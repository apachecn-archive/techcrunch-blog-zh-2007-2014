# SkyGrid 将实时新闻聚合器引入 Android 平板电脑

[SkyGrid，](https://web.archive.org/web/20230203091306/http://www.skygrid.com/)一家提供[强大商业新闻聚合器的初创公司，](https://web.archive.org/web/20230203091306/https://techcrunch.com/2009/04/20/skygrid-launches-real-time-financial-news-service/)今天将把它的新闻阅读器带到[安卓平板电脑](https://web.archive.org/web/20230203091306/https://market.android.com/details?id=com.skygrid.android.lge&feature=search_result)。该公司此前推出了[安卓、](https://web.archive.org/web/20230203091306/https://techcrunch.com/2010/12/04/skygrid-brings-realtime-news-aggregator-to-android-phones/) iPhone 和 [iPad 应用](https://web.archive.org/web/20230203091306/https://techcrunch.com/2010/04/23/skygrid-streamlines-realtime-news-consumption-with-ipad-app/)。

使用创业公司的专利算法，信息速度；SkyGrid 衡量哪些新闻在全球传播最快，并将这些内容直接引入其应用程序。用户将根据在全球传播最快的人物、话题和事件来了解相关的新闻。该应用程序本身可以从主流新闻、社交网站和博客中获取信息，并允许你通过电子邮件、Twitter 和脸书在应用程序上分享新闻文章和信息流。

您还可以通过类型、人物、主题等方式[关注新闻](https://web.archive.org/web/20230203091306/https://techcrunch.com/2011/05/18/skygrids-realtime-news-aggregator-now-allows-you-to-follow-topics-sites-and-more/)。您还可以创建自定义的新闻流，将不同类型的新闻组合在一起，跟踪新闻流，并访问 SkyGrid 的全球趋势特色新闻。

SkyGrid 首席执行官 Kevin Pomplun 表示，该应用程序的所有设计、布局和交互都是专为 Android 平板电脑量身定制的(而不是只有一个扩展到平板电脑的 Android 手机应用程序)。该应用程序还将发送带有新徽章图标的通知，或在平板电脑的 Android 状态栏上发送通知。

SkyGrid 的应用程序在 iPad 上特别受欢迎，目前在 App Store 中作为 iPad 新闻应用程序排名前十，因此这家初创公司预计 Android 平板电脑将受到用户的欢迎。
