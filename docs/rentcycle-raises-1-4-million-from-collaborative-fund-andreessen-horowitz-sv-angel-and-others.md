# Rentcycle 从 Collaborative Fund、Andreessen Horowitz、SV Angel 和其他公司筹集了 140 万美元 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/04/rentcycle-raises-1-4-million-from-collaborative-fund-andreessen-horowitz-sv-angel-and-others/>

被称为产品租赁的“ [OpenTable](https://web.archive.org/web/20230205032514/http://www.opentable.com/) ”的旧金山初创公司 Rentcycle 今天宣布，它已经筹集了 140 万美元的种子资金。这轮融资由[合作基金](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/company/collaborative-fund)牵头，这是一家由 [Craig Shapiro](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/person/craig-shapiro) 创立的年轻种子基金，投资于专注于点对点资源共享和商业创新的初创公司。

许多其他知名风投和天使投资人也参与了这轮投资，包括[安德森·霍洛维茨](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/financial-organization/andreessen-horowitz)、 [SV 天使](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/financial-organization/sv-angel)、[创始人集体](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/financial-organization/founder-collective)和 [Amicus Capital](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/financial-organization/amicus-capital-partners) ，以及天使投资人，包括[贝宝联合创始人兼 Yelp 董事长麦克斯·拉夫琴](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/person/max-levchin)，以及 Shopzilla [创始人法尔哈德·莫希特](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/person/farhad-mohit-2)。

作为种子资本注入的一部分，Rentcycle 还在其管理人员名单中增加了两名知名顾问:网飞创始人马克·伦道夫(Marc Randolph)和广受欢迎的餐厅预订服务创始人查克·坦普尔顿(Chuck templet on)。(邓普顿还将加入 Rentcycle 的董事会。据 Rentcycle 联合创始人兼首席执行官蒂姆·海尔(Tim Hyer)称，这家初创公司将利用这一轮来增加招聘，特别是在 rockstar 工程师领域。

那么，这家年轻的初创公司有如此令人印象深刻的投资者和顾问名单，并成为其集体名册的一部分，这是怎么回事呢？这位首席执行官告诉我，首先，产品租赁行业是一个 850 亿美元的空间，而它恰好是一个非常需要综合资源进行全方位租赁的行业。

正是因为这个原因，这家初创公司为复合行业(从工具和设备、派对和活动到运动和休闲)提供了目前在美国运营的许多夫妻店的完整目录。

因此，当从消费者的角度考虑 Rentcycle 时，该平台是一种允许人们轻松搜索、比较和在线租赁任何东西的服务——从滑雪板到手提钻和充气屋到设计师礼服。这一切都在同一个屋檐下。对于商家来说，Rentcycle 提供了基于云的管理解决方案，这不仅允许租赁企业将他们的实体运营带到网上，还允许他们在线跟踪他们的库存，并通过网络管理预订和支付。

Rentcycle 于 2009 年从硅谷孵化器 [Founder Institute](https://web.archive.org/web/20230205032514/http://www.crunchbase.com/company/founder-institute) 毕业，此后经历了几次测试版发布，最近一次迭代于今年 3 月上线。Hyer 说，自 3 月份以来，这家初创公司的月活跃用户已经增长到 3 万，而这只是专注于增加租赁供应并等待需求的到来。

毕竟，从消费者的角度来看，乍一看 Rentcycle 的价值主张是显而易见的:例如，对于那些可能仍然感受到经济衰退影响的人，或者那些可能不太去滑雪以至于不值得购买一双新滑雪板的人，租赁是最佳选择。当然，传统上，在网飞和 Airbnb 这样的网站登上主要舞台之前，租赁业务很少。

但消费者已经变得更愿意为产品和服务付费，作为直接购买的替代选择，Rentcycle 很可能会发现自己处于这一新浪潮的顶峰，因为它试图成为各种数字商务的一体化租赁资源。绝对值得一看。
