# 微软发布探索者触摸鼠标

微软今天宣布了其最新的触摸鼠标系列——探索者触摸鼠标。它的形状让人想起苹果的魔法鼠标，但功能类似于微软自己的 Arc 鼠标，前面和中间有一个触摸条。微软的 Explorer 触摸鼠标售价为 49.95 美元。

Explorer Touch 鼠标配有一个触摸条，可以进行垂直和水平滚动，鼠标底部有一个小口袋，可以在未连接到计算机时存放嵌入式 nano 收发器。这款鼠标宣称电池寿命长达 18 个月，并有五个可定制的按钮来保持快速运行。最值得注意的是，Explorer Touch 鼠标装载了 BlueTrack 技术，这意味着你基本上可以在你能找到的任何表面上使用这种鼠标。

Explorer Touch Mouse 要到 9 月份才会进入实体店，到那时，我们已经准备好了一篇实践评论，等着你。这款鼠标将有四种颜色:黑色、灰色以及微软所谓的桑格利亚红和铁锈红。
