# 脸书在翻译方面变得更加积极，增加了 22 种语言

![](img/796f440342a945eb68957fa4d318b143.png)

脸书不再只是摆弄几个[欧洲](https://web.archive.org/web/20221002130654/http://www.beta.techcrunch.com/2008/02/07/facebook-turns-1500-users-into-spanish-translation-slaves/) [语言](https://web.archive.org/web/20221002130654/http://www.beta.techcrunch.com/2008/03/02/facebook-now-in-german-thanks-to-2000-generous-users/)的翻译了。除了英语、法语、德语和西班牙语，他们正在使用新的用户驱动翻译引擎[将脸书翻译成另外 22 种语言。](https://web.archive.org/web/20221002130654/http://www.beta.techcrunch.com/2008/01/21/facebook-taps-users-to-create-translated-versions-of-site/)

用户翻译网站需要一些时间，脸书喜欢错开发布时间，以最大限度地提高公共关系。如果你想帮助这个项目，申请是[这里](https://web.archive.org/web/20221002130654/http://www.facebook.com/translations/)。
