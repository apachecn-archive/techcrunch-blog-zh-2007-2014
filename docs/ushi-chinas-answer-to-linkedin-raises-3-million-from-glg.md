# 优仕，中国版的 LinkedIn，从 GLG 和其他地方筹集了 300 万美元

商业专业知识市场[格尔森莱曼集团](https://web.archive.org/web/20230203091904/http://www.crunchbase.com/company/gerson-lehrman-group) (GLG)与中国商业社交网络服务提供商[优仕](https://web.archive.org/web/20230203091904/http://www.ushi.cn/index.jhtml;jsessionid=EDA0D582E600DA859FF2DC498A9F2A45)合作并收购了其少数股权。

GLG 的投资是最近一轮 300 万美元融资的一部分，使优仕的总资本增加到[450 万美元](https://web.archive.org/web/20230203091904/http://www.crunchbase.com/company/ushi)。优仕的早期支持者也参与了这轮融资，这是这家总部位于上海的公司迄今为止的第二轮融资。

这些早期投资者包括 Milestone Capital 创始合伙人[楼云丽](https://web.archive.org/web/20230203091904/http://www.mcmchina.com/team-1.asp)，Endeavor 联合创始人 [Peter Kellner](https://web.archive.org/web/20230203091904/http://www.rglobal.com/team) 和 [Jose Cheng](https://web.archive.org/web/20230203091904/http://www.lifunginvestments.com/asia.html) ，Li & Fung 私募股权基金董事总经理。

总部位于上海的优仕提供面向中国商人和企业家的社交网络服务，该公司表示，其成员包括超过 40，000 名首席执行官和 10，000 名首席技术官，以及来自中国 85%以上风险投资和私募股权公司的关键人物。

与 GLG 的投资和合作在战略层面上是有意义的。在中国，GLG 拥有一个由 25，000 多名主题专家组成的网络，他们可以根据需要实时参与交易采购、研究、尽职调查等。

优仕成员将被邀请加入 GLG 市场，以扩大公司在中国的专家网络。在 Ushi.com 的商业发展方面，GLG 将为优仕提供建议，优仕将给予解答。

后者是一种新的、只有受邀者才能参与的合作问答服务，最近向一组特定的 Ushi 成员推出。想想 Quora 的商业话题，显然是聚焦于中国的。
