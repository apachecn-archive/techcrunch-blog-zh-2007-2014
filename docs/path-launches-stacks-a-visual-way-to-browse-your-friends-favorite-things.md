# Path 启动了“Stacks ”,这是一种浏览朋友们最喜欢的东西的可视化方式

![](img/828b518c4c1db22281818f93db642154.png)今天在 TechCrunch Disrupt， [Path](https://web.archive.org/web/20221206205646/http://www.path.com/) 创始人兼首席执行官戴夫·莫林发布了最新版本的注重隐私的照片分享应用。它带来了一个关键的新功能，可以改变你使用服务的方式:堆栈。

这个功能最好被描述为 Path 的核心照片分享体验和脸书空间的结合。到目前为止，总是可以用人物、地点和事物来标记您共享的路径照片。但是从来没有办法真正浏览这些。现在，您将能够浏览按标签排序的相册，每个标签称为一个堆栈。

当你浏览一个朋友的个人资料时，你现在会看到一系列缩略图(实际上，它更像是一小叠照片)，代表他们最常互动的人、地点和事物。点击其中一个，您将看到包含此标签的朋友的照片。

莫林说，这个想法是让你接触你可能感兴趣的新地方和新事物。由于这个系统的工作方式，你的朋友最喜欢的东西会浮到顶部，这可以帮助你发现新的地点、要做的事情和朋友。
