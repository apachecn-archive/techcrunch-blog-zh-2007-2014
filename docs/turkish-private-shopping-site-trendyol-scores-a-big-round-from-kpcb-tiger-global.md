# 土耳其私人购物网站 Trendyol 获得了 KPCB 老虎环球的大力支持

谈到欧洲的初创企业，如果有一件事我一直反复听到的话，那就是没有认真看待土耳其市场的投资者可能会在未来 10 年错过欧洲大陆最大的一些数字公司。

我的计划是有一天去土耳其，近距离调查这个新兴的创业生态系统，我希望这次旅行能带我去一些神奇的城市和美丽的海滩。

我跑题了，我跑题了。

我们在土耳其的消息来源(当地科技博客 [Webrazzi](https://web.archive.org/web/20230203111246/http://www.webrazzi.com/2011/07/22/trendyol-yatirim/) )告诉我们[凯鹏华盈&拜尔斯](https://web.archive.org/web/20230203111246/http://www.crunchbase.com/financial-organization/kleiner-perkins-caufield-byers)和[老虎环球](https://web.archive.org/web/20230203111246/http://www.crunchbase.com/financial-organization/tiger-global)已经在私人购物网站 [Trendyol](https://web.archive.org/web/20230203111246/http://www.trendyol.com/default.aspx) 投资了 2500 万到 3000 万美元。

据报道，凯鹏华盈对伊斯坦布尔创业公司的估值约为 1.5 亿美元。

我的土耳其语有点生疏了，但看着这个网站，我会假设它是当地版本的[Vente-prive](https://web.archive.org/web/20230203111246/http://www.crunchbase.com/company/vente-privee-com)、 [Gilt](https://web.archive.org/web/20230203111246/http://www.crunchbase.com/company/giltgroupe) 、 [Beyond The Rack](https://web.archive.org/web/20230203111246/http://www.crunchbase.com/company/beyond-the-rack) 和 [HauteLook](https://web.archive.org/web/20230203111246/http://www.crunchbase.com/company/hautelook) 。当然，还有[谷歌的 Boutiques.com](https://web.archive.org/web/20230203111246/https://techcrunch.com/2010/11/17/google-boutiques/)、[易贝的时装店](https://web.archive.org/web/20230203111246/https://techcrunch.com/2010/04/05/ebay-launches-special-online-store-for-all-things-fashion/)和[亚马逊的 MyHabit](https://web.archive.org/web/20230203111246/https://techcrunch.com/2011/05/03/amazon-launches-private-fashion-sales-site-myhabit-com/) 。

据 Webrazzi 报道，Trendyol 早些时候筹集了 200 万美元和 1000 万美元的资金，一些股东在第三轮融资中套现。

据报道，这是凯鹏华盈在土耳其的第一笔投资。
