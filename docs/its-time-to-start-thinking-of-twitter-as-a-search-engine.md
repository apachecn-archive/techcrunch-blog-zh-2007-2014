# 是时候把 Twitter 当成一个搜索引擎了

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/03/05/its-time-to-start-thinking-of-twitter-as-a-search-engine/>

今晚与朋友共进晚餐时，话题转到了推特。他就是不明白，而且他肯定不是第一个这么告诉我的人。具体来说，我的朋友不明白 Twitter 在最近的[融资](https://web.archive.org/web/20230403233231/https://techcrunch.com/2009/02/13/twitter-raises-third-round-of-funding-from-benchmark-and-ivp/)中获得的巨额估值(2.5 亿美元或更多)。我告诉他为什么我认为这不仅仅是合理的:Twitter 最重要的是一个搜索引擎。

我告诉他我认为 Twitter 是一种微博服务:它是情感咕哝的集合。但它仍然很棒。有足够多的人迷上了它，Twitter 已经达到了临界质量。如果世界上有大事发生，你可以从推特上获得相关信息。

Twitter 还收集其他信息，比如人们与产品和服务互动时的体验。例如，几个月前，我被困在机场，受到了汉莎航空公司极差的服务。我在推特上表达了我的不满，这让我感觉好多了——至少我除了在无尽的队伍中等待之外，还在做一些事情。我也在推特上抱怨过 W 酒店(没有网络，房间很冷)和康卡斯特(常见的网络抱怨)。

越来越多的人开始在与品牌互动时使用 Twitter 实时谈论品牌。这些品牌想知道所有的事情，是单独回应(W 酒店纠缠我，直到我告诉他们别管我)，还是简单地收集信息，看看他们做得对还是错。

所有这些都可以在 Twitter 去年夏天收购的搜索引擎 search.twitter.com 发现。

人们搜索新闻。寻找反馈的品牌。那是有价值的东西。

推特也知道这件事。他们将在此基础上建立自己的商业模式。忘记用户为专业账户和其他功能支付的小额费用，他们所要做的就是不断扩大基础，并聚集越来越多的情感用户。总的来说，它非常有价值。正如谷歌所展示的那样，搜索是非常赚钱的——在 T2 的某个地方，所有在线广告收入的大约 40% 进入了今天搜索列表上的广告。

[正如约翰·巴特利](https://web.archive.org/web/20230403233231/http://battellemedia.com/archives/004832.php)所说，目前还不清楚谷歌或其他任何人是否能在这一点上与 Twitter 竞争(不过[脸书正在全力以赴](https://web.archive.org/web/20230403233231/https://techcrunch.com/2009/03/04/facebooks-response-to-twitter/))。

不仅仅是广告能带来收入。品牌需要工具来理解 Twitter 尚未提供的所有这些数据。我确信像 Scout Labs 这样的第三方将会自己挖掘这些数据。但 Twitter 还有很多其他方式可以对它们给品牌带来的效用征税。如果他们设法拒绝收购提议，就像脸书几年前做的那样，Twitter 没有理由找不到收入来源来支持他们成为一家独立的公司。甚至可能是公开的。
