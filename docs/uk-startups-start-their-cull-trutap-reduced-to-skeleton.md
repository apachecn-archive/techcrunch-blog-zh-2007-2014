# 英国初创公司开始裁员:Trutap 缩减为骨架

2007 年入选 TechCrunch 40 强的英国领先移动创业公司 Trutap 因未能获得第二轮融资，将解雇其 30 名员工中的近 80%。在出售或新投资之前，仅在重新推出一个月后，一名骨干员工将保持手机社交应用 Java 的可用性。这家初创公司在它的第一个投资者[都铎风险投资](https://web.archive.org/web/20221006021535/http://www.tudorventures.com/tudven/view/home.action)，一家受到经济危机重创的对冲基金，据传投入了 600 万美元，但无法回来进行第二轮融资后，就成了孤儿。今天，英国初创企业也受到了新闻的打击，有消息称，点评网站 [Reevoo](https://web.archive.org/web/20221006021535/http://www.reevoo.com/) 将五分之一的[员工职位列入“风险”名单](https://web.archive.org/web/20221006021535/http://uk.beta.techcrunch.com/2008/12/01/reevoo-hunkers-down-on-staffing-to-burn-slower-for-longer/)。
