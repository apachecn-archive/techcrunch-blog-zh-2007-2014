# YouTube 迪斯科停止跳舞，没人注意到

谷歌的 [YouTube](https://web.archive.org/web/20230205020351/http://www.crunchbase.com/company/youtube) 去年 1 月推出了一个[实验音乐发现项目](https://web.archive.org/web/20230205020351/https://techcrunch.com/2010/01/20/youtube-disco-music-discovery-project/)被称为 [YouTube Disco](https://web.archive.org/web/20230205020351/http://www.youtube.com/disco) ，使用户能够创建快速和肮脏的播放列表，并在飞行中发现新的艺术家和音乐视频。

这并不是什么大不了的事情，但一位读者告诉我们，这项功能是通过 YouTube 测试实验室 [TestTube](https://web.archive.org/web/20230205020351/http://www.youtube.com/testtube) 悄悄推出的，最近甚至停止了对“Lady Gaga”、“贾斯汀比伯”和“麦当娜”等查询的返回结果。伙计们，不要再寻找，混合或观看了。

**更新:**回来了！

也许这个团队觉得有必要关闭它，因为谷歌[砍掉了](https://web.archive.org/web/20230205020351/https://techcrunch.com/2011/08/25/slide-google-bloodbath/)Disco 应用。

幸运的是，迄今为止，似乎没有人注意到这一点。这可能是因为没有人使用它，当然，除了我们眼尖的读者。
