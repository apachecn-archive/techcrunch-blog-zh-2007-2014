# JavaScript 创始人兼 Mozilla 首席技术官布伦丹·艾希为 Ajax.org 出谋划策

![](img/6f3eda036f381b2fc56af6b70fc7bcac.png)

JavaScript 脚本语言的创造者、 [Mozilla](https://web.archive.org/web/20221224124202/http://www.crunchbase.com/company/mozilla) 的首席技术官布伦丹·艾希已经加入荷兰阿姆斯特丹[Ajax.org](https://web.archive.org/web/20221224124202/http://ajax.org/)的顾问委员会。

我们最近[写了关于](https://web.archive.org/web/20221224124202/http://www.techcrunchit.com/2011/02/28/ajax-org-debuts-development-as-a-service-platform-for-javascript-html5/) Ajax.org 对 [Cloud9 IDE](https://web.archive.org/web/20221224124202/http://cloud9ide.com/) 的介绍，这是一个商业的、基于云的 JavaScript 开发平台，整合了 HTML5，并支持 Python、Ruby 和 PHP。

Eich 因其在网景公司的工作而闻名，他于 1995 年 4 月开始工作，并发明了 JavaScript。1998 年初，他帮助创立了 Mozilla.org，担任首席架构师，后来又帮助创立了 Mozilla 基金会。

2005 年 8 月，在担任基金会董事会成员后，布兰登成为新成立的 Mozilla 公司的首席技术官。他的主要精力放在改进公司的明星产品，网络浏览器 Firefox 上。

他被任命为顾问委员会成员对 Ajax.org 来说当然是一个福音，该公司打算利用他的见解推动 Cloud9 的发展和推广。

该公司希望在 2011 年扩展到北美。
