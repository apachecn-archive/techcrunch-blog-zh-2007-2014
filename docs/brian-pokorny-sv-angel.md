# 在 Airbnb 人才交易 TechCrunch 之后，Brian Pokorny 以普通合伙人的身份重返 SV Angel

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/03/brian-pokorny-sv-angel/>

布莱恩·波科尔尼(Brian Pokorny)曾领导照片共享服务 Daily Booth 和 Batch 参与 Airbnb 的人才收购，他将以普通合伙人的身份重返 SV Angel。

他在这家公司工作了很长时间。自 2009 年以来，他一直是 SV Angel 的战略合作伙伴，后来他离职担任 DailyBooth 的首席执行官。自 2006 年以来，他还与罗恩·康韦合作进行天使投资。Pokorny 个人持有 Twitter、Square、OMGPOP、Milo、Chomp、TweetDeck、Circle、Bump、Posterous、Milk、Couple、Elepath 和 MessageMe 等公司的股份。

“投资对我来说感觉非常自然，”Pokorny 说。“你会看到各种各样的事情，你会非常接近处于早期阶段的初创公司。”

在第一次与 SV Angel 合作之前，他在谷歌工作。通过这种方式，作为 Daily Booth 的首席执行官和被 Airbnb 收购，他说他看到了公司的每个阶段，从早期阶段到高速增长期，然后成为一家成熟的大型公司。“我的激情在于早期公司和企业家，”他说，并补充说，他管理 DailyBooth 的经验给了他“创始人的视角”。你从招聘、招聘工程师、管理产品和管理发布会中学到了很多。"

作为一家 YC 支持的公司，DailyBooth [在 2009 年首次筹集资金时，有着非常有前途的吸引人数和发展势头。Pokorny 后来在初创公司](https://web.archive.org/web/20230218140408/https://techcrunch.com/2009/10/19/yc-funded-dailybooth-raises-1-million-from-sequoia-kevin-rose-ron-conway-et-al/)[内部领导了一场重大的战略变革，推出了照片共享应用 Batch](https://web.archive.org/web/20230218140408/http://uncrunched.com/2011/10/27/batch-may-be-the-perfect-mobile-photo-sharing-app-no-seriously/) ，专注于移动平台。

“DailyBooth 是一家以网络为先的公司，但用户行为是在摄像头前的行为，这种行为转移到了带有前置摄像头的手机上，”他说。“然后 Path 和 Instagram 推出了，那时我们有很多竞争对手。”

[该公司最终被 Airbnb](https://web.archive.org/web/20230218140408/https://techcrunch.com/2012/07/24/airbnb-brian-pokorny-batch-dailybooth/) 收购，Pokorny 在回到 SV Angel 之前从事招聘和其他社区及当地项目。
