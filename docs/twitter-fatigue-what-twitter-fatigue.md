# 有趣的是，我对 Twitter TechCrunch 并不感到疲劳

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/26/twitter-fatigue-what-twitter-fatigue/>

在《纽约时报》最近的一篇文章中，詹娜·沃瑟姆说她开始厌倦推特了。基本上，她把围绕贾斯汀比伯被捕的所有 Twitter 讨论作为一个跳板，引发一场更广泛的争论，即这项服务如何变得越来越不注重寻找有用的相关信息，而更注重争夺关注。

正如你所料，这篇文章得到了 T2 的赞扬和批评。最常见的批评似乎是“伙计，不要关注那些讨厌的人，”但正如[沃瑟姆](https://web.archive.org/web/20221006104200/https://twitter.com/jennydeluxe/status/427132622121074688)和[其他人](https://web.archive.org/web/20221006104200/http://blog.semilshah.com/2014/01/25/thoughts-on-jenna-worthams-article-about-twitter-dated-jan-25-2014/)所指出的，这说起来容易做起来难，因为不关注某人可能会令人尴尬或尴尬，即使你真的厌倦了他们的推文。

还有另一个原因可以解释为什么“不追随”的观点对我来说有点平淡。我认为，沃瑟姆小心翼翼地使用了“我们”这个词，尤其是在这篇文章的最佳段落中:

> 感觉好像我们都在努力成为深夜节目中的厚颜无耻的客人，真人秀选手或 Twitter 上戴着皇冠的蹒跚学步的孩子——通过一系列热辣、快速的评论、gif 或回应，让我们脱颖而出。我们相信，如果我们足够优秀，这可能是我们成功的门票，让我们在 BuzzFeed 或《赫芬顿邮报》的综述中获得短暂的一席之地，或者充其量是一份写作工作。但更多的时候，这意味着站在一个集体的肥皂箱上，互相推搡着让出空间，希望被认为发表了最聪明的俏皮话或反应。很多事情都是在欢声笑语中发生的。也许等量的疲惫随之而来。

在接下来的讨论中，“我们”似乎变成了“其他人”——当然，詹娜，*其他人*在 Twitter 上可能很烦人，所以你为什么不关注他们呢？默认的假设是，一定是其他人在 Twitter 上做错了，这很有趣。

就我个人而言，我觉得这篇文章很有价值，因为我立刻意识到了沃瑟姆所说的行为，而且不仅仅是在别人身上，而是在我自己身上。我怀疑我在 Twitter 上看到更广泛的对话时的默认模式是，“我怎样才能说一些有趣的事情，让每个人都谈论 meeeeeee？”(还要注意的是，沃瑟姆并没有直截了当地谴责这种行为——她承认这可能很有趣，但也会让人精疲力尽。)

因此，作为对个人经历的描述，我发现沃瑟姆的话是一个有价值的提醒，提醒我们至少要少自我推销，少自我关注。

另一方面，作为对“Twitter 的阿喀琉斯之踵”的更广泛的描述，我发现这篇文章不那么令人信服。正如其他人注意到的那样，沃瑟姆是一名跟踪近 4000 人的记者，拥有超过 50 万名追随者，因此她的经历几乎可以肯定是非典型的。她实际上在文章中提到了这一点，她认为，“我认为你的粉丝数量通常是无关紧要的，”但我对此表示怀疑。随着我自己的[比](https://web.archive.org/web/20221006104200/https://twitter.com/anthonyha)小得多，随着我的受众增加，动态也发生了变化，即使变化没有我预期的那么剧烈，随着我开始关注更多的人，噪音也是如此。

是的，沃瑟姆确实提出了一些非常有趣的轶事观点，但在我看来，她的文章在试图将这些观点转化为一个大的想法时遇到了困难。Twitter 变得信息量少了，自我宣传多了吗？以我的经验来看，它一直是一个混合体，我不确定这种混合体是否已经有了显著的改变。但同样，我们只是将个人经历的随机片段糅合在一起，并假装它对整个公司有意义。这条路充满了疯狂。(我说的“疯狂”是指“[询问青少年对脸书的看法](https://web.archive.org/web/20221006104200/http://www.slate.com/blogs/future_tense/2013/08/13/teens_on_facebook_ruby_karp_mashable_and_the_anecdotal_evidence_problem.html)”。)

但是，嘿，既然我们正在分享轶事证据，我将提供这个:我第一次听说沃瑟姆的文章是在哪里？我是在哪里第一次看到这篇文章中表达的大部分观点，无论是赞成还是反对的？当然是在推特上。
