Windows Phone副总裁揭穿了芒果9月1日的发布，我们仍然不相信

如果你正在关注 Windows Phone Mango (你完全应该关注它，因为它看起来非常棒)，你会记得我们在可能的9月1日发布中报道了的更新。可信但未透露姓名的袖珍棉绒来源证实了这一传闻。事情是这样的。我们完全正确。我们只是不确定我们对哪一部分是正确的——这是一个谣言，或者我们将在9月1日看到芒果。

微软Windows Phone副总裁Joe Belfiore发了一条推文驳斥9月1日的声明。" #芒果9月1日？只是谣言，”他在推特上写道。我们想相信你，乔。我们真的需要。但是Pocket-Lint倾向于用他们的谣言和很好的时机，第二天就是IFA。IFA是欧洲最大的电子产品展，Windows Phone在欧洲也非常受欢迎。把这些点联系起来？

公司经常在挖掘秘密和严格保守秘密之间走一条尴尬的路线。他们想让我们知道一些完全令人惊奇的事情将要发生，但是他们不想让我们知道在哪里，什么时候，或者什么情况。在这种情况下，如果微软不利用IFA来展示其最新最棒的操作系统，那将是相当令人震惊的。但是很容易理解贝尔菲奥里想要把猫带回袋子里，至少现在是这样。
