# Pinterest 还是 Pintrips？Pinterest 对旅游规划初创公司 TechCrunch 提起商标侵权诉讼

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/04/pinterest-files-a-trademark-suit-against-social-travel-startup-pintrips-claims-pinning-buttons-confusingly-similar/>

Pinterest 已经强调[不鼓励](https://web.archive.org/web/20221127014337/http://business.pinterest.com/brand-guidelines/)许多社交媒体公司在 Pinterest 上使用自己的品牌名称，其中一些比较突出的公司甚至[重塑了他们的服务](https://web.archive.org/web/20221127014337/https://beta.techcrunch.com/2013/03/19/pinfluencer-is-now-piqora-as-it-preps-to-add-tumblr-twitter-and-more-to-its-social-marketing-net/)。至于其他的呢？准备上法庭吧。

今天，Pinterest 对个人旅游规划初创公司 Pintrips 提起诉讼，指控其商标侵权、虚假指定原产地、不公平竞争和商标淡化。Pinterest 的律师在诉状中写道:“这起诉讼源于 Pintrips 决定采用一个与 Pinterest 相似得令人困惑的社交媒体品牌，并拒绝承认、讨论或补救它在消费者中造成的混乱。”。

就在同一周，Pinterest 赢得了一场 720 万美元的官司，并对一名叫钱进的域名抢注者发出了禁令，该抢注者注册了 100 多个听起来有点像 Pinterest 的名字。(这个人也申请了听起来很像 Quora 的商标。)而且这些并不是 Pinterest 卷入的唯一商标纠纷: [Path 已经申请延长反对 Pinterest 的美国商标申请的时间，因为这涉及到它的脚本 P](https://web.archive.org/web/20221127014337/https://beta.techcrunch.com/2013/09/06/pinterest-and-path-to-battle-over-letter-p-logo-trademark/) —这里的问题是它看起来太像 Path 的“P”。然而，我们知道这可以在没有任何法院诉讼的情况下得到解决。

我们在今年 4 月[写了关于 Pintrips 的文章](https://web.archive.org/web/20221127014337/https://beta.techcrunch.com/2013/04/16/pintrips-launches-a-collaborative-trip-planning-dashboard-for-tracking-flights-prices-across-destinations-in-real-time/)，当时它推出了一个“协作旅行计划仪表板，用于实时跟踪目的地的航班和价格。”

公平地说，如果你看看 Pintrips 的功能，你会发现它有很多是专门针对旅行规划的，与 Pinterest 目前的业务无关。

Pinterest 的 Pinboards 本质上是来自网络的图片链接集合，用户可以相互分享，并能够关注特定的 Pinners 或特定的主题。与此同时，Pintrips 是一个在线仪表板，用户可以在这里跟踪他们可能感兴趣的航班的价格变化。用户通过浏览器插件来核对航班。安装后，不同航班报价旁边会出现一个“Pin”按钮(目前支持该服务的包括美国航空、达美航空、捷蓝航空、西南航空、联合航空、美国航空、维珍航空，以及谷歌、Expedia、Kayak 和 Orbitz 上的搜索)。通过点击按钮，它们被添加到您的 Pintrips 仪表板。

事实上，对于 Pinterest 来说，更多的是它的名字，而不是功能。向美国加州北区地方法院提交的完整诉状(见下文)似乎基于几个关键点:

首先也是最重要的是名字的问题。在原告看来，Pintrips 听起来太像 Pinterest 了。如果你大声说出这两个名字，它们听起来确实很像。但更具体地说，Pinterest 已经列出了针对使用听起来太像自己名字的公司的规则。而且它已经开始追踪以这些名字进行交易的公司。“当 Pintrips 推出自己的社交媒体服务时，它可以采用任何数量的商标。相反，它选择了在外观、声音和商业印象上与 PINTEREST 相似的 PINTRIPS，”该公司在诉状中写道。“在这样做的时候，Pintrips 选择了一个在消费者中引起混淆的品牌，并暗示了一种并不存在的联系、从属关系或赞助。”

还有一个问题是，Pinterest 的社交网络上有很多与旅行相关的内容。“Pinterest 在旅游方面引起了特别大的轰动，”诉状写道。“迄今为止，Pinterest 用户已经在 Pinterest 的‘旅游’类别中发布了超过 6.6 亿个图钉。许多人使用 Pinterest 作为旅行规划工具。

它接着指出，有多少航空公司和酒店等使用 Pinterest 作为其服务的营销平台。(顺便说一句，随着付费广告的引入，这个平台[将很快开始赚钱，这也是为什么与一个类似的服务站在一起与 Pinterest 不合适的另一个原因。)](https://web.archive.org/web/20221127014337/https://beta.techcrunch.com/2013/09/19/pinterest-ads/)

也许最引人注目的是网站上的某些机制是相似的，并且有相似的名字。具体来说，Pinterest 认为 Pintrips 的“Pin”按钮与 Pinterest 的“Pin it”按钮“令人困惑地相似”。(它们在上面的图片中)

在某些方面，按钮可能是这种情况下更具破坏性的方面之一。Pinterest 依靠“锁定”按钮在网络上传播其服务，并让人们从他们访问的任何其他网站继续使用它。这些小按钮实际上也像是这项服务的小广告。在这个故事中，Pintrips 在很大程度上是 Pinterest 的歌利亚的大卫，但令人担忧的是，如果它增长，它在旅游网站上的 Pin 按钮可能会被误认为是 Pin-it 按钮。此外，如果 Pintrips 被允许保留其按钮，有什么能阻止其他人沿着这条路走下去呢？所有这些都意味着 Pinterest 的品牌稀释。

我们正在联系 Pintrips 寻求回应，当我们了解到更多信息时，我们会及时更新。Pinterest 对这个故事不予置评。
