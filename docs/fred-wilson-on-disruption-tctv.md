# 弗雷德·威尔逊谈颠覆:“你无法阻止人们最终希望发生的事情。”

纽约[互联网周](https://web.archive.org/web/20230203125446/http://www.internetweekny.com/)的特色是与联合广场的[弗雷德·威尔森](https://web.archive.org/web/20230203125446/http://www.crunchbase.com/person/fred-wilson)，预感的[克里斯·迪克森](https://web.archive.org/web/20230203125446/http://www.crunchbase.com/person/chris-dixon)和 SV 天使的[李尚义](https://web.archive.org/web/20230203125446/http://www.crunchbase.com/person/david-lee)进行小组讨论。我们带着相机参加了这次活动，并在整个[周](https://web.archive.org/web/20230203125446/https://techcrunch.com/2011/06/08/fred-wilson-platforms-valuations/)都在发布节选。在这段视频中，三位专家深入讨论了颠覆性问题。

迪克森通过谈论他认为已经成熟的颠覆领域开始了对话:教育、金融和政府——他说政府现任者和游说者阻碍了发展，没有推动变革的真正理由。Lee 拿起麦克风，将医疗保健抛入其中，然后说道:“这些系统中有很多低效之处……我们认为……任何时候有这种低效，就有机会。”

击球第三，威尔逊声称“政治和政府一直是一个可怕的投资场所，教育一直是一个可怕的投资场所，但这是因为根深蒂固的利益使其成为一个可怕的投资场所。你投资这些行业的方式是与既得利益者作对，你试图破坏既得利益者，而不是为他们服务。”

威尔逊特别提到了萨尔·汗，他通过他的汗学院创建了 2000 多个免费在线课程，并指出“汗学院的绝大多数人都不在美国。”他说，“如果你真的想看看世界是如何变化的，去撒哈拉以南的非洲，去东南亚，去东欧和世界上那些没有我们认为理所当然的东西的地方，看看他们是如何利用互联网来获得这些东西的，我认为在那里你会看到改变游戏规则的事情将要发生。”

三人接着讨论了电视、酒店(Airbnb)和音乐行业的颠覆，威尔逊指出，“你无法阻止人们最终希望发生的事情。”
