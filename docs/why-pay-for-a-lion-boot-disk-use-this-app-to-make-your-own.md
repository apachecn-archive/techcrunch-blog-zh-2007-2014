# 为什么要花钱买狮子启动盘？使用此应用程序制作您自己的应用程序

当 Lion 发布时，我们[将](https://web.archive.org/web/20230203044222/https://techcrunch.com/2011/07/20/lions-internet-recovery-feature-the-past-meets-the-future/)链接到一个详细介绍如何创建自己的可引导 Lion 驱动器的页面。但是如果你一定要有一个 GUI，并且不想在应用程序包里瞎折腾，你可能会发现[这个小工具](https://web.archive.org/web/20230203044222/http://blog.gete.net/lion-diskmaker-us/)很有用。

Lion Diskmaker 是一个简单的应用程序，它从一个胚胎 Lion 安装(即您从 Mac App Store 下载的包)中提取必要的项目，并将它们写入 USB 驱动器或 DVD。就是这样！一个单一服务的应用程序，我相信很多人会觉得很有帮助。

你所需要的是一个 4g 的可移动卷(SD 卡，拇指驱动器，DVD)和一个随时可以安装的 Lion。不要忘记通过捐赠或者至少在之后的[来感谢作者。](https://web.archive.org/web/20230203044222/http://twitter.com/ggete)
