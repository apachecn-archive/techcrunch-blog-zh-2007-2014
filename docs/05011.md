# 谷歌语音行动的下一步是什么？与谷歌移动团队(TCTV)的讨论

记者会可以来找你，为什么还要去参加记者会？

今天早上[谷歌](https://web.archive.org/web/20221007135711/http://www.crunchbase.com/company/google)在三藩市举行了一场新闻发布会，发布了一些新的 Android 功能，包括[谷歌语音操作](https://web.archive.org/web/20221007135711/https://beta.techcrunch.com/2010/08/12/google-voice-actions/)。目前由 13 个“动作”组成，这个时尚的应用程序允许用户口头命令他们的手机执行一系列广泛的[任务](https://web.archive.org/web/20221007135711/http://www.google.com/mobile/voice-actions/)，例如给朋友打电话，寻找方向，口述和发送电子邮件/短信，在谷歌搜索上呼叫任何企业，查找歌曲等。正如我的同事 MG Siegler 所写的，它“惊人地快速和准确”,并且只在 Android 上可用。

活动结束后，谷歌移动团队的几位关键成员——包括雨果·巴拉(产品管理总监)、迈克·勒博(高级软件工程师)、费尔南多·德尔加多(产品经理)和戴夫·伯克(工程经理)——拜访了 TechCrunch 的旧金山总部，让我们更深入地了解了 Android 的华丽新玩具。Barra 和 LeBeau，你可能会认为他们是可爱又傻傻的演示视频[中的明星](https://web.archive.org/web/20221007135711/http://www.youtube.com/watch?v=gGbYVvU0Z5s&feature=player_embedded)，在 [TechCrunch TV](https://web.archive.org/web/20221007135711/http://techcrunch.tv/) 上讨论了谷歌语音操作及其技术的未来。见上面的视频。

有一些有趣的披露:

**谷歌认为语音将在短信领域发挥重要作用**。虽然语音命令网络查询在 Android 平台上的受欢迎程度激增，四分之一的查询是通过语音搜索进行的，但巴拉认为谷歌可以在短信领域实现类似的数字。“我很想让它成为手机短信的四分之一，对吧，如果这项技术如此之好——我认为是这样的——我会觉得用它发送大部分短信很舒服，那该多好啊。”

将来会有更多可用的谷歌语音操作。预计下一次发布将关注开放性和与第三方开发者的集成。LeBeau 说:“我们认为这只是冰山一角…我们在这个版本中只是略微提到了与第三方应用程序的集成…我们肯定会想办法尽可能开放这一点。”。如果你想知道谷歌为语音动作准备了什么的另一个提示，请收听柯克船长频道。语音动作的设计师 LeBeau 说，他从《星际迷航》(Star Trek)的《T3》(不清楚他是指原版系列、下一代、深空九号还是其中一部故事片)中获得了**的大量灵感。**

**谷歌语音行动和谷歌语音的转录服务共享类似的技术。谷歌语音转录(将语音邮件转换为文本)一直不太准确，因为从语音到应用程序的路径不太直接(即直接对着手机说话可以提高准确性，而之前记录的内容不太清楚)。**“他们肯定分享了许多相同的核心技术，其中一些最大的区别是，在谷歌语音邮件或 YouTube 转录的情况下，我们试图转录音频的情况实际上要困难得多。因此，用户拿起手机，按下一个按钮，知道他们正在对系统说话，这是一种非常不同的体验，在音频和扬声器质量方面，他们试图通知的方式比你在语音邮件或 YouTube 上找到的有很多扬声器的东西更高质量的体验……我们正在分享从这两方面学到的知识，以建立能够很好地做这两方面事情的核心技术，”LeBeau 说。

**根据 Barra 的说法，谷歌还没有开始让谷歌语音操作兼容其他操作系统(即 iPhone)的进程，但这将会发生。“我们还没有开始将语音操作引入其他平台，但我们对此非常感兴趣。会有一些限制，有很多事情我们无法在其他平台上做，因为它们不像 Android 那样开放或容易使用，但正如我们对几乎每一个产品所做的那样，我们试图将它们带给尽可能多的用户，当然这意味着要去所有的平台。这是我们希望在不久的将来开始的工作，”Barra 说。**
