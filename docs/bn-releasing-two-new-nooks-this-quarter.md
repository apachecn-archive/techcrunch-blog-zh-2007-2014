# B&N 本季度发布两款新游戏

数字阅读器的忍者们发现了两款新的 Nook Color 版本的名字——售价 249 美元的 Encore 和售价 349 美元的 Acclaim。Nook Touch(现名为 Nook Smart Touch)仍将保持 139 美元。

这些新设备是什么？数字阅读器的 Nate 认为 Smart Touch 是 6 英寸的 Nook Touch，而 Encore 是 7 英寸的 Nook Color。除了 Acclaim 也是一家倒闭的游戏公司，我们对新设备知之甚少。所有这些产品都将在第四季度上市，鉴于[亚马逊](https://web.archive.org/web/20230203055007/https://techcrunch.com/tag/Amazon)竞相打造安卓阅读器，它们不会来得太快。Nook Color 和 Nook Touch 都是优秀的设备，但在视网膜屏幕和蜂巢的时代，这种颜色有点过时。一次强劲的更新会让 B & N 在接下来的几个月里保持势头。
