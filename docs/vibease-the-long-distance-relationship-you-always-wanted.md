# 你一直想要的异地恋

最近，我们已经看到相当多的初创公司试图破坏关系。还有 [Dejamor](https://web.archive.org/web/20221205193332/https://beta.techcrunch.com/2012/08/11/dejamor-keeps-your-sex-life-sexy-every-month/) ，每月送你一盒浪漫， [Boink Box](https://web.archive.org/web/20221205193332/https://beta.techcrunch.com/2012/07/27/boink-box/) ，送你一盒情趣用品，我好像还记得在 CES 上和 OhMiBod [的一次有点尴尬的采访。](https://web.archive.org/web/20221205193332/https://beta.techcrunch.com/2012/01/13/all-the-little-but-awesome-things-you-definitely-missed-at-ces/)

今天，又一家性感的创业公司加入了这个领域。它被称为 [Vibease](https://web.archive.org/web/20221205193332/http://www.vibease.com/) ，正如他们所说，它是一个“按摩器”，通过蓝牙连接到你的智能手机上。你可以发短信聊天，有点像[对](https://web.archive.org/web/20221205193332/https://beta.techcrunch.com/2012/03/23/pair-is-a-path-for-the-two-of-us/)，而你的爱人可以在全国范围内控制你的按摩器，反之亦然。

该应用程序提供文本和图片信息，以及创建自定义振动的能力。你也可以设置一个昵称，而不是使用你的真名(上帝禁止一些朋友或妈妈看到你的名字在屏幕上闪现在“Vibease”旁边)。

目前，Vibease 还没有为市场准备好蓝牙按摩器。“不会花很长时间，”他们说。但与此同时，Vibease 应用程序将使用手机的振动器“作为一种挑逗”此外，还有一个“独奏模式”，周围的噪音如下雨或深呼吸，让你放弃它，就像它是热的一样。

Vibease 今天才在 [Google Play](https://web.archive.org/web/20221205193332/https://play.google.com/store/apps/details?id=com.vibease.app) 上发布，iOS 将在不久的将来推出。该团队希望在推出蓝牙兔之前获得应用程序的牵引力和反馈。
