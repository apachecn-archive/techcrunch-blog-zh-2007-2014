# Rovio 发布了受梦工厂动画电影启发的“The Croods”游戏

【YouTube http://www.youtube.com/watch?v=LdF23AbGPtc&w=640&h=360]

Rovio 刚刚发布了基于梦工厂动画电影[“The Croods”](https://web.archive.org/web/20230306151326/http://www.rovio.com/en/our-work/games/view/52/the-croods)的最新游戏。

这家总部位于芬兰的手机游戏公司与梦工厂合作，让这款游戏成为现实，在游戏中使用了你可能在电影中看到的相同角色和场景。很简单:克罗德人是一个试图用新思想超越石器时代的穴居家庭。

从一处风景到另一处风景，克鲁德人努力狩猎和采集食物来维持生命，一路上发现和驯服新的生物。

我说的“生物”，实际上是指“生物”Rovio 用 Girelephant 或 Molarbear 这样的名字描述这些生物，称它们不像“你见过的任何东西”。在 Rovio 的[博客](https://web.archive.org/web/20230306151326/http://www.rovio.com/en/news/blog/277/angry-birds-toons-series-landing-on-a-screen-near-you/)上的新闻稿中甚至有一丝口袋妖怪的味道:“你认为你能抓住他们所有人吗？”

你也可以从家族族长 Grug 那里得到帮助，创造发明，有点像坏小猪创造交通工具。如果这还不够，用户还可以用配件定制他们的角色和房子。

Rovio 在与电影公司合作项目方面有着丰富的经验。该公司与 20 世纪福克斯公司的里约电影一起发行了《愤怒的小鸟》,而《愤怒的小鸟》《星球大战》自问世以来一直很受欢迎。

考虑到 Rovio 有自己的[卡通计划](https://web.archive.org/web/20230306151326/https://techcrunch.com/2013/03/11/rovios-angry-birds-toons-to-be-bundled-in-companys-mobile-games-will-arrive-march-17/)，动画短片将通过他们自己的游戏应用程序每周发布，合作也是有意义的。

这款应用现在可以在 iOS 和安卓系统上使用。
