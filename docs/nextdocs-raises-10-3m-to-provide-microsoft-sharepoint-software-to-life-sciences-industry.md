# NextDocs 筹集 1030 万美元为生命科学行业提供微软 SharePoint 软件

为生命科学行业销售基于微软 SharePoint 软件的公司 NextDocs 已经从 OpenView Venture Partners 获得了 1030 万美元的首轮融资。

NextDocs 帮助生命科学公司利用基于 SharePoint 的文档和管理软件。该公司实际上为制药、医疗设备和生物技术行业的公司定制 SharePoint。实际上，NextDocs 在整个生命科学行业拥有 100 多家客户(包括美国十大制药公司中的[五家)。](https://web.archive.org/web/20230203104005/http://technicallyphilly.com/2011/08/15/nextdocs-ceo-zikria-syed-rapid-growth-in-a-niche-life-sciences-market-from-king-of-prussia)

该公司正以惊人的速度增长。NextDocs 预计今年将获得 1500 万美元的收入，在过去两年中规模已经翻了一番。NextDocs 表示，它将把新资金用于产品开发、客户服务和支持，以及向新市场的扩张。
