Save For Later Service Pocket对其API进行了彻底改革，以使其10，000名开发合作伙伴的工作变得更加轻松

Pocket是一种保存内容以供日后查看的在线服务，它的前身是Read It Later，今天宣布了其API 的全新版本，旨在为其开发合作伙伴提供更简单、更好的服务，这些合作伙伴推动了其总活动的很大一部分。该公司称，Pocket 45%的储蓄来自第三方。

虽然Pocket曾讨论过今年夏天有望达到每天100万次存储的目标，但该公司现在透露，它实际上每天都超过了这一里程碑。事实上，Pocket声称它现在是包括Zite在内的某些应用程序中使用最多的共享服务之一，与Twitter和脸书等重量级应用相抗衡。

改进后的API引入了OAuth 2，让用户可以更快地在第三方应用中激活pocket，并进一步减少围绕节省的摩擦，以及一个用于轻松iOS和Mac开发的Objective-C SDK。Pocket最近还增加了一些新功能，包括收藏夹、缩小特定内容范围的过滤器、更好的标签和按域搜索。

显然，Pocket的API对其整体战略至关重要，占其参与度的近一半。开发人员越容易实现口袋集成，他们就越有可能实现，从而扩展现有的覆盖范围。该公司还建立了一个全新的开发者网站，为感兴趣的第三方提供更加集中和有组织的支持。

随着对API的重新关注，这家初创公司可能正在努力通过这一渠道创造收入，因为在从付费应用模式转变为Read It Later之后，尚不清楚它计划如何赚钱。创始人内特·韦纳(Nate Weiner)在早些时候的博客文章中解释了为什么Pocket尽管盈利却选择成为免费服务，但对未来的收入计划保持沉默。然而，该公司确实在7月份筹集了500万美元的B轮，以帮助燃料平台和设备的扩张。
