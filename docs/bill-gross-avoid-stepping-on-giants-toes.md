# 比尔·格罗斯谈到他将如何避免踩到巨人的脚趾

本周早些时候，比尔·格罗斯[在 Web 2.0 峰会上发布了](https://web.archive.org/web/20230204234059/https://techcrunch.com/2011/10/18/gross-chime-in-follow-part-person/)一款名为 [Chime.in](https://web.archive.org/web/20230204234059/http://chime.in/) 的新手机应用和网站。我在走廊里把格罗斯拖到我们的摄像机前，他给了我一个[的演示](https://web.archive.org/web/20230204234059/https://techcrunch.com/2011/10/20/video-demo-bill-gross-chimes-in/)，然后坐下来接受采访。“Chime.in 的目标是建立一个兴趣网络，”他在上面的视频中告诉我。

Chime.in 的全部意义在于帮助过滤社交网络中的所有噪音。它不仅让你关注人，还让你关注兴趣，并按主题对你的信息流进行分类。用逆向年表来阅读一切有其局限性。“时间递减法的问题在于，”格罗斯说，“它并不总是把最相关的东西放在最上面。”

在下面的视频中，我认为不可避免的是，脸书和 Twitter 等其他社交网络也必须在某个时候添加兴趣层。他将如何避免冒犯他们？格罗斯有与 Twitter 共舞的经验，Twitter 关闭了他的公司 UberMedia 的访问权限，最终在 Twitter deck 的出价上超过了他。

正如你将在视频中看到的，格罗斯并不太担心。
