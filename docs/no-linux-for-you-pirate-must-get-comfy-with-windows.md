# 没有适合你的 Linux:盗版者必须适应 Windows

因下载《星球大战》第三集入狱五个月——这似乎有点苛刻，不是吗？斯科特·麦考斯兰得到的就是这些，此外，他还被命令用基于 Windows 的系统替换他的 Ubuntu Linux 操作系统，以确保与法院下令安装的计算机监控软件兼容。

虽然原文会让你相信斯科特是因为下载电影而入狱的，但实际上是他*将电影上传到他管理的 [EliteTorrents 网站](https://web.archive.org/web/20221001124957/http://www.slyck.com/story1554_EliteTorrents_Interview)的，因此被判重刑。*

麦卡斯兰目前正试图在他的网站上筹集捐款来购买 Windows XP 或 Vista…

> “我对颠覆对我的限制不感兴趣。我不是一个程序员，我不是一个 l337 LINUX 大师，我也没有兴趣回到监狱。我只是认为，因为他们不支持而强迫我购买操作系统是完全荒谬的。我的认罪条件是:“我同意我的首席采购官定期检查我的电脑和/或安装软件来监控我的互联网活动。”我同意这一切，但碰巧他们用来监控的软件可能不支持我使用的操作系统。所以我不认为(我的律师也不认为)政府可以强迫我更换操作系统。我有义务在学校使用电脑(也可以上网)，Windows 不是必须的。"

斯科特，无论你做什么，都要抵制装盗版 Windows 的诱惑。法官可能想看看真品证书。
