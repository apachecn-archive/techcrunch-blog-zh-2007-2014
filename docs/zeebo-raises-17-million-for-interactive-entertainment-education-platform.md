# Zeebo 为互动娱乐、教育平台募集 1700 万美元

根据一份提交给证券交易委员会的文件[，Zeebo](https://web.archive.org/web/20230204215049/http://sec.gov/Archives/edgar/data/1470124/000101968711002836/xslFormDX01/primary_doc.xml) ，一家雄心勃勃的[目标](https://web.archive.org/web/20230204215049/http://zeeboinc.com/about-zeebo/)“帮助下一个十亿人玩耍、学习和联系”的公司，已经筹集了[1700 万美元](https://web.archive.org/web/20230204215049/http://www.crunchbase.com/company/zeebo)。

当该公司成立时，该公司声明的意图是创建一个[廉价的游戏主机](https://web.archive.org/web/20230204215049/http://news.cnet.com/8301-13506_3-10252999-17.html)，通过无线数字分发提供负担得起的游戏和教育内容(以规避盗版)。Zeebo 面向发展中市场，如金砖四国(巴西、俄罗斯、印度和中国)和墨西哥。

然而，Zeebo 最近似乎搁置了这些计划，关闭了其在巴西和墨西哥的网站/业务，现在正致力于一个“互动娱乐和教育平台”，将于 2012 年首次亮相。

Zeebo 由总裁兼首席执行官 Mike Yuen 和 [Reinaldo Normand](https://web.archive.org/web/20230204215049/http://www.linkedin.com/in/reinaldonormand) 共同创立。Yuen 从高通来到 Zeebo，在那里他是游戏和服务的高级总监。

Philip White(首席财务官兼业务运营 SVP)也来自高通，他在那里担任过各种高级管理职务。

值得注意的是，高通的风险部门是 Zeebo 的主要投资者。
