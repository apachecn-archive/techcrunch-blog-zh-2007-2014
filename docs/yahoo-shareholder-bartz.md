# 愤怒的雅虎股东质问巴茨并索要她的人头

![](img/4d1d41ee3dc9450c9254a6ac5b390b39.png)

雅虎首席执行官卡罗尔·巴茨昨天在公司年度股东大会结束时受到了指责。一名愤怒的股东自称史蒂夫·兰德里(Steve Landry)，是“拥有数百万雅虎股票”的机构投资者的顾问。鉴于该股过去三年的总体表现不佳，在那之前令人惊讶的感觉良好的事情结束时，他站起来要求砍掉她的头。我们已经上传了下面的音频剪辑。

他列举了一系列问题和担忧，称雅虎是“灾难”和“马戏团”。他从“房间里的大象”开始:寻找巴茨的继任者。

他说:“本周早些时候，一个博客透露，董事会正在秘密与其他首席执行官候选人谈判。”“我听说过类似的细节。我相信这是真的。雅虎现在最不需要的就是一个跛脚鸭 CEO。关于你的合同的收购谈判需要从今天开始。”

嗯，我想知道他说的是哪个博客

兰德里还提到了收购 Hulu 的传言([，这是错误的](https://web.archive.org/web/20230204140651/https://techcrunch.com/2011/06/21/source-this-huluyahoo-story-is-bs/))，并暗示杰森·基拉尔可能是雅虎的一个不错的首席执行官候选人。但他认为收购 Hulu 是个坏主意。

最后，他呼吁董事会解散或出售公司。“现状不再起作用了，”他说。

对此，巴茨回应道:“不客气，感谢你的观点、博客作者的观点和谣言。还有什么？太棒了。那确实令人沮丧。"
