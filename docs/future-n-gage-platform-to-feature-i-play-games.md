# 以 I-play 游戏为特色的未来 N-Gage 平台

诺基亚新的 N-Gage 平台将于今年秋天发布，它将提供 I-Play 上有趣的人们带来的游戏娱乐。I-play 的首批可用游戏将是，世界拉力锦标赛和超级麻将。随着游戏的重要性和盈利能力在移动行业的实现，更多的设备发布必然会与娱乐软件捆绑在一起。

“I-play 为 N-Gage 带来了多样化的游戏产品组合，我们很高兴能有他们加入，”诺基亚第三方出版总监 Gregg Sauter 说。“N-Gage 工具集提供了通过 I-play 强大的内容组合向消费者提供新功能和更丰富体验的机会。我们期待这次合作。”

I-play 游戏以其简单易用而闻名，同时对你的拇指和按键也很放松。但是这些游戏并不容易。掌握这些宝石需要智慧和技巧。
