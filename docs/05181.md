# Foursquare、Scribd 和 Spotify 在达沃斯被称为 2011 年技术先锋

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/09/01/technology-pioneers-2011-foursquare-scribd-spotify/amp/>

 <amp-img src="https://web.archive.org/web/20230404185206im_/https://techcrunch.com/wp-content/uploads/2009/12/wef.jpg" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 

世界经济论坛公布了 2011 年 31 位科技先锋的名单。技术先锋是它的新兴创业公司名单。去年的榜单包括 Twitter、Playfish 和波士顿电力。在之前的[年，Mint，Etsy，Brightcove 被命名。](https://web.archive.org/web/20230404185206/https://techcrunch.com/2008/12/04/mint-joins-the-world-economic-forum-knows-that-youve-cut-back-on-starbucks/)

今年加入先贤祠的有 foursquare、Knewton、Layar、Scribd 和 Spotify。然而，绿色科技在名单上同样强大，特别是智能电网公司，如 [OPower](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/opower) 和 [Tendril](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/tendril) 。以下是成功入选的信息技术公司的完整列表，并提供了它们的 Crunchbase 简介链接以获取更多信息:

[Aster Data](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/aster-data-systems) (大数据)
[Atlassian](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/atlassian) (产品管理)
[Foursquare](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/foursquare) (地理位置)
[GetJar](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/getjar-networks)
[Knewton](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/knewton)(自适应学习)
[Layar](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/layar) (增强现实)
[网秦移动](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/netqin-tech-co-ltd)(移动安全)
[OpenDNS](https://web.archive.org/web/20230404185206/http://www.crunchbase.com/company/opendns) (域名)【域名】
