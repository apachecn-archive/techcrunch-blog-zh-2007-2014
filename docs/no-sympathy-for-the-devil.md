# 对硅谷的恐惧和厌恶 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/06/18/no-sympathy-for-the-devil/>

欢迎来到 2013 年的夏天。当你不确定你的哪些网络活动被跟踪时，欢迎来到这个夏天。当你想开始 [Snapchatting](https://web.archive.org/web/20221006021451/http://www.snapchat.com/) everyone 时，因为至少那时数据会“消失”除了当它[不](https://web.archive.org/web/20221006021451/http://articles.latimes.com/2013/may/11/business/la-fi-tn-snapchat-photos-restored-300-20130510)的时候？

今年夏天，尽管去年甚至十几年前就有[的阴谋被明确报道，但美国国家安全局对我们的脸书和 Google+个人资料进行](https://web.archive.org/web/20221006021451/http://www.wired.co.uk/magazine/archive/2012/05/features/the-black-box)[某种链接和因素分析，或者至少是元数据收集](https://web.archive.org/web/20221006021451/http://www.nytimes.com/2013/06/08/technology/tech-companies-bristling-concede-to-government-surveillance-efforts.html?smid=tw-share&_r=2&)的披露，已经让我们达到了技术恐惧的顶峰。

整个春天都有前震。 [Valleywag](https://web.archive.org/web/20221006021451/http://valleywag.gawker.com/) 的暴力重现；对肖恩·帕克婚礼的[不幸和错误的抽象](https://web.archive.org/web/20221006021451/http://www.theatlantic.com/technology/archive/2013/06/new-government-documents-show-the-sean-parker-wedding-is-the-perfect-parable-for-silicon-valley-excess/276521/)，从所有意图和目的来看，这应该是一个私人事件，变成了硅谷“过度”的象征；另类出版物对我们的[培根包裹方式](https://web.archive.org/web/20221006021451/http://www.eastbayexpress.com/oakland/the-bacon-wrapped-economy/Content?oid=3494301)的惊人报道；甚至*《纽约客》* [权衡于](https://web.archive.org/web/20221006021451/http://www.newyorker.com/reporting/2013/05/27/130527fa_fact_packer)、[再](https://web.archive.org/web/20221006021451/http://www.newyorker.com/online/blogs/elements/2013/04/silicon-valley-start-ups-and-the-end-of-stanford.html)、[再](https://web.archive.org/web/20221006021451/http://www.newyorker.com/online/blogs/comment/2013/06/big-brother-and-silicon-valley.html)。

保罗·克鲁格曼和其他人正在预测技术启示录，或者 T2 奇点，或者两者皆有。如果你经营一个科技博客，请增加人手。我们[都是](https://web.archive.org/web/20221006021451/https://beta.techcrunch.com/2013/05/22/we-want-you-to-be-the-new-techcrunch-startup-battlefield-editor/)，尽管我们这个领域最大的新闻[现在被主流媒体打破了。](https://web.archive.org/web/20221006021451/http://www.techmeme.com/130606/p50#a130606p50)

许多人认为，我们的[公司班车](https://web.archive.org/web/20221006021451/http://nymag.com/daily/intelligencer/2012/12/silicon-valleys-exclusive-shuttles.html)、[膨胀的房价](https://web.archive.org/web/20221006021451/http://valleywag.gawker.com/silicon-valley-housing-explosion-means-this-cottage-cos-513817096)、社会泡沫[、日夜不停的 iPhones 手机](https://web.archive.org/web/20221006021451/http://www.vanityfair.com/culture/my-phone/2013/03/dave-morin-path-facebook-apple)以及用资本取代劳动力的热情，是值得大众怨恨的目标。一个[“独裁统治”的产生。](https://web.archive.org/web/20221006021451/https://twitter.com/tolles/status/341983111619416065)“2010 年， *[《社交网络》](https://web.archive.org/web/20221006021451/http://www.rottentomatoes.com/m/the-social-network/)* 高居票房榜首。2013 年， *[实习](https://web.archive.org/web/20221006021451/https://beta.techcrunch.com/2013/06/07/an-ex-googlers-take-on-the-internship/)* 因为普遍的硅谷[厌倦和疲劳，勉强跻身前四。](https://web.archive.org/web/20221006021451/https://medium.com/p/577832af331d)

然后是整个"[在我们不知情的情况下帮助政府收集全世界的私人数据](https://web.archive.org/web/20221006021451/http://www.guardian.co.uk/world/2013/jun/17/edward-snowden-nsa-files-whistleblower)"的事情。“相信我们。”还记得[安然吗？这就是现在的我们。](https://web.archive.org/web/20221006021451/http://en.wikipedia.org/wiki/Enron)

科技行业的谬误在于，我们认为我们“改变世界/连接世界”的意图已经足够了，或者至少它们应该保护我们免受指责，就像我们封闭的 Ubers、Airbnbs 和 TaskRabbits 社区一样。我们陶醉于财富的大规模集中、私人公共交通、私立高科技学校，以及政府愚蠢的潜在意识形态。我们是免税的。

嗯，除了当国家安全局要求合作的项目，[我们创立的公司可能应该遵守](https://web.archive.org/web/20221006021451/http://www.dailymail.co.uk/news/article-2342200/Revealed-Yahoo-FOUGHT-NSAs-warrantless-spying-program-lost-forced-secret-court-join-PRISM.html)——否则上帝知道下次我们因违反反垄断法被起诉会发生什么。

硅谷正遭受一种严重的合成谬误:仅仅因为它做了一些好事，并不意味着整体就是好的。科技并非高于危害社会。仅仅因为变化(即[颠覆](https://web.archive.org/web/20221006021451/https://beta.techcrunch.com/events/disrupt-sf-2013/event-info/))是不可避免的，并不意味着它总是受欢迎的。

机关枪是创新。他们打乱了步枪。他们还在二战中破坏了很多人体。药品拯救生命。但是他们也让人们麻木了情感上的痛苦而不是面对它，让他们的孩子安静下来而不是教导他们。社交游戏可以被看作是娱乐和放松。他们也可以被视为剥夺我们时间和注意力的小偷。

科技行业尤其不适合解决自己的问题，因为它错误地认为自己生活在精英统治下，从而避免了巨大的负罪感。见鬼，甚至那些写关于 it 的博客的人也很有钱。

就像技术取代工作的问题一样，技术假装无辜是没有解决办法的。作为书呆子和失败者，我们总是相信我们有最好的意图。这并没有否定问题:即使我们不是华盛顿特区，我们仍然是一个拥有大量权力、关注和金钱的行业。以及大量有意和无意滥用它的机会。
