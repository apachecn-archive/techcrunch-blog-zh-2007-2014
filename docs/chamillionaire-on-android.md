# Chamillionaire 恳求 Android 设计领导给他一个体面的设备

"我用过所有这些不同的设备——它们都很糟糕。"

短。太好了。切中要害。你很少在科技会议上发现这一点。幸运的是，百万富翁喜欢来我们这里。

这是嘻哈明星今天在我们的[移动第一次仰卧起坐](https://web.archive.org/web/20230203082604/https://techcrunch.com/2011/07/29/watch-the-mobile-first-crunchup-live-here/)期间对安卓设计负责人[马蒂亚斯·杜阿尔特](https://web.archive.org/web/20230203082604/http://www.crunchbase.com/person/matias-duarte)说的话。在闭幕会期间，Chamillionaire 站起来表达了他对 iPhone 的热爱，以及他对 Android 甚至无法创造出一款与之匹敌的设备的失望。

显然，这引起了观众的强烈反应。杜阿尔特泰然处之。他告诉 Chamillionaire，每个人都有不同的品味([类似于当天早些时候他对 iOS 与 Android 设计的争论](https://web.archive.org/web/20230203082604/https://techcrunch.com/2011/07/29/beauty-is-in-the-i-of-the-beholder/))。他承认苹果确实制造了很棒的设备，像微软和惠普这样的竞争对手也是如此。

但是百万富翁继续施压。他想要一个简单明了的答案。他应该使用哪种 Android 设备？“我希望很快我们就能有一部你会喜欢的安卓手机，”杜阿尔特回应道，似乎在暗示一款新的安卓设备即将到来。

当我们的 Erick Schonfeld 问 Duarte 他用的是什么设备时，Duarte 说是 Nexus S。目前，这是他推荐的一款设备。亿万富翁同意尝试一下。我们会看到——我肯定他会在推特上发表他的想法。
