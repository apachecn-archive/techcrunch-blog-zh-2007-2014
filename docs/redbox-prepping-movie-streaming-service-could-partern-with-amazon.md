# 红盒子准备电影流媒体服务，可能与亚马逊合作

[![](img/b9a1d38c17d24c6b469f6212d566551a.png "inside-redbox")](https://web.archive.org/web/20230203002825/https://techcrunch.com/wp-content/uploads/2011/02/inside-redbox.jpg)

Redbox 已经[拿下了](https://web.archive.org/web/20230203002825/http://www.crunchgear.com/2010/09/22/blockbuster-she-is-dying/)连锁 DVD 租赁店，现在 Coinstar 正在通过电影流媒体服务将这个受欢迎的 DVD 租赁品牌扩展到网络。网飞杀手？不太可能，但据报道，即将推出的红盒子媒体服务是与另一家公司。亚马逊是受欢迎的选择，这无疑将有助于吹嘘内容和可信度，而不是独自一人。这对于两家公司来说都是事实，因为有传言称亚马逊也在开发电影流媒体服务。

该公司的首席执行官[表示](https://web.archive.org/web/20230203002825/http://latimesblogs.latimes.com/entertainmentnewsbuzz/)这项服务将是基于订阅的，而不是像其 DVD 租赁业务一样的付费播放。毫无疑问，价格将与网飞竞争，但关键将是可用的内容和支持的平台。

三大电影公司已经要求红盒子和网飞在 DVD 上市后等待 28 天。迪士尼还提高了 Redbox 的 DVD 成本，消费者越来越远离 DVD 租赁服务。尽管与网飞竞争并不容易，但 Redbox 的转型是必要的一步。

Redbox 的下一步应该是像网飞过去两年所做的那样，拥抱尽可能多的硬件平台。简单地推出像 Blockbuster 这样的流媒体盒子是不够的。消费者不想在他们的电视下再放一个盒子。红盒子需要在现有的硬件上运行，比如网飞。这是留在游戏中的关键。
