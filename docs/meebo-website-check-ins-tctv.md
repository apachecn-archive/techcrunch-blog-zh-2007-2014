# Meebo 首席执行官 Seth Sternberg 解释他为什么推动网站签到(TCTV)

现在，人们刚刚开始适应地理定位的[签到](https://web.archive.org/web/20230205003252/https://techcrunch.com/2010/03/19/check-in-fatigue-location-war/)的概念，它开始蔓延到其他领域，如[产品签到](https://web.archive.org/web/20230205003252/https://techcrunch.com/2010/11/15/stickybits-product-rewards/)，电视节目签到，以及网站签到。就在几周前，Meebo [为其 Meebo 酒吧推出了网站签到](https://web.archive.org/web/20230205003252/https://techcrunch.com/2010/11/14/meebo-minibar/)新功能。在此之前，在旧金山的上一次 Disrupt 大会上，两家初创公司( [Badgeville](https://web.archive.org/web/20230205003252/https://techcrunch.com/2010/09/27/badgeville/) 和[oneruefan](https://web.archive.org/web/20230205003252/https://techcrunch.com/2010/09/27/onetruefan-is-the-foursquare-for-websites/))围绕网站签到的概念成立了整个公司。

那么你为什么想要登录一个网站呢？今天，在 SAI 于纽约市举行的点火会议上，我遇到了 Meebo 首席执行官 Seth Sternberg，他在上面的视频中做了解释。Meebo Bar 是一个 IM 和分享扩展，越来越多地出现在许多网站上。Meebo 现在正在测试网站签到的概念，你可以点击一个按钮，在脸书或 Twitter 上分享你所在的任何网站。这与共享特定链接不同，尽管它也支持共享特定链接。它只是告诉你认识的每个人你喜欢某个网站，这是一种隐性的认可。因此，你将能够通过人找到新的网站，有点像你可以用 StumbleUpon。这方面的签到和绊倒很像。

签到最多的人成为“贵宾”，他们的想法是个人出版商可以奖励他们最忠实的访问者认可或额外津贴。你可以在这里注册 Meebo 扩展的 alpha 版本(仅适用于 Firefox 和 Chrome)来查看签到功能。
