# PHP 创始人拉斯马斯·勒德尔夫离开雅虎

[![](img/e45f62404e3c499cb168ab6e9871f5e5.png "rasmus") ](https://web.archive.org/web/20230213021231/http://www.crunchbase.com/person/rasmus-lerdorf) [PHP](https://web.archive.org/web/20230213021231/http://php.net/) 创始人[拉斯马斯·勒德尔夫](https://web.archive.org/web/20230213021231/http://www.crunchbase.com/person/rasmus-lerdorf)已经离开他在雅虎长期担任的职位，根据[他的推特](https://web.archive.org/web/20230213021231/http://twitter.com/rasmus/status/5563093966)账户。勒多夫于 2002 年加入雅虎，此后一直在该公司担任工程师。Lerdorf 最著名的是创建了最初的 PHP 引擎，并且是一个著名的开源开发者、演讲者和作者。Lerdorf 在 1995 年开发了 PHP，之前他在 web 应用程序开发中使用了一系列 C 宏。首字母缩略词的原意是“个人主页”，其语言和环境仍然是当今网络上最流行的。

PHP 被 Zend 进一步开发并商业化，但是 Lerdorf 一直参与开源项目。自从最初开发 PHP 以来，Lerdorf 已经在许多公司工作过，但是他职业生涯的大部分时间是在雅虎度过的，他与该公司有着密切的联系。勒多夫是最近离开雅虎的众多明星工程师和开发人员之一，该公司稳定的知名和高知名度工程师已经减少。

Lerdorf 最近因他的博客文章而出名，比如他关于开发 PHP 应用的哲学的概述:无框架 PHP 框架。
