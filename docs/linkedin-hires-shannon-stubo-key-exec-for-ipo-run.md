# LinkedIn 聘用 Shannon Stubo，负责首次公开募股 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/07/23/linkedin-hires-shannon-stubo-key-exec-for-ipo-run/>

首席执行官[杰夫·韦纳](https://web.archive.org/web/20221006165241/http://www.crunchbase.com/person/jeff-weiner)告诉我们，LinkedIn 已经聘请[香农·斯图博](https://web.archive.org/web/20221006165241/http://www.crunchbase.com/person/shannon-stubo-brayton)为他们的新任副总裁。

Stubo 最近在 OpenTable 带领该公司完成了 IPO 过程。在加入 OpenTable 之前，Stubo 在易贝工作了七年，最终成为企业沟通副总裁。

公司通常会让曾在上市公司工作过的人组成高管团队，而真正经历过 IPO 过程的高管会受到特别重视。Stubo 将理解 SEC 对上市前公司宣传的复杂规定的细微差别。

当然，韦纳说他雇用 Stubo 是出于其他原因，他不会对公司的 IPO 计划发表评论。但很明显，LinkedIn 希望在未来几个财季的某个时候申请 IPO。当然，除非其中一家大公司向他们提出一个他们无法拒绝的收购提议。

这并不是 LinkedIn 团队中第一个表明该公司希望通过 IPO 提升其为公司提供咨询服务的经历的人。今年早些时候，LinkedIn [引进了第一位外部董事会成员](https://web.archive.org/web/20221006165241/https://beta.techcrunch.com/2010/04/08/former-ask-com-ceo-skip-battle-first-outsider-to-join-linkedin-board/)， [Skip Battle](https://web.archive.org/web/20221006165241/http://www.crunchbase.com/person/skip-battle) ，前 Ask.com 首席执行官，网飞、Expedia 和 OpenTable 的董事会成员。虽然韦纳当时告诉我们，Battle 为消费者和企业公司提供咨询的经验使他成为理想的选择，但很明显，这位经验丰富的高管在帮助多家科技公司通过公开募股的过程中发挥的作用可能也是一个因素。

今天早些时候，我和韦纳坐下来讨论了聘用 Stubo 的问题，以及 LinkedIn 的总体情况。该公司现在拥有 7000 万用户，每月约有 4000 万人访问该网站。

他不认为 LinkedIn 是脸书的专业版。我问他，如果他认为脸书或 Monster 是更直接的竞争对手，LinkedIn 的收入会不会更高？LinkedIn 的收入主要来自招聘信息和那些为招聘目的而购买人才洞察信息的公司。

他说，都不是。LinkedIn 是一个独特的东西。正如人们认为星巴克是咖啡，谷歌是搜索，他希望人们认为 LinkedIn 是人才——找工作，找人，让人们在商业上更成功。
