# 什么是游戏:科学主义错觉 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/28/what-games-are-the-scientism-delusion/>

**编者按:** *Tadhg Kelly 是资深游戏设计师，领先游戏设计博客[What Games](https://web.archive.org/web/20221226030004/http://www.whatgamesare.com/)的创建者，Jawfish Games 的创意总监。你可以在 Twitter 上关注他[点击这里](https://web.archive.org/web/20221226030004/https://twitter.com/tiedtiger)。*

制作一个游戏很少像有一个想法、编码并把它运出门外那么简单。看起来似乎应该是这样，每年都有一两个工作室这样做的媒体故事，但实际上这并不常见。通常有很多游戏测试，死胡同，需要返工或重新思考游戏如何工作和迭代。情况一直如此，尽管在这个联系更加紧密的时代，逆市场而动比闭门造车更容易。

烂游戏死于默默无闻。他们消失在 App Store 的 150，000+游戏中，再也没有出现过。他们的失败通常被等同于糟糕的执行和迭代，但这也发生在许多执行良好的游戏中。一款游戏可能会被勤奋地开发、迭代、分析用户行为等等，但它们永远不会有任何进展。相反，有很多你可能认为没什么好处的游戏却卖得很好。

因此，有理由得出结论，游戏的成功不仅仅在于交付。还有另一个因素在起作用，你可以称之为炒作或营销。你可能指的是在 ChartBoost 等渠道做广告、宣传以及努力取悦苹果，这样你就能在 App Store 中脱颖而出。这一切似乎都是同一个执行等式的一部分，但范围更广。迭代+执行+营销=成功。

然而它仍然不起作用。

公式化思维在游戏行业非常普遍。它很大程度上来自于工程思维，这并不奇怪，因为开发游戏和软件总是紧密相连的。在软件中，每个应用程序本质上都是试图通过提供功能来解决问题，随着时间的推移，应用程序会构建成一套用户喜欢的功能。有时产生的应用程序变得过于复杂，以至于新的开发人员为了简单而重新发明它们，然后循环又开始了。这几乎是科学的。

对许多游戏制作人来说，这是相似的。游戏似乎是为了解决娱乐问题，游戏开发者提供了这样做的功能。因为所有的问题都可以归结为测试和验证，就像科学一样，方法应该是清晰的。因此，战略就变成了努力寻找角落、利基、更好的问题，或者在现有问题上超越竞争对手。这就是为什么大多数游戏在野心上趋于保守，也是为什么流派思维高度盛行的原因。

从开发者的角度来看，类型思维在很大程度上将游戏视为一系列的问题类别。在你的 iPad 上出现了一种新形式的吃角子老虎机或一种游戏，其中一个化身沿着一条无尽的路径奔跑——突然间每个人都在谈论“社交吃角子老虎机类型”或“无尽的跑步者类型”。他们谈论他们如何需要“在那个空间”和“他们的游戏”他们通过观察竞争对手的表现来寻求外部的认可，并以此来证明为什么他们应该自己冒这个风险，并加入他们的秘密酱料(这导致了一些有趣的地方，例如尴尬的命名为[与朋友](https://web.archive.org/web/20221226030004/http://toucharcade.com/2013/03/06/running-with-friends-zyngas-take-on-temple-run-soft-launches-in-the-canadian-app-store-today/)一起跑步)。

但是，同样，这种方法通常是不成功的远远多于成功的。Temple Run 和 Jetpack Joyride 可能发现了一些东西，但对于后来成为无休止跑步者的其他几十名开发者来说，就没那么多了。Zynga 和 Supercell 可能凭借一款农业和战略游戏找到了主矿脉，但大多数试图复制这一成功的其他工作室都失败了。他们的成功不仅仅是迭代、执行和营销。

迭代-执行-市场思想家不倾向于进行逻辑跳跃，并得出他们的方法是错误的结论。他们似乎从来不会对自己说“也许问题在于我们太保守、太胆小、太厌恶风险。”相反，他们得出的结论是，他们只是没有足够努力地迭代、执行和营销。他们只是没有很好地组合配料来让秘制酱唱歌。但是肯定有秘制酱对吧？我的意思是，看看超级细胞，他们说。他们把酱汁弄对了。

现实是，这整个思维方式可能看起来像科学，但它不是科学。这是[科学主义](https://web.archive.org/web/20221226030004/http://en.wikipedia.org/wiki/Scientism)，这是经验主义的公理化信仰，看似科学的东西可能是对的。此外，在缺乏经验验证的情况下，假设总有一个公式存在于某处。这种方法是无可指责的。

在游戏世界的某些角落，公式化思维依然存在。体育、赌场和其他各种各样的事物在很长一段时间内显示出有规律的常青模式，但即使在那里，科学主义信徒认为他们看到的模式也被夸大了。在急于证实对成功的偏好的过程中，人们常常忘记的是，娱乐产业并不能解决问题，即使它们是基于软件的。《玩具总动员》没有解决 3D 动画玩具电影的问题，《少年派的奇幻漂流》也没有解决励志书籍的问题。他们做其他事情，这就是为什么那些娱乐行业总是充满大胆的制造商带来的大惊喜，以及大量试图了解扬·马爹利秘方的后续出版商。

我所说的是理解创造行为和创造过程之间的区别。你当然应该迭代。当然，你应该评估和游戏测试，并试图找出什么会使你的游戏有趣。当然，你应该着眼于市场，并努力找到正确的方法来实现你的目标。但是你不应该不折不扣地效仿别人的成功——要明白这是一种科学的方法。这是一种可怕的方法。

就像电影和书籍一样，突破游戏从拥有自己的想法开始。在 App Store 上，有成千上万的应用都在使用相同的推广渠道，但大多数都失败了。如果科学主义真的被证明是成功的方法，那么像 Zynga 这样的公司就不会发现自己继续陷入困境。艺电不会成为[裁员的代名词](https://web.archive.org/web/20221226030004/http://www.polygon.com/2013/4/25/4266122/ea-layoffs-popcap-vancouver-quicklime)，THQ 的债务[也不会如此不堪。这些公司(就 THQ 而言)的主要资产都是营销和推广能力，推销产品和控制注意力的能力，以及运用公式思维的能力。然而他们的失败远远多于成功。](https://web.archive.org/web/20221226030004/http://www.gamasutra.com/view/news/191327/Bankrupt_THQ_owes_tens_of_millions_in_debt.php)

更成功的游戏方法通常是围绕赛斯·戈丁所说的[部落营销](https://web.archive.org/web/20221226030004/http://www.amazon.com/Tribes-We-Need-You-Lead/dp/1591842336)，而不是将神秘的公式应用于自我选择的确认偏差，并假设秘制酱料只是盐的一小撮。不是说“我们有一个适合某个类型的想法，让我们创造这个想法，然后当我们创造这个想法时，让我们营销这个想法，如果我们做得足够好，我们就会成功，”真正成功的游戏制作者通常以“我有一种痴迷”开始。其他人也有同样的执念吗？我们看看那里有没有比赛。”

换句话说，不要依赖平均法则在行业发明的虚构类型中绊倒你，而是像宫本茂那样思考——他把自己对称重的迷恋变成了 Wii Fit。当然你可能会说宫本背后有任天堂，所以他有能力比你或我更有雄心，但这不是真的。他可以比你我在他的野心上花更多的钱，但即使是宫本也是从更小的地方开始的。今天的 Nimblebit 是昨天的任天堂，也就是说正确的创意方法可以扩展。

那怎么办呢？答案就在两个词里:“[对齐](https://web.archive.org/web/20221226030004/http://www.whatgamesare.com/alignment.html)”和“[营销故事](https://web.archive.org/web/20221226030004/http://www.whatgamesare.com/marketing-story.html)”。

第一个意味着与你的人结盟。科学主义迷通常有点像分析师、知识分子或工匠，习惯于冷静地看待市场，并想知道什么样的钥匙能打开它。他就像是法国政治家赖德律·洛兰-罗林所说的“我的人民走了，我必须知道他们要去哪里，这样我才能领导他们。”

成功的游戏设计师通常痴迷于一种游戏，无论是模拟游戏、驾驶游戏还是格斗游戏。结盟的游戏制作者与她的观众联系非常紧密，因为她本质上是他们中的一员，而非常成功的人之所以如此，是因为他们与足够多的观众结盟，这使得他们的痴迷可行。威尔·莱特、托德·霍华德、简·麦克戈尼格尔、大卫·凯奇和许多其他人都是忠实的信徒，他们找到了一大群志同道合的人。

营销故事呢？所有最成功的产品，从汉堡到 50 万美元的汽车，都能引起目标受众的共鸣。无论产品满足了他们的想法，还是代表了未来发展的方向，成功的产品告诉人们他们已经相信的故事。Skyrim 对角色扮演者来说意义重大，不仅仅是它本身。国际足联足球和 gt 赛车也是如此。大多数其他游戏的成功也是如此(主要的例外是彻头彻尾的新奇事物和平台楔子)。

但是，当然，正确调整和营销故事说起来容易，但实际上很难做到，因为这需要大量的自我检查。你不可能既保持一致又冷静超然。除非你真的相信一个营销故事，否则你不能支持它。你只会被弄糊涂，给人这样的印象。这意味着超越科学主义，同时不忽视尽职调查。

作为游戏制作者，我们太习惯于将游戏视为一项业务，而不是一种激情，因此我们在街上的联系感变成了科学主义。我们经常忘记我们自己的困扰和幻想的飞行与生俱来的快乐。我们经常会停止问这样一个问题“如果你做了一个游戏，会怎么样？”取而代之的是“我们需要挖掘什么类型的音乐？”我们太过深入地将玩家视为用户，以日活跃和月活跃群体来衡量，并细分为收入来源和客户群。我们忘记了我们是艺人，而不是工程师，对于那些只想享受乐趣和感受我们创造的世界的幻想的观众来说，流派大多是狗屁。

有时候，科学主义真的很有效，但更多的时候，它成功的真正源泉在于良好的时机。Zynga 可能看起来是一家由游戏科学家解决问题的公司，但事实是，当其他人都在等待验证时，它聪明地冒险，拥有先发优势。对于格力、BigPoint 和其他许多走上相同道路的公司来说，情况也差不多。他们方法的神秘主义很好理解，通常始于某人冒巨大的风险*在*他们开始将科学主义应用于他们的思考之前。

我们知识分子、分析师和工匠喜欢谈论秘方之类的东西，因为它们让我们感到放心。我们所做的只是相信我们自己的营销故事，这个故事说游戏在未来有一天会被解决，因为我们对不确定性感到不安。我们认为是辜莞允。不可能是单纯的运气和时机那么傻。那个刻痕男？也许他只是运气好。但是那个平卡斯？他有方法，秘方和计划。

没有。他只是有勇气在任何人面前豪赌一把。其余的大部分都是虚构的。
