# 贝佐斯、亚马逊和拒绝表现出你的年龄

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/11/01/bezos-amazon-and-refusing-to-act-your-age/>

1998 年初，亚马逊创始人杰夫·贝索斯和首席财务官乔伊·科维合写了一封致股东的信，讨论了前一年的成就。这封信已经成为公司内部的圣旨，每年都会在《T2》的年度报告中重新发布。

当我通读布拉德·斯通的优秀作品 *[The Everything Store:杰夫·贝索斯和亚马逊时代](https://web.archive.org/web/20221225193655/http://www.amazon.com/Everything-Store-Bezos-Amazon-ebook/dp/B00BWQW73E/ref=sr_1_1?ie=UTF8&qid=1383316422&sr=8-1&keywords=brad+stone)* 时，这封信及其内容给我留下了难以置信的先见之明，或者也许仅仅是有力的印象。毫不奇怪，这封信一直被视为该公司及其员工的试金石。该公司开发和发布产品的方式，以及日常业务所采用的方法，清楚地体现了 it 中阐述的原则。

亚马逊业务最常被讨论的一个方面是它对利润率的态度。众所周知，该公司每年都赚很多钱，并设法花掉其中的大部分，最终其季度资产负债表上显示利润微薄或没有利润。对于一些痴迷于数字在列中的显示方式的人来说，这是一件令人沮丧的事情。一家赚了这么多钱，并继续成为股市宠儿的公司，怎么会最终利润如此之少？

这封信给出了答案，让亚马逊和贝佐斯的观点变得异常清晰。以下是相关段落:

> 我们相信衡量我们成功的基本标准是我们长期创造的股东价值。这一价值将是我们扩展和巩固当前市场领导地位的直接结果。我们的市场领导能力越强，我们的经济模式就越强大。市场领先地位可以直接转化为更高的收入、更高的盈利能力、更高的资本流动速度，以及相应更高的投资回报。
> 
> 我们的决定一贯反映了这一重点。我们首先根据最能体现我们市场领导地位的指标来衡量自己:客户和收入增长、客户持续重复购买我们产品的程度，以及我们品牌的实力。我们已经并将继续大力投资，以扩大和利用我们的客户群、品牌和基础设施，从而建立持久的特许经营。

这封信接着用九点布局描述了亚马逊的决策方法。你可以[在这里](https://web.archive.org/web/20221225193655/http://benhorowitz.files.wordpress.com/2010/05/amzn_shareholder-letter-20072.pdf)查看这封信来阅读要点，但是如果你看看亚马逊最近的举动，几乎所有的举动都符合那里提出的原则。其中一些，如“当被迫在优化我们的 GAAP 会计外观和最大化未来现金流的现值之间做出选择时，我们会选择现金流，”直接表明了亚马逊保持精益公司的愿望。

斯通的书清晰地描绘了贝佐斯的形象，他是一个动不动就陷入困境的人。书中的一个重要例子是一则轶事，当时的营销高管马克·布雷尔(Mark Breier)向贝佐斯提供了一项调查结果，该调查显示大多数客户没有使用亚马逊，而且可能永远不会使用，因为他们不看书。

当时，亚马逊卖的都是书。这不是什么好消息。

贝佐斯的回应？“我给他带来了关于我们生意的坏消息，出于某种原因，他变得很兴奋，”布赖尔说。

贝佐斯没有把这个消息视为亚马逊继续成功的主要障碍，尤其是在 IPO 之后，而是把它视为开始探索实体店服务不足的其他产品类别的最佳时机。经过研究，其中一些由未来的亚马逊云执行官安迪·贾西和未来的 Hulu 首席执行官杰森·基拉尔执行，亚马逊扩展到了 DVD 和音乐领域。其他的，你知道俗话说的好。

关于贝佐斯处理亚马逊的轶事非常简单，我相信当人们思考该公司为什么做这样或那样的事情时，会忽略很多。

简而言之，杰夫·贝索斯认为亚马逊是一家年轻的公司。

作为互联网繁荣和萧条的幸存者，以及一家近 20 年的公司，亚马逊经常被科技作家和分析师视为互联网业务的一个令人尊敬的支柱。如今，一个日渐衰老的超级大国与从苹果到谷歌的所有人都发生了冲突。但这不是贝佐斯的看法，也不是他希望他的员工看到的。

在运营亚马逊的人看来，它仍然专注于增长，仍然想成为什么就成为什么。当你从这个角度看它时，决定将几乎所有收入再投资于扩张和新产品，如 Kindle Fire——即使这些决定对资产负债表产生了负面影响——就更有意义了。

在这个[估值极高](https://web.archive.org/web/20221225193655/https://techcrunch.com/2013/10/23/confirmed-pinterest-raises-225-million/)的时代，对于那些[看似没有什么内在价值](https://web.archive.org/web/20221225193655/http://allthingsd.com/20131029/snapchats-giant-funding-from-tencent-at-valuation-of-up-to-4-billion-could-click-this-week/)和没有[收入可言](https://web.archive.org/web/20221225193655/https://techcrunch.com/2013/10/24/twitter-to-offer-70m-shares-in-ipo-priced-at-17-20-per-share/)的公司，我们可以轻松地谈论数十亿美元的金额。但是当亚马逊被关注时，我们不愿意用同样的框架来讨论它，仅仅是因为它的“成熟”但作为一家仍处于起步阶段的公司——以这种方式重新定义——它实际上变得更容易理解。亚马逊时代？比你想象的要年轻。

斯通书中呈现的亚马逊仍然对自己的成功感到不安，描绘的贝佐斯仍然刚刚起步。这是一本引人入胜的读物。我向该公司的学生或那些只对商业故事感兴趣的人推荐这本书。
