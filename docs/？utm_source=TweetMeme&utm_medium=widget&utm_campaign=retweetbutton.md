# 现在我们是否都同意“高质量网络内容”实验已经失败了？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/27/who-wouldve-thought-it-figures/?utm_source=TweetMeme&utm_medium=widget&utm_campaign=retweetbutton>

很难想象还有什么比 Slate 决定解雇其受人尊敬的媒体评论家 T2 杰克·谢弗更完美的了。从好的方面来说并不完美——我认为自己是谢弗的众多粉丝之一——但从阿兰妮斯·莫利塞特[不理解](https://web.archive.org/web/20230203151921/http://www.youtube.com/watch?v=nT1TVSTkAXg)讽刺的含义这一方面来说是完美的，或者从十楼窗户掉下来的安全检查员这一方面来说是完美的。

“我很快就知道了…”

我的意思是，有什么比电子杂志解雇其媒体评论家更能说明在线媒体的困境，因为网络内容的经济学不支持他这样的地位和专业作家？至少谢弗可以感到满意的是，他的离开本身就是一个绝对完美的媒体批评:杰克谢弗既是媒介又是信息。

Slate 承认，即使拥有 60 名员工和华盛顿邮报公司的财务“实力”，它也无法从在线内容中赚钱，这也是完美的。这是一个绝好的机会，那就是一劳永逸地承认免费在线内容的伟大实验失败了。

在 Forbes.com，杰夫·贝尔科维奇[就这样解决了问题](https://web.archive.org/web/20230203151921/http://www.forbes.com/sites/jeffbercovici/2011/08/25/slate-lays-off-staff-does-its-model-still-make-sense/) …

> “对于送到你家门口的实物产品来说，一般兴趣是一个很好的概念，把所有不同的部分捆绑在一起是有意义的。在网络上这不是一个好的概念。网络讨厌人为的捆绑。如果你打算在网上做一个大众感兴趣的新闻产品，你必须准备好以低廉的价格去做，就像马特·德拉吉和阿里安娜·赫芬顿所做的那样(或者至少在后一种情况下曾经这样做过)。相反，如果你想推出一款制作昂贵、经过专业编辑的产品，最好坚持小众化，最好是拥有广告商想要触及的人群的产品，比如科技或商业产品。

…他是对的。在某种程度上。事实上，许多利基出版物也感受到了压力。就在不久前，贝尔科维奇的雇主《福布斯》放弃了其专业撰写和编辑金融出版物的地位，决定将自己打造成金融领域的《赫芬顿邮报》;拥抱廉价的嘉宾海报，不管它们会引起什么样的冲突。与此同时，如果我说出我们在技术博客圈的竞争对手的名字，那就太小家子气了，他们已经接受了狗屁幻灯片和十大榜单，而不是他们更昂贵的表亲:*实际他妈的报道*。(到目前为止，TechCrunch 被美国在线收购还没有导致我们的编辑部门不得不在上帝和财富之间做出选择，但愤世嫉俗的人可能会说，我们也会受到考验，这只是时间问题。)

直言不讳的事实是，在线广告是一个数字游戏。而且，即使在利基网站上，达到收支平衡所需的畅销页面浏览量也是巨大的。为广告制作的内容太多了，无法维持一个可行的长期商业模式。《纽约时报》在网上赚不到钱，《卫报》赚不到钱，《石板》赚不到钱，《沙龙》几乎赚不到钱。正如 Bercovici 指出的，即使 Slate 试图推出针对商业读者和女性的垂直服务，也是相对的[失败](https://web.archive.org/web/20230203151921/http://allthingsd.com/20100730/the-big-money-isnt-enough-slate-shuts-down-business-site-after-two-years/)。

可能有两家大众感兴趣的出版物可以合理地宣称已经破解了免费内容代码:[每日邮报](https://web.archive.org/web/20230203151921/http://www.dailymail.co.uk/ushome/index.html)和[赫芬顿邮报](https://web.archive.org/web/20230203151921/https://techcrunch.com/2011/02/06/i-for-one-welcome-our-new-huffington-overlord/)。但事实上，这些出版物能够支付日益壮大的真正的成年编辑大军的唯一方式是出售数百万页的动物故事和名人绒毛，这些都是由薪酬过低的黑客制作的。(有一天，我想制作一个《赫芬顿邮报》的幻灯片，展示《每日邮报》的最佳名人幻灯片——它会很棒的。)

然而。人们很容易悲叹互联网正在扼杀新闻业，但反乌托邦式的未来只有在我们假设互联网是社论内容唯一可能存在的地方时才会存在。事实上，在未来五年左右，我们可能会看到数字内容的分叉。

一方面，那些选择留在免费开放网络上的内容制作者将被迫做出越来越多道德上可疑的决定，以保持盈利。职业作家和内容与商业的政教分离将会消失；将会出现更多的 Groupon 式的“读者服务”，每个关键词后面的附属链接，以及一个越来越蠢的白痴链接诱饵。 *[用 Lady Gaga 索尼色情片赚外快的十种方法——小猫！](https://web.archive.org/web/20230203151921/http://coedmagazine.com/2011/08/25/lady-gaga-nude-naked-bondage-vogue-japan-photos/)*

但是另一边呢？《经济学人》北美发行量刚刚达到其[有史以来的最高水平](https://web.archive.org/web/20230203151921/http://www.foliomag.com/2011/economist-hits-highest-north-american-circulation-its-history)这一事实告诉我们，优质内容的观众不会离开。这也表明，那些喜欢我们的内容不受贿赂、欣赏精心制作的标题之美的人正在远离网络。越来越多最好的写作和报道出现在书籍和 Kindle Singles 上，读者乐于直接为高质量的信息和娱乐付费。随着网络内容变得越来越愚蠢，越来越不道德，网络之外的高质量内容的市场将继续增长。

当然，如果建议出版商希望效仿《经济学人》的做法，匆忙重新印刷，那将是愚蠢的。但是他们也不应该浪费钱在网上发布他们的内容。正如任何利润丰厚的应用程序开发者会告诉你的，网络是一个伟大的营销工具，但真正的金钱和注意力是在专用的便携式设备上找到的。一个聪明的出版商今天打算推出一本新杂志——专注于商业、技术或其他任何东西——将是明智的，专门为电子阅读器开发它，而不是浪费更多的钱追逐网络的愚蠢眼球。哦，他们应该雇用杰克谢弗。他很聪明。
