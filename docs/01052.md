# 即将推出的 MySpace“新闻源”的截图和细节

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/11/26/screenshots-and-details-on-upcoming-myspace-news-feeds/amp/>

今天早些时候，路透社[的一篇文章](https://web.archive.org/web/20230203091739/http://www.reuters.com/article/MediaandCommunications07/idUSN2641609120071126)给出了 MySpace 即将推出的“新订阅”产品的一些细节(脸书称之为他们一年前推出的[类似产品](https://web.archive.org/web/20230203091739/https://techcrunch.com/2006/09/05/new-facebook-redesign-more-than-just-aesthetics/)。我们与 MySpace 进行了交谈，对该产品有了更深入的了解，并获得了它的截图。

MySpace 称，这款名为“好友更新”的新产品将于本周四开始向用户推出。新西兰和爱尔兰的用户已经可以使用它了，那里的朋友更新正在进行测试。

新产品将为用户提供源源不断的来自朋友的活动信息。个人资料更新、博客文章和照片/视频上传都会被记录下来。请参见右边的屏幕截图(点击查看大图)以获取视觉效果。

MySpace 产品副总裁史蒂夫·皮尔曼告诉我，周四发布的产品只是第一次迭代；他们还计划在未来 18 个月内发布更多功能。

 **与脸书新闻供稿** 的主要区别

脸书新闻订阅和 MySpace 好友更新之间有一些关键的区别。该产品的一个关键焦点似乎是不惹恼用户。其他差异允许对新闻发布进行更精细的控制。

MySpace 只会在用户登录并查看通知用户变更的间隙页面后开始收集用户信息，直到用户看到并点击该间隙页面，才会收集好友更新的信息。用户还可以链接到一个设置区域，在这里可以更改默认选项，或者完全退出。这将有望帮助 MySpace 避免脸书在推出新闻订阅后不久就面临的用户反弹。

用户还可以更精确地控制谁可以查看数据。可以选择特定的朋友不接收更新。Pearman 举了一个例子，不一定要你女朋友知道你什么时候把状态改成“单身”(虽然快速看一下简介反正会让她知道)。用户还可以从一个或多个信息类别中选择要共享的信息。

这个过程归结为一个要求双重肯定——你必须同意与特定的朋友分享，他们也必须同意订阅。没有双方同意，信息不会出现。

Pearman 说，预计在不久的将来，第三方应用程序会访问新闻提要。他表示:事实上，这是他们开发该产品的关键原因之一。
