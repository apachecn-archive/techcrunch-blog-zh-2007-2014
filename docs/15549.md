# 电动汽车科技公司 Better Place 遭遇破产，绿色科技的洗牌仍在继续

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/05/26/better-place-bankruptcy-liquidating-greentech/amp/>

[Better Place](https://web.archive.org/web/20230324111803/http://www.crunchbase.com/company/better-place) ，这家总部位于特拉维夫的电动汽车电池技术公司自 2007 年成立以来已经筹集了超过[8 亿美元的风险投资](https://web.archive.org/web/20230324111803/http://www.crunchbase.com/company/better-place)，[今天](https://web.archive.org/web/20230324111803/http://www.betterplace.com/the-company/press-room)证实，在试图筹集更多资金失败后，它已经向法院提交了解散和清算公司的动议。《财富》杂志的丹·普里马克上周首次报道了即将到来的破产。

Better Place 的官方声明是在其大股东以色列企业集团[以色列公司](https://web.archive.org/web/20230324111803/http://www.israelcorp.com/)、[宣布其决定](https://web.archive.org/web/20230324111803/http://phx.corporate-ir.net/External.File?item=UGFyZW50SUQ9MTg4MTU1fENoaWxkSUQ9LTF8VHlwZT0z&t=1)“不参与 Better Place 向其投资者提供的投资轮次”，并补充说“由于没有投资者愿意大量参与 Better Place 的额外投资轮次”，清算申请将很快提交

对于该公司来说，这是一个非常令人失望的结局，该公司一度被视为全球绿色技术运动的重要参与者，也是以色列当地初创企业的宠儿。但最终，Better Place 的愿景[并没有与现实一致。尤其是在过去的一年](https://web.archive.org/web/20230324111803/http://www.haaretz.com/business/business-opinion/the-fall-of-israel-s-better-place-when-vision-isn-t-enough.premium-1.526072)，Better Place 的挣扎开始变得清晰[，尤其是在 2012 年 10 月，当其魅力四射的创始人](https://web.archive.org/web/20230324111803/http://www.haaretz.com/business/is-better-place-running-out-of-juice-before-it-hits-the-road-1.461584) [Shai Agassi](https://web.archive.org/web/20230324111803/http://www.crunchbase.com/person/shai-agassi) [被解除首席执行官职务](https://web.archive.org/web/20230324111803/https://techcrunch.com/2012/10/17/better-place-board-quietly-removes-ceo-shai-agassi-places-evan-thornley-at-helm/)。当时，TechCrunch 的约翰·比格斯(John Biggs)写了一篇关于 Better Place 早期炒作和随后斗争的简要概述:

> “该公司最初是作为电动汽车的独特解决方案而出名的，后来成为以色列科技界的宠儿。该公司计划在不同国家的车站提供可切换电池。当司机没电时，他们只需换下电池，继续开车。然而，技术和基础设施已经花了数年时间来规划，尽管阿加西被誉为以色列经济奇迹的鼻祖，但该公司在实际实施和交付方面几乎没有做什么。

Better Place 只是绿色科技初创公司持续洗牌中的最新受害者，这些初创公司在中期至晚期出现了很大的热情——通常是巨额融资——但现在遇到了困难时期或完全关闭。风险投资公司如凯鹏华盈(Kleiner Perkins)几年前曾在绿色科技浪潮中向 T2 投入大量资源(T3)，而现在 T4 正在改变方向(T5)。

但好消息是，除了许多输家，也有赢家——菲斯克汽车公司(Fisker Automotive)的[遭遇了](https://web.archive.org/web/20230324111803/http://www.crunchbase.com/company/fisker)[的](https://web.archive.org/web/20230324111803/http://gigaom.com/2013/04/07/fisker-hit-with-lawsuit-over-layoffs/)，特斯拉汽车公司(Tesla Motors)的[遭遇了](https://web.archive.org/web/20230324111803/http://www.crunchbase.com/company/tesla-motors)的[飙升](https://web.archive.org/web/20230324111803/http://www.prcarbon.com/press/tesla-stock-went-up-19-20-in-premarket-tesla-motors-inc-nasdaqtsla-reports-lucrative-quarterly-results-1005469.html)。能效和电池技术一如既往地重要，但似乎 Better Place 最终并不具备获胜的条件。
