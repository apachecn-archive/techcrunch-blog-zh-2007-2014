# InternMatch 筹集了 50 万美元将学生与实习联系起来

一家由 500 家初创公司孵化的公司已经从网飞创始人马克·伦道夫、罗杰·迪基、天使基金社交杠杆、500 家初创公司和其他投资者那里筹集了 50 万美元的资金。这给[带来了 90 万美元的资金。](https://web.archive.org/web/20230203223048/https://techcrunch.com/2011/02/02/internmatch-raises-400k-to-help-students-find-the-perfect-internship/)

InternMatch 本质上是帮助学生联系实习。该平台试图让[学生的实习搜索过程](https://web.archive.org/web/20230203223048/https://techcrunch.com/2011/04/08/a-new-design-for-internmatch-humanizes-internship-search/)人性化，并允许中小型企业招募符合其商业文化的学生。这家初创公司希望取代面向希望实习的大学生的招聘会。

在 InternMatch 的平台上，实习求职者和雇主都可以搜索匹配，接收基于技能和位置的匹配建议，并访问管理申请流程的工具(即简历创建、实习准备等提示)。

对于公司来说，InternMatch 提供了匹配保证——如果雇主在 60 天内没有找到实习生，他们可以获得全额退款。这家初创公司希望通过提供一个非常简单的用户界面，让学生可以轻松地搜索和申请职位，从而在竞争中脱颖而出。

InternMatch 已经在全国范围内扩展，并在今天推出了一个新产品——社区页面。基本上，社区页面是一个集中的地方，帮助雇主通过视频、照片、问答、社交媒体反馈等内容展示他们的文化。

社区页面为雇主提供了选择复杂的候选人筛选或与现有申请人跟踪系统简单集成的选项。InternMatch 正在与包括红杉资本(Sequoia Capital)、皮克斯(Pixar)、美国在线(AOL)、维基媒体(Wikimedia)、Yelp、美国在线(AOL)等许多知名公司合作推出社区页面。
