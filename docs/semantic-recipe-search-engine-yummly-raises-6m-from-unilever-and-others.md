# 语义食谱搜索引擎 Yummly 从联合利华和其他公司筹集了 600 万美元

语义菜谱搜索引擎 Yummly 已经筹集了 600 万美元的首轮融资，由 Physic Ventures 和食品巨头联合利华的风险投资部门联合利华公司(Unilever Corporate Ventures)牵头，投资者包括回归投资者哈里森金属资本(Harrison Metal Capital)和哈佛大学公共出版社(Harvard Commons Press)等。

Yummly 从网上收集了成千上万的食谱，允许你根据食物类型、课程和成分过滤结果，并根据饮食、过敏、营养、价格、烹饪、时间、味道和来源分解食谱。

你也可以编辑和保存任何配方，根据你的喜好进行配料替换和调整；Yummly 将重新计算配方以反映新的配料量。因此，如果你想将一份食谱缩减为一份，Yummly 会重新计算你需要的一份更小版本的菜肴的配料。

Yummly 计算每种食谱的营养价值，显示卡路里、脂肪、蛋白质和碳水化合物的分类。用户还可以从其他网站导入和添加他们喜欢的食谱，并保存在他们的美味食谱框中。

该公司表示，现在每月有 400 万独立访客，还推出了一款脸书应用。Yummly 拥有约 140 万脸书用户，自去年推出时间轴应用程序以来，脸书的推荐流量增长了 75%。新的资金将用于招聘和产品开发。
