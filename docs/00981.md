# Yahoo FireEagle -地理信息平台服务 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/11/04/yahoo-fireeagle-a-platform-service-for-geo-information/amp/>

今晚，雅虎不仅仅宣布了 Kickstart 。Salim Ismail 的 [Brickhouse](https://web.archive.org/web/20230126025524/http://techcrunch.com/2007/03/14/salim-ismail-to-head-yahoo-brickhouse/) 今晚宣布了一个非常有用的新平台服务，暂定名为 [FireEagle](https://web.archive.org/web/20230126025524/http://fireeagle.research.yahoo.com/) ，目前正在封闭的 alpha 测试中。该团队正在制定发布名称和最终发布日期，预计将于本月晚些时候发布。

完全基于 Ruby on Rails 构建的 FireEagle 最初的灵感来自于雅虎的研究产品。它是一个控制人们位置信息的平台。告诉它(直接或通过基于 fire eagle API 的第三方应用程序)你在哪里(给它具体的纬度/经度，或城市名称，或邮政编码等。)它会记下你的位置。或者，拥有 GPS 手机(或其他 GPS 设备)的用户可以将其设置为定期更新 FireEagle 的地理信息。

当然，用户可以随时关闭追踪，也可以删除任何或所有存储的关于自己的地理数据。雅虎表示，它将立即从他们的服务器上删除。

然后，其他应用程序可以在您的许可下获取该数据，并将其直接构建到它们的服务中。

这对于像 Flickr 这样的服务来说是完美的，Flickr 仍然很难让用户在照片上添加纬度/经度信息。FireEagle 还可以通过与 Plazes 等知名的位置博客服务合作而受益，既可以提供也可以接收用户的地理信息。

该服务将为添加和获取信息提供开放的 API。Ismail 说，在过去的几个月里，他们已经与大约 50 个第三方开发者秘密合作，其中许多人将在正式发布之日准备好使用 FireEagle 的应用程序。

上周在 Brickhouse 进行产品演示时，我用手机拍了几张照片。分辨率不是很好(事实上很糟糕)，但是你可以感受一下这个平台的样子。下面，我还附上了一张脸书应用程序的照片。

我想我可以有把握地说，有很多开发人员会对 FireEagle 感到非常兴奋。
