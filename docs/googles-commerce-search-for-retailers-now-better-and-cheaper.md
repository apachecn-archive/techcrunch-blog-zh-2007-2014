# 谷歌对零售商的商务搜索现在更好更便宜 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/06/17/googles-commerce-search-for-retailers-now-better-and-cheaper/>

对于任何电子零售商来说，产品搜索都是商务平台的基础。很多时候，消费者会直接进入搜索框来找到他们想要的对象。在零售商网站上轻松找到产品的能力会影响消费者与其网站的互动和体验，并产生更高的转化率。谷歌[去年开始为零售商的](https://web.archive.org/web/20221006175937/https://beta.techcrunch.com/2009/11/04/search-til-you-drop-with-googles-new-commerce-search/)搜索平台提供支持，推出了商业搜索，这是一个托管的企业搜索产品，为在线零售店和电子商务网站提供支持。今天，谷歌[推出](https://web.archive.org/web/20221006175937/http://googleblog.blogspot.com/2010/06/summer-is-here-and-so-is-google.html)下一代商务搜索，为零售商提供更强大、更便宜的产品版本。

Google Commerce 的第一个版本提供了许多针对零售和产品搜索而优化的功能，比如参数搜索、结果排序、拼写检查、词干提取和同义词建议。最新版本基于这些功能，为用户和商家提供了更深入的搜索体验。正如商务产品经理 Nitin Mangtani 告诉我的，2.0 版本专注于增强导航和用户体验。

**商品销售仪表板**:新的商品销售仪表板为商家在其搜索门户中提供了更多选择。零售商现在可以对产品进行排名，因此某些产品在搜索结果中的排名会高于其他产品。此外，零售商可以在给定的时间内，在搜索门户中突出促销和销售。零售商可以在搜索中添加定制的过滤器或“面”,谷歌的技术将相应地呈现适当的结果。所有这些功能都可以通过规则向导来实现，它不需要定制代码，因此营销人员或跟单人员可以轻松地实现更改。

搜索改进:谷歌正在增加实时查询自动完成功能，这是任何零售网站的必要元素。谷歌将根据索引的产品为每个网站定制自动建议。谷歌还提高了商业搜索中搜索结果的速度，并声称搜索结果会在不到一秒钟内返回给购物者。

价格下降:谷歌将商业搜索的价格降至每年 2.5 万美元的定价模式。以前，商业搜索的起价是每年 5 万美元。定价基于查询和索引项目的总量。

谷歌商务搜索还集成了谷歌产品，如谷歌分析和谷歌产品搜索。使用 Commerce，零售商可以通过 Google Analytics 测量点击量、转换率、交易数量、平均订单价值和其他数据。电子商务供应商可以提供单一的产品和分类项目，这将推动商业和谷歌产品搜索上的产品索引。

使用商业搜索的零售商包括 [Birkenstock](https://web.archive.org/web/20221006175937/http://www.birkenstockusa.com/products/women/sandals) 、[智能家具](https://web.archive.org/web/20221006175937/http://www.smartfurniture.com/)和 [Coveroo](https://web.archive.org/web/20221006175937/http://www.coveroo.com/) 。谷歌拒绝透露使用商务搜索产品的零售商的确切数量，但表示谷歌目前为 30，000 个网站提供企业搜索。谷歌还提供了一个通用的托管搜索产品,供那些想要在网站上添加定制的谷歌搜索功能的组织使用。

我很好奇谷歌降价是否是因为缺乏采用搜索产品的零售商。随着零售业稳步走出经济衰退造成的困境，我仍然认为 2.5 万美元对于较小的零售商来说是一笔不小的支出。但很高兴看到谷歌正在降低价格，以允许更多的零售商使用该产品。谷歌正在推广其产品，将其作为一种易于部署、用户友好的方式，将强大的搜索整合到零售网站中。谷歌面临着来自 Omniture、IBM、Endeca 和其他公司的竞争。
