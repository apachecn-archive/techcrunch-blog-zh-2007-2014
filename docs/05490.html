互联网电视和有线电视的死亡，真的

原文：https://web.archive.org/web/https://techcrunch.com/2010/10/24/internet-tv-and-the-death-of-cable-tv-really/amp/

是的，你以前听过这个。有线电视的消亡。然而，这并没有发生。但现在，视频领域发生了如此多的中断，有线电视真的正在走向悬崖。别指望有线电视行业会就这么放弃。

下周，当美国最大的有线电视运营商(2300万有线电视用户)康卡斯特报告其第三季度收入和用户数量时，我们将获得一些新的见解。与去年相比，康卡斯特在Q2的有线电视客户下降了近3%。在Q2，整个行业有创纪录的711，000用户放弃了有线电视，八分之六的运营商遭受了有史以来最严重的季度用户流失。

就在这个月，发生了很多事情:

  • 不到三周前，谷歌发布了其谷歌电视平台。你不能忽视谷歌。嘿，他们刚刚造了一辆自动驾驶的汽车。但是周四，在一场可能会变得更加频繁的新旧媒体之战中，美国广播公司、哥伦比亚广播公司和全国广播公司从谷歌电视屏蔽了他们的节目。MTV、Fox和HBO仍然可用，但这可能会改变。尽管如此，TechCrunch的一篇帖子宣称“我已经看到了未来，它从我的沙发上的谷歌电视开始。”
  • 史蒂夫·乔布斯本周向吹嘘说苹果已经卖出了25万台新的苹果电视。第一台苹果电视于2007年发货。它有自己的粉丝，但没有像iPod或iPhone那样迅速发展。第二代苹果电视上个月刚刚推出。MG西格勒非常喜欢这个设备，但是承认它还不是客厅里的杀手级设备。他说，要实现这一目标，需要电视网络订阅套餐。
  • “立即观看”在网飞蓬勃发展。本周，一项令人震惊的统计数据出炉。美国高峰时段20%的互联网流量来自网飞。
    关于网飞计划的更多信息，请看莎拉·莱西对首席执行官雷德·哈斯汀斯的采访。
  • Hulu Plus将于秋季登陆Roku 盒子。
    对于某些来说， Roku 盒可能是消除线缆的第一步。
  • Boxee 宣布新的Boxee盒子将于下个月发货，无论你是从亚马逊预订还是想在商店购买。
  • 疾风 报道苹果的iOS应用程序对最近电视收视率的下降趋势负有责任。实际原因可能更宽泛一点。
  • TechCrunch上周五的一篇帖子暗示电视的未来是HTML5。

有了这些选择，J.D. Power and Associates本月发布的一项调查显示，消费者对每月付费电视账单不太满意，有线电视用户比电信公司或卫星电视用户更容易感到被敲诈。

有线电视为什么会存在？

当有线电视在20世纪40年代刚刚起步时，它是一项颠覆性的技术。社区天线电视解决了一个问题。山区、城市和远离无线广播发射机的地方的一些家庭收不到好的信号。通过安装大型社区天线，并通过同轴电缆将信号转发到这些家庭，接收问题得到了解决。一些人声称第一个系统建立在宾夕法尼亚州的马哈诺伊市，用户每次付费100美元外加每月2美元。

随着行业的发展，有线电视运营商开始接收远距离信号，为消费者提供新的节目选择。甚至可以接收无线信号的家庭也注册了有线电视。有线电视开始制作自己的独家原创频道。1975年，HBO成为第一个通过卫星到有线系统向全国传送信号的有线网络，然后再传送给消费者。Ted Turner(“在有线电视变得酷之前我就是有线电视”)在1976年推出了第一个由卫星传播的基本有线电视网络WTBS。有线电视还提供当地社区节目，这是广播网所没有的。记得在家里看过一个我初中制作的新闻节目。当时，这让我大吃一惊。
(披露:我职业生涯的前14年是在一个名字以‘Cable’开头的地方工作，这是我欠有线电视行业的但后来我转行做了互联网。)

有什么变化

但是，所有这些曾经“仅限于有线电视”的好处都变了。DSL、电缆(是的，电缆——稍后会有更多)调制解调器，也许未来的光纤到户意味着视频有许多其他的途径进入家庭。

曾经只有有线电视才有的内容现在可以在其他地方看到。起初，有线电视节目在有线电视上首次播出后，开始出现在DVD上。但是现在，有了更多的选择。Cancelcable.com列举了一些非有线方式找到你喜欢的节目。并且本地和用户生成的内容现在具有全球影响力，并且可以在具有共同的广泛或狭窄兴趣的观众社区中共享。

然而，在互联网上获得更多电视节目有一个关键的绊脚石。钱。而且是上面提到的网络/谷歌电视之争的原因。一句格言文章明智地指出

“电视网屏蔽谷歌电视并不是因为它是谷歌。他们正在阻止谷歌电视，因为它正在电视上播放网络电视节目，网络电视节目经济学，这将对他们的业务产生难以置信的破坏性影响。”

在线观众每分钟的收入比广播少得多。随着更多的广告出现在网络分销上，它们变得更有针对性和互动性，这种情况可能会改变。此外，节目或频道订阅可以解决这个问题。

有线电视有一个优点:相对容易使用。打个电话，有人到你家把它挂上，然后你用遥控器控制它。(好吧，你可能等了几个小时才有人接你的电话，然后等了一整天安装人员的到来。)到目前为止，你必须是一个精通技术的人才能取消有线电视，连接自己的系统。谷歌、苹果或网飞的人不会去你家帮你安装新设备。正如丹尼·沙利文发现的那样，连接你的谷歌电视还没有为黄金时间或大众做好准备。但是，期待它的改善。苹果电视等其他产品让这变得更容易。电缆替代品正在从只有早期采用者使用的技术转向更广泛的市场能够发现的技术。

有线电视之前曾面临竞争对手
有线电视行业对竞争并不陌生。有线电视在DVD租赁店的竞争中幸存下来。(百视达申请破产。苹果刚刚取消了其新笔记本电脑中的光驱。)卫星电视提供商DirectTV ( 美国有1870万客户)和Dish Networks(1430万客户)拥有一些忠实客户，但还未能阻止有线电视。但是，互联网将是一个更严峻的挑战。

电缆成本
电缆行业面临的最大问题之一是其高昂的成本。一项研究报告称，八分之一的消费者表示他们将在2010年取消或缩减有线电视、卫星电视或其他付费电视服务，增长的成本是最大的原因。有线电视行业试图解释有线电视的成本正在下降。看看这张每小时收视价格的图表，在国家有线电视电信协会网站上。这不会让任何有线电视用户相信他们每月的账单。

有线电视行业抵制“按菜单点菜”定价，允许客户挑选和选择他们观看和付费的频道。有线电视网络的部分收入来自用户向有线电视运营商支付的订阅费，然后再支付给网络。因此，在这样的定价计划下，较小的网络可能会倒闭。还有一场关于“照单点菜”定价是会节省还是会让消费者花钱的辩论。互联网电视已经提供了挑选和支付的定价选项。新媒体可以比旧媒体更好地处理这一点。就我的钱而言，我认为在“照单点菜”系统下我会省下一大笔钱。

有线电视反击
虽然有线电视行业正在失去基本的有线电视用户，但他们的财务状况仍然不错。今年以来，康卡斯特的股票上涨了15%，而纳斯达克上涨了9%。康卡斯特的互联网业务(电缆调制解调器用户)正在增长。康卡斯特与时代华纳有线电视合作，也在推动 电视无处不在，将电视频道放在网上，并置于只有有线电视订户才能观看的付费墙之后。当然，康卡斯特正计划通过一项复杂的交易将广播公司和有线电视网络巨头NBC环球合并。但新的合资企业不会包括康卡斯特的有线电视系统。这项交易仍然需要监管机构的批准，联邦通信委员会已经询问这些公司，最大的宽带提供商和主要内容公司的合并将如何影响在线视频的分发。

还有人猜测，有线电视盒可能会内置“谷歌电视”，但需要额外付费。但是，当其他选择会更便宜时，消费者真的会选择支付更高的费用并保留他们的有线电视盒吗？

眼前的未来
电报之死仍然是一个热门话题。下个月，流媒体西部公司的一个小组将讨论“切断电视线路:在线视频真的会导致有线电视的消亡吗？”

实际上我还有有线电视。我差一点就要放弃它了，但是仍然有一些我的妻子喜欢的其他地方没有的节目。但是，很难想象这些节目在不久的将来不会在其他地方出现。我知道很多人用有线电视直播体育比赛，但有线电视作为一项体育服务幸存下来是很难理解的。

一年多前，作家兼企业家保罗·凯德罗斯基 写道“许多人正得出正确的结论，在Hulu、Boxess、Bittorrent等时代，有线电视是另一个娱乐时代价格过高的遗物。”现在，像苹果、谷歌、网飞和其他公司都在积极加入这场游戏，这个定价过高的古董的日子似乎更加屈指可数了。
