# Kiip 的 Brian Wong 谈年轻企业家的冒险精神

周一， [Kiip](https://web.archive.org/web/20230203015415/http://www.kiip.me/) 创始人 [Brian Wong](https://web.archive.org/web/20230203015415/http://www.crunchbase.com/person/brian-wong) 将的移动广告初创公司从秘密状态中拯救出来后，他有很多事情要做，主要是将最近 430 万美元的资金和一种新颖的产品和商业模式投入使用。Wong 甚至还不能合法饮酒！

今天早上，我们请他来到 TCTV 演播室，谈论 Kiip 背后的灵感以及其他事情。Wong 说，他是在做了一次“iPad 爬行”后产生这个想法的，也就是在飞机上走过过道时偷偷看一眼每个人的 iPad。他意识到每个人都在玩游戏，游戏是“参与的圣杯”，但游戏广告基本上占据了屏幕的一部分，没有增加任何价值。

Wong 随后提出了提供真实生活奖励的想法，*“这里必须有一个重置按钮，我们必须尝试一些新的东西……我们必须建立一个具有移动 DNA 的公司。”*

2009 年从不列颠哥伦比亚大学毕业后(跳过四个年级)，19 岁的他在 Digg 找到了一份业务发展的工作，直到一次裁员让他寻找下一个机会。尽管还很年轻，Wong 是一个坚持上学的人，他说:“完成一件事情的想法对企业家来说是不好的，我们太精神分裂了。[就像]嘿，让我们去这个项目，让我们去那个项目。好吧，让我们完成学业，让我们完成那个项目。”

Wong 还谈到了筹集资金的风险，以及人们可能会因为年龄而低估他和其他企业家。你走进这个合伙人会议(当我第一次打算筹集资金的时候，我不知道什么是合伙人会议)，如果有人接受你，他们将成为你的未来。你必须能够和他们交谈，能够理解他们。”

很明显，Wong 让某些人相信他在做一些事情，因为 True Ventures、Hummer Winblad 和 Carl's Jr、Sephora 和 PopChips 等蓝筹股品牌现在都给他投了信任票。

那么他的成功秘诀是什么呢？“我喜欢叫它‘盗梦空间’，就像电影一样。你必须首先播种这个想法，你必须让他们也想到它，同时你要揭示它。如果你有这种相似性，那么你就有了他们，愿景现在就在他们的头脑中。”
