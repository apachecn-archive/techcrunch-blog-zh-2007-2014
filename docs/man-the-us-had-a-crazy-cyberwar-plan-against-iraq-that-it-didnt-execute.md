# 伙计，美国对伊拉克有一个疯狂的网络战计划(它没有执行)

![armymen](img/df70525d22a847ac0a81f6fba85a1723.png "armymen")

这有多伟大:在 2003 年，五角大楼和情报机构已经有了一个计划，即“T2”向伊拉克发动一场网络攻击，这将会摧毁伊拉克的货币供应。换句话说，萨达姆·侯赛因不会有任何钱支付他的军队或其他任何东西。为毁灭欢呼三声！

现在，由于布什总统和他的顾问们害怕外部金融影响，军方实际上从未实施过袭击。(没有人确切知道会产生什么类型的附带损害。)一位网络战专家表示:“如果你不知道针对无辜第三方的反击会有什么后果，那就很难授权。”

网络攻击会有什么问题？“你不可避免地击中平民目标的可能性非常高——最糟糕的情况是摧毁一家与其他机构共享网络的医院，”一名硅谷计算机安全专家表示。

不用说，网络战在 1983 年达到顶峰。
