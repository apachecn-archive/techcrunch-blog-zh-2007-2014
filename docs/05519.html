Box.net升级个人和商业数据计划，在云中包含更多存储

基于云的存储和共享应用Box.net今天对其免费和付费计划进行大数据存储升级。该公司正在增加其个人、商业和企业计划的数据存储量。

Box的个人订阅计划现在提供5GB的免费网络存储空间。Box的商业计划正在被略微调整，这样公司就不用为个人用户付费了(以前每个用户给公司15GB的空间)。现在，企业将自动获得500GB的数据存储，是普通企业用户可访问的存储量的三倍。Box.net的首席执行官Aaron Levie说，企业客户将可以访问无限量的数据存储，使公司能够在云端管理万亿字节的数据(Box向企业客户收取每个用户每月35美元的费用)。

为了适应这些变化，Box建立了两个企业级数据中心，并获得了SAS 70 Type II认证。据该公司称，包括夏威夷航空公司、T-Mobile和ABC News在内的60，000多家企业使用Box作为数据存储库。Levie的未来愿景是，企业和个人将永远不必担心数据存储，Box将成为访问他所谓的“无限云”的首选平台。

今年早些时候，Box筹集了1500万美元的新资金，对于一家与微软Sharepoint等公司竞争的初创公司来说，它已经实现了稳步增长。自2005年推出以来，Box.net已经积累了超过400万用户，有数十万商家在使用该应用。去年，这家初创公司的收入增长了500%，由于与奥普拉温弗瑞网络、沃尔沃和诺基亚西门子的交易，今年的销售额也有所增长。

这家初创公司在过去一年里还不断推出新功能和新产品，包括文件同步功能、安卓应用、 HTML5功能等等。
