# 新 Beats By Dre Monster 耳机是无线的，丰富多彩的:我们竖起耳朵听

我很幸运地在 Monster 最近的假日发布会上看到了 Monster 的新 Beats by Dre 耳机，可以说它们是非常甜的罐头。该公司开发了两套新耳机:Beats 无线入耳式耳机和 Beats Studio HD Powered 隔离耳机，它们有多种颜色可供选择。

我有机会亲身体验了这两款新产品，老实说，我很享受与它们相处的时光。两套都超级舒服。

[Beats 无线耳机](https://web.archive.org/web/20230203144224/https://techcrunch.com/2011/09/06/beats-wireless/)在耳罩上带有控制装置，允许你调节音量、跳过曲目、打开和关闭耳机。由于易拉罐使用蓝牙技术，你甚至可以将它们与你的手机同步，这样每次你接到电话时音乐就会停止。耳机由电池供电，可以充电。

Beats Studio 耳机是 Beats 的旗舰产品，但该公司在假期推出了一些额外的颜色。以前只有黑色、白色和红色，现在有蓝色、橙色、洋红色和紫色。它们配备了一根额外的怪物 iSoniTalk 耳机线，带有内置麦克风和呼叫应答按钮，如果你在摇摆时接到电话，你可以在电话上聊天。

五颜六色的新款 Beats Studio 罐将于 10 月 16 日上市，零售价为 299 美元，而 Beats 无线耳机将于今年晚些时候在苹果商店独家发售，零售价为 279 美元。
