# 现场有 350 名投资者，TechStars 首次推出新的云加速器首批产品 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/>

9 月，国家创业加速器和种子基金 [TechStars](https://web.archive.org/web/20221205225817/http://www.techstars.com/) 在[推出了其首个项目](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2011/09/13/techstars-launches-cloud-startup-accelerator-bonus-new-trailer-of-its-reality-tv-show/)，专注于为在特定领域工作的公司提供指导和资金。随着云计算和服务的迅猛发展，该公司认为专注于基于云的初创公司是合适的，推出了 [TechStars Cloud](https://web.archive.org/web/20221205225817/http://www.techstars.com/cloud/) ，这是一个子加速器，专门专注于支持云计算和基础设施初创公司。

今天，第一批 11 个多云的 TechStars 初创公司在 TechStars 联合创始人兼首席执行官[大卫·科恩](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2011/09/13/techstars-launches-cloud-startup-accelerator-bonus-new-trailer-of-its-reality-tv-show/)告诉我们预计将有 350 多名投资者面前演示。今天的演示展示标志着为期三个月的计划的高潮，该计划于 1 月份开始，总部设在德克萨斯州的圣安东尼奥。

作为 TechStars 去年 9 月筹集的 2400 万美元资金的一部分，加速器现在向每个初创公司提供额外的 10 万美元种子资金，允许早期企业在整个计划期间专注于他们的产品，而不必担心向投资者推销产品。除此之外，初创公司在加入该计划后还会获得 1.8 万美元的资助，这使得每个创始团队的潜在资金达到 1.8 万美元。

这对 TechStars 来说是一个重大举措，考虑到尤里·米尔纳和 SV Angel [去年年初联手向同为创业孵化器 Y Combinator 的公司](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2011/01/28/yuri-milner-sv-angel-offer-every-new-y-combinator-startup-150k/)提供 15 万美元的可转换债券。

除了资金，2011 年对加速器来说也是重要的一年，因为它还开放了其模型，宣布了全球加速器网络，作为白宫刺激创业的美国创业计划的一部分。孵化器的模式现在被全球超过 35 个项目使用。此外，除了圣安东尼奥的云加速器，该孵化器还在波士顿、博尔德、纽约和西雅图开展项目。

今天，TechStars 为读者提供了在圣安东尼奥展示其产品的 11 家初创公司的预览，你可以在下面查看。我们从 accelerator 了解到，这 11 家公司中的每一家都已经承诺(或最近已经完成)一轮种子资金，Cohen 表示，他预计在演示日之后，外部投资者将为整批公司提供资金。

[![](img/a21d6c77b7e7d368c526f752dbcdeb37.png "Screen shot 2012-04-11 at 7.16.42 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-16-42-am/) [Appsembler](https://web.archive.org/web/20221205225817/http://appsembler.com/) 是一个“盒子里的生意”,面向那些想要销售软件即服务的公司。像其他平台托管公司一样，他们提供可扩展和按钮式应用程序托管，但除了简单的托管，他们的服务还为公司提供集成的计费、支持票和针对 SaaS 产品的 SEO 优化 web 内容。

[![](img/b3bbf4f0c3ff9c85a9479db55f7d739d.png "Screen shot 2012-04-11 at 7.18.29 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-18-29-am/) [Callisto.fm](https://web.archive.org/web/20221205225817/http://callisto.fm/) 是一家下一代媒体分析公司。专为媒体创作者打造，他们可以告诉谁是真正参与内容。该服务是基于云的，实时的，可以在任何设备、平台或服务上使用任何类型的媒体。他们围绕互动(播放、暂停、搜索、分享、喜欢等)捕获丰富的分析，以便为内容制作者提供可操作的对受众的洞察。

[![](img/b14477ff119e9b75f70ac41684c55798.png "Screen shot 2012-04-11 at 7.19.09 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-19-09-am/) [云能力](https://web.archive.org/web/20221205225817/https://www.cloudability.com/)监控和管理云中的支出。它们帮助公司更好地了解他们在云中的支出，识别浪费，并就如何优化支出提出建议。它们还允许公司将来自数百个供应商和数十个内部帐户的所有帐户纳入一个统一的视图。Cloudability 目前为 2，200 家公司管理着超过 5，300 万英镑的云支出，本周[宣布与 Rackspace 合作](https://web.archive.org/web/20221205225817/https://www.cloudability.com/blog/every-rackspace-user-just-got-a-free-cloudability-account/)，允许 Rackspace 的所有 175，000 名云客户免费使用 Cloudability。客户包括从 StockTwits 和 Slideshare 到 Adobe 和 Pega Systems 的公司。

[![](img/4c4da0d4775b83dff0421ec55217fec5.png "Screen shot 2012-04-11 at 7.20.07 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-20-07-am/) [CloudSnap](https://web.archive.org/web/20221205225817/http://cloudsnap.com/) 是集成即服务。它们允许用户将数百个 SaaS 产品中的任何一个连接在一起，而无需编写一行代码。这使您可以进行大量有用的工作流程和数据同步，例如将 Mailchimp 销售线索连接到 Salesforce 交易或将 Shopify 发票导入 Netsuite。所有这些都是一键操作，不需要开发人员花费时间，也不需要持续维护。

[![](img/9f7ac6e9929027d07d87ca3d2c82c1bc.png "Screen shot 2012-04-11 at 7.20.56 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-20-56-am/) [Conductrics](https://web.archive.org/web/20221205225817/http://www.conductrics.com/) 是一项决策优化服务，不仅仅是简单的 A/B 测试和报告。Conductrics 使用机器学习来获取客户提供的一组选择，并动态和适应性地学习最佳选择。当条件发生变化时，服务会学习并自动更新新的最佳选择。这使得网站或应用程序总是使用最高的转换内容。像安德玛这样的客户正在将 Conductrics 直接集成到他们的 CMS 中，这使他们能够将适应性行为直接引入内容创建过程。

[![](img/f7da6284d83e1923f0c3f4acb0f5e60c.png "Screen shot 2012-04-11 at 7.21.49 AM")](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-21-49-am/)[distilt](https://web.archive.org/web/20221205225817/http://www.distil.it/)是全球第一个内容保护网络。云服务位于客户的网络服务器前，识别并阻止抓取数据的企图。谷歌在搜索结果中重罚拥有非独特内容的网站，通过使用 Distil，公司可以通过阻止内容盗窃来遏制搜索引擎优化、访客和收入的损失。

[![](img/4ebefbfec56ff375cbbb181137bec9e1.png "Screen shot 2012-04-11 at 7.23.19 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-23-19-am/) [EmergentONE](https://web.archive.org/web/20221205225817/http://www.emergentone.com/) 是第一个 API 生成平台。使用简单的 web UI，客户可以连接到他们现有应用程序的数据库，并自动生成一个健壮的、设计良好的 API。初创公司可以在几分钟内为现有应用程序生成一个完整的定制 REST API。除了 API 本身，EmergentONE 还生成开发人员文档、特定语言库和后端分析视图。这家初创公司允许客户只公开他们想要的数据，除非他们明确要求，否则数据不会包含在 API 中。该解决方案适用于所有关系数据库，并提供细粒度的访问控制。相当酷。

[![](img/b597e007489cc441a60f231881b6a7fc.png "Screen shot 2012-04-11 at 7.24.03 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-24-03-am/) [Flomio](https://web.archive.org/web/20221205225817/http://flomio.com/) 是一个构建 NFC 和 RFID 应用的平台。该服务不受硬件限制，基于云。这意味着 Flomio 可以与任何支持 NFC 的设备一起工作，并屏蔽了 NFC 的所有复杂性，将服务作为简单的 web API 公开。他们还出售标签、阅读器和一个“方形”音频插孔插件，可以将任何手机转换成 NFC 阅读器。此外，Flomio 在分析仪表板中为其客户提供数据的汇总视图。

[![](img/ae35d09ea6e48685dd90d20f456a4057.png "Screen shot 2012-04-11 at 7.24.46 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-24-46-am/) [Keen.io](https://web.archive.org/web/20221205225817/http://keen.io/) 是一个面向移动和应用开发者的分析平台。基恩认为，应用程序创建者应该能够以自然的形式收集所有数据，而不是将数据转化为一些预先打包的分析服务。一旦客户的数据进入 Keen.io 服务，他们就可以成为自己的数据科学家，对数据进行切片和切片，并根据历史数据进行实验，以深入了解应用用户的行为。

[![](img/3e3b1e2316ab58becea0f83becc5a627.png "Screen shot 2012-04-11 at 7.25.23 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-25-23-am/) [Tempo](https://web.archive.org/web/20221205225817/http://tempo-db.com/) 是一个托管的时间序列数据库。时间序列数据正在成为“大数据”领域的一个巨大组成部分。每个设备和传感器都在创建数百万或数十亿个数据点，而现有的大数据系统并没有专门针对这种类型和数量的数据进行优化。Tempo 是专门为时间序列数据而构建的，允许公司持有前所未有的大量数据，让我们最终从我们新的测量世界产生的所有数据中获得价值。

[![](img/693976f93153fc8fb4854e3339751ecc.png "Screen shot 2012-04-11 at 7.26.06 AM") ](https://web.archive.org/web/20221205225817/https://beta.techcrunch.com/2012/04/11/techstars-debuts-11-cloud-startups/screen-shot-2012-04-11-at-7-26-06-am/) [Vidmaker](https://web.archive.org/web/20221205225817/http://vidmaker.com/) 让管理和编辑任何设备、任何地点和任何人的视频变得容易。该服务有一个集成的基于 web 的视频编辑器、Google Docs 风格的视频协作、一个内置的社区素材库和大量其他功能，使创建、编辑和发布视频内容的过程变得愉快而不是乏味。
