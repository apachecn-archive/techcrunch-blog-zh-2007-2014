# 惠普报告第三季度收益下降 5%，打印机下降 23%

惠普报告第三季度收益与去年相比下降了 5%。打印机销量下降了 23%，反映了整个市场的趋势。收入为 297 亿美元，低于 312 亿美元。首席执行官梅格·惠特曼表示

惠普的每股收益为 1.00 美元，高于此前提供的每股 94 至 94 美分的预期，与其预先宣布的一致。但每股收益较上年同期下降了 9%。对于 2012 财年，惠普估计非 GAAP 摊薄后每股收益在 4.05 美元至 4.07 美元之间，处于之前预测的低端。

与戴尔昨天的业绩一样，惠普显示了其消费产品的最大跌幅，硬件和打印机部门的收入同比下降 13%，下降 23%。

惠特曼表示，该公司将于今年秋季推出一系列以“设计”为重点的新个人电脑和平板电脑。惠普还将推出新的安全软件和企业服务产品。
