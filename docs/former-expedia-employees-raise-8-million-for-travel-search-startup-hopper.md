# Expedia 前员工为旅游搜索初创公司 Hopper 融资 800 万美元

总部位于加拿大蒙特利尔的初创公司 Hopper 已经从 [Atlas Venture](https://web.archive.org/web/20221207021752/http://www.crunchbase.com/financial-organization/atlas-venture) 和之前的投资者 [Brightspark Ventures](https://web.archive.org/web/20221207021752/http://www.crunchbase.com/financial-organization/brightspark-ventures) 获得了[800 万美元](https://web.archive.org/web/20221207021752/http://www.crunchbase.com/company/hopper)的新资金，试图“重塑旅游搜索”。

Hopper 仍处于秘密模式，但如果你想了解他们正在做什么，请查看 Xconomy Boston 的这篇[Boston.com 简介](https://web.archive.org/web/20221207021752/http://www.boston.com/business/technology/innoeco/2011/07/travel_site_hopper_with_backin.html)或[这篇文章](https://web.archive.org/web/20221207021752/http://www.xconomy.com/boston/2011/08/22/hopper-with-8m-in-new-vc-bucks-looks-to-leapfrog-online-travel-search-via-big-data/)。

事实上，Hopper 计划在波士顿开设一个新的办事处，预计明年工程团队将增加一倍。

该公司本质上是为消费者打造一个“旅游发现”引擎，让人们只需使用“欧洲最佳海滩”等关键词就能发现目的地和产品。

这家初创公司是由首席工程师塞巴斯蒂安·雷恩维尔(Sebastien Rainville)、首席执行官弗雷德里克·拉隆德(Frederic Lalonde)和产品总监约斯特·欧文克尔(Joost Ouwerkerk)构想的。后两者来自 Expedia，后者在 2002 年收购了拉隆德之前的公司 Newtrade Technologies。

Hopper 最初成立于 2007 年，之前在 2008 年筹集了 200 万美元，使公司的总资金达到今天的 1000 万美元。

Atlas Venture 的合伙人杰夫·法格南(Jeff Fagnan)将加入 Hopper 董事会。
