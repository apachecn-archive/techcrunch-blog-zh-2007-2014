# StrongLoop 为基于 Node.js 的移动应用平台筹集了 800 万美元

[StrongLoop](https://web.archive.org/web/20221205110655/http://strongloop.com/) 已经为其后端即服务(MBaaS)筹集了 800 万美元的首轮融资，该公司使用 [Node.js](https://web.archive.org/web/20221205110655/http://nodejs.org/) 作为在云中或数据中心开发移动应用的平台。这笔资金来自 Shasta Ventures 和 Ignition Partners，使 StrongLoop 的总资金达到 900 万美元。

该公司提供了一个移动 API 层，作为开发人员访问其应用后端数据的一种方式。新任首席执行官艾萨克·罗斯(Isaac Roth)曾是红帽公司的高管，曾帮助推动该公司的平台即服务 OpenShift，他表示，该公司正在寻求将移动设备与企业连接起来，就像网络通过浏览器连接到企业后端环境一样。

有了这笔资金，StrongLoop 正在推出 [LoopBack](https://web.archive.org/web/20221205110655/http://strongloop.com/strongloop-suite/loopback/) ，这是一个 MBaaS，可在亚马逊网络服务、Rackspace、Cloud Foundry、Heroku、Red Hat OpenShift 和在线开发平台 Cloud9 IDE 上使用。LoopBack 构建于 Node.js 之上，抽象了后端集成的复杂性。通过这项服务，开发人员可以将他们的设备和浏览器连接到云服务或企业数据中心的数据。

今天的区别在于数据的速度和容量以及人们使用的设备类型。数据通过智能手机和平板电脑更新到 Twitter、短信和电子邮件中。笔记本电脑和个人电脑仍在广泛使用，但转向移动技术是真实的。向移动设备的转变与 PC 时代形成了鲜明对比，当时人们主要使用网络浏览，使用电子邮件发送大文件。

StrongLoop 正押注于服务器端 JavaScript 编程语言 Node.js，以推动新一代应用的发展。Node.js 旨在管理大容量数据负载，主要用于移动服务，它在后端处理多个打开的会话，加载所需的数据。

成千上万的 JavaScript 开发者已经转向 Node.js，连接到 MongoDB 的 LoopBack 将会吸引这个社区。

MBaaS 市场开始看起来挤满了老牌玩家，如已经在市场上享有强大影响力的 [Kinvey](https://web.archive.org/web/20221205110655/https://beta.techcrunch.com/2013/06/05/google-app-engine-adds-kinvey-to-power-mobile-backend/ "Google App Engine Adds Kinvey To Power Mobile Backend") ，以及知名 Node.js 平台 [Feed Henry](https://web.archive.org/web/20221205110655/http://www.feedhenry.com/) 。
