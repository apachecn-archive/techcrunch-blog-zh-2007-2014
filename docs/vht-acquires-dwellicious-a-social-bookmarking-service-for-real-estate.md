# VHT 收购了房地产社交书签服务网站 Dwellicious

先驱社交书签网站[好吃](https://web.archive.org/web/20230205003627/http://www.crunchbase.com/company/delicious)得到[被收购](https://web.archive.org/web/20230205003627/https://techcrunch.com/2011/04/27/yahoo-sells-delicious-to-youtube-founders/) ( [两次](https://web.archive.org/web/20230205003627/https://techcrunch.com/2005/12/09/yahoo-acquires-delicious/))还不够。不， [VHT](https://web.archive.org/web/20230205003627/http://www.vht.com/) 刚刚*已经*着手[购买](https://web.archive.org/web/20230205003627/http://www.prnewswire.com/news-releases/vht-acquires-dwellicious---social-bookmarking-website-for-real-estate-125552148.html) [Dwellicious](https://web.archive.org/web/20230205003627/http://dwellicious.com/) ，这是一个奇怪命名的服务，使人们能够以同样的方式在线书签、标记和组织房地产。

此次收购意义重大，交易条款尚未披露。VHT 提供在线营销房地产的技术和服务，并将把 Dwellicious 整合到其 [ImageWorks](https://web.archive.org/web/20230205003627/http://vht.com/VHTImageWorks/) 在线营销平台中，为经纪客户提供与购房者沟通的工具。

Dwellicious 使用社交书签来帮助购房者在脸书、Twitter 或其他社交媒体服务上分享他们最喜欢的房产。买家可以组织、监控和比较房源，做笔记，添加标签，与朋友、家人和房地产专业人士分享和讨论房产。

全美有超过 100，000 名房地产专业人士使用 VHT 图像工程公司。
