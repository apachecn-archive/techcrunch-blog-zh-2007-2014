# Windows Live SkyDrive:随时随地观看、分享精彩内容

![sky.jpg](img/a5d43b23549a18a3d3319261b9777617.png)

如今网上有一些在线存储和共享网站，但 Windows Live SkyDrive(又名 Windows Live Folders)有很大不同，似乎更容易使用。您只需要一个 Windows Live ID，例如 Hotmail 或 MSN 帐户，就可以立即获得 500MB 的空间，与朋友和家人、公众或个人共享。

所有这些都可以在任何装有 IE 或 Firefox 的 PC 或 Mac 上通过网络访问。您可以完全控制用户是否可以修改/删除/添加文件到共享文件夹，这使它成为学生和小猴子的完美工具。SkyDrive 的关键在于它对文件使用的拖放方法。太棒了。你还在等什么？去报名吧。
