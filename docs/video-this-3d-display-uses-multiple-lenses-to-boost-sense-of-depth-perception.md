# 视频:这种 3D 显示器使用多个镜头来增强深度感

裸眼 3D 显示器，即使是[大尺寸型号](https://web.archive.org/web/20230204231546/https://techcrunch.com/2011/01/25/nict-shows-200-inch-naked-eye-3d-display/)，也不再是什么特别的东西，但它们通常都有一个共同的问题:观看图片时的 3D 效果不如需要用户戴眼镜的显示器强烈。T2 筑波大学的教授 Kakeya 正试图解决这个问题。

他的 3D 显示器的工作方式实际上非常简单:它使用多层和透镜来增强深度感。Kakeya 教授解释道:

> 它形成了前面的物体朝向前面，后面的物体朝向后面的图像。当前面的物体对焦时，后面的物体模糊不清，当你看后面的物体时，前面的物体模糊不清。所以这种显示器的一个特点是它再现了焦深。

目前原型的分辨率仅为 200×200，但另一个很酷的功能是，它不仅允许你水平移动头部，还允许你垂直移动头部来观看 3D 图片。
