# Flipboard 正式向感兴趣的测试者开放他们的 Android 测试版

嗯，真快。Flipboard Android 测试版即将推出的消息今天早上才开始流传，现在 Flipboard 团队已经向任何有兴趣尝试的人开放了测试程序。

有兴趣试用该应用预发布版本的用户可以逛逛 Flipboard 的 [Android 登陆页面](https://web.archive.org/web/20221209021108/http://flipboard.com/android/)，在那里他们可以注册测试版，并耐心等待在收件箱中收到下载链接。确认邮件提到等待时间可能长达 24 小时，尽管我们听说很多人在没有太多延迟的情况下就收到了他们的[下载链接](https://web.archive.org/web/20221209021108/http://www.droid-life.com/2012/05/30/flipboard-beta-released-for-android-download-it-now/)。

我设法在我信赖的 Galaxy Nexus 上安装并运行了新版本的应用程序，尽管它是测试版，但迄今为止它似乎反应很快(更不用说很帅了)。滚动源列表来筛选内容非常快，当我触摸这些源的图标来查看更多内容时，我只能察觉到最微弱的延迟。假设您已经有了一些健壮的硬件，您可能不会遇到太多问题。

然而，并不是所有的设备都会如此幸运。我下载了这款应用，并在我的备用手机上进行测试——这是一款不算太旧的 myTouch 4G 手机——安装后被告知，这款设备“不符合 Flipboard 不支持的最低要求。”在选择了一些类别添加到我的 Flipboard 后，应用程序停止了。我猜 Flipboard 决定了他们的路线是件好事，如果运气好的话，在未来几个月更广泛的发布会上，最终产品将不会出现问题。

您的里程数肯定会有所不同——这是整个练习的要点 Flipboard 团队将感谢您无畏的测试人员可能提供的任何反馈。如果应用程序碰巧强制关闭，请做一个好公民并报告问题，而不是关闭应用程序并从头开始，好吗？
