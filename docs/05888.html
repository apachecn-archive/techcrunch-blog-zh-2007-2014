原文：https://web.archive.org/web/https://techcrunch.com/2010/12/08/dennis-crowley-pretty-much-confirms-they-turned-down-a-140-million-acquisition-offer/

丹尼斯·克劳利似乎证实了他们拒绝了1.4亿美元的收购要约

在Le Web 10上，当主持人Loic Le Meur问他该公司最近拒绝的收购报价有多大时，每个人都看到了Foursquare 首席执行官 Dennis Crowley 脸上尴尬的表情。

Le Meur猜测约为1.4亿美元(没有真正提到报价来自脸书、雅虎还是其他公司，但他们是潜在的竞标者，当问题出现时，舞台上的话题是脸书)。

克劳利没有否认，然后继续承认这个数字是“大概的”。

这意味着Foursquare拒绝了大约1.4亿美元的报价，可能来自脸书，当时他们只筹集了大约135万美元的种子资金——在收购谈判平息后，他们继续筹集了2000多万美元。有远见，还是愚蠢？勇敢，还是泡沫？

值得一提的是:Foursquare刚刚超过了500万用户，每天有200万人签到，25000名新用户。
