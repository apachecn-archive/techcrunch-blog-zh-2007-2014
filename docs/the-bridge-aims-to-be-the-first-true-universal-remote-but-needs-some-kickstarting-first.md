# 该桥旨在成为第一个真正的通用遥控器，但首先需要一些启动

总的来说，通用遥控器是用词不当。它们并不是真正通用的。即使是那个愚蠢的亚当·桑德勒电影里的遥控器也不能控制 HTPC。通用遥控器通常只适用于传统的 AV 设备。但不是来自纽约的初创公司 Convergence Technologies 的桥梁。这种遥控器的概念是一个真正的通用遥控器，能够控制从有线电视盒到游戏系统到数字流媒体的高清电视。但是他们需要你的帮助。

融合技术公司[正在通过 Kickstarter 寻找 5 万美元的资金。这笔资金将允许该公司开发必要的软件，同时建造和销售遥控器。和大多数 Kickstarter 项目一样，这里也有一些风险。如果公司真的能造出这个东西的话，这个设备在纸上听起来和看起来都很棒。](https://web.archive.org/web/20230205045825/http://www.kickstarter.com/projects/50581889/the-bridge-the-first-truly-universal-remote)

初创企业需要为每个 AV 设备收集(或许可)必要的 IR 代码。该公司声称，该桥还能够控制 Boxee Box 等射频设备，因此也需要找到这些代码。更重要的是，创造者声称他们能够以远低于 100 美元的价格将这位人类的救世主推向市场。尽管如此，只要他们得到必要的资金，这个梦想是远大的，而且肯定是可以实现的。这就是你进来的地方。

Convergence Technologies 向 Kickstarter 求助。该网站的众筹模式允许任何人向该项目投入几美元，以换取一些赃物和温暖的模糊感觉。创作者[正在寻找 50，000 美元](https://web.archive.org/web/20230205045825/http://www.kickstarter.com/projects/50581889/the-bridge-the-first-truly-universal-remote)并且在这篇文章写作时，有 47 天达到这个目标。认捐 10 美元，你就能得到一枚胸针。20 美元可为您赢得一件 t 恤，而 65 美元可让您获得首批产品之一。
