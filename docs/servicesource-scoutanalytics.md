# 续订点播提供商 ServiceSource 以 3200 万美元收购 Scout Analytics

为 Renew OnDemand 等企业提供经常性收入管理软件的 ServiceSource 今天宣布，它已经收购了总部位于西雅图的订阅业务预测分析提供商 [Scout Analytics](https://web.archive.org/web/20221209103046/http://scoutanalytics.com/) 。交易条件是 3200 万美元现金。

在一份声明中，ServiceSource 表示，这些垂直市场“其潜在市场大约翻了一番，目前经常性收入机会超过 6000 亿美元。”Scout Analytics 目前的客户包括[金融时报](https://web.archive.org/web/20221209103046/http://www.ft.com/)和 [Reed Elsevier](https://web.archive.org/web/20221209103046/http://www.reedelsevier.com/Pages/Home.aspx) ，以及信息服务公司 [Wolters Kluwer](https://web.archive.org/web/20221209103046/http://www.wolterskluwer.com/Pages/Home.aspx) ，后者也是 ServiceSource 的现有客户。

然而，并不是每个人都对这笔交易感到满意。 [GeekWire](https://web.archive.org/web/20221209103046/http://www.geekwire.com/2014/early-backers-scout-analytics-arent-happy-claim-venture-capitalists-engaged-self-dealing-corporate-waste-prior-acquisition/) 报道称，一群 A 轮优先股股东上周提起诉讼，指控 Scout Analytics 董事会未能探讨 Catalyst Investors III 提出的反融资要约，该要约对该公司的估值为 4000 万美元。ServiceSource 和 Scout Analytics 都拒绝对诉讼置评。

Scout Analytics 管理着超过 35 亿美元的经常性收入，ServiceSource 表示，此次收购将使其能够在 200 多个客户中将其业务扩展到 140 亿美元，进入新的市场，并为订阅和传统业务提供经常性收入管理服务。

Scout 是 Scout Analytics 的产品之一，它跟踪客户的订阅使用情况、支出和其他活动，并提供公司可以用来改善帐户管理、留住客户和制定营销策略以向用户销售更多服务的信息。ServiceSource 表示，随着越来越多的人开始使用智能设备和云服务，经常性收入将在物联网中发挥“关键作用”。

现在收购了 Scout Analytics，ServiceSource 拥有了一个新的潜在客户群，其中包括以循环支付模式运营的消费科技公司。Scout Analytics 目前每天为软件即服务、信息服务和数字媒体行业的客户分析超过 4 亿台设备上约 2500 万用户的使用情况。

“Scout Analytics 大大扩展了我们在快速增长的 SaaS 市场的影响力，同时为信息服务和数字媒体的 B2B 订阅创造了新的机会。此外，通过添加复杂的预测分析，我们可以为公司提供所需的自上而下的客户数据视图，以通过经常性收入实现增长，”ServiceSource 董事长兼首席执行官 Mike Smerklo 在一份声明中表示。
