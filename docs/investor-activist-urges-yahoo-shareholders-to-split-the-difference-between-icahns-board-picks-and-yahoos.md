# 投资者维权人士敦促雅虎股东分割伊坎和雅虎董事会成员的分歧

维权股东埃里克·杰克逊(Eric Jackson)对所有持有雅虎股票的小人物有一些建议:到 8 月 1 日，在雅虎董事会现有董事和卡尔·伊坎(Carl Icahn)的替代人选之间分配分歧。如果大部分董事会成员被替换，那么[毒丸遣散费](https://web.archive.org/web/20221209120011/http://www.beta.techcrunch.com/2008/02/20/microsoft-yahoo-battle-getting-expensive-should-microsoft-give-up/)可能会被触发。因此，杰克逊建议股东投票支持伊坎的四位提名人(亚当·戴尔、约翰·查普尔、卢西恩·贝布丘克和爱德华·迈耶)，并保留五位现任董事会成员(Vyomesh Joshi、Robert Kotick、Maggie Wilderotter、Gary Wilson 和 Jerry Yang)。

这听起来像是造成完全僵局的一个原因。幸运的是，杰克逊只代表了 150 名雅虎股东中的一小部分，他们总共拥有 14 亿股流通股中的 320 万股(相比之下，伊坎拥有 6900 万股)。但是 Jackson 希望在接下来的两周内吸引更多的会员加入他的团队，他在 T4 的 Agoracom 上创建了一个“雅虎 B 计划”论坛，将雅虎的小股东聚集在一起，共同为雅虎提出一个更好的计划。

这和他去年的[激进计划是同一个名字，](https://web.archive.org/web/20221209120011/http://seekingalpha.com/article/27938-yahoo-time-for-plan-b)当时他呼吁当时的首席执行官特里·塞梅尔下台。他还呼吁更换七名雅虎董事会成员，包括罗伯特·科蒂克和加里·威尔逊(他认为现在应该留下的两名雅虎董事会成员)。

争夺股东的战斗正在进行。雅虎自己也在利用其主页直接向股东陈述自己的观点。接下来的两周应该会有更多的雅虎积极分子走出困境。夏天的乐趣才刚刚开始。
