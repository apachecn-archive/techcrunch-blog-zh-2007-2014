# 英特尔投资在电视电子商务平台交付代理方面领先 3550 万美元

电视电子商务公司[交付代理](https://web.archive.org/web/20230204073353/http://www.deliveryagent.com/)已经筹集了[3550 万美元](https://web.archive.org/web/20230204073353/http://www.marketwatch.com/story/delivery-agent-raises-355-million-led-by-intel-capital-with-liberty-global-and-grazia-equity-2011-11-03)的新资金，由英特尔投资公司牵头，战略投资者、Liberty Global 和 Grazia Equity 参与其中。E 轮融资包括所有当前投资者的再投资，包括 Bessemer Venture Partners，迄今为止总融资额为[1.073 亿美元。](https://web.archive.org/web/20230204073353/http://www.crunchbase.com/company/deliveryagent)

Delivery Agent 通过两种不同的渠道帮助电视网络将内容货币化。首先，该公司为电视节目中的产品和商品创建了在线市场。交付代理已经签署了所有的主要网络，包括 ABC，CBS，NBC 环球，二十世纪福克斯和 MTV 网络。

送货代理将电影和演出中出现的产品分类，然后创建一个可以购买产品的电子商务网站。通过将网络上看到的产品的所有电子商务外包给交付代理，网络能够在保持低管理成本的同时将其内容货币化。

送货代理的第二项服务是协调生产电视节目产品的品牌的广告。交付代理将在剧集播出前浏览将在节目中播出的产品的索引，然后联系品牌并为他们提供购买节目广告包的能力。到目前为止，Delivery Agent 已经处理了数百万次传统网络、社交、移动和连接电视的娱乐商务交易。

该公司在过去几年进行了大量收购，包括媒体公司 Band 和价值 10 亿美元的 flash 销售网站 Babes。
