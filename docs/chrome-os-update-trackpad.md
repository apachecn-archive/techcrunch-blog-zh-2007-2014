# Chrome 操作系统获得重大更新——两个词:触控板修复

几周前，[谷歌推出了最新版本的浏览器 Chrome 10](https://web.archive.org/web/20230203021010/https://techcrunch.com/2011/02/17/chrome-10-crankshaft/) 的测试版。今天他们也将它带到了 Chrome 操作系统中——它包含了更多的好东西。最重要的是，谷歌正在兜售触控板修复程序，可能是针对人们一直用来测试操作系统的 Cr-48 设备。终于！

我还没有机会测试触控板修复，但谷歌在他们的博客文章的第一句话中特别提到了“新触控板”，这似乎是一个好迹象。

在我对 Cr-48 的最初评价中，我用力撕扯了触控板。理由很充分:这太可怕了。希望这是软件可以解决的问题。

新 Chrome OS 版本的完整更新列表——有很多:

*   3G 调制解调器激活修复
*   3G 连接到运营商修复
*   Wi-Fi 连接/现成的修复
*   调整了新的触控板和灵敏度设置
*   自动更新引擎和调试改进
*   电源优化
*   GTalk 视频/聊天优化
*   音频 CPU 利用率提高
*   改进了屏幕指示器:亮度、网络状态、更新图标

此版本中还包含一个 scratchpad 应用程序安全漏洞修复程序。从技术上讲，这是 Chrome OS 的 0.10.156.46 版本。
