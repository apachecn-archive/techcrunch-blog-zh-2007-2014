# Warby Parker 在纽约开设零售店，波士顿紧随其后，在线下击败谷歌和亚马逊

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/13/warby-parker-opens-retail-store-in-nyc-with-boston-up-next-beats-google-amazon-to-the-offline-punch/>

过去两年，时尚在线眼镜初创公司 Warby Parker 一直在与精品店合作，开设“店中店”，即小型 Warby Parker 展厅，顾客可以在这里试戴 3d 眼镜。这些展厅出现在洛杉矶、纳什维尔、旧金山和其他许多地方。今天，Warby Parker [正式宣布其在纽约 SoHo 区的第一家旗舰店](https://web.archive.org/web/20230315095315/http://blog.warbyparker.com/post/47783604997/welcome-to-121-greene)。

随着电子商务的成熟，T2 的一些年轻公司开始转向更传统的广告形式。One Kings Lane、Gilt Groupe、Beachmint、Fab 和 Wayfair 只是过去六个月在电视上发起广告活动以吸引观众使用移动应用和网站的几个精品在线品牌。

现在，如果 Warby Parker 是任何迹象，电子商务公司可能会通过转移到线下和建立自己的实体零售店来继续扩大业务。亚马逊首席执行官杰夫·贝索斯,[在](https://web.archive.org/web/20230315095315/http://news.cnet.com/8301-1023_3-57551081-93/jeff-bezos-we-would-love-to-open-amazon-retail-stores/)前表达了兴趣，而 9to5Google 报道称，谷歌希望[在假期](https://web.archive.org/web/20230315095315/http://9to5google.com/2013/02/15/to-get-products-into-more-hands-google-will-open-its-own-stores-by-the-end-of-the-year/)前开设零售店。Warby Parker 并没有这样的市值，但显然它想与最大的线下零售商正面交锋。

[![tumblr_inline_ml5drgQ2sc1qz4rgp](img/5a87706a67a8ab3af816f8e39d1b2ac9.png)](https://web.archive.org/web/20230315095315/https://techcrunch.com/wp-content/uploads/2013/04/tumblr_inline_ml5drgq2sc1qz4rgp.jpg)

有趣的是，Warby Parker 的成立在一定程度上是因为(当时)只有不到 1%的眼镜在网上销售，所以这家初创公司希望通过创建一个强大的时尚品牌，并以更低的价格和更高的利润在网上销售，来挑战像 [Luxottica](https://web.archive.org/web/20230315095315/http://en.wikipedia.org/wiki/Luxottica) 这样的巨头——它拥有似乎无处不在的 LensCrafters，Ray-Ban，Sunglass Hut 和 Oakley 等。这很有效，这家初创公司一举成名，并开始吸引消费者和投资者的注意。

Warby 在 2 月底完成了一笔 4150 万美元的投资，部分由 J. Crew 首席执行官米勒德·s·德雷克斯勒(Millard S. Drexler，他曾是 Gap 的首席执行官，目前拥有一个令人敬畏的名字)和美国运通以及许多风投公司领导。凭借零售界的大力支持和越来越多的展厅，该公司现在正在更积极地尝试线下推广。但由于与分析初创公司 Nomi 的合作，它正试图将同样的学术美学/品牌转移到线下，同时保留一点技术。

它在纽约的新旗舰店设计得像一个经典的老图书馆——实际上是纽约公共图书馆——有黄铜图书馆灯、旋转梯子和发霉的书籍。整个交易。在某种程度上，这感觉有点自命不凡，但考虑到产品，这也很棒，很有意义；你用你的眼镜来阅读，书呆子——有时甚至是互联网以外的东西。

而事实上，几乎是相反的。这家初创公司的新店没有把眼镜放在眼镜后面，而是把它放在户外，让人们试驾。这家商店还包括一名内部验光师，每周七天提供 50 美元的眼科检查。非处方眼镜将出售外卖，顾客可以选择通过邮寄或在商店领取产品。

[![tumblr_inline_ml5dqw7p7j1qz4rgp](img/a01751999b7b29a5e2ce468c88fee648.png)](https://web.archive.org/web/20230315095315/https://techcrunch.com/wp-content/uploads/2013/04/tumblr_inline_ml5dqw7p7j1qz4rgp.jpg)

对于那些不住在五个区的人来说，这似乎是多余的信息，但似乎这家初创公司已经将其实体零售战略扩展到了纽约以外。[正如《波士顿商业杂志》最近在](https://web.archive.org/web/20230315095315/http://www.bizjournals.com/boston/blog/mass_roundup/2013/04/warby-parker-boston-store.html)报道的那样，沃比已经在 Beantown 列出了一个[“商店领导”的职位空缺，寻求“具有以客户为中心、运营出色的零售商的销售管理经验的人……来建立一个由优秀团队成员组成的团队。”他们还雇佣了一名店内配镜师。你不一定要住在贝克街 221B 号才能把这些拼凑起来，阿米利特？](https://web.archive.org/web/20230315095315/http://jobs.warbyparker.com/apply/7MxEKS/Store-Leader-Boston.html)

但正如 Om 在今天的报道中所描述的，Warby Parker 新的线下战略的另一个关键是，该公司希望通过将其商店联网来收集数据，从而将更多的技术带入线下世界。使用 Wi-Fi、传感器等。例如，它希望更好地了解人们的购物方式，以及店内人流、顾客设计和产品偏好方面的趋势。他们越是能够将智能数据收集方法带到线下，他们就能收集到越多的商业情报——特别是当它与在线购物数据的洞察力相结合时，正如 Om 指出的那样。

现在说这家眼镜初创公司处于即将席卷电子商务的线上到线下零售浪潮的尖端可能还为时过早——我还不确定这种情况是否会发生——但这也远非违反直觉。在网上创建一个强大、时髦的品牌，产生品牌认知度和收入，在市场上获得立足点，然后*再*转移到线下。这似乎是一个熟悉的剧本，即使它不是。
