# 诺基亚为低端手机准备新的“Meltemi”操作系统

众所周知，诺基亚相信 Windows Phone 7 能够承载他们未来的智能手机，但他们也没有忘记数百万依靠 Windows Phone 7 生产低端手机的客户。《华尔街日报》的一份新报告显示，诺基亚正在努力开发一款基于 Linux 的操作系统，代号为“Meltemi”

这并不是 Meltemi 的名字第一次流传开来:在今年 4 月宣布[大规模裁员](https://web.archive.org/web/20230204115621/http://www.theregister.co.uk/2011/04/27/nokia_cuts_memo/)后，诺基亚提到 Meltemi 项目将成为该公司一些被解雇的 MeeGo 开发人员的救生船。

与[三星的 Bada](https://web.archive.org/web/20230204115621/https://techcrunch.com/2011/09/20/samsung-to-make-bada-os-open-source-and-part-of-your-smart-tv/) 或新发布的 [Tizen 平台](https://web.archive.org/web/20230204115621/https://techcrunch.com/2011/09/28/samsung-taps-intel-to-build-a-new-linux-based-open-source-platform-tizen/)不同，Meltemi 并不意味着是一个影响将超越手机的包罗万象的操作系统。目前对 Meltemi 项目知之甚少，除了这意味着给他们更基本的设备一个下马威。然而，这并没有阻止投机机器建立联系。

例如，有传言称诺基亚正在开发无处不在的 S40 操作系统的触屏版。如果 Meltemi 被证明是该项目的延伸，它可以为用户提供两个世界的最佳选择:熟悉的外观和有用的新功能，如更强大的应用程序和服务。

虽然其他一些公司已经开始更多地关注他们的高端产品，但诺基亚在很大程度上要归功于他们的功能手机用户大军。在过去的这个季度，这些手机占该公司设备和服务总销售额的近一半。虽然这里总有一种被最新、最炫、最快的设备所吸引的趋势，但我认为现在是时候让所有诺基亚低端手机用户受益于更接近我们智能手机用户多年来享受的体验了。
