# Livefyre 收购 Storify，表示社交监管服务仍将作为独立产品运营

TechCrunch 和其他网站使用的评论平台 Livefyre 宣布，它已经收购了社交策展初创公司 [Storify](https://web.archive.org/web/20221227055304/http://www.storify.com/) 。

交易的财务条款没有披露，但 Livefyre 表示，五人 Storify 团队将加入该公司，使其员工人数达到 91 人。Livefyre 创始人兼首席执行官 Jordan Kretchmer 还通过电子邮件告诉我，Storify 将继续作为一项独立服务运营，stori fy 允许用户将社交媒体评论整合到一次对话中。

“我们不仅会保持 Storify 产品和品牌的完整无损，我们还会在这两方面进行投资，”Kretchmer 说。“这意味着，在我们努力继续发展 Storify 社区的过程中，Storify 将一直是免费版本，就像 Livefyre 一直是免费版本一样。”

与此同时，Storify 的联合创始人伯特·赫尔曼(Burt Herman)表示，使用 Storify 的媒体公司已经在询问这两种服务可以结合使用的方式。他和克雷奇默都指出 [Livefyre 的 StreamHub 产品](https://web.archive.org/web/20221227055304/https://techcrunch.com/2012/09/11/livefyre-streamhub/)(允许出版物自动将社交媒体内容添加到他们的网站)是一个自然的整合点。Kretchmer 表示，通过引入 Storify 更具实践性的监管方法，StreamHub 将成为“唯一一个能够实现编辑创作和自动化监管社交内容的实时平台。”

Storify 是作为记者的工具推出的，在这种背景下可能仍然是最知名的，但该公司一直在为品牌和机构增加更多工具(他们可能比新闻机构有更多的钱可以花)。赫尔曼表示，此次收购将加速这一转变，并帮助 Storify“整合一切”，允许使用 Storify 的品牌与该公司的出版商一起运营原生广告。(Livefyre 最近[推出了自己的广告产品](https://web.archive.org/web/20221227055304/http://blog.livefyre.com/livefyre-launches-social-native-advertising-solutions/)。)

Livefyre 在 2 月份筹集了 1500 万美元，使其总资金超过 2000 万美元。与此同时，Storify 从 Khosla Ventures、Promus Ventures 等公司筹集了约 200 万美元。

收购完成后，赫尔曼(他自己曾是一名记者)表示，他将继续与出版商和记者合作，而联合创始人兼首席执行官泽维尔·达曼(Xavier Damman)将继续负责 Storify 产品开发。
