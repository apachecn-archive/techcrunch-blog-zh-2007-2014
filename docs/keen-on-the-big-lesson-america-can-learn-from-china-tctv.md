# 热衷于…美国可以从中国学到的重要一课(TCTV)

有时候，说话温和的人会扔出最大的炸弹。在本周的科技经济会议上，Intuit 的联合创始人 T2·斯考特·库克的经济和政治分析相当激进。他认为所有公司都需要彻底改变自己，他告诉我，年轻的科技企业家——扎克伯格或肖恩·范宁——最擅长驾驭当今不断动荡的经济水域。在给不愿将权力移交给年轻同事的年长高管的信息中，他提醒我:“最辉煌的事情是在年轻时完成的”。

但正是在他的政治分析中，库克真正走向了核化。他认为“如果你付了花生，你会得到猴子”，并以新加坡或中国的经济增长模式为例，在这种模式下，伟大的(和年老的)领导人能够重塑他们的国家。以中国为例，库克以邓小平在深圳等地建立“经济特区”为例，他表示，这些经济特区让 3 亿中国人摆脱了“极度贫困”。

当然，并不是每个人都同意库克的观点。下周早些时候，我将采访反对苹果的垄断者[迈克·戴西，他可能也会指出中国深圳“实验”的可怕的人力成本。我不得不承认，对于库克对美国采用中国模式还是新加坡模式，我自己也有矛盾心理。毕竟，我们真的想让邓小平或李光耀来决定美国的科技政策吗？](https://web.archive.org/web/20230203210724/http://www.crunchbase.com/person/mike-daisey)
