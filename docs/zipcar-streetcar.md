# Zipcar 以 5000 万美元收购英国汽车共享公司 Streetcar

![](img/a879fda88a2bd5881514d06a211f6134.png)

汽车共享服务公司 [Zipcar](https://web.archive.org/web/20221209020527/http://www.zipcar.com/) 今天宣布，它已经收购了 [Streetcar](https://web.archive.org/web/20221209020527/http://www.streetcar.co.uk/) ，这是一家类似的合资企业，在英国伦敦之外运营。收购的条款没有透露，但是华尔街日报称该交易价值 5000 万美元。

这两家公司现在将合并成一个实体，很快将在 Zipcar 旗下独立运营。

Zipcar 董事长兼首席执行官斯科特·格里菲斯(Scott Griffith)表示，此次收购的短期利益是即时增长，现在超过 40 万名会员将受益于更多的汽车、更多的便利和更先进的技术。Streetcar 成员将可以使用 Zipcar 位于美国和加拿大城市的全球车辆网络。他补充说，从长远来看，这笔交易将有助于 Zipcar 扩张到更多的欧洲市场。

Streetcar 创始人兼高管安德鲁·瓦伦丁(Andrew Valentine)和布雷特·阿克(Brett Akker)在 2003 年从一本美国杂志上看到这个概念后，萌生了创办该公司的想法，他们将保留在 Zipcar 的领导职位。

在未来几个月，这些公司将开始将 Streetcar 品牌、停车地点、技术和运营转移到 Zipcar 品牌和平台，并将就影响他们的发展直接与成员、合作伙伴和其他团体沟通。
