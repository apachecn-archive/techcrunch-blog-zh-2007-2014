# 惠普将出售红帽 Linux Boxen

![hp-compaq-dx2250article-width.jpg](img/57ff37aebb68c66c30d29aa8e2d37614.png)惠普新的 dx2250 台式机和基于 AMD 的工作站将预装 Red Hat Enterprise Linux 5。这款电脑售价约 500 美元，目前在澳大利亚可以买到，因为众所周知，袋鼠生活在那里。

运行 Athlon 64 X2 双核、Althon 64 和 Sempron 处理器的电脑将面向企业，而不是 Vista 夫妇。激动人心？对我们这边的人来说还没有，但很高兴知道我们不用很快支付微软税。
