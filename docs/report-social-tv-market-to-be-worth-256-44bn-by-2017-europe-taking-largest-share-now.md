# 报告:到 2017 年，社交电视市场价值将达到 2564.4 亿美元；欧洲现在占据最大份额

全球社交电视市场已经是一个价值数十亿美元的产业，电视巨头们在第二屏幕观看上投入了大量资金，但根据市场研究公司 MarketsandMarkets 的一份[报告](https://web.archive.org/web/20230117043136/http://www.marketsandmarkets.com/Market-Reports/social-tv-market-815.html)，该市场将在未来五年内以两位数的速度增长。该研究公司预计，该市场将从今年的 1511.4 亿美元增长到 2017 年的 2564.4 亿美元，预计 2012 年至 2017 年的 CAGR 为 11.2%。

“通过整合电视上的社交互动，电视的未来是社交的。广播公司正在开发和丰富社交电视整合；报告指出:“他们瞄准了收看节目的客户、参与度和忠诚度，以提高收视率，他们还发现了社交电视的挑战。”。

目前，欧洲攫取了社交电视市场收入的最大份额。MarketsandMarkets 计算出该地区今年将获得 554.8 亿美元，并预计欧洲的份额到 2017 年将增长到 777.4 亿美元——CAGR 为 7%，略低于整体市场增长率。

除了社交电视初创公司从老牌媒体和科技行业公司吸引大量投资之外，该领域还出现了许多广播公司与 Twitter 和脸书等社交网络公司之间的合作。

“社交电视生态系统中存在各种各样的参与者，因为该行业对实验开放，并且正在见证广播公司和社交网络公司之间的许多合作。一些广播公司正在将社交电视整合到自己的平台中；有许多人[将 Twitter](https://web.archive.org/web/20230117043136/https://techcrunch.com/2012/10/07/twitter-dominates-live-tv-because-social-tv-is-failing/) 整合到他们的社交电视平台中，以增强定制体验和参与度。另一方面，BBC 和 CNN 等行业公司已经与脸书等社交网络公司签署了协议，因为社交网络公司正积极尝试进入这一领域，”报告指出。

“社交媒体真正成为电视和社交媒体的联盟，其中正在开发更新的格式，以提高观众的参与度并鼓励付费交易。许多媒体和科技公司，如赫斯特、时代华纳、英国天空广播公司和谷歌，都在投资巨资支持几家社交电视初创公司。
