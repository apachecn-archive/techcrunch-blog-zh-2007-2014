# Blackboard 买家 Providence Equity 从 PrepMe 收购个性化在线学习平台

数字学习公司 [Ascend Learning](https://web.archive.org/web/20230404075223/http://www.jblearning.com/) 正在收购 [PrepMe 的](https://web.archive.org/web/20230404075223/http://www.prepme.com/)适应性学习平台和 SaaS 虚拟教室。Ascend 实际上由 [Providence Equity Partners](https://web.archive.org/web/20230404075223/http://www.crunchbase.com/financial-organization/providence-equity-partners) 所有，该公司刚刚以 16.4 亿美元收购了 Blackboard。PrepMe 交易的财务条款没有披露。

正如我们[在 PrepMe 的评论](https://web.archive.org/web/20230404075223/https://techcrunch.com/2011/03/28/prepmes-coursification-offers-a-personalized-online-learning-platform/)中所写的，Coursification 是一个基于网络的个性化在线学习课程的应用程序。course 与其他课程管理和在线学习平台的区别在于，它可以帮助教师根据学生的表现和学习进度，为他们提供量身定制的个性化课程。

每个在线课程都以诊断性评估开始，确定学生的学习差距(专门针对该主题)。诊断测试后，每个学生都会收到一份个性化的学习时间表，教师只需将其导入平台即可。课程表是根据学生的熟练程度和课程的持续时间定制的。

当学生在线完成他们的作业时，他们的理解能力通过在线的小测验和长测验来评估。PrepMe 的平台将存储和分析学生与课程之间的每一次互动，并根据学生取得的进步调整学习计划。该平台还提供了与 Coursification 中的教师发送消息和聊天的能力，以及发送文件的能力。类似于 Blackboard 和其他在线学习 SaaS 产品，PrepMe 允许教师上传源材料、测试、测验等。超过 100，000 名学生使用了建立在 PrepMe 平台上的课程，该平台已在国际上用于核心科目和选修课。
