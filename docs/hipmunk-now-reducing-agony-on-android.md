# Hipmunk 现在减少了 Android 上的痛苦

热门航班搜索创业公司 [Hipmunk](https://web.archive.org/web/20230204222729/http://www.hipmunk.com/) 即将登陆 Android。该应用程序将是免费的，你可以在这里安装。

我已经测试了这款应用，它看起来很棒(这很好地提醒了我们，Android 应用*可以*设计巧妙，即使很多都不是……)。它的工作就像你预期的那样:输入你的目的地，出发/到达日期，并从经济舱、商务舱或头等舱中选择。该应用程序将呈现 Hipmunk 的航班网格，您可以按照痛苦程度、价格和时间进行排序。哦，所有载有维持生命的 Wifi 的航班都清楚地标有漂亮的橙色图标。

与二月份发布的 iPhone 应用程序一样，Android 应用程序也有一个缺点:当真正购买机票时，你不能在应用程序中购买。相反，你可以通过 Android 浏览器从运营商的移动网站上购买机票(这真的没那么糟糕)，或者你可以在你的桌面上完成事情——Android 应用程序会给你一个与你选择的航班相关联的密码，你只需要在[finish.hipmunk.com](https://web.archive.org/web/20230204222729/http://finish.hipmunk.com/)输入它(你会得到怀念你的 NES 的额外奖励)。

此外，机器人设计的 Hipmunk 标志非常棒。
