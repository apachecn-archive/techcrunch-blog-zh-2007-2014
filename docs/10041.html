Canonical发布了Ubuntu One的Windows版本

Ubuntu Linux发行版背后的商业支持者Canonical ，几年来一直在托管一项名为 Ubuntu One 的文件同步服务。一个免费帐户可以给你5GB的存储空间，客户端控件已经被嵌入到Ubuntu发行版的最后几个版本中。它的工作方式非常类似于Dropbox或类似的服务，但直到今天，它只适用于Linux。

在昨晚发布的公告中，Canonical透露现在有一个Ubuntu One的Windows客户端，允许你从Linux或者Windows电脑上访问你所有的文件。

我们很早就收到了Ubuntu用户的反馈，关于他们不断发展的需求，即从一个安全的地方跨多个平台和设备管理他们所有的内容。我们已经研究了许多用例，最常见的是使用多个设备或操作系统的Ubuntu用户。许多人不得不在Windows或Mac环境下工作，即使他们更喜欢使用Ubuntu作为他们的家庭桌面或操作系统。另一个例子是提供更多跨平台共享的机会。例如，在一个家庭中使用不同操作系统的家庭可以使用Ubuntu One作为他们存储所有音乐、文档、照片的中心位置，并轻松地与彼此和朋友分享。

Ubuntu One还提供了一个漂亮的音乐流媒体服务。每月支付4美元，你就可以获得20 GB的存储空间，还可以将音乐文件从你的账户传输到你的移动设备上。Ubuntu One应用程序适用于iOS和Android，Android设备获得了可控离线缓存的额外好处，允许您在不需要网络连接的情况下收听您的云存储音乐。很棒的东西。
