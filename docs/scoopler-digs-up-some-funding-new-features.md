# Scoopler 挖掘到一些资金，新功能 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/10/29/scoopler-digs-up-some-funding-new-features/>

实时，实时，实时——这似乎是你现在听到的关于网络的一切。但回到 5 月份，它刚刚成为一种新趋势，看起来即将爆发。走在前列的一家公司是由 T4 支持的实时搜索引擎公司 [Scoopler](https://web.archive.org/web/20221207081246/http://scoopler.com/) 。今天，走在曲线的前面得到了回报，因为该服务刚刚从一些知名投资者那里筹集了一轮种子资金。

当我们[最初写关于服务](https://web.archive.org/web/20221207081246/http://www.beta.techcrunch.com/2009/05/08/search-goes-real-time-with-scoopler-twitter-dominates-results/)(记住，在实时搜索现象的早期)，我们注意到结果的呈现令人印象深刻，但是结果本身完全被 Twitter 所主导。考虑到 Twitter 在这个领域的受欢迎程度，这真的没什么好惊讶的。但这项服务后来增加了一些新功能，使其更加强大。

你会注意到当你进行搜索时，有一个干净的两栏布局。在左侧，会出现受欢迎的共享项目。这些可以是在 Twitter 上分享的链接，Digg 项目等。您还可以通过视频、链接和图像来过滤这些内容。在右边，你会看到网上的实时更新项目。大多数仍然来自 Twitter，但是也有很多来自 Delicious 和 Digg 的搜索结果。自从我们第一次介绍该服务以来，这两列已经交换了位置,“您的搜索”列已被移除并放在顶部栏中。

更重要的是，Scoopler 增加了一个全新的顶部区域，将搜索分成不同的类别。Scoopler 称这一功能为“实时频道”，联合创始人 AJ Asver 将其描述为，“*像 Digg 的分类，但实时更新。他们展示新闻、商业、科技、政治等领域最热门的推文、链接、视频和图片。这个想法是通过建议人们应该搜索什么来驱动更多的搜索查询。*”

有道理。结果相当可靠。例如，我刚刚点击了娱乐频道，我看到直播帖子中充斥着关于迈克尔杰克逊新电影和新阿凡达预告片的讨论。页面顶部还有一个不断更新的区域，向您显示网络上实时谈论的最热门话题。

但也许 Scoopler 最好的功能仍然是“窥视”选项。这让你可以快速看到通过 tweets 和 Digg 链接的内容，而不必离开 Scoopler。这项服务还有小图片缩略图，可以插入到 Flickr 图片流中。

Scoopler 的种子轮已经吸引了投资者，如[罗恩·康韦](https://web.archive.org/web/20221207081246/http://www.crunchbase.com/person/ron-conway)、( Bebo 联合创始人)[迈克尔伯奇](https://web.archive.org/web/20221207081246/http://www.crunchbase.com/person/michael-birch)、[阿瓦隆风险投资](https://web.archive.org/web/20221207081246/http://www.crunchbase.com/financial-organization/avalon-ventures)和 [XG 风险投资](https://web.archive.org/web/20221207081246/http://www.crunchbase.com/financial-organization/xg-ventures)。虽然这轮融资的金额没有正式披露，但我们听说在 50 万美元到 100 万美元之间。

说到实时，不要忘记 11 月 20 日我们的第二次实时分析。
