# Twitter 收购开源培训公司 Marakana，为新的工程师“Twitter 大学”提供动力

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/08/13/twitter-buys-open-source-training-company-markana-to-power-new-twitter-university-for-engineers/>

Twitter 今天[宣布](https://web.archive.org/web/20230327232137/https://blog.twitter.com/2013/twitter-university-building-a-world-class-engineering-organization)其最新的收购，以及提供更丰富的资源来吸引更好的工程人才到公司。它已经收购了开源技术培训公司[马拉卡纳](https://web.archive.org/web/20230327232137/http://marakana.com/)；反过来，马拉卡纳将成为一个名为推特大学的新项目背后的力量。学校吉祥物:一只蓝鸟，不是鲸鱼。

该公司的工程主管克里斯·弗莱在博客中写道:“创始人马尔科和马赛丽·加根塔的创业领导力、对学习的承诺和技术专长给我们留下了深刻的印象。”

此举是几件事情的重要标志:

首先，随着 Twitter 的发展，它已经成为一家更加成熟的公司，远远超过了杂乱无章的初创阶段，它已经采取了几项措施来提供培训，以帮助公司员工继续感受到他们在学习新东西，并在项目中充分利用他们已经知道的东西。Fry 指出，现有的培训项目包括面向工程师的定向课程、iOS Bootcamp、JVM 基础、分布式系统和 Scala School。对 Twitter 来说，重要的是继续发展这类服务，以留住现有的用户。

其次，建立这样的机构，让马拉卡纳掌舵，可以为公司引入一个全新的工程师群体——在大大小小的科技公司继续寻找顶级人才的时候，这不是一件小事，尤其是在硅谷和湾区这样的竞争激烈的市场。事实上，弗莱也指出了这一点:

> “马拉卡纳团队在湾区培养了一个庞大的工程师社区，我们期待着在见面会和技术活动中与你们所有人合作。”

第三，Twitter 多年来为开源项目做出了巨大的贡献(这里的、这里的和这里的都是例子)，所以这也建立在可信度的基础上。

弗莱称赞马拉卡纳的企业家精神。虽然该网站现在实际上已经关闭，并建议过去的用户试用 [NewCircle](https://web.archive.org/web/20230327232137/http://thenewcircle.com/) 以获得类似的培训服务，但看看一个存档的关于页面，我们就能了解该公司的一些背景。它从 2001 年就存在了，为 Java、HTML5、Android 和敏捷开发者运行用户组。它有一个名为 Marakana Spark 的培训管理平台，现在也离线了。

有趣的是，看起来马拉卡纳出现的时候，我们正目睹互联网泡沫破灭后科技世界的又一次复兴(第一次)。仔细想想，正是像马拉卡纳这样的公司在扩大科技行业从业人员方面发挥了作用。

“这一切都始于 Craigslist 上的一则广告，”创始人写道。“旧金山当地的一家非营利组织正在寻找一名技术人员来帮助学习一些 Java 课程。我们喜欢他们的使命:弥合数字鸿沟——帮助贫困青年、少数族裔和妇女进入高科技工作岗位。需求非同寻常。这些课程在宣布后的几个小时内就销售一空。受参与者早期胜利和成功的鼓舞，我们开始向企业和政府提供开源培训。”Twitter 收购 Marakana 时，它正在“美国、加拿大和亚洲”提供开源培训。
