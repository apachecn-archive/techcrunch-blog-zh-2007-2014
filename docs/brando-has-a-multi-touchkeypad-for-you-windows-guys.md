# 白兰度有一个多点触摸+键盘给你们 Windows 的家伙

![usbmu](img/2dbd1ff9718954bdbb24743ad9945a59.png "usbmu")

嗯，看看这个，它是一个组合[多点触摸触摸板](https://web.archive.org/web/20221006120649/http://usb.brando.com/usb-multi-touch-smart-pad_p01106c034d15.html)和一个普通的数字键盘。(几年后，当我们都在使用笔记本电脑，或者，我猜，触摸屏界面时，孩子们会问，“什么是键盘？”)它来自白兰度，如果你对它的制造商有任何疑问的话。

她只运行 Windows XP 和 Vista，并不是说 Mac 用户[会太想念它](https://web.archive.org/web/20221006120649/http://www.apple.com/macbookpro/features.html)。

这是 40 美元，所以，你知道…

说到多点触控，这里有一个著名的 Jeff Han 视频，它已经流传多年了。
