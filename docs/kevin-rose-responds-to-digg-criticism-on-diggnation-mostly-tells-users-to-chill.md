# 凯文·罗斯回应对 Digg 的批评，主要是告诉用户冷静

[Digg](https://web.archive.org/web/20230209002608/http://www.crunchbase.com/company/digg) 创始人[凯文·罗斯](https://web.archive.org/web/20230209002608/http://www.crunchbase.com/person/kevin-rose)围绕[新推出的 Digg 4](https://web.archive.org/web/20230209002608/https://techcrunch.com/2010/08/25/new-digg-launch/) 兴高采烈地回应[山头](https://web.archive.org/web/20230209002608/https://techcrunch.com/2010/08/30/broken-is-repaired/)对[的批评](https://web.archive.org/web/20230209002608/https://techcrunch.com/2010/08/30/why-is-reddit-all-over-digg-right-now/)。他的总主题是用户需要处理它。

Rose 说，他“必须冒险”使用这项服务，寻求将它推向每月 3000 万左右的大众访问者。他希望头条新闻有 20，000 到 30，000 的点击率，而现在大多数头条新闻只有几百个点击率。为了做到这一点，Digg 推出了它认为与你更相关的故事，因为你关注的人和实体也推出了这些故事。大多数人说，这很像 Twitter，Digg 的灵魂已经消失了。

罗斯还说，他不介意人们离开。“如果 Reddit 是你的新家，是你真正喜欢的东西，我完全赞成，”他说。

他还谈到了规模问题，指出 Digg 在其各个数据中心至少有 500 台服务器。他说，所有工程师都专注于使网站稳定，他们没有时间重建旧的流行功能，如即将推出的故事。不过，在那之后，他说《即将到来的故事》很快就会回来。
