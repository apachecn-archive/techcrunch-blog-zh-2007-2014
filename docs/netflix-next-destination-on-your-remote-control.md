# 网飞的下一个目的地:在你的遥控器上

![](img/28fa5fa7fcd77041fd024271bd3604f1.png)

好主意:[网飞](https://web.archive.org/web/20230202231102/http://www.crunchbase.com/company/netflix)和一些消费电子公司[加入了](https://web.archive.org/web/20230202231102/http://www.prnewswire.com/news-releases/streaming-from-netflix-will-soon-be-even-more-convenient-with-netflix-one-click-remotes-introduced-by-major-consumer-electronics-makers-112860669.html)的开发努力，将网飞品牌的一键按钮放在遥控器上，用来操作联网电视、蓝光光盘播放器和其他设备。

这将使从电视上播放网飞更加方便。

网飞今天上午宣布，从今年春天开始，品牌按钮——包括一些带有该公司标志的按钮——计划被放置在遥控器上，用于操作百思买(Best Buy)内部的 Dynex 品牌、松下、三星、夏普、索尼、东芝等公司的某些新蓝光光盘播放器。

夏普、索尼和东芝也将在遥控器上安装网飞一键式按钮，用于选择新的联网电视。Boxee、Iomega 和 Roku 机顶盒的遥控器也将配备网飞一键式遥控器。

在一份声明中，网飞吹嘘说，现在市场上有超过 250 种“网飞就绪”设备。该公司在美国和加拿大拥有 1600 万会员。
