# Telerik 为 Web、混合和本地应用推出新的开发平台 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/28/telerik-launches-new-development-platform-for-web-hybrid-and-native-apps/>

Telerik 长期以来为开发人员提供了广泛的工具，从 Kendo UI jQuery 框架到跨平台开发服务 [Icenium](https://web.archive.org/web/20221007041325/https://beta.techcrunch.com/2012/06/08/like-phonegap-in-the-cloud-icenium-debuts-a-new-cross-platform-mobile-app-development-platform-invites/) 以及众多相关的移动和桌面应用后端服务。今天，该公司将所有这些工具组合在一起，命名为“Telerik 平台”。

新平台允许开发者为 iOS、Android、Windows Phone、Windows 和黑莓创建原生、混合和网络应用。对于开发者想要使用的语言，该服务也是不可知的。它支持。NET，JavaScript/HTML5，Java 和 PHP。

使用 Telerik 的 [AppBuilder](https://web.archive.org/web/20221007041325/http://docs.telerik.com/platform/appbuilder/getting-started/getting-started) ，开发者可以使用该公司自己的基于网络的 IDE 来创建他们的应用程序，或者使用带有 AppBuilder 扩展的 Visual Studio。在线版本具有一套非常强大的工具，包括 iOS 和 Android 的在线模拟器，以及版本控制工具和内置的 git 支持。

基于其现有的工具，该公司辩称，它已经为开发人员创建了一个端到端的解决方案，允许他们重用他们现有的技能、工具和工作流程。Telerik 平台包括对多种 ide、用户界面库和 API 的支持，以及用于创建交互式 UI 原型、调试、自动化测试、部署和实时[分析](https://web.archive.org/web/20221007041325/http://www.telerik.com/analytics)的工具。它还具有许多后端服务，从社交认证服务到推送通知。

该公司表示，完成所有这些工作需要 200 多人花六个月的时间来整合五条不同的产品线。

![image003](img/0498a2aa1dd8684fc17f60b9788efb0b.png)

正如该公司首席战略官史蒂夫·福特(Steve Forte)告诉我的那样，将所有服务整合到一个平台上是该公司合乎逻辑的一步。“在过去几年中，有如此多的新技术可供选择，企业开发人员正在用来自多个堆栈和供应商的技术修补应用程序，”他说。“太多的移动应用程序开发解决方案提供‘千篇一律’、即装即用的应用程序或‘一刀切’的解决方案。”另一方面，Telerik 希望行业从“足够好”的应用程序转向更强大的移动体验。

Forte 说，已经使用该公司服务的开发人员“将会看到大量的协作和生产力融入到平台中。”然而，对于个人开发者来说，变化不会太大。

新平台为那些只想构建基于剑道 UI 的移动应用程序，而不需要 Telerik 将应用程序部署到应用程序商店的用户提供了一个免费的入门层。然而，任何想要使用该平台创建跨平台应用并获得该公司大多数工具的人，都需要订阅付费计划。对于拥有多达 10 个用户的团队，每个用户每月 119 美元起。然而，完整的工具集每个用户每月只需 224 美元。该公司为订阅年度订阅的公司提供大幅折扣。
