# Kabooza 获得 84 万美元启动另一项在线备份服务

![](img/1a49b91f698e690c125a68d0b1c7182a.png)我很怀疑是否还有空间再推出一个，但据 [ArcticStartup](https://web.archive.org/web/20221207210427/http://www.arcticstartup.com/2009/02/03/online-backup-service-kabooza-gets-sek-7-million-investment/) 报道， [Kabooza](https://web.archive.org/web/20221207210427/http://www.kabooza.com/) 是一个相对较新的在线备份服务，刚刚从瑞典风险投资公司 [Aggregate Media](https://web.archive.org/web/20221207210427/http://www.aggregatemedia.com/) 筹集了 700 万瑞典克朗或 84 万美元的种子资金。

Kabooza 的主要卖点是它“非常简单”，这基本上也是许多类似服务的卖点。你下载一个程序到你的电脑上(抱歉，只有 Windows XP 和 Vista 用户，没有 Mac 或 Linux 版本),然后你选择你想备份到 Kabooze 的服务器上的文件夹，然后*瞧*，你就完成了。

也就是说，如果你想信任一家瑞典初创公司，并把你的数据交给他们——请注意，我绝不是在暗示他们会滥用或丢失数据，但消费者信任恰好是许多云服务，尤其是在线备份服务必须处理的问题之一，我发现 Kabooza 网站上的信息远远不足以做出这种判断。

无论如何，Kabooza 提供无限的存储空间和每个用户每月 25 GB 的上传限制。没有免费的试用计划，甚至没有用于测试的，价格定为每年 49.95 美元。
