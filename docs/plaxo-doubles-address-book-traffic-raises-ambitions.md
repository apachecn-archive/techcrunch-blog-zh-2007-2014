# Plaxo 使地址簿流量翻倍，提高了目标 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/04/06/plaxo-doubles-address-book-traffic-raises-ambitions/>

随着我们被越来越多的社交媒体网站所束缚，我们社交网络的联系信息不可避免地变得更加分散。像 [Gist](https://web.archive.org/web/20221209094214/http://www.crunchbase.com/company/gist) 、 [Xobni](https://web.archive.org/web/20221209094214/http://www.crunchbase.com/company/xobni) 和 [Plaxo](https://web.archive.org/web/20221209094214/http://www.plaxo.com/) 这样的数字地址簿和工具正试图组织这种白噪音，但没有人创造出最终的枢纽。

Plaxo 希望成为数字地址簿领域的谷歌，但新任首席执行官贾斯汀·米勒(Justin Miller)知道这将是一个艰难而漫长的过程。

Plaxo 自称为“你的生活地址簿”，它通过 Pulse 服务同步你的地址簿，并从 30 多个网站(如 Twitter、Yelp、Flickr)获取社交数据。据米勒称，该公司正试图设计新产品来增强服务，包括一个“智能搜索”工具，该工具基本上像一个执行助理一样，使用特殊算法全天候梳理网络和 Plaxo 数据库，以保持你的目录最新。该功能将于今年晚些时候发布。该网站目前在 Plaxo 网络中拥有 2000 万成员，托管着大约 5000 万份地址簿——根据营销副总裁 John McCrea 的说法，在线地址簿页面浏览量同比增长了 100%。

放眼未来，Plaxo 有着更大的野心。最终，我们的目标是帮助用户通过多种电子平台访问他们的地址簿，而不仅仅是你的手机和笔记本电脑。McCrea 设想有一天，你将能够跳进你的汽车，打开你的导航设备，说出一个朋友的名字，提示设备从你的云地址簿中提取数据，并指引你到他们的位置。

那当然不是近期的事。目前，该公司仍在为 iPad 开发一款应用程序。

与此同时，Plaxo(2008 年被康卡斯特以大约 1.5 亿至 1.7 亿美元收购)仍在试图与有点争议的过去保持距离。2006 年，Plaxo 与垃圾邮件指控作斗争，2008 年初，Plaxo 因其数据收集技术受到攻击。自收购以来，所有这些都已停止，但该公司承认，在公关方面仍有许多工作要做。

米勒和营销副总裁约翰·麦克雷来到我们的办公室，讨论他们的长期战略，改善他们的形象，并与脸书合作:

以下是文字记录的一些摘录:

关于创建权威智能地址簿的挑战:

米勒:我认为当你回顾始于 2002 年的 Plaxo 的历史时，有一个伟大的想法。也就是说，如果每个人都把他们的信息，他们的联系方式，放在 Plaxo 上，他们让他们的朋友这样做，当任何人更新他们的信息时，我们所有的信息都会被更新，这很好。问题是不是每个人都加入了 Plaxo，也不是每个人都保持他们的信息是最新的。所以我们能够做的，我们现在正在做的是看看我们如何能从网上的任何地方获取信息，为你把这些信息收集在一起。但是，经过这么长时间，我们已经开始寻找不同的机会。几年前，Plaxo 开始越来越多地涉足社交网络，这是一个很好的领域。最近，我们开始关注商业网络，这是另一个很好的领域，但脸书和 LinkedIn 现在在这方面做得很好。最近我们说，你知道吗，有这个需求，这个未解决的，未满足的需求，创造一个地方，人们可以得到他们所有的地址，通讯录，联系信息…在一起。

关于脸书:

McCrea: Scoblegate 发生的时候，我们还没有准备好理解用户拥有他们的数据，通过开放 API 访问这些数据的含义，但现在显然已经从封闭的花园走向开放和可操作性……在很多方面，这是一个技术问题，就像其他任何问题一样。

关于垃圾邮件:

米勒:“对垃圾邮件的认知仍然存在于一些人的头脑中。当我们思考品牌是什么的时候，这是我真正关注的事情。就历史而言，我们经历了几件事，一是垃圾邮件，对垃圾邮件的看法。正如 John 所说，每个人都在这么做，这并不是真正的垃圾邮件，但你在人们的脑海中有这样的看法。第二个是:“哦，Plaxo 是社交网络，不是吗？或者“哦，Plaxo 是一个商业网络，不是吗？“不，我们不是垃圾邮件，我们不是社交网络，我们不是商业网络，我们真的专注于创造最好的，智能的，有社会意识的和无处不在的通讯录。”
