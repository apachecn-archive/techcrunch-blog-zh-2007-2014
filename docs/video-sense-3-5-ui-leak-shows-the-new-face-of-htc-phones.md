# 视频:Sense 3.5 UI 泄露展示 HTC 手机新面孔

眼尖的读者可能已经发现了新泄露的 [HTC Vigor 的后端](https://web.archive.org/web/20230325162420/https://techcrunch.com/2011/08/26/htc-vigor-spotted-in-the-wild-possibly-packing-verizon-lte/)上的“With HTC Sense”徽章，而另一个泄露可能已经揭示了当这东西打开时会发生什么。HTC 更新的 Sense 3.5 UI 的[五分钟视频之旅](https://web.archive.org/web/20230325162420/http://v.youku.com/v_show/id_XMjk5NTEyMjAw.html)刚刚浮出水面，让我们一窥 HTC 下一款大设备为我们准备了什么。

从广义上讲，它看起来非常像在 HTC Sensation 等设备上看到的 Sense 3.0 皮肤。这并不令人惊讶，考虑到版本号的小波动，但 HTC 的 UI 团队似乎已经将大部分工作集中在平滑过渡和添加小细节上。

最大的视觉差异是主屏幕底部的标准三按钮 Sense 启动器栏消失了。它似乎已经被单独的应用程序抽屉和电话按键所取代。小工具也进行了更新，HTC 还尝试了添加和删除主屏幕的功能。

HTC 的 Sense 3.0 看起来很棒，但 3.5 版本中的额外润色让这款设备感觉更……完整。对于大多数设备来说，Sense 可以做的事情往往意味着混乱的主屏幕，但 Sense 3.5 乍一看起来更干净、更开放。除了活力之外，Sense 3.5 用户界面预计也将出现在[臭名昭著的 HTC Bliss](https://web.archive.org/web/20230325162420/https://techcrunch.com/2011/08/10/android-for-girls-htc-bliss-coming-to-verizon/) 上，这是威瑞森的一款女性专用智能手机。

Android 的受欢迎程度很大程度上来自于对其进行无休止定制的能力，但开箱即用的惊人体验肯定有市场。目前还没有消息表明 Sense 3.5 将于何时开始在手机上使用，但希望不会太久。
