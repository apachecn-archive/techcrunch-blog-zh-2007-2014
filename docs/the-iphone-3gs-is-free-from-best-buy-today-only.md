# iPhone 3GS 今天只在百思买免费

如果你是那些不需要最新最好的东西的人之一，那么知道百思买正在赠送 iPhone 3GS 可能会让你兴奋不已。销售将只持续到今天，并且只适用于在& T 与[签订两年新合同或续签合同的客户。](https://web.archive.org/web/20230203084724/https://techcrunch.com/tag/at&t)

iPhone 3GS 在美国电话电报公司的合约价为 50 美元已经有一段时间了，但免费获得 iPhone 确实非常罕见。话说回来，百思买必须为即将上市的另一款非常特别的苹果产品腾出空间。有人能猜到那会是什么吗？

不管怎样，3GS 已经过了它的第二个生日(这基本上是手机年的百年纪念)。有了两年的协议，你将随身携带一部 2009 年制造的手机，大约在 iPhone 7 发布的同时。到那时，谁知道你的 3GS 规格会过时到什么程度。
