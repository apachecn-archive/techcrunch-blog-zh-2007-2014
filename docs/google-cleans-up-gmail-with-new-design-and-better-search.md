# 谷歌用新设计和更好的搜索清理 Gmail

谷歌将在未来几天推出 Gmail 的新设计。它的特点是更多的空白，更少的混乱，线程对话，新的主题和更好的搜索。

信息不像以前那样集中在一起，更容易阅读。加入社交元素后，谷歌会在每条信息旁边添加个人资料图片，标签会更加突出。文本的密度也会根据屏幕的大小和分辨率进行调整，这样对眼睛来说更容易。新的设计在间距和整体感觉方面符合谷歌刚刚对谷歌阅读器做出的一些[改变。](https://web.archive.org/web/20230203063149/https://techcrunch.com/2011/10/31/and-here-it-is-the-new-google-reader-revealed/)

对话比以前更有线索，而不仅仅是集中在一起，所以你实际上可以来回跟踪。Gmail 还用大量来自 iStockPhoto 的背景图片来美化自己。还不清楚会有什么样的 [G+整合](https://web.archive.org/web/20230203063149/https://techcrunch.com/2011/07/11/gmail-plus/)。

但最大的变化之一是搜索 UI。当您开始在电子邮件搜索框中键入内容时，会弹出一个带有高级搜索选项的窗口，帮助您根据收件人、发件人、主题行、日期范围等来细化搜索。我想我还是更喜欢单框，让谷歌来弄清楚我的意思，但是我注意到现在越来越难找到邮件了。(拥有超过 20 万封未读邮件大概也无济于事)。获得那些高级搜索选项而不必点击另一个页面可能会达到目的。我希望如此。
