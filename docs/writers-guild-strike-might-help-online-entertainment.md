# 作家协会罢工可能有助于在线娱乐？

如果你现在还没听说，[好莱坞正在酝酿一场](https://web.archive.org/web/20230131172543/http://tvdecoder.blogs.nytimes.com/2007/11/01/strike-deadline-passes-wga-to-hold-member-meeting-tonight/?hp)罢工。美国编剧工会想要更高的家庭视频剩余报酬，电影和电视制片人联盟没有让步。

工会将于今晚 7:00 在洛杉机开会，进一步讨论罢工将如何进行，但其网站[已经发布了 2007 年的罢工规则](https://web.archive.org/web/20230131172543/http://www.wga.org/subpage_member.aspx?id=2493)。

有一条规则尤其引起了我的兴趣。上面写着:“这些规定禁止为受打击的公司提供与新节目相关的写作服务，这些新节目旨在通过非传统媒体(如互联网和手机)首次观看，并禁止选择或销售用于该目的的文学材料。”基本上，没有为大型媒体公司所有的网站写作。对于你自己创办的或由你所反对的公司之外的人或物所拥有的在线企业，没有任何规定。

路透社对此提出了一个非常有趣的观点。

> “上一次美国编剧工会罢工时，不安分的观众转向了有线电视，使这个类别进入了持续至今的井喷式增长。随着编剧罢工将于周五宣布，数字好莱坞面临的问题是:网络能成为 2007 年的有线电视吗？”

我喜欢这个主意。我非常喜欢它。首先，因为我想放弃我的有线电视提供商(Comcast ),我觉得它向我收取的电视费用远远超过了它的价值，但它提供了足够的互联网服务。第二，因为我内心的理想主义者希望看到 NBC 挣扎到杰夫·朱克下台的地步。我知道，这次罢工不是他的错，但我希望 NBC 继续不理智地把责任推到他身上。

是的，我说了。我不喜欢杰夫·朱克愚蠢的漫谈和他对技术应该如何工作的落后看法，这对消费者几乎没有任何好处，并且冒犯了我这个对这个主题充满热情的人。我听他说过的一件聪明的事情实际上是因为这次罢工而发生的；据路透社报道，他说“罢工可能是一个‘分水岭事件’，会‘驱使更多的人远离黄金时段’”[。听起来不错。那就让火车开动吧。](https://web.archive.org/web/20230131172543/http://today.reuters.com/news/articlenews.aspx?type=technologyNews&storyID=2007-11-02T154417Z_01_N02509124_RTRUKOC_0_US-STRIKE-WEB.xml&pageNumber=1&imageid=&cap=&sz=13&WTModLoc=NewsArt-C1-ArticlePage1)

对你们这些作家协会的人来说。不管我是否同意你罢工的原因，我祝你好运，我希望你考虑为网络创造内容。如果你这样做了，我希望你会意识到，你不需要电影和电视制片人的联盟来把这些内容放到互联网上。去掉那个慢吞吞的，老的，笨拙的，多疑的中间人。这是一个改变你的分销系统的好机会。如果你创造的所有内容都直接放到网上，你会为我们消费者节省很多时间和金钱。

随着罢工，网上娱乐可能会蓬勃发展
