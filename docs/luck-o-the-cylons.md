# 赛昂人的运气 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/03/17/luck-o-the-cylons/>

据 SciFi 频道 [Syfy](https://web.archive.org/web/20221006174423/http://www.crunchgear.com/2009/03/16/what-the-hell-is-a-syfy/) 报道，虽然大多数人今天会喝着蹩脚的绿啤酒，假装他们有一些爱尔兰血统，但[太空堡垒卡拉狄加](https://web.archive.org/web/20221006174423/http://www.scifi.com/battlestar/)的一些演员将前往纽约联合国参加[的一个小组讨论，讨论该剧如何审查诸如“人权、儿童和武装冲突、恐怖主义、人权以及不同文明和信仰之间的和解与对话”等问题。](https://web.archive.org/web/20221006174423/http://featuresblogs.chicagotribune.com/entertainment_tv/2009/03/battlestar-galactica-united-nations.html)

今晚 7 点，在联合国经济及社会理事会会议厅，执行制片人罗纳德·D·摩尔和大卫·艾克，以及明星玛丽·麦克唐纳(劳拉·罗斯林总统)和爱德华·詹姆斯·奥莫斯(威廉·阿达玛上将)将与联合国代表一起参加由乌比·戈德堡主持的小组讨论。讨论的完整记录预计将在不久的将来提供。
