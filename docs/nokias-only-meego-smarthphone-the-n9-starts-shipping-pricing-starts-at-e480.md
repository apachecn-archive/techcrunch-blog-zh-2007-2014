# 诺基亚唯一的 MeeGo 智能手机 N9 开始发货(€起价 480 英镑)

[诺基亚](https://web.archive.org/web/20230203091531/http://crunchbase.com/company/nokia)今晨[宣布](https://web.archive.org/web/20230203091531/http://www.marketwire.com/press-release/nokia-n9-begins-shipping-helsinki-nok1v-1565590.htm)其时尚的新智能手机[N9](https://web.archive.org/web/20230203091531/https://techcrunch.com/2011/06/21/meet-the-nokia-n9-a-colorful-slice-of-meego-magic/)——这几乎肯定是第一款也是唯一一款 [MeeGo](https://web.archive.org/web/20230203091531/https://meego.com/) 手机——已经[开始向预购该设备的客户和零售店发货](https://web.archive.org/web/20230203091531/http://conversations.nokia.com/2011/09/27/nokia-n9-is-heading-to-the-shops/)。

N9 有一个有趣的用户界面，可以通过简单的[滑动](https://web.archive.org/web/20230203091531/http://swipe.nokia.com/)来控制。这款无按钮智能手机具有三个主视图(应用程序、事件和实时应用程序)，旨在使人们能够轻松快捷地浏览界面。期待很快动手。

这款手机有三种颜色(黑色、青色和洋红色)，16GB 和 64GB 存储选项。前者的零售价为 480 欧元(约合 650 美元)，后者的零售价为 560 欧元(约合 755 美元)，不含税项或补贴。

诺基亚表示，这款手机将在世界各国销售。该公司在一份声明中表示，不同地区和运营商的定价和可用性明显不同。
