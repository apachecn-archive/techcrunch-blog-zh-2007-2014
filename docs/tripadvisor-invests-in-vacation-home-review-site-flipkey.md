# 猫途鹰投资度假屋评论网站 FlipKey

[![](img/31c63ff3f11b6ae702dd27843bd73f61.png "flipkey-logo") ](https://web.archive.org/web/20230326022542/http://www.flipkey.com/) [猫途鹰](https://web.archive.org/web/20230326022542/http://www.tripadvisor.com/)心情豪爽。7 月，它收购了两家小型创业公司， [VirtualTourist](https://web.archive.org/web/20230326022542/http://www.virtualtourist.com/) (用户-gen 旅行指南)和[once](https://web.archive.org/web/20230326022542/http://www.onetime.com/)(预订价格比较)。今天它宣布了对 FlipKey 的大部分投资，FlipKey 是一个度假屋租赁的客人评论网站。投资金额没有披露。

FlipKey 去年 3 月才推出。它覆盖了美国的 50，000 个度假租赁物业，首席执行官 TJ Mahoney 表示这代表了 600 亿美元的市场。所以这是一个相当大的利基市场。FlipKey 的目标是成为度假房屋租赁的声誉管理系统。房产所有者可以以小部件的形式获取评论数据，并将其放在他们的网站或房产列表上。(参见示例[此处](https://web.archive.org/web/20230326022542/http://bookings.mermaidcottages.com/site/Overview/PropertyID__27767/page__0/2390/DesktopDefault.aspx)和[此处](https://web.archive.org/web/20230326022542/http://mermaidcottages.com/faqs.html))。

根据 comScore 的数据，猫途鹰计划在自己的网站网络中加入 FlipKey 的评论，该网站在美国每月吸引 1，200 万游客，在全球每月吸引 2，800 万游客。

马奥尼以前是 Compete.com 的联合创始人。Brightcove 首席执行官杰里米·阿莱尔是这家初创公司的顾问。去年 12 月，FlipKey 从天使投资者那里筹集了 50 万美元，其中包括 Allaire、Care.com 首席执行官 Sheila Marcelo 和 Matrix Partners 的风险投资家 Nick Beim。
