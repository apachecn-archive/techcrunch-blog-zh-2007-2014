# YouTube 宣布合作伙伴资助计划，支持 4K 视频分辨率

对 YouTube 新闻来说，这是重要的一周。周三，该公司推出了一个新的基于 HTML5 的移动网站，以及外观漂亮的 YouTube [Leanback](https://web.archive.org/web/20230307214522/https://techcrunch.com/2010/07/07/youtube-leanback-tv/) 。今天，在 VidCon 的一次主题演讲中，该网站又发布了一些公告:YouTube 正在增加对 4K 视频格式视频拍摄的支持——这是一种非常高的分辨率，达到 4096 x 3072)。它还创建了一个新的 500 万美元的合作伙伴赠款计划，旨在通过资助有前途的视频企业来刺激 YouTube 上的原创内容的创作。

在一篇博客[帖子](https://web.archive.org/web/20230307214522/http://youtube-global.blogspot.com/2010/07/investing-in-future-of-video-youtube.html)中，YouTube 写道，资助计划的目标是“通过向处于创新前沿的一小批 YouTube 合作伙伴的制作预算注入额外资金来充当催化剂”，并表示这些资金将作为“合作伙伴未来收入份额的预付款”。

这个程序是如何工作的:

> *   YouTube 正在根据视频浏览量、订户、增长率、观众参与度和制作专业知识等因素来确定合格的合作伙伴
> *   YouTube 会联系选定的合作伙伴，邀请他们提交资助提案
> *   YouTube 会根据信号对提案进行评估，这些信号包括预期业绩、发行计划、营销计划、成本要求以及对广告商的吸引力
> *   如果获得批准，资金将转移到合作伙伴，以便他们可以开始他们的项目

至于 4K 视频分辨率支持，它真的很大:YouTube 说 4K 视频的理想屏幕尺寸是 25 英尺。更别说 4K 分辨率录制所需的摄像头还挺贵的。所以，是的，这可能对你的普通消费者录音有点大材小用，但很明显，它肯定可以有专业的应用。YouTube 还指出，如果你想观看在 4K 拍摄的视频，你需要非常快的宽带连接。

所有这些显然是为了帮助 YouTube 增加网站上原创的高质量内容的数量(换句话说，它正在超越 Roombas 上的 10 分钟猫剪辑，这是它的面包和黄油)。根据 AdAge 的一篇报道，YouTube“设想有人将资金投入到一部短片中，或者提高他们在 YouTube 上的制作水平”。听起来唯一的限制是这些视频的主要发布方式必须是 YouTube。
