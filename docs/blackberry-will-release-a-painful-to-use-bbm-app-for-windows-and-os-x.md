# 黑莓将发布一款痛苦易用的 BBM 桌面应用程序

尽管最近被收购，但黑莓宣布将在本周的 2013 年黑莓亚洲展上发布一款桌面 BBM 应用。不幸的是，这款姗姗来迟的应用还有很多不足之处。BBM 将从基于 Windows 的个人电脑开始，进入台式机领域。由于黑莓的大部分核心客户是企业客户，这些客户花了大量时间在办公桌后。这个版本应该更早发布。但更令人吃惊的是，消息服务的桌面实现糟糕透了。

安装该应用程序后，用户将不得不通过 Wi-Fi 将手机与电脑连接起来。如果你在咖啡店里，想用你的笔记本电脑发送一条快速信息，你首先要找到一种方法把你的手机和电脑连接起来。这是[剧本](https://web.archive.org/web/20230129060222/https://techcrunch.com/2011/04/13/blackberry-playbook-review/)的一派胡言。

给某人发信息应该无处不在，毫不费力。你不必担心你的手机是否还开着，就可以从你的电脑上发送信息。有些应用程序允许您从电脑上发送文本信息。但是 BBM 的观点是它不是短信——它应该更好。此外，BBM 的信息通过黑莓的服务器，就像脸书的信息，iMessages，WhatsApp 的信息等。这些服务不会通过你的手机重新发送你所有的信息。

这就是为什么将 BBM 限制在一次一个设备上是一个应该被修复的严重缺陷。许多人拥有一部手机、一台平板电脑和一台电脑，并希望在这三种设备上使用相同的服务。在目前的实现中，如果没有对 BBM 的恶意攻击，这是不可能的。由于多任务处理的限制，iOS 用户有时甚至需要重新启动应用程序来发送桌面消息——也就是说，如果黑莓能够修复他们的服务器，并实际上[为 iOS 和 Android 推出](https://web.archive.org/web/20230129060222/https://techcrunch.com/2013/09/22/bbm-android-ios-launch/) BBM。

目前还没有发布日期，该公司在发布会上只展示了 Windows 版本。WhatsApp 没有桌面客户端，所以你可能会说 BBM 的发布比什么都没有强。但是桌面 2.0 的 BBM 将不仅仅是一个简单的界面，将你的信息重定向到你的手机。目前，它只是一个空壳。
