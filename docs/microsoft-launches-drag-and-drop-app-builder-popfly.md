# 微软推出拖放式应用生成器 Popfly

微软刚刚在 Web 2.0 大会的舞台上演示了一个叫做 [Popfly](https://web.archive.org/web/20230213205338/http://www.popfly.com/) 的巧妙的 Silverlight 应用程序开发服务，该服务刚刚进入测试阶段。Popfly 让任何人，甚至非程序员，不用写一行代码就能创建网络混搭。都是在浏览器中拖拽(基于 Silverlight，微软对 Flash/Flex 和 Ajax 的回答)。这个演示展示了如何为你所有的脸书朋友建立一个数码相册。它从一个代表脸书的方框开始，在左侧的一个窗格中列出了可以连接到脸书的其他数据集，如另一个服务或 Technorati 排名上的照片。通过简单地拖放代表这些数据集的图标并用线条将它们连接在一起，一个 Silverlight 应用程序就被动态地构建起来了。当然，像任何演示一样，这个有点呆板。但是如果 Popfly 有微软在舞台上展示的一半那么简单，它应该会有很多人接受。
