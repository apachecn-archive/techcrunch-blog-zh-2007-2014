# iPad 真的能让飞行更省油吗？

苹果公司的新任首席执行官蒂姆·库克在今天的 iPhone 发布会上发表了许多关于 T4 苹果公司产品表现如何的强烈言论。但其中一个说法听起来有些牵强。

库克谈到了 iPad 是如何在许多不同的地方使用的，比如学校和飞机。库克说“在驾驶舱里，飞行员正在使用它。他们正在取代装满纸质手册、航海日志、航海图和清单的 40 磅重的飞行包，让飞行员更有效率，让飞机更省油。”为了强调或者怀疑，他补充道:“是的，这是真的。”

飞行员确实在使用 iPads。8 月，TechCrunch 写了一篇关于德尔塔如何开始飞行测试的文章。我们还介绍了一些 [iPad 新导航软件](https://web.archive.org/web/20230203070812/https://techcrunch.com/2011/04/18/efb-for-ipad-will-your-pilot-be-using-this-in-2012/)。这些工具显然使飞行员更有效率。但是，这真的能让飞机更省油吗？

假设飞行员有一个 [1.3 磅的 iPad 2](https://web.archive.org/web/20230203070812/http://www.apple.com/ipad/specs/) ，而不是携带一个 40 磅的飞行包。让我们大方地说，她没有那种智能外套，甚至决定把她的飞行包放在家里。iPad 重量减轻了 38.7 磅，如果是双人双 iPad 座舱，重量减轻了 77 磅。

一架更重的飞机需要更多的燃料来飞行，而更轻的重量确实创造了更好的燃料效率。但是，每个航班的数量很少。波音 757 最大起飞重量[25.5 万磅](https://web.archive.org/web/20230203070812/http://www.boeing.com/commercial/757family/pf/pf_200tech.html)时的 77 磅算什么？

美国联邦航空局表示，没有随身行李的乘客平均体重约为 186 磅。[ [pdf](https://web.archive.org/web/20230203070812/http://rgl.faa.gov/Regulatory_and_Guidance_Library/rgAdvisoryCircular.nsf/list/AC%20120-27D/%24FILE/AC120-27D.pdf) ]儿童的平均体重为 78 磅，与 iPad 节省的体重差不多。

Jet A 燃料每加仑重 6.8 磅。因此，iPad 重量减轻相当于 11.3 加仑的燃料。一架 757 飞机每小时要消耗 900 加仑的燃料，所以重量减轻只能持续 45 秒。

但是有很多航班。这些数字很快就会累加起来。[八月，联合航空公司宣布将向所有联合航空公司和大陆航空公司的飞行员发放 11000 台 iPads。美联航及其新合作伙伴大陆航空公司每天运营超过 21，200 次航班。该公司估计每年将节省 326，000 加仑的喷气燃料。这还不包括那些飞行包里的 1600 万张纸。](https://web.archive.org/web/20230203070812/http://ir.united.com/phoenix.zhtml?c=83680&p=irol-newsArticle&ID=1599253&highlight=)

据航空公司称，iPad 让飞行变得更有效率。是的，是真的。
