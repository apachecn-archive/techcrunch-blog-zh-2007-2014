# 相关磁学研究将最大磁场聚磁体带到你附近的墙上

去年，我们第一次告诉你[相关磁研究公司的可编程磁铁。该公司的专利技术允许他们灵活地控制他们创造的磁铁的磁场形状。](https://web.archive.org/web/20221225124417/https://techcrunch.com/2013/06/12/austin-based-cmr-demos-programmable-magnets-that-changes-polarity-and-strength-on-a-whim/)

从本质上来说，[该公司可以对磁铁的南/北极进行编程，就好像他们正在磁铁上“打印”一样](https://web.archive.org/web/20221225124417/https://techcrunch.com/video/cmr-demos-its-printed-polarity-magnets/517804225/)。它允许许多不同的配置，并有许多工业应用。

但是消费者对这种技术的用途是什么呢？我们在 SXSW 2014 的结尾捕捉到一个演示，展示了这样一个场景——在墙上挂照片。

众所周知，CMR 的专利让他们可以控制北/南磁场模式，但他们也可以控制磁铁的强度，使其能够非常牢固地附着在非常薄的金属上——薄如一层含有金属斑点的油漆。

现在，你可能已经看到了无数 Pinterest 上的儿童托儿所的图片，这些图片覆盖着金属漆，可以让字母磁铁贴在墙上。这并不新鲜，然而，据相关磁性研究公司的商业化和战略副总裁斯蒂芬·斯特劳斯说，典型的消费磁铁不会粘得很好。绝对不能支撑任何质量的物体。

进入 CMR 的 [MaxField Polymagnets](https://web.archive.org/web/20221225124417/http://www.correlatedmagnetics.com/hanging-solutions/) 解决方案，它可以让重达数磅的物体以惊人的强度附着在墙上。

目前在 amazingmagnets.com 有售，该公司计划在未来将这些可编程磁铁带给更多的零售商。
