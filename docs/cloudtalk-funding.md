# CloudTalk 为社交通讯工具筹集了 520 万美元

![](img/f59184e2f49dd9345796a34def98d7a9.png)

[CloudTalk](https://web.archive.org/web/20230202234242/http://pana.ma/html/index.html) 以债务和股权融资相结合的方式筹集了[520 万美元](https://web.archive.org/web/20230202234242/http://www.crunchbase.com/company/cloudtalk)的首轮融资，两份独立的[SEC](https://web.archive.org/web/20230202234242/http://sec.gov/Archives/edgar/data/1510361/000151036111000002/xslFormDX01/primary_doc.xml)文件显示。该公司表示，预计将在本月底完成这轮融资。

或许这家初创公司可以花一些现金收购 cloudtalk.com——其网站目前位于 [pana.ma](https://web.archive.org/web/20230202234242/http://pana.ma/) (其[原名](https://web.archive.org/web/20230202234242/http://www.crunchbase.com/company/pana-ma))。

CloudTalk 被宣传为一种“超越短信和语音邮件”的社交消息服务，从这个意义上说，它旨在通过网络和一系列智能手机实现个人和群体之间的免费语音通信。

老实说，这项服务看起来并不那么令人兴奋，所以看到该公司背后团队的经历是相当令人惊讶的。

这家羽翼未丰的公司的首席执行官是大卫·海登，他是一位连续创业家，也是 1993 年麦哲伦搜索引擎的联合创始人。)，早在 1996 年就卖给了 Excite。

他还是 Critical Path(1999 年上市)的创始董事长兼首席执行官，在 2004 年创建了社交网络公司 JetEye，并在 E*Trade Financial、Riya 和 eHatchery 等公司的董事会任职。
