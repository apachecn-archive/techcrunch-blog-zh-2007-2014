# 安德鲁·梅森希望 Groupon 投资者关注的一张幻灯片

对于 Groupon 首席执行官安德鲁·梅森(Andrew Mason)来说，这是一个艰难的平静期，但随着未来几周他的首次公开募股的临近，该公司正在进行最后的努力，通过在[的首次公开募股路演](https://web.archive.org/web/20230205004641/https://techcrunch.com/2011/10/21/video-tie-wearing-groupon-ceo-andrew-mason-pitches-ipo-to-investors/)(你可以[在线观看](https://web.archive.org/web/20230205004641/http://www.retailroadshow.com/roadshows.asp))。梅森公司在向潜在投资者提供尽可能好的故事方面做得很好:Groupon 正在应对万亿美元的市场，它是具有规模效应的最大的本地商务平台，它正在建立其日常交易优势，以进入相邻的产品和市场(活动、商品、旅游)，它正在建立一个以日常交易、即时移动优惠和忠诚度奖励为中心的本地商务三力。

但在所有的路演幻灯片中，梅森希望投资者关注上面的一张，因为它回应了他的批评，即 Groupon 正在逐渐被遗忘。它显示了 Groupon 营销投资的回报。

Groupon 提交给 SEC 的文件中最大的危险信号之一是它在营销上的巨额支出(根据最新的 S-1 文件，今年前九个月“在订户获取上”花费了 4.665 亿美元)。在第三季度，Groupon 能够将成本控制在可控范围内，从第二季度的 1.01 亿美元运营亏损降至几乎盈亏平衡(第三季度的运营亏损为 23.9 万美元)。

幻灯片显示，2010 年第一季度，Groupon 在营销上花费了 1800 万美元，这使得该公司增加了 370 万新客户。Groupon 随后跟踪了这一批顾客的消费情况。“我们预先支付固定金额，然后在两个季度内收回营销成本，”Mason 解释道。这些客户最终在 Groupon 上重复购买了足够多的商品，为 Groupon 贡献了 1 亿美元的收入和 4300 万美元的“贡献利润”。

如果我是路演的投资者，我会提出的问题是，这种分析是否也适用于后来的客户群体。如果是这样，那么 Groupon 异常高的营销预算将带来同样高的回报。如果回报随着后来的群体而减少，我不会感到惊讶，但这也是 Groupon 现在正在淡化的市场份额抢夺策略，因为它拥有 1.42 亿“订户”，其中只有 1600 万是回头客。Groupon 已经开始致力于让更多的一次性购买者回来。
