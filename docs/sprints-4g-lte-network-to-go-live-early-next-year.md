# Sprint 的 4G LTE 网络将于明年初上线？

Sprint 显然处境艰难。它在获得 iPhone 方面落后了一步，这应该很快得到补救，但黄色运营商也面临着 AT T 和 T-Mobile 之间的大型合并，这将使其远远低于竞争对手。然而，好消息是，看起来 Sprint 已经有了一个计划，其中包括在 2012 年初部署 4G LTE 网络。

据与 [CNET](https://web.archive.org/web/20230203143641/http://news.cnet.com/8301-1035_3-20112095-94/sprint-to-launch-own-4g-lte-network-in-early-2012-scoop/?part=rss&subj=news&tag=2547-1_3-0-20) 交谈的知情人士透露，在我撰写本文时，LTE 设备已经投入使用，工人们正在进行现场测试。目标发布日期是 2012 年第一季度或第二季度，但可能会更早。虽然新的改进网络不需要任何额外的资本投资，但它将在未来三到五年内花费 40 亿到 50 亿美元。

但显然，Sprint 的计划将在未来 7 年内实际节省约 100 亿至 110 亿美元。该运营商预计到 2013 年关闭其 iDEN 网络(为 Nextel ),同时安装可以运行其 3G CDMA 网络和即将推出的 4G LTE 网络的设备。4G LTE 网络基于与威瑞森网络相同的 LTE 变体:FD-LTE。

在所有这些美国电话电报公司/T-Mobile 的乐趣中，Sprint 已经[重整旗鼓对抗](https://web.archive.org/web/20230203143641/https://techcrunch.com/2011/09/06/sprint-files-lawsuit-to-block-attt-mobile-merger/)可能的双头垄断，把自己(谁在乎有多真诚)描绘成美国的失败者。从心理上来说，这无疑让 Sprint 的客户更加忠诚，并诱使竞争对手的客户至少考虑基于原则进行转换。

阻碍人们前进的是这样一个事实，首先，Sprint 没有 iPhone(抱歉，安卓粉丝，但这是事实)，其次，它的网络不像威瑞森或美国电话电报公司那样发达。随着强大的 4G 网络的推出，iPhone 的[发布，以及](https://web.archive.org/web/20230203143641/https://techcrunch.com/2011/08/23/rumor-sprint-to-sell-iphone-5-in-october/)[报道证实不变的无限数据](https://web.archive.org/web/20230203143641/http://www.cnet.com/8301-17918_1-20111753-85/sprint-confirms-it-will-stick-with-unlimited-data-plans/)，Sprint 可能会实现阿巴拉契亚圣与密歇根式的颠覆。
