通过收购Q-go，RightNow增加了一流的自然语言搜索解决方案；将出色的体验从支持页面提升到主页，并帮助消费者组织增加收入

蒙大拿州波兹曼。–(美国商业资讯)——right now(纳斯达克代码:RNOW)今天宣布，该公司已签署最终协议，收购一流的自然语言搜索解决方案提供商Q-go.com公司。该公司还提高了截至2010年12月31日的第四季度的收入和非GAAP收益指导。

指导更新

该公司目前预计2010年第四季度的收入约为5100万美元，经常性收入约为4100万美元，非GAAP每股收益约为0.16美元。这一更新的指导超过了该公司之前在2010年10月28日提供的第四季度收入约为4930万美元，非GAAP每股收益约为0.15美元的指导。

对于2010年第四季度，预期非GAAP每股收益不包括股票薪酬和与递延税资产估值相关的潜在税收优惠。该公司预计，在2010财年的最终结果中，它将确认与递延税资产估值相关的税收优惠。由于这一税收优惠预计很大，但税收优惠的金额目前无法确定，公司将在2011年2月2日RightNow发布截至2010年12月31日的第四季度和全年收益时提供GAAP每股收益和GAAP对账。

RightNow为RightNow CX增加了意图向导，这是同类最佳的自然语言搜索

消费者很容易在一个网站上迷失和沮丧。消费者知道他们在寻找什么，但往往找不到合适的内容，因为他们不了解公司的术语或网站结构。甚至在完成一个搜索请求后，他们经常会看到不相关和令人困惑的结果，这只会增加他们的挫折感。例如，“检查我的现金”和“兑现我的支票”有相似的词，但非常不同的意图和意义。如果没有了解客户真实意图的搜索解决方案，客户往往会浪费时间费力地搜索许多不相关的结果。

随着Q-go的加入，RightNow CX将优质服务从支持页面提升到了主页。利用Q-go自然语言搜索技术，新的RightNow CX意图指南了解网站访问者寻求的内容背后的意图，并立即提供高价值、高度相关的内容，使公司能够:

通过提高在线转化率来增加销售额；
为高价值互动提供优质客户体验；
通过对消费者意图的深入行为洞察来提高忠诚度。
RightNow CX意向指南通过四种互动叠加增强了网站:

问题匹配增强了现有的网站搜索和导航，以了解客户的意图，并提高在线转化率。
虚拟助理提供个性化的自然对话互动，增强客户体验，同时推动转化并降低现场协助成本。
Web Form Assistant通过使用之前捕获的访问者信息来预填Web表单，主动帮助用户完成表单，从而提高在线表单转化率。
情境在线优惠根据页面情境或用户意图提供实时、相关的优惠，显著提高转化率，从而推动销售。
随着Q-go的加入，RightNow还将为RightNow CX云平台增加一个一流的自然语言搜索引擎，将在线搜索从关键词提升到丰富、高度相关的结果集。它包括:

意图匹配:自然语言搜索技术，用于捕获查询、理解意图，并将其与高价值内容和交互进行匹配。例如，“开一个储蓄账户”和“开始一个储蓄账户”是不同的短语，但是具有相同的意图。
特定于行业的语言词典:品牌术语通过特定于行业的词典和语言微调进行定制，以优化自然语言搜索与用户意图的相关性。例如，对于金融机构来说,“银行”这个词总是指金融机构，而不是河岸,“利息”总是指储蓄的回报或一个人为贷款所支付的费用，而不是情感。
语言识别:利用本地化语言包，自然语言搜索会自动识别用户的本地语言，以识别他们的意图，例如“cuenta de ahorros”与“储蓄账户”
RightNow CX意向指南和RightNow自然语言搜索将在收购后立即可用，并将于2011年2月整合到RightNow CX。

关于非GAAP财务指标

非GAAP净收入和稀释后的每股净收入是我们业绩的补充指标，不是GAAP所要求的，也不是根据GAAP提出的。这些非GAAP财务指标不应孤立使用，也不应被视为净收入和每股净收入或根据GAAP确定的任何其他绩效指标的替代品。我们提出非GAAP净收入和每股净收入，因为我们认为它们都是我们业绩的重要补充指标。

管理层使用这些非GAAP财务指标来制定运营决策、评估公司业绩、准备预测和确定薪酬。此外，管理层认为，在规划、预测和分析未来时期时，参考这些非GAAP财务指标来评估公司的业绩，对管理层和投资者都有好处。

我们基于股票的薪酬支出预计会因新发放的奖励数量、我们股票价格的变化、股票市场波动、预期期权期限和无风险回报率而异，所有这些因素都很难估计。在计算非GAAP净收入和每股净收入时，管理层排除了基于股票的薪酬费用，以方便其审查公司在不同时期的经营业绩的可比性，因为在管理层看来，此类费用与公司的持续经营业绩无关。管理层使用其经营业绩的这一视图，以便与其业务计划和个人经营预算及资源分配进行比较。

此外，该公司预计，在2010财年的最终结果中，它将确认与递延税资产估值相关的税收优惠，这将有利于GAAP每股收益。如果公司承认与递延税资产估值相关的税收优惠，此类优惠将从非GAAP净收入和每股净收入中排除，因为管理层认为这是一个非经常性项目，不能反映我们的正常年度有效税率。

管理层进一步认为，这些非GAAP财务指标有助于投资者提高管理层在运营决策中使用的信息的透明度。我们认为，使用非GAAP净收入和每股净收入也有助于将RightNow的基本经营业绩与我们行业中的其他公司进行比较，这些公司使用类似的非GAAP财务指标来补充其GAAP结果。

计算非GAAP净收入和每股净收入作为一种分析工具有局限性，读者不应孤立地考虑这些措施或作为GAAP净收入和GAAP每股净收入的替代品。在未来，我们预计会产生额外的股票薪酬费用，在我们的非GAAP财务指标的陈述中排除这些费用不应被解释为推断这些费用是不寻常的、罕见的或非经常性的。投资者和潜在投资者应注意，使用非GAAP财务指标作为分析工具存在实质性限制，包括:

我们行业内外的其他公司计算非GAAP净收入和每股净收入的方式可能与我们不同，这限制了它们作为比较工具的实用性；并且
公司的所得税费用或收益最终将基于其GAAP应税收入和实际有效税率，这可能与我们非GAAP财务措施中使用的实际税率有很大不同。
此外，我们的GAAP财务指标的调整反映了排除基于股票的补偿费用，这些费用是经常性的，并将在可预见的未来反映在公司的财务结果中。公司通过提供从非GAAP财务指标中排除的GAAP金额的具体信息来弥补这些限制。公司通过更突出地展示可比的GAAP衡量标准，进一步弥补了我们使用非GAAP财务衡量标准的局限性。公司评估非GAAP财务指标以及最直接可比的GAAP财务指标。
