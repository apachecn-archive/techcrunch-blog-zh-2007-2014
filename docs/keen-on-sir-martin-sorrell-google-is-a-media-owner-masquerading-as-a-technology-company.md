# 热衷于……马丁·索罗爵士:“谷歌是一家伪装成科技公司的媒体所有者”

谷歌是媒体所有者还是科技公司？根据全球最大的营销和广告公司[马丁·索罗、](https://web.archive.org/web/20230205041327/http://www.crunchbase.com/person/sir-martin-sorrell)
[WPP](https://web.archive.org/web/20230205041327/http://www.wpp.com/wpp/)的联合创始人兼首席执行官的说法，谷歌是一个“伪装成技术公司的媒体所有者”

直言不讳的索雷尔在每年雅典郊外举行的 WPP 会议上发表年度国情咨文时做出了这一有争议的评论。在马丁爵士演讲结束后，我与他坐在一起，探讨了他所称的全球最大通信公司 WPP 和全球最大科技公司谷歌之间“富有成效”的关系。

那么索雷尔是正确的吗——谷歌真的是媒体所有者而不是科技公司吗？如果是的话，那么谷歌是否应该被要求表现得像一个媒体所有者，在它的技术支持自己内容的方式上更加透明？
