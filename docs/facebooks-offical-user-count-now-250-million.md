# 脸书的官方用户现在有 2.5 亿

![](img/117379c39db5eac651e2b63af9b1f8a9.png)

就在三个月前，脸书宣布其全球用户已经达到了 [2 亿](https://web.archive.org/web/20230130111308/http://techcrunch.com/2009/04/08/zuckerberg-welcomes-200-millionth-facebook-user-wants-to-know-how-it-affected-your-life/)。今天，首席执行官马克·扎克伯格说这个社交网络已经超过了 2.5 亿活跃用户。而在一月份，这个数字只有 1.5 亿。

过去，脸书一直被指责少算用户数。根据 comScore 的数据，今年 5 月，脸书在全球拥有 3.16 亿独立访客，但并不是所有人都有账户。

无论从哪个角度来看，在三个月内增加 5000 万用户都是惊人的。大多数网站会为了这一增长的一小部分而大开杀戒。脸书还需要多久才能达到 10 亿，它能击败 Twitter 达到那个里程碑吗？
