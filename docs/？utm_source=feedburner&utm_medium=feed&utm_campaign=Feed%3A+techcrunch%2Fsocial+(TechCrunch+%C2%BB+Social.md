# Twitter 的下一个重点:简化和统一所有界面

在今天与媒体的会谈中，Twitter 首席执行官迪克·科斯特罗提供了一些有用的信息。有些是在[以统计数据的形式](https://web.archive.org/web/20230203082134/https://techcrunch.com/2011/09/08/twitter-has-100m-monthly-active-users-and-40-of-active-users-dont-tweet/)，有些是在[他们对商业模式的思考](https://web.archive.org/web/20230203082134/https://techcrunch.com/2011/09/08/twitter-advertising/)(注意:偶尔会有广告在流中)。但他也透露了一些可能被忽视但很重要的东西:Twitter 各种界面即将统一。

事实上，几个星期以来，我一直听到关于这件事的传闻。就在杰克·多西[一举赶走几个产品经理](https://web.archive.org/web/20230203082134/https://techcrunch.com/2011/07/20/twitter-cleaning-house-product/)的时候，我听说这个[回归简单](https://web.archive.org/web/20230203082134/https://techcrunch.com/2011/07/18/dont-drink-and-power-use/)要成为焦点了。其中一部分意味着创造#newnewtwitter 来取代#newtwitter。

在谈到今天的“核心产品”时，科斯特洛指出前进的第一步是“简化界面”。“我们希望 Twitter 在不同客户之间非常简单——你会看到界面变得更加一致，”他说。据我所知，这意味着界面更像移动客户端，而不像新的 Twitter 和 iPad 应用程序的多窗格视图。这很有趣，因为 iPad 应用程序影响了 web 应用程序。现在情况可能会逆转。

我听说，Twitter 希望有一种方法可以再次拥有一个漂亮、干净的单一推文流。更高级的功能将以隐藏但简单的方式处理。最重要的是，该计划旨在统一网络、手机、平板电脑等的控制和功能。一条推特就能统治所有人。

你可以通过访问 Twitter 最新的网络应用程序来一窥这种想法。所有这些都使得抵制第三方 Twitter 客户端变得更有意义。Twitter 希望并且需要控制用户体验。

与此同时，Twitter 也在关注一种方法，将用户可能认为重要，但可能与社交图不匹配的信息呈现出来。科斯特洛指出，在最近的阿拉伯之春起义期间，人们来到 Twitter 上寻找信息，但必须通过标签来这样做，这对大多数人来说很麻烦。“我们想设计一个界面，既能捕捉人群的吼声，也能捕捉音量，”他说。

值得注意的是，40%的 Twitter“活跃”用户实际上并不发微博。相反，他们潜伏和浏览。科斯特洛觉得简化界面将有助于用户这样做，可能有办法通过起初的转发和回复来让他们更投入，而不是显式地发推。

没有消息说我们什么时候能看到#newnewtwitter，但我想这是迟早的事。
