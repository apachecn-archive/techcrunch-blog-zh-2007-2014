# 在移动广告中，尺寸重要吗？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/12/15/in-mobile-advertising-does-size-matter/>

重要的是海洋中的运动，对吗？不对。好吧，至少移动广告看起来是这样。(小心， [inneractive](https://web.archive.org/web/20221005185004/http://inner-active.com/) 下图信息图！)[如果你相信谷歌的话](https://web.archive.org/web/20221005185004/http://googlemobileads.blogspot.com/2011/12/engaging-consumers-on-tablets-new-ad.html)，那么 2011 年是平板电脑之年。或者用他们的话说，今年平板电脑“成为主流”。当然，谷歌*会*说，作为一个快速传播的移动操作系统的创造者，对吗？嗯，[到今年年底，预计售出的平板电脑总数将接近 6000 万台](https://web.archive.org/web/20221005185004/http://www.appleinsider.com/articles/11/11/21/combined_mac_ipad_sales_to_make_apple_top_global_pc_vendor_in_2012.html)，我认为我们可以假定他们是无辜的。

当然，对于考虑移动平台的广告商和开发商来说，收入将来自哪里——以及有多少——是至关重要的问题。就移动应用程序而言，免费增值肯定会继续存在，但正如你所听到的，移动广告正在迅猛增长，[2011 年，美国广告商预计将在移动广告上花费超过 12.3 亿美元](https://web.archive.org/web/20221005185004/https://beta.techcrunch.com/2011/11/14/mobile-games-may-be-taking-off-but-how-are-their-creators-going-to-make-money-infographic/)，到 2014 年，这一数字预计将增加到 44 亿美元左右。

或许更重要的是，数据显示，近 80%的消费者对移动广告感到满意或保持中立。当然，随着智能手机和平板电脑的采用继续以创纪录的速度增长，为了保持移动广告的舒适度，谷歌的研究表明，用户越来越期待在他们的移动设备上出现更多互动和触摸友好的广告。这就是为什么，[上周，谷歌推出了新的平板电脑专用广告格式](https://web.archive.org/web/20221005185004/http://googlemobileads.blogspot.com/2011/12/engaging-consumers-on-tablets-new-ad.html)，这种格式“使得广告商可以利用现有资产(照片、标识等)轻松地为平板电脑制作富媒体广告。)”并在移动平台上发布。

是的，这很好，但广告商和移动开发者想知道哪些平台、操作系统和设备的平均点击率最高，因为高点击率会带来货币化，而货币化会带来快乐的开发者、更好的广告，并有望带来更快乐的消费者。[正如我们在 Sarah 最近的报道](https://web.archive.org/web/20221005185004/https://beta.techcrunch.com/2011/07/22/android-vs-ios-the-heated-battle-over-mobile-advertising-infographic/)中看到的，苹果在移动广告表现上已经超过了安卓，但是 Windows 在第三季度的移动广告表现上仍然领先。随着 Windows 的领先，虽然芒果“推出时没有承诺的应用内购买支持”，但这一切都是为了广告盈利——尤其是对初露头角的 Windows 开发者而言。

因此，随着点击率对寻求在移动平台上赚钱的开发者变得越来越重要，一个有趣的补充问题变成了:当谈到移动广告的表现(点击率)时，设备的大小重要吗？或者，换句话说，开发者应该因为这个(或其他)原因将他们的注意力转向平板电脑吗？

谷歌认为，观众肯定是冲着平板电脑来的，他们(以及其他人)正试图通过在所有这些平板设备上开发优化的广告格式来实现这一目标。为了支持这一点， [inneractive](https://web.archive.org/web/20221005185004/http://inner-active.com/) ，一家移动应用货币化交易所，已经创建了一个信息图表来说明大设备的移动点击率是否更高。他们的研究表明，广告商(和开发商)在为更大格式提供优化广告时具有明显的优势。

因此，就点击率而言，尺寸似乎确实很重要——我们再次确认平板电脑将在 2012 年引领潮流。在很多方面。
