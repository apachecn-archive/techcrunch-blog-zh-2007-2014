# 脸书将在明天与诺基亚的位置游戏中再进一步

![untitled21](img/db8e4152429424daedab43a2d7eef641.png "untitled21")

关于脸书的选址计划，现在已经有不少传言了。随着竞争对手 Twitter 最近宣布了其地理定位 API ，更大的网络面临着交付某些东西的压力。我们了解到，明天将朝着这个方向迈出一步，因为诺基亚将在其诺基亚世界活动上宣布一项新的服务，该服务将利用脸书境内的位置。

现在，要澄清的是，这不是脸书正式进入游戏本身，但它足够大，脸书的移动总监亨利·莫伊西纳克，显然会利用他在诺基亚世界的主题演讲来揭开这一点，我们听说。

这款应用看起来是 2008 年夏天诺基亚收购 T4 基于位置的社交网络 Plazes 的直接结果。正如你在下面的截图中看到的，一张 [Ovi 地图](https://web.archive.org/web/20221006200257/http://maps.ovi.com/services/integrated)(诺基亚的地图财产)将驻留在脸书境内，并显示你所在的位置。它还可以根据你的位置更新你在脸书的状态，并在这些地图上提供链接。

![untitled1](img/2e783e70bdc0d8aadcd513aa40276cd0.png "untitled1")

其他服务，如雅虎的 Fire Eagle 和 Whrrl 都有应用程序来设置你在脸书的位置，但这些程序使用起来仍然很麻烦，而且还没有在脸书流行起来。据推测，这一诺基亚公告也将被捆绑到其移动设备上，这可以让它立即接触到数百万用户。

有趣的是，这个诺基亚/脸书的概念似乎也适用于“签到”地点，这也是 Foursquare(以及之前的 Plazes)等服务用来定位的地方。这与谷歌纬度或 Loopt 不同，它们只是跟踪你的位置。

在某个时候，脸书将不得不离开板凳，把它相当大的重量投入到基于位置的服务领域本身。这是肯定会发生的，我们已经从很多渠道听说了，但问题是，什么时候？就目前而言，合作将是必须的。
