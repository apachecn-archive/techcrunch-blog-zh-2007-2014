# 下一代 MacBook Pros 将采用 Air DNA？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/26/next-macbook-pros-to-feature-air-dna/>

Sandy Bridge [对 Air line 的更新](https://web.archive.org/web/20230203091731/https://techcrunch.com/2011/07/20/updated-macbook-air/)已经足以让一些人觉得轻量级笔记本电脑已经准备好迎接黄金时代了(我个人确信)，但对一些人来说仍然不够。MacBook Pro 用户习惯于更大的存储空间、更多的屏幕空间和更多的端口。如果谣言是可信的，两个世界的最好的可能正在路上，空气风格的设计正在走向 Pro 系列。

消息来源模糊不清，只是拐弯抹角地提到( [MacRumors](https://web.archive.org/web/20230203091731/http://www.macrumors.com/2011/07/26/apple-finishing-up-work-on-an-ultra-thin-15-notebook/) “了解到了” [TUAW](https://web.archive.org/web/20230203091731/http://www.tuaw.com/2011/07/26/rumor-apples-next-15-laptop-refresh-will-be-air-like/) 是“听到了”)，所以对此要有所保留。但即使没有源代码，它也是有意义的，同时也为大量的猜测留下了空间。妥协会是什么？

首先，光驱几乎肯定会被淘汰。对苹果来说幸运的是，这种特殊的东西在他们的用户中并不是优先考虑的。iTunes 和 Mac 应用商店很受欢迎，苹果商店本身也越来越少盒装软件。

此外， [Thunderbolt](https://web.archive.org/web/20230203091731/https://techcrunch.com/tag/thunderbolt/) 提供了一种极其简单的方式来添加高速外设。没有光驱？没问题:100 美元的外置硬盘，与内置硬盘相比，运行速度丝毫不减。

存储有点复杂。尽管流媒体解决方案很受欢迎，但本地存储对于编辑媒体仍然非常重要，苹果一直在通过 iLife 套件向消费者大力推销这一点——尽管专业人士可能会在 Final Cut X 的糟糕接收后跳槽。256GB 的闪存存储很好，但人们想要 TB。然而，2.5 英寸笔记本电脑驱动器仍然太厚，无法包含在类似空气的机身中。或者他们是？一些 1tb 容量的笔记本电脑硬盘厚度不到 10 毫米。你不能把它放在空气的尖端，但在右上角可能会有一个 2.5 英寸的空间，光驱将放在那里。苹果很乐意[定制 PCB](https://web.archive.org/web/20230203091731/https://techcrunch.com/2011/07/21/teardown-of-new-13-macbook-air-finds-almost-no-differences-from-old-model/)优化空间。

![](img/75cca4b89769258f68288062f987fdb0.png "Seagate-Momentus-Hard-Drive")

不过，只有旋转的硬盘才会影响性能。因此，我猜他们将配备混合 SSD-HDD 卷，系统，应用程序和临时保存在 SSD 上，大容量介质保存在 HDD 上。像这样分离数据并不困难，因为苹果不可能只承诺一种或另一种存储类型，他们必须对两者都做一些有趣的事情。

别忘了港口。苹果将会推出 Thunderbolt，虽然只配备那个端口可能是他们的白日梦，但我猜测他们会配备两个 Thunderbolt 和两个 USB 端口。2.0 还是 3.0？我不认为苹果会为了让 Thunderbolt 配件更有吸引力而限制他们的 USB 功能。正如其他人指出的那样，Thunderbolt 连接的 I/O 集线器将轻松处理 USB 3.0 速度，尽管[新影院显示器](https://web.archive.org/web/20230203091731/https://techcrunch.com/2011/07/20/the-new-apple-thunderbolt-display-is-an-ips-monitor-and-thunderbolt-hub/)上的端口实际上只有 2.0。

假日发布(如 TUAW 所建议的)将阻止他们添加下一代处理器(将于 2012 年第一季度推出)，但为了补充 Air 的风格，他们可能还想推广长电池续航时间，从旧的 Core i7s 切换到 Sandy Bridge 可能足以满足苹果的目的。想要最新的硬件？买一台电脑(我买了)。

所以:没有光驱，混合 SSD/HDD 存储，2 个 TB，2 个 USB 端口(当然还有 SD 插槽和以太网)。也许 17 英寸的更好，但我不认为这些规格听起来不太可能。
