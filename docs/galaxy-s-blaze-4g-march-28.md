# 3 月 28 日，Galaxy S Blaze 4G 点亮 T-Mobile 货架

三星 Galaxy S Blaze 4G 的消息第一次传出是在 TechCrunch 团队在 CES 的拉斯维加斯会议中心的大厅里漫游的时候，但当时 T-Mobile 对发布日期不太感兴趣。

不过，在这中间的几个月里，这家充满活力的运营商似乎开放了一些，因为他们今天早些时候在 Twitter 上宣布，他们最新的三星手机将于 3 月 28 日登陆他们的销售渠道。如果你真的很幸运(或者愿意跳上车碰碰运气)，你甚至可以更早买到 Blaze 4G，因为一些商店将提前一周出售这款设备。

你不记得 Galaxy 的 Blaze 4G 是情有可原的——在 CES 和 MWC 展出的所有漂亮的新设备中，像 Blaze 这样的中端 T-Mobile 手机可以很容易地穿过裂缝。尽管如此，虽然 Galaxy S III 不是这个东西，但人们肯定会比看到一个配备 1.5GHz 双核骁龙 S3 处理器、3.97 英寸 Super AMOLED 显示屏和支持 T-Mobile 42 Mbps HSPA+网络的设备更糟糕。

不管是好是坏，你仍将不得不应对 Android 2.3.6 姜饼和三星标志性的 TouchWiz UI，但嘿，市场似乎还没有在 Android 4.0 设备中游泳。Blaze 4G 的售价为 150 美元(扣除 50 美元的邮寄折扣后),正好处于 T-Mobile 系列的中间位置，虽然它看起来是任何人升级的可靠选择，但在这样的手机上扣动扳机仍然需要一些考虑。

多花几块钱就能买到一台真正的 Galaxy S II(甚至是白色的！)，而少花几美元就能生产出一些像 HTC Sensation 这样的过时硬件。对于潜在的手机消费者来说，这永远是一个艰难的决定——他们应该现在就购买中档手机，还是等待高端手机价格下降？
