英国微处理器公司Imagination Technologies以8000万美元的高价收购芯片设计商MIPS的运营业务“某些”专利

英国微处理器公司 Imagination Technologies 提高了之前收购芯片设计商 MIPS Technologies 运营业务和一系列与 MIPS架构相关的专利，以及其他专利的免版税许可权——从上个月的6000万美元提高到现在的8000万美元。当竞争对手半导体集团 CEVA 提出7500万美元现金收购相同的MIPS资产，出价高于Imagination的出价时，它的早期收购遇到了阻碍。

在今天发布的一份新闻稿中，Imagination Technologies Group PLC表示，它已经签署了一份“修订协议，以收购MIPS Technologies的运营业务和某些专利财产，以及所有剩余专利财产的许可权”。Imagination提出了8000万美元的现金报价——“将从现有的现金资源中融资”。

Imagination Technologies指出，其对MIPS的修改报价是为了回应CEVA的报价，并补充说，此次收购的所有其他实质性条款和条件仍与11月6日的报价相同。

苹果公司是该公司的股东之一，该公司表示，它仍然预计交易将在2013年第一季度完成。
