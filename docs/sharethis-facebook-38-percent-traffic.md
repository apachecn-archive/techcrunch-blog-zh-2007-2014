# 分享这项研究:脸书占了网络共享流量的 38%

![](img/8093503f51cd0ce5a7270956350b7138.png)

分享在网络上很重要。我们都知道。但是到底有多大呢？[分享这个](https://web.archive.org/web/20230203033203/http://sharethis.com/)在与 [Starcom MediaVest Group](https://web.archive.org/web/20230203033203/http://www.smvgroup.com/) 和 [Rubinson Partners](https://web.archive.org/web/20230203033203/http://blog.joelrubinson.net/about-rubinson-partners/) 共同进行的一项研究中找到了一些答案。纵观每月超过 3 亿人的分享和点击习惯，他们通过“分享”按钮在超过 100 万个网站上传递链接(每月产生 70 亿次页面浏览量)，有几件事很突出。

总体而言，现在分享产生了大约 10%的互联网流量和 31%的来自搜索和社交网站的推荐流量。搜索仍然是两倍大。

谈到网络分享，脸书说了算。脸书占所有分享推荐流量的 38%。电子邮件和推特并列第二，各占 17%。( ***更正**:分享这提供了一张编号错误的幻灯片，现已换出。Twitter 实际上是 11%，电子邮件是 17%。这些是实际点击率。原始共享数字更高。脸书占所有共享内容的 56%(高于 2010 年 8 月的 45%)，其次是电子邮件占 15%(低于 34%)和推特占 8%(低于 12%)。这两组数字的区别在于，有些内容是共享的，从来没有被点击过，因此原始数字更高。*

脸书的份额可能在增加，但 Twitter 在实际点击量方面保持不变。平均而言，Twitter 链接被点击 4.9 次，相比之下，脸书链接被点击 4.3 次，电子邮件链接被点击 1.7 次。

![](img/2a62f72c8f8f885d32ecd53705e96044.png)

但是分享并不像大多数人想的那样像病毒一样传播。链接不太可能在最初的共享者之外被点击。换句话说，如果你直接分享一个链接给我，我认识你，我很可能会点击它。但是如果我把这个链接传给离你有一两次距离的人，他们点击这个链接的机会就会大大降低。

说到分享，80%的人只分享一个类别的链接，超过 70%的人只会点击一个类别，无论是商业、政治还是娱乐。脸书在分享娱乐甚至购物链接方面尤其强大，而电子邮件和 Twitter 似乎在商业或健康方面取得了一些进展。
