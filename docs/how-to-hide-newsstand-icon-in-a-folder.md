# 如何隐藏 iOS 5 恼人的报摊图标

所以，你已经[升级到 iOS 5](https://web.archive.org/web/20230203091529/https://techcrunch.com/2011/10/12/apples-ios-5-update-now-available-for-iphone-ipad-and-ipod-touch/) 了。看看这些新的闪亮的东西！iMessage！通知中心！报摊！

等等，报摊？一个专门存放数字杂志和报纸的文件夹？咩！让我们继续删除这个…哦，你不能。没关系，我们会把它放到另一个文件夹里……*不*，你也不能这么做。

嗯，你*可以*——你只是不*应该*去。互联网已经找到了一种方法。

诀窍(正如第一次提到的[编码大屠杀](https://web.archive.org/web/20230203091529/http://thecodingmassacre.wordpress.com/2011/10/13/newsstand-folder-no-jailbreak-ios-5/))很简单(不需要越狱！)，但需要一些快如闪电的手指。很明显是某种错误，在创建一个文件夹后有大约半秒钟的窗口，其中没有设置自动拒绝报摊。

## 以下是如何做到这一点:

*   将一个非报刊亭应用程序拖到另一个非报刊亭应用程序上。这将创建一个文件夹。
*   在创建文件夹和播放小动画时，*立即*将报刊亭应用程序拖到这个新创建的文件夹中。(这有助于让您正在创建文件夹的应用程序靠近报刊亭应用程序所在的位置。诀窍是在新文件夹自动在屏幕上展开之前*把它放进去)*
*   嘭！你有了一个新文件夹，里面有完整的报刊亭应用程序。这么大的活动空间！

因为这是一个 bug，所以这个技巧有一两个陷阱:报刊亭不仅在塞进另一个文件夹时变得完全不起作用，而且如果你试图打开它，它实际上会让主屏幕崩溃并迅速重置。不过，这总比在你的主屏幕上有一个你永远不会用到的图标要好。

此外，你可能不应该期望这种伎俩无限期地工作。考虑到破坏主屏幕的副作用，苹果无疑会在不久后修复它——但希望他们会用一些不那么粗糙的东西来取代它。
