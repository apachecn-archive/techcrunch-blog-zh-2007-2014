# AppleCare+现已覆盖所有提供该服务的国家/地区的旅行者的 iPhones、iPads 和 ipod TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/09/24/applecare-now-covers-travelers-iphones-ipads-other-products-in-any-country-where-its-offered/amp/>

苹果公司最近对其 AppleCare+计划进行了[变更](https://web.archive.org/web/20230122045335/http://www.apple.com/support/products/faqs.html)，大约在新 iPhones 发布的时候。现在，您可以在任何提供 AppleCare+的国家/地区维修您的 iPhone、iPad 或 ipod，而不仅仅是在您的家乡。

这一变化意味着，如果您带着您的设备旅行，如果您在国内购买了 AppleCare+,您可以在另一个国家获得服务。以前，如果您想获得承保，您必须在购买该计划的地方接受该服务。请注意，[并非所有更换或维修服务](https://web.archive.org/web/20230122045335/https://twitter.com/SonnyDickson/status/382634673681092608)在每个国家都可用，因为许多国家没有特定 iPhone 型号的库存。例如，CDMA iPhone 5 在巴西无法买到，也无法在那里更换。

如果在这些地区有管理 AppleCare+的其他规则，这些规则也将适用。但是，在大多数情况下，旅行者现在将获得更多的维修和更换选择。

这些计划现在允许您为您的 iPhone、iPod 和 iPad 提供保险的国家包括奥地利、加拿大、中国、法国、德国、香港、爱尔兰、意大利、日本、荷兰、新加坡、西班牙、瑞士、英国和美国。苹果最近[扩大了 AppleCare+提供服务的国家](https://web.archive.org/web/20230122045335/https://techcrunch.com/2013/09/10/apple-raises-applecare-service-fees-launches-in-europe-and-adds-ipods/)，并将服务费从每次“事件”39 美元左右提高到 79 美元。
