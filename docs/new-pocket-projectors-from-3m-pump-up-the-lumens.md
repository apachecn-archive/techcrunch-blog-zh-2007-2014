# 3M 公司的新型袖珍投影仪提高了流明

CES 仅仅是这些小商品的盛会:纯粹而简单的小玩意。微型投影仪(或迷你、微型、口袋或你有什么投影仪)是少数几个我真正喜欢看到不断改进的小工具之一，我期待着每年的 CES 都有这些改进。今年没有失望:我们昨晚看到了奥图码的新设备，微视明天也将推出新设备，3M 也刚刚推出了他们的产品。

有两款新投影仪:MP220，热狗面包形状的微型投影仪系列的下一款，以及 boxier MP410。

![](img/c72224352d76ad3898c1a5edf2e056a5.png "2012_1_PP_MP220_32 (2)")

MP220 的亮度为 50 流明，是我上次评测的 MP160 的两倍，与 Optoma 和其他公司的新品具有竞争力。它有 1024×600 的分辨率，经过修改的设计(我认为是明智的修改)，他们声称它将持续两个小时。价格也有所上涨:令人望而生畏的 479 美元。在这一点上，你开始遇到“真正的”投影仪作为竞争对手，至少在价格上。

![](img/0df55d9089ecf4c2a3f2a099a58b3a2f.png "2012_1_MP410_prod05_rgb (2)")

然后是 MP410，一个更稳定的解决方案，但仍然是电池供电。它明显更大——4”x4”x1 ”,也更重(不到一磅),但它在 1280×768 的分辨率下输出 300 流明，足以播放你的高清游戏和电影，尽管我不相信它的在线解码器能完成这项工作——最好是插入你的笔记本电脑或 iPod。它有 VGA 和 HDMI 输入，售价 699 美元。

我对这些东西的唯一疑问是，它们是否会遇到和我刚刚评论的 AAXA P4 一样的问题:噪音。如此明亮的灯泡和重型加工，这些东西很快就会变热，必须高速旋转风扇。希望我们将有机会检查这些，所以请关注我们在未来几个月的审查。
