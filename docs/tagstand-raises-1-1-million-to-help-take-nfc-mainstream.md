# Tagstand 筹集 110 万美元帮助 NFC 成为主流 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/10/14/tagstand-raises-1-1-million-to-help-take-nfc-mainstream/>

新的 iPhone 可能没有，但下一代可能会有——而且在 Android 和其他智能手机平台上变得越来越常见。

是的，近场通信(简称 NFC)终于开始进入消费者手中。一家由 YC 支持的名为 [Tagstand](https://web.archive.org/web/20230205001133/http://www.tagstand.com/) 的公司正在寻求帮助开发者利用这项技术，为他们(和品牌)提供他们需要的所有工具和特殊的 NFC 贴纸。

今天，这家 YC 支持的公司，我们在 8 月份第一次[报道过](https://web.archive.org/web/20230205001133/https://techcrunch.com/2011/08/19/yc-funded-tagstand-greases-the-wheels-of-nfc-development/)，宣布它已经筹集了 110 万美元的资金，其中包括许多著名的天使投资人。完整名单:尤里·米尔纳，SV Angel，Naval Ravikant，Paul Buchheit，Yael Shazeer，Christina Brodbeck， [Anand Agarawala](https://web.archive.org/web/20230205001133/http://www.crunchbase.com/person/anand-agarawala) ，Mike Berolzheimer，Bee Partners，Quotidian Ventures，TEEC(中国天使网络)，Vaizra Investments(以色列基金)，迪恩·史密斯，Christopher Morton，Anand Swaminathan。

在这一点上，Tagstand 的目录包括各种支持 NFC 的贴纸，用于不同的使用情况(户外贴纸，可以消除干扰的贴纸，以便它们可以在金属表面上工作，等等)。)和 NFC 读取器/写入器。一旦进入现场，这些贴纸可以通过基于网络的控制面板进行管理。

联合创始人 Kulveer Taggar 表示，该公司每月收入已经达到五位数，在零售和广告领域，所谓的“智能海报”增长尤为迅猛。目前，市场上有 20，000 种不同的贴纸，而且这个数字还在以每月 50%的速度增长。或许更重要的是，这些标签的参与度每月增长三倍(换句话说，越来越多的人在使用 NFC)。

这种增长大部分是在国际上，NFC 在国际上有更强的存在——例如，Tagstand 与 Adshel 合作，为一家名为 Coles 的连锁超市发起了澳大利亚最大的 NFC 活动，该超市在墨尔本有 50 个站点。Taggar 表示，美国在 NFC 部署方面正开始加速。

更广泛地说，Taggar 表示，越来越多的支持 NFC 的手机(你可以在这里看到一个详尽的列表)正在促使开发者的兴趣普遍上升。该公司三分之一的 NFC 订单来自企业，三分之二来自对该技术感兴趣的工程师和爱好者。
