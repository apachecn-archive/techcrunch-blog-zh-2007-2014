# Twitter 收购列表共享和发现平台 Bagcheck，Sam Pullara 加入工程团队

Twitter 已经收购了分享和发现平台 [Bagcheck](https://web.archive.org/web/20230204233326/http://bagcheck.com/) 。收购条款没有披露，联合创始人[萨姆·普拉拉](https://web.archive.org/web/20230204233326/http://www.crunchbase.com/person/sam-pullara)将加入 Twitter 工程团队。我们在下面嵌入了公告。

Bagcheck 允许人们分享和发现你的“包”里有什么。人们可以评论和分享他们使用的物品和原因。所谓“包”，初创企业指的是任何一种有组织的列表或集合。例如，人们可以发布一组摄影设备或他们最喜欢的烹饪工具。

目前还不清楚 Twitter 将如何处理 Backcheck(目前该服务继续保持在线和运营)，但 Pullara 无疑是该公司工程团队的一大胜利。他之前是 Benchmark 的 EIR，也是雅虎的前首席技术专家。他还在雅虎担任应用平台主管。在那里，他主要关注网格计算、虚拟化、大规模可靠的数据系统以及雅虎开放平台战略的实施。

在加入雅虎之前，Pullara 曾担任 Gauntlet Systems 的首席执行官，该公司开发了第一个智能持续集成系统。Gauntlet 被 Borland 收购，Pullara 在那里担任首席建筑师。

Pullara 在 Bagcheck 的合伙人卢克·乌鲁布莱夫斯基[写道](https://web.archive.org/web/20230204233326/http://www.lukew.com/ff/entry.asp?1371)他不会加入 Twitter 的工程团队。

Twitter 最近的收购包括 [BackType、](https://web.archive.org/web/20230204233326/https://techcrunch.com/2011/07/05/twitter-acquires-social-analytics-startup-backtype/)TweetDeck、和 [AdGrok](https://web.archive.org/web/20230204233326/https://techcrunch.com/2011/05/31/adgrok-twitter/) 。

*我们非常激动地宣布，Bagcheck 已被 Twitter 收购，从今天起，Sam Pullara 将加入 Twitter 工程团队。*

Bagcheck 的初衷是将人们与他们的兴趣联系起来，让人们有一个地方来发现和分享他们最感兴趣的物品“包”,无论是摄影器材的集合还是烹饪工具的军火库。现在，Sam 将把他将人们与他们感兴趣的东西联系起来的热情带到他在 Twitter 的新角色中。

巴格奇怎么了？
Bagcheck 将继续提供服务。这意味着你所有的伟大内容都停留在同一个网址上，仍然是可搜索的，可共享的，可爱的。但就像任何收购一样，未来的某个时候情况可能会发生变化，所以如果您担心您的内容，我们还创建了一个即时导出功能，可以将您制作的所有包打包成一套整齐的 HTML 和 JSON 文件。您可以在个人资料页面上使用该功能导出所有内容，并在任何您喜欢的地方发布或保存您的包。

一切都是关于打包工的
当然，没有人就没有激情。我们想借此机会向所有尝试过 Bagcheck out、制作手袋、参与讨论、向我们发送反馈甚至只是会意地看着我们的人表示最诚挚的感谢。您绝对是最棒的，我们感谢您为打造出色的 Bagcheck 所付出的时间和贡献。我们希望您继续这样做。
