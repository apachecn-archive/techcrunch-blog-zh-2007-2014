# 好了，我们有了第一个基于 DNA 的约会服务:基因伴侣

有人推出了一个基于 DNA 匹配寻找潜在匹配的交友网站，这只是时间问题。那个时间显然是今天随着[基因合伙人](https://web.archive.org/web/20230326030617/http://www.genepartner.com/)的推出(好吧，它是[不是](https://web.archive.org/web/20230326030617/http://www.scientificmatch.com/)第一个，但它是最便宜的)。

这家总部位于瑞士的公司表示，他们可以使用 199 美元的 DNA 测试(相比之下，23andMe 的价格为 1000 美元)来帮助你找到你的完美匹配，从统计上来说。他们分析了“数百对夫妇”，并确定了成功关系中的基因模式。基于他们的算法和你的 DNA，他们将决定两个人之间满意而持久的关系的概率(让我怀疑)。

浪漫呢？化学？当你遇见某人时，你的胃会有刺痛的感觉？你做梦去吧。约会的未来是 DNA 测试和口腔拭子，所以要习惯它:

> 用于从唾液中收集 DNA 样本的刷子——称为口腔拭子工具包——将被送到您的地址。按照试剂盒中的简单说明，你将轻轻地从你的脸颊内侧收集 DNA。使用随附的写有地址的信封退还刷子。

GenePartner 正在寻求与约会网站合作，并让这些服务鼓励用户查看他们的 DNA 是否匹配。

他们能避开美国新出台的关于基因检测的严格法律吗？他们在瑞士。我猜他们不会太担心加州和其他州的法律禁止他们的服务。
