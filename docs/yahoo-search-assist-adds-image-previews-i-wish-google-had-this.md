# 雅虎搜索助手增加图片预览；我希望谷歌有这个

[![](img/c1c3386b96d75c67c1a4eba8270a14fc.png)](https://web.archive.org/web/20221006120111/http://www.yahoo.com/)

谷歌获得的[搜索流量可能是雅虎的三倍多](https://web.archive.org/web/20221006120111/http://www.beta.techcrunch.com/2008/11/25/google-gains-us-search-market-share-in-october-but-growth-slows/)，但是，撇开 [SearchWiki](https://web.archive.org/web/20221006120111/http://www.beta.techcrunch.com/2008/11/20/google-makes-major-interface-change-to-search-searchwiki/) 不谈，大部分创新似乎来自雅虎和微软的 Live Search，因为它们都在努力获得更大的搜索市场份额。

去年，雅虎[推出了](https://web.archive.org/web/20221006120111/http://www.beta.techcrunch.com/2007/10/01/yahoo-search-just-got-smarter/)搜索助手，这是一种先进的自动完成功能，当你在搜索框中键入你的查询时，它会推荐相关的搜索。自动完成并不是一个新概念——谷歌已经通过 Firefox 和它的浏览器工具栏提供了多年(最近[将](https://web.archive.org/web/20221006120111/http://www.beta.techcrunch.com/2008/08/25/4-years-later-google-suggest-graduates-to-googlecom/)集成到了它的主页中)。但是雅虎走得更远，不仅仅是猜测你在输入什么词，它还会建议可能的相关搜索。

今天，该网站为其图像搜索引入了缩略图预览，允许用户看到他们应该如何修改他们的查询，以获得他们正在寻找的图像集。例如，键入“奥巴马”会出现对“奥巴马家庭”、“巴拉克·奥巴马”和“米歇尔·奥巴马”的建议搜索，每个搜索都伴随有指示查询将产生什么的缩略图。在我自己尝试了几次搜索后，很容易看出这是如何派上用场的，尤其是在搜索具有多种不同含义的查询时(如“内华达山脉”、啤酒或山脉)。也就是说，这个特性有点古怪——如果我想让任何建议出现，我经常不得不刷新页面。

与此同时，本周早些时候，微软的 Live Search 引入了使用图片本身搜索相似图片的功能。
