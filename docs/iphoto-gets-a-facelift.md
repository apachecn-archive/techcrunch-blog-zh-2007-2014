# iPhoto 焕然一新 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/01/06/iphoto-gets-a-facelift/>

![](img/f212606623c40202baeb4e6a267a1644.png)

苹果公司的菲尔·席勒在今天的 Macworld 大会上宣布，新版 iPhoto 将包含面部识别软件。这项名为“面孔”的功能可以让你将一个名字与面孔对应起来，然后它会试图找到其他具有相同面孔的图像，并让你确认该软件是否正确。我真的希望这有用，因为我根本没有时间给我 iPhoto 中的 7000 张照片做标记，其中 5000 张可能是我的一个儿子。iPhoto 还会自动生成包含您选择的面孔的照片的相簿。面部识别是一个很难解决的问题。我看到了才会相信。

iPhoto 还增加了一个名为“地点”的功能，可以让你给照片添加地理标签，或者使用 GPS 坐标(如果有的话)。现在有一个地图视图，显示你在地图上用大头针拍摄的所有照片(由谷歌地图提供)。它也终于支持 Flickr 和脸书上传(希望，自动调整大小)。是的，苹果，我们想把我们的照片放在网上，让更多的人看到。但好的一点是，地理标签可以与 Flickr 同步，如果有人在你的一张照片上添加了脸书标签，它就会同步回来。

我喜欢这个方向，因为苹果正在把 iPhoto 变成真正的 Webtop 应用。它应该看看由前苹果技术专家 Jean-Marie Huillot 创建的 [Fotonauts](https://web.archive.org/web/20221005100657/http://www.beta.techcrunch.com/2008/09/10/tc50-fotonauts-is-a-georgeous-photopedia/) ，它正在推动这一领域的发展。Fotonauts 可以让你把每张专辑变成一个网页，而不需要昂贵的 Me.com 账户，并且可以很容易地围绕每张专辑建立一个社交社区。
