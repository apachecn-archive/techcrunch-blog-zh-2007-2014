# Spiceworks 为 IT 网络软件添加了社交媒体插件

![](img/ac55d81d9807750a8fa6a7d6fba352be.png)

IT 软件制造商 [Spiceworks](https://web.archive.org/web/20221005223905/http://www.spiceworks.com/) 为 Spiceworks 桌面开发了一套定制插件和各种类别的小部件。Spiceworks 的[广告支持的、](https://web.archive.org/web/20221005223905/http://www.beta.techcrunch.com/2007/08/27/spiceworks-raises-8-million-google-adsense-even-supports-it-software/)免费 IT 管理软件允许中小型企业的 IT 经理跟踪他们的网络资产、运行帮助台、监控活动、接收报告和排除网络问题。

这些插件让您可以跟踪警报、票证、新软件和新硬件，以及清单摘要。小部件包括帮助台小部件以及报告和清单小部件，并允许用户轻松访问这些信息。Spiceworks 还允许用户在桌面上添加主题和皮肤，创建定制的用户门户，并允许用户从 RSS 订阅源和 Twitter、Digg、脸书和 MySpace 的社交网络小部件中下载新闻小部件。

Spiceworks 还为其软件用户创建了一个 IT 新闻社区。在社区中，您可以看到其他 Spiceworks IT 专业人员提交的信息，以及面向中小型企业 IT 专业人员社区的内容。

Spiceworks 表示，在过去的一周里，其 53 个小工具的下载量超过了 16000 次。该公司表示，最受欢迎的插件是帮助台插件，该软件的用户已经创建了 16 种语言插件。另一款 IT 网络管理软件，为客户创建了移动接入，但似乎没有将社交网络功能整合到平台中。

Spicework 的软件是围绕在 IT 经理中创建一个社区而开发的，所以毫不奇怪，该公司现在正在向其平台推出社交媒体功能。该软件允许用户与其他 IT 经理联系。当问题出现时，用户可以张贴问题，提交答案和使用小组编辑的维基文章等。该软件现在包括一个仪表板、RSS 源、社交网络小工具和一个 IT 新闻共享社区，似乎几乎正在变成一个伪社交网络。
