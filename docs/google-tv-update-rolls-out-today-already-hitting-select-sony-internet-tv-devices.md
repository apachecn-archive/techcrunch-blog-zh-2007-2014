# 谷歌电视更新今天推出，已经击中选择索尼互联网电视设备

备受期待的谷歌电视更新今天系统地推出。索尼互联网电视设备是第一批，紧随其后的是罗技 Revue 设备。但是耐心等待。与大多数重大更新一样，OTA 更新不会一次完成所有任务，而是有选择地进行。你可以在“设置”菜单的“关于”部分下启动手动检查，但当你的盒子号码被呼叫时，会弹出一个通知。

我的索尼网络电视上仍然没有这个功能，但根据 AndroidCentral 的截图，这个更新已经在网上发布了。

谷歌上周在 T2 宣布了重大更新。它给谷歌电视带来了安卓市场和全新的用户界面。此外，该系统现在基于更新的 Android 3.1 版本，这应该会带来更流畅的体验。谷歌电视一直显示出如此大的前景，这一最新的涂料进一步提高了它的吸引力。但是只有消费者市场才能决定这次改革最终是否会刺激 GTV 的复兴。
