# 禧玛诺转向公路车的电子设置

[![](img/2b09f77115730be02f9f697e68f5f88d.png "shimano")](https://web.archive.org/web/20230119210703/http://old.crunchgear.com/wp-content/uploads/2008/08/shimano.jpg)

通过提供电子变速，禧玛诺(Shimano)肯定会轰动一时。每当体育运动中出现新情况，人们都会兴奋起来。为什么骑自行车应该有所不同？电子换挡的引入其实并不是什么新鲜事，但是续航和重量的限制已经让这项技术不可行了。但是这一切都将改变 T4。

有了数字装置，缺少运动部件将使骑自行车的人更有优势，减少磨损和灰尘堆积。自我校准，设置减少了变数，车手将传统的影响。哦，而且会很快。像瞬间一样快。

Di2 由一个锂离子电池、一个线束、两个改进的拨链器和两个制动和变速杆组成，实际重量将低于当前产品。在一项秒和重量非常重要的竞技运动中，D12 将于 2009 年 1 月开始在商店销售，看起来会立即引起轰动。
