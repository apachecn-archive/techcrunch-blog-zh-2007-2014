# HTC 的感觉 Redux 到体育节拍由 Dre 音频？

自从 HTC[收购 Beats Electronics 的多数股权](https://web.archive.org/web/20230314185447/https://techcrunch.com/2011/08/10/htc-announcement/)以来，最大的问题是他们的哪些时髦新手机将首先受益于音频改革。有传言称，活力可能是第一个尝到德瑞医生待遇的味道，但根据泄露的规格表，它可能是 HTC 的感觉的刷新版本，抓住了区别。

据 [CNET](https://web.archive.org/web/20230314185447/http://reviews.cnet.com/8301-19736_7-20099846-251/htc-prepping-sensation-special-edition-with-beats-technology-exclusive/) 报道，也许最出名的是作为第一批参与 HTC 引导程序解锁过程的手机之一，HTC 的轰动可能会看到特别版模型的新生命。

规格表概述了处理能力的提升(1.5 GHz，相比原来的 1.5 GHz)，板载内存(4 GB，从 2gb 上升)，以及更强大的电池。据报道，这款设备将以 520 英镑(846 美元)的价格首次亮相*无*合约，这在很大程度上可以解释为包含了 Beats tech。

随着定价细节已经到位，特别版的感觉似乎准备推出早不宜迟。虽然幻灯片几乎可以保证在英国推出，但硬件更新也可能很快进入中国——该设备被列为支持 TD-SCDMA 网络，这是中国无线产业的标志。
