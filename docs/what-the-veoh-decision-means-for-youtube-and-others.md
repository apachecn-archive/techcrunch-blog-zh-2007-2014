# Veoh 的决定对 YouTube 和其他公司意味着什么

 [](https://web.archive.org/web/20230320125329/http://www.veoh.com/) 代表全国在线视频网站的律师们今天对 [Veoh 简易判决决定](https://web.archive.org/web/20230320125329/https://techcrunch.com/2008/08/27/transcoding-is-not-a-crime-says-court-in-veoh-porn-case/)垂涎三尺(我知道这一点，因为我和他们中的一些人谈过)。简而言之，这是我们今天学到的:如果你对你的服务中的版权材料采取合理的预防措施，你可能是好的。哦，对了，如果你要被起诉，试着在北加州的联邦法院被起诉，因为那里的法官比我们见过的其他一些联邦法官更适合上网。

具体来说，法院表示，在线视频网站受到 DMCA 安全港条款的保护，如果它们做了以下事情(我对判决的解释):

*   向用户提供充分的通知，禁止上传受版权保护的材料
*   迅速遵守 DMCA 下架通知*“在收到通知的当天(或之后几天内)。”*
*   使用指纹和其他技术来检测受版权保护的材料，即使这些方法是有缺陷的。
*   采取措施控制侵权用户。具体来说，侵权的帐户需要被终止，并禁止任何新帐户的电子邮件。法院认为 IP 地址禁止没有必要:*“…Io 没有提出任何证据表明跟踪(或核实)用户的实际身份或阻止其 IP 地址是更有效合理的实施手段。”*
*   将文件代码转换为 Flash 格式不会将文件置于站点的控制之下；他们仍然受到 DMCA 安全港的保护(见[上一篇](https://web.archive.org/web/20230320125329/https://techcrunch.com/2008/08/27/transcoding-is-not-a-crime-says-court-in-veoh-porn-case/))。
*   网站被鼓励抽查视频，如果他们这样做，删除可能侵权的内容。
*   网站不需要检查每个视频。法院说*“本法院发现没有一个理性的陪审员可以得出结论，全面审查每一份档案是可行的。即使这样的审查是可行的，也不能保证 Veoh 能够准确地识别出有问题的侵权内容。”*
*   拥有大量非侵权内容非常重要。法院指出，Veoh 仅收到了约 7%内容的 DMCA 通知。这有助于它的论点，即它不同于 Napster，后者*“存在的唯一目的是为侵犯版权提供网站和设施……Napster 计划的唯一目的是为容易侵犯版权提供一个论坛。”*

YouTube 显然对这一决定感到兴奋，它通过电子邮件向我们发送了首席法律顾问 Zahavah Levine 的以下声明:

> 很高兴看到法院确认 DMCA 保护像 YouTube 这样遵守法律和尊重版权的服务。YouTube 已经超越法律保护内容所有者，同时赋予人们在线交流和分享经验的权利。我们每天都在为内容所有者提供选择，让他们决定是删除还是保留他们的视频，甚至从他们的视频中赚取收入，我们正在开发最先进的工具，让他们做得更好。

法院的声明称，检查每一个视频是否侵权是不现实的，这对谷歌/YouTube 来说是一个重要的声明，该公司表示，YouTube 上每分钟都有 13 个小时的视频内容被上传。如果 Veoh 无法监控所有内容，YouTube 将面临一个更大的问题。

在双方爆发之前，重要的是要注意，这是一个地区法院的决定，很可能会被上诉。我猜想 YouTube 可能会向 Veoh 借出一名或十名律师，以任何可能的方式协助上诉。

但这仍然是一个关键的裁决，可能会影响[YouTube-维亚康姆 10 亿美元的诉讼](https://web.archive.org/web/20230320125329/https://techcrunch.com/2007/03/13/5217/)以及其他一系列案件。
