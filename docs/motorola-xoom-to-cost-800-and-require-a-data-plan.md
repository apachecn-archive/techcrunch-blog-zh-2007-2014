# 泄露的百思买广告称，摩托罗拉 Xoom 售价 800 美元，需要一个数据套餐

![](img/eaa2563c6543b21a61bc2d2b02105678.png "motorola-xoom-best-buy")

当我们在 [CES](https://web.archive.org/web/20221204180538/http://ces.crunchgear.com/) 上第一次[看到](https://web.archive.org/web/20221204180538/http://ces.crunchgear.com/2011/01/a-quick-look-at-the-xoom/)[摩托罗拉 Xoom](https://web.archive.org/web/20221204180538/http://www.crunchgear.com/tag/xoom) 的时，我们想的和说的都是好话。双核全蜂巢平板怎么可能出错？然后我们听到谣言说摩托罗拉想卖很多，所以我们认为价格会有竞争力。然后我们开始看到 800 美元标价的谣言，这让我们很难过。现在几乎可以确定的是，当它在 2 月 24 日[上映时，将会花费](https://web.archive.org/web/20221204180538/http://www.crunchgear.com/2011/02/04/moto-xoom-tablet-coming-february-24th/)[800 美元](https://web.archive.org/web/20221204180538/http://www.engadget.com/2011/02/06/verizon-locking-wifi-on-motorola-xoom-until-you-buy-one-month-of/)。一则泄露的百思买广告分享了 Xoom 的价格以及一些有趣但奇怪的消息。

显示 800 美元价格标签的广告还提到了该设备的某种 Wifi 锁定功能。广告说:

> 要在此设备上激活 WiFi 功能，至少需要订购一个月的数据服务。

因此，为了获得我们最初付费的功能，我们必须向威瑞森订购至少一个月的数据服务？谢谢，但是不用了，百思买。我们希望这只是某个只有百思买在做的[骗局](https://web.archive.org/web/20221204180538/http://www.crunchgear.com/2011/02/07/best-buy-scam/)，其他零售商只是在卖 Xoom——800 美元加上一个数据套餐让我想买一个 [Mifi](https://web.archive.org/web/20221204180538/http://www.mobilecrunch.com/2009/05/07/the-verizon-mifi-2020-ev-do-portable-wifi-router/) 和一个 [iPad](https://web.archive.org/web/20221204180538/http://www.crunchgear.com/tag/ipad) (只有 Wifi)。
