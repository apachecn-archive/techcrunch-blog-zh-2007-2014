# 你知道什么很酷吗？不是 Myspace 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/30/face-uh-myspace/>

Myspace 救世主[具体媒体](https://web.archive.org/web/20230204112127/http://www.specificmedia.com/) [昨日裁员 8%](https://web.archive.org/web/20230204112127/https://techcrunch.com/2011/09/29/myspace-owner-specific-media-hit-with-layoffs/)，约 50 人。根据一个消息来源，那些被解雇的大部分是 Myspace 的员工，大概是急切地等待他们的遣散费；“(原文如此)被解雇的人在庆祝，因为特定媒体不知道该做什么。”

在过去的三个月里，特定媒体似乎已经从新收购气味的兴奋变成了产后抑郁症，最近由于不利的进展，缩减了今年夏天[的](https://web.archive.org/web/20230204112127/http://online.wsj.com/article/SB10001424053111904106704576579263962636624.html)盛大 Myspace 重新启动计划。
