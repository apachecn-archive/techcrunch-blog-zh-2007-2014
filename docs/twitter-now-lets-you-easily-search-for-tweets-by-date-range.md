# Twitter 现在可以让你轻松地按日期范围搜索推文

快！去年愚人节，人们在推特上谈论了什么？

到目前为止，这实际上是一个很难回答的问题。即使在 Twitter 的[高级搜索](https://web.archive.org/web/20221127170638/https://twitter.com/search-advanced)页面，也没有明显的方法来限制特定日期范围的搜索。

现在[有了。](https://web.archive.org/web/20221127170638/https://twitter.com/search-advanced)

奇怪的是。你以前真的可以这样做吗？这只是一种痛苦，而且对用户来说一点也不友好。你必须非常熟悉 Twitter 搜索的工作方式——足以知道它支持“自:”和“直到:”操作符，你可以手动将它们添加到你的搜索中。

鉴于谷歌自动建议的“搜索推特”的第二项是“按日期搜索推特”，可以肯定地说这还不够简单。

这可能看起来微不足道——但是下一次当你发现一条你刚刚*知道*是在 2012 年圣诞节前后发布的该死的推文时，你会很高兴知道这一点。
