# 谷歌收购面部识别软件公司 PittPatt

根据谷歌网站上的公告，谷歌刚刚收购了面部识别软件公司 PittPatt(匹兹堡模式识别)。

PittPatt 是卡内基·梅隆大学的一个项目，它开发了一种面部识别技术，可以通过照片、视频等多种方式匹配人。该公司已经在人脸检测、人脸跟踪和人脸识别方面创造了许多算法。PittPatt 的人脸检测和跟踪 SDK 在照片中定位人脸，在视频中跟踪人脸的运动。

以下是 PittPatt 在其网站上发布的公告:*加入谷歌是旅程中激动人心的下一步，旅程始于 20 世纪 90 年代卡耐基·梅隆大学机器人研究所的研究，并于 2004 年继续匹兹堡模式识别(PittPatt)的发布。我们一直致力于在许多重要方面推进研究和技术，并在一些非常有趣的产品中看到了我们的技术。在谷歌，计算机视觉技术已经是许多现有产品(如图像搜索、YouTube、Picasa 和护目镜)的核心，所以加入谷歌并把我们的研究和技术成果带给更广泛的受众是很自然的事情。我们将继续挖掘计算机视觉在从简单的照片组织到复杂的视频和移动应用等应用中的潜力。*

据报道，谷歌一直在[更认真地探索将面部识别添加到其产品(即谷歌眼镜)中，但由于隐私问题而推迟了。正如该公司在 3 月份告诉搜索引擎土地公司的那样，除非有非常严格的隐私控制措施，否则谷歌不会在移动应用中推出面部识别。](https://web.archive.org/web/20230203051714/http://www.pcworld.com/article/224007/google_wont_release_awesome_facial_recognition_app.html)

但今年 5 月，谷歌董事长埃里克·施密特(Eric Schmidt)表示，该公司“不太可能采用面部识别程序”。

谷歌发表声明确认收购:

*“匹兹堡模式识别团队在模式识别和计算机视觉领域开发了创新技术。我们认为他们的研究和技术可以在许多方面惠及我们的用户，我们期待与他们合作。”*
