# 乌克兰学生开发出能将手语翻译成语音的手套

在悉尼举行的微软[想象杯](https://web.archive.org/web/20230325002940/http://www.imaginecup.com/)的决赛中，不乏令人印象深刻的学生项目，但引起我注意的[六个决赛选手](https://web.archive.org/web/20230325002940/http://www.imaginecup.com/CompetitionsContent/Competition/WorldwideFinalists.aspx)之一是乌克兰团队 QuadSquad 的一个名为 [EnableTalk](https://web.archive.org/web/20230325002940/http://enabletalk.com/) 的项目。目前约有 4000 万聋哑人，其中许多人使用手语交流，但真正懂手语的人很少。使用配有[柔性传感器、触摸传感器、陀螺仪和加速度计](https://web.archive.org/web/20230325002940/http://enabletalk.com/documentation.html)(以及一些太阳能电池以延长电池寿命)的手套， [EnableTalk 团队](https://web.archive.org/web/20230325002940/http://enabletalk.com/developers.html)建立了一个系统，可以将手语翻译成文本，然后使用文本到语音引擎翻译成口语。整个系统然后通过蓝牙连接到智能手机。

该团队已经建造了一些原型，并在乌克兰对手语使用者进行了测试。团队成员 Osika Maxim 说，这个项目的想法来自于与团体学校的听力障碍运动员的互动。

现有的几个接近 EnableTalk 提议的项目通常成本在 1200 美元左右，通常传感器较少，使用有线连接，没有集成的软件解决方案。另一方面，EnableTalk 表示，其原型的硬件成本约为每台设备 75 美元。

然而，除了成本之外，让这个项目如此有趣的另一个特点是，用户可以教会系统新的手势，并修改团队计划在标准手势库中发布的手势。考虑到手语之间的高度差异，就像口语一样，手语也有地方方言，这将是一个受用户欢迎的功能。

这是微软的竞争，该系统显然主要使用微软的技术，但正如 EnableTalk 团队指出的那样，Windows Phone 7 不允许开发者访问蓝牙堆栈，当前版本实际上运行在 [Windows Mobile](https://web.archive.org/web/20230325002940/http://en.wikipedia.org/wiki/Windows_Mobile) 上，这是 Windows Phone 的前身，甚至微软的大多数人都不愿再考虑它。
