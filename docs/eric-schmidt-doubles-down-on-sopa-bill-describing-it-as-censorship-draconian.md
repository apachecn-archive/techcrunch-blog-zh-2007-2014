# Eric Schmidt 加倍抨击 SOPA 法案，称其为“审查制度”，“严苛”

我们已经阐明了我们对保护知识产权、SOPA、电子寄生虫或任何你想称之为法案的立场，该法案创造了一个将域名列入黑名单的危险先例，并将权力集中在权利持有人身上，并且仍然模糊不清，容易被滥用。埃里克·施密特已经公开表示反对，称谷歌不会遵守其限制。今天，他[在麻省理工学院斯隆管理学院发表演讲时，稍微提高了一下措辞。](https://web.archive.org/web/20230203064450/http://www.reuters.com/article/2011/11/15/us-media-google-regulation-idUSTRE7AE2G120111115)

“解决方案非常严厉。有一项法案要求互联网服务提供商从网上删除网址，据我所知，这也被称为审查。”

通过投掷 C-bomb，施密特可能已经承诺成为科技行业中最高调的法案反对者之一，如果不是的话，这一立场可能会在以后造成一些不便。幸运的是，他有足够的影响力来支持他的严厉言辞。

如前所述，这并不真正表明谷歌方面的立场发生了变化。但是施密特继承了民意的衣钵，直言不讳，称审查为审查。这基本上是个人对这项事业的承诺，他在业界的地位意味着他将在参议院关于这项法案的听证会上花相当多的时间。对他来说很好，尽管这有助于掌控整部剧最重要的部分。

事实是，如果谷歌不遵守，这项法案将只不过是互联网上的马奇诺防线。即使必应、雅虎、脸书和所有其他公司都遵从，谷歌的主导地位也会影响他们的遵从，并获得好处，不仅表现为一名白衣骑士，还会吸引那些无法使用雅虎主屏幕搜索找到 RapidShare 的用户。

因此，从演讲的角度来说，施密特拥有全权，因为该法案的支持者所做的任何事情都不会对他们的立场或该法案的无效性产生丝毫影响。大多数科技领袖很可能会像施密特和其他一些人一样，安全地宣称团结一致，坦率地说话，我们敦促他们这样做。
