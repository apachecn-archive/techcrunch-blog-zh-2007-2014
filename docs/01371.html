Kyte成为品牌的移动到网络视频平台(增加新投资者和实时移动流媒体)

原文：https://web.archive.org/web/https://techcrunch.com/2008/03/07/kytes-big-day%e2%80%94launches-kytecom-adds-live-mobile-video-streaming-new-facebook-app-and-new-investors/amp/

Kyte首席执行官丹尼尔·格拉夫(Daniel Graf)向将手机变成视频分发平台又迈进了一大步。“这是Kyte的大日子，”他告诉我，“这是我们自去年四月推出beta版以来最大的一次发布。”到目前为止，Kyte允许人们通过将视频从他们的手机上传到各种小工具和Kyte.tv来创建他们自己的个人电视频道。今天，Kyte增加了来自手机和网络摄像头的实时视频流，通过你的个人Kyte频道播放，并存档供以后观看。(在这里注册私人测试版。小心， Justin.tv 和 Ustream )。它还从Steamboat Ventures和瑞典移动运营商TeliaSonera那里额外筹集了610万美元，以总计2110万美元完成了其B轮融资。

但最重要的是，Graf正致力于使Kyte成为音乐家、媒体公司和移动运营商的平台。他认为Kyte更多地被知名人士和媒体公司用来制作初始内容，然后通过Kyte播放器被观众分享和分发。为此，他刚刚推出了Kyte.com，作为品牌合作伙伴(包括博主)进入Kyte平台的网站。所有四大唱片公司都在使用Kyte来创建品牌播放器，这些播放器可以被widget化并在整个网络上分发。(下面是我们的，展示了我用格拉夫的手机拍摄的对他的采访)。

格拉芙说:“哪儿都没有写‘Kyte’。”“这就像一个微型网站。它可以通过病毒传播。粉丝们觉得真的和它有联系。”他们也能有所贡献。Kyte播放器有一个“制作”按钮，允许粉丝将他们自己的视频上传到频道。格拉芙为TechCrunch制作了上面的这个(这是继50美分之后的第二个)。在接下来的几个小时里，您可以使用网络摄像头添加自己的视频评论(我们将对此进行监控，因此请保持干净)。

50 Cent已经对Kyte进行了大约三个月的测试，他的Kyte频道已经在10，000个网站上有超过400万次的观看。这位Kyte玩家在他的网站上是最重要的。他经常把自己和他的摄制组拍摄的视频片段放在手机上。昨晚，他通过Kyte播放器首映了他的视频“机械师”，粉丝可以将它嵌入到自己的页面上。这是他在网上保持对其品牌控制的方式，即使人们抓取视频并放到其他网站上。Kyte计划在4月份在其播放器中打开广告，并与Kyte频道的制作人分享这些收入。

Kyte还让其合作伙伴能够在大约五分钟内将他们的Kyte播放器变成脸书应用程序。这是50美分的T1，这是T2的TechCrunch。它还为合作伙伴网站提供了一个 API ，以便与Kyte播放器和服务进行更深入的整合。它甚至集成了Twitter。最后，Kyte正在开发一键式移动应用程序，使从手机上传视频变得更加容易。一个这样的应用程序已经在诺基亚60系列手机上进行私人测试。
