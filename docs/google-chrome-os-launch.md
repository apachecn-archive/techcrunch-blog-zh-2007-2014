# 谷歌周四举行 Chrome OS 活动。将公布完整的概述和发布计划。

我们刚刚接到通知，谷歌计划于本周四上午在加州山景城总部举办一场特别的 Chrome OS 活动。我们被告知，计划是提供一些技术背景信息以及展示一些演示。更值得注意的是，他们将给出新操作系统的“完整概述”，他们说将于明年推出。

谷歌产品管理副总裁桑德尔·皮帅和谷歌 Chrome OS 工程总监马修·帕帕基波斯将在活动上发言。之后会有问答环节。

正如我们几天前报道的那样，谷歌已经计划在本周至少发布 Chrome OS 的一部分。这可能仍然是这一事件的情况，但现在看来，更完整的发射将在明年进行。如果他们现在带着“完整的概述”举行这次活动，显然正在取得进展，所以你可能会期待明年在提前*发布。*

谷歌在 7 月份首次宣布了 Chrome OS，但没有透露太多细节。似乎有更多的想法是[向微软](https://web.archive.org/web/20221207211429/http://www.beta.techcrunch.com/2009/07/07/google-drops-a-nuclear-bomb-on-microsoft-and-its-made-of-chrome/)扔一颗核弹，微软[即将宣布](https://web.archive.org/web/20221207211429/http://www.beta.techcrunch.com/2009/07/09/why-chrome-os-now-because-microsoft-office-in-the-cloud-comes-monday/)其在线免费版 Office。在接下来的几个月里，人们对 Chrome 操作系统的兴趣仍然很高，但我们看到的几乎都是[操作系统的浏览器可能会是什么样子](https://web.archive.org/web/20221207211429/http://www.beta.techcrunch.com/2009/10/13/a-first-glimpse-of-chrome-os-in-the-flesh-at-least-the-browser-part/)。
