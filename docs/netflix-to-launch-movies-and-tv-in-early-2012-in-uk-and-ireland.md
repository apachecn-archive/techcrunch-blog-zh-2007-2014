# 网飞和亚马逊将于明年在英国和爱尔兰展开视频流对决

![](img/76f59f3f08b1307cef141141315ac70f.png)
电影和电视流媒体服务网飞将于 2012 年初在英国和爱尔兰推出，[今天](https://web.archive.org/web/20230203104921/http://netflix.mediaroom.com/index.php?s=43&item=415)向该公司透露。定价将在发布前宣布，但该公司表示，它将在一系列能够从网飞流媒体播放的消费电子设备上发布，以及在个人电脑、MAC 电脑、移动平板电脑和手机上发布。

在英国，它将与亚马逊拥有的 [Lovefilm](https://web.archive.org/web/20230203104921/http://lovefilm.com/) 竞争，后者是成熟的流媒体服务。亚马逊在 1 月份以 2 亿英镑收购了 LoveFilm，自 2008 年 LoveFilm 收购其欧洲 DVD 租赁业务以来，亚马逊一直持有 love film 42%的股份。

在美国，网飞每月花费 7.99 美元。LoveFilm 开创了其业务，在英国以每月 9.99 英镑的订阅费通过邮件提供无限制的 DVD 租赁。

收购时，LoveFilm 得到了 Index Ventures、Balderton Capital、Arts Alliance Media 和 DFJ 思捷的支持。
