电子商务平台Shopify收购移动应用开发工作室Select Start

提供让任何人都能在线创建店面的交钥匙电子商务技术的Shopify收购了移动应用和游戏工作室 Select Start Studios。

此次收购将用于推进Shopify的移动商务计划。虽然所有Shopify商店都已经包括移动优化的店面和收银台，但收购S3将为Shopify团队带来20多名新的移动工程师，继续为商家开发移动产品。

Shopify的基于网络的平台允许任何人在瞬间建立一个商店，添加要出售的物品，上传图片，添加标签和分组物品，并集成PayPal或其他信用卡处理器进行支付。店面和购物车可以定制，该平台确保所有交易的安全性。如今，这个电子商务平台在80个不同的国家拥有超过20，000家活跃的在线商店。

例如，愤怒的小鸟使用Shopify的平台进行电子商务销售(该公司表示，愤怒的小鸟在线商店的相当大一部分销售是通过移动设备进行的)。Shopify最近筹集了1500万美元的新资金。
