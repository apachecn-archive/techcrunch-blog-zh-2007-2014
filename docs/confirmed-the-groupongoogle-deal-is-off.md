# 证实:Groupon/谷歌交易取消

传闻已久的谷歌收购 Groupon 的交易已经取消，我们已经从知情人士处得到证实。这个消息早些时候被《芝加哥商业报》报道过，我们已经证实这笔交易确实取消了。

两家公司至少在上周一直在进行认真的谈判，据[报道](https://web.archive.org/web/20230406082657/http://www.nytimes.com/2010/11/30/technology/30google.html)称，谷歌出价高达 60 亿美元收购这家炙手可热的本地交易公司。

我们的消息来源还证实，Groupon 的年收入目前为 20 亿美元，远高于之前流传的数字(这个数字是 AllThingsD 几分钟前[报道的](https://web.archive.org/web/20230406082657/http://kara.allthingsd.com/20101203/exclusive-groupon-annual-revenues-actually-2-billion/))。2B 的数字是 Groupons 销售的总价值——Groupon 的一半成本回到了商家，但所有的收入都通过 Groupon，就像谷歌收集所有的 AdSense 美元，在将一部分传递给出版商之前将它们算作收入一样。

CBB 的报道称，Groupon 可能正在考虑 IPO，尽管它要到明年才会做出上市的决定。

当我们听到更多消息时，我们会更新。
