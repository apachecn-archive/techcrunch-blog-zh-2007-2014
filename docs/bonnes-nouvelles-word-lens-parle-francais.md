# 美好的新生活！法语世界

还记得[字镜头](https://web.archive.org/web/20230209125058/http://itunes.apple.com/app/word-lens/id383463868?ign-mpt=uo%3D5)吗？那个疯狂的、令人敬畏的移动应用程序，将图像中的单词，如路标、海报、菜单等，从西班牙语翻译成英语(反之亦然)？好吧，现在这款应用已经增加了第三种语言:法语。通过最新的应用更新，Word Lens 可以从英语翻译成法语，也可以从法语翻译回来，但不能在法语和西班牙语之间进行翻译。

一年前， [Word Lens 风靡一时](https://web.archive.org/web/20230209125058/https://techcrunch.com/2010/12/16/world-lens-translates-words-inside-of-images-yes-really/)，在期待已久的发布之后——这是创始人奥塔维奥·古德和约翰·杜威斯两年半工作的成果。这款应用让我们集体惊叹不已。这东西很神奇。Word Lens 使用 OCR 技术在智能手机的摄像头前“看到”单词，利用增强现实技术将翻译后的单词叠加在外文文本上。所有不是文本的背景图像也会被移除。(看吧，增强现实也不是完全没用！).更好的是，该应用程序可以离线工作，因为它有可下载的字典。

该应用程序本身是免费的，但~~每本词典售价 4.99 美元~~更正:每本词典售价 9.99 美元。(法语英语和西班牙语英语)。Word Lens 中的翻译并不完美，但很少有数字翻译者是完美的。但是它通常足以让你明白这一点…也许可以让你免于点错菜单上的菜或者在镇上迷路。

你可以从 iTunes [这里](https://web.archive.org/web/20230209125058/http://itunes.apple.com/app/word-lens/id383463868?ign-mpt=uo%3D5)抓取更新的 Word Lens。
