# Taboola:即将宣布融资& CNN 交易？

视频推荐/发现提供商 [Taboola](https://web.archive.org/web/20230323203627/http://www.taboola.com/) 肯定发生了一些事情，我们上一次[报道](https://web.archive.org/web/20230323203627/https://techcrunch.com/2007/11/14/taboola-lands-15-million-powers-video-discovery/)是在一年前。我从以色列风险投资界的多个渠道得知，该公司已经完成了数百万美元的融资，Evergreen Ventures Partners 是主要投资者。此外，在美国大选的最后一个月，我也碰巧在 CNN.com 的视频上看到了 Taboola 的推荐。该公司和 Evergreen 都对任何融资轮或 CNN 交易守口如瓶。

Taboola 自 2006 年成立以来一直保持低调。我被告知，绝大多数雇员都是技术专家，有很强的数学和以色列军事背景。

Taboola 的旗舰产品[视频探索](https://web.archive.org/web/20230323203627/http://www.taboola.com/ViDiscovery)在公司网站上描述如下:

> Taboola ViDiscovery 超越了“一刀切”的答案，一次匹配一个观众的视频。利用其突破性的正在申请专利的数学技术，Taboola 预测每个观众在你的网站上观看的最佳后续视频。Taboola 正在申请专利的专有技术分三步进行:
> 
> 1.  视频背景–独立于任何相关文本的视频背景分析。
> 2.  观众动态——匿名研究观众在你的网站上观看视频时的观看模式。
> 3.  个性化匹配–利用从视频分析和观众观看模式中收集的信息，为每位观众提供个性化的视频推荐。

要在 CNN.com 上观看它的运行，请点击[这里](https://web.archive.org/web/20230323203627/http://edition.cnn.com/video/#/video/politics/2008/10/21/griffin.palin.interview.cnn)并按下视频窗口上的“类似此标签的视频”。

如果确实与 CNN 达成协议，这将成为该公司视频推荐/发现技术真正强大的一个主要标志。CNN 随后将加入该公司不断增长的客户名单，其中包括 aniBoom、5Min 和 Sclipo 等。

我们将密切关注 Taboola，并更新任何官方资金公告。
