# 便携式西瓜冷却器

[![](img/4cd39f660c7b0dc631fa62bcae3c8bd5.png "watermelon_cooler")](https://web.archive.org/web/20221218012640/http://www.crunchgear.com/2010/07/19/portable-watermelon-cooler/watermelon_cooler/)

就在你以为你已经见识了所有的小玩意的时候，日本却炮制出了另一种独特的设备。这一次是[丸后藤玉蟾](https://web.archive.org/web/20221218012640/http://joybond.co.jp/product/00/c00-a-01.php)【JP】，西瓜冷却器和加热器。没错，机器不只是冷却西瓜，出于某种原因，它还可以加热西瓜。它也是便携的。

制造商 Joybond 表示，加热功能也可以用于保持其他物品的温度，例如饮料或食物。Marugoto Tamachan 的容量为 14l，尺寸为 50.2×35.4×39.2cm(重量:6.3kg)。

[![](img/74533c739001f65c7ade1b953f3674d3.png "watermelon_cooler2")](https://web.archive.org/web/20221218012640/http://www.crunchgear.com/2010/07/19/portable-watermelon-cooler/watermelon_cooler2/)

不用说，这款设备只在日本有售(售价:230 美元)。问进出口专家 [Geek Stuff 4 U](https://web.archive.org/web/20221218012640/http://www.geekstuff4u.com/) 如果你真的有兴趣得到一个。
