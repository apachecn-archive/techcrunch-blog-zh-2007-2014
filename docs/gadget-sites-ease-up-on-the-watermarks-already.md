# 小工具网站:已经放松了水印

这里有一点棒球内幕给大家，但我想得到一个我胸口的宠物:所有这些该死的小工具图像水印。今天早上，BGR 发布了一些据称是 iPhone 4S 零件的图片，这些零件看起来(令人惊讶地)非常像 iPhone 4 零件。然而，我被大量的水印震惊了，这些水印被大胆地打在了图片的正面，一个来自印度的某个网站(没有网址，所以我不知道它是什么，从而降低了标记的价值)，还有一个 BGR 的大喇叭水印。所以 BGR 从印度的某个家伙那里得到了这些图像，他们通过标记自己的领地把它们变成了自己的，就像一只猫在你的衣柜里喷麝香，以确保你再也不能穿你的衣服。
![](img/b8fabb5062314b156dc21000454f3315.png)
水印最初是作为广告设计的:如果你的网站很小，你可以添加水印来传播你网站的信息，而不是阻止其他人使用你的图片。当你在谷歌上搜索史蒂夫·乔布斯骑着独角兽的图片时，它们也有助于评估所有权，结果发现它被 GDGT 打上了水印。我完全不喜欢偶尔不显眼的图像(我们有时会这样做，在角落里，30%的不透明度，主要是我们自己拍的图像)，但来吧。为什么不完全疯狂地在 Javascript 中添加一个在早期非常流行的右键单击防护器呢？

不管怎样，回到你的常规报道。谢谢你让一位老人发言一会儿。我会回去对着我的草坪大喊大叫。
