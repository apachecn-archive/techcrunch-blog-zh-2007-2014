# OLX 和 hi5 联手进行国际扩张计划

![](img/b4f69a43c484cc6bef59dc56dc88735a.png)[【OLX】](https://web.archive.org/web/20221210001248/http://olx.com/)和 [hi5](https://web.archive.org/web/20221210001248/http://hi5.com/) ，两人都是各自领域(在线分类广告和社交网络)霸主的挑战者，他们联手向彼此展示彼此公认的庞大但地理上分散的受众。OLX 表示，目前每月有来自 90 个国家的 7000 万独立访客，这在很大程度上要归功于与服务提供商的现有合作关系，这些服务提供商的增长大多来自拉丁美洲和亚洲(Friendster、MySpace Lat-Am、Fotolog 等)。)，而 hi5 声称每月有来自 200 个国家的 6000 万独立访客。

即使考虑到合理数量的重叠，这些数字也很可观，尽管在那些潜在广告收入普遍远低于美国和欧洲的国家。

OLX(Craigslist 在美国的竞争对手)和 hi5(脸书和 MySpace 等网站的全球竞争对手)声称，新的合作伙伴关系有助于巩固两家公司在拉丁美洲市场的份额，同时为在世界其他地区的加速增长铺平道路。

作为协议的一部分，hi5 将实现 OLX 的功能，包括在用户的个人资料页面上显示广告和查看朋友的广告，以及通过新闻源告诉朋友他们自己的广告的功能。OLX 还将允许用户在分类列表中添加视频和图片，在超过 39 种语言和 90 个国家发表评论、发布/查看列表，并从移动设备访问网站。后两个功能显然是为了吸引大量的国际用户群。

在线分类广告领域的另一个主要参与者 Oodle ，最近与社交网络服务和门户网站达成了类似的交易，如[脸书](https://web.archive.org/web/20221210001248/http://www.beta.techcrunch.com/2008/12/02/confirmed-oodle-to-power-facebook-classifieds/)、 [MySpace](https://web.archive.org/web/20221210001248/http://blog.oodle.com/2008/07/28/new-myspace-classifieds-powered-by-oodle/) 和 [AOL](https://web.archive.org/web/20221210001248/http://www.beta.techcrunch.com/2009/02/23/after-myspace-and-facebook-oodle-to-power-brand-new-aol-classifieds/) 。
