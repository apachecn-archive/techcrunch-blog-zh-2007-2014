# TransferWise 希望成为货币交易的 Skype

![](img/2c6a042d593a50d65be62bab19d0117e.png "logo_uus") [TransferWise](https://web.archive.org/web/20221216172004/http://www.transferwise.com/) 今天推出了它的在线货币兑换，希望给银行一个挤兑他们的钱的机会，同时扰乱整个货币兑换市场的消费者端。标题命题是，该平台让任何人都可以获得与银行在银行间市场获得的相同的中间市场汇率，TransferWise 对每笔交易收取 1 英镑的固定费用，无论交易金额如何。这可能是一件大事。

但同样值得注意的是该公司的创始人，其中包括 Skype 的头号员工 Taavet Hinrikus。

他在 VoIP 公司工作了 7 年，最近担任战略总监，并成为许多其他知名和即将成立的初创公司[的天使投资人，包括 TweetDeck](https://web.archive.org/web/20221216172004/http://www.beta.techcrunch.com/2009/01/16/twitter-me-this-tweetdeck-secures-angel-funding/) 、Mendeley、wikimart.ru 和 blip.tv. TransferWise 的另一位创始人是 Kristo krmann，他在德勤和普华永道有直接的金融服务背景。

很自然地，Hinrikus 说他希望 TransferWise“像 Skype 对电信那样对货币兑换进行操作”这是一个很好的线，但在这种情况下，似乎有肉在骨头上。传统上，消费者通过银行支付 3-6%的外币付款。Hinrikus 指出，在其他已经转移到网上的市场中，去掉中间环节(在很大程度上)已经“降低了几乎任何地方的成本”。

为此，由英国金融服务管理局(注册为 Exchange Solutions Ltd .)许可的 TransferWise 按照彭博、XE 和其他公司报告的“全球中间市场汇率”为资金转移提供便利。它通过匹配那些需要以各种方式兑换货币的人来做到这一点——点对点，如果你愿意，或者几个人——尽管转账是双向的。在推出时，该公司使英镑和欧元之间的交流，更多的货币跟随。

TransferWise 的直接和间接竞争对手包括低端的 Moneybookers 和 PayPal，或者像 HiFX 这样的大额货币经纪商，以及 CurrencyFair 这样的新贵，当然还有银行。该公司仍然由 Hinrikus 和 rmann 自筹资金。
