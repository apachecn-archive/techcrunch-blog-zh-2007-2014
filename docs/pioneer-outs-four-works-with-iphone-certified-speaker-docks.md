# 先锋输出四个“与 iPhone 兼容”的认证扬声器基座

[![](img/cabe857ad9904d9efd5db20e373fe2ad.png "XW-NAS3_lg")](https://web.archive.org/web/20221202025308/https://beta.techcrunch.com/wp-content/uploads/2010/04/xw-nas3_lg.jpg) 
iPod 扬声器坞站仍然是大企业，先锋刚刚宣布了四个坚实的高端竞争者。无论你是在寻找一个音频专用模型还是一个功能齐全的系统，这些模型中的一个可能会符合要求——只要你能支付高昂的入场费。

[![](img/2bedd116635f75900bdb6e76ab695e4e.png "XW-NAS3-K_lg")](https://web.archive.org/web/20221202025308/https://beta.techcrunch.com/wp-content/uploads/2010/04/xw-nas3-k_lg.jpg)
449 美元 [XW-NAS3/-K](https://web.archive.org/web/20221202025308/http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.pioneerelectronics.com%2FPUSA%2FProducts%2FHomeEntertainment%2FDocks%2Fci.XW-NAS3-K.Kuro&esheet=6265672&lan=en_US&anchor=XW-NAS3%2F-K&index=1&md5=885381f5fc3a487870e5f92a0617a04c) 主要针对那些首先寻求高保真声音，其次才是无线流媒体和视频输出等额外功能的用户。该系统围绕一个双隔离框架构建，据说可以改善整体低音响应，并显著降低音箱共振。一看就知道是实心的。该系统还可以通过 99 美元的适配器通过蓝牙从 iDevice over component 和复合视频以及流媒体输出视频。四月发布

[![](img/658e076fee2be09531ba72d25d884fa8.png "XW-NAC3-K_lg")](https://web.archive.org/web/20221202025308/https://beta.techcrunch.com/wp-content/uploads/2010/04/xw-nac3-k_lg.jpg)
[XW-NAC1-K](https://web.archive.org/web/20221202025308/http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.pioneerelectronics.com%2FPUSA%2FProducts%2FHomeEntertainment%2FDocks%2Fci.XW-NAC1-K.Kuro&esheet=6265672&lan=en_US&anchor=XW-NAC1-K&index=3&md5=c7b70ce23ba8b55fead93c5e20ce9e8e)和 [XW-NAC3-K](https://web.archive.org/web/20221202025308/http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.pioneerelectronics.com%2FPUSA%2FProducts%2FHomeEntertainment%2FDocks%2Fci.XW-NAC3-K.Kuro&esheet=6265672&lan=en_US&anchor=XW-NAC3-K&index=4&md5=7654053437654efa10007521c634ba89) 略有不同。虽然它们可能听起来都很棒，但它们拥有更多小发明，售价分别为 349 美元和 449 美元。两者都有双对接端口，能够在两个设备之间混合音频，这是一种 DJ 模式，先锋称之为双洗牌。更高端的 XW-NAC3-K 甚至可以很好地与集成了 DLNA 1.5 服务器的局域网配合使用，该服务器可以向连接的显示器提供媒体。这些型号也不需要任何其他东西来通过蓝牙传输内容。适配器是内置的。两者都将在今年 6 月上市。

然后是 XW-NAV1K-K。这个家伙有点不同，他专注于消费者融合，而不是任何具体的东西。它具有一个坞站，调频收音机，CD/DVD 升级播放器，以及 CD 到 USB 的抓取功能，允许用户将 CD 抓取到连接的闪存驱动器。期待这家伙在六月 299 美元。
