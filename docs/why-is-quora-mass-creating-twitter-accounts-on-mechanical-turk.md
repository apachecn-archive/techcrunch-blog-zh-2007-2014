# 为什么 Quora Mass 要在 Mechanical Turk 上创建 Twitter 账户？

除了别的以外，亚马逊的机械土耳其人是一个获得廉价劳动力来完成简单任务的好地方，这些任务有助于黑帽 SEO 工作和一般的社交网络垃圾邮件。这个问题越来越严重，已经成为一个严重的污染问题，我们已经开始真正的研究，试图追踪幕后的大人物。我们正在使用新的搜索引擎 [Blekko](https://web.archive.org/web/20221207203321/http://www.blekko.com/) ，它对页面排名是透明的，以了解搜索引擎如何处理所有这些东西。

在我们的研究中，我们发现了一些非常奇怪的事情——[Quora](https://web.archive.org/web/20221207203321/http://www.quora.com/)正在使用 Mechanical Turk 大量创建 Twitter 账户:“给定一个名字、用户名、密码和电子邮件地址，创建一个新的 Twitter 账户，”帖子说道。你将得到 15 美分的报酬，到今天为止，还有 1483 个机会。

说什么？听起来很糟糕。Quora 是否在创建虚假的 Twitter 账户来增加链接、流量和搜索引擎优化？

![](img/b41259e8db4b3b0ed71672ed1a0ac618.png)

不，事实上。所以推特在禁止它们之前应该先拖延一下。今天早些时候，我与 Quora 首席执行官亚当·德安杰洛进行了交谈，他们这样做有一个非常合理的原因，尽管很有趣。

“Quora 正在考虑使用 Twitter 作为话题 RSS 源的替代，”他说。目前还没有办法通过他们的 API 大规模创建 Twitter 账户，所以要么是员工手动完成，要么是每个账户 0.15 美元的外包工作。

所以这里没有什么邪恶的事情发生，实际上 Twitter 对 RSS 的稳定替代至少对 Twitter 来说是一件好事。现在我们只是期待能够在 Twitter 和电子邮件上追踪一些 Quora 话题。然而，Quora 没有透露这款产品何时上市。
