现在每个人都可以更快更容易地将数据移入和移出亚马逊S3技术危机

原文：https://web.archive.org/web/https://techcrunch.com/2010/06/10/aws-import-export-s3/

现在，每个人都可以更快更容易地将数据移入和移出亚马逊S3

亚马逊网络服务今天早上宣布了亚马逊简单存储服务(又称 S3 )的 AWS导入/导出的普遍可用性，该服务具有一个新的网络服务界面，能够轻松管理数据传输和迁移。

根据新闻稿，新功能使用便携式存储设备加快了大量数据的进出运输。对于大型数据集，AWS导入/导出据称比互联网传输“快得多”。

AWS Import/Export服务并不是全新的，许多公司已经提供了该服务，但截至今天，该服务基本上已经提供。价格是每个处理的存储设备80美元，每个数据加载小时2.49美元。

此外，亚马逊网络服务今天还宣布在AWS管理控制台中支持亚马逊S3。
