# 马克·贝尼奥夫想要#占领企业

Salesforce 创始人兼首席执行官马克·贝尼奥夫(Marc Benioff)擅长将消费者网络上发生的事情应用到商业中。他甚至认为占领华尔街是企业应该学习和效仿的。他在上面的视频中阐述了他的观点，这是我昨天在纽约用我的 iPhone 拍摄的(抱歉光线不好)。他的观点是，如果抗议者可以使用 Twitter 和脸书来#OccupyOakland，为什么公司不能使用同样的社交工具来组织自己和激励他们的客户？

“脸书真的在吞噬网络，”他说，呼应了马克·安德森关于[软件正在吞噬世界的观点。贝尼奥夫指出，人们每天花 4 个小时在社交网络上。他提到](https://web.archive.org/web/20230204054940/http://online.wsj.com/article/SB10001424053111903480904576512250915629460.html)[迪斯尼乐园](https://web.archive.org/web/20230204054940/http://www.facebook.com/Disneyland)是一个品牌利用脸书在人们参观后有效吸引他们的例子。

但不仅仅是脸书，推特也是如此。像 KLM 这样的航空公司正在使用 Twitter 来管理愤怒的顾客并奖励忠诚的顾客。看看下面的视频，看看荷航在一次营销实验中走了多远，让乘客在 Foursquare 上登记或在推特上谈论他们的航班，并给他们一份礼物。(也请注意背景中的古代飞机。那些是 DC-10 吗？).但至少他们的社会营销是最新的。

贝尼奥夫正在谈论他自己的游戏，因为他正试图通过 Chatter 向企业出售他自己的社交工具。但他确实相信，如果公司忽视社交，他们就是忽视了他们的客户。“我们看到它正在改变我们的社会，”他说，“我们可以利用它让我们的公司变得更强大。…这是商业领域的一场社会革命。”
