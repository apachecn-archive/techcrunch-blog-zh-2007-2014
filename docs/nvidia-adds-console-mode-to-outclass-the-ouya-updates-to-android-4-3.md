# Nvidia 增加控制台模式以超越 Ouya，更新至 Android 4.3 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/28/nvidia-adds-console-mode-to-outclass-the-ouya-updates-to-android-4-3/>

Nvidia 已经更新了其基于 Android 的移动游戏控制台,增加了许多新功能，其中一项非常有趣，它可能会影响不断增长的基于 Android 的家庭游戏控制台市场。对于用户来说，这是一个非常好的更新，增加了许多有价值的功能，而对于 Nvidia 来说，这是一个真正的平台，而不仅仅是一个旨在吸引原始设备制造商的演示设备。

除了上述内容，这次更新还增加了 Shield 的 Gamestream PC gameplay 流媒体服务的官方非测试版，该服务可插入 Steam，让用户通过本地 Wi-Fi 网络以高达 60fps 的速度在他们的设备上玩完整的 PC 游戏。此外，您可以将完整的应用程序和游戏文件从本地存储转移到外部 micro SD 卡，这可以清理空间，Home 按钮现在可以访问最近打开的应用程序和 Google Now。

今天推出的另一个新功能是“游戏手柄映射器”(Gamepad Mapper)，它旨在回答如何处理不支持游戏手柄的游戏的问题。它通过允许用户浏览并手动将触摸屏控制映射到 Shield 的硬件按钮、d-pad 和操纵杆来解决这一问题，事实上，它对许多已经上市的顶级游戏都是自动进行的。这仍然不是完美的解决方案(也就是说，所有的游戏制作商都在他们的代码中内置了对 Shield 的支持)，但它确实让原本完全不可玩的游戏变得可以玩了。

Gamestream 功能现在支持官方发布的 50 多种游戏(和其他非官方的)，感觉比我第一次使用它时更加完美。除了围绕盾牌的任何其他考虑之外，PC 流媒体对那些发现自己粘在 PC 上几个小时沉迷于新游戏的人来说是一个巨大的好处；能够把它带在家里任何你想去的地方是一个巨大的恩惠。

这里的另一个大优势是控制台模式，它增加了基本的 HDMI-out 功能，可以将 Shield 变成一个成熟的客厅控制台。它旨在与合作伙伴 Nyko 的 PlayPad Pro 无线蓝牙控制器配合使用，但它应该可以与任何 Android 蓝牙控制器配合使用。然而，PlayPad Pro 是与 Nvidia 联合设计的，这使得它更有可能与 Tegra 优化的游戏完全兼容。

现在，您可以点击新的控制台模式图标在电视上运行它，此外，当连接到电视时，即使翻盖关闭，控制器也可以从睡眠中唤醒。因为它的软件支持和它的工作方式不需要开发者做太多额外的工作，这使得它成为 Ouya 的一个非常有吸引力的替代方案。完整的 1080p 输出将通过更新来实现，最重要的是，你可以拔掉它，带着它去任何地方，没有电视也可以玩。Shield 可能比 Ouya 贵 200 美元，但在我看来，这种新的控制台模式使它总体上更有价值。它已经是我首选的旅行控制台，真的可以帮助我在出差的酒店度过无聊的夜晚。
