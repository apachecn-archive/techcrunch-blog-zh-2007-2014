# PayPal 增加 Visa Exec 管理亚洲快速增长的支付业务

随着 Visa 高管鲁珀特·基利(Rupert Keeley)的加入，PayPal 今天正在提升其在亚洲的领导地位。基利将加入易贝旗下的 PayPal，管理其在亚太市场的支付业务。

这对 PayPal 来说是一个胜利，因为 Keeley 在管理亚洲和其他国际市场的支付服务方面拥有丰富的经验。在加入 PayPal 之前，Keeley 曾担任 Visa 国际业务总裁，负责 Visa 在美洲以外的所有业务。此前，他还担任过 Visa 亚太区集团总裁，以及 Visa 战略和企业发展主管。在 1999 年加入 Visa 之前，他在渣打银行工作。

PayPal 总裁斯科特·汤普森表示，该公司亚太业务的增长速度超过了全球任何其他市场，尤其是中国。他补充说，澳大利亚和日本的电子商务市场也代表着这一业务的潜力。

在贝宝，基利将推动贝宝的“积极的区域增长战略”，努力增加新的人才，并将发展与亚洲客户和合作伙伴的关系。
