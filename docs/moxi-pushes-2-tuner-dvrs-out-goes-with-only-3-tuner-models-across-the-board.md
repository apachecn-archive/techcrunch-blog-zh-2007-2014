# Moxi 推出 2 调谐器 DVR，全面只推出 3 调谐器型号

[![](img/84beff1ebf4b0f98b08dcd5f3d0e22cd.png "moxi-hd-dvr")](https://web.archive.org/web/20221208075451/https://beta.techcrunch.com/wp-content/uploads/2010/05/moxi-hd-dvr.jpg) 
谁想要双调谐器 DVR？有人吗？Arris 不这么认为，现在只提供 3 调谐器 Moxi DVRs。

有消息称，客户对 3 调谐器选项的反应非常“积极”, Arris 的每个型号都采用该选项。3 调谐器型号[的价格将保持不变，为 599 美元](https://web.archive.org/web/20221208075451/http://moxi.com/us/home.html)，比较小型号贵 100 美元，但它当然不需要每月订阅计划。亚马逊仍然有一些库存的[双调谐器型号](https://web.archive.org/web/20221208075451/http://www.amazon.com/dp/B001GQ8MT8?tag=pulleys-20)，但是，如果你想以更低的价格加入 Moxi party。
