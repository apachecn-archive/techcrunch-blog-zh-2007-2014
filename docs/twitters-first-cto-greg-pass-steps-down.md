# Twitter 首任首席技术官格雷格·帕斯下台

据一位熟悉 Twitter 的消息人士透露，我们听说 Twitter 首席技术官格雷戈·帕斯今天已经离开了 Twitter。

在成为首席技术官之前，帕斯在 Twitter 担任工程副总裁，此前他是 Summize 的首席技术官和联合创始人，sum mize 于 2008 年 7 月被 Twitter 收购，并最终转变为 Twitter Search。在 Summize 之前，Pass 是 AOL 的系统架构师。

作为工程副总裁，帕斯负责带领 Twitter 开发团队从十几名工程师发展到 10 倍于这个数字，并在 Twitter 的成功扩展(T4)中发挥了开创性的作用。

我已经联系了 Twitter 的公关部门了解更多信息，但还没有得到回复。我也没有听说谁将取代帕斯，或者帕斯的未来计划是什么。

**更新:**推特联合创始人比兹·斯通证实帕斯将离职，推特如下。公关代表肖恩·加勒特表示，Twitter 并没有寻找替代者，格雷格·帕斯目前也没有确定的未来计划。
