T-Mobile myTouch 4G滑盖已确认于7月27日发布，7月19日预购

经过一番混乱之后， T-Mobile 前往脸书告诉我们myTouch 4G Slide的确切上市日期。我们最初预计7月27日发布，幸运的是(也有消息透露)，我们是对的。T-Mo的脸书更新也确认了7月19日将开启预购期(仅限网络)。

从一开始，我们就一直在关注这款手机，我们很高兴看到它最终走向市场。规格包括运行 Android 2.3姜饼的1.2GHz处理器，顶部是 HTC的Sense 3.0 UI 。该滑块还配备了3.7英寸的WVGA super CD显示屏，分辨率为800×480像素，以及“任何智能手机上最先进的摄像头”

老实说，myTouch 4G Slide上的实际摄像头硬件是你的标准800万像素后置摄像头。“最先进”的部分通过软件发挥作用，因为相机应用程序具有许多不同的功能，使事情变得快速、清晰和有创意。以下是myTouch 4G Slide摄像头的运行视频:
