# 特德诉萨拉·丝沃曼的斗争变得很迟钝

特德诉萨拉·丝沃曼案的失败让 T2 变得更加丑陋。

快速背景–以震惊和侮辱幽默闻名的喜剧演员萨拉·丝沃曼，被邀请做一个 ted 演讲。她随后被 TED 组织者 Chris Anderson 诋毁，他在推特上写道*“我知道我不应该这样说我自己的一个演讲者，但我认为萨拉·丝沃曼太可怕了……”*

问题是西尔弗曼一直用“弱智”这个词来对抗萨拉·佩林最近对这个词的抨击。整个演讲完全被 TED 观众忽略了。

安德森删除了他的推文，但西尔弗曼今天用自己的推文进行了回击，称“感谢@tedChris 让 TED 成为所有人的不安全天堂！你是比尔·盖茨屁眼上的平庸之辈。”

试图置身事外的安德森一定很高兴。

然后，有点随机地，AOL 的创始人史蒂夫·凯斯也加入了捍卫安德森的战斗。他(也在推特上)对西尔弗曼说了“你真可耻”，并补充道“可悲的是你并不那么有趣。”

Silverman 的[回应](https://web.archive.org/web/20230203014132/http://twitter.com/SarahKSilverman/status/9120915563):*“@ Steve case 你应该对地球上最后一个拥有 aol 账户的人好一点。”*

…西尔弗曼和凯斯之间的斗争仍在继续，但你可以想象一下。

我们的观点:Silverman 被邀请到 ted 做她该做的事，她做到了。没人会对她的幽默感到惊讶。她随后遭到安德森的人身攻击。她还击并不奇怪。TED 演讲者要小心了——确保你的演讲在可接受的范围内，否则你将面临 TED 的愤怒。和他们的朋友。
