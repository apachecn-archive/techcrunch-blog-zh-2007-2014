# 微软将在与天空广播公司解决商标案后更名 SkyDrive

英国天空广播集团和微软今天宣布，他们已经在欧洲解决了天空广播公司在英国法庭对微软提起的商标侵权纠纷。因此，微软将不得不为其云存储服务 [SkyDrive](https://web.archive.org/web/20221005230337/http://skydrive.com/) 寻找一个新名字。Sky 将允许微软在“过渡期”继续使用该名称，但在此之后，微软需要重新命名该服务。尚不清楚微软计划何时宣布新名称。

[本月早些时候](https://web.archive.org/web/20221005230337/http://www.zdnet.com/microsoft-loses-skydrive-trademark-case-in-eu-7000017502/)，英国高等法院[发现](https://web.archive.org/web/20221005230337/http://www.bailii.org/ew/cases/EWHC/Ch/2013/1826.html)微软使用术语 SkyDrive 侵犯了 Sky 的“Sky”商标权。虽然微软可以对这一裁决提出上诉，但该公司选择了和解，并同意不进行上诉。和解协议包括一些财务条款，但这些条款的细节是保密的。

“我们很高兴这个命名争议得到解决，并将继续提供上亿客户期望的优质服务，提供让您的文件永远在身边的最佳方式，”微软今天在一份录音声明中表示。我们要求微软进一步澄清，但除了这份声明，该公司对今天的声明不予置评。

所有这些对于任何过去关注过微软品牌的人来说，听起来都非常熟悉。毕竟，在微软新操作系统发布之前，Windows 8 Metro 界面刚刚成为一个家喻户晓的名字，当时该公司不得不改变它，因为大多数专家认为这是与德国 Metro AG 的商标纠纷。今天，没有人知道如何称呼 Windows 8 的界面、应用程序和设计语言，所以大多数人仍然称它为“Metro”
