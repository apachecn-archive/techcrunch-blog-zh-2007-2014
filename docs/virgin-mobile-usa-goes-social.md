# 美国维珍手机公司走向社交

如果你是那种拥有维珍手机服务的手机用户，那么你可能是那种喜欢炫耀自己个性的人。你带着不同的包跑，你甚至不会被整个 iPhone 的事情所困扰。至少这是今年夏天早些时候维珍移动广告活动所说的。

维珍移动美国用户现在可以与志同道合的人分享他们的感受。该运营商与 Intercasting Corporation 合作，为其 ANTHEM 平台上的移动运营商带来多样化的社交网络和社区网站。

Virgin Mobile USA 移动数据服务部门副总裁 Dominick Tolli 表示:“我们成功的关键在于为客户提供最大的灵活性和控制力。“有越来越多的社交网站反映了不同的生活方式和兴趣，我们致力于让我们的客户处于主导地位:也就是说，让他们能够创造自己的个人体验，决定他们想要如何交流以及在他们喜欢的时候与谁交流。”

定价计划包括按天或按月付费的选项，24 小时费用为 1.00 美元(上限为半兆数据)，每月 3.99 美元，上限为 5 兆数据。我们很好奇超级用户是否会很快封顶，维珍是否会为那些极度社交化的 T4 人提供其他定价选择。同时，如果你超过了上限，你只能用手机打电话来社交。什么概念！
