# Dropbox For Android 更新包 ICS 用户视频流，朝鲜语支持

嘿，冰淇淋三明治用户——如果你是 Dropbox 用户，不知何故没有获得最新的(2.1.4)更新，帮自己一个忙[开始使用](https://web.archive.org/web/20221226014750/https://play.google.com/store/apps/details?id=com.dropbox.android&feature=nav_result#?t=W251bGwsMSwyLDNd)。

这次更新肯定会让那些发现 Dropbox 体验很少使用韩文的人高兴，因为它引入了韩文支持，但还有更好的东西可以玩。这一版本增加了一个漂亮的冰淇淋三明治专属功能——能够播放存储在 Dropbox 帐户中的视频。

作为一个数字媒体收藏者，我扔了一些大的视频文件(500MB。avi 和一个 400MB。mkv)到我的 Dropbox 账户，看看事情实际上是如何运作的。值得庆幸的是，等待它们被存储在云中是这个过程中最令人讨厌的部分——一旦一切就绪，从 Dropbox 应用程序中选择文件并暂时播放一集旧的 *Top Gear 就足够简单了。*

应该注意的是，质量有时会在这个过程中受到影响。视频从来没有达到我想完全放弃的地步，但那些希望他们的视频像素完美的人可能应该只是将文件下载到他们的设备上。不言而喻，如果你使用的是有上限的数据计划，这可能不是最好的选择，尽管我的 Galaxy Nexus 上的 LTE 连接保持得很好。
