# 自由落体中的蹦极实验室 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/08/28/bungee-labs-in-a-freefall/>

 [](https://web.archive.org/web/20221006002217/http://www.crunchbase.com/company/bungee-labs) [蹦极实验室](https://web.archive.org/web/20221006002217/http://www.bungeelabs.com/)，一家[资金雄厚的](https://web.archive.org/web/20221006002217/http://www.crunchbase.com/company/bungee-labs)犹他州初创公司，仅在六个月前[离开私有测试版](https://web.archive.org/web/20221006002217/http://www.beta.techcrunch.com/2008/02/18/bungee-connect-launches-ambitious-new-online-development-product/)，今天解雇了 15 名员工，以给自己更多的烧钱机会。我们最后一次检查时，他们有 38 名员工，所以这几乎是他们总人数的 40%。

首席执行官 [Martin Plaehn](https://web.archive.org/web/20221006002217/http://www.crunchbase.com/person/martin-plaehn) [解释道](https://web.archive.org/web/20221006002217/http://blogs.bungeeconnect.com/2008/08/27/changes/) : *“这种变化与技术发展速度关系不大，与实际采用率和预期采用率关系更大。我们的平台即服务——Bungee Connect 已经达到了我们预想的稳健性和功能水平，我们致力于继续定期改进和支持它。与大多数新的突破性产品一样，蹦极连接需要更长的培育时间才能被广泛接受。作为一家初创企业，我们昨天的行动将我们的运营计划延伸到了 2010 年，以更深入地在市场上建立蹦极连接。”*

该产品有点类似于一大堆竞争对手——平台即服务应用程序开发。我们在之前的帖子中提到过一些，包括 DabbleDB、Zoho Creator、LongJump、Coghead 和 WyaWorks 等等。Salesforce 的 Force.com 也是竞争对手。因此，简而言之，市场可能很成熟，但不能保证它们最终会出现在蹦极实验室。

至少有一名员工说他不后悔。亚历克斯·巴尼特写道:“没有遗憾，一点也没有。当我考虑加入蹦极实验室的机会时(这样做是为了离开一个相对安全的港湾)，我知道其中的风险。蹦极实验室的使命是——现在仍然是——那种旨在“改变世界”的使命。成为负责实现公司宏伟目标的团队的一员，是一项完全有价值且有教育意义的追求。"

在我听来，好像有人已经获得了一些股票。如果我被解雇了，我会很生气的。
