# StatSheet 更名为 Automated Insights，赢得 400 万美元

**独家—**[专注于实时内容自动化的 StatSheet](https://web.archive.org/web/20230205045825/http://www.statsheet.com/) 正在将[扩展到体育](https://web.archive.org/web/20230205045825/https://techcrunch.com/2008/12/17/sports-fans-keep-up-with-your-teams-on-stattweets/)之外，将其技术应用于其他数据密集型垂直行业，在这些行业中，创建高质量内容的成本仍然很高。

该公司已经决定，一个新的名字将更好地反映其扩大的重点，并从此被称为[自动化见解](https://web.archive.org/web/20230205045825/http://automatedinsights.com/)。此外，这家初创公司已经获得了 400 万美元的新资金来刺激增长。

这轮融资由 Court Square Ventures 和 OCA Ventures 牵头，IDEA Fund Partners 和其他现有投资者参与。他们早些时候筹集了 130 万美元。

根据记录，自动洞察将进入新的市场(想想金融、天气、房地产等。)但在这样做的同时，仍将保留其运动产品的 StatSheet 品牌。

“自动洞察”技术[自动将原始数据](https://web.archive.org/web/20230205045825/https://techcrunch.com/2010/11/12/automated-news-sports-statsheet/)转化为叙述性内容和可视化。输出的内容完全由软件编写，但随后可以格式化为标题、摘要和长篇文章，然后通过网络、移动应用程序和社交媒体分发。

他们最近将其技术应用于美国职业棒球大联盟(MLB)的报道。该公司现在正寻求为所有 32 个 NFL 和 244 个 NCAA 第一组大学足球队推出以团队为中心的网站和移动应用程序(见[一个例子](https://web.archive.org/web/20230205045825/http://bosoxball.com/))。

该公司表示，自去年 11 月以来，其软件已经生成了超过 10 万篇文章(涵盖 NCAA 篮球和 MLB)。

值得注意的是:Automated Insights 在不到一年的时间里只用了 12 名员工就做到了这一步，因为他们的大部分工作都是完全自动化的。
