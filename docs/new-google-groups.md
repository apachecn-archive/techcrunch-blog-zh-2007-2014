# 谷歌推出第一阶段的改进小组:新的 Gmail/类似阅读器的用户界面

![](img/7831df6f19e78ec788e79a0efde870d6.png "gg")

好像他们还没有忙着发布 [Chrome 网上商店](https://web.archive.org/web/20230202221758/https://techcrunch.com/2010/12/07/live-from-googles-chrome-event-chrome-os-web-store-and-more/)、 [Chrome OS 试驾](https://web.archive.org/web/20230202221758/https://techcrunch.com/2010/12/07/google-announces-chrome-os-pilot-program-12-1-inch-notebook-cr-48/)，以及无意中让[第一张 Google +1 图片泄露](https://web.archive.org/web/20230202221758/https://techcrunch.com/2010/12/07/google-plus-one-pic/)，谷歌今天又有了一个锦囊妙计:新的谷歌团队。

从技术上讲，谷歌只是预览新的谷歌群组用户界面。“*新的 Google Groups 用户界面代表了 Google Groups* 系列更新的第一步，”Google 在这里写道。这真的是一次彻底的改造。新的风格更符合 Gmail/谷歌阅读器的模式。事实上，有些元素甚至看起来有点像波浪。

新用户界面的关键是一个左侧菜单，使您可以轻松访问您的组、公告、最近查看的组和您最喜爱的组。还有各种新的键盘快捷键。个人资料现在可以更加个性化，并链接到谷歌个人资料。还有一个新的编辑器用于添加新的主题/讨论。

谷歌还彻底改造了群组的移动体验，让你可以在旅途中轻松管理它们。

如果你访问[群组主页](https://web.archive.org/web/20230202221758/http://groups.google.com/)，谷歌表示，在接下来的一周左右，你应该会看到一条消息，通过这个新的用户界面选择加入查看群组。

"*我们很高兴与您分享这些改进，但这仅仅是一个开始；我们正在努力改进垃圾邮件控制、审核、搜索和其他功能，以使谷歌群组成为参与讨论的最佳方式。*

如果这似乎是谷歌对新脸书群体的回答，请记住，即将推出的 [Google +1 社交产品](https://web.archive.org/web/20230202221758/https://techcrunch.com/2010/12/02/google-plus-one-brin/)也应该有一个分组机制([称为“循环”](https://web.archive.org/web/20230202221758/https://techcrunch.com/2010/12/04/google-plus-one-iphone-facebook-loop/))直接绑定。相反，这似乎只是谷歌试图恢复他们越来越垃圾的公共论坛区。
