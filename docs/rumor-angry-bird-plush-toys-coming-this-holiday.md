# 愤怒的小鸟毛绒玩具即将在这个假期上市

![](img/b474c071f09b25dd596f3462afb11daf.png "5083174881_46c42f72e7")

《愤怒的小鸟》(Angry Birds)这款游戏莫名其妙地赢得了数百万人的心(我还是更喜欢《粉碎城堡》),它显然即将来到你附近的一家玩具店。这两只鸟，在诺基亚总部用一只[诺基亚](https://web.archive.org/web/20221007234137/http://crunchgear.com/tag/nokia)N8 射击，可以用弹弓把它们扔到房间的另一头，攻击偷了你窝的本地猪。

没有关于价格或可用性的消息，但它们看起来做得非常好，所以我们只能假设它们要么是赠品的原型，要么是商业产品。

 [《愤怒的小鸟》](https://web.archive.org/web/20221007234137/http://www.rovio.com/index.html)现在可以在包括 [Android](https://web.archive.org/web/20221007234137/http://rovio.us1.list-manage1.com/track/click?u=63b9f50404dc0d105599c267a&id=eeb337164e&e=d4f7b7c8b5) 、Windows Phone 7 和 Palm 的 WebOS 在内的大多数平台上使用。随着所有这些产品的推出，很明显《愤怒的小鸟》现在是《孤岛惊魂》的新的移动版本
