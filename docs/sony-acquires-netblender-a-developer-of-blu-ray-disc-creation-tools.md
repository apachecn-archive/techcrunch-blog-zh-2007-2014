# 索尼收购蓝光光盘制作工具开发商 Netblender

[索尼创意软件](https://web.archive.org/web/20230205045119/http://www.sonycreativesoftware.com/)，显然是[索尼](https://web.archive.org/web/20230205045119/http://www.crunchbase.com/company/sony)的子公司，今天早上[宣布](https://web.archive.org/web/20230205045119/http://www.prnewswire.com/news-releases/sony-creative-software-announces-acquisition-of-netblender-132432783.html)收购 [Netblender](https://web.archive.org/web/20230205045119/http://www.netblender.com/) ，这是一家成立六年的公司，开发一系列蓝光光盘制作工具。

收购条款没有披露。

这笔交易使索尼创意软件公司控制了 NetBlender 的光盘创作应用程序 DoStudio 产品线，以扩大其目前的蓝光光盘软件产品组合(该公司还生产专业的视频编辑和音乐创作工具)。

2009 年，NetBlender 首次推出了其 [BD Touch](https://web.archive.org/web/20230205045119/http://www.netblender.com/main/products/bd-touch/) 技术，该技术允许消费者用他们的 iOS 和 Android 设备控制电视、蓝光播放器和有线电视盒。

总部设在弗吉尼亚州亚历山大市的 NetBlender 的产品据说迄今已被用于生产 2000 多张零售蓝光光盘。

索尼创意软件和 NetBlender 在一份声明中表示，在整个过渡期间，它们将继续提供客户服务和技术支持。
