# 视频:观看诺基亚 N9 成型

我们并不是每天都能看到我们最喜欢的手机是如何制作的，但是来自诺基亚的一个新视频向我们展示了他们华丽的一体式 N9 手机是如何制作的。

尽管你可能会对它运行节目的死胡同软件有什么感觉，但毫无疑问，N9 是一款视觉上引人注目的手机。我是一个真正喜欢好看设备的人，诺基亚的 N9 在我的列表中排名相当靠前——它有一点 iPod nano 的味道(甚至连配色方案都有)，但我认为它是一个干净的高级外观设计，只是工作正常。

诺基亚在一部几乎没有人会买的[手机上使用这样的设计有点可惜。](https://web.archive.org/web/20230203075419/https://techcrunch.com/2011/10/15/nokia-n9-the-most-amazing-phone-youll-never-buy/)尽管如此，还是有希望的:诺基亚世界就在眼前，如果运气好的话，N9 风格的诺基亚 800/SeaRay Windows Phone 将通过将非凡的设计与真正的操作系统竞争者结合起来而抢尽风头。
