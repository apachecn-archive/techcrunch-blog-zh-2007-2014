# 一个绝妙的骗局之后，Hotelicopter 加入了酒店聚合的竞争

[![](img/a9499648e5c8df7b4cd26992fa476b7a.png)](https://web.archive.org/web/20221006194948/http://www.hotelicopter.com/)

就在一周前，一些博客向我们展示了 Hotelicopter，这是世界上第一家[飞行酒店](https://web.archive.org/web/20221006194948/http://i.gizmodo.com/5187289/stay-at-the-hotelicopter-the-worlds-first-flying-hotel)，以一架经过改装的苏联直升机为特色，打造了一个五星级空中度假村。当然，这最终被证明是为了庆祝愚人节而搞的一场恶作剧(尽管还很早)——Hotelicopter 实际上是一家网络初创公司，今天向公众发布。

Hotelicopter 是一个新的酒店预订网站，允许用户在 30 个旅游网站上搜索超过 150，000 家酒店的房间。该网站还因为是第一个包含脸书连接集成的酒店聚合器而闻名，它可以用来提供社交酒店推荐，也可以让你的朋友知道你住在哪里。

该网站有一个干净的界面，允许用户使用滑动条根据价格、星级(基于猫途鹰的评论)和你可能想要的便利设施(互联网接入、含早餐等)的复选框来缩小搜索结果。价格清楚地标注在房间旁边，用户可以直接预订房间，而不必经过其他环节。

总的来说，它似乎工作得很好，但似乎没有太多区别 Hotelicopter 和其他网站，如 Kayak。也就是说，Hotelicopter 背后的人显然有很好的幽默感(和营销)，因此他们或许能够在拥挤的空间里扬名立万。
