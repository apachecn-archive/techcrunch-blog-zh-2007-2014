# 到 2019 年，FCC 将加强所有使用 E911 的运营商的定位精度要求

在拥有一部配备 GPS 的手机多年后，没有 GPS 的想法有点疯狂。即使智能手机的普及率大幅提高，仍然有很多人拥有地图和令人惊叹的记忆力。有了这些伟大的移动技术，那些在危机时刻需要帮助的人应该能够依靠紧急服务来获得准确的定位技术，并能够快速找到他们，这才是公平的。

联邦通信委员会因此规定，所有电话服务提供商，包括 VOiP 服务，必须在 2019 年确定的日期前满足增加的位置准确性规则，以便更好地帮助确定 911 电话的位置。

根据[法院新闻服务](https://web.archive.org/web/20230204193044/http://www.courthousenews.com/2011/09/30/40208.htm)的报道，没有 GPS 的 911 电话需要运营商从手机信号塔对呼叫者的位置进行三角定位，这比手机的 GPS 简单地将位置数据中继回紧急服务的效率要低。

不管怎样，联邦通信委员会估计，不管有没有新规定，到 2018 年，85%的手机用户将拥有配备 GPS 的设备。希望剩下的 15%能在任何需要拨打 911 的事情发生之前完成这个项目。

你可以在这里找到 [FCC 的声明](https://web.archive.org/web/20230204193044/http://transition.fcc.gov/Daily_Releases/Daily_Business/2011/db0712/DOC-308377A1.pdf)。
