# 孤独星球收购移动旅游应用 TouristEye 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/11/13/lonely-planet-touristeye/>

今年早些时候，我们写了一篇关于 [TouristEye](https://web.archive.org/web/20230223164715/http://www.touristeye.com/) 的文章，这是一款为[计划旅行并在旅行中发现新的事情做](https://web.archive.org/web/20230223164715/https://techcrunch.com/2013/07/02/touristeye-getaway-recommendations/)的移动应用。今天，这家初创公司宣布，它已被旅游重量级公司 Lonely Planet 收购，因为该公司试图扩大其产品组合。

TouristEye】自 2010 年成立以来，由一群西班牙企业家创建，旨在创建一个移动优先的应用程序，用于制定行程和收集在新城市旅行时要做的事情。该应用程序聚合了来自多个来源的数据，以建立世界各地地标和景点的档案。

有很多应用程序可以找到更多关于旅行时要做的事情。然而，TouristEye 的最大区别在于，它允许用户在行程中添加项目，稍后可以离线查看，因此用户可以访问他们的旅行计划和位置，即使他们在国外没有连接 WiFi 或没有数据计划。

该公司的重大转折点发生在它成为 500 Startups Accelerator 的一部分之后，在这一点上，团队对自己的内部指标有了更好的理解，哪些功能很重要，哪些功能推动了应用程序的参与度。据联合创始人阿里尔·加缪(Ariel Camus)称，在它进入时，只有 7%的用户会定期返回该应用。通过给用户提供周末旅行或当地度假时可以做的事情的建议，甚至在他们不旅行的时候，它也能够显著增加这一数字。

虽然 TouristEye 能够增加参与度并扩大用户群，但它仍然面临着与旅游行业巨头的竞争——如孤独星球(Lonely Planet)的竞争，孤独星球有自己的一系列指南和移动应用程序。因此，加缪和他的团队决定，与其竞争，不如加入一家重量级公司更有意义。

对于 TouristEye 来说，此次收购将为其带来更多资源，包括访问《孤独星球》的内容，以及借助其在旅游行业的庞大分销网络。与此同时，随着用户慢慢从购买昂贵、沉重、有时甚至过时的旅游指南转向随身携带应用程序，孤独星球获得了一些移动专业知识。

在每种情况下，团队都希望利用他们的社区和数据来改善用户的移动体验。

虽然交易条款没有披露，但加缪说，TouristEye 的投资者很高兴地离开了。该公司从 500 家初创公司、即插即用创新中心和一群欧洲天使投资人那里筹集了 50 万美元。
