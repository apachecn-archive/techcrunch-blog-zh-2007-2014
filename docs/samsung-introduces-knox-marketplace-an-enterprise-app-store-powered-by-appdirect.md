# 三星推出了由 AppDirect 支持的企业应用商店 KNOX Marketplace

由于推出了新的三星 KNOX Marketplace，三星今天正在大力推动其企业市场吸引力。这是一个特殊的以移动为重点的数字软件店面，为在大型组织内工作的技术经理提供业务应用程序、IT 供应和工具。三星和为市场提供动力的合作伙伴 AppDirect 旨在为关注 BYOD 趋势的公司提供一种保护数据安全的方法。

KNOX 允许用户将他们的工作应用程序和个人应用程序彼此分开，并通过差异化访问来尽量减少在工作中使用家庭设备带来的安全风险。KNOX marketplace 为企业提供了一种全新的、同样独特的方式来控制哪些应用程序可以在商业环境中下载和使用。它允许企业 It 经理搜索和购买 KNOX 应用程序和其他专门针对商务手机的软件，并一次性购买整个预配置的产品包。

![Samsung KNOX_search[1]](img/48ee92fc06268588d862d4115211d3a0.png)通过旨在为管理人员提供易用性的仪表板来管理已配置的应用程序，这可以配置为控制整个组织中部署的每一台支持 KNOX 的移动设备上的应用程序。AppDirect 是一家白标云服务市场提供商，它与许多合作伙伴合作，包括 T-Mobile、Comcast、Appcelerator 等。该公司去年 9 月筹集了 900 万美元的 B 轮融资，并收购了一家应用程序包装公司，这可能在一定程度上为与三星合作提供服务铺平了道路。

三星在交易中为其潜在的企业客户提供了一个不错的胡萝卜，这是关键，因为该公司继续完善其商业产品，以从苹果公司赢得更多的 BYOD 份额，并取代萎靡不振的黑莓，成为寻求移动解决方案的公司的首选平台。三星已经[讨论了其未来大规模瞄准企业移动服务的计划](https://web.archive.org/web/20221206040746/http://www.techweekeurope.co.uk/news/samsung-enterprise-knox-smartphones-mobility-132328)，该公司认为这是一个通过降低消费者对其智能手机产品的兴趣来工作的机会。
