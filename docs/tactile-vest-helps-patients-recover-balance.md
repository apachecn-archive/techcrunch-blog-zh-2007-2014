# 触觉背心帮助患者恢复平衡

加州大学洛杉矶分校的研究人员正在研究一种新设备，旨在帮助失去平衡感的病人。该装置将帮助病人康复，并学习如何重新正常活动。

该装置被缝在背心上，由病人穿着。肩膀上的加速度计可以检测身体何时旋转或倾斜，从而向佩戴者提供反馈。控制系统使用气动激活器给背心中一系列 2.5 厘米的硅胶气球充气，然后向患者提供反馈。我想成品看起来会比照片中的原型更精致一些。
