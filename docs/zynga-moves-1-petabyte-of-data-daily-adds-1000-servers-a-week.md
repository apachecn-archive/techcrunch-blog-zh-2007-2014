# Zynga 每天移动 1pb 的数据；每周增加 1，000 台服务器

我们都知道社交游戏巨头 Zynga(T2)是有史以来发展最快的科技公司之一，它将 FarmVille(T4)这样的游戏变成了主流现象。通过[国际扩张](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/06/15/confirmed-zynga-raises-150-million-more-from-softbank/)以及与[脸书](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/09/08/zynga-swallows-the-facebook-credits-pill-whole/)和[谷歌](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/07/10/google-secretly-invested-100-million-in-zynga-preparing-to-launch-google-games/)的交易，Zynga 继续其统治社交游戏市场的道路。我们对公司的[营收](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/05/03/zynga-revenue/)和其他[游戏统计](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/06/14/pincus-frontierville/)有所了解，但有一些涉及平台后端的数据没有透露。今天，Zynga 的 CTO Cadir Lee 在甲骨文全球大会上讲述了这个游戏巨头的基础设施、业务和挑战。

Lee 提供了以下统计数据:

*   全球 10%的互联网用户(大约每月 2.15 亿用户)玩过 Zynga 游戏。
*   该公司每周增加多达 1，000 台服务器，以适应不断增长的流量。
*   Zynga 的资产每天移动高达 1pb 的数据，该公司运营着自己的数据中心；使用混合私有/公共云基础架构。
*   Zynga 的技术支持 Frontierville 和 Farmville 等游戏上的 30 亿邻居连接。

公司本身一直在稳步增加员工，通过[收购](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/08/17/zynga-buys-bostons-conduit-labs/)和[新员工](https://web.archive.org/web/20230327175955/https://techcrunch.com/2010/08/13/zynga-hires-former-facebook-myspace-exec-owen-van-natta/)，现在拥有超过 1200 名全职员工，包括 13 个游戏工作室。

Lee 说公司的三个主要目标是与用户建立信任，使游戏可用，并为用户提供优质的体验。Lee 说，该公司面临的挑战包括工作负载(即必须在给定的一周内增加 1，000 台服务器)。而且需求也是一个需要缓解的因素，因为游戏发布会给平台带来大量用户。甚至功能发布也给 Zynga 的服务器带来了巨大的流量。李开复回忆起 1000 万 Zynga 用户种植的超级浆果的发布。仅在发布周末，游戏玩家就种植了 7 亿颗超级浆果。
