# Fitocracy 的网络应用程序希望你站起来，开始行动

社交健身追踪器 Fitocracy 的家伙们自从我们上次见到他们之后就一直很忙[。虽然我是这项服务的粉丝，但有时使用它可能会有失误。不要误会我的意思——它在电脑上绝对很棒，但该网站从未真正适合移动设备。谢天谢地，情况不再是这样了，因为该公司宣布他们的移动网络应用程序的第一个版本](https://web.archive.org/web/20230205015703/https://techcrunch.com/2011/06/28/fitocracy-brings-games-and-social-to-your-workouts-invites-within/)[刚刚上线](https://web.archive.org/web/20230205015703/http://blog.fitocracy.com/post/10993255833/announcing-fitocracy-mobile)。

对于外行人来说，Fitocracy 将健身跟踪与社交元素和 RPG 式的调平系统结合起来，吸引了我内心的 EXP grinder。当然，这是一个折衷的组合，但它帮助 Fitocracy 发展了相当忠实的粉丝群。

Fitocracy 的[移动网络应用](https://web.archive.org/web/20230205015703/http://www.fitocracy.com/m/play/)通过削减该服务所熟知的几乎所有社交费用而发挥作用。再见论坛、群组和状态更新—您只能访问您的活动跟踪器。从那里，你可以查找单独的练习，或者选择经常进行的活动或惯例添加到你的一天的列表中。

你的每一项练习都会给你带来一定的分数，足够让你提升自己。除了给你一种良好的身体成就感之外，提升体能水平实际上没有任何作用。移动网络应用程序还支持 Fitocracy 的成就，因此当你在一次郊游中跑完你的第一个 5 公里时，你将获得一个闪亮的小(数字)徽章。

总的来说，它很实用，虽然有点简陋。Fitocracy 在他们的博客上明确表示，移动网络应用程序仍在微调，因此偶尔可能会出现错误，但对于不介意推迟聊天的健身爱好者来说，这仍然是一个受欢迎的补充。
