# Gmail Magic 收件箱里有什么？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/05/21/whats-in-the-gmail-magic-inbox/>

发现一个新功能是否即将推出的一个几乎万无一失的方法是挖掘代码。这正是导致在 Gmail 中找到所谓“神奇收件箱”的原因。但这是什么呢？嗯，这可能只是谷歌[喜欢](https://web.archive.org/web/20221006001950/http://www.beta.techcrunch.com/2009/04/30/web-searches-in-gmail-now-feature-100-less-leaving/)在 Gmail 实验室以[极快的速度](https://web.archive.org/web/20221006001950/http://www.beta.techcrunch.com/2009/05/19/google-adds-message-translation-to-gmail/)推出的又一个漂亮但小巧的新功能。但有可能是更大的东西。

具体来说，[进行挖掘的谷歌操作系统](https://web.archive.org/web/20221006001950/http://googlesystem.blogspot.com/2009/05/gmail-magic-inbox.html)认为，该功能可能是一种根据你的社交图谱对你的 Gmail 收件箱进行排序的方式。《治罪法》中两次提到“朋友”,似乎对此有一定的可信度。据推测，这将允许您更好地过滤您的收件箱的基础上，如果你已经指定了电子邮件作为一个联系。作为一个每天都被电子邮件轰炸的人，其中大部分不是来自我真正认识的人，如果这样的功能得以实现，我会喜极而泣。我妈妈也会，因为如果是这样的话，她可能会收到我的电子邮件。

当然，其他人也在研究同样的想法。雅虎说它想把你的收件箱作为你社交图谱的一部分已经有一段时间了。微软的 Hotmail 和 Xobni 也在这一领域做着工作。但是考虑到谷歌最近为加强其庞大服务网络中的社交图谱所做的工作，Gmail 中的社交过滤器可能会非常非常有用。

用户也可能对此有安全顾虑。有些人希望他们的电子邮件客户端是完全私人的，而不是社交图的一部分。当然，谷歌已经使用 Gmail 作为你社交图谱的关键起点有一段时间了，即使你没有意识到。一年多前，谷歌 it 向谷歌阅读器推出了社交功能，根据你在 Gmail 中发邮件的人，把它认为是你朋友的人拉进来。

这被证明是一个糟糕的想法，因为你发邮件的人不一定是你的朋友。谷歌最终推出了这一功能的几项更新，以允许用户更好地定制他们的关系。这显然也是 Gmail 社交过滤器的关键部分。你需要能够将你真正的朋友与那些你过去仅仅联系过的人区分开来，或者甚至可能与很多人通信。

虽然谷歌[还没有完全确定社交功能](https://web.archive.org/web/20221006001950/http://www.beta.techcrunch.com/2009/05/04/google-reader-still-trying-to-figure-out-this-whole-social-thing-still-failing/)，但很明显该公司正在考虑这些功能——很多。你的[谷歌联系人](https://web.archive.org/web/20221006001950/http://www.google.com/contacts)，最初是 Gmail 的一部分，但后来被分离出来，是它的关键部分。

谷歌输入输出(Google I/O)将于下周举行大型开发者大会。谷歌可能会利用这次活动来展示它一直在开发的一些关键的新东西。那会不会是一个“神奇的收件箱”，也就是代码中所说的“冰箱收件箱”？我们会去找答案的。也许 Gmail 甚至会离开 beta 版——但很可能不会。
