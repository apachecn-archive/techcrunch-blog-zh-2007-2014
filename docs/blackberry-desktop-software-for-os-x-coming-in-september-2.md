# 面向 OS X 的黑莓桌面软件将于 9 月推出。！！！！)

多年来，RIM 一直被指责没有一个 OS X 本土的黑莓应用程序，但这应该会在 9 月份黑莓桌面软件[发布](https://web.archive.org/web/20221006004024/http://blogs.blackberry.com/2009/07/many-of-you-commented-on.html)时停止。该程序将完成黑莓迷所期望的一切，包括同步 iTunes 播放列表、日历、联系人、备忘录、任务，以及添加和删除软件。此外，该应用程序将能够备份和更新多个黑莓手机，就像 Windows 版本多年来所做的那样。我只想对所有的苹果黑莓用户说，是时候了。[PSGallery=90vn4upfd2]
