# 欧盟训练获得宽带

![scaledgriswold.jpg](img/3bbf0419ce7f7df7d0d4afc9df10d1ce.png)

幸运的肮脏嬉皮士！现在所有持有欧洲铁路通票的人，承蒙康涅狄格州的爸爸妈妈的好意，以及患有严重疱疹的人，承蒙马德里的保罗的好意，都可以跳上巴黎的火车，一路冲浪，冲浪，乘泰利斯的火车去阿姆斯特丹。该系统使用 GPRS 和 UMTS 以及 Wi-Fi 来提供标准的互联网连接，即使在 300 公里/小时(美国单位为 500 万英里/小时)的速度下也是如此。)
 他们现在正在提供试用版，应该会在今年秋天全面上市。所以这很棒:从巴黎到阿姆斯特丹你有 Wi-Fi，[华沙到布达佩斯](https://web.archive.org/web/20221002074526/http://crunchgear.com/2007/07/25/crunchgear-meet-up-budapest/)你有跳蚤、痢疾，而且没有睡眠。中欧，收拾好你的垃圾。
