# Twitter 正在尝试文字广告

作为无休止的商业模式猜测的主题，Twitter 今天又增加了另一个武器——当你登录时，主页上的房屋广告，以及一些搜索结果。

广告目前只在内部链接到 Twitter 搜索或 Twitter 小工具。但这并不意味着他们在测试未来可能出现的第三方广告。一年前在 T2 推出的 Twitter Japan 从一开始就包含了展示广告。

去年夏天，Twitter 首席执行官[伊万·威廉姆斯说](https://web.archive.org/web/20221207204159/http://www.beta.techcrunch.com/2008/07/15/interview-with-evan-william-summize-acquisition-api-issues-and-their-revenue-model/)他认为 Twitter 最大的收入潜力将来自对品牌商业使用的收费。但是[很明显](https://web.archive.org/web/20221207204159/http://www.beta.techcrunch.com/2008/12/16/twitter-hiring-product-manager-to-bring-in-the-revenue/)该公司将尝试许多不同的潜在流——我们认为 Twitter 搜索有巨大的[收入机会。](https://web.archive.org/web/20221207204159/http://www.beta.techcrunch.com/2009/03/05/its-time-to-start-thinking-of-twitter-as-a-search-engine/)
