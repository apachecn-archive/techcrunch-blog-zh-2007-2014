# Gmail 现在会告诉你想给谁发邮件

![](img/fad09fd9122f75989e601c36729c4d69.png)

不确定还有谁要添加到该群电子邮件中？Gmail 实验室现在有一个有用的[“建议更多收件人”](https://web.archive.org/web/20221006212333/http://gmailblog.blogspot.com/)功能，可以根据您过去分组为电子邮件收件人的人，建议您可能希望包含在群发电子邮件中的联系人。

例如，如果你经常给你的母亲、父亲、丈夫、姐妹和兄弟发送家庭电子邮件，当你开始给母亲和父亲撰写电子邮件时，Gmail 会建议添加你的姐妹、丈夫和兄弟。只有在您向电子邮件中添加了至少两个收件人后，才会触发该功能。

要启用该功能，请转到 Gmail 实验室，将其作为一项设置打开。这个聪明的功能只是 Gmail 实验室的许多漂亮创新之一，包括[离线访问、](https://web.archive.org/web/20221006212333/http://www.beta.techcrunch.com/2009/01/27/gmail-goes-offline-with-google-gears/) [联系时区、](https://web.archive.org/web/20221006212333/http://www.beta.techcrunch.com/2009/04/08/gmail-can-now-tell-which-of-your-contacts-are-awake/) [搜索建议、](https://web.archive.org/web/20221006212333/http://www.beta.techcrunch.com/2009/04/02/search-suggestions-come-to-gmail/)撤销按钮、 [多窗格查看、](https://web.archive.org/web/20221006212333/http://www.beta.techcrunch.com/2009/02/05/gmail-adds-support-for-multiple-pane-viewing/)等等。
