# 自动下沉。查看您隐藏的脸书“其他”收件箱，查找您丢失的电子邮件 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/07/02/facebook-email-sync/>

点击脸书上的“信息”,然后点击右边栏中的“其他”,你可能会发现你的 Gmail 或其他个人和工作账户中的邮件不见了。在没有给你任何警告或通知的情况下，[脸书上周更改了你的个人资料](https://web.archive.org/web/20221006180949/https://beta.techcrunch.com/2012/06/25/facebook-email-address/)，隐藏了你之前列出的所有电子邮件地址，只显示了你鲜为人知的“@facebook.com”地址。任何发送给你或你的@facebook.com 地址的信息，如果不是来自朋友、朋友的朋友或他们在脸书注册的电子邮件地址，最终都会被放在那里，很容易被错过。

现在，脸书对你首选联系信息的未经授权的更改会自动与一些移动设备的地址簿同步，所以人们甚至不会意识到他们不是在 ping 你的 Gmail，而是在向你的另一个黑洞发送消息。但是，[脸书并没有道歉，而是向读写网暗示用户只是感到困惑。是的，用户很困惑…因为脸书未经同意改变了他们的联系方式！**更新**:脸书称联系人信息同步](https://web.archive.org/web/20221006180949/http://www.readwriteweb.com/archives/facebook-blames-email-problems-on-user-confusion.php)[是一个错误](https://web.archive.org/web/20221006180949/http://www.theverge.com/2012/7/2/3132818/facebook-email-confusion-contact-sync-bug)。尽管如此，这仍然不能解决问题。

脸书为什么要这样做？因为脸书正处于苹果和谷歌争夺信息控制权的战争中。

大多数人没有意识到或者忘记了他们的另一个收件箱，这就是为什么自从两年前[首次出现以来，它就引起了很多问题。这就是为什么起初许多人和出版物认为脸书电子邮件更换的惨败导致他们的电子邮件被删除，因为它没有出现在他们主要的脸书信息收件箱中。他们没有检查他们的另一个收件箱。所以，是的，他们感到困惑，但这源于脸书未宣布的变化。](https://web.archive.org/web/20221006180949/http://www.insidefacebook.com/2010/11/16/in-depth-review-facebooks-new-message-inbox-product/)

![](img/8d51307467e027d64b3ad46c62919ff4.png "facebook-edit-visibility-email")

我不认为另一个收件箱会吞掉太多邮件，因为很少有人会主动向你不熟悉的@facebook.com 地址发送邮件。问题是，你的朋友甚至没有意识到他们在那里发送信息。他们的手机可能会自动将你的联系人文件与你的脸书个人资料同步，用@facebook 地址代替你的实际电子邮件地址，所以当他们选择按姓名给你发电子邮件时，他们永远不会看到他们 ping 的新脸书地址。

**更新**:据 The Verge 的 [Ellis Hamburger](https://web.archive.org/web/20221006180949/http://www.theverge.com/2012/7/2/3132818/facebook-email-confusion-contact-sync-bug) 报道，脸书的工程总监 Andrew Bosworth 表示，用你朋友的@facebook.com 地址替换真实电子邮件地址的联系人同步问题是一个错误，应该会在明天或更早的时候得到修复。脸书给了我们一份声明解释说

> *设备上的联系人同步通过 API 执行。对于大多数设备，我们已经验证了 API 工作正常，并且提取了与用户的脸书帐户相关联的主电子邮件地址。然而，对于某些设备上的人来说，一个错误意味着设备正在拉取添加到帐户的最后一个电子邮件地址，而不是主电子邮件地址，导致@[facebook.com](https://web.archive.org/web/20221006180949/http://facebook.com/)地址被拉取。我们正在解决这个问题，很快就会解决。之后，这些特定的设备应该提取正确的地址。*

*脸书还指出，用户有能力控制哪些邮件进入他们的另一个收件箱，他们可以将收件箱设置为完全拒绝陌生人的邮件:*

 *> *“如果你在隐私设置中指定你只想接收朋友或朋友的朋友发来的消息，那么这条消息就会被退回。我们注意到，在极少数情况下，退回给原始发件人的电子邮件可能不会被发送，因为它可能会被垃圾邮件过滤器拦截。我们正在努力确保电子邮件发件人始终收到退回邮件。”*

不管有没有窃听器，脸书，一切都完了。通过未经授权更改我们的联系信息，您侵蚀了我们的信任。虽然设备的自动同步可能不受你的控制，但这种变化现在对那些错过会议、失去客户或与朋友失去联系的人产生了真正的影响，因为他们没有在预期的地方收到电子邮件。即使修复了这个漏洞，仍存在未经同意更改可见联系人信息的潜在问题。

上周[我称之为废话](https://web.archive.org/web/20221006180949/https://beta.techcrunch.com/2012/06/25/facebook-email-address/)，并呼吁至少两个解决方案中的一个，我仍然在等待:

1.  恢复更改并重新显示每个人的电子邮件地址，除非有人在更改后手动更改了他们地址的可见性，这样您就不会意外地看到有人故意隐藏的任何地址。
2.  立即向整个用户群通知这一变化！在不提醒用户的情况下做出这样的改变是非常不负责任的。

与此同时，你应该[利用本教程来撤销脸书的更改](https://web.archive.org/web/20221006180949/https://beta.techcrunch.com/2012/06/25/facebook-email-address/)，并在你的个人资料上准确显示你想要的电子邮件地址。然后阅读为什么这种改变是苹果/谷歌/脸书控制信息传递的第一枪。
