Pixel Qi完成B轮融资，获得3M TechCrunch的投资

原文：https://web.archive.org/web/http://techcrunch.com/2011/09/12/pixel-qi-closes-out-series-b-round-with-investment-from-3m/

Pixel Qi完成B轮融资，获得3M投资

创始人兼首席执行官Mary Lou Jepson一直对Pixel Qi的低功耗、超反射液晶显示器寄予厚望。虽然这种显示器出现在像概念墨水亚当这样的小众平板电脑中，但它们还没有达到杰普森一直想要的那种大众市场渗透率。

Jepson的愿望可能很快就会实现，如果今天的声明有任何迹象的话:工业巨头3M公司正在向Pixel Qi投资一笔未披露的金额，作为该公司B轮投资的一部分。

Pixel Qi的显示器由于其高度适应性，近年来引起了相当大的反响。当背光打开时，屏幕看起来像任何其他LCD，但当背光关闭时，高反射显示器呈现eInk屏幕的外观。最终的结果是，屏幕在阳光下和在黑暗中一样好用，电池续航时间也很长。

这笔交易是通过3M公司的新风险投资部门进行的，其细节仍被掩盖着，但它应该是Jepson在圣布鲁诺运营的强心针。

“在我们与3M的合作中，我们有能力加速这项技术的大规模应用，”杰普森说。3M的投资将使Pixel Qi加快生产，这是有充分理由的:Pixel Qi将继续致力于将其显示器应用于更多面向消费者的设备，同时该技术也正在被用于数字标牌和“触摸应用”如果一切顺利，第一批面向大众市场的Pixel Qi设备将很快上市。
