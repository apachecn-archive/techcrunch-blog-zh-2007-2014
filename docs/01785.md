# Payoneer 从 Greylock 和 Carmel TechCrunch 获得 800 万美元的 B 轮融资

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/07/24/payoneer-raises-8-million-series-b-from-greylock-and-carmel/amp/>

 <amp-img src="https://web.archive.org/web/20230114050533im_/https://techcrunch.com/wp-content/uploads/2007/10/payoneer.png" alt="" class="alignleft amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 当 [Paypal 的竞争对手](https://web.archive.org/web/20230114050533/http://techcrunch.com/2007/10/22/payoneer-collects-3m-from-greylock-signs-up-istockphoto/) [Payoneer](https://web.archive.org/web/20230114050533/http://www.payoneer.com/) 已经从 Greylock 和 Carmel ventures 获得了 800 万美元的 B 轮融资。这使得该公司筹集的资金总额达到 1400 万美元。(格雷洛克[领衔 A 系列](https://web.archive.org/web/20230114050533/http://techcrunch.com/2007/10/22/payoneer-collects-3m-from-greylock-signs-up-istockphoto/))。

Payoneer 在小额支付领域相对于 Paypal 的优势在于，它[不需要银行账户进行验证，从而使国际支付变得更加容易](https://web.archive.org/web/20230114050533/http://techcrunch.com/2008/07/09/payoneers-virtual-us-bank-accounts-make-international-cash-out-easier/)。像 iStockPhoto 或 Metacafe 这样的网站可以向他们的客户发行预付费的万事达借记卡，并以电子方式将资金转移到信用卡账户。该账户可用作常规电子支付转账系统，与 Paypal 类似。

这家以色列初创公司(总部位于纽约市)将使用新的现金来资助向更多市场的扩张，并投资于检测和防止犯罪分子和恐怖分子欺诈和滥用的技术。

如果你打算在支付行业寻找一个利基市场，国际支付是你能找到的最大的利基市场。但在后 9/11 时代，这也可能是一场危险的游戏。
