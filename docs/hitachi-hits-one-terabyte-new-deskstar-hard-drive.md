# 日立推出 1tb:新 Deskstar 硬盘

![terabyte](img/c99a85844c55e3e133d77ecb02aacaec.png)

天哪，这对于一个桌面硬盘来说是很大的存储量。日立新的串行 ATA *Desktar 7K1000* 提供了一个完整的 1tb 的磁盘空间(经过一些标准的二进制转换欺骗后实际上是 931GB)，300MB/s 的传输速率，7200RPM 的主轴速度和一个巨大的 32MB 高速缓存。它也不贵——现在售价不到 350 美元。

如果你渴望一页又一页的基准测试分数，技术报告(下面的链接)测试了它的速度，它似乎表现得令人钦佩。
