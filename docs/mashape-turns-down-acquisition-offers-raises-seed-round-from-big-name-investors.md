# Mashape 拒绝收购提议；从知名投资者处筹集种子资金 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/08/mashape-turns-down-acquisition-offers-raises-seed-round-from-big-name-investors/>

API 很热。Twitter 每天吸引 150 亿次 API 调用，Saleforce.com 通过其 API 获得 50%的流量，举几个大名鼎鼎的例子。因此，这里有一个简单的想法:如果每个人和他们的母亲都开始利用 API，为什么不创建一个市场，让开发人员可以轻松地发现、分发和消费所有的 API 呢？这是一家名为 [Mashape](https://web.archive.org/web/20230203131647/http://www.mashape.com/) 的年轻意大利创业公司(被移植到旧金山)采用的思路。[正如我们在 6 月份关于这家初创公司的原始简介中所写的](https://web.archive.org/web/20230203131647/https://techcrunch.com/2011/06/02/mashape-the-etsy-of-cloud-services-goes-beta-lets-you-monetize-your-apis-in-a-click/)，简单来说:Mashape 希望通过提供一个统一的、一体化的市场，让用户可以找到、销售、分发和破解 API，从而变得有点 Etsy，有点 Github。

当这家初创公司 2009 年抵达美国时，它很快从几个天使和风投那里筹集了 10 万美元。现在越来越多的投资者开始买入，他们的名字在科技投资界也很熟悉。该公司昨天宣布，它已经筹集了 150 万美元的种子资金，由[新企业协会](https://web.archive.org/web/20230203131647/http://www.crunchbase.com/financial-organization/new-enterprise-associates)牵头，参与方包括[指数风险投资](https://web.archive.org/web/20230203131647/http://www.crunchbase.com/financial-organization/index-ventures)、[查尔斯河风险投资](https://web.archive.org/web/20230203131647/http://www.crunchbase.com/financial-organization/charles-river-ventures)、[点火伙伴](https://web.archive.org/web/20230203131647/http://www.crunchbase.com/financial-organization/ignition-partners)和埃里克·施密特的[创新努力](https://web.archive.org/web/20230203131647/http://www.crunchbase.com/financial-organization/innovation-endeavors)。包括亚马逊总裁兼首席执行官杰夫·贝索斯在内的几位天使投资人也参与了此轮投资。

这是一个令人印象深刻的投资者名单，特别是考虑到 Mashape 去年 6 月在私人测试中推出，至今仍未正式公开推出。Mashape 首席执行官 Augusto Marietti 还告诉我，在种子期之前，这家初创公司收到了一家“大型上市公司”和一家“中型”公司(约 200 名员工)的收购要约。创始人拒绝了这两个提议，因此不愿意透露姓名，以寻求风险和天使投资，让它继续作为一个独立的实体运营。

更重要的是，Marietti 说，这家初创公司已经开始产生收入，在三个月内已经从 110 个 API 跃升至近 200 个——包括公共和私有 API。

我说的公有和私有是什么意思——收入来自哪里？Mashape 上列出的 API 既有免费的，也有收费的，因此有公有的和私有的，还有 Mashape 最初的货币化策略。但该公司也想帮助开发商赚钱。Mashape 提供了一个简单的(免费的)工具，让用户能够将 JSON APIs 货币化，例如，允许他们列出一个 API，然后单击一下就可以设置账单，选择每月费用、API 调用限制，并给出每次额外调用的价格。

“如果我们看看那些超过 30 人的初创公司”，Marietti 说，“我们注意到他们中的大多数都在招聘两个新职位:API 布道者和 API 工程师。这些职位在 3 年前是不可能存在的。一个新的市场正在形成”。

毫无疑问，API 正在走向成熟，但 Mashape 也希望其市场能够利用 Web 2.0、3.0 或思想领袖希望我们去的任何地方的其他成熟方面。因此，Mashape 正在其 API 之上构建一个社交层，以解决不断增长的 API 市场固有的可发现性和信任问题，允许用户看到他们的朋友在使用什么。

Twitter 等更大的网站还没有购买 Mashape，并且承认初创公司正在一个由企业解决方案组成的市场上玩，但随着这些功能的到位，初创公司准备好利用由初创公司、中小企业和大公司构建的社区——每当他们最终决定上钩时。

对于创业公司和开发者来说，另一个有新闻价值的消息是: [Mashape 今天通过其博客](https://web.archive.org/web/20230203131647/http://blog.mashape.com/mashape-announces-to-be-a-proud-partner-of-th)宣布，它将成为 Rapleaf 新的“个性化基金”的合作伙伴，这是一个 100 万美元的基金，“致力于支持开发者围绕数据和个性化创建项目”。据 Marietti 说，Mashape 社区成员将优先获得 Rapleaf 的基金资源。

[正如 VentureBeat 最初报道的](https://web.archive.org/web/20230203131647/http://venturebeat.com/2011/09/06/rapleaf-personalization-fund/)，Rapleaf 正在寻求通过其新的计划资助 50 到 100 家初创公司，该计划提供 1.5 万美元的现金资助、免费数据和服务。该公司告诉 VB，被选中参与的初创公司将获得 Rapleaf 的合作伙伴网络、客户和联合营销机会。
