# 索尼爱立信澄清 Xperia 冰淇淋三明治计划

除了在脸书用荷兰语概述其计划[，索尼爱立信今天在其博客](https://web.archive.org/web/20230203172730/https://techcrunch.com/2011/10/20/sony-ericsson-plans-to-release-android-4-0-to-the-xperia-line-in-2012/)上澄清了其将冰淇淋三明治带给携带 Xperia 的大众[的立场。如果你还没有痴迷地阅读更新，我将继续提到几个](https://web.archive.org/web/20230203172730/http://blogs.sonyericsson.com/products/)[的新功能](https://web.archive.org/web/20230203172730/https://techcrunch.com/2011/10/18/a-quick-ice-cream-sandwich-feature-rundown/)，包括灵活的小工具、Roboto 字体、截屏功能、改进的通知和面部解锁。幸运的是，如果你的手机是在 2011 年推出的，并且名字中有“Xperia ”,那么冰激凌三明治正朝你走来。

以下是官方声明:

> 在博客和我们的支持论坛上有一些关于我们升级计划的问题。我们今天可以确认，我们计划将整个 2011 Xperia 产品组合*升级到下一个版本的 Android，即 Android 4.0 或冰激凌三明治。我们正在努力将我们当前的 Xperia 体验与 Android 4.0 中的新功能相融合。关于此次升级、时间和全球可用性的更多详细信息将在适当的时候在博客上公布。
> 
> *“2011 Xperia 产品组合”是指以下产品:Xperia arc 和 arc S、Xperia PLAY、Xperia neo 和 neo V、Xperia mini 和 mini pro、Xperia pro、Xperia active、Xperia ray 以及索尼爱立信 Walkman Live。

SE 最初通过其荷兰脸书页面发布了模糊的升级计划，但没有像这次一样澄清“2011 Xperia 组合”的含义。不幸的是，手机制造商没有提供更新可用性的时间表。我们一知道，你也会知道。
