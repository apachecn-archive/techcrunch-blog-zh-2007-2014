# 凭借《愤怒的小鸟》商品，Rovio 公司的彼得·维斯特巴卡计划击败海盗

正如愤怒的小鸟毛绒玩具系列的巨大成功所证明的那样，世界对真实世界的愤怒的小鸟产品有着强烈的需求。没有比北京更明显的地方了，那里无数玩具店和电子商店的大厅里堆满了各式各样的鸟主题商品，从运动衫和牛仔裤到糖果和气球。有什么问题吗？Rovio 没有授权。

在别人可能会看到问题的地方， [Rovio](https://web.archive.org/web/20230203145249/https://techcrunch.com/tag/Rovio) Mobile 的雄鹰(读作:CMO) Peter Versterbacka 看到了机会。他在上面提到的这些商店里游荡，寻找未经授权的商品——但不是为了抱怨或乱扔合法的订单。他在那里寻找最好的非官方愤怒的小鸟物品…所以他可以复制它们。

维斯特巴卡带着三个愤怒的小鸟气球，加入了我们的约翰·比格斯在《扰乱北京》的舞台上进行炉边谈话。当被问及气球时，彼得回答道:

“我要把这些气球带回去，告诉我们的人去做这些。”

这种打击海盗的行为，似乎成了彼得的一种习惯。

他为什么不能这么做？毕竟，被利用的知识产权是他的公司的。如果有人想从他团队的工作和创造力中快速获利，没有理由不扭转局面。
