# Samwer Square 的克隆产品 Payleven 通过现在在欧洲销售的芯片和 PIN 阅读器克服了签证障碍 TechCrunch

来自 Sam wer brothers ' Rocket Internet incubator 的移动支付公司 Payleven ，现在正在接受其芯片和密码阅读器的订单，这使其成为欧洲第一个提供快速方式以符合该地区 Visa 卡支付要求的 Square 式竞争对手之一。读者于 2012 年 10 月在[首次公布；今天，Payleven 开始接受订单——每个价格为 49€(65 美元)——并将于 2 月 18 日开始发货。](https://web.archive.org/web/20230201103624/https://techcrunch.com/2012/10/03/the-founders-of-rocket-internets-square-clone-payleven-on-technology-and-taking-mobile-payments-mass-market/)

“我们在一个技术解决方案上投入了大量资金，作为一个商家，你可以从一开始就接受一切，”Payleven 的首席技术官和联合创始人拉斐尔·奥特罗在一次采访中告诉我，当时该公司于 10 月在[首次宣布其解决方案。](https://web.archive.org/web/20230201103624/https://techcrunch.com/2012/10/03/the-founders-of-rocket-internets-square-clone-payleven-on-technology-and-taking-mobile-payments-mass-market/)

读卡器将首先在 Payleven 的欧洲市场推出:英国、德国、意大利、荷兰和波兰。Payleven 还在巴西等其他地区扩张，一位发言人表示，这些阅读器以后也会在那里使用。像其竞争对手一样，Payleven 向商户提供 2.75%的交易统一费用，截至今天，它与 Visa 和 MasterCard 合作，但不包括美国运通或其他支付提供商。

背景:芯片和密码交易依赖于嵌入信用卡或借记卡的芯片以及用户的密码来认证支付，这些现在是欧洲的标准，不像美国，磁条和签名仍然是亲自认证卡支付的主要方式。在欧洲，去年当 Visa 在大多数国家停止允许交易时，其他方形公司如 [iZettle](https://web.archive.org/web/20230201103624/https://techcrunch.com/2012/07/30/visa-europe-not-squares-investor-visa-inc-puts-brakes-on-izettle-as-mpowa-expands-to-android/) 受到了打击，因为他们没有芯片和密码解决方案。目前，他们的解决办法是让买家到他们手机上的支付网站验证销售点支付——这是一个解决方案，但更耗时。像 mPowa 这样的其他公司也开发了芯片和密码阅读器，与他们的移动应用程序一起工作；它的解决方案被认证为 PCI(支付卡行业)1，而 Payleven 将其描述为 3.1。

支持 Visa 支付是这些移动支付公司等式中的重要部分:Visa 在欧洲拥有约 4.5 亿张借记卡、信用卡和商务卡，涵盖 2012 年约 1.7 万亿€的支付，或欧洲销售点所有消费者支出的 14%。其中大约 80%是通过借记卡而不是信用卡进行的，而对于借记卡来说，PIN 是最重要的身份认证(而签名可以在信用卡上使用)。

鉴于 Square 及其欧洲同行都来自类似的使命——让较小的商户能够进行信用卡交易——这是在欧洲实现这一目标的重要一步。

“我们欢迎我们今天在欧洲看到的创新，为传统上不接受信用卡的小企业提供安全的信用卡接受解决方案，”SVP Visa 欧洲支付系统风险部的斯坦利·斯克格隆在一份声明中指出。“这种解决方案有助于提高卡的接受度，同时保留重要的芯片和 PIN 功能及其提供的所有安全优势。我们很高兴看到解决方案的出现，可以将芯片和密码从商店带到小商户手中。我们感到加倍高兴的是，EMV 的覆盖面和灵活性正在以负责任的方式扩大，让小型企业有机会提供与大型企业同等水平的安全卡接受度。”

在 Payleven 的解决方案中，智能手机或平板电脑仍然被用作认证过程的一部分，但用户不是将卡刷入手机上的加密狗，而是将它放入一个额外的小机器(如图)，通过蓝牙与它的[安卓](https://web.archive.org/web/20230201103624/https://play.google.com/store/apps/details?id=de.payleven.androidphone)或 [iOS 应用](https://web.archive.org/web/20230201103624/http://ax.search.itunes.apple.com/WebObjects/MZSearch.woa/wa/search?entity=software&media=all&restrict=false&submit=seeAllLockups&term=payleven)连接。

这个消息是在 Payleven 宣布[又一轮融资](https://web.archive.org/web/20230201103624/https://techcrunch.com/2013/01/20/payleven-the-samwers-answer-to-square-takes-a-high-single-digit-million-dollar-round-led-by-a-mysterious-new-investor/)的几周之后发布的。按照 Rocket 的标准做法，投资者和投资金额都没有透露，只是指出投资金额在“数百万美元以上”。除了 Rocket Internet 之外，该公司的指定投资者还包括 New Enterprise Associates、Holtzbrinck Ventures 和 ru-Net，它们共同向 Payleven 投入了“两位数百万美元”。

Payleven 成立于 2012 年 3 月，迄今尚未透露其服务的用户数量。虽然包括 Alston Zecha 和 Alexander Zumdieck 在内的创始人正在将公司建设成一个独立的实体，但 Samwers 也因建立模仿美国竞争对手商业模式的电子商务业务而闻名，这些竞争对手最终收购了他们，以实现无机的国际增长。此类退出包括出售给 Groupon 的 CityDeal 阿兰多，卖给了易贝；Betreut 卖给了 Care.com。
