# SurveyMonkey 首席执行官大卫·戈德堡(David Goldberg)谈到进军伦敦打造全球企业业务

> 原文：<https://web.archive.org/web/http://techcrunch.com/2013/12/05/surveymonkey-ceo-david-goldberg-on-swinging-into-london-to-build-out-a-global-enterprise-business/>

在线反馈和调查公司 SurveyMonkey 成立 14 年，其首席执行官大卫·戈德堡说，它“不再是一家初创公司”。但过去的一年标志着你可能会称之为这家硅谷公司的第二个童年:大量融资(一些[、8.5 亿美元](https://web.archive.org/web/20230324010246/http://www.techmeme.com/130116/p69#a130116p69)，都是次级融资)；一个全新产品分支的推出([企业](https://web.archive.org/web/20230324010246/https://techcrunch.com/2013/11/18/surveymonkey-enterprise/))；现在，它的 hattrick 开始了新的国际扩张。正是第三个原因，让戈德堡——一位经验丰富的企业家、投资者和脸书·首席运营官·雪莉·桑德伯格的丈夫——来到伦敦，在伦敦大桥附近的新摩天大楼“碎片大厦”( Shard)的一家酒吧与我会面。

我们在这里开会，是因为尽管 SurveyMonkey 宣布在伦敦设立新办公室，并计划招聘 50 名销售、营销和业务开发新员工，但它还没有找到自己的地方。公司会去哪里找？答案仍然悬而未决。不要住在租金太高的豪华公寓里；不一定是在肖尔迪奇，许多其他创业公司已经在那里扎根。

事实证明，碎片大厦——伦敦就在我们脚下——是一个合适的会面地点。调查猴子带着它眼中的机会来到这里。伦敦将是其在美国以外的最大投资，这是其更大企业推动的直接结果。这也是英国政府将更多科技企业带到伦敦的共同努力的一部分，包括鼓励新的工作签证计划、更快的宽带和更多的公司承诺在这里投资。

今天，英国是 SurveyMonkey 仅次于美国的最大市场，拥有 150 万用户，这一地位是建立在国民医疗服务体系、石油公司 BP、地方议会和大学以及“100%的富时 100 指数”等组织的企业和客户基础上的。

其中许多都是从个人账户开始的，因此 SurveyMonkey 希望通过其新的企业授权计划来加强和扩大这些联系；此外，通过更多的分析服务来充分利用调查收集的数据。

当我引起戈德堡的注意时，我得到了 SurveyMonkey 上可能发生的其他事情的最新消息:

— **收购**。这是美国公司拓展海外业务的一种显而易见的方式，更不用说是一种学习新技术的方式，所以 SurveyMonkey 会走这条路吗？

戈德堡说，目前还没有为了扩大客户规模而进行的收购。“那里没有任何规模的东西，”他告诉我。“这并不是说我们不会买东西。我们一直在国际上寻找，但什么也没找到。”

至于技术收购，那就是另一回事了。SurveyMonkey 的关键在于，它与企业一起推出了一个新的业务领域，这可能会导致它提供新产品和服务，其中一些可能会在内部构建；有些不会。

其中一个领域可能是数据，特别是大数据分析。“我们如何收集数据，如何帮助人们做出决策？”戈德堡反问我。“把我们所做的事情称为调查是非常狭隘的。这是数据，这是一个非常非常大的空间。”

数据智能也可能是 SurveyMonkey 保持与谷歌调查和(在较小程度上)Formstack 等竞争的一种方式。

尽管 SurveyMonkey 认为公共部门是其服务的强大垂直市场，但 Goldberg 排除了英国民调公司 YouGov 作为目标的可能性。“这不是我们想要的空间，”他说，并指出 SurveyMonkey 事实上已经有了自己的面板业务。

还有一个计划是通过 SurveyMonkey 的 API 继续扩大合作和集成。像这样的整合是一个重要的客户保持工具:“那些使用我们，也使用 MailChimp 和 Eventbrite 的人以较低的比率从我们这里流失，”Goldberg 指出。

——**资助**。SurveyMonkey 今年早些时候筹集的资金——其中[包括来自 Tiger Global、Social + Capital 和其他公司的大量捐款——是“100%次要的”，他说，这些资金流向了员工和投资者。](https://web.archive.org/web/20230324010246/http://www.crunchbase.com/company/surveymonkey)

“没有一笔钱进入公司，”他补充说，这意味着它将进行的任何收购都将来自自己的现金储备，如果需要的话，还会有债务。他告诉我，2012 年，SurveyMonkey 的息税折旧及摊销前利润(EBITDA)为 6200 万美元，收入为 1.13 亿美元。它没有披露 2013 年的收入。他指出，这一轮融资对该公司的估值约为 13.5 亿美元，这将把 IPO 的问题推迟一段时间。
