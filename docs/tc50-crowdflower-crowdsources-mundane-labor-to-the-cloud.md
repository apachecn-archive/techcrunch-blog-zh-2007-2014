# TC50: CrowdFlower Crowdsources 将平凡的劳动外包给云 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/09/15/tc50-crowdflower-crowdsources-mundane-labor-to-the-cloud/>

![](img/485117283a93c305eeac15d1d8630b35.png)

对于任何企业来说，从初创公司到大公司，转录视频或从博客中删除垃圾评论等平凡而耗时的任务可能会浪费员工的时间。 [TechCrunch50](https://web.archive.org/web/20230125053641/http://www.techcrunch50.com/2009/crowdflower) 初创公司 [CrowdFlower](https://web.archive.org/web/20230125053641/http://crowdflower.com/) CrowdFlower 通过让客户获得始终在线、可扩展的劳动力来提供劳动力即服务(LaaS)。

与传统的外包和招聘方式不同，CrowdFlower 的网络服务让企业无需拿起电话就能立即联系到数千名员工。它是这样工作的。如果一家公司有转录任务，管理员可以在 CrowdFlower 上填写任务表格。填写完所有字段后，CrowdFlower 将根据每单位执行任务(用户执行示例任务)所需的时间来为任务定价。初创公司会将任务分解成可以由一个人完成的单元，并相应地为任务定价。

然后 CloudFlower 让你选择工作的渠道。你可以选择亚马逊“土耳其机器人”,他们的劳动力，甚至是一个脸书应用程序，在那里人们执行任务来赚取虚拟货币。基于工人犯的错误，系统对工人的信任度降低。一旦任务完成，CloudFlower 将执行质量控制检查，并向用户显示任务的结果。初创公司会指出哪些员工效率最高、最准确。CrowdFlower 将继续雇佣评级高的员工。

CrowdFlower 通过从客户创建的每项任务中收取佣金来实现其平台的盈利。

**专家组 Q & A(转述)**

**专家:萨迪什·法拉杰、利奥·佐雷亚、布拉德利·霍洛维茨、蒂姆·奥莱利、凯文·罗斯**

SD:我认为这很有趣——你会把语音邮件翻译成文字吗？大概想考虑一个垂直的？

答:我认为有很多无聊的工作人们不会外包出去。

LZ:我认为这是一个伟大的想法。你如何保持竞争优势？

答:我们已经有大量的工人，我们知道谁能做好哪些工作。我们的人可以在日常工作中做到 99%的准确率。

BH:你在机械土耳其人身上做的很多事情都是可能的。这是他们要去的地方吗？

答:对“土耳其机器人”来说，困难在于他们不想知道结果是什么。当我们做一项任务时，我们有更高层次的语义，我们有更高层次的质量控制。亚马逊可能会将其视为一个平台之上的应用程序。

TO:这里的利基有多大？关于这一点还有很多无知？

答:有很多有趣的企业任务可以完成。比如 MySpace。存在安全性和集成等挑战。但最大的挑战是质量。

我们不知道如何接触小客户。

基米-雷克南:服务是令人敬畏的，但需要弄清楚你如何将信息传递出去。
