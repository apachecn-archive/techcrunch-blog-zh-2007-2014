# 诺基亚将 N-Gage 移动游戏的未来押在 N 系列 Ovi 平台上

[![Ngage](img/53c150a1799d9bd3a9fa5e926fc1b33a.png)](https://web.archive.org/web/20210412222431/http://www.flickr.com/photos/28429352@N00/1266161488/ "Photo Sharing")

N-Gage 有一些像样的游戏。当然，旧平台的主要问题是你必须买一部非常特别的虫火谷-塔科伊手机。他们在全世界只推出了大约 300 万个这样的傻瓜，这意味着他们远远没有达到创造一个有利可图的游戏生态系统所需的终端速度。嗯，已经有数千万部 N 系列手机散布在世界各地——这是开始他们新平台的一个不错的地方，不是吗？由于这些 N 系列手机中的一些拥有高达 8g 的内存，它们更多地依赖于下载播放式系统，而不是过去那种基于墨盒的系统。游戏价格从 6 欧元到 10 欧元不等(按照当前汇率，相当于 90 到 170 美元)，可以日租或周租，也可以完全免费试玩。

值得注意的是，诺基亚一直保持着 N-Gage Arena 在线游戏服务的活力，所以 N-Gage 的狂热分子应该会为这次更新感到高兴。

当然，所有这些下载都将通过全新的 [Ovi 平台进行。](https://web.archive.org/web/20210412222431/http://crunchgear.com/2007/08/29/nokia-launches-ovi-mobile-platform-gaming-maps-music-umbrella/)
