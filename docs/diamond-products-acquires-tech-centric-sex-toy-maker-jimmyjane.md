# 钻石产品公司收购以科技为中心的性玩具制造商 Jimmyjane

[钻石产品](https://web.archive.org/web/20230404010443/http://brookstonepartners.com/us/portfolio/diamond-products)，一家以游乐产品为中心的控股公司，宣布收购 Jimmyjane。交易条款没有披露。

Jimmyjane 正处于性玩具和技术的交叉点，开发下一代娱乐产品，如 Form 6 振动器和 HelloTouch 振动指垫。这些产品不仅整合了技术启发的设计实践，如无线充电和极简主义，而且它们完成了工作。

[相信我。](https://web.archive.org/web/20230404010443/https://techcrunch.com/2012/11/16/gift-guide-jimmy-jane-form-6-massager/)

钻石产品公司与纽约私募股权公司 [Brookstone Partners](https://web.archive.org/web/20230404010443/http://brookstonepartners.com/us/portfolio/diamond-products) 联手，还收购了另一家性玩具、内衣、娱乐产品公司 [Pipedream Products](https://web.archive.org/web/20230404010443/http://www.pipedreamproducts.com/main-22b.phtml) 。

两家公司(Jimmyjane 和 Pipedream)现在将携手合作，“为人们带来快乐。”
