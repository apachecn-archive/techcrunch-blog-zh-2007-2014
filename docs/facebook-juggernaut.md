# 有什么能阻止脸书的强大吗？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/11/25/facebook-juggernaut/>

![](img/e48d21ba8c785375962ba230593a3644.png)

所以。脸书。350 亿美元估值； [6 亿用户](https://web.archive.org/web/20221208061451/http://www.unifiedstream.com/facebook-reaches-600-million-users-bloomberg-profiles-mark-zuckerberg/)；25%的美国网络流量——而且所有员工比谷歌少的网站都有职位空缺。万维网的发明者最近警告说，网络可能会受到脸书巨大围墙花园的威胁。一名谷歌工程师最近获得了 350 万美元的报酬，以避免跳槽到那里工作。脸书似乎是一个不可阻挡的巨人。我有点想让他们死。

不是因为他们的政策。他们对用户的需求相当敏感，并愿意承认他们的错误(还记得脸书灯塔报吗？)最近出现了一些令人担忧的迹象，例如，他们对数据可移植性的两面派态度，以及他们对“Face”一词的注册商标，但我并不反对他们的所作所为。

我不喜欢脸书，因为他们很平庸。他们有一个不同于其他任何人的平台和机会，*—*他们做了什么？没什么。他们所谓的创新实际上一点也不像。复制 Twitter 是聪明的，但并不新鲜；Foursquare 也是如此。他们称脸书集团为创新；这是他们几年前就应该实现的基本功能。现在，他们可笑地声称[将电子邮件](https://web.archive.org/web/20221208061451/https://beta.techcrunch.com/2010/11/15/facebook-messaging/)整合到他们的信息系统中是一场震撼世界的革命。

像往常一样，威廉·吉布森说得好:“脸书感觉像一个购物中心。Twitter 给人的感觉就像大街上一样。”(我想这让 Zynga 成为了购物中心的街机。)偶尔去那里购物是一回事，但成为一个成熟的 mallrat 完全是另一回事——根据统计数据，这就是我们集体变成的样子。我想相信，最终我们会醒来，长大，并意识到新的和有趣的事情大多发生在其他地方。

因此，我满怀希望地推测:如果脸书是新的现场直播杂志会怎样？

你可能不记得现在奄奄一息的社交博客网站 [LiveJournal](https://web.archive.org/web/20221208061451/http://www.livejournal.com/) ，但是马克·扎克伯格记得:社交网络*的第二个场景描述了他在他的 LiveJournal 上直播一场黑客大战。(与电影的大部分内容不同，这个场景基本上是真实的。)那时我也在 LJ 上，主要是为了在地球上四处奔波时与我的加利福尼亚朋友保持联系。现在他们的账户变成了一座鬼城——虽然大多数人都搬到了脸书，但他们在那里的活动却少得多。他们并不孤单:LJ 的 [own](https://web.archive.org/web/20221208061451/http://web.archive.org/web/20041208071300/www.livejournal.com/stats.bml) [stats](https://web.archive.org/web/20221208061451/http://www.livejournal.com/stats.bml) 显示，虽然他们的用户基数增长了，但总用户活动实际上却下降了。*

如果 LJ 的衰落是脸书的警钟呢？如果人类的自然倾向是人们最初对社交网络感到兴奋和痴迷，但最终，几年后，他们越来越厌倦它，并开始慢慢疏远它，会怎么样？

这是一个可检验的假设。关键的统计数据是一个人成为脸书用户的时间长短和他在这个网站上花了多少时间之间的关系。不过，只有脸书知道这些数字，他们也没有说话。在此之前，我可以抱着这个希望。。。

——但问题是:这都不重要。脸书*仍然*无法停止。

即使我关于开明的自我实现在全球激增的启示录式预言成真，我们对脸书的集体痴迷开始消退，它仍将是一个强大的巨人。因为马克·扎克伯格也记得 LiveJournal，他和他的智囊团已经确保脸书将仍然是不可或缺的，即使他们的用户开始失去兴趣。

它不再仅仅是一个网站:像亚马逊或谷歌一样，脸书已经成为一个公用事业。这不是比喻。据我所知的所有初创公司/开发者称，依赖脸书连接及其 Graph API 的应用和网站数量正在飙升(见鬼，这里还有一些实际数据。)就连曾经强大的 MySpace [上周也向脸书连线网站交出了 T3。谷歌](https://web.archive.org/web/20221208061451/https://beta.techcrunch.com/2010/11/18/hell-freezes-over-as-myspace-fully-surrenders-to-facebook/)[三心二意的阻止他们的尝试](https://web.archive.org/web/20221208061451/https://beta.techcrunch.com/2010/11/09/facebook-slaps-google-openness-doesnt-mean-being-open-when-its-convenient/)太少，太晚。

脸书之于社交网络，就像微软之于桌面:庞大得令人难以置信，平庸得令人难以置信，而且几乎不可避免。就像 20 年前的微软一样，他们将会成功，因为糟糕的标准总比没有好:就像 10 年前的微软一样，他们通过笨拙地复制——然后试图挤压——真正的创新者来“创新”。

因此，让反弹繁荣！也许这最终会刺激扎克伯格公司用他们的无敌机器做一些真正有趣和创新的事情。
