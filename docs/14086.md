# Zynga 首席执行官马克·平卡斯证实裁员:5%的员工，英国、日本办公室可能关闭

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/23/zynga-ceo-mark-pincus-confirms-layoffs-5-of-workforce-potential-closures-for-u-k-japan-offices/>

Zynga 证实，在预计将发布疲软的第三季度收益报告之前，该公司开始裁员。该公司总共有 5%的全职员工被解雇。截至第二季度，该公司有 3200 名员工，因此最终人数可能接近 160 名。

关键细节是，该公司正在取消 13 款游戏，并大幅减少对其生活模拟游戏《The Ville》的投资，正如我们之前报道的那样,[该游戏正被送往 Zynga 的印度办事处进行维护](https://web.archive.org/web/20230329003314/https://techcrunch.com/2012/10/23/zynga-layoffs-boston-austin/)。

正如我们之前报道的那样，该公司还证实将关闭波士顿办事处。Zynga 还提议关闭其英国和日本工作室。

实际上，Zynga 正在关闭或考虑关闭的大多数办公室都是从收购开始的。 [Zynga Boston 曾经是 Conduit Labs](https://web.archive.org/web/20230329003314/https://techcrunch.com/2010/08/17/zynga-buys-bostons-conduit-labs/) ，而[Austin office 来自 Challenge Games 的收购](https://web.archive.org/web/20230329003314/https://techcrunch.com/2010/06/04/sec-filing-suggests-zynga-paid-at-least-20-5-million-for-challenge-games/)和[英国的开发工作来自收购 Wonderland Software 的交易](https://web.archive.org/web/20230329003314/https://techcrunch.com/2011/04/27/zynga-continues-international-expansion-acqhires-uk-mobile-gaming-studio-wonderland-software/)。Zynga 日本公司从 Unoh 收购案中走出来。

Zynga 本月早些时候表示将下调今年的收入预期时，暗示了即将到来的裁员。Zynga 表示，预订量(即用户为其游戏中的虚拟商品花费的金额)将在 10.85 亿美元至 11.00 亿美元之间，低于 11.50 亿美元至 12.25 亿美元。

该公司面临着充满挑战的一年，因为它在脸书平台上的先发优势开始瓦解。其他来自欧洲的游戏开发商，如 King.com 和 Wooga，在过去的一年里抢占了休闲游戏市场的份额，而 Kixeye 等中端开发商则开始腾飞。脸书在今天的财报电话会议上表示，其结果是，Zynga 对脸书总收入的贡献从一年前的 12%下降到了 7%。与此同时，许多游戏玩家已经转移到移动设备上，而 Zynga 在移动设备上并没有如此确定的主导地位。

在过去的一年里，投资者让 Zynga 的股价下跌了 80%或更多，他们质疑该公司的论点，即它拥有如此庞大的网络和规模经济，以至于它更容易受到游戏业务的点击驱动性质的影响。Zynga 的市值为 16.7 亿美元，与第二季度末手头的现金、短期和长期投资总额大致相当。然而，盘后交易者对裁员消息反应积极。Zynga 股价上涨 4.1%，至 2.29 美元。

过去一年，Zynga 的管理层也发生了巨大变化。艺电高管约翰·沙佩特(John Schappert)来到该公司工作了一段时间，在此期间，该公司推出了《Farmville 2》和《黑手党战争 2》等续集。在某些情况下，这些续集并没有成功，因为黑手党战争 2 疏远了游戏的原始玩家群。没过多久，Schappert 和他从 EA 带来的许多中层管理人员都被排挤出去了。与此同时，许多 Zynga 的长期员工在获得授权后离开，耗尽了公司的原始人才。
