# 出乎意料的是，Instacolor 被迫用“真实”的颜色来改名

这是一部黑色喜剧，涉及白热化的初创公司 [Color](https://web.archive.org/web/20230204163123/http://www.crunchbase.com/company/color-labs) 和仍然稚嫩的 [Instacolor](https://web.archive.org/web/20230204163123/http://www.gramfeed.com/instagram/instacolor) 。面红耳赤的是，后一个应用程序背后的[开发者](https://web.archive.org/web/20230204163123/http://twitter.com/#!/krisrak)，它结合了 [Instagram 和](https://web.archive.org/web/20230204163123/https://techcrunch.com/2011/07/08/instagram-color-instacolor/)Color 的关键功能，在颜色乡亲们对 Instacolor 名称产生灰色情绪后，被迫将其名称改为 Instalook。

本周早些时候，Color 的律师向 insta color insta look 的开发者发送了一份停止通知，要求他更改 iPhone 应用程序 pronto 的名称。

在信中，Color 的法律顾问写道:

> 如您所知，Color labs 是 Color 应用程序的创建者和所有者，这是一个支持 GPS 的照片共享应用程序，可以从 iTunes 和其他地方下载。Color Labs 花费了大量的时间和资源来提高消费者对彩色商标和产品的认知度和好感度。它也是美国商标申请号 85/222，392 的所有者，该申请的副本附后。
> 
> 虽然我们的客户赞赏您对其产品的热情，但我们希望您能够理解，未经授权的对其商标和软件的批发使用侵犯了其商标和版权，也违反了您在下载彩色应用程序时同意的使用条款。
> 
> 这种未经授权的使用错误地暗示 Color Labs 赞助或支持您的软件，并且它以某种方式附属于您的软件。鉴于 Color Labs 无法控制您的产品质量，我们相信您能够理解由此引发的重大问题。

在信的后面，Color 表示，如果开发商 Rakshith Krishnappa 未能“立即停止所有未经授权使用 Color labs 商标的行为”，它准备诉诸法律。

我猜想，让 color 恼火的不仅仅是 Krishnappa 在应用程序的名字中使用了“Color ”,并为此收费。

在被要求发表评论时，一位色彩代表回应道:

> “由于使用了 Color 的名称，并模仿了我们的应用程序和徽标，我们担心潜在的消费者混淆。”

作为旁观者，我可以理解为什么 Color 发出了停止信，但我也为 Krishnappa 感到难过，他开发这款应用程序是作为一个周末项目，没有 4100 万美元的资金支持。不管怎样，Krishnappa 明白他必须更改名称——他立即这么做了——但断然否认模仿 Color 的 iPhone 应用程序的外观和感觉。

他还解释说，他对应用程序收费只是因为 Instagram 有每小时 5000 个 API 的限制，他想确保每个使用该服务的人都有良好的体验。

自从 [Color](https://web.archive.org/web/20230204163123/http://www.color.com/) 推出其[照片分享应用](https://web.archive.org/web/20230204163123/https://techcrunch.com/2011/06/15/what-photo-sharing-app-should-you-use/)以来，这家[4100 万美元](https://web.archive.org/web/20230204163123/https://techcrunch.com/2011/03/23/color-looks-to-reinvent-social-interaction-with-its-mobile-photo-app-and-41-million-in-funding/)的初创公司就一直处于[艰难时期](https://web.archive.org/web/20230204163123/https://techcrunch.com/2011/06/25/fly-or-die-color-ishtar/)。据首席执行官比尔·阮(Bill Nguyen)称，联合创始人彼得·范(Peter Pham)离开了(T9)，或者被解雇了，他还告诉《纽约时报》(New York Times)记者(T10)，该公司将重新开始(T12)。

他们现在最大的挑战是:[似乎没有人在使用这个应用](https://web.archive.org/web/20230204163123/https://twitter.com/#!/kteare/status/72765971881922560)。
