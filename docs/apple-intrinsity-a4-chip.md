# 苹果继续变成巧克力工厂。预计堵塞泄漏会减少。

除了丢失的 iPhone 原型之外，大多数苹果泄密事件往往来自合作伙伴公司，他们的任务是帮助制造苹果特定产品所需的东西。毕竟苹果自己也做不到这一切。反正现在还没有。

今天《纽约时报》的一篇[报道](https://web.archive.org/web/20230208151519/http://www.nytimes.com/2010/04/28/technology/28apple.html?src=twt&twt=nytimesbusiness)证实了早先的[传言](https://web.archive.org/web/20230208151519/http://www.macrumors.com/2010/04/02/apple-appears-to-have-acquired-intrinsity/)苹果公司已经收购了位于得克萨斯州奥斯汀的芯片公司 [Intrinsity](https://web.archive.org/web/20230208151519/http://www.crunchbase.com/company/intrinsity) 。是的，看来这家公司在很大程度上是 iPad 内部新 A4 芯片的幕后推手。一位分析师给出的价格是 1.21 亿美元，但他也是一名分析师，所以谁知道呢。

但是真的，价格不重要。苹果银行有超过 400 亿美元的现金——这次收购很可能只是沧海一粟。更重要的是这一举措的影响:苹果继续挖走人才，并扩展其在内部完成更多工作的能力。你会记得早在 2008 年，苹果收购了 PA Semi，该公司也被认为是苹果用来设计 A4 芯片的另一部分(嗯，反正没有离开的人)。有传言称[苹果公司](https://web.archive.org/web/20230208151519/https://techcrunch.com/2010/04/21/apple-arm/)也在考虑收购 ARM 公司(尽管 ARM 公司淡化了这些传言)。在这三家公司之后，苹果新芯片唯一依赖的另一家公司被认为是三星——而且那只是为了[制造](https://web.archive.org/web/20230208151519/http://arstechnica.com/apple/news/2010/04/chipworks-confirms-more-ipad-a4-details-mystery-remains.ars)这东西。

在 iPad 发布后不久，[我写道](https://web.archive.org/web/20230208151519/https://techcrunch.com/2010/01/28/ipad-extras/)这件事最大的新闻可能不是设备本身，而是苹果控制了运行它的芯片的制造(或者至少是设计)。虽然目前，这只是 iPad 专用的，但这种芯片(或一些变体)很可能会在未来的 iPhone 中找到出路(甚至可能是今年夏天推出的那款)。如果你听听史蒂夫·乔布斯自己的评论——“苹果是一家移动设备公司——不难想象有一天苹果会负责为其整个产品线制造芯片。

如果是这样的话，苹果将继续积累必要的部分来完成所有的内部工作，苹果总部可能有一天会更像巧克力工厂。产品会出来，但你需要一张金券才能进去。苹果可以堵住那些讨厌的永久漏洞。
