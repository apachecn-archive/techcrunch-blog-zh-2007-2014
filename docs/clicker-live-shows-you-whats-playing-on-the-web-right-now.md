# Clicker Live 向您展示了网络上正在播放的内容

![](img/e545c45009b6345db4e3d904cf805b0d.png)

直播视频开始在网络直播视频网站上兴起，如 [Livestream、](https://web.archive.org/web/20221207214319/https://beta.techcrunch.com/2010/03/04/livestream-ap-oscars/) Ustream 和 Justin.TV。每个网络电视网站也不时提供直播视频。很难跟上。所需要的是用于网络上直播视频的电视指南。

在线视频指南[点击器](https://web.archive.org/web/20221207214319/http://www.clicker.com/)正在尝试推出[点击器直播](https://web.archive.org/web/20221207214319/http://www.clicker.com/live/)。它从网上成千上万的优质直播视频开始。点击网站将有一个新的直播标签，直播节目将被提升为“现在播出”并可以被搜索。还有一个日历显示即将发生的事情，你可以按主题或标题浏览。以下是将在 Clicker Live 上编入索引的节目和直播视频流类型的一些示例:
