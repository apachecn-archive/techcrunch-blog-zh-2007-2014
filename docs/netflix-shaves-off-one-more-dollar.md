# 网飞又刮掉了一美元

![02netflix.gif](img/00da3043a256913bf8e43d52bb73362c.png)

网飞在与百视达的斗争中变得绝望，或者他们正在学习沃尔玛的剧本，准备将百视达从在线租赁中驱逐出去。我倾向于相信后者是真的，嗯，在很大程度上。通过在已经很便宜的价格上再削减 1 美元，网飞正徘徊在一条可能最终打破它们的微妙线上。Blockbuster 不可能关闭他们的在线部门，因为人们仍然会去 B&M 商店，因为它超级容易，你可以看更多的电影，而不是坐在那里等待下一批 DVD 的到来。不过，我想我会回网飞。我很懒，很少离开我的公寓。
