# 英国安全局也接入了棱镜

# 英国安全局也接入了美国国家安全局的棱镜监视项目

棱镜计划看起来已经是一个影响深远的项目，但是一份新的报告称，美国国家安全局还允许至少一个外国安全机构进入这个系统。根据卫报的[报道，至少从 2010 年 6 月开始，NSA 向英国](https://web.archive.org/web/20230328053037/http://www.guardian.co.uk/technology/2013/jun/07/uk-gathering-secret-intelligence-nsa-prism)[政府通信总部](https://web.archive.org/web/20230328053037/http://en.wikipedia.org/wiki/Government_Communications_Headquarters) (GCHQ)提供了对 Prism 的访问权限。

美国现在已经承认了棱镜的存在，尽管谷歌、苹果和所有其他在最初泄密中被点名的公司继续声称他们没有像泄密者声称的那样让美国政府“直接访问”他们的服务器。美国国家情报总监詹姆斯·克拉珀昨天辩称，该计划只是为了监视外国公民而存在，但很明显，不管意图如何，该计划的影响远不止于此。

报告称，英国 GCHQ 在 2012 年上半年从 Prism 获得了 197 份情报报告。目前还不清楚该组织是否有权访问 Prism。《卫报》的报道表明，它“能够从节目的一个定制部分接收材料，以符合英国的利益。”

到目前为止，我们还没有听说任何其他美国盟友获得了这些数据，但如果 NSA 为英国提供了进入 Prism 的直接途径，并且该计划的目的确实是监视外国公民，那么如果其他人也能获得这些数据，也不会令人感到惊讶。
