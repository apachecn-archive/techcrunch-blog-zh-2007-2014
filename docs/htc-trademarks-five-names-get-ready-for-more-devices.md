# HTC 商标五个名字，准备购买更多设备

[![000064633495](img/e05719916b7ee4ab10aa4e5e849ff8c3.png "000064633495")](https://web.archive.org/web/20221002074411/https://beta.techcrunch.com/wp-content/uploads/2009/07/000064633495.jpg)

[HTC](https://web.archive.org/web/20221002074411/http://www.crunchgear.com/tag/htc/) 以强势名词命名设备的历史悠久。以前有神剑、骑士、黑石、阿帕奇、先知，最近有[英雄](https://web.archive.org/web/20221002074411/http://www.crunchgear.com/2009/07/20/review-htc-hero/)。好吧，准备好跳跃后再有五个遵循相同命名方案的设备。

我们有活力，黎明，插曲，意象和纹身即将到来的某个时候。关于这些设备没有任何其他信息，所以不要问。我们不知道他们会运行什么操作系统，什么时候发布，或者他们中的任何一个[是否会比性感英雄](https://web.archive.org/web/20221002074411/http://www.crunchgear.com/2009/07/20/review-htc-hero/)看起来更好。我们所知道的只是名字，对于一个缓慢的周一早晨的帖子来说，这已经足够了。
