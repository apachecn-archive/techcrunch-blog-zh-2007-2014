# iPhone 4 泄密案中的陈没有被起诉——其他人就没这么幸运了

经过一年多的不确定性，iPhone 4 被盗一案已经接近尾声。圣马特奥县助理地方检察官莫利·皮特宣布不会对 Gizmodo 的陈以桐提出指控，他的自由一直受到质疑，因为警察[在所谓的盗窃发生后不久突袭了他的房子](https://web.archive.org/web/20230203072208/https://techcrunch.com/2010/04/26/the-iphone-leak-gets-ugly-police-raid-gizmodo-editors-house-confiscate-computers/)。

皮特说“这是一个非常灰色的领域”，但关于记者保护法，“这不是我们要挑战极限的情况。”这不是最精确的免责声明，但非常明智。

另一方面，布莱恩·霍根(Brian Hogan)和塞奇·沃勒(Sage Wallower)在一家酒吧发现了新一代手机，*将分别被指控:挪用失物(即出售你发现的东西)，和占有赃物(一旦你决定出售它，它就变成了偷来的)。他们面临最多一年的监禁，外加罚款和缓刑；我不想对他们的命运做过多的猜测，但希望这种与新认定的合法新闻工作的联系会让法官对他们更加有利。*
