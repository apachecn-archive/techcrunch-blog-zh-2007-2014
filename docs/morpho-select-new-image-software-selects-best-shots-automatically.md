# Morpho 智能选择:新的图像软件自动选择最佳拍摄

[![](img/6ffdc29f858269fadc37fef8cb266333.png "Picture 10")](https://web.archive.org/web/20221202042845/http://www.crunchgear.com/2010/06/11/morpho-select-new-image-software-selects-best-shots-automatically/picture-10-4/)

总部位于日本的 Morpho [今天宣布了](https://web.archive.org/web/20221202042845/https://beta.techcrunch.com/wp-content/uploads/2010/06/morphosmartselect_10610.pdf)【JP，PDF】“Morpho 智能选择”，这是一款可以从数码相机拍摄的一组照片中挑选出最佳照片的软件。Morpho Smart Select 基于许多不同的因素做出决定，包括颜色对比度、图像中物体或人物的位置、人物的微笑和焦点。

“一个人微笑的质量”是通过分析这个人的眼角和嘴角的角度来确定的。这一决定是基于该公司从真人(从给定的一组照片中选择最佳照片)调查中收集的数据，并反馈到软件中。

Morpho Smart Select 还允许用户选择七种评估模式中的一种，例如“宠物模式”或“肖像模式”，从而对决策进行微调。它的制造商表示，该软件之所以脱颖而出，是因为它不需要太多的处理能力，而且速度很快:例如，根据 Morpho 的说法，内置在手机中的相机(并安装了该软件)只需要 100 毫秒就可以拍摄出最佳照片。

事实上，该公司已经与日本手机制造商达成了一些交易，几款预装了 Morpho Smart Select 的手机将于今年夏天上市。
