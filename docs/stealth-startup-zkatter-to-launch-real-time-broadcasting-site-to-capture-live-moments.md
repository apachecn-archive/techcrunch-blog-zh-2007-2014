# 秘密创业公司 Zkatter 将推出实时广播网站，捕捉“现场瞬间”

![](img/c8906fbe64d1c56fd7b6601b576310e1.png)

英国秘密创业公司 [Zkatter](https://web.archive.org/web/20221003215322/http://www.zkatter.com/) 将在未来几个月推出实时微博服务，这可能会引起轰动。理论上类似于 Twitter，Zkatter 问用户“你现在看到了什么？”对比 Twitter 的“你在做什么？”

这项服务自 2008 年以来一直在开发中，将允许任何人向“仅限朋友”或“公众”广播和存档包括位置、媒体(图像、视频和文本)在内的“实时时刻”，然后可以通过搜索和朋友时间线立即发现。Zkatter 的重点是捕捉你实际看到的信息，这为实时空间提供了一个有趣的补充。

Zkatter 在首轮融资中获得了[150 万美元](https://web.archive.org/web/20221003215322/http://www.crunchbase.com/company/zkattter)，它还在开发 iPhone 和 Android 应用程序，以便与独立网站协同工作，该平台可以在所有设备上实时更新。因此，如果你在 iPhone 上发布更新，它会自动显示在网站上。Zkatter 将与脸书和推特整合，所以你也可以在这两个社交网络上发布你的更新。该网站还有其他功能，但这是 Zkatter 能够做的事情的总体思路。

我们对网站进行了一点测试，结果令人印象深刻。不仅用户界面易于使用，而且手机和网络之间的无缝实时集成。但 Zkatter 仍将面临来自 Twitter 和过多内容共享 Twitter 应用的竞争，如 [TwitPic](https://web.archive.org/web/20221003215322/http://twitpic.com/) 或 [TwitVid、](https://web.archive.org/web/20221003215322/http://www.twitvid.com/)也可以让你直播“现场时刻”。该网站也类似于 [DailyBooth。](https://web.archive.org/web/20221003215322/http://dailybooth.com/)
