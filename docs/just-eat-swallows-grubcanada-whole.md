# 只吃整只燕子

总部设在伦敦的在线食品订购公司 Just-Eat 于 2000 年在丹麦成立，该公司正在很好地利用今年早些时候从 T2 筹集的 4800 万美元资金。在[接管了](https://web.archive.org/web/20230204115822/http://eu.beta.techcrunch.com/2011/10/05/just-eat-acquires-online-food-delivery-provider-urbanbite/)当地竞争对手，伦敦在线食品配送公司 [Urbanbite](https://web.archive.org/web/20230204115822/http://www.urbanbite.com/) 之后，Just-Eat 的[加拿大业务](https://web.archive.org/web/20230204115822/http://www.just-eat.ca/)已经[收购了](https://web.archive.org/web/20230204115822/http://www.marketwire.com/press-release/just-eat-canada-acquires-grubcanada-1577188.htm) [GrubCanada](https://web.archive.org/web/20230204115822/http://grubcanada.com/) 。

后者是加拿大最早的在线食品订购服务公司之一，由企业家霍华德·米格达尔和罗斯·利普森于 2008 年创立。今年早些时候，Just-Eat Canada 还收购了总部位于温哥华的 YummyWeb.com。

通过此次收购，Just-Eat 能够为在线消费者提供加拿大各地 1，000 多家外卖餐厅，由于 GrubCanada 业务的增加，安大略省和不列颠哥伦比亚省的 200 家新的社区餐厅也增加了。

收购条款没有披露。

Just-Eat 声称每天提供超过 100，000 份餐，自 2001 年网站推出以来，它已经处理了超过 3，000 万份订单(目前它每月为全球 20，000 家餐厅合作伙伴处理超过 400 万份订单)。

该公司得到了 Greylock Partners、Index Ventures 和 Redpoint Ventures 的支持。
