# 恭喜脆饼干赢家！Twitter 获得 2010 年最佳创业公司

今年的第四届年度[嘎吱嘎吱奖](https://web.archive.org/web/20230119035902/http://crunchies2010.beta.techcrunch.com/)刚刚结束，我们很高兴地说，这是一个压倒性的成功。对于那些没有出席活动或观看我们[直播](https://web.archive.org/web/20230119035902/https://techcrunch.com/2011/01/21/the-crunchies-awards-livestream-video/)的人，我们在下面列出了被提名者和获奖者的名单。我们向获奖者以及所有被提名的[候选人](https://web.archive.org/web/20230119035902/https://techcrunch.com/2011/01/07/2010-crunchies-finalists/)致以最诚挚的祝贺。对于许多类别来说，这是一场令人难以置信的激烈竞争，可以肯定地说，这个名单上的每个人都是他们所在领域的顶尖人物。

我们想花点时间指出 Twitter 赢得了“最佳整体创业公司或产品”，这是该公司第一次在这一类别中赢得冠军。Twitter 已经成为社会交流中不可或缺的一部分，也是网络结构中的一个重要组成部分。祝贺 Groupon 的安德鲁·梅森获得年度首席执行官；马克·平卡斯获得年度最佳创始人奖，Quora 获得 2010 年最佳新创企业奖。
