# DLD11:谷歌玛丽莎·梅耶尔谈埃里克·施密特辞去首席执行官

![](img/b35b487a7888c5885e06e8d62953f3f3.png)

在 DLD 会议上，由《快公司》的琳达·蒂施勒主持的关于“开放式创新”的小组讨论中，谷歌的[玛丽莎·梅耶尔](https://web.archive.org/web/20230202232553/http://www.crunchbase.com/person/marissa-mayer)被问及关于[埃里克·施密特](https://web.archive.org/web/20230202232553/http://www.crunchbase.com/person/eric-schmidt)向公司新角色[过渡](https://web.archive.org/web/20230202232553/https://techcrunch.com/2011/01/20/techcrunch-interview-with-eric-schmidt-larry-page-and-sergey-brin/)，[将控制权](https://web.archive.org/web/20230202232553/https://techcrunch.com/2011/01/20/google-ceo-change/)移交给联合创始人[拉里·佩奇](https://web.archive.org/web/20230202232553/http://www.crunchbase.com/person/larry-page)。

梅耶尔:

> 长期以来，谷歌一直由三巨头运营，上周我们宣布了重组。需要强调的是，角色已经发生了变化，但拉里、埃里克和谢尔盖将继续领导公司。
> 
> 媒体和其他媒体对此有很多猜测，但我们真的对这些变化感到兴奋。别忘了拉里之前也是谷歌的首席执行官。
> 
> 施密特在推特上发布了一些关于改组的有趣消息，称“不再需要成年人的监督”。这本来是一个玩笑，但我个人真的很兴奋看到拉里更成熟的管理方面。
> 
> Larry 的热情一直是产品，所以他将更多地关注等式的技术部分和全面的执行。

(其他小组成员包括贝丝·康斯托克，通用电气公司的 CMO，大象设计公司的创始人，延斯·马丁·斯基布斯特德。)
