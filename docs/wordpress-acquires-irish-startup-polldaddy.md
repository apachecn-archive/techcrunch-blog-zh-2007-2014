# WordPress 收购爱尔兰初创公司 Polldaddy

![](img/5c82020599d711c161d2b1924a4704cf.png)

WordPress 背后的公司 automatic 收购了爱尔兰初创公司 [Polldaddy](https://web.archive.org/web/20221130195102/http://www.polldaddy.com/) ，金额未披露。这次收购给 WordPress 注入了民意调查技术，而[似乎是合理的](https://web.archive.org/web/20221130195102/http://ma.tt/2008/10/polldaddy-goes-automattic/)，仅仅是因为博主们喜欢民意调查(我们在 TechCrunch 的许多帖子中都使用 PollDaddy)。

在高度分散的博客初创公司中，似乎有一种插件累积策略正在进行，这将导致将当前通过扩展提供的功能吸收到 WordPress 代码库中。automatic[最近收购了](https://web.archive.org/web/20221130195102/http://www.beta.techcrunch.com/2008/09/23/automattic-has-acquired-intensedebates-enhanced-comment-system/) [激烈辩论](https://web.archive.org/web/20221130195102/http://www.intensedebate.com/)，一家小型 [TechStars](https://web.archive.org/web/20221130195102/http://www.crunchbase.com/company/techstars) 初创公司，在一个高级评论平台上工作。更早的时候，它还收购了 [Buddy Press](https://web.archive.org/web/20221130195102/http://www.buddypress.org/) ，这是一个将社交网络功能分层到 WordPress 上的项目，[在 3 月](https://web.archive.org/web/20221130195102/http://www.beta.techcrunch.com/2008/03/04/wordpress-the-social-network/)收购了 [Gravatar](https://web.archive.org/web/20221130195102/http://www.gravatar.com/) ，这是一个通用的头像系统，[在去年秋天](https://web.archive.org/web/20221130195102/http://www.beta.techcrunch.com/2007/10/17/automattic-acquires-gravatar/)。

像激烈的辩论一样，Polldaddy 不仅仅向 WordPress 出版商提供技术，而且它也不打算在收购后逐步停止对其他平台的支持。但我们可以预计两家公司的努力主要是为了改进 WordPress——都是在 WordPress.org 的[提供的开源版本，但更重要的是在 WordPress.com 的](https://web.archive.org/web/20221130195102/http://www.wordpress.org/)[提供的托管版本(Automattic 可以通过它赚钱)。PollDaddy 已经因其 440 万博客用户而被纳入 WordPress.com。](https://web.archive.org/web/20221130195102/http://www.wordpress.com/)

鉴于许多初创公司(国内和全球)在这些动荡时期的经济担忧，我相信 PollDaddy 和激烈辩论都很高兴能在一家更大、资金更充足的初创公司中找到一个家。PollDaddy 总部位于爱尔兰，这一事实应该不会对 Automattic 的公司结构产生太大影响。正如首席执行官 Toni Schneider 在最近的 Startup2Startup 活动中解释的那样，Automattic 没有中央办公室，所有员工都在家远程工作，只是作为一个公司每年见几次面。
