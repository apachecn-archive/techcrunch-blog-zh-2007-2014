# Facedeals:通过面部识别在脸书办理登机手续。令人毛骨悚然还是令人敬畏？

现在，你可以通过面部识别扫描在脸书的某个地点办理登机手续。Redpepper 广告公司[声称](https://web.archive.org/web/20221006195038/http://redpepperland.com/lab/details/facedeals/)正在“纳什维尔商业”外面测试一个摄像头，在用户允许该公司访问他们的脸书数据后，该摄像头会自动检查顾客的位置并为他们提供交易(注:这是独立于脸书开发的)。下面是这项技术的视频解释， [Facedeals](https://web.archive.org/web/20221006195038/http://vimeo.com/47133014) 。

[vimeo 47133014 w=500 h=281]

通过收购 Face.com，脸书因采取面部识别措施而受到[国会的严格审查](https://web.archive.org/web/20221006195038/http://articles.latimes.com/2012/jul/18/business/la-fi-senate-facebook-20120719)。然而，仅仅过了一个月，[它就停止使用技术](https://web.archive.org/web/20221006195038/https://beta.techcrunch.com/2012/07/09/facebook-facial-recognition-api/)来自动建议在照片中标记哪些朋友。

正如 Redpepper [在博客文章](https://web.archive.org/web/20221006195038/http://redpepperland.com/lab/details/facedeals/)中解释的那样，所谓的技术完全是通过一个自愿的应用程序完成的:

> Facedeals 应用程序必须通过您的脸书帐户授权。在你的帮助下，该应用程序会验证你最近的照片标签，并使用这些标签来绘制你面部的物理外观。然后，我们定制开发的摄像头只需使用这些现有数据，就可以在现实世界中识别您的身份。现在，个性化交易可以从所有参与地点发送到您的智能手机上，您只需露个脸。

Redpepper 认为，这项技术将帮助消费者和企业利用脸书的魔力更有效地达成交易。你怎么想呢?令人毛骨悚然还是令人敬畏？

我们的脸书专家 Josh Constine 想要澄清的是，脸书并没有批准或认可 Facedeals。它只是利用社交网络作为一种简单的方式，让人们上传照片给 Facedeals 进行自己的面部识别处理。

事实上，考虑到名称和颜色，Facedeals 可能会遭到商标侵权诉讼。脸书正试图淡化其面部识别技术的使用，可能不会友好地对待这种混乱以及其平台可能引发的恐惧。
