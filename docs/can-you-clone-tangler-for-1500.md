# 您能以 1500 美元克隆唐格勒吗？

![](img/c081c41132e212107d89ca8a5aea40f4.png)

我们在多次场合[中称赞了新主持的论坛初创公司](https://web.archive.org/web/20221007152026/http://www.beta.techcrunch.com/2006/02/21/tangler-untangling-communication-on-the-web/)[唐格勒](https://web.archive.org/web/20221007152026/http://www.tangler.com/)——最近称其为新讨论板/论坛初创公司的[出血边缘](https://web.archive.org/web/20221007152026/http://www.beta.techcrunch.com/2007/11/07/wetpaint-combines-discussion-forums-with-wikis/)。

显然其他人也喜欢它。[土耳其某人](https://web.archive.org/web/20221007152026/http://www.getafreelancer.com/users/532152.html)愿意向任何能“克隆坦格勒”的人支付高达 1500 美元的费用。但是不要走得太远——列表还指出“不要窃取 tangler.com 图像，也不要侵犯版权。克隆应该具有相同的功能，但是设计应该看起来不同。”

已经进行了两次投标，一次出价仅为 1000 美元。当然，构建一个可扩展的论坛平台，通过 javascript 接口允许同步和异步消息传递，可能需要 1000 多美元的投资。但是嘿，为什么不试试呢？

这不是我们第一次看到这样的事情。2006 年，有人试图以 500 美元购买一个 Techmeme 克隆体，而一个更新的列表显示同样的东西需要 300 美元。

对任何赢得业务的人来说——我的建议是先要钱，因为当你交出一两周关于坦格勒或科技迷的工作时，你不太可能有一个开心的顾客。
