# 密歇根州成为最新一个保护公民免受雇主和学校窥探私人社交信息的州

美国最大的州密歇根州的雇主和学校现在被禁止向员工和学生索要个人电子邮件和社交媒体账户的密码。为了合理的隐私和常识，密歇根州州长里克·斯奈德刚刚签署了众议院第 5523 号法案，成为州众议员王高·奈斯比特提出的法律。

Snyder [在 2012 年 12 月 28 日发布的声明](https://web.archive.org/web/20221206181534/http://www.michigan.gov/minewswire/0,4629,7-136-3452-291976--,00.html)中表示:“网络安全对于密歇根的重塑非常重要，保护居民的私人互联网账户是其中的一部分。”。"对潜在雇员和学生的评判应该基于他们的技能和能力，而不是私人在线活动."

该法案还保护工作和学校申请人不必给出他们的密码。违反新法律的人可能会被指控犯有轻罪，并被处以最高 1000 美元的罚款。

诸如此类的法案是对 2012 年出现的一个令人不安的趋势的回应。雇主和学校被发现要求现在和未来的雇员和学生访问他们的在线账户。这通常被贴上自愿的标签，但不遵守通常会产生负面影响。今年早些时候，美国众议院[未能批准一项联邦禁令](https://web.archive.org/web/20221206181534/https://beta.techcrunch.com/2012/03/28/house-shoots-down-bill-that-would-have-stopped-employers-from-demanding-your-facebook-password/)，为各州承担责任铺平了道路。

加州、特拉华州、伊利诺伊州和新泽西[也有类似的法律。但是请记住，](https://web.archive.org/web/20221206181534/http://www.ncsl.org/issues-research/telecom/employer-access-to-social-media-passwords.aspx)[隐私和个人之间是有区别的](https://web.archive.org/web/20221206181534/https://beta.techcrunch.com/2012/12/28/theres-a-difference-between-private-and-personal/)，你放在网上的任何东西都不会是完全隐私的——对吗，Randi ？
