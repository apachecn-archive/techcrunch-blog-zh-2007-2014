# 微软的 Project Spark Beta 来到 Xbox One，提供跨平台游戏创作

微软雄心勃勃的项目 Spark open world game creator [早在 12 月就在 Windows 8.1](https://web.archive.org/web/20221006041919/https://beta.techcrunch.com/2013/12/03/microsofts-project-spark-diy-game-creation-software-beta-begins-today/) 上推出了测试版，并承诺它最终也会进入 Xbox One。今天是它在下一代游戏主机上发布的日子，这使得 Spark 创作者可以在 Windows 8.1 上构建游戏，并在他们的 Xbox One 上玩游戏，反之亦然。

据微软称，测试版目前已经积累了 250，000 个注册用户，每天都有“数百个”新用户加入。微软正在限制对开放游戏制作工具的访问，这可能是一个明智的想法，因为它是一个协作游戏创作引擎，必须承载沉重的服务器负载，而且可能会在确保用户创建的内容适当、有趣和引人入胜方面带来一些挑战。

Xbox One 的扩展可能会为潜在创作者的新受众打开大门，但它可能更有趣的是让游戏玩家考虑为不同平台设计，并根据游戏机技术的具体需求评估他们的创作。这也是一个很好的例子，说明微软决定采用基于 x86 的处理器架构如何更容易地构建扩展到 Windows 和 Xbox 的跨平台体验——在 Xbox 360 上，像这样无缝的跨平台游戏可能在技术上更具挑战性，如果不是不可能的话。

要开始使用 Spark 测试版，你仍然需要去[微软的预告页面](https://web.archive.org/web/20221006041919/https://joinprojectspark.com/)注册。
