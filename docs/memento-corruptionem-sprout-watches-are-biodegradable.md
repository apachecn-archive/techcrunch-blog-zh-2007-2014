# 腐败纪念品:萌芽手表是可生物降解的

几个世纪后，考古学家将发现这些 Sprout 可生物降解手表的宝藏，并只找到它们的机芯，薄硅片像 Antikythera 机械装置一样相互腐蚀，周围是大杯子和品客薯片罐，这是一个品牌致力于环保的证明。

这些手表的表壳、表带、表盘和表圈价格从 30 美元到 75 美元不等，由可生物降解的产品制成，包括竹子、棉花、树脂和矿物晶体。
