# Babelverse 致力于使翻译民主化

Babelverse 赢得了在 TechCrunch Disrupt 上亮相的机会，在几乎没有人注意的情况下，他做了一个巧妙的演讲。本质上，这是一个通用语音翻译的解决方案，由全球人类口译员社区提供支持:这意味着任何人都可以成为口译员。我们[在一月份报道了它的发布](https://web.archive.org/web/20221005185354/https://beta.techcrunch.com/2012/01/24/babelverse-to-offer-live-voice-translations-for-state-of-the-union-in-up-to-7k-languages/)，但是这里有一个快速的概要。

我们知道，机器翻译是不可靠的。所以我们现在看到的是一个翻译市场。

人们练习口译，并在系统中晋升，成为更专业的口译员。可以把它想象成一种翻译语言的需求媒体平台。

这是一个移动和网络应用程序，让用户受益于世界上任何一种口语的现场“实时”翻译。

[![](img/d6c3f7ddfa35eece7560d03817efef47.png "IMG_8895")](https://web.archive.org/web/20221005185354/https://beta.techcrunch.com/wp-content/uploads/2012/05/img_8895.jpg)

熟练的业余爱好者和专业口译员走上讲台，为他们的时间挣钱。专业翻译要贵得多，但这让翻译过程民主化了。

竞争对手包括谷歌翻译对话模式 Android 上提供的一项功能，但仅适用于有限的几种语言。

最终，他们希望能够做到现场翻译。
