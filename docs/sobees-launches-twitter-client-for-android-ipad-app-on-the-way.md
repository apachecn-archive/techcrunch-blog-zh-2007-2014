# Sobees 在路上为 Android 和 iPad 应用程序推出 Twitter 客户端 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/04/22/sobees-launches-twitter-client-for-android-ipad-app-on-the-way/>

# Sobees 即将推出 Android 和 iPad 应用程序的 Twitter 客户端

![](img/063ee3a5528010971adff4b3213a3286.png) Twitter 客户端Sobees 正致力于创造市场上最好的社交媒体客户端，与 [TweetDeck](https://web.archive.org/web/20221006230332/http://www.tweetdeck.com/) 和 [Seesmic 竞争。](https://web.archive.org/web/20221006230332/http://seesmic.com/) Sobees 提供了一个[原生 Windows 客户端，](https://web.archive.org/web/20221006230332/https://beta.techcrunch.com/2009/11/25/sobees-tackles-linkedin-in-powerful-new-clients-android-app-to-launch-soon/)一个基于网络的客户端，现在正首次进军移动领域，推出一个专为 Android 手机打造的 Twitter 客户端。可以在这里下载 app [。](https://web.archive.org/web/20221006230332/http://www.sobees.com/)

Sobees 的 Android 应用程序允许您访问多个 Twitter 帐户，共享照片和您的位置，缩短 URL，离线时阅读推文，使用 Twitter 搜索搜索最近的消息，并直接在应用程序中查看个人资料、图片、地图等。Sobees 还通过将实时搜索初创公司 OneRiot 的 Riotwise 广告添加到该应用中来赚钱。

Sobees 计划在未来增加其他社交网络，包括脸书和 LinkedIn，并且正在为 iPhone 和 Windows phone 开发客户端。Sobees 正在准备一个 iPad 应用程序。这家初创公司还更新了其桌面客户端，在客户端中添加了 RSS 阅读器和《纽约时报》内容(使用他们的 API 进行集成)。桌面应用程序也将展示 Riotwise 广告。

当然，Sobees 的 Android 应用程序将面临来自 Twitter 官方 Android 应用程序的竞争，后者是在上周的微博网络会议上宣布的。然而，目前还不清楚 Twitter 是否会在内部开发应用程序，或者收购一家移动应用程序开发商，就像它对 iPhone [Twitter 客户端 Tweetie 所做的那样。](https://web.archive.org/web/20221006230332/https://beta.techcrunch.com/2010/04/09/twitter-acquires-tweetie/) Twitter 还可以与 Sobees 或 Seesmic 等移动客户端开发商合作开发 Android 应用，类似于 Twitter 与 RIM 就黑莓手机的[官方 Twitter 应用达成的协议。](https://web.archive.org/web/20221006230332/https://beta.techcrunch.com/2010/04/09/twitter-fills-its-first-hole-with-an-official-blackberry-app/)
