# 东芝 T401 DAP 有 Wi-Fi

[![t401.jpg](img/8ebe24ec5366a5f1ef3eeaa983b38ec4.png)](https://web.archive.org/web/20130730120117/http://old.crunchgear.com/wp-content/uploads/t401.jpg "t401.jpg")

东芝总是擅长一两个 DAP，其最新的两款产品 T401S 和 T401 保持了这一趋势。这里的大新闻是 T401 带来了 Wi-Fi 认证，允许用户直接连接到互联网下载播客，更有可能的是，在他们的 PC 之间传输音乐。我指的是电脑，左下角有一个 Windows 标志按钮。

但是让我们诚实一会儿:自从苹果不久前发布了[彩色 iPod shuffle](https://web.archive.org/web/20130730120117/http://crunchgear.com/2007/01/30/apple-poops-out-multicolored-ipod-shuffles/) 以来，DAP 世界已经有点停滞不前了。也许 8000 个指向新 iPod 内置 Wi-Fi 的传言中的一个会成真。

至于 T401，东芝将于下个月在日本发布。除了 Wi-Fi，它到处都写着“标准”:4GB，兼容 WMA *和* WMV(甚至那些有 DRM 的)等。

不过，有趣的是，我确实注意到 iPod 在日本远没有在这里占据主导地位。
