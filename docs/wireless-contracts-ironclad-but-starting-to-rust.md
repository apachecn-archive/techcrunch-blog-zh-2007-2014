# 无线合同无懈可击，但开始生锈

华盛顿邮报报道，芝加哥的科里·泰勒受够了他的无线供应商糟糕的服务。他不想支付 175 美元的提前终止费，所以泰勒的病已经到了晚期。在一个朋友的帮助下，他的手机供应商收到了一份伪造的死亡证明。“我想，‘除了一部我鄙视的手机，我还能失去什么？’“诡计没有得逞，泰勒不得不支付 175 美元。

“最后，我还是付了钱，”泰勒说。“但我敢打赌，我传达了一个明确的信息，即人们有多讨厌被绑在一部不能用的手机上。”

许多移动电话合同有两年的期限。如果客户想解除合同，终止合同需要支付高达 250 美元的费用。这通常比直接违约付款更便宜，后者将迫使消费者在违约后支付合同价格的剩余部分。有些合同确实考虑了消费者履行合同的时间长度，并相应地按比例收取终止费。

许多人对他们目前的手机服务不满意。不良的联系经常被认为是不满意的原因。一些人了解到其他运营商提供更多的移动应用程序，并希望转换。美国消费者满意度指数显示，无线行业的排名低于快餐店和航空公司。根据 CFI 集团的说法，糟糕的客户服务是人们转向另一家运营商的主要原因。

挑战终止费的诉讼已经在加利福尼亚和佛罗里达提起。诉讼声称，这些费用是非法的，因为它们惩罚了消费者转向另一个供应商。为了解决这个问题，无线运营商要求 FCC 不要将终止费视为罚金，而是公司可以收取的服务费。国会正在关注这个问题，立法可能会在今年秋天出台，这将赋予各州更多的权力来监管无线费用。

我不希望看到联邦政府插手手机费用和合同的监管工作。同时，我希望用最少的钱获得最好的移动连接。无线行业竞争如此激烈，我认为在不久的将来，取消费用将会降低或完全取消。这种费用引起的客户的不信任将使这个人永远离开违规的公司。服务提供商需要停止考虑短期利润，考虑做一些能让人们长期使用他们的服务品牌的事情。
