# 这个性感的咖啡小玩意可能永远不会存在

这是什么？激光切割机？一个国防部照片放大器？显微镜？纹身去除剂？不，这是一个咖啡机，但一个非常苗条的咖啡机。事实上，我想象这种小小的[Nespresso](https://web.archive.org/web/20230210050134/http://www.nespresso.com/#/us/en/coffee_nespresso/nespresso_blend)-友好的咖啡因诱导物在你典型的“[冰旅馆](https://web.archive.org/web/20230210050134/http://www.hickerphoto.com/data/media/25/ice-hotel-T0712.jpg)”或广告公司大厅里会看起来很甜。

显然，我们已经走过了漫长的道路，从过去的日子里，一个 [Cuisinart](https://web.archive.org/web/20230210050134/https://techcrunch.com/2007/12/05/cg-holiday-2007-recommendation-cuisinart-grind-brew-thermal-10-cup-automatic-coffeemaker/) 足以让你在高级定制时装电路。

要回答工业设计专业学生程昕婷·希洛可爱的小咖啡机是否真的存在是另一个问题。在 Yanko 网上商店[看，我没有看到它——虽然不像 Yanko 的许多概念，这个实际上看起来像是现实世界的实用设计。](https://web.archive.org/web/20230210050134/http://store.yankodesign.com/)

也就是说，我不知道为什么它会被宣传为“极简主义”的机器，因为它是电动的，需要精美的咖啡包。[这些低技术含量的咖啡解决方案](https://web.archive.org/web/20230210050134/https://techcrunch.com/2010/03/22/low-tech-coffee-its-the-way-to-go/)可能更简约一点，但这个绝对很酷。
