# OMG！砧板上的草书教育

草书可能会步渡渡鸟和报纸的后尘:堪萨斯州正在考虑决定取消草书教育，优先考虑打字技能。“家长们想知道你们学校在教育孩子们为科技世界做准备方面做了些什么，”威奇托天主教教区学校负责人鲍勃·沃博里尔说。"对父母来说，这比我们所说的书法艺术更重要。"

周二，堪萨斯州教育委员会将考虑草书在小学教育中的作用，并收集各区的调查反馈。《威奇托鹰报》报道称，该市的草书课程已经减少，但不确定董事会成员对彻底将其从课程中删除的决定有多认真。

“我们必须能够以书面形式相互交流，”威奇托教育委员会成员沃尔特·沙佩尔说。“技术是伟大的，但它并不总是有效。有各种各样的情况，你必须知道如何手写。”

已经被 46 个州采用的新的国家课程[共同核心](https://web.archive.org/web/20220930203340/http://www.corestandards.org/)，包含[对草书教学没有正式要求](https://web.archive.org/web/20220930203340/http://www.kansascity.com/2012/11/11/3911774/state-board-of-education-will.html)。这种趋势让一些专家担心，因为最近的一系列研究发现，手写可以促进大脑发育。例如，印第安纳大学的研究人员发现，在为期四周的研究中，打印字母而不是说字母的儿童[表现出与成人](https://web.archive.org/web/20220930203340/http://www.latimes.com/health/boostershots/sc-health-0615-child-health-handwriti20110615,0,360921.story)更相似的大脑活动。“对孩子来说，笔迹极其重要。印第安纳大学教授卡琳·哈曼·詹姆斯说:“不是他们做得有多好，而是他们做了并实践了。”"打字不做同样的事情."

无论其价值如何，随着电子产品变得无处不在，草书可能会发现自己因极度不便而逐渐消失。

现在，我不打算完全编造阴谋论来看看边缘团体是否会上钩，但我会暗示一个令人担忧的原因:宪法是用草书写成的，有些人可能会说这是自由教育机构通过国家课程标准，让下一代人无法阅读美国的建国文件。我可以这么说，但我不会。
