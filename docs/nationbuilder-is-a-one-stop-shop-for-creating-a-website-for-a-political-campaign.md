# NationBuilder 是为政治活动创建网站的一站式商店

毫无疑问，政治家和政治运动需要[与网络](https://web.archive.org/web/20230203020453/https://techcrunch.com/2011/04/04/obamas-re-election-campaign-puts-facebook-front-and-center-literally/)接触，无论是为了筹款、活动推广、新闻、辩论等等。虽然总统和其他获得大量资助的竞选活动可以聘请像[蓝州数字](https://web.archive.org/web/20230203020453/https://techcrunch.com/2010/12/30/wpp-buys-obamas-campaign-agency-blue-state-digital/)这样的数字顾问来协调他们的在线工作，但许多现有和潜在的候选人根本没有资源和资金来从 scratchEnter [NationBuilder、](https://web.archive.org/web/20230203020453/http://nationbuilder.com/)这是一个 SaaS 平台，允许政治候选人在几分钟内建立一个支持筹款活动、博客、志愿者外联、支付处理、日历等的时尚网站。

NationBuilder 包括非技术人员创建品牌网站、博客以及导入联系人列表和直接从网站发送电子邮件的能力。在社交媒体方面，NationBuilder 允许您整合多个脸书和推特账户，并从这些账户发布推特和脸书消息。连群发短信的能力都完全烘焙到平台里了。

NationBuilder 最有用的功能之一是一个财务仪表板，可以跟踪捐助者的数量，每个支持者的平均捐款额，当天，每月，每年筹集的金额，整个选举周期的初选。用户可以为筹集资金的数量或捐赠者的数量设定一个目标，一个突出的温度计会向你的支持者显示进度。NationBuilder 直接与 Authorize.net、PayFlow Pro 和其他支付处理器挂钩。

其他功能包括票务活动管理、志愿者组织、地图、日历等。它本质上是一个为政治运动创建网站的一站式商店。价格从每月 20 美元到每月 500 美元不等。电子邮件和短信是额外收费的，这家初创公司允许用户免费试用该平台两周。

NationBuilder 是由 Jim Gilliam 创建的，他是一位纪录片制作人，也有丰富的网络经验。他作为首席技术官帮助推出了 Business.com，并在 Lycos 工作过。Gilliam 创建 NationBuilder 是一个充满激情的项目，并很快意识到需要一个廉价、易于使用的平台来帮助政治候选人(和非营利组织)在网络上建立自己的形象。去年秋天，他用几个候选人和竞选团队对该网站的早期版本进行了试验，现在，NationBuilder 向公众开放。随着 2012 年选举季的升温，我想说这是一个很好的时机。
