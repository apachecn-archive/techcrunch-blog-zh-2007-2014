# 松下将开始在欧洲(可能还有美国)销售安卓手机

在后 iPhone 时代，日本发生了很多变化，这个曾经是移动世界最受赞赏的国家。例如，在大规模整合和重组后，手机制造商的数量已从 13 家降至 5 家，所有剩余的制造商都在与不断萎缩和迅速老龄化的国内市场作斗争。

这就是为什么日本手机制造商长期以来一直在谈论走向国际市场的原因之一(几年前他们基本上都放弃了国际市场)，但现在[松下](https://web.archive.org/web/20221006183608/https://beta.techcrunch.com/tag/panasonic)似乎准备认真对待。

日本最大的商业出版物《日经新闻》报道称，该公司目前正试图与欧洲一家主要电信公司达成协议，将智能手机(Android 手机)引入欧洲大陆。显然，松下正在考虑是在东南亚自己的工厂生产手机，还是外包给另一家公司。

如果一切顺利，松下还计划最终在亚洲和北美提供 Android 手机。该公司去年销售了 440 万部手机，但它希望到 2015 年全球销量(每年)达到 750 万部。

上图显示了目前在日本销售的[松下 LUMIX 手机 101P](https://web.archive.org/web/20221006183608/https://beta.techcrunch.com/2011/09/29/android-phone-lumix-camera-panasonics-lumix-phone-101p-for-japan/) ，该手机采用 Android 2.3，4 英寸 QHD 液晶显示屏，分辨率为 960×540，13.2MP CMOS Lumix 传感器，防水机身(IPX5/7)，Wi-fi，红外连接，电子钱包功能，以及数字电视调谐器。
