# 音乐应用程序制造商 Smule 终于通过社交网络 Smule Nation TechCrunch 拥抱了网络

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/01/smule-nation-web-launch/>

Smule 推出了许多流行的音乐主题移动应用，包括 Sing！卡拉 ok，吉他，陶笛，魔琴，我是 T-Pain。现在，它正把自己的网站变成一个名为 Smule Nation 的社交网络，突出展示其所有应用的表现。

这些应用程序一直强调分享。我写的关于 Smule 的第一首热门歌曲 Ocarina 的一篇帖子[突出了 YouTube 上一些最佳表演的视频](https://web.archive.org/web/20221006180459/http://venturebeat.com/2009/01/13/watch-the-10-best-performances-on-smules-ocarina/)。但这也指出了一个不足之处:如果用户想要分享他们的表现，他们必须在应用程序内部进行，或者完全离开 Smule。

现在，通过 Smule Nation，该公司可以突出显示其所有应用程序的最佳性能。该网站包括网络上最流行的音乐部分，以及该公司的精选。如果您找到了喜欢的表演者，您可以访问他们的个人资料页面查看其他表演，并在他们添加新内容时关注他们以获取更新。与此同时，这些表演者可以在各种应用程序中统一展示他们的作品，如果他们想分享内容，他们现在只需分享一个链接。

联合创始人兼首席执行官杰夫·史密斯表示，Smule 之前有一个具有一些基本社交功能的网站，但他认为这是第一次真正的社交推广(当然，在此之前，网络从未成为 Smule 推广工作的重要组成部分)。

史密斯和首席产品和设计官杨珍妮(Jeannie Yang)带我参观了新网站，他们的主要目标之一似乎是突出应用程序中已经发生的活动量。唱吧！他们说，卡拉 ok 用户每小时唱 2 万多首歌，而 Magic Piano 用户则唱 5 万多首。唱歌！例如，实际上已经将在现实生活中从未谋面的表演者团体(是的，包括乐队名称和一切)联系起来。

该公司表示，总的来说，超过 1.25 亿用户分享了超过 10 亿首歌曲。史密斯说，大部分内容“不是特别好”，但他认为，随着用户收听和推广这些表演，“最终一些好的内容会出现在顶部。”在这种情况下，他建议该网站可以成为发现新音乐的目的地——因为它都是用户生成的内容，所以没有其他音乐服务的高昂成本。

尽管有更大的愿景，史密斯说他没有直接从 Smule Nation 赚钱的计划。相反，他认为这是统一和推广应用程序的一种方式。你仍然需要这些应用程序来创建网站上突出显示的表演。史密斯说，随着时间的推移，他有兴趣在网络上做更多的事情，但有些功能“我们永远只能在本地做。”
