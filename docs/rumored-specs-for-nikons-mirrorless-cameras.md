# 尼康无反光镜相机的传闻规格

现在所有酷孩子都没有镜子了。随着格式的成熟和人们看到好处，像 [E-PL3](https://web.archive.org/web/20230204121257/https://techcrunch.com/2011/09/09/the-olympus-e-pl3-dslr-power-in-a-point-and-shoot-package/) 这样的微型 4/3 相机和像 [X100](https://web.archive.org/web/20230204121257/https://techcrunch.com/2011/08/29/review-fujifilm-x100/) 这样的豪华紧凑型相机开始占据市场份额。但是佳能和尼康这两大摄影巨头还没有宣布任何计划。毕竟，他们镜子丰富的 DSLR 系列卖了一吨。他们不想太早行动。但尼康可能正准备先下手为强。

尼康谣言已经得到了它认为是[的尼康即将推出的无反光镜系列](https://web.archive.org/web/20230204121257/http://nikonrumors.com/2011/09/09/nikon-v1-and-j1-the-new-mirrorless-cameras-from-nikon.aspx/)的相当合法的规格。根据他们的提示，前两个相机将被称为 V1 和 J1。V1 将是更高端的型号，具有高分辨率 EVF 和多配件端口。J1 将有一个内置的闪光灯，而不是端口。两者都将是 1010 万像素的 3 英寸液晶显示屏，并将与新的镜头系列 CX 系统兼容。

我们看到了传感器的[泄露图，提到的 2.7 倍裁剪系数似乎可以解决这个问题，尽管确实那个镜头可能是他们获得信息的地方。在过去的一年左右，也有一些有趣的专利。早期有一个无反光镜系统](https://web.archive.org/web/20230204121257/https://techcrunch.com/2011/07/21/leaked-picture-of-new-nikon-sensor-suggests-truly-small-mirrorless-system/)[在截面图](https://web.archive.org/web/20230204121257/https://techcrunch.com/2010/08/02/take-a-sneak-peek-at-nikons-evil-patent/)中显示和说明，然后我们看到了一个[新型电动变焦](//web.archive.org/web/20230204121257/http://techcrunch.com/2010/12/24/nikon-working-on-combo-manual-electronic-zoom-for-its-lenses/)的专利。据推测，V1 和 J1 将有一两个他们的竞争对手没有的功能，变焦可能是其中之一。

当 M4/3 系统进入第三代，像索尼的 NEX 这样的系统变得越来越引人注目时，人们会购买第一代尼康系统吗？摄像机会自己说话。还没有日期的消息。
