# YC 支持的即时跑腿移动应用 Exec 筹资 330 万美元

根据美国证券交易委员会(SEC)提交的文件，一款能让人们立即帮你跑腿的移动应用程序 Exec 已经筹集了 330 万美元。该公司的联合创始人贾斯汀·坎(Justin Kan)也是 Justin.tv、Twitch.tv 和 SocialCam 的创始人，他说他还不准备透露投资者，只是说大约有 25 名不同的个人参与了这轮投资。文件只显示了 Exec 的团队，所以很难判断谁是这轮融资的公司或天使投资人。

Exec 是什么？这有点类似于 TaskRabbit 的[，因为你可以通过一个应用程序叫人帮你跑腿。但 Exec 不需要竞标过程，它会立即召集“执行人员”来完成你的任务。它还有每小时 25 美元的固定费率。Exec 负责各种差事——送货、家务、打扫卫生，甚至是艺术。有一位“Exec”曾经打扫过我的房子，也曾指导过 YC 的创始人为演示日做宣传。说真的。](https://web.archive.org/web/20221206232705/https://beta.techcrunch.com/2012/02/29/exec-justin-tv/)

“这笔钱是为了扩大旧金山湾区市场，”菅直人说。
