# Digg 的凯文·罗斯:DiggBar 对互联网有害，所以我们要消灭它

自从正式从杰伊·阿德尔森手中接过代理首席执行官的职位以来，Digg 创始人凯文·罗斯的第一天就在做一些重大的改变。在一篇新的博客[帖子](https://web.archive.org/web/20221224184621/http://about.digg.com/blog/digg-digg-iframe-toolbar-dead-unbanning-domains)中，Rose 写道 DiggBar(见上图)正在被扼杀。他没有像我们经常在公关机器上看到的那样给出通常的“它没有达到我们的期望”的借口，而是非常坦率地承认这是一个糟糕的举动。来自他的帖子:

> “用 iFrame 构建内容对互联网不利。它在添加书签时会造成混乱，破坏 w/iFrame busters，并且无法与下层框架通信(如果您浏览一个故事，旧的 digg 计数仍然存在)。这是一个不一致/不可靠的用户体验，我很高兴地说，当我们推出新的 Digg 时，我们正在杀死它(在这里注册测试)。也就是说，我们将继续为 Firefox、Chrome 和 IE 开发浏览器扩展。几个月后，你会看到这些产品的重大改进。”

Rose 关于杀死 DiggBar 的坦白并非没有先例。去年夏天，DiggBar 推出几个月后，网站[推出了](https://web.archive.org/web/20221224184621/http://techcrunch.com/2009/07/19/diggbar-commits-career-suicide-starts-redirecting-users-to-digg-homepage/)一个极具争议的改变，将 Digg 的快捷链接链接到 Digg.com 页面，而不是你实际想要链接的页面。用户们被激怒了，罗斯开始在推特上说他不知道这个变化，因为他已经休假两周了，他会调查这件事。听起来他并不是主要的支持者。

在今天的同一篇博客文章中，Rose 还指出，当新的 Digg 最终推出时，该网站将解禁任何以前被禁止的域名。我们联系了 Digg，询问他们是否愿意分享这一举动背后的逻辑。

以下是 Digg 首席收入官 Chas Edwards 对解禁网站的决定的评论:

> Digg 社区的成员提交他们在 CNN.com 等专业出版物、TechCrunch 等独立出版物和博客、YouTube 等视频网站，甚至 Toyota.com 或 IBM.com 等商业网站上找到的优质内容。其他读者查看这些提交的内容，并投票选出最佳主页。换句话说，Digg 的读者关心的是质量，而不是出版商是上市媒体公司、财富 500 强消费电子产品制造商，还是在博客上发布图片的独立摄影师。来自 4000 万读者的输入与 Digg 的策略算法相结合，将永远是一个比一小群人执行的政策更好的过滤器。这是我们欢迎品牌加入 Digg 对话的另一种方式。更多方式即将推出。
