# 反击:默多克，德沃尔夫宣布 MySpace 平台和新的隐私控制

今天晚上，在 Web 2.0 大会上，在与约翰·巴特利的问答环节中，鲁珀特·默多克和 MySpace 的联合创始人克里斯·德沃夫宣布了他们围绕 MySpace 平台(上周就有传言)的一些计划。

该平台显然是对 Facebook 平台的回应，并将最终在很大程度上模仿 Facebook 平台，Facebook 平台于 5 月在 T4 推出，并导致脸书上数以千计的第三方应用程序的创建。然而，MySpace 对发布采取了分阶段的方法，这可能有助于他们避免一些脸书在夏天不得不处理的[成长的烦恼](https://web.archive.org/web/20230221222147/https://techcrunch.com/2007/08/16/facebook-takes-action-against-black-hat-apps/)。

今晚还讨论了新的隐私控制。以下是每一项的更多细节。

**MySpace 平台**

MySpace 平台的关键事实:

*   在接下来的几周内，MySpace 将发布一个现有第三方小工具的目录，以帮助用户找到好的内容添加到他们的 MySpace 页面。
*   在接下来的一两个月里，MySpace 将推出一个合适的平台。正如我们在[之前的文章](https://web.archive.org/web/20230221222147/https://techcrunch.com/2007/10/09/myspace-platform-to-launch-next-week/)中提到的，它本质上将是一组 API 和一种新的标记语言，允许第三方开发者创建在 MySpace 中运行的应用程序。开发人员将能够在他们的应用程序中包含 Flash 小程序、iFrame 元素和 Javascript 片段，并访问 MySpace 的大部分核心资源(个人资料信息、好友列表、活动历史等)。).与 MySpace 上现有的小工具不同，开发者将能够访问用户的深层个人资料和其他信息，并将其融入应用程序中。
*   广告可以包括在应用程序页面(称为控制页面)和开发商将保持 100%的收入。然而，广告可能不会放在 MySpace 页面上的小工具中。
*   平台应用不会马上对所有 MySpace 用户开放。他们将有一个测试期，只有 100 万到 200 万 MySpace 用户能够访问这些应用程序。测试期过后，MySpace 的所有用户都可以使用这些应用程序。

 **新增好友选项和隐私控制**

像脸书一样，MySpace 将很快允许不同类型的朋友(家人、朋友、商业联系人等)。接受某人为好友时，还可以设置好友类型。当那个人访问你的 MySpace 个人资料时，他们看到的信息将基于你是什么类型的朋友——商业联系的简历类型的东西，你大学朋友的醉酒兄弟会聚会的滑稽动作，等等。
