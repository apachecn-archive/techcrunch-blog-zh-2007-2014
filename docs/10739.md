开发软件让 Android 用户在所有 Windows PC、平板电脑和笔记本电脑上运行应用程序的初创公司 BlueStacks 今天上午宣布，它已完成 640 万美元的 B 轮融资。这一轮由 AMD、Citrix Systems 牵头，包括现有投资者的参与，如 Andreessen Horowitz、Ignition Ventures、Radar Partners、Redpoint Ventures 等。BlueStacks 在 5 月份筹集了 760 万美元的首轮融资。

该公司计划利用新的资金加速开发其应用播放器(目前处于 alpha 阶段)和“云连接服务”，这两项服务由免费的可下载软件组成，基本上可以让用户在任何 Windows PC、平板电脑或笔记本电脑上一键访问 Android 应用程序，并能够以全屏模式查看这些应用程序。

最近发布的 blue stacks“Windows 应用程序播放器”的公开 alpha 版本[，我们上周报道过](https://web.archive.org/web/20230307110439/https://techcrunch.com/2011/10/11/bluestacks-releases-app-player-and-cloud-connect-service-to-let-you-run-android-apps-on-pcs/)，已经在 100 多个国家下载，该应用程序播放器的测试版定于今年冬天发布，此外还有该产品的“专业”版本，将包括访问水果忍者和切割绳子等优质 Android 应用程序。

BlueStacks 的技术可以与 AMD APU 驱动的平板电脑和个人电脑无缝合作(因此该公司投资了这家初创公司)，这意味着 Android 应用程序可以在许多基于 x86 的系统上利用 APU 的 horspower 和图形功能，这些系统运行在 Windows 和微软 Office 应用程序套件上。

这家初创公司希望为企业定义下一代 IT 架构，并希望帮助推动行业走向能够无处不在地运行移动应用程序，使用户能够在任何地方、任何设备上工作和娱乐。到目前为止，通过 Cloud Connect 可以获得 250，000 多个 Android 应用程序，以及 30 多个可供用户快速抓取和演示的特色应用程序，这已经是 Windows 和 Android 的一大进步。
