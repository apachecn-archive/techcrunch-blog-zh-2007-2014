# IAC 要求更多的谷歌，请

一段时间以来，IAC 已经与谷歌达成协议，将 Ask.com 和其他 IAC 网站上的所有搜索广告移交给这家搜索巨头。2007 年，IAC [将这笔交易](https://web.archive.org/web/20230203144847/http://searchengineland.com/iac-to-split-up-extends-google-ad-deal-worth-35-billion-12613)延期到 2012 年结束，这笔交易当时价值[35 亿美元](https://web.archive.org/web/20230203144847/https://techcrunch.com/2008/04/30/the-new-iac-riding-on-googles-coattails/)。根据今天早上发布的新闻稿，IAC 和谷歌已经续签并延长协议至 2016 年 3 月 31 日。

根据新闻稿，谷歌通过该协议向 IAC 的搜索产品和网络合作伙伴提供赞助列表和其他搜索相关服务，与之前的协议相当。

IAC 首席执行官格雷格·布拉特(Greg Blatt)在一份声明中表示，其搜索业务的营业收入已从 2006 年摊销前的 5800 万美元增长到 2010 年的 1.25 亿美元，并在此期间产生了超过 5 亿美元的税前自由现金流。

将搜索广告和技术交给谷歌对该公司来说是一个很好的赌注。谷歌实际上为 IAC 的 Ask.com 的搜索结果贡献了一部分，去年秋天，IAC 宣布将[完全外包](https://web.archive.org/web/20230203144847/http://www.bloomberg.com/news/2010-11-09/iac-s-diller-surrenders-to-google-juggernaut-ends-ask-com-search-effort.html) Ask 的搜索(尽管不清楚是哪家公司，可能是谷歌)。
