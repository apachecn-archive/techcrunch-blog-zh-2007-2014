# THQ CEO 称索尼的运动控制器实际上被命名为 Arc，而不是 Gem

[![](img/caecd570c20fd110e42b7cba6fd6a480.png "500x_arc") ](https://web.archive.org/web/20230322164148/http://www.crunchgear.com/2010/02/03/thq-ceo-says-sonys-motion-controller-is-actually-named-arc-not-gem/) [上次我们听说](https://web.archive.org/web/20230322164148/http://www.crunchgear.com/2009/12/10/the-sony-motion-controller-is-named-gem/)，索尼的运动控制器被命名为 Gem。这是根据 EA 的首席执行官约翰·里奇泰洛在 12 月初的说法。但是 THQ 的负责人今天在一次投资者电话会议上说它被命名为 Arc。不管怎样，对我们来说，它仍然像是一个新奇的 Wii 遥控器仿制品。通过。(除非有一个杀手级游戏，然后我们会先赶潮流)
