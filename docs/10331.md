# 还记得网飞想在另一个网站上租 DVD 吗？是的，那是有趣的一周

啊，七月。我们非常想念你。那时的世界更新，空气更清新，人们也更友好，而且曾经有一个叫 Qwikster 的东西。Qwikster 是为了解决向仍在使用光学媒体的人出租视频这一令人头疼的问题而建造的，它本应是网飞的一部分，仍然将光盘放在小袖子里。另一方面，网飞本身只会在互联网上传输视频，两人永远不会见面。

写道[网飞](https://web.archive.org/web/20230404010947/http://blog.netflix.com/2011/10/dvds-will-be-staying-at-netflixcom.html):

> 很明显，对于我们的许多成员来说，两个网站会让事情变得更加困难，所以我们将继续把网飞作为流媒体和 DVD 的一个去处。
> 
> 这意味着没有变化:一个网站，一个账户，一个密码…换句话说，没有 Qwikster。

没错:Qwikster 死了。网飞首席执行官雷德·哈斯汀斯表示，该计划是被误导的，这将是他们在很长一段时间内最后一次对价格进行争论。考虑到实际上没人会拼写 Qwikster (Quixter？快克斯特？Qkstr？)更不用说创建两个账户，一个用于流媒体，一个用于偶尔播放的 DVD，这对网飞来说是相当狡猾的举动。我只是想知道是什么把他们推到了悬崖边？摄取量低？当他们提到这个名字的时候，你觉得很傻吗？我在网飞有一条线，现在问他们，但在那之前，陶醉在一个少了一个独立 DVD 租赁服务的世界。

你会记得，早在 7 月份我们录制《不成功便成仁》时，Erick 对 Qwikster 的看法是正确的。
