# Videophlow 让 YouTube 成为一种群体体验 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/04/24/videophlow-makes-youtube-a-group-experience/>

[![](img/1cd8c2e0a19210580f1c39f753b882fc.png)](https://web.archive.org/web/20221206221933/http://www.crunchbase.com/company/oortle)

想象一下:你在网上找到了下一个热门视频。笑到流泪后，你又恢复了镇静，给你最好的朋友发了一个视频链接，期待别人称赞你非凡的幽默感。那你等着。五分钟过去了，仍然一无所获。他们喜欢吗？他们看了吗？最后你的付出换来一句空洞的“LOL”。

我们每个人都经历过。互联网在分享媒体方面做得很好，但它从观看体验中去除了人的因素。

奥尔特正试图改变这种状况。2007 年末，他们推出了一款名为 [Photophlow](https://web.archive.org/web/20221206221933/http://www.photophlow.com/invitation_requests/new) 的网络应用，目前处于私人测试阶段，其核心是围绕照片的群聊(尽管它也包括许多其他功能)。用户可以浏览 Flickr 相册，每个人都会在同一时间看到相同的内容，从而可以进行实时反应和讨论。

[![](img/334c31249e62f46e6cc34a5aa74bbd4b.png)](https://web.archive.org/web/20221206221933/https://beta.techcrunch.com/wp-content/uploads/2008/04/videophlowbig.png)

现在，他们正采取下一步措施，推出 Videophlow，将同样的动态社区体验应用到 YouTube 上。一群朋友在聊天、表情符号和手势互动的同时观看相同的视频——观众甚至可以向屏幕投掷虚拟番茄，并伴有动画啪嗒声。

所有用户将同时看到视频的相同部分，即使他们跳到不同的场景。最棒的是，群组可以无缝过渡到新视频。这有机会成为一个大热门——我可以很容易地想象一群朋友在深夜观看马拉松比赛时交换(和观看)他们最喜欢的视频。

今天下午，Oortle 将在 Web 2.0 发射台首次向公众展示 Videophlow，并将在两周内开放私人测试。我们会让你知道什么时候发生，到时候我们会发出至少 100 个邀请。

我们应该注意到，一个非常类似的应用程序可以从 [SeeToo](https://web.archive.org/web/20221206221933/http://www.seetoo.com/) 获得，它允许同时观看来自用户计算机而不是 YouTube 的视频内容。
