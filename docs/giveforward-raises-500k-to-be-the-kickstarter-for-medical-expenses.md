# GiveForward 筹集了 50 万美元作为医疗费用的 Kickstarter

![](img/832f11da996e6f7fb3930a919044012d.png)医疗费用在线筹款工具 GiveForward 已经从蒂姆·克劳斯科普夫、[新世界投资](https://web.archive.org/web/20221004113811/http://www.crunchbase.com/financial-organization/new-world-ventures)、[大卫·科恩](https://web.archive.org/web/20221004113811/http://www.crunchbase.com/person/david-cohen)、埃德·钱德勒、[社会杠杆](https://web.archive.org/web/20221004113811/http://www.crunchbase.com/financial-organization/social-leverage)和 [Excelerate Labs](https://web.archive.org/web/20221004113811/http://www.crunchbase.com/company/excelerate-labs) 处筹集了 50 万美元的资金。

GiveForward 是在芝加哥的 [Excelerate Labs](https://web.archive.org/web/20221004113811/http://www.exceleratelabs.com/) 孵化的[，想要成为医疗费用的 Kickstarter。该平台允许任何人创建可定制的筹款页面，来自世界各地的朋友和家人可以在线捐款。](https://web.archive.org/web/20221004113811/https://beta.techcrunch.com/2010/07/11/nine-startups-graduate-from-chicagos-excelerate-labs/)

为了赚钱，GiveForward 扣除了所有捐款的 7%。自 2008 年以来，已有 10，000 名用户加入了这家初创公司 GiveForward，帮助这些家庭筹集了近 350 万美元，用于支付他们亲人的自付费用。GiveForward 还为用户提供筹款技巧，允许他们在脸书和 Twitter 等社交网站上发布自己的页面。
