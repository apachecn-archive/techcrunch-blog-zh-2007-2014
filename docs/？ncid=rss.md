# Sookasa 轮融资 500 万美元，帮助公司安全使用公共云服务

Sookasa 希望帮助公司使用 Dropbox 和 Gmail 等流行的云服务，同时透明地加密敏感数据，以确保公司不会违反 HIPAA 和 FERPA 等法规。该公司今天宣布，它已经筹集了 500 万美元的 A 轮融资，由 Accel Partners 牵头，现有投资者首轮资本、SV Angel 和其他天使投资者参与。

众所周知，许多 It 组织都在努力确保合规性，同时让用户能够在多种设备上访问 Dropbox、box 和 Gmail 等工具。Sookasa 表示，它为企业提供了“一种自助式交钥匙加密和合规性解决方案，使其员工能够安全地使用他们最喜爱的移动设备和云服务。”该服务确保文件总是被加密，无论它们在什么设备上，甚至当它们被外部共享时。

Sookasa 于 2012 年由[父子团队 Israel 和 Asaf Cidon](https://web.archive.org/web/20230209131219/https://www.sookasa.com/our-team/) 创立。阿萨夫现在是 Sookasa 的首席执行官。两人担心他们共享的文件分散在多个云服务和设备中，并且没有集中的方法来管理它们。

“与保护网络或终端等基础设施的传统安全公司不同，Sookasa 专注于保护内容本身，同时保护用户体验，”Asaf 说。“我们的云合规服务旨在跨任何云服务、计算机或移动设备运行。”

该公司的第一款产品，也是今天发布的，是 Sookasa 云合规服务，允许公司控制和审计跨云服务、用户和设备的文件，并透明地加密它们。

在新一轮融资中，来自 Accel Partners 的 Sameer Gandhi 加入了 Sookasa 的董事会。此外，曾在 VMware 和 EqualLogic 担任高管职务的 Kirk Bowman 也加入了董事会，成为独立董事。
