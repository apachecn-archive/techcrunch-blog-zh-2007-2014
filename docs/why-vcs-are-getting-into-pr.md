# 风投为何进入 PR 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/12/why-vcs-are-getting-into-pr/>

风险投资家比以往任何时候都更热衷于此。为了保持竞争力和领先地位，风投不再出借他们的建议和资金，而是提供投资组合公司需要的实际“核心”服务。这种转变不一定是新的。例如，凯鹏华盈(Kleiner Perkins)和格雷洛克(Greylock)等顶级风险投资公司在帮助招聘人才方面发挥着实际作用。最近，我们看到公司卷起袖子准备另一项核心能力:公共关系。

更多的时候，我会被一家投资公司的内部公关代表介绍一个关于初创公司的故事，这家投资公司已经投资了这家初创公司。传统上，无论是初创公司还是外部公关机构，在发布会、潮流作品、融资或其他新闻时，都倾向于进行宣传。风险投资公司自身(相对于创业公司)推介的增加让我开始思考风险投资角色的演变，特别是在公关的背景下。但在过去一年左右的时间里，我们看到这些公司增加了顶级公关和传播人才，特别是帮助投资组合公司制定公关战略。

业内第一批大规模招聘之一[是最大的技术和媒体公关公司 OutCast Communications 的联合创始人 PR honcho Margit Wennmachers](https://web.archive.org/web/20230326045025/http://allthingsd.com/20100614/outcasts-wennmachers-joins-andreessen-horowitz-as-partner/) 加入 Andreessen Horowitz 成为合伙人。当时，马克·安德森告诉 AllThingsD 的卡拉·斯威舍，温马克的品牌建设和公关经验将是公司及其投资组合的宝贵资产。请注意，安德森·霍洛维茨在创办公司仅一年后就明智地意识到，建立一个风投中心不仅仅是开支票，还是一种咨询服务。

[大卫·克莱恩](https://web.archive.org/web/20230326045025/http://www.googleventures.com/team/david-krane)，谷歌前全球传播和公共事务总监，也是该公司初创时的第一批公关代表之一，在谷歌风险投资公司身兼两职，他现在是该公司的合伙人。他投资初创公司，但也帮助谷歌投资组合公司的公关策略。

2010 年 4 月，Sand Hill 风投公司凯鹏华盈(Kleiner Perkins)聘请 Hulu 前公关主管克里斯蒂娜李(Christina Lee)担任该公司的营销和公关主管。去年秋天，公关大师 Kiersten Hollars(前 Outcast，Yahoo，Digg 和 AOL) [加入了 Andreessen Horowitz](https://web.archive.org/web/20230326045025/https://techcrunch.com/2011/12/12/kiersten-hollars-moves-from-aol-comm-vp-to-andreessen-partner/) 的 Wennmachers 团队，专注于为当时公司投资组合中的 80 家公司扩展营销和公关工作。最近，我们看到谷歌公关部高级经理 Andrew Kovacs 跳槽到红杉，帮助该公司的营销合作伙伴 Mark Dempster 负责该公司在美国和以色列的公司的公关和沟通工作。

很明显，技术公关行业中一些最聪明、最受尊敬的人才不再仅仅在代理公司或公司内部。正如美国国家风险投资协会(National Venture Capital Association)交流副总裁艾米丽门德尔(Emily Mendell)告诉我的那样，风险投资市场对交易流的竞争非常激烈。“有很多公司在最热门的早期创业公司中追逐交易，公关代表和沟通策略只是这些公司在谈判桌上可以提供的另一项服务，”她说。

技术公关不是一件容易的事情——要做好这项工作，它需要对产品的技术理解，以及以可沟通的方式消化和传递这些信息的能力。门德尔解释说，初创公司意识到，投资公司手头有这样的东西对他们来说是一笔资产。

上面提到的许多公关经理只专注于帮助投资组合公司，而不是为公司本身做内部沟通。Lee 告诉我，越来越多的风险投资公司聘请公关人才的趋势，部分反映了传播在整个科技行业的角色正在发生变化。

她表示:“现在，聪明的企业家明白了对消费者透明的价值，沟通的作用已经上升到了战略层面。”。“公共关系预计将在品牌建设中发挥重要作用，越来越多的初创公司在早期阶段就开始考虑公共关系和品牌建设。”

她解释说，具体来说，凯鹏华盈正在采取一种全面的公司建设方法，其中一部分是为初创公司和公司提供沟通和公关帮助。

![](img/da98755cac946233d129df3f17adb447.png)当然，风险投资公关代表可以通过多种不同的方式帮助企业家和公司，从简单的任务，如向科技记者介绍，到实际领导产品发布的公关工作，到指导公司的品牌和信息战略，再到起草新闻稿。

另一位业内人士告诉我，虽然一些公司正在为投资组合公司引进公关人才，但其他公司正试图利用公关来提升自己的形象，并向媒体宣传他们的风投。

我认为，推动内部建立销售代表甚至小型代理机构趋势的一个因素是，这些风险投资公司种子阶段投资的增加以及这一层面的竞争市场。现在不缺愿意在种子级别开支票的风投和天使。许多公司不能仅凭声誉就获得交易流量。但创投公司成为“全方位服务”机构的能力可能是创业者坐下来谈判时要考虑的一个因素。一位业内人士表示:这对于初创公司来说尤其重要，因为它们可能没有钱聘请顶级公关公司来处理公关事务。

如上所述，这种全套服务的一部分还包括招聘，这是一种已经发生了一段时间的趋势。凯鹏华盈合伙人 [Juliet de Baubigny](https://web.archive.org/web/20230326045025/http://www.kpcb.com/partner/juliet-de-baubigny) 在过去的十年里一直在帮助投资组合公司进行人力资本和招聘。Greylock [最近创建了一个专门的人才团队](https://web.archive.org/web/20230326045025/https://techcrunch.com/2011/10/17/greylock-looks-to-help-portfolio-companies-recruit-talent-with-new-hires/)来帮助其投资组合公司有效地招聘人才。像安德森·霍洛维茨和红杉这样的公司也在积极地帮助创业公司建立网络和招募人才。

有趣的是，Y Combinator、TechStars 和 500 Startups 等孵化器已经通过内部人才和导师提供了这种“全面服务”模式。虽然把上面提到的风险投资公司称为孵化器有些牵强，但看到这些大公司的整体功能随着当前的投资状态而发展肯定是很有趣的。

随着安德森·霍洛维茨(Andreessen Horowitz)、凯鹏华盈(Kleiner Perkins)等公司创建这些提供全方位服务的风险投资基金，它们不仅发放现金，还配备内部公关机构、营销资源和人才招聘人员，毫无疑问，这种趋势将渗透到较小的公司和基金。
