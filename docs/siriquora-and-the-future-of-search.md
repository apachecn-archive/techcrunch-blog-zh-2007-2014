# Siri、Quora 和搜索的未来

***编者按**:撰稿人[丹·卡普兰](https://web.archive.org/web/20230205012446/http://www.quora.com/dan-kaplan)负责[特维利奥](https://web.archive.org/web/20230205012446/http://www.twilio.com/)的产品营销，偶尔写一些关于现在对未来的推断。*

随着 Google+的崛起，知名科技人士有争议的发布活动的减少，以及其他闪亮新事物的诱惑，大多数科技媒体已经将目光从我最喜欢的 2010 年初创公司 [Quora](https://web.archive.org/web/20230205012446/http://www.quora.com/) 转移开了。

既然苹果已经将有史以来最复杂的人工智能集成到了它的 iPhone 4S 中，我认为是时候掸掉 Quora 肩上的灰尘，照亮公司的未来了。

当大多数不了解 Quora 的人把它当成“另一个问答网站”时(或者不管他们说了什么，然后他们把它一笔勾销)，他们错过的是:当他们在 2009 年秋天首次推出 Quora 时，Quora 的创始人和他们的第一个雇员——设计师[丽贝卡·考克斯](https://web.archive.org/web/20230205012446/http://www.quora.com/Rebekah-Cox)——创造了有史以来最令人印象深刻的“主观知识提取”机器的核心。(是的，维基百科理应被誉为这个领域的第一巨头，但 Quora 最终会继承它的衣钵。与此同时，你可能会认为整个互联网是有史以来构建的最令人印象深刻的主观知识提取技术，但这只是语义上的)。

通过将答案投票机制和奖励成瘾循环(upvotes 是 crack)与严格的身份要求和一对多的追随者模型相结合，Quora 开始解决从人类的集体头脑中提取高质量的经验知识并将其转化为互联网上的结构化形式的问题。更何况 Quora 也是在利用人类的集体智慧进行排名。

有了这个引擎，Quora 正在建立一个人类经验的数据库，这个数据库可能最终包含未来携带 iPhones 的人们可能会有的许多问题的答案。

这又让我想起了 Siri。

对于那些还没有想通或者还没有玩过 iPhone 4S 的人来说，Siri 是一项改变游戏规则的技术:它知道如何将人类语言的乱码翻译成有针对性的 API 调用，然后从潜在的不断扩大的数据库中提取正确的信息(假设苹果有一天会将其他数据库集成到 Siri 中，我相信它会的)。Siri 和我们可能的问题的最佳答案之间的主要障碍是，包含这些答案的数据库仍在进行中。

正在进行的工作是 Quora，这可能是为什么我听到传言说，一些大规模的搜索和广告公司将匿名，直到下一段据称提出支付超过$1B 收购它。

如果这个传言是真的，这意味着谷歌看到了 Quora，明白了威胁的严重性。如果这不是真的，这意味着谷歌高层决策人员没有注意到这一点。正如我三月份在[一篇关于 Quora 和谷歌](https://web.archive.org/web/20230205012446/http://www.quora.com/Dan-Kaplan/The-Quora-Review/Forget-about-Facebook-Google-Start-fearing-Quora)的帖子中所写的:

> 考虑这样一个互联网，在这个互联网上，对我们大多数查询的最佳答案不是来自于浩瀚的、日益嘈杂的广阔的万维网，而是来自于最善于表达的专家的集中知识和经验。在这里，你不再通过 10 个蓝色链接(或数百个链接)来寻找你想要的东西；你只需输入你的问题，就会得到最好的回答。如果你发现自己在问一个以前没有人问过的问题，你只需要把它加入到这条溪流中，让它流向最能回答它的人。

把 Siri 作为这些查询的主要界面，这就差不多结束了:如果 Quora 的优秀团队成功地跨越了热情的早期采用者和其他善于表达的人之间的鸿沟，他们的公司最终可能会和 Siri 一起，成为谷歌核心业务的一个存在性危险。
