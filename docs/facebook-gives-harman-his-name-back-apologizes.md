# 脸书把名字还给哈曼，并道歉

昨晚我们帖子的更新——[脸书抓取用户的虚荣心网址，然后卖给哈曼国际](https://web.archive.org/web/20221006071750/http://www.beta.techcrunch.com/2010/01/23/facebook-vanity-url-harman/)。脸书表示，他们将很快归还哈曼巴杰瓦的[/哈曼](https://web.archive.org/web/20221006071750/http://www.facebook.com/harman)虚荣心网址:

> 谢谢你提醒我们，迈克。在这种情况下，我们犯了一个错误，正在将用户名返回给 Bajwa 先生。需要说明的是，此举并非出于金钱原因，而是为了保护商标。我们努力保护商标免受“抢注者”的侵害，抢注者是指那些试图在没有合法主张的情况下使用受保护条款的人。哈曼收到的信息是沿着这些路线，但显然不适用。一旦我们理解了我们错误的本质，我们就迅速行动去解决它。我们要向巴杰瓦先生道歉，因为我们的努力过于热心，并对他的帐户中断表示歉意。

[这里的](https://web.archive.org/web/20221006071750/http://www.facebook.com/help/?page=899)是关于商标抢注的脸书准则。这对耐克或可口可乐来说没问题，但对于作为人名合法使用的商标来说，这绝对是一个灰色地带。我真想花钱看看今天早上在脸书满天飞的电子邮件。但他们在这里做了正确的事，这才是最重要的。
