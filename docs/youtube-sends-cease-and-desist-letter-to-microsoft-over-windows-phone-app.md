# YouTube 通过 Windows Phone 应用向微软发送停止信

嗯，这很快:就在一周前，微软为其 Windows Phone 8 平台发布了一款 [YouTube 应用。今天，YouTube 告诉微软移除它，称该应用](https://web.archive.org/web/20230403153302/http://www.windowsphone.com/en-us/store/app/youtube/dcbb1ac6-a89a-df11-a490-00237de2db9e)[违反了其服务条款](https://web.archive.org/web/20230403153302/http://readwrite.com/2013/05/09/microsoft-youtube-app-rule-breaker-strips-ads-downloads-video)。

YouTube 全球平台合作伙伴总监弗朗西斯科·瓦雷拉(Francisco Varela)今天向 Windows Phone 应用程序总经理托德·白利(Todd Brix)发送了一封停止函，要求他的公司下架微软创作的应用程序。这封信声称，该应用程序允许用户从 YouTube 下载视频，同时也从它显示的视频中剥离广告。它还显示在某些平台上被限制播放的视频——比如当一家主要的媒体公司没有给 YouTube 在手机或平板电脑上显示视频的权利时。

所有这些行为都违反了 YouTube 的[服务条款](https://web.archive.org/web/20230403153302/http://www.youtube.com/t/terms)和 [API 服务条款](https://web.archive.org/web/20230403153302/https://developers.google.com/youtube/terms)，导致其要求微软禁用现有下载。该公司声称，这款应用也让用户感到困惑，因为它使用了 YouTube 的品牌和商标，尽管是由第三方创建的。鉴于所有这些，YouTube 表示给微软一周时间，直到 5 月 22 日星期三，撤回该应用程序。

YouTube 表示，超过 100 万个频道通过其合作伙伴计划获得收入，其中包括数千个年收入超过六位数的频道。这封信认为，微软的应用剥夺了他们在 Windows Phone 8 应用上赚钱的能力。这也侵犯了他们决定其内容在哪里以及如何在哪些设备上显示的能力。

就其本身而言，在 YouTube 拒绝投入任何资源来支持 Windows Phone 设备使用已获批准的原生应用程序后，微软做出了前所未有的决定，自己创作一个应用程序。但 YouTube 指出，它自己的 HTML5 标准网络体验是运行微软移动操作系统设备的用户的另一种选择。
