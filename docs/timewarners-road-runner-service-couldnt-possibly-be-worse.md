# 时代华纳的 Road Runner 服务差得不能再差了

[![twlies.jpg](img/e8a0f83febaafb28f6143c49a0c573ac.png)](https://web.archive.org/web/20210226214653/http://old.crunchgear.com/wp-content/uploads/twlies.jpg "twlies.jpg")

好吧，我就直说了:时代华纳的 Road Runner 宽带网络连接是我用过的最差的。在这里，我每月支付大约 100 美元的有线电视和互联网连接费用，但当我在 iMac 上启动传输时，我的互联网连接速度慢得像蜗牛。如果这个问题不能自行解决，我可能不得不考虑更换提供商，甚至在签约一个月后也不行。

我不会时不时地讨论我的电缆是如何切断的，因为这并不完全是一个技术问题。

事情是这样的:我最近从纽约北部的一个偏僻的地方搬到了皇后区。在北部，我有有线电视的 OptimumOnline。除了 OOL 限制了它在新闻组的下载速度，服务也很棒。我可以打开 Transmission 或 Azureus，播种一些种子，在任意数量的私人 BitTorrent 追踪器上建立我的比率。我说的不是成百上千的种子或者其他什么，而是一次可能有六七个。然后我移动，发现一旦我打开传输，我的连接融化得比切尔诺贝利还快。这里，照片可以证明这一点。
