# 东芝-富士通 IS12T:世界上第一个 Windows 芒果手机上升和关闭(视频)

两天前，日本东芝公司发布了 IS12T，这是世界上第一款运行 Windows Phone 7.5 的手机。我们很快给了你[第一个规格](https://web.archive.org/web/20230203205105/https://techcrunch.com/2011/07/26/official-japan-to-get-the-worlds-first-windows-phone-7-mango-handset-in-september)，但我们现在有一个视频显示 1 Ghz 高通 CPU 供电的设备在运行中-加上一些更详细的规格。

首先，这里是主要规格(更详细)，正如日本 IS12T 的供应商[宣布的](https://web.archive.org/web/20230203205105/http://www.kddi.com/corporate/news_release/2011/0727/index.html)【JP】，日本当地移动运营商 [KDDI](https://web.archive.org/web/20230203205105/https://techcrunch.com/tag/kddi) :

*   3.7 英寸显示屏分辨率为 800×480
*   1320 万像素摄像头采用 CMOS 传感器，由 Milbeaut Mobile engine 驱动，可拍摄分辨率为 1280×720 的视频
*   防水防震体
*   microUSB 端口
*   32GB 内部内存(无额外卡槽)
*   IEEE 802.11b/g/n 无线网络(无网络共享)
*   蓝牙 2.1+EDR
*   DLNA 支持
*   Zune 品牌的多媒体功能
*   Internet Explorer 9 作为浏览器
*   Xbox LIVE 连接
*   Windows Live SkyDrive 访问
*   待机时电池续航时间:280 小时，连续通话时间:400 分钟
*   尺寸:118×59×10.6 ~ 13.3 毫米，重量:113 克
*   已确认:有三种颜色(黄色、黑色、粉色)
