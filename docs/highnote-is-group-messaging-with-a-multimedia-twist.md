# HighNote 是带有多媒体特色的群发消息

群发短信应用 [HighNote](https://web.archive.org/web/20221006163233/http://www.highnote.com/) 除了当前的群发短信应用浪潮，如 [Kik](https://web.archive.org/web/20221006163233/http://www.kik.com/) 、 [PingChat](https://web.archive.org/web/20221006163233/http://www.pingchat.com/) 和 [Beluga](https://web.archive.org/web/20221006163233/http://belugapods.com/) 之外，它还增加了多媒体功能。

除了是一个免费的基于互联网协议的会员信息应用程序之外，这项服务(目前在 iPhone、iPad、iTouch 和 Android 上可用)在你可以发送和接收的信息的丰富性和范围方面是独一无二的。

有了 HighNote，你可以分享音乐、视频(通过 iTunes)、录音、图片、地图和几乎任何适合手机的媒体。

*“在 HighNote，我们已经认识到需要一种新的传播媒介，这种媒介能够捕捉文本的简单性和即时性，但具有更大程度的信息丰富性，旨在实现更强大、更有针对性的社交沟通，”*首席执行官 Marc Barach 说。

HighNote 有 20 多种可定制的问候方式，如*“看看这个”*和*“着火了”*以及更改文本对齐方式、颜色、样式和大小的能力，它可能对那些刚刚开始短信生涯的人最有吸引力，尤其是它有大量的表情符号。

是的，这款应用显然像其他社交服务一样瞄准了青少年市场。它有一个全面的“快速回复”部分(给你像“喜欢它”/“讨厌它”这样的选项)，以及一个名为“脉冲”的功能，据我所知，它让用户选择像“音乐”、“体育”、“名人八卦”和“时尚”这样的测验，更重要的是，它充满了广告定位的潜力。

HighNote 会员可以免费通过 HighNote 发送丰富的消息。这款应用不像 Kik 那样在战略上像病毒一样传播，它发送普通的短信，带有多媒体内容的浏览器链接，并向非会员发出邀请。黑莓，Symbian 和 Windows Phone 7 版本应该很快就会上市。
