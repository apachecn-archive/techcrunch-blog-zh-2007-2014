# 现在是你的智能手机的下一个技巧:看和理解，由 Google  提供

> 原文：<https://web.archive.org/web/http://techcrunch.com/2014/02/20/google-project-tango-smartphones-that-see/>

你的智能手机拥有一组传感器，在它光滑的外壳中做许多事情。但是多亏了一个新的项目，[称之为探戈](https://web.archive.org/web/20230316043343/https://techcrunch.com/2014/02/20/google-launches-project-tango/)，由[谷歌的先进技术和项目(ATAP)组](https://web.archive.org/web/20230316043343/https://techcrunch.com/2014/01/29/google-keeps-motorolas-advanced-technology-group-home-of-the-modular-phone-and-password-tattoo/)发起，你的下一个可能会有一个额外的超能力:视觉空间意识。比如，你的下一部智能手机可能不仅能看到，还能理解周围的环境。

如果这看起来像是科幻小说中的东西，那是因为它很可能是——例如， *Her* 中的人工智能助手设法变得如此令人信服，因为它分享了用户世界的意识，包括识别环境并将其置于适当的环境中。据 ATAP 项目负责人李炯楷称，谷歌的新[原型硬件和开发套件](https://web.archive.org/web/20230316043343/http://www.google.com/atap/projecttango/)，以及其 Myriad 1 [Movidius 视觉处理器平台](https://web.archive.org/web/20230316043343/https://techcrunch.com/2014/02/20/inside-the-revolutionary-3d-vision-chip-at-the-heart-of-googles-project-tango-phone)，正是试图赋予移动设备这种类型的“对空间和运动的类似人类的理解”。

## 这不仅仅是一台别致的相机

需要说明的是，这不仅仅是智能手机的新摄像头——它更像大脑的视觉皮层，像智能手机中的任何加速度计或陀螺仪一样被制成设备组件。在某种程度上，这是下一代移动技术的战场；例如，苹果拥有 M7 运动协处理器，高通正在将先进的相机处理技术内置到其 SOC 中，这将允许用户在拍摄后改变照片的焦点，并做更多其他事情。

但是谷歌的最新实验有潜力做得比这些现有的创新多得多。计算机视觉是一个学术、商业和工业研究的丰富领域，其含义几乎延伸到我们生活的每个方面。更好的是，谷歌合作伙伴用于实现这一目标的技术是专门设计的，电池寿命是首要考虑因素，因此它被设计为一种永远在线的技术，而不是只能在特定情况下调用的技术。

## 语境化请求

那么这些对于用户体验来说意味着什么呢？说“期待巨大的变化”是老生常谈，但却是事实，但在这个阶段，推动因素是让开发人员思考如何利用这项新技术。因此，现在确切地说他们将开发什么样的软件来付诸实践还为时过早。不过，有一点很清楚——背景将是关键。

谷歌现在已经提供了一些线索，表明当智能手机彻底了解其用户时，可能会发生什么，以及在给定时间、位置和包括电子邮件、日历和其他公开信号在内的输入的情况下，所述用户的需求可能是什么。综合起来，这些可以很好地描述我们所说的“背景”，或者构成任何给定情况的环境的总和。但最终，就目前的操作而言，你的智能手机实际上是在黑匣子中工作，表面上零星地切出一些针孔，让光束穿过，这些光束部分照亮，但不一定真正位于其中。

例如，通过对周围环境的了解，虚拟个人助理可以知道，尽管你在公共汽车站附近，但你目前正在与一位业务联系人握手，并在坐下来喝咖啡之前放下了你的包。例如，在这种情况下，提供当地公交车的到达时间不如打电话给电子邮件链确认会议重要。

但是视觉意识对虚拟个人助理的价值只是一个例子，而且鉴于最近对这一问题的虚构描述，这一点很容易理解。知道手机在哪里使用也有助于将更深远的概念带入生活，包括根据游戏地点改变设置、环境和角色的游戏；情境广告，其与附近的多媒体显示器交互，并基于用户在哪里以及他们必须查看什么类型的产品来针对/吸引用户；即使是宏观水平的设置变化，以确保您的手机准备好，以满足您的需求给你目前的情况。

至于最后一个例子，它可能有助于引入火狐、谷歌和其他公司一直在玩弄的动态移动操作系统。[我在最近的 TechCrunch Droidcast 上认为，上下文发布者还没有为黄金时段](https://web.archive.org/web/20230316043343/https://techcrunch.com/2014/02/09/tc-droidcast-episode-21-is-flappy-bird-really-gone-and-are-contextual-launchers-useful/)做好准备，因为他们经常弄错上下文；同样，他们在一个黑盒子里工作，只有短暂和零星的一瞥来理解他们周围的世界，这项新技术将他们带到了户外。结果可能是不需要手动静音的设备，或者自动提供正确的主屏幕或应用程序来满足您的需求，而无需告诉您这样做。

## 巨大的数据潜力

当然，如果这个项目没有数据角度，谷歌就不会做谷歌:这种信息可能会被一种设备收集，由用户携带到任何地方，不仅可以看到，还可以了解周围的情况，这种信息非常强大。

谷歌的整个业务是围绕着它了解客户的能力，以及知道他们在任何给定时间想看什么的能力而建立的。该搜索引擎依靠极具针对性的基于网络的文本广告赚钱，每当用户在其引擎中键入查询时，这些广告都会返回高度相关的结果——这几乎是他们对该主题感兴趣的简单标志。现在听起来似乎是显而易见的，但在搜索的黎明，这个简单的等式像闪电一样击中了整个广告行业，并继续为山景城带来巨大的收入。

即使是谷歌著名的雄心勃勃的“登月计划”，在其令人印象深刻的消费者潜力背后都有一个最初目标的线索，Tango 也不例外。声明的意图，就像人们在谷歌上搜索东西时表达的意图，只是找出消费者欲望的等式的一部分——在需求出现之前预测需求，并理解一个人可能甚至没有意识到自己的需求，这构成了谷歌这样的公司更广阔的蓝天机会。在这种情况下，拥有一部能够观察和解释周围环境的上下文感知智能手机，几乎就像是在任何给定的时间，让一名专门的市场研究员与任何给定的购物者共处一室。

至少与过去十年中每一项值得注意的移动技术发展一样，Project Tango 将寻求更多地进入用户的生活，以换取更多更好的服务。一旦开发者开始向我们展示一旦智能手机能够理解它的位置和正在发生的事情，可能会发生什么，我愿意打赌用户会发现数据成本完全可以接受。

## 谷歌并不孤单

谷歌不是唯一一家致力于感知移动设备的公司，也不会是唯一一家帮助将具有视觉智能的设备推向市场的公司。[苹果去年收购了 PrimeSense](https://web.archive.org/web/20230316043343/https://techcrunch.com/2013/12/03/apple-primesense-and-perceptive-computing-or-why-your-phone-will-see-in-3d/) ，这是一家开发运动传感技术并帮助绘制 3D 空间地图的公司。早在 2011 年，高通就收购了类似的公司 GestureTek。

当基于位置的技术第一次被引入移动设备时，它看起来很科幻，但现在它是必要的。我们未来购买的设备的情境意识也是如此。谷歌是第一个开始将这种力量放在开发者手中，看看这对软件的未来意味着什么的人，但它不会是唯一的一个。期待看到移动技术这一新领域的飞速发展，期待每一个主要参与者都能在谈判桌上占有一席之地。
