# 康卡斯特/Level 3 纷争的真正原因是:钱 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/11/30/what-the-comcastlevel-3-fracas-is-really-about-money/>

标题相当粗糙:康卡斯特讨厌网飞！网络中立快不行了！来自俄罗斯和古巴的共产主义力量正在攻击科罗拉多州的一个小镇，一群乌合之众的高中生联合起来反抗他们(尽管，可以说，这可能与康卡斯特/Level 3 无关)！但这到底是怎么回事？

首先，让我们了解数据是如何从云中到达您的手中的。回到过去，当你想在网上提供服务时，你租了一条 T1 线路，安装了一台机器，并希望有人会来查看你的商品。这台服务器依次连接到一个主干网，然后 ISP(过去是提供拨号服务的小夫妻店，现在是不知名的公司)把数据给你。就像一系列的管子，你知道吗？那是在 Slashdot 和 Digg 这样的网站对流行内容产生巨大影响之前，公众认为通过互联网在电视上观看电影很不错。结果，数字流量上升到惊人的速度，所有相关人员——从使用 T1 线路的人到 T1 线路提供商，再到在家使用电缆调制解调器的人——都必须升级。和升级。和升级。从这个角度来看，我们只有*真的*在过去十年左右有这个问题，而且技术进步如此之快，几乎就像运营商在冲刺——他们确实在冲刺。反过来，这也使得从[公共交换电话网(部分是机械的)](https://web.archive.org/web/20221007011300/http://en.wikipedia.org/wiki/Public_switched_telephone_network)到数字电话交换的 30 年转变看起来像是从纽约到南极洲的悠闲散步。

所以这种东西要花很多钱，运营商不是出于好心才这么做的。他们希望为他们的数据中心付费。这就是第三级的用武之地。第三层既是主干网——意味着一个巨大的全国性数据载体——也是一个[内容交付网络](https://web.archive.org/web/20221007011300/http://en.wikipedia.org/wiki/Content_delivery_network)。在过去，主干是网络上唯一的东西。但是，一旦人们意识到将所有数据托管在一台服务器上不是一个好主意，CDNs 就发展起来了，并允许内容提供商将数据缓存在不同的物理位置。你在加州打一个 CDN，我在纽约打一个。这样事情进展得更快。

cdn 也成为了巨大的流量来源，但是他们没有太多的网络资源，所以他们试图以“服务”而不是“保单”的形式支付更少的费用来提供流量。现在在一个完美的世界里，我的比特和网飞的比特一样有价值。在很大程度上，这是真的。但是，当康卡斯特将第 3 级视为 CDN 时，情况发生了变化。康卡斯特是这样说的:

> 康卡斯特与**三级内容交付网络(CDN)竞争对手**在向我们的客户交付相同类型的流量方面已经建立了长期且双方都能接受的商业安排。对于相同的流量，康卡斯特向第 3 级提供的条款与向第 3 级的 CDN 竞争对手提供的条款相同。但是，第 3 级试图通过声称有权得到不同的待遇，并试图迫使康卡斯特提供第 3 级无限且高度不平衡的流量，并将所有成本转嫁给康卡斯特及其客户，从而获得相对于其 CDN 竞争对手的**不公平的商业优势。**
> 
> 为了量化这一点，第 3 级想要的是向康卡斯特施加压力，迫使其接受第 3 级免费提供给康卡斯特网络的流量增加两倍以上。换句话说，第 3 级希望与其他 cdn 竞争，但要将所有业务成本转嫁给康卡斯特和康卡斯特的客户，而不是第 3 级及其客户。
> 
> 3 级的地位简直是口是心非。当另一个网络提供商试图通过这种方式将流量传递到第 3 级时，第 3 级说这不是在互联网世界中无结算对等的工作方式。3 级表示，当交通严重失衡时，它将坚持商业协商解决方案。

第 3 级是这样说的:

> “2010 年 11 月 19 日，康卡斯特通知第三级，它将首次从第三级收取经常性费用，将互联网在线电影和其他内容传送给请求此类内容的康卡斯特客户。通过采取这一行动，康卡斯特实际上是在其宽带互联网接入网络的边界设立了一个收费亭，使其能够单方面决定对与自己的有线电视和 Xfinity 提供的内容竞争的内容收取多少费用。康卡斯特的这一行为威胁到了开放的互联网，明显滥用了康卡斯特作为美国最大的有线电视供应商在宽带接入市场的主导控制权。
> 
> “11 月 22 日，康卡斯特告知其付款要求是‘要么接受，要么放弃’，三级在抗议下同意了这些条款，以确保客户不会受到任何干扰。
> 
> “三级运营着几个宽带骨干网络中的一个，这些网络是互联网的一部分，独立的在线内容提供商利用这些网络向消费者传输电影、体育、游戏和其他娱乐内容。当康卡斯特的客户需要这类内容时，例如在线电影或游戏，第 3 级会将内容传输给康卡斯特，以便交付给消费者。

所以康卡斯特就像“他们是 CDN！”而第三级就像“他们在威胁我们！他们是反竞争的！”他们都是对的。

平心而论，三级是 CDN。然而，它也是世界上最大的脊梁。用一系列管道的比喻来说，这类似于一个下水道运营商提供特殊的厕所，可以非常快速地通过系统为那些有那种…需求的人吹风。下水道是主干，而超级厕所是 cdn。那么，界限在哪里？无论如何，3 级应该为同样的流量支付两次费用吗？毕竟，像 Akamai 这样的 CDN 已经支付了康卡斯特的 CDN 费用。康卡斯特能阻止乡亲们获得那些特殊的超级厕所的好处吗，尤其是如果他们有自己的超级厕所出售的话？

现在，如果你阅读康卡斯特的一面，他们说“3 级是 CDN。他们希望提供流行的、大量的数据。他们需要签订新的合同。”而第三级表示没有任何变化。康卡斯特还表示，第三级的内容正在堵塞它的管道。毕竟电影比邮件大吧？

不对。康卡斯特真正在做的是挟持一定数量的比特作为人质。第三级确实像一个主干，而且是一个极其重要的主干。你在网上做的任何事情都可能在某个时候触及第三级。因此，强迫 Level 3 支付 CDN 通过 Comcast 的网络吹内容所做的事情是非竞争性的，这是[网络中立](https://web.archive.org/web/20221007011300/http://en.wikipedia.org/wiki/Network_neutrality)希望防止的问题之一。事实上，考虑到互联网连接对普通用户的价值，康卡斯特可以帮自己一个忙，花多一点的钱为其当前用户提供更快、更好的服务，而不是动摇第三级(然后可能会通过告诉我们可以提供每月 50 美元的“黄金++网飞流媒体服务”来动摇我们)。就目前的情况来看，有线电视和 DSL 服务速度极慢，表现不佳。显然，康卡斯特需要先把自己的事情处理好，然后再哭诉自己是受害者。

这是最糟糕的一种内线棒球，因为球员们首先没有引起太多的同情，而且还有另一种叫做网络中立的游戏正在进行，它也令人愉快地令人不快。一点是一点是一点，NN 人群说，而互联网服务提供商认为自己是愤愤不平的夏尔巴人，被迫携带富人的重型设备和穷人的轻型设备。然而，从理论上讲，每个人都应该为同样的服务支付同样的费用。在实践中，像这样的案例将有助于决定谁为谁支付什么，正如我们都知道的那样，我们最终将付出代价。
