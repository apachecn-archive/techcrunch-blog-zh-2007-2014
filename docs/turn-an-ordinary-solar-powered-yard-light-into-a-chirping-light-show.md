# 将一盏普通的太阳能庭院灯变成一场唧唧喳喳的灯光秀

[![Solar Powered Yard Light](img/f4228cb9835669235b7985042ccef3bc.png "yardlamp")](https://web.archive.org/web/20230122153231/https://techcrunch.com/wp-content/uploads/2008/07/yardlamp.jpg)

如果你非常擅长使用烙铁， [Make 杂志](https://web.archive.org/web/20230122153231/http://blog.makezine.com/archive/2008/07/weekend_project_electroni_1.html?CMP=OTC-0D6B48984890)有一个非常酷的项目，可以把普通的太阳能庭院灯变成一个唧唧喳喳的灯具。

在固定装置完全吸收了阳光之后，夜晚的来临释放出一阵唧唧喳喳的声音。实际的灯光也会随着电子板球一起闪烁，产生一种令人困惑的效果。住在城市里，我对蟋蟀没有太多的经验，但上次我检查过它们没有眨那么多眼。

也许蟋蟀本来就是要发光的，只是在它们的进化过程中出现了可怕的错误。萤火虫终究是会眨眼的。如果蟋蟀也这样做，它们会更酷。在某种基础上，创造一个闪烁唧唧的院子灯大军的想法确实吸引了我。这个项目完全是为了娱乐，如果它让邻居感到困惑，那就更好了。
