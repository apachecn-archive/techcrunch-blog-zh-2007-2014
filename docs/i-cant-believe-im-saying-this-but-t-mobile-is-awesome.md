# 我不敢相信我会这么说，但 T-Mobile 太棒了

我在我寒冷的家乡加拿大度过了最后一个星期，在这里，我的眼睛失去了知觉，我看到了光明，我有一个信息给你们所有生活在美国的人，一个极其重要的信息，用火镌刻在互联网的圣石上。这条信息是:*我的天啊 T-Mobile 太棒了。如果你要出国旅行，你现在应该换成 T-Mobile。*

是的，我知道我听起来像个被收买的骗子。我对此感到尴尬和尴尬。不过，我认为[我的记录](https://web.archive.org/web/20221209065019/https://beta.techcrunch.com/author/jon-evans/)不言自明:说到科技公司，我通常是个古怪、消极的家伙。但这不一样。这太棒了。

见鬼，就在六个月前我写了一篇名为“[美国的航空公司太可怕了”的帖子。大概是你的错](https://web.archive.org/web/20221209065019/https://beta.techcrunch.com/2013/05/11/americas-carriers-are-terrible-its-probably-your-fault/)。”但是 10 月份 T-Mobile 推出了免费无限国际全球漫游服务。

基本上，每月 T-Mobile 计划的国际旅行者不再需要支付高额的漫游费，也不再需要为获得和激活目的地当地的新 SIM 卡而烦恼。相反，你只需打开数据漫游，让你的手机找到你的供应商。令我相当惊讶的是，这就是我所需要做的；没有 APN 黑客，没有移动网络欺诈，没有服务电话；它只是工作。我甚至可以*系绳*。

当然，它并不完美。尽管价格非常合理，但你仍然要为通话和短信付费，而且你的免费数据只有 2G/128kbps。不用说，你可以从 T-Mobile 购买 4G 漫游，但我发现没有必要这样做；至少在多伦多，2G 已经足够好，可以在许多区域和咖啡馆之间查看电子邮件和 Twitter，现在你可以在那里拿起一杯 wi-fi 而且它比其他选择好得多，任何比较都是一种侮辱。

我不习惯听起来像这样的广告。老实说，这有点让我起鸡皮疙瘩。但这可能很重要，在极少数情况下，一家公司——一家美国航空公司——实际上做了正确的事情，我们对他们赞不绝口，并以他们的名义歌颂他们。

哦，我不幻想 T-Mobile 的董事会里全是天使。他们只是被驱使去做正确的事情，因为他们是恶性市场中遥遥领先的第四名竞争者；现在软银/Sprint 正准备收购他们，一切都可能改变。如果他们在合并结束后的第二天就取消这项计划，我一点也不会感到惊讶，理由是这实际上改善了他们用户的生活，而不是在每一个可能的借口下给他们带来极大的不便，然后要求他们减轻这些不便——我们不能有*那样的*，不是吗？这开了一个不好的先例。

我有点期待他们对顾客的态度会很快恢复到中世纪领主对农奴的态度。但也许，在此期间，我们这些旅行者可以用脚投票来传达一个信息。放弃你目前的运营商，购买一部解锁的手机，并涌向 T-Mobile 和它(目前)几乎难以理解的用户友好计划。这个黄金时代大概不会持续太久；但是尽你所能充分利用它。
