# 25 岁生日快乐，光盘

我正准备从明尼阿波利斯搬到波斯顿，在清理我的房子时，我发现了一个旧 CD 的宝库，我把它和我的 3DO 和世嘉土星系统一起塞在了楼梯下，有个白痴把啤酒洒了一地。我抓起*金属乐队的*同名专辑(碰巧是我买的第一张 CD ),扔进我家里剩下的最后一台 CD 播放器里，想起了为什么 CD 刚出来时人们那么喜欢它们。

听了这么久的压缩 MP3，我的耳朵已经完全忘记了音乐有多好听。虽然我不确定我是否愿意很快带着随身听到处走，但你可以打赌我会开始寻找更高比特率的 MP3。你可以问任何一个铁杆音乐迷在哪里可以得到最好的音乐，大多数人会告诉你黑胶唱片在哪里，但光盘通过提供轻松地从一首曲目跳到另一首曲目的能力，使好的音乐更容易获得，并把整个东西缩小到可管理的大小。

你还记得你的第一台 CD 刻录机吗？我花了 400 美元买了一个 4x HP 驱动器，毁了我试图刻录的前三张光盘，因为我在写入 CD 时稍微撞了一下我的 P166 索尼 Vaio(带 MMX 扩展)。美好时光。

总之，生日快乐 CD。抱歉，我们好久没一起出去玩了。
