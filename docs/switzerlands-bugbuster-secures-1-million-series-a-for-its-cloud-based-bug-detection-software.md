# 瑞士 BugBuster 为其基于云的 Bug 检测软件获得了 100 万美元的 A 轮融资

*你说种子期，我们说 A 轮。* 
[BugBuster](https://web.archive.org/web/20230314175213/http://bugbuster.com/) ，一家从瑞士联邦理工学院(EPFL)操作系统实验室分离出来的初创公司，在被称为 A 轮的融资中筹集了超过 100 万美元——这是欧洲一些地区可用资金的状况。自 2011 年成立以来，该公司已经筹集了 100 万美元。

这项投资来自天使财团(BAS 和 Go Beyond)，以及之前的支持者 Polytech Ventures 和 Hasler Foundation，BugBuster 将在本月完成测试后，将其基于云的自动漏洞检测软件推向市场，并用于进一步开发。

这家总部位于瑞士洛桑的初创公司与 Selenium 和 WebDriver 等公司竞争，为开发人员提供了一个基于云的解决方案，用于自动测试驱动他们 Web 应用程序的前端代码——目前支持 HTML5 和 Javascript。

因为它是基于云的，在最基本的情况下，你输入你的 Web 应用程序的 URL，BugBuster 会自动探索你的应用程序并执行其代码，找到并报告任何错误，并生成截图和精确的调试信息。

此外，SaaS 还提供了一个 API，这样您就可以指定应用程序的业务规则并指导 BugBuster，让您控制在哪里以及如何测试您的应用程序。这包括跟踪“目标”的能力，以确保您的应用程序做它应该做的事情，以及“观察者”来监控您的应用程序的状态并相应地做出反应。

有人告诉我，这项服务从六月份开始就已经进入测试阶段，目前有大约 400 家公司在测试它。在此期间，BugBuster 称在所有测试的应用程序中有 89%被检测到了 bug。现在，如果有人能编写一个应用程序来自动修复它们就好了。
