# 让你的脸书朋友偷偷进入 Google+的唯一后门就是雅虎

![](img/ef23405fe762dc0a565fb389fb43a3f2.png)

马克·扎克伯格可能是 Google+上最受关注的用户，但是祝你在 Google 的新社交服务上找到你的脸书朋友好运。脸书让任何人都很难将朋友的联系信息导入 Google+。没有像雅虎或 Hotmail 那样的直接联系人导入功能，脸书正在取缔让你的脸书朋友轻松进入 Google+的第三方服务。

上周末，脸书[屏蔽了谷歌 Chrome 浏览器的扩展](https://web.archive.org/web/20230203084303/http://www.zdnet.com/blog/facebook/facebook-blocks-google-chrome-extension-for-exporting-friends/1935)称为[脸书友好出口](https://web.archive.org/web/20230203084303/https://chrome.google.com/webstore/detail/ficlccidpkaiepnnboobcmafnnfoomga)。事实上，[给我我的数据](https://web.archive.org/web/20230203084303/http://givememydata.com/)的创建者欧文·蒙迪说，脸书改变了 OAuth 2.0 API，以至于它“毫无预警地突然从查询中删除了电子邮件地址”。其他数据仍然可以导出，只是不能导出你朋友的电子邮件地址。

仍然可能有一个后门打开，让你的脸书朋友偷偷进入 Google+。这就是雅虎。你可以把你脸书朋友的联系信息，包括电子邮件，导入雅虎。(首先，你链接你的脸书和雅虎账户，然后你从雅虎邮箱[导入你的联系人](https://web.archive.org/web/20230203084303/http://lifehacker.com/5817003/import-facebook-friends-to-google%252B-by-going-through-yahoo-first)。

我说“可能”是因为我刚刚试过，Google+中的雅虎导入器甚至在 Chrome 中对我不起作用。但在 Firefox(讽刺警报)中似乎确实有效。不过，我不知道它是否真的导入了我的脸书朋友，他们也在 Google+上，因为大多数人也在我的 Gmail 联系人中复制。其他脸书的朋友没有出现在 Google+中(尽管他们的电子邮件地址可以很好地导入雅虎)。

虽然我们以前看到过[脸书玩这些把戏](https://web.archive.org/web/20230203084303/https://techcrunch.com/2010/05/07/facebook%E2%80%99s-disconnect-open-doors-closed-exits/)，因为他们不想帮助启动竞争服务，但我真的不认为这有什么大不了的。在这种情况下，光是你的电子邮件联系人(Gmail、Yahoo Mail 或 Hotmail)就足以让任何人上 Google+。你真正的朋友在脸书和 Gmail 中，更不用说你其他更偶然的联系人，你可以开始把他们放在不同的圈子里。

虽然从数据可移植性的角度来看，脸书的恶作剧并不好，但总的来说，我认为这对 Google+来说实际上是一件好事。如果导入你在脸书的朋友太容易了，那么 Google+就会变成脸书的复制品，复制你在脸书的经历。通过迫使你从头开始创建一个新的社交图，Google+有更大的机会变得与众不同。
