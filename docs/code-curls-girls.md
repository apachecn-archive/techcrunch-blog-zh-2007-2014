# 是时候有人把脸书 TOS 翻译成兄弟用语 TechCrunch 了

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/16/code-curls-girls/>

感谢 Miles Lothe 通过阅读整个脸书 TOS 和[将其翻译成兄弟语言](https://web.archive.org/web/20221021211755/http://slacktory.com/2011/08/entire-facebook-terms-of-service-in-bro-speak/)，将整个[编程](https://web.archive.org/web/20221021211755/https://www.facebook.com/getwiththebrogram)提升到一个新的水平。

这太神奇了，因为除了[背包](https://web.archive.org/web/20221021211755/http://www.manpacks.com/)(是的，它存在)之外，脸书是地球上最有 brogrammy 的初创公司——如果你想让我详细说明这一点，请发电子邮件或即时消息给我，因为这很晚了，但我完全可以证明这一点。对于那些不知道什么是语法的人，请看看这个[非常非常有用的](https://web.archive.org/web/20221021211755/http://www.quora.com/How-does-a-programmer-become-a-brogrammer) Quora thread 或者这个[语法工具包。](https://web.archive.org/web/20221021211755/http://www.brogrammers.org/)

有趣的事实:[尼克·施勒克](https://web.archive.org/web/20221021211755/http://www.quora.com/Nick-Schrock)，一名脸书工程师，在脸书经营着非常受欢迎的[编程页面](https://web.archive.org/web/20221021211755/https://www.facebook.com/getwiththebrogram)。还有，[这个存在](https://web.archive.org/web/20221021211755/http://brospeak.com/)。

**老弟讲脸书 TOS 翻译集锦 vs 实际脸书 TOS 下面…** 
“您的隐私对我们非常重要。我们设计了我们的[隐私政策](https://web.archive.org/web/20221021211755/http://www.facebook.com/policy.php)，对您如何使用脸书与他人分享以及我们如何收集和使用您的内容和信息做出重要披露。我们鼓励您阅读隐私政策，并使用它来帮助做出明智的决定。”

“我们非常关心你的隐私，所以我们写了这个。读吧，这样你就知道我们他妈的会怎么处理你发的垃圾，这样你就不会说‘脸书，我不知道！’当你的东西出现在我们的新闻稿里。”

当您使用“所有人”设置发布内容或信息时，这意味着您允许所有人(包括脸书以外的人)访问和使用该信息，并将其与您相关联(即您的姓名和个人资料图片)

*“有时候发布东西，可以和‘大家’分享。你要知道，我们指的是所有人。每一个。他妈的。一个。但如果他们问这是谁的屎，我们只告诉他们你的全名，并显示一张照片。就是这样。所以要把它拍好。”*

“我们始终感谢您对脸书的反馈或其他建议，但您明白我们可能会使用它们，但没有任何义务为此向您提供补偿(正如您没有义务提供它们一样)。”

*“哎，有时候，也许你会有想法！是的，我们喜欢你有想法的时候。如果你告诉我们你的想法，也许我们会说，‘嘿！好主意，孩子！我们一定会实现这一目标。我们，呃，我们不需要付钱给你，就像你不需要告诉我们如何改进我们的网站，混蛋。谢了。"*

“未经用户同意，您不得标记用户或向非用户发送电子邮件邀请。”

*“是的，我们已经让你邀请你的朋友变得超级简单，还可以给他们添加图片和标签。太简单了，你会以为我们想让你邀请他们。你甚至可能会被诱惑去做。但是不要，除非你得到他们的允许。不要给任何人发电子邮件邀请去脸书，直到你得到他们的允许。不，请便，我们不介意你发邮件给他们，请求允许你发邮件给他们。我们就在这里等着。”*

“如果你从用户那里收集信息，你将:获得他们的同意，表明是你(而不是脸书)收集他们的信息，并张贴隐私政策，解释你收集什么信息以及你将如何使用它。”

*“这里有很多关于很多人的信息。有用的东西，信息，对吧？你不能拥有它，除非你告诉每个人你将如何使用它，并确保他们同意。你以为你是谁，我们？”*

"你不会向别人歪曲你和脸书的关系."

*“不要告诉别人我们比实际情况更紧密。你只是一些应用程序开发人员；我们不想发现你试图通过告诉他们你认识马克或什么来给那个辣妹或那些投资者留下印象。你没有。”*

“您给予我们链接或构建您的应用程序的权利，以及在您的应用程序周围放置内容(包括广告)的权利。”

*“基本上你创造的东西都不是私有的。我们可以检查您的内容，挖掘您的数据，分析您的应用程序，以及我们想要的任何东西，无论出于任何原因。是的，甚至从你的大便中赚钱。问题？”*

"您理解我们可能并不总是将付费服务和通信视为付费服务和通信."

*“另一方面，我们也不用跟你说什么狗屁。有时你会看到一些看起来像广告的东西，但也许不是，你会想，‘那是广告吗，脸书？’？“我们会说，“……好问题。””*

"我们可以将您的广告及相关内容和信息用于营销或促销目的."

***“我们可以在我们的广告中使用你的广告，就像我们为销售广告制作广告时一样。我们称之为 ADCEPTION。”***

“我们努力让 FACEBOOK 保持运行、无漏洞和安全，但你使用它需要自担风险。我们按原样提供 FACEBOOK，不附带任何明示或暗示担保，包括但不限于适销性、特定用途适用性和不侵权的暗示担保。我们不保证 FACEBOOK 是安全的。”

“从法律上来说，我们被要求喊出这一部分，因为它真他妈的重要。这是我们告诉你的部分，我们尽最大努力不打破狗屎，但我们不能作出任何承诺或保证任何事情。我们甚至不保证使用 FACEBOOK 是安全的，所以如果你因为你在 FACEBOOK 上做的一些与我们无关的事情而被斧头砍死，我们会试图警告你，甚至会大喊大叫。”
