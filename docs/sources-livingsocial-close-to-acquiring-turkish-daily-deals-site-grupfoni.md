# 来源:LivingSocial 接近收购土耳其每日交易网站 Grupfoni

据当地科技博客 [Webrazzi](https://web.archive.org/web/20230205011608/http://www.webrazzi.com/2011/08/04/livingsocial-turkiye-grupfoni/) 最先报道，多个知情人士告诉我们， [LivingSocial](https://web.archive.org/web/20230205011608/http://www.crunchbase.com/company/livingsocial) 即将收购土耳其当地领先的电子商务公司 [Grupfoni](https://web.archive.org/web/20230205011608/http://www.grupfoni.com/) ，即将继续其国际扩张。我们听说，交易还没有完成，但是 LivingSocial 已经非常接近完成交易了。

该交易对 Grupfoni 的估值超过 5000 万美元，尽管我们在现阶段无法确定确切的收购价格，但 5000 万至 7000 万美元之间是合理的，因为 Grupfoni 的年营收率估计在 2500 万至 3000 万美元左右。

LivingSocial 的一名代表拒绝证实或否认收购谈判，因为该公司通常不对传言或猜测发表评论。格鲁佛尼也保持沉默。

但是 LivingSocial，像它的主要竞争对手 Groupon 一样，定期收购当地的每日交易网站，作为全球范围内折扣分配的基础。

最近，[公司收购了](https://web.archive.org/web/20230205011608/https://techcrunch.com/2011/06/27/livingsocial-expands-daily-deals-empire-buys-ensogo-gonabit-and-dealkeren/) [DealKeren](https://web.archive.org/web/20230205011608/http://www.dealkeren.com/) (在印度尼西亚运营)、其母公司 [Ensogo](https://web.archive.org/web/20230205011608/http://www.ensogo.com/en/) (在泰国和菲律宾提供日常交易)以及 [GoNabit](https://web.archive.org/web/20230205011608/http://www.gonabit.com/) (在迪拜、阿布扎比、黎巴嫩、约旦和科威特运营)。

他们还收购了韩国的票务巨头，一举增加了 600 名员工。

今年早些时候，这家团购巨头还收购了欧洲 Let's Bonus 的[多数股权，并购买了](https://web.archive.org/web/20230205011608/https://techcrunch.com/2011/01/13/livingsocial-buys-majority-stake-in-let%E2%80%99s-bonus-expands-internationally/) [SocialMedia](https://web.archive.org/web/20230205011608/https://techcrunch.com/2011/04/15/livingsocial-acquires-socialmedia-for-3-million/) 和 [InfoEther](https://web.archive.org/web/20230205011608/https://techcrunch.com/2011/03/14/livingsocial-gains-wealth-of-ruby-on-rails-expertise-with-infoether-acquisition/) 。

根据两份 SEC 文件，LivingSocial 最近发行了价值 2940 万美元的股票，涉及一项或多项收购。

我们一有更多的消息就会更新。
