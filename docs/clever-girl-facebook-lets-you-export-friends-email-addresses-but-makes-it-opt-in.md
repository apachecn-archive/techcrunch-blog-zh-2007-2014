# 聪明的女孩:脸书允许您导出朋友的电子邮件地址，但让它选择加入技术危机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/11/clever-girl-facebook-lets-you-export-friends-email-addresses-but-makes-it-opt-in/>

现在可以导出您的脸书社交图了。您的*真实的*社交网络，包括您朋友的电子邮件，这意味着您可以离开脸书，将您的社交网络导入竞争对手的网站。

但前提是你的朋友选择加入。他们不会。

脸书悄悄地在账户设置页面增加了一个新功能，让用户可以选择“允许朋友在*上传我的电子邮箱地址，下载你的信息*”。检查一下，下次您的朋友决定下载他们的脸书数据档案时，他们也会收到您的电子邮件地址，以便保持联系。

对脸书来说，这是一个巧妙的举措，将有助于削弱其将用户数据作为人质的观点，同时确保其牢牢掌控其巨大的有价值的社交网络。

![](img/7352bacf61b99bab46fce68126e681f5.png)

这里涉及了很多历史，所以让我们回顾一下。

[下载您的信息](https://web.archive.org/web/20230203100230/https://techcrunch.com/2010/10/06/facebook-now-allows-you-to-download-your-information/)是去年秋天推出的一个功能，它听起来就是这个样子:点击它，几个小时之内你就会收到你的照片、状态更新和好友列表。到目前为止，这个朋友列表一直是字面上的——它是一个你朋友的名单，没有附带的电子邮件地址或联系方式。这意味着它实际上对其他服务毫无用处，这些服务最喜欢的莫过于导入您的社交图。比如说，谷歌。

去年秋天，谷歌和脸书就联系数据进行了一场公开的“T7”大战。谷歌先发制人，因为脸书不让联系数据流向相反的方向，所以阻止了脸书进口用户的谷歌地址簿。这意味着，当谷歌推出最终成为谷歌+的产品时，重建社交图将会困难得多。

脸书的观点是，你的朋友不一定有权将你的电子邮件地址导入另一项服务。这实际上没有意义，因为脸书用户已经可以将他们所有朋友的电子邮件地址导出到雅虎邮箱(而且只能导出到雅虎邮箱)。话说回来，谷歌关于用户*应该*能够导出他们朋友的电子邮件地址的论点并不真正有意义，因为它之前在[阻止脸书从 Orkut 导入数据的能力时，也制造了一个几乎相同的关于数据所有权的案例](https://web.archive.org/web/20230203100230/https://techcrunch.com/2009/10/26/orkut-slows-hemorraging-to-facebook-by-making-friend-export-tool-nearly-useless/)。

无论如何，抛开虚伪不谈，这两家公司的潜在动机是显而易见的。谷歌想要脸书的社交图谱(部分原因是脸书已经从谷歌联系人中吸走了太多数据)，而脸书不想给他们。

让我们回到今天的发布会。这个小复选框完全符合脸书的观点，即用户应该拥有对自己数据的控制权。当人们(或谷歌)抱怨他们不能导出朋友的电子邮件地址时，脸书可以说这不是因为它在挟持他们——而是用户根本没有兴趣分享这些数据。

但是，再一次，脸书非常清楚大多数人不会去挖掘设置来检查那个盒子。他们中的许多人可能不知道“下载你的信息”选项的存在，他们当然也没有太多考虑他们的朋友是否应该能够下载他们的电子邮件地址作为存档的一部分。该框未被选中，并且将保持不变。

同样值得注意的是:脸书并不羞于自动选择用户默认的新功能([即时个性化](https://web.archive.org/web/20230203100230/https://techcrunch.com/2010/04/27/senators-call-out-facebook-on-instant-personalization-other-privacy-issues/))，有人知道吗？)，但它在这里采取了“安全”的路线，让它选择加入。我肯定脸书会说他们已经从过去的错误中吸取了教训。或者也许它根本没有动力去帮助人们输出他们的社交图谱。

我们已经联系了脸书发表声明。
