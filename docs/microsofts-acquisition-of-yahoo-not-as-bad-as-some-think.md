# 微软收购雅虎:没有一些人想象的那么糟糕

【YouTube http://www.youtube.com/watch?v=x9BosxTWZE4&rel=1]

微软是邪恶的迷因本周仍然活跃，因为许多人都在消化微软对雅虎 446 亿美元的收购要约。有 [Flickr 用户抗议](https://web.archive.org/web/20230218074410/https://techcrunch.com/2008/02/04/flickrs-version-of-a-mass-protest/)，谈论雅虎与谷歌联手阻止微软的出价，微软到处都是坏情绪，甚至来自谷歌本身的。虽然微软收购雅虎可能不会给许多人带来最终的大团圆结局，但它真的没有一些人想让你相信的那么糟糕。

**谷歌害怕了。**

谷歌在周末对收购的[反应的真实性令人惊讶。谷歌和微软从来就不是朋友，但谷歌站出来攻击收购只能意味着一件事:谷歌害怕了，这是一件非常好的事情。谷歌内部认为，微软/雅虎的合并将为其在搜索和文本广告领域的主导市场地位带来真正的竞争，直到最近，谷歌一直依靠这一地位打造其庞大的股价和市值。谷歌可以鼓吹开放获取和开放市场，但谷歌的开放理念只是用户从其众多网络资产中的一个访问它。](https://web.archive.org/web/20230218074410/https://techcrunch.com/2008/02/03/google-cries-wolf-on-microsoft-yahoo-deal-irony-comes-up-blank-in-google-search/)

**更多竞争**

反对这项收购的一个理由是，它削弱了竞争。虽然就主要参与者而言，微软/雅虎的合并实际上会增加谷歌占主导地位的领域的竞争，在一些国家，谷歌几乎是垄断供应商。规模是这里的关键:谷歌有，雅虎和微软需要更多。例如，合并后的微软/雅虎在文本广告方面的市场份额仍不到谷歌的一半，但在与谷歌竞争时，合并后的公司将能够提供在覆盖范围方面更具吸引力的产品，即:扩大的覆盖范围将对寻求最大化曝光的广告买家更具吸引力。

**更强壮的尾巴**

反对这笔交易的一个理由是，雅虎出局后，初创公司被收购的机会将会减少。这是真的，但只是短期的。总会有第三名的玩家和低于第三名的位置，随着微软/雅虎排在第二位，现在有空间让较小的玩家进入这个地方。我想到了 Ask，但是我们没有理由看不到新球员的进步。本质上，现在将少了一个竞争对手。想要打入顶级市场的小型企业的实力增强，很可能会导致资金流动和收购的增加。在减少上层的数字竞争的同时，它也创造了下层的更多机会。这自然是基于这样一种假设，即在搜索等领域不会出现双头垄断，但这是不太可能的:一直以来都有较小的参与者，一直是，也将永远是，成为下一个谷歌的梦想将继续存在。

**雅虎靠**生存

实际上，微软将极大地推动雅虎品牌及其众多服务。相比之下，其他选择几乎是暗淡的:对冲基金或类似的投资会将雅虎剥离至核心部分(希望是盈利的)服务，并出售雅虎新闻等领域。谷歌作为合作伙伴(无论是在广告领域还是作为投资者这种不太可能的情况下)将使雅虎注定永远成为谷歌的二等玩家。雅虎最大的优势之一仍然是它的品牌，微软将不懈努力，将雅虎品牌打造回它作为互联网首页的光辉岁月。子品牌也是如此:例如，Flickr 可能会取代微软的照片分享套件。

**更好的服务**

有人在过去几天写道，打造最好的产品并期望人们使用它的日子已经过去了，这就是在谷歌占主导地位的时代，微软今天在线努力的问题。微软有一些很棒的互联网产品；从 Popfly 和 Silverlight，到稳固的广告平台(为 Digg 和脸书提供动力)和体面的搜索。合并后的微软/雅虎将提供两家公司的优势，理论上是一个更好的整体方案。有些人可能会认为达到这一阶段可能会很困难，但即使是[连线杂志也注意到](https://web.archive.org/web/20230218074410/http://news.wired.com/dynamic/stories/M/MICROSOFT_YAHOO_CLASH?SITE=WIRE&SECTION=HOME&TEMPLATE=DEFAULT&CTIME=2008-02-04-09-05-46)融合两种企业文化并不难。最终要与谷歌竞争，你必须有一个强大的全面的多方面的包，而微软/雅虎正好有。
 **结论**

这些都不是对谷歌的抨击。谷歌做了很多好事，正如谷歌喜欢争辩的那样，人们选择使用谷歌，没有人被迫使用他们的服务。然而，从长远来看，一家公司占主导地位的市场通常会出现停滞和/或低效的定价和服务。我们需要一个合并的微软/雅虎，以保持谷歌的警觉，保持谷歌的创新和提供最好的服务，保持谷歌的诚实。
