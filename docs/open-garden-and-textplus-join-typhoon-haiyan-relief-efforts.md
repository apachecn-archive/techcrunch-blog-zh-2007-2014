# 开放花园和 textPlus 加入台风海燕救援工作

许多技术社区的成员已经借出他们的技能来帮助菲律宾的超级台风海燕救援工作。其中包括 [Viber，它推出了一个名为 Viber Out](https://web.archive.org/web/20230129220016/https://techcrunch.com/2013/11/11/viber-opens-lines-of-communication-for-philippine-typhoon-victims/) 的新功能，以及与[合作的开发人员](https://web.archive.org/web/20230129220016/https://techcrunch.com/2013/11/11/developers-use-your-coding-skills-to-help-typhoon-haiyan-rescue-efforts-in-the-philippines/)。现在，点对点移动宽带网络[开放花园](https://web.archive.org/web/20230129220016/http://opengarden.net/)和短信应用 [textPlus](https://web.archive.org/web/20230129220016/http://www.textplus.com/) 也在提供服务，帮助菲律宾人从这场造成多达 1 万人死亡的毁灭性风暴中恢复过来。

Open Garden 响应菲律宾总统办公室向黑客发出的号召，推出了其旗舰应用来帮助幸存者和救援人员恢复手机连接。Open Garden 应用程序可以使用蓝牙从一部 Android 手机发送到另一部 Android 手机，使仍然可以访问互联网的移动用户的连接更加可靠，并允许没有连接的手机通过有连接的设备访问互联网。

Open Garden 的首席技术官和联合创始人 Stanislav Shalunov 告诉我，这家初创公司通过 [ICT](https://web.archive.org/web/20230129220016/http://plan-international.org/) 参与了救援工作，该公司在菲律宾设有现场办公室，提供第一手救援，并且正在与菲律宾政府合作。开放花园应用程序最重要的用途之一是改善救援人员的互联网连接。在一个拥有 300kb/s 卫星连接的外地办事处，Open Garden 通过让每个人同时查看电子邮件来帮助节省电池。

与此同时，免费短信应用 textPlus 正在向菲律宾捐赠免费语音电话。textPlus 用户可以通过 textPlus 应用程序免费拨打菲律宾的电话号码，该应用程序可在 [iOS](https://web.archive.org/web/20230129220016/https://itunes.apple.com/us/app/textplus-free-text-+-calls/id314487667?mt=8) 和 [Android](https://web.archive.org/web/20230129220016/https://play.google.com/store/apps/details?id=com.gogii.textplus) 上使用。textPlus 使用 Wi-Fi，因此移动网络覆盖区域的用户仍然可以使用该应用程序。更多信息，请参见 [textPlus 网站](https://web.archive.org/web/20230129220016/http://www.textplus.com/textplus-offers-free-calling-philippines-families-friends-affected-stay-connected/)。
