# 众议员古德拉特:移民辩论“绝对”会延续到秋天，可能是 2014 年

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/17/rep-goodlatte-immigration-debate-absolutely-extends-into-fall-possibly-2014/>

CrunchGov Essential 是一份关于科技对当今重大问题的影响力的扫描综述。在一篇专题文章下面，我们呈现了通过网络最佳内容讲述的最有思想、最令人愤慨、最鼓舞人心的故事。在这里注册早间新闻[，在](https://web.archive.org/web/20221007005838/http://link.beta.techcrunch.com/join/134/crunchgov-signup)[推特](https://web.archive.org/web/20221007005838/http://twitter.com/ferenstein)上关注 Greg Ferenstein。

众议院负责移民改革的国会议员向公众保证，法案可以通过，但可能需要(非常)长的时间。为了缓解人们对史上效率最低的国会将扼杀移民改革的担忧，众议院司法委员会主席鲍勃·古德拉特(Bob Goodlatte)在硅谷旋风式的科技和筹款之旅中与我坐在一起。

对于略读读者来说，有四个要点:

1)该法案可能需要长达一年的时间才能通过，并且肯定会延续到秋季。

2)众议院共和党人可能会推动让无证工人更难成为公民，但更容易在美国生活和工作

3)科技游说团体，比如马克·扎克伯格(Mark Zuckerberg)的 FWD.us，有助于教育国会议员，但他们无法影响最反对改革的地区的议员。

4)每个人都需要冷静；民主党人和共和党人可以在一些事情上达成一致。

**背景**

自 1986 年以来，移民改革已经失败了四次(1986 年、1996 年、2001 年和 2007 年)。古德拉特认为，过去的法案之所以失败，是因为行业团体试图过早地将有争议的法案强行灌输给公众。

古德拉特说:“2007 年，参议院通过了一项甚至没有通过委员会的法案，并将其提交给参议院，公众吓坏了，他们收到的关于该法案的电话和电子邮件比参议院历史上任何其他法案都多。”

为了避免引发同样的反改革愤怒雷区，众议院发誓要有条不紊地通过委员会听证会、修正案和辩论的“常规秩序”，最终将少数移民相关法案整合成一个单一的综合方案。

与此形成鲜明对比的是，参议院勉强通过了一项主要由“八人帮”有影响力的成员组成的法案。古德拉特说，他的同事中没有足够的人支持参议院的版本，所以众议院必须提出自己的版本。

然而，本月早些时候，迈克·艾伦的政治放弃了一份谴责报告，该报告预测众议院的做法将导致移民改革“缓慢死亡”，引发宿命论的洪流。Goodlatte 是来减轻这些恐惧的

**这要花很长时间**

“当然，它将进入秋季，这是毫无疑问的，”古德拉特说，粉碎了参议院民主党领袖哈里·里德早些时候的预测，即该法案将在夏末通过。

那么，需要多长时间？古德拉特对描述时间线的方式非常谨慎。我会全文贴出他的回应，让读者自己拿主意。我认为这意味着该法案可能需要长达一年的时间。

> “但这需要时间，把这件事做好比说我们有一个由总统制造的人为障碍更重要，他说这个障碍必须在 7 月前清除，或者其他人说它必须在今年年底前清除。或者在选举年做不到。为什么不能在选举年进行？导致这种情况的是双方的公众压力。”

从历史背景来看，[平价医疗法案](https://web.archive.org/web/20221007005838/http://en.wikipedia.org/wiki/Patient_Protection_and_Affordable_Care_Act)(又名奥巴马医改)比奥巴马总统预期的时间多花了一年才通过，所以这场内讧很有可能会将辩论延长到 2014 年大选年。

**回家容易，投票难**

“我们不会在众议院通过一项容易获得公民身份的法案，”他说。共和党中更加排外的边缘人士对给予居住在美国的 1100 万无证工人公民身份感到非常害怕——不仅仅是因为他们倾向于投票给民主党。

为了安抚共和党的这一部分人，古德拉特和众议院领导层倾向于一种解决方案，这种解决方案将允许更容易的居住路径，但更难的公民身份路径。“他们可以在这里工作，他们可以在这里拥有自己的企业，他们可以往返于自己的国家或世界上的任何地方，他们可以缴纳税款，然后他们可以拥有现在没有的机会。”

但是，至少，他认为无证工人将不得不与其他耐心等待数年(有时超过 10 年)获得公民身份的居民站在一起。

**不要高估科技游说集团**

许多新技术游说团体正花费巨资向国会施压，要求通过高技能移民改革。我问古德拉特，马克·扎克伯格的 FWD.us 或 Engine advocation 等组织是否在共和党会议上被提及，或者它们是否对他更抵制改革的同事产生了影响。他的回答是直截了当的“不”

“不，我认为他们认识到这不是什么新鲜事，”他说，“但是，这肯定是他们必须面对的现实。”

事实上，正如我以前所写的，[移民改革的主要障碍](https://web.archive.org/web/20221007005838/https://beta.techcrunch.com/2013/07/12/why-immigration-reform-may-fail-in-4-graphs/)是强烈反对参议院移民改革计划的白人占多数、选区划分不公的地区。尽管一些科技游说团体拥有大量现金，但许多美国中产阶级国会议员并不太在乎硅谷或纽约以外的游说团体怎么想。

**别大惊小怪了**

古德拉特确信，共和党人广泛支持移民改革，国会可以制定一项两党法案。例如，去年，众议院通过了 STEM Jobs 法案，取消了代表名额不足的国家的 55，000 个签证的多样性抽签计划，并将它们重新分配给拥有美国大学研究生学位的高技能移民。

尽管民主党人激烈地争取多样性彩票项目，但参议院已经通过了一个没有该项目的移民改革版本。

换句话说，国会已经并将继续寻找能够在众议院通过的妥协方案。

最后一句话，众议院司法委员会主席是美国在技术政策方面最有权力的职位之一，因为大多数技术法案都必须通过司法委员会。古德拉特的前任，众议员拉马尔·史密斯( [CrunchGov Grade](https://web.archive.org/web/20221007005838/https://beta.techcrunch.com/crunchgov/leaderboard/) : F)是科技行业的头号敌人，这不仅仅是因为他是《停止网络盗版法案》的拥护者。

尽管古德拉特任期尚早，但他已将自己树立为科技行业和党内其他人之间富有成效的桥梁。除了移民改革，他还站出来反对专利流氓，并主张简化有争议的网络销售税(T2)。他既不是啦啦队长，也不是行业的敌人，这可能会使他成为一个有效的领导者。
