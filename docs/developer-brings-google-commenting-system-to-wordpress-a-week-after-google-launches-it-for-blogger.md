# 在 Google 为 Blogger 发布 Google+评论系统一周后，开发者将 Google+评论系统带到 WordPress

本周早些时候，谷歌为 Blogger 推出了新的 Google+评论系统，但还不清楚该公司计划何时将其发布到其他网站。然而现在，无畏的开发者 Brandon Holtsclaw 已经设法将 Google+评论系统[引入任何 WordPress 博客](https://web.archive.org/web/20221007002231/http://www.cloudhero.net/gplus-comments)。虽然谷歌新评论系统的代码还没有针对 WordPress 或其他网站进行优化，但它实际上是相当可移植的，而且似乎工作起来没有问题。

看看 Holtsclaw 的代码，看起来将 Google+评论打包成 WordPress 插件实际上非常简单。谷歌没有采取任何措施来阻止用户在其他网站上重复使用它的代码，所以它可能不会采取任何积极措施来阻止网站所有者在非博客网站上使用它们。

如果你想在你的博客上尝试 Google+评论，你可以在这里下载插件[(并且它很快也会出现在](https://web.archive.org/web/20221007002231/http://www.cloudhero.net/gplus-comments) [WordPress 插件目录](https://web.archive.org/web/20221007002231/http://wordpress.org/extend/plugins/gplus-comments/))。不过，值得注意的是，这些评论与 Disqus 或 Livefyre 的类似系统不同，没有与 WordPress 集成，所以你无法从常规的 WordPress 仪表盘管理它们。

上周推出新的评论系统时，该公司表示，这将使博客用户更容易与他们的评论者互动，“看到直接访问者的活动，以及在 Google+上谈论(他们的)内容的人的活动。”

因为 WordPress Google+评论小部件使用了与 Google 的 blogger 实现完全相同的代码，所以所有的功能都是一样的。默认情况下，评论按“热门评论”排序，但用户也可以切换到“最新的优先”还有一个选项是只查看你的 Google+圈子里的人的评论。
