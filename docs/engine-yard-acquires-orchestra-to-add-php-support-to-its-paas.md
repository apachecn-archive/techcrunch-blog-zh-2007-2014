# Engine Yard 收购 Orchestra，为其 PaaS 添加 PHP 支持

[Engine Yard](https://web.archive.org/web/20230203145551/http://www.crunchbase.com/company/engineyard) ，一家平台即服务(PaaS)解决方案的提供商，今天早上[宣布](https://web.archive.org/web/20230203145551/http://www.engineyard.com/blog/2011/engine-yard-and-orchestra-join-forces/)它已经[收购了](https://web.archive.org/web/20230203145551/http://www.engineyard.com/orchestra) [Orchestra](https://web.archive.org/web/20230203145551/http://orchestra.io/) ，一家用于部署、扩展和管理 PHP 应用程序的平台开发商。

由 Benchmark Capital、New Enterprise Associates 和 Amazon.com 等投资者支持的 Engine Yard 选择不披露收购的财务条款。

这家总部位于旧金山的公司以其针对 Ruby on Rails 应用的 PaaS 而闻名，但此次收购将允许该公司[增加对 PHP 应用的支持](https://web.archive.org/web/20230203145551/http://blog.orchestra.io/post/orchestra-engine-yard)。

Engine Yard 在一份声明中表示，将继续开发 Orchestra PHP 平台，并继续投资于 Ruby on Rails 和 PHP 开源社区和项目。没有计划停止支持或开发 Orchestra 的任何产品。

Orchestra 由爱尔兰公司 echolibre 制作，该公司雇佣了 PHP 社区的热心贡献者和开源 API 框架 FRAPI 的创建者。

作为收购的结果，整个管弦乐队将加入 Engine Yard。

迄今为止，Engine Yard 已经筹集了 3750 万美元的资金。该公司与另一家风投支持的初创公司 [Heroku](https://web.archive.org/web/20230203145551/http://www.crunchbase.com/company/heroku) 竞争，后者于去年年底被 [Salesforce](https://web.archive.org/web/20230203145551/http://www.crunchbase.com/company/salesforce) 以[2 . 12 亿美元现金](https://web.archive.org/web/20230203145551/https://techcrunch.com/2010/12/08/breaking-salesforce-buys-heroku-for-212-million-in-cash/)收购。
