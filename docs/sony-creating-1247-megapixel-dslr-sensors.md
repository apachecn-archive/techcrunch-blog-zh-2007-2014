# 索尼制造 1247 万像素 DSLR 传感器

![CMOS](img/9f78023c0a3391648e7c46ca4e366f5e.png)

我们的星系是一个很大的地方。科学家说，如果太阳有一页纸上的一段时期那么大，那么银河系就有美国大陆那么大。也就是说，如果你带着一台配备索尼即将推出的 1247 万像素传感器的 DSLR 相机去我们银河系的边缘，你就可以拍下地球的照片，细节会非常好，以至于你可以看到你的房子。

虽然这可能有点夸张，但新的 1247 万像素 CMOS 传感器“具有 10.39 帧/秒(12 位)的高信号转换速度(在全像素扫描模式下)”，并将图像传递到相机的处理电路，同时将噪声和/或伪像降至最低。

如果有人想知道，我有一台 210 万像素的卡西欧 Exilim，可以打印出略高于平均水平的 4×6 照片。这个新传感器可能不适合我。有专业摄影师愿意发表评论吗？
