# 麻省理工学院科学家声称已经创造出实用的“人造叶子”

绿色能源支持者热切期待的一件事是“人造叶子”:一种真正小型、便携、模块化的光合发电厂。将一百个或一千个这样的单元贴在墙上、屋顶上、树上或其他什么地方，让它们整天在一个简单的燃料电池中储存能量，这将是一个很好的方法，可以使电力分配在没有到处都有电线的国家变得不那么复杂。

麻省理工学院的丹尼尔·诺切拉最近显然取得了重大进展，制造出一种人造叶子，不仅使用廉价材料，而且在光合作用过程中的效率是真叶子的十倍。

该技术通过利用阳光将水分解为氢气和氧气来模拟光合作用；就在不久前，诺切拉成功地创建了一个系统，使用“钴和磷酸盐的专有溶液”，能够以比太阳能电池更高的效率转换阳光。它也可以使用脏水，不像一些现有的电解/燃料电池解决方案。

他的研究刚刚被提交到《科学》杂志，还没有被审查或复制，但这似乎对印度工业巨头塔塔来说并不重要，他们已经决定与诺切拉合作——大概是为了在印度制造和销售该系统。这里阳光充足，人口众多，是个天然的好地方。
