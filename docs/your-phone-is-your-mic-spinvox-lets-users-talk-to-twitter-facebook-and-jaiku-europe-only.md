# 你的手机就是你的麦克风:SpinVox 允许用户与 Twitter、脸书和 Jaiku(仅限欧洲)通话

[![spinvox-logo.png](img/443198e5fc41a76fe0d5a5a373c343f9.png)](https://web.archive.org/web/20221006073419/http://www.spinvox.com/) 我确信，我们将会看到使用手机作为麦克风的语音应用的大幅增长。今天，在西班牙举行的世界移动通信大会上，总部位于伦敦的初创公司 SpinVox [宣布了其语音转文本服务的一项新功能](https://web.archive.org/web/20221006073419/http://www.spinvox.com/social-networks-facebook-twitter-and-jaiku-all-gain-voice.html)，该功能允许用户拨打 Twitters 或向脸书或 Jaiku 发送信息。

打电话比发短信容易吗？看你有什么样的手机，你是什么样的人。但是[口述录音机](https://web.archive.org/web/20221006073419/http://en.wikipedia.org/wiki/Dictaphone)现在是社交的。这至少会吸引那些喜欢用迷你录音机录笔记的人。他们可以与世界分享这些想法，计算机将转录它们。

据我快速浏览 SpinVox 的网站可以看到，它的服务只在欧洲可用，而且不是免费的。但这是解决移动用户界面问题的一个显而易见的方法。对手机来说，最好的输入技术就是对着它说话。也许竞争对手 SimulScribe、Jott 或 Yap 会在美国站出来。

[![myvox-small.png](img/78fb79aac2a4cacd995d0fe8a8eccb7f.png)](https://web.archive.org/web/20221006073419/http://www2.myvox.com/developers/default.asp) 等等，已经有办法把手机变成任何网络应用的麦克风了。它被称为 VoodoVox 的 MyVox API 。它是广告支持的，所以是免费的。而基于 API 构建的最佳应用可以[赢得 25000 美元](https://web.archive.org/web/20221006073419/http://www2.myvox.com/developers/getstarted_d.asp?ID=36)。
