# 如果您想在 iTunes 中查找图书，请在 App Store 中查找

![](img/a21af521049e372189160594c60b1812.png)

为你的 iPad 购买一本电子书是一种非常奇怪的体验。如果你打开 iTunes，你可以找到音乐、电影、应用程序，甚至有声书，但没有数字图书的类别。您需要先下载 iBooks 应用程序，然后在该应用程序中购买图书。因此，这就像是市场中的市场，同时也是读者。Kindle 应用程序也是如此。令人困惑。

但是如果你进入应用程序商店，你可以找到一整类的 iPad 应用程序，都是书籍。其中许多是互动的，往往是像 *[绿蛋火腿](https://web.archive.org/web/20221005164142/http://itunes.apple.com/us/app/green-eggs-and-ham-dr-seuss/id380751745?mt=8)*(3.99 美元)或 *[蜘蛛小姐茶会](https://web.archive.org/web/20221005164142/http://itunes.apple.com/us/app/miss-spiders-tea-party-for/id364736580?mt=8)*(7.99 美元)这样的儿童读物。越来越多的书籍会因为各种原因出现在 App Store 中。最大的一个原因是因为[应用更有趣。](https://web.archive.org/web/20221005164142/https://beta.techcrunch.com/2010/04/11/dear-authors-your-next-book-should-be-an-app-not-an-ibook/)

如果图书出版商想在 iBooks 之外增加任何功能，包括添加信息应用程序、外部网络链接或在 Twitter 和脸书上与朋友分享摘录，他们最好将图书作为应用程序发布。像 Rethink Books 这样的初创公司(我昨天[报道过](https://web.archive.org/web/20221005164142/https://beta.techcrunch.com/2010/11/11/rethink-books-social/))正在为出版商开发软件平台来实现这一目标——将书籍变成社交应用。

目前，图书在 iTunes 中仍处于某种隐藏状态。在主要的媒体类型中，除了有声读物，没有明确的书籍类别。你必须深入 iBooks 应用程序或在 App Store 中找到图书应用程序。但如果苹果真的想让 iPad 成为一款阅读设备，那么在一个地方找到所有可以在上面阅读的书籍应该会更容易一些。
