# Questli 推出了 iOS 应用程序，允许用户进行虚拟和现实世界的任务

今年在旧金山举行的 TechCrunch Disrupt 上，[观众选择获奖者之一](https://web.archive.org/web/20221007025922/https://beta.techcrunch.com/2011/09/12/quest-li/)quest Li 今天首次亮相其 IOS 应用程序。你可以在这里下载应用[，你可以使用代码“TC2011”来注册这个应用。正如我们在最初的评论中所写的，Questli 就是玩别人的任务。](https://web.archive.org/web/20221007025922/http://itunes.apple.com/us/app/questli/id464156578?ls=1&mt=8)

用户可以创建网络和虚拟的任务供其他人参与。所以你可以把一张 5 美元的钞票藏在 JFK，然后创建一系列复杂的任务，玩家必须完成这些任务才能找到钱。一个简单的基于网络的问题可能包括与一个在线的人联系，并询问答案。

Questli 还实现了密码保护任务，因此人们可以在活动期间玩这家初创公司所说的“私人寻宝”。来自西伯利亚的创始人 Danil Kozyatnikov 表示，这款应用受到了赞助任务企业的大量关注。
