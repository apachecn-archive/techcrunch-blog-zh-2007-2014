# 也许我们可以用科技找出尼克松 18.5 分钟的差距

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/07/28/maybe-we-can-use-technology-to-find-out-what-nixons-185-minute-gap-was/>

![nixonnn](img/0d6bf5b68e89ffeaf38636e13d7a6f04.png "nixonnn")

几年前，惠蒂尔·科尔吉最著名的毕业生理查德·尼克松总统在他的办公室里讨论一件事。他最喜欢的助手 H.R .霍尔德曼也在那里。确切地说，他们讨论的内容已经成为前美利坚合众国的一个传奇。你听说过 18.5 分钟的间隙吧？这就是我们今天关心的:在那 18 分钟里说了些什么？他们有没有在推特上谈论，或者事实是——震惊！—苹果有点[控制狂](https://web.archive.org/web/20221007093617/http://www.beta.techcrunch.com/2009/07/27/twitter-adopting-the-word-tweet-on-its-website/)？没有人知道…直到现在？也许吧，朋友。也许吧。

你可以在维基百科上阅读关于[这段 18.5 分钟的间隙的所有内容，但提要是:1972 年的一天，尼克松总统和他的幕僚长 H.R .霍尔德曼在椭圆形办公室谈话。现在，尼克松总统，就是他，有一个秘密录音系统在椭圆形办公室一直运行。如果你在里面谈论你最喜欢的棒球运动员，就会被记录下来。除了我们说的这 18.5 分钟的间隙。多年来，人们一直想知道在 18.5 秒的谈话间隙里说了些什么。一些让尼克松看起来很糟糕的事情，所以他擦掉了那部分带子，造成了所说的间隙？再说一次，没人知道。](https://web.archive.org/web/20221007093617/http://en.wikipedia.org/wiki/Watergate_tapes#18.C2.BD_minute_gap_tape)

这就是它变得“高科技”的地方多年来，人们一直试图“重现”磁带丢失的部分，但最新的方案可能最有可能成功。一位名叫菲尔·梅林杰的先生想看看霍尔德曼的笔记。(霍尔德曼先生是一个惊人的记录员。)霍尔德曼在有 18.5 分钟间隙的会议上做的笔记显示，也许，仅仅是也许，他在 18.5 分钟间隙做了笔记，但随后就销毁了。

这个想法是查看幸存的黄色便笺簿，并使用一种叫做静电检测分析的方法来检查压痕。什么？

> 正如《法医揭秘》一书所描述的过程，一张纸受到一个静电场的作用:“来自这个静电场的带电电子被吸引到纸上有压痕的受损或压痕的纤维上。”然后，可以将墨粉放在纸上——或一张薄薄的玻璃纸上——它将附着在有凹痕的区域:“书写变得明显。”瞧，笔记可以从碎纸机里退回来。原作不会受到伤害。

所以，如果你能看懂笔记，那么你就能弄清楚在 18.5 分钟的间隙里说了些什么。

当然，前提是霍尔德曼先生真的在那 18.5 分钟的间隙里*做了笔记！*
