# Wii 'Classic Controller Pro '体积增大，增加了更多按钮和抓握性

用标准的 [Wii Classic 控制器](https://web.archive.org/web/20230314024955/http://store.nintendo.com/webapp/wcs/stores/servlet/ProductDisplay?lastAction=setCurr&amp;jspStoreDir=NOASTORE&languageId=-1&catalogId=10001&categoryId=62707&productId=117708&currency=USD&storeId=10001&ddkey=SetCurrencyPreference)玩 Super Smash Bros. Brawl 还可以，但在我男人的手中感觉有点小——就像我在用格兰诺拉麦片棒或口琴什么的玩视频游戏。我肯定它对年轻一代来说很好，但当它在今年夏天上市时，我可能会被经典的控制器专业版所吸引。

它看起来更加充实，增加了一些手柄，更多的肩钮，以及模拟拇指棒的间隔。价格和确切的可用性是未知的，但我打赌它会运行 25 至 30 美元。我一想到我在 Wii 系统上花了多少钱就不寒而栗，这些额外的控制器和塑料枪状的小玩意儿。
