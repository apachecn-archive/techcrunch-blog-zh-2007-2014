# 谷歌风投喝了一口牛奶，投资了凯文·罗斯的新公司

凯文·罗斯的移动实验室“牛奶”有了新的投资者:谷歌风险投资公司。这家初创公司在去年 4 月的种子轮中从投资者那里筹集了 150 万美元，投资者包括克里斯·萨卡、伊万·威廉姆斯、戴夫·莫兰、约书亚·沙赫特、罗恩·康韦、查马斯·帕里哈皮蒂亚、TechCrunch 创始人迈克尔·阿灵顿、首轮资本和 True Ventures。种子基金现在筹集的资金总额是 170 万美元。

“我们在结束融资后开始与谷歌对话，”罗斯说，“但我们决定向他们开放，因为他们已经立即增加了大量价值。”

谷歌地图是米尔克的第一款移动应用 [Oink](https://web.archive.org/web/20230314040619/http://itunes.apple.com/us/app/oink/id451160341) 的重要组成部分。Rose 希望在未来的更新中加入更多的谷歌技术。

Oink 让人们评价和发现他们周围的事物。这是为现实世界的事物创建一个目录和推荐系统的第一步。当它上周推出时，Rose [给了我一个演示](https://web.archive.org/web/20230314040619/https://techcrunch.com/2011/11/04/kevin-rose-shows-oink/)，你可以在下面观看。
