# 视频回顾:Mode RCC-1075 多功能回收中心 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2009/04/20/video-review-mode-rcc-1000-all-in-one-recycling-center/>

![rcc-1075](img/22b4cb51c9d2845689a65177a3dbc305.png "rcc-1075")
[模式多功能高级回收中心](https://web.archive.org/web/20230204222238/http://www.modeproducts.com/products/premium_features.php)是一种回收解决方案，旨在帮助减轻回收负担。RCC-1075 的总容量为 13 加仑，分为 5 加仑的纸盒和 8 加仑的玻璃、塑料和金属纸盒，可以很容易地将可回收物分开。它还包括一个独特的压缩系统，一个跟踪你回收了多少的漂亮系统，以及一个提醒你回收的可编程警报。

【YouTube http://www.youtube.com/watch?v=HtGeZ0Iu3Lc&hl=en&fs=1]

**更新**:是啊，我是个傻逼。我所评论的型号实际上是 [RCC-1075](https://web.archive.org/web/20230204222238/http://www.modeproducts.com/products/premium_specs.php) ，而这篇文章的原始版本——以及上面的视频——指的是 RCC-1000。RCC-1000 不提供花哨的 EcoTrack 功能。否则，复审无效。

**简而言之**:RCC-1075 为管理和跟踪您的回收利用提供了一体化解决方案。内置压缩罐有助于你在更小的空间里储存更多的东西。数字显示器跟踪您回收物品的当前重量，以及您曾经回收的总重量。最重要的是，RCC-1075 包括一个提醒你回收日的警报！

**长版**:RCC-1075 大约是一个大型垃圾桶的大小和形状。铝制壁板让它看起来很漂亮。该装置的前面有一个数字时钟和一个 EcoTrack 功能显示器。该功能记录在任何给定时间单元内可回收材料的重量，以及记录迄今为止可回收材料总重量的运行记录。这是为了帮助您跟踪回收工作的有效性。

还有一个有用的提醒，告诉你这是回收日。你可以设置这个提醒在你需要的时候激活:在你当地回收发生的前一天晚上，或者是。它可以每周触发，或每隔一周触发。这个提醒仅仅是机器表面的一盏小灯，所以如果你是那种忽视汽车仪表板上“检查发动机”灯的人，这个提醒可能对你没有特别的帮助。

RCC-1075 使分离一些可回收物变得相当容易。前面的纸盒是用来放纸张的，前门有一个插槽，可以让您快速将纸张放入纸盒。这个 5 加仑的箱子有一个可调节的隔板，这样你就可以把你的邮件和你可能存放在这里的其他纸张分开。如果你每天都有报纸送来，这是你读完报纸后存放报纸的好地方。纸盒可以容纳两周的报纸，包括周日版！但是，请注意，EcoTrack 功能并不关注纸盒。

纸盒下方有一个脚踏板，可操作独特的压缩功能。您可以将铝罐和各种塑料瓶放入压缩箱中，踩下踏板，看着全副武装且可操作的~~战斗站~~回收中心缩小您的可回收物的尺寸！对于压碎易拉罐来说，这仅仅比用脚踩它们稍微有效一点(尽管远没有那么有趣);但是对于塑料瓶来说，它更有效。减少可回收物品的体积意味着你可以将更多的可回收物品放入 8 加仑的垃圾桶，这也意味着你不需要经常清空垃圾桶。

**一句话**:RCC-1075 型多功能高级回收中心是一个好主意。我觉得对于大多数休闲消费者/回收者来说有点太贵了。EcoTrack 功能非常酷，对于帮助小孩子理解一段时间后回收的有效性特别有用。

就我个人而言，我认为脚踏板和压罐功能浪费了很多空间，但这主要是因为我几乎从不喝易拉罐里的东西。[豪华回收中心](https://web.archive.org/web/20230204222238/http://www.modeproducts.com/products/deluxe.php)几乎是同样的东西，除了它用一个额外的储存箱取代了压缩系统。如果我要买一个 Mode 回收中心，很可能是这个。

Mode 还生产[可回收垃圾桶内衬](https://web.archive.org/web/20230204222238/http://www.modeproducts.com/products/bin_liners.php)，所以你可以额外确保你的回收努力不会被讨厌的不可回收塑料垃圾袋破坏！
