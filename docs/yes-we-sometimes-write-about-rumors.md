# 这是谣言星期天！苹果一直在迎合 DIY 指南应用 snap Guide 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/12/09/yes-we-sometimes-write-about-rumors/>

这是[谣言星期天！](https://web.archive.org/web/20221007164439/http://www.techmeme.com/121202/p14#a121202p14)再次来到 TechCrunch。我们从几个不同的来源听到了一个反复出现的报道，苹果公司一直有兴趣购买[snap guide](https://web.archive.org/web/20221007164439/http://snapguide.com/)——这是一个网络和移动应用程序，让人们创建和分享自己动手做的指南，如[巧妙的节日礼物](https://web.archive.org/web/20221007164439/http://snapguide.com/guides/make-instagram-magnets-for-less-than-10/)、[烹饪](https://web.archive.org/web/20221007164439/http://snapguide.com/guides/make-reeses-peanut-butter-style-brownie-cupcakes/)和[教你的狗弹钢琴](https://web.archive.org/web/20221007164439/http://snapguide.com/guides/teach-your-dog-to-play-the-piano-in-a-day/)。

在我们进一步了解细节之前，我们应该让你知道，Snapguide 的联合创始人丹尼尔·拉斐尔(Daniel Raffel)已经非常明确地表示，这家初创公司迄今为止已经从风投和天使投资人那里筹集了 700 万美元——包括 Index Ventures、Atlas Venture、CrunchFund、SV Angel 和戴夫·莫兰(Dave Morin)——没有与苹果进行任何正式谈判。

“Snapguide 没有被苹果收购，也没有与苹果进行收购谈判，”他说。“虽然我们非常尊重苹果，也喜欢在 iOS 上开发产品，但暗示收购或 M&A 对话已经发生(或正在发生)是不准确的。我们对我们正在建立的公司和摆在我们面前的机会感到兴奋，并期待在不久的将来发布重大产品更新。”

但是！(总有但是。亲爱的读者，以下是我们听到的各种建议，我们想与你分享。我们无法确认这些是否属实。我们把它放在这里:

*   苹果公司已经与该公司接洽出售事宜。我们听到的价格大约是 4500 万美元。
*   这并不是第一家向 Snapguide 示好的公司，其追求者名单中包括一些主要公司。
*   苹果已经对拉弗尔和他的联合创始人史蒂夫·克鲁维兹进行了尽职调查。
*   拉斐尔最近被看到与苹果工业设计副总裁 Jony Ive 共进晚餐。

所有有趣的(和有些虚假的)花絮都是肯定的，但是，基于 Raffel 的上述评论，看来无论两家公司之间发生什么*都不是一次直接的对话。那么，为什么会出现一个关于苹果和 Snapguide 的谣言呢？原因可能就像 Snapguide 本身一样简单。*

记得 iAd 吗？苹果的富媒体移动广告网络是一个例子，表明苹果有兴趣建立并进一步货币化其移动平台，并使其成为品牌更具粘性的地方。

Snapguide 是一个制作精美(有点让人上瘾)的地方，可以查看大量有用的/令人向往的信息，对于品牌来说，这是一个很有潜力的地方，可以通过更活跃的操作内容而不是简单的静态广告来营销自己。有点像微型版的 app，有些[像](https://web.archive.org/web/20221007164439/https://itunes.apple.com/us/app/johnsons-bedtime-baby-sleep/id488869049?mt=8) [做](https://web.archive.org/web/20221007164439/https://itunes.apple.com/us/app/mapmydogwalk-brought-to-you/id492760186?mt=8)。

我们可以很容易地将 Snapguide 视为像 Conde Nast 和 Martha Stewart 这样的品牌延伸平台。Snapguide 甚至一度在 Quora 上被称为“数字时代的玛莎·斯图尔特”。

还有一个事实是，苹果一直在逐步向其平台添加更多功能，以给人们提供更多使用手机的方式。(想想 Passbook、iBooks 或报摊，以及其中最大的应用程序商店。)类似 Snapguide 的东西可能是使用 iPhone 或 iPad 的一种更有趣、更实用的方式。

我们已经联系了苹果公司进行评论，如果我们从他们那里得到任何反馈，我们会及时更新。
