# 电子书收入首次超过了精装书

美国出版商协会今天发布的一份报告显示，电子书收入首次超过了精装书。年初至今，电子书收入达到 2.823 亿美元，而精装书收入达到 2.296 亿美元。几乎就在一年前，形势发生了逆转，电子书达到 2.2 亿美元，精装书超过 3.35 亿美元。

精装书的唯一增长是在年轻成人/儿童类别，精装书收入从 2011 年的 390 万美元增长到 1.877 亿美元，儿童电子书收入增长到 6430 万美元。

简而言之，电子书正在胜出。

虽然儿童/青少年实体格式的精装和平装书都出现了两位数的强劲增长(分别为 68.9%和 61.9%)，但 AAP 儿童/青少年电子书的首个月度数据显示，从 2011 年到 2012 年，增长了 475.1%。一些出版商将此归因于针对这些人群的设备有了更多选择，以及一些受欢迎的新版本。

AAP 调查了 1，149 家出版商收集数据，发现过去一年总贸易收入增长了 27.1%。考虑到大部分销售都是电子书，很明显，大多数购书者都忘记了媒介即信息的古老格言。事实上，就长篇小说和非小说而言，媒介显然是无关紧要的。

鉴于[宜家甚至减少了货架的尺寸，以反映人们对印刷书籍的缺乏兴趣](https://web.archive.org/web/20221218034157/https://techcrunch.com/2011/09/09/death-of-books/)，你最好现在就进货，以免这些东西步黑胶唱片、8 轨唱片和 merkin 唱片的后尘。
