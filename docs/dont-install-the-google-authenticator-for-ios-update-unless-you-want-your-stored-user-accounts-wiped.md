# 不要安装 iOS 更新的谷歌认证器，除非你想清除你存储的用户帐户

谷歌今天发布了 iOS 版谷歌认证器的更新，这是一款双因素认证配套应用，可以让你登录的谷歌账户和服务更加安全。但据联系过 TechCrunch 的多名知情人士透露，这是用户目前希望避免的更新，因为它会删除你现有的所有存储数据和关联账户。

Authenticator 应用程序的工作原理是为用户提供一个定时的、过期的随机生成的数字代码，他们使用该代码和他们的密码登录服务。它增加了一层额外的安全，因为有人试图未经许可访问你的帐户是非常困难的，以超越认证代码和密码检查。

更新该应用程序会删除你现有的所有账户，用户在更新评论中抱怨说，他们在更新后丢失了 Dropbox、Google Apps、DreamHost、Twilio、Evernote 和其他令牌，被迫再次同步每个账户。这可能会让你的账户完全被锁定，并被迫联系支持人员进行重置，正如 Dashlane 的联合创始人 Alexis Fogel 告诉我们的那样，他的服务正在发生这种情况，这也将导致大量不必要的繁忙工作来设置之前已经设置好的东西。

很有可能谷歌会在不久的某个时候推出一个 bug-busting 更新来修复帐户擦除问题，所以现在最好的行动是推迟更新这个特定的应用程序。目前，Dashlane 还建议其用户在更新前禁用谷歌认证，如果他们仍然觉得有必要的话，但让更新继续存在仍是最好的做法。我们已经联系了谷歌，看看他们是否可以为已经更新的用户提供修复，或者他们是否有一个时间表，可以确定修复漏洞的更新何时会发布到 iOS 应用商店。当我们知道更多的时候我们会更新。

**更新:**该应用已经从应用商店下架，谷歌正在努力解决问题。
