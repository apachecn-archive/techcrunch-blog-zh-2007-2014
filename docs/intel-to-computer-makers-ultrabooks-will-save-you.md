# 英特尔致电脑制造商:超极本将拯救你 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/19/intel-to-computer-makers-ultrabooks-will-save-you/>

当我写这篇文章时，我正坐在办公室里。在我周围，有 9 台电脑——其中 7 台运行 Windows。我也有三块石板——只有一块是 iPad。欢迎来到旧金山以外的后 PC 世界，在这里微软仍然是老大，苹果是小众的，但无可否认，是有影响力的玩家。

我们可能处于后 PC 时代，但 Windows PC 远未消亡。苹果公司发展迅速，但似乎专注于高利润产品和大型飞碟总部，而不是推动社会进入下一个计算时代。那是英特尔的工作。他们将会克隆 MacBook。

英特尔公司和主要的个人电脑制造商正在努力开发下一代个人电脑，如果营销得当，这将迎来电脑的下一个黄金时代。所谓的超极本本质上是 MacBook Air 的克隆版。它们围绕紧凑型主板、CPU 和闪存选项而设计。第一批产品可能会使用英特尔目前的 CPU，代号为 Sandy Bridge。当 22 纳米 Ivy Bridge 型号在今年晚些时候以低于 1000 澳元的价格上市时，真正的乐趣就来了。

惠普昨天基本上退出了消费者个人电脑领域；他们走了 IBM 和 Sun 的路。消费者个人电脑世界是一个残酷的市场，各公司不断用廉价的、有损品牌的硬件竞争到底。利润率极低，而且每季度都有全新的产品线推出，客户流失不停。只有苹果，看起来走的是一条高速路，坚持一个缓慢的，有利可图的步伐，每隔几个月推出一点硬件，每隔一年推出全新的电脑。但是惠普跳槽可能有点太早了。

超极本正是 Windows PC 世界所需要的。超便携笔记本电脑极具吸引力，即使规格适中，坚如磐石的 Windows 7 也能完美运行。

据报道，第一批超极本将于下个月上市，但根据之前的报道，价格可能会比预期略高。华硕有 UX21，而[就在今天](https://web.archive.org/web/20230204234216/http://www.digitimes.com/news/a20110819PB204.html)，Digitimes 报道宏碁也有一款超极本计划于 9 月发布。

说每个人都想要苹果电脑是虚伪的；Windows 巨大的市场份额讲述了一个不同的故事。但是很难想象任何一个逛过百思买苹果显示屏的人不会对其光滑的硬件印象深刻。然而，出于各种原因，数据显示他们经常回到笔记本电脑的主显示屏，购买一些更熟悉、更便宜的东西。这就是英特尔试图通过给计算机公司提供一个小得多的平台来对抗超极本的原因。

苹果有先见之明，认为移动电脑需要刺激。MacBook Air 和 iPad 无疑提高了它的利润，这也是苹果在 2011 年第二季度(T1)领先于其他公司的主要原因。苹果在这一领域同比增长 136%，最终售出 1350 万台移动设备。(笔记本和平板电脑)然而，Windows/Android 设备的整体销量是 OS X/iOS 产品的两倍。

Windows PC 市场略显陈旧。个人电脑领域已经有一段时间没有新的范例了。这就像那些微软商店的广告，女士说“我不需要一台新电脑。我的电脑做了我需要它做的一切。”今天的 Windows 笔记本电脑看起来就像三年前的 Windows 笔记本电脑一样。由于如今大多数计算都是在网络浏览器中进行的，很难告诉消费者他们需要一台新的五磅重的笔记本电脑来运行 Chrome。但他们真正需要的是一款新的两磅重的超便携电脑，其电池续航时间是原来的三倍。

超极本不会“拯救”Windows PCs 他们不需要拯救。超极本将导致许多消费者搁置他们笨重的 Windows 笔记本电脑，进入真正便携式电脑的精彩世界，而不是过去五年来他们一直背负的平庸技术。
