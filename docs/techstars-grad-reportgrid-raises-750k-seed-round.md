# TechStars Grad ReportGrid 筹集了 75 万美元的种子资金

最近刚从 [2011 年 TechStars Boulder 班](https://web.archive.org/web/20230205012957/https://techcrunch.com/2011/08/04/startup-accelerator-techstars-hatches-12-new-companies-in-boulder-heres-a-peek/)毕业的毕业生 ReportGrid 刚刚筹集了 75 万美元的种子资金。该公司是一家数据分析即服务产品公司，因通过云可扩展数据库和可视化引擎为其分析和报告 API 提供支持而闻名。

本轮主要投资者包括 Launch Capital、David Cohen、Walt Winshall、Doug Derwin 和 Ed Roman。

该服务旨在成为构建和维护本土分析解决方案的交钥匙托管替代方案，例如基于 Hadoop 等开源技术运行的解决方案。

[ReportGrid 的](https://web.archive.org/web/20230205012957/http://www.reportgrid.com/)产品的“魔力”在于它提供的 API 的简单性，它是 RESTful 的并且支持 JSON。( [ProgrammableWeb](https://web.archive.org/web/20230205012957/http://blog.programmableweb.com/2011/05/16/reportgrid-analyzes-your-data-with-artificial-intelligence) 如果你有兴趣了解更多，可以在 5 月份深入研究一下这个 API。)该 API 还可以嵌入到一家 SaaS 供应商的产品中，使可视化和分析成为他们提供给客户的产品的一部分。

目前，ReportGrid 正在为一系列客户提供分析支持，包括 SnapEngage 和 Fidatto。

ReportGrid 总部位于科罗拉多州博尔德市，其顾问包括 Jim Franklin (SendGrid)、Ari Newman (Jive Software)、Bill Boebel (Rackspace)和 Doug Derwin (Taylor & Company IP Law)。
