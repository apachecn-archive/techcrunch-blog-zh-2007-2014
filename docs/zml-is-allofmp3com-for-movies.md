# 对于电影来说，ZML 是 Allofmp3.com

![zml](img/c4dc6c0e8b9231e18abe4cc53117155e.png)

向美国电影协会下一个溃疡的源头问好。这是[ZML.com，](https://web.archive.org/web/20230206121829/http://www.zml.com/)一个类似于前重量级音乐冠军[Allofmp3.com](https://web.archive.org/web/20230206121829/http://www.crunchgear.com/2007/07/25/former-allofmp3com-owner-faces-jail-time/)的超简单和低价格的网站。

该网站提供不同质量水平的电影，分辨率较低的电影起价为 1.99 美元，分辨率最高的电影(DVD 质量)最高为 4.99 美元。如果听起来好得不像真的，那很可能就是真的。使用风险自担，因为这个服务的合法性类似于 Allofmp3，[被关闭](https://web.archive.org/web/20230206121829/http://www.crunchgear.com/2007/07/02/allofmp3com-is-no-more/)。

电影是为你的 iPod、PDA 和/或大屏幕预先格式化的，但是你想要的电影是否有特定的格式却是一个废话。它还没有像 Allofmp3.com 的动态文件编码那样进化。我们会像老鹰一样盯着这个。我几乎能听到聚集的律师大军微弱的洗牌声。

这就像电影的 AllOfMP3:亲身体验 ZML 的无数字版权管理电影
