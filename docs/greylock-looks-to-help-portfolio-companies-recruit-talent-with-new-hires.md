# Greylock 希望帮助投资组合公司招聘新员工

风投和投资者的一个关键角色是帮助投资组合公司和初创公司招募优质人才。尤其是在当前竞争激烈的招聘环境下，招聘对于大公司和小公司来说都是一项具有挑战性的任务。今天，风险投资公司 Greylock[宣布](https://web.archive.org/web/20230326045102/http://greylockvc.com/2011/10/17/greylock-builds-talent-team/)它已经建立了一个专门的人才团队，以帮助其投资组合公司有效地招聘人才，并在当前的人才争夺战中取得成功。

高管猎头公司海德思哲(Heidrick & Struggles)前风险投资业务董事总经理杰夫·马科维茨(Jeff Markowitz)已加入 Greylock，成为其“人才合伙人”。Markowitz 将专注于高管级别的人才，并将帮助 Greylock 的投资组合公司以及风投公司本身建立一个高级人才网络。

曾在 Rypple 和 Mozilla 担任人力资源和招聘主管的丹·波蒂略(Dan Portillo)加入了 Greylock 投资组合公司，担任人才副总裁，主要负责产品和工程方面的招聘。

显然，Greylock 正在认真对待硅谷和更大的技术世界的技术人才之战，并正在配备人员帮助初创公司发动这场战争。这是一种趋势，其他顶级风险投资公司[也在积极追求](https://web.archive.org/web/20230326045102/http://blogs.wsj.com/venturecapital/2011/07/14/venture-firms-battle-to-woo-top-tech-talent/)。该公司表示，它将致力于几项举措，以帮助公司建立招聘技能，寻找人才，利用网络等。

Greylock 最近也在自己的花名册上增加了新的人才；[让](https://web.archive.org/web/20230326045102/http://greylockvc.com/2011/10/13/welcome-sanjay-raman/) Howcast 联合创始人、前谷歌员工[桑杰·拉曼](https://web.archive.org/web/20230326045102/http://www.crunchbase.com/person/sanjay-raman)担任消费者团队的高级助理。他加入了其他新员工的行列，包括前 Twitter 和脸书产品负责人[乔希·埃尔曼](https://web.archive.org/web/20230326045102/http://www.crunchbase.com/person/josh-elman)和 LinkedIn 前搜索和平台副总裁[亚当·纳什](https://web.archive.org/web/20230326045102/http://www.crunchbase.com/person/adam-nash)。
