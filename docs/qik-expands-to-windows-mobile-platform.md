# Qik 扩展到 Windows Mobile 平台

![](img/870ce88ab4c6e4fb175d5599890c3dae.png)

明天 [Qik](https://web.archive.org/web/20230121135627/http://www.qik.com/) ，从你的手机播放视频的流行网络广播服务，将宣布支持微软的 Windows Mobile 平台。对手机的实际支持将在未来几个月推出。首先，该网站为摩托罗拉 Q 和三星 21 点推出了仅限受邀者的 alpha 测试版。我们已经收到了给[techcrunch@qik.com](https://web.archive.org/web/20230121135627/mailto:techcrunch@qik.com)发邮件的前 50 名 TechCrunch 读者的邀请。

Qik 是一个超简单的视频广播系统，它使用手机摄像头来录制视频，并实时传输到 Qik 的服务器上。观众可以观看来自嵌入式播放器或 Qik 网站的直播流，所有内容都会存档以供将来参考。这项服务已经成为许多人的重要工具，甚至是一些不太可能的人——罗伯特·斯考伯，他在 2006 年宣称自己[“只使用高清摄像机”](https://web.archive.org/web/20230121135627/http://www.newmediamusings.com/2006/12/why_scoble_shoo.html)最近已经成为一个狂热的(而且肯定不是高清的)Qik 用户:他最近使用这项服务参加了与 Twitter 创始人的一次重要[采访](https://web.archive.org/web/20230121135627/http://techcrunch.com/2008/05/30/scoble-interviews-twitter-founders-evan-willams-and-biz-stone/)，他为自己辩护说，如果他一直拖着他的高清摄像机，他根本不可能参加这次采访。

到目前为止，Qik 只能在基于 S60 平台的诺基亚智能手机上使用。Qik 表示，他们计划逐步引入对所有 Windows 手机的支持(可能带有摄像头和高速数据计划)，尽管他们没有提供时间表。Windows Mobile 平台代表着 Qik 潜在用户群的绝对大规模增长——微软预计仅在 2008 年就能售出 2000 万份许可证。

我们越来越喜欢 TechCrunch 上的 Qik:这项服务非常容易使用，对于通过蜂窝网络传输的视频来说，质量通常出奇的好。今天早些时候，我们用这项服务直播了对 Gmail 项目经理托德·杰克森的采访，本月早些时候，我们用它直播了比尔·盖茨在 Advance08 上的 T2 主题演讲。

Qik 在这一领域有许多竞争对手，包括 [Flixwagon](https://web.archive.org/web/20230121135627/http://www.flixwagon.com/) 和 [Kyte](https://web.archive.org/web/20230121135627/http://www.kyte.com/) 。
