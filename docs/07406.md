# 如何在互联网上建立受众:凯文·罗斯学派与弗雷德·威尔逊学派 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/11/how-to-build-an-audience-on-the-internet-the-kevin-rose-school-vs-the-fred-wilson-school/>

***编者按**:这篇客座博文由 MySpace 前总裁、创始人、第一个朋友[汤姆·安德森](https://web.archive.org/web/20230205023611/http://www.crunchbase.com/person/tom-anderson)撰写。你现在可以在[脸书](https://web.archive.org/web/20230205023611/http://www.facebook.com/myspacetom)、[推特](https://web.archive.org/web/20230205023611/http://twitter.com/#!/myspacetom)和[谷歌+](https://web.archive.org/web/20230205023611/https://plus.google.com/112063946124358686266)* 上找到汤姆

你是什么样的内容创作者？凯文·罗斯还是佛瑞德·威尔森？

任何形式的博客或“自我出版”都被认为是媒体行业的民主化和革命。内容创作者不再需要主流平台来让他们的声音被听到。博客作者不受制于编辑，自助出版商可以通过 AdWords 或任何一家广告网络合作伙伴将他们的内容货币化。

三天前，凯文·罗斯[在 Google+上发布了这个](https://web.archive.org/web/20230205023611/https://plus.google.com/110318982509514011806/posts/ZoUX52aowxy):“决定将 http://kevinrose.com 转发到我的 Google+。G+给了我比我的博客更多的(实时)反馈和参与。”他的决定引发了一场关于这样做的利弊的小辩论。

凯文的决定做得很快，并且当它适合他的需要时，很可能被逆转。他对 G+的感觉从在第一篇文章中质疑这项服务的价值(后来被删除)，到思考 Google+如何更像一个博客，再到周五彻底转变。(周四，我通过短信提醒他:“老兄，你完全可以用 Google+取代你的博客了。”

在哪里存放你的内容是一个棘手的问题。当博客开始成为一种严肃的努力，互联网人意识到他们可以聚集自己的观众，他们自然认为拥有自己的域名，控制自己的分发列表，维护已经建立的内容链接，或者总而言之，控制自己的命运是很重要的。

再说一次，这不仅仅是博客作者的问题。一般来说是关于网络存在的。还记得宣传 AOL 关键词的电视广告吗？品牌推广自己的网站会更聪明吗？音乐人用 Myspace 做唯一网站怎么样？乐队最初使用 MySpace 作为他们网站的附属品，但渐渐地，越来越多的乐队发现维护自己的网站是一件痛苦的事情，于是完全关闭了网站。甚至主要的唱片艺术家也在他们的 CD 封面上印上了他们的 MySpace 网址(而不是他们的网站网址)。现在访问这些 MySpace 的网址，它们是鬼城。Youtube 视频频道怎么样？你能想象有一天 Youtube 不再是寻找视频的最佳地点吗？

用一种新的方式来表述这个问题，也许更好的方式是:一个内容创作者应该走向他的观众，还是应该期待他的观众走向他？

在离线世界，大多数作家永远不会想到出版自己的杂志或报纸。一个作家利用一个更大的平台/品牌(华尔街日报、NYT、时代周刊)的发行来宣传他的故事和名字。但是在网络上，一些人认为技术已经改变了这一切。有吗？

这种把所有东西都贴在自己域名上的模式在互联网早期可能行得通。但是谁这么有趣，能让他们有足够多的观众来保留一个书签并查看他们的网站？Technorati 没有一个个人博客进入前 100 名。

个人品牌(普遍认可)、网站忠诚度和搜索引擎优化是旧的自助出版模式的缓慢吸引因素。很明显，今天，一个博客需要一种机制来通知人们他的帖子——电子邮件列表(仍然是半老的学校), Twitter(引导和吸引长格式的短帖子),在流量很大的网站(如 TechCrunch)上的访客帖子，以及像脸书(粉丝页面)和 Google+这样的社交网络。

最后，你把家安在哪里取决于你为什么创作内容。如果你想让你在互联网上的小点分享一些媒体或表达你的想法，那么 Tumblr 或任何其他平台都可能是你的选择。如果你是马克·苏斯特或 T2·弗雷德·威尔森，你可能不会想要像 Google+这样的参与度。作为被许多年轻企业家视为打开王国之门的风险投资家，马克和弗雷德可能意识到人们必须做一些工作和研究才能找到他们各自的博客并与之互动。他们可能宁愿限制讨论，也不愿被在 G+上发布所有内容所带来的参与度和流量所淹没。

但最重要的是，我认为，你需要保持灵活性，关注互联网的发展。没有放之四海而皆准的策略，你可能想要修改你自己的策略，以适合你的需要。随着某些流量来源(科技博客、社交网络)的起起落落，你可能需要驾驭它们，并在下降的过程中转变你的受众。当提拉·特基拉(一个拥有数百万朋友的 MySpace 用户)开始看到 MySpace 是一艘正在下沉的船时，她转到了 Twitter，后来又转到了脸书。如果你正在寻找某种类型的观众，那么你可能需要找到这些观众在哪里闲逛，把你的内容带给他们，如果你愿意，希望把他们中的一些带回你的“大本营”。

凯文瞬间就放弃了他的博客，而佛瑞德却年复一年地建立了他的 T2 博客。你属于哪个学校？
