# 新的无线技术可以让你的手机电池寿命延长一倍

![](img/b9946874f829f292beca062a895a87aa.png)

如果你觉得你正在使用的 wifi 网络上有人霸占了所有的能量，你可能是对的。与他人共享 wifi 下载大文件不仅会妨碍你欣赏最新的病毒视频，还会严重消耗你的电池。

杜克大学计算机科学研究生的一个新解决方案可以减轻你的挫折感，并通过允许你的 wifi 设备“小睡”直到有更多的带宽可用来潜在地延长你的电池寿命。这意味着你可能要等几分钟才能看完你的视频，但这无论如何都是一种生产力的提升。

![](img/debc05e81065607ccd5ad0da8097e748.png) [Justin Manwelier](https://web.archive.org/web/20230203120225/http://www.cs.duke.edu/~jgm/) 的 SleepWell 是一款软件，当相邻设备上的下载完成时，它会提醒手机中的 wifi 设备，从而帮助最大限度地提高下载效率。大多数 wifi 设备在等待下载时必须“保持清醒”，在此过程中会消耗大量电池。

Manwelier 在一份[声明](https://web.archive.org/web/20230203120225/http://www.eurekalert.org/pub_releases/2011-06/du-wd062911.php)中说:“支持睡眠的 wifi 接入点可以错开它们的活动周期，以最大限度地减少与其他接入点的重叠，最终在性能损失可以忽略不计的情况下实现有希望的能源增益。”。

无线设备之间的距离也会影响它们的性能，在城市等更拥挤的地区，下载时间会更长。而且，随着云计算的不断发展，连接 wifi 的理由也会越来越多。将连接 wifi 的设备置于待机状态，直到从另一个设备下载完成，这样可以延长两个设备的电池寿命。

关于 SleepWell 何时发布还没有消息，但一旦发布，我们都可以期待在寻找出路之前有更多的时间冲浪。
