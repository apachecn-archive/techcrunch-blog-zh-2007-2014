Xoom太贵？试试这5款便宜的安卓平板电脑吧TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/04/07/xoom-too-expensive-try-these-inexpensive-android-tablets-instead/

昨天关于Xoom发布失败的帖子的评论清楚地表明，人们想要便宜的安卓平板电脑——或者至少是高建议零售价导致了Xoom的失败。人们的想法是，如果你打算花500多美元买一台平板电脑，iPad是唯一的选择，因此800美元的Xoom就成了滞销品。我有点同意这一点，尽管我仍然认为Xoom是一款非常棒的平板电脑。不过，我觉得糟糕的营销和产品植入注定了平板电脑的命运，而不仅仅是价格。摩托罗拉从未公开证明价格合理。

也就是说，Xoom将是一个类似的，但仍然不同的失败鲸，即使它以300美元推出。那就不一样了。双核CPU、可爱的屏幕和大量的内存将是砧板上的第一选择。从那时起，Moto可能会将制造质量从坚固降低到廉价，并选择更弱的电池。价格较低的Xoom平板电脑不会是更成功的平板电脑。垂直iTunes生态系统打开了一个巨大的收入流，使苹果能够为产品定价，与大多数Android平板电脑制造商不同。真的不公平。

尽管如此，说公司忽视低端Android平板电脑市场是不诚实的。它实际上是活着的，而且不仅仅是欣欣向荣。但是拜托，不要把跳票后的平板电脑称为iPad甚至Xoom的竞争对手。他们甚至不是一个级别的。但这并不意味着它们不值得一看。

优派G平板电脑

优派G-Tablet是粉丝的最爱。在XDA的论坛上，它的讨论主题数量仅次于Nook Color。为什么？它相对便宜并且容易改装。内部是1GHz双核Tegra 2芯片组，512 MB DDR2内存，以及足够运行1080p视频的视频流。传统的Android 2.2版本运行起来很快，但不如iPad 2或Xoom流畅。那不重要。不要买gTab跑2.2。购买Gtab在各种开发论坛站点上运行几十个不同的定制构建。想让它看起来像亚当的想法吗？使用这个ROM 。

  • 建议零售价:499美元
  • 互联网价格: $299

优点:

  • 充满活力的支持社区
  • 附带相当数量的内置应用程序
  • 轻量级选手

缺点:

  • 轻量级是廉价的同义词
  • 有时有点迟钝
  • 屏幕低于标准

爱可视101

爱可视正在向市场推出低价安卓平板电脑。有2009年发布的最初的5款互联网平板电脑(和审查)，随后是48、7、10和，现在是流行的101 。该公司甚至推出了一条名为 Arnova 的新产品线，它只占用最少的物理内存。但101卖得像廉价的安卓平板电脑。

101有8GB和16GB两种规格，前者的网上售价为280美元，后者约为325美元。两者都有相同的10.1英寸屏幕、Cortex-A8 CPU和长寿命锂离子电池。端口丰富，有全尺寸USB 2.0端口、方便的USB 2.0从端口、MicroSD卡插槽和HDMI输出。此外，还有一个漂亮的支架。101开箱即用运行Android 2.2，但像许多其他平板电脑一样，修改者和开发人员目前正在为ROM乐趣敞开系统大门。一旦发生这种情况，101可能会像优派gTab一样受欢迎。

优点:

  • 综述中的最佳构建质量
  • 性感的硬件
  • 成吨的港口

缺点

  • 截至发稿，只是Bootloader被破解了。还没有rom。

eLocity A7

坦白说:我从未接触过eLocity A7平板电脑，所以我无法对构建质量或系统响应发表评论。然而，从规格和设计以及与所有者的交谈来看，这款平板电脑在价格上似乎与其他产品不相上下。它搭载了Android 2.2，在许多方面，与概念墨水亚当共享设计线索。端口丰富，有USB 2.0、HDMI、3.5毫米音频输出、MicroSD插槽和一个坞站端口。我能看到的主要缺点是7英寸屏幕带来的传统尺寸优势被相当大的边框有效地抵消了。然而在现实中，它只是比三星Galaxy Tab稍大一点，被证明是适合口袋的。

  • 建议零售价:349美元
  • 互联网价格: $299

优点:

  • 小的
  • 独特的风格
  • Tegra II平台

缺点:

Nook颜色

Nook Color是安卓平板电脑的Xbox。人们成群结队地购买它，而不是使用库存系统。太棒了。我现在有一个CyanogenMod在我的电脑上运行，我桌子上的某个地方有一个可启动的蜂巢MicroSD卡。修改社区如此活跃，即使是新手也不应该害怕订购一个来运行Android。NookDevs有很棒的DIY指南，XDA有一个大型的设备论坛。毫无疑问，它是市场上最好的入门Android设备。

然而也有不利的一面。Nook颜色对蜂巢来说动力不足。就是跑不好。当然，它可以工作，而且它是一个探索新平台的有趣设备，但我不能把它作为一个随身携带的平板电脑。Cyanogen 7是受欢迎的ROM ，2.3版本在该设备上运行良好。这有点古怪，可能不适合大众消费，但我们的许多读者仍然会感到满意。最重要的是，与其他廉价的Android平板电脑不同，Nook Color有大量的配件，包括大量的外壳，皮肤等等。改装不是你的事？刚刚宣布他们正在接受OEM系统的应用，这可能会成为一笔大交易。

  • 建议零售价:249美元
  • 互联网价格: $249

优点:

  • 薄而坚固
  • 在全国范围内的B&N商店销售，这使得退货变得容易
  • 高度可修改

缺点:

  • 250美元的平板电脑我看不到

三星Galaxy Tab

三星Galtab 是2010年炙手可热的安卓平板电脑，尽管它价格高，屏幕小。自从它首次亮相以来，变化并不大，这让我有点犹豫是否推荐这款平板电脑——特别是在两年承诺的情况下——然而，事实仍然是，它仍然是一款不错的入门级安卓平板电脑。威瑞森的合同售价仅为199美元。Honeycomb已经非正式地移植到平板电脑上，这表明即使三星忘记了它的第一个产品，互联网也不会忘记。就问问它最大的粉丝，我的第一个在线编辑， @KevinCTofel 。他抛弃了iPad，选择了GalTab。

优点:

  • 成吨的配件
  • 发展完善的平台
  • 3G功能
  • 一个坚实的未来，即使没有改装

缺点:

  • 对于7英寸的平板电脑来说很重
  • 接收官方Android更新可能会很慢
  • 当合同到期时会变得很老

现在有很多安卓平板可供选择，但等待可能是明智的。未来几个月，大量的蜂巢平板电脑将涌入市场。Android平板电脑可能会变得无处不在，每个看起来都和其他的一样。正是这些小细节让他们与众不同。一些将有数字笔，略有不同的屏幕尺寸，和无线选项。但这种泛滥很可能只会给消费者带来困惑，并有助于巩固iPad在市场上的主导地位——在短期内。最终，闸门将会关闭，Android平板电脑将成为iPad真正的竞争对手。至少我愿意相信这一点。
