微软在HTC Titan上展示Tango视频聊天应用

自从我第一眼看到Windows Phone Mango，我就喜欢上了它，但事实是，大多数正在实现的功能仅仅是为了赶上竞争平台。

Bing集成和语音转文本功能……非常酷。但除此之外，正如我所说，芒果正在追赶。在芒果驱动的HTC Titan 上添加视频聊天应用是另一个例子，但它即将到来。

尽管存在令人难以置信的不幸的名称共享情况，Tango视频聊天应用程序(与WP7下一次代号为Tango的更新无关)实际上非常酷，尽管有点晚了。在下面的视频中，Windows Phone团队的Aaron Woodman可能过于热情地详细描述了视频聊天在4.7英寸大屏幕上的工作方式(由 Windows Phone博客提供)。像苹果的FaceTime和Android的Tango应用程序一样，WP的Tango可以让你享受前置和后置摄像头之间的“看到我所看到的”切换。奇怪的是，Google Talk还没有解决这个问题。

很酷的是，芒果的Tango应用程序是跨平台的，支持iOS、Android和PC。微软碰巧拥有一个非常棒的叫做Skype的网络电话服务，所以这个Tango应用将来可能会在Windows平台上有一些非常激烈的竞争。
