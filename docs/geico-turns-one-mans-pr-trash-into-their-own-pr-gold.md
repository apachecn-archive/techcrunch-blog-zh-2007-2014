# Geico 把一个人的公关垃圾变成他们自己的公关黄金

朋友们，这就是我们所说的抓住时机。(Meme-mement？)

对于那些想知道为什么 Geico 突然忘记如何连贯地打字的人，这里是[的背景](https://web.archive.org/web/20221210022924/https://beta.techcrunch.com/2011/12/27/the-avengers-a-true-tale-of-bad-customer-service/)。TL；version 博士:一个厚颜无耻的营销人员在给客户的回复中(充满错别字)有点太自大了，消息传开了，互联网上辛辣的愤怒像地狱之火一样倾泻而下。Geico 抓住机会将他们的公关风格与一个小刺戳并列，互联网为此向他们发射了一座爱的大山(以转发和 Reddit 帖子的形式)。

这就是推特*应该做的*。对于那些仍在试图理解整个 Twitter 事件的公司来说:就像 Geico 那样做吧。让一个真实的人来回应(最好是把互联网当成他们第二个家的人)，并给他们像真实的人一样回应的灵活性。让他们搞笑吧。让他们开些内部玩笑。互联网 hivemind 会很感激的。

**更新:**对于那些希望从这场惨败中获得更多笑声的人来说，已经有一个用户制作的 [Cheezburger 子网站专门用于此事](https://web.archive.org/web/20221210022924/http://oceanmarketing.cheezburger.com/)。
