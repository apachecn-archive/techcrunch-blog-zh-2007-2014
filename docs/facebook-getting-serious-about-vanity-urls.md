# 脸书对虚荣心网址越来越认真

脸书越来越明白 MySpace 从一开始就知道的道理——用户喜欢虚荣的网址。当你告诉别人你的 MySpace 页面时，你只需说 MySpace . com/[用户/品牌/乐队/等。](我是[myspace.com/mikearrington)](https://web.archive.org/web/20221207193915/http://www.myspace.com/mikearrington)。在脸书，情况总是更加困难。

我的简介不是 Facebook.com/michaelarrington,，而是 facebook.com/profile.php?id=500065899 和 T4。不那么吸引人，结果是人们需要搜索才能找到你。MySpace 虚荣心网址之所以受欢迎，是因为人们喜欢域名，而不仅仅是输入 IP 地址。

脸书在这方面的政策显然在改变，至少对于页面(乐队、名人、品牌等。有他们脸书的存在)。一直以来，脸书都有一些与页面相关的虚荣心网址，比如[Facebook.com/U2](https://web.archive.org/web/20221207193915/http://www.facebook.com/u2)。然而，他们在去年八月明确表示，大多数人无法得到它们:

> 遗憾的是，我们目前无法批准虚荣心网址的请求。目前唯一有虚荣网址的网页是与我们合作推出新产品的乐队和企业。我们希望将来每个人都可以在自己的页面上使用这些 URL。当此功能可用时，我们将提供如何在 http://www.facebook.com/help.php?page=60.获得一个虚荣心网址的说明。同时，我们对任何不便之处表示歉意。

然而现在，脸书似乎让更多的人拥有了虚拟网址。例如，凯文·罗斯今天早些时候庆祝[得到一个](https://web.archive.org/web/20221207193915/http://twitter.com/kevinrose/status/1319143495)。阿什顿·库彻也有，黛米·摩尔也有。唉， [TechCrunch 页面](https://web.archive.org/web/20221207193915/http://www.facebook.com/pages/TechCrunch/8062627951)没有(悬停在那个链接上，哇)。
