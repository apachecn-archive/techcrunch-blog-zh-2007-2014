# 报道:任天堂 3DS 获得“滑动垫”附加功能(第一张图片)

日本最大的视频游戏杂志《Famitsu》本周的杂志将于明天出版，但游戏极客 [@south1996](https://web.archive.org/web/20230203064451/http://twitter.com/#!/south1996) 似乎已经提前拍摄了一张非常有趣的快照:这是第一张展示 3DS 游戏手柄插件的照片，游戏世界已经[猜测了](https://web.archive.org/web/20230203064451/http://www.engadget.com/2011/08/23/rumor-nintendo-working-on-3ds-redesign-10-joystick-add-on/)数周。

正如你所看到的，所谓的“滑动垫”被设计成像一个摇篮一样支撑着 3DS 的下部，使该设备整体上相当笨重(一个日本游戏博客[估计](https://web.archive.org/web/20230203064451/http://remoon.blog15.fc2.com/blog-entry-18210.html)它使 3DS 左边宽 1 厘米，右边宽 2-3 厘米，底部宽 2 厘米)。

滑动垫在 3DS (L1、R1 和 R2)的右侧增加了一个模拟圆形垫和三个肩部按钮。

目前还不知道该插件的上市日期或价格，但图片中的一行文字说它将可用于怪物猎人 3 (tri-) G，这是 3DS Capcom 目前正在开发的流行 RPG 系列的更新。
