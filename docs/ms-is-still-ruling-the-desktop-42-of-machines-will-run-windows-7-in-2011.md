# 微软仍然统治着台式机:2011 年 42%的机器将运行 Windows 7

Windows 7 现在是最流行的桌面操作系统，Gartner 估计目前 42%的电脑将运行该操作系统，而 94%的新电脑将运行 Win7。

相比之下，OS X 获得了 4%的份额，而 Linux 牢牢占据了 2%的份额。甚至 IT 部门也开始在他们的桌面上大规模部署 Win7，这一举动将这个相当新的操作系统推到了猫鸟的位子上。然而，随着 IT 部门转向托管计算和虚拟化，Gartner 预计这将是最后一次在商用电脑上安装独立的操作系统映像。

然而，并非一切都是阳光和玫瑰:

> Gartner 研究总监 Annette Jump 表示:“2010 年和 2011 年 IT 预算的稳步增长有助于加快 Windows 7 在美国和亚太地区企业市场的部署，从 2010 年第 4 季度开始，Windows 7 在这些市场开始大规模迁移。“但是，西欧的经济不确定性、中东和非洲(MEA)部分国家的政治不稳定以及日本在 2011 年 3 月地震和海啸后的经济放缓可能会导致 Windows 7 在这些地区的部署略显迟缓。”

如果你是订阅者，你可以在这里查看报告，但是一般来说，即使面对 OS X 和手机的流行，Windows 仍然很强大。
