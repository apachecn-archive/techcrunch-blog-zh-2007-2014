# Wakoopa 推出桌面应用“Alexa”

[![wakoopalogo.png](img/4b4c5b1cc30a4d33dc65472da593afcd.png)](https://web.archive.org/web/20221209001646/http://wakoopa.com/) 我们先把[写到](https://web.archive.org/web/20221209001646/http://www.beta.techcrunch.com/2007/04/27/wakoopa-lastfm-for-desktop-applications/)关于 [Wakoopa](https://web.archive.org/web/20221209001646/http://wakoopa.com/) 四月份推出的时候。这是一个可供应用程序爱好者下载的程序，可以跟踪你使用的软件或游戏。我们称它为应用程序的 Last.fm，暗指该程序的跟踪和推荐系统类似于音频搜索。

虽然存在明显的隐私问题(此处[解决](https://web.archive.org/web/20221209001646/http://wakoopa.com/about/privacy))，但已有超过 17，000 人注册了这项服务(没有关于下载的消息)。该网站每月吸引 50 万人浏览他们在 Mac、PC 甚至 iPhone 上跟踪的 7 万多个应用程序的简介。迄今为止，他们已经追踪了大约 1.1 亿小时的软件使用情况。Firefox 是排名最高的应用程序，使用时间超过 50 万小时。

所有这些使用产生了一些非常有趣的数据，Wakoopa 现在通过新的类似 Alexa 的图表展示这些数据。虽然 Wakoopa 将于明天正式推出图形功能，但 TechCrunch 的读者现在只需添加“？techcrunch=true "到任何 URL( [示例](https://web.archive.org/web/20221209001646/http://wakoopa.com/software/firefox?techcrunch=true))。像 Alexa 一样，图表显示了应用程序在样本人群中的相对排名和到达率(使用率%)。到明年第一季度，他们将允许多达 5 个应用程序的比较和嵌入。

诚然，样本人口是相当古怪的。目前的数据揭示了另一个世界，火狐的高级浏览器最终篡夺了 Explorer 的大部分市场份额(见下文)。然而，即使它可能有偏见，这些数据给了我们一个以前看不到的关于我们如何使用电脑的非常有价值的信息。例如，Wakoopa 发现周二是用户玩游戏最少的一天。他们还发现女性花在 Photoshop 上的时间是男性的两倍。

该团队明年将从阿姆斯特丹搬到硅谷，我很期待看到他们对产品的其他计划。
