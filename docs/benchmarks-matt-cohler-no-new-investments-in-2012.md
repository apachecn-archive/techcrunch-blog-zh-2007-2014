# Benchmark 的马特·科勒:2012 年没有新的投资

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/09/11/benchmarks-matt-cohler-no-new-investments-in-2012/>

TechCrunch 创始人[迈克尔·阿灵顿](https://web.archive.org/web/20230407173304/http://www.crunchbase.com/person/michael-arrington)今天上午在 TechCrunch Disrupt SF 与 Benchmark Capital 的[马特·科勒](https://web.archive.org/web/20230407173304/http://www.crunchbase.com/person/matt-cohler)坐下来讨论科技投资的现状。阿林顿问的第一个问题是关于科勒自己的创业投资，特别是他今年是否投资了任何新公司。苛勒说:“事实证明，他没有——但这不是他的一个具体决定。”。他还补充说，去年他做的投资比一个风险投资者通常会做的要多。

阿林顿后来追问他为什么没有在 2012 年投资(还没有)，问他估值是否是那个决定的一部分。苛勒表示，估值并没有导致他或他的合伙人改变策略，他表示，估值的影响在投资周期的后期最为强烈。基准测试在这一端不起作用。苛勒说，Dropbox 是 Bechmark 唯一的后期投资，但它通常参与 A 轮和 B 轮投资。

那他为什么没有扣动扳机呢？“时机成熟时我会这么做的，”苛勒说。“我的工作中令人痛苦的一件事是，随着时间的推移，你越来越清楚自己搞砸的所有事情和犯下的所有错误。我确信我已经传递了一些今年将被证明是非常重要和非常有趣的东西。”比如什么？“我还不知道。”

阿林顿还询问了苛勒关于 Benchmark 的一些更引人注目的公司，例如 Twitter，让他当场问苛勒是否认为 Twitter 正在进行的影响其开发人员社区的变化是邪恶的。本质上，Twitter 正在关闭其开发生态系统的大部分，影响开发者的生计。“他们有多邪恶？真的邪恶？还是有点邪恶？，”阿灵顿想知道。苛勒拒绝回答，只说 Twitter 是一家伟大的公司，贝马克很幸运投资了它。

聊天结束时，阿林顿问苛勒想投资哪些公司，苛勒说他对移动市场的机会感兴趣，他称之为“非常有趣”。从产品的角度来看，他建议创始人“把智能手机当成生活的遥控器”——也就是说，像基准投资 Instagram 和优步这样的应用程序。“从产品的角度来看，它们有一个共同点，”苛勒说，“就是你按下一个按钮，就会有令人惊叹的事情发生。”他称这种体验为“现实世界的遥控器”

那么，创始人应该如何联系来推销他们的应用呢？“在网上很容易找到我，”苛勒说。
