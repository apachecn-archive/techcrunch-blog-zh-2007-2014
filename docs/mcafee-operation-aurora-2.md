# 迈克菲称极光行动是“网络安全的分水岭时刻”，并提供了指导

![](img/fadc1e6d6b991369b72b846506d985d0.png)

计算机和软件安全公司[迈克菲](https://web.archive.org/web/20221229072139/http://mcafee.com/)上周[发现](https://web.archive.org/web/20221229072139/http://siblog.mcafee.com/cto/operation-%E2%80%9Caurora%E2%80%9D-hit-google-others/)微软 Internet Explorer 中的一个漏洞是[网络攻击](https://web.archive.org/web/20221229072139/http://techcrunch.com/2010/01/12/google-china-attacks/)的一个关键媒介，这些攻击以高调、多阶段和集中的方式攻击谷歌和其他 30 多家公司的特定计算机系统，以获取知识产权。

此后，雷德蒙发布了一份安全建议，随后发布了自己的零日威胁风险评估。今天早上，迈克菲宣布，它将为消费者和企业提供进一步的指导，这被称为[“极光行动”](https://web.archive.org/web/20221229072139/http://www.mcafee.com/us/threat_center/operation_aurora.html)。

它用最高级的词语来描述这些攻击。

McAfee 全球首席技术官 George Kurtz 一直在博客上讲述浏览器漏洞是如何被利用来进行网络盗窃的，今天早上的新闻稿引用了他的话，称这是“多年来针对特定公司的最大、最复杂的网络攻击”。

库尔茨没有说地球几乎停止旋转，而是称这次攻击是“网络安全的分水岭时刻”，已经“改变了世界”。

当然，McAfee 在传播有关攻击的信息以及其安全产品如何保护消费者和企业免受上述 Internet Explorer 漏洞的利用方面有着商业利益，但考虑到用于利用安全漏洞的代码已经进入公共领域，该公司也在提供服务。

不用说，这大大增加了进一步、更广泛攻击的风险。

详细指导可在[McAfee.com/OperationAurora](https://web.archive.org/web/20221229072139/http://www.mcafee.com/us/threat_center/operation_aurora.html)获得。
