# Switchkey 像弹簧刀一样把你的钥匙翻出来

已经有这样的产品出现了(很可能是在 Skymall)，但是它们中的任何一个都是如此精心制作的吗？它们有和[弹簧钥匙](https://web.archive.org/web/20221210070638/http://www.switchkey.ca/Site/what.html)一样的弹力吗？

基本上，它会把你的钥匙安全地放在它的小盒子里，直到你按下按钮，这时它大概会像弹簧刀一样弹出来。当然，为什么不呢？它们是在加拿大手工制作的，你必须[联系他们](https://web.archive.org/web/20221210070638/http://www.switchkey.ca/Site/contact.html)关于价格之类的。
