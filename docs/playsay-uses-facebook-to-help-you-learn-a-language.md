# PlaySay 使用脸书帮助你学习语言 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/13/playsay-uses-facebook-to-help-you-learn-a-language/>

今天在 TechCrunch Disrupt 上发布的, [PlaySay](https://web.archive.org/web/20230203170007/https://apps.facebook.com/playsay/) 是一款脸书应用，它利用了社交图和我们大多数人在脸书上花费过多时间的事实来帮助人们学习语言。

PlaySay 使用图像来表示语言学习的基本组成部分，允许基础语言学习者逐字逐句地建立或解构脸书状态更新。其理论是，用户会更热衷于在他们已经熟悉的环境中与朋友一起学习语言。

“你的脸书朋友是你的新同学。“签到、状态更新和图片都是你的课程材料，”该应用的创始人瑞安·梅恩泽(Ryan Meinzer)告诉我。事实上，在你朋友的个人资料页面上看到西班牙语的更新确实激起了你的好奇心，想知道那条状态更新说了些什么。点击一下状态，你就被传送到 PlaySay，它为你提供了通过图像解码状态的选项，即使你不会说一句西班牙语。

PlaySay 计划通过向用户提供兑现脸书积分的能力来实现盈利，以便解锁私人消息和级别等优质 PlaySay 功能，并为那些想要深入了解内容的人提供订阅选项。在语言学习流程中植入产品也是该公司正在探索的一条途径。

虽然目前只有西班牙语版本，Meinzer 计划通过内容合作伙伴关系(如最近与出版商 McGraw Hill 达成的合作关系)将该书翻译成各种语言。

Meinzer 告诉我，当他住在日本时，他想出了 PlaySay 的想法，并在第一次接触日语后受到启发学习日语。“我们希望让人们转而学习外语，”他说，“如果你使用谷歌翻译，你什么也学不到。”

PlaySay 有超过 57 万美元的资金。
