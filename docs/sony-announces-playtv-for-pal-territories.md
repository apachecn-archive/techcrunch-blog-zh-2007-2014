# 索尼宣布播放 PAL 领地的 PlayTV

![playtv.jpg](img/1450b363981562366389e1cba6c0ce43.png)

显然索尼更喜欢伙伴国家而不是我们。索尼[今天正式宣布](https://web.archive.org/web/20221006214137/http://crunchgear.com/2007/08/01/ps3-goes-dvr-in-2008/)PlayTV，这是一个电视调谐器和 PVR 的组合。英国、法国、意大利、德国和西班牙将在明年初获得第一批，其余的 PAL 地区将在我们 NTSC 之前获得乐趣。几周前首次宣布的细节仍然适用，因为 PlayTV 将在 DVB-T 上运行，并将使用 7 天电子节目指南进行 PVR 功能。两个电视调谐器允许 PS3 用户观看、录制和播放 1080p 内容。你可以用 SIXAXIS 或 BD 控制器控制一切。如果这还不够酷，PlayTV 将根据你的电视自动选择标准清晰度或高清。您还可以将录制的内容传输到您的 PSP，远程播放允许您在旅途中编程和观看 PlayTV 内容。该死，太酷了。
