# PayPal 与 FundRazr 和 Lottay 合作测试新的自适应支付 API

![](img/b328ab8ec2eae1f36c1418cd47d099b4.png)

最近，PayPal [在](https://web.archive.org/web/20221205071327/http://www.beta.techcrunch.com/2009/07/23/the-online-payment-wars-continue-paypal-officially-announces-flexible-api/)宣布了它的灵活支付 API，名为 Adaptive Payments(我们[抢先](https://web.archive.org/web/20221205071327/http://www.beta.techcrunch.com/2009/07/06/paypal-looks-to-crush-amazons-fledgling-payment-service-with-a-new-secret-api/)一步)让开发者可以完全使用 PayPal 的功能，允许他们在构建应用程序时有更多的自由，包括接受和分配支付的能力。

PayPal 将在 11 月举行的 PayPal X Innovate 2009 大会上发布这些 API。据 PayPal 称，开发者与会者将获得新 API 的独家使用权，非与会者要到 2010 年才能使用。PayPal 将公布其未来的平台路线图，这应该会很有趣。几周前，我们已经得到了 PayPal 未来愿景的预览图。

当然，看看开发人员可以用这个新的 API 做些什么，看看它真正的能力和连接性，总是很有帮助的。在过去的几个月里，PayPal 一直在与精选的开发者一起测试新的灵活支付平台，碰巧的是，其中两位测试者是 TechCrunch 50 demo pit 公司。
[FundRazr](https://web.archive.org/web/20221205071327/http://www.facebook.com/apps/application.php?id=18015191938) 是一款脸书应用程序，让任何个人、组织者或志愿者代表一个社会团体、运动队、学校俱乐部、慈善机构、运动或慈善团体筹款。这家初创公司使用 PayPal 的自适应支付 API 来处理交易。

另一家 TechCrunch50 demopit 公司正在基于 PayPal 的新 API 构建其平台，让你创建一个人们可以花钱购买的在线礼物。用户可以在 Lottay 上创建礼物页面，上面有特定目标或礼物的详细描述和图片，然后朋友可以通过 PayPal 向网站捐款。
