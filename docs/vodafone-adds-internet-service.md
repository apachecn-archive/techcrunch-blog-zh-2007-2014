# 沃达丰增加互联网服务

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/27/vodafone-adds-internet-service/>

![vodafone2.jpg](img/8d1fad9c5abaf1d7179db0b2b46f3fa5.png)想要更好的移动互联网接入？嗯，搬到不列颠群岛可能是合适的，或者至少去爱尔兰和英国旅行。沃达丰已经宣布与谷歌、Myspace、易贝和 YouTube 合作，努力为这些网站提供更强大的服务。

此外，沃达丰移动互联网用户将访问这些和其他网站的压缩和重新安排的内容，使其更适合移动设备。关于在手机上显示网站的最佳方法的争论仍在继续，但沃达丰选择了为其特定手机压缩内容的路线。希望这能让移动网络的使用更加流畅。我们不愿意想到有人会在易贝申奥中失败！

这项服务将提供给沃达丰的 3G 和 2.5G 用户，并允许用户以 99 欧元的费用在移动网络上冲浪，或者每月支付 9.99€的费用。当然会有沃达丰所谓的“合理使用”价格上限。考虑到 YouTube 和 MySpace 的组合，我们可以希望使用上限实际上是公平的。
