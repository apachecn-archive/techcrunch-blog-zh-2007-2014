# Mowser 创始人说移动网络已死。恰恰相反:移动网络昨天才诞生

[![mowser.jpg](img/1349e69d6f19b40f3d872dc175c5f34d.png)](https://web.archive.org/web/20230219130736/http://www.crunchbase.com/company/mowser) 移动互联网已死，[企业家【Russell Beattie 说，是他](https://web.archive.org/web/20230219130736/http://www.russellbeattie.com/blog/the-end-of-mowser)[死池](https://web.archive.org/web/20230219130736/https://techcrunch.com/tag/deadpool) [Mowser](https://web.archive.org/web/20230219130736/http://www.mowser.com/) 的时候了。阅读今天早些时候邓肯的帖子中的详细内容。

现在我当然认为，为移动设备创建专业的精简版网页的日子即将结束(这就是 Mowser 所做的)。带宽差的小屏幕等于一个不能用的产品。在今天的美国，几乎所有的移动浏览都发生在带有大屏幕和全键盘的智能手机上。尤其是 iPhone 的浏览界面非常友好，因为用户只需用手指移动屏幕，就可以在极其清晰的屏幕上放大或缩小。

简而言之，硬件的进步使得手机专用的标记语言变得多余。越来越多的人将拥有真正的智能手机，可以快速打开和浏览普通网页。看到整个屏幕。我们不再需要中间人软件将普通网站转换成普通手机能理解的东西。更好的办法是压低价格，让今天的 iPhone 在第三世界几乎免费。第一世界将会有越来越好的设计。

所以我不同意移动网络已死的说法。对我们中的许多人来说，它刚刚活过来。鉴于这些设备的发展速度和价格下降，我认为不值得人们花时间去开发优化体验的软件。相反，他们应该利用他们的专业知识来构建令人兴奋的新应用程序，这些程序将直接在这些新平台上运行。

所以不要认为这是你创业的死亡，罗素。当你想象如何改变世界时，把它当成一个让你的创造力放飞的机会。我猜你会在某个非常有趣的地方着陆，并开始构建用户会热情使用的软件。
