# MySpace 从 LinkedIn | TechCrunch 雇佣了一些社交高手

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/11/12/myspace-hires-some-social-smarts-from-linkedin/amp/>

MySpace 似乎有点招聘狂潮。在过去的几个月里，社交网络[聘请了](https://web.archive.org/web/20230305231428/https://techcrunch.com/2009/10/05/myspace-names-mark-rosenbaum-as-chief-financial-officer/) [马克·罗森鲍姆](https://web.archive.org/web/20230305231428/http://www.crunchbase.com/person/mark-rosenbaum)担任首席财务官，[亚历克斯·马根](https://web.archive.org/web/20230305231428/https://techcrunch.com/2009/09/29/myspace-loses-co-founder-and-cto-aber-whitcomb-names-alex-maghen-as-replacement/)担任首席技术官(他曾是 MySpace Music 的首席技术官)，前 AOL 和 Tsavo 高管[迈克·马卡达恩](https://web.archive.org/web/20230305231428/https://techcrunch.com/2009/07/10/myspace-hires-former-aol-and-tsavo-exec-mike-macadaan-as-vp-product/)担任产品副总裁，前脸书总监凯蒂·金明德担任用户体验和设计的 s VP。现在 MySpace 从 LinkedIn 招聘了一位产品经理， [Christina Wodtke，](https://web.archive.org/web/20230305231428/http://www.crunchbase.com/person/christina-wodtke)担任社交的总经理。

Wodtke 之前是 LinkedIn 的主要产品经理，负责管理活动流(最近有点喜欢上 Twitter 了)、通信产品和事件应用程序。她还在雅虎建立了搜索和市场设计团队，并帮助重新设计搜索、购物和本地产品。Wodtke 将归首席产品官 Jason Hirschhorn 管辖。
