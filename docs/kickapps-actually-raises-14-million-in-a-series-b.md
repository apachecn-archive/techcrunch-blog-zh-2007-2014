# KickApps 在 C 轮融资中融资 1400 万美元 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/11/25/kickapps-actually-raises-14-million-in-a-series-b/>

![KickApps](img/4b7b3ca8130bb4439b0d695977824f5e.png)

昨天，有消息透露，白色标签社交网络初创公司 [KickApps](https://web.archive.org/web/20221005152757/http://www.kickapps.com/) 筹集了新一轮资金。但是 PEHub 报道的金额[(“超过 1300 万美元”)，基于一个旧的注册。在证券交易委员会备案，并不准确。也不完全是](https://web.archive.org/web/20221005152757/http://www.pehub.com/24454/vc-deal-scoop-kickapps-cafepress-nabbr-more/)[错](https://web.archive.org/web/20221005152757/http://www.beta.techcrunch.com/2008/11/24/cafepress-raises-another-83-million-from-sequoia/)。

KickApps 首席执行官亚历克斯·布鲁姆(Alex Blum)向我证实，该公司确实通过新的主要投资者北大西洋资本(North Atlantic Capital)筹集了 1400 万美元的 C 轮融资。现有投资者软银、星火棱镜和贾尔·莫恩也参与其中。KickApps 要么与出版商分享在其平台上创建的页面上的广告库存(60%归 KickApps，40%归出版商)，要么让出版商以每分钟 3 至 6 美元的价格买断其广告库存部分。大多数较大的客户选择自己控制广告库存，这表明当社交网络功能被打包到一个较大的网站中时，广告费率不必以美分计算(就像他们在脸书和 MySpace 的许多网站上所做的那样)。

一些建立在 KickApps 上的品牌社交网络的例子包括[三藩市 49 人、](https://web.archive.org/web/20221005152757/http://49ersfaithful.net/)和[绯闻女孩](https://web.archive.org/web/20221005152757/http://cwtv.com/lounge/vip/gossip-girl)、蕾切尔·雷的[不可思议的人](https://web.archive.org/web/20221005152757/http://www.rachaelrayshow.com/incrediblepeople/)、以及 [NPR 的 get-out-of-vote](https://web.archive.org/web/20221005152757/http://www.npr.org/getmyvote/)网站和现已倒闭的 [McCainSpace](https://web.archive.org/web/20221005152757/http://www.beta.techcrunch.com/2008/08/28/close-encounters-of-the-republican-kind-mccainspace-relaunches/) 的粉丝网站。有些持续的时间比其他人长。它 40%的业务现在都在海外，包括吉尼斯世界纪录网站和阿桂婴儿中心。总而言之，KickApps 现在为近 50，000 个社交网络提供支持，据 Blum 称，这些社交网络每月吸引 1，000 万人，产生 3 亿次浏览量。

但布鲁姆认为，基于页面的指标是衡量受众参与度的错误方式。相反，在应用层面衡量人们如何与网站互动更有意义，这也可以在应用作为小工具病毒式传播时在场外衡量。布鲁姆认为:

> 随着 Omniture 变得无关紧要，我们正在开发一种新的测量形式。衡量基于页面的分析是一种商品。它是由谷歌免费提供的。

当然，他会这么说。但他确实有道理。

KickApps 对大型媒体公司尤其有吸引力，因为它允许他们将社交功能放在他们已经控制的网站上。去年，华纳兄弟从宁那里撤出了一批 CW 电视网站(包括《绯闻女孩》、《90210》、《全美超模大赛》和《超人前传》)，并将其转移到 KickApps，正是因为这个原因。它增加了宁的页面浏览量和会员数，而不是它自己的，它不喜欢这样。(**更新** : Ning 首席执行官吉娜·比安奇尼回应道:“这不是我们任何域名映射网络的工作方式。”她表示，浏览量是由网络创建者的网址认证的，会员数据可以导出为 CSV 文件。

KickApps 的另一个吸引力在于它有一个非常简单的 WYSISWYG 应用编辑器，甚至非设计师也可以拖放整个网站或小工具，完成带有广告的。它还支持 OpenSocial 和脸书应用程序，因此也可以添加这些应用程序。
