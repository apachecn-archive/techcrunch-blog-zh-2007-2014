# 孩之宝试图关闭 Scrabulous

[![scrabulous.png](img/7a2ccbb961099905cd9d0bea27d47436.png)](https://web.archive.org/web/20221006232500/https://beta.techcrunch.com/wp-content/uploads/2008/01/scrabulous.png "scrabulous.png") “孩之宝只是个笨土豆头吗？还是这是 Stratego 的一个高明游戏？”这是一个大问题[《财富》杂志的乔希·奎特纳](https://web.archive.org/web/20221006232500/http://techland.blogs.fortune.cnn.com/2008/01/11/will-someone-please-start-a-facebook-group-to-save-scrabulous/)(我的前老板)在报道拥有拼字游戏的玩具公司孩之宝(Hasbro)正试图关闭脸书最受欢迎的应用之一 Scrabulous
时问道。Scrabulous 让你和你的脸书朋友玩在线版的拼字游戏。该应用拥有 56.9 万日活跃用户，排名第九，仅次于排名第八的 Slide 的 SuperPoke，领先于排名第十的 iLike。

Scrabulous 的联合创始人、21 岁的 Jayant Agarwalla 证实，孩之宝“大约两周前给脸书发了一份通知。律师们正在努力。”奎特纳建议成立一个脸书组织来拯救 Scrabulous，但是我们都知道这可能会带来多少好处。
