# 这是谷歌工程师第二次因访问用户数据而被解雇

![](img/f037551282ba3d7e97d6a59656d478af.png)谷歌刚刚[证实](https://web.archive.org/web/20230214085325/https://techcrunch.com/2010/09/14/google-engineer-spying-fired/)其解雇了一名违反内部安全政策的工程师，这有效地证实了 Gawker [的一份报告](https://web.archive.org/web/20230214085325/http://gawker.com/5637234/)详细描述了前雇员 David Barksdale 如何利用他的许可访问多人的私人数据，包括四名未成年人。Barksdale 因该罪行已被解雇，谷歌表示，它“大大增加了我们花在审计日志上的时间，以确保这些控制是有效的”。

我们问谷歌这种情况发生了多少次，并确认这是第二次这样的事件。然而，之前的案件不涉及未成年人——罪犯就像巴克斯代尔一样被解雇了。老实说，鉴于谷歌已经存在了 10 多年，雇佣了大约 20，000 名员工，我原本以为这个数字会更高。尽管如此，听到这一切真的发生了，还是令人不安。

值得称赞的是，谷歌正在对这个问题进行评论。与此形成对比的是脸书——今年早些时候的一篇喧嚣的文章称[至少有两名员工因获取用户数据而被解雇。我反复问脸书这个问题，他们拒绝直接回答这种情况发生了多少次，如果有的话。](https://web.archive.org/web/20230214085325/https://techcrunch.com/2010/01/11/rumpus-facebook-privacy/)
