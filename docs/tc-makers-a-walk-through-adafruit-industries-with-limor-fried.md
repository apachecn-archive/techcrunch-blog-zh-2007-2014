# TC 制造商:与 Limor Fried 一起走过 Adafruit 工业

自从我们创办了 Makers，我就优先考虑去参观 Adafruit Industries，这家位于曼哈顿的电器商店由 Limor Fried 经营。弗里德在麻省理工学院的宿舍里创办了她的公司，为她的朋友们制造很酷的电子设备。她慢慢地扩展到其他爱好领域，包括 Arduino 插件、操作手册和案例。现在，她的商店里有 1302 件商品出售，每天约有 600 份订单。

他们的收入也令人印象深刻:每年 1500 万美元，没有任何形式的风险投资支持。简而言之，油炸又名 Ladyada 是杀死它。

我们有一个独特的机会去参观她在 Soho 的全新曼哈顿仓库/设计中心/工厂，在那里她和她的团队创造了令人惊叹的小发明和小玩意。他们还通过[他们的超级赛道操场](https://web.archive.org/web/20221226063349/http://adafruit.com/circuitplayground)开展学生拓展工作，以便让孩子们相信 STEM 其实很酷。

这次参观，我们看到了 Adafruit 的迷你生产线，他们令人惊叹的本土 CRM 系统，最重要的是，该公司的机器人吉祥物 [Adabot](https://web.archive.org/web/20221226063349/http://www.adafruit.com/blog/2012/09/01/ladyada-with-adabot/) 。
