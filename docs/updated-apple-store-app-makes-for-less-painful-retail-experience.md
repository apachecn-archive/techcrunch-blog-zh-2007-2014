# 更新的苹果商店应用程序减少了痛苦的零售体验

假期很快就要来了，随之而来的是我们国家的零售动脉不可避免的堵塞。多么幸运，苹果刚刚发布了苹果商店应用程序的[更新，允许顾客简化他们的购物体验。](https://web.archive.org/web/20230203065017/http://itunes.apple.com/us/app/apple-store/id375380948?mt=8)

下次你带着这款应用走进苹果商店时，你将能够通过使用这款应用的新功能 EasyPay，绕过所有“人际互动”的废话。拥有 iPhone 4 或 4S 的用户可以扫描许多配件上的条形码，用与他们的 Apple ID 关联的信用卡支付产品费用。收据会生成并存储在应用程序中，所以即使是最挑剔的保安也不应该在人们大摇大摆地出门时做出太多的挣扎。

该应用程序还支持苹果的个人提货功能，用户可以在应用程序内从某个苹果商店订购产品，并在稍后提货。从下订单到取货的周转时间可能只有一个小时，但一旦购物季进入高潮，我们很快就会看到这种说法有多站得住脚。如果你碰巧比某些人更挑剔，想要一种目前没有的产品，你会得到一个大概的估计，知道你要等多久。

并不是所有的新增功能都如此华丽:有一个专门用于跟踪订单的新菜单，加拿大和中国的客户也可以利用这一点。只要你有一台运行 iOS 4.2 或更高版本的设备，你就应该是金色的，但 iPad 用户应该知道，该应用程序并没有为所有额外的屏幕空间进行优化。
