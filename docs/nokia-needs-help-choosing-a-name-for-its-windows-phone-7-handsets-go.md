# 诺基亚需要帮助为它的 Windows Phone 7 手机选择一个名字…开始！

比我们希望的更多的是，手机被冠以绝对恐怖的名字。宏碁的 [beTouch](https://web.archive.org/web/20230205032317/https://techcrunch.com/2009/10/20/acer-announces-betouch-line-and-neotouch-touch-phone/) ，卡西欧 [G'zOne 突击队](https://web.archive.org/web/20230205032317/https://techcrunch.com/2011/04/26/casios-new-gzone-commando-handset-comes-to-verizon/)，以及宏达电 [ChaCha](https://web.archive.org/web/20230205032317/https://techcrunch.com/2011/02/24/chacha-sues-htc-over-htc-chacha/) 都属于这一类。

它们让你想知道在这个名字诞生的会议室里发生了什么，以及批准它们的人心里可能在想些什么。但老实说，要找到一部名字合适、好听、不别扭的手机，比你想象的要难多了。这就是为什么诺基亚让我们来决定。

当在美国谈到诺基亚的 Windows Phone 7 手机时，显然*我们*在名称方面发号施令。该公司已经发布了一项民意调查，有 20 多个不同的名字选项，你也可以提出自己的建议。到目前为止，诺基亚的“精英”、“融合”、“地铁”和“创世纪”领先，“凤凰”领先。

依我拙见，凤凰号只是象征性的，但创世号才是诺基亚真正迫切需要的。该公司一直在努力向 MeeGo 转型，这个梦想在诞生后不久就破灭了。在我们等待 Windows Phone 7 扭转局面的时候，诺基亚的市场份额继续下降。

好消息是 Windows 驱动的诺基亚就要来了。美国发展营销主管 Chanse Arrington 发起的一项民意调查进一步证明了这一点。我们只是希望诺基亚凤凰，或创世纪，或融合，或无论什么…早点到这里。
