周六电梯间推介:用Scour进行社交搜索

今天是电梯推介日周五周六，这意味着另一家初创公司制作了一个值得向你展示的视频。本周的演讲来自Scour.com，一个元社交搜索引擎。

Scour搜索并同时显示来自谷歌、雅虎和Live Search的结果，并使其每个成员能够根据关键词的相关性对每个列表进行投票，并对他们在该网站的体验进行评论。除了综合结果之外，Scour还提供了一种基于点数的付费搜索方案，类似于微软的SearchPerks。用户每次搜索、投票或评论都会获得一分，当他们达到6500分时，可以申请25美元的Visa卡。

成立于2007年，冲刷原名Aftervote.com。差不多整整一年后，Aftervote被Internext media收购，该公司是ABCSearch Network 的所有者和运营商，并更名为Scour.com。
