# 英国广播公司推出 Playlister，这是一项与 Spotify、YouTube 和 Deezer 合作的社交音乐服务，可以标记在 BBC TechCrunch 上听到的歌曲

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/08/bbc-launches-bbc-playlister-a-social-music-service-with-spotify-youtube-and-deezer-to-tag-tracks-heard-on-bbc/>

紧随 BBC [将其 iPlayer](https://web.archive.org/web/20221207114015/http://www.bbc.co.uk/mediacentre/latestnews/2013/dg-iplayer.html) 点播视频和音乐播放器作为其所有数字资产的首选之地，英国公共广播公司也准备将其音乐服务提升一个档次。今天[发布了 Playlister](https://web.archive.org/web/20221207114015/http://www.bbc.co.uk/mediacentre/latestnews/2013/dg-playlister.html) ，这是一项与 Spotify、YouTube 和 Deezer 合作的全球(不仅仅是英国)服务，让人们标记他们在 BBC 上听到的音乐以便以后收听，从中创建播放列表，并通过 BBC DJs 和主持人的推荐潜在地发现更多歌曲。

如果 Playlister 给你敲响了警钟，你不只是听到了什么。这是继一年前[报道](https://web.archive.org/web/20221207114015/http://www.telegraph.co.uk/culture/tvandradio/bbc/9579951/BBC-working-on-music-equivalent-of-iPlayer.html)BBC 正在研发这款产品之后的又一篇报道，该产品在当时被准确定位为 Playlister。相反，去年它为其广播服务发布了 [iOS 和网络应用。尽管如此，它当时还是两面下注。2012 年 10 月，BBC 节目和点播总经理丹尼尔·丹克(Daniel Danker)对 TechCrunch 表示:“从战略上讲，与音乐提供商有着明显的重叠和潜在的联系。“现在没有任何有形的东西，但我们总是有这样的对话。”](https://web.archive.org/web/20221207114015/https://beta.techcrunch.com/2012/10/08/bbc-builds-out-its-iplayer-empire-with-new-ios-and-web-apps-for-radio-service/)

BBC 将于周三举行一场活动，届时我们将有机会与 Playlister 一起玩，并与 BBC 的新服务合作伙伴交谈。

这是 BBC 音乐部门的一项重大举措，也是该公司实现更长期创收战略的重要一步——它必须这样做来补充其许可费，以跟上未来媒体的增长。直接向人们收取内容和广告费用是 BBC 作为公共广播公司在英国传统电视和广播服务中不做的两件事；但当涉及到国际和新媒体资产时，规则就变了。Playlister 几乎肯定会是推动这一想法的服务之一。

随着 Playlister 在世界各地上线，它使该公司的服务获得了更高的国际知名度，并有可能为未来创造收入。该计划是在未来将 Playlister 与 BBC 的广播应用程序集成在一起，但目前它只能通过 PC 和移动浏览器提供，用户可以将 BBC 的播放列表导出到 Spotify、YouTube 或 Deezer，并完整地听回歌曲。

但这仅仅是一个开始:“随着时间的推移，BBC 将寻求欢迎该产品的其他服务，”该公司在一份声明中指出。这将包括推荐功能，甚至可能包括从 BBC 非常丰富的录音室录音和现场录音的付费内容下载，这些内容迄今为止基本上尚未开发，但可能会在许多录制音乐数字流媒体平台的模仿产品之外建立这样的服务。

“BBC Playlister 是 BBC 的一项出色创新，完全是根据观众需求设计的。BBC 总干事 Tony Hall 在一份声明中说:“我们有着令人骄傲的音乐遗产，这可以追溯到 BBC 历史的最开始，多年来，我们已经找到了许多新的方法，将美妙的音乐带给我们的观众和听众。”。“通过与 Spotify、YouTube 和 Deezer 等合作伙伴合作，我们将再次改变观众与音乐和 BBC 的关系。"

但除了这可能是 BBC 赚钱的一种有趣方式这一事实之外，对于消费者来说，Playlister 将是他们可以用来更好地沿着他们今天可以获得的音乐内容的长尾找到自己的方式的又一种方式。

英国广播公司在这方面可能会发挥很大的作用。听众通过电台收听各种精选的音乐，而不仅仅是他们自己收藏的音乐。与此同时，discovery 是迄今为止尚未被任何单一提供商“拥有”的大领域之一。Playlister 将凭借 BBC 的声誉帮助你发现音乐。如果成功的话，这不仅可以让听众更深入地使用 Spotify 和 Deezer(以及 YouTube)这样的网站，还可以让 BBC 重获新生，成为带你去那里的渠道。
