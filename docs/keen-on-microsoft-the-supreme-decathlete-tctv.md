# 热衷于…微软:至高无上的十项全能运动员(TCTV)

罗杰·麦克纳米可能[认为](https://web.archive.org/web/20230203103951/https://techcrunch.com/2011/11/16/keen-on-why-html5-will-blow-up-your-world-tctv/)我们正处于革命之中，但是根据微软首席战略和研究官[克雷格·蒙迪的说法，](https://web.archive.org/web/20230203103951/http://www.crunchbase.com/person/craig-mundie)成为数字时代主导技术公司的竞赛甚至还没有真正开始。我昨天在亚利桑那州图森的[科技经济](https://web.archive.org/web/20230203103951/http://www.techonomy.com/)采访了蒙迪，他认为过去十年实际上是一场仍处于热身阶段的科技竞赛的前奏。当然，蒙迪相信，一旦比赛真正开始，微软很有可能赢得这场比赛。

“这是一项十项全能运动”，Mundie 描述了今天的大赛，让计算机成为反映我们的语言和视觉的感官机器。鉴于微软在整个技术领域毋庸置疑的实力——从个人电脑到平板电脑到云服务器到软件服务再到汽车——芒迪说，他的公司是唯一一个能够在他所谓的“深刻革命”的每个领域进行竞争的公司。

奥林匹克十项全能冠军通常被称为“世界上最伟大的运动员”。无论人们对蒙迪和微软还有什么看法，你都不能指责他们的野心。但是，继史蒂夫·鲍尔默和比尔·盖茨之后，与微软其他人一样有影响力的芒迪是正确的吗？这位来自雷德蒙德的老马还能通过保持/重新获得世界上最伟大的技术公司的头衔而让世界感到惊讶吗？
