# 我们什么时候会有第一次山谷自杀？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/03/02/when-will-we-have-our-first-valleywag-suicide/>

今天，我读到了关于一个著名的科技幻想家和一个似乎在寻求尽可能多的公众关注的女人之间所谓的性接触的所有肮脏的细节。我在哪里读到的？在硅谷八卦博客 Valleywag 上。这些帖子包括私人即时消息聊天和各种谣言，都是为了让提到的人看起来尽可能像个傻瓜。当然，这并不新鲜。Valleywag 正在通过挖掘人们的个人生活并发布给所有人(包括我在内，定期)来赚钱。

我认识的很多人都读过瓦莱瓦格，他们说听到所有的流言蜚语很有趣。但是所有这些人在博客第一次关注他们并把他们包含在谣言中时都会改变他们的论调。一个例子:TED 创始人克里斯·安德森(Chris Anderson)因 TED 与会者名单的公布而感到苦恼，他最近写信给瓦莱威的老板尼克·丹顿(Nick Denton)，称他“没想到(他)会成为瓦莱威八卦的接收端”。他的电子邮件很快被发布到了网站上。

大多数流言蜚语都是无害的。不过，大部分都不是(就像上面的性事件)。几十年来，名人不得不忍受这种胡说八道，这也解释了为什么他们中的一些人完全脱离社会，变得完全反社会。一些人认为，也许他们是通过追求名声来实现这一目标的。

但是对于硅谷的人来说，他们不是名人，除了建立一个伟大的创业公司之外，他们没有任何其他的愿望，瓦列里瓦格上的一篇文章给了他们巨大的冲击。在一个受欢迎的公共网站(由搜索引擎永久索引)上看到你的婚姻困境、酒后驾车或就业终止，简直超出了他们的处理能力。他们没有时间增强对袭击的抵抗能力。

《纽约时报》今天报道了广告主管保罗·蒂利的自杀事件，称这可能与博客上对这个人的严厉批评有关。博客作者，一位匿名女性，不承担任何责任。她说*“我对保罗·蒂利的死感到悲伤，但我不认为我的博客帖子促成了发生的事件”*并吹嘘了事件发生后网站的流量增长。该博客归 JupiterMedia 所有，并将 IAC 和《华尔街日报》视为广告商。

那么，瓦莱瓦格开车送我们社区的人去自杀还要多久呢？我担心的不是它是否会发生，而是何时发生。不过，瓦莱瓦格和尼克·丹顿可能会期待这场盛会，以及随之而来的巨大的交通增长。

显然，这类内容有市场。除了对诽谤法进行重大修改之外，没有什么能阻止它。这不是我所支持的。但是在瓦列伊瓦格时代之前，山谷是一个更好的生活和工作的地方。
