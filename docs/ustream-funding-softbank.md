# Ustream 的大规模融资:现在 2000 万美元，以后 5500 万美元。也许更多。

流媒体视频网站 [Ustream](https://web.archive.org/web/20221003201334/http://ustream.tv/) 刚刚获得新一轮大规模融资:7500 万美元。第二轮由日本电信巨头[软银](https://web.archive.org/web/20221003201334/http://mb.softbank.jp/en/)领导。此前，该网站已经从 2008 年的首轮融资和 2007 年末的天使轮融资中筹集了不到 1300 万美元的资金。

**更新**:虽然 Ustream 提到了 7500 万美元的数字，软银已经[澄清](https://web.archive.org/web/20221003201334/http://www.softbank.co.jp/en/news/press/2010/20100202_03/index.html)他们现在投资 2000 万美元购买该公司 13.7%的股份，并有*选择权*到 2011 年再投资 5500 万美元——这将使他们成为 Ustream 的最大股东，拥有超过 30%的流通股。

也许更疯狂的是，该服务表示，美国和亚洲的其他投资者正在等待额外的资金承诺，所以这一轮实际上可能超过 7，500 万美元。我们听到报道说，这一轮竞争相当激烈，显然所有的尘埃还没有落定。

那么 Ustream 究竟为什么需要 7500 万美元+？首席执行官约翰·汉姆在新闻稿中说，这笔钱将用于在世界的另一端扩张，特别是日本(很明显，考虑到软银的参与)、中国、韩国和印度。移动视频在一些亚洲国家尤其火爆，这些国家的无线网络速度比美国相对较慢的网络速度更快，可以提供更多功能。显然，Ustream 将在所有这些国家开设办事处并雇佣员工。

除了新一轮融资，我们还听说公司创始人 Ham、Brad Hunstable 和 Gyula Feher 能够出售一些股份作为奖励。

就在几天前，[Stream 推出了一个新的桌面客户端](https://web.archive.org/web/20221003201334/http://www.beta.techcrunch.com/2010/01/28/ustream-producer-desktop/)来帮助视频制作人给他们的作品一种更专业的感觉。在此之前，12 月，美国梦成为头条新闻，成为第一个在苹果手机 T3 上提供录音功能的大型视频流网站。

美国在线表示，其苹果手机应用程序迄今已被下载超过 150 万次(在直播应用程序可用之前，它也有其他应用程序)，并指出有 380 万人收看了去年 1 月奥巴马总统的就职典礼。

**更新**:软银的投资似乎让美国贸易代表团目前的价值约为 1.5 亿美元。然而，如果他们行使他们的选择权，他们对美国贸易代表团的全部投资将使其价值达到 2.3 亿美元左右。
