# 微软与纬创签订安卓和 Chrome 产品专利协议

[![](img/4f500b13f616ebfd31915a96b093ca80.png "service_2_02")](https://web.archive.org/web/20221207050337/https://beta.techcrunch.com/files/2011/07/service_2_02.jpg) 暗示女王。每个人都喜欢的 Android 相关专利所有者微软刚刚与 ODM 纬创资通公司签署了一项协议，涵盖其 Android 和 Chrome OS 系列产品。当然，和其他类似的协议一样，细节没有公布。

纬创是一家原创设计制造商，为其他公司设计和制造产品。这笔交易可能是微软利润更丰厚的协议之一，因为纬创资通就像著名的富士康一样，生产大量未公布的产品。然后，品牌和原始设备制造商购买未贴标签的产品，并在底部贴上他们的商标。纬创和其他 ODM 让电子世界不停转动。

微软的[记者](https://web.archive.org/web/20221207050337/http://www.prnewswire.com/news-releases/microsoft-and-wistron-sign-patent-agreement-125003559.html)注意到，这笔交易涵盖了纬创的 Android 和 Chrome 产品。“纬创的平板电脑、手机、电子阅读器和其他运行 Android 或 Chrome 平台的消费设备，”它说。微软已经成功地向安卓制造商施压。Chromebook 制造商可能是下一个。
