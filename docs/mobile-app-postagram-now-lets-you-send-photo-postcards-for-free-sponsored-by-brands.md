# 手机应用 Postagram 现在可以让你免费发送照片明信片，由 Brands TechCrunch 赞助

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/03/mobile-app-postagram-now-lets-you-send-photo-postcards-for-free-sponsored-by-brands/>

[Postagram](https://web.archive.org/web/20221209174346/http://postagramapp.com/) ，这款来自[的手机应用真诚地](https://web.archive.org/web/20221209174346/http://sincerely.com/)将你的照片转换成你可以用手机邮寄的明信片，刚刚接受了一年多来的首次大更新，最显著的变化是增加了免费明信片。此前，用户在美国邮寄这些个性化卡片需要支付 99 美分，在全球范围内支付 1.99 美元，包括邮费在内。但现在该公司推出了一套免费明信片，其费用将通过广告支付。

该交易的前两个合作伙伴是 Coppertone 和 Hyatt House，这两家公司将向真诚的用户群提供数十万张免费卡。这些公司将这笔交易视为通过直接邮寄将他们的品牌展现在消费者面前的一种方式，只是这一次，他们的广告不是看到他们的信息立即被丢弃在垃圾桶里，而是直接放在个人照片和家人或朋友的问候旁边。

这相当于今天品牌在 Instagram 或脸书等流行的照片共享网络上所做的线下工作——吸收人们喜欢观看的用户生成的内容，以便将自己与这些内容引起的同样温暖的感觉联系起来。除了 Postagram 以前进行过的一次性试验和试验，如 Bing 在 Sundance 赞助的明信片，或去年秋天与 Gap 进行的测试，例如，每月上限约为 20，000 张明信片，直邮公司以前从未有机会对用户的照片进行这样的处理。

但这一次，该计划是继续在 Postagram 提供免费的赞助明信片，即使前两个品牌在提供的卡片数量方面达到了合同要求。

“当你收到一封邮件时，这是一件多么令人高兴的事情。我们希望每个人都能体验到这一点，”首席执行官 Matt Brezina 真诚地解释道。“这些品牌表示，他们希望成为现实世界中情感联系的一部分，”他补充道。“这是一个他们已经习惯的世界——直邮是一个 140 亿美元的业务，所以他们可以把它放在那个桶里。”

在凯悦酒店的案例中，该公司正在全国 60 个地方的 10，000 个酒店房间中放置印刷的 Postagram，这将引导用户下载 post agram 移动应用程序。

Postagram 卡片的好处在于，尽管这些新卡片可能是赞助的，但收件人仍然可以像往常一样弹出照片，以便在扔掉卡片之前保存照片。对于那些*真的*不想让自己的卡被广告拖累的人来说，仍然可以选择支付 Postagram 的传统明信片。

布雷兹纳说，他有信心随着他们的前进，真诚可以继续用更多的明信片赞助商填充管道，但目前该应用程序一次只能支持一张广告支持的卡片。虽然交易条款不能透露，但每张卡产生的收入大致相同，除了现在真诚还将节省支付处理费。

此外，由于这家初创公司正在与知名公司建立关系，它可能会进一步发展交易，为其其他应用程序带来类似的选择，包括贺卡应用程序[真诚墨水](https://web.archive.org/web/20221209174346/https://sincerely.com/ink)和移动礼品服务[芝麻](https://web.archive.org/web/20221209174346/https://sincerely.com/sesame)。“这些对话是绝对发生的，”布雷兹纳证实。

免费明信片首先在 iOS 上推出，并计划在不久的将来支持 Android。随后，免费明信片选项也将提供给真诚的 API 合作伙伴，他们的 200 多个 iOS 照片应用程序总共拥有约 5000 万消费者。

更新后的 Postagram 应用程序现在在 iTunes 上[直播。该应用还包括](https://web.archive.org/web/20221209174346/https://itunes.apple.com/us/app/postagram/id429264904?mt=8&ls=1)[真诚的新“体贴引擎，](https://web.archive.org/web/20221209174346/https://beta.techcrunch.com/2013/06/05/sincerelys-new-thoughtfulness-engine-will-help-you-remember-birthdays-special-occasions-wont-spam-you-with-alerts/)”，这是一个最近添加的功能，有助于提醒用户即将到来的生日和其他特殊场合。
