# Kickstarter 项目赋予学生权利，用等离子体玩马里奥主题

这个很酷的 Kickstarter 允许你使用等离子能量来建造一个工作的扬声器。这有点深奥，但事情是这样的:这是一个小套件，包含使用等离子弧演奏音乐所需的一切。如果这看起来不像是你的好时光，你可能读错了文章。

也就是说，它是这样工作的:花 40 美元你可以得到使电弧振动的部件，花 80 美元你可以得到所有部件，包括电源和回扫变压器。这个工具包是由西雅图的[学生和](https://web.archive.org/web/20230204223605/http://facebook.com/StudentRND)一群酷孩子设计和建造的，他们和志愿者一起建造了一些疯狂的科技产品。这看起来是他们的第一个 Kickstarter 项目之一，他们要求 2000 美元来建立一些工具包和支持他们的任务。他们超过了他们的最小值，所以这个项目将得到资助，你将最终能够使用一个微小的等离子弧播放*在水面上冒烟*。

即使你不赞同在空中发射电流，StudentRND 看起来是一个伟大的组织，他们显然对科学很感兴趣，这是我绝对可以支持的。
