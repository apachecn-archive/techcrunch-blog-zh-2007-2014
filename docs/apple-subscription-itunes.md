# 尽管遭到酒吧的反对，苹果将强制 iTunes 订阅付费

![](img/14101c35647bb7e78864abf0334baefc.png)

当鲁珀特·默多克的*日报*上周推出时，苹果公司副总裁艾迪·库伊上台宣布[通过 iTunes 一键订阅 iPad 出版物](https://web.archive.org/web/20221231004925/https://techcrunch.com/2011/02/02/one-click-subscriptions-come-to-the-ipad/)。日报已经有了一键付费的功能，Cue 承诺它将很快提供给其他 iPad 报纸和杂志。Cue 随后开始走访纽约的印刷媒体公司，向他们解释订阅计费将如何在 iPad 上运行。

谁将控制订户收入和数据一直是印刷出版商和苹果之间的争论点。杂志和报纸公司希望能够通过将读者发送到他们自己的网站来处理 iPad 应用程序订阅来控制订阅账单，但苹果公司坚持要求他们使用 iTunes。为了打破这个僵局，我听说苹果公司提出了一个妥协方案。出版物可能被允许将订阅者引导到他们自己的网站或支付门户，以获取他们的信用卡号码，但他们也必须提供通过 iTunes 订阅的选项。

这是一个绝妙的对策。苹果基本上是在说，“让用户自己决定”，明知他们会选择 iTunes。毕竟，对消费者来说，从一个地方订阅数字出版物更容易。然而，已经有一些报纸公开抱怨被迫使用 iTunes。除了*日报*，它首先推出订阅，并且只通过 iTunes 提供(相比之下，新闻集团的 *WSJ* 应用程序会带你去一个网站处理订阅)。显然，苹果希望其他印刷媒体行业效仿《每日新闻》——无论是通过 iTunes 收费还是每周 99 美分的定价。(你不得不怀疑史蒂夫·乔布斯是否参与了说服默多克选择这个价格的行动。但如果其他媒体公司坚持自己完成订阅，他们将不得不提供 iTunes 订阅作为替代。

印刷媒体公司想要控制订阅有几个原因。他们不喜欢把每个用户 30%的钱交给苹果，但更重要的是，他们不能失去与读者直接付费的数据。如果不是在持续的基础上，他们可以分享收入。毕竟，杂志和报纸已经和报摊分享封面价格了。iTunes 就是新的数字报摊。不同的是，一次性报刊亭购买者有时会转化为订户。我们大多数人都知道，一旦你订阅了杂志或报纸，惯性会让你订阅很多年。

一旦你成为订户，印刷媒体公司就会知道你的很多信息:你住在哪里，你的名字和信用卡号。这些细节足以让他们的数据库营销人员创建相当准确的读者人口统计资料，并将这些有针对性的俘虏观众卖给广告商。另一方面，通过 iTunes 销售只是一个黑箱。虽然，作为注册应用程序的一部分，出版物可以收集一些相同的信息，如姓名、电子邮件和邮政编码。他们真正担心的是 iTunes 会让取消不想要的订阅变得更容易。
