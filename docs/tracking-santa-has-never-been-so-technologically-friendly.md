# 追踪圣诞老人从未如此科技友好 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/12/04/tracking-santa-has-never-been-so-technologically-friendly/>

传统是美好的东西。随着世界的加速，数字化，并最终空中传播，某些传统让我们这些年来一直脚踏实地。其中一个传统是用北美防空司令部的圣诞老人追踪器追踪全球的圣诞老人。

今年，圣诞老人追踪器有了很大的改进([主要归功于微软](https://web.archive.org/web/20230203145207/http://blogs.technet.com/b/microsoft_blog/archive/2013/12/03/norad-tracks-santa-project-goes-3d-touch-device-optimized-with-some-help-from-microsoft.aspx))，但没有失去我们已经习惯的传统圣诞欢呼。

首先，雷德蒙为克里斯·克林果创造了一个 3D 粘土动画世界，包括北极的圣诞老人村。

粘土动画与节日精神完美契合，因为许多最好的圣诞经典电影都是用粘土动画制作的，如*鲁道夫、红鼻子驯鹿*和*雪人*。

除了粘土动画之外，微软还将圣诞老人追踪器变成了一种触摸友好的体验，将 JavaScript 驱动的网站迁移到手机、平板电脑和网络上。

从现在开始，用户可以查看圣诞老人的 3D 粘土村庄，玩每天都会发布的游戏，以及查看音乐和其他内容。

然后，在 12 月 24 日，用户将能够跟随圣诞老人在全球各地递送圣诞礼物。

此外，北美防空司令部在圣诞老人追踪动画视频中加入了一些战斗机[。你知道，为了保持现实。](https://web.archive.org/web/20230203145207/http://www.reuters.com/article/2013/12/03/us-usa-christmas-fighters-idUSBRE9B20V820131203)

这是一个升级 50 多年传统的非常可爱的方式，考虑到现在的孩子更喜欢 iPads 而不是杂志，微软肯定已经帮助一代父母与他们自己的下一代孩子分享他们童年的传统。

大家圣诞快乐。
