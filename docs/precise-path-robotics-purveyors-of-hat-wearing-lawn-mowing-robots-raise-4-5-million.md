上述视频揭示了[精确路径机器人](https://web.archive.org/web/20221004031309/http://www.precisepath.com/)刚刚获得 450 万美元资金的确切原因。一旦你看过 [RG3 机器人果岭割草机](https://web.archive.org/web/20221004031309/http://www.precisepath.com/)，或者我喜欢称之为“戴帽子的高尔夫球场 [Roomba](https://web.archive.org/web/20221004031309/http://www.youtube.com/watch?v=GTxW3GWZ5hI) ，很难想象高尔夫球场会以其他方式割草。

你知道世界上有 32，000 个高尔夫球场吗？我也没有，但 RG3 是最快速和最时尚的方式来大幅削减每一个人花在割草上的钱。这相当于每年价值 20 亿美元的割草！世界上第一台机器人割草机的严肃生意。

![](img/5818891038e1a4c77adacb91dd058c12.png)

但是为什么要止步于割草呢？RG3 技术实际上优于 Roomba，因为它可以直线行驶，几乎可以应用于任何户外美容活动。

Precise Path 目前正处于 C 轮融资的尾声，总计 820 万美元。天使投资者包括查理·斯台普斯和查查联合创始人[斯科特·琼斯。](https://web.archive.org/web/20221004031309/http://www.crunchbase.com/person/scott-jones) 
精确路径公司总裁兼首席运营官杰森·齐尔克如是说。*“割草是我们设想的一长串应用中的第一项——从喷洒到除雪，再到在草坪上雕刻广告标识。”*

有一种东西是机器人的完美工作。
