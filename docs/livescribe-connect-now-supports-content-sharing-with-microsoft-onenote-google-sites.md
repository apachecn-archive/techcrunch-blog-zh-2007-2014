# Livescribe Connect 现在支持与 Microsoft OneNote、Google Sites 共享内容

Livescribe 今天向新闻界宣布了其 Livescribe Connect 服务的两大新增功能。Livescribe Connect for Echo 和 Pulse 智能笔的用户现在可以与 Microsoft OneNote 和 Google 网站共享内容。使用该公司的智能笔，任何用墨水写在特殊纸上的东西都可以通过各种社交和生产力服务在线共享。

Livescribe Connect 在 5 月份推出了，这是向数字笔的相关性迈出的一大步，因为它使分享内容变得更加容易。这两项新服务加入了 Twitter、脸书、Evernote、Google Docs 和 iOS 设备已经支持的平台，表明该公司致力于为该平台带来新的功能集。

正如约翰[在 5 月](https://web.archive.org/web/20230203102446/https://techcrunch.com/2011/05/23/livescribe-releases-connect-puts-ink-in-the-cloud/?replytocom=1785458)解释的那样，用户所要做的就是写下任何东西，然后画一条下划线，后面跟着一个类似“推特”或电子邮件地址的命令。然后只需在便笺簿上选择您想要共享的内容，一旦笔通过 USB 与计算机同步，预先选择的内容就会自动发送到所需的服务或地址。

OneNote 和谷歌网站非常适合 Connect 的用户群。Livescribe Connect 允许用户将录音直接发送到 OneNote，OneNote 是微软为这些内容开发的一种数字笔记本。Google Sites 是一个协作构建工具，能够添加来自其他 Google 服务和现在的 Livescribe Connect 的内容。

Livescribe Connect 似乎仍然受到智能笔共享物理链接要求的限制。这是一个新颖的想法，最终允许共享书面内容，但无线链接，甚至蓝牙，将随着功能的增加而扩大其吸引力。
