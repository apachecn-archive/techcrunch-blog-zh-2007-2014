# 微软发布。NET 小工具工具包

微软宣布了。NET Gadgeteer，一个类似于 T2 的小工具包，允许黑客使用一组预组装的部件来构建独特的硬件和软件解决方案。

kid 包括一个硬件编程框架，可以与微软自己的 Visual Studio 一起工作。例如，您可以将相机部件、热传感器和屏幕添加到您的软件项目中，从 PC 对其进行编程，然后在研究和实验应用中使用最终的设备。

该平台表面上是开放的，任何硬件制造商都可以为该框架设计硬件。例如，你可以在 9 月底花 249.95 美元从 [GHI Electronics](https://web.archive.org/web/20230203061514/http://www.ghielectronics.com/catalog/category/265/) 购买一个入门套件。它包括一个摄像头模块、一个 USB 主机、一个小屏幕和一些其他设备，包括一个 SD 读卡器。

你可以从这里下载软件[开始。](https://web.archive.org/web/20230203061514/http://netmf.com/gadgeteer/)

由于 Arduino 控制了快速电子原型制作的思想份额，微软进入这一领域是有意义的。我想说，对于大多数寻找设备框架的人来说，主要关心的是与现有系统和 Windows 的兼容性，而 Visual Studio 对大多数商业实体来说是有意义的。
