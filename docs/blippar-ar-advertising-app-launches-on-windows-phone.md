# Blippar AR 广告应用在 Windows Phone 上发布

Blippar 自 2011 年成立以来，唯一的目的就是让广告再次充满乐趣。

该应用使用[高速增强现实和图像识别技术](https://web.archive.org/web/20221210032315/https://beta.techcrunch.com/2013/09/22/blippar-ar-advertising-app-launches-social-sharing-improves-image-recognition-time/)，让品牌在现实世界的物体上添加互动广告。

据该公司称，Blippar 已经可以在许多平台上使用，包括 iOS、Android 和黑莓，今天将在 Windows Phone 平台上推出，成为第一个也是唯一一个可以在所有平台上使用的图像识别应用。

该公司最初成立于英国，后来[转移到了美国](https://web.archive.org/web/20221210032315/https://beta.techcrunch.com/2013/07/21/blippar-ceo-celebrates-one-year-in-the-u-s-with-ar-advertising-demo/)。Blippar 迄今已有超过 400 万次下载，该平台上有超过 10，000 个 blips，由 750 个品牌和出版商提供。

当然，很难相信一个广告应用程序会在消费者方面获得巨大的吸引力，但 Blippar 似乎证明了这一点。用户点击产品和图片超过 2500 万次。

除了英国和美国，Blippar 还在西班牙、土耳其、日本和印度推出了产品。
