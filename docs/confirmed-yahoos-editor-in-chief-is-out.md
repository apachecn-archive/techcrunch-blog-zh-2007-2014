# 证实:雅虎的总编辑不在

从明天开始，雅虎的总编辑 Jai Singh 将离开公司。继[早前媒体报道](https://web.archive.org/web/20221007114638/http://recode.net/2014/01/16/yahoos-editor-in-chief-jai-singh-departs-company/)他离职后，雅虎发言人向 TechCrunch 证实了这一消息。

雅虎目前正经历一段高管动荡时期，昨天解雇了首席运营官，今天的新闻紧随其后。

虽然[解雇 Henrique de Castro](https://web.archive.org/web/20221007114638/https://beta.techcrunch.com/2014/01/15/henrique-de-castro-leaves-yahoo/) 带来了巨大的财务潜力，但目前还不清楚辛格的离开会给公司带来多大的财务损失。当然，在你加强内容战略的时候，失去你的编辑领导很难说是一个好兆头，尤其是当他从公司的自我回避最初被描述为:辞职。

雅虎斥巨资聘请大众市场人才，如科技记者大卫·波格和媒体名人凯蒂·柯丽克。也许这些雇佣并不适合辛格。更有可能的是，梅耶尔领导下的雅虎的新方向——辛格是在梅耶尔成为公司首席执行官之前被聘用的——不适合辛格，或者他是内部多余的。雅虎拒绝就离职的性质提供更多指导。

雅虎是一家再造的公司。在梅耶尔的指导下，它重新调整了产品愿景，快速购买人才，并扩大了原生内容的努力。如果有新的资金流向内容之类的东西，辛格本可以在公司内部享有更高的地位。
