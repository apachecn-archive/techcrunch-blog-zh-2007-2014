# 推特的结局:首次公开募股、象棋还是轮盘赌？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/09/twitter-endgame-ipo-chess-roulette/>

***编辑注:**客座撰稿人[semi Shah](https://web.archive.org/web/20230205042942/http://www.semilshah.com/)是一位对数字媒体、消费互联网和社交网络感兴趣的企业家。他住在帕洛阿尔托，你可以在推特[@ semi shah](https://web.archive.org/web/20230205042942/http://www.twitter.com/semilshah)上关注他。*

有报道称，推特正在与 T10 谈判筹集 400 万美元，使该公司市值接近 80 亿美元。似乎就在昨天，推特从凯鹏华盈那里筹集了 2 亿美元。在凯鹏华盈投资的时候，一些权威人士窃笑不已，声称这家老牌风险投资公司为了在社交网络中获得一点股权而付出了代价，而他们被认为错过了这一点。

尽管笑声不断，但可能是[约翰·杜尔登](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/person/john-doerr)、他的合作伙伴、其他推特[投资者](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/company/twitter)、以及他们的员工分享了最后的笑声。除了他们的 T21 利基基金的成功之外，多尔在去年 12 月敲定这笔交易时，也清楚自己在下什么注。在购买推特的过程中，人们可能会把这些点联系起来，因为多尔对谷歌有着深刻的了解，他可能知道一些让他的投资流过的推特终端的事情。无数博客试图表明谷歌应该购买推特。与此同时，没有任何社交数据的其他科技巨头也可能将推特视为一个有吸引力的收购目标，给他们一个大网络，并帮助他们绕过许多充满社交、用户生成内容的围墙花园网站。该公司的大股东能在一个非常小的市场上为推特交易创造竞争吗？

Twitter 投资者的名单令人印象深刻，其中散落着蓝筹风险投资公司和消费者网络名人的名字，他们中的许多人主要通过这一项投资建立了自己的声誉——这是理所当然的。Twitter 有大约 400 名员工，其中一半专注于工程和产品，不一定需要近 5 亿美元的额外融资才能继续运营。迄今为止，它已经能够在很大程度上基于推广的 tweet 广告单元获得一些收入。

然而，它也一直在收购公司，以帮助消除其技术和产品供应中的盲点，迄今为止已花费超过 5000 万美元购买了 [Fluther](https://web.archive.org/web/20230205042942/https://techcrunch.com/2010/12/21/fluther-twitter/) 、 [TweetDeck](https://web.archive.org/web/20230205042942/https://techcrunch.com/2011/05/23/twitter-buys-tweetdeck-for-40-million/) 以及最近的 [BackType](https://web.archive.org/web/20230205042942/https://techcrunch.com/2011/07/05/twitter-acquires-social-analytics-startup-backtype/) 。人们可能会认为，随着公司继续让产品回到正轨，内部管理和新收购的结合有助于让产品达到稳定。这似乎是正在发生的事情，尤其是在 c0 创始人[杰克·多西](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/person/jack-dorsey)的产品指导下，他回来了，但[仍然是](https://web.archive.org/web/20230205042942/https://techcrunch.com/2011/03/03/jack-dorsey-twitter-punched-stomach/) [Square](https://web.archive.org/web/20230205042942/https://squareup.com/) 的首席执行官，也是前谷歌站长[萨提亚·帕特尔](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/person/satya-patel)。

那是因为现在，Twitter 产品正努力保持稳定。对于那些在浏览器中使用 Twitter 原生网站的人来说，速度要慢得多，可能是因为系统每天处理超过 2 亿条推文。当然，这是一个很大的数目，但在这 2 亿个项目中有很多嘈杂的、垃圾的、听不见的(或不真实的)账户。对于那些试图获得直接消息(“DMs”)的人来说，当他们最终加载时，随机的旧线程会不断弹出，这意味着他们可能会将所有的 DMs 移动到新的服务器上。@回复仍然无法与@提及区分开来，用户没有被教育去列出清单，并且很难通过收藏夹或旧推文进行搜索。与网络相比，通过手机发布的每条推文也有更多的附加选项，这可能是因为对于那些频繁使用 Twitter 的人来说，有太多不同的客户端可以访问这条推文。

这是巨大的挑战，但也是 Twitter 的巨大机遇。关于 Twitter 对其开发者社区的处理，有很多争议。对一些人来说，缺乏清晰的路线图已经吓退了从[投入时间、金钱和开发者资源的人。对其他人来说，谈论是廉价的，Twitter 最终将不得不拥有所有资产的现实是显而易见的，所以那些加入战斗的人必须心甘情愿地这样做。](https://web.archive.org/web/20230205042942/http://cdixon.org/2011/02/21/the-importance-of-predictability-for-platform-developers/)

最近，当 Twitter 被看到[攻击](https://web.archive.org/web/20230205042942/http://www.observer.com/2011/tech/twos-crowd-twitter-blocks-ubermedia-tweetdeck-deal-looms)优步媒体，然后[随后](https://web.archive.org/web/20230205042942/https://techcrunch.com/2011/05/02/twitter-to-buy-tweetdeck-for-40-million-50-million/)收购 TweetDeck，或者当 Twitter [决定](https://web.archive.org/web/20230205042942/https://techcrunch.com/2011/05/30/twitter-is-launching-its-own-photosharing-service/)与 [PhotoBucket](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/company/ontela) 合作推出自己的照片分享服务(从而绕过了像 [Twitpic](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/company/twitpic) 和 [yfrog](https://web.archive.org/web/20230205042942/http://www.crunchbase.com/product/yfrog) 这样的服务)，或者当它[决定](https://web.archive.org/web/20230205042942/https://techcrunch.com/2010/04/14/twitter-confirms-it-will-launch-its-own-link-shortener/)进入 URL 缩短服务时，这些矛盾就暴露出来了尽管 Twitter 在现实世界中非常强大，极具破坏性，但毫无疑问，在它自己的生态系统中，类似的力量也在发挥作用。

如今，科技世界迎来了“泡沫”横行的好年景，再加上如今火热的 IPO 市场，Twitter 需要不断推进自己的棋子。虽然公众对 Twitter 股票的需求可能很大，但现实是这家成立五年的公司不仅没有实现使其成为可行选择所需的收入加速，而且其产品也没有完全定型。这一轮最新的 4 亿美元融资可能是一种尝试，旨在帮助推动更多增强产品的收购，并从整体上开始整合分散的客户和服务，这些客户和服务目前仅依靠 Twitter 的 API 供氧生存。随着每一轮融资的增加，随着其估值的不断上升，可能的退出结果越来越少，赌注越来越高。Twitter 的残局从国际象棋变成了危险的轮盘赌游戏。

从我个人作为 Twitter 用户的角度来看，我真诚地希望 Twitter 仍然是一家独立的公司，最终上市。我会排队买股票。我通过 Twitter 浏览网页，花在这项服务上的时间比其他任何服务都多。虽然我希望它上市，但我有一种不现实的不祥感觉。这并不是说 Twitter 在寻找和调整所有最佳商业模式方面失败了。他们只用 400 人就做了一项令人难以置信的工作。真的很令人惊讶。相反，所有潜在机会的范围是如此巨大、广泛和深入——从互联网到电视，到后端分析、搜索、位置——以至于可能需要真正的合作才能将这些模式转化为真正的收入引擎。

如果 Twitter 继续保持私有和封闭，谁还会对它感兴趣呢？有些人认为苹果公司(T0)和苹果公司(T1)是有道理的，尤其是在苹果公司(T2)将 Twitter 整合到 iOS5 软件之后。像 Twitter 这样的公司可以给现金充裕的苹果一个巨大的网络来[在多种设备上部署](https://web.archive.org/web/20230205042942/https://techcrunch.com/2010/04/08/apple-announces-iad-mobile-advertising-platform/)其 iAd 广告，也许最终甚至在电视上。今年早些时候，有传言说脸书可能感兴趣，但他们手头没有足够的现金来做这件事(尽管脸书的股票可以做到这一点)。总会有微软，它手头当然有现金，也有购买社交通讯服务的欲望，最近以 80 亿美元收购 Skype 就证明了这一点。

这就剩下一家位于山景城的公司，许多人认为它可能是 Twitter 的完美收购者，尽管事实上它刚刚推出了自己的社交网络、 [Google+](https://web.archive.org/web/20230205042942/http://plus.google.com/) ，它也准备推出品牌页面。一个可能的谷歌-Twitter 收购案已经被分析得很透彻了，所以我不会在这里说，除了在纸上说，这很有意义。因为 Twitter 在世界各地播下了丰富的种子，最有可能收获这些种子(并在这样做的同时获得可观利润)的公司是谷歌，别无其他。

但是，随着 Twitter 以更高的估值筹集更多资金，如果 IPO 在近期不太可能，可能的结果就会减少。虽然从金额和媒体关注度来看，上述四家公司中的一家进行的收购可能是巨大的，但真正的游戏将是在四家潜在竞购者中的至少两家之间创造足够的竞争，以推高价格，实现股东回报最大化。如果 Twitter 使用新的资金继续巩固产品，并继续寻找最佳商业模式，它可能会给潜在的收购者提供信息，使他们认为他们可能也不会从 Twitter 获得利润。

通过筹集巨额资金，Twitter 为自己赢得了时间，但也加大了风险。Twitter 会在接下来的几年里继续前行，并偶然发现一种商业模式，使其能够上市吗？像苹果、微软、脸书或谷歌这样的巨头会收购这家公司，或者更好的是，互相下象棋争夺以接近 100 亿美元或更多的价格收购它的权利，为公司股东赚钱吗？或者，Twitter 是否开始慢慢消失，无法获得足够的现金来维持 Twitter 机器的运转？第一个选项是一个长镜头，我们在轮盘赌上只剩下两个数字。我的钱和希望都押在了黑色上:收购。
