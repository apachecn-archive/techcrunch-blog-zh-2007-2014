# Zynga 的独立游戏网络“Z 计划”今天宣布

我们今天早上在旧金山参加 Zynga Unleashed 活动，我们听说他们将推出的一款产品是[“Zynga Live”](https://web.archive.org/web/20230214224000/https://techcrunch.com/2010/05/07/zynga-gunning-up-and-lawyering-up-for-war-against-facebook-with-zynga-live/)——他们期待已久的平台，人们可以在这里玩游戏，而不需要通过脸书或其他社交网络合作伙伴。从那以后名字可能已经被改变了，但是概念，一个独立的游戏网络，仍然是相同的。

细节仍然很少，但我假设这和 Farmville.com 之类的东西之间的区别是 Zynga Live 将允许访问多个游戏标题，尽管目前还不清楚[围绕 Zynga 的各个标题的排他性安排](https://web.archive.org/web/20230214224000/http://allthingsd.com/20110718/zynga-and-facebook-exclusivity-goes-far-beyond-credits/)。

我们在 2010 年写过关于 Zynga Live 的[，当时 Zynga 与脸书的关系几乎达到沸点，原因是脸书要求将 Zynga 脸书游戏收入的 30%分成。](https://web.archive.org/web/20230214224000/https://techcrunch.com/2010/05/07/zynga-gunning-up-and-lawyering-up-for-war-against-facebook-with-zynga-live/)

问题是 Zynga 的游戏网络将在多大程度上依赖脸书的社交网络。我相信我们会在活动中听到更多，敬请关注。

**更新:** Zynga 首席执行官马克·平卡斯[证实。它被称为 Project Z，是 Zynga“直接面向消费者”战略的一部分，也就是说，它是该公司第一款直接面向消费者的产品。据我所知，这将是一个独立于脸书的网站，但你仍然需要一个脸书账号才能玩游戏。Zynga 只是透露，它将利用脸书连接，并将具有互操作性，这样你就可以在脸书上开始游戏，并在其他地方的同一点上继续游戏。](https://web.archive.org/web/20230214224000/http://twitter.com/#!/zynga/status/123816767251427328)

虽然这项服务还不能在浏览器上使用，但 Zynga 确实允许玩家在 Zynga.com 的所有 Zynga 游戏中保留他们的 Project Z[“Z tags，](https://web.archive.org/web/20230214224000/http://ztag.zynga.com/)”或他们的通用头像。
