白宫宣布全国公民黑客日，要求美国人解决来自美国宇航局等政府数据的问题

白宫希望你为一个更好的美国而奋斗。今天宣布6月1日至2日为全国公民黑客日，届时许多政府机构将向全美公民开放数据，用于建设有助于他们社区的技术。27个城市已经计划了一些活动，在这些活动中，黑客可以访问来自劳工部、人口普查局，甚至美国国家航空航天局的空间统计数据。

公民黑客国家日是由美国政府、 Code For America 、Random Hacks Of kind、和Eric Schmidt的早期风险基金Innovation efforts联合发起的项目。作为不断增长的提高政府透明度和公民参与度的举措的一部分，该组织解释说:

该活动将汇集来自全国各地的公民、软件开发商和企业家，利用公开发布的数据、代码和技术，合作创建、构建和发明新的解决方案，以解决与我们的社区、城市、州和国家相关的挑战。全国公民黑客日将为公民提供一个机会去做最典型的美国事情:卷起我们的袖子，参与进来，共同努力改善我们的社会。

参与者可以创建可视化政府数据集的应用程序，或者跨机构整合这些数据集。例如，可以将人口普查和劳工部的数据结合起来，研究什么样的人口统计和行业因素可以降低失业率。

或者，尽管在过去几年中大量政府数据已经“公开可用”，但其中许多数据并不是真正可访问的，因为它缺乏强大的API或用户界面。黑客可以构建这些来为未来的公共数据创新奠定基础。

创新努力和T3】mypc backup的管理合伙人Dror Berman告诉我，同时举行的全国性活动“是美国的合作聚会，以技术为核心。”有几种方法可以让你加入这个公民参与聚会(它会比听起来更有趣)。

你可以在你所在的城市策划一场活动或者通过成为东道主来支持当地的活动。例如，看看去年在帕洛阿尔托举行的创新努力“超级快乐街区派对”。

你可以为你控制的数据提供一个API，提供你写的代码，或者提出一个人们应该解决的问题。

或者，你可以报名参加当地的活动然后出来黑。白宫希望你们“卷起袖子，参与进来，进行公民黑客攻击……围绕着应对和解决与我们的街区、我们的社区、我们的城市、我们的州和我们的国家相关的挑战的共同使命。”
