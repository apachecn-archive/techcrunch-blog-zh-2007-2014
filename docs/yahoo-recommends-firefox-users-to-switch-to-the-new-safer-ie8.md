# 雅虎建议火狐用户改用“新的、更安全的 IE8”

不确定这是什么时候开始发生的，但是很明显，雅虎 T2 现在让火狐用户知道，换成 T4 会更好。除了昨天发出的这条推文之外，我们没有注意到任何人注意到它，所以我们怀疑它是新鲜的，是最近雅虎和微软爱情盛宴的直接结果。

多么讽刺。大约一年前，在微软收购尝试[失败后，雅虎](https://web.archive.org/web/20230216121927/https://techcrunch.com/2008/02/01/wow-microsoft-offers-446-billion-to-acquire-yahoo/)[在其首页](https://web.archive.org/web/20230216121927/https://techcrunch.com/2008/09/03/it-looks-like-yahoo-dislikes-internet-explorer-too/)发布了一条非常不同的建议，建议互联网浏览器用户转而使用“新的更安全、更快的火狐 3”。

显然，雅虎推荐浏览器是基于他们最新的商业联盟。(**更新:**或者如迈克尔所说，[显然最安全的浏览器是向雅虎支付最多的浏览器](https://web.archive.org/web/20230216121927/https://techcrunch.com/2009/08/20/apparently-the-safest-browser-is-whoever-is-paying-yahoo-the-most/))

也许他们应该开发和营销自己的浏览器，消除困惑？
