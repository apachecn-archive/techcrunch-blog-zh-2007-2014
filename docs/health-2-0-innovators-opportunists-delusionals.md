# 健康 2.0:创新者、机会主义者和妄想者 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/09/health-2-0-innovators-opportunists-delusionals/>

**编者按:** *这篇客座博文由曾入围 TechCrunch Disrupt 决赛的患者关系管理公司[Avado.com](https://web.archive.org/web/20230203064927/https://avado.com/#Avado)的首席执行官戴夫·蔡斯撰写。此前，他是埃森哲医疗实践咨询公司的管理顾问，为 25 家医院提供咨询服务，并且是微软医疗业务的创始人。你可以在推特上关注他。*

上周， [Health 2.0](https://web.archive.org/web/20230203064927/http://www.health2con.com/) 结束了其有史以来最大的活动，与会者比之前最大的活动多了 50%。[许多公司在会议期间推出了](https://web.archive.org/web/20230203064927/http://thehealthcareblog.com/blog/2011/09/22/health-2-0-launch-finalists-announced/),但像许多活动一样，最有趣的活动不是主舞台上发生的事情，而是许多会外会议/活动。另一个值得注意的项目是传统医疗保健领域内外的大公司的重组。这是医疗保健中断系列中强调的进一步证据(见下面的链接)。

最有趣的一句话是马克·贝托里尼的首席执行官安泰说的:“我们正在发展成为一家拥有健康保险的健康 IT 公司。”他表示，他们 340 亿美元收入中有 120 亿美元来自非保险收入流，去年他们完成了超过 16 亿美元的收购。大公司会带来更多。我与《财富》100 强公司进行了几次交谈，这些公司非常重视在医疗保健行业的存在(或扩大其现有的存在)。大多数还没有宣布任何事情，但不要惊讶在未来几个月会看到更多。

概括地说，我从健康 2.0 中得到的启示是，领导项目和创业的人有三类——创新者、机会主义者和妄想者。在这三个案例中，汤姆·埃夫斯林的名言“没有非理性繁荣就没有伟大的成就”抓住了这个行业的现状。自然，许多人会被淘汰，但一些巨大的成功将会出现。

最有趣的当然是创新者。稍后会有更多的介绍。机会主义者让我想起了互联网繁荣时期的许多公司。顾问、投资银行家之类的人追逐万能的金钱，因为他们把医疗保健视为一个快速赚钱的地方。与互联网热潮不同，医疗保健领域没有太多快速翻转的机会，尽管预计会看到一些能够为开发商提供适度回报的微交易。很容易嗅出机会主义者，因为他们在医疗保健方面没有什么背景或真正的热情，并撒上“有意义的使用”等正确的流行语，以表现出他们了解这种情况。

在展示“酷应用”的 Health 2.0 上，妄想随处可见，但不幸的是，许多初创公司都陷入了互联网泡沫破灭后的相同陷阱。他们有一句熟悉的过去时代的名言——“我们不担心收入。”我以前看过这部电影，知道它的结局。显然，他们没有读过《健康技术失败:健康创业公司给创业者的教训》。

创新者是真正的行动所在。我将重点介绍几个例子，我在健康 2.0 活动的四天中花了很多时间。诺华公司赞助了为期两天的代码马拉松。我相信诺华公司发布 API 将被视为向[制药 3.0](https://web.archive.org/web/20230203064927/https://techcrunch.com/2011/07/23/healthcare-disruption-pharma-3-0-will-drive-shift-from-life-science-to-healthtech-investing-part-i-of-iii/) 转变的开创性时刻。

我在 [Health 2.0 Code-a-thon](https://web.archive.org/web/20230203064927/http://thehealthcareblog.com/blog/2011/09/21/health-2-0-code-a-thon-novartis-invites-all-comers-to-innovate-around-their-api/) 中概述了这一点的含义:诺华邀请所有参与者围绕他们的 API 进行创新[ *披露:Avado 为实现与 API 集成的表单和服务提供了底层平台*。与 API 的含义相比，诺华在向市场发出信号方面做得更好，他们“对业务持开放态度”，愿意与创新的个人和初创公司合作，共同发展业务。

虽然颠覆性技术很有趣，但我最感兴趣的是颠覆性的新医疗服务模式。克莱顿·克里斯滕森的开创性著作是《创新者的困境》,但他随后与合著者医学博士 Jason Hwang 合著了一本名为《创新者的处方:医疗保健的颠覆性解决方案》的书。

在那本书里，他强调了许多这些颠覆者。早些时候，我在 MedLion 中强调了几个颠覆性的新医疗服务模式的例子:[硅谷最重要的组织，没有人听说过](https://web.archive.org/web/20230203064927/https://techcrunch.com/2011/06/19/the-most-important-organization-in-silicon-valley-that-no-one-has-heard-about/)和 [WhiteGlove Health 的融资轮，由技术支持的服务驱动](https://web.archive.org/web/20230203064927/http://blog.seattlepi.com/seattlestartupbuzz/2011/08/01/whiteglove-health%E2%80%99s-ipo-powered-by-technology-enabled-services/)。

很难对克里斯滕森和黄提出的案例提出异议，即为了杀死让地方、州和联邦政府破产的医疗成本巨兽，必须进行破坏性创新。世界上最受尊敬的经济学家之一劳拉·泰森说:“我们没有债务问题，我们有医疗保健问题。”新成立的集团[医疗保健交付创新者联盟](https://web.archive.org/web/20230203064927/http://www.hdia.org/) (HDIA)，成立的目的是确定和推进新医疗保健交付系统的标准，证明它们可以大幅降低成本，同时改善健康结果和消费者体验。

在互联网泡沫破裂后，我在互联网广告市场行业协会(IAB)担任了一个角色，有人请我分享这一经历如何对加速医疗保健领域颠覆性创新者的成长有所启发。《创新者的处方》的合著者 Jason Hwang 也出席并参加了 HDIA 的第一次面对面会议。披露:Avado 已经作为联盟的创始成员加入了 HDIA。 ]

我很高兴地发现，HDIA 的创始人有一个与我在下面的嵌入式演示中概述的“处方”相似的计划。随着互联网广告行业的转变，证据、标准和教育对于加速这些令人兴奋的新模式的发展至关重要。该联盟邀请各组织加入他们的运动。虽然欢迎任何真诚推动创新的组织，但 HDIA 对那些对扭转医疗保健恶性通货膨胀感兴趣的雇主尤其感兴趣。

自然，如果雇主(支付大部分医疗费用的人)向市场发出信号，表示他们将购买真正有所作为的医疗服务模式，这将加速这些模式的发展。现实是，任何颠覆性创新者都不会有什么损失，而在位者主要关注的是保持当前的收入流。医疗保健领域的收入流几乎占到了经济总量的 20%,请放心，现任者将会使用 [FUD](https://web.archive.org/web/20230203064927/http://en.wikipedia.org/wiki/Fear,_uncertainty_and_doubt) 和书中的每一个其他策略来保护这个收入海洋。

与任何带来变革的群体一样，创新者正在经历亚瑟·叔本华所阐述的真理的三个阶段——首先是被嘲笑，其次是被强烈反对，最后是被接受为事实。MedLion、WhiteGlove、Qliance 等组织正处于第二阶段，因为他们已经证明了他们的模型是可行的，这威胁到了现状。HDIA 的目的是让它走向真理的最后阶段。
