# Jolicloud 再次转向，现在是云服务的起始页——这次他们成功了吗？

“永不放弃，永不投降”是从[惊爆银河系](https://web.archive.org/web/20221231005928/http://www.imdb.com/title/tt0177789/)中抽出来的名言，这是一个真实的短语 [Jolicloud](https://web.archive.org/web/20221231005928/http://jolicloud.com/) 的创始人塔里克·克里姆用来描述基于云的“社交聚合服务”的重新启动。好吧，好吧，这太不厚道了，你得帮帮那家伙——他一直往前走。如果你见过的话，这是一位[专注于使命的企业家](https://web.archive.org/web/20221231005928/https://techcrunch.com/2012/09/11/zuckerberg-the-leader/)。但公平地说，Jolicloud 已经在方向上发生了一些变化，甚至有人可能会说，它即将如此艰难地转向，开始类似于 Krim 因其第一个开始页面而闻名的初创公司 [Netvibes](https://web.archive.org/web/20221231005928/http://netvibes.com/) 。解释一下:Jolicloud 最初是一个 Linux 发行版，甚至一度销售笔记本电脑，然后推出了一个聚合你所有社交账户的云服务，现在它再次发生变化，成为云中桌面和你所有云账户(从 Twitter 到 Dropbox)的主页/起始页之间的一种合并。他们称这是“你的云生活的开始页”,并表示这一最新举措是基于用户告诉他们想要的东西，包括下载了他们的 Linux 发行版的 400 万用户。

这是不是又回到了 Netvibes 模式？不完全是。但公平地说，他们认识到，虽然 Jolicloud 是一项不错的服务，但它并没有真正实现其作为一个真正的社交账户聚合器的承诺，在这个地方，你可以找到你已经联系到的一切。所有这些实际上都是之前 Jolicloud 内部的东西，但是新的界面将所有之前放在独立筒仓中的东西都呈现了出来。这完全是正确的举措，我们现在很可能会看到 Jollicloud 被更广泛地采用——假设一切按计划进行。

新的 Jolicloud 将让你能够通过仪表盘访问你的 Dropbox、Google Drive、box 和 Skydrive 账户以及你插入的任何社交账户的信息。这非常方便。举例来说，如果 MySpace 倒闭了，你在 Jolicloud 上收集的任何东西都将继续存在。许多其他服务也是如此。他们以前也是这样做的，但现在可能更具可扩展性和用户友好性。

“云的痛苦是碎片化，但碎片化是自由。把所有东西放在不同地方的能力就是选择的能力。所以我们给了人们自由选择的权利，”克里姆在都柏林网络峰会上告诉我。“云就像音乐行业——你有主要品牌和独立品牌，”他说，这是一个很好的类比，例如，你现在可以在一个地方拥有你的 Instapaper(一个“独立”)和你的 Google Drive(一个“主要品牌”)。

“我们接受了一切，决定重新聚焦，做一件事，并把它做到最好。它不仅仅是云中的一个桌面，还是你的入口，”Krim 说。

Krim 说，Jolicloud 的操作系统版本也有 400 万次下载，在学生中很受欢迎，他们有效地回收旧笔记本电脑作为“Jolibooks”。他们可能会在某个时候利用这种牵引力做一些事情，也许是在教育领域。

但就目前而言，Jolicloud 还需要继续努力。永不放弃。

2009 年，它从 Atomico 和 Mangrove Capital Partners 总共筹集了 420 万美元，显然必须见证这一最新的“转向”——如果你可以称之为“转向”的话。但这显然是正确的选择。
