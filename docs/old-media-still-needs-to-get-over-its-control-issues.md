# 旧媒体仍然需要克服其控制问题 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/05/26/old-media-still-needs-to-get-over-its-control-issues/>

[![endofcontrol.jpg](img/0b9f69eeecc025ed565e36fb4f5c1c74.png)](https://web.archive.org/web/20221207210111/https://beta.techcrunch.com/wp-content/uploads/2008/05/endofcontrol.jpg "endofcontrol.jpg")

互联网的奇妙之处在于没有人控制它。如果你不能控制媒介，你就不能控制信息。在这个有 1 亿博客、YouTube、Digg 和[推特狂热](https://web.archive.org/web/20221207210111/http://www.beta.techcrunch.com/2008/05/25/twitter-2/)的时代，这似乎再明显不过了。事实上，就在今天早上，我应邀参加了脸书一个名为[结束控制](https://web.archive.org/web/20221207210111/http://www.facebook.com/group.php?gid=15171398777)的小组，讨论随着控制权从媒体公司转移到消费者手中而出现的问题。(这个小组是由作家 Gerd Leonhard 发起的，他正在写一本关于同一主题的书。

然而，习惯于控制的行业不愿意放弃控制。旧媒体就是这样。即使在这个时代，它与控制问题的斗争仍在继续。旧媒体知道与观众的关系已经改变，但它仍然不太确定如何处理它。

为了说明我所说的，让我分享上周的两件趣闻。周三，我参加了 Mediabistro Circus 会议，参加会议的大部分都是纽约的媒体专业人士。我注意到一些演讲者反复提到的一个主题是，要在网络上取得成功，就必须放弃控制权。这是一个新发现，尽管行业观察家在过去的几年里一直在观察这个现象。是的，观众(喘息)会回应，他们通常更喜欢相互交谈，而不是简单地阅读媒体专业人士决定发布的新闻。

我想改变需要一段时间来适应。但让我感到奇怪的是，像这样的事情仍然需要解释。据我所知，参加会议的其他人大多来自报纸、杂志和其他广播媒体的数字部门。对他们中的任何一个人来说，要吸引网上的观众，你必须让他们评论、投票、发表评论，甚至选择突出报道什么，这真的是新闻吗？

我不这么认为。但是，知道一些事情和能够做一些事情之间是有区别的——在这种情况下，改变你自己根深蒂固的习惯，并说服同事(其中许多人在广播时代长大)也这样做。

现在，把这个群体和我周四在多伦多参加网上会议时结交的网络企业家做个对比。没有人需要向他们解释，为了在网络上取得成功，他们应该停止试图控制信息或观众。也许这是因为他们一开始就没有试图控制观众。相反，情况恰恰相反。他们中的一些人忙于创建“T2 ”,正是这些工具让观众能够自己报道事件并传播自己的信息。而其他人则帮助建立了整个业务，给予观众更多的*控制权。*

其中一位演讲者是丹尼尔·伯卡，Digg 的创意总监(Pownce 的联合创始人)。他指出，今天没有一个头脑正常的人会创办一家印刷报纸公司。看看报纸公司在卡车、纸张和印刷上花了多少钱。支付所有这些基础设施费用的唯一方法(更不用说创建内容所需的大批记者、编辑和摄影师)是控制观众，让他们为大量的新闻来找你。

但这在网络上没有任何意义。即使是 Digg，它应该把网络和其他媒体网站上最好的头条放在一起，也不会假装它是你唯一的信息来源。正如布卡所说:

> *用一页纸来统治所有人的想法毫无意义。我们的主页不是您访问的唯一主页。如果你认为你能创造一个一刀切的目的地，这是一个糟糕的设计决定。*

不是每个旧媒体的人都对这个新世界的现实视而不见。例如,《T8 有线》杂志的主编克里斯·安德森在 Mediabistro 会议上做了一个更开明的主题演讲，他在演讲中建议媒体公司采取一种更有层次的方法来展示内容(并从中获利)。

在金字塔的顶端将是传统的新闻业，在这一行业中，作家和编辑们沉迷于每一个词和每一个图像，并且为了代表他们的读者独家获取资源而竭尽全力。金字塔的底部是读者评论、推特和博客文章，链接到媒体渠道的主流故事。中间将是两者的混合，最好的评论和受众博客会冒泡，与记者博客和团体博客相遇。

金字塔顶端的故事会播放由媒体公司的 salesforce 销售的高 CPM 广告，而金字塔底部的长尾评论和博客帖子会播放来自 AdSense 和其他广告网络的长尾广告。随着故事沿着金字塔向上移动，它们会挖掘出更高的 CPM 广告。

也许安德森需要把他的金字塔倒过来，但至少他在考虑用网络的双向媒体来沟通旧媒体的世界。这意味着最好的东西将会到达中间层，然后反馈到他的媒体金字塔的顶端。也许这些观众的帖子和评论可以在网站上突出显示，甚至可能出现在印刷品上——尽管安德森在他的演讲中没有走那么远。

但是，放弃控制不仅仅是重播观众的最佳贡献。它是关于创造最好的谈话发生的地方。安德森当然理解利基媒体的潜力和力量。作为个人兼职项目，他为无人机爱好者创建了自己的 Ning 社交网络，名为 [DIY Drones](https://web.archive.org/web/20221207210111/http://diydrones.com/) 。它每月带来 400 美元的 AdSense 收入，但他只能获得 7 美元的 CPMs。最棒的是，他让聚集在他周围的社区贡献大部分内容。(注意，DIY 无人机不是 Conde Nast 或 Wired.com 的一部分)。

“成为最高的侏儒，”他向任何想创建自己的利基媒体网站的人推荐。

这是个不错的建议。但是旧媒体能在侏儒之地生存吗？他们很难控制。
