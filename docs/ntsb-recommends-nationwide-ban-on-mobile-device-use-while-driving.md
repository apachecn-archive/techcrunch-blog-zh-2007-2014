# NTSB 建议全国禁止开车时使用移动设备

如果美国国家运输安全委员会(National Transportation Safety Board)通过他们的方式，开车时打电话(或发短信，或发微博)将成为过去。监管机构今天建议在全国范围内禁止开车时使用任何手持设备。

该建议是在 NTSB 委员会讨论密苏里州格雷萨米特悲惨的多车相撞事故时提出的，事故起因是一名司机使用手机撞上了一辆拖车的后部。

几个州已经颁布了“[分心驾驶](https://web.archive.org/web/20221006184554/http://www.ghsa.org/html/stateinfo/laws/cellphone_laws.html)”法律，处理消费者在开车时能做什么和不能做什么，但 NTSB 首选的行动路线将比各州敢走的更远。委员会的建议不仅仅局限于移动设备——免提设备也将被禁止。

NTSB 主席黛博拉 A.P 赫斯曼谈到这个决定时说，这很困难，但“这是正确的建议，是时候了。”

现在，NTSB 纯粹是一个调查机构，没有法律权力来执行这样的禁令。即便如此，当 NTSB 讲话时，人们倾向于倾听(这个故事的所有情节都证明了这一点)。如果一些有事业心的国会议员已经开始起草反映 NTSB 观点的法案，我一点也不会感到惊讶，尽管它能走多远还很值得怀疑。

然而，人们不禁想知道，如果全国范围的禁令成为现实，会发生什么。随着免提套件不再被允许，Jabra 和 Plantronics 等无线配件公司将会看到相当大一部分市场消失。随着车型年代的推移，车内电话连接变得越来越普遍，那么这也会受到禁令的限制吗？该建议含糊不清的措辞提到，将对“旨在支持驾驶任务”的设备进行例外处理，这为争论每项相关技术的利弊留下了足够的空间。
