# 日产 Leaf 伴侣应用程序现可用于 Android 和黑莓

日产聆风车主们，注意了。Leaf companion 应用程序现在有安卓和黑莓两种版本。这些智能手机应用遵循了今年早些时候与 Leaf 一起推出的 iOS 版本的步骤。所有的关键功能都在这里:电池电量计，充电时间表，估计范围，和气候控制功能。

这些应用程序本身可以免费下载，不需要拥有一片叶子。但可以理解的是，它们需要一片叶子才能正常工作。不过，有远见的买家或好奇的旁观者可以免费下载和探索这些应用程序。

我和叶子在一起的时候看了[叶子 iOS](https://web.archive.org/web/20230205030459/https://techcrunch.com/2011/06/24/a-simple-look-at-the-simple-nissan-leaf-ios-app/) app [。这是一个很好的应用程序，允许用户远程访问 Leaf 的关键功能。唯一值得注意的缺失功能是切换车锁的能力。](https://web.archive.org/web/20230205030459/https://techcrunch.com/2011/06/29/nissan-leaf-review/)
