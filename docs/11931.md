# 脸书收购了 2009 年起诉它的 WhoGlue 公司背后的技术

今天早上，脸书向我们证实，它确实在本月早些时候收购了总部位于巴尔特摩尔的软件公司 T2 公司。WhoGlue 为会员制组织(比如大学校友网络)开发社交网络软件，并且显然很幸运地在 2001 年，也就是脸书存在的三年前，申请了一项名为[“分布式个人关系信息管理系统和方法”](https://web.archive.org/web/20230403060201/http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&p=1&u=%2Fnetahtml%2FPTO%2Fsearch-bool.html&r=1&f=G&l=50&co1=AND&d=PTXT&s1=whoglue.ASNM.&OS=AN/whoglue&RS=AN/whoglue)的专利。

2009 年，两人因专利问题闹上法庭，2010 年 3 月，案件以对 WhoGlue“非常积极的方式”结束。据 WhoGlue 创始人 Jason D. Hardebeck 称，两家公司之间正在进行的诉讼后对话最终促成了出售。

一名脸书代表给了我们以下声明，“我们可以确认我们获得了一些由 WhoGlue 开发的技术。WhoGlue 团队不会加入脸书，而是会继续为组织开发自己的社交网络软件。

根据其他报道，脸书还收购了包括西门子在内的 WhoGlue 股东的股份，但是脸书没有透露这些报道真实性的更多细节。

与渴求人才的脸书不同的是，WhoGlue 团队没有被收购，事实上 Hardeback 计划成立一家新公司，WhoGlue LLC，基本上将在同一领域继续运营。

WhoGlue 技术允许一个群体在一个更大的网络中进行私人联网，并且可以应用于子网功能，例如在不侵犯用户隐私的情况下在整个脸书更广泛地集成脸书群体的内容。

我们的常驻脸书专家 Josh Constine 也有这个非常有趣的理论，“如果它在中国推出，这项技术也可以帮助脸书为这个国家运行一个封锁的子网，不会违反防火长城。”
