# 品尝台是美食家的日常糖果 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2010/07/04/tasting-table-is-the-daily-candy-for-foodies/>

像 [DailyCandy](https://web.archive.org/web/20230204074625/http://www.dailycandy.com/all-cities/) 和 [Thrillist，](https://web.archive.org/web/20230204074625/http://www.thrillist.com/list/Nation)这样针对小众主题或受众的邮件通讯已经被证明是一种成功的模式。当然，这两个例子都有鲍勃·皮特曼[的点金术，他被](https://web.archive.org/web/20230204074625/http://www.crunchbase.com/person/bob-pittman)[称为](https://web.archive.org/web/20230204074625/https://techcrunch.com/2010/06/02/bob-pittman-geekchicdaily/)“通讯领域最成功的投资者。”皮特曼最近的项目之一，[品尝台，](https://web.archive.org/web/20230204074625/http://tastingtable.com/)已经不为人所知，但它的模式已经取得了显著的增长，甚至盈利。

自 2008 年 10 月推出以来，Tasting Table 每天提供免费的电子邮件，推荐当地的餐饮、葡萄酒、鸡尾酒、烹饪或美食旅行。目前，品尝台有纽约，洛杉矶，旧金山，华盛顿，DC 和芝加哥版本。首席执行官 Geoff Bartakovics 说，在每个城市的午餐前都会发出电子邮件，以便在用户饥饿和考虑食物选择时吸引他们。在 20 个月的运营中，该网站拥有 30 万订户，根据皮特曼的说法，鉴于品尝台的历史，这是他支持的增长最快的电子邮件出版物。

除了本地版，品尝台还有其他垂直市场。Bartakovics 表示，国家版(你可以在该国任何地方享用的食物和饮料推荐)和厨师食谱版(顶级厨师未公布的食谱，品尝台工作人员为家庭厨师修改的食谱)正在快速增长。

在商业模式方面，Tasting Table 的收入主要基于围绕其编辑内容购买广告空间的广告商。广告商包括，LVMH，美国运通，Le Creuset 和喜力啤酒。第一年，Tasting Table 的收入不到 100 万美元，这家初创公司实现了盈利。Bartakovics 表示，今年品尝台的收入有望达到 300 万至 400 万美元。

当然，对于任何一个以食物为导向的评论网站来说，一个知识渊博的员工对于建立信誉是至关重要的。在 17 名全职员工中，6 名员工拥有曼哈顿法国烹饪学院的学位，许多人来自主流美食媒体出版物，如美食和葡萄酒。Bartakovics 表示，Tasting Table 与其竞争对手(如 Urban Daddy 和本地美食博客)的区别在于，它的作者会多次尝试该机构，无论是酒吧还是餐厅，然后根据他们的经验发送建议。Bartakovics 说，时事通讯上的每一条推荐都经过了一位专职作家的检验。

虽然我认为自己是一个初出茅庐的美食家，而且在收到电子邮件几天后就已经是一个品尝台的粉丝，但我联系了我所知道的最认真的美食家，即由两位律师吉尔和安迪·弗里德曼创办的[wine anddined、](https://web.archive.org/web/20230204074625/http://www.winedanddined.com/)一个[迅速发展的](https://web.archive.org/web/20230204074625/http://newyork.timeout.com/articles/own-this-city/86772/seven-twitter-feeds-worth-following)纽约本地美食博客的人们。Wined and Dined 一致认为，时事通讯的餐厅和酒吧优惠和内容使其成为任何美食家的必读之作。例如，品尝台最近与里奥哈葡萄酒公司合作，推广里奥哈葡萄酒餐厅周，在纽约市的餐厅推出特别菜单和优惠。

而 Tasting Table 的内容甚至引起了合作伙伴的创业公司的注意。例如，流行的基于位置的移动社交网络 Loopt [在 Loopt 的移动应用程序 Loopt Pulse 中集成了](https://web.archive.org/web/20230204074625/https://techcrunch.com/2010/02/17/loopt-another-content-partner-integrates-local-foodie-news-from-tasting-table/)品尝表批准的餐厅和酒吧以及评论，加入了 Zagat 和 Citysearch 的内容。

随着美食杂志等传统食品出版物的关闭，初创公司有机会提供优质的食品内容，就像 Yelp 和 Citysearch 等非常受欢迎的评论网站一样。当然，挑战在于将它推广到全国甚至美国以外的城市，但看起来品尝台可能会变成美食家的尖叫。
