# 谷歌放弃“Flash 地图应用编程接口”

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/02/google-abandons-maps-api-for-flash/>

关于 Flash 即将衰落的更多消息是，谷歌宣布将为 Flash“放弃”谷歌地图 API(应用编程接口)。这个 API 以前允许开发者在他们基于 Flash 的应用程序中添加谷歌地图功能。

然而，谷歌表示，截至目前，该 API 的使用是有限的，只有少数应用程序利用了谷歌地图 Flash API 独有的功能。

最初[于 2008 年 5 月](https://web.archive.org/web/20230204230114/http://google-latlong.blogspot.com/2008/05/news-flash-maps-now-open-to-flash.html)发布的 API，类似于[的 JavaScript 版本](https://web.archive.org/web/20230204230114/http://code.google.com/apis/maps/)，但被设计用于 Flash 应用中。开发人员可以使用 API 通过各种服务操作地图并向地图添加内容，这使他们能够在网站上嵌入交互式地图应用程序。

Flash API 的地图实际上并没有被完全取消——而是被*废弃了。*这意味着它将继续根据 [Maps API 服务条款](https://web.archive.org/web/20230204230114/http://code.google.com/apis/maps/terms.html)运行，但不会开发新功能，只会修复“关键错误、退化和安全问题”。

换句话说，API 基本上被抛弃了。

相反，Flash 开发者被鼓励通过 Maps API v3 迁移应用程序，谷歌表示，这提供了其他好处，如[街景](https://web.archive.org/web/20230204230114/http://code.google.com/apis/maps/documentation/javascript/services.html#StreetView)、[融合表格集成](https://web.archive.org/web/20230204230114/http://code.google.com/apis/maps/documentation/javascript/overlays.html#FusionTables)、[位置搜索](https://web.archive.org/web/20230204230114/http://code.google.com/apis/maps/documentation/javascript/places.html)和[对移动浏览器的完全支持](https://web.archive.org/web/20230204230114/http://code.google.com/apis/maps/faq.html#browsersupport)。谷歌地图 JavaScript API v3 论坛提供了帮助。

如上所述，鉴于人们对 API 的独特特性兴趣有限，这一举动并不令人惊讶。然而，在某种程度上，这一决定似乎与谷歌近几个月来支持 Flash 的其他举措相矛盾。例如，去年夏天， [Adobe 和谷歌联合宣布](https://web.archive.org/web/20230204230114/https://techcrunch.com/2010/06/25/google-chrome-flash/)Flash Player 将很快内置到 Chrome 网络浏览器中，从而消除用户单独下载、安装和更新插件的需要。

此外，谷歌的移动操作系统因其对 Adobe Flash 和基于 Flash 的应用程序的[支持而闻名。](https://web.archive.org/web/20230204230114/https://market.android.com/details?id=com.adobe.flashplayer&hl=en)

在今天的博客文章中，谷歌还补充说，它也继续支持 Flash 作为 Chrome 的开发平台，但更多细节的链接会转到[一个死 URL](https://web.archive.org/web/20230204230114/http://www.google.com/support/chrome/bin/answer.py?answer=108086%20) 。

也许那个页面也被否决了？

***更新**:谷歌修复了链接。正确的网址是[http://www.google.com/support/chrome/bin/answer.py?答案=108086](https://web.archive.org/web/20230204230114/http://www.google.com/support/chrome/bin/answer.py?answer=108086) 。*
