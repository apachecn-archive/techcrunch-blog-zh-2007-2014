# 叛国工具技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/06/tools-for-treason/>

德文·科尔德韦投稿人

More posts by this contributor

我们的权利因我们使用的工具而扩大和限制。互联网放大了我们的言论自由能力，但降低了对隐私的合理预期。当然，我们在任何时候都毫无保留地自愿提供我们的数据。即使我们被烧伤了，我们还是会回来给予更多。我想我们这样做是因为我们信任那些我们给予的人，尽管他们现在几乎毫无例外地失去了这种信任。显然，他们重新获得的任何信任都必须是暂时的，为什么要为此烦恼呢？

似乎如果我们要重新开始，我们交流工具的基本原则不能是建立信任，而是不可能信任。

这是一个愤世嫉俗的开始，但显然是必要的。如果信任在任何时候都有可能进入等式，那么这种信任能够并且很可能会被利用。如果在一长串服务器、交换机、电缆、解释器、记录器、驱动程序、协议、接口和显示器中，从一端到另一端的任何地方，有一个地方您不能百分之百地控制您的数据，那么您的数据就会受到损害，系统就会出现故障。

诀窍在于将每一次交流都视为潜在的恐怖主义行为。毕竟，国安局不是这样做的吗？对他们来说，这是一个借口；对我们来说，这应该是一种方法。从那里开始，你可以建立一个有效的系统。从那里开始，你会被告知你在为叛国制造工具。你是。

很好。如果它不是非法的，它就不够强大。如果政府不谴责它，那就不够好。叛国工具是*唯一可以从现在开始保护我们的工具。*

但让我们明确一点:他们*将*用于邪恶，既卑鄙又邪恶:用于交易儿童色情制品，用于出售冰毒，用于策划暗杀，用于大规模谋杀。你会被告知你在武装敌人。你是。但是你的工具对于这些暴行来说既不必要也不足够。每把菜刀都锋利得足以割伤你的同伴；每一把锤子都足够坚硬，可以劈开头骨；每辆车都足够快，可以撞倒行人。他们必须实现他们的目的，这在这里也是一样的。

记住，我们谈论的不仅仅是脸书聊天和谷歌搜索。对于一个跨国合作的游击队员来说，保护个人电子邮件似乎有些过头了。当涉及到脸书泄露的备用电子邮件地址或他们 Gmail 账户的元数据时，许多人可能不关心他们的隐私。他们可能以后会回心转意，但必须有人确保，如果他们真的回心转意，隐私甚至是一种选择。

如果你的算法不允许恋童癖者不可逆地扰乱他的驱动器并避免起诉，它就不能被意识形态压迫下的自由思想者用来隐藏国家禁止的书籍。如果你的信息应用不能让某人安全地计划轰炸超级碗，它就不能被激进分子用来揭露侵犯人权的行为。如果你的地图不让偷猎者在不惊动护林员的情况下跟踪犀牛，它就不能被少数民族用来逃避清洗。这款工具的强大之处在于能够实现所有这些功能，这是一个很老很老的测试，我们之前已经多次测试过它的用途。一如既往，答案是“两者皆有”。

我们将如何制造这些神奇的工具？实际上只有两个主要的需求，如果我们假设(至少一开始是错误的)用户可以正确地操作它们。

他们必须完全无视法律。没有理由尊重它——甚至政府也不尊重它。警方对数据、传票和其他任何东西的请求必须是完全无能为力的，至少没有用户的同意。但是蔑视法律是不够的，除非一个人对法律免疫。

因此，它们决不能被集中。对于编辑照片和分享餐馆推荐，网络平台作为一种服务是好的，并将继续好下去，但对于个人通信或任何类型的机密数据则不是。几乎每个云服务都把遵守法律放在用户需求之上(SpiderOak 是个例外，我在这里写错了，他们可以根据命令解密；正如一位评论者指出的那样，它们不是，尽管它们可以披露的元数据也可能是关键的)。无论是在自己的硬件上，还是在租用的或虚拟化的硬件上，自托管都是远程确保数据安全的唯一方法。

在每个家庭中放置网络连接存储和弹出式网络服务器，并观察现有的整体结构被侵蚀。随着个人千兆连接、我们自己的兆兆字节服务、端到端加密以及在基础层面上实现的点对点，我们的通信将不再依赖于除关键基础设施之外的任何东西——甚至这些东西迟早也会过时。确定正确的协议、填补空白和扩展兼容性需要时间，但重要的是把它发布出去。和 Bittorrent 一样，猫不会被放回包里。torrents 成为一个家喻户晓的词花了十年时间，但以服务和代理机构意外透露他们的卡片的速度，让人们接触他们内心的密码专家可能不是一条漫长的道路。让它像 BonziBuddy 一样容易安装，你就会开始做一些不容易停止的事情。

一个简单的事实是，我们信任的政府和权力已经证明自己是不值得和不可靠的(如果不是完全应该受到谴责的话)。尊重他们的利益不应该再是理所当然的事情，推进互联网自然去中心化的本质是合乎逻辑的下一步。创造服务于私人(或受压迫的)个人利益的东西，而不是最终服务于那些希望强加于他或她的人，应该是下一个十年软件和平台开发的主要任务。

他们不会喜欢它，因为自由是做错和做对的自由，他们作为仲裁者害怕他们将不再能够辨别你在做什么。好吧，我们厌倦了他们知道——厌倦了他们试图发现。我们有技术。我们可以宣布独立。我们以前做过。
