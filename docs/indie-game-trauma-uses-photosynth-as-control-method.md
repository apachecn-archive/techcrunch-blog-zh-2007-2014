# 独立游戏《外伤》使用 Photosynth(？)作为控制方法

[http://vimeo.com/moogaloop.swf?clip_id=7319563&server = vimeo . com&show _ title = 1&show _ byline = 1&show _ 人像=0 & color=00ADEF &全屏=1](https://web.archive.org/web/20230307225756/http://vimeo.com/moogaloop.swf?clip_id=7319563&server=vimeo.com&show_title=1&show_byline=1&show_portrait=0&color=00ADEF&fullscreen=1)
您可能还记得 Photosynth，这是微软令人印象深刻但令人不满意的照片拼贴工具。虽然我总是对技术方面感到兴奋，但我从来没有喜欢过它。也许是我的摄影师更喜欢一张构图精美的照片，而不是一百张相互关联的快照。不过，一位游戏开发者看到了我从未考虑过的潜力。王南钧·马杰夫斯基设计了《外伤》，这是一款非常有趣的点击式冒险游戏，利用了 Photosynth 的功能，从图片中创造出“层次”。

抽象的、心理探索的氛围让我想起了迷人的[镜台](https://web.archive.org/web/20230307225756/http://www.increpare.com/tag/mirror-stage/)，梦幻般的叙述让我想起了麦克斯·瑞切特的 [*《蓝色笔记】*](https://web.archive.org/web/20230307225756/http://www.last.fm/music/Max+Richter/The+Blue+Notebooks) 。无论如何，这看起来像是一次引人注目的小冒险。

注意，我说的是 Photosynth，但是这个游戏显然是在 Flash 中运行的，这表明它是 Photosynth 的一个衍生物。我们走着瞧。很明显，游戏将在发布时发布；相应地添加书签。
