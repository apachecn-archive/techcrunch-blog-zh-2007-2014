# 谁给我买根橡木和鸵鸟皮的拱廊手杖，马上

每隔一段时间，我就会渴望在一个新地理模拟器上玩几个虚拟游戏，比如《T2》中的《武士归来》或《T4》中的《合金弹头 X》中的《T5》，但事实是，当我不得不使用键盘时，我从中得不到什么满足感。当然，我可以使用我的 360 控制器，但它就是感觉不对。不，要真正感受经典，你需要一个街机棒。这可能是我见过的最奇特的街机棍了。

[Hoon](https://web.archive.org/web/20221208131756/http://www.hoon.fr/) / [Neo Legend](https://web.archive.org/web/20221208131756/http://www.neo-legend.fr/) 操纵杆在巴黎手工制造，由涂漆橡木制成，配有*鸵鸟皮外壳。它可以在 PS3、360 或 PC 上运行，应该会在 18 日上市。*

价格没有公布，可能是因为鸵鸟皮本身的价格就相当于一根普通的街机手杖。我不确定你是否能订购一个，或者你是否必须去真正的地方订购。我有点希望是这样。如果我不能拥有一个，我也不想让任何人拥有一个——当然，除了巴黎人。

事实上，看起来你可以在 Hoon 预订。所以就这样了。
