# 百思买(Best Buy)和极客小队(Geek Squad)想出了如何通过游戏系统设置来进一步压榨消费者

[![](img/5ff4ff560c9cebb0c801956f1dd688a4.png "tattoosidekick")](https://web.archive.org/web/20221003190838/http://www.businesswire.com/portal/site/home/permalink/?ndmViewId=news_view&newsId=20081120005655&newsLang=en)

每个人都知道，极客小队已经准备好通过[为你](https://web.archive.org/web/20221003190838/http://www.businesswire.com/portal/site/home/permalink/?ndmViewId=news_view&newsId=20081120005655&newsLang=en)提供游戏系统设置来拿走你更多的血汗钱。定价过高的极客将为 29 美元的低价设置用户账户和家长控制。这是店内安装的费用，一如既往，Geek Squad 会向你收取高额的室内安装费用，这样懒人就不需要阅读使用手册了。

设置用户账户有多难？我的天，控制台会引导你完成整个过程，但不管怎样，百思买不会让人们购买这些设置。
