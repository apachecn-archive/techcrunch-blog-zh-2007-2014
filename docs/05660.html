巴茨有“什么是雅虎？”回答时间缩短到15秒。它既宽泛又模糊

今天在旧金山 Web 2.0峰会上，卡罗尔·巴茨走上舞台，与主持人约翰·巴特利坐在一起。虽然巴茨确实有一些关于内容的有趣的事情要说，但巴特尔并没有拐弯抹角。“你能告诉我们，雅虎是什么吗？”他问她。

”我可以。我的意思是也许它花了我2年时间，但我得到了它。这个问题让我很感兴趣，因为我认为雅虎是一个非常简单的故事。内容、通信、媒体、技术，这就是雅虎。

好吧，这比她今年早些时候在我们的TechCrunch Disrupt大会上给出的答案要好。如果不说别的，就是短了很多。但还是非常非常模糊。听起来，雅虎几乎就是一切。

嗯，除了搜索。巴茨说，当人们认为雅虎是一家搜索公司时，它就偏离了轨道。所以我们知道他们不是那样的。

但其他一切现在似乎都摆在桌面上。我认为雅虎代表了很多。它代表乐趣。”嗯。好吧。
