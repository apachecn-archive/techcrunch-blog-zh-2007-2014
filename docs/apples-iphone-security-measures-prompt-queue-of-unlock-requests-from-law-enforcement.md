# 苹果的 iPhone 安全措施促使执法部门的解锁请求排队

根据来自 CNET 的一份新报告，苹果面临执法官员大量的解锁 iPhone 设备的入境请求。被扣押的带有密码锁的 iPhones 显然足够安全，足以挫败美国的许多警察机构，因此苹果公司建立了一个等待名单，以帮助其处理当局的解锁请求。

根据《CNET 报告》,等候名单很长，导致烟酒枪支管理局去年夏天的一项申请推迟了 7 周。对 iPhone 用户来说，好消息是，在试图找到当地、州或联邦一级有能力在内部解锁手机的执法机构三个月后，ATF 将苹果作为最后手段。坏消息是，CNET 获得了一份宣誓书，解密似乎不一定需要客户的知识，而谷歌有一个密码重置，通过电子邮件通知用户解锁。

据 ATF 的一份证词称，据报道，苹果可以绕过安全锁获取手机上的数据，将其下载到外部设备上，然后交给当局，这意味着最终 iOS 设备上的信息不是 100%安全的。但总的来说，不断有报道称苹果设备特别能抵抗执法部门的窥探。

CNET 之前的一份报告也认为 [iMessage 在面对外界监控企图时具有弹性](https://web.archive.org/web/20221007034443/http://news.cnet.com/8301-13578_3-57577887-38/apples-imessage-encryption-trips-up-feds-surveillance/)，尤其是与 SMS 等更常见的文本通信方式相比。综合来看，这些报告表明，苹果的移动设备技术尤其擅长抵制不必要的进步，这对隐私爱好者来说非常好，尽管关于苹果何时以及为什么分享这些信息的政策需要进一步充实。

我们已经联系了苹果公司，看看他们是否对执法部门的解锁队列有任何官方评论，以及他们如何处理请求，如果我们听到更多信息，我们会更新。
