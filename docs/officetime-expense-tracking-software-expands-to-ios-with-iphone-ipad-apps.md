# OfficeTime 费用跟踪软件通过 iPhone 和 iPad 应用程序扩展到 iOS

时间和费用跟踪是乏味的，即使是在个人层面。在办公室环境中处理显然要糟糕得多。但 OfficeTime 希望让这一点变得更容易，今天宣布它将从 PC/Mac 桌面应用程序扩展到 iOS。对于像费用跟踪这样烦人的事情，OfficeTime 在手机上采取了一个聪明的方法。这里的主题是“点击越少越好”

iOs 版本似乎拥有与桌面应用相同的大部分功能。一触式报告将是其中之一，它允许用户即时访问项目任务、工作时间和费用，并可以与上个月的报告进行比较。对于自由职业者来说，还有一种方法可以跟踪他们的工作时间，甚至可以看到他们的薪酬与固定成本预算相比如何。

这些应用还具有团队报告功能，通过在[ipad](https://web.archive.org/web/20230203102007/https://techcrunch.com/tag/ipad)、[iphone](https://web.archive.org/web/20230203102007/https://techcrunch.com/tag/iphone)、MAC 和 PC 之间同步的能力，提供关于谁做了什么(以及何时做了什么)的跨团队报告。无论您在哪里插入信息，它都会显示在所有设备上。该应用程序还允许用户将任何内容导出到 Excel 或 Numbers。

iPhone 和 iPad 应用在[应用商店](https://web.archive.org/web/20230203102007/http://itunes.apple.com/lk/app/officetime-time-expense-tracking/id446200101?mt=8)的售价为 7.99 美元，桌面应用的零售价为 47 美元。不需要每月订阅。
