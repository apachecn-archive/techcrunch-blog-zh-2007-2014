# 英国公司发展透视照相机技术

![totalrecall](img/e1657530c05502badd62a323a0938a32.png)

还记得从*全面召回*的 x 光技术吗？据路透社报道，一家英国公司开发了一种被称为“ThruVision”的类似技术，“使用它所谓的‘被动成像技术’通过它们发出的自然电磁射线——被称为太赫兹或 T 射线——来识别物体”。“根据材料的不同，波的特征也不同，因此炸药可以与一块粘土区分开来，可卡因与一袋面粉也不同。”

它可以从 80 英尺外发现隐藏的物体，并且在人们移动时也能工作。它旨在用于机场、商场和体育赛事。
