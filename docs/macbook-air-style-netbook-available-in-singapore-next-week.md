# MacBook Air 风格的上网本下周在新加坡上市

![iiview](img/da16afe50153cd300ef1ee469c358752.png)

拿 MacBook Air 来说，把屏幕缩小一英寸，装上一些上网本组件，装上 Windows 7 候选发布版，然后——请敲鼓——你就有了新加坡的 iiView A2。哦，把价格降到 468 美元。这一点很重要。

我们会在美国看到这种机器吗？也许吧。大概不会。但它有一个很好的外观，让人想起 MacBook Air，这基本上是一个非常昂贵的上网本(我会为此而发火)。

[根据 CNET](https://web.archive.org/web/20221006121350/http://asia.cnet.com/crave/2009/07/30/iiview-a2-macbook-air-wannabe-netbook/) 的说法，“设计如此相似，甚至连显示器输出和 USB 端口都隐藏在侧面的翻盖内。”其他功能包括:

*   凌动 1.6GHz 处理器
*   2GB 内存
*   英特尔 945 芯片组
*   12.1 英寸，1,280 x 800 像素分辨率
*   320GB 硬盘
*   两个 USB 端口、迷你 HDMI 端口、2 合 1 耳机/麦克风插孔
*   802.11b/g，10/100 以太网
*   六芯电池
*   带 Vista 家庭高级版许可证的 Windows 7 RC1 版

iiView 的首席执行官表示，出于性能原因，上网本将预装 Windows 7 RC1。不过，Vista 许可证将允许用户在今年秋天 Windows 7 发布时升级到零售版。

iiView A2 预计下周在新加坡上市，售价 699 美元(新加坡元)，约合 467.78 美元。
