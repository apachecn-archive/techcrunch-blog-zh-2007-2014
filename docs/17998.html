AppsFlyer筹集710万美元以保持移动广告归属的真实性

SaaS移动应用测量、归因和分析初创公司AppsFlyer 已从Pitango Venture Capital和当前投资者Magma Venture Partners获得了710万美元的A轮融资。这笔资金将用于研发、产品、业务发展和客户支持。它还将在纽约市和旧金山开设两家新办事处。该公司声称现在已经盈利。

这家初创公司有一项服务，允许应用开发者、品牌和代理机构衡量所有媒体来源的移动用户获取情况，包括付费、有机、病毒和社交。简单地说，应用程序发行商可以利用这一点来确定他们从哪些网络获得了最佳回应。

它的平台集成了300多个广告网络和媒体资源，并且是脸书移动测量合作伙伴。它声称要以每年5亿美元的移动广告支出和每年10亿个移动应用安装的速度来监控移动广告活动。

首席执行官兼联合创始人柳文欢·卡尼尔非常重视它的独立性。“AppsFlyer是一个公正、独立的测量平台，”他说。Pitango think AppsFlyer的普通合伙人Eitan Bek“完全有能力改变移动广告市场。”他投资了。

它的主要直接竞争对手Ad-x被Criteo收购。但由于Criteo的业务也是销售媒体，人们有理由问它是否对市场有独立的看法。同时，HasOffers不再是脸书移动测量合作伙伴计划的成员。Kochava也是一个玩家，但在范围和能力上相对较小。

虽然有广告网络/DSP提供广告跟踪功能:Apsalar、Flurry、Inmobi、Fiksu，但它们并不直接与AppsFlyer竞争。
