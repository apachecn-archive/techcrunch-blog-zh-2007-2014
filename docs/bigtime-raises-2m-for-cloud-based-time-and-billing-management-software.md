# BigTime 为基于云的时间和计费管理软件筹集了 10 亿美元 2M

向专业服务公司提供基于云的生产力软件工具的 BigTime Software 今天早上[宣布](https://web.archive.org/web/20230203102042/http://www.businesswire.com/news/home/20110808005050/en/BigTime-Software%E2%80%99s-2-Million-Series-Financing-Oversubscribed)已经从 LTC Partners、Geneva Venture Investment Management、Hickory Grove 和 Howell Capital 获得了 200 万美元的首轮融资。

该公司向会计、建筑、营销服务和 it 服务等行业的专业服务公司提供时间、计费和实践管理[软件即服务](https://web.archive.org/web/20230203102042/http://www.edisonsattic.com/BigTime.asp)，它与 [Intuit QuickBooks](https://web.archive.org/web/20230203102042/http://quickbooks.intuit.com/) 相集成。

BigTime Software 最初成立于 2002 年，曾被称为爱迪生的阁楼。

额外的资金将用于加强销售、营销和产品开发工作。
