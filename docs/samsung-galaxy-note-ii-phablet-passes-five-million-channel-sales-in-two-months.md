# 三星 Galaxy Note II“平板手机”在大约两个月内通过渠道销售了 500 万部

本月初，三星宣布其迷你平板智能手机 [Galaxy Note II](https://web.archive.org/web/20221209235112/https://beta.techcrunch.com/2012/08/29/samsung-note-2/) 的渠道销售在发售的 37 天内[已突破 300 万台](https://web.archive.org/web/20221209235112/https://beta.techcrunch.com/2012/11/02/samsung-galaxy-note-ii-phablet-sells-three-million-in-one-month-of-sales-3x-faster-sales-rate-that-original-galaxy-note/)。现在，这家韩国手机制造商宣布，在推出约两个月后，该设备的全球渠道累计销量已超过 500 万部。

三星通常不公布面向消费者的设备销售，但其渠道销售指标显示了其销售渠道的终端用户需求有多大。

Note II 渠道销售公告是由三星在韩国[发布的新闻稿](https://web.archive.org/web/20221209235112/http://translate.google.com/translate?sl=ko&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.samsung.co.kr%2Farticle.do%3Fcmd%3Dview%26contentId%3D127659%26searchCategory%3D1)中发布的——h/t 到 [TNW](https://web.archive.org/web/20221209235112/http://thenextweb.com/asia/2012/11/26/absolutely-phabulous-samsungs-galaxy-note-ii-passes-5m-channel-sales-adding-3m-in-24-days/) 供发现。三星的发布表明，Note II 的销售在所有销售地区均匀分布，包括亚洲、欧洲、北美、拉丁美洲、中东和非洲。

最初的 Galaxy Note 和 Galaxy Note II 比典型的智能手机大，但比迷你平板电脑小，这导致了“平板手机”一词的产生，以描述这种手机兼平板电脑类别。

大约两个月内 500 万的渠道销量对于一款设备来说是一个令人印象深刻的数字，因为它的尺寸如此之大，与更普通尺寸的智能手机相比，这是一个相对容易获得的品味。举个例子，苹果最新的 iPhone，iPhone 5，在发售的前三天就卖出了 500 多万部。苹果的数字指的是对终端用户的销售，而不是渠道出货量。
