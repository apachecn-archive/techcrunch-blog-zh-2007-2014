# 固件更新应该可以修复 17 英寸 MacBook Pro 显示器问题

![firmwarefix](img/7dd11098e4cdcece58c763fe4ecde1cf.png "firmwarefix")

因此，苹果公司刚刚发布了新的 17 英寸 MacBook Pro 的固件更新,据说可以修复我们讨论过的[令人讨厌的屏幕撕裂问题](https://web.archive.org/web/20221006173921/http://www.crunchgear.com/2009/03/06/trouble-in-paradise-whats-the-matter-with-the-17-inch-macbook-pros-display/)。和所有固件更新一样，你被要求向你选择的上帝祈祷，以确保更新顺利进行。

该更新的官方名称是 MacBook Pro 图形固件更新 1.0。如果你用的是 17 英寸的 MacBook Pro，你所要做的就是从左上角的苹果菜单中选择软件更新。

如果你愿意，让我们知道你的结果如何，如果它解决了那些屏幕问题。我永远不会有足够的钱买这样一台电脑，所以我不知道你的痛苦。
