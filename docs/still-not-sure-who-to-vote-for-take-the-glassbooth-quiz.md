# 还不确定投票给谁？参加玻璃展台测试。

到目前为止，大多数美国人已经决定了在选举日投谁的票。但是，如果你仍在试图做出决定，或者只是想让自己确信，你确实是在为最能反映你观点的候选人投票，那就参加[玻璃展台测试](https://web.archive.org/web/20221007181645/http://glassbooth.org/)。这个网站是由一个无党派、非盈利组织运营的(或者 T2 声称是这样)。你告诉该网站你在一系列问题上的立场，它会返回每位总统候选人(包括第三方候选人拉尔夫·纳德、辛西娅·麦金尼和鲍勃·巴尔)的兼容性得分。你可以试试看，以确保你真的和你喜欢的候选人意见一致。

它是这样工作的。首先，你通过在 16 个不同的主题(税收和预算、科学、公民自由和国内安全、伊拉克和外交政策、互联网和媒体、贸易和经济、环境和能源、枪支管制等)中分配 20 个点来表明你最关心的问题。)

![](img/7b2fb31de1b1c794948460deacb11207.png)

然后你会被问及一系列问题，比如你是支持还是反对为中产阶级减税或加强网络中立等政策提案。每个问题都有一个从“强烈反对”到“强烈支持”的评分等级:

![](img/eeeea90aaa68c66ed20ae86569f2b5ed.png)

一旦你完成了测验，它会显示你与每个候选人的整体匹配情况，并逐期显示。然后，您可以按问题深入研究，阅读候选人对每个问题的陈述，并链接到来源。这是一个很好的方式来开始评估候选人超过一个或两个热点问题。你甚至会惊讶地发现你最同意(或不同意)谁。
