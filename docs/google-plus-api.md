# Google+ API 没有关闭…除非你是“可信的”，那么它已经准备好了

就在昨天，我们写道，那些等待 Google+ API 的开发者将不得不等待一段时间——几个月，甚至是几个月。这个令人失望的信息是由一个急切的开发者根据 Google+的项目经理告诉他们的告诉我们的。对于大多数开发者来说，这无疑是事实。但不是全部。

如果你碰巧是谷歌认为的“可信的测试者”，你可以更快地获得 API。也许就在接下来的几天里。

什么构成了“可信测试人员”？嗯，一个条件是，如果你有谷歌的风险投资部门谷歌风险投资的支持。这可能会激怒一些开发者。我确信会有 500 个关于矛盾的自然的故事，对吗？但实际上，这是有道理的。这或多或少是谷歌在完全准备好进入黄金时段之前，用一个小组测试 API。

尽管如此，这仍然是一个早期的跳跃，最终可能成为一个重要的社交分享层。

我是怎么知道这些的？我们有一封谷歌风投今天发给投资组合公司的电子邮件，宣布这项服务。下面找一下。

> 嗨–
> 
> Google+团队将在接下来的几天内开始向我们的可信测试程序推出他们的一些 API(接下来的几周内会有更全面的发布)。他们已经同意向我们投资组合中的少数公司(2-4 家)提供这种早期准入。
> 
> 如果你感兴趣，我们希望你至少能够快速(在接下来的 3-5 天内)构建一个小的“hello world”应用程序。程序的这个阶段更多的是关于文档和代码的反馈，目的是随着更多 API 的推出，与 G+进行更有意义的交互。
>
