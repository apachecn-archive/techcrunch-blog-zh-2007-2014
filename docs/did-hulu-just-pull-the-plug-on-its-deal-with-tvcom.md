# Hulu 刚刚终止了与 TV.com 的交易吗？

[![](img/95a72405e8872844a5b47b5eb1d79881.png)](https://web.archive.org/web/20221006201045/http://www.tv.com/)

TV.com 发生了一些事情，这是哥伦比亚广播公司旗下的网站，最近作为 Hulu 的竞争对手重新推出。自从去年夏天，TV.com 归 CNET 所有，仍然主要是一个社区中心，该网站通过与 Hulu 的合作展示了 NBC 和新闻集团的内容。现在，在 TV.com[作为 CBS 支持的大众媒体门户网站的直接竞争对手重新推出](https://web.archive.org/web/20221006201045/http://www.beta.techcrunch.com/2008/12/16/hey-cbs-hulu-much/)仅两个月后，看起来通过 Hulu 提供的大部分(如果不是全部)内容都不再工作了。

以前通过协议提供的剪辑和完整剧集都只提供一个说明“[视频不可用”](https://web.archive.org/web/20221006201045/http://www.tv.com/video/JBb_gixcXtTqz0n4jQb2tmCW9Aru_v5j/101/17005/top-banana?o=hulu&category=full_episode&tag=showspace;video;3)。有可能这只是一个技术故障——如果协议终止，TV.com 很可能会完全删除与这些剧集的任何链接。但是这些视频至少从今天早上开始就不可用了，修复一个漏洞似乎需要很长时间。其他视频网站，如康卡斯特的 Fancast 和 Sling.com 的 Hulu 内容似乎都很好。

如果 Hulu 事实上终止了与 TV.com 的协议，这将是对这个迅速发展的网站的一个巨大打击。TV.com 确实提供其他内容，包括 CBS、米高梅、索尼、Endemol、USA、PBS 和 Showtime 的节目，但许多粉丝最喜欢的节目仍然出现在 NBC、Fox 和 Hulu 合作的其他网络上。

我们已经尝试联系 Hulu 来解决这个问题，一旦得到回复，我们会尽快更新帖子。
