# 观看 Windows 8 在 10 秒内启动，体验四分之一英里的生活

我的超频四核台式机在加载 Windows 7 后需要 3 分钟才能启动，更不用说发布和运行所有预装 Windows 垃圾软件所需的时间了——这还是在安装了 [Soluto](https://web.archive.org/web/20230204181715/https://techcrunch.com/2010/05/26/techcrunch-disrupt-winner-soluto/) 的情况下。休息后的快速演示显示，笔记本电脑在 10 秒内从硬启动启动到 Windows 8。你大概花了这么多时间来阅读我被要求写的愚蠢的帖子，而不是仅仅发布视频和简短的信息“看这个”。太牛逼了。”

微软通过将内核会话写入磁盘来实现这种快速启动，本质上是将计算机置于休眠模式，而不是完全关机。这将引导时间缩短了 30-70%,因为计算机不必每次都加载所有的驱动程序和服务。[更多在此](https://web.archive.org/web/20230204181715/http://gizmodo.com/5838675/windows-8-boots-up-faster-than-a-bull-ride)

微软将在下周的 BUILD 大会上推出 Windows 8，并可能会通过类似的视频展示其即将推出的操作系统。
