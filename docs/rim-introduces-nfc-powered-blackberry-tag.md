# RIM 推出 NFC 技术的黑莓标签

早在二月份，RIM 首次承诺 2011 年“许多，如果不是大多数”黑莓手机将会采用 NFC 技术。事实证明，[黑莓 Bold 9900/9930](https://web.archive.org/web/20230203095115/https://techcrunch.com/2011/05/02/rim-announces-blackberry-bold-9900-and-9930-with-bbos-7-and-nfc/) ，以及黑莓 Curve 9350/9360/9370 都是如此，我们应该会看到更多这方面的产品。也就是说，RIM 联合首席执行官吉姆·巴尔西利本周在迪拜的 GITEX 会议上还有一个锦囊妙计，[首次推出了](https://web.archive.org/web/20230203095115/http://crackberry.com/rim-unveils-blackberry-tag-coming-soon-blackberry-smartphones)黑莓标签。

黑莓标签将成为下一次黑莓 7 更新的一部分。这个想法是，你可以通过一起点击你的黑莓设备来分享照片、文档、网址和联系信息——这与 iOS 和 Android 上的 Bump 应用程序没有什么不同。唯一的区别是，Bump 应用程序采用了蓝牙连接，而 BlackBerry Tag 使用了 NFC 技术。

现在，BBM 用户不用扫码，只需轻轻一碰设备就能建立联系，这将自动交换 BBM 的联系信息。更好的是，开发人员将可以访问黑莓标签 API，这意味着他们可以将标签功能融入自己的应用程序中。

对 RIM 来说，这是艰难的一年。但是也许这一点，加上其他新的软件产品，会让 RIM 重回正轨。
