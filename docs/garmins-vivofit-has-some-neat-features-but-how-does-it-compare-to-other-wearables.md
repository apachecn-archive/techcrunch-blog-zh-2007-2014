# Garmin 的 Vivofit 有一些简洁的功能，但它与其他可穿戴设备相比如何？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/03/26/garmins-vivofit-has-some-neat-features-but-how-does-it-compare-to-other-wearables/>

Garmin 抓住了世界的尾巴。有了他们令人惊叹的车载 GPS 设备和出色的运动手表，事情看起来只会越来越好。直到市场上的每部手机都有了 GPS 芯片，谷歌绘制了世界地图。现在，Garmin 必须迎头赶上，他们正在追逐迄今为止最受欢迎的可穿戴设备之一 Fitbit。

他们的新乐队名为 Vivofit，看起来像一个更便宜的 Fitbit 或 Nike FuelBand。Garmin 知道一些关于电池寿命的事情，所以他们放弃了明亮的 OLEDs 和 led，代之以反射式 LCD 解决方案。Garmin 知道一些关于繁忙的周末运动员的事情，所以他们增加了一个内置电池，可以持续一年。他们对跑步者略知一二，所以他们在胸前安装了一条可以监控你心率的肩带。

他们知道他们必须把这个产品做好，所以他们把它定价为 130 美元，与[陷入困境的 Fitbit Force](https://web.archive.org/web/20221007005313/https://beta.techcrunch.com/2014/03/12/fitbit-force-1-million-units-sold/) 完全一样。可悲的是，我认为他们的好运到此为止了。

首先，你要明白，我是那种无法忍受跑步机办公桌的人。虽然 Vivofit 在行走和跑步时工作良好，但它不会记录在跑步机上以每小时 2 英里的速度骑行时的具体活动。事实上，我发现很少有设备可以在跑步机办公桌上工作。例如，Fitbit Force 会定期记录 Vivofit 的几千步，虽然我不指望这两款设备的绝对准确性，但这种差异足以让我不信任这两款设备中较新的一款。这显然影响了我的判断——我知道只有少数人使用跑步机办公桌——但这对我来说是个阻碍。你可以把表带取下来放在口袋里，但这违背了初衷。

然后，为了检查你的心率，你必须一直佩戴舒适的胸带，即使是在睡觉的时候。这种前景可能会让一些人感到担忧，鉴于 Basis 和许多其他健康波段已经提供光学心率扫描，这甚至有点繁重。但是 Garmin 牺牲了易用性，却弥补了电池寿命的不足。该电池可持续使用一年多，是目前最好的电池。你不必把这个东西拿下来充电的想法是惊人的，但尺寸的减小显然是有代价的。

最后，定义其他一些健身带的易用性和简单性在这里消失了。Garmin 的应用很稳定，但在同步方面仍有问题。手环必须进入特殊的同步模式才能与你的手机连接，这样当你完成一定数量的步骤时，你就不会看到通知。没有内置的蜂鸣器来告诉你什么时候你已经达到了你的目标，显示器是一个简单的反射式液晶显示器，很难阅读。而且，鉴于它是一个 Garmin，大多数长期粉丝会哀叹缺乏一个精确的 GPS 追踪器。但是，要明确的是，这不是什么。

Garmin 生产很棒的健身手表。在我看来，它们已经让 Polar 黯然失色，就连大型户外手表之王 Suunto 也可以从它们身上学到一些东西。我目前正在使用 Garmin 的另一款产品菲尼克斯 2，我对此印象深刻。我使用 Garmin 跑步手表已经很多年了，甚至在跑一场倒霉的马拉松时也戴着它。我一直对 Garmin 印象深刻。有了 Vivofit 我就不那么想了。

这款产品是用一款简单的设备涉足可穿戴设备市场的结果。虽然我是 Fitbit 的粉丝，但如果他们增加背光和更好的通知系统，我可能会被说服转向，而想要心率监测器的用户会对增加一个而不增加手腕体积感到满意。同样，Garmin 在这里做了一些事情，但这并不能成为选择它而不是现任之一的理由。

如果你需要一个心率监测器，而且你喜欢这个品牌，那就买一个佳明吧。如果你对这两个方面都不确定，那么有很多选择，Garmin 将会在下一个周期用稍微好一点的功能和一些调整来说服你。
