# 新的美国化的诺基亚 N-95 与旧的 N-95:手牵手，肩并肩

![dsc_0284.JPG](img/52b81901b4c64a84d9c61c33bc99c415.png)

诺基亚刚刚宣布了一种新的美国化 N-95(照片中右侧的那种)。重要的是，它经过优化，可以接入我们的 GSM 3G 网络，因此您可以在美国使用 HSDPA 或 EDGE(它可以与美国电话电报公司和 T-Mobile 一起跳舞。)它也有一些外观上的变化——如你所见，像相机镜头。它将于 9 月在美国上市，价格类似于 iPhone，为 699 美元(主要是因为它只在无锁状态下销售，没有运营商补贴。)
