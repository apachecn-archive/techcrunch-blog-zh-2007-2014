# 基于云的视频编辑器 WeVideo 向公众发布

基于云的协作视频编辑平台 WeVideo 于 10 月宣布与 YouTube 整合，今天退出测试期并开始商业运营。该服务现在拥有超过 125，000 名用户，每天使用在线平台创建超过 1，000 个视频项目。

总部位于加州森尼维尔的 WeVideo 于 2011 年秋季推出，实际上是一家名为 [Creaza](https://web.archive.org/web/20230203221830/http://www.creazaeducation.com/) 的欧洲企业的副产品，该企业为欧洲 25 万学生提供类似的、但专注于教育的电影编辑器。与受计算机硬件限制的桌面编辑软件不同，WeVideo 的在线视频编辑解决方案可以根据最终用户的需求按需提高处理速度。随着用户在公司付费层级上的提升，速度也在提高，可用存储空间和视频分辨率也在提高。

此外，摄像师可以一起协作完成项目，所有项目成员可以通过单一界面共享他们的视频剪辑。(该功能仅对付费用户开放)。

随着 WeVideo 进入全面上市状态，它已经确定了免费增值服务的价格。免费版本提供 1 GB 的工作空间，并可扩展到每月 6.99 美元至 79.99 美元的 10 至 100 GB 工作空间，以及不同的导出质量和附加功能。还将提供企业版(价格可根据要求提供)，这将允许组织将 WeVideo 集成到他们现有的产品或服务中。

企业版包括一个配置层，允许公司添加自己的品牌、完整的 API 级别支持、集中式共享存储架构以及针对协作、编辑、管理审查和视频导出选项的高级管理控制。

首席执行官 Jostein Svendsen 说，该公司预计到 2012 年底将积累超过一百万的用户。
