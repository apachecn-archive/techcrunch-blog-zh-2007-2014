# 脸书将为整个该死的互联网发布一个“喜欢”按钮

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/03/25/facebook-to-release-a-like-button-for-the-whole-darn-internet/amp/>

 <amp-img src="https://web.archive.org/web/20230228042641im_/http://www.crunchbase.com/assets/images/resized/0000/4561/4561v1-max-250x250.png" class="shot amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 在即将于四月举行的 [F8 开发者大会](https://web.archive.org/web/20230228042641/https://techcrunch.com/2010/03/01/facebook-f8-tickets/)上，将会有大量关于脸书产品发布的消息泄露。这是因为他们已经开始和第三方开发者一起测试很多新东西，一旦两个人知道了一个秘密，它就不再是秘密了。

我们听说的一个新功能是脸书连接和脸书 API 的扩展，允许出版商为他们网站上的任何内容添加“喜欢”按钮。

听起来微不足道？它不是。这可能是脸书开放图形应用编程接口项目的一部分，该项目将通过共享内容和相关元数据来激励第三方网站与脸书进行深度互动。

今天，你可以通过一个简单的按钮与脸书“分享”内容(你可以在这篇文章的顶部看到我们的实现)。我们听说，新的喜欢功能远远超出了分享按钮。

对出版商有利？是的。但这对脸书来说也是非常非常好的，因为成千上万的网站会争先恐后地按照脸书的偏好来编排内容，然后毫不犹豫地把所有数据发送过来。

一位了解该产品的消息人士说，可以这么想。谷歌花费数十亿美元为他们的搜索引擎索引网络。脸书将让网络为自己做索引，专门针对脸书。

是的，这是一个伟大的想法。或者，正如 MG 所说，整个互联网将变成脸书的支流系统。这一切都源于一个简单的 Like 手势，以及我们不久将会谈到的其他一些功能。
