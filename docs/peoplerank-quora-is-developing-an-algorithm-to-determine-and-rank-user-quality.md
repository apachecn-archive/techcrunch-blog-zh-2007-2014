# PeopleRank: Quora 正在开发一种算法来确定用户质量并进行排名

众所周知，就使用和增长而言，Q & A site Quora 正在[爆炸](https://web.archive.org/web/20230202232039/https://techcrunch.com/2011/01/07/quora-scaling-yahoo-answers/)。目前，该网站每月有大约 164，000 名独立访问者(comScore 2010 年 12 月的统计数据)，我们从一个消息来源获悉，该服务每四个月用户就翻一番。这种增长的缺点是内容可能会受到影响(例如雅虎问答的问题)。虽然 Quora 的联合创始人[查理·契弗](https://web.archive.org/web/20230202232039/http://www.crunchbase.com/person/charlie-cheever)已经[暗示了](https://web.archive.org/web/20230202232039/https://techcrunch.com/2011/01/07/quora-scaling-yahoo-answers/)该公司计划缓解这一问题的一些方式，但他最近在一篇名为[“扩大规模”的新 Quora 帖子中更详细地介绍了这些举措](https://web.archive.org/web/20230202232039/http://www.quora.com/Charlie-Cheever/Scaling-Up)

奇弗写道，这家初创公司正在开展几个新项目，以支持新用户，调节大量内容的涌入，并帮助网站扩大规模。他提到的最有趣的项目之一是，该公司目前正在开发一种确定用户质量的算法，这将是技术方面的一个巨大补充。契弗是这样写的:

我们正在开发一种算法来确定用户质量。该算法有点类似于 PageRank，但由于人们与网络上的页面不同，Quora 上可用的信号也不同于网络上的信号，所以这不是完全相同的问题。我们将使用它来帮助决定在提要中显示什么，何时发送通知，以及如何对回答进行排名。

虽然在开发方面付出了巨大的努力，但 Quora 开发一种类似 PageRank 的技术来对网站上的人和内容进行排名是完全有意义的。对于那些不知道的人来说，PageRank 是由谷歌联合创始人拉里·佩奇和谢尔盖·布林开发的链接分析算法，是谷歌搜索的技术基础。

奇弗说，这家初创公司还在努力让更多人帮助评估网站上新内容的质量，有效地在 Quora 上进行众包审核。他补充道:*大多数使用 Quora 的人都有很好的判断力，我们相信人群中有一些智慧。初步看来，这种方法很有前途。*

奇弗说，教育是帮助 Quora 扩大规模的另一个关键部分。他断言，如果新用户在加入时明白 Quora 是什么，这将有助于影响用户向网站添加适当的内容。例如，Quora 在新用户添加新问题之前增加了一个简短的辅导测验，Cheever 表示，它在减少不符合指南或政策的问题数量方面发挥了很大作用。

他补充说，Quora 正在努力通过其他方式来改善网站，减缓增长。

如果 Quora 能够开发出一种“PeopleRank”算法，这无疑是一项非常重要的技术。一项专利算法肯定会让这家初创公司成为更理想的收购目标。
