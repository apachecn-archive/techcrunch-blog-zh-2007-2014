# 使用 Pubmatic 可以给你带来一些免费广告

[![pubmatic_logo.png](img/99ce412fa0f8a4a2de055a0c8a2646fe.png)](https://web.archive.org/web/20221006204950/http://pubmatic.com/) 自从互联网出现以来，广告变得复杂多了。外面有数百个广告网络。随着定位成为游戏的名称，有效地分配你的广告空间给访问者几乎是不可能的。作为回应，自动广告优化引擎已经开始帮助管理这些广告的复杂性。对于关心如何充分利用广告的出版商来说，使用电子邮件似乎是显而易见的。广告费率随着时间的推移变化太快，为不同的人群提供合适的广告的复杂性很难用手来处理，尤其是在占地面积较大的网站上。

已经有几个引擎问世，包括 [YieldBuild](https://web.archive.org/web/20221006204950/http://www.beta.techcrunch.com/2007/10/19/yieldbuild-will-try-to-pay-you-more-than-adsense/) 和 Rubicon 项目。现在，TechCrunch 40 决赛入围者 [Pubmatic](https://web.archive.org/web/20221006204950/http://pubmatic.com/) 正在举办一场竞赛，以吸引更多的出版商加入他们的平台。每个月，使用 Pubmatic 获益最大的两家出版商将在顶级“web 2.0 网站”(即博客)上获得数千美元的免费广告。因为他们还在最后确定他们要做广告的网站，所以数量还不知道。你可以在这里报名参加这个项目[。](https://web.archive.org/web/20221006204950/http://www.pubmatic.com/challenge)

PubMatic 的广告优化引擎类似于 YieldBuild 和 Rubicon 项目。与前两者不同，Pubmatic 实际上有 1300 个站点，并对其他任何人开放。该引擎作为一个元广告服务器，在给定时间、地理、人口统计和网站布局的情况下，为每个访问者提供最高收入的广告网络。它目前支持像谷歌 AdSense，雅虎出版商网络和价值点击这样的广告网络。他们还为他们提供了一个中央仪表盘来跟踪他们所有的广告网络和广告配置。下面你会从他们的 TechCrunch 40 视频中找到更多信息。

Pubmatic 已经选出了 11 月份的赢家 WinCustomize 和 BikerOrNot，他们声称自从使用 PubMatic 以来，他们的广告收入增加了 100%以上。YeildBuild 和 Rubicon 项目为他们的出版商带来了类似的增长。在某种程度上，这完全取决于你的网站开始时的盈利状况。然而，Pubmatic 正在从访问量相当大的网站中寻找明智的成功，并将仔细检查你以前的货币化指标，以确保收入的增加是真实的。
