# Evony 广告活动:乳房欺骗你玩文明克隆

![evony](img/c057b6f9e3d69d6b81160491f3ff1210.png "evony")

我相信你在互联网旅行的某个时候已经在网上看过这个广告了。这是为了游戏 *Evony* ，一个在线*文明*的克隆，虽然它没有在冰上彻底发臭，但也不是最好的。但是游戏的质量无关紧要，因为今天我们看到的是游戏的垃圾广告活动。基本上，它由一对贴满互联网的乳房组成。

所以这个游戏没什么值得称赞的。它是由环球多人游戏娱乐公司开发的，该公司有[链接](https://web.archive.org/web/20230319190520/http://www.guardian.co.uk/technology/gamesblog/2009/jul/15/games-evony-spam-internet)到中国的淘金者，而且，它基本上是一个*文明*的克隆。(以前叫 *Civony* 。)然而，广告活动几乎完全由邀请你玩游戏的胸部丰满的少女组成。

请注意，游戏中没有公主需要拯救。她只是让你对游戏感兴趣。

我能不能继续长篇大论地说，“这是社会的堕落”等等。但也许 CrunchGear 应该采取类似的广告活动？就雇一群拿着 Palm Pres 和 Wii 遥控器的丰满模特？
