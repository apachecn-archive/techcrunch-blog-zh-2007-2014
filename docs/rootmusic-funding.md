# RootMusic 筹集了 230 万美元帮助乐队建立壮观的脸书网页

![](img/7e8f6275e9ffa89962f0603dbf455fe9.png)

根据一份提交给美国证券交易委员会的文件，总部位于旧金山的初创公司已经筹集了[230 万美元](https://web.archive.org/web/20230202225121/http://www.crunchbase.com/company/rootmusic)的融资，该公司本质上是让乐队和音乐人更容易[在](https://web.archive.org/web/20230202225121/https://techcrunch.com/2010/06/27/facebook-musicians-page/)[脸书](https://web.archive.org/web/20230202225121/http://www.crunchbase.com/company/facebook)上建立外观惊人的页面。

如果你想了解更多关于 RootMusic 的信息，可以看看 [Orli 的帖子](https://web.archive.org/web/20230202225121/https://techcrunch.com/2010/06/27/facebook-musicians-page/)，但基本上，RootMusic 让音乐家和乐队建立起奇特的脸书页面，包括艺术品、音乐、照片、视频、音乐会日历等等。

对于想要深入定制他们所谓的“乐队主页”的乐队来说，有免费和付费两种选择。

如果你对一个例子感兴趣，或者查看 RootMusic 的巡演，这是汤姆·佩蒂和心碎者乐队的专辑。

我们就融资事宜联系了 RootMusic，但他们拒绝提供更多细节。当他们有时间的时候，我们会更新他们的 CrunchBase 档案。
