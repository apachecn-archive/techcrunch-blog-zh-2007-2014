# 安全风险情报公司 Rapid7 融资 5000 万美元

开发安全风险情报产品的公司 Rapid7，已经从 TCV 的技术交叉风险投资公司(Technology Crossover Ventures)获得了 5000 万美元的 C 轮融资[。这使该公司的资金总额达到 6000 万美元。](https://web.archive.org/web/20230205043222/http://www.businesswire.com/news/home/20111117005767/en/Rapid7-Secures-50-million-Series-Funding)

Rapid7 为企业提供了一种产品，可以识别公司 IT 基础架构中的任何安全风险，并根据攻击的可能性确定补救措施的优先级。该公司的旗舰产品 Nexpose 可以扫描整个 IT 环境中的安全风险，包括 Web、网络、应用程序和数据库。

该公司拥有 1，700 名客户，称其收入在过去四年中增长了 900%以上，并在 2011 年第三季度报告了其第十个季度的创纪录收入。

新资金将用于招聘、产品开发、战略收购和国际扩张。
