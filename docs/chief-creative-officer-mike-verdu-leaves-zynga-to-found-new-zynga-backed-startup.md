# 首席创意官 Mike Verdu 离开 Zynga，创立了新的、由 Zynga 支持的创业公司

Zynga [今天](https://web.archive.org/web/20221206153453/http://company.zynga.com/about/press/company-blog/starting-new-chapter)宣布，首席创意官[迈克·弗杜](https://web.archive.org/web/20221206153453/http://www.linkedin.com/pub/mike-verdu/0/113/2a6)将在三年后离开公司，领导自己的创业公司。这可能会给一些投资者带来问题，因为 Verdu 多年来一直是 Zynga 成功背后的驱动力。

但是，这并不是完全的分离。Zynga 将成为 Verdu 新公司的投资者。他的新公司尚未公布。

Zynga 首席执行官兼创始人马克·平卡斯(Mark Pincus)说:“迈克是我个人和工作上的好朋友，也是我们所有人有影响力的创意领袖。”。“我为迈克帮助建立的遗产以及创意人才和领导者的深厚基础感到骄傲，他们将高举火炬，塑造 Zynga 的下一波创意。”

“在 Zynga 工作的三年里，我见证了这家公司从一个互动娱乐前沿的小而杂乱的初创企业成长为一个大型的世界级组织，在世界各地拥有 20 个开发工作室和大量令人惊叹的人才，”Verdu 说。“早期在 Zynga 工作让我想起了我是多么热爱做一名企业家。经过一番自我反省，我决定回到自己的根本，创办一家新公司。”
