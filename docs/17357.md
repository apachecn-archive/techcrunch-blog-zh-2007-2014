看起来脸书已经收购了 [Branch Media](https://web.archive.org/web/20230404163638/http://branch.com/company) ，conversation service Branch 和[链接共享服务 Potluck](https://web.archive.org/web/20230404163638/https://techcrunch.com/2013/06/25/from-the-team-behind-branch-potluck-is-a-new-link-sharing-service-for-the-internets-lurkers-who-dont-tweet-or-blog/) 背后的初创公司。

联合创始人乔希·米勒在脸书发表了声明[。他说，他的团队将组建一个新的对话小组，该小组将设在纽约市，“目的是帮助人们围绕自己的兴趣与他人联系。”他补充道，“尽管我们打造的产品会让人想起 Branch and Potluck，但这些服务将在脸书之外继续存在。”](https://web.archive.org/web/20230404163638/https://www.facebook.com/joshamiller/posts/10151916252911547)

该帖子缺乏细节(米勒说他目前正在度假，现在才发帖，因为他“被告知这个故事将会泄露”)，但 [The Verge 报道](https://web.archive.org/web/20230404163638/http://www.theverge.com/2014/1/13/5303702/facebook-acquires-link-sharing-app-branch-for-15-million)价格为 1500 万美元。我已经联系了脸书以获取更多信息，如果有回音，我会及时更新。

Branch Media 的投资者包括 Obvious Corp .(由 Ev Williams、比兹·斯通和詹森·高德曼领导的孵化器——[——在 2013 年的一篇文章中，](https://web.archive.org/web/20230404163638/https://medium.com/obvious/7e542c703a7e))威廉姆斯表示，高盛“去年搬到了纽约市，大部分时间都在与……Branch 合作”)、Lerer Ventures、Betaworks、SV Angel 等。据报道，初创公司[筹集了 200 万美元的种子资金。](https://web.archive.org/web/20230404163638/http://www.businessinsider.com/branch-raises-2-million-from-sv-angel-and-others-2012-3)

**更新:**脸书发言人证实分支媒体团队将加入公司。他们拒绝讨论交易条款，但强调这是一次人才收购。
