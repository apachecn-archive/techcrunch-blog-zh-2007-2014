# CabEasy 公司希望让陌生人与半价出租车费配对

[![](img/2e399a5974bbbc8895f1b7e2c9e1f1a7.png)](https://web.archive.org/web/20221205073946/http://www.cabeasy.com/)

随着经济的动荡，大多数人都在想尽一切办法存点钱。本月早些时候成立的一人创业公司 CabEasy 希望帮助人们节省打车费用。该网站允许人们发布他们即将到来的出租车旅行计划的公开列表，并与沿着类似路线旅行的其他人配对来分担旅程。

这个概念与 [Hitchsters](https://web.archive.org/web/20221205073946/http://www.hitchsters.com/) 非常相似，后者是一家帮助[为旅行者](https://web.archive.org/web/20221205073946/http://www.beta.techcrunch.com/2008/11/10/hitchsters-20-half-priced-airport-limos-in-san-francisco-nyc/)配对的初创公司，这样他们就可以在去机场的路上共用一辆豪华轿车，费用只是他们通常支付的一小部分。Hitchsters 计划扩展到机场旅客之外(包括像球赛这样的活动)，但目前这些选项还不可用。而且，与 CabEasy 不同，Hitchsters 只为少数几个城市提供支持。

CabEasy 具有更大的灵活性，但该网站显然仍处于起步阶段，缺乏对大量受众有用的功能。首先，没有一种简单的方法来搜索某个半径范围内即将到来的旅行——你需要在地图上自己查看(现在这种方法很好，但如果 CabEasy 建立了相当大的用户群，就会变得混乱)。我也怀疑这是否会经常派上用场——大多数人很少提前计划他们的出租车行程，除非他们要赶飞机。如果 CabEasy 变得足够大，总是很容易找到匹配，那么它将是一个伟大的服务。但在那之前，如果只有很小的机会有用，大多数人可能不会花时间去查看这个网站。
