# 在旅途中裁剪虚拟优惠券:RetailMeNot 推出其首款 iOS 应用

流行的优惠券网站 [RetailMeNot](https://web.archive.org/web/20230209124854/http://retailmenot.com/) 在今年早些时候推出了它的移动网站，今天，该公司推出了它的第一个用于 iOS 的[原生应用。新的应用程序允许不喜欢支付全价的用户找到在线和店内优惠券。用户可以访问流行优惠券和“热门交易”的精选列表他们还可以为该公司称之为“时移购物”的优惠券添加书签根据该公司自己的数据，其网站已经从移动用户那里获得了大约 17%的流量。该公司的首席执行官 Cotter Cunnigham 说，其移动渠道正在“每月同比增长 200%”](https://web.archive.org/web/20230209124854/http://www.retailmenot.com/mobile/iphone/)

尽管该公司重新强调移动应用，但该应用不允许你发现身边的优惠券。也许这将在以后的版本中出现，但考虑到 RetailMeNot 自己认为 35%的成年人报告说，当他们手中有优惠券时，他们更倾向于在实体店购物，这感觉像是一个奇怪的省略。不过，该应用程序本身设计得非常好，由于其预测搜索功能，很容易找到优惠券。

该公司强调，这只是其首次涉足原生移动应用，并计划在不久的将来通过新功能显著改善其应用。今年早些时候，正如我们的记者 Josh Constine [在](https://web.archive.org/web/20230209124854/https://techcrunch.com/2012/04/04/retailmenot-card-linked-coupons/)报道的那样，RetailMeNot 宣布将很快允许用户将他们的信用卡信息整合到他们的优惠券中。选择加入这一功能的用户不用使用离线优惠券，很快就可以选择他们想在线或在旅途中使用的优惠券，然后前往商店，而不必打印任何东西或展示他们的虚拟优惠券。优惠券将在结账时悄悄使用。为了提供这项服务，RetailMeNot 与 [CardSpring](https://web.archive.org/web/20230209124854/http://www.crunchbase.com/company/cardspring) 合作。不过，就目前而言，这项服务仍处于测试阶段，还没有集成到移动应用程序中。

为了庆祝今天的发布，RetailMeNot 还委托进行了一项研究，调查美国消费者如何使用手机购物。从这项研究的结果来看，移动优惠券还有相当大的增长空间。例如，只有大约五分之一的受访者表示，他们在店内购物时使用过移动设备上的优惠券。然而，七分之一的人表示，他们在实体店购物时通过移动设备进行了在线购买，因为他们在网上找到了更好的价格。
