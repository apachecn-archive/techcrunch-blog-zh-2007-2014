# 问 Siri 的 Ziggy 要 Windows Phone，完成 500 万美元的融资回合，扩展到 iOS、Android

问问瑞格，Windows Phone 是否能回答 Siri 的问题。这是一个虚拟的语音助手，可以将脚换成码，为您提供股票报价，并回答生活中最困难的问题。今天，该公司宣布，他们已经收到了来自“一家公开交易的跨国公司”的 500 万美元新一轮融资，该公司的名字尚未披露。

Ask Ziggy 计划用这笔钱扩展到 Windows Phone 之外，希望到 2012 年第三季度在所有安卓、苹果和微软设备上发布。考虑到 Ask Ziggy 目前只在 Windows Phone 上提供，这个时间期限相对较短。

Ask Ziggy 正与 Nuance 密切合作，以语音转录服务(以及其他服务)而闻名，以实现这一梦想。该公司还增加了新的战略合作伙伴，包括 Kopin 公司及其 Golden-i 项目。Golden-i 设备旨在识别正常的输入方法，如触摸、击键等。而且还能理解语音输入。

“我们与 Ask Ziggy 的产品关系将在未来几个月内加强，”Kopin 的 Golden-i 项目经理 Jeffrey J. Jacobsen 说。“将齐格的 22 种语言的基于云的自然语音搜索引擎与 Golden-i 的世界级嵌入式实时语音和手势界面相结合，将会迅速将‘Siri’级语音识别从仅仅是娱乐转变为快速、准确、专业水平的信息获取，并且完全‘免提’了。”"

雅各布森接着说，Ask Ziggy 的基于云的平台是扩展 Golden-i 的最佳方式。

有了 500 万美元的资金，Ask Ziggy 不仅计划扩展到新的平台，而且将很快向希望利用语音助手的开发者发布一个 SDK。
