# 在移动应用中，免费并不免费，但是剑桥大学有一个计划来解决这个问题

围绕一些移动应用和社交网络等免费服务的信息隐私问题，经常会遭到来自另一方的反驳:如果服务是免费的，你用户就是产品，所以当你的信息作为商业模式的一部分被“出售”时，你不应该感到惊讶，这就是所谓的免费“隐性成本”。

然而，这似乎是一种令人不舒服的安排，所以现在英国剑桥大学的一些学者想出了一种解决办法，并透露了一些关于应用程序中数据收集的惊人研究，作为他们努力的一部分。

如果你担心你的信息如何被分享，应用商店是一个成熟的目标。着眼于 Android 市场，研究人员设计了一个 API 来分析商店中的免费和付费应用程序。通过梳理超过 25 万个应用，他们发现 73%的应用是免费的，其中 80%依赖定向广告作为主要商业模式。

在那些使用定向广告的应用程序中，剑桥大学的研究人员发现，其中 70%的应用程序正在收集与应用程序本身无关的数据。

他们指出，就下载量而言，免费应用更受欢迎:只有 20%的付费应用下载量超过 100 次，只有 0.2%的付费应用下载量超过 10，000 次，而 20%的免费应用下载量超过 10，000 次。**尽管如此，即使是付费应用程序也不能避免多余的数据收集，似乎**:40%的付费应用程序在收集应用程序运行实际上不需要的信息。

一些例子:在漫画类别中，35%的免费应用程序请求访问用户的位置；在其他情况下，游戏收集用户的电话号码和联系人。应用程序收集的其他“敏感”数据，包括访问用户的信息(电子邮件/短信)、联系人、日历、电话号码和 IMEI，这些数据并不是应用程序本身实际需要的。

然而，屏蔽所有不需要的东西的问题是，它会影响开发者如何在免费应用上建立业务。这就是剑桥研究人员提出的解决方案:将应用程序信息与广告信息分开，并确保广告网络只获得他们需要的信息，仅此而已。他们对此的术语是“脱钩”。

研究人员之一伊利亚斯·莱昂蒂亚迪斯博士解释说:“那么应用程序就不会把广告作为收集信息的借口。”“一个应用程序会收集它真正需要的东西。”进一步说，广告显示不必要的个人信息会被自动屏蔽。

他表示，当前模式的问题在于，开发商负责收集所有信息，包括位置、人口统计数据等，然后将这些信息转发给广告网络。“你不知道这些数据是给应用还是给广告商的，也不知道它们会被如何使用。”

Leotiadis 表示，分离信息的服务可以采取应用程序本身自带过滤器的形式，或者可以集成到移动平台中默认工作:leotiadis 表示，他更希望看到平台提供商默认提供这种服务。无论如何，他认为让开发者自己管理这些是不现实的:“市场上有超过 52，000 个开发者，但只有八个大的广告网络，”他说。“控制那些网络比控制那些开发商容易。”

值得注意的是，剑桥的调查只关注安卓市场。莱昂蒂亚迪斯告诉我，这是因为谷歌为用户设置了权限请求:每次用户下载应用程序时都会出现这些请求——他指出，问题是人们往往会在没有真正看他们同意的内容的情况下点击“确定”。相比之下，苹果公司管理这些独立于每次下载，因此更难跟踪。
