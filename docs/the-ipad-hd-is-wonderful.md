# iPad HD 烂透了* 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/03/06/the-ipad-hd-is-wonderful/>

*我实际上还没看过。我不确定它会有什么特点。我很确定我现在这么说是个傻瓜。但是，嘿，首先！！！！

曾几何时，在苹果发布会的前一天，每个人都可以在最后一刻发表自己的预测。在很大程度上，这是一个纯粹的奇迹时刻。这些天来，似乎是时候预先考虑苹果“将”宣布什么了。鲨鱼已经被跳过了。蛇吃自己的尾巴。

这个问题——如果你想这么说的话——当然是源于这样一个事实，即拥有最佳资源的技术网站已经非常善于抓住苹果最终公布的许多关键惊喜。(这就是为什么 OS X Mountain Lion 如此令人惊讶——这是一个真正的惊喜！)他们中的大多数人并没有把所有的事情都做对。他们在很多方面也经常出错。但是现在有这么多的人在四处打探，最终通过过程新闻的方式，达成了共识，大多数好的东西被发掘出来。

在明天的活动中，看起来以下所有的事情都将成为现实:将会有一款新的 iPad 发布。它将被称为“iPad HD”，而不是“iPad 3”。它将采用“视网膜”显示屏，分辨率是之前 iPad 显示屏的两倍。它将在一些型号上采用 4G LTE 技术。它将比以前的版本有更多的内存。会稍微厚一点。它会有一个更好的摄像头。它将于 3 月 16 日发布。价格是一样的。[它会有一个按钮](https://web.archive.org/web/20221207193715/http://parislemon.com/post/18487229136/exclusive-detailed-information-about-the-new-ipad)。

见鬼，我们甚至知道一款新的苹果电视也即将上市。

所有这些知识导致了一些不可避免的事情:一些人最初的失望。现在，这似乎成了苹果公司活动中的常事。再说一次，人们越来越擅长猜测即将发生的事情，但是他们想要惊喜。然而，尽管有这些“失望”，苹果比以往做得更好。iPhone 4S 令人失望，它是苹果最畅销的设备。去年的 iPad 也是“[失望](https://web.archive.org/web/20221207193715/http://www.macobserver.com/tmo/article/ipad_2_specs_are_a_disappointment/)”。再次，巨大的成功。苹果上个季度赚了 460 亿美元。也许他们正在做空自己的惊喜能力，大量买入失望股票。但我不这么认为。

看起来，科技媒体中的一些人和真实的人类之间存在着非常真实的脱节。

当然不仅仅是高深的知识。有些人可能已经准备好了帖子，并准备现在就发布，指出明天的公告将是多么“令人失望”。反向投资者。众所周知，在每个人都在报道重大新闻的日子里，这是吸引更多浏览量的最简单方法。

现在又有了别的东西来喂养这头野兽。因为苹果继续他们疯狂的向山顶攀升，每个人都想成为第一个喊“顶！”并成功预测公司的败落。毕竟，你建立公司只是为了摧毁它们。过去的几年让这些人看起来像个十足的混蛋。不过没关系。每个人都忘了两年前谁说了什么。或者六个月前。或者一周前。重要的是这次你是否是对的。

如果只是因为我们习惯了，这一切都很好。今年变得真正荒谬的是，对于一个尚未发生的事件，对于一个尚未推出的产品，提前称之为“失望”。这几乎就像是预先准备好的反向帖子，人们现在只是为了好玩才决定在事件之前发布*。为什么要等，对吗？*

[水星新闻社](https://web.archive.org/web/20221207193715/http://www.guelphmercury.com/news/business/article/682362--ipad-3-may-disappoint-analyst):“iPad 3 可能会令人失望”

CNET :“苹果公司应该会用 iPad 3 让我们大吃一惊，但它可能不会。

[美联社](https://web.archive.org/web/20221207193715/http://www.physorg.com/news/2012-03-ipad-modest.html):“新 iPad 预计会有适度升级”

等等等等。预先失望是新的失望。

同样，我们认为我们什么都知道，但是如果我们不知道呢？或者即使我们做到了，明天真的会令人失望吗？苹果正在升级一款在市场上占主导地位的产品，这种升级方式可能会让它变得更具主导地位。如果谣言是真的，他们给了我们几乎所有我们想要的东西。说真的，我们还想要什么？“聚变能源？心灵运输？一台能预见未来的相机？，[约翰·格鲁伯](https://web.archive.org/web/20221207193715/http://daringfireball.net/linked/2012/03/06/modest)问道。

所有这些都导致了[帖子](https://web.archive.org/web/20221207193715/http://www.marco.org/2012/03/06/disappointing-ipad-3)的出现，在这些帖子中，我们中的一些人大声疾呼，六个月后，当他们宣布明天令人失望时，他们可能看起来像绝对的白痴。[贴子](https://web.archive.org/web/20221207193715/http://parislemon.com/post/18877421671/liveblogging-the-letdown)号召那些预先宣称明天会令人失望的人。现在甚至有[的帖子](https://web.archive.org/web/20221207193715/http://gizmodo.com/5890927/what-to-expect-at-apples-event-tomorrow)像这样预测帖子。我头疼。

明天会很精彩，没有人会开心。当然，除了那些最终购买了绝对糟糕的产品的消费者。
