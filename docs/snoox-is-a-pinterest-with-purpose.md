# Snoox 是有目的的 Pinterest TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/11/14/snoox-is-a-pinterest-with-purpose/>

今天，一家新的创业公司正在冲击现场，旨在抓住我们在推荐领域看到的一些成功。它是像 LiveStar 和 T2 Stamped 这样的推荐服务的结合，并像 Pinterest 一样把它们放在网络上。为了让一切回到原点， [Snoox](https://web.archive.org/web/20230209125359/https://www.snoox.com/) 努力自动整合与百思买、亚马逊和 GrubHub 等网站的“点击购买”链接。

我早上大部分时间都在使用 Snoox，我不得不说它非常有价值。你建立像食物、旅馆、餐馆之类的网页，然后在这些保护伞下推荐单个品牌或产品。

该网站从互联网上自动生成缩略图选项，要求推荐，如果可能的话，试图拉进一个“购买”链接。否则，您可以自己添加链接，并指定它是商店链接、更多信息、视频等。

你甚至可以用特殊的背景自定义某些页面，增加一点个性。毕竟，这是一种对你将与其他人分享的事物和地点的表现。脸书互联让这些联系成为现实。这是你最好的印象。

我永远无法理解 Pinterest。这看起来像是无目的的浏览和创造一些值得浏览的东西，这是为了什么？同样，*给*一个推荐(不一定得到)的艺术，似乎更适合网络。

在 Snoox 上，每个职位都有一定的责任。这些是建议。不仅如此，它们是直接给零售商的带有链接的推荐。

还有一些工作要做。某些软件会找到一个链接(通常是通过亚马逊的 Android 应用)，而其他应用则找不到。有几个餐馆它找不到，等等。无论哪种方式，随着使用的增加，这几乎会立即变得更好(不要听起来像苹果的声音)。一旦你心中有了一个很好的推荐，就很难让一个小小的失败的搜索阻止你，因为一个橙色的“新增”按钮正盯着你。

正如我所说的，*给出*建议在网络上非常有意义。事实上，我给出的建议越多，我对我喜欢和推荐的事物的想法就越多。但在紧要关头得到一封推荐信，就意味着手机无处不在。这将是对 Snoox 的真正考验。

手机网页版相当糟糕。但根据发布的消息，Snoox 正在开发新的平台。
