# Broadcom 与 Skyhook Wireless 合作，将 Wi-Fi 定位技术引入更多手机

![](img/0396050ac15fd5e398957c2412e908ed.png)

iPhone 和即将推出的 Android 手机上的一些最引人注目的应用程序利用这些设备中的 GPS 芯片来创建基于您的位置过滤信息的服务。虽然 GPS 芯片特别擅长精确定位你的位置，但也使用其他技术。例如，iPhone 使用手机信号塔三角测量和 Wi-Fi 定位。没有一种技术总是有效的，所以有几种备用方法总是一个好主意。

虽然谈得不多，但通过与天勾无线的交易，Wi-Fi 定位成为可能。现在，在众多手机中制造许多 GPS、Wi-Fi 和其他类型芯片的 Broadcom 也将 Skyhook 的 Wi-Fi 定位添加到其芯片中。这意味着未来将有更多的手机能够利用基于位置的应用。

Skyhook 拥有一个不断更新的数据库，其中包含全国数百万个地理编码 Wi-Fi 热点。因此，任何装有 Broadcom Wi-Fi 芯片的手机都可以接入 SkyHook 的数据库。(你可以使用基于相同技术的 [MyLoki 服务](https://web.archive.org/web/20221006011423/http://www.beta.techcrunch.com/2008/03/12/myloki-lets-you-tell-the-world-that-you-are-here/)，在你的笔记本电脑上测试 SkyHook 的位置检测效果如何)。
