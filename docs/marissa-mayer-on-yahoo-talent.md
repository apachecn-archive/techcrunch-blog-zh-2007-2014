# 玛丽莎·梅耶尔表示，雅虎对人才的关注正在取得成效——员工数量下降了 19%,但顶级人才流失率已经减半

雅虎(Yahoo)的玛丽莎·梅耶尔(Marissa Mayer)今天表示，该公司即将结束她担任首席执行官期间的第一次“冲刺”。她说，冲刺的重点是“让人们相信雅虎”，让公司成为一个吸引人的工作场所。

梅耶尔提供了一些对比，以说明在人才方面，情况似乎确实在好转。她说，雅虎收到的简历数量在本季度增加了两倍。她还表示，“顶尖人才”的流失率是一年前的一半。

她提到了“回旋镖”——雅虎员工离开后很快回来。梅耶尔说，雅虎上个季度 14%的员工是回旋镖。

另一方面，与去年的数字相比，看起来不错可能并不难，因为听起来公司失去了很多人(其中许多人可能是由于裁员)。该公司本季度末拥有 11，300 名员工，比去年同期下降了 19%。

现在第一次冲刺已经结束，梅耶尔说公司的重点正在转移。接下来呢。"制造漂亮的产品，并根据我们的商业策略执行得很好."她说，如果雅虎打造出漂亮的产品，它将开始看到更多的参与，这将带来收入并最终实现增长。

梅耶尔在讨论公司第一季度收益报告的电话会议上发表了评论，该报告收入持平，但每股收益高于预期。
