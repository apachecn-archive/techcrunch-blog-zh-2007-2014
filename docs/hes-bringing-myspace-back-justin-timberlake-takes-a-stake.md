# 他要让 Myspace 回归:贾斯汀·汀布莱克入股

![](img/d043adf2d8bb685039f156e8fb2da2e6.png)

你知道什么[不酷](https://web.archive.org/web/20230205002220/http://twitter.com/#!/EvelynRusli/status/86164015826485248)？3500 万美元并不酷……因此，在一部名为《社交网络*的电影中扮演脸书总统肖恩·帕克的贾斯汀·汀布莱克(Justin Timberlake)采取了一个现实生活社交网络的[所有权股份，作为新闻集团](https://web.archive.org/web/20230205002220/http://www.businesswire.com/news/home/20110629006936/en/Specific-Media-Acquires-Myspace) [Myspace 出售的一部分。](https://web.archive.org/web/20230205002220/https://techcrunch.com/2011/06/29/myspace-goes-to-specific-media-for-35m-ceo-is-out-press-release/)*

即将离任的 Myspace 首席执行官迈克·琼斯刚刚在推特上祝贺了 Timberlake 的交易。我的裁决？这实在是太不可思议了，尤其是考虑到 Timberlake 在电影中扮演的那个家伙在上周的采访中长篇大论地解释了为什么社交网络会失败。

根据[的新闻稿](https://web.archive.org/web/20230205002220/http://www.businesswire.com/news/home/20110629006936/en/Specific-Media-Acquires-Myspace)，Timberlake 将在这个陷入困境的社交网络的战略和创意方向上扮演“主要角色”。新闻稿没有提及这位流行歌星将持有多少股份。

“我们需要一个地方，粉丝们可以去那里与他们喜欢的艺人互动，听音乐，看视频，分享和发现很酷的东西，并建立联系。“Myspace 有潜力成为那样的地方，”Timberlake 说。新的 Myspace 网站将在今年夏天晚些时候的新闻发布会上亮相。

那么，这个汀布莱克新闻是不是特定媒体试图利用所有名人时刻的母亲来开始重振房地产的艰难过程？嗯，这无疑分散了人们对围绕出售和裁员的悲观故事的注意力。
