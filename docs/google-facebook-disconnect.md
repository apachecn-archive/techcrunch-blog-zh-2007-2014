# 不可避免！谷歌浏览器扩展导出你的脸书联系人

![](img/336f36f63013fcd8ce15d1163defd891.png)
好吧，看看我们这里有什么…上次我们在[脸书/谷歌的闹剧](https://web.archive.org/web/20230202222746/https://techcrunch.com/2010/11/10/google-gets-feisty-kicks-data-portability-fight-with-facebook-up-a-notch/)上登记时，脸书已经[取消了从 Gmail 导入](https://web.archive.org/web/20230202222746/https://techcrunch.com/2010/11/20/facebook-google-2/)你的联系人的选项，并且在整体上仍然坚持[“拒绝谷歌访问联系人信息”](https://web.archive.org/web/20230202222746/https://techcrunch.com/2010/11/04/facebook-google-contacts/)这一最初引发这场争斗的复杂问题。到目前为止，如果他们想从脸书向谷歌大量输出他们的朋友数据，除了使用雅虎邮箱没有其他选择。感恩节快乐，数据互惠粉丝们！一个第三方开发者已经决定建立[“脸书不拥有我的朋友”，](https://web.archive.org/web/20230202222746/http://code.google.com/p/fb-exporter/)一个 Chrome 扩展，让你轻松导出你脸书朋友的联系信息。

不管是谁建造了这个，他对自己站在哪一边都相当激烈。看一下“关于”描述的讽刺语气。

> 不管脸书怎么说，如果某人是你的“朋友”,并且你能在他/她的脸书信息上看到他/她的电子邮件地址，他们可能会同意你给他们发电子邮件。
> 
> 脸书不让你导出这些数据，所以他们希望你点击你朋友的每个页面，复制他们的电子邮件地址(或其他联系信息)，然后粘贴到你的电子邮件客户端。有点可笑？是的。"

我刚刚试了一下，效果不错，可以将你的联系人信息抓取并导出到 Gmail 和 CSV 文件中，尽管这可能需要一段时间，如果你有数百个朋友，还需要多次尝试。虽然这个扩展没有正式隶属于谷歌(或脸书)，但看到这篇文章之后它会发生什么将是令人兴奋的。

由于这是一个 Chrome 扩展，脸书可能更难阻止，因为浏览器扩展可以使抓取看起来与实际用户点击浏览器没有区别。看起来数据可移植性的柏林墙已经开始倒塌了，所以赶快在你还可以的时候导出那些朋友的联系方式吧。

**更新:**没有证实这是否是脸书方面有意的阻止，但用户报告称，该帖子发布后大约 10 分钟，扩展就停止了抓取。我已经联系了脸书和谷歌了解更多信息，如果有回音，我会更新帖子。
