# 谷歌和惊人的彩色搜索选项

我是保持事情简单的超级粉丝，但这并不意味着事情必须平淡无奇。谷歌搜索结果相当平淡。当然，有时你会得到像 YouTube 缩略图或图片这样的返回结果，但许多结果仍然只是单调的蓝色链接流。谷歌试图通过它的搜索选项，一个可扩展的功能，给你一个左侧工具栏，来打破这种趋势。但即使这样也只是一系列平淡无奇的链接。谷歌终于开始考虑改变这种状况。

今天，谷歌已经开始测试搜索选项的新外观。这为侧边栏提供了更多的视觉方法，包括颜色和图形。正如你在截图中看到的，“一切”(常规的谷歌搜索结果)、“新闻”和“博客”是一些新的可视化标签。还有一个“更多”区域显示其他内容，如“地图”

是的，这些看起来更像雅虎搜索结果。

但是这个新外观最重要的一点可能是，它作为默认视图显示给那些看到这个测试的人。是的，它不再只是一个可扩展的选项。这可能是谷歌搜索的未来吗？
