# 数据中心自动化初创公司 Puppet Labs 收购开源项目 The 木偶集体

数据中心自动化公司[木偶实验室](https://web.archive.org/web/20230106033646/http://www.puppetlabs.com/)已经收购了一个名为[木偶集体](https://web.archive.org/web/20230106033646/http://marionette-collective.org/)的开源项目。该项目也称为 Mcollective，是一个构建服务器编排或并行作业执行系统的框架。

Mcollective 的主要用途是以编程方式在服务器集群上执行操作。Puppet Labs 表示，该技术将有助于为使用 Puppet Labs 系统管理框架的公司提供多服务器和多数据中心协调。MCollective 的实时网络发现功能将从本质上简化用户如何使用 Puppet 平台上的可用数据来安排复杂的活动序列。

Puppet Labs 的旗舰产品 [Puppet](https://web.archive.org/web/20230106033646/http://www.puppetlabs.com/puppet/introduction/) 是一个开源的数据中心自动化和配置管理框架，为系统管理员提供了一个透明、灵活的系统管理平台。该解决方案目前正被 Digg、Twitter、Shopzilla、Match.com 和 Oracle/Sun 等公司使用。Puppet 已经筹集了 700 万美元的风险投资。
