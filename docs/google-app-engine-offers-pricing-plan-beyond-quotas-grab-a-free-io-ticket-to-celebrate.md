# 谷歌应用引擎提供超越配额的定价方案；获取一张免费的 I/O 入场券以示庆祝

![](img/af12e04381523a09a98e13dba4579dba.png)

谷歌为其广受欢迎的[应用引擎](https://web.archive.org/web/20230306001059/http://code.google.com/appengine/)服务发布了一个新的升级版本，允许开发者付费来扩展其应用程序的资源配额，使之超过以往由免费服务提供的配额。据谷歌称，这是该平台最常被要求的功能，因为拥有快速增长的应用程序的开发者无法超越设定的阈值来满足需求。开发人员将能够每天留出一定数量的钱，这些钱将分布在与 CPU 使用、带宽、存储空间和电子邮件相关的费用中(你可以保留当天没有花费的任何钱)。更多详情，请点击查看该公司的博客文章[。](https://web.archive.org/web/20230306001059/http://googleappengine.blogspot.com/2009/02/new-grow-your-app-beyond-free-quotas.html)

【YouTube http://www.youtube.com/watch?v=xZwH_2MVmKI&hl=en&fs=1]

为了配合今天的宣布，我们将送出三张谷歌即将于今年五月举行的 [I/O](https://web.archive.org/web/20230306001059/http://code.google.com/events/io/) 活动的门票。如果你想要一个，请在下面留下评论，告诉我们你将如何使用谷歌新的高级配额扩展，我们将挑选出最好的。请务必使用您的真实电子邮件地址，以便我们可以直接与您联系。
