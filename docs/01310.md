# 猫途鹰收购假日看门狗，评论网站 M&A 开始

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/02/15/tripadvisor-buys-holiday-watchdog-reviews-site-ma-begins/amp/>

<amp-anim src="https://web.archive.org/web/20230317203553im_/http://www.holidaywatchdog.com/images/holidays/newhw.gif" alt="Newhw" class="amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-anim>
[据](https://web.archive.org/web/20230317203553/http://tripadvisor.com/)投资博客[报道，猫途鹰](https://web.archive.org/web/20230317203553/http://www.thealarmclock.com/euro/archives/2008/02/uks_holiday_watchdog.html)已经收购了[假日看门狗](https://web.archive.org/web/20230317203553/http://www.holidaywatchdog.com/)，一家总部位于英国的用户生成旅游网站，收购金额未披露。Expedia 旗下的猫途鹰产生用户对酒店和度假的评论。由创始人克瑞丝·布朗和克里斯·克拉克森于 2004 年从 eStuff Ltd .创立的 Holiday Watchdog 称，它每月拥有 100 万独立用户，并加入了 Seatguru、CruiseCritic 和 TravelPod 的猫途鹰网络。2005 年尼尔森网评称，以阅读为基础的假日监管机构是英国十大旅游信息网站之一，但我找不到更多最近的数据。该网站不接受酒店预订，但允许人们回顾假期和分享假期经历。就设计和平台而言，这是相当“Web 1.0”的，这表明它是为其受众而不是其平台购买的。
我认为这只是今年“评论”行业将发生的一轮并购中的一轮。
