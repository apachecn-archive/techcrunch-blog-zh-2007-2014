# 迭代:SnapChat 的成功挑战了许多硅谷的假设

**编者按:** *[塞米沙](https://web.archive.org/web/20221209132038/http://blog.semilshah.com/about)是* TechCrunch *的撰稿人。你可以在推特上关注他，地址是[@ semi](https://web.archive.org/web/20221209132038/http://www.twitter.com/semil)。*

昨天是[确认](https://web.archive.org/web/20221209132038/https://twitter.com/bgurley/status/300272472396283904)——[SnapChat](https://web.archive.org/web/20221209132038/http://www.snapchat.com/)募集了一轮大资金。到目前为止，每个人都知道 SnapChat 是一种新的沟通媒介，这种现象反映了人们对短暂性的渴望，也是目前增长最快的消费应用。当我昨晚写这篇文章时，周六夜现场的塞斯·梅耶斯在节目的“周末更新”新闻部分提到了这项服务。

我们不知道 SnapChat 会有多大。所以，只是为了好玩，让我们假设 SnapChat 继续增长，实际上是“下一个大事件”，就像 Instagram 和 Pinterest 在过去几年中的发展一样。如果发生这种情况，SnapChat 有一些有趣的事实挑战了科技创业社区中许多广为接受的假设:

1.  SnapChat 的创始人并没有从大学辍学——嗯，算是吧。 [ *更新，我刚刚得知一位创始人确实大学毕业了，另一位在毕业前休假离开了校园，还差三个学分就能拿到毕业证书。这个要点是帖子中讨论最多的，所以我想在这里更正一下。在这个时代，离开去创业的大学辍学生被浪漫化，当高等教育的商业模式受到攻击，当一些最聪明的人完全放弃大学的激励存在时，SnapChat 是在它的创始人还在上大学时创建的，他们已经毕业了。*
2.  SnapChat 可能始于硅谷，但它是在洛杉矶作为一种产品成长起来的。在过去的几年里，硅谷消费者创业活动的中心已经向北转移到了旧金山，纽约市也有了自己充满活力的景象。这两个城市都享受着健康水平的科技媒体炒作。这两个沿海城市之间现在有一种天然的竞争，但 SnapChat 获得了牵引力，因为它的总部设在一个完全不同的地方，有自己的创业公司 [DNA](https://web.archive.org/web/20221209132038/https://beta.techcrunch.com/2013/02/02/the-art-of-science/) 。SnapChat 可能在任何地方都可以建立，这本身就值得注意。
3.  **SnapChat 是在短信、苹果的 iOS 以及最终的安卓的支持下成长起来的，而不是脸书或推特。**近几年来，似乎几乎每个人都认为通过现有的社交图登录新服务可以创造更好的消费者体验。相比之下，正如史蒂夫·切尼尖锐地指出的那样，SnapChat 颠覆了现有的网络，因为它的用户群不希望内容本身出现在网络上，这反过来，正如 Fred Wilson 指出的那样，使得脸书恢复 Poke 的举动显得十分滑稽..
4.  **SnapChat 是围绕数字图片打造的。**是的，这是另一家照片分享初创公司，它的“关键照片过滤器”是一项删除照片并在有人拍摄图像截图时通知用户的功能。确实是数码图片游戏中的[早局](https://web.archive.org/web/20221209132038/https://beta.techcrunch.com/2012/12/30/earlyinnings/)。现在可能是 Instagram、Pinterest、*和*SnapChat——我预计在未来几年，我们将继续看到该类别中更多的新产品和服务。
5.  **SnapChat 是为消费者打造的。**“消费者死了！”“这是向企业的转变！”而且是在硅谷大部分地区倾向于更传统的 B2B 模式的时候成长起来的。当然，创始人继续为消费者创造价值，投资者继续下注，但这是我已经听了整整六到九个月的一个迷因，来自生态系统中的每一个人——包括风险投资公司有限合伙人，他们还无法调和私人消费科技估值与公共市场能够接受的水平之间的脱节。
6.  赢得这笔交易的风投公司是沙丘上最安静的公司之一。对于 SnapChat 公司的故事来说，这不是那么重要，但值得一提。在过去的几年里，正如莱斯利·希区柯克[所写的](https://web.archive.org/web/20221209132038/https://beta.techcrunch.com/2012/08/05/vcs-and-marketing-how-the-big-players-play/)，风险投资公司向科技社区推销他们的产品，作为接触下一个 SnapChat 创始人的一种方式，这已经成为一种时尚。而且，考虑到好的内容可以很好地(并且很便宜地)扩展，公司这么做很有意义。尽管有这种趋势，Benchmark Capital 最近却反其道而行之，[几个月前关闭了其网站](https://web.archive.org/web/20221209132038/http://www.benchmark.com/)，取而代之的是简单地将其单页网站变成其投资组合公司的精选 Twitter 流。

SnapChat 最初可能被视为一个“玩具”，让人想起 Chris Dixon 关于这个主题的著名文章。正如达斯汀·柯蒂斯(Dustin Curtis)巧妙地指出的那样，最初看起来像是噱头的东西很可能是一种全新的传播媒介。再说一次，我不知道 SnapChat 是否会像 Instagram 一样一炮而红。我甚至从未使用过该应用程序或服务。但围绕该公司创始人、位置和产品的事实对许多关于消费技术成功需要哪些要素的普遍假设提出了挑战。SnapChat 表明，它可以在大学范围内完成，可以在硅谷或纽约以外完成，可以在没有脸书或推特的情况下完成，可以通过利用最重要设备(手机)上最重要的传感器(摄像头)来完成。而且，如果 SnapChat 能够继续发展，成为下一个类似 Instagram 的轰动，它也会像其他许多突破一样——通过不太可能，通过最初被忽视，以及通过克服我们许多人内心深处认为是传统智慧的许多或所有假设。
