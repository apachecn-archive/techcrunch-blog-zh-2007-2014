# Lookout 的新应用程序可以可视化世界各地检测到的移动安全威胁

[Lookout，](https://web.archive.org/web/20221208171430/https://www.mylookout.com/)一家在[为许多智能手机平台提供安全服务的公司](https://web.archive.org/web/20221208171430/https://beta.techcrunch.com/2010/05/18/lookout-lands-11-million-from-accel-khosla-and-others-for-smartphone-security-platform/)正在推出一款新的安卓应用，让你看到在世界各地检测到的移动威胁。从 Lookout Labs 推出的这款新应用基本上可以可视化移动领域正在发生的事情，并且还可以分享恶意软件与间谍软件每周威胁分布的详细信息。

作为背景，Lookout 基于网络的云连接应用程序适用于 Android、Windows Mobile、黑莓和[最新的 iOS 设备](https://web.archive.org/web/20221208171430/https://beta.techcrunch.com/2011/10/18/lookout-finally-brings-intelligent-smartphone-security-and-tracking-application-to-ios/)，帮助用户防止丢失手机，并识别和阻止消费者手机上的威胁。用户只需将该软件下载到一个设备上，它就会像下载到电脑上的安全软件一样，充当跟踪应用程序、数据备份和病毒保护程序。

Lookout 从其移动威胁网络收集数据，这是一个基于云的网络，它不断分析全球威胁数据，以识别并通过无线应用程序更新快速阻止新的威胁。该网络包括全球超过 100 万个应用和 1500 万个用户设备。

使用 Lookout 移动威胁跟踪器，您现在可以看到 Lookout 每天通过网络识别和捕获的数以千计的威胁。在移动威胁跟踪器中，您可以快速查看三大趋势威胁。例如，如果你点击应用程序中的名称，你可以了解每个威胁的更多信息。例如，本周的头号威胁 RuPaidMarket 伪装成一个有用的应用程序，但实际上在不允许你选择退出或让你知道你将被收费的情况下发送高级短信。

数据每小时更新一次，因此您可以看到移动威胁随着时间的推移而出现和消失。该应用程序显示，过去一周的三大移动威胁是 RuPaidMarket、Legacy 和 Depositmobi。目前恶意软件和间谍软件的比例是 60%恶意软件和 40%间谍软件。

虽然这款应用可能不是日常检查的应用，但最新的 Lookout 产品可能是一种有用的方式，可以定期查看在任何给定时间你的 Android 手机可能面临的威胁。Lookout [预测](https://web.archive.org/web/20221208171430/https://beta.techcrunch.com/2011/12/13/lookouts-2012-mobile-security-threat-predictions-sms-fraud-botnets-and-malvertising/)恶意软件、间谍软件和其他移动安全风险在 2012 年只会增加，因此用户在未来一年必须更加警惕保护他们的智能手机。
