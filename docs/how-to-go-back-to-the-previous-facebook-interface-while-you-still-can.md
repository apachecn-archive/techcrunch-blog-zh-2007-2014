# 如何回到以前的脸书界面(当你还可以的时候)

如果你不满意[脸书对](https://web.archive.org/web/20230325040403/https://techcrunch.com/2011/09/20/facebook-news-feed-gets-smarter%E2%80%94%C2%A0and-the-ticker-makes-its-big-debut/)新闻订阅和添加“滚动条”所做的改变，有一个简单的方法可以恢复到“旧的”[脸书](https://web.archive.org/web/20230325040403/http://www.crunchbase.com/company/facebook)用户界面。

只需进入你的账户设置，将你的语言改为**，除了‘英语(美国)’**。如果你只会说英语，你可以**简单地选择‘英语(英国)’**，它会变回以前的布局。

抱歉，如果这已经在其他地方报道过了(我没有立即在通常的可疑新闻网站和博客上找到任何报道)，但这目前正在 Twitter 上流传。

显然，脸书在社交网络服务提供商覆盖的所有地区/语言中推出[新用户界面](https://web.archive.org/web/20230325040403/http://blog.facebook.com/blog.php?post=10150286921207131)只是一个时间问题，所以对于那些难以接受和接受[变化](https://web.archive.org/web/20230325040403/https://techcrunch.com/2011/09/20/facebook-news-feed-gets-smarter%E2%80%94%C2%A0and-the-ticker-makes-its-big-debut/)的人来说，这只是一个临时的“修复”。你是富足的。

有些人告诉我这不再管用了，但对我来说管用，而且从表面上看，对大多数人来说管用。如果这对你不起作用，道歉并记住，雨过天晴。
