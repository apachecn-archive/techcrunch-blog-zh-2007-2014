# 直播博客脸书的马克扎克伯格在 Web 2.0:收入，筹款，增长和产品 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/11/06/liveblogging-facebooks-mark-zuckerberg-at-web-20/>

今天下午 2 点左右，脸书首席执行官马克·扎克伯格和约翰·巴特利一起站在旧金山 Web 2.0 峰会的讲台上。

[就像昨天的杨致远](https://web.archive.org/web/20221007030722/http://www.beta.techcrunch.com/2008/11/06/the-full-jerry-yang-interview-at-web-20/)一样，扎克伯格也有很多[关于脸书及其未来的话题要和](https://web.archive.org/web/20221007030722/http://www.beta.techcrunch.com/2008/10/31/facebooks-growing-problem/)谈。

我们的实时采访记录如下。

巴特尔谈到脸书这些年来发生了多大的变化，开玩笑说扎克伯格现在穿网球鞋，而不仅仅是凉鞋。

第一个问题，巴特尔问扎克伯格关于新的[融资传闻和迪拜之旅](https://web.archive.org/web/20221007030722/http://www.beta.techcrunch.com/2008/10/31/facebooks-growing-problem/)。扎克伯格保持沉默。在回答“你需要钱吗？”扎克伯格说“不”

巴特尔深入挖掘，询问脸书是否会比他们一年前想象的更快地耗尽资金。Mark 说网站的发展速度比他们预期的要快得多，但是通过用户翻译网站的国际发展是一条有效的途径。马克还表示，该公司有两条非常强劲的收入线——直销和在线销售。巴特尔要求他分解这两个流，扎克伯格不予置评。他还提到了微软的收入，但只是说它在总收入中所占的比例在下降。

扎克伯格在评论微软的投资时说:“很多人都关注微软支付的价格。”巴特尔问脸书如何才能成长到这样的估值。扎克伯格表示，他们接受了所能得到的最佳估值，他们做这笔交易是有意义的。他说，它们几年内不会上市。“我们没有感到任何达到 150 亿美元(估值)的压力，”他说。

![](img/df00567d9cae78aa62450cb8af290aba.png)

马克证实“大约 700 多”人在脸书工作，并表示他们会继续招聘，尤其是工程师。他还表示，他们正在建立国际销售办公室。马克说，现在有 7%的法国人生活在脸书，如果明年有 30%，他也不会感到惊讶。他说，他们可能会在那里开设一个销售办事处。

关于脸书连线，马克说现在任何人都可以申请加入。

在回答关于脸书有多封闭的问题时，他说随着时间的推移，他们可能会开放，但用户现在更喜欢使用脸书的界面。他说，随着时间的推移，系统趋向于开放，并以个人电脑行业为例。

马克说，他们今天的目标是让人们进入脸书，让他们放心地把自己的个人信息放到网上与朋友分享。

他表示，该平台的下一个版本是脸书连接，为人们提供第三方网站上的脸书工具。“随着时间的推移，我们将看到这种非常自然的分散和分散，”他说。

马克说，脸书平台早期实施的一个缺陷是，应用程序开发人员过于专注于让用户添加它，这使他们登上了榜首。现在，脸书推广实际上有用户参与的应用程序。四个例子:原因，它正以“每天几十万人”的速度增长

他表示，脸书联通是进一步的发展。他说，他们放慢了脸书联通的推出速度，以尽量减少以后痛苦的变化。

关于推特，巴特尔问“推特只是脸书的一个特色吗？”

巴特尔将话题转向在线广告。马克表示，他们有第二代社交广告，让用户将礼物或内容与广告单元一起传递。

巴特尔询问脸书被公司禁止，指出军队认为脸书是一个问题。马克说他们看到了相反的趋势。他说，以前屏蔽脸书的公司现在允许他们使用它。

马克强调，在平台方面，它仍然是非常早期的，并且不排除在 OpenSocial 上与谷歌合作。

马克提到 50%的脸书用户每天都活跃在网站上。
