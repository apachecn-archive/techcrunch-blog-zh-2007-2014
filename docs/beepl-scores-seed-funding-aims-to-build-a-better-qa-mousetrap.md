# Beepl 获得种子基金，旨在建立一个更好的问答捕鼠器

Beepl ，一家标榜自己为专业技术平台和语义技术公司的初创公司(不要再板着脸了！)，已经从总部位于布拉格的风险投资公司 [Credo Ventures](https://web.archive.org/web/20230203091210/http://www.crunchbase.com/financial-organization/credo-ventures) 筹集了种子资金。Beepl 的联合创始人是首席执行官[史蒂夫·奥黑尔](https://web.archive.org/web/20230203091210/http://www.crunchbase.com/person/steve-ohear)，他[刚刚离开 TechCrunch](https://web.archive.org/web/20230203091210/http://eu.beta.techcrunch.com/2011/06/27/you-know-youre-in-a-bubble-when-tech-journalists-join-startups-bye-steve/) 去[体验一下科技创业的生活](https://web.archive.org/web/20230203091210/http://www.ohear.net/2011/06/27/why-i%E2%80%99m-leaving-techcrunch/)，甚至他也没有告诉我们他们已经筹集了多少资金，只是说是“六位数”。

他应该知道得更清楚，因为我们从消息来源得知，筹集的资金总计约 40 万美元(对不起，史蒂夫！).

Beepl 成立于 2011 年 4 月，由来自斯洛伐克、英国和捷克共和国的创始团队组成，总部位于伦敦。

该公司将在接下来的几个月内处于隐形模式，但据我收集的信息，他们正在建立一个平台，让用户能够获得他们问题的高质量答案，以及相关主题专家、爱好者和他们的社交图的意见。

用公关术语来说，他们计划实现这一目标的方式听起来像这样:

> “Beepl 通过一个定制的语义引擎来实现这一点，该引擎采用自然语言处理和机器学习来理解问题相关的主题以及用户的兴趣和专业知识。"

换句话说，或者我们敢说，一个更好的问答捕鼠器？我们很快就会知道了。
