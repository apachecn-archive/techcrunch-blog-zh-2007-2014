# 祝贺 BIMA 获奖者，现在我们能有一个很酷的技术奖吗？

衷心祝贺英国互动媒体协会(又称 BIMAs)年度大奖[的获奖者](https://web.archive.org/web/20230120193130/http://www.bima.co.uk/)。总之，第四频道新媒体赢得了 4oD*(点播电视追看服务)的“大奖”，而广告代理阳狮对话获得了两个奖项，代理 GT 获得了三个奖项。第四频道因“这个新界面的独创性和简单性”而获奖这很方便，因为 BIMAs“旨在认可和奖励交互设计中的优秀创意”。总共有 23 个奖项。[军职](https://web.archive.org/web/20230120193130/http://www.armyjobs.mod.uk/)(阳狮出品)和[伯恩茅斯水族馆的互动潜水笼](https://web.archive.org/web/20230120193130/http://www.bima.co.uk/membershowcase/0501080D1F12070C/flaming-pear/the-interactive-dive-cage/)(火梨出品)因“挑战了我们与媒体互动的传统思维”而获奖其余的获胜者将在周五的[这里](https://web.archive.org/web/20230120193130/http://www.bima.co.uk/)揭晓。

BIMAs 已有 23 年的历史，被称为“英国唯一庆祝数字创意卓越和最佳实践的重大数字创意大奖”。公平地说，董事长保罗沃尔什的热情和精力让他们重新出现在了地图上。但它们是英国众多在线媒体/营销/创意类奖项之一。NMA 有效性奖“表彰那些对交互式媒体的潜力有着深刻理解的公司。”[互动媒体和营销奖](https://web.archive.org/web/20230120193130/http://www.imaawards.co.uk/)表彰和庆祝使用互联网、移动和互动电视进行营销和广告的最有效范例同样相似的还有“T4 革命奖”(Revolution Awards)，它“奖励数字营销领域的佼佼者。”此外，还有美国在线出版商协会(Association of Online Publishers Awards)颁发的“数字出版各个领域的卓越表现”奖项。就在不久之前，美国还有个网商大会(Bafta Interactives，交易代码:T9)，但为了争夺一个视频游戏奖项，该大会于去年闭幕。(看来电影人在网络方面仍然有问题)。

换句话说，英国的媒体和在线营销行业有很多酷劲十足的颁奖典礼，但必须申明的是，这些行业都有自己独特的精彩之处，而且都认可我们在这些领域拥有的非凡才能。我甚至以亲自主持古怪的颁奖典礼而闻名……

但同样的，英国也没有多少奖项认可酷劲十足的科技公司，比如 LastFM，这是一个伟大的英国故事，它没有获得 BIMA 奖(不可否认的是，它们可能不适合这个奖项)，但却建立了令人惊叹的业务，并在去年以 1.4 亿英镑的价格卖给了 CBS。这是一项了不起的成就。我敢打赌，还会有很多类似的成就，可能不属于传统的“媒体”或“营销”范畴，但确实值得肯定。

在科技行业，我们在“妆点”颁奖晚会上最多能想到的就是英国计算机协会 IT 产业奖。又或者是一个笑容可掬、语无伦次的《韦比》版本。

我认为现在是时候来个硬碰硬了，不是吗？

*注:第 4 频道的 4oD 互联网服务使用与 BBC iPlayer 完全相同的技术(Kontiki Delivery Manager 和 Microsoft DRM)，天空随时服务也是如此。Kontiki 是 VeriSign 旗下的一家美国公司。或许大奖赛应该分享？
