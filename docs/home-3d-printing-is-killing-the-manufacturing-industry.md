# 家用 3D 打印正在扼杀制造业 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/02/home-3d-printing-is-killing-the-manufacturing-industry/>

如果你关注新闻，一群黑客最近租用了一台 3D 打印机来制造一把真正的工作枪。这个名为 [Defense Distributed](https://web.archive.org/web/20221226032424/http://defensedistributed.com/) 的组织开始了这个项目，但是一旦租赁者 [Stratasys](https://web.archive.org/web/20221226032424/http://www.stratasys.com/) 发现了他们正在建造的东西，他们就收回了这台打印机。

写着[危险的房间](https://web.archive.org/web/20221226032424/http://www.wired.com/dangerroom/2012/10/3d-gun-blocked/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+WiredDangerRoom+%28Wired%3A+Blog+-+Danger+Room%29&utm_content=Google+Reader):

科迪·威尔森计划在未来几周制造并测试一把 3d 打印手枪。现在，这些计划已经被搁置，因为桌面制造公司 Stratasys 取消了为维基武器出租的打印机租约，维基武器是威尔逊领导的互联网项目，致力于共享 3d 打印枪支的开源蓝图。Stratasys 甚至派了一个小组从 Wilson 的家里查封了这台打印机。

Stratasys 回应问题称:“Stratasys 保留拒绝订单的权利。国防分布式成员，像任何美国公民一样，能够遵循完善的联邦和州法规，在这个国家制造、分发或采购枪支。”

这整个混乱——这才是真正的混乱——笼罩了一场非常有趣的辩论，随着 3D 打印越来越成为主流，我们都将进行这场辩论。虽然我们目前认为家庭 3D 打印是无可救药的书呆子的领域，但事实是 3D 打印机已经度过了它的婴儿期，现在正处于所有范式转换技术所面临的细长而尴尬的青春期阶段。

2012 年的 3D 打印相当于 1982 年的家庭打印。那些年龄足够大，还记得拖拉机送纸，甚至是昔日糟糕透顶的热敏打印机的人，都与这种情况有关。3D 打印机虽然在技术上令人印象深刻，但对普通消费者来说并不那么令人兴奋。然而，到 1984 年，Broderbund 推出了 Print Shop，并使普通的点阵打印机更加引人注目。分辨率仍然参差不齐，直到大约十年后我们能够打印出信纸质量的页面，人们才把打印机本身视为“威胁”。简而言之，打印机悄悄靠近了我们，就像 3D 打印机一样。

然而，就这个问题的直接类比而言，我们可以指出家庭录音和 CD 刻录。人们的预期是——这是一个合理的预期——家庭 3D 打印将变得如此之好，以至于它生产的物品将与我们现在购买的简单物品相媲美。现在一个 Makerbot 需要几个小时才能打印出最基本的产品，但是当这些时间减少到几分钟时会发生什么？如果我们可以在厨房打印一套宜家银器，会发生什么？再说一次，也许我们还远远没有能力做这些事情，但是 Broderbund 的程序员不知道他们的软件很快就会被真正的家庭桌面出版和打印工具所取代，这些工具可以在几秒钟内创建一个精美的非常漂亮的产品。

这把枪怎么样了？有许多反对制造这种产品的论点，但没有一个站得住脚。第一，合法性。“枪”是一种通过推进剂的有限燃烧以高速发射一枚或多枚射弹的武器我们可以用一根金属管和一颗钉子制作一个周六夜特辑，或者我们可以在地下室制作一把枪，前提是它不用于出售或交易。这些人计划使用租赁的打印机来建造它*和*之后发布计划是一个完全合法的行动。正如[无政府主义者的食谱](https://web.archive.org/web/20221226032424/http://en.wikipedia.org/wiki/The_Anarchist_Cookbook)向我们展示的，分享制造枪支的过程是受保护的言论。不这样想会给信息的自由传播带来寒蝉效应(我们可以反驳说，如果这些人在制造原子弹，我们完全有权利让他们闭嘴，但原子弹表面上是非法的)。但是，最终，他们所做的并不违法。

然后是财务问题。如果谁有时间，有一点金属，有意志就能印枪，史密斯&韦森公司不会倒闭吗？我发现表面上这是一个非常似是而非的论点，但它值得进一步分析。人们称 3D 打印机为“盗版机器”可以说这是一个沉重的打击，但是如果我能打印出一个完美的复制品，比如说，一个米老鼠摇头娃娃*和*分享这个摇头娃娃的计划，那么迪士尼就没有摇头娃娃的销售了。然而，正如我们从家庭录像中学到的，数字和物理之间的接口是一个难以逾越的障碍。音乐和电影盗版猖獗，因为它没有摩擦。另一方面，打印一件物品需要大量的投资。除非你打算做一个恶心的摇头娃娃，否则可能不值得花时间和材料把它粘在迪士尼上。毕竟，家庭录音并没有扼杀音乐产业。Napster 有。

最后是道德问题，我们在上面已经部分讨论过了。有人认为枪支是我们与生俱来的权利，也有人认为枪支应该被取缔。但是那些你不需要买的枪呢？那些无法追踪，对开枪者和目标有潜在危险的枪支呢？我想，我们难道没有保护“孩子们”的道德责任吗？

我们知道，但这不是我们要做的。明确地说，我反对在我的家里使用枪支，但我不会嫉妒任何个人或团体尝试 3D 打印枪支的权利。硬件的创新来自实验。没有它，我们一个接一个地走进死胡同，什么也学不到。虽然我不同意 3D 打印枪是我家庭的重要组成部分，但我确实同意它是我们修补权利的重要组成部分。人类在发明犁之前就发明了武器，这是肯定的。但是如果没有这些武器，我们就不会扩展到如此遥远的地方，进入农业时代。

Stratasys 在这里失败了。他们声称非法是因为害怕诉讼。如果这个团体什么都没说，只是印出他们的部分，Stratasys 也不会知道。无论是法律、金融还是道德问题，任何黑客都有权利攻击，反过来，他们也要为自己创造的东西负责。正如霍尔写伏尔泰时所说的，“我不同意你说的话，但我誓死捍卫你说话的权利。”把“说”换成“建”，我们就有了这个看似无法解决的难题的答案。
