# BBM 将在未来几个月推出基于 Android 的诺基亚 X 和 Lumia 新设备 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/24/bbm-comes-to-new-android-based-nokia-x-and-lumia-devices-in-coming-months/>

诺基亚和 Windows Phone 用户一直在要求它——今天早些时候[诺基亚](https://web.archive.org/web/20230129073611/https://twitter.com/nokia/status/437853442862612481)和[黑莓证实](https://web.archive.org/web/20230129073611/http://blogs.blackberry.com/2014/02/bbm-windows-phone/) it:免费消息应用 BBM 将登陆诺基亚设备，基于微软 Windows Phone 的 Lumia 设备和基于今天早些时候宣布的 Android 分叉版本的新[诺基亚 X 手机](https://web.archive.org/web/20230129073611/https://techcrunch.com/2014/02/23/nokia-x/)。黑莓表示，这些应用将在“未来几个月”上线，最迟不会晚于今年夏天。

该应用将首先在诺基亚商店上线。尚不清楚它是否会扩展到其他 Windows Phone 设备，尽管这是其他应用程序遵循的轨迹。

这一举动对黑莓、诺基亚和微软来说意义重大，正值人们对获得数亿用户的即时通讯应用程序的关注热潮，其中最大的应用程序之一 WhatsApp 上周以 190 亿美元的价格卖给了脸书。

有趣的是，黑莓似乎将这一发展视为对其企业业务的推动。黑莓全球企业解决方案总裁 John Sims 表示:“随着数百万人使用我们的移动平台与朋友或同事聊天和联系，BBM 越来越受欢迎，我们很高兴很快将欢迎 Windows Phone 和 Nokia X 用户加入 BBM 社区。

然而，事实上，像 BBM 这样的应用程序已经在消费者中占据了一席之地。

对于诺基亚和微软来说，这款应用为公司提供了另一个关键应用，以继续充实其产品组合，并向现有和潜在的新用户证明，设备与他们更大、更成熟的竞争对手一样好，一样有用，即基于安卓和苹果 iPhones 的设备——这是应用的黄金标准，通常也是开发者的首选平台。

正如史蒂芬·埃洛普今天早些时候指出的那样，诺基亚还特别将这款应用视为瞄准发展中市场的另一种方式，诺基亚一直专注于用更低成本的设备赢得新的智能手机用户，以平衡许多成熟市场现在已经饱和的事实，即忠实的 Android 和 iPhone 用户。“我们致力于下一个 10 亿，”他今天谈到这一战略时说，这一战略也得到诺基亚 X 分叉安卓手机的支持。

像 BBM(和 WhatsApp)这样的聊天应用已经成为人们保持联系的重要方式。这是因为像 BBM 这样的聊天应用程序比短信功能更强大——允许进行多次聊天、群发消息、语音通话和贴纸等功能——而且在许多国家，以较小的增量购买比购买完整的语音和短信计划更容易。

对于黑莓来说，这是一个明显的举措，有助于扩大其 BBM 业务，继续为已经迁移到其他平台的前黑莓设备用户以及新用户提供一个交流平台。除了说“数百万人”正在使用这项服务之外，今天 BBM 的用户总数没有更新。
