# Twitter 为 iPad 推出了改进的 HTML5 版 Twitter.com

因为 Twitter 的实时聊天流需要更多的方式在更多的设备上使用，该公司刚刚宣布，T2 开始为 iPad 推出基于 HTML5 的 Twitter.com，所有 iPad 用户应该在一周内都可以使用。

该应用程序是完整的 HTML5，是对目前平板电脑上 Twitter 移动网站单一面板体验的改进。Twitter 移动工程师 [Manuel Deschamps](https://web.archive.org/web/20230203044202/http://twitter.com/#!/manuel) 告诉我，毫不奇怪，它让人想起 Twitter.com 和移动应用的混合体，基于 Twitter 的 iPhone 和 Android html 5 客户端。HTML5 网络应用将在本周内向所有用户推出，击败尚未推出自己的 HTML5 计划的脸书斯巴达项目。

Twitter 代表 Carolyn Penner 说:“HTML5 应用程序为那些喜欢使用浏览器而不是本地应用程序的人提供了一个很好的选择。它的两栏视图与你习惯的 Twitter 体验(在 Twitter.com 上)一致，它利用了触摸手势和其他设备功能。”
