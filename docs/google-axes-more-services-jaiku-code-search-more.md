# 谷歌推出更多服务:Jaiku、Buzz、代码搜索等等

根据谷歌产品副总裁布拉德利·霍洛维茨在[官方公司博客](https://web.archive.org/web/20230205045538/http://googleblog.blogspot.com/)上的一篇新帖子，谷歌正在对其他几个产品和服务实施致命一击，包括其代码搜索引擎、Buzz、Jaiku、iGoogle 功能和谷歌搜索的大学研究计划，后者为少数学术机构提供谷歌搜索结果的 API 访问。

以下是谷歌新关闭的详细信息:

*   ***代码搜索**，这个旨在帮助人们在全网搜索开源代码的工具，将于 2012 年 1 月 15 日与代码搜索 API 一同关闭。*
*   几周后，我们将关闭 Google Buzz 和 Buzz API，转而专注于 Google+。虽然在那之后人们显然不能创建新的帖子，但他们将能够在他们的[谷歌个人资料](https://web.archive.org/web/20230205045538/http://profiles.google.com/me)上查看他们现有的内容，并使用[谷歌外卖](https://web.archive.org/web/20230205045538/http://www.google.com/takeout)下载。
*   我们在 2007 年收购的产品 Jaiku 将于 2012 年 1 月 15 日关闭，它可以让用户向朋友发送更新。我们将努力使用户能够从 Jaiku 导出他们的数据。
*   几年前，我们让人们能够在 iGoogle 上进行社交互动。随着我们对 Google+的新关注，我们将于 2012 年 1 月 15 日移除 iGoogle 的社交功能。iGoogle 本身，以及非社交 iGoogle 应用，将保持原样。
*   *谷歌搜索的**大学研究项目将于 2012 年 1 月 15 日关闭，该项目为少数获得批准的学术研究人员提供访问我们搜索结果的 API。***

谷歌还表示，它将关闭谷歌实验室网站，就像之前承诺的那样。那个网站从今天起将会消失，所以现在就开始你的最后一次访问吧。此外，Boutiques.com 和 Like.com 现在将被谷歌产品搜索取代。

这些最新的关闭紧随该公司其他几次服务关闭之后，该公司以前的策略似乎包括将一束意大利面条扔在墙上，看看粘在墙上的是什么。六月，它关闭了谷歌健康服务。八月，谷歌[终止了收购 Slide】的项目，九月，](https://web.archive.org/web/20230205045538/https://techcrunch.com/2011/08/25/slide-google-bloodbath/)[终止了土豚](https://web.archive.org/web/20230205045538/https://techcrunch.com/2011/09/02/google-kills-aardvark/)。尽管进行了所有的精简和关闭，谷歌之前已经声明它的“20%时间项目”不会有任何进展。
