# 《生化奇兵》的创作者:新的游戏机会“对每个人都不好”

肯·列文因创作了《T2》生化奇兵而闻名，这是一部更具电影性和高制作价值的系列电影。所以一开始听到他说新一代更强大的游戏机会成为这样的问题似乎有点奇怪。

在[的非理性游戏播客](https://web.archive.org/web/20221206205947/http://irrationalgames.com/insider/irrational-interviews-6-randy-pitchford/)上，他说:

> 在这一点上，作为一个开发者，我没有欲望，作为一个游戏玩家，我也没有欲望看到下一代出现在我现在坐的地方。
> 
> 在这个平台上做游戏还是要花我们一大笔钱的。如果他们要扩大规模，提高艺术，增加内容，我不知道如何制作并以低于 100 美元的价格卖给任何人。谁想这么做？这对每个人都不好。

这才是真正的观点，对吧？这是我提出的关于 PSP2/NGP 的异议。投入到 AAA 游戏中的工作量和金钱是相当疯狂的，如果你的平台是围绕这类游戏构建的，那么就有一个权衡。现在看来，我们或多或少已经达到了游戏机和开发工具在合理预算下所能做的极限。

就我个人而言，独立游戏和可下载游戏正在疯狂起飞，这一事实让我在这场辩论中深受鼓舞。你不需要 3000 万美元的预算来制作一个伟大的游戏，虽然大人物总会有一席之地，但这些天的词似乎是“多样化”索尼和微软都把这放在心上，但有趣的是，任天堂似乎陷入了更大的困境。也就是说，实际上任天堂将从新游戏机中获益最多。朋友们，我们生活在一个有趣的游戏时代。尽情享受吧。
