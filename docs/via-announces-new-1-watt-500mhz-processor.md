# 威盛宣布推出新的 1 瓦 500MHz 处理器

![via](img/0df4932fd39196d8ea3f70582543bc3b.png)

向威盛的“世界上最节能的 x86 CPU”问好。它是一个 1 瓦、500MHz 的处理器，具有 400MHz 的前端总线，可以与威盛的 CX700/M 一体化数字媒体芯片组配对，创建一个功耗低于 10 瓦的完整系统。

这款无风扇处理器的尺寸仅为 21 平方毫米，将用于“超冷、超安静和高可靠性性能至关重要的应用”，如嵌入式系统和一体机。

比如说，一个 500MHz 的处理器不足以在 UMPC 上运行 Windows Vista，但对于运行嵌入式版本的 Linux 或类似的专有操作系统来说还是不错的。这里最大的吸引力是延长电池寿命。典型的 UMPCs 消耗大约 20-25 瓦的功率，因此如果您使用这种新的 VIA 芯片组的系统，总功率大约为 10 瓦，您将获得大约两倍的充电时间。
