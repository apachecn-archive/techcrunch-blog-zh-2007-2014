# 马克·扎克伯格首次公开募股后第一次采访的最佳片段[视频]

今天下午，在旧金山举行的 [TechCrunch Disrupt](https://web.archive.org/web/20230320085700/https://techcrunch.com/events/disrupt-sf-2012/) 会议上，脸书董事长兼首席执行官[马克·扎克伯格](https://web.archive.org/web/20230320085700/http://www.crunchbase.com/person/mark-zuckerberg)与 TechCrunch 创始人[迈克尔·阿灵顿](https://web.archive.org/web/20230320085700/http://www.crunchbase.com/person/michael-arrington)坐在一起，接受自该公司 5 月份备受关注(也备受批评)的首次公开募股[以来的首次采访](https://web.archive.org/web/20230320085700/https://techcrunch.com/2012/05/18/facebook-share-open-10-5-higher-at-42/)。

不用说，这是一件大事:会议大厅只是一个站立的地方，脸书的股票在盘后交易中明显上涨，似乎从扎克嘴里说出来的每一句话都成了新闻头条。

但由于 TechCrunch TV 和 [SnappyTV](https://web.archive.org/web/20230320085700/http://www.snappytv.com/corp/) 的人们的辛勤工作，我们可以同时展示*和*告诉你扎克大亮相的最佳原声片段。在上面嵌入的视频中，你会发现今天下午台上采访的所有最佳交易所的综述，一个接一个——从脸书下跌的股价如何实际上有利于其招聘,到[他不再向脸书提交代码](https://web.archive.org/web/20230320085700/https://techcrunch.com/2012/09/11/the-ceo-of-facebook-still-codes-on-the-side-but-tends-to-break-things/)的原因，到那些喋喋不休的[脸书电话谣言](https://web.archive.org/web/20230320085700/https://techcrunch.com/2012/09/11/mark-zuckerberg-a-facebook-phone-just-doesnt-make-any-sense/)等等。
