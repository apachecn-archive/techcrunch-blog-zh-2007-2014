# 东芝想象中的可穿戴计算手套赢得愚人节

如果有一个总的管理机构给愚人节恶作剧的人颁奖，我希望东芝也能得到一个。

该公司今天宣布了[数字](https://web.archive.org/web/20221006170833/http://www.toshiba.com/us/digit)，抛弃了通常非常无聊的商业风格，向小工具世界介绍了一些真正有趣的东西。

手指是一副可穿戴的计算手套。

它有一个“双手手势用户界面”，以及 3200 万像素的摄像头，4K 超高清摄像机和 4K 虚拟视网膜显示器。要使用屏幕，你真的要用手套遮住脸，就像那只看不见邪恶的猴子一样。使用电话时，你把手举到脸前，就像一个孩子假装在打电话一样。要听音乐，只要把拇指塞在耳朵里就行了。

太可爱了。

它还配有 64GB 的 SSD 存储(别问了)，1 TB 的云存储，所以你的东西总是在你的手掌里。(看到了吗？我也会开玩笑。)

但是这个数字最好的部分是社交媒体的整合。当你与另一位数字用户牵手时，它会自动改变你与他们的关系状态。
