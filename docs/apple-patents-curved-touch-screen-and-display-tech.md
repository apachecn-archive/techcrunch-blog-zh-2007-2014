# 苹果专利弯曲触摸屏和显示技术

> 原文：<https://web.archive.org/web/http://techcrunch.com/2013/12/10/apple-patents-curved-touch-screen-and-display-tech/>

# 苹果申请了曲面触摸屏和显示技术专利

苹果公司已经获得了曲面触摸屏和显示器的专利，据苹果内幕报道，该报道描述了一种制造曲面触摸感应玻璃的系统。苹果申请专利的方法旨在创建一个弯曲的屏幕表面，该表面保持触摸敏感，而不会使显示器产生的图像变形或扭曲，并将能够用于显示器、触摸感应鼠标(如当前的 Magic Mouse)触摸板和其他设备。

苹果的技术不同于一些现有的曲面玻璃技术，因为通常情况下，为设备增加触摸灵敏度的基板是平坦的，无论覆盖它的玻璃面板是否弯曲。因此，触摸灵敏度受到影响，因为有时人的手指实际上比看起来更远离触摸板，或者比传统的非弯曲设备更远离触摸板。

该专利还包括一种屏幕表面曲率变化的方法。具体来说，它展示了看起来像一系列气泡的东西，例如，你可以看到它们被用来在软键盘上创建凸起的图案。这可以方便地在显示器等设备上创建触摸感应按钮界面，以控制亮度和音量等事情，或者为触摸鼠标等设备添加更明显的输入方法，而不会偏离全触摸设计。

苹果还没有像它的一些竞争对手[包括 LG](https://web.archive.org/web/20230326075244/https://techcrunch.com/2013/10/13/lg-may-fire-back-at-samsung-with-its-own-curvy-smartphone-next-month/) 和[三星](https://web.archive.org/web/20230326075244/https://techcrunch.com/2013/10/08/that-curved-display-smartphone-from-samsung-is-real-meet-the-galaxy-round/)一样采取任何公开的行动来制造曲面玻璃移动设备，但它正在围绕不符合正常设计的屏幕制造一些嗡嗡声。彭博 11 月份的一份报告甚至暗示，该公司正在开发一款曲面玻璃 iPhone，将于 2014 年底发布，并且还在为未来的设备打造更好的触摸输入。一项为曲面玻璃设备设计更好的触摸屏的专利肯定符合这些报道。

曲面玻璃可能是一种时尚，但尚不清楚这种技术在满足消费者需求方面的优势。尽管如此，这显然是一个值得关注的领域，因为每个人和他们的兄弟似乎都在至少将 R&D 的支出投入到制造下一代显示器而不仅仅是平板显示器上。
