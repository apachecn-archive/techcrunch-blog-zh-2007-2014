# 据《华尔街日报》报道，在谈判破裂前，微软差点收购诺基亚

微软和诺基亚在过去的几年里变得相当亲密，当这家芬兰公司宣布决定使用 Windows Phone OS 来驱动其智能手机时，许多人猜测这是即将到来的收购的第一个序曲。据《华尔街日报》报道，这显然是事情的发展方向，但拟议中的交易已经破裂。

《华尔街日报》说，谈判最早本月就在进行，但谈判已经破裂，不太可能再次恢复，这意味着 Microkia 或 Nokrosoft 不太可能很快实现。报告称，在此之前的谈判已经非常深入，但最终因为 MS 和诺基亚面临的财政困难而破裂。

据报道，微软结束了这场求爱。诺基亚一直在苦苦挣扎，Windows Phone 计划也没有起到多少支撑其持续亏损的作用。诺基亚[上个季度没有达到分析师的预期](https://web.archive.org/web/20221006073434/https://beta.techcrunch.com/2013/04/18/nokia-misses-with-sales-of-7-6b-but-beats-on-0-03-loss-per-share-5-6m-lumias-sold/)，并公布了运营亏损，尽管比去年同期低得多。然而，该公司的设备销量连续下降，同比也有所下降。

从本质上讲，微软和诺基亚能够有一个延长的约会期，微软有足够的时间来认识到选择婚姻不太可能扭转任何一方的命运。事实上，无论是考虑合并的事实，还是合并失败的事实，对于那些一直在关注诺基亚在过去几年的发展，以及 Windows Phone 作为一个移动平台的发展的人来说，都不应该感到惊讶。

现在剩下的就是看微软在智能手机方面的下一步是什么，以及它是选择尝试以自己的项目 Surface style 来启动事情，还是寻找像诺基亚这样的人，他们可能会在为其移动操作系统点火方面做得更好。
