TripIt打包退出——被Concur以高达1.2亿美元收购

Concur ，一家整合旅行和费用管理解决方案的上市供应商，今天早上宣布已经同意收购 TripIt ，该公司帮助旅行者轻松组织和分享旅行计划。

Concur最初支付了大约8200万美元的预付现金、股票和未授予的限制性股票单位——随着时间的推移，额外的对价可能会使总价值达到1.2亿美元。财务状况的更多细节如下。

当一切烟消云散时，这对公司来说是一个很好的退出，到目前为止已经筹集了1300万美元的资金。投资者包括O ' Reilly alpha tech Ventures、欧洲创始人基金和 Azure Capital Partners 。

TripIt最初是在TechCrunch40上发布的，并受到了迈克尔·阿灵顿的喜爱。

在一份声明中， Concur 董事长兼首席执行官史蒂夫·辛格解释了收购TripIt的理由:

“移动解决方案的进步改变了商务旅行者购买、分享、管理和消费其旅行计划的方式。人们普遍需要给混乱的旅行带来秩序，让商务旅行者的生活更美好。无论是有管理的旅行还是无管理的旅行都是如此。

我们共同解决整个商务旅行流程中的挑战——从预订、旅行中的活动和分享旅行信息，到旅行后的费用管理和对账。"

Concur表示，超过1500万人使用其解决方案来帮助管理他们的商务旅行和费用。整个TripIt团队将加入Concur。

Concur已签署最终协议，在交割时以约2700万美元的现金和约4400万美元的Concur股票收购TripIt，外加在交割后30个月内结算的约3800万美元的或有现金，但须遵守最终协议中规定的某些调整和托管条款。

作为收购的一部分，Concur将把未授予的TripIt期权兑换成Concur限制性股票单位，在交易结束时总价值约为1100万美元。考虑到所有因素，这笔交易的总价值高达1.2亿美元。

Concur表示，将在2月初的2011财年第一季度财报电话会议上提供更多细节。
