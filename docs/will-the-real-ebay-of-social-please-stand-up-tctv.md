# 请真正的“社交易贝”站起来好吗？(TCTV)

在风险企业中，走在时间前面几乎和迟到一样糟糕。但风险投资业的另一个伟大之处在于，每条规则都有例外。克雷格·多纳托希望 Oodle 是个例外。他花了十多年时间建立了一家社交分类公司，为 Oodle.com、T4、MySpace 和脸书的市场提供动力，并发展到超过 1400 万独立用户。它得到了网络上一些最聪明的投资者的支持，比如格雷洛克公司的雷德·霍夫曼和大卫·司达，他们都投资了脸书和 LinkedIn，所以他们对社交图略知一二。

现在，脸书另一个热门的风险投资人[，](https://web.archive.org/web/20221209130832/http://https//beta.techcrunch.com/2010/11/22/accel-partners-fund-ix-facebook-extraordinary/) [Accel Partners](https://web.archive.org/web/20221209130832/http://www.accel.com/) ，[已经资助了](https://web.archive.org/web/20221209130832/http://https//beta.techcrunch.com/2010/11/22/yardsellr-5-million-ebay-facebook/) [Yardsellr](https://web.archive.org/web/20221209130832/http://www.yardsellr.com/) ，声称它将成为“脸书的易贝”与此同时，Groupon 的巨大成功让所有人都开始重新评估社交购物。那么，所有这些对 Oodle 来说意味着什么呢？

首先，该公司并没有减速。上周，Oodle [收购了 Grouply.com](https://web.archive.org/web/20221209130832/http://https//beta.techcrunch.com/2010/11/17/social-classifieds-service-oodle-buys-grouply/),帮助人们向朋友圈子之外的朋友圈子出售东西。这笔交易凸显了一个阻碍在线分类广告快速发展的核心矛盾:卖家和买家是想要信任和关系，还是想要巨大的买家和卖家市场？因为根据定义，只有从你认识的人那里买和卖，才限制了市场。

多纳托和我们一起谈论市场，团购交易，为什么他说分类广告是“社会化的，而不是病毒式的”，以及为什么他坚持这么长时间努力建立一个市场。
