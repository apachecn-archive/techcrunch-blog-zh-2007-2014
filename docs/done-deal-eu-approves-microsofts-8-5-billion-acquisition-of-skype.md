# 完成交易:欧盟批准微软以 85 亿美元收购 Skype

欧盟委员会已经批准了微软对 T2 85 亿美元收购 Skype 的提议。正如 5 月份宣布的那样，微软达成了以 85 亿美元现金收购 Skype 的协议。获得批准后，现在就可以进行收购了。

这项交易在六月得到了美国 T4 联邦贸易委员会的批准。在收购时，微软和 Skype 表示，他们“希望在本日历年内获得所有必要的监管许可”。显然他们的愿望实现了。

Skype 将成为微软内部一个新的业务部门，其现任首席执行官托尼·贝茨(Tony Bates)将担任微软 Skype 部门总裁的头衔，直接向微软首席执行官史蒂夫·鲍尔默(Steve Ballmer)汇报。许多人认为微软为 Skype 支付了过高的价格，尤其是考虑到脸书和谷歌也在窥探这家网络电话公司。这是微软迄今为止最大的一次收购。

以下是欧盟对此次收购的声明:

*欧盟委员会根据欧盟合并条例批准了微软公司对互联网语音和视频通信提供商 Skype 的拟议收购，因为该交易不会严重阻碍欧洲经济区(EEA) 1 或其任何实质性部分的有效竞争。*

在消费者通信领域，调查发现双方的活动主要在视频通信方面重叠，微软通过其 Windows Live Messenger 在这方面很活跃。然而，委员会认为，在这个包括谷歌在内的众多参与者参与的不断增长的市场中，不存在竞争问题。

对于企业通信，调查证实，Skype 在这些产品上的市场份额有限，并不与微软的企业通信产品 Lync 直接竞争，后者主要由大型企业使用。

委员会的调查还集中在可能的联合效应上，因为 Skype 和微软在邻近市场都很活跃。

该交易于 2011 年 9 月 2 日通知欧洲经济区委员会进行监管审批。
