# Cloudera 创始人在 Eric Schmidt 的支持下推出大数据管理和分析平台 WibiData

**独家:** [Cloudera](https://web.archive.org/web/20230204213058/http://www.cloudera.com/) 创始人 [Christophe Bisciglia](https://web.archive.org/web/20230204213058/http://www.crunchbase.com/person/christophe-bisciglia) 今天首次亮相他的新公司，名为 [Odiago](https://web.archive.org/web/20230204213058/http://odiago.com/) 。Odiago 开发了一款名为[widi data](https://web.archive.org/web/20230204213058/http://www.wibidata.com/)的产品，它利用 Apache Hadoop 来管理和分析大量数据。该公司正在与包括谷歌董事长埃里克·施密特、Cloudera 首席执行官迈克·奥尔森和 SV Angel 在内的许多主要支持者一起启动。

作为背景，Bisciglia 创立了 Cloudera，它是 Hadoop 的开发商和商业分销商，Hadoop 是一种开源软件，为世界上最大和最受欢迎的网站的数据处理引擎提供动力。在加入 Cloudera 之前，Bisciglia 是谷歌的一名高级工程师，他创建并领导了谷歌的学术云计算计划，该计划提供谷歌托管的计算资源，以促进世界各地大学的教育和研究。

Bisciglia 今年早些时候离开了 Cloudera，去追求其他机会，其中包括 Odiago。这家初创公司包括 Bisciglia 在谷歌个性化团队的一些前同事，他们正在推出 WibiData，以帮助管理和分析关于用户的复杂商业数据，以便你可以预测他们未来将如何与该产品互动。具体来说，Bisciglia 解释说，这项技术可以用于许多网络公司的个性化，包括消费者网络、电子商务和游戏公司。

Bisciglia 说，顾客也可以利用数据分析来创造个性化的产品和服务。该技术可以轻松地与商业智能和数据库产品集成。该公司已经有一些使用 WibiData 的知名客户，包括维基百科、Rich Relevance 和 Atlassian。

正如 Bisciglia 所解释的，Cloudera 是一个拓展应用领域的绝佳机会。借助 WibiData，他能够专注于拥有大量数据的公司的特定业务功能。
