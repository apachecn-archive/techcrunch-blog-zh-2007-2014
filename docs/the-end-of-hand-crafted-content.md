# 手工制作内容的终结|技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/12/13/the-end-of-hand-crafted-content/>

旧媒体最喜欢的莫过于写他们自己即将到来的死亡。我们总是喜欢添加我们自己的两分钱——不知道 YouTube 是什么的美联社,“T2 ”,每天阅读 TechCrunch 的《纽约时报》的 T3 记者,“T4”等等。

概括地说，我喜欢路透社的 T5、默多克的 T8 和埃里克·施密特的话:这个行业正处于危机之中，大胆的创新者将会胜出。就我个人而言，我仍然认为，对于最好的记者来说，即使不是他们目前工作的品牌，最好的前进方式也是离开这些品牌，做自己的事情。

但作为最后一轮的创新者之一，我认为有一个更大的问题潜伏在地平线上，比一群博客和聚合器破坏旧的媒体商业模式，无论如何都需要破坏。快餐含量的上升正在逼近我们，而且会变得越来越糟糕。

旧媒体对那些总结内容并链接回原始来源的博客和聚合器感到焦虑。他们说，在这个世界上他们不能做生意，所以他们走另一条路，试图找到一种方法来保护内容并对其收费。

这些是穴居人，或者其他什么人，当火被发现时，他们害怕火，因为它燃烧了，或者技术太先进了，无法真正理解。聪明的家伙用它来煮他们的肉，让它们保持温暖，然后繁殖。

就我们而言，当有人“窃取”我们的内容并链接回我们时，我们会举办一个派对。办公室里到处都是击掌声。至少在我们的方向上有一些小小的点头。像 [TechMeme](https://web.archive.org/web/20230314162549/http://www.techmeme.com/) 这样的聚合器可以算出是谁泄露了消息。页面浏览量减少，但[声誉增加](https://web.archive.org/web/20230314162549/http://www.techmeme.com/lb)。

但是对于每一个链接，都有几十个网站直接窃取我们的内容，而且没有任何归属。不仅仅是垃圾博客，就连《纽约时报》也是如此。这不是版权问题——故事是由真人来重写的。但简单地接受新闻并改写它——如果你能逃脱惩罚的话——比雇佣真正从事新闻工作的人要便宜得多。随着时间的推移，它变成了一种难以承受的竞争性税收。

但即便如此，像我们这样的公司也能找到竞争的方法。

那么，真正让我害怕的是什么呢？随着时间的推移，快餐内容的增长肯定会摧毁今天手工制作内容的妈妈和爸爸的业务。这是大规模廉价、一次性内容的兴起，是门户网站和搜索引擎向我们灌输的力量。

一端是美国在线的 T2 和他们的丰田战略，即通过旧媒体的废弃作品建立成千上万的利基内容网站。这导致了大量非常非常糟糕的内容被突出显示在庞大的美国在线主页上。[比如这篇文章，就太可怕了](https://web.archive.org/web/20230314162549/http://www.bloggingstocks.com/2009/12/10/aol-i-mean-aol-did-not-tank/)。美国在线自己的一个博客贬低了该公司的分拆，漫无目的地走了几英里，并添加了一个巨大的事实错误来掩盖事实(该公司正在赔钱)。雇用一群无法保住旧媒体工作、也没有勇气自己走出去的人，然后对这些低人一等的记者进行很少或根本没有编辑监督，会得出一个不可避免的结论——廉价、蹩脚的内容。这种蹩脚的内容在美国在线门户网站上获得了大量的受众。

另一端是需求媒体和类似的公司。参见《连线》杂志的[“需求媒体和快速、可弃置、盈利如地狱般的媒体模式](https://web.archive.org/web/20230314162549/http://www.wired.com/magazine/2009/10/ff_demandmedia/)”。该公司只根据搜索引擎上的热门内容，支付最低的费用每天制作“4000 个视频和文章”。他们推动搜索引擎优化汁这一内容，这是尽可能快和便宜，并祈祷交通。很明显，它就像一个符咒。

这些模式形成了一场向底层的竞赛，任何在内容上花费时间和精力的人都将被淘汰出局。

我们还没到那里，但我看到它来了。就像旧媒体在抱怨我们一样，找我们开始抱怨新的混蛋。

我给读者的建议是——做好准备，因为在不久的将来，你会一天读五次麦当劳。我对内容创作者的建议更微妙。想出一个更具破坏性的方法来赢得胜利，或者死亡。或者干脆放弃做你做的赚钱的事情。如果你是为了激情而写作，而不是为了金钱，你仍然会得到乐趣。即使你写的所有东西都立即被剽窃，没有署名，搜索引擎也不会像以前那样关注你。当然，你可能要在晚上继续你的爱好，找一份真正的工作。但是每个人有时候都要面对现实。

忘掉公平与不公平，对与错。这种情况正在发生。颠覆者正在被颠覆，每个人都必须适应它，否则就要面对后果。手工制作的内容已经死亡。快餐内容万岁，它在这里停留。
