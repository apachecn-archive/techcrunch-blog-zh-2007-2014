# 巴西初创公司 DeskMetrics 为桌面软件分析筹集种子资金

[![](img/214090c3727c056eeaf2c8b23dfaf882.png)](https://web.archive.org/web/20230203175448/http://www.deskmetrics.com/) 巴西初创公司 [DeskMetrics](https://web.archive.org/web/20230203175448/http://www.deskmetrics.com/) 提供工具和分析服务，帮助开发者跟踪他们的客户如何使用原生桌面应用程序，该公司已经完成了来自一群地区天使投资者的 20 万美元种子轮投资(该初创公司没有披露所涉投资者的姓名)。

DeskMetrics [于 10 月](https://web.archive.org/web/20230203175448/https://techcrunch.com/2010/10/29/deskmetrics-wants-to-be-a-google-analytics-for-desktop-software/)发布，为开发人员提供了一套原生组件，他们可以将其集成到桌面软件中，以执行高级分析跟踪。这很像桌面的网络分析服务——通过集成这些组件，开发人员可以知道用户何时安装或卸载了应用程序，甚至可以知道他们在应用程序中点击了哪些按钮。

显然有一点令人毛骨悚然的因素(我们原始文章中的评论指出，这听起来可疑地像间谍软件)，但 DeskMetrics 指出，微软通过其客户体验改善计划在自己的软件中使用了类似的技术。负责任的开发人员可以(也应该)在记录这些数据之前总是提示用户同意。

这一领域的另一个竞争对手是 [Eqatec](https://web.archive.org/web/20230203175448/http://www.eqatec.com/) 。
