# Foursquare 正在做大事，因此现有投资者 Spark Capital 购买了 5000 万美元的员工股票

Foursquare 创始人丹尼斯·克劳利(Dennis Crowley)今天下午在巴塞罗那向世界移动通信大会的与会者解释了他的定位公司的潜力。不过，其他人不需要听这个演示。据消息人士称，现有投资者 Spark Capital 正在购买价值 5000 万美元的股票，至少为部分员工提供流动性。

据我所知，其结果是估值已经比上一轮高出了 6 亿美元。尽管该公司继续专注于产品开发，而不是努力实现收入最大化。

那么，是什么让投资者兴奋呢？嗯，这款定位应用似乎做了很多正确的事情，尽管有人对其仍处于萌芽状态的商业模式提出了质疑。虽然 Foursquare 可能不是世界上最大的社交网络，但它所拥有的数据，以及围绕它成长的生态系统，构成了一个充满希望的故事。

“每个人都认为 Foursquare 是签到和徽章，”克罗利[在今天的会议上说。“不，不。我们正在回收数据，并为现实世界提供建议。”](https://web.archive.org/web/20221231111857/http://reviews.cnet.com/8301-13970_7-57387406-78/foursquare-on-why-recycling-your-data-is-good-for-you/)

克劳利在这里谈论的一个例子涉及到 Foursquare 的新“探索”功能，该功能最近在该公司的移动应用程序中推出。Foursquare Explore 本质上是一个本地推荐指南，它允许用户通过类别、名称，甚至更具体的东西，如“寿司”或“汉堡包”，来发现附近的餐馆、酒吧、咖啡店、夜生活场所和其他场所。这项功能利用了 Foursquare 自己的社交图，以便推荐你的朋友们去过和喜欢的地方，同时还提供来自 Foursquare 更广泛网络的提示和评论。最初，Explore 仅限于围绕您的确切位置提供建议，但随着[的最新更新](https://web.archive.org/web/20221231111857/https://techcrunch.com/2012/02/08/foursquare-rolls-out-new-version-of-explore-on-mobile-with-filters-improved-search/)，它现在允许您通过在屏幕上移动地图或输入确切位置来找到世界上任何地方的推荐地点。

迄今为止，Foursquare 的 1500 万注册用户已经登记了 15 亿次，这反过来又产生了大约 75 万家餐馆和其他场所的丰富信息。该公司正致力于在任何地图上创建这些数据的可视化，这样用户就可以看到附近他们可能想去的地方。

对企业来说，它可以细分单个商店的访客人口统计，让他们更好地为客户量身定制商品。当然，从去年开始，它也一直忙于通过美国运通的交易来经营优惠券。对于开发者来说，它可以通过其 API 提供数据，Instagram 和 Path 等流行的移动应用程序已经在使用这些数据。

然而，该公司仅表示“我们不能对资金事宜发表评论。”
