# 主流媒体仍然睁大眼睛

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/06/26/mainstream-media-still-has-eyes-wide-shut-proves-michael-jacksons-death-reporting/>

迈克尔·杰克逊的不幸去世在很多层面上都是一个悲伤的事件，这是一个反思这个男人丰富的生活和事业的时刻，也是一个向他的家人、朋友和所有需要支持的人传递支持的时候，不管是无声的还是大声的，因为这位流行音乐之王已经不在了。

对于 TechCrunch 的我们来说，这也是一个机会来看看新老媒体在这个时代是如何处理新闻报道和发布的(以及在不久的将来它可能会如何发展)。

像许多其他人一样，当杰克逊住院的第一个报道开始流传时，我在浏览器的一个标签中打开了推特。起初，没有迹象表明这条消息得到了任何人的证实，人们只是疯狂地转发带有许多问号的消息，同时试图找到一个在线新闻来源，作为进一步更新故事的灯塔。很快，人们开始链接到美国在线的在线八卦发电站 [TMZ](https://web.archive.org/web/20230403234556/http://www.tmz.com/) ，这是第一个[宣布他去世的](https://web.archive.org/web/20230403234556/http://www.tmz.com/2009/06/25/michael-jackson-rushed-to-the-hospital/)当其他人还在谈论住院的部分。

诚然，TMZ 可能不是世界上最可信的来源(快说，哪一个是？)还有一种可能性是，他们只是把杰克逊去世的故事当成了事实，但现在这一切都不重要了。他们先说的，他们说对了。我们很快了解到，尽管事实上许多新闻博客和网站正在努力跟上寻求更多数字报道的大量涌入。

于是 TMZ 宣布了这一消息，推特上一片热议，大约 40 分钟后,《洛杉矶时报》在其一个博客上引用自己的消息来源，跟进确认了杰克逊的死讯。没什么大不了，但人们确实认为最新的后续行动更可信地证实了每个人已经在假设的事情，这很好，即使他们中的一些人在几分钟前无法停止抨击 TMZ 所谓的造谣。

但值得注意的是，在我看来有些令人不安的是:在这两家媒体之后跟进此事的大多数主流媒体大多是在重复和过滤这两家博客在他们之前报道的内容，往往没有适当的信用或归属，更不用说链接了。他们中的一些人，如[纽约时报博客位](https://web.archive.org/web/20230403234556/http://bits.blogs.nytimes.com/2009/06/25/michael-jackson-tops-the-charts-on-twitter/)承认 Twitter 是一个对事件进行水冷型闲聊的地方，但几乎没有人敢承认博客和 Twitter 在传播事实方面比他们更快(是的，[我们意识到](https://web.archive.org/web/20230403234556/https://techcrunch.com/2009/05/15/according-to-twitter-prop-8-was-overturned-before-it-existed/)将两者用作坚如磐石的新闻来源也可能相当危险，但这与本案无关)。

《芝加哥论坛报》的这篇文章中发现了一个令人瞠目结舌的与现实脱节的案例，副标题是“TMZ 以早期细节为先导，而洛杉矶时报和美联社则承担重任”，我们在文章中读到了这样的废话:

> 时代华纳旗下的八卦网站 TMZ.com 站在了杰克逊新闻的前面，数字时代的管道传播了这个消息，就像以前其他主要名人的新闻报道一样。但是，是老媒体的中坚力量挑起了重担，美联社(Associated Press)和芝加哥论坛报的姐妹报纸《洛杉矶时报》(L.A. Times)的网站等巨头报道了关于急救电话和最终他死亡的最快、最可信的信息。

是的，我也笑出声来。捶胸顿足地指责旧媒体为博客和推特“负重前行”，在报道信息方面比那些新媒体更快，而事实恰恰相反，这简直是荒谬至极。TMZ 是第一个报道杰克逊死讯的媒体，在这种情况下，它的消息来源被证明是“可信”的，不亚于任何旧媒体的落后者。像上面这样的陈述证明了作者对新闻传播世界是如何演变的存在巨大的误解——他肯定不是唯一一个沿着这些思路思考的恐龙。

作者也很高兴找到了支持他的人，这一次是娱乐新闻网站 HollywoodChicago.com 的创始人兼发行人亚当·芬德尔曼，当他得知这一消息时，他对员工的第一反应显然是:“我们确定吗？”。

以下是据报道他对《芝加哥论坛报》记者说的话:

> “TMZ 的网络和电视现象非常擅长快速报道和即时新闻，但在日常消费者心中的信任度方面存在固有的问题。”

没错，而且还会变得更好，但就我而言，你可以用任何一家新闻媒体来取代 TMZ，包括那些老牌大媒体，它们正迅速变得越来越不那么大[和](https://web.archive.org/web/20230403234556/https://techcrunch.com/2009/06/02/from-terrible-to-terrifying-newspaper-ad-sales-plummet-26-billion-in-first-quarter/)[相关](https://web.archive.org/web/20230403234556/https://techcrunch.com/2009/06/03/the-apis-plan-to-save-newspapers-lets-put-humpty-dumpty-back-together-again/)。

对我来说，这整件事只是证明主流媒体有理由对正在发生的事情睁一只眼闭一只眼，而不是学习和适应新闻业的新时代。
