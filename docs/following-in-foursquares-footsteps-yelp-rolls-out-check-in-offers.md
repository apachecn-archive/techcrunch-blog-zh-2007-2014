# 跟随 Foursquare 的脚步，Yelp 推出了入住优惠

今年早些时候，当本地商业列表和评论网站开始允许用户在 T4 的餐馆、酒吧和商店登记时，Yelp 进入了 T2 的位置之争。Foursquare 长期以来一直提供交易作为签到的激励，允许用户在签到时解锁折扣和优惠券。看起来 Yelp 再次步 Foursquare 的后尘，推出了签到服务。

虽然细节有限，但入住优惠似乎相当简单。如果你在某个地点签到，商家可以为你的签到提供特别折扣或优惠券。Yelp 还表示，这与销售和特别优惠功能不同，后者允许商家宣布 Yelp 商品的折扣。虽然我们不知道这些优惠将如何运作，Yelp 也可能整合其类似 Groupon 的每日交易。

Yelp 有抄袭 Foursquare 除了签到以外的一些功能的历史。例如，Yelp 还允许用户成为某个场所的[“公爵”或“公爵夫人”，](https://web.archive.org/web/20221208174356/https://beta.techcrunch.com/2010/06/09/yelp-royalty/)类似于 Foursquare 的市长功能。该公司也在考虑为其武器库添加徽章。

很难指责 Yelp 采用了 Foursquare 的一些功能，这些功能帮助这个基于位置的社交网络变得流行。脸书和谷歌复制了 Foursquare，推出了自己的定位服务。事实是，本地和基于位置的网络是热门，每个人都想把他们的帽子扔在环。你知道他们说什么——模仿是最真诚的奉承。
