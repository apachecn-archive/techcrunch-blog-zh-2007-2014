# 商业智能初创公司 Looker 首轮从红点融资 1600 万美元

由网景和 LiveOps 的早期首席工程师创办的商业智能初创公司 Looker ，已经筹集了 1600 万美元的首轮融资，由 Redpoint Ventures 领投，首轮资本投资了该公司的[200 万美元种子轮](https://web.archive.org/web/20230131004433/https://techcrunch.com/2013/03/06/looker-takes-2m-from-first-round-and-pivotnorth-to-build-a-sequel-to-sql-for-business-intelligence/)，也参与了。这使该公司的资金总额达到 1770 万美元。

Looker 由 Lloyd Tabb 创建，旨在通过将过去的编程查询转变为基于自然语言的查询，从而颠覆商业智能领域。大多数遗留 BI 系统都是基于 SQL 的，需要用户具备工程和编程专业知识来制定查询，以提取数据并理解这些见解。Looker 允许任何人轻松查询这些大型数据集。

正如该公司解释的那样，这就像在你的浏览器中放了一个数据科学家。该方法的一部分包括一种称为 LookML 的专有语言，它简化了编写脚本和回收 SQL 查询的过程。使用 Looker 和 LookML，组织可以将 Looker 查询分布到企业的所有职能部门，包括销售、营销等。Looker 可以在几个小时内部署完毕。

在秘密运营一年后，Looker 于今年早些时候推出，有 20 多个付费用户。自推出以来，该公司的客户群增加了一倍多，现在有 40 多名付费客户在使用 Looker。

该公司筹集资金主要是为了扩大销售团队，满足客户需求，以及雇佣更多工程师和投资基础设施。

帮助企业收集更有洞察力的信息和对数据的理解并不是一个新趋势。在商业智能领域，好数据、Domo 和许多其他公司正在快速发展。我们不仅会看到更多的进入者进入这一领域，而且我们也可能会看到更多的收购。总而言之，它为这些公司创造了一个超级有趣的未来。
