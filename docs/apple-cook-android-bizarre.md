# 苹果公司首席运营官·蒂姆·库克认为安卓平板电脑“古怪”

![](img/c61fcdff9ab03771712e266f94f298e3.png)

苹果的 iPad 很受欢迎。该公司在 2010 年第二季度售出了 730 万台 iPads，比上一季度增长了 75%。但从 CES 上发布的所有 Android 平板电脑来看，它将面临激烈的竞争。然而，到目前为止，苹果公司的首席运营官·蒂姆·库克对目前市场上的平板电脑“没有任何担忧”，甚至称目前的安卓平板电脑“怪异”在今天的[苹果财报电话会议](https://web.archive.org/web/20230203031414/https://techcrunch.com/2011/01/18/apple-q1-2011/)上，库克将竞争分为两类:基于 Windows 的平板电脑，它们“大、重、贵”，并且“电池续航能力差”，以及当前的 Android 平板电脑，它们是“放大版”智能手机。

根据 MG 的笔记，以下是他对这个问题的回答:

> 问:iPad 的竞争对手呢？
> 
> TC:如你所知，外面没有太多。如今(市场上)有两种群体——一种使用基于 Windows 的操作系统。它们又大又重又贵。电池续航能力弱。需要键盘或手写笔。从我们的角度来看，客户对此不感兴趣。
> 
> 然后是安卓平板电脑。今天上市的这款操作系统并不是为平板电脑设计的——但谷歌说过。所以你最终得到了一个不太合理的平板电脑大小。或者甚至不是真正的平板电脑体验。这是一款“放大版智能手机”——在我们看来，这是一款奇怪的产品。这些是今天发货的产品。如果你拿着 iPad 进行对比，很大一部分人会选择 iPad。我们不关心那里。
> 
> 就下一代而言。还没有发货。所以我不知道。“今天他们是蒸汽。”然而，我们并没有坐以待毙。我们有巨大的先发优势。以及从 iTunes 到 App Store 的巨大用户优势。大量的应用和一个生态系统。我们非常有信心与任何人进行战斗。

他认为苹果公司在满足平板电脑的需求方面还只是皮毛。他指出，从消费者到企业都有兴趣。
