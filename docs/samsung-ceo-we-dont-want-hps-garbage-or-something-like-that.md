# 三星 CEO，“我们不要惠普的垃圾”——或者类似的话

现在，webOS 的未来不那么不确定了，因为三星首席执行官 Choi Gee Sung 抓住了收购惠普 webOS 业务的谣言，当着它的面笑了笑，然后冷冷地刺伤了谣言的心脏。

在回答记者关于最近的分析师报告的问题时，[表示](https://web.archive.org/web/20230204111243/http://www.bloomberg.com/news/2011-09-02/samsung-electronics-says-it-won-t-pursue-hp-s-webos.html)“收购操作系统成为一种时尚是不对的”，三星“永远不会”追求这样的交易。

在惠普宣布退出 webOS 硬件游戏后，一些分析师发布的报告显示，一些关键玩家有兴趣捡起倒下的操作系统。这些报道[将三星作为主要买家](https://web.archive.org/web/20230204111243/https://techcrunch.com/2011/08/29/samsung-may-buy-webos-recruits-former-hp-exec/)，但事实显然并非如此。此外，尽管惠普取消了 webOS 硬件部门，但该公司并没有放弃软件开发——目前还没有。事实上，如果惠普剥离其个人电脑业务，这家新成立的公司可以[重振 TouchPad](https://web.archive.org/web/20230204111243/https://techcrunch.com/2011/08/30/hp-could-revive-the-touchpad-says-former-webos-vp/) 以及其他 webOS 硬件设备。

相反，三星正专注于开发其 Bada 操作系统，我们的许多评论者在我们关于该主题的原始帖子中指出了这一点。该公司本周刚刚推出了三款新的 Bada 手机，包括一款具有 NFC 功能的手机。CEO Choi 实质上重申了 Android 和 Bada 是三星的未来，而不是买一个二手 OS。

但是，随着三星的出局，如果惠普想要卸载 webOS，那么没有多少公司还能扛着 Palm 倒下的大旗。宏达电牢牢掌握着安卓系统，RIM 正在 QNX 平台上开发下一代黑莓智能手机，摩托罗拉很快将被谷歌拥有，诺基亚与微软达成了利润丰厚的协议。还剩下谁？也许泰格电子公司会收购 webOS 并重新生产 Game.com 便携式电脑。我会买下它。
