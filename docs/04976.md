# 埃隆·马斯克谈吃玻璃的最佳方式[视频]

在[创始人展示会](https://web.archive.org/web/20221206175903/http://foundershowcase.com/)的舞台上，我采访了埃隆·马斯克。他可能把自己最好的一句台词留到了最后:“做一名企业家就像吃玻璃，盯着死亡的深渊。…所以如果这听起来很吸引人…”接着是他标志性的詹姆斯·邦德恶棍咯咯笑。

马斯克的企业家生涯非常成功——除了共同创立了 [PayPal](https://web.archive.org/web/20221206175903/http://www.crunchbase.com/company/paypal) ，他还创办了 [SpaceX](https://web.archive.org/web/20221206175903/http://www.spacex.com/) ，后者与美国国家航空航天局签订了 20 亿美元的合同，以及[特斯拉](https://web.archive.org/web/20221206175903/http://www.teslamotors.com/)，后者刚刚上市。但是这一过程并不顺利。在首次公开募股后的首次公开露面中，马斯克谈到了一系列事情。当马斯克说 2008 年 12 月是他一生中最糟糕的时刻之一时，一个特别人性化的时刻到来了:SpaceX 的发射失败，特斯拉的资金告罄，他陷入了他的[太公开的](https://web.archive.org/web/20221206175903/https://beta.techcrunch.com/2010/07/11/elon-musk-why-owen-thomas-is-silicon-valleys-jayson-blair/)离婚。这提醒了观众，即使是那些我们认为拥有一切的企业家也仍然有缺陷，仍然在奋斗。

以下是我们讨论的一些其他重点。完整视频在下面。

1.马斯克对风险行业颇有微词——除了一家公司。他没有透露是谁，但他列出了所有与他合作过的人，这显然是针对特斯拉的投资者[vantage point Venture Partners](https://web.archive.org/web/20221206175903/http://www.crunchbase.com/financial-organization/vantagepoint-venture-partners)。我在舞台上从他身上得不到更多了，所以不管是哪一方，如果你想发难，随时打电话给我。

2.迄今为止，他在倍数方面的最佳回报不是特斯拉或贝宝，而是 SolarCity。

3.马斯克之前曾表示，同时担任特斯拉和 SpaceX 的首席执行官是不可持续的。他证实这已经“超出了乐趣点”，但表示他会暂时保留这两份工作。

4.马斯克和受资助的 Adeo Ressi 在大学时开办了一家夜总会，马斯克是一个清醒的人，当警察来电话时，他总是和他们说话。“他是我见过的最傻的人，”雷西在观众中说。

5.马斯克仍然计划让他的下一家公司专注于电动垂直起飞飞机。他说他认为这可行，但问题是它们能飞多长时间。但在他的火箭和电动汽车之间，他还需要一段时间才能到达。

6.马斯克仍然希望[在火星](https://web.archive.org/web/20221206175903/http://www.guardian.co.uk/technology/2010/aug/01/elon-musk-spacex-rocket-mars)上退休，但他承认必须发生一些其他事情才能使这成为可能。值得注意的是，虽然 SpaceX 已经成为在空间站之间穿梭宇航员的可行业务，但马斯克的梦想仍然是促进[行星间](https://web.archive.org/web/20221206175903/http://www.youtube.com/watch?v=u2-blWgVk-A)的生活。他指出，从定义上来说，火星殖民地将是一个以创业、工程为重点的群体，因为只有敢于冒险的极客才会选择移居火星。那听起来像是一个我们的许多读者也想退休的地方。
