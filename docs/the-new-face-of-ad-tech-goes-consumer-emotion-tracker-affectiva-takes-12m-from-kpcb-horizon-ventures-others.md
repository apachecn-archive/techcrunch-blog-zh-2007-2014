# 广告技术的新面孔走向消费者:情感追踪器 Affectiva 从 KPCB、Horizon TechCrunch 获得 1200 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/08/07/the-new-face-of-ad-tech-goes-consumer-emotion-tracker-affectiva-takes-12m-from-kpcb-horizon-ventures-others/>

如今，广告技术和消费者参与游戏及其他在线活动的世界越来越近了。麻省理工学院媒体实验室的初创公司 Affectiva 今天宣布获得一轮价值 1200 万美元的融资，该公司通过跟踪在线用户的面部来测量他们的情绪反应。它将利用这笔资金将其首次用于衡量广告效果的技术推向消费者市场。

C 轮由 Horizons Ventures(和记黄埔的李嘉诚和脸书、Spotify、Siri 等早期支持者的投资工具)和 Kleiner Perkins cau field & Byers(KPCB)数字增长基金(脸书、Twitter、Zynga、Groupon、Square、Klout、Waze、Shazam、Spotify 等的支持者)领投。).包括 WPP、米利安资本和 Peder Wallenberg 慈善信托基金在内的一些现有投资者也参加了会议。

作为融资的一部分，Affectiva 将在其顾问中获得两位超级明星投资者:KPCB 的玛丽·米克尔将成为董事会顾问；弗兰克·米恩(Frank Meehan)加入了董事会，他在全职加入 Horizon 之前，曾是和记黄埔一些最具创新性的移动项目的幕后主使。

这使得 Affectiva 筹集的资金总额达到 2020 万美元，此外还有国家科学基金会的[拨款和其他支持。](https://web.archive.org/web/20221209101621/https://beta.techcrunch.com/2012/06/01/affectiva-nsf-grant/)

该公司表示，最新一轮的资金将用于开发其技术在未来的实施方式。到目前为止，Affectiva 主要与广告行业合作，开发衡量广告效果的工具，并推出了一款名为 Affdex 的产品。(WPP 分公司 Millward Brown 是使用它来测试和调整广告的公司之一。)它还创造了一种可穿戴生物传感器，名为 Q Sensor。

现在，这家由 Rosalind W. Picard 教授和研究科学家 Rana el Kaliouby 于 2009 年创立的公司正在将这一技术推向新的高度。它希望 Affdex 中的技术不仅可以用来衡量用户对所有形式的在线视频内容的反应，还可以整合到广告以外的内容中，例如游戏中。

“我们很高兴将这项技术带到更广阔的市场，”Affectiva 的首席执行官大卫·伯曼告诉 TechCrunch。“这一轮是把它放到消费者手中，放到社会环境中，让他们参与进来。”

有趣的是，El Kaliouby 和 Picard 最初开发这项技术是为了帮助自闭症患者更好地理解人们的反应。

El Kaliouby 承认，这些天来，考虑到我们独自看屏幕的时间，许多人对任何事情都没有太大的强烈反应。Affectiva 的目标是理解哪怕是最小的手势。正如谚语所说，一张图片可以表达 1000 个单词:“你越有表现力，内容就越有可能被广泛传播，”她指出。"但是，是的，对于最平淡的内容，你的脸会显示厌倦."

Affectiva 正在悄悄建立的另一个东西是一个大数据风格的面部内容存储库，其中包含所有含义的线索，在我们日益全球化的世界中，这对内容创作者来说是非常宝贵的。(是的，在你向 Affectiva 提交任何东西之前，它都会征求你的同意，正如你在这个[演示中看到的](https://web.archive.org/web/20221209101621/http://www.affectiva.com/affdex/#pane_tryit)。)

“看到不同文化的结果很有趣，”她说，并指出它已经收集了 20 多个国家用户的面部数据。“我们正在建立这个数据库，以观察人们对跨文化内容的反应。”例如，一个男人和一个女人之间的一个小小的动作在一个市场上几乎不会引起注意，但在另一个市场上会引起愤怒。

除了点击量之外，创造衡量用户参与度的方法才刚刚开始，但你可以想象，随着产品变得越来越复杂，以及点击率等指标继续下降，这可能会有很大的需求。

脸书自己也在这一领域进行了某种投资，去年 3 月，它[聘请走了眼球追踪初创公司 GazeHawk](https://web.archive.org/web/20221209101621/https://beta.techcrunch.com/2012/03/08/facebook-gazehaw-2/) 的两位创始人——尽管当时它并没有收购最初初创公司的产品或技术。与此同时，[脸书对人脸识别初创公司 Face.com](https://web.archive.org/web/20221209101621/https://beta.techcrunch.com/2012/06/18/facebook-scoops-up-face-com-for-100m-to-bolster-its-facial-recognition-tech/)的收购也表明，它正在考虑如何利用自动化系统来更好地订购网站上的视觉内容。

(应该指出的是，当我直截了当地问 Affectiva 的首席执行官，它是否正在与脸书谈判时，他说没有，但他强调说，它的最新投资者确实在那里取得了很大进展。)

然而，这个行业还有一个更人性化的因素。随着技术继续成为我们生活的中心支点，情感以及用技术包裹情感的能力也不容小觑:

“社交、移动和数字技术的采用正在继续加速，我们需要将真实的情感带回在线互动中，”KPCB 的合伙人玛丽·米克尔在一份声明中指出。" Affectiva 的思想领导力和商业吸引力使这成为一项引人注目的投资."

该公司的 Q 传感器产品已经被“数百”所大学和公司使用，也将获得一些投资。考虑到量化自我的趋势，以及帮助人们更好地跟踪自己物质生活的设备和服务的兴起，这是另一个值得关注的领域。
