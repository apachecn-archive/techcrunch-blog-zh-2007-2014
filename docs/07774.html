出乎意料的是，Instacolor被迫用“真实”的颜色来改名

这是一部黑色喜剧，涉及白热化的初创公司 Color 和仍然稚嫩的 Instacolor 。面红耳赤的是，后一个应用程序背后的开发者，它结合了 Instagram和Color的关键功能，在颜色乡亲们对Instacolor名称产生灰色情绪后，被迫将其名称改为Instalook。

本周早些时候，Color的律师向insta color insta look的开发者发送了一份停止通知，要求他更改iPhone应用程序pronto的名称。

在信中，Color的法律顾问写道:

如您所知，Color labs是Color应用程序的创建者和所有者，这是一个支持GPS的照片共享应用程序，可以从iTunes和其他地方下载。Color Labs花费了大量的时间和资源来提高消费者对彩色商标和产品的认知度和好感度。它也是美国商标申请号85/222，392的所有者，该申请的副本附后。

虽然我们的客户赞赏您对其产品的热情，但我们希望您能够理解，未经授权的对其商标和软件的批发使用侵犯了其商标和版权，也违反了您在下载彩色应用程序时同意的使用条款。

这种未经授权的使用错误地暗示Color Labs赞助或支持您的软件，并且它以某种方式附属于您的软件。鉴于Color Labs无法控制您的产品质量，我们相信您能够理解由此引发的重大问题。

在信的后面，Color表示，如果开发商Rakshith Krishnappa未能“立即停止所有未经授权使用Color labs商标的行为”，它准备诉诸法律。

我猜想，让color恼火的不仅仅是Krishnappa在应用程序的名字中使用了“Color ”,并为此收费。

在被要求发表评论时，一位色彩代表回应道:

“由于使用了Color的名称，并模仿了我们的应用程序和徽标，我们担心潜在的消费者混淆。”

作为旁观者，我可以理解为什么Color发出了停止信，但我也为Krishnappa感到难过，他开发这款应用程序是作为一个周末项目，没有4100万美元的资金支持。不管怎样，Krishnappa明白他必须更改名称——他立即这么做了——但断然否认模仿Color的iPhone应用程序的外观和感觉。

他还解释说，他对应用程序收费只是因为Instagram有每小时5000个API的限制，他想确保每个使用该服务的人都有良好的体验。

自从 Color 推出其照片分享应用以来，这家4100万美元的初创公司就一直处于艰难时期。据首席执行官比尔·阮(Bill Nguyen)称，联合创始人彼得·范(Peter Pham)离开了(T9)，或者被解雇了，他还告诉《纽约时报》(New York Times)记者(T10)，该公司将重新开始(T12)。

他们现在最大的挑战是:似乎没有人在使用这个应用。

昨天，我们报道了谷歌在推出Color之前出价2亿美元收购它。
