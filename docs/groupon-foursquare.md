# Groupon 现在在你的 Foursquare 中，两种美味一起品尝

嗯，这是一个潜在的巨大合作伙伴关系。经过长时间的谈判，Groupon 交易最终将开始出现在 Foursquare 上。不仅是普通的日常团购会出现，现在更新的移动团购也会出现。本月早些时候，Foursquare 宣布与 LivingSocial、Gilt City、zozi、BuyWithMe 和 AT&t 建立合作伙伴关系，这一合作伙伴关系有别于[的其他日常交易伙伴关系](https://web.archive.org/web/20230203061553/https://techcrunch.com/2011/07/12/foursquare-deal-partnerships/)

那些特价商品都是日常交易，不能立即购买。您需要购买交易，但在当天所有交易售出之前无法赎回。这就是日常交易的运作方式，当你从电脑上购买时，这是有意义的。但当你在手机上寻找附近的交易时，你希望能够自发地兑换它们。

Groupon Now 的设计就是为了做到这一点。这些移动交易可以在它自己的应用程序中获得，也可以通过与其他移动应用程序[如 Loopt](https://web.archive.org/web/20230203061553/https://techcrunch.com/2011/05/20/loopt-beats-groupon-to-notifying-you-of-nearby-groupon-now-deals/) 和现在的 Foursquare 的分销合作关系获得。Groupon Now 交易现在可以在 25 个城市使用，它们将出现在 Foursquare 上，与其他合作伙伴的其他特价商品和日常交易放在一起。但它们可以立即赎回的事实将使它们与众不同。他们还关闭了[兑换循环](https://web.archive.org/web/20230203061553/https://techcrunch.com/2011/07/24/redemption-loop-local-commerce/)，并将允许更好地跟踪交易，因为 Groupon 确切地知道兑换了多少交易以及何时兑换(而不是仅仅知道卖出了多少优惠券)。

这些交易在 Foursquare 的背景下是有意义的。通过与下载量超过 1000 万次的成熟移动应用合作，Groupon 获得了即时分销。(说真的，你们中有多少人打算下载 Groupon Now 应用？).人们使用 Foursquare 还有其他原因，现在他们也开始将交易视为额外的奖励。这是引入这些新型交易并逐步扩大的好方法。

那么，为什么 Foursquare 在与 LivingSocial 和其他公司合作时没有宣布 Groupon 回来呢？阻碍是 Foursquare 要求 Groupon 和所有合作伙伴使用 Foursquare 的 API，而不是相反。据我所知，这有点像平台对接。不过现在都好了。Foursquare 还必须做一些工程工作，为一些交易增加一个“赎回窗口”,这些交易只能在特定时间内有效，比如许多 Groupon Now 交易。预计很快会看到其他 Foursquare 特价商品利用兑换窗口。

Foursquare 的业务发展总监特里斯坦·沃克(Tristan Walker)今天将在我们位于帕洛阿尔托的 [Mobile CrunchUp](https://web.archive.org/web/20230203061553/https://techcrunch.com/the-mobile-first-crunchup-and-6th-annual-summer-party-at-august-capital-july-29th-2011/) 上发表演讲。现在我知道我会问他什么了。
