# hack FWD——缓慢燃烧的欧洲加速器正在加速

早在去年 6 月，LinkedIn 竞争对手邢的创始人 Lars Hinrichs[以 4830 万美元的价格离开了](https://web.archive.org/web/20230328085334/http://eu.beta.techcrunch.com/2009/11/18/burda-buys-25-stake-in-xing/)创建了一个新的创业投资工具，名为 [HackFwd](https://web.archive.org/web/20230328085334/http://hackfwd.com/) 。

当时宣布 HackFwd 将出售一家公司 27%的股份(总部位于美国的 YCombinator 持有约 6%的股份)，并为选定的初创公司开展为期一年的项目。这一年的资助高达 191，000€(取决于团队的规模)。创始人持有 70%的股权，其中 3%归顾问公司所有，27%归 HackFwd 所有。然而，也就是说，他们会处理“法律和行政事务…这样你就可以专注于你的产品。”

这是一个非常不寻常的举动-通常加速器是 3 个月左右。

从那以后，我们就没听到什么消息了。但是最近我是第一个被邀请参加 HackFwd“建设”活动的记者。一年四次，他们投资的团队通常会在马略卡这样阳光明媚的地方会面，回顾他们迄今为止的进展。我遇到的初创公司一般都是技术性很强的——都是消费类公司，但都非常非常注重技术。这使得一些加速器发生了令人耳目一新的变化，在这些加速器中，初创公司往往看起来不那么技术性。毕竟这是互联网。

HackFwd 现在的去向将是双重的。他们正在组织[hack now](https://web.archive.org/web/20230328085334/http://www.hacknow.org/)——一场针对全欧洲人民的竞赛。此外，在他们的下一次“建设”活动中，他们将在柏林举办一场露天活动。

我在 Mallorca Build 事件中采访了 Lars，以真正了解 HackFwd 的成功之处。
