# 雅虎向那些发送请求的人释放回收的用户名，让你花 1.99 美元就可以跟踪你自己的用户名

今天，雅虎将开始通知那些通过该公司新扩展的用户名池请求特定用户 ID 的人，如果他们的选择被批准的话。以防你第一次错过:今年夏天早些时候，雅虎宣布将重新开放未使用的雅虎 id，以及与之相关的电子邮件地址。这些计划最初遇到了一些围绕这样做的安全实践的争议，但是雅虎很快就通过额外的保护措施解决了 T4 的担忧。

七月中旬宣布，感兴趣的用户可以前往[wishlist.yahoo.com](https://web.archive.org/web/20221208221553/http://wishlist.yahoo.com/)申请最多五个电子邮件地址/用户 id。如果公司确定这些 id 符合重复使用的条件，将按照先到先得的原则授予这些 id。此举意在引发一场电子邮件和用户 id 的土地争夺战，这些东西早就被拿走了——比如你的名字，没有数字。(尽管从 johnsmith@yahoo.com 发来的邮件也可能表明你从第一天起就在雅虎工作。收件人永远也不会知道你是否已经完全过时了，或者看起来完全是最前沿的。)

现在，雅虎提醒用户，如果他们能够抢在别人之前抢到自己喜欢的用户 ID。如果他们没有，那么雅虎说它也宣布了一个新的[观察名单](https://web.archive.org/web/20221208221553/http://watchlist.yahoo.com/)，用户将被自动免费加入。如果您的首选用户名在将来某个时候变得可用，该功能会提醒您。

对于那些从未申请过新的雅虎 ID 的人来说，该公司在新的观察名单上迈出了相当大胆的一步:你现在可以支付 1.99 美元使用这项服务，这项服务可以跟踪五个名字三年。观察名单将保留 14 天内有效的名字，让你有机会提出要求。

等等，什么？

好的，当然。雅虎的目标是扭亏为盈，前谷歌高管、现雅虎首席执行官玛丽莎·梅耶尔(Marissa Mayer)有很大希望能够实现这一目标。而且已经有了一些进展——根据 comScore 的最新报告，雅虎在美国互联网流量上超过了谷歌。此外，该公司已经推出了一些不错的移动应用，比如更新了 Flickr 和新的雅虎天气。当然还有 Tumblr 的交易。但是，如果雅虎认为雅虎 ID 在这个时候是值得花钱的，那它就是被误导了(或者过于乐观了)。话又说回来，如果你对像 batman@yahoo.com 这样的名字有一些特别的计划，该公司说在请求列表中排名很高，也许你愿意[付钱跟踪它](https://web.archive.org/web/20221208221553/http://watchlist.yahoo.com/)。
