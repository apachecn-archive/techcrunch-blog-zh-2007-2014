# 智能吧:日本获得另一款特殊的安卓手机

[先是 KDDI 的 iida INFOBAR](https://web.archive.org/web/20221006073203/https://beta.techcrunch.com/2011/05/17/video-kddis-iida-infobar-android-handset-boasts-unique-design-ui/) ，然后是夏普的翻盖式[安卓手机混合 oo7SH](https://web.archive.org/web/20221006073203/https://beta.techcrunch.com/2011/05/20/aquos-phone-the-hybrid-007sh-sharp-shows-worlds-first-clamshell-android-phone/) ，现在是所谓的[智能栏](https://web.archive.org/web/20221006073203/http://emobile.jp/products/hw/s42hw/)【JP】:日本越来越多专门设计的安卓手机。正如你在图片上看到的，智能条采用了“传统”的糖果条外形，但实际上是由 Android 姜饼驱动的。

该设备由华为制造，将由移动运营商 eMobile 在日本分销。这款手机主要针对女性用户，配有 3 英寸 WVGA 触摸显示屏(LCD)、512MB RAM 和 ROM、500 万像素摄像头、microSDHC 插槽、蓝牙 2.1+EDR、IEEE802.11b/g/n Wi-Fi(最多可共享 5 台设备)、IrDA SIR 红外、GPS 和 W-CDMA/GSM 支持。仅 51mm 纤薄，114g 重。
