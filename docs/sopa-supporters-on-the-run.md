# 跑路的 SOPA 支持者 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/14/sopa-supporters-on-the-run/>

国会对 SOPA 反盗版法案的支持正在减弱。在网上持续数周的喧嚣之后，国会领导人上周开始[打退堂鼓](https://web.archive.org/web/20221208211500/http://www.techdirt.com/articles/20120113/23560217407/sopa-delayed-cantor-promises-it-wont-be-brought-to-floor-until-issues-are-addressed.shtml)，奥巴马政府周六[参与进来](https://web.archive.org/web/20221208211500/http://www.whitehouse.gov/blog/2012/01/14/obama-administration-responds-we-people-petitions-sopa-and-online-piracy)，回应网上要求停止法案的请愿。白宫发表声明，明确拒绝 SOPA 的一些主要原则。

虽然白宫支持该法案的主要目标是阻止国际在线盗版，但对该法律实施方式的抱怨越来越多，最终可能会对所有试图推动该法案通过的媒体公司游说起到平衡作用。事实上，白宫在这个问题上的博客几乎相当于对这些法案的预先否决(这些法案还没有被国会两院投票表决，更不用说批准了)。

具体来说，白宫[声明](https://web.archive.org/web/20221208211500/http://www.whitehouse.gov/blog/2012/01/14/obama-administration-responds-we-people-petitions-sopa-and-online-piracy):

> 虽然我们认为外国网站的在线盗版是一个严重的问题，需要认真的立法回应，但我们不会支持减少言论自由、增加网络安全风险或破坏充满活力、创新的全球互联网的立法。

SOPA 最大的问题在于它应该被强制执行的方式，即屏蔽侵犯版权网站的域名系统(DNS)服务器。但是 DNS 服务器是互联网的一个基本技术组成部分(它们将 beta.techcrunch.com 这样的网站名称翻译成计算机能够更好理解的数字 IP 地址)。一旦你开始搞乱 DNS，各种意想不到的问题就会出现。

没有在法庭上进行充分的对抗性听证就阻止 DNS 增加了审查言论和其他合法活动的可能性。这也是中国用来屏蔽中国互联网上“冒犯性”内容的相同方法。这种做法也破坏了新的安全协议。白宫承认:

> 我们必须避免制造新的网络安全风险或破坏互联网的基础架构

谢谢你。但是还不清楚如何在不破坏互联网的情况下实现法案的目标。国会应该想出一个不同的机制来追踪外国盗版网站，否则就完全否决这些法案。

SOPA 的支持者可能会重新考虑他们的立场，但他们并没有完全撤退。网上的 SOPA 反对者现在还不应该跳起任何胜利的舞蹈。
