# WordPress.com 母公司 Automattic 今年的收入有望达到 4500 万美元

Automattic 是一家软件公司，因运行 [WordPress](https://web.archive.org/web/20221209133849/http://www.wordpress.com/) 博客发布系统的盈利分支而闻名，它可能是博客可以成为一项真正业务的最好证明。

这家由 28 岁的 WordPress 创始人[马特·莫楞威格](https://web.archive.org/web/20221209133849/http://www.crunchbase.com/person/matt-mullenweg)于 2005 年创立的公司，在 AllThingsD 的 Liz Gannes 今天发表的[采访](https://web.archive.org/web/20221209133849/http://allthingsd.com/20120425/automattic-grows-up-the-company-behind-wordpress-com-shares-revenue-numbers-and-hires-execs/)中披露了一些大数字。Automattic 实现了盈利，今年有望实现 4500 万美元的收入。WordPress——通过其开源软件和 WordPress.com 平台——现在为 7000 万个网站提供支持(顺便说一下，TechCrunch 是其中之一。)

即使从近期的角度来看，这也代表着巨大的增长。大约 16 个月前，即 2010 年 12 月，Mullenweg [告诉我们自己的 Alexia Tsotsis](https://web.archive.org/web/20221209133849/https://beta.techcrunch.com/2010/12/09/automattic-hits-300-million-unique-visitors-roughly-10-million-in-revenues/)Automattic 的收入约为 1000 万美元。当时，WordPress 为 3000 万出版商提供支持。

因此，Automattic 聘请第一位全职财务主管来关注这些数字是有道理的。教育软件公司 Kno 的前首席财务官斯图尔特·韦斯特(Stuart West)加入 Automattic，担任首席财务官。该公司还聘请了保罗·西明斯基(Paul Sieminski)担任其第一任法律总顾问(或者，正如穆伦韦格在他自己的[博客文章](https://web.archive.org/web/20221209133849/http://ma.tt/2012/04/automattic-growth/)中宣布高管任命时所描述的那样，“顾问/自动锁定者。”)与此同时，Mullenweg 仍然担任公司总裁，首席执行官一职自 2006 年以来一直由 Toni Schneider 担任。
