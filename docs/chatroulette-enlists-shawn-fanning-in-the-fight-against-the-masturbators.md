# Chatroulette 招募肖恩·范宁加入对抗自慰者的战斗 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/06/13/chatroulette-enlists-shawn-fanning-in-the-fight-against-the-masturbators/>

由[安德烈·特诺夫斯基](https://web.archive.org/web/20230209033233/http://www.crunchbase.com/person/andrey-ternovskiy)创建的俄罗斯网站 [Chatroulette](https://web.archive.org/web/20230209033233/http://www.chatroulette.com/) 可能是最出名的观看男人暴露生殖器的地方。

但是这并没有阻止每天一百万人访问这个网站。而且它已经在[每日秀](https://web.archive.org/web/20230209033233/https://techcrunch.com/2010/03/05/the-daily-show-reveals-chatroulette-to-be-filled-with-creepy-journalists/)和[南方公园](https://web.archive.org/web/20230209033233/https://techcrunch.com/2010/04/09/facebook-chatroulette-south-park/)上播出。

硅谷和其他投资者也在向 wow Ternovskiy 示好。SGN 创始人谢文·皮什瓦尔帮助他来到帕洛阿尔托，并在一间公寓里安顿下来。五月份《纽约客》[谈到他第一次访问美国时，俄罗斯的 DST 是如何跟踪他的，我们还听说天使投资人和风险资本家像鹰一样围绕着特尔诺夫斯基。](https://web.archive.org/web/20230209033233/http://www.newyorker.com/reporting/2010/05/17/100517fa_fact_ioffe?currentPage=all)

问题是该网站正在迅速失去吸引力，因为越来越多的人对在网站上暴露自己或更糟的人感到厌恶。这个品牌正变得与人类更令人厌恶的部分永久地联系在一起(借助于那些每日秀和南方公园的特写)。

聊天轮盘可以变得更有意义吗？寻找功能变化很快，将试图发送所有这些阴茎到后台。该服务可能会添加软件，可以快速扫描视频，以确定阴茎是否正在显示。那些经常被快速跳过的用户(大概是因为他们暴露了自己或者令人厌恶)也可以被标记出来。有了这些和其他的改变，Chatroulette 也许能够让那些真正想相互交流的人更加频繁地接触。

这是真正的增长可能发生的地方。一位想加入聊天轮盘赌的投资者说:“互联网上再也没有地方让你结识新朋友了。”。*“在线约会的潜力是无限的，这在很大程度上推动了早期脸书的发展。”*

但是 Ternovskiy 的谨慎可能会毁灭这个网站。我们的消息来源称，他拒绝了大多数帮助，投资者和顾问也开始放弃。*“如果他不尽快采取重大举措整顿服务，这个品牌将永久受损，”*另一位感兴趣的投资者说。

Ternovskiy 似乎确实信任的一个人是 Napster 的创始人 Shawn Fanning，他目前正在努力工作，走上他最近的创业之路。他证实，范宁是 Chatroulette 的顾问，在过去一个月左右的时间里，他一直与 Ternovskiy 密切合作。*“我被聊天轮盘和安德烈迷住了，”*他昨天告诉我，*“我想尽我所能帮助他。”*范宁强调，顾问的角色是非正式的，没有报酬，他还和许多其他企业家一起工作。

现在还不清楚 Chatroulette 接下来会发生什么。Ternovskiy 要么独立推进，试图自己发展和清理网站，要么接受硅谷的生态系统，让其他人和投资者参与进来。

很明显，人们着迷于从相对安全的电脑屏幕上与陌生人交谈的概念，聊天轮盘满足了人类与他人联系的一些基本需求。但可悲的是，这也满足了一些人向世界展示他们阴茎的基本需求。阴茎的游行把其他人都赶走了。Chatroulette 需要决定自己长大后想做什么，而且需要尽快决定。否则它将只不过是一个笑话的笑点，甚至会很快变老。
