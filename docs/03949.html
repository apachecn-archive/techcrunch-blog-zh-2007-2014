新的谷歌Chrome版本增加了对1500多个扩展的支持，书签同步TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2010/01/25/google-chrome-for-windows-extensions/amp/

太平洋标准时间2010年1月25日上午8:22

PC用户，高兴吧，因为谷歌刚刚宣布有一个全新的、稳定的谷歌Chrome for Windows 版本，增加了扩展和书签同步功能。

如果你已经在使用谷歌浏览器Windows版，你可以手动检查新版本(进入设置>关于谷歌浏览器)，或者你可以等待新版本在下周内自动更新。

谷歌之前在测试频道推出了扩展，并表示超过1500个现在已经进入扩展库。Chrome Windows浏览器的另一个新功能是书签同步，如果你使用多台电脑，这个功能尤其有用，因为它可以让你同步所有电脑上的浏览器书签。

对于那些在Linux上使用谷歌Chrome的人来说，扩展目前只在测试版上启用。至于用于Mac的谷歌Chrome浏览器，这个搜索巨头要求你坚持下去:扩展、书签同步等功能应该会很快进入测试版。或者，你可以使用Mac版Chrome的开发版本，该版本已经支持扩展。

Web开发人员和设计人员可能想在Chrome博客上深入了解这个Chrome版本的新特性，因为它包含了许多新的HTML和JavaScript APIs。

如果你使用的是支持扩展的Chrome版本，也一定要安装TechCrunch one 。它非常有效。
