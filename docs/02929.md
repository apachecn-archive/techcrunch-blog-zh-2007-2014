# 特斯拉首席执行官埃隆·马斯克澄清未决诉讼

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/06/22/tesla-ceo-elon-musk-sets-the-record-straight-about-pending-lawsuit/amp/>

 <amp-img src="https://web.archive.org/web/20230304124927im_/https://techcrunch.com/wp-content/uploads/2009/06/25849v3-max-250x250-2.png" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 

我们[报道过](https://web.archive.org/web/20230304124927/https://techcrunch.com/2009/06/11/tesla-founder-sues-ceo-elon-musk-for-slander-and-breach-of-contract/)几周前，特斯拉联合创始人马丁·艾伯哈德对特斯拉和特斯拉首席执行官[埃隆马斯克](https://web.archive.org/web/20230304124927/http://www.crunchbase.com/person/elon-musk)提起诉讼，指控其诽谤、中伤和违约。提起诉讼时，特斯拉发表了一份简短的声明，称该诉讼是“特斯拉早年的虚构账户”现在，马斯克在特斯拉的博客上给出了他自己对混乱局面的看法。与诉讼无关的是，马斯克表示，由于材料成本下降和跑车销量增加，特斯拉将在下个月实现盈利。

马斯克回应了埃伯哈德的几项指控，即他谎报了自己的背景，并虚构了一些简历来美化事实:

> 事实是，当我通过 AC Propulsion 要求与埃伯哈德会面时，他没有自己的技术，没有样车，也没有与电动汽车相关的知识产权。他所拥有的只是一份商业计划，旨在将交流推进的 Tzero 电动跑车概念商业化。三年后，当埃伯哈德被要求离开特斯拉时，他所做的大部分工作都不得不重做。

特别是，马斯克提到了埃伯哈德对贝宝创始人谎报学历的抱怨。马斯克坚持说，他在宾夕法尼亚大学/沃顿商学院(我们向沃顿商学院证实了这一点——马斯克是本科毕业生，拥有沃顿商学院和文理学院的学位)完成了物理学和商学的本科学习，尽管从斯坦福大学的研究生院辍学，但他通过与斯坦福工程咨询委员会合作，保持了与该大学的联系。

马斯克还谈到了埃伯哈德的说法，即他没有参与跑车的制造，马斯克的管理不利于业务。马斯克写道，当埃伯哈德领导运营团队时，马斯克本人专注于车身设计、技术规格和建立特斯拉品牌。马斯克还表示，他花了大量时间“研究产品的细节，特别是车身造型——你会在跑车车身上看到我最喜欢的两辆车(保时捷和迈凯轮 F1)的元素——但将公司的大多数日常管理留给了埃伯哈德。”

马斯克的故事与埃伯哈德指控马斯克和特斯拉无缘无故将他赶出公司截然不同。根据马斯克的帖子，埃伯哈德严重错误地计算了跑车的生产成本，并故意向马斯克和投资者隐瞒了这一信息:

> 跑车开发成本如此之高的真正原因不是典型的企业家狂妄所能解释的，而是我们实际上不得不花费两次开发资金。两年前，埃伯哈德被要求辞去首席执行官一职后，几乎该车的每一个主要系统，包括车身、HVAC、电机、电力电子设备、变速器和电池组，都必须重新设计、重组或转向新的供应商。

这篇文章本身就是一篇引人入胜的文章，深入探究了特斯拉早期的一些失误，以及它们是如何被克服的。例如，该公司将电池组生产从亚洲转移到美国的决定是违反直觉的，但对其生存至关重要。马斯克写道:

> 避免从亚洲运输半吨包装的费用也意味着运输成本的显著节约。对于一个笨重的产品来说，这比一个小型消费电子设备要重要得多，因为在小型消费电子设备领域，外包到亚洲更有意义。非常重要的是，我们的供应链从结构上缓慢的六个月，以及必须支付数千万美元的库存运输费用，缩短到了几周的时间。

马斯克还包括一些电子邮件来支持他对埃伯哈德的反诉，但他表示，虽然这篇文章旨在澄清一些误解，但对即将提起的诉讼的正式回应还会更多。我在下面的电子邮件中加入了一些更有趣的片段，但你可以在马斯克的博客中看到更多的电子邮件。

正如我们在之前的报告中所写的，埃伯哈德的许多主张看起来像是酸葡萄。不清楚他要求多少赔偿金。
