# 报道:Hulu 准备申请 20 亿美元的首次公开募股

据路透社报道， Hulu 正准备在 2011 年上半年上市。消息人士告诉新闻机构，Hulu 将寻求筹集 2 亿至 3 亿美元的资金，这笔交易对该公司的估值约为 20 亿美元。关于 Hulu 的 IPO 野心的报道在去年夏天首次浮出水面，但这可能意味着申请可能即将到来。

Hulu 预计将在年底前向美国证券交易委员会提交申请，这将使该公司在 2011 年 1 月至 5 月期间进行 IPO。路透社的消息来源称，该提议“取决于节目转播权的更新，其中一些将在一年后到期。”

这可能是 Hulu 为 IPO 试水的另一种方式。虽然该公司可能声称收入很高(2009 年为 1 亿美元)，但目前尚不清楚 Hulu 的盈利能力如何，或者其长期生存的可行性如何。Hulu 已经不得不将多达一半或更多的广告收入交给其电视网络合作伙伴/支持者(福克斯、NBC 和 ABC)。

Hulu 刚刚推出了付费服务 [Hulu Plus](https://web.archive.org/web/20221006223523/https://beta.techcrunch.com/2010/06/29/hulu-plus-premium/) ，这是一项由广告支持的付费订阅服务，每月收费 9.99 美元，可以高清观看福克斯、美国广播公司和全国广播公司的整季节目。这最终会带来收入和利润，但对这项新服务的初步评价并不完全正面。

Hulu 的盔甲还有其他漏洞。你不能在 Hulu 上看 CBS 的节目，今年早些时候，维亚康姆决定从喜剧中心撤掉最受欢迎的节目。康卡斯特也有自己的网络流媒体电视计划，可能包括其新收购的 NBC。

Hulu 的竞争也在加剧，它一直与网飞有着长期的竞争。随着谷歌电视和苹果电视都在同一个空间播放，Hulu 将面临更多的观众竞争。

所以说真的，在我们看到实际的申报文件或财务状况之前，很难判断 Hulu 到底有多赚钱。也就是说，Hulu 正在主导的视频广告。根据最近的 [comScore 报告，](https://web.archive.org/web/20221006223523/https://beta.techcrunch.com/2010/09/30/comscore-facebook-passes-yahoo-to-become-the-second-largest-video-site-in-the-u-s/) Hulu 每月产生近 8 亿次广告展示，甚至超过 YouTube(这一测量没有考虑覆盖广告、横幅广告等其他类型的广告)
