# StockTwits 通过新的 iPhone 应用程序将金融理念网络移动化

![](img/a34907751b48192978be2b9f1901f52f.png) [专注于 Twitter 的](https://web.archive.org/web/20221007140006/https://beta.techcrunch.com/2010/02/18/stocktwits-evolves-becomes-must-use-site-for-traders/)股票和交易网络 [StockTwits](https://web.archive.org/web/20221007140006/http://stocktwits.com/) 今天以一款[本地免费 iPhone 应用](https://web.archive.org/web/20221007140006/http://blog.stocktwits.com/?p=3921)首次亮相移动领域。

该应用程序包含了与 StockTwits 网站相同的功能。你可以在应用程序上访问所有关注股票的 Twitter 信息流，以及股票观察列表。用户可以从应用程序中更新自己关于股票的推文，并关注和查看网站上的其他用户资料。

该应用还提供了 [Chart.ly、](https://web.archive.org/web/20221007140006/https://beta.techcrunch.com/2010/08/20/stocktwits-debuts-a-brand-new-chart-ly/)一个专门为 StockTwits 设计的[股票图表服务(而](https://web.archive.org/web/20221007140006/http://www.beta.techcrunch.com/2009/04/24/the-evolution-of-twitter-spin-offs-twitter-begat-stocktwits-begat-chartly/)[随后被初创公司](https://web.archive.org/web/20221007140006/https://beta.techcrunch.com/2009/05/12/stocktwits-takes-over-chartly-to-enrich-trading-chatter-on-twitter/)收购)。在应用程序中，你可以访问交易者的图表和[截屏](https://web.archive.org/web/20221007140006/https://beta.techcrunch.com/2010/08/20/stocktwits-debuts-a-brand-new-chart-ly/)。

StockTwits 内置了来自财经新闻网站《异常新闻》(该公司于今年年初收购了《异常新闻》)及其在线视频频道的实时财经新闻。

该应用程序本质上是关于在旅途中利用 StockTwits 的金融理念和信息网络的便利。StockTwits 有一个[快速增长的社区，](https://web.archive.org/web/20221007140006/https://beta.techcrunch.com/2010/06/21/finance-sites-turn-to-stocktwits-for-curated-stock-tweets/)加上该网站提供的金融信息的时效性，移动应用肯定会受到用户的欢迎。
