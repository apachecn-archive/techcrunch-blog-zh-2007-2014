# 清洁技术应该向纳米技术学习什么(在为时已晚之前)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/08/29/what-cleantech-should-learn-from-nanotech-before-it%e2%80%99s-too-late/>

早在我们有 Web 2.0 和清洁技术来困扰硅谷之前，纳米技术就已经是热门话题了——亚原子粒子会突然成为所有东西的组成部分。它会让我们房子上的油漆更持久，我们锅上的不粘锅粘得更少，我们的裤子不会起皱。在某个地方，有人可能向他们的董事会承诺，他们可以利用纳米技术让哈利波特的隐身衣成为现实。

这似乎是一个伟大的投资理论，原因有几个:实际上有真正的可申请专利的科学，因为可能性似乎是无限的，这是一个巨大的市场。2005 年 2 月的《商业周刊》封面认为，到 2020 年，这个数字将接近 3000 亿美元。(你知道，*现在的*。)

有一些风险投资人退缩了，有一些人——像史蒂夫·杰维特森(Steve Jurvetson)——全力以赴，但大多数人处于中间位置。换句话说，他们真的不知道这个小东西是否会很大，但为了以防万一，他们想让一些马参加比赛。根据同一期《商业周刊》的封面报道，总共有超过 10 亿美元投资于该领域，大约有 1200 家初创公司成立。(典型的风险研究公司不披露纳米技术投资，因此很难找到更好的数据。我认为《商业周刊》的数字实际上相当保守，因为每次 2000 万到 4000 万美元的资助并非闻所未闻。)

从某些方面来看，这场运动是成功的。根据皮尤慈善信托基金新兴技术项目的一份新报告，现在消费者可以买到 1000 多种纳米技术产品，而 2006 年只有 212 种。这项研究的负责人 David Rejeski 告诉 PEHub，他预计在未来两年内产品的数量将达到 1600 种。

太棒了。哦，等等。不太好——出口呢？硅谷现在应该被淹没的纳米现金在哪里？像 [Nanosys](https://web.archive.org/web/20221006021757/http://www.nanosysinc.com/index.html) 、 [Nanofilm](https://web.archive.org/web/20221006021757/http://www.nanofilmtechnology.com/index.aspx?bhcp=1) 和 [Konarka](https://web.archive.org/web/20221006021757/http://www.konarka.com/index.php/company/about-konarka/) 等公司的未决 IPO 从未发生过。(这三家公司都还在运营，已经筹集了数亿美元的风险资本和私募股权。)纳米技术的出口，嗯，很小。从来没有一个巨大的，标志性的纳米技术 IPO 来证明所有的希望，并让信徒们相信和投资。

看看皮尤的研究，产品潜力是显而易见的。发生了什么事？三件事之一:这些产品的市场太小，公司不能像希望的那样扩大规模，产品和科学只是太增量而不能成为一个大热门，或者一些大型 IPO 仍在拐角处。

在某些方面，这与 Web 2.0 一代的抱怨没有太大区别。怀疑者说，大多数初创公司都是小公司，仅仅是 Y Combinator 风格的功能和应用程序，顶多以 2000 万美元左右的价格被收购。

与 Web 2.0 的不同之处在于，这些网站和应用程序的构建和托管非常便宜且快速。设计将被制造成裤子的亚原子粒子不是。你知道还有什么不是吗？清洁技术领域的大部分机会。

今年清洁技术投资的交易额下降了 30%,金额下降了 60%——从能源生产到能源节约的巨大转变。几年后，它有可能看起来更像纳米技术。对于投入清洁技术的数十亿美元，我们目前有什么？有几个公共出口。我们有少量令人瞠目结舌的杀手级产品，主要是在汽车领域，像[特斯拉](https://web.archive.org/web/20221006021757/http://finance.yahoo.com/tech-ticker/article/yftt_229945/Behind-the-Design-of-Tesla%27s-Sexy-New-Sedan?tickers=F,GM,%5EIXIC)、[菲斯克](https://web.archive.org/web/20221006021757/http://karma.fiskerautomotive.com/)和[更好的地方](https://web.archive.org/web/20221006021757/http://finance.yahoo.com/tech-ticker/article/yftt_260496/Shai-Agassi%27s-Plan-to-Get-the-World-Off-Oil?tickers=GM,F,SAP,%5EIXIC)这样的公司。还有…具体还有什么？

随着油价在去年的大部分时间里处于更合理的区域，整个“不方便说真话”的热潮已经消退，清洁技术需要一个巨大的类似网景的 IPO 来让每个人兴奋起来，并点燃对针头移动科学和开发的真正投资，而不是为了更有效地管理智能电网而玩安全的软件程序。基金会资本的风险投资人保罗·霍兰德在[的新闻发布会上说:现在有几个竞争者即将出现。[讨论接近 4 分钟。]](https://web.archive.org/web/20221006021757/http://www.pressheretv.com/)

其他人[推测](https://web.archive.org/web/20221006021757/http://earth2tech.com/2009/06/26/ipo-comeback-starting-now-cleantech-investors-say/)几家清洁技术公司在崩溃前就准备上市，预示着可能活跃的 2010 年。但其中大多数都远低于 10 亿美元的市值水平。对于一个已经投入数十亿的行业来说，这就像亲吻你的妹妹。

毫无疑问，清洁技术改造世界上几乎所有传统行业的机会是巨大的。我不怀疑它会。问题是，它能否在纳米技术失败的地方取得成功，重塑风险投资回报的黄金时代。
