# Seesmic Web 完善了对 Twitter 关注者的管理

 <amp-img class="alignright size-full wp-image-160454 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="con" src="https://web.archive.org/web/20230301155328im_/https://techcrunch.com/wp-content/uploads/2010/02/con.png" alt="" srcset="https://web.archive.org/web/20230301155328im_/https://techcrunch.com/wp-content/uploads/2010/02/con.png 300w, https://web.archive.org/web/20230301155328im_/https://techcrunch.com/wp-content/uploads/2010/02/con.png?resize=150,83 150w, https://web.archive.org/web/20230301155328im_/https://techcrunch.com/wp-content/uploads/2010/02/con.png?resize=50,28 50w" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 你在推特上关注超过 100 个人吗？如果是的话，你有没有尝试过在 Twitter.com 管理它们？太可怕了。真的很糟糕。它们让你一页一页地浏览名字，没有真正的顺序(除了你最近添加的名字)。如果你想除掉一些你多年前追随的人，那是一种巨大的痛苦。Seesmic 刚刚提出了一种更好的方法。

今天发布的新的 Seesmic 网络应用程序带来了一个新的主标签:联系人。正如你所猜测的，这是一个 Twitter 的联系人管理器，它使得删除你的 Twitter 联系人变得轻而易举。它还会给你很多其他有趣的信息，比如你的关注者，那些关注你的人，以及你所有的 Twitter 列表。例如，对于每个人，你不仅可以看到他们所有的 Twitter 个人资料信息，还可以从 MrTweet 上看到他们的 Twitter 统计数据，这些数据会告诉你他们发 Twitter 的频率以及他们在 Twitter 上(公开)联系最多的人。

所有这些都以一种看起来非常类似于 Gmail 内置的谷歌联系人管理器的方式包装起来，所以你们很多人应该都很熟悉。但这实际上更好，因为他们在每个用户的名字旁边显示 Twitter 图标，这是一种简单的分类方法。

Seesmic Web 增加的另一个新的联系人管理功能是能够拖放任何用户，将他们添加到列表中。同样，这比在 Twitter.com 做要容易 1000 倍。正如你所想象的，Seesmic Web 现在还可以创建、删除和修改 Twitter 列表。

其他新功能包括更好的对话线程(如果你点击推文下方的“回复”部分，你会看到整个来回的覆盖图)。同样新的是 Tweetmeme 转发推文中每个链接的数据。只需点击链接旁边的“+”按钮，即可看到该数据的覆盖图(包括文本片段和图像)。还增加了从应用程序直接上传图片，并用地理位置标记推文的功能(假设你使用的是谷歌 Gears 之类的东西，可以将它添加到浏览器中)。其他较小的 UI 调整也已经完成。

随着过去几个月的转发和列表的增加，Twitter.com 再次成为一个非常有用的客户端。但是其他基于网络的 Twitter 客户，如 Brizzly 和 Seesmic Web，显然相信他们仍然可以比 Twitter 做得更好。有了这个联系人管理系统，Seesmic Web 给了我一个开始使用它的明确理由。
