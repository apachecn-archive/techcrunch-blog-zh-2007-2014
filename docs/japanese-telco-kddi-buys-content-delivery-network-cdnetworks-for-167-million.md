# 日本电信公司 KDDI 以 1.67 亿美元收购内容交付网络 CDNetworks

今天来自亚洲网络世界的重大新闻:日本电信巨头 KDDI T1([400 亿美元收入](https://web.archive.org/web/20230205034929/http://money.cnn.com/magazines/fortune/global500/2011/snapshots/10328.html) ) [宣布将以 1 . 67 亿美元的价格收购韩国内容交付网络](https://web.archive.org/web/20230205034929/http://www.kddi.com/corporate/news_release/2011/1012/index.html)[CDNetworks](https://web.archive.org/web/20230205034929/http://www.cdnetworks.com/)85 . 5%的股份。

KDDI 表示，该计划是将 CDNetworks 在首尔的总部变成一家子公司，并扩大其全球业务(CDNetworks 在美国、中国、欧洲以及 KDDI 的本土市场日本也设有办事处)。

CDNetworks 的核心产品是一个 [Web 性能套件](https://web.archive.org/web/20230205034929/http://www.cdnetworks.com/solutions/web-performance-suite/)，可为客户加速 Web 内容交付和应用性能。该公司目前在位于 31 个国家共 70 个城市的服务器上提供 130 多个内容存储点。通过此次收购，KDDI 希望为未来几年预计的[移动数据流量爆炸](https://web.archive.org/web/20230205034929/https://techcrunch.com/2010/03/30/mobile-data-traffic-rise-40-fold/)做好准备(KDDI 本身在日本拥有超过 3000 万移动用户)。

CDNetworks 于 2000 年在首尔成立，在韩国证券交易所上市，目前在全球拥有超过 1200 名客户(包括 LinkedIn、Yelp 或德国社交网络 StudiVZ)。该公司上一财年的收入达到 9900 万美元，并声称自 2007 年以来销售额平均每年增长 20%。
