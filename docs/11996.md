# 新的社交发现应用 Ntro 发布，获得 170 万美元的资金支持 TechCrunch

总部位于洛杉矶的初创公司 nProgress 今天上午发布了一款新的社交发现应用程序 ntro，该应用程序得到了贾尔·莫恩、戈登·克劳福德、凯文·沃尔、Activation Media Partners、巴罗达风险投资公司、SV Angel、杰夫·普尔弗和理查德·克罗威尔超过 170 万美元的资金支持。这款应用程序现在可以在 iPhone 和 Android 上使用，旨在让你和周围你可能通过共同的朋友认识的人或者有共同兴趣的人联系起来。

但是与这个领域的其他公司(有很多)不同的是， [ntro](https://web.archive.org/web/20230130221710/http://www.ntro.com/) 正在向它的目标市场展示。它今天在洛杉矶和旧金山湾区推出，并计划未来在其他城市推出。

Ntro 提供了一个有趣的功能组合，我们以前在其他“社交发现”应用中见过，比如[声纳](https://web.archive.org/web/20230130221710/http://www.sonar.me/)、[班卓琴](https://web.archive.org/web/20230130221710/http://ban.jo/)、 [Mingle](https://web.archive.org/web/20230130221710/http://itunes.apple.com/us/app/mingle!/id443064933?mt=8) 和 [Holler](https://web.archive.org/web/20230130221710/http://itunes.apple.com/us/app/holler/id461565363) 等等。像声纳一样，ntro 突出了附近那些你可能通过共同的朋友认识的人。和 Holler 一样，它将人们围绕利益联系在一起。和 Mingle 一样，它专注于做介绍。像班卓琴一样，它会自动完成这一切。

[<amp-img class="vertical alignright size-medium wp-image-461843 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="ntro-2" src="https://web.archive.org/web/20230130221710im_/https://techcrunch.com/wp-content/uploads/2011/12/ntro-2.jpg?w=200" alt="" srcset="https://web.archive.org/web/20230130221710im_/https://techcrunch.com/wp-content/uploads/2011/12/ntro-2.jpg 320w, https://web.archive.org/web/20230130221710im_/https://techcrunch.com/wp-content/uploads/2011/12/ntro-2.jpg?resize=100,150 100w, https://web.archive.org/web/20230130221710im_/https://techcrunch.com/wp-content/uploads/2011/12/ntro-2.jpg?resize=200,300 200w, https://web.archive.org/web/20230130221710im_/https://techcrunch.com/wp-content/uploads/2011/12/ntro-2.jpg?resize=33,50 33w" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230130221710/https://techcrunch.com/wp-content/uploads/2011/12/ntro-2.jpg)ntro 可能有一点优势，至少现在是在它提供的定制方面。有广泛的过滤器和控制提供，允许您设置您的最高利益，以及过滤和搜索周围人群非常狭窄的利益。也就是说，不仅仅是“音乐”或“摇滚音乐”，而是一个喜欢的乐队的名字。

此外，该应用还非常注重隐私——未经你的同意，它不会分享你的位置，也不会分享你的个人信息(如电话号码、电子邮件甚至姓氏)。

有了这些广泛的保护，你和你周围的人之间就有了一点距离，这与社交中令人毛骨悚然的班卓琴或声纳完全不同。在测试 ntro 时，我甚至看不到用户的个人资料照片，这要么是由于照片审批过程(有延迟)，要么是用户自己选择不显示他们的脸(这是一个选项)。但是如果你真的认为你已经找到了基于兴趣的灵魂伴侣/商业关系/新的最好的朋友，你可以通过应用程序给他们发信息。

ntro 的另一个突出特点是你浏览兴趣的封面流方式，每一个项目，无论是乐队，一本书，一项运动，一项活动等。，有自己的缩略图。然后，您只需点击缩略图，查看附近还有谁喜欢该商品。

目前，任何人都可以下载该应用程序并四处查看，但只有在洛杉矶和 S.F .的人会找到附近的匹配。首次展示的重点是大都市地区，下一个计划的城市是纽约、芝加哥特区和波士顿。

你现在可以通过下载 [iTunes](https://web.archive.org/web/20230130221710/http://itunes.apple.com/us/app/ntro/id417886713) 或者 [Android Market](https://web.archive.org/web/20230130221710/http://market.android.com/details?id=com.nprogress.android.ntro&utm_campaign=ntro.com) 来试用这个应用。
