# HTC 的第一款 NFC 手机 HTC 惊艳其实很不可思议

本周早些时候，智能手机制造商 HTC 展示了其首款基于 NFC 的设备，据报道称其名为“HTC 惊艳”这款手机将在中国发布，并将通过与中国银行卡网络中国银联的合作支持移动支付。

然而，我们不知道的是，这款惊艳的手机是哪一款。现在我们做到了:这只是带有 NFC 芯片的 HTC Incredible S。还有，它不叫“HTC 惊艳”这是中文“难以置信”的一个小小的误译

HTC 告诉我们，HTC 和银联之间的战略合作伙伴关系将包括技术 R&D、移动支付服务推广和未来的增长机会。

购买该设备的用户将能够支付中国移动、中国联通和中国电信的账单和公用事业账单，检查他们的可用信用，赚取忠诚度积分并预订酒店，购买电影票和其他活动的门票。银联的合作银行也将向手机用户提供手机银行和支付服务，用户可以在线注册。

此外，由于手机的 NFC 芯片([近场通信](https://web.archive.org/web/20230203160836/http://en.wikipedia.org/wiki/Near_field_communication))，中国版的 Incredible S 将允许在销售点进行移动支付。

HTC 研发和运营总经理 Fred Liu 表示，HTC 相信移动支付将成为智能手机提供的“基本功能”之一。他认为，通过与中国银联的战略合作伙伴关系，可以建立一个完善的移动支付系统，并推动智能手机移动支付的普及。
