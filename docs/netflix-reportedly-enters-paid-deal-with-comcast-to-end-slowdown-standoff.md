# 网飞与康卡斯特达成直接接入协议

*【更新:我们最初对网飞-康卡斯特协议的看法被称为[任务，因为它暗示](https://web.archive.org/web/20230128103110/http://blog.streamingmedia.com/2014/02/media-botching-coverage-netflix-comcast-deal-getting-basics-wrong.html)它将改变网络中立性如何应用于公共基础设施。正如 Dan Rayburn 解释的那样，网飞和康卡斯特之间的交易意味着两家公司绕过第三方分销商，转向直接的提供商到服务模式。这对于网络中立性的影响将在随后的文章中讨论]*

网飞[与互联网服务提供商康卡斯特](https://web.archive.org/web/20230128103110/http://online.wsj.com/news/articles/SB10001424052702304834704579401071892041790?mg=reno64-wsj&amp;url=http%3A%2F%2Fonline.wsj.com%2Farticle%2FSB10001424052702304834704579401071892041790.html&amp;fpid=2,7,121,122,201,401,641,1009)达成一项开创先例的交易，获得通过其网络提供流媒体视频的直接使用权。这笔交易将排除第三方传输提供商，使网飞能够更有效地向康卡斯特用户传输视频。

网飞吞噬了惊人数量的全球带宽。据估计，2012 年，网飞[占据了北美](https://web.archive.org/web/20230128103110/http://news.cnet.com/8301-1023_3-57546405-93/netflix-gobbles-a-third-of-peak-internet-traffic-in-north-america/)1/3 的流量。这一数字可能在上周达到顶峰，当时网飞 3300 万观看者中有 2%疯狂观看了他们的新(超级上瘾的)政治惊悚片《纸牌屋》。

《华尔街日报》解释道:“作为支付的交换，网飞将直接接入康卡斯特的宽带网络。”。如果网飞真的为更快的直接访问向康卡斯特付费，这将为互联网的未来树立一个重要的先例。

这笔交易将使网飞能够直接连接到康卡斯特的网络，而不是付钱给第三方传输提供商来卸载其内容。

以前，网飞通过 cdn 和转接提供商路由流量，这些提供商在选定的对等点连接康卡斯特的网络。但是随着网飞流量的增加，这些对等点已经不堪重负。

康卡斯特的交易开创了一个先例，网飞可以直接与其他主要的互联网服务提供商如威瑞森和美国电话电报公司连接

联邦通信委员会[正在考虑](https://web.archive.org/web/20230128103110/https://techcrunch.com/2014/02/19/fcc-wont-appeal-net-neutrality-ruling-seeks-diplomatic-way-to-keep-internet-open/)如何推进互联网开放接入的新规则以及公司和互联网服务提供商之间的付费服务交易；目前还不清楚他们是否会找到一种方法来阻止更多像网飞和康卡斯特之间的交易。当然，这将增加寻找某种解决方案来平衡网络拥塞和开放互联网接入的政治压力。
