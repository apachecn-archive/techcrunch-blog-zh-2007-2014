# 安全专家布莱恩·克雷布斯在发布了关于俄罗斯网络犯罪网站的帖子后，面临着现实生活中的痛击

安全作家布莱恩·克雷布斯(Brian Krebs)的电话被盗用，上周，一个特警队被黑客召集到他家，黑客们显然对他对各种在线黑客组织的调查感到不满。他的网站也遭受了短时间的拒绝服务攻击。

这些攻击发生在 Krebs 写了一篇关于[一个叫做 ssndob.ru](https://web.archive.org/web/20230318112623/http://krebsonsecurity.com/2013/03/credit-reports-sold-for-cheap-in-the-underweb/) 的网站的文章之后，该网站专门为不知情的受害者出售信用报告和信用信息。该网站出售的完整报告售价 15 美元，记录清单“包括名、中、姓，加上目标的地址、电话号码、SSN 和出生日期”每份 50 美分。

攻击开始于 [Krebs 的主机提供商收到一封假邮件](https://web.archive.org/web/20230318112623/http://krebsonsecurity.com/wp-content/uploads/2013/03/BogusFBILetter.txt)，发件人声称来自联邦调查局。几天后，在 3 月 15 日，他的提供商报告了一次 DDOS 攻击，可能来自[Booter.tw](https://web.archive.org/web/20230318112623/http://booter.tw/)，一家网络压力测试公司有一个漏洞，让[调查人员评估](https://web.archive.org/web/20230318112623/http://www.reversecurity.com/2013/03/analysis-of-bootertw.html)攻击来自哪里。攻击的来源是“countonme@gmail.com”电子邮件地址的用户。

与此同时，Krebs 在家里收到了敲门声，迎接他的是武装警察，他们正在回应一个虚假的电话，声称俄罗斯人已经枪杀了 Krebs 的妻子。警方短暂拘留了克雷布斯，然后在他们能够讨论此事以及该研究人员在 2012 年向警方报告的先前威胁后，让他离开。

他写道:

当我打开门把剩下的胶带撕下来的时候，我听到有人大喊:“不许动！把手举起来。”我从蹲着的地方抬头望去，看见一名费尔法克斯县的警官斜靠在一辆巡逻车的后备箱上，伸出双臂，用手枪指着我。当我慢慢地把头转向左边时，我看到了大约六辆警车，警灯闪烁着，更多的警察用枪指着我，包括一支猎枪和一支半自动步枪。我被指示面向房子，退下我的前台阶，并倒退到毗邻的停车场，之后我被戴上手铐，走到街的顶端。

简而言之，克雷布斯同时遭受了虚拟的和“动态的”攻击，老实说，这是相当罕见的情况。虽然打——报假警的行为——经常被谈论，但很少看到它在现实中发生。幸运的是，克雷布斯很冷静，警察的反应也同样乐观。

写黑客一直是一个危险的游戏，而且显然变得更加危险。虽然克雷布斯揭露的许多人只不过是小偷小摸，但他们对毫无戒心的研究人员造成的伤害可能是巨大的。幸运的是，事情没有变得更糟。
