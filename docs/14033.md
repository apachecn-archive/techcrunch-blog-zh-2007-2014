# Circa 的新 iOS 应用将改变你阅读新闻的方式 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/>

这些年来，我们在互联网上消费信息的方式发生了巨大的变化。主要原因是因为我们可以获得的内容比我们在地球上的时间所能消费的要多。

过去，我们唯一的新闻选择是报纸，然后最终是当地电视台或广播电台告诉我们在我们邻近地区发生了什么。Circa 今天发布了它的 iOS 应用程序，它旨在改变我们消费和保留信息的方式。这是一个极端的方法，但却是一个实际可行的方法。

我们[大约在 4 月](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/04/27/all-star-cast-invests-750k-in-ben-huh-and-matt-galligans-mobile-news-startup-circa/)的时候告诉过你，当时我们只知道该公司有一份令人印象深刻的投资者名单，其首席执行官兼联合创始人[马特·加利根](https://web.archive.org/web/20230213034036/https://twitter.com/mg)，前 SimpleGeo 的员工，有着远大的抱负:

> 我们试图让人们花 5 分钟自学，而不是玩愤怒的小鸟。

[![](img/4f8ed8c4dde3a27c6fda2cebb74e338e.png "AppIcon_1024x")](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/appicon_1024x/) 当时，我不知道 Galligan 是什么意思，因为他的团队在过去的六个月里一直在埋头打造这种专为移动设备打造的新体验。通常，公司会给你一些关于它正在做什么的信息，但是 Galligan 对此守口如瓶。他告诉我，他实际上很惊讶团队能够对 Circa 及其项目保密，因为这肯定不是“硅谷”的方式。

大约在的[会带来新的和令人敬畏的东西吗？作为一个新闻迷，我可以诚实地说是的，是的。](https://web.archive.org/web/20230213034036/http://cir.ca/)

### 向 Circa 问好

这是一个用手机阅读新闻的人的典型经历:你在脸书或推特上看到一个链接，然后你被带到一个很长的手机页面，希望如此，然后你就可以开始阅读了。Galligan 告诉我，这种经历是破碎的。为什么？因为当你在移动设备上时，你真的没有时间阅读整篇文章，你真的在寻找关于世界上正在发生的重要的点点滴滴。

在过去的几周里，我一直在玩 Circa，这种“吃零食”的方式让我印象深刻。当 Galligan 告诉我们你会花五分钟看新闻而不是愤怒的小鸟时，他是认真的。该公司有一个编辑和作家的花名册，他们将挑选一个热门的故事。一旦他们有了故事，他们会把它分成应用程序中的重要部分，你可以滚动浏览。

[![](img/9047c27cdb1f9179a7376e88baa0c28a.png "photo 5") ](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/photo-5-15/) [ ![](img/a93b42b3aa42fa8d6d94feedf1ca3949.png "photo 1")](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/photo-1-55/)

当你浏览一条新闻时，你会发现丰富的数据，如嵌入的地图，这将为你阅读的内容提供更多的背景信息。此外，Circa 的工作人员会指出他们故事的来源，你可以点击这些来源，然后转向不同的方向。这有点像选择你自己的冒险，但是有真实的世界新闻。

由于很难保留这么多信息，Circa 允许你跟踪你感兴趣的新闻，并在有新消息报道时更新你的信息。你可以深入一个故事，找到你感兴趣的领域，然后跟着它走。

[![](img/a68955b95572e1246a88a57ac2907c90.png "photo 2") ](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/photo-2-45/) [ ![](img/e5139d784ca08c79149af2a86693e599.png "photo 3")](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/photo-3-35/)

我越是使用这个应用程序，它就变得越有用。当你关注一个感兴趣的项目时，它真的很难得到一个通知，告诉你还有更多的内容要读。这就好像 Circa 把新闻阅读变成了一个游戏，在这里你要确保自己跟上所有的最新发展。现在的情况是，我在 iPhone 上看到一篇我喜欢的文章，然后我会用 Pocket 查看它，或者在 Twitter 上收藏它。这是一个不完整的系统，因为它要求我在电脑前阅读全文。

Circa 完全颠覆了这个过程，而且这个应用程序非常漂亮。Galligan 告诉我，这个团队只是想开发一个他们想要使用的应用程序，这个应用程序可以让他们获得维基百科提供的深度信息，但速度要适合 iPhone 或 iPad 等移动设备。我很高兴 Circa 在过去的六个月里一直保持低调，因为努力工作得到了回报。

### 新闻的未来？

这种连珠炮式的小新闻传播会开创一种新趋势吗？嗯，这是已经在 Twitter 上发生的事情。通常情况下，我首先在 Twitter 上获得突发新闻，长度不超过 140 个字符。有时会有一个完整文章的链接，但有时是讲述整个故事的信息。例如，当旧金山发生地震时，我会跟踪一个账户，它会告诉我地震发生的时间和震级。我不必进一步阅读任何东西，我得到了要点。Circa 是这两种模式的混合体，我个人很喜欢它。

[![](img/3266b9ca6b366c298ede40b86d2dc6ad.png "circa-menu")](https://web.archive.org/web/20230213034036/https://techcrunch.com/2012/10/15/circas-new-ios-app-will-change-the-way-you-consume-news/circa-menu-2/)

正如 Galligan 在 4 月份向我们指出的那样，Circa 不是像 Techmeme 或 Digg 那样的新闻聚合游戏，而是更大的东西。通过让真正的记者关注新闻，Circa 能够在应用程序中填充人们想要阅读的内容，以一种他们可以在旅途中快速阅读的方式。当我坐在门口等火车去市区或机场时，我只想知道“事实”。

在我们坐下来的时候，Galligan 浏览了一些新闻，向我展示了这种体验是多么的流畅。这显然是他和 Circa 车队热爱的工作，也是一种从未有人尝试过的体验。当你在一个充满聪明头脑和才华的小型创业公司得到这种组合时，伟大的事情就会发生。尽管这款应用今天才刚刚推出，但 Galligan 告诉我，可能会有机会将其平台授权给其他新闻机构，一旦你尝试一下，你就会明白为什么了。它不仅仅是一个应用程序，它还是一个变革的平台。

大约以一种奇妙的方式呈现事实，它的用户界面/UX 是一种全新的东西，你以前从未见过。有了 Circa 已经提供的资金和支持，我希望这个团队能做出非常棒的事情，即使它想像苹果一样保密。
