# 视频探索初创公司 Boxfish 从大西洋桥和三星融资 700 万美元

Boxfish 是过去几年涌现出的众多帮助电视观众更好地找到他们想看的节目的创业公司之一。现在，它希望将这项技术扩展到许多消费电子产品制造商、有线电视提供商和第三方应用程序开发商，并筹集了 700 万美元的新资金。

通过解析与电视直播一起发布的字幕中提供的信息，Boxfish 能够实时了解数百个电视频道上正在发生的事情。它可以利用这些信息引导观众观看他们想看的内容。

一年半前，Boxfish 利用这项技术[推出了一款 iPad 应用](https://web.archive.org/web/20221006065638/https://beta.techcrunch.com/2012/08/14/boxfish/)。不久之后，它又推出了一款提供相同功能的[安卓应用](https://web.archive.org/web/20221006065638/https://beta.techcrunch.com/2013/05/10/boxfish-android-api/)，以及一个供第三方开发者构建其平台的 API。

现在，不到一年后，Boxfish 已经有 100 多个第三方开发者使用它的 API 和许可数据来构建他们自己的视频发现工具。这包括许多大型有线电视公司和消费电子产品制造商，他们计划使用这些工具来更好地了解他们的观众，并向他们提供内容。

这项新业务让 Boxfish 超越了自己的第二屏应用，提供了一个额外的内容发现层，可以直接添加到有线机顶盒和联网电视机中。通过向 CE 制造商、广播公司和服务提供商许可 API 访问，这些合作伙伴能够构建下一代产品，让观众了解他们以前可能不知道的节目和电影。

考虑到这一点，该公司已经筹集了一些额外的资金——来自都柏林投资公司 Atlantic Bridge Ventures 和三星的 700 万美元。虽然三星提供了一些战略资金，并可能帮助 Boxfish 在其产品中获得更多分销，但这不是一项独家交易，该公司将继续与其他 CE 制造商合作。

据联合创始人 Eoin Dowling 称，随着公司开始服务于新的市场，这笔资金将用于增加员工人数。例如，它最近在都柏林设立了一个办事处，准备在那里也提供同样的发现数据。
