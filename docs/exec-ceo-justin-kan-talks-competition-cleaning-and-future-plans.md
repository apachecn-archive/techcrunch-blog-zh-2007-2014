# Exec 首席执行官贾斯汀·坎谈论竞争、清洁和未来计划

紧随 Exec 在纽约推出清洁服务之后，我们忍不住有机会邀请首席执行官兼联合创始人贾斯汀·坎(Justin Kan)在工作室进行一次简短的交谈。我们也忍不住查看了这项服务的运行情况，因此从该公司的独立清洁应用程序中聘请了高管来帮助整理我们在纽约的 Aol Ventures 婴儿床。

对于不熟悉 [Exec](https://web.archive.org/web/20230322160429/https://techcrunch.com/tag/exec/) 的人来说，这个概念相当简单。Exec 允许你根据具体情况雇人跑腿和打扫卫生。该应用最初是作为跑腿服务推出的，让登录该平台(并通过背景调查)的高管跑去给你拿咖啡、拿干洗的衣服或为你组装宜家床。你每小时的报酬是 25 美元，其中大部分归高管，一部分归公司。

在意识到清洁占据了 Exec 超过 50%的订单后，该公司在 2 月份推出了一款独立的清洁应用程序[。从那以后，T4 的清洁业务已经扩展到了波斯顿、洛杉机、芝加哥和纽约。](https://web.archive.org/web/20230322160429/https://techcrunch.com/2013/02/12/exec-cleaning-appseattle/)

“我们希望清洁应用程序是你能想象到的最简单的东西，”贾斯汀说。“我们不希望你不得不做任何决定。”

他提到，目前大多数清洁服务强迫你打电话和呼叫一些服务。与此同时，在 Exec 上，你只需打开应用程序，就能立即看到你的房屋清洁和报价。

但是，尽管清洁占了高管业务的很大一部分，我们也不能忘记跑腿。我问贾斯汀，Exec 是如何让用户改变行为的，因为年轻群体中的大多数人对指导别人做家务感到有些不舒服。但 Exec 在应用程序中添加了一些层，提示你可以用它做什么，比如让某人在早午餐时为你排队。(Justin 实际上就是为了这个目的而使用 Exec 的。)

随着该公司继续进入新市场，最大的挑战(据贾斯汀称)是随着应用规模的扩大处理后端的物流。

在过去的八个月里，超过 11，000 人申请成为执行承包商。该公司不仅要处理大规模的招聘漏斗，完成面试和背景调查，还要处理现场实时任务调度数千名工人。
