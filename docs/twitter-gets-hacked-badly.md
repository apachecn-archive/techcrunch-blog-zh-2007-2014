# 推特被黑了，很严重

![](img/74107f549d36e1bf5cf2a63730817724.png)

周末袭击 Twitter 的网络钓鱼攻击标志着一项服务*已经来到*(脸书也有同样的问题)。但是有人入侵了推特的内部管理工具，并[泄露了包括当选总统巴拉克·奥巴马在内的 33 个高调账户](https://web.archive.org/web/20230403221043/https://techcrunch.com/2009/01/05/either-fox-news-had-their-twitter-account-hacked-or-bill-oreilly-is-gay-or-both/)，使得推特用户[不知所措](https://web.archive.org/web/20230403221043/http://search.twitter.com/search?q=change+password)。

以下是 Twitter 官方[解释](https://web.archive.org/web/20230403221043/http://blog.twitter.com/2009/01/monday-morning-madness.html):

> 今天早上，我们发现 33 个推特账户被“黑”了，其中包括著名的推特用户，如里克·桑切斯和巴拉克·奥巴马(由于过渡问题，奥巴马自当选总统后就没有发过推特)。我们立即锁定了账户并调查了这个问题。里克、巴拉克和其他人现在重新控制了他们的账户。
> 
> 发生了什么事？
> 
> 这 33 个账户的问题不同于本周末针对 Twitter 用户的网络钓鱼骗局。这些帐户被一个人侵入了我们的支持团队用来帮助人们在忘记或卡住时编辑与他们的 Twitter 帐户相关的电子邮件地址的一些工具。我们认为这是非常严重的安全漏洞，并立即将支持工具下线。只有当它们安全可靠时，我们才会把它们放回去。

今天早上，我们大多数人都从巴拉克·欧巴马、布兰妮·斯皮尔斯、比尔·奥雷利和其他人的推特账户上留下的各种信息中得到了很好的笑声。但另一个信息清晰而响亮——Twitter 还没有为黄金时段做好准备，尽管用户继续涌向这项服务。
