# Zynga 计划游戏发布，首先分享一分钟的虚荣指标:64k 字，140k 总转等。

我们今天在旧金山公司总部出席 Zynga 的大型新闻发布会。马克·平卡斯(Mark Pincus)在台上分享了一些关于该公司的新数据——他说，还会有更多关于游戏发布的消息。

但是首先，这篇文章是关于统计数据的。他没有分享每日活跃用户数(根据 [AppData](https://web.archive.org/web/20221208005600/http://appdata.com/devs/10-zynga) 的数据，目前为 5330 万)、ARPU、ARPPU 或其他行业指标。

他分享了 Zynga 游戏在全球范围内最后一分钟的平均活动类型——虚荣指标，旨在说明 Zynga 的整体规模。

*   《与朋友的话》中的 64000 字
*   43，000 张图片在画什么
*   德州扑克 38，000 手扑克
*   跨游戏好友送 25 万礼物
*   总共 140，000 圈
*   5700 次实时聊天。
