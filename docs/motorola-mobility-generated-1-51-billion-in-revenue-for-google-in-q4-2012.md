# 在谷歌 2012 年第四季度 144.2 亿美元的收入中，摩托罗拉移动业务贡献了 15.1 亿美元

谷歌刚刚报告了其 2012 年第四季度的收益，这意味着我们也可以一窥摩托罗拉移动公司的手机制造商的情况。这一次，摩托罗拉创造了 15.1 亿美元的收入，占谷歌本季度总合并收入的 11%。

从一个角度来看，摩托罗拉业务部门上个季度带来了 25.8 亿美元的收入，但这是有原因的。就在几周前，谷歌宣布 T2 以 23.5 亿美元的价格将其家庭宽带业务出售给 Arris T3。这笔交易让这份特别的收益报告变得有些混乱，并促使这家搜索巨头在其投资者关系网站上发布了一份[罕见的会计公告](https://web.archive.org/web/20221006235524/http://investor.google.com/releases/2013/0118.html)，建议所有人注意他们如何处理这笔交易。

似乎是为了先发制人地回答我们迫切的问题，谷歌的财报几乎立刻指出，如果包括摩托罗拉家庭业务，该公司的总收入将达到 152.4 亿美元。记录收入的下降不仅仅是因为这里的会计实务——仅摩托罗拉的移动部门上个季度就创造了 17.8 亿美元的收入，仍然高于我们在这里看到的收入。此外，谷歌还报告了 3.53 亿美元的 GAAP 运营亏损，相当于营收下降了-23%。

我们至少可以将这种低迷部分归因于摩托罗拉移动最近推出的硬件数量的变化——在去年年底推出 Droid RAZR HD 和 Droid RAZR Maxx HD 之后，这家受欢迎的手机制造商明显缺乏硬件发布。不过，这一切可能很快就会改变，因为关于摩托罗拉“X Phone”的持续传言表明，一款高端智能手机很可能会出现在该国所有四大无线运营商的手机上。诚然，其中一些传言比其他的更古怪，但它们在媒体上受到的关注意味着谷歌和摩托罗拉有一群全神贯注的观众等着剖析和消化任何令人印象深刻的新硬件产品。
