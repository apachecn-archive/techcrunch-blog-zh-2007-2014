# 伊万·威廉姆斯，罗恩·康韦，卡特里纳虚假和真实的风险投资网页排版创业小批量

![](img/cab2f93e98f38334e0891b2c2c5fd006.png)

秘密创业公司[小批](https://web.archive.org/web/20221006204631/http://smallbatchinc.com/)已经[从](https://web.archive.org/web/20221006204631/http://blog.typekit.com/2009/06/24/small-batch-inc-announces-typekit-funding/) [True Ventures](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/financial-organization/true-ventures) 筹集了一轮未披露的股权融资，Twitter 联合创始人[伊万·威廉姆斯、](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/person/evan-williams) Flickr 和 Hunch 联合创始人 [Caterina Fake、](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/person/caterina-fake)WordPress/automatic 创始人[马特·莫楞威格、](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/person/matt-mullenweg)著名投资人[罗恩·康韦](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/person/ron-conway)、[克里斯·萨卡、](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/person/chris-sacca)乔希·费尔塞和戴夫·塞缪尔参与其中。Small Batch 正在推出 [Typekit，](https://web.archive.org/web/20221006204631/http://typekit.com/)一项让设计者用网络原生排版建立网站的服务。

Small Batch 由 [Jeffrey Veen、](https://web.archive.org/web/20221006204631/http://www.crunchbase.com/person/jeffrey-veen)共同创立，他是 Adaptive Path 的创始合伙人之一，也是 Measure Map 的项目负责人，Measure Map 是一款广受欢迎的网络分析工具，被[收购](https://web.archive.org/web/20221006204631/http://www.beta.techcrunch.com/2006/02/14/google-buys-measuremap/)于谷歌。收购后，Veen 在谷歌工作，并于 2009 年 1 月开始小批量生产。几周前，维恩还获得了 [TechFellow 奖](https://web.archive.org/web/20221006204631/http://www.beta.techcrunch.com/2009/06/05/2009-techfellow-awards-the-winners/)。

关于 Typekit 的细节仍然有限，但 Veen 说，该公司希望“建立一个灵活、安全的工具，使网页设计师更容易在网上进行惊人的设计。”Typekit 计划在未来几周推出这项服务的预览版，其中包含一个高质量字体和设计排版工具库。
