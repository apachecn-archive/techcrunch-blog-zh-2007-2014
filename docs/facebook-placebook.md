# 脸书胁迫 PlaceBook 改变他们的名字——或者发音

到目前为止，你还没有听说过 [PlaceBook](https://web.archive.org/web/20230201001201/http://placebook.com/) 的主要原因是因为他们仍然处于隐身状态。但是如果你以后再也听不到他们的消息，那是因为脸书在威胁他们，除非他们改名。对于一个基于位置的服务来说，PlaceBook 似乎是一个杀手级的名字。它也恰好与另一个社交网络服务押韵——他们不太喜欢那个。所以 PlaceBook 很有趣。

在下面的视频中，PlaceBook 创始人[迈克尔·鲁宾](https://web.archive.org/web/20230201001201/http://www.crunchbase.com/person/michael-rubin-2)解释了为什么他不应该改变名字——只需要改变发音。不是 PlaceBook，是 placèbok，区别很大。

博主 Robert Scoble [向世界展示了六月份的 PlaceBook。但是，他们仍然处于隐形模式，所以它在很大程度上是保密的。他们绝对值得一看，因为这是一群在外景领域工作的前网飞高管。只是还不清楚我们要用什么名字来看它们——哦，对了，Placè Book。](https://web.archive.org/web/20230201001201/http://scobleizer.com/2010/06/13/exclusive-first-look-at-location-based-service-placebook/)
