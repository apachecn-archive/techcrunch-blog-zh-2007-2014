# JiffPad 通过 iPad 创建个性化医疗图表 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/13/jiffpad-creates-personalized-medical-diagrams-via-ipad/>

Jiffpad 试图“重新发明医疗保健交流”，它是一款让医生、护士、药剂师和其他医疗专业人员通过 iPad 与患者交流的应用程序。

该应用允许用户从已经提供的医疗图表中进行选择或上传自己的照片，创建了一个“JiffTalk”，将医生和患者之间的交流数字化，并创建了一个医疗指导的数字记录，然后患者可以通过电子邮件发送回家或与家人分享。

JiffPad 由白内障或血管成形术等类别的混搭视觉组件组成，适用于大多数医疗实践。因为它本质上是可视化的，这款应用并不局限于一种特定的语言。

JiffPad 首席执行官菲尔·卡特说:“与墙报或解剖模型进行的对话不会成为你参观的纪念品，”他认为这款应用主要与传统的医学图表、纸张和铅笔以及电话和传真等简单的患者沟通工具竞争。

Carter 说，JiffPad 相对于其他医疗图表工具的优势在于，它是针对特定患者的，并且具有无限的可移植性，例如，允许医生创建他们经常使用的图表的播放列表。

卡特计划将 JiffPad 货币化，向该应用程序出售许可证，并向医疗专业人员提供额外的存储空间，同时保持对患者免费。他也在考虑赞助内容的可能性。当被问及是否会考虑将丰富多彩的混搭图扩展到其他垂直领域时，他告诉我，“在医学领域有很多事情要做。”

JiffPad 从 Aberdare Ventures 获得了 100 万美元的资金。

约西·瓦迪、马克·苏斯特、谷歌的陈德伟、Quora 的丽贝卡·考克斯等评委的提问，引发了“乔·格林”事件
