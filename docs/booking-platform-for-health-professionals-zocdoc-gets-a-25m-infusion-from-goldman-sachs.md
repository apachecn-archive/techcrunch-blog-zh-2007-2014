# 医疗专业人士预约平台 ZocDoc 获得高盛 2500 万美元注资

在 8 月初从 DST 筹集了 5000 万美元资金后， [ZocDoc](https://web.archive.org/web/20230204233822/http://www.zocdoc.com/) 又从高盛获得了 2500 万美元的 C 轮融资。这使得 ZocDoc 的 C 轮融资总额达到 7500 万美元，迄今为止总融资额达到 9500 万美元。[之前的投资者](https://web.archive.org/web/20230204233822/https://techcrunch.com/2010/07/14/khosla-ventures-and-founders-fund-invest-15m-in-health-appointment-platform-zocdoc/)包括科斯拉风险投资公司、创始人基金、马克·贝尼奥夫和杰夫·贝索斯。

ZocDoc 于 2007 年在 TechCrunch 上推出，它自动化了一项对消费者来说非常令人沮丧和耗时的任务。ZocDoc 允许用户在线预约医生，甚至当天预约。患者可以看到他们所在地区医生的实时可用性，确认谁接受他们的保险计划，并阅读其他患者对医生的反馈和评论。允许消费者在线查找和预约医生、牙医和其他医疗专业人员。这项服务对病人是免费的。

在医疗保健方面，ZocDoc 与医生的日历实时集成，并帮助挖掘医疗提供商的隐性供应，例如 10%至 20%的医疗预约在最后一刻被取消或重新安排。医生每月为 ZocDoc 支付 250 美元。

一年前，这项服务仅在四个城市提供，现在 ZocDoc 已在 11 个大都市地区提供，其中六个是在 2011 年推出的。 ZocDoc 目前提供超过 530 万个来自 40 多个医疗专业的医生和医疗从业者的可用预约，每月有超过 700，000 名患者使用。

新的资金将用于扩展到新的城市，以及走向国际扩张。
