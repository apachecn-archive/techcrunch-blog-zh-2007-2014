Local.com以高达3250万美元的价格收购Screamin ' Daily交易

当地在线媒体公司Local.com今天早上宣布，它已经收购了 Screamin' Media Group ，该公司是 Groupon克隆 daily deals网站 Screamin' Daily Deals 的幕后公司。

Local.com以现金、股票和债务的形式支付了1250万美元，如果在交易完成后的两年时间内满足确定无疑的高端财务表现标准，Screamin' Media Group的股东有机会获得额外的2000万美元。

Screamin' Daily Deals成立于2010年初，拥有约60名员工，为全美14个市场的“数十万”用户提供服务，这些用户与当地商家达成交易。该公司最近还推出了旅游优惠。

据说这家初创公司在2010年创造了约240万美元的未经审计的收入，在2011年上半年创造了440万美元的收入。

Screamin' Media Group将迁至Local.com位于加州尔湾的公司总部，并被后者的团购业务部门 Spreebird 所吸收。

这家初创公司的三位联合创始人，即前总裁、首席执行官和首席运营官，将分别成为Spreebird的销售和营销副总裁、产品副总裁和运营副总裁。

根据最近提交给美国证券交易委员会的文件，Local.com刚刚与谷歌签订了一份为期两年的协议，以实施“某些广告和搜索服务”，这导致股价上涨。
