# PRIMEHPC FX10:富士通向公司推销新的超级计算机

[富士通](https://web.archive.org/web/20230203153010/https://techcrunch.com/tag/fujitsu)宣布，它已经将其超级计算机“K”([是世界上最快的](https://web.archive.org/web/20230203153010/https://techcrunch.com/2011/11/03/fujitsu-k-the-worlds-fastest-supercomputer-is-now-even-faster/))的技术应用于 PRIMEHPC FX10，这是一种将在全球范围内提供的商业模型。PRIMEHPC FX10 可以扩展到 98，304 节点/1，024 机架配置，最高理论处理性能为 23.2 petaflops(每秒 23，200 万亿次计算)。

富士通正在向个别公司和研究机构推销这种超级计算机。

该公司解释道:

> 通过利用新系统，将有可能解决社会挑战——包括新药开发、灾难预防、灾难缓解和其他措施，以实现安全可靠的社会——并进行尖端研究，如无需制造原型就能开发和制造新材料。

PRIMEHPC FX10 将于明年 1 月上市，单机架机型的起价为 64 万美元。富士通希望在三年内卖出总共 50 台超级计算机。
