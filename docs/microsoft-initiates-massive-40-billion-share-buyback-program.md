# 微软启动 400 亿美元的股票回购计划

开市前不久，微软([纳斯达克:MSFT](https://web.archive.org/web/20221005145848/https://www.google.com/finance?q=NASDAQ:MSFT) ) [宣布](https://web.archive.org/web/20221005145848/http://www.microsoft.com/en-us/news/press/2013/sep13/09-17dividendpr.aspx)一项新的大规模回购计划以及增加股息。新的回购计划是在之前的 400 亿美元计划于 2013 年 9 月到期后推出的。换句话说，今天的新闻证实了微软将现金投资于自己股票的承诺。

季度股息也将从 0.23 美元升至 0.28 美元，增幅高达 22%。微软每个季度仍在创造数十亿的利润，投资者期望该公司向股东返还现金。

股票回购计划可能会提振微软的股票，因为许多投资者都在等待这方面的公告。与之前的程序相反，它没有截止日期。在盘前交易中，该股上涨 1.83%，至 33.40 美元。没有截止日期的事实可能表明回购计划可能需要一段时间。

当谈到股票回购计划时，这是对一家公司股票的巨大信任票。一方面，这意味着微软认为它不能以任何其他方式投资手头的现金。微软股票是微软自有资金的最佳金融投资。另一方面，它减少了股票市场上的股票数量，因此人为地提高了剩余股票的价值。现有投资者将拥有微软更大的股份。

提醒一下，苹果是仅有的几家不发放股息或没有股票回购计划的大型科技公司之一。去年，当该公司宣布将启动 450 亿美元的股票回购计划时，这一切都变了。早在 4 月份，该公司就将该计划增加了一倍多，达到 1000 亿美元。

雅虎有自己的 50 亿美元回购计划。所有这些项目都是微软回购计划必要的原因。它使公司在财务水平上保持竞争力。
