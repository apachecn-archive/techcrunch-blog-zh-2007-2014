# Klout 如何获得 Klout.com TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/29/how-klout-got-klout-com/>

当每个人都在为 Klout 是否是一个[完全无意义的服务](https://web.archive.org/web/20230203072923/https://techcrunch.com/2011/10/26/nobody-gives-a-damn-about-your-klout-score/)或神圣的[“影响力标准”](https://web.archive.org/web/20230203072923/http://marshallk.com/why-klout-is-really-and-truly-valuable)做出决定的时候，前几天在爱尔兰都柏林举行的 F.ounders 大会上，我和 Klout 联合创始人兼首席执行官[乔·费尔南德斯](https://web.archive.org/web/20230203072923/http://www.crunchbase.com/person/joe-fernandez)进行了一次有趣的聊天。

当我告诉他，如果有的话，我认为该公司的名字是很好的选择，他告诉我他如何获得域名 klout.com 的故事。因为我对这种幕后的小故事和域名有着浓厚的兴趣，所以我喜欢它，并且觉得它非常有趣，值得在这里分享。

当公司几年前在纽约市成立时，费尔南德斯的第一选择是 Klout，所以他注册了 klout.net，并试图与 klout.com 的老板取得联系，他的总部设在旧金山。

Klout 要获得资金或赚钱还有很长一段路要走，所以 Fernandez 提出用自己的 1000 美元购买这个域名。店主嘲笑他，暗示它值“五位数”，并说他计划为它造点什么。

在接下来的一年半时间里，费尔南德斯每周都骚扰域名所有者，试图说服他卖掉域名。然后，在 2009 年 8 月，费尔南德斯决定将公司从纽约搬到旧金山。大约在那个时候，Klout 还筹集了第一轮种子资金:天使投资人诺瓦·斯皮瓦克(Nova Spivack)的一张支票，费尔南德斯兑现了这张支票，将现金放在他的公寓里。他告诉我，让他的团队和托管公司非常沮丧的是。

接下来，费尔南德斯开始在推特上关注 klout.com 的主人，等待他透露自己的位置。当那个人在推特上说他在旧金山的一家餐馆时，费尔南德斯去了那里，把一个装有 5000 美元现金的信封扔在桌子上，并告诉他，如果他最终同意出售域名来换取这笔钱，他将不再打扰他。

那人同意了，费尔南德斯打开他的笔记本电脑，当场处理了域名转让事宜，十分钟后离开了餐馆。
