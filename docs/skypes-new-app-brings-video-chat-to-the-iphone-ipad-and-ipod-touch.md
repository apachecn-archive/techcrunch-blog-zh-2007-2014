# Skype 的新应用为 iPhone、iPad 和 iPod Touch 带来了视频聊天技术

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/12/29/skypes-new-app-brings-video-chat-to-the-iphone-ipad-and-ipod-touch/>

![](img/28c1e08106043d765652a57cc1db1a17.png)我们一直在听[报道](https://web.archive.org/web/20230202230417/http://gigaom.com/2010/12/29/how-skype-could-make-the-mobile-video-market-explode/)说 Skype 正在推出移动视频聊天服务，并且登陆日已经到来。该公司发布了一个全新版本的 iPhone 应用程序，它可以在任何 Skype 聊天中打开视频。

这款应用的美妙之处在于，它为 iPhone 3G 设备、iPad 和 iPod Touch 带来了免费的视频通话，这些设备都无法运行苹果的视频通话功能 Facetime(该功能仅适用于 iPhone 4 设备和 Mac 电脑)。当然，iPad 用户将无法发起视频通话，但这些用户可以接收任何来自联系人的视频聊天。

Skype 还表示，该应用程序允许用户通过 3G 和 WiFi 网络进行视频聊天，允许用户通过各种连接访问该功能。Skype 的 iPhone 应用程序是 2010 年五大免费 iPhone 应用程序之一，用户只需打开视频功能即可激活视频聊天。iPhone 4 用户可以使用前置摄像头或后置摄像头。

新版 iPhone app 兼容 iPhone 4、iPhone 3GS、i0S 4.0 以上的第四代 iPod touch。用户可以在第三代 iPod touch 和 iPad 上接收视频聊天。还可以在使用新的 Skype for iPhone 应用程序的设备和台式机之间进行通话，包括 Skype for Windows、Skype for Mac、Skype for Linux 和 ASUS Videophone。

【Skype 消费产品副总裁 Rick Osterloh 告诉我们，该公司已经花了一段时间开发应用程序的视频功能。视频通话占所有 Skype 通话的 40 %, oster loh 说增加这项功能对扩展 Skype 的使用是有意义的。Osterloh 说:“移动电话对 Skype 来说很重要。“我们确保无论您身在何处，Skype 都与您同在，随着智能手机变得越来越强大，手机将成为实现这一点的地方。”

Osterloh 补充说，我们可以期待在不久的将来会有更多的移动产品，这似乎意味着视频通话也可能会进入其 Android 应用程序。虽然这款新应用似乎确实与苹果的 Facetime 竞争，但奥斯特洛很快补充说，苹果一直是新版本 iPhone 应用开发的“伟大合作伙伴”(可能是因为苹果实际上批准了这款应用)。

对于 Skype 来说，这是混乱的几周，上周 Skype 遭遇了大规模中断，影响了数千万操作系统用户。今天，该公司的首席信息官拉尔斯·拉布[透露了中断背后的原因](https://web.archive.org/web/20230202230417/https://techcrunch.com/2010/12/29/skype-reveals-a-bug-in-its-windows-client-was-what-crashed-its-system/)，其中包括 Skype for Windows 客户端的一个错误。本周早些时候，该公司遭遇了一起[专利侵权诉讼。](https://web.archive.org/web/20230202230417/https://techcrunch.com/2010/12/27/skype-gradient-enterprises/) Skype 表示，该应用的发布是特意在新年前夕进行的，新年是 Skype 历史上一年中最大的视频通话日(以通话分钟数计算)。

2011 年对 Skype 来说应该是重要的一年，因为该公司正在为明年的首次公开募股做准备。随着这一事件的临近，Skype 一直致力于通过[企业产品](https://web.archive.org/web/20230202230417/https://techcrunch.com/2010/10/16/skypes-vp-of-enterprise-on-future-strategy-products-and-competitors/)和新产品开发(即[脸书整合](https://web.archive.org/web/20230202230417/https://techcrunch.com/2010/10/14/skype-5-0-for-windows-debuts-with-facebook-integration-call-recovery-and-more/))来扩大其服务的使用范围。我们知道 Skype 将在 2011 年大力推进云计算，可能会在年初推出基于网络的服务。

除了一般消费者使用之外，视频通话对 Skype 的企业客户也很有用。新功能无疑使该技术成为更具吸引力的会议解决方案。

Osterloh 非常明确地表示，移动电话也将是该公司扩大用户群战略的一部分(Skype 在任何特定时间平均有 2500 万消费者使用这项服务)。看看 Skype 在新的一年里有什么锦囊妙计应该很有趣。
