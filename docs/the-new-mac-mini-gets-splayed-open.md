# 新款 Mac mini 张开了

[![macminiinside](img/dd313243a81646d38e31b30f2d691609.png "macminiinside")](https://web.archive.org/web/20221004043742/https://beta.techcrunch.com/wp-content/uploads/2009/03/macminiinside.jpg)

最新的 [Mac mini](https://web.archive.org/web/20221004043742/http://www.crunchgear.com/tag/mac-mini/) 是[仅仅一天前](https://web.archive.org/web/20221004043742/http://www.crunchgear.com/2009/03/03/apple-unleashes-a-new-mac-mini/)我们已经得到了[第一次看里面](https://web.archive.org/web/20221004043742/http://www.macminicolo.net/macmini2009.html)更新的电脑。苹果保留了最初的设计，尽管仍然需要使用油灰刀和一点耐心来打开她。如果苹果能采用类似于新笔记本的简易替换设计就好了，但是你会怎么做呢？在内部，除了明显的新部件之外，没有太大的不同。点击查看电路板 pr0n。
