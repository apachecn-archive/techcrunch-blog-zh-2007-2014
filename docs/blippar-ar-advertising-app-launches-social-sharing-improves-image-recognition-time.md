# Blippar AR 广告应用推出社交分享，缩短图像识别时间

增强现实广告公司 Blippar 今天发布了一项重大更新，不仅提高了技术速度，还为用户提供了增强的社交分享功能。

Blippar 于 2011 年在英国推出，此后也扩展到了美国。这项技术的工作原理是让用户扫描杂志页面、番茄酱瓶子或任何有 Blippar 标签的东西，以查看各种额外内容，如促销、建议食谱、优惠券等。事实上，品牌将在他们的活动中包含什么样的宣传内容完全取决于品牌。

据创始人兼首席执行官 Rish Mitra 称，此次更新使图像识别技术的速度提高了 10%。但或许对 Blippar 合作的品牌更重要的是，这款应用现在允许用户分享 Blippar 的内容。

以前，Blippar 用户只能在 Twitter 或脸书上喜欢或关注某个品牌的社交媒体页面。有了这个更新，用户可以登录自己的脸书和 Twitter 账户，以病毒方式传播 Blippar 的内容。

此次更新还包括一个应用内目录，允许用户浏览有趋势的 Blippable 内容，但如果没有访问现实世界对象上的 Blip 标签，目前可能无法访问。

在品牌方面，Blippar 正在提供更深入的分析。除了位置、停留时间、整体亮点和应用内互动，由于与脸书和 Twitter 的新集成，品牌现在还将获得匿名人口统计数据的分析。

[Blippar](https://web.archive.org/web/20230129072022/http://www.crunchbase.com/company/blippar) 自 2010 年推出后三个月以来一直是现金正增长，但 Mitra 告诉 TechCrunch，Blippar 拥有“强大的领先速度和超过 70%的客户保留率，作为一种证明媒体价值的业务。”

UI 也进行了改进，允许 3D 建模，这在所有平台上都是一样的，包括 iOS、Android 和黑莓 Z10。
