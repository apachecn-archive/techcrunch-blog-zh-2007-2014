# 书盒子:因为你总是希望你的 iPhone 看起来像一本小圣经

厌倦了人们因为你整天盯着你的 iPhone 而对你评头论足？感觉你总是在 10 秒钟之内被某个大块头打脸，然后把*你的* iPhone *他的* iPhone？

输入 BookBook。这是一个让你的 iPhone 看起来像..一本书更具体地说，它让你的 iPhone 看起来像一本小小的圣经。这很好，因为([几乎](https://web.archive.org/web/20230203044054/http://www.reddit.com/r/atheism))没人会因为你读圣经而对你评头论足。谁会偷别人的圣经呢？没有必要。你可以得到一本免费的圣经，只要走进教堂，说“嗨，伙计们，我能要一本圣经吗？”

BookBook 是 twenty south[twenty](https://web.archive.org/web/20230203044054/http://twelvesouth.com/)的伙计们推出的第一款 iPhone 配件。如果你还记得的话，就是这些人制造了苹果键盘/魔力轨迹板融合的 MagicWand。

除了把你的 iPhone 包在一本相似的手册里，BookBook 还有另一个锦囊妙计:它是一个钱包。它有一个你的 ID 插槽，两张信用卡，还有一些奇怪的纸质绿色东西，我被 90 年代的人称为“现金”。

一个奇怪的地方:正如下面的视频所表明的，你的相机背面没有开孔，所以你必须将你的 iPhone 拉出一点才能拍照。

你可以在这里找到这个盒子，现在售价 60 美元。或者你也可以等着看那些地精会不会开始把它们藏在酒店的抽屉里。
