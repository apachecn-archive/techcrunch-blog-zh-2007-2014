# 谷歌雇佣在线旅游指南 Ruba 团队(更新)

![](img/afa5a822a650798f30007ba5c5c12af4.png)根据[这篇博文，](https://web.archive.org/web/20230227002144/http://www.ruba.com/blog/2010/05/21/ruba-team-joins-google/)谷歌收购了在线旅游指南和社区 [Ruba。](https://web.archive.org/web/20230227002144/http://www.ruba.com/) Ruba 是一个[视觉旅游指南](https://web.archive.org/web/20230227002144/https://techcrunch.com/2009/04/30/ruba-an-online-travel-guide-where-photos-speak-louder-than-words/)和旅游评论网站，为旅行者提供由其他旅行者撰写的视觉指南。博文嵌在下面。**更新**:谷歌告诉我们，他们实际上并没有收购 Ruba，但 Ruba 背后的团队将加入谷歌团队，从事 iGoogle 和其他项目。

Ruba 为用户提供了一种可视化浏览世界各地城市及其景点的方式，提供了照片丰富的指南，并强调让用户很容易快速发现新的地点。该网站由迈克·卡西迪领导，他创立了许多成功的公司，包括 2006 年以 1 . 02 亿美元卖给维亚康姆的 Xfire。

指南都是由用户编写和提交的，Ruba 从 Google 和 Flickr APIs 中提取来帮助精确定位，并提供一些样本照片(用户也可以提交自己的照片)。该网站在某些方面类似于猫途鹰，其特点是集成了 Twitter 和脸书连接，允许用户广播他们要去哪里，并要求朋友提供信息。

据报道，谷歌在进军在线旅游业务方面迈出了重要的一步，将酒店链接整合到地图中，并列出酒店和房价。据《今日美国》报道，这家搜索巨头据说也在就收购购票软件 ITA software 进行谈判。
