# RadioShack 商店现在接受旧设备的折价

[![radioshack](img/2c3c40f8e08ffa2f0e8cb41b505050eb.png "radioshack")](https://web.archive.org/web/20230323204546/https://techcrunch.com/wp-content/uploads/2009/04/radioshack.jpg)

对[无线电黑客](https://web.archive.org/web/20230323204546/http://www.crunchgear.com/tag/radioshack/)有利。所有 4400 家商店现在都是该公司的[以旧换新计划](https://web.archive.org/web/20230323204546/http://www.radioshack.com/uc/index.jsp?page=researchLibraryArticle&articleUrl=../graphics/uc/rsk/USContent/HTML/pages/tradein.html&noBc=true)的落客点，这听起来像是一个天然的契合点。以前的小工具折价项目通常会将你的设备发送到一个随机的网站，在那里你会*知道*你不会得到该网站所宣传的东西。现在，你只需前往当地的小屋，店员会立即确定小玩意的价值，如果你接受，就会给你一张礼品卡。对我们来说，这听起来是个不错的计划，因为每个社区都有一台 RadioShack，每个人都有积满灰尘的旧设备，现在可以用来购买新设备。

商店现在接受:

*   MP3 播放器
*   移动电话
*   摄像机
*   便携式摄像机
*   GPS 接收器
*   游戏媒体
*   游戏控制台
