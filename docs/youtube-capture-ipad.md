# 随着专门的视频录制应用程序在 iPad 上推出，YouTube Capture 大受欢迎

你有没有想过成为那种在音乐会上举着 iPad 录制视频并上传到 YouTube 上的人，对他周围的人和所有良好的品味视而不见？现在你可以了！*这是因为 YouTube 正在 iPad 上推出其专用视频录制应用[，就在](https://web.archive.org/web/20221206084335/https://itunes.apple.com/us/app/youtube-capture/id576941441)[为 iPhone](https://web.archive.org/web/20221206084335/https://beta.techcrunch.com/2012/12/17/google-launches-dedicated-youtube-video-camera-app-for-iphone-and-ipad-touch/) 推出该应用三个月后。

YouTube Capture 的推出是为了简化录制和上传视频到 YouTube 的过程，将所有的工具都放在一个应用程序中。一旦用户完成了视频录制，他们就可以添加字幕、标签和其他元数据，该应用程序会立即将文件直接上传到他或她的 YouTube 频道，或者如果他们不愿意与外界分享，也可以将其保存为私人视频。

对于那些想对视频进行微小编辑的人，该应用程序甚至包括颜色校正、自动稳定、修剪和音乐选项。当应用程序关闭时，它会在后台上传视频，因此用户不必担心在继续生活时保持视频打开。

这款应用的推出表明，YouTube 及其母公司谷歌对在 iOS 上提供应用的兴趣越来越大，尽管它是他们自己的 Android 移动操作系统的竞争对手。这包括用于观看 YouTube 视频的专用应用程序，以及谷歌地图、Gmail 和其他为在 iPhone 和 iPad 上使用谷歌服务而开发的应用程序。

视频录制和上传到 YouTube 是 Android 手机和平板电脑的直接内置功能，因此在这些平台上没有必要安装专用的录制应用程序。尽管如此，YouTube 的一位发言人表示，该公司正在寻求在未来将 YouTube 捕捉功能引入 Android。
