# 三星 P3 终于打破了封面

[![](img/d12ae73b33ac2dcd94257871341a2254.png "samsung-p31") ](https://web.archive.org/web/20221006104650/https://beta.techcrunch.com/wp-content/uploads/2008/11/samsung-p31.jpg) [三星 YP-P3](https://web.archive.org/web/20221006104650/http://www.crunchgear.com/2008/05/15/samsung-p3-a-haptic-competitor-to-the-ipod-touch/) 消息已经泄露了一段时间，但实际照片终于浮出水面。看起来下一个版本更像是一个进化更新，在触摸屏 PMP 上增加了触觉反馈和小工具。根据法国网站的说法，将有 4，8，16 和 32GB 版本，所有这些版本都在最大容量的播放器上配备蓝牙 2.0，FM 调谐器，麦克风，单声道扬声器和铝制外壳。现在仍在谈论价格或可用性，但该信息可能会在 CES '09 上正式宣布。
