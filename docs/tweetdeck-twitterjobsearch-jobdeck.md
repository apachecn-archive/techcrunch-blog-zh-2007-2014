# TweetDeck、TwitJobSearch 联手打造定制求职桌面应用

今早 [Clickz](https://web.archive.org/web/20221207205334/http://www.clickz.com/3636255) 报道称，基于 Twitter 的求职搜索引擎已经与 [TweetDeck](https://web.archive.org/web/20221207205334/http://tweetdeck.com/) 合作推出了一款名为 [JobDeck](https://web.archive.org/web/20221207205334/http://tweetdeck.com/jobdeck/) 的桌面客户端。

该应用程序基于 TwitJobSearch 的算法(我们的[早期报道](https://web.archive.org/web/20221207205334/http://www.beta.techcrunch.com/2009/03/17/two-new-ways-to-find-a-job-auction-yourself-off-at-jobaphile-or-do-a-twitterjobsearch/))实时索引整个 Twitter 领域与招聘相关的推文。

基本上，它只是一个定制品牌的 TweetDeck 客户端，带有两个额外的默认栏:“求职专家”和“TwitJobSearch”，尽管它也可以证明增加一个 LinkedIn 栏来承载来自专业社交网络的最新更新(甚至可能是你的脸书新闻订阅源)是有帮助的。

招聘人员可以遵循一些简单的步骤，确保他们的推文被 TwitJobSearch 服务索引，帮助他们通过社交媒体更有效地联系潜在的候选人，而且是免费的。TwitJobSearch 网站本身为广告客户提供了购买溢价或“Tweetured”职位列表的能力，以每次点击付费为基础出售。据 Clickz 称，目前的广告客户包括阿迪达斯、肯德基和劳埃德银行集团等品牌。

JobDeck 应用程序本身在这一点上不会提供任何额外的广告机会。
