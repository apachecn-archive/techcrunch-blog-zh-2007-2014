# 微软将重塑搜索品牌。会是 Kumo 吗？

据公司内部消息人士透露，微软将在明年年初的某个时候以新品牌重新推出 Windows Live 搜索。我们不知道的是这个新品牌将会是什么，尽管已经有几个名字被抛到了脑后。根据我们的消息来源，一个“最终”的决定已经作出，但微软内部很少有人知道，它可能会改变。

现在 LiveSide 称有证据表明新的搜索品牌将会是 Kumo，在日语中是“云”或“蜘蛛”的意思。

为什么微软会经历另一次品牌重塑的努力？除了搜索之外，Live.com 旗下还有很多不同的服务(一些服务器软件，一些客户端软件)。这也是一个[新兴的社交网络](https://web.archive.org/web/20221207192635/http://www.beta.techcrunch.com/2008/11/12/sweeping-changes-at-livecom-its-a-social-network/)。

我们听说，随着时间的推移，Live.com 将成为一个纯粹的社交网络和个人生产力门户。你可以在那里访问电子邮件、日历、照片、活动流等。但搜索属于其他地方，它肯定需要一个新的开始。

微软对更名不予置评，或者即使有更名也不予置评。但我们的消息来源提醒我们，一切都还没有最终确定，雅虎的命运也可能会发生这样或那样的变化。所以 Kumo 很可能是微软计划使用的名字，但是这个决定可能会改变。
