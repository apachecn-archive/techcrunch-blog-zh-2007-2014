# 社交媒体和原始媒体如何影响在线媒体品牌 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/28/how-social-vs-primary-sources-affect-online-media-brands/>

**编者按:** *[塞米沙](https://web.archive.org/web/20221007225503/http://blog.semilshah.com/about)是* TechCrunch *的撰稿人。你可以在推特上关注他，地址是[@ semi](https://web.archive.org/web/20221007225503/http://www.twitter.com/semil)。*

在整个高中和大学期间，我上了大量的政治学和历史课。因此，这些老师和教授强调了调查原始文献并根据它们自身的优点和二手资料(比如教科书)进行分析的重要性，尽管我们都收到了教科书、主题专家的论文和一系列其他的解释。最终，随着大学的结束，课程更加关注原始资料和我们自己对它的解释。

快进到今天。至少在创业技术新闻的世界里，一手资料仍然很重要，但社交资源——至少对我来说——胜过一切。创业技术新闻的发展速度太快，以至于教科书或杂志印刷版都无法捕捉到。当然，在早期的私人公司中，获得第一手资料是困难的。在我的科技新闻世界里，像许多人一样，Twitter 是我的主要信息来源，也是我上网冲浪的方式。然而，特别是在 Twitter 上，我并不直接关注任何“新闻来源”。外面的信息太多了。因此，我试着关注那些阅读和分享文章或随机博客帖子的人，我已经变得信任他们了。

为了让我阅读一些东西，我需要一个社交信号来触发和抓住我的注意力。“谁”和我分享很重要。“来源”仍然很重要，只是没那么重要了。而且，在某些情况下，网上的资源可以由一个品牌支撑，并掌握其分销权。房地产在线创造内容是无限的。创建信息、建立受众、产生页面浏览量，以及将这些与广告挂钩，没有任何障碍。因此，至少在我的在线科技新闻的小世界里，社交资源是至高无上的。

我猜很多读到这里的人可能会有同样的感觉。关注朋友或可信行业来源的社交信号激励我对链接产生兴趣，阅读故事，或留着以后看。在这个决定中，最关键的信息不是链接的来源和位置，而是谁共享了这个链接。在某种程度上，推文本身作为社会货币的单位，比来源本身更重要。一个展示了这种翻页板的普及性的产品。是的，Flipboard 有专门的媒体频道，但在他们的社交媒体上，一篇内容的作者几乎是灰色的，这样读者就可以专注于“谁”与他们分享了内容，而不是“谁”创建了它。

我分享的观点显然不是新的或者惊天动地的。“社会新闻”的想法甚至已经落满灰尘。我们都知道这是真的。然而，我认为这对网络媒体品牌有着重大而长远的影响。在我的大学历史经历中，图书出版商花时间与大学、教授和世界上的其他灯塔保持一致，以确保他们的材料被选为来源。快进到今天，这些策略可能并不那么有效。相反，媒体品牌被迫对其忠诚的核心受众的质量进行批判性思考，因为正是这些个人，作为社会资源，将与他们自己的朋友和受众分享和讨论内容、信息、事实和神话。这才是真正的、可持续的分销之所在。对于在线媒体公司来说，社交来源胜过主要来源——人们意识到，谁在网上分享信息往往比信息是什么更重要。对于许多媒体品牌来说，这是一个根本上的——有时甚至是可怕的——新现实。
