# Windows 7 通过测试，准备投入生产| TechCrunch

微软最新版本的操作系统 [Windows 7](https://web.archive.org/web/20230325052832/http://www.crunchgear.com/tag/windows-7/) 终于进入了[发布制造(RTM)阶段，](https://web.archive.org/web/20230325052832/http://windowsteamblog.com/blogs/windows7/archive/2009/07/21/when-will-you-get-windows-7-rtm.aspx)因此该操作系统将很快预装在新电脑上。尽管尚未正式发布，但 Windows 7 预计将会大受欢迎。例如，在亚马逊英国网站上仅仅 8 个小时后，Windows 7 的预购量就超过了 17 周内 Vista 的预购总量。

根据微软的说法，提供七个不同版本操作系统的 Windows 7 已经经历了重要的测试、质量保证和达到 RTM 阶段所需的验证。独立软件和硬件供应商最早将于 8 月 6 日开始下载 Windows 7 RTM。微软将在 8 月中下旬向其他合作伙伴推出 Windows 7。从 8 月 7 日开始，企业客户和开发人员将能够下载英文版的 Windows 7 RTM，随后不久将发布针对 Windows 7 的其他语言功能。

对于普通人/消费者来说，Windows 7 将于 10 月 22 日开始在零售店和新电脑上销售，这我们已经知道了。在收到 beta 测试人员的热烈响应后，微软也为 Windows 7 提供了一个“家庭包”,允许在最多 3 台电脑上安装。该公司还在[正式发布了](https://web.archive.org/web/20230325052832/http://www.microsoft.com/expression/products/Purchase.aspx#PageTop) [Expression 3 的新版本](https://web.archive.org/web/20230325052832/https://techcrunch.com/2009/07/10/the-complete-guide-to-microsofts-silverlight-3/)，这是微软为开发者提供的基于 Silverlight 构建应用的工具集。

当然，Windows 7 的正式 RTM 发布正值微软对该操作系统的垄断受到最近发布的谷歌 Chrome OS 的挑战。谷歌计划在今年晚些时候发布 Chrome 操作系统的开源代码，这可能会在 10 月份 Windows 7 发布时推出。第一批 Chrome OS 电脑要到明年才会上市。虽然谷歌表示 Chrome 操作系统目前的目标是上网本，但谷歌操作系统确实有潜力扩展到其他类型的电脑，这给了微软一些思考的空间。
