# 第一印象和动手操作:任天堂 3DS

[![](img/926ebf46dd447927459dace7abcff52e.png "IMG_9847")](https://web.archive.org/web/20221006204921/http://www.crunchgear.com/2010/06/15/first-impressions-and-hands-on-nintendo-3ds/) 
关于任天堂的新掌机没有太多可说的，不幸的是:我们的演示仅限于一分钟，灯光很差，当然 3D 效果也不能转化为图片或视频。但下面是我的感受…
 3D 效果对我来说并不是从屏幕中冒出来的东西，而是屏幕背后的深度。根据你看上去的样子和场景的不同，东西看起来肯定会突出一点，但总的来说，3D 屏幕就像是房间里的一扇窗户，里面的一切都是 3D 的。不过，3D 效果肯定是真实的。

马里奥和朋友们的各种场景正在运行，你可以改变相机的角度，但没有实际的游戏。将我的头向任何方向移动几英寸都会产生重影，最终图像会加倍；甜蜜点是…我不想说原谅，但你不需要一个颈托来保持它。你头部的正常运动不会影响它。

该设备本身是光滑的，我的单位是黑色的，感觉很像一个 DS。滑垫摸起来很光滑，模拟响应也很棒。不幸的是，现在只能说这么多了，所以我就说到这里，直到我们今天晚些时候在展示厅通过视频和其他方式获得更全面的实践。
