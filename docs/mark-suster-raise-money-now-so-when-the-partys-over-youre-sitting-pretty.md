# 马克·苏斯特:现在就筹钱，这样当聚会结束时，你就能过得很好

[维梅奥·http://vimeo.com/25219791 w = 620]

本周早些时候， [GRP Parter VC](https://web.archive.org/web/20230203072454/http://www.grpvc.com/) 和 VC 博主[马克·苏斯特](https://web.archive.org/web/20230203072454/http://www.crunchbase.com/person/mark-suster)在[方正研究所](https://web.archive.org/web/20230203072454/http://founderinstitute.com/join/TC)的创始人展示会上做了主题演讲[“在泡沫市场中获得资金”](https://web.archive.org/web/20230203072454/http://www.docstoc.com/docs/81854001/VC-Funding-in-a-Frothy-Market) Suster 以我们处于泡沫中的声明性陈述开始了主题演讲，“我们处于泡沫中，你可以引用我的话，我通常没有公开说过，”他肯定地说。

苏斯特接着提到了这一特定繁荣时期的独特性，因为它是本地化的，“如果你在俄亥俄州的克利夫兰，你不会特别感觉到有泡沫，”但现在，对于硅谷、纽约和洛杉矶的初创公司来说，通常需要 3-5 个月的融资时间已经缩短到 3-5 天

苏斯特敦促这些创业公司现在就筹集资金，这样当泡沫不可避免地破裂时，他们就可以生存下来，“当聚会结束时，聚会结束了，”他继续敦促企业家开始筹集资金……”当开胃小菜传来传去时，拿两个，第三个放在你的口袋里，“将市场泡沫比作一个跳跃的聚会，每个人都喝醉了，在许多情况下都是靠自己的库尔援助。

经历了前三个周期，苏斯特知道派对将如何结束(大多数人都会宿醉)，并传达了他的主要观点，即基本上等于为冬天储备物资——这样你就可以活着讲述这个故事:

> “以你认为合适的任何方式或形式，以你认为合适的任何价格，与你信任、喜欢和尊重的人一起去获得资金，然后去创建一家公司。实际上最终这是一种二元的结果。你筹集资金并全力以赴，尽你所能，有时你会成功，有时你不会。
> 
> 这是一次奇妙的经历。我只记得在过去的三个周期中，没有筹集到资金的公司是那些没有讲述这个故事的公司。
> 
> 我知道这是如何结束的，这是我的水晶球；当聚会结束时，每个人都回家了，你不能雇用人，你不能筹集资金，华尔街日报不再写关于你的文章，没有人回你的电话。老实说，如果你有钱，那是创业最有回报的时候。"
