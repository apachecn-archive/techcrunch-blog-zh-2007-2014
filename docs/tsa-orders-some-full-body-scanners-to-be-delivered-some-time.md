# 美国运输安全管理局订购了一些全身扫描仪，将在某个时候交付

L3 通信公司从美国运输安全管理局获得了一份价值 1 . 65 亿美元的合同，无限期交付无限量的毫米波安全扫描仪。所以，美国运输安全管理局肯定他们会得到一些，但他们不知道有多少，或什么时候。尽管存在各种各样的隐私问题，但目前世界各地部署了 200 多台供应扫描仪。L3 在 2009 年的销售额超过 150 亿英镑，所以这个协议在财务上看起来没什么大不了的。

当然，如果没有对全身扫描仪如何保护我们的隐私进行必要的讨论，任何关于全身扫描仪的报道都是不完整的:

> L-3 的规定提供了多种级别的隐私保护，可以定制以支持客户特定的操作程序。安全分析师看到一个 3-D 黑白剪影，这使得几乎不可能确定被扫描个人的身份。此外，图像是在远离检查站的地方观看的，因此分析员看不到被扫描的乘客。允许模糊面部特征的选项进一步确保了隐私。
