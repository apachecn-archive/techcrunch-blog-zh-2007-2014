# 脸书的 Brad Smallwood 提供了更多关于广告效果的数据，称 Datalogix 合作伙伴关系不存在隐私风险 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/01/facebook-brad-smallwood-datalogix/>

在过去的几天里，脸书一直在谈论其最近宣布的与 Datalogix 的合作关系，该公司发布了博客文章，解释了 T2 的隐私含义和 T4，总结了广告商已经看到的结果。脸书的测量与洞察主管 Brad Smallwood 今天出席了 IAB MIXX 会议，他在会上发表了关于合作的演讲。

合作的目标是跟踪脸书的广告活动是否推动了店内销售，并在有效的广告活动中找到共同的主题。脸书和 Datalogix 跟踪了大约 50 个广告活动，从中他们已经得出了三个一般性的经验。

首先，Smallwood 说是广告印象，而不是点击，推动了销售。事实上，在 DataLogix 的活动中，99%的销售额来自那些看过广告但没有与之互动的人。为了支持这一观点，他还指出[尼尔森的一项研究](https://web.archive.org/web/20221206031933/http://blog.nielsen.com/nielsenwire/consumer/research-shows-link-between-online-brand-metrics-and-offline-sales/)显示，广告点击量与品牌指标或线下销售几乎没有关联。Smallwood 补充说，在某些情况下，专注于点击是有意义的，例如直接回应活动，但这些点击不会告诉你任何关于“杂货店、汽车经销商或当地咖啡店发生的事情”的信息。

其次，他说触及(而不仅仅是针对狭窄的受众)是这些活动的重要组成部分，就像在电视上一样。最大化其活动范围的品牌的投资回报率提高了 70%。

最后，他说，品牌需要优化广告活动的频率，特别是通过将经常看到广告的人的印象重新分配给那些不经常看到广告的人。这似乎是一个显而易见的观点，但 Smallwood 表示，进行这种转变的品牌的投资回报率增加了 40%。他还表示，不同品牌和不同活动的“最佳频率”会有所不同——如果你是一家成熟的消费品包装公司，人们可能不需要像你发布一款全新的技术设备并需要让消费者熟悉细节时那样经常看到你的广告。

现在，50 个活动可能看起来是一个非常小的样本量，特别是因为它们不是随机选择的。但在他的演讲结束后，Smallwood 指出，这些公司是由脸书的“客户委员会”选择的，所以社交网络不能只选择那些会看到最令人印象深刻的结果的广告商。尽管最初的活动来自 CPG 品牌，Smallwood 说没有理由你不能扩大研究，以包括其他类型的广告商。

到目前为止，这种合作关系似乎已经引起了关注(至少是媒体的关注),主要是因为它的隐私含义。当我向 Smallwood 提出隐私问题时，他说脸书和 Datalogix 都不应该获得任何关于个人用户的额外信息——相反，两家公司都使用[散列](https://web.archive.org/web/20221206031933/http://en.wikipedia.org/wiki/Hash_function)数据来创建一个聚合图，显示有多少看过某个广告活动的人实际上购买了相关产品。

顺便说一句，如果你想要一个更公正的分析，这里是[电子前沿基金会如何安排过程](https://web.archive.org/web/20221206031933/https://www.eff.org/deeplinks/2012/09/deep-dive-facebook-and-datalogix-whats-actually-getting-shared-and-how-you-can-opt):

> 这将为脸书提供关于广告效果的数据，但因为结果是按组汇总的，脸书不应该有特定用户是否购买了特定产品的细节。除了脸书有兴趣知道用户是否购买了蔓越莓汁之外，Datalogix 不会知道任何关于用户的新信息。

我问脸书现在是一家上市公司，它在利用用户数据赚钱方面是否感受到了更大的压力，Smallwood 回答说，脸书从一开始就一直在“建设这些系统”(据推测，他指的不是 Datalogix 合作伙伴关系，而是广告和分析产品的通用套件)。

同样值得注意的是，Datalogix 已经在跟踪非脸书广告活动的这种购买行为，因此脸书合作伙伴关系受到如此多的审查似乎有点不公平(尽管并不令人惊讶)。脸书是否被迫比其他公司清除了更高的隐私壁垒？

“我当然觉得对隐私有很多审查，”Smallwod 说。“还有一个更高的标准，但这是我们坚持的内在标准。”
