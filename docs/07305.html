WordPress 3.1在不到5个月的时间里下载了超过1500万次

最新稳定版本的 WordPress ， 3.1 ，于2011年2月23日首次发布。

现在，不到5个月后，根据几分钟前发布的一条推文(和下载计数器)，这个博客软件已经被下载了超过1500万次。

Joomla最近刚刚宣布其软件已经被下载了2300万次(注意这是总数，不是针对软件解决方案的任何具体版本)。

供你参考:WordPress 3.1是TechCrunch用来支持其大部分网站的。

流行的博客软件产品的最新版本实际上是 WordPress 3.1.4 ，这是对之前所有版本的维护和安全更新。

就在昨天，一篇关于下一个版本WordPress 3.2的博文发表了，透露它将‘很快’发布(发布候选此处)。

WordPress 3.2将最终放弃对Internet Explorer 6的支持。
