# 看看这个家伙 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/08/24/check-this-guy-out/>

比特重力公司的技术人员之一比尔·施尼策正驱车前往洛杉机，并在互联网上对 T2 进行现场直播。他有一个网络摄像头显示他的行程，还有一个带有 GPS 设备的谷歌地图混搭显示他的进度。

位于加州伯林盖姆的 Bitgravity 还没有正式推出。但该公司是[改版 3](https://web.archive.org/web/20221203082954/http://www.crunchbase.com/company/revision3) 和其他视频网站的内容交付网络(CDN)。这个网站上的视频质量明显好于你在其他直播服务上看到的，如 [Justin.tv](https://web.archive.org/web/20221203082954/http://www.crunchbase.com/company/justintv) 和 [Ustream](https://web.archive.org/web/20221203082954/http://www.crunchbase.com/company/ustream) 。我期待着听到更多关于流媒体技术，以及谷歌地图/GPS 黑客(GPS mashup 的一些资源[在这里](https://web.archive.org/web/20221203082954/http://earth.google.com/support/bin/answer.py?hl=en&answer=21772)和[在这里](https://web.archive.org/web/20221203082954/http://earth.google.com/earth_plus.html))。
