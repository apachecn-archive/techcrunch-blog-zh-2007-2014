# 美国电话电报公司海洋克隆表面，运行 WinMo 6，看起来不错

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/29/att-ocean-clone-surfaces-runs-winmo-6-looks-good/>

![pantech1802.jpg](img/f5203537e2033dd1e3f01bf7406c64ab.png)

你是否羡慕我们这些带着奢华的[日光海洋](https://web.archive.org/web/20160316035437/http://crunchgear.com/?s=helio+ocean)，却坚持你的 [AT & T](https://web.archive.org/web/20160316035437/http://crunchgear.com/?s=AT%26T) 计划的人？你想要我们正在享受的双滑乐趣，却愚蠢地忽视了 Helio 服务的召唤？如果是你，我们有一部你可能会喜欢的手机。

美国电话电报公司正在为可能在假期推出的新手机泛泰 C810 做准备。像流行的 Ocean 一样，它是双向的，给你 T9 的通话功能，和全 QWERTY 的其他功能。据说它更苗条，这是另一个吸引人的方面。我们不关心的是 Windows Mobile 6 标准。即使他们把它做得很漂亮，它仍然是 WinMo 6。恶心。

BGR 听说两年期血液的价格约为 250 美元，这听起来对我们很合适。我们还听说 Helio 的 sans-WinMo 版本也即将推出，因此它将成为双滑动圣诞，就像奶奶喜欢的那样。你奶奶，不是我奶奶。
