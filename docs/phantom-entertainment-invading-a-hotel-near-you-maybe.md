# 幻影娱乐入侵你附近的酒店，也许

![phantom.jpg](img/45012254b8f1d315c9e46141ca54bff2.png)

哦，幻影娱乐，尽你所能让我们认为你实际上生产硬件，你去和 ProGames Network 签约，把你不存在的无线笔记本电脑和游戏服务带到我、你或任何人附近的酒店。我们等待虚幻的幻影游戏机有多久了？一年多后，你决定降级到一个 lapboard，它应该在今年的某个时候发货，但没有人见过。以下是 ProGames Network 总裁兼首席执行官马丁·格雷对新签署的协议的评论。

> 幻影无线笔记本电脑和幻影游戏服务内容符合我们的目标，即提供最人性化和内容丰富的酒店设施。幻影娱乐将帮助我们向酒店行业提供用户友好的无线技术和内容丰富的视频游戏目录。

想知道幻影娱乐公司总裁兼首席执行官格雷格·科勒是怎么说的吗？

> 我们非常高兴有机会向 Pro Games Network 提供我们的 Phantom 无线笔记本电脑和 Phantom 游戏服务内容，以放置在全球的酒店房间中。我们相信这个机会将证明酒店和其他基于位置的提供商可以提供视频游戏娱乐作为他们营销工作的一部分。
