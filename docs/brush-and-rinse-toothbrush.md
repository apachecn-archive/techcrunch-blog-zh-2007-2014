# 刷洗牙刷

当人类第一次从原始污泥中爬出来时，他们有很多事情要做。他们必须马上制造出轮子、呼吸带和飞机，更不用说通过文字来编纂思想和行动，通过色情和《实习医生格蕾》来编纂性感。老实说，从现在到那时，我们没有太多时间来制作刷子和冲洗牙刷，但我很高兴有人终于把治愈癌症放在一边，来处理这个紧迫的问题。

你在这里看到的是一个倾斜头部的牙刷。它被设计成做两件事——用有刚毛的部分刷牙，用倾斜的部分把水射进嘴里。如果你的嘴漏了，刷子里涌出丰富的泡沫流，会发生什么？嗯，你的睡衣湿了，你不得不整夜穿着猪小弟。科学的前进不会因为你无法瞄准而停止。

> 目前刷牙后用水漱口的方法是草率的，造成浪费，并给我们的身体带来不必要的压力。而且，人们喜欢喷泉。
