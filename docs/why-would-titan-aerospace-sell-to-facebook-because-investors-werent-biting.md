# 为什么泰坦航空要卖给脸书？因为投资者不感兴趣

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/03/04/why-would-titan-aerospace-sell-to-facebook-because-investors-werent-biting/>

关于脸书收购无人机制造商 [Titan Aerospace](https://web.archive.org/web/20230404105104/http://titanaerospace.com/) 的兴趣，我们有一个问题是为什么 Titan，一家成立于 2012 年的年轻公司，想要出售？毕竟，脸书对这项业务的兴趣是有道理的:它希望通过 Internet.org 倡议，利用无人机帮助发展中国家普及互联网接入，该倡议专注于将“下一个 50 亿”人口连接到网络。

但是泰坦在之前创建 Eclipse Aviation 的首席执行官 Vern Raburn 的领导下，似乎已经开发了一些非常创新的飞机，或者公司喜欢称之为“atmosats”。例如，该公司的 Solara 50 和 60 型号可以使用电池动力发射到天空，然后上升到 20 公里(65，000 英尺。他们可以在海平面上停留五年，不需要着陆或补充燃料，只使用太阳能。

不幸的是，创新本身似乎并不能让灯一直亮着。

泰坦之前已经在种子期、首轮和 A-1 融资中筹集了一笔未披露金额的资金，并计划在今年启动 B 轮融资。然而，TechCrunch 从未经授权访问内部信息的消息人士处获悉，该公司正在努力让投资者做出承诺。

显然，风投对该公司的发展方向感兴趣，但大多数人认为 Titan 还处于早期阶段，不值得投资，因为它距离实现收入还有两年左右的时间。(比如社交+资本，通过了)。此外，在泰坦能够证明它的飞机能够做它承诺的事情之前，他们对投资犹豫不决。例如，Solara 60 飞机据称可以在 65，000 英尺的高度飞行，据报道可以在那里停留数年。泰坦本身存在的时间还不足以证明这一点。

一些投资者也对即将出台的联邦航空局法规感到担忧。正如《财富》杂志(Fortune)最近解释的那样，将无人驾驶航空系统(UAS)整合到美国国家领空的法规尚未出台，监管框架将于 2015 年出台。但是对于更大的飞机，比如泰坦的 Solara 型号，预计会更加严格。

再加上飞机可能无法在 65000 英尺的高度持续飞行的风险——这将高于 60000 英尺的 A 类空域——这种风险可能超过了许多风投希望承担的风险。

少数几个仍然感兴趣的投资者之一是 [In-Q-Tel](https://web.archive.org/web/20230404105104/http://www.crunchbase.com/financial-organization/in-q-tel) ，这是一个独立的非营利组织，投资于风险投资支持的初创公司，这些公司支持中情局和更广泛的美国情报界的需求。In-Q-Tel 之所以留下来，只是因为泰坦正接近与[桑迪亚国家实验室](https://web.archive.org/web/20230404105104/http://www.sandia.gov/)达成协议。但它不会领先一轮。

这意味着 Titan 不得不关闭其他风投，或者转向其他资金来源，比如高净值个人。这可能需要一些时间。

随着该公司面临这种不确定性，与脸书的交易可能开始看起来是该公司及其员工的更好选择。这意味着泰坦将有资金继续运营，并成为脸书未来使命的战略组成部分。如果脸书能够像 Internet.org 计划的那样，在世界上其他缺乏互联网接入的地方部署互联网接入，那么脸书本身就可能成为一家比现在有价值得多的公司。

与脸书收购的其他公司相比，泰坦还可以继续更加独立地运营。例如，他们不一定需要像一家基于软件的初创公司那样，搬迁到脸书位于门洛帕克的总部。然而，他们仍然有在大公司工作的好处，包括像人力资源、会计等部门的一般配备。这可能有助于 Titan 将更多精力放在产品上，而不是运营自己的公司。

最后，对于即将加入脸书的泰坦员工来说，脸书的交易从长远来看也是有利可图的。除了买断要约(据报道为 6000 万美元)，脸书将向泰坦员工发放 2500 万美元的限制性股票。脸书在向 Titan 提出收购时对该公司的估值为 7500 万美元。
