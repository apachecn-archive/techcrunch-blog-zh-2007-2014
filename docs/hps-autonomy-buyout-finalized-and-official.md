# 惠普收购 Autonomy 的交易已经完成，TechCrunch 正式发布

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/03/hps-autonomy-buyout-finalized-and-official/>

# 惠普正式完成对 Autonomy 的收购

惠普刚刚宣布，根据上个月指定的条款，它已经完成了对英国企业数据处理器 Autonomy 的收购。也就是说:213，421，299 股每股 25.50 英镑，总计略高于该公司的 87%。这大约是 100 亿美元收购要约中的 85 亿美元(路透社[称](https://web.archive.org/web/20230203135506/http://www.reuters.com/article/2011/10/03/us-hp-autonomy-idUSTRE79269E20111003)120 亿美元)。

许多人认为收购价格过高，但这项收购是李艾科重组公司计划的关键之一。当然，李艾科[不久前](https://web.archive.org/web/20230203135506/https://techcrunch.com/2011/09/22/its-official-at-hp-apotheker-is-out-meg-whitmen-named-president-and-ceo/)离开了公司，就在几周前，为新任首席执行官梅格·惠特曼让位，并带走了约 1000 万美元的股票和奖金。

Autonomy 将作为一个“独立的业务部门”运营，尽管惠普将使用他们的新资产来“加速我们的战略愿景”和“重新发明如何处理、分析、优化、自动化和保护非结构化和结构化数据”，李艾科说，并“管理和提取数据的意义，以推动洞察力、远见和更好的决策”。

该消息在当天收盘时发布，惠普股价在尾盘交易中下跌，收于 22.20 美元，下跌 0.25 美元或 1.11%。
