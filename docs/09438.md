# Zappos 首席执行官谢家华领投私人飞机公司 Jet suite 700 万美元

Zappos 首席执行官[谢家华](https://web.archive.org/web/20230203060328/http://www.crunchbase.com/person/tony-hsieh)今天早上宣布他将回归投资，领导私人飞机公司 [JetSuite](https://web.archive.org/web/20230203060328/http://www.jetsuite.com/) 的 700 万美元投资。这将是谢家华自 24 岁亲自投资 Zappos 以来的第一笔投资。

除了这笔资金，谢长廷还将加入 JetSuite 董事会，一同加入的还有 JetSuite 首席执行官亚历克斯·威尔科克斯(Alex Wilcox)、JetBlue 创始人大卫·尼尔曼(David Neeleman)和 JetSuite 董事会主席阿特·萨姆伯格(Art Samberg)。

JetSuite 将利用这笔融资购买更多飞机，为其 Embraer Phenom 100 飞机机队增加两架飞机。Jetsuite 目前在美国西南部、美国大陆、加拿大和墨西哥私人飞行。

谢先生告诉我，投资主要是试图增加拉斯维加斯和旧金山之间的航班数量，“我在未来几年的重点是投资有助于拉斯维加斯市中心复兴的项目……对 JetSuite 的投资将有助于缩短拉斯维加斯市中心和硅谷以及洛杉矶之间的心理距离，”大概是通过增加两者之间的航班。

谢先生告诉我，这项服务是私人的，还有一个好处是你可以避开机场安检，“你可以在旧金山走出你的房子或办公室，两个小时后再走进你在拉斯维加斯的房子或办公室(反之亦然)。”

为了与“振兴拉斯维加斯”的主题保持一致，谢长廷最近还宣布投资 [First Friday](https://web.archive.org/web/20230203060328/http://www.lvrj.com/news/zappos-foursome-buying-arts-event-129708593.html) 拉斯维加斯。
