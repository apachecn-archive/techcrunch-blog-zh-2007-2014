# 评论钓鱼有一个心理学解释

评论巨魔是大多数网站的克星；由于绝大多数人可能永远不会对一篇帖子发表评论(TechCrunch 肯定就是这种情况)，所以那些确实会发表评论的帖子通常属于以下三类之一。

合法的评论者，他们有一些深思熟虑的话要说和/或添加到对话中(支持和反对)，链接垃圾邮件发送者评论他们的评论可能会因为他们的评论而获得一些流量(有时这些属于第一类..很难说)和巨魔，他们把批评任何写的东西和写它的人作为自己的事业，以某种可悲的方式试图通过为了自我肯定而对他人不友好来自我肯定。就像吸毒一样，钓鱼是一个可怜的情妇，她需要越来越多的东西来满足支撑她们自尊的自我满足的瘾。

《新科学家》对网络上的不良行为做出了心理学解释。虽然《邮报》主要谈论的是电子邮件，但《新科学家》指出了电子邮件和评论巨魔之间的关系。你可以在这里阅读整个帖子[，但这里有一些亮点:](https://web.archive.org/web/20230203100414/http://www.newscientist.com/blog/technology/2007/11/dont-flame-me-bro.html)

> 几十年来，社会心理学家已经知道，如果我们降低自己的身份感——这一过程被称为去个性化——我们就不太可能坚持社会规范……电子邮件等在线交流也是如此。从心理上来说，我们与谈话对象“距离遥远”,不太关注自己的身份。他说，结果是我们更容易有攻击性行为。
> 
> 根据 Epley 的说法，影响在线交流的另一个因素是基于文本的信息存在沟通错误的风险，这种信息本质上更加模糊。他指出，与此同时，电子邮件“给人一种不正式的感觉——我们只是发送一些东西”，尽管我们或许应该像对待书面信函一样小心对待它。而且，正如大多数人可能知道的那样，这会给发送方和接收方都带来问题。

我将把最后的话留给《新科学家》的迈克·马歇尔:

> 我不确定我们能做些什么来减少网上的误解和滥用。但是意识到我们并不像我们想的那样擅长网络交流似乎是一个好的开始。我知道我经常要克制自己不加入。
