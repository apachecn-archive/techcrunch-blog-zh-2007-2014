# 脸书的 iPad 应用终于来了。是的，这次是真的。

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/10/facebooks-ipad-app-is-finally-here-yes-for-real-this-time/>

不，这不是谣言。这不是泄密。它不会在最后一刻被取消，因为幕后谈判可能会发生，也可能不会发生。

脸书的 iPad 应用已经为每个人准备好了。

是的，在几个月的[延迟](https://web.archive.org/web/20230205042957/https://techcrunch.com/2011/09/26/facebook-ipad-app-saga/)，谣言，甚至是一个[泄露的版本](https://web.archive.org/web/20230205042957/https://techcrunch.com/2011/07/25/facebooks-hidden-ipad-app/)传到了各地 TechCrunch 读者的手中之后，脸书正式推出了它的 iPad 应用程序，这是一个长期以来令人瞩目的缺席(事实上，如此引人注目，以至于一些第三方开发者已经获得了他们自己的“iPad 版脸书”应用程序的数百万次下载)。

我还没有亲自试用过这个官方应用，但是脸书给了我一个关于它的亮点的口头演示，以及下面的截图。看起来不错。看起来像脸书。它将成为 iPad 有史以来下载量最大的应用程序，哦，大约两天后。你可以在这里下载。更新:脸书展示这款应用的网站已经上线，但看起来它仍在通过苹果的服务器传播。

这款应用在屏幕左侧有一个光滑的导航菜单，让你可以快速跳转到新闻提要、消息、你的群和你在脸书上经常访问的其他地方。该应用的其余部分将你期望从脸书的移动体验中获得的功能(如签到)与你在桌面上找到的功能结合在一起，如新闻提要旁边的聊天栏。脸书的 iPhone 应用程序也没有被冷落——它也获得了用户界面的更新，包括光滑的导航菜单。

还有另一个非常重要的新功能:脸书将应用程序发现带到了移动设备上。

它是这样工作的。

现在，如果你从桌面浏览你的新闻，你会注意到你偶尔会看到朋友用各种应用和游戏发布的更新。点击这些更新中出现的一个链接，你就会被定向到有问题的应用程序——这是应用程序在脸书病毒式传播的一种方式(另一种方式是通过“请求”渠道，让用户明确邀请他们的朋友加入某个应用程序或游戏)。

在此之前，这些功能在脸书手机版中被忽略了，但从今天的发布开始，它们得到了深度整合。

举例来说，你正在通过新的 iPad 应用程序浏览脸书，你看到一个朋友在你的新闻订阅中发布了一个与朋友的故事。如果你点击了那个链接，并且已经安装了原生版的 iPad 版的单词，那么脸书会快速将你切换到这个游戏。如果你还没有安装应用程序，它会将你链接到应用程序商店，你可以在那里下载它。如果根本没有本地应用可用，那么你会被定向到 HTML5 版本。

这个应用程序集成也适用于脸书的移动网页版。简而言之，脸书正在尽最大努力向你推荐最适合你在任何设备上使用的应用程序版本——所以如果你从 m.facebook.com 浏览脸书，你会被推荐到应用程序的移动网络版本；如果你在 iOS 上浏览，你将获得应用商店版本；当脸书完成了 Android 应用程序的修改版本(它正在工作中)，你将尽可能被引导到 Android Market 版本的应用程序。

这对应用开发者来说是一件大事，因为这意味着移动应用可以更容易地进入脸书的病毒式渠道。这也意味着邀请和请求将在所有平台上无缝工作，前提是应用程序开发人员已经构建了多个版本的应用程序(和/或 HTML5 版本)。

这也带来了另一个问题:这个[项目是斯巴达](https://web.archive.org/web/20230205042957/https://techcrunch.com/2011/06/15/facebook-project-spartan/)吗？

我的同事 MG Siegler 广泛报道了脸书的一项内部任务，即敦促第三方应用程序开发者转向 HTML5，从而将他们的应用程序带到更多的设备上，并放松苹果通过 App Store 对 iOS 的束缚(以及谷歌通过 Market 对 Android 的束缚)。

我与脸书首席技术官 Bret Taylor 讨论了今天发布的功能，以及它们之间的关系。他说，谣言可能误解了脸书的社交应用发现方法，虽然脸书是 HTML5 的支持者，但这些功能是平台不可知的。他解释道(这是现场转录的，所以略有转述):

> “有很多平台多样性，如果你希望你的应用程序是社交的，你确实希望你的应用程序为多个平台开发——我认为 HTML5 是其中的一个重要部分，因为它是跨所有这些设备工作的一个平台。但我不认为它会以牺牲这些平台为代价而成功。我们是 HTML5 的大力支持者，我们相信它是一个必要的组件…但是对于许多应用程序开发者来说，今天没有 iOS 应用程序将是疯狂的。大多数社交应用开发者可能会制作一个……”

不过，从长远来看，脸书似乎会在 HTML5 上做得更多。它已经写了一些关于使用 HTML5 制作 2D 游戏的开发者博客帖子，Taylor 说脸书可能会开源额外的工具来帮助开发者进入 web 开发环境。

我还向泰勒询问了 iPad 应用程序的历史，据报道，它被用作脸书与苹果谈判的一个重要筹码。泰勒回避了这个问题，说了一些类似的话，“我们做了很多事情，当他们准备好了的时候，我们就发布他们。”这并不完全合理，因为开发这款应用的脸书开发者说，它在 5 月份完成了功能，他辞职了，因为社交网络花了太长时间才发布它。
