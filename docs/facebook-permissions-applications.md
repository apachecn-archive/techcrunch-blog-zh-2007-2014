# 脸书推出简化的申请许可系统

今天，脸书宣布为第三方应用程序推出一个新的许可模式，要求应用程序明确指定他们希望访问的数据(并给予用户选择退出的能力)。这一改变已经酝酿了很长时间:脸书在[的博客文章](https://web.archive.org/web/20221001150043/http://blog.facebook.com/blog.php?post=403443752130)中提到，早在 2009 年 8 月，由于[加拿大隐私专员](https://web.archive.org/web/20221001150043/https://beta.techcrunch.com/2009/08/27/facebook-bows-to-canadian-privacy-concerns-will-change-the-way-all-apps-access-social-data/)提出的隐私问题，这一改变首次被宣布。新的对话框在四月的 f8 上[展示](https://web.archive.org/web/20221001150043/https://beta.techcrunch.com/2010/04/21/hacking-the-graph-live-from-facebooks-f8-conference/)。

第三方脸书应用程序总是需要征求用户的许可才能访问他们的私人数据，但在此之前，对话框是不具体的，有时需要用户在一系列框中单击“是”，这可能会令人困惑。现在，应用程序希望访问的所有数据都将出现在一个对话框中。像往常一样，应用程序将能够访问你与“每个人”共享的数据(这实际上包含了相当多的数据)。除此之外，用户还会被用简单明了的英语告知他们共享了哪些隐私数据，其中可能包括你的电子邮件地址、照片和视频。这绝对是一个进步。

一个月前，脸书还对其主要的[隐私控制面板](https://web.archive.org/web/20221001150043/https://beta.techcrunch.com/2010/05/26/live-blog-facebook-unveils-new-privacy-controls/)进行了修改，使管理你的设置变得更容易(并完全关闭 Facebook 平台)。尽管如此，这些变化还不足以让隐私倡导者闭嘴，尤其是在脸书备受争议的即时个性化功能方面。
