# Meebo 将文件共享添加到 Webchat 中

[![](img/9df8afc39235377e68ed4235f89a0e16.png)](https://web.archive.org/web/20221006210949/http://www.meebo.com/) 网络聊天初创公司 [Meebo](https://web.archive.org/web/20221006210949/http://www.crunchbase.com/company/meebo) ，仍然沉浸在将[即时消息引入 iPhone](https://web.archive.org/web/20221006210949/http://www.beta.techcrunch.com/2007/08/15/most-useful-iphone-site-yet-meebo/) 的喜悦中，今晚发布了一个备受期待的新功能:文件共享。

有几十种基于网络的文件共享应用程序，如 [YouSendIt](https://web.archive.org/web/20221006210949/http://www.yousendit.com/) 和 [DropSend](https://web.archive.org/web/20221006210949/http://www.dropsend.com/) (我们在这里回顾了[四种 P2P 解决方案](https://web.archive.org/web/20221006210949/http://www.beta.techcrunch.com/2006/08/24/lets-share-some-files-four-services-compared/))，但用户通常想在与朋友聊天时简单地向他们发送文件。基于桌面的聊天很容易做到这一点，它通常允许拖放文件传输。但是直到现在，网络聊天还没有对等物。

服务的后端托管在亚马逊 S3。Meebo 正在消耗这项服务的成本，但将每个文件的传输限制在 10 MB，每个用户每月的发送限制为 30 MB。在 Meebo 网站上与人聊天时，寻找一个新的“发送文件”按钮。

Meebo 已经从 Sequoia 和 Draper Fisher Jurvetson 总共筹集了 1250 万美元。他们说，每月有 600 万人使用这项服务，每天花大约 2 小时，每周几次。
