# 美国电话电报公司回应司法部:法庭将以事实为准

继今天的报道称[美国司法部已提交诉状](https://web.archive.org/web/20230203045531/https://techcrunch.com/2011/08/31/reports-us-justice-department-looks-to-block-atts-acquisition-of-t-mobile/)反对美国电话电报公司&以 390 亿美元收购 T-Mobile 之后，蓝色巨人运营商发布了回应，表示计划要求加速听证，“以便能够充分评估这次合并的巨大利益”

DOJ 在诉状中援引了反垄断问题，称“美国电话电报公司将 T-Mobile 作为一个独立的低价竞争对手排除在外，将会从市场中消除一股重要的竞争力量。”

尽管联邦通信委员会主席 Julius Genachowski 说审查程序尚未完成，但这仍然是美国电话电报公司和 T-Mobile 走向婚姻的另一大障碍。如果这个障碍太大而无法克服，美国电话电报公司将不得不向 T-Mobile 的母公司德国电信支付超过 60 亿美元。换句话说，美国电话电报公司的日子并不好过。

以下声明由美国电话电报公司高级执行副总裁兼总法律顾问 Wayne Watts 提供:

> 我们对今天的行动感到惊讶和失望，特别是因为我们多次与司法部会晤，而 DOJ 没有表示正在考虑这一行动。我们计划要求加快听证会，这样就可以充分评估这次合并的巨大利益。DOJ 有责任证明所谓的反竞争影响，我们打算在法庭上就此事展开激烈辩论。
> 
> 归根结底，我们相信事实将指导任何最终决定，事实是清楚的。此次合并将:
> 
> *   帮助解决我们国家的频谱枯竭状况，改善数百万人的无线服务。
> *   允许美国电话电报公司将 4G 移动宽带扩展到另外 5500 万美国人，或 97%的人口。
> *   导致数十亿的额外投资和数以万计的就业机会，而此时我们的国家最需要它们。
> 
> 我们仍然相信这一合并符合消费者和我们国家的最佳利益，事实将在法庭上占上风。
