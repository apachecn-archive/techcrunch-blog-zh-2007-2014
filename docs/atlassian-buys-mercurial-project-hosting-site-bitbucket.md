# Atlassian 收购 Mercurial 项目托管网站 BitBucket

7 月，产品开发软件公司 [Atlassian](https://web.archive.org/web/20230130100805/http://www.atlassian.com/) 表示，将把最近筹集的[6000 万美元](https://web.archive.org/web/20230130100805/https://techcrunch.com/2010/07/14/atlassian-accel-60-million/)投资用于企业领域的并购。看起来公司进展很快——Atlassian 刚刚收购了代码协作托管服务[Bitbucket.org](https://web.archive.org/web/20230130100805/http://bitbucket.org/)。交易条款没有披露。

Bitbucket 拥有超过 60，000 个帐户，是 Mercurial 分布式版本控制系统的主要托管代码协作提供商，并为希望共享和协作项目的开发人员提供服务。Bitbucket 类似于 GitHub 或 Google Code，托管 Adium、MailChimp、Opera 和其他伟大的开源项目的代码。

分布式版本控制系统分散了软件库，允许每个用户在其本地机器上拥有源代码库的完整副本。它允许用户在没有网络束缚的情况下工作。

BitBucket 将被纳入 Atlassian 的开发产品家族。Atlassian 的软件开发和协作工具，帮助团队构思、规划、构建和推出产品。产品包括 JIRA(问题跟踪)和合流(内容协作)。该公司的产品被全球 20，000 名客户使用，包括脸书、Zynga、思科和 Adobe。

作为整合的一部分，Atlassian 将使 Bitbucket 完全免费，为 5-dev 提供免费托管，拥有无限的私有存储库。作为一项推广活动，Atlassian 为 10 个用户的账户提供一年的免费服务。

Atlassian 表示，此次收购有助于该公司填补产品供应方面的空白，并使 Atlassian 成为开发者更全面的平台。因此，开发人员现在有一个地方来转储和托管他们的代码，并在 Atlassian 中跟踪他们的项目问题和 bigs。Atlassian 的 Jay Simons 告诉我，除了技术团队之外，该公司希望做 Adobe 为设计师做的事情。
