# CEATEC 2011:罗门推出世界上最小的电阻

我承认，在今年的 [CEATEC 2011](https://web.archive.org/web/20230205030033/https://techcrunch.com/tag/ceatec-2011/) 展会上，我刚刚走过微芯片制造商 [Rohm](https://web.archive.org/web/20230205030033/http://www.rohm.com/index.html) 的展台，但这家日本公司实际上展示了一些非常有趣的东西:它开发了世界上最小的[芯片电阻器](https://web.archive.org/web/20230205030033/http://www.rohm.co.jp/news/111003.html)【JP】，只有 0.3 x 0.15mm 毫米，比上一款小了 44%。

Rohm 在 CEATEC 的宣传是，50 万个新电阻器可以用在沙漏中(而不是沙子)。

事实上，该公司表示，这些设备太小，无法用目前可用的设备进行加工:日本商业日报日经报道称，罗门目前正在寻找合作伙伴，帮助开发将电阻器实际安装在电路板上的技术。

[![](img/30509e977c309d28fb51dc0ec85ca62d.png "rohm-2")](https://web.archive.org/web/20230205030033/https://techcrunch.com/wp-content/uploads/2011/10/rohm-2.jpg)

例如，智能手机使用多达 400 个芯片电阻，这意味着电阻越小，理论上未来手机可能会变得更薄更轻。

Rohm 目前计划在 2012 年秋季开始大规模生产这种电阻。
