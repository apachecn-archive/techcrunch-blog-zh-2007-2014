# 五个月内，200 万个网站使用了脸书的新社交插件

在今天的 Web 2.0 博览会上，脸书首席技术官布雷泰勒 T2 宣布，200 万个网站增加了“喜欢”按钮和脸书其他新的社交插件，这些插件是在五个月前推出的。脸书也在推特上证实了这个数据。

考虑到仅仅五个月前这个数字还是 5 万,这真是令人印象深刻。脸书的新[社交插件](https://web.archive.org/web/20230213094126/http://developers.facebook.com/plugins)，包括一个网络“喜欢”按钮和一系列其他插件，于四月在 [f8](https://web.archive.org/web/20230213094126/https://techcrunch.com/2010/04/21/zuckerbergs-buildin-web-default-social/) 发布。

除了“喜欢”按钮，脸书的其他社交插件还包括一个显示你朋友活动的活动源、一个推荐文章的小工具，以及一个向你展示也使用你浏览的网站的朋友的照片的 Facepile。脸书[最近发布了](https://web.archive.org/web/20230213094126/https://techcrunch.com/2010/08/17/facebook-like-button-2/)一款更轻薄、更时尚的纽扣。

在今天发布的一份说明中，脸书表示，一些使用新插件的出版商在嵌入新的社交插件后，流量有所增加。例如，美国广播公司新闻看到了 190%的跳跃和 Gawker 看到了 200%的流量增长。出版商也报告说，他们的男性参与度有所提高。例如，NHL.com 报告称，访客阅读的文章增加了 92%，在网站上停留的时间增加了 85%，观看的视频增加了 86%，产生的访问量增加了 36%。
