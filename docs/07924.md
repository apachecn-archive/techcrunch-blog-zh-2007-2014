# 英国警方声称已经逮捕了 LulzSec 成员“Topiary”。接下来会发生什么？

一名 19 岁的男子在设得兰群岛被逮捕，他被指控为 LulzSec 的发言人，“植园”。如果被判有罪，接下来会发生两件事之一…

**互联网结局:**

“托皮瑞”(topary)——他被揭露是一个黑暗、忧郁、穿着皮革衣服的 1995 年左右的约翰尼·李·米勒——通过与中央情报局(CIA)达成附带协议，帮助他们拿下一名威胁向伊朗人出售五角大楼机密的流氓黑客，从而避免了入狱。一切进展顺利，直到我们了解到"绿色花园"的中情局联系人实际上是一名双重间谍，为朝鲜工作。现在被三个政府追逐，“绿植”必须恢复五角大楼的秘密，拿下这两个流氓特工，并防止第三次世界大战。这部电影以“Topiary”的记录被抹去而告终，我们的英雄——事实证明，他是骗子世界中唯一诚实的人——与奥利维亚·维尔德一起驶向夕阳，他的前方是一生的高薪企业咨询工作。(*寓意:我们匿名。我们是军团。我们不原谅。我们可以和《创:传奇》里的小妞做爱。*)

**现实世界结局:**

托皮瑞——他被发现和你想象的一模一样——被引渡到美国，并在亚利桑那州监狱度过了接下来的十年。(*寓意:我们是男人。你认为制造可爱的盖伊·福克斯面具的公司是谁的？*)

当然，现在猜测哪种结局会发生还为时过早。很可能是第一次。
