# 你爱管闲事的男/女朋友可以在你睡觉的时候用拇指解锁你的 iPhone 5s

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/09/20/fingerprint-unlock-while-sleeping/>

密码还不会死。iPhone 5s 的触控 ID 指纹安全系统可以用手指解锁，即使你睡着了。这意味着一个嫉妒的情人可以在你睡觉的时候拿着你的手机，阅读你所有的短信、通话记录、电子邮件等等。

苹果公司证实，死拇指是不行的。对受害者施以氯仿麻醉或许可行，但国际间谍要想访问他们的安全文件/自拍照，切掉总理的拇指是不可能的。苹果也不会将指纹副本发送回其服务器，而是将它们存储在 A7 处理器中的一个“安全区域”，黑客或其他应用程序无法访问。

苹果公司努力使触控 ID 安全系统 T2 易于使用。如此简单，一只猫、你的[脚趾](https://web.archive.org/web/20230305162025/http://www.buzzfeed.com/jwherrman/yeah-you-can-unlock-the-new-iphone-with-your-toe)，甚至你的……成员都可以[解锁 5s，如果它已经在你的手机上注册了的话。然而，真正的问题是，触控 ID 无法判断一个人是否昏过去了。](https://web.archive.org/web/20230305162025/https://techcrunch.com/2013/09/19/watch-a-cat-unlock-the-iphone-5s-using-touch-id-and-the-fingerprint-sensor/)

兄弟会的伙计们，注意了。你可能在一夜饮酒后醒来，发现你的兄弟给你所有的前任发了信息，并且创造性地重写了你的脸书个人资料。然而，最大的威胁可能是被其他重要的人滥用。

我们经常听到这样的故事:一个多疑的女朋友或男朋友在他/她睡着的时候翻他/她没上锁的手机，发现他们在和别人调情，然后甩了他们。数字密码可以防止这种情况。

但是触控 ID 很容易受到“睡眠黑客”的攻击

只要有人知道你在触控 ID 注册了哪个手指，他们就可以从床头柜上拿起你的手机，按在你熟睡的手指上，瞧，手机解锁了。

如果你的手机里有可疑的私人物品，你应该…不要在手机里有可疑的私人物品。如果你的另一半在你睡觉的时候翻你的手机，那你就有更大的问题了。但是如果你被某个不择手段的人困住而无法入睡，你可能需要进入你的设置，启用密码锁定，并删除你文件上的指纹。

实际上，这一切都可以归结为一个想法，即人类还没有开发出既方便又 100%安全的密码。不是长字符串，不是面部识别，也不是指纹。缺乏完美的数字安全已经成为我们文化的一部分——这是一种风险和不便，目前没有人能置身事外。在这一点上，我将留给你这幅由涂鸦艺术家[班克斯](https://web.archive.org/web/20230305162025/http://www.banksy.co.uk/)创作的感人画作/诗歌:
