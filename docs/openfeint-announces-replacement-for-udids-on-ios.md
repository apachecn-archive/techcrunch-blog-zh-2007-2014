# OpenFeint 宣布取代 iOS 上的 udid TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/26/openfeint-announces-replacement-for-udids-on-ios/>

在苹果决定逐步取消开发者在 iOS 设备上访问 UDID(唯一设备标识符)之后，移动社交游戏网络 [OpenFeint](https://web.archive.org/web/20230203125233/http://openfeint.com/) 提供了一个替代解决方案。公司[今天宣布](https://web.archive.org/web/20230203125233/http://openfeint.com/company/press/45-OpenFeint-Launching-Single-Sign-On-for-Social-Games-as-a-Replacement-for-UDID)将为社交游戏开发者推出单点登录系统，取代 iOS 上的 UDIDs。

该系统将于今年秋季晚些时候上市。

最新的 iOS5 发行说明[中提到了对 UDID 访问的更改，并将其列为“已弃用](https://web.archive.org/web/20230203125233/https://techcrunch.com/2011/08/19/apple-ios-5-phasing-out-udid/)”UDID 是每个苹果设备独有的字母数字字符串，目前被移动广告网络、游戏网络、分析提供商、开发者和应用测试系统使用，例如 [TestFlight](https://web.archive.org/web/20230203125233/http://www.testflight.com/) 。在某些情况下，开发者甚至使用 UDID 来验证用户是否正在通过新设备访问他们的应用程序。

换句话说，这种变化是一件非常非常重大的事情。

![](img/e5e0cfb91d8338ea7f52d678d1c09a20.png "openfeint-sso")

然而，有了 OpenFeint 的新系统，游戏开发者社区至少会有另一种选择。用户将选择加入 OpenFeint 的新单点登录系统，然后让开发者访问 OpenFeint 的 OFUID，这是一个面向游戏网络跨平台用户的通用帐户系统。

使用 OFUID，开发者可以检查和分析用户在游戏中的行为，以便更好地了解他们的应用程序中有哪些工作，哪些需要修复。这也将有助于游戏开发商的广告定位工作。OpenFeint 表示，所有访问的数据都将被匿名化，以保护用户隐私。

不幸的是，与苹果的 UDID 不同，OFUID 只有在用户选择登录 OpenFeint 时才有效。显然，它不是 UDID 的通用替代品，因为 OpenFeint 只专注于手机游戏。

此外，OpenFeint 宣布了一个附带的“安装交易计划”，供开发者推广新系统。OpenFeint 表示，它将保证开发者在其“游戏频道”的每个新安装中安装 1.5 个新应用程序，这是单点登录系统工作所需的新应用程序。

对早期访问感兴趣的开发者可以在这里注册参与新计划[。](https://web.archive.org/web/20230203125233/http://www.openfeint.com/developers/features/single_signon)

OpenFeint 的网络上现在有超过 6900 款游戏，注册用户超过 1.15 亿。今年 4 月，It [被日本手机游戏公司 GREE](https://web.archive.org/web/20230203125233/https://techcrunch.com/2011/04/21/japanese-company-gree-buys-mobile-social-gaming-platform-openfeint-for-104-million/) 以 1.04 亿美元收购。
