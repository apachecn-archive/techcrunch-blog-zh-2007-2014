# 随着 Adobe flash paper 的停产，记录混乱中的初创企业

今天，随着 Adobe[停止开发其 Flashpaper 产品](https://web.archive.org/web/20230130100805/http://www.adobe.com/products/flashpaper/eod_faq/)的消息传出，许多欧洲初创公司——以及全球许多其他公司——将陷入混乱。

Adobe 将继续销售和支持当前的 FlashPaper 2 版本，但不会更新技术以支持 Microsoft Windows Vista 和 IE7，这将使它几乎毫无价值。

这一消息将打击美国的网站，如 ~~[Scribd](https://web.archive.org/web/20230130100805/http://scribd.com/)~~ 和 [Docstoc](https://web.archive.org/web/20230130100805/http://docstoc.com/) ，以及欧洲的网站，如英国的 [edocr](https://web.archive.org/web/20230130100805/http://edocr.com/) 和德国的 [Twidox](https://web.archive.org/web/20230130100805/http://www.twidox.com/) ，它们最近才获得资金。edocr 目前所有的文档共享都基于 Flashpaper。

Twidox 首席执行官 Nicholas MacGowan von Holstein 今天联系了 TechCrunch UK，称此举将产生重大影响:“那些用 Flashpaper 存储所有文件的网站怎么办？将所有这些文档转移到新的解决方案将是一项艰巨的任务。”他说，Twidox 不会受到严重打击，因为他们已经计划了替代方案。他说，他是在“直接给 Adobe 写了三封信之后”才知道这个消息的。

Adobe 的网站上说:

> “经过对市场和客户反馈的仔细考虑和分析，Adobe 计划停止为 FlashPaper 开发新功能。需求持续下降，Adobe 继续为 FlashPaper 提供开发支持在经济上已不再可行。”
