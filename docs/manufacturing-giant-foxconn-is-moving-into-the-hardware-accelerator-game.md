# 制造业巨头富士康正在进军硬件加速器游戏

电子产品巨头富士康(Foxconn)正在硬件加速器市场试水，将一个未被充分利用的诺基亚园区改造成其首个小型企业拓展中心。富士康首席投资总监潘石屹在举行的 TechCrunch/Technode(T3)大会上介绍说，该计划已经引导一家小型制造商走向市场，并将在未来一年帮助其他制造商。

这个孵化器还没有完全正式化，它将与小型制造商合作，让他们接受一段时间的强化辅导和指导。[智能手表公司 Tomoon](//web.archive.org/web/20221005183235/https://www.the-digital-reader.com/2013/11/12/tomoon-t-fire-smartwatch-shipping-soon-china-android-e-ink-screen/#.UosnObVDuhM%E2%80%9D) 与富士康合作开发了 T-Fire e-ink 智能手表，这是一款独特的 Android 智能手表，配有曲面屏幕。该公司在大约六周的时间里从一个简单的想法变成了一个成熟的产品。

“当他们来找我们的时候，就像一个玩具，”潘说。他说，该公司帮助改进设计，并提供制造专业知识，以创造最终产品。

加速器的启动日期尚未确定，但潘表示，公司正在为新公司制定标准操作程序。他预计富士康将获得一些股权，以换取硬件方面的帮助。
