# 明亮的 3D 驱动:爱普生推出世界首个 HTPS-薄膜晶体管 3D 面板

多年来，爱普生一直致力于开发 HTPS 高温多晶硅薄膜晶体管面板 T2，T4，昨天，爱普生宣布推出世界上第一款 3LCD 投影仪。将有两个模型，一个测量 0.74 英寸(如图)，另一个与 0.61 英寸，都将全高清分辨率。

根据爱普生的说法，这种新设备使用了一种被称为“明亮的 3D 驱动”的技术，它有助于将图像刷新率从 240 赫兹提高到 480 赫兹。因此，新面板产生的 3D 图像据说比 240 赫兹模型产生的图像亮 1.5 倍。

[![](img/c4fd00b2537ad87b2d76a524236002ed.png "epson 3d")](https://web.archive.org/web/20230204121218/https://techcrunch.com/wp-content/uploads/2011/09/epson-3d.jpg)

爱普生已经开始大规模生产他们的 HTPS 3D 面板，并将提供给其他投影仪制造商，太。
