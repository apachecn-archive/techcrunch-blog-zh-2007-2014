# YouTube 表示，从今年秋天开始，它将把广告客户纳入其合作伙伴计划

YouTube 刚刚宣布(作为本周戛纳狮子广告活动的一部分)，它正在扩大其合作伙伴计划，以包括广告商。

该计划已经[为顶级 YouTube 内容创作者](https://web.archive.org/web/20221206144840/https://beta.techcrunch.com/2007/05/04/youtube-launches-revenue-sharing-partners-program-but-no-pre-rolls/)提供了资源，以提高他们的制作技能和发行能力(并因此提高他们的盈利能力)。

YouTube 表示，广告计划的结构将与现有的合作伙伴计划完全相同——只是，你知道，针对广告商。该项目将于 9 月开始，在洛杉矶举行为期一周的研讨会。每个广告客户都将被指派一名合作伙伴经理来帮助他们制定内容策略。

“通过邀请广告商加入我们的合作伙伴计划，我们希望让他们获得资源和专业知识，帮助他们在 YouTube 上开发更具吸引力和真实性的内容，”YouTube 视频在线全球销售副总裁卢卡斯·沃森(Lucas Watson)在公告新闻稿中说。

通过这个项目，YouTube 所有者谷歌似乎在鼓励企业创建专门为该视频网站设计的广告，而不仅仅是重新包装现有的电视广告和视频内容。对于 YouTube 及其内容合作伙伴来说，这种定制活动可能更有利可图。该公司还表示，它将“促进与 YouTube 一些顶级创作者的战略联盟的发展”，以便广告和内容可以以新的方式整合。

最初的广告合作伙伴是美国运通、通用电气、强生公司和百事公司，以及它们的广告公司。这目前只是一个试点项目，但 YouTube 计划在 2014 年底前将它扩展到 100 家广告商。
