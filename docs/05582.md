# 苹果终于准备好切断我头上烦人的电线了吗？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/11/09/apple-wireless-headphones/amp/>

 <amp-anim class="alignright size-full wp-image-241875 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="im" src="https://web.archive.org/web/20230207135731im_/https://techcrunch.com/wp-content/uploads/2010/11/im.gif" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-anim> 在我生活中的所有烦恼中，我最经常处理的莫过于耳机线的解开了。一天有几次，我把手伸进口袋，拿出一团绳子，看起来好像是被某种袖珍侏儒故意缠在一起的。这让我发疯。

无线耳机并不新鲜，但最好的耳机通常不适合放在你的口袋里:对我来说，这是耳机的首要要求。所以我坚持用我的衣服自带的那种。但是，如果 9to5Mac 的消息来源是正确的，这些可能很快就会得到重大检修。

Seth Weintraub 了解到,[几个月前，苹果显然非常低调地收购了总部位于旧金山的公司 Wi-Gear。他们的专业领域？蓝牙无线耳机(特别是 A2DP 立体声耳机)和适用于旧设备的蓝牙 2.0 适配器。](https://web.archive.org/web/20230207135731/http://www.9to5mac.com/34608/apple-buys-wi-gear-to-build-stereo-bluetooth-headphones)

果不其然，该公司现在[停业](https://web.archive.org/web/20230207135731/http://www.wi-gear.com/about_us.php)，至少他们的联合创始人之一迈克尔·金(Michael Kim)现在在苹果公司担任 iOS 蓝牙工程师。

虽然这样的收购可能只是为了增加一些蓝牙人才，但你必须假设苹果迟早会想用他们的耳塞进行无线连接。这次收购是他们朝着那个方向前进的最强烈的信号。

作为世界上最受欢迎的 MP3 播放器和最受欢迎的智能手机之一的制造商，苹果的白色耳塞已经无处不在。无处不在的烦人。是时候切断脐带了。

**更新**:看起来温特劳布是[错了](https://web.archive.org/web/20230207135731/http://www.macworld.com/article/155623/2010/11/apple_wigear.html)，苹果没有买 Wi-Gear，他们只是雇佣了金，然后公司就倒闭了。
