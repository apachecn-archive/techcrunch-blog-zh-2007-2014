# 最后，苹果 OS X 的谷歌地球浏览器插件

![](img/315bfe6ebffb24a111cad02c263125d3.png)期待已久的苹果 OS X T2(英特尔和 PowerPC)谷歌地球浏览器插件发布已经到来，大约在 Windows 插件发布半年后，以及通用苹果应用程序推出[5 周后。Linux 用户仍将不得不等待一个兼容的版本。](https://web.archive.org/web/20221007104528/http://www.beta.techcrunch.com/2008/10/27/google-earth-comes-to-the-iphone-and-its-awesome/)

您可以通过访问任何谷歌地球插件应用程序或访问[通用电气 API 网站](https://web.archive.org/web/20221007104528/http://code.google.com/apis/earth/)下载。注意:由于英特尔和 PowerPC 的兼容性，文件大小为 47MB。前往[earth flock](https://web.archive.org/web/20221007104528/http://www.earthswoop.com/)或[360 city](https://web.archive.org/web/20221007104528/http://360cities.net/earthbeta/atlanta-georgia-usa)观看其运行情况。

有趣的是看看这个版本是否会导致插件得到更广泛的支持，我很想知道开发人员会为他们的应用程序想出什么。
