# MongoHQ 从 Y Combinator、Lerer 和 SV Angel 融资 41.7 万美元

数据托管平台 [MongoHQ](https://web.archive.org/web/20230131004215/https://mongohq.com/) 已经筹集了 41.7 万美元的融资，以便为使用 MongoDB 进行数据存储的开发人员扩展他们的托管配置服务。保罗·格拉厄姆的 [Y Combinator](https://web.archive.org/web/20230131004215/http://www.ycombinator.com/) 、 [Start Fund](https://web.archive.org/web/20230131004215/http://www.startfund.com/) 、 [SV Angel](https://web.archive.org/web/20230131004215/http://www.svangel.com/) 和 [Lerer Ventures](https://web.archive.org/web/20230131004215/http://www.lererventures.com/) 参与了出资。

首先是 MongoDB 数据库托管场景，MongoHQ 与 [MongoLab](https://web.archive.org/web/20230131004215/http://www.mongolab.com/) 处于同一空间，并通过添加企业级功能来区分自己，旨在通过其新的副本集功能为用户提供最大的正常运行时间，这使开发人员可以选择在三个不同的可用性区域中跨三个 EC2 机器创建数据库。

目前，MongoHQ 拥有超过 7K 的用户，超过 12K 的数据库托管着大约 20tb 的数据。该公司成立于 2009 年，已经实现盈利，并已在国际上占据重要地位

据联合创始人[杰森·麦凯](https://web.archive.org/web/20230131004215/http://www.crunchbase.com/person/jason-mccay-2)称，该公司计划利用这笔资金投资基础设施并雇佣更多工程师。他们还计划听取用户反馈，并在今年夏天扩展到专用盒子和数据优化。“我们的目标是成为应用程序的数据层，”McCay 告诉我。
