# 苹果可能会为 iStuff 开发一个基于网络的诊断工具

如果你的 iPhone 或 iPad 开始变得不稳定，你很有可能会收拾好它，前往最近的天才吧。这很麻烦，但这是为你心爱的 iStuff 付出的小小代价。易用性是苹果对设备的关键标准之一，这远远超出了导航界面。

也就是说， [Hardmac](https://web.archive.org/web/20230203103434/http://www.hardmac.com/news/2011/08/31/apple-will-put-in-place-new-tools-for-diagnostic-of-breakdowns-of-devices-under-ios) 报道称，苹果内部发布了一款新的基于网络的 iOS 设备诊断工具，可以远程使用。换句话说，你的 iPhone 或 iPod 的未来问题变得更容易解决了。

它的基本工作原理是向行为怪异的设备发送一封带有 URL 的电子邮件。当 Safari 加载 URL(也可以在浏览器中键入)时，手机将开始运行测试，检查内部是否有任何问题。结果会传回给苹果技术人员，他们可以通过电话来解决问题。显然，如果你的设备损坏严重，无法启动 Safari，这将不是一个选项。

新工具可以检查设备的健康状况、电池电量、自上次充电以来经过的时间、电池放电的最低水平、你运行的 iOS 版本以及最近一次手机是否正确关闭。

目前还不清楚该工具是否会测试设备是否越狱，尽管如果不能，这似乎有点傻。越狱你的 iOS 设备实际上结束了你的保修期，这意味着无论苹果提供什么支持来解决你的问题，都是没有保证的。

Hardmac 声称新工具将在未来几个月内推出。
