# Kamcord 现在通过音频评论工具让手机游戏录制变得更加有趣

Y Combinator 支持的 Kamcord 正式推出还不到一年，这个年轻的团队已经花了时间[筹集资金](https://web.archive.org/web/20221205104227/https://beta.techcrunch.com/2012/12/19/mobile-game-recording-yc-alum-kamcord-raises-1-5-million-from-andreessen-horowitz-google-ventures-and-others/)并悄悄地充实其 iOS 游戏 SDK。

该团队已经在 100 多个游戏中实现了游戏内屏幕录制技术，游戏玩家自早期以来已经录制了 5 亿个视频，但现在该团队正在开发一对新功能，他们希望能让更多的移动游戏玩家分享他们的视频。从今天开始，Kamcord 提供了一些工具，让玩家可以在运行中剪辑他们的视频，并在混音中添加他们自己的音轨……如果游戏开发者允许的话。

编辑游戏录像的能力非常简单——你的视频中有实质内容的片段可能总共有 30 或 40 秒，那么为什么要分享整个几分钟的剪辑呢？语音叠加功能似乎是两者中更引人注目的，因为它展示了对最常传播的游戏视频类型的相当牢固的理解。

想法是这样的:一旦该功能被启用，当游戏开始时，你的设备就会启动麦克风，并记录下你疯狂触摸触摸屏时发出的狂热低语。起初，这似乎是一种有问题的方式——我本以为游戏的声音会淹没麦克风的任何输入——但 Kamcord 首席执行官 Matt Zitzmann 指出，这明显缺乏回声或音频问题(尽管他仍然认为用户应该使用单独的麦克风)。

但是为什么要走这条路呢？快速浏览一下 YouTube 和 Twitch 上的游戏视频就会发现，它们中的许多都依赖于旁白——毕竟，在观看直截了当、未经删改的游戏录像时，只能有这么多娱乐。当人们沉浸在游戏体验中时，倾听他们的声音是一件非常有吸引力的事情，这也许解释了为什么[让我们玩](https://web.archive.org/web/20221205104227/http://www.youtube.com/results?search_query=let%27s+play&oq=let%27s+play&gs_l=youtube.3..35i39l2j0i3j0l5j0i3j0.616.1469.0.1671.10.8.0.1.1.1.148.579.7j1.8.0...0.0...1ac.1.11.youtube.EZq1QZwXiAY)视频类型如此受欢迎。

“它们只是更具观赏性，”齐兹曼在一次电话交谈中指出。如果 Kamcord 希望开发者认真对待 SDK，将其作为一种潜在的营销工具，那么在游戏录制中添加声音轨道正是 Kamcord 所需要的那种人类品质。对于一个潜在玩家来说，有什么比看着(听着)某人玩得很开心更好的方式来决定他们还没有尝试过的游戏呢？最终由开发人员决定他们是否希望启用其中任何一个功能，但该团队已经与一些感兴趣的方面进行了会谈，并正在慢慢增加人员以应对更多的挑战。
