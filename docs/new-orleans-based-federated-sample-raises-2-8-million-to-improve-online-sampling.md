# 总部位于新奥尔良的 Federated Sample 筹集了 280 万美元用于改进在线采样

总部位于新奥尔良的市场研究技术公司 Federated Sample 今天宣布，它已经筹集了 280 万美元的 A 轮融资。这笔投资由 [Sopris Capital](https://web.archive.org/web/20230204112545/http://www.sopriscapital.com/) 和新奥尔良早期风险投资公司 [Voodoo Ventures](https://web.archive.org/web/20230204112545/http://voodooventures.com/) 牵头。作为该轮融资的一部分，Sopris Capital 创始人兼合伙人乔恩·凯登将加入这家初创公司的董事会。

那么联邦样本是做什么的呢？简而言之，该公司专注于提高在线采样的效率和自动化程度。其旗舰产品 [Fulcrum](https://web.archive.org/web/20230204112545/http://www.federatedsample.com/products.php) 是一个企业级样本管理平台和白标路由解决方案，旨在改善调查体验并降低企业成本。

由于路由的使用越来越多，Federated Sample 向希望使用专有面板来减少对第三方样本提供商依赖的客户提供了优化引擎。该小组生成的每一个完整信息都消除了这些企业购买外部样品的需要，增加了用户的利润并提高了送达率。

Fulcrum 使用一套专有的 API 来连接面板公司和数据收集软件，以实现样品交付过程的自动化。当然，当一家公司购买市场研究调查时，他们希望知道受访者是他们所说的人——这是整个行业面临的最大挑战之一。因此，Federated Sample 正在从公共记录中开发实时验证，而不必收集“个人身份信息”。

Voodoo Ventures 的创始人克里斯·舒尔茨(Chris Schultz)表示，他对 Federated Sample 的使命感到兴奋，因为在创建一个买卖调查的有效市场的过程中，这家初创公司正在为市场研究做着与 10 年前大型广告网络为在线广告所做的相同的事情——为市场带来广泛的分布和效率。

此外，Federated Sample 已经将前五大市场研究公司中的三家列为客户，并看到了强劲的采用，每月有 350 万受访者，本季度收入同比增长 200%。
