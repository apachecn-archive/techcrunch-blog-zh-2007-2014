# WWWTWITTER.com:有史以来最好的网站

对于网站 URL，最常见的错别字之一是省略“.”在“www”和站点域之间。人们通常手动输入 URL 的大型网站通常都很聪明。比如，[wwwgoogle.com](https://web.archive.org/web/20230203054414/http://wwwyahoo.com/)指向谷歌，[wwwyahoo.com](https://web.archive.org/web/20230203054414/http://wwwyahoo.com/)指向雅虎，[wwwmicrosoft.com](wwwmicrosoft.com)指向必应(虽然有意思的是，微软并不拥有[wwwbing.com](https://web.archive.org/web/20230203054414/http://wwwbing.com/)，那是个蹲坑者)。今天有人给我们指出了其中一个伟大的例子:wwwtwitter.com。

去吧，现在就停止阅读这个[去访问它](https://web.archive.org/web/20230203054414/http://wwwtwitter.com/)。反正你很快就会回来。为什么？因为是的，wwwtwitter.com 是最伟大的重定向:它重定向到 TechCrunch。

不，我们不拥有它，也不知道谁拥有它。注册设置为 private，但是重定向是通过 GoDaddy 完成的。它于 2007 年 11 月注册，直到 2014 年才会更新。所以，不管是谁干的，小心点。
