# 离岸 3D 打印枪蓝图保护者金·多特康姆据报道删除文件[更新]

据报道，对于臭名昭著的自由信息黑客[金·多特康姆](https://web.archive.org/web/20221225121225/http://en.wikipedia.org/wiki/Kim_Dotcom)来说，3D 打印枪支甚至太可怕了。在美国国务院要求世界上第一支完全可打印枪支的设计者[从他的网络](https://web.archive.org/web/20221225121225/https://techcrunch.com/2013/05/09/state-department-demands-3-d-gun-blueprints-be-removed/)中删除这些文件后，总部位于新西兰的 Dotcom 承诺将这些文件安全地保存在他的离岸合法避风港。

“我认为这是对社区安全的严重威胁。我觉得人能打印出连金属探测器都探测不到的 3D 枪，这很可怕。根据一位发言人通过电子邮件发给我们的声明，Dotcom 说。

[据新西兰新闻谈话 ZB 网站](https://web.archive.org/web/20221225121225/http://www.newstalkzb.co.nz/auckland/news/nbnat/749825209-dotcom-orders-deletion-of-3d-gun-design)报道，“这些计划可以在 Dotcom 的 Mega 网站上获得，但这位新西兰企业家要求他的员工删除公开文件。Dotcom 说，他认为他们是对社区安全的严重威胁。

我们已经联系了 Dotcom，并将在收到更多信息时向读者更新。

上周，得克萨斯州法学院毕业生科迪·威尔森因免费分发用 3D 打印机制造致命武器的数字蓝图而成为全球头条新闻。仅在一周内，参议员查克·舒默[呼吁](https://web.archive.org/web/20221225121225/https://www.google.com/search?q=cody+wilson+law+school&oq=cody+wilson+law&aqs=chrome.1.57j0l3.3038j0&sourceid=chrome&ie=UTF-8)立即进行监管，而蓝图本身[已经被下载了超过 10 万次](https://web.archive.org/web/20221225121225/http://www.washingtonpost.com/blogs/innovations/wp/2013/05/09/plans-for-3d-printed-gun-downloaded-100000-times-state-department-in-contact-with-defense-distributed/)。

由于网络公司承诺保护它们免受美国的干涉，是否有任何政府实体能够阻止它们的传播是值得怀疑的。Dotcom 是一名企业家和黑客，他因警方大规模搜查他存放盗版娱乐内容的 Megaupload 网站而出名。在争取激进的公开信息的斗争中，他似乎是一个天然的盟友。

现在，即使他放弃了 3D 武器，也许有一些信息，互联网和政府可以合作拒绝。
