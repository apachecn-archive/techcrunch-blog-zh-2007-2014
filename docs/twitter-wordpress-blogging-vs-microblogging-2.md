# 博客与微博:Twitter 的全球增长趋于平缓，而 WordPress 的增长有所加快

仅在一年前，传统智慧认为博客已死，微博将很快取代它们。Twitter 本应扼杀博客，因为一次发布一个句子片段要比将整个想法汇集成业内所谓的“段落”简单得多。

今天，博客做得很好，而 Twitter 正在努力保持平稳增长，至少对其网站 Twitter.com 来说是这样(Seesmic 和 TweetDeck 等客户已经看到[没有放缓](https://web.archive.org/web/20230216131519/http://www.loiclemeur.com/english/2009/11/twittercom-traffic-down-seesmiccom-is-30month.html))。自去年夏天以来，Twitter 在美国经历的疲软终于也影响到了其全球访客的增长。

10 月份，comScore 估计 Twitter 在全球拥有 5830 万独立访客，低于 9 月份的 5840 万。与此同时，WordPress.com 增加了 1000 万独立访客，月底达到 1.518 亿——这是自 2009 年 3 月以来几乎没有任何进展的。

当然，我在这里使用 WordPress.com 作为所有博客的代理(我也可以很容易地使用 Blogger，它实际上在全球有 2.917 亿访问者。自 3 月份以来，Blogger 看到了类似的持有模式，10 月份的访问量突然激增 1820 万

那么，当微博走下坡路的时候，博客是不是又回来了？博客的受欢迎程度在一个月内飙升并不能说明什么，但无论如何，这都是一个错误的问题。博客从未真正消失过，事实上它得到了 Twitter 的帮助，Twitter 正成为许多人的首选阅读器(感谢像 Twitterfeed 这样的服务)。

先别把微博排除在外。Twitter 终于推出了对其网站的改进，如[列表](https://web.archive.org/web/20230216131519/https://techcrunch.com/2009/10/30/check-it-twice-twitter-lists-now-open-to-all-users/)和[新的转发按钮。](https://web.archive.org/web/20230216131519/https://techcrunch.com/2009/11/10/hate-it-or-love-it-twitters-new-retweet-style-rolling-out/)一旦[地理位置](https://web.archive.org/web/20230216131519/https://techcrunch.com/2009/11/19/twitter-location-api/)功能投入使用，Twitter 的增长可能会卷土重来。
