# LookSmart 收购的 Edgeio 资产

[](https://web.archive.org/web/20230218162009/http://www.crunchbase.com/company/edgeio)Edgeio 资产的[拍卖完成，](https://web.archive.org/web/20230218162009/https://techcrunch.com/2007/12/11/edgeio-bids-start-at-250000/) [Looksmart](https://web.archive.org/web/20230218162009/http://www.looksmart.com/) 为中标者。他们以 28 万美元收购了公司的大部分资产。

今早拍卖结束后，我与 LookSmart 的企业发展总监帕特里克·查普曼(Patrick Chapman)进行了简短的交谈。他说，现在说他们将如何处理这些资产还为时过早，但他们可能会很快发布新闻稿。查普曼说，他将与 Edgeio 的前员工交谈，并希望为他们提供一些或全部 LookSmart 的工作。

LookSmart 是一家小型上市公司，市值约为 6800 万美元。他们最近采取了一些措施，将业务恢复到 90 年代的水平，包括最近的股票回购公告。

这标志着 Edgeio 这个阶段的结束，Edgeio 是我和 Keith Teare 在 2005 年共同创立的一家公司，我们在本月早些时候把它放入了死亡池。但我很高兴看到资产转移到一家拥有推动想法前进的资源的公司。
