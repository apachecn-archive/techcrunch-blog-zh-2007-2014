# 精益创业思想家 Eric Ries:“不要急于做大，要急于拥有一个伟大的产品”

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/14/lean-startup-eric-ries-tctv/>

# 精益创业思想家埃里克·里斯:“不要急于做大，要急于拥有一个伟大的产品”

每个科技创始人都认为自己经营着一家精益创业公司。但是这到底是什么意思呢？在上面[的视频采访中，Eric Ries](https://web.archive.org/web/20230205035746/http://www.crunchbase.com/person/eric-ries) 提出了术语[精益创业](https://web.archive.org/web/20230205035746/http://theleanstartup.com/)，并且是即将在[出版的同名书籍](https://web.archive.org/web/20230205035746/http://www.amazon.com/dp/0307887898?tag=lessolearn01-20&camp=213381&creative=390973&linkCode=as4&creativeASIN=0307887898&adid=004DZWTQ0HQTRCNYZJPD)的作者，他解释了他的理论，即在将产品展示给更广泛的受众之前，向少量早期客户学习以改进和完善你的产品。“不要急于做大，”他在我们采访的最后总结道，“要急于拥有一个伟大的产品。”

里斯认为，企业家最大的优势是他们默默无闻。如果你的第一个产品很烂，至少不会有太多人知道。但那是犯错误的最好时机，只要你从中吸取教训，让产品变得更好。“第一个产品在某些方面不可避免地会变坏，”他说。精益创业方法是一种系统测试公司产品创意的方式。

早点失败，经常失败。“我们的目标是尽快学会，”他说。他还认为，创业公司应该悄悄地推出他们的产品，直到他们弄清楚客户真正想要的是什么。我们讨论了 Color 如何使用这个建议。Ries 还给我上了一堂速成课，讲授一些越来越流行的精益创业术语，比如“最小可行产品”、“产品/市场匹配”或“构建-测量-学习反馈循环”。

但是你能在泡沫中建立一个精益创业公司吗(如果我们还在泡沫中的话)？“这甚至更重要，”他说，当每个人都在向“成功剧场”砸钱，炫耀虚荣指标时，变得精益。“虚荣指标是你想在 TechCrunch 上发布的数字，让你的竞争对手感觉不好，”他说。不要相信你自己的宣传，或者用那些指标来经营你的公司。可操作的指标实际上能让你理解是什么推动了你的成长，这更为重要。

Ries 在[创业经验](https://web.archive.org/web/20230205035746/http://www.startuplessonslearned.com/)上写了关于精益创业的文章，并将在 [Disrupt SF](https://web.archive.org/web/20230205035746/https://techcrunch.com/2011/08/05/elon-musk-dustin-moskovitz-and-eric-ries-are-ready-to-shake-things-up-at-disrupt-sf/) 上发表演讲。
