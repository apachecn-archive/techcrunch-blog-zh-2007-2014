# 面对脸书的时间表，Memolane 增加了社交功能和来自数字车库的投资

在这个时间线的时代，很容易忘记脸书并没有提出记录你生活的社会时间线的概念。但是，如果你是一家像 [Memolane](https://web.archive.org/web/20230205033056/http://memolane.com/) 围绕时间轴建立[的初创公司，从网络上引入人们的推文、Instagram 照片、Last.fm 歌曲和许多其他社交媒体，你现在会怎么做？嗯，你变得更社会化了。](https://web.archive.org/web/20230205033056/https://techcrunch.com/2011/03/08/memolane-launches-its-personalized-internet-time-machine-to-the-public/)

Memolane 今天推出了一个重要的服务更新，增加了一个跟随模型。“当我们创建 Memolane 时，人们说它很棒，但它就像一个博物馆，”创始人 Eric Lagier 说。"我们希望它成为一个游乐园，而不是博物馆。"

现在不只是看到你的孤巷，你还可以添加好友，看到他们的巷，甚至为他们做贡献。您可以基于主题、标签或事件(带有开始和结束日期)创建多个通道。该网站现在也针对触摸屏平板电脑(iPad 和 Android)进行了优化

该公司增加了一个新的投资者，日本的[数字车库](https://web.archive.org/web/20230205033056/http://www.crunchbase.com/company/digital-garage)，它加入了现有投资者 August Capital 和 Atomico(去年投资了[200 万美元](https://web.archive.org/web/20230205033056/https://techcrunch.com/2010/09/13/memolane-raises-2-million-to-become-your-digital-memory/))。这笔投资相对较小，被认为是去年 A 轮投资的延续。日本是 Memolane 的第二大市场，仅次于美国，拥有 17%的用户。
