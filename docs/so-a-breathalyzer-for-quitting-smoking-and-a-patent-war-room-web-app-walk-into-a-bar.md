# 所以一个戒烟酒精测试仪和一个专利作战室网络应用走进酒吧...

> 原文：<https://web.archive.org/web/http://techcrunch.com/2014/03/23/so-a-breathalyzer-for-quitting-smoking-and-a-patent-war-room-web-app-walk-into-a-bar/>

现在，你们期待已久的时刻到了。

很快(取决于你的阅读理解水平)，我将揭晓 TechCrunch DC 和 NYC Meetup 的获胜者，你们都假装已经知道他们是谁。

当然，你不应该知道他们是谁。作为一项要求，他们大多都处于秘密或私人测试阶段，或通常在雷达下。但不再是了。

请击鼓！

在 DC 获得第一名，用一个“作战室”webapp 来帮助初创公司、律师和专利钓鱼者导航专利纠纷的陷阱，同时保持有组织、沟通，并最终节省时间和金钱，为岩石圈软件和他们的产品 PatDek 放弃吧！我们的第一名获奖者将获得两张 5 月份 Disrupt NY 的门票，以及 Disrupt Startup Alley 的演示桌。

测试和监控 API 的 API Fortress 在 5 月份获得了第二名和两张扰乱纽约的门票，而城市设计的“Houzz”Greater Places 获得了一张扰乱纽约的门票的第三名。观众选择奖的获胜者也收到了一张破坏票，是 SpeakerBlast，它使用一个 webapp 将任何数量的设备变成一个拥有“优于蓝牙的技术和质量”的声音系统。

现在对于纽约来说…

IntelliQuit 是一家制造帮助吸烟者戒烟的袖珍型呼吸器的公司，该公司在纽约赢得了第一名，并将获得一张创业小巷演示台和两张扰乱门票。嘿！如果观众在 Disrupt 喜欢他们，他们可能有机会在舞台上进入战场。

为大学生设计的七层一次性防水床单 AFreSheet 获得了第二名，一家名为 Project Shoe 的女鞋定制网站获得了第三名。我们的观众选择的是 ShopDrop，这是一个通知你最喜欢的商店、在线商店和商店销售情况的应用程序。

感谢每一个人给了我一个美妙的夜晚，也祝贺那些在见面会上发言的公司！你们做得很好。

我们的下一次聚会是在 4 月 8 日和 4 月 10 日，分别在波士顿的 T1 和 T2 的 T3。我们等不及了！
