# 由于专注于内容标识和货币化，ZEFR 签约了 UMG、索尼音乐、Ultra Music 和 NASCAR

自从 [Movieclips 更名为 ZEFR](https://web.archive.org/web/20230205141708/https://techcrunch.com/2012/08/15/movieclips-18-5m-funding-rebrand-zefr/) 以来，已经有大约九个月的时间了，它向在 YouTube 上分发或声称其内容的出版商提供内容 ID 和货币化工具。今天，这家初创公司宣布了它的第一批大客户，签约了 UMG、索尼音乐、Ultra Music 和 NASCAR 等大牌音乐公司。

对于那些已经忘记的人来说，ZEFR 的故事是这样的:从前，有一个叫做 Movieclips 的小初创公司。这家总部位于洛杉矶的初创公司成立的前提是许可电影剪辑，并通过 YouTube 提供这些剪辑，允许电影工作室将大量锁在金库里的长尾内容货币化。

但这还不是全部！Movieclips 还开发了工具来帮助识别粉丝发布的剪辑并将其货币化。它还确定了最佳实践，创造了让观众参与进来的技术——比如一面展示多个相关场景的后卷电视墙，观众可以点击进入。这是一项令人敬畏的技术应用，但电影剪辑公司意识到，它可以使用同样的工具来帮助其他领域的内容提供商——如音乐和体育——从他们制作的内容中获利。

就这样 ZEFR 诞生了。ZEFR 是 Movieclips 的更名，因为这家初创公司筹集了更多资金，并开始向其他人提供内容 ID 和货币化工具。

现在，在完善其技术并向各种垂直行业的内容所有者推销后，它设法让许多主要的第三方内容提供商加入进来。ZEFR 不仅可以找到用户发布的视频，帮助内容所有者赚钱，还可以帮助他们充分利用自己的频道。

对于像纳斯卡这样的客户来说，这意味着使用一些 ZEFR 自己开发的渠道管理工具。对于其电影剪辑业务，这家初创公司使用这些工具来确保最佳实践和改善搜索引擎结果，但像后卷视频墙这样的东西也可以让观众关注其客户的内容。对于唱片公司来说，与 ZEFR 合作使他们能够让超级粉丝创作并上传他们喜欢的艺术家的视频，同时也确保了唱片公司(和艺术家)从这些声称的视频中获得广告收入。

自 2009 年成立以来，ZEFR 已经筹集了近 3000 万美元，其中包括由美国风险投资伙伴牵头的一轮 1850 万美元的融资。其他投资者包括 MK Capital、Shasta Ventures、SoftTech VC、First Round Capital、Richmond Park Partners 和 Machinima 首席执行官 Allen DeBevoise。
