# 来源:Twitter 的广告平台今晚发布

Twitter 的期待已久的新广告平台将于今晚发布，我们从一位知情人士处获悉了该项目的消息。模特？这非常像昨晚刚刚发布的 [Tweetup 产品](https://web.archive.org/web/20221208002318/https://beta.techcrunch.com/2010/04/11/twitter-idealab-tweetup-adsense-bill-gross/)——想想 Google Adwords 用于 Twitter 搜索和标签，广告商可以对单个关键词进行竞价。**更新**:多份报告证实，Twitter 的广告平台即将推出——详情请看这里。

我们的消息来源称，赞助的(或付费的)Twitter 消息将出现在正常搜索结果之上。可能还有“Adsense”联合形式，以及第三方，如 Seesmic、TweetDeck 等。可以从中分一杯羹。

我们没有被告知新的平台，但我们听说其他人，可能会有更多的细节很快出来。该产品将于周三在 Twitter 的 Chirp 会议上正式亮相。

大约一年前，我们说是时候开始考虑将 Twitter 作为一个搜索引擎，利用搜索盈利机制。这就是 Twitter 押注的盈利模式。
