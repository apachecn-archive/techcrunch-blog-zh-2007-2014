# 日本的 Line 拥有 2 亿注册用户，但仍未公布每月活跃用户数

线继续其明显的增长撕裂。在日本本土市场非常受欢迎的移动通讯应用[宣布](https://web.archive.org/web/20230117202749/http://en.lineblog.naver.jp/archives/30767259.html#more)在推出约两年后，其全球注册用户已突破 2 亿大关。[Line 1 月突破 1 亿用户](https://web.archive.org/web/20230117202749/https://techcrunch.com/2013/01/18/line-100m-users/)。

应该指出的是，2 亿的数字不是月活跃用户，而是指~~应用下载~~注册用户 Line 的 iPhone/Android/Windows Phone/黑莓/诺基亚 Asha/功能手机应用的总和。**更新:**这意味着，如果一些用户在不同的移动设备上使用 Line 的服务，Line 实际上会在注册用户数据中多次计入这些用户。因此，2 亿人的数字可能有所夸大——尽管不可能说出夸大了多少。

Line 的用户帐户帮助部分说明:

> 每个手机号码/电子邮件地址只能验证**一部智能手机。**
> 
> 要在不同的设备上注册 LINE，请确保您为每台设备准备了另一个电话号码或电子邮件地址。
> ★如果您使用相同的电子邮件地址/电话号码从另一台设备访问相同的 LINE 帐户，您的 LINE 帐户将从之前的设备中自动删除。

该公司也没有公布每月发送/接收的消息，这是判断消息应用参与度的另一个有用指标。但是 Line 之前告诉 TechCrunch,活跃用户与注册用户的比例高达 80%。活跃高峰可能会出现在 Line 的亚洲据点市场，如日本，而不是它后来扩展到的国际市场，包括美国和欧洲。

Line 与一系列移动通讯应用竞争——包括老牌的 WhatsApp，该公司上个月宣布其月活跃用户已达 2 . 5 亿。虽然相对较新，中国的微信，现在已经拥有大约 3 亿注册用户群(T3)，其中 1.9 亿是月活跃用户。如果 Line 继续回避报告其每月活跃用户，很难避免得出这样的结论:它正在努力赶上其最大竞争对手的参与水平。

Line Corporation 确实公布了应用内购买的收入，比如贴纸包。在今年早些时候的 [Q1 收入报告中](https://web.archive.org/web/20230117202749/https://techcrunch.com/2013/05/09/line-reports-q1-2013-earnings-of-58-9m-half-from-game-in-app-purchases-30-from-stickers-80-from-japan/) Line 报告了 5890 万美元的收入，一半来自应用内游戏购买，不到三分之一来自贴纸销售。然而，这些收入的大部分(80%)来自日本。

Line 的核心贴纸人物是典型的卡哇伊，但它继续试图本地化吉祥物以扩大其吸引力——包括为印度尼西亚用户制作斋月友好的贴纸包，在下面的广告中进行宣传。
