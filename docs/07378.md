# 摩托罗拉继续在 Droid 3 上执行锁定引导程序的政策

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/07/08/motorola-continues-locked-bootloader-policy-on-droid-3/amp/>

 <amp-img src="https://web.archive.org/web/20230123232858im_/https://techcrunch.com/wp-content/uploads/2011/07/android_locked_bootloader.jpg" alt="" title="android_locked_bootloader" class="aligncenter size-full wp-image-48597 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 

听着，黑客们。这是给你的。

摩托罗拉的 Droid 3 终于发布了，它的所有技术规格都很荣耀，但有一样东西不见了:一个解锁的引导程序。没错，摩托罗拉又做了一次，把 Droid 3 的主人锁在一个安卓笼子里，就像 Atrix 的主人、Droid 2 的主人一样，这样的例子不胜枚举。

幸运的是，一个解锁的引导程序正在路上。“正如我们已经沟通的那样，我们计划在未来的软件版本中启用可解锁/可重新锁定的引导加载程序，从 2011 年底开始，届时渠道和运营商合作伙伴将允许它，”摩托罗拉发言人对 [Wired](https://web.archive.org/web/20230123232858/http://www.wired.com/gadgetlab/2011/07/motorola-droid-3-hack/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+wired%2Findex+%28Wired%3A+Index+3+%28Top+Stories+2%29%29&utm_content=Google+Reader) 说。摩托罗拉基本上已经明确表示，一旦你解锁了引导程序，你的保修期就结束了，因为许多“新手”都有这种讨厌的习惯*不遵循指示*并阻塞他们的手机。“如果你把手机弄坏了，我们不想在保修期内修理它，”一名摩托罗拉代表在博客中写道。

显然，摩托罗拉在这个引导加载器政策上走得很紧。像索尼爱立信这样的竞争对手已经开始培养与他们各自的改装社区的关系，提供如何解锁手机的教程视频。与此同时，摩托罗拉正在为 Droid X 这样的手机添加像 [eFuse 这样的东西，这是一种反篡改对策系统，专门用于在你选择 mod beyond stock 软件](https://web.archive.org/web/20230123232858/http://www.mobilecrunch.com/2010/07/14/droid-x-actually-self-destructs-if-you-try-to-mod-it/)时阻止你的手机

简而言之，消费者已经受够了。特别是一个人，在摩托罗拉创建了一个[网页，致力于结束引导程序政策，要求其他开发者签署请愿书。截至目前，已有超过 1 万人签名。虽然许多人只是留下姓名以示抗议，但一些 Moto 客户实际上已经花时间写下了他们的请求。](https://web.archive.org/web/20230123232858/http://www.groubal.com/motorola-lockedencrypted-bootloader-policy/#signatures)

这些是我们发现特别令人心酸的几个:

> “拜托，摩托罗拉，我们热爱手机，但让我们充分发挥这项新技术的潜力。在这个智能手机的新世界里，我们是你的 beta 测试者！但是我们为实验性的、开放开发的东西支付额外的费用，我们用你今天给我们的工具创造东西，换来一套更好的工具，供你明天再次出售。也许默认加密引导程序对你的大众市场来说是个好主意，但是我认为你应该给我们解密的选择，因为开放是 Android 的全部。这就是为什么我买了我的第一个机器人(Atrix)，我必须说，在这一点上，我希望我先研究一下这个常见的投诉，因为我会买其他真正的“机器人”。“所以，平心而论，我认为你要么让我们选择使用和开发我们需要的设备，要么至少给我们退款，并且在未来，在你的广告中使用免责声明。”布莱恩

> “我的钱意味着这是我的手机。据推测，锁定的引导装载程序是为了保护终端用户不损坏手机；然而，他们声称要保护的公众可能不会受到摩托罗拉决定的影响。给那些想花时间修改手机的人这个机会。如果我想被困在一个有围墙的花园里，我就不会买一个所谓的“开放的”Android 操作系统。史蒂文·巴纳班
