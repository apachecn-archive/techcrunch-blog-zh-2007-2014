# 媒体在 LeWeb 为维基解密辩护:“泄密者会赢”

![](img/4205772fc7efe06900fd5447882ab0c6.png "g2")今天在法国巴黎举行的[LeWeb’10](https://web.archive.org/web/20230202221005/http://www.leweb.net/)媒体座谈会上，每个人都在想一件事:维基解密。

Julio Alonso 说:“*这是互联网的一个转折点——这不再仅仅是关于维基解密*,[Weblogs SL 的](https://web.archive.org/web/20230202221005/http://www.weblogssl.com/)。他警告说:“*维基解密发生的事情将会在以后*应用到其他人身上。”。

Pierre Chappaz 补充道:“这是全球所有政府对互联网审查的首次尝试。他补充道:“尽管受到了所有的攻击，但我乐观地认为这些信息会保存下来。”。

当主持人阿德里安·蒙克(Adrian Monck)问及这是否会给美国带来特别的阴影时， [Techmeme 的](https://web.archive.org/web/20230202221005/http://techmeme.com/)加布·里维拉(Gabe Rivera)表示，他认为在某些方面已经有了阴影。里维拉指出，在某种程度上，LeWeb 上人群的语气就证明了这一点。他还特别提到了美国参议员约瑟夫·利伯曼(Joseph Lieberman ),暗示《纽约时报》可能成为目标，因为他们公布了一些电报。他说:“(T17)这强调了维基解密和《纽约时报》的所作所为没有本质上的区别。”。里维拉说，这是值得关注的事情。

“*我们已经把互联网审查的理论推到了最边缘*,[华尔街日报欧洲公司的](https://web.archive.org/web/20230202221005/http://europe.wsj.com/home-page) Ben Rooney 补充道。

查帕兹说:“我们必须谈论正在发生的事情。我对传统媒体的沉默感到惊讶。这是一次系统性的攻击。我们必须向传统媒体解释。利害攸关的是新闻自由，”他继续说道。

阿隆索同意了。“*第一道防线就是谈论它*，”他说。

里维拉补充说，泄密者不会输掉这场战争。他引用了镜像网站、Twitter 账户、脸书消息和各种各样不断出现的东西来继续数据传播。“泄密者将赢得胜利，”他说。

该小组似乎相当确信，即使 P2P 网络必须取代 DNS，信息最终确实会胜出。

”*互联网太强大了。他们将很难把它控制住。*
