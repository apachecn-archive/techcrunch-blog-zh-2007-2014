# 电子商务平台 Shopify 收购移动应用开发工作室 Select Start

提供让任何人都能在线创建店面的[交钥匙电子商务](https://web.archive.org/web/20230404233452/https://techcrunch.com/2006/01/04/shopping-20/)技术的 Shopify 收购了移动应用和游戏工作室 [Select Start Studios。](https://web.archive.org/web/20230404233452/http://www.selectstartstudios.com/)

此次收购将用于推进 Shopify 的移动商务计划。虽然所有 Shopify 商店都已经包括移动优化的店面和收银台，但收购 S3 将为 Shopify 团队带来 20 多名新的移动工程师，继续为商家开发移动产品。

Shopify 的[基于网络的平台](https://web.archive.org/web/20230404233452/https://techcrunch.com/2006/04/15/shopify-was-worth-the-wait/)允许任何人在瞬间建立一个商店，添加要出售的物品，上传图片，添加标签和分组物品，并集成 PayPal 或其他信用卡处理器进行支付。店面和购物车可以定制，该平台确保所有交易的安全性。如今，这个电子商务平台在 80 个不同的国家拥有超过 20，000 家活跃的在线商店。

例如，愤怒的小鸟使用 Shopify 的平台进行电子商务销售(该公司表示，愤怒的小鸟在线商店的相当大一部分销售是通过移动设备进行的)。Shopify 最近筹集了 1500 万美元的新资金。
