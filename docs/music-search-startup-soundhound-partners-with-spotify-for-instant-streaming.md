# 音乐搜索初创公司 SoundHound 与 Spotify 合作进行即时流媒体播放

音乐搜索和发现公司 [SoundHound](https://web.archive.org/web/20230203180010/http://www.crunchbase.com/company/soundhound) 已经[与数字音乐服务提供商](https://web.archive.org/web/20230203180010/http://www.businesswire.com/news/home/20110825005404/en/SoundHound-Spotify-Announce-Partnership) [Spotify](https://web.archive.org/web/20230203180010/http://www.crunchbase.com/company/spotify) 达成协议，使欧洲的 SoundHound 用户能够立即访问后者的目录(超过 1500 万首歌曲)。从声音上看，这种合作关系还不能让欧洲以外的用户跨越那座特殊的桥梁。

从今天开始，iOS 和 Android 设备上同时订阅 Spotify Premium 的 SoundHound 用户在通过音乐发现和识别服务识别歌曲后，将可以选择“立即在 Spotify 中播放”。这适用于免费和付费 SoundHound 应用程序的用户。

没有订阅 Spotify Premium 的欧洲 SoundHound 用户将被重定向到一个页面，邀请他们成为 Spotify 用户，然后才能继续收听歌曲。

SoundHound 的一个直接竞争对手是 Shazam，sound hound 的“Sound2Sound”技术，这种技术绕过了传统的声音到文本的转换技术。

SoundHound 总部位于加利福尼亚州圣何塞，由全球催化剂合作伙伴、TransLink Capital、Walden Venture Capital 和其他硅谷投资者提供支持。
