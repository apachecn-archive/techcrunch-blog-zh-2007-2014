# 微软和诺基亚为 Symbian 开发者发布 Windows Phone 移植指南 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/09/21/microsoft-and-nokia-release-porting-guides-for-symbian-developers-moving-to-windows-phone/>

微软和诺基亚合作开发了一套新工具，旨在让诺基亚开发者为过渡到 Windows Phone 做好准备。[今天宣布的](https://web.archive.org/web/20230204234757/http://windowsteamblog.com/windows_phone/b/wpdev/archive/2011/09/21/nokia-developers-learn-windows-phone-even-faster.aspx)是三个联合开发的工具和指南，最值得注意的是将 Symbian Qt 添加到 Windows Phone API 映射工具中。

已经支持 [iOS](https://web.archive.org/web/20230204234757/http://windowsteamblog.com/windows_phone/b/wpdev/archive/2011/04/29/leveraging-your-iphone-development-expertise-to-build-windows-phone-7-applications.aspx) 和 [Android](https://web.archive.org/web/20230204234757/http://windowsteamblog.com/windows_phone/b/wpdev/archive/2011/06/09/leveraging-your-android-development-expertise-to-build-windows-phone-applications.aspx) 的 [Windows Phone 映射工具](https://web.archive.org/web/20230204234757/http://wp7mapping.interoperabilitybridges.com/Home/Library?source=Qt)，充当 Windows Phone 平台和其他移动操作系统之间的翻译字典。有了它，开发人员可以在他们的应用程序中挑选出 API 调用，然后在 Windows Phone 中查找等效的类、方法和通知事件。

需要说明的是，这不是一个直接的移植工具——它不会为您做任何工作。开发人员可以简单地参考该指南来帮助移植他们的应用程序。

映射工具中包括用于 Symbian 的 Qt 4.7 的核心库(QtCore、QtGui、QtLocation、QtNetwork、QtSensors、QtSql、QtXml、QtWebKit、QML 元素和 QML 组件)。还提供了示例代码和教程。

微软和诺基亚也发布了一份 100 页的白皮书，作为有兴趣了解 Windows Phone 的 Symbian Qt 应用程序开发者的实际指南。在其 8 章中，两家公司详细介绍了 Windows Phone 指南、C#编程、应用程序生命周期、移植程序等。

今天也预示着一系列名为“诺基亚 Windows Phone 培训”路演的开发者活动的开始，第一场活动将在巴黎举行。这些活动将在接下来的几天内在欧洲和澳大利亚各地举行:[意大利米兰(9 月 26 日)](https://web.archive.org/web/20230204234757/http://www.cvent.com/d/jcq7j6)，[西班牙马德里(9 月 29 日)](https://web.archive.org/web/20230204234757/http://www.cvent.com/d/mcq7jy?lang=es)，[德国柏林(10 月 4 日)](https://web.archive.org/web/20230204234757/http://www.cvent.com/d/scq7jh?lang=de)，[英国伦敦(10 月 10 日)](https://web.archive.org/web/20230204234757/http://www.cvent.com/d/0cq7tk)，悉尼(9 月 24 日至 25 日【售罄】，【T10 月 8 日至 9 日 )，墨尔本([10 月 8 日至 9 日【售罄，等候名单】)和布里斯班](https://web.archive.org/web/20230204234757/http://blogs.msdn.com/b/dglover/archive/2011/09/19/windows-phone-developer-workshop-melbourne-oct-8th-and-9th-2011.aspx)

美国的日期，包括硅谷的日期，还没有公布。

Symbian 还不是一个死亡的操作系统——毕竟，它上个月刚刚接受了一次大的更新。但是诺基亚已经公开表示计划逐步淘汰它。Windows Phone 是诺基亚的下一步，看到这些工具和指南在第一款诺基亚 Windows Phone 发布之前发布并不奇怪，预计将于年底发布。
