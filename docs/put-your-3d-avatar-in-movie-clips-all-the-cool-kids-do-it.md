# 将您的 3D 头像插入电影剪辑。所有酷小孩都这么做。

[http://www.gizmoz.com//newsite/widgets/movies/movies.swf](https://web.archive.org/web/20221001174319/http://www.gizmoz.com//newsite/widgets/movies/movies.swf)
Gizmoz.com 自己搞定

如果你喜欢用 JibJab 将自己投入到电影片段中，你也会喜欢 Gizmoz 的新产品。

与 JibJab 不同，JibJab 拍摄静止的 2D 图像，仅上下移动嘴巴，Gizmoz 开发了专有技术来创建 3D 面部图像，包括各种表情变化和嘴唇同步。

他们的基本产品，允许用户创建会说话的化身，[在 2006 年 5 月推出了](https://web.archive.org/web/20221001174319/http://www.beta.techcrunch.com/2007/05/31/gizmoz-launches-create-your-talking-3d-avatar/)。

用户现在可以把他们的头像图像插入到许多视频剪辑中。随着时间的推移，将会添加更多的剪辑，Gizmoz 也正在授权好莱坞电影和其他专业内容的视频。创始人 Eyal Gever 一年前给我看了一个片段，我的脸出现在《皇家赌场》的詹姆斯·邦德片段上。那天我觉得很酷。

Gizmoz 的总部设在以色列，它从基准融资 630 万美元。
