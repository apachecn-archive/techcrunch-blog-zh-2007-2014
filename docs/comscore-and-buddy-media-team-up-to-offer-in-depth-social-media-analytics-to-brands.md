# comScore 和 Buddy Media 联手为品牌提供深入的社交媒体分析

网络和移动数据测量公司 comScore 正与向品牌提供脸书页面和社交媒体管理工具的公司 Buddy Media 合作。战略合作伙伴关系的基本目标是让品牌和机构在社交分析方面获得更好的数据衡量。

Buddy Media 最著名的是其[多功能](https://web.archive.org/web/20221225082814/https://techcrunch.com/2009/10/20/buddy-media-launches-integrated-facebook-and-twitter-client-for-brand-management/)社交媒体管理系统，帮助创建、管理和跟踪脸书的社交活动。关于脸书，该平台让品牌管理机构创建、管理和跟踪多种语言[的脸书页面](https://web.archive.org/web/20221225082814/https://techcrunch.com/2010/06/21/buddy-medias-facebook-management-platform-for-brands-goes-global/)，以推动和增加用户和品牌的参与度。

使用 Buddy Media 社交营销套件和 comScore 社交测量服务的共同客户将能够更好地量化他们的社交媒体工作的影响，访问关于覆盖范围/频率、人口统计数据、在线行为和竞争品牌基准的深入数据。作为合作伙伴关系的一部分，两家公司同意将 comScore 数据整合到 Buddy Media 产品套件和工作流程中。

这种整合是一种优质产品，目前的 Buddy Media 客户将继续免费获得社交营销套件中的公司内部分析。但显然，comScore analytics 可以提供更深入的信息。

巴迪媒体最近从 GGV 资本、机构风险投资伙伴、Bay Partners 和 Insight Venture Partners 获得了 5400 万美元的新资金。公司[也刚刚宣布了与谷歌的一个试点项目，允许企业和品牌使用伙伴媒体管理他们的 Google+页面。](https://web.archive.org/web/20221225082814/https://techcrunch.com/2011/11/15/through-buddy-media-hootsuite-more-google-pages-get-third-party-management-tools/)
