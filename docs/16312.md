# 微软达成 72 亿美元收购诺基亚设备和服务业务并授权其专利|技术危机

出乎意料的是，微软周一晚上宣布，它已经与诺基亚签署了一项协议，收购“几乎所有的诺基亚设备服务业务，许可诺基亚的专利，许可和使用诺基亚的地图服务。”

该交易的总价格为 54.4 亿欧元现金，目前价值 71.7 亿美元。其中，设备和服务业务收购占 37.9 亿欧元，专利许可交易占剩余的 16.5 亿欧元。

早在 2011 年，诺基亚和微软就签署了合作协议，但这显然是两家公司更为认真的合作。

此外，作为交易的一部分，诺基亚首席执行官史蒂芬·埃洛普将辞去诺基亚总裁兼首席执行官的职务，担任微软“设备服务诺基亚执行副总裁”的新职务埃洛普已经成为传闻中微软掌门人[即将空缺的位置](https://web.archive.org/web/20230316111359/https://techcrunch.com/2013/08/23/steve-ballmer-to-retire-as-microsoft-ceo-within-12-months/)的[头号竞争者](https://web.archive.org/web/20230316111359/http://www.ibtimes.co.uk/articles/503116/20130902/replace-steve-ballmer-microsoft-ceo-stephen-elop.htm)，这可能是一个关于微软执行领导层未来的强烈信号。

在宣布这一消息的新闻稿中，微软首席执行官史蒂夫·鲍尔默说:

> “这是迈向未来的大胆一步——对两家公司的员工、股东和消费者来说都是双赢。将这些伟大的团队聚集在一起将加速微软在手机领域的份额和利润，并加强微软和我们的合作伙伴在我们整个设备和服务家族中的整体机会。除了在各价位手机领域的创新和实力，诺基亚还在硬件设计和工程、供应链和制造管理、硬件销售、营销和分销等关键领域带来了业经验证的能力和人才。”

虽然这看起来像是我们所知的诺基亚的末日，但听起来该公司目前仍计划作为某种独立实体向前发展——尽管战略非常不同。在埃洛普离职后，诺基亚董事会主席兼现任临时首席执行官 Risto Siilasmaa 在一份新闻稿中称，“这是一个重要的重塑时刻，从财务实力的角度来看，我们可以建立我们的下一个篇章……这笔交易为许多诺基亚员工提供了未来的机会，作为一个拥有战略、财务资源和在移动领域取得成功的决心的公司的一部分。”
