# 利比亚域名注册管理机构分享其观点。这很夸张，但问题依然存在

![](img/09ec8600113983bb8a2dcc3a93437ace.png)

你可能已经在本周早些时候读到过，利比亚域名注册机构 [NIC.ly](https://web.archive.org/web/20221209140420/http://nic.ly/) 突然接管对域名 vb.ly 的控制权，突然关闭了由 [Ben Metcalfe](https://web.archive.org/web/20221209140420/http://benmetcalfe.com/blog/2010/10/the-ly-domain-space-to-be-considered-unsafe/) 和 [Violet Blue](https://web.archive.org/web/20221209140420/http://techyum.com/2010/10/official-vb-ly-link-shortener-seized-by-libyan-government/) 设计的网址缩短服务，这引起了轩然大波。

NIC.ly now [声明](https://web.archive.org/web/20221209140420/http://nic.ly/anvp.php)他们试图反复联系 vb.ly 的前所有者，但毫无结果，该域名被挑选出来是因为它的目的(“作为‘性友好的网址缩写’，主要供成人使用”)。

该注册处补充说，当地的网络社区在决策中发挥了作用，vb.ly 被认为是在“色情/成人”的旗帜下运营，因此违反了其规则和规定。

这意味着其他网址缩写商和其他拥有类似 [bit.ly](https://web.archive.org/web/20221209140420/http://www.crunchbase.com/company/bit-ly) 、 [ad.ly](https://web.archive.org/web/20221209140420/http://www.crunchbase.com/company/ad-ly) 等域名的公司可以[松口气](https://web.archive.org/web/20221209140420/http://thenextweb.com/socialmedia/2010/10/08/libyas-clarifies-the-vb-ly-takedown-bit-ly-can-breathe-easy)了，对吧？

除了这个声明留下了有待回答的问题。

NIC.ly 表示，其他网址缩写商已经制定了政策，要求他们遵守 NIC.ly 的规则和规定，但注册管理机构增加了以下相当模糊的部分:

> 关于保留短于 4 个符号的域名注册的决定。ly 仅适用于在利比亚当地有业务的实体，这符合 NIC.ly 的担忧，即来自国外的 URL 简称越来越受欢迎，使用所有这些名称剥夺了当地人注册其各种业务和利益的重要 3 个字母缩写的权利。
> 
> 作为注册机构，我们更希望看到 art.ly 被用于一个关于利比亚艺术的网站，或者 lda.ly 被利比亚扫雷协会使用，而不是在我们的国家 TLD 下添加更多的网址缩写。

注册处未能澄清的是，它是否打算剥夺业主的卖空。ly 域名控制他们的网站地址，或者如果这只是为了新的注册。

换句话说，NIC.ly 表示，它“更愿意”看到利比亚当地企业或个人使用 art.ly，但没有说明这是否意味着它将在可预见的未来某个时候实际转让该域名的所有权。
