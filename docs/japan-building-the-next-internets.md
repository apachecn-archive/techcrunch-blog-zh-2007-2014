# 日本建设下一代互联网

![060719_pb_internetcartoon.jpg](img/98ee884ee2dbd51ecde2f07be88ca9eb.png)

那些疯狂的日本人又在谈论建立一些新的技术，到 2020 年将取代我们心爱的互联网。这一切都很好，但他们更好地实现了更快、更安全的吞吐量以及对病毒和故障的恢复能力的承诺。

 我想巴西新闻发布会上传出的更大新闻是通信部长菅义伟承诺在 2011 年前将所有家庭的模拟广播转换成数字广播。他们甚至打算为低收入家庭提供必要的转换器，或者至少给他们提供一张优惠券。这不是很好吗？
