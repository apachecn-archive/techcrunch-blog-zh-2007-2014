# 耶普，更多三星便携式电脑

大山姆发布了一款新的便携式媒体播放器和一款新的 MP3 播放器，这两款播放器都配备了蓝牙 2.0 ( *无*适配器)、调频调谐器、录音机和视频播放功能。

Yepp YP-P2 将有一个支持手势但不支持多点触摸的 3 英寸触摸屏 LCD (480 x 272)和 4 或 8GB 的闪存，它将有三种经典颜色:黑色、酒红色或白色。它将支持多达 3 个同步蓝牙耳机，所以你可以和几个朋友挤在小屏幕周围。你可以观看 MPEG-4 和 WMV9 视频，但遗憾的是，你无法从 CinemaNow 等在线视频服务中获取内容。它不到半英寸厚，电池可以持续 35 小时播放音频，5 小时播放视频(可能没有蓝牙)。

耶普 YP-T10 是一款 4GB 的闪存音乐播放器，配有 2 英寸的屏幕，但是哇——午夜黑、神秘紫和白云？(他们真的打算用卫生纸品牌来命名其中一种颜色吗？)它可以处理多达两个蓝牙耳机，所以你和一个朋友可以在听同一首 ABBA 歌曲时看起来像完全的傻瓜。

11 月，三星将发布一个固件升级，将实现一些非常酷的手机连接——你可以使用任何一种设备作为替代手机！这两款播放器的价格还没有确定，但都将在 9 月下旬推出。
