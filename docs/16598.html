国家安全局试验了手机定位追踪程序

美国国家安全局(National Security Agency)在2010年试验了一个手机位置追踪项目，但最终搁置了这个想法。国家情报总监詹姆斯·克拉珀今天在国会山参议院作证时解密了该项目。

“在2010年和2011年，NSA收到了样本，以测试其系统处理数据格式的能力，但这些数据没有用于任何其他目的，也从未用于情报分析目的，”报告的一份草稿称，这是《纽约时报》早些时候获得的。

美国国家安全局一再否认其追踪美国人的位置，但其秘密文件显示GPS数据被用于建立一个与美国人有关联的嫌疑人网络。

参议员罗恩·怀登说:“在多年阻挠政府是否曾经或计划通过手机追踪守法美国人的位置之后，情报领导层再次决定对大部分真实情况保密——即使真相不会危及国家安全。”。

怀登已经率先提出了一个监控改革方案，该方案实际上可能会通过，但必须等待奥巴马总统的国家安全局特别工作组发布其建议。
