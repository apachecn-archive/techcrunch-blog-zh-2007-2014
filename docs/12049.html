Deezer计划到2012年6月在200个国家推出音乐服务(但不包括美国和日本)

原文：https://web.archive.org/web/https://techcrunch.com/2011/12/07/deezer-plans-to-launch-in-200-countries-by-june-2012-but-not-in-the-usa/

Deezer计划到2012年6月在200个国家推出音乐服务(但不包括美国和日本)

Deezer ，它在一份新闻稿中毫不夸张地宣称四年前“发明了音乐流媒体”(谁知道呢！)，计划在未来几个月内在全球范围内推广其音乐流媒体服务，到2012年6月将覆盖200个地区。美国和日本不在路线图上，因为 Deezer 认为这些国家由于“市场饱和和低增长预测”而不感兴趣。

这家初创公司雄心勃勃的全球推广计划的细节在巴黎举行的 Le Web 会议上公布，本周晚些时候，Deezer将在那里用英语音乐二人组The Ting Tings的独家演出庆祝该计划的推出。

Deezer目前在法国、比利时和英国推出，它表示将从12月8日开始每周在全球多个地区推出。来自新闻稿:

–2011年12月8日:荷兰爱尔兰

–截至2011年12月31日:欧洲其他地区:45个国家，包括西班牙(12月13日)、意大利(12月14日)、德国(12月15日)和俄罗斯(12月16日)

–截至2012年1月31日:加拿大和拉丁美洲(38个国家)

–截至2012年2月28日:澳大拉西亚和非洲(41个国家)

–截至2012年6月30日:世界其他地区(115个国家)

Deezer将在这些地区中的每一个地区提供定制定价方案，共有12种语言和13种货币。该公司将与当地电信公司合作，以提高其分销能力。它与脸书的深度融合和伙伴关系也将在这方面有所帮助。

Deezer声称，根据预测，到2016年，其基于网络的音乐流媒体服务最终将在全球75%的音乐市场上提供。

该公司补充说，它目前拥有140万付费用户，他们可以访问2000个不同品牌的1300万首歌曲的音乐目录，包括4个主要品牌。
