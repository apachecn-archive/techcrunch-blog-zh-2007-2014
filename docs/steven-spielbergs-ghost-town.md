# 史蒂文·斯皮尔伯格的《鬼城》

 [](https://web.archive.org/web/20230216094354/http://imdb.com/title/tt0112642/) 好莱坞超级制片人[史蒂文·斯皮尔伯格](https://web.archive.org/web/20230216094354/http://en.wikipedia.org/wiki/Steven_Spielberg)正准备推出一个新的社交网络，我们从多个渠道获悉。重点将放在有过或有兴趣分享超自然和外星经历的用户身上。新的社交网络也可能有调查所谓的鬼魂和 UFO 故事的原创视频内容。

斯皮尔伯格一直很喜欢鬼故事和 UFO 故事。他的[电影](https://web.archive.org/web/20230216094354/http://imdb.com/name/nm0000229/)包括《第三类亲密接触》(1977)、《吵闹鬼》(1982)、《外星人》(1982)、《卡斯帕》(1995)、《黑衣人》(1997)、《幽灵》(1999)、《世界之战》(2005)和即将上映的《星际穿越》(2009)。他 16 岁时的第一部电影是一部 140 分钟的科幻冒险片《火光》(后来激发了《亲密接触》)。除此之外，还有许多其他科幻电影。

显然，斯皮尔伯格自己至少有过一次超自然经历。有故事说他住在一家叫做怡东楼的旅馆里，被鬼魂吓得逃离房间，搬到 20 英里以外的地方。我们也听说过斯皮尔伯格小时候反复看到死去亲人的鬼魂的轶事。无论是基于现实还是非凡想象力的产物，这些经历可能对他一生的工作和这个即将到来的社交网络产生了影响。

该项目最初可能与雅虎有关，但在启动前就被扼杀了。但是，如果我们的消息来源是正确的，这个想法仍然存在，洛杉矶的一个团队正在努力在未来几个月内推出它。
