# 加州监管机构通过首个拼车规则，Lyft、SideCar 和优步 TechCrunch 大获全胜

> 原文：<https://web.archive.org/web/http://techcrunch.com/2013/09/19/cpuc-ridesharing-regulations/>

# 加州监管机构首次通过拼车规则，这是 Lyft、SideCar 和优步的一大胜利

加州公共事业委员会一致批准了围绕 Lyft、SideCar 和 UberX 等拼车服务的新法规(最初在 [Twitter](https://web.archive.org/web/20221130080816/https://twitter.com/lyft/status/380746471965868032) 上的大量[报道](https://web.archive.org/web/20221130080816/https://twitter.com/tomiogeron/status/380745935904448512)中提到)。

CPUC 早在 7 月份就提出了这些规定，为拼车服务在全州范围内的运营提供了法律框架。正如我们当时报道的那样，[大部分法规都围绕着公共安全](https://web.archive.org/web/20221130080816/https://beta.techcrunch.com/2013/07/30/california-regulator-proposes-new-ride-sharing-rules-in-a-victory-for-uber-lyft-and-sidecar/)，以及确保司机有背景调查并在发生事故时有保险。

根据 CPUC 发布的新闻稿，新法规建立了一种新的业务类别，称为运输网络公司，它要求这些公司从 CPUC 获得许可证，进行犯罪背景调查，建立司机培训计划，并持有每次事故最低 100 万美元的商业保险。

Lyft 联合创始人兼总裁约翰·齐默(John Zimmer)告诉我，加州是第一个建立此类规则的州。虽然该公司已经在旧金山(其总部所在地)、洛杉矶和圣地亚哥开展业务，但 Zimmer 表示，这一决定“为我们在现有市场的社区提供了清晰度”，同时也为在更多城市推出奠定了基础。他还说，当他与其他城市和州的监管机构交谈时，他们通常希望了解加州的情况，现在加州已经表明“有一种方法可以做到这一点，而不会在安全、创新或消费者选择方面做出妥协。”

至于今天通过的法规与最初提出的相比如何，齐默说，一些语言已经改变，但实质内容是相似的。CPUC 的一位发言人表示，自 7 月份以来的主要修订是加强围绕保险的一些条款。

“我们的决定强调安全是首要目标，同时促进这一新兴产业的发展，”专员马克·j·费隆在新闻稿中说。“我们已经明确了对保险属性的期望。现在，保险市场将决定最佳方法，以确保在这些车辆以商业方式运营时，乘客、司机和第三方始终都有保险。”

**更新:**旧金山出租车司机协会的一位发言人发给我以下声明:

> 旧金山出租车司机协会认为令人不安的是，CPUC 正在寻求创建一种新的出租运输服务，这种服务将不受当地监管机构的监督，同时与现有的当地监管的出租车服务进行不公平的竞争。
> 
> 这项措施从本质上解除了对加州出租车行业的管制，原因有很多。CPUC 只有五名安全执法调查员，负责美国第三大和人口最多的州，显然不足以执行他们提出的新规则。在加州，出租车是加州政府法规定义的随叫随到服务。任何额外类别的运输提供商，提供相同的随叫随到/按需客运服务，却没有相同的监管标准，使得现有的监管毫无意义。
> 
> 如果没有适当的地方监管，这只会导致跨国公司司机、公司和机会主义分子的滥用，导致对依靠出租车运输的残疾人、老年人和被剥夺权利者的乘客服务质量下降。
> 
> 这一决定没有考虑对所有加州人更广泛的环境、经济和道路安全影响。与跨国公司及其支持者的说法相反，仅旧金山市就增加了数千辆出租汽车，这已经加剧了拥堵和污染。
> 
> 我们发誓要继续为所有加州人平等获得安全、清洁、高效和价格合理的按需交通的权利而斗争。
