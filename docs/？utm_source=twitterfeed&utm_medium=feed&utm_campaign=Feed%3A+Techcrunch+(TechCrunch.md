# 脸书下周会发布什么？不是斯巴达计划。

显然，脸书正准备在下周展示一些“令人敬畏”的东西。至少，这是首席执行官马克·扎克伯格昨天参观西雅图脸书办公室时告诉记者的。路透社报道不管它是什么，它是由西雅图的 40 人团队开发的。他们认为这可能是在移动或平板电脑领域。

我们所能确定的是，它不会是[项目斯巴达](https://web.archive.org/web/20230203032627/https://techcrunch.com/2011/06/15/facebook-project-spartan/)，这个基于 HTML5 的应用平台，脸书已经与一小群外部开发者秘密合作了几个月。我们被告知，Spartan 最早也不会在 7 月中旬之前准备好——也不会以西雅图为基地。

总部设在西雅图的是与移动有着深厚联系的团队。[今年早些时候](https://web.archive.org/web/20230203032627/http://www.facebook.com/notes/facebook-seattle/made-in-seattle/207736969245338)，那里的团队为合并 m.facebook.com 和 touch.facebook.com 的站点做了大量工作。鉴于我们在[泄露的脸书手机照片应用](https://web.archive.org/web/20230203032627/https://techcrunch.com/2011/06/15/facebook-photo-sharing-app/)中看到的部分内容与 m.facebook.com 的新照片体验非常相似，脸书可能会从西雅图发布他们的新手机照片。我们最近听到的只是我们泄露的图片有些旧(但非常真实)。我们还认为，当它发布时，不会有一个独立的应用程序，而是，它将成为当前脸书应用程序体验的一部分。

我们也知道西雅图的脸书团队一直在试图建立一个桌面软件团队。但招聘仍在进行中，所以不要指望在相当长的一段时间内会有这个部门的任何消息。

通配符是 iPad 应用程序。[我们知道它即将到来](https://web.archive.org/web/20230203032627/https://techcrunch.com/2011/06/16/ipad-facebook/)，并且已经和许多看过它的人谈过了。尚不清楚西雅图团队是否建造了它，但这肯定是可能的。

**更新**:有消息称脸书下周发布的产品“极不可能”是 iPad 应用。
