# Tinychat 的视频聊天应用在脸书爆红

![](img/af3b5af294dc727df7baf8bdca6af38b.png)

Tinychat 的超级简单的消息服务[存在于网络](https://web.archive.org/web/20230202222126/http://tinychat.com/)(然后是[和一些](https://web.archive.org/web/20230202222126/https://techcrunch.com/2010/07/28/tinychat/))上，但是这个羽翼未丰的公司的脸书[视频聊天应用](https://web.archive.org/web/20230202222126/http://www.facebook.com/apps/application.php?id=140802830263)显然看到了它自己的一些稳固的增长。

该应用程序现在是脸书平台上最常用的视频聊天应用程序，根据应用程序页面显示，每月约有 240 万活跃用户。

这比其最接近的竞争对手 [vChatter](https://web.archive.org/web/20230202222126/http://www.facebook.com/vChatter) 多了大约 150 万月用户，我们已经报道过 vChatter，并将其比作“PG 级版本的聊天轮盘游戏”[这里是](https://web.archive.org/web/20230202222126/https://techcrunch.com/2010/09/21/vchatter-launches-a-pg-rated-version-of-chatroulette/)。

![](img/44f2c8e4e88c0a59b9bdb7e6e3ee97eb.png)

仅在 11 月份，安装了 Tinychat 脸书应用的用户总数就增加了 100 万，从上面的图表来看，该应用的增长率似乎是一致的。大约 75%的用户居住在美国。

很容易理解它为什么如此受欢迎——该应用程序旨在在脸书朋友中病毒式传播，因为你可以通过脸书聊天邀请在线联系人加入你的实时视频聊天，而不需要你的朋友安装该应用程序。

伤心欲绝。
