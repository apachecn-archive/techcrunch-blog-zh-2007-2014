# Twitter 的伊丽莎白·韦尔(Elizabeth Weil)加入安德森公司成为合伙人，她将把目光投向社交网络

安德森·霍洛维茨继续建造它的军火库。在宣布公司已经投资 1500 万美元给创新的 hip hop annotations 初创公司 Rap Genius 的几天后，我们得知它已经从 Twitter 雇佣了 T2。

Weil 是 Twitter 的营销主管，在此之前，她在另一家风险投资公司 Institutional Venture Partners 工作，专注于互联网和数字媒体投资。在 IVP 之前，韦尔是门罗风险投资公司的投资分析师。

除了每周为 Twitter 工作 80 个小时之外，Weil】还在她的车库里操作数字聚焦纸轮印刷机，手工制作给 Twitter 新员工的“欢迎”便条，还有其他事情。她和她的丈夫、Twitter 收入产品总监凯文·韦尔(Kevin Weil)也是狂热的马拉松运动员。

显然，Weil 在 Twitter 工作的三年里身兼数职(她的 Twitter 简历列出了“设计和文化”，她还在企业发展部门工作过一段时间)，我们还不知道她将在 Andreessen 从事哪些具体项目。

安德森和韦尔都向我们证实，她将加入他们的市场开发团队，帮助安德森投资组合公司的业务发展和营销战略。她将与 A16z 市场开发合作伙伴马克·克兰尼密切合作，并将专注于安德森的社会事业，但不会负责社会投资。

这位突破性的风险投资人最近还获得了另一个初创公司的聘用:四方发电机的特里斯坦·沃克饰演 EIR。
