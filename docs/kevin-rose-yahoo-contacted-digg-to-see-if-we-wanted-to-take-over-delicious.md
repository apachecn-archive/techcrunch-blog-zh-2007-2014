# 凯文·罗斯:雅虎联系了 Digg，看我们是否想收购 Delicious

看起来雅虎为美味寻找一个家已经有一段时间了。Digg 创始人[凯文·罗斯](https://web.archive.org/web/20230202234617/http://www.crunchbase.com/person/kevin-rose)本周在他的 [Diggnation](https://web.archive.org/web/20230202234617/http://www.youtube.com/watch?v=vVt9u71E24c#t=036m20s) 网络秀上说，雅虎联系了 [Digg](https://web.archive.org/web/20230202234617/http://digg.com/news) 看看该公司是否有兴趣收购 Delicious，当时一份内部幻灯片泄露，表明雅虎希望在[12 月裁员](https://web.archive.org/web/20230202234617/https://techcrunch.com/2010/12/14/yahoo-confirms-layoffs/)(时间将在 12 月初/11 月底)之前摆脱 Delicious。在视频中，罗斯在第 36 分钟左右谈到了这个问题。

Rose 说，起初他认为雅虎试图出售 Delicious(我们也是这么认为的)，但后来意识到雅虎只是想为书签服务找一个家。Rose 证实雅虎和 Digg 已经就 Delicious 的维护费用进行了初步谈判。

在这些早期对话后不久，Rose 说有消息称雅虎正计划“日落”Delicious 和雅虎产品家族中的许多其他服务。这时他在推特上说他“真的很想买下 del.icio.us，做一个非常棒的社会化书签网站，书签/音频注释的截图档案等等。”

Rose 有一些令人信服的想法来为 Delicious 注入新的活力，包括在给定时刻拍摄网页静态快照的能力，这样用户就可以随时访问他们最初添加书签时网站的样子。Rose 说，另一个可能的功能是音频录制功能，允许用户向书签添加音频注释，为书签共享添加额外的上下文。他还会添加专业账户，并试图从这项服务中获利。

罗斯说，他肯定会接管 Delicious，但没有提供任何关于这是否真的会发生的更新。他补充说，他联系了 Delicious 的原始创始人约书亚·沙赫特，看他是否想支持这个项目，但我们知道沙赫特正在寻求其他项目。罗斯说，他希望它能找到一个好的归宿，他很想用它做一些很酷的事情，但不知道雅虎是否会给他好吃的。
