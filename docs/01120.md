# FlixWagon 紧跟移动视频直播潮流 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/12/18/flixwagon-jumps-on-the-live-mobile-video-bandwagon/amp/>

[<amp-anim src="https://web.archive.org/web/20230121135630im_/https://techcrunch.com/wp-content/uploads/2007/12/flixwagonlogo.gif" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="flixwagonlogo.gif" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-anim>](https://web.archive.org/web/20230121135630/http://www.flixwagon.com/)桌面视频直播在美国可能只是[起飞](https://web.archive.org/web/20230121135630/http://techcrunch.com/2007/12/18/ustreamtv-gets-some-cash-and-general-wesley-clark/)，但在海外这已经成为一种移动现象。昨天 TechCrunch 英国 [的 Mike Butcher 在 Qik 及其直播视频流媒体服务上发布了](https://web.archive.org/web/20230121135630/http://techcrunch.com/2007/12/17/qik-streams-live-video-from-cell-to-web/) 。好吧，看起来 Qik 将能够享受一些博客世界阳光的宝贵时刻，因为一家名为 [FlixWagon](https://web.archive.org/web/20230121135630/http://www.flixwagon.com/) 的年轻以色列公司正在热中。

让我们从坏的开始… FlixWagon 有一个可怕的名字，并且有一个同样无效的标志。无论是名称还是标识都无助于传达该公司的真实产品:允许任何拥有 3G/WiFi 手机的人直接在网上直播视频。该过程中不涉及货车。

好消息是 FlixWagon 的技术远远优于其品牌。蜂窝到网络服务最重要的方面是对手机的广泛支持。FlixWagon 支持大多数诺基亚和索尼爱立信手机。用户必须下载的移动客户端应用程序对于 Symbian 是 76Kb，对于 J2ME 手机是 218Kb。FlixWagon 高管相信，这涵盖了绝大多数 GSM 手机。

[<amp-img src="https://web.archive.org/web/20230121135630im_/https://techcrunch.com/wp-content/uploads/2007/12/flixwagon-mobile-client.jpg" class="shot amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="flixwagon-mobile-client.jpg" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230121135630/https://techcrunch.com/wp-content/uploads/2007/12/flixwagon-mobile-client.jpg "flixwagon-mobile-client.jpg")为了给用户提供最短的播放路径，手机 app 的设计简单而有效。这不是强制性的，但用户可以从他们的手机编辑视频属性，如标题，标签，类别和访问权限(公共或私人)。手机应用程序还提供观众的实时反馈(见左边的截图)。

仅次于手机支持的重要性是视频质量。我有机会在几个场合看到 FlixWagon 的运行，并对网络上出现的比我预期的更清晰的视频印象深刻。我还看到了用同一部手机上传到 Qik 和 FlixWagon 的视频。主观上觉得 FlixWagon 更胜一筹。在视频清晰度和处理移动的能力方面都是如此——当手机稍微移动时，Qik 的质量就会大幅下降。FlixWagon 声称，这是因为它强调确保用户永远不会错过一秒钟的广播，即使在移动环境固有的最具挑战性的条件下。例如，如果用户进入电梯、地下室或由于任何原因失去接收，当接收恢复时，广播将继续，完整的视频将被存储并可供以后查看。

[<amp-img src="https://web.archive.org/web/20230121135630im_/https://techcrunch.com/wp-content/uploads/2007/12/flixwagon-web-portal.thumbnail.jpg" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="flixwagon-web-portal.jpg" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230121135630/https://techcrunch.com/wp-content/uploads/2007/12/flixwagon-web-portal.jpg "flixwagon-web-portal.jpg")[FlixWagon.com](https://web.archive.org/web/20230121135630/http://flixwagon.com/)作为服务的门户，允许网站访问者浏览直播和存储的广播。社交方面包括向朋友和家人广播警报，以及自动上传到 YouTube 和脸书等视频和社交网站。

FlixWagon 与 MTV Israel 合作，使记者和用户能够直接从他们的手机向即将在 mtv.co.il 推出的社交网络直播视频。(另见 [MTV Israel 对常规网络视频所做的](https://web.archive.org/web/20230121135630/http://techcrunch.com/2007/12/05/tvinci-makes-mtvisrael-rock-so-much-harder-than-mtvcom/))。我不会对 FlixWagon 被整合到 MTV 的全球网站上感到惊讶。

FlixWagon 将在 1 月份发布 Alpha 版本时，给予 TechCrunch 读者优先访问权。报名 [这里](https://web.archive.org/web/20230121135630/http://tc-register.flixwagon.com/) 。
