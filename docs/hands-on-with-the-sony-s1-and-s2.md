# 动手操作索尼 S1 和 S2 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/13/hands-on-with-the-sony-s1-and-s2/>

是官方人士！来自[索尼](https://web.archive.org/web/20230204115328/https://techcrunch.com/tag/sony)的可折叠[双屏 S2 平板电脑](https://web.archive.org/web/20230204115328/https://techcrunch.com/2011/04/25/sony-announces-s1-and-dual-screen-s2-android-tablets/)将于& T 在[上市，这是我们非常渴望得到的。当然，新闻稿中没有提到价格和可用性，尽管索尼确实承诺我们将在“今年晚些时候”看到基于](https://web.archive.org/web/20230204115328/https://techcrunch.com/tag/at&t) [Android 3.0 蜂巢](https://web.archive.org/web/20230204115328/https://techcrunch.com/tag/honeycomb/)的 S2 我们还亲身体验了 S1 和 S2，这基本上是我一周中最激动人心的部分。

我们从来没有机会看到 S2 完全折叠起来，它的便携性是我们一直想知道的。给你一些视角，S2 从一个主管的运动外套的胸袋进入舞台，比我想象的要小得多。它很容易放在口袋里，既可以放在裤子和夹克里，也可以放在钱包里。

不幸的是，索尼对规格非常吝啬，但我们确实听说这款平板电脑将同时具备 WiFi 连接和 AT&T 的 4G 支持。然而，有一些新信息可能会令人感兴趣。首先，索尼已经在 S1(折叠设计 10 英寸)和 S2 平板电脑上实施了一些新技术。“快速查看”是一个用户管理的设置，通过先加载图片，然后加载 java 脚本，它可以比普通平板电脑更快地加载网站。在一个演示中，一个带有快速浏览功能的 S1 打开页面的速度比一个关闭了快速浏览功能的 S1 快三到五秒。另一项技术被称为“快速触摸面板”，是一种特殊的软件，可以增加触摸屏对滚动和多点触摸手势的感知能力。

S2 将可以访问索尼的 Qriocity 媒体平台，并预装了一个电子阅读器应用程序。我们还了解到，S2 将预装裂纹，弹球英雄和坠毁 Bandicoot。S1 和 S2 都是经过 Playstation 认证的设备，因此用户将可以获得真正的 PS1 和 PSP 游戏，更多关于具体什么将在 8 月份推出的信息。所有的核心应用程序，以及预装的应用程序，都针对 S2 的 5.5 英寸双屏进行了优化。
