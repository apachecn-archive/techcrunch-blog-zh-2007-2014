# YouTube 和其他公司的配乐授权合作伙伴 Rumblefish 收购 Catalogik 以改进音乐搜索技术

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/03/21/rumblefish-soundtrack-licensing-partner-to-youtube-and-others-buys-catalogik-to-improve-music-search/amp/>

数字音乐领域的一次整合: [Rumblefish](https://web.archive.org/web/20230121164554/http://www.rumblefish.com/) ，一家与 YouTube、亚马逊、Vimeo、脸书和 Slideshare 等公司合作，让用户在他们的数字作品中添加授权音轨的公司，今天宣布收购 [Catalogik](https://web.archive.org/web/20230121164554/http://www.catalogik.net/) ，这是一个帮助寻找和监控特定数字音轨使用的平台。

交易的财务条款没有透露，但它包括两位创始人吉迪恩·阿罗姆和亚历克斯·斯通的收购；与 Catalogik 的音乐授权、投球和版权管理平台相关的 IP；正在申请专利的搜索相关技术；和商标等知识产权。

Rumblefish 平台涵盖了超过 180 万首音乐，该公司表示，它每天为超过 900 万个社交视频授权超过 2 万首配乐。

它说，这些视频获得了“数亿的月浏览量”，所以难怪 Rumblefish 正在寻找一种更好的方法来跟踪和收集这些浏览量的收入。改进跟踪过程也可能意味着更少的错误，例如[这个](https://web.archive.org/web/20230121164554/http://www.rumblefish.com/blog/2012/02/29/take-a-bird-off-it-a-video-with-birdsong-gets-inaccurately-idd-by-youtube-as-a-rumblefish-song-rumblefish-mistakenly-reinstates-the-claim-heres-what-happened/)和[这个](https://web.archive.org/web/20230121164554/http://www.techdirt.com/articles/20120425/00115218642/how-rumblefish-ended-up-claiming-copyright-song-uploaded-band-who-actually-held-copyright.shtml)。

Rumblefish 在下面的声明中表示，此次收购将有助于版权所有者——音乐家、出版商等——使用 Rumblefish 来跟踪他们的音乐在哪里以及如何被使用。Catalogik 所做的一些事情是提供基于算法的搜索功能，自动元数据填充和基于云的交易会计系统，这符合 Rumblefish 现有的在线模式，正如[Billboard](https://web.archive.org/web/20230121164554/http://www.billboard.com/biz/articles/news/digital-and-mobile/1553838/rumblefish-acquires-administration-startup-catalogik)所指出的那样，也可以用于改善该平台的整体搜索算法。

Rumblefish 表示，它将把 Catalogik 的技术集成到它的平台上，并特别强调了一个合作伙伴关系，该技术“加速了 YouTube 货币化平台的发展”。

Rumblefish 创始人兼首席执行官保罗·安东尼在一份声明中表示:“Catalogik 团队、代码库和 IP 组合非常出色，为 Rumblefish 提供了新的技术能力，这将帮助我们更好地在 YouTube 和其他社交视频网络和应用程序上赚钱。”“Catalogik 团队加快了我们的路线图，通过市场上最大的目录和最好的配乐搜索工具，改进了世界上最好的社交视频配乐平台。”

Catalogik 的总部设在洛杉矶，这是它的团队将留在这里，作为波特兰新的洛杉矶办事处的一部分。

下面发布。

> **Rumblefish 收购 Catalogik，加速其社交视频配乐货币化能力**
> 
> 收购有利于 Rumblefish 艺术家和唱片公司更好地从 YouTube 和其他视频应用程序和网络上的社交视频中赚钱
> 
> 俄勒冈州波特兰——2013 年 3 月 21 日——领先的在线社交视频配乐提供商 Rumblefish (www.rumblefish.com)今天宣布收购 Catalogik，这是一家软件即服务(SaaS)音乐许可和版权管理平台。此次收购进一步确立了 Rumblefish soundtrack 平台作为社交视频行业标准的地位，显著改善了其搜索工具，加速了其 YouTube 货币化平台的发展。
> 
> Catalogik 由企业家 Gideon Arom 和 Alex Stone 于 2009 年共同创立，他们是加州大学洛杉矶分校计算机科学和音乐学的毕业生，建立了一个领先的基于网络的音乐许可平台，使出版商、唱片公司和管理员能够节省时间、降低成本并增加同步收入。Catalogik 的技术包括基于动态算法的搜索功能、自动元数据填充和基于云的交易会计系统。
> 
> 在这项立即生效的交易中，Rumblefish 获得了 Catalogik 音乐授权、投球和权利管理平台、正在申请专利的搜索相关技术、商标和其他知识产权。Catalogik 的执行和开发团队将加入 Rumblefish，在新成立的 Rumblefish 洛杉矶办事处工作。Rumblefish 和 Catalogik 没有披露收购的财务条款。
> 
> Rumblefish 创始人兼首席执行官保罗·安东尼表示:“Catalogik 团队、代码库和 IP 组合非常出色，为 Rumblefish 提供了新的技术能力，这将有助于我们更好地在 YouTube 和其他社交视频网络和应用程序上赚钱。”。“Catalogik 团队加快了我们的路线图，通过市场上最大的目录和最好的配乐搜索工具，改进了世界上最好的社交视频配乐平台。”
> 
> Rumblefish 建立了世界上最大的版权许可音乐目录，拥有超过 180 万首配乐，每天将超过 2 万首配乐授权给超过 900 万个社交视频。Rumblefish 许可的配乐出现在视频中，每月浏览量高达数亿次。Rumblefish 音乐许可平台向其合作伙伴提供其 API 和 SDK，以便于使用和将许可的音轨集成到社交视频应用中。
> 
> Catalogik 的联合创始人兼首席执行官 Arom 说:“这对于我、我的联合创始人、我们的团队和我们的股东来说是一个伟大的成果。“我们很高兴能够解决同步许可市场面临的最具挑战性的问题，这是一个我们真正热爱的领域。通过与 Rumblefish 合作，我们的团队和技术将很快产生有意义的影响。”
> 
> 关于 Rumblefish
> Rumblefish 是最大的社交视频原声供应商，十多年来一直是音乐授权行业的先驱。通过其 API、SDK 和 Storefront 产品，该平台允许合作伙伴为 web 和移动应用程序提供配乐功能，并提供对全球最大的版权许可配乐目录(近 200 万首曲目)的访问。该公司使在线社交视频的配乐变得简单合法，并已颁发了超过 900 万份许可证，向其艺术家支付了数百万美元的版税。Rumblefish 的合作伙伴包括 YouTube、Animoto、Socialcam、Virtual Active、Klip 和 Google。该公司位于 www.rumblefish.com，总部位于俄勒冈州波特兰市。
> 
> 关于 Catalogik
> Catalogik 由企业家 Gideon Arom 和 Alex Stone 于 2009 年创立，是一个软件即服务(SaaS)音乐授权和版权管理平台。该平台使唱片公司和音乐发行商能够通过集成的 CRM 向音乐监管者推销同步许可机会，并利用定制许可店面、会计和交易跟踪、数据库管理和尖端音乐搜索技术等功能。
