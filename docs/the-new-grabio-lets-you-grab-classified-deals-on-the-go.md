# 新的 Grabio 让你在旅途中获得分类交易

Grabio 已经存在了大约一年，但他们最近更新了应用程序，并添加了许多功能。它是做什么的？本质上，它是一个移动 Craigslist，提供通知和基于位置的分类列表搜索。你可以进入任何一个社区，进行快速搜索，然后花几块钱就能找到一个坏掉的女孩自行车座位或者一个轻轻用过的全身石膏。很圆滑，是吧？

该公司由目前居住在芝加哥的霍拉修·博瑞乌和克里斯·波帕创建，在确定其目前的形象之前，该公司是自我创业并经历了多次迭代。在发布时，他们看到了 5000 名用户，尽管这个数字还在攀升。

有一些类似的服务，最著名的是 [EggDrop](https://web.archive.org/web/20221209005148/http://www.eggdropapp.com/) 和 [Zaarly](https://web.archive.org/web/20221209005148/http://www.zaarly.com/) ，但是 Grabio 使用一个位置感知 API 在你接近交易时通知你。

“这里的秘密是，与其他人相比，我们不会不断跟踪你的位置，”Boeriu 说。“我们使用苹果公司的一个很酷的 API，它可以告诉我们你何时移动，而无需主动跟踪你的 GPS。”

该应用程序还包括用户资料，移动支付，以及基于信誉的用户评级和关注。刊登物品是免费的，但如果你使用 Grabio 信用卡支付系统，卖家会支付 6%的费用。

“在不久的将来，我们还将与当地企业合作，以预定的成本为他们提供有针对性的线索，”Boeriu 说。未来的改进将包括地理围栏功能，当用户进入某个热卖区域时，它会通知用户。想象一下，如果你能知道什么时候能以半价买到一盒 zwieback 或 500 个部分充气的气象气球，你就会看到真正的价值主张。
