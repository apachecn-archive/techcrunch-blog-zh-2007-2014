Bose SoundDock便携式评论TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2007/09/08/bose-sounddock-portable-review/

人们普遍认为，最初的Bose SoundDock曾经是、现在仍然是市场上最好的iPod扬声器之一。我对它最大的不满就是它不是真正的便携，因为你必须把它插上电源。但不再是了。SoundDock Portable在声音和功率方面与纯AC版本惊人地相似；它的体积小巧，是迄今为止声音最大、最好听的东西。当然，399美元，你为它的完美付出了高昂的代价。九月底正式上市。

SoundDock Portable的光滑白色外壳(也有黑色)尺寸非常紧凑，为12 x 6.7 x 5.1英寸，重量为5.2磅。它看起来很像原始版本，但iPod基座从底部前端伸出，可以像恐怖片中的旋转书柜一样旋转到机身中。

整体设计非常简约，除了右侧的一对触摸感应音量控制键外，没有任何板载控件。附带的红外遥控器的橡胶表面上的八个按钮都有一个小凸缘，因此非常容易通过手感进行操作。它控制电源、回放、音量和播放列表跳过，但不能浏览iPod的菜单或音乐。在我的客厅里，从大多数角度使用遥控器都没有问题。

背面唯一的连接器是模拟八分之一英寸输入和附带的充电器；没有视频输出或USB端口。锂离子电池是可充电的，通过推动和转动背面的销钉可以轻松拆卸，这是一个巨大的优势。还没有备件的消息，但你可以在任何地方以最大音量播放3小时到正常使用8小时左右。考虑到低音表现的水平，这并不坏，可以很快耗尽电池。

我对侧边的音量按钮的灵敏度不太感冒，但你可以使用非常灵敏的遥控器来代替。我喜欢你的iPod即使在没有插上扬声器的情况下也能充电，但是要注意这会更快地耗尽扬声器的电池。通过背面的手柄，SoundDock非常容易携带，它作为系统的低音端口(或Bose所说的波导出口)发挥了双重作用。

当您按下遥控器或扬声器上的按钮时，扬声器格栅后面的LED就会亮起，当电池需要充电时，它会闪烁红光。您也可以通过按住遥控器的关闭按钮来检查电量。请注意，如果您14天没有使用扬声器，您只能通过插上电源再次打开它。

Bose不包括任何坞站插件；你应该用你的iPod附带的那个。通用电源插头上有一个绕线器，方便出行，但插头本身体积庞大，占用了大量插座空间。对于399美元，如果Bose给你一个旅行箱或包就好了，但结果是你必须为此多付59美元。Boooo。

像约翰·科尔特兰的“蓝色列车”这样的声学爵士乐听起来很饱满，尽管铙钹有点刺耳。一些简单的低音和高音音调控制会比强迫用户使用iPod不灵活的均衡器预设更好地解决这个问题。低音足够紧实，可以承受一点提升而不会变得浑浊。尽管如此，还是很难不被兴奋所淹没。出色的中音细节，高音萨克斯管在混音中的舒适位置。四周非常热闹。

在瑟吉·纳卡里亚可夫的古典小号和钢琴二重奏专辑卡门·范泰西中，你可以听到小号的每一个细微差别，包括年轻的俄罗斯人的呼吸和小号阀门的移动，钢琴声清晰如钟声。在较大型的管弦乐作品中，比如莫利斯·安德烈演奏的胡梅尔协奏曲，每个部分(木管乐器、铜管乐器、弦乐器、打击乐器)都是相互独立的，独奏者被清晰地呈现出来，但不要太靠前，尽管你会对乐队的规模有所损失。

国际刑警组织的《T2》古怪的《T3》专辑，尤其是像《邪恶》这样的重低音电音听起来非常令人兴奋。我用的是从eMusic下载的MP3版本，它仍然清晰有力。红辣椒乐队的声音强劲有力，因为SoundDock便携式吃跳蚤的过度驱动的低音部分早餐。这是一些

霍顿·希特牧师的复兴和费西合唱团的甘露图在乐器上比在声乐上听起来更好。那些中等水平的男歌手往往会被其余的音乐淹没，这是因为一些人声和吉他声在中间位置略有下降。低音和鼓听起来很美，清晰度很高。

在杰克逊五兄弟的“我要你回来”中，你可以听到经常迷路的康加舞，就像铃鼓一样清晰，人声通过这密集的轨道非常清晰。低音和高帽钹在鲍勃·马利的《出埃及记》中大放异彩，尽管人声只是略微有些低沉。Jay-Z的“证明我的暴徒”和“改变我的衣服”从这么小的地方发出惊人的力量和重击，Jay-Z的歌词有所有的存在和清晰，就像在俱乐部里一样。

399美元对于一个iPod扬声器来说太贵了，但这个听起来非常好——事实上，比目前任何其他便携式型号都好。如果你能付得起钱，你就不会失望，除了可能会因为没有手提箱来保护你昂贵的东西。
