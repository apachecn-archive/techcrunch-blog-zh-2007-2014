# 微软的大胆举措:如果他们不能赢得平板电脑竞赛，他们不会承认它的存在

昨晚观看 Windows 8 构建版时，我注意到的最有趣的一件事是，有人坚持认为 Windows 8 设备都是个人电脑。台式机上的 Windows 8？个人电脑。触摸屏笔记本电脑上的 Windows 8？个人电脑。ARM 平板电脑上的 Windows 8？个人电脑。任何带有 Windows 8 代码的东西都是电脑，而任何能打电话的东西都是 Windows Phone。

十年前，微软只关注平板电脑。几年前，微软曾试图向用户销售平板电脑，但在惨败后，他们放弃了。为什么？因为当新生的智能手机和衰落的 PDA 市场围着那些可笑的技术幻想打转的时候，他们一直把 Windows 贴在石板上，称之为平板电脑，而微软几年前就放弃了这些幻想。

但是，以一种非常迂回的方式，微软已经退出了平板电脑业务。不再有 Windows 平板，只有 Windows PCs。让世界上的其他人在外形上一决雌雄吧:微软正拿着他们的球回家。

需要说明的是，我测试的“平板电脑”运行的是 Windows，而 Metro 现在是固定在上面的。)但我觉得他们的定位既迷人又大胆。在过去两年左右的时间里，人们一直在叫嚣着要一台 Windows“平板电脑”。现在他们有了 Windows 电脑。

他们的意思是硬件无关紧要。我希望这将扩展到他们的销售渠道，我们不会在商店里有 Windows 8 专业版、Windows 8 游戏版、Windows 8 家庭版和 Windows 8 酒鬼版(但我不会屏住呼吸。)不过，我很怀疑会有叫 Windows 8 平板版的东西。

说到底，这都是营销。但是，在一次大规模的背离中，他们没有称之为“Windows 水平平板体验 2012 ”,相反，他们坚持 PC 的所有价值的概念。为什么？嗯，苹果用他们的“我是 Mac/我是 PC”广告铺平了道路，然后微软用他们的“我是 PC”广告进一步开拓了这一领域。现在他们实际上有了一个竞争对手，他们需要拥有自己的那部分计算空间。他们就是这样成功的。

现在来看这个营销计划如何成功还为时过早，但是微软非常明确地表示，他们不希望“后 PC 时代”意味着这是一个后微软时代。还有什么更好的方法可以通过吸收人们用来描述他们产品的术语来解决这个问题呢？
