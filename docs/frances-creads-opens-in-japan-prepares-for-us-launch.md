# 法国的 Creads 在日本开业，准备在美国推出

总部设在巴黎的 [Creads](https://web.archive.org/web/20221005035819/http://creads.org/) 的创立听起来与硅谷的成功故事没有什么不同:2008 年，23 岁的联合创始人朱利安·梅钦和罗南·佩鲁用不到 7000€和口袋里的钱创办了他们的公司。他们种子资金的主要来源是通过在法国[企业家沙龙](https://web.archive.org/web/20221005035819/http://www.salondesentrepreneurs.com/v2/index.php)举办的创业创意竞赛获得的，这也是本周在巴黎举行的活动。

两年后，Creads 的在线平台见证了顶级公司利用其超过 10，000 人的创意社区。该平台允许公司利用社区的创意人才开发 100 多种不同类型的与品牌相关的创意竞赛，包括公司名称、徽标、广告和设计。由于 Creads 更像是一个参与式的在线广告代理机构，而不仅仅是一个众包网站，大公司往往更容易利用它的专业知识。

该公司于 2009 年秋季在西班牙开设了第一个国际办事处，并刚刚宣布在东京开设第一个亚洲办事处。由于日本的潮流变化比季节更快，Creads 希望通过其 web 2.0 创意竞赛平台来响应该国的创意需求。

这位雄心勃勃的联合创始人正计划今年晚些时候在美国筹集新一轮资金并开设店铺。
