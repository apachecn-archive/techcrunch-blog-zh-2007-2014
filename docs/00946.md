# 脸书拿着微软的钱跑了。TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/10/24/facebook-takes-the-microsoft-money-and-runs/amp/>

[<amp-anim src="https://web.archive.org/web/20230121214415im_/https://techcrunch.com/wp-content/uploads/2007/10/facebooklogo2.gif" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="facebooklogo2.gif" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-anim>](https://web.archive.org/web/20230121214415/https://techcrunch.com/wp-content/uploads/2007/10/facebooklogo2.gif "facebooklogo2.gif")虽然如果[脸书](https://web.archive.org/web/20230121214415/http://www.crunchbase.com/company/facebook)为了谷歌而抛弃其当前的广告服务合作伙伴微软，故事情节会更加戏剧性，但是[微软已经赢得了](https://web.archive.org/web/20230121214415/http://online.wsj.com/article/SB119323518308669856.html?mod=hpp_us_whats_news)这笔交易。据《华尔街日报》报道，该公司将在脸书投资 2 . 4 亿美元，并扩大其与国际市场的现有关系。(之前的广告关系只针对美国，现在微软是脸书在美国和海外的独家广告合作伙伴，直到 2011 年。)2.4 亿美元是少数股权，对该公司的估值为 150 亿美元。因此，这相当于微软不到 2%的股份，比早些时候流传的 5%的股份要少得多。(**更新**:在电话会议上，脸书保留了更多投资者可能参与此轮融资的可能性，尽管它拒绝透露这些投资者可能是谁，也可能不是。但这确实为脸书筹集超过 2.4 亿美元的资金打开了大门。

虽然谷歌在整体理念上更合适(更开放)，但从目前的交易中买断微软为脸书在美国提供广告的成本可能太高了。鉴于谷歌与微软的深厚关系，坚持与微软合作总是阻力最小的途径。有人可能会说，谷歌从未对基于私人用户档案数据的定向广告感到舒服，这似乎是脸书在广告方面的巨大承诺。微软没有这些疑虑。
