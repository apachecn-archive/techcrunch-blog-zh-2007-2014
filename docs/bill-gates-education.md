# 比尔·盖茨:五年后，最好的教育将来自网络

![](img/8d48ba23e0cfef3c2cea2ce6a5ab0dd1.png "te")

[比尔·盖茨](https://web.archive.org/web/20221006093348/http://www.crunchbase.com/person/bill-gates)认为有些东西也会死去。

不，这不是像尼古拉斯·尼葛洛庞帝那样的实体书——相反，盖茨认为年轻人必须去大学接受教育的想法将会相对很快消失。如果他们是自我激励的学习者。

今天，盖茨在加州太浩湖举行的科技经济会议上说:“五年后，你将可以在网上免费找到世界上最好的讲座。“这将比任何一所大学都要好，”他继续说道。

他认为，不管你的知识是如何得来的，你都应该因此而得到荣誉。无论是麻省理工学院的学位，还是你从网上的讲座中获得的所有知识，都需要有一种方式来突出这一点。

他肯定地说，教育机构对孩子来说仍然至关重要，K-12。他热情洋溢地谈到了特许学校，在那里孩子们可以花 80%的时间深入学习。

但是根据盖茨的说法，大学需要少一些“地方色彩”。嗯，除了聚会，他开玩笑说。

但他的总体观点是，获得这些高层次的教育太贵、太难了。很快，基于地方的大学教育的重要性将会比现在低五倍。

根据盖茨的说法，教育系统的一个特殊问题是教科书。甚至在小学里，一本关于数学的书可能有 300 页。“它们是巨大的、令人生畏的书，”他说。我看着它们，心想:里面到底有什么？

根据盖茨的说法，我们的教科书比亚洲的教科书长三倍。然而他们在教育的许多方面击败了我们。问题是，这些东西是由委员会建立的，更多的东西只是简单地添加到已经存在的东西上。

盖茨说，技术是使教育回到控制之下并扩大教育的唯一途径。
