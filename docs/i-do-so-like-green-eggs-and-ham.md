# 点评:罗技无线触控板是我的新好友 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/09/22/i-do-so-like-green-eggs-and-ham/>

“这东西居然管用。”这是我上周拆箱后几分钟的想法。我把我的罗技 G700 推到一边，挑衅地把新的触控板放在我的游戏鼠标垫上。我想，未来就在这里。

一周后，我对我之前陈述的准确性非常满意。无线触控板并不完美。我有一个问题，虽然完全是第一世界，但对于那些患有关节炎或软弱体质的人来说，可能是一个潜在的交易破坏者。

罗技无线触控板之于 Windows，就像苹果魔术触控板之于 OS X 一样。它试图将多点触控触控板的优点带入桌面环境。确实如此。事实上，这款罗技设备提供了我在 Windows 上体验过的最好的触控板。

Windows 笔记本上的触控板几乎都是垃圾。我发现了一些可以接受的，主要是在宏碁/Gateway 机器上，但大多数都很糟糕(我宁愿做输精管切除术，也不愿使用惠普触控板)。Mac 和 Windows 触控板之间的体验差异类似于乘坐劳斯莱斯和 Geo Metro。这些成熟的概念让我对这款触控板的期望相当低。我完全期待它能工作——罗技制造了好东西——但我也期待 Windows 会妨碍它的功能。

然而，当打开无线触控板时，我惊喜地发现，这款设备不仅可以使用，而且效果很好。尽管有一些怪癖，我还是喜欢它。

与大多数罗技产品一样，无线触控板是一种即插即用的设备。插入 USB 接收器，它在大约一分钟内开始工作(至少在 Windows 7 上)。不过，我确实不得不从罗技的网站上下载了一个软件包来调整跟踪速度和多点触摸选项。

这款设备的物理尺寸和苹果 Magic Trackpad 差不多。它们都停留在相同的微小角度，但相似之处也就到此为止了。无论好坏，苹果 Magic Trackpad 的整个玻璃表面都是用来追踪的。在 Logitech 版本的跟踪区域周围有大约一厘米的边框，底部有两个大的、易于点击的按钮。比起 Magic Trackpad 的可点击表面，我更喜欢按钮，但塑料表面不如玻璃光滑。

让我最初感到惊讶的是，触控板甚至在我的双 24 英寸显示器设置上也能很好地工作。我完全预料到会讨厌在这么宽的工作区域使用触控板。一开始还不错，然后你的手因为保持三个手指在超大尺寸的表面上稍微抬高而开始抽筋。这是身体上的伤害，但是为了你们，我亲爱的读者们，我坚强地挺过了痛苦，因此和一个新朋友一起出现了。这个触控板几乎完全取代了我心爱的鼠标。

作为参考，我的 G700 是市场上最好的鼠标，现在大部分时间都没有使用，但仍处于就绪状态。你看，当我需要快速完成某件事时，我仍然默认使用它。17 年来，我几乎每天都使用鼠标，所以移动是第二天性，跟踪比这种触控板更精确。但对于休闲浏览，无线触控板实际上更舒适一些。多点触摸手势在浏览器中非常棒，尽管这听起来很懒，但我不需要移动手臂来移动我的 40 英寸屏幕。

这里有一些警告。多点触摸不像 OS X 那样内置在 Windows 中，所以不要指望 Photoshop 中的花哨手势，尽管双指滚动似乎完全复制了滚轮。此外，希望这是显而易见的，但游戏是不可能在触控板上；保持你的鼠标左右 Deus Ex。

罗技无线触控板是 Windows Magic 触控板。罗技制造了一个神奇的产品，定价为 50 美元。给它一个机会，熬过痛苦，你可能会爱上它。但为了以防万一，最好从零售商那里购买，不要支付愚蠢的进货费。**推荐**。
