# 脸书向美国 iOS Messenger 用户推出 VoIP 通话

脸书最初于今年年初开始为其 iPhone 版 Facebook Messenger 应用程序的用户测试新的免费[通话功能](https://web.archive.org/web/20221202232803/https://beta.techcrunch.com/2013/01/03/facebook-voice-messaging/)，现在这项服务也适用于美国用户。免费通话按钮应用程序现在显示在美国用户的应用程序中，在任何双方都使用 Facebook Messenger 应用程序的对话中。

对于社交网络的用户来说，这是一个非常方便的功能，特别是因为与 Skype、Viber 或其他 VoIP 通话应用不同，你可能已经有了一个广泛的联系人网络，你实际上希望通过内置的实时语音对话，并随时准备在脸书上进行。虽然它在蜂窝连接上不是完全免费的，因为它使用你的数据连接，但它在家里的 Wi-Fi 上是完全免费的，这意味着它可能会取代许多用户的家庭电话，如果他们仍然坚持一个，或者至少是付费的家庭 VoIP 服务。

正如 Josh 在他关于加拿大试点发布的帖子中指出的，这不是建立在 Skype 网络上的，人们可能会根据两家公司在语音和视频聊天方面的合作关系来猜测。相反，这是脸书自己试图为其用户拥有每一个沟通渠道。看起来加拿大的试验进行得很顺利，但我们将关注它在更大的美国用户群中的效果。
