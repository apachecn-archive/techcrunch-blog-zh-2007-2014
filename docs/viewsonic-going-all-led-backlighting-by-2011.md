# 优派将于 2011 年实现全 LED 背光照明

好消息:传统照明显示器正在走向渡渡鸟之路。优派正在逐步淘汰所有非 LED 显示屏，并希望到 2011 年初只有 LED 背光单元。这可能不是一个很难做出的选择；led 越来越便宜，在市场上的声望也越来越大。

下一个变化是局部调光与侧光 LED 显示屏的对比，或者甚至是(我敢不敢做梦？)OLEDs。带来对比！
