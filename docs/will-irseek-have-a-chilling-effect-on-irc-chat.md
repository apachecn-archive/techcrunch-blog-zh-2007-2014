# IRSeeK 会对 IRC 聊天产生寒蝉效应吗？

[![](img/040dfc3a0aff087520d540d007c21a26.png)](https://web.archive.org/web/20221204173853/http://www.irseek.com/) 以色列新创公司 [IRSeek](https://web.archive.org/web/20221204173853/http://www.irseek.com/) 正在以每天 600 万次对话的速度索引公共互联网中继聊天( [IRC](https://web.archive.org/web/20221204173853/http://en.wikipedia.org/wiki/IRC) )频道。该公司现已将 3 亿次对话编入索引。包括 EFnet、DALnet、Freenode 和 QuakeNetUndernet 在内的最受欢迎的网络都受到了监控——IRSeeK 现在正在“监听”10 个网络的 2000 多个频道。

该公司表示，目前很少有 IRC 搜索引擎，大多数专注于特定的利基市场或单一网络。IRC 服务器上近 20 年的数据实际上已经丢失。IRSeek 希望确保未来的对话能够被正确地索引和搜索。这是一个巨大的未开发的知识库。

因此，如果你想看看人们在 IRC 上谈论什么，比如说， [iPhone 解锁](https://web.archive.org/web/20221204173853/http://www.irseek.com/result.php?keywords=iphone+unlock)，现在你可以了。最受欢迎的搜索词会出现在网站的首页。

该公司由 Eran Cohen(首席执行官)和 Ariel Berkman(首席技术官)创建。开发始于 2006 年年中。

该公司表示，当检测到文件共享活动时，就会关闭一个频道，无论如何私人对话都不会被窃听。尽管如此，一些 IRC 用户，可能对隐私有不合理的期望，可能会被 IRSeeK 困扰。个人信息经常在 IRC 聊天中泄露。这些信息现已编入索引，可供搜索。搜索也可以通过 IRC 昵称进行，所有涉及该昵称的对话(或者即使他们只是在房间里)都是链接的。当然，昵称不是唯一的，随着时间的推移，许多用户可能会选择相同的昵称。但即便如此，输入的所有内容都可以被其他人找到的知识可能会对用户产生寒蝉效应。
