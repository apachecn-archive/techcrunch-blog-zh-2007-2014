# Bexy 的 iMirror:新公司，酷 iPod Dock

![imirror1.jpg](img/915ae3a694824852b78950ff0e60b912.png)

带着没有足够多的 iPod 基座解决方案，更不用说足够多的公司*生产* iPod 基座解决方案的想法，今天发现了一份关于一家生产 iPod 基座解决方案的新公司 Bexy 的新闻稿。和“性感”押韵。不知道为什么。

乍一看，新的 iMirror dock 看起来比一些更好，但它不是以前没有做过的。人们认为，Bexy 似乎已经找到了解决一些问题的方法，这些问题使一些码头变得不完美。

例如，遥控器可以与您的 iPod 对接。许多 docks 都有带 LCD 的遥控器，因此您可以看到您正在播放的内容。问题是他们通过无线方式获取信息，所以有相当大的延迟，尤其是在浏览的时候。iMirror 会将您的播放列表和资料库文件与 dock 中的 iPod 同步，镜像遥控器内存上的信息。因此得名，聪明。

我们希望这种微妙的创新成为新公司的一种趋势，并祝他们好运。鉴于有这么多 iPod 配件，它需要一些特别的东西来“惊艳”我们。希望 Bexy 能做到这一点。
