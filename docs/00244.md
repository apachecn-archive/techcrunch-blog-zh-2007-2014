# Genius 为游戏玩家打造的振动耳机

Genius 发布了一套革命性的耳机，内置振动系统，提升音质，让你真正融入游戏。耳机很小，可以折叠，所以你可以把它们带到下一个局域网聚会，而不用担心被普通人发现。USB 供电的手机有内置麦克风，噪音消除选项，音量控制和振动开关。HS-03U 将于 9 月上市，售价 39 美元。
