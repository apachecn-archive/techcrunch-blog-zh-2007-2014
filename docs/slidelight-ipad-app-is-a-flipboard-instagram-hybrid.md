# Slidelight iPad 应用是 Flipboard 和 Instagram 的混合体

来自波兰华沙的一款新的 iPad 应用程序可能会成为摄影界的 T2。由 Macoscope 发布的这款名为 Slidelight 的应用程序为你完成了大部分繁重的工作，它通过各种过滤器将 [Instagram](https://web.archive.org/web/20230203075844/https://techcrunch.com/tag/instagram) 图片拉进来，并以最令人愉快的方式格式化。事实上，根据应用程序的描述，Slidelight“充满了如此多的快乐，就像过量服用内啡肽一样。”

虽然我们不太确定你是否真的会被所有的乐趣击倒，但总的来说，这看起来是一个非常漂亮的应用程序。Slidelight 将 Instagram 上的照片过滤成自行车、通勤、小狗、猫、欧洲首都、形状和大小、音乐、微距摄影、体育、时尚和设计等类别。但是药物过量带来的快乐不止于此。Slidelight 还抓取在您所在区域拍摄的照片，并创建一个特殊的基于位置的小过滤器(有点像 [Color](https://web.archive.org/web/20230203075844/https://techcrunch.com/tag/color/) )。

另一个过滤器——我预测这将是 Slidelight 中最受欢迎的类别之一——是针对名人的 Instagram 照片的。还记得有一次[贾斯汀比伯拍了一张令人难以置信的无趣的交通照片](https://web.archive.org/web/20230203075844/https://techcrunch.com/2011/07/21/biebergram/)然后 Instagram 爆炸了吗？高速公路上的汽车照片就属于这一类。显然，你也可以登录 Instagram，使用一个基于你和你朋友照片的过滤器。

这款应用目前在苹果应用商店售价为[2.99 美元，需要 iOS 4.2 或更高版本。](https://web.archive.org/web/20230203075844/http://itunes.apple.com/us/app/slidelight/id443373778?ls=1&mt=8)
