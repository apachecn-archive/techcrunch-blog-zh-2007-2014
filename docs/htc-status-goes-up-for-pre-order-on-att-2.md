# HTC 在美国电话电报公司 TechCrunch 上的预购量上升

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/11/htc-status-goes-up-for-pre-order-on-att-2/>

这款以脸书为中心的 HTC Status 智能手机现在可以从美国电话电报公司预订，正式上市日期为 7 月 17 日。我们知道状态(一旦查查)是正式前往美国电话电报公司，我们已经确定在 7 月推出，但我们不知道定价或可用性的细节，直到今天。HTC Status 的新两年协议价格为 50 美元，考虑到这款手机的老式外形和笨拙的小屏幕，这个价格可以理解。

具体来说，屏幕尺寸为 2.6 英寸，比我们去年见过的几乎任何屏幕都要小得多，分辨率为 480×320 像素。在背面，你会发现一个 500 万像素的摄像头，以及一个用于视频聊天的 VGA 前置摄像头。Status 在 800MHz 处理器上运行 Android 2.3.3 Gingerbread。这款手机支持 WiFi、蓝牙 3.0 和 microSD 卡。

其他规格包括 2.6 英寸显示屏，提供 480×320 像素的分辨率，500 万像素的后置摄像头，以及用于视频聊天的 VGA 前置摄像头。Status 运行 Android 2.3.3 Gingerbread，由 800MHz 处理器提供支持，并支持 microSD 卡、Wi-Fi 连接和蓝牙 3.0。

很明显，这些规格并不令人敬畏——但是请等一下。先不要放弃手机。Status 的成名在于它的脸书集成，如果它没有卡在一个可以说是垃圾的手机里，那将会非常酷。基本上，无论用户此刻在做什么，无论是听音乐，浏览图片，还是网上冲浪，只要按下右下角的脸书按钮，就可以直接上传到脸书。

这一切都很好，但事实是，如果你对手机比对脸书更认真，这可能并不适合你。另一方面，对于十几岁的儿子或女儿来说，这可能是一个相当划算的选择。
