# Gulu 让飞行中的活动策划看起来不错

今晚在我们的 Disrupt 北京会议上，初创公司战地竞争者 Gulu 对卫冕 Disrupt 冠军 Shaker 说了一些激烈的话。

Shaker 想把你做的事情，也就是闲逛的行为，变成你可以在自己家里穿着睡衣就能做的事情。这个。是。令人不安。我们是 Gulu.com 的联合创始人，我们希望你能和我们一起出去。”

Gulu 希望填补脸书活动/Plancast 等服务和 Foursquare 等服务之间的空白。前者让你提前做好计划，而 Foursquare 让你在那里的时候广播你在哪里，而 Gulu 专注于即兴的活动规划，他们称之为“意图广播”。

这个想法很简单:你说你想做什么，你的朋友就点击写着“我也是”的按钮。所有有意愿/有希望的参与者都会自动加入一个聊天组，让活动可以随时计划。

如果你的朋友中没有一个愿意做你现在想做的事呢？你所有悬而未决的“我想做的事情”都会被添加到待办事项列表中，你的朋友可以在闲暇时细读。一旦你有了足够数量的与会者，你就可以开始你的活动了。

这是一个全新的想法吗？呃，也许不是——脸书的状态更新可能也一样有效。当然，你也可以对*大多数*社会规划服务说同样的话，所以我们期待着看到这种服务如何发展。
