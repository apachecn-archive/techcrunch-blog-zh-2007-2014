# Buddy Media 推出用于品牌管理的集成脸书和 Twitter 客户端

![](img/661c317c036d4ad45689165395e33a00.png)

随着越来越多的品牌向脸书和 Twitter 寻求社交媒体的存在，对简化这种存在的应用程序的需求也在增长。 [Buddy Media，](https://web.archive.org/web/20221208222343/http://www.buddymedia.com/)一家开发社交网络应用的初创公司，希望通过其全新的一体化社交媒体管理系统吸引品牌营销人员的注意。

Buddy Media 平台面向广告商和公关代表，帮助创建、管理和跟踪脸书和 Twitter 上的社交活动(该客户端将很快为 MySpace 和其他社交网络提供功能)。关于脸书，该平台让代理商创建、管理和跟踪脸书页面，以推动和增加用户和品牌的参与度。用户不需要事先掌握任何 HTML 知识就可以在脸书上创建页面，并且可以相当容易地创建流畅的交互式页面。网页本身不仅仅是广告；用户可以在他们的页面上添加各种应用程序，如投票、YouTube 视频、优惠券、幻灯片和虚拟礼物。该平台还允许你将更新直接发布到页面的墙上。Buddy Media Platform 允许代理商在品牌网站上实现脸书连接，以利用社交网络的图表。

我们在这里提到的 Buddy Media 的 Twitter 管理系统，可以让营销人员衡量和识别围绕特定品牌、相关话题和竞争对手的 Twitter 热门话题。你还可以通过 Twitter 跟踪一个品牌和整个行业的表现和趋势，轻松查看关注者、提及和转发数据。

Buddy Media 的工具可以让你跟踪 Twitter 上点击率的数量和频率，并监控和分析与竞争对手相比，关于特定品牌的推文的情绪。在该系统中，您可以创建各种配置文件来管理多个品牌和不同的 Twitter 帐户，并安排在活动开始前发布推文。该系统本身就像一个 Twitter 客户端，所以你可以有一个集中的地方来发布 Tweet，监控和绘制品牌。

该平台还提供其他服务，让广告商和公关人员更好地监控品牌参与度，例如对活动和资源的实时分析，以帮助经理们推动和跟踪社交活动和点击。Buddy Media 的系统提供了实时搜索功能，这对于希望深入了解社交网络上有关其业务的对话的公司来说尤其有用。

对 Buddy Media 来说，提供一个全面的平台，作为品牌经理管理社交媒体活动的一站式商店，是一个明智的举措。有一些服务也在提供类似的服务，比如 [PeopleBrowsr](https://web.archive.org/web/20221208222343/http://www.peoplebrowsr.com/) 和 [CoTweet](https://web.archive.org/web/20221208222343/http://cotweet.com/) (专注于 Twitter)。但是 Buddy Media 引人注目的脸书功能，例如制作视觉上吸引人的交互式页面的能力，可能会受到几乎没有技术背景的品牌经理的欢迎。
