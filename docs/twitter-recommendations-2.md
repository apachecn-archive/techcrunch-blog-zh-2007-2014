# 通过调整，Twitter“关注谁”变得更强大、更敏捷

![](img/188c9bfb2d38df7a51280b6770a44795.png "New-TV-Spot-For-The-Expendables--Awesome")

很自然，目前[关闭了](https://web.archive.org/web/20221207045644/http://status.twitter.com/post/1156670612/who-to-follow-disabled)，但是当它恢复时，你会注意到 Twitter 的“关注谁”功能有所不同——它甚至更好。

当它在 8 月份推出时，我注意到这个推荐用户让你关注的功能就像是推特社交图的[类固醇。看看我从那时到现在的统计数据，我已经添加了大约 50 个人来关注——而且几乎都是来自这个功能。此外，自从该功能发布以来，关注我的人数一直在飙升——](https://web.archive.org/web/20221207045644/https://beta.techcrunch.com/2010/08/05/twitter-recommendations/)[，我并不孤单](https://web.archive.org/web/20221207045644/https://beta.techcrunch.com/2010/09/06/twitter-people-recommendations/)。尽管如此，该功能也有点单调。你会一遍又一遍地看到同样的建议。现在好多了。

几天前，Twitter 推出了一项对该功能的更改，似乎使它能够更频繁地更新更多更新的推荐。在对我的观察的回复中，Twitter 的 Pankaj Gupta(研发团队)证实了这一点。Twitter 的回应进一步证实了这一点:

> 我们推出了许多改进中的第一项，即我们现在考虑用户查看某个推荐但没有采取行动的次数。人们更有可能更快地看到新的推荐，因为我们从他们没有做的事情中学到了东西。

这是个好主意。通常，我不想明确地点击“X”框来从我的推荐中删除它们，我只是想要新的。现在，Twitter 会根据用户向你展示其他内容的次数以及你是否点击它们来更快地提供这些内容。自从他们做了这次更新后，我又一次添加了更多的人。

这次更新尤其重要，因为除了新的 twitter.com 更新，Twitter 现在在右边栏显示四个建议，而不是两个。如果这四条建议不新鲜，它很快就会过时。在新的 twitter.com 发布会后，当我们见面时，我向 Twitter 的产品负责人詹森·高德曼提出了这个问题，他承认，如果他们打算继续向人们推荐，他们需要把这些推荐做得更好。这是迈向这一目标的第一步。

也就是说，至少有折叠该区域的选项还是不错的。或者能够在右窗格中移动项目。但是推特不会打破明显有效的方式。在新 twitter.com 的[发布会](https://web.archive.org/web/20221207045644/https://beta.techcrunch.com/2010/09/14/twitter-event/)上，首席执行官伊万·威廉姆斯表示，新的推荐系统到目前为止运行良好。他指出，“*我们才刚刚开始。*

Twitter 仍然在向所有用户推出新版本的过程中，但推荐更新是独立的，所以每个人都应该看到改进(当功能工作时)。

顺便说一下，Bing 刚刚推出了基于搜索推荐 Twitter 关注者的更新。Twitter 告诉我们，这完全是在 Bing 端完成的，与 Twitter 的关注谁功能无关。
