# Twitter 没有失败，社区欢欣鼓舞

当 Twitter [采取激烈措施](https://web.archive.org/web/20230213030339/https://techcrunch.com/2008/06/08/twitter-tempts-fate/)试图在一天的繁忙流量中生存下来(今天的史蒂夫·乔布斯[主题演讲](https://web.archive.org/web/20230213030339/https://techcrunch.com/2008/06/09/the-games-begin-live-coverage-of-apple-wwwc-event-in-san-francisco/)在 WWDC)，我说*“…他们可能会赢得这一天。如果他们这样做了，我们会保持沉默，如果他们失败了，我们会发布一篇无情的博客。”虽然有些评论者认为这对我不公平，但在一个脆弱的时刻，我随后承诺给 Twitter 一个高尔夫球掌声，他们一整天都没有中断。*

嗯，他们[做到了](https://web.archive.org/web/20230213030339/http://blog.twitter.com/2008/06/we-made-it.html)，我为他们鼓掌。公平地说，他们确实有一些小的停机时间(按 Twitter 的标准来说很小)——4%的请求失败。但它足够接近，可以称之为胜利，我在此给他们应得的。

Twitter 失败了失败了，今天我是一个快乐的用户。

我不得不把它交给那些家伙——他们是我所知道的唯一一个让用户感到高兴的服务，当他们设法让他们的服务保持活跃的时候。
