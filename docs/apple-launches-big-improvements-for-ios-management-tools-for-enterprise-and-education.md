# 苹果在面向企业和教育的 iOS 管理工具方面做出重大改进 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/26/apple-launches-big-improvements-for-ios-management-tools-for-enterprise-and-education/>

苹果在 IT 部门一直很忙。今天，它发布了一系列针对教育和企业客户的大规模部署工具的改进和变更。

一些调整在部署策略的杂草中消失了，所以它们只会让真正处理这些东西的专业人员感到兴奋。但总体情况是，苹果正在寻求让大规模部署对负责购买和维护 iPhones 和 iPads 的人更具吸引力。希望这将导致更多的大型组织购买和更少的失败的管理场景的负面故事。

苹果今天在其 [IT 部署页面](https://web.archive.org/web/20221205192137/http://www.apple.com/iphone/business/it/deployment.html)上发布了一系列文件，概述了这些变化。其中包括对其设备注册计划、批量购买计划和 Apple ID for Students 服务的更改。新的 iOS [部署技术参考指南](https://web.archive.org/web/20221205192137/http://images.apple.com/iphone/business/docs/iOS_Deployment_Technical_Reference_EN_Feb14.pdf)和[设备注册计划指南](https://web.archive.org/web/20221205192137/http://images.apple.com/iphone/business/docs/DEP_Business_Guide_EN_Feb14.pdf)已经发布，用新的设备管理选项更新了旧版本。一份新的[概述文档](https://web.archive.org/web/20221205192137/http://images.apple.com/iphone/business/docs/iOS_Enteprise_Deployment_Overview_EN_Feb14.pdf)为希望利用苹果提供的工具批量购买应用程序并一次性推出大量 iOS 设备的企业人员提供了一份快速备忘单。一份新的 iOS 安全文档[已经发布，提供了关于触控 ID 和 A7 的安全飞地如何工作的深入细节。](https://web.archive.org/web/20221205192137/http://images.apple.com/iphone/business/docs/iOS_Security_Feb14.pdf)

我曾与 IT 专业人士和过去大量安装 iPad 的人交谈过，他们提到 Windows 设备继续保持吸引力的原因之一是它们更好的远程安装和配置支持。为此，苹果做了一些更新来改善这种情况。

企业和教育计划现在都支持移动设备管理免提配置。这种“零接触”设置是许多专业人士长期以来一直要求的功能，因为它消除了通过苹果的配置器实用程序为每个部署的设备布线和安装描述文件的需要。

零接触配置允许组织注册一个配置文件并直接从 Apple 订购设备，这些设备预先配置了所有安全和配置选项。然后，他们可以直接管理它们，而不必实际接触它们。此外，将这些配置文件锁定到设备的能力意味着最终用户将无法删除它们，也无法随心所欲地对设备进行操作。

通过这种方式，IT 部门在部署阶段的负担得以减轻，用户能够定制和使用设备，而不会搞乱安全配置文件。它还将允许 It 部门通过简单的选择加入，轻松地以无线方式使用户自己的设备符合其公司的安全政策。

苏格兰 Cedars School of Excellence 的计算和 it 主管 Fraser Speirs 向 TechCrunch 指出:“另一个有趣的地方是，DEP 注册现在允许管理员在他们自己的设备上要求并强制注册 MDM。“这对学校来说意义重大。根据我的猜测，MDM 注册缺乏可执行性是如今臭名昭著的洛杉矶联合学区‘黑客’丑闻的核心问题。”

洛杉矶的学校已经部署了大量的 iPads，但在发现学生删除他们设备上的注册资料后，不得不[召回它们。这使得学生们可以随心所欲地使用它们浏览网页，并在他们的个人设备上安装未经批准的应用程序。无政府状态无法忍受，该地区开始重新考虑它的推广——T4 已经恢复了。这种高调的失误可能对苹果公司组织推出的势头没有任何帮助。这种强制应该允许定制，同时防止规避 IT 规则。](https://web.archive.org/web/20221205192137/http://www.latimes.com/local/lanow/la-me-ln-lausd-ipad-hack-20130925,0,6974454.story)

简而言之，这会让 iPads 对这些市场更具吸引力。

批量购买计划也得到了很好的检修。现在，它在所有 10 个 VPP 国家面向企业和教育合作伙伴提供，允许这些组织批量购买应用程序，一次安装在多台设备上。这些采购通常有很大的折扣，这使得它们对预算紧张的学校更有吸引力。

此外，企业客户现在可以通过采购订单购买 VPP，而这在过去只有 IT 专业人员才能做到。向 IT 部门提供采购订单支持将向以前无法从苹果公司购买软件的庞大市场群体开放 VPP 采购。

具体到教育市场，苹果开放了 13 岁以下学生获取自己 Apple ID 的能力。在学校注册后，他们可以向苹果公司申请 id，苹果公司将向家长发送信息，家长将在注册过程中得到指导。然后通知学校，学生已经同意。

虽然 13 岁以下的 id 在去年年底开放，但新的变化为学生开放了道路，特别是以符合 COPPA 的方式注册。首先，他们需要父母同意才能注册。而且这些 Apple IDs 都有一套特定的限制方式。

例如，学生 Apple ID 帐户的广告跟踪功能有限，不允许他们选择加入营销，并且默认情况下没有 iCloud 电子邮件支持。他们也不需要信用卡来注册，并且可以通知父母或监护人账户条款的任何变化。一旦学生年满 13 岁，这些帐户将转换为拥有所有正常权利和特权的完整帐户。苹果[提供了一份新的家长指南](https://web.archive.org/web/20221205192137/http://images.apple.com/education/docs/Apple_ID_for_Students_Parent_Guide_EN_Feb14.pdf)，详细说明了这些特殊苹果 id 的确切流程和津贴，以及一份针对将使用它们的机构的[指南。](//web.archive.org/web/20221205192137/http://images.apple.com/education/docs/u13_institution_guide_en__Feb14.pdf)

“我认为最重要的是，这些变化都是为了使部署规模扩大到真正的大规模，”斯皮尔说。“据我所知，学区目前最多有大约 5 万个单元，但在教育和企业的其他领域显然有更大的部署。”

目前，我们已经确认设备注册计划将只影响组织直接从 Apple 购买的设备。这意味着像斯皮尔这样通过零售商工作的学校，还不能使用该程序来预配置设备。但这很可能正在进行中。

Yankee Group 研究和数据科学副总裁 Carl Howe 告诉 TechCrunch:“我确实认为这些是 IT 组织向前迈出的重要一步。“在许多组织中，IT 部门大部分时间都是‘拒绝部门’，他们告诉员工，出于可管理性或安全性的原因，他们不能做某些事情。通过提高苹果设备的可管理性和安全性的透明度，苹果正在帮助这些 IT 组织从“说不”的部门变成“说是”的部门。鉴于易用性提高了安全性合规性，事实上，这可以通过不需要最终用户做任何特殊操作的方式来实现(即，他们的设备被调配给他们，完全链接到他们的组织，受到保护等)。)，它应该会让苹果设备在商业中比现在更受欢迎。"

通过利用自带设备的趋势，iOS 设备在组织中获得了很大的普及。如果老板得到了一个新的 iPad 并且很喜欢它，it 部门将别无选择，只能以某种方式支持它。但为了刺激增长，并保持 iPads 和 iPhones 的易用性，苹果必须继续完善其设备部署产品。

这些变化试图让人们使用机构 iOS 设备和消费者 iOS 设备的方式尽可能接近平等。如果你曾经千辛万苦地配置过公司配发的笔记本电脑或手机，那么你就会知道这有多痛苦。如果你想象这样做 50，000 次，你就能明白为什么这些改进会刺激采购部门的采用。
