# Flipboard 在 Android 上正式推出，增加了 Google+、YouTube 和更多本地化版本 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/06/21/flipboard-officially-launches-on-android-adds-google-youtube-and-more-localized-versions/>

对于 Flipboard 的谷歌相关公告来说，这是重要的一周。周二，该公司透露其社交杂志将整合 Google+,今天，该应用正式在所有 Android 设备上推出，包括 Kindle Fire 和 Nook。Google+集成以及新添加的 YouTube 集成也将于今天发布。

该应用程序将在 Google Play、亚马逊 Android 应用商店、Nook 商店和三星应用程序中提供。

在 Flipboard 首次推出其[官方安卓测试版](https://web.archive.org/web/20221209024755/https://beta.techcrunch.com/2012/05/30/flipboard-officially-opens-up-their-android-beta-to-interested-testers/)和作为刚刚发布的三星 Galaxy S III 上的预装应用在[有限推出安卓之后不到一个月，安卓公开发布。然而，由于来自同一设备的安卓 APK 文件的黑客版本，自](https://web.archive.org/web/20221209024755/https://beta.techcrunch.com/2012/05/30/flipboard-for-android-beta/) [Flipboard 的非官方发布](https://web.archive.org/web/20221209024755/https://beta.techcrunch.com/2012/05/09/flipboard-unofficially-available-for-any-android-device-and-heres-how-to-install-it/)以来，时间稍微长了一点。

Flipboard 首席执行官迈克·麦库埃说，对 Android 版应用程序的需求“令人惊讶”，他告诉我们，在测试期间，他们看到了超过 50 万次下载。“我们在各种手机上都有很好的覆盖。我们对这一切的进展感到非常满意，”他说。“但人们一直在问，为什么这是测试版？为什么不能现在就放生？”

答案当然是因为它是一款安卓应用，这意味着[需要做更多的工作](https://web.archive.org/web/20221209024755/https://beta.techcrunch.com/2012/05/11/this-is-what-developing-for-android-looks-like/)。“有了 iOS，你可以瞄准一个非常具体的屏幕尺寸，一个非常具体的处理器等等——这要容易得多，”麦库埃解释道。“在 Android 上，要真正构建一个适用于一系列设备的高质量应用，你必须花大量时间进行优化……这就是我们进行测试的原因。”新的 Android 版本可以在所有主要的 Android 智能手机以及 Nook 和 Kindle Fire 上运行，支持从小屏幕到 7 英寸的显示器。自测试版发布以来，它还进行了一些其他的性能和布局优化。

## YouTube 发布和更多本地化版本

[![](img/6b3d43765054d6281eae6856cd20e424.png "Android_TOC_1280")](https://web.archive.org/web/20221209024755/https://beta.techcrunch.com/2012/06/21/flipboard-officially-launches-on-android-adds-google-youtube-and-more-localized-versions/android_toc_1280/) 今天，iOS 和 Android 上的另一个新功能是 YouTube 集成。你可以在 Flipboard 中观看、收藏、标记和评论视频，这些更改会与 YouTube 网站本身同步。还可以关注用户和订阅的频道。

Flipboard 也再次扩展到更多国家。它之前在中国、日本和法国有本地化版本，但现在在韩国、荷兰、意大利、西班牙和德国也有了。“当我们在每个国家部署时，我们不仅针对每种不同的语言对应用进行本地化，更重要的是，我们将精力集中在每个国家的监管角度，以挑选最佳内容。“我们现在在这方面效率更高了，”麦库埃说。本地化列表上的下一个目标是拉丁美洲的部分地区，以及亚洲和欧洲的其他国家。

## 查看 Flipboard 的业务

虽然该公司仍然拒绝透露与总下载量或收入相关的硬性数据，但麦库埃表示，向智能手机的转移使用户全天都在使用该应用程序，而不是像在 iPad 上那样只在早上和晚上(主要是晚上 8-11 点)使用。封面故事功能首先在智能手机上推出，现在可以通过 Android widget 获得，现在是 Flipboard 上的头号种子。他还表示，Flipboard 看到了“非常、非常、非常强的应用回报指标。”

“每天都有很多人选择我们，每天都有很多次。这对我们来说是非常鼓舞人心的，特别是因为我们没有向用户发送太多电子邮件，也没有发出太多嘈杂的通知，”麦库埃说。然而，集成推送通知是该团队正在考虑的未来的事情，但他们希望在时机成熟时做得正确。

他还报告称，该业务在广告方面取得了良好进展，目前已有 6 家出版商在该平台上运营广告，迄今为止已整合的出版物总数为 2000 种。最近的一些广告是匡威、尼曼·马库斯(麦库埃说，和《时尚》杂志上的广告一样)，以及电影《白雪公主和猎人》最后一个表现特别好，点击率为 3.5%。它没有将用户引导到网站，而是引导到一个应用内页面，该页面以 Flipboard 的签名杂志风格展示了“白雪公主和猎人”的 Twitter 订阅，用户可以在应用中订阅。然而，所有 Flipboard 的点击率“都比你在网上看到的要高得多”，麦库埃说。

至于下一步，除了持续的国际扩张之外，一本如此专注于吸引人的视觉内容布局的杂志明显忽略了一点，那就是 Pinterest 整合。Pinterest 的竞争对手 The Fancy 的整合在 6 个月前完成，但尚未发货，因此停滞显然是在 Pinterest 这边。但是 Flipboard 并不是唯一迫切需要 Pinterest API 的第三方。(那东西已经在哪里了？)官方没有透露下一步会加入哪些网络，但是如果 Pinterest 到来的话，它肯定会引起轰动。
