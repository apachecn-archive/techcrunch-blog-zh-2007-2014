# 脸书会成为明天的谷歌，谷歌会成为明天的微软吗？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/05/15/facebook-google/>

![](img/d126a7697a1be1762aab894df2a7ce2e.png)

***编者按**:脸书能成为下一个谷歌吗？在这篇客座博文中，前谷歌员工[杜宾·雷迪](https://web.archive.org/web/20221007034541/http://www.crunchbase.com/person/bindu-reddy)，从脸书作为在线广告平台的潜力角度，令人信服地论证了这个案例。雷迪是由其他前谷歌员工资助的口碑广告网络 MyLikes 的首席执行官。在谷歌，她管理着一个产品经理团队，负责各种谷歌应用，包括谷歌文档、谷歌网站和博客。*

今天，如果你想寻找几乎任何东西的信息，谷歌是一个可以去的地方。通过显示与你正在寻找的内容相关的赞助链接，谷歌向我们展示了广告在有用的时候最有效。如此有效，以至于谷歌在过去十年里建立了价值 250 亿美元的搜索广告业务。

然而，谷歌的搜索广告业务天生就受到一个事实的制约，即它只有在用户已经在寻找某样东西时才能发挥作用。你必须先搜索化妆品，然后谷歌才能为迪奥最新的睫毛膏产品发布广告。

另一方面，脸书已经成为世界上的身份守门人——你的年龄、性别、所在地、你在哪里上学、你在哪里工作、你的朋友是谁——所有这些个人数据都被用来为你提供你可能感兴趣的信息。想看看你表妹婚礼的照片吗？想知道你的同事这个周末在看什么电影？你的朋友“喜欢”什么音乐？你需要去脸书。底线是你现在被训练去脸书发现事物。随着脸书应用平台的发展以及 Farmville 和 Mafia Wars 等应用的支持(到目前为止)，脸书也已经成为网络上娱乐和休闲的头号目的地。

就像谷歌引入赞助搜索作为与搜索结果无缝契合的广告模式一样，不需要太多想象力，就能看出脸书如何将基于品牌和发现的广告融入其产品，这对会员有益。像谷歌的 AdWords 一样，当脸书的广告被整合到核心产品中，并且与用户非常相关时，它们将是最有效的。

![](img/9da7eb32a8515046620734e6c475a3b3.png)

以右边索尼的赞助广告为例。这是一个我可以与之互动的赞助投票小工具。很像一个迷你的脸书应用程序，它用一个调查来娱乐我，同时确保我与品牌互动。随着营销人员想出越来越聪明的方法来吸引我们的注意力，创新的广告形式正在脸书的 T2 迅速发展。脸书发现如何在脸书进行有效的品牌广告只是时间问题。

与 Google AdWords 不同，这种模式不会受到你必须实际寻找某样东西的限制。你不必搜索化妆品，只需登录脸书，成为一名年轻的时尚女性，就能发现迪奥睫毛膏。迪奥可以通过这种方式接触到更多的潜在客户，而不仅仅是在谷歌搜索上做广告。

凭借有效和有用的发现广告模式，脸书是第一家有很大机会获得传统电视和线下品牌广告市场重要份额的网络公司(估计约为[【1320 亿美元】](https://web.archive.org/web/20221007034541/http://online.wsj.com/article/SB124172645603997429.html))。鉴于搜索广告仍然只占全部广告收入的 6.25%(250 亿美元，4580 亿美元)，脸书很可能在未来十年超过谷歌的广告收入和市值。

我们会看到过去十年的前五年重演吗？当时谷歌迅速成长，取代微软成为顶级科技公司。这一次可能会轮到脸书取代谷歌，目前还不清楚谷歌对此能做些什么。原因如下:

**谷歌没有获得品牌/发现广告**

谷歌的广告商业模式是基于意图和相关性，而不是发现。基于业绩的 AdWords 和 AdSense 模型更容易衡量，对谷歌的逻辑/分析头脑有吸引力。影响力、发现和品牌广告的力量需要比谷歌左脑思考者更多的右脑思考。

此外，谷歌过去几年在该领域的策略是简单地收购老派品牌广告(主要是横幅广告)的领导者 DoubleClick，而不是创新和探索新的品牌广告形式。这基本上意味着，与理解发现和推荐力量的脸书相比，谷歌在这一领域没有新的思维。

**谷歌是一家伟大的科技公司，也是一家平庸的产品公司**

正如谷歌欣然承认的那样，谷歌最有权力的人是工程师。像谷歌的其他非工程组织一样，产品管理更多的是一个服务组织。这实质上意味着谷歌擅长技术含量高的产品，而在产品含量高的产品上表现平平。

给谷歌一个技术挑战——建立最大的搜索索引(谷歌搜索)、最大的存储系统(Gmail)、最快的浏览器(Chrome)或最简洁的 javascript 界面(地图)和谷歌的优势。转过身来，给它一个产品挑战——建立一个社区视频网站(谷歌视频)、一个社交网络(谷歌 Wave、Buzz 和 Orkut)、一个电子商务平台(谷歌产品搜索)，谷歌的产品与其说优秀，不如说平庸。

谷歌的技术基础设施，是为大规模数据集而不是快速迭代优化的。它的工程师喜欢研究更好的算法和大型系统，而不是理解快速发展的用户生成社区网站背后的社会行为/经济(可能 YouTube 的人除外)。

**在核心产品方面，谷歌有一个增量产品战略。**

不像脸书，不断地对其核心产品做出激烈的[改变](https://web.archive.org/web/20221007034541/https://beta.techcrunch.com/2010/01/04/facebook-rolling-out-redesign-to-some-users/)，甚至冒着[惹恼](https://web.archive.org/web/20221007034541/https://beta.techcrunch.com/2010/05/11/facebook-social-plugins/)其中一些[用户](https://web.archive.org/web/20221007034541/https://beta.techcrunch.com/2009/03/19/facebook-polls-users-on-redesign-94-hate-it/)的[风险](https://web.archive.org/web/20221007034541/https://beta.techcrunch.com/2010/05/07/facebook%E2%80%99s-disconnect-open-doors-closed-exits/)，谷歌在搜索和 Gmail 等核心应用上的策略很大程度上是数据驱动和增量的。

谷歌无法利用其核心应用程序，通过这种渐进式方法与脸书竞争。直到最近，随着 Buzz 成为 GMail 的一部分，左导航条成为谷歌搜索的一部分，谷歌才开始采取更大胆的举措。然而，与 Buzz 一样，这些变化似乎太少太晚，而且不清楚他们是否有必要的资金坚持到底。

鉴于此，我非常担心谷歌是否有可能失去其主导地位。你怎么想呢?脸书会出色地执行并成为明天的谷歌吗？或者谷歌会像微软对网景公司那样醒来并战胜脸书吗？
