# 十字路口的硅谷:娱乐还是科学？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/10/05/silicon-valley-at-a-cross-roads-entertainment-or-science/>

[![](img/6d4f5b190e1c2ef6276f39da13e606e0.png "fork in road")](https://web.archive.org/web/20221209021928/https://beta.techcrunch.com/wp-content/uploads/2010/10/fork-in-road.jpg) 周日，我在写了一篇[帖子，讲述了硅谷的文化浪潮和由此产生的各个公司之间的区别。许多单个公司幸存下来，但浪潮本身总是会崩溃。这就是波浪的本质。但这也是波浪的本性，总会有另一个波浪出现。我不知道那会是什么。毕竟，十年前，硅谷最聪明的人认为社交网络是一项功能，而不是一项业务。](https://web.archive.org/web/20221209021928/https://beta.techcrunch.com/2010/10/03/if-web-1-0%E2%80%99s-kryptonite-was-the-bust-web-2-0-kryptonite-was-the-grind/)

但很明显，硅谷正处于十字路口:它会沿着 Web 2.0 的道路在媒体和娱乐的道路上走得更远，还是回到它的核心科学和技术的根源？

几十年来，硅谷的浪潮顺理成章地接踵而至。半导体的创新导致计算机的创新，计算机的创新导致个人和企业软件的创新。这导致了通过以太网、网络、电信和大量服务器、交换机和路由器将所有这些东西连接起来的愿望。最重要的是，我们建立了第一批功能强大的网络公司——这些网站使用了硅谷已经建立的技术，使我们能够更便宜、更容易地购买机票。

但在 21 世纪初，硅谷正处于十字路口。未来是清洁技术还是 Web 2.0？换句话说，未来是开始一个新的堆栈，还是继续在大部分已构建好的堆栈上迭代？清洁技术阵营得到了许多硅谷名人的支持，其中最著名的是约翰·杜尔登，他们认为这是改变世界的最重要的方式，也是一个不可否认的巨大市场，需要真正的创新。Web 2.0 阵营主要由雷德·霍夫曼和彼得·泰尔等 PayPal 黑手党的天使投资人以及马克·安德森等少数其他天使和导师支持。他们中的许多人珍视这样一个事实，即网络公司是清洁技术的对立面:由于几十年的技术发展，它们开始时价格低廉，技术含量低，彻底改变了这些公司的一切，从你需要有良好回报的出口规模，到你能多快知道你有没有东西。

最终，Web 2.0 赢得了硅谷的青睐和大肆宣传，而且由于一些巨头的存在，它很可能会赢得丰厚的回报。原因有很多解释。对于通常回避资助大型科学项目的风险投资来说，这可能是一个更好的机会。可能是因为这是一个更容易的机会。这可能是因为持有支票簿的人对此更加兴奋。这可能是因为政府没有尽自己的职责资助更多的清洁技术研究，或者是因为政府没有尽自己的职责创造补贴来刺激替代能源市场的早期发展。也可能有一个清洁技术的脸书，只是需要更长的时间来建设和证明自己，我们还不知道。但有一点可以肯定:清洁技术是硅谷几十年来第一个创新成熟的巨大市场，在这个市场上，硅谷失去了领导地位。几乎以任何标准衡量，世界其他地方都有更大的清洁技术市场和创新。

我上周在 Disrupt 上的一些主题演讲和后台对话强调，硅谷正处于另一个十字路口。这是上一个问题的更夸张的版本:是回归基于真实的、颠覆性科学的公司，还是进一步陷入网络媒体和娱乐的兔子洞。

彼得·泰尔说得最引人注目，当他——这个将在脸书的 50 万美元投资变成 30 亿美元投资的人，这个相信当很少人相信互联网仍然是一个巨大机会的人——说从风险投资的角度来看，互联网作为一个类别已经死亡。第二天，维诺德·科斯拉通过[嘲笑这个想法](https://web.archive.org/web/20221209021928/https://beta.techcrunch.com/2010/10/01/vinod-khosla-ceos-should-spend-more-than-50-of-their-time-recruiting-tctv/)来支持这个观点，即一个名为“Disrupt”的会议将会有微软下一版本 Windows 的演示。就连埃里克·施密特也带着对自动驾驶汽车的渴望加入了未来派的行列。另一方面，在周一下午的比赛中，我们有一群初创公司，它们基本上都是 Foursquare 上的迭代。公司依靠脸书或 iPhone 的成功是一回事，但 Foursquare 甚至还没有证明它能成为主流。

这不是零和游戏。这两次浪潮都将找到资金——毕竟在 2000 年代中期，投入清洁技术的资金比 Web 2.0 更多。值得注意的是，这两种浪潮也无情地交织在一起。昨天，我听了 Draper Fisher Jurvetson 的硅谷蛋头 Steve Jurvetson 令人兴奋、充满科学的主题演讲，他将围绕能源和水的创新描述为风险投资所见过的第一个*万亿美元*的机会。后来我问他，硅谷将如何解决这些问题。我们还会统治吗？答案是否定的，但这并不是因为硅谷的创新越来越少。相反，更多的创新正在其他地方发生，他说。他补充说，许多被人们嘲笑为不够颠覆性的东西——特别是 Skype、Gmail、脸书、Twitter 等应用——在团结世界各地的科技精英方面发挥了巨大作用，使全球颠覆性的激增成为可能。

但在未来十年，一条道路可能会主导硅谷的时代精神、影响和回报。你没看过清洁技术电影吧？

我会投泰尔和科斯拉的科学票。仿生学、人工智能、太空旅行、农业、能源和材料科学等领域正在酝酿许多令人惊叹的创新——其中一些创新之所以成为可能，很大程度上是因为构建了计算堆栈。但我猜 TechCrunch 的大多数观众和我的许多同事会选择另一条路。旧金山成为网络媒体的中心，就像纽约是新闻媒体的中心，洛杉矶是娱乐媒体的中心。这些业务更容易理解，评估更快，更有可能产生一系列有新闻价值的博客文章。我这么说并不是作为一种价值判断——我写过一本关于 Web 2.0 的书。这是硅谷最有趣、最性感的地方，也是大众观众最关心的地方。

硅谷在一波又一波浪潮中如此有弹性的原因是它的生态系统，该生态系统中的四个主要群体将在决定哪一波浪潮中获胜方面发挥作用。

**首先是媒体。写关于科学的文章并不容易，也不流行。一部分责任在我们身上，但是博客是一个社区。我写的关于科学和其他冷门话题的帖子，比如来自发展中国家贫民窟和村庄的创新，通常不会像关于最新的 YCombinator 公司的帖子那样带来相同的流量或参与度。如果页面浏览量没有了，想写这些东西的记者会得到支持吗？**

**第二类是聪明的工程师和有才华的高管。他们想在这样的公司工作吗？还是他们希望成为下一个脸书、Zynga、Groupon 或 Twitter——在这些公司里，你可以获得接触数百万人的即时满足感，而且越来越不必等待 IPO 或收购来获得意外之财？**

**第三是投资者。那是我最不担心的。抛开对新媒体、超级天使和空翻的大肆宣传不谈，硅谷仍有数十亿美元从未在消费互联网上获得巨大成功，也很快会支持一些拥有专有科学的东西。硅谷的一些名人正在支持这样的公司，无论他们是像科斯拉和泰尔这样的超级明星投资者，还是像埃隆·马斯克和杰夫·贝索斯这样的超级明星企业家，以及他们对太空旅行的押注。**

大多数情况下，这取决于第四个因素:企业家。最终，这就是我们围绕着的人。你可以说我是理想主义者，但我仍然相信让硅谷成为今天这个样子的核心组织结构:伟大、坚定的企业家扰乱了一个大市场，他们将获得资金，其余的人也会跟上——也许不会像网络一样快，但无论如何都会跟上。
