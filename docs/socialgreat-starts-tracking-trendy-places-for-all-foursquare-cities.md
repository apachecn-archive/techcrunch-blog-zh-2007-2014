# SocialGreat 开始追踪所有四方城市的时尚地点

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/08/04/socialgreat-starts-tracking-trendy-places-for-all-foursquare-cities/>

![picture-211](img/6bec5e370e9da7d91b6acf1cc504f1a6.png "picture-211")我们已经写了关于[方](https://web.archive.org/web/20230308122507/http://playfoursquare.com/)一个[数](https://web.archive.org/web/20230308122507/https://techcrunch.com/2009/03/18/sxsw-foursquare-scores-despite-its-flaws/)的[倍](https://web.archive.org/web/20230308122507/https://techcrunch.com/2009/07/16/foursquare-shows-the-business-potential-of-location-based-services/)。这是一个非常好的工具，可以追踪你的朋友在哪里，同时还可以玩这个奇怪的竞争性社交游戏。作为一个单机 app，很棒。但是它收集的数据可能同样有趣，社交网站是最先利用这些数据的网站之一。

SocialGreat 背后的想法非常简单:显示特定时间段内城市中最受欢迎的地方。正如标语所说，“人群在哪里？”但这就是为什么它比常规的评级系统更好:你用脚投票。比如，如果你去一个地方，并在那里登记，它会在排行榜上得到一分。

几周前，这项服务在纽约市推出，上周又增加了旧金山。这使得它能够跟踪近 3000 人的移动，这提供了一些关于群体如何从一个地方迁移到另一个地方的有趣数据。但是从今天开始，Foursquare 在所有城市都可以使用。这无疑意味着更多的数据点，甚至更多有趣的信息。

SocialGreat [使用 Foursquare 的 API](https://web.archive.org/web/20230308122507/https://techcrunch.com/2009/05/29/foursquare-to-serve-up-api-more-mobile-apps-free-beer/) 来获取你的信息。当你访问这个网站时，点击“加入”按钮，你就会被带到一个 Foursquare 页面，允许你的数据通过 OAuth 发送。然后，您的数据将被输入信息库。

![picture-20](img/6c703113ecaa1947bb6f3ac2dc4f83cd.png "picture-20")

SocialGreat 的主页很简单:你可以根据小时、天、周或所有时间跟踪热门地点。一个列表告诉你有多少人去过那个地方，以及最近人们去或不去那里的趋势(分别用正数或负数表示)。在这个列表的右边，所有受欢迎的地方都显示在地图上。从那里你可以点击他们中的任何一个来得到他们的地址。

一个不利之处是很多人在机场办理登机手续。这就是为什么在上面的图片中你看到“旧金山”领先(它实际上是 SFO，机场)。纽约也是如此。不过，如果服务愿意的话，这应该很容易被过滤掉。

正如我所说的，这是一个简单的想法和应用，但它可能是一个非常好的想法，可以根据一个非常真实的指标来跟踪流行的地方——人们实际上会去那里。当然，注册和使用 Foursquare 的人越多，它就会越有用，反过来允许 SocialGreat 访问他们的数据。

SocialGreat 是 Pepper 莉莉艾的 Bill Piel 和 Google 的 Jon Steinberg 的创意(这有点有趣，因为 Foursquare 本身是类似的，[的 Dodgeball 的后续服务，Google 在 2005 年收购了它，却让它死去)。这两人还得到了 Drop.io 首席执行官](https://web.archive.org/web/20230308122507/http://digital.venturebeat.com/2009/03/10/dodgeball-founder-pegs-google-in-the-face-with-foursquare/)[萨姆·莱辛](https://web.archive.org/web/20230308122507/http://www.crunchbase.com/person/sam-lessin)的帮助。

**更新**:正如 Piel 在评论中指出的，所有的城市还没有完全出现在网站的前端(明天就会出现)，但是你可以很容易地找到你要找的城市的网址。
