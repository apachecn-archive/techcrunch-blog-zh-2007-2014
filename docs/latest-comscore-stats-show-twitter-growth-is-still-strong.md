# 最新的 comScore 数据显示 Twitter 的增长仍然强劲

在线分析公司 [comScore](https://web.archive.org/web/20230404020310/http://www.comscore.com/) 发布了最新一批数据，本月的结果显示 Twitter 仍然表现出强劲的增长。根据 comScore 的数据，Twitter 全球独立访客从 4 月份的 8380 万增长到 5 月份的 9020 万，增长了 7.6%。相比之下，3 月份 Twitter 有 7940 万用户，4 月份有 8380 万用户，增长了 5.5%。一年前 comScore 显示 Twitter 有 3730 万用户。

需要注意的是，这些数字不一定能很好地衡量使用 Twitter 平台的总人数(它只包括去过 Twitter.com 的人的数据，不包括任何通过 API 访问 Twitter 的应用程序)。上周，推特首席运营官·迪克·科斯特罗[分享了公司的内部统计数据，显示推特每月有 1 . 9 亿访客，每天有 6500 万条推文。另一个警告:并不是每个访问 Twitter.com 的人都在发微博(有些人只是被动地使用这项服务)。](https://web.archive.org/web/20230404020310/https://techcrunch.com/2010/06/08/twitter-190-million-users/)

尽管这些数字之间明显存在差异，但 comScore 的数据可以用来追踪增长趋势。在经历了冬天的平静之后，Twitter 显然又回到了正轨。
