# 盒子里的维基解密:SecureDrop 是媒体的告密者交流工具

为了保护政府告密者免受前所未有的监视，新闻自由基金会推出了一款匿名提交工具 [SecureDrop](https://web.archive.org/web/20230210043947/https://pressfreedomfoundation.org/securedrop#faq) ，用于消息来源和记者之间的安全通信。SecureDrop 接受加密文件和消息来源的提示，并促进沟通，而不会使记者处于不得不在监禁威胁下透露消息来源的危险之中。

鉴于奥巴马政府根据《反间谍法》积极起诉泄密者，对安全的需求有所增加。例如，去年春天，司法部截获了美联社记者[的电话记录，这些记者参与了报道也门一起被挫败的炸弹阴谋。](https://web.archive.org/web/20230210043947/http://washington.cbslocal.com/2013/06/19/ap-ceo-justice-dept-suppressed-media-to-comfort-authoritarian-regime/)

“奥巴马政府起诉这么多举报者的原因之一是有一个简单的方法来找到记者如何会见消息来源并与他们交谈的数字线索，”[说](https://web.archive.org/web/20230210043947/http://www.forbes.com/sites/andygreenberg/2013/10/15/securedrop-project-will-pay-to-install-media-outlets-wikileaks-style-submission-systems/)新闻自由基金会执行董事埃沃·蒂姆。“我们需要找到一种方法，让记者在没有这种恐惧的情况下与消息来源交谈。”

SecureDrop 最初是[堕落黑客主义者艾伦·施瓦茨](https://web.archive.org/web/20230210043947/https://techcrunch.com/2013/01/14/aaron-swartz-asking-for-help-119-days-ago/)的项目(当时叫做 DeadDrop)。该项目已被更新，以考虑到最近国家安全局的间谍活动，尽管该组织提醒记者，没有什么是 100%安全的。代码库是开源的，已经通过了华盛顿大学安全专家的审查。

新闻自由基金会甚至提出帮助媒体安装相当复杂的加密工具。点击了解更多信息[。](https://web.archive.org/web/20230210043947/https://pressfreedomfoundation.org/securedrop#faq)
