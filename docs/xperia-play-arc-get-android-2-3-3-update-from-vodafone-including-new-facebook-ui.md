# Xperia Play，Arc 从沃达丰获得 Android 2.3.3 更新，包括新的脸书用户界面

![](img/c2e9852ceabe34e871df10d87a033f53.png "facebookinsidexperia")

[索尼爱立信](https://web.archive.org/web/20230205043600/http://crunchgear.com/tag/se)的 Xperia Play 和 Arc 智能手机将于今天早上更新到 [Android 2.3.3 姜饼](https://web.archive.org/web/20230205043600/http://crunchgear.com/tag/android)，但只针对我们在沃达丰网络上的海外朋友。抱歉，Verizonites 用户，但我确信你的 Xperia 手机很快就会升级到 2.3.3 版本。与此同时，[沃达丰](https://web.archive.org/web/20230205043600/http://crunchgear.com/tag/vodafone) Play 和 Arc 用户可以期待一些错误修复，以及索尼爱立信新的脸书集成 Xperia 中的脸书。

据沃达丰称，Android 2.3.3 更新将带来语音质量的改善、稳定性的提高以及电子邮件设置和 exchange 帐户的改进。但是这次更新的真正核心在于脸书的整合。[正如我们之前报道的](https://web.archive.org/web/20230205043600/http://www.mobilecrunch.com/2011/05/20/android-2-3-3-gingerbread-coming-to-xperia-play-arc-next-week/)，Xperia 内部的脸书基本上将社交网络置于你最常用的应用程序的核心，如通讯录、相机和媒体库应用程序以及音乐播放器。在那里，你可以把推荐的歌曲贴在墙上，“喜欢”任何东西，并在你的地址簿中看到朋友的额外信息，这些信息直接来自他们在脸书的个人资料。

这是一个手动更新，所以你需要将你的设备连接到计算机，并使用索尼爱立信更新服务应用程序来进行更新。如果你本质上是一个游戏玩家，并且是一只网上交际花，Android 2.3.3 更新应该是你姜饼驱动的 Xperia Arc 和 Xperia Play 上的一层美味糖衣。
