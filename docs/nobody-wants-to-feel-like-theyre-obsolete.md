# 没人想感觉自己过时了 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/24/nobody-wants-to-feel-like-theyre-obsolete/>

> **问:**爸爸，你用什么浏览器？**答:**善于觅食的动物。**问:**好的，它叫什么？**答:**一个浏览器。

我不知道你怎么想，但我尽量避免和父母讨论技术问题。尽管他们相对来说有技术意识，但他们倾向于团队 Windows，而我是团队 Mac。此外:虽然“我的酷奶奶拥有一台 iPad”这个比喻是完全普遍和真实的，但老年人往往对他们的科技敏锐度过于敏感。我想这是对死亡的恐惧。

这就是为什么当 Alexis Madrigal [提出把黑色星期五](https://web.archive.org/web/20221007021214/http://www.theatlantic.com/technology/archive/2011/11/forget-shopping-friday-is-update-your-parents-browser-day/248933/)变成“更新你父母的浏览器日”的想法时，我退缩了，尽管我认为普遍的现代网络浏览标准是值得向往的。作为可能是记得万维网出现之前生活是什么样子的最后一代人的一部分，不得不向那些给我买了第一台电脑——你知道，是他们给了我生命——的人解释像浏览器更新这样简单的事情，这让我感到有些难过。

如果科技让我们更人性化，就像 Twitter 的杰克·多西所说的那样，那么科技的缺失会让 u 盘里的人变成不同的物种。你讨厌认为你的父母在任何方面都比你弱，更不用说使用 Internet Explorer 6。

今天早上，我几乎不敢问我的父母他们用什么浏览器，因为害怕我可能真的要庆祝明天的假期。在[傻了](https://web.archive.org/web/20221007021214/https://twitter.com/#!/alexia/status/139835113470042112)(他毕竟和我有血缘关系)之后，我爸回答了谷歌 Chrome(咻！).我用的是 Chrome Canary——在《OS X 雄狮》中就有很多问题。

对于那些阅读(更不用说为其写作)TechCrunch 网站的偏狭的科技早期采用者来说，想到世界上 7.9%的人仍在使用十年前的 IE6，这是很可怕的。升级到最新的浏览器版本的好处不仅仅是提高了速度，而且没有自安装的“BonziBuddy”式工具栏…

一位黑客新闻评论者[描述了他父亲无意中被间谍软件利用的恐惧](https://web.archive.org/web/20221007021214/http://news.ycombinator.com/item?id=3274848)“他启动了老 IE7，进入汇丰银行页面，完成了登录的第一步，就在那里，一清二楚。一个神秘的第三个框提示用户输入完整的 PIN。“不寒而栗。

他继续说，“我们，作为家庭中更有技术头脑的人，有义务为我们关心的人做这件事，以一种超越那些聪明的 CSS 动画或原生视频支持的方式。我很幸运，我的家人知道要警惕网上的任何可疑行为，但我不知道他还在使用 IE7。一想到他(或者任何人)出于完全无辜的无知，无意中把自己完整的银行信息给了一些网络罪犯，我就害怕。”

但是并不是所有的父母都愿意接受技术变革；证据另一个黑客新闻[恐怖故事](https://web.archive.org/web/20221007021214/http://news.ycombinator.com/item?id=3274848)，“有一次我来到父母家，发现家里的笔记本电脑增加了一个奇妙的功能，每个搜索结果都会重定向到间谍软件。我尽最大努力清理混乱，让一切变得合理，然后几周后接到电话，说我的“色情观看”毁了他们的电脑，并删除了他们的书签。"

为了避免这种“你破坏了我的互联网”的问题，甚至像谷歌的马特·卡茨这样的精明的技术人员也求助于~~撒谎~~ [的巧妙解决办法。](https://web.archive.org/web/20221007021214/https://twitter.com/#!/mattcutts/status/139557025259601920)

https://twitter.com/#!/mattcutts/status/139557025259601920

无论你是技术势利者还是家庭聚会上的技术 00b，有一点是明确的——我父母这一代和我这一代之间的技术鸿沟将完全被我们的孩子提出的任何感觉优越的东西所掩盖。

因此，当我未来的孩子在[“更新你的电子邮件阅读隐形眼镜日](https://web.archive.org/web/20221007021214/http://www.guardian.co.uk/commentisfree/2011/nov/22/bionic-contact-lens-emails)”或其他什么日子帮助我时，我有一条信息要告诉他们:重新阅读这篇文章的标题。
