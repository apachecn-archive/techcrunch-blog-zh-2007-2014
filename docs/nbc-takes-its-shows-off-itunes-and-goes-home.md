# NBC 把它的节目从 iTunes 上撤下，然后回家

[![itunes-tv-list.png](img/c16e7bedb0b8dafbd5083bfdd03f368f.png)](https://web.archive.org/web/20230218024708/https://techcrunch.com/wp-content/uploads/2007/12/itunes-tv-list.png "itunes-tv-list.png") 在一次非常公开的裂痕之后，NBC 和苹果[终于分道扬镳](https://web.archive.org/web/20230218024708/http://www.tuaw.com/2007/12/02/nbc-officially-removed-from-the-itunes-store/)。随着合同到期，除了一些 NBC 新闻播客外，NBC 已经从 iTunes 上撤下了所有节目。(也不再有美国消费者新闻与商业频道、科幻频道、Telemundo 或美国节目)。此举在[意料之中](https://web.archive.org/web/20230218024708/https://techcrunch.com/2007/10/29/free-from-itunes-addiction-nbc-universal-seizes-control-of-digital-destiny/)。

看到右边 iTunes 上的电视网络列表了吗？NBC 曾经紧随国家地理之后。它的缺席留下了一个非常明显的漏洞，这是一个明显的提醒，苹果无法像它拥有数字音乐那样主导数字视频。

NBC 现在将发布自己的下载 [NBC Direct](https://web.archive.org/web/20230218024708/http://www.nbc.com/Video/) ，它只适用于 Windows 机器。给 Mac 用户竖起双指的方式。

Mac 用户将不得不等待 Hulu(NBC 与 Fox 的合资视频公司)公开发布，并满足于五周后消失的流媒体，或者访问像 AOL、MSN 或 MySpace 这样的 Hulu 附属公司。

我不确定我们已经看到了这场戏剧的结局。谁认为 NBC 会在一年内把它的玩具带回 iTunes？
