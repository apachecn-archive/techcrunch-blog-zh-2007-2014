# Curebit 为抄袭 37 个信号道歉:“愚蠢、懒惰、无礼”

这很尴尬:就在宣布[一轮 120 万美元的融资](https://web.archive.org/web/20221226065848/https://techcrunch.com/2012/01/27/yc-alum-curebit-raises-1-2-million-for-online-referral-system/)时，在线推荐初创公司 [Curebit](https://web.archive.org/web/20221226065848/http://www.curebit.com/) 被发现剽窃了 37Signals 的设计和代码，37 signals 是流行协作工具 Basecamp、Highrise 等的幕后公司。

37Signals 的合伙人[大卫·海涅梅尔·汉森](https://web.archive.org/web/20221226065848/http://www.twitter.com/dhh)在推特上指出了这一抄袭行为，在与 Curebit 的联合创始人艾伦·格兰特[交流后，他称 Curebit 团队](https://web.archive.org/web/20221226065848/https://twitter.com/#!/dhh/status/163297264624861184)“他妈的人渣。”格兰特最初的反应似乎并不特别悔悟，这可能没有帮助——他为抄袭辩护说这是一个“快速测试”，并在某一点上[告诉 Heinemeier Hansson](https://web.archive.org/web/20221226065848/https://twitter.com/#!/allangrant/status/163197219011428353) ，“冷静伙计:”( [VentureBeat 对最初的争议有一个很好的详细描述](https://web.archive.org/web/20221226065848/http://venturebeat.com/2012/01/28/cant-look-away/)。)

然而现在，格兰特已经在公司博客上发布了道歉——事实上，它吸引了太多的关注，以至于博客都崩溃了。他还[在黑客新闻](https://web.archive.org/web/20221226065848/http://news.ycombinator.com/item?id=3523542)上发表了文字:

> 最近，我们推出了一个网站，其中有几页是从 37signals' Highrise 复制来的。我们不仅仅是从他们的设计中获取灵感，我们实际上使用了 html 和 css 代码，并链接到他们网站上的图片。我们为剽窃大卫和 37signals 的作品向他们道歉。这是愚蠢的，懒惰的，不尊重他们的创造性努力。对我们来说，对 37 个信号这样做是特别痛苦的，因为他们是我们的大英雄。我们只是希望他们能接受我们的道歉。

格兰特给我发了一封电子邮件，强调他对 37Signals 的钦佩，承认他“越界了”，并总结道，“我会警告其他创业公司不要为了‘精益’而犯这样的错误。”"

这场争议引起了特别的关注，因为 Curebit 是由 Y Combinator 孵化的，并从 500 家初创公司(包括其他公司)筹集了资金，Heinemeier Hansson [毫不犹豫地指出了这一点。](https://web.archive.org/web/20221226065848/https://twitter.com/#!/dhh/status/163196823517929472)

看起来像是黑客新闻(由 Y Combinator 运营)的一位读者[发表了一篇帖子](https://web.archive.org/web/20221226065848/http://news.ycombinator.com/item?id=3523511)询问 YC 联合创始人保罗·格拉厄姆的意见。格雷厄姆封杀了这条帖子，称它违反了该网站“不使用帖子向我们提问”的准则，但他也写道，“我认为他们不应该这样做，他们没有足够认真地对待最初的投诉，从而加剧了问题。”

与此同时，回到 Twitter 上，500 Startups 创始人[戴夫·麦克卢尔说](https://web.archive.org/web/20221226065848/https://twitter.com/#!/davemcclure/status/163344277718577152)他与 Curebit 团队交谈并“强烈要求他们重新评估设计内容的政策&；希望他们拿出那颗 2 心”[后来补充道](https://web.archive.org/web/20221226065848/https://twitter.com/#!/davemcclure/status/163384885690646529)，“新创始人不是孩子，只是缺乏经验。此外，投资者不是父母，只是叔叔阿姨。我们都会犯错。”
