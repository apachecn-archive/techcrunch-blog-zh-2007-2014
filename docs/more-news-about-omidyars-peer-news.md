# 关于 Omidyar 同行新闻的更多新闻 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/03/18/more-news-about-omidyars-peer-news/>

[![](img/e79fe1a5593722dd339312c6ef4d8e9c.png "JOHNINHAWAIIsmall")](https://web.archive.org/web/20221006213445/https://beta.techcrunch.com/wp-content/uploads/2010/03/johninhawaiismall.jpg) 我今天在夏威夷参加新闻形态会议，一整天都在讨论新闻质量的现状，以及我们到底如何找到一种商业模式来继续为此付费。这在当地是一个大问题——今年早些时候，夏威夷五家最大的电视台中的三家[合并了业务](https://web.archive.org/web/20221006213445/http://the.honoluluadvertiser.com/article/2010/Mar/11/bz/hawaii3110316.html),檀香山明星公报[与另一家日报檀香山广告报](https://web.archive.org/web/20221006213445/http://www.honoluluadvertiser.com/article/20100227/NEWS01/2270336/Honolulu-newspaper-merger-would-mean-name-change-layoffs)合并，导致大量裁员和普遍的公民担忧。

因此，会议以易贝创始人皮埃尔·奥米迪亚的新[同行新闻网站](https://web.archive.org/web/20221006213445/https://beta.techcrunch.com/2010/01/21/peer-news-announces-new-editor-but-little-else/)的编辑约翰·坦普尔的演讲结束是很合适的，这是对未来地方新闻如何运作的一个测试。谢天谢地，我们终于得到了更多关于网站和方法的细节。

坦普尔明确表示，在修复媒体业务方面“没有灵丹妙药”，但也从波动性中看到了很大的希望——这来自现已关闭的《落基山新闻》(Rocky Mountain News)的负责人，这份报纸已经经历了许多日报所担心的事情。

“我们并没有试图重新发明一份本地报纸，并把它放到网上，”他说。事实上，同行新闻的使命甚至不包含“新闻”、“媒体”或“报纸”这些词。简单来说就是“创造一个新的市民广场。”Peer 发展的核心是三个问题:

-新闻自由在民主社会中的作用是什么？

-您如何利用目前所有可用的工具在当地最好地实现这一目标？

-你如何以可持续的方式做到这一点？

在内容方面，坦普尔谈到的最有趣的事情是取消我们所知道的“文章”。他批评了记者传统报道新闻的静态性和偶发性，让读者在档案中搜寻他们想要的信息。相反，Peer 的“构建模块”将是一个几乎像维基百科一样不断更新的页面，或者用他的话说，“更接近于一个随着发展而变化的主题的活生生的历史。”不会再有“遗漏”一篇文章的感觉，因为“文章”将是活的东西。这也解决了当地新闻围绕一个问题，然后继续前进的批评。“我们不会被热门话题所驱使，”坦普尔说。回到这些问题，坦普尔说，自由媒体的作用是告知公民，让他们能够做出明智的决定。“让我们不要让它变得如此困难，”他说。

Peer 方法的另一个标志是博客流行的原因——社区意识。但这肯定是一种不同的方法。首先，Peer 不会有传统意义上的“记者”,它将有“主持人”,帮助促进这个公民广场为社区回答问题。“在这个时代，报纸仍然重写新闻稿的事实令人尴尬，”坦普尔说。“我们不会成为速记员。我认为这是新闻业的衰落。”

但是对于一个打算非常面向社区的网站来说，有一个很大的震惊:同行不会有评论。“(评论)陷入种族主义、仇恨、丑陋，并对拥有它们的新闻机构进行糟糕的反思，”坦普尔说。为什么？他认为，因为人们在评论时不必露脸，所以没有责任感。“我们认为匿名是评论的一个大问题，”他说。

坦普尔还强调说，报道不会手下留情:“我们会像看到的那样称呼事情。我们认为表明立场有真正的价值。”

那么这种商业模式怎么样呢？正如坦普尔指出的，没有太多的商业模式可供选择。与大多数媒体网站不同，这将是一个人们“重视并愿意付费”的会员网站他补充道，“广告不会是我们的重点。”

Peer 应该会在下一季度推出，所以我们很快就能看到更多这些想法的实施。但是很明显，这个网站——或者更喜欢称自己为“新闻服务”——正在采取一种与旧媒体和我们现在认为的“新”媒体截然不同的方式。

得益于这些细节，奥米迪亚创办这家公司似乎不那么不合时宜了。毕竟，易趣是最早有力地利用网络社区的网站之一，开创了许多我们今天仍在使用的信任和声誉系统。
