# Homefront 的短战役提示问题:单人 FPS 是失传的艺术吗？

![](img/f1274ae29c9446db216d84e20f5bda5e.png "Home Front")

PSM3，PlayStation 杂志，已经评论了 THQ 的花式新射手 [Homefront](https://web.archive.org/web/20221208032528/http://www.crunchgear.com/tag/homefront/) 。它得到了令人尊敬的 83/100。然而，令人担忧的是单人战役[只有五个小时的长度](https://web.archive.org/web/20221208032528/http://www.computerandvideogames.com/292449/news/homefront-campaign-is-five-hours-long/)。这么短的活动，你开始问自己:值得开发人员的时间吗？

众所周知，今天的许多 FPS 迷倾向于直接进入游戏的多人部分。如果是这样的话，为什么不采用 id 软件大约 1999 年的方法:检查一下窗外的单人战役——反正没人玩它——而是把你的资源集中在创建一流的多人模式上？这就是《雷神之锤 3》所做的，它的遗产延续到了今天的《雷神之锤直播》,这可能比你在今天的射击游戏中发现的许多多玩家模式都要好。

哦，而且是免费的。所以就这样了。

但是回到 Homefront 和它五个小时的竞选长度。这对你们中的任何人来说是一个交易破坏者，还是说你们真的只对游戏的多人模式感兴趣？如果是这样，为什么？现在人们怎么还没有对硬汉军事多人模式感到厌倦呢？至少我两周前在一些预览活动上玩的 Brink(PS3 版本)不是什么超级严重的情况；有一个团队堡垒 2 的氛围，这是一个很好的氛围。

关于坚持自己的优势，还是有话说的。再次，雷神之锤 3 放弃了单人战役，以专注于其广受欢迎的多人游戏组件。《半条命 2》的单人战役显然是 Valve 努力的重点——你们中有人玩过《HL2 死亡竞赛》多久？这几乎被普遍认为是有史以来最好的单人射击游戏之一。
