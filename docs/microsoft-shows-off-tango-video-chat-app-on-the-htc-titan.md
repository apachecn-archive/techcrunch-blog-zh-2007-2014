# 微软在 HTC Titan 上展示 Tango 视频聊天应用

自从我第一眼看到 Windows Phone Mango，我就喜欢上了它，但事实是，大多数正在实现的功能仅仅是为了赶上竞争平台。

Bing 集成和语音转文本功能……非常酷。但除此之外，正如我所说，芒果正在追赶。在[芒果驱动的 HTC Titan](https://web.archive.org/web/20230204233116/https://techcrunch.com/2011/09/12/att-reveals-focus-s-focus-flash-and-mango-update-plans/) 上添加视频聊天应用是另一个例子，但它即将到来。

尽管存在令人难以置信的不幸的名称共享情况，Tango 视频聊天应用程序(与 WP7 下一次代号为 Tango 的更新无关)实际上非常酷，尽管有点晚了。在下面的视频中，Windows Phone 团队的 Aaron Woodman 可能过于热情地详细描述了视频聊天在 4.7 英寸大屏幕上的工作方式(由 [Windows Phone 博客](https://web.archive.org/web/20230204233116/http://windowsteamblog.com/windows_phone/b/windowsphone/archive/2011/09/15/htc-titan-tango-video-chat.aspx)提供)。像苹果的 FaceTime 和 Android 的 Tango 应用程序一样，WP 的 Tango 可以让你享受前置和后置摄像头之间的“看到我所看到的”切换。奇怪的是，Google Talk 还没有解决这个问题。

很酷的是，芒果的 Tango 应用程序是跨平台的，支持 iOS、Android 和 PC。微软碰巧拥有一个非常棒的叫做 Skype 的网络电话服务，所以这个 Tango 应用将来可能会在 Windows 平台上有一些非常激烈的竞争。
