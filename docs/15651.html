Modus为电子发现的数据驱动方法筹集了1000万美元

服务提供商获得巨额风险投资的情况并不常见。然而，上周，OpenStack cloud builder Mirantis筹集了1000万美元。今天， Modus eDiscovery 为其数据驱动服务筹集了1000万美元，用于降低成本和寻找远远超过任何数量的律师可以手工提供的信息。

这轮融资由哈勃克管理公司和杜鹃花资本牵头。Modus首席执行官Abtin Buergari也参与了融资。

华盛顿特区外的Modus本身没有知识产权。它所拥有的是一个法律市场，这个市场仍然在律师身上花费大量的金钱来研究法律文件。它是如此的劳动密集型，以至于顶级的律师事务所将会承包给电子发现公司，这些公司雇佣低工资的律师来做发现工作。

作案手法将法律文件转化为数据。代替工时，Modus为它搜索到的每千兆字节获得报酬。通过使文档变得可搜索，Modus有办法找到人类根本无法找到的文档，相比之下，分析可以务实地完成。

电子发现世界就像许多市场一样。占主导地位的传统玩家拥有他们已经使用多年的系统和流程。但是数据有办法改变一切。Modus正在利用这一转变提供服务，以节省资金并获得更多的洞察力。

Modus计划利用这笔资金向大型市场扩张，尤其是加州和纽约。正是在这些市场中，律师事务所更倾向于改变他们的方式，特别是随着整个司法系统的发展和适应这个新的数据驱动的世界。
