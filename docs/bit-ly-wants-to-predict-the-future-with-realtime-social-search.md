# Bit.ly 希望通过实时社交搜索来预测未来

链接缩短服务 Bit.ly [刚刚宣布](https://web.archive.org/web/20230203151624/http://bitlyenterprise.com/post/11403350749/social-search)为其付费企业客户推出测试版搜索平台和“信誉监测”服务。

显然，Bit.ly 一直在爬取每个 URL 的缩写“病毒式”(我猜这就是他们一直在做的)，试图提供一个趋势内容的快照，以及足够有趣的内容，这些内容将通过用户搜索成为趋势。

> “我们使用了不同的过滤器，而不是 pagerank 对于任何给定的搜索查询，我们都会显示我们预测将在未来 24 小时内获得最多关注的报道。然后，我们使用 bitly 的分析来实时完善我们的预测。我们的搜索技术基于最有价值的参与度指标:点击量。”

作为其实时社交搜索计划的第一个公开部分，该公司正在推出一个情绪分析系统，该系统将提供给希望进行测试的企业用户。

该公司的博客文章[认为](https://web.archive.org/web/20230203151624/http://bitlyenterprise.com/post/11403350749/social-search)不同于传统的剪辑或趋势引擎，它将基本上预测未来，例如，作为一个“早期预警系统”，提醒你人们对你品牌的评论“摇摆不定”。

作为一个想念谷歌实时搜索的人，我真的为这个产品感到兴奋，只是希望我(或我的公司)不必为它每月支付[955 美元](https://web.archive.org/web/20230203151624/https://bitly.com/pro/help#enterprise_cost)。
