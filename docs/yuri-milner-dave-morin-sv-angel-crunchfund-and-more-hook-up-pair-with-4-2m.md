# 尤里·米尔纳、戴夫·莫兰、SV Angel、CrunchFund 和其他投资者以 420 万美元入股 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/05/01/yuri-milner-dave-morin-sv-angel-crunchfund-and-more-hook-up-pair-with-4-2m/>

对于一个社交网络来说，由于缺乏更好的术语，一夫一妻制社交网络刚刚从如此多的知名投资者那里筹集了资金，我很难选择谁应该出现在这个标题中，并且严重缺乏空间。

该公司最近 420 万美元的种子轮投资者包括阿什顿·库彻的 A 级风险投资公司、戴夫·莫兰、保罗·布赫海特、[创始人集体](foundercollective.com)、 [SV 天使](svangel.com)、[勒勒风险投资](lererventures.com)、迈克尔·伯奇、山姆·奥特曼、[克朗彻基金](https://web.archive.org/web/20221206101142/http://www.crunchbase.com/financial-organization/crunchfund)、[腾讯](tencent.com)、尤里·米尔纳、 [Betaworks](betaworks.com) 、亚历克西斯·奥哈尼安、加里·谭、哈吉特·塔加、加里·维纳查克、布兰迪·巴克

这听起来像是一个令人惊奇的晚餐聚会。

YCombinator 初创公司由五名联合创始人组成的强大团队创建，他们是奥列格·科斯图尔(Oleg Kostour)、杰米·穆雷(Jamie Murai)、安东·克鲁天斯基(Anton Krutiansky)、阿斯温库马尔·拉坚迪兰(Aswinkumar Rajendiran)和迈克尔·彼得罗夫(Michael Petrov)，当然，他们计划利用这笔资金来扩大其移动开发和设计团队。

联合创始人科斯图尔告诉我，由于专为情侣服务的社交网络很少见，许多人认为该公司最接近的竞争对手是 Path。“我们都是移动应用程序，为你提供了一种更简单的方式来与更少的人分享东西，”他说，“然而……这两种工具侧重于不同的关系。”

除了 Path 创始人戴夫·莫林，他显然也认为这两个应用程序是互补的，“我认为我们会看到人们同时使用 Path 和 Pair，而不是在它们之间进行选择。举个例子，我们实际上是把 Path 作为一家公司，在内部互相分享东西，但这并不能改变我和女朋友使用 Pair 的事实。”

与普通的短信不同，Pair 允许给定的男女朋友或其他人通过应用程序交换照片，此外还可以同时绘画、分享照片和同时拇指亲吻——这是一种独特的 Pair 功能，它利用 iPhone 的触摸屏，每当给定夫妇的拇指亲吻时就会振动。说真的，这是通信的未来。

自推出以来，Pair 在过去的一个月中已有超过 22 万次下载，Kostour 表示，他希望“进一步探索”情侣们参与 la Thumbkiss 的动态方式，“归根结底，Pair 是为情侣准备的。这是给他们用来分享他们的记忆，享受乐趣和互相交谈的。这改变了我们设计产品的方式。挑战之一是检查情侣之间存在的更微妙的社会互动，因此我们可以将这些互动整合到 Pair 中。”
