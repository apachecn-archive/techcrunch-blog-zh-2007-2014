# 联邦通信委员会负责人:谷歌/威瑞森网络中立提案让我们慢了下来

今天在三藩市的 Web 2.0 峰会上，联邦通信委员会主席朱利叶斯·格纳考斯基与纽约杂志的约翰·海勒曼坐在一起。海尔曼问的一个问题是 Genachowski 对谷歌/威瑞森联合网络中立政策的看法，他们在八月份在 T4 发布了这一政策。

Genachowski 说:“*我更希望他们不要做他们做*时做的事情。他说:“它减缓了一些导致解决方案的进程。

也就是说，Genachowski 确实说，以前在重大问题上对立的两个大型实体现在走到一起似乎是一件好事。他似乎不喜欢他们做这件事的确切方式。

该提案在 8 月份宣布后，FCC 立即发布了以下声明:

> 有些人会声称这一声明推进了讨论。这是它的众多问题之一。是时候做出决定了——重申 FCC 对宽带电信的权威，保证互联网永远开放，将消费者的利益置于大公司的利益之上。

换句话说，联邦通信委员会想要的是行动，而不是更多的言辞。谷歌/威瑞森的提议带来了更多的词汇。

本周早些时候，谷歌首席执行官埃里克·施密特也谈到了这个话题。有人批评谷歌背弃了网络中立，因为他们没有要求无线监管。但是 Schmidt 说，更多的是把需要做的事情(有线)做好，对竞争更激烈的无线领域采取观望态度。

我们认为这个灰色地带仍然非常可怕。联邦通信委员会对仅仅因为两家大公司的提议而采取任何行动保持警惕，这很好。
