# 美国有线电视新闻网:黑莓停机影响几乎每个星球的用户

无处可逃。据美国有线电视新闻网报道，黑莓服务中断现在正在影响 T2 几乎每个星球的用户。塔图因和科洛桑似乎尚未受到影响，但据报道，吉奥诺西斯、纳布和雅文的用户受到了重创。我们关心他们的居民。
