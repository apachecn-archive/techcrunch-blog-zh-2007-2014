# 未来学家:N-盖奇还活着！但是它能生存吗？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/29/the-futurist-the-n-gage-lives-but-can-it-survive/>

[![Ngage](img/77aee5fe3657660b14f4ce97c54771a7.png)](https://web.archive.org/web/20130628144659/http://www.flickr.com/photos/28429352@N00/1266161488/ "Photo Sharing") 
后知后觉，原来 N-Gage 从一开始就注定了。它的游戏对于手机来说非常壮观，但设备本身体积庞大，屏幕非常小，看起来更像巧克力玉米饼，而不是手机或游戏机。一名诺基亚高管今天表示，他们认为一家公司需要发运 600 万台设备，才能创建一个财务上可持续的游戏生态系统。在全球范围内，芬兰售出了 300 万份墨西哥卷饼，远远没有达到他们的目标

尽管如此，在海外总是比在美国更受欢迎的 N-Gage 从未真正消亡。它持续出现在贸易展会上，值得注意的是，该设备的在线游戏平台 N-Gage Arena 一直在持续运行。“我们只是真正致力于游戏，”诺基亚游戏出版总监格雷格·索特告诉我。“看看游戏业务的价值——这是一个 300 亿美元的行业。如果你看看钱和利润在哪里，这就是大时代娱乐。”

那么，诺基亚打算如何将 N-Gage 这个拒绝死亡的小移动平台转变为占主导地位的手机游戏平台呢？

**多手机支持**

如果你以前想玩 N-Gage 游戏，你必须使用一部手机-N-Gage。考虑到仅诺基亚就生产数十种型号，这严重限制了它们的可用市场。N-Gage 不是作为硬件设备存在，而是一个可以在任何 N 系列手机上工作的游戏平台。有数千万这样的婴儿散布在世界各地，这意味着他们有一个良好的内在基础。

当然，虽然几乎每个购买了旧 N-Gage 的人都可能会购买一些游戏，但大多数拥有 n70 的人可能并不太关心游戏。尽管如此，诺基亚有信心，大量的单元应该创造一个足够大的市场。“大约 50%的 N 系列用户参与游戏，”Sauter 说。“当我们观察今天谁在购买手机游戏时，大约有 5%的用户在购买游戏。但如果你看看 60 系列的用户，这一比例是 20%或更多。”

**发行商支持:**

PS2 打败 GameCube 的部分原因可能与硬件优势有关，但它也有更好的第三方发行商支持。事实是，如果任何游戏平台要生存和繁荣，它需要出版商的支持。对于诺基亚来说尤其如此，它缺乏任天堂的第一方游戏历史，仅依靠《马里奥》和《塞尔达》就能轻松生存。

诺基亚的友好出版商名单中有一些游戏巨头。像艺电、THQ、智乐和卡普空这样的公司都参与其中。考虑到不可能是 N-Gage 的商业成功记录说服了他们，我很好奇他们是如何做到的。不管怎样，他们做到了，这才是最重要的。

**从墨盒中释放**

N-Gage 过去使用类似 SD 卡的卡座。考虑到 N 系列手机是 3G 高速设备，这有点过时了。对于新的 N-Gage 游戏，虽然发行商仍然可以选择盒式磁带，但大多数游戏现在都可以通过空中下载。这将给予发行商更大的自由，让他们既能创作出对微型游戏来说过于庞大和令人惊叹的游戏(例如 MMORPGs)，也能创作出可能太便宜而不能随便玩的小型休闲游戏(*蛇 II:复仇*)。)

我也确认至少会有一款在线游戏同时拥有基于 PC 和 N-Gage 的版本，并且在同一个世界中互动。所以在家里玩，然后当你跳上公共汽车时，从你停下的地方继续。

**应对坏按钮**

在大多数手机上玩游戏意味着玩弄为拨号而设计的小按钮，而不是射杀坏人。虽然旧 N-Gage 很粗糙，但其专用的游戏按钮和类似控制器的感觉让玩游戏变得轻而易举。通过将 N-Gage 转变为一个跨手机平台，他们已经失去了一些便利。尽管如此，诺基亚仍打算至少部分地补救这种局面。据 Sauter 称，这包括禁用当前游戏中不使用的按钮(当你打算射门时不再意外地呼叫妈妈)，以及以类似任天堂 DS 的方式集成未来诺基亚手机中突出的触摸屏。

那么，在 N-Gage 手机失败的地方，N-Gage 平台会成功吗？那得看情况。他们已经获得了出版商的支持，并从该平台以前生活的只有一部手机的监狱中解放出来，但其成功的秘密可能是诺基亚向其他制造商开放它。虽然这显然是一种可能性，特别是考虑到目前越来越多的非诺基亚手机正在使用 Symbian，我只是希望他们不会再花几年时间来实现这一点。
