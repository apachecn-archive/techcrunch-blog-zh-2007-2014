# 诺基亚经理谈论即将到来的 Windows 8 标签，定于 2012 年 6 月发布

诺基亚是微软团队的一员。芬兰电信此前放弃了大部分内部软件操作系统开发，转而支持微软的 Windows Phone 7 系统。但诺基亚似乎正在追赶另一个微软平台的潮流:Windows 8。

诺基亚的法国总经理 Paul Amsellem 最近接受了法国《回声报》的采访。采访当然涵盖了该公司最近发布的 Windows Phone 7 手机，但阿姆塞利姆也提到了 Windows 8 平板电脑。“2012 年 6 月，我们将有一款运行 Windows 8 的平板电脑，”他说。这呼应了诺基亚首席执行官史蒂芬·埃洛普[最近的声明](https://web.archive.org/web/20230205012457/https://techcrunch.com/2011/10/27/nokia-ceo-sees-broader-opportunity-with-windows-8-hints-at-tablets/)，该公司在即将推出的平板电脑中看到了新的机遇。

> Windows 8 的用户体验本质上是你今天在台上看到的诺基亚 Lumia 体验的增强版。从用户的角度来看，你看到了共性的相似之处和机会。你会说，哇，这不仅仅是智能手机，还有更广阔的机会。显然，我们也看到了更广泛的机会，但没有具体评论这在未来可能意味着什么。

这并不是诺基亚第一次尝试平板电脑。该公司一直在谈论一款能与 iPad 竞争的平板电脑。然而，随着诺基亚的移动平台在市场上苦苦挣扎，Meego 感受不到开发者的喜爱，该公司似乎决定不推出平板电脑。Windows 8 是一个完美的选择。

诺基亚清楚地看到了它与微软的未来。该公司怀念 Android 早期的狂野和疯狂。然后，微软决定通过向 T4 提供 10 亿美元的预付款来轻松开发 WinPhones。正如埃洛普指出的，诺基亚的新手机和 Windows 8 用户界面有明显的相似之处。如果诺基亚运作得当，该公司将很快成为微软团队中最受欢迎的一员。那是一个很棒的地方。
