# GoodGuide 的消费品“品质”数据库开始移动化

[![](img/0a8fd009d40f4acd207f09c6a586d079.png)](https://web.archive.org/web/20221006111140/http://www.goodguide.com/)

根据成分的毒性和环境影响等因素对消费品进行评级的初创公司 GoodGuide 在苹果应用商店发布了其原生 iPhone 应用程序。这个免费应用程序允许用户使用产品名称或 UPC 代码浏览和搜索该网站快速增长的数据库(现在有大约 60，000 种产品)，可以在这里找到。

GoodGuide 在今年的 TechCrunch50 大会上是最受欢迎的，通常被认为是一个[伟大的想法](https://web.archive.org/web/20221006111140/http://www.beta.techcrunch.com/2008/09/10/tc50-goodguide-shines-light-on-the-goodness-of-consumer-products/)，最终获得了该活动最高奖项的亚军。该网站的数据库根据健康、环境和社会表现对每种消费品进行评级，指出致癌成分和制造商对待员工和环境的方式。

![](img/9869b429a4efcca25f27f12526dbc002.png)

因为购物者通常在进入商店之前不会准确计划他们将要购买的商品，所以 GoodGuide 的移动功能将在这家初创公司的成功中发挥重要作用。在此之前，用户可以通过网络浏览器访问该网站的移动版本，但 iPhone 的原生应用程序速度更快，更直观——我可以看到这将成为“足球妈妈”人群和环保健康意识的最爱。GoodGuide 还计划在未来几个月内推出 Android 版本。

我对这个应用程序的一个主要不满是，它总是打开一个“特色产品”页面，迫使你在到达搜索框之前点击几个菜单项。我还希望有一种方法可以拍摄产品的条形码，并自动在数据库中查找它，尽管我不确定 iPhone 是否有能力做到这一点(iPhone 上只有几个条形码应用程序，似乎大多数都只支持二维码。在安卓系统上，它们被列为最受欢迎的应用之一。
