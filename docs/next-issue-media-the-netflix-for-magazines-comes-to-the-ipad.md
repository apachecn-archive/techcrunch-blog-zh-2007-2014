# 下期媒体，杂志的网飞，来到了 iPad

杂志迷们，欢呼吧:[下一期媒体](https://web.archive.org/web/20221204052201/http://www.nextissue.com/)，这可能是你阅读你最喜欢的书籍最简单、最经济的方式，现在可以在 iPad 上使用了。

这不是一个完美的类比，但解释这款应用最简单的方式就是，是的，作为杂志的网飞。下一期是五大杂志出版商(即 Conde Nast、Hearst、Meredith、News Corp .和 Time Inc .)的合资企业。每月支付 14.99 美元的订阅费，读者就可以无限制地阅读该应用目录中的所有杂志。你也可以购买单个的问题，订阅单个的标题，或者支付 9.99 美元几乎无限制地访问(你可以得到月刊和双周刊，但不是像《纽约客》和《时代》这样的周刊)。无论你的计划是什么，首席执行官摩根·根特说他想创造一种报摊式的体验，你可以在一个地方浏览和购买多种书籍，而不是为每一种都打开一个新的应用程序(并学习新的导航风格)。

[下一期于 4 月首次推出](https://web.archive.org/web/20221204052201/https://beta.techcrunch.com/2012/04/03/next-issue-media-launch/)，但仅适用于运行 Honeycomb 或更高版本的 Android 平板电脑，屏幕分辨率为 1024×600 或 1280×800。鉴于所有这些限制，Guenther 表示，早期版本更像是一个“行业发布会”>今天是下一期第一次面向大量消费者，更大的营销推广正在进行中。

几周前，当 Guenther 向我展示这款应用时，它看起来与 Android 应用非常相似。在我看来，演示的要点是展示你可以采用一个更通用的界面(而不是一个为特定标题设计的界面),并且仍然为每本杂志提供一个流畅的、高质量的体验。作为《纽约客》iPad 应用的忠实读者，我并不觉得用下期阅读《纽约客》会失去什么。

该应用程序现在包括 39 个标题——新的标题是 Bon Appétit、Brides、Golf Digest、GQ、Self、Vogue 和 Wired。Guenther 说，目录将继续扩大，目标是在年底前翻一番，但他希望专注于“优质、高读者群的书籍”，所以不要指望你最喜欢的默默无闻的书籍会很快出现在应用程序中。Guenther 的其他计划包括添加搜索、个性化和社交功能，以及扩展到其他平台。

你可以[在这里](https://web.archive.org/web/20221204052201/http://itunes.apple.com/us/app/next-issue/id533561885?mt=8)下载下一期 iPad 应用。
