# 彼得·芬顿在嘎吱嘎吱会议上被评为“年度风险投资”

Benchmark 的彼得·芬顿刚刚在 Crunchies 获得了年度风险投资奖。

芬顿最出名的可能是支持并继续在 Twitter 董事会任职，Twitter 于去年上市。他还是 Yelp、New Relic、Hortonworks、Zuora 和 Polyvore 等公司的投资者/董事会成员。(如果这还不够让他忙的话，他还是旧金山歌剧院和加州科学院的董事。)

去年，[推出了由芬顿支持的移动文字处理器 Quip，](https://web.archive.org/web/20221226032311/https://techcrunch.com/2013/07/30/quip-mobile-word-processor/)[基准测试公司投资 A/B 测试服务 Optimizely](https://web.archive.org/web/20221226032311/https://techcrunch.com/2013/04/10/optimizely-series-a/) ，芬顿也加入了董事会。

我在下面的 Disrupt Europe 会议上嵌入了 Fenton 在后台谈论他的投资策略的视频。

该奖项由 GitHub 联合创始人汤姆·普雷斯顿-沃纳和 Gigaom 资深作家德里克·哈里斯颁发。获得提名的还有红杉资本的吉姆·戈茨、格雷洛克合伙公司的雷德·霍夫曼、谷歌风投的比尔·马里斯和星火资本的比扬·萨贝特。
