# 系好安全带:Mozilla 的 Jetpack 可能是浏览器扩展的下一步

[![](img/79fd5f24712c53f8bde72532ae5bcf4f.png)](https://web.archive.org/web/20221006002728/https://jetpack.mozillalabs.com/)

Mozilla 公布了其实验室部门的一个名为 [Jetpack](https://web.archive.org/web/20221006002728/https://jetpack.mozillalabs.com/) 的新项目，该项目向我们展示了 Firefox 在未来几年将如何开始扩展网络功能。虽然该项目仍处于开发的早期阶段，但它似乎采取了一种简化的扩展系统的形式，允许 web 开发人员使用基于 web 的工具向 Firefox 浏览器引入新功能，并且只需要用户付出最少的努力。

Firefox 扩展一直是浏览器最好的部分之一，允许用户根据自己的需要添加和删除功能。虽然安装这些软件的用户体验相对简单，但仍然需要重启浏览器，这在你正忙着的时候可能会令人沮丧。有了 Jetpack，这就不是问题了——你点击 install 就完成了，几秒钟后新的小部件或应用程序就安装好了。

Jetpack 被描述为“使用 Web 技术来增强浏览器(例如 HTML、CSS 和 Javascript)的一次探索，其目标是让任何能够建立网站的人都能参与进来，使 Web 成为一个更好的工作、交流和娱乐场所。”换句话说，如果你知道如何为 web 开发，你应该能够构建一个 Jetpack 应用程序。开发者可以使用 Mozilla 基于网络的 [Bespin](https://web.archive.org/web/20221006002728/https://bespin.mozilla.com/) 环境和流行的 [Firebug 扩展](https://web.archive.org/web/20221006002728/http://www.getfirebug.com/)来编写代码。要了解构建这些应用程序有多简单，请观看下面的视频。

在这一点上，没有很多演示应用程序可用，但您可以通过访问 Jetpack 主页并安装 Jetpack 0.1 扩展来亲自试用它们(您必须重新启动浏览器才能让它工作)。从那里你可以找到演示[这里](https://web.archive.org/web/20221006002728/https://jetpack.mozillalabs.com/)和[这里](https://web.archive.org/web/20221006002728/https://wiki.mozilla.org/Labs/Jetpack/In_The_Wild)，可用的应用程序包括天气预报、美味通知和广告拦截器(如果你在寻找真正无用的东西，还有一个应用程序会混合你打开的浏览器标签中看到的图像)。

因为 Jetpack 仍然处于非常早期的阶段，所以很难说它最终会变得多么强大，以及它是否会取代当前的扩展模型。除了传统的扩展，这个新项目可能会允许可信网站在逐个网站的基础上定制浏览器体验，这可能会被证明是非常有用的(或者令人讨厌，取决于它如何工作)。
