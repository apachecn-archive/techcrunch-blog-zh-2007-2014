# Quotables 以美味报价的形式推出，但采用 CafePress 收入模式

于 8 月份试运行的![](img/899b9b4f7e498acae0934f108275bafd.png "Quotables") [【报价】](https://web.archive.org/web/20230329100925/http://quotabl.es/)旨在成为*在线寻找和分享报价的*目的地，通过社交媒体和众包来组织其主要由用户生成的报价数据库，并剔除错误报价和错误归因。这家位于苏格兰格拉斯哥的初创公司得到了第四频道和英格兰艺术委员会的支持，而网站本身是由 [Mint Digital](https://web.archive.org/web/20230329100925/http://mintdigital.com/) 建造的。

这是对传统引语词典的一种社交形式，可以手动添加引语，也可以通过新推出的“bookmarket”添加引语。bookmarket 允许用户在任何网站上高亮显示一段文字，只需点击一下，就可以将其添加到引语词典中，并附带属性和返回原始来源的链接。这既有趣又快捷，对于在线记录那些智慧或愚蠢的小宝石也有潜在的用处。

![](img/dc35d992572152fdea2462e57f0304be.png "Picture 62")此外，还可以通过创建列表来标记和管理报价。在许多方面，Quotables 可以被认为是报价的美味，这将我们带到了网站计划如何实际赚钱的棘手问题。

显然，Quotables 的目标是规模，它表示希望成为“最权威的在线报价资源”——这是一个直接报价——尽管其收入模式可能不那么可扩展(或雄心勃勃)。这家初创公司表示，它计划创建一个按需销售业务，我推测这包括让用户订购 t 恤、杯子、鼠标垫等物品，并根据 Quotables 数据库中的报价进行定制打印。这当然是一个好主意，但它是否是一棵潜在的摇钱树，我不太确定。

此外，该公司表示，它正在探索媒体合作伙伴关系，特别是计划扩大其与第四频道的现有整合。
