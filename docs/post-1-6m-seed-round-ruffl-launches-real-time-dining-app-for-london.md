# 在 160 万美元的种子期后，Ruffl 为伦敦推出实时餐饮应用

到现在为止，你可能已经听说过一些应用程序，在那里你可以根据当天的供应情况预订一些东西。例子包括 Hotel Tonight，当天酒店预订应用程序，Yplan，当天娱乐活动应用程序和 Hailo，“现在”打车应用程序。这些都是简单的应用程序——至少在前端是这样——它们不需要任何计划，通常会在当天提供折扣。现在推出了本地服务的实时预订系统 Ruffl，,第一项服务旨在餐馆预订，首先在伦敦推出。抢[安卓](https://web.archive.org/web/20221209131020/https://play.google.com/store/apps/details?id=com.anqa.hungrytony)和 [iPhone 应用](https://web.archive.org/web/20221209131020/https://itunes.apple.com/gb/app/hungry-tony/id621516284?mt=8)。

很简单——你在附近找到一家很棒的餐厅，而且马上就有空位。餐馆还可以实时提供优惠，以吸引附近的食客。这解决了目前困扰传统预订和预订平台的预订失约问题。到目前为止，他们说他们已经和伦敦的 450 家餐馆签约。

这家初创公司已经从一批天使投资者那里获得了 160 万美元/100 万英镑的种子基金，这些天使投资者包括卡伦·汉顿(TopTable 的创始人)、大卫·普里查德(OpenTable Europe 的前首席执行官)和马克·弗格森(曼联足球俱乐部前经理亚历克斯·弗格森的儿子)。你可以把它描述为下一波超本地、实时餐饮的一部分，与 Hasty 和其他餐厅并列。

共同创始人是艾伦·罗斯和劳伦斯·卡弗。Ross 是一位连续创业者，最近是大数据医疗保健初创公司 FirstCare 的首席执行官和联合创始人。

创始人罗斯说:“餐馆只是开始。我们建立了这个实时、超本地的经纪平台，以扩展到餐馆以外的其他垂直领域。”

这里的含义当然是扩展到其他“实时需求”，尽管让我们先看看他们在用餐方面做得如何。
