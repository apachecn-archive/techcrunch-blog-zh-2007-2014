# 移动广告服务初创公司 MADS 向€融资 100 万美元

移动显示和消息广告服务解决方案提供商 MADS 在由 [OTM 投资](https://web.archive.org/web/20230205035504/http://www.otminvestments.com/)牵头的 B 轮融资中，已经为[【€】筹集了 100 万](https://web.archive.org/web/20230205035504/http://www.crunchbase.com/company/mads)——约合 140 万美元(该公司和投资者均位于荷兰)。

MADS 表示，它将利用这笔新资金拓展移动广告服务以外的业务，并在英国、德国、意大利、法国和西班牙设立销售办事处，以当地语言支持客户。
