# 雅虎失去了一位顶级交易人——又多了一位高管

[![](img/90f8cfb3af0921e3f4ee8b8b0be3d129.png)](https://web.archive.org/web/20221206202448/http://www.crunchbase.com/person/kent-goldman) 正如任何人都可能猜到的那样，雅虎高管出走仍在继续。有传言称，雅虎的企业发展总监、交易高手之一肯特·戈德曼(Kent Goldman)将离开公司。高盛于 2004 年加入雅虎，担任业务战略总监，向托比·科佩尔(时任雅虎首席战略官——现为雅虎欧洲董事总经理)汇报工作。如果这是准确的，自 2007 年 1 月以来，离开公司的雅虎高管总数约为 115 人(T4)。仅这个月就有 18 位高管离职，包括布拉德·加林豪斯、杰夫·韦纳、唯帅·马基雅尼、陆弃、卡特琳娜·法克、斯图尔特·巴特菲尔德和约书亚·沙赫特。

事实上，雅虎现在正在失去交易人员，这并不一定是一个好消息或坏消息。2007 年，雅虎企业发展总监迈克尔·马尔克斯离开雅虎，成为 CBS Interactive 的战略和企业发展副总裁(他最近被提升为执行副总裁)。这可能只是高盛得到了一个他无法拒绝的去其他地方的邀请。

**更新**:有趣的是，Kent 最近加入了脸书不断壮大的雅虎校友团体，几天前他成为了首轮资本的粉丝。我们会联系他们看他是否和公司有联系。
