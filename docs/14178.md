# 美国法院命令苹果公司为 FaceTime 专利侵权支付 3.68 亿美元的赔偿金

苹果现在在法庭上找不到突破口。在对库比蒂诺的最新法律打击中，美国法院裁定其 FaceTime 视频通话技术侵犯了科技公司 VirnetX 的四项专利。美国德克萨斯州东区的一家法院已经下令苹果公司向 VirnetX 支付超过 3.68 亿美元的赔偿金。

正在讨论的专利是美国专利号[6502135](https://web.archive.org/web/20221128185141/http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PALL&p=1&u=%2Fnetahtml%2FPTO%2Fsrchnum.htm&r=1&f=G&l=50&s1=6,502,135.PN.&OS=PN/6,502,135&RS=PN/6,502,135)、[7418504](https://web.archive.org/web/20221128185141/http://patft.uspto.gov/netacgi/nph-Parser?Sect2=PTO1&Sect2=HITOFF&p=1&u=/netahtml/PTO/search-bool.html&r=1&f=G&l=50&d=PALL&RefSrch=yes&Query=PN/7418504)、[7921211](https://web.archive.org/web/20221128185141/http://www.google.com/patents/US7921211)和[7490151](https://web.archive.org/web/20221128185141/http://www.google.com/patents/US7490151)——其中包括一种建立安全通信链路的方法。

苹果可以选择对该决定提起上诉。我们已经联系了苹果和 VirnetX，寻求进一步的评论，并将更新任何回应。

VirnetX 首席执行官兼总裁肯德尔·拉森(Kendall Larsen)在一份声明中评论这场胜利时说:“我们对与苹果的诉讼结果非常满意。这一胜利进一步确立了我们专利组合的重要性。”

BBC 报道称，损害赔偿金大约是 VirnetX 最初索赔金额的一半。它还指出，VirnetX 此前已就类似的索赔从微软获得了 2 亿美元的和解金。

该公司拥有 20 项美国专利和 26 项国际专利，还起诉思科、Avaya 和西门子涉嫌侵犯相同的专利。

苹果最近的法律困境包括一名美国法官驳回了针对摩托罗拉移动公司的专利许可案，一家英国法院坚持裁定三星的 Galaxy Tab 平板电脑没有侵犯 iPad 的设计，一家荷兰法院裁定三星没有侵犯苹果的多点触控专利。苹果公司最近也因为一份法律声明的措辞而陷入困境，该声明被要求在其英国网站上发布。
