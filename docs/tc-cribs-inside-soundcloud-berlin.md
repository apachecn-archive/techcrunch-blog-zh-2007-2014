# TC Cribs: Inside SoundCloud，这家总部位于柏林的初创公司以音乐(和俱乐部伙伴)为燃料

柏林此刻充满了创业的活力，SoundCloud 是成为这座城市不断增长的科技创业场景的支柱之一。因此，当 TechCrunch 电视台最近在柏林报道 [Disrupt Europe 会议](https://web.archive.org/web/20230404225643/https://techcrunch.com/events/disrupt-europe-berlin-2013/disrupteurope2013/)时，我们一定要去 SoundCloud 的办公室看看。

自 2007 年首次亮相以来，SoundCloud 取得了显著增长，现在其服务在全球拥有超过 2 . 5 亿活跃听众[。它的员工也增长了不少，你可以从它目前总部的密度中看出来(该公司在旧金山、纽约、伦敦和保加利亚都有卫星办公室，明年将搬到一个更大的柏林办公室。)不过，就目前而言，近距离营造了一种令人兴奋的合作氛围——由于大多数 SoundCloud 员工都是音乐发烧友或音乐家，除了技术之外，还有很多创意。](https://web.archive.org/web/20230404225643/https://techcrunch.com/2013/10/29/soundcloud-now-reaches-250-million-listeners-in-its-quest-to-become-the-audio-platform-of-the-web/)

查看上面嵌入的视频，看看让 SoundCloud 总部每天都保持忙碌的艺术作品、乐器和队友[的瓶子。你还将了解到从 SoundCloud 的全职语言教师那里上德语课是什么感觉——总之，我会说这是一次非常棒的访问。](https://web.archive.org/web/20230404225643/http://en.wikipedia.org/wiki/Club-Mate)
