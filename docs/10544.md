# 迪安·卡门的充气风力涡轮机兼做数字广告牌

大多数风力涡轮机都有一个可识别的形状，但 Segway 的发明者 Dean Kamen 的一个新设计在其功能性方面不太明显:这种涡轮机有一个垂直轴和由塑料织物制成的转子，这意味着很容易被风移动。

原型风力涡轮机需要大量的钢材和混凝土来建造，一旦安装，就很难移动。卡门的设计使用空气给转子充气，转子就像巨大的船帆。转子可以很容易地放气，使得涡轮机不仅重量轻，而且可以移动。

卡门认为，涡轮机的移动性将有助于业主产生更多的能量，当预计附近有大风时，他们可以将涡轮机扔到卡车上。这可能会产生一种新的风暴追逐者，尽管这些涡轮机能够处理多少风力，以及如何将它们安装在临时位置以防止它们飞走仍是问题。

在他的[专利申请](https://web.archive.org/web/20230203082742/http://appft1.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PG01&p=1&u=%2Fnetahtml%2FPTO%2Fsrchnum.html&r=1&f=G&l=50&s1=%2220110215738%22.PGNR.&OS=DN/20110215738&RS=DN/20110215738)中，卡门描述了涡轮机的轻质如何使其能够安装在无法承受钢筋混凝土种类重量的屋顶上。

另一个独特的功能是一个 LED 系统，可以将涡轮机变成一个数字广告牌。当然，由嵌入其中的涡轮机提供动力，这些灯可以显示图像、消息、广告、天气预报和交通延误等内容，并为所有者提供另一个潜在的收入来源。

不同寻常的展示也可以激发关于风力发电的对话，意识是卡门设计的主要目标:在他看来，该设计与其说是绿色技术的突破，不如说是风力发电可能性的一种不同方法。
