# 裁员冲击 MySpace TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/05/09/layoffs-hit-myspace/>

[![](img/a3852db4ce24e69ade6e6fbb12c06ffe.png)](https://web.archive.org/web/20221209002852/http://www.myspace.com/)

我们已经证实，MySpace 上周解雇了多达 45 名员工。接近该公司的消息来源称，确切的数字可能会更小，但 MySpace 肯定已经解雇了相当数量的人。MySpace 拒绝对此事发表评论，因此不清楚这是因为项目取消、普遍裁员还是基于绩效的削减。一位消息人士称，早期迹象表明，空缺职位将不会由其他员工填补。

就在几周前，MySpace 突然[解雇了](https://web.archive.org/web/20221209002852/http://www.beta.techcrunch.com/2009/04/24/news-corp-pulls-the-trigger-owen-van-vatta-now-runs-myspace/)创始 CEO [克里斯·德沃夫](https://web.archive.org/web/20221209002852/http://www.crunchbase.com/person/chris-dewolfe)，并带来了由[欧文·范·纳塔](https://web.archive.org/web/20221209002852/http://www.crunchbase.com/person/owen-van-natta)领导的新[高管团队](https://web.archive.org/web/20221209002852/http://www.beta.techcrunch.com/2009/04/27/threes-company-meet-your-new-myspace-executive-team/)。目前还不清楚裁员是否与新团队做出的决定直接相关，或者是否在德伍尔夫离开之前就已经开始了。无论如何，我们可能会听到更多的重组，因为范纳塔和他的团队削减了他们不想继续追求的项目。

去年 7 月，MySpace 被传言裁员 5%,尽管该公司后来表示，裁员是基于绩效，相当于公司的“不到 5%”。
