# 设备识别初创公司 BlueCava 从马克·库班 S3 风险投资公司获得 950 万美元

**独家——**[blue cava](https://web.archive.org/web/20230205031600/http://www.bluecava.com/)，一家开发技术使其客户能够识别智能手机、电视机顶盒、游戏机、电脑等独特联网设备的初创公司，已经在由风险投资公司 [S3 风险投资](https://web.archive.org/web/20230205031600/http://www.crunchbase.com/financial-organization/s3-ventures)牵头的一轮融资中筹集了[950 万美元](https://web.archive.org/web/20230205031600/http://www.crunchbase.com/company/bluecava)。早期的支持者，天使投资人和亿万富翁马克·库班和蒂姆·赫丁顿也参与了这轮投资。

BlueCava 提供设备识别技术，使企业能够通过智能定位提高在线广告的效果，并将欺诈风险降至最低。

关于这项技术，有一个有趣的背景故事，我在[之前的帖子](https://web.archive.org/web/20230205031600/https://techcrunch.com/2011/07/08/mark-cuban-backed-device-identification-startup-bluecava-raises-1-5-million/)中概述过(它涉及新浪潮摇滚乐队的前道路经理里克·理查森)。

S3 风险投资公司的董事总经理 Brian R. Smith 将加入 BlueCava 的董事会，Putney Capital 的董事总经理、Equifax/IXI 数字公司的前总裁 Marsh Marshall 将担任公司的董事长。

其顾问委员会已经包括了像脸书首席安全官乔·苏利文和万事达卡欺诈管理解决方案副总裁埃伦·莫斯科维茨这样的人。

最近一轮融资使该公司筹集的资金总额达到了 1600 万美元。
