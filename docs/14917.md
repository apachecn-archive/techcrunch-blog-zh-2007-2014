# Twitter 关闭 Android、iPhone 和 AIR 的 TweetDeck，停止 TweetDeck 的脸书集成

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/03/04/twitter-shuts-down-tweetdeck-for-android-iphone-and-air-discontinues-tweetdecks-facebook-integration/amp/>

Twitter [于 2011 年收购](https://web.archive.org/web/20230403104217/https://techcrunch.com/2011/05/02/twitter-to-buy-tweetdeck-for-40-million-50-million/)的功能丰富的 Twitter 客户端 TweetDeck ，很快将主要作为基于网络的服务存在，而原生的 Mac 和 Windows 应用将退居网络和 Chrome 应用之后。该公司正在关闭桌面版 TweetDeck 的 AIR 版本，并将在 5 月份从各自的移动商店移除 Android 和 iPhone 应用。此外，TweetDeck 团队今天宣布，它也将“停止支持我们的脸书整合。”

TweetDeck 将继续支持其用于 Mac 和 Windows 的原生应用，但显然重点将放在网络和 Chrome 应用上，该团队写道，这将“提供迄今为止最好的 TweetDeck 体验”基于网络的应用也将首先获得新功能，“紧随其后的是 Mac 和 PC 应用。”鉴于对网络应用的明确关注，本地应用被砍掉可能只是时间问题。

Twitter 显然仍然致力于将 TweetDeck 作为高级用户的工具，今天的公告指出，该团队的规模在过去六个月中翻了一番。大多数用户已经在桌面上使用 TweetDeck，在手机上使用 Twitter。用于 AIR、iPhone 和 Android 的 TweetDeck 的当前版本仍然使用 Twitter 的 API 1.0 版本，该版本将在未来几周内[退役](https://web.archive.org/web/20230403104217/https://dev.twitter.com/blog/planning-for-api-v1-retirement)。Twitter 今天要退出的应用程序已经很久没有更新了，所以在某种程度上，对它们来说已经是板上钉钉了。

声明中只有一句话提到了停止对脸书整合的支持(“我们也将停止对脸书整合的支持”)，但我们已经向 Twitter 询问了更多有关此事的细节，一旦我们听到更多信息，我们将更新此贴。
