# 创始人的故事:马特·麦金尼斯离开苹果开始暗示

本周的创始人故事以暗示创始人兼首席执行官马特·麦金尼斯为特色。拥有 100 名员工的云出版公司最近宣布获得[1600 万美元的 C 轮融资](https://web.archive.org/web/20221007092737/https://beta.techcrunch.com/2013/07/17/inking-new-deals-with-pearson-and-elsevier-inkling-nets-16m-from-sequoia-and-more-to-digitize-the-worlds-academic-content/)。随着不可避免的增长，这是一个有趣的暗示时间。但这个故事仍在展开，所以马特和我回到了开始，更多地谈论这家公司是如何开始的，以及是什么激励他在八年后离开苹果，去创建一家创业公司:

> 你不会看到很多成功的初创公司首席执行官出自苹果公司。事后看来，我认为我学到的是，你在一个顺风的公司应用的许多方法，你得到的资源只是一种倾注，因为模型是有效的。这些产品是成功的，以苹果为例，对你正在做的事情守口如瓶。那不行。苹果公司没有人愿意来谈论他们正在打造的下一个伟大的东西，但作为一家初创公司，你必须这么做。"

继续我们的谈话，麦金尼斯详细讲述了从苹果公司到领导一家创业公司的转变，以及他在招聘、信任和透明度方面的心得。他还解释了为什么三个联合创始人能比两个人提供更好的平衡。观看以了解更多信息。
