# 扎克妹妹放弃了他

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/03/zucks-sis-quits-on-him/>

脸书向我们证实，马克·扎克伯格的妹妹兰迪·扎克伯格将离开脸书，创办自己的社交媒体咨询公司 RtoZMedia。

另一个扎克伯格在脸书工作了六年，2009 年成为营销总监，领导公司的脸书直播节目和其他社交媒体项目。

ATD 有她写给雪莉·桑德伯格的[辞职信](https://web.archive.org/web/20230204223316/http://allthingsd.com/20110803/exclusive-randi-zuckerberg-leaves-facebook-to-start-new-social-media-firm-resignation-letter/)，我假设是公关副总裁埃利奥特·施拉格。与此同时，我还没有说服脸书的任何人给我一个关于谁可能取代她成为营销负责人的提示(我听说这将是一个外部招聘)，所以我只是要发布 RaZu 在 2005 年唱“Valley Freude”的视频——在她成为明星之前——以及她在 2009 年在 Crunchies 拥抱麦克的[标志性图片](https://web.archive.org/web/20230204223316/https://techcrunch.com/2009/07/13/mark-can-have-chris-leftovers-randi-gets-michael/)。

这两个时刻都代表了我对我们行业的不敬(令人惊讶的是，“弗劳德谷”仍然是相关的)。

这是她关于工作变动的脸书状态更新。如果我偶然发现更多的信息，我会更新这个帖子。
