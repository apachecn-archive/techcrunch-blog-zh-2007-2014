# 脸书虚荣网址:记者不必排队等候

周五晚上是脸书虚荣 URL land rush([5 月 31 日首次报道](https://web.archive.org/web/20221207202107/http://www.beta.techcrunch.com/2009/05/31/prepare-for-the-facebook-vanity-url-landrush/)，[昨日宣布](https://web.archive.org/web/20221207202107/http://www.beta.techcrunch.com/2009/06/09/you-have-three-days-to-pick-your-facebook-vanity-url/)正式发布)。太平洋标准时间周五晚上 9:01，登录脸书，抢在别人之前抢到你想要的名字。

虽然大多数人(也就是你)会在周五晚上为那个完美的名字而战，但一些幸运的少数人不需要排队等候。[脸书](https://web.archive.org/web/20221207202107/http://www.crunchbase.com/company/facebook)员工已经抢了他们的(拿[Facebook.com/Mike](https://web.archive.org/web/20221207202107/http://www.facebook.com/mike)，这是我要的)。显然，“关键记者”也不用排队。

在今天的一封电子邮件中，脸书告诉我“我们想让你知道，我们决定为我们合作的主要记者和媒体保留用户名。请留意来自沟通团队成员的电子邮件，了解更多详细信息。”TechCrunch 的其他作者也收到了同样的邮件。

我觉得张贴这个有点不好，因为脸书实际上是在帮我们的忙。但我也认为，脸书让包括员工在内的一些人先说出名字，这有点胡扯。不过，我的负罪感仅限于此。你们这些笨蛋排队等着。我要提前抢到我的名字。
