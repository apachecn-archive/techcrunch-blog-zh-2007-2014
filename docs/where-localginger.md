# 在哪里购买 LocalGinger 以获得基于位置的社交商务知识

![](img/63e133cf74e3280e2e5facda03606d41.png)

基于位置的媒体公司[，其中](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/company/where-com)已经收购了 [LocalGinger](https://web.archive.org/web/20230404181901/http://www.localginger.com/) ，该公司运营同名的每日交易/团购服务。LocalGinger 专注于其社交商务活动的二级市场，它表示，这使它能够在选定的城市建立强大的追随者。

那么，为什么 Where 会收购这家私营的 flash commerce 初创公司呢？简而言之，增加其[广告产品](https://web.archive.org/web/20230404181901/http://where.com/site/advertise/)。

该公司表示，其 WHERE Ads 平台使其能够覆盖 5000 万人，通过整合 LocalGinger，本地企业将能够使用工具来创建报价，并在超本地级别上推动实时商务。

说[沃尔特道尔](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/person/walter-doyle)的 CEO 在哪里:

> “借助 LocalGinger 平台，Ads 在 Groupon 和 AdSense 等服务之间架起了一座桥梁。将这些服务结合在一起，有助于我们向受众以及出版商合作伙伴的受众提供相关且有价值的内容。在 WHERE Ads 上，对即时团购交易感兴趣的商家将有一个平台可以使用，观众也可以通过 WHERE 到达。

收购条款没有披露。

与 [Loopt](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/company/loopt) 和 [Foursquare](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/company/foursquare) 等公司竞争的 Where，迄今已从 [Venrock](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/financial-organization/venrock) 和 [Kodiak Venture Partners](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/financial-organization/kodiak-venture-partners) 等投资者那里筹集了[1850 万美元](https://web.archive.org/web/20230404181901/http://www.crunchbase.com/company/where-com)。

据其关于页面的[称，到目前为止，它已经吸引了 300 万活跃用户。](https://web.archive.org/web/20230404181901/http://where.com/about/)
