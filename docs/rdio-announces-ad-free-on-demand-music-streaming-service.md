# Rdio 宣布推出无广告点播音乐流

由 Skype 创始人尼可拉斯·曾斯特罗姆和杰纳斯·弗里斯创建的数字音乐服务 Rdio 今天宣布在美国推出一项点播、无广告的音乐服务。现在，当用户在线注册加入 Rdio 时，他们将可以访问 Rdio 的 1200 万首歌曲目录中的所有完整曲目。

以前，Rdio 只向付费用户提供完整的目录。

然而，Rdio 并没有放弃付费计划。这些仍然可用，无限网络流媒体的起价为 4.99 美元，无限网络和移动流媒体的起价为 9.99 美元，Rdio 家庭计划的起价为 17.99 美元。这里的关键词是“无限制”只有付费用户才能在一个月内“无限制”访问 Rdio 的所有歌曲。同时，免费用户在一段时间内可以无限制地访问，但之后就被切断了。

对于非订阅用户，Rdio 现在会在屏幕顶部放置一个绿色条，指示本月剩余时间内您有多少免费音乐可用。当您听曲目时，小节会减小。访问级别是根据每个人的收听行为定制的。

该公司解释道:

> “一个人将获得的免费量是动态的，将基于他们自己的个人使用，并显示在他们个人资料角落的计量器中。重度用户会得到更多，但可能没有他们想要的多，轻度用户会得到比他们想要的更多。当用户一个月用完时，他们可以在下个月回来获得更多免费服务，或者他们不必等待，可以注册一个无限制的帐户。”

这种音乐可以在网上或通过 Rdio 在 Mac 和 Windows 电脑上的原生应用程序获得，但不能在移动设备上获得。注册免费计划不需要信用卡信息。

歌曲也可以通过与脸书的 Open Graph 平台的新集成共享到脸书，在那里它们被发布到你的新闻订阅、Ticker 或 Timeline。

Rdio 表示，它将免费音乐视为其服务的一种营销工具，也是让人们尝试该产品的一种方式，而不是另一种服务。这就是为什么它决定走无广告路线，而不是中断收听体验。通过提供这种格式的免费音乐，Rdio 希望在未来几个月内将更多的免费用户转化为付费用户。
