# Taglocity 旨在通过使微软的 Outlook 更像 Gmail 来改进它

作为一个 Gmail 超级用户，你不需要让我相信通过一个快速的基于网络的管理系统而不是一个臃肿的桌面软件来处理你的电子邮件的优势。我早就和微软 Outlook 说再见了，尽管不可否认的是，它在商业环境中仍然非常受欢迎，以至于它可能在未来几年内仍然是主要的专业电子邮件管理软件。

现在，TechCrunch 50 DemoPit 校友 [Taglocity](https://web.archive.org/web/20221129052257/http://www.taglocity.com/) 发布了他们所说的结合了两个世界最好的东西的 2.0 版本。它本质上是一个附加组件，旨在通过许多关键的生产力和知识共享功能来增强 Outlook 2003/2007。

讨厌那个文件夹系统？Taglocity 2.0 将把消息、联系人、日历条目等。就像 Gmail 处理标签一样。

不喜欢默认的搜索功能？Taglocity 2.0 将为关键字搜索提供额外的上下文，这样你就不用花太多时间去寻找那一条消息或联系人。

线程对话的粉丝？Taglocity 2.0 可以将与某个对话相关的所有电子邮件以类似博客的时间顺序显示在一个浏览器中，并使您能够将它们分组在一起。

Taglocity 2.0 for Outlook 还提供了与另一个产品 Taglocity Groups 的集成，他们称之为“企业 Web 2.0 启发的群体电子邮件协作服务”，使用户能够共享信息，减少电子邮件过载，并提高生产力。

你可以自己找出这意味着什么:在这里免费获得 Taglocity 2.0 [(还有一个高级版本，每个用户 99 美元)。](https://web.archive.org/web/20221129052257/http://www.taglocity.com/downloads.html)

这家初创公司将要面临的一大挑战是，人们需要手动标记他们在 Outlook 中的所有现有项目，这对于那些经常使用 Outlook 的人(即 Taglocity 的核心受众)来说，势必会花费大量时间。也许像 [TagCow](https://web.archive.org/web/20221129052257/http://www.beta.techcrunch.com/2008/11/11/tagcow-focuses-photo-tagging-product-on-the-enterprise/) 这样的东西在这里也能派上用场。

其他试图让 Outlook 变得更好的公司包括 [Xobni](https://web.archive.org/web/20221129052257/http://www.crunchbase.com/company/xobni) 和 [Add-in Express](https://web.archive.org/web/20221129052257/http://www.crunchbase.com/company/add-in-express-ltd) 。
