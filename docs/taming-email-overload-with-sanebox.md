# 用 sane box TechCrunch 驯服电子邮件过载

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/04/11/taming-email-overload-with-sanebox/>

TechCrunch 创始人[迈克尔·阿灵顿](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/person/michael-arrington) [称电子邮件超载为“沟通危机”，早在 2008 年就发出挑战](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2008/03/23/a-crisis-in-communication/)“有人需要创造一种新技术，让我们享受生活，但不会错过重要信息。”在 [SaneBox](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/company/sanebox) [的企业家们阅读了这篇](https://web.archive.org/web/20221206072121/https://www.sanebox.com/why)和其他由 [Fred Wilson](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/person/fred-wilson) 和 [Bijan Sabet](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/person/bijan-sabet) 撰写的文章，并着手建立一个更好的收件箱。经过一个月的测试， [SaneBox](https://web.archive.org/web/20221206072121/http://www.sanebox.com/) 确实帮助我控制了收件箱，并接受了迈克的挑战。

已经提出了许多解决电子邮件过载的方案，但成效有限。阿林顿[写了](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2010/03/29/want-me-to-read-your-email-pay-me/)一个叫做“注意力拍卖”的服务，后来改名为 [attn.me](https://web.archive.org/web/20221206072121/http://www.attn.me/) ，人们付钱给你让你阅读他们的电子邮件。有些人干脆退出电子邮件。公司试图禁止它。其他人[写简短的电子邮件](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2010/09/18/3-sentence-emails/)并使用迫使你保持简短的系统[。有些人取消了你的电子邮件订阅，有些人甚至试图把他们的收件箱移到浴室。](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2011/07/11/shortmail/)

另一个驯服收件箱的方法是删除不重要的东西。随着电子邮件数量的不断增加，解决电子邮件过载的唯一真正方法是分清主次。Arrington 手动做到了这一点，他在[中写道](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2008/03/23/a-crisis-in-communication/)他将如何“扫描发件人和主题字段以获得高回报的信息”。我认识的不会浪费我时间的人，或者我与之有真诚友谊的人。

当谷歌[推出](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2010/08/30/gmail-priority-inbox/) [Gmail](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/product/gmail) 优先收件箱时，谷歌实现了自动化。这个功能很有帮助，也有它的粉丝，但对我来说还不够好。

在 Priority Inbox 推出之前，由 Stuart Roseman 创立的 SaneBox 团队正在私下测试一个类似的概念。Gmail 的解决方案让他们大吃一惊。SaneBox 的增长副总裁德米特里·列昂诺夫回忆说，那是公司的可怕时期。他们的许多用户转而使用 Gmail，但大多数人很快又回来了。

我试过优先收件箱，但 SaneBox 更适合我。它不会神奇地消除你所有的电子邮件或让你的收件箱立即为零。但是，你的收件箱缩小了，你可以专注于更高优先级的邮件。SaneBox 表示，其将普通用户 58%的电子邮件移出收件箱。您的里程可能会有所不同。

当被问及为什么 SaneBox 的小型自筹资金团队认为他们可以比巨头谷歌更好地解决问题时，列昂诺夫说“这是一个非常困难和昂贵的问题，一个巨大的边缘案例。我们所做的这种个性化分析需要基于每个用户的大量基础设施，这对于像 Priority Inbox 这样的免费服务来说成本太高了。”在免费试用后，SaneBox 用户每月支付高达 5 美元的服务费用，据称每周平均节省 2 小时。

Sanebox 的解决方案可以让你训练收件箱，但与优先收件箱不同，你可以看到你所有的训练并进行调整。它也比构建自己的自定义过滤器或电子邮件规则更容易、更快速、更强大。

通过减少收件箱中重要邮件的数量，你就减少了精神注意力转移的成本。从重要邮件切换到不重要邮件需要时间。所以花在转换上的时间越少，你的效率就越高。如果时间允许的话，你也可以查看不太重要的邮件。

SaneBox 说，这就像有一个非常好的行政助理，他把不受欢迎的访客拦在门口，让你在一天中更有效率。SaneBox 还效仿了[大卫艾伦](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/person/david-allen-4)的[卓越的 GTD](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2011/08/10/keen-on-david-allen-how-to-get-things-done-in-america-tctv/) (完成事情)理念，比如专注于[最重要的事情](https://web.archive.org/web/20221206072121/http://www.sanebox.com/blog/sanebox-ranking-mean/)“现在”并避免分心。

在解释它是如何工作的之前，你需要明白它不是什么。SaneBox 不是插件，也不是下载。SaneBox 不从事垃圾邮件控制业务。它不会阅读你的邮件正文，不会更改任何邮件头，也不会将邮件存储在它的服务器上。尽管大多数 SaneBox 用户使用 Gmail，但它几乎可以与任何电子邮件服务兼容。

设置好之后，它会扫描您的收件箱标题，以确定邮件是否重要，是否应该留在您的收件箱中。它使用自己的智能过滤算法并结合您的个人培训，将不重要的邮件移动到@SaneLater 文件夹(或标签)。

马上，SaneBox 工作得很好。如果你添加你的社交网络，它会优先显示你在脸书、LinkedIn 和 Twitter 上关注的人的电子邮件。如果你的收件箱里有不重要的东西，就把它移到你的@SaneLater 文件夹里。SaneBox 的算法可以识别来自同一域的个人和企业或批量电子邮件之间的差异。

我发现我每天只需要查看一两次我的@SaneLater 文件夹。当这些优先级较低的邮件被集中在一起时，浏览它们会快得多。您的@SaneLater 电子邮件不会丢失。您总是可以在他们的文件夹中找到他们，他们出现在搜索中。

SaneBox 还会按照您设置的时间间隔向您的收件箱发送一个@SaneLater 消息摘要列表。您可以随时将电子邮件从@SaneLater 移动到收件箱，这些培训可以针对这一封电子邮件或来自同一地址的所有邮件进行记忆。每个接触训练可以通过 SaneBox 设置页面进行调整。

SaneBox 还有另外两个非常有用的特性。将电子邮件移至@SaneBlackHole，您将再也不会收到来自该地址的电子邮件。这比构建自己的自定义过滤器或取消订阅要快得多。

还有一个@SaneRemindMe 文件夹，可以存放你想确保得到跟进的电子邮件。当您发送或回复电子邮件时，将地址添加到 to: cc:或 bcc:中，如 tomorrow@sanebox.com、Friday@sanebox.com、Dec31@sanebox.com 或 3h@sanebox.com(在 3 小时内)，SaneBox 将在未来的时间向您发送提醒电子邮件。这个功能类似于 [Nudgemail](https://web.archive.org/web/20221206072121/http://www.nudgemail.com/) 和 [followup.cc](https://web.archive.org/web/20221206072121/http://www.followup.cc/) 所提供的。但是有了 SaneBox，如果你的邮件得到回复，提醒就取消了。

在谷歌的免费优先收件箱之外，SaneBox 确实有一些竞争对手。OtherInbox 提供了一个免费的“组织者”功能，可以将收据、时事通讯和社交媒体邮件分类到文件夹中。 [OtherInBox](https://web.archive.org/web/20221206072121/http://www.crunchbase.com/company/otherinbox) 是 2008 年 TC 50 决赛选手。我发现我花了很长时间来搜索我的电子邮件，它的表现远不如 SaneBox。它还创建了比我想要的更多的文件夹。他们[被电子邮件认证和信誉监测公司](https://web.archive.org/web/20221206072121/https://beta.techcrunch.com/2012/01/10/return-path-expands-to-austin-acquires-tc50-finalist-otherinbox/) [Return Path](https://web.archive.org/web/20221206072121/http://www.returnpath.net/) 收购，该公司的使命是帮助营销人员和出版商接触受众。这与 SaneBox 这样的初创公司截然不同，后者的客户是付费电子邮件超载患者。

这个领域的另一个参与者是付费服务 [AwayFind](https://web.archive.org/web/20221206072121/http://awayfind.com/) 。它的目标是让你只收到 3%的超紧急邮件，即使你不在收件箱附近。这是一个不同于 SaneBox 的解决方案，sane box 试图从收件箱中取出大部分电子邮件。
