# Overstock 可能会用新的储物柜交付系统来挑战亚马逊(和谷歌)

储物柜是电子商务的新宠。亚马逊[最近推出了](https://web.archive.org/web/20221006125753/https://beta.techcrunch.com/2012/07/30/amazon-lockers-silicon-valley/)一个储物柜送货计划，允许顾客将他们的货物送到附近的储物柜。这项服务对于不在家接收快递的消费者尤其有用。谷歌刚刚以 2500 万美元收购了提供储物柜交付服务的服务公司 buffer box[。电子商务公司 Overstock 也盯上了储物柜领域。](https://web.archive.org/web/20221006125753/https://beta.techcrunch.com/2012/11/30/google-acquires-waterloo-based-e-commerce-startup-amazon-locker-competitor-and-yc-graduate-bufferbox/)

我们刚刚收到该公司的一份调查，其中询问了我们对亚马逊储物柜的感受以及我们是否使用过该服务。在调查中，Overstock 解释了他们潜在的储物柜系统:

*您的在线订单可能会被送到安装在安全位置(办公用品商店、药店、便利店等)的自动存物柜中。).无需将包裹运送到您的家庭住址，而是可以将包裹递送到您在结账时选择的指定储物柜位置。这项服务不收取额外费用。当包裹被送到储物柜时，您会收到一封确认电子邮件或短信，其中包含收件代码。使用储物柜的触摸屏，输入提货代码以打开包含包裹的储物柜。*

对于 Overstock 来说，考虑一个储物柜系统是有意义的。这家公司 2011 年的收入为 10.5 亿美元，销售从家具到珠宝的各种昂贵商品。储物柜服务不仅可以帮助那些没有人在家的人全天候接受快递，还可以省去你在营业时间去 UPS 或 FedEx 仓库取包裹和排队等候的麻烦。亚马逊在史泰博商店、7-11 便利店和 RadioShacks 都有储物柜。谷歌似乎也相信模型的有效性，因此我们可能会看到更多的电子商务网站采用这种趋势。
