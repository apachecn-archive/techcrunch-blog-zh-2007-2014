# Tween 虚拟世界制作商 Woozworld 融资 600 万美元；前迪士尼高管加入董事会

**独家-**[Woozworld](https://web.archive.org/web/20230204201326/http://www.woozworld.com/)，一家为青少年(更确切地说，是 9-14 岁的孩子)开发社交游戏/虚拟世界的开发商，已经从 Telesystem 和 iNovia Capital 以及不知名的天使投资人那里筹集了 600 万美元的新资金[。此外，迪士尼前总经理兼高级副总裁伯纳德·格申(Bernard Gershon)也加入了这家初创公司的董事会。](https://web.archive.org/web/20230204201326/http://www.crunchbase.com/company/woozworld)

这笔资金将用于扩大 Woozworld 的服务，使其包括移动平台，并加强其营销努力，以扩大覆盖范围。

Woozworld 本质上提供了一个用户生成的在线社交平台，青少年可以在这个平台上构建整个世界，与同龄人互动，建立虚拟企业，或者根据自己的选择建立餐馆、酒店和游戏。

迄今为止，该公司已经注册了超过 700 万个虚拟形象，在 18 个月内创建了超过 1400 万个虚拟空间。Woozworld 也像野草一样成长:在过去的六个月里，它的用户群增长了两倍多。

Woozworld 表示，它目前每月接待来自 180 多个国家的 1500 多万独立访客，每月页面浏览量超过 7600 万

格尔森加入了 Woozworld 的顾问委员会，被认为是流媒体的先驱，目前是数字媒体战略实践公司 GershonMedia 的总裁。

他也是华特·迪士尼公司的高级副总裁兼公司战略、业务发展和技术总经理。
