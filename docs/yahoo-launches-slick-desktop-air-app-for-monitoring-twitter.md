# 雅虎推出用于监控 Twitter 的光滑桌面 AIR 应用

![](img/f9a81525a7e21d215d5157e7bd245ef8.png) [雅虎](https://web.archive.org/web/20230211161008/http://yahoo.com/)昨天发布了一款名为[副业](https://web.archive.org/web/20230211161008/http://sideline.yahoo.com/)的 Adobe AIR 驱动桌面应用，再次验证了 [Twitter](https://web.archive.org/web/20230211161008/http://twitter.com/) 对于[实时搜索](https://web.archive.org/web/20230211161008/https://techcrunch.com/2009/03/05/its-time-to-start-thinking-of-twitter-as-a-search-engine/)的威力。在试用之后，我不得不说它看起来和感觉都很好，但除此之外，我没有真正的动力继续定期使用它。

那么它是做什么的呢？

Sideline 是一个简单的 Twitter 监控工具，让你有机会掌握微共享服务和/或你输入到应用程序中的关键词的最新趋势。它有一个自动刷新功能(你可以调整它，让搜索结果在 1 分钟到 1 小时之间重新加载)，一个通知系统，提醒你在覆盖图中出现新的关键词，无论你在做什么，以及只寻找包含你正在跟踪的关键词的收藏夹推文的能力。

到目前为止，没有什么特别的，但是 nice 的好处在于它可以让你创建所谓的搜索组，这些搜索组可以将多个关键字聚集在一起进行跟踪。这基本上允许任何人创建一个 techcrunch 群组，并跟踪不同的关键词和短语，如“TechCrunch”、“crunchgear”、“michael arrington”等。高级搜索功能也很棒，它可以让你广泛地过滤结果，例如通过人、标签、“问问题”、负面或正面含义(由微笑决定)等等。**更新:**正如一位评论者指出的，这实际上是在 [Twitter 的高级搜索功能](https://web.archive.org/web/20230211161008/http://search.twitter.com/advanced)之上的一层。

它很光滑*和*有用，但无论从哪方面看都没什么大不了的，尤其是因为它不是一个能让你直接或公开发送 Twitter 消息的功能性客户端。我想知道有多少人在试用后会有效地继续使用它，但我怀疑会有很多人。

桌面应用程序来自雅虎的用户界面(YUI)团队，他们的[详细解释了](https://web.archive.org/web/20230211161008/http://yuiblog.com/blog/2009/03/31/sideline-beta-released/)是什么激发了他们开发它，以及他们是如何做的。副业是用雅虎开发的。用户界面库(YUI)和 HTML/JS/CSS。在 BSD 许可和 GitHub 托管的[条款下，代码是开源的。](https://web.archive.org/web/20230211161008/http://github.com/cauld/sideline/tree/master)
