# 黑客通过提取指纹绕过苹果的触控 ID

指纹扫描仪总是容易受到黑客的攻击，这些黑客愿意付出额外的努力来绕过它们。多年来，我们已经看到人们使用[巨美熊](https://web.archive.org/web/20230304095829/http://www.zdnet.com/sweet-bypass-for-student-finger-scanner-1339306878/)、[培乐多](https://web.archive.org/web/20230304095829/http://www.engadget.com/2005/12/11/play-doh-fingers-can-fool-90-of-scanners-sez-clarkson-u/)和更复杂的技术来绕过这些生物扫描仪。因此，苹果在新 iPhone 5s 上的[触控 ID 指纹扫描仪](https://web.archive.org/web/20230304095829/https://techcrunch.com/2013/09/10/apples-touch-id-a-500ppi-fingerprint-sensor-built-into-iphone-5s-home-button/)也容易受到这类攻击也就不足为奇了。正如德国的混沌计算机俱乐部(CCC)今天宣布的那样，它已经成功地通过创造一个假手指来绕过 TouchID ，这个假手指使用提取的指纹来欺骗扫描仪，使其相信它正在与合法的主人打交道。

但是让我们来看看这个黑客。让这个工作起来并不像 CCC 黑客在他们的新闻稿或[这个视频](https://web.archive.org/web/20230304095829/http://www.youtube.com/watch?v=HM8b8d8kSNQ)中让你认为的那么容易:

首先你需要某种彩色粉末或强力胶来提取指纹。然后你必须扫描指纹，将它倒置，并以 1200dpi 或更高的分辨率打印在透明纸上。在那之后，你通过将粉红色乳胶或白木胶涂抹到墨粉在透明片上创建的图案中来构建你的假手指，并等待它凝固。最后，CCC 写道，“将薄乳胶片从纸上提起，吹气使其稍微湿润，然后放在传感器上以解锁手机。”这种方法应该适用于目前市场上几乎所有的指纹扫描仪。

如果有人愿意经历这一切来侵入你的手机，你可能有比指纹安全更大的问题。此外，鉴于大多数 iPhone 用户今天可能甚至不使用 PIN 码来保护他们的设备，触控 ID 仍然标志着智能手机安全方面的一大进步——即使有人可能会提取你的指纹并绕过它。
