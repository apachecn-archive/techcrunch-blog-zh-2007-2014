# 伦敦的科技初创公司投资者开始在硅环岛办公

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/03/28/londons-tech-startup-investors-start-taking-offices-in-silicon-roundabout/>

老街地铁站(Old Street tube station)、肖尔迪奇(Shoreditch)和霍克斯顿(Hoxton)周围的地区，就在伦敦平方英里(Square Mile)的北部，位于东伦敦的边界，已经被轻松地称为“硅环岛”,因为那里现在有一个可识别的创业公司集群。自 2008 年以来，我一直在跟踪这一趋势[。但缺少的是风投。虽然在硅谷，沙丘路的风险投资公司距离帕洛阿尔托和门洛帕克只有很短的车程，但伦敦的风险投资公司更喜欢呆在梅菲尔，通常靠近支持他们基金的有限合伙人的办公室。它离老街不远——3.2 英里——但它的文化和风景不同。但为了紧跟当今科技的发展，投资者与初创公司共处一地是非常有意义的。](https://web.archive.org/web/20230323051709/http://eu.beta.techcrunch.com/2008/07/30/so-where-are-londons-existing-organic-techhubs/)

大约一年前引领先锋的是[白熊场](https://web.archive.org/web/20230323051709/http://whitebearyard.com/)，这是一个由三位著名天使投资人支持的大型空间。不严格地说是在“硅环岛”，但是他们的 Clerkenwell 基地离这里只有 5 分钟的出租车或自行车路程。

但很快，这个地区将迎来著名的创业加速器项目 [Seedcamp](https://web.archive.org/web/20230323051709/http://seedcamp.com/) ，被一些人称为欧洲的 Y Combinator。我们已经听说他们准备从他们在伦敦西区的位置搬到他们投资的公司去。

今天新闻[传来](https://web.archive.org/web/20230323051709/http://blog.indexventures.com/doubling-down-on-london-east-west/)领先的科技风险投资 [Index Ventures](https://web.archive.org/web/20230323051709/http://www.indexventures.com/) 将在该地区购买一些办公桌(我们听说有三张)，与新房东 [Moo Studios](https://web.archive.org/web/20230323051709/http://uk.moo.com/) 共同位于 Scrutton 街，也是 Groupspaces、Lightbox、Soundcloud、Tech PR SparkPR 和 Tweetdeck 的所在地。2006 年，Index 是 Last.fm 和 Moo 的早期投资者，这两家公司都是“环岛场景”的后代，该场景还包括霍克斯顿的 Songkick 和肖尔迪奇的 Moshi Monsters。

Index 现在距离能够在 City Road(Editd 下面)的 startup-co working centre[Tech hub](https://web.archive.org/web/20230323051709/http://techhub.com/)进行开放办公时间也只有五分钟的步行路程，该中心包含 PeerIndex (Klout 的竞争对手)、Squadify、Pora Ora、Twileshare、Teamly 和 OpenSignalsMaps 等初创公司，其中包括 Tech PR、Ballou PR。[利益声明，我是 TechHub 的顾问]。

Index [说](https://web.archive.org/web/20230323051709/http://blog.indexventures.com/doubling-down-on-london-east-west/)他们想“做一些让我们更接近合作团队的事情，在开放的办公时间会见一些我们不认识的新公司，并让自己置身于欧洲最热门的创业区之一。”

我们被告知，所有在“西方”的指数团队都会在某个时候花时间在新的“东方”办公室——但公平地说，如果风投们不在办公室，他们通常会在飞机上。他们现在必须拿到规定的 [T 恤](https://web.archive.org/web/20230323051709/http://riglondon.spreadshirt.co.uk/)。
