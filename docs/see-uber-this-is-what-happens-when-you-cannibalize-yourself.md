# 看，优步——这就是你把自己拆成碎片的后果

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/03/15/see-uber-this-is-what-happens-when-you-cannibalize-yourself/>

优步的司机被激怒了。但这并不令人惊讶…至少对任何关注此事的人来说是这样。

今晚，一群 UberBLACK 司机聚集在该公司位于旧金山的 SOMA 总部外，抗议该公司的商业行为。除此之外，该组织还抱怨司机缺乏健康保险或股票期权。该组织还抗议较低的工资和一些最近被解雇的合伙人。

最后一部分对优步首席执行官特拉维斯·卡兰尼克来说是真实的，他在抗议期间并不在现场——相反，他正在费城吃晚餐，该公司去年夏天已经扩展到了这个市场。但他说，他从旧金山向他汇报的人那里了解到，20 多名抗议者中的大多数是优步合伙人，他们的账户因质量问题而被停用。

他说，“让一个伴侣失去活力从来都不是一件容易的事，”但他表示，当司机因不安全驾驶或不了解这座城市而获得较差评级时，这么做是必要的。

工作保障可能是该公司的一个问题，因为抗议者提到了 2 月份发生的大规模裁员。(公司对此有争议。)但更大的问题似乎是，优步正处于一场大变革之中，采取了与 Lyft 和 SideCar 类似的战略，这可能会蚕食其遗留的 UberBLACK 和 UberSUV 业务。

## 每个人的私人司机

大约两年半前，当优步第一次在三藩市推出[时，它被宣传为每个人的私人汽车服务。正如卡兰尼克所说，公司的成立主要是为了让他和他的几个朋友可以像芭蕾舞者一样乘坐他们自己的按需黑色汽车服务在镇上兜风。](https://web.archive.org/web/20230329064124/https://techcrunch.com/2010/10/15/ubercab-closes-uber-angel-round/)

但是，他们并不是唯一在寻找快速、可靠的替代旧金山糟糕的出租车服务的人。事实证明，旧金山有更多的乘客愿意支付高价购买舒适的汽车，他们知道汽车会准时到达，他们可以通过移动应用程序打招呼，而不必与调度员交谈。与从来没有来过的出租车相比，多付一点钱叫一辆优步似乎是显而易见的。

事情好了大约两年，然后竞争来了。使用与优步相同的基本理念——即使用移动应用程序来促进司机和乘客之间的沟通——一批新服务突然出现，提供他们自己的更便宜的按需交通版本。

但是，虽然这些服务承诺与优步的按需服务一样可靠，但它们的成本要低得多，因为它们使用出租车司机——或者在 Lyft 和 SideCar 的情况下，它们已经建立了一个由普通司机驾驶自己的车辆的车队。

## UBERx 效应

优步的回应是向可能没有出租车或豪华轿车运营执照的司机开放其平台。它还将其 UberBLACK 服务的费率降低了 10%。因此，该公司转向更便宜的选择并不令人惊讶——它去年秋天在三藩市首次推出 UBERx，此后不久宣布它打算在所有主要市场为其传统的黑车服务提供低成本替代品。

但与 Lyft 和 SideCar 采取相同策略的 UberBLACK partners，在过去几年中受益于其增长，他们对此感到恼火。这是因为随着公司为更便宜的替代品让路，他们看到自己的份额开始减少。

就其本身而言，优步最近的票价下调及其[对拼车社区](https://web.archive.org/web/20230329064124/https://techcrunch.com/2013/02/25/uber-uberx-ride-share/)司机的接纳可能是必要的，因为它看到了来自更便宜的替代品的竞争加剧。这意味着推动客户使用其低成本的 UBERx 服务，即使这是以 UberBLACK 和 UberSUV 为代价。

卡兰尼克承认，几个月前 UberBLACK 的价格不合适，称很难找到合适的市场均衡。他还承认，最近几个月，UberBLACK partners 可能感到手头拮据，因为每小时和每周的平均票价略有下降。他将这在一定程度上归因于第一季度的季节性，因为假期后事情会放缓。但他说，三藩市每小时的票价仍然远远高于优步其他主要城市，大约为 30%到 50%。

尽管来自其他服务提供商的竞争，优步继续增长。“旧金山今年的增长非常接近我们去年看到的增长，”卡兰尼克告诉我。“我们正在等待增长放缓，但还没有看到。”

这对优步来说是个好消息，但不一定对所有的司机都是。
