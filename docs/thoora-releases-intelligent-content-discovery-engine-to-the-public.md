# Thoora 向公众发布智能内容发现引擎

自从 2009 年[发布以来，](https://web.archive.org/web/20230203094554/https://techcrunch.com/2009/09/15/tc50-thoora-brings-the-days-news-cycle-into-clouds-of-content/)内容发现引擎 [Thoora](https://web.archive.org/web/20230203094554/http://www.thoora.com/) 一直处于私人测试阶段。但是今天，这个结合了聚合、管理和搜索个性化新闻流的网站，正在走出私人测试阶段，现在是公开的。

毫无疑问，当涉及到新闻网站、博客和社交网络时，内容会超载。Thoora discovery engine 为用户提供了一种新的方式来发现、监控和分享特定主题的网络新闻。

Thoora 的专利引擎搜索了 2800 万个在线资源，并使用 100 多个信号来查找和排列内容。Thoora 允许用户按主题整理结果，引擎从整理中学习，并开始提供更个性化的提要。

你可以用多个关键词建立主题，通过“收藏”或删除文章、帖子和特定来源来管理内容，通过 Twitter 流过滤，等等。用户也可以将他们的 Thoora 搜索分享到他们的社交网络，包括脸书和推特。

那么是什么让 Thoora 花了这么长时间向公众推出它的产品呢？这家初创公司表示，自 2009 年以来，社交媒体和内容呈爆炸式增长，Thoora 正在努力将这一点集成到应用程序中。这家初创公司还在开发自己的专有技术，用于管理和聚合新闻。
