# 谷歌承诺为 Nexus S 安装 Android 4.0，“理论上”也会为姜饼设备安装 Android 4.0

在我们熟悉了几个小时的冰淇淋三明治之后，每个人心中都有一个迫在眉睫的问题:Android 4.0 更新什么时候会在现有设备上推出？没人知道。但是在和一些权威人士讨论过这个问题之后，我们至少有了一些线索。

谷歌的 Android 产品经理 Gabe Cohen 和 Android 用户体验总监 Matias Duarte 已经承诺为 Nexus S 提供冰淇淋三明治，并暗示所有其他 Android 2.3 姜饼设备“理论上”也应该获得 ICS。这是官方的报价:“目前正在为 Nexus S 发布冰淇淋三明治。理论上应该适用于任何 2.3 设备。”据 [Engadget](https://web.archive.org/web/20230204205456/http://www.engadget.com/2011/10/19/google-confirms-nexus-s-will-get-ice-cream-sandwich-for-real/) 报道，Nexus One 仍然没有一个明确的计划。

HTC 已经决定对这款产品持观望态度[,称他们确实希望向大众发布 ICS，但首先必须找到一种方法将他们的 Sense UI 放在上面。就像科恩和杜阿尔特一样，HTC 拒绝给出时间表。](https://web.archive.org/web/20230204205456/https://techcrunch.com/2011/10/19/htc-remains-coy-about-ice-cream-sandwich-update-schedule/)

在寻找答案的过程中，我决定也联系四大运营商。T-Mobile 从未回应，而其他公司要么没有评论，要么没有关于 ICS 更新计划的信息。

许多今年早些时候购买的 Froyo 设备的所有者仍然在等待姜饼，所以希望运营商和手机制造商有一个更有效的 Android 4.0 计划。

但我不会打赌。
