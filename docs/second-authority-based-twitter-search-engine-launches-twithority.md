# 第二个基于权威的 Twitter 搜索引擎发布:Twithority

![](img/aaab5d1576a9824dc0de199712027ed4.png)

当博客们继续争论无中生有的细节时，又一位企业家走出去，为 Twitter 建立了一个过滤搜索引擎。几天前[tw ority 推出了](https://web.archive.org/web/20221006010153/http://www.beta.techcrunch.com/2008/12/28/ask-and-ye-shall-receive-twitority-launches-authority-based-twitter-search/)(虽然现在看起来已经关闭)。今天，是[推特](https://web.archive.org/web/20221006010153/http://twithority.com/)，由[察沃](https://web.archive.org/web/20221006010153/http://www.beta.techcrunch.com/2008/08/20/userplane-founder-peels-away-from-aol-to-found-new-startup-tsavo/)的同一批人创建了 [Daymix](https://web.archive.org/web/20221006010153/http://daymix.com/) 。

Twithority 返回结果的速度明显更快，而且看起来更早(1，000 个结果对 50 个结果)。它按照排名(排名最高的用户优先)和时间(最近的用户优先)对前 10，000 名 Twitter 用户的结果进行排名。

有些人会继续讨厌关于 Twitter 权限的辩论，但最终市场会决定是否有对这种产品的需求。
