# 擎天柱流行将花费不到 1000 美元，仍然无关紧要

[![](img/cb1572b0b5095c713f69d2157abf9f2b.png "Optimus Popularis 3D prototype")](https://web.archive.org/web/20221205093700/http://www.crunchgear.com/2010/05/13/optimus-popularis-will-cost-less-than-1000-remain-irrelevant)

我想到阿尔特·列别杰夫的古怪作品，就像我想到昂贵、怪异的性玩具一样——也许有人会买一个，但无论你怎么分析，它都不会畅销。无论如何，擎天柱流行键盘在每个键上都使用了小屏幕，就像老式的[擎天柱](https://web.archive.org/web/20221205093700/http://crunchgear.com/search/optimus%20maximus)。然而，它将使用发光二极管代替有机发光二极管。

Popularis 的售价将低于 1000 美元，应该会及时大规模生产，让所有人都忘记它。
