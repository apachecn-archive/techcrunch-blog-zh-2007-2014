# KickSat:将微型 DIY 卫星送入太空

地面控制少校你:只需 300 美元，你也可以将一颗卫星送入太空。怎么会？好吧，一个来自纽约伊萨卡的年轻人正计划将一盒小型、独立的太阳能无线电发射器发送到太空——总共约 300 个——并从近轨道上观看它们的发射。就像谋杀微小的太空乌鸦一样，Sprite 卫星会在它们环绕地球时，露出它们的位置，并以 300 美元的价格，露出你名字的首字母。

在项目页面中:

> 精灵只有两张邮票大小，但有太阳能电池、无线电收发器和带存储器和传感器的微控制器(微型计算机)——许多更大的航天器会有的功能，只是按比例缩小了。这第一个版本除了传输它的名字和一些数据之外不能做什么——把它想象成一个缩小的人造卫星——但未来的版本可以包括任何类型的传感器，从温度计到相机。

一旦卫星进入轨道，业余无线电操作员将使用 [Genso 项目](https://web.archive.org/web/20230204232739/http://www.genso.org/)跟踪它们。这些小家伙现在只有基本的功能，但创作者希望在下一代中添加温度计和其他传感器。

[Kickstarter 项目](https://web.archive.org/web/20230204232739/http://www.kickstarter.com/projects/251588730/kicksat-your-personal-spacecraft-in-space)将融资 3 万美元，还有一段路要走。然而，如果你想驾驶自己的人造卫星，这可能是值得的。
