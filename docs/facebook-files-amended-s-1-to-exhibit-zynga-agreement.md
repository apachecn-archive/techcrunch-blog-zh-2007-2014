# 脸书修改后的 S-1 展示了去年提交的 Zynga 协议

脸书刚刚提交了一份 S-1 修正案，展示了它与 Zynga 之间的协议。这两份[开发商](https://web.archive.org/web/20230323032836/http://www.sec.gov/Archives/edgar/data/1326801/000119312512046715/d287954dex1012.htm) [协议](https://web.archive.org/web/20230323032836/http://www.sec.gov/Archives/edgar/data/1326801/000119312512046715/d287954dex1013.htm)文件与去年 Zynga 自己的 S-1 修正案中提交的文件相同，不包括重要的新信息。

这些展品确实说明了脸书如何承诺帮助 Zynga 在 FarmVille.com 等 Zynga 网站上做广告，并分享这种合作关系带来的收入。这不应该与分享 Zynga 在 Facebook.com 的游戏广告收入相混淆。脸书还附上了 2005 年的股票计划，以及给主要高管的聘书。

开发者协议文件似乎与 Zynga 提交时的编辑内容相同。去年，TechCrunch 作家 Eric Eldon [报道了这些文件](https://web.archive.org/web/20230323032836/http://www.insidesocialgames.com/2011/07/19/zynga-s-1-update-traffic-revenue/)，解释了它们如何显示:

> 脸书似乎还向 Zynga 保证了一定的增长目标，以换取继续投资该平台上的游戏，无论是网络游戏还是移动游戏。脸书已经允许 Zynga 创建某种“Zynga 平台”。它还为开发者提供了新功能，包括拟议中的“游戏朋友协议”API，显然为社交游戏玩家提供了一种新的寻找和一起玩的方式。

关于广告收入分成:

> 乍一看，这些条款似乎表明 Zynga 与脸书达成了一项特殊协议，Zynga 从脸书的广告部门获得一部分广告收入，这些广告部门与 Zynga 的游戏一起在 canvas 应用程序中运行。然而，条款规定这不是 canvas 应用的广告收入，而是指 Zynga 网站[也称为 Zynga 游戏页面，不包括 Canvas 页面或 www . Facebook . com 上的任何其他页面]。
> 
> 我们就此事询问了脸书，得到了这样的回答:“我们没有与任何开发者达成协议，包括 Zynga，分享他们的脸书画布应用程序旁边的广告收入。我们确实与 Zynga 达成了协议，未来将合作在他们位于脸书以外的地产上提供广告，但我们目前还没有开始这项工作的时间表。”

这可能是未来脸书场外广告网络的第一个迹象，在这种网络中，网站可以托管广告，这些广告采用脸书自己的定向系统，向当前登录脸书的访问者显示相关广告。如果脸书将该系统推广到 Zynga 之外的网站，这将成为其巨大的收入来源。

该协议的一些具体要点是:Facebook 平台将被整合到 Zynga 移动游戏中，Zynga Properties 和 FB 将成为唯一和专属的社交平台

*   对于 Zynga 自己的网站，“Facebook 平台将被整合到 Zynga 手机游戏中，Zynga Properties 和 FB 将成为唯一和专属的社交平台”。这允许 Zynga 在 Google+等其他社交网络上开发游戏，但不允许使用其他身份提供商在其网站上开发游戏。
*   “FB 不得在脸书网站或 Facebook 平台上提供或以其他方式提供任何脸书游戏”，这意味着脸书不能创建自己的第一方游戏来与 Zynga 竞争。
*   Zynga 将在其所有“涵盖的 Zynga 服务”上独家使用脸书积分作为其游戏内购买处理方法，包括黑手党战争、FarmVille 和任何其他利用脸书数据的游戏。
*   Zynga 对自己游戏网站的“所有内容和材料、维护和运营”负责，除了它托管的脸书广告
*   Zynga 不会“删除、最小化、框住或以其他方式禁止任何页面的完整显示”，因此它可能会掩盖其网站上的脸书广告或点击这些广告时弹出的覆盖广告。
