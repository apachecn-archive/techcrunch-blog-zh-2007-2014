# 是的，BuzzFeed 也在做赞助问答游戏

定义 BuzzFeed 明显成功的广告方式的一个因素是其广告反映其编辑内容的方式。所以有了[赞助的列表](https://web.archive.org/web/20221207201123/https://beta.techcrunch.com/2013/08/13/buzzfeed-pepsi-listiclock/)，一个[赞助的 gif 馈送](https://web.archive.org/web/20221207201123/https://beta.techcrunch.com/2013/11/20/buzzfeed-gif-feed/)，以及最近的赞助测验。

我当然注意到在过去的几个月里，BuzzFeed 问答游戏有所增加，通常会问一些不同的问题，“你是哪种 X 或 Y？”(我可能应该透露一下，我为这个网站做了一点自由写作，但这并没有减少我对网站内容的困惑。)BuzzFeed 创意服务总监梅丽莎·罗森塔尔(Melissa Rosenthal)告诉我，多年来，小测验一直是内容的一部分，但它们最近开始流行是因为“大受欢迎”，即 1 月份发布的一个小测验，据称它决定了“[你实际上应该住在哪个城市？](https://web.archive.org/web/20221207201123/http://www.buzzfeed.com/ashleyperez/what-city-should-you-actually-live-in)

自然地，BuzzFeed 看到了编辑问答游戏的流行，并随之推出了广告——罗森塔尔说，她的团队遵循“编辑看到的许多趋势”，因此它“积极推销”赞助的问答游戏，迄今为止已发布了约 10 个。这些成功包括一个由 Mattel 赞助的问答游戏“你是哪个芭比娃娃？“还有一部 HBO 赞助的，”[在《权力的游戏》中你会怎么死 BuzzFeed 表示，这两条帖子的浏览量都超过了 100 万次，《权力的游戏》的脸书分享量达到了 7.5 万次，而《芭比娃娃》的分享量为 16.1 万次。](https://web.archive.org/web/20221207201123/http://www.buzzfeed.com/hbogameofthrones/how-would-you-die-in-game-of-thrones)

罗森塔尔说，编辑团队和广告团队之间有很大的“教会和国家”分歧，但同时他们“得到了同样的数据支持”在这种情况下，“我们已经看到品牌测验和社论一样引起共鸣。”事实上，尽管我提到的两个赞助测验被标记为由“BuzzFeed Partners”编写，但我猜许多参与者并没有注意到差异。

罗森塔尔还指出，这些测试对广告商来说很有用，因为它们“可以潜在地告诉品牌人们使用它们的习惯”。他们还可以纳入“真正的研究”，例如通过询问您的生活方式，并使用这些答案来识别可能非常适合的车型。

当被问及她是否担心测验可能只是一种时尚时，罗森塔尔回答说，“我们一直在考虑这个问题。”到目前为止还没有“下降”，但她补充道，“这并不意味着我们没有考虑它们可以采取什么新的形式。……我们的创造能力从未真正停滞不前，尤其是在品牌方面。”
