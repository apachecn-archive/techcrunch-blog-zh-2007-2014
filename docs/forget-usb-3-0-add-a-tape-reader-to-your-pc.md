# 忘记 USB 3.0:给你的电脑增加一个磁带阅读器

如果你正在市场上寻找一个稳定的存储介质来存储你所有的重要文件，你可能会很幸运。TR-01 纸带阅读器允许您将旧的(和新的)纸带直接读入您的电脑。系统可以像你将磁带拉过阅读器一样快或慢地阅读磁带。上面的视频展示了一些你可以用你的带子玩的可爱把戏，包括二进制计数，走 led 灯，为你和你的伙伴们创造一个可爱的灯光表演。

该套件在 2008 年花费了 180 美元——项目页面在这里——看起来开发者 Mike Davis 仍然在向[出售一些硬核复古硬件](https://web.archive.org/web/20230203193818/http://www.ebay.com/sch/mikesdavis/m.html?_dmd=1&_ipg=50&_sop=12&_rdc=1)。我问他是否还在卖 TR-01，还没有得到回复，但我确信一旦 EA 决定发布时髦的纸带版*战地 3* ，他可以提高产量。

可悲的是，没有纸磁带写入器，所以你可能无法将你的家庭照片转换为 8 位，以便后代可以找到磁带，假装它们是纸屑，然后在洒满咖啡后扔掉它们。

更新——Mike 证实，如果有需求，他仍然可以生产这些产品:

> 是的，我还在卖。我没有列出它，因为我只有 1 个单位的零件，并把它卖给了已经等了一段时间的人。下周我会订购更多的光电设备。如果你不介意等，我可以列出 TR-01，如果你愿意等一个星期才能交货的话。构建和测试它们需要一些工作。我自己做，在我妻子的厨房桌子上。:-)
