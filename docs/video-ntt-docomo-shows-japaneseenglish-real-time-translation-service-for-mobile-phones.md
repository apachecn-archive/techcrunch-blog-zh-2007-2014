# 视频:NTT Docomo 展示手机日语/英语实时翻译服务

日本最大的移动运营商 [NTT Docomo](https://web.archive.org/web/20230203082819/https://techcrunch.com/tag/docomo) 为手机开发了实时日语< - >英语翻译服务，这在同类服务中尚属首次。它的工作方式是，你对着设备说一些话，然后等待听到用另一种语言翻译你刚才说的话。

正如你在下面嵌入的视频中所看到的，这项服务使用云来完成繁重的工作，不是很“实时”，但非常接近。NTT·多科莫说，这项服务既可以用于电话交流，也可以用于面对面交流。

该公司还声称，日语的语音识别成功率约为 90%，英语的成功率约为 80%(其他语言将很快加入)。这大大超过了 5 月份的 15-20%，当时我们[首先推出了](https://web.archive.org/web/20230203082819/https://techcrunch.com/2011/05/30/video-ntt-docomos-mobile-simultaneous-translation-system/)这项服务。

日本 400 名终端消费者的测试从今天开始，将持续到明年 3 月。如果一切按计划进行，NTT Docomo 计划在 2012 年下半年向其 5600 万用户提供这项服务。
