# Groupon 与腾讯 Spars 合资企业没有激发当地的信心

据中国媒体报道，腾讯和 Groupon 的中国团购合资企业[![](img/82997e2397eccae786988d025f387ce0.png "groupon-china-logo")](https://web.archive.org/web/20230204185627/https://techcrunch.com/wp-content/uploads/2011/02/groupon-china-logo.jpg)遇到了麻烦。显然网站上线一天，腾讯犹豫了一下，又把它拉了下来。除此之外，人们告诉我们，运营完全是一片混乱:快速招聘，很少尽职调查，资金被随意挥霍。最大的不满是:它几乎完全由外国人经营。

根据我们几周前刊登的招聘广告，这听起来正是你所期望的。它吹嘘自己拥有近乎无限的资金，目标是在 3 月份前雇佣 1000 人，并强调咨询顾问和 MBAs 而不是精通本地市场的人。广告中甚至没有提到中文技能。

很容易说这只是另一个拙劣的合资企业和另一个傲慢的美国互联网公司大摇大摆地进入中国，认为它的品牌和大量现金将弥合文化差距。鉴于美国公司进入中国的历史记录，实际上很难说什么是[什么是*而是*什么是](https://web.archive.org/web/20230204185627/http://twitter.com/#!/niubi/status/38027207737610240)。我希望 Groupon 意识到，比起腾讯需要 Groupon 建立一个大型团购网站，它更需要腾讯来导航中国。腾讯已经通过 QQ 团购产品和对 Yelp 的克隆产品 Aibang.com 的投资两面下注。

如果负责运营的德国 Groupon 部门不明白自己有多需要腾讯的合作和当地人的支持，芝加哥办事处就需要介入 ASAFP。中国的市场变化很快，一旦你错过了第二次机会，就没有第二次机会了。这些都是几乎无止境的现金无法解决的问题。
