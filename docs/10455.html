Citrix收购“企业Dropbox”share file

Citrix Systems 已经收购了 ShareFile ，这是一家云存储、文件共享和协作解决方案提供商，这两家公司今天早上宣布。

ShareFile使企业能够跨多种设备安全地存储、同步和共享公司内外的业务文档和文件。

收购的财务条款没有披露。

ShareFile首席执行官Jesse Lipson将成为Citrix新成立的数据共享产品组的副总裁兼总经理，该产品组将负责ShareFile产品线。

此举符合思杰所谓的“个人云”的愿景，即每个人独有的文件、应用程序和个人数据的集合，可以从任何位置和设备访问。

ShareFile与Dropbox和Box.net等公司竞争，但它更专注于向各种规模的组织中的IT部门销售其产品，而不是消费者和小企业。其他替代方案包括 Egnyte 、 Nomadesk 、 Mozy 和 Syncplicity 。
