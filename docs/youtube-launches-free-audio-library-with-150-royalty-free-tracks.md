# YouTube 推出 150 首免版税的免费音频库

YouTube 目前在其网站上提供了超过 150，000 首音轨，视频制作者可以将其用作视频的背景音乐。然而，这些曲目不能下载或混音，这使得很难以创造性的方式使用它们。然而，对于那些想对背景音乐做更多事情的用户来说，YouTube 今天正在[扩展这个库](https://web.archive.org/web/20230316050130/http://youtubecreator.blogspot.com/2013/09/free-music-for-your-youtube-videos.html)，精选 150 首新的版税曲目。这个新的 YouTube 音频库中的音乐可以下载、混音并永久免费使用。

YouTube 表示，这些歌曲以 320 Kbps MP3 文件的形式提供，可以用于“任何创作目的”——甚至在 YouTube 之外。该团队表示，他们“广泛”寻找音乐家和制作人与这个项目合作。这也是[为更多想把他们的音乐加入收藏的音乐人征集](https://web.archive.org/web/20230316050130/https://docs.google.com/forms/d/1kjyiA-bQUMCpGraLI1-z2ey941pmsyYMAkoqUzVEKTA/viewform)意见。

总的来说，音乐的质量比 YouTube 上其他 150，000 首歌曲的质量都要高。从环境音乐、古典音乐到流行音乐和 R&B 音乐，各种类型的音乐相当丰富，歌曲也按情绪分类(愤怒、阴郁、快乐、浪漫等等)。).

拥有 150 首歌曲，这显然仍然是一个非常小的图书馆，但它确实给了用户更多的选择，让他们在视频中使用高质量的音乐，而不会遇到长期困扰这项服务的版权问题。
