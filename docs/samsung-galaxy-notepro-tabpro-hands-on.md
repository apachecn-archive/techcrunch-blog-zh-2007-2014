# 体验三星的新款 Galaxy NotePRO 和 TabPRO Android 平板电脑

三星在今年的 CES 上推出了几款[品牌](https://web.archive.org/web/20221209040024/http://www.samsungmobilepress.com/2014/01/07/Samsung-Galaxy-NotePRO-and-TabPRO-series-set-a-New-Rule-for-the-Tablet-Experience-at-CES-2014-2) [新安卓平板电脑](https://web.archive.org/web/20221209040024/http://www.samsungmobilepress.com/2014/01/07/Samsung-Galaxy-NotePRO-and-TabPRO-series-set-a-New-Rule-for-the-Tablet-Experience-at-CES-2014-2)，这两款都被定位为其现有 Note 和 Tab 系列设备的“Pro”版本。平板电脑确实提供了一些更针对企业用户的功能，但它们的用户界面也与普通的 Android 体验有很大不同。

Galaxy NotePRO 是一款 12.2 英寸的平板电脑，配有三星的 S Pen 和 2560×1600 的屏幕分辨率，TabPRO 有 12.2、10.1 和 8.4 英寸的尺寸，还在高端型号和两种较小的屏幕设备上提供了 2560×1600 的屏幕，考虑到你正在将更多像素塞进一个较小的空间，这是一个惊人的发现。各运行安卓 4.4。NotePRO 和 TabPRO 型号由用于 Wi-Fi 和 3G 版本的 Exynos 5 Octa 处理器和用于 LTE 版本的 Snapdragon 800 2.3GHz 处理器提供支持。

但这些新平板电脑真正令人印象深刻的地方并不在规格表上；取而代之的是新杂志《UX 》,它重新想象了 Android 平板电脑的基本主屏幕，其设计与 Windows Phone 甚至 Flipboard 等个别应用程序更为相似。还有一个多窗口模式，允许用户在同一个屏幕上播放多达四个不同窗口的独立活动内容。在实践中，它给我带来的体验与使用任何以前的 Android 平板电脑都不一样，虽然我没有花足够的时间来做出任何最终判断，但我确实很享受我用这些小工具获得的粗略体验。

三星将于 2014 年的某个时候在 Q1 把 NotePRO 和 TabPRO 设备推向市场，目前还没有关于定价的官方消息，尽管 [SamMobile](https://web.archive.org/web/20221209040024/http://www.phonearena.com/news/Samsung-Galaxy-TabPRO-and-NotePRO-prices-revealed_id51038) 已经把它们的价格定在 389 美元到略低于 900 美元之间，这取决于具体的装载情况。这是一种奇怪的设备，可能会挑战微软的 Surface 和 Windows 8 平板电脑产品，所以看看它们一旦发布会产生什么样的影响将是很有趣的。
