# 谷歌:Chrome 操作系统仍将在今年推出(可能只是测试版)

谷歌的 Chrome 操作系统被推迟，将错过“今年”的发布日期，现在有很多人在[欢呼](https://web.archive.org/web/20230202214006/http://www.techmeme.com/101122/p47#a101122p47)。这很大程度上是基于谷歌首席执行官 Eric Schmidt 上周在 Web 2.0 峰会上发表的[评论](https://web.archive.org/web/20230202214006/http://www.crunchgear.com/2010/11/15/schmidt-gingerbread-sooner-chrome-os-later/)，他说 Chrome OS 将在“未来几个月”的某个时候上市。所以我今天问谷歌，他们是否仍然坚持“今年晚些时候”推出 Chrome OS——我得到的答案是？一个热情的“是！”

但为了以防万一，我决定跟进，并问这是否意味着一个真正的航运产品即将到来或一些操作系统的测试版本？那里的答案要模糊得多。“*我们现在不讨论细节*，这是我被告知的。

查看 Chromium 操作系统论坛中的代码问题，看起来在操作系统发布之前，工作仍在进行中，以消除大量后期错误。这些 bug 中有许多是与 UI 相关的，但是也有一些看起来更严重。也就是说，有一些迹象表明，该操作系统的“测试版”可能即将推出。这里可以看到[，只有 6 个标记为“ReleaseBlock-Beta”的 bug。几乎所有这些都与从 Chrome OS 驱动的上网本购买 3G 计划的用户界面有关。还有一个“release block-Nominate”](https://web.archive.org/web/20230202214006/http://code.google.com/p/chromium-os/issues/list?q=label:ReleaseBlock-Beta)[列表](https://web.archive.org/web/20230202214006/http://code.google.com/p/chromium-os/issues/list?q=label:ReleaseBlock-Nominate)，里面有 38 个 bug。

还有其他迹象表明，谷歌正在删除某些包含“停止播放错误”的功能，以便推出测试版。

所以，如果要我猜的话，我打赌我们会在 12 月份看到某种形式的 Chrome OS 测试版。但这将让许多人失望，因为我们最初被告知 Chrome book(Chrome OS 驱动的上网本)将在假期前上市。除非一些供应商愿意发布一个非常 beta 的产品，否则这是不可能的。

但也许还有希望。微软的玛丽-乔·福利[说她最近和谷歌谈论了操作系统:](https://web.archive.org/web/20230202214006/http://www.zdnet.com/blog/microsoft/so-whatever-happened-to-googles-chrome-os/8046)

> 最近，我有机会向谷歌询问 Chrome OS，并被告知谷歌 OS 的预览版仍将于今年推出，并将在几款新机型上以测试形式提供。

当然，她也注意到“*谷歌，像微软一样，在 2010 年假日期间不会有一个可行的 iPad 竞争对手。*“但谷歌已经与 Chrome OS 是为平板电脑设计的说法拉开了距离。在同一个 Web 2.0 峰会上，施密特发表了上述评论，他还说 Chrome OS 是为键盘设计的，而 Android 是为触摸设计的。

也就是说，不可否认的是，ChromeBooks 和 iPads 很有可能最终在市场上针锋相对，因为两者的价格可能差不多。尽管施密特发表了评论，谷歌也一直在考虑在平板电脑领域使用 Chrome 操作系统。
