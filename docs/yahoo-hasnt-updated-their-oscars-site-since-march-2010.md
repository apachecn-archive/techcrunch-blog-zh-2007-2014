# 自 2010 年 3 月以来，雅虎一直没有更新他们的奥斯卡网站

似乎有很多雅虎的财产根本没人关注。例如，我们[注意到](https://web.archive.org/web/20230202231615/https://techcrunch.com/2010/11/01/halloweens-over-yahoo-so-turn-the-lights-back-on/)雅虎在万圣节过后仍保留他们的万圣节内容。有一个[美味的混乱](https://web.archive.org/web/20230202231615/https://techcrunch.com/2010/12/29/delicious-in-purgatory/)，一篇声称该网站不会下线的博客文章被下线，因为该网站已经被关闭。现在，我们注意到，[雅虎的奥斯卡颁奖网站](https://web.archive.org/web/20230202231615/http://oscars.movies.yahoo.com/)自 2010 年 3 月以来就没有更新过。

尽管今年早些时候雅虎有一个新闻发布会:“雅虎今年加倍了它的颁奖节目内容，”这篇文章说。不仅仅是内容被推送到雅虎的 OMG 网站。今年的奥斯卡金像奖也没什么内容。

没什么大不了的。不是世界末日。但这是一个迹象，表明有雅虎财产没有工作人员分配给他们。或者更糟的是，有员工，但他们根本不关心自己的工作。克莱斯勒，雅虎本赛季颁奖典礼的独家广告合作伙伴，一定想知道发生了什么。

好消息是，在这篇文章之后，雅虎会很快更新他们的奥斯卡网站。好吧，明天，周末结束后。我是说星期二，假期后。当然是明年之前的某个时候。
