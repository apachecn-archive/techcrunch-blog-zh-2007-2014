# Bojam 推出基于网络的协作声音工作室

![](img/68b3157a894135a7b3e1bdbd3496fd57.png)

去年的 [TechCrunch50 家创业公司之一，](https://web.archive.org/web/20221007133403/http://www.beta.techcrunch.com/2008/09/10/tc50-bojam-asks-why-cant-we-all-just-rock-together/)Bojam 今天发布公测，我们收到了 500 份邀请。Bojam 是一个基于网络的声音工作室，让音乐家们练习演奏音乐，寻找世界各地的其他音乐家一起演奏，并为同一首歌创作曲目。点击[此处](https://web.archive.org/web/20221007133403/http://bojam.com/tcsignup/)试用该服务。

Bojam 是一款功能齐全的音乐混音器。您可以调整每个音轨的音量，或者添加失真和混响等效果。同样，您可以独奏或静音乐器，循环，剪切，复制，粘贴和修剪音频剪辑。Bojam 还包括一个插件，允许你直接录制并上传到混音中，或者你可以上传 Garage Band 或其他软件中录制的曲目。

Bojam 的 mixer 即将推出的一个有趣功能是能够在博客、社交网络和网站中嵌入混合小部件，用户可以在可嵌入的小部件中重新混合和编辑音频。这个小部件应该会在接下来的几个月内推出

除了作为一个协作工具，Bojam 也是一个社区，让音乐人找到世界上任何地方的其他音乐人一起异步演奏。如果一首歌真的很好，音乐人可以决定在 iTunes 或 Rhapsody 上以 99 美分的价格出售。音乐教师也可以利用这项服务拍卖他们的服务(就像世界另一端的音乐工作室一样)。

Bojam 面临来自 [Indaba](https://web.archive.org/web/20221007133403/http://www.indabamusic.com/) (此处覆盖)[jam glue、](https://web.archive.org/web/20221007133403/http://www.jamglue.com/)和[最小噪音](https://web.archive.org/web/20221007133403/http://www.minimumnoise.com/Default.aspx)(此处覆盖)的竞争。
