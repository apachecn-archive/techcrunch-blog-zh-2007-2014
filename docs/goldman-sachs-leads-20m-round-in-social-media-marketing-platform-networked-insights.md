# 高盛在社交媒体营销平台 Networked Insights 领投 2000 万美元

一个营销决策平台的开发者已经筹集了 2000 万美元的新资金，由高盛资产管理公司牵头。这使得这家初创公司的总资金接近[3000 万美元。](https://web.archive.org/web/20230203172905/http://www.crunchbase.com/company/networked-insights)

Networked Insights 的 SocialSense 平台成立于 2006 年，它分析社交数据，帮助品牌制定营销、广告、研究和产品开发的战略和支出。Networked Insights 表示，它能够每月从 7.5 亿人和数十亿次在线对话中挖掘数据，这使营销人员能够利用见解并简化未来广告活动的研究。Networked Insights 利用这些海量数据来计算品牌应该在哪些方面投入媒体资金，无论是电视、手机还是社交媒体等。

据数据显示，该公司已经帮助三星、MillerCoors、MTV Networks、卡夫和 Starcom MediaVest 等公司形成了 50 亿美元的媒体支出。

该公司表示，他们计划使用新的资金投资开发机器学习技术、情感识别技术等。网络洞察也将推动其 IT 架构。
