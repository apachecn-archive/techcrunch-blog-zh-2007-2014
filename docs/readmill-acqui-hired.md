# 来源:Dropbox Acqui——以 800 万美元租用社交阅读应用 Readmill

随着新资本的到来，Dropbox 似乎掀起了一场收购狂潮，尽管是静悄悄的。在云存储公司[最近收购了工作场所聊天解决方案 Zulip](https://web.archive.org/web/20221007031203/https://beta.techcrunch.com/2014/03/17/dropbox-acquires-zulip-a-stealthy-workplace-chat-solution-still-in-private-beta/)之后，TechCrunch 从消息来源获悉，它正在与总部位于柏林[的社交和共享阅读平台 Readmill](https://web.archive.org/web/20221007031203/http://readmill.com/) 进行谈判。

据我们所知，该交易价值约为 800 万美元，主要由股票组成，但也有少量预付现金给将搬到旧金山 Dropbox 的初创公司创始人。Readmill 将会发生什么还不清楚——我昨天晚些时候联系了联合创始人兼首席执行官亨里克·博格伦，请他发表评论，但尚未得到回复，因此所有这些都未经证实。然而，考虑到这是一项人才收购，该服务很可能会被关闭，或者在短期内以最低限度的支持继续运营。这次收购也表明了柏林技术领域的整合趋势和大量退出。

Readmill 于 2010 年末由亨里克·博格伦和大卫·克尔克鲁德创立，作为电子书的社交层推出，可以方便地突出显示可以与朋友和同龄人分享的摘录，将书变成一个“社交对象”。它为 Android 和 iPhone 提供了一个非常优雅的阅读应用程序，以及 iPad 的平板电脑版本。该应用还采用了类似 Twitter 的关注模式，用户可以关注书籍、其他用户，最重要的是，作者。

该公司[于 2011 年 9 月在 TechCrunch Disrupt SF 首次亮相](https://web.archive.org/web/20221007031203/https://beta.techcrunch.com/2011/09/14/readmills-disrupt-encounter-makes-them-launch-early-2/)，与 YC 组合公司的保罗·格拉厄姆一起在台上办公。当时公司还没有推出，格雷厄姆反复告诉他们应该现在就推出。“为什么要等？”他说。Readmill 随后开始提供有限数量的邀请，同时宣布从伦敦的 Passion Capital 和 Index Ventures 获得 28 万€的种子资金。

2012 年，该公司[筹集了](https://web.archive.org/web/20221007031203/http://venturevillage.eu/readmill-wellington-partners)一轮未披露的 A 轮融资，由 Wellington Partners 牵头，现有投资者 Index Ventures 和 Passion Capital 参与。我还了解到，Atlantic Ventures(Christophe Maire 的天使投资工具)、Peter Read 以及 Souncloud 创始人 Alexander Ljung 和 Eric Wahlforss 也是投资者。
