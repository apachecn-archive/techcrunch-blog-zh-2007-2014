# Foursquare 与谷歌合作，通过谷歌 I/O 上的海报进行 NFC 签到

谷歌的年度 I/O 大会将于明天召开，届时可能会发布一系列重大消息。但是，在这次活动中展示的一个更酷的东西实际上是 Foursquare 今天提前宣布的[:基于 NFC 的活动签到。](https://web.archive.org/web/20230203044416/http://blog.foursquare.com/2011/05/09/experimenting-with-nfc-check-ins-for-google-io/)

谷歌和 Foursquare 正联手在未来几天的会议上实现这一目标。在会议中心周围，你会看到 Foursquare 的海报，里面实际上会嵌入 NFC 芯片。这意味着你只需将手机靠近这些海报，就可以通过 Foursquare 签到。非常酷。

当你这样做的时候，你自然会得到一个 Google I/O Foursquare 徽章。

当然，有一个很大的警告:你需要一个支持 NFC 的手机才能工作。Foursquare 指出，许多较新的 Android 设备都有 NFC 芯片，但这些设备仍未得到广泛使用——最大的设备是 Nexus S。但如果有一个地方会有一大群人使用新的 Android 设备，那将是 I/O。那些没有 NFC 手机的人仍然可以定期签到或扫描这些海报上的二维码。

Foursquare 指出，他们已经在自己的办公室测试了一段时间的 NFC 签到。这里有一条[推文](https://web.archive.org/web/20230203044416/https://twitter.com/#!/noah_weiss/status/42259702075232256)从二月份就证明了这一点。

这种合作关系很有趣，原因有几个。首先，谷歌有他们自己的基于位置的服务，Latitude，它今年终于开始提供签到服务。没有说你是否能在 I/O 以类似的方式登录到 Latitude。第二，正如一些人可能记得的那样，谷歌实际上收购了 Foursquare 的前身 Dodgeball，然后对它不做任何事情，这导致了一些敌意和[创始人离开谷歌](https://web.archive.org/web/20230203044416/https://techcrunch.com/2010/08/05/eric-schmidt-google-dodgeball-foursquare/)(这两者现在都回到了 Foursquare)。我猜伤口已经愈合了。

脸书去年在他们的 f8 会议上做了类似的事情，展示了 Presence(通过 RFID 进行的[签到)。但那也是在你参加会议时收到的一个特殊芯片，而不是通过你的手机。](https://web.archive.org/web/20230203044416/https://techcrunch.com/2010/04/21/facebook-presence/)

虽然这显然只是一项技术测试，但 NFC 签到的风险很高。这可能是这些服务的未来，因为 NFC 将使主流用户在场馆签到变得更加容易和方便。

我们将在谷歌 I/O 直播整个星期。
