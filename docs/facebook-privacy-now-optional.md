# 脸书:隐私现在是可选的

[![](img/2be55cab0f399880bf4b2357a6938321.png)](https://web.archive.org/web/20230313155207/http://www.facebook.com/) 脸书和 MySpace 之间的一大区别在于两家公司对待用户数据隐私的方式。在 MySpace，你的个人资料默认是公开的，用户可以选择将其设为私有(18 岁以下用户除外，他们默认为私有)。在脸书，您的个人资料信息一直是保密的，会与您的朋友共享，也可以与您的社交网络(您的学校、公司或地理位置)共享。

不过，从今天开始，用户可以改变他们的隐私设置，公开几乎所有的个人资料信息。对于像我这样不介意与世界分享所有数据的用户来说，这是个好消息。这也使得[我的简介](https://web.archive.org/web/20230313155207/http://www.facebook.com/home.php#/profile.php?id=500065899)更有趣，因为每个人现在都可以查看它。

一些用户在抱怨，虽然我不知道为什么，因为隐私设置是可选的。在一篇宣布这一变化的博客文章中，大多数评论说他们希望回到过去的脸书，不管这意味着什么(脸书在过去几年里发生了如此大的变化，真的不再有定义的“旧脸书”了)。和往常一样，一个星期后大家都会安定下来，找到新的东西去抱怨。
