# 专利和联盟:当好的意图变得可怕时

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/07/patents-and-unions-when-good-intentions-go-horribly-wrong/>

看我用一句话毁了你的周日下午。准备好了吗？开始了。

**专利。**

本周在 [NBC 新闻上:在这里](https://web.archive.org/web/20230203235417/http://pressheretv.com/?p=1840)我们谈论了大多数读者认为是漫长、缓慢、痛苦的根管治疗的话题。或者更糟。NPR 的劳拉·西德尔也参加了这个节目，如果你还没有听过她关于美国生活的一集，叫做[“当专利来袭”，](https://web.archive.org/web/20230203235417/http://www.thisamericanlife.org/radio-archives/episode/441/when-patents-attack)，现在就去听吧。我们会等的。

纳森·梅尔沃德——有人称他为科技界的伏地魔——对我来说有点神秘。Sydell 说，她不打算在[intelligent Ventures](https://web.archive.org/web/20230203235417/http://www.intellectualventures.com/Home.aspx)之后让这篇文章变得如此艰难，正如你在播客中听到的，他们认为无伤大雅的问题被当作“交叉询问”

我的直觉是，智力投资并不是为了让巨魔得逞。正如梅尔沃德所说，它的目的是给那些不想创业的发明家提供一种将他们的发明货币化的方法，并在一个破碎的系统中成为一个有效的调解人。这也是最近上市的 RPX 公司的承诺，该公司得到了查尔斯河风险投资公司的支持。Armony 还投资了智力投资，我们给了他机会为“专利流氓”的指控辩护。

但是，即使你愿意假定智力投资公司是无辜的，NPR 的文章也清楚地暴露了一些相当令人不安的连锁反应，智力投资公司的团队并不否认这一点。电台节目的美妙之处在于，你可以听到记者们多次试图给 IV 一个解释自己的方法，但他们只是挖了一个更大的坑。

当被问到时，我找不到一个他们承诺的双赢的例子:他们从一个聪明的修补匠那里买了一项发明，然后卖给一家公司，这样他们就可以为他把它推向市场。他们给出的例子让 NPR 陷入了一个兔子洞，这让他们在东德克萨斯的一个满是空壳公司办公室的走廊里着陆，这些空壳公司存在的全部理由是购买专利和起诉实际建造东西的公司，将这些收益的一部分用于智力投资。

这种美国生活以将专利形势比作军备竞赛而告终，引用了大型科技公司最近的专利组合之争。虽然这是一个恰当的比喻，但我不禁想到了另一个:工会。

我意识到我写这些是在邀请组织良好的仇恨邮件。像专利一样，我认为工会的意图非常好，而且一度对美国的发展至关重要。但让我们面对现实:管理不善的工会也是我们今天许多资本主义问题的根源。很难看出航空业、汽车业、公共教育发生了什么，也很难感觉到工会在企业界是一股完全良性的力量。

就像专利一样——也是为了保护小人物——工会的成立有着充分的理由。但就像专利一样，它们可能会扭曲成伤害创新、竞争，最终伤害消费者和整个国家的东西。工会本质上创造了一种“我们对抗他们”的动态，这种动态将战胜公司管理层作为首要目标，而不是服务客户、创新，或者在教育方面，教导孩子。

我有这种感觉并不奇怪，因为我职业生涯的大部分时间都在报道硅谷的企业，那里缺乏工会就像缺乏救助一样引人注目。最近，我读了很多关于硅谷历史的书，对工会的缺失在美国科技行业发展中的核心地位感到震惊。它实现了持续的创造性破坏，使硅谷在快速变化的世界中保持相关和繁荣。

飞兆半导体和英特尔联合创始人罗伯特·诺伊斯将中西部的价值观带到了硅谷，这是出了名的，但据报道，他讨厌工会，根据汤姆·沃尔夫 1983 年对诺伊斯的《时尚先生》[简介](https://web.archive.org/web/20230203235417/http://www.stanford.edu/class/e140/e140a/content/noyce.html)，他将工会视为“对英特尔和整个半导体行业的死亡威胁”。建立一个巨大的新的制造业，使得工人*不想*加入工会，这可能是诺伊斯&公司最大的遗产。这是他们在美国做生意的方式和 20 世纪后半叶做生意的方式之间划出的许多重要界线之一。沃尔夫写道:

> “劳资纠纷是东方古老地形的一部分。如果英特尔被分成工人和老板，这意味着每一方都必须从另一方身上榨取金钱，那么这个企业就完蛋了。动机将不再是内在的；它将以致命的工作规则和申诉程序的形式被具体化。唯一一次投票时，工会以四比一的大比分输了。英特尔的员工同意诺伊斯的观点。工会是过去死亡之手的一部分……诺伊斯和英特尔走在通往黄金国的路上。

这种心态是现在常见的股票期权的核心，我们努力工作并一起赢得创业公司的文化，这种文化已经在企业界蔓延开来。我们现在不认为这是“反工会”，因为我们不再在硅谷生产任何东西。但在当时，这是一个明显的岔路口，它极大地改变了硅谷，使它能够随着周围世界的变化而发展、变化并保持相关性，而不像底特律。

不幸的是，当围绕软件专利的善意但明显功能失调的规则被编纂成法律时，没有人有同样的能力或毅力来做出类似的立场——这是专利局反对的，也是许多行业希望从未发生的事情。当然，现在有一场改革专利制度的运动，但正如我们在过去几周所看到的，我们的政府现在并没有完全发挥最佳作用。我很难乐观，一切都会改变。

NPR 和西德尔打破了这个破碎系统的现实世界的影响，克里斯萨卡是唯一一个愿意公开谈论这个问题的投资者，这仍然值得称赞。对话越是能走出沙山路的怪圈，解决问题的机会就越大。也许吧。
