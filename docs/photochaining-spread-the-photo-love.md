# 摄影:传播摄影之爱

[![vacances2008011](img/839b650357eba6a71abfefe0ab7f7cf9.png "vacances2008011")](https://web.archive.org/web/20221007023249/https://beta.techcrunch.com/wp-content/uploads/2009/03/vacances2008011.jpg)

摄影是风靡全国和其他国家的最新时尚，而且非常简单有趣。你买一个便宜的存储卡，把它放在你的相机里，用它拍一些照片，然后给卡“命名”，并留下一张纸条，告诉别人拍一张照片，然后把它放在别的地方。结果随后被张贴到[Photochaining.com](https://web.archive.org/web/20221007023249/http://www.photochaining.com/)，世界和平随之而来。
 当处理正确时，产生的随机图片相当引人入胜，是[秘密](https://web.archive.org/web/20221007023249/http://postsecret.blogspot.com/)和地理藏宝的结合。很酷的东西。
