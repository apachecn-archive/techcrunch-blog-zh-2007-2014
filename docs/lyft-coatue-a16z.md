# 按需乘车共享初创公司 Lyft 正在筹集新一轮资金

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/15/lyft-coatue-a16z/>

消息人士称，Lyft 正在筹集新一轮资金。该公司正在寻求使拼车在美国和世界各地的城市成为主流，预计将利用新的现金为扩张到新的城市和地区提供资金。

我们听说 Lyft 已经投资了许多风险公司和后期机构投资者，但对冲基金 [Coatue Management](https://web.archive.org/web/20230328091051/http://www.crunchbase.com/financial-organization/coatue-management) 似乎在这笔交易中处于领先地位。Coatue 最近投资了热门公司，如 Box、 [Snapchat](https://web.archive.org/web/20230328091051/https://techcrunch.com/2013/12/11/snapchat-series-c-50-million/) 和[hotel night](https://web.archive.org/web/20230328091051/https://techcrunch.com/2013/09/05/hoteltonight-series-d/)，并且是[对冲基金押注具有吸引力的后期创业公司](https://web.archive.org/web/20230328091051/https://techcrunch.com/2014/01/18/the-rise-of-the-hedge-fund-startup-investor-again/)的增长趋势的一部分。

Andreessen Horowitz 领导了 Lyft 6000 万美元的 C 轮融资，预计也将贡献一大笔现金。Lyft 的其他投资者包括 Founders Fund、Floodgate、Mayfield Fund、K9 Ventures、Ooga Labs、fbFund 和 Keith Rabois。

该公司成立于 2007 年，名为 Zimride，但直到 2012 年，当它从远程转向按需拼车时，事情才开始真正起飞。

随着 Lyft 移动应用[的推出，它开辟了一个新的领域，使乘客能够从其他有车和空闲时间的人那里搭车。由于按需平台的成功，该公司更名为 Lyft，并于去年夏天将其遗留的 Zimride 资产出售给 Enterprise Holdings。](https://web.archive.org/web/20230328091051/https://techcrunch.com/2012/05/22/zimrides-lyft-is-going-to-give-uber-some-lower-priced-competition/)

在旧金山的最初阶段，Lyft 去年年初开始向其他城市扩张，现在在美国的 20 个市场提供服务。但像优步一样，Lyft 也通过移动应用程序提供按需乘车服务，它计划从今年开始大幅增加其运营的国际城市的数量。

随着它扩大旧金山总部和世界各地远程办公室的业务，额外的资金对于让它走上实现这一目标的正确轨道至关重要。

与此同时，Lyft 正试图让更多监管机构和地方官员接受让无证司机在镇上载乘客的想法。为此，它最近聘请了 Google X 法律总监大卫·埃斯特拉达担任其政府关系副总裁。

该公司还宣布成立一个[点对点拼车保险联盟](https://web.archive.org/web/20230328091051/https://techcrunch.com/2014/02/05/lyft-insurance/)，其中包括其他运输公司，以及监管机构和保险提供商，以解决其司机保险的棘手问题。

Lyft、Coatue 和 Andreessen Horowitz 均拒绝置评。
