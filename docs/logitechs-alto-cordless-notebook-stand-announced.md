# 罗技宣布推出 Alto 无绳笔记本支架

![](img/9f6ec274cd950b57b419533d821739ca.png)
罗技今天发布了他们 Alto 笔记本支架系列的最新产品。你知道，这款是无线的，还配有一个键盘和三个 USB 端口，可以满足你所有的打印机/显示器/iPod 需求。如果你想让你的手腕从笔记本电脑的纤薄键盘上放松一下，100 美元并不吝啬。
