# Mozilla Festival 向更多爆米花和更少开发者致敬 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/07/mozilla-festival-salutes-more-popcorn-and-less-developer-ghetto/>

![](img/a58a1ab0fb39038017abcc6ae11b8251.png)

上周末在伦敦举行的 [Mozilla Festival](https://web.archive.org/web/20230205011450/https://mozillafestival.org/) 的 600 名参与者是一群电影制作人、教育工作者、编码员、精通技术的媒体专业人士、对媒体持怀疑态度的黑客、不懂黑客的记者、游戏玩家、政府顾问——我可以继续说下去，但我想你已经明白了。它是多样化的。然而，他们都是准备探索开放网络前沿的思想家和创造者。

在“媒体、自由和网络”的口号下，非营利组织 Mozilla 围绕利用网络实现更具创造性和协作性的媒体景观的挑战，设计了这一轮年度节日。大量的想法被传播，挑战被克服，解决方案被众包，相当一部分令人敬畏的新东西被呈现，TechCrunch 为你挑选了一些亮点。

**网络是纪录片**

艾美奖获奖电影制作人凯特·西泽克在[# mozzfest](https://web.archive.org/web/20230205011450/https://twitter.com/#!/search/%23mozfest)上展示世界上第一部 HTML5 电影时形容自己“超级开心”。这是“第一百万座塔”的全球首映，这是一部像网络一样工作的纪录片:不断变化，每分钟更新。

这部电影是在 Mozilla 最近推出的 JavaScript 库 [Popcorn.js](https://web.archive.org/web/20230205011450/http://popcornjs.org/) 的帮助下开发的，该库类似于超文本原理，将在线视频与几乎任何在线内容无缝融合——最明显的例子是维基百科、Flickr 或 Twitter。它将电影变成一种更具语义的体验，允许更多的动态和互动。

“我们不能只在最后才考虑互动，而是从一开始就需要它来创造电影制作人和观众之间的对话，”参与爆米花开发的学生们表示同意。“这种讲故事才刚刚起步。”

**走出极客区**

《连线》英国版的自由编辑本·哈默斯利称之为“极客区”,我认为他一针见血。他的意思是，只有一小部分人知道如何编码，这种技能应该融入到各种职业中。

我们这些精通技术的人有责任确保下一代会发现编码是一种更融入他们日常生活的技能。这并不意味着教孩子们如何构建下一个 Firefox，而是要确保他们知道自己可以做到。这个想法是教人们能够分享想法的语言。“‘在这里构筑你的梦想’可以成为网络的口号，”Mozilla 的执行董事[马克·苏尔曼](https://web.archive.org/web/20230205011450/https://twitter.com/#!/msurman)说。本·哈默斯利一马当先，称之为“拯救世界”。

是的，这听起来有点理想化。然而，编码意味着能够创造，而创造意味着力量。如果他们不是在拯救世界，那么每天出现在这个博客上的优秀初创企业至少正在一点一点地改变这个世界。对于越来越多的变化，应该以各种可能的方式支持像[年轻重连线状态](https://web.archive.org/web/20230205011450/http://youngrewiredstate.org/)、[黑客龙](https://web.archive.org/web/20230205011450/http://hackasaurus.org/en-US/)或[梦工厂](https://web.archive.org/web/20230205011450/http://dreamyard.com/)这样的网络。

**“我们像信任政治家一样信任记者……**

…我们没有。“不是我的话；那些愤怒的开发者。或多或少懂技术的记者和或多或少懂媒体的黑客在 Mozilla Festival 上讨论罕见的“黑客记者”。

这一概念遵循的理念是，通过将这两种行业结合起来，让新闻变得更透明、更容易获取。这可以从教导记者更好地呈现数据驱动的新闻，一直到向媒体介绍某种形式的迭代。

作为一名科技记者和超级书呆子，我在这场讨论中扮演了一个非常轻松的角色。我学会了如何编码——不是很好，但足够有用了——因为它帮助我做得更好。毕竟，记者不应该是收集、分析和呈现数据的大师吗？一些基本的编程技能可以帮助你实现这一目标。

写数据新闻手册是一个开始。这本书的初稿是帮助记者尽可能好地处理大量数据，由 55 名撰稿人在节日的 48 小时内完成的。没有比 TechCity 更好的地方让黑客变得更像黑客了。
