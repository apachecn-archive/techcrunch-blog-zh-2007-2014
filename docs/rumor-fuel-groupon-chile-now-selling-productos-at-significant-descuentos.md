# 谣言火上浇油:Groupon 智利分公司现在以很高的价格出售产品

昨天，在我注意到 [Groupon](https://web.archive.org/web/20230205030703/http://www.crunchbase.com/company/groupon) 已经[注册了 groupongoods.info】、groupongoods.info 和其他相关域名后，我写了一篇帖子，推测这家快速增长的日常交易初创公司可能正在策划一个专门的网站(版块),用于打折销售家居用品、消费电子产品甚至杂货等有形产品。](https://web.archive.org/web/20230205030703/https://techcrunch.com/2011/08/21/does-groupon-hope-to-challenge-amazon-with-groupon-goods/)

我推测“Groupon Goods”将是该公司试图与电子商务巨头亚马逊竞争的一种方式，但当然，域名注册只是这样，并不一定意味着 Groupon 正在计划这样的事情。Groupon 发言人拒绝对我的猜测发表评论，这是可以理解的，但我很想知道他们如何解释目前在 Groupon 智利网站上直播的品牌*Nuevo*T6【Productos】部分的存在。

事实上，Groupon Chile 正在以零售价 40%到 50%的折扣销售煤油炉、厨房机器人甚至摩托车。

显然，这不是 Groupon 第一次将有形产品进行打折销售，但这绝对是我们第一次在任何 Groupon 网站上看到专门的“产品”(或“商品”)板块。

一年多前，Groupon 进入拉丁美洲，收购了智利的交易网站 ClanDescuento，并在巴西开设了一个名为 ClubeUrbano 的新网站。根据[最近的 comScore 研究](https://web.archive.org/web/20230205030703/http://www.comscore.com/dut/Press_Events/Press_Releases/2011/6/Groupon_Leads_Latin_America_as_Most-Visited_Coupon_Site_in_Region)，2011 年 3 月，巴西、阿根廷和智利展示了优惠券网站访问者的最高渗透率，这些市场超过 10%的互联网人口访问了该类别。

ComScore 还发现，Groupon 是这些地区最大的优惠券网站——超过了[Groupalia](https://web.archive.org/web/20230205030703/http://www.crunchbase.com/company/groupalia)——拥有大约 500 万的访问量。

在过去，Groupon 已经与巨头 Expedia 和 Livenation 建立了合作伙伴关系，分别提供旅行和活动门票的在线交易。

Groupon Goods 可能是他们的下一个实验品，尽管就 Groupon 智利网站而言，该公司似乎是直接与商家合作，而不是与其他公司合作。

我们已经要求 Groupon 进一步置评。

Groupon 最近宣布，2011 年第二季度其收入为 8.78 亿美元，净亏损 1.027 亿美元(今年上半年亏损约 2.05 亿美元)。该公司目前在全球拥有约 10，000 名员工。
