# 托普斯为公开的 Google+帖子推出实时搜索引擎

实时搜索和分析平台公司[托普斯实验室](https://web.archive.org/web/20230205020448/http://topsylabs.com/)今天早上[宣布](https://web.archive.org/web/20230205020448/http://www.marketwire.com/press-release/topsy-labs-releases-google-index-ranking-results-by-relevance-1571432.htm)已经将 [Google+](https://web.archive.org/web/20230205020448/http://www.crunchbase.com/product/google) 添加为其索引中的一个来源。

一个专用于公开 Google+帖子的搜索引擎，测试版可用，让用户搜索和访问社交网络中发生的实时和历史对话。

托普斯强调，帖子是根据初创公司专有的相关性和影响力排名技术进行排名的。你可以在下面找到一些截图，或者现在就试一试。

托普斯的算法计算谁是特定关键词、术语和链接的专家，并根据 Google+和 Twitter 帖子产生的关注度更新影响图。

托普斯实验室的联合创始人兼首席执行官维普·维德·普拉卡什说:

> “我们很自豪成为第一个对公开的 Google+帖子提供全面搜索的搜索引擎。除了我们目前正在索引的 2.2 亿条每日推文，随着 Google+的加入，我们已经扩展了我们的搜索技术，以深度索引和排名长篇社交对话。
> 
> 在托普斯，人们现在可以在 Google+和 Twitter 上搜索实时的相关排名结果，企业可以通过使用我们的 API 来挖掘 Twitter 和 Google+上海报的集体智慧，从而更好地了解什么是重要的。"

托普斯总部位于旧金山，成立于 2006 年，已经从 BlueRun Ventures、Ignition Partners、Founders Fund 和 Scott Banister 那里筹集了大约 3000 万美元。
