# 嘿脸书，搞什么鬼？远离技术专家

因此，当我们发现脸书正在以惊人的速度抢走谷歌的员工时，我认为这很有趣也很幽默。该公司明年的员工人数将增加一倍以上，达到 700 人。到目前为止，该公司至少雇佣了 10 名高管，每个月还会增加 2-4 名。

但不好笑的是，他们从其他创业公司和我更感兴趣的公司挖人。也就是我们。今天我发现他们已经雇佣了我们在 T4 的产品经理本·梅尔。本是我们今年夏天的明星实习生之一，实习结束后，我们让他继续全职工作。我们在 TechCrunch 的人不多，每个人都是关键。失去本非常非常难过。

但显然，脸书和他们的股票期权的引力足以吸引他离开 TechCrunch。祝贺本，我们会想念他，但他会在新的工作岗位上做得很好(我希望，能让我们进入脸书的管理系统:-)。

但是在这件事上，我对脸书一点也不满意。离我们的员工远点，脸书，去别处完成你的就业配额吧。任何人，我就宣战。
