# 人们现在正在玩地震直播 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/01/14/people-are-now-playing-quake-live/>

即将推出的免费浏览器版雷神之锤(Quake)正在向该服务发出测试邀请。用户被引导到[beta.quakelive.com](https://web.archive.org/web/20220930104827/http://beta.quakelive.com/)

这是一个游戏性的新实验。id Software 不再向用户收取月租费，而是与 IGA 全球合作，在游戏中加入广告和赞助来赚钱。游戏的早期[评论](https://web.archive.org/web/20220930104827/http://news.bigdownload.com/2009/01/05/hands-on-quake-live/)是一个大拇指，它看起来像是一列通往零工作效率的单程列车。我们正试图追踪测试环境的视频。

《雷神之锤》Live 的预告片在这里是。

> 雷神之锤现场测试者！
> 
> 你的名字已经提交给 id 软件，以帮助我们测试工作室正在进行的一个新的多人游戏项目。我们正在进行《雷神之锤》直播，在我们逐步测试技术系统、用户界面选项和游戏变化的过程中，我们将非常感谢您的帮助。
> 
> 什么是雷神之锤直播？
> 
> 当我们在 QuakeCon 2007 上宣布这个项目时，最初被称为“雷神之锤零”，雷神之锤直播将是免费提供的雷神之锤 III 竞技场的更新版本，你可以通过你的网络浏览器访问和玩。
> 
> 作为一名 beta 测试人员，我们请求您帮助我们测试游戏交付机制和核心游戏技术，并提供关于游戏玩法和游戏水平的反馈。服务器始终可用，但我们鼓励您加入我们的常规焦点测试和特殊活动(您将收到电子邮件通知)。
> 
> 请注意，QUAKE LIVE 是一项正在进行中的工作，您作为 beta 测试人员的体验并不反映最终将提供的功能集。我们鼓励你在地震现场论坛上发表评论、问题、建议和其他反馈。作为测试版测试人员，您需要对您的个人密钥、登录信息、帐户信息以及所有其他与您参与此计划相关的信息进行保密。
> 
> 下一步是什么？
> 
> 只需使用下面的个性化密钥来设置一个 id Beta 测试帐户，并访问 QUAKE LIVE 网站，您就可以开始玩游戏了。
> 
> 我们将要求您同意一份保密协议，并要求您对您的个人密钥、登录信息、帐户信息以及与您参与此计划相关的所有其他信息保密。
> 
> 访问 http://beta.quakelive.com/开始玩。

>
