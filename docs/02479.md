# 新的盒装套装以 2.99 美元的价格传输到你的 iPhone 上

![](img/f16edb38c7023531c93e155fc5b6ff05.png)

上世纪 90 年代的摇滚乐队美利坚合众国的总统们刚刚发布了一款 iPhone 应用程序( [iTunes link](https://web.archive.org/web/20221007020308/http://click.linksynergy.com/fs-bin/stat?id=SRj1kKq8JAY&offerid=146261&type=3&subid=0&tmpid=1826&RD_PARM1=http%253A%252F%252Fitunes.apple.com%252FWebObjects%252FMZStore.woa%252Fwa%252FviewSoftware%253Fid%253D304149125%2526mt%253D8%2526partnerId%253D30) )，它可能会成为盒装 CD 的替代品。这款 iPhone 应用程序可以访问四张专辑、一盘早期演示磁带和定期更新。你不用花 50 美元或更多的钱来购买一个乐队的全部音乐，只需花 2.99 美元就可以在你的 iPhone 上买到一套同等的音乐。问题是这些歌曲都是以随机顺序播放的(或者你可以选择不同的播放列表)。但这确实是乐队的营销手段。任何歌曲都可以直接从 iTunes 商店购买和下载。

当你真正想要的只是一两张专辑，或者一些单曲的时候，这是一个比浪费你的钱买一整套盒装唱片更好的模式。美国总统的主唱戴夫·戴德勒称这款应用程序为“惠特曼的耳朵采样器”他现在也是总部位于西雅图的 Melodeo 公司的高管，该公司发行这款应用程序，是其收集的 [NuTsie](https://web.archive.org/web/20221007020308/http://www.nutsie.com/main) 音乐流媒体应用程序的一部分。

然而，其他乐队可能会发现推出类似的应用程序更加困难。美国总统能够提供这个应用程序的一个原因是，乐队仍然拥有他们原始录音的所有权利。对于大多数乐队来说，情况并非如此，但也许某家唱片公司的一名有事业心的营销人员会为其他盒装流媒体设备开绿灯。
