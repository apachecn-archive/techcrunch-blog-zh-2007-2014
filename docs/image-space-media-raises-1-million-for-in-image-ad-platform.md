# 图像空间媒体为图像广告平台筹集了 100 万美元

根据 SEC 提交的文件[，TechCrunch50 demopit 公司](https://web.archive.org/web/20221006183705/http://www.sec.gov/Archives/edgar/data/1473932/000147393210000003/xslFormDX01/primary_doc.xml) [Image Space Media](https://web.archive.org/web/20221006183705/http://imagespacemedia.com/) (前身为 [Picad Media](https://web.archive.org/web/20221006183705/http://www.techcrunch50.com/2008/demopit_company.php?demopit=21) )已经筹集了 100 万美元的新资金。

ISM 的广告网络最近[推出了一个分析产品](https://web.archive.org/web/20221006183705/https://beta.techcrunch.com/2010/03/26/image-space-media-upgrades-in-image-ad-network-with-analytics-dashboard/)和一个[自助服务工具](https://web.archive.org/web/20221006183705/https://beta.techcrunch.com/2010/06/02/image-space-media-launches-self-service-ad-creation-tool/)，帮助出版商通过广告覆盖将他们网站上的图片货币化。它的专有技术允许观众定位，并将广告匹配到最合适的可用图像。

有了 AdStart，广告商现在可以在每次点击成本(CPC)或每千次点击成本(CPM)的印象水平上创建基于文本的广告。一旦广告商创建了他们的基于文本的广告活动，他们就设定他们的投标价格，并使用信用卡或 PayPal 为他们的账户提供资金，预付账户的最低金额为 5.00 美元。广告立即开始在图像空间媒体网络上作为上下文相关图像的覆盖图运行。

今年早些时候，这家初创公司从新大西洋风险投资公司、里奇兰资本公司和迈克尔·高顿公司筹集了 290 万美元的资金，并于去年任命了新的首席执行官杰西·切纳德，董在 2008 年的 TechCrunch50 上见过他。图像空间媒体面临来自 GumGum 和其他媒体的竞争。

我们已经联系了初创公司进行确认，当我们得到回复时将会更新。

ISM 创始人 Kevin Tung 证实，在 2011 年计划的 B 轮融资之前，ISM 已经从现有投资者那里筹集了另一轮资金。
