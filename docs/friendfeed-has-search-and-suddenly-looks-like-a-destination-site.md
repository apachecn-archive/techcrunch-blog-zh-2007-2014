# FriendFeed 增加了搜索，突然感觉像是一个目的地网站

![](img/13f80e3ce28a5a874ee63cc3611dfe2a.png)

FriendFeed，一个聚合社交网络信息的服务，刚刚[推出了](https://web.archive.org/web/20221221074504/http://blog.friendfeed.com/2008/03/friendfeed-has-search.html)一个相当不错的小搜索功能。用户可以按个人、朋友或所有用户进行搜索，并指定只搜索特定的服务，如 Twitter 或 Delicious。每个页面上都有一个基本搜索框。高级搜索是[这里的](https://web.archive.org/web/20221221074504/http://friendfeed.com/search/advanced)。

FriendFeed 显然不仅仅是一个从其他网站收集大量数据然后通过 RSS 访问的简单服务。用户可以直接对发布的内容发表评论(而且经常这样做)。现在他们也可以搜索了。越来越多的 FriendFeed 看起来像是建立在所有第三方数据基础上的目的地网站。

他们很聪明，这是我们过去见过的。最近的一个例子是 Meebo，它最初是用户在网上访问雅虎、AIM、GTalk 和 MSN 等即时消息服务的一种方式。但最终他们开始允许用户创建自己的账户，然后推出了自己的专有聊天服务[和](https://web.archive.org/web/20221221074504/http://techcrunch.com/2006/08/02/meebo-launches-meebome/)。Meebo 一直说这不是他们整体战略的重要部分，但我注意到我现在在 Meebo 上的大多数聊天都是通过他们的服务完成的，而不是大型 IM 玩家之一。

该公司在过去一周左右的时间里用户数量激增(记住，这项服务仅在 2 月 25 日正式推出)。他们不会透露确切的用户数量，但是他们说 FriendFeed 的用户基础在周末增长了 10%，在上周增长了 25%。他们显然正在享受他们的 [Twitter 时刻](https://web.archive.org/web/20221221074504/http://techcrunch.com/2008/03/14/friendfeed-is-this-years-twitter-but-why/)，并且到目前为止，已经避免了困扰 Twitter 两年的所有正常运行时间问题。不管你觉得它很美还是很可笑，FriendFeed 都在崛起。

暂时回到 Twitter 上的比较——FriendFeed 在推出几周后就有了强大的搜索功能。Twitter 仍然没有推出有意义的搜索，即使在几个月前[承诺推出之后](https://web.archive.org/web/20221221074504/http://techcrunch.com/2007/09/24/real-time-search-now-at-twitter/)。Twitter 需要小心——执行失败已经让许多受欢迎的创业公司[陷入困境](https://web.archive.org/web/20221221074504/http://www.crunchbase.com/company/friendster)。Twitter 是时候加快步伐了。

FriendFeed 总部位于硅谷，已经为 T4 筹集了 500 万美元的风险投资。
