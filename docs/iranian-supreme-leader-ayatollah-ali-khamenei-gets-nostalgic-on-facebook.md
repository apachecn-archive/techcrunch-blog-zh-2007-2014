# 伊朗最高领袖哈梅内伊怀念脸书

每个人都喜欢分享照片……甚至铁腕神权统治者也决心摧毁社交网络所体现的自由。伊朗最高领袖阿亚图拉·阿里·哈梅内伊(Ayatollah Ali Khamenei)的政府禁止脸书煽动民主抗议，他创建了自己的一页，对时间线的使用令人惊讶地复杂。他甚至在 Instagram 上展示了自己内心的潮人形象，并开始怀旧，[上传了一张自己少年时代的照片](https://web.archive.org/web/20221202021833/http://instagram.com/p/TOrXkELVVx/),照片上的他是一名年轻的革命者:

![Photo Credit: Instagram User khamenei_ir](img/efc7b5d7e0cc05d81458ed0c658ade28.png)

图片来源:Instagram 用户 khamenei_ir

当然，十有八九，这是一个极端独裁政权的又一个厚颜无耻的宣传工具，而阿亚图拉本人却不知道如何将照片上传到 Instagram。但是，伊朗的政策制定者们之前已经表达了他们对西方互联网服务的热爱，[停止互联网审查以获得更多的 Gmail](https://web.archive.org/web/20221202021833/https://beta.techcrunch.com/2012/10/01/google-1-theocracy-0-iranians-successfully-demand-gmail-be-unblocked/) 。所以也许，仅仅是也许，他像世界上其他许多人一样沉迷于脸书。我等待着他邀请我和朋友们玩文字游戏。
