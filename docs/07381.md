# 不要称它为 PivotPlz: PicPlz 是在混合媒体实验室准备他们的下一个产品时分离出来的

当[混合媒体实验室](https://web.archive.org/web/20230203083617/http://mixedmedialabs.com/)去年 11 月筹集到 500 万美元时，它引起了不少人的关注。毕竟，这笔钱来自支持 Instagram 的风投公司 Andreessen Horowitz。公平地说，他们支持 Instagram，当时它还被称为 Burbn，在它成为现在非常受欢迎的移动照片分享应用程序之前。然而，毫无疑问，混合媒体实验室的 PicPlz 产品是一个直接的竞争对手。安德森·霍洛维茨站在他们一边，而 Instagram [找到了其他投资者](https://web.archive.org/web/20230203083617/https://techcrunch.com/2011/02/02/instagram-funding/)。

[但正如莎拉当时指出的](https://web.archive.org/web/20230203083617/https://techcrunch.com/2010/11/11/andreessen-horowitz-isnt-hedging-its-bets-with-picplz-and-instagram-it-has-picked-picplz/)，安德森的赌注与其说是针对 Instagram，不如说是押注于混合媒体实验室创始人[道尔顿·考德威尔](https://web.archive.org/web/20230203083617/http://www.crunchbase.com/person/dalton-caldwell)。Caldwell 之前创建了 imeem，据说他对公司有着超越照片共享的广阔愿景。PicPlz 只是第一个利用他们所创造的东西的产品。虽然它取得了一定的成功，但它还没有看到 Instagram 在用户方面取得的成功。我们了解到，现在混合媒体实验室已经开始开发他们的下一个产品。

具体来说，PicPlz 已经被剥离出来，成为一家独立的公司，将由 Napster、imeem 以及混合媒体实验室(Mixed Media Labs)的前高管阿里·艾达尔(Ali Aydar)运营。几个星期以来，我们一直听说 PicPlz 正在四处寻找潜在的现金交易，因为混合媒体实验室试图专注于下一步。但最终，他们决定简单地将 PicPlz 剥离出来，成为一家独立的公司，同时保留一些所有权。

联系到 Caldwell 进行评论，他证实了 PicPlz 的分拆，但拒绝进一步讨论这笔交易，只是表示对他的团队下一步的工作感到兴奋。

那么混合媒体实验室下一步的工作是什么呢？我们所能确定的是，它符合应用程序领域广泛理念的最初目标。具体来说，我们还听说从第一天开始就会非常关注商业模式，不像 PicPlz。

与此同时，PicPlz 将作为一项服务继续存在，但 Caldwell 和混合媒体实验室团队的其他成员将不再参与其中。

离开 PicPlz 正值移动照片分享领域一个有趣的时刻。在 Instagram 继续快速获得用户的同时，其他玩家如 [Path](https://web.archive.org/web/20230203083617/http://path.com/) 和 [Color](https://web.archive.org/web/20230203083617/http://color.com/) 也一直在准备其他产品。Path 最近推出了[他们的第一个“联合”](https://web.archive.org/web/20230203083617/https://techcrunch.com/2011/06/07/path-with-twitter/)——hipster/德瑞医生行话“副业”— [与](https://web.archive.org/web/20230203083617/http://with.me/)。与此同时，据说 Color Labs 在 [Color 未能以他们希望的方式抓住](https://web.archive.org/web/20230203083617/https://techcrunch.com/2011/06/20/we-can-do-it-til-we-both-wake-up/)之后，正在准备一个全新的应用程序。
