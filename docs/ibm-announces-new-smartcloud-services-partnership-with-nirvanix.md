# IBM 宣布与 Nirvanix 合作推出新的智能云服务

IBM 今天[推出了新的 IBM SmartCloud 服务，扩展了](https://web.archive.org/web/20230407083143/http://www-03.ibm.com/press/us/en/pressrelease/35593.wss)面向企业的云计算服务。这些新产品包括 SmartCloud 应用服务、IBM 面向企业的新 PaaS、名为 SmartCloud Foundation 的硬件和软件组合，以及 SmartCloud 生态系统，其中包括面向 IBM 合作伙伴和 ISV 的新服务。

该公司计划到 2012 年底支持大约 2 亿用户。

IBM 表示，它已经将 SAP、ERP 和所有数据库应用程序移植到其 SmartCloud 平台上。借助其新服务，it 部门现在可以自动化与在云中运行 SAP 相关的劳动密集型任务，尤其是 SAP 克隆、更新和修补。在[的一些案例中](https://web.archive.org/web/20230407083143/http://www-03.ibm.com/press/us/en/pressrelease/35593.wss)，IBM 称根据任务的不同，节省的时间可能从几个月或几周到几天或几小时不等。

今天[还宣布了与云存储公司 Nirvanix 的新合作关系。展望未来，Nirvanix 的云存储技术将集成到 IBM 的 SmartCloud 企业存储服务中，允许客户在世界任何地方上传任何大小的文件，并从任何地方访问它们。存储即服务产品可以支持数百万用户、数十亿对象和数十亿字节的数据。](https://web.archive.org/web/20230407083143/http://www-03.ibm.com/press/us/en/pressrelease/35594.wss)

与此同时，IBM 的新平台即服务产品 smart cloud Application Services(SCAS)将于今年(第四季度)推出测试版，提供企业级安全性、开放式 Java 和跨平台支持(不受供应商限制),以及一套全面的应用基础设施和托管服务，以实现应用到云的开发和部署。它将在 IBM 的 SmartCloud Enterprise 和 Enterprise+上运行，这是一种企业级基础设施即平台，专门用于按照承诺的 SLA(服务级别协议)运行企业工作负载。IBM SmartCloud Enterprise+已在美国上市，并将于 2012 年底在全球部署。

最后，全新的[SmartCloud Foundation](https://web.archive.org/web/20230407083143/http://www-03.ibm.com/press/us/en/pressrelease/35596.wss)**私有云解决方案系列旨在帮助公司快速设计和部署私有云环境，包括 SmartCloud Entry、预打包的私有云软件，以及 smart cloud 配置和监控软件。**
