# 凭借《山狮》, OS X 向 iOS 靠拢 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/16/mountain-lion-os-x-ios/>

自从苹果[发布了 OS X 狮子](https://web.archive.org/web/20230127141259/https://techcrunch.com/2010/10/20/os-x-lion/)，它的桌面操作系统开始了[向苹果另一个更受欢迎的操作系统 iOS](https://web.archive.org/web/20230127141259/https://techcrunch.com/2010/10/25/os-x-lion-multi-touch/) 的漫长跋涉。随着下一个版本的 OS X、[、山狮](https://web.archive.org/web/20230127141259/https://techcrunch.com/2012/02/16/os-x-mountain-lion/)(今天作为开发者预览版发布)，OS X 将会更加接近它的 iOS 表亲。

OS X 的许多新功能直接来自 iOS。其中包括更深入的 iCloud 集成、消息、提醒、笔记、通知中心、游戏中心、AirPlay，以及内置的[分享到 Twitter](https://web.archive.org/web/20230127141259/https://techcrunch.com/2012/02/16/twitter-os-x-mountain-lion/) 、电子邮件等等。

随着时间的推移，苹果正在将越来越多的 iOS 功能内置到 OS X 中，以在两者之间创建更无缝的体验。正如苹果首席执行官蒂姆·库克本周早些时候在高盛会议上提到的那样，更多人对苹果产品的第一次体验是 iPhone 或 iPad，而不是 Mac。“iPhone 开始向数亿以前从未见过苹果的人介绍苹果，”他说。iPhone 的这种著名的光环效应要比 iPod 大几个数量级。购买 iPhone 的人更有可能购买 Mac，当他们购买 Mac 时，他们会期待一种熟悉的体验。

在所有移植到 Mac 的 iOS 功能中，最重要的是 iCloud、通知和社交分享。当然， [iCloud 已经是 iPhoto 等一些 Mac 桌面应用的一部分，但随着 Mountain Lion 的推出，它将包含 iWorks docs。你所有的文档都将被同步到云端，消除了跟踪不同版本的麻烦。苹果也可以向其他应用程序开放 iCloud 功能，使其成为该操作系统的一个关键功能(但它必须增加目前提供的区区 5 GB 的免费存储空间)。iCloud 是 iOS 和 OS X 应用之间的纽带，它只会继续增长。](https://web.archive.org/web/20230127141259/https://techcrunch.com/2011/06/06/icloud-digital-hub/)

苹果自己的桌面应用程序，如 Safari，将开始显示 iOS 用户熟悉的共享按钮图标(带有向右箭头的方框)。这将允许用户通过电子邮件、消息、Twitter 或其他服务分享网页、照片或文档。通过这种方式，共享将被嵌入到操作系统中。脸书目前还不在考虑范围之内，但如果两家公司能够解决分歧，这种情况是有可能发生的。

最后，在桌面上添加推送通知可能是最大的改变。任何拥有 iPhone 或 iPad 的人都知道应用程序通知有多令人上瘾(或者很烦人，如果不是你的 iPhone 的话)。通知已经成为[新的信息总线](https://web.archive.org/web/20230127141259/https://techcrunch.com/2011/12/18/remembering-email/)——持久的、总是在你面前的、实时的。它们是应用程序的即时消息，并且越来越多地成为我们的机器与我们交谈的方式，将我们拉进一个不断发出 pings 命令的应用程序，“嘿，刚刚发生了一些事情。”这些推送通知将从苹果应用程序开始，但很快将通过 API 提供给开发者。随着 iOS 开发人员创建更多的 Mac 配套应用程序，无论您是在 iPhone、iPad 还是 Mac 上，您都应该开始看到相同的通知弹出。
