# 告诉他们 MySpace 好友| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/23/tell-them-myspace-friends/>

一项新的手机服务为我们这些想和 MySpace 上的朋友保持联系的人提供了。“告诉他们”是一项让用户通过移动设备发送图片和文本的服务。这项服务是免费的，使用简单。

首先，去你手机上的告诉他们网站。输入您的 MySpace 电子邮件密码，并选择您想联系的朋友。写一条信息，如果你愿意，可以附上一张图片，然后 blamo，信息就发出来了。

这项服务非常新，所以现在还不能说它是否会流行。Tell Them 似乎是目前唯一一个专门迎合想发送 MySpace 消息的移动用户的服务。通过为像 MySpace 这样的流行现象提供特定服务，Tell Them 有潜力快速增长。如果这项服务受到欢迎，告诉他们的消息会很快在 MySpace 上传播开来。
