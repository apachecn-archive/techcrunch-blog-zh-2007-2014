# 波音黑色安全智能手机遭遇 FCC，目标是成为下一个黑莓

黑莓仍然享有一些政府智能手机合同，但波音公司即将推出的新 Android 设备可能会威胁到这种关系。这家航空巨头去年承认开发了一款安全智能手机，但现在所谓的“波音黑”已经袭击了 FCC(通过 [Myce](https://web.archive.org/web/20230128110820/http://www.myce.com/news/secret-secure-dual-sim-boeing-black-android-smartphone-hits-fcc-70611/) )，提供了更多的背景信息。

波音黑色飞机被设计为“主要出售给政府机构和与国防和国土安全相关机构进行合同活动的公司。这款设备的营销和销售方式将不会向公众提供关于该产品的低级技术和操作信息，”根据 FCC 的文件，因此它肯定不会像 Silent Circle 的 [Blackphone 和 Geeksphone](https://web.archive.org/web/20230128110820/https://techcrunch.com/2014/02/26/close-look-at-blackphone/?ncid=rss) 那样，更旨在保护注重隐私的普通消费者的安全。

然而，这显然是对黑莓在美国剩余市场份额的核心一击。这家加拿大智能手机制造商刚刚宣布了与五角大楼的新合同，在 1 月 31 日开始的首次展示中，[将向国防部雇员](https://web.archive.org/web/20230128110820/http://bgr.com/2014/01/20/blackberry-pentagon-contract-ios-android/)提供 8 万部新的黑莓手机，这意味着它仍然是美国安全政府机构的首选供应商。

三星也试图通过其设计用于 BYOD 设备的 [KNOX 软件和新市场](https://web.archive.org/web/20230128110820/https://techcrunch.com/2014/02/25/samsung-introduces-knox-marketplace-an-enterprise-app-store-powered-by-appdirect/)来打开这个市场。然而，这仍然没有黑莓的设备强大，也远远低于波音公司新推出的双 SIM 卡、LTE 供电的超机密智能手机套件。尽管如此，三星显然希望朝着这个方向前进，并正在发展其方法来帮助实现这一目标。

目前，黑莓仍保持着其作为敏感企业环境下使用的智能手机平台的选择角色，但这不太可能永远持续下去，除非它能够扭转其业务的其他部分。寄希望于成为需要增强安全性的行业的下一个主要平台提供商是一个明智的举动，但我们必须等待，看看波音在宣布其为智能秘密移动新国王之前到底走了多远。

这里有一个简短的版本，为那些在家跟踪的人提供上述产品的相关信息:

*   **KNOX:** BYOD 安全性适用于不需要两台设备且不需要处理国家机密的企业用户。
*   **Blackphone:** 为偏执的消费者准备的智能手机，他们愿意多付一点钱，注册订阅服务，以换取隐私和内心的平静。斯诺登启发的。
*   **波音黑:**可能自毁的詹姆斯·邦德手机，允许通信进入，但阻止间谍进入。十有八九也是斯诺登启发的，但来自相反的方向。
