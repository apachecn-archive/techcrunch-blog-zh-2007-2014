# 在 CRV | TechCrunch 460 万美元的支持下，Cybereason 将其恶意操作检测平台带出秘密

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/11/cybereason-launch/>

数字安全行业的大部分精力都集中在防止黑客攻击上。Cybereason 是一家专注于安全的初创公司，由一群前以色列情报机构成员创建，今天将推出其商业产品，它采取了不同的策略。

它认为，企业应该接受安全漏洞是不可避免的，而不是在监控和检测方面加强工具——给自己一个更好的机会，最终防止严重的数据泄露。

“在电影中，你可能会看到有人破门而入，然后不久就带着(公司机密)跑了。但在现实中，这些攻击者正在进行一个漫长的过程和一系列不同的行动和事情，”Cybereason 副总裁马克·泰伯告诉 TechCrunch。

“直到几年前，这种复杂性还只存在于国家层面，而不存在于企业层面。但现在它真的进入了企业。不幸的是，企业没有很好的装备来发现这种黑客行为，因为他们所有的技能和资金都用于预防。”

Cybereason 为企业建立了一个主动安全平台，以持续监控系统，目的是在黑客试图准备数据盗窃时检测黑客的行动和意图，从而让企业有机会在黑客有机会偷走他们公司皇冠上的宝石之前切断黑客攻击。

因此，Cybereason 的平台并不打算检测恶意软件本身，而是检测恶意操作的泄露秘密的签名，无论它们可能是什么——例如在网络中的多台机器上安装的不一致的程序序列，可能不符合用户特征，或者其他微妙的异常活动。

Taper 说:“恶意软件可能是恶意操作的一部分，但我们确实在关注黑客或黑客操作为实现其最终目标而做的一系列事情。“我们正在寻找(黑客活动的)微小痕迹……以揭露这一操作并阻止它。”

Cybereason 在其一个测试用户(一家未具名的大型媒体和娱乐公司)发现的一个正在进行的恶意操作的具体例子是，一个 IT 工具被安装在多个(非 IT)用户的计算机上，然后被用来远程打开摄像头和麦克风，因为黑客可能试图窃听大楼内的活动。

“正是在这一点上，我们当场停止了这件事。他补充道:“这已经上升到了恶意操作的地步，而该公司却能够终止整件事。”。

“当时的情况是，这个攻击者远程控制这个软件，试图四处移动，试图找出高管办公室的位置，然后在适当的时候打开摄像头和麦克风，记录在某个特定会议上办公室里发生的事情，然后窃取这些信息。”

今天，Cybereason 不仅推出了商业平台，还宣布从Charles River Ventures获得了 460 万美元的首轮融资，这是它的第一笔外部融资。

CRV 合伙人伊扎尔·阿莫尼(Izhar Armony)在一份声明中评论这笔资金时说:“cyber eason 团队为当今没有有效解决方案、损失高达数十亿美元的市场带来了独特的方法和新的见解。”

在这次 A 轮之前，Cybereason 是自筹资金的。据泰伯称，新的资金将用于平台的进一步开发以及将技术推向市场。

该平台可自动检测黑客踪迹，无需企业全天候雇佣经验丰富的安全分析师，而是使用特征分析和大数据分析来取代人工安全分析师的角色。

“我们有收集器，即位于端点的代理，它们查看各种差异数据。泰伯解释说:“我们对应用程序进行了分析，对用户进行了分析，我们查看程序(某个特定用户可能会使用与其他具有相同工作角色的用户相关的程序)，我们不断建立档案，寻找用户群或整个企业甚至整个企业的用户的罕见事件，以了解不同公司如何使用不同的应用程序。”。

“然后我们有一个大数据分析平台，然后我们添加上下文，添加所有不同的威胁情报，我们对这些黑客操作如何工作的知识，然后我们分析所有这些差异数据，直到系统开始建立证据，然后证据导致怀疑，然后怀疑会导致我们将识别的恶意操作。”

Taper 补充说，该平台还可以与雇用专业威胁分析师的企业(如银行公司)配合，作为支持工具和数据存储库，供这些分析师使用和查询。

Cybereason 成立于 2012 年，现已搬到美国，总部设在美国，但迄今一直处于隐形模式，拥有 15 名测试用户。

就其主要竞争对手而言，它命名为 Crowdstrike 和 kailing——但认为它们各自的方法不同。

“有两个[主要竞争对手]——crowd strike 正试图解决与我们相同的问题，但正在使用真人来支持他们的云服务，就像 Mandiant 2.0 一样。通过 Cybereason，我们已经将世界顶级安全分析师的情报构建到我们的平台中，因此企业基本上拥有一个由顶级分析师组成的虚拟军队，全天候搜索恶意操作，”它告诉 TechCrunch。

“另一个是 kaitlin，它正在用分析方法解决同样的问题，并为企业提供大量数据。但是，为了找到正确的信息，企业/安全分析师需要知道该问什么问题才能成功。Cybereason 降低了安全分析师的门槛，因为我们主动为他们提供所有正确的信息，并通知他们可能的恶意操作。”

为了进一步降低缺乏专业安全知识的企业更好地了解恶意入侵其系统的门槛，Cybereason 的平台包括一个可视化信息图表功能，以说明/叙述攻击的轨迹。

“这一切真的很复杂。大多数公司没有经验丰富的安全人员能够知道对所有这些证据和我们正在收集的所有东西提出正确的问题，所以我们所做的很大一部分是将这些恶意操作可视化，以使理解正在发生的事情变得非常容易，”Taper 补充道。
