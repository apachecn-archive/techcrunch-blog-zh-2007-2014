# Thermaltake 的移动风扇 12 是一个外部内部计算机风扇

西雅图的气温已经接近太阳表面了，但这个城市没有人拥有一台像样的风扇，因为…为什么？幸运的是，我的工作需要我一动不动地坐在电脑前，而我看起来很酷，而我的一个完全不合适的复古铬风扇在冷却我的公寓方面做得很差。

进入 Thermaltake，高质量计算机冷却产品的制造商。他们已经将一个标准机箱风扇封装在一个独立的线框中，[所以现在你的 CPU 和手掌都将有专用的冷却系统。](https://web.archive.org/web/20221206205033/http://www.thermaltakeusa.com/Product.aspx?C=1295&ID=1914)这听起来很傻，但我很喜欢。它是 USB 供电的，有一个转盘，所以你可以把它的速度设置在 850 到 1500 RPM 之间。很明显，它会变得很吵，所以我怀疑他们会有一场无声的比赛。

只卖 25 美元(不到 15 美元一条街)，我觉得很便宜。
