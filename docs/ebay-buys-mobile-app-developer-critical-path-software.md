# 易贝收购移动应用开发者关键路径软件

易贝正在进行一场假日购物狂欢。在[宣布收购](https://web.archive.org/web/20230319221339/https://techcrunch.com/2010/12/02/confirmed-ebay-acquires-milo-for-75-million-investors-make-a-killing/)米洛的几周后，这家电子商务巨头刚刚[收购了](https://web.archive.org/web/20230319221339/http://www.businesswire.com/news/home/20101215006520/en/eBay-Acquires-Industry-Leading-Mobile-Application-Developer) [关键路径软件。](https://web.archive.org/web/20230319221339/http://www.criticalpath.com/)

这家总部位于波特兰的移动应用开发商一直在帮助易贝开发其移动产品，包括其主要的 iPhone 应用 StubHub、易贝分类广告和 Shopping.com iPhone 应用。该协议的财务条款没有披露。

考虑到易贝大力开发其移动应用套件，这项似乎以人才为中心的收购是有意义的。最近，易贝[刚刚发布了用于安卓手机的](https://web.archive.org/web/20230319221339/https://techcrunch.com/2010/12/07/that-was-quick-ebay-adds-milo-results-to-mobile-redlaser-apps/)条形码扫描应用 RedLaser(易贝[于 6 月收购了](https://web.archive.org/web/20230319221339/https://techcrunch.com/2010/06/23/ebay-acquires-barcode-scanning-iphone-app-redlaser/))。

移动不仅是给易贝市场带来流量的好方法(主要的 iPhone 应用程序在两年内已经有 1400 万次下载)，而且也是一种赚钱的方式。易贝通过其移动产品发布了创纪录的假日销售额，预计 2010 年其市场的移动销售额将超过 15 亿美元。
