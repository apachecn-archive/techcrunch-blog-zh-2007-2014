# 在设计和数据的交汇处，脸书时间线诞生了

今天在脸书的 f8 会议上，首席执行官马克·扎克伯格展示了这项服务的巨大变化:[时间线](https://web.archive.org/web/20230204174146/https://techcrunch.com/2011/09/22/facebook-timeline/)。这是重生的轮廓，看起来很棒。随后，脸书的产品负责人[克里斯·考克斯](https://web.archive.org/web/20230204174146/http://www.crunchbase.com/person/chris-cox)上台讲述了该功能的诞生。

考克斯很大程度上归功于两个人:尼古拉斯·费尔顿和萨姆·莱辛。

考克斯对费尔顿的历史充满了诗意。他滔滔不绝地谈论着 2005 年开始的费尔顿年度报告。“14 页。一年。一本书，”考克斯指出。这一切都是为了用一种美丽的、视觉化的方式来组织他生命中的岁月。“除了它的本来面目，很难称它为别的——艺术，”考克斯继续说道。

“我们只有一个反应:我们必须试着雇用这个人。”

脸书就是这样做的。今年 4 月，当他们收购了费尔顿的初创公司 Daytum 后，费尔顿和联合创始人管·卡斯从纽约搬到了帕洛阿尔托，帮助他们将数据分析的本能融入到脸书档案的未来。

考克斯随后把注意力转向了莱辛。为了得到他，脸书去年还收购了他的初创公司 Drop.io。他只有一项工作:重塑侧写。莱辛指出，“人类故事讲述史上最大的一次错失机会”是他加入时个人资料的编排方式。为了证明自己的观点，他打印出了自 2004 年加入脸书以来在个人资料上分享的所有信息。它延伸到脸书的整个办公室。

考克斯指出，当他们在去年夏天偶然推出一款名为“回忆”的产品几个小时后，反响巨大。这是时间线带来的体验。“我们把它藏起来，继续工作，”考克斯说。

今天，这项工作已经完成——尽管 Timeline 将在接下来的几个月里向每个人慢慢推出，但从今天开始将会有一个测试版。脸书最近对设计的关注以及他们尽可能多地获得顶级设计人才的举动现在已经很明显了。

“我们是文化的建设者。现在让我们去创造一些令人敬畏的东西，”考克斯在结束语中说。
