# Dropbox 收购 Cove，将前 Facebookers 用户 Ruchi Sanghvi 和 Aditya Agarwal 带入团队

谣言是真的！Dropbox 因其人才而收购了隐形合作初创公司 Cove，即才华横溢的工程二人组和夫妇 [Aditya Agarwal](https://web.archive.org/web/20230227132026/http://www.crunchbase.com/person/aditya-agarwal) 和 [Ruchi Sanghvi](https://web.archive.org/web/20230227132026/http://www.crunchbase.com/person/ruchi-sanghvi) 。

Agarwal 和 Sanghvi 在脸书都很受尊敬，他们从 2005 年到 2010 年在那里领导产品工作。负责 Facebook 平台和新闻订阅的 Sanghvi 是脸书的第一位女工程师。她最近因为她的贡献获得了一个 [TechFellow 奖](https://web.archive.org/web/20230227132026/https://techcrunch.com/2012/02/22/the-winners-of-this-years-100000-techfellow-awards-are/)，我在颁奖仪式后对她进行了简短的采访。

在一份新闻稿中，Dropbox 联合创始人兼首席执行官[德鲁·休斯顿](https://web.archive.org/web/20230227132026/http://www.crunchbase.com/person/drew-houston)表示:“建立一个世界级的工程组织是我们的首要任务 Cove 的团队代表了硅谷最优秀的人才，我们期待他们将技术、技能和视角带到 Dropbox。”

Dropbox 没有给出关于 Cove 团队将从事什么工作的太多细节，但 Aditya 表示，“Cove 和 Dropbox 的产品愿景非常一致，我们在 Cove 建立的基础设施可以在 Dropbox 上使用。”

这是 Dropbox 的第一次收购，从我在新闻稿发布前听到的消息来看，这次收购纯粹是为了人才——据一位消息人士称，Aditya Agarwal 将成为 Dropbox 的工程副总裁。Sanghvi 的加入也值得注意，因为她在社交网络——Dropbox 据传即将计划自己的“Dropbox Connect”功能——期间曾在脸书连接上工作。

“我们认识 Ruchi 和 Aditya 已经有一段时间了，一直是他们工作的崇拜者，没有太多的人不仅想得很大，而且还创造了上亿人使用的东西，”休斯顿在电话采访中告诉我。

“随着我们过渡到 Dropbox，而不仅仅是你手机上的一个应用程序，深入融入各种不同的体验对我们来说很重要。我们的目标远不止文件共享，”休斯顿强调说。

看看 Dropbox 是否会将它的[2.5 亿美元](https://web.archive.org/web/20230227132026/https://techcrunch.com/2011/10/18/dropbox-raises-250m-in-funding-boasts-45-million-users/)资金用于平台扩张的进一步收购，这将是一件有趣的事情。在产品方面，初创公司[刚刚推出了其桌面和安卓客户端的新版本](https://web.archive.org/web/20230227132026/https://techcrunch.com/2012/02/23/dropbox-introduces-automatic-image-upload-on-android-and-it-has-more-up-its-sleeve/)，可以自动将你的照片上传到你的 Dropbox 账户，休斯顿暗示，还会有更多这种“协作和共享”类型的功能出现。
