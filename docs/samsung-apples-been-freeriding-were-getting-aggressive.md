# 三星:苹果一直在“搭便车”，我们正变得咄咄逼人 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/23/samsung-apples-been-freeriding-were-getting-aggressive/>

如果你仔细看看苹果和三星的毁灭之路(也就是他们的全球专利战)，你可能会得出苹果领先几个关键点的结论。苹果[在德国](https://web.archive.org/web/20230205041027/https://techcrunch.com/2011/09/09/apple-ftw-german-court-upholds-galaxy-tab-10-1-sales-ban/)下架了 Galaxy Tab 10.1，赢得了[EU-范围内对三款 Galaxy 智能手机](https://web.archive.org/web/20230205041027/https://techcrunch.com/2011/08/24/point-apple-court-issues-eu-wide-preliminary-injunction-on-galaxy-smartphones/)的禁令(尽管三星找到了继续销售它们的方法)，并且[也阻止了 GalTab 在澳洲的销售](https://web.archive.org/web/20230205041027/https://techcrunch.com/2011/08/29/apple-continues-to-block-the-galaxy-tab-10-1-in-australia/)。三星还没有从商店货架上撤下任何 iProduct。

显然，这是一场正在进行的战争，任何事情都可能发生，但现在苹果占据优势。任何一场胜利，无论如何被遏制，都会在心理上影响到其他法院系统和国家。因此，苹果的胜利不仅仅是在欧洲的胜利，在澳大利亚的半胜利——这是对所有其他法院的一个信号，苹果可能只是有理由支持他们的论点。对三星来说，这种影响是致命的。

美联社[报道称，三星移动通信全球营销负责人李英熙表示，三星计划对苹果采取更积极的姿态。在听到这些之后，三星在整个诉讼过程中有些犹豫的态度就更有意义了。众所周知，三星和苹果有着卓有成效的业务关系，三星是苹果的零部件供应商，而苹果则是三星最大的客户之一。](https://web.archive.org/web/20230205041027/http://www.washingtontimes.com/news/2011/sep/23/ap-interview-samsung-steps-up-apple-patent-fight/)

在这场斗争中，苹果没有采取多少措施来保护这种关系，它尽可能地在任何国家维护自己的权利，并指控侵权。当然，三星自己也提起了大量诉讼和上诉，但几乎在每一个案件中，它都表现出一种报复性的举动，而不是转向进攻。通过这种方式，三星稳扎稳打，避免越过任何界限，直到公司准备好永不回头。

[![](img/09a42de84993364e56241355f52f6e81.png "samsung headquarters")](https://web.archive.org/web/20230205041027/https://techcrunch.com/wp-content/uploads/2011/09/samsung-headquarters1.jpg) 但那就结束了。在这场战斗的早期，三星似乎对诉讼案感到受宠若惊。三星电子董事长李健熙说:“当一根钉子伸出来时，人们试图把它敲下去。(此类事件)就像是公司为了继续增长而必须经历的一个仪式。”那是在四月，当整个混乱开始的时候。现在，事情完全不同了。很明显，苹果正在直取要害，而三星只是在为自己辩护。一个强有力的辩护(关于三星发起的诉讼)，但仍然是一个辩护。

据李英熙说，“从现在开始，我们将以更积极的方式来争取我们的权利。”她还补充说，苹果一直在三星的无线通信技术专利上“搭便车”。这与一位未透露姓名的三星高管对《韩国时报》的评论非常吻合，他承诺 iPhone 5 在韩国上市后，将立即跟进。这位高管提到，避免此类诉讼的唯一方法是苹果从 iPhone 中移除“移动电信功能”。

在我看来，三星试图扭转苹果的局面。在扮演了受害者的角色(在一定程度上)后，三星可能会开始投掷炸弹，希望确保稳固的胜利。如果发生这种情况，苹果的优势就会消失，和解的可能性对侵略者苹果来说会变得更有吸引力。从目前的游戏方式来看，和解实际上是三星的一大胜利。它可能会保留与苹果的业务关系，同时向世界证明苹果确实害怕三星及其日益增长的业务。
