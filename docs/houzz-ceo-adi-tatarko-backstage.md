# Houzz 首席执行官 Adi Tatarko 谈如何建立成功的市场

扩大市场规模是一个巨大的挑战，但 Houzz 似乎已经破解了密码，将希望改造房屋的客户与能够完成这项工作的当地专业人士联系起来。在 TechCrunch Disrupt SF 的后台，我与 Houzz 的首席执行官 Adi Tatarko 讨论了该公司是如何变得如此庞大的。

Houzz 最初是一个宠物项目，Tatarko 和她的丈夫开始将其作为帮助自己和朋友改造家园的一种方式。但是在关于市场的消息传开后，Houzz 能够让设计师和建筑师加入进来，而不用做太多的工作。很快，Tatarko 在全国各地的市场上找到了用户和供应商。

“看到人们通过口口相传了解它的方式很有趣，因为很多时候专业人士会告诉他们在其他州的同行，”塔塔尔科说。这包括新城市和平台上没有的不同类别的专业人士。专业人士会告诉其他州的同事，他们也会告诉他们的客户关于 Houzz 的事情。

那么塔塔尔科什么时候意识到她的宠物项目是一个真正的生意呢？“当我们意识到它是跨州的，越来越多的人加入，并且有开放其他类别的需求时，”Tatarko 说。她发现其他州的人在没有任何真正推广的情况下也在寻找和使用这个平台。

塔塔尔科说，要建立一个成功的市场，你需要倾听你的用户，包括买家和卖家。Houzz 一直通过从用户那里获取反馈，并根据他们的要求实现新功能来做到这一点。

“我认为听取双方的意见并为你的用户创造一个产品是非常重要的，”Tatarko 说。“你不是为了投资者而创作，你不是为了在 powerpoint 上好看而创作。你是在为真实的人创作，你需要倾听他们的声音。”
