# PROskore 的新商业网络旨在成为 LinkedIn+Klout+Leads Generation TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/25/proskores-new-business-network-aims-to-be-linkedin-klout-leads-generation/>

PROskore 是一个新推出的商业网络，它结合你的职业经验对你的社交媒体活动和影响力进行排名，以便给你一个分数来表示你的整体职业影响力。乍一看，这听起来很像 [Klout](https://web.archive.org/web/20230204193948/http://www.klout.com/) ，一个测量网络影响力的流行服务。

但与 Klout 不同的是， [PROskore](https://web.archive.org/web/20230204193948/http://www.proskore.com/) 在其评分方法中考虑了你的业务关系和实际工作经验。它也是专门为商业网络而构建的，允许其用户专注于发展他们的公司。与此同时，Klout 通过其 Klout Perks 项目，更专注于帮助品牌和广告商发展他们自己的业务。

据首席执行官 Bill Jula 称，PROskore 和 Klout 之间有三个主要区别。套用一句:

*1)它的所有用户配置文件都是由人创建的(即手动创建)。与此同时，Klout 的许多个人资料都是从 Twitter 上导入的。*

*2) PROskore 讲的是你的职业声望，Klout 讲的是你的社会影响力。*

PROskore 是一个商业网络，而 Klout 是为了让它的广告客户受益。

该系统实际上比简单的“社交”排名更加细致入微。举个例子，正如[这篇公司博文](https://web.archive.org/web/20230204193948/http://proskore.blogspot.com/2011/10/its-all-relative.html)所解释的，纽约一个 45 分的网页设计师和佐治亚州一个同样分数的小镇的网页设计师是不一样的。换句话说，得分和排名是相对于你所在的地区而言的。

为了获得分数，PROskore 会利用你的社会影响力(这可能是与 Klout 最相似的方面)，加上你在 PROskore 上的活动(例如，你拥有的联系数量)和你的职业简历。在建立你的个人资料时，重点是增加与业务相关的联系，而不是社会影响。例如，它会询问你的脸书粉丝页面网址，而不是你的个人资料网址。

该网站的用户可以通过网站顶部的“寻找专业人士”部分搜索其他成员。对于那些已经填写了他们所提供的服务的人，此部分可以帮助生成线索，这些线索会自动直接发送到您的 PROskore 收件箱，并可以选择转发到您的个人电子邮件收件箱。

总部位于佛罗里达州萨拉索塔的 PROskore 是从一个名为 [Fast Pitch 的现有在线商业网络中分离出来的。](https://web.archive.org/web/20230204193948/http://www.crunchbase.com/company/fast-pitch)五年的收入和超过 100，000 名会员。但是，大部分前快速沥青！成员尚未完成配置文件向导，无法在新改造的网络上生成他们的 PROskore。

由于 Fast Pitch 带来的持续收入，PROskore 目前是自筹资金的。该公司采用免费增值模式，T2 可以选择每月、每季度和每年向 T3 支付费用，以获得该网站的高级功能，如自动生成潜在客户。付费用户也可以利用其他以商业为中心的应用程序，如电子邮件营销系统和新闻稿制作和发布工具。Jula 说该网站的转化率刚刚超过 10%。

在短期内，PROskore 专注于最大限度地实现病毒式增长，并将为会员推出六个个人资料徽章(Jula 向我们保证，虽然不会像 Foursquare 上那样“可爱”)。

该公司已经讨论了接受额外的资金来改善其营销工作，但还没有做出决定。
