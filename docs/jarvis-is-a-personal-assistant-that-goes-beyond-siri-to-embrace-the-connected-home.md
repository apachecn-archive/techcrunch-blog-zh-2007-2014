# 贾维斯是一个个人助理，超越了 Siri，拥抱互联家庭

如果你是钢铁侠的粉丝，你应该已经知道托尼·斯塔克的私人助理贾维斯(Jarvis)(他要么是人类，要么是虚拟的人工智能，取决于你关注这部漫画的时间长短)。贾维斯是保持斯塔克的商业、个人和超级英雄生活顺利进行的粘合剂。这就是为什么 Jarvis 是今年 Disrupt NY Hackathon 上一个团队构建的数字助理的完美名字。

Hack 的联合创始人 Felix Rieseberg 告诉我 Jarvis 是如何工作的，使用 Twilio、Weather Underground 和 Ninja Blocks 提供的 API 来帮助你控制你的家，查看当前状况、头条新闻和正在制作的新闻等，所有这些都只需通过从任何电话拨打号码并发出语音命令，它就像一个 Siri，但安装在 Windows Azure 上，能够插入更多功能。

里斯伯格说，实用的贾维斯还有很长的路要走，因为它不是真正的人工智能，在现实世界的条件下很容易受挫。但这种兴趣显然是存在的:他说已经有人接洽过他，并提供了种子投资，只是基于舞台上展示的快速演示，贾维斯关掉了一个与 Ninja Blocks 相连的飞利浦 Hue 灯。

Jarvis 体现的是联网家庭和数字个人助理的自然下一步，但 Jarvis 更多的是一个概念验证，而不是一个运输产品。尽管如此，很快有一天从任何地方控制你的家和收集信息将像打电话一样简单，贾维斯建议这一天不会太远。
