# 保护知识产权的反对者声称即将到来的众议院版本可能会伤害 Twitter，YouTube，脸书

显然“在被证明有罪之前都是无辜的”在网络上站不住脚。至少一个名为[要求进步](https://web.archive.org/web/20230203110517/http://act.demandprogress.org/sign/protectip_docs)的团体不这么认为，该团体声称即将出台的众议院版保护知识产权法案可能会给原告提供更多的权力。更糟糕的是，像 Twitter 和 YouTube 这样的网站可能很快就会遭到攻击。

该法案是对不受欢迎的 COICA 法案的重写，于五月由参议院司法委员会通过。即使在最初的状态下，该法案似乎在许多方面与这个国家的无罪推定相矛盾，不仅允许司法部，而且允许*私权持有者*指控一个网站侵权，并在法院批准下将其撤回。如果司法部提出指控(而不是私人权利持有人)，该命令的范围将从支付处理器和广告网络延伸到互联网服务提供商和搜索引擎，从而要求搜索引擎审查其索引中涉嫌侵权的网站。

然后，在被告网站通过 ringer 后，该网站可以“请求法院暂停或撤销该命令。”正是时候，对吧？就其本身而言，这看起来并不好，但需求进展声称众议院版本可能会使事情朝着错误的方向更进一步。据该组织称，新版本将要求托管用户生成内容的网站对其用户的侵权行为负责。

这意味着，如果有人在 YouTube、Twitter 或脸书上发布受版权保护的内容，司法部将有权追究网站所有者的法律责任，甚至关闭对这些网站的服务。如果 Demand Progress 对即将推出的内部版本 PROTECT IP 的解释是准确的，它将有效地推翻[数字千年版权法案](https://web.archive.org/web/20230203110517/http://en.wikipedia.org/wiki/Online_Copyright_Infringement_Liability_Limitation_Act)中保护互联网服务提供商和网站不对其用户发布的内容负责的部分。

保护知识产权的倡导者声称，新法案将保护美国消费者免受销售未经批准和潜在危险产品的外国“流氓网站”的影响，同时通过清除销售假冒产品的网站来帮助美国就业。这一切都很好，但难道不应该担心科技行业这种动荡的连锁反应吗？

显然，有。据《个人电脑世界》报道，消费电子协会、计算机和通信行业协会以及网络联盟在周一给参议院司法委员会的成员发了一封信。“科技行业正在带领美国走出衰退，对科技行业的无意损害不可能发生在一个更糟糕的时间，”信中写道。
