# 谷歌在 Android 和 iOS 上的语音搜索现在可以用德语、法语和日语回答问题

谷歌对语音搜索情有独钟已久，但直到现在，它唯一完全支持的语言是英语。尽管语音搜索本身可以支持几十种语言，但谷歌唯一可以用语音回答的语言总是英语。如今这种情况正在改变。谷歌刚刚[宣布](https://web.archive.org/web/20221209072148/http://insidesearch.blogspot.com/2013/12/voice-search-gets-international-upgrade_5.html)其用于 [iOS](https://web.archive.org/web/20221209072148/https://itunes.apple.com/us/app/google-search/id284815942?mt=8&ign-mpt=uo%3D4) 和 [Android](https://web.archive.org/web/20221209072148/https://play.google.com/store/apps/details?id=com.google.android.googlequicksearchbox) 的搜索应用现在可以说出法语、德语和日语的答案。

不出所料，这些也是谷歌知识图支持的语言。毕竟，要回答这些问题，谷歌需要能够理解他们的意图(或者至少对自己有很高的信心)。那么，在首次推出后，谷歌可能还会瞄准西班牙语、葡萄牙语、俄语和意大利语，所有这些语言都由 Knowledge Graph 提供支持。

如果你会说德语、法语或日语，你现在可以很容易地在手机上尝试这项新功能。只要问它“我在慕尼黑喝咖啡吗？”它会很乐意带你去慕尼黑的咖啡店。

目前，这只能在手机上使用，但是考虑到谷歌已经开始在其几乎所有的服务中建立语音支持，很有可能它也将很快把它带到网络上。
