# 前易贝/Skype 高管让你与 Tokoni 分享故事

[![](img/2d828f6119e754d6ed31cb081eb5d5cd.png)](https://web.archive.org/web/20221207211114/http://www.crunchbase.com/company/tokoni)

想分享吗？两位前 Ebay/Skype 高管创建了一个社交故事分享网站 [Tokoni](https://web.archive.org/web/20221207211114/http://www.tokoni.com/) ，他们希望这个网站能成为网络的虚拟“前台”。该网站于去年 12 月悄然上线，希望创造一个比其他类似的网络社区更温暖、更好的共享环境。

Tokoni 本质上是一个带有社交网络倾向的博客社区。创建个人资料后，会员可以撰写无限数量的故事。每个故事(基本上是一篇博客文章)都可以用关键字标记，并放在“hub”中，hub 实质上是相关故事的组合。故事可以包括嵌入的图像或 YouTube 视频，鼓励其他成员在每个故事的底部留下评论并参与讨论。

[![](img/ebaf11e26b622a8b618b23f3692174d9.png)](https://web.archive.org/web/20221207211114/https://beta.techcrunch.com/wp-content/uploads/2008/05/tokonibig.png)

乍一看，Tokoni 似乎是一个相当不成熟的想法。自从新闻组诞生以来，人们就一直在网上分享个人故事，允许成员按主题将故事分组并不是一个新颖的功能。为什么不用博客呢？

话说回来，令人痛苦的简单想法在过去也曾奏效过(比如 YouTube 和 photobucket)。有可能 Tokoni 将填补那些只想坐下来写作而不必处理博客软件或论坛的用户的空白。社区方面有助于区分网站和博客，允许作者快速找到并链接到其他人发布的故事，而不必在博客世界中筛选。

Tokoni 最令人鼓舞的资产是它的创始人。玛丽·露·宋是易贝的第三名员工，她的丈夫亚历克斯·卡兹姆曾担任过一系列高级职位:易贝工程总监、Skype 总裁、易贝新投资公司的 SVP 和贝宝公司的营销副总裁。该网站还有一个强大的投资者名单，包括易贝公司和一些现任易贝高管。

托科尼不是这一领域的唯一玩家。事实上，有数以千计(如果不是更多的话)的网站致力于故事分享，尽管其中许多都围绕着一个特定的主题或社区。Six Apart 还提供了 Vox T1，这是一个简单的博客服务，提供一些相同的标签和群组功能。Tokoni 拥有令人印象深刻的资质，但除非它能找到一种更好的方式来区分自己，否则它的故事将无人问津。
