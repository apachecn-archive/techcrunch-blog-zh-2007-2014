# 摩托罗拉亲自操作 Verizon Droid RAZR

在机器人仿生传奇(延迟！延迟！延迟！去它的，释放垃圾。)，我觉得我再也不会喜欢另一个摩托罗拉设备了。我想我错了。

我刚刚花了一点时间和刚刚发布的 Droid RAZR 在一起，至少第一眼看上去，它……出奇的棒。你想了解我的第一印象吗？

## 第一印象:

*   正如我们已经习惯了三星最新的显示器一样，三星制造的 Super AMOLED 屏幕令人惊叹。黑色很深，颜色对比鲜明。它仍然没有苹果的视网膜显示器那么高的分辨率，但是这种差异很难被注意到
*   整个设备的美学是…有趣的。不完全符合我的口味(可能因为我太习惯于圆角，而 RAZR 是故意倾斜的)，但一点也不令人讨厌。
*   没有滞后。一点都没有。当然，我们在这里谈论的是一个全新的设备，它几乎没有陷入困境，Android 作为一个整体比一年前要少得多——但尽管我可能会尝试，我不能让任何过渡或动画结巴。
*   Droid RAZR 的新功能是摩托罗拉称之为“智能动作”的功能，它允许你根据位置(基于 WiFi 网络或地理围栏)和时间等触发因素自动完成某些事件。上班？它可以把你的手机铃声从“性感，我知道”改成穿衣服的人*。回家了吗？它会让事情回归你的个人品味，关闭蓝牙，降低处理器速度(以节省电池)，并在手机滑入沙发时提高铃声音量。有第三方应用程序做类似的事情，但摩托罗拉的产品令人惊讶地直观，看起来很好。*
*   苗条。极其苗条。即使在顶部有一大块丑陋的东西(收音机/相机存放的地方)，这东西也是剃刀(哈！)瘦。
*   虽然我无法获得该设备的样本，但我确实使用了相机。快门延迟是最小的，照片之间的时间是微不足道的。
*   整个设备中我最喜欢的部分？奇怪的是，后面。因此，许多制造商(我正在看着你，三星)，已经开始将所有的努力集中在设备的正面。当我实际使用手机时，世界看到的那部分呢..你知道，一部电话？Droid RAZR 的背面几乎完全是凯夫拉尔纤维，手感非常好。这是软接触油漆的纹理，没有增加厚度。
*   电池是不可更换的——对于 Android 设备来说，这有点奇怪。他们牺牲了这一选择，以使设备更纤薄。
