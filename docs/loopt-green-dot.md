# 收购 Loopt 取得了一些成果:绿点推出了移动银行账户 GoBank

红杉(Sequoia)支持的绿点公司(Green Dot)专注于预付费借记卡，并最终于两年前上市，该公司刚刚推出了一个完全可以通过智能手机注册的新银行账户。来自以 4330 万美元收购基于位置的社交网络 Loopt 的团队[是这款名为 GoBank 的产品的幕后推手。](https://web.archive.org/web/20221210043453/https://beta.techcrunch.com/2012/03/09/how-green-dot-will-use-loopt-to-go-after-mobile-payments/)

[今天在有限的测试版中推出](https://web.archive.org/web/20221210043453/https://www.gobank.com/)。就像其他银行账户一样，存款由联邦存款保险公司保险。但是你可以用你的名字、地址、社会保险号、电子邮件地址和电话号码直接在手机上注册。虽然绿点历来专注于在沃尔玛等零售商处销售的预付卡，但 GoBank 可以向该公司介绍一种新的客户群。

“有数亿人拥有银行账户和智能手机，他们中的一些人正在寻找更好的选择，而不是所有的费用，”萨姆·奥尔特曼说，他加入了绿点，并在联合创立 Loopt 后成为移动业务的执行副总裁。绿点目前在预付卡端有 440 万活跃账户。

GoBank 借鉴了另一家红杉支持的公司 Humble Bundle 的经验，对游戏、软件或音乐的套餐进行了付费实验。GoBank 可以让你决定每月的支票费用，最高可达 9 美元。他们不知道最终的中间价格会是多少，但早期的测试表明，人们最终会支付大约 2 至 3 美元。

GoBank 将不会有透支费，罚款，最低余额要求，或要求每月费用。还有其他一些小好处，比如可以方便地给脸书的朋友汇款(他们需要 Paypal 或 GoBank 账户来收款)。

最重要的是，他们在应用程序登录屏幕上使用滑动解锁功能(见下文)可以轻松检查您的余额。

[![iphone_pay](img/f888c4101961d9964161e306d12df98b.png)](https://web.archive.org/web/20221210043453/https://beta.techcrunch.com/2013/01/15/loopt-green-dot/iphone_pay/)

此外，还有其他银行应用程序常见的标准功能，如从应用程序向人们邮寄纸质支票或通过拍摄照片存入支票的能力。然后是针对特定类型事务的通知和警报。

另一个被称为“算命先生”的功能可以帮助消费者决定是否购买符合他们预算的商品。
