# Android 版网飞终于获得了蜂窝平板电脑的支持

如果你的蜂巢驱动的安卓平板电脑已经没什么可做的了，为什么不加载最新更新的网飞应用程序呢？全新的 1.5 更新终于增加了对运行 Android 3.x 的平板电脑的支持，此外还将流媒体视频 love 传播到加拿大和拉丁美洲的平板电脑。

“等等，”你可能会对自己说。"网飞不是已经开发安卓平板电脑有一段时间了吗？"嗯，算是吧。你可以哄你的标签运行网飞提供你经历了挖掘和安装一个倾倒的麻烦。apk，公平地说，对于那些只有几分钟时间可以打发的人来说，这是一个非常琐碎的过程。这肯定是一个不太正式的解决方案，但是嘿——它成功了！

不过，这一次没有这么大惊小怪。它已经在市场上[上线，](https://web.archive.org/web/20230203094202/https://market.android.com/details?id=com.netflix.mediaclient&feature=search_result)，虽然这款应用是免费的，但你当然必须订阅网飞的流媒体月度计划。有些人可能会说网飞的目录质量已经慢慢下降，但我认为每月 7.99 美元是一个很小的代价，只要我想看《巨魔 2》。

**更新:** [Droid-Life](https://web.archive.org/web/20230203094202/http://www.droid-life.com/2011/10/19/netflix-on-android-finally-receives-official-honeycomb-support/) 已经开始使用更新后的应用程序，它似乎没有为所有额外的屏幕空间进行优化。
