蒂姆·奥莱利与克里斯托弗·普尔关于网络身份的未来

“脸书和谷歌做的身份不对，Twitter做的更好，”这是今天 4Chan 创始人克里斯托弗·普尔 Web 2.0峰会演讲中最常出现的重复片段。事实上，当谈到像网上身份这个概念这样无形且不断变化的东西时，真相要比这复杂得多。

因为很难在一个标题中概括未来个人将如何定义自己的多面钻石，我们与互联网先锋蒂姆·奥赖利和克里斯托弗·普尔坐下来探讨在线和离线人物角色的概念，讨论如何为即将到来的透明是默认的世界做准备。

要点:“世界变化如此之快，很难长期保持正确。”
