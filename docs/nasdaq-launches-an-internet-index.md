# 纳斯达克推出互联网指数

全美证券交易商自动报价协会(NASDAQ)证券交易所推出了一个新的仅用于互联网的指数，即 NASDAQ 互联网指数。

新指数将跟踪互联网公司在所有在线活动中的表现，包括电子商务公司、ISP、搜索引擎、虚拟主机和网页设计公司。该指数将衡量该行业公司的表现，快速反映包括雅虎和谷歌在内的股票的走势。

该指数本身确实表明了互联网在金融界的日益成熟和重要性，但尽管如此，纳斯达克也有各种细分市场的指数，包括计算机指数、技术部门指数、前技术部门指数和包括中国和以色列在内的国家的指数；如果说有什么不同的话，那就是这一举措可能不会来得太早。纳斯达克也没有在他们发布的中详细说明指数的构成，但不难猜测谷歌将是指数中的 1000 磅大猩猩，如果谷歌打喷嚏，指数肯定会暴跌。
