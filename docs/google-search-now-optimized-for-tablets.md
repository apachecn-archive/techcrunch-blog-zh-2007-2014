# 谷歌搜索现在针对平板电脑进行了优化

从你的 iPad 或 Android 平板电脑上使用谷歌搜索引擎从来都不难——但体验将会变得更好。今天，谷歌[宣布](https://web.archive.org/web/20230203171051/http://googlemobile.blogspot.com/2011/07/new-google-search-experience-for.html)将推出一个新的、触摸友好的谷歌搜索版本，对平板电脑的外形做了一些关键的改变。

最大的是导航的变化。屏幕左侧的导航条不见了(它有非常密集的菜单，因此很容易被意外点击)。它已经被页面顶部的一系列点击友好菜单所取代，让你在标准结果、视频、图像和其他选项之间跳转。相邻的菜单也可以让您缩小结果的日期范围。

还有很多更小的调整:更大的字体和缩略图，更少的错误，更大的按钮。

谷歌表示，新版本正在推出过程中，所以你可能不会立即看到它。
