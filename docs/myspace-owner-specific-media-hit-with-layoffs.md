# Myspace 所有者专用媒体遭遇裁员

Myspace 的所有者- [Specific Media](https://web.archive.org/web/20230204170700/http://www.specificmedia.com/) 今天解雇了一定比例的员工，我们已经向该公司确认。特定媒体称，这些裁员发生在销售和运营领域，是消除两个业务之间冗余的整合努力的一部分。正如 Specific Media 在一份声明中所说:*公司正在整合力量，以确保员工队伍与其前进方向更加一致。*特定媒体称该公司“不到 8%”的员工被解雇。Specific Media 大约有 750 名员工，因此大约有 50-60 名员工可能被解雇。

除了 Myspace，Specific Media 是一家广告技术公司，旨在帮助推动内容所有者的收视率、品牌参与度和消费者相关性。该公司允许营销人员通过网络、电视、在线视频和移动平台购买数字广告。当然，你可能知道特定媒体公司(连同 T4 贾斯汀汀布莱克)[今年早些时候以大约 3500 万美元从新闻集团手中收购了 Myspace](https://web.archive.org/web/20230204170700/https://techcrunch.com/2011/06/29/myspace-goes-to-specific-media-for-35m-ceo-is-out-press-release/)。

交易的一部分包括解雇 Myspace 当时 400 名员工中的一半。这些裁员是对 6 月份裁员的补充。

Specific Media 和 Myspace 的首席运营官 Chris Vanderhook 今天就这一消息发表了声明:“我们在收购后整合公司方面取得了很大进展，并找到了精简业务的新方法。除了实现协同增效之外，在实现我们的目标方面也有一些真正的创新和重大进展…我们正在顺利执行我们的计划，并将随着我们的继续发展投资于我们的战略。”

不幸的是，特定媒体最近决定缩减推出新 Myspace 的精心计划。陷入困境的社交网络的流量也一直在下降。
