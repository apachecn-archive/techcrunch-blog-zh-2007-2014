# 脸书公布建议兴趣列表

脸书已经根据会员朋友在他们的社交网络中参与的页面向他们推荐页面。但是今天，脸书正在帮助新成员从他们加入网络的那一刻起就策划他们的兴趣。现在，任何新成员都会看到与他们有相似人口统计的其他人也普遍喜欢的页面列表。

脸书说，它将只建议页面的人和组织，实际上看到参与网络。脸书工程师毛自清保证说，新功能是由算法驱动的，所以这些建议没有得到赞助。也不清楚脸书在向新成员提出建议时会考虑哪些人口因素。

不出所料，这个新功能类似于 Twitter 上的一个功能。这在理论上类似于 Twitter 的[建议用户列表](https://web.archive.org/web/20221006175747/https://beta.techcrunch.com/2010/01/21/twitter-sul-suggestions/)，它显示了基于主题的建议用户列表。像脸书的页面建议一样，Twitter 的建议也是基于一种算法，这种算法可以确定用户在微博网络上与谁交流。在脸书的建议列表中，你可以选择喜欢所有的建议

Twitter 的推荐用户名单一直备受争议，因为它保证名单上的任何人都有成千上万的 Twitter 粉丝。脸书的新功能预计也会有同样的效果。今年 1 月，Twitter 实际上改变了它的建议用户列表格式，从仅仅是一个用户列表变成了基于主题的建议。这可能是脸书未来可能会探索的东西，为用户提供更精简(更少随意)的页面建议。
