# “厕所纸德鲁克”就是你想的那样

如果你在 25 年前告诉我，这个来自俄亥俄州哥伦布市的双头年轻人，他晚上仰望星空，沉浸在看不见的宇宙的无边无际的地平线上，有一天他会在他的一生中能够在卫生纸上打印，他会朝你吐口水，然后踢你的狗。他是个相当糟糕的孩子。但这已经发生了:我们现在可以使用现成的组件在卫生纸上打印。这是历史的终结。

打印机使用几个坏掉的 CD 播放器，一个 Arduino 板和旧的卫生纸来打印信息。创造者[马里奥·卢卡斯](https://web.archive.org/web/20230203125554/http://www.mariolukas.de/2011/09/mach-flott-den-schrott-toilettenpapier-drucker/)为德国黑客大赛制造了这台打印机，他指出，你可以将它连接到 RSS 或 Twitter 上，告诉人们你对他们内容的真实想法。

我怀疑我们的一些评论者可能会用这个产品来表达他们对不得不每周 7 天、每天 24 小时阅读免费内容的公开不满。
