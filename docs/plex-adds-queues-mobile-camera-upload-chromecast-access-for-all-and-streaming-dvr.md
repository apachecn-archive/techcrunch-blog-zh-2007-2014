# Plex 增加了队列、移动摄像头上传、面向所有人的 Chromecast 访问和流媒体 DVR TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/03/13/plex-adds-queues-mobile-camera-upload-chromecast-access-for-all-and-streaming-dvr/>

流媒体平台 [Plex](https://web.archive.org/web/20221007041222/http://plex.tv/) 今天为用户进行了一次重大更新，包括向所有用户开放 Chromecast 支持，此前它在 12 月份首次作为付费 Plex Pass 用户的功能推出。对于普通用户来说，这可能是最大的增加，但 Plex Pass 成员获得了一些新的额外待遇，包括从 iOS 设备上传相机，以及共享同步，这让你可以给朋友和家人提供 Plex 媒体服务器的离线同步权限。

一项名为“面向 iOS 和 Plex web 客户端上所有用户的播放队列”的新功能(未来计划在其他平台上推出)还允许用户设置视频和音乐以连续播放，并随机播放不同艺术家的音乐或创建整个电视剧的马拉松式观看会话。这些可以在运行中进行修改和添加，这样观看就永远不会停止，使 Plex 更有可能取代电视，因为它能够在后台运行，而不需要用户进行太多的输入。

至于更新后的 Chromecast 支持，它不仅对所有人免费，还支持 web、iOS 和 Android 上的音乐和照片，并提供内容镜像，以准确显示你在 iOS 或 Android 应用程序上浏览的内容，甚至在 Chromecast 连接的显示器上的菜单中。大部分 1080p 内容现在也无需任何代码转换即可播放。

Plex 推出的摄像头上传功能在上面的视频中有所描述，但基本上是从您的手机到 Plex 媒体服务器的无线同步，iCloud 或 Google Drive 风格，但可以在您的家庭影院 PC 上即时访问您最喜爱的照片。目前，它仅限于 iOS，但 Plex 计划在未来将其扩展到其他平台。

另一个重要的新增功能是 PlayLater 集成，它允许订阅该服务的人在网飞、Hulu、HBO Go 等流媒体服务上录制内容，供以后观看。这将它保存在您的 Plex 媒体服务器库中，有效地让您随时访问通常需要网络连接才能查看的内容。它与 Plex Sync 和 Cloud Sync 一起工作，有效地让您随时随地访问您通过流媒体订阅支付的内容，这是它应该的方式。

最后，Plex 为 Plex Media Server 增加了许多后端功能，使其能够对磁带库进行部分扫描，增加了额外的平台支持和计划任务，以实现更干净的操作。

如果您还没有使用 Plex 来管理您的家庭媒体收藏，那么这些新的更新提供了许多加入的理由。Plex 是少数几家仍有兴趣为用户提供家庭娱乐中心的公司之一，该中心除了流媒体资源外，还可以与他们现有的内容库一起工作，很高兴看到他们仍在积极迭代他们已经建立的内容。
