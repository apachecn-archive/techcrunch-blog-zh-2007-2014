# Dropbox 已经聘请外部专家调查可能的安全漏洞

Dropbox 上的[垃圾邮件攻击](https://web.archive.org/web/20221005190430/https://beta.techcrunch.com/2012/07/17/dropbox-users-targeted-by-spam-possible-address-leak-to-blame)和相关的可能地址泄露可能比我们最初想象的更严重。根据该公司在其[论坛](https://web.archive.org/web/20221005190430/http://forums.dropbox.com/topic.php?page=4&id=64367&replies=111#post-455535)上发布的消息，他们现在已经引入了“外部专家团队”来协助 Dropbox 自己的安全团队进行调查。

截至目前，Dropbox 表示，它尚未收到任何关于用户账户未经授权活动的报告，并采取了一些预防措施来处理这一问题，但拒绝透露更多细节。

[如果你错过了，昨天我们报道了一些 Dropbox 用户开始看到他们的账户被垃圾邮件发送者盯上](https://web.archive.org/web/20221005190430/https://beta.techcrunch.com/2012/07/17/dropbox-users-targeted-by-spam-possible-address-leak-to-blame/)。麻烦的是，许多用户声称他们受到了攻击，尽管他们设置了只用于 Dropbox 的唯一私人电子邮件地址。如果这是真的，这意味着垃圾邮件攻击有可能与地址簿泄露有关，这可能是黑客攻击的结果。许多报告来自 Dropbox 的国际用户，特别是德国、英国和荷兰的用户。今天，论坛网站上有六页关于垃圾邮件的投诉。

虽然最初，用户可能是能够“猜测”他们的电子邮件的随机垃圾邮件的受害者，他们的电脑上的恶意软件感染窃取了他们的凭据，甚至是一个受损的第三方应用程序，但鉴于 Dropbox 承认它已经雇佣了帮手，这些情况现在似乎不太可能发生。

虽然垃圾邮件主要包含与欧洲赌场诈骗有关的消息，而且目前对受影响的用户来说，除了烦恼之外，似乎没有任何其他影响，但它的存在现在似乎指向了一个更明确的可能性，即 Dropbox 实际上被黑客攻击了。

根据 Dropbox 工程师乔·格罗斯(Joe Gross)的说法，Dropbox 昨天经历的中断是“偶然的，不是由任何外部因素或第三方造成的。”

看起来，如果不是因为[收购了 TapEngage](https://web.archive.org/web/20221005190430/https://beta.techcrunch.com/2012/07/17/dropbox-acquires-small-tablet-marketing-company-tapengage/) ，Dropbox 的一周将会非常糟糕。
