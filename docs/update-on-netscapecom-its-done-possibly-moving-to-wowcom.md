# Netscape.com 最新消息:已经完成，可能会搬到 WOW.com。美国在线即将大裁员。

 [](https://web.archive.org/web/20230302095156/https://techcrunch.com/wp-content/uploads/2007/08/netscapedb.png) 我们已经更新了本月早些时候写的关于[可能关闭](https://web.archive.org/web/20230302095156/https://techcrunch.com/2007/08/09/aol-may-kill-their-netscape-digg-clone/)[14 个月大的](https://web.archive.org/web/20230302095156/https://techcrunch.com/2006/06/14/aol-netscape-launches-massive-digg-like-site/)旧 Digg-clone [Netscape](https://web.archive.org/web/20230302095156/http://www.netscape.com/) 的的有争议的帖子。太多的 AOL 高管已经盯上了 Netscape.com 域名，该域名每天带来 300 万左右的页面浏览量。最有可能的情况是——aol.netscape.com 的当前主页成为 Netscape.com 的默认页面，一年前的 digg-clone 迁移到一个新的域名。

我们听说 wow.com 是网景服务的一个潜在登陆点，这个域名以前为 Compuserve 所有，后被 AOL 收购。然而，AOL 对 wow.com 可能有不同的计划，Netscape 门户可能会在别的地方登陆。无论哪种方式，在转换之后，从旧服务中寻找一个链接或模块来保留在 netscape.com 域中。

我们还预计在接下来的六周内会听到 AOL 的大规模裁员，可能会达到 16，000 名员工的 15%。下周，高管们应该会被告知裁员的具体规模，以及他们是针对特定业务群体还是全面裁员。
