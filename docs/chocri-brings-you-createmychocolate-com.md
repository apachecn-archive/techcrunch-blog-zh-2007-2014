# Chocri 为您带来 CreateMyChocolate.com

![](img/2f259fb3ecaced6fe206deadf7fe8bf2.png "createmychocolate")
还有什么比啃你定制的巧克力棒更好的方式来享受你的[定制礼服衬衫](https://web.archive.org/web/20221207014934/http://www.crunchgear.com/2010/06/18/shirtsmyway-design-your-own-dress-shirt/)？我的意思是，你穿的几乎是独一无二的衬衫，为什么你要背负着同样的巧克力棒被所有其他穿着 Abercrombie 衬衫的笨蛋吃掉？Chocri 给你盖了 CreateMyChocolate.com 的。你想要白巧克力配香蕉片和蓝莓吗？当然可以！或者黑巧克力配花生酱夹心的杏仁胡萝卜怎么样？你猜对了！
