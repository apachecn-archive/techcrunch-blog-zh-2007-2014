# 视觉匿名:YouTube 现在可以让你一键模糊人脸

YouTube 今天[推出了](https://web.archive.org/web/20221007221030/http://youtube-global.blogspot.com/2012/07/face-blurring-when-footage-requires.html)一项新功能，用户只需点击一下就可以轻松模糊视频中每个人的脸。YouTube 政策助理阿曼达·康威(Amanda Conway)在今天的一篇博客文章中表示，这可以有多种用途。例如，它可以帮助用户确保他们可以在抗议活动期间分享活动分子的视频[，而不会让他们面临日后引起反响的风险](https://web.archive.org/web/20221007221030/http://www.witness.org/cameras-everywhere/report-2011)。随着 YouTube 成为新闻传播的一个[越来越重要的](https://web.archive.org/web/20221007221030/https://beta.techcrunch.com/2012/07/16/pew-youtube-represents-new-kind-of-visual-journalism/)部分，确保这些视频中被拍摄者的安全变得势在必行。同时，当然这个新功能也可以帮助用户分享他们孩子的视频，而不用向全世界广播他们的脸。康威说，这种新的视觉匿名将允许“人们更广泛地分享个人镜头，并说出他们原本不能说的话。”

谷歌坦率地承认，这个可以在 YouTube 的视频增强工具中找到的新工具并不总是完美的。“这是一项新兴技术，这意味着它有时很难根据角度、光线、障碍物和视频质量来检测人脸，”康威说。例如，某些人脸在某些帧中可能不会模糊。在这些情况下，YouTube 建议用户保持视频隐私。顺便说一下，该工具可能使用了与谷歌街景相同的算法，但我们已经要求谷歌澄清。

目前，这种新工具似乎只允许用户模糊视频中的所有人脸。你不能只是选择性地模糊一些脸，而让其他人原封不动。尽管如此，对于那些想要分享视频而又不想让拍摄的视频面临风险的人来说，这可能会成为一个非常有用的功能。
