# 脸书开始把听、读、看故事变成广告

当你通过脸书 Open Graph 应用程序收听、阅读或观看时，你的一些朋友会通过新闻订阅源、滚动条或时间线了解情况。从本周开始，脸书允许广告商为更多提及他们的故事付费，将这些故事变成一种新形式的赞助故事社交广告。

open Graph action Sponsored Stories 可以进一步推动脸书将所有广告社会化的目标，并通过填充来自脸书的新库存来增加收入，现在允许[在新闻提要](https://web.archive.org/web/20221210071130/https://beta.techcrunch.com/2012/01/10/featured-news-feed-ads/)中出现赞助商故事广告。

脸书告诉我，目前这只是一个小规模的试点项目，只有几个广告商在测试，但它可能会发展成为广告商赞助任何开放图形行动的能力。上面你会看到一个新闻提要中的广告的模型。

此前，广告商只能将用户在自己的页面、应用和本地商业场所采取的行动转化为赞助故事。这排除了大量广告客户，他们的内容通过其他人拥有的应用程序进行互动。例如，唱片公司可以赞助他们艺术家的页面，但不能通过 Spotify 收听这些艺术家的音乐。

现在，乐队可以赞助收听他们的歌曲，电视工作室可以赞助在 Hulu 或网飞上观看他们的剧集。为了迎合更广泛的广告客户，任何在新闻报道中出现的人都可以付费向已经阅读过它的人的朋友展示它的链接。这些行动赞助的故事广告补充了[的 Open Graph 活动广告定位](https://web.archive.org/web/20221210071130/https://beta.techcrunch.com/2012/02/01/action-spec-ad-targeting/)，让广告客户能够接触到已经采取了传统广告的选定行动的用户。

脸书将开放图形应用视为其未来广告业务的灵魂。广告显示朋友的行为，并根据用户自己的行为进行定位，而不是让广告商根据一般的传记数据用不相关的推销来轰炸用户。这创造了一个三赢的局面——对用户来说更相关的广告，对广告商来说更好的转化率，对脸书和它未来的投资者来说更多的钱。
