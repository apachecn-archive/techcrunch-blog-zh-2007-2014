点评:LaCie 1TB加固型XL外置硬盘TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2009/08/06/review-lacie-1tb-rugged-xl-external-hard-drive/


作为LaCie最知名的产品之一，坚固的外置硬盘系列已经存在了很长时间，但直到最近才通过这款1TB设备扩展到3.5英寸硬盘领域。设计大同小异，但显然它要大得多，也更宽敞。但是，与其他1TB驱动器相比，它值得溢价吗？

Big ' n grippy
关于之前的车型，我能说的还不多。除了尺寸之外，坚固的XL在各个方面都与它们非常相似。以前的硬盘是平装本大小，而这个更像是精装本。

这款硬盘与你预期的3.5英寸硬盘一样重，但感觉很平衡，而且因为它是用来躺着而不是站着的，所以给人一种稳定感。它的小脚可以防止它平坐在桌子上，无论你把它放在什么表面上，它都能牢牢抓住。它打开时也有非常健康的光芒——可能有点太健康了。如果你不想做橙色的梦，睡觉前拔掉电源。

更粗犷的气派的，实在的

它的大小是第一个表明，虽然它是一个外部驱动器，但它并不是一个真正无处不在的东西。我的意思是，它可以去任何地方，但像大多数其他3.5英寸驱动器一样，它体积庞大，而且不是总线供电的。那么这有什么意义呢？嗯，这是一个好看的，相对宽敞的驱动器，你真的不用担心你的健怡百事可乐洒在或碰掉桌子。

“崎岖不平”可能有点夸张。可以肯定的是，它比更便宜的驱动器更坚固，更便宜的驱动器根本没有减震功能，可能容易受到飞溅的葡萄酒的影响，但它不是真正意义上的坚固。橡胶缓冲器有所帮助，但铝制外壳很薄(毕竟不能保持热量)，底部的通风口提供了一个弱点(尽管它们被屏蔽以防止碎屑和砂砾进入)。端口也暴露在外，这是我真的没有想到的——一个翻盖或小悬挂端口插头可能看起来很丑，但如果这东西在袋子里呆了很长时间，它会防止很多麻烦。

我没有真正测试减震，因为我想保持这东西完好无损，但它在LaCie的其他驱动器的评论中已经确立，只要它关闭，它就可以很好地承受轻微的撞击。在任何情况下，它都组装得很好，而且橡胶不会很快凹陷。

也就是说，除了受控的办公室环境，我宁愿在任何情况下都有一个这样的人(或他们的小兄弟)和我在一起。咖啡店？这东西。外景？这东西。户外？好吧，也许不完全是这个东西，因为你需要把它插上电源，但是你明白了。

试驾

安装基本上是不存在的，尽管第一次确实弹出了一个向导，可以选择简单地格式化为NTFS，或者在FAT32中有一个单独的分区，作为MAC和Linux机器之间的媒介。方便，但是第二个分区被限制为32GB或更少。我敢肯定这是有原因的，但是如果把它从中间分开，两者都用，那可能会更好。


T7】

我进行了一些非正式的读/写测试，发现声明的USB2上30MB/s和eSATA上80MB/s的最大值至少在正常容差范围内:

  • 一个1GB的文件传输到硬盘大约需要34秒，从硬盘传输出来需要31秒，这与估计的30MB/s完全一致。
  • 1GB的MP3需要的时间要长得多:不到一分钟，平均约为18MB/s，而以20MB/s的速度从驱动器中传输MP3需要50秒
  • 1GB的照片和混合文档(大概2000左右)用了一分多钟，平均14MB/s左右，一分钟平传回来，大概16.5MB/s。

不会让世界着火，但它似乎没有窒息。“exchange”分区倾向于较慢地复制大文件，较快地复制小文件。我没有对eSATA进行广泛的测试，因为，老实说，我直到几分钟前才想起它，但快速复制的几份文件显示它的平均速度为60-70MB/s，从驱动器传输时更高。

写支票前先检查一下自己……从自己身上

现在，这种驱动器的价格大约为140美元，比目前1TB外部驱动器的平均价格高出30美元。额外的花费值得吗？好吧，考虑一下你要用它做什么。如果它会在你的显示器后面安全地呆上一年，那很可能不会；它没有速度优势，而且由于占地面积大，会占用你桌子上更多的空间。但是如果你是移动的，在一个共享的空间工作，或者只是简单的笨拙，拥有一个这样的东西可能是值得的。当然，它的移动性受到它不是公共汽车供电这一事实的阻碍，但如果这也不是一个问题，那就去吧。
