抹去2010年的历史，自杀Web 2.0

更新:貌似脸书屏蔽了该网站的IP地址。

你是否厌倦了在公共场合生活，厌倦了社交网络上演的隐私闹剧，只想在网上结束这一切？现在你可以用 Web 2.0自杀机把石板擦干净了。(警告:这将真正删除您的在线状态，并且不可撤销)。只需输入你的脸书、MySpace、Twitter或LinkedIn的凭据，它就会删除你所有的朋友和消息，并更改你的用户名、密码和照片，这样你就无法重新登录了。

该网站实际上是由鹿特丹的一个新媒体实验室 Moddr 运营的，它执行删除你账户的底层脚本。Web 2.0自杀机器是一个数字化的科沃基恩博士。例如，在脸书，它会一个接一个地删除你所有的朋友，删除你的群组，让你加入它自己的“社交网络自杀者”，并让你留下一些遗言。迄今为止，已有321人利用该网站实施脸书自杀。在Twitter上，它会删除你所有的推文，并删除你关注的所有人和你的追随者。它实际上并没有删除这些帐户，只是让它们休息。

Web 2.0自杀机器运行一个python脚本，该脚本启动一个浏览器会话，并自动断开与这些社交网络的连接(这里有一个视频展示了它如何与Twitter一起工作)。你甚至可以通过一个Flash应用程序来观看虚拟自杀的过程，这个程序显示为一个远程桌面会话。你可以看着你的网络生活一条一条地消失。通过自动脚本接管他人的账户，即使得到许可，也很可能违反这些社交网络的服务条款。
