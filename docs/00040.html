Windows Mobile上的完整YouTube现在成为可能


Windows Mobile粉丝再也不用嫉妒iPhone了。Touch和Tilt即将推出，这两款手机本身都很酷，但现在你可以通过四个简单的步骤观看完整的YouTube、谷歌视频和Veoh。
T3】

第一步:如果你已经安装了核心的掌上媒体播放器，现在卸载它。一旦你的手机不在TCPMP，下载TCPMP的这个版本。为您的手机选择合适的版本，智能手机或掌上电脑。安装在你的手机上。

第二步:从这里下载Flash视频包。将它安装在与核心Pocket Media Player相同的位置(存储卡或手机内存)。

第三步:重启手机。

第四步:看一些视频！

注意:我已经在我的威瑞森摩托罗拉q上测试了这个方法。它应该可以在其他手机/运营商上使用，但我不能保证它可以在所有Windows Mobile手机上使用。
