# O2 用谷歌取代了雅虎在德国的移动搜索。拥有 70%的份额，我怀疑谷歌会在意

![](img/3c6dbe834f810f24d093182885ee250f.png)【德国】雅虎声称它已经“取代”了德国的谷歌搜索。让我们再检查一遍。事实是，雅虎已经与德国 O2 公司达成了一项独家、多年合作伙伴关系，成为移动搜索和服务的首选合作伙伴。谁是前搭档？谷歌。所以这实际上根本不是一回事。

这笔交易意味着那里的 O2 用户现在可以访问雅虎了！O2s 移动门户上的移动优化搜索引擎。用户还可以同步他们的个人电脑和手机主页。雅虎！还将提供 O2 的赞助搜索结果。它还包括其他雅虎网站的链接和内容。

[更新]雅虎告诉我，它不会为这笔交易从头到尾支付 O2，也不是任何合同安排的延续。(2007 年，雅虎与拥有 O2 的 Telefonica 达成协议，让雅虎搜索拉丁美洲的 Telefonica 手机和英国的 O2 手机。O2 德国当时刚刚与谷歌签署了一项协议)。所以 O2 有意识地选择了雅虎而不是谷歌，因为 O2 德国没有参与任何悬而未决的 Telefonica 交易。同样公平地指出，T-Mobile 也或多或少地取代了谷歌在欧洲搜索移动服务的位置，取代了雅虎。

但是雅虎面临的问题是！主要是，德国人自己可能不在乎。尽管其他人认为雅虎在德国没什么意义，而 O2 就用户数量而言是其最小的移动运营商。

正如我们在德国的同事马库斯·高贝尔(T4)所说:“雅虎-O2 的交易只是 O2 移动网站上的一个雅虎搜索框，反正没人使用。人们只需去谷歌或他们的书签网站。”

今年 8 月，comScore 的一项研究显示，谷歌在德国移动搜索市场占有 70%的份额，而雅虎仅占 8.2%。去年，谷歌占据了 85.1%的市场份额。雅虎！占 9.4%。
