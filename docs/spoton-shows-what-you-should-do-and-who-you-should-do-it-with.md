# 斯伯顿展示了你应该做什么，以及你应该和谁一起做

![](img/900e5591b6317cad31fc81ca91e50472.png)需要在西村找一个令人惊艳的哈他瑜伽班？我想在索玛找一家媒体友好的餐厅。新应用 [SpotOn](https://web.archive.org/web/20230202161722/http://www.getspoton.com/) 考虑了现有数据，如 Foursquare 签到、脸书喜欢等，以便为你和你的朋友提供量身定制的选择(无论他们是否下载 SpotOn 应用)。

朋友们也可以向 SpotOn 推荐商家，通过其创新和令人愉快的花瓣界面对他们进行评级，“这是一个以前不存在的完全不同的数据层。

虽然它确实提供用户账户，但 [SpotOn](https://web.archive.org/web/20230202161722/http://www.getspoton.com/) 通过从脸书和 Foursquare 导入你已经存在的社交图，解决了零起点问题。SpotOn maps 会考虑你的哪个朋友去过某个特定的地点，或者谁对某个特定的地点评价很高，以便提供建议。它可以根据签到、脸书信息、与你朋友去的地点的相似性、一天中的时间等，直观地推荐去的地方。

![](img/c17489197d284a708efcdbc1d7b19d54.png "tcdisrupt_web-001-1243")

“简而言之，我们获取你的数字偏好，无论是脸书喜欢，Foursquare 签到还是其他任何东西，并让它们在现实世界中有用，”SpotOn 首席执行官 Gauri Manglik 说。

SpotOn 通过每日交易会员费赚钱，因为它已经向用户显示了推荐，这只是进一步显示附近每日交易的推荐。但是现在公司更关注产品而不是盈利，Manglik 说。

日常交易服务多如牛毛，但有一款应用值得关注，它能根据你在所有移动平台上的社交图谱提供推荐，而不仅仅局限于脸书平台。

该应用程序目前正在引导，你可以在这里的[应用商店找到纽约版本。](https://web.archive.org/web/20230202161722/http://bit.ly/l97RSw)
