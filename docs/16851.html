互联网档案馆寻求捐赠以重建被火灾损坏的扫描中心

互联网档案馆寻求捐赠重建其火灾受损的扫描中心

非营利的互联网档案馆(T1)今天早上(T2)遭遇了重大挫折，其旧金山办公室(T3)发生火灾，造成了大约60万美元的损失。幸运的是，没有人受伤，也没有数据丢失，但试图“普及所有知识”的数字图书馆丢失了高端扫描仪和数字化设备。互联网档案馆正在寻求捐赠，以帮助重建其书籍、缩微胶卷和电影的扫描能力，并允许员工在另一个地点继续数字化工作。你可以在这里捐款。

正如互联网档案馆的粉丝们所知，这个网站不仅仅是一个你可以通过万维网的备份 Wayback Machine 对着你的旧Geocities页面笑(哭)的地方。到目前为止，该组织已经存档了超过10pb(或者说是100亿字节)的信息，包括用巴厘语写的所有内容。它的最新举措包括电视新闻搜索和借阅项目，该项目有超过495，000个存档广播可供借阅DVD，因此你可以核实新闻报道或政治家的声明等内容。

其他互联网存档项目包括开放图书馆，拥有超过200万本电子书。有许多公共领域的经典著作(以及可供借阅的现代书籍)，但开放图书馆最大的好处之一是能够浏览数以千计的古董和古董，如这本1912年的“古怪而美妙的歌谣”和一本1970年的精彩魔术手册。

在这100亿字节的数据中，其他很酷的东西包括: librivox有声图书集；故事片(这里是坎皮禁毒经典大麻疯)；广播节目，比如艾萨克·阿西莫夫的《基金会三部曲》；以及【2,200多本数字化教材。国会图书馆的Prelinger档案馆有60，000个“短暂”的镜头，如红头暴动，一个老式滑稽表演和脱衣舞剪辑的汇编，以及美国外观，面向世纪中期工业、室内和产品设计的粉丝。

在上个月政府关闭期间，Wayback机器还使人们能够在离线时查看重要的政府网站，包括国会图书馆、国家公园服务和联邦通信委员会。

正如互联网档案馆在关于那场火灾的博客文章中指出的:“这一事件提醒我们，数字化和制作副本是获取和保存的好策略。我们在多个地点的互联网档案馆都有这些数据的副本，因此，即使我们的主楼被大火烧毁，我们也不会丢失我们辛辛苦苦收集的惊人内容。”
