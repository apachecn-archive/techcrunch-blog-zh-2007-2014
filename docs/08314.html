NoSQL，是的资助！Couchbase筹集了1400万美元

今年早些时候通过合并CouchOne(由CouchDB的创建者联合创建)和Membase(由memcached背后的关键开发者创建)创建的NoSQL数据库公司 Couchbase ，已经在C轮融资中获得1400万美元。

这轮融资由Ignition Partners牵头，该公司现有的支持者Accel Partners、Mayfield Fund和North Bridge Venture Partners参与了融资。

此外，该公司说，Couchbase还额外预留了100万美元用于“战略客户和合作伙伴”的投资。

Couchbase为构建Web和移动应用程序提供数据库解决方案。就在上周，该公司发布了 Couchbase Server 2.0 ，这是一个分布式的、面向文档的数据库管理系统，也是该公司在合并后发布的第一个集成产品。

Couchbase表示，它将使用新的资金来加速NoSQL数据库技术的交付，在企业市场上发展公司，并进行国际扩张。

这轮融资使该公司的资本总额达到3000万美元。
