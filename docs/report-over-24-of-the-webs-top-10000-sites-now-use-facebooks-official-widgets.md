# 报道:超过 24%的排名前 10，000 的网站现在使用脸书的官方插件

根据网站监控服务 [Pingdom](https://web.archive.org/web/20221206121658/http://pingdom.com/) 的一项新研究，全球排名前 10，000 的网站中有 24.3%(据 Alexa 报道)在其主页上有某种形式的官方脸书整合。这已经是一个相当可观的数字了，但是一旦你把脸书的基本链接也算进去，这个数字会上升到 49.3%。该公司的官方[按钮](https://web.archive.org/web/20221206121658/http://developers.facebook.com/docs/reference/plugins/like/)出现在 7.3%的网站上。

正如 Pingdom 所说，脸书显然正在成为“网络 DNA 的一部分”，并且远远领先于其竞争对手。不过，值得注意的是，整合脸书的网站数量在过去一年中似乎保持相对稳定。例如，几乎就在一年前，bright edge[报告](https://web.archive.org/web/20221206121658/http://www.brightedge.com/2011-07-08-brightedge-socialshare)称，它在网络排名前 10，000 的网站中，发现了约 47%的脸书整合。

## Twitter、Google+和 LinkedIn

在 Pingdom 的研究中， [Twitter](https://web.archive.org/web/20221206121658/http://twitter.com/) 只出现在排名前 10，000 的 10%的主页上。大约 4.3%的网站使用 Twitter 的官方分享按钮。尽管这些数字相对较低，但值得注意的是，41.7%的网站有 Twitter 链接——这个数字与脸书没有太大区别。

至于谷歌的社交网络 [Google+](https://web.archive.org/web/20221206121658/http://plus.google.com/) ，取决于你问的是谁，它要么是活跃的，要么是死的——Pingdrom 报告说，它在主页集成方面实际上做得相当好。在这项研究中，超过 13%的网站在其主页上使用谷歌的+1 按钮，12.3%的网站包含 plus.google.com 的链接。然而，正如 Pingdom 正确指出的那样，许多网站所有者整合 Google+来提高他们的搜索引擎排名，“这可能会夸大数字，因为从网站所有者的角度来看，他们不一定与 Google+相关，而是被视为影响 SERP 的一种手段。”

另一家社交网站 LinkedIn 远远落后于竞争对手。在这里研究的网站中，只有 0.6%在其主页上整合了这一功能，0.3%使用了社交网络的官方分享按钮。即使包括基本链接，整合 LinkedIn 的网站总数也只有 3.9%。

算上各种链接和官方小部件，下面是 Pingdom 的最终计数:

> 1.  **脸书** : 49.3%
> 2.  **推特** : 41.7%
> 3.  谷歌+ : 21.5%
> 4.  领英网:3.9%
