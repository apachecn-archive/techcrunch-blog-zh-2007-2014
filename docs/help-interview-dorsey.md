# 今晚帮我采访杰克·多西

这些日子杰克·多西很受欢迎。Twitter 的发明者[现在回到](https://web.archive.org/web/20221207062832/https://beta.techcrunch.com/2011/03/28/jack-dorsey-takes-over-product-again-at-twitter-as-executive-chairman/)担任产品负责人，同时仍然担任他的另一家初创公司 Square 的首席执行官。今晚，多尔西和我将一起在雷切尔·斯特恩的哥伦比亚商学院的课程上发表演讲，主题是[社交媒体和企业家精神](https://web.archive.org/web/20221207062832/http://bit.ly/arEqye)。斯特恩也是纽约市的首席数字官，所以这应该是一场社交媒体盛宴。我们将于美国东部时间今晚 6 点左右在 TechCrunch 上直播这次演讲。

这实际上更像是一次采访，有来自学生和广大观众的问题。如果你对多尔西有问题，请在下面的评论中留下，或者在活动期间发推特给我 [@erickschonfeld](https://web.archive.org/web/20221207062832/http://twitter.com/#!/erickschonfeld) 。我将询问多尔西他在 Twitter 的新角色，该产品将何去何从，他将如何平衡他在 Square 的角色，以及他对设计社交产品的总体看法。

讲座的主题是“社会分裂”Twitter 和 Square 分别是媒体/通信和支付行业的颠覆性创业公司。我们将深入了解每家公司如何打破目前的秩序，并在此过程中创造新的价值。Square 允许任何人接受信用卡，降低了创业的门槛。Twitter 降低了沟通的障碍，开辟了一种全新的信息消费方式。我想知道他从这两次经历中学到了什么，以及 Twitter 的早期艰难岁月如何帮助他推出 Square，迄今几乎没有出现任何问题。我还想知道他对像 Intuit [这样的大玩家在 Square](https://web.archive.org/web/20221207062832/https://beta.techcrunch.com/2011/03/29/intuit-tablet-gopayment-square/) 之后冲进来有什么看法。
