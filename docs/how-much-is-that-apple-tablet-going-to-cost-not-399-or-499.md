# 苹果平板电脑的价格是多少？不是 399 美元或者 499 美元。

在苹果[季度收益](https://web.archive.org/web/20221007003851/http://www.beta.techcrunch.com/2009/07/21/apple-sold-twice-as-many-iphones-as-macs-last-quarter/)电话会议的问答环节中，你总能从冰冷的手指中窥探到一些有用的数据。今天也不例外。

像往常一样，出现了一个关于苹果进入“低端”笔记本电脑市场的问题，这是“上网本”市场的一种花哨说法。苹果首席运营官·蒂姆·库克再次重申了苹果的立场，即“*我们的目标不是制造最多的电脑。是为了打造最好的。*“他的意思是，苹果没有兴趣涉足低端市场，这当然也是一个低利润的市场，这意味着苹果可观的利润将受到打击。但措辞决定一切，库克再次没有完全排除苹果制造新型便携式计算设备的可能性。

他说，不管价位是多少，如果苹果觉得它能制造出与其品牌相称的产品，它就会这么做。但他也特别指出了他认为正在生产垃圾产品的两个价格点，并暗示这样的价格点将继续生产垃圾产品:399 美元和 499 美元。

因此，如果苹果正在开发某种平板电脑(或者是大屏幕 iPod touch)，价格可能会超过 500 美元。对苹果观察人士来说，这是显而易见的，但听听库克或多或少这样说还是很重要的。

随后的一个关于上网本的问题是，苹果公司是否预见到了一种新的大屏幕移动设备的未来？库克开玩笑说，虽然他从来不想排除未来的任何事情，但他也从来不打算回答关于新产品的问题。他接着说，他相信大多数购买便携式电脑的客户都想要一台功能齐全的笔记本电脑。而且很多上网本速度很慢，运行的是旧的操作系统。自然，他没有提到 [Chrome OS](https://web.archive.org/web/20221007003851/http://www.beta.techcrunch.com/2009/07/07/google-drops-a-nuclear-bomb-on-microsoft-and-its-made-of-chrome/) 。

这似乎进一步证明，无论苹果制造什么样的大型触摸屏设备，它都不会是上网本，而是苹果希望再次启动一个新型市场的产品。谁知道这是否行得通。让我们先看看这个东西。
