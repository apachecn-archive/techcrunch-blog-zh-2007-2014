# 苹果将发放股息，在 2012 年第四季度启动 100 亿美元的股票回购计划

苹果坐拥大量现金——近 1000 亿美元——他们已经安排了一个凌晨电话会议来讨论他们计划如何处理这些现金。然而，在电话会议之前，关于他们财务阴谋的消息不胫而走，因为 Cupertino 公司刚刚发布消息称，他们打算向股东发放股息，并在今年晚些时候启动股票回购计划。

苹果首席执行官蒂姆·库克表示:“我们已经使用一些现金对我们的业务进行了重大投资，包括增加研发、收购、开设新的零售店、战略预付款和供应链中的资本支出，以及建设我们的基础设施。”“即使有这些投资，我们也能保持战略机会的资金储备，并有充足的现金来运营我们的业务。”

苹果公司预计在未来三年内为这些项目支付 450 亿美元，尽管蒂姆·库克指出，该公司将定期重新审视他们的股息和股票回购努力。

苹果计划在 2012 年第四季度的某个时候向股东支付每股 2.65 美元的季度股息，对苹果来说，这将从今年 7 月开始。如果你想知道，苹果公司上一次支付股息[是在 1995 年](https://web.archive.org/web/20221210031751/http://investor.apple.com/dividends.cfm)——此举为股东净赚了每股 0.12 美元。

除此之外，苹果董事会还批准该公司在未来三年内回购 100 亿美元的股票。回购计划定于 2012 年 9 月开始，旨在帮助抵消“未来员工股权授予和员工股票购买计划的稀释影响”
