# MyLikes 给 YouTube 带来了付费视频广告

![](img/fd9e400532d272ccb3ef1b2ba5357e27.png)

这是不可避免的。首先我们有了[按帖子付费](https://web.archive.org/web/20221006233146/https://beta.techcrunch.com/2006/10/29/payperpost-is-now-officially-absurd/)，然后[按推文付费](https://web.archive.org/web/20221006233146/https://beta.techcrunch.com/2009/09/22/ad-ly-brings-sponsored-tweets-from-celebrities/)，现在我们有了按视频付费。随着 YouTube 上的名人开始吸引更多、更忠诚的观众，他们越来越被营销人员视为有效的广告渠道。MyLikes ，一个已经将有影响力的博客和推特与广告商相匹配的社交营销网络，现在正转向 YouTube。例如，博客作者 Chris Pirillo，他的[locker nome YouTube 频道](https://web.archive.org/web/20221006233146/http://www.youtube.com/user/lockergnome)有 120，000 名订户，他为 iPhone 应用 Siri 制作了一个[赞助视频](https://web.archive.org/web/20221006233146/http://www.youtube.com/watch?v=OBZZ_Snri5k)，展示了他在做虚拟个人助理的演示。

赞助 YouTube 视频并不新鲜。品牌一直在手工制作的活动中取得成功(事实上，今天早些时候我写了由通用电气和 Howcast 支持的[赞助视频广告，这些广告总共被观看了 800 多万次)。但是 MyLikes 采用了更自动化的方法。毕竟，它是由包括 AdSense 前顶级工程师在内的前谷歌员工创立的。](https://web.archive.org/web/20221006233146/https://beta.techcrunch.com/2010/04/26/ge-youtube-howcast-hot-chicks/)

Youtubers 用户需要[申请](https://web.archive.org/web/20221006233146/http://mylikes.com/video)才能进入该程序。如果你的 YouTube 频道有超过 10，000 名订阅者，这会很有帮助。每个 YouTube 视频制作者都会在 MyLikes 上创建一个个人资料，该资料与他或她的频道相关的类别相链接。他们设定每个视频的价格，并根据他们拥有的订户数量以及每个视频的平均浏览量和评论数等因素获得影响力得分。然后广告客户被匹配到视频创作者，然后他们选择是否要用他们自己的话代言广告客户的产品。这些视频应该被识别为赞助信息。

随着越来越多的人花时间在社交媒体上，营销人员也会被吸引到那里。我们已经看到了新的商业模式，比如 OpenSky，它试图将博客作者转变为直接的营销者。YouTube 是下一个。

MyLikes 最近刚刚[从其他前谷歌员工那里筹集了一轮种子资金](https://web.archive.org/web/20221006233146/https://beta.techcrunch.com/2010/04/13/mylikes-raises-630000-seed-round-all-from-ex-googlers/)。它还宣布刚刚聘请了另一位前谷歌员工大卫·斯卡科(David Scacco)担任首席收入官。斯卡科是谷歌的第一位广告销售主管。
