# IBM 发布 Lotus Notes，Domino 版本 8 | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/18/ibm-releases-lotus-notes-domino-version-8/>

![0.jpg](img/ca7b483001f71d1d633aa5d9c056fe4c.png)

我很少使用 Lotus Notes 和 Domino，因为我只在办公室使用过 MS Outlook，但我相信你们中的很多人都使用，所以我想，当你知道第 8 版终于发布时，你会很兴奋。这一过程历时两年，但最新版本是在开源 Eclipse 平台上构建的，这意味着 mashups 可以将旧应用程序与新应用程序链接起来。IBM Lotus 软件公司总经理迈克尔·罗丁对最新版本这样说，

> 我们的客户称 Lotus Notes 8 为“未来的桌面”，因为它将他们所有的协作工具整合到一个屏幕上。协作正在取代电子邮件，成为我们日常使用的杀手级应用，无论你的虚拟桌面是在桌子上、车里、手上还是耳朵里。Notes 和 Domino 8 是通向 Web 2.0 世界的个人门户。

IM 也包含在 Notes 和 ODF 支持中，允许用户导入和导出 MS Office 应用程序和 RSS 阅读器。定价:Lotus Notes 8 软件的 SRP 起价为每个客户端 101 美元。对于基于浏览器的替代方案，IBM Domino Web Access 客户机的价格是每个用户 73 美元。IBM Lotus Domino 服务器软件的 SRP 起价为每价值单位 14.75 美元。面向中小型企业的 Lotus Domino Express 解决方案，包括客户机和服务器，每个用户的 SRP 起价为 99 美元。
