# 路径:我们已经删除了所有地址簿数据

看起来 Path 已经注意到了投资者迈克尔·阿灵顿的话。

昨天，这家初创公司面临着一场重大的隐私反弹，当时有消息透露，该社交应用程序正在[上传用户的地址簿数据](https://web.archive.org/web/20221209215028/http://www.techmeme.com/120207/p43#a120207p43)，而实际上并没有告诉用户。联合创始人兼首席执行官戴夫·莫林对此深表歉意，而且关于[这是多么大的一件事](https://web.archive.org/web/20221209215028/https://beta.techcrunch.com/2012/02/08/et-tu-path/)(特别是因为这种做法符合苹果的政策)，有很多争论，但阿灵顿有一个更简单的建议:“[只要销毁所有数据](https://web.archive.org/web/20221209215028/http://uncrunched.com/2012/02/08/hey-path-just-nuke-all-the-data/)。”

在一篇新的博客文章中，Morin 说这正是 Path 所做的。首先，他解释说该应用程序将数据用于无害的目的(当用户的一个联系人加入 Path 时提醒用户)。Path 的一个新版本要求你在使用这些数据之前选择加入。但是莫兰也说:

> 我们认为，在分享您的个人信息时，您应该拥有控制权。我们也相信行动胜于语言。因此，作为我们承诺保护您隐私的明确信号，我们已经从我们的服务器上删除了所有用户上传的联系信息。您的信任对我们很重要，我们希望您能完全掌控自己的信息。

正如阿林顿指出的，Path 不会花太多时间从想要分享数据的用户那里再次收集数据。尽管如此，这似乎是一个有效的象征性姿态，表明“是的，我们搞砸了，我们想做出补偿。”
