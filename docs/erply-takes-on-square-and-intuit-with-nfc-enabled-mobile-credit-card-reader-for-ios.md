# Erply 与 Square 和 Intuit 竞争，推出支持 NFC 的 iOS 移动信用卡读卡器

开发零售库存管理软件的公司 Erply 今天推出了一款适用于 iOS 设备的新型移动信用卡读卡器，与 Square、Intuit 和 VeriFone 展开竞争。Erply 的信用卡设备连接到 iPad 或 iPhone，并将客户的加密信用卡数据传输到自己的销售点和库存管理软件套件，以处理支付。

通过 Erply 的读卡器，零售商将设备连接到 iPad 或 iPhone 的充电端口，并与公司的 POS 软件进行通信，以处理交易并记录库存销售。信用卡信息实际上是在设备内部加密的，确保交易安全可靠。

对于读者来说，连接库存管理的好处是，它为零售商提供了一个全面的解决方案，而不是必须使用不同的软件。这在有大量库存的零售环境中尤其有效。此外，创始人 Kristian Hiiemaa 表示，该阅读器支持移动交易的 NFC 功能。

Erply 的信用卡读卡器售价 50 美元，1.9%的交易费略低于 Square、VeriFone 和 Intuit 的费用。Square [最近降低了 0.15 美元的交易费](https://web.archive.org/web/20230203101034/https://techcrunch.com/2011/02/22/mobile-payments-startup-square-ups-the-ante-drops-transaction-fee-for-businesses/)，向商户收取 2.75%的固定费用。Intuit [做出了类似的举动](https://web.archive.org/web/20230203101034/https://techcrunch.com/2011/05/01/intuits-gopayment-cuts-transaction-fees-pricing-now-more-in-line-with-square/)。但是 Square 的阅读器和软件是免费的，而 Erply 的软件每月收费从 70 美元到 1000 美元不等。Erply 还可以利用其现有的 20，000 名零售商用户，他们使用该公司的软件进行库存管理。

当然，需要注意的是，Intuit 的 GoPayment 阅读器集成了流行的会计软件 Quickbooks，Square 也通过其最新的 iPad 应用程序开发了更全面的销售点系统。Square 在小企业中发展尤其迅速。

Erply 实际上是 Seedcamp 的冠军，并从红点和指数风险投资公司、马腾·米科斯、扎克·乌洛克(两人都曾是 MySQL 的成员)、肯尼·范·赞特(网络安全管理软件产品)、艾登·森库特、大卫·麦克卢尔和加速器集团那里筹集了[200 万美元](https://web.archive.org/web/20230203101034/https://techcrunch.com/2010/03/22/erply-the-next-skype-of-business-software/)。Erply 的移动信用卡读卡器现在可用于 iPad，并将在三个月内用于 iPhone。
