# 谷歌分析赶上实时，推出企业级

谷歌分析(Google Analytics)是谷歌提供的一套非常受欢迎的工具，用于帮助网站管理员跟踪他们的网站流量，该工具在本周得到了很大的提升:该服务已经推出了一个实时仪表板，随着用户事件的发生而更新。

这是一件大事，因为它使谷歌更加符合流行的实时分析产品，如 Chartbeat，允许你跟踪你的网站上目前有多少人，以及每篇文章在过去几分钟内获得了多少流量。在今天的实时发布之前，谷歌分析报告通常有三个小时以上的延迟窗口(通常更长)，这意味着这种跟踪是不可能的。

在宣布这一消息的博客文章中，谷歌强调了你现在可以如何使用分析来衡量各种社交渠道的影响，如 Twitter 和脸书——你将能够快速判断你的定制分析查询是否如预期那样工作，而不必等待 24 小时才能看到结果。为了查看新的实时功能，您需要激活位于屏幕右上角的“分析”中的“新版本”链接。

除了实时，今天分析团队还[宣布了](https://web.archive.org/web/20230204221210/http://analytics.blogspot.com/2011/09/introducing-google-analytics-premium.html)一个高级层，面向那些想要比谷歌免费服务提供更多支持(和数据点跟踪)的企业。谷歌表示，公司将能够以未指明的价格获得电话支持、SLA 保证和增加的数据限制。
