Windows Live Spaces过渡到WordPress创建了100万个新博客

去年9月在TechCrunch Disrupt上，微软宣布全部700万Windows Live Spaces博客将转移到WordPress.com。结果这个数字被夸大了，一封随后的内部邮件称转变后的博客的真实数量为30万。

已经转移到WordPress的Windows Live Spaces博客数量现在已经“超过50万”,另外还有50万新的WordPress博客是由Windows Live用户创建的。因此，作为合作的结果，新创建的WordPress博客总数达到100万。

到2011年3月，当Windows Live Spaces日志不再受支持时，总数不会达到700万。但不管怎样，这些博客中有很多已经死了。100万个新的WordPress博客是不容小觑的。自交易宣布以来，WordPress.com新注册的博客数量已经翻了一倍多，从每月40万增加到每月90万。
