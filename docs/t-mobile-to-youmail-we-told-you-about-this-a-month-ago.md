# T-Mobile 给你的邮件:我们一个月前就告诉你了

昨天发生了一件奇怪的事。拥有数百万下载量的可视语音邮件应用 YouMail 被从安卓市场撤出。困惑之余，YouMail 联系了谷歌，询问这是什么交易，结果却听说 T-Mobile 已经要求撤下这款应用。T-Mobile 向谷歌抱怨“不利的网络中断”，这完全归咎于 YouMail。

根据 [YouMail 的博客文章](https://web.archive.org/web/20221007170025/http://blog.youmail.com/post/13928968522/the-incredible-disappearing-youmail-app)，该公司没有从 T-Mobile 那里听到任何关于这个问题的消息。但是 [GigaOm](https://web.archive.org/web/20221007170025/http://gigaom.com/mobile/t-mobile-we-asked-youmail-to-fix-its-app-last-month/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+jkOnTheRun+%28GigaOM%3A+Mobile%29&utm_content=Google+Reader) 报道说他们已经收到了 T-Mobile 的一份声明，描述了一幅不同的画面。

> 我们在 11 月初联系了 YouMail，要求他们解决影响我们客户体验的应用程序问题。我们正在与 YouMail 联系，他们正在努力解决这些问题。一旦他们这样做了，我们将很高兴再次支持他们的申请。

所以我们正处在一个可爱的“他说，她说”的小争论中。好消息是 T-Mo 和 YouMail 已经在谈判了，所以这个问题应该会很快解决。一旦我们了解到更多情况，我们会通知你的。
