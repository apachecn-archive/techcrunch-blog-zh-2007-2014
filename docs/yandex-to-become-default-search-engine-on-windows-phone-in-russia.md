# Yandex 将成为俄罗斯 Windows Phone 的默认搜索引擎

俄罗斯在线搜索和广告巨头 Yandex ，已经[与微软、诺基亚、HTC 和三星](https://web.archive.org/web/20230407015105/http://company.yandex.com/press_center/press_releases/2011/2011-11-25.xml)联手，成为俄罗斯最新——以及即将推出的——Windows Phone 智能手机的默认搜索引擎。

Yandex 的搜索引擎目前安装在三星的 mnia W 上，该产品已经在俄罗斯销售，也安装在诺基亚基于 Windows Phone 的 Lumia 800 和 Lumia 710 智能手机上，这两款手机将于 12 月上市。

继去年秋天推出基于 Windows Phone 的 Titan、Mozart 和 Radar 智能手机之后，HTC 还计划发布一个更新版本，将 Yandex 作为其手机的默认搜索引擎。

上个月，Yandex 宣布成为 T4 bada 智能手机 T5 的主要搜索提供商。

Yandex 指出，它已经为 Windows Phone 开发了一些应用，包括地图服务。地图和在线市场 [Yandex。市场](https://web.archive.org/web/20230407015105/http://market.yandex.ru/)，并且它将在未来为该平台和其他移动操作系统开发更多的应用程序。

根据 LiveInternet 的数据，2011 年第三季度，Yandex 在俄罗斯搜索市场的平均份额为 62.7%，远远超过谷歌。今年早些时候，该公司在纳斯达克首次公开募股中筹集了超过 14 亿美元，最近报告今年第三季度 RUR 的收入为 52 亿美元(1.619 亿美元)，比 2010 年第三季度增长了 65%。

除了俄罗斯，Yandex 还在乌克兰、哈萨克斯坦、白俄罗斯和土耳其开展业务。
