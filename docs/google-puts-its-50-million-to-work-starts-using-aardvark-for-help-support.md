# 谷歌投入 5000 万美元，开始使用 Aardvark 寻求帮助支持

两个月前，谷歌[以大约 5000 万美元收购了](https://web.archive.org/web/20221206145348/https://beta.techcrunch.com/2010/02/11/google-acquires-aardvark-for-50-million/)有前途的社交搜索初创公司 [Aardvark](https://web.archive.org/web/20221206145348/http://www.vark.com/) 。这项服务允许你提出问题，并从其他对当前主题有所了解的用户那里立即得到回复。通常谷歌要花好几个月(或更长时间)才开始让收购生效，但我们已经遇到了谷歌首次在野外使用 Aardvark 的情况:帮助支持。

访问 YouTube [帮助页面](https://web.archive.org/web/20221206145348/http://www.google.com/support/youtube/?hl=en)，你会注意到一个显著的链接，提示用户“试试土豚！”。点击那个链接，你就会被带到土豚主页。谷歌在谷歌[工具栏](https://web.archive.org/web/20221206145348/http://www.google.com/support/toolbar/?hl=en)的帮助页面上发布了一条相同的信息。不，这不完全是深度整合，但它仍然是谷歌的一个有趣的举动，谷歌从来没有以其客户支持而闻名。

大多数谷歌服务提供帮助指南，但几乎从来没有电话号码或电子邮件地址来联系实际的人(这并不奇怪，因为大多数这些服务都是免费的)。Aardvark 可以作为谷歌的一个很好的妥协，增加了人情味，而不需要建立支持人员的电话银行。Aardvark 还没有出现在所有的 Google 帮助页面上，但是我想如果它表现良好的话，我们会看到更多这样的链接。

我决定用一个基本的(但可能是常见的)YouTube 支持问题来测试 Aardvark:“YouTube 支持什么文件格式，WMV 能工作吗？”。不到五分钟，我就有了答案，还附有 YouTube 上相关帮助文章的链接。
