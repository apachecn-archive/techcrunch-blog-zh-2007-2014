# 脸书为 iPad 准备好视网膜

脸书今天早上发布了一个更新，为其 [iPad 应用](https://web.archive.org/web/20221007121527/http://itunes.apple.com/us/app/facebook/id284882215?mt=8)增加了对新 iPad 视网膜显示屏的支持。根据 iTunes 的发布说明，新 iPad 的用户现在将看到“一个清晰的高分辨率界面。”此次更新(版本 4.1.1)也带来了其他错误修复，以及对其他语言的支持。

上周，当[有人在 iTunes App Store 的“新的和值得注意的”部分发现了即将推出的应用的预告片](https://web.archive.org/web/20221007121527/http://pocketnow.com/iphone/new-ipad-retina-optimized-facebook-app-coming)时，这款应用的视网膜版就已经被暗示了。当时显示的信息是“为新 iPad 增强”，大概意味着它将很快增加对视网膜图形的支持。

果然，它就是这么做的。

[![](img/a30ef2369030fa6bfc2ea2b2cd01ca2e.png "mzl.tbbiblzj.480x480-75")](https://web.archive.org/web/20221007121527/https://beta.techcrunch.com/wp-content/uploads/2012/04/mzl-tbbiblzj-480x480-75.jpg)

该更新还带来了许多其他错误修复，包括以下内容:

*   你可以在聊天中离线
*   每个人都会看到正确的个人资料图片
*   你的朋友列表总是包括你所有的朋友
*   在照片集中，你的名字会正确显示
*   喜欢脸书页面的人的照片可以正确加载
*   您的好友请求通知只有在您有请求时才会亮起

此外，脸书用户现在可以在更新的应用程序中将其语言更改为捷克语、丹麦语、希腊语、印度尼西亚语、马来西亚语、挪威语(博克马尔语)、葡萄牙语(葡萄牙语)或泰语。

然而，尽管新的 iPad 用户可能会很高兴看到他们最常用的应用程序之一更新为支持 Retina，但也许一个更受期待的功能(或者更确切地说是功能*请求*)是支持脸书时间轴。作为自 News Feed 以来对脸书用户界面最大胆的修改，时间轴提供了一种显示和消费脸书内容的新方式。鉴于 iPad 的屏幕尺寸，将该界面移植到 iPad 上似乎不成问题。

更新后的脸书应用程序[现在可以在 iTunes 中使用](https://web.archive.org/web/20221007121527/http://itunes.apple.com/us/app/facebook/id284882215?mt=8)。
