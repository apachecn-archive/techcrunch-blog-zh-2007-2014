# 马克扎克伯格对脸书的隐私控制:“我们只是错过了标记”

![](img/2a126d000a87539d5a63a04464c9a046.png)今天早上，在一封发表在[华盛顿邮报](https://web.archive.org/web/20230214044200/http://www.washingtonpost.com/wp-dyn/content/article/2010/05/23/AR2010052303828.html)上的信中，脸书首席执行官马克·扎克伯格回应了自 f8 发布声明以来针对该网站的一连串[批评](https://web.archive.org/web/20230214044200/https://techcrunch.com/2010/04/27/senators-call-out-facebook-on-instant-personalization-other-privacy-issues/)。关键信息:扎克伯格说，关于其隐私控制，“(脸书)刚刚错过了目标。”

这封信没有特别的歉意——你不会发现像“对不起”、“错误”或“错误”这样的词。相反，这更像是承认脸书听到了批评，并将做出回应。很快，脸书将推出一套新的简化隐私控制和一种“关闭所有第三方服务”的简单方法。以下是扎克伯格信中的一些关键引文:

> 我们听到了反馈。需要有一种更简单的方法来控制你的信息。在接下来的几周里，我们将添加更简单易用的隐私控制。我们还将为您提供一种关闭所有第三方服务的简单方法。我们正在努力使这些变化尽快实现。

> 我们最近听到的最重要的消息是，人们希望更容易地控制他们的信息。简而言之，你们中的许多人认为我们的控制太复杂了。我们的目的是给你许多粒度控制；但这可能不是你们中许多人想要的。我们刚刚错过了目标。

> 我们还听说有些人不了解他们的个人信息是如何被使用的，并担心这些信息以他们不希望的方式被共享。我想现在澄清一下。许多人选择向所有人公开他们的一些信息，这样他们认识的人就可以在脸书上找到他们。我们已经提供了控制措施来限制这些信息的可见性，我们打算进一步加强这些措施。

脸书的任何人都不会对此感到意外。几个月来，我一直在写关于迫在眉睫的反弹的文章。不是因为我特别有先见之明，而是因为这对于任何一个留心的人来说都是显而易见的。脸书赌了一把，结果不太好。至少目前是这样——预计他们会在大约六个月后再次挑战极限。希望用户能更好地了解自己的隐私，以及在那个时候如何控制它。
