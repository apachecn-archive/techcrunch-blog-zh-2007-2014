# Seesmic 的黑莓应用程序现在比其他所有应用程序都要甜

当 [Seesmic](https://web.archive.org/web/20221207210624/http://seesmic.com/) [在 11 月推出【Android 和黑莓设备的 Twitter 客户端时，我们写道，这两个应用可能是每个移动设备上最强的产品。Seesmic](https://web.archive.org/web/20221207210624/https://beta.techcrunch.com/2009/11/20/seesmic-ventures-into-mobile-with-powerful-new-apps-for-android-and-blackberry/) [刚刚更新了其 Android 应用程序](https://web.archive.org/web/20221207210624/https://beta.techcrunch.com/2010/02/04/seesmic-android-update/)，使其更接近 iPhone 的 Twitter 应用程序。今天，Seesmic 推出了功能强大的新版黑莓应用程序。

新版本包括支持多账户、 [Ping.fm](https://web.archive.org/web/20221207210624/http://ping.fm/) 集成(Seesmic [今年早些时候收购了 Ping.fm](https://web.archive.org/web/20221207210624/https://beta.techcrunch.com/2010/01/04/seesmic-acquires-ping-fm/) )以及选择照片上传大小的能力。因此，如果你有不止一个 Twitter 账户，你可以设置 Seesmic 来记住你所有的账户，并轻松地在它们之间切换。您还可以同时在不同的帐户之间交叉发布消息。该应用程序允许您设置默认帐户，该帐户将在您每次打开黑莓 Seesmic 时生成。使用新功能，您最多可以添加十个帐户。

Ping.gm 集成允许你同时向 50 个社交网络发布你的更新。为了使用该功能，您需要创建一个 Ping.fm 帐户。用于黑莓的 Seesmic 还包括通过 Tweetphoto now 分享图片的功能。你仍然可以用 yFrog 发布照片，这是客户端推出的照片应用程序，但 Tweetphoto 现在是默认的。现在你还可以选择你想上传的照片的尺寸。你可以选择小、中、大图片。

Sessmic 在 12 月对其黑莓客户端进行了[更新](https://web.archive.org/web/20221207210624/http://www.mobilecrunch.com/2009/12/14/seesmic-gets-updated-pushed-to-the-blackberry-app-world/)，但多账户和 Ping.fm 支持对该客户端来说是巨大的。这一最新更新应该使 Seesmic 的客户端成为功能最丰富的应用程序。当然，黑莓已经推出了其 Twitter 应用的私人测试版，该应用缺少了一些关键的功能，并且得到的评价平平。
