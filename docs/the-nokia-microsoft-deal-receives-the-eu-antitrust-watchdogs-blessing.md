# 诺基亚和微软的交易得到了欧盟反垄断监管机构的批准

今天，欧盟签署了微软收购诺基亚手机业务的协议，该交易价值超过 70 亿美元。最近，这笔交易得到了诺基亚股东和美国政府的批准。

微软收购诺基亚硬件业务的道路上已经没有什么障碍了。该公司的 Windows Phone 项目基本上已经成为一场独角戏，诺基亚的 Lumia 系列手机占其总销量的 90%以上。

诺基亚卖出这么多手机对公司来说是好事，但与此同时，微软正在慢慢失去对自己平台的控制。购买诺基亚的手机资产对微软来说是防御性的，也是进攻性的；它重新获得了控制权，并可以利用其更大的财政资源比诺基亚更努力地推进这条线。

根据欧盟的新闻稿，该交易没有“引起任何竞争问题，特别是因为双方的活动之间只有适度的重叠，微软的移动操作系统、移动应用程序和企业邮件服务器软件与诺基亚的智能移动设备之间的联系不太可能导致竞争对手被排除在市场之外。"

这笔交易继续快速进行，这意味着前诺基亚首席执行官史蒂芬·埃洛普将登陆微软，他被认为是微软首席执行官职位的竞争者。
