# 官方消息:Hulu 将于周三开放

[![](img/72d3d8fe5135f6c52f4bc1c1a1d0a8ac.png)](https://web.archive.org/web/20230213200203/http://www.hulu.com/)

没有人可以说他们没有[预见到这一天的到来](https://web.archive.org/web/20230213200203/https://techcrunch.com/2008/03/08/here-comes-hulu-mania-again/) : [Hulu](https://web.archive.org/web/20230213200203/http://www.hulu.com/) 将最终在本周对公众开放，准确地说是在周三，是的，这是官方的。

该公司已经进行了 4.5 个月的私下测试。如果你还没有设法得到四处流传的许多邀请中的一个，你可能在想“大事”。但是正如我在我最初的评论中所表达的，Hulu 是一个出色的网络服务，尽管它有很多缺点(没有下载，有限的节目和电影，没有国际访问等等)。

Hulu 还宣布增加来自华纳兄弟电视集团、狮门影业、NBA、NHL 和其他 20 家内容提供商(包括洋葱新闻网)的新内容！).
