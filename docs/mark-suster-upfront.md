# 马克·苏斯特(Mark Suster)谈到了他的 2 亿美元新基金计划，以及对洛杉矶创业生态系统的高期望

马克·苏斯特的风险投资公司刚刚完成了一笔 2 亿美元的新基金，它也有了一个全新的名字:[前期风险投资](https://web.archive.org/web/20221207210450/http://www.upfront.com/)。今天在 [500 Startups PreMoney 大会](https://web.archive.org/web/20221207210450/http://premoney.co/)上，我有机会与他坐下来谈论公司的新品牌，以及他对新基金的计划和对洛杉矶创业生态系统的继续支持。

根据 Suster 的说法，公司更名有几个原因。首先，GRP 合作伙伴陷入 RAS 综合征——即冗余缩写综合征。GRP Partners 实际上代表全球零售合作伙伴，嗯，合作伙伴。

此外，这个名字并没有反映出该公司实际代表的任何东西。GRP 的最初创始人是零售业务的早期投资者，如好市多、星巴克、迪克体育用品和其他实际的零售公司。不过，该基金本身一直非常关注科技股。

在寻找新的品牌名称时，该公司希望能展示其独特的优势和观点。这包括“坦诚”其商业实践，以及创业公司可以从与它的合作中得到什么。

“我们生活在一个透明的时代，人们想知道你是谁，你代表什么。所以我们想，我们想要透明，我们想要公开信息，这样你就知道我们是如何运作的。我们有强烈的观点，我们愿意与人辩论，”苏斯特说。

除了这个新品牌，advanced Ventures 还筹集了 2 亿美元的新基金，并将在圣莫尼卡开设一个令人敬畏的新办公室。它计划继续在洛杉矶大量投资。根据 Suster 的说法，这是一个将继续增长的市场，因为越来越多的第二次和第三次创业者开始建立新的企业，并向其他本地创业公司投入天使资金。
