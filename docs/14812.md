# 埃隆·马斯克展示了他的证据，证明《纽约时报》的特斯拉 Model S 试驾是“假的”

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/02/14/elon-musk-lays-out-his-evidence-that-new-york-times-tesla-model-s-test-drive-was-fake/amp/?guccounter=1&guce_referrer=aHR0cHM6Ly93d3cuZ29vZ2xlLmNvbS8&guce_referrer_sig=AQAAAGNfdcwljGAitDsrMK7hcX96JMYeawTVOhWMTGfH4Oe-LR5YLH0Q2xLMhXZeolHeKknTYqB1rHhkdmGFM0CpnGjbaZfGWuXKoD08VfNNFFJFbvaWaJBLBwM15u7yuJYu0VavAKQ0yX1pur8VRXXjMAp_72hPv-GtxA-j10rWZS8N>

特斯拉汽车公司首席执行官兼创始人埃隆·马斯克绝对不是试图耍花招的最佳人选。这位有远见的企业家在 Twitter 上引起了一阵窃笑，他在本周早些时候声称《纽约时报》记者约翰·布罗德捏造了特斯拉 Models 汽车在寒冷天气下的行驶里程的细节，导致了他所谓的“假”文章。马斯克承诺提供证据，现在他已经通过特斯拉官方博客[提供了证据。](https://web.archive.org/web/20230330195034/http://www.teslamotors.com/blog/most-peculiar-test-drive)

为了与他在最初的推文中对评论错误的简要描述保持一致，马斯克列出了车辆日志(特斯拉在[遇到 Top Gear](https://web.archive.org/web/20230330195034/https://techcrunch.com/2011/03/30/tesla-vs-top-gear/) 问题后的标准做法，该日志戏剧性地显示了实际上并不存在的故障)如何显示布罗德为他的文章驾驶的汽车充电不当，进行了一次计划外的附带旅行，基本上似乎是为了失败而设置的。

马斯克在一些要点中分析了问题所在，但基本上布罗德的汽车从未完全耗尽电量；被充电到一个他知道不足以到达目的地的程度；实际上超出了它的预期范围；经过充电站，充电站可以帮助它完成旅程；在曼哈顿绕了很长一段路，这不在最初的旅行计划中。

其他问题加剧了报道中的欺骗，包括气候控制设置与 Broder 在文章中关于他对车内加热所做的声明背道而驰(当他说他调低温度时，他调高了温度)。较小的细节不一定是最重要的，但马斯克在测试车辆的日志中记录了甚至这些较小的矛盾，这一事实有助于描绘出一个作家似乎从一开始就公然瞄准特斯拉的画面。

马斯克说，布罗德修改了测试的细节和条件，以帮助符合他之前存在的观点，这要归功于布罗德在 2012 年发表的一篇[文章中的引用。布罗德基本上试图贬低充满电动汽车的未来的阳光形象，声称“电动汽车的状况令人沮丧，是夸大的期望、技术失败、高成本和敌对政治气候的受害者”。“公平地说，在那篇文章中，布罗德还继续给了电动汽车的支持者足够的空间，甚至让《谁杀死了电动汽车》*背后的纪录片制作人克里斯·佩恩来做最后的决定。*，以潘恩暗示石油和天然气行业是阻碍汽车运输电动化未来的幕后黑手而告终。](https://web.archive.org/web/20230330195034/http://www.nytimes.com/2012/03/25/sunday-review/the-electric-car-unplugged.html?pagewanted=all&_r=4&)

但总的来说，马斯克的证据相当确凿，尤其是有 Model S 本身的可靠数据支持。他最后呼吁 NYT 对这篇文章及其写作展开调查，在这样的攻击之后，我想 NYT 必须这样做，以便能够拿出一个令人满意的回应。
