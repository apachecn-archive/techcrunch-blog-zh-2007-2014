Mobile Roadie现在为Android生态系统创建应用

我们是 Mobile Roadie、一家帮助开发iPhone应用的初创公司的忠实粉丝。但我们有一个抱怨，那就是移动Roadie仅限于iPhone平台。今天，我们的愿望实现了，因为Mobile Roadie正在为Android手机推出功能。

Mobile Roadie平台的美妙之处在于，它提供了一个非常简单的几乎自动化的系统来构建应用程序，并在短短一周内将它们发布到苹果的应用商店。这家初创公司于今年早些时候推出，为其他会议、活动和场所以及音乐家、运动员、政治家和其他名人开发移动应用程序。这些应用程序可以为用户提供新闻、音乐、直播和录制的视频、照片、活动列表等内容。这些应用还集成了YouTube、Brightcove、Flickr、Twitpic、Ustream、Topspin、Google News、RSS、Twitter和脸书。

Mobile Roadie平台的另一个好处是，它的CMS允许用户同时更新iPhone和Android应用程序。使用推送通知，客户可以发送出现在用户屏幕上的警报，将地理定位消息缩小到一英里半径。随着谷歌Android支持的推出，Mobile Roadie将为阿什顿·库彻、多莉·帕顿和麦当娜的iPhone和Android应用程序提供支持。我们听说泰勒·斯威夫特也将很快推出应用程序。

Mobile Roadie还为法国企业家和Seesmic创始人Loic Le Meur和他的妻子Geraldine组织的最重要的欧洲技术会议开发了iPhone官方应用。这个应用在会议上大受欢迎。Mobile Roadie最近还与兰登书屋达成协议，为作者提供iPhone应用。

毫无疑问，Mobile Roadie针对Android平台的功能将使其平台对用户更具吸引力。通过一个界面同时管理iPhone和Android应用程序的能力是一个引人注目的功能。凭借Mobile Roadie在过去一年的成功记录，我们将继续期待在不久的将来，这家自举创业公司会产生更多的创新。
