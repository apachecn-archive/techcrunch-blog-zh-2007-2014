# 创业只是出口吗？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2010/06/12/is-entrepreneurship-just-about-the-exit/>

戴维·帕克和埃里克·巴恩在他们的初创公司 *[击败了 GMAT](https://web.archive.org/web/20230203034551/http://www.beatthegmat.com/)* 挣得比他们在企业界挣得还多。企业的每一分钱利润都直接进入他们的银行账户。他们喜欢做自己的老板；成为销售、市场营销、客户支持、计算机编程和平面设计方面的专家；对帮助学生获得商学院的录取感到满意；并且很感激他们可以花时间*做事情*而不是*讨论事情——因为他们不用对任何人负责*。他们会问，为什么要卖掉自己的公司，回到像 Intuit 或麦肯锡这样的公司工作？**

瑞安·西特经营着一家名为 Picclick.com 的网站，他也有同样的感觉。他的视觉销售网站每月吸引 30 万独立访客，并为易贝和 Etsy 卖家创造了数百万的产品销售额，为他带来了六位数的收入。他在家工作，想花多少时间就花多少时间陪他的两个小孩和妻子。Ryan 珍惜做任何他想做的事情的自由——比如尝试新的网站创意。他最不想做的事情就是筹集资金或与更大的公司合并。“当你得到资助时，你就成了奴隶，拥有大量员工只是一种痛苦”，Ryan 说。

商学院教授的以及硅谷的传统智慧是，一家科技初创公司必须有一个明确的退出战略，并将其所有精力集中在实现这一最终目标上。换句话说，创业就是关于退出——财富需要通过将公司上市或出售给更大的参与者来积累。在任何天使或风险投资人投资一家企业之前，他会评估其退出策略，并盘问企业家，以确保他/她甚至没有考虑建立一家“生活方式企业”(最大的罪过)。创投资本家在他们的投资合同中包括“需求注册权”这样的条款。这些条款赋予了迫使公司上市的权利，以防创始人屈服于靠公司利润生存的诱惑而不退出。所以增长变得比利润更重要；目的地——出口——变得比旅程更重要；员工只是达到目的的一种手段。

但是一定要这样吗？技术企业家和战略顾问 Sramana Mitra [向](https://web.archive.org/web/20230203034551/http://www.forbes.com/2010/05/27/online-retailing-software-technology-entrepreneurs.html)提出了一个很好的问题:“如果退出的想法从等式中移除……如果投资者和企业家同意一种不同的模式——分享红利的模式——会怎么样？”她说，数学很简单:如果一笔 50 万美元的投资帮助建立了一家年收入 1000 万美元、年利润 20%的公司——这是完全可能的——天使投资人将获得数百万美元的股息。企业家将建立他们可以长期培育的公司，并“享受”。

这对我认识的许多企业家来说很有意义。是的，我的一些朋友雄心勃勃，需要*尝试*建立一个谷歌或 Zynga。但大多数人会很高兴建立一个生活方式的企业，支付账单，让他们学习，成长，并“享受”创业之旅。他们宁愿缓慢而稳定地积累财富，也不愿生活在恐惧之中，害怕投资者迫使他们引入另一位首席执行官，或者让公司走上歧途。[这篇对 Zappos 公司的谢家华的采访揭示了即使是最有能力的公司创始人也面临的压力。](https://web.archive.org/web/20230203034551/https://techcrunch.com/2010/06/07/tony-hsieh-zappos/)

Sramana 已经启动了一个名为 [1M/1M](https://web.archive.org/web/20230203034551/http://www.sramanamitra.com/1m1m/) 的项目，旨在通过向 100 万名创业者教授自举、客户验证和定位的基本知识，帮助他们实现 100 万美元的收入。这是通过一系列在线战略圆桌会议来完成的，这是一种真人秀，她在每场会议上指导五位企业家，多达一千人在听。斯拉马纳试图说服天使投资者更密切地关注那些能够盈利并产生股息，但永远不会成为下一个谷歌的公司

所有“令人愉快”的生意都必须是小型的吗，就像斯拉马纳正在培育的那样？当然不是。以 [Hannon Hill](https://web.archive.org/web/20230203034551/http://www.hannonhill.com/) 为例，它开发了一个叫做 Cascade Server 的网络内容管理系统。2000 年末，大卫·卡明斯在杜克大学(Duke)读本科时创办了这家公司，并把它和它的衍生产品发展到收入超过 600 万美元。它雇用了 51 名员工。他说，自成立以来，他的公司每年都在增长；他没有联合创始人，没有投资人，也不想退出。大卫将大部分利润投资于增长，因为他看到了建立一个亿美元企业的潜力。他开始建立一家拥有“令人敬畏的企业文化”的公司，给自己支付“不错的”薪水，从事有趣的项目。

还有北卡罗来纳州卡里的 SAS 研究所。这家商业智能和客户关系管理软件公司由 Jim Goodnight 和 John Sall 于 1976 年创立，现已发展成为一家高利润、23 亿美元的企业。几年前，当吉姆带我参观他的校园时，他对他的公司积累的稀有艺术品以及 SAS 为员工提供的福利感到非常自豪——比如两个现场托儿中心；员工保健中心；健康计划，包括一个 58，000 平方英尺的娱乐和健身中心；针对员工父母的“老年人关爱”计划；每个休息室都有冷饮柜和小吃。难怪 SAS 总是被评为美国最适合工作的地方之一。吉姆不需要向任何人道歉，因为他把员工的需求和福利放在了利润之前。反过来，他的员工回报以努力工作和奉献精神:SAS 的离职率是业内最低的。

我可能对生活方式企业的赞美有点过头了。建立一个有利可图的企业从来都不容易，而且有很多风险。如果没有精明的投资者的关注，一家公司可能会很快被淘汰。技术变化非常快，你可以在一夜之间从繁荣走向萧条。如果戴维·帕克和埃里克·巴恩从竞争对手 GMAT 考试准备公司如卡普兰或普林斯顿评论得到可观的收购报价，我会建议他们卖掉。有了几百万美元的银行存款，他们就可以奢侈地开始新的事业，而不用担心万一失败，他们的家庭会遭受损失。当他们有经济保障时，他们可以遵循吉姆·古德奈特的道路。

我也不想打击企业家的雄心。我认为他们应该向月球进发。我想说的是，如果你用正确的价值观建立一家公司，你更有可能成功，并享受这个过程，而不是只关注退出——用一点点雄心换取幸福是可以的。
