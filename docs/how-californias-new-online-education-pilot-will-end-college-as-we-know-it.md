# 正如我们所知，加州的在线教育试点将如何结束大学生活

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/01/15/how-californias-new-online-education-pilot-will-end-college-as-we-know-it/>

今天，[世界上最大的大学系统](https://web.archive.org/web/20230320210704/http://www.calstate.edu/PA/2011Facts/)，加州州立大学系统宣布在其一个校区试行 150 美元的低年级在线课程——这一举措意味着我们所知的高等教育的终结。较低级别的课程是许多兼职院系(尤其是人文学科)的财务支柱。作为一个在加州大学教授大型课程的人，我可以向读者保证，我的工作很容易实现自动化。随着预算紧张的学校将学生集中在网上，大学的大部分——广阔的校园和大型演讲厅——将崩溃成鬼城。

*【注意:在本文的最后，我提供了这一切是如何崩溃的时间表】*

传统上，成群结队毫无准备的青少年被塞进低级课程和补习课程的无名教室。“他们从高中毕业，但他们不能通过我们的小学数学和英语分级测试，”圣何塞州立大学新举措试点校区的教务长兼副校长埃伦·n·朱恩说。50%以上的入学学生不符合基本要求。

但是，即使学费在十年内增长了近三倍，班级规模也在不断扩大，但不断减少的助教和课程还是没能满足不断增加的大学新生的需求。

事实上，尽管花费了数十亿美元和学术思想来教育年轻人，但只有不到一半(48%)的人毕业于 SJSU。

受够了，杰里·布朗州长已经对流行的在线课程平台 [Udacity](https://web.archive.org/web/20230320210704/http://www.crunchbase.com/company/udacity) 给予了支持，与圣何塞州立大学合作提供超低成本的在线低年级和补习班。代数和统计课程的试点将仅限于 300 名学生，一半来自 SJSU，一半来自高中和社区大学。

加州教师协会还没有对试点采取立场，但至少有一名教职员工怀疑这一宣布是为了颠覆反对意见。社会学教授普雷斯顿·鲁迪(Preston Rudy)说:“我个人认为，在大多数教师不在校园的时候宣布这一消息并不是偶然的，但我没有证据证明这一点。我对 Udacity 的了解不足以采取任何立场，但总的来说，我知道该大学很关心如果他们上网，谁将教授课程，谁有控制权，以及他们是否会成为大学员工。”

尽管教师们担心在线课程的质量，但事实是，我们的教育体系主要是为了测试死记硬背而设计的，它是按比例构建的，不依赖于教师的互动。教育部 2009 年对研究的回顾发现，“平均而言，在线学习全部或部分课程的学生比通过传统面对面教学学习相同课程的学生表现更好。”

[最近](https://web.archive.org/web/20230320210704/http://www.nytimes.com/2013/01/15/technology/california-to-give-web-courses-a-big-trial.html?pagewanted=all&_r=0)，麻省理工学院和哈佛大学联合在线教育项目 EdX 的试点，发现将 SJSU 的课程与世界级的在线讲座相结合，获得 C 或更低的学生数量减少了 31%。

换句话说，[电脑可以——并且已经——成功取代了老师](https://web.archive.org/web/20230320210704/http://articles.cnn.com/2011-06-09/opinion/computers.replace.teachers_1_schools-change-online-teachers-cash-strapped-school-districts?_s=PM:OPINION)。

为了提高保留率，国家科学基金会资助的项目将提供一系列指导和监测服务，包括鼓励学生在特定任务中停滞不前的电子邮件。

在线课程并不是全新的，但很难低估加州高等教育系统的强大。前加州大学校长理查德·阿特金森(Richard Atkinson)威胁取消 SAT 的入学要求后，仅在威胁一年后，大学委员会就匆忙修改了整个国家的考试。不管加州的教育系统如何，这个国家也是如此。

**如果我必须预测这次试播的后果会如何，下面是我的时间表**:

1.  试点成功，扩展到更多的大学和班级
2.  兼职教师被解雇，更多的社区大学被关闭，课外大学服务被关闭，人文和艺术部门因招生不足而解散(理科招生增加——耶！？)
3.  一旦硕士生和博士生意识到没有教学工作，研究生项目就会枯竭。更少的研究生意味着更少的助教，因此，更少的班级
4.  基于能力的衡量标准开始发现，在线学生的表现与校园学生不相上下，甚至更好。主要的认证州立大学系统提供完全在线的大学学位，然后关闭越来越多的大学校园
5.  随着世界各地的大学向成功率最高的班级靠拢，一些常春藤联盟大学开始控制大部分在线内容
6.  在不久的将来，大学校园里的学习将回归其精英根基，由研究人员和专家教师亲自指导的学生比例会小得多

Udacity 项目的影响是玫瑰还是荆棘取决于你对高等教育投入了多少价值。如果你属于投资者彼得·泰尔的阵营，他以资助年轻企业家跳过大学而闻名，那么这是一个好消息，因为它可能会引导更多的年轻人进入职业生涯，为他们节省数千美元的债务，并扩大教育机会。然而，如果你落入伯克利研究员 Vivek Wadwha 的阵营，那么年轻人教育中最重要的一部分，面对面的大学互动，将会失去。不管怎样，变革正在到来。
