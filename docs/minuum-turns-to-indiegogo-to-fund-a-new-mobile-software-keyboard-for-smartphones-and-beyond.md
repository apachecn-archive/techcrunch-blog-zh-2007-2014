# Minuum 向 Indiegogo 寻求资助，为智能手机和其他设备开发新的移动软件键盘

【YouTube http://www.youtube.com/watch?v=niV2KCkKmRw？feature = player _ embedded & w = 640 & h = 360]

总部位于多伦多的初创公司 [Whirlscape](https://web.archive.org/web/20221006103405/http://minuum.com/) 认为，拿起原本为打字机设计的键盘，将其数字化，然后就此收工，是实现最佳智能手机文本输入体验的错误方式。这就是为什么这家由多伦多大学的教职员工和研究人员组成的年轻公司创造了一种新型虚拟键盘 Minuum，它现在正寻求通过 [Indiegogo](https://web.archive.org/web/20221006103405/http://www.indiegogo.com/projects/the-minuum-keyboard-project?website_name=minuum) 获得资金。

Minuum 取消了标准 QWERTY 键盘的传统三行字母和空格键布局，代之以单行字母和预测打字算法，这有助于回收大部分可用屏幕空间，同时据说保留了精确输入功能。它旨在保持 QWERTY 的顺序，以简化从标准软件键盘的过渡，但也包括强大的自动校正算法，以确保您可以非常草率地输入文本，仍然可以获得可用的结果。想一想[类似弗莱克西](https://web.archive.org/web/20221006103405/https://beta.techcrunch.com/2013/03/11/fleksy-gesture-keyboard-demos-leap-motion-controller-compatibility-introduces-ios-sdk-for-third-party-apps/ "Fleksy Gesture Keyboard Demos Leap Motion Controller Compatibility, Introduces iOS SDK For Third-Party Apps")的东西，但是用一种非常不同的方法。

这种差异的一部分是 Minuum 能够将其创新的打字模式扩展到智能手机和平板电脑以外的硬件设备。该公司正计划为希望在自己的应用程序中使用它的开发人员开发一款 Android 应用程序和一款 iOS SDK，但 Minuum 也有很多机会几乎可以在任何地方打字。在演示视频中，我们看到项目创建者展示了它通过像微软 Kinect 或谷歌眼镜中使用的相机捕捉技术工作，只需在测试者的手臂上用钢笔写下一个匆忙潦草的键盘。它还可以与设计用于检测精细运动的硬件一起工作，如即将推出的 [MYO 臂带](https://web.archive.org/web/20221006103405/https://beta.techcrunch.com/2013/02/28/myo-armband-pre-orders/ "YC-Backed MYO Armband Attracts 10,000 Pre-Orders In 2 Days, Which Adds Up To $1.5M In Sales")。

“人们一直在研究的所有不同类型的技术都集中在触摸屏上，每个人都开始在他们不同类型的键盘上构建滑动功能，”Whirlscape 联合创始人兼首席执行官威尔·沃姆斯利在接受采访时解释道。“我认为小型设备的未来不会局限于触摸屏。考虑到我们拥有和它们一样智能的设备，超越触摸屏的思维很有意义。”

然而，首先，Whirlscape 将专注于在 2014 年初准备好其原型 iOS 和 Android 软件(公共测试版将于今年晚些时候推出)，随后将推出硬件 SDK。Whirlscape 已经从多伦多大学的早期加速器项目 UTEST 和 T2 火星创新项目 T3 获得了一些种子资金，但它认为众筹是在准备公开发布时产生轰动的最佳方式。Walmsley 解释说，Indiegogo 有助于它与用户建立直接关系，并创建一个早期采用者池，从中吸引测试人员进行初始私人测试。

Walmsley 说，最终目标是直接与原始设备制造商合作，将 Minuum 推向尽可能多的用户。如果它能兑现其灵活性和跨平台功能的承诺，一旦推出，吸引追求者应该不成问题。
