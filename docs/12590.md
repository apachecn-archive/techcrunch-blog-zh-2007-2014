# 脸书推出验证账户和假名 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/15/facebook-verified-accounts-alternate-names/amp/>

脸书是一项基于真实姓名和真实身份的服务，将于明天开始允许知名公众人物验证他们的账户，然后选择显示他们喜欢的昵称，而不是他们的出生名。那些拥有验证账户的人将在脸书的“订阅者”建议中获得更显著的位置。

经过验证的账户并没有背离脸书的政策，即用户用真实姓名注册，因为出生名仍然会显示在用户关于页面的个人资料中。相反，这是一种确保人们不会订阅冒名顶替者的公开更新的方式。它还将武装脸书，与 Twitter 争夺对兴趣图谱的控制权。

脸书向我证实，明天，一些拥有很多用户的用户将通过他们的个人资料被告知可以选择验证他们的身份。没有办法志愿被验证，你必须被选中。这些用户将被提示提交一张政府颁发的带照片的身份证的图像，该图像在验证后被删除。他们还可以选择输入一个“备用名”,这个名字可以用来通过搜索找到他们，并且可以显示在他们的真实姓名旁边，用括号括起来，或者作为替换。

例如，Stefani Germanotta，又名 Lady Gaga，可以使用验证帐户来验证她是著名的 Stefani Germanotta，将她的姓名显示为“Stefani Germanotta (Lady Gaga)”，或者显示为简单的“Lady Gaga”，Stefani Germanotta 出现在她的个人资料的“关于”页面上。

[<amp-img class="alignright size-full wp-image-497836 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="Verified Accounts Alternate Name" src="https://web.archive.org/web/20230123113037im_/https://techcrunch.com/wp-content/uploads/2012/02/verified-accounts-alternate-name.png" alt="" srcset="https://web.archive.org/web/20230123113037im_/https://techcrunch.com/wp-content/uploads/2012/02/verified-accounts-alternate-name.png 306w, https://web.archive.org/web/20230123113037im_/https://techcrunch.com/wp-content/uploads/2012/02/verified-accounts-alternate-name.png?resize=150,141 150w, https://web.archive.org/web/20230123113037im_/https://techcrunch.com/wp-content/uploads/2012/02/verified-accounts-alternate-name.png?resize=300,282 300w, https://web.archive.org/web/20230123113037im_/https://techcrunch.com/wp-content/uploads/2012/02/verified-accounts-alternate-name.png?resize=50,47 50w" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230123113037/https://techcrunch.com/wp-content/uploads/2012/02/verified-accounts-alternate-name.png)Lady Gaga 随后会更频繁地出现在[脸书的订阅建议](https://web.archive.org/web/20230123113037/http://www.facebook.com/subscriptions/suggestions/)。此外，那些喜欢她的音乐的人不会不小心订阅了另一个同名同姓的人，或者用那个名字注册了一个账户，希望得到杜平的订阅者并向他们发送垃圾邮件。

脸书将手动批准替代名称，以确保他们是真正的艺名，笔名，或以其他方式为申请人建立的绰号。它需要小心避免错误地验证虚假账户，就像 Twitter 在邓文迪默多克事件中所做的那样。

与 Twitter 或 [Google+的验证程序](https://web.archive.org/web/20230123113037/https://techcrunch.com/2011/08/20/google-rolls-out-verified-profiles-still-struggles-with-real-name-policy/)不同，经过验证的脸书账户不会显示任何他们已经被批准的徽章或标志，但这可能会削弱该程序引导用户的能力。这个新系统不应该与经验证的开发人员计划或二级电子邮件验证混淆。

以前，想要显示昵称的人必须改变他们的正式名字。传统的做法是把它放在名字后面的括号里，但是脸书有一个很长的不可接受单词的黑名单，太多未经授权的名字转换会导致一个人的名字变得不可更改。

脸书确实允许用户在他们的个人资料和搜索中用另一种语言显示他们的婚前姓或真实姓名，但不像一个经过验证的别名那样在网站上显示。为了清理这项服务并加强其实名制政策，2010 年末[脸书不再允许](https://web.archive.org/web/20230123113037/http://www.insidefacebook.com/2010/11/29/denies-access-invalid-names/)在名字中使用特殊字符。

三个月前，[脸书推出了不对称关注功能](https://web.archive.org/web/20230123113037/https://techcrunch.com/2011/09/14/facebook-launches-twitter-like-subscriptions-lets-you-share-with-unlimited-users/)，用于订阅你感兴趣但不是朋友的人的公开更新。目前还没有关于冒名顶替者垃圾邮件的广泛报道，但这将防止问题的发生，而不会拒绝拥有著名名字的普通人的帐户。

[订阅对 Twitter 的长期增长是一个巨大的威胁](https://web.archive.org/web/20230123113037/https://techcrunch.com/2011/12/23/facebook-subscribers-follower/),因为它提供了一个对主流“足够好”的兴趣图订阅解决方案。我听说脸书有团队在积极招募名人来开发这个功能，经过验证的账户将会吸引更多的用户
