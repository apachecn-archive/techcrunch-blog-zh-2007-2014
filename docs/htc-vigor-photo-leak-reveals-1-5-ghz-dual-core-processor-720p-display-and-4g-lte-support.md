# HTC Vigor 照片泄露显示 1.5 GHz 双核处理器，720p 显示屏和 4G LTE 支持

自从我们第一次听到 HTC Vigor 的风声，事情就变得模糊了。我不仅仅是指泄露的照片——我们已经在一些不同的功能上来来回回，比如[显示分辨率](https://web.archive.org/web/20230204222140/https://techcrunch.com/2011/09/16/htc-vigor-renamed-incredible-hd-poised-for-october-release/?icid=tc_gadgets_art&category=gadgets)，或者它是否会是一款 [Droid 品牌的手机](https://web.archive.org/web/20230204222140/https://techcrunch.com/2011/09/23/htc-vigor-packs-beats-audio-lte-but-no-droid-branding/)。但是对于一部如此神秘的手机来说，我们最近的泄露确实泄露了秘密。

承蒙 [Droid-Life](https://web.archive.org/web/20230204222140/http://www.droid-life.com/2011/09/26/htc-vigor-poses-for-photo-shoot-4-3-720p-hd-screen-2mp-front-camera-4g-lte-and-1-5ghz-dual-core-processor-confirmed/) 的好意，我们现在有了一些半透明的图片，以及确实是 HTC Vigor 的硬件规格的完整列表。根据图片中显示的“硬件信息”页面，Vigor 将运行 1.5GHz 双核处理器和 Android 2.3.4 Gingerbread。HTC 用户界面的最新版本 Sense 3.5 也将被打包在新手机中。正如我们所希望的那样，我们也看到了对威瑞森 4G LTE 支持的确认。

除此之外(我知道你对此很好奇)，Vigor 将拥有 4.3 英寸 720p 高清显示屏。我们还将看到一个 800 万像素的后置摄像头和一个 200 万像素的前置摄像头，用于视频聊天。加上 Beats Audio 的可能性，活力似乎是一个完整的包。

但除了令人惊叹的规格，我们还仔细观察了这款手机的外观，我不得不说我并没有被打动。那种有纹理的背面肯定会很快变脏。

但不管你喜不喜欢，离开威瑞森时支付不到 300 美元的可能性微乎其微。与 Vigor 规格最接近的 LTE 手机将是摩托罗拉 Droid Bionic，合约价为 299 美元。如果感兴趣，就开始存钱吧。下面是完整的图库。
