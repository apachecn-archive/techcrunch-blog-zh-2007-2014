# 阿什顿·库彻媒体将于下周预告布拉女孩

![](img/c6137fa1fa60f66a154af1de4c8c27e2.png)

关于由 [Katalyst Media](https://web.archive.org/web/20221005235839/http://www.crunchbase.com/company/katalyst-films) (由[阿什顿·库彻](https://web.archive.org/web/20221005235839/http://www.crunchbase.com/person/ashton-kutcher)和[杰森·戈德堡](https://web.archive.org/web/20221005235839/http://www.crunchbase.com/person/jason-goldberg-2)创建的制作公司)创建的一些互动新媒体特性的更多细节。

上周，库彻和戈德堡给了我一些新房产及其背后的商业模式的预览。下周，两人将在纽约的互联网周上会见一些大型媒体买家，向他们展示五个左右的新房产。

他们将展示的属性之一:等等等等(按照上面的图像，将在域 Blahgirls.com，但该网站尚未上线)。这将是一个由“布兰妮、蒂芙尼和克里斯特尔”主持的每日卡通八卦节目，旨在“就像比维斯 T11 笨蛋是戏剧女王一样，有经前综合症、猫打架、时尚报道、购物狂欢，当然，还有很多有趣的…八卦！”这部剧的背景将是克里斯特尔的卧室——我看到的预览版让他们在卡通电视上观看一段真实的视频，并对其进行评论。是的，这很有趣。

Katalyst 还表示，他们也将为电视节目包装剧集，并与一家电视网达成协议。喜剧中心或 MTV 是显而易见的猜测，但我实际上不知道他们与谁合作。

下周应该会有更多。
