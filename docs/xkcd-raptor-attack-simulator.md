# XKCD 猛禽攻击模拟器

![](img/8e40e18df6f9242e3eb0db929770141e.png "sim")
深感极客。如果 XKCD 的书呆子式幽默和更书呆子式的元幽默还不够，这位有进取心的程序员(显然他有太多的时间)创造了一个关于猛禽攻击场景的[互动模拟](https://web.archive.org/web/20230128052141/http://jonathanryan.org/2008/02/08/raptor-attack-simulation/)[在这里提出](https://web.archive.org/web/20230128052141/http://www.xkcd.com/135)和[在这里解决](https://web.archive.org/web/20230128052141/http://www.mbeckler.org/velociraptors/velociraptors.html)。让那些方块看起来像迅猛龙需要一点想象力——虽然这对你自己来说并不是一个拉伸……哦， [burn](https://web.archive.org/web/20230128052141/http://www.explosm.net/db/files/Comics/Rob/epic.png) ！这并不新鲜，也不具有开创性，但是，嘿，今天是星期五。

[点击此处启动](https://web.archive.org/web/20230128052141/http://jonathanryan.org/raptor_3/raptor.htm)并按 F5 或重新加载再试一次。你的方块将朝着鼠标的方向运行。到目前为止我已经活了 2.5000000000000001 秒，创造者说他得到了 2.8 秒。来吧，辩论一小时后才开始，你还是做这个吧。告诉我们你的情况。
