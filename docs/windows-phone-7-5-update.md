# 微软开始推出 Windows Phone 7.5“芒果”

微软几天前承诺，Windows Phone 7.5 更新(也称为“芒果”，因为更新现在需要有趣的小名字)将“在未来一两周的某个时候”推出——果然，说推出才刚刚开始。

至于*和谁*合作……这仍然是个谜。

看看这句话:

*今天，我们开始向超过 98%的现有 Windows Phone 客户推出 Windows Phone 7.5。这是一次跨越运营商、手机型号和国家的同步、协调的全球更新。这一次，几乎每个人都是第一个。*

然后，仅仅几个句子之后:

所以我们开始时故意放慢速度。本周，我们将向 10%的客户提供更新。

好消息是:万岁！98%的人将从今天开始获得更新！大家先走！但是坏消息是:只有 10%的人真正安装并使用它。

当然，缓慢推出是有意义的；之前的 Windows Phone 7 更新已经变得和普通手机一样糟糕。他们的信息并不完全清楚，他们的“[我的更新在哪里？](https://web.archive.org/web/20230204233608/http://mango.microsoft.com/windowsphone/en-us/features/update-schedule-usa.aspx)”并没有太大的帮助:同样，他们说他们今天会“发布”更新，并带有一个特殊的小星号，表明更新可能在几周内不会上线。脏乱。

忘记了 **Windows Phone 7.5 的新功能**？以下是亮点的快速回顾:

*   脸书场所支持集成
*   面向第三方开发者的动态实时磁贴
*   联系人可以分组并固定在主屏幕上
*   Twitter/LinkedIn 支持
*   Windows Live Messenger 和脸书聊天
*   线索消息和电子邮件
*   一套 Bing 的东西(歌曲 ID，二维码阅读器，专辑/CD 封面标识符等。)
*   新相机用户界面
*   Internet Explorer 9
*   铃声管理
*   前置摄像头支架
*   可选，取决于运营商:可视语音邮件、网络共享
