# 仅仅过了 48 小时，Mountain Lion 已经占据了 Mac 网站 3.2%的流量

据广告网络 Chitika 的[报道，苹果桌面操作系统的最新版本](https://web.archive.org/web/20230127141214/http://insights.chitika.com/2012/mountain-lion-roars-captures-3-2-share-in-first-48-hours/) [Mountain Lion](https://web.archive.org/web/20230127141214/http://www.apple.com/osx/) 在正式发布后仅 48 小时，就已经占据了 Mac 网站 3.2%的流量。鉴于它的低成本(19.99 美元)和它可以下载并且非常容易安装的事实，这些数字有可能在未来几天内继续快速增长。山狮已经在 Chitika 的统计中超过了 OS X 10.4(老虎)，并在不久的将来超过豹。

去年，Chitika [报道](https://web.archive.org/web/20230127141214/http://insights.chitika.com/2011/mac-os-x-lion-fails-to-roar/)采用 OS。X 10.7 Lion 落后于预期，用户也没有像往常一样从雪豹更新。从 Chitika 今天发布的数据来看，这仍然是事实，因为雪豹仍然是该公司网络上最受欢迎的 Mac OS(45.5%)，其次是 Lion (35%)。正如我们自己的 MG Siegler 在本周早些时候的评论中指出的，“如果你不喜欢 Lion，你可能会更喜欢 Mountain Lion，因为它似乎解决了一些人在 OS X 的上一版本中遇到的许多性能/古怪问题，”虽然不是每台运行 Snow Leopard 的 Mac 电脑都可以升级到 Mountain Lion，但很有可能相当多的 Snow Leopard 用户会决定现在最终更新他们的电脑。

[![](img/5a120ee4b039e774d91b942efb2e4996.png "distribution7_27")](https://web.archive.org/web/20230127141214/https://techcrunch.com/2012/07/27/mountain-lion-already-accounts-for-3-2-of-mac-web-traffic/distribution7_27/)

我刚刚看了一眼我们自己的 TechCrunch.com 分析数据，根据谷歌分析，几乎 17%运行 Mac 的用户已经升级到 Mountain Lion。此外，虽然在 Chitika 的数据中，雪豹仍然是最受欢迎的 Mac OS，但目前几乎 60%的 TechCrunch 读者运行 Lion，只有 22%的人使用雪豹。

与所有这些统计数据一样，重要的是要记住，这些数据仅限于访问 Chitika 广告网络的用户。这可能会扭曲数据，但 Chitika 的数据通常非常接近我们从其他第三方分析公司看到的数据。
