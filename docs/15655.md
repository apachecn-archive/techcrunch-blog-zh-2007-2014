# 脸书宣布可搜索标签，承诺为公众对话提供更多功能

脸书刚刚宣布，它确实推出了通过标签跟踪对话的功能，正如三月份报道的那样。

需要说明的是，之前没有任何东西阻止你在脸书内容中包含标签——只是它们没有任何真正的功能。在宣布新功能的博客文章中，该公司承认这并不是一个新的想法，并指出它将“类似于 Instagram、Twitter、Tumblr 或 Pinterest 等其他服务”——当你点击一个标签时，你会获得使用该标签的评论。脸书表示，它的功能将包括搜索标签，点击来自其他服务的标签，以及直接从标签源写文章。

以下是脸书解释这一增加背后的原因:

> 每天，数亿人使用脸书分享他们对身边发生的重大事件的想法。无论是谈论最喜欢的电视节目，为家乡的运动队加油，还是在突发新闻事件中与朋友互动，脸书的人们都可以与朋友联系，了解世界各地发生的事情。…
> 
> 为了让这些对话更加引人注目，我们将推出一系列功能，展现人们对公共事件、人物和话题的一些有趣讨论。作为第一步，我们开始推出关于脸书的标签。

请注意，脸书说这只是“开始推出”的功能，所以你可能不会马上看到它。(该公司通常不会一次为其整个用户群打开功能，而是逐步推出。)请注意，该公司表示将推出更多这方面的功能。

为了说明脸书对话的规模，该公司指出，8800 万至 1 亿美国人在黄金时段使用这项服务。*《权力的游戏*》上周的《红色婚礼》被提及 150 万次，而围绕今年的奥斯卡有 6650 万次互动(包括赞、评论和帖子)。

这篇博文没有涉及我们在三月份开始听说这个功能时对的一些疑问——隐私。“公开对话”的概念在脸书和 Twitter 等服务上的运作方式不同，因为许多脸书帖子都有一定程度的隐私限制。

一位公司发言人告诉我，从隐私的角度来看，标签将与 Facebook 图形搜索类似。换句话说，你只能看到你有权看到的评论。因此，如果我在一个仅限朋友的帖子中包含一个标签，那么我的朋友仍然是唯一可以在标签搜索中看到它的人。
