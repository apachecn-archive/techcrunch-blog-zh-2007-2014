# 蒂娜·菲嘲笑迈克尔·贝在金球奖的消费电子展炸弹

CES 成就了辉煌！或者至少是金球奖。自 1967 年首届消费电子展举办以来，消费电子产品变得越来越容易销售，人们一直在讨论这个活动对科技行业和整个世界的重要性，甚至对顶级硬件制造商也是如此。

正如 TechCrunch 的约翰·比格斯在上周的活动前所写的那样，“好像大家伙们现在只是在走过场。他们就是。”

但似乎公司可能无意中发现了保持年度活动有趣的秘密:名人崩溃。自从迈克尔·贝在三星展示期间的提词器故障后逃离舞台已经[一周了——即使对于病毒式内容来说，互联网时间也是永恒的——但是人们仍然在谈论这个事件。](https://web.archive.org/web/20221207232650/https://beta.techcrunch.com/2014/01/06/michael-bay-joins-tom-hanks-lady-gaga-and-dr-dre-in-memorable-ces-wtf-moments/)

在今晚的金球奖上，蒂娜·菲嘲笑了贝的不幸之后，他的视频再次在社交媒体上传播。

Fey 的模仿只有几秒钟长，她实际上并没有跑下舞台，甚至没有提到 CES，但[的观众立即明白了这个笑话](https://web.archive.org/web/20221207232650/https://twitter.com/search?q=michael%20bay%20tina%20fey&src=typd)。

迈克尔·贝值得一些同情，仅仅因为你是世界著名的导演并不意味着你擅长公众演讲。但至少贝有擎天柱(和大量现金)来安慰他。

虽然 CES 历史上大多数最大的 WTF 时刻(如 2013 年的[高通怪诞秀](https://web.archive.org/web/20221207232650/http://www.theverge.com/2013/1/8/3850056/qualcomms-insane-ces-2013-keynote-pictures-tweets))可能会让公关人员尖叫，但他们确实避免了活动变得可预测和无聊——这对所有较小的硬件公司来说很重要，这些公司每年都在拉斯维加斯推出真正令人惊叹的、有创意的和创新的产品。
