# 如果不在图形搜索中，脸书会把你的查询交给必应

[脸书](https://web.archive.org/web/20221205124637/http://www.facebook.com/)今天早上[发布了“图片搜索”，](https://web.archive.org/web/20221205124637/https://beta.techcrunch.com/2013/01/15/facebook-announces-its-third-pillar-graph-search/)这是一款新的搜索工具，它可以以一种社交网络巨头非常谨慎地指出与网络搜索非常不同的方式找到人、照片、地点和兴趣。

但是，今天上午在脸书总部举行的新闻发布会接近尾声时，脸书首席执行官马克·扎克伯格宣布了“还有一件事”:脸书加深了与微软的合作伙伴关系，通过其搜索引擎[必应](https://web.archive.org/web/20221205124637/http://www.bing.com/)显示网络图表无法回答的搜索。

这意味着，虽然脸书本身可能不会进入传统的网络搜索领域，但用户毕竟不需要走出社交网络的界限——比如说，去谷歌这样的网站——来进行网络搜索。扎克伯格称 Bing 为“世界级的搜索引擎”,如果 Graph Search 不能给用户最好的回应，他很乐意让用户更深入、更快速地访问 Bing。

今天早上阿炳发表的博客文章透露了更多关于整合的信息:

> “现在，当你在脸书上进行网络搜索时，新的搜索结果页面采用两栏布局，左边是必应搜索结果，上面覆盖着来自脸书的社交信息，包括有多少人喜欢某个结果。在右边，你会看到与你的搜索相关的脸书页面和应用的内容。”

在新闻发布会后的问答环节中，扎克伯格扩展了双方的合作关系。“我们与微软的人有很好的合作伙伴关系……这(必应和脸书的整合)在某种程度上凸显了图形搜索和网络搜索的不同，”脸书的搜索结果有助于使必应的搜索结果更加社会化，必应的搜索允许脸书用户找到他们需要的信息的链接。

但他也补充说，他不反对与其他搜索公司合作。“我很乐意与谷歌合作。当我们整合 Bing 网络搜索时，我们非常公开地表示，我们希望让搜索变得更加社会化……这不仅仅是我想用 Bing 做的事情。”

事实上，脸书和微软多年来一直关系密切。早在 2007 年，微软[向当时私有的脸书](https://web.archive.org/web/20221205124637/https://beta.techcrunch.com/2007/10/24/facebook-takes-the-microsoft-money-and-runs/)投资了 2.4 亿美元，使这家社交网络公司的估值达到 150 亿美元(当时一些[人对这一估值望而却步](https://web.archive.org/web/20221205124637/http://www.inc.com/news/articles/200710/facebook.html)。)显然，这对微软来说是一项卓有成效的投资，因为脸书目前在公开市场上的市值为 650 亿美元。现在看来很明显，微软/脸书的关系一如既往地牢固，尽管最初的投资已经得到回报——无论是在业务还是工程层面。
