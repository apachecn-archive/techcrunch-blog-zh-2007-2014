# 通过 DeepMind，谷歌为我们在每一次计算交互中看到自己的未来做好了准备 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2014/01/27/why-google-bought-deepmind/>

谷歌似乎已经支付了至少5 亿美元收购 DeepMind ，这是一家人工智能初创公司，拥有许多知名投资者，并展示了显示计算机以与人类玩家非常相似的方式玩视频游戏的技术。据报道，脸书也试图收购该公司，大多数人心中的疑问是“为什么？”

当然，更智能的计算意味着更深入的数据收集和分析。任何旧电脑都可以收集信息，甚至可以做一些基本的分析工作，将它与其他数据集进行比较和对比，在因果或相关因素显而易见的情况下得出简单的结论。但是，仍然需要人类分析师从所有这些数据中找出意义，并从每天出现的大量、不分青红皂白的消费者数据中选择有意义的信息。

人工智能和机器学习专业知识可以帮助提高谷歌和其他依赖这些信息的公司收集数据的效率和质量，但它也可以为公司进入计算交互的下一个主要阶段做好准备:将物联网变成同伴的互联网。谷歌正在努力开发将使我们的生活更加以计算机为中心的技术，包括无人驾驶汽车和人形机器人来接管包裹递送等日常任务，但所有这些计算机交互的新机会都需要一个更好的界面，如果它们要变得可信和广泛使用的话。

谷歌已经在开发能够预测用户需求的软件和技术，并充当一种私人管家。谷歌现在从你的 Gmail 和搜索历史中分析信息，预测你会问什么，并提前提供信息。Now 一直在稳步发展，变得越来越聪明，并融入了更多的数据源，但它仍有很大的改进空间，而且没有比拥有一台像人一样思考的计算机更好的预测人类需求的方法了。

谷歌未来战略的另一个关键部分与硬件有关。该公司最近一次引人注目的收购是本月早些时候以 32 亿美元现金收购了 [Nest Labs。Nest 的智能恒温器还使用大量的机器学习来帮助预测用户的日程和需求，这是 DeepMind 可以在基础层面上提供帮助的事情。但是还有一个更大的机会，因为更人性化的元素可以帮助普通用户更容易理解物联网的概念。](https://web.archive.org/web/20230129225154/https://techcrunch.com/2014/01/13/google-just-bought-connected-device-company-nest-for-3-2b-in-cash/)

除了可以玩 DeepMind 视频游戏的电脑，我们几乎没有看到其他东西，但这个演示充分说明了谷歌可以与该公司合作。诚然，该公司已经进行的机器人和硬件投资很有趣，但 DeepMind 在许多方面是将所有这些独立举措汇集在一起的线索:在技术上令人印象深刻的创新和说服日常终端用户实际接受它们之间存在采纳脱节。DeepMind 可以帮助将看起来非常没有人情味的技术人性化(在无人驾驶汽车的情况下，甚至是反人类的)，以刺激吸收。

据报道，谷歌成立了一个道德委员会来监督 DeepMind 人工智能技术的使用，这可能是更多人类机器的一个重要原因。谷歌可能并不担心意外创造天网的可能性，但当你开始建造像人类一样思考和行动的计算设备时，你肯定会进入令人担忧的道德领域。就所述技术可以学习和了解其用户的内容以及我们有什么责任(如果有的话)来说，我们必须以不同于任何标准计算机的方式对待所述技术。

取决于你对谷歌及其业务的看法，DeepMind 的收购要么令人不安，要么令人兴奋。当然，它有可能两者兼而有之，人工智能和机器学习的任何潜在进步也是如此，但我不禁被谷歌以其最新的大动作描绘的画面的可能性迷住了。它似乎比任何其他公司都更致力于实现我们从小到大都在看的科幻大片的愿景，而且不可能否认这种雄心的诱惑。
