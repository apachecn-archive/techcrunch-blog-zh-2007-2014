# 网络效果不够好 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2012/11/04/the-network-effect-isnt-good-enough/>

**编者按:** *[Nir Eyal](https://web.archive.org/web/20230209124937/http://www.nirandfar.com/about) 在[NirAndFar.com](https://web.archive.org/web/20230209124937/http://www.nirandfar.com/)发表关于心理学、技术和商业交汇的博客。Sangeet Paul Choudary 在 [Platformed.info](https://web.archive.org/web/20230209124937/http://platformed.info/) 分析网络业务的商业模式。*在* [*的 Twitter 上关注 Sangeet，@sanguit*](https://web.archive.org/web/20230209124937/https://twitter.com/sanguit) *和 Nir 在* *[@nireyal](https://web.archive.org/web/20230209124937/https://twitter.com/nireyal) 。**

如果说硅谷有一个膜拜的祭坛，那就是神圣网络效应的圣殿。它的神秘力量将孤独的初创企业从默默无闻中拯救出来，并让它们名利双收。入选者名单包括了过去 15 年中几乎所有的科技成功故事。苹果、脸书、微软、易贝和贝宝凭借网络效应的巨大力量，各自的估值都飙升至数十亿美元。

但是今天，网络效应的力量正在消退，至少目前是这样。传统上，网络效应被定义为一个系统，在这个系统中，网络上的每个新用户都为所有其他用户增加服务的价值，网络效应通常会创造一种赢家通吃的动态，或者使一家占主导地位的公司凌驾于其他公司之上。此外，这些公司经常对其行业行使类似垄断的权力。

##### **开始时**

曾经，一家公司要利用网络效应所要做的就是促进大量客户之间的沟通。如果有足够多的人使用一个特定的系统来交换信息，一个领导者就会出现，并成为事实上的平台。能够形成市场或促进各方之间信息流动的公司作为数据传输的中心变得非常强大。

事实上，第一个网络效果平台是贝尔电话公司，它在近 100 年前建立了政府批准的垄断。从那以后，成功的网络效应企业基本上都唱着同一首赞美诗。

首先，通过建立必要的基础设施或发明新技术来建立沟通的媒介。例如，从东海岸到西海岸铺设电话线。然后，提供网络接入以提高信息传输的便利性——比如说，通过销售传真机。最后，抢在竞争对手之前扩大用户群。如果你比你的竞争对手成长得更快，瞧！你在天国之门里。

##### **狂喜**

至少这是计划。但是今天，事情不那么简单了。首先，在过去，消费者通过他们在硬件上的前期投资来付费接入网络。这些前期成本将用户锁定在网络中，一旦他们进入，他们就永远无法脱身，从而为潜在的竞争对手设置了进入壁垒。

然而，提供网络接入的成本已经急剧下降。客户购买昂贵的硬件来使用网络的日子已经一去不复返了，相关的锁定效应也是如此。

##### **转换**

除了接入成本降至零，曾经将用户锁定在网络中的另一个关键因素也消失了。曾经，将联系人移植到一个新的网络上，就像从 Yahoo！瞄准，是一个不平凡的任务。

然而今天，用户使用他们的脸书、推特或谷歌个人资料在几秒钟内就可以加入一项新的服务。以 Instagram 或 Pinterest 为例，一个新兴的网络可以利用社交图实现的单点登录，比以往任何时候都更快地达到临界质量。用户不仅可以输入他们的个人信息，还可以带来他们的关系。在社交网络时代，社交图的便利性在很大程度上打破了曾经让用户局限于一个网络而不是另一个网络的束缚。

##### **照看羊群**

如果没有对物理硬件的前期投资和用户转移个人信息和联系人的新能力，一家公司如何留住用户？网络效应锁定用户的能力死了吗？几乎没有。

利用网络效应的力量现在存在于“储值”中与网络接入成本不同，储值是一种投资，随着重复使用而小幅度增加，用户参与得越多，服务的重要性就越大。

##### **存储值**

储值有四种形式，公司利用这些微小的投资来锁定他们的服务并留住用户。

**创意内容**(例如 Pinterest、脸书、Instagram):用户投资[创建创意内容组合](https://web.archive.org/web/20230209124937/http://platformed.info/naturally-viral-how-to-design-your-platform-for-self-expression/)，这构成了他们在平台上互动的基础。内容的质量和数量导致与其他用户的更多交互，这反过来为内容创建者提供了更大的价值。

声誉(如 TaskRabbit、AirBnB、StackOverflow):尽管易贝等实体商品市场已经存在了一段时间，但服务市场最近变得越来越受欢迎。信任是这种新型网络效应业务的重要组成部分。因此，建立在平台上的声誉直接为所有用户带来更大的价值。在一个平台上建立声誉需要持续提供高评级的服务，还可能涉及符合该平台规定的一些最低标准。因此，一旦服务提供商在一个平台上建立了声誉，它就会阻止她迁移到一个竞争平台。

**使用数据:**用户[以数据](https://web.archive.org/web/20230209124937/http://platformed.info/seeding-platform-standalone-square-opentable-delicious/)的形式存储价值，要么通过主动收集信息，如 Dropbox 或 Reddit 的情况，要么随着用户的使用通过提供更多相关信息来改善服务，如 Quora 的情况，Quora 根据使用情况提供个性化的新闻。用户通过平台消费的信息越多，算法在向用户推荐相关内容时就变得越智能。在这两种情况下，由用户构建或为用户构建的数据集随着使用量的增加而提供更大的价值，这是竞争平台无法直接提供的。

**影响力**(例如 Twitter、YouTube 频道订阅):利用单边追随模式的网络创造了一种影响力动态。与导入联系人或“加好友”不同，收集关注者在很大程度上不受用户的直接控制。除了 Twitter 服务条款禁止的粗略策略之外，积累更多 Twitter 粉丝只能通过在 Twitter 上发布其他人觉得足够有趣的内容来分享。随着用户追随者数量的增长，网络中的存储价值和保持积极参与的动力也在增长。

##### **通往王国的钥匙**

创造网络效应已经不是以前的样子了。今天，用户创造的存储价值增强了网络效应的力量，从而留住用户并扩大市场份额。这种动态使得创造用户习惯变得更加重要，因为储值投资只有通过用户体验的连续传递才会发生(参见 Nir 之前的[文章](https://web.archive.org/web/20230209124937/http://www.nirandfar.com/2012/03/how-to-manufacture-desire.html)和[视频](https://web.archive.org/web/20230209124937/http://www.nirandfar.com/2012/09/desire-engine-in.html))。

随着社交图谱的便携性和加入社交网络的前期成本下降，企业必须利用新的方式来获取和留住用户。利用网络效应和储值的商业模式掌握着王国的钥匙。
