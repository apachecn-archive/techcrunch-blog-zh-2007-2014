# 来自印度的基于云的演示应用 Deck 从 Hotmail 联合创始人高通那里筹集了 60 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/12/16/deck-a-cloud-based-presentation-app-from-india-raises-600k-from-qualcomm-hotmail-co-founder/>

让用户在旅途中创建演示文稿的移动优先应用 Deck 已经从高通风险投资公司和由 Hotmail 联合创始人 Sabeer Bhatia 领导的天使投资人那里筹集了 60 万美元的种子资金。这家总部位于班加罗尔的同名初创公司表示，它将利用这笔资金雇人——它最近挑选了一些在《阿凡达》中崭露头角的设计师——并进行国际扩张。

Deck 在一个合适的时机进入市场。令人惊讶的是，PowerPoint 留出了太多的创新空间，允许更多的初创公司在在线演示空间启动。我们已经看到【Prezi 如何扩大其用户群，并积极招聘到今年年底达到 3600 万用户。

创业公司的创始人 Sumanth Raghavendra 告诉我，去年， [Deck 还从高通风险投资公司的 QPrize](https://web.archive.org/web/20221206162415/https://qualcommventures.com/qprize/winners-finalists-2012/india/) 获得了 10 万美元的奖金。Deck 在[安卓](https://web.archive.org/web/20221206162415/https://play.google.com/store/apps/details?id=air.com.deck.present)和 [iOS](https://web.archive.org/web/20221206162415/https://itunes.apple.com/us/app/deck-one-touch-presentations/id756407884) 上作为 app 提供。自今年 3 月推出测试版以来，该应用已经获得了约 30 万次下载。

巴蒂亚第二次支持 Sumanth(他向 Sumanth 的最后一个名为 [InstaColl](https://web.archive.org/web/20221206162415/http://www.instacoll.com/) 的企业投了种子资金，该企业已经关闭)，他说，虽然这个想法可能不是特别新，但 Deck 带来的是易用性，这符合许多用户，特别是印度等国家的用户今天上网的地方——手持设备。

具体来说，虽然这些应用可能仍然主要用于内容消费，但我们开始看到一波应用程序，这些应用程序正在帮助人们使用智能手机和平板电脑来创建动态内容，而不仅仅是简单的状态更新或自拍，用于他们选择的社交网络。

巴蒂亚说:“Deck 真的允许新手用户以一种以前不可能的方式表达他/她的创造力。”。(其他走这条路线的新应用包括 Bret Taylor 的 [Quip 文字处理应用](https://web.archive.org/web/20221206162415/https://beta.techcrunch.com/2013/11/21/quip-the-mobile-first-word-processing-app-now-imports-from-microsoft-word-google-drive-evernote-dropbox/))。

“为了实现这一点，我们已经从用户的视野中去除了所有的设计干扰和格式决定。Raghavendra 说:“每个主题都是独一无二的，保证会用出色的图形、动画和排版来支持你的故事。

Bhatia 说，在世界任何地方开发一个软件产品都很难，在印度更难，因为印度(UX)缺乏用户体验方面的人才。

“这些年来，我见过他们开发的一些质量相当不错的软件。不幸的是，他们以前的产品没有正确定位或营销，”Bhatia 说。

就其本身而言，Deck 正在通过从好莱坞设计工作室雇佣更多的设计师来创作电影，如《阿凡达》,来解决这个问题。目前，Raghavendra 拒绝透露这些设计师的名字，这样他就可以更悄悄地挖人了。

“我们的设计师以前为好莱坞大片和大预算广告创造视觉效果，但现在使用同样的技能为 Deck 创造演示模板，”他说。

那么 Deck 和 Prezi 相比怎么样呢？

我用 Deck 在 iPad 和 iPhone 上创建了一个演示文稿，体验相当流畅。这款应用程序可以让你从大约 60 个模板中挑选，而无需参与任何设计或格式化。对于那些只想专注于内容，并可能在移动或平板设备上创建演示文稿的用户来说，Deck 是一个不错的选择。但是如果你想添加你自己的音频或视频剪辑，你会失望的。Deck 主要适用于现场演示。

您还可以在 Slideshare 上发布幻灯片演示文稿，并与其他用户共享。

Raghavendra 说，Prezi 更适合那些有一定设计感和有一定演示技巧的用户。

“Deck 的目标群体是那些没有时间、没有兴趣或没有能力花几个小时来制作精美演示文稿的人。所以从某种意义上来说，deck 就像是记事本和 Prezi 的组合——编辑器类似于一个简单的笔记应用程序，但最终的输出像 Prezi 一样令人信服和难忘，”Raghavendra 说。

高通风险投资公司(Qualcomm Ventures)印度和以色列的董事总经理卡尔西·马达萨米(Karthee Madasamy)补充说，Deck 的移动和平板电脑优先战略正在帮助它甚至在美国和西欧的发达市场获得牵引力。

看到早期创业失败的企业家再次尝试并且不放弃总是很好的。但 Deck 要想成功，需要的不仅仅是善意，还包括印度和其他发展中国家快速增长的市场的高用户接受度，这些国家的大部分首次互联网用户都是手机用户。
