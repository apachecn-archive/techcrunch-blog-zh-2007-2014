# Foursquare 的下一个游戏:选择自己的冒险？

Foursquare 正在快速成长，从主流[交易](https://web.archive.org/web/20221006185221/https://beta.techcrunch.com/2010/07/07/foursquare-hardware-mayor-deals/)中获得动力，现在他们有银行里的[钱](https://web.archive.org/web/20221006185221/https://beta.techcrunch.com/2010/06/29/foursquare-20-million/)来维持它。但最大的问题仍然是:面对激烈的竞争，他们能保持下去吗？Twitter 已经有了定位功能，Yelp 最近[增加了](https://web.archive.org/web/20221006185221/https://beta.techcrunch.com/2010/01/15/yelp-iphone-app-4-check-ins/)签到功能，不久[脸书也会有](https://web.archive.org/web/20221006185221/https://beta.techcrunch.com/2010/05/09/facebook-places-check-in/)。Foursquare 如何在拥有数百万用户的竞争对手面前生存下来——如果不是比他们多几千万或几亿用户的话？显然，他们的答案是改变游戏。字面上。

今天，在[大款的颠覆者播客](https://web.archive.org/web/20221006185221/http://www.thebigmoney.com/articles/disrupters/2010/07/08/checking-in-with-foursquare)中，Foursquare 工程师 Anoop Ranganath 似乎谈了一些关于 Foursquare 及其未来的事情。他指出“*我们需要在签到*的基础上建立一个引人注目的产品”，并表示其中一个重要的关键是“*让游戏对用户来说更有趣——吸引更多人参与*”当然，他指的是这项服务的游戏元素，它允许用户在签到时获得积分和徽章。

当 Foursquare 刚开始的时候，积分排行榜是许多用户强调的一个领域——这确实是一个试图登上榜首的游戏。但这种新鲜感很快就消失了(部分是因为作弊，部分是因为它没有进化)，相反，用户对诸如获得徽章和市长职位(这反过来可以为你在某些场所赢得交易)等事情变得更感兴趣。Foursquare 早就知道直接游戏方面需要改进，但随着 Foursquare 的发展，这些功能不得不退居其次，让位于扩展和其他产品问题。但是根据 Ranganath 的说法，他们似乎正在对这个游戏元素进行彻底的改造。

Foursquare 是一款游戏。如果你考虑让你以不同方式升级的游戏和让你专攻的游戏。如果你认为城市是你游戏的棋盘，你就不能为所欲为。我们将引入游戏机制，允许你选择一条你要走的路，并获得一定的成就。

换句话说，Foursquare 听起来像是它的游戏将转向一种更像[选择你自己的冒险](https://web.archive.org/web/20221006185221/http://en.wikipedia.org/wiki/Choose_Your_Own_Adventure)的风格。

当主持人查德威克·马丁提出这个想法时，兰加纳特回应道，“*绝对是*”

Foursquare 并不是唯一有这种想法的公司， [Gowalla](https://web.archive.org/web/20221006185221/http://gowalla.com/) 、 [Whrrl](https://web.archive.org/web/20221006185221/http://whrrl.com/) 和特别是[谷歌支持的 SCVNGR](https://web.archive.org/web/20221006185221/https://beta.techcrunch.com/2010/05/12/google-backed-scvngr-takes-on-foursquare-looks-to-boost-fun-with-challenges/) 都在他们的产品中加入了类似的元素。但同样，Foursquare 现在势头强劲。如果他们在服务的基础上创建一个新的、身临其境的游戏元素，这可能会将服务推向一个新的水平。
