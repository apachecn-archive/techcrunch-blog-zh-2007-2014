苹果以50美元的折扣提供翻新的iPad 2型号

iPad 2没有太多会让你在购买前犹豫的东西，除了可能会扫一眼价格标签。今天，这已经不是什么大问题了。苹果已经开始在网上商店销售翻新的iPad 2s ,尽管看起来16GB型号目前还没有上市。退款只比原价低50美元，但折扣就是折扣。

据 MacRumors 称，苹果所有的退货和保修规定都适用于翻新机型。这些设备将有一个全新的后盖和电池，而设备的其余部分被清洁和测试任何问题。如果出了问题，苹果会更换那个特定的部件。

32GB WiFi + 3G型号的售价为679美元，两家运营商都有黑白两种颜色。649美元可以买到64GB的WiFi版，而64GB的WiFi + 3G版售价为779美元(两家运营商都有)。如果你不在乎iPad 2，只想要尽可能最便宜的iPad，苹果也在出售其16GB Wifi + 3G原版iPad 的翻新版，售价仅为399美元。

稍微在网上搜索一下，你可能会找到一台节省超过50美元的iPad 2。但对于如此昂贵的设备来说，苹果的支持非常重要，所以如果你有现金，我们建议你选择refurb路线。
