Windows Live Sync将存储空间升级到5GB，成为Windows Live Mesh TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2010/08/27/windows-live-sync-mesh/

Windows Live Sync将存储空间升级到5GB，并成为Windows Live Mesh

当微软在6月底推出测试版 Windows Live Sync 时，它将其与期待已久的 Windows Live Mesh 合并，但两者仍保持独立品牌。该产品将在未来几个月内采用Windows Live Mesh品牌。还有一些其他的变化。

目前，已经有240，000人在测试版中使用Windows Live Sync，在PC之间或PC和云之间同步文件(移动设备将在稍后添加，但那是最终的愿景)。个人电脑之间的同步是无限制的，但是到目前为止，每个账户的同步上限是2gb。这将很快增加到5gb。

测试版中的普通人只能将240兆的数据同步到云端，但你永远不会有太多的存储空间。然而，在电脑之间，他们平均同步675个文件，或1.2千兆字节的数据。一旦Microosft将移动加入等式，云中的存储量将大幅增加，因为如果可以的话，没有人想在手机上保存大文件。

Windows Live Sync也将升级为同步隐藏文件，并更好地显示哪些文件已同步，哪些文件正在同步。周一，在一个相关的公告中，微软将通过Exchange Active Sync使Hotmail电子邮件、联系人和日历可供大多数手机使用，这是一种推送技术(想想IMAP，加上联系人和日历)。

在设备之间同步文件和媒体是一个尚未完全解决的大问题。从云中获取手机上的所有数据将变得尤为重要。但我不确定选择文件夹进行同步的模式是否真的有效。有时候你不知道你想要访问什么，你只是想能够抓取你电脑上的任何东西。像Libox 这样的初创公司正在采用这种方法，并且已经提供了移动解决方案，但是还没有云备份。有人会得到这个权利。
