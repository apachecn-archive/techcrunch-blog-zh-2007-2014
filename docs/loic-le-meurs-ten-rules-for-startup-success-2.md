# 洛伊克·勒梅尔创业成功的十大法则

[《金融时报》](https://web.archive.org/web/20230203070711/http://www.ft.com/cms/s/0/459447d6-a28c-11dc-81c4-0000779fd2ac.html)对今天的法国([现为硅谷](https://web.archive.org/web/20230203070711/http://techcrunch.com/2007/06/13/loic-le-meur-moves-to-silicon-valley-to-start-video-content-company/))企业家[进行了简介。](https://web.archive.org/web/20230203070711/http://www.loiclemeur.com/english/2007/12/chris-nuttall-w.html)

Loic 是一位成功的企业家——他创建了 uBlog(与 Six Apart 合并),组织了一年一度的 Le Web 会议，现在还创建了 [Seesmic](https://web.archive.org/web/20230203070711/http://www.seesmic.com/) (注意，我是 Seesmic 的[投资者](https://web.archive.org/web/20230203070711/http://techcrunch.com/2007/11/20/100-seesmic-accounts-and-a-disclosure/))。所以尽管他是法国人，他的建议还是值得一听的。

文章中包括了他的创业成功十大法则。转载如下。

1.  不要等待一个革命性的想法。这永远不会发生。只需专注于一个简单、令人兴奋的空白空间，并尽可能快地执行
2.  分享你的想法。分享的越多，得到的建议越多，学到的东西也越多。与你的竞争对手会面并交谈。
3.  建立一个社区。使用博客和社交软件来确保人们了解你。
4.  倾听你的社区。回答问题，并根据他们的反馈构建您的产品。
5.  聚集一个伟大的团队。选择那些和你有很大不同技能的人。寻找比你优秀的人。
6.  成为第一个认识到问题的人。每个人都会犯错。公开解决问题，了解并改正它。
7.  不要花时间在市场调查上。尽早发布测试版本。在公开场合不断改进产品。
8.  不要沉迷于电子表格商业计划。无论如何，它们不会像你预测的那样出现。
9.  不要计划大的营销努力。你的社区喜欢这个产品更重要也更有力量。
10.  不要把注意力放在致富上。关注你的用户。金钱是成功的结果，而不是目标。
