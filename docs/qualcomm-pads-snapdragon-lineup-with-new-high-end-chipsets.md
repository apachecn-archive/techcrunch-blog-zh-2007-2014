# 高通用新的高端芯片组填充骁龙产品线

关于高通的 [S4 级芯片组](https://web.archive.org/web/20230204235319/https://techcrunch.com/2011/08/03/qualcomm-snapdragon-chipsets/)的消息一直让技术爱好者们坐立不安，现在该公司准备向[宣布他们引以为豪的 S4 阵容中的一些新成员](https://web.archive.org/web/20230204235319/http://www.prnewswire.com/news-releases/qualcomm-unveils-new-snapdragon-mobile-processors-across-all-tiers-of-smartphones-and-tablets-133949528.html)。

对于没有高通学学位的人来说，这里有一个快速回顾:该公司最近修改了骁龙品牌，以包括性能层，S4 层是他们保留所有真正好东西的地方。例如，APQ8064 是高通宣布的第一批 S4 芯片组之一，其处理器的四个内核以[2.5 GHz](https://web.archive.org/web/20230204235319/https://techcrunch.com/2011/02/13/snapdragon-bites-back-qualcomm-announces-a-quad-core-2-5ghz-chipset-for-mobile-devices/)的速度运行。所以，是的，当高通在什么东西上贴上 S4 贴纸时，期待一点疯狂吧。

高通在过去已经推出了三款 S4 处理器(MSM8960、MSM8930 和前面提到的 APQ8064)，但今天有八款新芯片组登场。它们是 MSM8660A、MSM8260A、MSM8630、MSM8230、MSM8627、MSM8227、APQ8060A 和 APQ8030，排名不分先后。

咻。

遗憾的是，由于高通没有深入细节，我们没有太多可以挖掘的内容。然而，我们确实知道每个芯片组都由 Krait CPU 驱动，并且它们支持世界上几乎所有的无线标准，所以这些坏男孩可能会在全球范围内看到一些游戏。希望一些更具体的细节很快出现，但现在我们必须安慰的是，新的 S4 芯片组将在明年的某个时候开始进入高端平板电脑和智能手机。
