# 扎克伯格:斯诺登揭露的美国国家安全局让科技行业更加紧密

据脸书的马克·扎克伯格称，美国国家安全局举报者爱德华·斯诺登披露的美国政府对互联网用户的监控范围产生了一个积极的影响:这使得科技行业的竞争对手更加开放地合作。

“国家安全局的问题，我认为这是一个真正的问题，尤其是对美国互联网公司来说，”他今天说。“当你考虑使用任何互联网服务和共享个人信息时，信任是如此重要。”

扎克伯格在巴塞罗那举行的世界移动通信大会的问答主题演讲中发表了讲话，被问及美国国家安全局的披露是否会危及 Internet.org 联盟的使命，该联盟有着将世界上每个人都连接到互联网的雄心，脸书是该联盟的主要/创始成员。

扎克伯格淡化了对政府监控的恐惧可能对 Internet.org 的使命产生的潜在影响，事实上，他提出了相反的观点，称他认为这可能会使目标更容易实现，因为在后斯诺登时代的科技世界里，有一种新的合作精神。

“国家安全局的问题让整个行业比以往任何时候都更好地合作，”他说，并补充说:“历史上，我们曾与一些竞争对手在政策问题上合作，这甚至有助于整个行业——互联网政策问题——但现在这是一件如此重要的事情，因为国家安全局的一些披露是如此极端，我确实感觉到很多行业更加一致了。”

扎克伯格没有指名道姓地说谁曾经对他的提议充满敌意，现在已经不那么敌对了，但一个可能的候选人是(大概)谷歌。

在对国家安全局问题的进一步评论中，扎克伯格补充说，从美国政府那里获得的协议，让脸书能够分享“政府要求我们的一切”——就[对用户数据](https://web.archive.org/web/20230216135650/https://techcrunch.com/2013/08/27/facebook-reports-government-data-requests-us-leads-with-12k-during-the-first-half-of-2013/)的请求而言——也“有助于”消除人们对脸书政府数据挖掘程度的担忧。

但他也批评政府在其监控项目上缺乏透明度，以及对互联网服务的影响。“这并不可怕，”他说。“政府在这件事上搞砸了……他们太过分了。”

“他们现在才刚刚开始到达他们应该到达的地方，”他补充道。“我认为这整件事本来是可以避免的。”
