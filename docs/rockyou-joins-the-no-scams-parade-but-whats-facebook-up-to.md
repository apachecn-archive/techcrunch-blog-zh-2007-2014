# RockYou 加入无诈骗游行。但是脸书在做什么呢？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/11/02/rockyou-joins-the-no-scams-parade-but-whats-facebook-up-to/>

 Zynga [今天早上改变了他们的 lead gen 骗局政策](https://web.archive.org/web/20221006013112/http://www.beta.techcrunch.com/2009/11/02/zynga-takes-steps-to-remove-scams-from-games/)(整个 [Scamville 的背景在这里](https://web.archive.org/web/20221006013112/http://www.beta.techcrunch.com/2009/10/31/scamville-the-social-gaming-ecosystem-of-hell/)，见底部的更新)。根据我们收到的一封电子邮件，现在 RockYou 正在采取措施清理他们的行为。

在给 RockYou 出版商的一封电子邮件中，他们表示将开始遵守脸书关于要约诈骗的规定(和你一样，我们不知道他们为什么一直没有遵守，但执法不严可能是原因)。

不过，邮件中有两个有趣的信息。首先，RockYou 表示，从现在开始，你只会看到*“来自顶级品牌广告商的干净、安全的调查。”我们看到的所有调查都是手机订阅诈骗，所以我不确定是否有干净、安全的调查。*

第二，邮件中说“*脸书合规团队将从今晚开始密切关注要约墙。”*我们听说脸书正在严厉打击应用程序开发商的欺诈行为，但脸书对此不予置评，只是说他们一直在监控应用程序报价并执行相关规定。从我们看到的情况来看，强制执行并没有带来多少结果，但也许他们现在对形势更加认真了。
