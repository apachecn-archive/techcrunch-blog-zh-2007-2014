# 优步聘请欧洲人 Kees Koolen 担任首席运营官，帮助其进行全球扩张

汽车服务派遣公司[优步](https://web.archive.org/web/20221006111552/http://www.uber.com/)已经聘请了一位欧洲首席运营官，今天在 [LeWeb 伦敦](https://web.archive.org/web/20221006111552/http://www.leweb.co/)宣布了这一消息。在任期内将 booking.com 的收入提高到 90 亿美元的 Kees Koolen ，现在将成为公司的“二号人物”，仅次于首席执行官特拉维斯·卡兰尼克。Koolen 来自阿姆斯特丹，他将有一半时间在旧金山的优步总部工作，另一半时间在全球各地奔波，领导这家初创公司的国际扩张。

该公司目前在伦敦和巴黎设有办事处。阿姆斯特丹会是下一个注重生活方式的个人交通初创公司吗？

"你为什么要雇一个欧洲人来演首席运营官？"专业煽动者、优步投资者迈克尔·阿灵顿难以置信地问道。

卡兰尼克表示，此次招聘将主要关注欧洲扩张，因为该公司面临来自竞争对手 Hailo、GetTaxi 和 MyTaxi 的激烈竞争，这些公司已经在许多欧洲国家的首都建立了广泛的网络。

“世界不仅仅是硅谷，”优步的联合创始人卡兰尼克断言优步的全球愿景，[就在该公司为伦敦奥运会在伦敦推出五天后](https://web.archive.org/web/20221006111552/https://beta.techcrunch.com/2012/06/15/uber-learns-to-drive-on-the-left-soft-launches-in-london/)。

该公司的历史与 LeWeb 大会有着千丝万缕的联系:优步巴黎于去年 12 月在巴黎 Le Web 大会上发布，优步作为一个整体在 2008 年 LeWeb 大会期间创始人 Kalanick 和 Garrett Camp 的一次对话中诞生。
