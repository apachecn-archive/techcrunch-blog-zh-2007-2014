# 脸书的照片再次放大

网络上最受欢迎的照片分享网站又更新了。

在今天晚上的一篇博客文章中，脸书——目前最大的图片网站——宣布将推出一款新的图片浏览器，图片宽度为 960 像素，而不是自 2010 年 3 月以来的 720 像素(之前为 620 像素)。浏览器本身也进行了更新，用不透明的白色取代了当前的黑色灯箱，据说这将更多的注意力放在照片本身上。脸书还表示，照片现在的加载速度是以前的两倍，但它没有说明它是如何以如此快的速度提供内容的。

脸书最近一次重大的照片更新是在 2010 年 9 月，当时[推出了](https://web.archive.org/web/20230205021752/https://techcrunch.com/2010/09/30/facebook-photos-hi-res/)基于黑色灯箱的照片浏览器，并增加了对宽度高达 2048 像素的照片的支持(它实际上不会在浏览器中显示这些照片，但你可以下载这种大小的照片)。这一更新最终使脸书成为分享高质量照片的可行方式(在此之前，你只能下载低分辨率版本)。

对脸书来说，这是重要的一周。周二，T4 宣布了一系列与隐私控制和照片标记相关的调整——你很快就可以在照片出现在你的个人资料之前批准照片，这是用户多年来一直要求的。这也彻底改变了脸书安排工作的方式，不再强调签到。今天早些时候，它证实在推出类似 Groupon 的每日交易仅仅四个月之后，它就要终止[了，尽管基于位置的交易仍然存在。](https://web.archive.org/web/20230205021752/https://techcrunch.com/2011/08/26/facebook-daily-deals-check-in-deals/)
