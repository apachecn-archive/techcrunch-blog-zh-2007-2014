# 贾斯汀·狄龙:互联网是当代奴隶制的罪魁祸首吗？(TCTV)

当然不是。然而，作为全球化的一个关键原因和结果，互联网确实在创造条件创造今天全世界 2700 万奴隶方面发挥了重要作用。但它也是我们对抗奴隶制的工具。至少，[贾斯汀·狄龙是这样认为的，](https://web.archive.org/web/20230205035418/http://www.crunchbase.com/person/justin-dillon)[奴隶制足迹的首席执行官，](https://web.archive.org/web/20230205035418/http://slaveryfootprint.org/)这是一个由国务院资助的组织，在 2011 年[克林顿全球倡议中发起，](https://web.archive.org/web/20230205035418/http://www.clintonglobalinitiative.org/)致力于揭露和打击当代奴隶制。

事实上，互联网现在正成为一个不可或缺的工具，不仅用来对抗奴隶制，还让我们意识到奴隶制在我们消费的许多产品中是多么普遍。正如 Dillon 在来到我们的 TechCrunchTV 工作室时解释的那样，Slavery Footprint 发布了一款 Android 和 iPhone 移动应用程序,它将使我们能够发现我们购买的产品中包含多少奴隶制。

例如，狄龙提醒我，生产一部普通的智能手机平均需要 3.2 名强迫劳动者。这难道不像是为保持联系而付出的高昂代价吗？
