# 你的创业公司需要 Beta 测试人员吗？BetaBait 会帮你牵线搭桥(现在在电子邮件和网络上)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/05/does-your-startup-need-beta-testers-betabait-will-hook-you-up-now-on-email-the-web/>

你可以不先测试就发布你的应用程序，但事实是，这往往会带来灾难性的后果。因此，作为精明的人，当企业家想要测试他们即将推向市场的东西时，他们会出去找测试人员对他们的想法进行彻底的审查。显然，这对于建立一个成功的企业是必不可少的，但是找到这些有价值的测试者呢？不像听起来那么容易。

这就是 BetaBait 的用武之地。上个月推出的 BetaBait 希望为初创公司提供一种替代解决方案，以雇佣公关代表，发送垃圾邮件数据库，或打扰朋友来测试他们刚刚起步的应用程序创意。简而言之，这家初创公司希望扮演媒人的角色，因为它已经建立了一种服务，可以在一个地方将拥有新网络、移动和社交应用的初创公司与渴望的消费者和专业早期采用者联系起来。一旦连接上，初创公司就可以接触到他们需要的测试用户，成为他们的试验品，而消费者或专业人士可以提前获得市场上最新的应用程序。

正如科技世界所发生的那样，测试人员和早期采用者可以在帮助初创公司解决问题、创造新功能方面发挥重要作用，如果他们喜欢他们所看到的，可以转化为第一批客户，甚至可能成为传播者。显然，对于初创公司来说，了解他们试图接触的受众，并尽一切努力让用户觉得这种体验可口(不，是愉快)是至关重要的。早期采用者往往非常熟悉这个领域，并且通常知道这个领域的其他公司在做什么。

因此，虽然科技博客(咳咳)可以帮助传播消息，并带来早期采用者和测试者，但并不是每个应用程序都准备好了媒体亮相。测试版用户可以帮助引导创业公司朝着这个方向发展。因此，BetaBait 创建了一个简单的服务，初创公司注册并向 BetaBait 团队提交他们的应用程序以获得批准。一旦获得批准，BetaBait 会在每天发给 1300 多名早期用户(还在增加)的电子邮件中重点介绍这款应用。由于社区成员已经选择接收电子邮件，初创公司可以放心，他们正站在希望成为测试过程一部分的用户面前。

今天，BetaBait 正在启动第二阶段，或者更确切地说，是其每日电子邮件时事通讯的在线版本。由于有大量的初创公司在争夺注意力，BetaBait 为其用户创建了一个在线门户，以便能够更容易地浏览、参与和分享其初创公司数据库，目前包括 200 多家初创公司。[点击这里查看](https://web.archive.org/web/20230209125148/http://www.betabait.com/submitted-apps/)。

该公司将继续发送日常电子邮件，但只会关注“新来者”，即在过去 24 小时内提交并获得批准的应用程序。这封邮件还将继续包括赞助商(目前主要是初创公司)以及“创始人精选”，联合创始人科迪·巴比里(Cody Barbierri)和罗里·汤普森(Rory Thompson)选择一款应用进行展示。

这让我们想到了 BetaBait 最棒的部分之一:它的服务对创业公司和用户都是完全免费的。巴比里说，在这个早期阶段，团队并不关注现金，而是通过电子邮件赞助商来支付运营成本，这些赞助商有能力写下对他们产品和服务的描述，并将其显示在初创公司的日常电子邮件上。

虽然有一些服务可以帮助初创公司管理测试人员流程，确保测试人员提供反馈，而不是自己做这些，但 BetaBait 认为这些是补充服务。(例如，像 [LaunchRock](https://web.archive.org/web/20230209125148/http://www.crunchbase.com/company/launchrock) ，它给创业公司提供了一种轻松制作测试版邀请请求启动页面的方式。)Barbierri 表示，BetaBait 的目标是帮助刚刚推出或即将推出的早期初创公司获得他们需要的测试人员。也就是说，为初创公司提供一个资源来了解 beta 测试过程中的其他补充服务将是一个受欢迎的网站，并真正有助于使其成为 beta 测试的首选资源。

这里的另一个优势是，BetaBait 可以成为初创公司应用和网站早期流量的巨大来源，这是获得初创公司渴望的注册和测试请求的关键。这位联合创始人说，BetaBait 已经给它的初创公司发送了数千次点击，而且流量还在继续增长。

BetaBait 本身还处于测试阶段，门户网站仍然需要一些润色，但它是初创公司的一个很棒的资源，[所以请查看一下](https://web.archive.org/web/20230209125148/http://www.betabait.com/submitted-apps/)，让我们知道你的想法。
