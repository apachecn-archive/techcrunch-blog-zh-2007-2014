# 索尼在淘汰 iTunes 之前关闭了 CONNECT

对你有好处，索尼。你终于放弃了你的音乐连接服务。

虽然你在一年多前就停止了尝试修补它，但它从一开始就没有正常工作过，但你在明年三月关闭了它，这是值得称赞的。这应该给那些从未使用过这项服务的人一个机会来备份他们从未购买过的所有歌曲。

对于那些使用 CONNECT 的人，请放心，你仍然可以通过旧的 SonicStage 软件和任何兼容 ATRAC 的设备来播放你购买的曲目。索尼电子书阅读器的连接服务，据我所知，实际上有点受欢迎，不会被关闭。
