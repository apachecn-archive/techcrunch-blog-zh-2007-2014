# OpenX 获得 2000 万美元的 D 轮融资，以推动增长和收购

![](img/a3f158b5f4786802eb4f1865ed37162c.png)

数字广告技术公司 OpenX 在第四轮融资中获得了[2000 万美元](https://web.archive.org/web/20230204055541/http://www.crunchbase.com/company/openx)，由 AOL 和 SAP 等战略合作伙伴的风险投资部门提供，此外还有亚洲投资者三井公司和 Presidio Ventures(住友商事)。

之前的支持者 Accel Partners、DAG Ventures 和 Index Ventures 也参与了这轮融资，这使得 OpenX 的总融资额达到了 5000 万美元。

额外的资本将用于促进公司的增长，特别是在亚洲(更准确地说是日本和中国)，那里的在线广告市场仍然相对新兴。此外，这笔钱将用于加速 OpenX 的[企业解决方案](https://web.archive.org/web/20230204055541/https://techcrunch.com/2011/02/08/openx-enterprise-groupon/)的采用。

OpenX 首席执行官 Tim Cadogen 告诉我，在不久的将来，这些收益也可能被用于对小公司的战略收购——他们基本上对那些能够在他们预见扩张的领域(比如移动广告交易所)与他们互补的技术公司感兴趣。

SAP Ventures 牵头投资，并将获得董事会席位。
