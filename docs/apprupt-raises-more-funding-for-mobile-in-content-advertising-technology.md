# Apprupt 为移动内容广告技术筹集更多资金

为[移动内容广告](https://web.archive.org/web/20230331093046/http://eu.beta.techcrunch.com/2010/06/28/apprupt-launches-self-service-affiliate-network-for-mobile-apps/)运营表演网络的 appr upt[今天早上](https://web.archive.org/web/20230331093046/http://www.apprupt.com/en/)[宣布](https://web.archive.org/web/20230331093046/http://www.apprupt.com/en/press/pressrelease/2011-2/09162011-apprupt-secures-further-funding/)已经[从](https://web.archive.org/web/20230331093046/http://www.crunchbase.com/company/apprupt)[现有的支持者](https://web.archive.org/web/20230331093046/http://eu.beta.techcrunch.com/2010/05/06/apprupt-affiliate-network-for-mobile-apps-closes-financing/) T-Venture(德国电信的子公司)、Neuhaus Partners 和 KfW 那里筹集了更多的资金。融资的规模没有披露，尽管该公司表示，它涉及“七位数的欧元投资”。

新资金将用于增加该初创公司的销售人员、国际扩张以及投资其移动广告技术。
