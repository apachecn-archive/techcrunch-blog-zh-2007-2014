# 负责 Galaxy Tab 产品线的三星首席技术官下台

曾负责 GalTab 和其他安卓智能手机崛起的三星首席技术官奥马尔·可汗将前往花旗银行处理该公司的全球数字银行业务。他将由尼克·迪卡洛(Nick Dicarlo)和加文·金(Gavin Kim)接替“三星移动产品和服务发言人的职责”

很明显这整个硬件的事情对他来说并不成功。

这种横向移动并不罕见，但对于汗来说，他已经领导了一个相当成功的平板电脑和手机业务，然后决定转向一家银行，这表明了一点政治内讧和对 GalTab 线的潜在失望，特别是三星正在与苹果公司(T2)就两大巨头的“抄袭”技术进行诉讼。

三星也面临着来自 HTC 和摩托罗拉的巨大压力，虽然他们的最新安卓手机在特定人群中很受欢迎，但并没有像 Galaxy S 那样风靡全球
