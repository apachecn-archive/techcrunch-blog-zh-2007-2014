# 微软在东京游戏展上的主题演讲——它会有什么大作为吗？

![](img/bf4356d97f077bfb4d37c0721c7ab8c8.png)
[E3](https://web.archive.org/web/20221210002745/http://www.crunchgear.com/tag/e3-2010/) 结束了，Gamescom 也结束了，现在我们把东京电玩展视为“发布酷游戏的时间”名单上的下一个明显的亮点微软似乎正在利用这一优势，并将在展会上为[做一个重要的主题演讲。“Xbox 360 的愿景和 2010 年的战略”是一个暂定的标题…不完全是提高屋顶，微软，但我们会给你好处的怀疑。](https://web.archive.org/web/20221210002745/http://www.1up.com/news/microsoft-deliver-tgs-2010-keynote)

不过，我无法想象那会是什么。显然，他们在 E3 真的放弃了很多东西，比如 Kinect 和新的 360 型号。我在想，他们会稍微谈一谈 Live 与 Zune 和 Windows Phone 7 的整合。搭售、新媒体服务、手机上的直播内容……基本上推动 Xbox 360 成为许多内容的中心。他们以前做过。

我们可能还会看到一些新的 Kinect 独家报道，可能会有一些大牌。他们真的需要尽快把它放大，否则铁杆游戏玩家不会在发布时购买它。

无论如何，谈话应该在 9 月 16 日，展览的第一天，虽然我们不会在那里(这是在东京，人们)，我们会有所有的新闻在这里下降。如果运气好的话，可能会早一点。
