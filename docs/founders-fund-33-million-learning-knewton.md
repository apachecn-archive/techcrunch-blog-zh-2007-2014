# 创始人基金领投学习创业公司 Knewton 的 3300 万美元 D 轮投资

在近年来最大的一轮教育创业融资中， [Knewton](https://web.archive.org/web/20230310024037/http://www.knewton.com/) 完成了由[创始人基金](https://web.archive.org/web/20230310024037/http://www.foundersfund.com/)牵头的 3300 万美元的 D 轮融资。(同样位于纽约市的另一家教育初创公司 [2Tor](https://web.archive.org/web/20230310024037/http://2tor.com/) ，今年早些时候筹集了 3250 万美元的 C 轮融资。教育出版商 Pearson 也进行了投资，现有的投资者还有 Accel、Bessemer 和 FirstMark。估价轻松超过 1.5 亿美元。

像许多教育创业公司一样，Knewton 尝试了在线备考(GMAT，LSAT，SAT)。但它真正的优势在于创造了一种适用于任何课程的适应性学习算法。“我们的最终愿景是为所有教育提供动力，”首席运营官·刘中达说。今年秋天，Knewton 开始为亚利桑那州立大学的 10，000 名新生提供在线数学准备课程。它计划与学校和教育出版商合作，扩展到整个高等教育和 K-12 阶段。该公司还希望开放其 API，以便出版商可以将他们的电子教科书和课程插入 Knewton 的技术平台，并最终创建一个适应性学习应用程序和课程的市场。。

通过提问和测验，Knewton 将学习体验个性化，并根据学生的学习方式调整呈现给他们的内容。因此，如果你对某一类型的代数公式有困难，它会后退一步，给你更简单的版本，直到你掌握它。或者，如果你正在快速解决问题，这将使问题越来越难，直到你受到足够的挑战。一些亚利桑那州立大学的学生已经提前完成了数学课程。

乍一看，创始人基金会支持一家教育创业公司，这似乎有些奇怪，该基金的合作伙伴也鼓励孩子们“离开大学”。但是 Knewton 正在利用技术来改善人们的学习方式。普通合伙人卢克·诺塞克告诉我，他对今天的教育没什么兴趣，“整个官僚机构、建筑和教师。我最感兴趣的是让地球上的每一个人都能最大限度地进行个性化学习。”他认为 Knewton 有机会成为那家公司。

Knewton 已经有 70 名员工，并且发展迅速。如果一个帮助学生更快学习的平台可以让每个学生/每门课程/每学期获得 100 到 200 美元的报酬，这可能会很快成为一个非常大的业务。学历断了。对于 Knewton 这样的初创公司来说，这是一个很大的机会，也是一大笔钱，可以介入并尝试解决这个问题。
