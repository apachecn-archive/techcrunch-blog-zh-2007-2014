# 雅虎:贫穷、孤独和悲伤

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/11/06/yahoo-poor-alone-and-sad/amp/>

 <amp-img class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" src="https://web.archive.org/web/20230305065310im_/https://techcrunch.com/wp-content/uploads/2008/05/yang-sad.png" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 半年前[杨致远在 D 大会上上台](https://web.archive.org/web/20230305065310/https://techcrunch.com/2008/05/28/jerry-yang-were-done/)谈他的事业状态。他描绘了一幅雅虎在原地打转，没有明确未来的画面。

[快进六个月](https://web.archive.org/web/20230305065310/https://techcrunch.com/2008/11/05/jerry-yang-speaks-at-web-20-our-live-notes/)什么都没变。

除了雅虎已经被留在了神坛上，因为谷歌放弃了他们的雅虎搜索营销交易。

除了雅虎的股价又下跌了 13 美元，损失了 180 亿美元的股东价值

除了雅虎的大多数高层管理人员已经离开公司。

他们还剩下什么？一个仍然不能清楚表述核心目标的 CEO。杨在台上无精打采，精神萎靡，时而称雅虎为平台公司，时而称其为目标网站。但他也表示，雅虎打算在搜索领域保持与谷歌的竞争力，尽管两家公司可以投入研究的资源存在巨大差异。

杨的希望寄托在他们的战略上，即拥抱第三方开发者，将搜索之类的东西转变为网络服务。

也许是传递的问题，但是信息没有被传递。观众们不禁想知道，在公司被内部撕裂之前，这位首席执行官还能撑多久。或者被外人。

雅虎需要的是一位新的首席执行官。他们需要他们的巴拉克·奥巴马，一个让所有人都相信一个真正的领袖正在掌舵，准备战斗的人。有可信计划的人。一个能够激励雅虎和雅虎用户，让他们相信雅虎能够再次成为互联网上的一股力量的人。

那个领袖不是杨致远。

早就该改变了。

杨必须走。
