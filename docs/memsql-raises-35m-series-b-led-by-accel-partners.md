# MemSQL 融资 3500 万美元，由 Accel Partners TechCrunch 领投

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/22/memsql-raises-35m-series-b-led-by-accel-partners/>

帮助实时检索和分析数据的大数据初创公司 MemSQL 已经从 Accel Partners 和 Khosla Ventures 获得了 3500 万美元的 B 轮融资。

这笔融资将用于扩大产品开发，为不断增长的用户群提供支持，并积极推动大数据市场的更大份额，预计到 2017 年将达到 324 亿美元。

除了 Accel 和 Khosla，MemSQL 的现有投资者 IA Ventures、First Round Capital 和 Data Collective 也参与了最新一轮的融资，使这家初创公司迄今为止筹集的总资本达到 4500 万美元。

MemSQL 由前 Facebookers 用户 Eric Frenkiel 和 Nikita Shamgunov 于 2011 年创建，它处于一场运动的前沿，这场运动不仅旨在帮助大型企业快速管理大数据，而且还旨在提高准确性。

MemSQL 的内存数据库已经在为 Zynga 和 Shutterstock 的大数据分析提供支持。

前脸书工程师 Eric 说，对 MemSQL 的需求是巨大的。

“客户被我们的解决方案所吸引，因为它在商用硬件上提供了极快的性能，并且能够在单个数据库中存储和查询多种数据格式，”他说。

MemSQL 是一种新型的内存数据库，对于那些疲于处理系统收集的万亿字节数据的企业用户来说，这种数据库开始变得更有意义。虽然“NoSQL”数据库(如 MongoDB)有助于存储和管理大量数据，但应用程序(如 MemSQL)可以完成所有这些工作，并提供 SQL 的优势。

然而，这两者(NoSQL 和 MemSQL 等新加入者)哪一个更好还没有定论。正如《连线》杂志的这篇文章提到的，关键的一点是，所有这些都在走向一个新的现实。

“传统数据库技术容易出现延迟，需要复杂而昂贵的体系结构，并且依赖于速度缓慢的基于磁盘的技术。结果是一个过时的计算基础设施，无法在真正的实时解决方案所需的时间范围内处理速度和数据量，”这家初创公司在一份声明中说。

MemSQL 还与其他内存数据库竞争，如 [SAP Hana](https://web.archive.org/web/20221205205333/http://www.saphana.com/welcome) ，Teradata 和 SAS。

对于 Accel 来说，这标志着其大数据基金[的又一项重要投资。](https://web.archive.org/web/20221205205333/https://beta.techcrunch.com/2013/06/17/accel-closes-100m-for-big-data-fund-2-to-invest-in-the-second-wave-of-big-data-startups/)

Accel Partners 普通合伙人 Kevin Efrusy 表示:“我们在交易方面支持突破性的开源领导者 Couchbase，在分析方面支持 Cloudera 及其企业数据中心，但我们也相信 MemSQL 在实时分析方面有非常独特的地位。
