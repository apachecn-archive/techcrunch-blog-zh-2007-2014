# 通用汽车宣布推出下一款伏特，凯迪拉克 ELR

通用从平台赚钱，不仅仅是汽车。带着这种想法，凯迪拉克刚刚宣布了 ELR 跑车，这是一款基于 Voltec 平台的时尚跑车，也为 Volt 提供动力。自 2009 年北美国际汽车展上展示了一个概念(如图所示)以来，双动力系统的凯迪拉克就一直被传言。好了，朋友们，开始了。

ELR 将使用相同版本的 Voltec 为 Volt 提供动力，包括相同的四缸 1.4 升发电机。这使得 ELR 听起来只不过是一个 reskinned 伏特，而不是调谐电动跑车价值的性感金属片。没有什么比 8.5 秒内从 0 加速到 60 英里更能说明美国的运动跑车了。

凯迪拉克尚未宣布任何关于即将到来的跑车的真正细节。“ELR 的开发刚刚开始，所以关于性能、价格和时间的细节将在稍后公布，”[在新闻发布会上说。](https://web.archive.org/web/20230205020938/http://media.gm.com/content/media/us/en/cadillac/news.detail.html/content/Pages/news/us/en/2011/Aug/0817_cadillac)

加长轿跑车设计实际上对 Volt 的 2+2 配置很有意义。不管怎么说，后排座椅并不那么好用。这种改进的设计可能使它在空气中滑行时阻力更小。这里显示的照片是 2009 年 Converj 概念。该品牌非常忠于其鲜明的线条和巨大的格栅设计，所以生产版本可能会遵循相同的风格线索，如果不是完全一样的东西。

雪佛兰 Volt 对通用汽车来说并不是一个巨大的成功。据报道，该汽车制造商上个月仅售出 125 辆，今年迄今为止不到 3000 辆。该车型将很快在全国范围内上市，这可能有助于减少一些经销商。这款凯迪拉克车型将是除 Volt 的欧洲和澳大利亚同类车型之外，第二款采用该系列和混合动力系统的量产车。由于它戴着徽章和花环标志，它甚至可能有一个设备齐全的小屋，而不是雪佛兰伏特(Chevy Volt)那样的塑料玩具。
