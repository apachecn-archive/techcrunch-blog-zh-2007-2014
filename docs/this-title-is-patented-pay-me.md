# 醉心于许可费和专利的微软成了一个笑话 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/03/21/this-title-is-patented-pay-me/>

今天早些时候，有消息称微软正在起诉另一家公司侵犯他们的专利。这次的目标是什么？巴恩斯&诺布尔。是的，微软正在起诉一家图书连锁店。为什么？因为他们声称 Nook 电子阅读器(运行谷歌的 Android 操作系统)复制了 Windows CE 的状态栏。或者什么的。请原谅我的直言不讳，这完全是一派胡言。

[Devin 在 CrunchGear 上对](https://web.archive.org/web/20230130222235/http://www.crunchgear.com/2011/03/21/microsoft-alleges-patent-infringement-on-barnes-noble-android-devices/)有一个很好的概述，即争议中的专利实际上是什么。整件事都很可笑。每个人都知道，除了一方:微软。该公司已经完全沉醉于他们的专利和随后的蹩脚诉讼。结果，他们很快失去了几乎所有不在雷德蒙工作的人的心。

那么这一切的背后是什么呢？微软真的想阻止巴诺出售 Nook 吗？不。相反，他们正试图迫使他们许可使用他们的专利，这样微软就可以从售出的每台 Nook 中分得一杯羹。我哪里知道？首先，这是微软已经使用了一段时间的策略。

去年十月，[微软起诉摩托罗拉](https://web.archive.org/web/20230130222235/https://techcrunch.com/2010/10/01/android-lawsuits/)使用安卓系统的手机。奇怪的是，他们并没有因为同样的事情起诉 HTC——因为正是因为这个原因，他们[与 HTC](https://web.archive.org/web/20230130222235/https://techcrunch.com/2010/04/28/microsoft-htc-android-apple-patents/)达成了专利许可协议。微软的策略只是试图让摩托罗拉也这样做。现在他们正在对 Barnes & Noble 做同样的事情。

至少今天他们不再拐弯抹角了。微软自己的法律总顾问 Horacio Gutierrez 已经在这个问题上发布了一个帖子，标题包含“[许可是解决方案](https://web.archive.org/web/20230130222235/http://blogs.technet.com/b/microsoft_on_the_issues/archive/2011/03/21/android-patent-infringement-licensing-is-the-solution.aspx)”

古铁雷斯指出，为了先发制人，微软不是一家轻易提起诉讼的公司。事实上，这只是我们 36 年历史中第七起主动专利侵权诉讼。但我们根本不能忽视这种范围和规模的侵权。

好吧，让我们用微软自己的话来检验一下“范围和规模”。在这篇文章中，微软指出了 Nook 的以下侵权行为:

*   通过一个带标签的独立控制窗口，让人们能够轻松浏览设备应用程序提供的信息；
*   允许在接收背景图像之前显示网页内容，允许用户更快地与页面交互；
*   允许应用程序在下载内容上叠加下载状态；
*   允许用户容易地选择文档中文本并调整选择；和
*   为用户提供在不改变基础文档的情况下注释文本的能力。

“轻松选择文本”？“浏览信息”？“注释文本”？我是说，这听起来像个笑话，但不是。接下来，微软将起诉打字、看东西甚至呼吸的能力。我肯定他们在过去 30 年的某个时候申请了这些东西的专利。他们有 20000 项专利之类可笑的东西。

再说一次，最糟糕的是这些衣服是中空的。微软真的不在乎 Nook 的销售因为他们抄袭他们的“创新”而停止——他们很高兴让他们继续销售，只要 Barnes & Noble 为每卖出一台支付费用。

其他公司，比如苹果，可能不再对这些虚假的软件专利负责。但至少他们真的相信像谷歌(通过 HTC )这样的公司抄袭了他们的知识产权，并希望他们停止这样做。对于微软来说，这是一种商业模式。

事实是，这个可笑的模型只是微软的延伸，微软:许可。微软能发展到今天，是因为电脑制造商授权了 Windows 和 Office。但是随着我们进入移动时代，这种模式不仅被打破了，而且被摧毁了。谷歌正在授权 Android——但他们是免费的。这正是为什么他们能够如此轻松地扼杀 Windows Mobile。

当然，微软不明白这种模式已经不再有效，他们仍然试图让原始设备制造商为他们的新 Windows Phone 操作系统付费。事实上，他们甚至让新伙伴诺基亚为每部手机支付授权费！(同时以数十亿美元的营销和开发成本回报他们。)是行不通的。

现在我们来看看许可费 2.0:专利。微软在专利授权上赚的钱还要多长时间才能超过他们的移动部门？严肃的问题。也许他们已经是了。

更糟糕的是微软处理这一切的方式:直截了当的欺凌。这是他们在最近的诉讼中将富士康和英业达包括在内的唯一原因。这些人与正在讨论的侵权行为无关，但微软正试图施加来自各方的压力。随着越来越多的公司使用 Android 开发产品，他们会继续做同样的事情。

古铁雷斯写道:“为了帮助公司解决安卓持续的侵权问题，我们建立了一个全行业的许可计划。

太棒了。多好的人啊。

“*去年，HTC 获得了一项覆盖其基于安卓系统的智能手机的许可，这证实了我们许可优先策略的可行性。在电子阅读器领域，Amazon.com 去年与微软签署了一项涵盖其 Kindle 设备的专利许可。许多其他设备制造商也在许多现有的许可计划下获得了微软专利的许可*，”他继续说道。

如果你的朋友从桥上跳下…

底线是这需要被停止。微软正在威胁一系列行业的创新，试图灌输他们的下一个伟大的商业模式。有史以来最扯淡的商业模式。各位，这是微软的未来。有什么奇怪的吗，为什么他们已经失去了公众的青睐？没有人喜欢支持蠢驴。

我最喜欢的部分是微软是多么健忘。他们不仅让古铁雷斯写像今天这样荒谬的帖子。他们让自己的公关公司 Waggener Edstrom 向媒体发送电子邮件，大肆宣传这起诉讼！万岁！

微软仍然有许多有才华的人在做伟大的事情。Kinect 甚至 Windows Phone ( [产品](https://web.archive.org/web/20230130222235/https://techcrunch.com/2010/12/23/iphone-windows-phone/)，而非战略)跃入脑海。但是诉讼和律师把所有这些都埋在 700 公吨的垃圾下面。
