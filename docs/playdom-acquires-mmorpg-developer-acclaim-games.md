# Playdom 收购 MMORPG 出版商 Acclaim Games

Playdom 继续疯狂收购游戏开发商。游戏巨头[今天收购了](https://web.archive.org/web/20221206205725/http://www.prnewswire.com/news-releases/playdom-announces-acquisition-of-acclaim-games-inc-94202589.html)社交游戏发行商 [Acclaim Games](https://web.archive.org/web/20221206205725/http://www.acclaim.com/) 。交易条款没有披露。

Acclaim 已经在休闲和 MMORPG 游戏领域建立了自己的声誉。该公司还在 2010 年 3 月推出了脸书吉他游戏 RockFree，现在拥有数万名每日活跃用户。Acclaim 目前正在开发另一款脸书游戏，将于今年夏天推出。

Acclaim 将成为加入 Playdom 越来越多的游戏工作室和人才的最新团队。Acclaim 首席执行官霍华德·马克斯(Howard Marks)是动视 2.0 的前联合创始人和动视工作室的主席，他将负责 Playdom 的 Acclaim 工作室，并担任母公司的高级战略角色，专注于 Playdom 的业务发展活动。Acclaim 的首席技术官尼尔·马尔霍特拉(Neil Malhotra)现在将担任该工作室的高级技术官。

在过去的几个月里，Playdom 一直在疯狂购物。该公司最近聘请了一位新的首席技术官，一直在稳步扩大其在脸书的业务，最近收购了脸书游戏开发商 [Offbeat Creations](https://web.archive.org/web/20221206205725/https://beta.techcrunch.com/2010/03/03/playdom-acquires-facebook-game-developer-offbeat-creations/) 和[开发商 Three 甜瓜。Playdom 还向脸书游戏开发商 MetroGames 投资了 500 万美元。Playdom 刚刚收购了知名品牌游戏开发商](https://web.archive.org/web/20221206205725/http://www.prnewswire.com/news-releases/playdom-announces-acquisition-of-three-melons-89614502.html) [Merscom。](https://web.archive.org/web/20221206205725/https://beta.techcrunch.com/2010/04/26/playdom-continues-shopping-spree-acquires-social-game-developer-merscom/)

11 月，Playdom 以 2.6 亿美元的估值筹集了 4300 万美元的巨额资金，显然是用于收购公司和人才。根据我们 11 月的统计，Playdom 每月有 2800 万游戏用户。60%的流量来自 MySpace，40%来自脸书。Playdom 的主要竞争对手是游戏巨头 Zynga，它是脸书领先的游戏开发商。
