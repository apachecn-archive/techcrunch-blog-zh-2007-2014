# Quora Backlash 迎头痛击 Quora Backlash Backlash TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/01/31/quora-quora-quora-quora-quora-quora-quora/>

你知道我是怎么知道 Quora 会变大的吗？没人能对此闭嘴。

这包括喜欢它的人和讨厌它的人。这种二分法很重要，因为它会让人们一直谈论它。这会让人们继续注册。它会让那些已经注册的人继续回去。这一点很重要，因为 Quora 是一项需要比其他服务花费更多时间的服务。

无论如何，过去的几个周末带来了一些关于 Quora 的真正伟大的[恶言](https://web.archive.org/web/20230215134412/http://www.techmeme.com/search/query?q=quora&wm=false)。上周末，Vivek Wadhwa 在这个博客上发布了他的帖子[为什么我不买 Quora 炒作](https://web.archive.org/web/20230215134412/https://techcrunch.com/2011/01/23/why-i-don%E2%80%99t-buy-the-quora-hype/)。这篇文章在评论区和 Twitter 上都引起了激烈的反应(有正面的也有负面的)。事实上，在上周末贴出后的某个时刻，我发誓我的整个推特都在关注它。

当然，Quora 上有一个关于它的大帖子。

然后就出现了回应它的博客帖子。个人博客上的想法[也迅速跳过](https://web.archive.org/web/20230215134412/http://semilshah.posterous.com/a-quick-response-to-vwadhwa-post-on-quora)[到 TechCrunch](https://web.archive.org/web/20230215134412/https://techcrunch.com/2011/01/24/quoras-challenge/) 。天啊，我喜欢婊子。

但是这个周末，事情又有了新的进展。自然，罗伯特·斯考伯是催化剂。Scoble 今天写了一篇文章，题为[为什么我对 Quora 作为博客服务的看法是错误的……](https://web.archive.org/web/20230215134412/http://scobleizer.com/2011/01/30/why-i-was-wrong-about-quora-as-a-blogging-service/)如果你还没有读过，你应该读一读，哪怕只是为了了解一下[丹·卡普兰令人捧腹的反驳](https://web.archive.org/web/20230215134412/http://quorareview.com/2011/01/30/sorry-scoble-quora-is-not-your-playground/)的背景。

Wadhwa 在他的帖子中提出了许多好的观点，而 Scoble 似乎只是通过反驳自己最初的观点来发泄。如果你不想读，基本上，它可以归结为:一个版主埋葬了我花了很多时间精心制作的答案，这很烦人。

一方面，这很幽默。另一方面，这也是问题的核心。

Quora 不是一个博客平台。最初，我同意[斯科博在这件事上的一些独到见解。但是我显然误解了他们的意思，他认为 Quora 是整个博客生态系统下一步发展的一部分。](https://web.archive.org/web/20230215134412/http://scobleizer.com/2010/12/26/is-quora-the-biggest-blogging-innovation-in-10-years/)[我还是觉得是](https://web.archive.org/web/20230215134412/https://techcrunch.com/2010/12/28/quora-blogging/)。Quora 是一个很好的信息来源，就像 Twitter、脸书和博客本身一样。但显然，Scoble 实际上只是认为 Quora 才是博客的未来。也就是说，你应该在那里做。

[真是傻](https://web.archive.org/web/20230215134412/https://techcrunch.com/2011/01/30/quora-is-really-about-a-better-wikipedia-not-robert-scobles-hopes-dreams/)。

对我来说，Quora 首先是关于信息的。它是关于把它从人们的头脑中取出来，放入一个集中的储存库，当它与某些社会信号混合在一起时，就变成了一朵盛开的知识之花。虽然我确实认为对这项服务有发表意见的空间(在许多情况下，这是特别要求的)，但用户不应该以高度个人化和奇怪的促销方式继续下去。这正是斯考伯在他被否决的帖子中所做的(卡普兰正确地指出)。

这样做没问题——在你自己的博客上。事实上，这是完美的。但如果你把这类东西放在 Quora 上，仅仅因为你的 8000 名粉丝中有一定比例的人会投票支持它，就指望它被视为最权威的答案，那你就错过了 Quora 的意义。更多的是信息，而不是提供信息的人。这只是判断信息好坏的信号之一(尽管是最重要的初始信号)。

Scoble 的回答是否应该被隐藏起来？大概不会。实际上看起来不再是了。但是因为他有如此多的追随者投票支持他的答案，这确实掩盖了其他人的观点，这些观点也很好，也更切题。也许隐藏 Scoble 的答案是主持人认为最能确保其他答案被看到的方式。谁知道呢。说真的，谁在乎呢？当然，除了斯考伯。

再说一次，Quora 不是一个博客平台。从来都不是。当你反驳自己的误解时生气是很奇怪的。

这里真正的关键是每个人都无法停止谈论 Quora——不管是什么原因。在一个非常基本的层面上，这意味着什么。一些重要的事情。其他流行的服务也分享了这一点。

在更基本的层面上，这意味着一大堆媒体。关于 Quora 出版社的新闻。现在甚至有关于 Quora 出版社的报道。不管你会在下面的评论中读到什么(如果你敢去那里)，这是你买不到的那种媒体。这是自然发生的，因为人们对你的网站感兴趣。既有想写它的人，也有想读它的人。也就是说，当他们不忙着使用它的时候。
