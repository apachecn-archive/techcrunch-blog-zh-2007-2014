# 世界欢迎梦幻豹猫:Ubuntu 11.10 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/10/the-world-welcomes-oneiric-ocelot-ubuntu-11-10-launched/>

自从 2004 年第一次发布以来，Ubuntu Linux 发行版已经走过了漫长的道路。它最初是一个包装精美的 Linux 桌面，由一组特定的包构建而成，这些包是从 Debian 发行版中的近三万个可用包中培育出来的。六个月一次的定期发布确保了 Ubuntu 将永远接近 Linux 和自由软件开发的前沿。每四个版本中就有一个是长期支持产品，它将获得三年的安全和支持更新。在过去的七年里，Ubuntu 的主要商业赞助商 Canonical 增加了 Ubuntu 的服务器版本，建立了跨平台云存储解决方案 [UbuntuOne](https://web.archive.org/web/20230205024747/https://techcrunch.com/2011/09/30/canonical-releases-windows-version-of-ubuntu-one/) ，并在[云计算](https://web.archive.org/web/20230205024747/http://cloud.ubuntu.com/)方面取得了长足的进步。

惠普最近[宣布](https://web.archive.org/web/20230205024747/http://cloud.ubuntu.com/2011/10/ubuntu-powers-hp-public-cloud/)他们已经选择 Ubuntu 来驱动惠普公共云；而印度最高法院最近已经[将](https://web.archive.org/web/20230205024747/http://efytimes.com/e1/70691/fullnews.htm)从红帽企业 Linux 切换到 Ubuntu 但是 Ubuntu 开发的面包和黄油仍然是他们的桌面产品。代号为 Oneiric Ocelot 的 11.10 版本将于 10 月 13 日周四正式发布，这是 Canonical 的最新版本，其中包含了许多有趣的迭代。

Oneiric Ocelot 将 Ubuntu 软件中心发展成为 Ubuntu 世界的一等公民。软件中心的外观和感觉已经更新，使其与整体 Ubuntu 美学保持一致。同时更新的还有可用应用程序的数量和种类，包括商业应用程序。

Canonical 一直在努力让出版社意识到软件中心，他们已经吸引了一些不错的标题。他们正在努力建立一个更令人印象深刻的标题选择，并尽可能方便用户找到他们想要的应用程序。

同时，Canonical 正在通过他们的[developer.ubuntu.com](https://web.archive.org/web/20230205024747/http://developer.ubuntu.com/)门户吸引应用程序开发人员，该门户解释了如何构建应用程序并将其提交给软件中心。Canonical 报告称，自开发者门户发布以来的三周内，应用提交量增加了 5 倍。据 Canonical 的通信主管 Gerry Carr 称，他们现在每天至少批准一个应用程序。

开发者平台和更新的软件中心背后的动机是提高用户可用应用的质量。还有一些工作是通过 Ubuntu Pay 让应用程序货币化，从而让开发者受益。鉴于“啤酒免费”软件的悠久历史，Linux 用户愿意为应用程序付费的程度还有待观察。虽然只有一个数据点，但我认为[简陋的捆绑包](https://web.archive.org/web/20230205024747/http://www.humblebundle.com/)提供了一些有价值的见解，我敢打赌，在软件中心可以赚钱:Linux 用户为简陋的捆绑包产品*花费的平均价格总是*超过 Windows 或 Mac 用户。当前简陋的捆绑包中，Linux 用户支付的费用是普通 Windows 用户的两倍。

Ubuntu 的最新版本 11.04 增加了 [Unity](https://web.archive.org/web/20230205024747/http://unity.ubuntu.com/) ，这是为现代硬件复兴桌面 Linux 的努力，同时为基于触摸的界面开发有意义的体验。它遇到了一些敌意，因为它是许多用户已经习惯的桌面范例的一个显著变化。Ubuntu 11.10 对整体体验进行了一些改进，但很明显 Unity 将继续存在。

我问 Gerry Carr 关于从传统的 GNOME 桌面到 Unity 的 Dash 模式的实质性转变。卡尔同意这可能是一个不和谐的转变，因为它打破了我们长大后使用的许多隐喻，感觉像是永远，但我们使用计算机的方式正在改变，界面需要适应。我们越来越多地使用计算机来访问互联网资源，因此界面应该以有意义的方式促进这一点。您应该能够访问的不仅仅是系统上的内容。

卡尔指出，镜头机制支持异构存储，这意味着你可以将数据保存在有意义的地方，但仍然可以通过一致、易用的界面访问数据。“Ubuntu 体验既是丰富的本地体验，也是丰富的云体验，”卡尔说。

这种整合的一个例子就是 Dash 本身。当搜索应用程序时，各种镜头会找到您当前安装的应用程序，并建议安装可用的应用程序。这完全避开了停止你正在做的事情，打开软件中心之类的东西，搜索一个应用程序，最后安装它的旧模式。通过将应用搜索直接集成到 Dash 中，Ubuntu 提供了一个单一的界面来减少用户体验的摩擦。

公平地说，这仍然是一个不断发展的解决方案。对于习惯了自己生活方式的人来说，过渡到 Unity 体验可能会令人沮丧。我认为 Canonical 迄今为止的一个大缺点是在解释镜头隐喻的价值以及帮助人们认识到它的效用方面缺乏真正的教育努力。

除了抱怨，Canonical 一直在努力制造能让 Ubuntu 体验更好的镜头。有一个新的音乐镜头，允许用户搜索音乐。music lens 将搜索用户的本地音乐文件，并查询 Ubuntu 音乐商店，允许用户快速识别本地收藏中可能缺少的专辑，并快速轻松地购买这些音乐。这是 lens 系统的本地+互联网搜索能力的完美融合。

所有搜索——无论是应用、文件、媒体还是其他——都支持丰富的上下文相关过滤。搜索文件时，可以按文件类型、大小或上次修改日期进行过滤。搜索媒体时，您可以按年代或流派进行过滤。搜索应用程序允许您按类别或评级进行过滤。这些都不是特别革命性的，但 Ubuntu 11.10 将搜索和过滤功能集成到镜头中的轻松程度令人印象深刻。

镜头也不一定来自 Canonical:有丰富的 API 来开发自己的镜头。一个很好的例子就是 [Ask Ubuntu lens](https://web.archive.org/web/20230205024747/https://launchpad.net/askubuntu-lens) 。这将对 StackExchange 支持的 AskUbuntu.com 网站的搜索集成到您的桌面上，允许您搜索您可能遇到的问题的答案。你不需要打开浏览器并导航到一个网站:你只需使用 Unity 界面进行
搜索，就像你搜索要安装的新应用或要听的新音乐一样。

Ubuntu 11.10 的另一个巧妙之处是 [Deja Dup](https://web.archive.org/web/20230205024747/https://launchpad.net/deja-dup) ，这是一个工作起来像苹果时间机器的备份应用程序。备份可以存储在您的电脑硬盘、外置硬盘或您的 UbuntuOne 帐户中。将你的备份存储到你的 UbuntuOne 账户，可以让你将系统恢复到不同的硬件上，这是一个很好的功能。同样值得注意的是，Deja Dup 不是一个规范的项目:Deja Dup 构建在 UbuntuOne API 之上，为 Ubuntu 用户无缝地工作。

对于任何过去使用过 Ubuntu 的人来说，你可能会高兴地得知 11.10 抛弃了进化，支持 Mozilla Thunderbird 作为默认的邮件客户端。当我问卡尔这种转变的动机时，他立即回答“用户需求”。显然用户对 Evolution 不满。卡尔还指出，更多的开发人员的努力正在应用于雷鸟，使它成为一个容易安装的选择。

Ubuntu 11.10 是下一个长期支持版本之前的最后一个版本，所以我们周四将看到的大部分内容是为下一个版本的内容奠定基础。像 Ubuntu 这样的自由软件的伟大之处在于，你现在就可以使用 11.10 版本*:只需按照[这些说明](https://web.archive.org/web/20230205024747/https://wiki.ubuntu.com/OneiricOcelot/TechnicalOverview/Beta2#Upgrading_from_Ubuntu_11.04)。同样，您可以在 12.04 版本正式发布之前很久就安装它，并且您可以通过提交错误报告来帮助确定该版本的方向！*

 *我一直在定期将我的笔记本电脑升级到最新版本的 Ubuntu，我从未真正失望过。升级到 11.04 一开始有点令人不安，但我用得越多，它就越不困扰我。随着 Canonical 致力于改进 Unity 的镜头机制，以及第三方镜头的激增，我希望我会更喜欢使用 11.10。*
