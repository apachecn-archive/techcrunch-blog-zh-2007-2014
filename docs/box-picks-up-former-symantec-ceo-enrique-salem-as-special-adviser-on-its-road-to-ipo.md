# Box 聘请前赛门铁克首席执行官恩里克·塞勒姆担任其 IPO 之路的特别顾问

今天，Box [宣布](https://web.archive.org/web/20221003083104/http://blog.box.com/2014/02/enrique-salem/)在这个房间里又找到了一个成年人，增加了它的领导深度，因为它正在向一个广泛泄露的、即将到来的首次公开募股倾斜。赛门铁克前首席执行官恩里克·塞勒姆将担任这家不断发展的公司的特别顾问。

Box 的首席执行官 Aaron Levie 表示，Salem 将帮助该公司“创新和推进云安全的新模式”。

为企业级业务提供云存储产品的 Box，最近筹集了另一笔 1 亿美元的[资金。它的主要竞争对手 Dropbox 最近](https://web.archive.org/web/20221003083104/https://beta.techcrunch.com/2013/12/05/box-confirms-that-it-has-raised-100m-to-help-accelerate-global-expansion/)[又获得了 3 . 5 亿美元](https://web.archive.org/web/20221003083104/https://beta.techcrunch.com/2014/01/18/dropbox-grows-up/)。据广泛报道，Box 已经提交了一份尚未公开的公开募股申请，应该会在今年进行。

尽管给 Box 目前的高管团队增加高级助手是个好主意，但塞勒姆是个有趣的选择。当然，他有给 Box 提供建议的经验，但是他从赛门铁克的退出并不是特别令人愉快。当他被解职的消息公布后，赛门铁克的股价上涨了 13%以上。为什么他被扫地出门？正如硅谷商业杂志[在当时](https://web.archive.org/web/20221003083104/http://www.bizjournals.com/sanjose/news/2012/07/25/symantec-stock-jumps-after-ceo-enrique.html)写道:

> [新任首席执行官史蒂夫]贝内特在一份准备好的声明中表示，此举是因为数据安全公司进展不够快，表现不佳。

这并不是说塞勒姆不会有效地帮助 Box，但这是一个有趣的背景。

根据白宫网站，塞勒姆是奥巴马总统管理顾问委员会的成员。最后，Salem 在一月份加入了 ForeScout 的董事会。
