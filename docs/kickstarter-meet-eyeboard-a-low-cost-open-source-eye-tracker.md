# Kickstarter:认识一下 Eyeboard，一个低成本的开源眼睛追踪器

我有点惭愧地承认，我在高中生涯的末期游手好闲，拼命想让自己看起来比实际上更酷。对我们的未来来说幸运的是，像路易斯·克鲁斯这样的人是存在的:这位最近的高中毕业生设计了[eye board](https://web.archive.org/web/20221007141610/http://www.kickstarter.com/projects/lcruz/the-eyeboard-0?ref=live)，这是一种低成本、开源的眼球追踪解决方案，旨在让残疾用户更容易沟通。

虽然许多著名的眼球追踪解决方案使用摄像头来监控用户的眼球运动，但额外的硬件意味着对于最可能使用它们的人来说，它们的成本可能过高。

Cruz 的 Eyeboard 采用了一种不同的方法:用户戴上一副内置一组电极的眼镜，外部电路板(由 ATMega328 微控制器供电)测量与他们的运动对应的视网膜静态电位的变化。这不是最精确的方法，但 Cruz 设计的简单性意味着吊环比更精确的替代方案要便宜得多。

该板运行到 PC 中，然后 Eyeboard 软件(Windows 和 Linux 版本都有)将这些电压读数解释为运动输入。最终结果？用户只用眼睛在屏幕上输入信息的一种廉价方式。正如我提到的，Cruz 的项目是完全开源的，因此有特定用例的人可以根据需要随意修改他的代码。

眼板原型已经流传了几个月了——他的工作在 CNN 的[节目中得到了强调——但是现在到了实际批量生产眼板套件的棘手部分。为此，克鲁兹转向 Kickstarter 寻求支持:虽然他的早期原型造价约为 200 至 300 美元，但感兴趣的投资者可以花 95 美元锁定第一轮 Eyeboard 套件。这肯定是一个利基产品，但如果幸运的话，这些东西不久就会在资金不足的学校和发展中国家开始使用。](https://web.archive.org/web/20221007141610/http://www.youtube.com/watch?v=r8d6j8b3sGI)
