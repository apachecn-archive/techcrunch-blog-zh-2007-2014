# 这就是我们跟你说过的脸书的“人人按钮”

![](img/a67cdde369ff6057466f899e363139d4.png)

![](img/ecfeee39860d909da5138bd0d3056e73.png)

第四阶段已经开始。[上周，我们告诉你](https://web.archive.org/web/20230406012817/https://techcrunch.com/2009/06/17/phase-4-of-facebooks-systematic-attack-on-twitter-the-everyone-button/)脸书新的“人人按钮”,让选择至少部分保密个人资料的用户偶尔向公众发送状态信息或其他内容。这项功能与[最近推出的公共状态信息搜索引擎](https://web.archive.org/web/20230406012817/https://techcrunch.com/2009/06/16/facebook-fixes-search-but-only-in-beta/)齐头并进，今天[向一小部分用户推出](https://web.archive.org/web/20230406012817/http://blog.facebook.com/blog.php?post=98499677130)。

不要被用户界面上看似微小的变化所迷惑。这是脸书极度渴望让用户尽可能公开他们的数据的又一个迹象。就像推特一样。

Publisher 的这一新版本允许用户在您的主页和个人资料中添加照片、视频和状态更新等内容。每一条内容都可以分享给所有人，朋友和网络，朋友的朋友和朋友的朋友。用户还可以进一步自定义设置。

现在，当您在网站上添加任何内容时，您可以控制要通知谁您的脸书朋友。它是这样工作的:在从发布者写下状态、上传照片或创建其他内容后，使用发布者右下角的锁图标来访问下拉菜单。在那里，你可以选择让每个人、朋友和某些网络、朋友的朋友以及自定义列表都可以看到这篇文章。
