# 移动广告网络 Mobclix 发布 iPad SDK

![](img/573993566d15ff514e0ca6477e3dd0c9.png)面对来自最近收购的 [AdMob(谷歌)](//web.archive.org/web/20221206213752/https://beta.techcrunch.com/2009/11/09/google-acquires-admob/)和 [Quattro Wireless(苹果)](https://web.archive.org/web/20221206213752/https://beta.techcrunch.com/2010/01/04/apple-acquires-quattro-wireless/)移动广告网络 [Mobclix](https://web.archive.org/web/20221206213752/http://www.mobclix.com/) 的竞争，Mobclix 一直在稳步加强其产品和服务。这家初创公司刚刚收购了 iPhone 应用销售分析软件 [Heartbeat](https://web.archive.org/web/20221206213752/https://beta.techcrunch.com/2010/03/01/mobclix-acquires-iphone-app-sales-analytics-software-heartbeat/) 和[宣布](https://web.archive.org/web/20221206213752/http://gigaom.com/2010/02/08/mobclix-and-nielsen-ink-mobile-ad-targeting-data-deal/)一项交易，将尼尔森的广告定位数据提供给其广告交易所，以增强广告定位。今天， [Mobclix](https://web.archive.org/web/20221206213752/http://www.mobclix.com/) 正式[发布](https://web.archive.org/web/20221206213752/http://blog.mobclix.com/2010/03/15/mobclix-for-ipad/)首批用于 [iPad](https://web.archive.org/web/20221206213752/http://www.apple.com/ipad/) 广告的 SDK 之一。

苹果的新平板设备将在几周内交付给消费者，这为广告创造了巨大的机会。平板电脑的尺寸不仅允许更大的广告位，而且广告也可以更具吸引力和媒体丰富性，有可能融入视频和其他互动功能。这对广告商和开发商来说意味着更多的收入。

Mobclix [允许应用开发者](https://web.archive.org/web/20221206213752/https://beta.techcrunch.com/2009/08/03/mobclix-takes-on-admob-by-roping-together-20-mobile-ad-networks-into-an-exchange/)注册他们的广告清单，广告网络根据年龄、性别、位置和其他因素竞标广告位。提供的广告会根据哪个广告网络出价最高来自动改变，以吸引特定应用的用户。它还允许广告商根据人口统计、地理定位和行为特征购买各种应用。

Mobclix 将其 iPhone SDK 拆开，并完全为 iPad 重新构建了一个新的 SDK。这家初创公司表示，新的 iPad 广告机会意味着更高的 eCPMs 和更多的广告尺寸选择。目前，Mobclix 合作伙伴 Traffic Marketplace 和 Valueclick 正在为 iPad 提供 300×250 的广告单元。但 Mobclix 将在 4 月份推出 468 x 60、120 x 600、728 x 90 格式。

当然，看看苹果自己的移动广告网络 Quattro Wireless 将如何在 iPad 的广告生态系统中发挥作用应该是很有趣的。虽然还不清楚苹果对 Quattro 有什么计划，但我们知道苹果看好移动广告，并且可能会在 T2 为开发者开发自己的移动广告产品。Quattro Wireless 是 Mobclix 交易所包含的网络之一。
