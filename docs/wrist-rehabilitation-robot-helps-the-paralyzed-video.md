# 手腕康复机器人帮助瘫痪者(视频)

今天，我们可以向你展示的来自日本的新型医疗机器人不仅仅是抬起病人的 RIBA II 。冈山大学[的研究人员](https://web.archive.org/web/20230203110350/http://www.okayama-u.ac.jp/index_e.html)开发了一种不同方向的模型:他们的[机器人](https://web.archive.org/web/20230203110350/http://mcrlab.mech.okayama-u.ac.jp/study_parallel_rehabili.htm)【JP】帮助脊髓损伤或中风康复的人。

这个想法是帮助病人再次活动手腕，这项任务通常由物理治疗师来完成。机器人的工作始于大约六年前，最新版本有六个关节和一个带致动器的气压缸。

患者可以在各种阻力水平之间进行选择后，使用机器人“练习”移动他们的手腕。最酷的是，一组传感器可以连接到患者手臂的肌肉上，模拟关于手腕运动的力反馈。
