# 清楚:为什么这个简单的任务列表应用让所有人都在谈论它

备受期待的基于触摸的待办事项应用 Clear 将于今晚在 iTunes 应用商店发布。我说的备受期待是指这款应用获得了基于演示、预览和预告视频的科技博客报道。

为什么一个通常相当乏味的应用类别——最低级的待办事项列表——如此吸引人？首先， [Clear](https://web.archive.org/web/20230306214257/http://itunes.apple.com/us/app/clear/id493136154?ls=1&mt=8) 是纯粹的视觉享受。但它也代表了智能手机应用程序设计的一次重大飞跃，因为它是为触摸界面而从头开始构建的。这款应用完全基于现在常见的手势:刷、拉和捏。没有 Clear 按钮，但是使用起来非常简单。事实上，这才是重点。

如果你当前的待办事项应用需求很复杂，你可能不适合 Clear。但如果你经常求助于一个简单的待办事项应用程序，甚至是 iPhone 内置的记事本来制作你的清单，Clear 肯定会让你惊叹不已。

[<amp-img class="alignright size-medium wp-image-497298 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="3" src="https://web.archive.org/web/20230306214257im_/https://techcrunch.com/wp-content/uploads/2012/02/3.png?w=234" alt="" srcset="https://web.archive.org/web/20230306214257im_/https://techcrunch.com/wp-content/uploads/2012/02/3.png 631w, https://web.archive.org/web/20230306214257im_/https://techcrunch.com/wp-content/uploads/2012/02/3.png?resize=117,150 117w, https://web.archive.org/web/20230306214257im_/https://techcrunch.com/wp-content/uploads/2012/02/3.png?resize=234,300 234w, https://web.archive.org/web/20230306214257im_/https://techcrunch.com/wp-content/uploads/2012/02/3.png?resize=531,680 531w, https://web.archive.org/web/20230306214257im_/https://techcrunch.com/wp-content/uploads/2012/02/3.png?resize=39,50 39w" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230306214257/https://techcrunch.com/wp-content/uploads/2012/02/3.png)这款应用的独特之处在于它放弃了常见的导航元素——比如位于屏幕底部或顶部的按钮——而支持全手势界面。如果你不知道如何滑动和挤压，你可能会在第一次启动时迷失一秒钟(呃，妈妈)。但 Clear 的设计旨在利用智能手机用户中已经存在的常识:你可以在屏幕元素上滑动、挤压和下拉以进行交互。谁需要纽扣？

在某个地方，史蒂夫·乔布斯，一个不喜欢按键的人(很明显，看看 iPhone，只有一个)正在微笑。

使用 Clear，您只需要几个手势就可以使用该应用程序:下拉列表以添加项目，向右滑动以完成项目或向左滑动以删除项目，分开两个项目以在中间插入新项目，垂直滑动以关闭当前列表并查看应用程序中的所有列表。列表还用热图进行颜色编码，一目了然地显示最紧迫的任务。(见下面视频)。

就是这样。这是一个快速的学习曲线，更重要的是，一旦你意识到这是如何完成的，取消按钮实际上可以加快使用待办事项列表的过程。Clear 甚至推动你简化这里的内容，将待办事项限制在 30 个字符以内。

“有太多的垃圾试图在其他待办应用程序中吸引你的注意力，你甚至都懒得在 iPhone 应用程序版本上输入东西，”该应用程序的联合创始人 Phill Ryu 说，“Clear 的速度如此之快，有时甚至很有趣。”

但是这个世界准备好接受一个无按钮的应用了吗？

“是的！”Ryu 说，“你见过婴儿玩 iPad 吗？他们喜欢在屏幕上一对一地直接滑动和操纵东西。在触摸屏设备上，没有什么比这更自然的了。”

他说得有道理。这个周末，我惊讶地看着一个一岁半的孩子打开 iPad，点击一个文件夹，启动网飞，浏览队列，推出他最喜欢的卡通。令人瞠目结舌，真的。Clear 是一款专为他设计的应用，也是为伴随手势长大的新一代智能手机用户设计的。对于那些还记得窗口和其他像“菜单”和“按钮”这样陈旧的用户界面概念的人来说，忘记向后兼容性吧面向未来的设计。你怎么能不喜欢这个主意呢？

这个应用程序本身也有一个有趣的背景。这是一个由 [Realmac 软件](https://web.archive.org/web/20230306214257/http://www.realmacsoftware.com/company)、[宇宙机器和 Clean Cut Code 的创始人米伦·朱梅罗夫](https://web.archive.org/web/20230306214257/http://milen.me/)和一个名为[即将到来公司](https://web.archive.org/web/20230306214257/http://www.impending.com/)的新工作室联合开发的项目。这是由 tap tap tap 的合伙人，上面提到的 [Phill Ryu](https://web.archive.org/web/20230306214257/http://www.linkedin.com/pub/phillip-ryu/13/a69/727) 和[icon factory](https://web.archive.org/web/20230306214257/http://dlanham.com/about/)的[大卫·拉纳姆](https://web.archive.org/web/20230306214257/http://iconfactory.com/)创办的《迫近》的第一次发布。在即将到来之前，该团队已经参与了许多高调和受欢迎的应用程序，包括 Twitterific，MacHeist，Classics 等。这是一个不错的方式来开始你的工作室的首次亮相。

在过去的一年半时间里，悬空还在进行另一个项目，但目前还没有谈论细节。然而，他们正与一个非常有才华的团队合作，包括一些“通常的嫌疑人”是的，我们可能也会对这个应用程序着迷。
