# 脸书身份证是我们的未来吗？

脸书已经申请了商标，在名片上使用“脸书”,更奇怪的是，在其他事物中使用“非磁性编码”身份证。如果获得授权，该商标将保护在特定格式中使用的“脸书”一词，而不是任何实际的发明。

那么，如果脸书只是想阻止人们制造假的脸书名片呢？好吧，看起来这个商标会涵盖更多，包括“名片和身份证设计服务”、“打印服务”和不祥的“通过提供用于自己的名片和身份证的数据来促进社交和商业网络。”

看起来该商标还将涵盖二维码和 NFC/RFID 的使用——这是通过磁感应而不是前述的磁编码来工作的——很像让你[在 F8](https://web.archive.org/web/20230204124129/http://www.slashgear.com/guidebook-app-for-facebook-f8-2011-attendees-hands-on-video-21181783/) 上传和标记照片的存在卡和照相亭(见左)。

很容易想象某种脸书总体计划，脸书会给用户一个便宜的物理 ID，可以被智能阅读器读取并用于各种实际用途。当被问到时，熟悉脸书事件的人不知道这是否真的在计划中。同样不清楚的是，像脸书这样的公司有多少次注册了商标，但实际上并没有利用这个商标。

如果脸书要开发某种实体身份识别系统，这对市场营销来说是一件好事，而且非常实用；想象一下，去听音乐会或看电影，在脸书买票，刷钥匙链身份证。

那么，脸书会在我们未来如何管理线下身份方面发挥更大的作用吗？这个想法并不牵强——毕竟，脸书已经是互联网上最主要的身份系统。
