# 查德·赫利认为 Delicious 的“价值是它的品牌(而不是它的技术)”，将利用它来“重启创新”

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/06/20/chad-hurley-thinks-delicious-value-is-its-brand-not-its-tech-will-use-it-to-restart-innovation/>

YouTube 联合创始人查德·赫利去年通过 AVOS 从雅虎手中收购了[Delicious，他今天表示，这次收购对他和他在 YouTube 和新公司的合作伙伴陈士骏来说是一次“有趣的实验”:他们想看看他们是否能利用一个(仍然)强大的品牌作为新创新的载体——超越作为 Delicious 今天所知的核心的聚合和书签服务。](https://web.archive.org/web/20221208004313/https://beta.techcrunch.com/2011/04/27/yahoo-sells-delicious-to-youtube-founders/)

早些时候在伦敦举行的 [LeWeb](https://web.archive.org/web/20221208004313/http://leweb.co/) 会议上，他与谷歌风投的凯文·罗斯和 Atomico 的尼可拉斯·曾斯特罗姆在一个小组上发言，Hurley 说:“我们将重启美味品牌的创新。”因此，这个想法是利用一个现有的品牌，而不是“从零开始创造一个新的品牌”来推行新的想法——大量的小想法。“我知道建立一个品牌有多难，所以这是一个有趣的实验。”

他没有详细说明这些新服务到底是什么，但他正在进行的另一个项目 Zeen 可能会指出他的意思。 [Zeen](https://web.archive.org/web/20221208004313/http://zeen.com/) 尚未上线，据其网站称，这将是人们创建自己的“美丽杂志”的一种方式——这听起来有点像雅虎的 Livestand 尝试和 Flipboard。赫尔利指出，他的公司的 40 名开发人员目前分布在三个地方，中国(北京)、新西兰和圣马特奥。

不过，这并不意味着他们打算放弃 Delicious 已经在做的事情:例如，他们最近推出了 Delicious 的中文版。

在迈克尔·阿灵顿主持的小组讨论中，这三位企业家/投资者还谈到了他们今天的投资目标，以及他们正在关注的科技领域的一些大趋势:

**谈当今国际扩张的作用。创立 Skype 并先后卖给易贝和微软的尼可拉斯·曾斯特罗姆说:“迅速进行国际扩张非常重要，因为在线市场是全球性的，如果你不加快步伐，就会遇到模仿者和其他竞争者的问题。”**

**论支持模仿者。** Zennstrom 又来了:“进化和复制是有区别的。Skype 没有发明网络电话，但我们让它变得更容易使用。但我们有很多公司抄袭我们。”

从企业家到投资人的转变。凯文·罗斯说，看到其他企业家追求创意，他也能感受到同样的“激情”,他在他支持的公司中有着惊人的业绩记录，包括 Zynga、Twitter、Foursqure、Chomp 和 Square，用阿灵顿的话说就是“热门游行”。“对我个人来说，我有点厌倦创业，”罗斯承认。

**下一件大事？** Rose:“我认为 Apple TV 有很多商机，”他说既然我们知道了尺寸和其他规格，人们现在就可以开始考虑解决方案了。他认为其中一个重点领域是电子商务。“看看今天的 QVC，在电视上的应用程序中添加了游戏组件，并且能够购买东西。”那是一家潜力巨大的公司。我不想做，但我想投资它。"

Zennstrom 呼应了[其他人最近也注意到的一种观点](https://web.archive.org/web/20221208004313/https://beta.techcrunch.com/2012/06/17/make-way-for-more-tech-investment-index-raises-another-e350m-fund/)，即我们正处于技术创新的平静期，现在更多的是丰富产品。“十年前我们专注于技术，但现在我们专注于产品。今天，人们专注于解决问题，而不是技术问题，而是围绕让事情更容易使用的问题，在医疗保健和教育、生命科学等领域……从大数据的角度来看，这些都很有趣。”

赫尔利说，下一件大事将是让人们以被动的方式保持参与的产品:“今天的技术对人们要求太多了，”他说。“我认为在未来几年内，事情将变得更加被动，无论是为健康应用程序还是社交网络收集数据……以更好地了解我在做什么，而不是我的朋友在做什么。”

罗斯还在列表中添加了对移动的持续关注:“我们每天使用的应用程序屈指可数。他们将来会是什么样的人？”他指出，对于移动交易，日常应用程序仍有空间取代现金。

在如今的必备应用中，Zennstrom names email、calendar 和 address book 是三个。“但这些仍然是缺乏创新的东西。”
