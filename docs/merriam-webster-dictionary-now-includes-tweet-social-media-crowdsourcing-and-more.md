# 韦氏词典现在包括“推特”、“社交媒体”、“众包”等等

韦氏词典大学词典已经更新，今年收录了 [150 个新词和新定义](https://web.archive.org/web/20230203043612/http://www.merriam-webster.com/info/newwords11.htm)，其出版商[今天早上宣布](https://web.archive.org/web/20230203043612/http://www.prnewswire.com/news-releases/something-to-tweet-about-128379408.html)。

四个与技术相关的新词是[【tweet】](https://web.archive.org/web/20230203043612/http://www.merriam-webster.com/dictionary/tweet)[【社交媒体】](https://web.archive.org/web/20230203043612/http://www.merriam-webster.com/dictionary/social%20media)[【众包】](https://web.archive.org/web/20230203043612/http://www.merriam-webster.com/dictionary/crowdsourcing)和[【移动商务】](https://web.archive.org/web/20230203043612/http://www.merriam-webster.com/dictionary/m-commerce)，这些词对本博客的读者来说应该已经很熟悉了。

事实上，根据韦氏词典的说法,“tweet”一词的首次使用可以追溯到 1768 年。显然，杰克·多西的祖先是真正的预言家。

说真的，我不禁想知道，是什么让出版商花了这么长时间才把上述单词加入词典。韦氏词典的自由编辑彼得·索科洛夫斯基在一份声明中解释道:

> “当然，我们多年来一直在跟踪社交媒体和推特这样的词，现在我们觉得它们的意思已经足够稳定，可以纳入字典了。”

根据该公司的说法，其他含义足够稳定的新词包括 bromance(“男人之间亲密的无性友谊”)、fist bump(感谢巴拉克·奥巴马)、cougar(“一个中年妇女寻求与年轻男子的浪漫关系”)和直升机父母(“一个过度参与孩子生活的父母”)。

Retweet、lulz、trollface、pedobear 和 philosoraptor 还没有入围。
