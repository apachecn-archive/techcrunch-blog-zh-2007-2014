# Zynga 收购你从未用过的社交浏览器 Flock 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/01/05/zynga-to-acquire-flock-the-social-browser-that-you-never-used/>

我们已经证实，Zynga 正在收购 [Flock](https://web.archive.org/web/20230202230536/http://www.flock.com/) ，这是一家陷入困境的初创公司，由 [Bart Decrem](https://web.archive.org/web/20230202230536/http://www.crunchbase.com/person/bart-decrem) 和 [Geoffrey Arone](https://web.archive.org/web/20230202230536/http://www.crunchbase.com/person/geoffrey-arone) 于 2005 年创立。这笔交易应该很快就会宣布。

该公司已经筹集了近 3000 万美元的风险投资，尽管最后一轮已于 2008 年结束。我们不知道收购的条款。我们从一个消息来源获悉，谷歌和 Twitter 也在竞购 Flock——或许是为了获得 Flock 备受推崇的工程人才。

Flock 在 2005 年首次推出了它的社交浏览器，我们[在那年的 8 月](https://web.archive.org/web/20230202230536/https://techcrunch.com/2005/08/26/flock-social-browsing-is-cool/)写了关于它的报道。当时，在 Twitter 和脸书崛起之前，“社交”意味着写博客和社交书签，除此之外别无其他。弗洛克并不成功。

它挣扎了几年，最终[从使用 Mozilla 转向使用 Chromium 作为它的核心。但是用法从来没有真正出现过。现在他们在一个从未被证明存在的市场上有了一个强大的竞争对手。](https://web.archive.org/web/20230202230536/https://techcrunch.com/2010/06/16/flock-switches-from-mozilla-to-chromium-for-new-browser-but-is-that-enough/)

这是 Zynga 在几个月内的第八次收购:北京的 XPD(5 月)，奥斯汀的挑战游戏(6 月)，东京的 Unoh 游戏(8 月)，波士顿的 Conduit Labs 月)，法兰克福的 Dextrose AG(9 月)，达拉斯的 Bonfire Studios(10 月)和得克萨斯州 McKinney 的 Newtoy Inc .(12 月)。大量的收购和大量的招聘。
