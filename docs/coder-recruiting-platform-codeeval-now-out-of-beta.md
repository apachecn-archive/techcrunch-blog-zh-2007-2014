# 程序员招聘平台 CodeEval 现已退出测试

![](img/be79c64ce756f47ed6d8ff0601f5ce42.png)
正好赶上即将到来的硅谷[人才启示录](https://web.archive.org/web/20230131032804/https://techcrunch.com/2011/06/07/a-look-at-whos-winning-the-talent-wars-in-tech-hint-it-rhymes-with-twitter/)、 [CodeEval、](https://web.archive.org/web/20230131032804/http://www.codeeval.com/)一个利用编程挑战来帮助招聘人员寻找和筛选最佳编程候选人的平台，今天向公众推出。

当技术招聘人员实际上不是技术人员时，CodeEval 会进行一些猜测，只将回答正确的候选人推给客户(并进入下一个筛选阶段)。创始人吉米·约翰说:“把它想象成一个真正高效的求职平台，所有的候选人都经过预先筛选。”

在与 CoderLoop 相同的领域，CodeEval 与竞争对手不同的是，它允许雇主创建自己的编程挑战，并提供广泛的语言支持(10 种不同的开源语言)。

目前，公司和招聘人员可以通过免费增值模式使用 CodeEval，基本功能免费提供，更广泛的功能(如无限制的应用程序)现在每个人都可以通过每月 199 美元和每月 399 美元的套餐获得。

CodeEval 位于旧金山，是 i/o Ventures 孵化器的一部分。约翰告诉我，他未来的创业计划包括进一步的语言和挑战自动化支持，以及有一天将平台扩展到设计师筛选。
