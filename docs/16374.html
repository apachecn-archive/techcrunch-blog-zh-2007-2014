在接到投诉后，微软向开发者提供了Windows 8.1 RTM版本

原文：https://web.archive.org/web/https://techcrunch.com/2013/09/09/following-complaint-microsoft-makes-the-windows-8-1-rtm-build-available-to-developers/

在接到投诉后，微软向开发者提供了Windows 8.1 RTM版本

微软今天宣布，Windows 8.1 RTM build对MSDN和TechNet订户开放，比普通公众提前了大约一个月。此前，该公司曾表示，在该代码正式发布之前，开发人员无权访问该代码。

这一决定遭到了开发人员的公然蔑视和愤怒，他们希望在即将到来的Windows 8更新的最终版本中测试他们的代码。

Windows 8.1包含了一系列关键的功能改进、用户界面升级和其他好处，使核心的Windows 8体验更受欢迎，如果我可以的话，更令人愉快。该更新将通过Windows商店免费提供给所有Windows 8用户。

今天发布的还有Visual Studio 2013发布候选版本和Windows Server 2012 RTM版本。所以，如果你在微软的平台上构建代码，今天对你来说是个好日子。

值得注意的是，微软在向硬件制造商发布Windows 8.1代码时犯了一个错误，没有向开发者发布，但该公司很快修复了这个错误。过去的微软不会做这样的事情。当然，最好不要犯这个错误，但是我们现在知道微软已经竖起了耳朵。

在一篇博客文章中，微软表示，它既在“倾听”，也“明白了”。该公司还承诺改变时间表，以更快的速度交付代码，因此这种情况不会再次发生。你还在等什么？去做测试。
