# YouTube 辩护:维亚康姆“秘密上传”内容，他们试图购买我们

![](img/180a37a1c5b711ea8cac855ecd2e42ab.png)

今天早些时候，在维亚康姆对 YouTube 的长期版权侵权诉讼中，几份先前密封的法律文件被公之于众。在公开发布这些文件的同时，YouTube 的首席法律顾问 Zahavah Levine 写了一篇[博客文章](https://web.archive.org/web/20221207113751/http://youtube-global.blogspot.com/2010/03/broadcast-yourself.html)，读起来更像是一份法律简报的摘要。

在其中，莱文概述了 YouTube 对维亚康姆指控的主要辩护，包括维亚康姆“秘密地将其内容上传到 YouTube，甚至在公开抱怨其存在的同时。”莱文还指出，“维亚康姆多次试图收购 YouTube”，暗示目前的[10 亿美元诉讼](https://web.archive.org/web/20221207113751/https://beta.techcrunch.com/2007/03/13/5217/)是其在事后多年试图在 YouTube 上赚钱。

以下是这篇博文的关键段落:

> 多年来，维亚康姆不断秘密地将内容上传到 YouTube，甚至公开抱怨它的存在。它雇佣了不下 18 家不同的营销机构来上传内容到网站上。它故意“粗暴对待”这些视频，让它们看起来像是被盗或泄露的。它使用虚假的电子邮件地址开设了 YouTube 账户。它甚至派员工到 Kinko's 上传无法追踪到 Viacom 的电脑上的视频片段。为了推广自己的节目，作为公司政策，维亚康姆经常保留普通用户上传到 YouTube 的节目片段。从喜剧中心的总裁到 MTV 网络的负责人，高管们“非常强烈地”认为，像《每日秀》和《科尔伯特报告》这样的节目片段应该留在 YouTube 上。
> 
> 维亚康姆掩饰其对 YouTube 的推广使用的努力非常成功，甚至连它自己的员工都无法跟踪它在网站上发布或留下的所有内容。结果，维亚康姆无数次要求删除上传到 YouTube 上的视频片段，但后来又不好意思地要求恢复。事实上，维亚康姆起诉我们的一些片段实际上是由维亚康姆自己上传的。

换句话说，在维亚康姆的律师发布撤下通知的同时，其营销人员在 YouTube 上发布视频剪辑，以推广维亚康姆的电影和电视节目。你一定想知道法官会如何看待这些证据。
