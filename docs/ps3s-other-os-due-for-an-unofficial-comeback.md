# PS3 的另一个操作系统将非正式回归

![](img/95921367b336b1a3c809223cec40b875.png "Other OS")

Lo！看起来“其他 OS”[将回归 PS3](https://web.archive.org/web/20221210000026/http://xorloser.com/?p=297&cpage=22#comment-3270) ，尽管是以[非官方身份](https://web.archive.org/web/20221210000026/http://www.ps3-hacks.com/2011/02/02/otheros-to-make-a-return/)。(你真的认为索尼会批准《其他操作系统》的回归吗？)著名的 tinker graf_chokolo 说他已经恢复了安装其他操作系统的选项，而且“很快就会实现”

格拉夫·乔科洛先生说:

> 我的方法将使其他操作系统拥有与 GameOS 相同的访问权限，这意味着可以访问 Dispatcher Manager、Update Manager、VFLASH、HDD 加密/解密、隔离 spu 和 RSX。
> GameOS 只对游戏好，对于 PS3 开发和黑客来说是 Linux 或者有 GameOS 权限的 FreeBSD 要好得多。我想要一个干净的方法来引导 Linux，不像石棉，它不是很干净。

对我来说，这听起来像一个完整的 Linux 系统，这是人们曾经想要的。
