# 新的“Showtime Anytime”应用程序将 Showtime 的电影和节目带到 iPad 上

Showtime 今天加入 HBO GO，成为另一个将内容带到移动设备的优质有线电视频道。通过新的 iPad 应用程序 [Showtime Anytime](https://web.archive.org/web/20221006172637/http://itunes.apple.com/us/app/id484232467) ，客户现在可以在任何有互联网连接的地方观看 Showtime 的原创节目，包括其电视剧、电影、体育和纪录片。

当然，这些内容只对 Showtime 的订户开放，目前这份名单只包括美国电话电报公司 U-Verse 和威瑞森 FiOS。康卡斯特的客户将能够通过 Xfinity 电视服务访问 Showtime 的内容，但该公司表示，Showtime Anytime 应用程序对这些客户来说“即将到来”。

该应用程序提供了对 Showtime 顶级节目的移动访问，如 Dexter，Weeds，Homeland，Californication，无耻，护士 Jackie，The Big C，Episodes，Borglas 和 The Real L Word，并将提供对即将播出的系列节目如谎言之家和内部喜剧的访问。

该网络的电影也将推出，首先是《国王的演讲》、《蓝色情人节》、《拆弹部队》、《无耻混蛋》、《我是第四号》、《暮光之城:新月和日蚀》等。体育节目包括 Showtime 冠军拳击赛、Inside NASCAR、Strikeforce 混合武术等。

iPad 用户可以使用“我的列表”选项在应用程序中导航，这让他们可以创建一个最喜欢的节目的个性化列表，并可以通过在脸书和 Twitter 上“喜欢”和“关注”他们最喜欢的节目来分享他们的观看习惯，或者从应用程序的界面发布到任何一个社交网络。

该应用程序还支持播放切换——这意味着当你更换设备时，它可以让你从你离开的地方继续。当这款应用在 2012 年初登陆 iPhone 和 Android 平台时，将会产生更大的影响。
