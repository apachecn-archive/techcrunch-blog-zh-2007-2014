**编者按:** *罗斯·鲁宾是[十字线研究](https://web.archive.org/web/20221007222629/http://www.reticleresearch.com/)的首席分析师，并在[科技进展](https://web.archive.org/web/20221007222629/http://www.techspressive.com/)发表博客。每个专栏都关注那些已经达到或未达到融资目标的众筹产品。在推特上关注他 [@rossrubin](https://web.archive.org/web/20221007222629/https://twitter.com/rossrubin) 。*

如今，似乎任何带有传统个人电脑味道的东西都具有一个月大香蕉的所有市场吸引力。微软和它的硬件伙伴们正试图用带触摸功能的一体式电脑、覆盖着 clickety 的平板电脑和像柔术演员一样扭曲的可变形笔记本来反击古板的塔式和笔记本形象。然而，由于没有 Windows 的利益需要保护，设备众筹者采取了不同的策略，将 Android 和其他移动操作系统推向异类配置。虽然对 OUYA 和 GameStick 的小型游戏机来说有点过时，但这款游戏现在可以用于更一般的计算任务。

![MiiPC](img/4c21f6f709f4643ec6ad597956940d14.png) **支持:** [MiiPC ZeroDesktop](https://web.archive.org/web/20221007222629/http://www.kickstarter.com/projects/2102024857/miipc-power-to-the-parents) 是一家比典型的单枪匹马进入众筹领域的企业家更成熟的企业。但它在云服务以及远程访问方面的经验使 MiiPC 与众不同，这是一台过度生长的 Android 电脑，具有广泛的控制功能，可以控制家中的学龄前儿童和青少年，还有一个绿色的背光灯，没有什么好的理由，只是看起来有点酷。MiiPC 将推出一款配套应用，让时刻警惕的父母和监护人像老板一样控制对应用的访问，而不管中间马尔科姆的主题曲信息。

MiiPC 项目本周结束，其迷你台式机的目标是 50，000 美元，支持者可以以 99 美元的价格收购。由于该设备使用的芯片组与当今谷歌电视盒子中的芯片组类似，该公司将在今年夏天让实习生使用该芯片组，看看可以为一款小型但带有插头的安卓设备找到什么样的替代用途。

![Aurus](img/986593908e13c667da9e62abb3cb2a11.png) **疲惫不堪:** [ Aurus 双屏平板电脑](https://web.archive.org/web/20221007222629/http://www.indiegogo.com/projects/dual-screen-tablet-pc)。这是什么疯狂？不是一个而是两个显示屏的移动设备？东芝(Toshiba)和宏碁(Acer)的 Windows 个人电脑以及索尼(Sony)的安卓(Android)设备都让人想到了不可思议的事情。所有这些都失败了，部分原因是底层操作系统没有优化到可以在一个屏幕上显示键盘或游戏控件，而在另一个屏幕上显示电子邮件客户端或游戏。活动页面承认了这个问题，问道:“需要一些双屏应用吗？，“向支持者保证他们正在开发一些。索尼方面表示，它正在与第三方开发商合作。但是，再次祝你好运，没有谷歌全力支持多屏设备。

支持者本可以买到起价为€399 英镑的双管安卓平板电脑，但它没有通过多少消费者的筛选。该项目仅获得了其崇高的€20 万美元目标的 1%多一点。

![CoolShip](img/2fcbe211650606739f2e6bb97ef49a15.png) **力挺:** [酷派](https://web.archive.org/web/20221007222629/http://www.indiegogo.com/projects/coolship-an-android-desktop-computer-that-looks-like-a-keyboard)。这并不是我们今天在 iMac 等一体化电脑中看到的集成水平，但一些最早的个人电脑没有单独的塔式机箱，将处理器和内存集成到键盘的同一个外壳中。也许这些早期设计中最巧妙的例子来自 Commodore，它将它们用在了圆形且受欢迎的 Commodore 64 和 VIC-20 上。事实上，该品牌及其泄露秘密的工业设计已被推出，用于昂贵的 Windows 就绪型 x86 PCs，这些 PC 被设计成吸引怀旧者的外壳。

另一方面，Android 宣传的 CoolShip 不仅便宜到 99 美元，而且甚至可以升级，这样你就可以在新的、更强大的处理器上市时更换它们。这种灵活性还应该有助于解决电脑集成到键盘上的另一个问题:因饮料溢出而死亡。CoolShip 的竞选目标是 1 万美元，几乎是这个数字的两倍，预计本月开始向支持者发货。
