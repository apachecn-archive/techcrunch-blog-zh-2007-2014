# 理光新推出的加固型 G700 相机看起来像一次性的

![](img/116ddeddfa982e575175af1d469fb617.png "ricoh_g700")
理光在这里有一种美感。我刚刚意识到最近发布的 [CX4](https://web.archive.org/web/20221207212247/http://www.crunchgear.com/2010/08/19/ricoh-cx4-10-megapixels-720p-high-res-lcd/) 让我想起了什么:那种一次性相机。这台新的 [G700](https://web.archive.org/web/20221207212247/http://www.ricoh.com/r_dc/g/g700/) ，虽然它看起来像一个完全固态的相机，但看起来绝对像你用完之后会扔掉它。

![](img/673d380cb8eb6c388fde61f8a90191e5.png "pop_bback")

没什么好说的:1200 万像素，5 倍光学变焦，ISO 64-3200，720p 视频。不过，最主要的是坚固性。它看起来和我们几个月前的综述一样坚固:水下 5 米，落差 2 米，灰尘和化学物质密封。

不错的触摸:你可以使用普通的充电电池组，但如果你发现自己远离插座，你可以切换到 AAAs。我喜欢理光。不知道为什么，我就是喜欢那些家伙。G700 还没有美国的价格或日期，但我认为我们很快就会知道了。
