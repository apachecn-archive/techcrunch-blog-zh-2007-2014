# Firefox 与众包 t 恤商店合作

![](img/c3803a4cc3e5d150e523c261774602ff.png)

借鉴 [Threadless](https://web.archive.org/web/20221006192529/http://www.threadless.com/) 的做法，Mozilla 正在开放自己的[在线商店](https://web.archive.org/web/20221006192529/http://communitystore.mozilla.org/)，展示众包 [Firefox T 恤设计](https://web.archive.org/web/20221006192529/http://communitystore.mozilla.org/gallery)。你可以上传自己的设计，或者从图库中挑选。

这个商店是建立在 Zazzle 的平台上的，这个平台负责打印、运输和结算。Zazzle 也推出了专注于其他在线社区的合作商店，但这可能是最好的例子。

t 恤衫在网上总是很赚钱。但是，谷歌的补贴有多少能被 t 恤销售所取代呢？
