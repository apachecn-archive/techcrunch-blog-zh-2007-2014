# UPcload 使用您的网络摄像头来帮助您订购真正合身的衣服

我是那种喜欢在网上买东西的人。电池？没错。杂货？你打赌。而且我刚买了一个靠近就会自动打开的垃圾桶(神奇！).

但是有一个很大的领域是网上购物一直欠缺的:服装购物。如果你曾经尝试过在网上订购一件衬衫或牛仔裤，那么你就会知道这种感觉——网站上说这是中号的，但你不确定是否应该换成大号的。然后当衬衫真的到达时，你发现它以所有错误的方式垂下来，看起来模糊地像一件雨披。即使有免费退货，换另一个尺码还是很痛苦。

要是有更好的方法就好了。

UPcloud 是一家新的创业公司，今天在 TC Disrupt Beijing 成立，它认为自己已经拥有了。

该公司正在做我们许多人都希望做的事情:它使用你电脑的网络摄像头来评估你，然后推荐适合你的衣服。你站在摄像机前，拿着一张 CD(形状一致)来校准服务。然后你可以在任何一个集成了 UPcload 的网站上购物。每个网站都会给出每件衣服的尺码信息，这些信息会与你的合身程度相匹配。

那么，问题是它是否真的有效。我们的第一次测试很有希望，该网站已经有了一个重要的信任票:它目前正在与 North Face 一起测试。
