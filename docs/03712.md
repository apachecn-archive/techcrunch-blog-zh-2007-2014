# 员工展示的 Twitter 新总部(图片)

今天，Twitter 搬进了位于旧金山的一个新的、更大的办公室。这个空间，以前是 Bebo 的科幻办公室，就在他们旧办公室的拐角处。

Twitter 团队的一些成员在周末花了很多时间用一些 Twitter 主题的元素装饰新住所，比如鸟和@符号。看看下面 Twitter 员工发布到网上的一些图片。是的，这里有一个 DJ 台，显然还有厕所隔间里的化妆镜。
