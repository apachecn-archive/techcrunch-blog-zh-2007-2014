# SFO 发出停止通知，阻止拼车公司在机场运营 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/04/sfo-hates-ride-sharing/>

# SFO 会发出停止通知，阻止拼车公司在机场运营

上周，InstantCab 向全世界宣布，它已经收到了来自旧金山国际机场的部分停止函，要求它停止在那里接送乘客。显然，这并不孤单，因为其他新的交通初创公司上个月也收到了同样的通知。

例如，SideCar 证实，它在 3 月初收到了一封停止通知函。Zimride 的 Lyft 没有回应我们的询问，但我们听说它也是受到 C&D 通知的公司之一。InstantCab 提供社区和出租车司机的混合服务，实际上是最后一家收到 SFO 通知的公司。该通知仅适用于其社区司机，而不是其业务的出租车电子呼叫方面。

这些信件旨在提醒 Lyft 和 SideCar 等公司，该网站是由旧金山市和县以及机场委员会运营的，同时也提醒他们没有在旧金山机场运营的许可。通知称，如果没有许可证，继续在 SFO 运营社区司机服务将被视为“非法侵入”。通知特别指出，这些服务违反了几项机场规则和条例，包括这条:

> 规则 3.3(E):未经局长或局长正式授权的代表的明确书面同意，任何人不得进入或停留在机场财产上，不得在机场财产上或从机场财产购买、出售、兜售或要约出售或购买任何种类的货物、商品、财产或服务。

像世界上所有其他机场一样，旧金山机场允许受监管的出租车服务在机场内接送乘客。几十年来，黑车和豪华轿车服务一直在运送乘客往返于旧金山。在过去几年里，优步甚至为进出机场的人提供了一种替代交通工具——尽管优步司机与豪华轿车服务几乎没有什么区别，部分原因是该公司与他们合作提供 UberBLACK 和 UberSUV 服务。

SFO 的停止函所打击的服务的关键区别特征似乎是，他们雇用的社区司机没有获得旧金山市交通局或加州公共事业委员会的许可。

在这方面，SFO 并不是第一个试图取缔这些服务的人。去年，Lyft 和 SideCar(以及优步和 Tickengo)都收到了来自 CPUC 的通知，之后该机构撤回并开始研究 T2 为拼车服务建立新监管框架的可能性。在费城和奥斯汀等市场，SideCar 受到了当地监管机构的一些抵制，这是它最近扩张到的城市。

这里的不确定因素是优步，据我们所知，它还没有收到 SFO 的通知。虽然自多年前在旧金山推出以来，它一直默默地为机场的乘客提供服务，但最近它通过与社区司机签约提供低价 UBERx 服务，加入了拼车大军。
