# 丰兹，死侍在召唤

以下是我对 2008 年的预测:死池将被 VoIP 服务填满。第一个加入的可能是 VoIP 电话会议服务 [Foonz。](https://web.archive.org/web/20230215213943/http://foonz.com/)新闻快报。给人们打免费电话是没有钱的。

Foonz 母公司 RPM Communications 的 CMO[告诉 GigaOm](https://web.archive.org/web/20230215213943/http://gigaom.com/2007/12/26/foonz-losing-its-voice/) 他已经“转移注意力”到一项名为 [Utterz](https://web.archive.org/web/20230215213943/http://www.utterz.com/) 的新服务上，这项服务可以让人们将手机上的语音信息、照片和视频发布到博客上。(想象一下，一个音频 Twitter 实际上迫使你听每个人的当天想法更新，而不是仅仅浏览或忽略文本中的所有内容)。虽然 Foonz 站点仍然运行，服务也可用，但我们现在把它放到了[死池](https://web.archive.org/web/20230215213943/https://techcrunch.com/tag/deadpool)中。如果几个月后服务还在响，我们会再次取消。
