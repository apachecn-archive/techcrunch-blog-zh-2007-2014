# 解锁的 iPhone 4Ss 现在已经上市，但是需要付费

如果冬天来临，你渴望温暖的气候，无论你想去哪个热带地区，苹果都能满足你。正如早在时承诺的那样，苹果已经开始在其在线商店销售所有 iPhone 4S 型号的解锁版本。

希望你能买到便宜的机票，因为苹果的环球旅行 4S 价格不菲。正如预期的那样，基本级别的 16GB iPhone 4S 的售价为 649 美元，而 32g 和 64GB 的版本分别为 749 美元和 849 美元。

解锁的 iPhone 4S 将接受来自世界各地任何 GSM 提供商的 microSIM 卡(即使是手工制作的),但苹果公司明确表示 CDMA 漫游是禁止的。考虑到世界上大多数地方都在 GSM 上运行，这并不是一个巨大的绊脚石，但它肯定会阻碍任何人尝试在威瑞森或 Sprint 上使用这些东西的计划。不过，如果有人想这样做，他们最好直接从运营商那里购买，并礼貌地要求他们在最初的 60 天期限过后解锁。

真的，没有什么理由让任何人选择这些东西，除非你经常坐飞机或者非常害怕签合同。不管是哪种情况，我都希望你的新 iPhone 一帆风顺，好运连连。
