# Windows Live Spaces 过渡到 WordPress 创建了 100 万个新博客

![](img/a0dd39d890efd93025c30d90fd5de2da.png)

去年 9 月在 TechCrunch Disrupt 上，微软[宣布全部 700 万 Windows Live Spaces 博客将转移到 WordPress.com](https://web.archive.org/web/20221004180130/https://beta.techcrunch.com/2010/09/27/windows-live-blogging/)。结果这个数字被夸大了，一封[随后的内部邮件](https://web.archive.org/web/20221004180130/https://beta.techcrunch.com/2010/09/30/leaked-internal-emails-show-microsoft-overstated-windows-live-spaces-numbers/)称转变后的博客的真实数量为 30 万。

已经转移到 WordPress 的 Windows Live Spaces 博客数量[现在](https://web.archive.org/web/20221004180130/http://windowsteamblog.com/windows_live/b/windowslive/archive/2010/11/29/over-500-000-windows-live-spaces-blogs-migrated-to-wordpress-com.aspx)已经“超过 50 万”,另外还有 50 万新的 WordPress 博客是由 Windows Live 用户创建的。因此，作为合作的结果，新创建的 WordPress 博客总数达到 100 万。

到 2011 年 3 月，当 Windows Live Spaces 日志不再受支持时，总数不会达到 700 万。但不管怎样，这些博客中有很多已经死了。100 万个新的 WordPress 博客是不容小觑的。自交易宣布以来，WordPress.com 新注册的博客数量已经翻了一倍多，从每月 40 万增加到每月 90 万。
