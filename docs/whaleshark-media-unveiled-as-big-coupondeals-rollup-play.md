# WhaleShark Media 作为大型优惠券/交易汇总游戏亮相

我们第一次听说 [WhaleShark Media](https://web.archive.org/web/20230126162159/http://www.crunchbase.com/company/whale-shark-media) 是在三月份，当时 MySpace 移动副总裁[约翰·费思](https://web.archive.org/web/20230126162159/http://www.crunchbase.com/person/john-faith)跳槽去了[加入神秘的新公司](https://web.archive.org/web/20230126162159/https://techcrunch.com/2010/03/02/myspace-john-faith/)。现在我们有了所有的细节。

该公司由前大资本家首席运营官[科特·库宁厄姆](https://web.archive.org/web/20230126162159/http://www.crunchbase.com/person/cotter-cuningham)领导，已经从奥斯汀风险投资公司和一些未披露的天使投资人那里筹集了超过 3000 万美元的资金。他们的策略是在互联网上建立顶级交易和优惠券网站。这不是世界上最性感的行业，但显然是有利可图的。

他们已经收购了四个网站——[deals 2 buy](https://web.archive.org/web/20230126162159/http://www.deals2buy.com/)、 [coupon7](https://web.archive.org/web/20230126162159/http://www.couponseven.com/) 、 [couponshare](https://web.archive.org/web/20230126162159/http://couponshare.com/) 和我个人最喜欢的[cheapstingybargains.com](https://web.archive.org/web/20230126162159/http://www.cheapstingybargains.com/)。除了现金和股票之外，该公司没有透露具体条款。他们说，更多的交易即将完成。

该公司的前进品牌很可能是[deals.com](https://web.archive.org/web/20230126162159/http://deals.com/)，这是他们从[需求媒体](https://web.archive.org/web/20230126162159/http://www.crunchbase.com/company/demandmedia)获得的域名。新网站仍在开发中，目前需求媒体内容仍在网站上。

这些类型的汇总正在成为奥斯汀风险投资公司的专长。他们创立了非常成功的 HomeAway rollup。更多[最近他们支持](https://web.archive.org/web/20230126162159/https://techcrunch.com/2010/03/03/chris-dewolfe-mindjolt-austin-ventures/)前 MySpace 首席执行官[克里斯·德沃夫的](https://web.archive.org/web/20230126162159/http://www.crunchbase.com/person/chris-dewolfe)社交游戏汇总 [MindJolt](https://web.archive.org/web/20230126162159/http://www.mindjolt.com/) 。汤姆·鲍尔和菲尔·西格尔。他们说，该公司已经“哦，是的”非常盈利，收入为 1000 万到 2000 万美元。
