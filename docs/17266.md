# 报告:美国国家安全局试图建立一个量子计算机来破解加密

《华盛顿邮报》从爱德华·斯诺登处获得的文件显示，美国国家安全局想要建造一台未来超级计算机，可以破解大多数类型的加密。然而，还没有理由担心，因为该机构甚至还没有弄清楚如何建立破解大多数安全程序所需的太空时代技术。

根据这些文件，该计算机项目是一个价值 7970 万美元的研究项目“穿透硬目标”的一部分，该项目涉嫌根据机密合同在马里兰州大学城的一个实验室进行。

[至少从古希腊](https://web.archive.org/web/20221007014603/http://www.howtogeek.com/howto/33949/htg-explains-what-is-encryption-and-how-does-it-work/)开始，加密就是一种用任意字符串的噪音对信息进行加扰的方法；只有有专门知识的人才能确定什么是噪音，什么是信息。噪声或比特越多，通过试错法破解代码就越困难。1024 位加密可能需要数年(或更长时间)才能解码。

量子力学利用了相当不直观的概念，即物质可以以多种状态存在。[量子计算机](https://web.archive.org/web/20221007014603/http://en.wikipedia.org/wiki/Quantum_computer)可以计算比特既是 1 又是 0，或者是多个 1 和 0 的问题，有可能成倍地提高计算速度。

届时，美国国家安全局可能会建造一台能够更快解码最先进加密技术的计算机。

一般来说，即使允许一个组织有能力破解加密，也有可能威胁到网络上的所有安全。不过，量子计算的进步也可能对科学和健康有益。因此，在这一点上，好处和坏处是非常理论化的。

据《华盛顿邮报》报道，无论哪种方式，这种计算机都极其脆弱，美国国家安全局只接近几个基本的构建模块。“这是很大的一步，但在建造大规模量子计算机的道路上，这只是很小的一步，”麻省理工学院机械工程师劳埃德·塞思解释道。

这就是为什么 NSA [也向安全专家](https://web.archive.org/web/20221007014603/https://beta.techcrunch.com/2013/12/28/rsas-deal-with-the-nsa-shows-how-mistrust-is-spreading-everywhere/)施压并付钱让他们制造后门和漏洞。
