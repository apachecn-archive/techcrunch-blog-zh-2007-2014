# Raspberry Pi 曾经取代了一个 30 英尺的 GSM 基站，并创建了一个有效的移动网络

【YouTube http://www.youtube.com/watch?v=GCcKgrzbix4？一家总部位于英国剑桥的咨询公司已经成功使用开源的 Raspberry Pi 计算机来复制通常由 30 英尺的 GSM 蜂窝基站执行的功能，以创建一个全功能的移动网络。使用两个开源软件程序和一些现成的硬件套件，DIY 爱好者可以相当容易地得到它们， [PA 咨询公司推出了他们自己的移动电话服务](https://web.archive.org/web/20221209132952/http://www.newelectronics.co.uk/electronics-news/basestation-created-on-raspberry-pi/46960/)。

该系统的工作原理是路由呼叫，类似于 Skype 处理呼叫的方式，这要归功于一个名为 FreeSWITCH 的开源程序，该程序还可以使短信通信和网络上的电话连接到互联网。所有这一切都必须建在一个无线电密封的房间里，以避免踩到任何频谱脚趾，从而违反法律。这个项目的全部目的只是为了证明它是可行的:一个 30 英尺高、极其昂贵的重要移动网络基础设施或多或少可以被一个 3 英寸的 Raspberry Pi 所取代，至少在提供一个实际的、正常运行的移动电话网络方面是如此。这预示着低成本基础设施的未来，也符合 Raspberry Pi 基金会的目标，即为以前昂贵的问题提供负担得起、易于使用的解决方案。
