# VMWare 收购 SpringSource

![picture-15](img/f2b4ec21ced4e0884148696cbc15fb09.png "picture-15")今天，VMWare[宣布](https://web.archive.org/web/20230326024349/http://www.vmware.com/company/news/releases/springsource.html)收购 SpringSource，一家网络应用开发和管理服务提供商。该交易以 4.2 亿美元的估值结束，其中包括 3.62 亿美元的现金和股权，外加约 5800 万美元的未归属股票和期权。

SpringSource 是“精益软件”的著名支持者，这是一个在企业空间获得牵引力的概念，作为加速在云中交付业务应用程序的一种手段。通过此次收购，VMWare 和 SpringSource 将合作提供一个平台，让客户能够“在内部和外部云架构中更高效地构建、运行和管理应用程序”

SpringSource 首席执行官 Rod Johnson 表示:“VMware 通过创新的虚拟化和云体系结构引领了数据中心基础架构的现代化，为客户提供了成本节约、灵活性和选择。“SpringSource 团队和社区致力于革新公司构建、运行和管理应用程序的方式。通过强强联合，我相信我们能够提供一套真正卓越的解决方案，显著简化企业 IT。”

在此之前不久，谷歌的一小部分人才从 T4 流向了 VMWare。有趣的是，[基准资本](https://web.archive.org/web/20230326024349/http://www.crunchbase.com/financial-organization/benchmark-capital)是 SpringSource 的最大投资者，彼得·芬顿是该公司的董事会成员。Benchmark 也是 FriendFeed 的投资者，因此该公司显然享受着本周的良好开端。
