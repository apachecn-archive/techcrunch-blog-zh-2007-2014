# Rovio 在芬兰开设了世界上第一家愤怒的小鸟商店

小心，三丽鸥。在统治移动世界不到两年后(憎恨者见鬼去吧)，愤怒的小鸟正在进军自己的零售领域。

虽然 Rovio 计划明年某个时候在中国开店(这里的品牌很大，但几乎所有的商品都是假的)，但第一家店是在他们的家乡芬兰赫尔辛基。它有一个巨大的弹弓！

可悲的是，可悲的弹弓似乎并不实际…你知道，吊索。大概是因为他们不愿意让一个 8 岁的孩子把他的鞋子射进店面的窗户，这个弹弓反而成了那些寻找新的脸书侧面照的人的秋千椅。

当然，可能值得注意的是，第一天可用的愤怒的小鸟商品的种类并不完全..多样。他们有《愤怒的小鸟》烹饪书(没错)，还有《愤怒的小鸟》铅笔/橡皮擦学校套装……但在大多数情况下，书架上并排摆放着他们去年推出的一系列毛绒玩具。来吧，彼得:动手干掉那些海盗！

祝所有员工好运，在第 37，000 次听到愤怒的小鸟主题后，他们不会去*绝对疯狂*。
