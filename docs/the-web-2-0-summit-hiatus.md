# Web 2.0 峰会，蒂姆·奥雷利和约翰·巴特利的技术盛会，正无限期地“中断”

Web 2.0 峰会今年不会举行。自 2004 年成立以来，这个会议已经发展成为网络和新媒体上层社会最热闹、最繁忙的年度活动之一，根据企业家/记者/作家[约翰·巴特利](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/john-battelle)的说法，这个会议将无限期地“暂停”，他和[蒂姆·奥莱利](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/tim-oreilly)一起主持这个会议。PandoDaily 是第一个[报道这个消息的](https://web.archive.org/web/20221006043121/http://pandodaily.com/2012/04/04/after-eight-years-the-web-2-0-conference-is-over/)。

在一篇博客文章中，巴特尔说暂停 Web 2.0 峰会的决定仅仅是因为时间紧迫。他正在写一本书[，他没有足够的时间来计划今年的会议，而会议需要真正启动:](https://web.archive.org/web/20221006043121/http://battellemedia.com/archives/2012/04/if-then-and-antiquities-of-the-future.php?r44b=no)

> “作为一个专注于核心产品——舞台上的节目——的人，我无法同时完成写书和创作一个完美的舞台节目。执行 Web 2 这样的会议需要月复一月的努力(不仅仅是我)。我在 O'Reilly 和 UBM TechWeb 的合作伙伴正忙于其他会议，经过几个月关于我们如何解决这个问题的讨论，我们都同意真的没有办法做到这一点。做一个阻止派对的人并不有趣，但在这种情况下，我必须站出来承担责任。”

巴特尔很快指出，Web 2.0 峰会不一定已经死亡——它只是暂时结束了，“处于中断状态”

不管怎样，这肯定是一个时代的终结。Web 2.0 峰会每年都是一个有形的、里程碑式的事件，以跟踪 Web 2.0 运动的松散概念。当然，技术世界并不缺少会议，但对我来说，Web 2.0 峰会一直是一个必须参加的会议，因为它一直吸引着来自各行各业的知名人士谈论所有技术的现状(仅在去年的[活动](https://web.archive.org/web/20221006043121/http://www.web2summit.com/web2011)上，发言者就包括 Twitter 首席执行官[迪克·科斯特罗](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/dick-costolo)，微软首席执行官[史蒂夫鲍尔默](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/steve-ballmer)，Salesforce 首席执行官[马克贝尼奥夫](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/marc-benioff)， 戴尔创始人[迈克尔·戴尔](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/michael-dell)，Foursquare 创始人[丹尼斯·克劳利](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/dennis-crowley)，LinkedIn 联合创始人[雷德·霍夫曼](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/reid-hoffman)，Napster 联合创始人兼早期脸书投资人[肖恩·帕克](https://web.archive.org/web/20221006043121/http://www.crunchbase.com/person/sean-parker)，还有很多很多其他人。 )看看哪个项目会取代它的位置会很有趣。
