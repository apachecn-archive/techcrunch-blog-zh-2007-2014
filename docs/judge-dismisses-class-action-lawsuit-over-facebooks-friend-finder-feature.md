# 法官驳回集体诉讼脸书的朋友发现者功能

昨天，加州联邦法官驳回了一起针对脸书的集体诉讼。在罗宾·科恩等人诉脸书一案中，该网络的一些成员指控脸书盗用用户的姓名和肖像来推广其朋友查找服务，该服务向选择上传其电子邮件联系人的用户推荐新的脸书朋友。

罗宾·科恩的案子实际上在 2011 年 6 月 28 日就被驳回了，但是原告提交了一份修正的诉状。周四的命令驳回了这一投诉。法官裁定，原告没有证明脸书涉嫌使用他们的名字和肖像造成了任何经济损害。

脸书[先前辩称](https://web.archive.org/web/20230203083518/http://www.mediapost.com/publications/article/153405/)原告没有受到伤害，并且网络的隐私政策规定成员的名字和照片是公开信息。

脸书的诉讼律师桑迪普·索兰基(Sandeep Solanki)对这一裁决做出了回应，“我们感谢法院的考虑，我们很高兴所有的索赔都因偏见而被驳回。”

这不是脸书最近第一次集体诉讼被驳回。9 月，洛杉矶高等法院驳回了一项集体诉讼，该诉讼质疑在脸书的第三方广告旁边使用青少年的名字和肖像，据称未经父母同意。

当然，当一个消失时，一些新的必然会出现。10 月早些时候，脸书看到三起[新的潜在集体诉讼](https://web.archive.org/web/20230203083518/http://www.mediapost.com/publications/article/160136/)被提起，原因是它追踪注销用户的报告。
