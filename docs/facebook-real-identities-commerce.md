# 脸书想要一个真实身份网络的另一个原因是

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/01/facebook-real-identities-commerce/>

上周，脸书的营销主管兰迪·扎克伯格声称网上匿名必须被废除，这引起了轰动。但是，强大的大型网络正在推动一个世界，在这个世界中，我们经过验证和认证的身份在线存在[不仅仅是为了阻止网络欺凌，并为用户创造更好的行为激励。这是钱的问题。该公司的部分驱动力也是帮助用户利用他们的在线身份来转变和加速在线商务。](https://web.archive.org/web/20230205023533/http://www.huffingtonpost.com/2011/07/27/randi-zuckerberg-anonymity-online_n_910892.html)

在线身份和匿名的话题是两极化的。强大的利益集团有强烈的动机去建立一个真实身份统治的网络。其他强大的利益集团将会不遗余力地保护和保留用户互动的空间，不管是匿名还是假名。这种说法引发了另一方的激烈争论，而在脸书，这些声音变得更大了。在电子前沿基金会的网站上有一篇由吉利安·约克写的关于这个话题的[的简短帖子](https://web.archive.org/web/20230205023533/https://www.eff.org/deeplinks/2011/07/case-pseudonyms)——她很快涵盖了支持和反对使用真实身份的观点。

不幸的是，在这种特定的背景下，这些辩论都没有解决问题的核心，即对于像脸书这样的网络来说，游戏是鼓励其用户在网上利用他们的真实身份，以便脸书能够加快其推动网上交易的能力。

我的意思不是玩世不恭。关于网络是否需要真实身份或匿名空间，我们可以争论到(脸书)面红耳赤。答案没有对错之分。为了构建这篇文章，我个人的观点是，我认为网上隐私的概念已经消亡多年，我愿意放弃一些隐私，使用我的真实姓名，只要它能让现代生活更有趣，并为我节省时间和/或金钱。

我们现在知道这笔交易了——脸书知道我们的朋友和家人，我们的兴趣，以及许多其他的生物数据。当我们在浏览器中打开脸书标签时，或者当我们在手机上启动脸书应用时，他们可以根据一系列个性化变量开始向我们精确定位广告。在购物中心启动脸书移动应用程序并获得个性化信息是有帮助的，但这仍然只是为了提高商业速度——帮助我们在更短的时间内定位、决策和交易。如何通过身份认证和支付从根本上改变和推动商业？

想想当你购买机票去旅行时会发生什么。你必须访问航空公司或旅游网站，登录网站(假设你有他们的账户)，然后选择特定日期的路线，选择座位，然后通过输入支付信息(通常是信用卡)，你的账单地址，有时是美国运输安全管理局强加的额外验证来完成交易。在飞机起飞前，你必须“办理登机手续”并打印出你的登机牌(或发送到你的手机上)。在机场，你可能必须托运行李，然后每个人都必须通过一系列安检，每次都要出示登机牌和/或政府颁发的某种身份证明，直到你到达座位。当你下飞机的时候，假设你托运了行李，你必须等着你的行李(假设他们已经到了你的地方)，并且经常坐在那里不知道它在哪里或者什么时候会到。然后你不得不在长长的租车队伍中等待，或者选择各种班车或公共交通方式。起泡，冲洗，重复。

事实上，有成千上万的这种类型的交易迷宫，我们不断地将自己置于其中。但是，不一定要永远这样。例如，想象一下，当你登录或使用手机时，只需点击航空公司网站上的脸书按钮，就可以支付机票费用。我认为，大多数用户愿意牺牲一点隐私和匿名，以换取他们的身份和支付信息得到认证，从而压缩我们旅行时启动和完成一系列交易的时间。只是今天，我们还没有选择。(脸书将需要比现在更进一步，验证人们的真实身份，并与现实世界的支付系统相结合)。

通过在线利用真实身份，压缩这些交易步骤并提高商务效率变得更加容易。虽然脸书公开倡导消除在线匿名可能是出于好意，因为这将有助于让网络成为一个更美好的地方，但我认为真正的原因是，该公司有一个难得的机会，可以通过帮助加速传统商务来让我们的生活变得更轻松，节省我们的时间，而且他们可以从每一部分的经济价值中分得一杯羹。我当然不是说他们会自动成功，也不是说这项任务不会充满无数的风险，但这是一个值得追求的机会，我们应该期待该公司完全做到这一点。

随着网络被慢慢清洗和清理，真实身份将成为等式中的重要部分。当然，也有一些地方，我们可以通过假名或匿名进行在线互动，自由讨论想法，尽管在每种情况下，网站所有者更有可能知道谁是淘气的或善良的。尽管听起来令人欣慰，但网上没有匿名这回事——只有虚假的度量。虽然我们可以在一些在线论坛上使用假名来帮助保护我们的身份，以实际推动商业并使这些交易更有效(对买方和卖方而言)，但交易需要真实身份的支持。

如果过去十年的“社交网络”真的会创造巨大的经济价值，LinkedIn 和脸书这样的网站将需要对我们所有的个人和集体数据做些什么——我不是说向用户展示超针对性的展示广告。

这些类型的公司的真正经济潜力在于验证和认证我们的身份，以便他们可以为我们提供有趣的在线体验，与我们的职业和个人生活的一部分联系起来，并有可能推动一种新型的商业，这种商业可以产生大量的收入和利润，同时为我们节省大量的时间、金钱和压力。
