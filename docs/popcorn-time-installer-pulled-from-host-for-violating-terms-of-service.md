# 爆米花时间安装程序因违反服务条款被从主机上撤下更新:它回来了

更新:安装程序又回到 Mega 上了。

[爆米花时间](https://web.archive.org/web/20221016120821/http://getpopcornti.me/)是[一个神奇的程序](https://web.archive.org/web/20221016120821/https://beta.techcrunch.com/2014/03/11/popcorn-time-is-like-netflix-for-pirated-content/)，它可以解决烘焙电影的所有麻烦。这是好莱坞最可怕的噩梦。这实际上是盗版电影的网飞。现在没了。

该程序的安装程序由金·多特康姆的 Mega 托管。不再是了。该安装程序因违反 Mega 的服务条款，今天早上被从托管网站上撤下。开发者目前正在寻找新的主机。

别的不说，安装程序本身可以去中心化，简单地通过 bit torrent 本身服务。虽然这种方法消除了开发人员追求的易用性，但他们不再需要担心遵守主机的服务条款。安装程序已经在几个种子网站上。

这不太可能是爆米花时间的终结。开源项目背后的开发者将找到一个新的主机，它将有朝一日与好莱坞对抗。正如一句古老的谚语所说，一旦它出现在互联网上，它将永远存在。
