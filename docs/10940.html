Scrolldit:我听说你喜欢滚动Reddit，所以我在你的Reddit中加入了滚动功能

Scrolldit 是由一个名叫乔纳森·布曼的年轻人发明的，它可以为你滚动Reddit，从而让你避免滚动Reddit的艰苦工作。你会问，为什么这是新闻？因为我们喜欢Reddit，它真的很酷。

该网站基本上采用了Reddit的订阅源(包括NSFW的订阅源，万岁万岁)，并把它们放在小盒子里，在屏幕上移动。因为大多数人不太会阅读，所以有大量的图片和偶尔的视频，可以方便快捷地阅读。大多数渲染发生在浏览器中，甚至出现在Reddit自己的广告中。

很明显，我们在这里谈论的是混搭，但正是像这样的重新配置改进了界面。虽然我怀疑我是否会在日常生活中使用它——目前的Reddit界面已经足够好了，安装了 Reddit增强套件——在一个流行混搭应用如 Pulse 和 Flipboard 的时代，桌面并没有太多替代界面。

该应用程序使用jQuery Masonry和Amazon cloudfront，并使用 JSON API 进行获取。它没有资金来源，因为为什么会有？
