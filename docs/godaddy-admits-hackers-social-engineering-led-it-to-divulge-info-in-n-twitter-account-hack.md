# GoDaddy 承认黑客的社会工程导致其在@N Twitter 账户 Hack 中泄露信息

刚刚从 GoDaddy 传来了@N 帐户黑客事件的最新消息，go daddy 是参与这个有点复杂的社会工程案件的公司之一。该公司承认，其一名员工受到“社会工程”的影响，提供了额外的信息，让一名黑客得以进入 Naoki Hiroshima 的 GoDaddy 账户。

我们[在今天早些时候](https://web.archive.org/web/20230131224536/https://techcrunch.com/2014/01/29/why-hasnt-twitter-just-given-n-his-name-back/)的一篇帖子中详细描述了这一黑客行为，该黑客行为是通过呼叫 PayPal 和 GoDaddy 来获得广岛的个人电子邮件，然后使用该电子邮件从他那里勒索@N Twitter 用户句柄。

广岛[在 Medium 上的一篇帖子中概述了这次黑客攻击](https://web.archive.org/web/20230131224536/https://medium.com/p/24eb09e026dd)，引起了很多关注。我们收到了 Twitter 的回应，称正在调查此事，PayPal 被迫发表声明否认提供了信用卡信息，并指出其员工接受了避免社会工程攻击的培训。

社会工程是一种黑客方法，攻击者利用个人或非个人信息来冒充帐户的合法所有者。他们给有问题的公司打电话，策划“重置”账户权限，让他们接管。

在广岛事件中，目标只是他的 Twitter 账号，但也可能是银行账户或网站。

GoDaddy 首席信息安全官托德·雷德福特(Todd Redfoot)向 TechCrunch 发布了以下关于黑客攻击的声明:

> 我们对情况的审查表明，黑客在联系 GoDaddy 时已经掌握了访问该帐户所需的大部分客户信息。然后，黑客通过社交工程让一名员工提供访问客户账户所需的剩余信息。该客户后来重新获得了对其 GoDaddy 帐户的完全访问权限，我们正在与行业合作伙伴合作，以帮助恢复其他提供商的服务。

Redfoot 还表示，GoDaddy 正在“对员工培训进行必要的改革，以确保我们继续为我们的客户提供行业领先的安全性，并保持领先于不断发展的黑客技术。”

令人不快的是，这些技术其实并不新鲜。正如我们在今天早些时候的文章中提到的，在过去的几年中，一些非常引人注目的黑客已经使用它们完成了。更重要的是一个广为流传的案例,《连线》杂志作家 Mat Honan 的账户几乎被采用社会工程技术的黑客摧毁。

如果说有什么不同的话，像河南和这个关于 T2 的乔什·布莱恩特(@jb)的黑客行为通过[大胆火球](https://web.archive.org/web/20230131224536/http://daringfireball.net/linked/2014/01/29/n)分享的案例应该给任何从事身份交易的互联网公司敲响了警钟。这些都不是新的战术，它们应该作为一个非常基本的预防措施来防范。
