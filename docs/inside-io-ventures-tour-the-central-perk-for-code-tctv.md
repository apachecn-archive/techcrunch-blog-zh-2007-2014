# 内部输入输出企业:参观代码中央公园(TCTV)

撇开上世纪 90 年代俗气的情景喜剧不谈， [i/o Ventures](https://web.archive.org/web/20230405161431/http://www.ventures.io/) 位于三藩市 Mission 区的孵化器空间围绕着社区的理念:在宽敞的 Summit 咖啡馆喝异国拿铁的社区，在底层另一半租用办公桌的年轻开发人员网络，i/o 孵化器项目的初创公司[在顶层设立法庭](https://web.archive.org/web/20230405161431/https://techcrunch.com/2010/01/06/io-ventures-incubator-san-francisco/)，以及进出大楼的有成就的导师。

这是少数企业家建造的家庭兼咖啡馆。

由[Ashwin Navin](https://web.archive.org/web/20230405161431/http://www.crunchbase.com/person/ashwin-navin)(BitTorrent 联合创始人)[Jim Young](https://web.archive.org/web/20230405161431/http://www.crunchbase.com/person/jim-young)(HotOrNot 联合创始人)[Aber Whitcomb](https://web.archive.org/web/20230405161431/http://www.crunchbase.com/person/aber-whitcomb)(MySpace 前首席技术官)和[Paul brag iel](https://web.archive.org/web/20230405161431/http://www.crunchbase.com/person/paul-bragiel)(Meetro 联合创始人)创立的 [i/o Ventures](https://web.archive.org/web/20230405161431/http://www.crunchbase.com/company/io-ventures) 项目于 1 月份开启，并于周二晚上举行了第一个[演示日](https://web.archive.org/web/20230405161431/https://techcrunch.com/2010/10/12/first-io-ventures-demo-day/)。这个项目已经走过了很长的一段路，它最初是由 Young 提出的在旧金山建立一个有趣的(Wifi 健壮的)咖啡馆的简单想法。

纳文和布拉吉尔在周二给了我们一个完整的游览，值得一看，见上文。
