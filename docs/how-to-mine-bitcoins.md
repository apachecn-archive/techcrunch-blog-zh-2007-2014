# 如何挖掘比特币 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/08/how-to-mine-bitcoins/>

挖掘比特币——一个有助于管理比特币交易以及创造新“财富”的过程——是新的豆豆娃。然而，对我们来说幸运的是，比特币似乎正在升值，并且应该会随着时间的推移保持其价值，不像你的薄荷状态微型填充吉娃娃。

但是怎么得到比特币呢？你可以一开始就直接购买它们，但目前市场非常火爆。每枚比特币的价格为 188 美元，目前任何人都无法猜测比特币的走向，而且与股票不同，这些东西不会分裂。简而言之，你应该是我的。但是什么是比特币挖矿呢？

可以把它想象成一群人寻找大质数或尝试用密钥解密文件的工作。你可以[在这里](https://web.archive.org/web/20230130100805/http://www.businessinsider.com/how-bitcoins-are-mined-and-used-2013-4)读到更多关于它的信息，但是你要明白，每开采一个区块，你可以得到 25 个硬币，或者按照现在的汇率，4722.25 美元。目前，一个比特币的价值为 188 美元，这是一个令人担忧的结果，可能是由与塞浦路斯有关的资金流动以及对该平台的普遍泡沫般的兴奋引起的。事实上，许多人打赌，对许多比特币相关服务的 DDOS 攻击是黑客为了降低价格而注入不稳定性的直接行动。

就目前情况来看,《独自采矿》几乎已经被否决了。寻找积木的过程现在非常受欢迎，而且寻找积木的难度如此之高，以至于可能需要三年多的时间才能产生任何硬币。虽然你可以简单地将一台机器放在一边，让它无休止地运行算法，但能源成本和设备折旧最终将超过比特币的实际价值。

然而，联营采矿利润更高。使用像“Slush's pool”这样的服务(稍后会有更多的介绍)，你可以在一群人中分配工作。使用这个等式:

(25 BTC +大宗费用-2%的费用)*(用户员工找到的股份)/(当前回合的总股份)

虽然这被简化了，但它基本上就是系统的工作方式。你在一个区块中为股份工作，当完成时，你会根据你旁边的工人数量得到一个百分比，减去费用。使用这种方法，我通过运行一台休眠的电脑，在周末筹集了大约 1.5 美元。你们当中精明的人会注意到我可能用了两倍的电。

作为一个新爱好者，我很惊讶我花了这么长时间才开始采矿。[我的好友汤姆](https://web.archive.org/web/20230130100805/https://twitter.com/tprinty)解释了如何建立一个集合采矿账户，所以我想分享一下说明会很有趣。

1.得到一个钱包。你既可以把钱包存放在本地，也可以存放在网上。Coinbase.com 是一款在线钱包，设置起来出奇的简单。钱包要求你使用或下载一个相当大的区块链文件——大约 6GB——所以下载和更新一个本地钱包是不可能的。像所有财富存储媒介一样，保持你的比特币“本地”可能比信任网络服务更好，但这是个人偏好问题。没有首选的钱包类型，两者都有明显的权衡。隐私倡导者可能会说本地钱包是最好的。

你可以在这里下载本地钱包[，但要确保你备份了你的数据。](https://web.archive.org/web/20230130100805/http://bitcoin.org/en/choose-your-wallet)

一旦你创建了一个钱包，你会得到一个这样的地址:1 bekugadfbreshqb 9 x R4 pkptm 8 jayinqsj。这个，没有句号，是直接把比特币发到你钱包里的方式。记下你的地址。在比特币基地，在关联账户下找到的钱包地址。

[![Screen Shot 2013-04-08 at 9.17.19 AM](img/19fc39142cbc9f3dd3844a78188fed48.png)](https://web.archive.org/web/20230130100805/https://techcrunch.com/wp-content/uploads/2013/04/screen-shot-2013-04-08-at-9-17-19-am.png)

2.加入一个游泳池。要在池子里采矿，你必须和一群其他矿工在可用的区块上工作。这里最受欢迎的是雪泥池。你也可以试试像 [BTC 公会](https://web.archive.org/web/20230130100805/http://www.btcguild.com/)这样的公会以及其他一些选择。每个泳池的主要特征是每块收费——例如，Slush 的泳池收费 2%——以及用户数量。用户较少的池也可能具有较慢的发现时间，但是用户较多的池通常导致较少的支付。

你怎么确定池主没有偷走你所有的比特币？你不能。然而，正如泳池所有者 Slush 所说:

理论上，作为比特币池的经营者，我可以把池中发现的 25 BTC 留给自己。我不打算这么做，但我完全接受人们不信任泳池运营商。这是他们的选择自由，比特币关乎自由。

为了简单起见，我使用 Slush 的池并创建了三个工人。首先，创建一个池登录。然后添加工人。工人是有自己密码的子帐户，通常由[您的登录]标识。[工作人员姓名]。目前，我有三个工作人员在运行——一个在我的 iMac 上，两个在我的旧电脑上。
[![Screen Shot 2013-04-08 at 3.05.25 PM](img/dcbe310e175edd1d4c443518aa8ec9c4.png)](https://web.archive.org/web/20230130100805/https://techcrunch.com/wp-content/uploads/2013/04/screen-shot-2013-04-08-at-3-05-25-pm.png)

你必须创造工人来采矿。大多数服务的说明都非常简单，所以不要不知所措。像任何在线俱乐部一样，随着经验的积累，你可以深入挖掘围绕比特币的亚文化。我喜欢把它想象成一个金融 MMORPG。

此外，请确保在池信息中输入您的钱包地址。这将确保你得到你的比特币。

3.**找个矿工。**虽然 OSX 的用户可能会发现自己有点麻烦，但是有很多针对多平台的挖掘选项。矿工使用空闲的 GPU 周期来驱动采矿操作，就像 SETI@Home 这样的服务使用空闲的周期来寻找智能生命一样。另一方面，矿工使用这些周期来帮助处理与比特币相关的点对点流程。因此，通过“工作”，你也在维护网络。

[GUIMiner](https://web.archive.org/web/20230130100805/http://guiminer.org/) 对于 Windows 用户来说是最简单的解决方案，因为它允许你使用几乎所有的标准显卡来创建矿工。你可以在这里下载[。50Miner](https://web.archive.org/web/20230130100805/https://github.com/downloads/Kiv/poclbm/guiminer-20121203.exe) 也是一个受欢迎的解决方案。两者都要求你输入你的工人信息和池，他们将开始挖掘。

Linux 用户可以运行 [CGMiner](https://web.archive.org/web/20230130100805/https://bitcointalk.org/index.php?topic=28402.0) 这样的矿工。在 Ubuntu 上安装 miner 的一个很好的指南是[这里有](https://web.archive.org/web/20230130100805/https://docs.google.com/document/d/1Gw7YPYgMgNNU42skibULbJJUx_suP_CpjSEdSi8_z9U/preview?sle=true)。

OS X 用户可以使用 [DiabloMiner](https://web.archive.org/web/20230130100805/https://github.com/Diablo-D3/DiabloMiner) ，这是一个两年前的命令行程序，将使用 OpenCL 进行挖掘。遗憾的是，它使用不推荐的比特币调用，速度相当慢。因此，你需要运行自己的代理， [Stratum](https://web.archive.org/web/20230130100805/https://github.com/slush0/stratum-mining-proxy) ，它允许 Diablo 与 Slush's pool 等服务连接。这两个程序通常在 OS X 上运行都没有问题，尽管你可能需要安装 [OpenCL for OSX](https://web.archive.org/web/20230130100805/https://developer.apple.com/library/mac/#documentation/Performance/Conceptual/OpenCL_MacProgGuide/Introduction/Introduction.html) 。

对我来说，我已经创建了一个在终端中运行的脚本，它只是在后台运行代理，然后连接 Diablo。注意最后两个参数对于 Mountain Lion 是必要的。

。/stratum-mining-proxy-master/mining _ proxy . py &
。/Diablo miner-OS x . sh-u worker name-p worker password-o localhost-r 8332-w 64-na

RPCMiner 运行起来要容易得多——你只需点击一个图标，输入一些数据——两者都有非常简单的基于文本的界面。在我的 iMac 上运行 Diablo 对 OS X 下的应用程序性能没有太大影响，尽管它确实大大降低了我的 Windows 8 机器的速度。

4.**把心思放在你的钱上。**比特币令人困惑，因为它们使用和开采起来极其简单。那么，投机者可能会扔出数百台机器来解决这个问题，并像雨点一样收集比特币，对吗？不对。随着越来越多的比特币被发现，它们变得越来越难找到。[这个盈利能力计算器](https://web.archive.org/web/20230130100805/http://www.bitcoinx.com/profit/)将帮助你了解你面临的是什么，但要明白这不是一件确定的事情。我运行我的系统一个周末，看到只有 1.50 美元——足够买一杯可乐——但其他用户可能已经改进了硬件和方法来取得成功。简而言之，如果运行你的硬件的成本高于你在比特币中的收益，你可能做错了。

祝你旅途好运，享受你第一次进入这个野生的羊毛世界。
