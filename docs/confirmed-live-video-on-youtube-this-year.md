# 已确认:今年 YouTube 上的视频直播

[http://www.viddler.com/player/b914691e/114.447/](https://web.archive.org/web/20230203023840/http://www.viddler.com/player/b914691e/114.447/)

[莎拉·梅耶斯](https://web.archive.org/web/20230203023840/http://www.pop17.com/)得到了独家新闻(上面的视频)，文字记录如下:

> 迈耶斯:“你们打算什么时候在 YouTube 上做视频直播？”
> 
> 陈:“2008。我们今年会做的。
> 
> “视频直播是我们一直想做的事情，我们从来没有资源来正确地做这件事，但现在有了谷歌，我们希望今年能真正做到这一点。”

现在来猜谜:一旦 YouTube 占据市场主导地位，哪家视频直播初创公司会先倒闭？YouTube 将是最后一个上市的，但见证了 YouTube 主宰视频的势头，现在也将应用于直播视频。就像视频一样，内容创作者希望在给他们最大曝光率的服务上，不管替代领域有多好(毕竟，YouTube 不提供最好质量的视频)。YouTube 已经有了用户群；直播视频流会像飞蛾扑火一样涌向 YouTube。
