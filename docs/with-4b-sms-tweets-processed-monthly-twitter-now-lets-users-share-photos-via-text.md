# 随着每月处理 40 亿条短信，Twitter 现在允许用户通过文本分享照片

我们这些生活在智能手机过剩世界的人经常忘记，功能手机目前主导着全球手机市场。为了向不使用 iphone、Androids 或任何其他基本功能为迷你电脑的手机的用户致意，Twitter 现在允许用户发送照片和短信。

希望通过彩信向 Twitter 发送照片的用户可以像在手机上一样将照片上传到短信中，然后像发送普通文本推文一样发送到 Twitter——例如，发送到美国的 404-04。不幸的是，该功能还不适用于 DM(考虑到一些用户[倾向于 DM 失败，](https://web.archive.org/web/20230203140414/https://techcrunch.com/2011/06/06/tautology/)也许这是一件好事)。

彩信上传功能现已在美国的美国电话电报公司、威瑞森无线和南方蜂窝、英国的沃达丰、O2 和 Orange、意大利的沃达丰、加拿大的 Rogers Communications、巴林的 VIVA 和巴西的 TIM Brazil 上提供。

Twitter 在一篇博客文章中提醒我们，每月有超过 40 亿条短信被发送和接收(现在这个数字可能包括彩信)，短信/彩信可以在 80 个国家和 9 种语言中使用。它还透露，它正在继续添加改善短信/彩信 Twitter 体验的功能，如 Fast Follow(允许您在没有 Twitter 帐户的情况下接收短信)、短信/彩信命令和进一步的运营商覆盖。

在一个充斥着脸书和谷歌哗众取宠的星期里，这只是个小新闻，但随着功能的改进，它可能会受到数百万用户的赞赏。
