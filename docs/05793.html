维珍新项目iPad Magazine的视频演示

维珍有点被寄予厚望

Project

iPad杂志

went live early this morning

。它的重点是“设计、娱乐、技术、企业家”，其形式类似于下一代杂志的其他尝试，尽管我认为这本书更成功。

当然，它需要一台4.2版本的iPad才能享受，虽然已经售出了五六百万台，但不是每个人都有机会体验一下。所以我给你们做了一个视频演示。

抱歉漫无边际。你也许可以把音量关小，播放一些音乐，我实际上没有说什么实质性的东西，除了它花了永远下载，并且有一些小问题需要解决。但我认为这将是有用的，能够看到这个东西是什么，而不需要麻烦一个巨大的下载。
