凯鹏华盈支持的Vlix是一个视频的Instagram为移动视频添加滤镜、效果等

移动视频应用程序 Vlix 发布了一个类似Instagram的移动视频共享iOS应用程序的新版本，该应用程序具有视频文本选项，可用于视频效果、滤镜、特效等。该应用程序目前在App Store中免费提供。

基本上，Vlix允许用户即时发布短视频(最长1分钟)，但可以添加酷效果、文本、编辑内容等。通过该应用程序，你可以捕捉视频，然后添加艺术滤镜，以及反转、慢动作、变亮和颜色增强等效果。您还可以添加可定制的文本，显示在视频的开头和结尾，并将视频分享到私人群组或电子邮件、YouTube、脸书和Twitter。

该应用最初于4月份首次亮相，其新功能是无需注册即可进入应用并浏览其他Vlix成员的特色视频，以及将视频保存到iPhone的相机胶卷中的能力。此外，Vlix允许您创建一个播放前的短视频，在您发布的所有视频之前播放。

当你准备从Vlix发布一个视频时(在使用该公司的dead simple编辑工具之后)，你渲染视频(使用Vlix的专有技术需要几秒钟)，然后发布。应用程序中的其他用户(你可以关注其他用户)可以评论和喜欢你的视频，你可以看到它获得了多少浏览量。

在许多方面，给视频添加效果以及编辑的便利性使该应用程序类似于视频的Instagram。Vlix的人也有在线视频的经验。他们运营 One True Media ，这是一个简单的视频幻灯片制作网站，每天有超过600万用户发布超过10，000个视频，每天有超过5，000个视频发布到YouTube和脸书。该公司的另一家初创公司 SpotMixer 允许企业为网络、电视和手机制作视频。

Vlix已经从Kleiner Perkins和DAG Ventures筹集了1600万美元的资金。
