# 谷歌投资旨在“加速科学”的秘密创业公司

![](img/dac2615b1509a9112683153e5b02535f.png)

[谷歌风投](https://web.archive.org/web/20230204145407/http://www.crunchbase.com/financial-organization/google-ventures)已经悄悄投资了一家名为 [Wingu](https://web.archive.org/web/20230204145407/https://wingu.com/login) 的秘密创业公司，报道 [StrategyFacts](https://web.archive.org/web/20230204145407/http://www.strategyfacts.com/) (需要订阅)。

事实上，虽然谷歌风投网站[为其位于马萨诸塞州剑桥的投资组合公司之一列出了](https://web.archive.org/web/20230204145407/http://jobs.googleventures.com/careers_home.php?Company=-1&Industry=%25&Function=%25&Location=%25&cmd=Search)四个职业机会，但没有提到 Wingu 的名字，这家秘密创业公司在其[求职栏](https://web.archive.org/web/20230204145407/http://wingu.theresumator.com/apply/)上发布了完全相同的职位空缺，没有留下任何想象空间。

Wingu 正在构建一个名为 [Elements](https://web.archive.org/web/20230204145407/https://wingu.com/static/policies/about.gsp) 的企业级云平台，这将使研究团队能够更有效地合作，并以“新的方式”使用数据。

根据 Wingu 的网站，其平台有四个主要卖点:

> 管理:将您的跨专业团队统一在一个公共平台上，以共享数据和想法。
> 
> 分析:利用我们的分析工作流和发现工具推动决策制定。
> 
> 分享:让您的研究人员跨越孤岛和地理界限，实现更好的协作、协调和沟通。
> 
> 保护:您可以放心，因为您的数据受到领先系统和安全专家的支持和保护。

Wingu 打算在订阅的基础上发布其研究集成软件。

这家初创公司的提议听起来与研究网络的[门德利](https://web.archive.org/web/20230204145407/http://www.crunchbase.com/company/mendeley)相似。其他类似的服务包括[Academia.edu](https://web.archive.org/web/20230204145407/http://www.academia.edu/)(我们把它比作[“研究人员的 Geni”](https://web.archive.org/web/20230204145407/https://techcrunch.com/2008/09/16/academiaedu-a-geni-for-researchers/))和 [Questia](https://web.archive.org/web/20230204145407/http://www.questia.com/Index.jsp) 。如果你知道任何其他的，请在评论中分享。
