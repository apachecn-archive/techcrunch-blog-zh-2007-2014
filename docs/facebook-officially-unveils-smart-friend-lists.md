# 脸书正式推出智能好友列表 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/13/facebook-officially-unveils-smart-friend-lists/>

本周早些时候，[我们报道了](https://web.archive.org/web/20230205043825/https://techcrunch.com/2011/09/08/facebook-begins-auto-grouping-colleagues-school-mates-and-local-friends-into-smart-lists/)脸书已经悄悄地激活了一个漂亮的新功能:智能好友列表，它使用算法生成你的朋友、同事、熟人和住在你附近的人的名单。现在，脸书公开宣布了智能好友列表，并提供了一些额外的细节。

当然，脸书上的好友列表已经存在多年了。但是大多数人从来没有为它们烦恼过，因为它们完全是一种痛苦——滚动浏览成百上千的朋友列表只是一件没有多少直接回报的琐事。但是脸书认为这个新功能将会改变这种情况，至少对一些用户来说是这样。

我已经了解了这个特性，但还没有机会亲自测试，它的工作方式和你预期的一样。首先，脸书会自动生成一些列表——一个是你的同事，一个是你的同学，还有一个是和你住在同一个城市的朋友。这是一个好的开始，也很容易让你自己列出清单。脸书说，如果你创建一个新的列表并添加几个朋友，网站会自动推荐它认为属于同一列表的其他朋友。

最后，有一个用星星装饰的列表，突出了你最好的朋友。脸书不会自动为这个名单推荐任何人——该公司认为用户只会希望包括五到十个人，最好让用户自己选择。该列表有一个特殊的属性:每当好友列表中的某人更新或发布照片时，无论变化有多小，你都会在你的订阅源中看到，并且你会(可选地)收到通知(通知是那些在屏幕顶部标有红色标记的提醒)。

还有一个名为“熟人”的新列表也有特殊属性，但顺序相反:添加某人到其中，只有最重要的身份变化(婚姻、新工作等)。)将出现在您的新闻订阅源中；你会忽略他们每天的帖子。

一旦您创建了新的朋友列表，您就可以像以前一样使用它们:即，您可以有选择地与某些组共享内容，并且可以更改每个列表的个人资料可见性权限。

毫无疑问，今天的发布会将你与 Google+相提并论，Google+非常强调将你的朋友分类到圈子里，这类似于脸书的朋友列表。但是尽管如此强调，脸书实际上已经抢先 Google+一步推荐*你把谁*放入这些群组——Google+很擅长发现你可能有兴趣关注的人，但是它还不能帮你分类。
