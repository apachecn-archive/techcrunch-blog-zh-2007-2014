用WizeHive (Beta版邀请)TechCrunch组织你的公司

原文：https://web.archive.org/web/https://techcrunch.com/2008/12/30/organize-your-company-with-wizehive-beta-invites/

Yammer肯定挑起了什么事端。今天，企业Twitter服务面临来自 Wizehive 的更多竞争，这是一个基于网络的群组信息和任务管理服务。WizeHive刚刚推出测试版。我们有500个邀请(当你注册时只需输入“TC2009”)。

WizeHive是一家自助创业公司，由创始人迈克·卡森(Mike Carson)和迈克尔·莱文森(Michael Levinson)出资10万美元。莱文森还是费城一家类似YCombinator的创业孵化器 DreamIt Ventures 的创始人。DreamIt本身没有投资，但Levinson在管理所有不同的DreamIt项目和申请人的挫折中创建了WizeHive。莱文森说:

我需要一些东西来为DreamIt组织我所有的想法和活动。东西飞得到处都是。我看了看Basecamp和Central Desktop，我认为它们不够直观，界面设计的方式对于我的合作伙伴(他们是业务合作伙伴，不是技术人员)来说不够直观。

虽然它在许多方面与 Yammer 、 Present.ly (我们的评测)、Basecamp、Central Desktop类似，甚至在某些方面与 Producteev (我们的评测)类似，但WizeHive是一个有价值的竞争对手，并增加了自己的一些变化。

像Yammer和其他软件一样，WizeHive允许你建立工作组，并以类似Twitter的方式向每个人显示消息。但是对话是有线索的。您可以点按以查看整个线程(最近的消息也会出现在流的顶部)。WizeHive在iPhone和其他移动浏览器上也能很好地工作(黑莓和Android优化即将推出)，你可以将其作为桌面Adobe Air客户端或在电子邮件中接收提醒。(如果您抄送notes@wizehive，并将工作区放在电子邮件主题行的括号中，wizehive将接收并分类电子邮件的内容，包括附件)。很快，你也可以在你的Twitter feed中获得WizeHive更新。

WizeHive还增加了任务管理功能，如 Basecamp 、 Huddle 和Producteev。您可以创建任务，设置截止日期，并将其分配给自己或其他人。每个任务都可以标记为“未完成”、“已完成”或“正在进行中”你也可以上传文件，每个人都可以在浏览器中查看。图像和文件可以附加到特定的邮件中。

不过，WizeHive最与众不同的特性是，它允许您为每个工作区创建一个小型数据库，并带有可定制的数据字段。这对于管理联系人或组织活动非常有用。WizeHive计划增加额外的应用程序，并收取订阅费。将于明年1月推出的前两个应用程序将是一个用于CRM目的的简单商机跟踪应用程序和一个用于时间表的应用程序。WizeHive已经有了一个 API ，并希望其他开发者也能开发自己的插件。

WizeHive对最多三个人免费，最多50MB的存储空间，然后f ees对一个小组每月8美元起，额外的存储空间需要额外付费。基于API的附加应用每月花费不到10美元，这也是莱文森认为他能赚最多钱的地方。
