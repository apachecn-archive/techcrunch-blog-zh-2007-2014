# 本·霍洛维茨:“脸书是技术领域运营最好的公司”

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/18/ben-horowitz-facebook-is-the-best-run-company-in-technology/>

今天，在 Web 2.0 峰会上，Opsware 的前联合创始人兼首席执行官、知名风险投资公司 Andreessen Horowitz 的创始合伙人 Ben Horowitz 今天走上舞台，与《纽约杂志》的约翰·海勒曼对话。Horowitz 谈到了他和他的创始合伙人的历史，以及这两位技术老手在他们投资和指导的公司中寻找什么。

有趣的是，在演讲的最后，海尔曼和霍洛维茨玩了一个单词联想游戏，问他对脸书(以及其他大型科技公司)的看法。霍洛维茨坦言，他认为这家社交网络巨头是硅谷——以及科技行业——运营最好的公司。

当被问及这是首席执行官马克·扎克伯格还是他周围的人的产物时，霍洛维茨说，当然，他认为这是两者的产物。“我非常关注年轻的首席执行官，我认为马克是业内最优秀的年轻首席执行官之一”。这其中的原因？霍洛维茨说，扎克伯格对产品战略考虑得非常周到。

他举了一个例子，当新员工来到脸书时，他们经常被迫接受比他们习惯的头衔更低的头衔。扎克伯格开创了这个先例，因为他希望来到脸书的人们不要把注意力放在他们的头衔上，而是放在他们将会制造出什么样的令人敬畏的产品上。他想确保他们来脸书是出于正确的原因——而不是因为他们看过《社交网络》。

霍洛维茨说，扎克非常关注小事情，这将使他长期成功。正是这种对细节的关注，以及对长期路线图的专注，推动了这位年轻的首席执行官和他周围的人，推动他们为公司各级人员提供相同的入职流程。

这位投资者表示:“如果你对制造产品不感兴趣，你就不会在脸书打球。”。

鉴于这位年轻的首席执行官多年来因在舞台上表现不佳、缺乏魅力等问题而备受指责，霍洛维茨的话有点令人惊讶。虽然安德森·霍洛维茨没有在脸书正式投资，但马克·安德森确实是董事会成员。由于 Horowitz 承认他和 Andreessen 是“最好的朋友”,就像一对“老夫老妻”,显然这家年轻的公司是两位投资者都非常熟悉的。

霍洛维茨还分享了他对 Groupon 和 Zynga 等其他科技行业名人的看法。他称 Groupon 是“世界上发展最快的公司”，称 Twitter 正在“改变世界”，除了“我爱它”之外，他无法对 Zynga(即将上市)说更多的话。关于史蒂夫·乔布斯，这位投资者说“他在这个世界上和他的生活中所做的一切是如此壮观。我喜欢有多少人一次又一次地被他证明是错的”。
