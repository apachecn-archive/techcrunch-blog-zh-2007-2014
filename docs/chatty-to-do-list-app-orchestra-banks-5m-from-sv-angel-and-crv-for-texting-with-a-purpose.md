# 聊天待办清单应用 Orchestra Banks 从 SV Angel 和 CRV 那里获得 500 万美元，用于“有目的地发短信”

圆滑的实时聊天+待办事项应用程序 [Orchestra](https://web.archive.org/web/20230203062909/http://www.orchestra.com/) 今天早上宣布其融资，刚刚结束的 500 万美元首轮融资来自 [Charles River Ventures](https://web.archive.org/web/20230203062909/http://www.crv.com/) 和 [SV Angel](https://web.archive.org/web/20230203062909/http://svangel.com/) ，此外还有来自 [Kapor Capital](https://web.archive.org/web/20230203062909/http://www.kaporcapital.com/) 和其他天使投资人的 50 万美元种子资金。

Orchestra 的联合创始人 Gentry Underwood 将这款免费应用描述为“有目的的短信”，但也将其比作“非常简单的任务清单”；“我们已经拿走了所有不必要的东西。”由于联合创始人安德伍德和坎侬都曾在 IDEO 和苹果公司工作过，难怪 Orchestra 的界面非常用户友好——它可以无缝地指导你在首次下载时通过点击、滑动和摇动手势进行导航。本着同样的“无摩擦”精神，该应用程序还会自动同步到 http://www.orchestra.com。

安德伍德告诉我，Orchestra 与 OmniFocus 等其他待办事项列表服务的最大区别在于，它的核心是社交。Orchestra 可以让你说话(就像“穷人的 Siri”)或输入任务，然后在应用程序内或在 Twitter 和脸书上与他人分享。用户不需要管弦乐队帐户来响应任务，你甚至可以通过将电子邮件转发给 tasks@orchestra.com 来将它们变成管弦乐队任务。

自发布以来的三周里，Orchestra 一直是 iTunes 全球各应用商店的“本周应用”，并被 Lifehacker 选为[“iPhone 最佳待办应用”。](https://web.archive.org/web/20230203062909/http://lifehacker.com/5851313/the-best-to+do-app-for-iphone)已经发布了超过 120 万个任务，其中 1/4 被分配给了其他人。设计超级明星[杰克·多西](https://web.archive.org/web/20230203062909/http://www.crunchbase.com/person/jack-dorsey)，他从未见过任何一位联合创始人，甚至[也在推特](https://web.archive.org/web/20230203062909/http://twitter.com/#!/jack/status/116214769656152065)上谈论过这件事。

http://twitter.com/#!/杰克/状态/116214769656152065

安德伍德计划利用这笔新资金充实他在帕洛阿尔托的七人团队，并专门寻找工程师和产品设计师。“这里有潜力建立一个平台，而不是一个产品，”他说。该应用的安卓和 iPad 版本也在开发中。
