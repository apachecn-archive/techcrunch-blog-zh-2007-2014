动态研究收购专业联系人经理Gist TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/02/14/rim-acquires-professional-contact-manager-gist/

动态研究获得专业联络经理要点

我们听到谣言称RIM公司参与了购买职业联络经理 Gist、的严肃谈判，今天这是官方消息。RIM已经收购了 Gist，我们已经和公司确认了。交易的财务条款没有披露。

以下是来自Gist的声明:今天，我们宣布Gist已经成为运动研究(RIM)的一部分。我们对RIM的未来以及全球数百万黑莓用户将如何使用Gist感到非常兴奋。这是朝着我们的目标迈出的一大步，我们的目标是利用基于网络的Gist体验，让用户建立更强大的专业关系。请继续关注更多细节，感谢你们所有人的支持和鼓励，让我们在Gist进入下一个伟大的阶段。RIM也在其黑莓博客上证实了此次收购。

Gist于2008年作为一个“专业”的沟通管理公司成立，旨在满足商业用户的过滤需求，筹集了1080万美元的资金。像网络服务一样，该应用程序根据你的电子邮件联系人组织你的信息流，因此这是一种跟踪你的同事、朋友和专业联系人的有用方式。

总部位于西雅图的Gist提供的服务可以帮助你密切关注你的职业网络中的人，并允许你查看每个联系人过去的消息和附件，关于他们公司的新闻，以及他们在Twitter等服务上的最新消息。该公司还提供许多移动应用程序，看起来RIM将为黑莓用户使用这项技术。

Gist由T.A. McCann创建，他是一名企业家，也是微软Exchange Server Group的前高级员工，最初由微软联合创始人保罗·艾伦的Vulcan Capital提供资金。去年，Gist 收购了Learn That Name，这款游戏利用你在LinkedIn上的联系人来帮助你记住商业上熟人的名字，还为Gmail推出了一款小工具。

正如奥姆·马利克在去年12月写的那样，RIM利用Gist将更多的社交元素融入黑莓电话簿，并利用将更多的社交元素融入电子邮件，这是有道理的。Gist表示，用户可以继续使用其平台，但看看RIM是否会关闭这项服务，转而专门为黑莓用户提供这项服务，这将是一件有趣的事情。我可以想象，如果是这样的话，许多Gist用户将会非常失望。
