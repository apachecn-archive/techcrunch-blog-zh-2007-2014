# 手动操作来自 AniApp 的 AppXRacer

如果你是鹦鹉 AR 的粉丝。无人驾驶飞机，你可能会从它的地球亲戚 AniApp 的 AirXRacer 中得到乐趣。我在深圳拿到了这部 iPhone 或 Android 控制的赛车的第一手资料，你可以在这个假期拿到它。

这辆车是一辆标准的遥控赛车，如你所见，速度非常快。你可以使用手机的加速度计或屏幕上的“滚轮”来控制它，这种方法提供了更多的技巧。AniApp 实验室的杰夫·罗(Jeff Luo)表示，这些汽车将于今年假期在 Radio Shack 和 Brookstone 上市，应用程序[现在可以下载](https://web.archive.org/web/20221006141507/http://aniapp.com/)。

![](img/90848623a47a8146148627a077daadd7.png)

在黑暗中录像有点困难，但请放心，控制方案是可靠的，汽车相当坚固，能够击中椅子腿和脚踝而不粉碎。还没有定价，但是在你附近的按摩椅经销商那里找找吧。
