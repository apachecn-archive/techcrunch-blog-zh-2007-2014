# 弗莱克走向死亡池，因为没有人想注释网络

[![](img/74f7787c7cec3486ee447db1b05a598f.png)](https://web.archive.org/web/20221007170955/http://www.fleck.com/)

就在 [ReFrame 推出](https://web.archive.org/web/20221007170955/http://www.beta.techcrunch.com/2008/10/08/reframe-it-retreads-web-annotation-as-a-browser-add-on/)让人们给网页加注释的服务后一周，另一家自 2006 年以来一直在做[几乎同样事情的初创公司](https://web.archive.org/web/20221007170955/http://www.beta.techcrunch.com/2006/11/16/fleck-offers-zero-friction-web-annotation/)， [Fleck](https://web.archive.org/web/20221007170955/http://www.fleck.com/) 也准备出售自己。Fleck 是一家由 Boris Veldhuijzen van Zanten 和 Patrick de Laive 创立的荷兰公司，他们是著名的西装革履的企业家，有一次他们在 Mike 的家里叫醒他去推销他们的创业公司。二人组还负责[下一次网络会议](https://web.archive.org/web/20221007170955/http://2008.thenextweb.org/)(披露:去年我是主持人)和博客，他们现在将专注于此。

我问鲍里斯，为什么弗莱克从未真正起飞，为什么他想脱手。他的回答是:

> *版本 1 太有限，版本 2 太复杂。最新版本很棒。现在它需要的只是另一次推动和一些关注。我们不能给它，因为我们现在有其他的兴趣(博客&会议和我们在 Twones 和 Wakoopa 的股份),而且我们没有资金。是时候换个新主人了！*

这项服务最初是作为一种给网页添加便签的方式出现的。这样当其他 Fleck 成员访问该页面时，你就可以看到他们的笔记。这个想法吸引了许多企业家(见，第三声音，Stickis，Diigo，ShiftSpace，TrailFire ),但迄今为止未能流行起来。Fleck 已经发展到包括书签共享、标记和交友功能。现在鲍里斯和帕特里克希望收回投资者在上一轮投资中对这项服务的 100 万欧元的估价。

祝你好运，因为如果他们卖不出去，这东西就要进死牢了。
