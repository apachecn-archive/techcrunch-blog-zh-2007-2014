# 发布 Atlas 应用程序，让管理日历变得更轻松 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/05/07/atlas-unsuck-calendaring/>

如果你和我一样，你的日程表是一团糟。你实际上有多个日历，通过这些日历，你试图记录各种不同的会议，你总是在你的电子邮件和日历之间来回切换，试图记录你在什么时候会见了谁，试图不重复预订——但往往失败了。

在过去的几个月里，出现了许多日历和所谓的生产力应用程序，如 Donna、Tempo、Fantastical 等等。但是它们都试图简化跟踪和参加所有会议的过程。很少有人真正简化设置过程。

这就是阿特拉斯的用武之地。

Atlas 是一款手机应用程序，它不仅可以通知你开会的时间和地点，更重要的是，它可以安排会议。该应用程序通过一个简单的、一步一步的过程让你与其他人联系起来，让你经历安排会议的复杂过程。

不用在多个浏览器窗口之间来回切换(一个用于电子邮件，一个用于日历)并试图以这种方式安排事情…不用在会议参与者之间来回发送多封电子邮件，Atlas 提供了一种简单的方法来在您的手机上创建会议，然后将这些会议导入到您选择的日历程序中。

使用 Atlas，用户只需创建一个活动，然后邀请人们参与其中。如果用户正在寻找多个可能的会议时间，他们可以发送几个合适的时间，让其他参与者选择一个适合他们的时间。发起会议的创建者可以看到每个人都有空的矩阵，然后可以选择对每个人都合适的时间。与此同时，在等待最后的会议时间时，这些时间被封锁起来，这样你就不会意外地重复预定。与会者也可以还价地点和时间等，因为他们可能知道比会议创建者更好的寿司店。

有点像以前的 Tungle.me，嗯，[你知道](https://web.archive.org/web/20221006143959/https://beta.techcrunch.com/2012/09/18/rim-to-shut-down-tungle-me-team-will-focus-on-blackberry-10-calendar-app/)。哦，对了，首先是手机。**

Atlas 没有试图接管你的日历，这很好。相反，它适用于您使用的日历。这包括 GCal、iCal、YCal 和 Outlook。它也适用于较新的日历应用——你知道，世界上的节奏、日出和幻想。

无论如何，虽然 Atlas 有很多很酷的功能，如果你是超级用户，它会取消你的日历，但它可能不是每个人的最佳选择。尽管如此，可能值得一试，因为嘿，任何事情都比两个浏览器窗口和来回的电子邮件好，对不对？
