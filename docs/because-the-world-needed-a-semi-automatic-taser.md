# 因为这个世界需要半自动泰瑟枪

世界各地的暴徒正在学习如何智取泰瑟枪。只要你们不止一个人，你们就可以逍遥法外！空气泰瑟枪的单发性质也总是让受害者感到沮丧:一旦你击倒一个傻瓜，你就完了！不，这是给你老爸的。但是不要害怕，泰瑟枪能听到你的叫声(以及那些被你打过的人的叫声)，因为他们现在已经发明了一种三发泰瑟枪，可以击倒多个攻击者。或者是单个攻击者的三倍惩罚。我看不出这对任何人有什么问题。

![x3_01-web](img/c9b03be791d5945ad5334b0e3f3b9e65.png "x3_01-web")

在上面的视频中，三个自愿的参与者被电击(电击？)在后面，而他们脸上的表情是无价的。泰瑟枪 X3，因为这种设备被称为，包装足够的果汁，使三个年轻女子失去能力，但也调整电压，所以没有真正的伤害。我认为这里没有人相信这一点，但至少他们承认伤害是一种遥远的可能性。
