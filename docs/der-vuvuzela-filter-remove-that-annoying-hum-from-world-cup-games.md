# 呜呜祖拉过滤器:消除世界杯比赛中恼人的嗡嗡声

如果你看过今年世界杯的任何一场比赛，你都会注意到[呜呜祖拉](https://web.archive.org/web/20221205093509/http://en.wikipedia.org/wiki/Vuvuzela)无休止的高音嗡嗡声。它基本上是一个非常烦人的喇叭。

总之，一名德国工程师发明了一种呜呜祖拉过滤器，它可以滤除环境噪音，并将其从音轨中移除。这很复杂，而且是德语的，但是如果我看足球的话，我希望这是一个去除噪音的内置硬件盒。
 可悲的是，它本身并不是一个“插件”，而是一个逻辑设置，所以你真的不能在观看它的时候去芜存菁。点击查看项目[。](https://web.archive.org/web/20221205093509/http://www.surfpoeten.de/tube/vuvuzela_filter)
