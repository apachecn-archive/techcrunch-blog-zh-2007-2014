# 贾斯汀·坎的执行官今天开始为公司和创业公司跑腿

你每年要付给你的开发商 15 万美元外加零钱。为什么要让他们被干洗或购物分心呢？

虽然像谷歌和脸书这样的大公司已经意识到他们需要在校园里引进理发师、洗车服务和美食来吸引人才，但是小公司不具备这些奢侈品的规模。

这就是为什么 Justin.tv 创始人贾斯汀·坎(Justin Kan)的任务管理服务公司[Exec](https://web.archive.org/web/20221210023612/https://iamexec.com/exec_for_companies)[今天](https://web.archive.org/web/20221210023612/https://iamexec.com/exec_for_companies)开设公司账户。公司将可以建立一个账户，每个月分配一定数量的钱给员工，让他们委派不想做的差事。如果他们不使用积分，那么他们将进入下个月。

[Exec 是一种任务管理服务，让人们按需跑腿](https://web.archive.org/web/20221210023612/https://beta.techcrunch.com/2012/02/29/exec-justin-tv/)，比如复制钥匙或打扫卫生，每小时收费 25 美元。与其他服务不同，它不会让用户管理投标过程和审查报价。

“如果你想让某人做一场演出，通常会有很多步骤，”Kan 说。“我在用精神能量来节省体力。我要用钱来代替脑力和体力。我希望这需要尽可能少的思考。”

Kan 说，他正在创建公司账户，因为有太多的初创公司要求他这么做。移动支付处理初创公司 Stripe 使用 Exec 从宜家获得啤酒和 50 把椅子，送到他们的办公室参加开发人员会议。Y Combinator 旗下的个人搜索公司 Greplin 也使用 Exec 从 Trader Joe's 为办公室购买食物。

目前，Kan 专注于保护用户体验，而不是过快增长。他说，该应用 91%的评论是五星评级，98%是四星或五星评级。Exec 目前仅在旧金山可用。

“对于这样的应用程序，第一次对你不起作用就是你最后一次使用它，”他说。他雇佣了大量工人，并通过多步培训流程确保目前供应超过需求(就像优步刚推出时必须做的那样)。
