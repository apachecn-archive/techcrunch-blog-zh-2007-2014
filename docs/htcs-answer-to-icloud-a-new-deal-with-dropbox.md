# HTC 对 iCloud 的回答:与 Dropbox 达成新协议

今晚在巴塞罗那，在 HTC Android 手机发布会中间，HTC 首席执行官 Peter Chou 透露了一个新的进展，这是该手机制造商对苹果 iCloud 的回应:与 Dropbox 达成协议，Dropbox 目前是云存储领域值得关注的名字。

HTC 表示，购买该公司新款旗舰设备 HTC One 的用户将获得两年免费的 25g 存储空间——考虑到目前 50g 存储空间每月花费 9.99 美元(2g 是唯一提供的免费服务)，这是一笔大交易。

Chou 说，存储空间足以存储 10，000 张高质量的照片，除此之外还有更多:这个消息是在 HTC 宣布其新系列 Android 设备的一系列内容和服务增强的同时发布的，包括改进的摄像头；一项新的音乐服务，包括去年 Beats audio 服务的扩展；增强的视频服务；和增强的集成以在车内使用该设备。

这都是 HTC 大反击战略的一部分，去年该公司在智能手机销售方面的势头相对于三星和苹果有所下滑。

它很久以前就意识到，为了与苹果竞争，它需要有引人注目的设备，但也需要有引人注目的服务——这就是它试图通过 Dropbox 和今天发布的其他内容来实现的目标。

我们稍后将采访 HTC 的高管，并亲自体验这些设备，我们会不断更新。
