# Firefox 操作系统与手机、应用程序市场和 18 家注册开放网络的运营商一起运行

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/02/24/firefox-os-smartphones-marketplace/amp/>

火狐操作系统，来自 Mozilla 的新的 HTML5 友好的移动操作系统，今天在其战略上向前迈出了一大步，成为目前由谷歌的 Android 和苹果的 iOS 主导的智能手机领域中可行的第三个玩家。Mozilla[宣布](https://web.archive.org/web/20230302063140/http://www.mozilla.org/en-US/firefox/partners/)18 家运营商现已承诺其开放网络 HTML5 设备推送；推出 Firefox Marketplace 应用商店，为平台聚合内容；一些首批低成本手机将于今年夏天从其运营商合作伙伴中推出。

Telefonica——其中兴制造的手机如图所示——将在拉丁美洲和西班牙销售其首款 Firefox 手机；德国电信将在波兰推出其手机，然后扩展到东欧的其他市场。今天宣布手机的其他运营商包括拉丁美洲的巨头 America Movil(首先从其本土市场墨西哥开始)和挪威的 Telenor，后者的业务遍及东欧和亚洲，包括马来西亚、巴基斯坦和孟加拉国。

这种想法是，在智能手机整体渗透率较低的市场，以及对低成本手机感兴趣的客户群(相对于 iPhone 或三星 Galaxy 设备等高端设备)，运营商可能有更好的机会。Telefonica Digital 首席执行官 Matthew Key 对 TechCrunch 表示:“在拉丁美洲，82%的人口目前没有智能手机，因此我们有潜力在这些发展中经济体开拓市场。”“他们处于不同的发展阶段。”

与此同时，Firefox Marketplace 已经签下了许多早期合作伙伴:Zeptolab,《割断绳子》的制作人；迪士尼手机游戏；EA 游戏；脸书；巴西音乐电视网；诺基亚在这里的制图工作；SoundCloud 和 Twitter 都将推出针对 Firefox OS 手机优化的基于 HTML5 的网络应用。鉴于流行的应用程序是吸引用户使用不同设备的关键部分，Mozilla 在早期阶段表明它正在挑选该领域的关键品牌是至关重要的。

今天在巴塞罗纳世界移动通信大会的新闻发布会上宣布的 18 家新运营商是拉美巨头美洲电信、中国联通、德国电信、Etisalat、和记三集团、日本 KDDI、韩国 KT、俄罗斯 MegaFon、Qtel、新加坡电信、Smart、Sprint、意大利电信集团、西班牙电信、Telenor、Telstra、葡萄牙 TMN 和俄罗斯 VimpelCom。

这是自 2012 年 7 月宣布七家(德国电信、Etisalat、Smart、Sprint、意大利电信、西班牙电信和 Telenor)的早期名单以来，Mozilla 合作伙伴网络的首次重大更新，它表明该集团继续与在智能手机市场寻找可行的第三条道路的运营商取得进展——理想的是，在如何在新设备上部署服务方面，给予这些运营商更多发言权和更大份额。

“欧洲第一款搭载新 Firefox 操作系统的智能手机的推出，是不同生态系统之间竞争加剧的重要一步，”德国电信董事长雷内·奥伯曼(Rene Obermann)在一份声明中指出。

尽管投了信任票，但在一个目前由 Android 和 iPhone 主导的世界里，争夺智能手机市场份额将是一场漫长而艰苦的战斗。来自 Strategy Analytics 的一份[报告估计，Firefox OS 将仅占今年智能手机出货量的 1%,而 Android 将占据 67%的份额。](https://web.archive.org/web/20230302063140/https://techcrunch.com/2012/09/27/firefox-os-will-only-take-a-niche-1-of-smartphone-shipments-in-2013-predicts-strategy-analytics/)

“毫无疑问，这将是一个长期的燃烧期，”西班牙电信的基承认道。"这不是一开始就能一炮而红的东西。"这也是 Telefonica 最初选择四个国家作为目标，而不是覆盖 25 个国家的原因之一。他说，这些只会在“一段时间后”出现。

事实上，火狐操作系统在很大程度上被用作瞄准后来的智能手机用户的杠杆，他们可能会在手机上花费更少，这也将给诺基亚带来更大的压力，T4 也在 Windows Phone 手机的 Lumia 系列中制定低成本设备战略，以与华为和中兴等公司竞争，现在他们既生产低成本的 Android 手机，也生产低成本的火狐操作系统手机。

Firefox OS 基于 HTML5 的框架建立在开放网络标准上，这意味着该设备的每个方面——甚至包括电话拨号——都是作为 HTML5 应用程序构建的。Firefox 表示，与 HTML5 通常在 Android 或 iOS 设备上的工作方式相比，这种方式可以实现更快的性能，在这些平台上，web 应用经常被视为不如本地应用。

它还让运营商有机会定制和本地化这些界面和服务——包括应用程序，也包括 Firefox market place——因为他们选择了 iOS 或 Android 所不具备的灵活性水平(除非他们决定开发后者的分叉版本，然后切断对 Google Play 应用商店的访问)。

例如，在德国电信的案例中，该运营商表示，它将预装“战略相关的构建模块，如安全模块”，但它也补充说，它正在为手机制造商提供“网络运营商的视角”技术要求。

Mozilla 首席执行官 Gary Kovacs 在一份声明中表示:“Firefox OS 为世界各地的移动用户带来了开放网络的自由和无限创新。“在我们充满活力的社区和专注的合作伙伴的支持下，我们的目标是创造公平的竞争环境，迎来内容和服务的爆炸式增长，满足未来 20 亿网民的各种需求。”

虽然德国电信最大的市场是在德国和美国，以及英国，在英国，它是法国电信合资企业 Everything Everywhere 的合作伙伴，但这些不是它第一批 Firefox OS 设备的目标市场——至少最初不是。相反，它的第一款设备 Alcatel One Touch Fire 将于今年夏天在波兰推出，“2013 年将在东欧其他国家推出，”该公司表示。

类似地，Telefonica 将在拉丁美洲的三个市场——巴西、哥伦比亚和委内瑞拉——以及西班牙推出其首款 Firefox OS 手机，西班牙市场对低价智能手机表现出了强烈的兴趣。像 DT 一样，Telefonica 将推出 Alcatel One Touch，此外，它还将提供 LG 和中兴的设备。

我们正在巴塞罗那的火狐操作系统新闻发布会上直播，我们会继续更新。
