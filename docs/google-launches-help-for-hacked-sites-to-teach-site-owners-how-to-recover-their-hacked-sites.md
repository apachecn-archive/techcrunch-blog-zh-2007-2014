# 谷歌推出“帮助被黑网站”来教网站所有者如何恢复他们被黑的网站

没有一个网站可以完全避免被黑客攻击，但是每个网站所有者都可以做一些显而易见的事情，让黑客更难闯入网络服务器，添加流氓链接或完全接管网站。今天，谷歌[推出了](https://web.archive.org/web/20221209115130/http://googledevelopers.blogspot.com/2013/03/friends-help-friends-recover-their.html)其新的“[帮助被黑网站](https://web.archive.org/web/20221209115130/http://www.google.com/webmasters/hacked/)”系列，以[教网站管理员](https://web.archive.org/web/20221209115130/http://googlewebmastercentral.blogspot.com/2013/03/new-first-stop-for-hacked-site-recovery.html)如何避免被黑——以及如果被黑，如何恢复他们的网站。

本系列的第一部分面向相对非技术用户，而后一部分面向能够阅读代码并习惯使用终端命令的用户。总的来说，该系列有大约 80 分钟的视频和十几篇文章，涵盖了从基本的事情，如确定一个站点实际上被黑客攻击到[与您的主机](https://web.archive.org/web/20221209115130/http://support.google.com/webmasters/bin/answer.py?hl=en&answer=3011658)一起工作来恢复一个站点，一直到使用漏洞扫描器、理解 SQL 注入、读取日志文件和使用 shell 登录到您的站点来确定[根本原因漏洞](https://web.archive.org/web/20221209115130/http://support.google.com/webmasters/bin/answer.py?hl=en&answer=3013681)。

[![hacked_google](img/d985ce5a068c50aed9c0a5a308f1427e.png)](https://web.archive.org/web/20221209115130/https://beta.techcrunch.com/2013/03/12/google-launches-help-for-hacked-sites-to-teach-site-owners-how-to-recover-their-hacked-sites/hacked_google/)

该系列还解释了用户如何确保谷歌在清理了他们的网站后，删除谷歌搜索上的[可怕的](https://web.archive.org/web/20221209115130/http://support.google.com/websearch/bin/answer.py?hl=en&answer=45449)*该网站可能会危害你的计算机**该网站可能会受到危害*链接。

“虽然我们试图概述恢复的必要步骤，”谷歌开发者项目技术负责人 [Maile Ohye](https://web.archive.org/web/20221209115130/https://plus.google.com/u/0/111983349552187459374?rel=author) 在今天的声明中承认，“除非网站所有者拥有系统管理员命令的高级知识和源代码经验，否则每项任务对他们来说都相当困难。”正因为如此，谷歌还让网站所有者到其网站管理员论坛寻求额外的帮助。
