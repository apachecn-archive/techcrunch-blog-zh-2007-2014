# Yelp 与 Locu 合作，允许企业实时向 Yelp 发布菜单、每日特色菜和照片 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/23/yelp-partners-with-locu-allowing-businesses-to-post-menus-daily-specials-photos-to-yelp-in-real-time/>

当地商业数据提供商 [Locu](https://web.archive.org/web/20221209041338/http://www.locu.com/) 今天宣布与 Yelp 建立合作关系，以帮助用 Locu 直接来自企业的实时信息更新 Yelp 的菜单和其他列表数据。这将允许 Locu 上的餐厅向 Yelp 分发他们的菜单和每日特色菜，但这也将有助于 Locu 进一步扩展到餐厅以外的其他垂直领域。

今年春天早些时候，Locu[与 WordPress 制造商 automatic](https://web.archive.org/web/20221209041338/https://beta.techcrunch.com/2013/03/07/locu-partners-with-automattic-to-bring-its-menus-to-wordpress-coms-restaurant-sites/)合作，允许餐馆更容易地将 Locu 支持的菜单添加到他们的 WordPress 博客中。该公司还忙于支持其他第三方，包括 [OpenTable](https://web.archive.org/web/20221209041338/https://beta.techcrunch.com/2012/11/15/locu-partners-with-opentable-for-data-lets-restaurant-owners-edit-menus-in-real-time/) 、[猫途鹰和 CitySearch](https://web.archive.org/web/20221209041338/https://beta.techcrunch.com/2013/01/15/following-opentable-deal-restaurant-data-provider-locu-adds-more-big-name-partners-tripadvisor-and-citysearch/) 。

这家初创公司获得了 400 万美元的首轮融资，从商业网站中提取数据，然后以结构化的格式提供这些数据，包括通过其 API。声称自己拥有预先填充的 Locu 个人资料的企业主，或者自己注册该服务的企业主，将有权根据需要编辑数据，然后将数据发布到他们自己的网站、移动网站、脸书页面、WordPress 博客和其他本地目录，比如现在的 Yelp。

联合创始人兼首席执行官 Rene Reinsberg 解释说，与传统的本地企业列表提供商相比，使用 Locu 的好处是 Locu 可以提供更新的速度。“如果你看看历史上的提供商，他们通常是每周一次，每月一次，在某些情况下每天一次——但这真的是例外——提供商会获得订阅和更新，”他说。“这里(Locu)的概念是拥有实时功能。”

“无论如何，这是我们需要前进的方向，”Reinsberg 补充道，“因为今天很多搜索都发生在手机上，所以它们是即时的——你现在正在寻找一些东西。”

对于仍是 Locu 大部分用户群的餐馆来说，Yelp 整合意味着他们将能够让 Yelp 用户在网络和手机上看到他们当前的菜单，包括每日特色菜。他们的菜单将出现在企业的个人资料页面上，Yelp 的数据(包括照片)位于页面上方。[这里有一个](https://web.archive.org/web/20221209041338/http://m.yelp.com/menu/radius-san-francisco/dinner)的例子。

Locu 的平台上现在有超过 15，000 个商家。该公司提供免费增值服务，付费用户每月支付 25 美元，就可以获得一系列额外的付费选项、模板和支持[。这包括试吃 Locu 外卖的能力，这是一种餐馆在线订购的选择，以及将照片发布到脸书、OpenTable，现在还有 Yelp。](https://web.archive.org/web/20221209041338/https://locu.com/pricing/)

当 Locu [去年 10 月](https://web.archive.org/web/20221209041338/https://beta.techcrunch.com/2012/10/02/locu-launches-to-merchants-now-local-businesses-can-update-once-to-publish-prices-to-web-mobile-facebook/)推出其商业平台时，该公司表示，其约 80%的商业数据来自餐馆，因为这些是其收集数据时的最初重点。如今，餐馆和其他行业的分野并没有太大变化。

因此，对于该公司来说，这种合作关系也将他们的服务开放给垂直餐厅以外的更多潜在客户——Yelp 提供了约 5000 万个企业的接入，使其成为 Locu 迄今为止最大的合作伙伴。“这对我们来说是一大步，因为我们有在其他垂直领域增长的雄心，”Reinsberg 说。“这是一笔非常重要的交易。”
