# MongoDB 制造商 10Gen 结束了来自美国情报投资者的一轮未披露的投资

[10gen](https://web.archive.org/web/20230321025126/http://10gen.com/) ，开源 NoSQL 数据库 [MongoDB](https://web.archive.org/web/20230321025126/http://mongodb.org/) 的发起公司，今天[宣布从](https://web.archive.org/web/20230321025126/http://www.iqt.org/press/2012/10gen.html) [In-Q-Tel](https://web.archive.org/web/20230321025126/http://www.iqt.org/) 获得笔未披露的投资，In-Q-Tel 是一家非营利性私募股权和风险投资公司，投资被认为有益于美国情报界的技术。In-Q-Tel 的典型投资据传在 100 万到 300 万美元之间。

该公司最初于 1999 年由中央情报局特许成立，但它是一个独立的组织，与多个情报机构合作，包括国家地理空间情报局(NGA)、国防情报局(DIA)和国土安全部科学和技术局(DHS·S&T)。[根据福布斯](https://web.archive.org/web/20230321025126/http://www.forbes.com/2010/11/19/in-q-tel-cia-venture-fund-business-washington-cia.html)报道，其预算是保密的，但是[华盛顿邮报在 2005 年报道](https://web.archive.org/web/20230321025126/http://www.washingtonpost.com/wp-dyn/content/article/2005/08/14/AR2005081401108.html)该公司每年从中央情报局获得 3700 万美元。

一篇 2000 年的国防情报期刊文章称:“中情局的领导层认识到，中情局过去没有、现在也不可能像商业市场中的企业那样，以同样的速度和敏捷争夺 IT 创新和人才，商业市场的业务受‘互联网时代’和利润的驱动。”。“该机构必须超越自身，不仅要表现为买方，还要表现为卖方。中情局必须向硅谷提供一些有价值的东西，一种硅谷理解的商业模式；这种模式为那些与 In-Q-Tel 携手的人提供了将他们的创新商业化的机会。”

除了资金，In-Q-Tel 还为其投资提供了来自情报界的“问题集”。

10gen 最后一次被看到是在由新企业联合公司(New Enterprise Associates)领投的一轮 4200 万美元融资中。在投资 In-Q-Tel 之前，该公司总共筹集了 7340 万美元。

毫不奇怪，In-Q-Tel[对大数据技术](https://web.archive.org/web/20230321025126/http://www.iqt.org/portfolio/practices.html)产生了兴趣。这一领域的其他投资包括 [Cloudera](https://web.archive.org/web/20230321025126/http://cloudera.com/) ，它是 [Apache Hadoop](https://web.archive.org/web/20230321025126/http://hadoop.apache.org/) 的开发者和支持者之一，以及[platform ora](https://web.archive.org/web/20230321025126/http://www.platfora.com/)，一家旨在使 Hadoop 更易于使用的公司。该公司最近的投资之一是对企业合作公司 Huddle 的投资。

它最著名的出口可能是 Keyhole，2004 年[卖给谷歌](https://web.archive.org/web/20230321025126/http://news.cnet.com/2100-1032_3-5428685.html)，最终成为谷歌地球。
