# 华为发布具有新“3D”用户界面的 Vision 智能手机

华为吸取了三星和 HTC 等巨头的经验，决定打造自己的用户界面。与 TouchWiz 或 Sense 不同，华为的 UI 带来了一些不同的东西:3D。现在，不要太兴奋，也不要太生气(如果你是一个反 3D 的家伙)，因为这不是令人讨厌的，在你面前，让你有点恶心的“3D”。这只是一个运行在 2D 屏幕上的 3D 界面。

浮华的 3D 界面真的比传统界面带来更多吗？我无法想象它会有那么大的革命性。希望它只是增强了用户体验的一点点美感，而不会造成不必要的混乱。

另一方面，规格和愿景的设计告诉我，华为对这一款寄予厚望。它的 1GHz 骁龙处理器很好地捆绑在一体成型的设计中。这款手机最薄处的尺寸为 0.39 英寸，重量略超过 0.27 磅，背面绑着一个 500 万像素的摄像头(支持 720p 视频)。

这款手机还配备了 microUSB 2.0 端口、512MB 内存和 2GB ROM，支持高达 32GB 的存储卡。愿景将推出三种颜色:银色、玫瑰金和木炭色。就膨胀软件而言，你实际上有一些非常好的应用程序——愤怒的小鸟、沥青 6:肾上腺素、在线秩序与混乱和游击队鲍勃高清。华为没有透露价格和可用性，但我们知道这款手机将在 9 月份的某个时候上架。
