# Mobile Roadie 现在为 Android 生态系统创建应用

![](img/c68cf537f86031cc2a5fea8ae759eb5e.png)

我们是 [Mobile Roadie、](https://web.archive.org/web/20221005224913/http://www.mobileroadie.com/)一家帮助[开发 iPhone 应用的初创公司的忠实粉丝。](https://web.archive.org/web/20221005224913/http://www.beta.techcrunch.com/2009/04/20/mobile-roadie-builds-bands-custom-iphone-apps-on-the-cheap/)但我们有一个抱怨，那就是移动 Roadie 仅限于 iPhone 平台。今天，我们的愿望实现了，因为 Mobile Roadie 正在为 Android 手机推出功能。

Mobile Roadie 平台的美妙之处在于，它提供了一个非常简单的几乎自动化的系统来构建应用程序，并在短短一周内将它们发布到苹果的应用商店。这家初创公司于今年早些时候推出，为其他会议、活动和场所以及音乐家、运动员、政治家和其他名人开发移动应用程序。这些应用程序可以为用户提供新闻、音乐、直播和录制的视频、照片、活动列表等内容。这些应用还集成了 YouTube、Brightcove、Flickr、Twitpic、Ustream、Topspin、Google News、RSS、Twitter 和脸书。

Mobile Roadie 平台的另一个好处是，它的 CMS 允许用户同时更新 iPhone 和 Android 应用程序。使用推送通知，客户可以发送出现在用户屏幕上的警报，将地理定位消息缩小到一英里半径。随着谷歌 Android 支持的推出，Mobile Roadie 将为阿什顿·库彻、多莉·帕顿和麦当娜的 iPhone 和 Android 应用程序提供支持。我们听说泰勒·斯威夫特也将很快推出应用程序。

Mobile Roadie 还为法国企业家和 Seesmic 创始人 Loic Le Meur 和他的妻子 Geraldine 组织的最重要的欧洲技术会议开发了 iPhone 官方应用。这个应用在会议上大受欢迎。Mobile Roadie 最近还与兰登书屋达成协议，为作者提供 iPhone 应用。

毫无疑问，Mobile Roadie 针对 Android 平台的功能将使其平台对用户更具吸引力。通过一个界面同时管理 iPhone 和 Android 应用程序的能力是一个引人注目的功能。凭借 Mobile Roadie 在过去一年的成功记录，我们将继续期待在不久的将来，这家自举创业公司会产生更多的创新。
