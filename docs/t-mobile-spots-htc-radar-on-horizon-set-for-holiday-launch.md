# T-Mobile 在地平线上发现 HTC Radar 4G，为假日发布做准备

看起来 HD7 不会是 T-Mobile 货架上唯一的 Windows Phone 手机了。就在昨天[大型智能手机发布会](https://web.archive.org/web/20230316000657/https://techcrunch.com/2011/09/26/t-mobile-galaxy-s2-amaze-4g-specs/)之后，T-Mobile 拉开了他们产品线中最新 Windows Phone 手机的帷幕:[HTC Radar 4G。](https://web.archive.org/web/20230316000657/http://newsroom.t-mobile.com/articles/htc-radar-4g)

这款雷达是 HTC 在 IFA 展示的两款 Windows 手机之一，虽然美国电话电报公司肯定会抓住这个大男孩，但雷达也不逊色。它拥有一个相当漂亮的白色铝制机身，而一个 800×480 的 3.8 英寸液晶显示屏则为雷达的前端增色不少。

与此同时，你会发现 1 GHz 的高通骁龙处理器正在运行，一个 500 万像素的摄像头牢固地安装在手机的后端，以完善包装。

当然，这些规格并不完全令人兴奋，但它们远远超出了 T-Mobile 最初的 Windows Phone 产品。HTC HD7 本质上是一款略有改动的 HD2，这意味着 Windows Phone 离开贝尔维尤时背负着过时的硬件。

鉴于宣布的时间，雷达运行雷德蒙期待已久的芒果版 WP7 应该不会令人惊讶。虽然我们在时间的问题上，T-Mobile 对 Radar 4G 的发布日期保持沉默，但他们很快向读者保证，它将在假期高峰时上市。
