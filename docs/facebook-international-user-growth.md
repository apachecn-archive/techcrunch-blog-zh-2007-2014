# 脸书向合作伙伴提交的可爱年度报告首次披露了各个国家的移动统计数据 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/12/29/facebook-international-user-growth/>

TechCrunch 获得了前所未有的指标，显示了脸书的国际增长。脸书给一些合伙人发了一本有趣的插图电子杂志，名为《年度》,但我从一个来源获得了一份副本，里面的统计数据是严肃的事情。该报告披露了一些主要国际市场的用户数量，如德国，目前拥有 2500 万用户和 1800 万移动用户。

此前，脸书只分享了多国地区的网络和移动用户总数，其移动用户总数为全球总数。问题是，它隐藏了世界上哪些地区正在推动其移动增长，截至 2013 年 9 月，在所有设备的每月 11.9 亿用户中，每月移动用户总数为 8.74 亿，每日移动用户总数为 5.07 亿。

因此，我连续几个季度呼吁脸书发布更详细的国际移动统计数据。该公司在 8 月份部分默许了，向我提供了美国和英国的具体数据，如下图所示。

![Facebook International User Counts By Country](img/b72dd392c4920defed4c4cdc41545c8d.png)

尽管它发誓要提高国际移动增长的透明度，并公开发布更多的国别统计数据，但它迄今没有这样做。但隐藏在向欧洲和中东营销伙伴提供的程式化“年度”报告中的是德国、法国、西班牙、意大利、瑞典、土耳其和以色列的数据。

该年度报告中有一些脸书高管的有趣言论，包括产品副总裁 Chris Cox，他说“我们永远不会成为一家媒体公司。媒体变得擅长创造内容的历史并不多。”还有大量以脸书为主题的插图(见本帖)案例研究，对马克·扎克伯格和凯文·斯特罗姆的采访，以及受脸书影响的购物和旅游支出统计。

然而，最吸引人的是之前未公布的国际增长数据，以下是这些数据:

*   德国——1900 万日活跃用户，2500 万月活跃用户(43%的德国互联网用户使用脸书，76%的月活跃用户每天返回)，1300 万移动 DAU，1800 万移动 MAU (27%的德国手机用户)
*   法国——1800 万 DAU，2600 万 MAU (63%的法国互联网用户使用脸书，69%的月度用户每天返回)，1100 万移动 DAU，1700 万移动 MAU (33%的法国手机用户)
*   西班牙——1200 万 DAU，1800 万 MAU (58%的西班牙互联网用户使用脸书，67%的月度用户每天返回)，810 万移动 DAU，1300 万移动 MAU (32%的西班牙手机用户)
*   意大利——1700 万 DAU，2300 万 MAU (71%的意大利互联网用户使用脸书，74%的月活用户每天返回)，1000 万移动 DAU，1600 万移动 MAU (32%的意大利手机用户)
*   瑞典——380 万 DAU，490 万 MAU (57%的瑞典互联网用户使用脸书，78%的月度用户每天返回)，300 万移动 DAU，400 万移动 MAU (53%的瑞典手机用户)
*   土耳其–1900 万 DAU，3300 万 MAU，950 万移动 DAU，2000 万移动 MAU
*   以色列——270 万 DAU，380 万 MAU (71%的月用户每天回访)，200 万移动 DAU，290 万移动 MAU

数据显示，脸书正在成功地向欧洲和中东部分地区的移动设备转移，所有列出的国家都有超过 50%的用户通过移动设备进行访问。脸书的小屏幕界面在瑞典尤其受欢迎，超过 81%的用户使用手机。

德国的统计数据显示，当地的社交网络 StudiVZ 和民众对隐私的强烈关注可能减缓了脸书在德国的发展。只有 43%的德国互联网用户和 27%的德国手机用户使用脸书——这一比例低于其他国家。不过，脸书已经在德国建立了 2500 万用户群。

![Facebook Mobile World](img/3396d6d74aa47dee4c48388973d62aa2.png)

虽然脸书依赖 HTML5 犯了一个大错误，并且迟迟没有开始在移动设备上显示广告，但它现在证明了它可以在桌面之外赚很多钱。事实证明，开发者愿意为移动应用安装广告付费，这有助于他们获得下载量，并在拥挤的应用商店中被发现。这就是为什么全球移动用户的增长会让华尔街兴奋不已。

它还可以将脸书与美国国内青少年的反复无常隔离开来，在美国，人们担心社交网络可能会失去对年轻人的“冷静”。调查显示，青少年讨厌父母出现在脸书。然而，调查结果可能会因[社会合意性偏见](https://web.archive.org/web/20221003021410/https://beta.techcrunch.com/2012/06/06/facebooking/)而对该公司不利，这可能会鼓励孩子们说他们使用十年前的脸书少于实际使用，以听起来很酷。

目前还不清楚脸书是否真的需要青少年的喜爱，或者它是否可以作为一个不时髦但无处不在的公用事业。随着孩子们从一个社交应用涌向另一个，成年人可能会跟随他们，或者他们可能会简单地认为青少年在追逐下一个时尚。不管怎样，脸书越能在全球确立自己移动优先服务的地位，它就越有机会摆脱混乱，转而成为互联网持久的、制度化的一部分。
