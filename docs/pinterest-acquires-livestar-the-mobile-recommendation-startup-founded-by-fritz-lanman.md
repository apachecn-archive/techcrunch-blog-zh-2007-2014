# Pinterest 收购了由 Fritz Lanman 创立的移动推荐初创公司 Livestar

Pinterest 今天宣布收购了 [Livestar](https://web.archive.org/web/20221124021530/http://www.livestar.com/) ，这是一款由微软校友 [Fritz Lanman](https://web.archive.org/web/20221124021530/http://www.crunchbase.com/person/fritz-lanman) 创立的移动推荐应用，Fritz Lanman 也是一位著名的天使投资人，他已经投资了像 [Square](https://web.archive.org/web/20221124021530/http://www.square.com/) 、 [Getaround](https://web.archive.org/web/20221124021530/http://www.getaround.com/) 这样的公司，而且碰巧的是，还投资了 Pinterest。

交易条款尚未披露，但这似乎是一个工程采购的情况。我们被告知，作为交易的一部分，只有 Livestar 的三人工程团队将加入该公司，该服务将立即关闭。兰曼本人不会全职加入 Pinterest，但他将继续担任公司收购后的顾问。我们被告知，从这里开始，他将转向一个尚未宣布的新项目。

Livestar 成立于 2011 年，于去年秋天在 TechCrunch Disrupt(T2)发布。这项服务旨在提供来自你的朋友和网上权威来源的个性化推荐，并且[旨在瓦解](https://web.archive.org/web/20221124021530/https://beta.techcrunch.com/2012/06/15/fritz-lanmans-livestar-to-launch-soon-to-fill-the-gap-between-foursquare-and-yelp/)four square、Yelp 和烂番茄等网站。据报道，该公司已经从投资者那里筹集了 200 万美元，投资者包括哈迪和阿里·帕托维、彼得·切宁、保罗·布赫海特、WME 和雷·奥茨。

以下是 Livestar 自己发布的公告:

> “我们在 Livestar 的目标是帮助人们从朋友、评论家和有相似品味的陌生人那里找到好的推荐。在过去的两年里，Livestar 社区分享了数百万次评级和评论。今天，我们想让你知道我们将采取下一步行动:Livestar 工程团队将加入 Pinterest。
> 
> 如果你不熟悉 Pinterest，它是一个在线工具，可以帮助人们发现他们喜欢的东西，并激励他们在日常生活中做这些事情。我们认为这与我们在 Livestar 的工作是一致的。虽然我们很高兴加入 Pinterest，但我们也很难过地宣布 Livestar 将立即关闭。
> 
> 我们非常感谢你们所有人。我们希望你发现了很棒的推荐，并喜欢与你的朋友和其他人分享你自己的经历。我们已经将您的所有评论放在附件的电子表格中，以便您可以随身携带。
> 
> 再次感谢您对 Livestar 的支持和贡献。我们会非常想念 Livestar，尤其会想念你所有的精彩评论。我们希望不久能在 Pinterest 上看到你！
> 
> 最佳，
> Livestar 团队”
