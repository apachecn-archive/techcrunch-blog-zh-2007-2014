# Fanhattan 增加了新内容合作伙伴的选择:Crackle、Lifetime 和 PBS

我一直很喜欢使用 Fanhattan 应用程序。它很智能，在[的 iPhone 和 iPad](https://web.archive.org/web/20221206072536/https://beta.techcrunch.com/2011/11/16/fanhattan-tv-discovery-app-migrates-to-the-iphone/) 上都有，让我更容易找到我想看的东西。句号。但是我有一个小小的不满。事实上，这甚至不是抱怨。这就是我们对任何好东西的感觉:我们想要更多。

Fanhattan 有一个很棒的智能浏览系统，可以让你根据流派、评级、发布日期、服务提供商过滤内容，然后选择你希望内容如何排序。唯一的问题是，你使用的过滤器越多，可供选择的选项就越少——有时，根本没有选项。

但随着三家新媒体合作伙伴加入范哈坦阵营，这种情况即将改变:Crackle、Lifetime 和 PBS。

《电光火石》从今天到去年都有免费的电影和电视节目，包括*抢*，*几个好男人*，*星河战队*，*全家老小*，*我梦见珍妮*，还有我个人最喜欢的一部*宋飞*。

然后是 Lifetime，这是一个比大多数网站对女性更友好的网站。Lifetime 将为 Fanhattan 用户提供访问项目附件*凝视莫莉·西姆斯，以及充满搞笑 NPH 的*我是如何遇见你母亲的*。*

我们都不应该对 PBS 感到陌生，它似乎已经存在了很长时间。通过 Fanhattan 应用程序，你可以进入像*新星*(是的，全部 39 季)和*古董路演*这样的节目。

总的来说，这些新的合作伙伴应该真正加强 Fanhattan 应用程序上的可访问内容。如果你还没有，而且这些节目中的任何一个对你来说听起来都不错(还有来自网飞、Hulu+、Vudu、iTunes 和 ABC Player 的内容)，那就在[苹果应用商店](https://web.archive.org/web/20221206072536/http://itunes.apple.com/us/app/fanhattan-for-ipad/id436928538?mt=8)免费下载 Fanhattan 吧。
