# 尼康展示了 Android 驱动的 Coolpix S800c，这是其第一款面临身份危机的相机

尼康昨晚正式发布了该公司首款安卓数码相机 Coolpix s800c，打消了很多猜测。我知道，很刺激。

姜饼驱动的傻瓜相机配备了 1600 万像素的传感器，10 倍光学变焦，可以拍摄 1080p 视频。当然，所有这些都不如相机后端的 3.5 英寸液晶触摸屏，它允许用户启动应用程序和浏览网页(多亏了一个漂亮的嵌入式 Wi-Fi 无线电)。尼康计划下个月在美国推出这个胖乎乎的小家伙，价格为 349 美元，但我想知道有多少人真的想要这样的设备。

不要误解我的意思，我完全支持模糊产品类别之间的界限，但是向某些方向发展似乎比向其他方向发展更有意义。苹果(Apple)和 HTC 等智能手机制造商一直在增加设备中的摄像头，因为他们知道用户无论如何都会随身携带这些东西。给这些用户(当然还有潜在客户)一个可靠的方法来捕捉展现在他们周围的瞬间是一个主要的卖点，这不会很快改变。

尼康在这里做的似乎是反其道而行之——这就像该公司的产品开发团队看着他们的相机，思考如何让它们更像那些每个人都一直嚷嚷着要的新奇手机。最终结果似乎在技术层面上足够好(Imaging Resources 对他们的 S800c 做了很好的报道)，尽管我有很多疑问，但只有时间才能证明 Android 的噱头是否会像尼康希望的那样发挥作用。
