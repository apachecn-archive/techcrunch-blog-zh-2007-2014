# 媒体行业最大的恐惧是:我们怎么知道明年苹果不会拿走 50%的份额？

![](img/25fa8348bd858442e3551e7d7bb2fa47.png)

冒着在上[和在](https://web.archive.org/web/20221206204603/https://beta.techcrunch.com/2011/02/15/apple-in-app-subscriptions/)上[谈论苹果有争议的新](https://web.archive.org/web/20221206204603/https://beta.techcrunch.com/2011/02/15/apple-ecosystem/)[订阅计费规则](https://web.archive.org/web/20221206204603/https://beta.techcrunch.com/2011/02/16/why-are-you-people-defending-apple/)的风险，让我再补充一点。是的，这些新规则可能会影响[音乐流媒体应用](https://web.archive.org/web/20221206204603/https://beta.techcrunch.com/2011/02/15/apple-keelhauls-music-streaming-services/)和电影流媒体应用，但首先也是最重要的是，它们针对的是[印刷出版商](https://web.archive.org/web/20221206204603/https://beta.techcrunch.com/2011/02/15/apples-digital-newsstand-just-disrupted-the-publishing-industry/)，他们希望将其现有的订阅业务模式移植到 iPad 上。

拥有大量订阅业务的大型出版商和其他媒体公司不喜欢将 30%的收入永久交给苹果。他们更不喜欢的是失去与客户的直接关系，以及随之而来的所有数据。(毕竟谁拥有了计费关系，谁就拥有了客户)。但是他们最害怕的是苹果的定价权。我从几个出版业内人士那里听到的重复说法是:“我们怎么知道明年苹果不会拿走 50%？”

换句话说，他们害怕他们和他们的读者会迷上 iPad，在短短的三个季度里，iPad 已经推动苹果成为世界上最大的移动电脑公司。现在他们仍然立场坚定，并威胁要坚持使用谷歌及其新的、更优惠的、One Pass 订阅付费或亚马逊的 Kindle。但是，如果他们不战而降，Appel 为他们的数字出版物带来了数百万新订户，那么苹果就可以在未来获得更大的收入份额。

苹果可以通过承诺在一定时间内不改变经济状况来减轻这些担忧。例如，将 30%的费用锁定三年，可以减少这个过程中的一些不确定性。
