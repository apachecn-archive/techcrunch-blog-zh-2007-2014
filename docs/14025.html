微软将全力投入15亿美元的Windows 8营销闪电战

Windows 8将于10月26日发布。据说这是微软迄今为止最重要的产品发布会。据报道，雷德蒙准备通过大规模营销活动来适当支持该产品。福布斯报道该活动耗资15-18亿美元，是行业历史上最大的产品发布会。

奇怪的是，到目前为止，微软对Windows 8一直保持相对平静。在其发布前夕，该公司似乎转而专注于必应和Internet Explorer 9的营销。但这种情况在未来几天应该会有所改变。Windows 8将于两周后发布。由于它的不同之处，与以前的Windows版本不同，微软不得不向普通消费者、发烧友和企业IT经理销售该操作系统。

Windows 8是不同的，而不是Windows Phone 8所展示的好的方面。微软即将推出的移动操作系统与iOS和Android不同。从通知到应用程序到工作流程，一切都不同，但也令人耳目一新。在过去的五年里，智能手机市场的规模几乎每年都在翻番。大多数消费者在一个特定的移动平台上的时间还不够长，无法让工作流完全扎根于他们的DNA中。在这一点上，对其他事情的改变可能是受欢迎的，甚至是值得庆祝的。但Windows 8的情况并非如此。

17年来，Windows一直采用同样的流程。作为世界上占主导地位的桌面操作系统，世界上有很大一部分人只用过Windows。我非常喜欢在移动设备上使用它，但不喜欢在台式电脑上使用。微软明智地知道，它将不得不花大价钱来说服用户，不同是好的，Windows 8值得麻烦去学习一个新的工作路径。

让Windows 8比以前的版本便宜很多也会有所帮助。微软正在对Windows 8的升级版本进行定价。从Windows XP、Vista或7升级只需40。

这场巨大的营销活动的绝大部分无疑是针对新的电脑购买者的。Windows 8营销闪电战可能旨在让人们进入百思买购买新电脑。一代人以来，微软第一次可以诚实地宣称Windows提供了一些不同的东西。用全新的触控一体机取代老化、恶意软件泛滥的奔腾4台式机。这才是微软会赢的地方。

微软Windows 8营销活动的细节仍然很少，但预计在假期期间将到处看到新的Windows旗帜和Metro界面。微软希望Windows 8成为2012年的Furby。Windows Phone 8在桌面操作系统之后不久推出。微软的Surface应该也会在不久的将来推出。微软希望在这个假日季节吸引所有的注意力，据报道，在营销上花了大笔钱，它肯定会把它的钱袋放在这个目标后面。
