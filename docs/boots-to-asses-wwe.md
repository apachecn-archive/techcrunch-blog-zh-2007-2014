# 从靴子到屁股——WWE 社交媒体战略将其他人远远甩在身后 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/25/boots-to-asses-wwe/>

https://twitter.com/#!/TheRock/status/136287368255782912

**编者按:** *特约撰稿人 [Joseph Puopolo](https://web.archive.org/web/20221006121222/http://twitter.com/#%21/jpuopolo) 是一名企业家和初创企业爱好者，他在博客上发表各种话题，包括绿色倡议、技术和营销。*

在过去的一年中，WWE 通过将 Twitter 和脸书进一步整合到其常规广播中，继续推进社交媒体的发展。“摇滚”德怀恩·约翰逊现在是 WWE 和好莱坞的交叉明星，他在 WWE Raw 上剪了一个宣传片，为即将到来的按次付费“幸存者系列”进行宣传。在宣传片中，摇滚乐队迅速创造了一个新的流行语“Boots to Asses ”,并表示它现在是全球的趋势。在他提到这件事之前，推特上没有提到这件事。在他说了这句话后不久，不仅整个竞技场开始了一段新的颂歌，而且证明了约翰逊先生的预言，因为几分钟后“ [Boots to Asses](https://web.archive.org/web/20221006121222/https://twitter.com/#!/search/boots%20to%20asses) ”一词就风靡全球。有人可能会说这是一次性的，但对于任何观察 WWE 的人来说，这只能被视为一个更大的社交媒体战略的一小部分。

在过去的一年里，WWE 通过非常精明的社交媒体攻势巩固了其已经很强的网络影响力。现在，每一个出现在 WWE 电视上的表演者都有自己的 Twitter 账号，他们用它来建立粉丝群，并实际上延续从节目开始的故事情节。在广播期间，每当摔跤运动员走向拳击台，他们的 Twitter 手柄就会在屏幕图形上他们的名字旁边突出显示。他们的策略显而易见且有效，提供了一种方法，让他们的节目更具互动性，并利用临时粉丝更有规律地收听，尤其是当屏幕上出现重大事件时。在整个节目中，经常听到主持人谈论某件事是否是全球趋势。

一些试图提高自己在公司地位的摔跤手实际上已经利用社交媒体来建立受众。扎克·莱德和他自称的“泽西海岸”角色创造了一系列 YouTube 视频来激发兴趣。值得称赞的是，他不仅成功地推动近 10 万人成为 Youtube 的订户，而且在 Twitter 上也有 30 万名粉丝。他基本上走出去，为自己建立了一个新的粉丝群，并因此获得了更多的播出时间和兴趣。

摔跤狂热是 WWE 举办的最大的活动。为了炒作主要事件，他们已经推出了一个单独的网站来突出他们的主要比赛，约翰·塞纳对“岩石”德万耶·约翰逊，并鼓励粉丝选择他们站在哪一边。该网站集成了 Twitter 和脸书网页。在每一方，他们都吸引了大量的观众。这是到目前为止磁带的故事，它是相当令人印象深刻的。

|  | “岩石”德韦恩·约翰逊 | 约翰·塞纳 |
| 推特粉丝 | 160 万粉丝 | 110 万粉丝 |
| 脸书喜欢 | 440 万个赞 | 920 万个赞 |

这就引出了一个问题，为什么其他体育或娱乐广播没有像这样积极地将社交媒体整合到他们的广播或活动营销中。WWE 使用社交媒体来增加争议和激发更多的兴趣，而其他人则回避它。很明显，WWE 是一种体育娱乐，更多的是一种照本宣科的产品，但为什么其他娱乐频道不能采用一些社交媒体策略来提高人们对其产品的兴趣和现场参与度。为什么周一晚上的足球赛在一场大型比赛即将到来时不做同样的事情？他们可以用这种策略来炒作，激发兴趣，吸引新的观众或追随者。我会提供两个理由，要么他们害怕社交媒体的自发性，要么他们觉得没有必要采用它。

[Aaron Rodgers](https://web.archive.org/web/20221006121222/https://twitter.com/#!/AaronRodgers12) ，他正在为[绿湾](https://web.archive.org/web/20221006121222/http://www.twitter.com/packers)包装工队度过他人生中的一个赛季，只有 385，000 名推特粉丝，超级碗卫冕冠军只有 160，000 名粉丝。当然，如果 NFL 中负责营销的人没有能够更好地实时联系和营销这位崭露头角的超级巨星和他的球队与球迷，这里就会有一点脱节。这似乎是一个错过的机会，虽然 Aaron Rodgers 是 WWE 的超级粉丝，喜欢炫耀他的[“世界冠军”腰带](https://web.archive.org/web/20221006121222/http://sports.yahoo.com/nfl/blog/shutdown_corner/post/Aaron-Rodgers-gets-a-real-actual-championship-b?urn=nfl-317685)，但他没有世界冠军的社交媒体形象。

另一方面，NHL 禁止玩家在比赛日使用脸书和推特。包括四大职业体育联盟(NFL，NBA，MLB，NHL)在内的几个传统娱乐频道都制定了严厉的处罚措施。2010 年，查德·奥乔辛科因在 NFL 比赛期间发推特而被[罚款 25000 美元](https://web.archive.org/web/20221006121222/http://sports.espn.go.com/nfl/trainingcamp10/news/story?id=5493157)。四大不仅落后，而且他们还不明白如何真正有效地利用社交媒体为他们的产品或受众服务，当你想到这一点时，这有点遗憾。

另一方面，UFC 和 WWE 一样，是另一个真正有个性的运动利用 Twitter 来激发兴趣的例子。他们采用了一些精明的社交媒体策略，包括在接近八角形时展示战斗机的 Twitter 手柄。UFC 赛事通常会推动整个赛事的热门话题。UFC 主席[白大拿](https://web.archive.org/web/20221006121222/http://www.twitter.com/danawhite)站在前面，像其他粉丝一样发推特直播他对比赛的反应。而且是双向对话。他实际上直接从粉丝那里获取反馈，并学习如何让他的产品变得更好。通过让粉丝与拳手密切互动，UFC 已经能够人性化，增加兴趣，并显著提高其付费节目的购买率。

任何不能与粉丝互动的娱乐品牌都错失了一个巨大的机会。尤其是在体育领域，品牌或联盟真的需要提供一个合适的渠道来炒作，让他们的粉丝对即将到来的事件或比赛感到兴奋。虽然 WWE 是一个非传统的例子，但它很容易领导这种策略，以激发人们对其产品的兴趣，并与他们的粉丝群互动。如果你看看 WWE 在社交媒体上做了什么，与他们在其他运动中的同行相比，他们真的是太棒了。
