# PicPlz 击败 Instagram 成为 API

![](img/1ccc4d8c26132d4ea094c716423aaaf8.png)

在新一代社交照片应用的竞争中，赢得消费者的心只是等式的一部分。赢得开发者的心是另一部分。虽然 Instagram 在用户方面远远领先于[早期的领导者](https://web.archive.org/web/20221007023738/https://beta.techcrunch.com/2010/12/21/instagram-one-million/)，但竞争对手 [PicPlz](https://web.archive.org/web/20221007023738/http://picplz.com/) 今天能够凭借 API[第一个出现。PicPlz 的 API 将允许开发者从其服务中提取照片，并从其不断增长的共享照片库中创建幻灯片和其他小工具。它还允许开发者上传图片，并利用 PicPlz 的照片过滤器。这意味着任何照片应用程序现在都可以添加 PicPlz 的滤镜作为一项功能。](https://web.archive.org/web/20221007023738/http://blog.picplz.com/post/3167778519/picplz-api-launch-creative-commons-support-more)

Instagram 也在开发一个 API，预计将于今天发布[。它已经不得不](https://web.archive.org/web/20221007023738/https://beta.techcrunch.com/2011/02/02/instagram-funding/)[屏蔽了一个名为 Followgram](https://web.archive.org/web/20221007023738/https://beta.techcrunch.com/2011/01/12/bam-instagram-blocks-followgram/) 的服务，该服务迫不及待地为有趣的照片擦洗服务。首席执行官凯文·斯特罗姆告诉我，测试版注册将于今天晚些时候开放(查看这个*创始人故事* [视频](https://web.archive.org/web/20221007023738/https://beta.techcrunch.com/2011/02/03/founder-stories-instagram-products/)，他在视频中谈到了他如何看待这场竞争)。他们仍然需要等待实际的 API 可用。同时，开发者可以开始玩 PicPlz 的 API 了。

为开发者提供工具，在 PicPlz 的基础上构建引人入胜的新应用，是从消费者应用到平台的第一步。但是这两件事是密切相关的。开发者希望在流行的平台上工作。PicPlz 还宣布了一个吸引高级用户的新功能:将知识共享许可应用于照片的能力。接受知识共享是 Flickr 早期成功的一个因素，它肯定会鼓励在这些新的移动社交应用中更多的分享。

最后，PicPlz 为品牌、名人和音乐人添加了一个分析仪表板，以跟踪哪些照片获得了最多的浏览、喜欢或评论。如果它被证明足够有用，该公司可能最终会对仪表盘的品牌收费，或者向所有用户开放。
