# 苹果新专利描述 Kinect 风格的控制系统

我意识到专利演讲之所以超级冗长是有原因的，因为那些申请人希望他们的技术/发明的每一个小角落都得到保护。但是……好吧，看看这个:“提供了一种三维成像和显示系统，其中通过测量作为其相移的函数的调幅扫描光束的路径长度，在成像体积中光学检测用户输入。呈现关于检测到的用户输入的视觉图像用户反馈

这是名为“三维成像和显示系统”的苹果[新专利申请](https://web.archive.org/web/20221208220833/http://appft.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PG01&p=1&u=%2Fnetahtml%2FPTO%2Fsrchnum.html&r=1&f=G&l=50&s1=%2220110298798%22.PGNR.&OS=DN/20110298798&RS=DN/20110298798)的摘要。如果你能从这些杂乱的单词中发现这项专利申请的重要性，你就能得到五颗金星。没有吗？好吧，我告诉你。

它基本上是苹果版的 Kinect，提供 3D 无触摸控制，可以在你的 Mac、iPad 或 iPhone 上操作用户界面。虽然比微软的 Kinect 限制更多，但苹果专利描述了一种包含高速红外激光器和高速光电探测器的设备。激光创造了一个“空间体积”,从中可以检测到任何运动。

这将允许用户使用基于移动的手势来控制他们的 iThing 界面。

当然，这很可能是苹果新 3D 控制系统相关的众多专利申请之一。早在 8 月份就提交了申请，很可能苹果实际上正在为此做一些核心工作，我们可能会在未来几个月看到更多类似的专利进入美国专利局的数据库。
