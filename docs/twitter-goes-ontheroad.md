# Twitter @ on the road TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/10/13/twitter-goes-ontheroad/>

![](img/2e28fbc63a25fb99f819518f69ba4eaf.png)感谢读者[亚当·杰克森](https://web.archive.org/web/20221005163856/http://twitter.com/#!/adamjackson)，我们刚刚在推特上发现了一个新的官方账号，名为 [@OnTheRoad。](https://web.archive.org/web/20221005163856/http://twitter.com/OnTheRoad)该账户的第一条推文是昨天发布的，所以看起来是最近才开通的。

从已验证账户发出的推文似乎涉及传递关于加拿大蒙特利尔 [Tweetup](https://web.archive.org/web/20221005163856/http://twtvite.com/c5hk8f) 的信息和细节。该账户的个人资料写道:*在北美进行旋风式旅行。询问人们如何发现正在发生的事情。*

该账户可以成为 Twitter 在美国和加拿大宣传推文的营销工具。蒙特利尔活动的 TweetUp 页面称，Twitter 的设计研究员 Mark Trammell“将在接下来的几周内旅行，与人们谈论他们如何使用 Twitter”，因此该帐户可能是 Trammell 宣传旅行的一种方式。或者，这个账户可以有更广泛的用途，作为查看 Twitter 员工将在哪里发言或发表演讲的一种方式。

另一个线索来自用户研究公司 [BoltPeters。从该公司的网站来看，Twitter 已经雇佣该公司进行用户研究。OnTheRoad 可能是记录这项研究的一种方式。](https://web.archive.org/web/20221005163856/http://boltpeters.com/)

就目前而言，Twitter 似乎对这项倡议难以捉摸。当我们询问@OnTheRoad 时，Twitter 发言人 Carolyn Penner 告诉我们,“继续关注并找出答案！”
