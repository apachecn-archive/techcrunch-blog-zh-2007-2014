# 俄罗斯 Nginx 从国际投资者那里筹集了 300 万美元

俄罗斯公司 [Nginx](https://web.archive.org/web/20230209011150/http://nginx.net/ "Nginx") 已经从德国 [BV Capital](https://web.archive.org/web/20230209011150/http://www.bvcapital.com/ "BV Capital") ，俄罗斯 [Runa Capital](https://web.archive.org/web/20230209011150/http://www.runacap.com/ "Runa Capital") 和 [MSD Capital](https://web.archive.org/web/20230209011150/http://www.msdcapital.com/ "MSD Capital") ， [Michael Dell](https://web.archive.org/web/20230209011150/http://www.crunchbase.com/person/michael-dell "Michael Dell") 的风险基金中筹集了 300 万美元，据 [RBK 日报](https://web.archive.org/web/20230209011150/http://www.rbcdaily.ru/2011/10/11/media/562949981699526 "RBK Daily")报道。

Nginx 由俄罗斯工程师伊戈尔·塞索耶夫于 10 年前创立，为大规模、快节奏、内容丰富的网站开发和维护基于 Unix 的开源 web 和邮件代理服务器。 [TechCrunch](https://web.archive.org/web/20230209011150/https://techcrunch.com/ "TechCrunch") 、[脸书](https://web.archive.org/web/20230209011150/http://facebook.com/ "Facebook")、 [Groupon](https://web.archive.org/web/20230209011150/http://www.groupon.com/ "Groupon") 和[WordPress.org](https://web.archive.org/web/20230209011150/http://wordpress.org/ "Wordpress.org")都是其客户，实现了高密度、可扩展性和可预测的性能。据 Nginx 称，其产品支持超过 4000 万个域名。
