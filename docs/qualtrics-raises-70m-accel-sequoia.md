# Qualtrics 从 Accel 和 Sequoia 融资 7000 万美元:“你没听说过的最大的软件公司”？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/05/15/qualtrics-raises-70m-accel-sequoia/>

两家顶级风险投资公司为一家成立 10 年的提供在线数据收集和分析的公司开出了一张大额支票。

这一轮 7000 万美元的投资是 Qualtrics 的第一笔机构投资——该公司表示，自 2002 年成立以来一直盈利。这笔钱来自 Accel Partners 和红杉资本(Sequoia Capital)，这两家公司都有针对 Qualtrics 等投资的成长型基金，这些成熟公司已经取得了一些成功，但迄今为止仍是靠自举发展。显然这是两家公司有史以来最大的一笔投资。(过去的联合投资包括 AdMob，[谷歌以 7.5 亿美元收购的移动广告网络](https://web.archive.org/web/20230405190129/https://techcrunch.com/2009/11/09/google-acquires-admob/)。)

联合创始人兼首席执行官瑞安·史密斯表示，Qualtrics 提供了一种新的市场研究方法。在过去，想要进行这项研究的公司必须雇用外部公司，这是一个昂贵的提议。有一些初创公司正在提供更便宜的替代方案，但它们通常是简单的工具，如调查，不会取代严肃的研究。另一方面，Qualtrics 已经开发出技术，允许公司自己进行复杂的研究，而不需要雇用外部公司，甚至不需要让他们的 IT 团队处理复杂的安装。

“我们的口号是:‘对博士来说足够复杂，对实习生来说足够简单，’”史密斯说。

Qualtrics 声称拥有超过 4000 家企业客户，包括 Barnes & Noble、CVS/Caremark、GEICO、微软、Neiman Marcus、皇家加勒比、西南航空、汤森路透、丰田、沃达丰、Zappos 和 600 所大学。史密斯表示，这种增长很大一部分出现在 2008 年和 2009 年金融危机之后，当时公司意识到他们需要变得更加数据驱动:“我们不能只是向墙上扔泥巴，看看有什么粘在一起。”

该公司最近扩大了其产品阵容，从核心研究套件扩展到包括 Qualtrics 360(一种员工评估工具)和 Qualtrics Site Intercept(向目标网站访问者提供定制内容)。史密斯表示，他决定接受新的资金，大力投资于增长——明年，他计划在 200 人的员工队伍中增加 250 名新员工。他计划在犹他州普罗沃的公司总部进行这项工作。

Accel 的 Ryan Sweeney 和 Sequoia 的 Bryan Schreier 都加入了董事会。Sweeney 说，他是在看到其他几家“我们知道并钦佩的”公司使用 Qualtrics 收集数据后听说它的。他很快意识到 Qualtrics 正在“为所有这些不同的东西提供动力，坦率地说,(我们)恳求他们与我们一起开会。”

Schreier(他在新闻稿中称 Qualtrics 是“你还没听说过的最大的软件公司”)告诉我，他从 2003 年就认识 Qualtrics 的联合创始人之一，当时他们在谷歌一起工作，但这家公司听起来像是一家“不起眼的街角商店”然后，几个月前，他看到了 Qualtrics 的数字，这些数字“令人震惊”他还表示，Qualtrics 对客户服务的承诺给他留下了深刻的印象——事实上，史密斯告诉我，每当客户服务电话铃响三次而无人接听时，办公室里就会响起一声不折不扣的警笛。
