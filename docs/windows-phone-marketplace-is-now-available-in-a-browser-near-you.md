# Windows Phone Marketplace 现在可以在您附近的浏览器中使用

早在五月份，微软就透露他们正在努力开发 Windows Phone marketplace 的网络版。既然新的芒果更新即将推出，微软[已经正式打开了他们全球市场网络市场的开关](https://web.archive.org/web/20230205024111/http://www.winrumors.com/windows-phone-web-marketplace-goes-live/)。

在真正的 Windows Phone 风格中，marketplace 的网页版紧密贴合 Metro UI 光滑、极简的设计根基。早在 5 月透露，微软暗示为开发者和他们的应用程序提供“额外的可见性和更多的销售可能性”，他们已经通过优秀应用程序的旋转幻灯片展示兑现了他们的承诺。

除了化妆品之外，所有预期的功能都在这里得到了体现:潜在客户可以按照类别或风格对他们可以支配的无数应用程序进行排序，每个应用程序的条目都为有眼光的客户提供了截图和用户评论。很大一部分 Windows Phone 应用程序允许你在实际购买前试用，在手机上玩演示版只需在浏览器中点击一下。

购买应用程序同样是一键操作，因为它会自动从用户 Windows Live ID 绑定的信用卡中收费。一旦某个应用程序被付费，它就会通过无线方式下载到用户的 Windows Phone 手机上。很简单，对吧？

现在可以通过[微软的 Windows Phone 门户](https://web.archive.org/web/20230205024111/http://www.windowsphone.com/en-US/marketplace)进入该市场，但用户可能要小心了:有时花几个小时浏览应用程序列表的诱惑太强大了，无法克服。
