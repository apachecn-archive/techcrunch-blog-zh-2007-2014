# TC50: LearnVest 是一份女性个人理财指南

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/09/15/tc50-learnvest-is-a-personal-financial-guide-for-women/>

![](img/62a4d2f19d4ec01a20bc1fd236585486.png)

现在比以往任何时候，个人理财教育和帮助对任何人的财务健康都至关重要。[Mint.com，](https://web.archive.org/web/20230406110930/http://www.mint.com/)[前 TechCrunch 40](https://web.archive.org/web/20230406110930/https://techcrunch.com/2007/09/18/techcrunch40-session-5-productivity-web-apps/) 公司[在大会上赢得](https://web.archive.org/web/20230406110930/https://techcrunch.com/2007/09/18/mint-wins-techcrunch40-50000-award/)最高奖项的[自推出以来已经](https://web.archive.org/web/20230406110930/https://techcrunch.com/2009/09/02/mint-is-worth-a-mint-140-million-valuation/)令人难以置信地成长，最近[被 Intuit 以 1 . 7 亿美元收购](https://web.archive.org/web/20230406110930/https://techcrunch.com/2009/09/13/intuit-to-acquire-former-techcrunch50-winner-mint-for-170-million/)。创业公司 LearnVest 在网上理财方面有着不同的目的；这家初创公司专注于帮助用户，尤其是女性，管理他们的财务，并学习如何成为财务专家。这有点像网络版的理财规划师苏茜·欧曼。

当你第一次登录 LearnVest 时，该网站会问你一系列关于你的财务健康状况(即你有多少信用卡债务)、你的人生阶段(即你是否租房、你是否很快计划成家、你是否拥有房子)和你的财务教育水平(即你最近是否检查过你的信用评分)的问题。LearnVest 将使用所有这些信息来诊断您的财务健康状况和教育水平，然后给你一个你需要学习和改进的快照。

LearnVest 会给你一个定制的计划，列出你需要做的事情，并通过清单阅读。有点像备忘单，清单将帮助你解决一系列问题，如摆脱信用卡债务、婚礼预算、买房等等。当您完成列表上的任务时，您可以勾选每个方框，并为您的改进赢得分数。积分会让你在 LearnVest 的社区中获得地位，最终你可以用积分换取奖励。

随着您不断登录 LearnVest，该网站将完全根据您的财务状况进行定制，为您提供与您的具体财务目标相关的新闻文章。LearnVest 还旨在成为一个社交社区，用户可以在论坛上向其他用户和专家提问，尽管我不确定有多少女性愿意与他人分享她们的财务困境。
