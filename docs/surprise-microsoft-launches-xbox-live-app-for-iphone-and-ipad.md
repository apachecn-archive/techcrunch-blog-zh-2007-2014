# 惊喜！微软推出适用于 iPhone 和 iPad 的 Xbox Live 应用

哇哦。今天早上我没有想到我会说这句话:微软本周将会大开杀戒。

在过去的 48 小时里，他们为 Xbox 360 推出了一个大规模的[大修界面，发布了](https://web.archive.org/web/20221207103054/https://beta.techcrunch.com/2011/12/04/the-xbox-metro-update-nudges-microsofts-console-closer-to-set-top-hedgemony/)[一个大大改进的网飞 360 应用](https://web.archive.org/web/20221207103054/https://beta.techcrunch.com/2011/12/07/netflix-xbox-360-kinect/)，以及[推出了一个让你从 Windows Phone 7 手机控制你的 360 的应用](https://web.archive.org/web/20221207103054/https://beta.techcrunch.com/2011/12/07/xbox-360-controlling-xbox-companion-app-now-available-for-windows-phone-7/)。他们的下一步行动是:他们刚刚推出了一款适用于 iOS 的 Xbox Live 应用程序(恰如其分地被称为“我的 Xbox Live”)。

现在，应用程序*似乎没有*复制他们今天早上发布的 WP7 应用程序的 360 控制功能——可以理解，这似乎是 Windows Phone 的独家功能。然而，它确实复制了 Windows Phone 发布时的许多以 Live 为中心的功能。

您可以:

*   更改您的 3D 头像的外观
*   向您的 Xbox Live 好友发送消息
*   查看您的成就
*   阅读 Xbox 新闻

虽然我敢肯定，许多痴迷于 Xbox 的 Windows Phone 7 手机用户会对失去这些功能的手机独占权感到有点恼火，但转向 iOS 几乎是不可避免的。整件事情中我最喜欢的一点是:在 iPhone 应用程序的底部栏上对 Windows Phone 7 的用户界面不那么微妙的点头。
