# Mint.com 社交化个人理财建议新增问答功能

随着[Answers](https://web.archive.org/web/20221206190732/http://answers.mint.com/)和 a new Q &的[发布，Mint.com 正在使其个人理财建议门户网站变得更加社会化，这是一个汇集专家建议和社区见解的网站。](https://web.archive.org/web/20221206190732/http://www.marketwatch.com/story/money-questions-mintcom-has-answers-2010-05-13?reflink=MW_news_stmp)

该网站博客 MintLife 的一个子集回答了来自专业顾问和其他 Mint.com 用户的社区成员问题的众包答案。问题按照主题、预算、债务管理、未来目标、
和储蓄进行分组。该论坛对任何人开放，答案由 MintLife 编辑人员筛选，并由其他用户根据回复的有用性进行排名。示例问题包括“我应该买还是租一辆新车？”到“已婚夫妇应该如何分配他们的钱和费用？”到“我应该花多少钱买一枚订婚戒指？”。

根据用户的排名，最有帮助的答案会出现在页面的顶部。最初，MintAnswers 将提供剑桥顾问联盟的专家建议。Answers 平台由海绵(一种白标问答技术)提供支持。

Mint 推出一个成熟的金融建议门户网站是完全有意义的；尤其是考虑到当前社交问答网站如 Quora 的成功。事实上，大多数美国人可能无法支付财务顾问接受个性化建议的费用，这使得该平台成为一种引人注目的资源。

Mint.com 是一家科技公司，去年秋天被 Intuit 以 1 . 7 亿美元收购。该公司最近还推出了一款安卓应用。
