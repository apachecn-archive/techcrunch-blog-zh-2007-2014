# 微软卖出了 4.5 亿份 Windows 7

今天，微软宣布了 Windows 7 的里程碑:自 2009 年 10 月发布以来，已经售出了整整 4.5 亿份许可证。当这些数字被分解时，不知何故更令人印象深刻；每天仅售出 65 万多份许可证。

过去两个月，Windows 7 的销量一直在上升；似乎所有的返校准备都给了 Windows 7 当头一棒。雷德蒙德的人们喜欢称 Windows 7 为“有史以来销售最快的 Windows 版本”，事实证明这种说法可能不只是一堆营销废话。

除了重大声明和财务报表，微软倾向于保持单位销售数据的安静，但让我们试着在这里把微软的成就放在一点点的角度。他们在 2008 年第四季度的收入报告中宣布，自发布以来，他们已经售出了 1 . 8 亿份 Vista 许可证。在 Vista 的这一阶段(发布后一年半多一点)，平均每天大约有 335，195 个许可证。如果不考虑具体情况，这看起来还不错，但在同样长的时间后，Windows 7 以每天 632，911 个许可证的数量几乎翻了一番。

平心而论，Vista 有点失败，但 7 和它的祖父 XP 相比也相当不错:微软宣布他们在 2004 年 5 月向[售出了 2 . 1 亿份 XP](https://web.archive.org/web/20221006172205/http://www.zdnet.com/blog/itfacts/210-mln-copies-of-windows-xp-sold/5688)，距离推出只有大约两年半的时间。与此同时，Windows 7 在其一岁时成功转移了 2.4 亿个许可证。

这是什么意思？首先，有大量安装 Win7 的机器，更重要的是，这些机器应该足够强大，可以运行明年左右到来的 Windows 8。此外，这意味着越来越多的用户连接到 Windows live 来使用 SkyDrive 共享和其他功能，确切人数为 542，000，000 人。这些数字非常大，令人印象深刻，很明显，微软已经成功了。
