# 一个明显而现实的危险 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/02/post-pc-has-nothing-to-do-with-windows/>

我经常想，如果我不是一个专业的博客作者，我的计算生活会有什么不同。我的意思是，我基本上一直带着我的笔记本电脑，以防我不得不闪开，随时写点东西(这种事情发生的比我愿意承认的还要多)。但如果我不是博主，我用什么呢？更广泛地说，如果我没有深陷目前控制我生活的科技领域，我会用什么呢？

答案很简单:一台 iPad。

啊！粉丝警报。啊啊！往山上跑！别读了！马上评论！我知道。我们已经经历了这个。我会说，你可以用“平板电脑”取代“iPad”，但问题是，没有任何其他平板电脑值得一文不值。那就 iPad 吧。

事情是这样的:起初，[我并不完全相信 iPad 是 PC 的替代品。对我现在的工作来说，我仍然不是。在这东西上打一百多个字简直太难了。你在报纸上一遍又一遍地听到这种重复。但这是矛盾的。媒体必须报道和评论 iPad，因为这是他们的工作。但他们也是 iPad 使用的最差候选人。](https://web.archive.org/web/20230204230859/https://techcrunch.com/2010/01/27/ipad/)

随着时间的推移，我慢慢意识到这一点。几个月前我去度假的时候，我把笔记本电脑和 iPad 都带来了。我向自己保证在旅途中不做任何工作——结果，笔记本电脑再也没有出来。一次都没有。iPad？我每天都用它，用了几个小时。

这很重要。关键是我热爱计算和网络。即使在我休息的时候，我也喜欢它。是的，断开连接——等等，等等。[我会做我想做的事情。但随着时间的推移，我被训练成认为传统的 PC 是做这些事情的方式，无论是工作还是娱乐。那根本不是真的。平板电脑的外形就是如此。很多。好多了。当你不需要做过多的打字工作时。在停机时间，当我像一个普通人一样使用电脑时，我发现那是经常的。](https://web.archive.org/web/20230204230859/https://techcrunch.com/2011/02/21/phones-at-dinner/)

快进到今天。我正在登机。在我前面的安全线是一家四口。他们总共有四台 iPads。我哪里知道？其中两个在排队时使用它们，另外两个(妈妈和爸爸)把它们拿出来扫描(尽管，在与安检人员简单讨论后发现，他们没有必要这样做)。

大约 20 分钟后，我登上了飞机。猜猜我注意到了什么？在我周围的座位上，还有半打左右的人在使用 iPads。是的，在飞机上使用笔记本电脑的人仍然较多，但并不多。

计算正在改变。这是有史以来最明显的声明。随着智能手机的兴起，我们已经看到这种情况很多年了。但是传统的计算也在发生变化。就像，人们正在抛弃个人电脑，转而使用这些更新的设备。这种情况还会继续发生。

我在个人电脑世界的中心写这篇文章:华盛顿州的西雅图，在雷德蒙以西大约 15 英里处——你知道的那个人的家乡[。微软和我已经争执了大约六年了。我以前讲过这个故事。我曾经是 Windows 的铁杆粉丝。然后我就跳到了 Mac 上，一直没有回头。现在我一边赞美苹果，一边写关于微软的](https://web.archive.org/web/20230204230859/http://microsoft.com/)[煽动性的东西](https://web.archive.org/web/20230204230859/http://parislemon.com/post/9558558373/apples-vision-for-the-future-of-computing-versus)。许多读者似乎认为我这样做是别有用心。我的动机简单明了:比起微软产品，我现在更喜欢苹果产品。

但我是一个人，我不重要。对微软来说，重要的是这种感觉的传播。哈里·麦克拉肯为《今夜时代》写道:“个人电脑并没有消亡——它只是在进化。真好。多么无伤大雅的故事。这是在回避真正发生的事情，我想他知道这一点。

这并不是说传统的个人电脑正在消亡——在我们的有生之年，它永远不会完全“消亡”。但它将成为主要的商业机器，或者如史蒂夫·乔布斯所说的“卡车”。它最终也将只是商业计算的一小部分。看看 iPads 在这一领域的业务渗透就知道了。

真正的问题是，随着传统个人电脑市场的衰退，微软是大输家。当然,“PC ”,意思是“个人电脑”,正在进化，但是它正在进化到一个超越微软的世界。iPad 比其他任何计算设备都更能给这个总部位于雷德蒙德的巨头带来明显而现实的危险。

这就是为什么我们会看到围绕 Windows 8 的所有行动和讨论。它不仅适合平板电脑，也适合个人电脑。微软正处于全力争夺模式。公司的个人私下承认这一点，但公司永远不会公开展示这种弱点。

事实是他们不需要这么做。如果个人电脑业务(再次，我们现在都认为)明天开始迅速衰退，微软仍然可以再滑行十年，没有问题。但在内部，我相信他们知道发生了什么，他们需要做什么。再次，因此，Windows 8。

我不同意[对于 Windows 8](https://web.archive.org/web/20230204230859/https://techcrunch.com/2011/08/30/we-need-an-invert-selection-button/) 的策略(至少目前我所听到和看到的)。我觉得分叉只会迷惑用户，把更多推向 iPad。但是我们会看到的。地铁看起来不错，不可否认。我只希望微软有胆量在 Metro 上全力以赴。

相反，我们听到了很多关于个人电脑没有死，它只是在“进化”的废话。我现在可以肯定，微软发布 Windows 8 时传达的信息将是“一切都是电脑”。台式机，笔记本电脑，手机。[这不是硬件的问题](https://web.archive.org/web/20230204230859/https://twitter.com/#!/fxshaw/status/108027320849285120)。猪食。

微软的问题在于公众并不愚蠢。他们看到计算机世界在他们面前发生变化，他们想要参与进来。

人们不会购买 Windows 8 平板电脑，因为它们就像他们熟悉和喜爱的个人电脑一样。事实上，如果有的话，人们*不会*因为这个原因买它们。微软认为人们热爱他们的个人电脑。他们没有。他们已经被束缚了很多年。他们想要自由。iPad 可以做到这一点。对微软来说，将 Windows 与他们的“不要称之为后 pc”战略捆绑在一起是危险的。鄙视开始超过舒适。此外，看在上帝的份上，这个世界上再也没有*窗口*了！

我完全同意惠普 CEO 李艾科离开 PC 业务是因为他在企业软件领域的根基。但我也相信，如果惠普个人电脑业务真的那么好，他绝不会提出出售或剥离的想法。对于第二大个人电脑制造商宏碁来说，事情看起来也不太妙。

坦率地说:个人电脑业务现在是一个糟糕的行业。正如麦克拉肯正确指出的，已经有一段时间了。不同的是，现在有了一个真正的替代品:iPad。其他玩家的替代品将会源源不断地出现，而且速度很快。即使其中一款安卓平板电脑不会很快上市，我打赌其中一款基于安卓系统的平板电脑会上市。

但对微软来说，最可怕的是 iPad 仍处于萌芽状态。它会继续变得更好，而且变得更快。在这一点上，肯定比 PC 快得多。抱歉，我指的是 Windows 8 消费电子系列便携家庭设备版本。

我在我的笔记本电脑(MacBook Air)上写这篇文章，但我现在真的只想在我的 iPad 上浏览网页。所以我结束这篇文章去做那件事。在接下来的五年内，我敢打赌，读到这里的大多数人也会有这种渴望——即使你认为你不会。

不要误以为平板电脑现在不能做某些事情，以后也不会。会的。而且很快。微软知道这一点。这就是为什么这是一个明显的和当前的危险。不仅仅是未来。
