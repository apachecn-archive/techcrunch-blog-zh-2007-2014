# 在 NEA 和安德森·霍洛维茨的支持下，Mattermark(前身为 Referly)希望成为风投 TechCrunch 的数据信号平台

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/06/04/backed-by-nea-andreessen-horowitz-mattermark-formerly-referly-wants-to-be-the-data-signaling-platform-for-vcs/>

如今，众所周知[风投正以一种有意义的方式进入数据领域](https://web.archive.org/web/20221006113538/https://beta.techcrunch.com/2013/06/01/the-quantitative-vc/)。公司再也不能坐等公司、个人和潮流来到沙丘路。要开始争夺交易，风投们必须在 Y Combinator 的演示日等活动开始之前，就开始发掘人才、公司和趋势。许多风投公司已经在内部建立了自己的数据解析平台，但今天推出的一家初创公司 [Mattermark](https://web.archive.org/web/20221006113538/http://mattermark.com/) 希望成为公司在量化增长和潜在利润丰厚的初创公司的信号时使用的软件。

Mattermark 是 Danielle Morrill 的创意，她之前创立了[referry](https://web.archive.org/web/20221006113538/http://refer.ly/)，这是一种为喜欢策划产品的人通过附属收入赚取额外现金的方式。Referly 在 3 月份被关闭(而且它的大部分员工都被解雇了)；莫里尔、她的丈夫凯文·莫里尔和安迪·斯帕克斯(二月份通过收购 Referly 的 LaunchGram】加入莫里尔)现在专注于 Mattermark。

Morill 是 Y Combinator via Referly 的毕业生，他对风投如何挖掘数据来识别公司越来越感兴趣。她之前是西雅图 2.0 的编辑，并帮助制作了西雅图 2.0 指数，这是一份所有当地创业公司的月度排名列表。根据她的经验，她意识到有机会在风险投资领域使用这些数据。正如她所解释的，“如果风投在 YC 的展示日之前看不到一家公司，他们赢得交易的机会就更小了。所以投资者需要变得聪明，提前发现这些机会，”她说。

该软件允许风险投资者从 Twitter、新闻网站、SEC 文件、LinkedIn、AngelList、CrunchBase 和公司自己的专有创业指数中获取数据，以发现潜在的机会并跟踪现有的创业公司。这是一个有趣的例子，是 Morill 和她的团队正在试验的一些数据潜水。

对于种子投资者来说，这就像一个早期预警系统，检测那些在成为热门交易之前很久就获得客户关注的企业(例如，该应用程序将识别突然开始获得大量 Twitter 提及的初创公司)。对于后期阶段的投资者来说，这是一种研究创业公司并在一个地方获得大量信息的方式，甚至可以将它们与同行业的其他公司进行比较(例如，LinkedIn 粉丝的增加与后期阶段的成功密切相关)。

您可以按阶段、垂直和地理位置查询、跟踪和组织公司，并将这些数据下载到 Excel 或 PDF 中。Mattermark 通过强大的过滤器使所有这些数据更容易消化。例如，你可以查询哪些初创公司正在构建尚未筹集 A 轮资金的开发者工具，并在 Twitter 上按参与度对这些公司进行排名。该网站还提供实时警报和每周电子邮件摘要。

Morrill 说，除了用于采购交易，该软件还可以用于研究潜在投资的背景或为客户创造销售线索。安德里森·霍洛维茨、Ignition 和 NEA 是 Mattermark 的忠实支持者，他们已经开始在内部使用这个应用程序。Morrill 补充说，这家初创公司也在寻求为某些公司开发定制应用程序。

有一个免费的订阅产品，提供每周简讯和访问所有公司排行榜。付费专业帐户是一个订阅，并提供完整的数据库访问，自定义报告和警报。

毫无疑问，对风投和投资者来说，成为彭博终端是一个重大机遇。直到最近，大部分投资都是通过口口相传或介绍获得的。如今对风投来说，这是一个不同的世界，知识和数据就是力量。

正如 Morrill 所解释的，“我们不认为我们可以取代投资者与企业家建立关系的能力，但我们确实认为我们可以帮助他们更有效地寻找机会，并减少错过鲸鱼的机会。“当然，Mattermark 的成功将取决于它是否允许风险投资公司帮助寻找下一个 Instagram。
