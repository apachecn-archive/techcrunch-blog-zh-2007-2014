# 保罗·亚当斯，谷歌人，他的演讲预言了“脸书集团”，前往脸书

![](img/8e8e45fbea4d57049e3e27c3caf34cf4.png)今天早些时候，谷歌用户体验主管[保罗·亚当斯、](https://web.archive.org/web/20230204234840/http://www.crunchbase.com/person/paul-adams)他的被广泛讨论的[幻灯片](https://web.archive.org/web/20230204234840/http://www.slideshare.net/padday/the-real-life-social-network-v2)揭示了脸书的社交图表现人际关系的方式的差距，宣布他将于一月份前往脸书。亚当斯在谷歌 UX 团队工作了四年多。

Adams 令人印象深刻的[“真实生活社交网络”](https://web.archive.org/web/20230204234840/http://www.slideshare.net/padday/the-real-life-social-network-v2)幻灯片，获得了超过 40 万的浏览量，1799 次脸书分享，是[大量媒体报道的主题，](https://web.archive.org/web/20230204234840/http://www.techmeme.com/search/query?q=paul+adams&wm=false)暗示了谷歌如何利用人类是多个社交团体的一部分这一事实成功进军社交市场。

除了预示了三个月后脸书自己对其集团产品的重新构思，这次展示给许多人的印象是谷歌终于认真对待社交了。随着谷歌继续加大这些努力，这一举措出现在一个有趣的、或许是不恰当的时机。

根据他的推文，亚当斯将在新年加入脸书。我已经联系了亚当斯和脸书以获得更多的信息，但是还不知道他在脸书的官方头衔是什么。他在 LinkedIn 上的简介仍然是“谷歌高级用户体验研究员”
