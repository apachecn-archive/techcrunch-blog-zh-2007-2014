# 谷歌准备推出 Chrome 网上商店(这次是真的)

早在八月，在欧洲的一次会议上，谷歌展示了 Chrome 网上商店的预览，并指出发布会可能会在十月举行。虽然有一些[暗示](https://web.archive.org/web/20221001094949/https://beta.techcrunch.com/2010/10/19/chrome-web-store-launch-2/)它会在 10 月份到来(包括一些定价细节，显然是偶然打开的)，但它从未到来。由于今天是 11 月的最后一天，我认为可以肯定地说这个月也不会到来。但现在看来，谷歌肯定正在为 12 月初的商店发布做准备，可能会推出 Chrome 8 或 Chrome 9 测试版。

今天早些时候，Chromium 问题列表中出现了一系列活动。具体来说，围绕“ [ReleaseBlock](https://web.archive.org/web/20221001094949/http://code.google.com/p/chromium/issues/list?q=label:ReleaseBlock-Stable) ”标签的项目有很多活动。如果你仔细阅读，你会发现这 16 个问题中的大多数都与网络商店或 Chrome 应用程序有关。谷歌似乎正在做收尾工作，以尽快推出这款产品。

那么正在研究什么呢？首先，正如你在上面看到的，谷歌为网上商店设计了一个新的标志。这个标志已经进入了 Chrome 的最新版本，应该会很快进入 Chrome 的开发频道。此外，将在浏览器中展示的 Chrome Apps 宣传片即将完成——他们正在[修复一些 bug。](https://web.archive.org/web/20221001094949/http://code.google.com/p/chromium/issues/detail?id=64736&q=label:ReleaseBlock-Stable&colspec=ID%20Stars%20Pri%20Area%20Feature%20Type%20Status%20Summary%20Modified%20Owner%20Mstone%20OS)

与此同时，谷歌正试图[完成更新文档](https://web.archive.org/web/20221001094949/http://code.google.com/p/chromium/issues/detail?id=59535&q=label:ReleaseBlock-Stable&colspec=ID%20Stars%20Pri%20Area%20Feature%20Type%20Status%20Summary%20Modified%20Owner%20Mstone%20OS)，详述[扩展](https://web.archive.org/web/20221001094949/http://code.google.com/chrome/extensions/whats_new.html)的新内容。扩展将成为网络商店的一部分(与主题和应用程序一起)，所以他们准备好文档的事实是另一个很好的迹象，表明发布非常接近。

另一个正在开发的相当大的功能是为应用程序创建桌面快捷方式的能力。你可以右击 Chrome 中的一个应用程序，并为它创建一个桌面快捷方式。这方面的编码工作似乎已经完成，现在正在实施。

谷歌也在为 Omnibox 的一些新的 API 特性做最后的工作，包括一些新的 Chrome 搜索扩展。

Chrome 9 中一个更大的变化是可以不用 crx 格式创建应用[。这现在被指定为 M10 特性。crx(当前 Chrome 扩展使用的)的一个问题是它们的大小被限制在 10 MB。](https://web.archive.org/web/20221001094949/http://code.google.com/p/chromium/issues/detail?id=61546&sort=-modified&colspec=ID%20Stars%20Pri%20Area%20Feature%20Type%20Status%20Summary%20Modified%20Owner%20Mstone%20OS)

鉴于这些东西被贴上了“M9”拦截器的标签，尚不清楚谷歌是否会在测试版中推出 Chrome 网络商店作为 Chrome 9 的一项功能，或者在稳定版中推出 Chrome 8。Chrome 7 是目前浏览器的最新稳定版本，而 Chrome 8 处于测试阶段，Chrome 9 处于开发频道。同样，你可能会期待该公司很快将这些全部转移。这甚至可能在下周早些时候发生，这将与一个月前[媒体备忘录](https://web.archive.org/web/20221001094949/http://mediamemo.allthingsd.com/20101027/how-to-find-the-google-chrome-app-store-wait-till-december/)报道的一致。

Chrome 网上商店也有望成为 Chrome 操作系统的重要组成部分，因此谷歌希望首先推出 Chrome，然后推出 Chrome 操作系统的第一个版本是有道理的，[仍计划在今年年底](https://web.archive.org/web/20221001094949/https://beta.techcrunch.com/2010/11/22/google-chrome-os-2010/)前推出。
