# 热衷于…如何制作电影和赚钱(TCTV)

“艺术家也需要成为商人，”电影制作人蒂芙尼·施莱恩走进旧金山的 TechCrunchTV 工作室时告诉我。史莱恩的电影《T2 与 T3》于本周早些时候上映，她是一位先锋电影制作人，不仅因为她的获奖电影，还因为这位创新的女商人找到了在当今数字经济中赚钱的方法。

“大门现在已经打开，”史兰告诉我 21 世纪的电影业务，她说，这与 20 世纪的电影公司模式“完全不同”。她解释说，这是因为电影制作人现在拥有了与观众直接沟通的技术。她告诉我，把你 50%的精力放在电影制作上，另外 50%放在推特和脸书等网络上推广。

Tiffany Shlain 证明了在当今的数字经济中，创造力和创业创新不仅可以，而且必须共存。电影制作人、作家和音乐家应该仔细听听她关于利用观众力量的建议。
