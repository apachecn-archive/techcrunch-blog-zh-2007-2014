# Skype 正在谈判收购 Gizmo5 | TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/10/13/skype-in-negotiations-to-acquire-gizmo5/>

多个消息来源称，诉讼缠身的 Skype 正在就收购 P2P 网络电话初创公司 [Gizmo5](https://web.archive.org/web/20230406113906/http://gizmo5.com/) 进行谈判。据说价格在 5000 万美元左右，但交易远未完成。其他潜在买家可能也在关注这家公司。Gizmo5 成立于 2003 年 4 月，迄今为止已经筹集了 600 万美元，外加创始人兼首席执行官 Michael Robertson 的一笔数目不详的资金。

Skype 正从易贝出售给一个新的投资者团体，它对其核心的 P2P 技术没有控制权，该知识产权现在是两起诉讼的主题——一起是针对 Skype 的[一起是针对迈克·沃尔皮和 Index Ventures 的](https://web.archive.org/web/20230406113906/https://techcrunch.com/2009/07/31/ebay-in-litigation-with-skype-founders-over-key-technology/)[一起是针对收购团体](https://web.archive.org/web/20230406113906/https://techcrunch.com/2009/09/18/joost-sues-former-chairman-and-ceo-mike-volpi/)的[。](https://web.archive.org/web/20230406113906/https://techcrunch.com/2009/08/28/investor-group-makes-play-for-skype/)

这使 Skype 面临巨大风险。没有这项技术，Skype 就无法进行 Skype 至 Skype 通话，这是其服务的核心。这些诉讼似乎不会很快达成和解——一些熟悉 Skype 的人认为这些案件很有可能最终会在法庭上审理。

我们听说 Joltid 想要多达一半的 Skype，除非 Skype 向原告 Joltid 做出巨大让步并解决此案，否则他们有两个后备计划。

首先是重建后端本身。第二是获得一个可以替代 Joltid 的人。Skype 正在双管齐下。

他们有一个名为 Gecko 的内部项目，为 Skype 构建一个基于 SIP 的 P2P 后端。他们正在疯狂招人。Theo Zourzouvillys、Bruce Lowekamp 和 Jason Fischl 只是他们最近在 Skype Labs 内从事 Gecko 项目的几位高级 VoIP/SIP 专家。

但是 Gecko 的进展可能没有 Skype 需要的那么快。这就是 Gizmo5 的用武之地。

Gizmo5 是一种类似 Skype 的可靠服务。只有 600 万注册用户，这远不是一个竞争威胁。但 Gizmo5 是一个经过测试的、部分规模化的 SIP P2P VoIP 系统，其工程师团队了解 Skype 面临的问题。

与 Skype 不同，Gizmo5 不能通过 P2P 进行所有通话。服务器节点仍然用于 NAT 和防火墙穿越。但是这些年来带宽价格大幅下降，Skype 架构的成本节约可能不再是保持盈利的必要条件。

随着事态的发展会有更多的报道。
