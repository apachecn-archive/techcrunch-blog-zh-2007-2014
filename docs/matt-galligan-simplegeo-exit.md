# SimpleGeo 联合创始人马特·加利根下台；将致力于帮助科技之星和一个非盈利组织

去年 11 月，定位初创公司 [SimpleGeo](https://web.archive.org/web/20230203045557/http://simplegeo.com/) 聘请前 Digg 首席执行官[杰伊·阿德尔森](https://web.archive.org/web/20230203045557/http://www.crunchbase.com/person/jay-adelson)接替联合创始人[马特·加利根](https://web.archive.org/web/20230203045557/http://www.crunchbase.com/person/matt-galligan)担任首席执行官。当时，Galligan 滑入首席战略官的角色，成为公司更多的公众形象。现在他完全离开了公司。

你可以在 SimpleGeo 的博客[这里](https://web.archive.org/web/20230203045557/http://blog.simplegeo.com/2011/08/22/matt-galligan-stepping-down/)阅读新闻，也可以在这里阅读 Galligan 自己的想法[。但 Galligan 告诉我们，从本质上来说，这是自 Adelson 几个月前接手以来一直在进行的工作。随着公司进入下一阶段，Galligan 觉得是时候完全退出了。他将继续担任顾问，但阿德尔森和 SimpleGeo 的联合创始人乔·斯坦普将继续掌舵。](https://web.archive.org/web/20230203045557/http://mgalligan.com/post/9266505212/matt-galligan-stepping-down-simplegeo)

当 Galligan 想出他下一步想做什么的时候，他会休息一段时间。2009 年 5 月离开 AOL 后，他直接进入 SimpleGeo(当时名为 Crash Corp .)。2008 年 8 月，他将自己的第一家创业公司 Socialthing 卖给了美国在线。

但 Galligan 在休假期间并没有完全退出创业领域。他将帮助 TechStars，给这个孵化器在西海岸带来急需的声音和存在。Galligan 与 TechStars 社区有着长期的联系，因为 Socialthing 是 2007 年的首届课程。

此外，他将很快与 Shervin Pishevar 一起成立一个非盈利基金会，Shervin pishe var 是一位资深企业家，现在是 Menlo Ventures 的董事总经理。这个名为“[百分之一无”的基金会将要求创始人承诺将他们公司百分之一的股份捐给慈善机构，以防流动性事件。Galligan 自己做到了这一点，他将 Socialthing 销售额的 1%捐给了科罗拉多州博尔德市的社区基金会。](https://web.archive.org/web/20230203045557/https://twitter.com/#!/1percentof)

到目前为止，SimpleGeo 已经从 T4 筹集了大约 1000 万美元的资金。当他们进入下一个阶段时，有传言说他们会筹集更多的资金。
