# 新加坡解除“质量效应”禁令

![](img/c3e1049ff1bc299340a1dd1a22a68c22.png)

路透社报道称，新加坡政府解除了最近对 Xbox 360 游戏 Mass Effect 的禁令，我们本周[告诉过你](https://web.archive.org/web/20221209051115/http://www.crunchgear.com/2007/11/15/xbox-360s-mass-effect-banned-in-singapore/)([两次](https://web.archive.org/web/20221209051115/http://www.crunchgear.com/2007/11/16/mass-effect-banned-in-singapore-for-hot-sweaty-cross-species-action/))。

这款游戏现在将贴上“M18”的标签，这样只有达到法定年龄的人才能见证一个场景，这个场景包含一个外星女性和一个人类女性，他们通过像“亲吻和爱抚”这样的成年动作来表达对彼此的强烈感情。

这是新加坡首次对游戏进行评级。显然，评级系统本应在 1 月份实施，但由于 dunebuggy 后座上所有的[草率的火星爱情](https://web.archive.org/web/20221209051115/http://snltranscripts.jt.org/98/98fharrycarey.phtml)而不得不快速跟进。
