# Flickr 通过 Twitter 和脸书的“分享这个”功能涉足社交

![](img/0084a0262d404e9885a6abc2b8405fb5.png)
照片分享网站 [Flickr](https://web.archive.org/web/20230203042958/http://www.flickr.com/) ，就储存在该服务上的照片数量而言具有无限的潜力(最新统计超过 50 亿张),通过新的 Twitter 和脸书[“分享这个”](https://web.archive.org/web/20230203042958/http://blog.flickr.net/2011/03/30/upload-once-share-everywhere/)功能，用户今天可以更容易地分享他们的照片。随着新特性的出现，共享界面也有了轻微的改变。

以前，你只能通过发送电子邮件、手动抓取链接/代码或在 Blogger 上分享 Flickr 上的照片流、照片组和照片集，而现在，用户可以通过每张照片左上角的“分享这个”下拉菜单，在脸书和 Twitter 以及 Tumblr 上发布个人照片和其他任何内容。

有趣的是，这些新功能可以在不同的隐私级别上运行:登录的用户可以在脸书上与朋友分享私人照片，但只有公共内容可以通过 Twitter 和 Tumblr 分享，或者在脸书上注销的用户可以分享。

Flickr 可能感受到了来自 Instagram 甚至 Color 等移动社交照片服务的压力，计划尽快将这些功能引入他们的移动应用和网站。欢迎来到派对 Flickr。
