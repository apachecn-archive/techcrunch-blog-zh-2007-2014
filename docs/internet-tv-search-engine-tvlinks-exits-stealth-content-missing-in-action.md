# 网络电视搜索引擎 TVLinks 退出隐身，内容在行动中失踪

![](img/5a65aaf8eaf991ef781a525a5e81442c.png "Picture 133")这是什么，欧洲创业公司思维全球化？互联网电视指南和视频搜索引擎 TVLinks 今天退出秘密模式，并表示它想与美国大型搜索引擎和电视列表网站竞争。

它目前拥有超过 30，000 部电影和 2，000 部电视节目的数据库，包括 80，000 集。它总共索引了超过 2000 家内容提供商网站，其中一些网站与以美国为中心的亚马逊视频点播(Amazon VOD)、网飞(Alibaba)和谷歌(Google)有正式合作关系，而其他网站可能是通过开放 API 抓取或抓取的。结果是，搜索结果中显示的许多内容可能会被放在各自的视频共享网站上，无论是否得到版权所有者的许可和/或被地理屏蔽。

![](img/99c1604e5387dd32cab5b53f8e542d08.png "Picture 132")例如，当我真正尝试观看视频时，我从我的英国 IP 地址尝试的所有搜索都没有结果。我被送到了 YouTube(“这个视频在你所在的地区不可用”)、Megavideo(这个视频显然已经被删除了)，以及其他一些收效甚微的网站。当然，这不完全是 TVLinks 的错，但反映了互联网电视的现状:它支离破碎，高度依赖物理边界，经常是一大堆撤下通知。也就是说，这些缺点并没有阻止用户利用电视链接搜索内容，不管是合法的还是非法的。

尽管在“秘密模式”下运营，TVLinks 声称每月有 400 万次访问和 2000 万次页面浏览。该公司称其网站是“发现和观看在线电视节目和电影的最佳目的地之一”，列举了其“直观的剧集指南和时间安排系统”、视频搜索引擎以及社交网络等功能。

现在，如果这一切听起来[非常谷歌电视](https://web.archive.org/web/20221005223717/https://beta.techcrunch.com/2010/05/20/google-tv/)，那是因为它确实如此。事实上，TVLinks 将其未来视为谷歌平台上的一项服务，以及基于 Android 的未来设备，包括机顶盒。

“我们是第一家与风险投资的美国聚合商和视频搜索引擎竞争的欧洲公司。TVLinks 的创始人之一 Adrian Constantin 表示:“我们的目标是获得巨大的市场份额，并利用谷歌电视和 Android 应用程序等产品来促进我们向最终用户的分销。”。
