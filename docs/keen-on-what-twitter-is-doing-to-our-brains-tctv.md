# 热衷于…推特对我们大脑的影响(TCTV)

不是每个人都同意尼古拉斯·卡尔的观点，互联网正在摧毁我们的大脑。以加州大学洛杉矶分校大脑研究所教授[迪安·布诺玛诺](https://web.archive.org/web/20221225234106/http://www.crunchbase.com/person/dean-buonomano)为例，他是[大脑缺陷:大脑缺陷如何塑造我们的生活](https://web.archive.org/web/20221225234106/http://www.amazon.com/Brain-Bugs-Brains-Flaws-Shape/dp/0393076024)的作者。虽然 Buonomano 承认互联网的好处是有“代价”的，但他相信我们的大脑最终会适应像 Twitter 这样的实时网络。

《脑虫》对企业家和营销人员来说都是一本有趣的书。Buonomano 说，鉴于大脑充当“计算设备”，它实际上控制着我们的一切——从我们的联想到我们的食欲，再到我们的购买决定。他解释说，正是我们大脑中的缺陷，那些大脑缺陷，为我们的行为方式以及我们想要和购买的东西提供了最好的介绍。
