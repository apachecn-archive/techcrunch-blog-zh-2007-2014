# 青少年博客网络被科技界收购

[![](img/69f241e89dfec9064500d7233f3631a8.png)](https://web.archive.org/web/20221006213810/http://www.teensintech.com/)

由 16 岁的 Daniel Brusilovsky 创立的博客网站 Tech 的青少年已经收购了[青年博客网站](https://web.archive.org/web/20221006213810/http://youthbloggers.net/) (YBN)。YBN 由 100 多名年轻博客作者组成，创始人是帕特里克·德维沃，他也是一名年轻的企业家。交易的条款没有披露，但鉴于这两个网络如此相似，我怀疑如果有任何货币交易，也是非常小的——听起来这两个网络正在联合起来，以帮助建立牵引力。

“科技青少年”旨在为青少年提供一种简单的方式，在一种既安全又容易接受他们想法的氛围中写博客。网站[在八月推出了](https://web.archive.org/web/20221006213810/http://www.beta.techcrunch.com/2008/08/04/15-year-old-entrepreneur-launches-teens-in-tech/)的私人 alpha 版，还没有向公众开放注册(Brusilovsky 说大门将在本周末暂时开放，但之后它将再次成为私人的)。虽然该公司最初计划在去年冬天向公众开放，但它目前正在探索建立自己的 CMS，并正在努力为其出版商建立一个广告计划。

正如我在该网站首次推出时所写的，在 Tech 中的青少年背后似乎没有太多独特的技术——该网站本质上是一个多用户 WordPress 安装，有一些定制的调整，如从所有网络博客中提取的 RSS 提要。也就是说，显然有一个热衷于写博客的青少年社区。我只是不确定科技领域的青少年将如何从他们身上建立一个可行的业务——鉴于其他地方有这么多免费的发布平台，让青少年(他们并没有现金)付费写博客将会很难。从规模上来看，科技领域的青少年很难确保其安全的青少年社区保持青少年社区的状态。
