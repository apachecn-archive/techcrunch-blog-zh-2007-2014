# 热衷于…互联网需要更多的全球监管吗？(TCTV)

在伦敦骚乱之后，关于互联网——尤其是像 Twitter 和脸书这样的社交媒体网络——是否需要更多的监管，出现了激烈的辩论。但是根据 [Chris Painter，](https://web.archive.org/web/20230205043734/http://www.crunchbase.com/person/chris-painter)美国国务院最近任命的网络问题协调员的说法，本月初的骚乱并不代表世界各国政府监管互联网的借口。

本周早些时候，我在[技术政策研究所的阿斯彭论坛](https://web.archive.org/web/20230205043734/https://techpolicyinstitute.org/aspen2011/)上见到了 Painter，他在众星云集的[小组](https://web.archive.org/web/20230205043734/https://techpolicyinstitute.org/video/aspen2011/110822-post_wikileaks.php)上谈到了后维基解密和后埃及世界。画家给世界的信息是明确的。希拉里·克林顿任命的人告诉我，互联网不仅带来自由，也带来繁荣。他向我承诺，美国将坚定地致力于让互联网成为其外交政策中“非常重要”的一部分，因为它积极支持世界各地的言论自由和民主。
