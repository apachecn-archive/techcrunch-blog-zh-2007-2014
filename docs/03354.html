重返校园:9元能量饮料大战TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2009/08/31/back-to-school-a-9-energy-drink-shootout/

哦男人。我不认为在九天内喝九种不同的能量饮料是健康的。我的预期寿命肯定会受到影响。哦，好吧，这是以科学和嘎吱嘎吱齿轮的名义做的。在这场枪战之前，我觉得所有的能量饮料尝起来都像尿，让你浑身发抖。有些人会，但我发现肯定有比其他人更好的选择。

几个注意事项:

  • 我每天至少喝一罐汽水，随便你怎么说，我不在乎
  • YMMV的价格，但所有这些都来自同一个加油站
  • 一些caffine内容信息来自EnergyFiend.com
  • 我现在必须在下午2点前喝一杯能量饮料才能正常工作

激浪

味道:柠檬酸橙
成本:1.49美元一瓶20盎司
添加剂:咖啡因和黄色5
刺激因子 : 1/10
咖啡因含量:55毫克
声称成名:自1964年以来一直让游戏玩家保持清醒并腐蚀牙齿

激浪是主要的能量饮料。它可能没有各种各样的草药和维生素，但它能完成任务。除非你像我一样对适量的咖啡因和HFCS产生了免疫力。现在我们很多人只是为了快乐而喝它。仅仅几个星期后，百事可乐就终止了激浪返潮，这真是太糟糕了。嗯，真正的糖。

NOS葡萄

口味:就像葡萄汽水
成本:2.29美元一罐16盎司
添加剂:瓜拉那、人参、牛磺酸&左旋肉碱
反冲因子 : 4/10
咖啡因含量:260毫克
号称名气:一大堆咖啡因

嗯，这东西给了我签名能量饮料崩溃头痛，没有首先提供一脚能量。就像——砰——头疼。一点也不难吃，但我完全不能称之为能量饮料。它更像是一种令人沮丧的饮料，这很奇怪，因为它比大多数其他饮料含有更多的咖啡因。我想我不会再买这个了。

摇滚明星榨汁——芒果橙

口味:有点像麦当劳的碳酸橙汁饮料
成本:1.99美元一罐16盎司
添加剂:瓜拉那、人参、牛磺酸& B族维生素
反冲因子 : 6/10
咖啡因含量:160毫克
号称50%果汁

有些人真的很喜欢橘子味的饮料。如果是你，拿起这个能量饮料。它有令人愉快的芒果橙味道。这几乎就像你阿姨为家庭野餐做的东西，除了这个装满了提神饮料。哦，我在想它会成为一种杀手螺丝刀一样的饮料。

全油门

口味:仿品露
成本:2.29美元一罐16盎司
添加剂:肉碱、蔗糖&牛磺酸
反冲因子 : 6/10
咖啡因含量:144毫克
号称:可口可乐产品

全速前进味道不差，但也不好。有点像塔可钟激浪的味道。你知道糖浆混合物尝起来总是一团糟吗？是的，那是全速前进。这种饮料给人一种温和但也令人满意的活力。这绝对不是我喝过的最烈的，但对新手来说可能不错。

Lo-Carb怪物

口味:浓浓的碳酸苹果汁混合甜味
成本:16盎司一罐的2.39美元
添加剂:肉碱、人参、牛磺酸& B族维生素
刺激因子 : 7/10
咖啡因含量:160毫克
成名:我的最爱

怪物，我多么爱你。你尝起来只比尿好一点点，但出于某种原因我不介意。请注意，我说的是蓝色的低碳水化合物怪物，而不是经典的绿色怪物。当然不是。绿色怪物比红牛还难吃。

安培能量牵引

口味:想葡萄冰棍
成本:16盎司一罐的2.29美元
添加剂:瓜拉那、人参、牛磺酸& B族维生素
反冲因子 : 7/10
咖啡因含量:160毫克
成名:小戴尔喝了这种东西——或者他的鼻子这么说

我不得不说这是最好喝的能量饮料之一。起初，葡萄的味道有点强烈，但它没有大多数能量饮料所具有的标准马尿味道。正因为如此，能量饮料的新来者应该不会有任何问题。它就像一种非常甜的葡萄汽水。

红牛

味道:马尿
成本:2盎司一罐2.09美元
添加剂:牛磺酸
反冲系数 : 9/10
咖啡因含量:80毫克
号称名气:原

如果看起来像马尿，闻起来像马尿，吃起来像马尿，那一定是红牛。哦，对了，我没有忘记冰箱里的红牛。

XS能量饮料

口味:就像Faygo Redpop
成本:2.29美元一罐8.4盎司
添加剂:大量B族维生素&牛磺酸
反冲因子 : 9/10
咖啡因含量:？？？
声名远播:一种“健康”能量饮料

这只小狗富含四种不同类型的B族维生素，就像5小时的能量一样。你知道吗，它就像5小时的能量，除了味道很好，你实际上得到喝东西的快乐，而不是吃药。更好的是，所有这些能量不含任何碳水化合物或糖，只有8卡路里。可惜店里没有，而且只能从安利买。我会再次购买这个。

5小时能量——浆果

口味:掺水的Robitussin
成本:2盎司一罐的2.99美元
添加剂:牛磺酸、维生素B12、维生素B6、&葡萄糖醛酸内酯
反冲因子 : 10/10
咖啡因含量:138毫克
声名鹊起:现在能量小时

该死的。这东西踢你的屁股。我已经喝了很多能量饮料，但这是我第一次尝试5小时能量。这不是最好吃的东西，但它很容易消化。一旦情绪低落，哇，没有通常的能量饮料发呆，能量几乎立刻就开始发挥作用。更好的是，现在有一个XL大小的5小时能量，以防有人的身体已经习惯了大量的B族维生素和魔法果汁。上帝保佑。
