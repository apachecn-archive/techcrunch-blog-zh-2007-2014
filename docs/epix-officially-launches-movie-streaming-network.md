# Epix 正式推出电影流媒体网络

![](img/38e9a27c2c7d76e7ffda23fa08f0c040.png)

我们最近[写了](https://web.archive.org/web/20221207195524/http://www.beta.techcrunch.com/2009/09/21/hollywood-we-have-plenty-of-half-services-its-time-for-a-whole-one/)关于 [Epix，](https://web.archive.org/web/20221207195524/http://www.epixhd.com/)这个电影和娱乐流媒体网络可以在你的电视、电脑和手机上运行。在狮门影业、米高梅和派拉蒙的共同支持下，Epix 今天推出了 150 部电影的服务，明年还将增加 3000 多部电影、音乐会和演出。

Epix 表示，最终它将能够提供超过 15，000 部作品的访问权限，但正如我们在对该服务的初步评估中所写的那样，这只是现有内容的一小部分，因为只有通过工作室提供的电影才是支持该公司的。而且，如果你想在你的客厅里看到这些内容，你需要[威瑞森 FIOS，](https://web.archive.org/web/20221207195524/http://www.forbes.com/2009/09/17/movies-hollywood-cable-television-business-entertainment-epix.html)只有大约 200 万人拥有。

但下个月，Epix 将允许有限数量的人从本周末开始，免费在线预览其 EpixHD.com 服务三天。在此期间，无论您是否是威瑞森 Fios 用户，您都可以访问该服务。我们实际上有 200 张 30 天的 VIP 通行证。前 200 名在 Twitter 上关注 [@epixHD](https://web.archive.org/web/20221207195524/http://twitter.com/epixhd) 并以“#techcrunch”为标签发布“@epixhd Experience Epix”的读者将收到 Epix 的 DM，其中包含 VIP 服务的促销代码。

我们之前对这项服务有一些意见，即它依赖于威瑞森·菲奥斯，它没有提供一个完整的网络电影观看平台(但没有人提供)。也就是说，这个平台是有用的，因为它超越了电视和电脑，而且相当容易使用。对于那些能够获得邀请的人，我们很有兴趣听听你们对这项服务的看法，所以一定要在评论中留下你们的看法。
