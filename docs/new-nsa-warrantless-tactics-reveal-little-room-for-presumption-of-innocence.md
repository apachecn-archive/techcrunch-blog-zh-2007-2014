# 新的国安局无证战术揭示无罪推定的空间很小

《卫报》 [发布了](https://web.archive.org/web/20221209173338/http://www.guardian.co.uk/world/2013/jun/20/fisa-court-nsa-without-warrant)关于美国国家安全局间谍活动的新细节，揭露了分析师如何在没有授权的情况下存储大量数据。具体来说，如果 NSA“无意中”发现了任何与潜在犯罪有关的东西，它可以存储这些数据供以后调查使用。

相当合理的是，最高法院已经宣布，如果在“[明显可见的地方](https://web.archive.org/web/20221209173338/http://en.wikipedia.org/wiki/Plain_view_doctrine)”进行犯罪，执法部门可以指控公民犯罪——例如，如果警察在一次交通停车时看到大麻坐在汽车的乘客座位上。也就是说，无罪推定不适用于警察无意中目击犯罪的情况。不幸的是，随着政府的眼睛变大，无罪推定的范围越来越小。

根据这些文件，国家安全局被要求“最大限度地减少”对美国公民的任何监控，这超出了这个面对外国敌人的机构的管辖范围。如果电话记录或互联网地址之间的通信似乎完全在美国境内或涉及美国公民，数据将被“立即销毁”

然而，电话记录和互联网数据可以保留，如果通信

*   “有理由相信含有已经、正在或即将实施的犯罪证据”
*   包含“理解或评估漏洞所必需的”信息
*   “被合理地认为包含重要的外国情报信息”

换句话说，如果 NSA 偶然发现可以阻止犯罪的证据，他们应该保留它。但是，将这种逻辑应用于数字信息是完全不同的，因为它使警察有能力监视每个人。事实上，第九巡回上诉法院的一项裁决警告说，将普通证据扩大到数字证据，“会产生一个严重的风险，即每一份电子信息授权令实际上都将成为普通授权令，使第四修正案变得无关紧要”。

当然，国家安全局间谍活动的法律理由都是绝密的，所以我们不知道他们用什么具体的先例来证明无证例外。抛开细节不谈，教训是显而易见的:广泛监控数据是有后果的，即使是出于好意。我们将如何使新技术符合宪法，这是我们作为一个社会必须努力解决的问题——如果我们能看到法律上的合理性的话。
