# 脸书正在努力打造针对平板电脑优化的体验

我们需要制作一个平板电脑版的脸书。这是我们现在正在做的事情

这是脸书首席技术官 [Bret Taylor](https://web.archive.org/web/20221207034634/http://www.crunchbase.com/person/bret-taylor) 昨天在旧金山举行的 [Inside Social Apps](https://web.archive.org/web/20221207034634/http://insidesocialapps.com/) 会议的舞台上接受一对一采访后接受我的采访。

“从长远来看，我们认为很多应用程序将会用 HTML5 来编写，”泰勒说，呼应了他在舞台上分享的一些想法。他还谈到要确保脸书数据的各种渠道和今天存在于网络上的互动转移到脸书的各种平台上。同样，这也符合他关于 2011 年重点精简产品线的声明。

具体就平板电脑而言，泰勒指出“*我们对此有点谨慎*”，他说，从事此类项目的团队一直相对较小，有些分散，因为他们致力于各种不同类型的设备体验(尽管首席执行官马克·扎克伯格[不认为 iPad 是移动设备](https://web.archive.org/web/20221207034634/https://beta.techcrunch.com/2010/11/03/facebook-ipad/))。

“*iPad 是内部*的牺牲品，”泰勒继续说道，听起来好像他很清楚人们对原生 iPad 体验的强烈渴望。泰勒说，但在未来一年，资源将会转移，更多的注意力将会放在手机和平板电脑上。
