# DreamIt 支持的 CloudMine 让应用程序开发人员绕过后端的痛苦，专注于他们的产品

我认为这是一个公平的假设，对于应用程序开发人员来说，他们工作中令人愉快的部分是构建实际的应用程序，而不是找到正确的网络服务和托管提供商，建立数据库，并苦干配置。我认为这是真的可能是错的，但是今天费城的初创公司 CloudMine 愿意打赌大多数开发者可能会同意我的观点。

因此，CloudMine 开发了一项服务，允许开发人员通过提供一组 RESTful APIs 来减少痛苦，这些 API 允许他们快速为自己的应用程序创建后端解决方案。具体来说，这家初创公司正在提供无模式数据结构存储、用户帐户创建和管理，以及服务器端业务逻辑，用于过于复杂或数据密集型的计算，以至于无法在移动设备上运行。

CloudMine 目前是 DreamIt Ventures 费城加速器计划中当前一批羽翼未丰的初创公司的一部分，该计划已向 CloudMine 的后端解决方案投资了 2 万美元的种子资金。该服务已经进行了几个月的封闭测试，其测试人员报告称，CloudMine 将开发人员配置后端解决方案的时间缩短了一半。

开发者可以直接使用该服务的基于 REST 的 API，或者为他们选择的平台下载其 SDK，让应用程序创建者有机会通过专注于产品(并将其呈现在用户面前)来增加他们的价值主张，而不必担心修补服务器、web 应用程序或扩展。

我喜欢 PaaS 服务提供“B2D”(企业对开发者)解决方案，使开发者能够将注意力从基础设施转移到产品测试和迭代上。非常酷的是，开发者可以[免费注册](https://web.archive.org/web/20230203111824/http://cloudmine.me/)，并立即为他们的第一个应用程序获得一个 API 密钥——一旦他们准备好，只需点击一下就可以快速为其他应用程序生成密钥。

至于谁是这家初创公司的幕后黑手:CloudMine 是由伊利亚·布劳德(前东方研究公司，被 Sycamore Networks 和 Drakontas 收购)、马克·韦尔(曾在苹果和甲骨文工作过)和布伦丹·麦科克尔(也是 Textaurant 的联合创始人)共同创立的。
