# 雅虎重新计票显示杨致远和博斯托克离被赶出董事会有多近

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/08/06/yahoo-vote-recount-shows-how-close-yang-and-bostock-were-to-being-ousted-from-the-board/amp/>

 <amp-img src="https://web.archive.org/web/20230204075905im_/https://techcrunch.com/wp-content/uploads/2008/08/bostock_yang2.png" alt="" title="bostock_yang2" class="alignleft size-medium wp-image-20808 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 雅虎对其股东投票的推迟显示了股东之间根深蒂固的愤怒，以及首席执行官杨致远和董事长罗伊·博斯托克多么接近被赶出董事会。在雅虎最大的股东之一资本研究和管理公司质疑最初的结果后，投票不得不重新计票。(计算错误是代理处理商 Broadridge Financial Solutions 的错——他们的可信度现在受到了质疑)。投给杨和博斯托克的反对票中有一半最初从未被计算在内。而不是最初报道的 15%和 21%的选票分别为杨和博斯托克保留，真正的“不”票是两倍:34%的杨和高达 40%的博斯托克。

这些数字非常接近将他们踢出董事会所需的水平。这也提出了一个问题，如果卡尔·伊坎决定不放弃全面代理权之争，会发生什么。虽然伊坎是否能够推翻整个董事会令人怀疑，但他可能已经成功罢免了雅虎董事长兼首席执行官。加上他自己和其他盟友，如约翰·保尔森和 t·布恩·皮肯斯(他最终亏本出售了自己的股份)，伊坎团队控制了至少 10%的选票。这足以摆脱博斯托克(取决于保尔森如何投票他的 4%的股份)。保持公众压力也可能赢得足够多的选票，让杨离开自己的董事会。

对杨公司来说，这种关系太密切了，以至于他们对自己在股东中的地位感到不安全，以防之前他们对投资者的真实想法有任何怀疑。根据重新计票结果，股东希望解雇的其他董事会成员包括罗纳德·伯克勒(38%的反对票)、阿瑟·科恩(32%的反对票)和加里·威尔逊(28%的反对票)。

总的来说，这是一半的董事会在打击距离内被取代。伊坎退缩是不是失算了？还是说，他现在手中的三个董事会席位比他本可以争取的五个席位更好？
