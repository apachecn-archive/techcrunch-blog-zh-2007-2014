# 突发事件:雅虎最终出售凯尔库

一段时间以来，有传言称雅虎将出售总部位于 T2 的比较购物网站 T4，现在看来他们已经找到了买家。雅虎在 2004 年以 4.75 亿€收购了 Kelkoo。

据了解该交易的消息人士透露，该公司已被出售给一家总部位于英国的私募股权公司 Jamplant Ltd(更新:[此处](https://web.archive.org/web/20230305215156/http://wck2.companieshouse.gov.uk/deca79b37c8de875f35dbabb464b8239/compdetails)是基金简介)，价格不到 1 亿€。

前 Kelkoo 首席执行官 Pierre Chappaz 在他的博客(法语)上宣布了这一消息，下面是宣布收购的内部电子邮件的副本。

自从收购雅虎以来，面对激烈的竞争，该公司已经失去了很多动力。
