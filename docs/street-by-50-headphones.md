# 街头 50 耳挂式耳机图片和细节

SMS 音频公司正准备推出 50 美分耳机线。顶层无线罐[泄露了](https://web.archive.org/web/20230203125819/https://techcrunch.com/2011/09/29/sync-by-50/)(最终导致其正式发布)，但该公司只是发布了有线版本的图片和技术信息。不过，价格和确切的发布数据仍未公布。

街上的 By 50 耳机看起来很像无线型号， [Sync By 50](https://web.archive.org/web/20230203125819/https://techcrunch.com/2011/09/29/sync-by-50/) 。它们都配有 40 毫米驱动器、记忆泡沫耳塞和可选麦克风。另外，它们是经过专业调音的——不管那是什么意思。有线版本似乎缺少 Sync 上的板载媒体播放控制。另外，不要对非常蓝的配色方案妄加评判。；它们也有黑色的。

该公司此前告诉我们，该计划是在未来几个月内发布整个系列，其中也包括耳塞，这可能意味着在节前推出。在那之前，短信音频似乎满足于向媒体透露信息。期待接下来的入耳式照片。
