糟糕的80年代电影的经典中有一个相当标准的蒙太奇。它包括主角们努力工作去建造/做/发明一些东西来击败那些高傲且不屑一顾的对手。例如， 夏季出租 ，这是一辆约翰·坎迪汽车，Candy和船员们将一家原本是船的海鲜餐厅改装成船，以赢得一场对抗势利小人的大型、富有的划船比赛。我不太记得他们为什么需要赢得比赛，但这无关紧要。最后[剧透警报]他们得意洋洋地对另一艘装备更丰富的船上的船员嗤之以鼻。这是一个失败者的故事——在那个黑暗的十年里要讲述的一个重要故事——在这里也适用。

这就把我们带到了HTC Hero T7，这是HTC第一款使用该公司新的Sense UI的Android手机。从某种意义上说，这位英雄“只是另一部安卓手机”；从另一个角度来说，这是HTC和这个平台的全新方向。

英雄是一个伟大的手机。它在许多方面与Palm Pre不相上下，甚至更胜一筹。它还把那些笨重的Windows Mobile和Symbian变成了你从年轻时就深情地记得的东西，一套现在已经灭绝的恐龙技术。

此外，我们可以很容易地扩展上面的比喻，说英雄是约翰·坎迪油漆甲板，而苹果和棕榈是富有的，高傲的游艇俱乐部成员嘲笑暴发户。我在这里告诉你，这些游艇俱乐部成员应该忽视这个暴发户，后果自负。

电话

HTC Hero相当纤薄。是的，除了下巴(我们稍后会谈到)和iPhone一样大。稍微厚一点，稍微短一点，但是大小差不多。
T13T15】

我们测试的模型覆盖着黑色和棕色的软橡胶。白人英雄是唯一一个被防污特氟隆覆盖的人。3.2英寸的屏幕本身在日常使用中保持惊人的清洁，这要归功于疏油屏幕。这款手机除了六个前置按钮(通话、主页、菜单、下巴上方的挂机和右下角的搜索和返回)之外毫无特色。)正面还有一个发光的小轨迹球。顶部扬声器上方有小型led，背面有一个500万像素的摄像头，没有闪光灯。顶部有一个全尺寸耳机插孔，底部有一个迷你USB。侧边控制音量的两个无标记按钮。

这款手机还有G1的“下巴”这是屏幕下的一个小突起，以30度角向外弯曲。HTC解释说，这更多的是一种设计选择，而不是功能选择。下巴定义了HTC的android系列，就像home键定义了iPhone一样:它是极简设计的一个刻意的优雅音符。我不认为这种设计会妨碍便携性或可用性，而且你在口袋里也不会注意到它。

总的来说，这个英雄小得惊人，而且相当有吸引力。这是一款设计精良的手机，没有粗糙的边缘，就像Pre一样，没有iPhone的脆弱性。你可以毫不犹豫地放弃这个想法，这是你想在iPhone上重新考虑的事情。

在手持电话和免提电话模式下，通话质量都很好。我的手机里有一个T-Mobile SIM卡，我发现它的接收效果总体来说很差。然而，这是T-Mobile在该地区覆盖范围内的一项功能，所以我现在无法对接收情况做出最终评估。这款手机运行在HSPA/WCDMA 900/2100MHz上，支持四频GSM/GPRS/EDGE。在这种模式下，3G在美国T-Mobile或美国电话电报公司无法使用。该设备还支持蓝牙和Wi-Fi b/g。

这款相机的标准分辨率为500万像素，具有自动对焦功能。我上传了一些照片到脸书。中间的两张照片是同一场景，拍摄于距离拍摄对象大约五英尺的地方。第一个一直放大，第二个缩小。

OS

安卓很乱。它就像一个好的Linux发行版——有时在视觉上令人印象深刻，而在技术上则一直令人印象深刻。但是，像一个好的Linux发行版一样，它对于外行用户来说是陌生的，这可能会导致它被认为是一个舶来品。我相信，Android最终会取代Windows Mobile和Symbian，成为大多数智能手机的首选操作系统。然而，这仍然是一项正在进行的工作。

距离我接触第一台Android设备T-Mobile G1已经过去了九个多月。如果你没有读到彼得·哈的评论，那么我在这里给你总结一下:G1和安卓在那个时候很糟糕。随着两个赛季的过去，我们发现这是一个比我们最初想象的更加强大和有用的操作系统。

安卓市场不是苹果应用商店的对手，但也不需要如此。其实比App Store好。你看，应用商店里的大多数应用都充满了娱乐价值，除此之外别无其他。如果你喜欢自娱自乐，这没什么，但如果你看看Android Market，你会发现它非常非常不同。

因为市场是开放和免费的，所以应用程序比实际需要的要多得多。虽然这导致了一些愚蠢的应用程序，但它让真正优秀的应用程序大放异彩。当你浏览市场时，你会发现到处都是奇怪的应用程序:这个ToggleWifi应用程序是什么？哦，它让我不用翻遍菜单就能打开/关闭无线网络？iPhone能做到吗？没有。谷歌语音？它就在那里，而且是免费的。找到我的iPhone？当然，安卓也有同样的功能。多亏了我的机器人在哪里，我可以给它发一条短信，它就开始鸣叫。需要从手机传输bittorrent应用程序吗？你可以用Transdroid做到。

我想你能明白我的意思。你可能会考虑Android Market自制软件，但这有什么错呢？这就是开放平台的伟大之处。总的来说，Android是一个很棒的平台。

UI

将用户界面从操作系统中分离出来是很困难的，但是让我试着评论一下HTC的自制用户界面，同时尽量避免Android给它带来的所有限制。

总的来说，Sense的用户界面和Palm Pre一样令人印象深刻，从纯功能的角度来看，比iPhone的默认界面要好。例如，Pre有页面，就像Hero一样，但是这里的页面实际上是“小部件容器”,可以在一个页面上保存多个数据点和控件。例如，你可以将七个页面中的一个专门用于娱乐应用，而其他页面可以专门用于电子邮件帐户。

【YouTube http://www.youtube.com/watch?v=0G2YibmW3DQ&hl=en&fs=1&]

该设备也有场景，这是对用户界面的一个重要补充。场景是为场景活动定义的“主题”。例如，工作主题有股票信息和世界时钟，而社交场景有突出的日历和社交网络特性。这是一个很棒的特性，可以解决操作系统对小部件和页面的一些限制。

[YouTube http://www.youtube.com/watch?v=6Tvi_x48dv8&hl=en&fs=1&]

HTC widgets与标准Android widgets的不同之处在于，它们是由HTC独家提供的，非常漂亮。例如，钟表就很吸引人。手表爱好者willy很高兴知道有些是向贝尔罗斯的飞行员风格手表致敬。

Sense UI还为您的联系人增加了社交网络意识。这意味着你可以实时观察你的朋友，无需思考就可以获取推文、更新和Flickr图片。例如，联系人应用程序有一个“更新”标签，允许您将联系人与他们的脸书个人资料联系起来。它还记录了你和那个人的所有交流，包括短信和电子邮件。最后，它显示那个人当前的Flickr流。这个系统并不是万无一失的，但它比Palm Pre要好。

应用程序内部的搜索系统相当不错，但在任何应用程序之外，它都默认为谷歌搜索。不太好。

屏幕顶部的通知栏向下滑动，显示最近的活动，包括电子邮件、推文和提醒。很高兴在一个地方看到所有这些，非常有用。所有的设置都很容易直接从页面上控制，包括Wi-Fi开关和蓝牙控制。

为了进一步扩展夏季出租的比喻，Sense是建立在已经充满活力的开发者社区平台上的WebOS。当Palm在研究那个该死的“丝带”和喝马提尼酒的时候，HTC的孩子们正在构建一个简单的覆盖层，模仿Palm Pre的所有优点。

例如，该浏览器在标准Webkit安装的基础上扩展，增加了智能重新格式化和缩放功能。浏览器通过放大字体来放大页面，而不是让页面变得更大，这样图像会很好地换行。有趣的是，Flash也在这个版本中工作。我可以浏览到的Flash欢迎页面，并在长时间等待后在那里加载一小部分代码。这确实是Flash，但还不够快。

一旦你掌握了窍门，屏幕键盘也非常有用。键盘自动完成大多数英语单词，并挑选出特殊的字符和数字，你按住字母键，直到交替键出现。大多数情况下，只有一个备用键——可能是冒号，也可能是加号——但是对于大多数信件来说，你可以找到几乎所有非正式通信所需的国际符号。

电子邮件设置很简单，你只需选择你的提供商或输入POP/IMAP设置。它可以支持多个帐户。它还支持Flickr、脸书和谷歌应用程序内容。

谷歌地图像预期的那样工作。它能精确定位你的位置，毫不费力地把你从一个地方带到另一个地方。

总的来说，UI是一个胜利。有那么好。他们让Android变得非常好用，这非常令人兴奋。

坏的

【YouTube http://www.youtube.com/watch?v=fbs5t4M4MEk&hl=en&fs=1&]

现在是坏消息:英雄部件引擎非常慢。它运行528MHz高通MSM7200A芯片，256MB内存，而iPhone 3GS运行600MHz，256MB内存。因此，这款设备没有理由更新小工具如此之慢。从一页滑到另一页已经够快了，但是一旦你到了那里，你会注意到明显的滞后。例如，当你滑动到电子邮件页面时，大约需要5秒钟才能看到最新的电子邮件。然后当你回到默认时钟页面时，你会注意到时钟停留在先前的时间——比如10分钟前——几秒钟后更新。看到这种滞后出现在设备的前端和中央令人沮丧。然而，一旦你深入挖掘，你会发现所有其他的应用程序或多或少都足够快。这只是一个症结，可能会挫败潜在用户。

底线

在《T4》的结尾，英雄们战胜了邪恶的游艇俱乐部成员，每个人都拥抱在一起。我不知道在这种情况下，道路是否清晰，但我认为HTC Hero是Android可用性新时代的曙光。Windows Mobile和Symbian现在应该动摇了，Palm高管最好计划在下一代WebOS手机中内置振动器/传送设备组合，因为他们能做的任何事情，已经表明，HTC可以做得更好。在我的书中，iPhone仍然是第一名，但这是第二名，取代了我长期以来对死党LX的爱。

英雄是精心设计的，有用的，强大的。操作系统和用户界面的结合几乎是完美的，手机运行感的未来是光明的。我的官方建议——如果有人在听的话——是让HTC放弃Windows Mobile开发，把他们的团队奉献给Android开发。他们几乎已经拥有了这个领域，其他制造商现在只是在赶时髦；有了适当的资源分配，他们可以在未来数年内引领潮流。

我对HTC一直持观望态度。我认为他们的智能手机是衍生产品，如果不是衍生产品，对他们来说也太“花哨”了。然而现在，他们用市场上最好的软件创造了一个完美平衡的智能手机。

据我回忆,夏季出租以Candy的乌合之众赢得比赛并驶向夕阳而告终。HTC要赢得这场比赛并不容易，但他们绝对领先。其他制造商现在也开始追随安卓的潮流；有了适当的资源分配，HTC可以在未来几年主导这一领域。这证明了Androids在移动生态系统中的价值。

这是一款面向大众的手机，但它仍然拥有Android的强大功能和极客风格。套用坎迪在电影中的角色，男主角是全家人的乐趣。
