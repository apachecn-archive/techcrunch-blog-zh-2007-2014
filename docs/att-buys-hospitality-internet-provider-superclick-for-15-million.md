# 美国电话电报公司以 1500 万美元收购酒店互联网提供商 Superclick

美国电话电报公司正在以价值约 1500 万美元的现金交易收购酒店业互联网提供商 [Superclick](https://web.archive.org/web/20230203102027/http://www.superclick.com/) 。

Superclick 为酒店、医疗保健和 MDU 市场提供网络设计、部署和基础设施管理。该公司服务于加拿大、北美、中东和亚洲的许多主要酒店品牌，如费尔蒙和莱佛士酒店、四季酒店、洲际酒店集团、卓美亚、金普顿酒店、文华东方酒店、万豪国际酒店、香格里拉酒店和喜达屋。

该公司的[收入并不多，](https://web.archive.org/web/20230203102027/http://www.marketwatch.com/story/superclick-reports-third-quarter-and-nine-month-financial-results-2011-09-15)但却在增加。上一季度的收入从本季度的 1，109，853 美元增长到 1，704，316 美元，增长了 53.6%，从今年迄今的 2，812，337 美元增长到 4，444，523 美元，增长了 58.0%，这在一定程度上得益于新业务。而 Superclick 是盈利的。

当然，与美国电话电报公司的[待决](https://web.archive.org/web/20230203102027/http://www.techmeme.com/110922/p18#a110922p18)[390 亿美元收购 T-Mobile](https://web.archive.org/web/20230203102027/https://techcrunch.com/2011/03/20/in-the-race-for-more-spectrum-att-is-acquiring-t-mobile-for-39-billion/)相比，这一收购简直微不足道。
