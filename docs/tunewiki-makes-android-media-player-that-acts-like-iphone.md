# TuneWiki 的 Android 版类似 iPhone 的媒体播放器

[![](img/e4d2a46fac9efced63be3015037a6fae.png)](https://web.archive.org/web/20221006201354/http://www.crunchbase.com/company/tunewiki)

谷歌 Android 平台上的软件霸权之争正在展开，开发商竞相开发最终将预装在品牌手机上的应用程序。今天[最受欢迎的 iPhone 第三方开发者之一 TuneWiki](https://web.archive.org/web/20221006201354/http://www.tunewiki.com/) 宣布了一款 Android 媒体播放器，该播放器将具有 iPhone 风格的界面，支持与 iTunes 同步，以及类似卡拉 ok 的歌词功能，使其闻名于世。

越狱 iPhone 的用户可能已经熟悉 TuneWiki 了——正如我们上周[提到的](https://web.archive.org/web/20221006201354/http://www.beta.techcrunch.com/2008/04/27/source-benchmark-invests-in-tunewiki-to-bring-song-lyrics-to-iphone/),自 2007 年 12 月以来，iPhone 版本已经有“数十万”的下载量。该软件的 iPhone 版本具有一个播放器，可以及时显示卡拉 ok 风格的歌词和设备库中的歌曲。TuneWiki 已与环球合作，合法提供歌词，并正在与其他音乐公司进行谈判。

该软件的 Android 版本具有相同的卡拉 ok 同步功能，并增加了对与歌词同步的音乐视频的支持。虽然这本身就很重要，但真正的新闻是软件附带的媒体播放器。从下面的视频中看到的简短演示来看，这款播放器似乎和 iPhone 上看到的一样好用(它看起来几乎完全一样，这不是一件坏事)。

Android 可能是一个“开放”平台，允许手机制造商(可能还有用户)安装他们想要的任何应用程序，但我们将看到每个领域都出现一些主导领导者。不会缺少模仿 iPhone 的应用程序，但 TuneWiki 通过建立一些难以复制的功能来帮助自己。这些是否足以让 TuneWiki 脱颖而出还有待观察。

您可以尝试该软件的仿真版本(减去视频中看到的一些导航功能)[这里](https://web.archive.org/web/20221006201354/http://android.tunewiki.com/)。
