# VMware 以 12.6 亿美元收购 Nicira，并提供了更多关于云战略的线索

VMware[收购了](https://web.archive.org/web/20230326024713/http://blogs.vmware.com/console/2012/07/vmware-and-nicira-advancing-the-software-defined-datacenter.html) [Nicira](https://web.archive.org/web/20230326024713/http://nicira.com/) ，这是一家以软件定义的网络技术而闻名的初创公司，在[的交易](https://web.archive.org/web/20230326024713/http://finance.yahoo.com/news/vmware-acquire-nicira-200100100.html)中价值 12.6 亿美元。这是一次收购，对市场上不断颠覆的企业领域有着各种各样的影响。

经过五年的开发，Nicira 于今年 2 月问世。它被广泛认为是传统网络技术的继承者，因为它能够创建一个软件定义的环境，该环境基本上可以通过软件中的调整进行编程来改变。这考虑到了灵活性和轻松移动数据的能力。通过在水平环境中横向扩展数据，它可以与水平存储环境很好地集成。Nicira 的软件实际上将网络变成了 IP 底板。物理网络中的层级会产生瓶颈。Nicira 创建了一个平面网络。设备直接连接到交换机，允许数据在其基础设施中有效分布。

Nicira 有 5000 万美元的资金和一份蓝筹股客户名单，其中包括 Rackspace、EBay 和 Fidelity。它的风险投资者包括 Andreessen Horowitz 和 Lightspeed Ventures，后者通过投资变革性的数据中心技术获得了巨大成功。VMware 联合创始人黛安·格林也是投资者之一。

Ben Horowitz 今天在接受我的采访时表示，VMware 已经购买了网络市场未来的期权。他说，在 Nicira 董事会上，考虑到公司还很年轻，他曾几次问自己是否要卖掉公司。他说，他们做出这一决定是考虑到成为 VMware 的一部分给了他们成为世界上最大的网络技术的巨大机会。他说，预计未来几年市场规模将达到 370 亿至 400 亿美元。

云计算对 VMware 的破坏不亚于任何传统技术公司。Nicira 有助于减轻 VMware 在制定全面战略以与 Amazon Web Services 和 OpenStack 竞争时面临的压力，open stack 正迅速成为构建云环境的可行替代方案。这也让 VMware 与思科等竞争对手正面交锋，而 VMware 与思科有着深厚的关系。

但这里有一定的讽刺意味。Nicira 是 OpenStack 的重要组成部分。OpenStack 的网络接口使用 Nicira API。这意味着 VMware 现在是 OpenStack 社区的代理成员，是网络市场长期计划的一部分。

OpenStack 是一种从不同云环境中汇集各种工作负载的联合方式。Rishidot Research 的 Krishnan Subramanian 表示，Cloudstack、Eucalyptus 和 OpenStack 将成为联合生态系统的基础。Nicira 的软件定义网络正好符合这一趋势。

Subramanian 说，未来，云将由大数据和服务环境来定义。VMware 认为这种转变是一项长期战略。

“你需要对网络进行编程，以满足你的需求，”他说。

VMware 首席技术官 Steve Herrod 表示，该公司相信未来是软件定义的网络。还需要进行编排。这就是它收购 DynamicOps 的原因。其自主开发的平台即服务 Cloud Foundry 也将在未来发挥作用。它提供了 Herrod 所说的“异构环境的应用语言”
