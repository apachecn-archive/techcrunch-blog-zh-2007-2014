# 迈克尔·戴尔在他臭名昭著的“我会关掉苹果”的名言上说:“我的答案在很大程度上被误解了。”TechCrunch

今天在 [Web 2.0 峰会](https://web.archive.org/web/20230327192315/http://www.web20summit.com/)的问答环节中，我问戴尔首席执行官[迈克尔·戴尔](https://web.archive.org/web/20230327192315/http://www.crunchbase.com/person/michael-dell)他是否后悔[说](https://web.archive.org/web/20230327192315/http://www.zdnet.com/blog/apple/aapl-tops-300-still-waiting-on-michael-dells-apology/8439)苹果应该关闭公司并把股票卖回给股东。

14 年前，当苹果公司的股价为每股 17 美元时，戴尔发表了臭名昭著的声明，我想知道他是否会修改他的话，因为苹果公司现在的股价为每股 420 美元，市值约为 4000 亿美元。

这个问题引起了观众的笑声，但戴尔说，他发现这是一个“很好的机会”来澄清他所说的话。他的回应？"我的回答被误解了。"

戴尔说，他最初在 1997 年拒绝回答最初的问题，在被要求回答两次后，他基本上回答了“我会怎么做？我会关闭它，把钱还给股东，”因为他无法想象自己成为任何其他公司的首席执行官。

戴尔继续说，“我回答的意思是，我是戴尔的首席执行官，我不考虑成为任何其他公司的首席执行官，我不是受雇的首席执行官，所以如果你问我会为任何其他公司做什么，这真的不是我考虑的事情。”

戴尔接着用忧郁的语气说，“我非常尊重苹果。显然，史蒂夫会被怀念，他是我的朋友。”他描述了自己 16 岁时在休斯顿的一个苹果用户群中遇到苹果联合创始人史蒂夫·乔布斯的情景，当时戴尔带着他的 Apple II 出席了会议。

当被问及其他公司，比如麦当劳，他是否会给出同样的答案时，戴尔说，“我是迈克尔·戴尔，戴尔的创始人，我会问:‘你为什么不给我另一个问题？’。“很公平。

“我不会为[戴尔]股票自 1987 年以来上涨了 16，000%而道歉，”他在结束讲话时强调道。
