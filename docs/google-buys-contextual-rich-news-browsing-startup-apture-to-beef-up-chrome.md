# 谷歌收购上下文丰富的新闻浏览初创公司 Apture 以增强 Chrome 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/11/10/google-buys-contextual-rich-news-browsing-startup-apture-to-beef-up-chrome/>

我们已经与两家公司确认，谷歌已经收购了一家为网络内容提供即时搜索的初创公司。交易的财务条款没有披露。[Apture 于 2008 年推出](https://web.archive.org/web/20230405022636/https://techcrunch.com/2008/06/30/apture-gives-contextual-popups-a-good-name/)，已经从博·弗罗利克、保罗·马里茨、史蒂夫·泰勒、Clearstone Venture Partners 筹集了 460 万美元

由[特里斯坦·哈里斯](https://web.archive.org/web/20230405022636/http://www.crunchbase.com/person/tristan-harris)和 [Can Sar](https://web.archive.org/web/20230405022636/http://www.crunchbase.com/person/can-sar) 共同创立的 Apture 开发了“Apture Highlights”，这是一个浏览器扩展，旨在堵塞“搜索漏洞”，该公司称这种漏洞正发生在网络内容上。Apture 强调了一个搜索漏洞，当用户在阅读内容时，想要更多关于关键词或短语的信息，然后打开另一个浏览器标签在谷歌、必应或雅虎上搜索信息。对于内容发布者来说，不利的一面是用户通过离开页面来脱离实际内容。

Apture Highlights 允许您突出显示页面上的任何单词或短语，并立即在窗口中显示搜索结果。这家初创公司带来了来自 60 多个来源的结果，包括 YouTube、Twitter、维基百科、谷歌和更多围绕内容的额外上下文。浏览器插件可用于 Chrome、Firefox 和苹果的 Safari 浏览器。Harris 说，事实上，每月有超过 10 亿的页面通过 Apture 得到了增强。

Apture 的商业模式还为出版商提供了一个白色标签版本的 [Apture Highlights，](https://web.archive.org/web/20230405022636/https://techcrunch.com/2010/08/24/apture-now-brings-instantaneous-search-to-any-web-page/)Scribd、金融时报、路透社、Economist.com、科学美国人、BleacherReport.com 和印度时报都在使用这个版本。Publisher 只需插入一行 Javascript 代码，读者就可以访问一个基于 HTML 的覆盖图，该覆盖图就像一个小型浏览器，使读者无需离开原始页面就可以找到并浏览相关的多媒体内容。

对出版商来说，使用这项技术的好处是，内容网站通过每次搜索增加相关文章的流量，从而增加了搜索量和网站页面浏览量。消费者实际上在有 Apture 的出版商页面上停留的时间是没有插件时的两到三倍。

该公司最近还推出了“热点”,“T2 ”,它允许 Apture 在网络上的任何页面上动态填充超链接。出版商必须添加一行代码，超链接将出现在读者面前，而出版商不必链接。这些链接指向一系列覆盖图，读者无需离开页面就能获得一系列数据:维基百科、CrunchBase、搜索结果、地图等。

在今天的采访中，哈里斯解释说，当他在 2007 年创办公司时，他觉得在网络媒体中有一个难以置信的机会，可以找到上下文信息来帮助你理解你在网页上看到的内容。哈里斯希望 Apture 成为“网络结构”的一部分。

但是在过去的几年里，哈里斯说他意识到浏览者本身是媒体的一部分，它将决定信息是如何被真正获取的。“如果不谈论网络浏览器，你就不能谈论信息的可访问性，”他解释道。

考虑到这一点，当谷歌在过去几个月向哈里斯和该公司提出收购要约时，整合是有意义的。结合谷歌自己让信息无障碍的使命，“Chrome 平台每月有 2 亿活跃用户，这是 Apture 的技术可以达到的巨大规模，”哈里斯说。

谷歌谈到此次收购时说，“我们对 Apture 团队增强网络浏览器体验的方法印象深刻，我们认为他们的专业知识将补充 Chrome 团队在这一领域的努力。”

由 10 名员工组成的 Apture 团队将加入谷歌 Chrome 团队，改善用户体验。谷歌表示，Apture 的插件和白标技术将在下个月左右关闭。

![](img/793e3bfb9802a7e9ec94cbea32f30a1c.png) Harris 说，“我们正在为 Chrome 团队带来我们一直在做的所有事情以及更多。随着 Chrome 浏览器的发展，很明显为什么这次收购如此有意义，在互联网的变革中，你还想和谁合作？我们将使浏览比以往多得多，这真的很令人兴奋。”

对谷歌来说，这也是一次人才收购。哈里斯之前在苹果公司的用户界面工程领域工作，拥有两项正在申请的专利。他帮助建立了 Wikia 的第一个广告服务器，也是斯坦福科技风险投资项目的梅菲尔德研究员，并从斯坦福计算机科学硕士项目退学来创建 Apture。Sar 还在斯坦福大学计算机科学博士项目中休假，并在操作系统和安全性方面发表了几篇学术论文，并且是几项正在申请的专利的作者。

总的来说，这次收购对谷歌来说是一次胜利。谷歌或脸书这样的网络巨头收购这家初创公司只是时间问题，增加一些有才华的开发人员和工程师也无妨。
