# Tweetvite:一个致力于计划和寻找 Tweetups 的活动网站

[![](img/008d190a67f68aca7ff60e6325dfdaef.png)](https://web.archive.org/web/20221006094126/http://www.tweetvite.com/) 一年多前，我们看到[推出了](https://web.archive.org/web/20221006094126/http://www.beta.techcrunch.com/2008/07/01/y-combinators-anyvite-takes-on-evite-keeps-it-simple/) [Anyvite](https://web.archive.org/web/20221006094126/http://www.anyvite.com/) 的，这是一家由 [Y Combinator](https://web.archive.org/web/20221006094126/http://www.ycombinator.com/) 资助的竞争对手 [Evite](https://web.archive.org/web/20221006094126/http://www.evite.com/) ，它希望简化活动规划。今晚，这家初创公司将推出一个名为 [Tweetvite](https://web.archive.org/web/20221006094126/http://www.tweetvite.com/) 的衍生网站——一个致力于帮助计划和发现推文的网站。

对于那些以前没有接触过这个术语的人来说，Tweetup 是使用 Twitter 的人在现实生活中的聚会。除此之外，规则是灵活的:推文可以是大型活动或小型聚会，可以是喝几杯或只是社交一下，可以是事先计划好的，也可以是自发的。创始人杰夫·莫林说，虽然有很多网站迎合传统活动，如生日和烧烤，但 Tweetup 利基服务不足。

任何使用过 Anyvite 或 Evite 等活动网站的人都很熟悉用 Tweetvite 设置活动。首先，您需要输入活动的名称、地点、主办者以及其他基本信息。但是这个网站包含了一些你在其他地方找不到的属性:它要求你为这个事件指定一个标签，以及一个定制的简短 URL。该网站还可以方便地发布你的活动，或者与脸书和 MySpace 等其他服务分享。与传统活动网站的另一个巨大区别是，Tweetvite 提供了一个即将发布的推文目录(鉴于它们被宣布的性质，它们通常对公众开放)。

一旦你创建了你的活动，你可以使用网站的控制面板来监控任何包含你的标签的推文，看看有多少人浏览了你的页面并回复了。该网站还提供了一个小部件，你可以将它嵌入到你的博客中，通知访问者你即将发布的推文。

![](img/43b74c3905b0d7db61c3a645693793f7.png)

Tweetvite 看起来很棒，有一个非常精致的界面和一个简化的事件创建过程，只需要一两分钟。在这一点上，我心中最大的问题是有多少人真的会发推文——它们可能越来越受欢迎，但远不如传统活动那样普遍，因此围绕这个利基市场建立业务可能很难。也就是说，Twitter 显然仍处于起步阶段，因此 Tweetups 的数量可能会在未来几年快速增长。
