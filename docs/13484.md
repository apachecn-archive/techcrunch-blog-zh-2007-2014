# 谷歌收购 iOS/Mac 电子邮件客户端 Sparrow，应用程序仍然可用，但开发已停止

谷歌刚刚收购了广受欢迎的 iOS 和 OS X 电子邮件客户端背后的公司 Sparrow。麻雀公司的首席执行官多姆·勒卡说，该公司的团队将加入谷歌的 Gmail 团队，“以实现一个更大的愿景——我们认为我们可以通过谷歌更好地实现这个愿景。”2011 年，Sparrow 从法国风险投资公司基马风险投资公司筹集了一小笔[种子资金，但此后没有接受任何其他外部投资。谷歌和 Sparrow 都没有透露收购的财务细节。](https://web.archive.org/web/20230315095317/https://techcrunch.com/2011/02/09/sparrow-mac-mail-app/)

Sparrow 表示，目前其应用程序将继续可用，但根据该公司刚刚发送给其用户 Sparrow 团队的电子邮件，我们不会很快看到应用程序的任何更新:“”然而，由于我们将忙于谷歌的新项目，我们不计划为 Sparrow 应用程序发布新功能。然而，Sparrow 将继续“为[它的]用户提供支持和关键更新”。"

## “我们不打算为麻雀应用发布新功能。”

Sparrow 的一些工作可能会流入谷歌的移动客户端。谷歌自己的 Gmail 客户端显然有一个艰难的开始，而 Sparrow 很快在 Mac 和 iOS 上获得了[的忠实追随者](https://web.archive.org/web/20230315095317/https://techcrunch.com/2011/03/22/new-sparrow/)。Sparrow 的 OS X 应用于 2011 年 10 月推出，其 iOS 应用于 2012 年 3 月推出。

谷歌计划支持 Sparrow 的 Mac 客户端多长时间仍有待观察。到目前为止，谷歌没有为任何操作系统提供自己的原生电子邮件客户端，鉴于谷歌对网络的关注，该公司似乎不太可能继续资助桌面电子邮件客户端的开发。目前还不清楚谷歌是否会继续对 Sparrow 的应用收费。Mac 应用程序目前售价 9.99 美元，iOS 客户端售价 2.99 美元。
