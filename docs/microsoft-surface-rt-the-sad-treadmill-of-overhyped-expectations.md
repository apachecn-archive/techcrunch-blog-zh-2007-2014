# 微软 Surface RT:过高期望的可悲跑步机

虽然我不愿意在 Windows Surface(这里指的是 RT 版)上指出像 Marco Arment 和 Gruber 这样的苹果书呆子，但我认为重要的是我们都后退一步，评估一下到底发生了什么:微软匆忙向市场推出了一款充其量只能说是粗制滥造的产品。

明确地说，我认为外形是计算的未来。[我同意杰夫·阿特伍德](https://web.archive.org/web/20221208074818/http://www.codinghorror.com/blog/2012/11/do-you-wanna-touch.html)的观点，Surface 振兴了奄奄一息的笔记本电脑行业，为更强大的平板电脑添加键盘是前进的方向。

我使用过运行 Win 8 的设备[，尽管 Metro(我称之为 Metro，因为它听起来更酷)和桌面之间存在明显的脱节，但我认为 Win 8 是一个可靠的操作系统，是一个惊人的进步。普遍接受将是缓慢的，但价值主张，特别是在统一 Win8/WinPhone 世界方面，是存在的。](https://web.archive.org/web/20221208074818/https://beta.techcrunch.com/2012/02/29/windows-8-the-road-ahead/)

但是我们开始了:微软以他们笨拙的方式，首先推出了他们最差的产品。顽固派正大张旗鼓地抗议 RT 受到的不公平待遇，这些批评者显然是站在苹果一边的(比如我们自己的马特·伯恩斯，他在职业生涯的大部分时间里都在使用 Windows)。就连马尔科也证实，硬件很漂亮，但处理器太慢，操作系统太不可靠，不足以成为微软今年剩余时间的帐篷杆。当克里斯·皮里略(有点)反对你的时候，谁能支持你？

微软有一个不幸的倾向，在上市期间做两件事。首先，他们过度打字和过度分享，制造了一堆兴奋的篝火，马上就熄灭了。然后他们发布由致力于自己事业的人组成的委员会设计的东西。兼容性团队想要台式机(阿特伍德反驳道:“是的，是的，它不运行 x86 应用程序。因此，您心爱的 Windows vista Designer 1998 将无法在 Surface RT 上运行。您需要等待几个月的时间，Surface Pro 才能完成这一操作，但您需要在价格、电池寿命和尺寸方面为英特尔 Premium 付出代价。”)用户界面恶魔想要地铁的前端和中心。营销人员想要一台 iPad。Office 人员希望与 Xenix 的多工具 Word 向后兼容。然后他们都坐下来做与最终目标无关的工作。例如，与 Win8 同时发布的《Windows Phone 评论指南》大约有 275 页。那是一个团队夏天的大部分时间。也许那些工时会更有利于让真正的 Surface 以 Surface RT 的价格上市？

简而言之，微软知道个人电脑行业充其量是不温不火的。除了金融计算和游戏领域的军备竞赛之外，没有什么能迫使普通用户升级。Win8 将履行大部分的内疚，但推出一个产品，这并不表明它的优势是一个大问题。瞧，从波格到伯恩斯，每个人都会说 Surface 很棒，但事实并非如此，他们会被吓退至少两代硬件。然后[粉丝们会点燃这样的帖子](https://web.archive.org/web/20221208074818/http://www.reddit.com/r/surface)并试图证明他们的购买是合理的，直到他们悄悄地承认，这不是他们想要的非机器人。

可以说，最初的苹果手机也是如此。它看起来很奇怪，没有应用程序，而且通常动力不足。然而，作为第一代产品，它定义了前进的方向。另一方面，Surface RT 定义了 Win8 平台的底端，每个乡村歌手都知道，你不会以一首悲伤的歌开始你的场景……你会以一个谷仓燃烧器开始。
