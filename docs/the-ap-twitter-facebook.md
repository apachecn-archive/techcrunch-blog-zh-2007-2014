# 美联社通过推特将人们送往脸书。等等。什么？

![](img/f13b77105c9cb77b305b0f361d258677.png "apface")哦美联社，我们最喜欢的[被禁的](https://web.archive.org/web/20221207111020/https://beta.techcrunch.com/2008/06/16/heres-our-new-policy-on-ap-stories-theyre-banned/)新闻来源。似乎几乎每个月他们都会做一些[违背](https://web.archive.org/web/20221207111020/https://beta.techcrunch.com/2009/04/06/behind-the-aps-plan-to-become-the-webs-news-cop/) [逻辑](https://web.archive.org/web/20221207111020/https://beta.techcrunch.com/2009/04/08/ap-exec-doesnt-know-it-has-a-youtube-channel-threatens-affiliate-for-embedding-videos/)和/或看起来是[自杀性](https://web.archive.org/web/20221207111020/https://beta.techcrunch.com/2009/02/22/hot-news-the-ap-is-living-in-the-last-century/)行为的事情。今天带来了另一件怪事。

美联社使用他们的推特来发布他们的故事——这显然不是什么新鲜事——但是他们每一个人都在他们的脸书笔记页面上链接到这个故事。不清楚他们这样做有多长时间了，但是[搜索引擎公司的](https://web.archive.org/web/20221207111020/http://searchengineland.com/)丹尼·沙利文[今晚注意到了](https://web.archive.org/web/20221207111020/http://twitter.com/dannysullivan/status/9408856266)这件事的奇怪之处，以及它有多烦人。美联社显然有大量的媒体合作伙伴，他们可以很容易地链接到其中任何一个，甚至是他们自己网站上的报道。但是没有，相反，他们把所有这些故事复制到他们的脸书页面上，然后无缘无故地链接到那里。

正如沙利文在随后的推特中指出的那样，“我真的很怀念人们拥有自己的网站并对其指指点点的时候。为什么要把你的灵魂出租给 facebook？或者巴斯。或者别的什么。掌握你的领域。”

真正奇怪的是，美联社最近与谷歌在托管美联社内容上发生了冲突。本月早些时候，双方似乎达成了某种协议(经过几个月的威胁和实际的[撤下内容](https://web.archive.org/web/20221207111020/https://beta.techcrunch.com/2010/01/11/google-news-pulls-ap/)，但是现在美联社只是为了好玩才在脸书上托管这些内容？

当然，也许他们认为通过在脸书上托管内容，他们在推特上是[公正的](https://web.archive.org/web/20221207111020/http://twitter.com/tolles/status/9409372885)。但是，为什么不使用他们自己的网站呢？

当我请沙利文详细说明这个问题时，他提出了一个很好的观点，“*有趣的是，他们似乎更喜欢社交(twitter & facebook)而不是基本的搜索引擎优化(他们与谷歌问题的核心)*。”哦，美联社；娱乐永远不会结束。
