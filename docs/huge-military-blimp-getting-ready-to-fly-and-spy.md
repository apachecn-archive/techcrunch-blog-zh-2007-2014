# 巨大的军用飞艇准备飞行和侦察

你可以说我缺乏想象力，但我一直认为软式飞艇最适合体育赛事和另类历史小说。如果这个珍珠般洁白的庞然大物是某种迹象的话，美国军方恐怕不敢苟同。女士们先生们，来见见蓝魔 2 号(T1)，这是一个浮动的监视平台，可能很快就会出现在世界各地饱受战争摧残的战区。

这些照片由《连线》杂志的[危险室](https://web.archive.org/web/20230205025519/http://www.wired.com/dangerroom/2011/10/giant-blimp-dwarfs-truck/)提供，乍一看，这艘飞艇名不副实。它既不是蓝色的，也不是特别危险，但蓝魔是彻头彻尾的巨大:它从头到尾有 370 英尺长，几乎是固特异飞艇的两倍。

这条鲸鱼将于明年年中携带一系列广域摄像机和窃听设备进入风景如画的阿富汗。一旦它到达目的地，蓝魔将悬挂在 20，000 英尺的空中并开始侦察。一台机载超级计算机将处理所有的信息，然后这些信息将被传递给我们下面的友军。

正如大多数军事项目一样,“蓝魔”已经酝酿了很长时间。有趣的是，危险室指出，这是两个飞艇相关的军事项目中的一个，这可能比任何平民预期的都多两个。
