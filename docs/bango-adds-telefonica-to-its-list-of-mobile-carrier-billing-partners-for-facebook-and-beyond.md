# Bango 将 Telefonica 加入其脸书及其他地区的移动运营商计费合作伙伴名单 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/01/16/bango-adds-telefonica-to-its-list-of-mobile-carrier-billing-partners-for-facebook-and-beyond/>

移动支付公司[Bango](https://web.archive.org/web/20221207110309/http://www.bango.com/)——脸书、亚马逊、黑莓和其他公司的合作伙伴，使应用程序和内容费用直接计入你的手机账单——已经[宣布](https://web.archive.org/web/20221207110309/http://www.iii.co.uk/investment/detail?code=cotn:BGO.L&display=news)另一个主要的运营商合作伙伴: [Telefonica](https://web.archive.org/web/20221207110309/http://www.telefonica.com/) 正在通过其支付 API 实现 Bango 支付。

这项全球协议(RNS 声明嵌入下文)将意味着，为 Telefonica 的 3.14 亿用户开发应用程序的开发者将能够将单一 API 集成到他们的应用程序和移动网站中，从而在 Bango 网络的应用程序商店中实现双击运营商支付。其中包括脸书的应用中心、Google Play、黑莓应用世界、亚马逊的应用商店、Windows Phone 商店、Opera 的应用商店，以及 Bango 表示目前正在保密的即将到来的应用店面。(我的猜测是:Firefox Mobile 的一个应用商店，因为 Telefonica 是该合资企业的合作伙伴，但谁知道呢——也许是苹果最终转向了运营商计费……)

这笔交易建立在 2012 年 7 月[传来的消息](https://web.archive.org/web/20221207110309/http://pressoffice.telefonica.com/jsp/base.jsp?contenido=/jsp/notasdeprensa/notadetalle.jsp&pagina=3&selectNumReg=5&id=66&origen=resnotap&idm=eng&pais=1&elem=18407&mesinicio=07&anioinicio=2012&mesfin=08&aniofin=2012&tlibre=)上，西班牙电信已经与谷歌、脸书、微软和 RIM 签署了全球协议，对其应用商店的内容收费。与此同时，Bango 的其他运营商客户中也包括法国电信。

这项新交易的财务条款没有披露，其收入分享/佣金细节也没有披露，西班牙电信表示，这将因国家和商店而异。

Bango 并不是 Telefonica 的第一个运营商计费合作伙伴。它还与博库合作，博库去年获得了由该运营商牵头的 3500 万美元的战略投资。西班牙电信数字公司 BlueVia 的负责人何塞·巴雷斯说，与其中一家合作并不能抵消另一家。

“这不是独家，”他在接受 TechCrunch 采访时说。“这是为了增强生态系统。不是所有的公司都与 Bango 合作，也不是所有的公司都与 Boku 合作，所以我们将与他们两个合作。”

这也不是两人第一次合作了。Bango 在大约 10 年前(没有使用 API 与第三方网站互联)首次向 Telefonica 提供运营商计费解决方案。

虽然 Telefonica 已经通过自己的支付 API 提供运营商计费有一段时间了，但这里的不同之处在于 Bango 已经与多家应用商店提供商达成了协议。这使得它的单个 API 功能更强大，实现起来更简单。

Telefonica 的开发者平台 Blue Via 的营销主管马特·迪克斯(Matt Dicks)表示:“这让我们专注于应用商店和无摩擦、低点击支付。“这不是电信公司擅长的领域。”Blue Via 正在将 Bango 的 API 与其他链接到 Telefonica 服务的 API 进行集成，如消息传递、VoIP 等。

Bango 表示，通过运营商计费实现的应用和其他内容的转化率为 77%，相比之下，需要信用卡或其他支付信息的内容的转化率为 40%。但要让运营商计费无处不在并得到更多使用，还有很多工作要做。

Bango 战略合作伙伴副总裁 Anil Malhotra 承认:“大型互联网公司仍然没有充分利用运营商计费的机会。从这个意义上说，这些交易对 Bango 来说是双向的。它积累的运营商合作伙伴越多，Twitter、任天堂和索尼等公司就越有可能将计费整合到他们的服务中——如果有这样的需求的话。

Telefonica 推出的另一项举措也推动了这一趋势——将更多运营商整合到其 API 平台上。第一个是 [Telenor](https://web.archive.org/web/20221207110309/https://beta.techcrunch.com/2012/10/10/dont-call-it-wac-telefonica-adds-telenor-to-its-bluevia-api-platform-carrier-billing-now-covers-460m-customers-worldwide/) ，更多的预计将在适当的时候宣布。

“BlueVia 是 Telefonica 的一项大胆而有价值的举措，旨在建立一套统一的计费 API。Bango 首席执行官 Ray Anderson 在一份声明中表示:“我们将在 BlueVia 的基础上实现标准化，与 Telefonica 在全球的 3.14 亿用户建立联系，并期待着欢迎其他运营商加入这一倡议，正如 Telenor 所做的那样。”。

虽然运营商计费的机会适用于任何市场，但这笔交易的一个主要目标是针对 Telefonica 业务范围内的新兴市场，尤其是拉丁美洲。

在萨尔瓦多、委内瑞拉和巴西等国家，信用卡和借记卡的普及率相对较低，因此 Telefonica、开发者和应用生态系统中的其他人需要其他支付途径。这就是运营商计费可能发挥作用的地方，作为“下十亿”智能手机用户在他们的设备上购买移动内容的大规模基础设施。

> **Bango 和西班牙电信宣布全球移动支付合作**
> 
> 剑桥和马德里，2013 年 1 月 17 日。移动支付和分析公司 Bango (AIM: BGO)和 Telefónica Digital 今天宣布，他们已经签署了一项全球框架协议。两家公司将在全球范围内合作，为移动应用商店创造更好的直接支付体验。此次合作将 Bango 的无摩擦支付体验与 Telefónica 的 BlueVia 支付 API 相结合，将全球超过 3.14 亿付费客户连接到 Bango 支付平台。
> 
> 通过移动电话账单或预付信用支付数字商品和服务的能力是内容所有者和开发者将其产品完全货币化的一种关键方式。在发展中市场尤其如此，如拉丁美洲，那里的银行账户和信用卡渗透率很低。Telefónica 运营业务中的直接付款试验证明了其推动销售的能力。在 Bango 向信用卡普及率较高的发达市场引入运营商计费的地方，数字产品的销售大幅增长。
> 
> 通过将单一的 BlueVia 计费 API 集成到 Bango 支付平台，应用商店将受益于 Bango 增强的移动设备用户体验，这将产生更高的支付转换率，特别是从 Wi-Fi 连接和其他“离网”设备。这对于移动支付的未来尤为重要，因为超过一半的智能手机浏览应用商店时使用 Wi-Fi 连接。
> 
> 合作的一个关键目标是加快所有应用商店和内容提供商的标准化和开放支付平台的可用性，并大幅改善客户体验。该平台将对所有应用商店开放，通过一个单一的通用平台支持运营商计费和其他支付方式。对客户的好处是无缝的支付流程，不需要输入个人信息或离开支付会话。
> 
> 除了改善用户体验的技术，Bango 还为这一合作伙伴关系贡献了其在大规模支付管理方面市场领先的专业知识，这些专业知识是与 RIM、脸书、Opera 等行业领先的合作伙伴共同开发的。Bango 平台通过自动化所有结算流程，包括税务对账、本地货币支持和提供复杂的分析和报告，使应用商店能够优化移动用户体验，从而扩大了运营商计费对第三方的吸引力。
> 
> 西班牙电信正在其运营业务中推出直接计费功能，并在今年早些时候宣布了与脸书、谷歌、微软和 RIM 的全球框架协议。
> 
> Bango 首席执行官 Ray Anderson 评论说，“Telefonica 和 Bango 有着共同的战略愿景:通过标准化和简化运营商付费的移动支付来扩大付费内容的使用范围。BlueVia 是 Telefonica 的一项大胆而有价值的举措，旨在建立一套统一的计费 API。我们将在 BlueVia 的基础上实现标准化，与西班牙电信在全球的 3.14 亿用户建立联系，并期待着欢迎其他运营商加入这一举措，就像西班牙电信已经做的那样”。
> 
> BlueVia 负责人何塞·巴雷斯代表 Telefónica Digital 表示:“每一项卓越服务的核心都是一流的客户体验，Bango 和 BlueVia 都有一个共同的愿景，即移动支付必须是无缝的，对客户来说是低摩擦的。通过这种合作关系，我们正在提供一个移动计费生态系统，使应用商店和内容所有者能够通过全球范围和远比信用卡更广泛的支付方式实现大规模业务”。
> 
> Telefónica 和 Bango 之间的伙伴关系是由 Telefónica 的金融服务和全球伙伴关系团队谈判达成的，这两个团队分别位于马德里和硅谷，由 Joaquín Mata 和 Wayne Thorsen 领导。
