# Glam Media 推出 BrashSports，瞄准男性观众

![](img/a83d931b6c40a1fd524e0145dc6b9329.png)

排名第一的女性媒体网络 Glam Media 正在锁定另一个目标:男性。今天早上，该公司推出了 [BrashSports](https://web.archive.org/web/20221207110827/http://sports.brash.com/articles/latest/) ，这是一个新的内容频道，涵盖了超过 25 个与体育相关的网站，包括 SportsFanLive 和 Bloguin。

这一发布紧随 Glam 上周收购体育媒体广告网络 [Sportgenic](https://web.archive.org/web/20221207110827/http://www.sportgenic.com/) 之后。这笔交易是为了 Sportgenic 的广告平台 AdPortal，Glam 正在将其整合到自己的 GlamAdapt 平台中，以更好地与谷歌的 DoubleClick 竞争。

Glam 历来提供主要迎合女性的酒店(在这方面做得很好)，但自 2008 年末以来，它也通过其傲慢的网络瞄准了男性。今天的发布极大地拓展了这一网络。

Glam 表示，随着体育属性的加入，其男子 newtork 现在在美国每月的独立访客人数超过 3000 万。Glam 说，这也使他们一跃成为美国第三大最受欢迎的男性网站，仅次于雅虎！体育网站(4200 万唯一)和 ESPN(3000 万)。
