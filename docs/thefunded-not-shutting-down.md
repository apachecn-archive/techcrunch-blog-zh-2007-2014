# 未关闭的融资 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/03/30/thefunded-not-shutting-down/>

我们总是很乐意给那些搞愚人节恶作剧的人留有余地，但即便如此，还是把它推到了 3 月 27 日。如果你去得太早，你需要表现得非常有趣，否则我们会不屑一顾，直到我们完全忘记这件事。

[受资助的](https://web.archive.org/web/20221207201515/http://www.thefunded.com/)的 Adeo Ressi 现在证实他上周张贴的关闭通知是愚人节玩笑的一部分。他说，他无法发表评论，因为他“在遥远的犹他州参加婚礼”我想，婚礼总是离线的借口，所以我们原谅了被资助者对情况的隐瞒。犹他州的婚礼一定很迷人。但至于笑话本身是否好笑，那就要由受资助的用户来决定了。

我们喜欢去年的愚人节玩笑，其中包括推出一个新网站，让风险投资家[审查投资他们的创始人](https://web.archive.org/web/20221207201515/http://www.beta.techcrunch.com/2008/03/31/venture-capitalists-fight-back-against-thefunded/)(来自首轮资本)。

> 这条消息是愚人节的一个周末恶作剧，被成员泄露给媒体，称该网站“在诉讼后”被关闭没有比平常更多的威胁，也没有提到诉讼…网站没有被关闭。
> 
> 事实上，恶作剧通知说，我们将结束审查，因为我们认为所有的投资者都很棒。我们选择周末是因为客流量少，我们认为人们会嘲笑所有投资者都很棒的说法。相反，人们只是做了最坏的打算。
> 
> 具有讽刺意味的是，我在遥远的犹他州参加婚礼，所以我不能发送电子邮件或更新网站…更新即将到来，明年我需要一个更好的恶作剧…有什么想法吗？
