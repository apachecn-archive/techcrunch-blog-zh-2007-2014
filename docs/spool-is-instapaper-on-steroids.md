# Spool 是 Instapaper 的兴奋剂 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/12/spool-is-instapaper-on-steroids/>

[Spool](https://web.archive.org/web/20230205040442/https://getspool.com/) 是一项新服务，旨在解决我们现在生活的多设备、多屏环境所带来的问题，在这种环境中，内容消费体验会因平台而异。例如，在 iOS 设备上，如果没有认真的变通办法，你就无法观看 Flash 视频。在 iPod Touch 或其他独立媒体播放器上，您需要 Wi-Fi 信号才能浏览网页。

但是有了[卷轴](https://web.archive.org/web/20230205040442/https://getspool.com/)，你就不用考虑这些事情了。任何互联网内容，包括音频、视频和文本，都可以通过 Spool 应用程序、浏览器插件或 bookmarklet 在移动设备上即时离线观看。因为 Spool 是智能的，它知道保存网页的哪一部分，丢弃哪一部分。

简而言之，Spool 的工作方式就像是进化版的 [Instapaper](https://web.archive.org/web/20230205040442/http://www.instapaper.com/) ，这是一种流行的服务，可以保存长篇网络文章，供以后在你的电脑、iPhone、iPad 或 Kindle 上阅读。像 Instapaper 一样，也有这样一种观点，即在网络上有效的不一定在移动设备上也有效。但是 Instapaper 清理和重新格式化文本以便于阅读，Spool 适用于任何媒体类型，无论是文本、音频还是视频。它甚至可以为您解析多页内容，例如，保存整篇文章或论坛主题，而不仅仅是第一页。

[![](img/2c87d66583600ce8467e6587203bd2d3.png "Spool_Flow")](https://web.archive.org/web/20230205040442/https://techcrunch.com/wp-content/uploads/2011/09/spool_flow.png)

该服务使用人工智能和计算机视觉引擎来像人类一样阅读网页，并提取相关部分，而丢弃其余部分(如广告、页眉、页脚等)。).或许最重要的是，它将视频转换成适合移动设备、基于 HTML5 的格式，可以在任何现代智能手机或平板电脑浏览器上播放。视频和其他内容也会缓存到设备中，以供离线访问。

从短期来看， [Spool](https://web.archive.org/web/20230205040442/https://getspool.com/) 解决了内容不兼容的问题，这一问题在很大程度上是由于苹果决定禁止 Flash 出现在移动设备上，以及出版商推迟转向 iOS 友好的网络标准 HTML5。它还为仍然存在的“离线”问题提供了一个可行的解决方案，该问题是由蜂窝覆盖差和盲点造成的。

![](img/8064f8200d46f0723a8936819e95bbcf.png "tcdisrupt_sf11-7-5-1")

Spool 的创始人[阿维沙尔·加格](https://web.archive.org/web/20230205040442/http://www.crunchbase.com/person/avichal-garg)和[柯蒂斯·斯潘塞](https://web.archive.org/web/20230205040442/http://www.crunchbase.com/person/curtis-spencer)承认，闪存问题正在慢慢消失，但他们认为连接问题将会持续一段时间。

目前， [Spool](https://web.archive.org/web/20230205040442/https://getspool.com/) 可以让你使用它的移动应用程序、Firefox 或 Chrome 扩展，或者浏览器书签来拍摄页面快照。这些保存的页面和相关媒体可以在应用程序中或在线查看，并在您完成查看后收藏以方便访问或存档。Spool 使用的存储空间也可以在设置中调整，对于 Android 用户，支持将内容存储到 SD 卡。

未来，Spool 将专注于添加深度链接(自动下拉您保存的文章中链接的内容)，以及智能“假脱机”您最喜欢的网站，而无需您提出明确的请求。

这款应用目前是免费的(私人测试版[，而创始人正在考虑包括免费增值服务、搜索服务和移动 CDN 模式在内的盈利选择。](https://web.archive.org/web/20230205040442/https://getspool.com/)

Spool 目前正在解决一些现实世界的问题，但可以说不是那些将永远存在的问题。另一方面，Spool 的技术可能比 Spool 的应用程序有更长的保质期。该公司预计其人工智能、计算机视觉、视频提取、视频转码和浏览器仿真基础设施将获得五项专利。收到的资金量目前尚未披露。

艾尔:如何发展业务？

答:人们已经在做这种行为了。Dropbox 和 Evernote 的忠实粉丝——这些网站解决了大部分在线用户的痛点。

DM:网络连接比较好的时候呢？

答:网络基础设施跟不上用户数量。即便如此，这也意味着 Spool 可以获得更快的管道，在手机上更快地加载页面。

MP:爱死了，迫不及待想试试。与竞争相比如何？

答:很多竞争对手关注的是文章内容(Instapaper)。这也是关于不同类型的内容。(视频、音频等。)

JS:人们真的会回头阅读他们存档的资料以便以后阅读吗？

答:我们还可以在未来的某个时间提前智能地为您取东西。但是，是的，它不是一个立即成为主流的产品。
