# 谷歌为更多的社交搜索酿造啤酒

谷歌社交搜索是一个好主意。你将常规的谷歌搜索结果与你的联系人在各种社交网络上分享的相关元素交织在一起。但是有一个很大的障碍。为了让你的联系人自动分享元素，他们必须将他们的各种社交资料链接到他们的谷歌个人资料页面。很多人根本不会这么做。所以谷歌正在改变一些事情，让获取社交数据变得更容易。

从今天开始，到本周结束，谷歌将开始查看你在 Buzz 上分享的内容，并从这些社交网络中抓取元素，用于社交搜索。例如，如果你将你的 Twitter feed 链接到你的 Buzz 页面，即使你没有将其链接到你的 Google 个人资料，你现在也可以在结果中看到来自你的 Twitter 社交图的结果。

显然，谷歌将会抓取公共信息。如果你想让他们停止抓取某些东西，你可以简单地取消它与你的 Buzz 帐户的链接(就像你可以取消它与你的 Google 个人资料的链接一样)。虽然这个解除链接可能需要一点时间，因为爬虫，我被告知。

谷歌的马特·卡茨告诉我们:“*这对用户来说是一个胜利，因为它在社交搜索*中创造了更多的价值。”。我问卡茨大部分社交搜索数据来自哪里，虽然他没有任何确凿的数据，但他说 Twitter 显然是最大的一个(尽管 Twitter/Buzz 仍然存在脱节)。他说，FriendFeed 还是一个有价值的数据来源。

但是现在你在 Buzz 上分享的任何东西都可以被拉进这些社交搜索结果，希望会有更多不同的数据——而且总体上更多。虽然这还没有发生，但卡茨设想有一天，在某个特定区域进行搜索(从你浏览器的地理定位中得知)会从你的 Foursquare 社交图中获得结果。
