# 谷歌浏览器通过浏览记忆巷庆祝其三岁生日

这有点难以置信，但自谷歌于 2008 年 9 月首次推出其 Chrome 浏览器以来，已经过去了三年(当然，我们提前几天发现了这一点，因为泄露了所有事情的漫画书，但 9 月 1 日是你可以下载 Windows 测试版的第一天)。

哇，Chrome 取得了长足的进步。在它的[一岁生日](https://web.archive.org/web/20230204214754/https://techcrunch.com/2009/09/15/chrome-hits-version-3-just-a-year-after-its-launch-still-no-mac-version/)前后，它已经发布了第三个版本，在 Windows 上运行良好，但是仍然没有 Mac 或 Linux 版本。在它的[第二个生日](https://web.archive.org/web/20230204214754/https://techcrunch.com/2010/09/02/google-chrome-birthday/)时，它升级到了第六版——支持 Mac 和 Linux。

如今，Chrome 是一个发电站。它以六周的发布周期运行，所以新功能正在快速推出(我目前运行的是 14 版测试版)。虽然这款浏览器继续增加打印预览和谷歌搜索上漂亮的“即时页面”等关键功能，但它仍然让人感觉轻巧、快捷。根据最近的统计数据，Chrome 已经拥有大约 22%的浏览器市场份额(根据同一份报告，Firefox 拥有 28%，IE 拥有 42%)。

对于 Chrome 生态系统来说，这也是重要的一年，12 月推出了 Chrome 网上商店，6 月推出了 T2 Chrome book。这两者都还没有起飞，但谷歌正在为 Chrome 成为 Windows 等传统操作系统的强大、完全基于网络的替代品奠定基础。昨天，[发布了离线 Gmail、日历和文档的](https://web.archive.org/web/20230204214754/https://techcrunch.com/2011/08/31/googles-new-html5-chrome-apps-for-gmail-calendar-and-docs-give-users-offline-access/)，最好地强调了这些努力的状态:它们仍然是早期和粗糙的(离线 Gmail 甚至还不支持键盘快捷键)，但你可以看到潜力。Chome 作为一个操作系统，未来的成功很难说，但谷歌对它非常重视。**更新:**谷歌指出，离线 Gmail*包含了上一封和下一封邮件的基本 J/K 快捷方式，尽管它仍然缺少很多快捷方式，比如我个人最喜欢的 Y(存档/下一封邮件)。*
