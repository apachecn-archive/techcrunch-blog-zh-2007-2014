看起来[诺基亚](https://web.archive.org/web/20230131221312/http://www.crunchbase.com/company/nokia)的高管们越来越多地掌握了一种由[微软](https://web.archive.org/web/20230131221312/http://www.crunchbase.com/company/microsoft)的高管们在过去几年里磨练出来的特殊技能[:说](https://web.archive.org/web/20230131221312/https://techcrunch.com/2010/07/14/iphone-4-microsoft/)[一些关于他们竞争对手的蠢话](https://web.archive.org/web/20230131221312/https://techcrunch.com/2007/10/28/link-ten-absolutely-stupid-quotes-by-steve-ballmer/),这些蠢话无疑会在某个时候反咬他们一口。直截了当:在与 Pocket-lint 的独家采访中，诺基亚娱乐公司产品营销组合总监尼尔斯·芒克斯加德表示，iPhone 和 Android 设备不再吸引年轻人:

> 我们看到的是，年轻人已经受够了 iPhones。“每个人都有 iPhone，”他说。此外，许多人对 Android 的复杂性和缺乏安全性感到不满。因此，我们越来越多地看到，希望走在前沿、尝试新事物的年轻人正转向 Windows Phone 平台。"

现在，我喜欢诺基亚的 Lumia 800，它运行 Windows Phone Mango，非常喜欢(稍后会有更多介绍)。但这位高管为何觉得有必要把 iPhone 和 Android 作为一个整体来嘲笑，我无法理解。

是的，Windows Phone 看起来和感觉都不一样，但并不是每个人都同意它更好。事实是，诺基亚有一切可以证明把智能手机农场押在微软身上，而 T2 会输得很惨。

到处说今天的年轻人受够了 iPhone，因为“每个人都有”——这到底是什么意思？俗话说:雄辩是银，沉默是金。
