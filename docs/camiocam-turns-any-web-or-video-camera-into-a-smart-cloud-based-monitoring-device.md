# CamioCam 将任何网络或视频摄像头转变为智能的、基于云的监控设备

如果有人把包裹放在你家门口时，你的手机会收到提醒，那会怎么样？或者，如果你能追踪你年迈的父母在某一天是否服用了药物？ [CamioCam 的](https://web.archive.org/web/20230225090552/http://www.camiocam.com/)智能视频监控服务希望能够完成这些任务，甚至更多。这家初创公司将在今天的发布会上发布，它可以将你家中的任何联网摄像头(平板电脑、智能手机、电脑)变成基于云的监控体验。

该公司由前谷歌员工卡特·马斯兰创建，他曾是谷歌本地搜索、地图和地球的产品管理总监。CamioCam 已经从 Freestyle、Marissa Mayer、Greylock、Floodgate、Ellen Levy、John Hanke、Box Group 和其他公司筹集了超过 100 万美元的资金。

它是这样工作的。您可以将任何连接 Wi-Fi 的手机、平板电脑、网络摄像头或 IP 摄像头连接到 CamioCam。只要有运动，摄像机就会将视频上传到 CamioCam，专有算法会分析运动，以确定是否应该标记运动——基本上旨在将数小时的视频缩减为小警报。用户将在手机或电子邮件上收到即时提醒，以及相关视频的片段，这样他们就不必浏览数小时的视频。用户还可以搜索或过滤最重要的事件。

您甚至可以通过在相机视图中选择区域来获得更具体的警报，如厨房的工作台面或婴儿床，并且您可以选择哪些区域在移动时向您发出警报。一台摄像机可以有多个区域(例如，在厨房里，你可以有一个垃圾桶区域和一个冰箱区域)。

正如 Maslan 解释的那样，CamioCam 速度很快，几乎实时的警报比传统相机快 7 倍，而且该服务使用的带宽比流媒体服务少 92%。这家初创公司还承诺减少误报。

就成本而言，你可以使用网络摄像头，在浏览器或 Android 设备上免费使用 CamioCam 应用程序。除此之外，每个额外的相机每月 9.90 美元。

CamioCam 的核心是将浏览器摄像头转变为运动检测摄像头，这在一定程度上是通过 WebRTC 实现的。目前，这家初创公司提供安卓和网络应用。

CamioCam 将与 drop cam(T2)正面交锋，后者是基于云的视频监控的最爱。但马斯兰认为，CamioCam 为用户提供了简单性，因为他们可以在手机或平板电脑中使用浏览器，而不必购买外部设备。
