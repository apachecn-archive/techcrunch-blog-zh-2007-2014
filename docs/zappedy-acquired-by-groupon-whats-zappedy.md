# 被 Groupon 收购的 Zappedy。Zappedy 是什么？

一位匿名消息人士告诉我们，一家为当地企业提供各种技术产品的公司 Zappedy 已经被 [Groupon](https://web.archive.org/web/20230204222302/http://www.crunchbase.com/company/groupon) 收购，根据前者网站上发布的消息，这一消息被证明是正确的(遗憾的是，该消息尚未被 Wayback Machine 存档)。

网上关于 Zappedy 的信息并不多，但由于我非凡的网络搜索技能，我收集了以下信息:

Zappedy 成立于 2010 年初，由前谷歌首席执行官 Eric Schmidt 创立的投资公司[创新努力](https://web.archive.org/web/20230204222302/http://www.crunchbase.com/financial-organization/innovation-endeavors)提供支持。根据其 [AngelList 简介](https://web.archive.org/web/20230204222302/http://angel.co/zappedy)，确切地说，Zappedy 是由 IE 的 [Dror Berman](https://web.archive.org/web/20230204222302/http://www.crunchbase.com/person/dror-berman-2) 和[科里·福特](https://web.archive.org/web/20230204222302/http://www.crunchbase.com/person/corey-ford)资助的。

投资公司[MECK](https://web.archive.org/web/20230204222302/http://www.meckltd.com/)[Wences(Lao)Casares](https://web.archive.org/web/20230204222302/http://en.wikipedia.org/wiki/Wenceslao_Casares)的联合首席执行官兼创始合伙人也出现了。根据他们在 LinkedIn 上的资料，创新努力公司(也是 TokBox 的创始人)罗恩·霍斯和 Project Slice 的联合创始人[哈宾德·辛格·马丹](https://web.archive.org/web/20230204222302/http://www.linkedin.com/pub/harpinder-singh-madan/0/9/b40)也是 Zappedy 的顾问。

我们被告知德罗尔·伯曼和罗恩·霍斯实际上是支持扎佩迪的人。

这是创新努力网站上对创业公司的描述:

> Zappedy 为革命性的应用提供动力，帮助实体商家更好地了解、吸引和留住客户。
> 
> Zappedy 使用从直接反馈和信用卡聚合器收集的数据，实现了下一代商业、娱乐和移动广告应用程序，这些应用程序可以识别客户的离线购买和在线活动。

值得注意的是，Zappedy [似乎与](https://web.archive.org/web/20230204222302/http://supercoolschool.typepad.com/startupschool/2010/08/startup-chile-helping-founders-to-touch-ground-in-south-america.html)和[初创公司智利](https://web.archive.org/web/20230204222302/http://www.startupchile.org/)有关联，这是智利政府的一个项目，旨在吸引早期创业者并说服他们在智利创业。

你可以在这里找到对 Zappedy 联合创始人纳阿玛莫兰[的简短采访](https://web.archive.org/web/20230204222302/http://smallbiztechnology.com/archive/2010/04/zappedy-5-minute-web-site-vi.html/)(也嵌入下面)。

据我所知，Zappedy 基本上让当地的小企业只需发送电子邮件，就能轻松地创建一个网站，并通过社交媒体和日常交易网站做在线广告。Groupon 收购该公司可能是因为它的团队，但也可能是因为它的技术(以便能够为还没有强大在线业务的商家提供更多工具)。

**更新 2:** 有人告诉我们，上面是 Zappedy 以前的产品，他们后来转向了“其他东西”，大概是更接近创新努力网站上描述的服务。因此，下面的视频不再与 Zappedy 最近的产品主张相关，显然 Moran 被错误地描述为 Zappedy 的联合创始人。

Zappedy 由[弗朗西斯科·拉腊因](https://web.archive.org/web/20230204222302/http://www.linkedin.com/pub/francisco-larrain/11/889/9b4)、埃文·麦克米伦[、丹尼尔·佩雷斯·拉达](https://web.archive.org/web/20230204222302/http://cl.linkedin.com/pub/daniel-p%C3%A9rez-rada/5/79/4a3)和[里卡多·齐勒鲁埃罗-拉莫斯](https://web.archive.org/web/20230204222302/http://cl.linkedin.com/pub/ricardo-zilleruelo-ramos/3/b07/31b)共同创立(至少根据他们各自的 LinkedIn 资料)。浏览 LinkedIn 时出现的另一个有趣的名字是[布拉德·格里菲斯](https://web.archive.org/web/20230204222302/http://www.linkedin.com/in/braddgriffith)，他是 [LOLapps](https://web.archive.org/web/20230204222302/http://www.crunchbase.com/company/lolapps) 的前业务开发人员，在此之前，他是[谷歌](https://web.archive.org/web/20230204222302/http://www.crunchbase.com/company/google)的高级金融分析师。
