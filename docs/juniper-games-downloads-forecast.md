# 分析师称，智能手机和平板电脑将成为游戏玩家的主要屏幕，到 2017 年将有超过 640 亿次游戏下载(是 2012 年的 3 倍)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/25/juniper-games-downloads-forecast/>

# 分析师表示，智能手机和平板电脑将成为游戏玩家的主要屏幕，到 2017 年将有超过 640 亿次游戏下载(是 2012 年的 3 倍)

根据分析师 Juniper Research 的一份新报告,智能手机和平板电脑的游戏应用下载量将在未来四年大幅增长，该报告预计 2017 年将有 641 亿次此类下载，是 2012 年 210 亿次的三倍多。这位分析师表示，推动这一高增长率的关键驱动力是越来越多的免费游戏(又称免费增值商业模式)，以及更复杂的设备和智能手机在全球的持续普及。

从分析师的数据来看，免费增值作为游戏应用商业模式的主导地位非常明显:2017 年，它预计只有 7%的游戏在购买时通过智能手机和平板电脑支付。应用内购买和/或广告大概是游戏开发者主要的收入来源。

Juniper 表示，由于“复杂游戏，通过使用云技术实现真正的多平台游戏”数量的增加，移动设备将成为游戏玩家的主要屏幕。设备存储容量的增长也使得消费者能够下载更多的游戏。虽然 Juniper 并不认为智能手机和平板电脑会扼杀专用便携式游戏设备，但它表示，毫无疑问，消费手机正在挑战和侵蚀后者市场——任天堂等玩家将其 3DS 的销售预测下调了 14%，Wii U 的销售预测下调了 27%。

根据 Juniper 的预测，社交和休闲游戏仍将是最受欢迎的下载类型，超过一半的智能手机游戏下载符合这一类型。这与移动设备的关键特征是一致的:永远在线的连接，这意味着连接到社交服务；和便携性，这意味着这些设备适合短时间的休闲游戏来消磨时间。

具体来看平板电脑，Juniper 发现他们的用户尤其热衷于下载游戏，平板电脑上的游戏下载量是智能手机的两倍多。“平板游戏增长如此之快，是因为它们是所有消费群体接触游戏的一种便捷方式。报告作者西恩·罗兰兹在一份声明中评论道:“特别是中核心玩家，他们以前花了很多钱和时间玩游戏，但现在有了工作、家庭或其他承诺，正在推动这一趋势。”“这些人真的很喜欢平板电脑，创新的游戏设备，如基于移动设备的 OUYA 控制台真的很吸引他们。”
