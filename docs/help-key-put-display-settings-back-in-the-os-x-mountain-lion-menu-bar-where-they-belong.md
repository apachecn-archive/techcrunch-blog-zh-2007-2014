# 帮助键:将显示设置放回它们所属的 OS X Mountain Lion 菜单栏中

我喜欢我的 Retina MacBook Pro，部分原因是当我远离家用 Thunderbolt 显示器并想要尽可能多的屏幕空间时，我可以选择切换到极高分辨率的显示模式。但苹果在 OS X Mountain Lion 中做了一件奇怪的事情:他们取消了在 OS X 菜单栏中放置显示偏好项的选项。幸运的是，第三方开发人员已经站出来，并创建了几个实际上超出使命召唤的修复程序。

Mac App Store 提供了显示菜单,这应该有助于那些担心安装操作系统调整的人感到更轻松，因为这意味着这至少通过了苹果的基本质量审查，但并非所有篡改 OS X 系统设置的实用程序都是如此。它提供了一个菜单，让你可以改变 Mac 的显示分辨率，以及任何连接的外部显示器，而不必打开系统偏好设置并点击显示偏好设置本身，自 Mountain Lion 推出以来，这个过程变得过于复杂。

显示菜单很简洁，在很多方面都有帮助，但是在一些方面也有限制。首先，它不支持 15 英寸视网膜显示器 MacBook Pro 在 HiDPI 模式下的原生视网膜显示分辨率。这意味着你不能快速切换回“最适合视网膜”选项。但是对于附加显示器来说，它绝对是完美的，而且是免费的。

如果你想要原生 Retina 分辨率选项，有一个名为“[显示模式-分辨率菜单](https://web.archive.org/web/20230127141146/https://itunes.apple.com/ca/app/display-mode-resolution-menu/id553412521?mt=12)的付费应用商店替代显示菜单，它提供了它-但目前只针对 15 英寸 Retina Pro。对于 13 英寸的手机，还不支持原生的 HiDPI 或 Retina 模式，该应用程序的价格为 2.99 美元。有可能一个更新将完全支持 13 英寸，但从 Display Modes 的开发者网站来看，这是相当稀疏的，我不会屏住呼吸。

尽管这两个应用程序都有局限性，但它们确实提供了对第三方显示器的完全控制，并且它们还提供了任何 Retina MacBook Pro 用户可能会喜欢的另一个技巧:在 15 英寸版本上，可以将分辨率更改为令人难以置信的最大 2880 x1800，或在 13 英寸版本上更改为 2560 x 1600。字体非常小，但是你也有大量桌面空间可以使用，如果你需要编辑打印分辨率的照片或者同时运行多个应用程序并密切关注每一个，这是非常完美的。

不管你是不是视网膜，如果你正在读这篇文章，你可能会错过苹果为了“进步”而莫名其妙地移除的显示偏好菜单栏选项。这些应用程序将它带了回来，根据你使用的 Mac 类型增加了一些额外的超能力，经济实惠，不需要在终端上折腾。
