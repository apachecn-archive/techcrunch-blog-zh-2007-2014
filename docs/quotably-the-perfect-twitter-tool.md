# 可引用的话:理解推特的完美工具

[![](img/ac93deb2b66db1b5ce4e2f12bf123ff5.png)](https://web.archive.org/web/20221006193458/http://www.quotably.com/techcrunch)

一项名为[的新服务可以说是](https://web.archive.org/web/20221006193458/http://www.quotably.com/)可能是迄今为止最好的第三方[推特相关服务](https://web.archive.org/web/20221006193458/http://www.crunchbase.com/company/twitter)。这是因为它将推特信息重新格式化为线程对话，使得关注推特上的实际讨论变得更加容易。

到目前为止，很难理解对话，即使你正处于其中。有时回复会附上您的用户名(@username)，但通常不会。如果你只是观察对话，几乎不可能看到所有的反应。

这项服务很容易使用——只需告诉它一个推特账号，它就会显示涉及该人的线索对话。您也可以通过简单的添加来查看任何推特用户的 RSS 源。rss”到可引用 URL 的末尾，如[quotably.com/techcrunch.rss](https://web.archive.org/web/20221006193458/http://quotably.com/techcrunch.rss)。

我是通过我新的最喜欢的新闻网站[黑客新闻](https://web.archive.org/web/20221006193458/http://news.ycombinator.com/)找到这个的。在[评论](https://web.archive.org/web/20221006193458/http://news.ycombinator.com/item?id=144253)中,[绿河](https://web.archive.org/web/20221006193458/http://www.greenriver.org/)的创造者本·塔克说这只是一个周末项目。

我有一个功能请求——每个线程会话的永久可报价 URL，允许人们直接链接到它。
