# 惠普触摸板将回归百思买？

据广泛报道，在惠普实施大幅降价之前，百思买坐拥超过 200，000 个触摸板，但大甩卖来了又走了，通常情况下就是这样。相反，百思买员工工具包系统中的通知显示，他们与 TouchPad 的争议关系可能还没有结束。

这张由百思买内部人士发送给 Droid Matters 的图片表明，百思买商店将再次开始接收 TouchPad 的发货。由于上一次蜂拥而至的抄底者，员工们被告知要坚持使用售票系统，并记下感兴趣者的信息。

虽然这份通知很有可能及时推出，在《星期日通告》的封面上引起了巨大的轰动，但你不应该屏住呼吸。不同的地区往往有不同的运输时间表，但如果这是真的，更有可能的是，这些单位将在一周的中间开始慢慢回到商店。在这一点上，仍然不知道通知是否只适用于一些商店或所有商店，但由于有一点企业的远见，你最近的商店可能很快就会在他们的电话系统中有一个新的录音，可以澄清细节。

惠普自己的网站承认，他们只是[【暂时】](https://web.archive.org/web/20230204232700/http://www.hp.com/united-states/webos/us/en/tablet/touchpad-availability.html)缺货，再加上一家大型零售商突然进货的消息，让我不禁思考:惠普还剩下多少这样的东西？对一些人来说更重要的是，有多少装有[安卓系统的手机莫名其妙地预装了](https://web.archive.org/web/20230204232700/http://www.youtube.com/watch?v=YTLZHH50V7A)？答案似乎会在本周晚些时候揭晓。
