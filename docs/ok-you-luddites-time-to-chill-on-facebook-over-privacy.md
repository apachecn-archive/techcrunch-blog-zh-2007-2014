# 好了，你们这些勒德分子，是时候在隐私问题上对脸书冷静一下了

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/01/12/ok-you-luddites-time-to-chill-on-facebook-over-privacy/>

2004 年，当 Gmail 推出时，每个人都吓坏了，因为谷歌会阅读你的电子邮件，以找出哪些广告适合你。*指出[连线](https://web.archive.org/web/20221207045854/http://www.wired.com/science/discoveries/news/2009/03/dayintech_0401)“隐私倡导者反对广告模式，该模式涉及谷歌的机器人眼睛扫描每封电子邮件的关键词，并在用户的收件箱旁边显示上下文相关的广告。”。*

 *你的曾曾祖父母可能对此耳熟能详。据说在 20 世纪初，许多人对使用电话感到担忧，因为他们知道电话公司可能会窃听他们的电话。今天有些人不使用电话，因为电话很容易被窃听。

但我们其他人似乎对 Gmail 没什么意见。还有我们的手机。这是因为这些产品的好处远远超过了隐私成本。人们也会和脸书相处得很好。即使他们确实在一个月前对隐私设置做了[转变](https://web.archive.org/web/20221207045854/http://www.beta.techcrunch.com/2009/12/09/facebook-privacy/),这仍然在科技媒体中[回响](https://web.archive.org/web/20221207045854/http://www.readwriteweb.com/archives/why_facebook_is_wrong_about_privacy.php)。

与发表的报道相反，脸书首席执行官马克·扎克伯格上周五晚上在 Crunchies 接受我采访时并没有说“隐私时代已经结束”。你可以自己在这里看视频[。他说的是，他希望脸书随着用户的变化而变化，并保持产品的新鲜感。这正是他们正在做的。](https://web.archive.org/web/20221207045854/http://www.ustream.tv/recorded/3848950)

事实是隐私已经真的真的死了。霍华德·林德宗(Howard Lindzon)前几天说得很对，他说“Equifax、Transunion、Capital One、American Express 和他们的兄弟们侵犯了我们的隐私，”我们做的每一件事，我们买的每一件东西，我们去的每一个地方都被跟踪，并保存在某个数据库里。我们的位置通过我们的手机，或我们的汽车 GPS。我们的信用卡交易。一切。老实说，与每天收集和利用的关于我们每个人的海量数据相比，你在大学里吸食大麻的照片简直是小巫见大巫。你真的看不到其他东西，因为这些公司不喜欢谈论他们收集的数据。例如，我没有看到一篇 Equifax 博客文章详细描述他们是如何收集和出售你的信息的。

重点是我们喜欢脸书。非常非常少的人会停止使用它。不可避免的是，他们会撕掉创可贴，试图让他们的用户公开数据。这是对脸书最好的选择。如果用户对此深恶痛绝，就会有其他人推出一项具有不同政策的竞争性服务，并蓬勃发展。你可以猜到发生这种情况的可能性有多大。

今晚早些时候，我和 Blippy 的首席执行官 T2 谈过话。Blippy 是一项服务，[让用户发布他们用信用卡购买的所有东西。](https://web.archive.org/web/20221207045854/http://www.beta.techcrunch.com/2009/12/11/blippy/)

疯狂吧？谁会想这么做。显然很多人都这么想。到目前为止，该公司已经招收了 2500 名员工。这 2500 人每天向他们的朋友发布价值 20 万美元的购买信息。它成立不到一个月，他们已经跟踪了 380 万美元的交易，平均交易规模为 46 美元。

超过 10，000 人正在等待获得一个帐户，并乐意与世界分享他们的消费行为。

他们为什么要这么做？分享他们买的东西，并谈论它们。或者让广告商看到他们喜欢的东西，并为他们量身定制报价。或者什么的。关键是，我们不再关心隐私了。脸书给了我们想要的东西。*
