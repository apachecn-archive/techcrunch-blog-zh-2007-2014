# 新的 Twitter 现在在侧窗格中显示完整的对话

![](img/4da75822ea459523392caa48dc51944e.png "side")几天前，我们注意到 Twitter 恢复了他们正在测试的[新人区](https://web.archive.org/web/20230202222615/https://techcrunch.com/2010/11/18/twitter-people/)的新功能。好吧，那就来个二对二吧，因为昨天他们推出了另一个新功能——自从新 Twitter 推出以来，许多用户一直在要求的功能:完整的对话线程。

是的，如果你点击一条带有聊天气泡图标的推文，你会在新 Twitter 的右窗格中看到该推文所回复内容的详细信息。以前，你只能看到对话的一个层次。但是现在，Twitter 将向你展示整个来回(假设用户每次都使用“回复”按钮进行回复)。根据你加入对话的位置，你会看到上面所有的“回复”信息，中间是你点击的推文，下面是这个窗格中的“回复”。

这是跟踪 Twitter 上的对话的一种更简单的方式，这是独立的 [Twitter 搜索](https://web.archive.org/web/20230202222615/http://search.twitter.com/)产品已经有一段时间了。一些第三方客户端也提供这一功能(Twitter 的移动应用也是如此)。但这也是不离开 twitter.com 的另一个原因。

回到 6 月份，[我们注意到](https://web.archive.org/web/20230202222615/https://techcrunch.com/2010/06/18/twitter-threaded-conversations/)Twitter 已经雇佣了应用 Twitoaster 的创建者，这很有趣，因为该应用专门在 Twitter 上显示对话。没有消息表明这是否在此次推出中发挥了作用。我很高兴它在这里。
