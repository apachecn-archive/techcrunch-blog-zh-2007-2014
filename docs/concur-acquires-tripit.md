# TripIt 打包退出——被 Concur 以高达 1.2 亿美元收购

![](img/e1df9f605415a9b7b030f5b1e0e8f95f.png)

[Concur](https://web.archive.org/web/20230202230521/http://www.concur.com/) ，一家整合旅行和费用管理解决方案的上市供应商，今天早上宣布已经同意收购 [TripIt](https://web.archive.org/web/20230202230521/http://www.crunchbase.com/company/tripit) ，该公司帮助旅行者轻松[组织和分享旅行计划](https://web.archive.org/web/20230202230521/http://www.tripit.com/)。

Concur 最初支付了大约 8200 万美元的预付现金、股票和未授予的限制性股票单位——随着时间的推移，额外的对价可能会使总价值达到 1.2 亿美元。财务状况的更多细节如下。

当一切烟消云散时，这对公司来说是一个很好的退出，到目前为止已经筹集了 1300 万美元的资金。投资者包括[O ' Reilly alpha tech Ventures](https://web.archive.org/web/20230202230521/http://www.crunchbase.com/financial-organization/o-reilly-alphatech-ventures)、[欧洲创始人基金](https://web.archive.org/web/20230202230521/http://www.crunchbase.com/financial-organization/european-founders-fund)和 [Azure Capital Partners](https://web.archive.org/web/20230202230521/http://www.crunchbase.com/financial-organization/azure-capital-partners) 。

TripIt 最初是在 TechCrunch40 上发布的，并受到了迈克尔·阿灵顿的喜爱。

在一份声明中， [Concur](https://web.archive.org/web/20230202230521/http://www.crunchbase.com/company/concur) 董事长兼首席执行官史蒂夫·辛格解释了收购 TripIt 的理由:

> “移动解决方案的进步改变了商务旅行者购买、分享、管理和消费其旅行计划的方式。人们普遍需要给混乱的旅行带来秩序，让商务旅行者的生活更美好。无论是有管理的旅行还是无管理的旅行都是如此。
> 
> 我们共同解决整个商务旅行流程中的挑战——从预订、旅行中的活动和分享旅行信息，到旅行后的费用管理和对账。"

Concur 表示，超过 1500 万人使用其解决方案来帮助管理他们的商务旅行和费用。整个 TripIt 团队将加入 Concur。

Concur 已签署最终协议，在交割时以约 2700 万美元的现金和约 4400 万美元的 Concur 股票收购 TripIt，外加在交割后 30 个月内结算的约 3800 万美元的或有现金，但须遵守最终协议中规定的某些调整和托管条款。

作为收购的一部分，Concur 将把未授予的 TripIt 期权兑换成 Concur 限制性股票单位，在交易结束时总价值约为 1100 万美元。考虑到所有因素，这笔交易的总价值高达 1.2 亿美元。

Concur 表示，将在 2 月初的 2011 财年第一季度财报电话会议上提供更多细节。
