# DMD Panorama 开放 API，支持任何应用中的全景照片

听到一家摄影初创公司的联合创始人兼首席执行官的声音是一件奇怪的事情，但 DMD Panorama 的 Elie-Gregoire Khoury 告诉我，全景照片将成为“一种商品”这并不意味着是时候退出照片业务了——相反，Khoury 希望看到全景照片成为各种网站和应用程序的标准功能，就像现在的普通照片一样。

如果 Khoury 如愿以偿，这一切都将通过 DMD 的新 API 实现。

Khoury 说，自从 2011 年 6 月在 iPhone 上推出以来，DMD Panorama 已经被下载了 450 万次。他的目标是建立最快、最简单的方式来拍摄全景照片，他可能已经成功了——例如，[《华尔街日报》的这篇文章](https://web.archive.org/web/20221208001808/http://online.wsj.com/article/SB10001424052702303816504577305702578426084.html)将该应用描述为“iPhone 上最易用的全景图片应用”

当我亲自试用这个应用程序时，我被深深打动了。要拍摄全景照片，你只需启动相机，将手机侧向移动，将屏幕上的阴阳标志合在一起。这个过程只是比拍摄普通照片稍微复杂和耗时一些。

DMD Panorama 是由黎巴嫩的一个五人小组建造的。Khoury 表示，该国的基础设施存在一些挑战——比如每天只有 6 小时的电力，以及办公室互联网连接每月 2g 的下载上限——但该公司成功制作了这款热门应用，并从投资者那里筹集了天使投资，其中包括早期的谷歌人 Georges Harik 和 bery tech Fund。

现在，Khoury 希望招募应用开发者使用 DMD 的免费 API。最终，Khoury 希望 DMD 能够在全景照片可能有用的任何应用程序中增强拍照体验——例如，Khoury 建议 DMD 可以将全景照片引入明信片应用程序，或者它可以帮助人们在 Airbnb 等租赁应用程序中拍摄全景照片来展示他们的房子。

用户将需要安装 DMD Panorama，以便利用集成，但一旦他们这样做，目标是在 DMD 和集成应用程序之间创建无缝体验。因此，当使用另一个应用程序时，用户可以在适当的时候点击“全景”按钮(或任何东西)，这将打开 DMD 全景或提示他们安装它。他们在 DMD 中拍摄照片，然后返回到原始应用程序。

Khoury 说，在更广泛地开放 api 之前，他正在与一些合作伙伴一起测试 API，因此感兴趣的开发人员应该发送电子邮件到 api (at) DerManDar (dot) com。
