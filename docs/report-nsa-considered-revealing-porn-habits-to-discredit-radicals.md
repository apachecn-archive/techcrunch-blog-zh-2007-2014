# 报道:美国国家安全局考虑揭露色情习惯来抹黑激进分子

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/11/27/report-nsa-considered-revealing-porn-habits-to-discredit-radicals/>

我觉得，当人们对政府大规模监控的未知威胁感到恐慌时，他们会暗暗担心秘密特工会向朋友和家人透露他们淘气的上网习惯。今天，这种担心暴露了出来。至少对恐怖分子嫌疑人来说是这样。

《赫芬顿邮报》报道称，新的绝密文件显示，美国国家安全局考虑通过披露穆斯林激进分子的互联网色情历史来勒索他们。

告密者、现俄语学习者爱德华·斯诺登泄露的机密文件显示，“一份关于激进化的评估报告表明，当激进分子的私人和公共行为不一致时，他们在权力领域显得特别脆弱。”

根据这份文件，美国国家安全局拥有至少两个可疑目标的令人尴尬的信息，其中一些数据是根据有争议的《外国情报和监视法》(FISA)通过联邦调查局的监视收集的。

没有信息表明国家安全局是否真的使用了红脸诱导色情抹黑计划。或许更重要的是，目前还不清楚美国国家安全局的目标是可疑的恐怖分子，还是那些通过互联网信息帮助他人变得激进的人。

前国家安全局总法律顾问、媒体最喜欢的监控辩护律师斯图尔特·贝克(Stewart Baker)表示，与轰炸嫌疑人相比，“总体而言，这更公平，或许也更人道”，他将“它”描述为“向他们透露真相”。

这是一个安全的赌注，美国国家安全局可能会发现恐怖分子在他们的休息时间偷偷放入一些色情；不是因为所有男人都会看色情片这是一个科学事实，而是因为官员们之前已经在 T2 发现了这种现象。美国官员在奥萨马·本·拉登在巴基斯坦的最后藏身处发现了一个“广泛的”色情图书馆。

对于那些读完这篇报道却没有抹去互联网历史的读者来说，《赫芬顿邮报》提醒我们，我们的情报机构过去一直保存着政治异见人士的有趣秘密，[包括](https://web.archive.org/web/20230306040230/http://www.motherjones.com/politics/2013/08/j-edgar-hoover-war-martin-luther-king)马丁·路德·金。

互联网显然有一种讽刺的感觉，因为《赫芬顿邮报》报道中第一个建议“你可能喜欢”的链接是“为什么女同性恋者会看同性恋色情片”，这可能是故意试图诱捕——或者因为色情片在互联网上随处可见。
