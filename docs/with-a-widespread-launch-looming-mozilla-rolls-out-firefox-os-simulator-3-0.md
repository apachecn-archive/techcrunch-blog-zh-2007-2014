# 随着大范围发布的临近，Mozilla 推出了 Firefox OS Simulator 3.0

Mozilla 在 3 月时热衷于宣传其 Firefox OS 模拟器[的 3.0 版本，但没有太多关于热切的开发者何时可以开始摆弄它的信息。对于 HTML5 爱好者来说，值得庆幸的是，六周的平静期结束了——该团队刚刚在官方](https://web.archive.org/web/20221208142019/https://beta.techcrunch.com/2013/03/13/mozilla-provides-an-early-peek-at-its-updated-firefox-os-simulator/) [Mozilla Hacks 博客](https://web.archive.org/web/20221208142019/https://hacks.mozilla.org/2013/05/firefox-os-simulator-3-0-released/)上宣布，最新更新的模拟器现在可供下载。

在[预览版](https://web.archive.org/web/20221208142019/https://beta.techcrunch.com/2013/03/13/mozilla-provides-an-early-peek-at-its-updated-firefox-os-simulator/)中出现的所有功能都被考虑在内——想想对旋转显示器的支持和用于测试位置感知应用的模拟地理位置 API 但模拟器套件自我们上次看到它以来已经有所改进。这些调整大多是日常工作的改变:下载的大小已经减少，这导致了更快的启动时间，模拟器现在支持常见的操作系统快捷方式，如 Cmd + Q 关闭，但模拟器也已经更新，可以运行新版本的 Firefox OS 和 Gaia 用户界面层。

也就是说，未来的 Firefox OS 开发人员可能会更多地使用模拟器的一个功能:将正在开发的应用程序推送到连接的测试设备上的能力。Mozilla 及其硬件合作伙伴华为、LG 和中兴通讯(在世界移动通信大会上展示了其首款 FFOS 设备)已经指出今年晚些时候将在巴西、哥伦比亚、匈牙利、墨西哥、波兰、塞尔维亚、西班牙和委内瑞拉推出设备，但这些手机上的体验质量将最终决定 Firefox OS 是否会失败。

即便如此，火狐操作系统开发者设备的强劲早期销售可能预示着今年晚些时候第一批面向消费者的手机的正式发布。看看西班牙硬件 OEM 初创公司 Geeksphone 就知道了——上个月末，它开始分别以 119 美元和 194 美元的价格销售 Keon 和 Peak reference 设备，该公司被迫限制发布当天售出的手机数量，以便 20 人的团队能够跟上出货。

这是一个充满希望的开始，特别是对于像 Geeksphones 这样年轻的公司来说，但毫无疑问，Firefox OS 将在其发布市场上面临一些严峻的竞争。Android 驱动着数量惊人的廉价智能手机，诺基亚已经重新专注于基于 Windows Phone 和老化的 40 系列操作系统开发低成本设备。与此同时，低价 iPhone 的持续传言仍在流传——当我第一次玩 Firefox OS 时，它似乎是新的和喜欢冒险的智能手机用户的一个新奇选择，但我们必须看看行业其他人如何回应。
