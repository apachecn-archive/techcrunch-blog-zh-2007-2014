# 新的简易捆绑包尝试不同的定价策略

如果你在过去一年左右的时间里关注过游戏世界，你可能会知道 [Humble Bundle](https://web.archive.org/web/20230203072449/http://www.humblebundle.com/) ，这是一个以慈善为导向的促销活动，你可以为一些杰出的独立游戏支付你想要的费用。到目前为止，少数几个已经得到了很好的接待，为慈善机构筹集了数百万美元(EFF 和儿童游戏)。一部新的游戏刚刚推出，广受好评的战略射击游戏*《冰雪奇缘》*是主要卖点，但定价方案有了新的变化。

你不是想付多少钱就能得到整个套装，而是根据你付出的多少，你要么只得到*冰冻突触*要么得到游戏加上~~整个~~以前的套装。低于平均水平就能得到游戏，高于平均水平就能得到捆绑包。有一点点聪明。

支付高于平均水平(在我写这篇文章时大约 4.40 美元)几乎肯定会是更受欢迎的选择，因为它会给你带来相当多的额外内容(优秀的 *Shadowgrounds* 及其续集)。当你支付超过平均水平时会发生什么？你提高了平均水平。这是一个很好的、软的方式来设定价格底线，而不会阻止现金短缺(或者只是便宜)的人花一美元买一个好游戏。就在我写这篇文章的时候，平均价格已经上涨了几个便士，当你看到成千上万的销量时，这些便士开始增加。

到目前为止，他们已经卖出了超过 30，000 包，筹集了超过 130，000 美元。销售才刚刚开始(持续两周)，所以这些数字应该会上升不少。前往[简陋的捆绑网站](https://web.archive.org/web/20230203072449/http://www.humblebundle.com/)购买或查看物品。这里有一段相关游戏的视频:
