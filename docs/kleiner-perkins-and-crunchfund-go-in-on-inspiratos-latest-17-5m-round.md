# 凯鹏华盈(Kleiner Perkins)和 Crunchfund 参与了 Inspirato 最新一轮 1750 万美元的融资

凯鹏华盈(Kleiner Perkins cau field)&拜尔斯、[克朗彻夫](https://web.archive.org/web/20230205001639/http://www.uncrunched.com/)、 [Access Venture Partners](https://web.archive.org/web/20230205001639/http://www.accessventurepartners.com/) 和(随机)扬基队一垒手马克·特谢拉(Mark Teixeira)已经为折扣豪华度假初创公司 [Inspirato](https://web.archive.org/web/20230205001639/https://www.inspirato.com/) 进行了一轮 1750 万美元的融资。

与高端度假屋领域的传统竞争者不同，Inspirato 能够通过长期租赁而不是拥有自己的物业，在豪华旅游目的地提供大幅折扣。

由于租赁节省了资金，创始人 [Brent](https://web.archive.org/web/20230205001639/http://www.crunchbase.com/person/brent-handler) 和 [Brad Handler](https://web.archive.org/web/20230205001639/http://www.crunchbase.com/person/brad-handler-2) 放弃了每笔交易的经纪费，取而代之的是 Inspirato 从其一次性 15000 美元的启动费和每年 2500 美元的维护费用中获利。

显然，这在富人中有一个巨大的市场，因为我的前老板迈克·阿灵顿刚刚加入并投资了，“这并不适合所有人，”他说，“但有很多人认为这几乎是一笔神奇的交易。”这家初创公司已经积累了 900 名会员，自 2011 年 1 月成立以来，每月新增 100 名会员。
