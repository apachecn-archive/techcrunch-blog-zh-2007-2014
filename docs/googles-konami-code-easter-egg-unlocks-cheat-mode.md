# 谷歌的 Konami 代码复活节彩蛋解锁'作弊模式'

视频

Konami 代码又出现了。去谷歌搜索栏，用语音搜索“上，上，下，下，左，右，左，右。”就这样，你将拥有无限的免费谷歌搜索。

当然这是开玩笑的。这只是在谷歌工具中发现的最新的隐藏宝石。

这不是搜索巨头第一次在其产品中使用 Konami 代码。当输入到 Pixel Chromebook 中时，笔记本电脑盖上的 LED 灯带会通过闪烁一系列随机颜色来进行灯光表演。臭名昭著的代码还[解锁了 Google Play 中的成就](https://web.archive.org/web/20230130100805/https://www.youtube.com/watch?v=HSG__1hcFo8)。

这一系列按钮是许多经典 Konami 视频游戏中使用的作弊代码。当在《NES 的魂斗罗》中使用时，它将解锁 30 条额外的生命，这正是我到达瀑布关卡所需要的。

谷歌以在其产品中隐藏这类幽默的视频游戏为主题的插科打诨而闻名。尝试搜索“[做桶滚](https://web.archive.org/web/20230130100805/https://techcrunch.com/2011/11/03/do-a-barrel-roll-in-google-right-now/)”。那是我最喜欢的。
