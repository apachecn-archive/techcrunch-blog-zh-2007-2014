# 运营商对奥巴马:给我们更多的频谱接入！

奥巴马总统在二月份宣布支持一项无线创新和基础设施倡议，他认为该倡议(除其他外)将“惠及所有美国人，加强公共安全，刺激无线服务、设备和应用的创新。”根据他的计划，未来十年将释放 500 MHz 的频谱，以刺激移动宽带扩张和 4G 网络的发展。

今天，威瑞森、美国电话电报公司、T-Mobile 和 Sprint 的负责人决定联系欧巴马总统，告诉他他们到底在瞄准哪个频段，而不是听天由命。在一封[联合署名的信](https://web.archive.org/web/20230204113134/http://files.ctia.org/pdf/CTIA_Letter_28July2011.pdf) [PDF]中，他们坦率地列出了他们希望看到的情况:

> …我们敦促您指导国家电信和信息管理局确定并努力清理 3 千兆赫以下的宽的、配对的、国际协调的频带。访问这些关键频段的频谱将极大地促进实现您的无线计划和联邦通信委员会的国家宽带计划中提出的重要目标。

但这意味着什么呢？简而言之，四大运营商认为释放这些频段将有助于推动移动宽带覆盖扩展到急需的地区，这是联邦通信委员会[国家宽带计划](https://web.archive.org/web/20230204113134/http://www.broadband.gov/plan/executive-summary/)的一个关键焦点。从表面上看，这是一个良性的、似乎利他的请求，但毫无疑问，他们肯定会受益。尽管如此，如果这意味着人们将能够在以前无法获得的地方获得急需的互联网服务，我愿意超越财务动机。
