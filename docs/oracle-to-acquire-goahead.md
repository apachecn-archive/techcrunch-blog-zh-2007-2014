# 甲骨文将收购 GoAhead

[甲骨文](https://web.archive.org/web/20230205002635/http://www.crunchbase.com/company/oracle)今晨[宣布](https://web.archive.org/web/20230205002635/http://www.oracle.com/us/corporate/press/495941)它已经[收购了](https://web.archive.org/web/20230205002635/http://www.oracle.com/us/corporate/acquisitions/goahead/index.html) [GoAhead 软件](https://web.archive.org/web/20230205002635/http://www.goahead.com/)，该软件向网络设备提供商(nep)和通信行业的其他参与者出售打包服务可用性软件。收购的财务条款没有披露。

GoAhead 是一家总部位于华盛顿贝尔维尤的私营公司，它提供软件来加快需要随时可用的电信级服务的上市时间。

该公司声称，迄今为止，它的软件已经被 nep 部署了 100，000 多次。

客户[包括](https://web.archive.org/web/20230205002635/http://www.goahead.com/customers/ourcustomers.aspx)摩托罗拉、LG-Nortel、Alcatel 和 Oki。

该交易受惯例成交条件的限制，预计将在 2011 年底前完成。在交易完成之前，每家公司都将继续独立运营。
