# Citrix 收购“企业 Dropbox”share file

[Citrix Systems](https://web.archive.org/web/20230205020801/http://www.crunchbase.com/company/citrix-systems) 已经[收购了](https://web.archive.org/web/20230205020801/http://www.crunchbase.com/company/sharefile) [ShareFile](https://web.archive.org/web/20230205020801/http://www.sharefile.com/) ，这是一家云存储、文件共享和协作解决方案提供商，这两家公司[今天早上](https://web.archive.org/web/20230205020801/http://www.sharefile.com/news/sharefile_acquisition.aspx)宣布。

ShareFile 使企业能够跨多种设备安全地[存储、同步和共享](https://web.archive.org/web/20230205020801/https://techcrunch.com/2011/01/18/sharefile-sync/)公司内外的业务文档和文件。

收购的财务条款没有披露。

ShareFile 首席执行官 Jesse Lipson 将成为 Citrix 新成立的数据共享产品组的副总裁兼总经理，该产品组将负责 ShareFile 产品线。

此举符合思杰所谓的“个人云”的愿景，即每个人独有的文件、应用程序和个人数据的集合，可以从任何位置和设备访问。

ShareFile 与 Dropbox 和 Box.net 等公司竞争，但它更专注于向各种规模的组织中的 IT 部门销售其产品，而不是消费者和小企业。其他替代方案包括 [Egnyte](https://web.archive.org/web/20230205020801/http://www.crunchbase.com/company/egnyte) 、 [Nomadesk](https://web.archive.org/web/20230205020801/http://www.crunchbase.com/company/nomadesk) 、 [Mozy](https://web.archive.org/web/20230205020801/http://www.crunchbase.com/company/mozy) 和 [Syncplicity](https://web.archive.org/web/20230205020801/http://www.crunchbase.com/company/syncplicity) 。
