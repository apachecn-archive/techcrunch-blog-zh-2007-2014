# 第一个第三方游戏在 Zynga.com 上线

早在三月份，Zynga first [就宣布了它的社交游戏平台。从那以后，Zynga 自己的游戏在 Zynga.com 的](https://web.archive.org/web/20221210022732/http://company.zynga.com/about/press/press-releases/zynga-unveils-new-platform-play)上线，该公司一直在[披露更多的细节和合作伙伴](https://web.archive.org/web/20221210022732/https://beta.techcrunch.com/2012/06/26/zynga-network-adds-social-lobby-for-users-across-all-devices-zynga-with-friends/)。然而，到目前为止，似乎一直缺失的是由 Zynga 以外的公司开发并在该平台上运行的真实游戏。

所以今天，在一个强调 Zynga 合作伙伴的新闻发布会上，该公司表示，第一批第三方游戏已经在 Zynga.com 推出——具体来说是四款游戏。这些标题是:

Zynga 还宣布了 Zynga 平台的九个新合作伙伴:Antic Entertainment、Big Bite Games、CrayonPixel、Eruptive Games、JamRT、The Method、Playnery、RocketPlay 和 TikGames。这使得平台合作伙伴的总数达到 24 个。

该平台允许开发者在 Zynga.com 发布社交游戏活动，还包括聊天和多人游戏等社交功能。它适用于网页游戏和手机游戏。事实上，Zynga 表示，它最近推出了第一款手机合作游戏 Phosphor Games 的 [Horn](https://web.archive.org/web/20221210022732/http://www.phosphorgames.com/horn/) 。

你可以在 Zynga 的这篇博客文章中读到更多关于新游戏的内容。
