# 通信主管肖恩·加勒特离开 Twitter

Twitter 通信负责人肖恩·加勒特(Sean Garrett)刚刚在一系列推文中宣布他将离开 Twitter——当然。当被要求确认他的离职时，Twitter 通信代表卡罗琳·彭纳说:“我认为他的推文说明了一切……”

加勒特还神秘地[在他的个人博客上发布了街头歌曲《我们的歌手》](https://web.archive.org/web/20230203072724/http://seangarrett.tumblr.com/post/12528276453/im-dreamin-of-something-now)和“我要走了”的推文。

在 2010 年 2 月[离开并加入 Twitter](https://web.archive.org/web/20230203072724/http://www.mediabistro.com/prnewser/sean-garrett-joins-twitter-as-vp-communications_b3093) 之前，加勒特是科技公关机构 463 Communications 的合伙人，该机构是他共同创立的。在 463 Communications 之前，Garrett 曾在 Applied Communications 和 Bite Communications 担任公关角色，这两家机构由专注于通信的控股公司 Next Fifteen 所有。

加勒特在他的两年任期内建立了多达八人的 Twitter 公关团队(你可以在马歇尔·柯克帕特里克策划的名单中关注他们)。Twitter 最近聘请了谷歌的凯伦·维克瑞担任媒体/传播职位(她的简历上写着“文字辩论者”))目前还不清楚谁将接替加勒特。
