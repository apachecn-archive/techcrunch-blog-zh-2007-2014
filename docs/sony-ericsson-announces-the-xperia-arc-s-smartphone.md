# 索尼爱立信发布 Xperia Arc S 智能手机

今天在柏林的 IFA 会议上，索尼爱立信宣布了其新的 Xperia 智能手机 Xperia Arc S。不幸的是，最初宣布的细节相当稀少，但我们知道的是:

这款手机将于今年 10 月首次上市。它将运行在 1.4GHz 处理器上，并将配备由索尼移动 Bravia 引擎驱动的索尼现实显示器。最初 Xperia Arc 上的现实显示器的像素密度为 854 x 480，比我们习惯在高端手机上看到的更标准的 800 x 480 分辨率略有升级。真正让现实显示器与众不同的是索尼移动 Bravia 引擎的支持，该引擎旨在以类似于 Bravia 电视的方式改善色彩和细节。

这款手机还将能够通过其 Exmor R 图像传感器拍摄 2D 全景照片。这些图像可以转换成 3D 格式，届时你可以在该公司的 3D Bravia 电视机上观看这些图像。

从外观上看，这与其前身 Xperia Arc 非常相似，但有一些额外的提升，如处理器升级。我们也看到了黑色和白色版本的图像，但还不清楚是否会有更多的彩色口味。

随着事态的发展会有更多的报道。
