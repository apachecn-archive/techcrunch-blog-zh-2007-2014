# Technorati 推出流媒体更新服务

[![](img/419efc8676929ebbf0eb303176644909.png) ](https://web.archive.org/web/20230324084019/http://technorati.com/topics) [Technorati](https://web.archive.org/web/20230324084019/http://www.crunchbase.com/company/technorati) 已经[宣布](https://web.archive.org/web/20230324084019/http://technorati.com/weblog/2007/09/373.html)推出 [Technorati Topics](https://web.archive.org/web/20230324084019/http://technorati.com/topics) ，这是一条实时新闻流，提供一系列感人的博客文章。

它取代了刚刚重新推出的 Technorati 主页。

Technorati 的 Dorion Carroll 说，这些变化是由于用户对“[Technorati]应该如何组织浩瀚的博客世界，帮助人们找到好东西，帮助伟大的博客作者被发现……”的反馈。特别是，有很多博客帖子要求我们建立博客作者想要的东西。”

对 Technorati 不友好是很容易的，有些人可能会认为今天的举动是在泰坦尼克号上洗牌。Technorati 今年陷入了一个又一个危机，[失去了一位首席执行官](https://web.archive.org/web/20230324084019/https://techcrunch.com/2007/04/06/technorati-ceo-search-confirmed/)并进行裁员，因为它试图成为每个人的一切，但未能在任何垂直领域推出一款杀手级产品。另一方面，作为一个长期的 Technorati 用户，很高兴看到 Technorati 回归到它作为博客搜索引擎的根基。在改进索引方面仍有一些工作要做，但就 focus Technorati Topics 而言，至少是朝着正确的方向迈出了一步。
