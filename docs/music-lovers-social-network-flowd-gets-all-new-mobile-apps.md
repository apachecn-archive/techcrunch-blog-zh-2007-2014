# 音乐爱好者的社交网络 Flowd 获得所有新的移动应用 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/11/14/music-lovers-social-network-flowd-gets-all-new-mobile-apps/>

音乐爱好者的社交网络 [Flowd](https://web.archive.org/web/20230204221237/http://flowd.com/) ，最近[在欧洲首次亮相后](https://web.archive.org/web/20230204221237/http://www.businesswire.com/news/home/20111005005706/en/Flowd-Leading-Music-Fan-Social-Network-Launches)抵达美国，刚刚更新了其移动应用程序，增加了一大堆新功能，值得一看。到目前为止，Flowd 只是稍微有点意思，因为它主要是在吹捧其 600 名左右的艺术家和 DJ，他们已经注册了该网络——这个数字在大计划中只是所有音乐中的沧海一粟。

但有了新的移动应用程序，Flowd 现在可以聚合来自其他社交网络的艺术家活动，包括 Twitter、YouTube、SoundCloud 和 Last.fm，允许用户访问艺术家的更新、简历和演出日历，即使艺术家不在 Flowd 上。

虽然在测试中，这种添加的导入/聚合功能并不意味着每个艺术家个人资料页面现在都充满了数据(这仍然是一种偶然的情况)，但肯定比以前有更多的信息。另外，应该注意的是，通过自动方式填写个人资料的问题可能会受到一些艺术家现在没有在线演出信息这一简单事实的影响。换句话说，没有东西可以拉进来。

在更新的 Flowd 移动应用程序中，有漂亮的 Twitter 风格的“关注”按钮，可以让你快速建立一个你最喜欢的乐队的更新流，以便获得新音乐会和其他新闻的细节。在某种程度上，Flowd 应用程序可以作为对音乐感兴趣的人的 Twitter 客户端，而你不必…你知道…实际上*使用* Twitter。

今天的应用更新(iOS 和 Android)中还有一个新功能，让 Flowd 扫描你的设备，以查看当前正在播放的音乐或最后播放的曲目。然后，它可以将这些信息发布回 Flowd，也可以交叉发布到脸书或 Twitter 上。没有 Spotify +脸书的整合(或者 MOG、Rdio、Rhapsody 等)那么优雅。+脸书)，但它可以为那些仍然喜欢从亚马逊或 iTunes 等服务购买 MP3 而不是付费订阅音乐的老派人士提供社交音乐分享的途径。(是的，现在还有很多这样的人)。

Flowd 移动应用程序在 [iTunes](https://web.archive.org/web/20230204221237/http://itunes.apple.com/us/app/flowd/id378303808?mt=8) 、 [Android](https://web.archive.org/web/20230204221237/https://market.android.com/details?id=com.digia.flowd&hl=en) 和 [Ovi](https://web.archive.org/web/20230204221237/http://store.ovi.com/content/118220) (诺基亚)上免费提供，但应用程序更新仅在 iOS 和 Android 上进行。

Flowd 是 Digia Ventures 的投资组合公司，Digia Ventures 是领先的芬兰技术服务公司 Digia，Plc 的风险部门，在芬兰赫尔辛基和加利福尼亚州圣克拉拉设有办事处。
