# GigaOm 购买了一个移动博客——世界上少了一个独立的博客

[![](img/5386aee276537327722856efa4f9599f.png)](https://web.archive.org/web/20221210044638/http://www.crunchbase.com/company/jkontherun) 博客网络 [GigaOm](https://web.archive.org/web/20221210044638/http://www.gigaom.com/) 将[宣布](https://web.archive.org/web/20221210044638/http://gigaom.com/2008/07/22/gigaom-acquires-jkontherun/)收购小而出色的移动博客 [jkOnTheRun](https://web.archive.org/web/20221210044638/http://www.jkontherun.com/) 。创始人[詹姆斯·肯德里克](https://web.archive.org/web/20221210044638/http://www.crunchbase.com/person/james-kendrick)和[凯文·托弗尔](https://web.archive.org/web/20221210044638/http://www.crunchbase.com/person/kevin-tofel)将留在休斯顿继续做他们的事情。

交易规模尚未公布，可能很小。但它展示了我在 3 月所写的[的开端——随着这个领域竞争变得异常激烈(你会爱上零进入壁垒)，越来越好的博客(主观定义)的出现。](https://web.archive.org/web/20221210044638/http://www.beta.techcrunch.com/2008/03/19/more-bloggers-raising-money-here-come-the-politics-and-here-comes-my-rant/)

我预测这只是这一过程的开始，这一过程将在未来 12-18 个月内加速。缺乏竞争欲望的大型博客将会卖给大型媒体公司。那些更有竞争力的大型博客想把这件事做到底，他们将开始收购较小的博客，并按照主题领域进行分组。谁能建立起最有趣、最惊人的声音网络，谁就能最终“获胜”或者也许每个人都会赢，但程度不同。

我坚持我的观点，博客应该获得现金盈余，然后开始收购其他人——筹集一大笔钱只会刺激人们花钱，而你会失去对一群投资者的控制，这些投资者可能对如何建立博客知之甚少或一无所知。

所有这一切中最清楚的是，那个小的、独立的、充满激情的博客写手日以继夜地写下任何能抓住她想象力的东西，在生态系统中扮演着重要的角色。他们保持较大的博客的诚实，其中最好的将凭借自己的力量成长为大公司。我们博客生活在一个美丽、肮脏、高度竞争和混乱的世界，大多数时候我不会为了这个世界而改变它。
