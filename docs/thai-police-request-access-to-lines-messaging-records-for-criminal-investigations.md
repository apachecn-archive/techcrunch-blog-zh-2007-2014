# 泰国警方要求查看 Line 的信息记录，以便进行刑事调查

据美联社报道，[泰国警方官员实际上在谈论他们监控公民的努力，这与当前的趋势形成了奇怪的逆转。泰国政府和警方要求流行的日本消息应用 Line 提供他们的在线聊天档案，目的是帮助他们收集涉嫌参与武器交易、毒品、卖淫等犯罪的人的数据，并发表与泰国君主制背道而驰的言论。](https://web.archive.org/web/20221210055455/http://news.yahoo.com/thai-police-seek-monitor-chat-app-crimes-113148175.html)

泰国法律禁止执政的王室成员说不，对这种言论可以判处长达 15 年的监禁。政府也积极参与删除网上对王室的批评。拟议中的监控计划将允许政府审查个人之间的私人通信，以及公开发布的材料。

美联社报道称，台湾高级技术犯罪警察 Pisit Paoin 表示，他们将根据具体情况针对个人，而不是进行某种全面监控(a la PRISM ),但这很难让泰国的 Line 用户放心。该计划的批评者建议，在这种情况下，警方至少应该获得搜查令。

Line 在泰国拥有超过 1，000 万用户，它对美联社说，它不会收集或存储用户的任何信息，也没有收到泰国政府对这些数据的正式请求。据报道，Pisit 本周五在东京安排了一次与 Line 的会议，讨论泰国警方的需求。

Line 并不是第一家遭遇与当地官员合作问题和数据隐私问题的公司。[例如，黑莓最近被迫在印度](https://web.archive.org/web/20221210055455/http://thenextweb.com/asia/2013/07/10/after-a-lengthy-battle-blackberry-will-finally-let-the-indian-government-monitor-its-servers/)提供通讯服务，在 T4【Skype】和其他公司在印度和其他市场也面临压力。

这可能不是一个孤立的案例，但对用户来说无疑是一个可怕的案例，特别是因为 Line 和其他即时通讯服务更专注于私人通信，而不是像脸书和 Twitter 这样更多基于广播的社交网络。
