# 以色列企业家:知道你在玩什么游戏 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/04/04/israeli-entrepreneurs-know-what-game-you-are-playing/>

![](img/d0c86bc4e5c7e246948d8ca4f2cea2df.png)

特拉维夫——前几天，我在一个名为 [Techonomy](https://web.archive.org/web/20221005152526/http://blog.go2web20.net/2009/04/6-israeli-startups-born-during.html) 的以色列活动上发言，六家精心挑选的以色列初创公司正在展示新产品。这些公司令人印象深刻，观众济济一堂，由蓝筹技术公司赞助，富有的专家在台上提供反馈。

事实上，除了一点之外，它本可以像硅谷的 TechCrunch 50 一样:专家和与会者被强烈地鼓励只提供积极的反馈。

要理解这种情况有多罕见，看看每天的博客帖子就知道了，这些帖子点名批评硅谷的初创公司——即使是最成功的——缺乏商业模式、重新设计主页或者只是普遍“差劲”或者阅读任何 TechCrunch 帖子上的评论。硅谷生态系统周围的每个人都是批评者，如果他们是匿名的，那就是恶毒的批评者。见鬼，迈克尔·阿灵顿被人唾弃，我职业生涯的一半都是从匿名的尖酸刻薄的憎恨者那里获利的。我们只写创业公司。我不喜欢这样，但这是我工作中不幸的现实。

这种对积极性的坚持很大程度上来自于以色列网络界的教父 Yossi Vardi。这是他在 KinnerNet 会议上的主要规则之一，他告诉人们，如果你不同意别人说的话，也许你就是问题所在。说得好，瓦迪先生。

令人惊奇的是人们是如何尊重规则的。Vardi 在以色列的网络企业家中唤起了恐惧、尊重和奉承的结合，所以当他要求人们友善时，他们实际上是在听。Techonomy 活动结束后，他向我走来，我有点担心他会对我大喊大叫，因为我说一家展示公司需要一个更好的标志，因为你无法破译它的名字。(如果你问我，我觉得这是消费者网络业务的一个问题。相反，我们开始了一场有趣的谈话，关于这种对积极的坚持。

简而言之，他希望营造一个舒适的环境，让那些容易被吓倒的聪明企业家不会被挤出这个行业。我能理解。我认为很多关于硅谷创业公司的批评往好里说是多余的，往坏里说就是没那么有趣。许多人试图将初创公司置于一个不可能的境地。他们毕竟是创业公司。他们应该做一些有风险的、看起来疯狂的、很容易失败的事情。如果不是，他们可能没有承担足够的风险。大多数懒惰的博主和评论者都明白了一件事:如果你想最大限度地提高你对一家创业公司的正确率，那就扔掉它们，因为大多数创业公司都会失败。相信一件事要难得多，说你相信更难。

也就是说，我不确定我理解了“建设性积极性”的概念。我认为很多创业公司都受益于诚实。诚实并不总是好的。拿我写的关于以色列的第一篇文章来说吧。当然，有些人认为道琼斯指数低估了回报。但是*没有人会否认以色列的回报在过去的八年里确实是不错的。*

你猜怎么着:他们在硅谷也不怎么样，这是人们一直写的关于[的事情。你永远听不到喧嚣，因为这是真的。风投、企业家和整个硅谷生态系统都明白这一点，知道这是一个问题，每个人都在悄悄地尝试各种方法来解决它。如果人们公开谈论以色列的低回报——就像他们在硅谷所做的那样——会有这样的喧嚣吗？或者人们会退缩、点头，并努力成为例外？](https://web.archive.org/web/20221005152526/http://www.beta.techcrunch.com/2009/04/01/another-dry-quarter-for-venture-exits/)

我花了很多时间和特拉维夫的人们谈论诚实的概念——这是以色列人应该以诚实而闻名的。从本质上说，人们都说以色列的一切都会点燃激情。好吧，这是一个公平的观点，可以理解。但是，政府、企业家和投资者不能指望美国主流博客和商业媒体进行报道，也不能指望我们写自己的创业公司时遵循同样的规则，即使是像脸书和 Twitter 这样的杰出公司。这有点像我说人们应该对我好，因为我是女人，不是吗？

但这正是瓦迪的世界观有意义的地方:他的目标不是让他的公司建立价值数十亿甚至数亿美元的帝国。更重要的是，他说期望他们这样是不公平的。他在活动结束后告诉我，以色列公司非常擅长开发技术，这样更大的公司、更大的市场和更多的资源就可以得到发展。他称它们为“番茄种子”——里面有很多，但它不会自己长成番茄植株。事实上，Vardi 有很多出场机会，并根据这个理论赚了钱。

当然，接下来的自然问题是，大型风险投资基金应该从以色列开始吗？我问过瓦迪，他说这是个好问题。他指出，鉴于资本总额较低且退出规模相对较小，大型风险资本不应该资助网络应用。同意。在我关于 Web 2.0 的书中，一个主题是在这一波公司浪潮中，大笔资金的重要性相对较低，这就是为什么你会看到天使投资人扮演了如此不同的角色，以及过去几年像首轮资本这样的强大种子基金的出现。

理想情况下，我认为以色列场景的积极和硅谷场景的诚实的结合对两个社区来说都是一个进步。但是，我同意 Vardi 的观点，那些仅仅致力于制造产品的公司不应该受到脸书级别的审查，而应该得到扶持。但是，这意味着作为一个主要为美国商业读者写作的人，我可能也不应该飞遍世界报道他们。简而言之，以色列企业家不能鱼与熊掌兼得。

就我个人而言，我仍然相信大事情可以从以色列发生，就像它们可以从任何地方发生一样。仅仅经过两周的搜寻，我已经发现了一些企业家在努力建立真正的、可行的公司，我希望在未来几年里看到这些公司。作为一个靠分析和提出令人不安的问题赚钱的人，我会像对待硅谷任何有潜力的公司一样对待这些公司。如果任何企业家不想要这样的报道:不要打电话给我开会。
