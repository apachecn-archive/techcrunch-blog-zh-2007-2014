# 移动网络插件 Brow.si 筹集 100 万美元，与 PrestaShop 合作帮助移动商务坚持下去

移动网站 add-on [Brow.si](https://web.archive.org/web/20221207082559/http://brow.si/) 背后的以色列初创公司 MySiteApp 已经从现有投资者 [thetime](https://web.archive.org/web/20221207082559/http://www.crunchbase.com/financial-organization/thetime) 和一些美国/以色列的天使投资人那里又筹集了 100 万美元，使该公司的总投资额达到 320 万美元。额外的资金将用于帮助该公司进一步整合第三方 CMS 和电子商务平台。

为此，Brow.si 今天还宣布通过与开源电子商务平台 PrestaShop 的合作赢得一个体面的合作伙伴。它遵循了与一系列 CMS 软件的现有集成，包括 Drupal、Joomla 和 Blogger，并且是 WordPress VIP 功能合作伙伴。

具体来说，与 PrestaShop 的合作有望成为移动商务的福音，因为它支持 Brow.si，可以轻松地将一系列参与功能引入移动电子商务—[brow . si](https://web.archive.org/web/20221207082559/https://beta.techcrunch.com/2013/05/21/brow-si/)的整个前提是，安装了该插件的移动网站能够提供与“本地”应用程序相媲美的功能，如社交共享、“稍后阅读”按钮、订阅和推送通知，在使用该插件的移动网站和 Brow.si Reader iOS 应用程序之间架起了一座桥梁。

使用 Brow.si 附加组件的 PrestaShop 支持的商店获得一个浮动菜单，该菜单以一致的方式向用户提供对该商家商店的所有页面上的 Brow.si 功能的访问。这包括全网站搜索、一个让购物者在多个社交网络上分享产品页面和其他商店内容的社交分享按钮、“保存以备后用”选项(口袋、可读性和 Brow.si 自己的阅读器)、“推送通知”和一个类似应用程序的移动购物车。

Brow.si 表示，在不久的将来，它将推出额外的功能，如定向优惠券、“热门交易”和产品推荐。

该公司解释说:“当从移动设备访问时，所有这些功能都可以在商店的任何地方使用，这代表了商家将注意力重新集中到他们商店的关键资产的新功能/方法，并使移动购物者更容易参与他们的产品/品牌，浏览商店并进行购买。”

或者更简单地说，这是关于反对放弃购物车，尤其是因为劣质的移动体验，以及通过降低分享产品页面的门槛来招揽更多业务。

为了帮助支持这一点，PrestaShop 商家可以获得包括参与度指标在内的分析，从而了解他们的移动客户和商店的表现。
