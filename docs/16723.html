谷歌地图引擎获得免费层和公共数据程序，以帮助政府，其他组织表面地图

今天，谷歌发布了免费的谷歌地图引擎，同时发布了小型商业工具Maps Engine Pro T1。免费层提供所有相同的功能，但数据访问配额较小，为“数十兆字节”,而完整版为数兆字节。

谷歌还推出了一个新的公共数据项目，允许组织注册一个免费的GME账户来创建地图并在网上发布。这可以让像拉脱维亚政府这样的组织将通常需要团队构建自定义工具才能呈现的数据地图呈现出来。

【T2

这个想法是提供一个新的投票区，交通路线，公共工程或任何其他类型的数据的公共地图，可以很容易地通过谷歌或其他工具进行搜索，并且可以以稳定的方式广泛传播。谷歌恰好在这方面做得很好，而许多组织和政府都不擅长。

任何组织都可以利用这个新项目，它不会像免费的谷歌地图引擎产品那样面临配额限制。唯一的规定是这些地图必须公开张贴。这满足了他们让这些地图易于搜索的需求，而谷歌需要让其地图产品成为世界上最完整的图片。谷歌自己只能收集这么多层次的数据，这个项目利用另一整群拥有独特、有价值数据的组织来添加到整体产品中。

T2】

今天，谷歌还宣布将与一些非常流行的地理信息系统软件整合，如 ESRI 和 QGIS 。它还与Safe Software 建立了合作关系，Safe Software是全球最大的空间可视化公司之一。他们的数据转换工具FME将获得完整的谷歌地图引擎集成。

联邦快递、美国国家铁路客运公司和NOAA等公司都使用地图引擎。谷歌最近发布了一个地图引擎API ，该API旨在允许公司导入数据并将其置于谷歌自己的地图层之上。谷歌地图引擎产品已经出现了大约两年，并且已经上市一年多了。

免费层的好处是显而易见的:它提供了完整产品的味道，以吸引新的企业客户。帮助政府和其他人发布公共地图的公共数据工具的好处有点微妙。是的，这些组织从谷歌的地图工作中获得了一个稳定而强大的平台，但他们也有可能向谷歌的整体地图平台贡献数十万个单独的数据集，使其更加占据主导地位。
