# 中国的山寨 iPads 没有人被愚弄…或者有吗？

[![](img/7e78bc700bd87c0eb9ef466b1de824f8.png "ipadchina")](https://web.archive.org/web/20221205092339/http://www.crunchgear.com/2010/04/26/chinas-knockoff-ipads-have-no-one-fooled-or-do-they/ipadchina/)

很肯定中国在盗版和非盗版商品之间有自己的游戏规则，但是这个故事仍然值得一笑。自从苹果公司表示将推迟 [iPad](https://web.archive.org/web/20221205092339/http://www.crunchgear.com/tag/ipad/) 的国际发布，以满足美国的需求——这种神奇而革命性的设备似乎正在下架——中国制造商已经用一个又一个 iPad 仿制品充斥了他们的国内市场。它们比真的要稍微便宜一些，但是重要的是它们现在已经上市了。不是善意的，而是善意的*发现了*。我喜欢这种毫无意义的感觉。

交易是路透社的普通人在深圳找到了一个特别的山寨 iPad，深圳是一个与香港隔街相望的快速发展的城市。在那里找到盗版产品的新奇感几乎不存在——该报告甚至提到了以 2 美元(没错，就是 2 美元)出售的盗版 Windows 7！

突发新闻:盗版商品在中国销售。

问题中的 iPad 仿制品看起来和真的一样，足以骗你一秒钟，当然，直到你打开它。这里没有 Mac OS X 的衍生产品，而是一个 Windows 版本。这大概意味着你可以做任何你能用标准 Windows 电脑做的事情。

问题变成了:今天支付大约 400 美元购买山寨 iPad，或者等到苹果在你的地盘发布，或者以预期的加价购买进口版本。

决定，决定。
