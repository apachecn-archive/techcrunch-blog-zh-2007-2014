# 秘密创业公司 Glassdoor.com 获得 300 万美元的 B 轮融资

Glassdoor.com 在由 Benchmark Capital 领投的一轮 B 轮融资中获得了 300 万美元。

对这家位于加州索萨利托的公司了解不多。首席执行官兼创始人罗伯特·霍曼(Robert Hohman)曾任 Hotwire.com 总裁，团队成员包括 Zillow 首席执行官理查德·巴顿(Richard Barton)和 Expedia 前高级管理人员蒂姆·贝斯(Tim Besse)。PEHub [暗示](https://web.archive.org/web/20221208073714/http://www.pehub.com/article/articledetail.php?articlepostid=11107)该公司是一家“关注工作场所就业状况的社交网络公司”，而 as Rent Bits [暗示](https://web.archive.org/web/20221208073714/http://www.rentbits.com/blog/glassdoor/glassdoorcom-a-stealth-service-for-real-estate)它可能与房地产有关。

该公司仍然处于秘密模式，除了说“除了我们认为它非常独特，建造起来会很有趣”之外，没有在他们的网站上给出任何提示该公司可能已经在努力将这个神秘的产品推向市场，他们的网站上说他们“预计在 2008 年初推出一些产品”，但仍然显示一个保留页面。
