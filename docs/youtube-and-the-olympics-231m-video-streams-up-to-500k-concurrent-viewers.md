# YouTube 和奥运会:2.31 亿视频流和高达 50 万的并发观众

关于 NBC 在美国的奥运报道质量，人们说得越少，[越好](https://web.archive.org/web/20221130175333/https://beta.techcrunch.com/2012/07/27/nbc-olympic-opening-ceremony/)，但就今年直播和录制的奥运视频数量而言，NBC 和 YouTube 之间的合作显然进行得很好。即使按照 YouTube 的标准，为 NBC 直播奥运会也是一项相当大的任务，该公司刚刚[发布了本月早些时候关于其奥运报道的新数据。在巅峰时期，这项服务同时推出了超过 50 万个直播流。YouTube 表示，在 NBCOlympics.com，观众总共观看了超过 1.59 亿个视频流，其中超过三分之一来自移动设备，超过 50%是高清视频。](https://web.archive.org/web/20221130175333/http://youtube-global.blogspot.com/2012/08/how-olympics-played-out-on-youtube.html)

YouTube 说，它在美国以及非洲和亚洲 64 个国家的用户总共观看了 2.31 亿个视频流。仅[国际奥委会的 YouTube 频道](https://web.archive.org/web/20221130175333/http://www.youtube.com/olympic)就负责其中的 7200 万次，而“[美国队”美国奥委会频道](https://web.archive.org/web/20221130175333/http://youtube.com/teamusa)的浏览量超过 675 万次(尽管这些浏览量中的大部分似乎来自[少数](https://web.archive.org/web/20221130175333/http://www.youtube.com/watch?v=fJlieHbnElk&feature=plcp) [精选](https://web.archive.org/web/20221130175333/http://www.youtube.com/watch?v=Mqj4kGXTQW8&feature=plcp) [视频](https://web.archive.org/web/20221130175333/http://www.youtube.com/watch?v=r95yVCUNI0U&feature=plcp)，相当一部分“美国队”上传的浏览量目前不到 1000 次)。

根据 YouTube 的说法，直播视频的质量也“比以往任何时候都好，在低缓冲和高帧率的基础上，质量提高了 7 倍。”YouTube 没有说今年的数字到底是什么，所以对这个数字持保留态度可能是个好主意..
