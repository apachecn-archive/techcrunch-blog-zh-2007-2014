# 招聘一名办公室经理

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/03/28/techcrunch-is-looking-for-an-executive-assistant/>

**为 TechCrunch 的编辑提供行政支持:**

–管理编辑的日历、费用、差旅和运输需求。

–让经理了解日历或差旅的变化。

–及时完成请求，注意细节。

**成为编辑人员和 AOL 之间的联络人:**

–回答与人力资源、会计和法律相关的问题。

**对费用有着永恒的爱:**

–及时完成自己和经理的费用。

–了解这个过程，这样当问题出现时，你可以向作者解释。

–了解什么是适当的费用，并及时批准报告。

–确保所有费用记入正确的部门，以保持平衡的会计报表。

**协助中断事件的操作:**

–能够解决与中断相关的各种问题。

–以专业的方式处理自己。

–在需要的地方填写。

–协助贵宾进出大楼。

**确保所有第三方供应商按时付款:**

–接收和整理收到的发票。

–与客户合作，设定第三方供应商的付款方式。

–提交发票并跟进以确保及时付款。

**处理自由职业者的月工资:**

–计算自由职业者的职位数量，并提交给 AOL 会计付款。

–保持合同数据库有序，便于参考和每月付款。

–确保支出保持在预算范围内。

**每周跟踪编辑分析:**

–为编辑团队提供每周分析。

–根据经理需要的数据编制报告。

**根据需要帮助整个编辑团队出差:**

–确保在活动期间与酒店一起组建编辑团队。

–确保及时为活动预订差旅。

–研究我们报道的活动的所有旅行选择，包括 CES、SXSW、E3 等。
