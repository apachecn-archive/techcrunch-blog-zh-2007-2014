# 肖恩·帕克现在在推特上说:“对不起，扎克”

肖恩·帕克的声音越来越大。

十年来，你一直听说*关于*他 Napster 和 Plaxo 的联合创始人，脸书的创始总裁，最近是 Spotify 的投资者(和倡导者)。在他的[会议](https://web.archive.org/web/20230203151502/https://techcrunch.com/2011/06/28/sean-parker-on-why-myspace-lost-to-facebook/)露面或采访中，他也没有手下留情。

但现在看来，我们将会更经常地直接听到他的声音:帕克刚刚开通了一个推特账户。

他的第一条推文，你可以在上面看到，不需要太多的解释([斯科特·布劳恩](https://web.archive.org/web/20230203151502/http://en.wikipedia.org/wiki/Scooter_Braun)是一位人才经理，实际上他以前在 TechCrunch Disrupt NYC 说过)。

扎克伯格可能不会太难过。例如，帕克最近在他的[脸书页面](https://web.archive.org/web/20230203151502/https://www.facebook.com/Sean)上发布了一些冗长的[帖子](https://web.archive.org/web/20230203151502/https://techcrunch.com/2011/09/22/facebook-founding-president-sean-parker-smart-lists-represent-a-change-in-philosophy/)(在撰写本文时，他已经有 30，651 名订户)。帕克负责通过一种新颖的股权结构给予扎克伯格对脸书的长期控制权。直觉告诉我他们关系还不错。

显然，帕克的推文是一个更大战略的一部分(据报道，他也正在写博客)。我的直觉是，他渴望通过不受媒体过滤的渠道分享他的想法(就像媒体采访一样)。

或者是他的经纪人让他这么做的。
