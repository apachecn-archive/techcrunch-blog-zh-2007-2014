# 多有面子(。com)的认可可能适合脸书移动 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/05/28/face-com-may-give-facebook-a-mobile-klik-but-ceo-tells-us-nothing-new-to-announce/>

当我们询问时，Face.com 的首席执行官对[传言](https://web.archive.org/web/20221006031624/http://www.techmeme.com/120528/p7)被脸书以高达 1 亿美元的价格收购不屑一顾。但是，随着社交网络的用户基础从桌面转移到其他地方，脸书的移动应用增加了面部识别技术，这可能会确保好友标记的继续。

事实上，大约 45%的 Face.com 应用程序 [KLIK](https://web.archive.org/web/20221006031624/http://itunes.apple.com/us/app/klik-by-face.com/id484990787?mt=8) 的用户最终会在脸书上分享他们的照片，这显示了移动面部识别可以有多受欢迎。

郑重声明，Face.com 人对自己的秘密守口如瓶。Face.com 的首席执行官吉尔·赫希(Gil Hirsch)直截了当地告诉 TechCrunch:“目前我们没有新的消息要宣布或分享。”

但是，尽管脸书一直在推动其移动产品的许多新奇增强功能(其[相机移动应用程序](https://web.archive.org/web/20221006031624/https://beta.techcrunch.com/2012/05/25/facebook-camera-app-overload/)是最新的)，仍然有数量惊人的功能尚未被该公司涵盖。

因此，也许是因为大自然厌恶真空，我们现在得到了该公司可能会收购或推出什么来弥补这一点的大量报道，包括今天的[Face.com 新闻报道，脸书正在寻求收购移动/个人电脑浏览器公司](https://web.archive.org/web/20221006031624/http://www.techmeme.com/120528/p7) [Opera](https://web.archive.org/web/20221006031624/http://www.pocket-lint.com/news/45795/facebook-browser-opera-software-buyout) 并雇佣前苹果硬件工程师[来开发自己的手机。(Btw Opera 也给出了类似的无新闻台词:“我读过的一切对我来说都是新闻，”一位 Opera 人士告诉我。)](https://web.archive.org/web/20221006031624/http://bits.blogs.nytimes.com/2012/05/27/facebook-tries-tries-again-on-a-smartphone/)

我们可能只是，很有可能，正处于脸书新闻泡沫的中心，我们读到的关于脸书和移动的一半可能永远不会实现——或者可能需要很长时间才能实现:据报道，从 11 月开始的 AllThingsD 帖子中的时间来看,《安卓杀手巴菲》仍然需要 6 到 12 个月的时间。这不是我们第一次听说 Face.com 在收购脸书。

但是，如果你咽下这一大粒盐，社交网络中有很大一部分人希望通过这些公司来增强其移动武器库，这些公司提供了脸书目前没有的功能，因此提供了让移动用户在社交网络上花更多时间的承诺——这是公司关心的事情。

目前，脸书广受欢迎的照片标签建议功能无法在手机上使用，只能在个人电脑上上传；一位发言人告诉我们，在手机上，用户可以通过输入标签选项的名称来进行标签。

与此同时，Face.com 提供面向个人电脑和移动设备的面部识别软件，关键是它覆盖了移动设备。

Face.com 有自己的 iPhone 手机应用程序 KLIK，测试版于 1 月 3 日在 T2 推出，本月早些时候在 T4 的应用程序商店推出 1.0 版本，展示了它的全部功能。

这些包括面部识别、面部友好的照片过滤器和基于位置的照片网络——所有这些服务你都可以在脸书现有的服务中自然看到。

Face.com 告诉我们，Klik 在头三天就有超过 100，000 次的应用下载——这是粘性和受欢迎程度的标志。

同样有趣的是，Face.com 还提供了一个 API 来将其技术集成到其他应用程序中。(一家公司[建议](https://web.archive.org/web/20221006031624/https://twitter.com/face/status/200577948724637696)进行整合:路径。)

随着脸书寻找更多方式来扩展其功能和其围墙花园之外的接触点，特别是因为 Face.com 很可能很快就会推出 Android 版本，API 功能可能会派上用场。

Opera，上周的收购传闻，为脸书提供了一个覆盖新领域的类似承诺。

在这种情况下，它是关于一个网络浏览器——正如[其他人报道的那样](https://web.archive.org/web/20221006031624/http://www.techmeme.com/120525/p28#a120525p28)，如果脸书要推出自己的移动平台，这将是一个必不可少的功能。

那将是在智能手机领域与谷歌的 Android 和苹果的 iOS 竞争。但拥有网络浏览功能也可以帮助脸书在低端功能手机领域取得更多进展——在这一领域，它已经通过 Facebook Zero 等服务和收购 [Snaptu](https://web.archive.org/web/20221006031624/http://paidcontent.org/2011/03/20/419-facebook-buys-feature-phone-developer-snaptu-for-up-to-70-million/) 来改善发展中市场目标用户的功能手机体验。

巧合的是，Opera 在 2 月 3 日于 T2 推出了 Opera Mini Next 浏览器，专门为功能手机用户提供“智能页面”社交媒体分享功能，将自己标榜为一家“社交移动”公司。

Opera 称其浏览器目前拥有约 2 亿用户，但如果脸书追求的是其社交功能，那么可能还有其他值得关注的用户。例如，社交移动浏览器公司 [Rockmelt](https://web.archive.org/web/20221006031624/http://www.rockmelt.com/) ，已经从 Andreessen-Horowitz、Accel、Khosla Ventures 和其他公司筹集了近 4000 万美元，目前提供了一款 [iOS 应用](https://web.archive.org/web/20221006031624/http://itunes.apple.com/us/app/rockmelt/id416256246?mt=8)。

与此同时，自从上周谣言爆发以来，Opera 的财富有了一点提升:今天它的[股价](https://web.archive.org/web/20221006031624/http://www.google.com/finance?q=FRA%3AOS3)似乎是一年来最高的，上涨了 24.4%，收于每股 5.70€。
