没错。我刚刚在iBaseball投了一个时速110英里的快球

SGN 肯定对这些类似Wii的iPhone游戏有所了解。他们的《我的T4》和《我的猫头鹰》的下载量已经超过了200万次。今晚他们发布了该系列的下一款游戏，iBaseball ( iTunes link )。

像其他游戏一样，它是免费的，你可以使用iPhone中的加速度计来控制游戏的进行。该应用程序包括掌声和游戏声音，击中时振动，并支持左手游戏。你也可以挑战你的朋友来玩，目前是异步的，但很快就会有直接的面对面游戏，CEO Shervin Pishevar 说。

更多游戏即将推出。
