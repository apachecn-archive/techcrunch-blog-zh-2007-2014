# imo.im 悄然打造一款稳固的多网络即时通讯应用

当我在二月份第一次报道 [imo.im](https://web.archive.org/web/20230118222818/https://imo.im/) 的时候，我写道这是[最好的即时消息](https://web.archive.org/web/20230118222818/http://techcrunch.com/2009/02/06/imoim-is-the-best-im-web-service-youve-never-heard-of/)网络应用之一，你可能从来没有听说过，而且很可能从那以后你就把它忘得一干二净了。老实说，我也是，但团队上周联系了我们，让我们知道在过去的几个月里，它已经为服务添加了一些有用的新功能，我们应该重新审视它。我们的目标是取悦他人，所以开始吧。

Imo.im 是一个多网络即时消息工具，这意味着您可以使用它登录多个消息服务，如 Windows Live Messenger / MSN、AIM / ICQ、MySpace、Yahoo Messenger、Jabber、Gtalk 甚至 Skype，并从浏览器中获得一个完整的联系人列表，并使用文本、语音或视频与列表中的人聊天。它还拥有一个基本的[桌面客户端](https://web.archive.org/web/20230118222818/https://imo.im/download.html)，不幸的是它仍然只支持 Windows。

最近，imo.im 在其支持的 im 服务列表中增加了脸书聊天，这大概是我第一次写这款应用时它唯一严重缺乏的服务。现在它也支持与你的脸书朋友进行聊天会话，它比以往任何时候都更接近于提供基于网络的即时通讯客户端的知名初创公司，如 Meebo 和 eBuddy Webmessenger。除了经典的 Skype 聊天功能，它还支持 Skype 聊天，如果我错了，请纠正我，我还没有看到任何其他基于网络的应用程序集成了 Skype 聊天功能(请注意，像 [fring](https://web.archive.org/web/20230118222818/http://www.fring.com/default.asp) 和 [Nimbuzz](https://web.archive.org/web/20230118222818/http://www.nimbuzz.com/en/) 这样的移动设备应用程序支持 Skype 聊天)。

imo.im 的另一个新功能是“广播”功能，允许用户向其他 imo.im 用户发送免费消息(例如，职位空缺、聊天邀请等。)和一个互补的照片共享服务，允许人们通过所有网络与其他用户共享图像。最后，这家初创公司正在试验一种[“白板”](https://web.archive.org/web/20230118222818/https://whiteboard.imo.im/)功能，这种功能使用户能够相互处理图表、绘图等。下一步:完善服务的搜索和聊天记录功能。

路线图上还有一个 iPhone 应用程序，该初创公司计划在未来几周内将其放在 App Store 上(我们将密切关注)。

该公司由[乔治·哈里克](https://web.archive.org/web/20230118222818/http://www.crunchbase.com/person/georges-harik)共同创立，他是谷歌首批 10 名员工之一，也是谷歌早期几款产品的经理。该公司声称，到目前为止，它已经吸引了 50 万用户试用其服务，尽管它仍处于查看徽标的 alpha 模式。
