# 谷歌在大学生最受欢迎雇主排行榜上超过脸书

在过去几个月左右的时间里，谷歌(Google)经历了一段艰难的时期，或多或少被媒体定位为一家面临顶级人才流失到脸书的危险的公司。尤其是在硅谷有史以来最糟糕的一次人才危机中，员工全面加薪 10%的消息被说成是绝望的表现。等一下，从什么时候开始，一家公司给员工加薪是一件让人不齿的事情了？

Aftercollege(一个帮助学生联系他们毕业后想去工作的公司的网站)的布莱恩·黑弗伦，首席运营官，与我们分享了一些关于美国其他地方如何看待硅谷当前人才军备竞赛的见解。

尽管事实上超过 12%的脸书人现在是由前谷歌员工组成的，但是流行的假设是错误的。根据希望与某些公司匹配的学生的 Aftercollege 数据集，谷歌仍然是雇主联系请求的首选，特别是在前 50 所学校的工程专业中。

> 两个月前，我的公司为我们的用户(希望开始职业生涯的大学生)推出了一项新功能，允许他们请求他们想为之工作的公司。
> 
> *然后，我们代表他们走出去，从这些公司中寻找机会。无论如何，虽然只有两个月，但当谈到学生更愿意为谁工作时，我们看到了一些非常明确的观点。谷歌遥遥领先，排名第一。脸书目前排名第七。”*

虽然与当前的媒体炒作不一致，但黑弗隆表明，受雇于谷歌的兴趣并未减弱。虽然第一名和第七名的差距令人震惊，但当缩小到顶尖学校工程专业的学生时，结果显示差距甚至更大。当数据通过顶级工程项目如柏克莱、史丹福、麻省理工、乔治亚理工、密西根等按学生要求排序时，脸书排名第 12，谷歌排名第 1。因此，也许谷歌人力资源可以深呼吸放松一下，至少目前是这样。
