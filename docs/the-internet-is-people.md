# 互联网是人 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2012/01/01/the-internet-is-people/>

在互联网上，人类和算法之间一直存在紧张关系。在早期，雅虎是一个人为策划的索引，记得吗？但是人类跟不上，算法接管了。如今，随着脸书、Twitter 和 Instagram 等无数移动应用的出现，人的因素的重要性再次上升。一切都是社会性的。今天的紧张是在社交和搜索之间——人类对计算机。只不过事情没那么简单。

互联网不仅仅是数十亿的链接页面、数据库和(越来越多的)移动应用。互联网是人。是你和我。一个网页的有趣程度取决于最后一次有人链接到它的时候。我们是创造所有数据的人，这些数据供给互联网。甚至谷歌的搜索引擎——理解这一切的终极算法——也根据我们的行为决定什么是重要的。我们链接和访问最多的页面往往会出现在搜索结果的顶部。

是的，社交威胁着搜索的主导地位，因为它用我们信任的人的链接取代了搜索引擎的算法，作为发现新信息的一种方式。但是社交分享变成了另一组输入算法的信号。谷歌可能无法完全访问脸书或 Twitter 上的所有分享，但这就是它推出 Google+的原因。社交流的价值就在数据里。

互联网让我们更聪明，但我们也让互联网更聪明。信息来回流动。正在出现的是一种分工，人类做我们更擅长做的事情，而互联网——作为每个人都可以使用的巨大的全球性计算资源——做它更擅长做的信息任务。

我们每天都在为互联网编程。但是这种关系是双向的。它也在给我们编程。风投 Bryce Roberts 称这种观念为“[可编程的人，](https://web.archive.org/web/20230204091457/http://bryce.vc/post/12600078312/bionic-software-and-programable-people)”，他定义为:

> 人类和计算机的这种相互作用增强了彼此的行动和理解。

亚马逊的[机械土耳其人](https://web.archive.org/web/20230204091457/https://www.mturk.com/mturk/welcome)是人类劳动被用作计算输入的最著名的例子。但是，最近，越来越多的服务涌现出来，这些服务本质上是利用互联网来有效地分配劳动力和其他资源。想想像 [Skillshare](https://web.archive.org/web/20230204091457/http://www.skillshare.com/) 、 [TaskRabbit](https://web.archive.org/web/20230204091457/http://www.taskrabbit.com/) 或 [Zaarly](https://web.archive.org/web/20230204091457/http://www.zaarly.com/) 这样的服务，它们都创造了对人类劳动力的新需求，并利用互联网作为路由机制来传播工作。甚至是点对点市场，如 Airbnb、Etsy 和 Kickstarter。布莱斯解释道:

> 在过去的十年里，像沃尔玛这样的零售商使用局域网来编程自动化他们的供应链和库存管理系统。我们开始看到同样的事情发生在可编程的人身上。除了今天，清单由时间、技能和可用任务组成，并广泛分布在开放的网络上。有时间和技能的人可以在 Skillslate、Zaarly 和 Task Rabbit 等服务上搜索可用的任务。对于更个性化的东西，像 Etsy 这样的服务可以直接将你与工匠联系起来，这些工匠有时间和技能来完成更定制的任务。

这些只是早期的例子。可编程人的概念超出了这些有限的情况。我们是一个[人的互联网](https://web.archive.org/web/20230204091457/http://cdixon.org/2011/12/19/an-internet-of-people/)。随着我们越来越多地使用互联网，使互联网平稳运行的算法变得越来越好。他们变得更聪明，因为我们明确地或通过我们的行动告诉他们应该关注什么。

一些未来学家认为有一天计算机会超越人类的智力——所谓的[奇点](https://web.archive.org/web/20230204091457/http://en.wikipedia.org/wiki/Technological_singularity)。如果这真的发生了，那将不是任何一台计算机，而是互联网本身——一个计算机网络——实现了这种超级智能。但是变得更聪明的不仅仅是电脑。也会是我们。唯一的问题是人机界面将会是什么。我们从屏幕和键盘开始。现在我们转向触摸和语音。这些机器能做的越多，就越像人类。反之亦然。
