# 随着新资金的到位，拼车服务 SideCar 准备在西雅图推出 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/19/sidecar-seattle/>

拼车不再只是旧金山的专利。紧随[获得 1000 万美元新资金](https://web.archive.org/web/20221207210038/https://beta.techcrunch.com/2012/10/10/sidecar-10m-new-markets/)， [SideCar](https://web.archive.org/web/20221207210038/http://side.cr/) 准备开始向新市场扩张。在今天早上发送给 SideCar 社区的电子邮件中，该公司宣布了在未来几周内进入西雅图的计划。

西雅图似乎是拼车的一个明智选择:这是一个大多数居民都有车的城市，但它不是那么分散或蔓延(比如像洛杉矶那样)，以至于不可能确保拼车覆盖率。当然，它拥有懂技术的民众，因为它是微软和亚马逊等科技巨头的所在地或附近。

为了准备在翡翠城的推出，SideCar 已经要求用户推荐西雅图的司机，他们可能有兴趣成为那里服务的一部分。推荐最终被选中为司机的朋友的用户将获得 100 美元的乘车积分，只要司机最终在 11 月底之前开始。这告诉我们，一个真正的推出 SideCar 在西雅图将发生在未来六周内，但它可能会启动得更快。

SideCar 也在[寻找那个城市的社区经理](https://web.archive.org/web/20221207210038/http://www.linkedin.com/jobs?viewJob=&jobId=3958403&srchIndex=0&trk=njsrch_hits&goback=%2Efjs_*1_*1_*1_I_us_*1_10_1_R_true_*1_*2_*2_*2_us%3A91_*2_mrkt_*2_*2&trk=from_simplyhired)。我已经要求 SideCar 提供更多的细节，但是还没有回音。

随着许多玩家试图扰乱旧金山的交通行业，拼车在一个新城市的引入正在发生。当然，优步是第一家，但最近它瞄准了拼车服务，在国内市场推出了成本更低的优步出租车服务。

拼车在旧金山变得如此流行，以至于 Zimride 的 Lyft 服务[都跟不上需求](https://web.archive.org/web/20221207210038/https://beta.techcrunch.com/2012/10/18/lyft-waitlist-cancellation-fee-etc/)。为了解决这个问题，Lyft 已经为新用户建立了一个等待名单，并引入了取消费用，以阻止用户在司机到达之前订购并取消。但是，当 Lyft 正努力跟上国内市场的需求时，SideCar 正准备向其他城市扩张。

进军西雅图之际，SideCar 和其他拼车服务也面临着国内的监管审查:加州公共事业委员会(California Public Utilities Commission)在 8 月份向 SideCar、Lyft 和 Tickengo 发出了停止通知。所有人都还在路上，据报道，所有人都在与 CPUC 讨论他们的服务如何融入其监管框架。随着当地政府关注这些新技术驱动的服务，SideCar 可能会吸引其他市场的审查。

但它必须首先在新市场推出。雇佣司机和当地社区经理将是实现这一目标的第一步。
