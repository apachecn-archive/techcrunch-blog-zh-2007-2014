# 存储提供商 Nutanix 筹集了 3300 万美元，并显示出准备 IPO 的迹象

Nutanix 已经在 C 轮融资中筹集了 3300 万美元。这笔投资来自投资者光速风险投资公司和科斯拉风险投资公司，以及新的投资者电池风险投资公司和高盛。

这轮融资为 Nutanix 提供了总计 7100 万美元的资金，并为该公司可能的 IPO 做好了准备。

Nutanix 涉足软件定义的存储领域，这是一个相对较新的类别，反映了整个存储市场的巨大变化。虚拟化如此盛行，以至于需要新形式的存储来存储来自虚拟机的数据。

企业客户长期以来一直将存储与服务器分开——在许多方面都是不可见的，由不同的团队管理，按需分配存储。Nutanix 采取了不同的方法。它将该存储包装到商用 x86 服务器中，有助于减少大型存储连接网络(SAN)和网络连接存储(NAS)环境所需的空间。

随着深度虚拟化环境中数据量的增加，公司寻求整合其数据中心，同时提高所需的速度和效率，Nutanix 正在融合浪潮中前行。

计算、网络和存储将越来越多地集成到盒子本身中。新的工作负载需要它。随着负载在巨大的网络上被定向，数据越来越多地从一个虚拟机移动到另一个虚拟机。

NetApp 和 EMC 等公司的 SAN 和 NAS 系统有其坚定的追随者，但虚拟化环境的新要求为 Nutanix 等提供商留下了一个开放的市场。

首席执行官 Dheeraj Pandey 昨天在接受采访时表示，该公司可以选择像苹果公司一样包装自己的技术，或者通过原始设备制造商(OEM)提供技术。他们选择走苹果路线。这意味着建立自己的销售网络。据 Pandey 称，该战略使其成为过去十年中增长最快的基础设施初创公司，售出了 150 个系统，部署了 600 台服务器，存储容量超过 3.3。

这种增长部分是由于 Nutanix 与 [Fusion-io](https://web.archive.org/web/20230131171058/http://fusionio.com/) 的关系，Fusion-io 是一个非常受欢迎的服务器附加闪存存储提供商，集成到 Nutanix 系统中。

Fusion-io 于去年首次公开募股。潘迪说，你可以从字里行间体会到 Nutanix 首次公开募股的含义。雇佣高盛说明了一切。

Nutanix 仍然面临着一场艰难的战斗。它将利用这 3300 万美元的资金建立一个销售和营销渠道网络。公司会需要的。Nutanix 将与 IBM 和 EMC 等价值数十亿美元的公司竞争。这些公司对新的存储市场和将这些产品推向市场的渠道有自己的计划。
