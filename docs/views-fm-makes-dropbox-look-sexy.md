# Views.fm 让 Dropbox 看起来很性感 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/03/23/views-fm-makes-dropbox-look-sexy/>

[![](img/f786c3671f9b027ea95d6c697ca12b95.png "Picture 3")](https://web.archive.org/web/20230203004927/https://techcrunch.com/wp-content/uploads/2011/03/picture-37.png)

网络是分享的媒介。无论是照片、视频、音乐、链接、代码，我们都在分享我们生产和消费的内容。社交图的兴起让人们更加关注我们分享的方式以及与谁分享。毕竟，分享是关爱，amirite 朋友？

[在最近的一篇文章](https://web.archive.org/web/20230203004927/https://techcrunch.com/2011/03/21/sendoid-finally-sharing-big-files-isnt-a-huge-pain/)中，我谈到了在内容共享领域的一个特定(也是熟悉的)部分仍然存在的一些问题:文件共享。在帖子的最后，我提到了流行的云存储、同步和文件共享初创公司 [Dropbox](https://web.archive.org/web/20230203004927/http://www.dropbox.com/) ，这是我经常使用的一项服务。Dropbox 如此强大的部分原因是它的简单性——你下载该实用程序，创建一个帐户，就可以轻松地在虚拟云文件夹中共享所有电子文件，与朋友和同事合作，并在设备和硬盘之间进行同步。嘣。

尽管 Dropbox 很快成为硅谷的宠儿，并在 2010 年 1 月底突破了 400 万用户大关，但 Dropbox 的部分共享体验仍有欠缺。(不可否认，就像其他网站一样。)例如，共享整个 Dropbox 文件夹仍然很困难，除非这是你的公共文件夹。共享选项是有限的，该工具的照片显示也没有一个很好的界面。

进入: [Views.fm](https://web.archive.org/web/20230203004927/http://views.fm/) ，这是一个可以快速优雅地展示 Dropbox 内容的浏览器。Views.fm 允许您创建公共或安全的私人共享文件夹，这些文件夹会自动组织在一个易于扫描的缩略图视图中。

Views.fm 的创建者，[戈登·西普拉克](https://web.archive.org/web/20230203004927/http://grdn.cc/)和[泰勒·洛夫](https://web.archive.org/web/20230203004927/http://tylr.org/)，都是设计师和工程师(西普拉克目前是 [8tracks](https://web.archive.org/web/20230203004927/http://www.crunchbase.com/company/8tracks) 的创意总监，洛夫是 [Tumblr](https://web.archive.org/web/20230203004927/http://www.crunchbase.com/company/tumblr) 的高级开发人员)，都是自由职业者和媒体机构的雇员。Cieplak 说，在这样做的过程中，他们已经非常熟悉可用于共享媒体的工具，特别是在专业环境中:“Views.fm 是我们努力使文件共享过程变得更简单的工具”。

两人都承认是 Dropbox 的普通用户和忠实粉丝，但他们表示，他们认为该工具是为“傻瓜协作”而优化的，不允许向客户或朋友和家人进行高级演示。因此，他们在 Dropbox API 上构建了 Views.fm，以利用这样一个事实:如果你是 Dropbox 用户，你的文件已经在它的云中进行了组织。这使得文件共享既快又容易。

Views.fm 为您提供了公开共享您的文件的功能，即允许任何拥有该文件夹链接的人浏览和下载共享文件夹，或者私下共享，即只有您通过电子邮件邀请的人才能访问您的文件夹。

增强 Dropbox 体验，私人共享还使您能够查看和编辑谁有权访问您的共享列表，并提供对文件夹本身以及单个文件的评论能力。因此，这意味着你可以与 80 个人共享一个文件夹，而不必让每个人都知道他或她不是唯一与你共享文件的人。

Cieplak 说，大多数云服务都迎合一种类型的媒体(比如视频的 Vimeo，音乐的 Soundcloud，照片的 Flickr)，这对于最终的展示和用户消费来说是很好的，但对于迭代来说就不那么好了。因此，Views.fm 为各种媒体提供了独特的布局，无论是音频文件、音频文件夹、图像还是常规文件。你可以在右边看到受 [Muxtape](https://web.archive.org/web/20230203004927/http://en.wikipedia.org/wiki/Muxtape) 启发的音频文件界面。

Views.fm 让你很容易把它的网站当成作品集，因为你所要做的就是分享一个装满图片的文件夹，剩下的事情就交给它了。Views 实际上是为了让有创造力的人(Cieplak 特别提到了许多报酬过低的承包商)在向客户进行最终演示时拥有一些优雅和优势，以及快速迭代的协作工具。

目前，Views.fm 仅适用于 Dropbox，但创作者表示，他们希望尽快添加尽可能多的服务。但是，此时此刻，他们都还在为各自的雇主全职工作。

最后，看来 Views.fm 为 Dropbox 观看体验所做的优化可能只会对共享上瘾者或云共享服务的习惯用户真正有用。对于其他人来说，这可能只是一个性感的 Dropbox 皮肤。没有人说 Dropbox 不能简单地自己添加一些功能，或者收购初创公司为它们进行集成。

目前，Views.fm 是完全免费的，创始人说他们计划在可预见的未来保持这种状态。但是，在没有收入模式的情况下，他们可能还不应该放弃他们的日常工作。
