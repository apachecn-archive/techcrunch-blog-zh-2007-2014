# 黑客篡改肯尼亚警方网站以纪念马克·扎克伯格

[![](img/1a494d370da725ce3b247e868518ea43.png)](https://web.archive.org/web/20221207180922/https://beta.techcrunch.com/wp-content/uploads/2011/01/kenya-hack.png)

据肯尼亚内罗毕的商业记者拉里·马多沃[报道，一名黑客显然已经破坏了肯尼亚执法机构的网站(](https://web.archive.org/web/20221207180922/http://twitter.com/#!/larrymadowo) [kenyapolice.go.ke](https://web.archive.org/web/20221207180922/http://www.kenyapolice.go.ke/) )。

果然，黑客这么做似乎是为了向脸书首席执行官马克·扎克伯格致敬。

网站主页上明显无意的欢迎信息(网站的其余部分似乎运行良好)目前为:

> “上车后，我满脑子都是扎克伯格！！！这是给你的，马克！"
