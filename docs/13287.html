报道:超过24%的排名前10，000的网站现在使用脸书的官方插件

根据网站监控服务 Pingdom 的一项新研究，全球排名前10，000的网站中有24.3%(据Alexa报道)在其主页上有某种形式的官方脸书整合。这已经是一个相当可观的数字了，但是一旦你把脸书的基本链接也算进去，这个数字会上升到49.3%。该公司的官方按钮出现在7.3%的网站上。

正如Pingdom所说，脸书显然正在成为“网络DNA的一部分”，并且远远领先于其竞争对手。不过，值得注意的是，整合脸书的网站数量在过去一年中似乎保持相对稳定。例如，几乎就在一年前，bright edge报告称，它在网络排名前10，000的网站中，发现了约47%的脸书整合。

Twitter、Google+和LinkedIn

在Pingdom的研究中， Twitter 只出现在排名前10，000的10%的主页上。大约4.3%的网站使用Twitter的官方分享按钮。尽管这些数字相对较低，但值得注意的是，41.7%的网站有Twitter链接——这个数字与脸书没有太大区别。

至于谷歌的社交网络 Google+ ，取决于你问的是谁，它要么是活跃的，要么是死的——Pingdrom报告说，它在主页集成方面实际上做得相当好。在这项研究中，超过13%的网站在其主页上使用谷歌的+1按钮，12.3%的网站包含plus.google.com的链接。然而，正如Pingdom正确指出的那样，许多网站所有者整合Google+来提高他们的搜索引擎排名，“这可能会夸大数字，因为从网站所有者的角度来看，他们不一定与Google+相关，而是被视为影响SERP的一种手段。”

另一家社交网站LinkedIn远远落后于竞争对手。在这里研究的网站中，只有0.6%在其主页上整合了这一功能，0.3%使用了社交网络的官方分享按钮。即使包括基本链接，整合LinkedIn的网站总数也只有3.9%。

算上各种链接和官方小部件，下面是Pingdom的最终计数:

  1. 脸书 : 49.3%
  2. 推特 : 41.7%
  3. 谷歌+ : 21.5%
  4. 领英网:3.9%
