# Debian 将作为谷歌计算引擎的默认操作系统

谷歌将 Debian 引入谷歌计算引擎，并使其成为开发者使用该服务的默认操作系统。谷歌将同时支持本周发布的 Debian 6.0 和 7.0。

谷歌让 Debian 成为默认操作系统有一些非常明显的原因。首先，它是免费的，云分析师和 [Rishidot Research](https://web.archive.org/web/20221004203433/http://rishidot.com/) 的创始人 Krishnan Subramanian 说。“有了 Ubuntu 和 Red Hat，谷歌不得不与那些想自己赚钱的厂商打交道，”他说。此外，Debian 拥有庞大的客户群。这也符合谷歌的极客文化。

在关于该声明的博客文章中，谷歌引用了 Debian 7.0“喘息”版本的改进。它强化了安全性、更好的 32 位和 64 位兼容性，并解决了社区反馈问题。

谷歌表示，它将评估其他操作系统，它可以启用谷歌计算引擎。

值得注意的是，谷歌计算引擎仅[对 400 美元黄金支持包的订户开放](https://web.archive.org/web/20221004203433/https://beta.techcrunch.com/2013/04/04/google-opens-up-compute-engine-to-all-developers-who-buy-its-400month-gold-support-package-drops-instance-prices-by-4/)。

这一切看起来都像是为下周的谷歌 I/O 活动做准备，预计届时将发布谷歌的云计算战略。

Debian 与其他基于 Linux 的操作系统竞争，如 Ubuntu、Mint 和 Fedora。根据 [DistroWatch](https://web.archive.org/web/20221004203433/http://distrowatch.com/) 的统计，Debian 的页面点击量排名第五。薄荷是第一名。
