# 花旗集团加入安德森·霍洛维茨基金，支持欺诈分析公司银尾系统公司

[Silver Tail Systems](https://web.archive.org/web/20230205013038/http://www.silvertailsystems.com/) ，一家检测和预防在线欺诈的预测分析提供商，今天早上宣布，它已经从[花旗集团](https://web.archive.org/web/20230205013038/http://www.crunchbase.com/financial-organization/citi-ventures)[的子公司](https://web.archive.org/web/20230205013038/http://www.crunchbase.com/financial-organization/citigroup)花旗风险投资获得了额外的[轮融资](https://web.archive.org/web/20230205013038/http://www.crunchbase.com/company/silver-tail-systems)。

这笔投资的规模尚未披露，是之前宣布的由 Andreessen Horowitz 牵头的 2000 万美元 B 轮融资的一部分。

Silver Tail Systems 提供内部和 SaaS 解决方案，以保护拥有关键信息的在线资产(如电子商务、金融、政府和面向消费者的网站发布者和所有者)的运营商免受在线欺诈和滥用。

该公司表示，目前每年监控超过 7.5 亿用户和超过 18 亿次网络会话。
