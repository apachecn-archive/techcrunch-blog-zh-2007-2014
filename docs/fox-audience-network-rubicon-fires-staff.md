# 为了准备出售给 Rubicon 项目，福克斯观众网络解雇了一半员工

![](img/c4f490ea289658b453169f7a85891628.png)

新闻集团即将把福克斯互动媒体旗下的另一项数字业务[福克斯观众网络](https://web.archive.org/web/20221007010550/http://www.foxaudiencenetwork.com/)(范)转让给洛杉矶的广告优化初创公司 [The Rubicon Project](https://web.archive.org/web/20221007010550/http://www.rubiconproject.com/) 。据直接了解该交易的消息人士称，该交易尚未签署，仍有可能破裂，但两家公司正处于谈判的最后阶段。(在过去的一个月里，谈判一直在升温。为了准备出售，或继续作为一个独立的业务，其 300 名员工中约有一半昨天被解雇，其中大多数是销售人员。Rubicon 对广告技术更感兴趣。

如果与 Rubicon 的拟议交易获得通过，它将获得某些资产，包括范的广告服务器技术，其名为 [MyAds](https://web.archive.org/web/20221007010550/https://www.myads.com/) 的自助式横幅广告平台，以及大约 100 名员工来帮助运营这些业务。作为回报，新闻集团将获得 Rubicon 大约 20%的股份。(当福克斯互动媒体[出售烂番茄](https://web.archive.org/web/20221007010550/https://beta.techcrunch.com/2010/01/04/rotten-tomatoes-flixster/)时，它与 Flixster 进行了类似的资产换股权交易。考虑到 Rubicon 过去的公开姿态是[广告服务器已死](https://web.archive.org/web/20221007010550/https://beta.techcrunch.com/2010/02/19/the-rubicon-project/)，广告服务器技术是这里的关键资产之一这一事实尤其值得注意。

范还经营着互联网上最大的广告网络之一。然而，广告网络上大约一半的广告印象是由 MySpace 产生的。这是否会成为交易的一部分是最后要确定的项目之一。不管怎样，MySpace 都不太可能继续作为第三方网络运行。

这句话在范的墙上挂了很久了。新闻集团从去年三月开始收购。六月，进行了一轮小规模的裁员，随后，范的总裁亚当·贝恩离职，成为推特的首席营收官。当包括 Rubicon 在内的追求者对其感兴趣时，新闻集团正计划将范并入 MySpace。

Rubicon 是我于 2007 年创办的一家广告技术公司。根据点击量、浏览量和其他指标，帮助广告商确定在哪些广告网络和网站上投放展示广告。它的最终目标是建立一个谷歌旗下 DoubelClick 的 DART 的替代品，FAN 建立的一些技术可以帮助它实现这一目标。这就是为什么它不需要所有的销售人员。一些公司可能会转向残余广告和其他低质量广告，但 Rubicon 主要是一个技术游戏。该公司已经筹集了总计 4200 万美元的风险资本。
