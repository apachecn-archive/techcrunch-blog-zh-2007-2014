# PS 越狱实现了它的承诺:越狱你的 PS3

仅仅使用一个 USB 驱动器和一个软件漏洞，PS 越狱显然做了许多人之前尝试过的事情:允许未经授权的软件——包括备份——在 PS3 上运行。

这个漏洞的美妙之处在于它让 PS3 像一个调试控制台一样运行游戏。这意味着您可以从内置硬盘或通过 USB 从外置硬盘运行合法的备份。这显然是 neta-o-rama 的诡计，也是自从 Firefox 还很酷的时候 PS3 用户就一直想要的。

最棒的是？它可以在任何固件版本上运行，并且只有当 magic USB 驱动器插入端口时，这意味着使用它不会使您的保修无效。

不幸的是， [PS 越狱](https://web.archive.org/web/20221006134549/http://www.psjailbreak.com/)网站关闭了，所以我无法确认你是否需要从他们那里购买 USB 驱动器，或者你是否可以将一些文件解压到你自己的驱动器上。与此同时，你可以在这篇文章底部的视频中看到备份的运行——以及一位友好的澳大利亚同胞的一些评论。

在这种情况下，人们通常会持怀疑态度，因为许多 mod-chip 未能给 PS3 带来备份和自制程序，但似乎有越来越多的[证据](https://web.archive.org/web/20221006134549/http://twitter.com/xorloser/status/21554119576)表明这款设备确实有效。

到目前为止，唯一的警告似乎是你不能备份蓝光/DVD 电影，或 PSX/PS2 游戏。对我来说听起来很不错。
