# Bug.gd:协作搜索和 Bug 解决方案

被称为“人类对抗计算机的最后希望”的 [Bug.gd](https://web.archive.org/web/20221006005340/http://www.bug.gd/) 旨在建立一个计算机 Bug 的协作搜索和答案数据库。

这个想法很简单，并且解决了一个实际问题。用户看到一条错误消息，他们搜索 bug.gd，看看是否有人解决了这个问题。如果有人有一个伟大的解决方案，如果没有人找到一个解决方案，那么不那么伟大；然而，Bug.gd 认为，当他们无法提供解决方案时，用户最终会找到一个。用户会在 48 小时内收到一封提醒电子邮件，请求他们发布解决方案(假设他们找到了一个)，以便其他人可以从这一知识中受益，从而提供一个关于问题和解决方案的流畅且定期更新的协作数据库。

这项服务是免费使用的，没有广告。Bug.gd 希望从公司内部网销售 Bug.gd 工具中获得收入，这些工具将提供一个集中的内部 bug 修复数据库。用于提交和查询 bug.gd 错误数据库的 API 目前正在构建中。

这是一个可行的简单想法，尽管结果真的取决于用户是否足够慷慨地分享错误修复。Bug.gd 已经植入了 60，000 条错误信息和来自微软的解决方案(这本身就是一个可怕的数字)，并且每天都在缓慢地添加新的解决方案。
