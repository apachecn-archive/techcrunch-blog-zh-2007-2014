# 用于项目管理的 Teambox 获得了€14 万英镑的资助

协作项目管理平台 Teambox 是另一个西班牙本土的项目，旨在与类似于 [Basecamp](https://web.archive.org/web/20230307053600/http://basecamphq.com/ "Basecamp") 的公司在国际范围内竞争，整合了一个类似 Yammer 的社交模式。这家初创公司最近通过大型国际商业天使网络 [SeedRocket](https://web.archive.org/web/20230307053600/http://www.seedrocket.com/ "SeedRocket") Investors Day 和 [Keiretsu Forum](https://web.archive.org/web/20230307053600/http://www.keiretsuforum.com/frontend/index.aspx "Keiretsu Forum") Barcelona 获得了€14 万英镑的小轮融资。资金主要来自 Infojobs 的前首席技术官 Albert Feliu、前 Grupo Intercom、Abiquo 的首席执行官 Diego Mariñ、SeedRocket 的导师和私人投资者 Jordi Priu。

Teambox 正在采取一种类似 WordPress 的部署方式。他们提供了一个开源平台，以及一个托管和白标解决方案，这可以使他们在与其他一些知名玩家的竞争中处于有利地位。

他们的定价模式很简单:基本版免费，高级版每月 20-150€不等。他们还对定制安装收费。Teambox 确信他们的市场是全球性的，因为他们推出了英语、法语、西班牙语，很快还会推出德语、俄语、意大利语甚至中文。凭借他们希望在 2010 年获得的 140.000 英镑和 [NEOTEC](https://web.archive.org/web/20230307053600/http://www.cdti.es/index.asp?MP=7&MS=24&MN=3 "NEOTEC") 和 [Plan Avanza](https://web.archive.org/web/20230307053600/http://www.planavanza.es/LineasEstrategicas/AreasDeActuacion/EjeDesarrolloDelSectorTIC/Paginas/desarrollo.aspx) 资金，Teambox 将在技术、产品开发以及有希望的战略性国际推广方面取得增长。其实，敬请期待，更多的产品正在酝酿中。
