# Makerbot 的 Bre Pettis:“未来已经到来”

我们很好地了解了新的 Makerbot 复制器，更重要的是，我们与创始人 Bre Pettis 坐下来讨论了未来。他拿的？未来就在眼前，3D 打印是将改变世界的事物之一。

佩蒂斯在布鲁克林的黑客空间 [NYC Resistor](https://web.archive.org/web/20230204093308/http://www.nycresistor.com/) 建造了第一个 Makerbot。他意识到这种潜力在某种意义上是无限的，在销售了数千台之后，Makerbot 现在正在销售完全组装的新型 3D 打印机，可以制造“一条面包大小”的物体。

佩蒂斯指出，未来懂 3D 打印的孩子会比没有这方面经验的孩子占优势。他认为 Makerbot 是一长串越来越便宜的设备中的第一个，这些设备将使我们脱离与现代小工具销售相关的漫长物流链。公共生产手段已经到达媒体。现在又到了制造业。
