R.I.P. Microsoft Zune，2006-2011 tech crunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/03/14/r-i-p-microsoft-zune-2006-2011/


彭博正在报道微软最终决定正式终止其 Zune 媒体播放器产品线。“一位知情人士”告诉他们，微软不会推出任何新硬件，今后将专注于将Zune功能与Windows Phone 7平台集成。

并不完全出乎意料；自2009年年中发布 Zune HD 以来，Zune硬件没有改变，尽管它已经接受了几次重大的软件升级。这篇文章已经挂在墙上很长时间了，但是微软是否会加倍(再次)或者减少他们的损失还不清楚。

让我们沿着记忆之路快速旅行一下。在西雅图一家潜水酒吧里，一名微软员工不明智地(或者说是明智地)将一个Zune放在桌子上，周围都是博客作者，之后，CrunchGear是世界上第一个看到Zune的公司之一。我们狂热地报道了它，因为它似乎是iPod的黑马竞争对手，实际上可能有机会。独特的(但笨拙的)共享特征，棕色的外壳(尽管遭到嘲笑，但却是他们最受欢迎的Zune)和迟钝的广告都暗示着……嗯，一些新的有趣的东西。

各种更新将我们带到了今天的Zune HD和Zune软件，这两个软件都必须称赞其真正原创和有效的界面。我仍然使用Zune HD，它仍然得到称赞。但是销售从来不是微软所希望的，也许是因为微软对应用程序和附加功能的骑墙态度。虽然iPod touch显然被宣传为一部没有手机的iPhone，但Zune被宣传为完全是关于音乐的——除了它完全是关于云的——除了它有应用程序——除了它没有时钟——除了它在Windows Phone 7中——除了它在Kin中……等等。这并不完全是一个沉重的负担(它已经卖出了几百万，流媒体服务也很受欢迎)，但面对不屈不挠的iPod，这并不算什么。

这是一个悲伤的故事，这是微软无法找到合适子弹的众多案例之一。然而，随着选择与WP7整合，以及WP7即将通过诺基亚推出，Zune服务可能会继续存在，尽管几乎可以肯定是作为一个更大的媒体和数据云服务的一部分(可能是他们为雇佣的项目【Ventura】)。他们在流媒体和订阅音乐方面的前瞻性探索(现在反映在Rdio和Spotify等云音乐服务中)也没有被遗忘。Zune DNA，和hardy DNA，将会在Metro界面和其他任何薄的、剪裁过的无衬线文本中存在。

人们可以说这是他们一直以来的计划，作为他们“三屏”环绕服务的一部分，但事实是这一直是B计划，现在Zune未能从媒体播放器方面削弱苹果的份额，他们将尝试不同的策略。至于玩家呢？没有新的硬件意味着Zune HD是其同类产品中的最后一款，尽管它可能会销售更长时间。见鬼，179美元，我仍然认为32GB的版本很划算。现在你知道不会有Zune HD 2，在某种程度上，原版是经得起未来考验的。

再见了，祖恩。这是一次漫长的棕色旅行。

更新 : Zune的硬件业务开发人员发表了评论，淡化了这份报告，但很大程度上证实了我们在这里所写的，Zune作为一个独立的播放器和品牌正在融入WP7。
