# 数据库管理员！红门想把你们中的一个送上太空

我们很少在 TechCrunch 上宣传一家公司的营销活动，但话说回来，这是一场奖金丰厚的竞赛。如果您是数据库管理员，您最好注意以下几点。英国软件公司 [Red Gate](https://web.archive.org/web/20230204143359/http://www.crunchbase.com/company/red-gate-software) 奖励一名幸运的 ~~SOB~~ DBA(无论他或她是在银行、互联网创业公司还是汽水经销商工作)一个进入太空的商业航班座位，由[太空冒险](https://web.archive.org/web/20230204143359/http://www.spaceadventures.com/)提供。

太空飞行。你要从这里的[开始](https://web.archive.org/web/20230204143359/https://www.dbainspace.com/quiz)。

不过，注册或发微博说些搞笑的事情不会有什么影响。来自[新闻稿](https://web.archive.org/web/20230204143359/http://www.businesswire.com/news/home/20111012005926/en/Send-DBA-Space):

> 但首先，数据库管理员必须解开一个棘手的情节，其中涉及变形火星人、处于危险中的宠物、外星人的身体部位和绑架他们中的一员——所有这些都在为期五周的一系列时髦(坦率地说是超现实的)60 年代风格的太空黑色电影中展开。

我知道，听起来工作量很大，但是奖品是飞向太空。

15 名决赛选手将从那些对上述视频中的测验提供最佳答案以及来自 dataspace 的最成熟推文中选出。获胜的数据库管理员将由公众投票选举产生。

那么为什么 Red Gate 要为数据库管理员做这些呢？收到。浆糊。

> 为什么是 DBA？因为他们是你从未听说过的最重要的人。政治家或金融家并没有让这个世界保持正常运转。让这一切成为现实的是谦逊的数据库管理员——数据的主人，他能让医疗记录在眨眼间被调出，保持交通顺畅运行，为数十亿人管理电网下的数据，并提供即时获取新闻、音乐、电话、金钱和无尽娱乐的途径。
