# 脸书在 2500 部 Java 手机上推出免费移动应用

拥有超过 2 . 5 亿移动用户的脸书知道，随着网络变得越来越移动，它必须安装在地球上的每一部手机上，才能继续驾驭社交浪潮。这意味着不仅仅是在 iPhone 和 Android 手机上。

今天，脸书正在解决 2500 部功能手机可以下载 Java 应用的问题，为每部手机开发了一款名为[脸书的新应用。它包括一个轻量级的新闻源，一个脸书信息收件箱，以及一个直接从手机上传和分享照片的方式。你可以在 m.facebook.com 的](https://web.archive.org/web/20230203070912/https://www.facebook.com/notes/facebook-mobile/facebook-for-every-phone/200217473360613)[或者手机应用商店下载，比如 GetJar、Appia 和 Mobile Weaver。](https://web.archive.org/web/20230203070912/http://m.facebook.com/)

为了让人们试用，脸书将为该应用使用的前 90 天的数据付费。(脸书将其设计为数据高效型)。

这个新应用看起来像是脸书在三月份收购 Snaptu 的直接副产品。这与脸书的其他项目保持一致，以确保脸书在世界各地的所有设备上都可以访问和快速访问，如 Facebook Zero(用于手机上的纯文本访问)或其现已解散的脸书 Lite 实验。
