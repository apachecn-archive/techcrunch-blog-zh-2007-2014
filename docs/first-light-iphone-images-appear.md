# 第一个“光”iPhone 图像出现

我们出发了:第一批“廉价”iPhone 的照片开始出现——不是特指 iPhone 5，而是 iPhone 4 的轻便版——最新的泄露来自越南的一个论坛。

我们通常不会相信这种事情，但是 [Tinhte 在过去一直是正确的](https://web.archive.org/web/20230204172352/http://www.tuaw.com/2011/04/19/white-iphone-test-version-of-ios-tinhte-claims-both-in-video/)，所以它至少值得一目了然。

我们在这里看到的是没有前后玻璃面板的 iPhone 4。相反，有两个半透明的塑料片和边缘周围的金属带。这将大大降低 iPhone 的制造成本。

在 2011 年的大部分时间里，我们一直听到“更便宜”的 iPhone 的传言，伟大的预言家[本人](https://web.archive.org/web/20230204172352/https://techcrunch.com/2011/06/27/iphone-5-iphone-lite/)认为，iPhone“精简版”即将推出，可能定价为预付费、发展中市场水平。毕竟，蒂姆·库克说过他不想让 iPhone“只为富人服务”。

更新——正如 iawa2k 指出的，这可能只是一个简单的转换套件。然而，我发现看似粗糙或斑驳的表面是这次“泄漏”中最有趣的部分是真的吗？谁知道呢，但是我们不能忽略潜在的候选人。
