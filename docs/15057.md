# 传言谷歌也在生产智能手表 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/03/22/google-rumored-to-be-making-a-smartwatch-too/>

从苹果入侵 Watch 的传闻和谷歌眼镜出现的消息来看，谷歌似乎真的在开发自己的智能手表，以便与联网的安卓设备搭配使用。据《金融时报》报道，谷歌的安卓部门将是负责该设备的团队，而不是负责谷歌眼镜开发的 X 实验室部门。

可穿戴计算机市场正在迅速升温。除了 T2 的谷歌眼镜项目 T3，一些小型原始设备制造商也推出了蓝牙智能手表，作为智能手机的配套产品。

比如，化石能源公司有一款制作精良的 T4 元手表“T5”，智能家居软件 InPulse 有一款热销的 Pebble smart watch“T7”，甚至还有一些可以量化衡量人们日常活动的智能设备。比如耐克的 FuelBand、Jawbone UP 和 Basis。再加上苹果即将推出的 iWatch 和三星的 T9 智能手表，看来谷歌已经在酝酿推出一款手表了。

谷歌眼镜(Google Glass)让可穿戴计算技术比普通手表更上一层楼。然而，它的使用率几乎肯定会低于手表或智能手机，因为体验是我们与数字内容和世界互动方式的巨大变化。而智能手表给人的感觉更像是使用一部真正的小型智能手机，这种熟悉感让它成为智能手机和计算耳机之间的一座伟大桥梁。

谷歌没有对这一猜测发表评论。

不过，有一项专利是谷歌(Google)于 2011 年申请的，内容是带有“翻盖显示器”的“智能手表”。看起来该专利还提供了触摸屏体验。

现在的问题并不是谷歌是否会生产智能手表。随着小型原始设备制造商和大型竞争对手向市场大量推出可穿戴智能手表，谷歌可能也需要加入这场战斗。不过，目前还不清楚具体情况会怎样？翻盖显示器看起来像翻盖手机吗？

从[的专利申请](https://web.archive.org/web/20221005201917/http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&u=%2Fnetahtml%2FPTO%2Fsearch-adv.htm&r=23&p=1&f=G&l=50&d=PTXT&S1=%2820121002.PD.%20AND%20Google.ASNM.%29&OS=ISD/20121002%20AND%20AN/Google&RS=%28ISD/20121002%20AND%20AN/Google%29)来看，“翻盖显示器”似乎像一个数字怀表一样工作，当打开时显示两个显示器，当关闭时在顶部显示一个显示器。

然而，仅仅因为谷歌提交了这项专利，并不意味着谷歌的 Android 智能手表会像它一样。

在软件方面，谷歌已经证明了它可以开发新的计算形式，比如谷歌眼镜。甚至一些已经发布的应用程序，如 Google Now 和 Field Trip，看起来也能很好地适应智能手表。另外，我们不能忘记收购摩托罗拉给谷歌留下了一个相当大的硬件团队。
