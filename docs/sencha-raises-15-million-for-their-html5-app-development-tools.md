# 森查为他们的 HTML5 应用程序开发工具筹集 1500 万美元

森查度过了多么美好的一周。就在几分钟前，他们还宣布推出新的 HTML 5 云服务套件 Sencha.io。明天上午，该公司将宣布，他们已经筹集了 1500 万美元的 b 系列。

声明将在明天的第三届年度 Senchacon 大会的主题演讲中公布。

“森查”是什么意思？Sencha 为 HTML 5 开发人员提供了 Javascript 框架和工具。换句话说，它们让那些试图让网络变得更漂亮的人的生活变得更容易。[例如，Sencha Touch](https://web.archive.org/web/20230205044745/http://www.sencha.com/products/touch/) 让开发者可以快速地将 Native app 风格的触摸手势添加到他们的网络应用中； [Sencha Animator](https://web.archive.org/web/20230205044745/http://www.sencha.com/products/animator/) 则让开发人员用构建 Flash 动画的相同方法来构建复杂的 CSS3 动画。它也是一种绿茶(确切地说，是不需要先把茶叶碾碎而成的绿茶。)

本轮融资由 Jafco Venture 牵头，现有投资者红杉资本(Sequoia Capital)和 Radar Partners 也参与了融资。桑查在 2010 年 6 月的 A 轮融资中筹得 1400 万美元。
