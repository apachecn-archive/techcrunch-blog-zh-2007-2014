# 奥巴马带着 35000 美元+雪莉·桑德伯格晚餐再次造访硅谷

硅谷，奥巴马总统就是戒不掉你。在湾区，这个周末免费？有 35000 元可以挥霍吗？好吧，如果你曾经想见一见在任总统，现在你有机会参加奥巴马的筹款晚宴，25 日周日，在阿泽顿脸书·首席运营官[雪莉·桑德伯格](https://web.archive.org/web/20230204100253/http://www.crunchbase.com/person/sheryl-sandberg)的房子里。

去年秋天，奥巴马在谷歌高管玛丽莎·梅耶尔的帕洛阿尔托别墅(人均 3 万美元)露面，并于今年 2 月在凯鹏华盈风投约翰·杜尔登的别墅参加了一场独家见面会(T8)——出席嘉宾名单包括脸书首席执行官马克·扎克伯格、苹果首席执行官史蒂夫·乔布斯、雅虎首席执行官卡罗尔·巴茨、思科首席执行官约翰·钱伯斯。 Twitter 首席执行官迪克·科斯特罗、甲骨文首席执行官拉里·埃里森、网飞首席执行官雷德·哈斯汀斯、基因泰克董事长阿特·莱文森、谷歌首席执行官埃里克·施密特和风投史蒂夫·韦斯特利·杜尔。 四月份，奥巴马还去了 Salesforce 首席执行官马克·贝尼奥夫的家。

那么为什么会有这么多 SV 爱呢？奥巴马在他的竞选活动和候选人资格中一直将自己定位为技术行业友好型，最近,[脸书首席执行官马克·扎克伯格在脸书总部会见了他,](https://web.archive.org/web/20230204100253/https://techcrunch.com/2011/04/20/obama-i-want-people-to-feel-the-same-about-the-next-internet-breakthrough-as-they-did-about-the-moonwalk/)奥巴马主持了一次 Twitter 市政厅会议，并[聘请](https://web.archive.org/web/20230204100253/https://techcrunch.com/2011/06/01/former-threadless-cto-is-now-obama-campaign-cto/) Threadless 的前首席技术官担任他的竞选首席技术官。

预算有限，但仍想和总统交往？赛门铁克前首席执行官约翰·w·汤普森和他的妻子桑德拉将举办一个更早更便宜的招待会，一般门票 2500 美元，照片 7500 美元(如果你们是两个人，奥巴马会给你一个 10K 美元的优惠)。
