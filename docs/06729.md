# 巨大的信任投票:优步从基准资本 TechCrunch 融资 1100 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/02/14/huge-vote-of-confidence-uber-raises-11-million-from-benchmark-capital/amp/>

我们从消息来源证实，一家年轻的初创公司<amp-img class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" src="https://web.archive.org/web/20230124210403im_/https://techcrunch.com/wp-content/uploads/2010/07/ubercab.png" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>[优步](https://web.archive.org/web/20230124210403/http://uber.com/)，一家让人们通过他们的移动应用程序随时预订黑车服务的公司，已经完成了一轮 1100 万美元的融资，对该公司的预投资估值为 4900 万美元，后投资 6000 万美元。Benchmark Capital 领投了这一轮，之前的投资者也参与了进来。基准合伙人[比尔·格利](https://web.archive.org/web/20230124210403/http://www.crunchbase.com/person/bill-gurley)加入优步董事会。

我们听说，红杉资本合伙人阿尔弗雷德·林也以个人身份参与了这轮融资。

对于这家最初由 StumbleUpon 创始人加勒特·坎普创立的公司来说，这是不平凡的一年。他没有自己的汽车，想要一个私人汽车服务，而不是使用出租车，但不想专用汽车的成本。

优步有什么特别之处？首先，用户喜欢它。你在手机应用程序上要求一辆黑色汽车向你驶来，然后看着它向你驶来，应用程序通过 GPS 跟踪这辆汽车。付款是通过在你存档的卡上收费来自动处理的，这比打车要贵 50%左右。该公司非常重视物流技术。为了最大化利用率和最小化等待时间，他们花了大量时间研究动态价格、需求预测等。在用户层面上，它似乎是可行的。

该公司表示，自去年夏天推出以来，已有数万次付费乘车。这还只是在旧金山。
