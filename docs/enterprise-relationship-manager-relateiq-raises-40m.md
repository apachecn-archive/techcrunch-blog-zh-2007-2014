# 企业关系经理 RelateIQ 融资 4000 万美元

企业关系经理 [RelateIQ](https://web.archive.org/web/20221226065758/https://www.relateiq.com/) 已经从红点风险投资公司、KPCB、菲利西斯风险投资公司和新闻集团筹集了 4000 万美元的新资金。据[华尔街日报报道，](https://web.archive.org/web/20221226065758/http://blogs.wsj.com/digits/2014/03/25/big-data-startup-relateiq-raises-another-40-million/?utm_content=buffer5e521&utm_medium=social&utm_source=twitter.com&utm_campaign=buffer) RelateIQ 在这一轮中的估值为 2.45 亿美元。

之前的投资者包括 Accel、摩根塔勒 SV Angel、Formation 8、Battery Ventures、AMC Cloud Ventures(通过雅虎联合创始人杨致远)、Thrive Capital、Allen & Co .和达斯汀·莫斯科维茨。

RelateIQ 使用大规模数据挖掘技术来自动化企业和 CRM 世界中的关系跟踪。该公司注重前端的简单性，消除了为更深入了解职业关系所需的手动数据输入。

其中一部分涉及使用算法来更好地理解和洞察你在工作场所的行为，从电子邮件、语音、社交网络和日历中捕捉数据，以及分析这些通信中的语言。
