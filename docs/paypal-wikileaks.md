# 贝宝副总裁谈屏蔽维基解密:美国国务院称这是非法的

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/12/08/paypal-wikileaks/>

Milo Yannopoulos 在台上向 PayPal 平台副总裁 [Osama Bedier](https://web.archive.org/web/20230207114448/http://www.crunchbase.com/person/osama-bedier) 提出的第一个问题是为什么 PayPal 阻止了维基解密的支付并冻结了它的账户。这个问题遭到了大多数欧洲观众的嘘声。

在他的回答中，贝迪耶似乎已经遵从了政府的要求，拒绝向维基解密提供服务，*“我们有一个可接受的使用政策，他们的工作是确保我们的客户受到保护，确保我们遵守世界各地的法规，确保我们保护我们的品牌。”*

贝迪耶还表示，PayPal 的决定受到了美国国务院在 11 月 27 日发出的一封信中认为维基解密非法这一事实的影响，Yiannopoulos 没有对这一声明进行跟进。目前还不清楚维基解密到底违反了哪些美国法律。

当被问及[Mastercard.com 今天早些时候宕机](https://web.archive.org/web/20230207114448/https://techcrunch.com/2010/12/08/4chan-takes-down-mastercard-site-in-support-of-wikileaks/)以及 Paypal 是否担心遭到报复时，*“你是一家成功的支付公司的标志之一是黑客开始瞄准你，这个案例也不例外。”*

**更新:**在与 Bedier 在后台交谈后，他澄清说，国务院没有直接与 PayPal 交谈，这里所说的那封信实际上是由国务院发送给维基解密的。我已经改变了这篇文章的标题来反映这个观点。奥萨马说他引用的信的全文和演讲的视频如下:
