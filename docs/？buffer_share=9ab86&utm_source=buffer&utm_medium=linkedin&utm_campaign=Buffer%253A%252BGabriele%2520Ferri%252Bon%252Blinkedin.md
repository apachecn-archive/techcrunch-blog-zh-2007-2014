# Twitter 正在野外测试其官方谷歌眼镜应用

Twitter 发布自己的谷歌眼镜应用只是时间问题，因为凯鹏华盈(Kleiner Perkins)的约翰·杜尔登在本月的谷歌眼镜集体发布会上暗示，该公司正在考虑建造一个应用。

一条来自官方 Twitter Glass 应用的推文被发现了，有趣的是，这位先生给你带来了第一个[非官方 Twitter 应用，GlassTweet](https://web.archive.org/web/20221209200546/https://beta.techcrunch.com/2013/04/27/skip-google-sharing-and-tweet-photos-directly-from-google-glass-with-glasstweet/) 。

当我查看该用户的个人资料时，他们的个人资料中没有任何信息，但它已经被删除了，还有下面的推文:

[推特 https://twitter.com/jonmarkgo/status/328692024024920066]

你会注意到“谷歌眼镜的推特”标签，这表明该推文来自哪个应用程序。这一点，再加上该账户已经被删除的事实，表明有人可能过早泄露了秘密。

我联系了 Twitter，但该公司没有就其对谷歌眼镜的意图向我们提供任何声明或评论。看看真正的 Twitter Glass 体验将会是一件有趣的事情，因为我无法想象任何人会希望看到他们流中的每一条推文从他们眼前经过。我可以看到获取直接信息的效用，也许是回复和提及，当然还有分享照片和视频。

预计未来几个月将会看到 Twitter 和脸书等公司推出大量谷歌眼镜应用，因为这些公司正在试图找出如何最好地利用一种可以增加使用量的设备，让用户与媒体分享一个全新的视角。

我经常和人们使用的例子是，在音乐会或其他现场活动中，谷歌眼镜会非常有趣，我不必从口袋里拿出手机，把自己从这种情况中移走。在不影响我的视野和注意力的情况下，抓拍会很自然。
