# Twitter 让标签变得更有用

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/07/02/twitter-makes-hashtags-more-useful/amp/>

你可能已经注意到 Twitter 已经开始超链接标签。这些词前面有一个“#”，表示推文是关于什么的，使搜索关于特定主题和事件的推文变得更容易。例如，尝试搜索 [#realtimecrunchup。现在它们都有了超链接，当你点击一个标签时，你会被带到特定标签的搜索结果页面。其他人一直在实施这一点；FriendFeed(大惊喜)已经做了一段时间了。一些 Twitter 客户端，包括桌面版的 Tweetie 和 Seesmic Desktop，也提供了到标签的超链接。](https://web.archive.org/web/20230211235957/http://twitter.com/#search?q=%23realtimecrunchup)

对于 Twitter 来说，搜索是一个导航工具，这一功能是让人们轻松发现其追随者群体之外的新推文的又一种方式。这一趋势始于去年四月他们在每个人的主页上增加了搜索框。

实时搜索正在升温。就在今天早些时候，FriendFeed 推出了自己的实时搜索。通过链接到标签，Twitter 为人们提供了另一个进入现有搜索的入口。现在，如果你能追踪特定标签的提及次数。
