# 还记得神奇收件箱吗？是的，这就是现在的 Gmail 优先收件箱

【2009 年 5 月，[我写了一篇文章](https://web.archive.org/web/20221209232423/https://beta.techcrunch.com/2009/05/21/whats-in-the-gmail-magic-inbox/)，推测一种叫做“神奇收件箱”的东西它有什么特别之处？这是博客[谷歌操作系统](https://web.archive.org/web/20221209232423/http://googlesystem.blogspot.com/2009/05/gmail-magic-inbox.html)在 Gmail 的代码中发现的一个参考，似乎指向一个新的 Gmail 功能，它将在你的社交关系的帮助下对你的电子邮件进行分类。我们中的许多人想知道这样的功能是否会很快推出。但这从未发生过。嗯，直到今晚，那是。神奇收件箱是新的 Gmail 优先收件箱。

Gmail 产品总监 Keith Coleman 在我们上周与他讨论新的优先收件箱时透露了这一消息。Magic Inbox 在此次发布之前已经在谷歌测试了很长很长时间。科尔曼说:“*我们经历了这么多版本的*。事实上，科尔曼指出，根据你联系最多的人来对电子邮件进行优先排序的想法是 Gmail 本身的原创想法，但没有人能够将用户界面和功能结合起来。显然现在，他们认为他们有。

科尔曼指出，多年来，谷歌尝试了各种分组视图，甚至在一个收件箱中按重要性排序电子邮件。但是这个优先过滤器是他们几个月前最终决定关注的。打开该功能后，优先电子邮件会查看很多内容，例如你经常给谁发电子邮件，你和谁聊天，你实际上从谁那里阅读电子邮件，等等。这与去年挖出的神奇收件箱代码略有不同，它似乎专注于“朋友”的想法。

众所周知，“朋友”对谷歌来说是一个棘手的概念——尽管他们准备再次尝试他们即将推出的社交产品。对于这一点，明智的做法是选择一些不那么基于具体社会结构的东西，而选择一些更模糊的东西。

它是如何工作的？这是魔法。我们终于发现了盒子里的东西。
