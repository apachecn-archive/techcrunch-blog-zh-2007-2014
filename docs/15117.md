# 谷歌隐私总监 Alma Whitten 离职

正如福布斯[今天下午第一次报道](https://web.archive.org/web/20220808112906/http://www.forbes.com/sites/kashmirhill/2013/04/01/googles-privacy-director-is-stepping-down/)的那样， [Alma Whitten](https://web.archive.org/web/20220808112906/https://plus.google.com/111367628080250080638/posts) ，谷歌产品和工程隐私总监，已经决定辞去她现在的职位。谷歌现在已经证实了这一点。

惠滕 10 年前加入谷歌，在一个动荡的时期监督该公司的隐私政策，当时其街景车被指控监视人们的 Wi-Fi 网络，谷歌决定将其 70 多项隐私政策整合到一个[单一文件](https://web.archive.org/web/20220808112906/http://www.google.com/policies/privacy/)中。

惠滕拥有卡内基梅隆大学的博士学位，她的论文研究了“[使安全性可用](https://web.archive.org/web/20220808112906/https://www.google.com/search?lr=&ie=UTF-8&oe=UTF-8&q=Making+Security+Usable+Whitten)”在 Wi-Fi 街景事件曝光，谷歌因 Buzz 的隐私控制受到严厉批评之后，她被提升为隐私总监，之前她在该公司担任了七年的工程师。

当时，谷歌称她是“隐私和安全计算机科学领域国际公认的专家”。过去两年，她一直是我们隐私方面的工程主管，我们将大幅增加在这个新职位上与她合作的工程师和产品经理的数量。”

据福布斯[报道](https://web.archive.org/web/20220808112906/http://www.forbes.com/sites/kashmirhill/2013/04/01/googles-privacy-director-is-stepping-down/)惠滕将被[劳伦斯·尤](https://web.archive.org/web/20220808112906/https://plus.google.com/115317725503531115879/posts)取代，他是谷歌的一名工程师，之前[曾在 Pixo、苹果和 tali gent](https://web.archive.org/web/20220808112906/http://alumni.soe.ucsc.edu/~you/bio.html)工作。《福布斯》报道称，惠顿将继续留在公司监督交接工作。
