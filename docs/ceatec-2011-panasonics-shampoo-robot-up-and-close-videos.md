# CEATEC 2011:松下的洗发机器人(视频)

在目前正在日本举行的 [CEATEC 2011](https://web.archive.org/web/20230203091448/https://techcrunch.com/tag/ceatec-2011/) 科技展上，松下展示了其超级奇怪(也很棒)[洗发机器人](https://web.archive.org/web/20230203091448/https://techcrunch.com/2010/09/30/videos-panasonics-hair-washing-robot-in-action/)的新版本(第一个版本大约一岁)。机器人自主处理洗发的整个过程:润湿、洗发、调理和干燥。

新版本用总共 24 个(而不是仅仅 16 个)手指来洗头。松下还表示，他们改进了扫描系统(机器人在施展魔法之前会扫描头部的形状)，从而获得更好的“体验”。旧型号也缺乏调节和干燥功能。

[![](img/432de713c0a430e14bb27ac55804dc80.png "robot panasonic")](https://web.archive.org/web/20230203091448/https://techcrunch.com/wp-content/uploads/2011/10/robot-panasonic.jpg) 
用户可以将自己的数据(首选洗头方法、头型)储存起来，以便下次洗头时节省时间。
