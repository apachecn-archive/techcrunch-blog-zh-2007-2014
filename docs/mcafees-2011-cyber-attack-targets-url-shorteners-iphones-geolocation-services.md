# 迈克菲 2011 年网络攻击目标:网址缩写、iPhones、地理定位服务

![](img/573c13adf012c4542f3e6ee50fc0dc86.png) [英特尔收购迈克菲](https://web.archive.org/web/20221209004715/https://beta.techcrunch.com/2010/08/19/intel-buys-cyber-security-giant-mcafee-for-7-68-billion-in-cash/)刚刚[发布了其对 2011 年网络罪犯首要目标](https://web.archive.org/web/20221209004715/http://www.businesswire.com/news/home/20101228005009/en/McAfee-Labs-Predicts-Geolocation-Mobile-Devices-Apple)的预测，许多热门平台都榜上有名。该反病毒和安全软件公司的实验室集团 McAfee Labs 表示，谷歌的 Android、苹果的 iPhone、Foursquare、谷歌电视和 Mac OS X 平台都有望成为新一年的目标。

McAfee 实验室表示，每分钟创建超过 3000 个 URL 的 URL 缩短服务将是 2011 年网络犯罪的重要目标。该公司表示，因为社交媒体网站已经充斥着网络犯罪分子，这些链接将被用于垃圾邮件、诈骗和其他恶意目的。有趣的是，McAfee [最近推出了自己的网址缩写工具 McAf.ee](https://web.archive.org/web/20221209004715/https://beta.techcrunch.com/2010/09/21/mcafee-url-shortener/)

地理定位服务，如 Foursquare、Gowalla 和脸书 Places，预计也将成为安全目标，因为网络犯罪分子可以实时看到谁在发推文，他们在哪里，他们在说什么，他们的兴趣是什么，以及他们正在使用什么操作系统和应用程序。McAfee 表示，这些公开共享的个人信息使得网络罪犯能够发起有针对性的攻击。

当然，不出所料，随着越来越多的消费者和企业使用智能手机，手机预计将在 2011 年成为一个巨大的目标。McAfee 表示，这种使用量的增加，加上“历史上脆弱的蜂窝基础设施和加密的缓慢进展”，将使移动电话上的用户数据面临高风险的攻击。

苹果是迈克菲在其名单中列出的另一个目标。Mac OS 平台以抵御网络犯罪而闻名，但 McAfee 实验室警告说，针对 Mac 的恶意软件在 2011 年将继续增加复杂性。随着 iPads 和 iPhones 在企业中的普及，存在数据和身份暴露的风险。

迈克菲表示，谷歌电视等联网电视平台也将成为可疑和恶意应用的目标。该公司表示，这些应用程序将瞄准或暴露隐私和身份数据，并将允许网络犯罪分子通过受损或受控制的应用程序操纵各种物理设备。

随着有争议的网站维基解密的出现，迈克菲表示，明年还会有更多出于政治动机的攻击出现在网络上。迈克菲实验室称“黑客行动主义”将成为政治示威的一种新形式。
