# 南非希望获得 BBM 的信息

“这是最好的时代。这是最糟糕的时代。”然后时间变得越来越糟，T2 变得越来越糟，T4 变得越来越糟。至少对 RIM 来说是这样。

在经历了沙特阿拉伯政府的重重困难，然后[又在印度做了同样的事情](https://web.archive.org/web/20230204114627/https://techcrunch.com/2010/10/01/indian-government-unhappy-with-rims-solution-for-spying-on-blackberry-users/)之后，南非决定加入进来，要求获得 BBM 的信息。政府的理由当然与安全有关，但也与英国也计划这么做的事实有关。

通信部副部长 Obed Bapela 周一在南部非洲电信网络和应用会议上告诉代表们，“有证据表明，犯罪分子现在正在利用 BBM 策划和实施犯罪。我们希望像英国和沙特阿拉伯那样审查 BBM，”法新社报道。

老实说，这个国家可能需要它。南非平均每天发生 46 起谋杀案，而 BBM 已经证明了自己是组织犯罪的流行方式。
