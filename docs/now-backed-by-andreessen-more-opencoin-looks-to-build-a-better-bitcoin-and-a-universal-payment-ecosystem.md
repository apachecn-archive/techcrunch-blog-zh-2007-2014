# 现在有了 Andreessen & More 的支持，OpenCoin 希望建立一个更好的比特币——以及一个通用的支付生态系统

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/11/now-backed-by-andreessen-more-opencoin-looks-to-build-a-better-bitcoin-and-a-universal-payment-ecosystem/>

又是一天，又是一笔与比特币相关的投资，这进一步证实了[风投们渴望](https://web.archive.org/web/20230406230515/https://techcrunch.com/2013/04/05/why-do-vcs-care-about-bitcoin/)投资于围绕这种去中心化数字货币的生态系统。今天， [OpenCoin](https://web.archive.org/web/20230406230515/https://opencoin.com/) ，更广为人知的是开源支付协议的开发者， [Ripple](https://web.archive.org/web/20230406230515/https://ripple.com/) ，宣布它已经从 Andreessen Horowitz，FF Angel，Lightspeed Venture Parnters，Vast Ventures 和[比特币机会基金](https://web.archive.org/web/20230406230515/https://www.secondmarket.com/company/bitcoin-opportunity-fund)，SecondMarket 创始人 Barry Silbert 的比特币相关创业公司和公司的投资工具，获得了一轮未披露的天使轮融资。

OpenCoin 将利用其新资本来扩展 Ripple 背后的开源代码，对于那些不熟悉的人来说，Ripple 是一个虚拟货币和支付系统，旨在使任何人都可以轻松负担得起以任何货币进行任何金额的交易。OpenCoin 由首席执行官克里斯·拉森等金融服务资深人士领导，他也是 P2P 贷款巨头 Prosper 的创始人，open coin 的使命是创造一种新的全球货币。

有人会说，这使得 OpenCoin 相当于[“比特币模仿者”](https://web.archive.org/web/20230406230515/http://www.businessweek.com/articles/2013-04-11/introducing-ripple-a-bitcoin-copycat)，因为它是一个分布式的开源支付网络，拥有自己的本地虚拟货币“涟漪”(像比特币一样)。但就 OpenCoin 而言，它急于避免这个绰号，也不想被误贴上“下一个比特币”的标签。

尽管(或者说因为)它越来越受欢迎，并且第一次成为一个十亿美元的市场，比特币已经受到越来越大的波动性和安全问题的影响。比特币交易也有需要很长时间确认的趋势。

该公司表示，OpenCoin 希望通过创建一个统一的分类账来解决这些问题，该分类账跟踪每一个账户、交易和余额，系统会自动 pings 确认交易的合法性，使他们能够在一分钟内清算。通过这样做，它希望自己也创造了一个系统，可以消除交换费，让人们能够以最低的成本和退款进行跨境交易，同时解决比特币的一些安全问题。

该公司声称，Ripple 可以容纳任何货币，包括美元、日元、欧元，甚至比特币，使其成为首批分布式货币交易所之一。该公司告诉我们，到目前为止，一些涟漪已经在流通，但该公司将从 5 月开始发行大量货币(计划发行 500 亿涟漪)，计划在长期内投入 1000 亿枚硬币流通。

此前 [OpenCoin 收购了 SimpleHoney](https://web.archive.org/web/20230406230515/https://techcrunch.com/2013/04/03/opencoin-acquires-simplehoney/) ，这是一家制作“愿望清单购物应用”的初创公司，旨在帮助其推广虚拟货币，并让普通人更容易使用和兑换比特币。此举正值 OpenCoin 寻求扩大围绕其交换服务的工具生态系统，该服务现在允许用户发送和接收支付，监控余额，将波纹转换为其他货币等。

虽然许多人认为比特币的胜算不大，它可能只是一个短暂的时尚(或者至少是一个泡沫)，这种观点很可能适用于 OpenCoin 和 Ripple，但也有很多对它有利的因素。Ripple 是由杰德·麦卡勒(Jed McCaleb)等黑客开发的，他们在比特币的早期发展中发挥了重要作用(麦卡勒是最大的比特币交易所之一 Mt. Gox 的创始人)，这一事实让它在虚拟货币世界中获得了大量早期信誉。

OpenCoin 认为，它正在推动比特币的发展，通过创建一个多币种、注重安全的交易所，它可以更好地向主流市场推销分布式全球虚拟货币的概念。
