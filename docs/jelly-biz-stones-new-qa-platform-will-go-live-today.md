# 比兹·斯通的新问答平台 Jelly 利用图片寻找答案

Twitter 联合创始人比兹·斯通的新问答平台 Jelly 今天即将亮相。这款应用可在全球范围内的 [iOS](https://web.archive.org/web/20230328005435/https://itunes.apple.com/us/app/jelly/id685652528) 和 [Android](https://web.archive.org/web/20230328005435/https://play.google.com/store/apps/details?id=com.jellyhq.starfish) 上使用，它将与 Quora 和 ChaCha 等网站正面交锋，但也更进一步，融入了视觉图像等功能，引导人们从社交网络内外获得更好的答案。

Jelly 有八名员工，斯通说，“一半是技术人员，一半是产品、设计和商业运营人员”，自 2013 年以来一直在开发。

Jelly 不会是第一个利用“蜂巢思维”寻找答案的服务——像雅虎问答和 Ask.com 这样的网站已经这样做了很多年，像 Quora(以及不太成功的脸书)这样的服务已经接过了接力棒，融入了针对社交网络的问题和答案的想法，既提供回应，又充当这些回应的感兴趣的读者。最近，Twitter 也以这种方式使用，尽管不太正式。

Jelly 的不同之处在于，它将利用新的算法来帮助将问题定向到特定的网络，特别是使用图像。这里有两点需要注意:

**首先是用图像提问的想法。**

Pinterest 和 Instagram 等基于图像的社交网络的兴起表明，在社交网络上使用图像作为主要沟通形式——或者至少是对话的起点——越来越受欢迎。显而易见，我们将看到越来越多的智能算法被创造出来，以帮助解析这些网络上的长尾内容。(举个例子，就在昨天，Pinterest 宣布[收购了视觉搜索初创公司 VisualGraph 的](https://web.archive.org/web/20230328005435/https://techcrunch.com/2014/01/06/pinterest-visualgraph/)。)Jelly 也在利用这一想法:如今，图像是用来提示用户做出回应的，但我敢打赌，该公司会寻找自动解析这些图像的方法，然后利用这些数据来帮助将问题更好地导向那些可能适合回答(甚至直接提供答案)的人。

使用图像的另一个好处是它指向了当下的平台:手机。Gartner 对未来一年设备出货量的预测表明，个人电脑销量将继续下降，而手机和平板电脑等移动设备的销量将继续上升。事实是，使用移动设备拍照既有趣又简单，但输入一些东西就有点麻烦了。Jelly 将视觉提示置于其平台的核心，并将其定位为未来的问答平台。

关于图像的最后一点是，它指出了一种有趣的方式，即服务可以潜在地扩展。虽然英语是互联网的通用语言，但图像确实超越了语言障碍。

**第二个是水母本身作为如何获取信息的概念想法的无定形想法。**

虽然 Twitter 等社交网络上的一群追随者感觉像是一个固定的想法，但这里的想法是网络可以根据环境改变形状(像水母一样)。

“我们选择水母来代表我们的产品，因为它有一个松散的神经网络，充当‘大脑’，类似于我们想象的松散分布的网络，人们通过水母来相互帮助，”斯通指出。这不是第一次斯通的记者 T2 对水母的模糊形象感兴趣了。
