# 扰乱北京:尼可拉斯·曾斯特罗姆、凯文·斯特罗姆、侯赛因·拉赫曼和更多的人都与中国有关

我们还没宣布完《扰乱北京》的全明星阵容。除了像腾讯联合创始人兼首席执行官马化腾这样的中国互联网名人，像 YouTube 的这样的成功人士，以及像 Rovio 的 Peter Vesterbacka 和 Evernote 的 Phil Libin 这样的崭露头角的西方名人，我们还有更多创业专家。

我很高兴地宣布，Skype 联合创始人[尼可拉斯·曾斯特罗姆](https://web.archive.org/web/20230203131954/http://www.crunchbase.com/person/niklas-zennstrom)也将在北京加入我们。作为 Kazaa 在 Web 1.0 时代的联合创始人，Zennstrom 处于让网络变得更酷的点对点音乐革命的前沿。随后，他提出了一个靠不住的计划，利用点对点技术来拨打廉价的国际电话，最初没有人愿意为此提供资金。

这个想法就是后来的 Skype——一家 2005 年以 26 亿美元卖给易贝的公司，后来被回购，又以 85 亿美元卖给微软。很少有企业家在职业生涯中成功实现了数十亿美元或更多的退出——更少有人在同一家公司做到过两次。

Skype 也是同代人中仅有的在中国取得巨大成功的西方网络公司之一。如今，Zennstrom 有兴趣通过他在伦敦的风险投资公司 Atomico 在中国进行更多的投资，因此这是一个中国本地企业家不想错过的基调。

我们还从硅谷带来了 Instagram 的联合创始人凯文·斯特罗姆。Instagram 在本地和全球都获得了巨大成功。在舞台上，Systrom 将谈论这一旅程以及 Instagram 的下一步。台下，他将首次探索中国。

并非所有说美国话的人对中国都不熟悉。事实上，中国是我们另外两家扬声器公司成功的核心:Jawbone 的侯赛因·拉赫曼和 ShoeDazzle 的 T2。

炙手可热的 [Jawbone](https://web.archive.org/web/20230203131954/http://www.crunchbase.com/company/aliph) 去年筹集了超过 1 亿美元的资金，背后有一些硅谷最大的名字——从安德森·霍洛维茨到红杉资本。它取得了过去十年中很少有硅谷公司取得的成就:建立了一家尖端的硬件公司。没有中国的合作，拉赫曼不可能做到这一点。

同样，布莱恩·李的[shoedazze](https://web.archive.org/web/20230203131954/http://www.crunchbase.com/company/shoedazzle)利用中国的供应链实力，发明了一种新的[高利润的](https://web.archive.org/web/20230203131954/https://techcrunch.com/2010/12/09/can-shoedazzle-finally-make-celebrity-co-founders-pay-off-tctv/)电子商务商业模式。李和拉赫曼将告诉我们他们在中国做生意的经历，以及为什么更多的中国企业家没有利用这些特有的优势来创建自己的品牌产品公司。
