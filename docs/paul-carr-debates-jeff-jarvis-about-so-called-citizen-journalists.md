# 保罗·卡尔与杰夫·贾维斯就所谓的公民记者展开辩论

![](img/638608e306c2ef05b0697c58b613de33.png)

TechCrunch 专栏作家[保罗·卡尔](https://web.archive.org/web/20230403221514/http://www.crunchbase.com/person/paul-carr)昨天在纽约公共广播电台 [WNYC](https://web.archive.org/web/20230403221514/http://www.wnyc.org/shows/bl/episodes/2009/11/10) 讨论突发新闻背景下公民新闻的利与弊。主持人布莱恩·莱勒和记者兼学者杰夫·贾维斯也加入了卡尔的行列，杰夫·贾维斯是公民新闻的长期倡导者。

你可能还记得保罗上周末发布的有争议的帖子，该帖子讨论了胡德堡士兵在周四大屠杀期间在基地内发推文的事情——包括发布一名受害者的推文和对所发生事件的不准确报道。卡尔深入研究了这个公民新闻变坏的例子及其更大的影响的伦理问题。他的专栏引发了关于使用社交媒体的善与恶的进一步辩论和评论。

卡尔和贾维斯就目击者报告他们对新闻的实时观察的新闻美德以及该女子是否可以被视为“公民记者”展开了激烈的争论。贾维斯开始发难，卡尔过了一会儿才开始反击，可能是因为现场采访是在加利福尼亚州一个不适宜的早上 6 点，这个小时让我们这位深夜思考者处于明显和昏昏沉沉的劣势。但在贾维斯猛击了几下后，卡尔终于醒了过来，振作起来。我们认为卡尔赢得了这场比赛，[但是请听一听，告诉我们你的想法！](https://web.archive.org/web/20230403221514/http://www.wnyc.org/shows/bl/episodes/2009/11/10)
