# 微软凭借 Seadragon Mobile 涉足 iPhone

微软显然不会完全忽视 iPhone。他们[今天发布了](https://web.archive.org/web/20230305075655/http://livelabs.com/blog/seadragon-goes-mobile/)一款用于 iPhone 的 [Seadragon](https://web.archive.org/web/20230305075655/http://livelabs.com/seadragon/) 应用程序，让用户能够以接近零的转换时间“无限缩放”高质量的图像。下面的嵌入显示了 Seadragon 的运行情况，在 iPhone 上的体验非常相似。

我们之前已经介绍过 Seadragon 应用程序，如 [TouchWall](https://web.archive.org/web/20230305075655/https://techcrunch.com/2008/05/14/microsoft-touchwall-can-inexpensively-turn-any-flat-surface-into-a-multi-touch-display/) 和 [pptPlex](https://web.archive.org/web/20230305075655/https://techcrunch.com/2008/08/14/microsoft-turns-super-awesome-touchwall-into-super-snoozy-pptplex/) ，三月份微软[将其内置到 Silverlight](https://web.archive.org/web/20230305075655/https://techcrunch.com/2008/03/05/seadragon-in-silverlight-the-coolest-bit-from-mix-keynote-one/) 中。
