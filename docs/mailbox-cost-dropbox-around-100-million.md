# 邮箱花费了 Dropbox 大约 1 亿美元

Disrupt 校友 [Dropbox](https://web.archive.org/web/20230326025426/https://www.dropbox.com/) [今天进行了一系列超级精明、超早期阶段的收购中的第二笔](https://web.archive.org/web/20230326025426/http://blogs.wsj.com/digits/2013/03/15/dropbox-buys-mailbox-an-app-with-some-buzz/)，收购了[大肆宣传的](https://web.archive.org/web/20230326025426/https://techcrunch.com/2012/12/13/omg-mailbox-is-amazeballs/)电子邮件管理应用[邮箱](https://web.archive.org/web/20230326025426/http://www.mailboxapp.com/)，我们称之为“DropMail”

我们听说 Mailbox 正在筹集资金，激起了 Andreessen Horowitz 和其他人的兴趣，这就是为什么今天的[新闻](https://web.archive.org/web/20230326025426/http://www.techmeme.com/130315/p25#a130315p25)说该公司出售给名字和谐的 Dropbox 并不令人惊讶。有时候，收购是筹集增长资源的最简单方式，尤其是当你在处理像电子邮件这样昂贵的问题时。有六位数的等候名单。

我们听说这次收购并不便宜——根据多个消息来源，后 pivot 创业公司花费了存储公司“远远超过”5000 万美元。我们听说价格是大约 1 亿美元的现金和股票。

雅虎也在电子邮件平台方面取得了进展，该平台由 IDEO 资深人士 Gentry Underwood 创立，考虑到雅虎邮箱的品牌衰落以及后者在移动领域的低迷表现，这是有道理的。但 Dropbox 的吸引力和 sympatico 愿景对这家羽翼未丰的初创公司来说更有意义，其令人印象深刻的数字给了联合创始人德鲁·休斯顿和阿拉什·菲尔多西勇气，向电子邮件发起冲击。

无论如何，我们可能很快就会看到 Dropbox 处理电子邮件附件。
