借助新的音频引擎，Findaway World计划为移动和网络上的有声读物商店提供动力技术危机

原文：https://web.archive.org/web/https://techcrunch.com/2014/03/31/findway-world-audioengine/

借助新的音频引擎，Findaway World计划为移动和网络上的有声读物商店提供动力

Findaway World成立于2006年，旨在销售预装有声读物的MP3播放器。(其更值得关注的客户之一:美国陆军。)如今，随着 AudioEngine 的发布，公司正在认真启动一项新的数字战略。

联合创始人兼首席执行官米奇·克罗尔表示，Findaway World并不是要和Audible这样的有声读物商店直接竞争。相反，通过AudioEngine SDK 和 API ，它试图将有声读物带到现有的各种在线商店和服务中。

Findaway数字产品集团副总裁拉尔夫拉扎罗(Ralph Lazaro)表示，该公司此前曾尝试构建定制有声读物应用程序，但它“很快意识到有声读物与其他数字内容在很大程度上是一种互补的体验”。

对于Findaway为之开发应用程序的品牌而言，有声读物成了另一个需要担心的应用程序。如果有声读物市场要持续增长(克罗尔和拉扎罗认为这种情况已经发生，而且随着智能手机、电子书和播客的爆炸式增长，未来有声读物市场潜力巨大)，Findaway就需要改变消费者行为。Lazaro说，最好的方法就是把有声读物与电子书、音乐和其他内容放在一起。

他补充说:“我们相信，由于我们构建了这个平台，因此存在着机会。“我们都知道，机遇来自数字内容的最大卖家，或者只是想开个商店，把有声读物加进产品目录的人。我们可以为所有这些提供服务。”

他说，AudioEngine从出版商那里获取音频内容，然后将其分解成若干章节，使内容“在不同类型的设备上易于消费者理解”。随后，它就可以将最终得到的有声读物出售给合作伙伴——内容按固定价格出售(包括向出版商支付费用以及Findaway收取一定比例)，合作伙伴随后可以随心所欲地将其转化为收入，无论是通过一次出售一本书，还是收取订阅费进行访问，或者其他任何方式。

Kroll表示，整合可以相对快速地进行，合作伙伴可以在60天内推出基于AudioEngine的服务。已经整合的公司包括3M(因为它的云图书馆)、Mackin (MackinVia)、Baker & Taylor (Acoustik)和Follet (Catalist Digital)。

Findaway说它的图书馆现在有超过50，000本有声读物。
