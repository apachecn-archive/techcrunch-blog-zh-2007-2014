# Zoho 日历走出测试版，提供分享、订阅等功能

Zoho 的生产力应用套件无疑是全面的，为用户提供了 20 多种不同的产品。应用程序包括 Zoho 邮件、CRM、项目、规划、会议等。但压延应用程序是一个重要的层，它可以位于所有这些应用程序之上，并将其统一起来。 [Zoho Calendar，](https://web.archive.org/web/20230331225620/http://www.zoho.com/calendar/)于去年推出测试版，今天[退出测试版](https://web.archive.org/web/20230331225620/http://www.businesswire.com/portal/site/home/permalink/?ndmViewId=news_view&newsId=20100707005627&newsLang=en)，带来一系列[新特性](https://web.archive.org/web/20230331225620/https://show.zoho.com/public/raju/Zoho%20Calendar)和功能。

Zoho Calendar 的一个主要新增功能是，现在可以与拥有 Zoho 帐户的其他用户私下共享您的日历。您可以以只读模式共享日历，这样用户就可以只查看忙/闲时间，或者查看您的所有事件和会议。读写模式允许其他用户编辑您的现有事件或事件创建并编辑您日历上的事件。与 Google 日历类似，您可以通过将其他用户的电子邮件地址添加到帐户中，然后设置偏好设置，来与他们共享您的日历。

虽然 Zoho Calendar 允许您通过文件或 URL 导入活动，但您现在也可以“订阅”另一个日历，它会定期更新您的日历。您可以订阅“内部”日历(来自 Zoho 应用程序的日历)或通过提供 URL 订阅 ICAL 格式的任何其他日历，这将在一天内更新您的活动几次。您也可以使用朋友的电子邮件地址来订阅他们的公共日历。

如果你想让你的日历完全公开，Zoho 现在允许你在外部网站上嵌入你的日历。其他新功能包括基于事件的视图(相对于日历/日期视图)，清除日历的能力等等。

当然，Zoho Calendar 面临着来自谷歌日历、微软 Outlook 和 iCal 的强大竞争。但是对于 Zoho 的 200 多万用户来说，一个跨所有套件应用的统一压光应用肯定会被大量使用。

今年，Zoho 对其应用程序进行了大量更新，包括升级到 [Zoho Wiki、](https://web.archive.org/web/20230331225620/http://www.techcrunchit.com/2010/06/17/zoho-adds-workspaces-access-controls-widgets-and-more-to-zoho-wiki/) [跨应用程序的统一搜索](https://web.archive.org/web/20230331225620/http://www.techcrunchit.com/2010/06/01/zoho-adds-unified-search-to-productivity-suite/)，与[脸书连接](https://web.archive.org/web/20230331225620/https://techcrunch.com/2010/02/23/zoho-adds-a-social-layer-to-productivity-suite-with-facebook-connect/)集成，以及在新推出的[谷歌应用程序市场中添加一个应用程序。](https://web.archive.org/web/20230331225620/https://techcrunch.com/2010/03/09/google-apps-marketplace/)

对于一个与谷歌和微软这样的科技巨头竞争，并且从未接受过外部资金的初创公司来说，这种增长令人印象深刻。那么有没有可能通过收购退出呢？Zoho 的 Raju Vegesna 告诉我，这家初创公司目前无意出售。
