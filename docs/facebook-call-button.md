# 为了对抗通话时间和 Google+视频聊天，脸书测试在个人资料中添加视频“通话”按钮

为了增加视频聊天的使用率，脸书正在测试在人们的时间表档案上设置一个“通话”按钮。自从一年前 F [acebook 与 Skype](https://web.archive.org/web/20221231012341/https://techcrunch.com/2011/07/06/facebook-launches-skype-powered-video-calling/) 合作推出这项功能以来，我还没有收到过脸书的视频聊天，但现在这种情况可能会改变。如果一个朋友在桌面版的脸书上在线，那些测试者会在他们的个人资料上看到朋友名字旁边的“通话”按钮，而以前文本聊天窗口中的一个小小的摄像头图标基本上是开始视频聊天的唯一方式。

脸书似乎已经意识到它需要突出这一功能，否则当人们想与朋友面对面地微笑和大笑时，他们首先想到的可能是 Google+及其[常去的地方](https://web.archive.org/web/20221231012341/http://youtube-global.blogspot.com/2012/06/have-your-own-youtube-party-in-google.html)，或者前脸书总统[肖恩·帕克](https://web.archive.org/web/20221231012341/https://techcrunch.com/2012/06/05/sean-parker-airtime-app-platform-cool/)新推出的[播放时间](https://web.archive.org/web/20221231012341/https://techcrunch.com/2012/06/05/airtime-video-chat/)。

![](img/09aa8225dcf6d326f870f9eb0cf4bc2f.png "Facebook Call Button Timeline")

Airtime 上周首次亮相，其使命是通过实时视频聊天的亲密关系，让互联网重新人性化。发布会的一个微妙信息是“脸书真的不是个人”，因为用户主要是通过异步新闻发布或基于文本的聊天进行互动。

![](img/27b86cde1053bc8cb7bbb6756a797d8c.png "Facebook Old Profile Call Button")

这可能触动了脸书的神经，它以自己是网络连接的首选方式而自豪。也许脸书也认为实时视频聊天是更生动的分享方式，很遗憾很少有人意识到它的存在。脸书在旧版本的配置文件上实际上有一个呼叫按钮，但它被隐藏在齿轮设置下拉菜单中。现在这个功能更容易找到了，当你浏览一个朋友的个人资料时，你可能会决定给他们打电话，让他们快速与脸书面对面。

我们还没有看到在 2011 年 7 月推出视频聊天功能时承诺的任何[额外的脸书 Skype 功能，包括群组视频聊天、脸书至 Skype 客户端通话或付费脸书至标准电话号码音频通话。](https://web.archive.org/web/20221231012341/http://www.insidefacebook.com/2011/07/06/facebook-launching-ad-hoc-group-chat-new-chat-design-skype-video-calling/)

也许现在脸书会加强它的游戏，并考虑增加这些功能，并复制播放时间和视频聊天的突出功能(谷歌今天用协作视频播放列表和更容易的共享对其进行了美化):允许两个人同时观看视频，如 YouTube 剪辑。

盯着别人的脸看很有趣，肖恩·帕克在发布会上证明了有时候你只想看一只地鼠尖叫“艾伦！”和你最好的朋友在一起。
