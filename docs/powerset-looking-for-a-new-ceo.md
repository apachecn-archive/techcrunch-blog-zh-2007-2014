# Powerset 寻找新的首席执行官

[![powersetlogo.png](img/9a20e18f8a76a8f4206d304ee0f442ab.png)](https://web.archive.org/web/20230216131947/http://www.powerset.com/) 自然语言搜索初创公司 [Powerset](https://web.archive.org/web/20230216131947/http://www.powerset.com/) 正在经历一些成长的烦恼。巴尼·佩尔将辞去首席执行官一职。他现在将成为首席技术官，他和 Powerset 的董事会将寻找新的首席执行官。Powerset 的另一位创始人、首席运营官史蒂夫·纽康(Steve Newcomb)不在最高职位的竞争之列。他已经离开公司了。

在 Web 2.0 大会上，Pell 给了 Powerset 的搜索技术一个令人印象深刻的演示，尽管它仅限于有限的数据集。搜索引擎将如何应对整个网络，这是一个更大的技术挑战，还有待观察。

但是，这次重组确实提出了一个大问题。为什么在找到新的首席执行官接手之前，辞去首席执行官一职，留下一个巨大的领导空白(也没有首席运营官)？也许这更多是出于内部原因。一次宣布所有事情向员工传达了公司发展方向的信号，并最大限度地减少了未来的意外。首席执行官搜索也表明，Powerset 可能最终准备在明年的某个时候向公众开放其搜索引擎。或者 Powerset 的董事会对公司的发展失去了耐心，想要新的领导。你可以在这里阅读佩尔关于转变[的解释。(你可以在这里](https://web.archive.org/web/20230216131947/http://www.barneypell.com/archives/2007/11/management_chan.html)阅读我们之前的报道[)。](https://web.archive.org/web/20230216131947/https://techcrunch.com/tag/powerset)
