# 加州大学积极拓展在线课程，发现不及格率下降

美国最大的大学系统正在基于一个非常有前途的试点课程，积极扩展其对大规模在线开放学习(MOOCs)的实验性尝试。加州州立大学系统将在另外 11 所大学提供特殊的“翻转”版本的电气工程课程，学生可以在家观看哈佛和麻省理工学院的在线讲座，而课堂时间则用于动手解决问题。圣何塞州立大学的一名试点发现，翻转课堂使通过率提高了 46%，大学校长穆罕默德·卡尤米认为这足以全速前进。

《电路导论》课程由哈佛大学和麻省理工学院的在线课程仓库 [edX](https://web.archive.org/web/20221206210921/https://beta.techcrunch.com/2012/05/09/move-over-harvard-and-mit-stanford-has-the-real-revolution-in-education/) 提供，这是一个向公众免费提供大规模开放在线课程(MOOC)的项目。

正如我们之前所写的，在线教育运动正在以疯狂的速度改变物理学院[，比教育科学家能够研究的影响要快得多。甚至连 edX 总裁阿南特·阿加瓦尔也呼吁对结果保持谨慎。“我不会把这个号码带到银行，”他告诉我。](https://web.archive.org/web/20221206210921/https://beta.techcrunch.com/2013/02/11/a-huge-month-online-education-is-replacing-physical-colleges-at-a-crazy-fast-pace/)

圣何塞 edX 课程似乎正经历着类似于大学从枯燥的老式讲座式教学向更具互动性的形式转变的结果。例如，加州大学洛杉矶分校的一项生物化学课堂实验发现，当逃课时，通过率提高了大约 18%。

但是，一次性实验一旦形成规模，往往看起来比现实更有希望。当新时代的飞行员被扩大到教师和学生都不那么热情的环境中时，事情可能会分崩离析。

因此，圣何塞州立大学开设了一个 MOOC 培训中心，即适应性和混合学习卓越中心。不管你喜不喜欢，MOOCs 就要来了，因为某种形式的在线教育是抑制学生和学校不断增加的债务的唯一方法。正如加州副州长加文·纽瑟姆在 edX 公告的新闻发布会上所说，“坦率地说，旧的教育融资模式不再是可持续的。”
