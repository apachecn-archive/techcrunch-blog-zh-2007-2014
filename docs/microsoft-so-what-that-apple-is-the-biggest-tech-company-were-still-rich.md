# 微软:那么苹果是“最大”的科技公司是什么？我们还是有钱的！

[![](img/576a14f2e00bd98dcfb71a8b035872ef.png "ballmer")](https://web.archive.org/web/20221202043513/http://www.crunchgear.com/2010/05/28/microsoft-so-what-that-apple-is-the-biggest-tech-company-were-still-rich/ballmer-4/)

哦，[微软](https://web.archive.org/web/20221202043513/http://www.crunchgear.com/tag/microsoft/)。别担心:人们仍然爱你。而且，人们仍然恨你，所以最后一切都解决了。首席执行官史蒂夫·鲍尔默[告诉华尔街日报](https://web.archive.org/web/20221202043513/http://online.wsj.com/article/SB10001424052748704269204575270083925943178.html?mod=WSJ_latestheadlines)大致是这样的，“那么苹果现在是美国“最大”的科技公司。大不了。我们仍然能赚到令人难以置信的钱。事实上，我现在就要去我的巨款池游泳了。以后再聊。”准确的报价？不，但它肯定抓住了鲍尔默试图传达的精神:让苹果有它的阳光时刻，但我们在这里做得并不太坏，你知道吗？

原话如下:

> 我会赚更多的利润，当然，这个星球上没有哪家科技公司像我们这样赚钱。股市会处理好剩下的事情。

然后他证明了自己的信心:

> 我们执行得非常好。这将带来伟大的产品和巨大的成功。

也许吧，鲍尔默先生，但不是你们做的每件事都是黄金。当然，有些是——我不知道你是否会发现有人说“Windows 7 糟透了！”他们说“Windows Vista 烂透了！”Xbox 360 仍然表现强劲(尽管现在没有了 j . Allard……)——但是你的手机阵容还有一点不足。Windows Mobile 现在在哪里？你新的 Zune HD 风格的用户界面可能会做得很好，但你还需要一点时间来赶上 iPhone 和 Android。考虑到移动设备是未来的发展趋势，以及所有这一切…

但是作为一个日常的终端用户，我为什么要关心哪家公司赚更多的钱呢？谁在乎呢。我要找的是一种价格合理、能像广告宣传的那样起作用的产品。这是来自苹果公司还是来自某个不知名的初创企业微软公司，这无关紧要。
