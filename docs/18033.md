在昨天 Spotify 收购音乐探索技术公司 Echo Nest 的消息传出后，我们现在知道了这笔交易的价格:1 亿美元，其中 90%是 Spotify 的股权。

Spotify 拒绝对这一数字发表评论，该数字来自接近交易的可靠消息来源。我们已经联系了其他人，当我们了解到更多信息时，我们会更新这个故事。

今天早些时候，我们从一个消息来源获悉，该交易价值 1.25 亿美元。其中包括三个人分享的 1500 万美元现金:两位创始人——布莱恩·惠特曼和特里斯坦·杰汉——以及首席执行官吉姆·卢切斯；2000 万美元用于 70 名左右的员工在四年内的现金奖励，9000 万美元用于延期奖励。当我们四处询问以确认这些数字时，我们听到了 1 亿美元这个数字。

Echo Nest 成立于 2005 年，已经从投资者那里筹集了大约 2560 万美元的资金，投资者包括英联邦资本投资公司、Matrix Partners、Norwest Venture Partners 和 Jim Pallotta。

除了 Spotify，它还因其音乐个性化和发现技术积累了广泛的客户。如今，其中一些是 Spotify 的直接竞争对手。它们包括 Clear Channel 的 iHeartradio、Beats Music、Rdio 和 SiriusXM.com，以及 BBC、Foursquare、MTV、Twitter、VEVO 和雅虎等其他网站。美国、诺基亚以及可口可乐、英特尔、微软和锐步等品牌。Spotify 已经[声明](https://web.archive.org/web/20230404103654/https://techcrunch.com/2014/03/06/spotify-acquires-the-echo-nest/)Echo Nest 的免费 API 将保持免费和开放，这些 API 被这些公司用来支持他们的服务。

正如 [Josh 在与 Spotify 首席执行官 Daniel Ek 和 Echo Nest 首席执行官 Jim Lucchese 的采访中指出的](https://web.archive.org/web/20230404103654/https://techcrunch.com/2014/03/06/spotify-connect/)，这笔交易是 Spotify 的明智之举——在[据报道计划的 IPO](https://web.archive.org/web/20230404103654/http://www.bloomberg.com/news/2014-03-06/spotify-said-to-seek-credit-facility-in-step-toward-ipo.html) 之前——进一步实现其成为音乐平台以及直接的品牌提供商的雄心——尽管竞争对手是否会配合这一战略，或者在其他地方寻找竞争技术，仍有待观察。

在 2013 年 11 月融资 2.5 亿美元时，Spotify 的估值约为 40 亿美元。
