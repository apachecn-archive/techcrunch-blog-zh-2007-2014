# Fleksy 在 iOS 上推出首个 SDK 合作伙伴，展示第三方键盘的威力 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/12/12/fleksy-ios-sdk-apps/>

总部位于旧金山的 Fleksy 已经通过四个新的合作伙伴推出了 iOS 应用内 SDK 集成，他们今天在自己的应用中实现了该软件。这些包括[发射中心 Pro](https://web.archive.org/web/20221005093432/http://contrast.co/launch-center-pro/) 、 [Wordbox](https://web.archive.org/web/20221005093432/http://vulpi.net/?page_id=805) 、 [GV Connect](https://web.archive.org/web/20221005093432/http://www.gvconnect.com/) 和 [BlindSquare](https://web.archive.org/web/20221005093432/http://blindsquare.com/) ，它们是从许多潜在合作伙伴中挑选出来帮助 Fleksy 展示其虚拟键盘的功能和范围的。

在 iOS 上操作，作为键盘等默认系统组件的替代品，并不是一件容易的事情；苹果不会允许第三方开发者像用户在 Android 上那样替换一些系统功能，如键盘、浏览器、信息或呼叫应用。Fleksy 试图通过提供一个 SDK 来绕过这一限制，第三方开发人员可以使用该 SDK 将 Fleksy 一次一个地构建到他们自己的应用程序中，就像谷歌让开发人员可以内置一个选项，让他们的软件在 iOS 上的 Chrome 中打开链接一样。

这四个启动合作伙伴应用程序现在都可以在 App Store 中使用，并利用 Fleksy 的独特能力来解释用户的预期输入，无论他们在屏幕上的哪个位置击中不同的目标。在一次采访中，Fleksy 的联合创始人和首席运营官·约安尼斯·维尔德利斯向我解释了第一批应用程序的选择过程。

“自从我们在 iOS 上首次发布这款应用以来，确实有很多(来自第三方开发者的)兴趣，”他说。“我们已经选择了四个合作伙伴，他们在 SDK 的测试过程中与我们合作，我们试图推出一个解决无障碍市场的应用程序 BlindSquare，然后我们选择了其他受欢迎的应用程序，我们认为它们在设计中有意义地使用了文本输入。”

GV Connect 是一个谷歌语音客户端，你可以使用 Fleksy 发送短信；Wordbox 是一个文本编辑器；Verdelis 说，Launch Center Pro 是“一切的一部分”，它的快捷方式可以帮助人们浏览 iOS，更快地完成工作。他指出，对于前四个合作伙伴来说，重要的不仅仅是找到会以有趣的方式使用键盘的合作伙伴，帮助向其他人展示可能性，还包括使用帮助定义产品的人。

从这里开始，Fleksy 打算继续选择 SDK 合作伙伴，并与第三方开发人员合作推出他们的集成，但最终计划是开放给任何人使用，独立于 Fleksy 的参与。Verdelis 说，Fleksy 的收入取决于每个开发人员的收入方式，有些像 Launch Center Pro 一样尝试将它作为应用内购买，然后分享这些销售的收入，其他人则选择更直接的许可费。

上周，Fleksy 在 Android 上推出了一款独立的第三方键盘，Verdelis 表示，自那以来，他们已经获得了超过 10 万次下载。在 iOS 上，自从推出他们的独立应用程序以来，他们已经有超过 50 万个应用程序，但就商业目标而言，SDK 是这里的焦点，所以观察使用 Fleksy 的应用程序的稳定性如何从这里增长将是关键。
