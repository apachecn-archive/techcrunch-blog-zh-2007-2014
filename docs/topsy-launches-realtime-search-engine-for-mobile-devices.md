# 托普斯推出移动设备实时搜索引擎

[托普斯实验室](https://web.archive.org/web/20221007104420/http://www.crunchbase.com/company/topsy)今天发布了一个面向移动设备的社交、实时[搜索引擎，让用户能够基于来自](https://web.archive.org/web/20221007104420/http://m.topsy.com/) [Twitter](https://web.archive.org/web/20221007104420/http://www.crunchbase.com/company/twitter) 和 [Google+](https://web.archive.org/web/20221007104420/http://www.crunchbase.com/product/google) 的数据发现关于任何话题的相关聊天。

该搜索引擎带有一个漂亮的“社交时间机器”功能，使用户能够在一个可以用滑块控制的时间线上查找结果(见下面的截图)。

同样有用的是:搜索查询可以自动保存，因此用户可以在任何时候重新访问特定的结果，任何社交内容都可以被任何主题、术语或链接引用。

托普斯引用了一项 Performics 研究，称 32%的人在手机上搜索的次数比在电脑上多，75%的人认为手机搜索让他们的生活更轻松。

托普斯总部位于旧金山，成立于 2006 年，由 BlueRun Ventures、Ignition Partners、Founders Fund 和 Scott Banister 提供支持。
