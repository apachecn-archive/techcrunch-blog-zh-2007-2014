# Biba 为其移动互联网会议服务增加了一个桌面版本

互联网会议初创公司 Biba T1 于去年春天从[的阴影中走出来](https://web.archive.org/web/20221207213832/https://beta.techcrunch.com/2013/06/25/biba-comes-out-of-stealth-with-15m-for-mobile-first-password-free-approach-to-conferencing-and-messaging/)，今天发布了一款新的桌面应用，具有屏幕共享功能和高清音频。这项服务最初设计用于 iOS 和 Android 操作系统，现在可以在 Mac 和 Windows 机器上使用。

Biba 遵循的理念是，今天的服务比以往任何时候都更依赖于它们在设备本身上的表现。通过增加桌面支持，Biba 可以扩展人们在不同平台的不同设备上参与的方式。

这项服务的核心围绕着用户的联系人列表。安排活动并通知与会者。与传统的电话会议服务不同，Biba 会在会议安排好之后采取主动，提醒与会者注册 Biba 服务，并在会议开始前两小时再次提醒。在约定的时间，比巴召集参与者；不需要用户名或密码，没有注册的人会在通话前五分钟收到一个免费号码。

有了 Biba，客户可以从他们的移动应用程序向参与者发起电话会议，然后转移到桌面，而不必重新输入呼叫。领导 Biba 呼叫的用户可以看到谁在呼叫以及谁在讲话。组织者可以让有很多背景噪音的人静音。

![screen_share_600x360](img/965d58001fa087a7a3841c2063b88124.png)

新的屏幕共享服务增加了客户可以利用的协作。用户可以进行视频通话，切换到屏幕共享，并在不同设备之间进行。Biba 还增加了高清音频。该公司认为，他们的音频质量比传统手机通话好六倍。

在很大程度上，像 WebEx 这样的服务不能像其他通信技术和商业服务一样在设备之间工作。例如，一个人可以在平板电脑和手机之间使用 Gmail 或 Dropbox 等存储服务，而不会破坏体验。

Biba 展示了应用程序和移动设备对人们变得多么有用。这种组合正在扼杀个人电脑市场，并通过代理扼杀 IT 世界。云服务进一步推动了这种消亡。互联网电话会议和其他电话服务可以使用 Amazon Web Services (AWS)作为编程的基础，这种服务几乎可以在世界任何地方工作。

企业曾经拥有最复杂的电话网络，后来演变成复杂的电话会议服务。在客户机/服务器时代，Webex 和其他提供商使这一过程自动化，并成为基于 web 的协作技术的一些最早的先驱。

今天，Biba 代表了绕过旧交换机的新一代服务，受到基于互联网的呼叫和电话会议服务需求的推动。

Biba 面临着激烈的竞争，如微软 Lync，但毫无疑问，它正骑在正确的潮流上。这一举措是以移动为先的互联网会议，没有回头路可走。
