# 雅虎裁员预计本周开始

雅虎以裁员开始了这一年[，并将以更多裁员结束这一年。自从雅虎从贝恩公司雇佣打手来帮助裁员以来，裁员就在意料之中。确切的裁员人数还不知道——1000 到 3000 是正在讨论的数字](https://web.archive.org/web/20221209120700/http://www.beta.techcrunch.com/2008/01/21/yahoo-layoffs-for-real%E2%80%94but-whats-the-real-number/)。在周二的财报电话会议上，预计雅虎将宣布裁员人数。此外，预计运营预算将全面削减 15%。

一月份，雅虎[裁员 1000 人](https://web.archive.org/web/20221209120700/http://www.beta.techcrunch.com/2008/01/21/yahoo-layoffs-for-real%E2%80%94but-whats-the-real-number/)，一整年都在遭受人才流失[。但是它仍然有大量的员工——大约 14000 人。再减少 10%左右的支出会降低开支，从而产生有意义的财务影响，但也会降低士气。](https://web.archive.org/web/20221209120700/http://www.beta.techcrunch.com/2008/06/19/tracking-former-yahoo-execs-so-many-have-left/)

杨致远这些天并没有太多选择。当最终数字公布后，我们将更新裁员跟踪系统。
