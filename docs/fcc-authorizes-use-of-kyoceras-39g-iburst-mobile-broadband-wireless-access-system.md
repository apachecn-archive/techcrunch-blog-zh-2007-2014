# FCC 授权使用京瓷的 3.9G iBurst 移动宽带无线接入系统

![kyocera-iburst](img/ab19d53f4ecb1186641c725d09dfdda1.png "kyocera-iburst")
不完全是 4G，但比 3G 好，京瓷的 3.9G [iBurst](https://web.archive.org/web/20221007023703/http://en.wikipedia.org/wiki/Iburst) 移动宽带无线接入系统(MBWA)获得了 FCC 认证部门——美国电信认证机构的设备授权。

据京瓷的[新闻稿](https://web.archive.org/web/20221007023703/http://www.businesswire.com/portal/site/home/permalink/?ndmViewId=news_view&newsId=20090317005545&newsLang=en):

> 京瓷的 iBurst 因其高频谱效率和单一频谱重用因子而被归类为 3.9G 设备，并提供比现有 HSDPA 系统高三倍的频谱效率。使用 SDMA 技术的 TDD-TDMA 系统能够在仅 5MHz 的带宽内实现每个基站 32Mbps 的总吞吐量(下行链路 24Mbps，上行链路 8Mbps)，使其小到足以在 GSM 频带中工作。

那么这对无线上网者乔来说意味着什么呢？在这一点上，不是特别多；只是另一种竞争的无线宽带技术。但最终，竞争越多越好(或者至少我们愿意这么认为)。

关于这项 MBWA 技术使用的更多信息，请查看 iBurst 协会。
