# 谷歌聘请硅谷科技博主路易斯·格雷担任 Google+布道者 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/18/google-hires-silicon-valley-tech-blogger-louis-gray-as-google-evangelist/>

著名的硅谷科技博客和创业顾问路易斯·格雷终于将他对谷歌产品和服务的热情转化为职业。格雷刚刚接受了 Google+团队产品营销经理的职位，这是一个福音传播者的角色，涉及谷歌在脸书的新竞争对手 [Google+](https://web.archive.org/web/20230205023650/https://plus.google.com/) 的推广。

这个角色的定义很松散，但将涉及与 Google+工程师合作改进产品，为 Google+设定未来目标，最重要的是，尽可能随时随地推广社交网络。格雷说，后一项职责还包括在活动和贸易展会上谈论 Google+。

这份工作是为格雷量身定制的，他在自己的个人博客[上报道谷歌的技术和创新以及其他事情已经有一段时间了。但应该注意的是，格雷不是典型的谷歌“粉丝”。事实上，就在几个月前，人们本可以指责他站在苹果的阵营里。](https://web.archive.org/web/20230205023650/http://blog.louisgray.com/)

然而，在 2009 年的某个时候，他对苹果的热爱开始动摇。格雷开始感受到谷歌的吸引力，并公开怀疑他这个自称“Mac 家伙”是否会在不久的将来“走向谷歌”。

[他在自己的博客上写道](https://web.archive.org/web/20230205023650/http://blog.louisgray.com/2009/10/could-real-apple-fan-completely-go.html)“谷歌在很大程度上让我越来越喜欢，它们是(苹果)真正的替代品——这是微软从来没有过的。”。

此后不久，Gray [用他的 iPhone](https://web.archive.org/web/20230205023650/http://blog.louisgray.com/2010/07/why-i-turned-in-my-iphone-and-went.html) 换了一个 Android 设备。到了年底，他兴高采烈地宣传 Android 是 iPhone 的有力竞争对手，称它现在可以“一个功能接一个功能地对抗 iPhone”，让选择智能手机成为个人喜好的问题。但即使他在谷歌 Chrome CR-48 审查单元(以谷歌基于云的 Chrome 操作系统为特色的演示硬件)上打字，他仍然发现自己在购买苹果产品，如 iPad、Apple TV 和 MacBook Air。

然而，这些天来，你更有可能发现他使用 Chrome OS、Android 手机和平板电脑，在[Google+](https://web.archive.org/web/20230205023650/https://plus.google.com/100535338638690515335/)…哦，收集[这些可爱的 Android 人物](https://web.archive.org/web/20230205023650/http://www.thegrayeffect.com/2011/04/android-invasion.html)。

在被谷歌聘用之前，格雷是自己公司 [Paladin Advisors Group](https://web.archive.org/web/20230205023650/http://www.paladinag.com/) 的董事总经理，这是一家为初创公司提供战略咨询的独立公司。帕拉丁过去一年的主要客户是汽车护理[资源提供商](https://web.archive.org/web/20230205023650/https://techcrunch.com/2008/06/12/repairpal-its-like-google-health-for-your-car/) [RepairPal](https://web.archive.org/web/20230205023650/http://repairpal.com/) 和 [My6Sense](https://web.archive.org/web/20230205023650/https://techcrunch.com/2010/09/07/my6sense-brings-personalized-content-ranking-app-to-android-phones/) ，这是一款移动应用程序，其目标是从你的社交网络和 RSS 订阅源中发掘最相关的内容。

格雷还持有青少年科技、MyLikes 和 SocialToo 的股份，每当他在博客上提到这些公司或它们的竞争对手时，他都会一丝不苟地披露这些信息。直到 2009 年，他还担任 NAS 供应商 BlueArc 的企业营销总监。

格雷说，他希望在业余时间继续写博客(披露完整无缺)并为初创公司提供免费建议。然而，我们认为，鉴于最近几周 Google+的增长，这个时间可能很难到来。
