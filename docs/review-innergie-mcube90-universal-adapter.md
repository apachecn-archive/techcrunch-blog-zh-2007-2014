# 回顾:Innergie mCube90 通用适配器

![](img/440ac039256ab5fe6d59c7e0c52ac688.png)

**简而言之:**Innergie MC ube 90 是一款体贴的通用电源适配器，可以同时为你的笔记本电脑和 USB 供电的设备充电，它的特点是有一个微小的辅助 DC 适配器，可以从主适配器上拆下来，供汽车和飞机使用。

![](img/618fab9a4f452fdf89815eb8bfdda1a5.png)

**长版:**

公路勇士们，注意了。这个 119 美元的通用适配器可能是你下次旅行需要随身携带的唯一电源。与其他各种通用适配器的价格相似，mCube90 提供了一些你在其他地方找不到的额外功能，而不会增加你的包的体积。

![](img/b7f06eea02ec6c178e1ae5686ec9535d.png)

该适配器允许您同时为笔记本电脑和 USB 供电的手机(或 MP3 播放器、录音机等)充电。)当您在酒店房间时，如果您需要上路，适配器底部的四分之一会从顶部分离，充当 Innergie 所说的“世界上最小的汽车/飞机电源适配器”

![](img/cdba869c2b613fdb1a2e514137952a1f.png)

这是一个很好的功能，考虑到在你的杯架上放一个大砖头，电线从里面出来，这是不方便的，也是丑陋的，甚至是危险的。当然，你仍然会有电线，但是实际的电源适配器的大小几乎无关紧要。

![](img/ffa7aa187b349ce28390cc4cd6e84df6.png)

较大的交流适配器输出 65 到 90 瓦，兼容几乎所有标准笔记本电脑，如惠普、索尼、宏碁、戴尔、联想等。有七个附带的电源提示和一个易于阅读的指南，以及交流电源线，DC 电源线和飞机兼容的 DC 电源线。

![](img/d08457aaaf482d1c0a1b3b9bc52b1709.png)

吸头、手册和电线都装在一个带拉链的尼龙袋中，便于运输。

![](img/0ae6e0c946b207349772028309fd1c18.png)

不幸的是，没有包括 MacBook 兼容的提示。我猜这是因为 Mac 笔记本上的 mag-safe 提示可能生产成本更高。这也是一个遗憾，因为 mCube90 有一个漂亮的“MacBook white”外壳。

![](img/fb8f3db2ac8bb0fe6e9fe9b481cd3d8d.png)

适配器本身的大小大约与标准笔记本适配器一样长，但厚度大约是标准笔记本适配器的一半，宽度可能是标准笔记本适配器的 15%到 20%。所有线缆都精心包裹了 velcro 带，便于线缆管理。

**结论:**

Innergie mCube90 适合除了最紧身的旅行包之外的所有包款。119 美元，乍一看可能有点贵，但竞争对手的通用适配器一般在 80 美元以上，你基本上可以在一个适配器中获得三个适配器。

特别是可拆卸的迷你 DC 适配器，如果你花很多时间在车上，这是一个非常好的功能。值得注意的是，如果你不需要完整的交流适配器，DC 适配器[可以作为独立产品出售，售价 60 美元](https://web.archive.org/web/20230322164156/https://www.myinnergie.com/Innergie_Store/Default_mcube90DC.aspx)。
