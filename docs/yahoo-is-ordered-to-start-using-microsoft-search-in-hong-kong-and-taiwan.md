# 雅虎被命令在香港和台湾开始使用微软搜索

微软和雅虎于 2010 年签署的[十年搜索联盟](https://web.archive.org/web/20221211123140/http://yahoobingnetwork.com/en-apac/home)本应提升谷歌在搜索广告上的地位。但是从那以后，这项交易逐渐被削弱，最新的迹象来自[一份法庭文件，称雅虎试图推迟在台湾和香港](https://web.archive.org/web/20221211123140/http://in.reuters.com/article/2013/10/21/yahoo-microsoft-ruling-idINL1N0IB23620131021)采用必应搜索。

美国地方法院法官罗伯特·帕特森(Robert P. Patterson)周一裁定，雅虎必须根据协议条款在这两个市场使用微软的搜索技术。法庭文件显示，雅虎不愿意在香港和台湾推出必应，因为它想看看微软首席执行官史蒂夫鲍尔默的继任者是否同样致力于这项交易。雅虎原定于本月在这两个国家开始使用必应，但在 9 月 20 日告诉微软，它希望将推出时间推迟到 2014 年初。帕特森法官支持一名仲裁员早先的裁决，即雅虎“违反协议‘对微软造成了不可挽回的伤害’”。周一的法庭裁决再次表明了雅虎对搜索联盟的不满。2013 年 2 月，梅耶尔[在一次会议上表示，该协议表现不佳](https://web.archive.org/web/20221211123140/https://beta.techcrunch.com/2013/02/12/ceo-marissa-mayer-says-yahoo-microsoft-search-deal-is-underperforming/)，没有带来预期的市场份额增长或收入增长。根据 comScore 的数据，谷歌仍然保持着 2010 年在美国搜索市场三分之二的份额，相比之下，微软和雅虎的份额分别为 16.3%和 12.2%。

自从加入搜索联盟以来，雅虎和微软都经历了重大的战略转变，该联盟是在梅耶尔成为首席执行官两年前签署的。微软尚未宣布鲍尔默的继任者，但几乎可以肯定的是，该公司的下一任领导人将领导自 20 世纪 90 年代以来最大规模的硬件和软件战略重组。
