# 使用 mShopper  的移动商店

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/22/mobile-shop-with-mshopper/>

随着 mShopper 发布新的手机平台，现在用手机购物变得更加容易了。网上购物的所有方面现在都可以用手机买到。用户可以在几秒钟内完成安全的购买。

但是每个优秀的购物者都知道，购物不仅仅是购买。在发行硬通货之前，必须对价格和质量进行研究。mShopper 应用程序通过实时提供数千种产品的价格和规格来帮助实现这一点。可以轻松获取最新的变更信息。

说到容易，mShopper 搜索引擎可以让你找到一个只有两三个字符的产品。例如，如果一个购物者对松下的高清电视感兴趣，键入“pan”和“hdt”将会得到匹配结果。如果搜索结果太多，可以添加额外的描述来缩小搜索范围。这个功能在购买某种颜色或尺寸的东西时很有帮助。

目前，超过 100 家供应商正在利用 mShopper。展示了 700 多万件产品。这使得购物者可以比较相同商品的价格，做出最划算的交易。如果对某个项目有疑问，mShopper 包括 Click2Call。点击 2 呼叫是每个页面上的客户支持链接。如果您有一个只有真人才能回答的问题，点击 2Call 将让您无需争辩即可回答。

mShopper 的首席执行官大卫·高尔德说:“超过 95%的零售金额是在传统商店消费的，但在线商家通常会提供便宜 20%的价格。”使用 mShopper，用户可以节省时间和金钱。现在人们可以去商店，做他们的研究，然后通过他们的手机直接购买。"

网上购买有一些缺点。运费通常不包括在价格中，所以如果你买的东西很重，必须运到你家，便宜 20%的东西可能就不那么便宜了。当我买东西的时候，我讨厌等待它的到来。有时候，等待带来的额外挫折给我带来的成本超过 20%。
