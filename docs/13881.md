# 继大规模移动重新设计之后，StumbleUpon 给网络带来了新的类似 Pinterest 的外观

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/09/25/following-big-mobile-redesign-stumbleupon-brings-new-pinterest-like-look-to-the-web/amp/>

老牌内容发现平台 [StumbleUpon](https://web.archive.org/web/20230207093949/http://www.stumbleupon.com/) 在过去的六个月里过得很艰难。为了与 Pinterest 这样性感、年轻的新人保持同步，StumbleUpon 在去年年底推了一个重大的重新设计，引入了新的用户体验和推荐算法，以及品牌频道和一个“绊倒”按钮，该按钮将用户带到基于主题的随机(个性化)文章。用户不喜欢新版本，上周，成立 10 年的公司[发布了另一个重大的重新设计](https://web.archive.org/web/20230207093949/https://techcrunch.com/2012/09/19/stumbleupon-revamps-ios-app/)——这一次为其 iOS 应用程序带来了大胆的新外观。

如今，StumbleUpon 的转型仍在继续，[该公司推出了](https://web.archive.org/web/20230207093949/http://www.stumbleupon.com/blog/next-up-a-more-social-organized-and-personalized-stumbleupon/)“一个重新想象的测试版 Stumbleupon.com”，将 iOS 应用的新设计和用户体验带到了网络上。StumbleUpon 的新产品副总裁 Cody Simms 表示，重新设计表明 StumbleUpon 的新重点(以及未来的新战略)，这意味着用户在网站上有更强的个人身份，以及更好的方式来发现(或“偶然发现”)有趣的内容，并查看其他人正在查看和分享的内容。

受脸书、推特和 Pinterest 不可忽视的成功的启发，该公司在公告中宣布“社交发现(现在)是 StumbleUpon 的核心。”因此，重新设计旨在通过一个名为“活动”的新功能，让用户更容易看到人们在绊倒、喜欢和分享什么它可以让用户看到他们的朋友最近喜欢、评论或分享到脸书和推特的页面。

在那里，用户可以看到由 StumbleUpon“专家”推荐的页面，或者那些在发现和评价特定主题或类别的内容方面有良好记录的人。最重要的是，StumbleUpon 将会有一个新的“趋势”页面，正如你所期待的那样——一个可以找到网站上正在流行的内容的地方。

StumbleUpon 新方向的另一个重要部分是整合列表，通过一个功能，用户可以将他们的“喜欢”放入收藏中，他们可以与朋友分享或保留隐私。从大的方面来看，列表创建显然并不新奇，因为它是我们最喜欢的生产力工具的固有功能。那么，现在 StumbleUpon 为用户提供了保存页面以备后用(或者“你想收到的礼物或者你想去的地方的照片”)并关注他人列表的功能，这听起来像是什么样的熟悉体验呢？

就用户体验和设计而言，StumbleUpon 已经在很多值得注意的方面与 Pinterest 相似:列表将成为用户体验的核心。Simms 表示，这项功能将在 StumbleUpon 的平台上推广。

也就是说，该公司很聪明，它正在追求更深层次的东西，而不仅仅是一个漫无目的的内容推荐引擎或钉，咳咳，绊板。StumbleUpon 希望根据我们消费的内容向我们展示我们是谁。它是通过 StumbleDNA 做到这一点的，它将每一个喜欢和每一个分享都归入一个兴趣类别。例如，你可能 23%是科技，13%是运动，47%是户外运动。

这将有助于 StumbleUpon 推荐更有可能让你兴奋的内容，并且它为许多大图人口统计信息奠定了基础，这些信息肯定会引起广告商、内容制作者等的兴趣。聪明还是令人毛骨悚然，取决于你坐在哪里。

StumbleUpon 的新方向显然是要给网站带来新的生命，这个平台现在包含了很多在其他地方已经成功的元素。他们也没有理由不能在 StumbleUpon 的旗帜下取得成功。这是迈向内容探索新模式的一小步，这种模式将超越文章，并明显跨平台，这可能是该公司现在似乎让其移动体验为用户在线观看内容设定基调的原因。
