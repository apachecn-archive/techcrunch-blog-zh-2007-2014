# Vuzix iWear AV310 宽屏现已发货

![vuzix](img/f9b41da8e2954ffa5c8bb32210d34c9d.png "vuzix")我们知道 Vuzix AV310 已经上市[一段时间了](https://web.archive.org/web/20221006235112/http://www.crunchgear.com/2008/09/16/vuzix-av310-video-eyewear-to-do-widescreen/)，但是等待已经结束。iWear AV310 采用全 16:9 宽屏显示器，现在的售价为 249.95 美元，与预期一致。

它复制了一个 52 英寸的屏幕(尺寸为 428×240 ),从 9 英尺外观看，使用一节 AA 电池可以运行 5 个小时。AV310 与所有 iPod 型号、便携式媒体播放器、带视频输出的手机、游戏机、数码相机和 DVD 播放器兼容。前往 [Vuzix 的网站](https://web.archive.org/web/20221006235112/http://www.vuzix.com/iwear/products_av310w.html)为你自己或你被宠坏的孩子订购一个(如图)。我喜欢这个概念和功能，但看起来像独眼巨人不是我的风格。也许你的感觉不同。
