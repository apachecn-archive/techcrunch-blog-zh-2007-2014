# NuCaptcha 动态改变验证码以保证安全性

开发[视频验证码技术](https://web.archive.org/web/20230203102847/https://techcrunch.com/2010/06/30/nucaptchas-technology-uses-video-to-decipher-between-humans-and-robots/)的初创公司 NuCaptcha 希望通过推出向网站所有者承诺安全性的动态验证码来颠覆当今的验证码行业。

验证码是你在网站上发现的安全问题，要求你破译和键入单词或数字，并检测用户是人还是垃圾邮件。你看到的大多数验证码是转录的，基于文本的验证码。

当然，验证码经常会令人困惑，并且不便于用户使用。NuCaptcha 正试图通过向网站所有者提供一种更加用户友好的替代方式来解决这一痛点，这种方式被称为视频验证码。NuCaptcha 在过去的几年里优化了它的安全性，这样 3 个未变形的字母可以在“大多数”网站上使用。需要更多安全性的网站最多可以添加 8 个字母。

NuCaptcha 的行为分析系统可以动态检测异常行为，并可以增加验证码的复杂性，降低视频速度，使机器人和人类农场都难以大量解决验证码。

根据网站所有者所需的网站外观，他们可以使用各种基于视频的模板，并且可以快速部署。目前，NuCaptcha 每月提供数百万个验证码，提供免费、专业和企业产品，用户可以通过 NuCaptcha 的[广告计划确定大小、视频选择、安全级别，甚至可选的品牌、广告和创收。](https://web.archive.org/web/20230203102847/https://techcrunch.com/2011/04/14/nucaptchas-advertising-network-for-video-captchas-goes-self-serve/)

NuCaptcha 面临 ReCaptcha 的竞争，后者于 2009 年被谷歌收购。
