# CrowdStar 和其他人押注于 HTML5 移动游戏平台 Spaceport

多设备社交游戏平台 Sibblingz 宣布许多脸书游戏开发者已经选择他们的平台进行移动化。BitRhymes、IKIGames、LuckyLabs、Fortune Planet、DeezGames 和 CrowdStar 现在都在使用 Spaceport 为 iOS、Android 和其他平台的用户开发游戏。

Sibblingz [允许开发者](https://web.archive.org/web/20230205035958/https://techcrunch.com/2010/05/12/sibblingz-launches-multi-platform-social-game-engine-to-the-masses/)在脸书以及 iOS 和 Android 设备上同时创建游戏，允许玩家在设备之间切换时继续相同的社交游戏。该平台还为开发者提供了利用虚拟商品从免费游戏中赚钱的能力。

该公司基于云的游戏创作平台 [Spaceport](https://web.archive.org/web/20230205035958/https://techcrunch.com/2011/04/27/sibblingz-launches-new-version-of-multi-device-social-gaming-platform/) ，允许游戏开发者使用 Javascript 构建一个游戏，并让它在所有智能手机操作系统上运行，具有类似原生应用的体验和快速的性能。基于网络的游戏作为混合 html 5-native 应用程序运行，游戏代码用简单的脚本语言编写一次，因此它们可以在任何设备上运行。开发者也可以使用 Spaceport 创建原生 iOS 应用。

开发者基本上是把他们的脸书游戏变成基于 HTML5 的移动网页游戏，以及 iOS 应用商店和 Android 的原生游戏。

正如我的同事 MG Siegler [在今年夏天早些时候报道的](https://web.archive.org/web/20230205035958/https://techcrunch.com/2011/06/15/facebook-project-spartan/)，脸书有一个秘密计划，叫做 [Project Spartan](https://web.archive.org/web/20230205035958/https://techcrunch.com/2011/06/30/facebook-ipad-spartan/) ，通过 HTML5 将应用程序带到移动网络上。我们知道[这将很快推出](https://web.archive.org/web/20230205035958/https://techcrunch.com/2011/09/19/project-spartan-f8/)，Sibblingz 表示，Spaceport 允许开发者通过 iOS 和 Android 可用的 HTML5 画布渲染引擎创建与 Project Spartan 兼容的 HTML5 游戏。

Sibblingz 创始人 Ben Savage 告诉我们，这家初创公司正在与多家脸书游戏开发商合作，他们正在为 Project Spartan 做准备。这是一个巨大的 Zynga 规模的机会，显然许多开发者都在努力争取。
