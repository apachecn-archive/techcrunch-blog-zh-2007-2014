# Skimlinks 为网络出版商发布完整的 API

面向网络出版商的内容盈利平台 Skimlinks T1 发布了一个全面的 T2 API 套件 T3。部分原因是出版商可以更多地了解用户的购物习惯。这家初创公司最近成为新闻，当时它被热门初创公司 Pinterest 短暂用于将其外部链接货币化——尽管这种安排在[引起争议](https://web.archive.org/web/20221007024317/https://beta.techcrunch.com/2012/02/17/pinsanity/)后失败了。

这四个 API 包括:链接、商家、报告和产品。Link API 使用开发人员的自定义重定向脚本，或者在无法运行标准 Skimlinks javascript 的应用程序上，实现自动链接“加入”。商家 API 允许出版商根据类别和名称搜索和过滤 Skimlinks 的商家列表。Reporting API 可以在高粒度级别上过滤和细分数据，以帮助了解用户的购买行为并相应地调整设置。产品 API 根据产品参考、类别、商家或价格创建实时产品搜索。产品结果以 XML 格式返回，允许网站开发者选择如何显示和设计他们的网站。

Skimlinks 成立于 2008 年，现在每月处理超过 70 万个网站的 1.1 亿次点击，包括 AVForums、WordPress、Ning、Technorati 和 Hearst Digital，推动 27 个附属网络的 18，000 多家商家实现了近 3，000 万美元的消费者销售额。

该公司在伦敦、旧金山和纽约设有办事处，并得到了苏塞克斯投资公司、NESTA 投资公司、加速器集团和贝塔斯曼数字媒体投资公司的支持。
