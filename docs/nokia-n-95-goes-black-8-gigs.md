# 诺基亚 N-95 变黑，8-Gigs

![](img/ff8ded4c8828510e986164ba3c609838.png)

N-95 有 8g 内存，今天有一些(主要是装饰性的)更新，比如漂亮的黑色外壳，以及其他一些小的机身模块。尽管如此，缺乏任何真正的内部创新应该不会太麻烦——如果你问我，N95 已经有点超前了，但考虑到第一代手机是在历史上最炒作的手机之前推出的，他们在顶级产品上的整个反应似乎是几场演出和一件黑色外套——现在这有点令人担忧。正如我所说的:N95 不一定需要任何新的东西才能成为一款令人疯狂的手机，但黑色的外壳不会有太多的宣传效果。当然，整个媒体机器在手机上有点烧坏了，所以他们可能不得不加入一些心灵传输和无线电力动作来真正实现这一点。它将在第四季度上市。

哦，诺基亚总统刚刚说他最喜欢的专辑是拉什的《T1 信号 T2》。我喜欢芬兰人。真的。
