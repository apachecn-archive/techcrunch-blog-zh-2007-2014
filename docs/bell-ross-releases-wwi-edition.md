# 贝尔&罗斯公司发布一战版

贝尔&罗斯公司并不满足于他们的航空成就，发布了 WWI-92 版，这是一款基于第一次世界大战时期怀表的手表。这款手表非常简单——只有一个镀银指针和表壳的黑色表面——97 款有一个 reserve de marche，所有型号都有焊接的表耳，这一奇妙的细节让人想起手表实际上是焊接有金属条的怀表，因此可以戴在手腕上的时代。

如果你想完全复古，你可以看看 [PW1](https://web.archive.org/web/20230205005753/http://www.bellross.com/us/#/shop/watches/details/416) ，这是一款漂亮的 B & R 怀表，它的设计灵感来自 92 年二战。这两块表可能都非常贵，但是我真的非常渴望得到它们。
