# 在 Web 2.0 峰会上采访迪克·科斯特罗[视频]

【YouTube = http://www . YouTube . com/watch？v=HRzs8sz4WaA&w=640]

Twitter 首席执行官和前即兴喜剧演员迪克·科斯特罗在本周 [Web 2.0 峰会](https://web.archive.org/web/20230203072249/http://www.web2summit.com/)的开幕晚宴上发表了近一个小时的讲话，充满了他独特的幽默和洞察力。

当主持人[约翰·巴特利](https://web.archive.org/web/20230203072249/http://www.crunchbase.com/person/john-battelle)引用最近一篇[芝加哥论坛报的文章](https://web.archive.org/web/20230203072249/http://articles.chicagotribune.com/2011-10-07/business/ct-biz-1006-confidential-costolo-20111007_1_dick-costolo-twitter-coo-tomorrow-jack-dorsey)开始讲话时，“科斯特洛的工作是把从斯努基、安东尼·韦纳和金·卡戴珊的指尖散发出来的 140 个角色变成一台强大到足以承受泡沫破裂的赚钱机器，”科斯特洛自己也附和道，“……对于一个如此有限的杰出人物来说，这个任务似乎无法完成。”

**亮点:** Battelle 对 Costolo 关于 Google 最近在 Google+上发布的标签[和实时搜索](https://web.archive.org/web/20230203072249/http://www.googleplusers.com/hashtags-on-google-plus.html)的质询；“现在还为时过早，”科斯特洛开始说道，对此巴特尔回应道，“这是一个糟糕的答案……‘现在还为时过早’……‘这不是一个零和游戏’..我可以替你回答吗？”

“一个人不能再抛出主题句了，”科斯特洛打趣道，增加了已经非常自我意识的采访的整体自我意识(在某些时候，两人抛出了采访可能产生的博客文章标题——巴特尔误称我为“亚历克西斯”)。

至少有半个小时左右的时间是值得观看的，如果只是为了体验一位杰出的科技公司首席执行官对荒谬的敏锐感觉。
