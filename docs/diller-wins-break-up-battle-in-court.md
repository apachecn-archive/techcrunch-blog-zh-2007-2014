# 迪勒在法庭上赢得分手战

[![barry-diller.jpg](img/a5706f28c02ecefaa04c1859e831b697.png)](https://web.archive.org/web/20221007192730/http://www.flickr.com/photos/jdlasica/50183963/) 巴里·迪勒[今天在](https://web.archive.org/web/20221007192730/http://www.bloomberg.com/apps/news?pid=20601213&sid=aA_1z5Qf7LsA&refer=home)与自由媒体的约翰·马龙的法庭大战中获胜。现在，迪勒终于可以实施他的[计划，将 InterActive Corp.](https://web.archive.org/web/20221007192730/http://www.beta.techcrunch.com/2007/11/05/barry-diller-uncomplicates-his-life%e2%80%94splits-iac-five-ways/) 拆分成五个部分——HSN、Ticketmaster、Lending Tree、Interval International 和新的 IAC (Ask.com、Bloglines、Citysearch、Evite、iWon、Match.com、BustedTees、Vimeo、GarageGames 和 CollegeHumor)。IAC 的最大股东马龙试图阻止分拆的发生。

财务策略是否会为股东“释放”任何价值仍有待观察。(如果是的话，我会很惊讶的)。但毫无疑问的是，IAC 是一头笨拙的多头野兽，其分散业务的集合从未真正有过太多的相互联系。正如我去年 11 月报告的那样:

> Diller 将继续担任 IAC 的首席执行官和主席，IAC 仍然拥有大约 30 个网站。但至少这些企业开始最终能够自立。对于他们来说，因为抵押贷款危机而被 Lending Tree 拖累或被家庭购物网络所掩盖没有太大意义。IAC 的控股公司模式用其更成熟业务的收益为其初创公司提供了庇护，但无论规模较大的企业发展得多快，它们都很难克服任何困难。

正如在审判中出现的问题一样，这些基础网络业务的增长也没有迪勒希望的那么快。Ask.com 未能实现将其搜索市场份额翻一番的目标，Ticketmaster 错过了二级门票市场的增长，最近不得不以 2 . 65 亿美元收购 TicketsNow，与 StubHub(由易贝所有)竞争。

一个独立的 IAC 能更有效地与 Web 创业公司竞争吗，或者它只是 Web 1.0 狗的集合？
