# 多亏了鲨鱼池，呼吸仪获得了 200 万美元的资助

呼吸仪希望帮助减少路上醉酒司机的数量，这要归功于一种口袋大小的设备，它通过耳机插孔连接到你的手机，可以测量你的血液酒精水平。为此，我们获得了 200 万美元的新资金，并宣布了一个全新的高管团队。该公司从包括结构资本 VC 和 Dillon Hill Capital 在内的投资者那里筹集了 100 万美元，此外还从那个怪异的 ABC 节目中的五个“鲨鱼”那里获得了 100 万美元。它还任命了一大批新高管，包括新的首席财务官马特·阿克曼、营销副总裁布莱恩·斯特迪万特、销售副总裁马特·萨蒙斯和软件工程副总裁戴夫·布瑞恩。
