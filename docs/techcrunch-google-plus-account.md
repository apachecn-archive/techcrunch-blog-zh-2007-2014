# 我们的最新员工:Techathew Cruncherin，Google+ Maven

今天早些时候，[谷歌开始禁止那些在不该注册的时候注册了 Google+个人资料的品牌。一些人对此感到愤怒，但事实是，谷歌从一开始就表示，Google+目前只为真实的人服务，品牌/公司支持将会到来。这就是为什么 TechCrunch 还没有设立 Google+账户。谷歌让我们等等。所以，我们等着。](https://web.archive.org/web/20230308111913/http://searchengineland.com/google-removes-mashable-sesame-street-other-prominent-accounts-from-google-plus-86788)

像个十足的笨蛋。

结果，[谷歌实际上对让那些品牌/公司保留他们建立的账户没什么意见](https://web.archive.org/web/20230308111913/https://plus.google.com/105923173045049725307/posts/gTyhduYbfnj)，只要有人被指派到账户上。[还是什么](https://web.archive.org/web/20230308111913/https://plus.google.com/113217924531763968801/posts/UUnWt9phLux)。此外，他们现在正在[重新思考整个品牌的事情](https://web.archive.org/web/20230308111913/https://techcrunch.com/2011/07/21/vic-gondotra-on-how-google-handled-brands-it-was-probably-a-mistake/)。

换句话说，他们彻底耍了我们。当我们的竞争对手为他们的新闻订阅设置 Google+账户时，我们像傻瓜一样无所事事。在某些情况下，他们获得了数万或数十万的追随者。正如我之前解释的那样， [Google+已经成为](https://web.archive.org/web/20230308111913/https://techcrunch.com/2011/07/05/google-plus-sharing/)[利用它的网站](https://web.archive.org/web/20230308111913/https://techcrunch.com/2011/07/07/social-sharing-war/)流量的重要驱动力。这项服务似乎每天都在以数百万计的速度增长。

我们会第一个承认，如果没有 Twitter 的 SUL(推荐用户列表)，我们不会从 Twitter 获得如此多的流量。事实上，当我们离开 SUL(后 Twittergate)时，我们的数字受到了打击。从本质上说，谷歌刚刚让我们离开了他们自己的 Google+SUL，告诉我们不能创建账户，同时给我们的竞争对手大量的流量和追随者，然后赦免他们违反规则。
