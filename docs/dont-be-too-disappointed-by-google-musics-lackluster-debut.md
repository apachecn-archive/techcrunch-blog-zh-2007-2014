# 不要对谷歌音乐平淡无奇的首次亮相太失望

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/16/dont-be-too-disappointed-by-google-musics-lackluster-debut/>

网络对谷歌音乐不太感兴趣，它今天像往常一样大张旗鼓地首次亮相。批评以其有点标题的方式逐渐消失:“Spotify 杀手在哪里？”“Rdio 已经这样做了！”“所以基本上是 iTunes 匹配？”

有两件事批评者拒绝承认:谷歌正在提供一个简单的，免费的服务，以及所有谷歌产品开始并不引人注目，并逐渐扩大。Android 出现的时候一片混乱。而且它仍然是一个烂摊子——一个每天多达 50 万台设备的烂摊子。我们都知道谷歌不提供完整的产品。那么，当它将成为一个新的谷歌音乐时，我们为什么要在第一天的诽谤或赞美中浪费我们的口舌呢？

当然，例外比比皆是，但它是像 Wave 这样的华而不实的东西，无法真正超越其初始形式，最终会被卷起。音乐看起来像是在一两个月内拼凑起来的，然后谷歌花了一年左右的时间与音乐高管开会，以获得他们的内容。

谷歌音乐更像 GMail:它是你已经有的，只是它在他们的服务器上，你可以在任何地方访问它。网络邮件在问世时就已经存在，GMail 也很有竞争力，但算不上是一个飞跃。这里还有音乐:一个存放 MP3 的储物柜，一个简单的界面，一个和其他人选择和定价差不多的商店。

问题是，iTunes 在近十年前抓住了行业的短毛，并且从未放弃，在这段时间里没有真正改变。我是说真的，有什么不同？现在最大的变化，也许也是最重要的事情是，它是 iPhones 和 iPads 的媒介。但这并不像 Genius 和 Ping 已经进入了时代精神，正如我很久以前写的那样，当许多人只是想玩音乐时，程序的膨胀就失去了控制。iCloud 很好，但只要我们在谈论不做新东西，iCloud 就是苹果在追赶谷歌和其他人。

真正的竞争是下一代音乐播放器，即 Spotify、Rdio 和各种小播放器。你真的认为谷歌没有这方面的计划吗——谷歌从一开始就帮助推广基于云的媒体？我猜是在谈判中，但唱片公司不喜欢谷歌的货币化想法。谷歌喜欢免费赠送东西，让你通过其他方式付费，通常是广告。百代真的需要另一个潘多拉挂在他们的脖子上吗？

最后但并非最不重要的一点是，谷歌的“逗留”策略，几乎在每一个谷歌平台上都在实施。比如 Google+，不管你喜不喜欢，你基本上已经上了。音乐也是如此:今天你可能不会觉得它很吸引人，但它就在那里，而且是免费的，你会继续看到它。而且会越来越好。

脑海中浮现出这句名言:“如果你在河边坐得够久，敌人的尸体就会漂过来。”当然，在科技领域，总有敌人在下游等着你的身体漂过。

不过，有两件事可能会在未来杀死它。我担心付费服务越来越多的自我整合可能会引起反垄断式调查的注意，这是谷歌一直处于其中的事情。如果他们不能利用搜索将用户引向他们的音乐服务，那将是一个巨大的损失。缺乏基于订阅或免费的流媒体计划也可能会使这艘船沉没。谷歌正试图想办法免费提供这一切。不幸的是，他们不像 YouTube 那样免费提供用户提交的内容，所以这不仅仅是物流的问题(谷歌喜欢物流)。像谷歌电视一样，他们可能会发现自己与人们想要的内容隔离开来，而一个简单的音乐储物柜虽然对一些人有用，但不会吸引大量资金或用户。

基本上我要说的是，谷歌认为一切都是“缓慢而稳定地赢得比赛”类型的情况。当它试图向前冲刺时，它发现自己没有同行，缺乏灵感和改进的东西，这些雄心勃勃的项目就终止了。但谷歌音乐远非雄心勃勃。这是一个简单、熟悉的服务，带有一点吸引人群的诱饵。谁不想在免费服务上备份他们的音乐，以防他们决定不再喜欢苹果或 Spotify？无论你是赞扬还是批评，在第一天就评判谷歌产品都是一个糟糕的举动。我们都知道谷歌在提供完整产品方面有多糟糕。我不知道他们为什么会有这些活动。如果备份不在你的兴趣范围内，你最好的行动是在六个月后重新审视这项服务，并比较它与竞争对手的差距。

嘿，这是免费的*。这很了不起，对吧？*
