# 美国众议院现在可以使用 Skype 的视频通话服务

国会办公室可以使用的技术类型受到政府的严格限制(作为一名前国会工作人员，，我非常熟悉严格的采购政策)。今天，美国众议院[宣布](https://web.archive.org/web/20230331130955/http://cha.house.gov/)国会议员将可以在政府计算机系统上使用 Skype 的视频会议技术。

Skype [称](https://web.archive.org/web/20230331130955/http://blogs.skype.com/en/2011/06/skype_is_in_da_house.html)其工程师与国会网络安全小组密切合作，以确保 Skype 被安全地用于公务(而不是用于类似[韦纳](https://web.archive.org/web/20230331130955/https://techcrunch.com/2011/06/06/tautology/)的行为)。作为安全防范措施的一部分，每个国会办公室都可以访问自己的 Skype 管理器帐户，因此每个办公室都有一个中心人员可以管理 Skype 帐户。

此外，国会议员及其工作人员可以亲自配置重要的隐私设置。

Skype 的加入对许多国会议员来说是一个福音。Skype 将允许立法者与无法前往国会办公室的选民举行会议，当议员不在她或他的选区时参加虚拟市政厅会议，与选区工作人员举行视频会议等。

虽然 Skype 还没有在参议院签字；我们假设，如果对众议院有利，对参议院来说可能足够安全。
