# Viber 桌面在 Windows 8 上发布

Viber 最近一直忙于推出新的创收功能，如[贴纸市场](https://web.archive.org/web/20221209073857/https://beta.techcrunch.com/2013/11/12/viber-flips-monetization-switch-launches-sticker-market-in-4-0-update/)和 Viber Out，但今天该公司专注于通过发布 Windows 8 的 Viber 来扩大其覆盖范围。

Viber [于五月首次在桌面上推出，支持 PC 和 Mac。](https://web.archive.org/web/20221209073857/https://beta.techcrunch.com/2013/05/07/now-200m-users-strong-viber-launches-desktop-app-with-video-calling-in-version-3-0/)

如今，该公司正在将这一服务扩展到 Windows 8 用户，为他们提供专门为该操作系统定制的功能。

在 Windows 8 版本的 Viber desktop 中，用户可以将对话钉在他们的开始屏幕上，使用 Snap View 将 Viber 捕捉到屏幕的一侧，并使用通用搜索功能搜索 Viber 联系人。

与 Viber 的其他版本一样，Windows 8 用户也可以通过一个按钮在桌面和移动之间切换。

以下是首席执行官兼创始人 Talmon Marco 对此次发布的评论:

> 这是继我们支持 Windows Phone 8 之后，Viber 的下一个自然步骤，为微软的触摸友好型操作系统带来完全兼容和定制设计的 Viber 版本。Windows 8 是一个令人兴奋的开发平台，首次为 Viber 带来了一套全新的 UI 元素和独特的功能。我们很高兴在 Windows 8 上支持 Viber 用户，这是我们对广泛跨平台可用性承诺的一部分。

Viber [最近推出了](https://web.archive.org/web/20221209073857/https://beta.techcrunch.com/2013/12/10/viber-officially-launches-viber-out-letting-users-call-mobile-and-landline-numbers/)一项竞争性通话功能，允许 Viber 用户通过手机或座机呼叫非 Viber 用户，与 Skype 的工作方式相同。两者的主要区别是价格，在大多数情况下，Viber 的成本远低于 Skype。
