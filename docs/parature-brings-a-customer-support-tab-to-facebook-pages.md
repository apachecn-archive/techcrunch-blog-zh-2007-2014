# Parature 为脸书页面带来了一个客户支持标签

随着脸书日益成为品牌与消费者互动的渠道，公司利用社交网络来管理客户支持是有意义的。 [Salesforce 的服务云](https://web.archive.org/web/20221007135313/https://beta.techcrunch.com/2009/09/08/everything-you-need-to-know-about-salesforces-service-cloud-2/)、 [Get Satisfaction](https://web.archive.org/web/20221007135313/https://beta.techcrunch.com/2010/03/10/get-satisfaction-turns-to-facebook-to-socialize-customer-support/) 和许多其他公司已经启动了与脸书的深度整合，以允许公司与社交网络上的 [4 亿多](https://web.archive.org/web/20221007135313/https://beta.techcrunch.com/2010/02/04/as-it-celebrates-its-sixth-birthday-facebook-surges-to-400-million-users/)消费者互动。今天， [Accel Partners](https://web.archive.org/web/20221007135313/http://www.crunchbase.com/financial-organization/accel-partners) 支持的 [Parature，](https://web.archive.org/web/20221007135313/http://www.parature.com/)一款随需应变的客户服务软件，正在为脸书页面推出[客户服务产品](https://web.archive.org/web/20221007135313/http://www.parature.com/facebook.aspx)。

随着客户对话越来越多地发生在脸书网页的墙上，Parature 的应用程序允许公司实际集成一个品牌的“支持”标签。客户可以在公司的脸书支持墙上发表评论，然后作为一张列有脸书客户的票被推送到 Parature 的软件中。客户服务代表可以通过直接在墙上回复来推送更新。

Parature 的支持应用程序还允许公司包括一些关于产品或品牌的常见问题。这允许客户从脸书页面中搜索服务答案。如果客户的问题没有得到回答，Parature 的技术将允许用户创建一个将被推送给代表的票据。

随着实时网络聊天越来越成为公司网站上的一种常见支持技术，Parature 允许客户直接从脸书粉丝页面开始与客户服务代表进行实时聊天。当然，这些功能的目标之一是让客户通过脸书信息流与他们的朋友分享他们积极的支持体验。

毫无疑问，Parature 在脸书集成支持技术的举措是客户服务的未来。Parature 的客户包括罗塞塔·斯通、NPR、特纳和 T Mobile，该公司一直直接与脸书(两家公司都有 Accel 的支持)合作开发一款应用程序，该应用程序提供全面的服务应用程序，同时仍能无缝融入社交网络的界面。这并不是 Parature 第一次将社交媒体与客户服务相结合；Parature 的软件最近也增加了 Twitter 支持。
