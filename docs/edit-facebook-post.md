# 脸书允许你在 Android 和 Web 上分享后编辑帖子，很快就会在 iOS 上

错别字毁了你的社交媒体杰作？脸书开始允许你在发表文章后编辑文章内容。这项功能今天在网络和安卓系统上更新，不久将在 T2 的 iOS 系统上更新。以前你必须删除你的帖子，失去你所有的喜欢和评论，然后重新发布来编辑一些东西。Android 用户还可以分享情感和活动，浏览活动，创建相册。

很快你就可以编辑来自网络、Android 或 iOS 的所有帖子和评论。要编辑帖子，你可以点击其中一个帖子右上角的向下箭头，选择“编辑帖子”来更改文本，然后点击“完成编辑”，如[【陈】](https://web.archive.org/web/20230404070659/https://twitter.com/spencerchen/status/383356312492318720)的截图所示。但是，如果朋友查看了你的帖子的编辑历史，你的修改和原始帖子仍然可以查看。对于评论，将鼠标悬停在一篇文章上或点击它，会在右边显示一个铅笔图标，你可以点击它开始编辑。

![Edit Posts On Facebook](img/bde73bc9836c9c1f1aa27281865c6001.png)

长期要求的脸书功能可能会被推迟，因为该公司害怕用户会互相引诱和交换。想象一下，如果一个朋友发了“如果你讨厌贾斯汀比伯，就喜欢这篇文章”，你喜欢它，但他们编辑成“如果你喜欢贾斯汀比伯。”这会损害你的音乐品味。

但脸书显然发现，编辑历史部分足以阻止人们这样的诡计。它于 2012 年 6 月开始允许用户编辑评论，页面所有者也有能力编辑照片帖子上的文字。脸书似乎没有遇到什么问题，所以现在每个人都有了选择权。

![Screen Shot 2013-09-26 at 11.16.32 AM](img/871d4f004424eee19dc218fb989f54a5.png) Google+的帖子编辑已经有一段时间了，这促进了它更冗长、更深入的讨论风格。不过，不要指望编辑功能会很快出现在 Twitter 上，因为它不会与该服务的实时永久记录风格相融合。

这个功能在手机上可能是一个很大的帮助，因为触摸屏键盘很容易出错。仅仅为了一个“它的”一个“它的”而不得不洗去你立即得到的评论，这总是一种耻辱。由于打字错误，脸书的帖子可能会因为删除而减少，所以这种变化也可能会增加一点点内容，为其广告带来眼球。
