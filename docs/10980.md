# 诺基亚:让我们称之为回归吧

这是一个漫长的过程，但在经历了十年的缓慢迷失之后，我认为诺基亚正在从失落之地回归。

这并不是说我认为两款新的 Lumia 手机将会把诺基亚从低迷中拉出来。然而，我认为该公司最终做出了一些大胆、明智的决定，不管喜欢与否，Windows Phone 将在未来几年内挽救这些决定。

首先，诺基亚减少了损失。在这十年的大部分时间里，诺基亚一直在生产人们购买的手机，简而言之，因为几乎没有其他可用的产品。欧洲购买诺基亚是因为它是本地的、可用的，而且最重要的是，价格便宜，而亚洲和美洲喜欢这种手机，但很少认为它们是可以买到的、买得起的。三星开始吃诺基亚的午餐，苹果吃完了。现在，请原谅我的比喻，微软将不得不提供晚餐，否则他们就完蛋了。

通过与 MeeGo 和 Symbian 决裂，诺基亚确保了 IT 部门将再次认真看待诺基亚手机。Windows Phone 可能看起来并不流行，但请放心，一旦 Windows 8 上市，它将成为微软生态系统的重要组成部分。虽然我们都是粉丝，但我很清楚 Windows 对科技领域的影响，一旦 Windows 8 和 Windows Phone 合并，诺基亚的形象将会大大改善。

**其次，诺基亚一直制造良好的硬件，N9 和这两款新手机没有什么不同。**坚固的金属外壳、明亮的触摸屏和一丝不苟的制造标准使诺基亚成为移动时代的主力，这些手机也不例外。

**第三，诺基亚可以攫取苹果的残羹——以及 RIM 的全部市场份额。**仍有一群尚未开发的消费者在寻找智能手机，但不太确定该买什么。想想诺基亚目前的用户群，至少在欧洲是这样的——他们不会把产品卖给年轻的专业人士。相反，他们向那些已经忠于这个品牌、从 8120 年代起就拥有诺基亚手机的人出售手机。智能手机革命错过了他们，同样，三星试图通过 Windows Mobile 驱动的[21 点](https://web.archive.org/web/20230407084339/http://www.wireless.att.com/cell-phone-service/cell-phone-details/?device=BlackJack(TM)+II+by+Samsung&q_sku=sku1140027)等手机来利用“足球妈妈”市场，诺基亚现在有一个尚未开发的市场。然而，不同之处在于品牌。两个熟悉的品牌，微软和诺基亚，胜过一个——RIM。

诺基亚的回归没有得到保证，但如果你能给埃洛普一点信任，那就是给了公司更多的时间来扭转局面。
