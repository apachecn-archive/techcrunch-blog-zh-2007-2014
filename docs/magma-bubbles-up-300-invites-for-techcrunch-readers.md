# Mag.ma 向 TechCrunch 读者发出 300 份邀请

[![](img/95ec5373a8f65fb1b708bbf5eab7b044.png)](https://web.archive.org/web/20230228082345/http://www.mag.ma/) 去年五月，我们有了第一家[口味](https://web.archive.org/web/20230228082345/https://techcrunch.com/2009/05/24/our-first-taste-of-magma/)的 [Mag.ma](https://web.archive.org/web/20230228082345/http://www.mag.ma/) ， [Rocketboom](https://web.archive.org/web/20230228082345/http://www.rocketboom.com/) 创始人 [Andrew Baron 的](https://web.archive.org/web/20230228082345/http://www.crunchbase.com/person/andrew-baron)新在线视频初创公司，旨在帮助实时呈现有趣的新视频。该网站仍处于私人测试阶段，因为它继续推出新功能(上次我试用时它还很简单，但后来已经有了很大的改进)，今天它向 TechCrunch 读者开放了 300 个邀请，以纪念我们的[实时搜索](https://web.archive.org/web/20230228082345/https://techcrunch.com/crunchup/)。要获得一个，只需进入 [Mag.ma](https://web.archive.org/web/20230228082345/http://www.mag.ma/) 并使用 code CRUNCHUP09。

大多数人使用 Mag.ma 主要是因为它的视频“墙”,它提供了网络上最近和最热门的剪辑列表。正如我在第一次报道这个网站时写的那样，如果你想消磨时间，Mag.ma 是一个很好的去处，因为它的特色是实时更新的有趣和/或滑稽的视频不断变化。这些视频按来源分成几个频道(Mag.ma 从 YouTube、CollegeHumor 和 Digg 等网站获取)，但你也可以选择“关注”其他用户创建的视频流。

Mag.ma 还特别强调统计跟踪，允许您观看视频获得更多评论和视频点击，增加视频的整体“嗡嗡声”。因为这些数据来自多个来源，你可以更好地了解一个视频的表现，而不是在一个网站上看它的点击率，比如 YouTube。
