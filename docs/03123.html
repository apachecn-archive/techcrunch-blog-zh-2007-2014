T-Mobile宣布黑莓Curve 8520将于8月5日上市，售价130美元TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2009/07/27/t-mobile-announces-the-blackberry-curve-8520-available-august-5-for-130/

华盛顿贝尔维尤。和安大略省滑铁卢-T-Mobile美国公司，以及RIM公司(纳斯达克股票代码:RIMM；TSX: RIM)今天宣布了新的黑莓Curve 8520智能手机，这是黑莓Curve系列智能手机的新成员，预计将于8月5日上市。

纤薄的新款黑莓Curve 8520是一款令人难以置信的平易近人的智能手机，有T-Mobile提供的两种诱人的颜色——黑色和霜色。对于喜欢联网的人来说，它提供了轻松的电子邮件、消息(IM、SMS、MMS)和流行的社交网站(包括脸书和MySpace)的移动访问，并具有高度触感的全QWERTY键盘，可以舒适、准确地打字。它还具有丰富的多媒体功能，可以在旅途中访问音乐、游戏和其他移动应用程序进行娱乐。

黑莓Curve 8520智能手机是一款全球手机，也具有Wi-Fi功能，可以让客户快速浏览网页和进行Wi-Fi通话。T-Mobile的客户可以通过T-Mobile的无限热点呼叫服务继续获得巨大的移动覆盖和无限的全国Wi-Fi呼叫，以及通过T-Mobile my waves服务无限的全国五人呼叫。*

除了提供业界领先的移动电子邮件和消息传递解决方案，BlackBerry Curve 8520智能手机还设计为高效便捷。它引入了创新的触摸感应光学触控板，使滚动和选择变得流畅和容易，从而获得出色的导航体验。它也是第一款具有专用媒体键的黑莓智能手机，平滑地集成在手机顶部，为客户提供了一种简单方便的方法来控制他们的音乐和视频。

T-Mobile USA的产品营销总监Travis Warren表示:“人们越来越期待丰富多样的手机体验，因为他们正在以新的不同方式使用手机。“将BlackBerry Curve 8520添加到T-Mobile系列中，可以为我们的客户提供一款能够帮助平衡繁忙的职业和个人生活的混乱需求的手机。”

“越来越多的手机用户选择升级到智能手机，新的黑莓Curve 8520无疑会说服更多的人进行转换。它提供了所有伟大的信息，社交网络和多媒体功能，使黑莓成为美国最畅销的智能手机品牌。“T-Mobile的客户将会喜欢在脸书和MySpace上保持联系是多么容易，而且全QWERTY键盘对于在旅途中打字来说惊人地快速和舒适。”

BlackBerry Curve 8520智能手机的主要功能包括:

全QWERTY键盘和触控式光学触控板，提供可靠、灵敏的打字和导航功能

256MB闪存和512Mhz下一代处理器增强了性能
高级电话功能，包括声控拨号和蓝牙(2.0)支持免提使用耳机、车载套件、立体声耳机和其他蓝牙外设
2 MP数码相机，支持变焦和录像

用于音乐、图片和视频的高级媒体播放器，具有专用的媒体键和3.5毫米立体声耳机插孔，以及BlackBerry Media Sync，可轻松快速地将iTunes或Windows Media Player中的音乐与智能手机同步**
访问BlackBerry App World，该应用程序具有广泛且不断增长的专门为BlackBerry智能手机开发的第三方移动应用程序目录。类别包括游戏、娱乐、社交网络和共享、新闻和天气、生产力等等

黑莓互联网服务支持访问多达10个受支持的电子邮件帐户，包括最受欢迎的ISP电子邮件帐户，如雅虎！、Windows Live Hotmail、AOL和Gmail以及BlackBerry Enterprise Server支持，为企业部署提供高级安全和IT管理功能
通过热插拔microSD/SDHC存储卡插槽的可扩展内存，目前支持高达16 GB的卡，预计将支持下一代32GB卡；包含1GB卡
内置Wi-Fi (802.11 b/g)，支持T-Mobile无限热点呼叫服务

四频全球电话:EGDE/GPRS/GSM(850/900/1800/1900 MHz)

T-Mobile的新款黑莓Curve 8520智能手机将于8月5日在T-Mobile零售店、沃尔玛商店、精选授权经销商和http://www.T-Mobile.com在线商店上市。
