# HTC 终于提供 Bootloader 解锁，美国用户必须等待

在过去的几周里，HTC 用他们的引导程序解锁工具的可用性来暗示和戏弄我们，但是等待终于结束了。如果你拥有一个国际 HTC Sensation，那么恭喜你:你可以自由地去[逛 HTC Dev](https://web.archive.org/web/20221207014801/http://www.htcdev.com/bootloader/unlock-instructions) 来解决你的问题。

Android 开发者和爱好者应该不会有太多的麻烦，尤其是考虑到 HTC 的漫游质量非常好，但一点点额外的谨慎不会有什么坏处。另一方面，Android 开发领域的新手应该慢慢来:这个过程涉及到使用 Android SDK，对于门外汉来说需要一点时间来适应。尽管 HTC 在引导你完成相关过程方面做得很好，但如果你不小心的话，它仍然是那种容易出错的东西。

HTC 仍未透露该工具何时能用于更具国内吸引力的设备，但请放心，Peter Chou 和他的公司正在跟进此事。对于那些拥有 T-Mobile sensation 和 Sprint EVO 3Ds 的人来说，你的日子将会到来，但你们中更没有耐心的人总是有[其他](https://web.archive.org/web/20221207014801/http://forum.xda-developers.com/forumdisplay.php?f=1112) [选项](https://web.archive.org/web/20221207014801/http://forum.xda-developers.com/forumdisplay.php?f=1096)。
