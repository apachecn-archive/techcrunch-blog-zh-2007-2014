# 吉尔摩帮:动态集群

吉尔摩一伙——约翰·博思威克、基思·特雷、凯文·马克斯、约翰·塔斯切克和史蒂夫·吉尔摩——朝着推送通知的黄金时代越走越远。一些人认为这是一个支离破碎的应用海洋，另一些人则认为这是一个成熟的机会，可以将通知作为电子邮件、社交流媒体的继任者统一到核心。随着 Windows 被卷入移动化的浪潮，要靠单个应用程序将自己融入无情的潮流。

从大量的转发、@提及和其他图形感知信号中，一种新的层次结构正在出现，成为传统电子邮件贵族的替代者。该企业不相信精英驱动的个人出资制度的可行性，但幸运的是，还没有任何其他东西浮出水面来排除这种可能性。与此同时，平板电脑的缓慢迭代变薄正在形成一个导航新世界的新模板，在狂喜中逆流而上。
