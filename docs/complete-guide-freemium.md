# 免费增值商业模式完全指南 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/04/complete-guide-freemium/>

***编者按**:这篇客座博文是由* *[Future Simple](https://web.archive.org/web/20230205031745/http://www.futuresimple.com/) 的首席执行官兼创始人[简自豪·什米洛维奇](https://web.archive.org/web/20230205031745/http://www.crunchbase.com/person/uzi-shmilovici)撰写的，该公司为小型企业* *创建在线软件。这篇文章是基于芝加哥大学布斯商学院经济学教授埃里克·布迪什的一项研究。它还包括安德森-霍洛维茨基金的风险合伙人、斯坦福 GSB 大学教授彼得·莱文的观点和评论*

免费提供你的产品或它的一个版本的想法引起了很多争论。

定价总是很棘手。不幸的是，许多企业家没有给予足够的思考。他们通常会复制类似产品的定价策略，根据“专家”的浮夸陈述做出决定，或者依赖不成立的理由(我们努力工作，所以我们应该收取 X 美元)。

免费甚至更棘手，有这么多关于它的观点，我们认为这将是令人耳目一新的，采取一种批判的方法，深入研究为什么一些公司在采用这种模式方面非常成功，而其他公司却失败了。我们研究了经济学学术论文、行为心理学书籍和为公司工作的策略，得出了以下关键概念。

**边际成本定律**

定价在争夺顾客方面起着重要作用。这里有一条几乎和万有引力定律一样适用的经济定律:在完全竞争的市场中，长期产品价格(又名“市场出清价格”)将是生产的边际成本。

你猜怎么着？由于主机和带宽成本的下降，对于大多数互联网产品来说，今天的边际成本实际上是……零。

换句话说，如果服务客户的成本(支持除外)为零，产品在市场上的长期价格也将为零(因为竞争压力)。

**一个体验好的**

“免费”模式的核心是提供给客户的产品或服务。大多数互联网产品或服务都符合体验商品的定义:一种产品需要一段时间的使用，然后客户才能确定他们可以从中获得的价值。

Dropbox 就是一个很好的例子。想想德鲁·休斯顿的话:“事实是，Dropbox 提供了一种产品，人们在尝试之后才知道他们需要这种产品。”

有很多学者研究了体验商品的定价。1983 年，经济学家卡尔·夏皮罗写了一篇关于这个主题的精彩论文。他的结论是，由于顾客倾向于低估产品的价值，体验商品的最优定价是一个较低的介绍价格，当顾客意识到产品的价值时，这个价格就会提高。

在某些情况下，顾客可能会高估产品的价值。在这种情况下，最佳定价策略是在开始时收取尽可能多的费用，或者用长期合同锁定客户。

这就是为什么当有人要求客户预付服务或产品或签订长期合同时，客户不愿意购买。

因此，介绍价格是一种信号机制。结论呢？较低的入场费表明你相信你的产品会为顾客创造价值。

**自由的心理**

关于自由的心理学已经写了很多。两本专门研究这个主题的书是克里斯·安德森的《T4》、《自由》和丹·艾瑞里的《可预测的非理性》。简而言之，免费是一个情感热点按钮，可以立即降低客户的心理障碍。免费让人们认为他们没有什么可失去的，因为许多人忽略了时间是一种投资。

从这个角度来看，免费是采用的巨大加速器。另一方面，免费使用产品后，很难让客户开始付费。这种现象非常普遍，以至于有了自己的名字:“一分钱的差距”——最难的部分是让你的顾客付给你第一个便士。这就是明智地选择您的高级功能如此重要的原因。

**决定因素**

如果这些都是真的，那么免费(或免费增值)似乎就是答案。嗯…没那么快。这个决定绝对不容易。这里有一个基本框架来帮助你做出更明智的决定。不过，提醒一句:对于每个复杂的问题，都有一个简单的解决方案……但这是错误的。这个框架作为一个思考工具是有帮助的，但是没有神奇的公式。

这里有一系列你需要问自己的问题:

1.  我希望我的公司有多大？如果你想建立一个月收入 8000 美元的生活方式企业，并且你有一个好的产品，你可能不需要免费增值。如果你想建立一家占据主导地位的公司，拥有可观的市场份额，免费增值可以帮助你加速采用。
2.  **免费用户的价值是什么？**所有成功的免费增值公司都有一种从免费用户身上赚钱或省钱的方法。要么通过节省营销成本(Dropbox)，要么通过广告或数据赚钱(Pandora、Evernote、Mint)，或者两者兼而有之。如果你不能将你的免费用户转化为营销成本的节省或来自第三方的收入——想办法吧！
3.  **服务免费用户的成本是多少？**这是模型的一个重要方面。如果你花了很多钱和/或时间为免费用户服务，你会损失很多钱。为免费用户服务的成本必须低于他们提供的美元价值。
4.  **我的市场有多大？Evernote 的首席执行官 Phil Libin 说:“让 100 万人付费的最简单的方法就是让 10 亿人使用。免费为您的收入增加了另一个转换步骤。你需要一个大市场，才能有足够多的人在一天结束时付钱给你。**
5.  **使用该产品的其他客户对一个客户有价值吗？这将决定免费用户会推荐多少新用户。价值有三个层次:**

1.  *内在价值*–只有与您通话的人也使用 Skype，您才能使用 Skype。您只能与其他 Dropbox 用户共享 Dropbox 文件夹。在这种情况下，免费增值可能是一个强有力的策略。
2.  *附加值*——你不会想成为 LinkedIn 的唯一用户。你从使用它的其他人那里获得价值。在这种情况下，如果你使用有效的邀请机制，免费增值可以帮助你获得牵引力。
3.  *没有价值*——你不在乎别人是否在用 Evernote。一个人告诉另一个人关于产品或服务的唯一原因是他们认为它很棒。

**自由的类型**

让免费增值发挥作用的关键因素之一是产品的结构。你提供的免费和收费服务是什么？有不同类型的免费策略。让我们来看看受欢迎的:

1.  真正的免费增值——免费提供产品的一个版本，对其他版本收费。有两种方法可以解决这个问题:

1.  基于价值——最成功的免费增值战略。客户使用产品越多，她获得的价值就越多，转换成本就越高，在某一点上，她将达到使用极限，并转化为付费客户。Evernote 和 Dropbox 就是很好的例子。
2.  基于特征——例如，为一个用户免费提供产品(例如，基于公司规模)。让我们考虑一个 B2B 应用程序。如果我是一名自由职业者，我将永远使用这个应用程序，永远不用升级。如果我是一个 3 人公司，我不能添加更多的用户并真正尝试应用程序，因此可能无法看到使用它的价值。
3.  交叉补贴的免费产品——免费提供一种产品，对补充产品收费。
4.  基于时间的免费试用——免费试用 X 天，试用结束后开始收费。这里的问题是弄清楚 X 是什么。一方面，你想创造一种紧迫感，另一方面，你需要客户看到系统的价值。

**开源为免费模式**

最近，我看到许多企业家将开源和免费混为一谈，所以我认为区分开来会有所帮助。开源模式肯定可以加速你产品的发布，是一个可行的免费模式。它有两个主要优点。你可以让开发者为你的产品做贡献。通过这样做，你可以加速产品的开发。另一个优势是您让客户放心，因为他们可以控制源代码。然后你可以通过销售专业功能或增值服务来赚钱。这里有一个关键的区别，那就是你的代码就在那里，任何人都可以开一家公司来商业化这些代码。记住，要改变开源的决定是非常困难的(通常是不可能的)。
**成功的最后一点和秘诀**

当你评估是否使用免费增值模式时，有许多因素需要考虑。然而，还有最后一个秘密我没有告诉你。在研究过程中，在观察成功的免费增值公司时，出现了一种模式。他们都有非凡的产品。如果你提供的产品或服务非常棒，所有这些决定因素都是无用的。如果你的产品没有为用户创造巨大的价值，世界上没有任何策略能让免费增值为你工作。
