# 诺基亚上的 Windows Live

巨头微软和诺基亚今天联手在 S60 兼容设备上提供 Windows Live。这项服务将于明年提供给诺基亚系列 40 兼容手机。

Windows Live 是一套全面的个人互联网服务和软件。它旨在将人们关心的所有关系、信息和兴趣汇集到一个地方。安全和保密功能是精心制作的，以保护用户而不干扰通信。Window 的 Live Hotmail、Live Messenger、Live Contacts 和 Live Spaces 现在都可以在诺基亚设备上使用。

微软在线服务业务高级副总裁 Steve Berkowitz 表示:“面向诺基亚设备的 Windows Live 服务的推出，表明了我们致力于提供出色的移动体验并扩展人们的在线生活——将他们从 PC 带到设备上。“这一联盟将使更广泛的消费者群体体验到 Windows Live 所提供的好处，使他们能够从几乎任何地方轻松地联系到最重要的信息和人。”

这项服务今天首先在 11 个国家推出，包括:丹麦、芬兰、法国、德国、荷兰、挪威、西班牙、英国、瑞典、沙特阿拉伯和阿拉伯联合酋长国。这项服务现在是免费的，但将来可能会产生月租费。有关信息和更新，请按下面提供的诺基亚链接。

随着移动设备变得越来越受欢迎，像 Windows Live 这样的服务必然具有移动性。随着流行的个人电脑功能进入移动设备市场，订阅服务和广告收入将会增加。
