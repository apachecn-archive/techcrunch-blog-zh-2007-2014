# Posterous 加入了 SXSW 的行列，为 iPhone 提供了 Posterous 事件

![](img/5d330fd7d11638641e59554728543c9a.png)
因为你不是任何人，除非你有[你的 SXSW 应用程序的特别版本](https://web.archive.org/web/20230203025518/https://techcrunch.com/2011/03/09/groupme-2-1-sxsw/),[Posterous](https://web.archive.org/web/20230203025518/http://www.posterous.com/)今天发布了自己的 Posterous Events，它允许人们在像 SXSW 这样的大型活动甚至像家庭野餐这样简单的事情上使用他们的 iPhone 应用程序创建一个简单的网站。

所有想要围绕一个事件创建 posterous 站点的用户只需打开他们的 iPhones，为一个特定的位置和帖子创建一个. Posterous 地址(你最近发布的站点将保持在你的事件之上)。

![](img/eb0d920b26be018006b0b48141972c42.png)

Posterous 正在将 SimpleGeo 位置数据整合到应用程序中，这样任何使用 Posterous iPhone 应用程序的人都可以在特定事件附近看到该事件，并张贴照片和媒体，然后以幻灯片格式呈现在网站上(如上图)。

营销负责人 Rich Pearson 说:“我们的目标是让 Posterous Events 成为音乐会、婚礼、派对和会议上的默认分享场所——尽管如此，这还是一个早期版本。”Posterous 还计划在以后的版本中提供事件通知和私人事件功能，但是现在，请享受 SXSW Posterousers。

你可以在这里找到 app store [里的 app。](https://web.archive.org/web/20230203025518/http://itunes.apple.com/us/app/posterous/id394252521?mt=8)
