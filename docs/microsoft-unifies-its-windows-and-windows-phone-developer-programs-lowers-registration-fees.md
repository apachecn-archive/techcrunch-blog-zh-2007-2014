# 微软统一 Windows 和 Windows Phone 开发者程序，降低注册费

微软[今天](https://web.archive.org/web/20230316041135/http://blogs.windows.com/windows/b/appbuilder/archive/2013/11/06/unifying-developer-registration-windows-and-windows-phone.aspx)宣布将整合其 Windows 和 Windows Phone 开发者项目。这两个组的混合是微软的一个举措，以鼓励开发者为一个以上的支持设备类别而构建。这也代表了微软在共享的 Windows 核心平台上统一平台的又一步。

微软现在把 Windows Phone 和 Windows 作为一个软件包来推销。该公司还合并了产品的营销团队，可能有助于进一步统一他们的信息和防止串扰。

Xbox One 也依赖于共享的 Windows 内核。因此，当这款游戏机发布时，Windows 将扩展到你的智能手机、平板电脑、笔记本电脑、台式机和电视。因此，看到微软将 Windows 和 Windows Phone 开发团队整合在一起并不奇怪。

如果你是注册的 Windows 应用商店开发者，你可以免费提交 Windows Phone 应用，反之亦然。微软也降低了(现在)两个平台的注册价格。如果你已经是 Windows Phone 和 Windows Store 的开发者，微软会给你一个代码，免费续费一年。

应用生态系统问题长期以来一直是阻碍 Windows Phone 的关键问题，在微软修复了一系列可用性问题后，它已经成为 Windows 8.1 的最大问题，这些问题使使用 Windows 8 变得令人沮丧。

因此，微软需要消除构建其平台的所有障碍。我在上面提到，Windows 和 Windows Phone 开发者注册系统的统一并不令人惊讶。这并不意味着这不是一个明智之举。
