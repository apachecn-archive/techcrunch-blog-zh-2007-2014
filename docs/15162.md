# 欧文·托马斯(Owen Thomas)以主编的身份加入 ReadWrite，称他将带领该网站“回到我们的根源”

[Say Media](https://web.archive.org/web/20230217010016/http://www.saymedia.com/) 刚刚宣布，前 [Business Insider](https://web.archive.org/web/20230217010016/http://www.businessinsider.com/) 的西海岸编辑欧文·托马斯(Owen Thomas)已加入 [ReadWrite](https://web.archive.org/web/20230217010016/http://www.readwrite.com/) (一家科技博客，Say 于 2011 年收购)担任主编。

我在周五[报道了托马斯(他是我以前的老板——我还是助理编辑的时候他是 VentureBeat 的执行编辑)正在就这个职位进行谈判，尽管当时所有的 Say 代表都告诉我他们正在与一些候选人进行谈判。](https://web.archive.org/web/20230217010016/https://techcrunch.com/2013/03/29/owen-thomas-probably/)

该博客原名读写网，由理查德·麦克马努斯于 2003 年创建，在过去的十年里，它已经成为了许多伟大的科技作家的家园(包括弗雷德里克·拉迪诺瓦、莎拉·佩雷斯和亚历克斯·威廉姆斯，他们目前都在为 TechCrunch 写作)。去年秋天，该公司宣布了新的名称、新的设计，并聘请丹·莱昂斯担任编辑。[莱昂斯上个月离开](https://web.archive.org/web/20230217010016/https://techcrunch.com/2013/03/19/dan-lyons-leaves-readwrite/)，在营销软件公司 HubSpot 谋得一个职位。

在某些方面，托马斯和莱昂斯是相似的选择:他们有丰富的媒体经验(是的，欧文，这是我说你老了的礼貌方式)，但他们也可能是刻薄和有争议的。

“欧文是一位才华横溢的作家，拥有涵盖技术行业各个方面的丰富经验，”SVP 和编辑总监凯特·路易斯在一份新闻稿中说。“他有独特的观点和聪明才智，这让他的作品在他的社区内外都引起了共鸣。”

ReadWrite 也[发表了自己的博客文章宣布转会](https://web.archive.org/web/20230217010016/http://www.readwrite.com/2013/04/03/owen-thomas-is-a-shockingly-nice-fellow)，其中欧文被描述为“一个令人震惊的好人”我不确定我会走那么远，但他实际上很好，这确实让一些人感到惊讶，特别是因为他曾经是 Valleywag 的编辑。

“我们肯定要做的一件事是回到我们的根本，”托马斯告诉我。换句话说，他想探索公司名称中蕴含的理念，即“今天，每个人都是参与者，每个人都是建设者”意味着什么

当莱昂斯作为编辑加入时，他告诉我们他想让 ReadWrite 变得更有趣。我指出，人们可能也希望托马斯这样做，但他说，“我也有严肃的一面，安东尼。”他补充说，他绝对想确保这个网站有幽默感，有时搞笑或尖刻是对科技新闻最恰当的回应:“你必须笑，这样你就不会哭。”
