# 瓦拉布将您的所有信用卡合二为一，确保您获得最多的回报

[Wallaby](https://web.archive.org/web/20221206134939/http://www.walla.by/) (今天即将发布测试版)让我想起了银行初创公司 Simple。就像 [Simple](https://web.archive.org/web/20221206134939/https://www.simple.com/) 自动为您管理财务并自动在最赚钱的借记卡和储蓄账户之间洗牌一样，瓦拉布将您所有现有的信用卡合并到一张卡上，然后根据您在哪里可以获得最多的飞行常客里程或现金返还奖金来决定将您的交易转移到哪个信用卡账户。

Wallaby 由加利福尼亚州圣莫尼卡的加速器[maskerlab](https://web.archive.org/web/20221206134939/http://www.muckerlab.com/)支持，为您提供单一物理信用卡和数字钱包。首先，您向 Wallaby 注册所有物理卡，然后，例如，当您想支付汽油费时，它会自动使用为您提供三倍里数的卡购买汽油。之后，当您在餐厅使用该卡时，它会自动将付款转至您数字钱包中的奖励卡，该卡可为您提供 5%的餐厅现金返还奖金。

[![](img/920107754c43ae3f783e86c9e73d5696.png "Wallaby Financial")](https://web.archive.org/web/20221206134939/https://beta.techcrunch.com/2012/06/26/wallaby-combines-a-cloud-based-wallet-with-an-intelligent-physical-credit-card/wallaby-financial/) 除了自动模式，用户还可以指定他们的花费应该有多大比例流向哪张卡。一个配套的移动应用程序将允许用户随时改变他们的偏好。

该公司表示，现在几乎 90%的美国消费者都携带积分卡，大多数人发现这些信用卡积分计划令人困惑。另一方面，Wallaby 的创始人兼首席执行官 Matthew Goldman 说，它“通过将消费导向每笔交易的最佳卡，确保每个消费者从他们的信用卡年费中获得最大收益，使人们很容易获得最大回报。"

我已经看到一些[设计研究](https://web.archive.org/web/20221206134939/http://www.yankodesign.com/2009/10/22/all-in-one-credit-card/)和[产品](https://web.archive.org/web/20221206134939/https://beta.techcrunch.com/2012/03/13/icache-geode/)旨在做类似的事情，但我不知道另一种产品将物理卡与 Wallaby 承诺的智能后端相结合。

首先，使用信用卡显然有利也有弊，但是如果你确实使用信用卡——尤其是那些带有飞行常客奖励或返现奖励的信用卡——Wallaby 可能值得一看。

顺便说一下，注册该公司测试版的前 1000 名用户将获得终身会员福利，接下来的 5000 名用户将获得 12 个月的免费卡。这些首批测试版用户之外的新用户将获得六个月的免费服务。之后，Wallaby 将每年花费 50 美元。
