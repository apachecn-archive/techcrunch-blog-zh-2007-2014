# 政府将讨论谁负责创建新的顶级域名

![](img/aebd4fe65e2d5cfe26734b34de5c857f.png "USA")  [弗利克](https://web.archive.org/web/20221206220317/http://www.flickr.com/photos/mingchuno/4798537723/)

最近发生了什么让美国政府突然对互联网监管感兴趣？我们看到国土安全部上周查封了网站的域名，尽管这些网站并不在美国本土。现在有消息称，美国正在就批准任何和所有新顶级域名的权利讨价还价。你知道，像. com 或. org 这样的网站。本质上，利害攸关的是谁在掌管互联网？

要了解正在发生的事情并不难。下个月将召开一次会议，以确定团体申请新顶级域名的最终申请流程。Dot XXX 已经存在好几年了，但是还有其他更良性的顶级域名可供争夺:dot nyc、dot health、dot movie 等。新的顶级域名将使组织更容易在互联网上安家。你不必注册“some movie name-the movie-for real . com”，你可以注册并宣传“SomeMovieName.movie”

问题变成了是否允许“有争议”的 TLDsdot gay 就是这样一个 TLD。一个“保守”的国家会希望看到这样一个 TLD 的诞生吗？谁应该决定是否批准网络圣战，凭空举个例子，TLD？

决定到底应该由谁来负责这件事相当棘手。有人会说，ICANN 为美国政府的利益而工作，到目前为止已经做得足够好了。也许，但同样，国土安全部刚刚查封了一些甚至不在美国的网站域名，那么为什么美国应该被完全信任呢？如果不是美国，那是谁？联合国，这个以人权理事会中不太受欢迎的个人和国家而闻名的组织？

我们需要的是公正无私的法庭来调解这类事情，因为我很难相信所有被提议的党派。
