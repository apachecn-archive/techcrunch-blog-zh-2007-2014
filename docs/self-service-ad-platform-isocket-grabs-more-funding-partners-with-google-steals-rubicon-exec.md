# 自助广告平台 isocket 攫取更多资金；与谷歌的合作伙伴；窃取 Rubicon Exec TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/28/self-service-ad-platform-isocket-grabs-more-funding-partners-with-google-steals-rubicon-exec/>

使在线广告买卖变得容易的初创公司 isoket 今天宣布了几个重大消息，包括结束了现有种子投资者数百万美元的首轮融资(该初创公司早在 2009 年就筹集了 180 万美元)，其中包括一些知名天使/风投，如 Draper Fisher Jurvetson 的蒂姆·德雷珀、Blumberg Capital 的大卫·布隆伯格、SoftTech VC 的杰夫·克拉维尔和 TechStars 的大卫·科恩。(披露:TechCrunch 使用 isocket 来推广其直接广告。)

除了新的资本注入，isocket 宣布它已经从 [Rubicon 项目](https://web.archive.org/web/20230203053140/http://www.rubiconproject.com/)挖走了 Ben Trenda。Ben 将以销售副总裁的身份加入 isocket。 [TechCrunch 报道了他于去年 4 月从 Aol](https://web.archive.org/web/20230203053140/https://techcrunch.com/2010/04/05/former-aol-experian-executives-join-rubicon-project/) (在那里他是代理关系副总裁)加入 Rubicon。在 Rubicon，Trenda 负责启动该公司的实时竞价业务，并帮助该项目发展成为仅次于谷歌的第二大广告交易所。

除了为 Mashable、VentureLoop、GameZebo 和 Appolicious 等出版商提供直接广告外，isocket 还正式宣布，它已经将包括 TUAW 在内的 Aol Tech properties(并将很快加入 Joystiq 和 Engadget)加入其客户名单。isocket 还与谷歌建立了供应方合作伙伴关系，以整合其 DoubleClick 广告服务器，这意味着该公司将很快直接插入 DoubleClick For Publishers(大多数主要出版商使用的服务)，从而直接向客户销售广告，而没有传统的麻烦。

isocket 创始人兼首席执行官 John Ramey 告诉我们，像 Cheezburger Network 这样的出版商已经在使用这种双击集成，isocket 和 Google 的采用情况都非常好。

为了让大家了解 isocket 的工作，这家初创公司的服务让广告商可以轻松地与优质出版商联系购买广告，反过来，也让这些出版商可以轻松地销售他们的数字广告。传统上，对于想卖广告的出版商，销售人员(或者创始人等。)必须通过电话、电子邮件和 Excel 来管理与买家的关系。

它包括收集广告模型、盘点、冗长的广告大小、格式、定价等。和管理整个过程的电子表格。简单地直接购买一个广告，一个只需点击两下就能完成的行为，变成了一个长达数周的过程，这对每个参与其中的人来说都是一件痛苦的事情。

isocket 的产品寻求简化这一过程，允许出版商建立个人资料，创建他们的广告清单，然后基本上提供一个微型市场，广告商可以在这个市场上细读这些广告和所有相关信息，点击几下就可以购买他们想要的广告。这是一个自助服务平台，一旦有兴趣的买家选择购买广告，出版商就会直接收到电子邮件，允许他们接受或拒绝。一旦他们批准，isocket 就会处理剩下的事情。

今天，这家初创公司宣布了一项为广告商增加额外功能的功能，允许他们直接从他们已经使用的代理软件下单——他们现在可以搜索优质出版商的列表和他们的最佳库存，而不必担心使用其他平台做生意。

本质上，随着这些新功能添加到其 [BuyAds](https://web.archive.org/web/20230203053140/http://www.buyads.com/) 平台，isocket 已经创建了一个强大的优质库存平台，并通过其 API 受益于程序化直销。这意味着出版商现在可以直接与广告商已经使用的购买工具分享他们的库存，如需求方平台和媒体策划。换句话说，麻烦更少，销售更快，订单更多。

现在，通过 isocket 的 API，用户可以搜索平台的优质出版商管理目录，也可以搜索第三方出版商，查看哪些库存以什么价格提供，直接下订单(和编程)，进行支付，并访问广告绩效指标。

其中一些听起来可能晦涩难懂，有点小众，但对于广告商、营销商和出版商来说，isocket 正在建立一个非常漂亮的平台，真正将直接广告销售带入数字时代。
