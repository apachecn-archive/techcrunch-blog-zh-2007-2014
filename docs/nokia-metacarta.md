# 诺基亚收购地理搜索技术公司 MetaCarta

![](img/87a849f58d508aa7470f5d9dee8cdd4f.png)今天早上[宣布](https://web.archive.org/web/20221209072329/http://www.prnewswire.com/news-releases/nokia-acquires-metacarta-inc-90346994.html)已经收购了私有地理情报公司 [MetaCarta](https://web.archive.org/web/20221209072329/http://www.metacarta.com/) ，金额未披露。诺基亚表示将在本地搜索和其他服务领域使用 MetaCarta 的技术。

MetaCarta 成立于 2001 年，总部位于马萨诸塞州的剑桥，拥有 30 多名员工。

MetaCarta 的解决方案使公司能够弥合内容和地图之间的差距。它的[产品](https://web.archive.org/web/20221209072329/http://www.metacarta.com/products-overview.htm)分为四大类:MetaCarta 平台、地理数据模块、GeoWeb 应用和托管内容集合。

要查看 MetaCarta 技术能力的示例，请参见 [NewsMap](https://web.archive.org/web/20221209072329/http://www.metacarta.com/products-geoweb-applications-newsmap.htm) ，这是一个托管地图混搭/扩展，使网络出版商能够动态显示地图，显示他们新闻文章中提到的地方。或者看看它的地理搜索新闻服务，它从成千上万的来源收集新闻，并显示故事中提到的地点的地图。

MetaCarta 还与微软建立了合作关系，提供地理搜索和参考平台，在微软 Vine 服务中提供基于地图的本地新闻。

对于诺基亚来说，这是两周内的第二笔小额收购——它最近收购了移动网络浏览器公司 Novarra。它对 MetaCarta 的收购再次证明了这家芬兰移动巨头对定位和导航服务的关注，这些服务面向全球的[消费者。](https://web.archive.org/web/20221209072329/http://www.beta.techcrunch.com/2010/01/28/nokia-q4-2009-earnings/)
