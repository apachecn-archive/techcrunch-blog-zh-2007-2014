观看面向Android的Opera Mobile:快速、缩放、地理定位等技术

原文：https://web.archive.org/web/https://techcrunch.com/2010/11/08/opera-mobile-for-android/

看看Opera Mobile For Android:速度快、缩放、地理定位等等

Opera软件已经在发布了其 Android 平台的(第二款)免费手机浏览器产品。获取应用程序，被称为 Opera Mobile 10.1 beta ，现在在Android Market或这里。

是的，Opera已经可以在Android上使用了，只是以一种相当有限的方式。 Opera Mini ，自7月起已上市，但现在你也可以从应用商店下载该公司的成熟浏览器 Opera Mobile 。

这是它的包装:

–与其他浏览器相比，页面加载速度更快，尤其是在较慢的网络上(承蒙Opera Turbo 的支持)

–平滑滚动、缩放和平移(而不是点击放大，这绝对是一个巨大的进步

–快速拨号

–可视化标签便于标签管理—>

–内置地理定位支持

–能够跨设备同步书签

–SVG支持

正如我之前提到的那样，Opera Mobile面向Android的公开测试版的发布只是时间问题，因为产品已经存在，尽管只面向挪威公司的OEM合作伙伴。

既然最终用户可以直接下载，不去看看就错了。

TechCrunch作家Jason Kincaid今天早上在他的Android智能手机上安装了一个应用程序，并证实它比安装在它上面的原生浏览器感觉更快。显然，这里欢迎更健壮的并行测试。

浏览器提供中文、捷克语、丹麦语、荷兰语、英语、法语、德语、意大利语、日语、韩语、挪威语、波兰语、葡萄牙语、俄语、西班牙语和瑞典语版本。
