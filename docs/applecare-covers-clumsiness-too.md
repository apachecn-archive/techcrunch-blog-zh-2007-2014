# AppleCare+也包括笨拙

我经常摔我的 iPhone。事实上，自从我决定保留我的 3GS 并坚持到昨天发布了一个非常令人失望的公告后，我就越来越不在乎它何时落地了。一旦我得到了 iPhone 4S…嗯，这是另一个故事。尽管如此，事故总是会发生，对于苹果公司要求的现金，一点保险也无妨。输入: [AppleCare+](https://web.archive.org/web/20230205041254/http://store.apple.com/us/product/S4575?mco=MjU5MTk4MjQ) 。

如果你曾经用过苹果产品，那么你一定听说过 AppleCare。就 iPhone 而言，AppleCare 为你提供两年的技术支持和硬件维修，只要手机的损坏不是由你自己的笨拙或液体造成的。现在，苹果公司将费用从 69.99 美元提高到 99.99 美元，并对协议进行了一些调整。

AppleCare+提供长达两年的相同技术支持和硬件维修，只是这次苹果将“由于搬运造成的意外损坏”包括在其覆盖范围内。然而，不要太激动，把你的手机扔到房间的另一边，只是为了好玩。说到“意外处理”，你只有两次机会，每次需要额外支付 49 美元。所以，如果你把 iPhone 掉进马桶里，这种情况比你想象的要多得多，不要担心。拿起 50 美元，前往苹果公司，这个问题应该会很快得到解决。

就定价而言，如果我们仍然要为意外处理支付费用，那么苹果为什么要将 AppleCare+的价格提高到 99.99 美元还有点不清楚。另一方面，即使有额外的 30 美元首付，50 美元仍然比更换一个屏幕或完全购买一部新 iPhone 的成本要低。
