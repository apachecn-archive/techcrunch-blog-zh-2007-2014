# (创始人故事)Birchbox:帮助人们“找到他们不知道自己想要的东西”

在克里斯·迪克森与 T2 的 Birchbox 创始人海莉·拜尔娜和卡蒂亚·博尚的访谈的第二部分，迪克森问 Birchbox 是否符合订阅商务的趋势。

波尚认为他们有。与此同时，拜尔娜指出订阅“有许多不同的风格”。与 Diapers.com 相比，拜尔娜说[尿布](https://web.archive.org/web/20230204180035/http://www.crunchbase.com/product/diapers-com)解决了“方便”和“补充”，而 Birchbox 为人们提供了一种“找到他们不知道自己想要的东西”的方式三人讨论了占据这一领域的其他公司，如 ShoeDazzle 和 BeachMint。

Beauchamp 认识到他们的商业模式是可行的，他们可以扩展到销售美容用品以外的产品，他告诉 Dixon 他们选择 Birchbox 这个名字是为了给用户提供灵活性。例如，拜尔娜说“我们已经测试了非美容产品”，如“茶、巧克力或文具”

继续下面的对话，Dixon 询问了快速增长的挑战。Beauchamp 说，其中一个挑战是，“我们等了太久才雇用，这就像一个恶性循环，你没有时间雇用，因为你真的忙于做每件事，但当然如果你雇用了你会有更多的时间。”

然后，讨论转向了订阅成为新的每日交易商业模式的话题，以及随之而来的问题(例如，充斥着竞争对手)。Beauchamp 说这是他们的想法，但认为 Birchbox 独特的“声音”和“个性”提供了辩护理由。
