# KidZui 筹集 240 万美元，成立 Zui 工作室，为孩子们制作更多视频

独家消息——kid zui 是一家家庭友好型互联网产品的制造商，如[安全浏览器](https://web.archive.org/web/20221007025915/https://beta.techcrunch.com/2009/11/18/kidzui-browser-kids/)和[儿童视频目的地](https://web.archive.org/web/20221007025915/http://zui.com/)，该公司从现有投资者那里又获得了[240 万美元](https://web.archive.org/web/20221007025915/http://www.crunchbase.com/company/kidzui)的资金，由 Mission Ventures 和 Costella Kirsch 牵头。

这笔资金将用于开发一个新的视频制作部门，名为“醉工作室”。

部分资金还将用于首秀(Zui Buzz，见下文)的推广和 Zui.com 的[进一步发展，该公司称其为最安全的儿童一站式内容聚合和搜索引擎。](https://web.archive.org/web/20221007025915/https://beta.techcrunch.com/2011/08/10/kidzui-launches-zui-com-an-internet-experience-for-kids-raises-2-million/)

KidZui 已经聘请亚当雅各布斯(Adam Jacobs)担任 Zui 工作室的执行制片人，他将负责该部门的所有内容创作。雅各布斯曾两次获得艾美奖最佳导演奖提名，他曾负责时代华纳有线电视的宣传，并为 HBO 电视剧《黑暗面的故事》撰稿。

Zui Buzz 是 Zui 工作室的第一部作品，每天为孩子们更新网络趋势，包括最好的视频、网站和游戏，将在 Zui.com 主页上播放。

该公司还将在 YouTube 上寻找拥有大量观众、非凡激情和/或才华的孩子，将他们推广到[【Zui Stars】](https://web.archive.org/web/20221007025915/http://zui.com/stars)，旨在给孩子们“他们所寻求的病毒式视频明星，而无需父母担心的曝光率”。

所有评论和视频回复都由 KidZui 内容策展团队筛选，该团队包括家长和老师。

KidZui 创始人兼首席执行官 Cliff Boro 告诉我，KidZui properties 在 2011 年 11 月接待了大约 180 万独立访客，并且还在全面增长。

Kidzui 成立于 2006 年，总部位于加州圣地亚哥。
