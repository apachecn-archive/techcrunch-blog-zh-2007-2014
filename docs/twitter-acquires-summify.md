# Twitter 收购、关闭社交新闻初创公司 Summify

[sum mify](https://web.archive.org/web/20230307063453/http://www.summify.com/),[利用社交数据创建个性化新闻摘要的初创公司](https://web.archive.org/web/20230307063453/https://techcrunch.com/2011/03/30/summify/)，刚刚[宣布](https://web.archive.org/web/20230307063453/http://blog.summify.com/2012/01/19/summify-joins-the-flock-at-twitter/)被 Twitter 收购。

这听起来像是 Twitter 的人才收购——换句话说，收购的主要目的可能是雇佣 Summify 团队。Summify 的一些功能已被立即禁用，不再接受新用户，几周后，Summify 表示将完全关闭当前产品。与此同时，这家初创公司将从温哥华搬到旧金山，在 Twitter 办公室工作。

该公司的投资者包括 Accel Partners、Rob Glaser、斯图尔特·巴特菲尔德、Steve Olechowski 和加拿大超级天使 Boris Wertz。

不管 Twitter 最终是否会使用 Summify 的技术，这笔交易看起来都是一笔不错的交易。Twitter 已经做了更多([特别是在新版本的服务](https://web.archive.org/web/20230307063453/https://techcrunch.com/2011/12/08/live-at-twitters-come-see-what-were-building-press-conference/))来突出显示 Twitter 上的新闻，而 Summify 则是通过查看社交图来为每个用户找到最相关的新闻。

Summify 团队在其博客上写道:

> 我们在 Summify 的长期愿景一直是以最省时的方式为人们提供最相关的新闻。随着全球数亿人注册和消费 Twitter，我们意识到这是在真正的全球范围内实现我们愿景的最佳平台。由于 Twitter 与我们分享了这一愿景，加入该公司是非常明智的。
