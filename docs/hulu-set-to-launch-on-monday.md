# Hulu 将于周一推出

一位消息人士称，在准备了[整个夏天](https://web.archive.org/web/20230325061458/https://techcrunch.com/2007/08/29/clown-co-gets-website-and-new-name-hulu/)， [Hulu](https://web.archive.org/web/20230325061458/http://www.crunchbase.com/company/hulu) (即将与 YouTube 竞争的 NBC-Fox 合资公司)本应于昨天推出私人测试版。这就是为什么 NBC [上周从 YouTube 上撤下了它的视频](https://web.archive.org/web/20230325061458/http://valleywag.com/tech/online-video/nbc-pulls-youtube-channel-313276.php)。(这是在 NBC [为亚马逊](https://web.archive.org/web/20230325061458/https://techcrunch.com/2007/09/04/nbc-runs-from-itunes-into-the-arms-of-amazon/)放弃 iTunes，并已经开始[在网上提供其节目的免费下载](https://web.archive.org/web/20230325061458/https://techcrunch.com/2007/09/19/is-hulu-doomed-nbc-to-begin-offering-free-downloads/)之后)。

但是创建一个视频网站可能很难——即使它只是一个私人测试版。现在，我们的消息来源称，Hulu 正在争取下周一推出私人测试版，这样它仍然可以满足首席执行官 Jason Kilar 自己设定的 10 月最后期限。这个日期可能会推迟到下周三，从技术上来说当然是准时的，但 Kilar 可能想避免恶作剧还是招待的玩笑。我们还在等我们的邀请。
