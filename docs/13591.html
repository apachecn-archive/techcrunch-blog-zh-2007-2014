python For Microsoft Excel Company iron spread退出测试版，将名称改为DataNitro

y CombinatorWinter2012年夏季毕业生 Data Nitro (原名IronSpread)有一个简单的提议:它能让你在Microsoft Excel中使用流行的编程语言Python。该插件对个人非商业用户是免费的，企业将为其特权付费。到目前为止，它只适用于Windows的Excel 2007和2010。该公司没有支持OSX的计划。

尽管高级用户长期以来一直能够使用微软的Visual Basic for Applications (VBA)来编写Excel脚本，并且微软也有望在Office中支持JavaScript，Python是Excel jock工具箱的强大补充。根据T2的多项指标来看，Python对于程序员来说就算不是顶级语言，也是顶级语言。它干净、直接的语法使它成为初学者和编程入门课程的热门选择，但它在许多地方的产品中也很受欢迎，包括Google 和NASA 。这种语言有很多令人喜欢的地方，花在学习Python for Excel脚本上的努力可以很好地移植到其他应用程序中。

好消息是:你仍然可以使用你所有的旧VBA脚本，Data Nitro不会阻止它们工作。另外，您可以使用现有的Python库。坏消息是:你发送电子表格的任何人都需要安装插件才能运行Python脚本。

在Excel中处理Python代码的其他工具包括 Pyvot 和 Pyxll (链接见黑客新闻社区)。
