# (创始人故事)迈克·拉泽罗在谷歌+“我不知道谁在用它”

在 Chris Dixon 的[上一集《创始人故事》](https://web.archive.org/web/20230205003726/http://www.crunchbase.com/person/chris-dixon)[中，巴迪媒体的](https://web.archive.org/web/20230205003726/https://techcrunch.com/2011/08/30/founder-stories-lazerow-facebooks/)[迈克·拉泽罗](https://web.archive.org/web/20230205003726/http://www.crunchbase.com/person/michael-lazerow-3)告诉 Dixon，巴迪媒体是“脸书+”，这意味着巴迪媒体帮助品牌管理消费者在社交媒体平台上的互动和参与，而不仅仅是脸书。

在这一集里，迪克森问拉泽罗，与脸书相比，品牌如何看待使用 Twitter 作为平台和广告场所。

拉泽罗承认，“Twitter 更多的是一项正在进行的工作。”他承认 140 字的字数限制了创意内容，但指出 Twitter 有潜力。“当你看到你可以用照片做什么，你可以用生态系统做什么……无论是 Twitter-Tumblr 整合，Twitter-脸书整合，你都可以突然之间……在全球范围内建立一个可扩展的存在。”

两人都认为 Tumblr 也是一个有前途的品牌平台，但其有效性仍有待评判。对话以 Google+结束。拉泽罗并不以为然。他说，除了少数人，“我不知道谁在使用它。”

下面，拉泽罗告诉迪克森“我爱谷歌，因为谷歌做得很好”，包括搜索和 Gmail。话虽如此，拉泽罗并不认为 Google+或任何类似的社交网站会取代脸书，他说“以一种对你作为一个人来说很重要的方式把你和人们联系起来，这是很难设计的。”事实上，如此难以设计，以至于拉泽罗认为脸书是“永远安全的”
