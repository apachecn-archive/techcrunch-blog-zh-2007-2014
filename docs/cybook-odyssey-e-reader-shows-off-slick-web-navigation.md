# Cybook Odyssey 电子阅读器展示了触摸屏和网络导航

这可能不会对美国的电子阅读器大战产生太大影响，但我不能对这样一款漂亮的设备说不。当他们把荷马也搬上银幕时，那只是锦上添花。[法国电子阅读器制造商 Bookeen 的 Cybook Odyssey](https://web.archive.org/web/20230203144037/http://bookeen.com/blogfr/bookeen-oddyssey-cp01/) 有一两个锦囊妙计，我希望我们的美国设备有一天能学会。

奥德赛运动被布克恩称为高速墨水系统，或 HSIS。他们在一篇评论中模糊地描述了这一点:布克恩长期以来一直致力于改善 E-Ink 屏幕的性能，将其作为硬件(处理器……)的一部分，而不是“软件(固件优化)”所以我猜他们基本上超频了一个 E-Ink 的珍珠屏幕，使其更新更快。这对于显示动态图像非常方便——你可以制作翻页和按钮之类的动画。

这也带来了更有用的网络浏览体验。[你实际上可以在](https://web.archive.org/web/20230203144037/http://www.youtube.com/watch?v=CxwceUvxlCo&feature=player_detailpage)左右移动页面，你不必等待缓慢的电子墨水刷新率赶上来。它并不完美，但可能会很方便。你可以通过古登堡计划和谷歌图书网上找到很多书，这种方式也是可行的。

当然，拖动并不是全新的事情。你可以看到我在最新的 Kobo 电子阅读器上拖着一个 PDF 文件[在这里](https://web.archive.org/web/20230203144037/https://techcrunch.com/2011/06/13/review-kobo-ereader-touch-edition/)(大约 3:30 进入视频)。奥德赛的版本看起来更流畅一点，当然它也适用于网络，而不仅仅是书籍。

总之，这是一个好看的小装置。还没有定价，但这将是欧洲唯一的。
