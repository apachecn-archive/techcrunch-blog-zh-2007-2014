# 脸书通过给用户一年的回顾来纪念 2012 年

今天，脸书发布了 2012 年回顾，记录了过去一年最重要和最受欢迎的事件、歌曲、电影、人物、技术和地点。你在那里看到的大部分都是你所期待的:饥饿游戏、巴拉克·奥巴马和 YOLO 这个词在过去的 12 个月里对人类的发展都至关重要。

但脸书也向其超过 10 亿的用户赠送了数据分析的礼物，在每个人的个人时间线上提供个性化的年度回顾。就像推特在回顾其[年](https://web.archive.org/web/20221209024954/https://beta.techcrunch.com/2012/12/11/twitter-releases-its-2012-year-in-review-and-its-an-immersive-and-personal-experience/)时所做的一样。

如果你去 Facebook.com/yearinreview[旅游，你会很快看到你一年中最重要的 20 个时刻，尽管还不清楚脸书是如何做出这些决定的。](https://web.archive.org/web/20221209024954/http://facebook.com/yearinreview)

例如，在我的时间线上，我的状态更新为“恋爱中”可能是我 2012 年最重要的时刻之一，但那次约翰·比格斯劫持了我的脸书账户，并发布了“我喜欢玉米。”不是。无论如何，脸书包括他们两个。从那里，用户还可以通过一个简单的分享按钮分享他们一年中的重要时刻。

我们与脸书谈论了个性化的回顾年时间表，并被告知这个想法是“给人们一个回顾 2012 年的互动体验”。人们会注意到他们已经忘记的故事，这使它更加怀旧。”
