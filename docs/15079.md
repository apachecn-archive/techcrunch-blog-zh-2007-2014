早在 2012 年 1 月，[一项名为 Burn Note 的服务推出了](https://web.archive.org/web/20230118194151/http://thenextweb.com/apps/2012/01/31/privacy-isnt-an-issue-with-burn-note-because-all-messages-will-self-destruct/)，旨在通过在一定时间后销毁来保护你的私人信息。几乎在同一时间，随着 [Snapchat](https://web.archive.org/web/20230118194151/https://techcrunch.com/2012/06/08/snapchat-110-million/) 的推出和小小的争议，短暂信息的概念在主流用户中流行起来。虽然 Snapchat 允许你发送自毁的照片和视频，但脸书迅速执行了~~复制~~[的东西，Burn Note 又回来了，它仍然专注于沟通的直接消息方面。](https://web.archive.org/web/20230118194151/https://techcrunch.com/2012/12/21/facebook-poke-app/)

今天， [Burn Note](https://web.archive.org/web/20230118194151/http://www.burnnote.com/) 推出了新的 iOS 和 Android 应用，这些应用具有一些非常有趣的功能，可以限制消息的查看区域，以进一步防止它们被收件人截屏。虽然这听起来可能过于偏执，但在这个世界上，像这样与性短信毫无关系的技术绝对有用武之地。

我与 Burn Note 的创建者 Jacob Robbins 进行了交谈，他解释了查看邮件的新“聚光灯”方法，这真的很酷。不仅如此，它还使用了正在申请专利的技术，因为自去年年初推出这项服务以来，罗宾斯显然已经在这项服务上花了时间。Spotlight 的作用是强迫你用手指或鼠标在信息上移动，一次只显示一部分。这可以很好地阻止人们截屏或复制你的信息，也可以阻止那些讨厌的人从你身后偷看。

下面简单介绍一下它的工作原理:

[<amp-anim class="aligncenter size-full wp-image-785587 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="burnnote_techcrunch" src="https://web.archive.org/web/20230118194151im_/https://techcrunch.com/wp-content/uploads/2013/03/burnnote_techcrunch.gif" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-anim>](https://web.archive.org/web/20230118194151/https://techcrunch.com/wp-content/uploads/2013/03/burnnote_techcrunch.gif)

Burn Note 中的邮件使用一个计时器自毁，一旦收件人打开邮件，计时器就会启动。该服务将猜测读者需要阅读多长时间，或者你可以自己设置时间。一旦计时器超时，你的信息将被永久销毁。通过销毁，罗宾斯说，所有的信息数据都被安全地从刻录笔记服务和两个参与者的设备中删除。对于像 Burn Note 这样的服务来说，这是一个关键的组成部分:[如果失去了这种信任](https://web.archive.org/web/20230118194151/https://techcrunch.com/2013/01/22/not-so-ephemeral-messaging-new-snapchat-hack-lets-users-save-photos-forever/)，那么 Burn Note 成功的机会也就没有了。你可以向其他 Burn Note 用户、电子邮件地址发送消息，或者向任何平台上的任何人发送链接。

罗宾斯说，Burn Note 的目标是:“允许在线交流与面对面交谈一样亲密；比在社交网站上分享、发送电子邮件或使用短信更私人。”

我向罗宾斯询问了一项服务和信息平台的一些潜在应用，如 Burn Note 和他创造的反间谍聚光灯技术:

Jacob Robbins: 这款应用的第一版发布令人惊讶。我没有做过太多的研究，不知道人们听到短暂信息这个概念时的第一反应是什么。我非常惊讶地发现，人们经常将这项技术与抱怨他们的老板或其他同事的方式联系起来，因为他们是工作场所压力的来源。我没有意识到这种工作压力几乎是普遍存在的，也许我在初创公司的回音室里呆了太长时间。但是人们不太理解这个系统是如何工作的，因为第一个版本是非常单向的。他们会听说最初的产品，然后想“我能说出谁是谁，让他们永远不知道是我吗？”幸运的是，他们没有这样做，因为 Burn Note 不是匿名的，但显然这不是一个好的初始反应，我意识到我需要重新包装它，使其更类似于 IM 客户端或 SMS 应用程序，这两者都是非常熟悉的对话工具。

**TC:** 一路走来，你从用户身上学到了什么？

**Robbins:** 当我第一次制作 Burn Note 的时候，我非常专注于让用户相信他们的信息不会保留在服务器上的技术。来自用户的反馈几乎完全忽略了这一点，而是关注于他们如何相信他们的消息没有被保留在接收者一方。

阻止收件人保留您发送给他们的邮件的副本是一个猫捉老鼠的游戏，没有完美的解决方案，因为通信的目的是让收件人以某种方式保留信息的副本(例如，在他们的大脑中)。但更大的问题是，发送者往往比接收者更关心他们的消息的安全性，他们需要工具来帮助他们。这就是为什么我们觉得聚光灯很适合这种情况。它在这方面提供了几个不同的好处，包括抵制复制和粘贴，抵制截图，还限制了收件人附近的人偷偷查看邮件。

我们收到的一个更有趣的回复来自一位理发师，他在第一次使用聚光灯时说，“当我在沙龙阅读信息时，这对我真的很有用，这样人们就不会走到我身后看到我的对话”。我们从来没有考虑过用户在工作场所进行个人短信对话时会遇到的隐私问题，在那里，每个人都整天站着，到处走动。打破办公桌/隔间思维模式对于理解移动应用的真实使用情况非常重要。

**Robbins:** 在选择 Spotlight 之前，我们尝试了很多显示信息内容的不同方法。在其中一种方法中，我们将信息分解成短语，然后一个接一个地显示出来，就像在抽认卡上一样。我们甚至试着把信息做成视频，在每一帧中显示信息的随机部分，但是当连续观看时，你可以看到整个信息是一种不稳定的静态图像。最后，聚光灯法是人们第一次使用时立即产生情感反应的一种方法，这也是它吸引我们的原因。

**TC:** 你如何看待其他应用程序，特别是 Snapchat，因为你提供的消息类型而受到指责？

**Robbins:** 看到媒体如何报道 Snapchat 很有意思，因为很多记者和博客作者显然不知道该怎么做。写得最多的故事是，青少年主要使用 Snapchat 发送性短信，但这显然不是真的，除非性短信是主要在上学时间进行的日常活动，在这种情况下，学校管理人员可能会大声权衡。一方面，你有非常高的使用量，这显然是值得注意的，另一方面，你有记者推出的故事说，“孩子们都是性短信！”这让我想起了汤姆·沃尔夫对 60 年代早期主流媒体抱怨披头士狂热的描述，抱怨尖叫的青少年人群，称“我想牵你的手”“我想牵我的鼻子”等等。

**TC:** 谈谈这款应用的一些你从未想过会看到的用法。有企业使用吗？

据我所知，我们还没有任何企业级的应用。运营 Burn Note 的部分挑战是，确保邮件内容保持私密是我们工作的重中之重，而分析使用模式必须退居其次。因此，我们很少了解人们使用它的目的。我们非常依赖来自客户服务和调查的轶事反馈来获得用例细节，尽管我们确实有总量来衡量数量。今年早些时候，我们与一家英国私人调查公司进行了客户服务互动，这是一个有趣的专业使用案例，但大多数反馈来自使用它与朋友和家人交谈的人。

———

这显然不是一个会从运营商那里夺走大量短信使用的应用程序，但对于那些可能羞于说出某些事情或只想抱怨同事而不担心社会反响的人来说，这绝对是一个有趣的提议。当然，你可以为像 Burn Note 这样的应用想出许多令人讨厌的用法，但最终，你可以为任何服务这样做。

要欣赏这种服务，尝试一下是必须的。在有些情况下，我可以看到自己通过 Burn Note 而不是 SMS 或电子邮件发送消息，这让他们自己可以访问互联网或其他人的收件箱。当然，我们应该相信我们的朋友不会这样做，但它确实发生了。我的个人收件箱里有些东西对我来说太私人了，我通常会说些什么，而大多数人可能不会这么做。

如果你想给某人发送一条非常私人的信息，也许是像你有多喜欢他们这样单纯的事情，像这样的服务可能会给你这样做的勇气。如果你对某人的爱没有得到回报，那么就没有伤害或犯规。当然，另一个人可以对他们的朋友说“嘿，他们给我发了一条信息，约我出去”，但是没有看到这条信息，社交恐惧是有限的。
