# 语法推出预算 65 英寸，LCoS 1080p 高清电视

![olevia-lcos-65-hdtv.jpg](img/a2f1b4065dac6ca644842becbe689873.png)

首先，我要说的是，我非常自豪地拥有一台 37 英寸的欧莱雅液晶高清电视。我喜欢这东西。我在 Fry's 买的全新的，含税不到 600 美元。这是一个了不起的交易，尤其是因为电视本身是 kickass。但现在我在考虑升级，因为 Syntax 下周将推出 65 英寸的 LCoS 电视，它承诺也会很棒。

LCoS 是 DLP 和 LCD 的近亲，使用 DLP 背后的镜像理论，但使用硅(不是硅)上的标准液晶。它能产生明亮、清晰的图像，而且制造成本相对低廉。这意味着这款 65 英寸、1080P 高清电视的零售价格将低于两倍大，非常划算。

我们期待着在黑色星期五的“破门者”特辑中看到这个单位，所以请让我们一直关注你的屏幕，看看它有多低。
