# 你现在可以直接从 Lexmark 打印机向 Evernote 发送扫描结果

Evernote 是一项伟大的网络服务，让人们能够在各种环境下捕捉和存储信息、记忆和内容，今天早上宣布 Evernote [SmartSolution](https://web.archive.org/web/20221007063325/https://smartsolutions.lexmark.com/ssds/index.jsp) 应用程序可用于利盟的三款喷墨打印机:Interact、Prestige 和 Platinum。

Evernote SmartSolution 应用程序基本上允许用户从这些 Lexmark 设备直接发送硬拷贝扫描到他们的 Evernote 网络帐户，只需轻触一个按钮，不需要计算机。

Evernote 应用程序由 Lexmark 开发，该服务的快捷方式可以在几分钟内添加到打印机触摸屏上。设置完成后，点击 Evernote 应用程序可以通过网络向预先配置的 Evernote 账户发送扫描结果。

去年年底，[风险投资支持的](https://web.archive.org/web/20221007063325/http://www.crunchbase.com/company/evernote) Evernote 有大约[200 万人](https://web.archive.org/web/20221007063325/https://beta.techcrunch.com/2009/12/30/evernote-two-million/)使用其服务，可以想象，其中只有一小部分人真正拥有一台 Lexmark 打印机。也就是说，我喜欢能够在电脑和手机之外的设备上管理网络服务的想法。

现在他们能帮我接冰箱了吗？
