# Orbotix 发布了售价 99 美元的新型超高速可编程机器人 Sphero 2B

Sphero，顾名思义，是一个机器人球体。现在 [Orbotix](https://web.archive.org/web/20221209122141/https://beta.techcrunch.com/tag/Orbotix) ，这个心爱的球的制造商，已经决定把它拉长成一个管子(实际上是一个圆柱体)，把价格降到 99 美元，并做一个我见过的最有趣的小玩具。如果最初的 [Sphero](https://web.archive.org/web/20221209122141/https://beta.techcrunch.com/tag/Sphero) 是 Eve，那么他们的新产品 2B 就是 Wall-E。这款充满活力的设备的速度是当前版本 Sphero 的两倍，并将在今年秋天[发布时售价约为 99 美元](https://web.archive.org/web/20221209122141/http://gosphero.com/2B)。

[gallery ids="936559，936560，936561，936562，936563，936581，936582，936583，936584"]

Orbotix 联合创始人 Ian Bernstein 表示，Sphero 是首批联网设备之一，这是一种与智能手机相连的设备。通过扩展原有产品，同时简化设计，他们认为他们能够让 2B 进入更多的客厅。

“2B 看起来和驾驶起来都像是来自未来的东西，”伯恩斯坦说。该公司将为 2B 创造配件，包括户外游戏的小块轮胎和内置的红外传感器和灯，让你可以用多个 2b 玩标签，并在检查站之间进行比赛。

我们有机会在 2014 年 CES[上与 2B 一起玩，我被深深打动了——它看起来很有趣。](https://web.archive.org/web/20221209122141/https://beta.techcrunch.com/tag/ces-2014)

该公司还更新了他们的应用程序，以支持 Sphero 2.0 的更多功能，并宣布了 [Sphero Rangers](https://web.archive.org/web/20221209122141/http://www.gosphero.com/education/) ，这是一个用 Sphero 作为现实生活中的标志海龟来教孩子们编程和几何的学校计划。2B 将使用大多数这些新的编程功能。
