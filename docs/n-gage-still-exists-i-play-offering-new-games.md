# N-Gage 依然存在，I-play 提供新游戏

![nokia_ngage_qd_grey_front.jpg](img/5d0918bfdbe6105b5f01440a4ca7e4c6.png)

你最后一次看到有人拿着诺基亚 N-Gage 是什么时候？我很确定我最后一次看到它是在 2003 年在~~T 商店的~~Cingular。我实际上忘记了他们的存在，直到 15 分钟前我看到了这个。N-Gage 平台将从今年秋天开始从 I-play 获得一套全新的游戏，包括 *WRC* (世界拉力锦标赛)和*超级麻将*。I-play 的游戏将专注于容易玩但难以掌握的东西。I-play 主要针对“一个拇指”人群，它肯定会将 N-Gages 的销量提高两倍。也许我会看到有人拿着 PSP 以外的东西。
