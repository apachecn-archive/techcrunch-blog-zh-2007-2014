# 谷歌手机地图在没有 GPS 的情况下精确定位你的位置

[![google-maps-logo.png](img/54a6c256fc21c04c62b2f5cbd4e3e42a.png)](https://web.archive.org/web/20230307213235/https://techcrunch.com/wp-content/uploads/2007/11/google-maps-logo.png "google-maps-logo.png") 谷歌一直在以迅猛的速度向其地图添加功能。昨天，它是[地形和维基风格的合作](https://web.archive.org/web/20230307213235/https://techcrunch.com/2007/11/28/google-earth-heading-towards-extinction/)的网络地图。今天，它[更新了它的手机地图](https://web.archive.org/web/20230307213235/http://www.google.com/intl/en/press/annc/20071128_maps_mobile_my_location.html)，通过手机信号塔之间的三角定位来确定你的位置(或者如果你的手机上有 GPS，它会使用它，但今年售出的手机中只有 15%支持 GPS)。我刚刚把“我的位置”应用程序下载到我的黑莓手机上(不带 GPS ),它把我定位在曼哈顿我所在位置的一个街区内。我是地图上一个闪烁的小蓝点。不过，如果我在办公室走动，它会接收到另一组信号塔，把我带到近六个街区以外的地方。哦，等等，现在我几乎完全在正确的街区了。现在它又把我带到了拐角处。至少它有合适的社区——而且曼哈顿确实有很多手机信号塔。
