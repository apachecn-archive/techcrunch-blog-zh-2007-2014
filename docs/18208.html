Tappy Fit使用Fitbit数据将Flappy Bird上瘾转化为健身动机TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2014/03/21/tappy-fit-channels-flappy-bird-addiction-into-fitness-motivation-using-fitbit-data/

现在在应用商店上发布的一款新游戏的特点几乎与病毒热门游戏《Flappy Bird》中的相同，但有一个关键的变化——游戏可以通过纯粹的体力劳动变得更容易。使用 Fitbit API ，开发者Aaron Coleman内置了一个可变难度引擎，允许你兑换一天内计算的步数，以换取游戏内的奖励，比如你必须通过的管道之间更宽的间隙来导航你的鸟化身。

除了是T4众多Flappy Bird克隆体中的又一个，Tappy Fit还有趣地展示了开发人员可以利用Fitbit、Nike、Jawbone等公司向他们提供的新健身API做些什么。特别是在游戏领域，未来的集成有很大的潜力。

科尔曼说，Flappy Bird成为目标不仅仅是因为它的高调——更多的是因为这款休闲游戏在很多圈子里的坏名声。

“据报道，这种类型对生活产生了一些上瘾和负面影响，”他在一次采访中解释道。“这个想法是在思考我们如何将它重新定义为对健康有益之后产生的。从一个开源项目开始，我们只花了几天时间就完成了开发，然后对它进行了改造。”

Tappy Fit背后的想法不是仅仅为了积分而提供积分，尽管这样做似乎已经为Flappy Bird带来了很多参与。相反，科尔曼和他的小步骤实验室初创公司正在研究如何能够创造一种基础体验，而不仅仅是短暂、表面的回报。

“为了分数而分数不是我们感兴趣的东西，”科尔曼说。“将人们重视的东西与他们的长期健康习惯结合起来，对我们来说非常有趣。就Tappy Fit而言，如果你喜欢快速游戏，那么让你在游戏中更成功，因为你更成功地实现了你的分步目标，这是我们认为与你的生活有意义联系的一个例子。”

至于这个项目有多原创，科尔曼指出，虽然有很多游戏可以跟踪你的健身情况，并旨在将锻炼游戏化，但这是他能想到的唯一一个利用你在一定时期内的总运动量来影响游戏的例子。科尔曼之前为孩子们开发了一款名为Food Buster的卡路里追踪游戏，该游戏在2010年赢得了米歇尔·奥巴马的“让我们动起来”应用程序挑战人气大赛，因此他一直在思考如何将健康和游戏结合在一起。

“有大量的文献表明，让人们注意自己的日常身体活动水平，会让他们更有可能提高身体活动水平，”他说，并描述了为什么他认为这里有更多的机会。“也有很多‘健康游戏’是在健身房、跑步时或在屏幕前玩的。例如，Wii Fit和XBox Kinect能让你立刻投入其中。”

Tappy Fit是一项实验，旨在看看在健身数据的基础上可能会有什么样的体验，科尔曼说，我们刚刚开始看到这个特定冰山的一角。
