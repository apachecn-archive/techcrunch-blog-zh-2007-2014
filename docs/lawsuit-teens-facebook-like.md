# 诉讼称青少年不应该被允许喜欢脸书上的广告

![](img/e929013d9b301aea4a9aaac56145405c.png)

有些人不喜欢广告。但是一项新的[集体诉讼](https://web.archive.org/web/20230213095526/http://www.prnewswire.com/news-releases/facebook-accused-of-misappropriating-likenesses-of-teens-for-profit-101644698.html)在加州起诉脸书(见下文),认为青少年在没有得到父母同意的情况下喜欢任何广告都是非法的。

我们以前写过关于在广告中使用脸书成员肖像的法律含义，但这一次它不是一个笑话。问题是脸书的[“喜欢”按钮](https://web.archive.org/web/20230213095526/https://techcrunch.com/2010/04/21/facebook-like-button/)，而不是用一个人的肖像来宣传产品。在脸书，你可以“喜欢”你信息流中的任何状态更新或帖子，也可以“喜欢”广告。当你这样做的时候，如果广告链接到脸书页面，它会显示为你所有朋友的状态更新，从而将“喜欢”按钮变成一种社交认可。(如果它没有链接到页面，喜欢广告只是脸书用来帮助他们确定广告的质量，它不会出现在你的流中)。

集体诉讼律师声称，在青少年的案件中，脸书是“盗用未成年人的名字和图片牟利”脸书可能会说，这是它的服务条款，这就是网站的工作方式。但这起诉讼的关键在于加州法律的一个漏洞，该法律要求父母同意才能获得未成年人的同意，才能在广告中使用他们的名字或肖像，而脸书不会这样做。像这样的诉讼可能会导致任何 18 岁以下的人必须得到父母的允许才能注册脸书，这可能不是一个坏主意。

一些事情告诉我，这里的律师比青少年更愤怒(因为愤怒可能是由贪婪引发的)。这不是第一起针对脸书的集体诉讼，也不会是最后一次。我已经要求脸书做出回应。

脸书回应道:“我们认为这一诉讼完全没有法律依据，我们将全力抗争。投诉误解了法律，它的意图和脸书的工作方式。例如，原告声称未成年人通过搜索引擎营销脸书，但我们不允许未成年人在搜索引擎中包含他们的个人资料。”[阅读更多](https://web.archive.org/web/20230213095526/https://techcrunch.com/2010/08/27/facebook-exhibit-a-teen-lawsuit/)。
