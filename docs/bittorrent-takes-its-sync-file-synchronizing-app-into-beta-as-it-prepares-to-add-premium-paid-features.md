# BitTorrent 将其同步文件同步应用程序带入测试版，准备添加优质的付费功能 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/17/bittorrent-takes-its-sync-file-synchronizing-app-into-beta-as-it-prepares-to-add-premium-paid-features/>

一点一点地，[努力消除盗版内涵的 P2P 平台](https://web.archive.org/web/20221210032720/https://beta.techcrunch.com/2013/06/25/piracy-what-piracy/)BitTorrent，正在围绕其架构建立一套服务，让用户分享、存储和下载大文件，将自己定位为 Dropbox、box、谷歌等公司的竞争对手。最新的进展是该公司将其文件同步服务[Sync](https://web.archive.org/web/20221210032720/http://labs.bittorrent.com/experiments/sync.html)[从 alpha 升级为 beta](https://web.archive.org/web/20221210032720/http://blog.bittorrent.com/) 。与此同时，它正在通过添加更多功能和一个 Android 应用程序[向更多用户开放平台，iOS 也将很快推出。](https://web.archive.org/web/20221210032720/https://play.google.com/store/apps/details?id=com.bittorrent.sync)

Sync 可以免费使用，就像今天 BitTorrent 的其他服务一样，但一位发言人告诉我，这也为该公司开始推出针对企业和高级用户的付费优质服务奠定了基础。

“我们正在探索潜在的商业模式，其中可能包括针对特定用例的优质功能或应用，”一位发言人说。"例如，对于需要特定功能的企业来说，Sync 具有巨大的潜力."

瞄准企业似乎是正确的方向。这个关于 Sync 的 [BitTorrent 论坛似乎有许多使用案例，其中 IT 人员在远程环境中使用该服务来保持主办公室和卫星之间的数据有序。这与 BitTorrent 捆绑包等服务形成对比，BitTorrent 捆绑包一端面向创意产业(如音乐人)作为分发内容的方式，但最终也面向下载这些捆绑包的消费者。](https://web.archive.org/web/20221210032720/http://forum.bittorrent.com/topic/18320-the-types-of-things-people-are-using-sync-for/)

当你考虑到今天推出的一项新功能时，企业关注同步也是有意义的:一个存档选项，允许用户收集文件的以前版本。这不仅为现在的 Sync 增加了更多的功能，也向用户展示了一种服务增长的方式，即在未来增加更多的高级(付费)服务。

自从今年早些时候从 BitTorrent Labs 中推出 alpha 版以来，Sync 已经被用来同步大约 8pb 的数据了，Sync 是对文件传输服务 [SoShare](https://web.archive.org/web/20221210032720/https://beta.techcrunch.com/2013/02/15/bittorrent-sharpens-enterprise-focus-launches-soshare-to-send-large-files-offering-first-terabyte-free/) 的补充，也在 beta 版中。虽然像 BitTorrent 捆绑包这样的服务更具欺骗性

虽然我们已经看到了初创领域的一些收购——从谷歌这样的公司到突然关闭服务——但 BitTorrent 目前正在利用其持久性作为其独特的卖点之一。一位发言人告诉我，“即使我们关闭了这个程序——例如，最近的 RSS 谷歌阅读器例子——Sync 用户仍然可以使用 Sync。这是因为没有任何数据被存储在 BitTorrent 服务器上，一旦用户下载 BitTorrent 客户端来管理一段数据，他们就可以终生拥有它。

也正是因为这个原因 BitTorrent 服务器缺乏存储——该公司还在兜售其他东西:通过 P2P 架构分发文件的服务的固有安全性；以及随后这些文件和数据永远不受国家安全局等权威机构控制的能力。这是一个及时的专题，给出了围绕 Prism 服务的所有争议，以及大型互联网公司在促进数据收集方面所扮演的角色。

发言人指出:“使用 BitTorrent Sync，您的数据只存储在您拥有的设备上，或者存储在您被授权与您的数据同步的设备上。”。“您的数据没有存储在第三方可以侵入或找到后门访问的云/服务器上；我们不托管任何数据。我们不能允许访问我们没有的数据。”

与存档/版本服务一样，BitTorrent 称移动应用是 Sync 最受欢迎的功能之一。这个应用程序将允许人们使用该应用程序同步远程计算机上的文件夹，以及向其他移动设备发送文件——这是对上传更多数据并在云中生活的一种抗衡。
