# 传言:富士 X10 即将发布，应该会以 599 美元的价格推出复古热卖版

Fujufilm X100 (如图)由于其强大的传感器、合理的价格和令人惊叹的复古风格，正悄然成为摄影爱好者的最爱。但是，即使它的价格完全符合其性能，它也超出了一些有抱负的射击运动员的能力。别担心。富士 X10 将在未来几周内发布，价格应该会低很多。

富士胶片尚未正式发布以下规格，但西班牙网站 [Quesabesde](https://web.archive.org/web/20230203050726/http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=es&tl=en&u=http%3A%2F%2Fwww.quesabesde.com%2Fnoticias%2Ffuji-finepix-x10-rumores%2C1_7877) 独立证实了 X10 之前的传言。较小的相机据说有一个 2/3”传感器，正如 PhotoRumors】所说的[，比微型四分之三相机中的传感器小，但比竞争对手佳能 G12 大。同样像 G12 一样，X10 据说会以 599 美元的建议零售价下降，并配备一个 F/2.0 光圈的 4 倍变焦镜头。](https://web.archive.org/web/20230203050726/http://photorumors.com/2011/08/22/fujifilm-x10-in-the-next-few-weeks/)

X10。或 X50，只要富士包装的性感能让 X100 一炮而红，就有可能撼动 prosumer 领域。
