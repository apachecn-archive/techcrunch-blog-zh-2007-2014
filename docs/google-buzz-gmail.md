# 不，Buzz 没有被从 Gmail 中删除。不过，它可能会有一个单独的网络应用。

嗯，今天几乎每个人都把这个故事搞砸了，所以是时候澄清一下了。不管你在网上读到了什么，谷歌是**而不是**从 Gmail 中移除 Buzz。至少短期内不会。

这种困惑源于谷歌产品营销副总裁[布拉德利·霍洛维茨](https://web.archive.org/web/20230126203048/http://www.crunchbase.com/person/bradley-horowitz)今天早些时候在 TED 对搜索引擎公司的丹尼·沙利文的评论。Horowitz 说谷歌正在考虑将 Buzz 从 Gmail 中分离出来。但他的意思是，他们会考虑让*成为另一个可以在 Gmail 之外使用的*应用，而不是说它会从 Gmail 本身剥离出来，谷歌澄清道。我的意思是，它已经发布了整整三天，使用量也在飙升，真的有人认为谷歌会这么快就让它静音吗？

不，我们不打算从 Gmail 中移除 Buzz。谷歌的一位发言人告诉我们:“我们正在考虑的一些功能包括，在未来的某个时候，在 Gmail 的基础上建立一个独立的 Buzz 体验。

也就是说，有很多人希望完全关闭 Gmail 中的 Buzz。这些用户选择使用它，但现在被它的噪音淹没了。但是你可以在 Gmail 的页脚轻松完成。也不乏有人[担心](https://web.archive.org/web/20230126203048/https://techcrunch.com/2010/02/11/reply-google-buzz-exposing-email/)这项服务的[隐私](https://web.archive.org/web/20230126203048/https://techcrunch.com/2010/02/12/google-buzz-privacy/)后果。谷歌昨晚已经做了一些调整来缓解一些问题——预计很快会有更多的调整。
