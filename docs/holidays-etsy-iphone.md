# 在假期，Etsy 在你的 iPhone 上

时尚购物者们，欢呼吧。你可以在你的 iPhone 上，或者任何一部有合适浏览器的 T2 手机上，为那些独一无二的工艺品做你所有的 [Etsy](https://web.archive.org/web/20230205033913/http://www.etsy.com/) 假日购物。Etsy 刚刚发布了它的 [iPhone 应用](https://web.archive.org/web/20230205033913/http://itunes.apple.com/us/app/etsy/id477128284?mt=8)，以及一个新的移动网站。

你可以浏览各种东西，从艺术钩针，针织，被子到手工制作的童装和珠宝。就像在普通的手工市场上一样。iPhone 应用程序和移动网站都有商品的大照片，以方便视觉浏览。您可以通过标签或常规关键字进行搜索，或者通过类别进行浏览。

该应用程序还具有一些经过深思熟虑的社交分享功能。任何项目都可以通过推特、脸书、电子邮件或短信分享。该应用程序鼓励你使用 Etsy ID 登录，这样你就可以收藏商品并将其添加到购物车中。当然，你可以直接用手机购物。

对于一个移动应用程序来说，搜索实际上非常深入。例如，如果你搜索“帽子”，你会不断看到无穷无尽的新帽子(出于某种原因，大部分是针织帽)。然后就可以按位置过滤搜索(买本地！)、价格、相关性或新近性。

Etsy 卖家也可以使用该应用程序来管理他们的商店，查看正在出售的商品，获得提醒，甚至进行微小的更新。

是的，移动商务将会在这个假日季节大受欢迎。
