# 谷歌创造了一种更好的方式将 IE 转化为 Chrome:一种商业就绪的 Windows 安装程序

![](img/b7022186d10a1f31770c646de1494139.png "g")

2009 年 9 月，我们写了一篇关于谷歌为了渗透商业市场正在做的非常有趣的事情:他们本质上是把 IE 变成 Chrome(T2)。Chrome Frame 是微软浏览器的一个插件，可以在 IE 浏览器中重现 Chrome 浏览体验。谷歌这样做的原因是，他们意识到许多工作中的用户无法安装 Chrome，因为他们的计算机被他们的 IT 部门锁定。今天，谷歌宣布了一个更直接的方法让 Chrome 工作:一个 MSI 安装程序，也就是一个标准的商业 Windows 安装程序。

“今天，我们宣布 Chrome 提供控件，使 IT 管理员能够根据他们的业务需求在 Windows、Mac 和 Linux 上轻松配置和部署浏览器，”谷歌[在 Chrome 博客上写道](https://web.archive.org/web/20230330020923/http://chrome.blogspot.com/2010/12/chrome-is-ready-for-business.html)。该公司表示，这种新的安装程序允许使用标准部署工具的企业为其所有托管用户安装 Chrome。谷歌还概述了一些[政策](https://web.archive.org/web/20230330020923/http://www.google.com/support/a/bin/answer.py?hl=en&answer=187206)，以显示 Chrome 在安全性和可由管理员设置的设置方面将尊重什么。

谷歌还指出，这些政策也适用于 Chrome 操作系统(尽管显然不适用于 MSI 安装程序)。是的，谷歌真的在这里追赶微软。他们不仅希望系统管理员取代 IE，还希望他们取代 Windows。他们显然也想让他们取代自己。

谷歌表示，他们已经与几家大公司就这种新型的 Chrome 安装进行了数月的合作。目前使用它的公司包括 Vanguard、Boise State University、Proctor & Gamble，当然还有 Google。

但谷歌似乎意识到，即使是这个工具也不足以让大多数企业转向 Chrome。他们指出，这只是他们计划为企业提供 Chrome 的“开始”。“*我们正在努力完善下一套政策，让谷歌 Chrome 在未来变得更加可定制，对用户更加有用*，”他们指出。
