# 掌上游戏机将超级马里奥兄弟压缩到 64 像素

嘟嘟嘟嘟嘟嘟嘟嘟。黑客 [Retrobrad](https://web.archive.org/web/20230205014225/http://www.bradsprojects.com/electronics/Bradsprojects-SuperPixelBros.html) 创造了一个手持控制台，以一种非常特殊的方式显示*超级马里奥兄弟*:每个精灵缩小到一个像素。这款名为“超级像素兄弟”的游戏机包括所有 20 个关卡以及 boss 战。

这个游戏是在一个 8×8 的多色 LED 板上进行的，要击中敌人，你需要将你的单像素马里奥放在他们的单像素头上。

他在他的[网站](https://web.archive.org/web/20230205014225/http://www.bradsprojects.com/electronics/Bradsprojects-SuperPixelBros.html)上提供了完整的操作方法和原理图，并且他正在考虑出售实际的 PCB，这样你和你的水管工兄弟 Luigi 就可以以一种绝对低带宽的方式重现你们的冒险。

该套件还允许您使用转换器构建自己的 1 像素版本的 SMB，这是我们许多人在大量空闲时间可能想要模仿的一个技巧。
