# 无线 iPod/iPhone/iPad 同步很棒——可惜可能会被拒绝[视频]

基本上，自从 iPod(以及现在的 iPhone、iPad 等)上市以来，最令人讨厌的事情之一就是你必须通过 USB 将其插入电脑才能同步。当然，这也允许你给你的设备充电，但我很想说，在我床边的支架上充电(因为我也把它用作闹钟)，并通过 WiFi 无线同步它。一个新的应用程序可以让你做到这一点。可惜很有可能被拒绝。

正如你在下面的视频中看到的，WiFi Sync 可以让你做到顾名思义的事情。完成鉴定 iPod/iPhone 和电脑的几个步骤后，您就可以开始无线同步了。它只是工作。当然，苹果可能会遇到的问题是未经授权与他们的软件 iTunes 进行互动。

毫无疑问，如果苹果想在 iPod touch/iphone/ipad 中加入这样的功能，他们早就这么做了。技术似乎不是问题，但可能还有其他因素在起作用，比如安全性和速度。虽然安全性是有争议的，但与 WiFi 相比，速度绝对是使用 USB (2.0)的一大优势，因为一个传输数据的速度接近 500 Mbps，而另一个传输数据的速度只有这个数字的一小部分。(显然，它会因技术(802.11n 对 802.11g 等)和与路由器的距离而异。)

其他需要考虑的事情:当苹果[不可避免地](https://web.archive.org/web/20221207213533/https://beta.techcrunch.com/2009/09/08/the-inevitable-move-of-itunes-to-the-cloud/)将 iTunes 转移到云端时，无线同步他们的设备将非常必要。也许苹果只是在等待，并在那之前享受大 USB 带宽。
