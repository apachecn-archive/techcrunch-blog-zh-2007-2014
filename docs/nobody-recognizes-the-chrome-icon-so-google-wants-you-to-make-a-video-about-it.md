# 没有人认识 Chrome 图标，所以谷歌希望你制作一个关于它的视频

【YouTube http://www.youtube.com/watch?v=bGJOmGL6mYM&hl=en&fs=1&]

快，画出谷歌 Chrome 浏览器的图标。它有一堆原色，但我不看它就能记住这么多。大多数人都没见过，甚至不知道 Chrome 是什么。(是谷歌的浏览器)。谷歌希望通过病毒式营销活动改变这一切。

它要求人们提交展示创建 Chrome 图标的创造性方法的视频。它大概会把最好的一个拉到最后一卷。在这里，你可以[在 7 月 22 日之前提交你的视频](https://web.archive.org/web/20230314082529/http://www.google.com/landing/chrome/ugc/)，看看还有什么被提交了。

但仅仅是让早期的 Chrome 通过在网络上传播视频来帮助营销，就表明了谷歌的整体营销哲学。这不是阿炳式的电视和其他地方的营销闪电战。尽管如此，谷歌不再凌驾于 T4 在电视上投放广告之上。对谷歌来说，浏览器是一个新的市场，它在这个市场上的占有率几乎为零。它将尽一切可能获得这种认同，无论是在电视上还是在网络上。

我想知道哪一组广告会更有效。
