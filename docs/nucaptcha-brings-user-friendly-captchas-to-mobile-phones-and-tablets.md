# NuCaptcha 为手机和平板电脑带来了用户友好的验证码

[NuCaptcha，](https://web.archive.org/web/20230205022524/http://www.nucaptcha.com/)一家开发“人类友好”[视频验证码技术](https://web.archive.org/web/20230205022524/https://techcrunch.com/2010/06/30/nucaptchas-technology-uses-video-to-decipher-between-humans-and-robots/)的初创公司，宣布推出一款专用于移动设备的验证码产品，可用于包括 Android 和 IOS 平台在内的所有移动平台。

作为背景，验证码是你在网站上发现的安全问题，要求你破译和键入单词或数字，并检测用户是人还是垃圾邮件。你看到的大多数验证码是转录的，基于文本的验证码。当然，验证码经常会令人困惑，并且不便于用户使用。NuCaptcha 正试图通过向网站所有者提供一种更加用户友好的替代方式来解决这一痛点，这种方式被称为视频验证码。

NuCaptcha 在过去的几年里优化了它的安全性，这样 3 个未变形的字母可以在“大多数”网站上使用。如果通过 [NuCaptcha 行为分析系统](https://web.archive.org/web/20230205022524/https://techcrunch.com/2011/08/11/nucaptcha-dynamically-alters-captchas-to-promise-security/)检测到威胁，需要更多安全性的网站可以添加多达 8 个字母。

NuCaptcha 的移动版和网页版一样简单。但新的优化移动验证码在多种设备上提供一致的用户体验。例如，当用户点击移动触摸屏来解决验证码时，它将完美地适应键盘留下的空间，以便可以快速轻松地完成验证码。NuCaptcha 不需要 Flash 或者 JavaScript 该公司预计在 2011 年第四季度初发布其验证码技术的 HTML5 移动版。

目前，NuCaptcha 每月提供数百万个验证码，提供免费、专业和企业产品，用户可以通过 NuCaptcha 的[广告计划确定大小、视频选择、安全级别，甚至可选的品牌、广告和创收。](https://web.archive.org/web/20230205022524/https://techcrunch.com/2011/04/14/nucaptchas-advertising-network-for-video-captchas-goes-self-serve/)

NuCaptcha 面临 ReCaptcha 的竞争，后者于 2009 年被谷歌收购。
