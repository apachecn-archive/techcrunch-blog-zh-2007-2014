# 更新的 Gamepark GP2X 将于今年秋季发布

![gp2x](img/eda93644b4e368b0b495f4a44ecd5004.png)

开源便携式游戏设备的 [GP2X 系列](https://web.archive.org/web/20221001062729/http://www.gp2x.com/)将于今年秋季推出新的 GP2X F-200。值得注意的更新包括触摸屏和重新设计的 D-pad。这些基于 Linux 的便携式电脑的主要吸引力在于内置的[模拟器支持](https://web.archive.org/web/20221001062729/http://www.gp32x.de/cgi-bin/cfiles.cgi?0,0,0,0,5)，允许你在包括 NES、MAME、Game Boy、世嘉创世纪、SNES 和雅达利 Lynx 在内的系统上玩游戏。

GP2X 还可以播放电影，内置电子书阅读器和照片浏览器。它将很快在 Play-Asia.com 接受预订。还没有价格，但 GP2X F-100 是 179 美元，所以我们预计 F-200 可能在 200 美元左右。
