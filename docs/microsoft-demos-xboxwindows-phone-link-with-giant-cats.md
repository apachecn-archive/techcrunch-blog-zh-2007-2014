# 微软演示 Xbox/Windows Phone 与大猫的链接

微软从第一天起就一直在吹捧 Windows Phone 7 的 Xbox Live 功能，但总觉得有点欠缺。你可以看着你可爱的小头像玩游戏来提高你的玩家分数，但是我们在像这个这样的演示中看到的 Xbox 体验的连接性在哪里？

似乎雷德蒙的人们还没有完全实现，但是周一早上在西雅图他们的游戏节上的演示告诉我们这个想法还没有半途而废。

微软和 Kinectimals 团队展示了一种将一个毛茸茸的朋友从 Kinectimals Xbox 游戏中移动到 Windows Phone 上的新方法。越来越喜欢 Xbox 上宠物的玩家可以让游戏生成一个二维码，Kinectimals 移动应用程序可以扫描这个二维码。在打电话的时候，玩家可以用相机拍照，并有策略地把他们的宠物放在里面玩耍。

当玩家厌倦了随身携带他们的小数码老虎时，该应用程序会生成一个类似的二维码，由 Kinect 摄像头扫描，宠物会被重新导入到游戏中。 [Geekwire](https://web.archive.org/web/20230205032607/http://www.geekwire.com/2011/video-xbox-windows-phone-games-linked-kinect-qr-codes) 有完整的视频演示，对于那些难以想象的人来说:

【YouTube http://www.youtube.com/watch?v=fZuqi8IzTYY&w=640&h=345]

Kinectimals 连接是微软计划的一部分，旨在加强其广受欢迎的 Xbox Live 平台和 Windows Phone 7 之间的联系。坦率地说，这是时候了:微软利用 Xbox Live 品牌为他们的移动操作系统赢得一些玩家信誉是一回事，但如果这种体验没有实现，那就几乎没有用了。正如 [WPcentral](https://web.archive.org/web/20230205032607/http://www.wpcentral.com/kinectimals-feature-kinect-kitty-i-mean-connectivity?utm_source=wpc&utm_medium=twitter?utm_source=wpc&utm_medium=twitter) 所指出的，在 Kinectimals 之前，唯一一款与家庭版整合的游戏是 Full House Poker，它允许玩家在游戏过程之间转移他们的奖金。

在今天的演示和他们已经提出的概念之间，微软正在考虑是否有可能拥有移动游戏领域的很大一部分，如果他们能让 A-game 上线的话。
