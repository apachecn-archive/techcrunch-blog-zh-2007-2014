# 达斯手表:酷手表，诉讼等待发生

除非伦敦的风暴官方得到许可称他们的手表为达斯，否则我会有点担心。无论如何，这是一个非常酷的作品，有三个以 coo 为中心的圆圈显示时间。它有黑色、蓝色、铬色和白色，目前售价为 170.00 美元。
