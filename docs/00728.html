微软收购企业群聊公司Parlano

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/08/30/microsoft-acquires-enterprise-group-chat-company-parlano/trackback/

邓肯·赖利

微软宣布已经同意收购总部位于芝加哥的 Parlano ，这是企业群组聊天平台 MindAlign 的制造商。

微软表示，将利用Parlano的群聊功能来改进微软Office Communications Server和微软Office Communicator。

Parlano的平台提供群聊技术，“使人们能够进行特定主题的多方即时通讯讨论，并持续一段时间。”Parlano的现有客户包括金融服务公司和呼叫中心。

Microsoft计划将群聊作为Office Communications Server 2007标准客户端访问许可证的一部分提供。

收购的条款没有公布。该交易预计将于2007年第四季度完成。
