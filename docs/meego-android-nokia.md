# MeeGo 副总裁辞职，诺基亚首席执行官接听 Eric Schmidt 的电话

![](img/17bc1a907067add085867b4a4324a229.png "Screen shot 2010-10-05 at 11.14.39 AM")

今早有消息称，诺基亚负责 MeeGo 设备的高管阿里·雅克西于上周辞职。这家全球最大的移动电话公司试图在快速发展的移动领域建立自己的身份，这是一系列高调人士离开该公司的延续。内部动荡和最近聘请前微软高管[史蒂芬·埃洛普](https://web.archive.org/web/20221007071812/http://www.crunchbase.com/person/stephen-elop)担任诺基亚的新首席执行官，引发了不少猜测，诺基亚可能会放弃自己的操作系统，转而采用新的 Windows Phone 操作系统——或者至少将其产品分支为[,在即将推出的 MeeGo 之上选择这一操作系统](https://web.archive.org/web/20221007071812/http://venturebeat.com/2010/09/23/nokia-windows-stephen-elop/)。但是也不要排除谷歌的 Android 操作系统。

我们从可靠消息来源获悉，谷歌首席执行官 Eric Schmidt 已经致电埃洛普，讨论 Android 在诺基亚手机上运行的可能性。我们实际上在一周前就听到了这个消息，但是今天的新闻让它变得更有趣。雅克西辞职的时候，埃洛普和施密特在谈话。

此外，当诺基亚手机运行 Windows Phone 7 的想法被一个了解诺基亚内部战略的内部人士提起时，那个人笑了——“歇斯底里地”据知情人士透露，如果诺基亚选择 MeeGo 以外的其他方向，Android 比 Windows Phone 7 更有可能——即使埃洛普也在其中。也就是说，这位人士认为，如果 MeeGo 在未来几个月(或几个月)还没有准备好，Android 可能是一个临时措施，用来炫耀诺基亚的伟大硬件。N9 等设备的早期迹象似乎恰恰表明了这一点。

根据这个消息来源，MeeGo 仍然是未来。或者至少*是*——记住，这个消息是在得知雅克西辞职之前。但是考虑到诺基亚上个月刚刚请来了[彼得·斯基尔曼](https://web.archive.org/web/20221007071812/http://www.crunchbase.com/person/peter-skillman)来负责 MeeGo 的用户体验(斯基尔曼设计了 Palm Pre，而[我们在 8 月份透露了他离开惠普/Palm](https://web.archive.org/web/20221007071812/https://beta.techcrunch.com/2010/08/09/peter-skillman-palm/) 的消息)，看起来该公司至少会给它更多的时间来看看会发生什么。

与此同时，诺基亚会采用另一种操作系统作为临时解决方案吗？几周前，该公司表示“[不](https://web.archive.org/web/20221007071812/http://www.bloomberg.com/news/2010-09-24/nokia-says-not-planning-to-add-more-software-platforms.html?cmpid=yhoo)”，但目前情况似乎很不稳定。诺基亚没有太多时间在这里行动。他们一天不超越 Symbian，就一天落后于苹果、谷歌和 RIM。很快，这份名单将再次包括微软。

目前，使用 Android 的唯一真正不利之处，即使只是暂时的，是微软的潜在诉讼(这是[起诉这些天试图使用 Android 而没有首先许可其专利的所有人](https://web.archive.org/web/20221007071812/https://beta.techcrunch.com/2010/10/01/android-lawsuits/))。埃洛普可能认识一些能帮上忙的人。
