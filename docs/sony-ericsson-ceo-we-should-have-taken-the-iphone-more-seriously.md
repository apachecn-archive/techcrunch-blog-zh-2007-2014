# 索尼爱立信 CEO:我们应该更认真地对待 iPhone

在接受《华尔街日报》、[采访时，索尼爱立信](https://web.archive.org/web/20230204190145/http://www.crunchbase.com/company/sony-ericsson)总裁兼首席执行官伯特·诺德伯格就 iPhone、Android、Windows Phone 和摩托罗拉发表了许多有趣的言论。

在为选择 Android 作为其手机操作系统的决定辩护时，诺德伯格认为，考虑到其快速增长，这是“他们可能做出的最佳选择”，但也承认该公司“在 2007 年推出 iPhone 时应该更认真地对待它”。

当被问及他是否愿意购买摩托罗拉移动[而不是谷歌](https://web.archive.org/web/20230204190145/https://techcrunch.com/2011/08/15/breaking-google-buys-motorola-for-12-5-billion/)时，他坦率地回答道:

> “当然，但是在你去购物之前，你必须变得富有。
> 
> 我们之间的交易会非常复杂，因为我们是一家拥有两大股东的私人公司，而他们是一家美国上市公司。"

诺德伯格表示，索尼爱立信的目标是逐步剥离其功能手机业务，成为一家 100%的智能手机公司——我应该指出，这与诺基亚非常不同。他估计这将在明年年中实现(如今其 70%的销售额已经来自智能手机)。

当被问及索尼爱立信为什么还没有成为世界上最大的 Android 设备制造商时，Nordberg 告诉 WSJ，他们低估了它渗透美国的速度，他承认该公司在美国是一个“非常小的参与者”。

据估计，索尼爱立信在整个安卓市场占有 11%的份额。

当被问及该公司是否会考虑转向微软的 Windows Phone 平台时，诺德伯格说(强调我们的观点):

> “在这一点上，我不会放心投资一个不如我们目前使用的平台好的平台**。因此，我们仍然使用 Android，但我对 Windows Phone 相当好奇。”**
