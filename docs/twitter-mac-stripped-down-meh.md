# 面向 Mac 的 Twitter:剥离下来，咩

![](img/716ec6d7e792e4469a9471f767e05496.png)

苹果应用商店[今天早上刚刚推出了](https://web.archive.org/web/20230202225747/https://techcrunch.com/2011/01/06/apple%E2%80%99s-mac-app-store-opens-with-more-than-1000-apps/)(这是一个独立于 iTunes 的应用商店，它提供了苹果 OS X 的最新更新)，而排名第一的免费应用已经是[苹果推特](https://web.archive.org/web/20230202225747/http://itunes.apple.com/us/app/twitter/id409789998?mt=12)。我刚刚安装了它，并开始使用它。我的第一印象是 meh。

Mac 版 Twitter 是优秀的 iPad 版 Twitter 的精简版。想象一下，如果你在 Twitter for iPad 应用程序中只使用左边的流列，并将其作为一个小部件在你的桌面上启动。我觉得这是一种半心半意的尝试。我给它这个:它非常快，而且不碍事。但这是不和谐的，因为它的运作方式不同于 iPad 版 Twitter，甚至 Twitter.com 的运作方式。当你点击一条带有链接的推文时，它不会像 iPad 应用程序那样展开成第二个窗格，页面在下面打开。相反，它会在你的浏览器中打开一个新标签。这是标签超载的一个原因。此外，当一条推文回复另一条推文时，唯一的指示是当你悬停在该推文上时出现的过于微妙的对话气球。找到对话视图并不完全明显。你必须双击推文或者点击对话气泡。按右箭头键也有效。

其他投诉:没有推文箱。为了发推，你必须进入菜单，选择新推或直接消息。还有一个键盘命令。至少在 iPAd 应用程序的左下角有一个新的信息图标，在 Twitter.com 有一个漂亮的大“发生什么事了？”盒子。发微博应该是一个一步到位的过程，而不是两步走的过程。另一个烦恼是，你不能点击一个人的名字来查看他们的个人资料或推文流。我花了一段时间才明白，你必须双击他们的头像图片才能显示他们的视频流。同样，如果你习惯了 Twitter 在其他地方的工作方式，像这样的小矛盾实际上会让你慢下来。

应用程序的其他部分都很好。您可以查看您的信息流、提及、直接消息、列表、个人资料和搜索。你可以回复个人推文，收藏它们或者转发它们。大概就是这样。也许这就是你所需要的。但是当你拥有台式电脑的全部功能时，在功能上吝啬就没什么意义了。也许我已经被 Twitter 在网络和 iPad 上更丰富的功能宠坏了。我想我现在还是坚持用那些。
