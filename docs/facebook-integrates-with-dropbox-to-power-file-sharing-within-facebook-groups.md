# 脸书与 Dropbox 集成，支持脸书团队内部的文件共享

[脸书](https://web.archive.org/web/20230326025014/http://www.facebook.com/)今天宣布与个人文件存储公司 [Dropbox](https://web.archive.org/web/20230326025014/http://www.dropbox.com/) 进行新的整合。现在，人们可以使用 Dropbox 基于云的存储系统向脸书团体共享文件。

Dropbox 的 Chris Varenhorst 在[的一篇博客文章](https://web.archive.org/web/20230326025014/https://blog.dropbox.com/index.php/share-stuff-from-dropbox-in-your-facebook-groups/)中写道:

> “今天，我们很高兴地宣布，您可以在脸书群组内部分享 Dropbox 中的内容！现在，无论你在哪里，你都可以与你的学习小组共享笔记，将最新的比赛日程添加到你的篮球队小组，或者以闪电般的速度将生日视频发布到你家人的小组——只要你将它们保存在 Dropbox 中。
> 
> 就像脸书群的其他帖子一样，人们可以喜欢或评论你在群内分享的任何内容。如果您对已共享的文件进行了任何编辑，群将会自动获得更新！"

脸书告诉我们，这并不能取代目前从你的电脑上传文件到群组的功能。这只是让人们能够上传他们已经存储在云中的文件到群组——只要这些文件存储在 Dropbox 中。

这是一个非常直接的消息，但从战略上来说，这在几个层面上都是一个有趣的举动。几个月前，脸书推出了自己的群组文件共享功能，当时我的同事(同时也是 TechCrunch 驻脸书的专家)Josh Constine [推测](https://web.archive.org/web/20230326025014/https://techcrunch.com/2012/05/10/facebook-file-sharing-to-all-groups/)假以时日，它“可能足以限制 Dropbox 等云存储/文件共享服务的长期增长潜力。”现在看来，脸书最终决定不再独自应对为整个云文件共享平台提供动力这样的巨大挑战。

另外值得一提的是，Dropbox 目前的工程副总裁是[阿迪蒂亚·阿加瓦尔](https://web.archive.org/web/20230326025014/http://www.crunchbase.com/person/aditya-agarwal)，这位技术高管[曾在](https://web.archive.org/web/20230326025014/https://techcrunch.com/2012/02/27/dropbox-buys-cove-to-bring-former-facebookers-ruchi-sanghvi-and-aditya-agarwal-to-the-team/)担任脸书的工程总监。他可能在促成这笔交易以及领导 Dropbox 和脸书的技术整合方面发挥了重要作用。

对脸书来说，完全专注于成为网络社交基础设施的核心使命是明智之举。这也是 Dropbox 的一大妙招，它现在可以在脸书庞大的用户群中获得有利可图的曝光。据我所知，目前脸书还没有整合到其他云存储服务的计划——所以这是脸书非常信任 Dropbox 技术的一个有力证明。
