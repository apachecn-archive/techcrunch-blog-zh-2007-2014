# 现在我知道了:企业家在新的视频系列中讨论成功、失败和经验教训

创业公司和企业家远不缺乏资源来帮助他们度过创业过程中的痛苦，但总是有回报的。例如，他们可以查看[创业基因组计划，看看哪些具体的组成部分是建立成功企业不可或缺的](https://web.archive.org/web/20230205015526/https://techcrunch.com/2011/08/29/what-kills-startups-blackbox-releases-reportapp-to-help-founders-avoid-the-deadpool/)，查看像这样的酷[工具，将他们的条款清单与行业标准进行比较](https://web.archive.org/web/20230205015526/https://techcrunch.com/2011/09/26/free-startup-tools-how-does-your-term-sheet-compare-to-the-standards/)，甚至是创始人研究所[帮助创始人规范他们与顾问的关系的努力](https://web.archive.org/web/20230205015526/https://techcrunch.com/2011/09/27/free-startup-tools-an-easy-way-for-founders-to-set-the-terms-of-their-collaboration/)。显然，这只是许多例子中的一小部分——说到与资金的联系，还有孵化器、天使名单、Kickstarter——还有很多。

此外，还有通过视频系列进行的创业教育:早在 8 月，多城市创业孵化器 tech stars[推出了一部“现实纪录片”电视剧(与彭博合作)](https://web.archive.org/web/20230205015526/https://techcrunch.com/2011/08/02/startups-on-tv-techstars-teams-up-with-bloomberg-to-offer-a-look-inside-the-startup-hustle/)，旨在让世界一窥在科技孵化器的快节奏氛围中创业是什么样的。

为科技公司制作商业视频的初创公司 Sprinkle Lab 改编了像这样的视频系列，上周推出了一个系列，旨在与任何对创业世界感兴趣的人交谈。这部名为[“现在我知道了”](https://web.archive.org/web/20230205015526/http://nowiknow.sprinklelab.com/)的剧集将在接下来的六周内，每周一和周三发布新一集，其中包含科技领域不同知名人物的经验教训、难忘经历或个人哲学。这有点让人想起《创业小子》( Startup Kids )( T12 ),但它没有聚焦于年轻的企业家，而是分享了当今科技行业面临的重大问题的思考，由各个年龄的专家提出。

该系列首播了六集，其中包括《精益创业》的作者埃里克·里斯、Craigslist 创始人[克雷格·纽马克](https://web.archive.org/web/20230205015526/http://www.crunchbase.com/person/craig-newmark)、连续创业者、讲师和作家[史蒂夫·布兰克](https://web.archive.org/web/20230205015526/http://www.crunchbase.com/person/steve-blank)、摇滚健康创始人[哈雷·泰科](https://web.archive.org/web/20230205015526/http://www.crunchbase.com/person/halle-tecco)、Causes.com 联合创始人兼产品副总裁[克里斯·陈](https://web.archive.org/web/20230205015526/http://www.crunchbase.com/company/causes)以及 TechCrunch 专栏作家和新成立的 CrunchFund 风投 [MG Siegler](https://web.archive.org/web/20230205015526/http://www.crunchbase.com/person/mg-siegler) 。

例如，现在我知道的最新视频中，前 TechCruncher 和现任 Accel 投资大师 [Dan Levine](https://web.archive.org/web/20230205015526/http://www.crunchbase.com/person/daniel-levine) 谈论失败的概念，以及它如何可能实际上*而不是*是最好的学习方式。

Sprinkle Lab 的联合创始人 Cameron Woodward 告诉我们，他和他的团队在今年 3 月成立了 Sprinkle Lab，因为他们认为科技领域没有足够多的好视频。虽然有 Revision3 和其他类似的优秀视频专家，但伍德沃德说，他希望自己的初创公司专注于制作电视节目，而不仅仅是 vlog 级别的节目。

“我们是千禧一代，所以我们没有那种耐心”，他说。“我们想看一些短小精悍、充满书呆子气的东西”，所以我们成立了一家视频公司，使用 [MediaCore](https://web.archive.org/web/20230205015526/http://mediacore.com/) (一家专注于[让中小企业建立自己的视频平台的初创公司，我们在 8 月份](https://web.archive.org/web/20230205015526/https://techcrunch.com/2011/08/10/mediacore-raises-seed-funding-launches-platform-to-let-smbs-create-their-own-youtube/)对其进行了简介)——Sprinkle Lab 现在推出，我知道这是基于他们希望看到这种教育内容找到归宿的愿望。
