认识来自I/o Ventures TechCrunch的最新创业公司

原文：https://web.archive.org/web/http://techcrunch.com/2011/06/28/meet-the-newest-startup-brood-from-io-ventures/

I/o Ventures演示日不到半小时就要开始了，我将前往瓦伦西亚街观看在场的初创公司。i/o Ventures是一家基于旧金山的孵化器，在其7000平方英尺的阁楼/咖啡店空间中为五六家初创公司提供为期4-6个月的位置，以及25000美元的种子资金，以换取公司约8%的股份。

我们之前已经介绍过一些公司，我在这里发布一个初步的列表(在我观看他们的演示时会更新)。

CodeEval ( 我们的报道)——一个面试和测试程序员的平台，CodeEval让公司通过给他们一个编码挑战来自动化筛选求职者的过程。CodeEval的联合创始人吉米·约翰(Jimmy John)估计，仅在美国就有1350万开发者，全球市场价值106亿美元。“我们想让这成为事实上的全面招聘标准，”约翰说。

Pieceable刚刚发布了Pieceable Viewer，这是一个试图通过为客户提供iOS应用程序的现成组件和一个交互式应用程序Q的平台来简化移动开发的平台。

预付 —一项移动忠诚度计划，为客户提供预付商店积分，允许他们在购买时节省高达10-20%的费用。例如，商家可以以120美元的价格向常客出售100美元的午餐积分，或者回购不需要的商店积分。

Cortex ( 我们的报道 ) —一个共享平台和浏览器扩展，允许您通过其菜单快速共享视频、图像和链接等网络内容。Cortex是Chrome网络商店中增长最快的应用程序，目前每秒钟有1次分享，每天约有87K次分享。

《口袋妖怪遇上电子鸡》(Pokemon meets Tamagotchi)，这是一款手机游戏，你可以通过地理定位的附加优势来饲养和对抗被称为Mobbles的古怪生物。“我们没有发明一个新概念，我们只是把已经成功的内容混合在一起，”联合创始人亚历山大·柯特林说，他提到口袋妖怪已经带来了300亿美元的收入。

i/o Ventures的联合创始人保罗·布拉吉尔(Paul Bragiel)告诉我，今年春季课程的首要主题是消费者互联网游戏和移动，孵化器更喜欢至少有一名技术联合创始人的初创公司。

当被问及班级规模相对较小时，他说:“我们对成为工厂不感兴趣，我们有超级选择性。”i/o从今年春季团体的500名申请者中挑选了5名。当被问及成为一家i/o风险投资公司需要什么时，他回答说，“在一天结束时，我们会问自己‘我们年轻时会和这些人一起创办公司吗？’“拉德。

在最后一批i/o创业公司的六家公司中，两家被收购，其余的都筹集了50万到100万美元的资金来实现他们的想法。聚会上见。
