# Ovi 敲门——

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/29/ovi-knocking-at-the-door/>

今天，诺基亚宣布了其新的互联网服务品牌 Ovi。(Ovi 是芬兰语门的意思)。诺基亚新的互联网服务将包括诺基亚音乐商店、N-Gage 和诺基亚地图。音乐和地图可以从诺基亚音乐商店和诺基亚地图下载。正如昨天 mobile runch([franken phone](https://web.archive.org/web/20201204184205/http://mobilecrunch.com/index.php?s=Frankenphone)的回归)报道的那样，N-Gage 是一款手机游戏服务。这三种服务都可以在旅途中使用，也可以下载到电脑上。将文件下载到 PC 上，然后安装到移动设备上，可以快速传输较大的文件。

“我们知道，人们希望能够以集成的方式访问精彩的内容，而不必坐在电脑或电视机前。借助诺基亚音乐商店和 N-Gage 服务，您可以直接从您的诺基亚 N 系列多媒体电脑上获得最佳音乐和游戏，”诺基亚多媒体部执行副总裁兼总经理 Anssi Vanjoki 说道。“我们预计移动设备将成为最受欢迎的平台——第四屏——来享受您的数字内容。Ovi 将我们的互联网服务愿景带入生活，人们可以随时随地创造、分享和消费事实、感受和情感。”

随着移动设备计算能力的提高，消费者必然会需要 PC 类型的应用程序。提供信息和娱乐的互联网服务对移动设备和固定设备一样有利可图。我希望有一天我能拥有一台移动设备来满足我所有的计算需求。在旅途中，我可以看到它小巧而清晰的屏幕。当我回家时，它可以和我的 60 英寸显示器连接，用来看电影和玩游戏。
