# 马克·库班向社交游戏制造商提提移动投资 25 万美元

**独家-**社交游戏初创公司[提及移动](https://web.archive.org/web/20230203115355/http://www.mentionmobile.com/)已经[从亿万富翁、企业家和达拉斯小牛队老板](https://web.archive.org/web/20230203115355/http://www.crunchbase.com/company/mention-mobile)[马克·库班](https://web.archive.org/web/20230203115355/http://www.crunchbase.com/person/mark-cuban)手中筹集了 25 万美元，以换取该公司的少数股权。

提及移动说，它专门从事“注入脸书内容”的应用程序，资本注入是为了资助新的单一标题应用程序的开发。

该公司的移动应用程序很有趣，因为它们利用来自脸书的公共信息来定制游戏，并根据用户的朋友、偏好、兴趣等创建个性化内容。

提到移动的应用程序目前只适用于 iOS，但该公司表示，即将推出的应用程序也将适用于 Android。

提及移动成立于 2010 年，总部位于洛杉矶。
