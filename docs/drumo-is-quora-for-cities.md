# Drumo 是城市的 Quora

当你刚到一个城市，没有朋友，这很难。当我访问一个新城市时，我倾向于躲在树里和垃圾桶后面，从远处观察人们，看他们去哪里吃饭和社交。然后我会在晚上去那些地方，当它们关门的时候，嗅嗅空气中的气味。然而， [Drumo](https://web.archive.org/web/20230204095000/http://www.drumo.com/index.htm) 提供了一种改进的方法，使用 [Quora](https://web.archive.org/web/20230204095000/https://techcrunch.com/tag/Quora) 风格的讨论，结合主要大都市地区的地理数据，找出关于一个城市的具体事情。

目前，这项服务仅限于悉尼、新加坡和东京，因此限制了你的选择。然而，它仍然处于测试阶段，看起来是一种在地面上解决问题的有趣方式。例如，你可以询问寿司店和最好的手表店，并从真人那里得到真实的答案。回答正确会获得积分和徽章，还会与当地企业合作。
![](img/5b0a9730bb311710af9d741b697742f9.png)

问题范围从[平凡的](https://web.archive.org/web/20230204095000/http://singapore.drumo.com/question/where-can-i-find-some-really-nice-koi-fish-for-my-pond.html)到[相当紧急的](https://web.archive.org/web/20230204095000/http://singapore.drumo.com/question/where-can-i-find-some-really-nice-koi-fish-for-my-pond.html#search:hospital)，现在似乎很少被开发人员的朋友提及。然而，设计是异想天开和可爱的，服务，从表面上看，似乎是有用的。

澳大利亚皇家设计公司建造并资助了这个应用程序，目前没有公共投资。他们正在缓慢但稳步地发展这项服务，在接下来的几个月里，他们应该会增加新的城市，包括大阪和香港。现在有一个 iPhone 应用程序可用，一个 Android 应用程序即将推出。

现在，在数百名自愿者的帮助下，我可以满足自己邪恶的欲望，而且，鉴于 Quora 式的界面，我现在可以找到有户外座位的位置来嗅嗅。我的朋友们，这就是互联网的诞生。
