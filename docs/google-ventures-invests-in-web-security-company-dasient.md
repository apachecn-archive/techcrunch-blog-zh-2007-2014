# 谷歌风投投资网络安全公司 Dasient

为网站和广告网络开发反恶意软件解决方案的公司 Dasient 已经从谷歌风险投资公司获得[额外资金。融资金额没有披露，但该公司此前已经从 Benhamou Global Ventures、Floodgate、Radar Partners 和其他公司筹集了 400 万美元。](https://web.archive.org/web/20230221234640/http://www.prnewswire.com/news-releases/dasient-raises-new-round-of-funding-led-by-google-ventures-115477669.html)

Dasient 由前谷歌员工尼尔·达斯瓦尼(Neil Daswani)和沙里克·里兹维(Shariq Rizvi)以及阿梅特·拉纳迪夫(Ameet Ranadive，前麦肯锡公司)创建，旨在保护网站免受基于网络的恶意软件攻击。该公司的专有技术监控网站的恶意软件感染，如果发生攻击，将提醒网站管理员，并隔离感染。在该公司上一季度，Dasient 报告了超过 120 万个网站上的恶意软件。

新一轮的资金将用于扩大 R&D 的努力和销售和营销活动。尽管面临来自 McAfee 等巨头的竞争，该公司似乎仍在增长。2010 年，Dasient 表示，其产品在金融、媒体和电子商务垂直领域的销售额创下纪录。Dasient 还实现了三位数的客户增长，并将团队规模扩大了一倍。
