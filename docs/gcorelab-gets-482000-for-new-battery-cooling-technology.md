# Gcorelab 因新的电池冷却技术获得 482，000 美元 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/23/gcorelab-gets-482000-for-new-battery-cooling-technology/>

一家名为 Gcorelab 的新加坡清洁技术公司刚刚获得了 48.2 万美元(58.9 万新元)的资金，用于一项新技术，该技术有望控制困扰锂离子电池的热量问题。

使用锂离子电池的电池组，如笔记本电脑中的电池，以及电动汽车等大型部署中的电池，可能会出现电池故障。当细胞过热时，它们会引起与邻近细胞的连锁反应，这一过程被称为热跑道传播，这可能导致爆炸。

Gcorelab 已经为其冷却技术申请了专利。它依靠冷却板，该团队声称，与液体冷却相比，其“斜鳍技术”可以实现 50%至 80%的更好结果，同时使用相同的能量。

Gcorelab 的联合创始人 Ray Kung 说，电池温度是通过将热量从组件中转移出去来管理的，传感器有助于调节冷却过程。他补充说，它在寒冷的气候下也可以工作，加热器和冷却器可以保持最佳状态。(电池是很挑剔的东西；北极的温度也会让它们爆炸。)

“目前，电池热管理正在从空气冷却系统过渡到液体冷却系统。虽然价格合理且易于实施，但与液冷相比，空气冷却在传热性能方面要差得多，”Kung 说。

因此，当行业转向液体冷却方法时，Gcorelab 希望其技术能够为后者提供一种更实惠的替代方案。

如果成功，该公司的技术可能会给电池行业和 reliant 行业带来革命，因为需要冷却来防止过热和爆炸。

这些年来，大量锂离子电池爆炸，从[iphone](https://web.archive.org/web/20221005121225/https://beta.techcrunch.com/2011/02/16/this-iphone-4-exploded/)到[电动牙刷](https://web.archive.org/web/20221005121225/http://www.reddit.com/r/mildlyinteresting/comments/1breg1/the_battery_inside_my_electric_toothbrush_exploded/)。糟糕到当你刷牙的时候有东西突然着火，但是你可以看到为什么航空业和汽车业对这方面的安全标准非常焦虑。

波音公司的全新梦想飞机 787 最近遭遇电池过热故障，监管机构下令所有 50 架飞机停飞——这是 34 年来美国机队首次停飞。

据报道，停飞使波音公司损失了约 6 亿美元，这还不包括其他费用，如对业务中断的航空公司的赔偿。

特斯拉首席执行官埃隆·马斯克对波音公司的电池困境发表了评论。马斯克对波音设计错误的诊断是，787 电池的大电池相互靠近。他[说](https://web.archive.org/web/20221005121225/http://blogs.wsj.com/corporate-intelligence/2013/02/26/elon-musks-solution-to-boeings-battery-problem/)为了最小化热跑道的级联效应，特斯拉汽车使用更小的电池单元，它们之间有间隙，并试图在它们之间使用更好的隔热材料。

Gcorelab 正在与中国的一家公共汽车制造商合作，它预计中国未来将成为全球最大的电动汽车市场。中国设定了到 2020 年达到 500 万辆电动汽车的目标，以减少对石油的依赖。

Gcorelab 估计，到 2020 年，这个电动汽车电子冷却市场将达到每年 57 亿美元。它还强调了可以解决的二级市场，如风力涡轮机和航空航天用途的冷却，同年价值约 2000 万美元。

这家初创公司由李宝成和龚在 2011 年初创立。Lee 拥有机械工程博士学位，Kung 拥有工商管理硕士学位，是 Gcorelab 的业务开发人员。

这轮种子资金来自新加坡政府指定的早期孵化器之一红点创投(Red Dot Ventures)。政府的国家研究基金会(NRF)部门以 85-15 的比例匹配其孵化器的投资，提供更大比例的资金，以提高该国初创公司的种子资金数额。
