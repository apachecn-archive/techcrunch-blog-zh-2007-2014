# 出于对睡眠的热爱，我将参加为期一周的杰欧挑战

![](img/ba0ba112d1178457534be633ecd867f3.png "IMG_0091")

今晚，我将开始通往更好睡眠的道路——至少杰欧是这么说的。杰欧是一个私人睡眠教练(闹钟)，观察你如何睡觉，并帮助“指导”你更好的睡眠模式。杰欧最近做得非常好，甚至在一月份又筹集了 1230 万美元。你可能听说过[类似的装置](https://web.archive.org/web/20221207091356/https://beta.techcrunch.com/2009/11/24/wakemate-sleep-aid/)，尽管[有些没有那么好的运气。](https://web.archive.org/web/20221207091356/https://beta.techcrunch.com/2010/12/31/wakemate-usb-fire/)

![](img/f90d4b9049ad884a620ff0a045053275.png "IMG_0105")

基本上，杰欧会等着叫醒你，直到你离深度睡眠最远，让你醒来时感觉更精神。在消费电子展上，我们采访了杰欧的联合创始人本·鲁宾，他解释了一下杰欧是如何运作的。杰欧与其他睡眠监测器的不同之处在于，它提供了所有的工具来创建一个完整的睡眠指导生态系统。有一个网络应用程序可以添加剥夺你适当休息能力的日常习惯的细节(下午 3 点后喝咖啡，睡前 2 小时使用电脑，等等)。).你所要做的就是戴上杰欧头带睡觉，数据会被发送到接收器/时钟，并存储在 SD 卡上。卡上的数据可以传输到个人电脑上。

本周，我将每天晚上使用杰欧，并用图表记录我的分数来报告它的有效性。我会在这个星期六把它给你。我通常醒来时都很虚弱，所以我希望一切都好。
