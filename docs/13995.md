# 谷歌为 Gmail TechCrunch 带来了 100 多种虚拟键盘、音译和即时消息

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/09/google-brings-more-than-100-virtual-keyboards-transliterations-and-imes-to-gmail/amp/>

谷歌刚刚宣布了对 Gmail 的更新，这将使我们这些经常用不同语言写邮件的人的生活变得更加轻松。Gmail 用户现在可以在 100 多个虚拟键盘和音译之间轻松切换。此外，你现在可以使用谷歌的输入法编辑器( [IME](https://web.archive.org/web/20230123111506/http://en.wikipedia.org/wiki/Input_method_editor) )将你的按键转换成亚洲字符。谷歌表示，这意味着你不会“发现自己受到键盘语言的限制”

由于虚拟键盘支持阿拉伯语、希伯来语、泰语和韩语等语言，Gmail 现在支持 75 种语言的输入。输入法编辑器目前支持简体中文和拼音的[拼音](https://web.archive.org/web/20230123111506/http://en.wikipedia.org/wiki/Pinyin)和[吴彼](https://web.archive.org/web/20230123111506/http://en.wikipedia.org/wiki/Wubi_method) IME 以及繁体中文和日文的[朱茵/注音字母](https://web.archive.org/web/20230123111506/http://en.wikipedia.org/wiki/Bopomofo) IME。

要在 Gmail 中打开这些新的输入法，只需前往你的 [Gmail 设置](https://web.archive.org/web/20230123111506/https://mail.google.com/mail/u/0/#settings/general)，寻找“启用输入工具”选项，然后按照这里[的指示](https://web.archive.org/web/20230123111506/http://support.google.com/mail/bin/answer.py?hl=en&answer=139576)进行操作。看起来谷歌正在慢慢推出这项新功能，所以如果你还没有在你的 Gmail 账户中看到它，请今天晚些时候再来查看。

此前，谷歌已经通过其[谷歌输入工具 Chrome 扩展](https://web.archive.org/web/20230123111506/https://chrome.google.com/webstore/detail/google-input-tools-by-goo/mclkkofklkfljcocdinagocijmpgbhab)提供了类似的功能。如果你经常需要在语言之间切换，Chrome 扩展仍然值得一看——即使现在这已经成为 Gmail 的内置功能，因为它可以在许多其他工具上工作，包括 Google Drive、Search 和 Translate。谷歌还提供了一个 [Windows 桌面客户端和一个具有类似功能的安卓应用](https://web.archive.org/web/20230123111506/http://www.google.com/inputtools/cloud/)。
