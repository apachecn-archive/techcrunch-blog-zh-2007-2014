# Forrester 发现，尽管平板电脑入侵了工作场所，但员工也更喜欢键盘

平板电脑！它们无处不在，并迅速取代销量下滑的个人电脑。但仅仅因为平板电脑越来越受欢迎，并不意味着企业 IT 部门应该全力以赴开发基于触摸的界面。一份新的 Forrester 报告表明，人们希望他们的平板电脑配有键盘，然而，这是有道理的，因为无论任何人说什么，在所有可能的宇宙中，在软键盘上打字都不可能胜过使用全尺寸笔记本 QWERTY。

Forrester 对美国和欧洲的 1，000 多名信息工作者进行了调查，发现目前或希望在工作中使用平板电脑的人中，有 62%的人还希望使用无线键盘或键盘基座附件。35%的人想要可转换的笔记本电脑设备，另外 34%的人说他们想在需要的时候使用平板电脑，然后在大量打字时切换回标准电脑。

这对非苹果平板电脑市场来说可能是个好消息，因为华硕和 Xplore(在由 [ABI 研究](https://web.archive.org/web/20221208135311/http://www.businesswire.com/news/home/20130816005312/en/Samsung-Narrowly-Tops-ABI-Research%25E2%2580%2599s-Media-Tablet)针对商业的平板电脑供应商调查中排名第三)等原始设备制造商生产具有替代外形和键盘附件的平板电脑设备。但在微软将此视为 Surface 平板电脑的一个亮点之前，它需要看看这些设备的[销售和性能，特别是 Windows RT](https://web.archive.org/web/20221208135311/https://beta.techcrunch.com/2013/08/10/microsoft-doesnt-want-to-admit-windows-rt-is-dead/ "Microsoft Doesn’t Want To Admit Windows RT Is Dead")，并意识到不。只是不

对于平板电脑专用键盘配件制造商来说，这可能是一个更好的消息——仅仅向企业和商业环境销售，就可能有巨大的市场，有助于为当前的 iPad 部署配备员工渴望的键盘。此外，一个真正的类别桥接设备仍然有兴风作浪的空间。
