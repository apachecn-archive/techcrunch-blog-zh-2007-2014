# 购买一台新的 Mac，用这个简单的方法在其他 Mac 上安装 iLife

好消息！如果你在你的生活中缺少了一点点快乐，请不要害怕，[苹果](https://web.archive.org/web/20221208142733/https://beta.techcrunch.com/tag/apple)已经为你想好了。iLife 以 GarageBand、iPhoto 和 iMovie 的形式包含在新款 MacBook Air 和 Mac mini 中。幸运的是，一旦你的 Mac App Store 用户帐户附带了 iLife '11 的许可副本(就像你的新 Mac 附带的一样)，它就可以在其他也链接到该帐户的 Mac 上下载。

一旦您购买了 iLife '11(无论是在商店的包装盒中，还是作为新购买的 MacBook Air/Mac mini 的一部分)，它将显示在 Mac App Store 的“购买”标签下。也就是说，只要你运行的是 OS X 10.6.6 Snow Leopard 或更高版本，你就可以首先访问 Mac App Store。所以你坐在你的新 MacBook Air 前，看着 Mac App Store 的“购买”,那里有 iLife '11。

还和我在一起吗？你应该害怕，那是最简单的部分。(我老弟，这*都*真容易。)

好了，现在我们来看看你的另一台 Mac，可能比较旧，它也需要一点 iLife '11。只要您用相同的 Apple ID 注册了新 mac，iLife’11 也会突然出现在该电脑的“购买”标签下供下载。您现在所要做的就是安装它，iPhoto、iMovie 和 GarageBand 的最新版本很快就会准备好，等待您的到来。
