# 埃里克·舍恩菲尔德:2011 年将会是 1999 年的重演吗？(TCTV)

![](img/b398b74df8a6482f91e8ae30186d1b39.png "KeenSchonfeld")2010 年只有四天多了；离 2011 年还有四天。那么在新的一年里有什么值得期待的呢？关于 2011 年，我们最希望和害怕的是什么？

对于 TechCrunch 联合编辑 [Erick Schonfeld，](https://web.archive.org/web/20221007021009/http://www.crunchbase.com/person/erick-schonfeld)来说，2011 年可能是触摸成为计算体验中心的一年。这也可能是移动和社交—[约翰·杜尔登的第三次浪潮](https://web.archive.org/web/20221007021009/https://beta.techcrunch.com/2010/10/21/john-doerr-sfund/)—最终成为科技行业主导部门的一年。

2011 年还会是 2000 年吗？我们会看到围绕移动和社交的所有乐观情绪崩溃吗？弗雷德·威尔逊关于科技业当前非理性繁荣的[警告](https://web.archive.org/web/20221007021009/http://www.avc.com/a_vc/2010/11/storm-clouds.html)正确吗？Schonfeld 不这么认为，虽然他承认有太多的模仿公司，但他认为推动当前繁荣的老牌公司——脸书、Groupon、Zynga 和 Twitter——是真实的。

舍恩菲尔德是对的吗？我们应该像 1999 年那样狂欢吗？或者，2011 年能像 2000 年一样，作为音乐死亡的一年被人们记住吗？
