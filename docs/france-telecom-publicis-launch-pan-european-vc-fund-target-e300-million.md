# 法国电信、阳狮启动泛欧风险投资基金。目标:€3 亿

彭博、T2 和 T4 的阳狮集团已经联手成立了一个风险投资基金。

这些公司承诺共同投入 1.5 亿欧元(可能对半分)，并正在寻求外部投资者将基金规模增加一倍，该基金将专注于支持在法国和欧洲其他国家建立数字公司的初创企业家。

彭博估计该基金的规模“超过 1 亿欧元”。如果有足够多的投资者排队支持这只新基金，它将达到 3 亿欧元(4.115 亿美元)的实际目标规模。
