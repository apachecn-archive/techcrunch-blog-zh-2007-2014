# OneRiot 的“社交兴趣得分”为广告商定义了移动受众细分

正如我们今年早些时候所写的， [OneRiot](https://web.archive.org/web/20221231100631/http://www.oneriot.com/) [推出了](https://web.archive.org/web/20221231100631/https://techcrunch.com/2011/02/25/oneriot-rolls-out-social-targeting-service-for-mobile-ads/)移动广告社交定向服务，在移动应用程序中提供高度定向的广告。与 Klout 的社交影响力评分类似，OneRiot 开发了一个“社交兴趣评分”，根据社交兴趣类别来定义移动受众群体。

通过新的社交定位服务，OneRiot 允许广告商在移动设备上接触目标受众
,从忙碌的母亲到技术影响者，到体育爱好者，再到时尚人士。细分和定位基于诸如观众兴趣概况、人口统计、社会影响和实时对话等因素。OneRiot 的受众资料是通过挖掘和分析来自服务(即 Twitter)的公共大数据社交流创建的。该公司表示，这些数据来自于大量使用移动设备内容的用户，这些内容与他们当前的社交活动相关，包括状态更新、推文、照片、广告等。

现在 OneRiot 正在开放这种细分算法，允许用户查看你的“社交兴趣细分”该算法基于许多因素，包括关注图、兴趣图、影响图以及这些图之间共享的链接背后的内容。你只需输入一个你在手机上关联的社交媒体用户名(如你的 Twitter 账号)，OneRiot 就会返回你的“社交兴趣群”——以及一个我们可能会向你展示的广告示例。H

例如，如果你在 Twitter 上关注 BBC Top Gear，你关注的其他朋友发布了 nytimes.com 汽车页面的链接，那么 OneRiot 会将该活动归类为“汽车”类别。OneRiot 随后将与沃尔沃等汽车品牌广告商合作，帮助他们向手机上的“汽车”受众群体提供基于兴趣的移动定位活动。

受众定位并不新鲜——这在网络广告中时有发生。但有趣的是，看看你作为一个 Twitter 用户属于哪一类，以及广告商是如何把你放入特定人群的。
