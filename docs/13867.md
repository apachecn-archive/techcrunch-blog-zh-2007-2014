# OpenStreetMap 的贡献者 MapBox 从骑士基金会 TechCrunch 获得 57.5 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/09/21/openstreetmap-apples-mapping-partner-for-iphoto-gets-575k-from-the-knight-foundation/amp/>

虽然苹果继续因其在 iOS 6 中对地图的笨拙表现而受到非常负面的报道(以及令人尴尬的照片证据),但地图世界的另一项发展也与苹果的一项服务有关。MapBox 为苹果 iPhoto 服务中使用的开源 OpenStreetMap 做出了贡献，它正在获得现金注入。今天，MapBox 宣布从骑士基金会获得 575，000 美元的赠款，表示将用于帮助改善 OpenStreetMap 的核心基础设施——促进开放数据。

你可能还记得，苹果公司使用 OpenStreetMap 为 iPhone 的 iPhoto 提供位置数据——苹果公司过了一段时间才承认这一点。苹果获得了 OpenStreetMap 数据，并在此过程中放弃了谷歌，这似乎预示着它将在远离搜索巨头的更大地图努力中做些什么。但它并没有将所有产品都移植到自己的地图解决方案中，而是仍在为某些服务使用其他数据，比如 iPhoto。在最新版本中，OpenStreetMap 仍然包含在 iPhoto 的确认信息中。

“我们的目标是利用这笔投资，让向 OpenStreetMap 添加数据变得更容易，让 OpenStreetMap.org 更加社会化，以支持其继续快速增长的社区，并让人们更容易从 OpenStreetMap 获取数据来制作自己的地图，”领导 MapBox 数据团队的 Alex Barth 在宣布这一消息的[博客帖子](https://web.archive.org/web/20230213063049/http://mapbox.com/blog/knight-invests-openstreetmap/)中指出。

他指出，这些工具将继续保持开源，数据存储在 GitHub 等平台上。

过去几个月，该公司一直在与骑士基金会合作，其基础是开放数据的承诺。

鉴于围绕苹果地图的所有喧嚣，以及[谷歌可能会或可能不会做什么](https://web.archive.org/web/20230213063049/https://techcrunch.com/2012/09/20/crap-maps/)来回应这一消息，以及[诺基亚](https://web.archive.org/web/20230213063049/http://www.nokia.com/)也在这一领域大规模发展的事实——诺基亚的地图数据是其在第三方交易中设法利用的关键资产之一，而此时其手机业务仍然面临巨大挑战；由于其在地图数据方面的优势，它也在努力将自己的手机呈现为独特和特别的——这是该领域的一个关键时刻。

从这个意义上说，这可以帮助 OpenStreetMap 项目继续在所有这些其他专有努力中保持领先地位。

巴斯指出，“我们认为[这笔资金]所允许的工作将产生我们迄今为止最大的影响。”

其他使用 OpenStreetMap 数据的网站包括 Foursquare、NPR 和维基百科。
