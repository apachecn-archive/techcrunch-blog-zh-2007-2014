# “决定”继续前进 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/12/deciding-to-move-on/>

AOL 发表了以下声明:

> “收购 TechCrunch 对 AOL 和我们的股东来说都是成功的，我们对它的未来感到非常兴奋。TechCrunch 的创始人迈克尔·阿灵顿决定离开 TechCrunch 和 AOL，转投他新成立的风险基金。迈克尔是一位世界级的企业家，我们期待通过投资他的风险基金来支持他的新努力。Erick Schonfeld 被任命为 TechCrunch 的编辑。TechCrunch 将在未来几个月扩大其编辑领导力。
