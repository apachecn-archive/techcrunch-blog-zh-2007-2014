# 新雅虎！Mail 增加了短信功能

![yahoomail](img/cc840ff0ec746bdf054052788309b73c.png)

在 Gmail 出现之前，我使用一种叫做 [Oddpost](https://web.archive.org/web/20220928040256/http://en.wikipedia.org/wiki/Oddpost) 的创新电子邮件服务，这种服务在 2002 年发布时是格兰德河这边最酷的 JavaScript 应用之一。Oddpost 被雅虎收购了！2004 年，底层技术帮助了今天的雅虎！邮件。

昨晚，雅虎！推出了一个更新的邮件界面，除其他功能外，还包括集成的文本消息(SMS)。新的界面和功能现在是所有雅虎的标准！邮件账户也是。我拿着它玩了一会儿，心想:“真不错。”

然后我去查了一下我的 Gmail 账户，真的很快，很简单，很直观。

重点是，雅虎！邮件太棒了。不过，也许太棒了。它非常繁忙(到处都是广告),我认为有许多功能会受到年轻一代的欢迎，他们希望集成文本消息和即时消息，或者希望电子邮件不仅仅是电子邮件的人。不过，在我看来，如果你为了电子邮件而想要坚如磐石、快速可靠的电子邮件，Gmail 仍然是卫冕冠军。
