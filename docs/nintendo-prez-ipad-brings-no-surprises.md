# 任天堂 Prez: iPad 带来“没有惊喜”

![](img/9a2db389228328d967f465669f1dd09a.png)

任天堂总裁岩田聪对 iPad、3D 或高清电视完全不感兴趣。事实上，他很高兴以同样保守的方式经营公司，并认为 3D 和高清是时尚。

“我怀疑人们是否会戴着眼镜在家玩游戏。其他人会怎么看？”他在美联社的报道中说。

在过去的一周里，我一直在运行一个 3D PC 平台，我很高兴地说，我的印象很好。*倦怠天堂*很华丽*蝙蝠侠*看起来很惊艳。然而，任天堂不做任天堂不想做的事情，所以如果你想在 3D 中看到马里奥击败林克，你会被解决。

任天堂，继续做你正在做的。依靠吸引男性儿童的特许经营权来支撑公司，并以制造人为短缺的方式继续销售低成本硬件。我完全愿意在*超级纸马里奥 2:火的追求*、*超级马里奥纸银河*、*超级马里奥巴什兄弟奥林匹克*以及这一季你推销的任何东西上多花几百。我还需要更多的塞尔达头衔，这次包括牧羊和飞船。然后，当孩子们很乖，厌倦了你的单调乏味时，就歇业吧。

哦。由于 Wii 销售缓慢，任天堂的利润下降。
