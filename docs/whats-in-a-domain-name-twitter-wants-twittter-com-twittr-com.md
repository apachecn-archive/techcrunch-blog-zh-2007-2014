# 域名里有什么？推特想要 Twittr.com 的 Twittter.com

在与世界知识产权组织(WIPO)提出争议后，Twitter 最近成功获得了域名 Twiter.com，并且显然对更多的域名产生了兴趣。

上周，[公司提交了一份关于 Twittter.com 转会的投诉](https://web.archive.org/web/20230203081235/http://fusible.com/2011/11/twitter-is-going-after-another-typo-domain-twittter-com-with-an-extra-t/)(带有一个额外的‘t’)，现在我们注意到他们也在[瞄准 Twittr.com](https://web.archive.org/web/20230203081235/http://www.udrpsearch.com/wipo/d2011-1992)。

你可能记得 Twitter 是[最初叫做](https://web.archive.org/web/20230203081235/https://techcrunch.com/2011/03/13/twitters-beginning/) [Twttr](https://web.archive.org/web/20230203081235/https://techcrunch.com/2006/07/15/is-twttr-interesting/) ，所以让我感到惊讶的是，他们并没有已经拥有后一个域名，也没有在过去提起诉讼来获得它。事实上，该公司今年早些时候才向世界知识产权组织提交了第一份域名争议。

目前，Twittr.com 指向一个充满上下文广告的页面。我猜这个域名很快就会被转移到 Twitter 上。

与此相关的是:Twitter 的联合创始人和前首席执行官伊万·威廉姆斯最近写了一篇博文，认为如今域名越来越不重要。我倾向于不同意。
