# 谷歌收购移动音乐和娱乐平台 PushLife

谷歌以近 2500 万美元收购了总部位于多伦多的移动娱乐初创公司 [PushLife](https://web.archive.org/web/20230204224829/http://www.pushlife.com/) ，初创公司 North 正在[报道](https://web.archive.org/web/20230204224829/http://startupnorth.ca/2011/04/08/pushlife-acquired-by-google/)。这家让你能够将 iTunes 和 Windows Media player 库移植到安卓和黑莓等非苹果手机上的公司，已经被搜索引擎吞并，试图超越搜索领域。

PushLife 由 RIM 前员工雷·雷迪(Ray Reddy)于 2008 年创建，让你在手机上管理壁纸、音乐、视频、铃声和其他媒体，直到被收购，并着手*“为人们构建身临其境的移动体验，以便在多种设备上播放、组织、分享和购买数字内容。”相反，它被谷歌收购了。*

鉴于[报道](https://web.archive.org/web/20230204224829/https://techcrunch.com/2010/09/15/google-music/)称谷歌可能很快就会推出自己的音乐服务，人们很容易认为谷歌会希望 PushLife 在整合到加拿大的谷歌工程团队后执行类似的功能，因此这次抓取尤为有趣。
