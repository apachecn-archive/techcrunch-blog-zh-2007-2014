# 脸书禁止 Netlog 违反其政策

**独家-**TechCrunch 获悉，一项涉嫌违反脸书政策的行为导致了一个相对高调的伤亡: [Netlog](https://web.archive.org/web/20230203104044/http://www.crunchbase.com/company/netlog) ，一个规模庞大的欧洲社交网络，在本周早些时候被禁止使用[脸书](https://web.archive.org/web/20230203104044/http://www.crunchbase.com/company/facebook)平台。

Netlog 在美国并不广为人知，但它已经存在了十多年，拥有 8000 万用户，其中大多数在欧洲和中东。据母公司[大众传媒](https://web.archive.org/web/20230203104044/http://www.crunchbase.com/company/massive-media)称，脸书没有解释他们违反了哪条具体规定，或者以何种方式违反。

然而，我们被告知，大约 150 万使用脸书连接注册 Netlog 的用户由于禁令而无法登录该网站，禁令于上周二生效。

最初，Netlog 被告知他们违反了脸书平台政策中的一个全新条款，该条款是大约两周前悄悄添加的，规定脸书应用不能链接、推广、整合或重定向到任何竞争社交平台上的任何应用，但这似乎是脸书方面的一个痛苦的沟通错误。

事实上，Netlog 从未在脸书平台上推出过 canvas 应用程序，但曾考虑过构建一个——出于显而易见的原因，这些计划现已被放弃。

脸书证实，在最初关于突然禁令的沟通中出现了错误，但他们已经通知 Netlog 他们被禁的真正原因。

当被问及这些原因是什么时，脸书发言人只会说:

> Netlog 采取了大量措施来访问内部脸书 API，并故意损害我们平台的预期限制。我们采取了适当和必要的步骤来保护脸书的人民。

与此同时，大众传媒表示不知道有任何故意的不当行为，但他们会调查此事，然后再做进一步评论。因为脸书最初表示，Netlog 因链接到竞争对手的社交平台而被禁止，大众传媒实际上考虑向欧盟委员会投诉脸书的“反竞争行为”。

重大危机和公关风波得以避免。
