# 筹集到 2000 万美元后，MyEdu 推出了新工具，帮助大学生与雇主建立联系 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/10/with-20m-raised-myedu-launches-new-tools-to-help-college-students-actually-connect-with-employers/>

总部位于奥斯汀的 MyEdu 于 2010 年成立，旨在帮助学生降低获得大学文凭的成本，消除高等教育过程中从决定正确的学校到找到研究生就业的各个方面的摩擦。从那以后，这家创业公司一直在有条不紊地收集学生的数据和大学的官方学术数据，以确定导致更高成本(和更低保留率)的模式。

今天，MyEdu 为学生提供了查看教授和课程评级的能力，以帮助他们选择合适的课程，同时提供了一个访问课程描述、评估、时间表规划以及学位路线图和进度仪表板的单一位置，使学生能够制定学术计划并跟踪他们向学位迈进的情况。另一方面，MyEdu 让教授、学生和公司能够访问 2.25 亿份官方 GPA 记录，以跟踪学生的成绩和成果。

MyEdu 的学术数据和见解共同构成了它认为是美国目前最大的课程、成绩历史和学位信息的学术知识库，它利用这些信息来支持安全的基于云的教育应用程序，帮助学生更好地规划和管理学期和学位，就像 Edmodo 一样。

撇开最高级不谈，这种模式(免费的)已经导致了持续的采用，MyEdu 现在正在收集来自 800 多所(两年制和四年制)院校的数据。这家初创公司在 2011 年增加了 530 万用户，创始人表示，到今年年底，用户数量将翻一番。因此，MyEdu 还透露，它现在已经筹集了超过 2000 万美元的外部投资，其中大部分资金来自贝恩资本风险投资公司及其战略投资者德克萨斯大学。

到目前为止，虽然这家初创公司声称可以帮助学生管理学术过程的每个部分，但它在毕业后的就业选择方面相当有限。因此，今天这家初创公司推出了一项新服务，旨在通过帮助学生找到高质量的工作和实习机会，并允许雇主改善他们的大学招聘工作，更有效地弥合高等教育和就业之间的差距。企业试点计划成员包括安永、VMware、梅西百货和 ADP 等公司。

对于学生来说，MyEdu 的新平台现在提供了一种熟悉的、类似 LinkedIn 的体验，允许他们创建学术档案，在那里他们可以展示自己的课程、技能、才华或突出个人经历(例如语言技能和证书)，同时直接与其他学生和雇主联系。MyEdu 的新平台对学生和机构都是免费的。

与此同时，雇主每月支付一笔 SaaS 费用来访问这个新平台，这使他们能够利用特定的搜索标准，如学校、专业、毕业年份、所学课程、工作和课外背景等，来定位招聘信息和确定候选人。公司还可以使用该平台直接与寻求课程和活动建议的学生进行互动和指导，这将有助于他们在放学后找到工作，并访问 MyEdu 的学术数据数据库。

随着 MyEdu 更加积极地进入学生网络和招聘领域，它不仅要与提供学位管理和学生贷款服务的初创公司竞争，还要与大学网络领域的公司竞争。像 BranchOut 和 Identified 这样的公司瞄准的是年龄范围内的年轻一代，并为年轻一代提供了许多建立个人资料、分享成就和与雇主联系的方式。 [AfterCollege 是另一家试图在高等教育和雇主之间架起一座桥梁的公司](https://web.archive.org/web/20221005221449/https://beta.techcrunch.com/2012/08/14/aftercollege-relaunch/)，它建立了一个面向大学生和应届毕业生的 LinkedIn。

但 MyEdu 的优势在于，它可以在一个庞大的数据集基础上建立自己的新档案和招聘选项，这个数据集已经包含了一个更大的课程和成绩历史信息库——这是它与竞争对手不同的地方，也是它相对于竞争对手的一个很大的专有优势。

[![](img/cb724c4fed79703542fc3c70df55f1a9.png "Screen shot 2012-10-10 at 3.35.17 AM")](https://web.archive.org/web/20221005221449/https://beta.techcrunch.com/wp-content/uploads/2012/10/screen-shot-2012-10-10-at-3-35-17-am.png) 随着高等教育越来越关注利用数据来提高学术成果、保持和个性化学习的重要性，MyEdu 正在建立一项宝贵的资产，并使其业务变得越来越有防御能力。

当然，最终，最重要的是这些工具在帮助学生降低教育成本和增加学术投资回报方面的有效性。随着 60%的学生实习在全国范围内获得工作机会，这家初创公司需要提高这一统计数据，并展示在他们的学术生涯早期促进学生与雇主互动的价值。我们将从 MyEdu 中寻找更多关于其对学生成绩和保留率的影响的数据，因为这确实需要成为我们判断新兴教育技术平台的试金石。
