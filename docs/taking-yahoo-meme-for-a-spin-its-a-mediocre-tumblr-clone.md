# 带着雅虎迷因转一圈:它是一个平庸的 Tumblr 克隆

![](img/2307b146a42ac5d0a4ff19222b65cf3f.png)我们最近披露了关于[雅虎推出葡萄牙语](https://web.archive.org/web/20230130113139/http://techcrunch.com/2009/05/14/yahoo-makes-a-twitter-clonein-portugese/)服务的消息，该服务被称为 [Meme](https://web.archive.org/web/20230130113139/http://meme.yahoo.com/) ，从表面上看似乎倾向于类似 Twitter 的服务。但是现在我们收到了来自巴西葡萄牙用户的邀请码，我们知道这根本不像 Twitter。

当然，这都是关于微分享的，但这只是比较的终点。雅虎 Meme 更像微博服务 [Tumblr](https://web.archive.org/web/20230130113139/http://www.crunchbase.com/company/tumblr) 而不是其他任何东西，而且它是一个平庸的克隆。

它是这样工作的:你用你喜欢的用户名创建一个账户(需要雅虎个人资料)，你基本上从一个空的博客开始，你可以用文本、图像、视频、音乐或这些媒体的混合来填充。除了内容之外，你能添加到博客中的只是一个标题、一段 100 字的描述和一个头像。你可以在这里访问我的账号[。没有其他设置可以定制你的 Meme 博客，至少现在没有，而且到目前为止，除了一个指向更新的专用 URL 之外，人们没有办法在你发布的内容下创建评论线程，你也没有办法分享你的帖子。](https://web.archive.org/web/20230130113139/http://meme.yahoo.com/robinwauters/)

你可以搜索其他人的公共账户并关注他们——这本质上与订阅博客的 RSS 源相同——一旦你这样做了，这些用户的更新也会出现在你自己的信息流中。它类似于 Tumblr，有一个“转发”按钮(见第二个截图)，其工作方式与 Tumblr 内置的“重新博客”功能完全相同。

也就是说，Tumblr 给了你更多的定制选项，让你以多种方式更新你的博客，提供基本的分析，并附带一个 API。雅虎 Meme 目前还没有这些。

目前还不清楚雅虎是否只是在一种语言中对这项新服务进行桶式测试(尽管有超过 2 亿人在使用这种语言)，并计划从长远来看进行更广泛的推广，或者这是否是它所能做到的。我们也不确定是否会有更多的特性出现，或者它是否会继续像现在这样初级。

无论哪种方式，它都需要大量的工作才能吸引比一些早期测试者更广泛的受众。
