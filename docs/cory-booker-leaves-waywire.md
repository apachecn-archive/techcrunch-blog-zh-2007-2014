# 科里·布克将从 Waywire 董事会辞职，该公司正在寻找新的首席执行官

据了解竞选计划的消息人士透露，纽瓦克市长和参议员候选人科里·布克今天宣布，他将从他去年创立的公司董事会辞职。此次离职使得 Waywire 的三位联合创始人中只剩下一位，该公司仍处于私下测试阶段，正在寻找新的首席执行官。

退出 Waywire 董事会的决定在人们的意料之中，因为一些政治对手质疑布克继续参与初创公司。布克此前曾表示，如果当选，他将辞去 Waywire 董事会的职务，但随着他继续推进政治竞选，这加速了这些计划。随着他离开董事会，布克将把他在公司的股份捐赠给慈善机构。

布克的决定正值 Waywire 的困难时期，该公司尚未公开推出其产品。该声明是在 Waywire 联合创始人兼首席执行官[内森·理查森上个月离开公司](https://web.archive.org/web/20221209213707/https://beta.techcrunch.com/2013/08/16/waywire-ceo-leaves/)之后发布的。此前曾在 Content Next Media 和 Gilt Groupe 担任高管职位的理查森加入了 TechCrunch 的母公司 AOL，他将成为即将推出的 AOL 直播视频频道的[。](https://web.archive.org/web/20221209213707/https://beta.techcrunch.com/2013/08/30/nate-richardson-aol-live/)

根据 Waywire 发言人的说法，该公司非常接近于带来一位新的首席执行官，他可能会在未来几周内宣布。这是个好消息，尤其是该公司正处于从内容创作到视频监管平台的转型过程中。与此同时，联合创始人莎拉·罗斯(Sarah Ross)一直负责日常运营。

对 Waywire 来说，还有其他好消息:该公司已经从四家新投资者那里获得了种子期的延期，尽管它尚未准备好宣布筹集了多少资金或谁参与了融资。(这些细节可能会在宣布新 CEO 的同时公布。)该公司最初从首轮资本、埃里克·施密特的创新努力、Lady Gaga 的经理特洛伊·卡特和其他天使投资人那里筹集了 175 万美元。

尽管如此，Waywire 仍有大量工作要做，因为在布克、理查森和罗斯共同创建该公司 18 个月后，它仍处于私下测试阶段。这在一定程度上是 Waywire 战略变化的结果。最初，该公司希望通过制作和发布自己和用户的视频来挑战传统媒体渠道。但是它的目的更多的是成为[从多个来源收集和管理视频](https://web.archive.org/web/20221209213707/https://beta.techcrunch.com/2012/08/13/waywire-video/)的平台。

当 Waywire 准备推出其产品时，我们将看看它的表现如何。有了新的首席执行官和新的资金，没有关于布克参与的问题，这应该会在未来几个月内发生。
