# 莫斯科:目前为止我们的演讲者

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/11/15/techcrunch-moscow-our-speakers-so-far/>

![](img/d244faf8d375338deb7eee4bb759240d.png)

12 月 13 日，TechCrunch Europe 将来到俄罗斯莫斯科，这是我们第一次举办“TechCrunch Moscow”。该活动将在俄罗斯第一家私人科技孵化器“数字十月中心”举行，该中心位于一座历史悠久的制造业建筑 Krasny Oktyabr(俄语中称为“红色十月”/красныйоктябрь)。你现在可以在[这里](https://web.archive.org/web/20221208211037/http://techcrunch.timepad.ru/event/4011)买票了。

TechCrunch 莫斯科的与会者将享受一整天的时间，其中充满了行业领袖的演讲、交流机会以及预览有前途的俄罗斯科技初创公司的机会。

该会议由 TechCrunch Europe 与 [Kite Ventures](https://web.archive.org/web/20221208211037/http://kiteventures.com/) 联合举办，将聚焦于俄罗斯科技创业的关键问题

TechCrunch Moscow 旨在支持莫斯科和俄罗斯的初创企业和风险投资生态系统。这也是为了揭示目前市场上的趋势和最热门的公司和创业公司。如果你是 Le Web 的欧洲用户，并且想在莫斯科度过富有成效的几天，那么 TechCrunch Moscow 是你了解那里的机会。

由于该计划已满，因此不再有任何发言时间，所以请不要向 TechCrunch 团队询问此事。对于所有赞助和参展商的询问，请联系 TechCrunch.com 的活动和赞助总监 Alena。请确保所有电子邮件都是英文的。谢谢大家！

最终计划的更多细节将会公布，但目前我们可以说，俄罗斯领先的搜索引擎 Yandex 的首席执行官 Arkady VolozhSUP/Livejournal 首席执行官 Annelies van Den Belt 塞米恩·沃伊诺夫，Zeptolab 的创始人，《割断绳子》的创作者；《生意人报》首席执行官德米扬·库德里亚夫采夫；Lukasz Gadowski，Team Europe Ventures 创始人；KupiVip 创始人兼首席执行官奥斯卡·哈特曼(Oskar Hartmann)和许多其他人都将是演讲嘉宾。

**合作伙伴:**

美俄商务委员会(USRBC)是一个主要的贸易协会，总部设在 DC 华盛顿州，在莫斯科设有办事处，代表其美国和俄罗斯成员公司的贸易和投资利益。USRBC 通过代表其成员参与美国和俄罗斯政府的宣传工作，寻求扩大和加强美国与俄罗斯的商业关系；协助成员公司解决故障和开发新业务；提供信息和分析以支持业务决策；促进接触和建立联系的机会，包括向政府官员和私营部门领导人介绍情况。在[www.usrbc.org](https://web.archive.org/web/20221208211037/http://www.usrbc.org/)或[www.usrbc.org/ru](https://web.archive.org/web/20221208211037/http://www.usrbc.org/ru)了解 USRBC 事件、活动和业务发展机会。
