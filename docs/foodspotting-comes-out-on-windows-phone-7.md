# Foodspotting 在 Windows Phone 7 上推出

![](img/133ddbf8e4847ac46f9c34e15f8871f8.png)

[Foodspotting](https://web.archive.org/web/20230205013511/http://www.foodspotting.com/) ，一款食物照片分享应用，围绕特定菜肴的分享建立了一个社区，今天在 Windows Phone 7 上发布。Foodspotting Windows Phone 应用程序将保留 iPhone 和 Android 应用程序的所有功能，允许用户查看附近的菜肴，寻找特定的食物，查看餐厅的各种菜肴，并标记他们有兴趣尝试的食物。

与现有的 iPhone 和 Android 应用程序相比，该应用程序还具有更精简的界面和简化的反馈和语言，Foodspotting 表示，这些功能将很快移植到其他应用程序中。该公司计划让用户从应用程序中浏览他们的“想要”列表，并在更高版本的 Windows Phone 7 中关注人物和指南。

Foodspotting 外联负责人 Fiona Tang 解释了公司决定开发 Windows Phone 应用程序的原因，“我们实际上收到了社区的请求，它太漂亮了。[Foodspotting 首席执行官 [Alexa Andrzejewski](https://web.archive.org/web/20230205013511/http://www.crunchbase.com/person/alexa-andrzejewski) 非常注重设计，所以我们认为这是个好主意。”

Foodspotting 目前在这两个平台上有超过 62 万条评论和照片，超过 85 万次下载。这家初创公司(已经将其团队扩大到 10 名员工)目前正在对一款黑莓应用进行 beta 测试。
