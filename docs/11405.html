Synaptics在视频中演示Windows 8触控板手势

不久前我们听说 Windows 8 将通过触控板支持多点触控。当然，你现在可以做一些事情，但是微软和Synaptics做出的承诺在很大程度上已经被推迟了。但他们发布了一个视频，展示了如何使用多点触控触控板与Windows 8和Metro进行交互。

你可以在这里观看视频，但是为了你的方便，我也把它嵌入在这里:

我不得不说，其中一些东西看起来非常方便。我用的是PC台式机和Mac笔记本电脑，出于几个原因，两者都嫉妒对方。我喜欢我的MacBook的一个特点是多任务多点触控手势。用几个手指在桌面或应用程序之间切换已经成为第二天性，看起来这已经很好地融入了Synaptics的概念。

本质上，他们所做的只是将你在大触摸屏上的输入映射到普通的触摸驱动程序中；困难可能是精度和过滤，确定手指的确切位置，以便您可以提供触摸屏级别的精度。

自然有障碍。我很久以前写过关于的文章，直接和间接操作UI元素的根本区别，我不能说这个视频对我的观点有任何影响，我认为我们可能永远都需要这两者。毕竟，告诉你手指放在哪里的反馈环有点不完整:它需要让你知道你的手指悬停在哪里，因为你不能直接触摸这些项目。事实上，我认为你可以触摸，然后按下点击，就像我们已经做了一段时间，但这不够连贯。

然而，边缘手势看起来像是一种乐趣。在应用程序之间切换看起来就像在平板电脑上一样自然和有用，我喜欢这样提出魅力栏的想法。存在意外激活这些的危险，但这是软件优化的问题，Synaptics毫不逊色。事实上，通常是ODM和OS制造商没有正确实施他们的解决方案。

他们将在本周晚些时候的微软生态系统峰会上展示这一成果。同时，更多信息可在 Synaptics新闻稿中找到。
