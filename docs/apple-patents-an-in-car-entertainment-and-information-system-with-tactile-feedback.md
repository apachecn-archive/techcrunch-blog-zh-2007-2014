# 苹果申请了一项带有触觉反馈的车内娱乐和信息系统的专利

苹果公司今天(通过 [AppleInsider](https://web.archive.org/web/20221007023557/http://appleinsider.com/articles/13/07/09/apples-safe-vehicle-telematics-patent-mates-touchscreen-with-tactile) 获得了一项车载触摸屏远程信息系统的专利，该系统将为司机提供触觉反馈，帮助他们保持对道路的关注。这有点像通过让苹果设计汽车仪表板信息和娱乐控制台功能的每一个元素，将 iOS 7 的新汽车专用功能带到他们的逻辑结论。

该系统将使用旋钮、滑块、触摸屏控制和当前车载系统中基本存在的其他东西，但扩展了其完全可由用户编程的能力，并能够将其范围扩展到像挡风玻璃雨刷控制这样的系统。该专利谈到了通过应用程序和信息的可定制性，例如通过控制台显示的股票应用程序。

专利申请本身是一个老问题，但苹果现在才开始真正表明它将如何开始在这个新市场扩张的意图。车载 iOS 希望使用 Wi-Fi 和 AirPlay 从本质上取代 iPhone，接管汽车内现有的基础设施。重新配置的 iOS 主屏幕突出显示了相关功能，显示在仪表板触摸屏上，完全绕过了任何类型的内部信息娱乐系统。

当谈到新产品时，所有的目光都集中在客厅和手腕上，苹果的下一个地盘，但汽车可能是一个更合乎逻辑的新领域。根据最近的一项研究，像黑莓这样的竞争对手已经凭借其 QNX 操作系统在这场游戏中占有一席之地，该操作系统为世界上 60%的信息娱乐远程信息处理系统提供动力。

苹果已经在与汽车制造商合作集成 Siri，车载 iOS 将进一步扩展与合作伙伴的关系。换句话说，这种方法正在发展，苹果公司是一家硬件和软件完美结合的公司，所以我们可以看到，随着他们在这个市场上继续大步前进，它将承担更多的责任。
