# 尝试，再尝试:XYBOARD 平板电脑会犯同样的错误 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/12/06/try-try-again-the-xyboard-tablets-make-all-the-same-mistakes/>

听着，我还没有玩过新的 Xoom 2 型号——或者，正如威瑞森称之为 [XYBoards](https://web.archive.org/web/20221206195609/https://beta.techcrunch.com/2011/12/06/droid-xyboard-tablets-get-official-hitting-verizon-stores-this-month/) 的新设备。我敢肯定 8.9 和 10.1 两款都是精品平板。他们可能会像冠军纯种马一样驾驭上百个蜂巢应用。不过没关系。没有人会买这种新的平板电脑。它们会比原来的 Xoom 翻得更惨。

目前市场上主要有两款平板电脑:500 美元的 iPad 和 200 美元的 Kindle Fire。因此，只有两个价格点。iPad 为全功能平板电脑设定了 500 美元的门槛，而 Fire 也为廉价机型设定了同样的门槛。任何试图将自己塞进这种定价方案的平板电脑目前都注定要失败。低端想要亚马逊血统，而高端知道他们想要 iPad。我承认，中端安卓机型很强，但它们并不能吸引普通消费者。

以摩托罗拉 Droid XYBoard LTE 平板电脑为例。不要介意，它本质上是与原始 Xoom 相同的硬件，只是愚蠢地增加了一个红外发射器。[不要介意这个愚蠢的名字](https://web.archive.org/web/20221206195609/https://beta.techcrunch.com/2011/11/14/xoom-to-give-way-to-xyboard/)。见鬼，忘记它缺少原来的 SD 卡插槽。它的价格完全超出了它的市场。只有无知的粉丝才会买。

10.1 型号的价格为 529 美元，但需要两年的协议，每月 30 美元起。这意味着，两年后，当 Tegra 2 平板电脑比当前型号落后无数代时，客户将至少花费 1250 美元。更糟糕的是，由于数据包与合同绑定，如果 XYBoard 所有者决定提前取消数据包，他们需要支付昂贵的提前终止费。

尽管可能令人震惊，但这种荒谬的定价方案并不是 XYBoard 独有的。HTC Jetstream 于 9 月份在美国电话电报公司上市，最初两年的合同价格为 699 美元(后来降至 599 美元)。iPad 在威瑞森和美国电话电报公司均有销售，价格略高于 HTC 和摩托罗拉，但不需要同样的两年合约。用户可以随时打开或关闭 3G 无线网络，而无需支付罚金。

我是安卓的忠实粉丝，但过去的九个月是痛苦的。几乎每个月都会有一个发热的平板电脑漏水，这将导致 Android 社区无法控制地发痒。但随后会宣布官方细节，而 MSRP 通常会扼杀这种言论。除了华硕将 Transformer 定价为 399 美元这一明显的例外，Android 制造商无知地将他们的第一代设备直接与 iPad 进行定价。他们没有机会。缓慢但坚定地，iPad 把每一个都砍倒了，直到大火烧到了现场。现在，作为一个 Android 支持者，我事实上的冠军是一个 200 美元的预算平板电脑，它去掉了蜂窝中所有漂亮的用户元素，支持一个增强的亚马逊店面。

XYBoard 可能会很快被另一个型号取代。它建立在一个旧的 NVIDA 平台上，随着冰淇淋三明治即将推出，谷歌和摩托罗拉移动可能会推出一款旗舰设备。但这就是现在的安卓游戏:用改造过的部件制造的平板电脑充斥市场。这可能有利于制造商和无线运营商的底线，但将顶级型号与 2 年合同捆绑在一起的做法最终会降低采用率。最近的研究表明，Android 平板电脑的市场吸引力非常有限，这些买家不太可能愿意支付提前解约费来获得稍微升级的设备。

从各方面来看，2012 年应该是安卓平板电脑的大年。据说冰淇淋三明治将统一 Android 平板电脑和手机的发展，这将有望导致平板电脑应用的涌入。此外，虽然安卓的消费者兴趣有限，但自 2011 年上半年以来已经翻了一番。然而，如果 Android 平板电脑制造商继续遵循与 2011 年相同的定价方案，该平台将开始试水，为大白鲨创造一顿轻松的大餐，这就是 Windows 8。
