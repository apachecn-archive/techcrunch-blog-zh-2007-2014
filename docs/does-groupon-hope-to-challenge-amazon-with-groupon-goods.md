# Groupon 是否希望用‘Groupon 商品’挑战亚马逊？

上周，每日交易巨头 Groupon 注册了一系列包含“groupongoods”的域名([)。com](https://web.archive.org/web/20230203053612/http://whois.domaintools.com/groupongoods.com) ，。网，。我们，。信息，。有趣的是。fr)。显然，保护这些域名并不一定意味着该公司将很快或甚至在遥远的将来推出名为“Groupon Goods”的服务，但让我们假设一下，这正是他们计划要做的。

想象一下，如果 Groupon 开始指示其遍布世界各地的庞大销售队伍开始关注餐厅、水疗、按摩和普拉提课程以外的机会。

更好的是:如果他们最终寻求与像 Alice.com、[、Drugstore.com](https://web.archive.org/web/20230203053612/http://drugstore.com/)、[、Pets.com](https://web.archive.org/web/20230203053612/http://pets.com/)、[、Overstock.com](https://web.archive.org/web/20230203053612/http://overstock.com/)或[Newegg.com](https://web.archive.org/web/20230203053612/http://www.newegg.com/)这样的公司建立互惠互利的合作伙伴关系，以便扩大业务，销售打折食品、家庭用品、消费电子产品等等，那该怎么办？

从理论上讲，这家快速增长的公司可以应用其成功的“本地商务”模式，并利用其庞大的国际用户群(约 1.2 亿用户，并且还在增加)，冒险提供杂货、家居用品和其他经常在网上购买的商品的交易。实际上，在我看来这很合理。

首先，随着 Groupon 准备上市，这将是 Groupon 实现收入来源多元化和加快销售增长的一种方式，而不一定会增加大量的管理费用(特别是如果他们寻求合作伙伴关系而不是在内部建立这样的服务)。

值得注意的是，这不是 Groupon 第一次做出这样的举动。他们已经与巨头 Expedia 和 T2 livening 合作，分别提供 T4 旅游、Groupon 度假和活动门票的在线交易。

我们已经联系了 Groupon——也许他们足够冷静，不会拒绝对谣言或猜测发表评论。希望是永恒的，等等。

一个运作良好的“Groupon Goods”有可能成为亚马逊的挑战者，亚马逊不仅因其 Amazon.com 电子商务业务而闻名，还因其杂货购物网站 [Amazon Fresh](https://web.archive.org/web/20230203053612/http://fresh.amazon.com/) 和(收购的)小众在线商店如[Diapers.com](https://web.archive.org/web/20230203053612/http://diapers.com/)、[Soap.com](https://web.archive.org/web/20230203053612/http://soap.com/)、[Wag.com](https://web.archive.org/web/20230203053612/http://wag.com/)和[Zappos.com](https://web.archive.org/web/20230203053612/http://zappos.com/)而闻名。

亚马逊还拥有 Woot，提供一系列商品的日常交易(主要是消费电子产品)。

另一方面:这不会构成单方面的攻击。毕竟，亚马逊也在追逐 Groupon 的面包和黄油，目前正在美国悄然推出亚马逊本地服务。

Groupon 最近宣布，2011 年第二季度，其收入为 8.78 亿美元，净亏损 1.027 亿美元(上半年亏损约 2.05 亿美元)。Groupon 目前在全球拥有约 10，000 名员工。

那么，Groupon 手头有没有[的猫](https://web.archive.org/web/20230203053612/https://techcrunch.com/2011/06/20/groupon-responds-to-ipo-backlash-with-a-cheeky-feline/)来助长或揭穿上述 Groupon 商品的谣言？
