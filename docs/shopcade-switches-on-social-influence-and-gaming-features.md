# Shopcade 开启社交影响力和游戏功能

Shopcade 的平台让人们通过向脸书的朋友推荐产品获得奖励。到目前为止很简单，也不完全是原创。更有趣的是，它正在迅速扩大在美国的规模，超越其在伦敦的欧洲基地，并允许用户获得积分奖励，这些积分可以兑换成商家提供的优惠。这包括像 iTunes 中的音乐这样的数字产品。

该网站上有大约 6500 万件产品，还有许多主要品牌，如 Urban Outfitters 和 Adidas，Shopcade 有大量库存。但是他们试图通过只奖励用户每次购买而不是每次点击来避免“社会垃圾邮件”。

现在，该网站正在引入一种他们称之为“购物 Klout”的东西，它在网站内部工作。

他们新推出的“引领潮流得分”根据用户对彼此购物习惯的影响来给用户打分。这是该网站游戏化的新水平，可能会大大提高参与度。

用户现在需要完成一系列“任务”或挑战才能获得奖励，其中一些是合作伙伴品牌的独家奖励。此外，一个“Matcheroo”游戏允许用户将他们最近在 Shopcade 中展示的产品与脸书朋友进行匹配。他们又一次赢得了分数。

不出所料，该公司最近聘请了游戏设计师劳伦斯·克拉克(Lawrence Clark)专注于在网站中开发和构建更多的游戏机制。

这整个举动可能是为了应对[梦幻商店](https://web.archive.org/web/20221210070000/http://fantasyshopper.com/)的巨大增长，在那里用户可以通过建造装备和花费虚拟货币来“玩”，也可以购买现实世界的物品。社交购物的游戏化似乎是新的黑色。

不幸的是，这一购物社交影响力评分对 Shopcade 来说过于内部化。我想看到它变成更多的产品。

创始人兼首席执行官 Nathalie Gaveau 是 PriceMinister 的联合创始人(已被日本电子商务网站 Rakuten 收购)。天使投资人包括欧洲零售商家乐福前首席执行官丹尼尔·伯纳德(Daniel Bernard)、Eidos Games 联合创始人伊恩·利文斯顿(Ian Livingston)和 BBC 前总干事约翰·伯特勋爵(Lord John Birt)。
