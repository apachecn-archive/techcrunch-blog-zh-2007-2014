# 马克·扎克伯格是 Google+上最受关注的用户

根据 Google+的统计数据[，脸书首席执行官马克·扎克伯格是 Google+上关注最多的用户，这让](https://web.archive.org/web/20230210074001/http://socialstatistics.com/)[拉里·佩奇](https://web.archive.org/web/20230210074001/http://www.crunchbase.com/person/larry-page)和[谢尔盖·布林](https://web.archive.org/web/20230210074001/http://www.crunchbase.com/person/sergey-brin)有些尴尬。

这位脸书 CEO 有 21213 名粉丝，相比之下，谷歌 CEO 有 14798 名粉丝，谷歌社交沙皇维克·冈多特拉有 13783 名粉丝，谷歌联合创始人谢尔盖·布林有 11629 名粉丝，博主罗伯特·斯科博有 11389 名粉丝，谷歌垃圾邮件复仇者马特·卡茨有 9153 名粉丝，TWIT 创始人利奥·拉波特有 7566 名粉丝，谷歌的布拉德利·霍洛维茨有 7187 名粉丝，TechCrunch 的 MG·西格勒有 6579 名粉丝，博主吉娜·特拉帕尼有 5 名粉丝

Google+ Statistics 的创建者 Boris Veldhuijzen van Zanten 解释了这位首席执行官不太可能受欢迎的原因，“他有世界上最多的朋友，他们拍了一部关于他的电影，他比拉里和谢尔盖更帅。”我认为答案应该是这样的。一个人收到的关于 Google+的媒体报道越多，他的追随者就越多，因此 MG Siegler 排名第九。
