# Rdio 推出家庭计划

正如我们几周前提到的，Rdio 家庭计划现在可以为鲍勃·迪伦所说的“妻子和家人”以及“整个孩子”提供。

该计划两个账户的费用为 17.99 美元，三个账户的费用为每月 22.99 美元。无限网络和移动计划每月仍需 9.99 美元(家庭计划中增加的额外账户也是如此。

一个中央帐户控制其他帐户，并为家庭成员分配登录。你可以在 [Rdio 博客](https://web.archive.org/web/20230404170609/http://blog.rdio.com/us/2011/08/introducing-rdio-family-plans-.html)上了解所有信息。

您目前可以在多台设备上使用 [Rdio](https://web.archive.org/web/20230404170609/https://techcrunch.com/tag/Rdio) ，但不能同时在多台设备上使用，这使得家庭计划非常适合返校的孩子或希望在家或工作时在设备上收听 Rdio 并希望分享和降低成本的夫妇。
