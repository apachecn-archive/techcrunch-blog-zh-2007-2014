# 花旗分析师:谷歌在摩托罗拉的举动是“防御性的”

市场正试图理解谷歌[125 亿美元](https://web.archive.org/web/20230205003612/https://techcrunch.com/2011/08/15/breaking-google-buys-motorola-for-12-5-billion/)收购摩托罗拉移动公司的协议。(谷歌[股票](https://web.archive.org/web/20230205003612/http://www.google.com/finance?q=goog)在午盘交易中下跌了 2.8%，至 548 美元，而[摩托罗拉股票](https://web.archive.org/web/20230205003612/https://techcrunch.com/2011/08/15/market-thinks-googorola-deal-will-go-through-interdigital-shares-tank/)上涨了近 60%。例如，花旗分析师马克·马哈尼认为这是一种“防御性”举措。

在宣布后的今天，马哈尼发表了一份研究报告，对全现金交易持怀疑态度。他写道:“这或许是一种不错的现金使用方式。”。"但这是谷歌对现金的最佳利用吗？"有利的一面是，作为交易的一部分，谷歌将获得超过 17，000 项专利，成为领先的移动手机制造商，并在摩托罗拉的机顶盒业务中占据更强的地位。

不利的一面是，马哈尼担心这会对整个 Android 生态系统产生什么影响。Android 在全球已经拥有 1.5 亿部手机，发展势头如此强劲，为什么要冒险通过吸收一家领先的 Android 制造商来减缓这一势头呢？制造硬件“远离谷歌的核心竞争力”监管部门的批准至少要等到今年年底，如果不是更久的话，这笔交易才有可能完成。

那么，谷歌为什么要经历这么多麻烦呢？“专利可能是这笔交易的关键，”马哈尼写道，“但它们也暗示了这笔交易的防御性质。”嗯，当其他主要科技公司似乎都在与 Android[对抗时，你真的不能责怪谷歌在专利上的防守。](https://web.archive.org/web/20230205003612/https://techcrunch.com/2011/08/03/google-rips-android-competitors-over-patents/)
