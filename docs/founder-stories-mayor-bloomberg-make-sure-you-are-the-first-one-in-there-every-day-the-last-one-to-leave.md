# 市长彭博:“确保你每天第一个到，最后一个离开。”

在主持人[克里斯·迪克森](https://web.archive.org/web/20221006072752/http://www.crunchbase.com/person/chris-dixon)、[市长彭博](https://web.archive.org/web/20221006072752/http://www.crunchbase.com/person/michael-bloomberg)的最后一次*创始人故事*采访中，他首先向那些纠结于职业选择的人提供建议——成为一名企业家还是加入一家更大的公司。

尝遍了两个世界，市长彭博指出“大公司有更多的结构，但是你有更多的安全感。小公司，结构少得多，但回报高得多。”尽管他认为从一家大公司过渡到一家小公司比反之更容易，但他指出，“如果你有愿景，想要(自己创业)，就去做吧。”然而，他诚实地承认，“如果我没有被解雇，我不知道我是否有勇气去做这件事。”

根据第一手经验，市长抛出了另一个选项——公共服务。与私营部门类似，他认为“这是一项艰苦的工作，是愿意坚持自己的信念”，但也认识到“回报不是金钱，而是满足感。”

他在这三个领域的成功出现在《不夜城》中似乎是恰当的。

“我不比任何人聪明，但我比你更聪明——对你或任何人来说，我成功的关键是确保你每天第一个到办公室，最后一个离开。不要午休或上厕所，你要继续工作。你永远不知道这个机会什么时候会到来。”他继续说道，“我所拥有的每一个机会，我认为很多都是因为我当时在场。这是你能控制的一件事。你不能控制你的运气，但是你越努力，你就越幸运。”

同样重要的是，“看到弯曲面的动力”他补充道，“从来没有一天我不期待去工作，即使是我知道我会被痛打的那一天，即使是我知道我会被解雇的那一天——我以前从来没有被解雇过，我想知道那是什么样的，好吧，让我们去找出答案。”
