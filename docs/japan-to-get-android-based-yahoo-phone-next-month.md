# 日本下月将推出基于安卓系统的“雅虎手机”

日本获得基于安卓系统的[“雅虎手机”](https://web.archive.org/web/20230204234152/http://www.softbankmobile.co.jp/ja/news/press/2011/20110818_05/)【JP】:考虑到[雅虎是日本最大的网站](https://web.archive.org/web/20230204234152/https://techcrunch.com/2008/08/23/3-reasons-why-the-internet-in-japan-is-ruled-by-one-single-company-yahoo/)，起初听起来可能很奇怪的事情实际上并不令人惊讶。此外，日本第三大移动运营商[软银](https://web.archive.org/web/20230204234152/https://techcrunch.com/tag/softbank/)，手机的供应商，是雅虎日本的最大股东，这样一款手机的出现更有意义。

这款手机由夏普制造，是该公司几周前展示的所谓[“AQUOS PHONE THE PREMIUM SoftBank 009 sh”](https://web.archive.org/web/20230204234152/http://mb.softbank.jp/mb/smartphone/product/009sh/)[JP]的更名版。它采用 Android 2.3 作为操作系统，具有 QHD 分辨率的 4 英寸液晶显示器，800 万像素的 CMOS 摄像头，蓝牙 3.0，IEEE 802.11 b/g/n Wi-Fi，microSDHC 插槽，数字电视调谐器，电子钱包功能，红外通信等。

买家可以在手机上找到预装的雅虎日本精选安卓应用和服务，例如雅虎日本主页、雅虎日本拍卖、日历、字典、雅虎本地定位(该公司的 LBS)等。

选择这款手机的软银用户可以免费成为雅虎日本高级会员两年(而不是每月支付 4.5 美元)，并且在接下来的两年里，在雅虎购物上每购买一次还可以获得额外的积分。

软银将于下月下旬开始在日本销售雅虎手机。
