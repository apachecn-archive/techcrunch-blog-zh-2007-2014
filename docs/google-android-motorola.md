# 谷歌对摩托罗拉的收购让安卓变得无所不包

今天早上，谷歌出人意料地出价 125 亿美元收购摩托罗拉移动公司，这是一次大胆的尝试，试图在几个方面改变谷歌的处境。谷歌正在全力以赴，动用其 390 亿美元现金进行有史以来最大的收购。这笔交易表明，移动业务将是谷歌的主要增长动力之一，而增长将来自进入新市场，特别是移动硬件。通过收购，谷歌获得了 17，000 项专利和另外 7，000 项正在全球申请的专利，在这一领域，谷歌是目前落后的。但最重要的是，它表明了谷歌从头至尾控制 Android 体验的重要性。

现在可能有[1.5 亿部安卓手机](https://web.archive.org/web/20230314190952/http://googleblog.blogspot.com/2011/08/supercharging-android-google-to-acquire.html)，它们占据了[最大的智能手机市场份额](https://web.archive.org/web/20230314190952/https://techcrunch.com/2011/08/11/gartner-android-os-sales-top-ios-rim-and-nokia/)，但是迄今为止最受欢迎的单一智能手机仍然是 iPhone。(在所有智能手机设备中，iPhone 的市场份额最大，而 Android 操作系统在所有设备中的份额更大)。收购摩托罗拉是谷歌方面的一个承认，如果它希望取代苹果，它必须控制从软件到硬件的体验。

到目前为止，谷歌已经在移动领域采用了 Windows 的方法——创建一个所有制造商都可以在其上构建并专注于软件的操作系统——但这种方法是不够的。39 家制造商每天有超过 55 万部 Android 手机被激活，但没有一家手机或 Android 制造商能胜过 iPhone。通过拥有摩托罗拉，谷歌可以按照自己的确切规格制造安卓手机，并利用操作系统的最新进展，就像苹果一样。当首席执行官拉里·佩奇说收购摩托罗拉将“增强”安卓系统时，我怀疑他就是这个意思。

佩奇小心翼翼地指出，“安卓将保持开放”，摩托罗拉将只是众多被许可方之一。但毫无疑问，如果该交易通过反垄断审查，摩托罗拉的安卓手机将成为同类手机中的佼佼者。谷歌还凭借摩托罗拉的机顶盒业务在客厅获得了强大的立足点，这可能有助于其黯淡的谷歌电视努力

赌注很大。谷歌搞砸了怎么办？好吧，它仍将拥有所有这些移动专利，可以说，这些专利可能占总收购价格的很大一部分。记住，北电的专利以 45 亿美元的价格卖给了谷歌的竞争对手。在今天解释这笔交易的电话会议上，佩奇指出，摩托罗拉“强大的专利组合”将有助于谷歌抵御“微软、苹果和其他公司”对安卓的攻击电话中的前两个问题也直接指向了专利问题。随着安卓系统在专利战线上受到、苹果、微软、甲骨文和其他公司的攻击，收购摩托罗拉也是一个防御性的举动。
