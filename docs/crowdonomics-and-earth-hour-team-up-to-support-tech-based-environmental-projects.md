# 众筹和地球一小时联手支持环保项目

总部位于新加坡的众筹平台 Crowdonomic 已经与非营利组织[地球一小时](https://web.archive.org/web/20221006215301/http://www.earthhour.org/)合作，帮助资助基于技术的环境项目。

名为[地球一小时蓝色](https://web.archive.org/web/20221006215301/https://earthhourblue.crowdonomic.com/)的新平台允许用户向世界野生动物基金会支持的项目捐款。寻求资金的项目目前包括[菲律宾的 Bancas](https://web.archive.org/web/20221006215301/https://earthhourblue.crowdonomic.com/project/1015-bancas-for-the-philippines)，该项目将向在超强台风“海燕”中失去捕鱼设备的社区介绍玻璃纤维船；[停止杀戮！](https://web.archive.org/web/20221006215301/https://earthhourblue.crowdonomic.com/project/1071-stop-the-killing-1)，努力打击东南亚野生动物走私；恢复罗德伯格公园，该公园正在为保护南非的桌山国家公园筹集资金。

Crowdonomic 大约在一年前推出，为亚洲的项目提供基于奖励或股权的众筹。该公司希望与地球一小时的合作将有助于提升其在新加坡以外的知名度，并引领与世界各地更多创业公司和企业的合作。

“这个项目展示了我们新的[企业众筹解决方案](https://web.archive.org/web/20221006215301/http://www.crowdonomic.com/enterprise)，它允许大型机构利用 Crowdonomic 的专业知识和基础设施，在全球范围内利用众筹的力量，”联合创始人 Leo Shimada 和 Nicola Castelnuovo 在一封电子邮件中说。

对于同样位于新加坡的“地球一小时”来说,“蓝色地球一小时”让非营利组织有机会支持世界各地的更多倡议。“地球一小时”称其项目已经覆盖了 154 个国家的 7000 个城市。
