# 到达让人们知道你何时到达

“当你在纽约的时候给我发短信，”你告诉你的朋友，但是你忘了发短信，接下来你知道的是你在 JFK 的出境航班上，你甚至没有见过你上学的人或交往的人。

或者让我们说，你想和你的闺蜜去布鲁明戴尔，但有一堆事情要做，没有时间紧迫，所以你可以随时去。[抵达](https://web.archive.org/web/20230204192441/http://www.getarrived.com/)试图解决这个问题。

创始人 [Clarence Wooten Jr.](https://web.archive.org/web/20230204192441/http://www.crunchbase.com/person/clarence-wooten) 认为有一个更好的解决方案，当人们在普通或超本地区域时，给人们一个提醒，使人们能够制定“软计划”

使用已到达的应用程序，您可以选择一个地方，并添加当您到达特定位置时要通知的人。被选中的人可以选择接收通知，也可以在到达特定地点时通知您。通过结合地理围栏、自动签到和通知，Arrived 希望在不要求消费者改变任何行为模式或签到的情况下为他们提供价值。

在包括 Shervin Pishevar、Rich Skrenka 和 Eghosa Omoigui 在内的顾问名单中，Arrived 计划通过“抵达奖励”来货币化(it better)，商家将跟踪消费者抵达，以换取获得奖励的可能性。
