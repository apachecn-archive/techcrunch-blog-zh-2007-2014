# 未验证？这是我坐的地方的推特视图

所以你在 Twitter 上没有被验证。别担心。现在还有时间——不像国会为政府提供资金。但是我跑题了。

无论如何，你可能不知道的是，如果你*通过了*的验证，你就可以在 Twitter 网络界面的交互标签中使用一个简洁的过滤选项。它允许您只看到来自其他已验证用户的与您的帐户的交互。这基本上是一种只在酷孩子俱乐部里闲逛的方式。而且很棒。
