# 社交问题:Quora 获得了一款非官方的 iPhone 应用

![](img/648d1577af2b738156d74a12dd9c5f6f.png)

社交问答初创公司 [Quora](https://web.archive.org/web/20230203003506/http://www.crunchbase.com/company/quora) 最近[聘请了](https://web.archive.org/web/20230203003506/http://www.quora.com/Adam-DAngelo/Quora-is-looking-to-hire-an-iPhone-iOS-developer)一名 iOS 开发者，所以我们很有可能很快就会看到正式的 iPhone(和 iPad？)应用程序不断涌现，即使其网站在智能手机上已经运行良好。

在你等待官方应用的时候，一定要看看[社交问题](https://web.archive.org/web/20230203003506/http://itunes.apple.com/WebObjects/MZStore.woa/wa/viewSoftware?id=415504188&mt=8&ign-mpt=uo%3D6)。由 Thomas Ricouard 开发的这款应用可以让你访问和浏览 Quora 移动网站，它有几个方便的附加快捷方式和更快的导航。

您可以使用该应用程序访问您的个人资料的主页、收件箱、通知，并且可以轻松添加新问题。

根据 Quora 上的这个帖子, Ricouard 请求并获得了 Quora 的许可，在 App Store 上发布 iPhone 应用程序，只要他不说出它的名字，这样人们就会将其与官方产品混淆。因此得名“社会问题”。

据我所知，Quora 也有两个非官方的 Android 应用程序。
