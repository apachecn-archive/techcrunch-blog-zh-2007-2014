# Zendesk 为 iPad 带来了帮助台软件

客户支持初创公司 [Zendesk](https://web.archive.org/web/20230203013057/http://www.zendesk.com/) 今天推出了一款 iPad 应用，支持代理可以在旅途中管理公司的服务台。

Zendesk 于 2008 年推出，它提供了一个基于网络、由 SaaS 交付的帮助台/支持票务应用程序，为公司提供了一种简单的方法来管理来自最终客户的支持请求。Zendesk 每天增加大约 20 名新客户，最近达到了使用客户服务平台的[5000 家企业](https://web.archive.org/web/20230203013057/http://www.techcrunchit.com/2010/05/18/zendesks-saas-help-desk-system-now-being-used-by-over-5000-businesses/)的里程碑。

Zendesk for iPad 加入了 iPhone、黑莓和 Android 的现有应用程序，这些应用程序的总下载量已超过 80，000 次。iPad 应用程序包含许多与网络和移动应用程序相同的功能，但也包括一些独有的功能。用户可以查看已分配票证的实时列表，支持代理也可以将票证标记为书签，以便稍后对其做出响应，或者将其用于将来参考。

该公司的首席运营官·扎克·乌尔洛克表示，随着越来越多的公司采用 iPad，他预计 iPad 应用将成为该公司最受欢迎的移动应用。

Zendesk 总共筹集了 2550 万美元。
