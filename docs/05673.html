Twitter对每个用户都有一个(秘密的)信誉评分

哇哦。在 Web 2.0峰会的舞台上，当被采访者约翰·巴特利问及“你如何挑选追随时，推特创始人伊万·威廉姆斯透露？，“Twitter对每个用户都有一个私人信誉评分。

根据威廉姆斯的说法，Twitter的“科学和数学人”有一个系统来衡量你关注谁和你关注谁的人，并试图在重叠部分找到“关注谁”的相关性。他没有说清楚个人用户信誉得分是如何衡量的。

“我们可能会把它公之于众，”他在谈及配乐时说，“但它必须进一步发展。”
