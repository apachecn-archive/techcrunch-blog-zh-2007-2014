# 黑客因 iPad 数据泄露面临刑事指控

![](img/4bba249043c1e6d5fb842acc4c14ce05.png "iPad")  [弗利克](https://web.archive.org/web/20221007005518/http://www.flickr.com/photos/yagankiely/4647065335/)

还记得几个月前发生的大规模 [iPad 数据泄露事件](https://web.archive.org/web/20221007005518/http://www.crunchgear.com/2010/06/09/att-security-breach-leaks-thousands-of-ipad-owners-emails-but-luckily-nothing-more/)吗？很好，因为事实证明有人提出了刑事指控。

新泽西州的地方检察官保罗·菲什曼对两名男子提出了一项欺诈指控和一项未经授权密谋进入电脑的指控。这两个人是丹尼尔·斯普利特和安德鲁·奥恩海默。

数据泄露暴露了大约 12 万名 iPad 用户的电子邮件地址。据称，Splitter 和 Auernheimer 早在 6 月份就侵入了美国电话电报公司的服务器，暴露了这个国家一些最优秀和最聪明的人的数据:政治家、商人、名人等等。

没有任何过于敏感的信息被窃取，只有姓名和相应的电子邮件地址。
