# 专为智能手机设计的迷你燃料电池

[![](img/17ee0b72f95d6b2ff642d20c8c000ec4.png "aquafairy0")](https://web.archive.org/web/20221207085801/http://www.crunchgear.com/2011/02/23/mini-fuel-cell-specifically-designed-for-smartphones/aquafairy0/)

[燃料电池](https://web.archive.org/web/20221207085801/http://search.beta.techcrunch.com/query.php?y=%2Ftc_eng_id%2Fsearch%2Fv1%2Fquery%2Ffuel%2520cell%3Fcategory_id%3DTechCrunch%2520Search%26sort%3Ddate%26client%3Dtechcrunch)仍在成为主流，但我们正在慢慢实现。一个很好的例子:AF-M3000，一种迷你燃料电池，只需要用户加水就能工作。有趣的是，位于京都的 [Aquafairy](https://web.archive.org/web/20221207085801/http://www.aquafairy.co.jp/en/index.html) 已经开发了专门用于智能手机的设备。

根据制造商的说法，这种电池产生 3W 的能量，需要 90 分钟才能给一部 iPhone 充电(这足以让手机使用大约三个小时)。为了让这个装置更小更轻(128 克)，Aquafairy 使用了氢氧化钙。用户将水加入水合物中，产生氢气，然后氢气与氧气反应发电。

[![](img/479452582286f520ca5af3d35e45c4c3.png "Picture 1")](https://web.archive.org/web/20221207085801/http://www.crunchgear.com/2011/02/23/mini-fuel-cell-specifically-designed-for-smartphones/picture-1-67/)

Aquafairy 计划下个月在日本举行的 FC 博览会上展示这种燃料电池。从四月份开始，它将以 320 美元的价格出售。
