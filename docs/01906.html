随着Adobe flash paper的停产，记录混乱中的初创企业

今天，随着Adobe停止开发其Flashpaper产品的消息传出，许多欧洲初创公司——以及全球许多其他公司——将陷入混乱。

Adobe将继续销售和支持当前的FlashPaper 2版本，但不会更新技术以支持Microsoft Windows Vista和IE7，这将使它几乎毫无价值。

这一消息将打击美国的网站，如 Scribd 和 Docstoc ，以及欧洲的网站，如英国的 edocr 和德国的 Twidox ，它们最近才获得资金。edocr目前所有的文档共享都基于Flashpaper。

Twidox首席执行官Nicholas MacGowan von Holstein今天联系了TechCrunch UK，称此举将产生重大影响:“那些用Flashpaper存储所有文件的网站怎么办？将所有这些文档转移到新的解决方案将是一项艰巨的任务。”他说，Twidox不会受到严重打击，因为他们已经计划了替代方案。他说，他是在“直接给Adobe写了三封信之后”才知道这个消息的。

Adobe的网站上说:

“经过对市场和客户反馈的仔细考虑和分析，Adobe计划停止为FlashPaper开发新功能。需求持续下降，Adobe继续为FlashPaper提供开发支持在经济上已不再可行。”
