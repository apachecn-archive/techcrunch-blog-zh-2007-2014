# 谷歌失去了欧洲首席运营官本莱格欧洲目录

![](img/62040df8aad3f8a644f21d9096b4a485.png) [欧洲目录](https://web.archive.org/web/20230320164534/http://www.europeandirectories.com/)今天早上宣布了一项重大胜利，任命[本·莱格](https://web.archive.org/web/20230320164534/http://www.linkedin.com/ppl/webprofile?action=vmi&id=10529792&pvs=pp&authToken=agmh&authType=name&trk=ppro_viewmore&lnk=vw_pprofile)为首席运营官。

Legg 从其主要竞争对手之一谷歌加入荷兰/英国公司，从今年 1 月起他就在欧洲首席运营官公司任职，并在此之前的两年中担任运营总监。

在荷兰、芬兰、丹麦、瑞典、奥地利、捷克共和国、斯洛伐克和波兰等八个欧洲市场提供本地搜索和铅生产产品的欧洲目录公司表示，Legg 将直接负责该公司三个国家的业务。

他还将领导未透露姓名的“全集团计划”，该计划旨在将传统的印刷目录业务转变为一项多媒体的潜在业务。

他将在位于奇斯威克公园的欧洲目录伦敦办事处工作。

欧洲目录公司表示，该公司目前为 70 多万中小企业客户提供了一系列解决方案，以满足他们在当地的广告需求。该公司目前总共雇用了 4700 名员工。

本月早些时候，该公司宣布了与 Skype 的战略合作伙伴关系，将为客户提供免费通话服务。在线广告商将有机会在他们的号码出现在互联网上的任何地方用 Skype 的蓝色“免费通话”按钮突出显示。
