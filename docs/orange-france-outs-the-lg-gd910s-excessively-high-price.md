# Orange France 击败了 LG GD910 过高的价格

哦，天啊，戴文会不高兴的。他迷恋 LG GD910，但我觉得这个价格对他来说有点太贵了。有传言说它会很贵，果不其然，Orange France 将在€以 899 英镑(1290 美元)的合同出售它。

那是很多，但是看看它。这是一个该死的手表手机，有前置摄像头，蓝牙，甚至 HSDPA 连接。绝大多数“智能手机”都不具备这三个特征。

LG 尚未宣布在美国销售这款手表手机的计划。我敢打赌，这一代手机将不会在这里正式上市，尽管进口商可能已经梦想着他们可以逃脱的疯狂利润。也许当我们的经济稍微好一点，手表手机的价格稍微低一点的时候，我们会得到下一代。
