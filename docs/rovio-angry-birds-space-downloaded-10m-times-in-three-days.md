# Rovio:愤怒的小鸟空间 3 天下载 1000 万次

强大的老鹰着陆了！上周晚些时候发布的最新版 Rovio 的《愤怒的小鸟》在头三天就被下载了 1000 万次。不幸的是，由于公告是通过推特发布的，它没有包括关于平台渗透率或为额外级别支付 0.99 美元的人数的详细数据(我有)。但是三天 1000 万！

[推特 https://twitter.com/AngryBirds/status/184235948324814848]

TechCrunch 的 Jordan Crook 上周与 Rovio 的 Peter Vesterbacka [就这款游戏进行了对话。他表示，这款游戏是该公司迄今为止最重要的产品。该公司无疑对结果感到高兴，并希望至少有一些粉丝用一个昂贵的愤怒的小鸟太空毛绒玩具来庆祝游戏。](https://web.archive.org/web/20221207003211/https://beta.techcrunch.com/2012/03/15/peter-vesterbacka-angry-birds-space-demo/)

不过，Rovio 在达到 1000 万次下载的过程中碰巧遇到了一点争议。据报道，维斯特巴卡说，Rovio 不打算为 Windows Phone 制作一个版本的《愤怒的小鸟》( Angry Birds Space 这是一个夸张的说法，甚至导致一位所谓的分析师对第三方 WinPhone 供应商诺基亚持悲观态度。Rovio 的首席执行官[随后更正了声明](https://web.archive.org/web/20221207003211/https://beta.techcrunch.com/2012/03/23/analyst-no-angry-birds-space-on-windows-phone-will-cause-others-to-think-twice-about-nokias-recovery/)，称 Windows Phone 版本即将推出。

Rovio 不会像对待第一款游戏那样，依赖《愤怒的小鸟》空间的成功。该公司计划在 2012 年发布另外四款愤怒的小鸟游戏。我们还不知道即将到来的游戏的细节，但是很有可能这个游戏会向绿猪投掷心烦意乱的小鸟。
