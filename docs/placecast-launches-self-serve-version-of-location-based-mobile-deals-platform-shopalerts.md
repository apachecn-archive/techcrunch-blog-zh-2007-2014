# Placecast 推出基于位置的移动交易平台 ShopAlerts 的自助版本

Placecast 的 [ShopAlerts 服务](https://web.archive.org/web/20230203055214/https://techcrunch.com/2010/02/25/placecast-debuts-location-based-mobile-marketing-technology-shopalerts/)通过手机向消费者提供零售商和商家的优惠和折扣，已经被许多知名品牌使用，如[、](https://web.archive.org/web/20230203055214/http://www.nytimes.com/2010/02/23/business/media/23adco.html?_r=1)和[。甚至在& T.](https://web.archive.org/web/20230203055214/https://techcrunch.com/2011/02/27/att-taps-placecast-for-location-based-shopping-alerts/) 今天，该公司推出了一个自助版本的 ShopAlerts，允许商家创建、提供和跟踪移动促销的兑现。

shop alerts Self-service 是一个白标地理围栏平台，面向希望通过手机向客户提供地理触发优惠的企业。消费者可以选择以多种方式接收短信——在商店、网上、通过短信、手机网站或脸书。地理围栏是围绕物理位置设置的虚拟区域；当选择加入的消费者在地理围栏区域内时，她会收到来自企业的消息，包括折扣、销售通知、商店方向或其他内容。

自助版本的 ShopAlerts 允许商家上传和管理商店位置和信息，创建地理围栏和促销，跟踪优惠券和兑换代码，并访问对参与者、地理围栏和 ROI 的分析。用户还可以实时更改消息。

shop alerts Self-service 实际上并不直接面向小型企业，而是面向大型垂直公司，如移动运营商、黄页提供商、零售商、金融机构或交易服务，这些公司可以与中小型企业合作，并为商家部署这一工具。

Placecast 表示，目前商店警报可以覆盖美国电话电报公司、斯普林特、威瑞森和 T-Mobile 网络中的 2.68 亿部手机。
