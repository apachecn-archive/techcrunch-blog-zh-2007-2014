# 牛肉在哪里？艺电推迟 OS X 发布

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/16/wheres-the-beef-ea-delays-os-x-titles/>

![](img/8acf4faa198467f84bfca7828896e259.png)

艺电和乔布斯在 WWDC 舞台上做爱几个月后，消费者还没有看到艺电承诺的任何游戏。像*Madden’08*和*战地 2142* 这样的游戏应该会出现在 OS X，但是还没有出现。现在事情越来越可疑了。显然艺电没有对 OS X 相关的游戏发表评论，苹果将所有的电话和询问都转给了艺电。

在所有的现实中，EA 可能需要几个月的时间将游戏移植到 OS X，所以我不担心。苹果公司告诉 journos 与 EA 对话的整个推诿有点奇怪，但希望一切都会及时得到澄清。
