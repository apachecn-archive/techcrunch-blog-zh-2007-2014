# DARPA 主任雷吉娜·杜根离开国防部加入谷歌

今天，谷歌带来了一些相当高调的招聘:国防部研究机构 DARPA 的主任[在领导该机构三年后即将离开，以“高级管理职位”加入谷歌](https://web.archive.org/web/20230204105433/http://www.wired.com/dangerroom/2012/03/dugan-darpa-google/)。

这一消息来自 DARPA 的一位发言人，他报告说，杜根觉得她无法拒绝这样一家“创新公司”的邀请。自 1996 年以来，她断断续续地在该机构工作，最近，她将该机构的预算和资源用于更实际的问题，如保护军事网络。她将在谷歌扮演什么角色还不得而知，但很可能至少部分与安全相关。

Dugan 最初是一名研究员和项目经理，2009 年晋升为主任。尽管她鼓励创新和一些所谓的“蓝天”项目(可能会停滞不前或永远见不到天日的研究)，但她也努力资助按照 DARPA 的标准来看更加脚踏实地的项目。她在工程的各个领域拥有[几项自己的](https://web.archive.org/web/20230204105433/http://patft1.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&u=%2Fnetahtml%2FPTO%2Fsearch-adv.htm&r=0&p=1&f=S&l=50&Query=in%2Fdugan-regina-e&d=PTXT)专利，并出版了一本关于工程热力学的书。

11 月，她呼吁黑客和安全专家考虑军方过时的数据保护方法的问题。她似乎已经直觉到，虽然[机器人猎豹](https://web.archive.org/web/20230204105433/https://techcrunch.com/2012/03/05/darpas-cheetah-robot-will-stab-you-with-its-pointy-legs/)、[角膜显示器](https://web.archive.org/web/20230204105433/https://techcrunch.com/2011/11/22/researchers-see-retina-display-raise-them-a-cornea-display/)和[蜂鸟无人机](https://web.archive.org/web/20230204105433/https://techcrunch.com/2011/06/01/darpa-video-shows-the-evolution-of-the-hummingbot-nano-uav/)都很好，但物理战场并不是唯一真正的战场。即使在现实世界的安全中，网络安全也越来越重要；美国国家航空航天局最近表示，它已经被黑客攻击了数千次，很可能将有价值的敏感数据泄露给了中国黑客。

杜根是一项调查的对象(被描述为许多“定期审计”中的一项，但仍然是一项调查)，该调查与授予她是联合创始人的一家公司的合同有关，但 DARPA 发言人表示，谷歌的离职与之无关。

预计她将在接下来的几周内做出最后的调整，届时我们可能会有更多关于她在谷歌新角色的信息。
