# 摩托罗拉 Droid RAZR 在威瑞森接受预购

今天早上，Droid RAZR——摩托罗拉称之为世界上最薄的智能手机——已经可以从威瑞森预订了。这款手机仍然列在威瑞森的网站上，但它肯定会很快。所以，如果这是你一直在等待的，那么我们建议你赶快行动。

作为补充， [Droid RAZR](https://web.archive.org/web/20230204184000/https://techcrunch.com/2011/10/18/verizon-officially-unveils-the-motorola-droid-razr/) 厚 7.1 毫米，配有双核 1.2GHz 处理器和 Android 2.3 姜饼(即将推出冰淇淋三明治)。这款手机配备了 4.3 英寸 qHD Super AMOLED 960×540 显示屏，1GB 内存，800 万像素后置摄像头和 130 万像素前置摄像头。这款手机还支持威瑞森的 4G LTE 网络。

你现在可以从威瑞森以 299 美元的合同预购这款手机，如果你碰巧是美国电话电报公司的顾客，[同样的事情可能会比你想象的更快发生。](https://web.archive.org/web/20230204184000/https://techcrunch.com/2011/10/24/motorolas-droid-razr-or-something-like-it-likely-headed-for-att-soon/)
