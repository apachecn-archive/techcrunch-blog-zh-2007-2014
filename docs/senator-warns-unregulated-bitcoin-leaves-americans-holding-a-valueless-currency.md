# 参议员警告说，不受监管的比特币让美国人持有一种“毫无价值的货币”

参议员乔·曼钦(Joe Manchin)呼吁要么严格监管，要么彻底禁止有争议的数字货币比特币。他在给美联储的一封信中称，随着中国等国家禁止比特币，“美国人将为这种毫无价值的货币背黑锅”。

比特币是一种匿名的数字加密货币，已成为越来越受欢迎的在线购买产品和转移资金的方式，但它更容易购买黑市药物和武器。比特币的麻烦因价值大幅波动而加剧，也因著名的比特币交易所 Mt . Gox(T3)数百万比特币被盗而加剧。

美国财政部已经证实，它不会追查那些[开采比特币](https://web.archive.org/web/20221004082506/https://beta.techcrunch.com/2013/04/08/how-to-mine-bitcoins/)的人。然而，纽约官员已经宣布计划在不久的将来对其进行监管。

抛开任何此类监管的合法性不谈，当前的[国会是历史上最没有成效的国会之一，](https://web.archive.org/web/20221004082506/http://www.huffingtonpost.com/2013/07/08/113th-congress-bills_n_3563008.html)所以他们不太可能聚集必要的支持来解决一个低优先级的问题。

一位发言人向《福布斯》证实，参议员[并不确定](https://web.archive.org/web/20221004082506/http://www.forbes.com/sites/andygreenberg/2014/02/26/senator-calls-for-bitcoin-ban-in-letter-to-financial-regulators/)他将如何监管比特币。我们联系了他的办公室，看看他是否真的在呼吁彻底禁止或者只是严格监管。
