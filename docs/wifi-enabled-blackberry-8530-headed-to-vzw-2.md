# 支持 WiFi 的黑莓 8530 前往 VZW

[![blackberry-8530-aries](img/039d990d872bdb4c928d71350e400d40.png "blackberry-8530-aries")](https://web.archive.org/web/20221007161317/https://beta.techcrunch.com/wp-content/uploads/2009/07/blackberry-8530-aries.jpg) 
但愿[【威瑞森】](https://web.archive.org/web/20221007161317/http://www.crunchgear.com/tag/verizon/)已经把它调好了 WiFi。一般来说，运营商不允许出售带有威瑞森标志的 WiFi 设备——有几个小的 Winmo 例外——但似乎黑莓 8530 [将支持 802.11 b/g](https://web.archive.org/web/20221007161317/http://www.boygeniusreport.com/2009/07/29/blackberry-8530-headed-to-verizon-totes-wi-fi/)。不仅如此，它可能是第一款搭载 OS 5.0 的威瑞森黑莓手机。价格或可用性信息尚未公布，但一旦这些信息进入我们的提示框或 RSS 订阅，我们会让你知道。
