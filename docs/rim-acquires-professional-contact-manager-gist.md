# 动态研究获得专业联络经理要点

![](img/b6e681f27138d7f9cac5c3999b785e02.png)我们听到[谣言](https://web.archive.org/web/20230202235259/http://gigaom.com/2010/12/17/rim-to-buy-gist/)称 RIM 公司参与了购买职业联络经理 [Gist、](https://web.archive.org/web/20230202235259/http://gist.com/)的严肃谈判，今天这是官方消息。RIM 已经[收购了](https://web.archive.org/web/20230202235259/http://blog.gist.com/2011/02/14/gist-is-now-part-of-research-in-motion-rim/) Gist，我们已经和公司确认了。交易的财务条款没有披露。

以下是来自 Gist 的声明:*今天，我们宣布 Gist 已经成为运动研究(RIM)的一部分。我们对 RIM 的未来以及全球数百万黑莓用户将如何使用 Gist 感到非常兴奋。这是朝着我们的目标迈出的一大步，我们的目标是利用基于网络的 Gist 体验，让用户建立更强大的专业关系。请继续关注更多细节，感谢你们所有人的支持和鼓励，让我们在 Gist 进入下一个伟大的阶段。RIM 也在其[黑莓](https://web.archive.org/web/20230202235259/http://blogs.blackberry.com/2011/02/rim-welcomes-gist/)博客上证实了此次收购。*

Gist 于 2008 年作为一个“专业”的沟通管理公司成立，旨在满足商业用户的过滤需求，筹集了 1080 万美元的资金。像网络服务一样，该应用程序根据你的电子邮件联系人组织你的信息流，因此这是一种跟踪你的同事、朋友和专业联系人的有用方式。

总部位于西雅图的 Gist 提供的服务可以帮助你密切关注你的职业网络中的人，并允许你查看每个联系人过去的消息和附件，关于他们公司的新闻，以及他们在 Twitter 等服务上的最新消息。该公司还提供许多移动应用程序，看起来 RIM 将为黑莓用户使用这项技术。

Gist 由 T.A. McCann 创建，他是一名企业家，也是微软 Exchange Server Group 的前高级员工，最初由微软联合创始人保罗·艾伦的 Vulcan Capital 提供资金。去年，Gist [收购了 Learn That Name，](https://web.archive.org/web/20230202235259/https://techcrunch.com/2010/03/17/gist-acquires-startup-weekend-app-learn-that-name/)这款游戏利用你在 LinkedIn 上的联系人来帮助你记住商业上熟人的名字，还为 Gmail 推出了一款[小工具。](https://web.archive.org/web/20230202235259/http://social.venturebeat.com/2010/05/18/gist-gmail-contextual-gadgets/)

正如奥姆·马利克在去年 12 月写的那样，RIM 利用 Gist 将更多的社交元素融入黑莓电话簿，并利用[将更多的社交元素融入电子邮件](https://web.archive.org/web/20230202235259/http://www.techcrunchit.com/2010/04/15/gist-launches-a-xobni-like-plugin-for-lotus-notes/)，这是有道理的。Gist 表示，用户可以继续使用其平台，但看看 RIM 是否会关闭这项服务，转而专门为黑莓用户提供这项服务，这将是一件有趣的事情。我可以想象，如果是这样的话，许多 Gist 用户将会非常失望。
