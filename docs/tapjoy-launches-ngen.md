# Tapjoy 表示，其新的 nGen 平台可以在正确的时间投放正确的移动广告

移动广告公司 Tapjoy 今天发布了一个名为 nGen 的新平台，首席执行官史蒂夫·沃兹沃斯告诉我，这反映了公司更广泛的转型。

“从很多方面来说，这是一家新公司，”沃兹沃斯说，他于 2012 年底接任[。“我们一直在重建高级管理团队……我们做了很多事情来重建公司，以推出这款产品。这一切汇集在一起，为发行商提供了一个更加全面的应用货币化解决方案。”](https://web.archive.org/web/20221005195245/http://gigaom.com/2012/11/27/tapjoy-replaces-ceo-with-former-disney-executive-steve-wadsworth/)

显然，Tapjoy 以(和批评)闻名的游戏内服务仍将是产品的一部分，但 nGen 是一个更广泛的移动广告平台。该公司表示，目标是让出版商能够“在正确的时间向正确的用户展示正确的广告。”

因此 nGen 可以发送各种不同的广告，从全屏插播广告到出版商的促销信息。这些广告与应用程序中的事件相关联，可以针对特定的用户群，也可以进行测试和动态更新。

例如，游戏开发商 [Concrete Software](https://web.archive.org/web/20221005195245/http://www.concretesoftware.com/web/) 的首席执行官基思·皮切尔曼(Keith Pichelman)说，他的团队只在他们的 PBA 保龄球挑战赛中加入了“一些广告”，因为“我们不想吓跑用户。”他说，困难之一是，如果 Concrete 想要测试或调整广告，工程师必须编写代码并更新应用程序。

另一方面，有了 nGen，你可以在没有新代码的情况下更新广告，所以 Pichelman 说他对尝试不同的选项和想法感到很舒服(你可以在上面 PBA 应用程序的截图中看到一些广告)，最终导致每日用户的平均收入增加了 130%。

首席产品官 Jeff Drobick(去年[加入 Tapjoy](https://web.archive.org/web/20221005195245/http://www.insidesocialgames.com/2013/03/20/tapjoy-hires-jeff-drobick-as-cpo/))表示，nGen 的目标之一是创造足够广泛的东西，让开发者不再需要集成“太多的 SDK 来获得整个解决方案堆栈”，而是将 tap joy 视为“可以做更多事情的单一合作伙伴”他还表示，这将有助于 Tapjoy 继续向非游戏应用扩展。

Tapjoy 说，nGen 在去年第三季度进入 alpha 测试，现在正进入全面可用阶段。
