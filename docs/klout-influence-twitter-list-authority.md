# Klout 带来了影响力:衡量和修改 Twitter 上的列表权威

![](img/a1274e9eafa98e17df59b009e58bb8eb.png)

总部位于旧金山的 Klout 对衡量社交网络的影响力并不陌生。该公司在 SXSW Interactive 2009 上发布，旨在帮助 Twitter 用户发现全世界在听的声音(无论如何是在 Twitter 上)。本质上，Klout 在主题层面衡量影响力，对讨论特定主题时在各自社交图中表现出推动行动能力的个人进行排序。

今天，Klout 宣布了迄今为止最重要的版本。除了衡量 Twitter 上的[权威](https://web.archive.org/web/20230331133827/https://techcrunch.com/2009/03/10/are-blogs-losing-their-authority-to-the-statusphere/)之外，该公司还发布了一个新的智能 Twitter 列表引擎，可以识别和排名任何感兴趣话题的前 25 名影响者，并产生一个新的合格排名列表。

衡量个人在社交媒体中的影响力既令人垂涎，又难以捉摸。虽然许多工具声称计算权威，但需要澄清影响的定义，以便掌握现有工具和服务的相关性和差异。

为了让这个讨论保持在正确的轨道上，让我们定义一下影响力。根据韦氏词典的观点，影响力是指产生某种效果的力量或能力。

在正式[发布 List](https://web.archive.org/web/20230331133827/https://techcrunch.com/2009/10/30/check-it-twice-twitter-lists-now-open-to-all-users/)之前，Twitter 向 Klout 提供了对其 List APIs 的早期访问。因此，Klout 能够立即解决列表的一个早期问题——修改别人创建的列表而不必重新创建它的能力。新系统现在增加了将任何列表引入 Klout 的分析引擎的手段，并立即查看每个列表上每个人的分析。然后，用户可以按原样遵循该列表，也可以使用 Klout 的主题搜索和排名功能对其进行定制，以确保该列表包含所有必要的权威机构。

Klout 为一个人的影响力添加背景的能力对于那些希望利用影响者传播公司或产品的企业来说至关重要。

![](img/4e66098f6165db0546a5ee2aba3a2501.png)

Klout 的算法涉及语义计算的三个复杂阶段。第一种被称为“真实接触”，衡量每种关系之间的影响，评估参与观众与总观众的比例。

该服务还采用了一个被称为“放大概率”的二级指标，该指标记录了特定推文通过转发传播到主要网络之外的可能性，以及产生直接受众回应的可能性。然后，该服务会检查特定受众实际提及或转发的百分比，以及由用户操作生成的原始消息的百分比。

最后，Klout 考虑了“网络价值”，衡量那些追随原始用户的个人的影响力以及他们分享内容的倾向，这有助于第一个人的整体权威。

总的来说，Klout 的分数非常重视一个人通过他们在 Twitter 上分享的链接进行点击的次数，从而真正衡量与每次更新相关的行为。

Twitter 列出的影响不仅仅是影响力，还为新一层的权威创造了机会。不管是否有人直接激发了可衡量的行动，创建高度准确和有意义的列表是有意义的，也有助于 Twitter 社区。识别和汇集相关主题的权威人士的能力使追随者能够与可核实和可信赖的来源以及反过来影响其活动的声音、信息和趋势保持联系。更重要的是，Klout 使追随者能够进一步修改列表，不仅可以跟踪活动，还可以确定未来参与和行为跟踪的机会。

毕竟，影响力不在于旁观者的眼中，而在于激励行动并衡量其效果的能力。
