# 索尼全力以赴打造半透明相机镜

索尼今天在西班牙发布了一个令人惊讶的公告，称其阿尔法系列[数码单反](https://web.archive.org/web/20221202091731/http://www.crunchgear.com/tag/Photography/)的所有未来型号都将在[配备分束半透明反射镜](https://web.archive.org/web/20221202091731/http://translate.google.com/translate?js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&sl=es&tl=en&u=http://www.quesabesde.com/noticias/sony-alpha-slt-espejo-translucido,1_7341)。这是大多数 DSLRs 中标准全反射反射镜的替代物，该反射镜会向外摆动以暴露传感器。相反，大部分光线通过镜子，不需要等待反射模块离开，就可以被捕捉到。

a55 和 a33 使用了这项技术，由于明亮的电子取景器而不是光学棱镜取景器，他们避免了以前阻止半透明镜流行的问题。索尼分离出的光线仅够相位检测自动对焦系统使用，而不足以通过取景器实际观看。这是一个勇敢的举动，似乎得到了回报。a33 和 a55 得到了相当不错的评价，但这样一个大胆的尝试没有完全失败的事实是重要的一点。

下一款采用“SLT”格式(单镜头半透明)的相机将是 a77，它是佳能 60D 和尼康 D7000 的竞争对手。它将有 1080p 视频，2400 万像素，高 ISO 能力，等等。祝它好运。
