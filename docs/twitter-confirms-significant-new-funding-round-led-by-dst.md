# Twitter 确认由 DST 牵头的“重大”新一轮融资

经过数周的猜测，Twitter 刚刚发布了一篇博客文章，证实它确实筹集了“由风险投资公司 DST Global 牵头的一轮重大融资，我们的几名现有投资者也参与了。”

在今天的官方声明之前，上个月有几份报告。7 月初,《华尔街日报》[报道称【Twitter 正寻求以 70 亿美元的估值进行新一轮融资。一天后，纽约时报](https://web.archive.org/web/20230203045726/http://online.wsj.com/article/SB10001424052702304803104576428020830361278.html)[报道](https://web.archive.org/web/20230203045726/http://dealbook.nytimes.com/2011/07/07/investment-values-twitter-at-8-billion/)该公司正在完成一轮 4 亿美元的融资，对该公司的估值为 80 亿美元，由 DST 牵头，现有投资者 Kleiner Perkins 参与。

AllThings D 在 7 月 20 日发布的一份报告补充道，这轮融资实际上将总计 8 亿美元，其中一半将分配给员工和投资者。我们从另一个消息来源独立地听到，事实确实如此。

这已经成为了 DST 的一项专长——该公司持有脸书、Zynga 和 Groupon 的大量股份，允许员工套现，减少了每家公司上市的内部压力。

下面是 DST 首席尤里·米尔纳的一段话:

> “Twitter 是少数几家真正改变了世界的公司之一，”DST Global 的创始人尤里·米尔纳说 Twitter 的惊人增长证明了它每天对越来越多的人变得多么重要，以及为什么我们不能错过成为其未来更大一部分的机会。"

Twitter 也刚刚宣布在纽约开设了一个办事处。
