# Kik 为什么爱 Windows Phone 7(剧透:它促进了 Android 和 iOS 用户增长)

**独家—**说说一个光环效应。跨平台移动通讯应用开发商 [Kik](https://web.archive.org/web/20230205002634/http://www.crunchbase.com/company/kik-interactive) 表示，去年 6 月[推出](https://web.archive.org/web/20230205002634/http://www.kik.com/blog/2011/06/windows-phone-7-is-here/)Windows Phone 7 应用时，发生了非常奇怪的事情:iOS 和 Android 平台的用户增长都显著加快。

事实证明，Windows Phone 7 Kik Messenger 应用程序的每个注册用户都“值得”拥有 2.53 个用户(参见下面的方法解释和随附图片)。

Windows Phone 的市场份额可能很低(根据 Millennial Media 的数据，仅占美国市场的 2%左右——[PDF](https://web.archive.org/web/20230205002634/http://www.millennialmedia.com/wp-content/images/mobilemix/MM-MobileMix-June2011.pdf)),但对于有风险投资支持的[Kik 来说，它绝对值得在这个平台上推出。据该公司称，在 WP7 应用发布后，Android 和 iOS 用户增长实际上比预期高 22%。](https://web.archive.org/web/20230205002634/https://techcrunch.com/2011/03/07/mobile-messaging-startup-kik-raises-8m-launches-group-chat-and-photo-sharing/)

Kik 得出这些数字的方法是，在 2011 年 6 月 28 日 Windows Phone 7 Kik Messenger 发布之前的五周内，跟踪 iOS 和 Android 上的平均用户增长，并随后将这一趋势与发布后的一个月进行比较。

该公司发现，除了 Android 和 iOS 的预期平均增长之外，这些平台还因为发布而获得了 22%的额外增长。

每位 Windows Phone 用户带来了额外的 1.53 名 iOS 或 Android 用户，总共获得了 2.53 名注册 Kik 用户。

Kik Interactive 首席执行官 Ted Livingston 总结道:

> “Windows Phone 用户可能只是整体中的一小部分，但他们带来了其他移动平台上的所有朋友，这使得 Kik 活动出现了巨大而意外的峰值。
> 
> 跨平台应用是时候开始关注 Windows Phone 7 了。"
