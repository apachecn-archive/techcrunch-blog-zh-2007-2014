# Tucows 正式推出 Ting，这是一款更具创意的无线运营商 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/02/tucows-officially-launches-ting-a-more-thoughtful-wireless-carrier/>

Tucows 最出名的可能是他们的一系列网络服务和广泛的经销商网络，但首席执行官埃利奥特·诺斯看到了另一个领域的增长空间:移动。在花了几个月的时间为数百名用户进行私人测试后，Tucows 正式向所有用户开放了他们的 [Ting 无线服务](https://web.archive.org/web/20221225015850/https://ting.com/)。目标？为无线用户提供“一种完全不同类型的运营商关系”

诺斯告诉我:“知名航空公司提供的服务是为了最大化他们的利润，而不是他们对客户的服务。”。

这种以客户为中心的愿景从 Ting 的语音计划选择(有六个，分钟时段从 0 到 3，000 不等)延伸到客户服务(没有等级，所有代表都可以提供“极客支持”)再到他们对超额的处理。

就像我之前提到的，Ting 有六层[语音方案](https://web.archive.org/web/20221225015850/https://ting.com/plans)。如果你注册了 500 分钟计划并完成了你的配额，你会自动进入下一个计划。虽然经过深思熟虑，但它有时会比其他运营商强加的标准超龄模型更糟糕，尤其是如果你只是踮着脚超过自己的限制。即便如此，Ting 也试图通过让它在相反的方向上工作来弥补这一点——如果你使用的分钟数比前一个月少，你将自动进入一个较低的计划，并相应地获得积分。

为了好玩，我把自己的威瑞森账单扔进了储蓄计算器，看看如果我决定做出改变，我能存多少钱。对于我的两人 700 分钟威瑞森家庭计划，每人 2GB 的数据和 1000 条消息，Ting 估计我每年可以节省近 436 美元。

也许这不是我见过的最令人印象深刻的节省，但这肯定足以让我暂停。如果你或你所爱的人不是巨大的数据猪(像我一样)，会有更多的节省，因为他们的数据包很容易是 Ting 公式中最昂贵的部分。

游戏现阶段，[硬件选择](https://web.archive.org/web/20221225015850/https://ting.com/devices)有点受限。Ting 服务使用 Sprint 的移动网络来提供实际的连接，因此 Ting 产品系列中的大多数手机对 Sprint 客户和手机爱好者来说应该是熟悉的。产品线中的最新设备是摩托罗拉的 angular Photon 4G，而中端 Android 设备，翻盖手机和移动宽带设备填补了他们的名册。在典型的 MVNO 模式下，Ting 并不将用户与长期合同捆绑在一起，但这让那些用户为他们的硬件支付全部零售价。

虽然通常情况下，获得手机补贴并在两年内每月支付更高的账单仍然是一个更好的交易，但当涉及到像 ting 这样的未知实体时，人们可能会(可以理解地)感到害怕。即便如此，得益于母公司的帮助，Ting 或许能够推动用户的接受。

MVNOs 在历史上没有太大的吸引力，但 Ting 可能会依靠 Tucows 庞大的经销商合作伙伴名单。据 Noss 称，Tucows 经销商还将能够向他们自己的客户提供 Ting 无线服务，而 Tucows 则负责支付处理和计费。如果经销商对 Ting 的接受度是积极的 Noss 告诉我，到目前为止是积极的——那么 Ting 缺乏实体零售可以通过利用现有的 Tucows 附属机构来弥补。

Noss 对 Ting 的计划是从小规模开始，他们的第一次促销活动定于本月中旬开始。坦白地说，我希望他们一切顺利——虽然 Ting 不是一个完美的服务，但他们对无线服务的扭曲有一些方面，我认为每个运营商都可以从中受益。不过，与此同时，诺斯满足于保持他的期望非常合理。

“如果在一年之内，我们被认为是高端用户使用的运营商，那对我来说没问题。”
