# iPhone 现在是一个带有 PushupFu 的健身游戏网络

![](img/f66d99b6c2aebbedbe10f02be5738b01.png)正好赶上这些新年决心，[俯卧撑](https://web.archive.org/web/20221007160918/http://www.gymfu.com/)是一款新的付费 iPhone/iPod Touch 应用程序[ [ App store 链接](https://web.archive.org/web/20221007160918/http://itunes.apple.com/WebObjects/MZStore.woa/wa/viewSoftware?id=300935801) ]，旨在通过俯卧撑让你保持健康。现在有很多适用于 iPhone 的健身应用程序。当前领导者 [WeightBot](https://web.archive.org/web/20221007160918/http://tapbots.com/weightbot) 让你每天称体重。其他像 iPump 这样的网站可以让你通过电子邮件将你的进度发送给虚拟的健身伙伴。然而，据我所知，还没有任何原生应用程序将锻炼变成了 Wii Fit 风格的游戏，你可以通过使用 iPhone 的加速度计来验证你是否真的做了锻炼，从而与相同服务的其他人竞争——这正是 PushupFu 所做的。
