# MSN 视频现在是 Bing 视频

![](img/13d44f978e7bc9b12cdf083549cea248.png)

利用 Bing 品牌，微软正在将 MSN 视频整合为 Bing 视频(T2)。如果你去 video.msn.com 的[，它会重定向到必应视频。新的视频搜索目的地让你既能在网上搜索视频，又能在 Bing 播放器中观看。](https://web.archive.org/web/20230315095322/http://video.msn.com/)

主视频页面旨在帮助您浏览和发现视频，顶部是大型播放器中的特色视频，下面是标签类别，包括编辑精选、病毒剪辑、“昨晚的电视”和“必应最佳”很多这看起来像一个直接从 MSN 视频端口。甚至还有一个标签叫做“MSN 视频最受关注”他们现在可能想改变这种情况。

Bing Video 从 YouTube、Hulu、ABC 等网站引入视频，直接主持 900 个电视节目。当你搜索像 *[【真爱如血](https://web.archive.org/web/20230315095322/http://www.bing.com/videos/search?q=true+blood&go=&form=VBREQY)* 这样的节目时，你会在左边看到 Bing 熟悉的导航，它可以让你根据季节、剧集、预告片、长度和格式缩小搜索范围。

必应视频播放器也有熄灯模式。你有点觉得 Bing 不想让你离开。当然，雅虎视频、谷歌视频、Blinkx 或其他大多数视频搜索引擎现在也没有。但是必应视频是一个更好的品牌。下一步什么 MSN 属性会被归入 Bing？
