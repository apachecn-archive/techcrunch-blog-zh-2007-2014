# 奥姆饿了；购买另一个博客(TheAppleBlog)

![](img/4ebd044c22de741f71b59aa423148c0f.png)

他们称之为 Giga Omni Media 是有原因的。Om Malik 的博客帝国 j [今天随着收购](https://web.archive.org/web/20221210063225/http://gigaom.com/2008/09/15/gigaom-acquires-theappleblog/)[苹果博客](https://web.archive.org/web/20221210063225/http://theappleblog.com/)而变得更大了一点。这发生在他[将移动博客 JKOnTheRun](https://web.archive.org/web/20221210063225/http://www.beta.techcrunch.com/2008/07/22/gigaom-buys-a-mobile-blog-the-beginning-of-the-blog-rollup/) 添加到他的博客名册中 GigaOm 品牌下一个多月之后(以及他[作为合伙人加入 True Ventures](https://web.archive.org/web/20221210063225/http://gigaom.com/2008/09/04/evolving-my-work-life/) 两周后)。

具体条款没有披露，但他不太可能为这个应用博客付了很多钱。它吸引的流量远低于一些更知名的苹果博客，如 AppleInsider 或 MacRumors。事实上，一些苹果专属博客带来的访客比 GigaOm 还多(见上图；GigaOm 是蓝色的，AppleBlog 是红色的)。如果 Om 能够帮助苹果博客在同一水平上竞争，这可能会对他的业务产生巨大影响。明智之举。

当其他博客如 Ars Technica 和 Paidcontent 向更大的媒体公司出售自己的时候，Om 正在建立小型博客并把它们加入到他的网络中。当他在 7 月份买下 JKOnTheRun 时，迈克尔写道:

> *我预测这只是这一过程的开始，这一过程将在未来 12-18 个月内加速。缺乏竞争意识的大型博客会卖给大型媒体公司。那些更有竞争力的大型博客想把这件事做到底，他们将开始收购较小的博客，并按照主题领域进行分组。谁能建立起最有趣、最惊人的声音网络，谁就能最终“获胜”*

博客圈自然地把自己[借给了一个总结策略](https://web.archive.org/web/20221210063225/http://www.beta.techcrunch.com/2008/03/19/more-bloggers-raising-money-here-come-the-politics-and-here-comes-my-rant/)。Om 消化完这一个，下一个会买谁？

(**披露**:我曾经和 Om 一起在*业务 2.0* 工作，他是私交)。
