# 微软任命 ValueAct Capital 的梅森·莫菲特进入董事会

今天，微软在一份联合新闻稿中宣布，它已经任命 ValueAct Capital 的总裁梅森·莫菲特(Mason Morfit)为董事会成员，并将发布每季度 0.28 美元的股息。

这两条消息都不令人惊讶。微软和 ValueAct 签订了一份“[合作协议](https://web.archive.org/web/20230316052412/http://www.microsoft.com/en-us/news/press/2013/aug13/08-30announcementpr.aspx)，根据该协议，投资团队可以与高管们“定期会面”，并且 Morfit 可以选择加入董事会。鉴于 ValueAct 在微软的股份中有相当多的资金，他接受要约的决定完全不令人意外。

他到底想让微软做什么，而微软目前却没有做？在二月份，彭博在这个问题上有一个很好的 T4 报告。摘录如下:

> 据知情人士透露，38 岁的 Morfit 和 ValueAct 希望世界上最大的软件制造商减少对 Windows 的关注，Windows 是该公司大部分产品的基础操作系统。由于信息是私人的，这些知情人士要求不要透露姓名。二十年来，微软一直专注于销售专为 Windows 设计的应用程序和服务器软件。
> 
> 现在，ValueAct 希望微软加快努力，解除 Windows 产品和服务的束缚，以便它们能够在智能手机和平板电脑上得到更广泛的采用。

鉴于微软正在双倍下注 Windows，这似乎令人怀疑。目前，微软的新 CEO 脑子里有一个新的声音。
