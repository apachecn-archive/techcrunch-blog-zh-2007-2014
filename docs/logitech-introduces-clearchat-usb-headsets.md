# 罗技推出 ClearChat USB 耳机

![](img/f45e8f5291517a561fbd2f4b370aa908.png)
clear chat Pro 的耳机内置音量控制功能

无论你是用 Ventrilo 和《魔兽世界》*中的公会成员通话，还是用 Skype 给海外的家人打电话，有一件事是肯定的:你需要一个好的耳机。今天，罗技推出了两款新的耳机，这两款耳机同时兼容 OS X 和 Windows。ClearChat Pro 和 ClearChat Comfort 通过使用“激光调谐驱动器”和衬垫耳罩，应该比竞争对手的声音更好。当然可以。*

 *两者之间的区别(除了价格)是 Pro 型号带有一个均衡器开关，这样你就可以轻松地从音乐设置切换到游戏设置，以及耳机本身的音量调节开关。无论哪里有卖电子产品的，你都可以以 40 美元买到 ClearChat Comfort，50 美元买到 Pro。
