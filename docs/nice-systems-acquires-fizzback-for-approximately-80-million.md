# NICE Systems 以大约 8000 万美元收购 Fizzback

![](img/3a235e216ab0c30e09cf6f9d187e4c65.png)

[尼斯系统](https://web.archive.org/web/20230203114600/http://www.nice.com/)今晨[宣布](https://web.archive.org/web/20230203114600/http://www.nice.com/blog/enterprise/nice-acquire-fizzback-amplifying-voice-customer)将以大约[8000 万](https://web.archive.org/web/20230203114600/http://www.crunchbase.com/company/the-fizzback-group)美元现金收购英国公司 [Fizzback](https://web.archive.org/web/20230203114600/http://www.fizzback.com/) 。Fizzback 得到了 Advent Venture Partners、Nauta Capital 和 TAG 的支持。

Fizzback 通过手机、网络或社交媒体向消费者发送与特定互动或交易相关的反馈请求。然后，Fizzback 对反馈进行分析，以确定相关的响应，公司随后在体验点吸引消费者，例如在联络中心、分支机构、销售点、移动应用程序或网络上。
