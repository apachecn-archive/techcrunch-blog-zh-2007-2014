# 黑莓对剩余粉丝采取强硬措施，对泄密者采取法律行动

黑莓，这不是你处理狂热粉丝的方式。首席执行官陈志强今天发布了一篇博客文章，承诺公司将对“窃取未来黑莓产品机密信息并将其公之于众的一方”采取法律行动啊。

这很烦人，因为它代表着公司对可以说对其有利的事情采取了令人费解的强硬立场。事实上，正如陈所说，泄密并不是“分散注意力，最糟糕的是，完全误导了我们的利益相关者”，而是小工具和移动设备生态系统的自然组成部分，也是可能真正想购买您产品的人表达真正兴趣的一种方式。

这封公开的警告信是一个例外的例子，这是有原因的——其他设备制造商知道，最终，谣言和早期报告在最坏的情况下会在产品周围引起轰动，在最好的情况下会吸引一些眼球，否则这些眼球可能一开始就不会关注你的公司。陈在他的文章中认为，最近的泄密事件已经跨越了“犯罪”的界限，因此有必要采取法律行动。

但它告诉我，我不得不做一些谷歌功夫，以找出泄漏可能正在这里讨论，这大概是这个[揭示的黑莓操作系统 10.3 从几天前](https://web.archive.org/web/20230129235503/http://www.gsmarena.com/blackberry_os_103_screenshots_leak_voice_assistant_coming-news-8112.php)。陈最后表示，我们将会看到更少包含关于下一代黑莓智能手机的照片和谣言的博客帖子——据我所知，除了黑莓本身，这对任何人都没有负面影响。

这种“对泄漏采取强硬态度”的立场本身可能是试图吸引更多的注意力到即将推出的产品上(毕竟，陈链接到了 Z3 官方档案)，但不管是什么，听起来都是充耳不闻。的确，苹果公司在苹果手机 4 被一名员工留在酒吧后，随后被卖给了吉兹莫多，并因此受到了严厉打击，但这是一个极端的例子。一般来说，泄漏循环会不间断地进行，即使原始设备制造商确实觉得有必要做出反应，他们也几乎不会注意到他们正在这样做的事实，因为疏远狂热分子最终没有什么好处。
