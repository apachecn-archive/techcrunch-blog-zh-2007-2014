# Twitter 发布扩展的 Tweets，让你可以阅读新闻、查看图片和在线观看视频

在今天的 Twitter 博客上，该公司宣布了一种新的方式，用户将能够访问来自合作伙伴的更多内容。最新的创新——扩展的 Tweets——将允许用户直接查看来自某些合作伙伴的内容，而无需离开其网站。

像《纽约时报》、《华尔街日报》、《MSNBC 新闻》、《旧金山纪事报》、《时代周刊》和《时代》这样的新闻合作伙伴现在可以让用户直接在他们的推文中看到更多内容，而不必点击进入他们的网站。例如，当你扩展一条基于新闻的推文时，Twitter 的扩展推文将提供标题和介绍的预览，以及发布者和作者的 Twitter 账户。然后他们可以点击阅读文章，关注新闻，在 Twitter 页面上回复或转发。

Twitter 扩展推文的目标不仅仅是在线新闻发布者或文本。用户还可以直接在推文中查看图片和视频。Twitter 为这一功能命名的合作伙伴包括图片方面的 WWE、BuzzFeed 和 TMZ，以及视频方面的 BET、Lifetime 和 Dailymotion。

目前，这项功能只能通过 Twitter 网站和移动网络获得。但是 Twitter 产品经理 Michael Sippey 在 Twitter 博客上写道，用户很快就会通过 iPhone 和 Android 的原生应用程序获得相同的在线查看功能。
