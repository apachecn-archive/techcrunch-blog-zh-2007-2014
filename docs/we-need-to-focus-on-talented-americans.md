# 美国需要将教育重点放在有才华的美国人身上

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/11/11/we-need-to-focus-on-talented-americans/>

**编者按:** *乔纳森·韦(Jonathan Wai)是杜克大学人才鉴定项目的研究员和作家，也是《寻找下一个爱因斯坦:为什么聪明是相对的》【今日心理学的 的作者。在推特上关注他。*

Vivek Wadhwa 在他的新书《移民外流》中认为“美国的未来取决于技术移民。”这本书以他作为来自印度的天才移民的引人注目的个人旅程开始，讲述了他年轻时是如何抓住机会来到美国的。

瓦德瓦是一个在美国达到成功顶峰的移民。他是《华盛顿邮报》和《彭博商业周刊》的定期专栏作家，在杜克大学、斯坦福大学、埃默里大学和奇点大学担任学术职务，并在加入学术界之前创建了两家软件公司。

换句话说，他是他最新著作主题的光辉典范。

然而，他的个人经历并不是他展示的唯一证据。基于他对有才华的移民企业家和商业领袖的大量研究，Wadwha 提出了一个令人信服的理由，即美国的未来可能确实取决于，至少部分取决于有才华的移民:

“今天，许多专家和观察家质疑我们是否正在目睹美利坚帝国衰落的开始。我向你提交，这可能确实是这样的。在疏远和排斥技术移民企业家和发明家的过程中，我们不仅阻断了构建这个伟大国家经济支柱的命脉，我们还扼杀了创造下一个伟大事物的神经末梢。如果我们恢复了这种流动，我们就恢复了我们的国家。”

也许瓦德瓦是对的。但我认为，仅仅关注有才华的移民，我们就忘记了一群人，他们对美国的未来同样重要，如果不是更重要的话:有才华的美国人。我们尽全力教育他们了吗？事实证明，答案是我们并没有真正做那么多。

根据全国天才儿童协会的网站，联邦政府将 0.02%的教育预算用于天才美国人的项目。瓦德瓦关注有才华的移民是正确的，因为他们对我们国家的 GDP 有很大影响。但是有才华的美国人呢？

研究人员海纳·林德曼和詹姆斯·汤普森已经证明，一个国家前 5%的智力人才对该国的 GDP 产生了不成比例的影响。换句话说，最聪明的美国人，包括出生在美国的人和移民，对美国的未来极其重要。

然而，这些最终来到美国的有才华的移民在非美国的教育系统中发展了他们的才华。换句话说，瓦德瓦认为重要的人已经是来自各自国家的最优秀的个人。

正如我在关于投资美国最有才华的人的文章中所说的:

“在竞技体育中，有板凳球员、普通球员和明星。在教育中，有低于平均水平的学生、中等水平的学生和优等生。如果一名教练决定只专注于发展替补球员和普通球员的天赋，球迷会不会同意是值得怀疑的——这将降低球队的竞争力。然而，我们在美国致力于同等的教育——我们专注于教育低于平均水平的学生和中等水平的学生，而往往忽视我们的尖子生。如果这在竞争激烈的体育界行不通，那在我们残酷的全球经济中又有什么意义呢？”

Wadwha 写了一本引人注目的书，有力地支持为有技能和有才华的移民敞开大门。但是，在我们专注于帮助其他国家的最优秀和最聪明的人之前，难道我们不应该专注于帮助我们自己国家的最优秀和最聪明的人，并为他们打开尽可能多的大门吗？

我想是时候了。
