# 微软收购企业群聊公司 Parlano

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2007/08/30/microsoft-acquires-enterprise-group-chat-company-parlano/trackback/>

### 邓肯·赖利

[![parlano.jpg](img/c93c710147463de6fec7c2943e074ca7.png)](https://web.archive.org/web/20080705150900/http://www.crunchbase.com/company/parlano) 微软宣布已经同意收购总部位于芝加哥的 [Parlano](https://web.archive.org/web/20080705150900/http://www.crunchbase.com/company/parlano) ，这是企业群组聊天平台 [MindAlign](https://web.archive.org/web/20080705150900/http://www.parlano.com/asp/products/mindalign2007/index.asp) 的制造商。

微软表示，将利用 Parlano 的群聊功能来改进微软 Office Communications Server 和微软 Office Communicator。

Parlano 的平台提供群聊技术，“使人们能够进行特定主题的多方即时通讯讨论，并持续一段时间。”Parlano 的现有客户包括金融服务公司和呼叫中心。

Microsoft 计划将群聊作为 Office Communications Server 2007 标准客户端访问许可证的一部分提供。

收购的条款没有公布。该交易预计将于 2007 年第四季度完成。
