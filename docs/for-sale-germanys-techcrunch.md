# 待售:德国的“TechCrunch”

我们已经得到了很多关于此事的提示，德国媒体也疯狂地报道了这件事，所以，如果你曾经想买一个德国博客，现在是时候把你的钱投在你的鼠标上了。因博客 [Basic Thinking](https://web.archive.org/web/20221207212610/http://www.basicthinking.de/blog/) 而在德语国家出名的 Robert Basic，已经将网站[放在易贝](https://web.archive.org/web/20221207212610/http://cgi.ebay.de/ws/eBayISAPI.dll?ViewItem&item=260343987629)上出售，据说是因为他想从头开始。

Basic Thinking 是一个技术博客，它实际上被一些人称为“德国 TechCrunch ”;我们很荣幸，但我们不确定，因为我们没有跟上它，我们的德语有点生疏。Basic 对拍卖清单上的博客收入非常直接:2008 年，他通过展示和文本链接广告获得了约 37，000€毛收入。他一直没有在网站上积极销售广告，所以他相信这可以带来更多的收入，特别是考虑到他作为德国 ICT 博客*的地位，以及他的谷歌页面排名可以通过简单地删除文本链接广告从 4 恢复到以前的 6。*

基本思维的访问者数量并不多，但考虑到利基市场和有限的(仅限德语)范围，他们已经足够体面了:据报道，上个月该网站接待了 185，000 名独立访问者，提供了 254，000 次页面浏览。2008 年的独立访问者总数不到 250 万，页面浏览量为 360 万。嵌入的图表显示了该网站自 2005 年启动以来的增长情况(它实际上是在 2003 年启动的，但已经中断了一段时间)。
