# 颚骨向上又回来了，但是比以前更好吗？

我这周大部分时间都在戴着 [$129 的 Jawbone UP](https://web.archive.org/web/20221208233810/https://jawbone.com/up) ，我想我应该给那些不确定买哪个型号计步器的人提供一些印象。我自己也是一个 [Fitbit](https://web.archive.org/web/20221208233810/https://beta.techcrunch.com/tag/Fitbit) 人，因为我已经习惯了把这个小小的加密狗从一个口袋搬到另一个口袋，而且我在旅行中只丢过一个。但是——这是一个很大的但是——如果 Jawbone 解决了一些小问题，我可能会被说服去 Jawbone。

如果你还记得的话，UP 是一个能感应你运动的手镯。这可以用来在白天观察你跑了多少，在晚上观察你睡得怎么样。你也可以记录你的食物摄入量和情绪，从而给你一个相当好的关于你的卡路里输入输出和一般状态的图片。

它在一个大约像手镯一样大的包装中可以容纳大约一周的电荷，并且有一个按钮可以从白天模式切换到晚上模式。它外面包着橡胶，是防水的。它还有一个无声的闹钟，可以轻轻地把你从睡眠中唤醒，还有一个懒虫警报，会在一定的时间间隔内发出嗡嗡声，提醒你离开懒惰的巢穴，四处走走。

新的 UP 还有什么不同？好吧，Jawbone 完全重新设计了内部结构，确保没有水可以进入，并且取下和装上它的持续压力不会损坏连接器、芯片或电池。变化几乎完全是内部的，但理想情况下，这一个将比以前的版本更坚固。

现在问题来了:与 99 美元的 Fitbit One 和其他设备不同，你可以通过将其插入手机的耳机插孔来同步它。有一个小帽盖在插孔上(在某些时候，它不可避免地会脱落和丢失)，还有一个 USB 加密狗，你可以用它给笔记本电脑充电。你不会真的在桌面上与 UP 交互，这已经足够好了，但我喜欢无线同步的方式。据推测，这种设备的尺寸——它出奇的小——阻止了它成为一种选择，但它可能会成为一些人的交易障碍。遗憾的是，行走的步数没有外部指标，所以当谈到行走时，你大部分时间都处于黑暗中。
![](img/be6d1271e985aa8db5969550c752f430.png)
对其他人来说，取下手机，打开盖子，插上电话并不难。你不必每天都这么做(但你会想这么做的),这款应用足够酷，你可以原谅它的一些缺点。

但是市场上有这么多其他设备，为什么是这个呢？简单是一个因素。这看起来不像是书呆子设备。如果你对橡胶感兴趣，它实际上可以伪装成一件珠宝。UP 也非常准确，每次我检查它时，它几乎都与 Fitbit 的一模一样。换句话说，它是有效的。

我也喜欢提醒功能，它比 Fitbit 的 quiet calm 好用一点。手腕上的嗡嗡声帮助我站起来四处走动，这是一件很棒的事情。

到底哪个更好？我不确定。这些事情不断提醒我们作为生物有机体的失败。我确实喜欢 UP 的简单，但 Fitbit 的楼梯传感器和全面的在线界面仍然让我不断回来。我会两个都穿——我就是那样的一个呆子——但我想在这种情况下，经验法则很简单:如果你倾向于经常失去小东西，就买大的。否则，权衡利弊——价格是其中之一——看看哪些功能你会用得最多。可悲的是，没有一个明确的赢家，但对于过着量化生活的人来说，选择过多是件好事。
