# Indaba:一个音乐协作中心，可以为你建立自己的乐队 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/01/13/indaba-a-music-collaboration-hub-that-can-build-your-band-for-you/>

[![](img/ebb48ccbdf6df6dc8a23a8d265518070.png)](https://web.archive.org/web/20221006203326/http://www.indabamusic.com/)

音乐合作服务 [Indaba](https://web.archive.org/web/20221006203326/http://www.indabamusic.com/) 刚刚为其网站引入了许多新功能，包括一个新的类似脸书的聊天系统，增强的评论，以及一个帮助兼容艺术家找到彼此的推荐引擎，有效地使其能够自动挑选出潜在的乐队成员。该网站自 2007 年推出以来一直不受我们的关注，但已经发展到拥有超过 125，000 名音乐人。

Indaba 是一个音乐社区，面向寻求远程合作的音乐家，分享他们创作的音乐。然而，尽管该网站确实让人们很容易听到其他成员创作的歌曲(歌曲可以用直观的播放器播放，而不是坐在屏幕底部)，但 Indaba 并没有致力于推广你的乐队和赢得粉丝(至少目前没有)。

为了真正创作音乐，用户通常使用 GarageBand 等程序录制他们的曲目，然后将他们的曲目上传到在线会话。在这些会话中，多个用户可以使用集成的聊天框来谈论曲目，也可以使用基于 Flash 的混音器来重新编排每个音乐家的作品。此时，该服务的导出功能可能需要一些工作(您只能导出为常规音频轨道，而不能导出为 GarageBand 或 ProTools，尽管这是因为专有文件格式问题)。

![](img/b1a4975494916f626566d4caa47e482f.png)

该网站还允许用户建立支付协议，这意味着你可以在自己家里舒适地作为一名录音室音乐家工作。用户也可以使用该网站来指定与每首歌曲相关的权利。

今天推出的新功能包括歌曲内评论，同时允许用户留下带有视觉指示器的评论，指定他们指的是歌曲中的哪个位置(例如，我可以留下一个便条，说我喜欢某个吉他独奏或鼓点)。音乐家也可以为他们的观众留下注释，当轨道到达相关点时就会出现(我可以留下一个注释说，“我仍在进行这一部分”)。

另一个新功能是一个增强的网站范围的聊天，类似于脸书聊天。使用这个新系统，用户可以监控他们的朋友何时在线，并且无论他们在 Indaba 的什么地方浏览都可以聊天。

最后，该网站正在使用 EchoNest(麻省理工学院媒体实验室的一家音频公司)的技术来产生音乐家推荐。不像大多数的音乐推荐引擎那样，将用户与他们可能喜欢的歌曲配对，EchoNest 系统是用来将可能合作得很好的音乐家配对的。系统会将你之前在 Indaba 录制的歌曲以及你在用户档案中指定的兴趣考虑在内。

吲达巴的竞争对手包括[最低噪音](https://web.archive.org/web/20221006203326/http://www.minimumnoise.com/Default.aspx)(此处涵盖)、 [WeMix](https://web.archive.org/web/20221006203326/http://www.wemix.com/) 和 [Indomite](https://web.archive.org/web/20221006203326/http://www.indomite.com/) 。
