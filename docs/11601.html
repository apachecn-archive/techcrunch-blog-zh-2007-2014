诺亚2011为伦敦带来了一个泛欧洲的网络空间


我喜欢诺亚的一点是感觉有点像一个大型鸡尾酒会。是的，有面板和演示，优雅地隐藏在分隔展览区和会议厅的黑色窗帘后面，舞台上的大人物在那里与观众分享成功故事。但真正的活动是在酒吧邀请交往的地方。在诺亚，人际关系网更有趣。

过去两年，由诺亚咨询公司创始人马可·罗德兹内克(Marco Rodzynek)组织的年会，已经赢得了名副其实的泛欧洲顶级盛会的声誉，充满了投资机会。它名副其实。诺亚2011是充满活力的，它很大，有超过1700名与会者和100名发言人。但真正令人惊叹的是，这种感觉如此国际化。在旧比林斯盖特体育场，只有少数观众来自英国。

相反，这听起来像是整个德国初创企业在伦敦相遇，来自德国三大科技中心的企业占据了主导地位:柏林、慕尼黑和汉堡。

来自汉堡的企业家，比如出租车预订应用 MyTaxi 或者超级简单的网站编辑器 Jimdo ，让我相信这个海港已经成为柏林德国顶级科技中心头衔的有力竞争者。柏林充满了创造力，但汉堡可以为那些对商业机会有着卓越直觉的人而自豪。还记得汉堡投资公司 HackFwd 吗？创始人Lars Hinrichs在台上解释了为什么该基金只投资软件开发商。因为他们是“21世纪的艺术家”。听到听到。汉堡也有优秀的开发人员，结合Lars的态度，我们有一个新的热点值得关注，因为它目前正在快速增长。

所以德国赢得了本周最具统治性的奖项，但是第二个引人注目的创业热点是以色列。publisher network Conduit 和他们最近收购的 Wibya 的首席执行官们都是小组发言人，反馈管理平台 Kampyle 的首席执行官Ariel Finkelstein和我聊了聊以色列不断增长的热心企业家网络。以色列创业人群的热情和勤奋达到了一个新的高峰，诺亚2011证明了这一点。

正如它的名字所承诺的，如果你想在不离开伦敦的情况下结识一些来自欧洲各地的伟大企业家，诺亚仍然是一个值得去的地方。我希望在明年的诺亚2012年会上保持这种状态。日期已经定在2012年11月6日和7日。

本周会议的舞台本身是一个平台，大多数成熟的初创公司都在这里展示自己。这些演示有时感觉像是自我推销，这是一种耻辱。有这么多重要的国际创业公司参与，一些伟大的舞台讨论的潜力是巨大的，结果并不完全符合人群的要求。

这并不是说没有一些好的面板。很快你就可以在诺亚网站上亲眼看到了。我被告知，所有面板的视频将于下周末在网上发布。

总而言之，做得很好，诺亚，2012年再见。告诉你吧，如果你在明年的演讲中加点火，我的评分甚至会从“棒极了”变成“绝对牛逼”。
