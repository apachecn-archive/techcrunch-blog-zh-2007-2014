# 达美首席执行官禁止飞行中通话

联邦通信委员会可能已经迈出了取消飞行中使用手机禁令的第一步，但这并不意味着你会很快听到任何邻居的私人谈话。

至少德尔塔上没有。

该航空公司的首席执行官理查德·安德森向所有 80，000 名达美员工发出了一份全公司备忘录，称达美将不会允许任何达美航班上的手机或 VoIP 电话。

然而，安德森确实表示，一旦禁令正式解除，“达美将迅速采取行动，使客户能够使用文本、电子邮件和其他门到门的无声数据传输服务”。

目前来看，联邦通信委员会已经[投票赞成](https://web.archive.org/web/20221209040246/http://www.techmeme.com/131212/p38#a131212p38)向公众开放讨论，但最终运输部将拥有最终决定权。甚至在交通部批准后，个别航空公司将有权执行他们自己的规则。

对达美来说，这意味着不能打电话。
