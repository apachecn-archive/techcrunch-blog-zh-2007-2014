# OpenFeint 的游戏频道使用 Groupon 模式销售其手机游戏

由 Groupon 开创的团购模式已经进入了移动游戏和应用领域。几个月前，Y Combinator 支持的 [TapZilla](https://web.archive.org/web/20230203185157/http://tapzilla.com/) [推出了](https://web.archive.org/web/20230203185157/https://techcrunch.com/2010/10/13/tapzilla-offers-daily-deals-for-paid-mobile-apps/)，将当日交易模式应用于付费移动应用。今天，移动社交游戏平台 [OpenFeint](https://web.archive.org/web/20230203185157/http://openfeint.com/) 加入了 Game Channel 的发布，这是一个移动游戏应用的团购网站。

Game Channel 允许开发人员在 OpenFeint 平台上开发游戏，通过每日和每周的游戏交易来推广他们的游戏。具体来说，游戏频道的大甩卖功能使用团购模式来推动游戏开发商的下载量。

每周，Fire Sale 都会推出一款面向 OpenFeint 5000 万用户的高端游戏，通常定价为 1.99 美元至 9.99 美元。当足够多的游戏频道用户投票表示想要这款游戏时，游戏的价格就会下降。OpenFeint 表示，它将向投票者发送推送通知，通知他们游戏限时发售。

首先发售的是《大白鲨》(Jaws)，这是一款官方游戏，改编自 Bytemark Games 的夏季大片。游戏频道还将包括免费游戏，这使得一个游戏免费一天。

团购热潮延伸到应用和游戏领域只是时间问题。但我很好奇，当移动应用生态系统中有这么多免费游戏时，这种模式是否能够推动付费游戏的销售。我想我们会找到答案的。
