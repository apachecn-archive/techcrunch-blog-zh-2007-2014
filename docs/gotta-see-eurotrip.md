# 三星 Galaxy Tab 7.7，Galaxy Note 不在美国上市？从来没有？

叹气。三星显然不打算在美国推出两款最新的平板电脑。这句话来自一位 IFA 三星代表，他表示没有计划将平板电脑带给善良的美国公民。当然，这位代表可能不了解情况，但大多数时候，这些宣传者都避免使用绝对的陈述，而是使用他们花哨的营销圈言论。但是这个代表几乎扼杀了所有希望。

这里提到的两款平板电脑 Galaxy Tab 7.7 和 Galaxy Note 是今年 IFA 最受欢迎的产品。它们都在细长、大小合适的外壳中展示美丽的 AMOLED 屏幕。7.7 是对最初的 Galaxy Tab 的一次很好的更新，Galaxy Tab 也于 2010 年在德国的 IFA 展上首次亮相。然后，Note 继承了被取消的戴尔 Streak 5 放弃的横幅，但带有一个活动的数字化手写笔。

一个厂商会在不同的地区销售不同的车型，这不是没有先例的。如果这是真的，三星的这一举动可能是对其与苹果在欧洲的法律交易的回应。如果萨米不能销售 GalTab 10.1，那么他们将向市场推出一系列非常规平板电脑。事实上，三星应该从一开始就采用这种策略，通过更小的外形提供不同使用案例的反 iPad 产品来冲击市场，而不是与 iPad 正面交锋。

不过，这让我们这些买家蒙在鼓里。Sprint、威瑞森和美国电话电报公司仍在销售最初的 7 英寸 Android 2.x Galaxy Tab 和新的 10.1 iPad 克隆版。美国市场正在获得刚刚宣布的 3G 和 WiFi 版本的 8.9 英寸 Galaxy Tab。也许三星的核心人物认为超过三款的机型会让普通美国购物者感到困惑，尽管大多数技术专家认为新的 GalTab 7.7 将取代非常旧的 7 英寸机型。哦，那听起来不像是会发生的事。但是，正如 Android 的所有产品一样，另一家制造商可能会推出几乎相同的型号，包括像三星 Note 这样的较小的平板电脑，并在美国推出，而三星正在欧洲试水。
