# Blaupunkt 存储卡汽车立体声 nix 光盘

![blaupunkt](img/2b883df25850c8992c8f23aa6dbfee34.png)

我一方面可以数出我使用过我的汽车 CD 播放器的次数(五次或更少)，所以看到上述设备售价 150 美元，对我来说也是一种极大的安慰。如果我的立体声系统真的报废了，我会很乐意改用全存储卡格式。

Blaupunkt Melbourne SD27 接收器处理加载到 SD 卡或 MMC 卡上的 MP3 和 WMA 文件，有一个前置辅助输入，还可以配备一个 20 美元的 iPod 连接器和/或蓝牙模块。蓝牙部分和接收器本身一样昂贵，所以如果你明白我的意思，你真的应该抱着 30 英尺个人区域网络的想法坐在树上。

其他功能包括:

–18 瓦 RMS 功率输出，45 瓦峰值
–2 个前置放大器输出(2 伏)
–3 波段均衡器
–无线遥控器
–可拆卸面
–除了前面的一个
外，后面的辅助输入–CD 换碟机控制
–分别带有 15 和 10 个预设的 FM/AM 收音机

对一个半本杰明来说还不错，不过如果你愿意忍受拥有 CD 播放器功能的“不便”，你可以在 [Target](https://web.archive.org/web/20221007195129/http://www.target.com/gp/detail.html/ref=br_1_2/602-2081710-1481444?ie=UTF8&frombrowse=1&asin=B000FFEXEY) 或 [Walmart](https://web.archive.org/web/20221007195129/http://walmart.scosche-cars.com/products2/?sfID1=24&productID=1652&walmartProductID=4744710) 上花 80 美元买到类似功能的音响。讽刺。不过，说真的，我认为布劳蓬克装置听起来会好一点，也更可靠一点。它是否值额外的 70 美元由您决定。
