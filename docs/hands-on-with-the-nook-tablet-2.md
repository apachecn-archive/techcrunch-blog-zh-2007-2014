# 动手操作 Nook 平板电脑

我们已经抓拍了一些 Nook 平板电脑的镜头，有趣的是看到了该设备的响应速度和速度，尤其是与 Nook Color 相比。这是一款全多媒体平板电脑，拥有充足的视频和丰富的内容，一切都加速进行，没有太多的延迟或口吃。
