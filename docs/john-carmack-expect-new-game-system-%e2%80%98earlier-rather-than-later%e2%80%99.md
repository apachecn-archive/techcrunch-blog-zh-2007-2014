# 约翰·卡马克:期待新的游戏系统“越早越好”

![gamers](img/ec5c485e3cab3ead1d70e303d27cdce9.png "gamers")

假设你下班或下课回到家，做了晚饭，坐下来玩一个小时左右的*游戏 2:游戏*。你对此很冷静；今年秋天，你没有理由再花 500 美元去买一个视频游戏系统。但是，哦！等等！这是什么？约翰·卡马克似乎认为，尽管这一代游戏机还没什么问题，但*有人*可能[会“早一点而不是晚一点”发布新的视频游戏系统](https://web.archive.org/web/20221006214740/http://www.gamesindustry.biz/articles/carmack-new-next-gen-console-will-end-up-shipping-earlier-rather-than-later)

这位 id 软件专家对波兰网站 [CD-Action](https://web.archive.org/web/20221006214740/http://www.cdaction.pl/news-7673/john-carmack-opowiada-o-rage---pol-godziny-goracego-materialu-prosto-ze-studia-id-software.html) 也是这么说的。事情是这样的，卡马克认为有人——姑且称之为“索尼”——不希望竞争对手再次在这方面领先一年。(Xbox 360 于 2005 年秋季上市，而 PS3 于 2006 年秋季上市。)鉴于 PS3 一直表现平平，或许索尼不想面对又一代被微软打败的人。

当然，由于我持有这两家公司的股票，我完全满足于看着它们无情地一决雌雄。

卡马克还表示，下一代游戏机很有可能完全跳过物理媒体，转而支持全下载交易，就像 PSP Go 一样。
