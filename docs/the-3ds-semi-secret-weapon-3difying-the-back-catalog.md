# 3DS 的半秘密武器:3 定义后台目录 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/06/28/the-3ds-semi-secret-weapon-3difying-the-back-catalog/>

当 3DS 在[宣布](https://web.archive.org/web/20221210002300/http://www.crunchgear.com/2010/06/15/nintendo-unveils-the-3ds-which-puts-3d-gaming-in-your-hand/)的时候，任天堂做了大量的粉丝服务。我不知道你，但我几乎一生都在期待一款新的*儿童游戏《伊卡洛斯》*。除了翻拍或重拍经典之作，如*时光之笛*、 *StarFox* 、 *Paper Mario* ，还有许多第三方游戏和其他有趣的东西。但是任天堂从来没有依靠第三方来获得突破性的成功。偶尔会有类似*黄金眼*的东西出现，并艰难地面对它们，但大 N 一直是关于它的核心属性，老实说，挖掘怀旧情绪。

所以:考虑一下。有什么明显的举动？新的 3D 技术…旧的标题。我保证我们将会看到一个 DSWare/downloadables 的巨大爆炸，以 3DS 兼容的立体 3D 渲染的 back catalog 游戏的形式出现。
 任天堂所要做的就是将一些标准图形引擎移植到 3DS 的令人印象深刻的新硬件(我们认为它的图形保真度接近 Wii 水平)，将一些控件映射到底部屏幕，并将输出设置为与 3DS 的视差栅栏显示器兼容。我的意思是，这不是一个*琐碎的*任务，但任天堂可以在内部完成这一切，已经领导了许多上述图形引擎的开发，并已经有了一个庞大的专门移植团队——正如已经宣布的标题所证明的那样。

那个视频是在一个装有 iZ3D 驱动程序的模拟器中完成的。如果他们能做到，任天堂也能。启用您选择的 3D 观看方法；“平行”最适合我，没有任何眼镜。([在 YouTube 上观看](https://web.archive.org/web/20221210002300/http://www.youtube.com/watch?v=G5usQPQdhwo)非失真版本)

所以十有八九，你已经有了像 *F-Zero X，疾风军团，身体收割，Wave Race 64* (这将是令人敬畏的)，甚至可能是 GameCube 的东西，如 *Wind Waker* 和 *Metroid Prime* 即将推出。我们甚至会看到基于深度增强(我喜欢深度增强)的精灵游戏吗？我不会放过他们的。SNES 游戏中的背景层通常已经被分离以提供视差移动，尽管当然 3D 效果不会那么完整。不过，超级特效和模式 7 游戏会很有趣。

作为奖励，如果他们决定支持大屏幕的 3D(很可能他们的下一台游戏机至少会有这个能力——3-Wii？Wii-D？)，他们会有一船游戏准备发行。

说真的。任天堂不可能不利用其庞大的后备目录——他们知道他们可以转换，他们知道人们会购买它们。他们显然已经优先考虑了大卖家的游戏，但我们可能会在 3DS 发布日期之前看到更多的公告，最早可能是 2010 年假期。
