# Hulu 现在生活在日本，只提供订阅服务

这标志着 Hulu 的首次海外扩张，Hulu 现在可以在日本使用。正如[所预期的](https://web.archive.org/web/20230205013349/https://techcrunch.com/2011/08/10/hulu-to-start-streaming-in-the-land-of-the-rising-sun/)，视频流媒体服务将在各种设备上提供电视节目和电影，每月收费约 20 美元(1480 日元)。

与美国不同的是，在美国有免费和付费两种版本，在日本这项服务将只接受订阅。它的价格是 Hulu Plus 在美国的两倍多(每月仅 8 美元)。Hulu 收取这么高的费用是因为它认为可以在日本获得更高的订阅费，还是因为它不得不为国际许可权支付高昂的费用，并且没有其他选择？

Hulu 将以英语和日语播放 CBS、NBC 和 ABC 的美国节目，以及《T4》《加勒比海盗》、《世界末日》、《黑衣人》和《特洛伊》等电影。Hulu 计划加入日本制作的节目和电影。(令人惊讶的是，他们没有与任何日本内容合作伙伴合作，但这些合作伙伴应该会出现)。

与此同时，Hulu 还宣布与日本最大的移动运营商 NTT Docomo 建立独家移动合作关系。订阅内容包括在智能手机和平板电脑上观看，以及在电脑、联网电视和视频游戏机上观看。
