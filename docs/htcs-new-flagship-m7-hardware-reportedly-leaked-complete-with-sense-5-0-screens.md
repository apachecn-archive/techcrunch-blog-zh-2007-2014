# 据报道，HTC 的新旗舰 M7 硬件泄露，配有 Sense 5.0 屏幕

HTC 去年经历了令人失望的一年，尽管 One X 和稳健的硬件发布取得了初步成功。今年，它似乎将推出一款新的旗舰设备，名为 HTC M7，有传言称这款手机可能会在下个月的 MWC 发布。上周泄露的[渲染图中据称描绘了这款手机，但现在，看起来非常不同的硬件出现在安卓警方](https://web.archive.org/web/20221202221114/https://beta.techcrunch.com/2013/01/16/htcs-flagship-m7-smartphone-supposedly-outed-but-wheres-the-htc-look/)获得的[照片中。](https://web.archive.org/web/20221202221114/http://www.androidpolice.com/2013/01/20/exclusive-this-may-actually-be-the-htc-m7-with-sense-5-0/)

乍一看，美联社发现的 M7 泄漏看起来更像我们期待的 HTC。它拥有该公司的标志性外观，圆角和前玻璃看起来在屏幕边缘稍微变细。这款手机的背面看起来也像 HTC signature 风格，在看起来像橡胶表面的表面上有一层哑光黑色，在设备底部附近有一个位于中央的后置摄像头和扬声器格栅。

传闻这款手机的规格包括 4.7 英寸的 1080p 分辨率显示屏(令人难以置信的 468ppi 像素密度)和 1.7GHz 四核高通处理器。在新泄露的图片中，它被描述为运行 Sense 5.0，这是 HTC 自己的 OEM 专用定制 Android 皮肤的下一代产品。Sense 似乎为用户界面的组合带来了一些相当大的美学变化，包括一个采用数字外观的时钟，支持翻盖时钟风格，HTC 已经在相当长一段时间内使用它作为自己的 Android 标志性产品。

另一个值得注意的特性是新的信息窗格部件，它看起来可以提供来自各种来源的信息流，有点像微软的 live tiles 所提供的 Android 近似。不过，这些还没有在截图中填充，所以如果这些泄露被证明是准确的，我们将不得不等待并看看它们在实践中的实际情况。

HTC 需要在 2013 年走出摇摆的大门，以提升品牌，赢回消费者的信心，但其最近的 Droid DNA 和 Butterfly 手机已经在这个方向上迈出了一些戏剧性的步伐。M7 应该结合了行业领先的显示器和可能比巨型设备更电池和用户友好的外形因素，可能正是医生所订购的。
