# 如何不优雅地处理辞职

每个故事都有两面性，但是[这封](https://web.archive.org/web/20230213024828/http://pastebin.com/PSY4iYZ0)的[马哈罗](https://web.archive.org/web/20230213024828/http://www.mahalo.com/)创始人兼首席执行官[杰森·卡拉卡尼斯](https://web.archive.org/web/20230213024828/http://www.crunchbase.com/person/jason-calacanis)和他的一名员工(现已离职)之间的电子邮件交流是如何不处理辞职的一个教训。

Jason [说](https://web.archive.org/web/20230213024828/http://twitter.com/Jason/status/12733869779)这是一次私人交流，他只是对 Evan 说实话。艾凡说杰森无法控制自己的情绪。

如果你要诋毁你的员工，口头上说出来，这样以后就不会在网上留下记录。或者，根本不要扔掉它们，和团队一起组织饮料为它们送行，这样你的其他员工就知道你关心他们。从下往上读。
