# 百思买发布内置 TiVo 的 Insignia 电视

百思买的内部技术品牌 Insignia 刚刚发布了一款内置 TiVo UI 的电视。为什么？我想人们不希望在他们的 CE 书架上有一个单独的 TiVo。两种很棒的味道放在一起味道很棒，对吧？

32 英寸的售价为 499 美元，而 42 英寸的售价为 699 美元。这是一个 1080p/120Hz 的 LED 屏幕，具有网络功能，并支持一种叫做 Rocketboost 的东西，可以与百思买自己的声音外设配合使用。这基本上是一种追加销售和锁定的方法。

大不了？不，但一台 499 美元的 32 英寸内置 TiVo(无 DVR)电视还不算太坏，如果你手头拮据，你可以通过坚持购买一台电视和媒体播放器来节省一点钱。你会注意到雅虎！也有叫做[联网电视](https://web.archive.org/web/20230204222754/http://connectedtv.yahoo.com/)的东西，但是他们是雅虎！所以谁在乎呢？现在可以用了，大概是在返校的时候。
