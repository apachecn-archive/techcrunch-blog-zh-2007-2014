# Clover 从 Andreessen Horowitz 筹集了 580 万美元；作为一键式移动支付平台推出 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/03/08/clover-raises-5-8m-from-andreessen-horowitz-launches-as-a-one-click-mobile-payments-platform/>

一家新的移动支付初创公司正在进入市场。作为一种快速、简单的移动支付技术，Clover 今天退出了 stealth，该技术旨在让消费者在五秒钟内通过移动网站或应用程序一键结账。该公司还宣布，它已经从 Andreessen Horowitz 和 Sutter Hill Ventures 筹集了 580 万美元。

正如首席执行官布莱恩·拉姆金(Bryan Lamkin)解释的那样，Clover 正试图为移动和平板设备建立最快、最简单和最安全的支付网络。这家初创公司认为，移动结账和支付仍然存在巨大的摩擦，导致购物车被放弃，商家、零售商和开发者的转化率很低。

今天，Clover 为 Android、iPhone 和 iPad 发布了一个 SDK，允许开发者将 Clover 集成到他们的应用程序中。它是这样工作的。为了使用 Clover，消费者必须将 Clover 应用程序下载到他们的手机上。注册时，他们分享信用卡信息、手机号码和其他账单信息，然后就可以使用该平台进行一键式支付。

在开发者方面，Clover 表示，它已经将网络往返次数降至最低，以增加在不稳定的移动网络上成功交易的机会。还有简单的基于短信的初始认证，用于将 Clover 帐户锁定到一个电话号码，该平台默认采用双因素认证，使用相互认证的 SSL 和 PIN。这家初创公司表示，因为没有用户名和密码，而且因为他们将账户锁定在电话号码上，账户被接管的威胁(这是在线支付服务的常见问题)在很大程度上得以避免。

对于集成了 Clover 的开发人员来说，支付体验将只包括一个“用 Clover 支付”的结账界面，SDK 将检测那些安装了 Clover 的用户。对于现有的 Clover 用户，他们只需输入他们的电话号码和/或一个单独的移动 PIN 进行身份验证，并在五秒钟内完成结账体验。

对于那些没有 Clover 帐户的人，他们将被提示下载该应用程序并注册该服务。为了鼓励用户下载 Clover 应用程序(并补偿开发者帮助平台建立网络)，当消费者安装 Clover 时，Clover 将为他们的信用卡交易提供最高 5 美元的即时退款。为了帮助激励开发者开始使用 Clover，这家初创公司宣布了一项 10 万美元的开发者基金。

Clover 收取每笔交易 3%的手续费。早期采用者包括 Lolapps、Coupon Trade 和 Yorder。

当被问及来自 PayPal 的竞争时，Lamkin 解释说，Clover 是专为那些感受到移动支付转换痛苦的开发者设计的，PayPal 也提供一键移动支付选项。“设计围绕速度和安全性展开，以原生应用程序作为入口，”他解释道。

在移动支付领域，Clover 与 PayPal、Google 和其他公司竞争无疑是雄心勃勃的。但该公司发现了移动支付行业的一个巨大问题——简单易用。看来，能够提供快速移动结账体验(同时承诺安全性并融入消费者行为)的初创公司或公司将成为在线移动支付的真正赢家。
