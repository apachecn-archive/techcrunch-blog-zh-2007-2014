# 奇怪的双屏笔记本电脑实际上已经上市了

几年前，我遇到一位股票经纪人，他告诉我，有人应该制造一台双屏笔记本电脑，这样他就可以在度假时进行交易。虽然我对他对资本主义的献身精神感到有点不安，但我很自豪地说，他的梦想已经实现了。

几年前，我们第一次谈到了 gScreen Spacebook，现在，这家阿拉斯加公司实际上开始向大众销售他们巨大的两英寸厚的笔记本电脑。屏幕均为 17 英寸，笔记本电脑相当标准，配有光驱、三个 USB 端口和 GeForce GTS 250M GPU。英特尔酷睿 i5-560M 的起价为 2400 美元，尽管现在预购价格有 50%的折扣，降至 1200 美元。

如果你需要像前面提到的股票经纪人一样的双屏，这可能是你最容易的选择。然而，我建议雇一个夏尔巴人帮你带着这个，因为这个东西看起来像个怪物。
