# 阿什顿·库彻和麦当娜的经纪人投资小型柏林创业公司。怎么回事？

人们对柏林的一家秘密创业公司[阿门](https://web.archive.org/web/20230204230818/http://amenhq.com/)知之甚少。

我们所知道的是天气很热，柏林人都在谈论它。少数人已经获得了它的测试版，并提出了“奇怪的上瘾”之类的说法，至少根据初创公司自己的说法。

不管它是什么，消息来源告诉我们，它吸引了投资者的目光，因为它吸引了科技投资名人[阿什顿·库彻](https://web.archive.org/web/20230204230818/http://www.crunchbase.com/person/ashton-kutcher)和[盖伊·奥塞里](https://web.archive.org/web/20230204230818/http://en.wikipedia.org/wiki/Guy_Oseary)，后者恰好是麦当娜的经纪人。这是一个相当大的名气配额，所以很明显，阿门是在一些可能相当主流的东西上。
