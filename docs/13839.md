一位知情人士称， [Twitpic](https://web.archive.org/web/20230206173845/http://twitpic.com/) 正确地预测到 [Twitter 的 API 镇压](https://web.archive.org/web/20230206173845/http://blog.twitter.com/2012/09/because-you-have-more-to-show.html)将触及照片共享服务，正在积极地与至少两家潜在的收购者进行谈判，并在过去几周内收到了报价。据说创始人诺亚·埃弗雷特(Noah Everett)曾想卖掉公司，转向新项目。

Twitpic 可能希望在锤子落下之前达成交易，但今天 [The Next Web 注意到【Twitter 从其应用程序中删除了选择第三方图片共享服务的选项，扼杀了 Twitpic 的主要分销渠道。](https://web.archive.org/web/20230206173845/http://thenextweb.com/twitter/2012/09/18/twitter-now-forces-use-image-service-ios-apps/)

关于谁出价、潜在价格以及将出售哪些资产的细节很少。我们听说 Everett 正在与两家不同的公司就出售事宜进行讨论，并且可能已经与其他公司进行了接洽。我的消息来源称，Everett 不一定会出售 Twitpic，因为创始人希望探索新的想法。我们正在等待 Twitpic 对这些问题的回复。

对埃弗雷特来说，收购可能是有利可图的，因为他的公司是白手起家，2009 年实现了 150 万美元的广告销售额，其中 105 万美元归自己所有。【2010 年的一份报告称，他希望以 1000 万美元的价格出售这项服务。

一个潜在的收购者可能是 Twitter 本身。去年 6 月，微博平台[与 Photobucket](https://web.archive.org/web/20230206173845/https://techcrunch.com/2011/06/01/twitter-photos-search/) 合作，将照片直接发布到 Twitter.com，这样用户就不必将照片上传到其他地方并粘贴链接。去年 8 月，Photobucket 帮助它推出了 Twitter 画廊。

仍然有许多人认为 Twitter 的图片服务可以做得更好。我们的铁杆 Twitterer 用户 Drew Olanoff 指出，发现、图像质量、标记、标签都可以改进。与此同时，Twitter 可以为直接信息提供照片(也称为私人照片)、滤镜、特效、水印或幻灯片。收购能更好地处理这些功能的初创公司可能会给 Twitter 带来一个飞跃。此外，Twitter 显然有很大的影响力，因为它今天关闭了 Twitter pic 的移动应用程序。

其他可能的买家包括媒体公司，如 [Say Media](https://web.archive.org/web/20230206173845/http://www.crunchbase.com/company/saymedia) 可以将其添加到其资产网络中，或 [CNN](https://web.archive.org/web/20230206173845/http://www.cnn.com/2009/TECH/01/22/social.networking.news/) 可以作为公民新闻工具吸引 Twitpic:Twitpic 主持了 2009 年紧急降落在哈德逊河的飞机的著名和非常受欢迎的早期照片。UberMedia 或 Hootsuite 可以购买 Twitpic 来增强他们的 Twitter 工具。

无论如何，在经历了艰难的一年后，Twitpic 在谈判中将处于不利地位。分析公司 Sysomos 的一项研究发现，2011 年 5 月，Twitpic 托管了 45.7%的推文照片，相比之下，Yfrog 占 29.3%，Lockerz 占 17.4%。但是后来 Twitter 和 Photobucket 合作了。据 Skylines 报道，到 11 月，Twitter/Photobucket 已经成为头号照片分享服务，占据 36%的份额，而 Twitpic 下滑至 30%。

从那以后，Instagram 的使用量飙升，这意味着更多的照片是从其他应用程序发布的，而不是通过 Twitter 上传和发布，直到今天，一些用户默认使用 Twitter pic 托管。虽然该公司可能很难说服收购者支付高价，但一旦有数据显示 [Twitter 的平台收紧](https://web.archive.org/web/20230206173845/https://techcrunch.com/2012/09/18/twitter-announces-new-design-with-focus-on-profile-pages-ipad-update-on-the-today-show/)对 Twitpic 的挤压有多严重，情况可能只会变得更糟。

预计会有更多被 Twitter API 变化所鄙视的公司转向或出售。Twitpic 可能是第一批，但不会是最后一批。
