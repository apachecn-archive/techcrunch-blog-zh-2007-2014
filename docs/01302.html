你的手机就是你的麦克风:SpinVox允许用户与Twitter、脸书和Jaiku(仅限欧洲)通话

我确信，我们将会看到使用手机作为麦克风的语音应用的大幅增长。今天，在西班牙举行的世界移动通信大会上，总部位于伦敦的初创公司SpinVox 宣布了其语音转文本服务的一项新功能，该功能允许用户拨打Twitters或向脸书或Jaiku发送信息。

打电话比发短信容易吗？看你有什么样的手机，你是什么样的人。但是口述录音机现在是社交的。这至少会吸引那些喜欢用迷你录音机录笔记的人。他们可以与世界分享这些想法，计算机将转录它们。

据我快速浏览SpinVox的网站可以看到，它的服务只在欧洲可用，而且不是免费的。但这是解决移动用户界面问题的一个显而易见的方法。对手机来说，最好的输入技术就是对着它说话。也许竞争对手SimulScribe、Jott或Yap会在美国站出来。

等等，已经有办法把手机变成任何网络应用的麦克风了。它被称为VoodoVox的MyVox API 。它是广告支持的，所以是免费的。而基于API构建的最佳应用可以赢得25000美元。
