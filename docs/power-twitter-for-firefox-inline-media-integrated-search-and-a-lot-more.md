# 强大的 Twitter:内嵌媒体、集成搜索等等

由[纳伦德拉·罗谢尔](https://web.archive.org/web/20230327002745/http://www.crunchbase.com/person/narendra-rocherolle)在 [83 度](https://web.archive.org/web/20230327002745/http://83degrees.com/)创作的火狐附加软件 [Power Twitter](https://web.archive.org/web/20230327002745/https://addons.mozilla.org/en-US/firefox/addon/9591) 的 1.0 版本刚刚发布。如果你使用 Twitter，并且对 Tweetree 这样的[服务感到兴奋，这是你想立即添加的东西。](https://web.archive.org/web/20230327002745/https://techcrunch.com/2008/12/28/actual-conversations-on-twitter-not-possible-until-twitter-lets-us/)

Power Twitter 将不方便地位于不同子域名的 Twitter 搜索移到了 Twitter 网站本身。你也可以从一个用户的 Twitter 页面上搜索这个用户的更新。Twitter 早就应该将搜索框直接集成到已登录的 Twitter 主页上。他们没有，但是力量推特解决了这个问题。

Power Twitter 还修复了@replies 功能，该功能目前只显示发给你的以@[yourusername]开头的 Twitter 消息。有了 Power Twitter，你可以看到所有包含你的用户名的消息，修复了 Twitter 上的一个大漏洞。

YouTube、Flickr 和 TwitPic 链接也被内嵌到 Twitter 消息中，这样你就不必点击离线才能看到它们。这是竞争对手 Friendfeed 做的事情，也是经常被要求的功能。

我喜欢的另一个功能是:你可以将鼠标悬停在用户图标上，查看他们最近写的几条 Twitter 消息。这有助于将当前信息放入上下文中理解对话。

其他功能:你可以点击右边的脸书标签，直接在 Twitter 上看到你脸书朋友的更新。

总之，*这是修复 Twitter* 的方法，直接通过用户界面，而不是从用户会忘记去的第三方网站。
