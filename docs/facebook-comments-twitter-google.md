# 有了脸书的评论，Twitter 和谷歌的登录就不翼而飞了

鉴于全球已有 6 亿人使用这个身份，[脸书进军出版商评论空间](https://web.archive.org/web/20230205014536/https://techcrunch.com/2011/03/01/facebook-rolls-out-overhauled-comments-system-try-them-now-on-techcrunch/)无疑将是一个非常非常强大的举动。事实上，雅虎也参与了此次发布，这放大了这种力量。但它本可以更强大。

脸书最初[计划](https://web.archive.org/web/20230205014536/https://techcrunch.com/2011/01/31/facebook-comments-2/)也使用 Twitter 和谷歌的登录信息作为评论系统。但据知情人士透露，两者都在发射前的最后几个小时被取消，原因不明。事实上，对于那些在过去几个月里一直关注这个领域的人来说，选项消失的原因很明显:所有参与者都不喜欢对方。

好吧，也许这有点极端。更好的说法是，双方都不希望对方利用对手的社交网络来支持自己的社交产品。去年，当 Twitter [试图推出一个新的脸书好友导入工具](https://web.archive.org/web/20230205014536/https://techcrunch.com/2010/06/23/twitter-facebook-friends-linkedin/)，而[脸书在几分钟内就屏蔽了它](https://web.archive.org/web/20230205014536/https://techcrunch.com/2010/06/23/facebook-blocks-twitter/)，我们就看到了这一点。如果脸书试图使用 Twitter 的公共 API 为脸书评论推出 Twitter 登录，你可以打赌这个恩惠会很快得到回报。

然后是脸书和谷歌。是啊——同样的交易，[但是](https://web.archive.org/web/20230205014536/https://techcrunch.com/2010/11/04/facebook-google-contacts/) [更差](https://web.archive.org/web/20230205014536/https://techcrunch.com/2010/11/09/googles-response-to-facebooks-response-to-googles-facebook-api-ban/)。事实上，这种情况已经持续了很多年。并且它[保持](https://web.archive.org/web/20230205014536/https://techcrunch.com/2010/11/10/google-gets-feisty-kicks-data-portability-fight-with-facebook-up-a-notch/) [继续](https://web.archive.org/web/20230205014536/https://techcrunch.com/2011/02/22/google-android-facebook-contacts/)。

从用户的角度来看，这太糟糕了。拥有在一个系统下登录所有这些的选项将会很棒。当然，从像 [Disqus](https://web.archive.org/web/20230205014536/http://disqus.com/) 这样的初创公司的角度来看，这场社交图战争无疑是受欢迎的，因为他们是一个中立的第三方，可以免费使用所有的服务。

对于他们在这场战斗中的角色，脸书只能说:

> 这是自 2009 年 2 月以来广泛使用的评论产品的更新。作为更新的一部分，我们增加了雅虎！作为第三方登录，并希望在未来增加额外的供应商。

谷歌拒绝置评。Twitter 尚未回复我们的置评请求。

**更新**:以下是推特的评论:

> 我们不参与脸书在网站上留下评论的新系统。我们将继续与脸书就合作方式进行对话。
