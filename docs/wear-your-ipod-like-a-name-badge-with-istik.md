# 用 iStik 像佩戴姓名牌一样佩戴您的 iPod

[![](img/f84b7c16c7b05e476fd01cb2a334d8d0.png "istik")](https://web.archive.org/web/20230315100830/http://old.crunchgear.com/wp-content/uploads/2008/08/istik.jpg)

用 iPod 做活动的时候，有时候很难找到一个好的地方放。你的口袋是不好的，因为它可能会掉出来，如果活动涉及运行或跳跃。另一个选择是一个臂带，我觉得不舒服和分散注意力。

现在有了一个新的选择，叫做 iStik。这是一个塑料外壳，背面有四个磁铁。它的工作原理就像那些磁性姓名牌一样。有了 iPod Nano(第二代或第三代)上的外壳，你就可以把它贴在衣服上的任何地方。

唯一让我担心的是磁铁部分。我知道当我戴着名牌时，它有时会掉下来。我想如果这个磁铁更强，这可能不会发生。如果失败了，他们应该把名字改成 iSuk。

如果你感兴趣，你可以在 [iStik](https://web.archive.org/web/20230315100830/http://www.goistik.com/) 买到这个坏男孩，第二代 24.95 美元，第三代 26.95 美元。
