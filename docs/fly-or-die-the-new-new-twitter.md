# 不飞则亡:新的 Twitter

如果你没有注意到，这是一个全新的 Twitter 世界，我们只是生活在其中。新的 Twitter 界面非常“appy”，尽管桌面应用糟糕透顶，但这个网站非常强大和有趣。本周，Erick 和我仔细观察了这一进步，并评估了它在我们生活中的价值。

改变网络界面是一件大事，似乎公司不再害怕把事情搞混了。同样有趣的是，随着越来越多的推特用户离开桌面，转向移动设备，推特正在走向平板风格的界面。
