# 法国的 Netatmo 筹集了 580 万美元来扩展其联网气象站的覆盖范围

【YouTube http://www.youtube.com/watch?v=jcvCGbT37CU]

总部位于巴黎的物联网初创公司 Netatmo 刚刚完成了一轮€450 万欧元(合 580 万美元)的融资。net atmo 是一家制造个人气象站和空气质量传感器设备(如上视频所示)的公司，可用于 Android 和 iOS 应用程序。它计划在今年下半年利用这笔资金推出新的联网设备，包括额外的室内空气模块(将于本月宣布)，以及雨量计和风速计。

这一轮投资者——net atmo 的第一笔外部资金——包括 [Iris Capital](https://web.archive.org/web/20221209234835/http://www.crunchbase.com/financial-organization/iris-capital) 、[FSN·PME](https://web.archive.org/web/20221209234835/http://www.cdcentreprises.fr/portefeuille/fiche/fonds_directs/f332/fsn_pme.php)，这是法国国家数字社会基金，以及威望迪公司和翠丰公司的非执行董事、苹果欧洲、中东、印度和非洲前副总裁&帕斯卡尔·卡格尼。

Netatmo 去年秋天推出了以消费者为中心的气象站监控设备。该设备允许用户跟踪室外天气条件和室内环境条件，如空气质量和二氧化碳水平，并通过相应的应用程序监控和绘制数据。

尽管 Netatmo 尚未公布设备销售数据，但它表示，其气象站目前正在监测超过 105 个国家的环境。Netatmo 首席执行官弗雷德·波特在一份声明中指出:“在上市几个月后，需求继续增长，我们正在经历销售额的大幅增长。”“我们新的金融合作伙伴将允许我们追求进一步的创新，开发新的设备，并扩大我们的分销渠道和区域，”

Netatmo 表示，它计划专注于欧洲、亚洲和美国的发展和运营，目标是随着今年业务的增长而扩大员工人数。

帕斯卡尔·卡格尼在一份声明中对这笔资金进行了评论，他补充说:“物联网是一个更加互联的数字世界崛起的下一步……由于 Netatmo 的人才团队以及将先进软件与最先进的硬件相结合的能力，这家公司将在这场革命中发挥主导作用。”
