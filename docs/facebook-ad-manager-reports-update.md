# 脸书用更加可定制的实时数据更新其广告经理报告

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/06/26/facebook-ad-manager-reports-update/>

脸书继续努力简化其广告产品:今天它宣布更新广告经理报告，这个产品的名字很好地解释了它的作用——帮助广告商跟踪和管理他们在脸书的活动。

据产品经理 David Baser 说，这些变化的目标是给广告商提供“更简单、更易用、更灵活”的工具。这似乎包括给那些广告商更多的控制权来控制他们收到的数据。

具体来说，Baser 说，脸书现在让广告商更容易看到任何日期范围内的广告、活动或整个账户的触达数据。以前，你只能查看 reach for an ad 或 a campaign，而且只能按照脸书指定的固定时间表，比如一周或一天。Bazer 说，覆盖范围尤其重要，因为这往往会推动其他大多数结果，如店内购买，这是广告商所希望的。控制时间安排是很重要的，因为它可以让广告客户立即了解特定努力的成功(比如说一个为期四天的活动，以推动周末的购买)。

此外，Baser 表示，脸书正在“在我们的报告中处处引入每次行动成本指标”，这样广告商就不必再查看不同的报告(或在 Excel 中合并这些报告)来了解他们为实现预期目标而支付的费用。它还允许广告商根据年龄、性别、国家或在脸书的位置来查看广告、活动或账户的表现。它会实时更新这些结果。

当我指出向报告中添加更多的数据可能不符合每个人对“更简单”的定义时，Baser 承认在灵活性和简单性的目标之间可能存在矛盾。但在这种情况下，他说脸书两者都做到了。

“即使现在有更多的指标，访问它们也更简单，展示也更简单，”他说。“在这种情况下，简单压倒了(呈现给广告商的)东西的数量。”

总的来说，脸书表示，这些变化应该会给广告客户带来更轻松的体验，正如[一家公司的博客文章](https://web.archive.org/web/20230224014331/http://www.facebook-studio.com/news/item/updates-to-ads-manager-reports)中概述的那样:

> 营销人员不再需要导出多个报告来获取他们想要的数据。有了这些新的报告，他们可以选择他们想要包含在报告中的脸书广告指标；按照他们想要的方式分解数据；拖放以组织列；以及保存、导出和安排其报告的交付。

Priceline.com 一直在测试新界面，脸书也引用了这一说法。Priceline 的 Kristen Jones 表示，这些变化意味着公司必须减少 50%的报告，减少查看电子表格的时间。

上周[脸书宣布重新设计其页面洞察工具](https://web.archive.org/web/20230224014331/https://techcrunch.com/2013/06/19/facebook-page-insights-update/)，所以我也很好奇这些不同的努力被协调到什么程度。Baser 说，实际上有一个产品经理和一个团队在努力确保“我们有一个简化的整体愿景”——尽管他补充说，更新将在不同的产品之间交错进行:“它们不会成为一个大的变化。”
