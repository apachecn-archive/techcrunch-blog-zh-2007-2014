# TechCrunch 上争夺奖杯的最后 7 场比赛被打乱

![](img/08858ad5ddd26bdd031ec2694d59948e.png "22")在过去的两天里，我们在旧金山举行的 [TechCrunch Disrupt](https://web.archive.org/web/20221007005242/http://disrupt.beta.techcrunch.com/2010-sf/) 大会上见证了 25 家初创公司(其中两家来自 Startup Alley)。我们的专家小组对每个问题进行了质疑和投票。现在我们进入了最后的七强，接受来自去年五月我们最后一次 Disrupt 大会的获胜者[Soluto](https://web.archive.org/web/20221007005242/http://www.soluto.com/)的 [TechCrunch Disrupt 杯](https://web.archive.org/web/20221007005242/https://beta.techcrunch.com/2010/05/23/unveiled-the-techcrunch-disrupt-cup/)。

事不宜迟，最后的七家创业公司:

今天 3:30，七人将登台面对我们最后一组评委:[鲁洛夫·博塔](Roelof%20Botha)、[罗恩·康韦](https://web.archive.org/web/20221007005242/http://www.crunchbase.com/person/ron-conway)、[詹森·高德曼](https://web.archive.org/web/20221007005242/http://www.crunchbase.com/person/jason-goldman)和[玛丽莎·梅耶尔](https://web.archive.org/web/20221007005242/http://www.crunchbase.com/person/marissa-mayer)。只能有一个。

我们还想借此机会感谢所有参与今年 Disrupt 的公司，无论是在舞台上还是在创业路上。说“每个人都赢”是老生常谈，但我们认为时间会证明你在这里的参与最终会有回报。我们对你们每个人都寄予厚望。
