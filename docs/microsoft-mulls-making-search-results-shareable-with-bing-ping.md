# 微软考虑让搜索结果与“必应”共享

[![bing-flights](img/dbf41e1571c88de2ab627b734ad202c4.png "bing-flights")](https://web.archive.org/web/20230315095322/https://techcrunch.com/wp-content/uploads/2009/09/bing-flights.jpg)

大大小小的网站都很快意识到，让某样东西像病毒一样传播的可靠方法是让它容易在脸书和推特上分享。为什么搜索结果会有所不同？事实上，通过 Twitter 或社交网络分享结果的能力正迅速成为许多实时搜索引擎的标准功能(T4)。

微软的 Bing 可能很快会加入自己的分享搜索结果的方式，名为[“Bing 和 Ping”](https://web.archive.org/web/20230315095322/http://www.bing.com/community/blogs/search/archive/2009/09/03/share-your-search-with-bing-and-ping.aspx)该功能即将进入有限的 beta 测试，并将出现在某些结果类型下，如体育比分或航班信息。底部会有小链接，让你通过推特、脸书或电子邮件分享结果。

大多数时候，搜索是一项孤独的活动。但是有时候你会遇到一些值得分享的东西，尤其是当它不仅仅是一个链接的时候。Bing 试图将不同搜索类别的信息汇集到自己的独立框中。这些当然是可以分享的，尤其是当你试图证明一个观点，赢得一场争论，或者只是在你朋友最喜欢的球队输了的时候揭他们的疮疤。
