# 休斯顿:“18 个月后，你将会看到到处都是 Dropbox 的小按钮。”

无论从哪方面来看，德鲁·休斯顿和他在 T2 的 Dropbox 团队都在 T4 取得了巨大的成功。在由 TechCrunch 编辑的《休斯顿的创始人故事》的最后一集中，Erick Schonfeld 表示，他计划通过专注于移动和其他一切来保持这一势头。

休斯顿表示，Dropbox 已经与 HTC 达成了一项新协议，根据该协议，Dropbox“将被植入数千万部(Android)手机中”，与脸书一样，Dropbox 计划在所有产品上打上自己的印记。" 18 个月后，你会看到到处都是小收纳盒按钮."

但是 Dropbox 的策略会强大到足以抵挡像 Instagram 和 Twitter 这样的热门网站吗？这些网站也存储大型用户社区提交的数字内容。或者更直白地说“五年后，我会需要 Dropbox 吗？”肖恩菲尔德问道。

休斯顿认为这两个问题的答案都是肯定的。

“我们可以帮助您从每个小仓库或小盒子中取出内容，并将它们放在一个地方，这样所有内容对您都更有用，我们可以帮助您分享这些内容，我们可以帮助您在 iPad 或电视上观看这些内容。”

我们将会看到事情如何发展。
