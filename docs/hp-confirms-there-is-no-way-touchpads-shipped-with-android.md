# 惠普证实触摸板不可能与 Android 一起发货

当 Touchpad 爱好者发现 Android 在一些惠普触摸板上运行，并吵着要源代码时，惠普开源项目负责人 Phil Robb 很生气。负责处理开源软件和公司分销的 Robb 表示，从来没有为 Touchpad 制造的 Android，它也不可能出现在 Touchpad 上，即使是工厂错误。

引用他的话说:

> “我们已经证实，惠普从未授权在惠普 TouchPad 上分发任何版本的 Android，”他告诉 coders。
> 
> “根据对我们制造流程的审查，我们认为所有触摸板设备出厂时都只安装了 WebOS 操作系统，没有触摸板设备安装了 Android 系统，即使是错误的。”

[由于一大群程序员和黑客的努力，Android 目前可以在 Touchpad 上运行](https://web.archive.org/web/20230203155749/https://techcrunch.com/2011/08/29/cyanogenmod-team-gets-android-working-on-hp-touchpad/)，但 Robb 证实惠普与这些端口几乎没有任何关系。
