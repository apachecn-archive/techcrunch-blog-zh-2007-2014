# 凯鹏华盈领投 Klout 最新一轮“3000 万美元”融资 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/03/kleiner-klout-30-million/>

在投资者中，Klout 的声誉正在[增长](https://web.archive.org/web/20221217060633/https://techcrunch.com/2011/12/29/klout-attacks-influence/)。这家初创公司为各种社交网络上超过 1 亿个公共档案提供社交信用评分，筹集了大约 3000 万美元的 C 轮融资[。Business Insider 最先报道了这笔资金。该公司拒绝透露它筹集的确切金额，但我们已经确认 3000 万美元“接近”正确的金额。](https://web.archive.org/web/20221217060633/http://www.businessinsider.com/klout-series-c-funding-rumors-2012-1)

凯鹏华盈合伙人钱启华(Chi-Hua Chien)自去年 1 月领投 Klout 850 万美元的 B 轮融资以来一直是董事会观察员，现在是董事会的正式成员，还有凯鹏华盈的另一位合伙人宾戈登(Bing Gordon)、梅菲尔德的艾伦摩根(Allen Morgan)和首席执行官乔费尔南德斯(Joe Fernandez)。IVP 和文洛克作为新投资者加入了这轮融资，之前的投资者 ff Venture Capital 和 Greycroft 也投入了更多资金。

今天，我与首席执行官乔·费尔南德斯和 Klout 的几名投资者进行了交谈，以了解该公司正在经历什么样的牵引力。由于它仍然是相当预收入，该公司衡量的一些指标包括 API 调用的增长，API 合作伙伴的数量，注册用户和索引配置文件。费尔南德斯说，该公司在 12 月的 API 调用次数超过了 100 亿次，而 2011 年 1 月和 2010 年 4 月的 API 调用次数分别为 1 亿次和几十万次。API 调用是对其数据需求的一个很好的指标，因为这是其他服务消费 Klout 分数并将其与自己的服务混合的方式。

该公司拥有超过 4000 个 API 合作伙伴，而 2010 年初只有 100 个左右。它已经索引了超过 1 亿个公共档案。几百万人已经在 Klout 上积极注册，以便将他们的各种社交资料联系在一起(并提高他们的 Klout 分数)。

ff 风险投资公司的约翰·弗兰克尔说:“这比我预期的要更早成为主流。”。“我们投资的原因是，我们相信到 2015 年左右，人们会关心自己的网络声誉，并需要工具来管理它。”

Klout 是一个有影响力的市场。它试图总结一个人在 Twitter、脸书、LinkedIn 和其他社交网络中对哪些主题有影响力或感兴趣。弗兰克尔解释说，它试图确定“这个人的思想在哪里”。“它是有价值的，也是易腐的。”人们的兴趣减弱了，他们今天对一个主题感兴趣，但下周他们可能已经转移了。Klout 试图捕捉这种转瞬即逝的兴趣和影响，并将其装入 Klout 分数中，该分数因主题而异。我在技术方面有不错的 Klout，但不太擅长烘焙。

了解某人的 Klout 在不同方面都是有价值的。营销人员希望接触不同类别的影响者，但他们也希望根据人们的兴趣进行营销。Klout 已经在尝试通过 Klout Perks 在品牌和用户之间进行调解。但是这些数据也可能对其他服务有价值。想象一下，一个易贝卖家或在 Airbnb 上提供房间的人旁边有一个 Klout 评分。随着网络世界与现实世界的冲突越来越多，对适用于整个互联网的信誉评分的需求也在增加。([社会证明](https://web.archive.org/web/20221217060633/https://techcrunch.com/2011/11/27/social-proof-why-people-like-to-follow-the-crowd/)，FTW)。

Klout 的分数远非完美，但它们正在变得更好。它们很容易理解。如果 Klout 能够真正成为主流，让普通人相信这些分数，他们就可以建立一个庞大的业务。如果[没人关心](https://web.archive.org/web/20221217060633/https://techcrunch.com/2011/10/26/nobody-gives-a-damn-about-your-klout-score/)分数，那么世界上所有的风投资金都不会帮助他们。但是人们确实在乎，即使他们不知道分数意味着什么。(是真人 [Whuffie](https://web.archive.org/web/20221217060633/http://en.wikipedia.org/wiki/Whuffie) )。现在去检查你的 Klout 分数。自从你上次做了以后，它可能已经改变了。
