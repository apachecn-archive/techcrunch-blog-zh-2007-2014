# 火狐 23 登陆极光频道:杀死闪现元素，引入大量新开发工具

Mozilla 今天将流行浏览器的测试前发布渠道 [Firefox Aurora](https://web.archive.org/web/20221206082033/http://www.mozilla.org/en-US/firefox/23.0a2/auroranotes/) 升级到第 23 版。因此，它为开发人员引入了许多新工具，这些工具将在接下来的几个月里慢慢进入稳定的发布渠道。可悲的是(也可能不是)，这也是 Firefox 的第一个版本，取消了旧的 [< blink >](https://web.archive.org/web/20221206082033/http://en.wikipedia.org/wiki/Blink_element) 元素，这是 20 世纪 90 年代可怕的 GeoCities 网站的主要元素。

在面向用户的方面，这个 Aurora 版本还包括对 Firefox 的[混合内容拦截器](https://web.archive.org/web/20221206082033/https://blog.mozilla.org/security/2013/05/16/mixed-content-blocking-in-firefox-aurora/)的支持，当网站包含 HTTP 和 HTTPs 资源时，这应该会让用户更安全一些。对于 Mac 用户来说，它包括滑动导航的新动画，以及最终对 OSX 10.7 新滚动条样式的支持。

不过，这个版本的焦点显然是在开发人员身上。Firefox Aurora 现在提供了一个新的网络监视器，可以在任何给定的页面上提供网络活动的标准瀑布时间轴视图。这些数据一直都是可用的，但是只能通过 Web 控制台，这不太容易解释。

[![Screen-Shot-2013-05-13-at-2.33.43-PM](img/fdef457f8915c8a96f1e1dc3bfab7056.png)](https://web.archive.org/web/20221206082033/https://beta.techcrunch.com/wp-content/uploads/2013/05/screen-shot-2013-05-13-at-2-33-43-pm.png)

这个版本中还有一个新的功能是远程风格编辑器，它允许开发者通过远程协议实时测试他们的 web 应用。Mozilla 表示，该编辑器应该与 Firefox for Android 23 兼容，Android 23 也将很快进入 Aurora 频道，该团队正在努力将这项技术的一些方面纳入 Firefox OS 模拟器。

其他新工具包括专用于开发人员工具的新首选项菜单，以及 SourceMap 支持的首次实现和对对象检查器的更改。
