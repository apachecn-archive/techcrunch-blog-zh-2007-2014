# Rackspace 收购了 Mailgun，这是一家 Y Combinator 初创公司，为应用开发者提供了一个创建和管理在线邮箱的 API

Rackspace 已经收购了 [Mailgun](https://web.archive.org/web/20230215112939/http://mailgun.com/) ，这是一家总部位于旧金山的 Y Combinator 初创公司，该公司开发了一种为应用程序和网站创建和管理在线电子邮件收件箱的 API。这是一个表面上听起来很简单的概念，但 Mailgun 的易用性和复杂的路由可以为 web 应用程序和网页提供许多新功能，这些功能都是通过编程完成的。交易条款没有披露。

该公司是 2011 年 winter Y Combinator 课程的一部分。当年 5 月，它获得了 110 万美元的资助。投资方包括 SV 天使、尤里·米尔纳；梅纳德·韦伯；Y Combinator 合伙人保罗·布赫海特(Paul Buchheit)，他创建了 Gmail 杰夫·罗尔斯顿，早在 1997 年就创建并推出了雅虎邮箱。

Rackspace 将向其 Rackspace 托管客户提供 Mailgun，用于将基于云的电子邮件服务集成到应用程序和网站中。首席执行官 Ev Kontsevoy 表示，该公司的六名员工将加入 Rackspace，并搬进托管服务提供商旧金山的办公室。

Rackspace 的企业发展高级副总裁帕特·马修斯(Pat Matthews)说:“令人敬畏的产品应该并将与开放的云平台相集成。“我们希望让成千上万的客户能够轻松使用 Mailgun。”

在其核心，Mailgun 提供了一个动态创建弹性邮件服务器的服务。它抽象了设置电子邮件收件箱的复杂性，简化了应用程序和网页中电子邮件的路由。该服务允许开发人员以编程方式发送、接收、解析、存储和运行电子邮件搜索查询。

对于大多数年轻的开发人员来说，创建电子邮件收件箱是一项他们不了解的古老任务:电子邮件早于互联网。从那时起，配置电子邮件的过程几乎没有变化。Mailgun 提供了一项服务，可以发挥电子邮件的价值，为年轻的开发人员提供一个他们知道如何实现的新功能 API。

例如， [Picplum](https://web.archive.org/web/20230215112939/http://picplum.com/) 使用 Mailgun 来解析客户发送到他们账户的照片。Mailgun 读取收到的电子邮件，并将其路由到适当的位置。Everyblock 使用 Mailgun 帮助向用户发送电子邮件，并跟踪所有外发邮件背后的分析。其他客户包括 37Signals、Parse 和英国《金融时报》。

Kontsevoy 表示，热门的新 web 框架, [Meteor](https://web.archive.org/web/20230215112939/http://www.meteor.com/) 正在添加 Mailgun，为开发者提供添加更多功能的能力。Mailgun 还集成到 Heroku、AppFog 和 Engine Yard 等平台。

Mailgun 按用户收费。随着用户扩大电子邮件的使用，价格会下降。价格从每千封邮件 1 美元到 10 美分不等。Sendgrid 和 CloudMailin 是两个类似于 Mailgun 的服务。

Rackspace 面临的挑战是将 Mailgun 顺利整合到其组织中，并展示电子邮件为应用开发者提供的价值。年轻一代不太理解电子邮件，但 API 是日常工作的一部分。在这方面，添加电子邮件是有意义的，只要它可以通过编程来完成。
