# Glam Media 凭借新的专注于品牌的移动广告平台 TechCrunch 挑战苹果的 iAd

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/17/glam-media-takes-on-apples-iad-with-new-brand-focused-mobile-ad-network/>

作为全球最大的出版和广告网络之一，Glam Media 今天推出了移动广告平台 GlamMobile，旨在让品牌广告商与 Glam 移动网站的读者互动。该公司还宣布推出 GlamEnable，这是一个新平台，让出版商能够为移动网络开发网站，并为 iOS 和 Android 创建本地应用程序。

作为背景，Glam 的各种垂直出版在美国每月有 9400 万独立访问者，在全球有超过 2 亿的独立访问者，在女性读者中特别受欢迎。Glam Media 拥有超过 2，500 家出版商，涵盖多个在线垂直类别，包括面向女性的 Glam.com、面向成年人的 Glam Entertainment、面向男性的 Brash.com 以及面向健康和保健的 Bliss.com。

Glam 已经运营了一个基于网络的广告服务平台 Glam Adapt，但现在正在将其触角延伸到移动领域。由于 Glam 在女性中的影响力特别大，这家媒体公司可以向这一人群投放极具针对性的、引人入胜的广告。该网络还能够根据地理位置、设备类型和移动操作系统来确定目标。

Glam Mobile 的所有广告都专注于交互式的富媒体格式(通过 GlamAdapt 提供的所有广告中超过 80%是富媒体广告)。Glam 的首席执行官 Samir Arora 表示，广告网络完全专注于品牌购买，并将与苹果的 iAd 直接竞争。GlamMobile 平台可以在苹果 iOS、谷歌 Android、Windows Mobile 和惠普 Web OS 上运行。

事实上，一些高端品牌已经签约 Glam Mobile。雷克萨斯和梅西百货都是发布会的赞助商，他们正在使用 Glam Mobile 的广告形式来吸引女性。有趣的是，我们从消息人士那里得知，Glam 在开发 Glam Mobile 的早期阶段就在寻找一些移动广告网络，包括 Greystripe 和 Smaato，并决定不进行外部收购，因为内部开发更便宜。

GlamEnable 今天还推出了一个新的自动化平台，允许内容出版商为任何 HTML5 兼容的网络浏览器和 iOS 及 Android 移动操作系统的原生应用程序创建优化版本的网站。Glam 媒体网站(Glam.com、Brash.com 和 Brash.com)以及该公司的 75 个出版合作伙伴，包括 Bag Snob、SheFinds 和 Young Black and familiable，已经使用 GlamEnable 创建了优化的移动网站。

Glam 正在为明年的首次公开募股做准备，T2 计划最早在今年秋天申请 S-1。该公司的年收入也达到了[1 亿美元。](https://web.archive.org/web/20230205035955/http://venturebeat.com/2011/04/15/glam-hits-100m-revenue-plans-ipo-as-early-as-this-fall/)

据报道，苹果 iAd 平台[正在遭受](https://web.archive.org/web/20230205035955/https://techcrunch.com/2011/02/10/mobile-insiders-say-apples-iads-are-hurting/)和[甚至降价](https://web.archive.org/web/20230205035955/http://www.techmeme.com/110707/p27#a110707p27)来吸引广告商，所以看看 Glam Mobile 如何为品牌服务应该很有趣。
