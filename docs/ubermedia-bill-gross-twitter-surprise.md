# UberMedia 首席执行官比尔·格罗斯:关闭 Twitter“让我们大吃一惊”，但“我们会改变”

![](img/19ce5e0ed39d80aed3136395ba348ff7.png)

我刚刚与 UberMedia 首席执行官兼 Idealab 创始人[比尔·格罗斯](https://web.archive.org/web/20230204215534/http://www.crunchbase.com/person/bill-gross)通了电话，此前 Twitter 因[违反 Twitter 的 API 规则](https://web.archive.org/web/20230204215534/https://techcrunch.com/2011/02/18/twitter-suspends-ubermedia-clients-ubertwitter-and-twidroyd-for-violating-policies/)暂停了 UberMedia 的两款产品 UberTwitter 和 Twidroyd。这些侵权行为涉及商标、隐私和货币化。

格罗斯告诉我，“我们刚刚与 Twitter 进行了交谈，讨论了他们提出的各种问题。这让我们很惊讶，因为他们以前没有养过它们。我们开始做出改变”

UberTwitter 将更名为 UberSocial。隐私问题与处理超过 140 个字符的推文有关。格罗斯认为，货币化问题与 Twitter 认为 UberMedia 在使用附属链接有关，但他表示事实并非如此。但他渴望遵从 Twitter 的要求，重新获得它的青睐。“不管是什么，我们都会改变它，”他说。

格罗斯表示，他与 Twitter 进行了交谈，以更好地了解问题所在，他“感到欣慰的是”,那里的一名高管告诉他，如果他让 UberMedia 的产品恢复合规，Twitter 将恢复它们。那么，Twitter 以如此公开的方式追踪 Twitter 生态系统中如此重要的一员，想要传递什么信息呢？“我认为 Twitter 的意思是，我们真的真的希望你遵守服务条款。”嗯哼。
