# 名词项目:组装一个免费的通用视觉速记技术

> 原文：<https://web.archive.org/web/http://techcrunch.com/2010/12/07/the-noun-project-assembling-a-free-and-universal-visual-shorthand/>

![](img/ae18e530139522fdcfdffecc129d033f.png "np1")
虽然我怀疑[名词项目](https://web.archive.org/web/20230204225838/http://www.thenounproject.com/)会对你的日常生活产生深远的影响，但这是他们自愿承担的一项有用的服务，我认为它值得一些关注。

这个想法很简单:制作一个清晰、简单的图形库，代表从急诊室和药物到不同种类的鸡尾酒或主菜的一切。

有人可能会说，像谷歌图片搜索这样的东西已经让这种努力变得过时了，但我认为它还是有一席之地的。世界日益缩小，当提供本地化文本不可行时，像这样的一套全面的符号可能会变得非常宝贵，例如在国际分发的包装中，或者在无法或不愿跟踪用户位置的单域服务中。网址缩写的国际符号是什么？社交网络？无麸质？苹果商店？

当然，他们大量借用已有的图标集和公共领域的符号:没有必要创造一个新的、更生动的“生物危害”符号，或者试图改进我们在每个机场看到的“带箭的行人”。但是还有很多东西没有被图标化，或者如果它们已经被简化为图标形式，它们就不容易在一个中心位置被访问。

![](img/58e21683d06172b558c8cf2e78466775.png "np2")

无论是对于标牌还是用户界面，这样的集合都很方便。我很想知道一些文化差异是如何影响这些东西的可理解性的。具有严重局部差异的概念(例如写上下和左右而不是左右)或法律和习俗不同的概念(左驾车，“V”手势)将需要专门的符号。而且必须有一些指定的地方现存的符号(像同步)，缺乏上下文，可以采取其他东西(回收，也许)。显然，这不仅仅是收集图标。

我见过像这样的设计项目来来去去，尽管这并不影响我的希望，也许这将是每个人都使用的一个(除了政府和机场，它们可能有自己的)。我猜它会永远免费，因为许可这些图形的想法，其中许多已经在使用，似乎是可疑的。他们确实有一个 [Kickstarter 页面](https://web.archive.org/web/20230204225838/http://www.kickstarter.com/projects/tnp/building-a-free-collection-of-our-worlds-visual-sy)(我应该说，他们已经达到了他们的目标)来帮助支付托管和设计的成本，所以请随意投入一两美元。

他们的网站目前似乎受到了抨击，但上面的截图中没有太多内容(尽管背景颜色调节器很酷)。看起来他们现在有大约 500 个图标，但是有很多重复的(多少个锚太多了？)，还有一些坦率地说令人费解。看来需要一点治疗了。
