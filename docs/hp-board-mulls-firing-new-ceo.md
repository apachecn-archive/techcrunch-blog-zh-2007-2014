# 惠普董事会考虑解雇新 CEO

有传言称，惠普对李艾科的公司计划不太满意，他可能会在上任不到一年后被解雇。

你会记得，李艾科的背景是 SAP 和大规模商业应用，因为当你有一把锤子时，所有东西看起来都像钉子，他认为惠普可以通过向企业销售相同的软件赚取各种现金，而[则扼杀和剥离其硬件业务的各个](https://web.archive.org/web/20230204123915/http://www.google.com/url?sa=t&source=web&cd=9&ved=0CGIQFjAI&url=http%3A%2F%2Fbeta.techcrunch.com%2F2011%2F08%2F18%2Fits-official-hp-kills-off-webos-phones-and-the-touchpad%2F&ei=wAR6ToavDsfX0QHvsJnqAg&usg=AFQjCNGLxOPUaSKos4k86DqTK5pMfp2CsQ&sig2=EewoqARCdFw_vOCPv-zN5A)。

正如[丹·弗罗默](https://web.archive.org/web/20230204123915/http://www.splatf.com/2011/09/apotheker-whitman/)所写:

> 看起来并不是每个人都对“放弃世界上最大的个人电脑业务，努力成为一家企业软件公司”的方法感到兴奋。

据《彭博》报道，易贝首席执行官梅格·惠特曼将角逐这一职位。自从李艾科掌权以来，惠普的股票已经下跌了将近一半，从 42 美元跌至 22 美元，投资者对他收购信息管理公司 Autonomy Corp. 表示了“几乎一致的反对”。
