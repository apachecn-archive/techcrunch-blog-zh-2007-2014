# 马克·博德尼克将成为 Quora 的新成年人

[![](img/2188a0786ee9ceb9dda2261ea13ea7c7.png "bodnick")](https://web.archive.org/web/20221207071344/https://beta.techcrunch.com/wp-content/uploads/2011/01/bodnick.jpg) 昨天给 Elevation 带来了坏消息，有报道称超级巨星搭档马克·博德尼克即将离开公司。今天这个故事似乎变得有趣多了。我们听到报道说，博德尼克不仅仅是去了一家更性感的新风险投资公司——他去了 Quora，成为公司的首席财务官。对于那些认为 Quora 不仅仅是下一代雅虎答案的人来说，这无疑增加了一些可信度。众所周知，博德尼克对该公司印象深刻，并在网站上花了大量时间。

随着今天早上这个故事的展开，关于博德尼克的确切头衔有一些争论，我们根据接近球员的消息来源继续更新这个故事。他的背景是金融，所以他无疑会帮助首席财务官的职责，但不清楚为什么 Quora 在其发展的这个阶段需要如此优秀的全职首席财务官。他很可能最终会成为必不可少的“成年人”,帮助 Quora 扩大规模，建立真正的业务，并给公司一个完美的发言人。博德尼克很熟悉这个角色——毕竟他的嫂子是马克·扎克伯格在脸书的第二号人物雪莉·桑德伯格。

我们已经联系了 Quora 进行确认，当我们知道的时候会添加更多的消息。在 Quora 发展的这个阶段，头衔并不那么重要——任何一个 C 级官员最终都会戴上一百万顶组织帽子。重要的是，这条消息清楚地表明，Quora 不打算在短期内出售，并且已经为建立一家更大的公司奠定了基础。显然，我们并不是唯一听到它的人。Elevation 拒绝置评，Bodnick 也没有回复我们的邮件。

关于 Elevation 的补充说明:这可能不像昨天猜测的那样，表明博德尼克认为该公司筹集第二只基金的机会注定要失败，而是表明他想加入硅谷最热门的创业公司之一。尽管这对 Elevation 无疑是一个打击，但我们知道，该基金的财务状况非常好，这主要归功于脸书估值的飙升。虽然博德尼克被认为促成了这笔交易，但他们的关系实际上可以追溯到脸书的早期，当时罗杰·麦克纳米和博诺都是天使投资人，麦克纳米是扎克伯格的非正式顾问。
