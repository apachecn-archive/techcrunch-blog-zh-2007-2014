# 长尾视频推出其开源视频播放器的新版本，支持苹果 HLS

多年来，[长尾视频](https://web.archive.org/web/20221210033938/http://www.longtailvideo.com/)一直提供其开源 [JW 播放器](https://web.archive.org/web/20221210033938/http://www.longtailvideo.com/jw-player/)作为企业的免费选择，这些企业不想把他们的视频放在 YouTube 上或向 Brightcove 或 Ooyala 等 SaaS 的内容管理播放器付费。但它没有他们可能喜欢的所有特性和功能——例如，内置对苹果 HTTP 直播(HLS)自适应比特率协议的支持。JW 播放器的新版本 JW6 试图纠正这一点，同时也改善了用户界面，加快了播放速度。

JW6 最大的变化，至少从表面上看，是播放器 UI 本身。自从 2005 年发布以来，这款播放器并没有太大的变化，看起来也是如此。新版本遵循了现代无铬视频播放器的趋势，具有更大的控件，使用户更容易浏览视频或打开和关闭功能。长尾团队还为时间滑块、音量、质量和标题菜单增加了鼠标悬停显示。

但是球队也在幕后对球员做了很多改变。他们重写了代码以使它更快，特别是在 HTML5 模式下，它利用了现代浏览器的改进，如 HTML5 全屏 API 和 CSS3 过渡。该播放器还得到增强，可以根据哪个浏览器试图访问它，在 HTML5 和 Flash 播放之间自动切换。虽然它将默认为 HTML5 播放，但如果不支持 HTML5，嵌入式播放器将在 Flash 中播放。

此次更新还包括对苹果 HLS 的支持，使出版商首次能够利用 iOS 设备上的自适应比特率功能。当观众在 JW 播放器中观看视频时，特别是在低带宽的情况下，HLS 将提供更流畅的播放和更高的质量。这将允许他们通过一个嵌入式设备访问桌面和 iOS 设备。

长尾视频还简化了其支持的视频和音频格式。在视频方面，它支持 MP4、FLV 和 WebM 格式，在音频方面支持 AAC、MP3 和 Vorbis。它还支持 HLS 和 RTMP 流媒体，以及 YouTube 视频渲染。新播放器还允许观众选择他们喜欢的视频质量，无论格式如何都可以。

最终的改变不是来自播放器本身，而是长尾视频的盈利方式。虽然 JW Player 有免费版本，但该公司也提供付费版本和各种插件。它取消了插件系统，将许多最流行的插件捆绑到不同版本的播放器中。

JW6 将会有一个免费版，它将会提供基本的功能集。但它也有几个付费版本。专业版，单个网站嵌入费用为 99 美元，允许用户禁用徽标或添加自己的徽标，并自定义右键菜单。还有一个额外的优惠，可以在多达 10 个商业网站上使用。它还支持苹果 HLS 和社交分享，并显示相关视频。最后，还有一个广告版，用户每月支付 49 美元，它拥有与 Premium 相同的功能，但也支持许多广告平台，供出版商从中获利。
