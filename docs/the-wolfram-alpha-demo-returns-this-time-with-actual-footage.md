# Wolfram Alpha 演示回来了，这次是服务的实际片段

【YouTube http://www.youtube.com/watch?v=hYhLsQPHNas&hl=en&fs=1]

昨天，几天的大肆宣传在 Wolfram Alpha 搜索引擎的发布上达到高潮，它在哈佛大学伯克曼中心的一次演示上首次亮相。不幸的是，由此产生的视频片段变成了沮丧(或无聊)的[练习。请注意，这并不是因为它无趣，而是因为我们看不到创造者](https://web.archive.org/web/20221208235134/http://www.beta.techcrunch.com/2009/04/29/harvard-posts-the-wolfram-alpha-preview-video-without-a-single-shot-of-the-service/)[史蒂夫·沃尔夫勒姆](https://web.archive.org/web/20221208235134/http://www.crunchbase.com/person/stephen-wolfram)所谈论的明显创新的搜索引擎。

显然，有人对媒体的挤压改变了看法，因为伯克曼中心发布了一个新版本的视频(或至少 10 分钟的视频)，这次是服务的镜头。

视频的这一部分包括许多查询，包括计算法国的国内生产总值(这导致一段时间内国内生产总值的图表)和欧洲的互联网用户数量，生成相关的直方图。在查询马萨诸塞州列克星敦的天气时，搜索引擎生成了一个随时间变化的温度图，包括对近期的预测。一个每小时 17 美元的查询产生了一个不同比例的相同转换表(尽管在视频中很难辨认)。

对于理科生和研究人员来说，这个引擎看起来棒极了。对 5M H2SO4 的搜索产生了一个计算，确定每种试剂需要多少来制造它(基本化学，但仍然很方便)。看起来它可以处理各种类似的计算，比如确定一种化学物质在不同压力和温度条件下的当前状态，包括相图。

下一个查询展示了引擎如何用于健康问题。搜索低密度脂蛋白 180(衡量你的“坏”胆固醇)会显示一个图表，描述这个数字相对于一般人群的情况。Wolfram 随后展示了该引擎如何针对不同的年龄范围执行相同的任务。

总而言之，我印象深刻。当然，这看起来不像是谷歌的替代品，至少对于一般的搜索词或新闻来说是这样。正如丹尼·沙利文[指出的](https://web.archive.org/web/20221208235134/http://searchengineland.com/overhype-your-search-engine-18076)，这些术语可能是精心挑选的，因为它们在引擎上表现良好。但我仍然很兴奋，并期待着自己尝试一下。
