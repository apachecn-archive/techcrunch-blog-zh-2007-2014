# Jaja:巧妙变形的压敏 iPad 手写笔 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/31/jaja-a-pressure-sensitive-ipad-stylus-with-a-clever-twist/>

> “谁想要手写笔？”

*–史蒂夫·乔布斯，MacWorld 2007*

事实证明，很多人都是这样。提醒你，不是为了在用户界面中导航——Steve(等人)在这一点上是绝对正确的。但是对于世界上那些想用 iPhone 或 iPad 作为他们新奇的便携式玻璃画布的艺术家来说，手写笔随时都会胜过手指。

然而，各种各样的 iPad 触控笔普遍缺乏艺术家们对他们的数字笔所期望的东西:压力敏感度。一个名为“jaja”的[成功启动的项目](https://web.archive.org/web/20221224173233/http://www.kickstarter.com/projects/jonatherton/jaja-worlds-first-pressure-sensitive-stylus-for-ip)自称为“世界上第一个用于 iPad 的压力感应手写笔”,看起来要改变这一点了。

最后一点好处是:他们试图不使用 WiFi 或蓝牙进行无线连接。那么它是如何与 iPad 进行通信的呢？*音*。

你可能知道，这个世界充满了我们听不到的声音。一般来说，人耳能听到 20 到 20，000 赫兹之间的声音。据 jaja 背后的人说，iPad 的麦克风可以接收远远超过这一频率的频率，在高端给他们一点空间，以声音的形式传递信号，而不会让你周围的人发疯。(但是狗呢？不会有人想到狗吗？！)

除了压力敏感性，jaja 还将有两个内置按钮用作热键(例如，用于切换画笔，或一键撤销/重做功能)。)

当然，任何你希望使用它的 iPad 应用程序(除了基本的、非压力敏感的手写笔功能)都需要包含对 jaja 正在开发的 SDK 的支持。你最喜欢的绘图应用程序现在可能没有使用麦克风做任何事情，更不用说解析出高音调的抱怨了。

有一件事我很好奇:环境声音呢？以飞机为例。不依赖 WiFi/蓝牙，jaja 可以在飞机上安全使用。但是飞机引擎会产生大量荒谬的声音——大部分在更高的范围内。这会造成干扰吗？

无论如何，jaja 已经远远超过了它在 Kickstarter 上的最初目标[25，000 美元](https://web.archive.org/web/20221224173233/http://www.kickstarter.com/projects/jonatherton/jaja-worlds-first-pressure-sensitive-stylus-for-ip)，所以它进入现实世界的几率相当大。40 美元可以获得前 500 枚 jajas 中的一枚，其中 471 枚已经被抢购一空。
