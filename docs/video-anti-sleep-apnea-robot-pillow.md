# 视频:防睡眠呼吸暂停机器人枕头

睡眠呼吸暂停对于患有这种疾病的人来说是一个大问题，但是日本正在提供帮助。东京[早稻田大学](https://web.archive.org/web/20221006115652/http://www.waseda.jp/top/index-e.html)的一组研究人员开发了一种机器人枕头，它可以监测病人的睡眠，并在检测到睡眠周期不规律时通过触摸病人的面部或颈部来帮助他们。

这个枕头被称为 Jukusui-kun，形状像一只泰迪熊。仅在日本，估计约有 200 万人患有睡眠呼吸暂停症(总人口为 1.28 亿)，这意味着目标群体相当大。

[![](img/83eeedbe2a36193174485661be874ecd.png "robot pillow 2") ](https://web.archive.org/web/20221006115652/https://beta.techcrunch.com/wp-content/uploads/2011/12/robot-pillow-2.jpg) [ ![](img/e40d706483553fe8acc645f82fe0a25f.png "robot pillow 3")](https://web.archive.org/web/20221006115652/https://beta.techcrunch.com/wp-content/uploads/2011/12/robot-pillow-3.jpg)

Jukusui-kun 研究小组的 Teruko Otani 解释了枕头的工作原理:

> 在这个系统中，戴在手指上的脉搏血氧计测量血液中的氧浓度。此外，泰迪熊体内的麦克风会接收到打鼾的声音。来自这两个设备的数据由 PC 评估，并反映在发送到致动器的信号中，因此致动器的移动可以根据人的情况而变化。致动器的运动帮助患者在睡眠中翻身，如果他们的呼吸严重中断，它会唤醒患者。
