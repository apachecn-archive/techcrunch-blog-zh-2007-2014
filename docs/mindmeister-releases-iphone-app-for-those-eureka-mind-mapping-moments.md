# MindMeister 为那些灵光乍现的思维导图时刻发布了 iPhone 应用程序

MindMeister 是一个在线思维导图工具，允许你创建、分享和合作思维导图。今天在 App Store 中可用的 MindMeister iPhone 应用程序最初是由 Ultravague 为 Mindmaker iPhone 应用程序开发的，然后在今年 1 月被[mind meister](https://web.archive.org/web/20221209174536/http://www.beta.techcrunch.com/2009/01/26/another-iphone-app-sold-mindmaker-goes-to-mindmeister/)以未披露的金额收购。

德国-奥地利创业公司 MindMeister 认为，IPhone 应用程序将真正让他们在开发自己的应用程序方面领先一步。看起来确实如此，因为有一些关键的区别。也就是说，这款新应用支持分享思维导图，还支持 mindmeister 的“geistesblitz”或“脑波”功能，允许你将你在浴室里得到的那些绝妙的灵感插入到 mindmeister 网站上的默认思维导图中。非常适合 iPhone。

旧的 mindmaker iPhone 应用程序用户将获得这个 mindmeister 应用程序的免费更新以及所有新功能，如果他们开设了一个 MindMeister 帐户，他们将获得 mindmeister 平台的在线和离线同步功能。

MindMeister 的联合创始人 Michael Hollauf 告诉我们，到目前为止，用户的接受度很好，他们有新的编辑功能，如为应用程序计划的可折叠节点，他认为未来在 android，Symbian 或 blackberry 上的任何平台开发都将取决于 iPhone 应用程序的表现。他确实认为 MindMeister 是一个商业协作工具，因此黑莓应用程序可能也是一个好主意——该公司有兴趣听取那些希望将 MindMeister 集成到他们的应用程序中的人的意见。

该公司运营一种免费增值模式，更高级的功能，如导出到其他思维导图应用程序，需要至少一个专业帐户。他们目前拥有超过 300，000 名用户，其中三分之一来自北美，另外三分之一来自欧洲，但最后三分之一来自亚洲，尤其是中国，这是一个他们通过合作伙伴网络进入的增长型市场。该公司最初由一轮未披露的天使投资资助。

在这个领域还有其他的应用，最著名的是 Mindjet 的桌面应用 MindManager，它已经存在很久了。在网上我们有 [bubbl.us](https://web.archive.org/web/20221209174536/http://bubbl.us/) 和 [mindomo](https://web.archive.org/web/20221209174536/http://www.mindomo.com/) 。MindMeister 有一些优势，例如使用 AJAX 而不是 flash，因此不需要播放器，事实上，它们提供了在思维导图上真正合作的能力，而不是仅仅分享它们，在桌面上导出到 MindManager 是有意义的，并且非常容易使用。
