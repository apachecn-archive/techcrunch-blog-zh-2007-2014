# 通往 TwitBook 之路:脸书的新分享功能是防御性的还是先发制人的？

***编者按**:这篇客座博文由 MySpace 前总裁、创始人、第一个朋友[汤姆·安德森](https://web.archive.org/web/20230204225959/http://www.crunchbase.com/person/tom-anderson)撰写。你现在可以在[脸书](https://web.archive.org/web/20230204225959/http://www.facebook.com/myspacetom)、[推特](https://web.archive.org/web/20230204225959/http://twitter.com/#!/myspacetom)和[谷歌+](https://web.archive.org/web/20230204225959/https://plus.google.com/112063946124358686266) 上找到汤姆。*

几周前，我问 Google+是否让脸书处于守势。在昨天的[脸书公告](https://web.archive.org/web/20230204225959/http://tcrn.ch/n6bQ0S)之后，我不认为有任何问题。许多记者将承诺的改变描述为给予用户“对隐私的新控制”，并暗示脸书只是给我们一个像 G+一样更加隐私的机会。有些人甚至认为，这种对隐私的新关注是脸书以前的哲学的“大转变”。我想说脸书根本没有改变他们对隐私的态度。我的直觉告诉我——这确实是一种直觉，这里没有内部消息——宣布的变化意义重大，因为脸书实际上是在让用户为更大的事情做准备。脸书现在比以往任何时候都更加积极，但这不仅仅是因为 G+的缘故。也是因为推特。实际上，这是因为 Twitter 为 G+的未来提供了一个可能的愿景。听起来很困惑？让我解释一下。

G+基本上是脸书推特的组合。许多记者似乎都忽略了这个非常基本的想法。他们认为 G+是脸书的竞争对手，很难将用户从他们的朋友网络中拉出来。从我的角度来看(只有谷歌可以通过查看他们的使用日志告诉我们)，更多的人对 G+类似 Twitter 的公开分享感兴趣，而不是朋友之间的私人分享。换句话说，G+的流行是由类似 Twitter 的活动推动的，而不是类似脸书的活动。G+目前大概有 3000 万用户。

那没什么可轻视的。但是 Twitter，当然，数量级更大，并且仍在疯狂增长。脸书担心的是，G+可能会变得像 Twitter 一样大，然后 G+ *可能会*开始逐渐将用户拉进更私人的行为，这通常发生在脸书。对脸书来说，G+可能的未来比 Twitter 的现在或未来更可怕，因为至少人们认为 Twitter 永远不会尝试做“隐私”的事情。

因此，我的猜测是，脸书正在向用户宣布对“隐私功能”的更多前端和中心访问，因为脸书实际上正计划在类似 Twitter 的公共领域进行更大的推动。想想这个:当被问及他认为昨天发布的最重要的部分是什么时，脸书的产品副总裁克里斯·考克斯[重申](https://web.archive.org/web/20230204225959/http://dthin.gs/r2UGVJ)脸书是一个“分享工具”，考虑与你分享的观众是很重要的。他没有谈隐私，除了说在网上没有什么能真正保证你的隐私。(对于扎克伯格来说，这是关于隐私的非常标准的说法。)

实际上，这对脸书意味着什么？我认为这意味着我们将会看到脸书在公共方面以及在私人和群体分享方面变得更加“被动”。我认为这意味着我们将看到脸书将用户推向公共和“大众”分享。更具体地说，我认为我们会看到“粉丝页面”和新闻订阅(甚至可能是“双重新闻订阅”)的变化，这样脸书就可以让用户从他们在脸书关注的公众人物、品牌和兴趣来源中获得“类似 Twitter”的体验。

脸书很早就理解了 Twitter 的力量，并融入了 Twitter 的精髓——“状态更新”——这种方式给脸书带来了巨大的变化。它融合了 Twitter 的元素，但没有偏离其核心。渐渐地，Twitter 的用处变得越来越小&不再是关注你的朋友，而是关注你的兴趣。推特行为是从“你在做什么？”到“发生了什么？”在这种情况下，Twitter 对标准的新闻把关者构成了可信的威胁。我的感觉是，这是扎克伯格一直希望他的“新闻订阅”所做的——脸书不应该*只是*关于你的朋友在做什么，但它也应该是一个更广泛意义上的“分享工具”。脸书还没有像 Twitter 或 Google+ now 那样创建一个真正给你个性化新闻的“新闻订阅源”。

那么 G+让脸书处于守势了吗？是的，但这仅仅是因为 G+是 Twitter &脸书和脸书的结合，在某种程度上，可能一直有点嫉妒 Twitter。现在，如果脸书能够进化出 TwitBook，那么他们将在自己的游戏中击败 G+,在 G+真正起飞之前，创造出第一个广受欢迎的一体化公私“社交网络”。这自然引出了一个问题:“这种公私结合的服务是人们真正想要的吗？”只有时间能证明一切。
