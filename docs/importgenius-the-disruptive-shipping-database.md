# 进口天才:颠覆性运输数据库技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/05/28/importgenius-the-disruptive-shipping-database/>

[![](img/1116979472a81c6517458fe60881873c.png)](https://web.archive.org/web/20221209024311/http://www.crunchbase.com/company/importgenius)

如果您可以监控任何公司在通关当天进口的货物和原材料，该怎么办？乍看之下，亚利桑那的初创公司 [ImportGenius](https://web.archive.org/web/20221209024311/http://www.importgenius.com/) 听起来并不特别令人兴奋:它是一个跟踪美国海运码头进出口活动的网站。但是根据 ImportGenius 的总经理瑞安·彼得森所说，这是同类服务中的第一项，也是一项不缺乏潜在应用的服务。

每艘进出美国的船只都必须提交记录其货物的运输记录。这些文件大多是公开记录的——如果你愿意，你可以自己查阅。但是每年有数百万这样的文档被提交，没有可搜索的索引，使得数据几乎没有用处。

直到现在。ImportGenius 已许可从多个来源(以及美国海关等免费来源)导入/导出数据，并将其添加到每天更新的数据库中。按月付费(标准接入 99 美元，优质接入 250 美元)，用户可以搜索数据，确定标准，包括货物类别、涉及的公司和原产地。

该网站六个月前首次推出时，ImportGenius 主要是一个小型进口商/出口商的工具，他们想了解他们的竞争对手在做什么。竞争对手可以密切关注哪些工厂正在被使用，或者某一特定产品有多少需求。虽然这个市场已经起飞，但数据还有许多其他可能的(也是更令人兴奋的)用途。

本周早些时候，ImportGenius [分析了苹果目前的发货记录，推断今年春天已经有异常大量的“电子计算机”(苹果从未使用过这一分类)到货。这并不奇怪，但它说明了一个很好的问题:无论一家公司对其产品有多保密，它仍然必须保持准确的运输记录。](https://web.archive.org/web/20221209024311/http://www.importgenius.com/blog/iphone)

股票分析师一直在密切关注这些数据——如果他们能够确定收入和进口之间的趋势，他们就有可能在公司季度新闻发布之前很久就评估公司的收益(这显然会彻底改变股市)。其他公司一直用它来加强版权保护。这些公司不是简单地打击假冒商品的犯罪商人，而是追捕商品的原产地工厂。这些数据也被用于许多诉讼中。

我们应该注意到，这种数据对安全目的没有用处——美国海关总署投资了 150 亿美元的跟踪系统，这(希望)让这个系统望尘莫及。
