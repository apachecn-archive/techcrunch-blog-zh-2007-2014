# Sphero Sports 新车身，接近正式发布

Orbotix 在过去的几个月里一直在努力，这表明:他们在 4 月份获得了 500 万美元的[B 轮投资](https://web.archive.org/web/20230204120751/https://techcrunch.com/2011/04/07/orbotix-scores-5-million-because-you-all-want-smartphone-controlled-robotic-balls/)，现在他们已经宣布粉丝们不用再等很久就可以购买他们的[机器人球。](https://web.archive.org/web/20230204120751/http://www.orbotix.com/events/the-countdown-begins-as-we-prepare-for-launch/)

没错，机器人球。对于陌生人来说，这家总部位于科罗拉多州的公司一直在努力开发 Sphero，这是一种可以通过智能手机控制的小型机器人球。可控球的应用似乎有点有限，但 Orbotix 幻想自己创造了一种“混合现实”体验，游戏既可以在手机上进行，也可以在现实世界中进行。

最终的 Sphero 生产设计与我们过去看到的略有不同，半透明的白色底盘取代了原来的底盘。Spheros 还将配备一个漂亮的感应充电器:只需在两轮办公室高尔夫球赛之间将球扔进坞站，你就能获得金牌。

虽然 130 美元的价格标签可能会使 Sphero 摆脱“冲动购买”的状态，但开放 API 的加入意味着人们可以开发利用 Sphero 硬件的新游戏和应用程序。如果可破解机器人球的概念激起了你的兴趣，Sphero 目前已经开始预购，但那些持观望态度的人可以观看我们在 CES 上的实践视频，想象一下其中的乐趣。
