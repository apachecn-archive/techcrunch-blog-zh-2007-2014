> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/06/30/tictacti-secures-funding-to-take-on-heyzap-in-game-aggregation/>

# Tictacti 获得了在游戏聚合领域与 Heyzap 竞争的资金

许多公司都想搭上 flash 游戏的潮流，但却没有核心竞争力。以色列推出了一种新的白色标签解决方案，名为“T2”。这家初创公司现在已经筹集了 50 万美元，由基马风险投资公司和其他未披露身份的投资者牵头。迄今为止，这家初创公司已经吸引了 160 万美元的资金。

Tictacti 计划与现有的玩家 [Mochi Media](https://web.archive.org/web/20230203140220/http://mochimedia.com/) 和 [heyzap](https://web.archive.org/web/20230203140220/http://heyzap.com/) 竞争，借助一个平台，游戏开发者可以通过一个 API/SDK 或基于 Flash 的游戏在移动、网络或 IPTV 上赚钱。但是这里关键的一点是，他们不需要使用 API 就可以做到。

最终的想法是成为一个大型游戏发行商聚合商，涵盖网络、社交网络、移动设备、游戏机和 IPTV，并为开发者提供一站式解决方案。

到目前为止，耐克、星巴克、宝洁和其他公司都在使用 TicTacTi，加载他们的富媒体内容，从而获得广告收入。
