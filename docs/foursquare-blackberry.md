# Foursquare 为签到大战增加了另一个武器:黑莓客户端

有一段时间，玩基于位置的 Foursquare 的唯一方式是通过它的 iPhone [应用程序](https://web.archive.org/web/20230315060635/https://techcrunch.com/2009/03/18/sxsw-foursquare-scores-despite-its-flaws/)或稀少的移动网站。然后是安卓应用程序[，向一群全新的用户开放游戏。现在，黑莓用户得到了喜爱。](https://web.archive.org/web/20230315060635/https://techcrunch.com/2009/09/02/android-now-plays-foursquare-too/)

这项服务在数百名用户进行了数周的私人测试后，今天正式在黑莓手机上推出。具体来说，该应用程序可以在所有 8000 和 9000 系列设备上运行(任何带轨迹球的黑莓手机)，也可以在触摸屏黑莓风暴上运行，但 Foursquare 指出，为该设备打造的特殊版本也在开发中。

正如你在截图中看到的，该应用程序与其他平台上的 Foursquare 应用程序非常相似。虽然没有最新版本的 iPhone 漂亮，但它完成了任务。我们自己的 Leena Rao 已经是测试版的测试者之一，并且在羡慕了几个月的 iPhone 之后，兴奋地说能够使用这项服务了。也就是说，她确实注意到了一些错误，但不能确定这是黑莓的问题还是 Foursquare 的问题。许多测试人员在论坛上指出了位置感应的问题，但随着上周发布的最新更新(“发布候选”)版本，该服务对 UX 进行了一些调整，试图使其更明显地找到你想要登记的实际地点。

随着 Foursquare 的持续增长，黑莓的推出对他们来说是一个重要的事件。Windows Mobile 和 Palm Pre 版本也已经由使用 four square API 的独立开发者开发了一段时间。一旦它在所有的移动平台上运行，它将对 Gowalla 和 Yelp 应用程序有一个非常强大的防御，Gowalla 目前只在 iPhone 上运行，但 T2 在 Android 上运行，Yelp 应用程序上周刚刚添加了签到功能，这是 Foursquare 的关键。这仍然只能在 iPhone 应用程序上使用，但它有大约 125 万用户——远远超过 Foursquare 的总人数(大约 20 万)。就在昨天，Foursquare 还确认聘请了一位首席运营官(他们称其为总经理)。
