# 微软开始认真对待网络办公技术

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/11/09/microsoft-starts-taking-office-on-the-web-seriously/>

本周早些时候，微软更新了其 Office 365 套件，增加了一些新功能和许可条款。总体而言，此次更新与微软对其面向消费者和企业的基于订阅的 Office 版本所做的其他 100 项更改非常一致，但有一项功能非常突出。从本周开始，所有 Office Web 应用程序都将具备实时协作编辑功能，这一功能以前只适用于 [Excel](https://web.archive.org/web/20221206151527/http://blogs.office.com/b/microsoft-excel/archive/2010/01/21/collaborative-editing-using-excel-web-app.aspx) 和 [OneNote](https://web.archive.org/web/20221206151527/http://office.microsoft.com/en-us/web-apps-help/introduction-to-onenote-web-app-HA010378339.aspx) web 应用程序。

这本身是一个有趣的举动，但在谈到这次更新时，微软 Office 部门的企业副总裁约翰·凯斯告诉我，这也标志着微软内部对网络应用的一种新的思维方式。

不过，让我们先退一步。如果您不熟悉 Office Web Apps，只需看看 SkyDrive 并上传一个 Word、Excel 或 PowerPoint 文档。一旦文件上线，你可以在 SkyDrive 中查看它，但最重要的是，你还可以在微软旗舰生产力应用程序的轻量级版本中编辑它。

Office Web Apps 于三年前推出，但大部分都不为人所知，尽管它们比谷歌的产品功能更全面。鉴于微软对文件格式的控制，它在显示和保存文件时也明显更好，不会弄乱任何格式。它们还使用与常规桌面办公应用相同的功能区菜单，因此常规办公用户应该从一开始就使用它们。

![Print](img/6a46b3265052a78dba84d931a746de7a.png)

然而，到目前为止，这些网络应用基本上是客户的伙伴，微软几乎没有做任何推广工作。看起来这种情况正在改变。正如 Case 告诉我的，微软现在终于开始将 Office Web 应用程序视为独立的服务。Case 没有透露微软是否考虑转向免费增值模式，并在免费服务之外提供一揽子付费服务。不过，如果微软认为网络应用是 Office 的独立版本——从长远来看，它们可能会成为唯一的版本——如果这是微软正在考虑的事情，我也不会感到惊讶。

微软一直非常关注 Office 桌面客户端。毕竟，所有的钱都是在那里赚的。现在它已经转向订阅模式，给 Office Web 应用程序添加高级功能似乎是合乎逻辑的下一步。

凯斯告诉我，微软计划在未来几个月加大对办公网络应用的投资，所以我们可能会听到更多关于该公司为这些服务所做工作的消息。

如果微软开始在这一领域加大投资，这也将对谷歌构成更大的挑战，谷歌的在线办公套件除了非常基本的编辑之外，仍然不适合做任何事情。然而，谷歌一直拥有的一个优势是，它提供协作编辑。这种优势现在已经没有了，但是微软必须开始强调这些特性。到目前为止，它可能没有这样做，因为它不想蚕食常规的 Office 销售(毕竟，Web 应用对许多用例来说“足够好了”)。如果这种情况开始改变，谷歌最好小心。
