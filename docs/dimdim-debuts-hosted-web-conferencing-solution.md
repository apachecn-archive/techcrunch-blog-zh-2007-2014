# Dimdim 推出托管网络会议解决方案

[![](img/bf890141a60d49ed0ce9ea28988f5d57.png)](https://web.archive.org/web/20221208215941/http://www.crunchbase.com/company/dimdim)

[Dimdim](https://web.archive.org/web/20221208215941/http://www.dimdim.com/) 是专有网络会议包 [WebEx](https://web.archive.org/web/20221208215941/http://www.webex.com/) 和 [GoToMeeting](https://web.archive.org/web/20221208215941/http://www.gotomeeting.com/) 的开源替代品。我们第一次报道他们是在 2006 年 9 月，当时他们正在提供内部部署解决方案的 alpha 版本。

今天，该公司发布了该解决方案的托管版本，无需设置即可使用，甚至不像其他产品那样需要浏览器插件。首席执行官 DD Ganguly 表示，通过摆脱插件，Dimdim 将更容易被那些技术不熟练或担心在线安装第三方软件的人使用。

[![](img/d29a8c65e9d7b15313af44ddded22bca.png)](https://web.archive.org/web/20221208215941/https://beta.techcrunch.com/wp-content/uploads/2008/04/dimdim_shot.png)

托管 web 会议体验与内部部署的体验具有相同的功能，只是公司不能将其定制为看起来像他们的网站。Dimdim 可用于运行幻灯片演示、举行协作手绘会议、聊天(小组或私下)以及共享桌面。

该产品的免费版本允许多达 20 个用户同时使用，并有广告支持。为了摆脱广告并获得技术支持、保证正常运行时间和定制品牌等好处，组织将不得不根据他们需要的座位支付 100 至 500 美元不等的费用。

Dimdim 表示，其内部解决方案已在 165 个国家被下载超过 20 万次。大约有 26，000 人测试了这个托管解决方案。

Dimdim 遵循的开源策略在客户希望在内部管理软件时最有意义，而当一切都托管在云中时，它就不那么重要了。但是很高兴看到竞争在巨人 WebEx 的伤口上咬了一口。
