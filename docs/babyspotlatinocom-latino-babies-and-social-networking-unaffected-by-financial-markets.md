# BabySpotLatino.com:“不受金融市场影响的拉丁裔婴儿和社交网络”

[![](img/0e66bb76af713a2b079d55f66a1cf019.png "logo1")](https://web.archive.org/web/20221007105452/https://beta.techcrunch.com/wp-content/uploads/2008/09/logo1.png) 随着我今年看到的最引人入胜的公关头条之一，我们发现自己正处于完美的 Web 2.0 风暴的中心:拉丁美洲人、婴儿和社交网络。介绍 [Babyspot Latino](https://web.archive.org/web/20221007105452/http://babyspotlatino.com/) ，一个为拉丁裔婴儿的父母或拉丁裔父母的婴儿服务的社交网络。加上“Twitter”和“API”这两个词，你就得到了一个大满贯。

不要误解我，这是一个崇高的目标。它为传统上标准社交网络服务不足的市场带来了婴儿护理的新声音。该网站首次亮相于一个西班牙语脱口秀节目，节目中可爱的小剃须刀像丘比特一样雀跃,该网站将包括关于婴儿健康和怀孕的教育材料，以及标准的 BabySpot 功能，如照片共享和小组创建。

我的主要问题是，这个标题(“[拉丁裔婴儿和社交网络不受金融市场影响——BabySpotLatino.com 在 Univision 的 Despierta America](https://web.archive.org/web/20221007105452/http://www.prnewswire.com/cgi-bin/stories.pl?ACCT=109&STORY=/www/story/09-30-2008/0004895089&EDATE) )上首次亮相并获得巨大成功”)是如何通过一些公关高射炮的绝对牛市传感器的。从这里到哈利斯科，年轻的母亲们在逗弄跪着的孩子时，可能不会想到将一个拉丁裔婴儿网站的推出与自由世界的经济崩溃联系起来。
