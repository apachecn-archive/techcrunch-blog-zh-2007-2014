# 思科以 3100 万美元现金收购康普特尔的 Axioss 软件资产

[思科](https://web.archive.org/web/20230205041432/http://www.crunchbase.com/company/cisco)今晨[宣布](https://web.archive.org/web/20230205041432/http://www.marketwire.com/press-release/cisco-announces-intent-to-acquire-axioss-software-assets-nasdaq-csco-1551855.htm)其打算以[2130 万欧元](https://web.archive.org/web/20230205041432/http://www.comptel.com/default.asp?docId=18242)或大约 3100 万美元现金从合作公司 [Comptel](https://web.archive.org/web/20230205041432/http://www.comptel.com/) 的英国子公司购买服务履行软件资产和相关员工。Axioss 是由 Axiom Systems 开发的一种履行解决方案，该公司已于 2008 年被 Comptel 收购。

通过此次收购，思科希望将网络和服务管理技术扩展到其基于 IP 的网络平台，并使服务提供商能够使用单一管理架构更快地向客户推出新的视频、数据、移动和云服务。

该公司表示，AXIOSS 软件套件还将增强思科 Prime，使服务提供商能够更好地管理他们的网络和网络服务。

收购预计在 2011 年第三季度完成，届时 AXIOSS 团队将整合到思科 NMTG 和思科高级服务集团。

Comptel 首席技术官兼执行董事会成员 Gareth Senior 也将调任思科。

Comptel 和 Cisco 将继续在 OSS/BSS 的其他领域进行合作，例如云调解和收费。Comptel 表示，它将继续开发和销售 Comptel Fulfillment 解决方案，保持现有的 Axioss 客户关系，并继续支持这些客户。
