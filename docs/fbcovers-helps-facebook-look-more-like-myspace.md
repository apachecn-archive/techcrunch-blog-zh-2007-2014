# 用 FB covers TechCrunch 的时间轴封面照片让脸书看起来更像 MySpace

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/28/fbcovers-helps-facebook-look-more-like-myspace/>

上周，[脸书推出了](https://web.archive.org/web/20230203153946/https://techcrunch.com/2011/09/22/facebook-timeline/)自公司推出新的脸书时间轴以来，对脸书个人资料页面最大胆的改造之一。更新后的个人资料可以让你更深入地了解你的过去，导航功能可以让你回到在脸书度过的前几个月和前几年。

然而，[你的脸书个人资料最引人注目的变化](https://web.archive.org/web/20230203153946/https://techcrunch.com/2011/09/22/facebook-timeline-pictures/)并不是时间线导航——而是显示在页面顶部的大横幅尺寸的照片，从今天起，一系列较小的个人资料图片出现在那里。但是如果你没有一张好的照片放在那里呢？这就是在[mybcovers](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/)新推出的封面照片资源的用武之地。

需要说明的是，我们并不完全推荐这个网站，它漏洞百出，充斥着广告，而且它的脸书应用程序要求允许向你发送电子邮件。相反，MyFBCovers 对脸书设计的未来提供了一个有些可怕的展望。

本质上，MyFBCovers 是一个脸书时间线封面照片的壁纸网站。它本身并不十分引人注目，很可能只是众多类似网站中的一个。但它是第一批推出的产品之一，针对的是那些尝试开发脸书时间轴的人。([而且我们知道你是](https://web.archive.org/web/20230203153946/https://techcrunch.com/2011/09/22/how-to-enable-facebook-timeline/))。

该网站提供了两种风格的横幅。第一种风格提供传统的壁纸风格的照片，包括动物、卡通形象、自然风景、体育、学校、公司、名人和其他几十种类别。另一种风格可以让你创建包含你自己的脸书相册中的拼贴照片的横幅。

[![](img/05edc04f51aae8bcf591b2567c2f2bb2.png "fb-covers-ick")](https://web.archive.org/web/20230203153946/https://techcrunch.com/wp-content/uploads/2011/09/fb-covers-ick.png)

在我看来，结果有点令人讨厌。脸书的个人资料曾经在设计上看起来干净简约，而新的横幅照片——尤其是当它是一张类似壁纸的照片时，让你的个人资料页面看起来更像 MySpace 而不是脸书。

FBCovers 甚至坚持把它的小标签放在照片的右下角，这样你就可以告诉别人你是如何毁掉你的个人资料的。和其他封面照片的变化一样，新闻会发布到你的脸书新闻订阅源，这样你的朋友就可以喜欢和评论这张照片。

我知道很多人会发现 [FBCovers](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/) 和类似的服务很方便。我知道，并不是每个人都想在自己的脸书个人主页上放一张专业、高质量的大而醒目的照片。但是我看到了我的朋友们以前对他们的 MySpace 个人资料所做的事情，坦白地说，很多人的品味都很差。

还有多久我们才能看到[比基尼女郎](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/view/Jersey+Shore-156)、 [Lady Gaga 致敬](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/view/Lady+Gaga-8)、[粗俗漫画](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/view/Blow+Me-46)、[啤酒照片](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/view/Beer-47)、[愚蠢语录](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/view/Believe+In+Yourself-23)和[其他诸如此类的废话](https://web.archive.org/web/20230203153946/http://www.myfbcovers.com/view/This+Girl+is+Crazy-21)？

哦，从今天开始？太棒了。
