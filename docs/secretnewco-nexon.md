# Zynga 前首席游戏设计师为 SecretNewCo 接受 Nexon 的战略投资

早在一月份，Zynga 的首席游戏设计师布莱恩·雷诺兹(Brian Reynolds)悄悄地离开了公司，他也是《国家崛起》和《文明 II》等老游戏的幕后策划者。

这个月，他带着一家名为 [SecretNewCo](https://web.archive.org/web/20221005163722/http://www.secretnewco.com/) 的新公司重返舞台(很合适，是吧？)，这家初创公司今天宣布与东京游戏巨头 Nexon 建立战略合作伙伴关系。这家总部位于东京的公司正在向雷诺兹的公司投资少量股份，具体金额未披露。虽然 Nexon 没有透露投资金额，但可以假设这是一笔非常早期的投资。

这笔交易将使 Nexon 获得 SecretNewCo 旗舰游戏的全球出版权，该游戏的工作名称是“SecretNewGame”。SecretNewCo 的旗舰游戏本身听起来就让人联想到文明。标题是一个“移动战略社交网络游戏”，玩家开始是石器时代的一个部落的领袖，必须引导他们的文明通过人类历史的所有时代。就像在文明中一样，他们可以与朋友交易，对敌人发动突袭，结成联盟，抵御其他玩家的攻击。

Nexon 在中国、日本和韩国建立了像 MapleStory 这样的长期特许经营后，达成这项协议是为了吸引西方观众。

Nexon 的首席财务官 Owen Mahoney 说:“我们已经关注了西方和欧洲的一些人，我们已经考虑了一段时间。”"当雷诺兹离开 Zynga 时，我们开始讨论他的下一步计划."

虽然尼克森[在今年前三个月创造了近 4.5 亿美元的收入](https://web.archive.org/web/20221005163722/http://ir.nexon.co.jp/en/library/pdf/20130510.pdf)，但欧洲和北美市场同期对该公司收入的贡献不到 5%。相比之下，中国占了 Nexon 收入基础的近一半。

因此，在国际市场壁垒降低之际，该公司正寻求扩大在西方的业务。历史上只在日本或中国表现良好的游戏，现在有能力通过 Android 和 iOS 平台接触到完全全球化的观众。日本的热门游戏《拼图与龙》目前在美国票房排行榜上排名第 16 位，而西方游戏如《国王的糖果粉碎传奇》和《Supercell 的部落冲突》在美国也卖得不错。

与西方游戏开发商的更多交易可能会修补 Nexon 在这些市场的产品。Mahoney 表示，Nexon 正在寻求各种规模的收购和战略投资，从非常早期的交易到可能高达数亿美元的交易。截至 3 月底，该公司的资产负债表上有 1105 亿日元(11 亿美元)现金。

“我们的规模不受限制。马奥尼说:“我们有大量现金头寸，所以我们资金充足，能够在大宗商品上市时出手。”。“但我们认为，机会将是一系列小型、中型和大型交易，这取决于团队的创意方向，他们是否有相同的方法来建立公司和特许经营权，估值，以及我们的团队和他们的团队如何相处。”
