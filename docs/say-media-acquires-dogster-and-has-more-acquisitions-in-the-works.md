# 假设媒体收购了 Dogster(并且正在进行更多收购)

在[创立](https://web.archive.org/web/20230203020646/https://techcrunch.com/2010/09/21/video-egg-will-acquire-six-apart-and-rename-itself-say-media/)仅仅六个月后，据说媒体已经走向狗:它收购了[狗主人](https://web.archive.org/web/20230203020646/http://www.dogster.com/)，一个为狗主人服务的网络社区，每月吸引大约 200 万独立访客。这一举措起初听起来令人困惑，但也让媒体的使命变得更加清晰。交易条款没有披露。

SAY Media 诞生于去年秋天，当时流行的广告网络 VideoEgg 收购了苦苦挣扎的博客平台 Six apartment，这一合并并没有立即产生很大的意义。但是，正如 VideoEgg 创始人兼 SAY Media 首席执行官 Matt Sanchez 解释的那样，该网站现在正寻求利用 Six Apart 的技术来创建一个“现代垂直媒体公司”，专注于热情用户的利基社区。Dogster 将是第一个加入 SAY 媒体网络的社区。

Dogster(以及它的姐妹网站 Catster)最初是一个面向狗主人的社交网络。但是在脸书崛起之后，这个网站演变成了一个面向宠物爱好者的基于编辑的社区。该公司目前有一个由大约五名全职员工组成的团队，专注于创建内容(此外还有许多贡献者)，他们将加入 SAY Media 的 350 人团队。

桑切斯说，我们可以预计 SAY Media 将收购许多其他网站，因为它建立了“有趣、充满激情的垂直领域”的名册，其中将包括食品、家居、生活方式和健康等主题。一些网站也将在内部创建。

桑切斯说，Dogster 将继续使用其现有品牌，尽管 SAY Media 将在鼓励用户参与和改善网站布局方面纳入一些最佳实践。

披露:TechCrunch 创始人兼编辑迈克尔·阿灵顿持有 Dogster 少量[股份](https://web.archive.org/web/20230203020646/https://techcrunch.com/2006/09/18/why-i-invested-in-dogster/)。
