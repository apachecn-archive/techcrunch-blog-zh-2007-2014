# 米切尔·卡普尔领投视频转录初创公司 SpeakerText 的 60 万美元天使轮投资

![](img/3ccf596a7ebe102b6e6216b434ef0b9a.png)

[SpeakerText](https://web.archive.org/web/20221206221819/http://www.speakertext.com/) ，一家通过结合[智能算法和众包](https://web.archive.org/web/20221206221819/https://beta.techcrunch.com/2010/09/07/speakertext-crowdsources-micro-tasks-to-automate-video-transcripts-100-beta-invites/)自动录制视频的初创公司，从由[米切尔·卡普尔](https://web.archive.org/web/20221206221819/http://www.crunchbase.com/person/mitch-kapor)领导的天使投资人那里筹集了 60 万美元。其他投资者包括戴夫·麦克卢尔的 500 Startups，Crowdflower 首席执行官卢卡斯·比沃德，罗伊·罗登斯坦，克里斯·叶和前谷歌人乔治·哈里克

大多数出版商面临的挑战是视频仍然很难找到。SpeakerText 通过以 SEO 友好的格式自动转录每个视频来解决这个问题，以便视频变成可搜索的抄本。视频首先通过语音转文本引擎，然后被分解成小块，发送给远程在线转录者，由他们修正任何错误。
