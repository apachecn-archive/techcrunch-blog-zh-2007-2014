# Chill 向社交领域迈出了新的一步，允许用户建立直播休息室

上周在[期间，TechCrunch Disrupt SF](https://web.archive.org/web/20230204121806/http://disrupt.beta.techcrunch.com/) 社交视频服务 [Chill](https://web.archive.org/web/20230204121806/http://www.chill.com/) 设立了一个特殊的 Disrupt Chill 休息室，允许人们在 Chill 的虚拟家庭影院环境中观看和聊天 TechCrunch Disrupt live。这是一个成功，也就是说，因为与观众一起观看现场活动是多么引人注目，即使观众是虚拟的。

由于用户对 Disrupt lounge 的积极反馈，Chill 联合创始人 [Brian Norgard](https://web.archive.org/web/20230204121806/http://www.crunchbase.com/person/brian-norgard) 决定向所有用户开放这项围绕直播内容创建 Chill lounges 的功能。从今天起，任何 Chill 用户只需选择 Chill 休息室创建流程中的直播视频流选项，即可创建直播休息室。该功能支持 Ustream、Justin.tv、Livestream.com、YouTube Live 和 Twitch.tv，根据 Norgard 的说法，更多选项即将推出。

“我们有很多内容合作伙伴直接联系我们，要求我们为他们的品牌建立类似的体验，”诺加德谈到 TechCrunch Disrupt lounge 的影响时说。Chill 计划在下周发布更多品牌休息室，从一些知名的网络系列开始，希望专注于使事件驱动的内容更加社会化。

Chill livestreams 功能正好赶上 F8 怪兽脸书大会，您可以在 Chill live streams 休息室[观看太平洋标准时间上午 10:30 开始的直播。](https://web.archive.org/web/20230204121806/http://chill.com/lounge/f8-facebook-developers-conf)应该是激动人心的。

最近，Chill】从 Kleiner Perkins[、](https://web.archive.org/web/20230204121806/http://www.kpcb.com/) [Redpoint](https://web.archive.org/web/20230204121806/http://redpoint.com/) 和 TechCrunch 创始人[迈克尔·阿灵顿](https://web.archive.org/web/20230204121806/http://i.telegraph.co.uk/telegraph/multimedia/archive/01208/costume-rabbit_1208808i.jpg)等人那里筹集了 150 万美元的天使投资。
