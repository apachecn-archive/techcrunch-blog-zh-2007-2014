# 更多的性爱视频给大家！YouTube 向出价最高者出售视频搜索结果

![](img/9825b4cff42c27d258296b84f859b151.png)

忘记那些[YouTube 视频网站](https://web.archive.org/web/20221212165329/http://www.beta.techcrunch.com/2008/10/07/still-searching-for-a-video-business-model-google-introduces-the-youtubevertorial/)。当谈到把 YouTube 变成真正的商业，你知道，赚钱的时候，谷歌不再开玩笑。YouTube 本身将是仅次于谷歌的第二大搜索引擎。你知道吗？谷歌已经知道如何从搜索中赚钱。你把最好的结果卖给出价最高的人。

YouTube 一直拒绝出售搜索结果，但现在它已经向 T4 屈服了。现在任何人都可以在点击付费的基础上为特定的搜索关键词推广他们的 YouTube 视频。这应该会给谷歌带来可观的新收入。YouTube 有很多搜索目录。(观众每月在网站上搜索超过 25 亿次)。这也会产生更多的垃圾邮件结果。

谷歌喜欢假装付费搜索广告和有机搜索结果一样相关。在文本世界里，这有时是真的。但是当涉及到视频的时候，性的因素被放大了，不管搜索条件是什么，你最终有一半的时间都是性视频。

例如，当我搜索[“体育”](https://web.archive.org/web/20221212165329/http://www.youtube.com/results?search_query=+sports&search_type=&aq=f)时，第二大赞助结果是“WWE 天后，性感的体育女孩。”(参见上面的屏幕截图)。我不认为它链接到的视频中的女孩(题为“性感网络摄像头女孩热宝贝大胸巨乳裸体屁股华丽屁股魅力模型”)真的是一个摔跤手。YouTube 禁止真实的色情广告，但女孩穿着比基尼模拟性爱的视频是可以的。

搜索词是什么并不重要，像这样的视频会有很多点击。YouTube 上已经有很多这样的视频了。现在它们会变得更容易找到。
