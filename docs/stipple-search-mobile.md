# 点画推出图像归属服务点画搜索，加上一个移动应用程序

图像标记初创公司 [Stipple](https://web.archive.org/web/20221208065912/http://www.stipple.com/) 今天宣布了两款新产品——Stipple Search，它允许任何人识别图像的所有者，以及 Stipple Mobile，一款智能手机照片应用程序。

Stipple 的技术可以识别图像，并添加可以指向相关内容的标签。例如，[Stipple Mobile](https://web.archive.org/web/20221208065912/https://stipple.com/photos/53218230)的这张图片包括相关照片的标签、应用程序本身的链接，以及表明图片源自 Stipple 的注释。出版商可以使用这些标签来丰富他们的内容，而广告商可以使用它们在新的相关上下文中传递他们的信息——电影工作室可以在电影明星的图像上为新发布的电影做广告。

[点画搜索](https://web.archive.org/web/20221208065912/http://www.stipple.com/search)使用类似的技术，但目的是“反向图片搜索”——上传一张图片，然后得到显示图片来源的结果。基本上，联合创始人兼首席执行官 Rey Flemings 说，如果你上传一张图片到点画搜索，如果点画的合作伙伴之一([像 Getty Images](https://web.archive.org/web/20221208065912/https://beta.techcrunch.com/2013/07/31/stipple-partners-with-getty-images/) )已经在点画系统中标记了这张图片，那么你应该会看到所有这些标记。

弗莱明斯指出，到目前为止，“你能看到点画作品魔力的唯一途径是你是一名出版商。”但是任何在网上找到有趣图片的人都可以去点画搜索并找到额外的信息，比如允许你购买照片中产品的链接。

谷歌图像还包括一个反向图像搜索选项，但正如弗莱明斯在他的演示中所展示的那样，谷歌只是在定位同一张图像的其他用途——点画实际上是在检索有用的数据。

弗莱明斯说，这也是一个有趣的“归因游戏”，因为现在更容易找到图像的所有者，并请求他们允许使用它。这给了图像所有者更多的理由与 Stipple 合作，因为该公司的数据现在可以在出版合作伙伴之外获得。

至于新的点画移动 iOS 应用程序(Android 正在开发中)，弗莱明斯说，目标不一定是与现有的照片共享应用程序竞争，而是将点画移动变成你用来上传照片的初始应用程序，然后在其他地方分享。目前，用户可以在点画手机上给他们的图像添加标签，然后在脸书和推特上分享它们——弗莱明斯认为，这让他们对这些图像拥有更多所有权，因为他们想要分享的信息将随着图像的分享和再分享而保持不变。

他说，“我在‘挑选你喜欢的内容’手机照片分享应用上创建内容，这种想法实际上是属于网络的。”。弗莱明斯补充说，点画搜索和点画移动都符合公司的大愿景:“我们希望成为网络照片的信息来源。”
