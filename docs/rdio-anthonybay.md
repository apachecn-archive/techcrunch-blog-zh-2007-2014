# Rdio 任命前亚马逊高管 Anthony Bay 为新任首席执行官 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/12/03/rdio-anthonybay/>

在经历了上个月的一轮裁员后，Skype 联合创始人 Janus Friis 创办的流媒体音乐初创公司 [Rdio](https://web.archive.org/web/20221205233021/http://www.rdio.com/) 找到了新的首席执行官 Anthony Bay。贝的经验包括在亚马逊、微软和苹果担任高管职位，他继承了让 Rdio 盈利的挑战，因为它要与像 [Spotify](https://web.archive.org/web/20221205233021/http://www.spotify.com/) 这样的市场领导者竞争。

流媒体音乐业务的利润率非常低，Rdio 缺乏 Spotify 已经形成的音乐行业联系，可能难以获得足够的付费用户来维持运营(该公司尚未披露其当前用户群的规模)。Rdio 没有透露上个月解雇了多少员工，但有一个人告诉我们，裁员影响了 35 人，而另一个消息称，这个数字约为 Rdio 员工总数的五分之一至三分之一，其中工程部门的裁员幅度最大。

该公司的一名代表告诉我们，裁员是为了“改善其成本结构，并确保长期可扩展的商业模式。”还不清楚这家初创公司的视频流媒体服务 [Vdio](https://web.archive.org/web/20221205233021/http://www.vdio.com/) 受到了怎样的影响。

另一个潜在的令人担忧的迹象是，这家初创公司已经有一段时间没有发布任何用户数据了。相比之下， [Deezer](https://web.archive.org/web/20221205233021/http://www.deezer.com/) 在 11 月早些时候报告说，尽管没有进入美国市场，它的用户已经超过了 500 万。Spotify 在 2013 年 3 月报告了 600 万付费用户，[今天](https://web.archive.org/web/20221205233021/https://beta.techcrunch.com/2013/12/03/spotify-gets-in-tune-with-musicians-with-the-new-b2b-spotify-artists-website-has-paid-out-500m-to-rightsholders-so-far-in-2013/)宣布向音乐人和经理人开放其分析。然而，Rdio 告诉 TechCrunch，自 2012 年底以来，它的新用户数量增加了两倍，其中 90%的用户现在都是 Rdio 无限用户，每月费用为 9.99 美元。

贝曾是亚马逊全球视频业务的负责人，也是 Loudeye 的董事长，在被出售给诺基亚之前，他领导着该公司的 B2B 数字媒体内容部门。在微软，贝负责公司的电子商务技术平台，以及开发 Windows Media 技术，包括 Windows Media Player。他还在苹果工作了八年，负责苹果的在线服务。

贝跟随德鲁·拉纳，他现在是 Rdio 的副主席和董事会的战略顾问。拉纳今年早些时候宣布，他计划辞去首席执行官一职。

Friis 在一份声明中表示，Rdio 现在计划专注于在全球范围内扩大其付费用户群，并补充说，Bay 将“在释放我们全球地面无线电合作伙伴关系的价值方面发挥关键作用。”Rdio 于 9 月宣布与 Cumulus Media 建立战略合作伙伴关系，后者在美国运营着 525 家广播电台。该协议使 Rdio 能够获得 Cumulus 的节目，以及其电视台的促销活动。作为回报，持有 Rdio 母公司 Pulser Media 股份的 Cumulus[现在拥有了一个在线平台。](https://web.archive.org/web/20221205233021/http://www.nytimes.com/2013/09/16/business/media/a-digital-deal-for-cumulus-and-rdio.html?_r=1&)

迄今为止，Rdio 已经从 Atomico、红树林、Janus Friis 和 Skype 筹集了 1750 万美元。
