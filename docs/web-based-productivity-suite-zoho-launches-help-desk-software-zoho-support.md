# 基于网络的生产力套件 Zoho 推出服务台软件 Zoho Support

Zoho 是一家令人印象深刻的公司——这家初创公司基于网络的生产力直接与微软和谷歌竞争，但从未接受过任何[外部资金](https://web.archive.org/web/20221006110714/http://blogs.zoho.com/general/why-we-haven-t-taken-venture-capital)。Zoho 现在有 25 个不同的生产力应用程序，每天有 350 万用户积极使用 Zoho 的应用程序。今天，我们可以通过 [Zoho Support](https://web.archive.org/web/20221006110714/http://www.zoho.com/support/index.html) 的[发布](https://web.archive.org/web/20221006110714/http://www.businesswire.com/news/home/20101110006453/en/Zoho-Announces-Zoho-Support)再添一员，这是一款基于网络的帮助台软件。

Zoho Support 旨在帮助企业管理和回复通过电话、电子邮件或网站记录的客户支持查询。与 Zendesk 类似，Zoho Support 允许 IT 人员和技术人员对客户票据进行优先排序，找到适当的响应并尽快回复客户。

“请求”选项卡可帮助技术人员对来自客户的支持请求进行优先级排序，允许员工分配不同的任务，将任务标记为已完成或未完成，并创建任务完成顺序。“请求”模块还可以帮助组织筛选这些支持请求，以便将它们快速发送给相应的团队。

帐户和联系人模块允许组织定义、跟踪和执行他们可能与客户达成的特定服务级别协议。例如，两个看似相似的支持请求可能具有不同的优先级，这取决于向每个客户承诺的协议。

通过仪表板和报告选项卡，Zoho Support 将向组织显示哪些产品收到的请求最多，哪些问题发生得最频繁，谁的响应时间最慢等等。任务模块使销售代表能够跟踪和安排活动，从回复客户或对某个问题进行内部研究。

Zoho Support 的一个有用特性是创建解决方案数据库的能力，这是一个类似 wiki 的目的地，销售代表可以在这里创建和发布客户支持问题的解决方案。该数据库也是可搜索的。

使用 Zoho 的任何应用程序的优势在于，它可以与其他应用程序无缝集成。Zoho 支持集成了 Zoho CRM 和 Zoho Chat。Zoho Support 可通过免费增值模式获得，提供免费计划，以及每个代理每月 12 美元起的其他计划(基于每天创建的门票数量)。

Zoho 的传播者 Raju Vegesna 告诉我，该公司还计划在[应用市场](https://web.archive.org/web/20221006110714/https://beta.techcrunch.com/2010/03/09/google-apps-marketplace/)中创建一个谷歌应用，以支持 Zoho。这家初创公司已经在市场上提供了许多应用，包括 [Zoho CRM 和 Invoice。](https://web.archive.org/web/20221006110714/https://beta.techcrunch.com/2010/09/18/developers-on-google-apps-marketplace-the-good-the-bad-and-the-ugly/)
