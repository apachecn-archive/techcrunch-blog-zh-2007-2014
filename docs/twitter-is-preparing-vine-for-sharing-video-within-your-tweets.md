# Twitter 正在准备让 Vine 在你的推文中分享视频

10 月，Twitter [收购了一家名为 Vine](https://web.archive.org/web/20230404205245/https://techcrunch.com/2012/10/09/if-twitter-gets-into-the-video-hosting-game-it-could-be-a-wonderful-and-horrible-thing/) 的公司，该公司生产一款视频分享应用，但没有多少人有机会使用。AllThingsD [报道称，杰克·多西正在大力推动交易的完成](https://web.archive.org/web/20230404205245/http://allthingsd.com/20121025/jack-dorsey-still-has-pull-at-twitter-just-ask-the-vine-guys/)，认为视频可能是 Twitter 未来成功和采用的重要组成部分。[多尔西](https://web.archive.org/web/20230404205245/https://techcrunch.com/2012/09/10/jack-dorsey-we-need-revolution-not-disruption/)据报道也极力推动 Twitter 收购 Instagram，但这并没有发生。

今天，该公司似乎正在为大众准备其视频产品，因为其首席执行官一直在通过分享链接来玩弄它。据 AllThingsD 报道，这款独立应用最早将于明天(T7)在 iOS 系统上推出。

显然，这是全世界都想看到的视频类型:

[推特 https://twitter.com/dickc/status/294124523714916353]

正如你所看到的，在一个扩展的 tweet 中，视频自动播放和循环，这可能很快变得非常令人讨厌。希望 Twitter 也能发布一个 API 更新来提供一个合适的“播放按钮”来避免这种情况。

[正如《金融评论》所指出的](https://web.archive.org/web/20230404205245/http://www.afr.com/p/technology/is_twitter_about_to_take_on_instagram_lDBOkflDFniwuLJlYfk7vL)，这将是一场“视频 Instagram”游戏，与各地应用商店中允许你在 YouTube 上分享和发布简短视频或较长内容的所有其他应用竞争。报告显示，你将可以在 Twitter 上分享六秒钟的视频，目前还不知道它是否会像 AllThingsD 建议的那样作为一个独立的应用程序推出，或者是否会集成到 Twitter 的现有应用程序中。

《金融评论》向 Twitter 的编辑总监卡伦·维克瑞(Karen Wickre)询问了关于本周推出视频服务的问题，维克瑞回答道:

> …你说的是 Vine …但不，我认为(它即将推出)还不是时候。但是我不是产品方面的，所以我不知道所有的细节。

当 Vine 被收购时，我们讨论了为什么这对 Twitter 和它的用户来说既是好事又是坏事。基本上，通过增加向数百万人实时拍摄和发送视频的能力，会有一种隐私感，这种隐私感会因意外发布而丧失，就像你现在看到的错误定向的直接消息一样。

一旦这个项目启动，第一个关于一个意外视频的新闻报道将会花多长时间，这个视频包括六秒钟地球上没有人应该用他们的眼睛看到的东西。你把剩下的部分弄清楚。有了强大的工具，就有了强大的责任，视频是一种媒体类型，一旦推出，就需要全新的视角来监管。

需要指出的是，Twitter 之前的媒体合作伙伴和主持人 [Photobucket](https://web.archive.org/web/20230404205245/https://techcrunch.com/2012/08/16/photobucket-gets-a-facebook-facelift-easier-uploads-and-a-new-timeline-like-story-feature/) 拥有视频功能，但该公司决定通过[与 Aviary](https://web.archive.org/web/20230404205245/https://techcrunch.com/2012/12/10/twitter-brings-aviary-powered-photo-filters-to-its-android-and-iphone-apps/) 合作，走自己的路。这是完全有道理的，葡萄收购将很快见天日。

我们联系了 Twitter 进行评论，但目前它没有什么可分享的。
