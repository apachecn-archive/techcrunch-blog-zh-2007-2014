# 正如 OS X Mountain Lion 所证明的，Twitter 是苹果的社交网络

在去年 6 月苹果的 WWDC 活动上，Twitter 交了一个新的最好的朋友:苹果。这家科技巨头宣布，它将通过新的 iOS 5 软件将社交网络植入每一台 iOS 设备。这让许多人感到震惊，原因有二。1)苹果很少与第三方进行如此深度的合作。2)不是脸书。

对于 Twitter 来说，这是一个全面的巨大胜利。这也是苹果的一个胜利，因为多年来已经证明他们没有社交化——这也是昨天[辩论](https://web.archive.org/web/20230127002401/http://www.techmeme.com/120215/p69#a120215p69)的一个话题，巧合的是。苹果*需要*来建立自己的社交网络，丹·弗罗默[争辩道](https://web.archive.org/web/20230127002401/http://www.readwriteweb.com/archives/apple_social_network_address_book.php)！不，他们需要把他们的社交平台搞对。今天，他们迈出了一大步:在 Twitter 上加倍下注。

苹果公司刚刚透露，Twitter 也将被烤成[OS X 的最新版本，山狮](https://web.archive.org/web/20230127002401/https://techcrunch.com/2012/02/16/os-x-mountain-lion/)。这意味着从今年夏天开始销售的每一台新 Mac 都将内置 Twitter 功能。所有升级的旧版本也会有。现在市场上有 6000 多万台 MAC 电脑。Twitter 又一次大获全胜。

Twitter 首席执行官迪克·科斯特罗[在 12 月表示，在 iOS 5 整合之后，Twitter 每月注册人数增加了 25%。iOS 已经拥有比 Mac 大得多的用户群，但 Twitter 仍然可以从这一整合中看到另一个两位数的增长。](https://web.archive.org/web/20230127002401/https://techcrunch.com/2011/12/08/apple-ios-5-integration-boosted-twitter-signups-by-25-percent/)

更重要的是，Twitter 现在已经牢牢确立了自己的地位，不仅是 iOS 的社交服务，也是苹果的社交服务。当然，苹果可以在未来的任何时候将 Twitter 从 iOS 和 OS X 中剥离出来，但这样做现在会激怒那些被训练成期待它成为苹果核心操作系统功能的用户。对于一个仍然没有赚到很多钱的服务来说，它的未来一直受到质疑，这一点很重要。这是一个巨大的认可。

这对 Twitter 来说也是一个大新闻，因为它为这项服务作为一种认证方法打开了一个全新的世界。有了 iOS，应用程序可以选择使用 Twitter 来简化新注册——让人们只需点击一两下就可以开始注册。现在，Mac 应用程序将能够做同样的事情。我不认为脸书仍然主导这个认证领域有任何问题，但 Twitter 的苹果联盟正使他们在这方面更具吸引力——至少对开发者来说是这样。

就在上周，一些人批评推特看似任性的关注。今天，他们刚刚与这个星球上最有价值的公司达成了第二次大规模合作。对我来说,[确实看起来像是一个好的焦点的标志。](https://web.archive.org/web/20230127002401/http://parislemon.com/post/17456056398/dear-henry-blodget-dont-mean-to-be-rude-but-its)
