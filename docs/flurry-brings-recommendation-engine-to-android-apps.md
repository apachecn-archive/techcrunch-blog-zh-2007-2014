# Flurry 为安卓应用带来推荐引擎

在与苹果和史蒂夫·乔布斯(Steve Jobs)就其分析产品发生争议后， [Flurry](https://web.archive.org/web/20221006231034/http://www.flurry.com/) 正朝着安卓平台进一步迈进，推出了针对安卓手机的应用推荐引擎 [AppCircle](https://web.archive.org/web/20221006231034/http://www.flurry.com/product/appcircle/index.html) 。Flurry 上个月为苹果手机推出了 AppCircle，吸引了 600 多个参与应用程序，出版商的总影响力达到 2200 万苹果设备所有者。

Flurry 的 AppCircle 试图将希望获得更多用户的应用程序开发人员与希望通过推广其他应用程序赚钱的应用程序开发人员联系起来。该服务利用 Flurry 的推荐引擎，结合 Flurry Analytics，在适当的时间从参与的 Flurry 应用网络中向适当的用户推荐适当的应用。

AppCircle 建立在 Flurry Analytics 的基础上，Flurry Analytics 是一项在苹果、安卓、黑莓和 J2ME 平台上被 45000 个应用程序使用的服务。目前还不清楚 Flurry 网络中有多少应用是安卓应用，所以应用圈的覆盖范围值得怀疑。

对于想赚钱的应用程序出版商来说，他们可以将 Flurry 集成到他们的应用程序中，这样 Flurry 就可以向他们的消费者提供有针对性的应用程序推荐。出版商从每次应用下载中获得 60%的价格。对于推广者，应用程序开发人员会创建活动，并对他们愿意为新用户支付多少费用进行投标。Flurry 的推荐引擎将匹配相关的推广应用程序，在出版商应用程序中显示，然后使用竞价来排列印象显示的顺序。出版商也可以使用自己的虚拟货币来鼓励下载。

在苹果因 Flurry 违反隐私政策而对其进行打击后，这家初创公司将视野扩大到其他智能手机平台也就不足为奇了。但是安卓可能不会长久成为一个避风港。理论上，帮助希望更多用户的开发者与希望赚钱的出版商建立联系似乎是一个好主意；但是我想知道谷歌开始使用 AdMob 是否只是时间问题。
