# JiWire 收购基于位置的移动购物平台 NearbyNow

移动广告公司 [JiWire](https://web.archive.org/web/20230202214852/http://www.jiwire.com/) 今天宣布收购基于位置的移动购物平台[。](https://web.archive.org/web/20230202214852/http://www.nearbynow.com/)交易条款没有披露。

NearbyNow 允许品牌在应用程序或广告中展示产品，并确认产品在实际商店中的可用性。用户还可以在商店中预订产品，以便提货。例如，一个 Seventeen Magazine 手机应用程序用户在网上看到一个 boots 的广告，可以点击查看当地库存。如果有现货，顾客会预订靴子，当她到达时，靴子已经在柜台上准备好了。该公司表示，广告点击率超过 20 %,转化为购买的平均比率为 5.8%。

这个广告和购物平台将是 JiWire 自有广告网络的补充。JiWire 表示，它计划将其现有的基于位置的广告平台与 NearbyNow 的平台相结合，该平台为 iPads、智能手机和笔记本电脑等设备提供 Wi-Fi 和移动服务，以增加广告商规模化接触新的本地客户并推动店内转化的方式。

NearbyNow 已经从 Draper Fisher Jurvetson 和其他人那里筹集了近 2000 万美元的风险投资。
