# 视频:川崎的助力机器人套装帮助人类举起重物

日本科技巨头[川崎](https://web.archive.org/web/20230205034514/http://www.khi.co.jp/english/index.html)正在研发所谓的动力辅助服，这是一种可穿戴的[机器人](https://web.archive.org/web/20230205034514/https://techcrunch.com/tag/robots/)，它可以帮助人类搬运 30-40 公斤重的物体，而无需肌肉的任何努力。这套衣服实际上是目前日本市场上为数不多的[类似](https://web.archive.org/web/20230205034514/https://techcrunch.com/2011/03/21/video-newest-version-of-cyberdynes-robot-suit-hal-up-and-close/)[型号](https://web.archive.org/web/20230205034514/https://techcrunch.com/2010/10/26/power-loader-light-panasonic-starts-selling-mecha-robot-suits/?replytocom=1533003)中的一种。

川崎机器人在覆盖臀部和膝盖的部分共有四个电机(见下图)。控制单元和锂电池放置在下背部周围的部分。 [](https://web.archive.org/web/20230205034514/https://techcrunch.com/wp-content/uploads/2011/09/1kawasaki_0921-1.png) [![](img/c5dc4ccdae48eba3b8ddcad03fa8ddc6.png "Picture 1")](https://web.archive.org/web/20230205034514/https://techcrunch.com/wp-content/uploads/2011/09/picture-15.png)

根据川崎的说法，这种自立机器人的设计方式是穿着者在走动时不会感觉到衣服本身的重量。川崎重工承诺，这种动力辅助服可以根据穿着者的腿的长度进行调节，穿上和脱下都不会超过一分钟。

该公司希望在自己的工厂进行一到两年的测试后，将机器人商业化。
