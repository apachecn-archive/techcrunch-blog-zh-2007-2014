# 让你可以与律师即时交流的初创公司 Lawdingo 加入了 Y Combinator

为用户和律师提供在线咨询(有时是即时咨询)的初创公司 Lawdingo 宣布，它是 [Y Combinator](https://web.archive.org/web/20221007062452/http://www.ycombinator.com/) 当前创业公司类别的一部分。

我上一次[写关于 Lawdingo 的文章是在 11 月](https://web.archive.org/web/20221007062452/https://beta.techcrunch.com/2012/11/09/lawdingo-makes-it-easy-to-find-and-instantly-consult-a-lawyer-online/)。你可以根据律师的专业知识和位置在网站上浏览他们，如果你找到一个你喜欢的，你可以安排一个约会，或者在某些情况下点击“现在说”按钮。目标是让找律师更方便、更实惠。(在支付能力方面，网站上的许多律师都提供免费咨询，而且由于你是在广泛的地域范围内选择，你可能会找到更低的价格。)

创始人兼首席执行官 Nikhil Nirmel 告诉我，他没想到会进 YC。这听起来可能像是虚假的谦虚，只不过作为一个单枪匹马、非技术型的创始人，他的情况对孵化器来说确实有点不寻常——所以他们肯定看到了他们真正喜欢的东西。(他在东欧的一个团队工作，但 Nirmel 仍然是 Lawdingo 在美国唯一的全职员工。)

[![lawdingo - profile](img/d2f31d06931ecb52afe15656b833e368.png)](https://web.archive.org/web/20221007062452/https://beta.techcrunch.com/2013/03/15/lawdingo-y-combinator/lawdingo-profile/)

现在网站上有超过 450 名律师。正如他所说，“我们的扩张速度超过了任何一家律师事务所。”

商业模式也在不断发展。最初，Lawdingo 收取订阅费，但 Nirmel 表示，由于不同律师事务所之间的营销预算不同，以及潜在客户的不同线索具有不同价值的事实，固定费用对每个人都没有意义。相反，他转向了一种拍卖模式，即律师出价将自己的个人资料展示给特定用户。随着时间的推移，随着网站上律师数量的增长，如果他们希望自己的资料出现在最热门的法律话题上，他们需要出价更高。
