# 当加州 PUC 打击拼车时，Sidecar 和 Lyft 承诺继续上路 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/08/cpuc-ride-sharing-c-and-d/>

先是优步，现在是 Sidecar 和 Lyft。

两年前试图关闭优步的加州公共事业委员会已经向诸如 T2 Lyft、T4 Sidecar 和 tick engo T7 等拼车初创公司发出了停止令。该委员会的任务是监管私营电力、电信、水务、公交和客运公司，现在正试图弄清楚拼车公司是如何纳入其监管框架的。但与此同时，它一直要求他们离开道路。

发送给拼车初创公司的信件显然没有什么效果:虽然一个多月前就已经送达，但它们现在才刚刚进入公众的视线。与此同时，Lyft、Sidecar 和 Tickengo 都一直在路上行驶，没有明显的停产。

### Lyft 表示，它超越了

Zimride 的联合创始人约翰·齐默和洛根·格林今天早上在博客文章中发表了这封信，为 Lyft 的运营辩护，尽管监管机构进行了审查。Zimmer 和 Green 将 Lyft 描述为乘车共享的又一次演变，从大学的物理乘车板，到 Craigslist 上的帖子，到 Zimride 自己的在线社交平台，再到通过 Lyft 等应用程序的移动设备。

根据 Zimmer 的说法，发出的信件是 CPUC 努力了解新技术如何适应当前监管框架的一部分。“很明显，我们使用的技术在制定这些法规时是无法想象的，”齐默在电话中告诉我。

他表示，自收到这封信以来，Lyft 一直在与欧盟委员会进行富有成效的对话，该监管机构对其服务的运作方式有了更好的理解。这种理解的关键是让委员会相信 Lyft 的运营完全符合该法律，并且它已经坚持了比其他运输服务更高的标准。“当你看我们的背景或司机检查标准时，它是超越豪华轿车服务的，”齐默告诉我。

就其本身而言，Lyft 会让潜在司机通过一系列检查，然后允许他们在车上贴上粉红色的小胡子。这包括车管所和犯罪背景调查，随后是面对面的采访和车辆检查。据我们所知，大约有 5%的申请人被接受为司机，尽管它一直在迅速扩大其车队，所以这个比例可能已经发生了变化。Lyft 现在有超过 250 名司机，相比之下，它在 8 月份正式向公众推出时只有 100 名司机。

### 边车是多么的不同

Sidecar 也收到了通知，甚至在 Lyft 创始人 Sunil Paul 通过电子邮件告诉我之前。据保罗说，这封信要求 Sidecar 停止经营“租船承运人”，这需要许可证。由于他认为 Sidecar 不是租船承运人，这家初创公司一直在路上。

保罗写道，Sidecar 正在受到审查，因为它是一个成功的创新者“打破现状”，但它被精心设计为 100%合法。“我们是致力于公共安全的创新者。我们建立了一个考虑到安全的系统，并将在未来几个月继续扩展安全和保险功能，”保罗写道。

他还试图将 Sidecar 与其他拼车服务区分开来。他写道，首先，司机知道潜在乘客的目的地，因此可以选择是否让他们搭车。它也不像其他应用程序那样建议捐款或设定价格，而是使用软件来计算乘车的平均价格。

### tick engo——一个真正的拼车平台？

与此同时，Tickengo 的联合创始人 Geoff Mathieux 声称，这是一个真正的点对点协作运输平台，“旨在填补普通汽车的空座位”，而不是用于商业目的。虽然一些其他服务的司机一天要接几个乘客，但这不是 Tickengo 的目标。事实上，它将司机每年的收入限制在 AAA 的官方年度车辆拥有成本之内，目前为每年 8776 美元。

Mathieux 还将 Tickengo 与众不同，称它只是一个连接个人的平台，与上面提到的 Lyft 旧游戏板没有什么不同。他在一封电子邮件中写道:“我们不从事驾驶专业人士或商业运输车辆的运营、雇佣或维护。”。

尽管其他服务已经为共乘期间的事故购买了额外的保险(Lyft)，或者正在考虑购买(Sidecar)，Tickengo 表示没有必要购买。“蒂肯戈的司机不会像其他人一样一天骑几次车，”马修写道，因此旅行不应该被视为制服或商业旅行。在此基础上，驾驶员保险单应足以承保在其一次乘坐过程中发生的任何损坏。

### 拼车只是最新的目标

当然，这些新的拼车公司并不是第一批面临当地监管机构审查的科技初创公司:优步也收到了 CPUC 的停战函，并受到了其运营的许多其他市场的监管机构和政府机构的抨击，包括纽约州、芝加哥和波士顿。像之前的优步一样，莱夫特计划在监管问题解决后继续运营。

尽管 CPUC 仍在试图弄清楚拼车如何纳入其监管范围，但莱夫特和其他人可能会从旧金山新成立的[合作消费工作组](https://web.archive.org/web/20221206115002/http://www.sfciti.com/initiatives#collaborative-consumption-working-group-the-sharing-economy)那里得到一些帮助。该工作组正试图了解 Airbnb、TaskRabbit、Getaround 等公司面临的问题。这些公司已经创建了点对点市场，允许用户更有效地共享资源、资产或技能。旧金山市长埃德·李(Ed Lee)是共享经济的狂热粉丝，这个团体似乎致力于帮助这些公司克服与根深蒂固的行业和不考虑新技术的法规相关的挑战。
