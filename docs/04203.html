移动广告网络Mobclix发布iPad SDK

面对来自最近收购的 AdMob(谷歌)和 Quattro Wireless(苹果)移动广告网络 Mobclix 的竞争，Mobclix一直在稳步加强其产品和服务。这家初创公司刚刚收购了iPhone应用销售分析软件 Heartbeat 和宣布一项交易，将尼尔森的广告定位数据提供给其广告交易所，以增强广告定位。今天， Mobclix 正式发布首批用于 iPad 广告的SDK之一。

苹果的新平板设备将在几周内交付给消费者，这为广告创造了巨大的机会。平板电脑的尺寸不仅允许更大的广告位，而且广告也可以更具吸引力和媒体丰富性，有可能融入视频和其他互动功能。这对广告商和开发商来说意味着更多的收入。

Mobclix 允许应用开发者注册他们的广告清单，广告网络根据年龄、性别、位置和其他因素竞标广告位。提供的广告会根据哪个广告网络出价最高来自动改变，以吸引特定应用的用户。它还允许广告商根据人口统计、地理定位和行为特征购买各种应用。

Mobclix将其iPhone SDK拆开，并完全为iPad重新构建了一个新的SDK。这家初创公司表示，新的iPad广告机会意味着更高的eCPMs和更多的广告尺寸选择。目前，Mobclix合作伙伴Traffic Marketplace和Valueclick正在为iPad提供300×250的广告单元。但Mobclix将在4月份推出468 x 60、120 x 600、728 x 90格式。

当然，看看苹果自己的移动广告网络Quattro Wireless将如何在iPad的广告生态系统中发挥作用应该是很有趣的。虽然还不清楚苹果对Quattro有什么计划，但我们知道苹果看好移动广告，并且可能会在T2为开发者开发自己的移动广告产品。Quattro Wireless是Mobclix交易所包含的网络之一。
