# 谷歌通过新的 DevArt 项目 TechCrunch 帮助将比特转化为艺术

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/05/google-to-help-turn-bits-into-beaux-arts-with-new-devart-project/>

谷歌正在以一种特别适合其角色和业务的方式发起一项支持艺术的广泛的新努力，一个名为 DevArt 的项目[强调现代艺术家使用技术，特别是代码来创作他们的作品。](https://web.archive.org/web/20221206172107/http://googleblog.blogspot.ca/2014/02/devart-art-made-with-code.html)

【YouTube http://www.youtube.com/watch?v=30yGOxJJ2PQ？feature = player _ 嵌入式]

该项目将在伦敦巴比肯表演艺术中心展出，包括四个独立的艺术装置。为了做好准备，谷歌呼吁开发者填补其中的一个位置，与三位已经被选中展示他们作品的特色艺术家一起展出，他们在下面的视频中出现。

**扎克·利伯曼**

【YouTube http://www.youtube.com/watch?v=C5U0BH1HvAc]

**瓦尔瓦拉·古尔贾耶瓦和玛尔·卡内**

【YouTube http://www.youtube.com/watch?v=S1Ta8AMbkGU]

**卡斯滕·施密特**

【YouTube http://www.youtube.com/watch?v=USrM43hZSOQ]

谷歌委托艺术家们使用谷歌 API 及其网络产品和服务，为开幕展创作全新的互动装置。涉及的一些技术包括 Kinect、Unity、WebGl、Arduino、Raspberry Pi、Google+、地图、Twitter 和 YouTube 等等。那些被谷歌选择参与的人将通过每个艺术家运行的项目页面，维护他们与展览进展的持续日志。

如果你认为你拥有成为下一个黑客的编程技能(这很糟糕，但这是我能做到的最好的)，那么你可以通过[谷歌的 DevArt 网站](https://web.archive.org/web/20221206172107/https://devart.withgoogle.com/#/enter-now)报名成为今年夏天展会的第四个参展商。通过 GitHub 跟踪提交的作品，参赛作品必须在 3 月 28 日前提交才能被考虑，获胜者将于 2014 年 4 月 15 日揭晓。
