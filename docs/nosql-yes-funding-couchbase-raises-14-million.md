# NoSQL，是的资助！Couchbase 筹集了 1400 万美元

今年早些时候通过合并 CouchOne(由 CouchDB 的创建者联合创建)和 Membase(由 memcached 背后的关键开发者创建)创建的 NoSQL 数据库公司 [Couchbase](https://web.archive.org/web/20230205031117/http://www.couchbase.com/) ，已经在 C 轮融资中获得[1400 万美元](https://web.archive.org/web/20230205031117/http://www.crunchbase.com/company/couchbase)。

这轮融资由 Ignition Partners 牵头，该公司现有的支持者 Accel Partners、Mayfield Fund 和 North Bridge Venture Partners 参与了融资。

此外，该公司说，Couchbase 还额外预留了 100 万美元用于“战略客户和合作伙伴”的投资。

Couchbase 为构建 Web 和移动应用程序提供数据库解决方案。就在上周，该公司发布了 [Couchbase Server 2.0](https://web.archive.org/web/20230205031117/http://www.couchbase.org/get/couchbase/2.0.0) ，这是一个分布式的、面向文档的数据库管理系统，也是该公司在合并后发布的第一个集成产品。

Couchbase 表示，它将使用新的资金来加速 NoSQL 数据库技术的交付，在企业市场上发展公司，并进行国际扩张。

这轮融资使该公司的资本总额达到 3000 万美元。
