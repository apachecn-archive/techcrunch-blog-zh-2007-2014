# CloudBees 筹集了 1120 万美元的 C 轮融资，由威瑞森风险投资公司牵头，以扩大其以 Java 为中心的企业平台即服务(PaaS)

[CloudBees](https://web.archive.org/web/20230306005432/http://www.cloudbees.com/) ，一个以 Java 为中心的企业平台即服务，我们在过去报道过几次，今天宣布它已经筹集了由威瑞森风险投资公司领导的 1120 万美元的 C 轮融资。现有投资者 Matrix Partners 和 Lightspeed Ventures 也参与了此轮投资，还有新投资者 Blue Cloud Ventures。

自 2010 年成立以来，这项服务已经筹集了 2570 万美元。该公司计划利用这笔资金通过扩大其功能集和全球销售足迹来推动收入增长，并“扩大 CloudBees 品牌的覆盖范围”

CloudBees 之前与威瑞森有过合作关系，尽管两家公司[在今年早些时候宣布了合作关系。作为交易的一部分，威瑞森在其最近推出的威瑞森云平台上提供 CloudBees 的服务。该公司告诉我，它选择威瑞森作为主要投资者，因为它“认真对待成为顶级投资者和投资最好的公司。”](https://web.archive.org/web/20230306005432/http://www.cloudbees.com/press-room/cloudbees-and-verizon-add-enterprise-paas-new-verizon-cloud)

就像类似的以企业为中心的 PaaS 服务一样，CloudBees 允许开发人员在公司的平台上构建和测试他们的应用程序。然而，让 CloudBees 与众不同的是它对 Java 的专注。

该公司也支持几乎所有基于 JVM 的语言，包括 Spring、JRuby、Grails、Scala、Groovy 等。CloudBees 上的开发者使用 Jenkins 来跟踪 Git 和 SVN 库中的代码变化，并使用 Maven、Gradle 和 Ant 的链接来触发构建。
