# 从你的浏览器搜索你的云:Greplin 增加了一个 Chrome 扩展

[![](img/3145f4828bad563591f61a4cc4f1b43f.png "Picture 9")](https://web.archive.org/web/20230203115955/https://techcrunch.com/wp-content/uploads/2011/03/picture-9.png) 对于[格雷普林](https://web.archive.org/web/20230203115955/https://www.greplin.com/)来说，这是活跃的 2011 年。二月份，我们报道了社交搜索服务的[公开发布](https://web.archive.org/web/20230203115955/https://techcrunch.com/2011/02/16/greplins-social-search-opens-its-doors-to-all/)，仅仅几天后，我们报道了它的关闭[一轮由红杉支持的 400 万美元的风险投资](https://web.archive.org/web/20230203115955/https://techcrunch.com/2011/02/14/greplin-grabs-4-million-from-sequoia-for-social-search/)。

自那以后，Greplin 一直在迅速扩大其可以搜索和索引的社交应用，上个月在其已经健康的网络和服务中添加了 Yammer、Highrise 和 Google Contacts。除此之外，你可以授权它搜索 Dropbox、Twitter、脸书、LinkedIn、谷歌日历、谷歌文档、Evernote、Basecamp 和 Gmail。只需一个简单的搜索查询，砰！，Greplin 几乎可以在您的整个个人云中搜索您似乎找不到的地址、联系人或邀请。

更重要的是，今天 Greplin 宣布在其搜索存在中添加一个 Chrome 扩展，这样你就可以直接从浏览器中搜索你的社交数据，而不必去 Greplin 的网站登录。看看右边图片中的扩展[和 Greplin 的小部件。](https://web.archive.org/web/20230203115955/https://chrome.google.com/webstore/detail/bjclhonkhgkidmlkghlkiffhoikhaajg?hl=EN)

[![](img/5e2ab47cf566912795c061c1fbd74fca.png "Picture 16") ](https://web.archive.org/web/20230203115955/https://techcrunch.com/wp-content/uploads/2011/03/picture-16.png) Greplin 采用了新的 [Chrome Omnibox API](https://web.archive.org/web/20230203115955/https://sites.google.com/a/chromium.org/dev/developers/design-documents/extensions/omnibox-api) ，所以你可以得到自动完成的结果，不管你在哪个网站。对于那些不熟悉 Omnibox 的人来说，它是 Chrome 搜索栏的另一种说法，它允许你在一个统一的地方输入 URL 和常规的旧关键字搜索，而不必去 Google.com。然后，Omnibox 的 API 允许扩展开发人员通过添加他们自己的关键字命令来使工具更加强大，这样用户就可以键入以关键字为前缀的查询，使扩展能够建议潜在的完成并对用户的输入做出反应。

该公司还宣布加入 [Aaron Sarazan](https://web.archive.org/web/20230203115955/http://twitter.com/#!/asarazan) (之前在 [Spark Plug Games](https://web.archive.org/web/20230203115955/http://www.sparkpluggames.com/) )，他将致力于创建一个本地移动应用程序。联合创始人罗比·沃克(Robby Walker)告诉我，Greplin 目前正在寻求增强其员工队伍，因此所有希望参与一家试图为我们提供[“搜索的另一面】](https://web.archive.org/web/20230203115955/https://techcrunch.com/2010/08/31/greplin-ycombinator-personal-search/)的年轻热门创业公司的专业工程师都应该听从这一召唤。

沃克表示，Greplin 将继续增加其可搜索社交应用的列表，Basecamp 和 Salesforce 是目前私人测试版中提供的服务之一。(接下来是 Box.net 和谷歌语音。)*和*一个火狐插件。TC 的读者可以通过给 support@greplin.com 发电子邮件来预览以上所有内容。
