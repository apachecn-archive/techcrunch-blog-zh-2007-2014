# 夏普的新设备可以让你通过网络跟踪太阳能发电

最近日本似乎完全依赖太阳能发电。4 月 1 日，[夏普日本公司将发布 JH-rwl 1](https://web.archive.org/web/20230304004642/http://www.sharp.co.jp/corporate/news/090210-a.html)【JP】，这是一款联网设备，允许家庭远程监控其太阳能发电系统的性能。

该设备采用 7 英寸彩色液晶显示器(W-VGA， 800×480 分辨率)，显示生产水平、当前储存的能量以及有多少剩余电力可以出售给电力公司。它还可以连接到个人电脑、AQUOS 电视，当然也可以连接到手机。

![sharp_jh-rwl1_2](img/4b91c51cccf6fb2494549f167e2cac32.png "sharp_jh-rwl1_2")

夏普表示，JH-RWL1 的建议零售价为 1100 美元，他们的目标是每月向家庭客户销售 700 台。该公司没有说这款设备是否会在日本以外生产。
