# Cloudera 现在为 Apache Hadoop 认证公司；启动合作伙伴计划

[Cloudera，](https://web.archive.org/web/20230203045351/http://www.cloudera.com/)商业分发和服务[基于 Apache Hadoop](https://web.archive.org/web/20230203045351/http://www.crunchbase.com/product/hadoop) 的数据管理软件和服务的初创公司，今天宣布为使用其服务的公司提供全面的合作伙伴计划。

Hadoop 是一个 Java 软件框架，诞生于 Google 发布的计算基础设施的开源实现，它是在 Apache Software Foundation 中发展起来的。Hadoop 支持在处理大量数据的大型商用计算机集群上运行的分布式应用程序。Cloudera 帮助分发 Hadoop，并围绕该技术提供实用服务，类似于[红帽](https://web.archive.org/web/20230203045351/http://www.crunchbase.com/company/red-hat)为 Linux 框架所做的事情。

合作伙伴计划包括一个自我认证套件，使供应商能够测试和验证硬件和软件在 Apache Hadoop 集群中的可支持操作。合作伙伴还将获得技术培训、营销工具等资源。该公司展示了整个生态系统中的某些供应商，因此潜在公司可以了解实际部署情况。

Cloudera 的业务开发主管 Ed Albanese 解释说，认证计划对公司来说非常重要，这与 Red Hat 验证 Linux 操作系统的方式类似。对于合作伙伴计划，他解释说，Cloudera 正试图引入商业链接，以了解开发过程中发生的事情，并深入了解供应商。
