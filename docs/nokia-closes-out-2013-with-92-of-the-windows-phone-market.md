# 诺基亚以 92%的 Windows Phone 市场份额结束 2013 年

[AdDuplex](https://web.archive.org/web/20221005201646/http://www.adduplex.com/) 的优秀员工向 [Paul Thurrott](https://web.archive.org/web/20221005201646/https://twitter.com/thurrott) 提供了他们【2013 年【Windows Phone 市场的最终数据，让我们在公众面前最终一睹微软智能手机平台 OEM 和设备构成的关键统计数据。

这些数字和全年差不多，甚至更高。诺基亚继续控制着 Windows Phone 硬件的最大份额，本月末上升了几个百分点，达到 92.1%(这是通过 AdDuplex 的网络跟踪的使用份额的计算结果)。Lumia 520 手机继续以 39.3%的份额高居同类产品之首。Windows Phone 8 在如今日渐衰落的 Windows 7.x 系统的竞争中取得了增长，其相对市场份额达到了创纪录的 78.3%。

如果你必须将 Windows Phone 的 2013 年放入基本趋势中，那将是诺基亚大获全胜，其 Lumia 520 是首选武器。

瑟罗特很好地描述了目前 Windows Phone 的低端(粗体原文):

> **几乎没有高端手机受欢迎**。在全球范围内，只有 Lumia 920 在所有 Windows Phone 手机中排名前十，如果你只看 Windows Phone 8 手机，只有两款高端设备上榜:Lumia 920 和 925。在美国，有三种型号:Lumia 920、928 和 1020。剩下的都是新的低端设备或者老设备。Lumia 1520 平板手机没有出现在这份报告的任何地方。

这意味着 Windows Phone 的销售势头是以单位收入为代价的。价格越低，利润压力越大。生产 Lumia 520 必须做出的牺牲清单[不小](https://web.archive.org/web/20221005201646/http://mynokiablog.com/2013/03/09/nokia-lumia-720-520-qa-with-product-managers/)。

因此，当我们统计诺基亚统治 Windows Phone 的最后一个月时，重要的是要注意到销量的上升是有代价的。Lumia 1020 在一部分技术精英中很受欢迎，但其他人可能很少。

你能在低端手机上建立一个手机帝国吗？苹果做到了相反的事情，所以这也是可能的。
