# 脸书打击垃圾应用，给游戏一个自己的仪表盘链接

继昨天与开发者的会议之后，脸书[宣布将对其主页进行一些改变](https://web.archive.org/web/20221006195629/http://blog.facebook.com/blog.php?post=166797817130)。正如[所料](https://web.archive.org/web/20221006195629/http://www.beta.techcrunch.com/2009/10/28/d-day-for-facebook-app-developers/)，最大的变化之一是试图对抗应用程序垃圾。

你注册的应用程序的通知正在被缩减，这样你就可以选择只查看那些你的朋友明确发送给你的通知。这应该可以消除所有那些关于乔伊把一些虚拟香蕉作为礼物送给你不关心的人的随机更新。脸书还将允许用户指定哪些应用程序可以通过电子邮件联系他们，以及出于什么目的，就像当你注册任何网站时，他们会问你是否可以给你发送营销通讯。

然而，应用程序将更容易访问。“应用程序”链接将出现在左侧导航栏中。点击它会显示你所有的脸书应用。有迹象表明，社交游戏在脸书越来越重要，“游戏”将在左边的仪表板上有自己的链接，尽管它们是一种应用程序。游戏在这方面很特别。
