# 又一个开放数据基金会？

![](img/97369ce52cf2f5b8e97fea1ea77a6ef0.png "jaltaconference")

[克里斯·萨阿德](https://web.archive.org/web/20221208223447/http://chrissaad.wordpress.com/),[数据可移植性](https://web.archive.org/web/20221208223447/http://www.dataportability.org/)项目[的联合创始人](https://web.archive.org/web/20221208223447/http://chrissaad.wordpress.com/2008/07/23/open-data-foundation/)发布消息称，明天在 OSCON，一个新的开放~~数据~~网络基金会将由大卫·雷科登等人宣布成立。

新基金会的目标是制定实际的数据规范，围绕数据可移植性的法律结构，并帮助推广 set 格式。Saad 表示，该倡议不同于数据可移植性项目，因为它是围绕特定技术和法律实施的细节，而不是来自数据可移植性的更广泛的宣传工作:

**更新**:大卫·雷科登[回应说他明天不在](https://web.archive.org/web/20221208223447/http://daveman692.livejournal.com/338020.html) OSCON。

**更新 2** : David Recordon 已经确认，开放网络基金会将于周四上午发布。
