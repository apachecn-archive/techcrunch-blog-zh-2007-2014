# 报道:富士通下月将在日本推出世界首款 Windows 芒果手机

首先，富士通-东芝在 5 月份发布了世界上第一款 Symbian/Windows 7 双引导手机，现在:日本商业日报日经新闻报道，该公司准备推出世界上第一款运行 Windows Phone Mango 的手机，这是微软[在 5 月份正式发布的移动操作系统。](https://web.archive.org/web/20230204214731/https://techcrunch.com/2011/05/24/microsoft-officially-announces-windows-phone-7-1-mango-with-500-new-features/)

尽管微软宣布第一批芒果手机不会在秋天之前上市，日本移动运营商 [KDDI](https://web.archive.org/web/20230204214731/https://techcrunch.com/tag/kddi/) 最早将于下个月在当地市场推出所谓的 IS12T。

据日经新闻报道，该设备将具有防水功能，配备 3-7 英寸的液晶显示屏，并预装脸书和 Skype(在日本，KDDI 是这两家公司的战略合作伙伴)。

IS12T 的价格标签为“380-505 美元”。我们会随时通知你，也包括富士通-东芝可能发布的公告，如果他们的芒果手机也将在日本以外的市场上市。
