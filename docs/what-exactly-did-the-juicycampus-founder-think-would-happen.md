# JuicyCampus 创始人到底认为会发生什么？

 [](https://web.archive.org/web/20221007011409/http://www.crunchbase.com/company/juicycampus) 我不确定 [JuicyCampus](https://web.archive.org/web/20221007011409/http://www.juicycampus.com/) 创始人马特·依万斯特(Matt Ivester)在推出大学生八卦网站时期望什么，但显然事情正在[失控](https://web.archive.org/web/20221007011409/http://www.deepjiveinterests.com/2008/03/02/juicycampus-founder-pleads-cant-you-just-stop-being-mean-to-each-other-please/)。可以匿名发帖的人很“刻薄”尽管有一些法律保护，但依万斯特可能会对诽谤和中伤诉讼感到有点紧张。

在周五的一篇博客文章中，[他说](https://web.archive.org/web/20221007011409/http://juicycampus.blogspot.com/2008/02/hate-isnt-juicy-letter-from-founder-of.html) *“已经发布的一些东西是卑鄙的，我们已经收到了声称在该网站上被诽谤的人的电子邮件，”*并补充道*“我们希望你让 JuicyCampus 变得多汁，而不是充满仇恨。”*

他还很快指出，他对网站上说的任何事情都不承担责任:*“请注意，Juicy Campus 不是网站上出现的帖子的作者。更确切地说，Juicy Campus 是一个交互式计算机服务的提供商。因此，根据《美国法典》第 47 编第 230 条，Juicy Campus 对用户发布的内容不承担任何责任。*

随着用户匿名发帖，指责他人猥亵儿童，并讨论谁是学校里最淫荡的女孩，页面浏览量可能会上升。但是马特将会面对来自愤怒的学生和家长的几乎持续不断的法律威胁。呼吁只是友好并不能阻止它。这些法律保护可能不像他想的那样防弹。当错误的政客的女儿受到攻击时，整个事情可能会分崩离析。

菲利普·卡普兰在 2000-2001 年和 FuckedCompany 处理了几乎完全相同的问题。马特，你可能想给他打个电话，让他推荐一个好律师。
