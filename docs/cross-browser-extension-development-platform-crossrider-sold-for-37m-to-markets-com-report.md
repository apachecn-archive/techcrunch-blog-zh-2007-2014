# 跨浏览器扩展开发平台 Crossrider 以 3700 万美元出售给 Market.com:已确认

据报道，Crossrider 已经以 3700 万美元的价格卖给了 Market.com，这是一个平台，开发者可以在 Chrome、Internet Explorer、Safari 和 Firefox 等不同的网络浏览器上进行扩展。以色列报纸 [Globes](https://web.archive.org/web/20221129210717/http://www.globes.co.il/serveen/globes/docview.asp?did=1000806538) 首先注意到了出售的消息；[国土报](https://web.archive.org/web/20221129210717/http://www.haaretz.com/business/start-up-crossrider-is-sold-for-37m.premium-1.485270)随后报道了买家。我们已经联系了该公司请其置评。

更新:首席执行官 Koby Menachemi 已经确认了这笔交易，但也澄清了这笔交易是卖给 Market.com 的。《国土报》早前的报道称，买家是交易平台 Markets.com。Market.com 专注于电子商务，属于同一个母公司。该服务将一如既往地继续存在。

Crossrider 成立于 20 个月前，只有[在今年早些时候推出了测试版](https://web.archive.org/web/20221129210717/https://beta.techcrunch.com/2012/06/28/crossriders-cross-browser-extension-development-platform-comes-out-of-beta/)，获得了大约 200 万美元的支持。但它显然触及了一个热点——网络浏览器的扩展，人们用它来定制他们与网络的交互方式，并增加他们浏览器的功能。

11 月，Crossrider 表示，其平台上的~~开发者~~用户突破了 [2 亿大关](https://web.archive.org/web/20221129210717/http://blog.crossrider.com/post/36293997690/say-hello-to-our-200-000-000th-user)；今天，这个数字超过了 2.17 亿。这不仅表明了人们对开发浏览器扩展的巨大兴趣，也为那些为开发者提供开发这些扩展并避免处理碎片化问题的方法的公司提供了机会——这与平台的吸引力不同，平台让开发者可以在不同的移动平台上制作和部署应用。

虽然还不清楚为什么一个金融交易网站会有兴趣收购一个跨平台扩展开发者，但这可能是一个吸引开发者人才的案例，或者可能是 Markets.com 业务的其他部分——它由企业家 Teddy Sagi 的 Safecap Investments 所有。

该公司的平台可用于开发许多强大的服务，包括为电子商务服务的套利。据《国土报》报道，Crossrider 的另一个产品是浏览器扩展,“当用户在 Amazon.com 上查看某个特定产品时，会打开一个窗口，并在另一个网站上以更低的价格提供该产品。如果销售成功，CrossRider 将分享收入。

Crossrider 过去的支持者包括柳文欢·泽夫，他是 Orens Capital 的创始合伙人，也是 Apax Partners 的前普通合伙人。《国土报》称，由于公司在出售前获得的资金很少，联合创始人首席执行官科比·梅纳赫米和首席技术官 Shmueli Ahdut 拥有 Crossrider 超过一半的股份，并将从出售中各获得 900 多万美元。
