# 大数据巨擘穆适马从红杉资本通用大西洋融资 1.08 亿美元

[Mu 适马](https://web.archive.org/web/20230315015900/http://www.mu-sigma.com/)，一家为企业客户提供分析和数据驱动的决策支持服务的供应商，[已经在由成长型股权公司](https://web.archive.org/web/20230315015900/http://www.marketwire.com/press-release/mu-sigma-closes-108-million-investment-round-led-global-growth-investor-general-atlantic-1602123.htm) [General Atlantic](https://web.archive.org/web/20230315015900/http://www.crunchbase.com/company/general-atlantic) 牵头的一轮融资中筹集了1.08 亿美元。

大约半年前，穆适马[从红杉资本](https://web.archive.org/web/20230315015900/https://techcrunch.com/2011/06/06/mu-sigma-helps-companies-analyze-big-data-raises-25-million-from-sequoia/)[筹集了 2500 万美元](https://web.archive.org/web/20230315015900/http://www.crunchbase.com/financial-organization/sequoia-capital)，红杉资本也参与了此轮融资。

Mu 适马总部位于芝加哥，1500 多名员工大多在印度班加罗尔工作，是一家专业服务公司，帮助企业分析[【大数据】](https://web.archive.org/web/20230315015900/http://en.wikipedia.org/wiki/Big_data)以“将决策支持制度化”。

Mu 适马在一份声明中说，该公司已经实现盈利，能够自行为运营融资，但它筹集了更多风险资本来“加速增长”。

一部分新资本将用于购买现有股东持有的股份，但该公司没有透露哪些股东以及 1.08 亿美元注入资金中有多少将用于回购。

穆适马表示，所有现有股东都将保留公司股份。
