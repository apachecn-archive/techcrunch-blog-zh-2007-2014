# 瑞典粉丝自制字幕网站被版权警察关闭

一个应该让暮光之城/科幻作家们暂停的举动是，瑞典版权执法机构，名为[intr ngssernskning](https://web.archive.org/web/20221005100039/https://sv.wikipedia.org/wiki/Intr%C3%A5ngsunders%C3%B6kning)，突袭并查封了 [Undertexter.se](https://web.archive.org/web/20221005100039/http://undertexter.se/) 的服务器，这是一个专门收集粉丝制作的英语和瑞典语字幕的粉丝网站。突袭发生在昨天早上的斯德哥尔摩。

Fansub 网站提供特殊编码的文本文件，观众可以使用这些文件来同步音频和翻译的字幕。他们显然很受盗版内容消费者的欢迎，但即使是网飞在芬兰也用一些粉丝为他们的电影添加字幕。更重要的是，字幕并不是 DVD 或蓝光光盘上的文本副本，而是大部分屏幕动作的再现(这通常非常糟糕)。然而，他们仍然是由某些节目和电影的粉丝制作的无偿的爱的劳动，更类似于混音而不是批发盗版，他们在服务不足的市场上是非常有价值的。

简而言之，好莱坞完全选错了要麻烦的人。顺便说一下，[数据仍然在线](https://web.archive.org/web/20221005100039/http://www.opensubtitles.org/sv)。

Undertexter.se 不再直播。他们在自己的脸书账户和域名上发布了一条信息，由 [Falkvinge](https://web.archive.org/web/20221005100039/http://falkvinge.net/2013/07/10/fan-subtitle-site-raided-by-copyright-industry-aided-by-police/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+Falkvinge-on-Infopolicy+%28Falkvinge+on+Infopolicy%29) 翻译:

今天早上，警方突袭了 Undertexter.se，查封了服务器和台式机。我们在网站上工作的人并不认为对 dialog 的解释是非法的，尤其是在免费分享的时候。这次袭击的幕后黑手 Henrik Pontén(瑞典版权产业的主要亲信)不同意这种观点。抱歉，好莱坞，这张牌打错了。我们永远不会投降。[……]我们必须尽一切力量阻止这些反海盗。[…]很快会有更多信息。/ UT 船员

瑞典盗版党也发表了一份声明，称“今天的垄断以一种完全不合理的方式扼杀和抑制了创造力。对 undertexter.se 的突袭是又一个证据，表明彻底改革版权垄断的时候到了。”

虽然有很多关于自由和控制手段的高调谈论，但显然粉丝是好莱坞最不关心的。他们认为让瑞典政府调出一组对电影业无关紧要的文本文件是合适的，这就像让一位为韦斯利·克拉舍(Wesley Crusher)亲吻数据画肖像的艺术家关门一样——这显然令人不快，它破坏的善意远远超过它节省的金钱。

2007 年，一个类似的案例出现在波兰，当时一个受欢迎的 fansub 网站被关闭。波兰当局最近结案了。来自[DOP repramy](https://web.archive.org/web/20221005100039/http://www.dobreprogramy.pl/Prokuratura-umorzyla-sledztwo-w-sprawie-napisy.org,Aktualnosc,41631.html):

Zabrze 检察官办公室副主任 Elizabeth owicka 今天宣布，检方已决定停止审理此案。作出这一决定的理由是，译者对其行为的非法性一无所知。

简而言之，译者们并不认为这是错误的——最后，我希望更多的版权执法者同意这不是错误的。
