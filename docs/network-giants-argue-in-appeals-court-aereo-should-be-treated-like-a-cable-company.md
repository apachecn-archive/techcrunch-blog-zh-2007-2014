# 网络巨头在上诉法庭争辩:Aereo 应该像有线电视公司一样对待 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/11/30/network-giants-argue-in-appeals-court-aereo-should-be-treated-like-a-cable-company/>

今天，纽约南区法院听取了来自福克斯、美国广播公司、Univision 和少数其他有线广播提供商的口头辩论，他们希望看到流媒体网络电视初创公司 Aereo 的业务被关闭。这只是一场正在展开的法律战的最新进展，这场法律战是关于这家初创公司的服务是否侵犯了网络版权，该服务使用数百个微型天线向联网设备传输直播和录制的电视节目。

原告今天改变了立场，关注 Aereo 作为转播服务的存在，而不是关注该服务类似 DVR 的功能。

换句话说，原告们不太关注该案的公共表演方面(其中心是一个单一的固定副本被传输给多个用户)，而是更关注 Aereo 是否曾经有过传输信号的许可。

地区法院最初在初步禁令听证会中裁定[有利于](https://web.archive.org/web/20230118114855/https://techcrunch.com/2012/03/05/aereo-for-the-win/)Aereo，这将暂停 Aereo 的业务，直到主要审判结束，这可能需要几个月甚至几年。原告现在对该裁决提起上诉，希望在主要案件的过渡期间再次关闭她。

在第一次初步听证会上，Aereo [说服法院](https://web.archive.org/web/20230118114855/https://techcrunch.com/2012/03/01/aero-lawsuit-see-you-in-court/)不存在公共表演问题，坚持认为用户捕捉内容的单独副本，包括直播和录制，以供以后观看，而不是一个单一的固定副本被一次重播给多个用户。该公司援引 Cablevision 的先例称，Cablevision 的远程 DVR(将单个剧集的录制内容从用户的客厅转移到服务器)并不违反版权法，Aereo 声称其服务做了同样的事情。

在今天的口头辩论中，电视网转而关注这样一个事实，即 Cablevision 首先拥有播放内容的长期许可证，认为 Aereo(没有任何此类许可证)无权捕捉并向其用户转播空中信号。这些网络甚至引用了 Aereo 的上诉简报中的一句话，承认这家总部位于纽约的初创公司确实为用户传输了信号。

在简报的第 28 页上，它说“Aereo 系统中的所有传输都来自消费者的唯一拷贝，而不是直接来自输入的广播信号。”

另一方面，Aereo 的首席律师认为，个人用户是捕捉和转发内容的人，这些内容是用户为用户制作的唯一副本。

听证会后，我与原告律师保罗·史密斯(詹纳&布洛克 LLP 律师事务所)进行了交谈。我问他 Cablevision 和 Aereo 的主要区别是什么。

“有线电视是一种获得许可的转播服务，”史密斯先生说。“它唯一做的事情就是把电线从机顶盒移到服务器。在这种情况下，法院只关注从复制品到使用者的传播。

“Aereo 是未经许可的，所以它所做的一切，从接收内容到从服务器传输内容，都是未经许可的。在这种情况下，我们会说‘你不能看了允许他们这么做的法令，就说这样做没问题，因为他们正在复制他们发送的所有内容。’这基本上就否定了规则，因为每个人都会这么做。"

这些网络仍然不同意 Cablevision 的先例，但他们承认，Cablevision 所做的事情是以同样的方式获得许可的，所有有线电视公司都有播放网络内容的许可，无论是直播还是存储以备后用。

“Aereo 应该得到和其他有线电视公司一样的待遇，因为它们就是这样，”史密斯说。

Aereo 的律师解释说，用户实际上是以同样的方式访问内容，如果你在电视机上有兔子耳朵，除了兔子耳朵是远程的(就像有线电视案件中的 DVR 记录是远程的)。法官似乎对此持抵制态度，因为这些传输实际上是通过 Aereo 作为一种服务来捕获的，而不是客户购买并在电视上安装天线。

但 Aereo 建立了包括数百个微型天线的技术，并选择制作公共可用内容的个人副本(如在中，通过公共可访问的无线电信号免费传输)。该公司承认，它试图维护法律，尤其是 Cablevision 甚至 ASCAP 的先例。

法院和网络提供商似乎都不赞成这个漏洞初创公司的想法——法官甚至将这项服务比作避税的公司，“这不同于逃税，”他说。然而，当一家公司被特别设计为在法律范围内时，很难辩称它违反了法律。

法院将很快做出决定，在主要听证会进行期间，Aereo 是否需要关闭服务。
