# 德国迅速采取行动

诺基亚今天宣布，SNAP 手机游戏将于今年秋季向德国消费者推出。SNAP 游戏目前在亚洲和北美销售。

流行的手机将会安装免费的演示游戏。要下载完整的游戏，用户只需点击“购买”链接。然后发送带有链接信息的文本消息。

SNAP Mobile 的负责人 Antoine Doumenc 表示:“在亚太地区取得持续成功后，我们显然将目光投向了欧洲市场。德国是我们的第一个目标国家，我们计划在年内扩展我们在多个欧洲分销渠道的服务。此外，在不久的将来，我们将为德国消费者提供更多通过休闲游戏建立联系的机会。”

Sudoku 和 JellyPop 是 SNAP Mobile 的 Pocket Party 应用程序支持的两种流行游戏。数独产生随机的难题，创造了众多的挑战。JellyPop 是一款简单易学但很难掌握的益智游戏，有单人和多人模式。高分、排名、好友列表、头像、代币、聊天都融入了口袋党宇宙。朋友可以动员和游戏。
