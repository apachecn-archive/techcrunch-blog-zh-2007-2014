# 尼可拉斯·曾斯特罗姆:“点对点今天没有破坏性”

上周，在 Disrupt Beijing 上，莎拉·莱西采访了 Skype 联合创始人兼 Atomico 投资人尼可拉斯·曾斯特罗姆。(可以看上面视频里完整的炉边聊天。)Zennstrom 是建立一系列基于 P2P 技术的创业公司(Skype、Kazaaa、Joost)的先驱，但在采访快结束时，他说:“如今 P2P 并不具有破坏性。”有时候用有意义，有时候没意义。许多网络是混合的。但是，最初的竞争优势和绕过带宽瓶颈的方法不再那么重要。Zennstrom 了解到，随着 Joost 的出现，宽带成本大幅下降，直接向用户播放视频变得更加经济。

Joost 失败的另一个原因是，在与 Hulu 的竞争中,“现有公司做得非常出色”。Joost 永远无法为其第三方网络获得最好的内容。

Skype 在中国一直很大，他解释了 Skype 是如何打入市场的。他还谈到风险企业需要如何在全球范围内进行变革。“市场已经从一个投资者的市场变成了一个企业家的市场，”他回忆道，当时他向风投推销 Skype，但没有人真正明白，尤其是在欧洲。“需要改变的是，传统的风险资本投资者非常本地化，”他说，“但如今科技行业是一个全球性行业。你往往会看到公司成为全球领导者或地区失败者。”这是非常二元的。
