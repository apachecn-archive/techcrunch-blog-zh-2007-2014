# 太糟糕了！把你从尴尬处境中拯救出来的应用程序 Tickle 并不真正存在

陷入尴尬境地？一个新发布的名为 [Tickle](https://web.archive.org/web/20221210070056/http://www.usetickle.com/) 的应用程序承诺给你一个借口，以紧急但虚假的电话形式离开，你可以通过“挠痒痒”(用手指摩擦)你的 iPhone 来狡猾地激活它。听起来不错，对吧？事实证明，这好得令人难以置信。[尽管](https://web.archive.org/web/20221210070056/http://www.news.com.au/technology/gadgets/tickle-app-will-help-you-get-out-of-awkward-situations/story-fn6vihic-1226847114767?from=public_rss) [什么](https://web.archive.org/web/20221210070056/http://www.bustle.com/articles/17420-tickle-app-provides-perfect-excuse-for-leaving-awkward-situations-and-i-absolutely-must-have-it-now) [你已经](https://web.archive.org/web/20221210070056/http://www.huffingtonpost.co.uk/2014/03/07/tickle-phone-app_n_4917646.html) [读过](https://web.archive.org/web/20221210070056/http://www.gizmodo.com.au/2014/03/tickle-a-weird-app-to-get-you-out-of-weirder-situations/)在一个[号](https://web.archive.org/web/20221210070056/http://appadvice.com/appnn/2014/03/appadvice-daily-gmail-update-aluminum-bumpers-done-right-and-an-app-for-awkwardness) [的](https://web.archive.org/web/20221210070056/http://elitedaily.com/news/technology/the-tickle-app-is-exactly-what-you-need-to-get-out-of-awkward-situations-video/) [地点](https://web.archive.org/web/20221210070056/http://www.dailydot.com/lol/tickle-app-awkward-situations/)的网页上，这个应用并不真实。

但是来吧，你会喜欢这个主意的。

“Tickle”是 Firespotter Labs 的联合创始人亚历克斯·康奈尔(Alex Cornell)的最新恶搞视频，他之前曾用一款名为 Jotly 的恶搞应用[娱乐过我们，该应用因其移动、本地、社交(](https://web.archive.org/web/20221210070056/https://beta.techcrunch.com/2011/10/18/jotly-parodies-our-mobile-local-social-app-obsession/) [SoLoMo](https://web.archive.org/web/20221210070056/https://beta.techcrunch.com/2012/03/24/die-solomo-die/) )而受到科技行业的指责！)2011 年的应用痴迷。

(而且 Jotly 后来成了[真事](https://web.archive.org/web/20221210070056/https://beta.techcrunch.com/2011/12/13/jotly-lives-parody-app-goes-live-in-itunes/)，因为，为什么不呢？)

扰乱冠军 UberConference 背后的公司 Firespotter Labs 也有一个关于扰乱杯与该公司合作的非常棒的视频，康奈尔大学也参与了。

康奈尔说，他最近离开了他在费尔斯波特的联合创始人职位，去追求电影和其他“怪异的互联网项目”的事情，他与另一位诙谐的同谋者菲尔·米尔斯合作制作了这个滑稽的视频。

“我喜欢探索怪异和新的产品开发方式的想法。我的技能是设计和视频；康奈尔说:“我不会编程。“我喜欢把一个想法扔进这个世界，相对成形并为在互联网上生存做好准备，然后看看会发生什么，”他补充道。(嘿，[我也是](https://web.archive.org/web/20221210070056/http://sarahintampa.tumblr.com/post/74129836731/random-startup-ideas)——但是我的没那么好笑。)

仅仅因为胳肢不存在，并不意味着它不应该存在。正如康奈尔指出的，它有一个实际的用例，所以如果任何人想建立这个东西，他们有他的祝福。
