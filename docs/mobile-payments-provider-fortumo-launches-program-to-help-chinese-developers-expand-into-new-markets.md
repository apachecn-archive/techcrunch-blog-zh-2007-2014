# 移动支付提供商 Fortumo 推出计划，帮助中国开发商拓展新市场

移动支付提供商 Fortumo 推出了一个项目，为希望在其他国家发布应用的中国开发者提供“一站式服务”。该计划名为 [Fortumo Expand](https://web.archive.org/web/20220930111936/http://developers.fortumo.cn/expand/) ，为 Android、Windows Phone 和 Windows 8 开发者提供访问这家爱沙尼亚/美国公司在 80 个国家的移动运营商网络，以及本地化、测试、分销和营销服务。

Fortumo Expand 是对该公司与中国三大电信的[直接计费协议的制衡，该协议允许外国开发者为其应用程序提供运营商计费，从而帮助他们获得更大的立足点。在与西方客户如](https://web.archive.org/web/20220930111936/https://beta.techcrunch.com/2013/07/25/fortumo-china/) [Rovio](https://web.archive.org/web/20220930111936/http://www.rovio.com/) 、 [EA](https://web.archive.org/web/20220930111936/http://www.ea.com/) 、 [Gameloft](https://web.archive.org/web/20220930111936/http://www.gameloft.com/) 、 [Zeptolab](https://web.archive.org/web/20220930111936/http://www.zeptolab.com/) 和 [Badoo](https://web.archive.org/web/20220930111936/http://badoo.com/) 合作时，Fortumo 发现他们在中国的同行很难在国际上扩展，尤其是如果他们在其他国家没有合作伙伴来帮助处理语言、文化和商业惯例的差异。

Fortumo Expand 将由马尔蒂·穆斯蒂拉(Martti Mustila)领导，他曾是诺基亚全球开发者项目的负责人。
[![Fortumo Expand](img/93c05fdb06eae861e7fe9eadc24a3f88.png)](https://web.archive.org/web/20220930111936/https://beta.techcrunch.com/2013/10/23/mobile-payments-provider-fortumo-launches-program-to-help-chinese-developers-expand-into-new-markets/fortumo-expand/) 
“我们希望帮助中国开发者获得知名度。对于 Android 开发者，我们帮助他们在独立的应用商店和运营商商店发布。在最好的情况下，如果一个应用做得很好，我们也可以为他们建立一个促销活动，”穆斯蒂拉说。Fortumo Expand 的货币化计划与该公司的其他开发者项目遵循相同的模式。移动运营商从中提成，而 Fortumo 收取 2%至 5%的佣金。

Fortumo 已经与 30 家中国应用程序开发公司合作，主要是在游戏领域。注册 Fortumo Expand 的开发者可以决定他们想进入哪些国家。Fortumo 在巴西、俄罗斯、拉丁美洲和东南亚尤其活跃，那里的大多数消费者通过运营商而不是信用卡进行移动支付。

除了获得 Fortumo 的 300 家移动运营商的国际网络，开发者还可以选择在 23 个第三方应用商店分发他们的应用程序，并接收月度报告，以便他们可以看到哪些新兴市场最有收入潜力。Fortumo Expand 通过 [Symbio](https://web.archive.org/web/20220930111936/http://www.symbio.com/) 为开发者提供第一个应用的免费本地化和测试服务，并从 [AdDuplex](https://web.archive.org/web/20220930111936/http://www.adduplex.com/) 为 Windows Phone 和 Windows 8 应用提供免费广告。
