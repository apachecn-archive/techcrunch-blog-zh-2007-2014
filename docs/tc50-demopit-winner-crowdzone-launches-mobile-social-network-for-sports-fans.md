# TC50 Demopit 冠军 CrowdZone 为体育迷推出移动社交网络

![](img/7b083bd9425424f3e5b75b9391549572.png)在去年秋天的 TechCrunch50 大会上， [Chyngle 凭借其为体育场馆开发的品牌移动营销应用程序，被大众选为](https://web.archive.org/web/20221006203901/https://beta.techcrunch.com/2009/09/15/tc50-demopit-winner-chyngle-creates-in-pocket-mobile-marketing-apps-for-stadiums/)最佳演示初创公司。现在，Chyngle 已经与 [mobiTeris](https://web.archive.org/web/20221006203901/http://www.mobiteris.com/) 合作，将其应用更名为 [CrowdZone](https://web.archive.org/web/20221006203901/http://www.crowdzone.com/) ，这是一款免费的 [iPhone 应用](https://web.archive.org/web/20221006203901/http://itunes.apple.com/us/app/crowdzone/id359819981?mt=8)，为体育赛事中的观众提供移动社交网络。

CrowdZone 的应用程序允许用户“加入”一个皇冠或在体育赛事周围发起一群人。虽然该应用程序是基于位置的，但你也可以从你的教练那里参与进来，即使你没有参加某个特定的活动，也可以使用该应用程序的功能。在应用程序的人群中，你可以签到游戏，类似于在流行的移动社交网络 Foursquare 上签到。登录后，你可以看到其他正在使用该应用程序的人，以及他们在人群中的位置。

你可以与你的观众分享评论和图片，并参加“粉丝活动”，如 cheers 或 the wave。当您参与申请时，您将获得活动积分和徽章，目的是在活动中获得“最有价值人群”的地位。

虽然与运动队和体育场的合作不是该应用功能的必要条件，但 CrowdZone 已经有了一个专业运动队。NHL 的卡罗莱纳飓风队将通过电视和广播广告以及他们的网站、脸书和推特粉丝频道向他们的粉丝推广 CrowdZone 的应用程序。

Chyngle 和 CrowdZone 的联合创始人托德·沙利文(Todd Sullivan)表示，收入尚未实现，但这家初创公司正在寻求通过各种方式将该应用货币化。沙利文说，团队可能会利用 CrowdZone 网络来识别和发送有影响力的粉丝。沙利文补充说，CrowdZone 还为团队提供了一个渠道，让他们与粉丝围绕虚拟和现实世界的商品进行交易。CrowdZone 从人群行为和位置数据中生成的实时分析可以为团队提供有价值的见解。

CrowdZone 将面临的最大挑战之一实际上是让用户下载并使用该应用程序。但如果这家初创公司继续获得专业运动队的认可和宣传，这款应用可能会成为铁杆体育迷的利基移动社交网络。
