# 热衷于…为什么女性擅长社交媒体(TCTV)

它必须发生。可以说，社交媒体已经性别化了。本周[最畅销非小说类榜单](https://web.archive.org/web/20230205000240/http://online.wsj.com/article/SB10001424053111904265504576566661009105054.html?mod=WSJ_Books_LS_Books_13)中的新成员是[薇琪·米拉佐的](https://web.archive.org/web/20230205000240/http://www.crunchbase.com/person/vicki-milazzo-2) [邪恶的成功在每个女人心中](https://web.archive.org/web/20230205000240/http://www.amazon.com/Wicked-Success-Inside-Every-Woman/dp/1118100522)，这是一本女性自助书籍，在一定程度上认为女性比男性更有效地使用社交媒体。

上周晚些时候，当我们通过网络聊天时，维姬·米拉佐告诉我，女性“擅长建立关系”，因为她们在家庭中“几乎管理一切”。她表示，与倾向于将合作视为弱点的男性相比，女性是天生的关系建立者，因此天生擅长社交媒体。米拉佐向我解释说，这是因为女人同时使用大脑的两侧，而男人只使用一侧。

作为一个不擅长建立关系，也很少使用左右脑的人，我不得不承认我同情米拉佐的观点。但我怀疑，并非所有 TechCrunchTV 的观众都会同样开明。那么，Vickie Milazzo 是对的吗——女性天生就比男性更有效地使用社交媒体吗？
