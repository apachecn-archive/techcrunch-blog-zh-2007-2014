# 社交实验室筹集 700 万美元帮助零售商管理社交商务

[社交实验室](https://web.archive.org/web/20230203085831/http://www.sociablelabs.com/)是一家帮助在线零售商利用社交商务增加推荐流量和销售额的初创公司，已经在由 Battery Ventures 牵头的 B 轮融资中筹集了 700 万美元。这家初创公司最初由脸书通过 fbFund 提供种子资金。

社交实验室专注于通过脸书和其他社交网络为零售商提供和测量传统的电子商务指标，如流量、转化率和销售额。这家初创公司为零售商提供了一套托管应用程序，用于社交化买家的意见，并在购买时根据买家朋友的建议帮助发现产品。

这家初创公司表示，其客户报告称，与标准社交按钮(如“喜欢”或“分享”)相比，网站上的社交分享增加了 20 倍。分享的增加使得流量回到他们的网站，比脸书粉丝页面的流量高出 300%。
