# 在 Seed Labs 之后，Cvent 以 1000 万美元的价格收购了 CrowdCompass，在移动活动应用上加倍努力

对你从未听说过的世界上最大的活动管理公司之一的 Cvent 来说，这是活跃的七天。该公司为在线活动注册、管理等提供基于网络的软件，一直致力于超越 B2B 服务。自从去年筹集了 1.36 亿美元以来，Cvent 重新关注更多面向消费者的活动解决方案，尤其是在移动领域。作为这一新方向的一部分， [Cvent 上周进行了有史以来的首次收购](https://web.archive.org/web/20221208084026/https://beta.techcrunch.com/2012/06/13/event-management-giant-cvent-scoops-up-seed-labs-to-help-it-go-mobile/)，以 420 万美元收购了总部位于奥斯丁的活动应用开发商 [Seed Labs](https://web.archive.org/web/20221208084026/http://www.seedlabs.com/) 。

鉴于其新的消费者愿望、风险投资支持和最近的强劲增长，[我们上周表示](https://web.archive.org/web/20221208084026/https://beta.techcrunch.com/2012/06/13/event-management-giant-cvent-scoops-up-seed-labs-to-help-it-go-mobile/)“在不久的将来，Cvent 继续进行战略收购并不令人惊讶”。虽然我们是占卜师，但没想到会这么快。今天，活动软件公司宣布了七天内的第二次收购，以 1000 万美元收购了波特兰的应用开发商 [CrowdCompass](https://web.archive.org/web/20221208084026/http://www.crowdcompass.com/) 。

CrowdCompass 已经筹集了大约 130 万美元的资金，就像是会议的种子实验室。该应用程序开发商提供了一套移动应用程序，旨在通过使活动、会议和贸易展览变得简单、社交化和易于在旅途中分享来加快这些活动、会议和贸易展览的受欢迎程度。

具体来说，这意味着活动和会议组织者可以创建定制品牌的移动应用程序，将时间表、地图、参展商列表、赞助机会、二维码扫描、消息传递和社交网络整合到一个解决方案中。Cvent 已经为客户提供了创建移动优化网站的能力，但此次收购将使该公司能够提供原生体验，让组织者快速为他们的商业活动创建 iPhone，iPad，Android 和黑莓应用程序。

CrowdCompass 由戴夫·尚利(Dave Shanley)、布莱恩·费尔德(Brian Felder)和汤姆·金斯利(Tom Kingsley)于 2009 年创立，已经开发了 435 款应用，去年活动参与者下载了超过 50 万次。该公司为 E*Trade、戴姆勒和 Intuit 等公司以及美国律师协会和总承包商协会等行业组织和协会开发了定制应用程序。反过来，访问 Cvent 在 90 多个国家的 10，000 个客户的名册将允许 compass 在比其他情况下更大的规模上运行。
