# DataSift 提供对历史推文的访问

Twitter 开发者长期以来一直渴望获得历史推文。目前，根据关键词搜索，他们最多只能得到 7 天的信息。Twitter 的数据合作伙伴之一 DataSift 目前为开发者和第三方提供实时访问完整 Twitter fire hose 的[,很快也将提供历史推文。开发者可以从今天开始注册 DataSift 的](https://web.archive.org/web/20221226062809/https://techcrunch.com/2011/11/16/datasift-launches-powerful-twitter-data-analysis-and-business-intelligence-platform/)[历史数据](https://web.archive.org/web/20221226062809/http://datasift.com/historic)的 Alpha(实际服务将在明年第一季度开始推出)。

DataSift 的历史服务将为开发人员、社交媒体监测公司、营销人员和品牌提供 60 天的 Alpha 推文访问权限，可以对这些推文进行分析和过滤，而不仅仅是简单的关键字搜索。当这项服务在明年晚些时候更广泛地推出时，它将追溯到两年前。DataSift 允许各种类型的数据分析，因为它将所有的推文倒入一个结构化的数据库中。所以你可以给它这样的查询:“给我所有不关注@techcrunch 的人提到 TechCrunch 的推文”或者“所有提到时尚的英国女性。”

该公司已经在内部每天收集 1tb 的数据——这是每天 2.5 亿条推文的消防水管产生的数据——迄今为止总计 400。“这是一个真正的‘大数据’引擎，我们正在使它变得简单，我们正在利用 map reduce，但这是我们自己的定制处理引擎，”DataSift 创始人 Nick Halstead 说，他指的是该服务部分基于的 Hadoop 技术。
