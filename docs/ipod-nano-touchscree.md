# 苹果缩小了 iPod Nano，但设法增加了触摸屏

![](img/cd40134369255e71a8a795b77b2989cd.png "gallery-big-01-gen6")

今天，苹果公司[宣布](https://web.archive.org/web/20221202100108/http://www.crunchgear.com/2010/09/01/live-from-apples-september-2010-press-event/)下一代 iPod nano。正如[泄露的](https://web.archive.org/web/20221202100108/http://www.crunchgear.com/2010/08/26/leaked-ipod-cases-show-new-ipod-nano-design/)，新版本更小更方。传闻中的[触摸屏](https://web.archive.org/web/20221202100108/http://www.crunchgear.com/2010/07/09/whats-apple-going-to-do-with-a-3x3-cm-touchscreen-display-new-ipod-new-macbook-pro-touchpad/)也在车上——尽管它相当小(小 46%—轻 42%)。苹果认为它*非常*可用。

与过去的 nano 类似，新的 nano 将具有调频广播/实时暂停和 iTunes 标记，苹果称之为“更智能的广播方式”。此外，新的 nano 将不会有摄像头-不知道有多少人使用了摄像头，尽管如此，我们讨厌看到它消失。

![](img/e5e48f1ef8cc5fb30889a2800bf021ad.png "gallery-big-05-gen6")

这也难怪为什么[新洗牌](https://web.archive.org/web/20221202100108/http://www.crunchgear.com/2010/09/01/the-ipod-shuffle-goes-retro/)被做得*甚至*更小了；新款 nano 几乎和旧款 shuffle 一样小。新的 Nano 完全基于手势，运行 iOS4，这意味着它应该快速且易于使用。nano 的存储容量保持不变，同样是 8GB 和 16GB，售价分别为 149 美元和 179 美元，有七种颜色。您可以今天预购，下周发货时就可以买到。

对于那些更喜欢运动生活方式的人来说，新款 nano 将非常棒:内置夹子、计步器，和以前一样，支持 Nike+ Sport kit 成为标准配置。电池寿命据说是 24 小时。虽然苹果公司正在为他们的返校促销推出免费的 Nano 和 Mac 购买，但新的 Nano 不包括在内。好吧，我们尽力了。
