# 脸书用并排布局重新设计信息

脸书[今天宣布](https://web.archive.org/web/20230209125116/http://newsroom.fb.com/News/A-New-Look-for-Facebook-Messages-1b2.aspx)已经重新设计了脸书信息，采用并排布局，搜索更方便。

新布局在左侧显示所有邮件的滚动菜单，在右侧显示所选邮件的完整内容。风格类似于许多电子邮件客户端。

脸书说，用户可以“用多张照片和表情符号让对话变得生动”照片长期以来一直是脸书内容的驱动形式，所以这并不奇怪，但这是一个很酷的功能。

这家社交网络公司声称，它允许用户通过发件人姓名或关键词来查找信息，从而简化了搜索，并通过添加键盘命令来简化导航。

脸书表示，更新将从今天开始在小范围内推出，并将在接下来的几周内向所有用户提供。

脸书工程公司[今天也发布了一条消息](https://web.archive.org/web/20230209125116/https://www.facebook.com/notes/facebook-engineering/under-the-hood-improving-messages/10151032941658920)，解释了他们在构建新信息布局时所面临的技术方面和挑战，他们称之为“水星项目”
