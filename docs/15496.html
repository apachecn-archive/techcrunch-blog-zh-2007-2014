大数据分析专家Tableau Software通过IPO筹集了2.54亿美元，股价收盘上涨64%；Marketo上市首日上涨78%，至23.10美元TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2013/05/17/big-data-visualization-goes-public-tableau-software-raises-254m-as-shares-pop-58-while-marketo-raises-85m/

陷入困境的脸书首次公开募股(IPO)已经过去一年了，公开市场上科技公司IPO的气氛明显没有那么激烈，尤其是对企业领域的公司而言。今天，不是一个，而是两个，Tableau软件和Marketo，正在纽约证券交易所上市。商业情报提供商 Tableau Software ，以“数据”交易，是今年最受期待的科技IPO之一，迄今为止，它没有令人失望。它将其首次公开募股定价为每股31美元，它，在早盘交易上涨58%，收盘时比开盘价高出64%，或每股50.75美元。

与此同时，基于云计算的营销服务公司Marketo将其IPO定价为每股13美元。它将作为MKTO在纳斯达克交易所交易，但在撰写本文时尚未交易。在早期活动中，它上涨了50%以上，然后继续攀升:收盘时为21.48美元和23.10美元，比IPO定价和交易价高出近78%。(我们一整天都在更新这些数字。)

总的来说，这两者是对企业服务市场和其中一些新兴趋势的有力支持。

Tableau Software，正如其股票代码明确暗示的那样，更多地是针对大数据游戏，提供可视化和分析，据称非技术人员也可以轻松使用。到目前为止，它仍然以可下载的内部软件形式提供大部分服务，而不是基于云的应用程序。

Marketo被定位为软件即服务，就像营销部门的销售人员一样，提供各种服务——集客营销、线索管理、社交营销、事件管理、即时CRM集成、销售仪表板以及营销ROI报告和分析——所有这些都在一站式云中商店中完成。

Tableau Software以每股31美元的价格筹集了约2.542亿美元，此前该公司首次公开募股的初始价格区间为23-26美元；这使其估值达到20亿美元。与此同时，Marketo以5.5亿美元的估值筹集了不到8500万美元的资金。

(顺便提一下，脸书的股票在去年损失了30%的价值，目前大约是每股26.25美元。)

Tableau的IPO与其他备受瞩目的企业上市相比如何？筹集的资金略低于企业安全公司帕洛阿尔托网络公司在2012年7月筹集的2.6亿美元。这与人力资源专家Workday在2012年10月的首次公开募股仍有一段距离，当时筹集了6.37亿美元。

tableau Software 10位数的IPO为Box和Twitter等其他10亿美元以上的科技公司的IPO奠定了基础。在此之前，Tableau已经从包括NEA和Meritech在内的投资者那里筹集了大约4500万美元( Crunchbase 给出的总额仅为1500万美元，但 Geekwire 称NEA在该公司的总投资为2900万美元，其 SEC文件讲述了另一个故事)。

相比之下，Marketo在六轮融资中筹集了1 . 08亿美元，投资者包括机构风险投资伙伴、InterWest Partners、Mayfield Fund、Storm Ventures和Battery Ventures。
