# Sobees 简化了 Windows 的本地 Twitter 客户端，集成了实时搜索

![](img/0bc1fb73e1010433dae06351890c4556.png)
我们已经写了关于 Twitter 客户端 [Sobees，](https://web.archive.org/web/20221006225359/http://www.sobees.com/)它正在努力创造市场上最好的社交媒体客户端，与 [TweetDeck](https://web.archive.org/web/20221006225359/http://www.tweetdeck.com/) 和 [Seesmic 竞争。今天，Sobees 发布了内置的 Windows 原生桌面应用程序的新版本。NET，完成实时搜索，重新设计和更多。](https://web.archive.org/web/20221006225359/http://seesmic.com/)

新的客户端包括对脸书、推特、MySpace、FriendFeed 和 LinkedIn 的支持(LinkedIn 是去年年底添加的)。最重要的增加是客户端的实时搜索功能，能够从应用程序中搜索 Twitter、Friendfeed、 [OneRiot](https://web.archive.org/web/20221006225359/http://www.oneriot.com/) 和 [FacteryLabs](https://web.archive.org/web/20221006225359/http://facterylabs.com/) 。Sobees 整合了其最新发布的[实时网络仪表盘](https://web.archive.org/web/20221006225359/http://search.sobees.com/)的元素，以支持客户端搜索。

Sobees 还为列添加了拖放技术，以便在客户机中改变列的位置。其他技术更新包括预览推文和地图，翻译消息，以及从推文中查看 TwitPic 上发布的图片。Sobees 将打破你与朋友的线程对话。

Sobees 正在一个拥挤的空间中竞争，每个客户都在继续创新，为用户提供更多管理社交媒体账户和实时网络的选择。例如，Seesmic 已经[整合了 Ping.fm，](https://web.archive.org/web/20221006225359/https://beta.techcrunch.com/2010/01/04/seesmic-acquires-ping-fm/)允许用户同时更新 50 多个不同的社交网络。现在[的 TweetDeck 在它的客户端中整合了](https://web.archive.org/web/20221006225359/http://eu.beta.techcrunch.com/2010/02/08/new-tweetdeck-version-puts-twitter-on-crack-bring-in-youtube-and-flickr/) YouTube 和 Flickr。目前，Sobees 没有任何移动客户端，但我们被告知 iPhone 和 Android 应用程序即将推出。
