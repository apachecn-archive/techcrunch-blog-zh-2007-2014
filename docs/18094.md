# 刚刚完成 IPO 的 Wix 收购以色列初创公司 Appixia，为 4200 万用户带来本地移动商务支持 TechCrunch

Wix 于 2006 年作为新一代创业公司的一部分出现，其使命是为网站创建做 Blogger、WordPress 和 Tumblr 为博客作者和内容创作者所做的事情。随着对易于使用的 DIY 网站建设者的需求日益增长，Wix 现在是网络上同类平台中最大的一个，另外两个是 Weebly 和 SquareSpace。现在有超过 4000 万人在使用免费的网页设计工具，11 月， [Wix 在纳斯达克](https://web.archive.org/web/20230129103627/https://techcrunch.com/2013/11/05/twitter-friends-ahead-of-its-public-debut-wednesday-wix-prices-ipo-at-16-50-per-share-with-valuation-near-800m/)首次亮相，这是该公司估值略低于 8 亿美元的首次公开募股的一部分，这是迄今为止以色列公司最大的一次公开募股。

今天，在上市以来的第一个大动作中，这家总部位于特拉维夫的公司宣布，它将收购 [Appixia](https://web.archive.org/web/20230129103627/http://www.appixia.com/) ，这是一家以色列初创公司，该公司开发了一个平台，旨在使零售商更容易将他们的商店移动化，并建立自己的本地“移动商务”应用程序。

Appixia 是微软在特拉维夫的创业加速器项目的毕业生，其平台旨在使小型夫妻店和中型零售商能够轻松地在现有的电子商务市场上快速增加移动店面。为此，这家初创公司的平台支持一系列平台，如 Magento、Prestashop、Zen Cart、Shopify 和易贝，自动与现有的实时商店和后端系统同步，这样商家就不必与在线商店分开管理他们的移动业务。

像 Wix for mobile 一样，Appixia 允许零售商建立兼容 HTML5 的移动网站和 iOS 和 Android 的原生应用程序，而不需要任何先前的编码技能或编程知识。然而，Appixia 与 Wix 的一个不同之处是:Wix 允许任何人从头开始建立一个网站，Appixia 目前不支持没有现有电子商务商店的零售商。

尽管如此，Wix 已经为那些希望快速建立电子商务商店的人提供了广泛的电子商务功能，并自动创建其用户创建的网站的移动优化版本。因此，收购 Appixia 使 Wix 能够扩展其当前的电子商务和移动功能，使用户能够创建在线商店，然后快速添加他们的商店的移动优化版本以及原生应用程序。

从更大的意义上来说，这次收购是一次战略行动，因为两家公司都提供了无代码解决方案，使复杂和昂贵的开发过程变得简单和容易，特别是对小企业和企业家来说。Appixia 还为 Wix 目前的移动自适应解决方案提供了补充，该解决方案自 10 月份推出以来一直在快速增长。

作为今天公告的一部分，该公司表示，用户现在每天创建超过 3 万个移动网站，截至 2013 年底，在其平台上创建了 160 万个移动网站。通过整合这家初创公司的技术，Wix 将能够添加其平台所缺乏的扩展移动功能和特性，如支持推送通知、条形码扫描、基于位置的服务(GPS)、信用卡扫描、脸书和 Twitter 集成等。

至于交易条款，尽管该公司拒绝分享价格，Wix 通信总监埃里克·梅森表示，这主要是一笔人才和技术交易，该公司预计此次收购不会对其非公认会计准则的经营业绩产生影响。然而，该公司确实希望将 Appixia 的技术“完全整合”到其平台中，该初创公司的联合创始人之一塔尔·科尔将领导整合工作。

梅森还表示，“上市时间”也是收购的一个重要因素，因为这项技术已经建成，收购 Appixia 使 Wix 能够在比内部开发这项技术“短得多的时间”内扩展移动支持。对于 Appixia，当它开始与 Wix 讨论时，显然正处于第一轮融资阶段，联合创始人兼首席执行官伊盖尔·伯杰(Yigael Berger)表示，团队认为现有的协同效应和超过 4000 万英镑的机会太诱人了，不能错过。
