# 雅虎通过垂直网站吸引主流受众，推出雅虎美食和雅虎科技 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/07/yahoo-goes-vertical-for-the-mainstream-launches-yahoo-food-and-yahoo-tech-hires-rafe-needleman-as-editorial-director/>

为日益拥挤的科技新闻网站世界添加另一个新闻门户，如果你仍然感到饥饿，添加另一个食物门户。今天，雅虎正式推出了雅虎科技(Yahoo Tech)网站(T1)，这是一个新网站，将与雅虎新闻(Yahoo News)和雅虎商业(Yahoo Business)并列，为主流受众提供科技领域的新闻。它还推出了一个新的数字杂志，雅虎食品。

作为雅虎技术的一部分，它宣布了一些新的雇员，包括 Rafe Needleman 作为该网站的编辑主任。资深记者 Needleman 是从 Evernote 挖来的，在 Evernote，他最初是这家初创公司的[平台倡导者](https://web.archive.org/web/20221206211905/https://beta.techcrunch.com/2012/07/31/evernote-hires-cnet-rafe-needleman/)，后来晋升为开发者关系总监。

但与其他消费出版物不同的是，这里有一个条件:没有展示广告。这是雅虎新努力的一部分，旨在移动指针，为其资产带来新的受众，看起来其中一部分将包括提供更独特体验的内容的方式，这可能比竞争对手更受欢迎，竞争对手的网站上充斥着横幅、摩天大楼和间隙广告。

玛丽莎·梅耶尔(Marissa Mayer)在拉斯维加斯的 CES 大会上发表了主题演讲，并带来了明星雇员大卫·波格(David Pogue)，他离开了《纽约时报》的技术专栏作家的角色，加入了雅虎，在台上谈论雅虎技术的所有事情。

“在我看来，重点是高端确实得到了很好的照顾……但谁在照顾中间的 85%？没人！”(是的，他尖叫了这最后一部分。)

我目前正在雅虎的主题演讲上听他们介绍新的垂直市场，但是当我等待会议开始时，我遇到了 Needleman。这是他告诉我的关于新网站的情况:

“购买世界上大多数科技产品的人都非常聪明，但他们并没有融入这个世界，他们不关心员工的变化，而且在很大程度上，他们对硅谷不感兴趣，”他说。“他们想知道买什么电视最好。”

至于雅虎食品，这似乎是该公司一系列垂直努力的开始。“数字杂志是我们长期愿景的核心部分，”梅耶尔在她的主题演讲中说。该网站将整合雅虎其他网站的元素，如 Flickr，从外观上看，它也将整合其他网站的社交元素。

至于广告将如何在这两者中发挥作用？还不完全清楚。我浏览了雅虎科技，看不到任何我称之为广告的东西——除非一些应用文章实际上是赞助内容。该网站的架构遵循 Flickr 的架构——也就是说，当你点击一个特定的图片存根时，它就会展开。

我的猜测是，这种马赛克将成为广告生存的基础，而不是文章本身。这将让雅虎可以选择放入赞助内容，也可以放入小型自动播放视频，当你浏览所有的故事时，这些视频会出现在那里，但当你选择专注于一篇特定的文章时，这些视频会出现在一个干净的白色页面上。
