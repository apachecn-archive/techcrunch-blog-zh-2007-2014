# 扰乱北京:我们将带来陈士骏、彼得·维斯特巴卡、菲尔·利宾等等

正如我们上周宣布的那样，我们一直在忙着为今年 10 月的 [Disrupt Beijing](https://web.archive.org/web/20230205033830/http://disrupt.beta.techcrunch.com/BJ2011/) 大会寻找一些最令人兴奋的中国名字，包括腾讯创始人兼首席执行官[马化腾](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/person/pony-ma)和中国企业家兼天使投资人[雷军](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/person/lei-jun)。但如果没有一点硅谷的魔力，它就不会是一场 TechCrunch 盛会。

在选择带去中国的人时，我们希望有些人是新来的，有些人有在中国做生意的长期经验；既有成功的人，也有快速成长的人。我们还希望有一些人能够讲述让硅谷如此独特的文化和奇思妙想。

我们将从《愤怒的小鸟》的制作人 [Rovio](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/company/rovio-mobile) 的雄鹰[彼得·维斯特巴卡](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/person/peter-vesterbacka)开始。维斯特巴卡对中国并不陌生，他说 Rovio 正忙于在年底前建立一个价值 1 亿美元的业务。这项业务从游戏到甜点到毛绒玩具，无所不包。我最后一次见到维斯特巴卡是在柏林的 Next11 会议上，他越是告诉我他的中国战略，我就越坚持要他加入 Disrupt。他甚至可能有一些消息要宣布…

[陈士骏](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/person/steve-chen)，YouTube[的联合创始人](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/company/youtube)也将和我们一起登台。陈是这方面的专家:一家发展如此迅速的公司可能会让你破产。YouTube 是 Web 2.0 世界有史以来最大的现象之一。从美国总统竞选到贾斯汀比伯的崛起，它在所有事情上都发挥了关键作用。这是第一个退出金额超过 10 亿美元的 Web 2.0。

硅谷并不全是浪费时间的有趣方式。 [Evernote](https://web.archive.org/web/20230205033830/http://www.crunchbase.com/company/evernote) 的 Phil Libin 称他的公司为“反 Zynga ”,因为它试图让你在一天中的小洞里变得更聪明。Evernote 在用户中拥有狂热的追随者，是硅谷目前最热门的公司之一，最近刚刚进行了一轮融资。正如利宾在[的这段视频](https://web.archive.org/web/20230205033830/https://techcrunch.com/2011/07/13/evernote-takes-50-million-to-become-the-anti-zynga/)中解释的那样，Evernote 在下载量上与 Foursquare 不相上下，而且与 Foursquare 不同，它没有盈利问题。
