# Fortumo 打开(愤怒的小鸟)存钱罐，让 Android 开发者通过短信提供运营商账单

![](img/db56c934ce9a43b4778cd2f4bad73515.png "Fortumo")你还记得 [Bad Piggy Bank](https://web.archive.org/web/20230203232015/http://eu.beta.techcrunch.com/2010/12/10/angry-birds-50-million-downloads-and-creates-simple-in-app-purchases-on-android/) ，这是安卓系统上 [Rovio](https://web.archive.org/web/20230203232015/http://www.crunchbase.com/company/rovio-mobile) 的《愤怒的小鸟》的游戏内支付模式，它允许用户要么删除游戏免费版本上的广告，要么购买即将到来的 Mighty Eagle 更新。是的，它绕过了安卓自己的应用商店支付系统，转而支持运营商付费。

嗯，它实际上是由欧洲移动支付提供商 Fortumo 提供支持的，该公司今天宣布，它将通过向其 Android 库添加离线支付(通过短信)的支持来进一步打开存钱罐。

这与《愤怒的小鸟》使用的[应用内支付库](https://web.archive.org/web/20230203232015/http://fortumo.com/in-app-payments)相同，任何 Android 开发者现在都可以免费访问。

它是这样工作的:要求付款的应用程序向一个短代码发送一条短信。然后，当应用程序返回确认文本消息时，支付被验证，而用户只需单击一下就可以开始支付，而无需离开应用程序。
 [更多的证据](https://web.archive.org/web/20230203232015/http://eu.beta.techcrunch.com/2010/11/26/guest-post-the-alternative-payment-scene-in-europe-%E2%80%93-bridging-the-gap/)或许，欧洲引领着移动支付的潮流？

Fortumo 的计费覆盖了 50 个国家和大约 200 家移动运营商。值得注意的是，该公司表示，没有启动费，月租费或最小量承诺，“使其适合小型和大型开发商”。

Fortumo 成立于 2007 年，在欧洲(爱沙尼亚塔尔图)和美国(加利福尼亚州桑尼维尔)设有办事处，迄今为止已拥有超过 49，000 家注册服务提供商，他们已经创建了超过 156，000 项移动支付服务。
