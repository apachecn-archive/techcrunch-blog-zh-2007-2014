马克·平卡斯曾经是肖恩·帕克的老板

Napster联合创始人、前脸书总统、现任Airtime首席执行官肖恩·帕克今天在 Web 2.0峰会的舞台上代替Zynga首席执行官马克·平卡斯，当时平卡斯因为Zynga的静默期而不能发言(旁注:预计下个月内IPO，哟)。

你知道什么很酷吗？奇怪的是，15岁的帕克曾经在马克·平卡斯在华盛顿特区的初创公司——第一家消费者推送信息服务公司——白吃——实习过，你知道，在他出名之前。
