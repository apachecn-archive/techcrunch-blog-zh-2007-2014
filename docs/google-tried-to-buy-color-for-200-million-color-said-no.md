# 谷歌试图以 2 亿美元收购 Color。色说没有。

如果你对这个标题很熟悉，你可能还记得今年早些时候我们关于谷歌出价 1 亿多美元收购 Path 的故事。Path 拒绝了这笔交易。

几乎与此同时，多个消息来源证实，谷歌也在试图收购由[比尔·阮](https://web.archive.org/web/20230209185816/http://www.crunchbase.com/person/bill-nguyen)创立的移动社交网络 [Color](https://web.archive.org/web/20230209185816/http://www.color.com/) 。这远在 Color 推出之前，谷歌就在关注公司的潜力和团队。根据我们的消息来源，谷歌出价 2 亿美元收购该公司。

我们的消息来源称，Color 拒绝了这笔交易。然后，他们从贝恩资本、红杉资本和硅谷银行筹集了 4100 万美元的债务和股权资本。这一轮对该公司的估值为 1.67 亿美元。

Color 应该接受谷歌的交易吗？绝对是的。该公司自成立以来一直步履蹒跚，未能实现自己的宣传，并失去了创始人彼得·范和首席产品官 DJ·帕蒂尔。

也就是说，有后知之明总是好的。在谷歌试图收购 Color 的时候，该公司有一个杀手级团队，一个强大而大胆的愿景，以及愿意向这家初创公司投资的顶级投资者。[企业家不会在公司成立之前就出售公司。](https://web.archive.org/web/20230209185816/https://techcrunch.com/2010/10/31/are-you-a-pirate/)

事后看来，虽然…哇。
