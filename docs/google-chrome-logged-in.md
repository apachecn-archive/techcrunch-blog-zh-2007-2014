# 谷歌开始重视 Chrome 档案，即登录体验

谷歌 Chrome 浏览器的迅速崛起有一个副作用，我认为很多人都忽略了。面对脸书的巨大挑战，谷歌想要继续成为网络的中心，这给了它最后的武器。虽然谷歌永远不会承认这一点，但他们也不必承认。我们已经看到了。

Chrome 的最新版本(金丝雀频道的版本 14)有一个新的、实验性的配置文件切换和创建图标。这种将谷歌账户与 Chrome 绑定的功能已经存在了一段时间，它是 Chrome OS 的一个重要部分，但鉴于该功能的新图形布局，很明显谷歌打算让它更加有用和引人注目。

在 Chrome 外壳的右上角，你会看到一个烧杯图标，谷歌喜欢用它来做实验性的功能。点击这个按钮，你可以看到你的 Chrome 档案，自定义它，切换到另一个，或者删除它。定制仍然是初级的——你可以设置你的同步选项(还是旧的)或者你可以创建一个新的配置文件。当你创建一个新的配置文件时，你会被带到一个页面，允许你设置一个配置文件名称并选择一个配置文件图标——这类似于 Chrome OS。造物主上的文字写着:

> Chrome 配置文件包含一组书签、浏览历史、保存的密码和主题。如果需要管理单独的设置，可以创建多个配置文件。例如，您可以使用一个配置文件，而您的朋友可以使用另一个。”

这一切听起来简单而伟大。但同样，不要忽视这里的潜在力量。谷歌已经授权 Chrome 能够在所有谷歌用户浏览网页时为他们提供完全登录的体验。现在，他们正在做基本的工作，但是将来，他们可以很容易地打开开关，实现更强大的功能。也许 Chrome 将获得全面的 Google+集成(跨谷歌网站的工具栏？求你了。全网一个怎么样！)?或者谷歌单点登录怎么样？或者深度整合的广告网络怎么样？

同样，谷歌会否认所有这些意图，因为我让它们听起来有点邪恶。但事实是，它们只是明智之举(尽管如果 Chrome 变得足够大，反垄断监管机构无疑不会同意)。谷歌如果不追求这些，那就太傻了。

脸书是谷歌在网络上最大的挑战，Chrome 给了他们一些脸书没有的东西:浏览器。现在，脸书无疑正在开发一个(除了常识之外，我在这里没有真正的信息——如果他们*没有*开发一个，那将是疯狂的),他们已经在[这个领域做了一些合作](https://web.archive.org/web/20230204214404/https://techcrunch.com/2011/06/14/facebook-rockmelt/)。通过 Connect，脸书一直致力于通过网络提供登录脸书的体验。但是连接是很棘手的。网站所有者需要实现它。如果用户可以在自己的浏览器中简单地做这些事情会怎么样？

这是谷歌在 Chrome 上的黄金机会。一步一步来。
