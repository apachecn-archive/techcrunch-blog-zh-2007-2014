# LinkedIn 收购了实时托管搜索初创公司 IndexTank

职业社交网络 LinkedIn 正掀起一场收购狂潮。在两周内的第二次收购中，LinkedIn】收购了托管的搜索初创公司 [IndexTank。这笔交易的财务条款没有披露。](https://web.archive.org/web/20221210052927/http://indextank.com/)

IndexTank 是一种实时的托管搜索引擎服务，允许开发人员快速构建基于搜索的应用程序，而不必担心托管自己的搜索软件。IndexTank 的功能包括:实时提要索引，搜索数据的即时可用性，地理搜索，自动分面，范围搜索和关于搜索索引数据的即时见解。

这家初创公司已经筹集了 160 万美元的资金，将 Reddit 视为客户。细节尚不清楚，但似乎 IndexTank 的搜索技术将被整合到 LinkedIn 的搜索中。IndexTank 表示，LinkedIn 已经同意在未来六个月内保持 IndexTank 服务的运行，并将开放该技术的关键组件。
