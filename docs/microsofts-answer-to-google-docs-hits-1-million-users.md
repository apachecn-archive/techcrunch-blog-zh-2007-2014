# 微软对谷歌文档的回答点击了 100 万用户

[![](img/50b15e11c81c0bbf23ff48b9253d4db3.png)](https://web.archive.org/web/20221207191753/http://workspace.officelive.com/)

微软[宣布](https://web.archive.org/web/20221207191753/http://www.techmeme.com/080903/p89#a080903p89)其 Office [Live Workspace Beta](https://web.archive.org/web/20221207191753/http://workspace.officelive.com/) 已经拥有了第一百万个用户。这项服务既是 Office 的延伸，也是 Google Docs 和 Zoho 等初创公司的直接竞争对手，于去年冬天宣布，并于去年 3 月开放公测。

当我们[最初报道](https://web.archive.org/web/20221207191753/http://www.beta.techcrunch.com/2007/12/10/office-live-workspace-beta-finally-goes-live-still-needs-work/) Live Workspace 的时候，我们批评了它乏善可陈的基于网络的文本编辑器。微软并没有把精力集中在改进竞争对手谷歌文档(Google Docs)提供的 webtop 文档编辑上，而是试图用 Live Workspace 来支持其 Office 桌面套件。用户被鼓励通过 Word 等几乎无处不在的办公产品编辑他们的文档，Live Workspace 将更改同步到在线档案和其他贡献者。

该网站确实提供了一个在线编辑器，但是微软并没有特意指出来——在最初的声明中根本不清楚是否存在这样一个编辑器。这一战略对微软来说是有意义的:由于在桌面出版软件上占据压倒性垄断地位，而且几乎没有基于网络的协作解决方案，它宁愿将这作为 Office 的额外功能，而不是谷歌文档的直接竞争对手——人们认为他们需要桌面软件编辑文档的时间越长越好。
