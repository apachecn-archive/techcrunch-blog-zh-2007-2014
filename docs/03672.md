# 不是闹着玩。EA 以 3 亿美元收购 Playfish，并获得 1 亿美元的收益。TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/11/09/not-playing-around-electronic-arts-buys-playfish-for-275-million/amp/>

经过漫长的谈判，艺电以 2.75 亿美元现金完成了对社交游戏初创公司 Playfish 的预期收购。另外 2500 万美元的股票将用于留住创业公司的顶级人才，如果业务达到某些里程碑，另外 1 亿美元的收益也是交易的一部分。所以说到底，这笔交易的总价值可能高达 4 亿美元。不过，收入往往会不足(见 Skype)。

Playfish 总部位于伦敦，已经从 Accel Partners 和 Index ventures 筹集了 2100 万美元。Accel 的投资来自其欧洲基金。Playfish 的预计年收入为[7500 万美元](https://web.archive.org/web/20230130084615/http://techcrunch.com/2009/10/26/social-games-how-the-big-three-make-millions/)。

去年，在英国汉普郡的创始人论坛上，首席执行官克里斯蒂安·塞格斯特勒(Kristian Segerstrale)展示了一张恐龙幻灯片，表达了他“杀死 EA”的愿望。现在他加入了他们。有趣的是这是怎么回事。

Playfish 在脸书、MySpace、Hi5 和其他社交网络上运营社交游戏，安装次数超过 1.5 亿次，并声称每月有 6000 万活跃用户。它的顶级游戏包括*宠物协会、餐厅城市、乡村故事*和*谁的大脑最大？*

社交游戏越来越受欢迎，电子艺界需要购买顶级玩家之一:Zynga、Playfish 或 Playdom。问题是，这三家公司都是通过交易游戏中的虚拟货币来换取广告优惠来赚钱的。这些报价中有许多都是彻头彻尾的骗局，这可能解释了为什么 Playfish 获得的前期现金少于它期望的 3.5 亿至 5 亿美元，而最后 1 亿美元是以盈利的形式获得的。Playfish 或许也不像它的竞争对手那样容易受到这些欺诈性报价的影响。

Accel 的合伙人凯文·科莫利(Kevin Comolli)告诉我们，Playfish 的“绝大部分”收入并不来自潜在客户或其他类型的虚拟货币广告。他对整个斯卡维尔事件的看法是“曝光最终是有益的。它需要是一个持久的业务。清理这一点很重要。”
