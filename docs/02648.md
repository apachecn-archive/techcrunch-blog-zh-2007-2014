# 日本厕所分析粪便，通过个性化网址将结果发送到手机

就像生活在日本的其他外国人一样，我开始喜欢上了日本厕所。它们有多个按钮和功能可以玩，在寒冷的日子里让你的屁股保持温暖，而且通常都是过度设计的。虽然这对我来说很酷，很好，但日本的 Inax 现在用一个非常有趣的在线功能做得太过了。

美国第二大“卫生设备”制造商开发了一种厕所终端，它通过个性化的网址告诉你你的大便结构(现在考虑人类的另一个大问题得到解决)。 [FKF-20M](https://web.archive.org/web/20230122053856/http://www.inax.co.jp/products/toiletroom/toiletroom_goods/remoconstands/ "FKF-20M") [JP】(如下图)是一个遥控支架，可与许多 Inax 高科技马桶兼容。

<amp-img class="aligncenter size-full wp-image-81915 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="inax_remote" src="https://web.archive.org/web/20230122053856im_/https://techcrunch.com/wp-content/uploads/2009/04/inax_remote.jpg" alt="inax_remote" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>

它是这样工作的:

工作完成后，马桶会分析你的排泄物(细菌数量、身体脂肪等)。)，会创建一个包含分析结果信息的 URL，并通过红外连接自动发送到您的手机上(在日本的大多数现代厕所中，红外面板充当非接触式冲水传感器)。然后，用户可以在他们的移动浏览器上打开相应的页面，检查他们的健康状况。太神奇了。

FKF-20M 当然是日本独有的，售价 560 美元。它将在本月内上市。我会亲自测试的，当我找到一个，承诺。
