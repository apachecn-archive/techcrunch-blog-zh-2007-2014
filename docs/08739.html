Ustream Shutters Live Mobilizer与Mobile Roadie合作，允许广播公司创建移动应用程序

去年，流行的直播视频平台Ustream 推出了一个名为Ustream Live Mobilizer的移动应用开发平台，为品牌、名人和乐队提供一个可定制的iPhone应用，该应用具有Ustream的直播功能。但今天，Ustream关闭了Live Mobilizer，并宣布与 Mobile Roadie、合作，允许任何人开发和创建流畅的富媒体 iPhone 和 Android 应用。

Mobile Roadie将与Ustream广播公司合作创建移动应用程序，展示他们的Ustream直播频道以及许多其他免费功能。例如，Mobile Roadie的Ustream应用程序将在广播公司上线时自动发送推送消息，允许访问者“签到”流媒体并分享到脸书和Twitter，收集广播公司直播流媒体的实时分析和位置数据，并允许用户在观看直播流媒体时相互聊天。

广播公司也将能够利用该应用赚钱，并可以向Ustream内容的用户收取一次性或每月费用。Mobile Roadie表示，使用Live Mobilizer的现有Ustream客户端将可以选择迁移到新平台。

将移动应用程序开发外包给Mobile Roadie对Ustream来说很有意义，他们可以专注于他们最擅长的事情——直播。Mobile Roadie已经成为许多知名品牌和名人使用其平台的事实上的移动应用程序开发平台，包括泰勒斯威夫特(参见本文中Ustream集成的例子)、麦当娜、Live Nation、李维斯、暮光之城和维拉王。2009年，Mobile Roadie还为 LeWeb、开发了官方iPhone应用程序，与兰登书屋达成协议，为作者开发iPhone应用程序，与索尼音乐为音乐品牌的艺术家网络开发移动应用程序。
