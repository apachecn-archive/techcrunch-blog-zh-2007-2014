推特时代的离婚

编者按:特约撰稿人迈尔斯·贝克特是媒体公司f EQAL 的首席执行官和联合创始人，该公司围绕名人和品牌建立影响者网络。他是原创网络视频系列《孤独女孩15》的联合创作者，之前是一名医学博士。

离婚。它发生在我们最好的人身上。尽管在情感上令人心痛，但现在更糟糕的是，我们在脸书、推特之类的公共舞台上度过自己的人生。如果最近阿什顿和黛米的公开分手有任何迹象的话，事情只会变得更糟。作为一名前医生、现任互联网企业家和对人类状况永远充满好奇的观察者，我着迷于互联网如何广泛地塑造我们的文化，以及这对我们人际关系的日常影响。

过去很简单，我们的公共生活和私人生活被物理界限明显分开。在iPhones、Twitter、Instagram和脸书出现之前的日子里，让公众关注我们的关系状态真的很难，我们大多数人不需要考虑我们的“公众形象”，也没有多少人在乎。与脸书和推特最相似的历史事件是全国各地报纸上的社交版，但这些只有精英阶层才能看到，他们更习惯于维护精心构建的公众形象。现在，我们都有能力传播我们的生活，但我们中的许多人没有配备工具来处理它。

毫不奇怪，有一种对名人关系的全国性迷恋。就像他们生活的其他方面一样，他们的婚姻和离婚是我们的写照，现在我们比以往任何时候都更能从他们的成功和失败中吸取教训。就像一个巨大的绊倒和坠落伴随着巨大的轰隆声！，当一个名人在Twitter上犯了一个错误，他们比我们这些“普通人”更容易崩溃阿什顿和黛米为我们所有人提供了一个教训，既有他们作为媒体早期采用者的成功，也有他们最近的失败。

我学到了很多关于如何通过正面和负面的公关来驾驭一个名人品牌的经验。我也花了一些时间与日常生活中不出名的人谈论他们在离婚期间如何管理自己的个人品牌。过去还有另一层隐私，但现在关系可以很快变得非常公开。我发现，从我的隔壁邻居到一位资深的娱乐公关人员，每个人都同意，我们现在必须控制自己公开隐私的冲动，除非我们准备好承担后果。

考虑到这一点，以下是我在公众面前处理你们的关系、分居或离婚时的“该做”和“不该做”的事情:

做

  1. 一旦你们的关系正式化，所有的细枝末节都处理好了，就要公开你们的关系状况(不管你喜不喜欢，它都会公之于众)
  2. 从你的脸书订阅源中“隐藏”你的前任，将他们移到“有限个人资料”中(你需要一点隔离和隐私！)
  3. 在公共场合展示你和前任之间的礼貌(在照片上标记他们或者在适当的时候回复他们是可以的)
  4. 离婚后继续发推特和脸书(自信地继续“一切照旧”很重要)
  5. 试着继续前进，享受乐趣，并与你的朋友和追随者分享(每个人都喜欢赢家)

不要

  1. 在推特、脸书、布斯、巴奇或克上给任何人发自己衣着暴露的照片(它们将被用来对付你……)
  2. 使用一个包含你配偶名字的推特账号(@mrskutcher陷入了两难境地)
  3. 在Craigslist上卖你的戒指(我们能说“俗气”吗？)
  4. 在脸书和推特上发布关于前任的令人讨厌的更新(没有人喜欢输不起的人)
  5. 在向你的另一半吐露心声之前，先向你的推特和脸书观众吐露心声(没有人喜欢惊喜)
  6. 取消好友或者取消关注你的前任(这是公开的，而且很讨厌，问问金或者，嗯，克里斯就知道了)

底线:在你发微博之前停下来想一想。
