#OccupySocialMedia: GO推出匿名照片和视频分享移动应用TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/11/15/ocuppysocialmedia-go-launches-a-mobile-app-for-anonymous-photo-video-sharing/

GO 是iPhone的新移动应用程序，允许用户广播并与他人分享照片、视频和评论，并将它们发布到直播门户网站。该应用程序提供实时访问地理位置，以流和地图视图显示的标记媒体。但是，这个世界需要另一种移动/社交/照片分享体验吗？嗯，也许是这样。

你看， GO 与众不同:它支持完全匿名。需要澄清的是，它不仅仅提供了使用一些聪明的Web句柄来代替认证用户帐户的选项——它实际上提供了“匿名”发布的选项。#OccupySocialMedia？哦是的。

就应用程序的设计而言，你要么会喜欢它，要么会讨厌它。这款应用非常漂亮，甚至可能有点太过漂亮，其过于风格化的外观和感觉有时会妨碍本可以更简单的用户体验。我碰巧认为围棋看起来很热门，但你可能会有不同的想法。各有所好。

至于功能本身，没什么惊天动地的:贴吧、标签、分享。然而，当你点击大的粉红色分享按钮时， GO 为你提供了“抓拍”(发布照片)、“拍摄”(视频)或“说话”(音频)的选项。这是媒体输入的三个选项，你最喜欢的拍照/应用滤镜照片分享应用可能没有。

不过，真正有趣的是GO的匿名选项，这是新移动应用体验的一个关键部分。当今天的大多数应用都竭尽全力为你提供登录选项，让你通过脸书或Twitter加快身份验证过程时，通过GO，自我识别是一个选项，而不是一个要求。虽然这可能会导致用户可能不是“马克·扎克伯格”(是的，他加入了)，但这并不重要。GO不是通过喜欢和分享来提升你的自我，“与朋友联系”，或与家人分享照片——它是关于即时移动广播。这是关于记录这个世界，而不必透露你这样做了。

尽管它的推出相对温和(GO于2011年11月11日上线，没有太大的声势)，但已经有一些有趣的视频从占领华尔街运动中浮出水面，GO似乎是为他们量身定制的。当然，GO并不是满足这一人群需求的第一工具。像Twitter风格的Vibe 这样的应用程序在过去也很好地服务了占领运动的抗议者。但是Vibe很丑，主要用于文字。GO做的更多。

尽管GO有着与众不同的特点，但它仍将遭遇任何照片/视频分享领域新来者最初都会遭遇的同样的劣势:临界质量。然而，假设对匿名记录世界的工具的需求不会随着今天的占领华尔街关闭而消亡，那么GO可能还有未来。

GO 是来自 Hollr (不要和 Holler )的第一款产品，由Michael Bachman和Justin Dionisio创立。这两人之前是位于长滩的精品互动机构KURO的董事。Hollr从Imprint Venture Lab获得了一些种子阶段投资，但不愿透露金额。
