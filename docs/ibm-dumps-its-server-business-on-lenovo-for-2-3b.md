# IBM 以 23 亿美元将其服务器业务出售给联想

IBM 最终成功将其低利润的服务器业务出售给了全球最大的个人电脑制造商联想。过去几年，两家公司一直在就这笔交易进行谈判，去年，当联想不同意支付 IBM 想要的 60 亿美元时，一笔潜在的交易流产了。

现在，联想已经通知香港证券交易所[它将向 IBM 支付 20.7 亿美元现金购买其服务器部门](https://web.archive.org/web/20230326053730/http://in.reuters.com/article/2014/01/23/us-ibm-server-lenovo-idINBREA0M01U20140123)。

这是 IBM 第二次通过将低端硬件业务出售给联想来摆脱这一业务。2005 年，IBM 以 17.5 亿美元的价格将其 ThinkPad PC 业务出售给联想。

IBM 来自服务器业务的收入在过去七个季度中一直在下降，迫使该公司迅速采取行动，摆脱这一部门。

在全球范围内，包括脸书和谷歌在内的一些最大的企业越来越多地转向精简版的硬件，而不是昂贵的硬件。这些公司转而要求像[广达](https://web.archive.org/web/20230326053730/http://www.quantatw.com/Quanta/english/)这样的制造商为他们制造更便宜、更合适的定制服务器。由于几乎没有品牌关联，来自中国大陆和台湾制造商的服务器被称为“其他”服务器。

对于联想来说，由于智能手机在不断增长的亚洲市场的普及率越来越高，联想已经在努力应对日益恶化的个人电脑销售情况，而 IBM 的服务器业务将提供一个建立另一个收入来源的机会。

正如路透社的这篇文章所说，根据技术研究公司 Gartner 的数据，2013 年第三季度，IBM 的服务器业务是世界第二大的，在 123 亿美元的市场中占有 22.9%的份额
