Classmates.com家长联合在线收购专注于学校的交友网站SchoolFeed

原文：https://web.archive.org/web/https://techcrunch.com/2012/06/11/classmates-com-parent-united-online-acquires-school-focused-friend-finder-schoolfeed/

Classmates.com家长联合在线收购以学校为重点的交友网站SchoolFeed

来自的RockYou Lance Tokuda 的前联合创始人兼首席执行官SchoolFeed 被收购。一个并不令人意外的转折是，这家利用脸书为用户提供更便捷的方式与老同学联系的公司，被总部位于西雅图的Memory Lane收购了，后者更为人所知的名字是“Classmates.com背后的人”。

Memory Lane是联合在线公司(United Online)的子公司，该公司经营许多社交网络服务，包括Memory Lane、Classmates.com和欧洲StayFriends，但也经营其他垂直领域的业务，包括FTD花卉服务公司和Interflora，以及互联网接入提供商NetZero、Juno和NetZero Wireless。

今年3月，当TechCrunch报道了Schoolfeed的新功能时，该公司称其用户已达1000万。schoolfeed最大的吸引力在于它如何改进脸书的人物搜索功能，使其比脸书本身更适合与高中或大学的老朋友联系。例如，schoolFeed可以让你按高中年级进行搜索，而脸书却不能。这对老一辈的人尤其有用，他们的友谊和社交网络从一开始就没有在脸书上记录下来。

如今，schoolFeed表示，在过去的三个月里，它平均每天以大约10万个新注册用户的速度增长，现在已经拥有1900万注册用户。Tokuda和其他schoolFeed团队成员将在交易结束后加入Memory Lane。

根据联合在线首席执行官马克·戈德斯通的声明，收购是希望它“将为同学们创造一个在脸书占据领先地位的机会，他说。”它还应该为schoolFeed成员和Classmates.com成员提供一个机会，与他们高中的更多人重新联系和互动，”他补充说。

同学，于2004年被联合在线收购，早在2007年就放弃了IPO计划。它未来成功的最佳情况是在脸书的学校朋友利基中寻找机会(是的，通过利用脸书)，而不是与社交网络巨头正面交锋。目前，同学录拥有5500万注册用户，25000所高中，并且已经数字化了超过130000本高中年鉴。
