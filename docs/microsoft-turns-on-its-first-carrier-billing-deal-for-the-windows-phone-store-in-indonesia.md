# 微软与印度尼西亚 Bango 签订了第一份 Windows Phone 商店运营商付费协议 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/06/27/microsoft-turns-on-its-first-carrier-billing-deal-for-the-windows-phone-store-in-indonesia/>

本周，微软在其 [Build conference](https://web.archive.org/web/20221204100702/http://www.buildwindows.com/) 上发布了一系列[产品公告](https://web.archive.org/web/20221204100702/https://beta.techcrunch.com/tag/microsoft/)，同时也在其他领域取得了一些进展。今天，总部位于英国的移动支付和分析公司 Bango 宣布首次在 Windows Phone 商店上实现其与微软的服务，该公司与脸书、亚马逊、黑莓和谷歌等公司合作，以便应用程序购买可以直接计入用户的手机账单。具体来说，微软正在印度尼西亚开启这项服务，该国是东南亚最大的手机市场，拥有超过 5000 万用户。计划是在不久的将来将它扩展到更多的市场。

Bango 和微软达成框架协议已经有一段时间了，但这实际上是基于该协议的第一次商业部署。Bango 首席执行官 Ray Anderson 在一份声明中指出:“今天 Windows Phone Store 的发布具有重要的战略意义，因为它标志着我们与微软的首次整合。”

这也说明了第一笔交易是在印度尼西亚达成的。该国不仅拥有该地区最多的移动用户，也是移动公司瞄准新兴市场提供新服务的一个例子。

随着智能手机在更成熟市场的采用放缓，那些仍有快速增长、更有机会赢得新用户的国家成为焦点。这也是苹果公司最终在俄罗斯开设网上商店的原因之一。就移动用户而言，印度是迅速崛起的第三大经济体，仅次于中国和美国，这也凸显了这一趋势。对于微软来说，进军仍有消费者的市场是一个明智的战略，其亲密合作伙伴诺基亚也在追求这一战略。在采用智能手机平台方面，微软远远落后于安卓(Android)和苹果(Apple)。

就 Bango 而言，它一直在努力瞄准新兴市场的运营商计费交易——它在这一领域的其他努力包括为 Firefox Mobile 的应用商店提供[电力支付的交易(也瞄准新兴市场)；一项](https://web.archive.org/web/20221204100702/https://beta.techcrunch.com/2013/02/27/bango-to-power-payments-in-mozillas-firefox-os-app-store-carrier-billing-by-default/)[与 Telefonica](https://web.archive.org/web/20221204100702/https://beta.techcrunch.com/2013/01/16/bango-adds-telefonica-to-its-list-of-mobile-carrier-billing-partners-for-facebook-and-beyond/) 的交易(覆盖拉丁美洲)；以及专门用于新兴市场扩张的[1020 万美元融资](https://web.archive.org/web/20221204100702/https://beta.techcrunch.com/2013/02/08/bango-raises-10-2m-to-take-its-mobile-payments-service-into-brazil-india-and-other-asian-markets/)。

这不仅仅是因为上述智能手机用户数量仍在快速增长，还因为许多这些市场的信用卡、借记卡和其他支付卡的渗透率也很低。因此，与用户已经拥有 iTunes、亚马逊和其他基于信用卡账户的账户的市场相比，将购买的应用程序直接计入你的预付费手机账户不仅更方便，而且实际上可能会更频繁地被选择。

Bango 说，实现了运营商付费服务的应用商店往往有更高的付费服务转化率。在有大量信用卡的市场，运营商付费服务的转换率比其他服务高 300-400%;在卡渗透率较低的市场，它声称运营商账单转化率高出 1000%。

印度尼西亚的交易也涉及第三方，运营商 indos at——这意味着它只适用于该网络的用户。Indosat 的想法是，这将促使更多的用户使用该网络，并让用户花更多的钱；对微软和 Bango 来说，这一交易将使 Indosat 的竞争对手也愿意谈判。微软也在使用其他技术提供商来与其他运营商整合计费:例如，XL Axiata 正在使用 Amdocs 的移动支付解决方案来支持 Windows Phone store 计费。

微软 Windows Phone Store 总经理 Todd Brix 在一份声明中表示:“移动运营商计费为消费者提供了一种便捷的支付选择，转换率远高于信用卡，极大地惠及了 Windows Phone 8 客户和开发者。“我们很高兴看到 Bango 和 Indosat 这样的公司携手合作，为开发者扩大盈利机会，并为 Windows Phone 用户提供无缝的购买体验。”

鉴于我们讨论的是运营商计费，让这些服务的运营商与他们的后端连接是推进这些服务的核心。Bango 拒绝评论它目前与亚马逊的合作情况——这是几年前宣布的合作关系，但[尚未在任何地方进行商业推广](https://web.archive.org/web/20221204100702/https://beta.techcrunch.com/2013/03/20/amazons-carrier-billing-deal-with-bango-to-kick-in-this-year-changes-for-the-appstore-and-amazon-ahead/)。答案可能在于这些运营商交易进展如何——或者不进展，视情况而定。
