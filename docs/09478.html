Windows 8有一个更友好的蓝屏死亡

虽然人们普遍预计Windows 8会出现黑屏死亡，但昨天发布的开发者版本显示，雷德蒙选择了传统的蓝色。然而，它确实有一个奇特的转折。与其让人们(希望他们记得保存他们的工作)淹没在为什么他们的电脑停止工作的故障中，看起来微软选择了把事情往更有同情心的方向发展。

不像过去的经典、罗嗦的蓝屏，最新的版本对用户做了一个悲伤的表情。除了显示大皱眉头之外，新的BSoD还提供了一些关键的搜索词，以防用户喜欢挖掘刚刚发生的事情。在电脑自动重启之前，用户有几秒钟的时间把它写下来或记在记忆中，然后瞧:又开始工作了。

这是朝着正确方向迈出的一步，因为经典的蓝屏对大多数用户来说几乎是难以理解的。这个最新版本设法使这个过程不那么令人头痛，但我(也许很天真)渴望有一天微软能用简单的英语告诉我为什么我的电脑失败了。
