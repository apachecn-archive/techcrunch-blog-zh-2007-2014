# 还记得上网本吗？是的，联想已经停止在网上销售了

哇哦。上网本。那是来自过去的爆炸。这就像再听一遍魔力红的歌，然后想“等等，这是从他们表面上很好的那段短暂时期开始的，还是从那首关于选美皇后 18 岁女人的单曲之后的那段混乱时期开始的？”说实话，没关系。上网本是电子产品的魔力红:仍被模糊地认可，但其价值正在迅速丧失。

无论如何，联想已经停止在网上销售上网本，理由是 K1 和 A1 平板电脑的销售流量不错。11 寸 X130e 上网本缺货，大概不会再进货了。《个人电脑世界》指出，去年第四季度，上网本占个人电脑销量的 6.8%。

虽然上网本狂热者大声疾呼“带键盘的低成本笔记本电脑”的价值，但事实是，带蓝牙键盘的平板电脑将比任何硬塞到处理器上的 Windows 版本提供更专业、更流畅的体验，处理器的速度和处理能力尤其强劲。

我怀疑即使是 Chrome OS 也救不了这个[沉没空间](https://web.archive.org/web/20221206062542/https://beta.techcrunch.com/2010/09/17/netbook-sales-are-crashing-quick-blame-the-ipad-not-the-lousy-netbooks/)，但是祝福谷歌的尝试。
