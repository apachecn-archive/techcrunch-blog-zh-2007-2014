# 在当地与 NYT 的争吵中，华尔街日报拿出了一些四方徽章

![](img/83a1971b98592de69934ac3cc54a3144.png)

《华尔街日报》今天推出了《大纽约》[的地铁版，这是鲁珀特·默多克追求《纽约时报》的持续努力。现在的头条新闻是“](https://web.archive.org/web/20221005213033/http://online.wsj.com/public/page/new-york-main.html?refresh=on)[老鼠在上东区闹事”](https://web.archive.org/web/20221005213033/http://online.wsj.com/article/SB10001424052748704448304575196110880930850.html?mod=WSJ_NY_MIDDLELEADNewsCollection)(说真的)。《纽约时报》今天向记者发出了一份员工备忘录，嘲笑《华尔街日报》的努力，并提供“给我们的杂志同事有益的提示”，如“道奇队现在在洛杉机比赛，Soho 是休斯顿南部的首字母缩写，时装周已经转移到林肯中心，Idlewild 现在是 JFK，Cats 不再在百老汇比赛。”

《纽约时报》可能有 160 年报道这座城市的历史，但它没有《华尔街日报》所做的一件事。没错，[四方徽章](https://web.archive.org/web/20221005213033/http://online.wsj.com/public/page/foursquare.html)。作为其本地发布的一部分，《华尔街日报》为铁杆纽约人和《华尔街日报》读者创造了三种新徽章:银行家徽章(用于三次进入金融区)、城市冒险家徽章(用于所有五个区)和午餐盒徽章(用于两次进入《华尔街日报》午餐盒专栏评论的餐厅)。希望，午餐盒评论家会远离上东区。《华尔街日报》还在纽约市的一些地方增加了相关文章的链接。

这些提示和徽章可能不会说服纽约人转向《华尔街日报》进行金融版面以外的本地报道，但它们代表了年轻的 Foursquare 的另一次品牌政变。这笔交易紧随 VH1 和历史频道之后。如果你是一家寻找街头信誉的媒体公司，似乎你需要一个 Foursquare 徽章。

显然，纽约时报不能忽视这一点。它必须用自己的品牌徽章来回应。也许是连续五天入住百吉饼店的百吉饼徽章。
