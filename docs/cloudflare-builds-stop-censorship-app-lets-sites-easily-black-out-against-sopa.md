# CloudFlare 开发“停止审查”应用，让网站轻松“屏蔽”SOPA

无论你对 SOPA 持什么立场，或者对网站是否应该封锁 SOPA 持什么立场，这个问题今天已经达到了沸点，维基百科和 Reddit 等网站承诺在周三封锁，以进一步提高对这项措施的缺陷的认识。

由于 SOPA 和 PIPA 威胁到链接到侵犯版权内容的网站(如 Twitter、维基百科、脸书和互联网上的所有其他网站)的存在，这些法案目前在国会搁置，引发了大规模的在线反弹。

除了 Reddit、维基百科和 [Icanhazcheezburger](https://web.archive.org/web/20221206152547/http://www.icanhascheezburger.com/) ，网络安全初创公司和 TechCrunch Disrupt 亚军 [CloudFlare](https://web.archive.org/web/20221206152547/http://www.cloudflare.com/) 也加入了反 SOPA 的行列，推出了[“cloud flare Stop reviews”](https://web.archive.org/web/20221206152547/https://www.cloudflare.com/apps/stop_censorship)应用程序——基本上[解决了投资者 Fred Wilson 不知道如何封锁的问题](https://web.archive.org/web/20221206152547/http://www.avc.com/a_vc/2012/01/i-want-avc-to-go-dark-on-the-18th.html)。截至太平洋标准时间下午 4 点，任何使用 CloudFlare 的人都可以从 CloudFlare 应用市场一键下载该应用(没有[的人可以在这里](https://web.archive.org/web/20221206152547/http://mikesofaer.github.com/stop_censorship/)从 Github 抓取代码)。

CloudFlare 创始人马修·普林斯(Matthew Prince)告诉我说:“我们认为这是一种有效的方式，可以接触到大量观众，提高人们对 SOPA/PIPA 这样的法律的危险的认识。我们每个月都有超过 4 亿独立访客的页面访问量超过 250 亿次。给你一些感觉，这比 Amazon.com、推特、维基百科、Zynga 和美国在线加起来还要多。4 亿唯一用户约占互联网总人口的 25%。”

普林斯在周末的大部分时间里都在确保一切正常，确保应用的使用不会影响网站搜索排名。

普林斯计划在接下来的一个月左右继续使用该应用程序，直到 SOPA 的威胁过去，他告诉我，“我是一名正在康复的律师，仍然教授互联网和技术法律，所以这是一个我一直密切关注的主题，并了解其真正的风险。我很高兴我们正在帮助那些想参与周三停电的网站变得更容易。”
