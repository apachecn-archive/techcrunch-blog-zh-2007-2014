# Visible Measures 为其数据驱动的视频广告平台筹集了 2150 万美元

视频分析和广告公司 [Visible Measures](https://web.archive.org/web/20221006224803/http://www.visiblemeasures.com/) 刚刚宣布，它已经筹集了 2150 万美元的新资金。

创始人兼首席执行官 Brian Shin 描述该公司是谷歌分析和谷歌视频 AdSense 的结合。当它在 2005 年开始时，Visible Measures 专注于分析，然后该公司在去年推出了[其可视媒体广告平台](https://web.archive.org/web/20221006224803/https://beta.techcrunch.com/2011/04/04/viewable-media/)。可视媒体承诺以一种“为盈利媒体优化”的方式推广广告客户的视频，换句话说，就是在正确的环境中向正确的观众提供视频，以便他们作为常规内容被观看和分享。

申说，这是一个“在视频领域，旧的又是新的”的时代。就像肥皂公司曾经是电视肥皂剧(因此得名)背后的驱动力一样，现在品牌再次进入内容业务，像那些以[这个老香料男](https://web.archive.org/web/20221006224803/http://en.wikipedia.org/wiki/Isaiah_Mustafa)和[依云的滚筒婴儿](https://web.archive.org/web/20221006224803/http://www.youtube.com/user/EvianBabies)为特色的病毒视频，或者，从不那么幽默的角度来看，许多品牌创建的指导视频，以推广他们的产品并与消费者建立联系。当谈到衡量内容的影响时，Shin 强调了可见措施数据的力量和独特性。这些数据用于图表，如[广告时代的热门病毒视频](https://web.archive.org/web/20221006224803/http://www.visiblemeasures.com/adage)，其中包括统计数据，如视频在付费、自有和收入媒体中的“真实覆盖范围”。

“我们比任何人都更多地利用这些数据来帮助合适的人代表品牌观看合适的内容，”Shin 说。

该公司表示，其广告客户包括(品牌方面)P&G、福特、高盛、微软和联合利华，以及(代理方面)Starcom MediaVest Group、Zenith-Optimedia、群邑和宏盟。它还声称去年的收入增长了 500%，今年有望增长 300%。

新的融资主要来自现有投资者的内部融资。之前的支持者 DAG Ventures 领投了这一轮，参与方包括 condéNast owners Advance Publications、General Catalyst Partners、Mohr Davidow Ventures、Northgate Capital 和 new investor Common Fund。Visible Measures 现在已经筹集了超过 6500 万美元。
