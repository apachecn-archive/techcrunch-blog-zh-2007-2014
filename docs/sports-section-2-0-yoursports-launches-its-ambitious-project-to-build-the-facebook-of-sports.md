# 体育 2.0 版:YourSports 启动雄心勃勃的项目，打造体育脸书 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/10/11/sports-section-2-0-yoursports-launches-its-ambitious-project-to-build-the-facebook-of-sports/>

克里斯·麦考伊在喝咖啡时告诉我，“社交图谱中缺少了一个重要部分，那就是体育。”自然，麦考伊是一名前棒球运动员，Quora 的爱好者，也是所有体育用品的宗教消费者。但是，也许令人惊讶的是，他并不是唯一一个以这种方式看待当前体育景观的人。

麦考伊说，尽管 ESPN、BleacherReport、Twitter 和无数其他媒体正忙于将体育内容的消费数字化，但问题是主流体育媒体只覆盖了体育市场的一小部分。不仅如此，它还没有找到一种有效的方式来整合本地内容、个性化和社交发现——这些是我们在脸书时代所期待的东西。

因此，麦考伊和他的团队今天正式推出了 [YourSports](https://web.archive.org/web/20221209003816/http://www.yoursports.com/) ，这是一个雄心勃勃的项目，旨在重新思考“体育部分”，并通过将 Quora、脸书和 ESPN 的元素结合在一个包中，彻底改变我们创建和消费体育内容的方式。

任何拥有脸书账户的人都会熟悉 YourSports 界面。所以，简而言之，大多数人，包括你的猫。你的运动背后的理论是，运动本身天生就是社交的，然而生态系统缺乏一个专门的社交网络，或者用时下流行的术语来说，体育图表。反过来，我们体育迷都有自己的体育身份，这些身份是由社会关系定义的——无论是对球队、比赛、城市、其他粉丝还是特定运动。

体育的“社会化”最好地体现在 Twitter 和脸书已经成为与其他粉丝实时讨论游戏、事件和新闻的工具——作为第二屏幕体验。也就是说，一边看比赛。但 Twitter 上有很多噪音，虽然有“# mlbp offsets”这样的标签来关注人们对棒球季后赛的看法，但这不是一种专门的体育体验，而且有很多东西与你无关。

因此，YourSports 试图创建一个社交/本地网络，与你的朋友进行这些对话——关于任何运动，而不仅仅是 NBA、NHL、MLB 和 NFL。用户通过登录或脸书登录您的 Sports。后者会用你的名字和脸书的照片自动创建一个个人资料。从那里，你可以跟踪球队、城市、体育媒体等。，发布到这些页面的内容最终会出现在您的新闻订阅源中。例如，如果你成为某个团队的“粉丝”，你会看到与该团队相关的内容，如果你成为“敌人”，你就看不到了。

[![](img/596f0a8ca1fa278dc74ecfa3b65c4312.png "Profile")](https://web.archive.org/web/20221209003816/https://beta.techcrunch.com/wp-content/uploads/2012/10/profile1.png)

像脸书一样，有一个“首页”或主订阅源，你可以查看人们发布的文章的链接，并从你的网络中看到通知，实时阅读新闻订阅源风格的文章。你可以回复这些文章(以及发布它们的人)，在脸书和 Twitter 上分享你的回复，当然，也可以发布你自己的体育相关内容，无论是什么内容。

同样，这感觉很像体育的脸书，因为该团队花了两年的大部分时间建立了一个体育数据库，现在包括运动员、学校和场馆的 50 多万个档案，从高中体育到奥运会赛事。当然，这并不意味着所有这些品牌和个人都声称拥有自己的 YourSports 个人资料。远非如此。但在某种程度上，这类似于 Identified and [BranchOut](https://web.archive.org/web/20221209003816/https://beta.techcrunch.com/2012/10/08/turning-a-utility-into-a-community/) 如何看待网络发展，从脸书获取数据，为人们建立基本档案，他们的想法是，然后要求这些档案并成为管理员，建立它并丰富数据。

尽管如此，麦考伊告诉我们，他们已经签约了一些运动员、作家、团队和联盟作为早期采用者，他们将尝试创业公司的体育内容消费版本，并可能声称自己的个人资料。

因为麦考伊是 Quora 的粉丝，这个受欢迎的问答网站为 YourSports 建立网络的方式提供了很多灵感。特别是，Quora 使用户能够将他们的 Quora 内容分发给某个主题的关注者，使用一种复杂的算法来确定谁看到了什么内容。但麦科伊说，对大多数人来说，Quora 主题的工作方式可能会令人困惑。

[![](img/9476015bd34d336feed1d7fb959873e7.png "Frontpage")](https://web.archive.org/web/20221209003816/https://beta.techcrunch.com/wp-content/uploads/2012/10/frontpage.png)

所以他对 YourSports 说的想法是使内容分发民主化，从某种意义上说，他们希望一个随机的球迷能够向旧金山巨人队的追随者分发与旧金山纪事报 T4 的首席体育记者一样多的内容。一个特定帖子或链接的播放量取决于转发、分享和反应的数量，但真正的关键是，理论上，它提供了一种比 Twitter 更好的方式来将你的内容传递给你最喜欢的运动队的粉丝。

即使你在标签中使用“SFGiants ”,你的内容也不会被 Giants 的粉丝看到——这是你的 Sports 想要改变的。当然，更大的说，当平台达到一定规模时，会给巨人的粉丝带来很多噪音。

YourSports 方法与众不同的另一部分是它如何试图创建“动态网络”，将团队层面发生的事情与学校、联盟、协会等联系起来。例如，一个联盟可以通过 YourSports 网络中的所有学校和球队发送消息。最重要的是，YourSports 已经在地理位置的基础上建立了自己的平台，因此当某个特定城市的球队发生新闻时，它也会在州和国家层面上显示出来。

因此，我们的目标是能够在城市、州、国家和区域网络层面上告诉人们社区在谈论什么，哪些特定的团队、问题和内容最受欢迎和最不受欢迎。McCoy 说，在未来的某个时候，他们希望添加签到和标签，以便用户可以看到哪个高中或 D3 篮球队最活跃等。

[![](img/23419ffc945316905cc856491fc87837.png "networks")](https://web.archive.org/web/20221209003816/https://beta.techcrunch.com/wp-content/uploads/2012/10/networks.png)

YourSports 在产品方面还有很长的路要走，所以团队还没有决定商业模式。不过，就未来的商业扩张而言，麦考伊看到了一个将签到和趋势团队扩展到餐厅等地方的机会，这样用户就可以看到哪些酒吧或汉堡店最受斯坦福球迷的欢迎。YourSports 还准备推出其移动应用程序，因为它知道你正在练习或参加 49 人队的比赛，所以当你离开体育场、溜冰场等时，它可以将这些信息推送到你的手机上。

综上所述，McCoy 认为 YourSports 是第一个尝试为体育运动建立一个基于关系标签(在当地有效)的动态社交网络的公司。大图概念(和最终目标)肯定是独特的，即使在这个早期阶段的用户体验是熟悉的(即，几乎太像脸书)。就像任何初期的网络一样，你的许多运动功能都显得多余。

如果它发展成自己的用户群，那就越来越不真实了。但是，以 Foursquare 为例:在它真正成为一个有用的应用程序之前，该公司花了几年时间收集签到数据，并开始进行地理标记交易，向你显示你的朋友在哪里签到，最受欢迎的景点等。在规模上，YourSports 真的有可能改变我们消费体育的方式，将目前分散的部分连接成一个整体。

大多数人都有体育身份，包括当地垒球队和喜爱的国家职业运动队，但他们通常不会(也不喜欢)将两者混为一谈。主要的体育产业没有触及本地是有原因的。但是，由于它已经从 5，000 份报纸、博客、Twitter 账户中获取了新闻，并分发给 1，000 多个团队，YourSports 已经开始为一种模式奠定基础，这种模式可以证明将主要运动级别(青年、高中、大学、职业等)纳入其中是合理的。)在一起。它甚至显示出使其成为一项有利可图的业务的潜力，尽管这需要时间来证明我的怀疑是错误的。

当然，总会有风险投资:YourSports 已经从一些天使投资者那里筹集了 40 万美元的种子资金，目前正在筹集首轮融资。

YourSports 代表了一个重要的实验和一些伟大的想法，但它也感觉像是刚刚到达起跑线，还有一场马拉松要跑。
