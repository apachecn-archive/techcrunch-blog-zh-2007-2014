# EyeEm 的相机应用程序是一个漂亮的新摄影界面

随着 iPhone 和其他智能手机[逐渐成为](https://web.archive.org/web/20230203072937/https://techcrunch.com/2011/04/17/iphone-4-camera/)我们一直带在身边的默认傻瓜相机，照片应用出现了爆炸式增长。

人们不禁要问，另一家初创公司怎么可能推出另一款应用，并希望获得成功。然而，炙手可热的柏林初创公司 [EyeEm](https://web.archive.org/web/20230203072937/http://eyeem.com/) 认为，它可能只是偶然发现了一种新的方法，Instagram 等公司错过了这种方法:照片的内容。

经过超过 5000 名用户的测试后，EyeEm 是一款智能手机相机和照片共享应用程序，它可以监控用户的摄影品味，并使用这些数据来推荐朋友和志同道合者的类似照片相册。

EyeEm 还从 Passion Capital、Wellington Partners 和著名的柏林天使投资人 Christophe Maire(他也是该公司的执行主席)那里筹集了未披露金额的种子资金。

这款应用本身也非常擅长展示你可能会喜欢的新照片。这可能是我见过的相机应用程序中最好的内容发现界面，包括 Flickr 的。我敢说，这几乎是美丽的。EyeEm 非常关注摄影消费的界面，它显示了。Instagram，小心。

十个内置的过滤器可以快速增强照片，图像会使用 EyeEm 所谓的“vibes”自动分类。这就像一个智能相册，将图像放入主题照片流中。因此，你可以根据自己的兴趣来策划一个摄影订阅源，而不仅仅是你在 EyeEm 上关注的人。

捕捉，过滤，位置标记和分享到脸书和推特是非常容易的。

创始人兼摄影师 Florian Meissner 在丢失了他的数码单反相机并发现了整个社区的 iPhone 摄影师后，受到启发开发了这款应用程序。他将其描述为智能相册或“照片的魔力磁铁”。

加入 Florian Meissner 的创始团队包括联席 MD Lorenz Aschoff、CTO Ramzi Rizk、创意总监 Gen Sadakane、首席 iOS 开发人员 Wojtek Sczygiol 和前端开发人员 Guillaume 卡姆邦。
