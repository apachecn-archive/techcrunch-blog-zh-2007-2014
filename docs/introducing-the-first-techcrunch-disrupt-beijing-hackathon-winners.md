# 介绍首届 TechCrunch Disrupt 北京黑客马拉松获奖者

在弥合翻译差距、早餐咖啡的缺乏和丰富、互联网的集体斗争以及许多黑客使用天极的应用编程接口(API)之间，[TechCrunch Disrupt Beijing Hackathon](https://web.archive.org/web/20230203061847/http://disrupt.beta.techcrunch.com/BJ2011/)刚刚发生了，这简直令人惊讶。

大约 300 名黑客签署了 24 小时在一起的协议，100 名黑客为了在北京时间上午 11 点展示他们的黑客技术，勇敢地度过了一个充满不稳定连接和蔬菜面条的夜晚。每个团队都有一分钟的时间在多语言观众和评委面前展示他们的作品。

评委包括 TechCrunch Mobile 的 Greg Kumparak，icanhazgimber 的 [Ben Huh](https://web.archive.org/web/20230203061847/http://www.crunchbase.com/person/ben-huh) ，Zynga 的 Andy Tain，腾讯的 [Peter Zhong](https://web.archive.org/web/20230203061847/http://www.linkedin.com/pub/dir/Peter/Zhong) ，Sonos 的 Carter Agar，Google 的和 DigiCha 的 [Bill Bishop](https://web.archive.org/web/20230203061847/http://www.crunchbase.com/person/bill-bishop) ，他们尽最大努力分析了 46 个黑客，选出了其中的 6 个。除了永久的名誉和荣耀，获胜者还有机会在 TechCrunch Disrupt proper 上登台亮相。

见见第一届 TechCrunch Disrupt 北京黑客马拉松的获胜者，如下(当黑客上线时会有链接)。

**大奖 WeDiary —** 为你的博客写一篇按时间顺序排列的视觉日记。

**鞭炮—** 一款模拟玩鞭炮的手机游戏。

**TouchMusic —** 一款外观精美的智能手机应用，功能类似 theramin。

**HackMaster —** 一款教人如何编写 Javascript 代码的游戏，有一个爱情故事。

共享软件— 共享软件意识到你是在中国浏览，并加载了中国的分享按钮，而不是脸书和推特，这解决了一个大问题。

**Pomo —** 追踪你工作时间的手机应用程序——记录哪些时间是有效率的，哪些时间是没有效率的。
