# 生化奇兵评论:值得炒作？

![1-1.jpg](img/e42f1ddce8e4af7fc9650153b27b6626.png)

我很少再对游戏标题感到兴奋了——一个*半衰期*可能会让我兴奋不已，但我肯定不会在当地的游戏商店预订任何东西的副本——这就是我对*生化奇兵*真正感兴趣的原因。实际上，这似乎是一场我可以参与的游戏，不考虑孩子、妻子和逼近我的 500 万美元的最后期限。

这款游戏可在 360 和 PC 上下载，使用了*虚幻 3* 引擎，描述了一个社区陷入遗传失败的过程。

> 如果核心机制不好，所有这些故事和强大的写作都没什么关系，但幸运的是，《生化奇兵》是一款非常深入的游戏。你会遇到更多的质粒，你可以一次“装载”到你的身体里，这意味着你必须做出一些非常艰难的决定，什么时候你想要什么能力。

这有点让我想起了*镇压*，但可能会少吸。我还没看过比赛，但看起来确实不错。
