# 这是谷歌的 Project Tango 手机 TechCrunch 拍摄的一个房间的实际 3D 室内地图

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/21/heres-an-actual-3d-indoor-map-of-a-room-captured-with-googles-project-tango-phone/amp/>

我们昨天爆料称，谷歌正在生产一款名为 的 3D 传感智能手机原型 。我们还分析了设备 中视觉处理器的 功能，并讨论了它 对未来手机 的意义。

现在，我们在下面的视频中独家看到了 Matterport 用其中一个原型设备拍摄的一个房间的真实 3D 室内地图。Matterport 制造计算机视觉和感知计算解决方案，如绘制地图和创建室内空间 3D 重建的软件，是谷歌选择提供该设备早期原型的少数合作伙伴之一。

我们采访了 Matterport 首席执行官比尔·布朗，讨论了 Project Tango 重建 3D 场景的能力。Brown 表示，该项目极大地加速了移动 3D 捕捉领域的发展。移动设备已经成为可行的 3D 地图捕捉平台，其速度远远超过了大多数关注这项技术的人的预期。

Matterport 的软件管道允许它同时捕捉彩色相机数据和完整的 3D 数据，并将这些数据一起重建为精确的模型。Matterport 一直在制造自己的相机，以满足其所需的质量和价格点，但 Brown 表示，Project Tango 设备非常强大。

“我们得到的网格质量和我们从相机中得到的一样，”布朗说。“它‘看起来’并不那么好，但那只是因为它是一个原型。”

由于 Project Tango 设备上的低分辨率相机，该模型的外观和感觉略显粗糙，但 Brown 表示，这些问题应该很快就会得到纠正。

正如你从模型中看到的，Project Tango 设备的功能已经非常令人印象深刻，随着时间的推移，我们只会看到我们的手机在感知和解释我们的环境方面变得更加熟练。

Matterport 将出现在高通即将在巴塞罗纳举行的世界移动通信大会的主题演讲中。它将展示其在移动设备上的 3D 捕捉和重建软件，并更多地谈论它如何帮助人们运行利用这些模型的应用程序。
