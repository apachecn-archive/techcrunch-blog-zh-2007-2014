# 富士通的 IS12T Windows Phone 芒果今天在日本上市(快速上手)

上个月，[富士通](https://web.archive.org/web/20230205003421/https://techcrunch.com/tag/fujitsu) [在](https://web.archive.org/web/20230205003421/https://techcrunch.com/2011/07/18/fujitsu-windows-mango/) [日本](https://web.archive.org/web/20230205003421/https://techcrunch.com/2011/07/26/official-japan-to-get-the-worlds-first-windows-phone-7-mango-handset-in-september/) [发布了](https://web.archive.org/web/20230205003421/https://techcrunch.com/2011/07/28/toshiba-fujitsu-is12t-worlds-first-windows-mango-cell-phone-up-and-close-video/)IS12T，宣布成为世界上第一款运行 Windows Phone 7.5 的手机，又名[芒果](https://web.archive.org/web/20230205003421/https://techcrunch.com/tag/mango)。该国第二大移动运营商(也是该手机的独家供应商) [KDDI au](https://web.archive.org/web/20230205003421/https://techcrunch.com/tag/kddi) 没有浪费太多时间:IS12T 今天在这里上市(这里是富士通的官方英文新闻稿，从今天开始)。

正如我们之前报道的那样，Mango 手机配备了分辨率为 800×480 的 3.7 英寸液晶显示器，13.2 百万像素的 CMOS 摄像头，防水防震的机身，32GB 的内存，microUSB 端口，IEEE 802.11b/g/n Wi-Fi(无网络共享)，DLNA 支持，GSM/CDMA 等。
[![](img/3b0e7247da538c7f462e8c71735b6276.png "IMG_1746")](https://web.archive.org/web/20230205003421/https://techcrunch.com/wp-content/uploads/2011/08/img_1746.jpg)
UI 有九种语言，IS12T 有三种“霓虹”色(橘色、黑色、品红色)。

[![](img/64bdb385ba13cf63fa22ba30d7fd0cf3.png "IMG_1747")](https://web.archive.org/web/20230205003421/https://techcrunch.com/wp-content/uploads/2011/08/img_1747.jpg)

今天，我去了我家附近唯一的一家手机店看了看这款手机，但不幸的是，我只拍了几张照片(见上图),玩了几分钟就不得不离开了(无照片无视频政策)。

[![](img/76298450ea97fc8361a6d9186ed27159.png "20110823-01al")](https://web.archive.org/web/20230205003421/https://techcrunch.com/wp-content/uploads/2011/08/20110823-01al.jpg)

我最初的印象是，这款手机本身感觉很轻很薄，但也非常便宜和可塑(我无法让 Xbox Live、Internet Explorer 和其他应用程序旋转，因为 Wi-Fi 和 3G 被关闭了——在我玩它的几分钟里，用户界面感觉很棒)。

就价格而言，IS12T 在日本定位于高端手机市场:商店(以及其他所有零售商，就此而言)出售的手机价格高达 955 美元(但在某些情况下会有大幅折扣)。因此，如果富士通真的开始在日本以外销售手机(他们确实有国际计划)，不要指望这是一个预算模型。
