# LaCie 坚固安全的外置硬盘既坚固又安全

[![](img/7c05847b5cb4e09a0378d75b125ff8a8.png "laclie rugged safe")](https://web.archive.org/web/20221207214223/https://beta.techcrunch.com/wp-content/uploads/2010/05/laclie-rugged-safe.jpg) 
[LaCie](https://web.archive.org/web/20221207214223/http://www.crunchgear.com/tag/lacie/) 想要确保你的数据安全。非常安全。该公司采用了其坚固的硬盘产品线，增加了指纹扫描仪和 128 位 AES 加密技术，从而制造出了可能是硬盘疯狂的麦克斯的产品。

这款硬盘有 500GB 和 1TB 两种规格，但遗憾的是，它既没有 USB 3.0 也没有 eSATA，尽管这款硬盘除了 USB 2.0 之外还支持 Firewire。500GB 版本的零售价为 189 美元，1TB 的售价为 299 美元，所有额外的安全性和男子气概的成本略高。显然，如果您需要相当于 A-Team 的团队来保护您的数据，这取决于您。
