# 当隐私遇上虚伪:暴雪真实 ID 版| TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/07/08/when-privacy-meets-hypocrisy-blizzard-real-id-edition/>

[![](img/bb22b26763a47fbb1833dff284f0dec7.png "orcab")](https://web.archive.org/web/20230315033158/http://www.crunchgear.com/2010/07/08/when-privacy-meets-hypocrisy-blizzard-real-id-edition/orcab/)

如果暴雪想引发一场讨论，那么，任务完成了。这家[公司计划](https://web.archive.org/web/20230315033158/http://www.crunchgear.com/2010/07/06/want-to-qq-on-blizzards-message-board-better-get-ready-to-use-your-real-name/)在其留言板上引入真实身份，但如果不*吓到人们*的话，这一计划毫无成效。继续抓狂吧。

在我再次进入这个话题之前，我想先说一下:如果暴雪在宣布新系统的同时并没有真正实现它的意图，我不会感到惊讶。或许它想做的只是衡量公众的反应？既然它知道了公众在竭力反对~！过了它，也许它会回来说，“由于压倒性的反应，我们想宣布一些改变，我们真正的 ID 政策将在未来的工作方式…”或者，类似地，这可能是动视公司比暴雪公司做得更多——我们都知道我不是动视公司的朋友。在这种情况下，你的愤怒最好指向 Bobby Kotick 公司，而不一定指向暴雪公司的人。

但为了便于讨论，我们姑且说这是全速前进。

从 [*星际争霸 2*](https://web.archive.org/web/20230315033158/http://www.crunchgear.com/tag/starcraft-ii/)发布开始，你在暴雪的论坛上发帖时需要用真名。(关于魔兽世界 方面的事情，新的制度要到下一个资料片大灾变*发布后才会生效，而这还需要几个月的时间。)这是不可追溯的，因为没有办法实际上回到和重新工作的旧论坛与真正的 ID。您在当前论坛上的所有帖子将永远保持原样。至少在服务器离线之前。所以，如果你在论坛上以伟大的血精灵法师哈克索的身份发帖时出丑，那么你的身份是安全的。*

暴雪表示，它正在转向新系统，主要是为了清理你的恶劣行为。悲伤却真实。也许如果暴雪不需要花一整天的时间来管理它的论坛，那么你们中的许多人决定在匿名的面具下表现得像傻瓜一样是可以接受的，那么你们就不会在第一时间陷入这种混乱。所以某种意义上是你自己的错。

让我们继续讨论隐私问题，其中一些是由民主技术中心的优秀人员引起我的注意的。我见过很多大意如下的废话:“用真名发帖会危及我的职业*处境*”这似乎源于你对游戏的热爱在社交和/或职业上的尴尬。类似于“如果我的老板看到我在凌晨 3 点发帖抱怨掉线率，他/她会怎么想？”为什么你的老板会在乎你在自己的时间里做了什么——只要它不会削弱你在上班时间生产小部件的能力？被人看到询问在哪里可以找到某个暴徒，或者报告新的五人地牢中的一个 bug，真的会有损你的社会地位吗？你们住在哪个星球上的人认为这是件大事？我以为我们已经过了被人看见玩电子游戏被认为是“坏事”的时代了？这不是隐私问题，而是社会不安全感的问题。“邻居们会怎么想？”谁在乎他们怎么想！过你认为合适的生活，不要担心这个人或那个人是否认为这是“酷”或“可接受的”！

让我们不要忘记这样一个事实，我们现在生活在网上，你们中的许多人声称“侵犯隐私~！”，对不起，没有一条腿站得住脚。你们中有多少人有推特账号或 T2 账号或脸书账号？在你说“嗯，不应该在那里”之前，Twitter 要遭受多少次大规模的安全漏洞？你们中有多少人会在脸书上发布自己和家人的照片，让全世界都看到——当然，除非你采取大规模的主动措施，关闭你的账户？你们中有多少人使用 [FourSquare](https://web.archive.org/web/20230315033158/http://search.beta.techcrunch.com/query.php?s=foursquare) 向全世界广播你们的*确切位置*？还有什么比告诉人们“嘿，我就在这里”更侵犯隐私的呢？你们中有多少人使用 [BitTorrent](https://web.archive.org/web/20230315033158/http://www.crunchgear.com/tag/bittorrent/) ，当你的 IP 地址有目共睹时，尽情下载，更不用说让像美国版权组织这样的人看到并卷入[诉讼](https://web.archive.org/web/20230315033158/http://www.crunchgear.com/2010/07/07/hurt-locker-bittorrent-lawsuits-continue-but-with-restraints/)？你们中有多少人使用 Gmail 服务，这种服务会主动扫描你的电子邮件来推销广告？如果你那么关心隐私，你应该建立自己的邮件服务器。但是你当然不知道。你只在方便的时候关心隐私。只是现在，它涉及到你花你的空闲时间施法的事实，突然你的“隐私”处于危险之中。

我没有时间选择性的愤怒。

另一件事:玩魔兽世界的 1100 多万人中有多少人真正参与了论坛？把这场争论说成是“整个社区”都在竭力反对，嗯，不。

你真的认为 4Chan(或任何其他团体或个人)能够逃脱对在新论坛上发帖的人的骚扰吗？这是我见过的一种常见的抱怨。“现在现实生活中人们会烦我！”这听起来像是一张诉讼的单程票，由动视暴雪提供。仅仅因为你的名字是“在那里”,并不意味着人们可以威胁你。你肯定认识这个吧？

对于企业家群体来说:这听起来是一个创造一站式“完全未经授权”的暴雪留言板的好机会…

不要介意所有这些乱七八糟的问题有一个非常现实的解决方案:不喜欢它，不要在那里发帖。没有人，甚至暴雪，会强迫你在论坛上发帖。
