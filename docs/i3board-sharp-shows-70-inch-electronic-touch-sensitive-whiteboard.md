# i3 板:夏普展示 70 英寸电子触摸感应白板

夏普日本[昨天宣布了面向日本市场的 i3board PN-L702B，这是一款面向企业和学校的 70 英寸电子白板。该设备本质上是一个 70 英寸的触摸屏(LCD)，配有手写笔和特殊软件，使用户能够通过手势在屏幕上做笔记或绘图，滚动或改变对象的大小。](https://web.archive.org/web/20230205025723/http://www.sharp.co.jp/corporate/news/110831-b.html)

白板可用于显示从 Windows XP/Vista/7 PC(通过 USB)或直接通过局域网端口从网络传输到白板的任何类型的内容(包括视频)。它具有全高清分辨率、LED 背光、3，000:1 对比度、300cd/m2 亮度和一个 HDMI 端口。

夏普计划于 9 月 20 日在日本开始销售 i3 板 PN-L702B，价格为 9700 美元。一个更小的版本(60 英寸)零售价为 7100 美元。
