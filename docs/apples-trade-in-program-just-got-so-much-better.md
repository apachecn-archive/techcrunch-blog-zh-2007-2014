# 苹果的以旧换新计划变得更好了

如果你想扔掉那台旧笔记本电脑，先等等。[苹果](https://web.archive.org/web/20230203074533/https://techcrunch.com/tag/apple)刚刚更新了它的[以旧换新政策](https://web.archive.org/web/20230203074533/http://www.apple.com/recycling/gift-card/)，现在你可以带进任何苹果产品，以及任何台式机或笔记本电脑，免费回收，甚至商店积分。是的，*任何台式机或笔记本电脑，包括个人电脑——而且不会花你一分钱。*

多亏了在 [MacRumors](https://web.archive.org/web/20230203074533/http://www.macrumors.com/2011/08/09/apple-expands-recycling-options-for-old-computers-and-mobile-devices/) 的一些敏锐的眼睛，我们现在知道事情将会是这样的:你现在可以用苹果礼品卡上的“公平市价”(由苹果商店确定)来交换你的 [iPad](https://web.archive.org/web/20230203074533/https://techcrunch.com/tag/ipad) 和/或 [iPhone](https://web.archive.org/web/20230203074533/https://techcrunch.com/tag/iphone) 。苹果已经在台式机和笔记本电脑上这样做了一段时间，包括 PC 和 Mac，但现在正在将该产品扩展到 iPhones 和 iPads。

但是请记住，这只适用于你的旧技术适合用于新的东西。如果它没有任何货币价值，就被扔进回收堆。但是不用担心——苹果对此也有简单、无痛苦的计划。

以前是当你带进来的旧电脑和显示器一文不值的时候，如果你买的是 Mac 就可以免费回收。如果你买的不是 Mac，你必须支付 30 美元的运费。现在，无论你是否购买 Mac，任何品牌的电脑或显示器都将免费回收。

大多数品牌对他们的旧产品都有某种形式的以旧换新计划，但就像苹果一样，他们提供信用而不是现金。因此，如果你不是苹果粉丝，也不真的需要 Apple gear 的积分，花两秒钟时间查看制造商的以旧换新计划细节并获得你喜欢的品牌的积分是值得的。其他第三方程序如 [Gazelle](https://web.archive.org/web/20230203074533/http://www.gazelle.com/) 也可以让你很容易地用旧东西换钱。

但是无论你选择在哪里倾倒你的旧东西，尽量不要瞄准垃圾桶。这个星球可以休息一下了。
