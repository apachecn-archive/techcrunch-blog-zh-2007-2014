# SoundCloud 推出超酷的现场录音和 iPhone 应用程序录音功能

![](img/d09203ea71571b7289105119729594d2.png)

今天， [SoundCloud](https://web.archive.org/web/20230203032536/http://soundcloud.com/) ，一个迄今为止专注于音乐制作人的音频平台，向更广泛的音频共享领域做出了重大转变。到目前为止，SoundCloud 已经做了相当不可思议的工作，吸引了知名艺术家到它的平台上，如喷火战机乐队、凯莉、Deadmau、莫比和 Caribou，他们利用它有效地将他们的音乐社会化。但是今天，Soundcloud 发布了通过网站直接录制声音的功能，并通过其更新的 iPhone 应用程序。这使它进入了纯音频共享平台的领域，如 [Cinch](https://web.archive.org/web/20230203032536/http://www.cinchcast.com/) 和 [Audioboo](https://web.archive.org/web/20230203032536/http://audioboo.fm/) 。在其他新闻中，SoundCloud 的用户也超过了 200 万。

SoundCloud 在网站或 iPhone 应用程序上的新记录功能让用户可以在任何地方捕捉声音，并将其放入波形播放器，以便在社交网络、网站、SoundCloud 社区或朋友之间私下分享。iPhone 应用程序还能够为录音添加图像和位置。举例来说，这正是 Audioboo 和 Cinch 已经在做的事情，但 SoundCloud 上的社交功能可能会超越任何现有的播放器。看看下面的截图。看过应用程序后，我可以看到我可能会将自己的音频记录转移到 SoundCloud，因为这种体验真的非常好。

SoundCloud 的首席执行官兼创始人 Alexander Ljung 明确表示，他的目标是一个比音乐人更大的市场，他说:“SoundCloud 为他们提供了一个平台，可以创建和分享任何东西，从个人音频消息和博客，到诗歌、喜剧和他们周围的世界的声音。”

在某种程度上，此举是对用户的回应，就像一对夫妇骑自行车环游世界，记录他们旅途中的声音，皇家文学学会一直在上传他们的讲座。

然而，当我问这是否是远离音乐的一个大转变时，他说“这不是一个彻底的转变。人们已经在网站上这样做了，我们只是让他们以及所有想要分享瞬间创意的音乐人能够以最简单的方式做到这一点。”

在我看来,*相当激进。我很清楚，SoundCloud 的创新速度比许多音频共享平台要快得多。Soundcloud 有可能成为我们真正的音频 YouTube 吗？他们现在正在采取所有正确的行动。*

SoundCloud 的 API 还允许第三方开发者向其应用程序库添加 100 多个移动、桌面和网络应用程序。最近的其他创新包括重新设计的仪表板和给声音添加音符。
