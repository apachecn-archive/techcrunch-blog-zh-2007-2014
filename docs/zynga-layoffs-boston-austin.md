# Zynga 的波士顿，奥斯汀今天裁员:“几乎每个人都被蒙在鼓里”

**更新:** [Zynga CEO 马克·平卡斯已经证实了这一消息](https://web.archive.org/web/20230326143549/https://techcrunch.com/2012/10/23/zynga-ceo-mark-pincus-confirms-layoffs-5-of-workforce-potential-closures-for-u-k-japan-offices/)。

今天我们有更多关于 Zynga 裁员背后的背景故事。直到今天早上，没有一个普通员工知道裁员会在哪里发生，尽管高级管理层已经提前计划了裁员(正如 Zynga 本月降低今年预订量预期时所表明的那样)。

今天早上，Zynga 的游戏高级副总裁托德·阿诺德来到波士顿办公室，告诉 50 人的工作室，公司将关闭它。

同样，在奥斯汀，大多数员工(其中大约三分之二在 Ville 工作，三分之一在 Bingo 工作)直到今天才知道。~~我们仍然没有裁员总数的最终数字，Zynga 也没有回复置评请求。Zynga 最终确认裁员，我们可能会看到大约 150 名员工。~~

许多奥斯汀员工收到了调回旧金山的提议，Ville 正在该公司印度办公室进行维护——就像 Zynga 终止了许多不太受欢迎的游戏一样。

“几乎每个人都被蒙在鼓里，”一位接近奥斯汀办公室的消息人士说。

同时在波士顿，整个工作室都被解雇了。事实上，该办公室即将推出一款令公司内部许多人兴奋不已的产品。但在此之前，该办公室主要是帮助 FarmVille 续集。

“这种游戏不会有出头之日。他们不想取消这个头衔。这很痛苦，但他们想省钱，”一名前员工表示。“我认为他们只需要裁掉数百名员工，这是最简单的方法。他们看的是谁没有在玩游戏，”

我们也从以前的员工那里听说芝加哥办公室确实在裁员。但是这个办公室相对来说还很年轻，直到几个月前才作为开发工作室使用。芝加哥没有裁员。

尽管此次裁员恰逢苹果发布 iPad Mini，但时间安排与 Zynga 即将发布的收益公告有更大的关系。

“(首席执行官马克)平卡斯希望能够说公司是盈利的，但随着收入的下降，这变得更加困难，”一名前员工表示。“如果你认为这是一家银行里有 16 亿美元现金的稳定公司，他们正在做的短期事情根本没有任何意义。”

目前，华尔街投资者对该公司的估值为 16.7 亿美元——这几乎正好是该公司在第二季度末手头的现金、短期和长期投资。本质上，华尔街是在说，除了手头的现金，该业务本身几乎没有任何价值，这引发了 Zynga 可能成为一个有趣的收购目标的猜测。

也就是说，这位消息人士补充说，平卡斯真诚地希望成为一名成功的上市公司首席执行官。事实上，平卡斯的父亲在他的职业生涯中为许多上市公司的首席执行官提供投资者关系方面的指导。

“马克有足够的投票权，他不必出售。他是保持选择余地的王者。一名前员工表示:“他有一个愿望——就像他自己说的那样——在有生之年把这家公司经营成一个互联网宝藏。”。
