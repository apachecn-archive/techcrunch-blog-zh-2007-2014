# ReTweet.com 发布，看起来很像 TweetMeme

![picture-130](img/f7dac56d3639607c5055d5f3dc8bfca2.png "picture-130")[【ReTweet.com】](https://web.archive.org/web/20230220024115/http://retweet.com/)，一个[由](https://web.archive.org/web/20230220024115/http://tweetmeme.com/) [Mesiab Labs](https://web.archive.org/web/20230220024115/http://mesiablabs.com/) (臭名昭著的推特垃圾软件程序蜂鸟背后的初创公司)组建的推特网挑战者，在几周前用[一个公告和登陆页面](https://web.archive.org/web/20230220024115/https://techcrunch.com/2009/07/26/retweetcom-looks-to-be-a-tweetmeme-competitor-with-a-killer-domain-name/)戏弄推特之后，已经在网上首次亮相。

你可能还记得，就在 ReTweet.com 离开 woodworks 后不久，TweetMeme 就已经[威胁 Mesiab Labs](https://web.archive.org/web/20230220024115/https://techcrunch.com/2009/07/27/defending-its-turf-tweetmeme-is-already-threatening-to-sue-retweet/) 起诉后者[公然抄袭其转发按钮代码和网站设计](https://web.archive.org/web/20230220024115/http://uk.beta.techcrunch.com/2009/07/28/retweetcoms-reputation-will-be-permanently-stained-by-the-tweetmeme-affair/)。两家初创公司的目标都是成为转发之王，这是越来越受欢迎的 Twitter 服务上越来越受欢迎的活动，他们显然不是友好地竞争。

(为了全面披露:我们在 TechCrunch 网络上使用 TweetMeme 的转发按钮)

由于两家初创公司之间的在线争论开始升温时，ReTweet.com 尚未推出，因此很难断定两家网站会有多相似，尽管提供的截图表明它们看起来非常相似。现在，ReTweet.com 终于上线了，它和 TweetMeme 的相似之处确实惊人:

![](img/0559481663e356ad825d229d1207c4aa.png)

![](img/6794fa077ebcf7a6a3accfcd443dde80.png)

现在，在另一个服务上激发一个人的网站设计本身没有什么错，但考虑到这个游戏中只有两个值得注意的玩家，而且其他事件已经引起了关注，对 Mesiab 实验室来说，至少尝试让它看起来足够不同可能是更明智的。但不，ReTweet.com 的布局、措辞和配色方案显然是明目张胆的剽窃，我敢肯定他们也是故意的。

我不愿意在这里表明立场，但在跟踪了 Mesiab 实验室和他们卑鄙的蜂鸟产品几个月之后(稍后会有更多)，我倾向于同情 TweetMeme。他们当然可以利用一些竞争，但我更希望看到两家公司基于优点和尝试新事物来竞争，而不是 T2 用语言和诉讼威胁来争斗。

重申一下 Erick 之前的观点:Twitter 很可能最终被[注册为术语](https://web.archive.org/web/20230220024115/https://techcrunch.com/2009/07/01/twitter-to-developers-tweet-your-heart-out-but-dont-twitter-it/)‘retweet’，就像他们注册‘tweet’这个词一样。

**更新:**还有， [@Retweet](https://web.archive.org/web/20230220024115/http://twitter.com/retweet) 貌似是暂停账号。

未完待续，我确定。
