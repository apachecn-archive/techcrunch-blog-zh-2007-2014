# Twitter 照片分享平台 TwitPic 增加了视频支持

Twitter 媒体分享网站 [TwitPic](https://web.archive.org/web/20230203014818/http://twitpic.com/) 正在向照片世界之外扩张。这家初创公司[今天推出了](https://web.archive.org/web/20230203014818/http://blog.twitpic.com/2011/02/now-share-videos-on-twitpic/)视频支持，这将允许用户从网站本身以及支持 TwitPic 的移动和桌面客户端上传视频。TwitPic 现在有 1700 万用户，它说每天有超过 30000 名新用户。

添加视频是 TwitPic 合乎逻辑的下一步，到目前为止，TwitPic 一直专注于允许用户向 Twitter 上传照片。Twitter [照片分享赛马](https://web.archive.org/web/20230203014818/https://techcrunch.com/2010/08/18/the-twitter-photo-sharing-horse-race/)无疑是竞争激烈的。竞争对手 [Yfrog](https://web.archive.org/web/20230203014818/http://yfrog.com/) 已经支持视频上传，Plixi】刚刚被社交网络 Lockerz 收购。Twitter 最近还与 TwitPic 和其他公司合作，在新版 Twitter 界面中加入了内嵌图片。

当然，TwitPic 在创始人 Noah Everett 的领导下，已经推出了核心产品的扩展，包括 [TwitPic Labs](https://web.archive.org/web/20230203014818/https://techcrunch.com/2010/08/12/twitter-photo-sharing-site-twitpic-launches-labs-to-showcase-side-projects/) 来展示平台的创新项目。埃弗雷特正在从事一项名为 [Heello](https://web.archive.org/web/20230203014818/http://heello.com/) 的[秘密创业](https://web.archive.org/web/20230203014818/https://techcrunch.com/2010/08/10/twitpic-announces-stealth-communications-startup-heello/)。

那么 TwitPic 会改名来体现视频的加入吗？埃弗雷特告诉我们，这个名字将保持不变。他说*“我们发现无论使用哪种第三方服务，当用户在 Twitter 上分享照片时，术语‘Twitpic’无处不在。我们的名字对我们来说是一个巨大的好处，品牌本身现在已经传播到世界各地，甚至被包括在流行的说唱歌曲歌词中。”*
