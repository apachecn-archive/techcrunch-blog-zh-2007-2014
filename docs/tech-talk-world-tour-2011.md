# 苹果将通过“Tech Talk World Tour”向全球开发者传授 iOS 5

苹果的全球开发者大会(WWDC)对开发者来说是一个相当大的事件——但是，与“全球”这个名字的含义相反，它对那些不能来三藩市的人来说就没那么大了。

苹果希望向那些无法证明此行合理性的开发人员唱出 iOS 5 的声音，它再次开始了他们所谓的“技术对话世界之旅”。他们在 2008 年和 2009 年举行了类似的活动，2010 年出现了无法解释的中断。

与 WWDC 不同的是:科技讲座是免费的。你需要是一名注册的 iOS 开发者(他们会优先考虑商店里已经有应用的人)，但除此之外是在~~母舰~~的房子里。

看看议程，很明显会谈的焦点是什么:iOS 5 和 iCloud。如果你从未参加过这些活动，并且对发展有浓厚的兴趣，那么(不存在的)报名费是非常值得的。苹果带来了一些他们最好的工程师，你可以和那些比任何人都了解这些东西的人面对面交流。你可以在这里注册或找到更多信息。

不要期望在这些活动中听到太多秘密的新细节——但是，嘿，如果你*做* …你知道[给谁打电话。](https://web.archive.org/web/20230205001109/https://techcrunch.com/contact/)
