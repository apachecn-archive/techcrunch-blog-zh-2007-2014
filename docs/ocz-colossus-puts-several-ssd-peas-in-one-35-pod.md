# OCZ 巨像将几个固态硬盘豌豆放在一个 3.5 英寸的豆荚中

这是我们在超高端存储系统中看到的东西，但现在正在发烧友消费者层面实现。OCZ 庞然大物在其毫无特色的 3.5 英寸外壳中，配备了两个安装在单个 PCB 上的 RAIDed SSD 驱动器。你可以打赌它会很快，尽管 RAID 控制器显然被额定为“仅”260MB/s。使用两个驱动器(或在巨像 4X 中为四个)，你会认为他们很容易达到这一点，但你也有理由期待这样的配置更多。

128GB 的价格是 299 美元，所以这仍然是一个相当奢侈的项目，但多个非高端驱动器在 RAID 中的方法似乎正在获得牵引力。当然，以这个价格，你可以得到一个[普通的高端固态硬盘](https://web.archive.org/web/20221006110201/http://www.crunchgear.com/2009/07/20/lexar-rolls-out-fastest-crucial-ssds-to-date-the-m225/)，但如果你想保持 260MB/s 的速度直到 1tb，那么巨像可能是你最好的选择(尽管它会非常贵)。
