# 监管工具 Storify 与 MSNBC 的突发新闻合作，获取新闻内容

如你所知，去年秋天[在 TechCrunch Disrupt](https://web.archive.org/web/20230204120304/https://techcrunch.com/2010/09/27/storify/) 上推出的内容监管平台 [Storify，](https://web.archive.org/web/20230204120304/http://storify.com/)汇集了来自社交网站的推文、脸书状态更新、视频等，以创建一个故事或问题的实时视图。今天，[宣布](https://web.archive.org/web/20230204120304/http://storify.com/storify/breaking-news-is-now-a-source)首次与[突发新闻](https://web.archive.org/web/20230204120304/http://www.breakingnews.com/)合作，这是一个 [MSNBC 拥有的](https://web.archive.org/web/20230204120304/https://techcrunch.com/2010/03/11/msnbc-breaking-news-facebook/) [推特账号](https://web.archive.org/web/20230204120304/http://twitter.com/#!/breakingnews)和网站，用于实时更新世界上的突发新闻。

Storify 是这样工作的。Storify 允许您围绕一系列推文、Flickr 照片、脸书状态更新、YouTube 视频和更多内容构建和嵌入一个故事。在 Storify 平台上，您可以简单地搜索并将内容拖到 Storify 故事中。一旦你用所有这些精选的内容创建了一个故事，你就可以通过一个类似的 Javascript 将真实的故事嵌入到你的博客或内容管理系统中。

通过突发新闻合作伙伴关系，Storify 用户现在可以添加来自 [@BreakingNews](https://web.archive.org/web/20230204120304/http://twitter.com/#!/breakingnews) Twitter 账户的更新，以及来自新闻专线的其他更新。所以突发新闻现在是一个来源，类似于推特、脸书、YouTube 和其他。

这种合作的独特之处在于，在 BreakingNews.com，你可以直接从新闻故事中创造故事。Storify 创建了一个特殊的按钮(类似于 Tweet 按钮或 Like 功能)，允许任何人从突发新闻中创建并嵌入 Storify 故事。

联合创始人伯特·赫尔曼解释说，这是 Storify 的第一次双向合作，出版商将创作工具集成到了网站上。他正在寻求与其他主要出版商建立类似的合作关系。

Storify 最近从 T2 的科斯拉风险投资公司筹集了 200 万美元，T3 已经被许多新闻机构使用，包括 T4 的半岛电视台。

Storify 自 4 月 25 日以来一直处于公开测试阶段，即将创建超过 10 万个故事，所有网站平均每月有 650 万次故事浏览量。
