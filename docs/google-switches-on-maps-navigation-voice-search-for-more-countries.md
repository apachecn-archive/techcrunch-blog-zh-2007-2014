# 谷歌开启地图导航，语音搜索更多国家

![](img/9868b86b980f9fd5be0fed8a44b4cb16.png)

【Android 版谷歌地图导航现在可以在美国或英国以外的地区[使用，因为免费的逐路段导航系统已经](https://web.archive.org/web/20221209103117/http://eu.beta.techcrunch.com/2010/06/09/google%E2%80%99s-free-sat-nav-for-android-reaches-mainland-europe/)[为加拿大、法国、意大利、德国、西班牙、荷兰、丹麦、奥地利、瑞士和比利时的用户开启](https://web.archive.org/web/20221209103117/http://www.engadget.com/2010/06/09/google-maps-navigation-visits-canada-and-mainland-europe-remain/)(**更新:**和[葡萄牙](https://web.archive.org/web/20221209103117/http://googlemobile.blogspot.com/2010/06/fun-on-autobahn-google-maps-navigation.html))。

此外，谷歌语音搜索现在对法语、德语、意大利语和西班牙语用户可用。已经支持英语、中文普通话和日语。

谷歌还详细介绍了如何为语音搜索添加新语言:

> 我们的目标是将谷歌语音搜索带给所有语言的使用者。我们遵循严格的流程来添加每一种新的语言或方言。我们直接与每个国家的母语人士合作，花费数周时间收集口语话语，以创建支持该服务的特定模型。
> 
> 我们的助手被要求在各种声学条件下，如在餐馆、繁忙的街道和车内，用他们的母语朗读流行的查询。我们还为每种语言构建了超过一百万个可识别单词的词汇表。这不是一个小壮举，但我们喜欢这样做。

谷歌表示，这种方法的一个副作用是，如果你在墨西哥说西班牙语，你可能会发现这项服务并不像宣传的那样有效，因为口音不同。

语音搜索可以在 iPhone、黑莓和诺基亚 S60 设备上运行。显然，它也可以在 Android 设备上工作，尽管你最好查看宣布增加新支持语言的博客帖子，以了解如何激活它。此外，该应用程序仅适用于法国、德国、意大利和西班牙的 Android 市场。

谷歌地图导航在谷歌地图 4.2 版本中可用，在 Android 设备 1.6 和更高版本上可用。
