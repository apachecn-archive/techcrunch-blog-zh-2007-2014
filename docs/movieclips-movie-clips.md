# 看你的了，孩子——MovieClips.com 测试版上线了

影迷和影迷们，欢呼吧，坐下来，拿出爆米花。[MovieClips.com](https://web.archive.org/web/20230205141811/http://movieclips.com/)宣布，在与六大好莱坞电影公司达成许可协议后，它已经推出了在线电影剪辑目的地网站公测版。

它继续前进，让我的一天。

该网站在推出时拥有大约 12，000 个两分钟的流媒体剪辑，这些剪辑来自 20 世纪福克斯、米高梅、派拉蒙、索尼电影公司、环球电影公司和华纳兄弟电影公司的图书馆。它承诺数以千计的人也将很快到来。

由于这家初创公司正在对冲广告收入和用户代销商购买收入的赌注，这些剪辑都是免费的。

重温经典的场景，如[一把钱](https://web.archive.org/web/20230205141811/http://movieclips.com/#p=1&q=movie%253A%253A%2522A%2520Fistful%2520of%2520Dollars%2522)、[悲惨](https://web.archive.org/web/20230205141811/http://movieclips.com/#p=1&q=movie%3A%3A%22Misery%22)、[卡萨布兰卡](https://web.archive.org/web/20230205141811/http://movieclips.com/#p=1&q=movie%3A%3A%22Casablanca%22)、[做正确的事](https://web.archive.org/web/20230205141811/http://movieclips.com/#p=1&q=movie%253A%253A%2522Do%2520the%2520Right%2520Thing%2522)和 [Mallrats](https://web.archive.org/web/20230205141811/http://movieclips.com/#p=1&q=movie%253A%253A%2522Mallrats%2522) (是的，那也是我书中的经典)。

一旦你找到了你一直在寻找的片段，就像第一部《终结者》电影中臭名昭著的[“我会回来的”场景](https://web.archive.org/web/20230205141811/http://movieclips.com/watch/the_terminator_1984/ill_be_back/)，你可以通过“炒作”或喜欢它来竖起大拇指，在视频下方分享你的评论，或者在脸书、Twitter、Digg 等网站上与你的朋友分享整个事情。它也有琐事！

据我所知，你应该也能嵌入剪辑，但是由于代码没有出现在适当的框中，用户目前似乎不能。

该网站允许您按电影或演员浏览电影剪辑，但也可以根据动作(例如攻击)甚至情绪(例如恐怖)来查找场景。当你忙于搜索时，你会得到一个方便的剪辑缩略图，你可以悬停在上面预览。该网站唯一的缺点是剪辑加载非常慢，至少在我看来是这样。

MovieClips 计划从附属合作伙伴关系中获得收入；你可以直接在合法网站上购买或租借完整长度的电影，有时也可以直接从 iTunes 上购买或租借。

这项服务背后的创业公司是私人持有的，总部设在加利福尼亚州圣莫尼卡。该公司已经联合了一些著名的顾问，包括作家兼哈佛商学院教授克莱顿·M·克里斯坦森、希德·甘尼斯(美国电影艺术与科学学院前院长)、前漫威影业有限责任公司主席迈克尔·a·赫尔凡特、Omniture 联合创始人兼首席执行官乔希·詹姆斯和瑞安·赖特、SVP、索尼 BMG 音乐。

我敢肯定 MovieClips.com 不是第一个这样的(也查看尚未推出的 [TC50 2009 年决赛选手](https://web.archive.org/web/20230205141811/https://techcrunch.com/2009/09/15/tc50-find-the-perfect-scene-every-time-anyclip-is-a-search-engine-for-movie-clips/) [AnyClip](https://web.archive.org/web/20230205141811/http://anyclip.com/) )但它是免费的，合法的，好看的，因为它是用户友好的。

而在我眼里，那是一份[的大礼](https://web.archive.org/web/20230205141811/http://movieclips.com/watch/its_a_wonderful_life_1946/a_great_gift/)。
