# 每周观察综述 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/30/weekly-watch-round-up-2/>

这是很难拉一个良好的未来潜水手表设计，因为更传统的外观往往更永恒。然而，Clerc 在收集 Hydroscaph 方面做得很好。点击此处查看对[Clerc hydro caph 限量版计时手表](https://web.archive.org/web/20230203172539/http://www.ablogtoread.com/clerc-hydroscaph-limited-edition-chronograph-watch-review/)的评论。

对于另一个(也是更昂贵的)未来主义潜水者，我们认为[宝珀 X Fathoms 手表](https://web.archive.org/web/20230203172539/http://www.ablogtoread.com/blancpain-x-fathoms-watch/)有一个合适的模糊但听起来有趣的名字。它有一个机械深度计和其他一些简洁的功能。我们提到过它在手腕上大约 55 毫米宽吗？

那些看起来干净的[百年灵手表会喜欢越洋自动](https://web.archive.org/web/20230203172539/http://www.ablogtoread.com/breitling-transocean-watch-hands-on/)。它有一个复古的外观，但现代的尺寸，易于阅读，设计非常好。

个性化的终极体现是获得一枚在机械机芯表面刻有定制艺术的腕表。像宝珀这样的高端品牌就提供这样的服务。为了展示他们的作品，他们正在为全球某些城市制作特别版的[宝珀·维勒特·格兰德装饰腕表](https://web.archive.org/web/20230203172539/http://www.ablogtoread.com/blancpain-villeret-grande-decoration-watch/)。

新的英国品牌施菲尔德发布首款纪念灯塔的腕表。施菲尔德 Signalman GMT 腕表[采用现代、简约和阳刚的设计，是极具吸引力的首发产品。](https://web.archive.org/web/20230203172539/http://www.ablogtoread.com/schofield-signalman-gmt-watches/)

对许多人来说，开放式表盘和镂空机芯的机械表是钟表的终极魅力。明年最好的简单高端骨骼化手表可能是记录保持者(薄) [Piaget Altiplano Skeleton 超薄手表](https://web.archive.org/web/20230203172539/http://www.ablogtoread.com/piaget-altiplano-skeleton-ultra-thin-automatic-watch/)。
