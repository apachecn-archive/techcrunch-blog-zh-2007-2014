脸书否认他们知道美国国家安全局在FISA之外收集数据

原文：https://web.archive.org/web/https://techcrunch.com/2014/03/20/facebook-disputes-claims-that-they-were-aware-of-nsa-data-collection-outside-of-fisa-orders/

《卫报》昨天头版头条报道了一则新闻,称科技公司对国家安全局间谍活动的了解并不完全真实。新闻媒体迅速报道了美国国家安全局总法律顾问Rajesh De的指控，称科技公司“完全了解”对其用户的有争议的监控。

从去年夏天NSA丑闻开始，科技公司就强烈否认NSA直接获取了他们的数据。他们还否认知道任何明显允许NSA强行要求用户数据的项目，即PRISM。此外，他们还公开游说美国政府允许他们披露由外国情报监视法院(FISC)授权的NSA监控的用户数量。

起诉政府提高透明度能力的公司最终赢得了这场官司，并在此后披露了更多有关政府数据要求的信息(T4，仍在严格的限制范围内)。

因此，脸书迅速否认了新的指控。在一份声明中，该公司告诉TechCrunch:

“在新闻报道之前，我们从未听说过‘棱镜’或任何互联网公司自愿或不自愿地让政府直接访问服务器或以任何方式促进大规模收集用户数据的计划。与此同时，我们从未说过我们不知道我们根据FISA承担的义务，这正是今天听证会的主题。事实上，我们一直在为美国政府根据该法案提出的与国家安全相关的合法请求争取更大的透明度。暗示我们在误导公众是令人沮丧的，也是不真实的。”

我们收到这份声明后不久,《卫报》对他们的故事进行了重大“修正”。

“这篇文章于2014年3月20日进行了修改，删除了原文中的陈述，即Rajesh De的证词与技术公司关于他们对NSA数据收集的知识的否认相矛盾。它也被更新，以澄清这些公司质疑围绕第702条命令的保密性。还做了其他一些小的澄清。”

第702条是指一项允许一些更有争议的情报机构监控项目的法律。

当被问及《卫报》是否仍然坚持他们对这个故事的最初解释时，发言人根纳迪·科尔克尔回复说，“这篇文章被修改以澄清和纠正我们的报道，符合《卫报》的政策和实践。”

《卫报》在原文中写道:

美国国家安全局(National Security Agency)的高级律师周三明确表示，美国科技公司完全清楚这家监控机构广泛收集数据的行为，这与这些公司一个月来愤怒的否认相矛盾。

…

美国国家安全局周三的评论与科技公司关于公司对棱镜的了解相矛盾，这有可能加深与国家安全局依赖的公司的紧张关系，国家情报总监的总法律顾问罗伯特·利特(Robert Litt)告诉委员会，这是“我们拥有的最有价值的收集工具之一。

文章的修改缩小了德的说法，反映出报道一个被秘密笼罩的故事的难度——这种秘密让公民和科技公司都感到沮丧。奥巴马总统已经提议对情报机构监控进行几项改革，但是任何重大的透明度改革都必须等到今年晚些时候国会开始讨论这个问题。
