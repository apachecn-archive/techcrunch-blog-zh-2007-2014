# Weebly 推出迎合设计师的白标平台

从一开始， [Weebly](https://web.archive.org/web/20230205022903/http://designers.weebly.com/) 就致力于使用所见即所得的拖放式界面，尽可能轻松地构建自己的定制网站。它在这方面做得很好——首席执行官大卫·鲁森科说，根据最近 [Netcraft](https://web.archive.org/web/20230205022903/http://news.netcraft.com/archives/2011/07/08/july-2011-web-server-survey.html) 的调查，Weebly 为网络上大约 2%的网站供电。

但是，尽管 Weebly 相对容易使用，仍然有很多人不知道如何建立自己的网站，他们宁愿花钱请人帮他们建网站。这就是 Weebly 推出新的[设计师平台](https://web.archive.org/web/20230205022903/http://designers.weebly.com/)的原因。它本质上是一个服务的白标版本，为设计者提供了一种快速构建客户网站的方法，然后提供一个 CMS，一旦项目完成，客户就可以实际使用。

它是这样工作的:首先，设计师创建他们自己的帐户，上传自定义徽标，并为他们的任何同事创建额外的帐户，这些同事也可以访问客户页面。然后，设计师可以着手构建他们客户的页面——他们可以像往常一样使用 Weebly 的标准拖放界面，或者如果他们想给页面一个独特的外观，他们也可以包含自定义代码。

一旦网站接近完成，设计师就可以让客户访问(客户甚至不会意识到网站正在 Weebly 上运行，因为它将带有设计师自己的标志)。根据设计师如何配置网站，客户将能够进行简单的调整(调整颜色，字体等。)使用 Weebly 接口。然后，一旦项目完成，客户将能够使用 Weebly 的 CMS 添加额外的内容、页面、表单和其他功能。

定价方案起价为每月 7.95 美元，其中包括设计师网站的托管费用(你使用的页面和存储空间越多，价格就越高)。设计师可以自由地向他们的客户收取更多的费用。
