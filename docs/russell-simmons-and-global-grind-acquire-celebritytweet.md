# 拉赛尔·西蒙斯和环球研磨收购 CelebrityTweet

![](img/5b1c892ea31cfc10705927e8743e2d4c.png)

一个名人的推特整合！大亨拉赛尔·西蒙斯的嘻哈媒体平台 [Global Grind](https://web.archive.org/web/20221209130814/http://globalgrind.com/) 正在[收购](https://web.archive.org/web/20221209130814/http://www.prnewswire.com/news-releases/russell-simmons--globalgrindcom-announce-the-acquisition-of-celebritytweetcom-78763532.html) [CelebrityTweet，](https://web.archive.org/web/20221209130814/http://www.celebritytweet.com/)在其平台上收集和聚合名人的推文。收购条款没有公布。

CelebrityTweet 于 2008 年 10 月推出，利用越来越多的名人通过 Twitter 与粉丝群联系。Global Grind 将使用 CelebrityTweet 建立自己的平台和数据库，让用户可以访问他们喜爱的名人的推文。

Global Grind 还希望将该平台作为名人与粉丝联系和互动的一种方式。随着越来越多的名人涌向 Twitter，像 CelebrityTweet 这样的平台可以帮助粉丝找到他们的推文。SoBe Lifewater 还将赞助 CelebrityTweet.com 及其名人博客。几个月前，Global Grind 筹集了 260 万美元的新资金。
