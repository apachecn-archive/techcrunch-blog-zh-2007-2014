# 三星不明飞行物手机:当心热射线

[![samufo1.jpg](img/13acbbf838901d91e9c7802562cfaf02.png)](https://web.archive.org/web/20221003024823/https://beta.techcrunch.com/wp-content/uploads/2007/08/samufo1.jpg "samufo1.jpg")

还记得在 90 年代，当每一个两位数的电视网络放“他们见过不明飞行物！”在广播里？是的，他们完全是愚蠢的。当三星想出其 SCH-W300、SPH-W3000 和 SGH-U700 手机的名字时，我非常怀疑它有这些程序和相关的不明飞行物亚文化。他们是众所周知的——令人惊讶！——作为不明飞行物。我真的不知道他们为什么这么叫。

[![samufo2.jpg](img/88ec3ffe6d24af936c3a6c933be71362.png)](https://web.archive.org/web/20221003024823/https://beta.techcrunch.com/wp-content/uploads/2007/08/samufo2.jpg "samufo2.jpg")

实际上，请注意——不明飞行物，很明显，代表难以想象的高速哦。哦，现在我明白了…

超地面速度归功于手机的 HSDPA 兼容性。(我想这让苹果手机太人性化了。)除了我听说已经过时的双速网络速度之外，你还会发现普通的:蓝牙、microSD、yada yada。没什么可疯狂的。因为是韩国，所以就目前而言，您还可以获得内置的 DMB 电视接收功能。

恒星会对齐吗？在美国我们会看到不明飞行物吗？不要问我；我只是很高兴我们已经完成了整个“zomg，麦田怪圈再次-巫术！”阶段。
