# 时代华纳有线电视以 2.3 亿美元收购企业托管和云服务公司 NaviSite

时代华纳有线电视刚刚宣布，它已经以每股 5.50 美元的现金，或 2.3 亿美元，收购了企业级托管、托管应用程序、消息和云服务提供商【NaviSite】。该交易预计将于 2011 年第二季度完成。

NaviSite 为公司提供企业托管能力、应用管理和云服务。这些包括 IBM Lotus 和 Domino 的电子邮件和协作主机；Microsoft Exchange 和 SharePoint 软件质量保证和测试、服务器和存储管理、数据保护等等。该公司在 2010 财年获得了 1.26 亿美元的收入，约有 1400 名客户使用其服务。

该公司目前在美国和英国运营着十个数据中心，包括两个 SAS 70 Type II 认证数据中心，以及位于印度古尔冈和马萨诸塞州安多弗的网络运营中心。

根据新闻稿，NaviSite 的服务将提供给时代华纳有线电视的商业服务业务客户。时代华纳首席执行官 Glenn Britt 在一份声明中表示:“NaviSite 为我们提供了一个成功的托管服务业务和一个新的、创新的托管云平台，代表着重要的新增长机会。我们希望在 NaviSite 成功的企业级产品及其运营能力、基础架构和专业知识的基础上，更快地为中小型企业创建强大的托管服务产品。”

NaviSite [最近在 12 月将其达拉斯主机托管业务出售给了 Cologix。](https://web.archive.org/web/20230203001051/http://www.masshightech.com/stories/2010/12/20/daily12-NaviSite-sells-Dallas-colocation-site-for-1275M.html)
