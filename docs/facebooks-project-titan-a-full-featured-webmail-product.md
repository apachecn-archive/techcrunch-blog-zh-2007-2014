# 脸书的“泰坦计划”:一个全功能的网络邮件产品

据了解该产品的消息人士称，脸书正在彻底改写他们的信息产品，并准备推出一款功能齐全的网络邮件产品来取而代之。内部称之为泰坦计划。或者，非正式地，也许是过分热情地，Gmail 杀手。

脸书消息多年来一直是用户生存的祸根。我第一次公开抱怨是在 2008 年，当时我说迫切需要变革。最大的问题是简单地删除旧邮件。这需要很长时间，以至于我的收件箱里有成千上万的未读和已读但未删除的邮件。

但是脸书信息也只是间接链接到电子邮件，这仍然是人们不在脸书时交换数字信息的标准方式。

脸书偶尔会涉足消息传递的改进，比如增加搜索消息的[功能。但在很大程度上，它保持不变。而且不是很有用。](https://web.archive.org/web/20230208105151/https://techcrunch.com/2008/05/08/facebook-email-gets-better-with-search/)

甚至 MySpace 也在 2008 年从他们老化的消息平台转移到真正的网络邮件服务(尽管它缺乏 POP 或 IMAP 支持)。

但现在，脸书正让自己重新回到游戏中。如果我们听到的细节是准确的，泰坦计划，或者它发布时的名称，可能是人们蜂拥而至的那种产品。

首先，我们的理解是，将有全面的 POP/IMAP 支持，这意味着用户可以访问帐户，而不是通过脸书本身。你的电子邮件账户名将成为你的[虚拟网址](https://web.archive.org/web/20230208105151/https://techcrunch.com/2009/06/09/you-have-three-days-to-pick-your-facebook-vanity-url/)——vanityurl@facebook.com。

电子邮件是关于身份的。通过脸书连线，脸书在身份游戏中领先于其他所有人。脸书[表示](https://web.archive.org/web/20230208105151/http://www.facebook.com/press/info.php?statistics)每月有超过 6000 万人通过脸书互联登录 80000 个第三方网站。

将一个真正的网络邮件产品放在那些虚荣心的网址和脸书连接之上，这甚至会让谷歌不寒而栗。Gmail 杀手？我不这么认为。但仍是一个强劲的产品动向。
