# 微软回应谷歌回应微软回应

如果你认为微软和谷歌这两家大型上市公司会在过去几天非常公开的交锋之后迅速而平静地退居幕后继续战斗，那你就错了。嗯，也许他们也在幕后争斗，但是公开的斗气也在继续。对我们有好处！

为了涵盖故事的双方(当然不是因为这个故事情节充满了伟大的素材，产生了巨大的兴趣和浏览量)，这里是微软的最新反驳。微软的沟通主管弗兰克·肖再次回击谷歌的法律主管大卫·德拉蒙德，因为他昨天更新了他在[的博客文章](https://web.archive.org/web/20230130121815/http://googleblog.blogspot.com/2011/08/when-patents-attack-android.html)。肖又在推特上这么做了。

对于那些不知道我在说什么的人，提醒一下:

肖最有趣的[推文](https://web.archive.org/web/20230130121815/https://twitter.com/#!/fxshaw/status/99217585115119616)回应如下:

> 为什么？因为他们想买一些可以用来对抗其他人的东西。

这是 Shaw 声称谷歌拒绝了 Novell 的专利合作投标，以便他们可以找到其他专利来对抗其他人。谷歌在专利问题上的立场一直是，他们只是出于防御目的才需要它们，所以你知道这种攻击肯定让他们很恼火。

希望这意味着另一个回应！
