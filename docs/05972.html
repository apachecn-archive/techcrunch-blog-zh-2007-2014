OpenFeint的游戏频道使用Groupon模式销售其手机游戏

由Groupon开创的团购模式已经进入了移动游戏和应用领域。几个月前，Y Combinator支持的 TapZilla 推出了，将当日交易模式应用于付费移动应用。今天，移动社交游戏平台 OpenFeint 加入了Game Channel的发布，这是一个移动游戏应用的团购网站。

Game Channel允许开发人员在OpenFeint平台上开发游戏，通过每日和每周的游戏交易来推广他们的游戏。具体来说，游戏频道的大甩卖功能使用团购模式来推动游戏开发商的下载量。

每周，Fire Sale都会推出一款面向OpenFeint 5000万用户的高端游戏，通常定价为1.99美元至9.99美元。当足够多的游戏频道用户投票表示想要这款游戏时，游戏的价格就会下降。OpenFeint表示，它将向投票者发送推送通知，通知他们游戏限时发售。

首先发售的是《大白鲨》(Jaws)，这是一款官方游戏，改编自Bytemark Games的夏季大片。游戏频道还将包括免费游戏，这使得一个游戏免费一天。

团购热潮延伸到应用和游戏领域只是时间问题。但我很好奇，当移动应用生态系统中有这么多免费游戏时，这种模式是否能够推动付费游戏的销售。我想我们会找到答案的。
