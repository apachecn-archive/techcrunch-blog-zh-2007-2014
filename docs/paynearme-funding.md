# PayNearMe 为现金支付产品 TechCrunch 从 Khosla 等人那里筹集了 1600 万美元

> 原文：<https://web.archive.org/web/http://techcrunch.com/2010/11/16/paynearme-funding/>

![](img/6c939d671020e249d8cb486d8727946f.png)[【PayNearMe】](https://web.archive.org/web/20230202215604/http://www.paynearme.com/)公司的替代支付产品 [Kwedit，](https://web.archive.org/web/20230202215604/http://www.crunchbase.com/product/kwedit-promise)刚刚筹集了[1600 万美元](https://web.archive.org/web/20230202215604/http://www.prnewswire.com/news-releases/paynearme-secures-16-million-in-funding-to-expand-its-cash-payment-network-108400399.html)的资金，由 [Khosla Ventures](https://web.archive.org/web/20230202215604/http://www.crunchbase.com/financial-organization/khosla-ventures) 牵头，新投资者 [August Capital](https://web.archive.org/web/20230202215604/http://www.crunchbase.com/financial-organization/august-capital) 和当前投资者 [True Ventures](https://web.archive.org/web/20230202215604/http://www.crunchbase.com/financial-organization/true-ventures) 和 [Maveron](https://web.archive.org/web/20230202215604/http://www.crunchbase.com/financial-organization/maveron) 也参与了这轮投资。这使得该公司的资金总额达到[2330 万美元](https://web.archive.org/web/20230202215604/http://www.crunchbase.com/company/paynearme)。在宣布融资的同时，博库首席执行官马克·布里托将加入公司董事会。

PayNearMe 正在利用为无银行账户者提供的服务浪潮，这一群体包括美国的 6000 万人。“无银行账户者”是指没有传统银行账户或没有资格使用信用卡的消费者。PayNearMe 允许没有或不想使用信用卡或借记卡的人在美国大陆超过[6000 家 7-Eleven 商店](https://web.archive.org/web/20230202215604/https://techcrunch.com/2010/09/14/paynearme-launches-cash-payment-system-signs-up-amazon-facebook/)用现金购买产品

在参与的电子商务或商家网站上，消费者可以使用 PayNearMe 选项来支付电子商务购物、电话订单、贷款还款、转账等费用。您只需在 PayNearMe 下订单，并打印出给定的收据。然后你拿着收据去 7-11 便利店，他们扫描后你用现金支付。一旦您付款，您与零售商或商家的订单将被履行。

今天，PayNearMe 推出了一种[无纸化移动支付方式](https://web.archive.org/web/20230202215604/http://www.prnewswire.com/news-releases/paynearme-expands-cash-payment-network-to-enable-mobile-payments-108400394.html)；用 PayNearMe 品牌的卡。因此，在参与的商家网站上，如果用户想使用支付选项，他们需要提供自己的手机号码。然后，他们可以从当地的 7-Eleven 商店获得一张免费的 PayNearMe 卡，并用卡的唯一号码回复 PayNearMe 的短信。并且他们将卡和用于支付的现金交给 7-Eleven 商店员工来处理交易。与所有 PayNearMe 交易一样，收款人会立即收到付款通知。

该公司于 8 月份开始在 T2 测试其支付脸书积分的形式，但现在正在整合 Amazon.com、Progreso Financiero、MOL AccessPortal、m-Via、Lexicon Marketing、LLC、Adknowledge 的 Super Rewards、Money to Go 和 SteelSeries。例如，你可以使用 Amazon.com 的 PayNearMe 在线购买礼品卡。

作为背景，Kwedit 在今年早些时候推出了另一款支付产品[，引发了很多争议。据 PayNearMe 创始人兼首席执行官 Danny Shader 表示，PayNearMe 已经能够与一些知名商家签约，并且正在稳步发展(他拒绝透露有多少消费者正在使用该支付平台)。](https://web.archive.org/web/20230202215604/https://techcrunch.com/2010/02/03/kwedit-launches-the-first-completely-unreliable-payment-network/)

Shader 表示，他计划使用新的资金来扩大其零售商支付地点网络(即 7-Eleven)以及增加接受 PayNearMe 的新商户和公司。这笔资金也将用于产品开发。

虽然去一趟商店为你在网上订购的商品付款的过程可能看起来很复杂，但对于那些无法获得信贷的人来说，这是一个可行的支付选择。如果 PayNearMe 继续与知名、受欢迎的商家和支付地点合作，它的产品可能会起飞。绿点公司通过为欠银行贷款的人提供预付费信用卡业务，成为了一项可行的业务，最近申请了 20 亿美元的首次公开募股。
