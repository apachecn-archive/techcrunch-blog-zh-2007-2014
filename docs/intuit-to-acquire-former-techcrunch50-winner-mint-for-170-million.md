# Intuit 以 1.7 亿美元收购(前 TechCrunch50 赢家)Mint

据知情人士证实，Intuit 将以大约 1.7 亿美元的价格收购免费在线个人理财服务公司 [Mint](https://web.archive.org/web/20230405230816/http://www.mint.com/) 。[硅巷知情人](https://web.archive.org/web/20230405230816/http://www.businessinsider.com/tipster-intuit-buying-mintcom-2009-9)首先报道了这方面的一个传闻。这笔交易将在未来几天内宣布。

**更新**:首席执行官 Aaron Patzer 刚刚在 TechCrunch50 的舞台上确认了这笔交易，并写了一篇客座博文，描述了[TechCrunch 50 的价值](https://web.archive.org/web/20230405230816/https://techcrunch.com/2009/09/14/the-value-of-techcrunch50-mint-acquired-by-intuit-for-170m-two-years-after-winning-tc40/)，其中包含更多细节。

这对两年前在 TechCrunch50 首次推出的 Mint 来说是一个极好的退出。Mint [在那次活动中获得了最高奖项](https://web.archive.org/web/20230405230816/https://techcrunch.com/2007/09/18/mint-wins-techcrunch40-50000-award/),并且从那以后发展迅速。他们最后一轮融资对公司的估值为 1.4 亿美元。

Mint 总共在三轮风险投资中筹集了 3200 万美元。

今年早些时候，Mint 和 Intuit 就 Mint 广告声称每天获得 3000 名新用户，并在几个月内从 60 万用户跃升至 85 万用户发生了一场幽默的冲突。Intuit 给 Mint 写了一封信，要求对这一明显不可思议的壮举做出解释，我们获得了这封信，并在此发表。

不过，我们只有一个问题要问创始人兼首席执行官[亚伦·帕兹](https://web.archive.org/web/20230405230816/http://crunchbase.com/person/aaron-patzer)。我们能要回我们的 5 万美元大奖吗？看来你真的不再需要它了。:-)
