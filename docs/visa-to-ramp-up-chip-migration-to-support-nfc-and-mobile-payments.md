# Visa 将加快芯片迁移以支持 NFC 和移动支付

我们知道 Visa 对移动和数字支付越来越重视。该计划的一部分包括采用 NFC 和其他芯片技术，实现移动交易。今天，该公司在[宣布](https://web.archive.org/web/20230204222320/http://www.prnewswire.com/news-releases/visa-announces-plans-to-accelerate-chip-migration-and-adoption-of-mobile-payments-127297418.html)它将在美国加速向 EMV 接触式(那些有时在你的信用卡中可以找到的芯片)和非接触式芯片技术的迁移。

Visa 表示，采用这些芯片技术将有助于零售商和支付公司为基于 NFC 的移动支付的到来做好准备，建立必要的基础设施来接受和处理支持销售点签名或 PIN 的芯片交易。

Visa 全球产品负责人吉姆·麦卡锡(Jim McCarthy)在一份新闻稿中表示:“随着 NFC 移动支付和其他基于芯片的新兴技术在未来几年即将起飞，我们今天正在采取措施创建一个商业框架，以支持增长机会，并为支付链的所有参与者创造价值。”

正如 Visa 移动产品全球负责人 Bill Gajda 上周告诉我的那样，该公司看好 NFC，但意识到该技术的大规模采用最早可能要到今年晚些时候才会发生。

当然，在全球范围内，Visa 将继续支持其一系列持卡人验证方法，包括低价值、低风险交易的签名、PIN 和无签名。但从长远来看，Visa 预计，随着芯片技术实现新的动态持卡人验证形式，签名和 PIN 等验证方法的使用将会减少或完全取消。

Visa 还表示，它将要求美国收单处理器和子处理器服务提供商能够在不迟于 2013 年 4 月 1 日支持商户接受芯片交易。

虽然许多[认为](https://web.archive.org/web/20230204222320/http://blogs.forbes.com/tomiogeron/2011/07/20/ebays-paypal-payments-go-offline-preps-point-of-sale-product/)NFC 在扩大规模时会有一场艰苦的战斗，但 Visa 正在推进其战略是一个积极的迹象。万事达卡还与谷歌在 NFC 支付方面合作，即将推出谷歌钱包[。](https://web.archive.org/web/20230204222320/https://techcrunch.com/2011/05/26/google-wallet-offers/)
