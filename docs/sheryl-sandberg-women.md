# 雪莉·桑德伯格:“除非女人和男人一样有抱负，否则她们不会有同样的成就”

此时，技术领域的女性，尤其是那些身居要职的女性，可能已经厌倦了听到“技术领域的女性”这个词。但是，今晚，在与查理·罗斯的一次谈话中，脸书·首席运营官·雪莉·桑德伯格被问到了同样的问题:如果在今天的科技行业，女性能够找到她作为世界上最知名的公司之一的高管希望她们能够找到的机会水平。

桑德伯格认为，女性在高等教育和管理方面取得了令人难以置信的进步，在大学和研究生学位以及领导职位方面获得了更大的份额(如果不是领先于男性的话)。但是玻璃天花板拒绝完全破裂。桑德伯格告诉罗斯，“在过去的十年里，女性在高层已经止步不前了”。

她继续说道，在美国企业界，女性只占董事会席位和首席执行官等高层职位的 15%至 16%——在过去十年左右的时间里，女性在这些高层职位中所占的比例确实没有太大变化。

> Rose: 让我谈谈两件事，一是你，赋予妇女权力，二是对你所代表的人的感觉。这是一个你所在的地方——你之所以突出，是因为你和玛丽莎(梅耶尔)以及其他一些人，你知道——科技是一个女性可以找到你希望她们找到的那种机会的地方吗？它需要像雪莉·桑德伯格这样的人能够提供的指导和其他各种东西吗？
> 
> **桑德伯格:**听着，我认为女性在经济和国家中的地位是个大问题。这是我非常关心的事情，马克也非常关心，也帮助了我。你知道，我们基本上有一个停滞不前的女性革命。你知道，1981 年，女性在这个国家的大学毕业生中占了 50 %,然后稳步前进，更多的大学学位，更多的研究生学位，更多的经理职位。结束——我们仍在进步。在过去的十年里，女性在高层一直止步不前。在美国公司中，女性拥有 15%到 16%的董事会席位和首席执行官(按发音拼写)等高级职位，这一比例十年来没有变化。
> 
> **玫瑰:**为什么？
> 
> 桑德伯格:哦，这可能比我们的时间还长。有很多原因，但我真的认为我们需要更多的女性投身于她们的事业，并真正致力于留在职场。我觉得成绩差距是很多东西造成的。这是由制度障碍和各种各样的东西造成的。但是也有很大的抱负差距。如果你调查一下当今美国大学里的男性和女性，你会发现男性比女性更有野心。除非女性和男性一样雄心勃勃，否则她们不会取得和男性一样多的成就…
