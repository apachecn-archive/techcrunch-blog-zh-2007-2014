# 事物如何变化 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2014/02/19/how-things-change/?utm_source=dlvr.it&utm_medium=twitter>

四年前，布莱恩·阿克顿正在找工作。

他在雅虎的工作已经结束了，在那里他担任了 11 年多的工程职位。

他在和招聘人员联系:

他会见了推特。推特说不:

他会见了脸书。脸书说没有:

因此，他和雅虎的同事简·库姆开始做自己的事。

今天，他们以 190 亿美元的价格把那东西卖给了脸书。

生活是疯狂的。
