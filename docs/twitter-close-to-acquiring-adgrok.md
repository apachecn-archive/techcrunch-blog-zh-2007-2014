# Twitter 接近收购 AdGrok

我们从多个渠道获悉，Twitter 正在就收购 Y Combinator 支持的关键词竞价平台 [AdGrok](https://web.archive.org/web/20230205012340/http://www.adgrok.com/) 进行谈判，交易金额不到 1000 万美元。目前还不清楚他们在交易过程中的确切位置，也不清楚这是一项技术收购还是一项收购。

AdGrok 本身[自动化了在 Google AdWords 上竞价上下文关键词](https://web.archive.org/web/20230205012340/https://techcrunch.com/2011/03/07/adgrok-raises-470k-to-be-the-turbotax-for-search-engine-marketing/)的过程；也许 Twitter 可以在这方面找到一些用途，用于自己推广的趋势/推文？

Twitter 似乎处于收购模式，最近[在 pro-client Tweetdeck 上投入了 4000 万美元](https://web.archive.org/web/20230205012340/https://techcrunch.com/2011/05/25/twitter-confirms-tweetdeck-acquisition-its-all-about-the-power-users/)。

有趣的是:Twitter 公司发展主管[杰西卡·韦里利](https://web.archive.org/web/20230205012340/http://www.crunchbase.com/person/jessica-verrilli) [在 Twitter](https://web.archive.org/web/20230205012340/http://twitter.com/#!/jess/following/people) 上关注的最后一个人是 AdGrok 的联合创始人 Argyris Zymnis。联合创始人安东尼奥·加西亚·马丁内斯还没有回复我的电话。
