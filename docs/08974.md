# Zynga 推出最新的脸书游戏:冒险世界

Zynga 刚刚在脸书发布了其最新社交游戏《T2》的预告片。据该游戏的脸书·佩奇介绍，该游戏的口号是“组队，探索失落的世界，发掘 Zynga 最新游戏《冒险世界》中的冒险经历”。"

从视频来看，这个游戏似乎是围绕着许多不同的谜题、地图和土地开发的。与 Zynga 的其他脸书游戏类似，你的朋友也是体验的一部分。基本上，认为印第安琼斯符合 FarmVille。

Zynga 最近一直在大量生产游戏，发布了 [The Pioneer Trail、](https://web.archive.org/web/20230204214446/https://techcrunch.com/2011/08/12/after-frontierville-zynga-channels-the-wild-west-again-for-facebook-game-the-pioneer-trail/)Empires&alliances、[一个中国版的 CityVille、](https://web.archive.org/web/20230204214446/https://techcrunch.com/2011/07/25/zynga-partners-with-tencent-to-launch-localized-chinese-version-of-cityville/)一个 [CityVille iOS 应用程序](https://web.archive.org/web/20230204214446/https://techcrunch.com/2011/06/15/zynga-wants-you-to-build-your-neighborhood-with-cityville-hometown-ios-app/)、 [Words With Friends on、](https://web.archive.org/web/20230204214446/https://techcrunch.com/2011/08/01/zynga-brings-scrabble-like-words-with-friends-to-the-web-with-new-facebook-game/)和[hang With Friends。](https://web.archive.org/web/20230204214446/https://techcrunch.com/2011/06/06/zynga-turns-hangman-into-a-social-ios-game-with-the-debut-of-hanging-with-friends/)

从视频中还不清楚游戏何时发布，但我们会随时为您更新《冒险世界》的发布。
