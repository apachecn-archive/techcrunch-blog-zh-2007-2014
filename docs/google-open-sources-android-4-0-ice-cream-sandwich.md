# 谷歌开源安卓 4.0，冰淇淋三明治

Android 最新版本的代码——4.0，也被称为冰激凌三明治——现在已经无人知晓。

根据 Android 开源项目的软件工程师让·巴普蒂斯特·m·“JBQ”·奎鲁的帖子，代码仍在上传过程中，开发者被建议等到完全完成后再开始下载。但是很快就能买到了。

出于几个原因，这是一件大事。

首先，Android 4.0 包括了对 Gingerbread 和 Honeycomb 的一些重大改进，这两个版本的 Android 都在当前的 Android 设备上运行。其中包括新的 API 调用、性能改进等等。

但更重要的是:这是谷歌首次开源针对平板电脑和手机优化的安卓版本。现在，显然运行 Android 3.x Honeycomb 的设备从今年年初就已经出现了。但是谷歌选择不发布蜂巢的源代码。原因？它很快就被整合在一起，谷歌不得不采取一些捷径来推出它(他们也担心第三方会试图将 Honeycomb 移植回手机，而这并不适合手机)。

集成电路的开源将允许不直接与谷歌合作的制造商生产他们自己的平板电脑(有一些低成本的平板设备附带姜饼，只是因为他们不能使用蜂巢)。这也意味着像 CyanogenMod 这样的定制 ROM 开发者可以调整代码，将其移植到旧设备上，等等。

其他一些有趣的注意事项:正在开源的 ICS 版本是 4.0.1，开发者可以下载 Galaxy Nexus 的图像，其构建目标是 full_maguro。

哦，如果你*真的*想看看蜂巢的来源，它也在那里(因为它是冰淇淋三明治的祖先)。但是谷歌不鼓励任何人实际使用它。
