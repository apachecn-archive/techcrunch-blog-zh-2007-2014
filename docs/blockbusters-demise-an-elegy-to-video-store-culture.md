# 百视达的消亡；音像店文化的挽歌

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/11/06/blockbusters-demise-an-elegy-to-video-store-culture/>

在坚持生活多年后，Dish Network 宣布将于明年初关闭剩余的 300 家大型视频实体网站，这标志着家庭视频租赁时代的又一次丧钟。这个曾经强大的视频巨头在 2004 年的巅峰时期拥有超过 9000 个位置，这个数字在过去十年中[大幅下降](https://web.archive.org/web/20230306013416/http://www.nytimes.com/2011/04/09/business/09blockbuster.html?_r=0)。

90 年代末，我十几岁时在新泽西为百视达工作。这份工作有起有落，有些部分糟透了，但总体来说，这是我人生中的一段快乐时光。随着 Redbox 的出现，一台不比 ATM 大多少的机器提供了一项服务，而在我的有生之年，这项服务需要一栋大楼、一份工资单、一个管理层级和 24 名员工来完成。尽管明显缺乏逛到一台机器前按几个按钮做出选择并付款的“经验”，商店模式还是没有机会。

虽然我们中的大多数人可能已经有一段时间没有涉足实体音像店了，但我们中的一些人在一定的年龄无疑会有许多与现在几乎古怪的外出“租录像带”行为相关的记忆。

我小时候，去音像店是一件大事。我们会兴奋地挤进车里，一路上喋喋不休地谈论我们想租哪部电影或哪款游戏。作为拥有在家重新观看我们在电影院看过的电影的技术的第一代人的一部分，我们充分利用了这个勇敢的家庭娱乐新世界。我的父母成长在一个充斥着经典电影的电影时代；詹姆斯·邦德电影系列的开端，《毕业生》，《人猿星球》和无数其他电影。一旦他们在电影院看到它们，再次看到它们的唯一方法是在当时可用的少数几个电视频道之一的一周电影中。

我们现在生活的“点播”家庭娱乐世界的根源始于那些被称为 VHS 磁带的坚固的黑色矩形。

音像店的一切都很新奇。不同的会员卡、它们展示盒子的方式以及磁带上的安全机制都是独一无二的，似乎有着无穷无尽的排列组合。“哦，这个地方切掉了展示盒书脊的一边，把它们放在一个透明的盒子里”；“哦，这一个有盒子，你必须在边上捏一下才能松开带子”；“哦——这个地方横着展示他们的磁带？!"是我和我的朋友们经常挂在嘴边的话。

每次旅行都有些不同。

租赁电子游戏一直是最大的一笔交易。我已经记不清有多少次我在租一个我选择的游戏时割草、铲车道或清理排水沟。谁知道我父母在那些交易中从我身上得到了多少大打折扣的劳动？

商店里陈列的总是几十个，不，几百个电子游戏盒。回到 NES 时代，我能得到一个新的盒装游戏的唯一方法就是在圣诞节或者生日的时候。标准的 50 美元价格对我的精灵血统来说太贵了。所以参观视频商店游戏区就像去香格里拉旅行一样。精心绘制的盒子与眼睛齐平，以全彩绘画为特色，画有龙或带剑的骑士或赛车。当然，实际的游戏图形从来没有匹配到框艺术，但这并不重要。我在音像店，打算带一只小狗回家玩。

可能除了玩具店之外，在我成长的过程中，没有一次去零售商店的旅行给我带来如此的快乐和幸福。每一次都是一次冒险，每个角落背后都潜伏着神秘。谁知道在商店的后面，在一对摇摇晃晃的老西部沙龙风格的门后面，在一个显眼的“成人专用”的位置下，住着什么不为人知的秘密签字？

由于过去十多年的家庭娱乐真正强调了“家”，大多数美国人现在都可以接触到几乎无底洞的电影、电视节目、纪录片和成人娱乐。即使作为一个公认的相当不经常购买新设备的人，我个人也有六个不同的平台可供我使用，通过它们我可以出租我的小心脏。六个平台，我没有努力去积累这么多的选择。不要误解我。不用离开我们的沙发就可以进入基于视频的娱乐世界，这真是太棒了。但是失去的是获得和观看它的体验品质。

视频“商店”的概念(就连“视频”这个词如今也只是作为一个不合时宜的口语词而存在)向如今陈旧过时的巨大鸿沟又迈进了一步。有了它，对于那些曾经从还书口偷看过，或者不得不踮着脚尖把还回来的磁带放在柜台上的孩子来说，这是一个珍藏童年记忆的地方。
