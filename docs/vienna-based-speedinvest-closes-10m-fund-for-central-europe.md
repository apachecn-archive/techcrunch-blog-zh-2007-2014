# 总部位于维也纳的 Speedinvest 为中欧关闭了 1000 万美元的基金

慢慢地，但肯定地，维也纳正在建立自己作为创业热点的声誉，最终柏林、伦敦、哥本哈根和维也纳之间可能会有一场战斗，尽管伦敦和柏林目前领先。在奥地利出生的创始人推动下，出现了许多大规模的退出(从 UCP、Jahjah、3united 和 last.fm 的退出超过 10 亿美元)。像 [Wikitude](https://web.archive.org/web/20230204162940/http://www.wikitude.com/) 、 [Runtastic](https://web.archive.org/web/20230204162940/http://www.runtastic.com/) 和 [Lookk](https://web.archive.org/web/20230204162940/http://www.lookk.com/) 这样的公司。然而，不断增长的创业群体，尤其是来自中欧和东欧国家的创业者，面临着缺乏足够投资者的问题。现在，早期超级天使基金 Speedinvest 刚刚宣布，他们已经关闭了第一个 1000 万美元的基金，目标是中东欧的早期移动和网络初创公司。
