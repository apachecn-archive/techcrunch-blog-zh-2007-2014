# 更多关于 Palm“Mini-Pre”的信息:代号 Pixie

 **更新:**这个故事发展得非常快——我们已经[获得了更多细节](https://web.archive.org/web/20230204200743/http://www.mobilecrunch.com/2009/04/30/palm-eos-is-this-the-mini-pre/)，包括名称和硬件规格。

我们得到了更多关于今年晚些时候即将推出的 Palm WebOS 设备的信息。我们昨天[透露了它存在的消息](https://web.archive.org/web/20230204200743/https://techcrunch.com/2009/04/29/forget-the-palm-pre-theyve-got-a-second-device-coming-out-this-year-were-calling-it-the-mini-pre/)。现在我们有了这个项目的内部名字:Pixie。

消息人士称，Pixie 将使用与 Palm Pre 相同的 WebOS 操作系统和软件，但尺寸更小，目标价格为 99 美元。它将在 6 月 Pre 之后几个月发布，所以这不是一款升级设备。它的目标是低端市场。

昨天我们不知道这款设备是否会有固定的物理键盘。今天我们证实了这一点。从物理上看，该设备将类似于现有的 Palm Centro 系列手机，但硬件和软件都是全新的。小精灵是先知的弟弟。

我们有相互矛盾的消息来源在谈论发射计划。一个消息来源说，这是全速前进。另一个人说，Palm 正在等着看 Pre 的表现，然后再宣布 Pixie。

今天早上《天才少年报告》发布了一张新 Palm 设备的模糊图片[(我们在上面使用过的),它符合我们独立听到的所有事实。在我们找到更值得信任的东西之前，我们会继续下去。](https://web.archive.org/web/20230204200743/http://www.boygeniusreport.com/2009/04/30/palms-next-webos-device/)
