# Opera 创始人乔恩·s·冯·泰茨赫纳因与董事会意见不合而辞职

歌剧创始人乔恩·s·冯·特茨纳已经[从公司](https://web.archive.org/web/20230204234932/http://ht.ly/5pG7Z)辞职。

在给 Opera 员工的一封电子邮件中，von Tetzchner 说:“很明显，董事会、管理层和我没有共同的价值观，我们对如何继续发展 Opera 也没有相同的意见。因此，我已经和董事会达成协议，结束我在 Opera 的工作。我觉得董事会和管理层比我更关注季度。”你可以阅读下面的完整邮件。

Von Tetzchner 在 1995 年共同创立了 Opera Software，并领导该公司直到 2010 年，他辞去了 CEO 的职务。然后，他成为了公司的全职战略顾问。

至于下一步，Opera 表示，von Tetzchner“对新项目有想法，但目前还不准备透露他的任何想法。”

尚不清楚 Opera 创始人和董事会在哪些价值观上存在分歧。独立的移动和网络浏览器[拥有数千万用户，](https://web.archive.org/web/20230204234932/https://techcrunch.com/2010/10/14/opera-in-numbers-its-pretty-huge/)并且似乎在[快速增长](https://web.archive.org/web/20230204234932/https://techcrunch.com/2011/01/20/opera-mini-had-over-85-5-million-users-in-december-up-84-percent-from-2009/)。
