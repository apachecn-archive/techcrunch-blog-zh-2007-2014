# 史蒂夫·鲍尔默说微软是平板电脑的“铁杆”

[![](img/a839f69c407f3c23e9733435512a3741.png "ballmertab")](https://web.archive.org/web/20221209050502/http://www.crunchgear.com/2010/07/12/steve-ballmer-says-microsoft-is-hardcore-about-tablet-computers/ballmertab/)

别担心，伙计们，因为[微软](https://web.archive.org/web/20221209050502/http://www.crunchgear.com/tag/microsoft/)是关于平板电脑的[。嗯，关于基于 Windows 7 的平板电脑的“硬核”。史蒂夫·鲍尔默在西雅图召开的全球合作伙伴大会上如是说。微软表示，它在这方面的一大优势是，it 消费者在购买基于 Windows 的平板电脑时将有多种选择。当然，这与另一家公司的平板电脑产品](https://web.archive.org/web/20221209050502/http://www.techflash.com/seattle/2010/07/ballmer_microsoft_hardcore_about_slates_running_windows_7.html)[形成了对比，该产品有任何颜色，只要是黑色的。你知道我的意思。](https://web.archive.org/web/20221209050502/http://www.crunchgear.com/tag/ipad/)

鲍尔默重复了比尔·盖茨几周前说过的话，微软想制造一款能与你所有的多媒体一起工作的平板电脑，是的，但它也适合在会议室使用。这与苹果的做法截然相反，但我认为该公司必须在某些方面与众不同。

鲍尔默说道:

> 我们希望为您提供一款出色的设备，一款面向消费者的设备，同时也是一款适合当今企业 IT 解决方案并易于管理的设备……这对我们来说是一个非常重要的领域。这当然是一个领域，我该怎么说呢？—我们感受到前所未有的能量、活力和推动力，去创新、去努力、去竞争。

不用说，要与一款在三个月内售出 300 万台的设备竞争是非常困难的。

微软在拓展业务方面并没有太好的记录。[同族](https://web.archive.org/web/20221209050502/http://www.mobilecrunch.com/tag/kin/)怎么了？我们真的需要微软 Office 版吗？Zune HD 实际上可能是一款很棒的硬件，但你知道有多少人拥有它？Devin 和我有一个，所以是两个人。

雷德蒙的有趣时代即将到来。
