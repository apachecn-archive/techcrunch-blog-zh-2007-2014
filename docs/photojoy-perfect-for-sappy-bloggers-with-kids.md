# PhotoJoy:非常适合有孩子的多愁善感的博客作者

有了孩子显然会影响我对应用程序的评估……没有什么比 [IncrediMail](https://web.archive.org/web/20221006201749/http://www.incredimail.com/) 的一款名为 [PhotoJoy](https://web.archive.org/web/20221006201749/http://www.photojoy.com/) 的新产品更能证明这一点了，如果我不是两个孩子的父亲，我绝对不会考虑这个应用程序。

PhotoJoy 是一个可免费下载的应用程序，它使用您的照片制作整洁的桌面小工具、3D 照片屏保和壁纸拼贴画。您可以使用电脑上的照片，也可以使用上传到 Flickr 帐户的照片。PhotoJoy 还可以使用您的 Flickr 流来保持您的照片新鲜。值得注意的是，它缺少从你的脸书账户导入照片的功能，比如说，我在那里保存了我所有的在线照片。

该应用程序应该可以在大多数 Windows XP 安装程序上运行。抱歉，目前不支持 Mac。我已经在我的机器上运行了一周，没有发现任何负面的性能影响。

要查看 PhotoJoy 的运行情况，请查看本文底部嵌入的视频，该公司使用我孩子的一些照片制作了一个演示。

在许多方面，该产品类似于[早已消失的电影循环](https://web.archive.org/web/20221006201749/http://www.beta.techcrunch.com/2007/01/06/filmloop-dips-toes-into-the-deadpool/)(还记得他们吗？这两家公司都很难应对 Slide 和 RockYou 带来的日益激烈的竞争，以及一家风投公司明显缺乏支持。Filmloop 也是一个小把戏，而 PhotoJoy 只是几个 IncrediMail 产品中的一个——HiYo 是另一个。

由于看不到直接的商业模式，并考虑到公司的历史，我要求澄清 PhotoJoy 的广告业务及其商业模式策略。IncrediMail 的首席执行官 Ofer Adler 解决了我的担忧:

> “我们绝不会用这种产品推销任何其他安装。
> 
> 我们将使用一种类似于 picasa / yahoo messenger /现在甚至 AVG 反病毒软件的广告搜索商业模式，并有一个免费版本，将建议用户在安装期间接受我们的谷歌/其他合作伙伴搜索属性，如主页/工具栏/默认搜索提供商，假设一定比例的安装将接受它们。(当然，任何不想要的人都可以不接受它们)。此外，可能会有一些更多的功能/内容的高级版本。"

所以尽管叫我笨蛋吧，但是 PhotoJoy 让我脸上挂着微笑，有时这不仅足够好，而且足够多！
