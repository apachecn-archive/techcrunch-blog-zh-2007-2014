# 谷歌称其智能广告关联系统为“智能屁股”(没错，是认真的)

今年我(和其他人)在 D10 学到的最好的东西是什么？(当然，除了史蒂夫·乔布斯[曾经拥有一只孔雀](https://web.archive.org/web/20230404044803/https://techcrunch.com/2012/05/30/want-to-be-like-steve-jobs-well-its-probably-not-going-to-happen/)。)

在 D10 的舞台上，广告的谷歌 SVP[苏珊·沃西基](https://web.archive.org/web/20230404044803/http://www.crunchbase.com/person/susan-wojcicki)在回答沃尔特·莫斯伯格关于广告相关性的可悲状态的问题时透露，谷歌在内部将其机器学习智能广告定位技术称为“智能驴”。

除了那些自动驾驶汽车，这可能是目前谷歌发生的最酷的事情。

Wojcicki 然后告诉 Mossberg,“大量的”谷歌工程师正在致力于改进“智能屁股”,“网络上有各种各样令人惊奇的东西。广告不是其中之一，”她说。"展示广告非常粗糙，对于所提取的价值来说，CPM 价格非常高."

“我不是反对广告，只是不明白为什么不好。”莫斯贝里回答道。“当我在网上漫游时，为什么看不到我可能愿意花钱购买的东西？”

Wojcicki 同意他的观点，希望有一天 Mossberg 会收到关于新技术产品的广告，而不是在阅读关于伊拉克战争的新闻时去伊拉克旅行。“我们正在变得更好，从用户的角度来看，我们正在发展，”她说。

我仍然在想象，每当谷歌的工程师们说“聪明的屁股”而不是“聪明的广告”时，他们会咯咯地笑。太神奇了。
