# 微软用更好的同步、存储、颜色等更新了它的 Outlook.com 安卓应用

今天[微软宣布](https://web.archive.org/web/20220930230344/http://blogs.office.com/b/microsoft-outlook/archive/2013/10/10/server-side-search-and-more-updates-to-outlook-com-android-app.aspx)其安卓 Outlook.com 应用的更新，包括服务器端搜索、改进的离线邮件存储、颜色、假期回复和别名。

Outlook.com 是微软的重点项目，拥有超过 4 亿用户。Hotmail 的继任者，Outlook.com 是微软所有移动平台的邮件愿景。据微软称，68%的 Outlook.com 用户通过移动设备发送电子邮件，因此拥有强大的应用程序很重要。

如果你是 Outlook、com 和 Android 用户，这个更新至少会包含一些你喜欢的东西。如果您想刻录存储容量，现在可以将所有邮件同步到您的设备。如果你想找到任何你想要的信息，但不想下载，微软已经在 Android 应用中增加了服务器端搜索。另一个不同的问题:以前*怎么没有*有这个功能？

微软还为该应用程序添加了新的颜色，并引入了对别名以及假期回复的支持。总而言之，Android 现在似乎是一个功能齐全的邮件体验。我无法想象以前用它是什么样子。之前，超过 10，000 个评级，微软在 Play store 上的评级低于三星。

你可以[在这里](https://web.archive.org/web/20220930230344/https://play.google.com/store/apps/details?id=com.outlook.Z7)下载应用。如果你认为添加颜色是一个小更新，顺便说一句，你是在和雅虎争论，雅虎最近推出了对其邮件体验的颜色改变。显然这很重要。

最后说明:前三名的网络邮件体验继续蚕食边缘。什么时候会有人重新发明我们称之为电子邮件的这种该死的瘟疫？
