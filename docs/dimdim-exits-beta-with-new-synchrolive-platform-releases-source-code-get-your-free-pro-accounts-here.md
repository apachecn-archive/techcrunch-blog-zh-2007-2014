# Dimdim 退出测试版，推出新的 SynchroLive 平台，发布源代码(在此获得免费的专业帐户)

![](img/5b352412e1ce9dddd4a57ae9ebe69702.png)

不需要安装任何桌面软件的基于浏览器的网络会议平台提供商 Dimdim 在发布其应用程序的新版本时放弃了 beta 标签，同时向开源社区发布了完整的源代码。

如果你想要一个免费的专业版(每年 99 美元起，不限会议次数)**，点击这个[特殊的 TechCrunch 链接](https://web.archive.org/web/20221006100956/http://www.dimdim.com/techcrunch)来注册。Dimdim 将升级每 100 个注册的百分比，总共多达 1000 个免费的专业帐户。**

随着新的 4.5 版本的发布，Dimdim 引入了他们所谓的 SynchroLive 通信平台，旨在自动扩展其性能，以确保所有实时通信完全同步。同步网页浏览功能是我在任何基于网络的协作工具中见过的最好的东西之一。SynchroLive 共同浏览基本上使您能够通过简单地建立一个房间并将它的专用 URL 发送给其他参与者来与多人共享您的浏览会话。SynchroLive 会让你的客人看到你浏览网页时看到的内容。这也适用于视频，能够同时观看视频流并与其他与会者交谈/聊天。另一个受欢迎的新增功能是双向视频聊天功能。

Dimdim 有一个免费版本，支持多达 20 个会议与会者，并提供专业版和企业版。定价明显低于 [GoToMeeting](https://web.archive.org/web/20221006100956/http://www.crunchbase.com/product/gotomeeting) 和 [WebEx](https://web.archive.org/web/20221006100956/http://www.crunchbase.com/company/webex) 等知名竞争对手。

今年早些时候，Dimdim [推出了其网络会议软件的托管版本](https://web.archive.org/web/20221006100956/http://www.beta.techcrunch.com/2008/04/10/dimdim-debuts-hosted-web-conferencing-solution/)，允许公司在 Dimdim 的服务器上“在云端”举办会议。现在，Dimdim 正在推出 [Liberty](https://web.archive.org/web/20221006100956/http://dimdim.com/opensource/dimdim_open_source_community_edition.html) ，这是基于新的商业托管版本的最新开源社区版本，并在 GPL3 许可下发布源代码，对参与者、会议或混搭的数量没有限制。

Dimdim 号称全球有 100 多万人在使用它，[在今年夏天由 Index Ventures、Nexus India Capital 和 Draper Richards 牵头的 B 轮融资中筹集了 600 万美元](https://web.archive.org/web/20221006100956/http://www.beta.techcrunch.com/2008/07/09/dimdim-gets-6-million-for-open-sourced-web-meetings/)，使其总资金达到 840 万美元。该公司总部位于马萨诸塞州的伯灵顿，在加拿大新罕布什尔州和印度海德拉巴设有办事处。
