# 脸书本周三举行特别手机活动

![](img/f117687b2b267aa49df5775d1f813595.png)

脸书刚刚向我们发出邀请，邀请我们参加本周三举行的一场特别活动。主题？手机。

我们不知道脸书计划推出什么(尽管我们会尽力提前发现)，但我们有一些猜测。最明显的是该公司期待已久的 iPad 应用程序(它仍然没有官方应用程序)。它还可能包括更多与地点相关的功能——脸书最近正在测试与地点相关的交易服务，移动显然是用户访问这些服务的最佳地点。

当然，这可能与难以捉摸的[脸书手机](https://web.archive.org/web/20221207194151/https://beta.techcrunch.com/2010/09/19/facebook-phone/)有关，尽管当我们上个月与马克·扎克伯格谈论这个时，听起来脸书的(未经证实的)定制版安卓系统还很遥远。或许这是一笔将脸书更深入地集成到一些合作手机中的交易？

脸书的最后一次大事件是在 10 月 6 日，当时该公司推出了一个完全改进的团体功能。
