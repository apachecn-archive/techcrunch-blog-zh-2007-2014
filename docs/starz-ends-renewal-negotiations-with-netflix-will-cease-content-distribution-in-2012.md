# Starz 结束与网飞的续约谈判；将在 2012 年停止内容分发

斯塔兹和网飞卷入了关于电视网优质电影内容的谈判。在过去的一年里，自由媒体旗下的 Starz 一直在讨论该协议的条款，该协议将于 2012 年 2 月到期。《T2 洛杉矶时报》今年早些时候报道称，网飞每年将花费超过 2 亿美元进行更新。今天，Starz [发布了一份声明](https://web.archive.org/web/20230404012943/http://www.marketwatch.com/story/starz-entertainment-issues-statement-on-status-of-netflix-affiliation-agreement-renewal-discussions-2011-09-01?reflink=MW_news_stmp)，声明与网飞的续约谈判已经结束，一旦明年协议到期，该网络将从网飞图书馆删除其内容。

Starz 娱乐公司已经结束了与网飞的续约谈判。当协议于 2012 年 2 月 28 日到期时，Starz 将停止在网飞流媒体平台上发布其内容。这一决定是我们战略的结果，即通过保留我们独有的高价值内容的适当定价和包装来保护我们品牌的优质特性。凭借我们目前的工作室权利和不断增长的原创节目，该网络在评估新机会和扩展其整体业务方面处于有利地位。

为期四年的 Starz 协议每年给网飞带来 1000 多部电影，包括索尼和迪士尼的电影。与 Starz 的交易对网飞很重要，因为它实际上为电影流媒体服务提供了更新的标题。

内容所有者正在逃离网飞的平台，这绝对不是一个好兆头。假设 Starz 和网飞不能达成一个对双方都合适的财务协议(也许 2 亿美元太多了)。Starz 也可能受到来自迪士尼等公司的压力，据报道，迪士尼希望限制电影频道向网飞提供电影的能力。Showtime 最近也从网飞撤掉了一些内容。

但是网飞已经能够为其国际市场达成新的内容交易。

九月的第一天对网飞来说不是一个好日子，他的[有争议的提价](https://web.archive.org/web/20230404012943/https://techcrunch.com/2011/09/01/psa-netflix-price-hike-rolls-out-today/)今天刚刚出台。网飞[的股票在盘后交易中下跌超过 8%。](https://web.archive.org/web/20230404012943/http://www.google.com/finance?q=netflix)
