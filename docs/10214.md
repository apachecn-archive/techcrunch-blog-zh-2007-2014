# 索尼凭借 PS3 广告赢得了互联网

不管你喜欢还是讨厌游戏，这则索尼广告恰当地抓住了视频游戏的精髓。它是关于经历的。这是关于回忆。电子游戏允许一个人，哪怕是短暂地，变成另一个人，进入他们的生活，看到一个不同的世界。林克、幽灵和戈登·弗里曼都成了玩家故事的一部分，其重要性不亚于汤姆·索亚和哈弟小子。

当然，这个特别的视频是为索尼和 Playstation 3 准备的，但它描绘了游戏的巨大吸引力。干得好，索尼。你今天赢得了互联网。
