# 现在 Twitter 可以让链接变得短小精悍

![3213679186_4678fd1bfc](img/d9737af58ee0c0f8cc8ce7a0f5d1cf28.png "3213679186_4678fd1bfc")

Twitter 现在[拥有 twee.tt 域名](https://web.archive.org/web/20230203051806/http://nic.tt/cgi-bin/search.pl?name=twee.tt)，以“.”结尾。tt”特立尼达和多巴哥的国家域名。Twitter 计划用自己的网址缩写器来填补更多的漏洞吗？或者这仅仅是为了保持对“Tweet”一词使用的控制而采取的防御措施？

Twitter 最近推出了自己的黑莓应用程序，并收购了 Tweetie iphone 应用程序，使其与一些第三方应用程序开发商展开了更直接的竞争。Twitter 目前使用 [bit.ly](https://web.archive.org/web/20230203051806/http://bit.ly/) 作为其默认的网址缩写，但[可以轻松地将](https://web.archive.org/web/20230203051806/https://techcrunch.com/2009/05/06/url-shortening-wars-twitter-ditches-tinyurl-for-bitly/)切换到 twee.tt。从品牌的角度来看，缩短所有 twee.tts 的链接是有意义的。Twitter 试图(但失败了)将“Tweet”注册为商标，去年夏天，Twitter 警告开发者不要使用这个术语，之后 Twitter 放弃了。Twitter 在自己的网站上使用了[这个词，当然，它与该公司有着密切的联系。](https://web.archive.org/web/20230203051806/https://techcrunch.com/2009/07/27/twitter-adopting-the-word-tweet-on-its-website/)

需要澄清的是，仅仅因为它拥有 Twee.tt 网址并不意味着 Twitter 打算用它做任何事情。Twitter 拥有许多域名，它只是坐在那里，阻止其他人获取，或保留供未来使用。Twitter 也拥有 twitter.tt，有人[在 2010 年 3 月 27 日申请了 twe.tt](https://web.archive.org/web/20230203051806/http://nic.tt/cgi-bin/search.pl?name=twe.tt) 。一个都没有。tt 域名还活着。

Twitter 也可以用这个域名来销售新产品。例如，如果它最终在本周的 [Chirp](https://web.archive.org/web/20230203051806/http://chirp.twitter.com/index.html) 会议上发布广告，它可以指定特定的 URL，比如 twee.tt 或其他带有链接的广告信息。
