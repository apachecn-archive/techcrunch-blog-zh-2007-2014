# Six Apart 和 Vox——承诺是如何被浪费的

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/09/03/six-apart-and-vox%e2%80%94how-promise-gets-squandered/>

[![](img/756e159870edc20f99010bedd4c8803f.png "231-focus") ](https://web.archive.org/web/20230405220308/https://techcrunch.com/wp-content/uploads/2010/09/231-focus.jpg) Six Apart 正在关闭其免费博客服务 Vox，正如 Mike [指出的那样](https://web.archive.org/web/20230405220308/https://techcrunch.com/2010/09/03/six-aparts-vox-heads-to-deadpool/)这一声明实际上是为即将到来的与 VideoEgg 的合并做准备。随着全球 2.5 亿个独特的博客和不断增长的广告业务，Six Apart 并不是一个失败。但是，像 Slide 和 Digg 一样，它也没有兑现自己的承诺。像 Vox 这样的产品是一个很大的原因:随着博客越来越开放，评论者越来越刻薄，Vox 旨在成为博客世界中一个干净、明亮的地方。它有一个很棒的用户界面和一些不错的功能，比如“每日一问”,可以让不情愿的新博主开始写作。但是后来它就枯萎了。

我从关闭中得到的启示不是“六个分开正在清理待售”(他们是，六个分开日本是下一个)，而是“天哪，六个分开！怎么这么久？”

早在 2007 年，当 Six Apart 出售 LiveJournal 并任命克里斯奥尔登(Chris Alden)为首席执行官时，人们的口头禅是该公司终于要专注了。健康的多样化和做得太多做不好任何事情之间有一条细微的界限。Six Apart 一直有一个直觉问题——他们有几个有价值的资产，但在大型消费互联网品牌的黄金时代，它们不一定能成为一个大型消费互联网品牌。他们本质上是一家软件即服务公司，为媒体提供 MovableType，为网络发布工具提供 TypePad，为青少年提供 LiveJournal，为成人提供 Vox。Six Apart 本质上使自己成为了一家无人能直接收购的公司，因为它在做许多不同的事情。

这里有一个提示:如果没有人想买你，也许你不应该作为一个独立的公司有这么多不同的，开发不足的产品。过去三年里，我接触过的 Six Apart 的每一位高级经理都说这是公司最大的问题。然而，我们现在才看到关闭摇摇欲坠的房地产的举措。从外部很难说谁该受到指责，但 Six Apart 显然缺乏领导力和果断性。

我喜欢奥尔登。他了解媒体，他是一个好人，他带着善意和热情进入这份工作。但也许这就是问题的症结所在——他更关注出版商而不是读者，而且他太好了，不会更快地做出艰难的决定。据我所知，公司内部的事情变得越来越糟糕，奥尔登指责他的一些高级团队，而该团队的大部分人转向奥尔登。当(我应该说是“如果”，但很可能是“当”)与 Video Egg 的这笔交易宣布时，他们会笑逐颜开，会有一个关于为什么这两者在一起有意义的伟大叙事，也许他们会这样做。但我所听到的都不是现在在六处总部发生的事情。

事情不应该是这样的。Six Apart 是最早的博客工具之一，也是最早对简单易用收费的工具之一。博客的很多外观和感觉是由六个不同的创始人本和梅纳·特罗特塑造的。Six Apart 拥有一个更强大、更有趣的董事会，其中包括超级天使雷德·霍夫曼、超级巨星、知识共享创始人伊藤和奥古斯特资本的斗志昂扬的大卫霍尼克。网络上一些最聪明的人清楚地看到了《六人行》中的伟大之处。而且它有充足的资金——它从投资者那里筹集了 2000 多万美元，在推出 LiveJournal 时又筹集了数百万美元。

欢迎来到 Web 2.0 令人伤感的阶段。大赢家——脸书、Twitter、Zynga 和 LinkedIn——已经和明显的输家——Friendster、Plurk、Friendfeed 和许多我们已经忘记的名字分开了。直到现在，我们才开始对中间的公司做出判断。那些成功地建立了真正的公司，拥有相当大的影响力和可观的收入，并且比许多竞争对手存活的时间都长的风险投资，但是却没有兑现他们的承诺。最好的方式是滑梯，这是一个很好的出口，没有人会因此赔钱，有些人还会因此赚钱。然后是 Six Apart 现在的处境——准备与另一家私人公司进行令人不安的合并，这对任何人来说都不是“退出”,而只是意味着又要苦干四年才能做成大事。
