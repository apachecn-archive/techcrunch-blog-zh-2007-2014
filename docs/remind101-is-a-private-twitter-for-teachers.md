# Remind101 是教师的私人推特

今天早上，来自帕洛阿尔托的 Imagine K12 孵化器的十家公司在 [TechCrunch Disrupt SF](https://web.archive.org/web/20230205100435/https://techcrunch.com/2011/09/14/watch-tc-disrupt-sf-2011-live-day-3/) 的舞台上展示了他们的产品。为了参加为期三个月的 Imagine K12 计划，一家初创公司必须有教育倾向，但除此之外，可以专注于任何产品。

当我问 Imagine K12 的参与者布雷特·科普夫(Brett Kopf)为什么要创办一个只专注于教育的孵化器时，他回答说，“教育中有很多问题，所以有很多解决问题的空间。”很公平。

Kopf 的初创公司， [remind101，](https://web.archive.org/web/20230205100435/http://remind101.com/)是一个私人的“教师推特”，为教育工作者提供一种“安全”的方式来广播消息，如考试提醒(因此得名)和鼓励学生的笔记；使用 remind101，任何参与者都无法访问任何其他参与者的个人和敏感联系信息，如社交网络档案、电话号码或电子邮件地址。

“不是我们不信任老师。这是因为老师没有一个好的沟通方式，因为人们可能会做出一些潜在的假设，”科普夫说。在该服务推出后的三周内，该产品已被 1500 名教师、15000 名学生和家长使用；累计发送超过 13 万条消息。
