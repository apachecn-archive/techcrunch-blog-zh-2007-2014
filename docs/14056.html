Mac版Tweetbot现售价19.99美元:Twitter客户端令牌限制是高价的罪魁祸首TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2012/10/18/tweetbot-mac-twitter-osx-mac-app-store/amp/

经过长期的公开alpha和beta测试阶段后，面向Mac的Tweetbot终于以零售形式发布。著名的iOS和Mac开发者Tapbots的原生OS X Twitter客户端可以从 Mac应用商店获得，价格为19.99美元，有点令人惊讶。它值20美元吗？一千次是的。但不熟悉Tapbots工作的用户可能会拒绝为这种应用支付那么多钱——这正是Tapbots所期待的。

当我向Tapbots的联合创始人保罗·阿达德询问定价时，他说这故意高于公司最初的预期，主要是由于 Twitter的新API令牌限制，这限制了Tweetbot for Mac客户端可以支持的最大用户数量。这一限制与iOS版Tweetbot所享受的限制是分开确定的，因此只能容纳更少的用户群。一旦令牌消失，Tapbots将不再能够销售Tweetbot，所以从这个意义上说，这是一个非常真实的、非营销意义上的限量版软件。

Tapbots花费了大量的时间和精力为Mac开发Tweetbot，他们打算在停止销售后继续支持它，因此价格更高。因此，尽管价格标签可能会让Tapbots和Tweetbot的新用户望而生畏，但忠实用户会更愿意付费，我不认为供应会持续那么久。

与此同时，Tapbots希望用户撤销旧版本Tweetbot 上的令牌，如果他们不打算使用它们，还希望联系Twitter 并鼓励他们建议提高令牌限额。我有点怀疑第二种方法会对社交网络的政策变化产生如此大的影响，但试试也无妨。

如果你没有像我一样在Tweetbot发布后使用过它的alpha和beta版本，我只能说它太棒了。自从Tweetie被Twitter收购后，我一直在寻找一个合适的替代者，但没有什么能与Tweetbot相提并论。这是我早上启动的第一个应用程序，也是我晚上关机时关闭的最后一个应用程序。它有实时流媒体，可定制的栏目，静音功能和其他一系列令人敬畏的东西。此外，它拥有周围最好看的设计。即使是20美元要价的两倍，Tweetbot也是显而易见的。
