# 刚刚被民粹主义收购的 Mokono 收购德国博客网络 Adnation

**独家——**紧随[民粹](https://web.archive.org/web/20230203160712/http://www.crunchbase.com/company/populis)以[€820 万](https://web.archive.org/web/20230203160712/http://eu.beta.techcrunch.com/2011/09/15/populis-acquires-germanys-largest-blog-network-mokono-for-e8-2-million/)收购[木科诺](https://web.archive.org/web/20230203160712/http://www.mokono.com/en/)之后，后者完成了自己的一次收购。TechCrunch 获悉，Mokono 在周末完成了对德国高级博客网络 Adnation 的收购。

此次收购为 mokono/popular 网络增加了 300 万独立月访客，其中大部分是来自德国一些知名博客的读者，如[Spreeblick.com](https://web.archive.org/web/20230203160712/http://spreeblick.com/)、 [BILDblog.de](https://web.archive.org/web/20230203160712/http://bildblog.de/) 和 [crackajack.de](https://web.archive.org/web/20230203160712/http://crackajack.de/) 。Mokono 的网络现在每月有 1700 万独立用户。阅读更多关于欧洲 TechCrunch 的信息。
