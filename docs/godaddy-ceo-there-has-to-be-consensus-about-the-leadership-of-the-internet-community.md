# GoDaddy 首席执行官:“必须就互联网社区的领导地位达成共识”

在 TechCrunch 的一次简短采访中，GoDaddy 的新任首席执行官沃伦·阿德尔曼(Warren Adelman)就该公司[高度公开逆转](https://web.archive.org/web/20230326023320/https://techcrunch.com/2011/12/23/godaddy-no-longer-supports-sopa/)其在 [SOPA](https://web.archive.org/web/20230326023320/https://techcrunch.com/tag/sopa/) 的立场做了一点损害控制。戈达迪此前曾发表声明，强烈支持这项有争议的法案[，你可以在这里找到](https://web.archive.org/web/20230326023320/http://www.thedomains.com/2011/11/15/here-is-godaddys-statement-in-support-of-the-stop-online-privacy-act-house-hearing-tomorrow/)。最后一天，越来越多的草根阶层以抵制的形式拒绝了该公司及其立场。但是今天该公司的声明明显做了彻底的改变。

阿德尔曼指出，他“担任这家公司的首席执行官整整一周”，投诉和反馈在过去 24 小时内不断增加，正是在这段时间里，他开始参与进来。他说，反馈让他意识到他们应该“后退一步，看看当前的立法。”当然，对 SOPA 及其在参议院的孪生兄弟的强烈抗议可以追溯到更久以前。

我认为说他们应该后退一步很容易，但事实上他们已经公开表示强烈支持条例草案的几乎每一项条文。阿德尔曼拒绝回答这个问题，他说“到目前为止，他不能评论这件事的历史。”当我问该公司是否仍然认为向 SOPA 提出的反对意见是“没有根据的”，他说“还有其他人需要表达他们的主张”，大概是指专家和互联网名人的证词很少这一事实。

当被问及此事时，阿德尔曼不能承诺改变其在国会的立场，但他说“我会把这带回给我们的立法人员，但我同意这是重要的一步。”但当被追问时，他说“我们将退后一步，让其他人担任领导角色。”他认为，公开声明取消他们的支持就目前而言已经足够，尽管还会考虑采取进一步的措施。

最后，关于该公司相当模糊的承诺“当互联网社区支持它时，如果互联网社区支持它，Go Daddy 将会支持它，”我说互联网社区似乎不太可能支持这种类型的法案，当然也不会以任何形式支持 SOPA——所以这一声明很难解读。阿德尔曼说,“对于互联网社区的领导权，必须达成共识。这是一个大型社区，也是一个全球性的社区。”这

GoDaddy 及其新任首席执行官的声明远非强有力，但它们可能会随着时间的推移而固化。“不支持”并不等同于反对，在这样一个问题上，互联网，他们自己承认是他们逆转的来源，将要求反对。然而，他们无法回避这样一个事实，即他们是该法案坚定的公开支持者，这可能会在一段时间内成为他们的一个负担。虽然面对广泛的抗议，这种快速的逆转可能只是一个商业决定，但它仍然是一个商业决定，在反对 SOPA 和类似的立法方面可能是有用的。
