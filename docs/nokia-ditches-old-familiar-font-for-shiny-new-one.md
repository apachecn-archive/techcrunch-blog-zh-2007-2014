# 诺基亚抛弃旧的熟悉字体，采用闪亮的新字体

![](img/bcaff9b2832a63cafc3ce5fb44c9df1b.png "Nokia")

嘣~！

是的，[芬兰巨头](https://web.archive.org/web/20230203215634/http://www.crunchgear.com/tagnokia/)已经表示将停止使用其老的备用诺基亚 Sans，而开始使用诺基亚 Pure。转向 Pure 是该公司正在进行的品牌重建的一部分，这也包括拥抱微软和它的 Windows Phone 7 平台。

诺基亚说，这种新字体在今天的数字设备上看起来和感觉都会比以前的 Sans 更好。
