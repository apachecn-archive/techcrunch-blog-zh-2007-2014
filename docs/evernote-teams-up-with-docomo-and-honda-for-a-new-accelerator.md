# Evernote 磨练其平台野心，与 Docomo 和本田合作开发新的加速器

生产力应用 [Evernote](https://web.archive.org/web/20221205093336/http://www.evernote.com/) 今天宣布，它将与本田和日本移动运营商 Docomo 合作，推出 [Evernote 加速器](https://web.archive.org/web/20221205093336/http://dev.evernote.com/accelerator/)，这是一个为期一个月的辅导计划，位于 Evernote 在红木城的总部，面向世界各地使用 Evernote 的 API 制作应用的开发者和小型创业公司。Evernote 的战略是在自有品牌产品之外发展业务，并为其他开发生产力服务的人提供更广泛的平台，此举是这一战略的一个进步:该公司今天每天已经有 60 亿次 API 调用，但其中大多数来自 Evernote 应用程序本身。

Evernote、本田硅谷实验室(Honda Silicon Valley Lab)和 Docomo Innovation Ventures 将为参与者提供工作空间、生活空间、Evernote 工程师的指导、市场营销和生活津贴，但该项目不会直接投资。

“目前，我们认为没有必要为它们提供资金。最好的结果可能是 Evernote 产品的成功。Rafe Needleman 说，他是一名前科技记者，去年以平台倡导者的身份加入 Evernote，并帮助运营加速器。“我们期待伟大的 Evernote 产品，这将使我们的用户获得成功。”不过，他补充说，初创公司或独立开发者进入 Evernote 生态系统的部分优势可能还包括向其他可能成为支持者的人做介绍。

但是，即使 Evernote 可能没有风投式的直接投资，也可能有另一种形式的投资:Needleman 指出，Evernote Food 是该公司的独立应用程序之一，最初是在 Evernote 的一次黑客马拉松中开发的。Evernote 最终购买了这个 IP 来创建我们今天所知道的服务。

参与者将从 2013 年 Evernote Devcup 的获胜者中选出，Evernote 举办了一场跨地区的活动，以鼓励更多人使用其 API。它表示，该课程将于 2013 年 10 月开始，包括辅导和发展。然后在 11 月会有一个硅谷式的演示日。

虽然 Needleman 表示 Evernote 对潜在应用在 Evernote 平台上运行的可能性持开放态度，但他也将注意力转向了加速器的两个联合发起人。本田汽车车载系统的应用编程接口也将出现在组合中，Docomo 的影响也意味着将会有一个强大的移动元素。这将产生移动和运输方面的特定奖项。

虽然加速器在创业公司的世界中已经变得非常普遍，但这一次的独特之处在于 Evernote 提议如何寻找候选人。Needleman 说，这个想法是利用尽可能远的地方的人们，比如在东京、新加坡、苏黎世和墨西哥城举办的黑客马拉松；然后用体验科技圣地的生活作为吸引他们的另一种方式。

“我们正试图吸引硅谷以外的初创公司，”他说。“这真的被视为一个机会，来看看成为这里一切的一部分是什么感觉。”
