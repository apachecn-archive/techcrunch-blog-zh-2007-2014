索尼售价149美元的智能手表终于登陆美国

原文：https://web.archive.org/web/https://techcrunch.com/2012/04/12/sonys-149-smartwatch-finally-lands-in-the-u-s/amp/

令人失望的LiveView在2010年推出时没有给人留下太多印象，索尼在今年的CES上发布了另一款可穿戴设备——智能手表，让我们大吃一惊。在英国短暂兜了一圈后，索尼刚刚宣布他们最新的小型腕戴设备现在在美国上市，售价149美元。

与之前的LiveView一样，智能手表本质上只是一个小型的辅助显示器，一旦安装了相应的应用，它就可以连接到任何运行Android 2.1或更高版本的蓝牙Android设备。幸运的是，索尼认为这次应该包括一个完整的多点触摸显示屏，而不是智能手表的前身中出现的笨拙的仅边缘导航方案。

接下来的问题是，你会怎么处理这个东西？除了用它来看时间，用户还可以在1.3英寸的有机发光二极管显示屏上阅读完整长度的电子邮件和短信，以及控制手机上的音乐播放。这也只是冰山一角——就在今年的CES进入高潮时，索尼发布了智能手表平台的 SDK ,我被告知，Google Play商店中有近60个迷你应用程序等待所有可穿戴计算的粉丝来玩。

其中一些似乎已经成为必备——当我的Galaxy Nexus在家里不可避免地丢失时，我会使用这个 Find Phone 应用程序，快速访问 Endomondo应用程序可以为我的跑步节省不少麻烦。

最近几个月，很多人已经接受了这个概念，并带着它跑了起来——摩托罗拉想到了，WIMM的人们也想到了，还有这个特别棒的 Kickstarter项目。索尼似乎肯定是在以智能手表相对低廉的价格和一系列引人注目的腕带(如果你好奇的话，每个20美元)瞄准大众，如果你觉得从口袋里掏出手机太麻烦，那绝对值得再看一眼。
