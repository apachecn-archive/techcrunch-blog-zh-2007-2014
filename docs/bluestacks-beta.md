# 在 Alpha 吸引了 100 万用户后，BlueStacks 进入测试阶段，将 45 万个 Android 应用程序带到你的电脑上

自从 pre-launch [BlueStacks](https://web.archive.org/web/20230307110439/http://bluestacks.com/) 在 A 轮融资中筹集了 760 万美元以来，这家初创公司一直备受关注。为什么？因为它开发的软件旨在让 Android 用户在所有 Windows PCs、平板电脑和笔记本电脑上运行他们的应用程序——无需任何修改。更不用说初创公司[几个月后又从 AMD 和 Citrix Systems 筹集了 640 万美元](https://web.archive.org/web/20230307110439/https://techcrunch.com/2011/12/06/android-apps-on-your-pc-bluestacks-app-player-blows-past-half-a-million-downloads/)，现有投资者如 Andreessen Horowitz、Ignition Ventures 等也提供了资金。

B 系列紧随阿尔法推出的 Windows 应用播放器之后，这是一款免费的软件下载，用户可以在任何 Windows PC 上一键访问 Android 应用，并可以全屏查看这些应用。伴随 App Player 的是 Cloud Connect，这是一种基于云的服务，允许 PC 成为任何基于 Android 的移动设备的延伸，反之亦然。今天，BlueStacks 正在推进其下一个主要版本，即其应用程序播放器的测试版，该版本正在追求一个雄心勃勃的目标，即将大约 45 万个 Android 应用程序提供给超过 10 亿的 PC 用户。

测试版软件的发布值得注意，因为它现在允许桌面电脑运行图形密集型 Android 应用程序，将它们加速到以前版本的软件无法达到的程度。这项新功能是由该公司正在申请专利的新技术 Layercake 实现的，该技术允许 Android 应用程序在基于 x86 的 PC 上运行，包括为 ARM 处理器编写的应用程序，例如愤怒的小鸟空间或水果忍者。

BlueStacks 的测试版使用个人电脑的显卡，让运行 Unity 等图形密集型引擎的应用程序运行起来没有任何故障，使用的硬件加速与浏览器中发生的没有什么不同。新版本还通过鼠标和键盘箭头将智能手机体验的加速度计倾斜和缩放功能带到了桌面上。

这家初创公司在为期三个月的 alpha 阶段获得了相当大的吸引力，吸引了 100 多个国家的 100 多万用户，打开了超过 450 万个应用程序。随着测试的进行，这家初创公司希望在早期成功的基础上再接再厉，同时让体验更加顺畅。这意味着用户现在可以从 BlueStacks 下载应用程序，即使他们没有 Android 手机，这是一个反映 Windows 界面的 dock launcher，此外，云连接使用户可以在他们的 PC 上发送和接收短信。

BlueStacks 营销和业务发展副总裁表示，该公司已经与 Fruit Ninja、SliceIt！、Barnes & Noble 的 Nook、Townsmen、Evernote、Defender 和 StumbleUpon 在测试版中预装他们的应用。这些开发者被 BlueStacks 的软件所吸引，因为它为一个巨大的潜在市场——超过 10 亿的个人电脑用户——打开了 Android 应用发现的大门。不仅如此，开发人员不必移植或修改他们的应用程序就可以在那些电脑上运行，这就是银行的钱。
