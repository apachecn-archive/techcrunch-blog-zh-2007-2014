# Dropbox 正在研究超越文件夹的方法

Dropbox 首席执行官[德鲁·休斯顿](https://web.archive.org/web/20230205003307/http://www.crunchbase.com/person/drew-houston)今晚在 GigaOm 路线图会议上与 Gigaom 的[奥姆·马利克](https://web.archive.org/web/20230205003307/http://www.crunchbase.com/person/om-malik)交谈，并透露了一些关于他公司产品路线图的关键见解，即他希望超越文件夹作为 Dropbox 唯一组织原则的隐喻。

休斯顿将同步文件夹的想法描述为公司的“第一章”，“我们认为这只是第一步……(在 Dropbox 的初期)我们不想称 Dropbox 为‘备份’，甚至不想使用‘同步’这样的词……(因为)这与其说是存储，不如说是体验。”

他透露，该公司正在寻找超越文件和文件夹的方法，“如果 Dropbox 只是一个文件夹，那它有什么意义？他说，“我们真的建立了许多新的方法来管理和查看你的重要资料。”

然后，Malik 问这些新的管理和组织原则是否围绕时间和空间，“当你考虑你的[文件]时，你可以将它分成这些类别，然后你不仅可以拥有照片本身，还可以拥有与照片相关的所有数据的索引，例如，不仅这张照片是在哪里拍摄的，而且还有在 10 英里半径内拍摄的所有其他照片。”
