# 怪兽与 SDJ 合作开发“怪兽数字”闪存产品

如果你发现自己渴望一个新的存储卡，但你所有的选择都太平淡无奇，那么怪物和 SDJ 技术可能会照亮你的一天。这两家公司刚刚宣布，他们已经签署了一份长期许可协议，这意味着怪兽数字闪存产品将很快上架。

没错——虽然两家公司都没有进入具体的产品类型，但我们可以猜测一下他们在做什么。SDJ 首席执行官杰伊·坦登提到，人们越来越关注“移动计算、智能手机、数码相机和视频记录设备”，因此预计一些时髦的怪物品牌存储卡和闪存驱动器将很快亮相。当然，这些将承担怪物品牌，所以也准备好溢价标签。

这里要记住的是，这只是一个许可协议，所以 SDJ 将实际上制造产品，而怪物公司的人将做他们最擅长的:**营销这些东西。**

读者可能已经知道 Monster 的惯用手法，但那些从未享受过这种乐趣的人，Monster 采用电缆等商品，并利用其营销能力给它们带来某种高级光环。似乎总是有一场激烈的辩论，关于怪物产品是否实际上比一般的替代品表现更好(我个人认为他们没有)，但足够多的人是怪物的粉丝，该公司不断推出产品。

第一批巨型数码产品将于下月在纽约举行的消费电子展上亮相。怪物迷也会很高兴地知道，怪物数码产品的更广泛的展示将很快开始。
