# 苹果现在正在一个操作视频中推销 Square 的 iPad 应用程序

几个月前，总部位于库比蒂诺的公司[开始在苹果零售店和网站上展示这家移动支付初创公司的设备](https://web.archive.org/web/20221206120328/https://beta.techcrunch.com/2011/04/16/square-apple/)，看起来苹果和 Square 更加亲密了。现在，苹果正在为 iPad 用户在其“[操作指南”播客](https://web.archive.org/web/20221206120328/http://itunes.apple.com/us/podcast/process-sales-transactions/id419217687?i=95045566)中推广 Square。

苹果制作的免费播客视频推荐了一些功能和应用程序，以帮助用户在 iPad 上执行业务功能。具体到 Square，苹果推荐支付产品帮助用户在 iPad 上处理现金或信用卡交易。

该视频本身以 Square 的新 iPad 应用程序 [Square Register](https://web.archive.org/web/20221206120328/https://beta.techcrunch.com/2011/05/23/squares-disruptive-new-ipad-payments-service-will-replace-cash-registers/) 为特色，这是一款高性能的销售点替代收银机和销售点终端。

苹果出售 Square 是该公司的一个胜利，但需要注意的是，苹果还出售其他移动支付技术，如 VeriFone 的读卡器。有趣的是，相比其他移动支付产品，苹果正在选择推荐 Square 作为销售交易产品。这是一件大事。

说 Square 势如破竹可能是一种保守的说法。Square 每天处理 300 万美元的支付，据报道，square 正在筹集 5000 多万美元的新资金，估值高达 20 亿美元。该公司刚刚在董事会中增加了维诺德·科斯拉和拉里·萨默斯。
