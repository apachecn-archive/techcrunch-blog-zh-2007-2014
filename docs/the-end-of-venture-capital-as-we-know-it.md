# 我们所知的风险投资的终结？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/12/06/the-end-of-venture-capital-as-we-know-it/>

![](img/04572e3bb06b31bea036ccf631c38b3a.png)

上周，事情有了转机。我们发现，我们不仅处于经济衰退中(T2)，而且这场衰退从一年前就开始了。[科技裁员](https://web.archive.org/web/20221207202417/http://www.beta.techcrunch.com/layoffs/)进入高潮(AT T 有 12，000 人，Adobe 有 600 人，Real Networks 有 130 人)，根据我们的统计，总失业人数至少达到 90，000 人。

甚至脸书也决定无限期推迟允许员工私下出售部分股票的早期计划。脸书态度大转变的一个可能的考虑是，外部投资者可能不再愿意以该计划要求的已经降低的 40 亿美元内部估值购买脸书的股票，更不用说微软去年获得的过度膨胀的 150 亿美元。

资本正在枯竭，事情在好转之前可能还会变得更糟。到目前为止，在这次衰退中，我们已经看到初创公司做好了准备(正如他们应该做的那样),并希望生存足够长的时间，以走出另一端。

但是风险投资公司呢？我们什么时候才能开始看到风投裁员和基金倒闭？

这在某种程度上已经发生了。一些风投网站上列出的合伙人数量已经在悄然缩水。一些新的风险投资基金难以筹集资金，甚至现有基金在从资金紧张的有限合伙人那里募集资金时也遇到了问题。

华尔街的大屠杀正在对风险投资公司产生涓滴效应。通常投资风险投资基金的有限合伙人——大学捐赠基金、养老基金、投资银行、其他机构和富有的个人——现在现金短缺。仅在过去的四个月里，哈佛的捐赠基金就损失了 80 亿美元。许多有限合伙人根本无法兑现风险投资公司的注资。(当一家风险投资公司创建一只新基金时，它不会一下子募集到所有的资金。相反，它从有限合伙人那里得到承诺，他们将在需要资金时进行投资。

有限合伙人越来越多地试图在二级市场上以很低的折扣出售他们的承诺，而不是面对违约的惩罚。相反——知道他们可能无法收回他们的账单——风险投资者有动力放慢他们的投资活动。

在经济衰退期间，所有这些都在意料之中。企业家等待反弹，然后他们的初创公司再次获得资金。

但是如果这次衰退(和熊市)持续的时间超过一两年呢？如果创业公司的创始人不想等着风投和他们的有限合伙人重新站起来怎么办？

现在创业公司的运营成本如此之低(有了开源软件、云计算和遍布网络的虚拟团队)，以至于更多的公司可以在没有任何风险投资的情况下实现盈利。直到最近，当有人递给他们现金时，他们仍然高兴地接受。但是某些类别的创业公司，尤其是网络创业公司，现在可能发现他们甚至不需要那笔钱。Y Combinator 的保罗·格拉厄姆[认为](https://web.archive.org/web/20221207202417/http://www.paulgraham.com/divergence.html):

> *风投和创始人就像两个曾经被栓在一起的组件。大约在 2000 年，螺栓被拆除。因为到目前为止，这些部件都受到相同的力，它们看起来仍然连接在一起，但实际上一个部件只是靠在另一个部件上。剧烈的撞击会使它们分开。目前的衰退可能就是这种影响。*
> 
> 。。。现在这一代创始人想从风投那里融资，具体是红杉，因为拉里和谢尔盖从风投那里拿了钱，具体是红杉。试想一下，如果下一个热门公司根本不接 VC，会对 VC 业务造成什么影响。

新创企业获得的风险资本越少，脱钩就开始得越快。
