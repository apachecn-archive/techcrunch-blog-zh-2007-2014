# 脸书更新了其 Android SDK，使其与 iOS SDK 保持一致，以将更多的 Android 开发者纳入其移动领域

脸书已经更新了它的 Android SDK，使其与 4 月在 T2 推出的 iOS SDK 的特性集保持一致。脸书向 TechCrunch 证实，添加到 Android SDK 中的所有新功能已经可供 iOS 开发者使用。更新后的 Android SDK 可以通过脸书的 [Android 开发中心](https://web.archive.org/web/20221006141059/https://developers.facebook.com/android/)立即获得。这是自去年 12 月的 3.0 版本更新以来脸书 Android SDK 的首次更新。脸书的克里斯·朗今天在一篇博客文章中宣布了这些变化。

SDK 更新旨在让安卓开发者更容易将脸书的服务集成到他们的应用中，这当然有助于脸书嵌入第三方移动应用生态系统。对于社交网络来说，这可以说是一个更重要的任务，因为它试图颠覆其他人的应用程序——通过其 Facebook 主页接管 Android 主屏幕——已经如此惊人地失败了。

根据社交网络的数据，总收入前 100 名的 Android 应用程序中有超过 73%集成了脸书，而总收入前 100 名的 iOS 应用程序中有 87%集成了该软件。今年 4 月，总收入前 100 名的 Android 应用中有 70%包含脸书集成，总收入前 100 名的 iOS 应用中有超过 81%包含它。因此，尽管 Android 在不断进步，但它仍在追赶 iOS 集成。脸书对开发商的推销是这样的:帮助我们赚钱，我们也能帮助你。

脸书现在有 8 亿多手机用户，高于 4 月份报告的 6.8 亿多。该公司没有按移动平台对这些用户进行分类，但在被问及 Android 增长的具体情况时，该公司表示:“使用我们工具的 Android 开发者数量正在迅速增长。”

根据 IDC 最近的数据，随着 Android 在智能手机市场的份额迅速接近 80 %,脸书将迎来更多的应用和开发者。Android 在发展中市场也占主导地位，这些市场也是智能手机的增长市场——这使得它们对脸书更加重要。这确实提出了一个问题，为什么它仍然优先考虑 iOS 开发者工具，而不是 Android 工具。但也许这是将来会改变的事情。

脸书 Android SDK 的新特性包括:

*   **分享对话框—** 开发者可以更轻松地将应用内活动分享到他们的应用中，这样用户就不必登录脸书来将他们在应用中所做的事情分享到社交网络。“只需一行代码，开发者就能让人们跳过额外的三个步骤，立即分享到脸书”
*   **对象 API**——这使得开发者更容易将 Open Graph 集成到 Android 应用中。根据 iOS，Android 开发者“现在可以直接创建开放图形对象，不再需要托管带有开放图形标签的网页”
*   **重新设计的登录用户界面—** 和 iOS SDK 一样，重点是脸书在移动设备上的加载速度

此外，脸书表示，Android SDK 也将获得一个测试功能，允许开发者发布和测量他们应用的“**应用事件**”，以建立更清晰的移动广告参与情况。这显然也已经在 iOS 上实现了。“(Android)开发者现在可以衡量应用程序中预定义动作被执行的总次数，以帮助开发者了解脸书移动广告的参与度和投资回报率，”它在博文中指出。

回到 7 月 11 日，脸书在报告其 Q2 收入时表示，移动广告占该季度总广告收入的 41%——高于 Q1 的 30%。该社交网络表示，预计移动广告收入将超过桌面广告收入。支撑这种增长需要脸书继续将其触角伸向第三方应用生态系统，让更多开发者参与进来，将他们的应用与它的服务相集成。
