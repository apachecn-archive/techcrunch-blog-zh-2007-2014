# 一些大网站正在使用谷歌趋势来指导社论

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/10/09/some-big-sites-are-using-google-trends-to-direct-editorial/>

 [【谷歌趋势】](https://web.archive.org/web/20230315031306/http://trends.google.com/)，向你展示在任何给定时间谷歌上的热门搜索查询，现在已经有[两年多的历史了](https://web.archive.org/web/20230315031306/https://techcrunch.com/2006/05/10/google-trends-launches/)(今年他们还增加了[网站/域名跟踪](https://web.archive.org/web/20230315031306/https://techcrunch.com/2008/06/20/google-faces-off-with-compete-alexa-comscore-quantcast-brings-nothing-new/))。公关专业人士和品牌经理经常使用它来跟踪他们的资产有多热，这项服务还有无数其他用途。

不过，一个我们听说越来越受欢迎的用途是，博客、主流媒体网站和其他网站定期监控谷歌趋势，并根据热门词汇撰写报道。谷歌显示了一个[每日摘要](https://web.archive.org/web/20230315031306/http://www.google.com/trends/hottrends?sa=X)以方便数据收集。

我们的目标不是挖掘互联网用户认为有趣的东西并写下来。相反，这一切都是为了从谷歌获得更多的点击。

它是这样工作的:

博客和主流媒体网站被谷歌索引的频率非常高。事实上，每天很多次。这些网站通常已经有了很好的页面排名。将常规索引和页面排名与谷歌最近的新闻类搜索结果排名高于老的常青树的政策结合起来，你就有了一个容易被滥用的系统。

假设我经营一个流行的政治或名人八卦网站(这两个话题在 Google Trends 上经常出现)。我寻找人们正在输入的热门查询，不管是出于什么原因。然后我写了一篇博文，确保在文章标题中使用查询词(在将内容与特定查询匹配时，这个词的权重更大)。文章本身的内容大多无关紧要，只要你的正常读者不插科打诨。

在几分钟内，内容被谷歌编入索引，网站的高页面排名以及内容的新颖度将它推上了搜索结果的第一页。可能一直到顶端。

我们也不是在谈论微不足道的流量。我昨天与之交谈的一个人说，他每天可以获得多达 30，000 个额外的独立访问者，仅仅是通过将内容集中在热门查询上，这足以让几个全职人员投入这项工作。

我(和自己)讨论过这种行为到底有多“糟糕”。这样做的网站显然是利用了谷歌搜索方法中的一个弱点，但这并不像他们在从事黑帽搜索引擎优化策略，以欺骗谷歌认为他们的内容比实际更相关。相反，他们只是利用他们的网页排名分量和欺骗的边缘。

我不会说我听到的是哪些网站在这样做，但你可以自己检查一下。如果你看到一个似乎有点跑题或怪异的标题，接着是一些非常仓促编写的内容，快速浏览一下谷歌趋势，看看确切的查询是否在帖子或文章的标题中。你可能会惊讶于谁在利用这一点。
