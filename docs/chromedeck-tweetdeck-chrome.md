# ChromeDeck: TweetDeck 最终以 Chrome 应用 TechCrunch 的形式出现在网络上

> 原文：<https://web.archive.org/web/http://techcrunch.com/2010/12/07/chromedeck-tweetdeck-chrome/>

![](img/273d7e99bfe2cc1a5656d0218a5687e7.png)

你可以在你的 iPhone、iPad 或安卓手机上安装桌面客户端软件 [TweetDeck](https://web.archive.org/web/20230202222658/http://www.tweetdeck.com/) ，这是一款流行的实时流媒体阅读器。但到目前为止，还没有网络浏览器版本(不像 [Seesmic](https://web.archive.org/web/20230202222658/http://seesmic.com/app/) ，它最出名的是一个基于浏览器的应用)。今天，TweetDeck 在新的 [Chrome 网络商店](https://web.archive.org/web/20230202222658/https://chrome.google.com/webstore)中发布了第一个网络客户端 [Chrome 应用](https://web.archive.org/web/20230202222658/https://chrome.google.com/webstore/detail/hbdpomandigafcibbmofojjchbcdagbl)。

“这绝对是我们迄今为止最好的桌面 TweetDeck 版本，”首席执行官 Iain Dodsworth 说。你可以使用现有的 TweetDeck 帐户登录，并在不同的栏目中添加不同的实时流——Twitter、脸书、Foursquare checkins、Google Buzz。很快它也将支持 Gmail。ChromeDeck 在开发过程中被命名为 chrome deck，它从其最新的 [Android 应用程序中借鉴了一些 UI 元素。](https://web.archive.org/web/20230202222658/https://techcrunch.com/2010/08/11/tweetdeck-android/)有几个组合栏，分别标注为主页(来自不同账户的所有时间表)、我(所有针对你的提及和消息，如 Twitter @回复)和收件箱(直接消息，很快还有脸书和 Gmail 消息)。

Chrome 应用程序应该比桌面版 AIR 更快、更稳定、占用内存更少。一旦你把它“安装”到你的浏览器上，它就存在于它自己的标签中。当你启动一个新的空白标签页时，它和你安装的任何其他应用程序都是可用的。

如果你是一个经常使用 TweetDeck 的用户，你首先注意到的是它完全没有声音。然而，这种沉默不会持续太久。Dodsworth & Co .正致力于“录制一些 TweetDeck 声音，并将其添加到所有应用程序中”，以努力“创造一种社交声景，让你甚至不需要看着屏幕，就能知道发生了什么。”天啊，我妻子会喜欢的。哔，哔，哔。

我更喜欢安静。(请使用静音模式)。你注意到的另一件事是，当你需要它们时，它们会突然出现，而当你不需要它们时，它们又会消失。点击顶部的撰写框，它会展开给你空间来写，添加图像和你的位置，并选择你想发送你的消息的帐户。在你的信息流中点击回复，一个回复框会随着你回复的推文或消息一起放大到栏目的顶部，所有的都是内嵌的。平稳。
