# Sponsify 的目标是成为 YouTube 上的产品植入和其他原生广告的平台

一家名为 [Sponsify](https://web.archive.org/web/20221205171143/http://sponsify.co/) 的初创公司正寻求将原生广告大规模引入 YouTube。

我怀疑任何访问 YouTube 的人都会抱怨广告短缺，但这些广告大多采取播放前视频的形式，并将内容放在一边。与此同时，Sponsify 将专注于以更有意义的方式与内容整合的广告，例如通过在视频中植入产品。

这些活动对广告商来说很有价值，对创作者来说也很有利可图。据 Sponsify 联合创始人兼首席执行官 Muhammad Huzaifa 称，迄今为止的挑战是，这些创作者开展本土风格活动的唯一方式是与 Maker Studios 和 Machinima 等多渠道网络建立联系。然而，这可能是一个比创作者想要的更大的承诺——胡法伊扎说，60%的 MCN 合同包括[永久条款](https://web.archive.org/web/20221205171143/http://readwrite.com/2012/10/19/youtube-networks-an-inside-look-at-their-unsavory-business-practices)。

他说:“(创作者)没有一个独立的平台，可以为他们的内容寻找广告客户。”。

因此，Sponsify 应该填补这一空白，让品牌找到触及理想受众的创作者。然后，他们可以提议活动，如产品评论或代言，YouTube 频道促销，创建病毒视频，以及赞助脸书或 Twitter 消息。

该公司仍处于早期阶段——Huzaifa 说他和他的兄弟 Muhammad Jehanzaib“六、七周前”才开始开发这个平台。这个两人团队目前位于巴基斯坦卡拉奇，计划搬迁至伦敦。它还没有开展任何活动。尽管如此，胡法伊扎说，已经有 100 个频道注册了，总共有超过 1000 万订户，创作者对此很感兴趣。

当被问及透明度和披露是否存在任何问题时，Hufaiza 指出，YouTube 上已经存在类似的活动——但同样，广告商没有有效的方法来找到并直接联系独立创作者。是的，视频的文字描述应该包括某种类型的披露，但他补充说，“这对视频的普通观众来说并不重要。”

至于跟踪这些活动的效果，Hufaiza 说，该公司可以通过使用 YouTube 的 API 提供“大量数据”。

Sponsify 只是最新一家寻求将这类活动引入新的在线形式的初创公司。本月早些时候， [MediaSpike 筹集了 520 万美元](https://web.archive.org/web/20221205171143/https://beta.techcrunch.com/2013/11/04/mediaspike-series-a/),将植入式广告引入手机和社交游戏。
