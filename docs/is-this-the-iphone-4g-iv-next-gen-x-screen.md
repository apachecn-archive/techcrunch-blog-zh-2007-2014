# 这是 iPhone 4G IV Next-Gen X 屏幕吗？

【YouTube http://www.youtube.com/watch?v=sKrLnHQ8d3U&hl=en_US&fs=1&]

[在别处](https://web.archive.org/web/20221004155457/http://www.nowhereelse.fr/ecran-iphone-4g-piece-detachee-30959/)发现了这个令人愉快的视频，视频中有人打开了[iPhone](https://web.archive.org/web/20221004155457/http://crunchgear.com/tag/iphone)“4G”屏幕，大概是发给他们的同一个人给 [iResq](https://web.archive.org/web/20221004155457/http://www.nowhereelse.fr/iphone-4g-2010-photos-facade-29143/) 发了东西。

屏幕基本上看起来像标准的 [iPhone 3G](https://web.archive.org/web/20221004155457/http://crunchgear.com/search/iphone%203g) 屏幕，但更长更薄。这可能是一个原型，一个错误，或者是一家中国公司出于未知目的制造的测试屏幕。然而，如果你手边有荧光棒的话，伴随视频的音乐会让你的早晨充满活力。
