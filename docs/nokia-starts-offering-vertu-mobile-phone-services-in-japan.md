# 诺基亚开始在日本提供 VERTU 手机服务

![nokia_japan](img/f7275661bbb39e98f3e592ea5c6844db.png "nokia_japan")

对于许多非日本手机制造商来说，日本是一个很难打开的市场，甚至世界领先的手机制造商诺基亚在未能在这个国家站稳脚跟后也不得不放弃其日本业务(去年 11 月)。但是他们再次尝试，这次是用他们的奢侈品牌 Vertu 。

成为日本 VERTU 俱乐部的会员有很多好处。例如，用户可以打电话给门房，门房会 24 小时负责餐厅或酒店预订等事宜(用英语或日语)。另一项被称为“堡垒”的服务，可以将电话号码或地址等数据存储在英国一个地下掩体中的服务器上。

这个废话的价格:每月 560 美元，但你可以免费拨打 20 小时的国内电话。我看不出诺基亚会有什么进展，尤其是在当前的经济形势下，但我可能是错的。
