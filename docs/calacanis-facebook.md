# 不知何故未能删除脸书账户的男子抱怨新功能

等一下。似乎每个人都忽略了最近[关于脸书的虚假争议](https://web.archive.org/web/20221209225953/https://beta.techcrunch.com/2010/10/07/conspiracies-privacy-spam-and-nambla-facebook-groups-are-fun/)的一个关键因素。杰森·卡拉卡尼斯回到脸书做什么？

卡拉卡尼斯今天发布了一封发给脸书首席执行官马克·扎克伯格和首席运营官·雪莉·桑德伯格的电子邮件。在这篇文章中，他指责社交网络没有对其新的群组功能采取选择加入政策。但是等等——他怎么会知道新功能是如何工作的呢？他不是又说了一遍[比](https://web.archive.org/web/20221209225953/http://calacanis.com/2010/06/04/steve-jobs-mark-zuckerberg-and-serpico/)重要[比](https://web.archive.org/web/20221209225953/https://beta.techcrunch.com/2010/06/04/the-mythical-sheryl-sandberg-wink-and-smile-starring-al-pacino/)重要[比](https://web.archive.org/web/20221209225953/https://beta.techcrunch.com/2010/06/05/facebook-steve-jobs-dog-day-afternoon/)重要吗，说他和脸书已经结束了？

他删了自己的账号——视频上的[甚至](https://web.archive.org/web/20221209225953/http://www.youtube.com/watch?v=C193b8KQCk0)！然而，几个星期后，他仍然在那里。起初[他让这听起来像是](https://web.archive.org/web/20221209225953/https://beta.techcrunch.com/2010/06/11/calacanis-facebook-profile/)一个第三方网站重新激活了他的账户，但随后他澄清说他已经登录脸书连接来清理一些东西，并相信是重新激活了他的账户。[脸书说那不是真的](https://web.archive.org/web/20221209225953/https://beta.techcrunch.com/2010/06/11/facebook-calacanis-is-lying/)，他肯定主动取消了删除请求。卡拉卡尼斯在那之后就安静了。

我们本以为，既然他如此坚持，他会立即再次删除他的账户。但显然不是。正如你现在看到的，大约一个月后，也就是八月，他把这个墙贴留给了他的追随者:

> 注意:如果你偶然发现我的脸书页面，它处于混合模式。我只是不相信脸书的人会给我数据。

他对脸书如此偏执，但他决定保持他的账户活跃？奇怪。

更奇怪？几周后，他又在他的脸书墙上发了一个帖子。这一次，链接到他最新的咆哮。

所以很明显，卡拉卡尼斯已经和脸书断绝关系了，除非这能帮助他宣传他的内容。或者当它成为新内容的素材时，正如我们今天所看到的。

我们同意，人们可以被添加到任何组是相当蹩脚的。但这不是隐私问题——卡拉卡尼斯(和迈克的)[自己的朋友把他们加到了这个群](https://web.archive.org/web/20221209225953/http://searchengineland.com/irony-facebooks-new-groups-give-me-less-control-not-more-52495)！它之所以站不住脚，是因为不知不觉中被添加到某个群中会导致数量惊人的垃圾邮件。

但这不应该是卡拉卡尼斯遇到的问题。他几个月前删除了他的脸书账户，记得吗？
