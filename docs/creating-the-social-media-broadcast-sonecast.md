# 创建社交媒体广播(Sonecast) | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2008/03/21/creating-the-social-media-broadcast-sonecast/>

[![sonecast-logo-1.png](img/323c58537bd6e9cf8618f44c2c211fee.png)](https://web.archive.org/web/20230316035240/http://www.crunchbase.com/company/sonecast) 在社交媒体时代——当每个人都忙于在脸书、MySpace、YouTube、Twitter、Flickr、博客、Justin.tv 等网站上创建个人广播时，广播的概念会发生什么变化？我说的不是电视或广播，传统的广播仍然以它们一贯的方式运作。我说的是在网上广播一个事件或一条消息。在一个任何人都可以按照自己的时间表阅读、观看或收听任何信息的窄播世界里，广播的概念还有意义吗？如果真的有意义，你如何将个人网络媒体整合成一个连贯的体验，而不是以自上而下的方式制作一些东西，发布给不再被动的观众？

这些都是 Tola Oguntoyinbo 一直在努力解决的问题。Tola 是 Sonecast(T4)的首席执行官和联合创始人，这是一家位于北卡罗来纳州教堂山的基本社交媒体营销初创公司，由信用卡资助。他的 sonecasts(社交网络广播)将几乎任何来源的社交媒体拉进一个专门的品牌网站。最接近它的可能是 [Meebo Rooms](https://web.archive.org/web/20230316035240/https://techcrunch.com/2008/01/31/meebo-turns-chat-rooms-into-a-web-service/) 和 [SplashCast](https://web.archive.org/web/20230316035240/http://web.splashcast.net/) 的组合(本周早些时候刚刚筹集了 400 万美元)。Sonecasts 往往在现场活动中工作得更好，就像这个 DJ 幽灵般的演唱会，或者这个他在事后开始的嘎吱嘎吱的音乐。

[![sonecast-crunchies-small.png](img/baf1ae92b06bfdbfee471e17c9282bc6.png)](https://web.archive.org/web/20230316035240/https://techcrunch.com/wp-content/uploads/2008/03/sonecast-crunchies-vid.png "sonecast-crunchies-vid.png")

在 Crunchies sonecast 中，页面上有一个浮动框架，带有小标签，可以点击显示 Flickr 照片、YouTube 视频和活动博客帖子的缩略图。还有一个直播标签，可以显示颁奖典礼期间来自 Mogulus 的直播视频。你点击的每一张照片、视频或博客标题都会打开一个新窗口，让你对每一项发表评论。DJ ghost one 直播了杜克大学一场音乐会的视频，该视频现已存档，仍可观看。其他标签包括来自 Flickr 和脸书的照片、DJ 幽灵视频、音乐和来自 MySpace 的帖子、来自该活动的播客采访，以及一个功能齐全的音乐小工具，其中包含完整的流媒体歌曲、音乐视频和音乐博客上关于这位艺术家的帖子的链接。

请注意，这些是演示网站，它们加载有点慢，可能无法处理大量的流量。(该服务仍处于封闭的 alpha 阶段)。但是用户界面采用的方法是有指导意义的。它既经济又包罗万象。虽然它出于营销目的利用社交媒体，但它并没有试图按照 Mzinga 或 Networked Insights 的路线创建一个公司化的社交网络。

“我们实际上是在利用现有的内容网络，”Tola 说。他区分了“以自我为中心”的社交网络和“以对象为中心”的社交网络，在“以对象为中心”的社交网络中，人们组织的“对象”是照片、视频和链接。相比之下，以自我为中心的社交网络，如脸书或 MySpace，是围绕用户档案组织的。他说，这是关于“人们上网还是内容上网”。

sonecasts 可以将专业制作的视频或其他广播材料与观众制作的关于同一事件的相关材料结合起来。Flickr 照片流、Twitters 和 YouTube 视频都可以很容易地整合到 sonecast 中。这个想法是围绕事件创造社会活动。Sonecasts 也可以围绕品牌或产品进行创作。例如，Pleo 玩具恐龙的 sonecast(点击下面中间的虚拟屏幕截图)可能包括爱好者的 Flickr 照片、Pleo 博客的帖子或 Pleo 被黑客攻击的视频。

Sonecast 是一个收集产品或品牌生命流的地方，如果你愿意的话，就像人们开始从网络上收集自己的生命流到像 [FriendFeed](https://web.archive.org/web/20230316035240/https://techcrunch.com/2008/02/25/friendfeed-raises-5-million-now-open-to-everyone/) 这样的服务中一样。除此之外，还有广播部分——将人们吸引到 Sonecast 的最初的、引人注目的内容。这是最大的挑战:首先让人们去参加 Sonecasts，并确保他们有事情可做。

一个很好的起点是在网络上整合其小部件，以吸引人们回到每个完整网站的丰富体验中。(相比之下，SplashCast 只做小部件。同时拥有窗口小部件和目标页面可能是一个优势)。它还需要更好地显示谁在 Sonecast 中以及他们在那里做什么。但它确实让你看到了以一种完全不同的方式进行广播的可能性。
