# 新的应用程序使用艾森豪威尔的方法来完成你的事情

有很多“把事情做完”的应用程序，比如 [Things](https://web.archive.org/web/20230321064317/http://culturedcode.com/things/) 和 [Priority Matrix](https://web.archive.org/web/20230321064317/http://www.appfluence.com/) ，它们要么专注于 GTD，要么不依赖于时间管理哲学。还有一些像柏林的热门创业公司 [Wunderlist](https://web.archive.org/web/20230321064317/http://wunderlist.com/) 来自 [6wunderkinder](https://web.archive.org/web/20230321064317/http://www.6wunderkinder.com/) 采用非常以设计为导向的方法。

现在自举 [Eisenhower.me](https://web.archive.org/web/20230321064317/http://eisenhower.me/) 基于“Eisenhower matrix”时间管理原则推出了它的 [iPhone 应用程序](https://web.archive.org/web/20230321064317/http://itunes.apple.com/app/eisenhower/id547099449)，带有相应的网络客户端。这款软件售价 1.99 美元，可以将你的任务与免费的网络应用程序同步。

最初的艾森豪威尔四象限矩阵由于其不太友好的移动外形而没有在移动应用程序中得到广泛应用，但这些人似乎已经提出了一个巧妙的解决方案。

创始人安德烈亚斯·科维亚特科夫斯基和蒂姆·布吕克曼在德国科隆发布了这款应用。请在评论中告诉我们你的想法。
