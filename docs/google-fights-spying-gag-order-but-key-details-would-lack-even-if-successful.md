# 谷歌对抗间谍封口令，但即使成功，关键细节也会丢失

正如它所承诺的那样，谷歌正在对抗政府关于公布多少用户受到国家安全局监控的禁令。与脸书和微软不同，[谷歌和推特公开拒绝了](https://web.archive.org/web/20230329172209/http://www.theverge.com/2013/6/15/4432368/google-opts-out-of-fisa-disclosure-deal-made-by-facebook-and)一项披露用户数据间谍授权总数的政府协议，这将包括(但不详细)来自有争议的外国情报监视法庭(FISA)的请求数量。

“将国家安全请求与犯罪请求混为一谈——就像一些公司被允许做的那样——对我们的用户来说是一种倒退，”[在请愿书后的一份公开声明中解释道。](https://web.archive.org/web/20230329172209/https://plus.google.com/+google/posts/eV7cNmQPqKU)

不幸的是，[正如我](https://web.archive.org/web/20230329172209/https://techcrunch.com/2013/06/11/why-googles-request-to-release-more-spying-data-is-probably-an-empty-pr-gimmick/)和[华盛顿邮报所暗示的那样](https://web.archive.org/web/20230329172209/http://www.washingtonpost.com/business/technology/google-challenges-us-gag-order-citing-first-amendment/2013/06/18/96835c72-d832-11e2-a9f2-42ee3912ae0e_story.html)，即使谷歌成功了，最紧迫的问题仍然是个谜。谷歌的透明度报告披露了法院命令和受影响用户的数量，但没有透露哪些数据被放弃。政府能像泄密者爱德华·斯诺登声称的那样阅读电子邮件、监控 Gchats 和谷歌语音电话吗？

此外，如果政府真的可以[要求大范围的数据，比如搜索日志](https://web.archive.org/web/20230329172209/http://www.nytimes.com/2013/06/08/technology/tech-companies-bristling-concede-to-government-surveillance-efforts.html?pagewanted=all&_r=1&)，受影响的用户数量可能数以百万计。公布受影响用户的总数相当于披露重要的监控来源和方法。

请愿书引用了他们的第一修正案权利，指出“谷歌的声誉和业务受到了媒体虚假或误导性报道的损害……谷歌必须用比泛泛而谈更多的话来回应这些指控。”

有理由乐观地认为，允许谷歌详述 FISA 的请求将有助于修复其声誉。脸书[报道](https://web.archive.org/web/20230329172209/https://newsroom.fb.com/News/636/Facebook-Releases-Data-Including-All-National-Security-Requests)在 9000-10000 个政府请求中，只有 18000-19000 个用户受到影响。这似乎让人怀疑，一个政府的请求是否允许对整个人口的活动进行大规模监控。因此，虽然我们不知道有什么被泄露了，但大多数用户可以放心了，他们不是目标。

我同情谷歌的立场；当然，他们可能想披露一切，或者干脆停止窥探。但是，即使在最好的情况下，公众仍然一无所知。
