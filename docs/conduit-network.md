# Conduit 报告的数字很大:25 万个应用发行商，1.7 亿用户

![](img/7309841d6544bf6c850c0a0b36a096a1.png)

[Conduit](https://web.archive.org/web/20230306011740/http://www.conduit.com/) ，由[风投支持的](https://web.archive.org/web/20230306011740/http://www.crunchbase.com/company/conduit)公司，使出版商能够通过浏览器工具栏在网络上创建和分发应用程序，今天公布了一些大数字。

这家初创公司表示，其网络现在拥有超过 250，000 家网络出版商，他们能够接触到大约 1.7 亿用户。该公司补充说，每秒钟大约有 23 个新用户安装 Conduit Engine。

这些数据让 Conduit 总裁亚当·博伊登(Adam Boyden)对其相对于移动应用市场的规模做出了一些大胆的声明:

> “虽然媒体的大部分讨论都是关于移动应用，但现实情况是，使用 Conduit 驱动的应用的人数比 iPhone 和 Droid 应用的总和还要多 1.18 亿。”

这不完全是苹果之间的比较，但它确实表明 Conduit Network 已经悄悄地成为许多大品牌的分销渠道的巨头，包括亚马逊、零度可乐、易贝、脸书、福克斯新闻频道、Hulu、Skype、YouTube 和 Zynga。

根据 Conduit 的说法，它的[应用市场](https://web.archive.org/web/20230306011740/http://apps.conduit.com/)(可以把它想象成一个通过跨浏览器工具栏提供的网络应用的[应用商店)增长显著，尤其是。该公司表示，每天有 6000 万终端用户使用市场上的应用*，这不仅让主要品牌受益，也让不太知名的软件开发商受益。*](https://web.archive.org/web/20230306011740/https://techcrunch.com/2009/06/23/conduit-creates-marketplace-for-toolbar-content/)

 *自 2009 年 8 月以来，Conduit 声称拥有超过 100 万用户的网络出版商的数量已经从 1 家增长到 2010 年 5 月的 40 多家。

在这种规模下，Conduit 能够提供一些有趣的 Web 应用使用趋势，这正是他们为我们提供的。全面披露:请注意，在 Conduit 的市场[的某个地方](https://web.archive.org/web/20230306011740/http://apps.conduit.com/search?q=techcrunch)有一个 TechCrunch 应用程序，该公司称之为。

逐字引用[新闻稿](https://web.archive.org/web/20230306011740/http://www.conduit.com/Home/AboutUs/press-releases/Conduit-Network-Grows-to-More-Than-250,000-App-Pub.aspx)中的广告:

> 娱乐应用是 Conduit 应用市场中数百个垂直领域中最大的部分；自 marketplace 推出以来，24.4%的应用程序是在这一类别中创建的。
> 
> 终端用户也渴望在线新闻，应用发行商也做出了回应；市场上 19.5%的应用提供来自福克斯新闻频道、iVillage 和 TechCrunch 等来源的新闻。
> 
> 社交媒体应用的数量紧随其后，占所有可用应用的 18.2%。同样值得注意的是，社交媒体应用类别在去年的最终用户中增长最快，采用率平均每月增长 87%。
