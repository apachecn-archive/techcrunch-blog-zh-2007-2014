# TC 婴儿床:Tumblr 值得再博客的内部

我们带着新一集的 [TC Cribs](https://web.archive.org/web/20230205012026/http://www.techcrunch.tv/show/tc-cribs) 回来了，这一集肯定会受到大众的欢迎:火热的博客平台 [Tumblr](https://web.archive.org/web/20230205012026/http://www.tumblr.com/) 。这家总部位于纽约的初创公司一直在疯狂增长(他们现在每月的页面访问量高达 116 亿次)，他们给了我们一个窥探他们总部内部的机会。

对于一家拥有超过[4000 万美元](https://web.archive.org/web/20230205012026/http://www.crunchbase.com/company/tumblr)资金的公司来说，Tumblr 团队仍然小得惊人，办公室也相对简陋。但这里有很多魅力:手工家具、桌游比比皆是，还有看起来可疑地像我最喜欢的巫师的机器人。

一定要看完，当我遛 Tumblr 创始人戴维·卡普的狗的时候。啊？

一如既往，感谢 Ashley Pagán 和 John Murillo 的摄影工作，感谢 Murillo 先生出色的剪辑。
