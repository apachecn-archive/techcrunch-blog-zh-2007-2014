# HouseFix 是家装公司 Carfax，它想为你找到最好的本地承包商(视频)

上个月在旧金山的 TechCrunch Disrupt 上推出了一家名为 [HouseFix](https://web.archive.org/web/20230203124200/http://www.housefix.com/) 的初创公司。正如你可能会猜到它的名字，HouseFix 是一个创业和服务，旨在帮助你找到合适的人来修理你的房子。这家初创公司由房主和承包商发起，试图让寻找熟练工人进行房屋维修的过程对双方都更有利。为了做到这一点，HouseFix 希望为房主提供一份关于他们房屋的 CARFAX 报告——一份关于他们房屋的完整历史记录，其中包括一份改进项目列表或未来房主可能会发现的任何相关内容。

为了区别于其他供应商，如 Angie's List 或 ServiceMagic，HouseFix 以简单直观的方式将社会和当地背景带给房主，以便他们可以快速找到符合他们个人标准的当地承包商，了解他们需要什么样的劳动力、价格等。创业公司希望你能在几分钟内找到你可以信任的承包商，而不是几天或几周。

因此，HouseFix 希望成为一个全面的承包商名录，包括每个承包商的评论和简介，业主可以很容易地搜索到。用户将能够看到他们的哪些邻居正在使用该网站，他们最近使用了哪些承包商，除了社交推荐，为了帮助房主克服噪音，HouseFix 采用了一系列不同的因素，包括性能、可靠性等。来产生。每个承包商都有一个分数，就像信用分数一样，这样业主就知道他们可以信任谁。([你可以在这里阅读 Jason 之前对 HouseFix 的报道](https://web.archive.org/web/20230203124200/https://techcrunch.com/2011/09/12/housefix-bringing-social-recommendations-and-accountability-to-home-improvement/)。)

另一方面，HouseFix 也希望成为这些承包商的资源，为他们提供平台，让他们可以跟踪自己的项目，并将线下业务带到线上。有了 HouseFix，承包商可以管理他们的工作，查看哪些工作按时完成且在报价预算范围内，存储注释以及之前和之后的照片，并最终允许他们建立一个在线配置文件，用于创建和管理他们自己的网站。

与 ServiceMagic 相比，HouseFix 目前是免费测试版，不收费。ServiceMagic 为房主提供建议——三个电话号码——但提供的承包商信息很少。HouseFix 总裁兼首席执行官阿德里安·塔拉潘(Adrian Talapan)表示，作为一名房主，他希望找到一名承包商，该承包商曾在他们所在的社区建造过与他们类似的房子，并参与过与他们正在寻找的项目类似的项目，而且工作出色。

对于承包商来说，一般来说，他们最终会成为第九个或第十个就工作请求联系房主的服务提供商，到那时，房主不会接电话，或者已经找到了他们想要使用的人。这对承包商来说是浪费时间，很少有服务给他们机会有效地推销自己，分享他们最擅长的工作，以找到合适的工作。
