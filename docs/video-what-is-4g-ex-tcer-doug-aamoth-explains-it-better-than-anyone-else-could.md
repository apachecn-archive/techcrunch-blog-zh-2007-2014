# 视频:4G 是什么？前 TC'er 道格阿莫思解释得比任何人都好

我每周至少会被问几次同样的问题:4G 到底是什么？“4G”对所有运营商来说都是一个意思吗？

我一直想做一个快速而幽默的视频来解释整个混乱，但只是一直推迟。我现在意识到了原因:在我不知道的情况下，我的大脑在等待 TechCrunch 校友道格·阿莫思去做这件事。时间到了，大脑——时间……到了。

对于那些对这些地区不太熟悉的人来说，他们可能不记得以前的 TechCrunch，道格在我们的姐妹网站 CrunchGear 上为我们写了文章。在 AOL 母舰抵达并开始向我们提供这些东西之前不久，他被像*金钱*和*健康保险*这样的琐碎小事的诱惑之歌召唤到[的另一个站点](https://web.archive.org/web/20230205013933/http://techland.time.com/)。更多美好时光，请点击这里查看道格的经典[企鹅汽水机评论](https://web.archive.org/web/20230205013933/https://techcrunch.com/2008/10/16/review-penguin-home-soda-maker/)。想你，伙计。
