# iTunes 播放电影停滞了？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/08/itunes-replay-for-movies-stalled/>

苹果一直计划推出一种新的基于云的电影流媒体服务，这种服务允许你将购买的电影重新下载到其他设备上，而无需额外付费。这项服务是“iTunes Replay”的一部分，据[多家](https://web.archive.org/web/20230204233213/http://news.cnet.com/8301-31001_3-20067614-261.html?tag=mncol;txt)T2 报道，最初计划在“未来几周”内推出。

但现在看来，情况不会是这样的。

电影下载服务将类似于苹果电视上新增加的功能，它允许你播放你之前用 iTunes 账户购买的任何电视剧集。这些节目出现在桌面和移动设备上 iTunes store 的“已购买”部分，以及 Apple TV 本身。

在通过苹果电视最近的软件更新添加了这一功能后，对电影的支持有望指日可待。

不幸的是，这种支持可能需要很长时间。

[![](img/b4eb342fa1e2d1037144a008696bbc79.png "apple_tv_4_3_purchased")](https://web.archive.org/web/20230204233213/https://techcrunch.com/wp-content/uploads/2011/08/apple_tv_4_3_purchased.png)

据 CNET 援引多个电影行业消息来源称，苹果尚未与六大电影公司中的四家签署协议，谈判可能会持续数月。

其中一个问题与“HBO 窗口”有关，即电影发行 DVD 后的一段时间，在此期间 HBO 拥有电影的独家电子发行权。HBO 与六大电影公司中的三家达成了协议，即 20 世纪福克斯、环球和华纳兄弟，这意味着只有 HBO 可以在指定的时间框架内播放他们的电影。iTunes 无法同时提供这些电影的流媒体访问。

虽然这样的封锁期不一定会阻止苹果推出基于云的电影流媒体服务，但它会降低这项服务对消费者的吸引力。然而，这不会阻止苹果与 HBO 以外的电影公司签署协议:迪士尼、派拉蒙和索尼影业。苹果可以选择推出混合产品，其中一些工作室版本先于其他版本推出，即使它们同时推出 DVD。
