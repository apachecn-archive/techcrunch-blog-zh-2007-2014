# AIM 上的脸书大受欢迎:100 万用户

二月份，AOL 增加了给你在 AIM 上的脸书朋友发消息给 T2 的功能，给脸书用户提供了一个很好的独立客户端来和他们在脸书的朋友聊天。AOL 告诉我，从那以后，已经有 100 多万人使用 AIM 上的脸书连接来访问脸书的朋友列表，所有这些都是 AIM 客户端的新安装版本。

根据康姆斯克的数据，这大约是 AIM 每月 2100 万用户的 5%。这些数字应该会大幅上升——最近 AOL 还增加了用你的脸书证书直接登录 AIM 的功能。

AOL 在三月份发布的 Lifestream 产品上也取得了成功。社交网络聚合器内置在他们的 iPad 应用程序中，这是目前仅次于 Twitterific 的第二大社交网络 iPad 应用程序。AOL 表示，很快，用户也可以使用他们的脸书凭证登录 AIM iPad 应用程序。

上个月晚些时候，我们采访了负责 AIM 和其他社区产品的 AOL 高管布拉德·加林豪斯(Brad garling house )( T6 ),谈论他对聚合社交内容的愿景。
