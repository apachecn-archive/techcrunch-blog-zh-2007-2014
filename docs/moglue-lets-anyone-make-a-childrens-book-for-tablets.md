# Moglue 让任何人都能为平板电脑制作一本儿童书籍

iPad 的崛起(以及在较小程度上其他平板电脑的崛起)导致了无数新类型的应用蓬勃发展。而且，作为一个仍然喜欢不时翻阅一本好的苏斯博士书籍的人，我最喜欢的趋势莫过于那些丰富的、互动的、正在流行的儿童书籍。

这些书通常以音乐、音效、一些动画和其他漂亮的细节为特色，让孩子们玩起来更有趣(也可以补充学习)。有很多人可以写一本好书并为其配插图，但对 Objective C 编程了解不多。这就是 TC Disrupt 决赛选手 [Moglue](https://web.archive.org/web/20230204190536/http://www.moglue.com/) 的用武之地:它让任何人都可以使用简单明了的用户界面创作儿童书籍。

显然，如果你不擅长画画，你的书不会看起来很棒，但对于所有希望转向平板电脑的艺术家来说，或者任何想为孩子制作一本带有家庭照片的定制故事书的人来说，这似乎是完美的。可以将文本和图像拖放到屏幕上，然后使用多种不同的效果制作动画。

这些应用程序使用适用于 Mac 和 Windows 的客户端构建，可以“一键”发布到 iOS 和 Android。该构建器可以免费使用，Moglue 将通过向出版其书籍的用户收取 rev-share 来赚钱。
