# CloudFlare 希望成为面向大众的 CDN(只需五分钟即可完成设置)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/09/27/cloudflare-wants-to-be-a-cdn-for-the-masses-and-takes-five-minutes-to-set-up/amp/>

[<amp-img class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" src="https://web.archive.org/web/20230128140437im_/https://techcrunch.com/wp-content/uploads/2010/09/cloduflarelogo.png" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230128140437/http://www.cloudflare.com/)众所周知，性能是网站成功的重要因素——让你的用户等着，他们会不耐烦，转而关注其他网站。有一些解决方案可以帮助保持速度，比如 cdn，但是大多数小网站都不使用它们。TechCrunch Disrupt 决赛入围者 [CloudFlare](https://web.archive.org/web/20230128140437/http://www.cloudflare.com/) 希望将这些快速加载时间带给大众，它还提供了一些其他好处，包括针对在线威胁的强大安全保护。首席执行官 Matthew Prince 说，简而言之，CloudFlare 让普通的 web 管理员成为一个成熟的运营团队。

普林斯说，速度问题会对你的网站产生很大影响——一项研究表明，每 100 毫秒的加载时间，你就会失去高达 2%的访客。他说 CloudFlare 提供了平均 30%的速度提升，可以“阻止几乎所有的网络垃圾邮件攻击”。他说你可以在大约五分钟内将它整合到你的网站中。哦，它是免费的，至少它的基本服务是免费的。

Prince 说，CloudFlare 运行在网络级别，因此它支持任何平台。设置包括更改您的 DNS 以路由到 CloudFlare 的服务器。在您的站点上设置了 CloudFlare 之后，您可以前往控制面板，该面板显示已经为用户提供了多少数据请求，以及 CloudFlare 为您节省了多少带宽。这也使得谷歌分析很容易下降

CloudFlare 还将提供一个“Pro”计划，增加 SSL、更好的页面优化和对象预取等功能，以进一步提高速度。该公司在三大洲建立了五个数据中心。到目前为止，它一直处于私人测试阶段，并在 1000 个网站上进行了测试，这些网站为 600 万独立访客提供了服务。
