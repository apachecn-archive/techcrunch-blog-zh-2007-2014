# 创业加速器 TechStars 在博尔德孵化 12 家新公司

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/04/startup-accelerator-techstars-hatches-12-new-companies-in-boulder-heres-a-peek/>

今天，种子基金和加速器 [TechStars](https://web.archive.org/web/20230203140926/http://www.techstars.org/) 将在博尔德举办“演示日”，展示其 12 家最新的创业公司。加速器预计将有超过 500 人到场，包括投资者和风险基金，他们将到场审查年轻的创始团队，并可能拿出那些支票簿。

TechStars 于 2006 年在博尔德成立，现已将其业务扩展到纽约市、波士顿和(最近)西雅图，加速器为 10 至 15 家初创公司提供季节性项目，以 6%的股权向每个团队投资高达 18，000 美元的种子资本。而且，到目前为止，TechStars 已经证明了其对创业者的价值；根据 [TechStars 的网站](https://web.archive.org/web/20230203140926/http://www.techstars.org/results/)，81%的校友仍然活跃或盈利，10%已经被收购，8%已经倒闭。

昨天，TechStars 宣布了一部[即将上映的电视真人秀](https://web.archive.org/web/20230203140926/https://techcrunch.com/2011/08/02/startups-on-tv-techstars-teams-up-with-bloomberg-to-offer-a-look-inside-the-startup-hustle/)，该剧将跟随最近一批纽约公司经历作为加速器计划一部分的考验、磨难和转折。许多 TechStars 的纽约导师，如 Foursquare 的丹尼斯·克劳利(Dennis Crowley)和 Union Square Ventures 的弗雷德·威尔逊(Fred Wilson)，都会客串，这无疑会成为一些不错的创业电视节目。

然而，今天的演示日是关于博尔德的创业公司，所以废话不多说，让我们来看看今天在科罗拉多州成立的 12 家创业公司:

[![](img/b3ab91b36963ab1b275b041b987dec99.png "Print") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/creativebrainstudios.jpg) [创意大脑工作室](https://web.archive.org/web/20230203140926/http://creativebrain.com/)是一个游戏开发引擎，解决了游戏工程师面临的许多常见问题。他们的交钥匙解决方案提供了轻松的音频和视频文件管理，整合了游戏播放、分析、托管和货币化，并利用 HTML5 发布一次游戏，并将其部署到智能手机、平板电脑和台式机上。

[![](img/82eacc4ecceaa4af334845586f33c9e0.png "Flextrip")](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/flextrip.png) 由于旅游和活动没有一个成熟的分销系统，在线旅游公司正在失去一个价值 200 多亿美元的机会。 [FlexTrip](https://web.archive.org/web/20230203140926/http://www.flextrip.com/) 向在线旅游公司提供其 API，以便从旅游和活动这一最赚钱的领域中获利。这家初创公司声称拥有“世界上最大的旅游和活动选择”以及独特的推荐引擎，以确保旅行者找到最适合他们的活动。FlexTrip 已经完成了一些很酷的集成，[包括一个与 TripIt 的集成](https://web.archive.org/web/20230203140926/http://www.tripit.com/uhp/tools)。

[![](img/0f4a8e5b9373d3259b80f97121e25e40.png "FlixMaster") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/flixmaster.jpg) [FlixMaster](https://web.archive.org/web/20230203140926/http://www.flixmaster.com/) 创造了一种技术，让任何人都可以构建他们所谓的“分支路径视频混搭”。这使得人们可以从整个网络上获取视频、音频文件、表格和图像，使观众能够通过不同的路径浏览视频——所有这些都在视频播放器中进行。他们的分支路径视频混搭为消费者提供了更丰富、更吸引人的体验。

[![](img/fe3409c32d07c0b0f10ce41359902188.png "FullContact") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/fullcontact.png) [FullContact](https://web.archive.org/web/20230203140926/http://fullcontact.com/) 是一个保持联系信息最新的 API。开发人员可以轻松地利用 FullContact 超过 1 亿条联系人记录的数据库，快速将部分过时的联系人记录转换为全新的完整联系人记录。他们使用众包模式来更新联系信息，只分享网络上公开的联系信息，不像 Plaxo 和 Jigsaw 这样的竞争对手，他们允许任何软件应用程序通过他们的 API 利用他们的联系数据库。

[![](img/2f2611d4dd2c4a6928b32316a3e38570.png "GoSpotCheck") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/gospotcheck.png) [GoSpotCheck](https://web.archive.org/web/20230203140926/http://www.gospotcheck.com/) 是众包秘密购物的品牌。他们通过智能手机为品牌提供实时洞察，如价格检查、货架定位、照片和店内展示视频。购物者只需在他们已经去过的商店完成这些简单的任务，就可以获得现金。与允许一系列活动的 GigWalk 不同，GoSpotCheck 为零售环境中的品牌提供定制解决方案。

[![](img/a57d9779e1b518a04ad16d60f97203a4.png "inboxfever") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/inboxfever.png) [InboxFever](https://web.archive.org/web/20230203140926/http://www.inboxfever.com/) 是一个构建和销售电子邮件应用的平台和市场。电子邮件驱动的应用程序是一种新型的应用程序，可以帮助人们在网络上更快地完成普通任务。它们让你只需发送一封电子邮件就能控制你的网络应用——无需登录，无需新的用户界面，无需下载。测试它，把任何 word 文档发送到 pdf@inboxfever.com，看着它在几秒钟内以 PDF 格式返回给你。

[![](img/3822f6c256c3fec5b8e8065976a4eaec.png "MealTicket") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/mealticket.jpg) [MealTicket](https://web.archive.org/web/20230203140926/http://www.getmealticket.com/) 为餐饮配送商提供了一个移动社交网络平台，让他们能够更好地与自己的餐厅合作。他们通过餐饮经销商传递给他们餐厅的交易和促销社交营销工具来做到这一点。交易和促销通过 MealTicket 的移动应用程序、Twitter 和脸书应用程序分发，最重要的是，他们的 API 合作伙伴确保优惠在许多网络渠道分发。餐饮服务分销商和餐厅可以合作创建交易和促销，最终保护餐厅的底线，利用餐饮服务分销商的销售力量，并使餐厅能够利用鲜为人知的社交媒体专业知识来利用社交网络。

[![](img/d8de7a533e8eb585f267aa8ba3c4a8b9.png "mocavo")](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/mocavo.png) 据创业公司介绍， [Mocavo](https://web.archive.org/web/20230203140926/http://mocavo.com/) 是“第一个也是最大的免费家谱搜索引擎”。以前，系谱学家和家族史爱好者不得不访问几十个不同的网站进行搜索，或者支付昂贵的月费来寻找有限的内容。今天，人们可以在 Mocavo.com 开始免费的祖先搜索。

[![](img/f19c30ffdde06ec13967910854bc10da.png "ReportGrid") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/reportgrid.png) [ReportGrid](https://web.archive.org/web/20230203140926/http://www.reportgrid.com/) 提供了一个 API，让 SaaS 厂商和技术平台可以轻松地为他们的客户提供深度分析和有洞察力的报告。与普通分析工具不同的是，该 API 嵌入到 SaaS 供应商的产品中，允许可视化和分析成为向客户提供的产品的一部分。

[![](img/701065a609d32389b8be7ef77b74b8a2.png "SimpleEnergy") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/simpleenergy.png) [简单能源](https://web.archive.org/web/20230203140926/http://www.simpleenergy.com/)通过改变公用事业公司吸引客户的方式，正在改变数百万人节约能源的方式。他们的网络和移动应用程序使用游戏机制来比较朋友和邻居的实际能源使用情况，使能源效率变得社交化、有趣和简单，同时解决了公共事业部门每年 110 亿美元的问题。

[![](img/3056db840b27dd6508ddca43b679b216.png "SocialEngine") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/socialengine.png) [社交引擎](https://web.archive.org/web/20230203140926/http://www.socialengine.com/)通过一系列产品将任何公司变成“社交引擎”。他们的旗舰产品是一个可下载的社交网络平台，目前有超过 7000 名客户在使用。他们工具集的最新成员是一个推荐引擎，它给出了关于发布什么、关注谁、阅读什么、回复什么的战术和实用建议——实际上把任何人都变成了社交媒体大师。

[![](img/0b44435bce1700429933555e5b737362.png "TruantToday") ](https://web.archive.org/web/20230203140926/https://techcrunch.com/wp-content/uploads/2011/08/truanttoday.png) [逃课今天](https://web.archive.org/web/20230203140926/http://truanttoday.com/)当学生逃课时，立即联系家长，把学生和资金带回教室。目前的逃学产品联系家长太晚，在错误的地点，或在非母语的家庭。因为今天的逃学可以立即联系家长，他们有能力让学生回到教室，并继续在学校提供资金。如今，16 岁的扎克·库考夫(Zak Kukoff)是科技之星(TechStars)有史以来最年轻的创始人(是的，他确实计划在运营这家初创公司的同时重返校园)。
