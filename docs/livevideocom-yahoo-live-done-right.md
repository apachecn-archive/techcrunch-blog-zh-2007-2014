# LiveVideo.com:雅虎直播做得对

MySpace 创始人布拉德·格林斯潘的公司 LiveUniverse 的 LiveVideo.com 是另一个直播流媒体服务，但它结合了现有服务的最佳功能和雅虎直播风格的功能，是一个值得一看的包。

LiveVideo.com 具有以下特点:

*   节目存档，因此用户可以录制节目，以便以后播放
*   嵌入直播流
*   与流相关联的聊天
*   视频会议雅虎直播风格，用户可以添加其他流媒体用户到他们的页面，并与他们互动
*   类似于 YouTube 的评论板/个人资料页面，用户可以在这里发表评论
*   照片库，没有 Flickr，但允许用户分享照片

LiveUniverse 将 LiveVideo.com 定位为“第一个完全互动的全球直播平台它不是，但它是一个功能丰富的产品，很可能会受到青睐。
