# 安桥日本宣布基于 Windows 7 的“平板电脑”

在过去的几个月里，安桥公司已经发布了不少 Windows 操作系统的[平板电脑](https://web.archive.org/web/20230203162409/https://techcrunch.com/tag/tablets)，该公司昨天[宣布了](https://web.archive.org/web/20230203162409/http://www.jp.onkyo.com/news/newproducts/pc/20111116_tw2a/)【JP】又一款。TW2A-A25Z7CK 采用 Windows 7 家庭高级版作为操作系统(32 位)，并配有一支手写笔、一个支架/外壳和一个蓝牙键盘。

安桥所谓的 Slate PC 有以下规格:

*   10.1 英寸液晶显示屏，分辨率为 1024×600
*   凌动 Z670 CPU (1.50 GHz)
*   英特尔 SM35 高速芯片组
*   2GB 内存
*   32GB 固态硬盘
*   IEEE 802.11b/g/n 无线网络
*   蓝牙 2.1+EDR
*   两个 USB 端口，一个 microSD 插槽
*   130 万像素网络摄像头
*   大约 6 小时的电池寿命
*   尺寸:267×173×15.5 毫米，重量:892 克

安桥公司计划于 12 月 2 日在日本开始销售 Slate PC，价格开放(“超过 780 美元”)。
