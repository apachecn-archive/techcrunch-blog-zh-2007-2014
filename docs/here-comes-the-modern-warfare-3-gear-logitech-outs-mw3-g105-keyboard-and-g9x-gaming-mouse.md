# 现代战争 3 装备来了:罗技胜过 MW3 G105 键盘和 G9X 游戏鼠标

收集所有的东西！罗技的最新冒险包括在他们的商品上贴上《现代战争 3》的标志，并以同样的价格出售。是的，你可以显示你对 FPS 的忠诚，而不需要支付额外费用。即将上市的 MW3 G105 游戏键盘和 G9X 游戏鼠标的价格将与标准版相同。这一举措既有利于消费者，同时也相当吓人。

G105 游戏键盘采用背光按键、防重影技术和六个所谓的 G 按钮来执行宏命令。G9x 游戏鼠标并不新鲜。这款鼠标是罗技游戏产品的主要产品，具有可互换的手柄、可更换的重量和板载内存。G105 现在售价 79.99 美元，而 G9x 应该会在下个月初上市。这些价格与标准版惊人地相同，这表明罗技正在走一条危险的道路。一旦你踏上赞助之路，就很难回头。这里希望这个 MW3 设备是一次性的东西，罗技不再计划了。我最不想从罗技看到的就是有 NHL 标志和芝麻街键盘的鼠标。
