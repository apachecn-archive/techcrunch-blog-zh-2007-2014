# Zynga 继续国际扩张；收购德国游戏引擎开发商葡萄糖公司

随着社交游戏巨头收购德国游戏引擎开发商 [Dextrose AG 的资产，Zynga 对世界统治地位的](https://web.archive.org/web/20221217080141/http://www.dextrose.com/)追求今天仍在继续。 Zynga 今年一直在快速发展并不断进行收购，收购了北京的 [XPD](https://web.archive.org/web/20221217080141/https://techcrunch.com/2010/05/20/zynga-enters-asia-with-acquisition-of-gaming-startup-xpd-media-opens-office-in-beijing/) 、东京的 [Unoh Games](https://web.archive.org/web/20221217080141/https://techcrunch.com/2010/08/05/zynga-confirms-unoh-acquisition/) 、波士顿的 [Conduit Labs](https://web.archive.org/web/20221217080141/https://techcrunch.com/2010/08/17/zynga-buys-bostons-conduit-labs/) 以及奥斯丁的 [Challenge Games](https://web.archive.org/web/20221217080141/https://techcrunch.com/2010/06/04/sec-filing-suggests-zynga-paid-at-least-20-5-million-for-challenge-games/) 。

葡萄糖公司为无插件浏览器游戏开发商业游戏引擎。游戏引擎技术被称为“Aves 引擎”，允许开发者使用 HTML、CSS 和 JavaScript 构建任何类型的基于网络的 2D 或 2.5D 游戏。

与该公司之前收购的性质类似，葡萄糖公司团队将成为 Zynga 德国公司(总部位于法兰克福)。这将是该公司在欧洲的第一个工程存在(该公司在都柏林有一个运营办公室)。加入 Zynga 的是来自葡萄糖公司的联合创始人 [Paul Bakaus，](https://web.archive.org/web/20221217080141/http://www.crunchbase.com/person/paul-bakaus)他将成为 Zynga 德国公司的首席技术官。Bakaus 是 jQuery 核心团队成员，帮助创建了用户界面框架 jQuery UI。右旋糖创始人罗科·迪里奥将成为 Zynga 德国公司的国家经理，负责未来浏览器游戏技术开发的所有流程和组织。

当然，凭借丰厚的利润和 T2 的巨额资金(由于软银的投资，Zynga 最近获得了 T4 1.5 亿美元的资金)，Zynga 有足够的现金进行这些收购。该公司正在通过一些高调的合作伙伴关系进行扩张，包括与谷歌、、脸书和雅虎的交易。
