# Twitter 启示不是现在:危机已经避免。大部分是。TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/06/12/twitpocalypse-not-now-crisis-averted/amp/>

 <amp-img class="alignright size-full wp-image-73135 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="picture-101" src="https://web.archive.org/web/20230404030307im_/https://techcrunch.com/wp-content/uploads/2009/06/picture-101.png" alt="picture-101" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 因此，就在推特上的每个人都在[谈论](https://web.archive.org/web/20230404030307/http://twitter.com/timeline/search?q=Twitpocalypse)今天的事情时，一场[推特末日的可能性就在眼前](https://web.archive.org/web/20230404030307/https://techcrunch.com/2009/06/12/all-hell-may-break-loose-on-twitter-in-2-hours/)。跟踪某些第三方 Twitter 应用程序可能失败的[网站](https://web.archive.org/web/20230404030307/http://www.twitpocalypse.com/)认为，危机将在明天一早(美国)发生——但 Twitter 通过今天触发它来预先阻止这个问题，所以所有人都会在甲板上解决这个问题。而且成功了。

“按照计划，我们今天下午通过部署避免了推特末日，”推特联合创始人[比兹·斯通](https://web.archive.org/web/20230404030307/http://www.crunchbase.com/person/biz-stone)告诉我们。好消息。但是这个问题有一个暂时的小伤亡，我被告知 Twitter 搜索已经暂停了大约一个小时。但斯通说，这应该会很快赶上并恢复正常。

所以，现在每个人都可以冷静下来。就像大约 10 年前世界大部分地区都吓坏了的千年虫一样，推特启示录来了又去，没有出现任何问题。

**更新**:正如一些评论者注意到的，我也刚刚证实，流行的 Twitter 应用 Twitterrific 现在因为 API 错误而完全崩溃——至少 iPhone 版本是这样。一些报道称，TweetDeck 正面临类似的问题。
