# 移动企业应用开发商 Taptera 从 Salesforce 天使投资公司筹集了 200 万美元

开发企业 iPhone 和 iPad 应用程序的 Taptera 在由著名硅谷投资者(前甲骨文高管)特伦斯·加内特(Terence Garnett)领导的首轮融资中获得了[200 万美元](https://web.archive.org/web/20230203204651/http://www.crunchbase.com/company/taptera)，他也将加入这家初创公司的董事会。

这轮融资的其他参与者包括 Salesforce.com 和一系列公司首席执行官和专业投资者，包括兰加斯瓦米和前惠普高管大卫墨菲。

Taptera 的第一款应用名为“同事”,是一款移动员工目录工具，将于今年晚些时候上市。

Taptera 团队包括联合创始人兼首席执行官奥孔纳，他之前在基因泰克工作了八年，在那里他领导了该公司通过移动应用增强员工能力的举措。他的联合创始人兼首席产品官丹·麦考尔也是基因泰克的一名兽医。

Taptera 的 CTO 是 Mike Janson，他从现在的投资者 Salesforce.com 加入了这家初创公司，在那里他领导了开发协作工具 Chatter 的团队。

该公司总部设在旧金山。
