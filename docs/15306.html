在你的创业宣传片中不要做什么

【YouTube http://www.youtube.com/watch?v=CLQ1V9OGwKc&w=560&h=315]

感谢我的工作，我看到了很多愚蠢的废话。大多数都被过滤掉了，但是时不时会有一些可笑愚蠢的东西冒出来，它只是乞求被召唤出来。这就是来自Rippln 的好心人的这个宣传视频的情况，这或多或少是一个两分钟的案例研究，关于如何不向公众、媒体、潜在员工或投资者推销你即将成为超级病毒的应用程序。

因此，这里有一个在你的创业视频中不要做什么的步骤:

  • 不要告诉我们你的应用会在发布前就传播开来。[0:11]
  • 别指望媒体会在你那愚蠢的应用发布前报道它。除了可能在模仿它的帖子里。[0:15]
  • 不要假设你的朋友和家人会谈论它。同样，不要假设陌生人会在街上走过来和你谈论这件事。[0:20]
  • 不要说你的愚蠢应用程序将改变我们的交流方式，或者称之为“自电子邮件以来最大的突破”[0:26]
  • 不要承诺你的应用会改变线上或线下的商业模式。肯定不是两者都有。[0:30]
  • 不要承诺(再一次)你的愚蠢的应用程序会像病毒一样传播。不要把它的增长与脸书或Twitter相比。[0:36]
  • 不要假装知道人们是如何在哈佛的自助餐厅里与马克·扎克伯格交谈的。或者，呃，他会怎么回应。[0:50]
  • 请不要把亚当·安吉洛扯进来。还有，如果一定要说，别说他看起来13岁。[1:02]
  • 别想用亚当·安吉洛的净资产来打动我。等等，为什么我们又在谈论亚当·安吉洛了？[1:18]
  • 不要让你的愚蠢的应用听起来像某种有“内部圈子”的独家俱乐部，并谈论我们如何幸运地被邀请和狗屎。[1:27]
  • 天鹅绒绳？否[1:33]
  • 内圈？他妈的不！[1:35]
  • 邀请五位“领导”？你搞什么鬼？！？！[1:40]
  • 不要说只有一个短暂的窗口让我加入！但是，拜托，上帝，如果有，让我们希望这个可恶的窗口尽快关闭，因为这个创业公司太烂了。[1:45]
  • 不要假装你有商业秘密。商业秘密？[1:48]
  • 当你真正的意思是“在我们的B.S .营销中使用的个人信息收集表”时，不要提到“NDA”还有，谁会在YouTube上给NDA签名？[1:58]
  • 或者，只是不要参与一般的废话——这几乎描述了整个视频。
