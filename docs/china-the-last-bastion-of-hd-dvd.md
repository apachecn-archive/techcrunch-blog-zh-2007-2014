# 中国:HD-DVD 的阿拉莫 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/07/27/china-the-last-bastion-of-hd-dvd/>

虽然格式战在西方已经尘埃落定，但在中国似乎并非如此。CBHD 格式是那里占主导地位的高清光盘，尽管[是它自己的东西](https://web.archive.org/web/20221007155850/http://en.wikipedia.org/wiki/CBHD)，它与 HD-DVD 的共同点比蓝光多。当然，那边最大的市场可能还是 vcd，但我认为这个小事实很有趣。

老实说，如果 HD-DVD 失去了美国和欧洲，但赢得了中国，这并不是一个坏消息。HD-DVD 狂热分子(我知道你在那里)将继续有一个奇怪的非品牌电子产品和 fakey 假媒体供应使用。
