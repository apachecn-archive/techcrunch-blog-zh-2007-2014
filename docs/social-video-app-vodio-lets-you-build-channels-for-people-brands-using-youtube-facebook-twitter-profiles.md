# 社交视频应用 Vodio 让你可以使用 YouTube、脸书和 Twitter 的个人资料为人们和品牌建立渠道

社交视频应用 Vodio 今天发布了新版本，该版本利用了该应用与用户社交账户(包括脸书、推特和 YouTube)联系的能力。在更新后的应用程序中，你现在可以自动将任何脸书页面或 Twitter 个人资料转换为视频频道，并直接从 YouTube 添加频道。

此前，Vodio 的一个与众不同的特点是，它不只是聚合社交网络上你朋友之间的视频趋势，它还提供了对“人群”认为流行的东西的洞察。该服务不仅跟踪用户的行为，还跟踪其社交图中的趋势，并将这些与它从观察网络趋势中获得的洞察力相结合，以显示符合用户兴趣的视频。这些推荐会出现在主题频道中，如科技、喜剧、应用、音乐、酷&励志、新闻、名人、病毒视频等等。

然而，随着更新，Vodio 现在不仅提供推荐和受欢迎的视频，还允许用户更多地个性化。现在，你可以在新增加的搜索界面中搜索朋友、名人、企业或品牌，还可以选择订阅那个人或公司的脸书页面、Twitter feed 或 YouTube 频道，作为一种更具体的方式来跟踪他们在社交网络上分享的视频。搜索时，该应用程序不仅会显示相关品牌/人物的精确匹配，还能调出相关内容，例如关于或由相关公司或人物制作的视频。

总部位于特拉维夫的 Vodio 于今年 1 月首次推出。这项服务与 [ShowYou](https://web.archive.org/web/20230308165953/http://showyou.com/) 、 [Telly](https://web.archive.org/web/20230308165953/http://telly.com/) (前身为 Twitvid)、以及 [Shelby Genius](https://web.archive.org/web/20230308165953/https://itunes.apple.com/us/app/shelby-genius/id556665733?ls=1&mt=8) (前身为 Shelby.tv)等公司竞争。正如你可以从“以前”这里看出的那样，社交视频应用市场仍然是一个年轻的市场，各公司正在试水，看看是否有办法在 YouTube 和脸书等巨头的阴影下建立自己的受众。

更新后的应用程序现在可以在 iPhone 和 iPad 上使用。
