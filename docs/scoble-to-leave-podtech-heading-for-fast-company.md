# 斯考伯离开波德科技，前往快速公司

著名博主罗伯特·斯考伯(Robert Scoble)表示，他将于明年 1 月中旬离开目前在波德科技的工作岗位。他可能会加入快速公司，创办快速公司电视。

Scoble 在 PodTech 只工作了一年半。他之前在微软工作，担任他们的非官方发言人。

斯考伯没有说他是否会继续使用他目前的节目形式，即他每天采访企业家的节目。另一个大问题是他的赞助商希捷是否会跟随他去 Fast Company。据传希捷公司将支付巨额费用赞助该节目。

这对 PodTech 来说是又一个打击，它已经失去了创始 CEO，并且几乎没有钱了。该公司已经花掉了 750 万美元的资金。据传，他们即将获得另外约 100 万英镑的过渡性融资。但是他们非常非常接近死池。
