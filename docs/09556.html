移动图片大战、莫斯科维茨宣言和其他来自TC的重大主题颠覆了SF TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/09/17/themes-tc-disrupt-sf/

TechCrunch Disrupt 2011已经结束。经过三天的发布、讨论、采访和演示，我终于有机会补上一觉，消化刚刚发生的事情。以下是我从今年的会议中获得的主题，排名不分先后:

第一天

手机图片大战正在进行:我天真地认为手机图片分享大战始于去年，终于去年，Instagram是赢家。没那么快。基于位置的签到公司 Gowalla ，在一个签到的价值被称为受到质疑的市场中，努力追赶 Foursquare ，巧妙地转变为更多的社会策划的旅游指南(围绕位置和图片)。博客网站 Posterous ，努力跟上 Tumblr ，重新专注于推出“ Spaces ”，更专注于在各种团体内公开和私下分享图片和视频。(顺便提一下，Tumblr可能会与Pinterest和T21竞争，因为它会保持增长。)而相机大战还在继续。埃里克·冯(Eric Feng)创建了一个新网站，试图围绕经历来组织图片。红杉资本合伙人 Doug Leone 继续在唱片上讲述他们对 Color 的投资，后者将发布其下一款图片应用 Blue ，与脸书有着深厚的分享联系，据传言将打造自己的专用图片应用。 Twitter 做了一个聪明的举动，用一个让人想起with . me(Twitter内的图片标记，来自 Path )的查看器来聚集用户账户周围的图片，我认为这仍然是2011年设计最漂亮的图片应用，但还没有真正坚持下来。也许甚至卡特琳娜·法克的隐形创业公司 2bkco 也可以利用手机图片，毕竟她是 Flickr 的创始人。移动照片分享仍然是一个竞争非常激烈的领域，至少目前来看，可能会呈现出难以捉摸的“薄楔子的边缘”,一家新的创业公司可以借此突破并建立一个社交网络。

2、移动应用与HTML5: 在的一次讨论中，与 MG Siegler 、 Pandora 的Tom Conrad和Rovio(愤怒的小鸟的制作者)的Andrew Stalbow讨论了为新的和新兴平台构建应用的思维过程。底线是，这两家公司都是有史以来下载量最大的应用，都致力于为用户居住的平台建设。这是关于用户的。如果一群用户最终使用Windows Mobile 8，那么Pandora也会出现。与此同时，Conrad提出，构建HMTL5应用程序(可能成为跨多个平台的“通用语言”)的实验也是一种可能性，尽管它还没有实现。Qwiki 宣布了一个新的编辑器和对其平台的大量HTML改进，其中一些是全新的。虽然在HTML5中开发的好处对开发者来说是显而易见的(更低的成本，更高的效率，以及绕过应用商店规则)，但对许多设计师和开发者来说，用户的成本可能很高，在某些情况下需要谨慎，特别是在iOS平台上。在我用过的几个HMTL5应用上，体验就是不一样(甚至不接近)，开发人员可能会在某些平台上对这条路三思而行。观察HTML5能实现多少技术进步，以及在某些情况下，用户是否能对移动设备上的这些体验无动于衷，这将是一件有趣的事情。

3、莫斯科维茨宣言:我不太了解达斯汀·莫斯科维茨(除了他共同创立的【脸书】和阿萨纳)，但是听到他和埃里克·舍恩菲尔德谈论他对创业生态系统的看法是很有趣的。这个视频值得一看，但简单来说，在硅谷呆了七年后，莫斯科维茨认为，当前的创业生态系统，早期资金如此容易获得，正驱使创始人基于更狭隘的价值主张建立公司，以至于很难找到合适的人才，因为它很分散。他甚至猜测，如果脸书成立于2011年，他不确定今天的公司是否会找到合适的人才来实现这一愿景。莫斯科维茨还认为，现在成为创始人的前脸书员工通常被一种共同的风气所驱使，即建立一个持久的公司，而不是为了出售而建立，即使这一过程中存在轻微的非理性风险。鉴于他的低调，我对他愿意说出自己的想法印象深刻。

第二天

4、【实验室】公司的出现: 牛奶联合创始人凯文·罗斯，一位成功的天使投资人(也是 @CrunchFund 的投资人、 Digg 的创始人)坐下来和迈克尔·阿灵顿聊天。正如你将在视频中看到的，Rose小心翼翼地警告说，Milk的第一个移动产品“Oink ”将会有两到三个其他产品跟进，并且体面地承认，没有人真正知道在这样一个快速变化、竞争激烈的环境中，应用程序的发布会如何。我注意到新的移动公司采用“实验室”命名法的更大趋势，我认为这软化了人们对大型首次发布的期望，并向投资者、科技媒体和早期采用者发出信号，表明他们的努力在很大程度上是实验，并期待他们在永恒的探索中建立更多的杀手级应用。

5、关于精益创业的争论:“精益创业”运动的兴起既激发了大批开发人员快速构建和快速迭代，但也产生了微小的反弹(“肥胖创业”)，并引起了硅谷一些最成功投资者的惊讶，比如基思·拉博伊斯，他最近在推特上回复一个询问使用这种方法的成功互联网公司数量的问题:“一个也没有。”该运动目前的代表人物，主要是埃里克·里斯和斯坦福大学的史蒂夫·布兰克，带来了大量的研究项目和多年的行业经验。里斯与Intuit创始人斯考特·库克和Instagram创始人凯文·斯特罗姆在Disrupt登台。这场运动尤其引起了技术创始人的共鸣，部分原因可能是这种方法为那些没有这类经验的人提供了一个基本的业务运营和管理框架。当然，我在这里画的是一般化。另一方面，我和一些投资者和企业家谈过，他们觉得受到这种思维水平的限制，想立刻做大。就像大多数事情一样，真相可能就在中间，最好是用具体情况具体分析的方法，而不是一种信仰体系。

6、投资者营销的演变:到目前为止，所有人都知道，个人和机构投资者都在想方设法向企业家推销自己。他们更多地写博客，参与社交媒体，参加更多的活动，在种子阶段投资，并配备与业务相关的专业人员(和专业的特设网络)以跟上时代。这些以不同的方式表现出来。比如在会议期间，维诺德·科斯拉哀叹被称为风险“资本家”，而更喜欢“风险助理他还认为，许多投资者的主要驱动力是“做交易”，而不是帮助企业家。你不得不称赞科斯拉，他无疑是一位成功的风险投资家，这并不可耻。顶级公司继续在早期阶段投资，来自查尔斯·里弗风险投资公司、Accel Partners和谷歌风险投资公司的投资者也这么说。一年前在Disrupt， Greylock Partners 宣布了它的早期探索基金，今年， Shervin Pishevar 宣布了一个新的人才基金和 Menlo Ventures 。这两种工具的存在都是为了让机构投资者更快地投资于有趣的早期创业公司。第二天，基准资本合伙人马特·科勒 与阿灵顿坐下来讨论他最近的投资，其中有几笔是海外投资，比如 Peixe Urbano (巴西)。他也去过柏林，他认为那里是创业的好地方。一些投资者(出于各种原因)更喜欢在离家近的地方投资，一些投资者将目光投向硅谷以外的地方，并慢慢走出国界，寻找那些独特的机会。

7、从(花园的)边缘切入:投资者经常说，他们希望看到更大胆、看似不可能的概念。好吧，你来了:在周二的创业战场期间，其中一位演讲者是种植星球，起初这是Disrupt(一个帮助你种植自己食物的社交网络)的一个看似奇怪的创业公司，但经过一番思考后，我觉得这是整个活动中最独特的概念之一。我并不是说他们会变成Zynga，但是正如评委Chi-Hua Chien在演示后漫不经心地评论的那样，他们在某种程度上“创造了现实生活中的农场村”自从美国第一夫人米歇尔·奥巴马把照料后院和社区花园作为她推广良好营养的关键问题之一以来，这种想法正在慢慢地传遍全国。我也推荐看视频。演示结束后，Chien给出了建议:简而言之，他鼓励创始人思考如何(1)与当地服务提供商合作，帮助购买园艺用品和安装设备，以及(2)与当地餐馆联系，这些餐馆希望从过剩的供应中获得当地种植的产品。现在，这是一个新的“线上到线下商务”的旋转虽然很多人都希望这家公司能成功，但我并不保证它能成功。我认为，钱学森和创始人之间的互动突显了企业家找到合适的投资伙伴的重要性，这些投资伙伴可以帮助他们拓展商业思路，同时也回击了目前流行的一些流言蜚语，即投资者只对他们的钱有利。

第三天

8、脸书IPO的承诺:预计随着年底的临近，对脸书IPO的期待会慢慢增长。该公司最杰出的两位领导人戈库尔·拉贾拉姆和迈克·斯科洛普夫到访，讨论产品、工程、社交广告和一系列其他话题。在过去的几个月里，脸书已经在“地方和“问题上改变了路线，然而却将其对Beluga 的收购变成了一个超级圆滑的iOS 信使。在脸书高管们登台的同时，一些媒体猜测脸书希望将任何IPO推迟到2012年9月，甚至可能更晚，尽管有相互矛盾的说法。这家社交网络年收入约为30亿美元，除了既得雇员和投资者被压抑的需求之外，它没有任何迫在眉睫的申报压力。对IPO的预期正在上升，毫无疑问，脸书只是触及了它潜在能力的表面，但脸书方面很快将面临越来越大的压力，要求其不辜负这些预期，即使其他人不公平地夸大了这些预期。无论扎克伯格最终做了什么，他可能想向团队传递一个信息，即还有更多的东西要建立，或者也许他觉得随着收入的增长，社交广告并没有完全解决，最好重新聚焦，并确保每当IPO到来时，所有松散的细节都被捆绑起来。

9、天使得到他的翅膀:一些最成功的超级天使(或微型风投)像戴夫·麦克卢尔 ( 500家初创公司)、乔希·费尔塞 ( 自由式)、艾登·森库特 ( 菲利西斯)、杰夫·克拉维尔 ( 软科技)和即使这些话题中的一些明显已经过时，但还是值得看一下视频。特别是，有一个关于无上限可转换债券和能够就这些条款进行谈判的企业家的详细讨论，尽管它给已经承担巨大风险的早期投资者带来了更多风险。其次，阿灵顿提出了一个有趣的问题，即超级天使投资(有时需要一种产品或牵引力)与来自家人、朋友和那些在行业和社区中取得成功的人的更真实意义上的天使投资之间的区别。从视频中可以清楚地看到，尽管我们倾向于将超级天使归入一个投资类别，但他们都非常不同，由不同的信号和投资主题驱动，因此，有希望的企业家应该以不同的方式接触他们。

好了，你知道了。请记住，这只是我在采访中注意到的个人对更大主题的看法，也是一个普通人试图在噪音中找到一些信号。如果你能在下面评论你从这次活动中挑选出的其他主题(好的或麻烦的)，那将会更有趣，我会确保跟进和评论。提前感谢，继续关注中国的 TechCrunch扰乱北京。
