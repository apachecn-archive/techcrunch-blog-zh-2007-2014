# 最初的谷歌 Cr-48 Chrome 笔记本印象有一个重复的说法:Flash 太烂了

正如我们今天早些时候在提到的，谷歌几乎没有浪费时间让用户拿到他们的 Cr-48 Chrome 笔记本电脑。谷歌发布这款设备不到两天，就有大量报道称用户已经得到了它。有趣的是，一些以为自己注册了 Chrome 贴纸的人也收到了贴纸——相当大的一笔奖金！但是 Cr-48 并不完全是一个惊喜。

一些报道称，试图在设备上播放 Flash 视频和应用程序或多或少很糟糕。显然，这种体验从错误到根本不工作不等。这包括 YouTube、Hulu、Vimeo、CNN——基本上是网络上所有主要的视频网站。

现在，必须指出的是，谷歌已经非常明确地表示，Cr-48 是一台测试机。还没准备好大规模生产。此外，内部的处理器(英特尔凌动处理器)不能完全运行 Flash 也不足为奇。也就是说，谷歌也明确表示 Flash 是 Chrome 的重要组成部分(以及 [Chrome OS](https://web.archive.org/web/20230404070111/https://techcrunch.com/2010/03/30/chrome-os-flash/) )。其实，[是烤熟的](https://web.archive.org/web/20230404070111/https://techcrunch.com/2010/06/25/google-chrome-flash/)。你别无选择，只能安装它。

是的，你可以通过 Chrome 禁用该插件，但与苹果的移动设备不同，这种退路基本不存在。

当苹果决定不在他们的新 MacBook Airs 上安装 Flash 时，他们引起了轩然大波，但是不可否认的是，没有 Flash 的机器运行起来更加流畅(无论是从电池寿命还是网络浏览性能来看)。也许谷歌应该考虑在第一批面向消费者的 Chrome 笔记本推出时遵循类似的路线。
