# FlipBoard 的迈克·麦库埃:建造者

在迈克·麦库埃发现如何将 iPad 翻转成一个让阅读电子杂志变得轻而易举的设备之前，他本人在纽约州的 T4 硅谷伍德斯托克工作时就被一些科技界大腕发现了。

在克里斯·迪克森的这一集*创始人故事*中，你会听到他们在用 Extended Basic 为 TI99 编写视频游戏，麦库埃如何去了 IBM 而不是大学，讨论他如何在资金紧张时实现收支平衡，这种情况如何改变了一百万次，以及他的第一个创业公司 Paper Software 背后的想法。“我们的想法是创造像一张纸一样简单实用的技术，”他说。几经波折后，它被网景公司收购，当一个叫 JavaScript 的小东西风靡编程界时，他发现了自己。

然后，在 AOL 收购 Netscape 之后，麦库埃告诉迪克森，他决定重新开始他的创业之痒，并描述了他重新回到他的初创公司“TellMe”的创始人游戏中。“我们想建立 Dialtone 2.0，”他说。

听听他自己的话。

在下面的采访中，麦库埃从他停下来的地方继续讨论，在监管 200 名员工和在 2000 年烧焦的市场上花费“每年 4500 万美元和很少的收入”的情况下，勉强引导 TellMe 远离 dot.com 崩溃着陆。幸运的是，TellMe 能够筹集到最后一轮 1.25 亿美元。“当钱从银行流入时，”麦库埃回忆道，“市场崩溃了。”当他准备“穿越沙漠”时，他要做出一些艰难的决定。

结果一切都很好。TellMe 最终被微软以[8 亿美元](https://web.archive.org/web/20221001021045/https://beta.techcrunch.com/2007/03/14/microsoft-acquires-tellme/)收购。但是到达那里并不容易。
