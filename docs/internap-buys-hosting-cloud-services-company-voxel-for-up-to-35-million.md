# Internap 以高达 3500 万美元的价格收购了托管云服务公司 Voxel

IT 基础设施解决方案提供商 [Internap](https://web.archive.org/web/20230126030413/http://www.crunchbase.com/company/internap) 今晨[宣布](https://web.archive.org/web/20230126030413/http://www.prnewswire.com/news-releases/internap-acquires-enterprise-hosting-and-cloud-services-provider-voxel-136583578.html)以[3000 万美元现金](https://web.archive.org/web/20230126030413/http://www.crunchbase.com/company/voxel-dot-net)收购总部位于纽约的企业托管和云服务公司 [Voxel](https://web.archive.org/web/20230126030413/http://www.voxel.net/) 。根据交易条款，在未来两年内，根据某些基于绩效的标准，可能会有额外的 500 万美元现金摆在桌面上。

该公司于 1999 年在创始人 Raj Dutt 的宿舍中成立，为企业和以网络为中心的企业提供按需专用托管和云服务。

Voxel 表示，它目前服务于 1000 多家客户，拥有大约 50 名员工。

该公司得到了总部位于纽约的私募股权公司海港资本的支持。

通过将 Voxel 在阿姆斯特丹和新加坡的服务点添加到 Internap 的现有服务点，合并后的公司旨在为全球客户提供主机托管、托管、专用托管和云服务。Internap 表示，它还将利用 Voxel 的自动化技术和专业知识来增强现有产品。
