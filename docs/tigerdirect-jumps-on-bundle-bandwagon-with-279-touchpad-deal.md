# TigerDirect 凭借 279 美元的 TouchPad 交易紧跟捆绑销售潮流

看起来百思买设法让一些 32GB 的触摸板从他们的手指间溜走了，因为 TigerDirect 已经在惠普的死产平板电脑周围炮制了[一个 279 美元的捆绑包](https://web.archive.org/web/20230204214748/http://www.tigerdirect.com/sectors/campaigns/hp/touchpad_available.asp)。

是的，想要低价买到 TouchPad 的人可能需要去别处看看(顺便说一句，祝你好运)。当谈到 TouchPad 交易时，似乎总是有一个陷阱，TigerDirect 非常乐意满足。

花 279 美元，你不仅可以得到一个 32GB 的触摸板，还可以得到一个 Belkin 平板电脑外壳，一个 Belkin FlipBlade 平板电脑支架和一对 Creative Lab D80 蓝牙扬声器。哦，谁会忘记 TigerDirect 为了增加交易的甜头而推出的 16GB 惠普闪存驱动器呢？

所有这些额外的赠品是否值得额外的 130 美元完全取决于你，但这种捆绑似乎比百思买的个人电脑购买策略更容易接受。就我个人而言，这似乎不是一个坏交易——只要写下一个你不情愿为之购买节日礼物的人的名单，并相应地分配这些配件。或者，事后跳到 Craigslist 上，把这些好东西翻过来获利。

TigerDirect 将于美国东部时间今天下午 2:30 打开 TouchPad 捆绑包的闸门，所以那些渴望最后一次体验 webOS 的人最好在那之前准备好。
