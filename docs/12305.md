# DailyWorth 获得 200 万美元，通过收件箱 TechCrunch 为女性提供高质量的理财建议

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/05/dailyworth-grabs-2-million-to-bring-quality-financial-advice-to-women-via-the-inbox/amp/>

去年 3 月， [DailyWorth](https://web.archive.org/web/20230213192946/http://dailyworth.com/) 、女性个人金融每日电子邮件和社区[筹集了 85 万美元的种子资金](https://web.archive.org/web/20230213192946/https://techcrunch.com/2011/03/02/dailyworth-raises-850k-to-become-the-daily-candy-for-personal-finance/)，由罗宾汉风险投资公司(Robin Hood Ventures)牵头，投资者包括埃里克·施密特(Eric Schmidt)的 TomorrowVentures、Social Leverage、500 Startups、Venture51、Investors' Circle、乔安妮·威尔逊(Joanne Wilson)、大卫·科恩(David Cohen)、斯科特·贝克尔(Scott Becker)、彼得·尚克曼(Peter Shankman)和卡罗尔·周(Carol Chow)。

现在，这些投资者中的一些人似乎想在 DailyWorth 上加倍下注，因为这家初创公司宣布已经完成了由 [Joanne Wilson](https://web.archive.org/web/20230213192946/http://www.linkedin.com/pub/joanne-wilson/10/861/b08) 和 StockTwits 联合创始人兼首席执行官[Howard lindson](https://web.archive.org/web/20230213192946/http://www.crunchbase.com/person/howard-lindzon)的社交杠杆以及前投资者的捐款牵头的 200 万美元的首轮融资。新一轮融资使 DailyWorth 的总资金超过 300 万美元。

昨天，beta beat 首次披露了这个故事，上个月，博客作者兼天使投资人 Joanne Wilson [告诉了 O'Reilly's Radar](https://web.archive.org/web/20230213192946/http://radar.oreilly.com/2011/12/joanne-wilson-angel-investing.html) 她对 DailyWorth 的兴奋之处，称这家初创公司已经“创造了巨大的牵引力，并以闪电般的速度销售广告”——显然，这家初创公司的库存正在减少。

投资者和广告商发现这家初创公司很有吸引力，因为它以智能、务实的方式为女性创造了一个围绕金融知识和资金管理的社区。这是 DailyWorth 创始人兼首席执行官阿曼达·斯坦伯格(Amanda Steinberg)的一部有针对性的作品，她已经是六次企业家了，她开始为女性(已经是资金管理者)创造一种资源，打破基本的金融概念，让忙碌的妈妈和女性专业人士容易消费(在旅途中)。

这份时事通讯包括有关金融和职业建议的教育花絮，在美国已经拥有超过 20 万订户，荷兰国际集团和 H&R·布洛克等金融机构也加入了赞助行列。曾在 Pilot Group 投资过 Daily Candy 和 Thrillist 等电子邮件公司的安德鲁·拉塞尔(Andrew Russell)也以独立董事的身份加入了这家初创公司。

电子邮件时事通讯最近已经卷土重来，这也是 DailyWorth 区别于其他社区的地方，daily worth 被称为“赚钱的 DailyCandy ”,比如资金雄厚的 LearnVest。

DailyWorth 试图超越简单地成为另一个为女性提供金融建议的生活方式或交易网站，试图帮助女性以更细致的方式理解金钱。如果不是老一套的话，每天直接发送一份简讯到你的收件箱是一个很好的方法，可以确保这种金融教育直接和持续地到达目标受众——特别是当它补充了很好的编辑内容时，因为，正如斯坦伯格所说，女性不仅想知道如何预算和储蓄，还想知道如何投资，要求更高的工资，并为未来建立一个金融战略。

为了推进这一目标，DailyWorth 将在今年年初推出两个新版本的模型，一个是针对“富裕女性”的“MoreWorth ”,另一个是针对女性企业家的“CreateWorth”。《纽约时报》前专栏作家、国会议员邓利维将领导这家初创公司的编辑团队。
