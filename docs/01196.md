# 正如承诺的那样，Zivity 创始人光猪六壮士 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/01/16/as-expected-zivity-founder-does-the-full-monty/amp/>

[<amp-img src="https://web.archive.org/web/20230120123542im_/https://techcrunch.com/wp-content/uploads/2008/01/zivitycyann.jpg" class="border amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230120123542/http://www.zivity.com/)

两周前，我写道，Zivity 创始人 Cyan Banister 不仅是她成人摄影初创公司的高管，她还开始为该公司做模特。不过，当时她还穿着衣服。

现在，她开始全力以赴了。在一组名为“在巴黎时”的新照片中，她袒胸露乳我在上面放了一张(经过审查的)图片。

震撼？有点吧。但是就像我之前说的，至少她通过为网站做模特来吃自己的狗粮。不过，她这样做不仅仅是为了推广网站。这也是自我表达的一种形式。在一封电子邮件中，她说“我脱掉衣服不仅仅是为了提升活力。我这样做也是为了模特界，也是为了我自己。我想改变人们对裸体和美丽的看法，就像主要的电视节目打开了我们对其他问题的思维一样…我真的很喜欢上 Zivity，在照片中裸体和性感是令人愉快的！我看着我的照片，我笑了。那是我的身体。我可以做我想做的事。很漂亮。谁也不能把这从我这里夺走。”

*“我创办了 Zivity，因为我相信授权商业模式能够让人们成为企业家，”她补充道。*

不管什么原因，人们喜欢她的照片。她目前在顶级模特排行榜上名列第八。Banister 还在做其他照片集，这些照片集正在被添加到 Flickr。参见**不安全工作**图片[此处](https://web.archive.org/web/20230120123542/http://www.flickr.com/photos/merkley/344970526/in/photostream/)和[此处](https://web.archive.org/web/20230120123542/http://www.flickr.com/photos/merkley/347934569/in/photostream/)为例。

Zivity 有一个独特的商业模式，根据模特和摄影师获得的用户投票数直接向他们支付报酬。投票越多，他们在排行榜上的位置就越高，赚的钱就越多。大约 40%的总收入直接给了人才。他们在去年秋天的 TechCrunch40 大会上首次推出了 T1。2007 年 8 月，该公司筹集了[100 万美元的种子资金。](https://web.archive.org/web/20230120123542/http://www.crunchbase.com/company/zivity)

Zivity 仍处于私下测试阶段，并在等待名单上慢慢增加了大约 20，000 人。
