# Google Ventures 加入 DocuSign 的新融资，融资额增至 5570 万美元

看起来电子签名公司 [DocuSign](https://web.archive.org/web/20221206051138/http://www.docusign.com/) 在上个月[披露其 D 轮融资](https://web.archive.org/web/20221206051138/https://beta.techcrunch.com/2012/07/12/docusign-kleiner-perkins-mary-meeker/)时还没有完全完成融资——今天它宣布了一个新的投资者。

这笔融资最初是通过[向美国证券交易委员会](https://web.archive.org/web/20221206051138/http://www.sec.gov/Archives/edgar/data/1261333/000126133312000001/xslFormDX01/primary_doc.xml)提交的一份文件披露的，随后该公司证实，它已经在由凯鹏华盈(Kleiner Perkins cau field&Byers 牵头的一轮融资中筹集了 4750 万美元，凯鹏华盈合伙人玛丽·米克尔加入了董事会。Accel Partners、Comcast Ventures、SAP Ventures 以及“一家大型全球机构投资者”也参与其中。

从凯鹏华盈筹集资金已经非常令人印象深刻，但 DocuSign 增加了另一个知名投资者——谷歌风险投资公司，其投资将这轮融资的总规模增加到 5570 万美元。

该公司目前已经筹集了总计约 1.14 亿美元的资金。当这轮融资首次宣布时，首席执行官 Keith Krach 告诉我，它将用于研发、拓展新行业和国际增长。
