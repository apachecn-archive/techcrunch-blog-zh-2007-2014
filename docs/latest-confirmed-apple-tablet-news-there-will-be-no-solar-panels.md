# 最新确认的苹果平板电脑消息:将不会有太阳能电池板

[![6a0120a5580826970c0120a7f77161970b-800wi](img/2a0604d833c48a98ed20dab55c20130d.png "6a0120a5580826970c0120a7f77161970b-800wi")](https://web.archive.org/web/20230322164148/https://techcrunch.com/wp-content/uploads/2010/01/6a0120a5580826970c0120a7f77161970b-800wi.jpg)

有人猜测苹果平板电脑将是市场上最环保的设备之一。为什么？因为在[专利浮出水面](https://web.archive.org/web/20230322164148/http://www.patentlyapple.com/patently-apple/2010/01/apple-thinks-green-again-with-solar-cell-panels-for-media-players.html)描述了一个前面带有太阳能电池板的 iPod 之后，有一些传言称这项专利将在平板电脑中发挥作用，也许是以玻璃屏幕后面的太阳能电池板的形式。
 对不起，生态学专业的，那都是空话。我们已经确认它不会有太阳能电池板。这听起来可能很傻，但这可能是该设备的一大优势。很抱歉我们在追踪这些谣言，但是每一点你都会知道的。
