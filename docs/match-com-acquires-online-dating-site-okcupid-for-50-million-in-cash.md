# IAC 的 Match.com 以 5000 万美元现金收购在线约会网站 OkCupid

今天，IAC 的 Match.com 继续其收购策略，以 5000 万美元现金收购在线约会网站 [OkCupid](https://web.archive.org/web/20230405113713/http://www.okcupid.com/) 的[收购](https://web.archive.org/web/20230405113713/http://www.prnewswire.com/news-releases/iacs-matchcom-acquires-okcupid-115090674.html)。该交易还包括视业绩而定的未来收益。

虽然 Match 拥有庞大的用户群，但 OkCupid 的单身人士往往更年轻，这就是为什么 Match 发现这是一笔有益的收购。该网站是免费的，显然是通过广告创收，根据 IAC 的说法，它已经“成为基于广告的交友网站中增长最快的一个”

OkCupid 已经筹集了 600 万美元的资金，由马克斯·克罗恩、萨姆·亚甘、克里斯·科恩和克里斯蒂安·鲁德共同创立。Match 之前以 8000 万美元收购了约会网站[People Media](https://web.archive.org/web/20230405113713/https://techcrunch.com/2009/07/07/matchcom-acquires-people-media-for-80m-in-cash/)，最近又收购了 [Singlesnet。](https://web.archive.org/web/20230405113713/https://techcrunch.com/2010/02/25/confrirmed-match-com-acquires-singlesnet/)

亚甘将继续在新创公司的办公室里运营 OkCupid。希望这次收购不会阻止 OkCupid 发布其关于在线约会场景的匿名数据。或者更好的是，报告将包括来自 Match.com 和 OkCupid 的数据。
