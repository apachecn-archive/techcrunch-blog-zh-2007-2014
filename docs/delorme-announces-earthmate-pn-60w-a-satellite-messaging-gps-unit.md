# 德洛尔姆公司宣布推出 Earthmate PN-60w，这是一款卫星信息 GPS 设备

[![](img/3657b4281f92a806ccb7048fafa1d73a.png "pn60w_with_spot")](https://web.archive.org/web/20230131162049/http://www.crunchgear.com/2010/06/02/delorme-announces-earthmate-pn-60w-a-satellite-messaging-gps-unit) 
所以你被困在西藏的高山上，只有你的智慧和一片牛肉干介于你和死亡之间。你是做什么的？为什么不发短信给你的朋友“巫师非常需要 FUD，哈哈！”在 DeLorme 地球伴侣 PN-60w 上。

该设备是一个内置键盘的便携式全球定位系统装置。它支持 [SPOT](https://web.archive.org/web/20230131162049/http://www.findmespot.com/en/) 卫星信息技术——显然与微软的 MSN Direct 技术无关——可用于在紧急情况下寻求帮助。然而，使用这项技术，您可以从世界上几乎任何地方发送信息。

该单元目前处于预发布阶段，将包括美国和加拿大的地形和街道覆盖范围，从而在很大程度上限制了其在西藏的用途。

> 地球伴侣 PN-60w 是世界上第一个能够通过卫星输入和发送短信的手持全球定位系统。PN-60w 上的一个特殊接口将其与 SPOT 卫星通信器无线连接，该通信器是专门为 DeLorme 设计的。消息在 PN-60w 的内部键盘上生成，然后通过卫星转播，使用成熟的 SPOT 技术传送。
> 
> 因为信息是通过卫星传送的，并且不依赖于手机的覆盖范围，所以带有 SPOT 卫星通信器的 PN-60w 即使在世界各地的偏远地区也能保持用户的可靠连接。这两款手机都经过加固和防水设计，因此比大多数手机更适合户外使用。
