# OrangeScape 推出了 Kissflow，这是一款谷歌应用程序的工作流构建器

从今天开始，使用谷歌应用程序的企业应该可以更轻松地管理复杂的任务，初创公司[推出的](https://web.archive.org/web/20221210070317/http://www.orangescape.com/)[kiss flow](https://web.archive.org/web/20221210070317/http://www.kissflow.com/)orang escape——这应该是第一个与应用程序“深度集成”的工作流创建器。

OrangeScape 将此作为使用电子邮件和电子表格来跟踪批准和其他任务的替代方案，随着公司的发展和任务变得更加复杂，电子邮件和电子表格可能会变成一个令人头疼的问题。有了 Kissflow，企业应该能够通过五个相对简单的步骤来创建费用报告等工作流:命名流程、设计表单、定义工作流、配置权限，然后发布。

当然，还有与谷歌应用程序的集成。Kissflow 客户可以将文档直接附加到他们的工作流中，并与他们的 Google Apps 域中的任何人进行协作。该界面旨在让人想起 Gmail。为了安全和控制，Kissflow 上的每个企业都在 Google Cloud SQL 上有一个单独的数据库。

Kissflow 将在 Google Apps Marketplace 上提供，对最多 10 个用户免费，之后每个用户收费 3 美元。

至于产品背后的公司，OrangeScape 是由首席执行官苏雷什·萨姆班达姆(Suresh Sambandam)和首席技术官马尼·多莱萨米(Mani Doraisamy)共同创立的。它已经从天使投资人那里筹集了 100 万美元的“过桥轮”(包括 YahooIndia 前 Sharad Sharma 文卡特·拉朱；Pramod Bhasin，前 Nasscom 主席兼 Genpact 首席执行官；Quattro 首席执行官拉曼·罗伊(Raman Roy)；StraTerra Partners 首席执行官 Linda Bernardi)计划在今年晚些时候再融资 1000 万美元。
