# Handmark 抢滩 I-play 的手机游戏

全球移动媒体分销商 Handmark 与移动娱乐领域的领导者 I-play 合作。Handmark 现在有权通过 Handmark 的全球分销渠道在全球范围内为黑莓设备分销 I-play 的许多最成功的游戏。

Handmark 将负责发行《速度与激情:东京》(下载量已突破 800 万次)和《宝石探险》( Jewel Quest)等热门游戏。《宝石探险》非常受欢迎，自 2004 年首播以来，下载量已经超过 4000 万次。其他标题包括，3d 池:高辊，麻将追求，我的狗，和枕头大战。其他头衔肯定会接踵而至。

Handmark 通过美国移动运营商提供独特的分销服务，包括设备上的客户端和门户网站，以及 Handmark 和运营商品牌的桌面和设备上的商店。Handmark distribution 还通过与主要设备制造商的关系，在欧洲包括设备上和基于网络的电子商店。

“I-play 很高兴与 Handmark 合作，并通过我们市场领先的休闲游戏吸引黑莓用户，”I-play 首席执行官大卫·戈森说。“我们简单易玩、难掌握的游戏开发理念非常适合在黑莓设备上玩游戏。”

Handmark 联合创始人兼首席营销官道格拉斯·爱德华兹(Douglas Edwards)表示:“我们致力于打造强大的产品组合，为全球客户和合作伙伴提供最好的移动内容。“I-play 是一家受人尊敬的知名移动娱乐开发商，一直致力于开发高质量的游戏，其产品是我们产品目录中的一大亮点。”

正如 MobileCrunch 的读者所知，今年手机游戏的销售额预计将达到 40 亿美元。预计这一数字在未来四年将翻一番。移动软件开发和分销领域的知名企业将建立类似于 Handmark 和 I-play 之间的合作伙伴关系，以满足移动游戏娱乐的需求。
