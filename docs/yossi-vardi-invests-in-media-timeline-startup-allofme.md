# Yossi Vardi 投资媒体时间线初创公司 AllofMe

[![](img/d9d253b355f5b0e9a2b8e26d84101b1d.png)](https://web.archive.org/web/20221006141847/http://www.allofme.com/)

ICQ 旗下奇异基金的创始投资者 Yossi Vardi 向特拉维夫的初创公司 [AllofMe](https://web.archive.org/web/20221006141847/http://www.allofme.com/) 投入了一笔未披露金额的资金。

该公司正在开发一款产品，目前处于私人测试阶段，你可以用它展示你的数字资产(电子邮件、照片、视频、文档等)。)沿着数字时间线。虽然关于 AllofMe 的细节还没有透露，但看起来你将能够包括来自 Flickr 和 YouTube 等在线服务的媒体文件。

创始人 Addy Feuerstein 这样描述 AllofMe:

“这个想法是，如果我或其他人有一张包括我儿子的照片，单独或与朋友在一起，我或其他任何人都可以标记照片中的人，并将这些数字资产转化为我儿子的一部分。当他长大并控制自己的时间线时，他将拥有一个由童年时期的标记材料组成的时间线……我们还将通过互联网网站上的小工具将每个人创建的时间线转换成视频电影，并使您的时间线与您的熟人的时间线或按时间顺序排列的数据文件相比较。例如，你将能够把你自己的时间线与历史事件进行比较，并看到一些重要的世界事件发生时你在哪里。”
