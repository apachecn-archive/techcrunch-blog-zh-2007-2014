英国人来了！TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2008/03/11/the-british-are-coming/

我敢肯定，没有必要让更多的英国人在硅谷闲逛——但我担心，在4月19日至25日这一周，这将是不可避免的，因为20家英国初创公司涌入，这是英国政府的好意。 Web Mission 08 是一个让20家英国科技初创公司与硅谷的公司和个人建立联系的项目。该项目由英国企业家奥利·巴雷特和顾问 Polecat 组织，由 TechCrunch UK 和科技企业家兼投资者道格·理查德参与。超过100家公司申请加入。目的是让这些公司“与硅谷的关键人物一起探索新的增长机会。”除了政府的英国贸易投资部门，其他赞助商还包括 Heller Ehrmann 、英国电信和汇丰银行。一些支持T21的声音来自Bebo的Michael Birch等人。

来的标准是什么？评委们在寻找能够在美国开展有意义的业务的公司:从设立办事处，到寻找资金，到扩展网络，以及发展可持续的“跨池塘”关系。他们还根据企业的生存能力和管理团队的实力来评估这些公司。他们寻找能够展示英国在这一领域的广阔前景的公司组合。

结束后，这个小组有一个日程安排，还将参观Web 2.0博览会。我们甚至可以说服阿林顿先生打个招呼。如果你想和初创公司见面，就发电子邮件给info@polecatting.com。来打个招呼吧——我们帮你省了一趟去伦敦的路。
