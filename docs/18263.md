# 《毁灭战士》的创造者现在是脸书的一名员工

这不是 bizzaro 世界。我们很快就会生活在这样一个世界里,《毁灭战士》、《地震》和我大部分童年记忆的创造者为马克·扎克伯格工作。约翰·卡马克很快就会成为脸书的员工。

脸书[刚刚宣布](https://web.archive.org/web/20230125234853/https://techcrunch.com/2014/03/25/facebook-to-buy-oculus-vr-maker-of-the-rift-headset-for-around-2b-in-cash-and-stock/)计划以大约 20 亿美元的现金和股票收购 Oculus VR。该公司将在脸书独立运营，专注于游戏业务。脸书通信公司向 TechCrunch 证实，Oculus 的所有人都保持不变，约翰·卡马克仍将是 Oculus 的首席技术官。

卡马克在 2013 年末为 Oculus VR 留下了 id 软件。显然 id 的母公司 Zenimax 没有分享卡马克对虚拟现实的热情，并试图限制可用于制作 VR 游戏的资源。

“他们不能在这一点上走到一起，这让我很难过。这只是不幸，”卡马克[在二月](https://web.archive.org/web/20230125234853/http://www.usatoday.com/story/tech/2014/02/03/john-carmack-on-oculus-rift/4938887/)告诉《今日美国》。“当我在 id software 工作时，显然没有机会做任何虚拟现实方面的工作，我决定不再续约。”

如果卡马克说虚拟现实可以成为一件大事，那听听是明智的。作为游戏诸多方面的冠军和开发者，卡马克给这个价值数十亿美元的行业留下了比其他任何人都深刻的印象。

同样，可以理解为什么 Zenimax 和其他人对 VR 更加悲观。到目前为止，这项技术还没有证明自己是一个很酷的技术演示。它成为一种可行的消费产品的机会仍需数年时间。

但脸书将长期参与其中。马克·扎克伯格今天在电话会议上告诉记者和投资者，他的公司的愿景是建立下一个计算平台。随着约翰·卡马克的加入，扎克现在有了业内最优秀的员工之一为他工作。
