# 苹果公司的蒂姆·库克认为 iPad 在 Mac 蚕食市场方面有“巨大的机会”

在今天的财报电话会议上，苹果首席执行官蒂姆·库克回应了关于 iPad 设备可能蚕食 Mac 销售的问题，这个问题因第一季度 Mac 销售大幅下降而变得更加及时。他重申，供应限制正在导致销量下降，但他也将同类产品的竞争作为一个广泛的话题，指出 iPad 在几个重要方面存在机遇。

库克重申，苹果“永远不要害怕被拆”，因为拆你自己的产品总比让别人拆你的好。但他接着谈到了更大的图景，谈到了个人电脑市场的总体情况。“特别是在 iPad 上，我们拥有所有机会之母，因为 Windows 市场比 Mac 市场大得多，”他说。“我过去说过，我相信平板电脑市场将在某个时候超过个人电脑市场，我现在仍然相信这一点。”

他肯定提到的另一点是所谓的“光环效应”，iPhone 已经被证明具有这种效应，即首次购买苹果设备的人在购买一台设备后，往往会购买其他产品。iPad 也很有可能引发这种现象。

“如果有人买了 iPad mini 或 iPad，这是他们的第一款苹果产品，我们多年来的经验告诉我们，他们很有可能会购买另一款 iPad 产品，”他说。“我们非常有信心这将会发生，我们在 iPad 上也看到了一些证据，所以我认为自相残杀是一个巨大的机会。”

自相残杀是苹果一直信奉的东西，但那是因为取代它的产品总是倾向于比它们推向边缘的产品获得更多的销量。Mac 可能正在走下坡路，但只要 iPad 继续发光发热，苹果及其高层就不会太在意它。
