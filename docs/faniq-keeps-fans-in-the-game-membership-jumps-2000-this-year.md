# FanIQ 让粉丝继续留在游戏中，今年会员人数增加了 2000%

[![](img/948611b1aac3a426548fe7428007817e.png)](https://web.archive.org/web/20221003093450/http://www.faniq.com/)

体育网站 FanIQ 更注重娱乐用户，而不是用 ESPN 等网站上的统计数据和观点让用户陷入困境，这是具有里程碑意义的一年。该网站于 2006 年推出，但直到现在才真正实现飞跃:自 1 月份以来，该网站的增长超过了 2000%，最近达到了 240 万独立用户和 150 万注册会员。与大型体育网站和流行的梦幻联盟相比，这些数字显得苍白无力，但增长率仍然令人印象深刻。

FanIQ 通过提供一系列休闲游戏和社区功能以及更传统的体育头条来区别于其他体育网站。为了鼓励参与，该网站有一个积分系统，奖励写博客帖子和玩网站综合琐事游戏的用户。这些积分只是为了展示(尽管首席执行官 Ty Shay 说许多用户仍然对它们竞争激烈)，但在未来该网站将提供虚拟商品和其他奖励来换取积分。

该网站已经筹集了大约 300 万至 500 万美元的资金(具体金额没有透露)，投资者包括 Vantage Point Ventures、彼得·泰尔、保罗·马蒂诺、基思·拉布瓦和杰夫·弗鲁尔。这个领域的其他初创公司包括 [Open Sports](https://web.archive.org/web/20221003093450/http://www.opensports.com/) (它有一个类似的门户网站)和 [Watercooler](https://web.archive.org/web/20221003093450/http://www.watercooler-inc.com/) 和 [Citizen Sports](https://web.archive.org/web/20221003093450/http://citizensportsnetwork.com/) ，这两家公司都为社交网络设计面向体育的应用程序。
