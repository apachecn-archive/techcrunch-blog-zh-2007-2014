# 三星副总裁 Gavin Kim 跳槽加入 Windows Phone 团队

三星对此不能太高兴。该公司消费者和企业服务副总裁 Gavin Kim 刚刚宣布他将离职，以便加入微软，担任 Windows Phone 团队的总经理。

Kim 已经在三星工作了几年，在担任目前的职位之前，他是内容服务副总裁。在与 [BGR](https://web.archive.org/web/20230205035515/http://www.bgr.com/2011/11/07/samsung-vp-gavin-kim-departs-to-lead-microsofts-windows-phone-product-management/) 的谈话中，金简要概述了他和家人搬到西雅图后他将承担的一些新职责。

“我将负责帮助设定 Windows Phone 平台的未来方向，并加快微软赢得消费者、运营商、设备制造商、开发者和合作伙伴的心的轨迹，”Kim 说。

这些都是崇高的抱负，金先生肯定会有适合他的工作。一份最近的尼尔森报告显示，Windows Phone 的市场份额仅为 7%，远远落后于 Android、iOS 和黑莓操作系统，位列第四。

即便如此，这也没有阻止金看好 Windows Phone 芒果。他认为微软正在“迅速缩小差距，消除对他们在移动领域地位的所有怀疑。”他很可能是对的——美国电话电报公司已经开始在假期前推出新的 Windows 手机系列，诺基亚也准备在明年推出全套 Windows 手机，重新进入美国智能手机市场。如果 Windows Phone 要脱离群体，它必须迅速行动，微软不能在接下来的几个月搞砸了。

尽管我希望他一切顺利，但我还是要给金先生一点建议:小心你发的微博。
