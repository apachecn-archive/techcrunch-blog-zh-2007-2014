# Spotify For BlackBerry 正式发布，仅支持旧硬件

如果你是一名黑莓用户，并且你一直在等待一种比 BBM 音乐更有吸引力的音乐服务，那么你可能很幸运——强调“可能”黑莓的 Spotify 今天正式发布，但是你只能在特定的情况下使用。

根据 Spotify 的博客，新发布的应用程序将只适用于以下黑莓手机用户:Bold 9780、Bold 9700、Curve 9300、Bold 9000 和 Curve 8520。

如果这些手机听起来都不太熟悉，我不能说我责怪你——最新的手机(9780)是一年多前[发布的](https://web.archive.org/web/20221007125814/https://beta.techcrunch.com/2010/11/17/blackberry-bold-9780-now-available-from-t-mobile/)。Spotify 没有明确提到该应用程序是否适用于新款黑莓 7 设备，但 Spotify 的一名代表告诉新款 Bold 9900 的所有者，他可以“偷偷摸摸地尝试一下。”

即便如此，这种偷偷摸摸的方法也不会对所有人都有效。Spotify 的博客提到 CDMA 黑莓手机根本不兼容，Torch 9800 也不兼容。也就是说，拥有更炫硬件的黑莓用户可能仍想尝试一下——一些在 [CrackBerry](https://web.archive.org/web/20221007125814/http://crackberry.com/spotfiy-blackberry-leaves-beta-now-available-most-gsm-devices#comments) 上的用户显然已经让这款应用既能在 Bold 9900 上运行，也能在表面上被禁止的*9800 上运行。*
