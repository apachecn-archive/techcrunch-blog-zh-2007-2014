# Kik 的跨平台消息应用在 Symbian 上首次公开亮相(是的，Symbian)

经过一个月的[私人测试](https://web.archive.org/web/20221003192445/https://beta.techcrunch.com/2011/10/25/kik-debuts-new-symbian-windows-phone-messaging-apps-at-nokia-world/)，可以在 iOS、Android、BlackBerry 和 Windows Phone 上运行的跨平台消息应用 Kik 今天在 Symbian 上公开发布。新的 Kik Messenger for Symbian 带来了改进的群发短信、发送、递送和接收回执，当有人打字时发出通知，支持照片共享，以及找到已经使用 Kik 的联系人的能力。

但是我知道你在想什么:*为什么是 Symbian* ？

根据 Kik 首席执行官 Ted Livingston 的说法，鉴于该应用程序最近回归黑莓设备，Symbian 是“合乎逻辑的下一步”。该公司认为，要成为真正的跨平台应用，你不能只关注领导者 iPhone 和 Android，你必须满足整个智能手机市场的需求。

Livington 说，“坦率地说，我认为放弃 Symbian 是一种非常北美中心主义的观点，而且还为时过早。全球有数亿 Symbian 用户，随着最新一代 Symbian 设备的推出，预计还会有数百万用户加入该平台。”

“这是一个庞大、多样化的全球用户群体，他们渴望以新的方式使用智能手机，”他说。“如果你想真正改变世界交流的方式，你就不能忽视 Symbian 用户。”

就这样。这就是原因。

对于用户采用来说，这也是一个不错的策略。当今许多消息应用程序(或语音应用程序、视频聊天应用程序、“社交”应用程序等)的问题。)的一个问题是，你应该和其他人一起使用它们，但这款应用只能在一两个平台上运行。作为一个 iPhone/Android 家庭，我们对今天以 iOS 为中心的营销焦点感到无尽的沮丧。安卓用户也是人！，我想尖叫。我无法想象可怜的、被忽视的塞班人会有什么感受。(不过说真的，也许你*应该*升级。)
