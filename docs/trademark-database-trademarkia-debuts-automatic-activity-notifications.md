# 商标数据库 Trademarkia 首次推出自动活动通知

![](img/51aba1372432163cce7ce6245ca84bb1.png)搜索引擎 [Trademarkia，](https://web.archive.org/web/20230326220304/http://www.trademarkia.com/)[于 2009 年在 TechCrunch50](https://web.archive.org/web/20230326220304/https://techcrunch.com/2009/09/27/from-the-tc50-demopit-trademarkia-tm-simplifies-trademark-search/) 上推出，是一个包含自 1870 年以来提交的超过 650 万个美国商标的综合数据库，包括死商标。今天，该数据库推出了一项新功能，肯定对初创公司和大公司都有用——自动商标活动通知。

Trademarkia 扫描所有的商标，并在视觉网格中返回结果，这比 USPTO 的商标数据库更容易理解。你可以通过公司、主题、产品类别，甚至申请律师来搜索。公司也可以通过该网站向美国专利商标局申请商标。Trademarkia.com 提供商标标识搜索，允许用户在数百万个标识中搜索特定的特征或设计元素。

有了这项新功能，搜索引擎将通过电子邮件发送有关新商标申请和其他活动的即时通知。任何时候发生活动，如新的备案、批准或“办公室行动”，Trademarkia 都会向您发送电子邮件提醒您选择的公司、品牌或备案。而且服务是免费的。
