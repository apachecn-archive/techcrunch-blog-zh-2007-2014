一些报道称，试图在设备上播放 Flash 视频和应用程序或多或少很糟糕。显然，这种体验从错误到根本不工作不等。这包括 YouTube、Hulu、Vimeo、CNN——基本上是网络上所有主要的视频网站。

现在，必须指出的是，谷歌已经非常明确地表示，Cr-48 是一台测试机。还没准备好大规模生产。此外，内部的处理器(英特尔凌动处理器)不能完全运行 Flash 也不足为奇。也就是说，谷歌也明确表示 Flash 是 Chrome 的重要组成部分(以及 Chrome OS)。事实上，[是烤熟的](https://web.archive.org/web/20230404104302/https://techcrunch.com/2010/06/25/google-chrome-flash/)。你别无选择，只能安装它。

是的，你可以通过 Chrome 禁用该插件，但与苹果的移动设备不同，这种退路基本不存在。

当苹果决定不在他们的新 MacBook Airs 上安装 Flash 时，他们掀起了轩然大波，但不可否认的是，没有 Flash 的机器运行起来更加流畅(无论是从电池寿命[还是网络浏览性能](https://web.archive.org/web/20230404104302/https://techcrunch.com/2010/11/08/adobe-flash-macbook-air/)来看)。也许谷歌应该考虑在第一批面向消费者的 Chrome 笔记本推出时遵循类似的路线。
