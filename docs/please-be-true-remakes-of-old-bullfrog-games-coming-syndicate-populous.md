# 请是真实的:老牛蛙游戏的翻版来了

以前我玩[辛迪加](https://web.archive.org/web/20221006100639/http://en.wikipedia.org/wiki/Syndicate_%28video_game%29)的时候很开心。我仍然把它安装在我的 D: drive 上(就在*星际*和*部落*之间)，游戏的概念和执行真的和 20 年前一样新鲜。即使是图形仍然站得很好。事实上，当我写这篇文章的时候，我几乎可以说服自己，这个游戏的重拍版真的没有必要。但如果这是我们在《仿生突击队:重新武装的 T9》中看到的那种充满爱意的精心翻拍，我完全支持。

Populous 是另一个伟大的游戏，最初是皮特·莫利纽克斯的“建立你的城镇，然后摧毁另一个”游戏。它可能需要改头换面，但游戏性确实被 Molyneux 的后期作品所掩盖，所以最好只是重新绘制一些资产，更新代码，并在各种数字发行网站上以 5 美元的价格发布。

不幸的是，对我们所有人来说，表明这种情况正在发生的信息并不可靠。在最近的一次采访中，Bright Light Studios 的负责人(包括许多已解散的牛蛙的成员)表示，他喜欢旧的牛蛙 IP，并且[希望看到它们被翻拍。但是除此之外，没有承诺，没有预算，什么都没有——除了他眼中的一丝微光。我想我们只能这样了。与此同时，原创就在那里等着你，像](https://web.archive.org/web/20221006100639/http://www.1up.com/do/newsStory?cId=3175526) [FreeSynd](https://web.archive.org/web/20221006100639/http://freesynd.sourceforge.net/index.php) 这样的东西也是如此。

既然我们在挖掘过去，我们能得到那个[爆能大师](https://web.archive.org/web/20221006100639/http://www.metafilter.com/55782/RBeeeak)的翻拍版吗？
