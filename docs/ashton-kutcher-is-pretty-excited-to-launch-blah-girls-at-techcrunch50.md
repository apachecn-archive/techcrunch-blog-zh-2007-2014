# 阿什顿·库彻非常兴奋能在 TechCrunch50 上推出 Blah Girls

 [](https://web.archive.org/web/20230205012559/http://www.blahgirls.com/) [TechCrunch50](https://web.archive.org/web/20230205012559/http://www.techcrunch50.com/) 还有一周半就到了。出席公司的名单在活动当天之前严格保密，以确保最大限度地吸引观众的注意力。但是今年我们破例一次。阿什顿·库彻和杰森·戈德堡的 [Katalyst Media](https://web.archive.org/web/20230205012559/http://www.crunchbase.com/company/katalyst-media) 将在 TechCrunch50 上发布 [Blah Girls](https://web.archive.org/web/20230205012559/http://blahgirls.com/) ，这是他的一款新的互动在线视频产品。

像所有在活动中推出的产品一样，TechCrunch50 将是公众或媒体第一次看到它。我们在几个月前的看到了它，认为它将会大受欢迎。在接下来的几个月里，请关注 Katalyst Media 的更多活动。
