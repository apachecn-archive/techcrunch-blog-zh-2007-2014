# 战争，有什么好处？三分！

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/04/10/gamification-isnt-a-word/>

![](img/a0f3f47419cf3a88bfdd9ee515f100c5.png "adults") *“我为孩童的时候，说话像孩童，悟性像孩童，思想像孩童，到了成年，就把孩童的事丢弃了。”*—**哥林多前书 13:11**

昨晚，我一周以来第一次访问了 BBC 新闻网站。想象一下，当我在拉斯维加斯忙着[和脱衣舞娘](https://web.archive.org/web/20230204143803/http://www.huffingtonpost.com/paul-carr/the-strip-diary-day-four-_b_846495.html)出去玩[看 80 年代男孩乐队明星](https://web.archive.org/web/20230204143803/http://twitter.com/paulgoestovegas/statuses/56942348608876544)的时候，我有多惊讶，我错过了整个联邦政府几乎被关闭的机会。

这种与现实的隔绝既是这个城市的一个问题，也是它的一个卖点:在拉斯维加斯没有发生的事情，就留在拉斯维加斯之外。

因此，在我追踪新闻的过程中，我颇感兴趣地注意到了 PBS 创意实验室的标题:“[为什么新闻编辑室抵制创建新闻游戏？](https://web.archive.org/web/20230204143803/http://www.pbs.org/idealab/2011/04/why-are-newsrooms-resistant-to-creating-newsgames097.html)“在一篇论证充分的文章中，克里斯·奥布莱恩概述了新闻机构应该探索将视频游戏元素纳入报道的各种原因。

> “大多数美国人在手机上玩某种视频游戏:游戏机、浏览器。如果你一个游戏都不玩，那你就是越来越少的少数群体中的一员。许多这种形式的游戏，尤其是社交、移动和休闲游戏，现在已经深入到所有年龄的主流观众中。
> 
> 这些游戏绝不仅仅是琐碎或简单的娱乐消遣，它们提供的好处应该会吸引任何新闻编辑室。最好的游戏能创造深度参与，它们具有强烈的社会性，在某些情况下，它们展示了从数字内容中赚钱的新途径。这些项目中的任何一个都会引起出版商的共鸣。"

确实如此。事实上，我不能不同意奥布莱恩说的一句话:大多数成年人确实玩游戏，反复的研究表明，很大一部分人对直接的新闻报道感到厌恶，而对各种形式的游戏机制感到兴奋。Zynga 和 Rovio 等公司已经从社交游戏中赚了数十亿美元:tham thar villes 中有金子。

然而，尽管我同意奥布莱恩的观点，我还是忍不住回答他的修辞标题:

因为不是一切都是游戏。

可能是我老了。当然，我是一个老媒体记者势利小人。但事实是，当面对越来越多的人无法在没有游戏元素的情况下处理新闻的事实时，我的直觉是回答……好吧……去他们的。

曾经是这样的:一个世界分成了两部分，一部分人拥有阅读和消化当天新闻所需的智力和注意力，并理解这样做的重要性，另一部分人是永久的孩子:骄傲的白痴，他们会说“我对政治不感兴趣”(尽管已经完全抵押了)，或者问“我为什么要关心几千英里之外的国家发生的事情？”(尽管这些国家拥有所有债务)。你可以从他们的口水、保险杠贴纸，以及他们对视频游戏的长期沉迷来判断这些人，这是他们逃避残酷现实的一种方式。

然而，越来越多的人都是游戏玩家；如果不是通过传统的电子游戏，而是通过现实生活中日益增长的“游戏化”(ugh)。和朋友们一起享受一顿[大餐已经不够了](https://web.archive.org/web/20230204143803/https://techcrunch.com/2011/02/21/phones-at-dinner/):除非我们登记入住(也许——*喘气！*–成为市长！)体验是空洞的，没有意义的。我们喝的每一瓶汽水，打开的每一盒麦片，或者我们得到的每一包可卡因都印有自己的二维码，允许我们解锁点数或赢得一些无关的东西；不然为什么要选择那个品牌？生活中几乎没有一个领域——金融、[税收](https://web.archive.org/web/20230204143803/http://hplusmagazine.com/2010/03/25/gamification-turning-work-play/)、[教育](https://web.archive.org/web/20230204143803/http://www.forbes.com/2010/10/28/education-internet-scratch-technology-gamification.html?boxes=Homepagechannels)、[健康](https://web.archive.org/web/20230204143803/http://www.wired.co.uk/news/archive/2010-09/07/health-month-rpg)不被游戏化。现在，很明显，轮到新闻了。因为，说实话，除非我能扣动扳机，亲自把卡扎菲揪出来，否则谁会在乎呢？

当然，至少在最后一点上，我夸大了。奥布莱恩并不是建议所有的新闻都应该游戏化——只是说记者不应该害怕让它成为他们的武器库的一部分。同样，人们不是为了与朋友社交，或者重新安排他们的财务，或者学习新语言*而仅仅是*来赚取积分:游戏元素只是一种有趣的方式，鼓励他们更频繁地做这些事情，或者引导他们选择某些品牌或场所。

然而……

游戏的受欢迎程度——以及玩游戏所花的时间——[持续增长](https://web.archive.org/web/20230204143803/http://www.huffingtonpost.com/jane-mcgonigal/video-games_b_823208.html)(普通年轻人在 21 岁时玩游戏的时间达到 1 万小时)。与此同时，你只要看看《每日秀》( Daily Show)、福克斯新闻频道或迈克尔摩尔(Michael Moore)的成功，就能意识到，观众越来越喜欢从用娱乐来粉饰现实的来源获取新闻。随着新闻机构努力吸引和留住那些宁愿花时间种植虚拟作物也不愿做得…好的观众..几乎任何其他东西——从“游戏化是一个有用的附加产品”到“如果它不是一个游戏，人们就不会玩它，我们都会破产”之间的斜率是不稳定的。

在这个斜坡的顶端是一个成熟的世界，我们足够聪明地意识到，某些决定(保持消息灵通、照顾我们的健康、监控我们的财务状况)应该因为它们重要而不是因为它们有趣而做出。在斜坡的底部是一个反乌托邦的未来，我们生活的每个方面——从我们吃的食物，到我们形成的关系，到我们处理的信息——都被一个幼稚的问题所驱动:有趣吗？一个像流口水、咯咯笑的彼得潘一样，我们永远也不会长大的世界。

当然，我们能在这个斜坡上滑多远，完全取决于人类选择追求信息本身的能力，而不是总是追逐乐趣和奖励的幼稚要求。

是啊。

如果你在接下来的几天里在拉斯维加斯的大街上遇到我，你会明白我看起来有点害怕。
