# 美国电话电报公司虎视眈眈菜网潜在收购？

在美国电话电报公司/T-Mobile 的合并不光彩地结束前不久，Dish Network 的首席执行官 Joseph Clayton 漫不经心地表达了他对与 T-Mobile 在 T2 合作的兴趣，以此为 Dish 的客户带来无线语音服务。

如果美国电话电报公司参与其中，这些计划可能不会成功。来自[彭博](https://web.archive.org/web/20221006062428/http://www.bloomberg.com/news/2012-01-17/at-t-s-need-for-spectrum-signals-dish-bid-at-decade-high-premium-real-m-a.html)的一份新报告表明，AT & T 显然非常缺乏额外的频谱，以至于他们正在考虑支付“十多年来最高的溢价”来收购卫星电视提供商和他们持有的频谱。

曾经是机会主义者的克雷顿似乎接受了 at 收购的可能性——他上周在接受彭博·韦斯特采访时指出，公司“对所有可能的选择都持开放态度”

“我们可能被收购，也可能成为收购者，”他补充道。

克莱顿最初的愿景是创建一个由视频、语音和数据服务组成的“全国无线网络”。为此，克莱顿表示，他将考虑与 T-Mobile 或 Sprint 等公司合作，以实现这一目标。他们将获得 Dish 的频谱所有权，Dish 将获得他们在管理语音网络方面的专业知识。

如果美国电话电报公司真的收购了 Dish，Dish 是否能够提供这些服务还值得怀疑，因为这将对美国电话电报公司自己的网络服务组合构成切实威胁。尽管如此，在这一点上，美国电话电报公司显然更需要迪什，而不是迪什需要美国电话电报公司。

Clayton 似乎非常乐意与较小的运营商建立合作伙伴关系，以实现他的网络愿景，但如果没有 T-Mobile 的网络支持，美国电话电报公司在推出 LTE 网络时可能会面临坎坷的道路。作为他们公开分手的结果，美国电话电报公司不得不把他们持有的一部分频谱让给德国电信，这当然于事无补。
