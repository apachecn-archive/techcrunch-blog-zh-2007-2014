# 未来 6 个月的特性现在就在脸书的代码中(但是我们看不到)

几天前，[脸书在他们的总部举办了一场科技讲座](https://web.archive.org/web/20230117005529/http://www.facebook.com/video/video.php?v=10100259101684977&oid=9445547199&comments)。谈话的主题是推动变革——缺陷修复、新特性、产品改进等。每天，脸书的工程师都会推出数百个这样的产品；有些大，有些小。6 亿多用户中的大多数从来不会注意到任何事情。很明显，我们甚至不太可能注意到由于脸书的一个特点而产生的变化。“除了 TechCrunch，其他人都可以看到”功能。

正如脸书工程师 Chuck Rossi 在视频中大约 23:00 分钟的细节，脸书有一个他们称为“看门人”的工具，可以让他们控制谁可以在任何给定的时间看到服务上的代码。正如 Rossi 指出的那样，现在在 Facebook.com 已经有了*的代码，脸书将在未来六个月甚至更长时间内推出的每一个重大项目都有了代码！是看门人阻止我们看到它。*

我指的是“我们”。虽然一些把关者参数是显而易见的——按国家、年龄、数据中心过滤——但有一个参数非常有趣。“我最喜欢的一句话是‘除了 TechCrunch 的人，其他人都可以看到这个’”，Rossi 说。他是认真的。

脸书所做的很可能只是把我们所有的个人资料放在一个永远没有资格看到隐藏代码的人的名单上。当然，[那也不一定管用](https://web.archive.org/web/20230117005529/https://techcrunch.com/2010/05/09/facebook-places-check-in/)。但这也是允许他们与我们一起[【推出】新的传真服务](https://web.archive.org/web/20230117005529/https://techcrunch.com/2009/09/10/facebook-now-lets-you-fax-your-photos-i-have-no-idea-why-anyone-would-want-to-do-this/)的相同类型的功能，即使其他人看不到它。有趣的事情。

我们感谢脸书对细节的关注，让我们远离他们的代码。当然，现在他们真的是自找的。他们真的认为你不能在脸书做假账吗？当然…请继续关注接下来六个月来自脸书的特写…

整个演讲非常精彩，值得一看，我们已经嵌入到下面了。
