# 为什么中国为电子商务做好了准备

中国的电子商务已经准备好起飞，更重要的是，它已经准备好以自己的方式达到伟大的高度。就在我们说话的时候，拉梅的陆东、的叶海峰和绿盒子的吴芳芳是三个电子商务先驱，他们正在重新定义中国的网上销售。

“中国已经为电子商务做好了准备，”吕东说。"人们几乎开始在网上购买任何东西。"

叶海峰同意了。“在中国市场，我们有更大的机会来创造新的东西。在美国，电子商务市场已经相当成熟，”他表示。“我们可以利用电子商务建立一种新的零售模式。”他称网上销售是一个巨大的机会。

三人一致认为，中国电子商务海洋中最大的鱼是[淘宝](https://web.archive.org/web/20221226095254/http://www.taobao.com/index_global.php)，淘宝作为中国销售平台的重要性不容忽视。该公司在结算、物流和服务方面提供帮助，三人都认为淘宝“很棒”

这些企业家认为未来几年情况会有所改善，并指出广告费率上升了 30-50%，而外部投资略有下降，但不足以让这些小组成员担心。

吕东说得好:“筹集资金很重要，但赚钱更重要。”
