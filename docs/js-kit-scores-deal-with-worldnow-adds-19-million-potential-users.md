# JS-Kit 分数处理 WorldNow，增加 1900 万潜在用户

[![](img/6424d008777742068c34559a39dd946f.png)](https://web.archive.org/web/20221006052608/http://www.crunchbase.com/company/js-kit)

[WorldNow](https://web.archive.org/web/20221006052608/http://www.worldnow.com/) 是当地电视新闻网站的领先内容管理服务提供商，每月访客总数超过 1900 万，该公司与 [JS-Kit](https://web.archive.org/web/20221006052608/http://js-kit.com/) 合作，为他们提供大量基于社区的插件，这些插件将嵌入到他们的网站中。JS-Kit 成立于 2006 年，创建了简单的小部件，只需要几行代码就可以实现(你可以在这里看到我们对它们的介绍性回顾)。

除了 JS-Kit 的旗舰评论工具外，WorldNow 还将为他们的网站添加评分、分数、评论、导航和投票。WorldNow 有 320 个电视客户，包括许多美国广播公司和全国广播公司的附属公司。来自 JS-Kit 插件的广告收入将由两家公司平分。

JS-Kit 首席执行官 Khris Loux 认为这种合作关系是该公司迈出的一大步，将大大增加其影响力。虽然他承认 WorldNow 的 1900 万读者中有许多人可能不会使用评论系统(至少在一开始)，但他表示该公司的用户群仍将有非常可观的增长。

这可能是事实，但很难衡量他们能从 WorldNow 庞大的读者群中获得多少——很少有人习惯于加入新闻网站的社区。Loux 还认为，随着经济的收费氛围，公司将更有可能转向像 JS-Kit 这样的公司来建立评论系统，而不是建立自己的内部系统。

作为一家公司，JS-Kit 显然正在成熟，已经从一个基本的评论系统发展成为一个功能齐全的套件，包括 API、安全性和 OpenID 集成。一月份，该网站还宣布与 T2 的 HaloScan 合作，为 489，300 个网站提供易于部署的评论系统。这些合作关系应该使他们在与这个领域的其他参与者较量时处于有利地位，这些参与者包括 [Disqus](https://web.archive.org/web/20221006052608/http://www.disqus.com/) 、 [SezWho](https://web.archive.org/web/20221006052608/http://www.sezwho.com/) 和 [IntenseDebate](https://web.archive.org/web/20221006052608/http://www.intensedebate.com/) 。
