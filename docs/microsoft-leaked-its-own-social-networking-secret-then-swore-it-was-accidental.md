# 微软泄露了自己的社交网络秘密，然后发誓这是意外

紧随谷歌+的疯狂之后，[微软](https://web.archive.org/web/20230205033733/https://techcrunch.com/tag/microsoft)已经“无意中”泄露了自己的社交网络平台。无论这是炒作还是某个 IT 人员的真正错误，我们仍然有一个问题:Tulalip 到底是什么？我的孩子……[Fusible](https://web.archive.org/web/20230205033733/http://fusible.com/2011/07/new-microsoft-social-search-service-called-tulalip-revealed-on-socl-com/)最先了解到这个故事，它发现图拉利普这个名字也是一个位于微软大本营华盛顿州雷德蒙附近的美国土著部落的名字。

据 Fusible 报道，下图中的预告页面今天发布在 socl.com[的网站上，该网站显然属于微软。有传言称微软是 Social.com 的幸运买家，这意味着他们仅域名一项就支付了 260 万美元。](https://web.archive.org/web/20230205033733/http://www.socl.com/)

[![](img/30bf1040a03fd5cbd5f27974750a2b98.png "tulalip-575x342")](https://web.archive.org/web/20230205033733/https://techcrunch.com/wp-content/uploads/2011/07/tulalip-575x342.jpg)

从表面上看，这很可能是一次“社交搜索”的冒险，由 Bing 主持。“有了 Tulalip，你可以比以往任何时候都更容易地找到你需要的东西并分享你知道的东西，”这是一句挑逗性的标语。注意到 Find 和 Share 的大写了吗？是的，我们非常确定这将是搜索和分享的结合点，这是一个好消息，因为你不能在 Google+中进行[谷歌](https://web.archive.org/web/20230205033733/https://techcrunch.com/tag/google)搜索(那到底是怎么回事？).奇怪的是，在预告页面上有[脸书](https://web.archive.org/web/20230205033733/https://techcrunch.com/tag/facebook)和[推特](https://web.archive.org/web/20230205033733/https://techcrunch.com/tag/twitter)登录字段，这让我们相信这将是对社交网络的一次相当轻松的尝试，而不是一次沉重的微软体验。如果你打不过他们，就加入他们，好吗？

该预告页面已经被撤下，而微软发誓这是一个意外。他们把这句话换成了:“谢谢你的来访。Socl.com 是微软研究院一个团队的内部设计项目，它被错误地发布到了网上。我们不是故意的，真的。”好吧，我们还不确定我们是否会相信“我们不是故意的”这部分，但不管怎样，我们很高兴它发生了。
