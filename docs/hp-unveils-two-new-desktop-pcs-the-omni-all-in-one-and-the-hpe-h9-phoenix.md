# 惠普推出两款新台式电脑:Omni 一体机和 HPE h9 Phoenix

超极本在 2012 年风靡一时，我们可能会在下周的 CES 上看到相当多的超极本。但仍有一些人喜欢更强大的计算体验，这就是为什么惠普今天宣布了 Omni 27 英寸一体机和迄今为止最强大的 Pavilion HPE h9 Phoenix。

事实上，几周前，我非常幸运地接触到了这些小狗，27 英寸的 Omni 是一个值得一看的景象。它很时尚，看起来结构良好，符合我内心的极简主义。该电脑拥有您期望一体机具备的所有端口，包括 HDMI-in、蓝光光盘驱动器、两个 USB 3.0 端口、四个 USB 2.0 端口、一个 SD 读卡器和一个以太网插孔。

您还可以找到高达 2TB 的内部存储，以及英特尔酷睿 i5 2400S 处理器和 Beats Audio。27 英寸的屏幕可以倾斜和调节，这样每个人在电脑前都很舒服。惠普还包括其 LinkUp 软件，可以让你通过 WiFi 网络在设备之间同步内容。

基本款起价为 1199 美元。

[![](img/cbc61af8f3c82821adaf99ef213be817.png "Phoenix")](https://web.archive.org/web/20221209044242/https://beta.techcrunch.com/wp-content/uploads/2012/01/hpe-phoenix4.jpg) 另一方面，凤凰在设计部门确实给事情增添了情趣。这并不完全符合我的口味，但我很高兴看到惠普在设计方面变得更加激进。它是惠普 Pavilion E 系列的最新产品，装有 FX-8100 (8 核)处理器、AMD 镭龙 7670 显卡、160GB 固态硬盘和 8GB 内存。

连接并不短缺，有各种形状和大小的存储卡读卡器，两个 USB 3.0 端口，四个 USB 2.0 端口和两个光驱。包括无线鼠标和键盘，这也是 Omni 的情况。

基本单元起价 1149 美元。
