# Tradeshift 提升了企业即时支付的档次

[Tradeshift](https://web.archive.org/web/20230204222130/http://tradeshift.com/) ，旨在成为一种 B2B 流程社交网络的免费发票平台，[去年曝光](https://web.archive.org/web/20230204222130/http://eu.beta.techcrunch.com/2010/02/25/tradeshift-launches-to-disrupt-an-entire-financial-system-and-morten-lund-is-behind-it/)，此后从[idea Capital](https://web.archive.org/web/20230204222130/http://www.crunchbase.com/financial-organization/notion-capital)以及其他公司 [PayPal](https://web.archive.org/web/20230204222130/https://techcrunch.com/2010/10/11/paypal-invests-tradeshift-lanng/) 筹集了 700 万美元。

被称为“Skype for invoicing”的 Tradeshift 正在加速解决一个大问题。快速获得支付可能是当今公司面临的最大问题，其新的即时支付解决方案旨在解决这一问题。这项新服务将在测试阶段提供给选定的供应商，并将在未来六个月内推出，但是，我们可以为 Techcrunch 读者提供独家早期访问。要获得资格，请注册一个免费的 Tradeshift 帐户，然后前往[tradeshift.com/instantpayments](https://web.archive.org/web/20230204222130/http://tradeshift.com/instantpayments)
