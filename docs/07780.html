随着ShareMyPlaylists获得天使投资，Spotify生态系统不断壮大

任何新的美国Spotify用户都将忙于创建播放列表，并与朋友分享。这是服务中令人敬畏的一部分，也是Spotify与Mog和Rdio等其他服务相比变得如此令人上瘾的真正原因。不久前，一家聪明的小初创公司意识到他们的列表中有黄金，并创建了一个网站，允许用户更容易地在Spotify之外分享他们的列表。

ShareMyPlaylists 现在已经获得了25万英镑的天使投资，这次来自天使投资人马克·皮尔森，Markco Media Ltd / MyVoucherCodes的董事长。培生集团还投资UGC媒体网站 Blottr 。
