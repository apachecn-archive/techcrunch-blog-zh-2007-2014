苹果公司准备从应用程序商店撤出Evi。它是否拒绝了Siri？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2012/02/26/apple-prepares-to-pull-evi-from-app-store-did-it-slap-down-siri/

上个月我们报道了关于 Evi 的发布，这是一款新的iPhone ( iTunes链接)和Android应用(链接)，其功能非常类似Siri。也许是因为太擅长模仿Siri了？

是的，Evi做不到Siri在你的iPhone日历上添加东西或挂钩提醒的技巧。但是，问“我怎么做苹果派？”Evi提供了一个带有网络链接的食谱列表。Siri无法直接回答你，只会问你是否要搜索网页。也许是因为Evi非常好，看起来与Siri相差无几，苹果告诉True Knowledge，它将从App Store中撤出Evi。等等，什么？

苹果已经多次批准了appstore的Evi版本——包括Siri之后的版本。然而，世界移动通信大会(Mobile World Congress)上的传言是，有不祥的迹象表明，苹果对竞争感到不满，包括发布后提交的update True Knowledge(它只是增加了一个历史搜索功能)。这花了三周时间才获得批准。

周五晚上，True Knowledge接到了苹果公司代表理查德·奇普曼的电话。(如果你在谷歌上搜索理查德·奇普曼的名字，你会发现他也是苹果的代表，负责打关于应用程序的有争议的电话。

他告诉True Knowledge，苹果“将把Evi从appstore下架”,因为它与Siri相似。

被引用的规则是App stro T&Cs中的第8.3条“与现有苹果产品或广告主题混淆相似的应用将被拒绝”。

讽刺的是，Siri和Evi都许可了Nuance的语音识别技术。但是Siri背后没有Evi的真知的搜索引擎。

True Knowledge首席执行官威廉·汤斯顿-佩多告诉我:“我不认为需要太多的想象力就能意识到‘令人困惑的相似’是‘竞争对手’的代名词，所有用户和媒体的评论都是‘现在你不需要买4S了——你可以下载Evi’，‘比Siri更好’等等。导致苹果公司改变主意，不允许用户使用这款应用程序。”

苹果从商店里撤下一个应用程序无疑是非常激进的。通常的流程是拒绝应用程序(以书面形式)，并允许开发人员修复。然而，看起来苹果的路线本质上是修复Evi不是一个选项。

取消这一应用还将导致近20万已经购买了该应用的苹果用户无法获得更新。

Evi现在仍然在苹果应用商店上，但它很可能不会存在很长时间。

Android版本的Evi当然不受影响，这意味着Android用户将获得好处，但iPhone用户被告知，要么是Siri，要么什么都不是。
