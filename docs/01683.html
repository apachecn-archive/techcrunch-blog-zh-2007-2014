NHK展示超高清(7，680 x 4，320像素)原型系统

T2】

日本国家电视台 NHK 在新加坡举行的科技博览会上展示了一套令人惊叹的投影系统。

NHK使用了2台Lcos投影仪(8000流明)和一个6.6 x 3.7米的屏幕来展示他们的3300万像素、8K视频投影技术(7,680 x 4,320像素)。超高清格式产生的图像是全高清格式的四倍宽，四倍高。NHK还在演示屏幕上连接了22.2声道的影院音响系统。

该电视台使用了用自主开发的8K演播室摄像机录制的镜头。这是NHK首次在亚洲(不包括日本)向公众展示他们的技术。
