# iOS 5 拥有强大的人脸检测功能

在苹果购买面部识别软件供应商极地玫瑰(Polar Rose)后，我们不确定苹果的计划。现在，通过一点点[9 到 5 毫秒的挖掘](https://web.archive.org/web/20230203085642/http://9to5mac.com/2011/07/25/face-detection-software-and-api-lands-in-ios-5-following-apples-2010-purchase-of-polar-rose/)我们知道了面部识别(FaceRec？FaceTimeRec？面部抓取？)将被深深植入 iOS 5，并支持许多巧妙的功能，包括可能与 Lion 的 Photo Booth 应用程序中相同的古怪效果。

例如，Photo Booth 利用 Polar Rose 的技术添加了在你头上飞来飞去的鸣叫鸟，并特别改变了你的眼睛或鼻子，而不是整个脸。这一功能已经在一些网络摄像头上使用了几年，现在才慢慢进入到封闭的苹果世界。

API 支持[对嘴部位置的调用](https://web.archive.org/web/20230203085642/http://developer.apple.com/library/prerelease/ios/documentation/CoreImage/Reference/CIFaceFeature/Reference/Reference.html#//apple_ref/occ/instp/CIFaceFeature/mouthPosition)以及左眼和右眼位置。在操作系统本身中没有这些 API 调用的明确例子，但是开发者可以，例如，使用它们来制作更好、更准确的屁应用程序，并将[对话汤姆](https://web.archive.org/web/20230203085642/http://itunes.apple.com/us/app/talking-tom-cat/id377194688?mt=8)带到令人眩晕的新高度。
