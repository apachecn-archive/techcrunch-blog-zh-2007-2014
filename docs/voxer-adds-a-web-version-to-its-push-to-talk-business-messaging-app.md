# Voxer 为其一键通商业消息应用| TechCrunch 增加了一个网络版本

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/14/voxer-adds-a-web-version-to-its-push-to-talk-business-messaging-app/>

[Voxer](https://web.archive.org/web/20230328030828/http://www.voxer.com/) ，这个在 2013 年开始时声名狼藉的移动消息应用程序(由于[脸书切断了](https://web.archive.org/web/20230328030828/https://techcrunch.com/2013/01/24/my-precious-social-graph/)其 API 访问)，今天在向付费企业一键通和聊天产品的转型中迈出了下一步:该公司正在推出 [Voxer for Web](https://web.archive.org/web/20230328030828/https://business.voxer.com/) ，这是一个将该公司的付费 Voxer for Business iOS 和 Android 应用程序与桌面版连接起来的服务版本。

作为该公司更大的 Voxer for Business 服务的一部分，其价格目前为 4.95 美元/用户/月，但在 2014 年 1 月 1 日之后，注册费用将上涨至 9.95 美元/用户/月。网络版本将帮助 Voxer 从一个可以用作人对人通信系统的产品转变为一个更统一的产品，也包括那些在固定场所并需要同时与许多工人通信的人。想想出租车或其他送货服务的调度员。(至于价格上涨，Voxer 还表示，对于那些在 12 月底之前注册的人，他们将获得适用于 2014 年 10 月之前的较低定价选项)。

Voxer for Web 基本上从该公司现有的商业用户移动应用程序中提取了一些功能，并将其扩展到台式电脑，能够推送直播和消息音频，发送图像和短信以及聊天。新增功能包括能够在一个屏幕上打开多个聊天；以及整个团队发送的语音消息、文本和图像。用户还可以在实时模式下收听多个聊天——类似于你在质量较低的现有调度服务中获得的内容。

Voxer 公司的首席运营官·伊塔马尔·坎德尔告诉我们，该公司已经根据企业的要求推出了网络版。他说:“我们从各行各业的客户那里了解到，他们需要让移动工作人员和办公室工作人员能够协作。“Voxer for Web 通过提供统一的体验来解决这一问题，同时还包括独特的功能，使管理员和调度员能够从一台计算机上监控多个对话、人员或团队。”

他指出，这一想法不仅意味着功能的增加，还意味着 Voxer 瞄准的业务种类的增加。他说:“我们的愿景是，移动设备将适合移动工作人员，而台式机则非常适合拥有大量工作人员的企业，他们需要与调度员等核心人员快速沟通。”。“通过增加从桌面发送消息的功能，我们的潜在市场扩大了十倍，现在不仅可以满足移动员工的通信需求，还可以满足整个组织的需求。Voxer for Web 将作为移动和固定劳动力之间的连接组织。

Voxer 的消费产品——区别于 WhatsApp 等其他通讯应用，主要专注于基于语音的交流——已有数千万次下载，2012 年下载量激增[，同年它筹集了](https://web.archive.org/web/20230328030828/https://techcrunch.com/2012/01/10/popular-like-voxer/)[3000 万美元的巨额资金](https://web.archive.org/web/20230328030828/https://techcrunch.com/2012/04/11/walkie-talkie-app-voxer-goes-big-ivp-and-intel-lead-30-million-round/)。

该公司今天没有透露有多少企业已经签署了 Voxer 的业务(我们正在询问)。它不太可能像消费品那样批量生产，但它代表了公司的几个不同的重要发展。首先也是最显而易见的是，这家初创公司的产品瞄准了付费受众，同时推动 Voxer 成为一家企业级公司，拥有更高的服务水平、安全性和一系列付费客户需要和要求的功能。

第二个是 Voxer 如何成为众多利用更大的消费化趋势的公司之一，将最初为普通消费者创造的产品应用于企业市场——但这是一个转折。Voxer 也是一个标志，表明实际上甚至消费产品也可以来自企业的开端:创始人汤姆·卡蒂斯显然是第一个[想出这个主意](https://web.archive.org/web/20230328030828/http://www.cnn.com/video/#/video/bestoftv/2012/01/05/nr-simon-voxer-walkie-talkie-app.cnn)的人，他是阿富汗军方的一名通信专家。

在转向商业用户的过程中，Voxer 并没有完全进入未知领域。它与这些服务竞争的还有 Sprint 等消费者应用的起源，以及 Nextel 的一键通服务。

对 Voxer 来说，也许更重要的是，有许多其他消息服务和调度消息服务专门针对与它相同的市场。以出租车服务为例，这是 Voxer 的主要目标之一。(在宣布新网络版本的新闻稿中，它引用了 Future Cab 的所有者汤姆·布伦南(Tom Brennan)的话:“Voxer 在我的公司中发挥了核心作用，使我的团队能够不断地相互联系并与我联系。”显然，他住在弗吉尼亚海滩，并在纽约经营一项服务。)现在已经有几十家公司了，一些公司正在开发需要特殊车载设备的遗留系统；但是更多的人依靠移动网络和手机的爆炸来运营服务。竞争对手包括 [T 调度](https://web.archive.org/web/20230328030828/http://tdispatch.com/)和 [PC 调度](https://web.archive.org/web/20230328030828/http://www.pc-dispatch.com/)(我猜是势均力敌)[数字调度](https://web.archive.org/web/20230328030828/http://www.digital-dispatch.com/)、[出租车司机](https://web.archive.org/web/20230328030828/http://www.taxicaller.com/fleets.php)、 [AutoCab](https://web.archive.org/web/20230328030828/http://www.autocabinternational.com/) 等等。

Voxer 的额外功能，让用户发送图片等，它在消费者方面的强大品牌认知度，以及 Katis 与风投资金和后续网络的联系，是否足以让它脱颖而出并赢得新客户，都值得关注。

迄今为止，Voxer 已经筹集了约 3000 万美元的资金，支持者包括机构风险投资伙伴、英特尔投资、SV 天使、TC 创始人迈克尔·阿灵顿的 CrunchFund、Chris Dixon、Roger McNamee、Windcrest Partners 和 Webb 投资网络。
