# 谣言:苹果正在秘密测试苹果电视，不急于推出

在过去的几年里，关于潜在的苹果电视机已经说了很多，但是来自富士康的消息称苹果的高清电视已经开始测试了。这位不愿透露姓名的消息人士解释说，苹果正在使用 46 英寸到 55 英寸的液晶面板。

然而，现在还不要太兴奋。据直接与消息来源对话的 [FocusTaiwan](https://web.archive.org/web/20221005183200/http://focustaiwan.tw/ShowNews/WebNews_Detail.aspx?Type=aALL&ID=201212190038) 称，2013 年发射不太可能。

尽管苹果公司称其更多的是一种“爱好”，但关于苹果电视[的传言已经流传多年了自那以后，蒂姆·库克](https://web.archive.org/web/20221005183200/https://beta.techcrunch.com/2012/08/18/apple-tv-deep-thoughts-by-jack-handey/)[公开暗示](https://web.archive.org/web/20221005183200/https://beta.techcrunch.com/2012/05/29/apple-tv-pulling-the-strin/)苹果肯定有开发电视机的计划，但没有透露更多细节。

到目前为止，我们已经听说苹果已经开始测试电视的设计，但这将是富士康的第一次泄露，富士康是中国制造商，开发了苹果的大多数标志性产品，如 iPhone 和 iPad。

但是，不要太激动。苹果电视机顶盒基本上是从苹果货架上飞下来的，直到他们有一款产品击败三星的电视产品，仍然有时间等待。

看，苹果和三星几乎在每个消费电子产品领域都有竞争，但电视是关键。用户不一定在单一领域寻找最好的产品，而是跨电子产品的连接体验。随着三星和苹果在移动领域占据了最大的市场份额，真正的战争开始于客厅。
