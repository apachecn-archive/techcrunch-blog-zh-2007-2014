# 谷歌准备开启 Chrome 扩展

对于那些真的想换成 Chrome 但又害怕过没有扩展的日子的火狐用户来说，这是个好消息，那一天就要到了。或者至少是接近。Google today [宣布](https://web.archive.org/web/20221006191952/http://blog.chromium.org/2009/09/extensions-status-on-runway-getting.html)从现在开始，在所有新的开发者版本中(在 Windows 中)默认开启 Chrome 的扩展支持。

当然，由于扩展支持以前很大程度上是隐藏的，所以 Chrome 上可以使用的扩展并不多。但是谷歌在这里列出了一些例子。其中包括一些简单的东西，比如 Gmail 收件箱检查器，以及谷歌阅读器的 URL 自动订阅按钮。

谷歌表示，这是 Chrome 特有功能“启动过程”的第一步。当然，值得注意的是，常规(非开发)Chrome 用户停留在 2.x 版本，而开发版本现在是 4.x，正如读写网[昨天指出的](https://web.archive.org/web/20221006191952/http://www.readwriteweb.com/archives/google_chrome_extensions_get_polish_getting_ready_for_wide_release.php)。所以现在还不清楚 Chrome 3.0 是否会有扩展，无论何时正式发布(目前已经在测试版中发布)。

谷歌还指出，它已经获得了一些帮助，以加快 Mac 和 Linux 版 Chrome 的扩展速度。在 Chromium for Mac 的最新版本中，扩展支持并没有突出显示，但是你可以通过在 URL 框中输入“chrome://extensions/”来获得它。
