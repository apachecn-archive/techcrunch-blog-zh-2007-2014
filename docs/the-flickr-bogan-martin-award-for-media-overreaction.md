# 因“媒体过度反应”获得 Flickr Bogan-Martin 奖

关于 Flickr 团队，你可以说一件事——他们之间有些争斗。他们显然对[我们的报道](https://web.archive.org/web/20221208144028/https://beta.techcrunch.com/2010/07/20/flickr-awards-this-years-grant-pattishall-award/)他们的年度(和非官方) [Grant-Pattishall 奖](https://web.archive.org/web/20221208144028/http://grant-pattishall-award.com/)不太满意，该奖每年颁发给“以最引人注目的方式破坏 Flickr”的雅虎工程师我不确定为什么，我觉得这个奖很有趣。

所以现在他们有了一个新的奖项，叫做[博根-马丁奖](https://web.archive.org/web/20221208144028/http://bogan-martin-award.com/) : *“博根-马丁奖每年颁发给 Flickr 的员工，他们无意中引起了媒体对个人评论或内部笑话最引人注目的过度反应。”*

那么谁赢了？丹尼尔·博根也是今年的另一个奖项的得主。去年是克里斯·马汀。两个获奖者的名字链接到我们以前做过的职位。暗示我们是参与了惊人的过度反应的媒体。

好吧，Flickr。这一轮你赢了。
