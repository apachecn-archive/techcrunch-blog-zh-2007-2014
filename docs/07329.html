WordPress 3.2发布到野外；24小时内下载超过33万次TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2011/07/05/wordpress-3-2-released-into-the-wild-downloaded-more-than-330k-times-in-24-hours/amp/

WordPress 3.1在不到5个月的时间里被下载超过1500万次。但时间在前进，音乐也在前进。昨天，也被称为“格什温”的WordPress 3.2 向公众发布，在短短24小时内，这个网站和博客平台的最新版本被下载超过33万次。这些天他们成长得相当快。

为什么这么多人下载最新的迭代？据马特·莫楞威格和公司介绍， WordPress 3.2 的目标是“更轻、更快”，也就是说格什温在移除额外代码、重写某些查询以优化速度等方面做了大量工作。

在新版本中，用户界面也做了一些调整，如下文3.1(左)和3.2版本的比较所示:

3.2版本的另一个很酷的功能是WordPress所说的“不受干扰的写作”(DFW)，它取代了“全屏”模式，用一种新的模式让仪表板在用户写作时淡出背景，以提供一种不那么混乱的观看体验。该团队也将此称为“禅模式”，显然用户感受到了禅的力量:

根据Mullenweg的说法，除了试图让整个WordPress体验更快，并对管理设计进行一些调整，该团队还更新了默认主题，以兼容微博、HTML5和任何屏幕尺寸。所有急需的改进。(为了完全公开，TechCrunch运行在WordPress上。)

这是WordPress的第15个主要版本，网站平台现在拥有超过15，000个插件。更重要的是，WordPress 3.2将不兼容Internet Explorer 6，微软实际上似乎对很高兴。

WordPress的贡献者现在也将在一个“信用屏幕”中获得信用，因此最终信用可能会在信用到期时给予，特别是对于所有那些参与了15个WordPress迭代和计数的开源开发的人。
