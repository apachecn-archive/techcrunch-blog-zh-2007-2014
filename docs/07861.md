# 我在 MySpace 学到的五件事可以帮助 Google+TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/26/five-things-myspace-google-plus/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+Techcrunch+(TechCrunch>)

***编者按**:这篇客座博文由 MySpace 前总裁、创始人、第一个朋友[汤姆·安德森](https://web.archive.org/web/20230205004345/http://www.crunchbase.com/person/tom-anderson)撰写。你现在可以在[脸书](https://web.archive.org/web/20230205004345/http://www.facebook.com/myspacetom)、[推特](https://web.archive.org/web/20230205004345/http://twitter.com/#!/myspacetom)和[谷歌+](https://web.archive.org/web/20230205004345/https://plus.google.com/112063946124358686266)* 上找到汤姆

这只是一个猜测，但我敢打赌，维克·冈多特拉和布拉德利·霍洛维茨可能觉得他们的头要爆炸了。Google+团队中任何真正关心 G+的人可能睡眠都很少，并且用他们一根筋的 G+思维惹恼了他们的朋友和家人。有如此惊人数量的[反馈](https://web.archive.org/web/20230205004345/http://www.techmeme.com/110725/p67#a110725p67)，Google+团队不禁应接不暇——我们所看到的只是网站和媒体。想象一下每个 G+页面底部的“发送反馈”收件箱里会有什么？

在 MySpace，我试着自己消化“收件箱”和“社区”,这在几年里效果相当好。那时候要容易一点，但是今天的 G+用户是完全不同的一类人。G+上有大量的早期采用者和技术专家，他们之前都经历过社交网络铃声。G+用户提供 powerpoint 幻灯片、图文并茂的屏幕模型，以及关于需要做什么的长篇大论。在 G+从全球社区获得的“免费建议”中，存在着真正的、高质量的思考。G+团队如何才能穿过噪音，决定什么是重要的？(尤其是当有一些非常高质量的噪音指向谷歌的时候。)

如果我是谷歌，我现在会做以下几件事。

**1)开始认真地讨好那些使用和/或宣扬 G+的记者、时尚人士和名人**。这并不意味着谷歌应该忽略“常规的”用户反馈，甚至也不意味着谷歌应该按照三巨头说的去做。这只是意味着他们应该和那些人建立真实的私人关系。在 MySpace 发展的过程中，记者们不断地误解了 MySpace 的事实。他们写了一个又一个关于脸书比 MySpace 大的故事，而事实上脸书还不到 MySpace 的十分之一。

为什么？因为记者的常春藤盟校毕业的孩子正在使用脸书，记者有最后期限和其他事情要考虑。如果你了解人们，当他们写那些故事时，他们会想到一个真正的人，他们会更在乎一点。我不想说人们是“绵羊”，但如果普通互联网用户相信 G+正在发生并持续下去，那么他们会花时间去尝试它。对大多数人来说，流行的观点是最大的“过滤器”——如果他们已经被告知某样东西不酷，他们就没有必要去尝试。如果你愿意的话，流行观点是终极的“社会搜索”。这不仅仅适用于用户数(G+点击量 2000 万！)，当然。流行的观点会塑造人们 G+感知的方方面面。

**2)彻底考虑隐私问题**并处理好 G+在这方面的任何细节。我见过很多人在不知情的情况下和我分享他们的电话号码。他们可能也不想分享其他东西，尽管这些东西不太明显是私人的。我会确保人们了解他们的帖子如何被分享，以及他们的其他谷歌账户(个人资料、Gmail、docs)和内容(Youtube、Picasa)是如何连接到 G+的。我不认为隐私对大多数人来说是一个真正的问题，但大多数人认为这是一个真正的问题。因此，它在从竞争对手处叛逃的心理理由中扮演着重要角色。“安全”歇斯底里摧毁了新闻界的 MySpace。它导致 MySpace 被学校、苹果商店和好心的家长禁止使用，这些家长被他们阅读的内容吓到了。隐私倡导者过去曾试图摧毁脸书和谷歌。你需要世界上最好的公关人员加入你的团队，谷歌，但更重要的是，你需要确保这个软件不会让隐私卫士有理由生气。

**3)将谷歌的顶级分析师(目前可能专注于盈利)转移到 Google+项目**上，组建一个臭鼬工作团队。像挖掘黄金一样挖掘关于 G+使用的数据，因为它确实是谷歌长期收入和利润增长的未来。(实际上，我不认为 Google+上会有广告，这是另一篇文章的主题。)脸书非常善于理解他们的入职流程，知道哪些关键活动会导致后来的使用(添加 x 个好友，放一张照片并获得回应，等等)。).谷歌需要密切跟踪没有采用这项服务的用户，以及那些采用了这项服务的用户，并试图了解什么类型的用户是走在曲线前面的用户——确定通过他们的开创性活动展示未来“常见”用例的用户。G+需要了解所有这三类用户，并针对每一类用户制定计划。

**4)雇佣世界上最好的产品执行者&远见者，**这显然不是谷歌过去的强项。(事实上，似乎一些[的好员工已经离开了](https://web.archive.org/web/20230205004345/https://techcrunch.com/2010/12/20/paul-adams-googler-whose-presentation-foretold-facebook-groups-heads-to-facebook/)，因为他们觉得自己在谷歌不受重视)。我指的不是普通的产品经理和 UI 开发人员或“社交媒体专家”，而是那些拥有可证明的经验，带领用户走向互联网天堂的少数人。谷歌有工程天赋和能力来扩展 G+服务(我认为，比人们现在所理解的更有价值)。但是谷歌有产品人吗？谷歌的技术基础设施将允许他们做其他社交网站不能做的事情——事实上，他们已经在做了。他们需要能够理解这一点的产品远见者。虽然我喜欢 G+，但 G+的某些部分现在真的很乱，而且在这个阶段非常重要的两个部分需要做很多工作:入职照片。

**5)必须有一个戒指来统治他们所有人。**一个领导做决策。也许那个人已经在谷歌就位了；我不知道公司内部的等级制度。但是领导本人，每个员工都必须明白这是谁。制作一个网站类似于制作一部电影——数百人一起工作，一个人做最后的决定，他们每天每分钟都在做决定。我用 LOTR 做类比，因为可能有 12 个非常重要的产品人员(第四点)。但是需要有人来做决定。进一步类比，如果魔戒坏了，霍比特人需要把魔戒扔进末日火山，然后找一个新的领袖。好吧，也许这个类比不成立，但是你明白了。所有的观点和分析都会使任何不能胜任这项任务的人陷入瘫痪。这个人必须把所有的事情整合在一起，根据自己的直觉和对整个公司使命的理解做出决定。不，领导者不会总是做对，但实现清晰和节省时间是至关重要的。互联网以闪电般的速度发展。如果你搞砸了，一个果断的领导可以迭代修复。这值得一读(阅读兰德尔·斯特罗斯关于史蒂夫·乔布斯/苹果 vs 谷歌的文章[就在这里](https://web.archive.org/web/20230205004345/http://www.nytimes.com/2011/07/24/technology/what-apple-has-that-google-doesnt-an-auteur.html?src=tptw))。

当你陷入困境时，你会学到很多东西。我搞砸了很多，所以这些只是我学到的一些东西。顺便说一句，这些教训不仅仅是给谷歌的。它们也适用于任何初创公司，祝大家好运。我希望看到你们都在这个世界上留下自己的印记。
