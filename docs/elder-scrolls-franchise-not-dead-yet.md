# 《上古卷轴》特许经营权尚未消亡

# 《上古卷轴》系列还没死

上周在 QuakeCon 大会上，Bethesda 软件公司的员工 Todd Howard [发表了一个被错误引用的声明](https://web.archive.org/web/20221001180407/http://news.bigdownload.com/2009/08/14/quakecon-2009-todd-howard-keynote-highlights/)，并引起了轩然大波。当被问及是否会有新的《上古卷轴》游戏时，他回答说人们不应该在“不久的将来”寻找。这让人们相信根本不会有新游戏。

然而，今天贝塞斯达的公关负责人[通过博客](https://web.archive.org/web/20221001180407/http://bethblog.com/index.php/2009/08/17/clarifying-about-next-elder-scrolls-game-mmo-etc/)回应了这些传言。他的意思是，仅仅因为在不久的将来不会有什么事情发生，并不意味着专营权已经完成。他指出，TES 一直是一个非常受欢迎的标题，他们不会完全停止生产一个受欢迎的标题。

这也符合其他流传的谣言，关于 MMO 上古卷轴的可能性。这些谣言始于 Bethesda 开始雇佣前魔兽世界开发者的时候。
