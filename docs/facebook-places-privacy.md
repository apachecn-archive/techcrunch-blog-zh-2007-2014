# 选择去脸书的困惑阶段

今天早些时候，我们在 TechCrunch 总部进行了一次消防演习。当我在脸书的 Places 上登录 TCHQ 时，我也标记了我的几个同事，完全期待他们的登录会被推迟，直到他们第一次真正选择加入该功能(不像脸书过去一些有争议的功能发布，你必须选择加入，然后才能在 Places 上被标记)。除此之外，正如我们很快发现的，你真的*不需要*在你被标记之前选择加入。

我一在 Places 上登记，脸书就向我的朋友们发布新闻订阅条目，表明我已经和我的同事们登记了——甚至是那些还没有选择 Places 的*的同事。我的同事收到了一封要求他确认标签的电子邮件，但他从未点击过。然而，他的许多朋友都被告知，他刚刚和我一起进入了 Techcrunch 总部。我确信这是一个隐私漏洞，于是我找来了脸书公关，他耐心地向我解释说，事情实际上应该是这样的。啊？*

事实证明，选择加入这项服务有三个不同的阶段。让我们把它们拼出来。

*   选项一:你明确选择允许人们给你贴标签。你的任何一个朋友都可以做这个。非常简单。
*   选项二:你勇敢地面对脸书隐私控制面板的混乱局面，完全关闭了一些地方。你不能被标记——如果一个朋友试图在 Places 帖子中标记你，你的名字就不会出现在帖子中。
*   **选项三**:如果你从未决定“允许”你的朋友给你加标签，但你也没有完全屏蔽“地点”功能，那你就处于一种悬而未决的状态。这是美国绝大多数脸书用户目前的处境。一旦他们第一次被标记，他们就会收到一封电子邮件，并在脸书上得到提示，询问他们是否愿意允许他们的朋友在未来的地方标记他们。接受这一点将允许您的任何朋友标记您，除非您进入您的隐私设置并取消它(见选项一)。但是即使你点击了“不是现在”按钮，你仍然会在相关位置更新中被标记。事实上，即使你没有*看到*要求你批准位置标签的提示，你仍然会被标记。脸书将此视为你在基本状态更新中被标记，因此它将显示在你的墙和你朋友的新闻订阅上~~——你只是与你朋友标记你进入的任何地方没有关联(即，如果你的朋友访问该场所的场所页面，他们不会看到你以前在那里登记过)。这里的逻辑是，无论如何，你的朋友可以在正常状态更新中手动标记你。
    **更新:**脸书已经澄清它不会出现在你的墙上。
    **更新 2** :脸书进一步澄清。状态更新只会出现在你与在该地点标记你的人共享的朋友的新闻源中，而不会出现在你所有的朋友中。~~

**脸书刚刚发布了一段解释这一点的视频，你可以[在这里](https://web.archive.org/web/20230219191727/http://www.facebook.com/video/video.php?v=697692691093)** 观看。

从脸书的角度来看，这是有道理的，但这对用户来说非常混乱——通常当你选择不在某个地方被标记时，你可能会认为*根本不会在那里被标记。不管怎样，脸书知道这是令人困惑的——他们计划在不久的将来发布一个视频来解释它。尽管如此，我认为网站应该隐藏任何位置标签，直到用户明确表示希望与它们相关联。
