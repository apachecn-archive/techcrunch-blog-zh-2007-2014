# 参与海盗湾 2.0 的前格罗斯特高管暗示新模式

![tpb20](img/1e18434a24973f62534476fe06b7adac.png "tpb20")

格罗斯特公司的前总裁[韦恩·罗索](https://web.archive.org/web/20221006120804/http://www.waynerosso.com/about/)，刚刚给[一个关于海盗湾 2.0 的小暗示](https://web.archive.org/web/20221006120804/http://www.themusicvoid.com/2009/07/exclusive-the-pirate-bay-2-0/)。罗索先生在 Music Void 上写道，该公司的首席执行官汉斯·潘德亚已经联系了他，该公司准备以近 800 万美元的价格收购海盗湾。有什么大不了的？

我提请你注意这几句话:

> 起初，我对汉斯想要完成的事情还是有点模糊。但是经过几次长谈，我现在明白了。这是相当巧妙的。我称这种新模式为“资源支持”。简而言之，用户贡献给海盗湾的计算机资源越多，他的内容消费得到的补贴就越多。由于商业机密，我不会进一步深入研究，但它实际上是可行的。如果有，那将是巨大的。

所以，你贡献了 CPU 周期和/或带宽——我有多余的带宽——而你得到了内容补贴？什么，把 New_Album 的价格降低 2 美元，因为你为这项事业贡献了每 10 亿字节？我是这么解释的，但谁知道是怎么回事。
