# 社交网络悖论 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/28/the-social-network-paradox/>

**编者按:** *尼娜·科斯拉(Nina Khosla)是一名设计师，也是社交博客初创公司[neigie](https://web.archive.org/web/20230203125250/http://teethie.com/)的创始人，该公司专注于建立基于兴趣的社区。可以关注她 [@ninakix](https://web.archive.org/web/20230203125250/http://twitter.com/#!/ninakix) 。*

这些年来，我们与网上朋友互动的方式发生了根本性的变化。过去，我们有几个朋友(在线或离线朋友)在一项服务上，允许我们通过互联网与朋友联系，并查看他们的活动。互联网曾经是一个充满陌生人的有点可怕的世界，但我们突然有了友好的主播来探索这个世界。当然，我们的大多数朋友都不在线，或者至少没有使用相同的服务，但这种熟悉感令人欣慰，能够看到我们的一些朋友在做什么让我们能够找到新的内容和新的朋友。

我们爱上了那些让我们觉得有人和我们相似、我们在交谈、我们有共同经历的网站。但是后来，其中一些网络——特别是脸书和推特——开始爆炸性增长。脸书促进了一种文化规范，即使用其服务将我们认识的每个人都“加为好友”。突然间，无论我们去哪里，我们都有很多朋友。有了与几个朋友分享在线空间的经验，逻辑将决定有更多的朋友在线将使这种经验更加丰富。但事实并非如此。

相反，有一种新趋势正在发生:我们并没有真正关注我们在网上联系的朋友。以 Twitter 为例。Twitter 曾经是许多早期用户谈论技术的好地方。就在不久前，Twitter 上还没有足够多的人可以阅读你的信息流中的每一条推文。

但是随着网络开始变得更加密集，人们在 Twitter 上找到更多他们认识和喜欢的人，他们开始关注数百人，阅读所有这些推文变得不可能。这是生活中的一个事实，整个公司都是建立在这样一个前提之上的:你在脸书和推特上有太多的朋友，以至于无法真正关注他们在说什么。

例如，Flipboard，以及其他一些软件，突出了它与你分享你朋友最好的 Twitter 和脸书帖子的能力。这些公司，甚至脸书的 news feed intelligence，都在帮助我们处理由于我们的连通性而与朋友之间的脱节——他们正在整理这个扩展的网络为我们创造的海量信息。

这就是没有人愿意承认的社交网络的悖论所在:*随着网络规模的增加，我们的社交能力下降。*

和其他事物一样，网络和流经网络的信息遵循供求法则。随着来自这些网络的比特、照片和链接数量的增长，每一个都在无形中开始贬值。

也许这解释了人们对新产品的兴奋。当一小群人在使用一个工具时，比如 Foursquare，我们可以跟踪我们朋友的位置和行踪。在较小的范围内，知道这些信息，并且能够期望其他人也看到它，让我们所有人都知道一个小秘密，这让早期使用 Twitter 感觉有点神奇。但是随着朋友数量的增加——特别是超过了 150 人的神奇邓巴数(T1)——魔咒开始消失。在这种规模下，我们很难跟踪所有的情况。当我们的连接数超过 150 时，一切都变成了简单的评论，因为真实的对话加重了我们本已有限的网络交互能力。

曾经是一个网络探索者和叛逆者的小社区已经变成了一个有点无意义的 Foursquare 签到和过度使用标签的大聚会。那种神秘的东西，社会联系，不会在这些网络上流动；信息在这些网络上流动。这个网络感到有意义的唯一原因是，在小范围内，这个网络像一个社区一样运作。但这在很大程度上是分裂的。

这就把我们引向了社区:社区，那种成员界限分明的社区，总是存在于更大的网络环境中，并且总是被分割成一点一点，让他们感觉熟悉。社区，以及给予他们形成的空间，是我们能够与物理世界的网络合作的唯一方式。我们的足球队，我们的学校，我们的工作场所，我们的街道，我们的城镇，都有自己的社区。我怀疑只有这些东西才能让数字世界变得同样易于管理。

社区给了我们观众和视角。我们知道我们在和谁说话。这看起来没什么大不了的，但却是维系我们沟通的粘合剂。这就是对着虚空大喊和与站在你面前的人对话的区别。

直播一场体育比赛和参加一场 SB 国赛线程有什么区别？推文不是一种体验，它是向模糊不清的受众传播个人的体验。当我想到我在推特上发布的事情时，它们就像“我刚刚读了一篇很酷的文章，看看吧，”或者“正要上飞机”，或者“GOALLLL！”如果我的球队(圣何塞鲨鱼队)刚刚得分。

关于所有这些的事情是，它们不是一种共享的体验——它们是我的体验，我正在与你分享，但你可能无法与我一起体验——我对我刚刚发布的文章的想法或迷恋，登上那架飞机的感觉，或观看鲨鱼队比赛的兴奋感。也许你可以比较你自己对这些事情的经验笔记；对我来说，这似乎是大多数 Twitter 上的对话，但这些经历并没有被分享。

这不同于社区中的讨论，例如发生在 SB Nation game day 主题中的讨论。对话的中心不是任何一个人的经历，而是社区的集体状况。对话*就是*体验。每个评论都是为了唤起和表达社区经历的情感，特别是他们共有的情感。

这种唤起和表达共同情感的习惯是内部笑话及其互联网化身——迷因的驱动力。当然，社区中存在分歧和差异，但神奇之处在于相似之处:知道那里的每个人都是鲨鱼队的球迷，并且刚刚在电视上发誓要实现这个目标，这是令人感动的，也是有价值的。这就扩大了社区中人们的归属感和成员感，并成为整个讨论的基础(甚至，特别是差异)。

SB Nation 是实时的，但不一定是实时的:社区已经在传统、缓慢的 PHP 公告栏上涌现多年。*迷失*的粉丝挤满了留言板和博客，团结在他们对*迷失*的共同热爱，以及这部剧对抗他们的方式——那个舱口里是什么？！

如果我们所有网络的模式变得更大，就像脸书推动周围的其他网络变得更大一样，消费者将会触及这些网络意义的极限。如果我们在创造社交产品，我们需要创造能让人们社交的产品，真正的社交。

我们需要开发不仅仅允许用户写作和发布的产品，我们需要创造鼓励讨论、体验和持久、有意义的关系的产品。这些才是真正为用户创造利益的东西，才是激励用户的产品。因此，社交网络的未来不再是网络，而是社区。
