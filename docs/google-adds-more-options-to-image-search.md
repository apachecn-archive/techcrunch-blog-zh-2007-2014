# 谷歌为图片搜索增加了更多选项

![](img/45cbcb170fe5b4508e3941d995342a1d.png)

图片搜索是谷歌、雅虎和现在的必应之间激烈竞争的领域。今天， [Google Images](https://web.archive.org/web/20221006010806/http://images.google.com/) 增加了一些[搜索选项](https://web.archive.org/web/20221006010806/http://googleblog.blogspot.com/2009/07/search-options-now-on-google-images.html)，使得根据颜色、类型(脸、照片、剪贴画、线条画)和文件大小过滤搜索变得更加容易。

这些过滤器中的大多数以前在高级搜索中可用，但现在它们在左侧栏中可用。(类似的选项栏[于 2009 年 5 月引入](https://web.archive.org/web/20221006010806/http://www.beta.techcrunch.com/2009/05/12/google-launches-search-options-declares-real-time-search-biggest-challenge/)到主搜索页面)。您可以选择多个选项来自动缩小搜索范围。谷歌可以增加的另一个选项是通过知识共享许可按图片排序。它[已经在它的高级搜索选项](https://web.archive.org/web/20221006010806/http://www.beta.techcrunch.com/2009/07/09/google-finally-adds-creative-commons-to-image-search/)中这样做了，但是它被隐藏在那里。

雅虎和必应有类似的过滤选项，尽管谷歌的彩色过滤器是最先进的。它允许你用 12 种不同的颜色进行分类，而必应和雅虎只允许你用彩色或黑白图像进行分类。然而，必应在相关搜索方面做得更好，谷歌正在以一种略有不同的方式进行试验，实际上是试图在图像搜索中找到谷歌没有的自己的一套新颖功能。你可以在谷歌图片[、雅虎图片](https://web.archive.org/web/20221006010806/http://images.google.com/images?imgcolor=&imgtbs=&gbv=2&hl=en&sa=1&q=fruit&btnG=Search+images&aq=f&oq=&aqi=&start=0)和必应图片上搜索“水果”来比较这三种图片。
