# 脸书将推出即时通讯服务

脸书一直在测试一项新的即时通讯服务，并将很快向公众推出，也许就在下周。

我们的理解是，这项服务将被纳入用户的脸书网页，并允许他们与他们的脸书朋友进行网上聊天。我们还听说，像 Gtalk 一样，它将建立在 Jabber 开源平台上，允许用户将该服务添加到许多流行的即时消息客户端，如 Trillian (Windows)和 Adium (Mac)。我还希望像 Meebo 和 eBuddy 这样的网络聊天服务能够增加对这项服务的支持。

这给第三方在脸书建立的一系列即时通讯服务带来了麻烦。社交。例如，IM 是一家([资助的](https://web.archive.org/web/20230404075901/http://www.crunchbase.com/company/mogad))创业公司，我们已经[写过几次](https://web.archive.org/web/20230404075901/https://techcrunch.com/2008/01/23/facebook-im-client-socialim-now-available-for-mac/)。(英国初创公司 Techlightenment 的 FriendVox 也是如此)。那些应用程序现在基本上都死了。

这个时机的确很有意思。昨天，AOL 广泛讨论了将他们的 AIM 平台与他们新收购的 Bebo 社交网络结合的问题。

有一个屏幕截图在某个地方。我们现在正试图得到它。

**更新:**我现在听说这不会是 Jabber 兼容的，至少一开始不会。这意味着，除非他们创建一个 API 和/或第三方找到侵入服务的方法，就像他们过去对雅虎、MSN 和 AIM 所做的那样，否则访问将只能是脸书的。此外，澄清一下，我没有听说脸书打算围绕这一点推出任何桌面软件。
