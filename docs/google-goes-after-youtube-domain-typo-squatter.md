# 谷歌追踪 YouTube 域名输入错误抢注者

**专业提示:**如果你的新商业计划需要购买一大堆域名，而这些域名显然与一个主要品牌的域名相差一个字母，那么你*很可能*做错了。

一位来自伊利诺伊州的绅士就是这种情况，他抢到了不是一个，不是两个，而是六个与 YouTube.com 看起来几乎一样的*域名，填补了他们可疑的调查。正如你所料，谷歌现在已经开始抢占域名。*

超级域名侦探在 [Fusible](https://web.archive.org/web/20221207081312/http://fusible.com/2011/11/google-going-after-youtube-typo-domains-that-lead-to-survey-scams/) 首先注意到，谷歌已经正式向[国家仲裁论坛](https://web.archive.org/web/20221207081312/http://domains.adrforum.com/decision.aspx)提出申诉(你可以找到案件编号 1416796)。如果成功(几乎肯定会成功)，谷歌将获得对域名的完全控制权。

**域名列表:**

*   Youtbe.com
*   Youtub.com
*   Youtue.com
*   Youube.com
*   Yutube.com

鉴于三个主要标准是“与商标相同或令人混淆地相似”(检查)、“所有人对该域名没有权利或合法利益”(检查)，以及“域名已被注册并被恶意使用”(检查！我的意思是，只要*看看下面的截图*，这个可能不会花太长时间来清理。
