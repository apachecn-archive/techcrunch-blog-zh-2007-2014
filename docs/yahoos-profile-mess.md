# 雅虎档案一团糟 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/10/18/yahoos-profile-mess/>

![](img/6a20b1fed14658395a1415ee51f46710.png)

雅虎上周推出了他们的新个人资料产品，这是他们将整个雅虎世界社交化的大计划的一部分。每个雅虎档案都被重置，用户被要求重新创建。

来自[用户](https://web.archive.org/web/20221007104008/http://www.ymessengerblog.com/blog/2008/10/17/changes-to-your-yahoo-profile/)的[评论](https://web.archive.org/web/20221007104008/http://www.yprofileblog.com/blog/index.php/2008/10/17/managing-your-alias-and-profile/)毫不含糊——他们被激怒了。

常见的抱怨包括，新的个人资料必须包括你的全名，你的联系人(朋友)可以看到，以及删除一些我从未听说过的功能(如酷链接)，显然有一部分用户非常非常喜欢这些功能。但是到目前为止，最大的抱怨是，旧档案中的所有信息都被清除了，用户看到的是一张白纸，就像上面这张一样(这是我的)。一些用户还注意到，空白个人资料上没有任何信息让用户知道他们的数据刚刚发生了什么。你必须去博客上了解发生了什么。

与此同时，雅虎迅速做出回应。在其中一篇博文中，雅虎社区经理梅丽莎·丹尼尔斯(Melissa Daniels)说:“我们知道这对你们中的一些人来说是一个艰难的转变，我们致力于帮助你们使用、理解和(希望)享受你们的新档案。”

这似乎更加激怒了用户。一个例子评论:*“我同意制作一个二流的 facebook 副本，同时打破人们来雅虎的其他一切，这是一个很好的倒闭方式。”*

尽管我最近对雅虎颇有微词，但我主要还是支持他们进行个人资料的转换。

与用户的沟通无疑执行得很差，用户应该在他们的个人资料页面上有一条消息告诉他们发生了什么。但该团队显然正在努力解决这个问题。社区副总裁吉姆·斯通汉姆(Jim Stoneham)六周前刚刚加入雅虎领导这个项目(欢迎来到雅虎，吉姆！)，在他的[个人资料](https://web.archive.org/web/20221007104008/http://profiles.yahoo.com/u/ZQUCUVVWWHX7RYBKHS54CMNCGY) : *上只有一条信息“正在收听”*雅虎至少保留了一份旧个人资料的副本，以备用户需要:

> 我们也知道你们中的很多人努力工作，想要你们的数据。如果您觉得丢失了数据，我们已经保存了您的旧档案(和别名)的副本，我们的客户服务团队可以检索这些信息。但是，您不能恢复到旧的配置文件格式，但是您可以获得您认为丢失的任何数据。要做到这一点，请到这里 http://help.yahoo.com/l/us/yahoo/profiles/general.html 联系客户服务。

无论如何，大量的个人资料仍然存在。我浏览了新的个人资料设置，它保留了我的姓名、年龄和兴趣，并在我完成后显示出来。
