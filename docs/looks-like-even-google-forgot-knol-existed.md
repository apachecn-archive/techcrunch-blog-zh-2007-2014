# 看起来就连谷歌都忘记了知识单元的存在

![](img/77fb4cafefeeaa88ad35cc9ff4184e62.png)知识单元，谷歌分享特定话题文章的平台，似乎已经关闭。目前还不清楚这是暂时还是永久的情况。我们已经联系了谷歌寻求评论。

我们一直在[猜测](https://web.archive.org/web/20221007170050/https://beta.techcrunch.com/2009/08/11/poor-google-knol-has-gone-from-a-wikipedia-killer-to-a-craigslist-wannabe/)谷歌何时会关闭知识单元。这个知识共享网站基本上已经沦为“Craigslist 的追随者”知识单元背后的最初想法是，人们可以合作写关于他们喜欢的任何主题的权威文章，并从他们创作的每个页面中获得一部分 AdSense 收入。不幸的是，似乎没有人阅读任何关于知识单元的东西，产品也从未真正成功。

尽管这次停电可能是暂时的，但也许是时候让知识单元脱离苦海了。谷歌此前已经关闭了其他表现不佳的项目，如 [Lively](https://web.archive.org/web/20221007170050/http://www.beta.techcrunch.com/2008/11/19/google-kills-lively/) 和[谷歌笔记本](https://web.archive.org/web/20221007170050/http://www.beta.techcrunch.com/2009/01/14/google-axes-dodgeball-jaiku-video-and-more/)。

**PDT 下午 2:49 更新:**谷歌通知我们这次宕机实际上是暂时的，“我们正在努力尽快为用户恢复。”
