# 以下是政府如何修复硅谷:别管它 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/06/07/heres-how-the-government-can-fix-silicon-valley-leave-it-alone/>

我采访了大多数总统候选人，询问他们在 2008 年大选中对技术问题的立场。采访结束后，我支持了 T2 两党的一名候选人——巴拉克·奥巴马和约翰·麦凯恩。他们当时都没有在初选中领先，但仅从技术政策的角度来看，他们似乎是最佳候选人。

我一直认为，政府往往会搞砸它所触及的任何事情，但奥巴马似乎特别不同。他理解让其他候选人困惑的技术问题。一部分可能是他的年龄。但不管是什么原因，我真的希望他能带领我们进入一个技术领先和增长的新世纪。

我真的不再相信了。

主要是因为失信。根据我对他的采访:

> 他坚定地支持网络中立，并承诺在他执政的第一年优先恢复网络中立。他提出了增加科技教育和儿童接触的智能计划。他认为，联邦通信委员会在开放 700MHz 频谱拍卖的拟议规则方面走得不够远。他希望看到每年发放的 H1 B 签证数量增加。他强烈支持可再生能源的研究，并且他有一个现实的，基于市场的方法来限制碳排放。

在他的任期内，这些事情都没有发生，也不太可能发生。

但这不仅仅是违背承诺。我们的政府对在硅谷瞎折腾太感兴趣了，在很少或根本不了解后果的基础上制定和执行规则。一个很好的例子——最近由参议员克里斯·多德提出的[金融改革法案](https://web.archive.org/web/20230203071358/https://techcrunch.com/2010/04/29/angel-investing-a-fly-on-the-elephants-ass-video/)增加了一些随意的条款，这些条款可能会摧毁硅谷脆弱的风险投资生态系统。

今年早些时候，我应邀与奥巴马任命的美国知识产权执法协调员维多利亚·埃斯皮诺(Victoria Espinel)举行了一次小型闭门会议。出席会议的有硅谷许多大小公司的首席执行官和其他高级管理人员。这次会议应该是关于她的办公室如何帮助硅谷繁荣的。但很快就变得非常明显，当涉及到版权问题时，Espinel 只有一个议程——帮助音乐公司和电视/电影工作室按照他们自己的方式处理互联网。

这次会议是严格保密的，这就是为什么我离开房间后没有立即写下来。正如我所承诺的，她在那次会议上说的话不会被记录在案。但我要说的是——一个小时后，我沮丧地走了出来。我在那个房间里说了很多，其中之一就是:

> 政府可以继续在我们的花坛撒尿，但很快所有的花都将死去。

Espinel 的问题是，她必须遵循游说资金，而这些资金来自老牌参与者——电视和电影工作室以及唱片公司。正如她在会上对我说的那样(我将引用她的原话)，“我的头衔*终究是*知识产权*执法*”

所以，好吧，好吧。但为什么我们需要一个总统任命的人，他的主要工作似乎是试图关闭那些吓坏了唱片公司和工作室的初创公司？那些家伙有成堆的私人资金和各种民事诉讼选项来保护他们的权利，他们当然不需要政府再增加一层欺凌。为什么要费心来到硅谷，假装你想找到一种方法，让初创公司在一个为没有互联网的世界设计的过度限制性版权保护的国家茁壮成长？

因此，今晚我饶有兴趣地读了一封好朋友发来的邮件:

> 白宫科技政策办公室就总统和联邦政府如何增加美国的高科技创业征求了一些意见，我想你们可能在这方面有一些好的想法。
> 
> 他们在寻找目标和战术。这些策略可能涉及立法、联邦支出、公私合作、政治意愿等。鉴于您在社区中的地位，我相信您对目标有一些想法。我期待在接下来的几天里为他们收集一些想法。

我的回应基本上就是这篇文章的标题——我们不需要他们的帮助，因为他们倾向于把他们接触到的任何东西都变成有毒废物。请不要管硅谷。*请*。

最近，我看到太多的朋友在华盛顿与奥巴马和希拉里·克林顿合影。天知道他们在忙着扮演重要人物时，实际上在跟这些政客说些什么。但是我怀疑这是他们应该说的——*让我们单独呆着，不要在我们的花坛上撒尿*。

如果政府想要帮助这个国家的创新，他们应该忙于基础设施建设。将光纤铺设到美国的每个家庭和企业，实际上开始建设一些高速列车网络，使旅行更容易。在这个国家的每一个孩子能够按下按钮的时候，就把电脑送到他们的手中。见鬼，把女人送上月球。我不知道最后一个会不会有太大作用，但至少他们会忙着不去搞砸硅谷。

我会说让世界上任何想住在这里的受过高等教育的人进来，但我知道这是不可能的。我们将继续回避下一代优秀的外国企业家，因为有些荒谬的担心他们会抢走我们的工作。无论如何，几年后这些企业家将不再想住在这里。

在过去的几十年里，硅谷推动了我们经济的大部分增长，并创造了令人惊叹的(高利润的)公司，使世界成为一个更美好、更有趣的居住地。所有这些都是在政府无视我们的情况下发生的。

我们不想要施舍。我们不想要“公私合营”，我们当然也不想要立法。让我们做好自己的事情，或许对那些创造了成千上万个就业岗位、缴纳了巨额企业所得税的公司说声谢谢。因为你只会把美好的事情搞砸。真的。
