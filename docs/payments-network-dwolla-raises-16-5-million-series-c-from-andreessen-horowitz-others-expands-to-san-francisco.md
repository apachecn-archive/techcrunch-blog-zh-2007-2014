# 支付网络 Dwolla 从 Andreessen Horowitz & Others 获得 1650 万美元的 C 轮融资，扩展到旧金山 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/30/payments-network-dwolla-raises-16-5-million-series-c-from-andreessen-horowitz-others-expands-to-san-francisco/>

为数字支付构建新的基础设施听起来可能并不性感，但这是一个创新时机已经成熟的领域。现有的传统支付网络因技术过时而停滞不前，减缓了发展速度。总部位于得梅因的 Dwolla 认为，支付创新的方法是彻底摧毁过时的基础设施，从头开始构建新的网络。今天，这项工作已经为该公司赢得了 1650 万美元的新资金，由安德森·霍洛维茨(Andreessen Horowitz)领导的 C 轮融资。该公司之前的投资者 Village Ventures、Thrive Capital 和 Union Square Ventures 也参与了这轮投资。

Dwolla 有时会被混淆为 PayPal 的替代品——尽管它可能会在一些项目上与 PayPal 更直接地竞争，例如 MassPay 降低了 PayPal 在服务业务中代替开支票的费用—这只是该公司建立的新支付基础设施的结果，而不是其本身的全部愿景。

例如，去年，Dwolla 推出了许多新产品，这些产品有助于将业务提升到一个新的水平，不仅包括其自己的大规模支付服务，还包括 [FiSync，这是 ACH 支付的实时替代产品](https://web.archive.org/web/20221006083618/https://beta.techcrunch.com/2012/06/15/dwolla-fisyncs-first-banking-customer-goes-live-eliminates-ach-delays-with-real-time-bank-transfers/)，加上[与爱荷华州的合作伙伴](https://web.archive.org/web/20221006083618/http://blog.dwolla.com/governmentpayments/)，以处理最初的 1.3 亿美元的税收，以及[与移动银行和支付服务提供商 mFoundry](https://web.archive.org/web/20221006083618/https://beta.techcrunch.com/2012/10/10/dwolla-partners-with-mfoundry-bringing-real-time-p2p-mobile-payment-capabilities-to-800-u-s-banks/) 的合作，这将该服务放在该公司 800 多家有兴趣提供服务的银行和信用合作社合作伙伴面前

今天，Dwolla 的年度交易处理运行率已经超过 10 亿美元；该公司以 15%的月环比增长率增长，从 2012 年初的 80，000 人增加到 25 万人，并带来了 100 多个大客户，包括企业和政府。

但走到这一步并不容易。

首席执行官本·米尔恩解释说，公司发现它尝试的一些方法比其他方法更有效。例如，试图用一种面向消费者的产品来扰乱支付是很困难的。

“我们发现，就让用户用另一种支付形式付费而言，零售真的很难转化用户，”米尔恩承认。沿着这些思路，该公司在[线上](https://web.archive.org/web/20221006083618/https://beta.techcrunch.com/2012/11/30/dwolla-launches-guest-checkout-option-for-merchants-allows-shoppers-to-pay-with-dwolla-without-signing-up/)和[线下](https://web.archive.org/web/20221006083618/https://beta.techcrunch.com/2012/07/03/with-places-payments-platform-dwolla-finally-lists-where-it-works-lets-you-request-merchant-support/)发起了各种努力，鼓励消费者使用 Dwolla 而不是现金、支票或信用卡等传统方式支付。虽然 Dwolla 的努力今天仍在继续，但它们并不是导致新投资的增长领域。

“我们看到的大量交易都是一对多关系，基本上这些交易更多的是支票替代，而不是信用卡或借记卡支付，”Milne 说。在这方面，该公司今天没有新的关系或合作伙伴关系要宣布，但暗示有几个正在进行中，因为额外的资金已被指定专门用于扩大初创公司的业务发展和关系团队，以服务于更大的客户。

“这些交易需要很长时间，并且需要很多关注……它们无法以小型创业公司的速度运作。米尔恩解释道:“他们是大型老牌公司。“这一轮是关于不要搞砸我们拥有的机会。我们是否率先将大量新技术推向市场并不重要，重要的是我们有机会成为第一个推向市场和第一个规模化的企业。”

这家初创公司现在正在与薪资公司和政府等合作——后者最初是在自己的州，爱荷华州州长特里·布兰斯塔德在 2013 年初宣布，政府官员将探索使用该服务来征收财产税、发放退款、支付承包商以及更新车辆注册的方法。但是，虽然 Dwolla 的系统对这些大客户很有效，但它也适用于较小的客户，例如，包括需要向供应商支付费用的制造商，支付顾问费用的广告公司，或支付数千人费用的微咨询平台。

Andreessen Horowitz 的合伙人、Dwolla 董事会新成员斯科特·韦斯(Scott Weiss)在一份关于投资的声明中表示:“A16z 押注于那些改变其市场基础结构的公司，像脸书、Twitter 和 GitHub，我们认为 Dwolla 将在银行业做到这一点。”。“事实上，Dwolla 的网络可以同时满足复杂企业或政府的需求，同时允许父母用手机支付保姆的费用，这反映了该解决方案在市场上是多么简单和显著不同。”

除了发展工程团队，以及支持其大客户所需的人员，Dwolla 还将扩展到其第五个办公地点，旧金山，在 6 月开业之前，它已经在那里进行了面试和招聘。(该公司在得梅因、纽约、奥马哈和堪萨斯城都有员工。)在湾区，该团队将由 Dwolla 的首席运营官 Charise Flynn 领导，并将主要专注于产品和业务开发及营销。

Milne 表示，在 Dwolla 能够真正与传统支付提供商竞争之前，仍有许多工作要做。例如，虽然它的网络现在支持支付，但像 Visa 这样的传统提供商也支持其他两种类型的交易:授权，保证资金到位；和捕获，它们在授权后取走资金。随着时间的推移，Dwolla 将为这些类型的交易提供支持，这些交易在市场上仍有需求。该公司还将更新其移动应用程序，并做更多的工作来教育市场。

但这个“市场”可能并不意味着消费者。

“事实是，我们的基本业务是允许任何有互联网连接的人获得他们的钱，并与他们想收到钱的任何人交换，”米尔恩说。他补充道:“很多应用将来自第三方平台和产品。”。“我没有看到去 Dwolla.com 的人越来越多——我看到他们去的越来越少，而我们的软件只是方便支付。”
