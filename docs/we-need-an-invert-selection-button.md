# “尊重探险家的遗产”

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/30/we-need-an-invert-selection-button/>

当昨天开始流传 Windows 8 中 Windows Explorer 新界面的照片时，我确信它们是一个玩笑。当然，这是一个洋葱模型。或者可能是一些苹果粉丝取笑微软的设计美学，更多往往是更多。我的意思是，这是微软不可能真正发布的东西——更不用说在 2012 年了，对吗？

不对。

正如 Windows 首席执行官史蒂文·辛诺夫斯基和 Windows 管理团队成员亚历克斯·西蒙斯[在构建 Windows 8 博客上详细描述的那样，这些截图非常真实。不仅如此，整个 Windows 团队显然已经做了详尽的研究，并为此功能投入了大量时间。这正是他们在博客上吹嘘的原因。整个帖子都在尖叫:“看看我们做了多少作业！”](https://web.archive.org/web/20230203075349/http://blogs.msdn.com/b/b8/archive/2011/08/26/improvements-in-windows-explorer.aspx)

与此同时，这篇文章的言外之意是:微软很紧张。他们现在发布这些截图，以便获得反馈。对我来说，这表明虽然微软确信他们做了功课，但他们实际上对产品本身没有信心。考虑到人们的反应，也不应该如此。

虽然苹果的死忠不喜欢新的 Windows Explorer 的设计并不奇怪，但即使是许多 Windows 观察者也对 T4 持谨慎态度。但在我看来，这比你典型的“[微软没有品味](https://web.archive.org/web/20230203075349/http://www.youtube.com/watch?v=upzKj-1HaKw)”批评还要糟糕。这里的设计和用户界面是我在任何软件中见过的最糟糕的事情之一——更不用说将被数亿用户使用的东西了。

这不仅仅是因为它极其混乱、不均衡和令人困惑(所有这些都是)，而是因为它表明，当谈到计算的未来时，微软的很大一部分正朝着错误的方向前进。我相信，随着计算继续融入我们的日常生活，简单和优雅将越来越占上风。这次 Windows 资源管理器的灾难表明微软的想法正好相反。

也许我错了。毕竟，微软是这个星球上最大的公司之一，每个季度都有数十亿美元的收入。但是我已经说了好几年了。在那段时间里，苹果，一家散发着我正在谈论的东西的公司，已经从灰烬中崛起，成为这个星球上最有价值的公司——远远领先于你知道的那个人。

与此同时，个人电脑制造商——微软在未来竞争中的重要合作伙伴——也像苍蝇一样纷纷倒下。世界头号个人电脑制造商惠普公司正试图出售他们的个人电脑业务。第二大个人电脑制造商宏碁刚刚[公布了有史以来第一次](https://web.archive.org/web/20230203075349/http://www.reuters.com/article/2011/08/24/acer-idUSL4E7JO0HP20110824)*季度亏损。*

 *但是，也许我疯了。可能是我脱离现实了。所以[我昨天](https://web.archive.org/web/20230203075349/http://gopollgo.com/what-do-you-think-of-microsofts-new-windows-explorer-ui-in-windows-8)进行了一项民意调查，看看更多的人对这个新的 Windows 资源管理器界面有什么看法。经过 1300 多次投票，结果如下:

*   47%的人投票支持“太糟糕了”
*   8%的人投票支持“太棒了”
*   19%的人投票支持“没关系”
*   26%的人投票支持“这是我见过的最糟糕的事情”

73%的人对该功能有负面反应。

当然，我想你可能会说我的许多推特粉丝很可能是亲苹果的。但我也在 Google+上进行了投票，这是一个对苹果明显不友好的地方。看看更广泛的互联网周围的反应。可能网络上那些唱衰的已经被苹果洗脑了。或者也许，仅仅是也许，这个设计真的很糟糕。

今天早上，一名苹果恶意软件博主[对我将 iPad 主屏幕和新的 Windows Explorer 进行对比](https://web.archive.org/web/20230203075349/http://www.zdnet.com/blog/bott/competing-visions-of-the-future-of-personal-computing/3837)表示反对[。不公平！，他喊道——然后继续将 iPad 主屏幕与 Windows 3.1 进行比较。当然，这种反应没有解决两件事:1)新的 Windows 资源管理器设计很糟糕。2)重点。](https://web.archive.org/web/20230203075349/http://parislemon.com/post/9558558373/apples-vision-for-the-future-of-computing-versus)

我将为苹果恶意软件博主说得更清楚一点:苹果对未来的愿景是 iOS——更广泛地说，是简单和触摸。微软对未来的愿景是这个令人厌恶的 Windows 资源管理器。

苹果有自己的文件系统管理器吗？是的，OS X 中的 Finder。但是苹果在 2011 年不会在博客中吹捧它。事实上，他们从不谈论它。在 OS X 的最后几次迭代中，他们的目标是消除它对 99%用户的有用性。相反，他们推出了 Stacks 和 Launchpad。如果你真的需要一个单独的文件，为什么不用 Spotlight 呢？

那就是 OS X，苹果现在的小众操作系统。在 iOS 中，没有文件浏览器——更不用说有 19 个工具栏按钮的浏览器了。为什么？因为在苹果看来，“文件”的概念正在发生变化。它们的保存和传送是用户永远不必考虑的事情。[这就是 iCloud 的全部](https://web.archive.org/web/20230203075349/https://techcrunch.com/2011/06/08/apple-icloud-google-cloud/)。

也许我夸大了微软对 Windows Explorer 的展望，但如果这是真的，那么微软自己的领导也是如此。

“Windows 资源管理器是 Windows 桌面用户体验的基础，”辛诺夫斯基写道。“资源管理器是 Windows 最古老的部分之一，”西蒙斯写道。“有机会重新审视和思考我们产品的这一基石有点令人畏惧，但也相当令人兴奋，”他继续说道。

这两种说法和 Windows 8 帖子本身都指出了微软在这里面临的主要问题。在帖子中，他们两次提到“尊重探险家的遗产”。微软成了行李搬运工。

每次更新时，不管他们多么想改变，他们总是被操作系统过去的包袱所束缚。这导致了一种既不新也不旧的奇怪融合。而且往往比两者都更糟。

“保持资源管理器的功能和丰富性，并在 Windows 的当前体系结构和安全模型允许的情况下，恢复 Windows XP 时代最相关和最需要的功能。”——帖子实际上说这是这位新探险家的一个目标。这听起来也像是一个笑话。不是的。行李。行李。行李。特征蠕变。杂乱。

众所周知，苹果厌恶委员会设计和焦点小组的想法。微软显然庆祝这两者。

他们是两种不同的风格和工作方式，这很好。再说一次，两家公司现在都做得很好。这只是我的信念，在不太遥远的将来，窗户会在这个包袱的重压下倒塌。我认为这个新的 Windows 资源管理器是第一个明显的标志。

智能手机和平板电脑正在迅速改变计算的格局。微软在 Windows Phone 的 Metro 用户界面上所做的工作表明，他们能够抛弃行李，跳上火车。但与苹果不同的是，他们在两面下注。他们在 Windows 上加倍复杂，而在 Metro 上强调简单。

苹果已经在采取行动，为未来的计算统一他们的整个生态系统。他们从头开始重新想象经历，摧毁旧的范式。如果 Windows 8 中的 Windows 资源管理器有什么启示的话，微软还没有准备好做同样的事情。

当然，微软可能已经为“Windows 8 平板电脑”准备好了 Metro 皮肤，但现在看来，它将是成堆行李之上的一个表层，而不是对计算的根本反思。这将是[微软鲍勃](https://web.archive.org/web/20230203075349/http://en.wikipedia.org/wiki/Microsoft_Bob) 2.0。

但是，嘿，我有偏见，对不对？在过去的七年里，我对苹果产品的偏好显然与对计算未来的展望无关——我只是被洗脑了。数百万其他人也是如此，而且每年都有数百万人。我们应该看看 Windows 8 中 Windows Explorer 的这些截图，看看未来。我们应该看看 iPad，看看 Windows 3.1。*
