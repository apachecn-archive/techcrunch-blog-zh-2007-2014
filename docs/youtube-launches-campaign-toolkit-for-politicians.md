# YouTube 推出政客竞选工具包

YouTube 已经成为国家和地方政治活动不可或缺的工具。谷歌主持人也被用于以多种方式主持社区市政厅会议。今天，谷歌[在](https://web.archive.org/web/20230324000623/http://youtube-global.blogspot.com/2010/06/inform-engage-and-mobilize-voters-with.html)推出了 [2010 年竞选工具包](https://web.archive.org/web/20230324000623/http://www.youtube.com/youchoose2010)和升级版[谷歌竞选工具包。](https://web.archive.org/web/20230324000623/http://www.google.com/events/campaigntools/)

这些工具包旨在为政治候选人提供成功使用 YouTube 和其他谷歌产品来吸引选民和公民的资源。在 YouTube 上，竞选团队将可以访问一些功能，如政客频道(允许竞选团队宣传他们的频道并上传更长的视频)、谷歌版主和分析工具 YouTube Insight。该工具包还包括付费广告活动，如流内广告和促销视频。谷歌工具包展示了谷歌应用程序家族中的产品，如 Docs 和 Gmail，如何让员工和志愿者保持联系。

这些产品肯定会对明年竞选公职的政治家和候选人特别有用。啰嗦的政客应该喜欢视频时间限制比正常时间长的事实，他们可以在 YouTube 上播放同样的 15 秒和 30 秒的电视广告。值得注意的是，奥巴马总统在他的竞选活动和现在的白宫内部都使用了 YouTube。白宫也使用[谷歌版主和应用引擎。](https://web.archive.org/web/20230324000623/https://techcrunch.com/2009/03/24/white-house-using-google-moderator-for-town-hall-meeting/)
