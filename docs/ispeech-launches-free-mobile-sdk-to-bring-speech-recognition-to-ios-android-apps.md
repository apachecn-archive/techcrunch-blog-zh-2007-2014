# iSpeech 推出免费的移动 SDK，为 iOS 和 Android 应用程序带来语音识别

[iSpeech，](https://web.archive.org/web/20230203041340/http://www.ispeech.org/)开发了用于短信的语音识别应用[drive safe . ly](https://web.archive.org/web/20230203041340/https://techcrunch.com/2010/10/20/drivesafe-ly-2-0-responds-to-text-email-while-you-drive/)，随着新 SDK 的发布，iSpeech 正在将其文本到语音技术引入 iOS、Android 和黑莓应用。

新的自助服务平台允许开发人员将文本到语音和语音识别集成到他们自己的应用程序中。这项服务对所有支持 iOS、Android 和黑莓设备的移动开发者都是免费的。

iSpeech SDKs 和 API 提供 40 多种文本到语音的语音，支持超过 25 种语言。还提供了高级服务，包括名人和自定义文本到语音语音，以及开发人员针对其使用情况提高语音识别性能的能力。

iSpeech 已经向某些合作伙伴开放了 API，包括 TeleNav、Car and Driver、DailyHoroscope 等等。iSpeech 表示，迄今为止，已经完成了 10 亿次文本到语音和语音识别的转换。
