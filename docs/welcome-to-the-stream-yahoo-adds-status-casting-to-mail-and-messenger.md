# 欢迎来到小溪。雅虎在邮件和信使中加入了“状态播报”

![](img/ee7b9c3a8acf2cbfd9c0788fcd55eb60.png)

当你在游戏中迟到时，试图重新命名并不能为你赢得任何分数。今天，雅虎宣布，它终于在邮件和信使产品中加入了基本的状态更新，并称之为[“状态播报”](https://web.archive.org/web/20221206201636/http://www.beta.techcrunch.com/2009/08/24/live-from-yahoos-what-matters-most-event/)在雅虎邮箱和 Messenger 10 中，你都可以更新你的状态，所有使用这两款产品的联系人都可以看到你的更新。您还可以选择在您的邮件主页或即时消息流中查看您朋友在各种社交媒体网站上的更新，如 Yelp、YouTube 和 Twitter。

雅虎通过结合私人和公共信息流，使其通讯产品更加社会化，就像美国在线上个月为 AIM 增加生活流一样。(没错，AOL 在这个功能集上比雅虎领先了一个多月)。

一方面，雅虎希望利用雅虎邮箱(Yahoo Mail)的人气(雅虎邮箱是排名第一的网络邮件服务，全球有 3 亿人使用它)进入微消息游戏。就像年初的雅虎简介一样，你现在可以通过雅虎邮箱给雅虎上的其他人添加 140 个字符的更新。它还可以让您跟踪您的联系人在其他社交网站上的活动。这些出现在雅虎邮箱登录页面的一个新的更新部分。

Yahoo Messenger 允许您做同样的事情—在您的 Yahoo 网络上创建更新，并查看您的 IM 好友在其他地方的更新。但这似乎不是一种双向联系。你可以在 Twitter、脸书、Yelp、Flickr、YouTube、Digg 和其他地方阅读你朋友的更新，但是你不能总是在 Yahoo Messenger 或 Mail 中回复，这样你的回复就会出现在原来的服务中。如果你想回复朋友的推文，你可以给他们回一条附有他们推文副本的即时消息，但你不能像任何 Twitter 客户端那样简单地生成一条新推文来回复。

这不是真正的状态广播，或者不管你想叫它什么，除非你可以通过非雅虎服务广播你的状态。
