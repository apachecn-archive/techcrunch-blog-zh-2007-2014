自从 [Movieclips 更名为 ZEFR](https://web.archive.org/web/20230205141708/https://techcrunch.com/2012/08/15/movieclips-18-5m-funding-rebrand-zefr/) 以来，已经有大约九个月的时间了，它向在 YouTube 上分发或声称其内容的出版商提供内容 ID 和货币化工具。今天，这家初创公司宣布了它的第一批大客户，签约了 UMG、索尼音乐、Ultra Music 和 NASCAR 等大牌音乐公司。

对于那些还没有忘记这一点的人来说，ZEFR 的故事是这样的:很久以前，有一家衣衫褴褛的小初创公司，名叫 Movieclips。这家总部位于洛杉矶的初创公司的创立前提是为电影片段授权，并通过 YouTube 提供这些片段，从而使电影制片厂能够将封存在自己金库里的大量长尾内容转化为利润。

但这还不是全部！Movieclips 还开发了一些工具，帮助用户识别粉丝发布的视频片段并将其转化为收入。它还确定了最佳实践，创造了能让观众参与进来的技术——比如一面滚动后的视频墙，可以显示多个相关场景，观众可以点击浏览。它对技术的运用非常棒，但 Movieclips 意识到它可以用同样的工具来帮助音乐和体育等其他领域的内容提供商从他们制作的内容中获利。

于是，ZEFR 诞生了。ZEFR 是 Movieclips 的更名，原因是这家初创公司筹集了更多资金，并开始向其他人提供内容 ID 和盈利工具。

如今，在优化技术并在各种垂直市场争取内容所有者支持后，它已经成功拉拢了许多主要的第三方内容提供商。他们希望 ZEFR 不仅能找到用户发布的视频，帮助内容所有者赚钱，还能帮助他们充分利用自己的渠道。

对于纳斯卡这样的客户来说，这意味着使用一些 ZEFR 自行开发的渠道管理工具。这家初创公司在自己的 Movieclips 业务中使用了这些工具，以确保获得最佳实践并改善搜索引擎结果，但卷后视频墙之类的东西也能让观众与客户的内容保持互动。对于音乐唱片公司而言，与 ZEFR 合作可以让超级粉丝与他们最喜欢的艺术家一起创作和上传视频，同时也可以确保唱片公司(和艺术家)从这些宣称拥有版权的视频中获得广告收入。

自 2009 年成立以来，ZEFR 已经募集了近 3000 万美元，其中包括由美国风投公司 Venture Partners 牵头的 1850 万美元的融资。其他投资者包括 MK Capital、Shasta Ventures、SoftTech VC、First Round Capital、Richmond Park Partners 和 Machinima 首席执行官艾伦·德布雷沃斯。
