# 红杉在移动业务数据可视化应用 Roambi 上领投 3000 万美元

[MeLLmo](https://web.archive.org/web/20230204232929/http://www.crunchbase.com/company/mellmo) ，商业数据可视化平台 [Roambi、](https://web.archive.org/web/20230204232929/http://www.roambi.com/)的开发商[已经通过红杉资本(Sequoia Capital)牵头的首轮机构融资筹集了 3000 万美元](https://web.archive.org/web/20230204232929/http://www.businesswire.com/news/home/20110915006792/en/MeLLmo-Secures-30M-Funding-Led-Sequoia-Capital)。这使梅尔莫的资金总额达到 5000 万美元。作为该轮融资的一部分，红杉合伙人格雷格·麦卡杜(Greg McAdoo)将加入梅尔莫的董事会，大卫·斯佩克特(David Spector)将作为董事会观察员加入。

Roambi 于 2009 年推出，允许商业专业人士从任何地方访问可视化数据。Roambi 的网站和 [iPhone](https://web.archive.org/web/20230204232929/https://techcrunch.com/2009/11/16/roambi-pros-iphone-app-lets-businesses-visualize-data-on-the-go/) 和 [iPad](https://web.archive.org/web/20230204232929/http://www.techcrunchit.com/2010/04/02/roambi-brings-data-visualization-app-to-the-ipad/) 应用程序允许在网络和移动平台上以可定制的图表和图形轻松查看电子表格和文档中的数据。

Roambi 用户只需将 Excel 电子表格、Google 电子表格、HTML 表格数据、CSV 文件和 Salesforce CRM 报告中的数据上传到在线 Roambi Pro Publisher。Roambi 还支持从业务应用程序上传数据，包括来自 IBM、Microsoft、Oracle 和 SAP 的数据。下载数据后，您可以选择一个预先设计的模板来交付数据(如饼图、图表等。).一旦发布，这些数据就会转化为 iPhone 和 iPad 的交互式可视化效果。

Roambi Trends 可以识别和可视化数据趋势，并提供用户选择的任何时间段的数据比较——年、月甚至分钟；用户可以通过滑动图表底部的日期变更线来轻松关注特定时间段。CataList view 将数据组织到分层的分类列表中，能够深入到详细的摘要中，并同时比较或分析不同的数据集。

Roambi 的好处是它可以被小型和大型企业使用。对于企业用户，Roambi 提供了 Pro 版本，这是一种连接移动设备的安全托管服务；Roambi ES 是一种安全的本地服务器解决方案。正如我们过去所写的，Roambi 应用程序中的数据视图令人印象深刻，因为充满数据的电子表格会自动转换为交互式图形和图表，使您可以轻松理解正在提供的信息。

该公司最近推出了 Flow，允许用户通过 iPad 上的一个光滑的报告来创建和分享所有类型的商业信息和情报。新应用将在未来几周内公开发布。

最新一轮融资将用于国际扩张区域招聘以及 Roambi Analytics 和 Roambi Flow 本地化版本的开发。

商业智能是一个热门领域，吸引了风险公司的大量投资。8 月，Andreessen Horowitz 领导了商业智能和分析初创公司 Good Data 的一轮 1500 万美元投资。
