# 索尼宣布推出 32GB 记忆棒 PRO-HG Duo HX

[![blisterhxok2](img/0a59113e83fa2df98d1db8803bc32574.png "blisterhxok2")](https://web.archive.org/web/20230324115539/https://techcrunch.com/wp-content/uploads/2009/07/blisterhxok2.jpg)

极速前进。！！！

> 最高可用容量——32GB 记忆棒 HX 系列
> 超高速 20MB/秒数据传输大大缩短了等待时间
> 非常适合高清摄像机——或处理大量图像的任何人
> 
> 索尼推出了有史以来最大容量的记忆棒 PRO-HG Duo HX，以一个巨大的 32GB 模型为该系列增添了光彩。HX 记忆棒已经提供了令人印象深刻的速度，因此 32GB 版本的引入使其成为处理高清视频拍摄时创建的非常大的文件的完美选择。巨大的空间也是静态摄影的理想选择，无论你使用的是索尼α这样的复杂 DSLR，还是小巧的随处可去的 Cybershot。
> 
> 摄像机用户可以使用单个 32GB 记忆棒 PRO-HG Duo HX 在全高清模式(1920 x 1080)下拍摄令人印象深刻的 225 分钟，几乎消除了对剩余空间的担忧。
> 
> 商标记忆棒 HX 的性能可以让您自由专注地制作出色的高清视频。使用附带的 USB 适配器，可以实现 20MB/s 的读取速度和 15MB/s1 的写入速度，因此您可以减少等待视频传输到 PC 的时间。
> 
> 对于拥有技术先进的数码单反相机(如具有连续拍摄功能的索尼α)的用户来说，这种尺寸和速度的有力结合同样是个好消息。或者，任何拥有紧凑型数码相机的人，如果只是不想担心填满内存，现在有了理想的解决方案。
> 
> 记忆棒 PRO-HG Duo HX 系列还包括 4GB、8GB 和 16GB 型号，所有版本都配有免费的 USB 适配器，以实现最佳性能并轻松连接到各种硬件。
