# 比较新款 iMac 键盘和老款键盘

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/20/comparing-the-new-imac-keyboard-to-the-oldies/>

[![ikeyboard11.jpg](img/8c5f8638e736cad25ac190e47205e2dc.png)](https://web.archive.org/web/20151110170124/http://old.crunchgear.com/wp-content/uploads/ikeyboard11.jpg "ikeyboard11.jpg")

上周，老好人文斯[给了](https://web.archive.org/web/20151110170124/http://crunchgear.com/2007/08/17/the-new-apple-keyboard-reviewed/)新款 iMac 键盘一个合格的大拇指，而且，傻傻的你，你可能想知道这款键盘与其他老款苹果键盘相比如何。你当然知道。有一个人贴出了这样一个图片库，向我们展示了新款车与一些老款车的对比。怀旧，全速前进。

有几个亮点:

记得六年级的时候用苹果 IIs。我想是 1997 年。显然，我参加了平破产小学。

[![ikeyboard2.jpg](img/548147a18f548bb99522ab2564d0869f.png)](https://web.archive.org/web/20151110170124/http://old.crunchgear.com/wp-content/uploads/ikeyboard2.jpg "ikeyboard2.jpg")

这是下一个键盘，你可能记得上周在易贝上展示的[。我个人认为黑色比新款的银灰色更好看。](https://web.archive.org/web/20151110170124/http://crunchgear.com/2007/08/15/complete-next-computer-set-shows-up-on-ebay/)

[![ikeyboard3.jpg](img/5e383bee3f9bb1c545a9bbe7aacb39ee.png)](https://web.archive.org/web/20151110170124/http://old.crunchgear.com/wp-content/uploads/ikeyboard3.jpg "ikeyboard3.jpg")

最后(为了不抢那家伙的风头，我只在这里贴了几个)是我对 Mac 平台的正确介绍所联系的键盘，这是 2004 年左右的键盘吗？它看起来和我的 iMac 的键盘非常相似，只是键下有污垢。

我认为，我们距离未来风格的虚拟键盘可能还有几年的时间。
