# Songkick 从 Index 融资 400 万美元

总部位于伦敦的音乐会列表网站 Songkick 允许音乐迷跟踪乐队、巡演日期和购买门票，该网站已获得 Index Ventures 400 万美元的首轮融资。他们之前总共从 Y Combinator、SoftTech VC 和 Accelerator Group 筹集了 102 万美元。

这项投资背叛了 Index 投资娱乐导向型初创公司的战略，包括 Listen.com(Real Networks)、Last.fm (CBS)和 Viagogo(二级票务市场)，因为娱乐通常是人们在经济低迷时期钻研的东西。现场音乐也是音乐行业中为数不多的可以防御的领域之一。

Index 的索尔·克莱恩将加入 Songkick 的董事会。Songkick 目前在 5 个国家可用，投资将用于扩展平台和推出新产品。
