# 热衷于…互联网的问题:消费者想要免费的文化(TCTV)

今年夏天，在著名的[爱丁堡国际图书节上，](https://web.archive.org/web/20221206113938/http://www.edbookfest.co.uk/)获奖的苏格兰作家兼电影制作人[伊万·莫里森](https://web.archive.org/web/20221206113938/http://www.crunchbase.com/person/ewan-morrison)做了一个极具争议的演讲，关于传统作者和这本书的[之死](https://web.archive.org/web/20221206113938/http://www.guardian.co.uk/books/2011/aug/22/are-books-dead-ewan-morrison)。事实上，莫里森的演讲在欧洲引起了如此大的轰动，以至于我邀请他来到 TechCrunchTV，让我们的全球观众熟悉他有争议的观点。

质朴的莫里森没有让人失望。他认为互联网是“极端美国资本主义的典范”，并告诉我，如今高质量的书籍和作者正被三股力量扼杀:谷歌、亚马逊和消费者。虽然莫里森关于谷歌和亚马逊的观点之前已经被许多批评家(包括我自己)提出过，但他对在线消费者的批评可能是最有争议的。但他是对的吗？真的有可能是网络消费者——他们对免费或非常便宜的内容贪得无厌——正在扼杀我们的文化吗？
