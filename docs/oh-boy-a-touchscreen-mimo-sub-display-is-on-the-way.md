# 哦天啊！触摸屏 MIMO 子显示屏即将问世

![touchy](img/1a2750f627a684dd5bfcb61c9a0ee6dd.png "touchy")
当然，我不指望我们的读者，尽管他们很聪明(也很性感),能立即知道什么是 MIMO 子显示器——所以我把你带到[我们关于这个有趣的小玩意儿的最新版本的视频评论](https://web.archive.org/web/20221007094417/http://www.crunchgear.com/2009/07/28/video-review-thinkgeek-mimo-mini-usb-monitor/)。基本上它是一个小显示器(7-8 英寸)，用于放置你的目标列表、媒体播放器或种子窗口。方便，如果你喜欢这种东西，可以使用你的桌面上的一点额外空间。

就我个人而言，没有手机我也没意见……直到他们宣布[将推出触屏版](https://web.archive.org/web/20221007094417/http://www.slashgear.com/nanovision-confirm-touchscreen-mimo-720-s-7-inch-usb-subdisplay-1051607/)。720-S 是制造商 Nanovision 即将推出的两个 MIMO 版本之一，它本身将有一个漂亮的小触摸屏(我猜是电容性的，如果你必须使用手写笔，你会打翻它)。

这将是一个令人惊叹的外设，特别是在 Windows 7 这样的系统中，它将针对多点触摸和手势控制进行优化。价格尚未公布，但其非触摸屏的关系，710-S，将花费 150 美元。所以至少 200 美元的酷版本，但不要引用我。如果运气好的话，我们会有一个人来评论，然后你们就会知道整个故事了。
