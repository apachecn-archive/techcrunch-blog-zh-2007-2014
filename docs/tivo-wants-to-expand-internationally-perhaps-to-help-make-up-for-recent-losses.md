# Tivo 希望进行国际扩张(也许是为了弥补最近的损失)

[![](img/537b5f05cab9e9f2630db665a1bb65d5.png "tivoeu")](https://web.archive.org/web/20221208010949/http://www.crunchgear.com/2010/08/26/tivo-wants-to-expand-internationally-perhaps-to-help-make-up-for-recent-losses/tivoeu/)

欧洲人，请注意: [Tivo](https://web.archive.org/web/20221208010949/http://www.crunchgear.com/tag/tivo/) 可能很快就会以一种非常非常大的方式来到你的社区。该公司总裁兼首席执行官汤姆·罗杰斯表示，该公司[希望在更大的欧洲市场(包括英国和西班牙)获得更多立足点](https://web.archive.org/web/20221208010949/http://www.broadbandtvnews.com/2010/08/26/loss-making-tivo-looks-internationally/)，以便稍微拓宽视野。减轻最近 1530 万美元损失的痛苦也不会有什么坏处。

罗杰斯说，虽然许多美国电视供应商(你的康卡斯特，你的时代华纳等。)可以提供他们自己的基本 DVR 解决方案，但不是每个欧洲供应商都有这种能力。

> 与美国的一些大型运营商不同，国际运营商必须依赖外部合作伙伴提供他们的机顶盒软件。很少有公司能够提供可靠的解决方案，将日益复杂的机顶盒和网络基础设施结合起来，处理一系列难以置信的动态和具有战略重要性的面向消费者的功能。

这就是 Tivo 的用武之地，根据 Matt 的说法，[它是一个很棒的遥控器](https://web.archive.org/web/20221208010949/http://www.crunchgear.com/2010/08/24/tivo-slide-qwerty-remote/)。想象一下这个场景:“你好，欧洲电视供应商。您的客户需要合适的 DVR 解决方案吗？我们一起吃午饭，讨论这个问题。”

Tivo 也在寻求进入印度和中国。话说回来，有哪家公司*不是在努力向印度和中国扩张呢？*

差不多就是这样:Tivo 正在寻求扩张，就像任何企业一样。
