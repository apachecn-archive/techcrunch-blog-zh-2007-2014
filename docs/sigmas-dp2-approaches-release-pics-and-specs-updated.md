# 适马 DP2 接近释放；图片和规格更新

![headline](img/89f85a7446992765b2491f993f82aca4.png "headline")
我们知道 [DP2](https://web.archive.org/web/20221005100117/http://www.crunchgear.com/2008/09/23/sigma-unveils-new-compact-camera-the-dp2/) 已经有一段时间了，虽然它的前身是[酷酷的](https://web.archive.org/web/20221005100117/http://www.crunchgear.com/2008/05/19/sigmas-dp1-hits-hybrid-point-and-shootdslr-ruled-interesting-but-lacking/) [接收了](https://web.archive.org/web/20221005100117/http://www.crunchgear.com/2008/07/24/the-sigma-dp1-is-a-technical-marvel-but-thats-about-it/)，但我对这一款感到兴奋。每个人都认为 DP1 是一项技术成就，但它的实际性能是缺乏的。自动对焦、快门延迟、一般速度是个问题，弱光下表现不佳——这样的例子不胜枚举，甚至比我记忆中的还要多。但是，如果适马已经做出了必要的改变，这可能是一个非常令人印象深刻的相机。

[PSGallery=3mbl45b9ll]

但是我越看规格越觉得一样。它使用类似的传感器(在[收购 Foveon](https://web.archive.org/web/20221005100117/http://www.crunchgear.com/2008/11/11/sigma-buys-out-sensor-maker-foveon/) 后，他们可能挪用了他们的一些改进技术)，类似的固定镜头(尽管明显更快——f/2.8 比 f/4)，机身看起来几乎完全没有变化。按钮配置有所改变，但仅此而已，LCD 屏幕(在评论中遭到唾骂)似乎也没有改变。你可以把焦距调近一点(11 英寸)，但这说明不了什么。

我真的希望这款相机能成为比 DP1 更好的套件，但除非大量的变化未经宣传，否则我不认为它会超越它太多。尽管如此，我们还是会尽我们所能来回顾它，并报道世界上其他人对这款有趣的相机的看法。
