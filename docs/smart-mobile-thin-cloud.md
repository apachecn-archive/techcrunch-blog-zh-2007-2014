# 智能手机和瘦云

***编者按**:客座作者[基思·蒂尔](https://web.archive.org/web/20230204111903/http://www.crunchbase.com/person/keith-teare)是其孵化器[阿基米德风险投资](https://web.archive.org/web/20230204111903/http://archimedesventures.com/)的普通合伙人，也是新出资 [just.me](https://web.archive.org/web/20230204111903/http://just.me/) 的首席执行官。他是 TechCrunch 的联合创始人。*

所有固体都会融化成空气。不会有 web 3.0！

当惠普首席执行官李艾科宣布该公司正在为其消费者个人电脑业务寻求选择，并放弃硬件移动业务时，其股票下跌了 20%。当史蒂夫·乔布斯[辞去苹果公司首席执行官](https://web.archive.org/web/20230204111903/https://techcrunch.com/2011/08/24/steve-jobs-resigns-from-apple/)职务时，经过 12 个小时的回调，苹果公司的股票在 48 小时内上涨，超过了之前的收盘价。

这两个关于公司变革的故事被广泛讨论，通常认为每个公司的核心人物就是故事。然而，更大的事情正在发生，它将在未来 5 年内改变整个软件生态系统。这些变化将如此巨大，以至于目前关于泡沫的讨论将显得愚蠢。大公司会倒闭，甚至更大的新公司会成立。

我们所处的时代的基础具有以下特征{

*   包括笔记本电脑在内的台式计算设备正被简化为用于执行严肃工作任务的机器。将来会有更少的人购买它们，那些购买的人会更少的使用它们。
*   为 Web 2.0 时代编写的软件，假设云中的服务由坐在办公桌前使用浏览器的人消费，将越来越不相关，使用频率也越来越低。甚至像脸书和 Google +这样相对较新的项目也会很快变得“过时”。
*   移动设备，尤其是智能手机，将会完成越来越多个人想要完成的事情，而且会更容易和更有效率。部分由于结合了强大软件和服务的始终在线设备的易用性和便利性，采用速度将会很快，数量将会很大。
*   软件和服务将在这些设备上运行，并使用云进行存储和交付。富客户端将使用瘦云。云将变得更大但更简单。旧的 web 2.0 web 应用和 web 服务模式将会衰落。
*   苹果的 iPhone 架构最适合这种新兴的人类体验。
*   谷歌的 Android 主要是谷歌厚云(Docs、Gmail、Calendar、Contacts、Picasa、G+)的瘦客户端，这将取悦极客，但需要改变以成为有眼光的消费者的主流选择。与早期的塞班不同，它的巨大销量将更多地取决于商品价格，而不是激情。
*   脸书是典型的厚云生态系统，在这一过渡期间将非常脆弱，因为几乎其整个业务都依赖于基于云的架构，该架构持有一个人的社交图，并成为在该图上采取行动的手段。对于用户来说，作为瘦客户端的脸书移动应用程序不会像真正的移动社交网络那样简单或强大。
*   任何在 2011 年建造任何东西的人都应该考虑“[移动优先](https://web.archive.org/web/20230204111903/https://techcrunch.com/2011/07/29/watch-the-mobile-first-crunchup-live-here/)”并且可能“仅移动”。它们应该让智能手机用户能够以最小的忙乱和最大的效率完成复杂和简单的任务。他们不应该要求一个人去坐在办公桌前做任何事情。

这整个转变是由始于 iPhone 的生态系统推动的。相比之下，Android 是一个向后看的架构。这是一种向移动受众分发旧 web 服务的方式，但它并没有改变软件，以满足移动提供的机会或它开辟的更简单的新架构。

从这个角度来看，惠普想要退出硬件业务是正确的。在接下来的几年里，那里的将是[的天下。苹果遥遥领先。](https://web.archive.org/web/20230204111903/https://techcrunch.com/2011/08/18/apple-wins-without-throwing-a-punch/)

谷歌仍然有机会因为 Android 而变得重要，但它需要对 Android 哲学进行巨大的改变来实现这一点。HTML5 不足以使移动未来的 web 2.0 生态系统现代化。一个新的生态系统正在形成，这是由一个更加分散的、以用户为中心的移动应用生态系统的用户乐趣所驱动的。

除非脸书能够转向一种完全不同的、去中心化的架构，否则它将面临保持其在社交网络领域领先地位的挑战。但作为 web 2.0 最集中的成功案例，这一点并不明显，也不容易，直到为时已晚。

接下来的 10 年将会非常有趣。感谢的是。。。苹果和史蒂夫·乔布斯。认为不同不再是一种选择。
