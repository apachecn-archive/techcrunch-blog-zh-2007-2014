# 修改社区为 TouchPad Android 端口提供 1500 美元奖金

本周末的 TouchPad 大甩卖或多或少已经结束，数以千计的惠普小型平板电脑找不到新家。然而，似乎并不是每个人都喜欢 WebOS，因为越来越多购买便宜货触摸板的人都渴望有更多的平板电脑体验…我们可以说，相关。

由于其开放性，Android 是 TouchPad 端口的首选操作系统，现在[一个名为 HackNMod](https://web.archive.org/web/20230205032800/http://hacknmod.com/hack/hp-touchpad-android-1500-prize/) 的修改社区的人正在向能够实现这一目标的开发者提供奖金。

可供争夺的总金额为 1500 美元，但它被分成几个部分，因此多个开发者将有机会获得支付。

谁能第一个在 TouchPad 上安装并运行“基本 Android 端口”,谁就能净赚 400 美元。HackNMod 没有详细说明“基本端口”的确切构成，但可以推测，该设备必须以相当高的稳定性启动到 Android。子奖金涵盖了所有支出:使用 WiFi 工作的团队获得 350 美元，音频团队获得 300 美元，多点触控团队获得 100 美元。开发人员还有一件事要处理:他们的解决方案必须足够简单，普通用户可以自己安装。

就像所有的事情一样，有一个小小的陷阱:虽然 Android 端口本身的 450 美元来自 HackNMod 的口袋，其余的则留给了赞助商。在某种程度上，对于这样的项目来说，这是一个完美的系统:需求最高的特性(希望)将会得到资助。感兴趣的人也可以为他们希望看到的其他功能投入资金，从而为渴望开发的人带来更多的子奖励。
