# 游客之眼首次在 Android 和 iPhone 上推出离线移动旅游指南

![](img/a29b55210997c8bf37cf5f7ce865f462.png) [《游客之眼》](https://web.archive.org/web/20221007004253/http://www.touristeye.com/)今天早上退出了他们的封闭测试版，因为他们同时为 [iPhone](https://web.archive.org/web/20221007004253/http://beta.touristeye.com/iphone) 和 [Android](https://web.archive.org/web/20221007004253/http://beta.touristeye.com/android) 推出了他们的应用。两位年轻的西班牙创始人 Javier Fernández Escribano 和 Ariel Cámus 瞄准了国际智能手机用户群。该公司为创建和分享旅行计划以及编写旅行指南提供了一个合作空间，并将与 Tripwolf 等其他初创公司展开竞争。

[![](img/5b980319551ce94bcf36d1032a20fd52.png "everywhere")](https://web.archive.org/web/20221007004253/http://eu.beta.techcrunch.com/wp-content/uploads/everywhere.png)

用户可以在网上创建自己的旅行指南，与他人一起计划旅行，然后在旅途中通过智能手机访问他们的指南。一个好处是，大多数经常旅行的人可能会喜欢，通过他们的应用程序，用户可以离线浏览他们拼凑的指南，放弃高额的漫游费。如果你不是在漫游，而不是在旅行结束后上传你的轨迹，你可以边走边报告，包括签到、图片上传等。游客之眼让你在旅途中通过与脸书的联系分享你的行动。然而，如果你没有数据连接或者你正在漫游，应用程序会用日期和时间以及你的地理位置来标记你的活动，这样一旦你上传，你的网络旅行日志实际上将是你旅行活动的精确蓝图。

由于强调分享和社区，不仅用户在旅行时分享，而且随着社区的发展，用户将在旅途中收到来自其游客之眼连接的实时移动通知，其中包含评论和旅行建议。

竞争很激烈，提供的服务种类也各不相同。以巨头如 [Tripadvisor](https://web.archive.org/web/20221007004253/http://www.tripadvisor.com/) 或其他在线旅游社区如 [Tripwolf](https://web.archive.org/web/20221007004253/http://www.tripwolf.com/) 或 [Minube](https://web.archive.org/web/20221007004253/http://www.minube.com/) 为例；这些更侧重于网络。TripIt 之类的网站更加面向移动，但更倾向于关注你的官方旅行预订和行程。最接近的赌注是 mTrip，它专注于旅行指南、旅行计划和移动平台上的分享。

该团队正努力在短期内推出黑莓和 Windows Mobile 7 版本。
