# YouTube 现在将 2D 视频转换为 3D，取消了验证用户 15 分钟的限制

我讨厌 3D 视频。很有可能，你也讨厌 3D 视频。然而，世界上的电子产品制造商坚持将 3D 显示技术应用到从电视到智能手机的所有产品中，尽管事实上仍然没有什么值得一提的 3D 内容。除了《阿凡达》——但是说真的，我几乎不能在第一次没有打瞌睡的情况下看完那部电影。

但是无论视频走向何方，Youtube 都会紧随其后。YouTube 已经为 T2 提供 3D 视频支持有一段时间了，最大的阻碍是只有非常非常少的人拥有能够捕捉额外维度的相机。为了在他们的网站上获得更多的 3D 内容，YouTube 今天推出了一个 2D 到 3D 转换系统的测试版。他们还取消了所有验证用户的 15 分钟视频限制。

与传统的 3D 视频捕捉设置相反，传统的 3D 视频捕捉设置需要用花哨的软件将两个同时捕捉的视频合并，Youtube 的新转换工具可以测量单个视频的运动和颜色，以模拟深度，并拼凑出粗糙的 3D 视频。Youtube 承认，它不能与以正确的方式拍摄 3D 视频相提并论——但是，嘿，这是一个测试版。你仍然需要 3D 眼镜(或者至少是不戴眼镜的 3D 显示器)才能正常观看视频。

YouTube 还发布了另一个相当引人注目的公告，尽管他们只是顺便提了一下:他们取消了对任何信誉良好(即:任何没有上传色情或盗版的人)和“经过验证”(即:任何愿意交出电话号码的人)的 15 分钟视频长度的限制。终于，长达一小时的《你的芯片&戴尔斯营救游侠粉丝音乐视频》导演剪辑版可以在网上有个家了！

他们还确认了他们与 Magistro 的新视频编辑合作关系(我们在这里打破了[)，以及与效果/标题编辑工具](https://web.archive.org/web/20230203160930/https://techcrunch.com/2011/09/21/youtube-magisto/) [Vlix](https://web.archive.org/web/20230203160930/http://www.youtube.com/create_detail/VlixVideo) 的合作关系。
