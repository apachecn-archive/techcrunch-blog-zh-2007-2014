# doubleTwist 为 Android 设备推出了一款精巧的媒体播放器

“安卓版 iTunes”软件 doubleTwist 可以让你在包括安卓手机在内的数十种设备上同步你的媒体文件，它向成为安卓媒体同步的成熟解决方案又迈进了一步。该公司发布了一款原生安卓应用，可在[安卓市场](https://web.archive.org/web/20221207131851/http://apps.doubletwist.com/doubleTwist/-395703074765864636)免费获得，为用户提供了一款能够播放音频和视频的精致媒体播放器。

Android 因其笨重的默认媒体播放器而臭名昭著，doubleTwist 轻而易举地击败了它，拥有更干净的界面，能够导入 iTunes 播放列表，并支持音频/视频播客。也就是说，已经有很多第三方解决方案击败了传统的 Android 播放器。那么是什么让 doubleTwist 与众不同呢？

首席执行官 Monique Farantzos 表示，新的 doubleTwist Android 播放器可以直接与 doubleTwist 的桌面软件一起工作，就像 iPod 和 iTunes 一起工作一样。用户可以在手机和电脑之间来回传输收视率、播放次数和其他信息。Farantzos 还表示，在线广播和对无线媒体同步的支持(这将是很大的)将在不久的将来到来。

值得指出的是，Android 将很快允许用户从他们的个人电脑上[播放他们的音乐](https://web.archive.org/web/20221207131851/https://beta.techcrunch.com/2010/05/20/google-buys-simplify-media-to-power-music-syncing-for-new-itunes-competitor/),而无需物理连接他们的设备，但我怀疑仍然会有很多人希望在本地存储和管理他们的媒体。
