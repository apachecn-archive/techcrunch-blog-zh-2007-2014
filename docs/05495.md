# Digg 创始人杰伊·阿德尔森(Jay Adelson)同意不提前出售，即使是在裁员的情况下

今天在 [FailCon](https://web.archive.org/web/20221004032332/http://www.failcon2010.com/) 的舞台上，[杰伊·阿德尔森](https://web.archive.org/web/20221004032332/http://www.crunchbase.com/person/jay-adelson)回顾了他从 Equinix 到 Digg 的传奇职业生涯。阿德尔森一直强调，尽管 Digg 未能确定 Current 和谷歌的收购要约，但他并不后悔，而[新闻在小组讨论期间爆出](https://web.archive.org/web/20221004032332/http://twitter.com/#!/ronaldmannak/statuses/28719548024)Digg 刚刚解雇了 37%的员工。

我问阿德尔森，当他在台上时，他是否想用最近报道的信息重新评估他的“不后悔”的评论，特别是关于提前出售公司的评论。

阿德尔森看起来真的很关心这个消息，他肯定地说他不会做任何不同的事情，

“我知道那些忠诚的员工希望退出。我认为他们从那里获得了知识和智慧，马修·威廉姆斯和政府做出这样的决定是因为他们有新的数据。”

当被问及他是否认为 Digg 已经死了，正如许多媒体在 Digg 第四版改版后所暗示的那样，Adelson 坚定地说没有:

*“这个网站有 2000 万人(每月唯一)，这是一个很大的数字。当媒体报道 2000 万人因为他们访问的网站“死了”而变得愚蠢时，这是一种侮辱和怪异。" Digg 可能没有 100 倍的回报前景，但它有未来."*

阿德尔森之前讲述的职业轨迹是一个个人痛苦的故事，他还明确表示，销售从来都不是一个可行的选择。空谈不值钱。”
