# 谷歌的电视战略注定要失败

我觉得我们像是在看谷歌电视从附近的沟里滚出来。该公司对该平台并没有太多的投入，而且像苹果电视一样，GTV 也未能获得足够的关注。

正如赖安指出的那样，在一些 I/O 会议之外，谷歌今年很少提及谷歌电视项目，我几乎可以肯定这意味着他们已经放弃了这个项目，但不想告诉他们的合作伙伴。他们在 I/O 平台上举办研讨会，但在正式宣布结束之前，他们必须保持形象。

明确地说，一段时间以来，我一直是 GTV 相当受欢迎的用户——这曾是我继 Boxee Box 之后的首款智能电视解决方案——但很明显，谷歌无法在变幻莫测的机顶盒世界中立足。以下是一些原因。

首先，电视广播公司不想与任何想从内容中赚钱的人合作。当然，他们会在这里或那里出售一个节目，但是除非你家里的机顶盒正从他们的卫星上流出纯净的节目流，否则他们不想参与其中。通过联合选择搜索和发现，谷歌在广播公司看来就像一个寄生虫有机体，而不是一个赚钱的人。TiVo(勉强)生存下来，因为它充当了一个智能录像机。苹果电视和 GTV 的传播速度很慢，因为它们充其量是被美化的媒体播放器，如果没有真正的广播公司的支持，它们将一事无成。

第二，没有人知道智能电视应该是什么，但 GTV 不确定。没有人能完全确定他们想要电视做什么。它应该将内容流式传输到家庭吗？允许你在大屏幕上看 YouTube 吗？提供在沙发上发推特的方式？所有这些都可以在笔记本电脑和平板电脑上更快、更有效地实现。为什么选择房子里最大的屏幕？

GTV 有点像电视世界的叠加，就像谷歌眼镜应该是现实世界的叠加一样。不幸的是，这种叠加很少在实践中起作用，因为它会分散人们对手头程序的注意力。电视观众并不活跃。所有这些关于提供流媒体统计数据的第二个屏幕的讨论在一分钟内都是很酷的，当你想知道谁主演了*飞行计划*。一旦你意识到空姐是埃里卡·克里斯滕森，她也曾在*游泳迷*中，你就差不多完成了第二个屏幕。没有人想从他们的电视上查看节目、搜索相关视频或推特。没人。

我可能错了。也许谷歌在 GTV 领域为我们带来了一个真正的亮点。但是机顶盒将很快被更强大的 DVR 或游戏控制台超越，它们提供真实价值而不是感知价值。比方说，我更希望我的 Xbox 能让我不受约束地访问电视内容，而不是让 GTV 坐在我和碟订阅之间。DVR 已经完善了电视最好的东西。其他的都只是分散注意力。
