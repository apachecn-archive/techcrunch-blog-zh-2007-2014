# WITN:伦敦会很生气，但我们更喜欢柏林的初创公司

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/05/22/witn-london-is-gonna-be-pissed-but-we-prefer-berlin-for-startups-tctv/amp/>

 <amp-img class="alignleft size-medium wp-image-305672 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" title="berlin" src="https://web.archive.org/web/20230313100516im_/https://techcrunch.com/wp-content/uploads/2011/05/berlin.jpg?w=300" alt="" srcset="https://web.archive.org/web/20230313100516im_/https://techcrunch.com/wp-content/uploads/2011/05/berlin.jpg 585w, https://web.archive.org/web/20230313100516im_/https://techcrunch.com/wp-content/uploads/2011/05/berlin.jpg?resize=150,95 150w, https://web.archive.org/web/20230313100516im_/https://techcrunch.com/wp-content/uploads/2011/05/berlin.jpg?resize=300,190 300w, https://web.archive.org/web/20230313100516im_/https://techcrunch.com/wp-content/uploads/2011/05/berlin.jpg?resize=50,32 50w" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 上周我们花了很多时间探索柏林，参加了 [Next11 会议](https://web.archive.org/web/20230313100516/http://nextconf.eu/next11/)并会见了一些当地的初创公司。从视频娱乐的角度来看，最大的问题是什么？我们都认为柏林比伦敦更适合创业。

“感觉像”在这里是一个重要的免责声明。柏林当然没有这方面的记录，也没有当地成熟的投资者阶层，也没有很多当前的后起之秀。最近的记录大多充斥着美国网络模仿者。毫无疑问，伦敦现在有一个更先进、更复杂的创业生态系统。但这也是一个我们认为它将会得到的最大发展。另一方面，柏林就像一家初创公司一样，刚刚起步，充满活力和希望。

我们遇到的一个突出的公司叫做[研究门](https://web.archive.org/web/20230313100516/http://www.researchgate.net/)。在这个网站上，学者们可以相互联系，分享论文和成果，并搜索在网上其他地方找不到的非常具体的信息。这不像成为“学术界的脸书”那么简单，因为学术界只需要一个一年专门化几次的网络。相反，它的目标不仅是成为学术界的脸书，也是学术界的 LinkedIn，提供专业的求职服务，以及学术界的亚马逊，提供电子商务交易和对非常具体的实验室设备的评论。

它是由在哈佛做研究的德国出生的医生 Ijad Madisch 发起的，他意识到学术界只公布他们的成功。没有一个引擎来分享为什么 80%的实验会失败的知识。

Research gate 周一宣布其用户已超过 100 万，背后有知名投资者，包括 Index Ventures 的[西蒙·莱文](https://web.archive.org/web/20230313100516/http://www.crunchbase.com/person/simon-levene)、Benchmark 的[马特·科勒](https://web.archive.org/web/20230313100516/http://www.crunchbase.com/person/matt-cohler)和 Accel 的[安德鲁·布拉西亚](https://web.archive.org/web/20230313100516/http://www.crunchbase.com/person/andrew-braccia)。根据马迪士的说法，苛勒是建议 Research Gate 离开波士顿的人之一，那里的编码人才既昂贵又有限。他的两个建议是:柏林或硅谷。伦敦甚至不在名单上。事实上，当我们请马迪士比较这两个城市时，他说这是他第一次真正考虑这个问题。

就生态系统而言，柏林拥有一些大多数西欧国家首都所缺乏的东西。首先，在那里工作和生活很便宜。在文化和历史上，柏林是欧洲成熟的西方经济体和东欧不断涌现的人才之间更天然的门户。柏林有一个激增的创意阶层，主要由移民组成。它已经成为欧洲不合群的人——想成为艺术家和创作者的人，不适应伦敦等城市僵化的社会结构的人——蜂拥而至做他们想做的事情的地方。

这种敢于冒险、不合群的创意阶层并不总能催生出创新型的创业公司，但它是旧金山和特拉维夫创业生态系统发展的重要组成部分。这也是伦敦长期以来被批评为缺乏的东西。

如果伦敦感觉像欧洲的纽约；柏林给人的感觉就像欧洲的波特兰、奥斯汀或巨石。也许有一天会成为欧洲的旧金山。
