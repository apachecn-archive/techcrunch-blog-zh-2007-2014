# 美国有线电视新闻网在谈判购买 Zite iPad 应用程序？

Flipboard 式杂志风格的聚合应用现在非常流行。显然 Flipboard 有明确的空间，Pulse 也是一个不错的选择。但是一款聚合应用似乎引起了一家知名新闻出版物的注意——Zite。

同样是这家公司，曾经有一个充满来自像华盛顿邮报和美联社这样的恶劣出版物的[停止通知](https://web.archive.org/web/20230203095428/http://moconews.net/article/419-zite-has-a-new-ceo-and-a-response-to-cease-desist-demands/)，现在可能走向一个大的支付。据加拿大新闻网站 [Techvibes](https://web.archive.org/web/20230203095428/http://www.techvibes.com/blog/vancouvers-zite-to-be-acquired-by-cnn-for-20-25-million-2011-08-22) 报道，Zite 正在与 CNN 谈判以 2000 万到 2500 万美元的价格出售公司。

不幸的是，我们没有听到太多关于这笔交易的其他消息，两家公司都没有透露一点消息。如果它通过了，那将肯定是齐特人生命中的一个重大时刻。在过去的几年里，该公司处理了几个小问题:最明显的是马克·约翰逊新任首席执行官 Worio 的变动，当然，还有不愉快的新闻网站对 Zite 用户界面的大规模打压。与 CNN 的 2500 万美元的交易应该会让这些磕磕绊绊变得值得。
