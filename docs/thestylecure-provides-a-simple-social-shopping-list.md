# StyleCure 提供了一个简单的社交购物清单

前 MySpace 副总裁 Ryan Sit 试图重现人们与朋友在商场购物时的社交活动。当然，网上购物是另一回事，但随着脸书越来越成为一个电子商务目的地，社交购物可能会起飞。Sit 推出了自己的购物中心，名为 [TheStyleCure。](https://web.archive.org/web/20230204113506/http://thestylecure.com/)

这家初创公司通过 ShopStyle 的 API 在线列出了 300 家商店的 100 多万种产品和 7，000 个品牌。你可以搜索品牌、衣服、鞋子、配饰等等，你可以喜欢任何产品，这些都将通过脸书连接共享。你还可以看到你喜欢的所有衣服的简介，以及你朋友喜欢的所有衣服。

这个网站非常简单(Sit 在不到 30 天的时间里建立了 StyleCure ),但它确实增加了一个社交购物清单，不像你喜欢的衣服和配饰都在一个地方(而不是分散在网上)。你也可以点击购买这些产品。

虽然 StyleCure 还需要一些额外的东西，还需要继续增加库存，但它确实提供了一个集中的平台，让你可以和朋友分享你最喜欢的衣服。如果 Sit 可以扩展，这可能是一个将社交和你的愿望清单结合起来的好方法。也就是说，直到脸书创建自己的购物中心。
