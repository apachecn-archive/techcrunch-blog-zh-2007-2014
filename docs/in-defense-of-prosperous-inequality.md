# 捍卫繁荣的不平等

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/05/30/in-defense-of-prosperous-inequality/>

无人驾驶汽车将[挽救每年因汽车事故而失去的大约 32，000 条生命](https://web.archive.org/web/20230330114234/http://mobile.bloomberg.com/news/2011-12-08/u-s-highway-deaths-decline-2-9-falling-for-fifth-year-1-),减少污染，并节省我们在交通中浪费的无数时间。它也将消灭出租车行业——就像它消灭汽船行业一样。

给大众带来曾经被认为是奢侈品的服务(个人助理、前沿医学和世界一流的教育)，意味着将一些不存在的工作自动化。

许多对互联网经济的猛烈抨击，[最著名的是来自*纽约客*的乔治·帕克(George Packer)将硅谷的劳动力描绘成自由主义的乌托邦人，他们对高薪照片分享应用程序设计师大军正在压榨旧金山中产阶级的事实视而不见。](https://web.archive.org/web/20230330114234/http://www.newyorker.com/online/blogs/georgepacker/2013/05/silicon-valley-taxes-and-libertarianism.html)

在一篇引人入胜且不偏不倚的文章中，帕克指责硅谷创始人未能回答“为什么在个人电脑和互联网出现的几十年里，美国经济增长如此缓慢，平均工资停滞不前，中产阶级被掏空，不平等现象激增？”

Packer 的 A-list 访谈没有一个答案的原因是因为技术本身创造了不平等。“在过去二十年中，发达国家和发展中国家不平等性的明显上升在很大程度上可归因于技术变革的影响，”[Florence jau motte、Subir Lall 和 Chris Papageorgiou 在国际货币基金组织的一份工作文件中解释道](https://web.archive.org/web/20230330114234/http://blogs.wsj.com/economics/2008/07/24/technology-not-globalization-feeds-income-inequality/)(这[总体上证实了](https://web.archive.org/web/20230330114234/http://www.nytimes.com/2013/01/13/sunday-review/americas-productivity-climbs-but-wages-stagnate.html)为什么经济学家发现生产率稳步提高而工资停滞不前)。

然而，与几十年前相比，生活在某些方面正在变得更好:[癌症存活率翻了一番](https://web.archive.org/web/20230330114234/http://www.dailymail.co.uk/health/article-455028/Cancer-survival-rates-double-30-years.html)，休闲时间[增加了](https://web.archive.org/web/20230330114234/http://www.economist.com/node/5476124) 4-8 小时，女性可以自由地将更多时间花在职业上，[部分归功于家务琐事](https://web.archive.org/web/20230330114234/http://www.sciencedaily.com/releases/2009/03/090312150735.htm)(伦巴万岁！).

帕克从未解释过硅谷如何以其他方式运作，但行业工会确实有一个阻止不平等的计划:阻止创新。

工人工会 AFL-CIO [正在游说阻止高技能移民](https://web.archive.org/web/20230330114234/https://techcrunch.com/2013/05/17/union-tech-industry-is-greedy-for-wanting-to-end-hiring-wait-period-for-immigrants/)与美国人竞争，出租车工会想要阻止[优步提供汽车接送](https://web.archive.org/web/20230330114234/https://techcrunch.com/2012/07/15/two-victories-in-one-week-the-internet-flash-lobby-becomes-a-political-force/)，教师工会的目标是阻止[廉价在线大学课程的传播](https://web.archive.org/web/20230330114234/http://chronicle.com/article/As-MOOC-Debate-Simmers-at-San/139147/)。

在许多情况下，技术创造的工作机会比它摧毁的多，尤其是对那些第三世界的人来说。

政府在提供安全网和职业培训方面发挥着重要作用，但间接指责创新者并不是一个有效的信息。

但是，金融平等从来不是，也永远不会是互联网经济的目标。目标是*财富*创造。对于那些现在可以在世界任何地方接触到世界级教授和医疗的人来说，硅谷实现了它的承诺。
