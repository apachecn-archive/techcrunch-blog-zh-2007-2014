# 里尔登商务走向移动技术

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/05/20/rearden-commerce-goes-mobile/>

[![readen-mobile-2.png](img/969202fd5539200c36e270e1721b592b.png)](https://web.archive.org/web/20221005235416/http://www.crunchbase.com/company/reardencommerce)

不陪你上路的旅行助理有什么好？里尔登商务公司(Rearden Commerce)的企业旅行应用程序的一个漏洞是缺乏移动版本。现在它有了，从黑莓开始。(毕竟，里尔登是一项针对企业出差族的服务)。今年晚些时候，面向 iPhone 和普通网络浏览器的应用程序即将问世。一旦 Android 手机上市，它也可以轻松地将其移动应用移植到这些手机上。有趣的是，由于市场份额下降，该公司对开发 Windows Mobile 应用程序持观望态度。

Rearden 与您的公司旅行档案相关联，因此它知道您的完整行程——航班、酒店、租车、晚餐预订、篮球比赛门票。每当您的旅程发生变化(航班延误、登机口变更、新酒店)时，这些变化都会同步到手机应用程序，您也可以在飞机上离线查看。因此，如果你使用黑莓*和*手机，你的公司使用里尔登(很快摩根大通的所有客户也可以使用)，你需要的一切都在你的手机里。它还能告诉你你所在地方的天气，给你任何电话号码的点击呼叫链接，并让你通过黑莓电子邮件客户端发送任何行程段的电子邮件。

里尔登正在为未来开发的功能包括让你对餐馆进行评级，并查看同事或 Zagats 或 OpenTable 如何对这些餐馆进行评级，这样你就可以看到附近哪些餐馆得到了很高的评级。它还在开发对机票或活动门票的支持，以条形码图像的形式发送到你的手机上。

凭借刚刚筹集的 1 亿美元资金，里尔登有能力大举进军移动市场。现在，那个 iPhone 应用在哪里？
