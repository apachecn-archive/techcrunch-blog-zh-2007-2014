# 针。干草堆。发现:Splunk 在为 2012 年的首次公开募股做准备的过程中继续增长

你可能不知道 [Splunk](https://web.archive.org/web/20230130101858/http://www.splunk.com/) ，但你应该知道。该公司实时索引和搜索来自任何应用程序、服务器或网络设备的数据，包括日志、配置文件、消息和警报；正在飞速发展，并着眼于 2012 年的首次公开募股。

2010 年第三季度，Splunk 实现了 96%的同比增长，新增了 255 家客户，其中包括 104 家美国以外的客户，使该公司的付费客户总数超过 2000 家。客户包括瑞士瑞信银行、LinkedIn、T-Mobile、Swisscom、Shutterfly、Heroku 以及美国劳工部和能源部。

该公司现金流为正且盈利，2009 年收入为 4000 万美元，较 2008 年增长 125%。Splunk 预计 2010 年收入将超过 6000 万美元，并计划于 2012 年上市。

那么，Splunk 为什么会增长呢？因为该公司使 IT 管理员的生活变得更加轻松。例如，您可以在索引的机器数据中搜索 503 错误代码，而不是手动遍历所有服务器数据来查找错误代码。

云提供商也是 Splunk 快速增长的客户群。Heroku、GoodData、SendGrid、
Netsmart Technologies 和 Genius.com 都在使用 Splunk 来管理他们复杂的云环境。客户端还可以监控分布在多个数据中心的数千台服务器上的分布式部署；管理云计算平台即服务(PaaS)的基础设施；监控云交付的 SaaS 解决方案的性能，并监控混合 SaaS/托管模式。
