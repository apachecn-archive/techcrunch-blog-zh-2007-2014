# YouTube 扼杀了我们的视频下载工具 TechCrunch

在过去的几年里，我们提供了一个非常基本的[工具](https://web.archive.org/web/20230214225228/https://techcrunch.com/get-youtube-movie/)，用于下载 YouTube 视频，允许用户输入任何视频的 URL，立即获得一个可下载文件的链接。该工具做的事情与其他几十个网站和插件完全一样，这些网站和插件的出现是因为 YouTube 长期以来拒绝通过自己的网站提供类似的功能。现在，鉴于[昨天的新闻](https://web.archive.org/web/20230214225228/https://techcrunch.com/2009/02/12/youtube-hopes-to-boost-revenue-with-video-downloads/)称该网站将开始支持少量视频的下载，YouTube 显然已经决定打击这些工具——看起来他们在拿我们做例子。

在过去的 24 小时里，我们的工具停止了工作，但我试过的其他方法都工作得很好，包括一些其他网站和流行的 Firefox 扩展。有可能 YouTube 只是禁用了我们碰巧使用的创建视频文件链接的某种技术，但更有可能的是，他们挑出了我们(我们的工具已经变得非常受欢迎，当你在谷歌上搜索“[下载 YouTube](https://web.archive.org/web/20230214225228/http://www.google.com/search?hl=en&safe=off&client=firefox-a&rls=org.mozilla%3Aen-US%3Aofficial&hs=MPR&q=download+youtube&btnG=Search&cts=) ”)时，排名第二)。

当被问及为什么我们的工具被禁用时，YouTube 的一位发言人向我们介绍了该网站服务条款的这一部分(早在 2006 年，该公司就曾向我们发出过类似的停止通知):

> 第五节。您对网站内容的使用
> 
> b 部分。您可以通过 YouTube 网站提供的功能访问用户提交的内容，仅供您参考和个人使用。除非您在 YouTube 网站上看到 YouTube 为该用户提交的内容显示的“下载”或类似链接，否则您不得复制或下载任何用户提交的内容。

这一举动并不特别令人惊讶。YouTube 的新下载功能——更重要的是，它可以对下载收取费用——不会成为一个可行的摇钱树，除非该公司能够遏制这些未经授权的下载。但是，如果这是一种趋势的开始，那就为时过早。虽然你可以在 YouTube 上下载一些视频，但绝大多数仍然没有提供这种选择，用户可能很快就会别无选择。

即使 YouTube 可以消除所有这些工具，下载也可能太少、太晚:大多数人已经使用 iTunes、亚马逊或网飞进行视频下载，虽然 YouTube 可能会获得大量流量，但尚不清楚是否有许多人愿意真正开始为散布在网站上的简短剪辑支付费用。
