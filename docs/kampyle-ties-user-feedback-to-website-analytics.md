# Kampyle 将用户反馈与网站分析联系起来

[![](img/dcf374009ecad5bffb06ac9334643203.png)](https://web.archive.org/web/20221007062330/https://beta.techcrunch.com/wp-content/uploads/2008/07/kampyle_shot1.png)

[Kampyle](https://web.archive.org/web/20221007062330/http://www.kampyle.com/) 已经将其[用户反馈管理平台](https://web.archive.org/web/20221007062330/http://www.beta.techcrunch.com/2008/03/1/kampyle-closes-the-feedback-loop/)整合到 [Google Analytics](https://web.archive.org/web/20221007062330/http://www.google.com/analytics) 和 [Nuconomy](https://web.archive.org/web/20221007062330/http://www.nuconomy.com/) 中，以提供用户反馈和网站分析的并行分析。

自去年 3 月推出其平台以来，Kampyle 已被 1，500 多个网站采用，这些网站的用户总共产生了超过 100，000 条反馈信息。该公司声称，小型网站平均每天有 1 到 5 条反馈信息，中型网站平均有 20-30 条，大型网站平均有 50-1000 条。Kampyle 向这些大型网站展示了最大的价值，这些网站必须有效地管理他们的反馈，否则就会淹没在建议的海洋中。

Kampyle 很自然地从用户反馈中获得了整合到网站分析中的想法。这家初创公司发现，用户通常会同时打开两个浏览器标签:一个用于谷歌分析，另一个用于自己的仪表板。同时跟踪分析和反馈使网站所有者能够更密切地关注使用模式，并快速识别技术和可用性问题。这种整合也有助于 [A/B 测试](https://web.archive.org/web/20221007062330/http://en.wikipedia.org/wiki/A/B_testing)和对网站其他变化的测量，比如添加新的页面或部分，以及它们如何影响用户行为。

[![](img/7b869c668205bdfa9d504028b45c74bd.png)](https://web.archive.org/web/20221007062330/https://beta.techcrunch.com/wp-content/uploads/2008/07/kampyle_shot2.png)

Kampyle 已经通过 greasemonkey 脚本集成到 Google Analytics 中。发行商必须安装 Firefox 插件才能启用该功能(目前不支持其他浏览器)。与下一代网络分析平台 Nuconomy 的整合要“干净”得多，因为两家公司都在 T2 的 Yossi Vardi 的 T3 投资组合中。这两家公司开发了一个私人 API，使用户能够查看与特定页面相关的反馈信息，然后跳转到 Nuconomy 中分析相同页面的参与度和互动分析。

除了这些整合，Kampyle 还在其反馈表中添加了 60 种语言翻译。用英语创建的表单现在可以立即用法语、德语或任何其他可用语言部署。类别、子类别、颜色、图标和反馈按钮样式的定制程度更高。
