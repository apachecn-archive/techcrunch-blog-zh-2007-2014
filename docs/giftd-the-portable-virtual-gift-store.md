# Giftd:便携式虚拟礼品店

![](img/7b0dad620b47e14ec74ac1321f55a971.png)

虚拟礼物在像脸书这样的社交网络上相当流行，它提供了数百种礼物，其中大部分售价为 1 美元(其中一些是免费的)。但是这些礼物通常都与他们购买的网络有关，这意味着你不能在不同的社交网络或你的博客上展示同一个礼品盒。 [Viximo](https://web.archive.org/web/20221007094326/http://www.viximo.com/) 是一家为网站提供虚拟商品和礼物交钥匙解决方案的初创公司，它发布了一个面向消费者的网站，名为 [Giftd](https://web.archive.org/web/20221007094326/http://www.giftd.com/) ，允许用户创建虚拟礼品盒，几乎可以嵌入到网络上的任何地方。

该网站提供了一个装满数字礼物的商店，其中许多都有完整的动画、声音效果和一定程度的用户互动(例如，点击六包啤酒会让你“大口”喝下饮料)。购买礼物需要 Giftd 积分，15 个积分兑换 3 美元左右，大多数礼物需要 2-5 个积分。

礼物显示在可嵌入的 Flash 小部件中，这些小部件可以放在大多数博客、主页(比如 iGoogle 和 Netvibes)和社交网络(包括 MySpace 和 Multiply)中。最大的例外是脸书，因为它不允许嵌入式闪存。

那么，Giftd 会流行起来吗？可移动虚拟礼品盒的概念并不新鲜——其他初创公司也开发了类似的产品。真正的区别在于内容，这也是 Giftd 可能擅长的地方——它的礼物具有很高的生产价值，这可以归功于 Viximo 不断壮大的艺术家群体。但是，即使有了高质量的图标，Giftd 仍将很难与社交网络上越来越常见的本土礼品店竞争(T5【Hi5】T6 刚刚推出了一家，而我的空间也在酝酿中)。
