# 忘记易贝吧。使用 Swaptree。

[![](img/7ffd8e353d0a3634a57b9c8446ba785f.png)](https://web.archive.org/web/20221006010556/http://www.swaptree.com/) 自 2006 年年中以来，我们一直在报道波士顿 [Swaptree](https://web.archive.org/web/20221006010556/http://www.swaptree.com/) 。该公司正在创建一个交换书籍、CD、DVD 和视频游戏的用户网络，刚刚获得了第三轮融资——来自安全科学公司的 330 万美元。三轮筹资总额现已达到 560 万美元。

为了创建一个交易，公司会找到多达四个用户，他们拥有并想要匹配的商品。然后，每个人都从网站上打印出地址表，并支付运费(通常为 2.50 美元)，直接邮寄给下一个人。

最酷的功能之一是能够输入一些你愿意交易的物品——向你展示所有你可以收到的物品。

这是一款由最近收购的 Peerflix 公司[尝试过的车型，后来](https://web.archive.org/web/20221006010556/http://www.beta.techcrunch.com/2008/09/05/liveuniverse-buys-another-loser-peerflix/)[放弃了](https://web.archive.org/web/20221006010556/http://www.beta.techcrunch.com/2006/12/19/peerflix-swaps-one-problem-for-another/)。不过，Peerflix 只从事 DVD 交易，并收取交易费。Swaptree 不收取交易费，涵盖书籍、音乐、电影和游戏的群体交易方式意味着更多的匹配。

该公司在一年多前首次推出了，并表示迄今为止已经完成了超过 10 万笔交易。
