# Color Labs 首席产品官 DJ Patil 辞职

我们已经证实，Color Labs 首席产品官 [DJ Patil](https://web.archive.org/web/20230124020957/http://www.crunchbase.com/person/dj-patil) 将很快以全职员工的身份离开公司。帕蒂尔在 3 月份之前一直是 LinkedIn 的首席科学家，他只与 Color 共事了大约 5 个月。他将转行成为公司的顾问。

[一个月前失去了](https://web.archive.org/web/20230124020957/https://techcrunch.com/2011/06/14/troubled-startup-color-loses-cofounder-peter-pham/)联合创始人[彼得·范](https://web.archive.org/web/20230124020957/http://www.crunchbase.com/person/peter-pham)。这是这家苦苦挣扎的公司第二次高调亏损。
