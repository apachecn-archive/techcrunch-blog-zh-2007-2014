go Try On筹集了300万美元，为大众带来了Gap和丝芙兰的私人造型师

去试试吧，一家允许用户分享自己的照片并获得对自己长相的意见的初创公司，已经从SPA Investments、Index Ventures和其他公司获得了300万美元的资金。

在Go Try On上，你可以上传一张自己的照片，添加你所穿品牌的描述，并包括服装选择的背景(如音乐会、假日聚会)。该网站的社区可以在网站上发表评论，并对其他成员的服装提供反馈。用户可以选择在社区上询问更大的Go Try It，在脸书上与他们的朋友分享，或者向某个品牌寻求建议。该网站的 iPhone应用允许你从应用访问你的长相和评论，也可以直接从应用上传和分享长相。您还可以突出显示您的位置，并与您的通讯录共享您的外观。

这家初创公司还推出了一个专业的个人造型师网络，允许成员上传衣柜和化妆造型，并接受Gap和丝芙兰的专家作为“个人造型师”的建议。

在未来，尝试给会员提供个性化的着装分析，有点像基于服装标准的谷歌风格和时尚分析。该公司计划通过允许品牌根据这些结果提供定制建议来创收。
