# 出于某种原因，LinkedIn 上的每个人都喜欢称自己“有责任感”

“创造性”LinkedIn 个人资料的统治似乎已经结束。

该公司发布了 2013 年最常被滥用的流行语，基于在专业社交网站的个人资料中出现频率最高的词。2011 年和 2012 年，用得最多的词是“创意”，但今年“创意”被“负责”打败了。一方面，这比把自己描述成有创造力的人更容易弄巧成拙。另一方面，它不仅仅是一点点…没有启发性。

就我个人而言，我觉得用谄媚的形容词来提升你的 LinkedIn 个人资料的想法有点奇怪，但话说回来，[我的个人资料](https://web.archive.org/web/20221208091227/http://www.linkedin.com/in/anthonyha)说，“我写东西”，所以我知道什么呢？然而，在 LinkedIn 自己的新闻稿中，该公司的“职业专家”妮可·威廉姆斯表示，在你的个人资料中包含流行语并不是一个好主意:“如果你听起来像其他人一样，你就不会在竞争机会的其他专业人士中脱颖而出。”

如果你正惊恐地盯着下面的流行语列表，因为它基本上与你的整个 LinkedIn 个人资料中的*相同，那么，该公司有一些建议。除此之外，它说你应该让其他用户为你担保(鉴于我收到的随机请求和推荐，我也不太喜欢这个功能，但是，嗯，让我们继续吧……)，包括实际工作或结果，表明你有创造力或(bleh)负责任，并且措辞更加具体，以便更好地适应你感兴趣的工作、公司和行业。*
