# Salesfusion 收购营销分析平台 LoopFuse

营销自动化平台 [Salesfusion](https://web.archive.org/web/20230326082531/http://www.salesfusion.com/) 收购了 [LoopFuse](https://web.archive.org/web/20230326082531/http://www.loopfuse.com/) ，这是一个帮助客户找到有前途的新线索的分析平台，价格未披露。此次收购发生在 [Salesfusion 完成 825 万美元的 B 轮融资](https://web.archive.org/web/20230326082531/http://venturebeat.com/2014/01/16/marketing-automation-stays-hot-with-funding-for-salesfusion/)后两周，使其迄今筹集的资金总额[达到 1010 万美元](https://web.archive.org/web/20230326082531/http://www.crunchbase.com/company/salesfusion)。

正如《福布斯》的 Ben Kepes 指出的那样，CRM 领域最近有很多并购活动。例如，Salesforce.com 最近扩展了它的现金资源，[获得了一笔 3 亿美元的贷款](https://web.archive.org/web/20230326082531/https://techcrunch.com/2013/07/15/salesforce-com-takes-300m-loan-for-exacttarget-acquisition-cash-resources-getting-tight/)来支付它对 ExactTarget 的收购，而[甲骨文最近的收购包括 Compendium 和 Eloqua](https://web.archive.org/web/20230326082531/https://techcrunch.com/2013/10/17/oracle-buys-compendium-a-content-marketing-startup-to-build-up-its-arsenal-against-frenemy-salesforce/) 。

Kepes 指出，微软自己没有强大的营销自动化工具。通过寻找收购目标，该领域的公司可能会反过来使自己成为 Redmond 有吸引力的收购目标。

即使 Salesfusion 没有试图为自己定位 M&A，LoopFuse 也会帮助它扩大客户名单。首席执行官 Christian Nahas 告诉我，收购 LoopFuse 将使 Salesfusion 在 2014 年的收入翻一番，新客户增长三倍。该公司表示，2013 年它实现了强劲增长:收入翻了一番，息税摊销前收益翻了三番(EBITA)；每月新增经常性净收入和新增客户增加两倍。

纳哈斯表示，该公司被 LoopFuse 的人才和知识产权所吸引，其中包括“利用社交智能和大数据分析做一些令人兴奋和强大的事情，以帮助营销人员用优质线索填充他们的漏斗。”

纳哈斯说，两家公司都专注于中型企业，但也吸引了小型企业和企业客户。收购 LoopFuse 意味着客户现在将能够获得 Salesfusion 面向中端市场的营销自动化平台和 LoopFuse 的社交智能和预测分析工具的组合。这意味着 Salesfusion 的客户现在可以超越他们的客户关系管理(CRM)工具，找到与现有销售线索相似的新线索。
