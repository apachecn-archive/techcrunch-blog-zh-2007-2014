# Fantastical:我最喜欢的新日历应用

我最近一直在使用一款非常酷的应用，名为 Fantastical。这是 OS X 独有的，但我怀疑 Windows 人群也会从中得到乐趣。

基本上它是一个自然语言日历应用程序。你可以粘贴(几乎)任何东西到它的小窗口，它会自动创建一个事件。“周四下午 5 点与乔共进晚餐”就像你想象的那样，除非今天*是*周四，否则 Fantastical 会把事情搞清楚。它有时会在词序上出错，最后会用“星期二”来设置一整天的午餐，但大多数情况下，敲几下键盘就能修复。

该应用程序确实复制了 OS X 邮件的一些功能，尤其是自动日期输入功能，但我发现我用它来记录航班时间，方法是从航空公司网站上复制并粘贴它们，我还能创建特别的事件(“下午 5 点和 T-Pain 喝酒”)，甚至简单的提醒(“明天给达赖喇嘛打电话”)。

该公司由校友[迈克尔·西蒙斯](https://web.archive.org/web/20230205041412/https://twitter.com/#!/macguitar)和[肯特·萨瑟兰](https://web.archive.org/web/20230205041412/https://twitter.com/#!/ksuther)创立，这款应用自 5 月份就已问世。很快会有一个新的更新，将启用内嵌日历条目编辑。这款应用售价 20 美元，可以免费试用。

这两个人构建了他们自己的 CalDAV 实现，因此它几乎可以与任何日历服务一起工作。它还具有自定义报警设置以及完整的搜索功能。同样，这是以前就存在的东西，但现在它在一个地方，有一个快速简单的界面。它和 QuicKeys 等应用程序一起，极大地简化了我的日常个人信息管理和写作活动。我们不会在 TC 上评论很多应用程序(随着时间的推移，我们可能会开始添加应用程序评论)，但这是一个我非常喜欢的应用程序，我认为它非常简单，非常有用。
