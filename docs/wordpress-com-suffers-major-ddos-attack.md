# WordPress.com 遭受史上最大规模的 DDoS 攻击

你不知道发这个帖子有多难，因为我们的博客主持人[WordPress.com](https://web.archive.org/web/20221207103652/http://wordpress.com/)目前正遭受拒绝服务攻击。过去 10 分钟几乎无法访问 TechCrunch 后端(现在一切似乎都很稳定)，用户一直收到*“对服务的写入已被禁用，我们将尽快使一切恢复在线”*错误消息。

来自 VIP 博客帖子:

> WordPress.com 目前正成为一个非常大的分布式拒绝服务攻击的目标，在某些情况下影响连接。攻击的规模是每秒几千兆，每秒几千万个包。
> 
> 我们正在努力减轻攻击，但由于规模过大，这被证明是相当困难的。此时，随着攻击的平息，一切都应该恢复正常，但我们正在积极地与上游提供商合作，采取措施防止此类攻击影响未来的连接。
> 
> 我们将优先考虑我们的 VIP 网站，一如既往，您可以通过 xxxxx@wordpress.com 联系我们了解最新进展。我们也将更新这篇文章，提供更多的信息

WordPress 没有提到攻击的来源(DDoS =！匿名)和我联系了创始人马特·莫楞威格了解更多信息。WordPress.com 目前为 1800 万出版商提供服务，包括 VIPs TED、CBS 和 TechCrunch，并负责全球 10%的网站。WordPress.com 本身每月有大约 3 亿次独立访问。

**[更新:](https://web.archive.org/web/20221207103652/https://beta.techcrunch.com/2011/03/03/wordpress-com-suffers-major-ddos-attack/)** Automattic 和 WordPress 创始人马特·莫楞威格告诉我们，这是 WordPress 有史以来最大规模的攻击，而且很可能是出于政治动机:

*“有一个持续的 DDoS 攻击，其规模足以影响我们在芝加哥、圣安东尼奥和达拉斯的所有三个数据中心，目前已被制服，但它可能会在稍后再次爆发，我们正在采取主动措施实施。*

*这是我们 6 年历史中遇到的规模最大、持续时间最长的攻击。我们怀疑这可能是出于政治动机，针对我们的一个非英语博客，但我们仍在调查，还没有确切的证据。”*
