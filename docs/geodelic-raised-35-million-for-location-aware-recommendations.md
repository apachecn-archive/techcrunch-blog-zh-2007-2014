# Geodelic 为位置感知推荐筹集了 350 万美元

![](img/df7e9191a3503adbba3484bf408d440e.png)

在手机上，你越能自动搜索，人们就越有可能使用它。或者至少这是指导 [Geodelic Systems](https://web.archive.org/web/20221208232314/http://www.geodelic.com/) 的原则，这是一家为手机创造“无搜索”体验的初创公司。今天，在一份新闻稿中，它透露，它在早期一轮融资中筹集了 350 万美元，可能是在 2008 年，来自 Clearstone Ventures(它是在那里孵化的)和 Shasta Ventures。该公司由 Rahul Sonnad 创建，他之前创建了网络视频发布服务平台，并于 2006 年将其出售给康卡斯特。

Geodelic 正在为餐馆、电影、商店、航班、酒店和当地景点创建一个位置感知搜索引擎，根据它们与您的距离推荐结果。“位置转盘”会在地图上按类别显示附近的结果，它会从你的行为中学习你最喜欢的地方、商店和品牌，并据此锁定你。该应用程序被设计为尽可能被动，消除或最大限度地减少所需的打字量。然而，它没有像 Layar 或[Sekai Camera](https://web.archive.org/web/20221208232314/http://www.beta.techcrunch.com/2009/02/17/tagging-the-real-world-sekai-camera-for-the-iphone-is-alive-and-very-cool/)等一些增强现实标记应用[走得那么远，这些应用通过手机的摄像头在视图上添加了一个数据层。](https://web.archive.org/web/20221208232314/http://www.beta.techcrunch.com/2009/06/21/layars-augmented-reality-browser-literally-more-than-meets-the-eye/)

T-Mobile 的下一代 Android 手机 myTouch 3G 将采用 Geodelic [开发的名为 Sherpa](https://web.archive.org/web/20221208232314/http://www.beta.techcrunch.com/2009/06/21/t-mobile-will-drop-its-second-android-phone-the-mytouch-3g-this-august-for-199/) 的应用。而且，从其网站上所有的 iPhone 截图来看(我在下面贴了两张)，一个 iPhone 应用程序也在开发中。
