# 教室里的 Skype:教师的国际社交网络

![](img/974cd76ec82904c5e60ee3e7881e2858.png)

[Skype](https://web.archive.org/web/20230204231518/http://www.crunchbase.com/company/skype) 充分意识到其软件被全球许多学校教师和学生使用，今天宣布它已经建立了一个专用的社交网络，帮助他们通过网络联系、协作和交流知识和教学资源。

今天早上，该公司[推出了一个名为](https://web.archive.org/web/20230204231518/http://blogs.skype.com/en/2011/03/skype_is_connecting_teachers_a.html) [Skype in the Classroom](https://web.archive.org/web/20230204231518/http://education.skype.com/) 的免费国际社区网站，这是一个在线平台，旨在帮助教师[根据他们所教的年龄组、地点和感兴趣的科目等搜索标准找到彼此](https://web.archive.org/web/20230204231518/http://education.skype.com/users)和[相关项目](https://web.archive.org/web/20230204231518/http://education.skype.com/projects)。

该平台自 12 月底以来一直处于测试阶段，已经拥有超过 4000 名教师的社区，遍布 99 个国家。

教师只需在网站上注册他们的 Skype 帐户，创建一个包含他们的兴趣、地点和所教年龄组的档案，并通过浏览目录开始与其他教师联系，他们还可以在目录中找到与他们的技能、需求或兴趣相匹配的项目和资源。

Skype in the Classroom 是一个只对成员开放的社区，教师可以轻松地将彼此添加到他们的 Skype 联系人列表中或相互发送消息。

这是一个很棒的想法，我真诚地希望它能够成功(不会造成伤害，而是有希望激发许多现有的教师社交网络和协作网络)。
