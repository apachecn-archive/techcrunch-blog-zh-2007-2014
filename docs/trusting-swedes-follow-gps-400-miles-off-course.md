# 信任的瑞典人跟踪 GPS 偏离航线 400 英里

于是，一对游客想开车去美丽的卡普里岛，无意中在他们的导航系统中输入了“Carpi ”,并跟着它走了整整 400 英里，才发现可能出了什么问题。是什么给了他们线索？卡普里岛是一个岛屿，他们的路线从未穿过一片水域？实际上，当*到达卡皮时，他们问在哪里可以找到[蓝色洞穴](https://web.archive.org/web/20221007191849/http://en.wikipedia.org/wiki/Blue_Grotto)。*

这很傻，但也表明了一种越来越普遍的态度，那就是我们的设备是不会出错的。这导致我们更依赖于他们而不是我们自己，并在这条路上走得更远，导致我们不再相信自己去做曾经是第二天性的事情。我们刚刚在 Boing Boing Gadgets 上看到一篇关于这个主题的文章。那边的丽莎觉得她的全球定位系统对她的大脑有害，而且确实如此。一圈一圈的导航让你的注意力偏离了道路，而且不是以一种好的方式。我们的其他设备呢？当然，他们正在实现一种完全前所未有的分享和交流水平，我认为总的来说这是一件好事——但是他们(可以说)把我们的思想拴在他们身上，这很少是一件好事。这有点像通过一个接一个地搜索所有的东西来做一个纵横字谜。

我想我应该为这种事情制定一个规则。是的，我现在就要弹出一个:

> 利用技术来扩展你的能力，但永远不要取代它们。

这似乎很合理。所以他们可以做我用 GPS 做的事情:找到一个位置，仔细检查，并写下如果我不知道该区域我可能会错过的部分。一个接一个地被诅咒。
