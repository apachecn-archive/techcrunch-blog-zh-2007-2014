# 在 Blackhawk 和 GroupCard 的帮助下，苹果将 iTunes 礼品卡带给其脸书粉丝

今天早些时候，iTunes 脸书粉丝页面[推出了一个漂亮的新功能:你现在可以创建定制的 iTunes 礼品卡，并通过脸书或电子邮件直接发送给朋友。这本身就很不错，但有趣的是这个应用程序部分是由](https://web.archive.org/web/20221006192138/http://www.facebook.com/itunes) [GroupCard](https://web.archive.org/web/20221006192138/http://www.groupcard.com/) 构建的，这是一家让朋友和同事共同创建贺卡的初创公司。GroupCard 是 2008 年冬季 fbFund 的[获得者](https://web.archive.org/web/20221006192138/http://www.beta.techcrunch.com/2008/12/09/facebooks-fbfund-names-winners-of-225000-grants/)，后来参加了 [fbFund REV](https://web.archive.org/web/20221006192138/http://www.beta.techcrunch.com/2009/09/01/live-from-fbfund-revs-demo-day/) 孵化器项目。更新:该应用程序的前端 UI 由 GroupCard 构建，后端由美国最大的第三方礼品卡提供商 Blackhawk Network 处理。

使用 iTunes Gifts 脸书应用程序非常简单:你从六个不同的模板中选择一个，其中包括假期、生日和一些无处不在的彩色 iPod 彩色舞者的设计。然后你选择你想要的礼物数量(从 5 美元到 50 美元不等)以及你脸书朋友的名字或电子邮件地址作为收件人。最后，你可以用自己的个人信息定制卡片。完成后，您可以选择立即发送，或者指定发送日期(这非常适合假日/生日购物)。

这似乎是购买数字 iTunes 礼品卡的唯一方式之一——苹果网站本身只提供实体版本，Gifts.com 和 Giftcards.com 等网站也没有。除了这个脸书应用程序，我们唯一能看到这个选项的地方就是 iTunes 桌面应用程序本身。

GroupCard 构建了应用程序前端，但它们实际上并不在其中。那是因为礼品卡的处理者是礼品卡商城(隶属于[黑鹰网](https://web.archive.org/web/20221006192138/http://www.blackhawknetwork.com/)，专营预付卡)。苹果、黑鹰和 GroupCard 联手推出了这项新功能。**更新** : Blackhawk 通过其 API 为应用程序的后端供电。该公司与苹果公司有着长期的合作关系，这就是为什么苹果公司选择它来支持 iTunes 页面上的礼品卡应用程序。

同样值得注意的是，GroupCard 是脸书测试新功能的首选应用之一——今年早些时候，它率先在 T4 测试脸书支付。GroupCard 最终有可能允许用户使用脸书积分购买 iTunes 歌曲，这显然是一件大事。GroupCard 首席执行官约翰·安德松表示，目前还没有任何这样做的计划，除此之外他无可奉告。但他表示，GroupCard 期待未来与其他品牌合作。**更新**:鉴于 GroupCard 是应用程序的前端，而不是交易，这在不久的将来似乎不太可能，至少在 iTunes 歌曲的情况下。

Anderson 还说，这项服务每月获得 25 万个 uniques，并增长了 300%。
