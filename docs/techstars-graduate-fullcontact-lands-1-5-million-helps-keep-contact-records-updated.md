# TechStars 毕业生 FullContact 获得 150 万美元，帮助更新联系记录

[FullContact](https://web.archive.org/web/20230204201906/http://fullcontact.com/) ，一家[刚刚从商业加速器 tech stars](https://web.archive.org/web/20230204201906/https://techcrunch.com/2011/08/04/startup-accelerator-techstars-hatches-12-new-companies-in-boulder-heres-a-peek/)毕业的初创公司，已经从 500 家初创公司、Tango / High Country Venture、BulletTime Ventures、SK Ventures、Zelkova Ventures 和天使投资人那里筹集了[150 万美元](https://web.archive.org/web/20230204201906/http://www.crunchbase.com/company/fullcontact)。

FullContact 提供了一个 API，软件开发人员可以在他们的服务和应用程序中使用它来为用户提供完整和最新的联系信息。

基本上，FullContact 允许开发人员利用其超过 1 亿条联系人记录的数据库，将部分过时的联系人记录转化为完整的联系人记录。

信息包括电话号码、电子邮件地址、邮政地址、姓名、职务、公司、社交资料 URL、图像、生日、即时消息句柄等。

这家初创公司采用众包模式来更新联系信息，只分享在网络上公开可用的联系信息(社交网络、其他 API 等)。)，并允许任何软件应用程序通过其 API 利用其联系人数据库。目前，超过 750 名开发人员使用他们的 API。

这家初创公司表示，他们的技术不能用于广告网络、广告交易或把 cookies 与联系数据联系起来的服务。

FullContact 由巴特·洛朗、特拉维斯·托德和丹·林恩于 2010 年创立。
