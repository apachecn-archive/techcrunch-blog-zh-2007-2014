# 主板会很快提供板载存储吗？

![](img/5e59496dcbbb456bbfaf10f73e394219.png "IP35-Pro-Big")
很长一段时间，使用一块主板的板载资源被认为是……不文明的。但现在，你可以在集成显卡上运行体面的 3D，声卡越来越不必要，因为主板提供了出色的内置环绕声。[存储](https://web.archive.org/web/20221203092646/http://www.crunchgear.com/tag/storage/)还会远吗？嗯，是的，但也许不久之后你至少会在中拥有自己的系统分区[。](https://web.archive.org/web/20221203092646/http://news.cnet.com/8301-30685_3-20014032-264.html)

小型化的冒险使 SanDisk 创造了一些极小的“iSSD”模块，可以很容易地放在主板上，并立即提供高达 64GB 的存储空间。非常适合作为系统驱动器，这样您就可以将所有媒体和游戏存储在外置驱动器甚至网络驱动器上。

显然，这种解决方案已经可以在手机和一些平板电脑上使用，但将其转移到台式机和笔记本电脑上可以节省大量空间，并简化布线和电源安排。速度也相当不错:160MB/s 的读取和 100MB/s 的写入(顺序)，比过去几十年我们安装操作系统的 3.5 英寸硬盘要好。

当然，这将提高主板的价格，但这可能是比手机或完全集成的内置固态硬盘中的 NAND 闪存更便宜的解决方案。不过，SanDisk 很可能仍在向 mobo 制造商推销这款手机，所以我暂时不会这么认为。
