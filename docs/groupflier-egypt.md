# 受#Egypt 的启发，GroupFlier 推出了策划短信群

![](img/7df2f96a809d940d0f54ca5550501598.png)就像 [1000Memories](https://web.archive.org/web/20230202235747/http://www.1000memories.com/) 和它的[网上纪念](https://web.archive.org/web/20230202235747/https://techcrunch.com/2011/02/09/1000memories/)埃及抗议者一样，群发短信应用 [GroupFlier](https://web.archive.org/web/20230202235747/http://www.groupflier.com/) 意识到它的用户正在呼吁一个#埃及相关的功能，特别是一个提供策划的 Twitter 和其他与反穆巴拉克抗议相关的内容的功能。

GroupFlier 创始人、哈佛网络安全专家莫里斯·彭纳认为这是一个好主意。在你的手机上关注一个特定问题的推文是超级嘈杂的，无论你是通过短信(尽管我不知道谁会这么做)还是使用 iPhone 应用程序。Penner 让他的一名员工建立了一个名为[“Egypt 4 peace”](https://web.archive.org/web/20230202235747/http://groupflier.com/join/Egypt4Peace)的小组，该小组关注活动家兼谷歌高管瓦伊尔·高尼姆以及现场记者丹·诺兰和沙里夫·库杜斯发布的推文。

要加入“Egypt4Peace ”,您可以访问[http://groupflier.com/join/Egypt4Peace](https://web.archive.org/web/20230202235747/http://groupflier.com/join/Egypt4Peace)或发送短信“Egypt4Peace”至(617) 394-1386。要创建自己的策划小组，只需到 http://groupflier.com[建立一个小组，命名并开始推送内容。其他用户可以通过发短信或访问网址来跟进。若要离开群组，只需发送短信“*再见”](https://web.archive.org/web/20230202235747/http://groupflier.com/)

策划短信内容可以非常有用。例如，在穆巴拉克今天的电视讲话中，福克斯和赫芬顿邮报等新闻媒体报道穆巴拉克将下台，而穆巴拉克本人则用华丽的辞藻表达了他不会下台的声明。事实上，我自己也不太确定，直到我收到相反的*“穆巴拉克没有下台…人群愤怒地在空中挥舞鞋子”*来自埃及 4Peace 的短信。

差不多可以概括了。

Penner 的未来计划是建立多个跨不同垂直行业的策划小组，如本地体育，他希望电话积分网站 Twilio 最终能够获得国际支持，这样国外的用户也可以通过电话跟踪策划的新闻。

虽然这一功能仍处于早期阶段，但 Penner 充满希望:“新工具将使人们能够放大人类行为的力量，尽管没有人知道具体有多大或以何种方式放大。”
