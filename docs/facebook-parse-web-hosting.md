# 脸书最近收购 Parse，为开发者的网站提供托管服务

[解析](https://web.archive.org/web/20221006210147/https://www.parse.com/)，移动后端[创业公司，脸书最近收购](https://web.archive.org/web/20221006210147/https://beta.techcrunch.com/2013/04/25/facebook-parse/)建立一个新的以开发者为中心的业务，刚刚[推出托管](https://web.archive.org/web/20221006210147/https://www.parse.com/products/hosting)。它旨在帮助拥有桌面网站或网络伙伴体验的移动开发者。这笔收购已经给 Parse 带来了提振，自交易宣布以来，它托管的应用数量增加了 33%，达到 8 万个。

“人们使用 Parse 构建移动应用。但当他们想要一个网站或. com 登陆页面时，他们使用 Parse 登录，但网站是由 Heroku 或 App Engine 等其他东西提供的，”Parse 联合创始人伊利亚·苏哈尔解释说。"所以我们推出了一个功能齐全的虚拟主机平台."

Sukhar 说，这个项目已经进行了四到六周，甚至在脸书谈判进行的时候。

新的托管服务允许开发人员托管登录页面，并显示使用 Parse API 检索的用户数据。比方说，如果一个开发者想在网上显示他们游戏的排行榜，他们可以使用新的托管服务和标准的 Parse data 产品来实现。

![Screen Shot 2013-05-07 at 10.21.29 AM](img/bc7c693f9af3dac03d8040d85c5a1425.png)

Parse Hosting 是基于其他产品之上的，这些产品帮助移动开发者管理推送通知、用户身份和登录。

他补充说，我们独立听说的脸书交易价值 8500 万美元，不包括留用奖励，[没有吓跑开发者](https://web.archive.org/web/20221006210147/https://beta.techcrunch.com/2013/04/27/parse-facebook/)。他们现在有 80，000 个应用程序，而在脸书交易宣布时，他们说他们有 60，000 个应用程序。“关于人们是否会放弃 Parse 有一场有趣的辩论，但我们所有的指标都在上升，”他说。

脸书赢得了收购 Parse 的交易，尽管许多硅谷最知名的公司，如苹果、雅虎和 Dropbox，都表示出了兴趣。通过收购 Parse，他们开始了第一个企业对企业的收入流，并且像 Instagram 一样，承诺团队有相当大的自主权来开发他们认为合适的产品。他们没有篡改 [Parse 基于 SaaS 的收入模式](https://web.archive.org/web/20221006210147/http://www.crunchbase.com/company/parse)。

他还说，公司还没有庆祝这笔交易。“我们有很多事情要做，”他说。
