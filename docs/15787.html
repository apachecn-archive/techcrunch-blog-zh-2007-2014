TechMedia Network收购Bestofmedia Group，加快内容商务整合

原文：https://web.archive.org/web/https://techcrunch.com/2013/07/02/techmedia-network-buys-bestofmedia-group-and-steps-up-content-commerce-integration/

数字技术和科学出版商TechMedia Network 昨天宣布，它以未披露的金额收购了 Bestofmedia Group ，为其名册增加了三份新的技术出版物。

此次收购提供了一个利用TechMedia Network的商业功能和Bestofmedia的社区品牌的机会，明确了TechMedia Network对数字媒体的定义是垂直媒体。今天早上，我给TechMedia Network的首席执行官格雷格·梅森(Greg Mason)打了一个电话，以了解这笔交易的更多细节。

TechMedia的电子商务网站TopTenREVIEWS目前每月推动100万次购买，它希望在销售与Bestofmedia的网站、Tom的硬件、Tom的指南和Tom的IT Pro整合后，这一速度将会跃升。目标是将内容、广告、社区和购买推荐整合到一个赚钱的用户体验中。

问题是如何将汤姆的读者转化为销售。在像TopTenREVIEWS这样的网站上，当读者已经在寻求购买时，将内容与商业配对，比当用户不一定有购买的想法时，将用户从内容转移到购买要容易得多。

TechMedia Network的解决方案非常简单:使用TopTenREVIEWS。

汤姆的网站目前从事商业活动，通过链接到商家。梅森表示，新收购的网站可能会从产品评论和专题报道到TopTenREVIEWS的评论平台建立更直接的联系，该平台已经有效地将消费者转化为销售。

想法是，如果你正在阅读一篇关于身份盗窃的文章，你可能会有兴趣采取措施来保护你的身份。那是他们把你链接到TopTenREVIEWS的地方。

因为科技新闻本质上是基于产品的，这个计划比其他新闻网站更有机会。但这是假设一个阅读专题报道的人仅仅通过进入TopTenREVIEWS环境就会转变为购买心态。

梅森说，展望未来，TechMedia Network正在寻求进入英国市场。Bestofmedia的重心在欧洲，但在英国几乎没有业务；TechMedia Network那里没有。他们还计划为Tom's Guide和Tom's IT Pro开发编辑人员和内容，因为Tom的硬件目前是Bestofmedia的主要流量驱动因素。

梅森不愿透露具体数字，但他说TechMedia Network和Bestofmedia都是盈利的。除了广告和商业收入，TechMedia还向包括雅虎、TechCrunch的母公司美国在线、MSNBC和福克斯在内的媒体网站提供内容。

作为对比，TechMedia Network(和AOL)的竞争对手Vox Media一段时间以来一直在建立自己的垂直整合。发布SBNation、Verge和Polygon博客的Vox于4月宣布在TC Disrupt成立一个新的公司部门，作为内部广告服务。因为它有自己的发布平台Chorus，Vox能够使其广告与内容紧密结合，并跟踪每个活动。

从梅森所说的将商业整合到汤姆的网站中来看，购买点似乎不会过多地扰乱阅读体验。Bestofmedia给TechMedia Network带来了一大批活跃的社区成员，可以从中赚钱。在接下来的几个月里，我们将看到这两家公司如何有效地利用彼此。
