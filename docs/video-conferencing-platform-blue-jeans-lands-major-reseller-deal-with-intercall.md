# 视频会议平台 Blue Jeans 与 InterCall 达成主要经销商交易

[Blue Jeans Network](https://web.archive.org/web/20221006103655/http://bluejeans.com/) 是一个跨多种平台集成的企业视频会议解决方案，它宣布与最大的音频会议和协作服务供应商之一 Intercall 建立重要的经销商合作关系。

Blue Jeans Network 被称为“任何(软件)视频会议”，它将思科/Tandberg、Polycom、Lifesize 的会议技术与 Skype 和 Google Talk 等消费者平台连接在一起。远程工作人员、旅行者和远程办公人员只要能够使用 Skype 和支持视频的台式机或移动设备，就可以在传统的企业视频会议设备上与同事参加视频会议。

Blue Jeans Network 的每个用户在 Blue Jeans cloud 中都有一个私人“会议室”,他们可以用它来安排、主持和管理会议。与会者只需拨一个号码或点击他们系统中的一个链接就可以参加会议，无论是 Skype 还是 Tandberg。

通过与 Blue Jeans Network 的合作，InterCall 将向其客户转售 Blue Jeans 的产品。该公司表示，虽然 Blue Jeans 与德国电信(Deutsche Telecom)等其他提供商有转售协议，但这是迄今为止最大的合作。

目前，已经筹集了[2350 万美元](https://web.archive.org/web/20221006103655/https://beta.techcrunch.com/2011/06/29/blue-jeans-nabs-23-5m-for-the-video-conferencing-platform-to-rule-them-all/)资金的蓝色牛仔裤网络称，InterCall 在 2011 年从客户那里获得了 200 亿分钟的音频，占全球 800 亿分钟音频总量的 25%。
