# 为什么大媒体要对 DMCA 科技博客发起核攻击

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/06/big-media-nuclear-dmca/>

***编者按:**以下客座博文由[Ashkan Karbasfrooshan](https://web.archive.org/web/20230203064503/http://www.crunchbase.com/person/ashkan-karbasfrooshan)[WatchMojo](https://web.archive.org/web/20230203064503/http://www.watchmojo.com/index.php?id=1)创始人兼 CEO 撰写。*

当国会在 1998 年更新版权法并通过数字千年版权法(DMCA)时，它开启了一个投资、创新和创造就业的时代。在此后的十年里，谷歌、YouTube 和 Twitter 等公司因该法案而崛起，但在这个过程中，它们扰乱了传统媒体公司的商业模式和收入流。如今，TMCs 正试图在众议院和国会快速通过几项法案，以扭转这一切。

通过他们在华盛顿特区的说客，媒体公司正试图通过两项新法案改写 DMCA。内容行业的游说者在没有任何来自技术行业的投入的情况下稳步前进，参议院的游说者被称为保护知识产权，众议院的游说者被称为电子寄生虫。电子寄生虫法将扼杀 DMCA 的避风港，并允许传统媒体公司通过切断新兴技术公司的交易和收款能力来攻击它们，如果你愿意，这有点像维基解密发生的事情。这会让风投不敢投资这类科技公司，这反过来会破坏就业机会的创造。

可以理解的是，科技行业对其影响感到震惊，这包括版权所有者发布的任何网站的自动黑名单，这将延伸到支付提供商甚至搜索引擎。这是怎么回事，我们到底是怎么到这里的？

什么是 DMCA，什么是安全港？

当国会在 1998 年通过数字千年版权法(DMCA)时，它更新了版权法，提供了四个安全港，包括对“根据用户的指示驻留在系统或网络上的信息”免受侵犯版权的法律保护 DMCA 建立了一个重要的平衡，如果在线服务提供商在收到通知后删除内容，他们可以免除责任。

在这样做的时候，DMCA 基本上允许用户生成的网站增长和繁荣，当用户上传侵权内容时，保护它们免受传统媒体公司的不公平要求和过度诉讼。

**为什么媒体公司对 DMCA 不满意**

DMCA 将识别侵权内容的责任推给了 TMC，例如 NBC 环球必须通知 YouTube 有人上传了《懒惰的星期天》的片段，并要求他们将其删除。只要 YouTube 删除了有争议的视频，就没有人受到伤害，尽管有人认为这一系列事件实际上伤害了 TMCs。

**为什么媒体公司要以先发制人的方式发展核能**

愤世嫉俗者会认为，TMC 本质上是在通过不同的渠道应用与科技公司相同的策略。换句话说，当风险资本家资助企业家编写旨在“颠覆陈旧行业”的代码时(引用肖恩·帕克 LinkedIn 个人资料)，没有人反对传统内容公司被要求“输入”

显然，情况并不完全相同:对于下一代互联网初创公司来说，这些法案将影响整个行业(如果不是整个经济的话)，而当风险投资者投资一家公司时，这是一种更有限的行为，即使这家初创公司有可能像 Napster 或 YouTube 那样“改变世界”。

再者，新兴公司颠覆 TMC 的事实是进化，是适者生存的表现。虽然有些人会认为 TMC 依赖律师，而科技公司在市场上竞争，但事实是许多科技公司躲在 DMCA 后面购买时间，这进一步挫败了 TMC。

TMC 变得“积极主动”的另一个原因是，无论是通过撤下通知，还是通过随后的诉讼，追捕侵权者都需要大量资源。在某些情况下，最严厉的手段正被胡萝卜所取代。但当你考虑到维亚康姆对 YouTube 的诉讼“太少太晚”时，也许 TMC 正在追求这种先发制人、严厉的先发制人战略，以使他们瞄准的科技公司更愿意合作。

事实上，既然 TMC 已经表现出发展核能的意愿，他们希望风投和科技公司能够更倾向于按照他们的条件与 TMC 合作。

**法案对创业、创造就业的影响**

投资者 Fred Wilson 将[的注意力吸引到了这两项新法案上，他辩称“这些法案是由内容产业起草的，没有任何来自技术产业的意见。他们试图在不与科技行业进行任何协商的情况下，通过国会快速将其纳入法律。”他补充说，“最后一次谈判产生了一个极好的妥协，经受住了时间的考验，并允许像谷歌、脸书、YouTube 和 Twitter 这样的重要新服务被创建，并成为大公司和大量就业机会的创造者。”](https://web.archive.org/web/20230203064503/http://www.avc.com/a_vc/2011/10/protecting-the-safe-harbors-of-the-dmca-and-protecting-jobs.html)

他是对的。没有人怀疑这些法案会终结我们所知的互联网。过去 20 年来，科技公司创造的就业机会也可能远远超过 TMCs 流失的就业机会。

但公平地说，如果 TMC 没有暴跳如雷，那么或许科技公司和支持它们的风投就不会如此在意重启对话，听取 TMC 的愿望清单。举个例子，Wilson 先生在他的文章中伸出了橄榄枝:“如果另一个谈判是为了修改 DMCA，那就让我们来做吧。”

有句话说，请求原谅比请求允许更容易；这总结了多年来科技公司的一些想法。现在可以说，TMC 并不是在请求修改法律的许可，而是希望他们的先发制人的罢工能让他们在尘埃落定时请求原谅。

双方都受到贪婪和恐惧的驱使，但如果 TMC 如愿以偿，赶走了 DMCA，那么即将到来的不确定性可能会回来困扰他们。如果需要的话，技术行业将会适应，谁知道这对媒体行业意味着什么。毕竟，认识的魔鬼总比不认识的好。
