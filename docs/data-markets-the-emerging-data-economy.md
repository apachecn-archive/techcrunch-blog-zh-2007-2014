# 数据市场:新兴的数据经济

**编者按:** *吉尔·厄尔巴兹(Gil Elbaz)是一位企业家，也是自然语言技术的先驱。1998 年，他共同创立了应用语义学，该公司开发了上下文广告产品，包括 ASI 的 AdSense。2003 年，谷歌收购了 ASI，在谷歌工作了四年后，吉尔在 2007 年发现了 [Factual](https://web.archive.org/web/20221206114006/http://www.factual.com/) 。在推特上关注他。*

数据市场这个术语让人想起一种传统的结构，在这种结构中，供应商出售数据来赚钱。事实上，随着大大小小的公司纷纷加入，这种形式的市场正在兴起。想想 [Azure 数据市场](https://web.archive.org/web/20221206114006/https://datamarket.azure.com/)(微软)[data.com](https://web.archive.org/web/20221206114006/http://data.com/)(sales force . com)[InfoChimps.com](https://web.archive.org/web/20221206114006/http://www.infochimps.com/)和[DataMarket.com](https://web.archive.org/web/20221206114006/http://datamarket.com/)。

虽然这种模型允许组织获取有价值的数据，但该术语正在演变为包括多种形式，每种形式都有不同程度的采用成功。在 it 的核心，数据市场使组织能够以新的方式访问数据，其中货币不仅必须是金钱，还可以是数据或洞察力的形式。

还有一种趋势是，公司可以将数据管理的某些方面，尤其是关于参考或规范数据集的方面，外包给专门收集和管理数据集或以其他方式从数据中创造价值的第三方。因此，新的数据经济正在形成，数据可以以更简单、更经济的方式创建、访问、租赁和永久维护。

互联网允许快速收集和交换数据的能力，以及能够搜索数据并将其准确传递到所需位置的 API，推动了新形式的数据市场。

考虑下面的例子:

*   Jigsaw 创造了一个数据市场，在这个市场中，个人和组织在一个中央存储库中提供联系信息。Jigsaw 对这些数据进行筛选，然后部分和全部分发，以换取数据和金钱。
*   Kaggle 允许公司向数据科学家社区提供数据，这些数据科学家分析数据，发现预测性的、可操作的见解，并赢得奖励。数据和回报是用来换创新的。

数据市场的出现让企业开始质疑对数据普遍存在的“不在这里发明”的态度。如果第三方可以创建和组装有价值的数据，为什么不租用而不是拥有呢？如果你不拥有数据，你也不必维护数据。

数据市场也在改变人们对数据作为一种必须保密的资产的态度。虽然有些数据显然永远是专有的，但在许多情况下，最大的价值来自共享数据，并从中获得某种新的价值。

数据市场新参与者的关键问题包括:

*   您的数据在您的组织中有什么价值？
*   分享有什么风险？
*   你对数据有什么控制权？
*   你能换来什么？
*   你应该在数据市场中扮演什么角色？

现代数据市场将采用全新一代的技术、流程和数据科学，取代上一代的数据管理系统。其中包括:

*   **云计算:**首先，云计算正在被广泛采用。集群可以立即启动，无需交付时间，并且可以根据需要进行扩展，以应对需求的意外增长。虽然在公共云中管理最大的数据集可能成本高昂，但相同的核心技术可用于管理私有云，从而提供大量管理和成本优势。
*   **大数据软件:**Hadoop 开源项目获得了令人难以置信的发展势头，成为许多从海量数据中提取价值的新的大规模努力的核心。像 Microsoft 和 EMC/Greenplum 这样的老牌软件公司以及 Cloudera 和 Hortonworks 这样的新兴公司都在加班加点地利用高级管理、云和支持产品为 Hadoop 堆栈增加价值。
*   **数据科学和机器学习:**在整理大量数据以找到模式和关系时，预测建模和机器学习正在成为标准工具集的一部分。自然语言处理和统计技术可用于发现非结构化数据中的关系。随着数据量的急剧增长，所有这些技术都至关重要。
*   **API:**API 是一种粘合剂，使应用程序能够实时集成大型数据库的适当部分或复杂的第三方数据处理能力。API 还支持以小块或大块的形式收集数据，以便中央监管工作流能够保持最大的数据新鲜度。
*   **众包和社交流程:**正如 Twitter 让人们能够以新的方式联系和交流，数据市场可以使用众包和其他受社交媒体启发的方法来创造新的分享形式。

新的数据市场模型仍在不断发展，并被商业界所接受，但我预测在未来几年内，它将成为访问和管理数据的事实标准。
