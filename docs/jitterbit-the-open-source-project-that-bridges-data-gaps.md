# Jitterbit:一个弥合数据鸿沟的开源项目

[![](img/26309fa710647de40a2d7ef42669d95d.png)](https://web.archive.org/web/20221006234950/http://www.crunchbase.com/company/jitterbit)

帮助企业连接和整合不同来源数据的服务公司 Jitterbit 发布了其软件的新 2.0 版本和新的定价模式，希望这将使其对企业和消费者更具吸引力。

Jitterbit 是一个开源项目，它允许用户有效地使用和修改来自通常不直接通信的多个来源的数据。虽然该公司的部分目标是企业市场(在处理 Salesforce 等 SaaS 应用程序时经常处理此类问题)，但它也努力吸引较小规模的消费者业务。

该软件的新版本允许用户利用拖放界面来生成功能，这些功能将从一个输入中收集数据，并在其他地方修改数据，而不必输入任何代码。例如，我可以使用 Jitterbit 来收集易贝上 iPhone 的平均价格，然后自动调整我自己的网上商店的价格，使其比这个数字低 2%，以保持我的在线店面的竞争力。

![](img/210cab2854e0eb51375db845c178007e.png)

新版本的发布恰逢该公司货币化战略的转变。在过去，Jitterbit 试图为开源项目的标准版本和“前沿”夜间版本提供支持，这些版本经常容易出现错误，因为它们没有经过彻底的测试。从现在开始，该公司将不再支持这些夜间构建，而是提供一个付费的“坚如磐石”版本，其中包括支持(喜欢冒险的用户仍然可以免费下载未经测试的版本)。

这个领域有很多类似的服务，包括 [Talend](https://web.archive.org/web/20221006234950/http://www.talend.com/) 和 [SnapLogic](https://web.archive.org/web/20221006234950/http://www.snaplogic.com/) 。
