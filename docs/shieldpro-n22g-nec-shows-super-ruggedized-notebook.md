# ShieldPRO N22G: NEC 展示超耐用笔记本电脑

市场上所有这些加固型计算机对您来说还不够坚固吗？那么你可能想看看今天 NEC [为日本市场发布的 ShieldPRO N22G。它本质上是一台 Windows 笔记本，但专门设计用于恶劣环境。](https://web.archive.org/web/20230204193935/http://www.nec.co.jp/press/ja/1109/2001.html)

根据 NEC 的说法，温度从-20 到 50 摄氏度，灰尘或沙子(“完全保护”)，从 90 厘米落到混凝土上，或水(由 6.3 毫米的喷嘴喷射)都不是问题。对于极端情况下的使用，该公司承诺 12 小时的电池寿命。

就规格而言，买家可以预期 12.1 英寸触摸屏(背光液晶显示器，分辨率为 1，024×768)，双核 i7-660UE CPU (1.33GHz)，4GB 内存，英特尔 QM57 芯片组，SD 卡/记忆棒 Duo 插槽，三个 USB 端口，IEEE 802.11a/b/g/n Wi-Fi，蓝牙 2.1+EDR，千兆以太网和双语(英语/日语)键盘。

ShieldPRO N22G 的尺寸为 290×255×47 毫米，重 2.7 千克，将采用开放式价格模式销售(取决于您选择 XP 还是 Windows 7，取决于您需要多大的硬盘或固态硬盘以及其他额外因素)。目前还没有关于国际发行的消息，但是之前的型号 [N22A](https://web.archive.org/web/20230204193935/http://www.nec.com/global/prod/indpc/notebook/n22a/index.html) 实际上已经在国外发行了。
