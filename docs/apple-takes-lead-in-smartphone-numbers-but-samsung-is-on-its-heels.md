# 苹果在智能手机出货量方面领先，但三星紧随其后

根据 IDC 的一份报告,[苹果](https://web.archive.org/web/20230203060424/https://techcrunch.com/tag/apple)的智能手机出货量超过 Q2 任何其他制造商，从苦苦挣扎的[诺基亚](https://web.archive.org/web/20230203060424/https://techcrunch.com/tag/nokia)手中抢走了吹嘘的资本。苹果出货量为 2030 万部，市场份额为 19.1%，同比增长 141.7%。三星和诺基亚紧随其后，RIM[和 HTC](https://web.archive.org/web/20230203060424/https://techcrunch.com/tag/rim) 紧随其后。

值得注意的是，HTC 本季度发布了创纪录的数字，同比增长 166 %,声称市场份额从上个季度的 8.9%上升到 11.7%。HTC 的轰动和 Evo 3D 与 HTC 在中国日益突出的地位有很大关系。尽管该公司排名第五，但它似乎仍在蚕食 RIM 和诺基亚的大部分份额。但是 HTC 并不是唯一的一家——三星也吃了一大口。

事实上，三星经历了惊人的一年，同比增长 380.6%。这一成功在很大程度上要归功于三星 Galaxy S II，它在上市后的头 55 天里售出了 300 万台。如果他们能保持这样的增长再久一点，他们将轻松超越苹果。

现在是坏消息。RIM 本季度的智能手机出货量比 2010 年同期多了 100 多万部——如果智能手机市场本身没有出现更大的增长，本季度的总出货量达到 1.065 亿部，这将是一个可观的增长。因此，虽然 RIM 公司确实卖出了更多的手机，但他们实际上失去了大量的市场份额。

现在是真正的坏消息。不出所料，与去年相比，诺基亚本季度的销量和市场份额都有所下降。出货量下降 30%后，诺基亚现在只控制了 15.7%的市场份额。显然，从 MeeGo 过渡到 Windows Phone 7 与此有相当大的关系。随着 MeeGo 的消亡，消费者开始选择全新的手机，而不是等待 Windows 操作系统的诺基亚手机。更糟糕的是，等待不会很快结束，因为美国不在诺基亚的[名单上获得第一部 Windows 操作系统手机的国家之列。](https://web.archive.org/web/20230203060424/https://techcrunch.com/2011/06/17/nokia-to-launch-windows-phone-7-handsets-in-just-6-european-nations-at-first/)

苹果和[安卓](https://web.archive.org/web/20230203060424/https://techcrunch.com/tag/android)吃诺基亚和 RIM 的午餐并不是什么新闻，但是看到实际数字总是很有趣。根据 comScore 的最新数据，截至 6 月，Android 已经占据了 40%的市场份额。碰巧的是，增长最快的制造商——HTC 和三星——也恰好主要运行安卓系统。如果 Android 能继续以现在的速度增长，三星和 HTC 也会紧随其后。
