# MakerBot 的佩蒂斯:“我们将会看到大量的硬件创业公司。”

克里斯·迪克森(Chris Dixon)在《创始人故事》的最后一站接受了 [Bre Pettis](https://web.archive.org/web/20230320160534/http://www.crunchbase.com/person/bre-pettis) 的采访，两人谈到了这样一个事实，即创始人口吐白沫创办硬件创业公司的门槛正在被清除。

佩蒂斯说:“我们正在看到这种创业文化进入硬件领域，而通常只有真正的大公司才有能力进行这种 R&D 和开发，我认为在未来十年内，我们将会看到大量的硬件创业公司。”

谈到各个创业领域的相似之处，迪克森说:“过去(创办一家创业公司)需要花费 3000 万美元，你必须运营一个数据中心并做所有这些其他事情，现在你只需要几千美元就可以做到，我认为这在某种程度上创造了互联网创业公司的复兴。这也让一类新的没有超级技术的人可以做一些事情。”

正如上面的对话所证明的那样，进入游戏变得越来越容易——如果你正在寻求这样做，佩蒂斯提供了以下建议:“让人们可以用你的硬件做你意想不到的事情，留下机会，尝试并兼容未来。”

最后，两人将聚光灯对准了真正的 [MakerBot](https://web.archive.org/web/20230320160534/http://www.crunchbase.com/company/makerbot) ，并从其内部取出一艘刚刚完成的火箭船——同时还讨论了蠢朋克乐队的好运护身符。
