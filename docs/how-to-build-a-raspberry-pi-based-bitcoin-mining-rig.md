# 如何构建基于 Raspberry Pi 的比特币挖矿钻机

虽然你不会因为运行一个 Raspberry Pi 比特币平台而变得富有，但你或许可以在这个精巧的小机器上收回你的初始投资。RaspPi 比特币平台并不新鲜，但戴夫·康罗伊刚刚建造了一个，更重要的是，在一个相当易读的页面上分享了他的计划。

该系统使用 Pi 作为主处理器和一个 [330MH/s ASIC](https://web.archive.org/web/20221007014857/http://www.daveconroy.com/beginners-guide-to-turn-your-raspberry-pi-into-an-affordable-bitcoin-mining-rig/) 。他总共花了 127 美元在他的装备上，据我们所知，电力支出可以忽略不计。

该系统使用一个名为 [MinePeon](https://web.archive.org/web/20221007014857/http://mineforeman.com/minepeon/) 的以 ARM 为中心的采矿操作系统。它基本上是 Arch Linux ARM 的一个版本，内置了一些挖掘应用。那么你需要的只是一个比特币钱包和一点点时间。假设你将以最高 330 英里/秒的速度采矿，你可能一天只能看到几便士，如果那样的话。如果你对比特币完全陌生，你可以在这里了解更多关于如何设置所有账户和钱包的信息[。然而，我认为真正的重点是这套工具的新颖性。](https://web.archive.org/web/20221007014857/https://beta.techcrunch.com/2013/04/08/how-to-mine-bitcoins/)

看到这个[小装备运行](https://web.archive.org/web/20221007014857/http://www.daveconroy.com/beginners-guide-to-turn-your-raspberry-pi-into-an-affordable-bitcoin-mining-rig/)很酷，我认为它上面的小 USB 供电风扇是点睛之笔。把几十个这样的东西放在一个房间里，你可能真的会造成一些破坏，你的农场会看起来有点像 Pikmin，这是一个额外的奖励。
