# 我不认为我们会在美国看到这些 Galaxy Tab 智能外壳

虽然我还没准备好指控三星的犯罪裙带关系(正如 [9 to 5 Mac](https://web.archive.org/web/20230203093540/http://9to5mac.com/2011/07/18/family-ties-earn-this-smart-cover-knock-off-a-samsung-certification-and-a-place-on-their-store-shelves/) 和 [Gizmodo](https://web.archive.org/web/20230203093540/http://gizmodo.com/5822404/samsung-now-selling-apple-smart-cover-knock+offs-for-their-ipad-tablet-knock+offs/) 兴高采烈地做的那样，但我不得不说，在高风险的专利诉讼期间，将他们的品牌借给这样一个*难以置信的*公然山寨产品似乎有点不明智。

显然，这款用于 [Galaxy Tab 10.1](https://web.archive.org/web/20230203093540/https://techcrunch.com/tag/galaxy-tab/) 的[“智能外壳”](https://web.archive.org/web/20230203093540/http://www.anymode.net/goods/view.php?goodsno=12346934)正在韩国各地的三星商店出售，而美国专利法可能无法触及这些地方。但是苹果会忽略在如此大的市场申请专利吗？

无论如何，这是无耻的(这些案件可能是因为三星内部的一些个人关系而被批准的)，我认为这不会被三星批准很久。如今，山寨产品是科技世界的一个正常部分，但用货架空间来美化它们却并非如此。
