# FitBark 又一次证明了它的狗健康追踪器并没有发疯

厌倦了量化你自己吗？为什么不量化你的宠物呢？FitBark 是一款 Fitbit 风格的健康追踪器，适用于你正在行走的狗狗。在之前，我们已经报道过这个(坦率地说)疯狂的小发明[，早在五月，当它的创造者在](https://web.archive.org/web/20230327025301/https://techcrunch.com/2013/05/01/the-fitbark-pet-activity-monitor-is-a-reasonable-device-for-pet-owners/) [TechCrunch Disrupt NY](https://web.archive.org/web/20230327025301/https://techcrunch.com/events/disrupt-ny-2013/) 的硬件巷展出时，但他们现在已经开始在 [Kickstarter](https://web.archive.org/web/20230327025301/http://www.kickstarter.com/projects/fitbark/fitbark-a-fitbit-for-dogs) 上筹集资金，将这个设备放到野外。又来了。

这实际上是 FitBark 第二次尝试启动这个小发明。正如 Gigaom 指出的那样，为了重新思考商业模式，它的创造者取消了早期的众筹尝试，取消了每月订阅费，选择了 Kickstarter 上 69 美元的固定价格或普通零售 99 美元。

FitBark 这一次的制造成本为 3.5 万美元，距离实现目标还有 32 天，已经过半——不管是否疯狂，这是一家硬件初创公司，至少在第一次制造活动中几乎是板上钉钉的。

现在我说吠叫疯了，但这主要是开玩笑，因为 FitBark 不是唯一一个为宠物主人的现金而钓鱼的健康追踪者。 [Whistle](https://web.archive.org/web/20230327025301/https://techcrunch.com/2013/06/05/on-a-mission-to-build-the-next-big-pet-brand-whistle-launches-a-99-fitbit-and-health-monitor-for-pooches/) ，一家获得 600 万美元首轮融资支持的初创公司，[上个月刚刚推出了一款售价 99 美元的可穿戴狗狗活动追踪器](https://web.archive.org/web/20230327025301/https://techcrunch.com/2013/06/05/on-a-mission-to-build-the-next-big-pet-brand-whistle-launches-a-99-fitbit-and-health-monitor-for-pooches/)。还有 [Tagg](https://web.archive.org/web/20230327025301/http://www.pettracker.com/) ，通过在其设备中包含 GPS 来结合活动和位置跟踪。因此，低估拥有宠物的美元，后果自负。

那么 FitBark 实际上是做什么的呢？将它挂在你的狗的项圈上，它可以跟踪每天的活动水平，当你的智能手机在范围内时，或者如果你购买了专用的 FitBark 基站，它可以全天将数据发送回 FitBack 的服务器(当你外出时，可以将你的宠物关在家里)。后一种情况将允许主人在他们不在家时远程监控他们宠物的活动水平，但除非你拥有一座豪宅(或雇佣一名遛狗者)，否则你的狗将无法在没有你的情况下到处跑。FitBark 然后处理所有的活动数据，提供可定制的每日活动目标，并通过应用程序将结果反馈给你。到目前为止，还算正常。

在其更加狂吠的边缘，FitBark 还让宠物主人比较——嗯，他们说“统一”——他们自己的健康与他们的狗的健康/活动。所以，是的，吹嘘你比菲多更健康显然是现在的事情…

> FitBark 也是第一个利用人类健身追踪器现有 API 的平台，为您带来您和您的狗的健身水平的统一视图。从一开始，FitBark 就会无缝接收来自 Nike Fuelband、Fitbit、Withings Pulse 或 Bodymedia Fit 的输入。当我们了解到新的开放 API 或合作机会时，我们会扩展列表。如果你不仅是一个忠实的狗父母，而且也认真跟踪自己的健康状况，你会喜欢这个。
