# Gist 收购 Startup Weekend 应用程序“了解这个名字”

每隔几周(有时甚至更频繁)，几十名技术人员聚集在一起参加地区性的[创业周末](startupweekend.org)——快节奏的代码编写狂潮，企业家和开发人员在不到 60 个小时的时间里构思并构建一个新的应用程序(并在此过程中失去不少睡眠)。许多应用程序在那之后很快就消亡了，但其中一些仍然存在。现在他们正成为收购目标:[Learn the Name](https://web.archive.org/web/20230203020845/http://www.learnthatname.com/)，一款利用你的 LinkedIn 联系人帮助你记住商业熟人名字的游戏，已经被 [Gist](https://web.archive.org/web/20230203020845/http://www.gist.com/) 收购。交易的条款没有被披露，但 LTN 的家伙说，他们对结果“非常，非常高兴”。

Learn That Name 是去年 8 月在微软赞助的初创公司周末活动中开发的，并赢得了最高奖项(这很有趣，因为它是为绝对非微软的 iPhone 开发的)。这款应用的创意来自律师埃里克·科斯特(Eric Koester)，他在活动中忘记了某人的名字，因此受到启发，创建了这款应用。一个 14 人的团队在那个周末一起开发了这个应用程序，从那以后，最初的 LTN 团队的一个子集继续致力于它，发布了更新的 iPhone 版本、Palm WebOS 应用程序和 Flash 应用程序。

这笔交易是为了 LTN 的科技资产，收益将在 14 名原始团队成员之间进行分配。展望未来，独立的 iPhone 和 Palm 应用程序仍将可用，游戏也集成到 Gist 自己的 iPhone 应用程序中，你可以在这里找到。Gist 版本将接入 Gist 的联系人数据库(原版使用 LinkedIn)。

对于那些没有使用过 Gist 的人来说，Gist 提供的服务可以帮助你密切关注你职业网络中的人。该服务的网络界面允许你查看每个联系人过去的信息和附件，关于他们公司的新闻，以及他们最近在 Twitter 等服务上的信息。Gist 还提供了一个 Outlook 插件，与 [Xobni](https://web.archive.org/web/20230203020845/http://www.xobni.com/) 有相似之处。鉴于学习这个名字的商业导向性质，这似乎是一个很好的(和有趣的)适合。

鉴于 Learn That Name 团队的成功，看看是否有更多的创业周末团队在活动结束后继续合作将是一件有趣的事情。
