# Bubble Motion 为手机推出类似 Twitter 的语音博客服务

![](img/1773a318fca7c7bd4ad57a4a37be433c.png)

红杉资本支持的语音信息公司 [Bubble Motion](https://web.archive.org/web/20221208211711/http://www.bubblemotion.com/) 今天进入了微博领域，这是基于语音的微博领域。这家初创公司推出了 [Bubbly，](https://web.archive.org/web/20221208211711/http://www.bubblemotion.com/products-bubbly.html)一个为手机打造的语音博客社交网络。

使用 Bubbly，您可以从任何移动设备录制和广播消息。当用户录制音频消息和更新时，追随者可以随时收听。要开始写语音博客，你需要在手机上输入一个短代码，然后开始记录你的信息。要关注另一个语音博客，用户可以拨打他们想关注的人的电话号码。每当有新的音频更新，关注者都会通过短信收到如何收听的指示。Bubblers 还可以控制谁可以听到他们的消息，并可以选择隐藏他们的实际电话号码，以防止追随者直接打电话给他们。通过与移动运营商的合作部署，Bubbly 是 [BubbleTalk、](https://web.archive.org/web/20221208211711/http://www.bubblemotion.com/products-bubbletalk.html)一种不需要任何呼叫的“点击、通话、发送”信息服务的延伸。

这个想法有点古怪，但已经在亚洲市场发布，过去一周在印度积累了 15 万用户。我可以看到，在移动设备使用率高于电脑上网使用率的市场上，Bubble 正在被使用。香槟酒可以让名人与粉丝保持距离。Bubbly 目前正在与印度的移动运营商合作，并与使用 Bubbly 语音博客的印度地区和国家名人合作。印度似乎是一个完美的泡沫市场，尤其是考虑到印度类似 Twitter 的短信微博网络的成功和快速增长。

总部位于加州山景城的 Bubble Motion 公司自 2003 年成立以来，已经筹集了总计 3500 万美元的资金。
