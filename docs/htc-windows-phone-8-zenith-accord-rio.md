# 据报道，HTC 有三款 Windows Phone 8 设备正在研发中

伙计，HTC 是不是雇佣了一些信口开河的人？

在微软盛大的 Windows Phone 8 预览活动后不到一天， [The Verge](https://web.archive.org/web/20221207223518/http://www.theverge.com/2012/6/21/3105474/htc-windows-phone-8-rio-accord-zenith) 设法获得了 HTC Windows Phone 8 硬件计划的一些细节。如果他们的消息来源属实，这家台湾公司计划在今年年底前发布一款顶级、中档和入门级 WP8 设备。

从顶部开始是“Zenith”，据报道，它配备了 4.7 英寸的 Super LCD 2 显示屏，支持 HSPA+ 42Mbps，800 万像素摄像头，以及一个未指定的高通四核处理器。与此同时，“雅阁”将占据中间位置，配备 4.3 英寸 Super LCD 2 显示屏、1GB 内存和高通双核处理器——据报道，它保留了与其哥哥相同的网络支持和摄像头模块。

最后但并非最不重要的是相对较小的“Rio”，据说是 HTC 的入门级 Windows Phone 8 设备。它配备了 4 英寸的 WVGA 显示屏，虽然预计将由高通的骁龙·S4 驱动，但 Rio 将只有 512MB 的内存，足以处理现有版本的 Windows Phone，但看看处理器/内存组合如何处理 WP8 将是有趣的。

不言而喻，你应该对所有这些持保留态度(相当大)，但考虑到 HTC One 系列的三层发布似乎做得很好，如果这一信息证明是正确的，也不会太令人震惊。幸运的是，暂定的发布时间表将里约和雅阁的发布时间定在 10 月份左右，天顶紧随其后，所以更多的泄露应该不会太久，要么证实要么打破这些说法。
