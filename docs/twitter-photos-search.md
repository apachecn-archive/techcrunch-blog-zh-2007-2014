# Twitter 在照片上与 Photobucket 合作，在搜索上与 Firefox 合作

Twitter 刚刚在他们的博客上宣布了两项新的关键功能:改进的搜索和照片。后者，我们显然在最近几天已经讨论过了，因为[我们几天前第一次透露了](https://web.archive.org/web/20230210170539/https://techcrunch.com/2011/05/30/twitter-is-launching-its-own-photosharing-service/)的消息。前者，我们提到过，也可能会出现。但是有一些细节我们不知道。

首先，今天带来了全新版本的 Twitter 搜索，它承诺“更相关”的推文，但也在右侧窗格中向您显示相关的图片和视频。更有趣的是，Twitter 已经与 Mozilla 合作开发了 Firefox 浏览器的新版本[和内置在搜索栏中的 Twitter。(还有一个针对当前火狐用户的插件](https://web.archive.org/web/20230210170539/https://twitter.com/#!/download/firefox/)[。)](https://web.archive.org/web/20230210170539/https://addons.mozilla.org/firefox/addon/twitter-address-bar-search/?src=external-blog)

Notes Twitter:

> 如果你在 Twitter 上使用新版 Firefox，你可以直接在地址栏中输入#hashtag 或@username，直接进入搜索结果页面(试试#idol)或某人的个人资料页面(比如@nba)。简单。

然后就是照片。虽然这一功能还没有上线，但在“未来几周”内，你将能够上传一张照片，附在一条来自 twitter.com 的推文中。显然，这也适用于移动客户端。

昨天，我们推测 Twitter 将使用亚马逊来托管这些照片，而不是把它们放在自己的服务器上。相反，他们与 Photobucket 合作来托管这些照片。有意思。

Twitter 还表示，他们正在与世界各地的移动运营商合作，为简单的图片附件添加彩信功能。

没有关于计划中的 iOS 5 整合 Twitter 的消息，但这是意料之中的。在苹果下周 WWDC 发布之前，Twitter 是不允许发布这个消息的。
