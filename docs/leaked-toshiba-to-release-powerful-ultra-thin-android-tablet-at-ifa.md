# 泄露了！东芝将在 IFA 发布功能强大的超薄安卓平板电脑

东芝 Thrive 并不完全是一款好看的安卓平板电脑。当然，它有很多端口，可拆卸电池，价格低廉，但它也很笨重，感觉像廉价的塑料。没有东芝的标志，它的感觉和外观就像一个随机的中国山寨。但是不用害怕，东芝的粉丝们！根据 IFA 之前的泄露，将有一款新的平板电脑问世，它与华硕变压器或三星 Galaxy Tab 一样性感，同时至少打包了 Thrive 上的一些 I/O 端口。

[![](img/efbeea6875a20282b83d2865dfd6831b.png "toshiba_tablet-new")](https://web.archive.org/web/20230205032226/https://techcrunch.com/wp-content/uploads/2011/08/toshiba_tablet-new.jpg) 
和大多数泄密事件一样，[细节有点匮乏](https://web.archive.org/web/20230205032226/http://notebookitalia.it/toshiba-tablet-killer-in-arrivo-12485)。没有关于价格、目标地区或确切硬件规格的消息。我甚至不确定照片是不是真的，因为端口的尺寸很大。但似乎有一个微型 HDMI、微型 USB 端口和一个相当宽的 MicroSD 卡插槽。甚至还有一条不寻常的大缝，这有望表明在 Thrive 上找到的可移动电池仍在使用。 [Android 和 Me](https://web.archive.org/web/20230205032226/http://androidandme.com/2011/08/devices/toshiba-goes-ultra-thin-in-berlin-for-3rd-android-tablet/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+androidandme+%28Android+and+Me%29&utm_content=Google+Reader) 声明 10 英寸屏幕下是 TI OMAP4460 双核 1.5GHz 处理器。

东芝可能会在本周晚些时候 IFA 发布会之前正式发布这款平板电脑。当然，前提是另一次泄密不会破坏他们的兴致。
