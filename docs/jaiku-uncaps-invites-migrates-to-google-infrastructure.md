# Jaiku Uncaps 邀请，迁移到谷歌基础设施

[![](img/42eedffe9b7ef81b312518db8ae7c92b.png)](https://web.archive.org/web/20221206220820/https://beta.techcrunch.com/wp-content/uploads/2008/08/jaiku_shot.png)

自从去年秋天谷歌收购了 T4 后，Jaiku 的事情一直很平静。人们担心谷歌可能会毁了 Twitter 最强大的竞争对手，因为它要求这家芬兰初创公司花费数月时间将其服务整合到谷歌的技术栈中，而不是推出新功能。与此同时，新注册被禁用，邀请被限制，这切断了该服务健康的新用户流。

但是现在我们看到了一些生命的迹象。Jaiku 已经将其软件转移到谷歌数据中心，并向现有用户发送无限数量的好友邀请。

新的数据中心并不是 Jaiku 的最终落脚点，仅仅是向在 Google App Engine 上运行东西迈出的一步。然而，对于一家似乎首先为可扩展性做准备的公司来说，这是一个重要的问题(众所周知，这个问题一直困扰着 Twitter)。

正如 VentureBeat [指出的那样](https://web.archive.org/web/20221206220820/http://venturebeat.com/2008/08/27/jaiku-comes-back-with-a-new-tos-and-unlimited-invites/)，Jaiku 还引入了新的服务条款和隐私政策，因此它也在理顺自己的法律事务。
