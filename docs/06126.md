# 已证实:Skype 收购移动视频初创公司 Qik TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/01/06/confirmed-skype-buys-mobile-video-startup-qik/amp/>

 <amp-img src="https://web.archive.org/web/20230129114149im_/https://techcrunch.com/wp-content/uploads/2011/01/qik.png" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 看起来 Skype 刚刚拿起手机视频创业 [Qik。](https://web.archive.org/web/20230129114149/http://www.crunchbase.com/company/qik)根据一份新闻稿，Skype 已经与[达成协议](https://web.archive.org/web/20230129114149/http://www.businesswire.com/news/home/20110106006563/en/Skype-Acquire-Qik)收购这家初创公司。收购的条款没有透露，但是消息来源告诉商业内幕[这笔交易大约是 1 . 5 亿美元。我们已经嵌入了下面的版本。](https://web.archive.org/web/20230129114149/http://www.businessinsider.com/skype-to-acquire-qik-for-100-million-says-reliable-source-2011-1)

根据最近的统计，[使](https://web.archive.org/web/20230129114149/https://techcrunch.com/2010/09/09/qik-metrics/)手机用户能够[实时录制和流式传输视频](https://web.archive.org/web/20230129114149/http://qik.com/)的 Qik 拥有[500 万用户。](https://web.archive.org/web/20230129114149/https://techcrunch.com/2011/01/06/qik-funding/)

看起来 Skype 将使用 Qik 的实时视频流技术(该技术“优化了无线网络上的视频传输”)。该公司表示，此次收购将“通过为 Skype 的产品组合增加录制、分享和存储功能，加速 Skype 在视频领域的领导地位。”

> Skype 今天宣布已达成收购 Qik 的最终协议，Qik 是一家移动视频软件和服务提供商，使个人能够从任何地方捕捉、即时分享和保存视频中的精彩瞬间。Qik 有 60 名员工，总部设在加利福尼亚州的红木城，并在俄罗斯莫斯科设有办事处。该交易预计将于 2011 年 1 月完成。收购的条款将不会被披露。
> 
> “Qik 非常努力地解决复杂的问题，让数百万人每天都能与对他们来说最重要的人分享他们的生活”
> Qik 成立于 2006 年，提供创新和灵活的解决方案，通过移动设备、网络和桌面平台与任何人捕捉和分享视频。视频可以实时共享或存储，以便稍后观看，允许视频消息、共享和存档。Qik 服务可在 Android、iPhone、Symbian、Blackberry 和 Windows Mobile 平台的 200 多种手机上使用，并通过与领先的手机制造商和运营商合作，预装在各种手机上。
> 
> Skype 和 Qik 都有一个共同的目标，那就是在任何设备上丰富交流和视频分享。收购 Qik 为 Skype 的产品组合增加了录制、分享和存储功能，有助于加速 Skype 在视频领域的领导地位。通过此次收购，Skype 还将能够利用 Qik 智能流媒体 <amp-img src="https://web.archive.org/web/20230129114149im_/https://s.w.org/images/core/emoji/14.0.0/72x72/2122.png" alt="™" class="wp-smiley amp-wp-enforced-sizes amp-wp-843f19c i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" noloading="" layout="intrinsic" data-amp-original-style="height: 1em; max-height: 1em;" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 技术背后的工程专业知识，优化无线网络上的视频传输。
> 
> Skype 首席执行官托尼·贝茨(Tony Bates)表示:“Qik 团队为其移动合作伙伴和数百万终端用户提供了卓越的视频体验。“据估计，全球 25%的国际长途语音通话分钟数 1 是由 Skype 软件实现的，其中大约 40%的 Skype 至 Skype 通话是通过视频进行的。Qik 深厚的工程能力和强大的移动关系将与 Skype 形成令人印象深刻的互补。”
> 
> Skype 和 Qik 将共同致力于提供更丰富、更集成的体验，使全球各地的人们能够在不同平台上实时视频分享体验，并存储这些时刻，以便以后随时观看。
> 
> Qik 首席执行官 Vijay Tella 表示:“Qik 非常努力地解决复杂的问题，让数百万人每天都能与对他们来说最重要的人分享他们的生活。”“加入 Skype 后，Qik 的团队将与 Skype 的优秀团队联合起来，为我们的客户和合作伙伴开发新的创新产品。”
