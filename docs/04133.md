 <amp-img src="https://web.archive.org/web/20230404150257im_/https://techcrunch.com/wp-content/uploads/2010/02/octazen.jpg" class="shot amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 脸书收购了其第三家公司，马来西亚初创公司 [Octazen Solutions](https://web.archive.org/web/20230404150257/http://www.crunchbase.com/company/octazen) 。脸书表示，根据 GigaOm 的说法，这在很大程度上是一次人才收购。Octazen 在他们的主页上有一个稍微不同的故事，说脸书收购了*“公司的大部分资产，并将这些资产用于不同的方向。”*

不管怎样，这让一些人摸不着头脑。我们今晚采访的一家竞争公司的一名高级工程师说,*脸书刚刚购买了网络上最有才华和创造力的铲运机，绕过了每个人的限速和检测系统*我们今天晚上采访的另一位了解 Octazen 产品的人说,*脸书对通过脸书 Connect 保护他们自己的用户数据非常虚伪，但是 Octazen 多年来一直在从服务条款中删除用户数据，然后再转售。”*两位消息人士均要求匿名。

脸书方面尚未对我们的置评请求做出回应。

Octazen 到底在做什么？该公司主要致力于从一项服务到另一项服务的桌面联系人导入——例如，从脸书登录 Gmail，在那里导入您的联系人并将其添加为脸书朋友。大部分工作都是通过 OAuth 和 API 完成的，但是 Octazen 以深入挖掘数据而闻名。

一个例子是——多个消息来源称，Octazen 有时会直接收集和存储用户凭据，并以用户身份登录大型社交网络和其他网站。然后他们会下载通讯录和社交图。该服务的一定比例的朋友可能是该服务(现在是脸书)的付费用户，你会被要求成为他们的朋友。但是，还有一个大问题是，其他数据会发生什么，以及 Octazen 是否违反服务条款，存储了一个影子社交网络来推荐未来的用户连接。他们可能会更深入地研究数据，比如邮件头信息，以更好地了解你和谁交流最多。

但 Octazen 最令人不安的部分，我们的消息来源说，是他们非常非常擅长在不被发现的情况下大规模收集数据。例如，他们可能会使用大量不同的 IP 地址来攻击某项服务，而不被发现。他们说，Octazen 可以抓取像 Twitter 这样非常公开的网站，在那里社交图出现在每个个人资料上，以一种 Twitter 不会知道的方式。

例如，在 2007 年，人们购买并运行 [Octazen 脚本，以一种非常粗略的方式收集联系人](https://web.archive.org/web/20230404150257/http://lindner.livejournal.com/12948.html):*“所以我们使用 Octazen 的这个工具包从各个网站收集联系人列表。我们曾经热切的用户(ab)如此频繁地使用这一功能，以至于 hotmail 屏蔽了我们。”*海报找到了访问 Hotmail 的 API 的方法，而不是仅仅通过抓取来获取数据，Octazen 回应道*“确实很好”*

我们的理解是，脸书已经使用 Octazen 神秘地确定了你失散多年的朋友，并建议你重新与他们联系(导致我们的收件箱收到大量电子邮件，脸书不知何故在阅读电子邮件或以其他方式获取他们不应该得到的数据)。

最大的问题是，如果脸书所做的只是通过 OAuth 和 API 导入标准的地址簿，或者像脸书使用的专有但记录良好的协议，为什么他们需要收购一家位于世界另一端的公司。言下之意是，这些家伙在大规模数据收集方面非常专业，有时可能会违反被收集网站的服务条款。

这显然只是一个可能的故事的一面，尽管是基于 Octazen 以前不光彩行为的确凿证据和通过多种渠道。但在脸书更详细地解释这次收购之前，我们没有太多的进展。

**更新:**拉里·于在脸书报道:

作为背景，超过 30%的用户目前依赖 Hotmail 以外的电子邮件提供商，雅虎！，AOL 或 Gmail 作为他们进入脸书的认证方式，寻找网站上的朋友并邀请新朋友加入。

随着我们在全球范围内寻求增长，这一比例只会增加。所以我们请来了 Octazen 工程师，因为他们非常擅长为这些包含“长尾”的替代服务建立联系进口商。这很重要，因为新用户可以很快加入，因为他们可以找到并邀请朋友。我们打算让他们专注于建立这些“长尾”进口商，并改进我们现有的解决方案。

Octazen 不会直接收集和存储用户凭证以方便联系人导入。他们的大多数客户将软件安装在自己的服务器上，因此 Octazen 在这些情况下无法访问任何用户数据。有一种按需服务，联系人导入过程将通过 Octazen 的机器运行。但对于这种按需服务，他们只存储用户名和用户密码的单向散列。这些存储的数据不能用于代表用户登录。它仅用于审计目的，因此他们可以根据成功的导入而不是尝试对客户端进行计费。
