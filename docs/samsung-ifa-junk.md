# 三星调侃 IFA，发光的手机和漂亮的人

三星为 2011 年 IFA 奥运会准备了一些大手笔——但他们并不总是如此。该公司刚刚公布了其最新营销活动的两个视频，根据病毒 101 的规则，没有显示任何有趣的东西。尽管如此，IFA 是欧洲的 CES，他们公司肯定会有一些有新闻价值的东西，如 Galaxy II 的继任者或 7 英寸的 Galaxy Tab。

跳跃之后的第二个视频有大部分动作，显示人们拿着发光的手机走来走去，这可能预示着新的智能手机。不管是什么，我们下周都会在现场，喝着小玩意和德国啤酒。
