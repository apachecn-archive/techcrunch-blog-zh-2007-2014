# 鲍勃·曼斯菲尔德将离开苹果公司高级技术副总裁的职位，但将继续从事特殊项目

鲍勃·曼斯菲尔德将辞去苹果公司技术高级副总裁的职务，但将继续留在公司从事特殊项目，向蒂姆·库克汇报工作。

当曼斯菲尔德的传记[从苹果公司网站上的高管名册](https://web.archive.org/web/20221006121307/http://www.macrumors.com/2013/07/28/apple-senior-vice-president-bob-mansfield-removed-from-leadership-webpage/)中消失时，关于他的动向的谣言开始满天飞。该公司尚未发表官方声明，但[向路透社](https://web.archive.org/web/20221006121307/https://twitter.com/PoornimaGupta/statuses/361676821571448833)证实，曼斯菲尔德已经离开。我们已经给苹果发了电子邮件，询问更多信息。

苹果[在 2012 年 6 月](https://web.archive.org/web/20221006121307/https://beta.techcrunch.com/2012/06/28/bob-mansfield-apple-exec-who-has-led-hardware-engineering-for-mac-and-ios-devices-to-retire-this-summer/)宣布，于 1998 年加入公司的 Mansfield 计划从他的硬件工程高级副总裁职位上退休。然而，两个月后，曼斯菲尔德改变了主意，继续留在公司担任技术高级副总裁。当时，有[报道称](https://web.archive.org/web/20221006121307/http://arstechnica.com/apple/2012/11/apples-bob-mansfields-un-retirement-partially-due-to-forstall-exit/)在 iOS 软件主管斯科特·福斯特尔[离开](https://web.archive.org/web/20221006121307/https://beta.techcrunch.com/2012/10/29/scott-forstall-to-leave-apple-as-ive-cue-mansfield-and-federighi-take-on-new-roles/)后，曼斯菲尔德决定留下来。

如果曼斯菲尔德坚持他最初离开公司的决定，这将标志着苹果公司的一个重大转变。曼斯菲尔德领导了苹果一些最赚钱的设备的硬件工程，包括 iPhone、iPad、iPod 和 Mac。
