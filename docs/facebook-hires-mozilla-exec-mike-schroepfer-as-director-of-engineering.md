# 脸书聘请 Mozilla 高管迈克·斯科洛普夫担任工程总监

迈克·斯科洛普夫是 Mozilla 公司备受尊敬的工程副总裁，现在是脸书的工程总监。

他将负责 Facebook 平台和主要产品前端，他今天早上在电话中说，尽管他的确切职责范围尚未确定。从 9 月份开始，他将直接向首席执行官马克·扎克伯格汇报。

众所周知，Schrep 是 Mozilla 负责 Firefox 的工程团队的负责人。在此之前，他是 Sun 微系统公司数据中心自动化部门的首席技术官(负责雄心勃勃的 N1 项目)。为了成为领先的互联网平台，脸书需要在其工程队伍中注入更多开源 DNA。施瑞普应该帮助它做到这一点。

由于 [Mozilla 不会上市](https://web.archive.org/web/20221005184535/http://john.jubjubs.net/2008/01/03/firefoxmozilla-is-not-going-to-ipo-heres-why/)，随着时间的推移，它可能会面临更多顶尖人才的流失。
