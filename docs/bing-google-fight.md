# 哇，微软和谷歌就在我们面前打对方的脸！TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/02/01/bing-google-fight/>

到现在为止，你肯定已经听到这个消息了。谷歌设立了一个诱捕行动(多酷啊？)企图当场抓住微软窃取他们的搜索结果。根据他们的说法，他们就是这么做的——而且让大家都知道了。微软似乎回避并否认了这一指控——然后将指控发回给谷歌。整件事太神奇了，老实说，我还在试图解析这一切。但是你可以通过阅读[Techmeme](https://web.archive.org/web/20230203133112/http://www.techmeme.com/110201/p22#a110201p22)上的内容，从[丹尼·沙利文关于这个话题的原始帖子](https://web.archive.org/web/20230203133112/http://searchengineland.com/google-bing-is-cheating-copying-our-search-results-62914)开始，来了解整个要点。

但是现在最有趣的是谷歌和微软正在进行一场全面的战争。是的，他们差不多已经打了很多年仗了。但这主要是一场安静的战争，发生在幕后，只是偶尔包括在公开场合对另一方的快速抨击。但是现在他们在推特上，以及他们自己非常受欢迎的博客上，直截了当地互称对方是骗子！

新闻发布后，今天早上第一次爆发了打斗。微软立即在他们的 Bing 博客上发表了反驳，标题是:[关于搜索质量的想法](https://web.archive.org/web/20230203133112/http://www.bing.com/community/site_blogs/b/search/archive/2011/02/01/thoughts-on-search-quality.aspx)。那篇文章直接提到了沙利文的文章，但没有直接提到谷歌。
