# 黑莓蓝牙迷你立体声扬声器给你一个扬声器，去任何地方的价格为 79.99 美元

黑莓的配件几乎和它的第一个 BB10 出货硬件一样有趣，除了外部电池充电器，它们还拥有一个蓝牙迷你立体声扬声器/扬声器，非常便携，提供体面的声音，并且不会花费你 79.99 美元。此外，独特的设计意味着它可以夹在车内的遮阳板上，实现免提通信。

扬声器的音质相当不错，比许多更昂贵的设备好得多(尽管为了音频保真度，我还是更喜欢 Jambox)，而且它实际上可以发出很大的声音。没有挤满房间或任何东西，但足够大声的私人聆听。但其主要优势在于，它可以作为一个非常好的蓝牙扬声器，通过 micro USB 充电(可以用外部 Z10 充电器充电)，可以连续几周不充电，并有一个 3.5 毫米的耳机插孔，这基本上使它成为一个蓝牙接收器，可以与立体声或耳机一起使用。
