# 据报道，黑莓互联网服务在埃及被封锁

这只会越来越糟。在屏蔽了推特和脸书之后，看起来埃及政府或运营商可能也屏蔽了黑莓的互联网服务。根据推特上的聊天记录，通过黑莓手机上网已经被暂停。

正如我们在之前的报道中所写的那样，埃及抗议者走上开罗街头，抗议政府腐败和政策。与最近的突尼斯抗议活动类似，埃及的示威活动部分是在脸书和 Twitter 等社交网络上组织的。移动互联网接入是抗议者相互联系的一个重要渠道。

尚不清楚黑莓的所有服务是否都被屏蔽了。有些人[报告说](https://web.archive.org/web/20230203004338/http://twitter.com/marmite_news/status/30660538262355968)用户仍然可以访问应用程序。沃达丰[本周早些时候发推特](https://web.archive.org/web/20230203004338/http://twitter.com/VodafoneEgypt/status/29927017323433984)称推特没有被屏蔽，但这可能是一个网络问题。但是推特上的[对话，](https://web.archive.org/web/20230203004338/http://twitter.com/search?q=blackberry%20egypt)更新速度很快，表明这次宕机可能不仅仅是网络过载。

我们已经联系了 RIM 公司，如果我们得到封锁的确认，我们会及时更新。
