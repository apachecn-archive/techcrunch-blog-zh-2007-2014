# 脸书允许嵌入公共帖子来引诱新闻人物，开始在五个网站上推广

让新闻人物嵌入你的内容，他们会从你那里寻找素材，可能会发布更多给你，你会有更多的内容来做广告。这就是为什么[脸书今天开始](https://web.archive.org/web/20221007022854/https://developers.facebook.com/blog/post/2013/07/31/introducing-embedded-posts/)允许“公共”帖子嵌入第三方网站，这是 Twitter 在 2010 年首创的策略，最近被 Instagram 复制。嵌入是预定全面推出，但现在你可以在五个网站预览。

Bleacher Report、CNN、Huffington Post、Mashable 和 People 是第一批被允许嵌入脸书帖子的网站，但很快新闻提要和时间轴帖子上的下拉箭头将包括一个“嵌入”选项，让你可以将 HTML 复制到任何网站或博客中。考虑到像 WordPress 这样的网站可以通过简单地粘贴一条 tweet 的永久链接来呈现 Twitter 嵌入，这仍然有点笨拙。

![Screen Shot 2013-07-31 at 10.09.08 AM](img/93268c89d21a11c160ee4814c6d2f1c0.png)

这些嵌入将伴随着所有的铃铛和哨子。您可以在帖子中嵌入文本、图片或视频。喜欢和评论，关注公众人物，喜欢网页，点击标签回到脸书，看到相关材料。

脸书在嵌入方面比较安全和保守，只允许公开的帖子在异地粘贴。当然，相当多的人在没有意识到的情况下公开发帖，所以你可能会看到一些有趣的 Tumblrs 嵌入的白痴脸书帖子，没有隐私锁。这一切都意味着脸书需要更好地宣传隐私控制的工作方式。

![Screen Shot 2013-07-30 at 12.03.37 PM[1]](img/b1fe550817b53d9c727d62f4ba9db7ce.png)

为什么嵌入内容如此重要，以至于脸书在本月早些时候从 Instagram 和今天从自己的网站上打开它们？你可以从脸书提醒用户 embeds 文档中了解到，“它也有在脸书上寻找高质量公共帖子进行嵌入的最佳实践。”

类似于脸书的嵌入式 Like 按钮(反过来，Twitter 复制了它的 Tweet 按钮)，一个基本目标是增加它在网络上的足迹，并为它的网站创造更多的访问途径。但这里也有一个更长远的愿景。如果它能成为新闻人物的热门内容来源，他们会花更多时间梳理它，并可能更频繁地发布他们自己的作品和评论，希望能嵌入其中。就像脸书讨好第三方应用开发者，希望他们能贡献内容一样，脸书希望在新闻提要中加入更多有趣的内容，并在旁边显示广告。

此举正值脸书越来越多地试图接受公共内容的想法，尽管它起源于私人分享。标签的推出是朝着这个方向迈出的又一步。事实是，主持突发新闻和目击报道会带来巨大的流量和影响力。正是那些激动人心的时刻，而不仅仅是关于我们朋友的私人趣闻，激起了我们的情感。
