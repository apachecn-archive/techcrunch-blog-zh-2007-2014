# 着眼于视频发现，Flixster 重新推出，帮助用户浏览在线电影世界

好消息是:网上的电影比以往任何时候都多。坏消息是:电影服务也比以往任何时候都多。对于消费者来说，这意味着一种非常分散的体验，他们永远不知道在哪里可以购买或租赁一部电影，甚至不知道是否可以通过网飞这样的视频订阅服务获得。随着其网站的重新推出，时代华纳旗下的 Flixster 希望帮助用户在任何地方获得他们想看的任何电影。

[去年五月，时代华纳收购了 Flixster 和烂番茄](https://web.archive.org/web/20221002232254/http://www.timewarner.com/newsroom/press-releases/2011/05/Warner_Bros_Home_Entertainment_Group_to_Acquire_Flixster_05-04-2011.php)，然后将 Flixster 网站和应用程序作为其[紫外线推广](https://web.archive.org/web/20221002232254/http://gigaom.com/video/warner-bros-flixster-ultraviolet/)的一部分。不幸的是，新 Flixster——以及与之相关的紫外线数字版权锁——在发布时反响相当负面。Flixster 在过去的几个月里一直在努力改进它，并在测试版中推出了一个网站范围的更新，这将使它回到视频发现的根源。

随着去年年底 UltraViolet 的推出，Flixster 主要专注于让用户通过自己的 UltraViolet VOD 服务购买电影。但后来情况发生了变化:除了可以在 Flixster 上购买的图书，该网站现在还突出了从 Vudu、亚马逊和 iTunes 购买和租赁的选项。它甚至可以通过像网飞这样的订阅服务向用户显示何时有电影。这意味着几乎任何在线电影都可以在该应用程序上找到——这对电影爱好者来说是一个大消息。

主页允许用户观看和评价电影，保存它们以备后看，或者说他们不感兴趣。还有很多工具可以优化视频搜索。用户可以根据烂番茄的自动计时器以及电影上映的年份来过滤结果。它让用户可以真正地按流派深入研究，有十几个不同的类别可供选择。用户还可以过滤掉他们是否想看正在大范围上映的电影，过滤掉他们不感兴趣的电影，只显示 Flixster 上可用的标题——但谁会真的想这么做呢？

无论如何，仅仅因为它让用户在不同的服务上查看电影，并不意味着 Flixster 仍然对让他们注册 UltraViolet 感兴趣。该网站简化了紫外线注册过程，通过脸书连接实现了一键注册。任何加入新 Flixster 的人都将在他们的 UltraViolet 数字储物柜中收到一部免费的礼物电影，用户可以通过执行下载 Flixster 移动应用程序或创建“想看的”电影列表等任务来赚取更多电影。

当然，Flixster 并不是唯一一个专注于优质内容的视频发现引擎。还有 Fanhattan，它也与多种视频服务挂钩，可用于 iPhone 和 iPad。除了重新设计测试版网站之外，Flixster 还致力于重新推出其移动应用程序，这将为这些设备带来许多相同的功能。
