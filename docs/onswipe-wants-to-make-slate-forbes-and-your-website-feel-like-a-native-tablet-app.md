# Onswipe 希望让 Slate、Forbes 和你的网站感觉像一个原生的平板电脑应用

[vimeo http://http://vimeo . com/25372812 w = 620]

发布平台 Onswipe 允许任何内容网站在 iPad 和其他平板电脑上创建类似应用的体验，它将于本周结束测试，并将于美国东部时间明天下午 3:00 向公众开放。随着 Onswipe 的推出，它还将增加前 CNN 总裁约翰·克莱因作为顾问。

与赫斯特的[嘉人](https://web.archive.org/web/20230204092059/http://www.marieclaire.com/)、 [Slate](https://web.archive.org/web/20230204092059/http://www.slate.com/) 、齐夫戴维斯[、Geek.com](https://web.archive.org/web/20230204092059/http://www.geek.com/)和[极限科技](https://web.archive.org/web/20230204092059/http://www.extremetech.com/)、汤姆森路透 [PE Hub](https://web.archive.org/web/20230204092059/http://www.pehub.com/) 、[福布斯](https://web.archive.org/web/20230204092059/http://www.forbes.com/)、[Hollywood.com](https://web.archive.org/web/20230204092059/http://www.hollywood.com/)、[斯托克斯维茨](https://web.archive.org/web/20230204092059/http://www.stocktwits.com/)以及广告商美国运通和 Sprint 合作，Onswipe 承诺客户在三分钟内实现触摸设备优化。所有出版商和广告商需要做的就是选择和定制一个布局，导入内容，然后在他们的网站上嵌入一段 javascript 代码。

随着明天的发布，Onswipe 将推出其平板电脑内容观看的三管齐下的方法:1)其发布平台，让出版商为用户提供流畅的平板电脑般的体验。2)一个互动广告平台，让客户从平板电脑特定广告和 Onswipe 广告网络中赚钱。3)一个允许用户共享和保存内容以备后用的社交平台。

联合创始人 Jason Baptiste 谈到这家初创公司早期以平板电脑为中心的方法背后的动机时说:“平板电脑是这一代人的电视，我们让出版商在上面制作他们的内容，但也从中赚钱。最后，我们说，“嘿，如果我们有了这个新的网站网络，我们如何利用它？”

Onswipe 的平台将允许出版商和广告商不仅从文章中添加和定制内容，还可以从 Twitter、YouTube、Quora、Instagram 和 Tumblr 等社交媒体来源添加和定制内容。此外，此次发布还将增加 myOnswipe，这是一种类似 Instapaper 的离线阅读服务，面向 Onswipe 网络中的网站。

Baptiste 告诉我，Onswipe 不为其发布平台向公司收费，而是通过从其广告平台获得部分收入来赚钱，“我们已经让出版商在平板电脑上比以前赚得更多。这是一个很大的数目，”巴普蒂斯特说。
