脸书收购分支媒体团队，领导新的“对话”小组

看起来脸书已经收购了 Branch Media ，conversation service Branch和链接共享服务Potluck背后的初创公司。

联合创始人乔希·米勒在脸书宣布了。他说，他的团队将组建一个新的对话小组，该小组将设在纽约市，“目的是帮助人们围绕自己的兴趣与他人联系。”他补充道，“尽管我们打造的产品会让人想起Branch and Potluck，但这些服务将在脸书之外继续存在。”

这个帖子很少提到细节(米勒说他目前正在度假，现在才发帖，因为他“被告知这个故事将会泄露”)，但 The Verge报道价格是1500万美元。我已经联系了脸书以获取更多信息，如果有回音，我会及时更新。

Branch Media的投资者包括Obvious Corp .(由Ev Williams、比兹·斯通和詹森·高德曼领导的孵化器——在2013年的一篇文章中，)威廉姆斯表示，高盛“去年搬到了纽约市，大部分时间都在与……Branch合作”)、Lerer Ventures、Betaworks、SV Angel等。据报道，初创公司筹集了200万美元的种子资金。

更新:脸书发言人证实分支媒体团队将加入公司。他们拒绝讨论交易条款，但强调这是一次人才收购。
