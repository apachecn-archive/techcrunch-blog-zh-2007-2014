# 点评:LaCie 1TB 加固型 XL 外置硬盘 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/08/06/review-lacie-1tb-rugged-xl-external-hard-drive/>

![lacie-001](img/df6f27d06fb7088c23abb1db042b3821.png "lacie-001")
作为 LaCie 最知名的产品之一，坚固的外置硬盘系列已经存在了很长时间，但直到最近才通过这款 1TB 设备扩展到 3.5 英寸硬盘领域。设计大同小异，但显然它要大得多，也更宽敞。但是，与其他 1TB 驱动器相比，它值得溢价吗？
 **Big ' n grippy**
关于之前的车型，我能说的还不多。除了尺寸之外，坚固的 XL 在各个方面都与它们非常相似。以前的硬盘是平装本大小，而这个更像是精装本。

这款硬盘与你预期的 3.5 英寸硬盘一样重，但感觉很平衡，而且因为它是用来躺着而不是站着的，所以给人一种稳定感。它的小脚可以防止它平坐在桌子上，无论你把它放在什么表面上，它都能牢牢抓住。它打开时也有非常健康的光芒——可能有点太健康了。如果你不想做橙色的梦，睡觉前拔掉电源。

![lacie-003](img/7ad3d32263c11c29041b1519769bdb87.png "lacie-003")

**更粗犷的*气派的*，实在的**

它的大小是第一个表明，虽然它是一个外部驱动器，但它并不是一个真正无处不在的东西。我的意思是，它可以去任何地方，但像大多数其他 3.5 英寸驱动器一样，它体积庞大，而且不是总线供电的。那么这有什么意义呢？嗯，这是一个好看的，相对宽敞的驱动器，你真的不用担心你的健怡百事可乐洒在或碰掉桌子。

“崎岖不平”可能有点夸张。可以肯定的是，它比更便宜的驱动器更坚固，更便宜的驱动器根本没有减震功能，可能容易受到飞溅的葡萄酒的影响，但它不是真正意义上的坚固。橡胶缓冲器有所帮助，但铝制外壳很薄(毕竟不能保持热量)，底部的通风口提供了一个弱点(尽管它们被屏蔽以防止碎屑和砂砾进入)。端口也暴露在外，这是我真的没有想到的——一个翻盖或小悬挂端口插头可能看起来很丑，但如果这东西在袋子里呆了很长时间，它会防止很多麻烦。

我没有真正测试减震，因为我想保持这东西完好无损，但它在 LaCie 的其他驱动器的评论中已经确立，只要它关闭，它就可以很好地承受轻微的撞击。在任何情况下，它都组装得很好，而且橡胶不会很快凹陷。

![lacie-002](img/6e1b05aebfe886e8f8016277b5801a94.png "lacie-002")

也就是说，除了受控的办公室环境，我宁愿在任何情况下都有一个这样的人(或他们的小兄弟)和我在一起。咖啡店？这东西。外景？这东西。户外？好吧，也许不完全是这个东西，因为你需要把它插上电源，但是你明白了。

**试驾**

安装基本上是不存在的，尽管第一次确实弹出了一个向导，可以选择简单地格式化为 NTFS，或者在 FAT32 中有一个单独的分区，作为 MAC 和 Linux 机器之间的媒介。方便，但是第二个分区被限制为 32GB 或更少。我敢肯定这是有原因的，但是如果把它从中间分开，两者都用，那可能会更好。

[![optimize](img/b894ede6abfc64605979efda72bffd70.png "optimize")](https://web.archive.org/web/20221006193753/https://beta.techcrunch.com/wp-content/uploads/2009/08/optimize.png)


我进行了一些非正式的读/写测试，发现声明的 USB2 上 30MB/s 和 eSATA 上 80MB/s 的最大值至少在正常容差范围内:

*   一个 1GB 的文件传输到硬盘大约需要 34 秒，从硬盘传输出来需要 31 秒，这与估计的 30MB/s 完全一致。
*   1GB 的 MP3 需要的时间要长得多:不到一分钟，平均约为 18MB/s，而以 20MB/s 的速度从驱动器中传输 MP3 需要 50 秒
*   1GB 的照片和混合文档(大概 2000 左右)用了一分多钟，平均 14MB/s 左右，一分钟平传回来，大概 16.5MB/s。

不会让世界着火，但它似乎没有窒息。“exchange”分区倾向于较慢地复制大文件，较快地复制小文件。我没有对 eSATA 进行广泛的测试，因为，老实说，我直到几分钟前才想起它，但快速复制的几份文件显示它的平均速度为 60-70MB/s，从驱动器传输时更高。

**写支票前先检查一下自己……从自己身上**

现在，这种驱动器的价格大约为 140 美元，比目前 1TB 外部驱动器的平均价格高出 30 美元。额外的花费值得吗？好吧，考虑一下你要用它做什么。如果它会在你的显示器后面安全地呆上一年，那很可能不会；它没有速度优势，而且由于占地面积大，会占用你桌子上更多的空间。但是如果你是移动的，在一个共享的空间工作，或者只是简单的笨拙，拥有一个这样的东西可能是值得的。当然，它的移动性受到它不是公共汽车供电这一事实的阻碍，但如果这也不是一个问题，那就去吧。
