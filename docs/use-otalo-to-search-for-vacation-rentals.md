# 使用 Otalo 搜索度假租赁

度假租赁是一项大生意。HomeAway 拥有十几个度假屋上市网站，刚刚[在一轮风险投资中筹集了 2.5 亿美元](https://web.archive.org/web/20221209132821/http://www.beta.techcrunch.com/2008/11/10/monster-round-for-homeaway-250-million/)，对该公司的估值超过 10 亿美元。

但是房源分散在很多网站上——这个行业需要的是一个好的中央搜索引擎来搜索所有的度假屋出租。在不同的网站上做大量的搜索也很痛苦，因为你必须输入你想去的地方和时间，这比标准的搜索引擎查询复杂得多。

这正是刚刚推出的 [Otalo](https://web.archive.org/web/20221209132821/http://www.otalo.com/) 正在做的事情。他们从 20 万份房源开始。该公司由迈克尔·贾尔斯(Furl 的[创始人)和贝尔·蒂尔克尔创建。Otalo 这个名字来源于禅宗符号 Enzo——“O”——芬兰语中房子的意思是“talo”。](https://web.archive.org/web/20221209132821/http://www.beta.techcrunch.com/2005/06/19/profile-furl/)

该公司将从向各种度假租赁网站推荐客户中赚取佣金。顺便说一句，如果你旅行时总是住在旅馆里，你有时应该试着租房子住。它要平静得多，通常也便宜得多。你也可以在最后一分钟得到很好的交易。
