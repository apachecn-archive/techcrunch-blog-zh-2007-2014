# 卡尔·伊坎与摩托罗拉谈论其庞大的专利组合

在移动领域，专利从未如此重要。拥有最多专利的人权力最大。句号。仅在过去的几个月里，我们就已经看到[苹果和三星一决雌雄](https://web.archive.org/web/20230203035251/https://techcrunch.com/tag/apple)，[苹果和 HTC 一决雌雄](https://web.archive.org/web/20230203035251/https://techcrunch.com/2011/07/18/judge-rules-htc-infringed-on-2-of-apples-patents/)，[谷歌与 InterDigital](https://web.archive.org/web/20230203035251/https://techcrunch.com/2011/07/21/after-nortel-loss-google-looks-to-interdigital-to-strengthen-patent-portfolio/) 搭讪，我甚至不会从[北电](https://web.archive.org/web/20230203035251/https://techcrunch.com/2011/07/01/apple-microsoft-rim-google-nortel-patents/)开始。

亿万富翁投资者卡尔·伊坎——以向公司注资并对其发号施令而臭名昭著——对摩托罗拉移动公司的专利投资组合颇感兴趣。拥有超过 17，000 项无线相关专利和 7，500 项正在申请中的专利，摩托罗拉可谓囊中之物，伊坎希望该公司尽可能地利用这些专利。

根据一份股票市场文件，伊坎与摩托罗拉举行了会议，要求(或者告诉)该公司“探索有关其专利组合的替代方案，以提高股东价值。”就规模而言，摩托罗拉的投资组合让北电 45 亿美元的专利宝库相形见绌，其中还包括许多关于 4G 技术的宝贵专利。该公司将从销售中获得数十亿美元，尽管这种行为的后果可能是成本大于利润。

有了专利，你有两个选择:1)大量销售，快速致富，或者 2)使用它们。给他们发执照。看到稳定的现金流。几乎在每一种情况下，坚持你的投资组合，要么许可专利，要么对侵权的竞争对手提起诉讼是最好的选择。事实上，北电最初出售的唯一原因是，从长远来看，破产将使该公司永远不必支付许可费。

在与伊坎会面后，摩托罗拉实际上从未放弃出售。然而，该公司确实提到，收入增长 22%与摩托罗拉强大的专利收藏有很大关系。
