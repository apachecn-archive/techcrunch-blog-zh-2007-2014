# 脸书将取消 5000 个好友的限制 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/05/09/facebook-to-lift-5000-friends-limit/>

本周，一位知情人士告诉我们，脸书将很快取消用户好友数量不得超过 5000 的限制。

关于这种限制存在的原因有很多故事。官方理由是，脸书希望确保人们只在自己的账户上添加“真正的”朋友，并且限制在任何一个人可以合理拥有的朋友数量的上限。非官方的(也是实际的)原因是:缩放问题使得这是必要的。我直接从脸书的员工那里听说了这一点，还有其他人。

但我们从消息来源处获悉，这些规模问题已经得到解决，上限将很快被取消。

脸书说“不到 1000”的用户现在有 5000 个朋友。大约有 7000 万活跃的脸书用户，所以受影响的用户数量大约是千分之一。但是不成比例的博主和媒体已经达到了极限，所以这个问题会得到更多的关注。

知名博主 Robert Scoble 是达到上限的 1000 名脸书用户之一，他过去曾抱怨过这一限制。

脸书表示,“页面”功能是为那些希望拥有比普通账户更多“朋友”的人和品牌准备的。一个例子是巴拉克奥巴马的脸书页面，目前显示有 82 万支持者。

但是对于很多人来说，做朋友和做粉丝有很大的不同，允许的互动程度也有显著的不同。新的[好友列表](https://web.archive.org/web/20221209153626/http://www.beta.techcrunch.com/2008/03/18/facebook-to-launch-new-privacy-controls-confirms-chat-is-coming/)功能允许用户对好友进行分类和分组，无论如何都让组织变得更加容易。
