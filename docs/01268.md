# 雅虎没有超级碗:高层权衡他们有限的选择

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/02/03/no-super-bowl-for-the-yahoo-top-brass-weigh-their-limited-options/amp/>

因此，显然雅虎高层并没有像预期的那样接受微软的公开收购要约(嘿，至少他们的股东终于松了一口气——股价在周五上涨了近 1/3)。他们整个周末都在工作(我怀疑他们最后一分钟取消了超级碗的门票)，想找个人，任何人，提出一个竞争性的报价。除此之外，他们只是想尝试提高报价，从目前的[31 美元/股](https://web.archive.org/web/20230126182429/http://techcrunch.com/2008/02/01/wow-microsoft-offers-446-billion-to-acquire-yahoo/)到至少，据雅虎内部人士称，到 35-40 美元/股。

但事实是，各种因素的完美风暴将导致微软在不久的将来收购雅虎，或许不会比最初的报价有任何价格上涨。债务危机意味着筹集大量债务来杠杆化交易是不可能的(最近最大的债务融资交易在 40 亿美元左右，远远不足以完成这笔交易)。雅虎的财务状况继续下滑，这让潜在买家对买狗持怀疑态度。事实是，这笔交易对微软来说是有意义的。对其他人来说可能没有意义。

所有现实的阻碍都是政府对该交易的否决(根据[哈特-斯科特-罗迪诺](https://web.archive.org/web/20230126182429/http://www.ftc.gov/bc/hsr/hsr.shtm)和相关欧盟法规的 DOJ 审查)。或者，也许，某个白夜出现并拯救这一天的机会很渺茫。前雅虎首席运营官丹·罗森斯维于 2006 年末离开雅虎，据传他正致力于促成一项涉及 NBC 环球和脸书的复杂交易。更多信息请见下文。

以下是雅虎的潜在选择。所有这些都是谣言，有些比其他的更有意义。

**新闻集团**

 **新闻集团正忙着在周五达成一项交易并提交竞标。据我们采访的私募股权投资人说，没有买家，在他们花了周末时间考虑之后，还不清楚他们是否真的想要这笔资产。目前，把新闻集团排除在竞争之外，尽管如果雅虎被分拆出售，他们可能会竞购部分资产。

**苹果**

作为一家市值 1180 亿美元的公司，他们拥有促成这笔交易的市值。但苹果自 2002 年以来就没有收购过一家公司，他们也不太可能将雅虎视为战利品。人们会喜欢这笔交易，但它不会发生。

**纯私募**

本周末，私人股本公司分成了两大阵营。一些人正在考虑报价，但债务市场一直阻碍着他们。其余的人只是想弄清楚周一是否会出现任何形式的竞标，这样他们就可以交易股票获利。他们中的一些人今天突然打电话给我，询问新闻集团的出价。他们都不认为私人股本收购是现实的。

**易贝**

 **雅虎和易贝是长久以来的恋人——2000 年 3 月，雅虎对该公司的收购在公开宣布前的最后一分钟被扼杀了(我们听说是文化因素)。当时，人们认为，易贝的市值只有 200 亿美元左右，而雅虎的市值为 700 亿美元。如今，这两家公司的规模差不多，都不到 400 亿美元。易贝拥有参与竞购并保持控股地位的市值(T4 只有 T5)。但他们是一家转型中的公司，有自己的成长问题。他们长期的首席执行官[即将卸任](https://web.archive.org/web/20230126182429/http://techcrunch.com/2008/01/24/meg-whitmans-exit-interview/)。易贝不太可能进入拳击场。

**主权权益基金**

中国、挪威和一些中东国家拥有足够大的主权财富基金，可以对雅虎发起全现金收购。但是他们为什么要这么做呢？

**白骑士丹·罗森斯维？**

<amp-img src="https://web.archive.org/web/20230126182429im_/https://techcrunch.com/wp-content/uploads/2008/02/rosensweig.jpg" class="snap_nopreview shot2 amp-wp-enforced-sizes amp-wp-ab76592 i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="" layout="intrinsic" data-amp-original-style="float:right;" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>丹·罗森韦格(Dan Rosensweig)于 2006 年末以首席运营官的身份离开雅虎，此后加入大型私募股权商店 Quadrangle Group。他有达成交易的动机(胜利回归雅虎)，也有一些工具(他可以支配的大量资金)。但债务危机对他的影响不亚于其他考虑这笔交易的私人股本公司。

但这是有趣的地方。据我们所知，一项计划正在考虑将雅虎的媒体资产出售给 NBC Universal(音乐、TC、体育、金融、娱乐、OMG、食品、健康等)。–除视频外的所有内容)。剩下的——雅虎的核心搜索和广告资产——将被打包并与脸书合并。是的，脸书。这可能会导致脸书在不久的将来成为一个公共实体。

这笔交易极具创意，但很难卖给雅虎股东，他们了解微软股票的价值，但不了解脸书的价值。分拆雅虎并将其分拆出售也意味着一项非免税交易，这使得它不如微软提供的那样有吸引力。我很想看到罗森斯韦格尝试四分卫这笔交易，或类似的东西，但有太多的障碍。

**游戏结束**

超级碗并不是今天唯一一场不可避免的结局。越来越明显的是，本周将会有竞争性报价。即便如此，微软也可能会提高出价。雅虎的命运似乎是注定的——成为微软公司的一个运营子公司。
