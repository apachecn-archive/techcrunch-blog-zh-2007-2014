# 什么云？Nimble 获得 2500 万美元的存储和备份盒

我们都听说过，云是未来。鲍尔默的一切都在 T1，乔布斯说 T2 的一切都在云 T3，施米特 T4 把它作为 2010 年谷歌战略 T5 的核心。但是我们也看到了云中断的危险，多亏了亚马逊。因此，尽管云的前景令人眼花缭乱，但 Nimble Storage[正以真正的反叛形式悄悄地提供一种替代方案——投资者正在买入。](https://web.archive.org/web/20230203103048/http://www.nimblestorage.com/)

今天，融合存储和备份解决方案的开发商 Nimble Storage 宣布，它已经完成了 2500 万美元的 D 轮融资，继去年 11 月完成的 1600 万美元的 C 轮融资之后。这使投资总额达到 5800 万美元。这轮融资由 Artis Capital Management 牵头，现有投资者、Accel Partners、Lightspeed Venture Partners 和红杉资本分别出资。注入的资本将用于拓展国际市场，支持现有的增长，并有望使公司盈利。

Nimble Storage 由前 Data Domain 和 NetApp 高管 Varun Mehta 和 Umesh Maheshwari 创建，一直在热销其数据存储和备份设备，主要是因为它为可能负担不起 it 团队的中小型企业以及那些不想处理大量服务器的企业提供企业存储。Nimble 的四个设备套件将存储、备份和灾难恢复整合到一个解决方案中，每个盒子都提供不同级别的原始存储容量(高达 24tb)、闪存容量和连接速度。

Nimble 的解决方案提供闪存加速的主存储性能、即时备份和恢复、应用集成的数据保护和异地灾难恢复，所有这些都来自单个 iSCSI 系统，有效地降低了设备成本并简化了存储管理。
