# 雷德·哈斯汀斯:我们把网飞带到了脸书，除了美国

网飞正以一种真正的社交方式来到脸书。今天，在脸书的 F8 开发者大会上，马克·扎克伯格展示了脸书的会员将能够看到他们的朋友在网飞观看的电影或电视节目，并点击电影在脸书观看。会弹出一个叠加播放器，你甚至不用离开脸书就可以看电影。

网飞首席执行官兼脸书董事会成员雷德·哈斯汀斯也发表了讲话，并将在脸书发现新电视节目和电影的体验与网飞自己的推荐算法进行了比较。“我的朋友确实胜过了算法，”黑斯廷斯说。如果你曾经因为在你的脸书新闻订阅中看到某个 YouTube 视频而点击过它，你可以想象你会如何点击网飞的视频(如果你有时间观看一个更长的视频)。但如果你住在美国，你就必须想象一下，因为美国相关视频观看数据的隐私法阻止网飞在美国打开这款应用程序。不过，它将立即在其他 44 个国家推出。黑斯廷斯指出，1988 年通过的《T2 视频隐私保护法案》正在审查过程中，可能会在美国被推翻..

查看数据会引发各种隐私问题。但如果脸书正确处理隐私问题，让你只分享你想分享的观看习惯，这种社交电视可能会创造一种全新的方式来找到要观看的节目和电影。人们通过各种应用程序广播他们正在观看的内容已经变得越来越普遍。现在在脸书，当你看到这些状态更新时，你应该也能看到。如果美国法律允许的话。

潜力肯定是存在的。黑斯廷斯分享了一个轶事，他让扎克伯格定义社交电视的成功。扎克伯格回击道，“你明年打算长到多大？”黑斯廷斯告诉了他一个数字。扎克伯格告诉他，成功在于网飞的增长是预期的两倍(大概是流媒体视频的数量)。如果网飞能在美国的脸书上推出其视频分享应用，它可能就会成功。
