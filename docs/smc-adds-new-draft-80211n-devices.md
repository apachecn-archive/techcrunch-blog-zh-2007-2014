# SMC 增加了新的 802.11n 设备草案| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/21/smc-adds-new-draft-80211n-devices/>

![SMC](img/11bcb24de2fe7a6a1562c2e36013a069.png)

位于加州尔湾的网络硬件公司 SMC 最近在其产品线中增加了一款新的 802.11n 路由器、接入点和 USB 适配器。SMCWGBR14-N 路由器包括 4 端口 10/100/1000Mbps 千兆局域网交换机、802.11n 无线草案、硬件防火墙、WPA/WPA2 加密和 USB2.0 打印服务器。它的零售价为 174.99 美元。你可以用 SMCWEB-N 接入点或 SMCWUSB-N USB 适配器连接路由器，价格分别为 114.99 美元和 62.99 美元。这三种设备现在都可以使用。好耶。
