# 天美时发布售价 275 美元的 GPS 手表

[![](img/a43f509882f4103fb5e5b18f702165f7.png "timex-gps")](https://web.archive.org/web/20221210045850/https://beta.techcrunch.com/wp-content/uploads/2010/06/timex-gps.jpeg) 
我爱我的 GPS 手表。虽然它们在实践中没有理论上那么令人兴奋，但它们仍然是锻炼身体和保持正确轨迹的好方法。到目前为止，我只用过 [Garmin](https://web.archive.org/web/20221210045850/http://crunchgear.com/tag/Garmin) 型号，但这款售价 275 美元的新款 [Timex](https://web.archive.org/web/20221210045850/http://crunchgear.com/tag/Timex) 手表让我非常兴奋。

Timex Ironman Global Trainer 售价 275 美元，包括一个 SiRF Star III GPS 接收器，支持多达 100 个 GPS 航路点。花 50 美元你可以得到一个心率监测器。这款手表防水 50 米，非常结实。
