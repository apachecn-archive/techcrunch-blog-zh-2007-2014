# 苹果终于准备好切断我头上烦人的电线了吗？

在我生活中的所有烦恼中，我最常处理的是耳机线的解开。一天有几次，我把手伸进口袋，拿出一团绳子，看起来好像是被某种袖珍侏儒故意缠在一起的。这让我发疯。

无线耳机并不新鲜，但最好的耳机通常不适合放在你的口袋里:对我来说，这是耳机的首要要求。所以我坚持用我的衣服自带的那种。但是，如果 9to5Mac 的消息来源是正确的，这些可能很快就会得到重大检修。

几个月前，苹果显然非常低调地收购了总部位于旧金山的公司 Wi-Gear，[塞思·温特劳布已经了解到](https://web.archive.org/web/20230208033019/http://www.9to5mac.com/34608/apple-buys-wi-gear-to-build-stereo-bluetooth-headphones)。他们的专业领域？蓝牙无线耳机(特别是 A2DP 立体声耳机)和适用于旧设备的蓝牙 2.0 适配器。

果不其然，该公司现在[关闭了](https://web.archive.org/web/20230208033019/http://www.wi-gear.com/about_us.php)的业务，至少他们的联合创始人之一 Michael Kim 现在在苹果公司担任 iOS 蓝牙工程师。

虽然这样的收购可能只是为了增加一些蓝牙人才，但你必须假设苹果迟早会想用他们的耳塞进行无线连接。这次收购是他们朝着那个方向前进的最强烈的信号。

作为世界上最受欢迎的 MP3 播放器和最受欢迎的智能手机之一的制造商，苹果的白色耳塞已经无处不在。无处不在的烦人。是时候切断脐带了。

**更新**:看起来温特劳布是[错了](https://web.archive.org/web/20230208033019/http://www.macworld.com/article/155623/2010/11/apple_wigear.html)，苹果没有买 Wi-Gear，他们只是雇佣了金，公司也就倒闭了。
