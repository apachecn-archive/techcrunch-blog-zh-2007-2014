德国电信推出移动组件市场TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2012/07/03/deutsche-telekom-launches-mobile-component-marketplace/

移动应用组件市场和前Disrupt决赛选手Verious刚刚宣布与移动运营商Deutsche Telekom建立战略合作伙伴关系，这将扩大其国际足迹和影响力。这家移动运营商现在正在欧洲14个国家推出其Verious-powered 开发者花园组件市场，在那里它不仅将展示iOS、Android、Windows Phone和HTML5的移动应用组件，还将展示电信自己的API。

这些API提供了对Telekom大量服务的访问，如机器对机器(M2M)通信、基于云的语音应用、短信、运营商计费等。例如，新添加的 Telekom Tropo API 使用了美国Voxeo Labs的技术，允许应用程序和网站转移呼叫、与呼叫者交谈、评估响应等。

在Verious发布消息之前，该运营商最近宣布了与微软MSDN网络的合作，为开发者提供用于智能手机、平板电脑和个人电脑的API。该公司表示，通过这一合作以及与英特尔的另一项合作，开发者花园网站现在面向1700万全球开发者。在组件市场发布期间，将为希望使用电信API进行构建的开发人员提供一些特殊的促销活动。稍后，德国电信计划允许开发者也通过其在线服务分发和销售他们的应用程序。

除了出现在德国电信品牌的开发者网站上，电信的API还将出现在Verious.com主页及其开发者网络上，后者通过时事通讯和第三方网站每月接触超过100万应用开发者。

许多移动运营商向开发者开放他们的API——Telekom并不是第一个这样做的。例如，就在最近，克里斯报道了AT&T在这方面的一些举措。然而，Telekom是第一个向其开发者社区提供完全成熟的移动组件市场的公司，其API与其他组件一起列出，有助于加快开发者的上市时间。据Verious首席执行官阿尼尔·佩雷拉(Anil Pereira)称，此举表明开发者除了API之外还需要代码。“随着事情朝着‘应用世界’发展，具有前瞻性思维的服务提供商正试图给开发者尽可能多的价值，”他说。

这一观点得到了德国电信核心电信产品业务部门高级副总裁Rainer Deutschmann的回应，他表示，该公司推出该市场是为了帮助支持开发者，使他们的工作“更容易，更有利可图”市场现在可用这里。
