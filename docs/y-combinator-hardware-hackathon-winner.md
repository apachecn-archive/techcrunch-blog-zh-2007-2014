# 用 Upverter + YC 硬件黑客马拉松获奖的电子人手套挤压虚拟现实[视频]

如果你真的能抓住你在电子游戏中捡起的剑，或者如果外科医生能感觉到他们的机器人击中骨头，那会怎么样？这是战术手套 T1 的承诺，它赢得了本周末的 Upverter+Y Combinator T2 硬件黑客马拉松 T3。观看我们演示弗兰肯斯坦遇见任天堂的装置，让你触摸虚拟现实中的物体。

胶带，麻绳，还有一大堆热胶水。这就是杰克·米纳尔迪和他的团队如何在短短八个小时内打造出[战术](https://web.archive.org/web/20221209064906/https://www.facebook.com/Tactilous)。白天为 entthought 工作的 Python 研究员米纳尔迪说，他被硬件黑客所吸引，因为“我可以准确地告诉计算机该做什么，但我希望能够告诉世界该做什么。”

其他团队制作了将图像刻录到吐司上的小工具，但米纳尔迪的团队赢得了一台 iPad、一些 Pebble 手表，以及一些硅谷顶级投资者的访问权。队友们还不会辞职。然而，他们的[代码](https://web.archive.org/web/20221209064906/https://gist.github.com/jminardi/5022297)在 GitHub 上，他们正在寻找公司来帮助他们使用气动人工肌肉建造第二个原型。有一天，这种设备可以帮助中风患者康复。

把这东西绑起来太不现实了。当它把我的手指拉回来模拟握住硬物的阻力时，我突然有一种想要机器人横冲直撞的冲动。看着上面的伺服火和驱动器紧张，让我感觉到一些不存在的东西。
