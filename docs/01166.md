# 雅虎直播来了，我是说雅虎生活 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/01/08/here-comes-yahoo-live-i-mean-yahoo-life/amp/>

如今，每个人都想变得更像脸书，雅虎也不例外。谷歌的战略是[开放社交](https://web.archive.org/web/20230126193412/http://techcrunch.com/2007/10/30/details-revealed-google-opensocial-to-be-common-apis-for-building-social-apps/)，一套开放的 API，允许第三方在其他第三方上嵌入内容。谷歌开放社交的基石是 Gmail，所有的行为都是为用户而发生的(其次是 Orkut 和 iGoogle)。雅虎似乎也在朝着同样的方向前进，将雅虎邮箱作为他们新的社交网络战略的中心。

杨说，新产品目前只是一个概念验证。目标是联合邮件、搜索等服务。在社交框架下，以雅虎邮箱为中心枢纽。您的电子邮件和即时消息联系人充当您的联系人，并填写社交图表。他们和你的亲密程度是基于你和他们交流的频率。

丹·法伯举了一个例子:

> 他举了一个为 CES 策划晚宴的例子。你可以把线拖到地图上，它会显示邮件上的人的简介，记录偏好(在这种情况下是食物)，并推荐该地区的餐馆。你也可以接收一封电子邮件，弹出邮件中的个人资料，从邮件中提取一个地址并显示地图。

第三方应用程序也不例外。例如，上图清楚地显示了与 MySpace 的集成。这就是雅虎对 Zimbra 的野心，它在 2007 年底以 3 . 5 亿美元收购了 Zimbra。开发者的新平台部分基于 Zimbra 的技术。

早在 2006 年末，这一战略的第一步就已经开始实施，当时雅虎开始在 T2 推出雅虎邮件和即时通讯的整合。信息现在可以在即时消息和电子邮件之间无缝地来回传输，这是其他网络邮件应用程序所不具备的。很明显，即使在那时，雅虎也在考虑邮件界面的未来。

不幸的是，雅虎，在社交网络方面有过一个[式的混乱过去](https://web.archive.org/web/20230126193412/http://techcrunch.com/2007/11/14/inbox-20-makes-me-sad/)，把这个产品称为“生活”这太接近于微软在新品牌下抵制在线服务的策略了。在过去的几年里，它除了横着走之外，没有任何进展，这表明在今天的互联网中，重新包装和营销对推动用户采用没有什么作用。

但是抛开名字不谈，尽管我之前批评过，雅虎生活看起来像是雅虎的智能社交架构。假设所有这些社交网络的东西不仅仅是一种时尚(它不是)，那么雅虎将资源集中在这场战斗中是有意义的。当然，除了现有的脸书和谷歌平台之外，开发人员现在还有另一个平台可以构建。
