# TechCrunch 电视台:热衷于和李夏琳的连接

***[![](img/9e028420b057b0aa302812b2f415b846.png "keen")](https://web.archive.org/web/20221206220139/https://beta.techcrunch.com/wp-content/uploads/2010/07/keen.jpg) 在这篇客座博文中，作者 Andrew Keen 介绍了他的 TechCrunch 电视节目第二集 [Keen On](https://web.archive.org/web/20221206220139/http://techcrunch.tv/keen-on) …*** 

如果说 21 世纪第一个十年的 Web 2.0 时代是关于用户生成内容的，那么本世纪第二个十年的社交媒体时代则是关于技术改变我们生活的方式。昨天的 Web 2.0 都是关于数据的；今天的社交媒体都是关于人的。

一个日益协作和社会化的互联网似乎——至少对那些相信其功效的人来说——正在成为社会和商业可以彻底转变的工具。

上个月出版的两本重要书籍，克莱·舍基的 [*认知盈余:互联时代的创造力和慷慨*](https://web.archive.org/web/20221206220139/http://us.penguingroup.com/nf/Book/BookDisplay/0,,9781101434390,00.html?Cognitive_Surplus_Clay_Shirky) 和李夏琳的 [*开放式领导:社会技术如何改变你的领导方式*](https://web.archive.org/web/20221206220139/http://www.josseybass.com/WileyCDA/WileyTitle/productCd-0470597267.html) 都雄辩地证明了这一点。在他的*认知盈余*中，Shirky——他在纽约大学任教，是 2007 年热门作品 *[人人来了](https://web.archive.org/web/20221206220139/http://en.wikipedia.org/wiki/Here_Comes_Everybody)* 的作者——认为集中式大众媒体生态系统的崩溃将我们从看电视中解放出来，让我们成为更有创造力和更慷慨的公民。李，[高度计集团](https://web.archive.org/web/20221206220139/http://www.altimetergroup.com/)的创始人，2008 年畅销书 *[《惊涛骇浪](https://web.archive.org/web/20221206220139/http://www.forrester.com/Groundswell/index.html?cm_guid=1-_-100000000000000000062-_-3035823578&cm_mmc=google-_-Groundswell_-_Opt-_-Groundswell_-_Opt-_-groundswell%7C-%7C100000000000000000062&gclid=CIfgxvjO2qICFRX_iAodNk0Yvg)* 的合著者，认为社交媒体技术让企业领导人能够让他们的公司更有效、更果断，从而盈利。

那么，社交媒体真的正在彻底改变世界吗？我邀请李和 Shirky 参加我的节目的第二集，以了解这两本新书的更多信息，并了解他们对互联网如何从根本上改变商业、社会和文化的看法。
