# 更新:脸书证实没有私人信息出现在时间轴上。它们是旧的墙柱。

**更新完毕。今天早上，一些[脸书](https://web.archive.org/web/20230221203204/http://www.facebook.com/)的用户感到震惊，因为他们发现写于 2009 年及更早时候的私人信息显示为“朋友发布的”**

这个帖子的最早版本包括一些用户的账户，他们声称自己的私人信息被转换成了时间线帖子。TechCrunch 进行了更多的调查，我们没有发现任何证据表明据称被曝光的帖子实际上是私人信息。我们的脸书专家 Josh Constine 发现，电子邮件收据显示所谓的暴露信息实际上是墙贴，这些帖子不会出现在用户的脸书信息收件箱中。

脸书还明确表示，绝对不存在隐私漏洞。人们看到的是旧的墙贴，而不是私人信息。一位发言人告诉 Constine:

> “我们看过的每一份报告，我们都回去检查过了。我们还没有看到一个被证实的(私人信息被曝光的)报道。很多困惑是因为在 2009 年之前，墙贴上没有赞和评论。人们拿着墙贴来来回回，而不是进行对话(在单个墙贴的评论中)。]"

脸书还告诉 Constine，有技术障碍阻止私人信息作为墙贴出现。"这两个系统是完全分开的。"

此前脸书发表声明:

> “一小部分用户在他们误认为是私人信息的内容出现在他们的日程表上后提出了担忧。我们的工程师对这些报告进行了调查，发现这些消息是在用户个人资料页面上一直可见的旧帖子。脸书对没有侵犯用户隐私感到满意。"

今天早上，我们第一次从读者的电子邮件中听到这方面的报道。关于这一假想问题的报道首先发表在法国报纸上，如 [*《世界报》*、](https://web.archive.org/web/20230221203204/http://www.lemonde.fr/technologies/article/2012/09/24/messages-prives-rendus-publics-et-plongeon-boursier-pour-facebook_1765038_651865.html) [*《晨报》*](https://web.archive.org/web/20230221203204/http://www.lematin.ch/high-tech/web/Facebook-publie-les-messages-prives-des-membres/story/11015793) 和 [*《法国地铁报》*](https://web.archive.org/web/20230221203204/http://translate.googleusercontent.com/translate_c?depth=1&hl=en&ie=UTF8&prev=_t&rurl=translate.google.com&sl=auto&tl=en&twu=1&u=http://www.metrofrance.com/high-tech/facebook-d-anciens-messages-prives-publies-sur-la-timeline/mlix!MDk3LU0hjxIQ/&usg=ALkJrhiss0HsNxWyk-4YIdSV9Uaa0aPrjA) 。

今天这成为一个问题的原因可能是因为时间轴的全球推广。第一个担心旧信息潜在曝光的人来自法国——脸书的一位发言人表示，这是时间轴推送给所有用户的最近的地方。

Constine 指出，很难记住脸书的工作方式或我们以前使用它的方式。如果没有评论，人们会通过交换墙贴来进行对话，任何脱离上下文的帖子都可能看起来像是私人信息。在每个人的同事、老板和奶奶都有脸书账户之前，我们可能也没有对我们张贴到墙上的内容如此小心——我知道，当我今天回头看时，看到我张贴在别人墙上的信息时，我个人感到很惊讶。四年前，脸书感觉像是一个不同的、更舒适的地方。

但是不要担心，确保你的朋友知道真相。脸书的私人信息没有泄露。
