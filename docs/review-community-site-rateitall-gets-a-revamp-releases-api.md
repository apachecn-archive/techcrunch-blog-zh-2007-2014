# 审查社区网站费率全部得到改造，发布 API

自称为“分布式 Yelp for everything”的消费者评级和评论网络 RateItAll ，今天正在进行改造，推出一系列新的社交功能，包括评论订阅和产品兼容性测试。

RateItAll 最初成立于 1999 年，在互联网泡沫破裂后于 2001 年关闭，随后于 2007 年重新启动。它允许用户对从音乐到旅游到食物的各种话题进行评分。该网站目前拥有 350，000 多名会员，而【Quantcast 和 Compete 都显示出良好的流量增长模式(该公司自称上个月的访问量为 130 万，自 2008 年 6 月以来增长了 44%，去年的独立访客总数为 1，130 万)。

新的外观和新的功能在过去的 5 个月里一直在工作，在旧金山的公司[于去年 9 月筹集了第一轮](https://web.archive.org/web/20221007103426/http://www.beta.techcrunch.com/2008/06/18/rateitall-gets-800k-for-yelp-for-everything/)[资金](https://web.archive.org/web/20221007103426/http://www.crunchbase.com/company/rateitall)达 140 万美元后，开发工作开始了。你现在可以创建一个评论流，它非常类似于脸书的新闻源，但是是按类别/关键词或个人网络进行评论的。该服务还增加了一个名为“测验”的社交兼容层，让你可以将自己的评分和评论与朋友或 RateItAll 社区其他成员的评分和评论进行比较。

该服务最受欢迎的新增功能是[读/写 API](https://web.archive.org/web/20221007103426/http://rateitall.pbwiki.com/) (该公司还为社交网络和博客提供 RSS 源和小工具)和“全球添加”功能，该功能允许用户将任何产品、企业、品牌、个人和/或地点上传到 RateItAll 数据库，供其用户群讨论。

作为基于 Google AdSense 的收入分享计划[的一部分，RateItAll 向评论者支付 50%。](https://web.archive.org/web/20221007103426/http://code.google.com/intl/nl/apis/adsense/docs/casestudies/rateitall.html)
