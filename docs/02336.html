待售:德国的“TechCrunch”

我们已经得到了很多关于此事的提示，德国媒体也疯狂地报道了这件事，所以，如果你曾经想买一个德国博客，现在是时候把你的钱投在你的鼠标上了。因博客 Basic Thinking 而在德语国家出名的Robert Basic，已经将网站放在易贝上出售，据说是因为他想从头开始。

Basic Thinking是一个技术博客，它实际上被一些人称为“德国TechCrunch ”;我们很荣幸，但我们不确定，因为我们没有跟上它，我们的德语有点生疏。Basic对拍卖清单上的博客收入非常直接:2008年，他通过展示和文本链接广告获得了约37，000€毛收入。他一直没有在网站上积极销售广告，所以他相信这可以带来更多的收入，特别是考虑到他作为德国ICT博客的地位，以及他的谷歌页面排名可以通过简单地删除文本链接广告从4恢复到以前的6。

基本思维的访问者数量并不多，但考虑到利基市场和有限的(仅限德语)范围，他们已经足够体面了:据报道，上个月该网站接待了185，000名独立访问者，提供了254，000次页面浏览。2008年的独立访问者总数不到250万，页面浏览量为360万。嵌入的图表显示了该网站自2005年启动以来的增长情况(它实际上是在2003年启动的，但已经中断了一段时间)。

那么，你还在等什么？

竞标将在大约六天后结束，目前桌面上的报价刚刚超过€20，000英镑(Basic希望价格在10，000到100，000英镑之间，所以我们很想知道它将在哪里结束)。
