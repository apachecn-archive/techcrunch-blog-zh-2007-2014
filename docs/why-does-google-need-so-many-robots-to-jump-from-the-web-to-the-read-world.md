# 谷歌为什么需要这么多机器人？从网络跳到现实世界

Google [为什么需要机器人](https://web.archive.org/web/20230329150227/https://techcrunch.com/2013/12/14/google-buys-boston-dynamics-creator-of-big-dog/)？因为它已经控制了你的口袋。移动市场，除了可穿戴设备的缓慢崛起，已经饱和。世界上有数百万部手机，每一部都连接到互联网，大多数运行 Android 或 iOS。除了对表单的增量更新，未来十年移动领域将不会有什么创新。

然后是玻璃。这些设备通过让我们成为载体，将网络带到了现实世界。谷歌已经在我们的小屏幕上走在了我们的前面，但谷歌眼镜让我们成了被俘虏的观众。通过依赖谷歌的数据进行日常互动、地图绘制和餐厅推荐——更不用说我们的一举一动都被数字化了——我们成为了历史上最好的谷歌消费者。但这仍然不够。

谷歌受到了限制，因为找不到更好的词，肉。我们不善于探索，也不善于收集数据。我们倾向于每天遵循相同的路径，就像蚂蚁一样，我们很少远离巢穴。谷歌是一家数据公司，需要的数据远远超过人类单独能够收集的数据。那么，机器人将在未来几十年里成为许多令人印象深刻的壮举的驱动力，包括太空探索、改进的测绘技术和制造工作空间的巨大变化。

像 Baxter 这样的机器人将取代数百万昂贵的人类——我怀疑这一举动将引发制造业失业率上升的问题——像制造业巨头 T2 富士康 T3 这样的公司正在快速投资机器人技术。无人驾驶飞机，无论是人类控制的还是自主的，都是我们感官的真正延伸，将我们放置在远离大本营的地方，并让我们随时了解情况。当我们生病时，家庭佣工很快就会[把我们从床上抱起来](https://web.archive.org/web/20230329150227/http://www.pcadvisor.co.uk/videos/3295217/video-japanese-care-robot-can-lift-patients-from-bed-to-wheelchair/)，帮我们[打扫卫生](https://web.archive.org/web/20230329150227/https://techcrunch.com/tag/iRobot)，并在我们生命的最后时刻帮助我们。更小的硬件项目将帮助我们[减肥](https://web.archive.org/web/20230329150227/https://techcrunch.com/2012/10/18/meet-autom-the-quantified-self-robot-that-is-your-next-diet-coach-made-by-pch-and-intuitive-automata/)和[巡逻我们的街道](https://web.archive.org/web/20230329150227/http://news.cnet.com/8301-11386_3-57614255-76/this-crime-predicting-robot-aims-to-patrol-our-streets-by-2015/)。今天没有投资机器人技术的科技公司将在未来十年发现自己远远落后于形势。

这就是谷歌需要机器人的原因。他们将把公司放在人机交互的最前沿，就像 Android 把他们放在数百万眼球面前一样。许多专家认为，谷歌没有理由在安卓还年轻的时候就成立移动部门。他们错了。这些看似不靠谱的机器人实验也是如此。

谷歌收购波士顿动力公司和其他七家机器人公司，是为了让它能在我们的城市里跑一千只四足大狗吗？没有，但我可以看到他们使用 [BD 的 PETMAN](https://web.archive.org/web/20230329150227/http://www.bostondynamics.com/robot_petman.html) ，一种可以在崎岖地形上行走和奔跑的双足机器人，以帮助绘制难以到达的区域。它也可能成为现实世界的 Google Now，以助手的形式出现在我们身边，整天跟着我们，让我们保持正轨，帮助我们完成任务，并在我们无法同时出现在两个地方时成为我们的化身。谷歌越能调节我们的日常体验，它就变得越有价值。

需要更多证据吗？跟着钱走。机器人是一个大行业，分析师估计波士顿动力公司在未来几年可能会成为一家价值 50 亿美元的公司。有了合适的合同和合适的产品组合，谷歌目前的机器人大军几乎任何成员都可以打入几乎任何市场，从大规模的消费机器人到制造业的大规模安装——更不用说那些利润丰厚的 DARPA 合同了。

今年我们会看到 RoboGooglers 在帕洛阿尔托闲逛吗？不会吧。太早了。但凭借谷歌司机(运行该公司自动驾驶汽车的软件)的一点聪明，以及一些更好的双足机器人设计，我可以在十年内看到谢尔盖和拉里站在他们的机器人助手旁边。现在他们要做的就是让它们变得有知觉。
