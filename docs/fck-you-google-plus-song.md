# “去你的，谷歌+”，一首关于 YouTube 新评论的可爱歌曲

一些 YouTubers 用户对因评论而被迫进入 [Google+感到不满，而](https://web.archive.org/web/20221209173052/https://beta.techcrunch.com/2013/11/06/youtube-starts-rolling-out-its-new-commenting-system-based-on-google/)[的一个女孩](https://web.archive.org/web/20221209173052/http://www.youtube.com/watch?v=LTq8TrA3hb4)以可以想象的最可爱、最亵渎的方式表明了立场。“你毁了我们的网站，还称之为整合/我写这首歌只是为了发泄我们的挫败感/去你的，Google Plusssssss！”

就我个人而言，我认为转向 Google+评论是[的一个巨大胜利](https://web.archive.org/web/20221209173052/https://beta.techcrunch.com/2013/09/25/google-plus-youtube/)。这将极大地阻止欺凌和钓鱼，把粪坑变成文明的讨论。它还可以让 YouTube 根据人气对评论进行排名，并向你展示来自朋友和名人的最相关的评论。

但是改变是艰难的，尤其是对于情绪化的孩子。还记得“学生反对脸书新闻”吗？750，000 人，也就是当时脸书用户群的近 10%，抗议该功能在 2006 年的推出。现在，它是互联网上最受欢迎的内容之一，也是脸书上瘾的关键。

YouTube 一直以来都有自己的评论系统。这是基本的，已经成为同性恋恐惧症和种族主义的避风港，但有些人就是不想适应新事物。

Emma Blackery 对 YouTubeland 中的强制过渡和其他问题有一些好的观点。
