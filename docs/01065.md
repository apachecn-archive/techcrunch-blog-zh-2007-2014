# 谷歌移动地图在没有全球定位系统的情况下精确定位您的位置|技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/11/28/google-mobile-maps-pinpoints-your-location-without-gps/amp/>

[<amp-img class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" src="https://web.archive.org/web/20230306165911im_/https://techcrunch.com/wp-content/uploads/2007/11/google-maps-logo.png" alt="google-maps-logo.png" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230306165911/https://techcrunch.com/wp-content/uploads/2007/11/google-maps-logo.png "google-maps-logo.png")谷歌一直在为其地图快速添加功能。昨天，基于网络的地图出现了[地形和维基风格的合作](https://web.archive.org/web/20230306165911/https://techcrunch.com/2007/11/28/google-earth-heading-towards-extinction/)。如今，它更新了手机地图，通过三角测量手机发射塔来定位你的位置(或者如果你的手机上有 GPS，它会使用它，但是今年只有 15%的手机支持 GPS)。我刚刚把“我的位置”应用程序下载到我的黑莓手机上(没有全球定位系统)，它把我定位到了离我在曼哈顿的位置不到一个街区的地方。我是地图上一个闪烁的小蓝点。尽管如此，如果我在办公室里走动，它会拿起另一套手机信号塔，把我带到近六个街区以外的地方。哦，等等，现在我几乎完全在正确的街区了。现在我又在拐角处了。至少它有合适的邻居——曼哈顿确实有很多手机发射塔。
