# Kyte 成为品牌的移动到网络视频平台(增加新投资者和实时移动流媒体)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/03/07/kytes-big-day%e2%80%94launches-kytecom-adds-live-mobile-video-streaming-new-facebook-app-and-new-investors/amp/>

[<amp-img class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" src="https://web.archive.org/web/20230121135634im_/https://techcrunch.com/wp-content/uploads/2008/03/kyte-logo.png" alt="kyte-logo.png" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230121135634/http://www.crunchbase.com/company/kyte)Kyte 首席执行官丹尼尔·格拉夫(Daniel Graf)向将手机变成视频分发平台又迈进了一大步。“这是 Kyte 的大日子，”他告诉我，“这是我们自去年四月推出 beta 版以来最大的一次发布。”到目前为止，Kyte 允许人们通过将视频从他们的手机上传到各种小工具和 Kyte.tv 来创建他们自己的个人电视频道。今天，Kyte 增加了来自手机和网络摄像头的实时视频流，通过你的个人 Kyte 频道播放，并存档供以后观看。(在这里注册[私人测试版](https://web.archive.org/web/20230121135634/http://www.kyte.com/platform/mobile_producer_beta_form)。小心， [Justin.tv](https://web.archive.org/web/20230121135634/http://www.justin.tv/) 和 [Ustream](https://web.archive.org/web/20230121135634/http://www.ustream.tv/) )。它还从 Steamboat Ventures 和瑞典移动运营商 TeliaSonera 那里额外筹集了 610 万美元，以总计 2110 万美元完成了其 B 轮融资。

但最重要的是，Graf 正致力于使 Kyte 成为音乐家、媒体公司和移动运营商的平台。他认为 Kyte 更多地被知名人士和媒体公司用来制作初始内容，然后通过 Kyte 播放器被观众分享和分发。为此，他刚刚推出了[Kyte.com](https://web.archive.org/web/20230121135634/http://www.kyte.com/)，作为品牌合作伙伴(包括博主)进入 Kyte 平台的网站。所有四大唱片公司都在使用 Kyte 来创建品牌播放器，这些播放器可以被 widget 化并在整个网络上分发。(下面是我们的，展示了我用格拉夫的手机拍摄的对他的采访)。

格拉芙说:“哪儿都没有写‘Kyte’。”“这就像一个微型网站。它可以通过病毒传播。粉丝们觉得真的和它有联系。”他们也能有所贡献。Kyte 播放器有一个“制作”按钮，允许粉丝将他们自己的视频上传到频道。格拉芙为 TechCrunch 制作了上面的这个(这是继 50 美分之后的第二个)。在接下来的几个小时里，您可以使用网络摄像头添加自己的视频评论(我们将对此进行监控，因此请保持干净)。

50 Cent 已经对 Kyte 进行了大约三个月的测试，他的 Kyte 频道已经在 10，000 个网站上有超过 400 万次的观看。这位 Kyte 玩家在他的网站上是最重要的。他经常把自己和他的摄制组拍摄的视频片段放在手机上。昨晚，他通过 Kyte 播放器首映了他的视频“机械师”，粉丝可以将它嵌入到自己的页面上。这是他在网上保持对其品牌控制的方式，即使人们抓取视频并放到其他网站上。Kyte 计划在 4 月份在其播放器中打开广告，并与 Kyte 频道的制作人分享这些收入。

Kyte 还让其合作伙伴能够在大约五分钟内将他们的 Kyte 播放器变成脸书应用程序。这是 50 美分的 T1，这是 T2 的 TechCrunch。它还为合作伙伴网站提供了一个 [API](https://web.archive.org/web/20230121135634/http://wiki.kyte.com/wiki/API:Introduction) ，以便与 Kyte 播放器和服务进行更深入的整合。它甚至集成了 Twitter。最后，Kyte 正在开发一键式移动应用程序，使从手机上传视频变得更加容易。一个这样的应用程序已经在诺基亚 60 系列手机上进行私人测试。
