# Bottlenose 将实时电视和广播数据添加到其社会趋势监测中

帮助大品牌发现新兴趋势的初创公司 Bottlenose 如今正在向社交媒体之外扩张，增加了广播分析，让客户可以看到电视和广播数据。

为了实现这一目标，瓶鼻公司与媒体监测公司[合作。根据 Bottlenose 联合创始人兼首席执行官 Nova Spivack 的说法，Critical reference 每分钟摄取和转录 40 小时的视频和音频节目，然后 Bottlenose 可以将 Critical reference 的数据视为另一个文本消防水带进行分析。](https://web.archive.org/web/20230316161016/http://www.criticalmention.com/)

例如，该公司表示，客户可以使用这些数据来跟踪一家公司在直播中的提及率，了解不同主题之间的关系，跟踪公司或个人的情绪，并将电视对话与网络对话相关联。反过来，这种分析可以用于广告购买等活动。

Spivack 说，Bottlenose 与众不同的一点是能够在一个产品中查看社交媒体和广播数据(因此能够连接我上面提到的电视和网络对话)。另一个事实是它是实时的——Spivack 说，大多数用相同术语描述自己的竞争对手实际上是指他们实时摄取数据，然后进行分析，而 Bottlenose 的分析发生在摄取的时候。

更广泛地说，他认为 Bottlenose 正在做一些与社交监听服务“非常不同”的事情，社交监听服务专注于帮助公司看到他们在社交媒体上被提到的位置，然后对投诉、问题等做出回应。

“我们所做的是在海量数据中发现趋势，”Spivack 说。“我们实时了解什么是重要的。”

最后，他透露，自从去年夏天我们报道了[瓶鼻公司 360 万美元的融资](https://web.archive.org/web/20230316161016/https://techcrunch.com/2013/07/30/social-media-dashboard-bottlenose-raises-3-6m/)以来，该公司已经筹集了更多的资金，使其总融资额达到约 600 万美元。
