# 谷歌现在是 Android.me 的骄傲所有者

这不完全是谷歌罗拉大小的新闻，但我刚刚注意到[谷歌](https://web.archive.org/web/20230204115415/http://www.crunchbase.com/company/google)最近获得了 Android.me 域名的所有权(不值得链接，因为它目前还没有被使用？).

现在，谷歌可能只是要求——或强迫——前所有者转让这个引人注目的域名的所有权，所以这并不一定意味着它将推出一个同名的产品(此外，他们已经拥有 android.com，所以何必呢？).

出于这个和其他原因，我认为谷歌不太可能计划让 Android.me 成为一项新服务的大本营，尽管我以前也曾被认为是错误的。不管怎样，有点意思。
