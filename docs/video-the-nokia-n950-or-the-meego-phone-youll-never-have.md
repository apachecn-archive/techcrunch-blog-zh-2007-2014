# 视频:诺基亚 N950，或者你永远不会拥有的 MeeGo 手机

[![](img/0be89ccefdad987964da7ec6160c0c25.png "N950")](https://web.archive.org/web/20221205093151/http://www.mobilecrunch.com/2011/06/22/video-the-nokia-n950-or-the-meego-phone-youll-never-have/)

当诺基亚 N9 的产品照片在周一泄露[时，有些事情让我们有点措手不及。不，这不是因为它运行的是 MeeGo——也不是因为这款设备时髦的橙红色。奇怪的部分？它没有键盘。](https://web.archive.org/web/20221205093151/http://www.mobilecrunch.com/2011/06/20/nokia-n9-pictures-meego/)

几个月来，谣言工厂周围的传言和泄漏将 N9 定为 MeeGo 驱动的设备，配有滑出式 QWERTY 键盘。事实证明，*在某些时候就是*N9。然而，在过去几周的某个时候，正式名称为 N9 的设备变成了 N950，诺基亚决定只向 MeeGo 开发者提供。
