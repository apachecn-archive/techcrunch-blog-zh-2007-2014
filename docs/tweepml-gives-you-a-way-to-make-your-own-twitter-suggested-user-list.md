# TweepML 为您提供了一种创建自己的 Twitter 建议用户列表的方法

Twitter 的推荐用户名单引发了很多争议。但是其背后的基本原理是合理的，即使执行并不合理:这是一种将用户捆绑在一起的方式，这种方式可能很有意思。一项名为“TweepML”的新服务将这个想法传递给大众。

基本上，如果你曾经在 RSS 阅读器中使用过 OPML 文件捆绑订阅源，你会立刻理解这个概念。TweepML 将你选择的一群 Twitter 用户捆绑在一起，并允许你与你选择的任何人分享这个捆绑包。当接收端的人装载包裹时，他们会跟随你建议的所有人。

一个很好的用例是个人博客。这里是 TechCrunch 的 [TweepML](https://web.archive.org/web/20230222201807/http://tweepml.org/TechCrunch/) 的例子。如果你点击那个链接，你会看到一群美国 TechCrunch 作家被选中让你关注。您也可以单独取消选中我们中的任何人。然后，您可以使用列表下方的 Twitter 凭据登录，并关注这些用户。奇怪的是，TweepML 选择不使用 Twitter 流行的 OAuth 认证系统，而是让你登录他们自己的网站。但它承诺，一旦它跟踪你选择的用户，它就会“永远忘记你的密码”

TweepML 背后的人 Marcelo Calbucci 创造了一些其他有趣的 Twitter 用户群，包括[美国参议员](https://web.archive.org/web/20230222201807/http://tweepml.org/US-Senators/)和 [Twitter 员工](https://web.archive.org/web/20230222201807/http://tweepml.org/Twitter-Team/)

这项服务 TweepML 今天将与几个合作伙伴一起发布，包括 Twitter 分组服务 [Twibes](https://web.archive.org/web/20230222201807/http://twibes.com/) 和 Twitter 语音服务 [Chir.ps](https://web.archive.org/web/20230222201807/http://chir.ps/) 。他们还声称得到了 OneRiot、Gnip 和 Gist 等人的支持。这是一个好主意，以一种足够简单的方式执行，它就可以工作。

TweepML 格式承诺是可扩展和开放的，因此其他人也可以免费使用它。它基于简单的 XML，Calbucci 希望网站通过实现它并包含它来使它成为一个标准。tml 文件供访问者访问。
