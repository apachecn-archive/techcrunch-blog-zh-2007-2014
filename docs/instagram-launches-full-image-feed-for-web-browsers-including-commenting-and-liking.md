# Instagram 为网络浏览器推出了完整的图片馈送，包括评论和喜欢功能

Instagram 刚刚宣布其完整的 feed 已经发布到网络上，这意味着用户不再需要通过专用的移动应用程序来查看他们的所有内容，并参与围绕发布的照片的对话。

基于网络的 feed 有桌面和移动优化版本，这意味着如果你使用的是黑莓之类尚未被官方应用程序增色的东西(当然，不包括发布照片的功能)，你就已经准备好了。Instagram 在很长一段时间内一直只提供移动服务，但这不仅仅是简单的调情。

[![Capture](img/ee5982959bb13d8659114b7c2cf61803.png)](https://web.archive.org/web/20221006133100/https://beta.techcrunch.com/wp-content/uploads/2013/02/capture1.png)

在网络上，它看起来基本上和你在任何一个移动应用中看到的一样，正如你从上面的截图中看到的，移动版本本质上是一样的。您可以双击照片来喜欢它们(或者也可以使用方便的“心形”按钮)，并在线查看对话。新的网络视图也意味着你将能够可靠地在 iPad 或 Android 平板电脑上浏览(有第三方应用程序试图实现这一点，但大部分都不太好)，至少可以消费内容，如果不是主动创建内容的话。

*在 mobile Safari 中查看新的 Instagram web feed。*

Instagram 先发制人，在 2012 年 11 月推出基于网络的个人资料页面时就暗示了这一点。当时，该公司称“压倒性的用户需求”是推出该功能的原因，但没有给出任何关于未来网络产品的具体计划。

随着完整提要的推出，链接到提要中的特定图片变得更加容易。以前你必须进入用户的个人资料并找到你想要的图片，现在你可以简单地启动 Instagram.com，从任何帖子的“…”按钮下的“查看照片页面”链接中复制链接地址，并在周围分享。

尽管推出了大量的桌面产品，但 Instagram 的联合创始人凯文·斯特罗姆强调，这项服务的核心仍然是“在移动中、在现实世界中、实时地制作照片”，这就是为什么照片上传功能没有出现在基于网络的产品中。
