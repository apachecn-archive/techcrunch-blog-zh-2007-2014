# 快速社会准备在 SXSW 聚会(独家演示视频)

如果你是一家名副其实的群发短信初创公司，你必须为即将在奥斯汀举行的 SXSW 大会准备好一款新应用。GroupMe、Beluga(刚刚被脸书收购的[)、](https://web.archive.org/web/20221207042918/https://beta.techcrunch.com/2011/03/01/facebook-beluga/) [Kik](https://web.archive.org/web/20221207042918/https://beta.techcrunch.com/2011/03/07/mobile-messaging-startup-kik-raises-8m-launches-group-chat-and-photo-sharing/) 、 [TextPlus](https://web.archive.org/web/20221207042918/https://beta.techcrunch.com/2011/03/03/textplus-groups-for-good/) ，以及像 [Yobongo](https://web.archive.org/web/20221207042918/https://beta.techcrunch.com/2011/02/21/yobongo-sxsw/) 这样的分支机构都将大展拳脚。为了不被超越， [Fast Society](https://web.archive.org/web/20221207042918/http://fastsociety.com/) 刚刚推出了一款全新的 [iPhone 应用](https://web.archive.org/web/20221207042918/http://itunes.apple.com/us/app/fast-society/id389925564?mt=8)和一款 [Android 应用](https://web.archive.org/web/20221207042918/http://fst.sc/android/)也将于今天晚些时候推出。

![](img/1423482800bab3b5bb2e3e99114347d0.png)

我最近拜访了纽约市狗补丁实验室的三人小组。首席执行官马修·罗森伯格给我演示了新应用程序(见上面的视频)。它不是简单地将群聊引入短信，而是利用推送通知将消息流引入应用程序本身，这很像 [GroupMe 的最新版本](https://web.archive.org/web/20221207042918/https://beta.techcrunch.com/2011/03/01/groupme-push-notifications-maps-foursquare/)。

这款应用比之前的版本有了很大的改进。您可以创建最多五个临时或永久群组，每个群组最多 25 人。这款应用可以让你聊天、在线添加照片、发起电话会议或分享你的位置。Fast Society 还提供了一些独特的功能，例如可以录制一段简短的音频“大声喊出来”，就像群组语音邮件一样，直接发送到流中。它还能让你在地图上区分你所在的位置和你想去的地方。

创建一个新的团队是很容易的“集会”功能，发送一个提示给现有团队中的每个人。你也可以通过 Twitter 或脸书分享一个邀请，然后只让你想一起出去玩的人进来(我知道，这可能会变得很刻薄)。还有一个存档功能，可以让你重温过去与朋友群发短信的时刻(都保存在服务器端)。

快速社会将如何突破[所有噪音](https://web.archive.org/web/20221207042918/https://beta.techcrunch.com/2011/03/07/sxsgah/)？首先，它向任何下载该应用的人提供从奥斯丁机场到 SXSW 的免费乘车[。这家初创公司面临着资金更充足的竞争对手——group me 筹集了 1000 万美元，Beluga 现在是脸书的一部分。到目前为止，Fast Society 只从 Eniac Ventures、Quest VC 和对冲基金投资者 Jim Pallotta 那里筹集了 27.5 万美元的种子资金。罗森博格说，他已经能够通过与短信聚合商签订协议来控制短信成本，该协议对无限量短信收取固定费用，而不是每条短信收取一分钱或更多。这应该有助于 Fast Society 经济高效地扩大规模。现在它要做的就是赢得 SXSW 和其他地方的群发短信者。](https://web.archive.org/web/20221207042918/http://fastsociety.com/sxsw/)
