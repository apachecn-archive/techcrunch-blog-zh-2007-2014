# 在照片应用淘金热中，iPhone 年度应用 Hipstamatic 搬到了旧金山

热门的 iPhone 应用软件 [Hipstamatic](https://web.archive.org/web/20230204121459/http://hipstamaticapp.com/) 背后的公司合成公司，这一年过得非常好。他们的付费应用不仅是今年最畅销的应用之一，几周前，苹果还授予了它“[年度应用](https://web.archive.org/web/20230204121459/https://techcrunch.com/2010/12/09/apple-top-apps-2010/)”的桂冠。现在，他们又以一条大新闻结束了这一年:他们要搬到旧金山去了。

首席执行官卢卡斯·艾伦·别克告诉我们，该公司实际上已经在旧金山的索玛区(TechCrunch 也在这里)买下了一整栋两层楼的建筑。显然，这个应用程序行业有些钱。该计划是利用位于[这里](https://web.archive.org/web/20230204121459/http://maps.google.com/maps?hl=en&q=74+langton+st.&um=1&ie=UTF-8&hq=&hnear=74+Langton+St,+San+Francisco,+CA+94103&gl=us&ei=m8sPTeDdL4z6sAPMvtWtAg&sa=X&oi=geocode_result&ct=image&resnum=1&ved=0CBQQ8gEwAA)的建筑，作为全球总部和零售/画廊空间，在那里为他们的社区举办活动。

此前，合成公司在芝加哥设有办事处，在纽约也有一些员工。但我们被告知，他们将从 2 月份开始全部搬往西部。他们目前正在翻修这座建筑。

别克和另一位联合创始人瑞安·多尔霍斯特(Ryan Dorhorst)实际上在 7 月份就搬到了湾区，过去六个月一直在这座城市的一个临时场所工作。鉴于人们对照片应用领域的兴趣激增，这一点以及更大的举措应该不会太令人惊讶。

虽然 Hipstamatic 可能摘得了年度应用的桂冠，但他们面临着来自总部位于湾区的 [PicPlz](https://web.archive.org/web/20230204121459/http://picplz.com/) 和 [Instagram](https://web.archive.org/web/20230204121459/http://instagram.com/) 等公司的激烈竞争。事实上，Instagram 刚刚[在 SoMa 地区也开设了他们的新办公室](https://web.archive.org/web/20230204121459/https://techcrunch.com/2010/11/17/instagram-twitter-office/)(在 Twitter 原来的办公室里)。

除了 Hipstamatic，Synthetic 还开发了另外两个基于照片的应用程序: [IncrediBooth](https://web.archive.org/web/20230204121459/http://incredibooth.com/) 和 [SwankoLab](https://web.archive.org/web/20230204121459/http://swankolab.com/) 。
