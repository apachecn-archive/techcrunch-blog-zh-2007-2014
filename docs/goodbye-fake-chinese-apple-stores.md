# 再见，假冒的中国苹果店

[彭博](https://web.archive.org/web/20230205044244/http://www.bloomberg.com/news/2011-07-25/china-finds-five-unauthorized-apple-stores-in-kunming-city-shuts-two.html)报道称，最近流行的[昆明地区山寨苹果店](https://web.archive.org/web/20230205044244/https://techcrunch.com/2011/07/22/the-variegation-of-the-species-fake-apple-stores-abound-in-china/)正受到中国当局的调查，该店和另一家店已被勒令关闭。

这些商店的所有者，包括一个叫于成的人(他声称他在帮苹果的忙)，多年来一直试图成为官方授权的零售商，他们说他们没有违反任何法律。

并非所有人都满意:

> “当局应该阻止这些山寨店随心所欲地经营和销售，”苹果授权代理商之一南方源的员工谢永林说。未经授权的商店“对我们的产品和品牌有负面影响。”

最搞笑的事？俞说苹果商店非常容易被复制，称其等同于学校的化学实验室。我们还必须记住，除了整个授权经销商的问题，拥有一个漂亮、明亮的环境，让潮人做很多事情并不完全违法。如果是这样的话，星巴克可以起诉西欧几乎所有的现代咖啡店。
