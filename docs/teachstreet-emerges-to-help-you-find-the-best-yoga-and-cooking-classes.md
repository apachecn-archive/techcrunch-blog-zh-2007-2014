# TeachStreet 帮助你找到最好的瑜伽和烹饪课程

 [](https://web.archive.org/web/20230217025419/https://techcrunch.com/wp-content/uploads/2008/04/class.jpg) 亚马逊和 JibJab 校友[戴夫·沙佩尔](https://web.archive.org/web/20230217025419/http://www.crunchbase.com/person/dave-schappell)将于明天某个时候推出他最新的创业公司[蒂奇街](https://web.archive.org/web/20230217025419/http://www.teachstreet.com/)。他们还将宣布第一轮融资:[来自 Madrona Ventures、贝佐斯探险公司和一些天使投资人的 225 万美元](https://web.archive.org/web/20230217025419/http://www.crunchbase.com/company/teachstreet)。

该公司是现实世界课程(烹饪、狗的服从、音乐课、交际舞、外语、高尔夫、瑜伽等)的一种叫嚣。)，允许讲师上传课程信息。用户可以查找可用的课程，并阅读和撰写对课程和教师的评论。

目前，该网站将支持广告。未来，教师街可能会向教师收取额外费用。Schappell 说，随着时间的推移，他们将变得特别擅长向用户推广优质服务，他们可以为此收费。他们还将能够向用户定向投放极具针对性的广告——例如，向寻找训狗课程的用户投放一本关于训狗的书。

该服务首先在西雅图地区推出，拥有 8 个主要类别和 70 个子类别的 25，000 个课程。然而，大多数搜索和浏览是通过标签完成的，标签允许创建几乎任何类别的类。

荷兰初创公司 Libersy 正在为现实世界的服务创造一个分布式预订系统，这是一个间接的竞争。Schappell 说，目前，TeachStreet 不会直接提供预约和日历服务，但这是他们将来会增加的一项功能。
