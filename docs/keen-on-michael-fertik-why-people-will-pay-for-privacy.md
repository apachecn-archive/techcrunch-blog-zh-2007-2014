# 热衷于…迈克尔·费提克:为什么人们会为隐私付费

人们会为网络隐私付费吗？是的，他们会的——至少根据[迈克尔·费蒂克、](https://web.archive.org/web/20230204204214/http://www.crunchbase.com/person/michael-fertik)[Reputation.com 的创始人兼首席执行官](https://web.archive.org/web/20230204204214/http://www.reputation.com/)、新型在线隐私生态系统的早期领导者之一的说法。事实上，Fertik 认为隐私是网络经济中的下一件大事——是对雷德·霍夫曼无孔不入的个人数据的 Web 3.0 经济的必要解毒剂。

本周早些时候，Fertik 来到旧金山的 TechCrunchTV 工作室时告诉我，在我们的 Web 3.0 世界里，像 Reputation.com 这样的公司将控制权交还给了消费者。通过每年 75 美元的 MyPrivacy 和 MyReputation 等产品，Reputation.com 为消费者提供了一种相对实惠的方式，既能屏蔽 cookies，又能在我们日益公开的社交媒体世界中保护他们的在线声誉。

费蒂克很可能是对的。Reputation.com 已经在三轮融资中从包括 Kleiner 和 Bessemer 在内的一些蓝筹风险投资公司筹集了 2500 万美元。虽然 Fertik 在技术媒体上没有受到太多关注，但他经常在主流媒体上就隐私问题发表评论，甚至最近在美国广播公司(ABC)关于网络声誉受损的新真人秀节目中担任[“网络复仇者】](https://web.archive.org/web/20230204204214/http://abcnews.go.com/2020/We_Find_Them)的主角。

这是对费提克两部分采访的第一部分。明天，他将解释为什么数据是新的石油，以及为什么消费者是唯一没有从个人信息的雪崩中受益的人。
