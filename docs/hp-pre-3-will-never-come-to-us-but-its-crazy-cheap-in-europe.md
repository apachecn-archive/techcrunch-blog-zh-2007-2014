# 惠普 Pre 3 永远不会来找我们，但它在欧洲便宜得离谱

在惠普的第一次也可能是最后一次 webOS 大甩卖之后，来自大洋彼岸的消息说，他们也在大部分欧洲市场大打折扣[新推出的 Pre 3 智能手机](https://web.archive.org/web/20230204165730/https://techcrunch.com/2011/08/17/hp-pre3-makes-stealthy-entrance-into-the-european-market-u-s-to-follow/)。

虽然它最初在英国的售价为 299 英镑，在法国的售价为€349 英镑(两者都在 500 美元左右)，但惠普的降价使价格降到了非常合理的 75 美元*无 T5 合约。信不信由你，德国不会从降价中受益，因为所有的单元都已经售出。*

尽管这可能是一笔非凡的交易，但对国内 webOS 爱好者来说，这也带来了一点坏消息。惠普希望从头到尾控制 webOS 生态系统，如果没有 TouchPad，他们似乎认为没有必要在美国推出 Pre 3。真的很遗憾:Pre3 是惠普在智能手机市场上的最后一次机会，很多 webOS 的顽固分子或多或少都在耐心等待。

如果你口袋里有钱，心中渴望 webOS，那么当它“很快”上市时，你可以放心地进口它。不过要注意的是，尽管 Pre 3 的规格高于平均水平，但在覆盖范围方面还是会有一些瑕疵。T-Mobile 用户将被降级到边缘速度，美国电话电报公司将最多有部分 3G 覆盖。
