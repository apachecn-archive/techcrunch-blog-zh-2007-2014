# AskMarkets 为 Crunchbase 推出预测市场。谁要去死亡池？

希腊初创公司 [AskMarkets](https://web.archive.org/web/20221207200945/http://www.askmarkets.com/) ，让用户创建并参与任何类型的非现金预测市场，已经在 [CrunchBase](https://web.archive.org/web/20221207200945/http://www.crunchbase.com/) 为初创公司创建了一个[预测市场。](https://web.archive.org/web/20221207200945/http://techcrunch.askmarkets.com/)

该应用程序让用户押注任何特定创业公司被收购、获得额外资金或在下一年走向[死亡池](https://web.archive.org/web/20221207200945/http://www.beta.techcrunch.com/tag/deadpool)的机会。举个例子:用户说 [Animoto](https://web.archive.org/web/20221207200945/http://techcrunch.askmarkets.com/home/select_stock/323) 有 51.5%的机会被收购，25%的机会被资助，23.4%的机会死池。我买了一些“被收购”的股票，因为我喜欢这家初创公司专注于在竞争对手徘徊的时候把[一个产品做得完美](https://web.archive.org/web/20221207200945/http://www.beta.techcrunch.com/2008/11/11/animoto-makes-a-perfect-product-perfecter/)。

作为一家初创公司，我还为 TechCrunch 创建了自己的市场。是的，我在利用内部消息。
