# 哥伦比亚广播公司测试社交观察室:与陌生人一起观看节目，并在节目中交谈

![](img/5099941a186d9f86e23284b5a1466026.png)
 [一直在测试](https://web.archive.org/web/20221005195718/http://labs.cbs.com/)[新高清流媒体](https://web.archive.org/web/20221005195718/http://www.beta.techcrunch.com/2008/03/27/cbs-testing-high-def-streaming/)产品的哥伦比亚广播公司实验室，也推出了一款名为社交视听室的新产品的实验室版本。这个想法是，你出现在网站上，选择一个正在播放的节目，和你的朋友或其他人一起看。可以评论，LOL 或者考积分。

[链接是直播的](https://web.archive.org/web/20221005195718/http://labs.cbs.com/viewing_room/)，但是我现在看不到任何实际的表演。

鉴于点播电视有多棒，我不认为这种东西会很受欢迎。有一个很大的例外:现场直播，根据定义，它是在特定时间播出，届时所有人都可以观看。除了直播内容，异步评论似乎也是一个不错的选择，就像 YouTube、Hulu 和 Joost 所做的那样。
