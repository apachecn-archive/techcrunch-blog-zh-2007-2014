经过长期的公开 alpha 和 beta 测试阶段后，面向 Mac 的 Tweetbot 终于以零售形式发布。著名的 iOS 和 Mac 开发者 Tapbots 的原生 OS X Twitter 客户端可以从 [Mac 应用商店](https://web.archive.org/web/20230125130000/https://itunes.apple.com/us/app/tweetbot-for-twitter/id557168941?ls=1&mt=12)获得，价格为 19.99 美元，有点令人惊讶。它值 20 美元吗？一千次是的。但不熟悉 Tapbots 工作的用户可能会拒绝为这种应用支付那么多钱——这正是 Tapbots 所期待的。

当我向 Tapbots 的联合创始人保罗·阿达德询问定价时，他说这故意高于公司最初的预期，主要是由于 [Twitter 的新 API 令牌限制](https://web.archive.org/web/20230125130000/https://techcrunch.com/2012/10/03/tapbots-founder-paul-haddad-on-app-net-netbot-and-what-twitters-new-api-means-for-tweetbot/ "Tapbots Founder Paul Haddad On App.net, Netbot, And What Twitter’s New API Means For Tweetbot")，这限制了 Tweetbot for Mac 客户端可以支持的最大用户数量。这一限制与 iOS 版 Tweetbot 所享受的限制是分开确定的，因此只能容纳更少的用户群。一旦令牌消失，Tapbots 将不再能够销售 Tweetbot，所以从这个意义上说，这是一个非常真实的、非营销意义上的限量版软件。

Tapbots 花费了大量的时间和精力为 Mac 开发 Tweetbot，他们打算在停止销售后继续支持它，因此价格更高。因此，尽管价格标签可能会让 Tapbots 和 Tweetbot 的新用户望而生畏，但忠实用户会更愿意付费，我不认为供应会持续那么久。

如果你没有像我一样在 Tweetbot 发布后使用过它的 alpha 和 beta 版本，我只能说它太棒了。自从 Tweetie 被 Twitter 收购后，我一直在寻找一个合适的替代者，但没有什么能与 Tweetbot 相提并论。这是我早上启动的第一个应用程序，也是我晚上关机时关闭的最后一个应用程序。它有实时流媒体，可定制的栏目，静音功能和其他一系列令人敬畏的东西。此外，它拥有周围最好看的设计。即使是 20 美元要价的两倍，Tweetbot 也是显而易见的。
