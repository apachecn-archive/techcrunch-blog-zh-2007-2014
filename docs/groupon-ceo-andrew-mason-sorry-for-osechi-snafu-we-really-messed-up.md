# Groupon 首席执行官安德鲁·梅森对 Osechi 的混乱表示抱歉:“我们真的搞砸了”

![](img/c416e7de25c73d691ade8c66187e7370.png)

社交商务巨头 [Groupon](https://web.archive.org/web/20230202231352/http://www.crunchbase.com/company/groupon) 的创始人兼首席执行官[安德鲁·梅森](https://web.archive.org/web/20230202231352/http://www.crunchbase.com/person/andrew-mason)今天通过视频[向日本顾客道歉](https://web.archive.org/web/20230202231352/http://info.groupon.jp/topics/20110117-515.html)。如果你像我们一样一直用鹰眼关注着 Groupon，你就会知道这与一桩出了大问题的新年交易有关。

梅森以他的幽默感而闻名，但是在这个视频中，他展示了他严肃的一面。

在给 Groupon 日本客户的一封邮件中，梅森承认该公司“真的搞砸了”这笔交易，并概述了它正在采取的措施，以重建其在日本和其他国家的形象。

梅森表示，他们过去曾成功推出 Bird Café的送餐业务，但在 Groupon 售出 500 张“osechi”套餐优惠券后，这家日本餐厅无法处理新年交易的订单量。梅森承认，许多菜肴送得太晚，而其他菜肴则“状况糟糕”。

几周前，由于这起广为人知的混乱事件，Bird Café的总裁水口健司(Kenji Mizuguchi)辞职，而 Groupon 则对顾客进行了赔偿，为这起事故道歉，并发放优惠券以恢复形象。

在下面的视频中，Groupon 的创始人继续说，该公司不会让类似的错误在未来发生。

他补充说，Groupon 已经开始对客户进行“容量规划”方面的教育，以避免此类问题在它开展业务的所有新国家发生。

像这样的失误当然凸显了 Groupon 面临的困难，以及它在未来将面临的困难，因为它试图管理其快速的全球扩张。

顺便说一句:如果你是一个企业主，请看看这个视频，学习一到两个教训，如何在你搞砸了的时候真诚地向客户道歉——你会反复地道歉。它是无价的。抱歉，如果我看起来像一个梅森粉丝，但他只是做对了很多。
