# 英特尔:第二代酷睿 CPU 出货量达到 7500 万，是公司历史上销售最快的 CPU

显然，英特尔已经成功了。PC 客户端事业部副总裁兼总经理 Mooly Eden 今天在 2011 年英特尔信息技术峰会上宣布了几件事情，其中包括，正如这篇文章的标题所述，该公司目前代号为 Sandy Bridge 的核心 CPU 是英特尔迄今为止销售最快的平台，已售出超过 7500 万片芯片。这一重大里程碑事件发生之际，英特尔的下一个重要产品超极本即将登陆零售商市场。

Eden 接着说，据英特尔估计，每天大约售出 100 万台电脑，新兴市场引领着这一增长。英特尔可能希望超极本能够刺激复兴，并让这个数字更高。

当然，更多的 Sandy Bridge 芯片将会在它的下一代上市之前上市。这些超便携笔记本电脑的第一代将采用 Sandy Bridge 芯片，进一步扩大销售记录，但以后的版本将采用基于新 22 纳米架构的第三代芯片。

英特尔在超极本上投入了大量资金。根据 Eden 的说法，第一阶段目前正在进行中，第一批产品将于今年假期上市，第二阶段将于 2012 年常春藤桥芯片上市。但在 2013 年，第三阶段将从节能的 22 纳米“Haswell”处理器开始，该处理器今天在舞台上演示，据说可以实现前所未有的电池寿命。到那时，Windows 8 将会被淘汰，两者很可能会在个人电脑的下一个阶段合作——或者至少我希望如此。
