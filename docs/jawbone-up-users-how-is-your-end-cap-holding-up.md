# Jawbone UP 用户:你的端盖怎么样了？

在工程师们将 Jawbone UP 带回[绘图板以纠正原始](https://web.archive.org/web/20221007140938/https://beta.techcrunch.com/2012/11/16/the-jawbone-up-is-back-but-is-it-better-than-ever/)版本中的大量故障之后，Jawbone UP 现在已经回归了一段时间。Jawbone 说，它比以往任何时候都更好，在很大程度上，这似乎是真的。古怪的同步曾经破坏了我的原创，尽管我从未经历过和其他人一样的灾难性失败。这一次，我最初注意到了一个问题，办公室里的其他人也看到了这个问题，其他一些用户已经报告了这个问题:端盖表面粗糙，容易剥落。看起来很便宜。

我们的两个员工团队已经在展示这个，都是在使用了几天之后。我们还在 Twitter 上发现了一篇[的报道，在 Jawbone 的官方论坛](https://web.archive.org/web/20221007140938/https://twitter.com/unravel101/status/279809494039535617)上[也有其他人抱怨同样的事情。这条线索给出了两个建议:永远不要摘下你的颚骨(lol ),购买颚骨官方商店提供的三包替换帽。两者显然都不太理想。一个理想的解决方案是从一开始就不要发生这种事情。](https://web.archive.org/web/20221007140938/http://forums.jawbone.com/t5/TROUBLESHOOTING/Coating-peeling-off/m-p/48794)

Jawbone 论坛上的许多人在 [other threads](https://web.archive.org/web/20221007140938/http://forums.jawbone.com/t5/WELCOME/Cap-issues-with-new-Jawbone-Up/m-p/48282) 上回应了对帽子的抱怨，也包括用户抱怨帽子在激烈的活动中松动。这是最初版本的一个问题，据说第二版做得更好。Jawbone 表示，你的第一顶替换帽是免费的，但之后，你必须去商店购买新的。据推测，这也不会使他们对这种剥落问题更加抵制。

这些 Jawbone UP 可能不会随机永久消亡，当然，这个问题主要是化妆品，但 UP 是 130 美元:在这个价格下，你不会期望它在打开包装后 48 小时内开始显示正常使用的磨损。我们已经联系了 Jawbone 寻求解释，但同时让我们知道你是否看到了这个问题，或者你的 UP 腕带有任何其他问题。
