# 欧盟对谷歌展开反垄断调查。微软的指纹无处不在。

欧盟对谷歌展开反垄断调查，调查三家总部位于欧洲的互联网公司的指控。毫不奇怪，这一调查的关键部分据说是关于搜索，这是由谷歌主导的大部分欧盟市场。[每日电讯报](https://web.archive.org/web/20230304184700/http://www.telegraph.co.uk/technology/google/7301299/Google-under-investigation-for-alleged-breach-of-EU-competition-rules.html)和[华尔街日报](https://web.archive.org/web/20230304184700/http://online.wsj.com/article/SB10001424052748704188104575084062149453280.html?mod=WSJ_business_whatsNews)有更多细节。

值得注意的是，这次调查只是初步的，不会有任何结果。但是至少有三家公司对谷歌提起了诉讼——值得注意的是，其中一家为微软所有。另一个是由微软部分资助的组织的成员。

这当然很有趣，因为微软十多年来一直卷入反垄断调查，在欧洲是(当然，之前在美国也是)。事实上，这整个[浏览器投票的事情](https://web.archive.org/web/20230304184700/https://techcrunch.com/2010/02/22/microsoft-ballot-screen/)是欧盟正在努力[确保微软公平竞争的结果。](https://web.archive.org/web/20230304184700/https://techcrunch.com/2009/12/16/eu-microsoft-antitrust/)

Ciao(现在叫做 Ciao Bing)是微软旗下的德国购物网站之一。微软在 2008 年以近 5 亿美元收购了它们。另外两家是英国购物网站 Foundem 和法国法律搜索网站 EJustice.fr。这三家公司都声称谷歌的搜索优势正在损害他们的业务。Foundem 声称谷歌在其网站上设置了“搜索惩罚”,使其陷入瘫痪。

正如《每日电讯报》所指出的，另一件有趣的事情是，“ *Foundem 是 ICOMP 的成员，这是一个接受微软*资助的互联网压力团体。”

这可能会变得很难看。

**更新**:这是[谷歌公共政策博客](https://web.archive.org/web/20230304184700/http://googlepublicpolicy.blogspot.com/2010/02/committed-to-competing-fairly.html)上的一篇文章。毫不奇怪，谷歌很快注意到 Ciao 和 Foundem 与微软的关系。

以下是一些关键摘录:

> 欧盟委员会通知我们，它已经收到了三家公司的投诉:一家英国价格比较网站， [Foundem](https://web.archive.org/web/20230304184700/http://www.foundem.co.uk/) ，一家名为 [ejustice.fr](https://web.archive.org/web/20230304184700/http://ejustice.fr/ "ejustice.fr") 的法国法律搜索引擎，以及微软的 [Ciao！来自 Bing](https://web.archive.org/web/20230304184700/http://www.ciao.co.uk/) 。虽然我们会就这些投诉提供反馈和其他信息，但我们相信我们的业务符合用户和合作伙伴的利益，并且符合欧洲竞争法。
> 
> 考虑到这些投诉会引起媒体的兴趣，我们想给他们提供一些背景资料。首先，搜索。foundem——一个名为 [ICOMP](https://web.archive.org/web/20230304184700/http://www.i-comp.org/) 的组织的成员，该组织部分由微软资助—[认为](https://web.archive.org/web/20230304184700/http://www.nytimes.com/2009/12/28/opinion/28raff.html?_r=2)我们的算法在我们的结果中降低了他们网站的排名，因为他们是一个[垂直搜索引擎](https://web.archive.org/web/20230304184700/http://en.wikipedia.org/wiki/Vertical_search)，因此是谷歌的直接竞争对手。ejustice.fr 的抱怨似乎呼应了这些担忧。

并且:

> 我们知道排名对网站有多重要，尤其是商业网站，因为更高的排名通常会带来更高的流量。我们也是第一个承认我们的搜索并不完美的人，但这是一个很难破解的计算机科学问题。想象一下，要在 14×12 屏幕的计算机屏幕上，在短短几毫秒内对像 iPod 这样的流行查询的 2.72 亿个可能结果进行排序。这是我们每天都要面对数百万次的挑战。

并且:

> 关于 Ciao！他们是谷歌的长期 AdSense 合作伙伴，我们一直保持着良好的关系。然而在微软[收购了](https://web.archive.org/web/20230304184700/https://techcrunch.com/2008/08/29/microsoft-says-hello-to-ciao-buys-european-shopping-site/) Ciao 之后！2008 年(改名为[)再见！从 Bing](https://web.archive.org/web/20230304184700/http://www.ciao.co.uk/) 开始，我们开始收到对我们标准条款和条件的投诉。他们最初[将他们的案子](https://web.archive.org/web/20230304184700/http://www.nytimes.com/2010/01/19/technology/19antitrust.html)提交给德国竞争管理局，但是现在已经移交给了欧盟。
> 
> 尽管每个案例提出的问题略有不同，但它们最终提出的问题是，谷歌是否在做任何扼杀竞争或伤害我们的用户和合作伙伴的事情。事实并非如此。如果有人有真正的担忧，我们总是努力倾听，我们努力把用户的利益放在第一位，并在市场上公平竞争。我们相信我们的业务实践反映了这些承诺。
