# 独家报道:ProfileBuilder 收购 ZingFu.com

[![profile-builder.jpg](img/8d504813f9609c79bd876cc2e8211688.png) ](https://web.archive.org/web/20230221203133/http://www.crunchbase.com/company/profilebuilder) * **更新**:这笔交易告吹了。辛福从未被收购。*

在线身份管理平台公司 [ProfileBuilder](https://web.archive.org/web/20230221203133/http://www.crunchbase.com/company/profilebuilder) 收购了拥有 60 万注册会员的社交网络资源网站[ZingFu.com](https://web.archive.org/web/20230221203133/http://www.zingfu.com/)。

总部位于密尔沃基的 ZingFu.com 提供“滑稽内容”，产品包括照片处理工具 Zing、宠物、“tomgatchi 风格的玩具”和滑稽卡片，就像标题所示。

ZingFu.com 发布了一个独家的 [TechCrunch 40 zany 卡部分](https://web.archive.org/web/20230221203133/http://www.zingfu.com/cardz.php?category=11)来庆祝这次收购；见下图。

这两个网站的整合将为新浪微博的用户“提供一个平台，进一步增强他们的在线影响力。”

ProfileBuilder 还与澳大利亚悉尼的一家提供在线视频会议的初创公司[Mebeam.com](https://web.archive.org/web/20230221203133/http://mebeam.com/)建立了合作关系。

根据协议，ProfileBuilder 和 Mebeam 将交换资源，建立共同交付的网站，为 ProfileBuilder 用户提供广泛的视频会议工具。

收购 ZingFu.com 的条款没有披露。

ProfileBuilder 是 TechCrunch40 大会的赞助商。
