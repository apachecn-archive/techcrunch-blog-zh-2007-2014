# 新兴城镇:自 2012 年初以来，旧金山的科技工作岗位增加了三分之一 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/08/27/san-francisco-tech-job-data/>

**更新完毕。繁荣时代重返旧金山的视觉证据遍布整个城市:公寓几乎很难找到，租金和房价飙升，豪华的新餐厅每天都在涌现，交通系统[挤满了人](https://web.archive.org/web/20221210034410/http://www.caltrain.com/about/news/Caltrain_Ridership_Hits_Historic_High.html)。**

现在，实际数据正在支持它。

根据旧金山评估记录员 Phil Ting 办公室发布的新数据，自 2012 年初以来，旧金山的技术工作岗位~~增加了近两倍~~增长了三分之一——从~~1.3 万到 4.4 万~~3.1 万到 4.4 万[。似乎所有的](https://web.archive.org/web/20221210034410/http://sfappeal.com/news/2012/08/sf-tech-job-numbers-triple-in-2012.php)[招聘活动](https://web.archive.org/web/20221210034410/https://beta.techcrunch.com/2012/04/17/a-sign-of-the-hiring-pocalypse/)，以及老牌公司和初创公司[从南湾](https://web.archive.org/web/20221210034410/https://beta.techcrunch.com/2012/07/09/its-official-pinterest-mayor-ed-lee-confirm-move-to-san-francisco-lee-opens-pinterest-acct-in-celebration/)搬到市区，在科技就业方面真的越来越多。

技术工作的首选社区是市场南部、南海滩和米申湾。据丁的办公室说，就业增长直接影响了房地产市场，推高了房价。

根据旧金山在线报纸《呼吁》今天发表的《海湾城市新闻》对这些数字的报道，丁说:

> “我们的城市是发展和吸引 21 世纪企业的典范，这些企业提供高收入的好工作。我们可以指望旧金山在技术、生命科学、生物技术以及吸引世界上最优秀和最聪明的工人的生活质量方面树立榜样。”

旧金山看到这种繁荣并非偶然。在市长李爱德的带领下，该市的政客们一直在积极吸引技术人员在市中心开店，而不是在历史上被称为“硅谷”的郊区其他城市，特别是由市长迈克尔彭博领导的纽约，也在寻求效仿。

***更新:*** *此前，本文报道，根据[丁俊晖办公室发布的新闻稿报道的信息，工作岗位增加了 3 倍。许多](https://web.archive.org/web/20221210034410/http://www.sfexaminer.com/local/2012/08/tech-jobs-sf-triple-first-half-2012)[当地出版物](https://web.archive.org/web/20221210034410/http://www.sfexaminer.com/local/2012/08/tech-jobs-sf-triple-first-half-2012)也对此进行了报道。事实证明，实际数据显示就业增长了三分之一——这仍然是一个显著的推动，但绝对不同于说他们增加了两倍。*
