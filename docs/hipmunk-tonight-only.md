# Hipmunk 通过其新的“仅今晚”功能 TechCrunch 增加了最后一分钟的酒店交易

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/08/01/hipmunk-tonight-only/>

希望在最后一刻预订酒店的旅行者应该能够从今天开始在旅游初创公司 [Hipmunk](https://web.archive.org/web/20220930172412/http://www.hipmunk.com/) 找到更好的交易，这要归功于一个名为“仅今晚”的新功能的推出。

该功能仅在 Hipmunk 的移动应用程序中提供，将包括酒店交易，在标准价格的基础上提供高达 60%的折扣。这些交易可以被视为 Hipmunk 的一般酒店搜索结果的一部分(在这里，您还应该看到提前 72 小时预订的酒店的稍微少一点的最后折扣)，或者在单独的“仅今晚”部分。

联合创始人兼首席执行官亚当·高德斯坦表示，该公司开始注意到，在 Hipmunk 上搜索酒店的人分为两个不同的群体——那些提前预订的人和那些在最后一刻寻找住宿的人。第二个群体规模很大(它占 Hipmunk 移动应用搜索的 50%以上)，但该公司直到现在还没有建立任何功能来使他们的特定体验更容易。

最后一分钟酒店交易的想法并不新鲜(事实上，这是初创公司 HotelTonight 的主要关注点[，但 Goldstein 说 Hipmunk 的方法不同于大多数交易提供商，因为它汇总了来自各种来源的交易，给用户提供了更广泛的选择。](https://web.archive.org/web/20220930172412/https://beta.techcrunch.com/2013/05/08/hoteltonight-snap-your-stay/)

“这基本上是一种商业模式的区别，”他说。“对于一家大型旅游公司来说，让他们的业务运转起来的是，他们是负责处理你的信用卡、收取你的佣金的商家。我们是一个元搜索网站，而不是一个旅游卖家。”

[![hipmunk deals](img/f9ff0836689576abb0e69c0013a11e20.png)](https://web.archive.org/web/20220930172412/https://beta.techcrunch.com/2013/08/01/hipmunk-tonight-only/deals-2/) 换句话说，Hipmunk 非常乐意从一些网站上找到交易，然后在你准备购买时将你转到这些网站上(尽管今年早些时候它已经[在某些列表上增加了从 Hipmunk 网站](https://web.archive.org/web/20220930172412/http://blog.hipmunk.com/hipmunk-news-book-hotels-directly-on-hipmunks-website-and-mobile-app/)直接预订酒店的功能)。

Goldstein 还表示，Hipmunk 提供了更好的选择这些交易的体验，因为它已经开发了更广泛的酒店搜索功能。例如，一个商务旅行者可以看到每个可能的酒店离他们预定的会议有多近。或者，休闲旅行者可以查看 Hipmunk 的社区热图，看看哪些酒店位于最理想的社区。

这一推出反映了 Hipmunk 今年更广泛的关注点，Goldstein 说这是为了改善酒店搜索:“我们希望尽可能成为最好的旅游网站，尽可能成为最好的旅游应用程序——不仅仅是酒店，还有航班和旅行的其他方面。但作为一家小型初创公司，今年对我们来说最重要的项目是酒店。”他补充说，从商业角度来看，酒店预订比航班更有利可图。

然而，正如他上面的声明所表明的，Hipmunk 肯定不会放弃它最初以其闻名的航班搜索。事实上，它今天还推出了一个新的票价提醒功能，用户可以通过电子邮件或推送通知获得他们想要监控的路线上新机票的提醒。

**更新:**帖子的措辞已被调整，以明确这仅适用于 Hipmunk 的移动版本，而不是其网站。
