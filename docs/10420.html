原文：https://web.archive.org/web/http://techcrunch.com/2011/10/12/ibm-announces-new-smartcloud-services-partnership-with-nirvanix/

IBM宣布与Nirvanix合作推出新的智能云服务

IBM今天推出了新的IBM SmartCloud服务，扩展了面向企业的云计算服务。这些新产品包括SmartCloud应用服务、IBM面向企业的新PaaS、名为SmartCloud Foundation的硬件和软件组合，以及SmartCloud生态系统，其中包括面向IBM合作伙伴和ISV的新服务。

该公司计划到2012年底支持大约2亿用户。

IBM表示，它已经将SAP、ERP和所有数据库应用程序移植到其SmartCloud平台上。借助其新服务，it部门现在可以自动化与在云中运行SAP相关的劳动密集型任务，尤其是SAP克隆、更新和修补。在的一些案例中，IBM称根据任务的不同，节省的时间可能从几个月或几周到几天或几小时不等。

今天还宣布了与云存储公司Nirvanix的新合作关系。展望未来，Nirvanix的云存储技术将集成到IBM的SmartCloud企业存储服务中，允许客户在世界任何地方上传任何大小的文件，并从任何地方访问它们。存储即服务产品可以支持数百万用户、数十亿对象和数十亿字节的数据。

与此同时，IBM的新平台即服务产品smart cloud Application Services(SCAS)将于今年(第四季度)推出测试版，提供企业级安全性、开放式Java和跨平台支持(不受供应商限制),以及一套全面的应用基础设施和托管服务，以实现应用到云的开发和部署。它将在IBM的SmartCloud Enterprise和Enterprise+上运行，这是一种企业级基础设施即平台，专门用于按照承诺的SLA(服务级别协议)运行企业工作负载。IBM SmartCloud Enterprise+已在美国上市，并将于2012年底在全球部署。

最后，全新的SmartCloud Foundation私有云解决方案系列旨在帮助公司快速设计和部署私有云环境，包括SmartCloud Entry、预打包的私有云软件，以及smart cloud配置和监控软件。
