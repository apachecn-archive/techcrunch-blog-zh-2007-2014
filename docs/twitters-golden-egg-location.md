# Twitter 的金蛋:位置 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/01/05/twitters-golden-egg-location/>

***编者按** : [塞米·沙阿](https://web.archive.org/web/20230203062739/http://about.me/semilshah)是一位对数字媒体、消费互联网和社交网络感兴趣的企业家。他住在帕洛阿尔托，你可以在推特上关注他*

当“新 Twitter”[在 2010 年推出](https://web.archive.org/web/20230203062739/https://techcrunch.com/2010/09/14/twitter-event/)时，伴随着新平台的[有趣的视频演示](https://web.archive.org/web/20230203062739/http://twitter.com/newtwitter?utm_source=twitterfeed)，用户可以在网页中查看线程对话和多媒体。新的 Twitter 提供了一个界面，允许用户留在原生网站上([www.twitter.com](https://web.archive.org/web/20230203062739/http://www.twitter.com/))，而不是链接和打开新标签。此外，Twitter 为 iPhone 和 iPad 发布了漂亮的新应用，这些应用也提供了将用户留在原生平台上的方法。当然，他们创造的部分动机是，不断地链接和远离网站获取信息会产生摩擦，并为其他人创造机会，在 Twitter 的大规模背后创造不同的、更专注的体验。

然而，事实上，Twitter 的核心优势——将帖子限制在 140 个字符以内——为其他网站和应用创造了机会，围绕源自 Twitter 的活动吸引和俘获用户。这些新网站和服务利用 Twitter 的分销渠道，免费，回报很少或没有。

根据我的观察，最常见的推文倾向于分享与以下一个或多个相关的信息:

*   分享图片/多媒体
*   介绍/认可
*   链接内容
*   提出问题
*   娱乐与体育评论
*   时事(技术、政治、金融)
*   共享位置
*   分享情绪/心情

2010 年，这些常见的推文行为为新企业家创造了利用的机会。在图片方面， [Instagram](https://web.archive.org/web/20230203062739/http://www.instagr.am/) 凭借其一系列滤镜和分享功能成为热门，而用户也可以选择将他们的应用内照片放在外部博客网站上，如 [Posterous](https://web.archive.org/web/20230203062739/http://www.poserous.com/) 。对于介绍， [Hashable](https://web.archive.org/web/20230203062739/http://www.hashable.com/) 提供了一项服务，用户可以通过这项服务轻松交换联系信息，并跟踪自己参与了哪些介绍。在推特上提出问题的行为已经被 Quora 捕获。Twitter 用户对娱乐和体育的迷恋促使 Twitter 在洛杉矶地区雇佣了自己的员工( [@omid](https://web.archive.org/web/20230203062739/http://www.twitter.com/omid) )(并且[为贾斯汀比伯](https://web.archive.org/web/20230203062739/http://twitter.com/#!/dcurtis/status/23192425566)专用单独的服务器)。在政治方面，Twitter 已经从 DC 环城路招募了关键人员，像 [Votizen](https://web.archive.org/web/20230203062739/http://www.votizen.com/) 这样的初创公司可能会利用这个平台来帮助组织在线政治利益。体育迷有像 Fanvibe 这样的网站。对于时事和信息，Twitter 有助于推动大量科技博客、新闻博客和金融博客的流量，这些博客本身就是真正的生活方式业务。我敢肯定，根据 Twitter 用户的情绪和情感表达来收集他们并与他们互动是一个商机，尽管这可能会有一些健康和隐私问题。

但是，位置呢？Twitter 在定位方面有真正的、可持续的、长期的商业模式吗？

在关注了 Quora 上的两条独立线索后，我开始思考这个问题:(1)“有哪些创造性的方法可以将 Twitter 货币化？”([链接](https://web.archive.org/web/20230203062739/http://www.quora.com/Twitter-1/What-are-some-creative-ideas-to-monetize-Twitter))和(2)“用户在基于位置的 app 中想要什么？”([链接](https://web.archive.org/web/20230203062739/http://www.quora.com/What-do-users-want-on-a-location-based-app?q=What+do+users+want+location))。一方面，大多数人都从 Twitter 中受益匪浅，包括我自己。我喜欢这个产品。我希望它能成功并永远持续下去。我甚至会付费使用它。然而，许多人想知道 Twitter 将如何转变为一个真正的企业，尽管它最近的估值为 37 亿美元，并拥有全球足迹。另一方面，用户可以通过[脸书 Places](https://web.archive.org/web/20230203062739/http://www.facebook.com/places) 、 [Foursquare](https://web.archive.org/web/20230203062739/http://www.foursquare.com/) 或者使用 [Gowalla](https://web.archive.org/web/20230203062739/http://www.gowalla.com/) 分享他们的位置，这让他们可以签到并全面广播。用户感觉到“位置”将会变得很大，但他们不知道如何变大，有些人已经厌倦了各种不同的方式。

不清楚基于位置的服务将赢得什么，加上 Twitter 对长期商业模式的探索，为 Twitter 提供了一个绝佳的机会，可以暂时放弃其他领域(如图片)，专注于位置，以测试、完善和确定一个明确的收入计划。如今，在基于位置的应用中，用户更喜欢交易和优惠券，而不是游戏机制和徽章。Foursquare 很棒，但它不是 Groupon。

从这个意义上说，Twitter 可以成为一个强大的平台，实时提供本地交易。每当用户通过网络、本地应用、文本或第三方客户端在系统中输入推文时，Twitter 都会通过 IP 地址或手机的 GPS 芯片知道位置。Twitter 可以通过两种方式将这一机会转化为将用户导向特定商家并达成交易:

1.  根据用户在推文中的位置，Twitter 可以从本地商家向用户推送选择加入的文本/电子邮件提醒和推文，这些提醒和推文可以立即使用或保存起来以供兑换。
2.  当用户进入一家商店时，用户可以在推特上发布一些简单的信息，如“斯坦福商城”，并自动接收附近商家的回复或文本/电子邮件提醒。这个选项的好处在于，那些没有智能手机或复杂数据计划的人不必在 LBS 应用程序之间进行选择，而是可以通过任何手机的短信发送推文。

作为推特和定位的信徒和两者的忠实用户，我自私地希望看到这一切发生。然而，我并不意味着这将是完全直截了当的，或者这是 Twitter 唯一的收入途径。这只是一个想法，但考虑到当前的收购环境以及本地和快闪交易网站的人气飙升，这可能是一个很大的想法，也很及时。
