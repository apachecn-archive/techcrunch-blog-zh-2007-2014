# Siri，能给我倒杯啤酒吗？

今天是星期五。是时候坐下来，把你的脚抬起来，打开一个冰冷的。更好的是，为什么不让 Siri 为你倒啤酒呢？你只需要一部 iPhone 4S 和一罐好东西。和一辆遥控汽车。一台 Arduino 连接到一个 Twitter 账户。还有锋利的大头钉，还有…是的，你明白了。
