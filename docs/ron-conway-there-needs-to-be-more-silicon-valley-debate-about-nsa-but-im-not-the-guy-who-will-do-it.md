# 罗恩·康韦:硅谷需要更多关于国家安全局的辩论，但我不是领导这场辩论的人

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/09/09/ron-conway-there-needs-to-be-more-silicon-valley-debate-about-nsa-but-im-not-the-guy-who-will-do-it/>

硅谷最有影响力的投资者之一罗恩·康韦今天为他不介入国家安全和消费者隐私问题的立场辩护，称他已经忙于枪支管制和移民改革等太多其他事情，同时还试图保住他的“日常工作”，帮助运营他的风险投资公司 [SV Angel](https://web.archive.org/web/20221221032054/http://www.crunchbase.com/financial-organization/sv-angel) 。

当迈克尔·阿灵顿问及他对美国国家安全局及其在以国家安全的名义窃取私人数据方面的作用，以及随后硅谷在试图反击方面缺乏参与的看法时，康威说:

“我相信我们必须平衡国家安全，但是有一件事叫做 9/11..你必须用透明度来平衡，”他今天在旧金山的 TC Disrupt 对观众说。“对我个人来说，这将取决于他们如何消化数据以及他们将数据保留多长时间。根据我的参与，去年我选择了三个问题，枪支安全、移民改革和公民参与……我们做了很多工作，让技术融入当地社区。”然后他把挑战转向阿林顿:“如果这是一个你感兴趣的问题，你应该做点什么。”

[更具体地说，阿林顿的问题更具对抗性:“你为什么袖手旁观，什么也没做？”他问。“我们正在谈论文明的终结和你生活中隐私的终结。”]

声明发表的同时，[脸书](https://web.archive.org/web/20221221032054/https://fb-public.app.box.com/s/0g1gquqyqmt3jk6jwlej)、[谷歌](https://web.archive.org/web/20221221032054/http://googleblog.blogspot.co.uk/2013/09/petition-for-greater-transparency.html)和[雅虎](https://web.archive.org/web/20221221032054/http://yahoopolicy.tumblr.com/post/60753842342/yahoo-files-suit-demanding-greater-accountability-from)都加大了对美国政府的要求，要求允许他们发布更多关于国家安全要求的信息。

这并不是说康威没有自己的观点，但从某种程度上来说，他缺乏反击似乎说明了一种无能，即风投或较小的初创公司如何最好地应对这个故事。“过去 60 天的事件表明，需要在透明度和国家安全之间取得平衡……但我将在今年剩下的时间里致力于移民改革。我不是一个吹牛的人，告诉人们我将在国家安全局改革中做同样的事情。现在对我来说，移民改革更重要……你听起来像一个完美的领导人。”

上午早些时候，旧金山市长李爱德也在这个问题上发表了意见。“我们需要更多的人来讨论这个问题……我们不知道这场辩论将走向何方。与此同时，这里有一个目标，许多人都在谈论我们必须拥有的平衡。我不喜欢个人信息被以令人震惊的方式使用。”他指出，旧金山与街头摄像头达成了一种“平衡”，例如，该市决定在抗议期间关闭街头摄像头，但在其他情况下为街头罪犯保留摄像头。

迈克·阿灵顿(TechCrunch 创始人)今天早些时候阐述了他的[意图](https://web.archive.org/web/20221221032054/http://uncrunched.com/2013/09/09/trying-to-face-down-the-evil-at-techcrunch-disrupt/)严厉打击围绕国家安全局、安全和信息的问题，以及硅谷在这个故事中扮演的角色。他认为，硅谷的公司，无论大小，都没有做出足够的努力来抵制信息要求的浪潮，以及政府机构以国家安全的名义所做的任何其他事情。

他今天早些时候写道，“我害怕我们的政府，我对小硅谷对抗政府的所作所为感到厌恶。”。

具体到罗恩·康韦，他指出，“我要问罗恩·康韦，他通过他的桑迪胡克承诺推动枪支管制已近一年，但他对硅谷在美国政府大规模破坏我们的人权中的作用只字未提。他可以通过领导真正透明的努力和对政府的真正抵制来做很多事情。但他一根手指都没动过。我想知道为什么。”

[Sandy Hook Promise](https://web.archive.org/web/20221221032054/https://techcrunch.com/2013/03/14/silicon-valley-investors-launch-innovation-and-investment-campaign-to-reduce-gun-violence/) 成立于 2013 年 3 月，是一个投资者合伙组织，致力于投资减少或消除枪支暴力的初创公司。阿林顿今天称这一努力“模糊不清”。
