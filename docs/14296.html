三星Galaxy Note II“平板手机”在大约两个月内通过渠道销售了500万部

本月初，三星宣布其迷你平板智能手机 Galaxy Note II 的渠道销售在发售的37天内已突破300万台。现在，这家韩国手机制造商宣布，在推出约两个月后，该设备的全球渠道累计销量已超过500万部。

三星通常不公布面向消费者的设备销售，但其渠道销售指标显示了其销售渠道的终端用户需求有多大。

Note II渠道销售公告是由三星在韩国发布的新闻稿中发布的——h/t到 TNW 供发现。三星的发布表明，Note II的销售在所有销售地区均匀分布，包括亚洲、欧洲、北美、拉丁美洲、中东和非洲。

最初的Galaxy Note和Galaxy Note II比典型的智能手机大，但比迷你平板电脑小，这导致了“平板手机”一词的产生，以描述这种手机兼平板电脑类别。

大约两个月内500万的渠道销量对于一款设备来说是一个令人印象深刻的数字，因为它的尺寸如此之大，与更普通尺寸的智能手机相比，这是一个相对容易获得的品味。举个例子，苹果最新的iPhone，iPhone 5，在发售的前三天就卖出了500多万部。苹果的数字指的是对终端用户的销售，而不是渠道出货量。
