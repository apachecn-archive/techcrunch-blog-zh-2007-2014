# 热衷于…什么让人们真正兴奋？(TCTV)

你叉子的末端是什么？你上网是为了看年轻女性还是看“辣妈辣妹”和“T2 熟女”？你喜欢对阿尔法男性的强迫幻想吗？还是更喜欢同性恋的言情故事？

到目前为止，这些关于我们网上性行为的问题充其量只是猜测。但最终，我们有了数据——确凿的、权威的数据——当我们认为没人在监视我们时，我们在网上真正做了些什么。两位波士顿大学训练有素的认知科学家，[Ogi Ogas](https://web.archive.org/web/20221003045117/http://www.crunchbase.com/person/ogi-ogas)博士和[Sai gad dam](https://web.archive.org/web/20221003045117/http://www.crunchbase.com/person/sai-gaddam)博士分析了来自 Dogpile、AOL、OKCupid 和 Pornhub 等网站的数百万个数据点，揭示了我们这个互联网时代性欲的真实本质。他们在一本极具煽动性的新书《十亿个邪恶的想法》中公布了他们的发现，该书描绘了当你把十亿人放在一个“几乎黑暗的房间”中时，他们会做些什么。

我采访奥吉·欧加斯的时候，他告诉我，《十亿个邪念》是第一部对“性欲的真实本质”的科学调查。更令人震惊的是，他向我揭示了真正让人兴奋的是什么。Ogas 博士解释说，男人喜欢观看，他们对业余爱好者有“无限的胃口”。比起 19 岁的人，更多的男人搜索 50 岁的照片。女人呢？嗯，他解释说，女人喜欢完全不同的东西。
