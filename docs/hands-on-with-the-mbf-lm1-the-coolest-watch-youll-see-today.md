# 动手使用 MB&F LM1:今天你会看到的最酷的手表

我采访了史蒂夫·哈洛克，他是 T2 钟表公司的北美总裁，也是瑞士钟表品牌向西扩张的幕后推手。他为我们带来的手表是 LM1，也就是传统机器#1，这是我们上个月早些时候报道过的一款新品，现在正运往世界各地的寡头和行业巨头。

首先，简单介绍一下这件作品:

> 为了与 19 世纪的高品质怀表保持一致，LM1 配备了一个宁静振荡(2.5 赫兹)的大直径摆轮，传统的宝玑表链悬挂在雄伟的双拱门上；它神秘的调节机制一览无余，但与运动没有明显的联系。两个子表盘上的小时和分钟可以完全独立设置——双时区复杂功能通常不允许独立调整分钟——它们的圆顶表盘进一步强化了制表黄金时代(1780-1850)的视觉参考。看起来像一个微型六分仪，世界上第一个垂直动力储备指示器可以跟踪可用动力，同时为支撑天平的优雅拱门提供视觉三维对位。

LM1 在现实生活中引人注目。双表盘经过手工上釉处理，悬挂在表盘中间的美丽摆轮营造出一种巧妙的、蒸汽般跳动的心脏，将整件作品融为一体。这个东西多少钱？运气好的话，大约 9 万美元，不过我相信史蒂夫可以给我们所有人打折。
