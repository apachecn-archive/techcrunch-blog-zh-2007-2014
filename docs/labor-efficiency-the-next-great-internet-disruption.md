# 劳动效率:下一次互联网大颠覆 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/04/labor-efficiency-the-next-great-internet-disruption/>

尼克·克罗宁是前公司律师，现在是总部位于芝加哥的[ExpertBids.com](https://web.archive.org/web/20221207042945/http://www.expertbids.com/)公司的总裁和创始人。

十多年来，互联网通过轻松地将拥有互利的个人或商业关系的各方联系起来，在提高我们日常生活的效率方面做出了巨大的贡献。同样的想法现在正处于颠覆劳动和改变我们所知的就业定义的边缘。

**独立工作者的崛起。**

在过去的几年里，作为某种独立的、自由代理的承包商或顾问的工人数量有了巨大的增长。虽然数字差异很大，但共识似乎是大约占美国劳动力的 20%，并且还在增长(有人估计到 2020 年将达到 50%)。想想看，目前每五名员工中就有一人不属于任何一家公司！

专家对这一增长的解释和数字本身一样多种多样，但我认为目前最重要的两个因素是:

1.  **技术。**从来没有一个物理空间表现得如此之少。一座办公楼，就其本身而言，通常不会有比某人所能携带的更多的完成工作所必需的工具。电脑、电话、云以及整体连通性创造了一个位置变得越来越无关紧要的环境。不必依赖他人的生产力工具，这使得比以往任何时候都多的人有能力成为一名独立的工作者。(无疑仍有例外)。
2.  **经济。最近的经济衰退导致了裁员和高失业率。此外，新一代的成年人认为，在一家公司长期工作是过去的遗留物。不管这是因为他们读到过裁员的消息，还是因为他们与家人或朋友一起经历过裁员，还是因为其他什么原因——许多人，无论年龄大小，都不再对传统工作感到舒适或稳定。因此，经济状况迫使一些人不得不独立工作，这也促使无数其他人探索传统工作之外的工作选择。**

有了技术和联系的武装，人们正以创纪录的数量独自出发。但是他们去哪里找工作呢？

**公司“雇佣”劳动力方式的变化。**

劳动效率就是让合适的工人来完成需要完成的任务。这包括所有类型和所有领域的任务。比以往任何时候，这是通过拥有根据项目需求来来去去的精干、灵活的劳动力来实现的。越来越多的雇主开始分解任务，让基于项目的临时工来完成任务。

以美国一家大型公司卡特彼勒公司为例，该公司最近报告称，他们在 2011 年最后一个季度雇佣了近 3 万名灵活的临时工。几乎每一项研究都表明，各种规模的公司都在强调精益劳动力，并越来越多地以项目为基础进行招聘(尽管并非所有公司都像卡特彼勒那样激烈)。这种趋势不仅限于工厂工人或计算机程序员或任何一个群体——每个行业和职业的工人都看到这种增长。

对于一个公司来说，要雇佣一个人，除了工资和福利之外还有很多成本(这些成本本身就很可观！).有招聘工作，入职成本，如供应和培训，最后是员工离职时的成本，如失业保险费，遣散费和人力资源成本。现在，公司可以选择以更低的价格雇佣数百万独立工作者中的一部分，而不是选择雇佣其他人。企业不必支付所有相关成本，而是可以将任务分解为项目，并找到专家来非常高效地完成它们——只需为完成的工作付费，而无需支付上述的次要成本。此外，他们可以更容易地根据供求关系扩大和收缩员工队伍。

不仅是大型企业雇佣了更多的独立人士，这一趋势也渗透到了数百万自举创业公司，他们在发展公司时只雇佣独立人士(除了创始人之外)。精益、灵活的劳动力时代已经到来，猜猜看，公司和独立企业在哪里越来越多地相互定位。是的:互联网。

中断的时间。

已经有很多公司将需要服务的一方与能够提供服务的另一方联系起来。TaskRabbit 和 Zaarly 是两家发展非常迅速的初创公司。但是我们刚刚开始触及互联网将如何扰乱劳动力的表面。随着越来越多的传统工作岗位创造者、小型企业乃至财富 500 强企业意识到灵活劳动力的好处，以及越来越多的个人投身于独立、自由的领域——无论是出于必要还是出于选择——真正的改变将会到来。

有许多公司致力于促进项目工人和公司之间的联系。像 ODesk 和 Elance 这样的市场为各种不同领域的自由职业者提供了一个全球平台。这些市场中的一些更倾向于商品化的服务，但是它们越来越多地包含所有类型的服务。OnForce 允许公司为项目保留 IT 专业人员的服务。劳动力市场是一个劳动力资源平台。Crowdspring 和 99Designs 是创意服务市场。最后(尽管还有无数其他的可以包括在这里)，我的公司 ExpertBids.com 是一个顾问、律师和会计师的专业服务市场。似乎每天都有新的垂直劳动力市场诞生。这些公司仍然需要克服许多障碍，但变革即将到来。

一些人批评这种转变，称这种劳动和就业方式只会增加不平等，弊大于利。我们需要开始深入研究这一趋势以及它是如何影响人们的，但是一个高效的劳动体系对双方都有很大的好处。

一个任务由正确的人在正确的时间以正确的价格(不是最低价格，而是正确的价格)完成的市场可能看起来对雇主有利。但是想想一个独立的人，他的开销很少，可以在任何地方、任何时间为任何人工作，他的收入潜力不再受单一工资的限制。消除浪费的时间和费用对双方和整个经济都有巨大的好处。这就是包括专家在内的所有在线劳动力市场需要帮助的地方。我们必须建立有效的平台，为这两个政党建立互利的 T2 关系扫除障碍。
