# Evernote 宣布实体产品市场，包括便利贴…和袜子？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/09/26/evernote-market/>

拥有 7500 万用户的热门笔记应用 Evernote 今天揭开了其努力成为人们存储所有记忆的去处的最新篇章——这是向实体商品迈出的决定性一步。

它正在与 3M 公司及其便利贴品牌建立合作关系，开发一种特殊的记事本系列和一种相机来记笔记、记录笔记，然后将笔记无缝传输到 Evernote 的平台上。这一消息实际上是昨晚泄露的，但今天 Evernote 更进一步，宣布这些产品和更多产品现在将通过一个名为[Evernote Market](https://web.archive.org/web/20221206115212/https://www.evernote.com/market)的特殊在线商店销售。(该公司表示，该市场也将在其桌面和移动应用程序中提供。)

Evernote 在旧金山举行的 EC3 大会上宣布了这一消息。首席执行官 Phil Libin 表示，公司涉足实体产品可能看起来“很奇怪”，但他指出[公司去年与 Moleskine](https://web.archive.org/web/20221206115212/https://beta.techcrunch.com/2012/08/24/evernote-smart-notebook-moleskine/) 合作，创造了一种特殊的笔记本，其功能使 Moleskine 漂亮的皮革笔记本上的笔记更容易数字化。

“无纸化不是我们的目标——卓越的体验不是我们的目标。”利宾说，后来又补充道，“我们希望消除纸张的愚蠢用途，但我们希望扩大纸张的伟大用途。”

除了 Post-it 产品之外，市场上的其他新产品还有 ScanSnap 和 Adonit 的设备。

该公司表示，ScanSnap 的 Evernote 版本是与 ScanSnap 合作的结果，“将硬件速度和精度与 Evernote 的组织和同步功能结合在一起。”据说这将是市场上最先进的扫描仪到服务的集成。

与此同时，Adonit 的 Jot Script Evernote Edition 手写笔是“市场上第一款也是唯一一款细点手写笔，并经过优化，可与 Evernote 屡获殊荣的手写应用程序 Penultimate 配合使用。”

Evernote 还推出了一系列生活方式产品，这些产品并没有很强的技术角度，而是根据公司品牌定制的。这些产品包括来自科特和西尔的 Evernote 背包和 Evernote 平板包。[而且还有袜子](https://web.archive.org/web/20221206115212/https://www.evernote.com/market/feature/socks?sku=SOCK00106)！这是一个真正的一应俱全的方法。

把 Evernote 的所作所为想象成企业版的相同策略，这种策略已经被像迪士尼和 Rovio 的愤怒的小鸟这样的消费品牌所体现。或者甚至像苹果这样的公司所体现的垂直整合的概念。这个想法是，实体产品不仅扩展了大公司的品牌。但它也将为用户提供更多使用 Evernote 的方式，让它变得更有用。

根据 Rovio 本身计划从商品销售中获得的收入超过其实际游戏的收入这一事实来判断——当然，苹果从硬件中获得的利润远远超过其服务——这最终可能是 Evernote 的一个明智之举。迄今为止，该公司最大的收入来源是付费用户。这给了该公司一个更加多样化的长期货币化战略。

Evernote 产品副总裁杰夫·兹沃纳(Jeff Zwerner)表示，该公司在考虑物理集成时，遵循五大原则——为物理产品提供数字未来，平衡生活和工作，为用户带来“全球发现”，逐步改进基本产品，以及一起设计软件和硬件。

“没错，我们现在是一个时尚品牌，”李斌说。“没有人预见到这一点。”

**更新:**主题演讲结束后，我们问李斌，鉴于如今 Evernote 品牌的急剧扩张，是否有一些事情是该公司绝对*不会*做的。他回答说，他致力于“直接”赚钱，而不是“间接”，这意味着 Evernote 不会经营广告。(利宾描述 Evernote 市场上产品背后的协议是简单的收入分成:“我们对交易采取了非常没有创造性的方法。”)

他还表示，Evernote 将继续是一家“内省”的公司，不会过于深入地研究社交功能——它将在明天发布一些商业合作公告，但它们应该“尽可能提高你的互动效率，以便你可以继续高效工作。”

利宾补充说，这种合作关系的广泛愿景是将 Evernote 变成“思想的耐克”——即消费者与思想联系在一起的大品牌。

当被问及如何看待潜在的首次公开募股时，他强调说，他认为这是公司的“加油”站，而不是目的本身。李斌说，“是的，我认为我们应该上市，但这不是目标。…这将是多年以后的事了。”

我还与 Moleskine 首席执行官 Arrigo Berni 进行了交谈，他引用 Evernote 合作伙伴关系作为例子，说明事情不仅仅是“数字数字数字”和“纸张纸张”。相反，他说两者之间有一个连续体。这种合作关系已经产生了一个零售价值为 450 万欧元的业务，而且这只是几个类型的笔记本电脑(如上所述，有各种各样的产品在工作)。Berni 说，Moleskine 现在正在寻求其他合作伙伴关系，以进入数字世界，例如与纸质应用程序的制造商 53 合作。
