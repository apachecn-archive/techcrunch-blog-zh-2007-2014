# Flickr 宣布为每个用户免费提供 1tb 的存储空间，正式击败了所有人

雅虎的 Flickr 照片分享服务现在为用户提供了一个完整的 1tb 的存储空间，足以容纳世界上所有的照片。除了全分辨率照片存储服务之外，该服务还提供这项优势。

虽然普通用户可能一辈子都不会触及这个存储空间的外部界限，但仅此一点可能就足以吸引专业摄影师来使用这项服务，更重要的是，将流失的用户带回雅虎。

这一步很重要。鉴于大多数照片分享服务的奇怪性质，你要么被限制在几十千兆字节，要么在 Instagram 和其他移动服务的情况下，有一个不属于营销宣传材料的未说明的上限。虽然我不怀疑谷歌或脸书在不久的将来会宣称拥有 1tb 的容量，但率先将这一特殊功能推向市场是一个重要的里程碑。

这一举动很明显是雅虎为了让自己的产品变得有意义而采取的一个策略。长期处于困境的 Flickr 立刻吸引并挫败了声称被网络巨头抛弃的专业用户。

正如玛丽莎·梅耶尔在她的演讲中指出的，这是关于“将一生的美丽带入 Flickr”这也是为了说服休闲摄影师相信 Flickr 是他们旧照片的通用鞋盒——这是一件有利可图且令人惊讶的重要事情。
