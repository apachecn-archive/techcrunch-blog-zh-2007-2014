Windows Phone Manager因诺基亚电话推特离开微软TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/09/20/windows-phone-manager-leaves-microsoft-over-nokia-phone-tweets/

通常是一家公司的社交媒体代表在推特上发布他们不应该发布的东西，但前微软员工乔·马里尼在食物链上的位置比这高一点。前Windows Phone网络团队的首席项目经理Marini在推特上发布了他对即将推出的诺基亚制造的Windows Phone的第一印象后，辞去了他在微软的职位。

微软通常不会对此类人事问题发表评论，但一位发言人打破沉默，向Geekwire证实马里尼已不再受雇于该公司。

微软有自己的一套关于员工应该如何接触社交媒体的指导方针 [PDF]，但它真正说的是关于内容，员工应该“聪明”，不要分享“尚未公开披露的新特性、功能或创新”

有趣的是，推文本身很难称之为有趣。最初的推文发布于9月7日，只是提到他开始玩Windows操作系统的诺基亚设备:

https://twitter.com/#!/Joe marini/status/111555182919294976

随后的推文更加详细，比如当天晚些时候的两条:

https://twitter.com/#!/Joe marini/status/111578149971689472

https://twitter.com/#!/Joe marini/status/111587869272969216

到目前为止，他只确认了新设备有摄像头、屏幕和机身——几乎没有什么突破性的东西。Marini(可能非常小心地)除了对设备构建和用户界面的一般性评论之外，没有提及任何细节。尽管如此，这显然足以让他与微软的社交媒体政策发生冲突，该政策(不难理解)禁止使用Twitter等服务分享机密信息。

然而，乔并不是第一次犯罪。早在三月份，乔在推特上说他将在参加SXSW时展示一个用于Windows Phone的未发布版本的IE9 。在这种背景下，这组新的推文可能是压垮骆驼的最后一根稻草，但这种情况给我留下了不好的印象。

你可以说我幼稚，但我认为微软和Windows Phone需要更多像Joe这样的人。他的推特账户描绘了一个致力于工作的人，他对微软和诺基亚即将到来的事情感到非常兴奋。令人不快的推文给了人们足够的刺激，然后就此打住。考虑到诺基亚和微软几个月前就达成了官方协议，任何人都不会对手机的研发感到惊讶。为什么要因为一个人说了显而易见的话而解雇他呢？

我打赌微软和诺基亚不会太喜欢他对设备屏幕尺寸的抨击——这是一个合理的评论，但可能不会让高层满意。尽管如此，鉴于马里尼对他所看到的总体模糊，另一个有趣的问题浮现在脑海中:在什么情况下泄漏才是真正的泄漏？
