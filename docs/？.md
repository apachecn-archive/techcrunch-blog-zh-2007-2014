# 来源:脸书推出阅读、聆听、观看和需求按钮

脸书将会在本周的开发者大会 f8 上推出[重大产品](https://web.archive.org/web/20230205032953/https://techcrunch.com/2011/09/19/expect-this-years-f8-to-be-huge-%E2%80%94-the-biggest-since-facebook-platform-launched/)的消息已经泄露。我们已经听说过[社交音乐服务](https://web.archive.org/web/20230205032953/https://techcrunch.com/2011/09/17/facebook-music-soundcloud-deezer-rhapsody/)可能会在几天后推出，但正如[纽约时报在过去的周末所传达的](https://web.archive.org/web/20230205032953/http://www.nytimes.com/2011/09/19/business/media/facebook-is-expected-to-unveil-media-sharing-service.html?_r=1&pagewanted=all)，脸书正在计划更好地展示个人内容的方式。我们从一个消息来源得知，脸书将在墙上引入新的按钮，这将开始引入一些“喜欢”概念的粒度。我们被告知这些新按钮是“读”、“听”、“看”在引入上述按钮之后，该网络还将很快推出新的社交商务按钮，如“Want”。

重要的是要证明这是来自某个来源(而不是来自脸书)，但据我们所知，脸书的用户将能够点击阅读、收听和观看他们的新闻内容。很快，“想要”也是。

目前还不清楚 Like 按钮会发生什么，以及这些新按钮会对 Like 按钮产生什么影响。我们不知道脸书会用这些数据做什么，但是网络*可以用这些按钮上的数据做很多事情。显而易见，广告定位以及为用户提供更个性化体验的能力将是一个巨大的机会。更不用说品牌，零售商，娱乐公司和其他企业将能够获得类似的细分数据。*

如果这一切开始听起来有点像脸书臭名昭著的 Beacon 项目，那也不应该太令人惊讶——据我们所知，这些脸书新功能的一个关键部分是提供类似 Beacon 的功能，根据意图和行动自动填充新闻提要故事。但是他们这样做没有广告和隐私的影响。至少目前来说…

这些新的粒状按钮的引入肯定会增加媒体共享应用程序和零售商提供的内容的深度，这与之前关于即将推出的内容的报道一致。我们会让你知道我们听到的 f8 的其他消息(我们知道还有什么没有宣布:斯巴达项目)。

**更新** : [利兹·甘尼斯报道](https://web.archive.org/web/20230205032953/http://allthingsd.com/20110919/read-watch-listen-facebooks-official-motto-for-f8/)F8 的座右铭将是“阅读。看好了。听着。”—这听起来与我们听说过的按钮完全一致。
