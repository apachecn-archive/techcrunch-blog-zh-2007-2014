# 雅虎高管结构瓦解:陆、加林豪斯和马基雅尼离开 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/06/19/yahoos-executive-structure-crumbles-lu-garlinghouse-and-makhijani-to-leave/>

我们发布[前雅虎高管追踪电子表格](https://web.archive.org/web/20230209002125/https://techcrunch.com/2008/06/19/tracking-former-yahoo-execs-so-many-have-left/)才几个小时，但它已经严重过时了。据接近该公司的消息人士透露，又有三名高管将离职:执行副总裁[陆弃、](https://web.archive.org/web/20230209002125/http://www.crunchbase.com/person/qi-lu) SVP [布拉德·加林豪斯](https://web.archive.org/web/20230209002125/http://www.crunchbase.com/person/brad-garlinghouse)和 SVP [唯帅·马基贾尼](https://web.archive.org/web/20230209002125/http://www.crunchbase.com/person/vish-makhijani)。

顺便说一句，上周宣布离职的雅虎技术传播者杰里米·扎沃德尼(Jeremy Zawodny)宣布，他将于下月加入 Craigslist。

**唯帅·马基雅尼**

据我们的消息来源称，SVP 兼搜索部总经理唯帅·马基贾尼将离职，出任 [Yandex](https://web.archive.org/web/20230209002125/http://www.yandex.com/) “俄国的谷歌”的首席执行官更正:他将领导 Yandex 实验室，而不是 Yandex。请参见下面的更新]。Yandex 主导着俄罗斯的搜索市场，据传将于今年晚些时候上市。Makhijani 曾长期担任雅虎高管，在 2002 年雅虎收购 Inktomi 时加入该公司。唯帅向执行副总裁杰夫·韦纳汇报工作，韦纳本周早些时候辞职。

**布拉德·加林豪斯**

[![](img/b6035c01c640ed98608ef88e396a653e.png)](https://web.archive.org/web/20230209002125/http://www.crunchbase.com/person/brad-garlinghouse) 布拉德·加林豪斯(Brad Garlinghouse)是雅虎通信界的 SVP&，控制着大量资产，如 Mail(2.6 亿用户)、Messenger (1 亿用户)、Groups(1.1 亿用户)、Flickr(4700 万用户)和 Zimbra。大约有 600 人向加林豪斯汇报工作，在过去的两年里，加林豪斯一直呼吁对公司的经营方式进行重大改革。在加入雅虎之前，布拉德曾担任 Dialpad Communications 的首席执行官。在职业生涯的早期，Brad 在@Ventures 领导了通信和互联网业务的风险投资。他还在@Home Network 和 SBC Communications 担任过领导职务。

目前还不清楚加林豪斯接下来会去哪里，有传言称多家私募股权公司正在争夺他的注意力。坦率地说，考虑到他的运营经验(他将自己控制的大部分资产发展成了市场上的第一名，尽管雅虎搜索多年来分崩离析)，他没有最终成为某个地方的首席执行官真是太糟糕了。

**陆弃**

这个谣言已经流传了几个月，但是搜索和广告技术集团的执行副总裁陆弃将很快离开雅虎。Lu 向 CTO [Aristotle Balogh](https://web.archive.org/web/20230209002125/http://www.crunchbase.com/person/aristotle-balogh) 汇报。他是雅虎巴拿马搜索营销平台的首席架构师(这意味着自雅虎/谷歌交易以来，他的工作变得明显不那么有趣了)。

布拉德·加林豪斯和唯帅·马基贾尼被认为是接替即将离任的执行副总裁杰夫·韦纳的角色的有力竞争者——显然他们现在都不在考虑范围内。但更大的担忧是，除了首席执行官杨致远和总裁苏·德克尔之外，雅虎目前没有多少管理人才，他们越来越孤立，似乎已经失去了除雅虎董事会以外所有人的信任。

**更新:**[Yandex 董事会成员埃丝特·戴森](https://web.archive.org/web/20230209002125/http://www.crunchbase.com/person/esther-dyson)发电子邮件告诉我们，唯帅·马基雅尼确实加入了 Yandex，但他是硅谷 Yandex 实验室的总裁兼首席执行官。不是母公司 Yandex 的 CEO。
