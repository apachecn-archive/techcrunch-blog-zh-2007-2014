# Siine 耗资 55 万英镑，旨在改变我们在触摸屏键盘上打字的方式

![](img/3cc422f21720c2d820762feac267d31a.png)

总部位于西班牙/英国的软件开发商 Siine 旨在改变我们使用触屏手机和平板电脑交流的方式，已经从 Skype 创始人[Niklas zenn strm](https://web.archive.org/web/20221005212557/http://www.crunchbase.com/person/niklas-zennstrom)和 [Janus Friis](https://web.archive.org/web/20221005212557/http://www.crunchbase.com/person/janus-friis) 建立的投资工具 [Atomico Ventures](https://web.archive.org/web/20221005212557/http://www.crunchbase.com/financial-organization/atomico-investments) 获得了 55 万英镑(约合 88 万美元)的种子资金。

该公司表示，它已经创造了独特的系统，使人们更容易通过触摸屏设备输入和传输文本，减少了使用虚拟屏幕键盘所需的按键次数。当然，这个音调听起来和 Swype 的非常相似。

不过，该公司走了一条不同的路线，在语言学和通信研究的基础上创建了一个图形用户界面，用户可以点击建议的符号来传达他们的意思，而不必键入单独的字母。

我曾在去年 12 月的 Le Web 会议上简要地看过这款目前处于 alpha 测试阶段的软件，不得不承认我对此非常感兴趣。下周在巴塞罗那举行的世界移动通信大会上，我将了解更多关于 Siine 的信息。

据我所知，Siine 将在本季度首先推出其 Android 平台软件，并且已经在与 Galaxy Tab、Galaxy S 和 HTC Desire 的用户合作。

这家总部位于巴塞罗那和伦敦的初创公司由埃德·马克洛夫(Ed Maklouf)于 2007 年创立，他在斯坦福大学学习群体通信，后来开发了游戏、音乐和投票技术。

当地天使投资者，包括前巴塞罗那足球俱乐部副总裁马克·英格拉，早些时候向 Siine 注入了一笔未披露金额的资金。
