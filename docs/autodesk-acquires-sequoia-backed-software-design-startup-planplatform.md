# 欧特克收购红杉支持的软件设计初创公司 PlanPlatform

![](img/9b97fc0e9809dbe28217d60ddcc76b7d.png)

软件设计应用程序开发商欧特克[收购了竞争对手](https://web.archive.org/web/20230130100805/http://www.globes.co.il/serveen/globes/docview.asp?did=1000523388&fid=942) PlanPlatform。据报道，Autodesk [为这家以色列初创公司支付了](https://web.archive.org/web/20230130100805/http://www.haaretz.com/hasen/spages/1135725.html)2000 万至 3000 万美元。Autodesk 是一家上市公司，为各种行业提供软件设计应用，包括建筑、工程、媒体和娱乐。

PlanPlatform，原名 [Visual Tao，](https://web.archive.org/web/20230130100805/http://www.visualtao.com/)开发了为工程师提供二维和三维软件绘图工具的 SaaS。这家初创公司最近获得了 430 万美元，红杉资本领投。Autodesk 已经收购了许多公司，包括 Algor、SoftImage 和 3D Geo。
