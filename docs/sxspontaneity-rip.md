# # SXSpontaneity，RIP 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/03/02/sxspontaneity-rip/>

More posts by this contributor

*“你要去 SXSW？傻逼。”*

这是一位长期参加 SXSW 的人的情绪，他决定今年不参加，这得到了科技界许多其他人的回应。随着其不断扩大的节目，住房选择的集群和曾经被认为是任何知情人都必须参加的会议的商业化，SXSW 不仅没有跳过鲨鱼，而是受到了资深与会者的公开诽谤。那些选择离开的人，像我一样，正被同情怜悯地看待，就像南方人用“保佑你的心”来表示“你像尘土一样愚蠢”

上帝保佑我，真的。

我不会坐在没完没了的讨论会上(尽管我确实买了一枚徽章，这显然越来越罕见了，如果我的朋友有任何暗示的话)，而是给它最后一次机会来挽救它的价值。我认为，仍有交易要做，还有人要见，这或许是一种过于乐观的看法，因为我亲眼目睹了过去 11 年来的变化。哦，它是如何改变的。除了规模(难以忍受——想想死亡金属演唱会上的 mosh pit，只有更傻的 t 恤)或巨大、夸张的大品牌激活(与我曾经崇拜的音乐节在视觉上不协调),真实性和亲密感已经消失。

但我最怀念的是自发性。最好的意图会被打断，这太棒了……这就是这个事件的原因。你可能正走向一个(很可能非常拥挤的)聚会，遇到一个你多年未见的人，却花了接下来的五个小时去了解你的下一任老板。它告诉你在你的 3000 人派对人群中看到的那个帅哥“留在原地”，这导致了一场持续多年的热恋。这是意想不到的，你一时兴起做的决定，导致了一些最有价值的经历。我甚至可以说是它成就了 SXSW 或者说，这些天来，它是什么。

现在一切都感觉过度规划和过度生产，不仅仅是一个组织(可以理解)需要增长和盈利，并为品牌和个人提供体验。从一月份开始，你的收件箱里就会收到来自公司和品牌的邀请。事实上，所有市中心的酒店在 7 月开始售票的几个小时内就销售一空。正是这种意外收获的丧失导致许多 SXSW 的长期爱好者呆在家里。

我把这个对话带到了推特上。YxYY(许多人称之为当代 SXSW)的两位创始人希拉里·哈特利(Hillary Hartley)和艾米·穆勒(Amy Muller)都表示，失去“魔力”是跳过今年的原因。失去了[的人脉&的意外收获](https://web.archive.org/web/20221206143014/https://twitter.com/hillary/status/436570357961854976)让哈特利很难证明自己有理由去，因为她[只希望能赶上三藩市的朋友](https://web.archive.org/web/20221206143014/https://twitter.com/hillary/status/436584330614153216)。

穆勒同意；[偶然发现的魔法现在很少了](https://web.archive.org/web/20221206143014/https://twitter.com/AmyGSFN/status/438065539093512192)。其他人则哀叹大品牌和营销项目，比如德鲁·奥拉诺夫(Drew Olanoff)，他在推特上写道:“一旦糟糕的营销人员和公关人员将 SXSW 视为开发应用的捷径，一切就都走下坡路了。”然而其他人——比如阿里·伯曼——认为这仅仅是一种自然进化。“看过去‘它太大了，太公司化了，太蹩脚了’的争论，就像许多事情发生的那样，我们已经超越了它。我敢肯定，有很多 26 岁的人很想去。”

也许吧。对我来说，我没有太多的计划，我只打算参加一个活动(这是一个小型的，只有我的一些好朋友才能参加的晚宴。)而不是举办一个盛大的派对，我打算某天醒来，查查天气，邀请一些朋友和我一起去外面喝一杯。SXSW 还有希望吗？我不知道，但愿上帝保佑你，也许我会在那里见到你。

**编者按:** *[奥布里·萨巴拉](https://web.archive.org/web/20221206143014/http://about.me/aubs)是一名营销人员，他发展了包括谷歌、脸书、Digg 和 AOL 在内的一些全球最大的科技品牌。她 11 年前开始去 SXSW，虽然 SXSW 已经改变了，但她对早餐玉米卷的偏好仍然很强烈。在推特上关注她对意外收获的追求——更不用说奥斯汀最好的玛格丽塔酒了。*
