# MyID .现在处于公开测试阶段，目标是成为数字认证标准

数字认证平台 [MyID.is](https://web.archive.org/web/20221208224245/http://myid.is/) 正在尝试为人们提供一种在线声明真实身份的方式，以便能够防止 ID 被盗，并验证他们在博客、社交网络账户、照片视频共享网站等上发布的内容。

此外，该网站还提供了(另一种)管理你在线身份的方式，同时也是认证的 OpenID T4 提供商。该网站在过去的 8 个月中一直处于 alpha 测试阶段，并于昨天进入公开测试阶段。

它是这样工作的:你在网站上注册一个 MyID.is 认证的帐户，并输入你的个人信息，这些信息随后会被团队验证(我稍后会谈到这个问题)。他们通过交叉检查你提交的姓名和你信用卡上的姓名来做到这一点——他们将收取 2 至 5 美元的费用来验证这是你的，类似于谷歌检查你的 AdSense 账户信用卡详细信息的方式——并通过向你的邮政地址发送一个 6 位数的代码来验证你在他们平台上的身份。除了小额安装费，这项服务是免费的。

你会得到一个专用的 MyID.is 网址，看起来像这样:[http://myid.is/charles.nouyrit.id](https://web.archive.org/web/20221208224245/http://myid.is/charles.nouyrit.id)(这是创始人兼首席执行官查尔斯·nouÿrit 的网址)，你还会得到一些徽章，你可以把它们放在你的博客或社交网络个人资料上，以表明你的身份已经得到该公司的验证([例](https://web.archive.org/web/20221208224245/http://blog.myid.is/))。该平台还提供了许多定制的小工具，为传播您的在线 ID 提供了“无限的可能性”。

这是一个雄心勃勃的项目，很高兴看到这样的倡议来自欧洲。

但房间里的大象当然是事实，即 MyID .面临着巨大的问题，必须说服人们有效地信任他们，包括他们的私人数据、信用卡细节和物理地址(显然，他们明确承诺永远不会出售)。我确信有安全措施，Nouÿrit 说他们不保存机密信息，甚至从来没有接触过它，因为它通过一个复杂的银行系统，但这真的是题外话。

人们仍然需要对注册这项服务充满信心，我不确定一家总部位于伦敦的小公司如何能够在全球范围内达到这种信任程度。Nouÿrit 反驳道，PayPal 也需要几年时间才能被广泛接受，这是一个很好的观点，但不完全是 MyID.is 的通配符

在全球范围内验证你的在线身份似乎没有太多的竞争，我们能找到的解决方案也相当昂贵。在美国，有特鲁菲娜和 BeenVerified.com 的[，在芬兰有类似 NorthID.com](https://web.archive.org/web/20221208224245/http://beenverified.com/)的[(在瑞典也可参见](https://web.archive.org/web/20221208224245/http://northid.com/) [BankID](https://web.archive.org/web/20221208224245/http://www.bankid.com/en/What-is-BankID/) )。
