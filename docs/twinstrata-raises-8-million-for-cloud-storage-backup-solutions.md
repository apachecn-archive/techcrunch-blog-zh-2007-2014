# TwinStrata 为云存储和备份解决方案筹集了 800 万美元

[TwinStrata](https://web.archive.org/web/20230204195056/http://www.twinstrata.com/) ，提供基于云的数据存储、备份和灾难恢复解决方案，今天早上[宣布](https://web.archive.org/web/20230204195056/http://www.prnewswire.com/news-releases/twinstrata-raises-8-million-in-series-b-funding-131778803.html)在由 [Avalon Ventures](https://web.archive.org/web/20230204195056/http://www.crunchbase.com/financial-organization/avalon-ventures) 领投的 B 轮融资中获得了[800 万美元](https://web.archive.org/web/20230204195056/http://www.crunchbase.com/company/twinstrata)。该公司将利用新的资本来加强其 [CloudArray](https://web.archive.org/web/20230204195056/http://www.twinstrata.com/CloudArray-overview) 解决方案的销售和营销工作。

TwinStrata 的 CloudArray 云存储网关将客户的应用程序和备份基础架构连接到云，使他们能够将数据即时存储和恢复到任何位置。

CloudArray 支持公共和私有云存储提供商，如 Amazon、RackSpace、OpenStack、EMC Atmos、Nirvanix 和 Scality 等。

新的融资使该公司的资本总额达到大约 1450 万美元。
