# Path 最终关闭了难以捉摸的系列 C TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/10/path-finally-closes-that-elusive-series-c/>

对于戴夫·莫林的[之路](https://web.archive.org/web/20221007092907/https://path.com/)来说，这是一条坎坷的道路，因为去年疯狂的谣言已经将“私人”社交网络立刻视为[增长滞后、员工减少](https://web.archive.org/web/20221007092907/https://beta.techcrunch.com/2013/11/07/path-loses-biz-head-amid-rumors-of-over-7m-in-funding-from-dustin-moskovitz/)和潜在收购的主题，同时以 5 亿美元的估值为[筹集了 5000 万美元的巨额融资。](https://web.archive.org/web/20221007092907/https://beta.techcrunch.com/2013/07/16/path-is-raising-50m-at-a-500m-valuation/)

如今，该公司漫长的 C 轮融资之路似乎终于走到了尽头。最近，Path 通过推出高级订阅计划和私人分享等产品增加了额外的收入来源，在过去的一个季度里，Path 一直在采取措施，似乎让投资者对其长期前景放心。

今晚，[卡拉·斯威舍报道【Path 已经筹集了 2500 万美元，并为印度尼西亚的 Bakrie Global Group 增加了至少一名新投资者，使其资金总额达到 6500 万美元。](https://web.archive.org/web/20221007092907/http://recode.net/2014/01/10/exclusive-path-completes-25-million-funding-led-by-indonesias-bakrie-global-group/)

Morin 也是一名前 Facebooker 用户，他还告诉 Swisher，虽然该公司完成了一轮比其 3000 万美元的 B 轮融资规模小的融资，但事实上这是一轮“上升轮”。据报道，Path 在 B 轮融资中的估值为 2.5 亿美元，如果这是一轮上涨，在投资者眼中，该公司的价值自 2012 年 4 月以来已经增加，可能高达 4 亿美元。

此外，11 月份，TechCrunch 的联合编辑 Alexia Tsotsis 从消息来源得知，Path 正在完成一轮超过 700 万美元的融资，其中也包括莫林的朋友和前 Facebooker 用户达斯汀·莫斯科维茨。正如我们当时报道的那样，TechCrunch 再次从消息来源获悉这是真的——莫斯科维茨是 Path 的 C 轮投资者。

Swisher 还报道说，Path 的现有投资者，包括 Greylock Partners、Kleiner Perkins、Index Ventures、Insight Venture Partners、Redpoint Venture Partners 和 First Round Capital，也参与了这一轮投资。

至于为什么 Path 选择了一个印度尼西亚的主要投资者作为它的 C 轮投资？Morin 在接受 Re/code 采访时解释说，他“一直在寻找”亚洲的战略投资者，尤其是在东南亚。Morin 说，东南亚是 Path 的影响力最大的地区之一，甚至比人们所理解的还要大。让 Bakrie 成为其主要投资者，让 Path 有了一个了解市场的合作伙伴，可以帮助它战略性地扩大和增加其在东南亚的足迹。

虽然很难不把这次提价(以及估值的提高)视为一个积极的迹象，但说这是 Path 想要达到的目标(尤其是在美国)的一个迹象是不公平的。在与 Swisher 的交谈中，Morin 承认 Path 仍然需要“关注”许多“美国市场的挑战”

随着 Morin 确认 Path 现在拥有 2300 万客户，挑战可能不是缺乏牵引力，而是在面对不断变化的社交市场和 Snapchat 等年轻移动社交网络的爆炸式增长时，堵住用户群的漏洞。

我们已经联系了 Morin 和 Path 来确认以上所有信息，一旦我们了解到更多信息，我们会尽快更新。
