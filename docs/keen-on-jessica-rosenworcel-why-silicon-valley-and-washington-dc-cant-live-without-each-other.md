# 热衷于…杰西卡·罗森沃塞尔:为什么硅谷和华盛顿·DC 离不开彼此

当然，硅谷有许多自由主义者将华盛顿 DC 视为创新的死敌。但是根据 2012 年被巴拉克·奥巴马任命为联邦通信委员会委员的杰西卡·罗森沃尔瑟尔的说法，硅谷需要“更多地参与”DC。她告诉我，企业家“需要一个裁判”，尤其是在频谱政策等关键领域——她坚持认为，在我们日益流动的未来，频谱政策是“完美的稀缺资源”，对“创造创新”至关重要。

但 Rosenworcel 承认，正如硅谷需要华盛顿 DC 一样，华盛顿需要更好地与硅谷合作，她说，硅谷“理应受到尊敬”。Rosenworcel 自己可能是一个更加兴奋的立法者的典范，她不仅与初创科技社区有联系，甚至还在 [Twitter](https://web.archive.org/web/20230319125513/https://twitter.com/JRosenworcel) 上。因此，与其说是频谱，不如说是 Rosenworcel 才是*完美的稀缺资源*——这位冷静的联邦通信委员会委员在硅谷就像在 DC 一样自在。
