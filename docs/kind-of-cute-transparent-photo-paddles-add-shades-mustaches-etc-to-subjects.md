# 有点可爱:透明的“照片桨”给主体添加阴影、胡子等

![](img/0fdcd3ea43e3d709bab35d7b65498d4a.png "photo_paddles")
等不及回家给照片添加一些新奇的效果？我有一个产品给你！这些照片桨本质上只是一些透明的塑料，里面装有道具——嘴唇、太阳镜、漂亮的大胡子，诸如此类。是的，很俗气，但是*可能*很有趣。http://www.photopaddles.com/index.php

它们实际上已经存在很长时间了(它们是为去年的一个设计展设计的)，但是现在购买它们比以前更容易了。每个 6 美元似乎有点贵(你可以用更低的价格做自己的)，我不知道怎么才能买到太阳镜(“你好”)。)，但是哎。这是艺术，不应该是有意义的。

![](img/fc3c054efa49b931501972fd26ffaa5a.png "1400x720-3z1XYd5T")

请记住，如果你有任何景深，这些都不会起作用。[创造者](https://web.archive.org/web/20221202063558/http://stevenhaulenbeek.com/artwork/1259790_Photo_Paddles.html)指示你将它们与你的拍照手机一起使用，这似乎是一个合理的建议，尽管如果你有一个花哨的*自动对焦*拍照手机，你可能会有麻烦。
