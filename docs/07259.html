Tictacti获得资金在游戏聚合领域挑战Heyzap TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/06/30/tictacti-secures-funding-to-take-on-heyzap-in-game-aggregation/

Tictacti获得了在游戏聚合领域与Heyzap竞争的资金

许多公司都想搭上flash游戏的潮流，但却没有核心竞争力。以色列推出了一种新的白色标签解决方案，名为“T2”。这家初创公司现在已经筹集了50万美元，由基马风险投资公司和其他未披露身份的投资者牵头。迄今为止，这家初创公司已经吸引了160万美元的资金。

Tictacti计划与现有的玩家 Mochi Media 和 heyzap 竞争，借助一个平台，游戏开发者可以通过一个API/SDK或基于Flash的游戏在移动、网络或IPTV上赚钱。但是这里关键的一点是，他们不需要使用API就可以做到。

最终的想法是成为一个大型游戏发行商聚合商，涵盖网络、社交网络、移动设备、游戏机和IPTV，并为开发者提供一站式解决方案。

到目前为止，耐克、星巴克、宝洁和其他公司都在使用TicTacTi，加载他们的富媒体内容，从而获得广告收入。
