# 使用 Plug，用您的 USB 驱动器创建一个个人的、免订阅的 Dropbox

Plug 刚刚为其 69 美元的适配器发起了一场 Kickstarter 活动。它将把你的 USB 驱动器转变成你所有设备的个人收存箱。由于文件系统的深度集成，您不必将文件移动到特殊的文件夹或虚拟硬盘。启动 Plug 的应用程序后，一切都是透明的，你不必改变你的工作流程。除了你所有的设备现在都有相同的文件。

就像 TechCrunch Disrupt aluminum[Bitcasa](https://web.archive.org/web/20221006071824/https://www.bitcasa.com/)一样，你可以在本地硬盘上缓存一些文件夹和文件，或者从 Plug 上下载。Plug 的唯一限制是你的 USB 驱动器上的存储空间。

在幕后，Plug 是一个基于 Linux 的小型机器，它创建了一个 VPN 网络。然后，如果你想从你的 iPhone 上访问你的文件，Plug 的客户端会静默地连接到网络，并向你显示你所有的文件。它看起来很像 Dropbox 的应用程序，只是你的文件存储在家里，而且你不需要支付订阅费。你可以说它非常类似于网络附加存储设备，但是用一个软件技巧来替换你的整个文件系统。

“我们在 Plug 上的创新与其说是硬件创新，不如说是软件创新，”联合创始人兼首席执行官 Séverin Marcombes 告诉我。“如果这个系统不是如此昂贵和缓慢，我们可以在云端设计它，”他继续说道。

启动应用程序后，你所有的文件都将通过插件。使用 USB 2.0 连接器和以太网 100 端口，播放电影可能会有点慢，特别是如果你有多个驱动器插入适配器。这就是为什么你可以缓存你的文件。即使 Plug 声称你可以像以前一样使用你的文件，这种新的模式可能需要一段时间才能适应。Marcombes 将缓存功能与 Spotify 的离线播放列表功能进行了比较，Spotify 用户已经非常熟悉这个按钮。

[Kickstarter 活动](https://web.archive.org/web/20221006071824/http://www.kickstarter.com/projects/cloud-guys/plug-the-brain-of-your-devices)刚刚开始，但它的目标相当低。在 69，000 美元，总部设在巴黎的团队肯定会吸引 1000 名支持者来实现其目标。你只需要等到 12 月份就能拿到这款设备了。
