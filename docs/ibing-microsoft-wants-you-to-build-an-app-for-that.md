# 伊冰？微软希望你为此开发一个 iPhone 应用程序

如果你在 iPhone 上进行网络搜索，你很可能会使用 Safari 浏览器的内置搜索栏。如果你正在使用它，你很可能会用谷歌搜索，因为它是预装选项。而事实上，如果你想在设置中更改它，唯一的其他选项是雅虎搜索。如果你想使用 Ask 或 Bing 之类的东西，你就必须访问它们各自的网页(至少在 Bing 开始支持雅虎搜索之前)。但是今天微软[发布了](https://web.archive.org/web/20221209135947/http://www.bing.com/community/blogs/developer/archive/2009/08/27/sprinkle-some-cocoa-on-bing.aspx)一种方法，通过一个新的 SDK 从你的 iPhone，甚至你的 Mac 上的应用程序中获得 Bing。

Bing iPhone 和 Mac SDK 可供任何人在这里下载[，它甚至是开源的(在微软公共许可下)。以下是它包含的内容:](https://web.archive.org/web/20221209135947/http://ibing.codeplex.com/)

*   从 Cocoa 或 Cocoa Touch 应用程序中轻松查询 Bing 的能力。
*   执行同步和异步查询。
*   在 Bing 中搜索网页、图片、视频、新闻和电话簿结果。

微软还写道，“*SDK 旨在消除手动解析 XML 或 JSON 以便与 Bing API 通信的头痛问题。*

幽默的是，包含 SDK 的域名以“ibing”开头，这无疑是对苹果品牌的一种玩弄。当然，许多公司在他们的应用程序中使用相似的品牌，但是微软和苹果是激烈的竞争对手。但很高兴看到微软创建了一个开源 SDK，为开发者提供了更多 iPhone 和 Mac 应用程序的选择。自今年早些时候推出以来，必应已经采取了许多明智的举措。现在，希望苹果不会在使用 Bing 的应用程序审批过程中做任何有趣的事情(尽管我不明白他们为什么会这样做，例如，很多应用程序都使用 Bing 地图)。
