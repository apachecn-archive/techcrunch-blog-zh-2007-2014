# (创始人故事)Reddit 的亚历克西斯·奥哈尼安向“乔布斯勋爵”鞠躬，并抨击投资者

在本期《T2》创始人故事中，主持人[克里斯·迪克森](https://web.archive.org/web/20221205132511/http://www.crunchbase.com/person/chris-dixon)和 [Reddit](https://web.archive.org/web/20221205132511/http://www.crunchbase.com/company/reddit) 联合创始人[亚历克西斯·奥哈尼安](https://web.archive.org/web/20221205132511/http://www.crunchbase.com/person/alexis-ohanian)(他现在在 Y Combinator 和 Hipmunk 工作)中，Color 被抨击为所谓的创业泡沫的促成者，而 Web 1.0 时代投资者的兄弟会态度也好不到哪里去。

在一个跨越各种话题的对话中，你会听到奥哈尼安描述“乔布斯勋爵”如何间接促成了 Zappos 和 Instagram 的成功，初创企业需要解决的巨大机会，以及当今投资者的态度调整；对于这类初创企业来说，动用大量现金(和其他东西)是不够的。

请务必观看视频，听听对上述问题的见解——以及为什么如果推测的泡沫破裂，我们可能会看到更软的着陆。

下面，奥哈尼安描述了他的领导风格，为什么他认为初创企业会失败(在迪克森的追问下，奥哈尼安表示，这更多是因为“糟糕的标识”)，以及奥哈尼安在招聘时寻求的是什么。
