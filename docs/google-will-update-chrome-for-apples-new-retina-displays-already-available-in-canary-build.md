# 谷歌将为苹果新的 Retina 显示屏更新 Chrome，该显示屏已经在 Canary Build 上市

苹果新推出的配有 Retina 显示屏的 MacBook Pro 无疑推动了现代屏幕的发展，但这也意味着开发者必须让他们的应用适应这些高分辨率，以确保他们的应用看起来不错。在本周早些时候的声明中，苹果表示 Adobe 和 Autodesk 都已经在开发他们产品的视网膜优化版本(尽管双方现在都表示[他们的软件还需要一段时间](https://web.archive.org/web/20221206190642/http://www.macworld.com/article/1167250/developers_retina_optimized_mac_apps_will_take_time.html)。看起来谷歌会更快一点。该公司今天[宣布](https://web.archive.org/web/20221206190642/http://chrome.blogspot.com/2012/06/chrome-and-new-shiny.html)它首次尝试让 Chrome 为这些高分辨率显示器做好准备，已经可以在它高度实验性的[金丝雀频道](https://web.archive.org/web/20221206190642/https://tools.google.com/dlpage/chromesxs?platform=mac)上看到。谷歌表示，它“致力于打磨 Chrome，直到它在那台机器上发光。”

Canary builds 是谷歌最具实验性和前沿性的 Chrome 版本(比 dev 频道更甚)。这些构建在发布之前没有经过测试，甚至很有可能无法运行。不过，如果你是已经拥有配备 Retina 显示屏的新 MacBook Pro 的少数幸运儿之一，你现在可以看到 Chrome 在这些机器上应该是什么样子(当然，苹果自己的 Safari 已经在这些新屏幕上呈现高分辨率文本)。

谷歌表示，目前的版本已经显示了其为 Chrome 带来高分辨率支持的“早期成果”。仍然有更多的工作要做，但是开发者们“正在全力以赴让 Chrome 变得尽可能的漂亮。”
