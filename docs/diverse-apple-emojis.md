# 苹果承诺让 iOS 表情符号更具种族多样性

在麦莉·塞勒斯和演员塔吉·莫里抱怨卡通表情缺乏多样性后，苹果公司发誓要更新其表情符号。MTV Act 博主乔伊·帕克[就此事给苹果 CEO 蒂姆·库克发了一封电子邮件](https://web.archive.org/web/20221006162237/http://act.mtv.com/posts/apple-responds-to-lack-of-diversity-in-emojis/)，得到了苹果全球企业沟通副总裁凯蒂·科顿的回复:

> “蒂姆把你的邮件转发给了我。我们同意你的观点。我们的表情符号基于 Unicode 标准，这是它们在许多平台上正确显示所必需的。表情符号字符集需要更加多样化，我们一直在与 Unicode Consortium 密切合作，努力更新标准。”

Cotton 没有透露苹果将于何时更新表情符号。事实上，使表情符号更加多样化的非官方运动已经持续了一年多，麦莉·赛勒斯回应了丝芙兰的一条推特，要求[增加更多指甲油表情符号，要求苹果增加代表黑人的表情符号](https://web.archive.org/web/20221006162237/http://betabeat.com/2012/12/miley-cyrus-emoji-racist-iphone/)。

帕克指出，让表情符号更加多样化的努力遭到了嘲笑。例如，对一个名为“表情符号是种族主义者吗？”的快速公司故事的大多数评论我们对此不屑一顾，从“lmao wow 少数民族从不快乐”到更公平的回答，指出表情符号首先出现在日本的 iOS 设备上，如果他们有种族的话，这使他们成为亚洲人。但是帕克写道,“虽然它可能‘只是’一个表情符号，但代表所有种族和性别是迈向更平等社会的重要组成部分。”。"
