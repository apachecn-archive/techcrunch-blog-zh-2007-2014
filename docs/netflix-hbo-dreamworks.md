# 网飞用梦工厂协议打击 HBO

网飞最近因其拆分流媒体和 DVD 业务的计划受到了一些打击，但现在它正在反击。该公司与梦工厂签署了[协议，这将使其在付费电视窗口期间通过互联网获得电影和电视节目。网飞击败 HBO 赢得了这笔交易，HBO 目前是第一家播放梦工厂电影的电视网络。](https://web.archive.org/web/20230203140435/http://www.nytimes.com/2011/09/26/business/media/netflix-secures-streaming-deal-with-dreamworks.html)

没错，网飞出价高于 HBO。据估计，网飞梦工厂每部电影的价格为 3000 万美元，而 HBO 目前每部电影的价格为 2000 万美元。这正是网飞现在需要做的事情，让流媒体与有线电视并驾齐驱。网飞将不得不与其他电影公司达成更多类似的协议，但这表明它未来将更加积极。通常情况下，网飞会购买较晚时段的流媒体播放权，通常是在付费有线电视频道开始播放电影之后。

然而，这笔交易要到 2013 年才会生效。与此同时，到 2012 年初，由于与 T4 Starz 的谈判失败，网飞将失去迪士尼和索尼电影的流媒体播放权。

索尼和迪斯尼是更大的电影公司，每年生产的电影比梦工厂多得多。如果网飞想被认为是常规电视的重要替代品，它还必须从更大的好莱坞电影公司那里赢得更多的流媒体协议。
