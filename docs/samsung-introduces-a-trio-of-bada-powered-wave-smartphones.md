# 三星推出三款 Bada 驱动的 Wave 智能手机

今天对三星来说是个大日子。如果你还没听说，这家手机制造商将于今晚在纽约发布其旗舰手机 Galaxy S II 的美国版本(你可以打赌我们会出席)。但如果这还不够，三星今天还宣布了三款新的 Bada 智能手机，每款都带来了一些特别的东西。

其中的明星将是三星的 Wave 3，它有着相当不错的规格，尽管它无法与糟糕的 Galaxy S II(我一直在谈论它)相提并论。这三款手机都配备了三星新的 [BBM 风格的跨平台消息服务 ChatOn](https://web.archive.org/web/20230204195817/https://techcrunch.com/2011/08/29/samsung-to-launch-chaton-for-cross-platform-mobile-chat-service/) ，我们甚至看到了一些 NFC 功能。

Wave 3 被认为是 Bada 的旗舰产品，这可能就是为什么我们已经[听到了一些关于它的细节](https://web.archive.org/web/20230204195817/https://techcrunch.com/2011/08/25/samsungs-ifa-app-reveals-new-galaxy-tab-wave-handset-galaxy-note/)。与此同时，Wave M 虽然是一款中档设备，但对 NFC 的支持紧随其后。Wave Y 是一款入门级智能手机，规格有限，但我们预计这个小家伙的价格会非常有吸引力。

这里是一个快速的规格运行:

**三星第三波:**

*   Bada 2.0
*   4 英寸 Super AMOLED 显示屏
*   1.4 GHz 处理器
*   9.9 毫米拉丝金属外壳
*   社交中心、音乐中心和 ChatOn
*   带 LED 闪光灯的 500 万像素后置摄像头
*   用于视频聊天的 VGA 前置摄像头

**三星波 M:**

*   Bada 2.0
*   3.65 HVGA 320×480 显示屏
*   832 MHz 处理器
*   NFC 功能
*   社交中心和 ChatOn
*   带 LED 闪光灯的 500 万像素后置摄像头
*   用于视频聊天的 VGA 前置摄像头

**三星 Y 浪:**

*   Bada 2.0
*   3.2 英寸 HVGA 320×480 显示屏
*   ChatOn
*   200 万像素后置摄像头，带 LED 闪光灯

虽然 Wave 3 和 Wave M 肯定可以保持自己的优势，但它们可能不太适合智能手机爱好者。Bada 是一个相当智能的 OS，该表扬的地方我们一定会给予表扬。但是 Android 和 iOS 有一些你无法击败的硬件。

另一方面，这些手机将完成任务，而且价格可能相当合理。显然，这完全取决于你在寻找什么。随着今晚 GSII 的发布和这些 Wave 手机的上市，你将有很多选择。
