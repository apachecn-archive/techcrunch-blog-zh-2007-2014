# 散居者再次传递帽子 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/12/diaspora-passing-the-hat-again/>

自从 [Diaspora](https://web.archive.org/web/20230204233715/http://diasporafoundation.org/) [筹集了 20 多万美元](https://web.archive.org/web/20230204233715/https://techcrunch.com/2010/06/02/diaspora-project/)用于创建一个替代的、分散的社交网络以来，已经过去一年多了。不到三个月后，[项目的第一张截图](https://web.archive.org/web/20230204233715/https://techcrunch.com/2010/09/15/diaspora-revealed/)在网上发布，但从那以后——嗯，说我们没有听到任何消息是不真诚的(例如，他们上个月才发出 alpha 邀请)，但他们肯定没有在那里制造太多噪音。

虽然他们 Kickstarter 项目的最初目标只是 1 万美元，但这个数字的 20 倍仍然不足以开发和推出互联网上最大网站的竞争对手。所以他们要求更多。

当然，这并不可耻:当咖啡喝完的时候，世界各地的创业公司都会向他们的风投、投资者，甚至朋友和父母乞讨。许多软件项目完全由捐赠资助。Diaspora，[在一封略带 twee 的成员邮件和博客文章](https://web.archive.org/web/20230204233715/http://blog.diasporafoundation.org/2011/10/12/we-love-you.html)中解释说，他们需要一点现金来保持奇迹的发生:

> 在其核心，Diaspora*是一个新的社区，一个独特的自由社区，基于我们如何在线体验社区的积极愿景。
> 
> 这也是社区的努力。这就是为什么这么多人尽其所能做出贡献。以及为什么我们希望你能花点时间捐 25 美元。-，或任何你能做的，来支持今天的愿景。

如果你愿意的话，这是捐赠链接。如果你还没有加入阿尔法，你会收到邀请。

在我看来，他们是在面对推出产品的现实。虽然他们可以在一两个月的咖啡因和正义驱动的高生产率中敲出指导原则、基本 UI 等，但魔鬼总是在细节中。让它变得安全、可伸缩、灵活和可访问与它背后的想法一样重要。毕竟，除非你是有色人种，否则你只有一次大的机会。

希望社区会用一点钱渡过难关，让他们继续下去。我渴望看到这个项目，不是因为我对脸书或 Google+特别不满意，而是因为它与那些服务不同，真正以用户为中心，而不是广告或数据农业。这就像是近来的一股新鲜空气。
