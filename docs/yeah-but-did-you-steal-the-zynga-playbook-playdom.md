# 是的，但是你偷了 Zynga 的剧本吗，Playdom？

晚了一天，但社交游戏网站(以及[Zynga](https://web.archive.org/web/20230304114758/http://www.zynga.com/)-敌对者】 [Playdom](https://web.archive.org/web/20230304114758/http://www.playdom.com/) 终于回应了我们对本周早些时候他们遭遇的[诉讼和临时限制令](https://web.archive.org/web/20230304114758/https://techcrunch.com/2009/09/10/zynga-accuses-playdom-of-stealing-trade-secrets-judge-issues-temporary-restraining-order/)的评论请求(所有法律文件都在这里)。

这份声明是今天早些时候通过电子邮件发给我们的，简短而甜蜜，包含的信息非常少:

> 鉴于 Zynga 对诉讼的偏好，这起诉讼并不令人意外。我们不相信把不必要的诉讼作为一种商业策略，我们很担心看到像我们这样光明和有前途的行业被这种策略拖垮。
> 
> 我们对 Zynga 的“剧本”或“秘方”不感兴趣。我们的力量来自我们 111 名有才华的人，我们将有力地捍卫自己不受这种干扰。

这起诉讼源于七名前 Zynga、现已成为 Playdom 的员工，他们可能会也可能不会带着一些专有文件去新的工作岗位。在 Playdom 被指控窃取的文件中，有一份迅速成为传奇/神话的“Zynga Playbook”:*“Zynga Playbook 实际上是包含 Zynga“秘方”的食谱，它的内容对 Playdom 这样的竞争对手来说是无价的，”*Zynga 在诉讼中表示。

Playdom 偷了吗？他们只是说他们对这份文件“不感兴趣”。在我看来，他们能确定这一点的唯一方法是阅读它。我是说，如果纽约时报有剧本，我肯定会感兴趣的。除非我读了之后发现它没什么意思。所以我再问一遍，Playdom。你偷了 Zynga 的剧本吗？

如果你看过，能给我一份吗？
