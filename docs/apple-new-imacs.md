# 苹果用新的英特尔处理器、高速 802.11ac Wi-Fi 和更快的闪存更新了 iMac

苹果公司刚刚发布了其 iMac 一体机的升级版。此次更新是一次小的更新，不像去年年底的大规模重新设计，但它为 iMac 系列带来了全新的性能和内部规格。新功能包括英特尔四核处理器、802.11ac Wi-Fi 和速度更快的 PCIe 闪存。

这些新的 iMacs 现在在基本的 21.5 英寸型号上配备了 2.7GHz 四核英特尔酷睿 i5 处理器，这是第一款采用英特尔 Crystalwell 架构的台式机，以及新的 Iris Pro 显卡。Iris Pro 是一款新的英特尔 GPU，在中档游戏电脑上很受欢迎，应该会比以前的型号有很大的改进。还有最高 3.4GHz Core i5 Haswell 处理器的 CTO 选项，顶配机型上的 NVIDIA GeForce 700 系列显卡，最高 4GB 专用内存。

802.11ac 网络意味着新的 iMacs 可以与最新的 AirPort Extreme 和 Time Capsule 路由器配合使用，这是今年早些时候推出的无线标准。据苹果公司称，基于 PCIe 的闪存应该比上一代机型提供高达 50%的速度提升，以加快启动、加载应用程序和整体系统性能。全闪存 CTO 选项现在的存储容量最高可达 1 TB，融合驱动器有 1TB 和 3TB 两种。

8GB 内存和 1TB 驱动器现在是标准配置，最大 32GB 配置选项可用于定制订单。根据苹果的新闻稿，苹果今天开始在其网站和商店销售新款 iMac。如果客户安装在自己的摇臂或其他硬件上，他们可以选择支架安装版本和内置 VESA 适配器的版本。基本配置的起价为 1299 美元，最高可达 1999 美元。

最高端的 27 英寸机型提供了高达 40%的图形速度，结合 PCIe 和新的英特尔 Haswell 四核处理器，对于任何玩游戏或进行重型视频编辑的人来说，这可能是一个有吸引力的包。此外，这一切都包裹在更薄的机身设计中，正如我在[对去年型号](https://web.archive.org/web/20230131034421/https://techcrunch.com/2012/12/04/apple-2012-21-inch-imac-review/ "2012 21.5-Inch Apple iMac Review: Slim, Sleek, And Stylish, But Far From Shallow")的评论中提到的那样，这是 iMac 美学上的一大进步。
