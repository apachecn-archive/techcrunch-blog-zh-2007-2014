# Imeem 在任何地方都会吞并一家年轻的初创公司。调频

![anywhere_imeem.png](img/18b39b2dbb25d41ac97a48e4c7cb33bd.png)经过不到一年的运营，团队在[随处可见。FM](https://web.archive.org/web/20230216130304/http://anywhere.fm/) 今天提前获得了回报，老牌社交音乐服务 [Imeem](https://web.archive.org/web/20230216130304/http://imeem.com/) 以未披露的金额收购了他们，很可能是现金和股票的混合形式。iTunes 风格的网络音乐播放器已经从 Y Combinator 和天使投资人那里筹集了不到 10 万美元的资金，这使得提前退出可能低于 500 万美元。任何地方。拥有 60，000 名用户和超过 900 万首上传歌曲的 FM 将继续存在，但创始人将加入 Imeem 以完成他们的盈利，并继续在 Imeem 的旧金山办公室开发他们的 iTunes 风格的音乐播放器。

无论是在 Imeem 还是任何地方。FM 在这笔交易中看到了很多协同效应。任何地方。FM 拥有我所见过的最好的上传和播放器界面，但是缺乏一个可靠的盈利方法。Imeem 将带来其音乐交易和销售团队来支持这项服务，并希望利用 Anywhere 的客户端 iTunes sync uploader、buddy radio，特别是推荐技术。只需轻轻一点，Anywhere 的上传程序就可以上传您的整个 iTunes、WinAmp 或 Windows Media Player 库，包括个人播放列表、歌曲评级和播放次数。在一封电子邮件中，Imeem 的首席执行官道尔顿·考德威尔(Dalton Caldwell)暗示了该公司的未来，他说:“我认为，一个完全授权的优秀而完整的产品将会赢得我们现在看到的支离破碎的市场。”我完全同意。

虽然目前没有宣布，但任何地方。FM 将有可能获得与主要唱片公司达成的相同的许可协议。这些交易允许用户免费播放朋友的 500 万首歌曲中的任意一首。被包括在交易中意味着任何地方。FM 可以避免网络广播规则限制歌曲播放的频率和顺序。

与所有大公司达成协议标志着 Imeem 历史上的一个重大转折，结束了早先困扰该网站的诉讼。但这些交易至少附带了环球公司以某种股票形式收取的一磅肉，以及一大笔预付现金。金融时报[称支付金额为 2000 万美元，尽管 Imeem 不同意。MP3tunes.com 的迈克尔·罗伯逊和早先的 MP3.com 称之为“T4”死刑判决。](https://web.archive.org/web/20230216130304/http://www.ft.com/cms/s/0/ff0a7e34-a6c3-11dc-b1f5-0000779fd2ac.html)

虽然他们的高管自己承认的广告支持模式还有待证明，但 Imeem 在竞争中有很大优势。他们是一个大型图书馆的合法用户，根据他们自己的统计，目前每月有超过 2000 万个唯一用户，每天有 65，000 个新注册用户。去年九月，康姆斯克将他们列为增长最快的社交网站。如果广告支持的音乐要成功，Imeem 是值得关注的创业公司。
