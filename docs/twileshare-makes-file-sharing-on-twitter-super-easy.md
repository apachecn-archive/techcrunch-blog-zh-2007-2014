# Twileshare 让在 Twitter 上分享文件变得超级简单

我们已经习惯了在 Twitter 上分享照片(从 [Twitpic](https://web.archive.org/web/20230203000111/http://twitpic.com/) 到 [Instagr.am](https://web.archive.org/web/20230203000111/http://instagr.am/) )、视频( [YouTube](https://web.archive.org/web/20230203000111/http://youtube.com/) 到 [Qik](https://web.archive.org/web/20230203000111/http://qik.com/) )和音频( [Cinch](https://web.archive.org/web/20230203000111/http://www.cinchcast.com/) 到 [Soundcloud](https://web.archive.org/web/20230203000111/http://soundcloud.com/) )。但是我们通过 Twitter 做了很多文件共享吗？不是约会。

有 [Tweetshare](https://web.archive.org/web/20230203000111/http://www.tweetshare.com/) 、 [Filesocial](https://web.archive.org/web/20230203000111/http://filesocial.com/) 和 [Tweetcube](https://web.archive.org/web/20230203000111/http://www.tweetcube.com/) 但它们不是你在普通用户流中看到的东西——或者至少不是我的，我[关注](https://web.archive.org/web/20230203000111/http://twitter.com/mikebutcher)近 4000 个账户。

Twileshare 仍处于测试阶段，并在圣诞节前推出，但据我所知，它已经有了大量的病毒式流量。现在谷歌上有 24000 次提到它。

通过 Twitter 登录后，你可以上传和分享大小不超过 20MB 的图片、pdf、Word 和 Excel 文件。几乎任何类型的文件都可以共享，pdf 和 Word 文档也可以通过 Scribe 的 API 放在页面上，这样你就可以在页面上简单地查看文档，而不是下载到桌面上你想看但还没有看的其他 pdf 文件中。您还可以查看您的文件被查看和发布的次数，对上传的文件进行评论。

尽管 Twileshare 的创始人丹尼·布尔(Danny Bull)和瑞安·福斯特(Ryan Foster)在上面投放了谷歌广告，但他们对收费服务持反对态度——这一点尚无定论。更有可能的是组织的品牌渠道。

你还不能 DM 一个文件链接，但它有可能在计划中——这将使它变得更加有趣，我们应该说。
