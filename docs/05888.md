# 丹尼斯·克劳利似乎证实了他们拒绝了 1.4 亿美元的收购要约

![](img/a9c4de76dd91469fadfbb863c674e9bc.png)

在 Le Web 10 上，当主持人 Loic Le Meur 问他该公司最近拒绝的收购报价有多大时，每个人都看到了 Foursquare 首席执行官 [Dennis Crowley](https://web.archive.org/web/20221209003955/http://www.crunchbase.com/person/dennis-crowley) 脸上尴尬的表情。

Le Meur 猜测约为 1.4 亿美元(没有真正提到报价来自脸书、雅虎还是其他公司，但他们是潜在的竞标者，当问题出现时，舞台上的话题是脸书)。

克劳利没有否认，然后继续承认这个数字是“大概的”。

这意味着 Foursquare 拒绝了大约 1.4 亿美元的报价，可能来自脸书，当时他们只筹集了大约 135 万美元的种子资金——在收购谈判平息后，他们继续筹集了 2000 多万美元。有远见，还是愚蠢？勇敢，还是泡沫？

值得一提的是:Foursquare 刚刚超过了 500 万用户，每天有 200 万人签到，25000 名新用户。
