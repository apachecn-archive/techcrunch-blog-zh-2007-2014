# Windows 8.1 将于太平洋时间明天凌晨 4 点开始推出

明天，太平洋夏令时凌晨 4 点，Windows 8.1 将开始向 Windows 8 用户推出可选更新。新软件代表了微软对其 Windows 8 操作系统所下赌注的重置和重生。

Windows 8.1 包含一系列新功能，包括新的内置应用程序、全新的用户界面变化以及更强大的搜索技术，该技术将 Bing 提升为微软服务套件的一个组件。SkyDrive 在 Windows 8.1 中也有突出的地位，因为微软努力确保它继续管理用户的文件，无论它们存储在哪里。(为了快速了解为什么这很重要，[点击这里](https://web.archive.org/web/20221209035307/https://beta.techcrunch.com/2013/09/16/box-announces-box-notes-a-lightweight-editing-tool-that-is-an-opening-salvo-against-microsofts-office/)。)

相信我，你想升级。Windows 8.1 之于 Windows 8，如同 Office 2010 之于 Office 2007。

明天我将运行最终代码(微软没有向 TechCrunch 提供最终版本的副本),并报道更大的发布本身。然而，由于这是一个阶段性的展示，不是每个人都能在太平洋时间明天凌晨 4 点看到代码。新的操作系统将在不同时间作为更新出现在 Windows 应用商店中，这取决于您的位置。你所要做的就是好好睡一觉，当你醒来的时候，操作系统要么已经准备好等你了，要么已经在路上了。

Windows 8.1 甚至比 Windows 8 更依赖于服务，这带来了更多潜在的故障点——如果 SkyDrive、Bing、Skype 或其他服务明天出现问题，这将破坏操作系统本身的发布。也就是说，微软的服务最近或多或少变得稳定了。

有趣的是，大多数 Windows 8 用户是否会转向 Windows 8.1，以及以什么样的速度。我们将能够独立跟踪 Windows 8.1 的市场份额，但我怀疑微软会分享一些统计数据，因为它们变得相关和扩大。

所以今晚标志着 8.1 级风暴前的平静。消费者会有积极的反应吗？新法规会刺激销售吗？它能与即将推出的新硬件兼容吗？大约 16 小时后，我们将开始回答这些问题。
