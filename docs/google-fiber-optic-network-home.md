# 谷歌计划为超过 50，000 个家庭提供 1Gb/秒的光纤宽带网络

![](img/58eb597df7a9465dd8cd227e1a24793f.png)

谷歌正在进军宽带业务。该公司计划将自己的“实验性”光纤网络部署到至少 50，000 个家庭，也许多达 500，000 个。该光纤网络将提供每秒 1 千兆比特的速度，比美国目前提供的住宅光纤服务快 20 多倍。该公司在其博客中写道:

> 我们计划在美国的一小部分试验地点建设和测试超高速宽带网络。我们将通过每秒 1 千兆比特的光纤到户连接，提供比大多数美国人今天获得的网速快 100 多倍的网速。我们计划以有竞争力的价格向至少 50，000 人甚至可能高达 500，000 人提供服务

这项服务在价格上与今天有线电视和电话公司的宽带服务有竞争力，但速度会快得多。威瑞森和康卡斯特一定很激动。谷歌表示，他们正在试验基础上推广新的杀手级应用，这些应用将利用更快的速度，试验更好的方式将光纤部署到家庭，并为更开放的宽带接入创造压力。它认为自己的努力是对美国政府国家宽带部署计划的补充，它也支持美国政府的国家宽带部署计划。希望被考虑使用谷歌服务的社区和市政当局可以在这里申请。

谷歌在全球拥有自己庞大的暗光纤网络，可以连接其数据中心，加快搜索速度，并降低每月在 YouTube 上播放数十亿视频的成本。通过这个项目，谷歌迈出了将光纤主干连接到消费者家中的第一步。目前还不清楚谷歌宽带订阅服务将会提供什么样的服务，但可以肯定的是，谷歌将会是默认的搜索引擎，Gmail 将会是默认的电子邮件。也许他们可以提供谷歌语音服务和一部能让你通过 WiFi 通话的安卓手机。
