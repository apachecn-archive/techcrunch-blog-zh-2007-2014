# DotWar:基于 Twitter 的策略游戏

[![](img/8296b0bc367304131f1489be3cd585c2.png "dotwar")](https://web.archive.org/web/20221207232734/http://www.crunchgear.com/2010/06/16/dotwar-twitter-based-strategy-game/dotwar/)

这当然不是第一款基于 Twitter 的游戏《T4》，但《T5》这款日本制造的策略/动作游戏是一款非常令人满意的消磨时间的游戏。这个游戏是双语的(英语/日语)，实际上是在 Twitter 之外的一个单独的网站上玩的。游戏玩法非常简单:只需输入你的 Twitter 账号，就可以派出数百甚至数千个战斗“单位”去对抗另一个玩家/Twitter 用户。

[![](img/45d38b49483134c032badadaf0a336f3.png "dotwar2")](https://web.archive.org/web/20221207232734/http://www.crunchgear.com/2010/06/16/dotwar-twitter-based-strategy-game/dotwar2/)

你可以用鼠标控制你的单位的移动。由于不需要注册或签约，你基本上可以让任何 Twitter 用户与任何其他用户一决高下，例如在“对抗模式”中。目标是在一定时间内摧毁每个玩家拥有的三块水晶。

DotWar 是由一个叫[“Sipo”](https://web.archive.org/web/20221207232734/http://sipo.jp/blog/)【JP】的日本软件工程师用 Flash 创建的。

[![](img/16be76e6851889c1a8dd8dfd3c92e68f.png "dotwar3")](https://web.archive.org/web/20221207232734/http://www.crunchgear.com/2010/06/16/dotwar-twitter-based-strategy-game/dotwar3/)

目前它非常流行，所以在你真正开始玩之前，你可能需要重新加载网站几次(这也很难)。
