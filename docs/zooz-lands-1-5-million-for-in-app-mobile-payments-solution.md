# ZooZ 获得 150 万美元的应用内移动支付解决方案

**独家-**[ZooZ](https://web.archive.org/web/20230203090222/http://zooz.co/)，不要与 [ZOOZ Mobile](https://web.archive.org/web/20230203090222/http://www.crunchbase.com/company/zooz-mobile) 混淆，已经在由 [lool ventures](https://web.archive.org/web/20230203090222/http://lool.vc/) 领投的种子轮融资中筹集了 150 万美元(是的，他们确实是这么叫的)。

参与这轮融资的其他投资者包括 Rhodium、基马风险投资公司、Orefa Investment、Jaina Capital、A.M. Holdings 以及私人投资者 Eilon Tirosh 和 Yuval Tal。

ZooZ 提供了一个全面的端到端支付解决方案，让移动应用程序开发人员可以通过 SDK 轻松地将数字支付和结账选项集成到他们的 Android 或 iOS 应用程序中。

这家初创公司声称，其支付解决方案只需要几行代码和最少的开发时间，但仍然为应用程序开发者提供了开始接受最终用户的应用内支付所需的一切。下面是[的工作原理](https://web.archive.org/web/20230203090222/http://zooz.co/howitworks.html)。

值得注意的是，ZooZ 坚持认为，它在构建其解决方案时已经考虑到了安全性和欺诈检测，并将所有敏感数据存储在符合 PCI 的云环境中。

ZooZ 平台将在最初的六个月测试阶段推出，在此期间该服务将完全免费。过了这段时间，ZooZ 将收取象征性的处理费用。

ZooZ 由首席执行官柳文欢·利维和首席技术官罗恩·莫雷斯基于 2010 年创立，总部位于以色列。
