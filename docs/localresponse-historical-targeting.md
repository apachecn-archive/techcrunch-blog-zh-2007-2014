# LocalResponse 现在可以根据历史推文和签到来定位广告

LocalResponse 刚刚宣布了一种新的客户定向广告方式。

以前，初创公司的广告客户可以根据社交网络活动来接触受众——例如，当有人在特定商店登记入住后，他们可以在商店中看到相关的广告。现在，这些广告商也可以查看历史数据，包括来自 Twitter、Instagram 和 Foursquare 的帖子。

该公司表示，其历史意图定位(HIT)将自己的技术与 DataSift 的[社交媒体监控工具相结合。](https://web.archive.org/web/20221207101059/https://beta.techcrunch.com/2012/08/21/datasift-debuts-more-powerful-tools-to-help-businesses-analyze-social-data/)

首席执行官兼联合创始人尼哈尔·梅塔(Nihal Mehta)告诉我，该功能将适用于该公司的所有广告客户，但他目前真的专注于几个行业，特别是零售和娱乐，因为这是该功能最明显有用的地方。有了 HIT，零售商可以通过向过去几个月来光顾他们商店的人提供特别优惠来吸引顾客再次光临。

或者，当电影制片厂发行蓝光/DVD 时，它可以将广告瞄准那些在影院上映时在推特上谈论这部电影的人。LocalResponse 表示，它在今年夏天与索尼电影公司一起在一个试点项目中测试了这项技术。

该公司声称与超过 125 家广告商合作。这一数字高于 7 月份，当时[梅塔称该数字为 75](https://web.archive.org/web/20221207101059/https://beta.techcrunch.com/2012/06/19/localresponse-7-billion/) 。(他还表示，LocalResponse 每月的浏览量超过 70 亿次。)
