# Seesmic 通过面向 Android 和黑莓的强大新应用进军移动领域 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/11/20/seesmic-ventures-into-mobile-with-powerful-new-apps-for-android-and-blackberry/>

Seesmic 度过了重要的一周。本周早些时候，为网络和桌面开发 Twitter 和脸书客户端的初创公司[在微软](https://web.archive.org/web/20221006194746/http://www.beta.techcrunch.com/2009/11/17/seesmic-launches-native-twitter-client-for-windows/)[专业开发者大会](https://web.archive.org/web/20221006194746/http://msdn.microsoft.com/en-us/microsoft-pdc-2009.aspx)上发布了一款原生 Windows 客户端。在今天的[实时计算](https://web.archive.org/web/20221006194746/http://www.beta.techcrunch.com/real-time-crunchup-sf/)上，Seesmic 首次进军移动领域，推出了令人印象深刻的安卓和黑莓应用，现在可以在这里下载[。对于 Seesmic 来说，这是一个关键时刻，因为这家初创公司现在正在征服所有的媒体——](https://web.archive.org/web/20221006194746/http://seesmic.com/)[网络、](https://web.archive.org/web/20221006194746/http://www.beta.techcrunch.com/2009/07/10/seesmics-browser-client-is-like-gmail-for-twitter/) [桌面和](https://web.archive.org/web/20221006194746/http://www.beta.techcrunch.com/2009/06/07/seesmic-to-launch-new-desktop-version-iphone-app-and-browser-based-client/)移动。我和 Seesmic 的联合创始人 Loic Le Meur 坐在一起测试这些应用。

**黑莓应用**

运行 OS 4.6 或更高版本设备的黑莓应用程序有一个非常光滑漂亮的界面，为各种效率的黑莓用户进行了优化。你可以从不同的时间线快速切换，轻松地从收件箱切换到提及，直接消息。另外，你可以在一个应用程序中监控不同的 Twitter 账户。当你发送推文时，你可以通过 Bit.ly 缩短一个链接，并通过 yFrog 上传图片或视频。一个引人注目的功能是能够直接通过电子邮件将推文发送给联系人。

该应用还包括搜索功能(Seesmic 将所有信息存储在缓存中)，当你滚动浏览电子邮件时，还会有新推文和 DMs 的通知(Le Meur 说，一只小浣熊会弹出一些新推文)。毫无疑问，黑莓应用程序的两个最佳功能是直接从应用程序和地理定位中查看您的 Twitter 列表的能力，因此您可以直接从应用程序中发布您的位置。Seesmic 刚刚添加了地理定位，因为 API 是昨天发布的。脸书集成将在不久的将来加入。

**安卓应用**

![](img/ccac9ac70508f59e08f62906d3d7efea.png)Android 应用程序还具有非常时尚的设计和用户友好的用户界面。最明显的优势是应用程序的速度。Le Meir 表示，该应用程序是 Android 的原生产品，并称其为“Android 的 Tweetie”通过触摸界面，您可以轻松浏览线索时间表、直接消息、不同帐户和@回复。你可以通过电子邮件发送推文，也可以轻松切换到横向模式。

当你发推文时，你会看到公开或通过 DM 发推文的选项，并通过 Bitl.ly 缩短链接。你可以通过 yFrog 轻松附加图片或视频。更妙的是，你还可以通过这个应用程序直接上传视频到 YouTube。您还可以访问其他用户的个人资料，查看他/她的推文，他/她关注的人，然后关注这些用户。此外，您还可以阻止或取消关注某个用户。

虽然 Twitter 昨天推出了[地理定位 API](https://web.archive.org/web/20221006194746/http://www.beta.techcrunch.com/2009/11/19/twitter-location-api/) ，但 Seesmic 尚未将其 API 集成到应用程序中。但该应用程序仍具有地理定位工具，允许你通过 Android 的 GPS 将你的位置嵌入推文中，这将在谷歌地图上显示你的位置。

该应用程序适用于所有型号的 Android，并针对新推出的 Verizon Droid 进行了优化。虽然 Android 应用程序没有列表功能或脸书集成，但 Le Meur 表示，这些功能将在不久的将来添加。![](img/3f41eea1402f31732800a76ccb5d1d9f.png)

**iPhone 应用程序**

几个月前，Le Meur [告诉我 Seesmic 的 iPhone 应用程序。但该应用仍在调整中。虽然我们不知道 iPhone 应用程序的详细信息，但我们知道它将具有 Twitter 和脸书状态更新的功能。](https://web.archive.org/web/20221006194746/http://www.beta.techcrunch.com/2009/06/07/seesmic-to-launch-new-desktop-version-iphone-app-and-browser-based-client/)

Seesmic 的黑莓应用将面临来自 [UberTwitter、](https://web.archive.org/web/20221006194746/http://www.ubertwitter.com/) [OpenBeak](https://web.archive.org/web/20221006194746/http://www.orangatame.com/products/openbeak/) (之前的 TwitterBerry)、 [Tweetcaster](https://web.archive.org/web/20221006194746/http://www.handmark.com/company/apps/tweetcaster/) ，甚至可能还有一个 [RIM 开发的](https://web.archive.org/web/20221006194746/http://crackberry.com/rim-working-native-twitter-client-should-they) Twitter 客户端的竞争。但 Seesmic 的应用程序充满了漂亮的功能，比如列表，甚至可能在未来集成脸书。它肯定会成为该领域的有力竞争者。Android 应用将面临来自一小群 Twitter 客户端的竞争，其中包括 Twitdroid、 [Swift、](https://web.archive.org/web/20221006194746/http://www.swift-app.com/)和 [TwitterRide。但是像黑莓应用一样，Seesmic 的安卓应用速度快、时尚、易于使用，并且很快将内置脸书，这使得它非常有吸引力。](https://web.archive.org/web/20221006194746/http://twitterride.net/)

正如我在上面写的，Seesmic 现在正全力以赴为网络、移动和桌面开发众多产品。Le Meur 说，Windows 客户端在 12 小时内有 1 万次下载。我完全期待 Android 和黑莓应用程序会得到同样的回应。随着这家初创公司继续开发新的创新产品，它正在慢慢蚕食竞争对手 Tweetdeck 的市场份额，并吸引了一大批新的追随者。但是科技公司之间有点友好的竞争从来都不是坏事。
