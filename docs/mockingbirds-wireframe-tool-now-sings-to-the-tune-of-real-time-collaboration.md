# 知更鸟的线框工具现在唱实时协作的曲调

![](img/adee9bfed4386a6a410bd2b5859563e8.png)

如果你正着手设计一个新网站，很少有比线框服务更好的起点:它们让你快速了解你的网站将会是什么样子，而不必处理剪纸或无休止的铅笔擦除。今天，一个更好的线框服务是离开测试与一些主要的新功能拖在它的正式推出。遇见知更鸟。

大约一年前，这家两人创业公司推出了一个基于 Cappuccino 网络框架的简洁界面(换句话说，它不像许多竞争对手那样使用 Flash)。自从推出测试版以来，用户已经增长到 55000 人，创建了近 100000 个项目。

今天，Mockingbird 推出了实时协作，这意味着多人可以启动他们的网络浏览器(无需安装)并开始编辑相同的页面布局。鉴于设计的协作性质，这显然是一件大事，在实践中它看起来非常巧妙(你可以在下面的视频中看到它的实际操作，或者你可以查看[演示页面](https://web.archive.org/web/20230202213904/https://gomockingbird.com/mockingbird/demo.html)亲自尝试一下)。

此次发布的另一个重大变化是:“知更鸟”准备打开收入水龙头，并开始为其更强大的计划收费。仍然会有一个免费版本，支持最多两个用户的一个项目，高级计划每月 9 美元起。
