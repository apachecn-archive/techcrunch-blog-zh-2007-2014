# Foursquare 现在允许企业“驱逐”假市长(抱歉，@Arrington)

这个选举日，将会有比大多数专家预测的更多的政治变动:Foursquare 启用了一个新功能，将结束虚假市长选举。我们已经与 hot location 初创公司确认，如果经过验证的企业认为他们的入住是欺诈性的，他们可以驱逐市长——一旦企业解雇了骗子，下一个候选人就会立即获得市长职位。

我的老板——一直用他的假签到表过着双重生活——渗透董事会议和内衣店——可以理解他心烦意乱。

自从 foursquare 推出以来，这款游戏一直不得不应对某种程度的“[欺骗](https://web.archive.org/web/20221208002738/https://beta.techcrunch.com/2010/02/16/foursquare-cheating/)”——人们喜欢在他们实际上没有去过的地方签到。在很大程度上，这是完全良性的，但更顽固的骗子已经成功地在他们很少去的地方获得了梦寐以求的“市长”位置，这导致了沮丧的用户按规则玩游戏。

但是现在这些虚假市长职位开始影响到更多的人——企业越来越多地给他们的市长提供特殊的促销活动，给予他们最忠诚的客户交易。但是，如果没有人能够获得这些奖励，这显然就失去了意义。不可避免的是，Foursquare 最终将不得不解决这个问题。

Foursquare 的发言人表示，这项新功能目前还处于早期阶段，未来可能会有所调整:

> 昨天，我们给了经过验证的企业主罢免市长的权力，如果他们有理由相信市长职位不是通过合法登记获得的。如果一个市长被一个企业主标记，他们将立即被免职，下一个用户将接任市长。
> 
> 我们仍处于试验这一功能的早期阶段，很可能会根据企业主的反馈进行调整。

Foursquare 在 4 月份开始打击这些虚假签到，当时它开始考虑用户的物理位置(在此之前，用户可以在 100 英里以外签到)。这是下一个合乎逻辑的步骤，尽管我确信不久我们就会听到用户被错误地驱逐出他们的统治。
