# Twitter 利用其第一个电视广告作为推出标签页面的机会

周日，推特在[发布了它的第一个电视广告](https://web.archive.org/web/20221210071320/http://www.youtube.com/watch?feature=player_embedded&v=1a51TozHcBU)，广告内容都是关于纳斯卡的。但除了获得纳斯卡推特粉丝的关注，这个广告还意味着展示如果他们与社交网络合作，品牌可以获得什么。根据我们所听到的，我们非常确定 Twitter 在 Pocono 400 期间在 TNT 上发布的广告完全是为了吸引人们对一个改版页面的注意，该页面显示了一个热门关键词的搜索结果——用 Twitter 的说法是一个标签。

商业内幕是第一个注意到 Twitter.com/#NASCAR 重定向到[twitter.com/hashtag/nascar](twitter.com/hashtag/nascar)的，一个赛车联盟的新登陆页面。在周四的博客上，[推特给出了一个预告](https://web.archive.org/web/20221210071320/http://blog.twitter.com/2012/06/off-to-races-with-nascar.html)，如果纳斯卡迷们收听[Twitter.com/#NASCAR](https://web.archive.org/web/20221210071320/https://twitter.com/hashtag/nascar)页面，他们将会期待什么。这包括来自车手、他们的工作人员和专家评论员的评论，以及一大堆美丽的幕后照片，所有这些都是通过赛车联盟的算法和策展的组合浮出水面的。这是 Twitter 和 NASCAR 帮助粉丝筛选所有噪音的一种方式，如果他们只是在搜索中查看#NASCAR 标签，就会发现这些噪音。

然而，更重要的是，这是 Twitter 首次基于其标签发布品牌页面。纳斯卡运动和电视广告不仅仅是针对赛车迷的——它们也是对一大群其他品牌营销人员的宣传，以此展示 Twitter 的标签页面的能力。

可以肯定的是，将来会有更多这样的合作关系。纳斯卡不会是最后一个利用 Twitter 标签页的体育联盟或大品牌。基于本周末实验的成功，我们预计更多的品牌会加入进来，根据他们自己的标签策划社交营销页面。他们不再只是购买赞助标签或推文，而是能够围绕人们所说的话来策划体验。

hashtag 页面的推出也将为社交营销者增加另一个渠道，并可能在一个已经拥挤和分散的市场中混淆用户的注意力。(老实说，整件事让我想起了 AOL keywords，那是一个品牌所有新闻和评论的去处。)仅仅有一个网站已经不够了——越来越多的品牌必须出现在用户期望它们出现的任何地方，这意味着一个脸书页面，现在，很明显，还有一个 Twitter 标签。
