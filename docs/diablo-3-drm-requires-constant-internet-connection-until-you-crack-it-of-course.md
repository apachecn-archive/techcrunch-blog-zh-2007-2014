# 《暗黑破坏神 3》的数字版权管理需要持续的互联网连接——当然，除非你破解了它

在线 DRM 的局限性也许实际上只是假设。可以想象一个必须一直在线的 DRM 方案，它可以优雅地处理中断并且永远不会限制你的游戏。不幸的是，暴雪并没有在今年最受期待的游戏之一暗黑破坏神 3 上安装这种 DRM。在一个给一些游戏网站的简报会上，有消息透露这款游戏确实需要一个活跃的网络连接才能玩。

还记得上一次顶级游戏[这么做](https://web.archive.org/web/20230203075626/https://techcrunch.com/2010/03/07/ubisofts-drm-servers-crash-locking-players-out-of-their-games/)吗？还记得中断、错误、挫折和缺乏沟通吗？暴雪说这是必须的，因为否则他们不能确定你没有作弊来创造你的角色。多好的解决问题的方法啊！

下面是 1up[报道的资金报价:](https://web.archive.org/web/20230203075626/http://www.1up.com/news/diablo-3-requires-online-when-playing)

虽然 Pardo 承认人们有时希望或需要离线玩游戏(例如互联网中断，或在飞机飞行期间玩笔记本电脑)，但他指出，安全性的提高以及上述好处超过了其他担忧。“我想在飞机上用我的笔记本电脑玩暗黑破坏神 3，但是，嗯，还有其他游戏可以玩。”

说的多好啊！多么令人难以置信的清晰陈述！他完全正确。还有其他游戏可以玩。与暗黑破坏神 3 不同，我会购买其他游戏。

我想到的第一件事就是让系统选择加入:你在 Battle.net 上创建一个角色，然后登录玩单人游戏，收集战利品，并一直被跟踪。持久网游就是这样的。已经很久了。但是如果你只是想离线玩呢？为什么，你创造了一个离线的角色，永远不能在网上玩。线上角色完全存储在暴雪的服务器上(比如说，我在*坏公司 2* 里的家伙)，离线角色存储在本地(哦，就像我在*坏公司 2* 里的*其他*家伙)。

暴雪并不愚蠢——或者我应该说，他们并不愚蠢到会阻止他们考虑这个解决方案。但如果他们认为这行得通，那他们就太傻了。Battle.net 可能是世界上最大的在线游戏公司之一，但是如果暴雪认为这个小障碍不会被破解，Battle.net 会被欺骗，那他们就大错特错了。如果他们认为破坏付费用户的游戏会减少盗版，那他们就是十足的疯子。在[电脑游戏玩家写的简报](https://web.archive.org/web/20230203075626/http://www.pcgamer.com/2011/08/01/diablo-3-cannot-be-played-offline/)中可以找到更多的戏谑。

暴雪公关需要注意的是:建议人们玩你自己的游戏以外的其他游戏，因为你的游戏会被故意屏蔽，这并不是真正的推销。
