# 诺基亚带着(非法)音乐来越？

![](img/f65bc2a4e11b18d78825fe9977dc6bbd.png)据报道，越南(RIAV)版权监督记录协会起诉 FPT 电信和诺基亚越南公司侵犯知识产权，索赔 300 万美元。代表全越南 60 家音乐制作公司的 RIAV 公司起诉这两家公司侵犯录制音乐的版权。

本月早些时候，RIAV 表示将对 FPT 电信提起诉讼，指控其通过网站 [Nhac So](https://web.archive.org/web/20221007090009/http://nhacso.net/) 在互联网上分享录制的音乐，并在 FPT 电信的互联网电视服务 IPTV 上使用版权材料。在两家公司之间的和解尝试失败后，RIAV 还对诺基亚越南公司提起诉讼，原因是该公司允许其客户通过 Nhac So 从 10，466 首歌曲数据库中下载 1，000 首受版权保护的歌曲。

诺基亚上周致信 RIAV，将矛头指向了 FPT 电信。据该公司称，FPT 向他们承认，它可能没有其网站上所有音乐内容的必要权利，尽管他们此前向诺基亚确认，他们确实拥有其网站上所有音乐内容的合法权利。

作为回应，RIAV 说，它承认诺基亚的善意，但申明诺基亚试图逃避责任。

这件案子引发了很多问题。最重要的问题是:如果诺基亚在越南发行音乐，它为什么要依赖第三方许可协议，而不是自己达成全球许可协议？
