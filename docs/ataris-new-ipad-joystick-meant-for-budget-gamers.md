# 雅达利的新 iPad 游戏手柄是为预算游戏玩家设计的吗？

ThinkGeek 的 iCade 当然有视觉天赋，但有眼光的 iPad 游戏玩家可能很快会有另一种选择，他们可能不需要复古。雅达利街机双动力操纵杆的存在最近在雅达利最受欢迎的应用程序的变更日志中得到证实，希望这意味着它将很快正式亮相。

细节仍然很少，但据 [Technabob](https://web.archive.org/web/20230203131056/http://technabob.com/blog/2011/09/04/atari-arcade-duo-joystick-for-ipad-icade/) 报道，Atari Arcade 操纵杆主要采用塑料机身，采用开放式设计，可以轻松移除 iPad。在这一点上，似乎很可能像 iCade 一样，操纵杆将(不管是好是坏)通过蓝牙连接到 iPad 来传输控制输入。

从本质上来说，操纵杆就像一个蓝牙键盘，某些键绑定到不同的屏幕动作上——就控制而言，这可能不是一个理想的解决方案，但比通过苹果的 [MFi 配件认证计划](https://web.archive.org/web/20230203131056/http://developer.apple.com/programs/mfi/)要容易得多。

现在下判断还为时过早，但考虑到设计和按钮布局，Atari Arcade 操纵杆可能很快就会成为一款经济型外设。Discovery Bay Games 还没有透露这款游戏手柄的价格或上市日期，但不要指望它一旦开始零售就能赚个盆满钵满。
