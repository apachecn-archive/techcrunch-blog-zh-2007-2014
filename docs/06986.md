# RIM 宣布 Playbook 的视频聊天和脸书应用

一款本应在发布时出现在 Playbook 上的应用——视频聊天，将于明天早上通过 OTA 更新的形式提供给平板电脑用户。除了前后摄像头支持，聊天应用程序还允许您进行 VOIP 和视频通话，以及基于用户 BBID 的连接。这款应用具有“强大的通话功能”，比如——我没有瞎编——“静音/取消静音”。
