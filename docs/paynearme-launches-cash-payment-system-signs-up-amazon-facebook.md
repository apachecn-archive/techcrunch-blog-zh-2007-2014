# PayNearMe 与 7-Eleven 推出现金支付系统；注册脸书亚马逊

8 月份，我们[给](https://web.archive.org/web/20221006065609/https://beta.techcrunch.com/2010/08/03/woot-test-paynearme-and-get-a-ton-of-free-facebook-credits/)写了关于 [PayNearMe、](https://web.archive.org/web/20221006065609/http://www.paynearme.com/)的文章，PayNearMe 是该公司的一款替代支付产品，其前身为 [Kwedit](https://web.archive.org/web/20221006065609/http://www.crunchbase.com/product/kwedit-promise) 。作为背景，Kwedit 在今年早些时候推出了另一款支付产品，引发了很多争议。但是这一个可能是赢家；PayNearMe 已经与一些知名商家签约，并于今天向公众发布。

PayNearMe 允许没有或不想使用信用卡或借记卡的人在美国大陆的 6，000 多家 7-Eleven 商店用现金购买产品。在参与的电子商务或商家网站上，消费者可以使用 PayNearMe 选项支付电子商务购物、电话订单、贷款还款、转账等。您只需在 PayNearMe 下订单，并打印出给定的收据。然后你拿着收据去 7-11 便利店，他们扫描后你用现金支付。一旦您付款，您与零售商或商家的订单将被履行。

该公司于 8 月开始测试这种支付脸书积分的形式，但现在正在与 Amazon.com、Progreso Financiero、MOL AccessPortal、m-Via、Lexicon Marketing、LLC、Adknowledge 的 Super Rewards、Money to Go 和 SteelSeries 进行整合。例如，你可以使用 Amazon.com 的 PayNearMe 在线购买礼品卡。

对于 7-Eleven，PayNearMe 与便利店的销售点集成在一起，当消费者付款时，它会实时通知收款人。商家还可以指示 PayNearMe 在消费者的注册收据上打印自定义消息，这些收据可用于打印礼品代码、门票、保险证明、贷款声明、法律披露等。

PayNearMe 门户网站打印定制信息的能力，可能使其成为销售交通票、保险产品甚至启动资金转账的理想选择。该平台希望不仅仅是一种在线购买商品的方式，PayNearMe 希望成为用户用现金支付但仍在线订购商品的首选方式。

PayNearMe 的创始人兼首席执行官 Danny Shader 表示，该选项对于 25%没有信用卡或借记卡的美国家庭来说非常理想，这些家庭发现在线交易更加困难。

虽然去一趟 7-11 为你在网上订购的商品付款的想法可能对一些人来说很奇怪，但 Shader 认为，对于只能用现金的用户来说，这种选择将能够作为一个可行的支付平台获得牵引力。PayNearMe 已经能够与亚马逊等商家达成一些高调的交易(Shader 表示更多交易正在进行中)；因此，看看替代支付方式是否会赢得忠实用户群应该会很有意思。
