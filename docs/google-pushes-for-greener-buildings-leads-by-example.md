# 谷歌推动绿色建筑，以身作则

今天早上，谷歌在其公共博客上谈论了公司对为员工建设更环保、更可持续的工作场所的关注。该公司参与了多项绿色计划，包括能源和水资源审计、绿色建筑材料的使用，以及让团队相互竞争成为绿色办公室的竞争计划。

此外，谷歌表示，它现在已经在加州山景城的谷歌总部增加了第一座获得 LEED[认证的建筑。](https://web.archive.org/web/20230205044033/http://www.usgbc.org/DisplayPage.aspx?CategoryID=19)

LEED 代表“能源和环境设计的领导地位”，是国际公认的绿色建筑评级认证计划。谷歌表示，这不是它在 LEED 的第一座建筑，但却是谷歌总部的第一座。

谷歌表示，新的 LEED 白金成就奖授予了使用健康建筑材料和做法的室内装修。该公司目前在全球拥有 450 万平方英尺的建筑面积，希望很快获得 LEED 认证。

谷歌使用的其他绿色建筑实践包括:

*   通过使用不含[生活建筑挑战红色清单材料](https://web.archive.org/web/20230205044033/https://ilbi.org/lbc/v2-0)和 [EPA 关注化学品](https://web.archive.org/web/20230205044033/http://www.epa.gov/opptintr/existingchemicals/pubs/ecactionpln.html)的材料，避免使用含有挥发性有机化合物( [VOCs](https://web.archive.org/web/20230205044033/http://www.epa.gov/iaq/voc.html) )和其他可能危害人类健康的已知毒素的材料
*   推动建筑行业在建筑材料标签方面采取更加透明的做法，并通过 [Pharos 项目](https://web.archive.org/web/20230205044033/http://www.pharosproject.net/)要求供应商满足透明度要求
*   投资于最高效的供暖、制冷和照明系统
*   定期进行能源和水审计
*   实施保护措施并开发最佳实践
*   尽可能使用可再生能源(如[这些太阳能电池板](https://web.archive.org/web/20230205044033/http://googleblog.blogspot.com/2006/10/corporate-solar-is-coming.html)

谷歌还开展了一项内部可持续追求计划，通过衡量其绿色清洁计划、用水和废物管理战略，让团队相互竞争，根据他们办公室的绿色表现来获得积分。
