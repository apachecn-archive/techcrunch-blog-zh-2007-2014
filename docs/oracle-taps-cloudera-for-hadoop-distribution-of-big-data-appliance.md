# Oracle 利用 Cloudera 发布大数据设备的 Hadoop 版本

甲骨文已经与商业分发和服务基于 Apache Hadoop 的数据管理软件和服务的初创公司 [Cloudera，](https://web.archive.org/web/20221225194258/http://www.crunchbase.com/company/cloudera/)合作，为甲骨文新发布的大数据设备提供 Apache Hadoop 分发版和工具。

Hadoop 是一个 Java 软件框架，诞生于 Google 发布的计算基础设施的开源实现，它是在 Apache Software Foundation 中发展起来的。Hadoop 支持在处理大量数据的大型商用计算机集群上运行的分布式应用程序。Cloudera 帮助分发 Hadoop，并围绕该技术提供实用服务，类似于[红帽](https://web.archive.org/web/20221225194258/http://www.crunchbase.com/company/red-hat)为 Linux 框架所做的事情。

Oracle 已经将 Cloudera 的发行版(包括 Apache Hadoop 和 Cloudera Manager 软件)集成到其大数据设备中，这是一个刚刚推出的集成系统，旨在为大数据提供高性能和可扩展的数据处理环境。该设备旨在帮助公司获取、组织和分析海量企业数据。

该设备是利用 Apache Hadoop 的硬件和软件的混合体，包括 Cloudera Manager、R 的开源发行版、Oracle NoSQL 数据库社区版、Oracle HotSpot Java 虚拟机和运行在 Oracle Sun 服务器上的 Oracle Linux。

2010 年，Cloudera [与 Quest Software](https://web.archive.org/web/20221225194258/http://news.cnet.com/8301-13846_3-20008242-62.html)合作，生产并发布了一款面向 Hadoop 的 Oracle 连接器。考虑到一些人认为甲骨文[收购 Cloudera](https://web.archive.org/web/20221225194258/http://www.theregister.co.uk/2011/05/11/oracle_buy_hadoop_cloudera/) 对这个企业巨头来说是有意义的，这种合作关系很有趣。
