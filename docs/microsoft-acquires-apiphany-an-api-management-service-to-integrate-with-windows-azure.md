# 微软收购 Apiphany，这是一个与 Windows Azure 集成的 API 管理服务

微软已经收购了 Apiphany，这是一家 API 管理公司，它将与 Windows Azure 集成。交易条款没有披露。

在 [Windows Azure 博客](https://web.archive.org/web/20230404042104/http://blogs.msdn.com/b/windowsazure/archive/2013/10/23/microsoft-acquires-apiphany.aspx)上有一篇关于此次收购的简短帖子，引用了 API 不断增长的价值以及可靠、安全和大规模公开 API 所需的新管理水平。在帖子中，微软总经理表示，Apiphany 为任何规模的企业提供了他们需要的工具来暴露特定的端点，同时选择性地控制对更敏感的数据和服务的访问。"

华盛顿特区的 Apipahny 与《华盛顿邮报》等客户合作，为公司管理其 API 提供了一个门户。API 分析师 Kin Lane 在去年 5 月的一篇评论中表示，Apiphany 有三个主要组件用于 API 部署和消费它有能力管理多个 API 带来的操作，为开发人员提供访问 API 资源和操作的各种组合的服务，以及访问在平台上注册的应用程序。

Apiphany 的收购指向 API 管理市场的 整合 。今年早些时候， 英特尔以超过 1.8 亿美元的价格收购了 Mashery，CA 收购了 Layer 7。 Mulesoft 收购可编程 Web 。然后脸书跳进来买了 解析 。

Apiphany 提供的服务将会被人们怀念，因为它是最后一批为开发者提供管理其 API 的廉价平台之一。每月花费 50 美元。屈服于市场和投资者需求的其他服务，比开发商更关注企业游戏。
