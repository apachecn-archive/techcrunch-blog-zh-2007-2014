# 诺基亚表示，由于准备将部门出售给微软 TechCrunch，Lumia 和其他手机销量在第四季度有所下降

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/23/nokia-says-lumia-other-phone-sales-declined-in-q4-as-it-prepares-to-sell-the-division-to-microsoft/>

这可能是诺基亚将其手机制造业务移交给微软之前的最后一个季度收益，这家芬兰手机制造商报告称，其第四季度 Lumia 设备的销量有所下降。该公司还报告了 4.08 亿€的非国际财务报告准则运营利润，但主要基于其 NSN 设备业务。因此，这些结果似乎证明了分拆手机制造业务的决定是正确的。

诺基亚本季度调整后的营业利润为 2.74 亿€——€2.43 亿来自 NSN。其非国际财务报告准则每股收益为€0.08。收入为€34.76 亿美元。

在对收益的预测中， [TC 的 Alex Wilhelm](https://web.archive.org/web/20221206102939/https://beta.techcrunch.com/2014/01/22/nokia-earnings-preview-the-lumia-question/) 建议诺基亚在整个假期季度应该很容易卖出超过 1000 万 Lumias。

如果诺基亚没有在第四季度财报中公布 Windows Phone 设备的销售数据，只提到智能设备净销售额(以及基本手机净销售额)有所下降，那么可以推测它的销量低于上个季度报告的 880 万部 Lumia。

**更新:**据《华尔街日报》报道诺基亚现已证实该季度售出 820 万辆 Lumias。2013 年全年 Lumia 销量为 3000 万部。

> 2013 年第四季度终止运营净销售额同比下降主要是由于手机净销售额下降，其次是智能设备净销售额下降。我们的手机净销售额受到竞争激烈的行业动态的影响，包括价格越来越低的智能手机的激烈竞争和我们低端产品组合的激烈竞争。我们的智能设备净销售额受到竞争行业动态的影响，包括竞争智能手机平台的强劲势头，以及我们的产品组合从 Symbian 产品过渡到 Lumia 产品。

在之前的收益报告中，对于其第三季度财年，诺基亚公布了€1.18 亿英镑(1.6 亿美元)的意外利润，该季度售出了 880 万部 Lumia 智能手机——高于其 Q2 的 740 万部 Lumia。诺基亚当季运营亏损€1.15 亿英镑(1.56 亿美元)。

在 Q1，诺基亚报告售出了 560 万部 Lumia——因此其 Lumia 设备的销量逐季增长。其第四季度的业绩扭转了这一趋势，并使诺基亚将手机业务移交给微软的决定看起来像是一个坚定的决定。

[早在九月](https://web.archive.org/web/20221206102939/https://beta.techcrunch.com/2013/09/02/microsoft-enters-into-7-1b-deal-to-buy-nokias-devices-and-services-business-and-license-key-patents/)诺基亚宣布将把其大部分手机制造业务出售给微软。该交易仍有望在今年第一季度完成。

诺基亚董事长兼临时首席执行官 Risto Siilasmaa 在一份声明中评论第四季度的业绩时说:“2013 年第四季度是诺基亚历史上的一个分水岭。在 11 月份的特别股东大会上，我们获得了股东们压倒性的强烈支持，同意将我们的手机业务出售给微软，我们正在努力确定诺基亚未来的发展方向。我对我们迄今为止在战略评估方面取得的进展感到满意，并对我们三项持续业务(NSN、HERE 和 Advanced Technologies)的未来机遇感到兴奋。

“在第四季度，诺基亚的持续业务产生了 12%的健康基本运营利润率。虽然今年第一季度对我们的持续经营来说是季节性疲软，但我们仍然预计微软交易的完成将显著改善诺基亚的盈利状况。”

一度是全球最大手机制造商的诺基亚，在 2011 年 2 月决定为智能手机设备转向微软的 Windows Phone 平台，并放弃自己的软件后，其支持率大幅下降。那一年，其智能手机总销量在 Q1 首次下降，随后随着诺基亚从 Symbian 过渡到 Windows Phone，连续几个季度亏损，现金储备减少和一系列破坏性的大规模裁员使这一任务变得更加困难。

最后，去年秋天，诺基亚决定减少损失，退出手机制造行业——以 37.9 亿€的价格将该部门出售给微软(并同意以 16.5 亿€的价格获得专利许可)——将精力集中在其他业务上:即 NSN 网络部门；这里地图；它的专利&正在 R&D 一个叫做先进技术的部门的努力下进行。

诺基亚今天表示，其 NSN 业务现在与两年前开始重组时“根本不同”。该部门本季度根据国际财务报告准则调整后的利润为 2.43 亿€，看起来将成为诺基亚未来增长的核心引擎，仅次于手机。

其 HERE maps 部门公布的 IFRS 调整后利润为 1800 万€——这使得它现在在诺基亚的池塘里是一条非常小的鱼(因此有“长期转型增长机会”的说法)。诺基亚补充说，它计划在 2014 年增加在这里的投资水平。

负责诺基亚知识产权许可工作的先进技术公司公布了按国际财务报告准则调整后的利润 6500 万€。

对于 Windows Phone 平台来说，过去三年多的时间是一条漫长而血腥的道路，以建立其既定目标，即成为第三个生态系统，与谷歌的 Android 和苹果的 iOS 竞争。诺基亚的手机制造业务无疑是这一努力中最大的受害者(还有诺基亚本土的塞班平台——以前是全球最大的智能手机操作系统)。

然而，在微软准备接管诺基亚手机制造部门之际，假日季度 Lumia 销量的下降——即便是小幅下降——无疑是对其手机雄心的一个打击。这不是新的微软首席执行官接手后想要宣扬的 Windows Phone 动力故事。
