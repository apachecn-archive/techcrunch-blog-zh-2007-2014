# 阿尔卡特朗讯收购 Web API 存储库项目 webtechcrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/06/29/alcatel-lucent-programmableweb/amp/>

太平洋时间 2010 年 6 月 29 日凌晨 3 点 58 分

[ProgrammableWeb](https://web.archive.org/web/20230204230957/http://www.programmableweb.com/) ，一个著名的 Web APIs 仓库，已经被[阿尔卡特朗讯](https://web.archive.org/web/20230204230957/http://www.alcatel-lucent.com/)意外收购。收购的条款仍未披露，但根据[新闻稿](https://web.archive.org/web/20230204230957/http://www.prnewswire.com/news-releases/alcatel-lucent-acquires-leading-web-20-api-repository-programmableweb-97374309.html)，ProgrammableWeb 将继续作为一个独立的实体运营，该网站的创始人 [John Musser](https://web.archive.org/web/20230204230957/http://www.programmableweb.com/about) 将转移到阿尔卡特朗讯。

在 2005 年发布的[库](https://web.archive.org/web/20230204230957/http://www.programmableweb.com/apis/directory)，现在包含大约 2042 个 API(当[在 2008 年 11 月达到 1000 个](https://web.archive.org/web/20230204230957/https://techcrunch.com/2008/11/03/let-a-thousand-web-apis-blossom/)时，我们已经报道过了)，将由它的新主人维护。

阿尔卡特朗讯表示，它收购 ProgrammableWeb 是为了帮助培育一个由服务提供商、企业和开发商组成的生态系统，以推动当前和通过 LTE(长期演进)在“宽带移动新世界”中开发应用。该公司补充说，它将提供支持和资源，以促进 API、mashup 和开发者生态系统的发展，并促进服务提供商参与其中。

此次收购还让阿尔卡特朗讯获得了通用 API 使用模式，允许它为自己的[开放 API 服务](https://web.archive.org/web/20230204230957/http://www.alcatel-lucent.com/wps/portal/!ut/p/kcxml/04_Sj9SPykssy0xPLMnMz0vM0Y_QjzKLd4w3MfQFSYGYRq6m-pEoYgbxjgiRIH1vfV-P_NxU_QD9gtzQiHJHR0UAAD_zXg!!/delta/base64xml/L0lJayEvUUd3QndJQSEvNElVRkNBISEvNl9BXzNBRC9lbl93dw!!?LMSG_CABINET=Solution_Product_Catalog&LMSG_CONTENT_FILE=Products/Product_Detail_001089.xml)构建 API 包。ProgrammableWeb 的 API 监控服务、API 试用服务和自动化 API 更新将集成到阿尔卡特朗讯的[开发者平台](https://web.archive.org/web/20230204230957/http://www2.alcatel-lucent.com/application_enablement/dpp.php)中。
