# 我要退出电子邮件 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/07/06/i-wouldnt-say-ive-been-missing-it/>

它开始是一个想法。嗯，[一条推文](https://web.archive.org/web/20230204193018/https://twitter.com/#!/parislemon/status/88818046021537793)，真的。如果我让[停止](https://web.archive.org/web/20230204193018/https://twitter.com/#!/parislemon/status/88818507457900544)回复邮件会怎么样？

当然，起初是在几杯酒后半开玩笑地说的。这是我在引导我内心的[彼得·吉本斯](https://web.archive.org/web/20230204193018/http://www.imdb.com/character/ch0001889/)——“我只是不想去了”——当我坐在华盛顿特区这里，在一整天没有收到邮件之后，完全被电子邮件淹没了。但是推文和想法背后的动机是非常真实的。邮件是绝对的[魔鬼](https://web.archive.org/web/20230204193018/https://techcrunch.com/2011/06/10/tumblrs-new-messaging-system-another-way-to-avoid-my-evil-email-inbox/)。不堕落的唯一方法就是…逃跑。所以这正是我要做的。

在这个月剩下的时间里，我不会回复任何邮件。没有。

是的，这听起来有点像一个娇纵的问题，也许有点鸡巴的举动——我应该很幸运能收到这么多电子邮件，对吗？但现实依然是:电子邮件绝对是我生活中的噩梦。我早上害怕，睡觉前更害怕。它一直在我脑海深处，挥之不去。

在我最初的推文之后，GigaOm [的 Colleen Taylor 给我发了一个链接，链接到《艾米丽》杂志](https://web.archive.org/web/20230204193018/https://twitter.com/loyalelectron/status/88820133136580608)上的这篇[旧文章，这篇文章很好地总结了这一点:](https://web.archive.org/web/20230204193018/http://www.emilymagazine.com/?p=371)

> 当人们在聚会上问我是做什么的时候，我想我会开始说我是一个“发邮件的人”

既悲哀又真实。

我并不孤单。我今晚发出的关于这个话题的推文得到了几乎一致的同意(我将在下面粘贴其中的一些)。好像我认识的*大部分人都希望*退出 email。他们只是害怕。

事实是我也害怕。如果我停止回应会发生什么？老实说，我不太确定。我认为，如果我考虑到所有潜在的后果，我就不会这么做。所以我不会为此担心。我正要去做。

如果没有别的，这将是一个有趣的实验。我们都在谈论我们对电子邮件的憎恶——多年来一直如此——但很少有人真的放弃了。有些人半心半意地尝试。其他人确信解决方案总是指日可待。但我不想再等待和找借口了。我正要去做。

我必须作弊吗？我当然不希望。那会非常令人失望。但我真的不认为我必须这么做。

我的计划是仍然不时地查看我的收件箱，以确保没有什么紧急情况。我会根据需要转发信息(除了电子邮件地址之外，不需要输入任何东西)。但是对于其他事情，我将简单地设置一个自动回复器，类似于“不再回复邮件，如果你需要我，你会想出办法的。”(今年早些时候，Shervin Pishevar 想测试一个类似的想法，但我退缩了。)

这是所有这一切的关键。并不是我真的想甩了人家。是邮件吹了。一定有更好的方法。而且我觉得是有的！如果人们真的需要联系我，他们会知道怎么做的。有很多选择。所有这些都比当前状态下的电子邮件要好([加油 Gmail Lite，加油！](https://web.archive.org/web/20230204193018/https://techcrunch.com/2010/11/27/gmail-lite/))。

这并不意味着我会回复所有这些备选的交流方式——我怀疑它们也会很快建立起来。但至少这将是一个很好的进入壁垒，有助于减轻我的收件箱超载。其他一些消息平台的伟大之处在于，它们中的许多都遵循“流”的理念。那就是:你发消息，也许会有回应，也许不会。电子邮件的一个大问题是，人们每次都希望得到回复。有了推特，人们就不会了。

我不知道你怎么样，但我很兴奋。我完全不知道接下来会发生什么。我会回复我收件箱里的一些邮件，但是 7 月份剩下的时间里，我会一直关着灯。世界会终结吗？TechCrunch 会内爆吗？我想不会。我认为这个实验的最终结果不会那么令人震惊:发送的电子邮件会更少。
