# NYU 如何处理盗版:不要被抓否则...

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/27/how-nyu-handles-piracy-dont-get-caught-or-else/>

[![nyupiracy.jpg](img/b0f2e4e6269572721ad949d670db0869.png)](https://web.archive.org/web/20220926164543/https://beta.techcrunch.com/wp-content/uploads/2007/08/nyupiracy.jpg "nyupiracy.jpg") 
盗版害死小猫，你知道

别问，别说。哦，但是如果 RIAA 抓住了你，你就完了。这就是被哈利·乔尔·奥斯门特、斯派克·李斯和约翰·比格斯称为 NYU 的纽约大学对待校园盗版的方式。(今年秋天我就大四了。)今天，我登录了学校的主门户网站，想知道我的 730 亿美元贷款是否已经结清——他们已经结清了！债务规则！我发现了一个有趣的小纸条，来自这里的一个高层。“关于非法下载的注意事项”，这个标题只是尖叫虚假的学术，基本上告诉新生和返校学生:看，我们知道你下载音乐和电影，只是不要被抓住。因为，嘿，如果你这么做了，你就只能靠自己了。

我相信你们中的一些人是第一次去学校，所以我想我会分享我的如何处理盗版的祸害。

NYU 的立场要点如下:

> 该大学在这个问题上的立场很简单:未经许可下载版权材料是非法的，你不应该这样做。未经版权所有者的许可，你也不应该使用你的计算机分发受版权保护的材料。如果你现在正在或者曾经非法下载或分发，你应该停止…
> 
> 如果 RIAA 认为你参与了非法下载或分发受版权保护的材料，并向 NYU 提交了一份有效的传票，以寻求你的身份，大学将遵守传票，并向 RIAA 的律师提供你的姓名和联系信息。
> 
> 在回答这些问题时，这所大学诉诸于亚伯拉罕·林肯曾称之为你们天性中的“善良天使”以及你们对学术文化的承诺。
> 
> 但是，正如你在学术和大学生活中遵守某些行为标准一样，当涉及到互联网上其他人的知识产权时，你也应该遵守高标准。

我的回应？哈哈，NYU！在我当学生的三年里，其中两年我有 NYU 提供的互联网连接，你知道，我肯定到处下载了几张专辑，也许是一部电影或一两个游戏。顺便说一下，这些都是很低的估计。这并不像是我下课后回到家，拿起高清的 24 张 T1 或 ATB 的新专辑或任何东西。或者，也许，让我们说，整季的电视节目我没有兴趣看*，只是因为我可以*。(可以想象，NYU 的关系非常好。)

对于我所有的下载，我从来没有收到过学校，RIAA/美国电影协会，任何人的警告。数兆字节的数据，宝贵的内容，日复一日地涌入我的 PowerBook 和 iMac。太棒了。

有趣的是，去年 IT 部门斥责了我的室友，因为他下载了电影*smoking’Aces*。我说他下载了那个*垃圾*是罪有应得，但是我发现讽刺的是他被抓了——我的意思是说，他是一个运动员类型的临时下载者——在他仅有的一次数字尝试中。他使用直接连接，这可以解释为什么警察…哦，我是说美国电影协会，发现了他。

总之，NYU 不想知道你到底用你的网络连接做了什么。如果一个法外卡特尔偶然发现你下载了它的垃圾，那你就有麻烦了。我建议，如果你坚持下载，你可以使用私人 BitTorrent 跟踪器，它只增加了最薄的保护面纱，PeerGuardian 并打开 Azureus 的加密功能。换句话说，[照文斯做的做](https://web.archive.org/web/20220926164543/http://crunchgear.com/2007/05/14/help-key-the-essential-guide-to-piracy/)。

你们还在上学吗？这种“捂住耳朵，闭上眼睛”是对你疯狂下载的普遍反应吗？

注意安全，朋友们。不要让妖怪吓走你的爱好。此外，大学就是实验。
