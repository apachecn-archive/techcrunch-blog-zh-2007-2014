# 苹果更新了时间胶囊，2TB，售价 500 美元

[![time-capsule](img/6e6011198cd31bf34fe5a0ca0fa9c9d8.png "time-capsule")](https://web.archive.org/web/20221006120707/http://www.crunchgear.com/2009/07/30/apple-updates-the-time-capsule-2tb-for-500/)

嘿嘿，看那个。[苹果](https://web.archive.org/web/20221006120707/http://www.crunchgear.com/tag/apple/)最终[更新了时间胶囊](https://web.archive.org/web/20221006120707/http://store.apple.com/us/product/MB996/Time-Capsule-2TB?mco=NzUxMDMwNg)，将 500 美元型号的存储容量增加了一倍，达到 2TB，将 1TB 的型号降至 300 美元。其他一切都是一样的，但现在是苹果开始使用这个程序的时候了。如果你没有注意到，硬盘现在很便宜，500 美元只买 1tb 的硬盘有点荒谬——即使对苹果来说也是如此。
