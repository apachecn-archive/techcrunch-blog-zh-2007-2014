# KakaoTalk 正考虑在其拥有 1.3 亿用户的移动信息平台上增加新闻内容服务

韩国的 [KakaoTalk](https://web.archive.org/web/20230117202731/http://www.crunchbase.com/company/kakaotalk) 正在考虑推出一项新闻内容服务，以进一步充实其移动通讯应用的功能集，该功能集已经包括群组语音聊天、事件安排&投票、贴纸和动画表情符号，以及用户购买礼物或折扣的能力。

《T2 朝鲜先驱报》(Korea Herald)早些时候报道了新闻内容发布计划，Kakao 向 TechCrunch 证实，它正在开发一项新的内容服务——但一位女发言人形容该计划“处于非常初期的阶段”，并补充说，“可能会也可能不会包含新闻”。

“目前我们没有什么可以宣布或确认的，”她补充道。

她说，KakaoTalk 是其本土市场中最强的，现在拥有 1.3 亿注册用户。它没有公布每月活跃用户的数字。

不管 Kakao 最终推出(或不推出)什么样的内容服务，它甚至考虑涉足媒体领域的事实，更证明了移动通讯应用凭借自身的能力成为平台——这要归功于它们能够利用大量高度参与的用户。对于最大的移动通讯应用程序，这些池包含数亿每月活跃用户。

例如，老牌移动通讯应用 WhatsApp，与该领域的一些新竞争对手相比，其功能相对较少，但仍设法积累了超过 3 . 5 亿的月活跃用户群。与此同时，据最新统计，中国的微信拥有超过 2.7 亿的 MAUs 而日本的 Line 报告 [3 亿注册用户](https://web.archive.org/web/20230117202731/https://techcrunch.com/2013/11/25/line-300m/)。

后两个平台已经扩展到多个附加服务，以扩大其核心通信产品，如游戏(应用内游戏相关的购买是 Line 最大的收入来源)，以及微信的移动支付服务。

KakoTalk 还有一个社交游戏平台——考虑到游戏在其本土市场韩国的受欢迎程度，这并不奇怪。如果它进入媒体领域，这将是消息应用类别的进一步扩展。随着移动通讯应用继续从纯粹的通信渠道向社交网络、丰富的娱乐、信息和实用平台进军，界限将进一步模糊。
