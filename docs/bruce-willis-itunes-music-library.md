# 布鲁斯·威利斯没有就 iTunes 音乐所有权起诉苹果

今天早些时候，有传言称布鲁斯·威利斯正在考虑起诉苹果，以澄清谁拥有从 iTunes 下载的内容。英国《每日邮报》报道——和所有涉及英国小报的事情一样，你应该对此持保留态度——威利斯“据说正在考虑对科技巨头苹果公司采取法律行动，因为他想把他的数字音乐收藏留给他的女儿们。”在考虑他的死亡时，威利斯显然注意到了大多数 iTunes 用户也很容易忽视的一点:尽管苹果现在向你提供无 DRM 的文件，但你所拥有的只是一个在你控制的最多五台设备上播放你的音乐的许可证，你不能合法地将它们传递给其他人。

*我们更新了这个故事，因为威利斯的妻子否认了这个故事是真的。*

据《每日邮报》报道，威利斯的法律团队显然在考虑“建立家族信托，作为某人下载音乐的‘持有者’”。然而，根据威利斯的妻子艾玛·海明-威利斯的说法，这个“[不是一个真实的故事](https://web.archive.org/web/20230123125819/https://twitter.com/EmmaHeming/status/242631258310594562)”

## 你死后，你的数字音乐和书籍会怎么样？

尽管如此，在某人死后，他的数字资产(包括音乐、电子书和社交网络档案)会发生什么，这是一个我们还没有完全弄清楚的问题，值得思考，即使在这个缓慢的新闻日故事之外。

例如，像[dead Account](https://web.archive.org/web/20230123125819/http://deceasedaccount.com/)这样的网站列出了许多主要的网络服务，这些服务目前没有关于如何处理已故用户账户的政策。包括 Twitter 在内的大多数服务都有相关政策，允许家庭成员关闭已故用户的账户，但不允许他们访问用户的数据。脸书可能比其他在线服务更需要解决这个问题，现在它提供了一个功能[将一个死者的个人资料转化为在线纪念。](https://web.archive.org/web/20230123125819/http://www.facebook.com/blog/blog.php?post=163091042130)

至于人们从 iTunes 下载的音乐，苹果公司处理音乐版权的大部分规则并不是由苹果公司决定的，而是由音乐产业决定的。尽管如此，它总是有助于保持最新的 iTunes 条款和条件，因为谁知道会发生什么，如果你不…
