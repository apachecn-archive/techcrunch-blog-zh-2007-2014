# 为什么达尔文打败了丹尼·卡尔顿

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/08/23/why-darwin-beats-danny-carlton/>

在 jacklewis.net/weblog.，丹尼·卡尔顿用笔名“杰克·路易斯”写了一个鲜为人知的个人博客，但是如果你使用火狐浏览器，不要试图访问它，因为他禁止这款流行浏览器的用户访问他的网站。火狐用户现在被重定向到[这里](https://web.archive.org/web/20210925005100/http://whyfirefoxisblocked.com/index1.php)。

为什么？因为他反对一些火狐阅读器使用广告屏蔽扩展功能来屏蔽网站上的广告。为了解决这个问题，他把婴儿和洗澡水一起扔了出去，并把他网站上大约 13%的互联网流量踢出去。

虽然卡尔顿肯定很享受他的十五分钟成名，但在我看来，这不是一个建立博客的好策略。用户是纯金。即使是那些屏蔽广告的。他们有时会写评论，这是免费内容。他们从自己的博客链接到你。他们把你的网站告诉朋友。所有这些都会带来更多的读者，最终带来更多的收入。如果用户想跳过广告，愿意安装广告拦截软件，那就这样吧。我仍然爱他们。我很乐意免费给他们我的内容。

卡尔顿显然不同意。尽管我想知道他为什么继续在 jacklewis.net/weblog/atom.xml 提供一个完整的内容提要，没有广告。这些用户也在“窃取”他的内容。他们怎么了？也许他现在会把注意力转向 RSS 的弊端。

如果没有卡尔顿充满激情的社论，互联网肯定会变得不那么丰富多彩。细读他的博客帖子(通过 Safari ),我发现他认为巴拉克·奥巴马是一个共产主义者,“四年级学生可能会被骗，被告知进化论是事实。”问题是，达尔文是对的。只有适者生存。卡尔顿刚刚把他的博客列为濒危物种。
