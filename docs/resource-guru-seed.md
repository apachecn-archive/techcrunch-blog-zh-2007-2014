# Index 在资源专家基于云的团队日程安排应用 TechCrunch 的 87 万美元种子轮中领先

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/19/resource-guru-seed/>

Index Ventures 已经向[资源大师](https://web.archive.org/web/20221209030242/http://www.crunchbase.com/company/resource-guru)提供了 87 万美元的种子资金，这是一家总部位于伦敦的初创公司，希望打破公司使用电子表格来管理员工、设备和其他资源的做法，并用其基于云的团队调度软件取而代之。

Index Ventures 的合伙人罗宾·克莱恩(Robin Klein)在一份声明中表示:“大多数经理仍然依赖复杂的电子表格或笨重的软件来管理他们的团队和资源，这两者都不会使资源分配变得特别容易和有效。”“我们对 Resource Guru 如何利用其简单易用的在线工具解决这一问题印象深刻，该工具节省了大量时间并提高了生产率。”

在此轮融资之前，成立于 2011 年 5 月的 Resource Guru 已经从朋友、家人和天使投资者那里筹集了资金。种子轮的其他投资者包括现有的和新的投资者，但这些尚未披露。迄今为止，其资金总额为 114 万美元。

Resource Guru 表示，将利用新资金“加速招聘、产品开发和客户获取”。这家初创公司在 2012 年 5 月推出了日程安排应用，但目前没有公布客户数量(尽管它提到了盛世长城、沃达丰、英特尔、罗氏、ASOS 和国家地理频道等未知客户)。

它还表示，其平台的用户已经为超过 31，000 个项目创造了超过 296，000 个预订，所以你可以随心所欲。

值得注意的一点是，它并没有走免费增值的路线，而是向其软件的所有用户收取订阅费。“迷你”服务(最多 10 人)起价为每月 19 美元，覆盖 80 多名用户的“高级+服务”起价为 99 美元。

当你试图说服公司放弃电子表格，转而使用专用的日程安排工具时，你最大的优势无疑是用户体验的相对流畅。因此，这就是资源专家所说的集中精力的地方。

“当我们设计 Resource Guru 时，除了人们使用的电子表格之外，没有任何参考点。我们从零开始，重新思考了整个过程，”联合创始人安德鲁·罗格夫说。

该软件包括一个预订部分，用于可视化资源在给定时间点的使用情况，一个“冲突管理引擎”，以防止过度预订，以及一个等待列表功能，以便需求资源不会被同一只早起的鸟从帐户中霸占。可以根据技能组合等因素筛选人员和资源的可用性，以便用户可以在要求的时间范围内为给定的工作找到合适的人。

该软件还可以生成“利用率”——这样你就可以识别使用不足和过度使用的资源(或员工)，以及跟踪在哪些项目/客户上花费了多少时间，以及谁在做什么。换句话说:密切关注你的员工。

这是一个与时间管理软件产品有些重叠的功能集——如[Harvest](https://web.archive.org/web/20221209030242/http://www.crunchbase.com/company/harvest)——尽管罗格夫拒绝进行比较，称 Harvest 专注于“时间表和过去”，而 Resource Guru 的目标是“预测未来”，并补充说“直到现在，这一直是一个严重服务不足的领域”。

就直接竞争对手而言，除了电子表格，他还提到了像[synergistic](https://web.archive.org/web/20221209030242/http://www.crunchbase.com/company/synergist-express)和 [Deltek](https://web.archive.org/web/20221209030242/http://www.crunchbase.com/company/deltek) 的资源规划服务，称其“将调度作为整体解决方案的一部分”。但是他再次拒绝任何直接的比较。

“我们并不真正认为它们是直接竞争对手，因为它们价格昂贵、笨重，而且充斥着不必要的功能，”他补充道。
