# 彼得·泰尔的突破实验室资助了“纳米管道”和一个类似 Siri 的自然语言处理创业公司

[Breakout Labs](https://web.archive.org/web/20230307015709/http://www.breakoutlabs.org/home.html) ，一个彼得·泰尔支持的努力，支持做尖端研究的初创公司，这些研究可能仍然过于冒险，无法获得传统风险投资的支持，[刚刚向另外两家公司投资](https://web.archive.org/web/20230307015709/https://www.breakoutlabs.org/recipients.html)。

他们正在资助一家名为 [SkyPhrase](https://web.archive.org/web/20230307015709/http://skyphrase.com/) 的类似 Siri 的自然语言处理初创公司和另一家名为 Stealth Biosciences 的生物技术公司，后者正在创造“纳米管”,可以很容易地将材料插入单细胞而不破坏它们。

去年成立的 breakout Labs[向初创公司](https://web.archive.org/web/20230307015709/https://www.breakoutlabs.org/recipients.html)提供高达 35 万美元的资助。突破背后的动机是弥合初创公司从事风险更大、资本更密集的研究与他们发现商业上可行或可以制成产品的东西之间的差距。Breakout 的本意是在一家初创公司继续寻找更多传统风险投资支持之前，成为第一个外部投资者。

泰尔基金会负责投资的高级副总裁呼唤·菲什伯恩表示，去年有几家公司已经筹集了 100 万至 150 万美元的天使融资，还有两家公司获得了美国国立卫生研究院(National Institutes of Health)等知名机构的额外资助。

她说，Breakout 资助了 Stealth Biosciences，因为它的技术可以帮助研究人员和医生避免使用病毒或病毒载体将材料插入细胞并治疗患者。

“这实际上是一种利用纳米技术将材料直接插入细胞内部的可扩展方法，”菲什伯恩说。“他们开发了纳米材料，使他们能够穿透细胞并获得所需的细胞，避免使用病毒或病毒载体，这种技术还有其他问题。”

Stealth 开发了被称为“纳米管”或“隐形探针”的纳米级设备，可以在单细胞水平上测量和控制生物过程。隐形技术背后的两位斯坦福大学教授，尼克·梅洛什和克雷格·加纳说，这可以用于个性化医疗、肿瘤学和神经科学。

第二个公司突破叫做[天片语](https://web.archive.org/web/20230307015709/http://skyphrase.com/)。有两个学派致力于自然语言处理；一种依赖于强力统计方法，另一种依赖于逻辑推理和语言规则。 [SkyPhrase](https://web.archive.org/web/20230307015709/http://skyphrase.com/) 称他们有一项融合了这两种技术优势的技术。

他们[去年曾短暂地为 iPhone](https://web.archive.org/web/20230307015709/http://allthingsd.com/20121108/artificial-intelligence-professor-makes-a-search-app-to-outsmart-siri/) 开发了一个 Gmail 搜索应用，可以搜索内容丰富的短语，比如“简在假期发给我的含有图片的电子邮件”但是店里已经没有了。
