# 据报道，苹果公司正在测试电视机设计，这应该不会让任何人感到惊讶

《华尔街日报》今早报道称，苹果正在测试苹果电视机(相对于机顶盒)的设计，尽管测试仍处于“早期阶段”根据报道，苹果正在专门研究高分辨率电视，尝试一些不同的设计，与包括鸿海(富士康)和夏普在内的多家供应商合作开发新的电视设计。据报道，电视测试对苹果来说并不新鲜，我们之前已经听过很多了，此外，苹果首席执行官蒂姆·库克在[最近的采访中对苹果的电视计划也不再闪烁其词。](https://web.archive.org/web/20221207070225/http://www.businessweek.com/articles/2012-12-06/tim-cooks-freshman-year-the-apple-ceo-speaks#p2)

底线是:苹果正在开发一款电视。Piper Jaffray 分析师吉恩·蒙斯特多年来一直在谈论这个问题，但每年他们都认为他的可信度有所下降。但他没有错。《华尔街日报》的报道只是强化了这样一种观点，即苹果一直在修补电视，已经超过了他们已经推出的第一代、第二代和第三代苹果电视机顶盒。史蒂夫·乔布斯在他的沃尔特·伊萨克森传记中讨论了苹果对空间的争夺，其他消息来源，包括[《纽约时报》](https://web.archive.org/web/20221207070225/http://bits.blogs.nytimes.com/2011/10/27/whats-really-next-for-apple-in-television/?smid=tw-nytimesbits&seid=auto)，几乎把苹果电视称为未来产品发布的一把锁。

那么有什么变化吗？或者，这仍然只是一个 R&D 的项目，还没有接近见天日？嗯，根据这份新报告，有一些迹象表明，苹果可能会在这个过程中走得更远。首先，即使与外部供应商合作也表明了某种程度的成熟，因为苹果通常在与制造或组件合作伙伴进行沟通之前就已经在内部工作了。其次，在董事长郭台铭(Terry Gou)对一家前夏普(Sharp)所有的液晶显示器工厂进行战略投资的支持下，富士康已采取措施扩大高清电视生产。目前 Apple TV 的销量也在上升。

但是还有很长的路要走，因为苹果还没有为电视机达成内容协议，至少没有与主要的有线电视和媒体提供商达成协议。所以最后，虽然苹果公司积极修补电视机并不令人惊讶，但没有人应该对这样一种设备的到来抱以屏息，至少现在没有。
