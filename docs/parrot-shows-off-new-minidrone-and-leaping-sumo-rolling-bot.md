# Parrot 展示了新的小型无人机和跳跃的相扑滚动机器人

法国蓝牙公司转型无人机制造商 Parrot 为 CES 2014 推出了一些新硬件，包括两个全新的智能手机控制的机器人。一个是滚动、跳跃的相扑，另一个是 MiniDrone，这是其流行的 ar 无人机的超小型版本，可以飞行、滚动并拥抱天花板和墙壁。

迷你无人机非常小，只有完整版本的十分之一大小，这使得它非常便携。如果你真的想的话，你可以把它放在口袋里，不会有什么问题。为了获得这种尺寸优势，它放弃了摄像头，但增加了可移动的大轮子，使它可以在地面上滚动——或者玩杂技，也可以沿着墙壁或天花板跑。

相扑是一种双轮地面专用装置，它有一个摄像头，就像大型 AR 无人机一样，并且有一只内置的脚，允许它几乎立即停止，并在空中跳跃高达 80 厘米。它也更加坚固，符合其杂乱无章的生活方式，让我想起了该公司刚刚推出的 Sphero 2B。

迷你无人机或 Sumo 都没有确定的发货日期，但据首席执行官 Henri Seydoux 称，它们“很快”就会上市，价格尚未最终确定。我问 Seydoux 这两个设计是怎么来的，它们是否是用户请求和反馈的结果，他说事实上 Parrot 想出了它们，因为“我们喜欢娱乐。”
