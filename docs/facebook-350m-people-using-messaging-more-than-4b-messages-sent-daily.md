# 脸书:3.5 亿人使用短信；每天发送超过 40 亿条消息

今天，在脸书的[消息发布会](https://web.archive.org/web/20221105000859/https://beta.techcrunch.com/2010/11/15/facebook-email/)上，马克·扎克伯格透露了一些关于社交网络消息平台的惊人数据。扎克伯格说，3.5 亿会员(T4 超过 5 亿会员中的 3.5 亿)正在使用脸书的信息系统。因此，超过一半的脸书用户在使用其消息平台。

他补充说，每天有超过 40 亿条消息被发送，而且这个数字还在以惊人的速度增长。该统计包括使用聊天的消息和即时消息。扎克伯格表示，该平台上的绝大多数消息都是两个人之间一对一的消息。

有 3.5 亿用户使用脸书现有的消息系统，每天有 40 亿条消息通过网络发送，所以脸书想要拥有消息和社交也就不足为奇了。
