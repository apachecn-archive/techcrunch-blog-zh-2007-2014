# Line Messaging 应用程序与电信运营商 Telefónica 达成协议，在 Firefox OS 的主要市场上独占 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/02/04/line-telefonica-ffos/>

去年一直在日本本土市场之外拓展的 Line 消息应用已经签署了一项协议，旨在加强其在拉丁美洲市场的存在。今天，它宣布与电信巨头西班牙电信合作，致力于通过低成本 Firefox OS (FFOS)平台上的排他性来建立牵引力。

这一合作伙伴关系使西班牙电信的用户可以在被称为“主要的西班牙电信 Firefox OS 市场”独家访问 Line 应用程序，包括委内瑞拉、秘鲁、哥伦比亚、乌拉圭、巴西和墨西哥。该交易还包括西班牙。两人表示，将“很快”宣布其他市场。

如果 FFOS 起飞，这可能是 Line 的一大胜利。西班牙电信拥有 3.2 亿全球用户使用其网络基础设施，包括在拉丁美洲地区的大量存在，Line 认为其可爱的卡通信息风格，以一系列可爱的贴纸人物和游戏为特色，可以找到一个西班牙语家庭。

去年这个时候，该公司试图进入美国市场的尝试显然更加困难，因为 Line 似乎正在围绕其他更容易实现增长的地区重新配置其战略，如拉美和 T4。

谈到这种合作关系中 Mozilla 风格的部分，这个平台在去年的世界移动通信大会上大张旗鼓地推出，并获得了大量运营商的支持。自那以来，它一直通过运营商的推出渗透到各个市场，重点是发展中地区，这些地区的网络运营商认为在低端智能手机领域与 Android 存在竞争差距。

自 6 月份以来，西班牙电信已经在 7 个国家推出了 FFOS，而 Telenor、德国电信(Deutsche Telekom)和 TIM 等其他运营商也开始推出运行基于 HTML5 平台的手机。

然而，目前还不清楚 FFOS 到底有多少用户，因为它的运营商支持者没有透露销售数据。增长似乎不太可能迅猛，否则运营商们会对此大喊大叫。尽管如此，FFOS 的支持者显然仍在继续推动该平台，并通过类似 Line 和西班牙电信今天的信息合作伙伴关系，寻求在该平台上实现差异化的方法。

西班牙电信大概是希望通过利用 Line 在该地区的受欢迎程度，以及手机短信的普遍受欢迎程度，来提高人们对其 FFOS 设备的兴趣。因此，虽然 Line 应用程序现在可以通过 Firefox OS Marketplace 在全球范围内使用，但其 FFOS 应用程序将只面向上述市场(主要是拉美市场)的西班牙电信用户。

这可能看起来很奇怪——也就是说，对于一个显然是为了支持开放网络而构建的平台来说，允许控制应用程序的分发方式——但这正是那种以运营商为中心的灵活性，使 Mozilla 的平台能够吸引如此重要的移动网络运营商的支持。FFOS 还给他们一种谷歌 Android 所缺乏的控制元素。

预装 Line 应用程序的 FFOS 电信设备预计将在 Q1 早期推出。该运营商还将在向现有 Firefox 操作系统用户发布的新操作系统版本中包含 Line 的应用程序，因此，如果西班牙电信也能在 FFOS 周围建立牵引力，Line 在这一合作关系中有很大的潜力。

为此，两人表示，合作伙伴关系将得到双方所称的“重大营销活动”的支持(尽管他们没有透露确切的支出)。Line 显然还将通过在“特定市场”的官方 Line 账户向用户推出特别优惠，鼓励用户加入西班牙电信的网络。

Telefónica S.A .集团设备部门总监 Francisco Montalvo 在一份声明中评论了这一合作关系，重申了该电信公司对 FFOS 的承诺，并表示该生态系统拥有“快速增长的客户群”，他补充说:“LINE 的消息应用程序使我们能够加强我们已经为客户提供的服务，并将在 2014 年开辟更多的合作机会。”

Line Euro-Americas Corp .首席执行官 Jeanie Han 在一份声明中表示:“通过与西班牙电信合作，我们强调了我们对扩大欧洲和拉丁美洲客户群的承诺。
