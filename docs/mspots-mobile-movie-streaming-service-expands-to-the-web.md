# mSpot 的移动电影流媒体服务扩展到网络

移动娱乐初创公司 [mSpot](https://web.archive.org/web/20221006062945/http://www.mspot.com/) 正在扩展其移动电影网站，该网站将允许用户在他们的手机和网络上播放完整长度的电影，这样你就可以在你的电脑上享受 mSpot 的在线流媒体电影服务。

mSpot 已经与派拉蒙(Paramount)、环球(Universal)、影像娱乐(Image Entertainment)和 Screen Media Ventures 达成协议，将完整长度的电影租赁内容传输到用户的个人电脑和手机上，允许你在观看电影时在这两种设备之间切换。PC 流媒体功能建立在去年推出的移动电影服务 mSpot [之上。该服务包括 1000 个可以传输到电脑和移动设备的标题。](https://web.archive.org/web/20221006062945/https://beta.techcrunch.com/2009/09/29/mspot-launches-web-based-mobile-movie-streaming-service/)

这项服务适用于 50 多种不同的手机设备，包括 iPhone、黑莓、Palm、Android、Windows Mobile 等。用户可以以每部 4.99 美元的价格观看单部电影，或者加入每月电影俱乐部:每月 9.99 美元，最多 4 部电影；12.99 美元，6 部电影；15.99 美元，8 部电影。

当然，mSpot 的主要竞争对手是苹果公司，它允许 Mac、iPhone 和 iPod touch 用户通过 AppleTV 下载电影和节目并同步到他们所有的设备上。但是 mSpot 和 mSpot 将面临来自网飞的其他系列竞争，当该公司最终进入移动领域时。
