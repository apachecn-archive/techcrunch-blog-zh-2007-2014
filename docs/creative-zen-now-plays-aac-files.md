# 创意禅现在播放 AAC 文件| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/29/creative-zen-now-plays-aac-files/>

[![newzen16gb.jpg](img/e486436e08539d0f9b1e87617edb9a2a.png)](https://web.archive.org/web/20151002132717/http://old.crunchgear.com/wp-content/uploads/newzen16gb.jpg "newzen16gb.jpg")

创意部不是每天都发布新的 Zens 吗？看起来的确如此。(尽管那可能是我现在服用的非处方药鸡尾酒——生病可不好玩。)这款特殊的 Zen 有 4GB、8GB 和 16GB 三种尺寸，与 Creative 的其他产品不同，它与[非 DRM] AAC 文件兼容。是的，那包括 [iTunes 加](https://web.archive.org/web/20151002132717/http://crunchgear.com/2007/05/31/the-audiofile-is-apples-itunes-plus-a-minus/)文件，比如[我几周前买的这个](https://web.archive.org/web/20151002132717/http://phobos.apple.com/WebObjects/MZStore.woa/wa/viewAlbum?playlistId=205149823&s=143441&i=205149873)。

Zen 的大小和信用卡差不多，深度适中，可以播放 iPod 梦寐以求的文件，比如 WMV 和 Divx。就功能而言，它比 iPod nano 更好，如果不是完全更好的话。至少目前是这样。

新款 Zen 的屏幕也让 iPod 的价格不相上下:2.5 英寸，320×240 像素分辨率，能够显示 1670 万色。(iPod 只能做 65000 色。)

现在，这看起来是一笔好交易。在做出任何有意义的结论之前，我们必须看看下周苹果公司有什么锦囊妙计。
