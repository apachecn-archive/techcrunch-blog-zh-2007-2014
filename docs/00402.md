# 创意 Travelsound i50:当你洗牌不够大

TravelSound iS0 是一个奇怪的 Shuffle 扬声器基座，它基本上使 Shuffle 成为一个像 iPod 一样大的东西。为什么？我不知道，但似乎给我们提供了一个不错的高杯 WTF。它现在只在亚洲销售——大约 69 美元——可能在很长很长一段时间内都不会登陆我们的市场。
