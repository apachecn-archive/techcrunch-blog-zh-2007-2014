# 苹果公司为一款内置显示器的 iPhone 相机遥控器申请了专利

苹果公司已经获得了一项无线 iPhone 相机遥控器的专利(通过 [AppleInsider](https://web.archive.org/web/20230129061808/http://appleinsider.com/articles/14/03/18/apple-patents-remote-iphone-camera-control-accessory-) ),它有一个内置的预览和查看照片的显示屏，静止和电影的快门按钮，以及远程切换相机模式和全媒体播放控制的方法。该设备将通过蓝牙或 Wi-Fi 连接到 iPhone 或其他移动 iOS 设备。

如果相机没有激活，它甚至可以用来启动相机应用程序，这是目前第三方 iPhone 蓝牙相机遥控器不一定具备的功能。此外，苹果在这方面的独特技术创新是，遥控器可以接收和显示关于相机状态的通知，告诉用户照片已成功拍摄，或已实现对焦锁定等。，这是当前迭代所不提供的。

从实际制造的角度来看，这个配件对苹果来说有点偏离，但该公司过去一直以制造专门用于摄影的配件而闻名，包括 iPod 和 iPad 相机连接套件适配器。相机和摄像技术长期以来一直是 iPhone 的主要卖点之一，尽管竞争对手的设备中功能噱头的数量有所增加，但 Cupertino 仍然因其移动产品拍摄的照片的图像质量而获得了很多好评。

然而，它仍然是一个非常专业的设备，本质上相当于一个 iPod nano，你可以随身携带来控制你的 iPhone，这似乎是许多多余的硬件，违背了苹果对简单性的承诺，所以不要对这一款屏住呼吸。尽管如此，结合苹果之前申请专利的用于 iPhone 的[相机镜头卡口支架，想象一下苹果将 iPhone 作为无反光镜可互换镜头相机的全面替代品是令人兴奋的，在这种情况下，像这样的配件可以帮助充实一个生态系统。](https://web.archive.org/web/20230129061808/http://www.macrumors.com/2014/03/13/apple-bayonet-lens-mount/)
