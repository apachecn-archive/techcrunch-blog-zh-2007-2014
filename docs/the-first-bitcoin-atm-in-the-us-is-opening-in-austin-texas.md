# 美国第一台比特币 ATM 机在德克萨斯州奥斯汀开张

美国！美国！我们现在正式几乎和加拿大一样好了！为什么？因为在得克萨斯州奥斯丁的[把手](https://web.archive.org/web/20230105035415/http://handlebaraustin.com/)处，机器人币的人们正式开启了美国第一台比特币自动取款机。

放置在奥斯汀酒馆显著位置的自动柜员机，将与您使用过的任何其他自动柜员机非常相似。你走上前去，投入一些钱，然后取出比特币。比特币所有者将能够通过该机器出售，你将收到一张收据，一笔 BTC 将被存入你的虚拟钱包。该公司已经在温哥华安装了自动取款机，并将它们运往亚洲，在那里它们曾一度被视为非法。

该公司将在 2 月 20 日星期四下午 2 点打开他们的自动取款机。这台自动取款机只是个开始。

“机械战警将无处不在！我们刚刚抵达得克萨斯州，已经运往西雅图，华盛顿和加拿大艾伯塔省，”创始人乔丹凯利说。

“奥斯汀的机器熊将会在市中心的车把手上开到一个非常酷的地方。我们有一堆更多的航运到加拿大惊人的地方。我们仍在与香港合作，但我们低估了国际硬件认证流程需要的时间。”

凯利说，注册系统允许你使用政府 ID 和生物特征数据销售比特币，“为 Robocoin 运营商提供全面的客户可见性、可跟踪性和报告工具，以支持全球合规性”。显然，这可能会让一些更偏执的 BTC 卖家望而却步，但这可能就是重点。

“现在，我们正在建立一个由充满激情和敬业的机器货币运营商组成的全球基础设施，他们都完全符合让比特币安全并为大众所用的使命。Robocoin 正在彻底消除比特币的进入壁垒，并帮助比特币进入现实世界，”他说。

这些机器采用“银行级”硬件和网络安全技术制造。电脑和相关的现金都在一个保险箱里，服务器不能从互联网上公开访问。

不过，在你的企业中安装一台机器警察需要一点时间。凯利说，业主需要一个官方的汇款许可证(MTL)或注册为货币服务企业(MSB)。但凯利认为，机器警察的受众，即 18-40 岁的早期使用者，是具有前瞻性思维的咖啡馆或酒店的理想客户。

凯利期待着增长，并不太担心政府。

“我们乐观地认为，政府会将我们视为盟友。听着，比特币会一直存在，就像互联网一样。我们认为，政府将寻求监管这个新兴经济体，并将确定和支持准备合作的企业。
