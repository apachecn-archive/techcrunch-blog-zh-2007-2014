日本宫崎的Infinity Ventures峰会:来自日本初创公司的12个演示TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2009/11/12/infinity-ventures-summit-in-miyazaki-japan-12-demos-from-japanese-startups/amp/

我目前在宫崎/日本，参加 Infinity Ventures峰会 (IVS)，亚洲最负盛名的web行业盛会之一。组织风险投资公司Infinity Venture Partners预留了一些时间给总共12家日本初创企业一个机会，在舞台上向评委团和300多名国际观众展示他们的服务。

一家名为莫吉莫吉电视台的语音识别和转录服务公司被选为IVS 2009年秋季发射台的“最佳创业公司”(刚刚结束)。以下是此次活动中演示的所有服务的简要介绍。(请注意，有些公司还没有推出英文主页，但有些确实也提供全球化服务。)

一名冠军和五名亚军:
莫吉莫吉电视台byCatalog
最佳节目由莫吉莫吉电视台获得，这似乎是上个月在私人alpha中推出的非常强大的视频语音识别和转录服务。该引擎仅支持日语，但英文和中文版本正在开发中。莫吉·莫吉从视频中提取音频(自制电影，YouTube剪辑等。)并自动将说出的单词显示为文本，然后用户可以对其进行编辑。该文本可以用来标记和子电影，也可以在其中搜索某些单词或表达式。还有一个名为 Shabetter 的iPhone应用程序，可以自动将你说的话转录到iPhone麦克风中，并发布到Twitter上。更多关于莫吉莫吉电视台的英文信息可以在这里找到。

这段视频(英语)展示了莫吉·莫吉电视台的运作方式:

V-Sido by 渡泽喜崎(亚军)
大多数用于娱乐、研究和其他领域的机器人都有一个共同的问题:它们只能以缓慢、机械和劈砍的方式移动。由机器人狂热者Wataru Yamazaki开发的V-Sido是一个基于软件的实时控制系统，用于帮助人形机器人更平稳地移动。

SpyseebyOhma(亚军)
Spysee 是今年TechCrunch 50上推出的英文版的人搜索引擎(spy see是一家演示坑公司)。中文版正在制作中。这项服务从网络上获取个人信息和媒体，并呈现在个人页面上(例如:巴拉克·欧巴马)。Spysee最初的日语版本于去年上线，此后经历了多次迭代。一个例子是Spysee现在将其服务货币化的方式:它允许用户向需要帮助来资助其个人目标的个人捐赠真金白银(欢呼Spysee)，作为回报，获得捐赠的一部分。

m-policebymilog(季军)
m-police 是一个全自动化的网站监控系统，能够检测网络上的亵渎、侮辱和“法律上有问题”的句子，并将相关的表达细分为16个不同的类别。该公司表示，日本博客圈大约5%的帖子可以被视为与移动警察相关，其中22%可以被称为潜在危险。一些日本社交网络雇佣了多达300名“监视官”来保持网站的清洁。移动警察可以将由人工监控的每个帖子的成本从6日元(7美分)降低到3日元。

aegis guardbyKLab【第四名】
AEGISGUARD 是一款杀毒软件，不仅可以免费下载而且完全有英文版本。该计划的主要目的是保护您的重要文件和文件夹免受病毒(其中超过500万存在今天)和恶意软件，只授予白名单程序访问它们。AEGISGUARD的开发者 KLab 说，这样，未知或新的病毒被有效地隔离了。该解决方案可以与传统的、基于黑名单的防病毒软件一起安装在同一台PC上。

Symphonic MotionbyAITIA(季军)
Symphonic Motion 是一个基于增强现实的娱乐引擎，它不是为移动设备而设计的，而是为大尺寸显示器(直径可以达到几米)而设计的。这项技术使用“物理”AR，这意味着你可以站在摄像机前，通过移动手臂来移动你在屏幕上看到的CG元素。制造商AITIA正在将该解决方案作为互动、有趣的数字标牌应用程序向企业、展览和活动组织者进行营销(演示电影)。

其他六家演示公司:
OpenSocial HostbyHeartRails
OpenSocial Host是一个用于开发、分发和管理OpenSocial应用程序的集成平台，例如用于 MySpace 、 hi5 等网站(其中早在今年9月就向外部开发者开放了其网站)。OpenSocial Host支持移动应用程序，提供付费和免费选项，但它仅支持日语。

位置放大器byKoozyt
位置放大器背后的关键思想是通过将“丰富的”、基于位置的内容推送到手机上来“放大”现实世界，例如基于位置的游戏或指南。在IVS，制造商Koozyt展示了如何追踪一个人的行动，即使是在建筑物内(全球定位系统经常失灵)。也可以使用定位放大器在博物馆的展览之间走动，让你的移动设备告诉你当前正在看什么(演示电影)。这项技术基于 PlaceEngine ，这是一项依靠Wi-Fi来估计你的位置的服务，即使没有GPS。

Zeke CMS社交游戏套件 by 无处不在的娱乐
无处不在的娱乐 CEO清水亮演示了 Zeke CMS社交游戏套件，这是一个(仅限日本)平台，用于开发部署在社交网络中的游戏，如【脸书】或日本最大的社交使用该系统开发的一个游戏名为“东京宝藏”，这是一个寻宝游戏，使用iPhone摄像头将AR元素与现实世界结合起来(“当你被标记时，游戏就结束了！”).


Droidget AR被GClue
Droidget AR描述为全球首个Android上增强现实服务的widget框架。这些小部件可以在物理位置被其他人“捡起”并立即使用，而无需用户访问网络。例子:当你在一家餐馆里，打开你的Android手机上的摄像头，对准门，点击你看到的AR标签，立即进入菜单(演示电影)。

Jitsu kuku Kan Toushi KeitaibyKDDIau one Labs
KDDI日本第二大电信公司展示了一款增强现实应用，于今年6月针对数量有限的日本手机推出。他们的Jitsu kuku Kan Toushi Keitai应用程序(大致翻译为:允许真实空间透明的手机)的工作方式与其他许多移动AR应用程序一样。它使用GPS，手机的摄像头，6轴传感器和屏幕来定位你在哪里，并通过标签指出你想去的地方。但如果你将手机指向一堵墙或一个人，你也可以“看穿”他们，这意味着该应用程序将显示位于他们身后的标签。

Odette解决方案byORSO
Odette解决方案让你创建移动Flash站点(几乎100%的日本手机都安装了Flash Lite)。仅限日语、基于云的ASP服务提供了各种模板，您可以通过各种方式编辑其元素，例如通过拖放。但是你也可以通过组合不同的元素来创建移动网页。Odette解决方案将于明年春天推出。
