# Isostick 是对光盘霸权的嘲弄

过去，要安装软件和操作系统，你必须将一种叫做“光学介质”的东西放入“DVD 驱动器”中。多亏了一个奇怪的叫做 Isostick 的小 u 盘，那些日子早就过去了。

尽管该项目并不正式存在——它几乎是由 Kickstarter 资助的 IsoStick 承诺允许你从任何计算机加载任何 ISO 磁盘映像，就像从光盘加载一样。你为什么想这么做？嗯，有些电脑不能直接从 USB 拇指驱动器启动，IsoStick 通过先伪装成光驱，再伪装成拇指驱动器来解决这个问题。几乎所有的个人电脑都可以从光盘启动，这使得 Isostick 非常有用。

你可以将多个 iso 写入 FAT32 驱动器，一个特殊的选择器可以让你选择加载哪个。这意味着您可以在一个 USB 驱动器上保存多个安装映像，并在需要时安装它们。

写创作者:

> isostick 的目标是 IT 人员、计算机技术人员和一般的极客，他们厌倦了随身携带大量总是丢失、损坏、划伤或停止工作的光盘。很多时候，你必须用最新的补丁或病毒定义或诸如此类的东西来更新你的光盘。与 isostick 这是一件轻而易举的事，只需将新的 iso 放在闪存驱动器上，你就可以开始了！

225 美元可以买到一个 32GB 的记忆棒和一些贴纸，而最便宜的 8g 机型售价 125 美元。如果你不确定你会怎么做，你可能不是目标市场，但他们到目前为止已经承诺了 21，000 美元(目标是 25，000 美元)，所以这可能一直是一个需要挠的痒处。
