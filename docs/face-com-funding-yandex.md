# 面部识别技术初创公司 Face.com 从 Yandex 和其他公司筹集了 430 万美元

![](img/ca21395078faed9a5936fc5419d7f081.png)

俄罗斯最大的搜索引擎运营商 Yandex 投资了总部位于特拉维夫的面部识别技术初创公司[Face.com](https://web.archive.org/web/20230406075343/http://www.face.com/)，这是其对以色列公司的首笔投资。在由之前的投资者 [Rhodium](https://web.archive.org/web/20230406075343/http://www.crunchbase.com/financial-organization/rhodium) 牵头的一轮 B 轮融资中，Face.com 总共筹集了[430 万美元](https://web.archive.org/web/20230406075343/http://www.crunchbase.com/company/face-com)。

这个消息最早是由 [TheMarker](https://web.archive.org/web/20230406075343/http://www.themarker.com/tmc/article.jhtml?ElementId=gg20100926_753186) (希伯来语)报道的，但我们已经向 Yandex 的一位代表证实了这个消息，他拒绝透露对这家初创公司的投资金额。

Yandex 确实表示，他们和 Face.com 正在开展一个“大型联合项目”，但未能具体说明该倡议具体包括哪些内容。最后，Yandex 首席执行官 Arkady Volozh 加入了这家初创公司的董事会。

成立于 2007 年的 Face.com，为[脸书](https://web.archive.org/web/20230406075343/http://www.crunchbase.com/company/facebook) ( [拍照器](apps.facebook.com/photo_tagger)和[寻像器](https://web.archive.org/web/20230406075343/http://face.com/apply.php)开发了多个[人脸识别 app](https://web.archive.org/web/20230406075343/https://techcrunch.com/2009/03/24/facecom-brings-facial-recognition-to-facebook-photos-we-have-invites/)；后者当前处于私有 alpha 模式)。

去年五月，该公司称其技术已经帮助用户扫描了超过[70 亿张照片](https://web.archive.org/web/20230406075343/https://techcrunch.com/2010/05/03/7-billion-scanned-photos-later-face-com-opens-up-to-developers/)，并着手推出了一个[开发者社区](https://web.archive.org/web/20230406075343/http://developers.face.com/)和开放 API，为第三方开发者提供他们的核心面部识别技术。

该初创公司最接近的竞争对手之一， [Polar Rose](https://web.archive.org/web/20230406075343/http://www.crunchbase.com/company/polarrose) ，据报道被[苹果](https://web.archive.org/web/20230406075343/http://www.crunchbase.com/company/apple)以大约[2900 万美元](https://web.archive.org/web/20230406075343/http://eu.beta.techcrunch.com/2010/09/20/apple-buys-polar-rose-for-a-rumoured-22-million/)收购。

**更新:** [声明](https://web.archive.org/web/20230406075343/http://blog.face.com/2010/09/27/we-raised-some-money/)来自 Face.com 创始人兼首席执行官 [Gil Hirsh](https://web.archive.org/web/20230406075343/http://www.crunchbase.com/person/gil-hirsch) :

> Face.com 确实接受了一轮资助。总金额为 430 万美元，参与者包括 Rhodium、Yandex 和目前的私人投资者。我们将使用这笔钱来增强我们的开发团队，雇用一些新的和有才华的人，并建立额外的基础设施来支持我们的面部识别平台。
> 
> 我们已经有数千名开发者注册加入我们的平台，并获得我们的面部识别 API，我们希望支持这些开发者，并提供出色的技术。我们很兴奋也很自豪能得到投资者的支持。
