Kindle Code TechCrunch中发现亚马逊“Prime电子书”贷款计划的线索

原文：https://web.archive.org/web/http://techcrunch.com/2011/09/24/clues-to-amazons-prime-ebooks-loan-program-found-in-kindle-code/

Kindle代码中发现亚马逊“Prime电子书”贷款计划的线索

一些论坛用户在移动阅读平台上发现了在线T2 Kindle T3代码中所谓的Prime电子书的证据。在搜索新款Kindle平板电脑的图片时，他发现了一个描述PRIME_EBOOKS_COMPATIBLE的变量。

当试图删除“借出”的图书时，还会显示一些文本，特别是:

prime_active_delete = "您确定要永久删除此贷款吗？这样做将归还贷款，并将其从您的帐户中删除。”

这段代码与同一应用程序中的图书馆借阅文本明显不同。遗憾的是，代码中没有Kindle平板电脑的证据。然而，我们确实证实了这些贷款将作为一种亚马逊“Prime”订阅服务存在，这表明了一种类似于网飞的以固定价格下载大量书籍的方法。
