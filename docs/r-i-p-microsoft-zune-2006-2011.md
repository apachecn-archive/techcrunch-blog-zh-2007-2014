# R.I.P. Microsoft Zune，2006-2011 tech crunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/03/14/r-i-p-microsoft-zune-2006-2011/>

![](img/e6d6686d426b6658e0ae46e139d70e95.png "zune-HD-009-620x412")
彭博正在[报道](https://web.archive.org/web/20230203003822/http://www.bloomberg.com/news/2011-03-14/microsoft-said-to-stop-releasing-new-zune-models-as-demand-ebbs.html)微软最终决定正式终止其 [Zune](https://web.archive.org/web/20230203003822/http://www.crunchgear.com/tag/zune/) 媒体播放器产品线。“一位知情人士”告诉他们，微软不会推出任何新硬件，今后将专注于将 Zune 功能与 Windows Phone 7 平台集成。

并不完全出乎意料；自 2009 年年中发布 [Zune HD](https://web.archive.org/web/20230203003822/http://www.crunchgear.com/2009/09/18/review-zune-hd/) 以来，Zune 硬件没有改变，尽管它已经接受了几次重大的软件升级。这篇文章已经挂在墙上很长时间了，但是微软是否会加倍(再次)或者减少他们的损失还不清楚。

让我们沿着记忆之路快速旅行一下。在西雅图一家潜水酒吧里，一名微软员工不明智地(或者说是明智地)将一个 Zune 放在桌子上，周围都是博客作者，之后，CrunchGear 是世界上第一个看到 Zune 的公司之一。我们狂热地报道了它，因为它似乎是 iPod 的黑马竞争对手，实际上可能有机会。独特的(但笨拙的)共享特征，棕色的外壳(尽管遭到嘲笑，但却是他们最受欢迎的 Zune)和[迟钝的广告](https://web.archive.org/web/20230203003822/http://www.crunchgear.com/2006/11/17/the-zune-ads-microsoft-never-wanted-you-to-see/)都暗示着……嗯，一些新的有趣的东西。

![](img/1498158923bf4665bf1fc32e3cf2944b.png "interface")

各种更新将我们带到了今天的 Zune HD 和 Zune 软件，这两个软件都必须称赞其真正原创和有效的界面。我仍然使用 Zune HD，它仍然得到称赞。但是销售从来不是微软所希望的，也许是因为微软对应用程序和附加功能的骑墙态度。虽然 iPod touch 显然被宣传为一部没有手机的 iPhone，但 Zune 被宣传为完全是关于音乐的——除了它完全是关于云的——除了它有应用程序——除了它没有时钟——除了它在 Windows Phone 7 中——除了它在 Kin 中……等等。这并不完全是一个沉重的负担(它已经卖出了几百万，流媒体服务也很受欢迎)，但面对不屈不挠的 iPod，这并不算什么。

这是一个悲伤的故事，[这是微软无法找到合适子弹的众多案例之一。然而，随着选择与 WP7 整合，以及 WP7 即将通过诺基亚推出，Zune 服务可能会继续存在，尽管几乎可以肯定是作为一个更大的媒体和数据云服务的一部分(可能是他们为](https://web.archive.org/web/20230203003822/http://www.crunchgear.com/2010/03/06/its-time-for-microsoft-to-turn-itself-upside-down/)雇佣的[项目【Ventura】)。他们在流媒体和订阅音乐方面的前瞻性探索(现在反映在 Rdio 和 Spotify 等云音乐服务中)也没有被遗忘。Zune DNA，和 hardy DNA，将会在 Metro 界面和其他任何薄的、剪裁过的无衬线文本中存在。](https://web.archive.org/web/20230203003822/http://www.crunchgear.com/2011/03/08/microsoft-planning-successor-to-zune-service/)

人们可以说这是他们一直以来的计划，作为他们“三屏”环绕服务的一部分，但事实是这一直是 B 计划，现在 Zune 未能从媒体播放器方面削弱苹果的份额，他们将尝试不同的策略。至于玩家呢？没有新的硬件意味着 Zune HD 是其同类产品中的最后一款，尽管它可能会销售更长时间。见鬼，179 美元，我仍然认为 32GB 的版本很划算。现在你知道不会有 Zune HD 2，在某种程度上，原版是经得起未来考验的。

再见了，祖恩。这是一次漫长的棕色旅行。

**更新** : Zune 的硬件业务开发人员[发表了](https://web.archive.org/web/20230203003822/http://www.crunchgear.com/2011/03/15/zune-team-chill-out-about-the-zunes-death/)评论，淡化了这份报告，但很大程度上证实了我们在这里所写的，Zune 作为一个独立的播放器和品牌正在融入 WP7。
