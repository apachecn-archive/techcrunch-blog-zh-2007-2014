# Windows Mobile 上的完整 YouTube 现在成为可能

![logo_youtube.gif](img/91eecadc287944453df4e1f106dd13cd.png)
Windows Mobile 粉丝再也不用嫉妒 iPhone 了。Touch 和 Tilt 即将推出，这两款手机本身都很酷，但现在你可以通过四个简单的步骤观看完整的 YouTube、谷歌视频和 Veoh。


第一步:如果你已经安装了核心的掌上媒体播放器，现在卸载它。一旦你的手机不在 TCPMP，[下载 TCPMP 的这个版本](https://web.archive.org/web/20200806012553/http://picard.exceed.hu/tcpmp/test/)。为您的手机选择合适的版本，智能手机或掌上电脑。安装在你的手机上。

第二步:从[这里](https://web.archive.org/web/20200806012553/http://pdaphonehome.com/forums/ppc-6800-xv6800/90808-full-youtube-access.html)下载 Flash 视频包。将它安装在与核心 Pocket Media Player 相同的位置(存储卡或手机内存)。

第三步:重启手机。

第四步:看一些视频！

> 注意:我已经在我的威瑞森摩托罗拉 q 上测试了这个方法。它应该可以在其他手机/运营商上使用，但我不能保证它可以在所有 Windows Mobile 手机上使用。
