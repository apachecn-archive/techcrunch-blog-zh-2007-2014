# 奥巴马的开放数据秩序如何能够拯救生命，节约能源成本，以及制作酷的应用程序

读者可能已经注意到，今天早上他们最极客的政治取向的朋友像贾斯汀比伯音乐会上的少女一样吓坏了。今天，白宫的大人物签署了一项罕见的行政命令，所有联邦数据都可以以一种软件开发人员可以利用的形式免费访问，这种形式被亲切地称为“机器可读格式”。下面，我们整理了一份用政府数据开发的最酷的应用列表，下面是一个简短的解释为什么这个新项目如此重要。

**1。全球定位系统数据**:如果你是谷歌地图的粉丝，那么你可以感谢罗纳德·里根向数字制图员发布政府的卫星数据。在 1983 年大韩航空公司 007 航班在苏联领空被击落的臭名昭著之后，Gipper 认为[航空公司能够实时定位任何飞机具有战略意义。现在，从地图到 Yelp，GPS 被广泛应用。](https://web.archive.org/web/20221209050558/http://www.faa.gov/about/office_org/headquarters_offices/ato/service_units/techops/navservices/history/satnav/)

**2。拯救生命**:加州圣拉蒙，消防员[与当地黑客](https://web.archive.org/web/20221209050558/http://abclocal.go.com/kgo/story?section=news/local/east_bay&id=7918699)合作开发了一款 iPhone 应用程序，该程序可以自动定位离他们附近发生心脏病发作的最近的 CPR 认证公民。更多救生应用的潜力是巨大的。哈佛大学的信息/法律博客[推测](https://web.archive.org/web/20221209050558/https://beta.techcrunch.com/2013/01/18/we-must-choose-privacy-or-medical-breakthroughs/)与现已失效的万络药物相关的患者心脏病发作数据可以拯救 25，000 人的生命，因为它给了研究人员一个更大的数据集来识别负面副作用。

**3。节能:美国人是众所周知的能源消耗大户，但有时这是因为能源便宜得离谱，我们太忙了，没时间考虑如何减少能源消耗。简单能源移动应用程序插入圣地亚哥的开放数据智能电网，并提醒用户他们自己的能源使用与邻居的相比如何。在一个试点项目中，竞争导致总能耗降低了 11%。**

奥巴马总统被他的工作人员描述为“高极客商”，他在 2009 年的第一个行政命令《开放政府指令》中未能激励联邦机构向公众开放更多数据。这一新指令更有力度，要求开放数据被整合到所有新项目中，[包括流行的社交开发者网站 GitHub 上的一套工具](https://web.archive.org/web/20221209050558/http://project-open-data.github.io/)。从本质上说，现在联邦机构有责任解释为什么他们没有发布数据集，从而增加了完成这项工作的动力。

开放数据专家和记者 Alex Howard 提醒我，新数据也可能让谷歌与昂贵、根深蒂固的政府数据分析公司竞争，如彭博和 Lexis-Nexis。

这是朝着正确方向迈出的一步。现在我们将看到执行是否遵循乐观主义。
