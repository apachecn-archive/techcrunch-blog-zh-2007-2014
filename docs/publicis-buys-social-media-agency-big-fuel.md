# 阳狮收购社交媒体机构 Big Fuel

全球最大的通信公司之一阳狮集团(Publicis Groupe)收购了总部位于纽约的社交媒体机构[Big Fuel](https://web.archive.org/web/20230203050115/http://www.crunchbase.com/company/big-fuel)51%的股份，并有权从 2014 年起将其持股比例提高至 100%。

Big Fuel 将在阳狮 [VivaKi](https://web.archive.org/web/20230203050115/http://www.vivaki.com/about/) 组织下进行整合，并作为其其他一些资产的战略“社交媒体中心”，即数字机构 Digitas、Razorfish、Starcom MediaVest Group 和 Zenith Optimedia。

交易的财务条款没有披露。

我们应该注意到, [Ad Age 报道了](https://web.archive.org/web/20230203050115/http://adage.com/article/agency-news/publicis-groupe-talks-acquire-big-fuel/228698/)上周阳狮正在谈判收购大型燃料公司。

Big Fuel 拥有 170 多名员工，是一家“纯粹”的社交媒体机构，2011 年的收入预计将达到近 3000 万美元。该机构为通用汽车、T-Mobile、微软和飞利浦等公司提供战略建议以及创意、分销、管理和分析服务。

阳狮集团是巴黎泛欧交易所的上市公司，拥有大约 50，000 名员工，业务遍及 104 个国家。
