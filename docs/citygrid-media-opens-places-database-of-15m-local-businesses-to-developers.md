# CityGrid Media 向开发者开放 1500 万本地企业的位置数据库

IAC 拥有的本地媒体和广告资产 [CityGrid Media](https://web.archive.org/web/20230203091853/http://www.citygridmedia.com/) 今天发布了一个全新的 API，向开发者提供超过 1500 万本地企业的位置数据和 CityGrid 广告网络中 100 万本地广告商的数据。

作为背景，IAC [去年推出了 CityGrid](https://web.archive.org/web/20230203091853/https://techcrunch.com/2010/01/29/citysearch-citygrid-local-advertising/) 作为一套 API，使 Citysearch 的所有本地列表内容和广告可供其他网站和移动应用程序使用。该公司[随后将其类似 Yelp 的本地商业内容平台城市搜索更名为城市网格媒体的一部分。](https://web.archive.org/web/20230203091853/https://techcrunch.com/2010/06/02/citysearch-recasts-itself-as-citygrid-media/)

Citysearch 中的所有本地列表都可以通过 CityGrid 的 API 获得，因此任何创建移动应用程序或本地网站的人都可以获取企业列表、地址、电话号码、照片、评论等等，并围绕它们创建自己的应用程序。CityGrid 还将本地广告商与这些本地出版商相匹配。

今天的公告提供了该公司迄今为止最全面的地点数据库，因为 CityGrid 已经与 Infogroup 合作，将 Infogroup 的 1500 万家当地企业的完整数据库添加到 API 中。所有这些都将与 CityGrid 的内置广告客户群配对，并通过新的 API 套件交付。

该 API 包括超过 1500 万个本地企业名录的位置数据、全国 75，000 个城市的近 100 万个广告客户、有针对性的网络和移动格式以推动与商业广告客户的互动、iOS 和 Android SDKs 等。

CityGrid 还计划向开发者免费提供其 API，并将在 [TechCrunch Disrupt 黑客马拉松上主持一场研讨会。](https://web.archive.org/web/20230203091853/https://techcrunch.com/2011/09/08/techcrunch-disrupt-hackathon-details/)
