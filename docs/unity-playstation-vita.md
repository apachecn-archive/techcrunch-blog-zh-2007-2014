# Unity 开发者现在可以为索尼的 PlayStation Vita 开发游戏了

Unity 的“一次开发，为一切而建”的武器库还在继续增长。

今天早上，Unity 的伙计们兑现了他们在去年 3 月首次做出的承诺:支持索尼最新的掌上电脑 PlayStation Vita。

这可能不是最好的时机，因为索尼目前正试图解释为什么没有人购买 Vita。

对于那些可能不知道的人来说，Unity 是一个从头开始构建的游戏引擎，旨在实现游戏开发者多年来的梦想:能够一次性构建一个游戏*，并以最小的努力让该游戏可以在许多不同的平台上运行。当然，你仍然需要对每个游戏进行优化和平台特定的事情，如触摸控制或运动传感器，但与为每个平台重写大量代码相比，这只是沧海一粟。*

 *(想了解更多？ [Slashdot 有一个伟大的——虽然现在有点过时——看看 Unity 迄今为止的故事](https://web.archive.org/web/20221209040554/http://slashdot.org/topic/cloud/how-unity3d-become-a-game-development-beast/)

有了这个补充，Unity 已经没有平台可以扩展了。它现在支持:PC，Mac，Linux，iOS，Android，BlackBerry，Windows Phone，PS3 和 PS4，Xbox 360 和 Xbox One，以及 Wii U(尽管使用游戏机比使用电脑或智能手机复杂得多——在大多数情况下，在开始使用之前，你需要得到游戏机制造商的认可)。哦，还有一些游戏可以通过 Unity 的浏览器插件在网上玩。

Unity 不仅仅提供游戏移植支持，它还将 Vita 特有的功能整合到了游戏中。有了它，开发人员就可以使用 Vita 的运动传感器、摄像头和 Vita 的疯狂后触摸板。

唉，即使你已经建立了 Unity 并且附近有一个 Vita，你可能还不能开始为 Vita 构建游戏。像市场上几乎所有其他游戏机一样，索尼把事情锁得很紧——如果你想在这个平台上构建，你必须说服索尼让你成为一名授权开发者。对于我们这些刚刚涉猎的人来说，那…可能有点挑战。*
