# Canonical 寻求 3200 万美元来开发一款为融合计算设计的 Ubuntu 智能手机 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/22/ubuntu-smartphone-canonical/>

【YouTube http://www.youtube.com/watch?v = eQLe3iIMN7k？feature = player _ 嵌入式]

Ubuntu Mobile 并不新鲜；Canonical 已经做了大量工作来推动其打入智能手机移动软件领域的努力。但是今天，该公司正在发起一项 Indiegogo 活动，为其第一个自有品牌 Ubuntu 移动硬件 Ubuntu edge 的开发提供资金。Edge 是一款智能手机，它还希望取代台式电脑。

为 Edge 的创建提供 3200 万美元资金的活动是一个固定资金项目，这意味着它要么全有，要么全无，它必须在接下来的 31 天内实现目标才能成功。Canonical 每天需要筹集超过 100 万美元才能实现目标，这意味着，对于第一天每台预购价格为 600 美元的高概念设备，之后每台设备的价格为 830 美元。这可不是件容易的事。

但根据 Canonical 的项目页面，Canonical 希望打造的不仅仅是一款基于 Ubuntu 的智能手机，而是一个“由爱好者和移动计算专业人士众筹的低容量高科技平台”，旨在“加速新技术的采用，并将其纳入主流”。这是关于为那些为日常世界制造小器具的修补匠和建筑商制造一个小器具，它的价格标签和雄心勃勃的设计反映了这一点。

Edge 是一项真正融合计算的实验，它将智能手机变成了一台完整桌面 PC 的中枢神经系统，从而产生了一部自称的“超级手机”。它被包裹在金属外壳中，配有 4.5 英寸 1280×720 屏幕，其设计不是为了提供最大的像素密度，而是为了提供更好的动态范围和色彩准确度。屏幕由蓝宝石玻璃覆盖，Edge 将拥有“最快的”多核移动处理器、4GB 内存和 128GB 存储空间，以及旨在将寿命推向极限的电池。内部还有一个双 LTE 芯片，用于广泛的无线覆盖范围。

在软件方面，Edge 结合了 Ubuntu Mobile 和完整的 Ubuntu desktop，甚至可以引导到 Android，实现真正的融合计算体验。这听起来很神奇，毫无疑问，各地的移动书呆子现在已经对此垂涎三尺，或者一旦他们发现这是真的，就会开始垂涎三尺。

当然，它还不是真实的，需要 3200 万美元才能成为现实，这是一个很高的要求。但 Canonical 是一家有着可靠记录的公司，这是一个令人兴奋的项目，肯定会让早期采用者社区兴奋不已。这是否会提供足够的动力来帮助 Canonical 推进移动计算仍有待观察-但它已经在很短的时间内积累了近 20，000 美元，因此显然有相当数量的早期兴趣。
