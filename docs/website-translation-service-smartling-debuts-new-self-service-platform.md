# 网站翻译服务 Smartling 推出新的自助服务平台

实时网站翻译服务 [Smartling](https://web.archive.org/web/20230203135128/http://www.smartling.com/) 首次推出其平台的新自助服务版本，该版本既适合小企业，也适合大公司。

Smartling 为网站提供大规模的搜索引擎优化友好的实时翻译服务。该公司有一个混合模式，允许你在专业翻译、机器翻译和众包翻译之间进行选择。

该平台的新功能包括风格指南、面向翻译人员的“学习”术语表、所有翻译页面的 SEO
合规性以及众包管理工具。Smartling 还为高流量和复杂的站点提供企业计划，包括 API 访问、SSL 支持和额外的培训和集成服务。Foursquare、IMVU、Scribd 和 SurveyMonkey 等公司都使用了 Smartling 进行翻译。

Smartling 刚刚筹集了[1000 万美元](https://web.archive.org/web/20230203135128/http://gigaom.com/2011/07/27/smartling-gets-10m-to-bring-translation-to-more-websites-apps/)的新资金，并且[最近与【CloudFlare 合作，允许这家网络安全初创公司的用户启用翻译。](https://web.archive.org/web/20230203135128/https://techcrunch.com/2011/08/05/smartling-partners-with-cloudflare-to-package-security-and-language-translation-for-users/)
