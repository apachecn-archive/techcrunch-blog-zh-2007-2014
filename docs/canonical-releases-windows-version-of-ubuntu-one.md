# Canonical 发布了 Ubuntu One 的 Windows 版本

[Ubuntu](https://web.archive.org/web/20230203155230/https://techcrunch.com/tag/ubuntu) Linux 发行版背后的商业支持者 Canonical ，几年来一直在托管一项名为 [Ubuntu One](https://web.archive.org/web/20230203155230/http://one.ubuntu.com/) 的文件同步服务。一个免费帐户可以给你 5GB 的存储空间，客户端控件已经被嵌入到 Ubuntu 发行版的最后几个版本中。它的工作方式非常类似于 Dropbox 或类似的服务，但直到今天，它只适用于 Linux。

在昨晚发布的公告中，Canonical 透露现在有一个 Ubuntu One 的 Windows 客户端，允许你从 Linux 或者 Windows 电脑上访问你所有的文件。

> 我们很早就收到了 Ubuntu 用户的反馈，关于他们不断发展的需求，即从一个安全的地方跨多个平台和设备管理他们所有的内容。我们已经研究了许多用例，最常见的是使用多个设备或操作系统的 Ubuntu 用户。许多人不得不在 Windows 或 Mac 环境下工作，即使他们更喜欢使用 Ubuntu 作为他们的家庭桌面或操作系统。另一个例子是提供更多跨平台共享的机会。例如，在一个家庭中使用不同操作系统的家庭可以使用 Ubuntu One 作为他们存储所有音乐、文档、照片的中心位置，并轻松地与彼此和朋友分享。

Ubuntu One 还提供了一个漂亮的[音乐流媒体](https://web.archive.org/web/20230203155230/https://one.ubuntu.com/services/music/)服务。每月支付 4 美元，你就可以获得 20 GB 的存储空间，还可以将音乐文件从你的账户传输到你的移动设备上。Ubuntu One 应用程序适用于 iOS 和 Android，Android 设备获得了可控离线缓存的额外好处，允许您在不需要网络连接的情况下收听您的云存储音乐。很棒的东西。
