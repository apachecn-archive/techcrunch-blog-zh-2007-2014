# GigaOM 发布...一家电视台 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/06/09/gigaom-launchesa-tv-station/>

 [](https://web.archive.org/web/20221006013344/http://station.newteevee.com/) 有趣的新网站在 GigaOM 的热门视频新闻博客 NewTeeVee 上发布-[NewTeeVee 站](https://web.archive.org/web/20221006013344/http://station.newteevee.com/)。该网站包含来自 YouTube、Blip 等各种视频网站的视频。由《综艺》和《每日卷轴》的编辑利兹·香农·米勒挑选。

NTV Station 正在利用为 Ostatic 创建的一些工具，这是 GigaOM 最近推出的另一个网站——包括概述、编辑、用户评论和评级。他们还维护着一个节目、演员、工作人员和其他信息的数据库。他们还使用 VodPod 来收集和存储视频。

下面是米勒的视频概述。看起来 NTV 电视台并没有走向利基市场，而是试图专注于热门视频和未来的东西。在我看来，要找到真正有趣的东西，很难与 Digg video 竞争。但是，嘿，所有这些内容都是由他人创建的，这使得维护网站变得容易多了。

一些附加功能，来自 [Om Malik 的帖子](https://web.archive.org/web/20221006013344/http://gigaom.com/2008/06/09/introducing-newteevee-station/):

*以利兹·香农·米勒领导的团队撰写的在线视频编辑评论为特色，她来自《综艺》和《每日卷轴》。
*仅作为发布工作的一部分，就有超过 100 个 NTV 节目&视频。
*拥有一个演员、工作人员和其他细节的数据库，描绘出不断扩大的网络视频世界。
*邀请读者通过添加他们自己的评论、意见和评级，成为编辑过程的一部分。
*拥有八个最令人难忘的喜剧、广告、戏剧、音乐、新闻&谈话、人物、现实和时代精神精选频道。
*展示当下最热门的内容，就在首页。
*可通过电台 RSS 订阅或捆绑 NewTeeVee 订阅。
