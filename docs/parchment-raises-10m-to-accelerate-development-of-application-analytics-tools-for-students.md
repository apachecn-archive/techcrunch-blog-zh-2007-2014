# 羊皮纸筹集 1000 万美元，用于加速学生应用分析工具的开发

羊皮纸是一家初创公司，它已经开始了在线获取和分享学术和其他证书的离线老化过程，今天宣布了一轮 1000 万美元的后续融资，[使其迄今为止的总投资](https://web.archive.org/web/20221005200707/https://beta.techcrunch.com/2012/10/18/used-by-30-of-u-s-high-schools-parchment-lands-23-5m-to-bring-transcripts-student-data-online/)达到 4500 万美元。新一轮由 Raine Group 牵头，包括该公司以前的投资者，将用于引进新人才和开发新产品，既涉及将羊皮纸支持的[类型的证书扩展到学术界以外，也涉及帮助使用其数据的学生在入学时更好地利用这些信息。](https://web.archive.org/web/20221005200707/http://www.parchment.com/)

该公司自 2003 年开始运营，现在由 Matthew Pittinsky 领导。皮汀斯基没有创建公司，但他在 1997 年创建了 Blackboard，并在 2011 年带着羊皮纸加入董事会。在皮汀斯基的领导下，该平台得到了发展，注册消费者用户在 2014 年增长了 90%，从 80 万增加到不到 160 万。

“许多教育项目都在使用 Blackboard，从某种程度上说，我觉得很奇怪的是，我们投资创新这些项目背后的教学和学习流程，并将其转移到网上，但你获得的证书仍然非常依赖纸张，”“你仍然会收到一个信封，信封里装着成绩单，数据会因此而被截留，许多认证和许可证都是如此，这就是我对将其带到网上的想法感到非常兴奋的地方。”

自从加入该公司以来，Pittinsky 还帮助它将重点从单纯关注学生和学术记录转移到关注培训和认证非常重要的其他领域的机会——这基本上扩展到了你能想到的所有商业和企业环境，包括内部和外部。羊皮纸已经开始探索这些机会，但计划在新的现金注入的支持下做得更多。

它还希望帮助其不断增长的学生用户群从该平台中获得更多价值。[它已经提供了分析，可以根据学生获得的证书，告诉他们被某个项目或学校录取的可能性有多大。皮汀斯基说，产品添加将利用这些数据达到更高级的目的，并帮助学生获得更多关于什么可能给他们带来优势以获得他们所追求的项目认可的方向。](https://web.archive.org/web/20221005200707/https://beta.techcrunch.com/2012/10/18/used-by-30-of-u-s-high-schools-parchment-lands-23-5m-to-bring-transcripts-student-data-online/)

将离线数据转移到网上是一个艰苦的过程，在发展缓慢的教育领域更是如此。但羊皮纸正在取得进展，向用户和客户展示这些记录中包含的大量离线信息中有有用的数据可以挖掘，这是让他们加快转变步伐的好方法。
