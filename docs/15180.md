# 脸书国际电话公司，将发展中国家的在线社交带到第一位

脸书的下一个十亿用户还没有智能手机，甚至没有互联网，但脸书的手机可能会改变这一点。通过精明的运营商交易、补贴手机和免费的有限数据访问，脸书可以确保新兴市场在第一时间收到好友请求。这可能会把东南亚、非洲和拉丁美洲的人口变成脸书的终身使用者。

国际计划和运营商协议是我们在今天脸书的大[“Android 之家”发布会](https://web.archive.org/web/20230404211559/https://techcrunch.com/2013/03/28/facebook-android-phone)之前没有听到太多的东西。虽然手机和软件必须由大团队来制造，但商业交易可能更容易保密。很少有人猜测，因为几乎没有证据可循。

与此同时，脸书让这些国际用户提前介入。它们现在可能不值钱了。在“世界其他地区”，脸书的每名用户收入仅为 1.84 美元。但到移动世界成熟，他们拥有智能手机时，他们的社交网络将会激增，他们将被锁定在脸书。该公司开始完成连接世界的使命，为下一波广告受众做准备。

那么脸书的电话是从哪里来的呢？这让脸书加速了这种成熟。脸书可以与 HTC 或其他手机制造商合作开发廉价的 Android 智能手机，并与运营商合作补贴有限的脸书接入。这可能会让脸书推出一款便宜的智能手机，或者在某种程度上，可以免费拥有和使用。在这些设备上，脸书不会像其应用程序、Facebook Zero 或脸书那样成为每部手机的选项。它会渗透到整个经历中。

作为分发智能手机时代打折介绍的交换，脸书可以在其仍有增长空间的地方赢得用户。谷歌和其他公司也有他们自己的无障碍倡议，但这是我如何看待脸书追赶世界其他地方。

我们今天可能不会听到太多这方面的内容。但从长远来看，品牌手机是脸书正在进行的全球无障碍计划的自然延伸。每个人最终都会拥有智能手机，如果智能手机在设计上是社会性的，对脸书来说会有很大好处。
