# Twitter 与 BBC 全球新闻签约，加入其视频电视广告节目 Amplify 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/24/twitter-signs-bbc-global-news-to-its-in-stream-video-tv-ad-program-amplify/>

Twitter 已与 BBC 全球新闻签约，加入其视频电视广告定向节目 Amplify，这是 Twitter 首次与全球新闻机构合作。上个月，它把哥伦比亚广播公司加入了 T2 的名单。该计划最初的合作伙伴包括 BBC 美国、福克斯、Fuse 和天气频道。

今年 5 月首次宣布的[Amplify 计划，允许广播公司近乎实时地在他们的推文中嵌入视频短片。然后，这些剪辑由广告商赞助，广告商既宣传推文，又在视频前播放简短的前置广告。](https://web.archive.org/web/20221006100000/https://beta.techcrunch.com/2013/05/23/twitter-launches-twitter-amplify-for-real-time-videos-in-stream-partnering-with-bbc-fox-fuse-and-weather-channel/)

BBC 全球新闻有限公司数字技术总监詹姆斯·蒙哥马利在一份声明中说:“这一新的合作利用了我们的全球影响力和新闻采集能力，为我们的广告合作伙伴带来了聪明而令人印象深刻的产品。”。“在 BBC 美国与 Twitter 合作的基础上，我们很高兴将我们的广告合作伙伴带到新闻前沿。”

这一合作将使 BBC 全球新闻视频剪辑内容出现在 Twitter 上。从 BBC 的角度来看，该公司表示，这种合作关系为其提供了一种让其内容触及新观众的方式。

作为合作的一部分，BBC.com 正在制作一个新的推特广播，名为#BBCTrending。这将采取一系列简短视频广播的形式，由 Anne-Marie Tomchak([@ AMTomchak](https://web.archive.org/web/20221006100000/https://twitter.com/AMTomchak))主持，利用 BBC 的新闻采集，挖掘每天社交媒体上的最新“趋势现象”。

#BBCTrending 将于今年秋天晚些时候推出，分发给 BBC 国际新闻 Twitter handle @[BBC world](https://web.archive.org/web/20221006100000/https://twitter.com/search?src=typd&q=BBCWorld)的 480 万粉丝，并通过 Twitter Amplify 进行推广。

Tomchak 在一份声明中说:“在趋势方面，我们正在利用最强大的内部洞察工具和大规模的 BBC 国际新闻收集和语言操作，来破译世界各地社交媒体上趋势发生的原因和方式。”

本月早些时候，Twitter 的 Deb Roy(电视社交影响分析公司 Bluefin Labs 的联合创始人，今年早些时候被 Twitter 收购，继续建立其社交电视工具库)，将电视描述为其业务的“T2 中心”。8 月， [Twitter 还收购了 Trendrr](https://web.archive.org/web/20221006100000/https://beta.techcrunch.com/2013/08/28/twitter-continues-to-beef-up-its-social-tv-efforts-with-trendrr-acquisition/)——一家专注于电视的社交媒体参与度追踪内容公司。
