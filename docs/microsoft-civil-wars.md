# 迷失:如果微软不能活在一起，微软可能会孤独终老 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/02/04/microsoft-civil-wars/>

今天《纽约时报》有一篇非常有趣的专栏文章，题为[微软的创造性毁灭](https://web.archive.org/web/20230130125719/http://www.nytimes.com/2010/02/04/opinion/04brass.html?partner=rss&emc=rss)。在这篇文章中，作者详细阐述了他认为微软多年来未能与竞争对手同步创新的原因。有什么大不了的，对吧？毕竟，很多人一直在写这种类型的作品。这次有什么不同？这是迪克·布拉斯写的，他从 1997 年到 2004 年在微软担任了七年的副总裁。

布拉斯的评论在引起了如此大的轰动，以至于微软的企业沟通副总裁甚至被迫[在他们的官方博客上做出回应](https://web.archive.org/web/20230130125719/http://blogs.technet.com/microsoft_blog/archive/2010/02/04/measuring-our-work-by-its-broad-impact.aspx)。虽然微软的公关团队显然会淡化布拉斯的一些评论，并用反例反驳其他人，但这篇文章完全忽略了我认为布拉斯的主要观点:微软已经成为一个公司内部发生数十场甚至数百场内战的地方。

布拉斯举的一个例子是他参与开发的 ClearType。虽然微软反驳说，ClearType 现在是 Windows 的一部分，但布拉斯说，如果公司内的其他团队没有“感到受到我们成功的威胁”的话，ClearType 应该而且早就实施了

他进一步解释说，pocket devices 的副总裁甚至说，只有他的团队完全控制了它，他们才会支持它。他还注意到办公室主任对此表示反对。这很有意思，因为他也指责办公室主任扼杀了微软早在 2001 年就开始的平板电脑概念。

正如布拉斯所说，办公产品副总裁拒绝修改他的软件，使其与平板电脑兼容。这与苹果在上周的 iPad 发布会上展示的完全相反，史蒂夫·乔布斯带着 iWork 团队上台展示他们如何专门为新产品重新设计软件。

显然，iWork 不是 Office。无论你如何看待这两款产品，iWork 在苹果的大收入图景中只是非常小的一部分，而 Office 则是微软的很大一部分。但这正是重点。正如布拉斯指出的，微软正在创造大量收入，但几乎所有收入都来自两个地方:Windows 和 Office。正因为如此，这两个分支无疑拥有巨大的权力，他的两个例子展示了这一点。

这方面的问题是，如果谷歌(以及在较小程度上的苹果)得逞，Windows 和 Office 都将变得不那么重要。显然，这远非板上钉钉的事情，但一切向移动和/或网络发展的趋势非常明显。如果他们不同意，微软也不会在网络部门投入这么多。但是，尽管有这些投资，从收入的角度来看，微软在网络领域的情况并不太好。事实上，他们正在变得可怕。根据布拉斯的说法，原因如下:

> 内部竞争在大公司很常见。可以明智地鼓励让思想去竞争。当竞争变得不受控制和具有破坏性时，问题就来了。在微软，它创造了一种功能失调的企业文化，在这种文化中，大型老牌集团被允许掠夺新兴团队，贬低他们的努力，与他们不公平地竞争资源，随着时间的推移，迫使他们消失。过去十年来，几乎所有负责微软音乐、电子书、电话、网络、搜索和平板电脑业务的高管都离开了，这并非偶然。

微软不是苹果。也就是说，他们的员工会经常谈论他们为软件巨头工作的经历。如果说他们中的许多人都赞同布拉斯关于团队相互对抗的观点(在某些情况下，同一团队中的人相互对抗)，这是一种保守的说法。事实上，我可以说*他们中的大多数*都承认这是公司内部的一个主要问题。即使他们说在他们实际的团队中这不是问题，他们也会承认在公司的其他团队中这是一个大问题。

那是管理问题。如果它涉及到某些部门比其他部门获得优先权或资源，这是一个上层管理问题。微软的支持者总是很快指回收入，好像在说，“如果我们赚了这么多钱，事情不会变得那么糟糕。”底线显然很重要，但它没有解决任何潜在的问题。没有什么是永恒的，包括 Windows 和 Office。当创新的新产品遭受损失时，依靠这些是愚蠢的。

另一位前微软员工唐·道奇在最近几个月也在这些问题上对 T2 提出了批评。以下是他最近不得不[告诉 VentureBeat 的](https://web.archive.org/web/20230130125719/http://venturebeat.com/2009/12/17/former-microsoft-evangelist-don-dodge-on-google-vs-microsoft-qa/):

> 从高层次来看，今天的微软相当于 80 年代末、90 年代初的 IBM。当我刚刚开始我的职业生涯时，IBM 统治着世界。IBM 是世界上占主导地位的计算机供应商——硬件、软件、网络，凡是你能想到的，IBM 都是王者。我认为在 80 年代末和 90 年代初，我们看到了这种转变，微软成为了霸主。在 2009 年、2010 年，展望未来，微软有点像 IBM。这是一家历史悠久的公司，拥有伟大的传统，仍然非常盈利，但它不是领导者。

事实是，现在最热门的两个话题是手机和平板电脑。微软很早就涉足这两个领域，但不知何故要么完全失败(平板电脑)，要么让他们的产品在绷带时代失血过多而死(Windows Mobile)。好像不是缺乏远见，是缺乏执行力。

那么，所有这些内讧的根源是什么？微软的当权者是否太老了(一位微软观察家[思考过](https://web.archive.org/web/20230130125719/http://redmondmag.com/articles/2009/12/01/is-microsoft-too-old-for-web-20.aspx)两次[现在](https://web.archive.org/web/20230130125719/http://blogs.zdnet.com/microsoft/?p=4859))。整个公司是不是太大了(即使在裁员 5000 多人之后，至少有一个微软人这么认为)。是工人们缺乏热情吗？更重要的是，微软打算对此做些什么？

随着 ABC 的热门节目 *Lost* 进入最后一季，几周以来基本上不可能避免关于它的商业广告。但其中一个让我想起了微软的这种情况。也许公司需要一个杰克·谢帕德式的领导，把他们召集在一起说，“如果我们不能一起生活，我们就会孤独终老”
