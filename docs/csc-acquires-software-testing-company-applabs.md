# CSC 收购软件测试公司 AppLabs

[计算机科学公司](https://web.archive.org/web/20230203065647/http://www.csc.com/) (CSC)今天早上[宣布](https://web.archive.org/web/20230203065647/http://www.businesswire.com/news/home/20110914005893/en/CSC-Acquires-AppLabs)它已经[收购了](https://web.archive.org/web/20230203065647/http://www.crunchbase.com/company/applabs)印度的 [AppLabs](https://web.archive.org/web/20230203065647/http://www.applabs.com/) ，一家软件测试和质量管理服务提供商。交易的财务条款没有披露。

此次收购旨在发展 CSC 的应用开发和管理业务，并向其客户提供软件测试和质量管理服务。

AppLabs 的所有 2，500 名员工都将加入这家总部位于美国的 IT 公司。

该公司于 2001 年由 Sashi Reddi 创立，并得到了 WestBridge Ventures 和 Sequoia Capital 的投资。客户包括美国航空、益百利和摩根大通。

AppLabs 总部位于印度海得拉巴，在美国和英国设有办事处。
