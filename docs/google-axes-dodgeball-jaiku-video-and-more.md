# Google Axes 躲避球、Jaiku、视频等 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/01/14/google-axes-dodgeball-jaiku-video-and-more/>

![](img/e7ff69de24e4548d77561ccfe1535394.png)

在谷歌，祸不单行。在今天下午宣布第一轮[裁员](https://web.archive.org/web/20221007153818/http://googleblog.blogspot.com/2009/01/changes-to-recruiting.html)之后，谷歌发布了几篇博客文章，详细介绍了即将关闭的一些服务(这里[由丹尼·沙利文编译](https://web.archive.org/web/20221007153818/http://searchengineland.com/google-ends-google-video-uploads-shutters-notebook-catalog-search-dodgeball-jaiku-16166))。即将关闭的包括:谷歌笔记本、谷歌目录、躲避球、谷歌视频、谷歌混搭编辑器和 Jaiku 的未来发展(尽管这项服务将继续存在)。

下面我们总结了即将到来的变化:

Jaiku 目前正在移植到 Google App Engine，并将作为 Google Code 上的开源项目发布。然而，虽然谷歌不会有进一步的发展，但它将继续保持在线。

允许用户通过短信分享他们当前位置的闪避球将被完全停止，尽管谷歌尚未确定比“几个月”更具体的时间表。

Mashup 编辑器(处于私有测试阶段)正被 App Engine 所取代。(点击[此处](https://web.archive.org/web/20221007153818/http://google-code-updates.blogspot.com/2009/01/changes-for-jaiku-and-farewell-to.html)查看谷歌在 Jaiku、Dodgeball 及其混搭编辑器上的帖子)。

[谷歌笔记本](https://web.archive.org/web/20221007153818/http://googlenotebookblog.blogspot.com/2009/01/stopping-development-on-google-notebook.html)将继续为当前用户服务，但不再接受新用户。然而，现有用户将无法使用浏览器扩展，这使得该服务的实用性大大降低。谷歌的替代建议包括搜索维基、谷歌文档、任务(Gmail)和谷歌书签。

[目录搜索](https://web.archive.org/web/20221007153818/http://booksearch.blogspot.com/2009/01/farewell-google-catalog-search.html)的目的是演示[光学字符识别](https://web.archive.org/web/20221007153818/http://en.wikipedia.org/wiki/Optical_character_recognition)，并很好地满足需求。现在，随着人们的注意力转移到谷歌的图书搜索上，它已经失宠了。

最后，谷歌视频的上传功能将在几个月后被禁用，尽管用户仍然可以观看系统中已经存在的内容。这是一个漫长的过程，因为在 2006 年谷歌[收购 YouTube](https://web.archive.org/web/20221007153818/http://www.beta.techcrunch.com/2006/10/09/google-has-acquired-youtube/)后，谷歌视频在很大程度上被认为是多余的。

目前还不清楚这是否是一个新趋势的开始:谷歌是否已经在未来几个月大幅削减了所有计划，或者其不太受欢迎的产品的未来仍有疑问？[谷歌在维基百科上的竞争对手知识单元](https://web.archive.org/web/20221007153818/http://knol.google.com/k)，在很大程度上没能赶上。Grand Central 是一种先进的电信服务，拥有一群虔诚的粉丝，但仍未向公众开放，这种情况如何？
