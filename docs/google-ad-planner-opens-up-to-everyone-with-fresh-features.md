# 谷歌广告策划向每个人开放了新的功能

去年六月，谷歌推出了一个新的研究和媒体测量工具，名为[广告策划](https://web.archive.org/web/20221208235821/https://www.google.com/adplanner)，我们怀疑[使用谷歌工具栏收集数据](https://web.archive.org/web/20221208235821/http://www.beta.techcrunch.com/2008/06/24/is-google-ad-planner-getting-its-data-from-the-google-toolbar/)。今天，该公司为任何拥有谷歌账户的人开放广告策划器，并增加了一系列功能。

谷歌广告策划现在支持搜索查询(惊喜！)和地理定位，这意味着您可以深入到特定的州或城市(美国以外的地区或城市)。你还可以在三种新的排名方法中进行选择，以显示你正在考虑开展活动的网站的结果，还有一个新的交互式“气泡图”，它应该可以帮助你直观地比较人口统计数据、频率、流量和独立访客。

谷歌还扩大了人口统计受众数据，包括法国、德国、意大利、西班牙和英国。

要使用该服务，请访问[广告策划主页](https://web.archive.org/web/20221208235821/http://www.google.com/adplanner)并使用您的谷歌账户登录。
