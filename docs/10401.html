后端服务提供商StackMob来到Android TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/10/12/backend-service-provider-stackmob-comes-to-android/

面向移动开发者的后端服务提供商StackMob 今天发布了重大消息:它现在可以在Android上使用了。随着今天的发布(仍处于私人测试阶段——见下文邀请函), StackMob将向Android开发者提供对其服务的全面访问，包括一个开源Android SDK 和对Android推送通知的支持。

这个平台，名爵西格勒曾称之为“移动的Heroku”(在 StackMob与Heroku 合作之前)！)，是一个基于云的系统，旨在解决移动开发者的所有后端需求，包括社交整合、存储、消息传递、API创建、分析、货币化等。这是一个灵活、可扩展的解决方案，如果需要，它甚至允许开发人员使用自定义代码向他们的应用程序添加复杂的逻辑。

联合创始人Ty Amell解释说，新的Android SDK提供了与iOS SDK相同的功能，这意味着开发人员可以通过用户名向给定用户发送推送通知，而不必担心他们当时使用的是什么设备。StackMob会处理那部分。

推送通知功能适用于运行2.2及更高版本的Android设备，Amell表示，Android平台上约78%的设备都使用该功能。

有兴趣注册Android stack mob私有测试版的开发者可以从这里开始:http://www.stackmob.com/tc-android。有300个空位。

我们被告知，未来，StackMob计划通过更详细的应用分析和API分析来进一步构建其分析系统。它最近对其后端进行了彻底检查，将架构从专有系统改为开源Flume，主要是因为该公司喜欢支持开源，并且因为…嗯，为什么要在有可用系统的地方构建自己的系统呢？该公司仍未公开宣布退出私有测试版时的定价情况，但私有测试版的开发者可以免费测试该平台。

StackMob此前在5月份的一轮融资中筹集了750万美元，由Trinity Ventures牵头，现有投资者Harrison Metal和Baseline Ventures参与。它现在在旧金山有15名员工，但正在寻求招聘更多员工。
