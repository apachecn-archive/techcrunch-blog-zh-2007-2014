# iPhone 和 Android 现在占美国智能手机网络广告的 81%

![](img/f4a302b5feeb420defb6d2704889d716.png)

说到移动网络，越来越多的只有两个重要的移动平台:苹果的 iPhone 和谷歌的 Android。根据今天由 [AdMob](https://web.archive.org/web/20230316172515/http://metrics.admob.com/2010/01/december-2009-mobile-metrics-report/) (正在被谷歌收购的[)发布的市场份额数据，2009 年第四季度，iPhone 和 Android 占据了美国智能手机上 81%的移动广告展示，高于 2009 年第一季度的 55%。大部分增长来自安卓系统，从第三季度开始，安卓系统的市场份额翻了一番，达到 27%。iPhone OS 占据了另外 54%的份额。(在全球范围内，iPhone 占 51%的份额，Android 占 16%，合计占 67%)。](https://web.archive.org/web/20230316172515/https://techcrunch.com/2009/11/09/google-acquires-admob/)

早在 2009 年第一季度，Android 在移动广告展示中仅占 4%的份额(这也是整体移动网络和应用使用的一个不错的代表)。它夺走了黑莓和 Windows Mobile 的大量市场份额，前者在去年从 20%上升到 10 %,后者从 12%上升到 3%。而 Palm，Android [很久以前就超过了](https://web.archive.org/web/20230316172515/https://techcrunch.com/2009/04/23/android-catches-up-to-palm-in-mobile-ad-market-share-iphone-still-blows-it-away/)，相比一年前 7%的份额，现在几乎是一个舍入误差。在这段时间里，不仅 Android 和 iPhone 的市场份额一直在增长，整体移动网络流量也在增长。仅在北美，移动网络流量在过去一年就翻了一番。

这越来越像是 iPhone 和 Android 之间的两匹马的比赛，所有其他手机都无关紧要。难怪谷歌只关心针对这两个平台的移动广告。
