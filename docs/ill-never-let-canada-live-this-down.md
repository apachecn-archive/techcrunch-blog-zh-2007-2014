# 我永远不会让加拿大活下来:邪恶拼车创业罚款

关于一桩荒谬诉讼的荒谬更新:8 月份，我写了一篇关于拼车初创公司的文章，名为[pick upal](https://web.archive.org/web/20230203033652/http://www.pickuppal.com/pup/intro.html;jsessionid=6B3B3EE3B296EDF0E8B944F7AB2D5F94)。这个想法是，人们可以聚集在网站上找到其他人去同样的地方旅行，并拼车去那里节省汽油。

好主意，对吧？不对。巴士公司吓坏了，并根据安大略省的一项法律提起诉讼，该法律限制拼车者只能从家到工作地点往返，每天与同一名司机共乘，只按周付费，以及其他限制。尽管政府已经在拼车车道上花费了“数十亿”的资金。

反正官司判决了[皮卡帕尔输了](https://web.archive.org/web/20230203033652/http://save.pickuppal.com/2008/11/12/ohtb-takes-the-side-of-the-bus-company/)。他们被罚款 11，000 加元，并被迫将这种卑鄙的拼车活动限制在法律的严格范围内:

> 我们从安大略省公路运输委员会(OHTB)得到了我们的裁决，他们说在安大略省拼车是非法的，(这是官方决定[PDF 0.98 MB] )。
> 
> 你能和某人共乘的唯一方法是，如果你满足以下所有极其不切实际的特定标准:
> 
> *您必须仅从家到工作单位出差—(不是从家到学校， 或者是去医院或机场的家)
> *你不能跨越城市边界——(住在城外，开车进去——抱歉，你不能和邻居一起坐车)
> *你必须每天和同一个司机一起坐车——(想混在一起，一天和一个人一起去，另一天和另一个人一起去——不，抱歉，做不到——每天必须是同一个人)
> * 你给司机的钱不能超过每周一次——邻居开车送你去上班更好，不要马上付钱给她，以防她在一周的晚些时候开车送你
