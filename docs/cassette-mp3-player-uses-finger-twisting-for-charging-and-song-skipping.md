# 盒式 MP3 播放器使用手指扭转来充电和跳过歌曲

![nvrd_10](img/1406783cd577498e7b23455fe70880f1.png "nvrd_10")

这个 MP3 播放器的想法有很多美妙之处。这还不是一个实际的产品，但这里是基本的。首先，就像以前学校的磁带一样，你需要选择磁带长度:45 分钟、60 分钟或 90 分钟。数字越大，基于 flash 的播放器可以容纳的歌曲就越多。

![nvrd_8](img/bd92acc48f80b206cfcc7486273862e7.png "nvrd_8")

该设备有一个翻转式 USB 连接器，用于传输歌曲，还有一个沿着磁带底部的滑动按钮，对应于播放和停止。

换曲目？你并不真的改变整个音轨，但是你可以在左边的磁带卷轴里转动手指来快进和倒回当前的歌曲(见上图)。如果你喜欢复古的东西，这就是它。就功能而言，这东西甚至可以追溯到 CD 时代。卷轴之间的一个小显示屏显示了经过的跟踪时间，这应该会有所帮助。

![nvrd_12](img/015ae4e799f28a4327b9049245f8edfa.png "nvrd_12")

当今一个受欢迎的特征是动力充电。用手指旋转右侧卷带轴，给电池充电。谁知道充满电需要多长时间，但当你给所有东西充电时，状态灯会从红色变为橙色再变为绿色。

![nvrd_9](img/c113f3706f9975f5ab210b3ce1e0e615.png "nvrd_9")

播放器坐在一个保护性的塑料磁带盒里，就像过去的磁带盒一样。这种情况下有一个插槽，所以你可以使用你的耳机，同时保护播放器。

再说一次，这还不是一个真正的产品，它的功能肯定不如一个真正的 MP3 播放器，但对于任何伴随着磁带长大的人来说，它可能会带回一些美好的回忆。如果他们能把它缩小到真正的磁带大小(它看起来有点大)并让你把它放入真正的磁带播放器中，就像那些已经可用的解决方案一样，那将会很酷。但是这可能不是一个巨大的优先事项，因为磁带走带设备已经慢慢地走上了水牛的道路。

这个概念来自设计师 Stefano Pertegato、Massimiliano Rampoldi、Eloisa Tolu、Francesco Schiraldi 和 Giovanni Mendini。
