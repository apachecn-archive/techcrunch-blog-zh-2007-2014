# HTC 对苹果的投诉与你如何使用手指无关

当 HTC 今天早些时候宣布，它正在以专利侵权为由反诉苹果公司，作为对苹果公司早些时候在 T2 提起的专利诉讼的回应，它没有提到它正在使用哪些专利进行反击。好了，现在我们有了正式的投诉(嵌入在下面)>，尽管一个宣传视频声称 HTC [在 2007 年改变了我们使用手指的方式](https://web.archive.org/web/20221005130759/https://beta.techcrunch.com/2010/05/12/htcs-defense-fingers/)，但它的投诉与多点触摸手势无关。

它挖出来反击苹果的五项专利比这无聊多了。它们涵盖了在手机上访问电话簿的方法和电源管理。这五项专利是:

专利号 [6，999，800](https://web.archive.org/web/20221005130759/http://www.google.com/patents/about?id=3kx4AAAAEBAJ&dq=6,999,800) “智能手机电源管理方法”
专利号 [5，541，988](https://web.archive.org/web/20221005130759/http://www.google.com/patents/about?id=xz8EAAAAEBAJ&dq=5,541,988) “具有电话簿存储器个性化页面组织的电话拨号器”
专利号 [6，058，183](https://web.archive.org/web/20221005130759/http://www.google.com/patents/about?id=xz8EAAAAEBAJ&dq=6,058,183) “具有电话簿存储器个性化页面组织的电话拨号器”【奇怪的是与上述名称和发明人相同】
专利号 [6](https://web.archive.org/web/20221005130759/http://www.google.com/patents/about?id=rEcIAAAAEBAJ&dq=6,320,957)

最后一个专利，第 7，716，505 号，昨天刚发的。
