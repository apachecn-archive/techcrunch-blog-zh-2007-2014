# Fring 将 Last.fm 集成到其移动 VoIP 客户端中

移动网络电话和即时通讯服务 [fring](https://web.archive.org/web/20221007161311/http://www.fring.com/) 是目前最受欢迎的移动通讯服务之一，它正在试验将一系列第三方应用移植到 fring 客户端，并将在今年宣布推出一系列应用。今天，fring 推出了一个定制的移动优化 [Last.fm](https://web.archive.org/web/20221007161311/http://last.fm/) 音乐电台插件，为 fring 客户端带来了大量的音乐推荐服务，但也集成了后者的社交和通信功能。

Fring 用户(也称为 fringsters)现在可以启动流媒体音乐广播频道，包括他们自己的 Last.fm 库，标记收藏夹，禁止不喜欢的歌曲，查看专辑插图，基本上可以将他们的 Last.fm 音乐帐户移动化。除了将这些功能植入客户端之外，fringsters 还可以借助 Last.fm 插件中嵌入的朋友列表，选择实时查看他们的联系人当前正在收听的内容。

![](img/e877d459e7a7d537b936453f02c0d67f.png)就像 Last.fm 自己的 Android 应用程序一样，该应用程序可以在后台运行，用户可以在使用其他 fring 功能的同时收听流媒体音乐频道。

第三方可以使用 fring 的 API 在 fring 的基础上构建附加组件，你可以在这里找到更多信息。
