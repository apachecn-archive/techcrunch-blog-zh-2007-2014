# NEC 展示另外三款 3D 桌面电脑 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/09/13/ne-shows-another-three-3d-desktop-pcs/>

[![](img/6702b2eda1e05be800237d14547aa64f.png "VW970_CS")](https://web.archive.org/web/20221208063413/http://www.crunchgear.com/2010/09/13/ne-shows-another-three-3d-desktop-pcs/vw970_cs/)

在日本的个人电脑制造商中，NEC 似乎在 3D T4 上下了特别大的赌注。继今年夏天早些时候公司在日本推出 3D 个人电脑之后，NEC 今天[宣布](https://web.archive.org/web/20221208063413/http://pc.watch.impress.co.jp/docs/news/20100913_392629.html)【JP】另外三款面向日本市场的 3D 桌面个人电脑。这里是所有的细节。

所有这些 Windows 7 机器将在本月内登陆日本商店。

**NEC Valuestar VW 970/CS
**

本文顶部的图片显示，这台电脑配备了酷睿 i5-460M (2.53GHz) CPU、4GB 内存、1.5TB 硬盘、GeForce GT 330M、3D 蓝光光盘驱动器、23 英寸 3D 全高清屏幕、两个电视调谐器、一副 3D 眼镜和一个 HDMI 端口。

VW970/CS 售价 3100 美元。

[![](img/9d26dc0d131b0d9c2d8f0650e7129240.png "VN790_CS")](https://web.archive.org/web/20221208063413/http://www.crunchgear.com/2010/09/13/ne-shows-another-three-3d-desktop-pcs/vn790_cs/)

**NEC Valuestar VN790/CS**

该型号具有以下功能:酷睿 i5-460M CPU(与上述型号相同)、4GB 内存、1TB 硬盘 1TB、非 3D 蓝光驱动器、分辨率为 1600×900 的 20 英寸 3D 屏幕、一副 3D 眼镜和一个电视调谐器。

VN790/CS 的价格为 2，600 美元。

[![](img/135c629c42d4c8abec9a38596d2efa52.png "VL750_CS")](https://web.archive.org/web/20221208063413/http://www.crunchgear.com/2010/09/13/ne-shows-another-three-3d-desktop-pcs/vl750_cs/)

NEC Valuestar VL750/CS

VL750/CS 配备了酷睿 i7-870 (2.93GHz) CPU、8GB RAM、1TB HDD 1TB、GeForce GT 330 显卡、非 3D 蓝光光盘驱动器、23 英寸全高清 120Hz 3D 屏幕、HDMI 端口、三个 USB 3.0 端口和八个 USB 2.0 端口。

它将花费 3400 美元。
