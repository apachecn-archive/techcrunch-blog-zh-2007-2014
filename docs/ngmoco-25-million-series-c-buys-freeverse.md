# iPhone 游戏公司 ngmoco 融资 2500 万美元，收购 Freeverse

![](img/ccfe50d6fe3f2972538dc72698e10ce1.png)

iPhone 游戏创业公司 [ngmoco](https://web.archive.org/web/20230318144003/http://blog.ngmoco.com/) 的首席执行官尼尔·杨希望“积累足够的规模”来加速“脱离群体”他刚刚筹集了 2500 万美元的 C 轮融资，并收购了另一家顶级 iPhone 游戏开发商 [Freeverse](https://web.archive.org/web/20230318144003/http://www.freeverse.com/) ，以帮助 ngmoco 继续前进。这轮融资由机构风险投资伙伴牵头，现有股东 Kleiner Perkins、Norwest Venture Parters 和 Maples Investments 也参与了融资。新一轮筹资总额达到 4060 万美元。

这家初创公司不断推出新的 iPhone 游戏，这些游戏已经被下载了数百万次。仅两款游戏——*触摸宠物*和*消灭*——就被安装了 900 万次，每天都有数十万人玩。去年， [ngmoco 通过其 Plus+社交游戏网络转向免费游戏，通过应用内购买虚拟商品](https://web.archive.org/web/20230318144003/https://techcrunch.com/2009/11/04/at-the-top-of-his-game-and-the-app-charts-ngmoco-bets-its-future-on-in-app-purchases/)。

Freeverse 的所有游戏，如 Skee-Ball、Flick Fishing、Flick Bowling 和 Moto Chaser，都是付费应用程序。Skee-Ball 是 App Store 排名第四的付费应用。但 Young 计划将 Freeverse 游戏转向免费模式，然后通过应用内购买赚钱。Young 还从另一家游戏开发商手中收购了 0.99 美元的游戏 Charadium，并计划将其转变为免费游戏。

这种模式在 ngmoco 运作良好。Young 表示，TouchPets 上周末是“收入最高的一天”，其接下来的两款游戏 We Rule 和 GodFinger(一款“社交上帝游戏”)是专门为免费游戏模式设计的。“在任何一天，你都有大约 2%的观众向你付费，”他说。他预计 ngmoco 今年将推出约 20 款新游戏，新收购的 Freeverse 团队也将接近这一数字。

除了推出一波又一波新的 iPhone 游戏，Young 还计划为 ngmoco 的 Plus+系统开放一个 SDK，以便其他开发者可以更容易地利用它。Plus+系统可以管理虚拟物品支付、玩家之间的消息传递以及 iPhone 游戏的其他社交方面。
