# 易贝 PayPal 收购移动支付初创公司 Fig Card

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/04/28/ebays-paypal-buys-mobile-payments-startup-fig-card/amp/>

 <amp-img src="https://web.archive.org/web/20230404185836im_/https://techcrunch.com/wp-content/uploads/2011/04/about-fig-card.jpg" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 在其[第二次收购](https://web.archive.org/web/20230404185836/https://techcrunch.com/2011/04/20/ebay-acquires-location-based-media-and-advertising-company-where/)的两周内，易贝的 PayPal 部门已经收购了移动支付初创公司 [FigCard。](https://web.archive.org/web/20230404185836/http://figcard.com/)收购条款在 PayPal [博客上公布，](https://web.archive.org/web/20230404185836/https://www.thepaypalblog.com/2011/04/welcome-max-metral-and-hasty-granbery-to-paypal/)没有披露。

总部位于波士顿的 Fig Card 允许商家通过使用插入收银机或销售点终端的简单 USB 设备在商店接受移动支付。消费者需要的只是智能手机上的 Fig 应用程序。与贝宝的联系是，当消费者设置他们的支付信息时，他们可以添加贝宝作为支付选项。你可以在下面的视频中看到 Fig Card 的技术演示

这既是一项技术购买，也是一项人才收购。创始人 Max Metral 和 Hasty Granbery(都将加入 PayPal 移动团队)都是经验丰富的技术高管。在创办 Fig 之前，Metral 是 Firefly 的联合创始人兼首席技术官，该公司被出售给了微软。他还设计了登录系统 Microsoft Passport。Metral 和 Granbery 是在 PeoplePC 上认识的，people PC 被卖给了 Earthlink。诸如此类。

显然，移动和线上到线下是易贝市场和 PayPal 业务战略的重要组成部分。该公司刚刚购买了位置和广告公司在哪里，这将被安置在贝宝。事实上，PayPal 将被整合到 Where's 移动应用中，作为其本地交易的支付机制。

看看 PayPal 如何(以及是否)将 Fig Card 整合到其产品中应该会很有趣。我们知道，支付服务正在寻求与当地商家建立联系，为他们带来一个以 PayPal 为核心的销售点系统，可以帮助他们与 NFC、Square 等新兴技术竞争。
