# 视频平台 OpenWatch 旨在支持全球公民新闻 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/23/video-platform-openwatch-aims-to-support-global-citizen-journalism/>

今天在媒体创业加速器 [Matter](https://web.archive.org/web/20221003020205/http://matter.vc/) 的 [NYC 演示日](https://web.archive.org/web/20221003020205/https://beta.techcrunch.com/2013/07/23/media-startup-accelerator-matter-comes-to-new-york-for-demo-day/)上展示的 OpenWatch ，希望成为全球公民新闻的视频平台。该公司成立于去年 11 月，目前正处于融资过程的初期，希望能尽快完成一轮种子期融资，从而启动项目。

OpenWatch 的联合创始人兼首席执行官里奇·琼斯告诉我们，OpenWatch 的目的是建立一个视频记者和调查人员的全球网络，他们可以曝光世界各地的重要问题。在新闻不自由的海外国家，OpenWatch 的吸引力尤其明显。

“我们(在美国)仍然有一个自由媒体的表象，”琼斯说。“这个项目有严重的国际影响。我真的相信，移动技术和网络技术为以前没有机会的国家创造了一个公共自由媒体的巨大机会。”

他补充说，尽管 Android 在海外的采用正在起飞，但那些没有智能手机的人可以拨打 OpenWatch 热线，并将手机用作录音机。

琼斯说，自从一个月前新版本的 OpenWatch 推出以来，成千上万的人一直在使用该网络——例如，美国国家安全局棱镜抗议活动产生了来自亚特兰大、波士顿和芝加哥的视频。

多面讲故事提供了一个人写的新闻报道中可能无法描述的视角。与此同时，开源新闻意味着事实可能会被无意歪曲，或者更糟，被劫持为宣传。

为此，OpenWatch 包括所有视频的完整原始资料，因此如果观众对故事不满意，他可以自己核实。所有事情，包括电话和电子邮件，都应该公布并记录在案。

OpenWatch 平台也是官方新闻机构的资源。

“也许《纽约时报》可以派人去(国外)，但《乡村之声》不行，”琼斯说。“这真的降低了做这种国际报道的成本。”

在公司正式成立之前，OpenWatch 作为一个开源项目已经存在了很多年。在那段时间里，创始人还创建了一个名为 [Cop Recorder](https://web.archive.org/web/20221003020205/http://chrisballinger.info/apps/coprecorder/) 的应用程序，该程序为人们提供了一个平台，可以秘密拍摄他们与警察的互动并上传这些视频。琼斯说，这些都被纳入了全球警察遭遇地图，反映了警察执法和公民权利滥用的区域趋势。

“我不喜欢看到媒体目前的角色是公共关系或宣传者，”琼斯说。“没有多少新闻业应该做的事情，让当局负起责任。”
