东芝-富士通IS12T:世界上第一个Windows芒果手机上升和关闭(视频)

两天前，日本东芝公司发布了IS12T，这是世界上第一款运行Windows Phone 7.5的手机。我们很快给了你第一个规格，但我们现在有一个视频显示1 Ghz高通CPU供电的设备在运行中-加上一些更详细的规格。

首先，这里是主要规格(更详细)，正如日本IS12T的供应商宣布的【JP】，日本当地移动运营商 KDDI :

  • 3.7英寸显示屏分辨率为800×480
  • 1320万像素摄像头采用CMOS传感器，由Milbeaut Mobile engine驱动，可拍摄分辨率为1280×720的视频
  • 防水防震体
  • microUSB端口
  • 32GB内部内存(无额外卡槽)
  • IEEE 802.11b/g/n无线网络(无网络共享)
  • 蓝牙2.1+EDR
  • DLNA支持
  • Zune品牌的多媒体功能
  • Internet Explorer 9作为浏览器
  • Xbox LIVE连接
  • Windows Live SkyDrive访问
  • 待机时电池续航时间:280小时，连续通话时间:400分钟
  • 尺寸:118×59×10.6 ~ 13.3毫米，重量:113克
  • 已确认:有三种颜色(黄色、黑色、粉色)
