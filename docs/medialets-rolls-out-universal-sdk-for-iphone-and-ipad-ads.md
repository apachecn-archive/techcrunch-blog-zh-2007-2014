# Medialets 为 iPhone 和 iPad 广告推出通用 SDK

广告和分析初创公司 Medialets 在两年前推出 App Store 时，在 iPhone 上下了很大的赌注，并从那以后一直在苹果浪潮中发展。该公司是最广泛使用的[广告平台之一，](https://web.archive.org/web/20221206214943/https://beta.techcrunch.com/2009/04/14/medialets-beats-apple-to-a-billion-when-it-comes-to-the-iphone/)[拥有](https://web.archive.org/web/20221206214943/http://www.medialets.com/blog/2010/02/18/the-associated-press-npr-cnn-variety-and-others-integrate-medialets-into-their-mobile-apps-2/) [NPR、](https://web.archive.org/web/20221206214943/https://beta.techcrunch.com/2010/01/13/iphone-music-video-ad/) CNN 和福克斯的客户。今天，广告平台[宣布](https://web.archive.org/web/20221206214943/http://www.medialets.com/blog/2010/03/30/announcing-medialets-universal-sdk-for-iphone-and-ipad-a-new-era-in-app-advertising/)一个适用于 iPhone 和 iPad 的通用 SDK，允许出版商和开发者在广告网络和第一方广告服务器之间无缝切换。

用于 iPhone 和 iPad 的 Medialets 通用 SDK 将允许出版商修改和更改来自各种网络的广告标签，而不必更新他们的应用程序。新的 SDK 可与任何第一或第三方广告服务器、网络或中介合作，包括 [AdMob](https://web.archive.org/web/20221206214943/http://www.admob.com/) 和 [Quattro Wireless](https://web.archive.org/web/20221206214943/http://www.quattrowireless.com/) 标签。

Medialets 承诺在移动电话上提供丰富的媒体和显示广告，从而与用户进行更深入的互动。事实上，参与度是广告平台在决定其广告活动成功与否时关注的主要衡量标准。Medialets 表示，其应用参与度显示，每个应用使用会话的平均使用时间为 3.5 至 7 分钟。但开发商和广告商可能更看重点击率，Medialets 声称点击率在 1%到 8%之间。但考虑到包括 Greystripe 和 AdMob 在内的其他富媒体移动广告平台的份额在 1%到 2%之间，8%似乎有点高。

对于那些不想浪费时间在不同网络的 SDK 之间切换的开发人员来说，通用 SDK 绝对是一个有吸引力的特性( [Mobclix 允许你](https://web.archive.org/web/20221206214943/https://beta.techcrunch.com/2009/08/03/mobclix-takes-on-admob-by-roping-together-20-mobile-ad-networks-into-an-exchange/)在它的平台内做这件事)。我想其他移动广告网络也会在适当的时候开发他们自己的通用 SDK。

去年春天，Medialets 已经筹集了[400 万美元](https://web.archive.org/web/20221206214943/https://beta.techcrunch.com/2009/05/12/medialets-big-bet-on-the-iphone-pays-off-with-a-4-million-round/)的首轮融资，并计划在下个月推出一款适用于 Android 的通用 SDK。
