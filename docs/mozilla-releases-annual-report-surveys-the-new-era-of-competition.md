# Mozilla 发布年度报告，调查竞争的新时代

广受欢迎的 Firefox 网络浏览器的制造商 Mozilla 刚刚发布了年度 Mozilla 状态报告，这是一份基于网络的交互式报告，概述了该组织的进展，以及它对未来几年的展望。它还包括一个关键数据:Mozilla 2010 年的总收入为 1.23 亿美元，比 2009 年增长了约 18%。Mozilla 通过搜索合作伙伴关系获得大部分收入(谷歌是最大的合同，但它也与必应、雅虎和其他提供商有交易)。

Mozilla 表示，谷歌的合作伙伴关系将于 11 月续签，它“非常有信心，在可预见的未来，搜索合作伙伴关系仍将是 Mozilla 的稳定收入来源”。

这份文件没有太多惊喜，但它提醒人们，Mozilla——长期以来主要与桌面版 Firefox 相关联——正在为新项目投入更多努力。也就是说，它更加强调移动，同时也在寻求解决网络上的各种挑战，比如创建一个分散的网络身份平台，并改善网络应用的功能和发现。它还希望保护用户隐私，部分是通过“不要跟踪”倡议。

该报告隐晦地提到了谷歌 Chrome 浏览器的增长，该浏览器上市仅三年，市场份额就将超过 Firefox。当然，谷歌长期以来一直是网络的倡导者，但 Mozilla 肯定会指出，在主要浏览器供应商中，只有它“完全是为了整个网络的利益而组织的”，而不是为了市场份额和利润。

一些重要的段落:

> 今天，我们比以往任何时候都更多地生活在网上。这给了 Mozilla 在生活的更多领域建立自由和用户主权的机会。与此同时，移动技术的进步将我们推向几个平行的世界，每个世界都由不同的大型商业实体控制。
> 
> 我们以前见过这种威胁。Mozilla 诞生于这种环境，是对充满希望和潜力的在线世界的一种反应——但随后受到了少数几家限制消费者选择、互操作性和网络整体健康的大公司的威胁。我们当时共同面对并赢得了挑战，在这个新时代，我们需要再次面对挑战。

…

> 我们还开始将 Android 作为第一层平台，与我们的桌面平台一起使用。这一重要转变有助于 Mozilla 将使命扩展到网络的下一个阶段。在这种新的环境下，Firefox 不仅仅是你桌面上的一个浏览器，而是发展成为一个可信的环境，在这个环境中，用户可以对自己的体验进行一致的核心控制，而不管使用什么设备。

…

> 我们目前有几个重要的业务合作伙伴，并正在积极探索搜索合作机会和其他潜在的收入机会。我们将继续开发伟大的产品，帮助人们享受互联网的丰富多彩，我们相信这将使我们找到合适的收入来源。
