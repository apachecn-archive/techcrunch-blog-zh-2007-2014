# 红杉资本支持的 Piazza 利用 iOS 和 Android 应用为学生提供移动问答平台

[Piazza](https://web.archive.org/web/20230203093643/http://piazza.com/) ，一个 [Q &学生和教师的平台，](https://web.archive.org/web/20230203093643/https://techcrunch.com/2010/02/05/piazzza-college-questions-answers/)正在推出 iOS 和 Android 移动应用，允许学生在旅途中分享想法和获得答案。

Piazza 的平台帮助同学们以一种介于维基和论坛之间的形式分享他们的问题和答案。每个班级都有自己的问答中心，如果学生也渴望找到答案，他们可以将任何问题加入书签。多个学生可以在维基风格中贡献每个答案，但有一个版本历史显示每个学生写了什么。

学生和教授都可以为他们的班级创建广场中心。教师的答案与学生的答案是分开的，以便于查找。教授们还可以查看哪些问题被大多数学生收藏，以判断他们应该在课堂上更好地解释哪些主题。

许多教育机构的学生已经开始使用 Piazza，包括斯坦福大学、佐治亚理工学院、伯克利分校、麻省理工学院、康奈尔大学、哈佛大学、哥伦比亚大学、伊利诺伊大学、密歇根大学、普渡大学、弗吉尼亚理工大学、滑铁卢大学、不列颠哥伦比亚大学、普林斯顿大学、德克萨斯大学和中佛罗里达大学。

Piazza 的移动应用程序包括许多与 web 应用程序相同的功能，允许用户阅读、发布、搜索和编辑问题。这些应用程序还提供了一个问题源，让学生和教师可以直接打开他们尚未阅读的问题或答案。

该公司还发布了关于网站参与度的新指标。用户目前每周做出超过 40，000 个回答和编辑，同时查看超过一百万个问题。

Piazza 得到了红杉资本、Kapor Capital、Felicis Ventures 和 SV Angel 的支持。
