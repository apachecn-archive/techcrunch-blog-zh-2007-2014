# 谷歌时代精神 2010: iPhone 击败安卓，Twitter 击败脸书，Chatroulette 击败所有

![](img/897615ba6cb74da9d60d62b5a436066f.png "c")

现在快到年底了，你知道这意味着什么:年终清单。苹果刚刚[发布了](https://web.archive.org/web/20221007063034/https://beta.techcrunch.com/2010/12/09/top-iphone-ipad-apps-2010/)和[一批](https://web.archive.org/web/20221007063034/https://beta.techcrunch.com/2010/12/09/apple-top-apps-2010/)，现在谷歌也发布了。像每年一样，谷歌公布了他们的“[2010](https://web.archive.org/web/20221007063034/http://www.google.com/intl/en/press/zeitgeist2010/)时代精神”。他们又有了一些漂亮的交互式数据可视化——现在他们在 HTML5 中。

谷歌在关于此事的[帖子](https://web.archive.org/web/20221007063034/http://googleblog.blogspot.com/2010/12/zeitgeist-2010-how-world-searched.html)中挑出了奥运会、世界杯、地震、石油泄漏和 Lady Gaga 等。但是他们的图表主要集中在增长最快的查询上。值得注意的是，在热门查询中，最热门的 iPhone (iPhone 4)击败了最热门的 Android 手机(Evo 4G)，Twitter 击败了脸书，Chatroulette 击败了所有人。事实上，这项服务(似乎没有人再谈论它了)远远领先于除 iPad 之外的所有其他快速增长的查询，iPad 紧随其后。

再次值得注意的是，这些列表只是针对本年度*上升最快的查询*。举例来说，虽然 Twitter 在这方面打败了脸书，但脸书在总体查询方面远远领先于 Twitter。
