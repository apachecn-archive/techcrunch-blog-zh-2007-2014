# Plaxo 推出脉冲小工具 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/10/22/plaxo-launches-pulse-widget/>

今年夏天早些时候，[推出的](https://web.archive.org/web/20230203074317/http://techcrunch.com/2007/08/03/plaxo-prepares-to-launch-pulse-will-users-trust-it/)的 [Pulse](https://web.archive.org/web/20230203074317/http://pulse.plaxo.com/) ，是 Plaxo 进军社交网络的一次尝试。在多年收集用户的联系信息和通讯录联系人后，他们采取了下一步措施，围绕所有这些数据创建了一个社交网络。

他们增加的一个重要功能是，用户可以在几十个其他社交网络上添加关于他们在做什么的信息，比如 Twitter、Flickr、脸书、MySpace、Digg、YouTube 等网站。然后，您会在 Pulse 页面上看到一个新闻源，其中包含您的朋友在做什么的信息。这很像脸书的[新闻提要](https://web.archive.org/web/20230203074317/http://techcrunch.com/2006/09/05/new-facebook-redesign-more-than-just-aesthetics/)，除了它是基于第三方数据。

今晚，他们推出了一种方式，让用户通过一个小工具获取个人社交网络数据，并将其包含在另一个网站上。一旦你有了一个 Pulse 账户，并输入了一些你的 feeds，你就可以点击这里的小工具。该信息的 RSS 提要也是可用的。

30 个盒子可能是[第一个在这个领域创新的创业公司](https://web.archive.org/web/20230203074317/http://30boxes.com/blog/index.php/2007/05/17/flash-buddy-cards-from-30-boxes-bling/)。本月早些时候我们报道过的一家未上市的初创公司 FriendFeed 也在做类似的事情。
