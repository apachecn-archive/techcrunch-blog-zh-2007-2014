# 脸书不是一个社交网络。不要试图在那里结交新朋友

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/09/15/facebook-isnt-a-social-network-and-dont-try-to-make-new-friends-there/>

一个名为 [PackRat](https://web.archive.org/web/20230213180302/http://www.new.facebook.com/apps/#/apps/application.php?id=2431403991) 的脸书流行应用正在酝酿一场争议，在这个应用中，用户[收集](https://web.archive.org/web/20230213180302/http://socialmediablog.tamar.com/2008/02/packrat---most.html)套带插图的卡片来换取点数和等级。该应用背后的公司称，用户每天在该应用上产生多达 500 次的页面浏览，试图寻找合适的卡片来完成收藏。

游戏的很大一部分是从朋友那里“偷”卡，所以很多用户添加其他用户为朋友，这样他们的卡就可以被获得。该应用程序的流行也导致一些用户为了玩游戏而创建脸书账户。

据申请网站上的论坛[称，其中一些账号现在被脸书禁用了。](https://web.archive.org/web/20230213180302/http://www.new.facebook.com/topic.php?app_id=2431403991&xid=packrat_general&c_url=http%253A%252F%252Fapps.new.facebook.com%252Fpackrat%252Fdiscuss&r_url=http%253A%252F%252Fapps.facebook.com%252Fpackrat%252Fdiscuss%252F&sig=bc5f8bd3530199418631078da97aa349&topic=31747)

令人好奇的是脸书发给一个被删除用户的电子邮件，其中声明脸书不是一个社交网络(它是一个“社交工具”)，也不打算建立一大群新朋友。相反，脸书意在强化“预先存在的”社会联系:

> 请注意，脸书帐户仅用于真实用途。这意味着我们希望账户主要反映“真实世界”的联系(即你的家庭、同学、同事等)。)，而不是主要的“纯互联网”联系人。正如我们主页上所说的，脸书是一个连接你和你周围人的社交工具，而不是一个“社交网站”。它旨在帮助巩固已有的社会关系，而不是建立新的大群体。如果这与你所期望的合法的脸书用法形成了直接的对比，我为任何混淆道歉。这就是网站背后的意图。
> 
> 仅用于申请目的的账户违反我们的使用条款。不幸的是，我将无法重新激活您的帐户。很抱歉给您带来不便，但这是最终决定。
> 
> 谢谢你的理解，
> 
> 劳伦
> 用户操作
> 脸书

没错，脸书已经明确表示，他们的意图是成为现实世界社交图谱的一面镜子。但不可避免的是，新的友谊也是在网站上建立的。甚至是为了玩第三方开发商制作的游戏而建立的友谊。

就连前脸书总裁肖恩·帕克(现在的股东)最近在 TechCrunch50 上也说，他在脸书的朋友远多于现实世界的朋友。

脸书在这里真正的信息可能是“请不要只是为了玩这个游戏而做假账”，但这不是他们所说的。我已经发邮件给他们要求澄清。

**更新:**脸书回复了我发的邮件。一位发言人说*“简单来说，脸书的用户不能拥有一个以上的账户，为了玩这个游戏而创建另一个账户违反了我们的使用条款。我们认识到并欣赏每个人基于自己的兴趣和偏好使用脸书，并很高兴看到人们在脸书结识新朋友。为了确保用户在网站上感觉舒适，不会受到不请自来的联系的困扰，我们鼓励用户添加反映他们现实世界联系的人，并创建可信的网络。”*
