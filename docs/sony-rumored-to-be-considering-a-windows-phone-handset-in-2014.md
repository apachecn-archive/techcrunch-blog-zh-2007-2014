# 有传言称，索尼正在考虑在 2014 年推出 Windows Phone 手机

据消息称，索尼可能会发布一款 Windows Phone 设备，使其移动设备阵容多样化，支持微软，并可能表明硬件公司越来越希望对抗 Android 主导的未来。

索尼似乎对这个项目非常感兴趣。The Verge [今晨](https://web.archive.org/web/20221209041153/http://www.theverge.com/2013/12/31/5260828/sony-may-release-windows-phone-2014)写道，尽管微软计划收购诺基亚的手机业务，该公司仍继续该项目。尽管 Windows Phone 平台对微软自己的硬件有坚定的本土优势倾斜，但它愿意考虑开发一款 Windows Phone 设备，这是一个迹象。

对于索尼来说，此举将使其移动产品线多样化，不再局限于 Android，Android 平台现在通常与三星硬件相关联。具有讽刺意味的是，诺基亚是事实上的 Windows Phone OEM 厂商，因此索尼将进入第二领域，成为第二大玩家。

索尼潜在进入者中的赢家是微软，即使 Vaio 品牌的 Windows Phone 手机的发布可能会减缓其即将拥有的 Lumia 手机的销售增长。微软将收取单位费用，或许享受更快的整体平台销售增长，当然，Windows Phone 和 Xbox 产品线之间一直存在隐含的联系。据我估计，我们还没有看到确凿的证据表明一个会导致更多地使用另一个，但共享的 Xbox 平台体验肯定会对消费者活动产生一些影响。

因此，索尼制造 Windows Phone 会对 Xbox 产生一些积极影响。这可能会以牺牲 Playstation 的势头为代价。

根据[最新的公开数据](https://web.archive.org/web/20221209041153/https://beta.techcrunch.com/2013/12/30/nokia-closes-out-2013-with-92-of-the-windows-phone-market/)，诺基亚在 Windows Phone 硬件生态系统中控制的使用份额现在超过了 92%。这对微软来说是危险的，因为将你的移动平台押在单一设备上可能会导致平台风险(糟糕的硬件更新周期可能会减缓一年的增长，等等。)，意味着索尼加入 Windows Phone 骨干可以更好地停泊 Windows Phone。

当 Windows Phone 推出时，它与戴尔等各种 OEM 合作伙伴一起推出。已经进行了筛选。如果微软能够扭转这一趋势，它将证明它在 2013 年取得的进步并非侥幸。
