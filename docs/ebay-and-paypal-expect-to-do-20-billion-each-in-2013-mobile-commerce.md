# 易贝和 PayPal 预计在 2013 年各自实现 200 亿美元的移动商务收入

易贝[今天报告了](https://web.archive.org/web/20221209173839/https://beta.techcrunch.com/2013/01/16/thanks-to-growth-in-mobile-and-marketplaces-ebays-q4-revenue-up-18-percent-to-4b-net-income-up-22-percent/)强劲的收益，这要归功于移动应用的增长以及市场业务的改善。该公司还发布了今年的移动商务数据，超出了最初的估计。

2013 年，易贝预计每笔交易将超过 200 亿美元。

PayPal 和易贝都预计今年的销售额/支付额将分别达到 100 亿美元。然而，易贝移动以 130 亿美元的交易量结束了这一年，比上一年增加了一倍多，PayPal 移动处理了近 140 亿美元的支付量，比上一年增加了两倍多(确切地说是 250%)。

易贝表示，市场移动销售的增长是由于易贝移动应用程序的采用率增加，以及新功能和新产品的参与度增加。事实上，易贝的移动应用套件在 2012 年吸引了超过 400 万新客户。

约翰·多纳霍在易贝的财报电话会议上表示，2012 年，贝宝的移动支付量占总支付量的 10%。
