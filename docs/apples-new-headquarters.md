# 苹果新宇宙飞船总部的壮丽图像

[正如我们在 6 月份](https://web.archive.org/web/20230308065421/https://techcrunch.com/2011/06/07/steve-jobs-cupertino/)所提到的，苹果正在加州 Cuptertino 建造有史以来最酷的办公楼。这座大楼对公司如此重要，以至于几个月前，史蒂夫·乔布斯(Steve Jobs)亲自参加了库比蒂诺市议会的一次会议，以敲定这笔交易。他们完全听命于他。在这一点上，其余的都只是形式。“我们不可能说不，”库比蒂诺市长 Gilbert Wong [会后说。](https://web.archive.org/web/20230308065421/https://techcrunch.com/2011/06/08/cupertino-to-jobs-there-is-no-chance-were-saying-no-to-the-apple-spaceship-campus/)

今天，库比蒂诺[在城市网站上发布了关于该项目的更详细的计划。该项目被称为该市的“苹果校园 2”，将建在 280 号高速公路附近的 175 英亩土地上。总建筑面积约为 280 万平方英尺，将设有一个 1000 座的礼堂(除了苹果最大的活动之外，所有活动都很完美)、一个健身中心、30 万平方英尺的研究设施、一个发电厂和地下停车场。但所有这些听起来都很无聊，你真正需要看到的是它看起来的样子:一个巨大的飞碟状环路。幸运的是，Cupertino 包括了苹果的提案文件，其中包括许多该项目的新渲染图。](https://web.archive.org/web/20230308065421/http://www.cupertino.org/index.aspx?page=1107)

库比蒂诺说，他们仍然必须审查该项目的环境影响，并在实际批准之前举行更多的公众听证会。但是，是的，再一次，这是不可能的。苹果的计划是明年破土动工，2015 年建成新总部。它将在一个巨大的建筑中容纳超过 12，000 名员工。
