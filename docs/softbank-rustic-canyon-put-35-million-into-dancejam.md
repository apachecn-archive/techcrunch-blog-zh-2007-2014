# 软银乡村峡谷向 DanceJam 投资 350 万美元

[http://v1.tinypic.com/player.swf?file=6b9gyls&s = 1](https://web.archive.org/web/20230204124712/http://v1.tinypic.com/player.swf?file=6b9gyls&s=1)

将新创公司 [DanceJam](https://web.archive.org/web/20230204124712/http://www.dancejam.com/) ，一个舞蹈视频的[YouTube](https://web.archive.org/web/20230204124712/https://techcrunch.com/2007/11/06/mc-hammers-dancejam-opens-its-doors-to-a-select-few/)，再投资 350 万美元，使他们的总融资额达到[，450 万美元](https://web.archive.org/web/20230204124712/http://www.crunchbase.com/company/dancejam)。他们已经完成了软银资本、乡村峡谷伙伴和一些新的天使投资人的 B 轮融资。**注意，我是这家创业公司的投资人。**

该公司由 M.C. Hammer 和 Flock 的 Geoffrey Arone 和 Anthony Young 创建，目前处于私人测试阶段，但应该很快就会推出。用户可以上传自己跳舞的短片；其他用户可以对这些剪辑进行评级。总票数将用于显示全国和世界各地的地理和舞蹈类型的顶级舞者。

除了资金和即将到来的公开发布，该公司表示，他们已经签署了一些关键的业务发展协议，以协助分销和营销。

《连线》杂志在这里写了一篇很好的公司概述[。](https://web.archive.org/web/20230204124712/http://www.wired.com/entertainment/theweb/commentary/listeningpost/2007/11/listeningpost_1112)
