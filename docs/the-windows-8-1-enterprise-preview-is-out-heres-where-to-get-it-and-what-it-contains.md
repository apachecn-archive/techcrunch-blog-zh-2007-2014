# Windows 8.1 企业版预览版已经发布，这里是获取它的位置和它包含的内容

今天，微软发布了 Windows 8.1 企业预览版，这是即将到来的 Windows 8 更新的早期版本，旨在为需要广泛控制功能来确保机器、软件和数据安全的大公司提供服务。

如果你是那种会使用这种东西的人，你可以在这里找到代码。

Windows 8.1 不会延长 Windows 8 操作系统的一般寿命。相反，微软今天在一篇博客中声明，对寡妇 8.1 的支持将于 2023 年 1 月 10 日结束。也就是说，一旦 Windows 8.1 发布，Windows 8 用户将只有两年的时间转换到 Windows 8.1。未能做到这一点的用户将不会获得“Windows 8 生命周期支持”

随着企业预览版的发布，微软还详细介绍了它包含的许多新功能，包括侧加载，这将允许公司将应用程序侧加载到“加入域”的机器上，以及 DirectAccess，这将允许用户在不使用另一个虚拟专用网络的情况下“远程访问企业网络内的资源”。

此外，在更新的操作系统中，如果公司认为更安全，则限制商店访问，并提供一种工具，允许 Windows 8.1 设备安全地访问公司的安全数据。

Windows 8 在发布时并不是一个以企业为中心的操作系统。相反，在我看来，它非常关注消费者和平板电脑。然而，Windows 7 更新周期最终会结束，微软需要创建一个新的核心、稳定、支持企业的操作系统。Windows 8 企业版 8.1 中即将推出的一系列功能将大大有助于其成熟，这是它在发布时没有的。

今年我们将看到 Windows 8.1。它对下滑的个人电脑销售会有什么影响仍不确定。
