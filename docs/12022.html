吃得更聪明:Foodie.fm首次推出个性化杂货购物平台

本周在巴黎举行的 Le Web 大会上， Foodie.fm 将正式推出其个性化的杂货社交购物平台。这家芬兰初创公司的基本目标是更好地告知购物者他们购买的食品，并帮助零售商直接沟通他们出售的食品。

Foodie.fm的核心是一个推荐系统，该公司表示，该系统依赖于正在申请专利的技术，可以从用户的饮食和购买习惯中学习，并建议符合他或她的“口味特征”的食谱和杂货。该系统考虑了个人偏好——比如食物过敏或不耐受、预算限制和偏好。
