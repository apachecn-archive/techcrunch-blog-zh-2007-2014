# IBM:众所周知的广告的终结

本周早些时候，IBM 发布了一份有趣的新报告，预测我们所知的广告业将在 5 年内结束。

引用 IBM 的话

> 随着预算迅速转向新的互动形式，传统广告公司面临收入大幅下降的风险，预计互动形式的增长速度将是传统广告的近 5 倍。为了在这种新的现实中生存，广播公司必须改变他们的大众观众心态，以迎合利基消费者群体，分销商需要为一系列多媒体设备提供有针对性的互动广告。广告公司必须进行创造性的实验，成为消费者洞察的经纪人，并在爆炸式的选择中引导广告资金的分配。所有玩家都必须适应这样一个世界，即广告库存越来越多地在公开交易所而非传统渠道进行买卖…
> 
> 该报告观察到四种改变驱动力打破了广告业的力量平衡:“注意力控制、创造力、衡量和广告库存。”消费者的注意力已经转移，个人上网时间与看电视时间不相上下。消费者已经厌倦了打断广告，并且越来越多地控制他们如何交互、过滤、分发和消费他们的内容以及相关的广告消息。IBM 的调查结果表明，一半的 DVR 用户在重播时观看了 50%或更多的节目，传统的视频广告无法在网上播放:40%的受访者认为在线视频片段中的广告比任何其他形式的广告都更令人讨厌。业余爱好者和半专业人士正越来越多地创作低成本广告内容，这有可能绕过创意机构，而出版商和广播公司正在拓宽自己的创意角色。广告客户要求在广告平台上进行问责和更具体的个人消费者测量。自助广告交易所正在吸引曾经只通过专有渠道或交易销售的收入。
