# 谷歌以超过 5 亿美元收购人工智能初创公司 DeepMind

谷歌将收购总部位于伦敦的人工智能公司 DeepMind。 [The Information](https://web.archive.org/web/20230329150222/https://www.theinformation.com/google-beat-facebook-for-deepmind-creates-ethics-board) 报道称，收购价格超过 5 亿美元，脸书也在去年年底就收购这家初创公司进行谈判。DeepMind 向我们证实了这笔收购，但不能透露交易条款。

谷歌最初确认的收购是[重组/编码](https://web.archive.org/web/20230329150222/http://recode.net/2014/01/26/exclusive-google-to-buy-artificial-intelligence-startup-deepmind-for-400m/?utm_source=rc_twitter)。

谷歌聘用 DeepMind 将有助于它与其他主要科技公司竞争，因为它们都试图通过专注于深度学习来获得商业优势。例如，脸书最近聘请了 NYU 大学教授 Yann le cunn[领导其新的人工智能实验室](https://web.archive.org/web/20230329150222/https://techcrunch.com/2013/12/09/facebook-artificial-intelligence-lab-lecun/)，IBM 的[沃森超级计算机](https://web.archive.org/web/20230329150222/https://techcrunch.com/2013/02/07/alchemy-api-raises-2-million-for-neural-net-analysis-tech-on-par-with-ibm-watson-google/)现在正致力于深度学习，而[雅虎最近收购了照片分析初创公司 LookFlow](https://web.archive.org/web/20230329150222/https://techcrunch.com/2013/10/23/yahoo-acquires-startup-lookflow-to-work-on-flickr-and-deep-learning/) 领导其新的深度学习小组。

DeepMind 由神经科学家、前国际象棋神童戴密斯·哈萨比斯、沙恩·莱格和穆斯塔法·苏莱曼创立。Skype 和 Kazaa 的开发者 Jaan Tallin 是投资者之一。

这是谷歌填补其人工智能专家名册的最新举措，据 Re/code 报道，此次收购据称由谷歌首席执行官拉里·佩奇(Larry Page)领导。如果 DeepMind 的三位创始人都为谷歌工作，他们将加入[发明家、企业家、作家和未来学家 Ray Kurzweil](https://web.archive.org/web/20230329150222/https://techcrunch.com/2012/12/14/ray-kurzweil-joins-google-as-engineering-director-focusing-on-machine-learning-and-language-tech/) ，后者于 2012 年被聘为专注于机器学习和语言处理的工程总监。

Kurzweil 说他想建立一个先进的搜索引擎，它可以像一个“控制论的朋友”

在本月早些时候收购 Nest 后，批评者[表达了对这家智能设备制造商将与谷歌分享多少客户数据的担忧。该公司上个月](https://web.archive.org/web/20230329150222/https://techcrunch.com/2014/01/13/nest-says-customer-data-from-devices-will-only-be-used-for-nest-products-and-services/)[对 Boston Dynamics 的收购](https://web.archive.org/web/20230329150222/https://techcrunch.com/2013/12/14/google-buys-boston-dynamics-creator-of-big-dog/)也引发了关于[为什么一家搜索公司需要一家机器人制造商](https://web.archive.org/web/20230329150222/https://techcrunch.com/2013/12/14/why-does-google-need-so-many-robots-to-jump-from-the-web-to-the-read-world/)的困惑。

谷歌看起来更好地准备减轻用户对其最新收购的担忧。根据信息来源，谷歌已经同意建立一个道德委员会，以确保 DeepMind 的人工智能技术不被滥用。

但该公司可能还必须澄清*deep mind 的人工智能技术到底是干什么的。这家[公司的网站](https://web.archive.org/web/20230329150222/http://deepmind.com/)目前只有一个登陆页面，其中有一个相对模糊的描述，称 DeepMind 是“一家前沿的人工智能公司”，为模拟、电子商务和游戏构建通用学习算法。信息显示，截至去年 12 月，这家初创公司约有 75 名员工。*

据 Re/code 报道，Founders Fund 和 Horizons Ventures 都是这家初创公司的主要投资者。根据 LinkedIn 的资料，DeepMind 成立于大约三年前。

2012 年，卡耐基梅隆大学教授拉里·乏色曼写道 “创业公司试图建立一个思考的系统。这是 AI 最初的梦想。正如 Shane [Legg]向我解释的那样，神经科学和 ML 都取得了巨大的进步，他们的目标是将这些东西融合在一起。我认为这听起来很疯狂，直到他告诉我投资了该公司的著名亿万富翁名单。”
