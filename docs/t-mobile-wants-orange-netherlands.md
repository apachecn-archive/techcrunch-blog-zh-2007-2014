# T-Mobile 想要 Orange 荷兰

T-Mobile 想收购法国电信的分公司 Orange Netherlands。法国电信自 2 月以来一直试图出售 Orange Netherlands。欧盟监管机构已经批准 T-Mobile 荷兰公司收购 Orange 荷兰公司。据《休斯顿纪事报》报道，欧盟委员会称该交易不会“严重阻碍”欧盟市场的有效竞争。

Orange 为郁金香和风车之乡的 190 万手机客户和 61.2 万互联网用户提供服务。此次收购将使 T-Mobile 成为荷兰市场上第二大移动电话提供商。
