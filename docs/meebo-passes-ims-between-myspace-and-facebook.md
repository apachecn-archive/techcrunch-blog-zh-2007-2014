# Meebo 在 MySpace 和脸书之间传递即时消息

![](img/069148ed756d5c5de42c61efa518a32f.png)

MySpace 和脸书可能永远是友敌，但现在他们有了一个新的共同朋友，可以在他们之间传递信息。基于网络的即时通讯服务 Meebo 现在支持两个社交网络的即时通讯账户。这意味着你可以在 Meebo 中输入你的脸书和 MySpace 账户 id，并在一个 IM 框中与来自这两个社交网络的朋友聊天。整合将在 Meebo.com T2 和 T4 社区即时通讯合作网站上进行。

虽然与 MySpace 的合作是正式的，但 Meebo 基本上是对脸书即时通讯的逆向工程。因此，如果脸书决定改变其 IM 协议，Meebo 集成可能会中断，直到其工程师应用创可贴。相比之下，MySpace 的集成是建立在官方 API 之上的，并且包含了 MySpace IM 的所有安全特性。

Meebo 还支持 AIM、Gtalk、MSN Messenger、Yahoo Messenger、Jabber 和 ICQ 帐户。它声称有 4000 万会员，每月发送 50 亿条信息。ComScore 报告全球每月有 740 万游客到 Meebo.com，并显示增长有些平缓。进军脸书和 MySpace 应该有助于 Meebo 继续发展。
