# 想要帮助抵消你的交通碳足迹吗？GiveO2 是一个应用程序

[give 2](https://web.archive.org/web/20230224135430/http://www.giveo2.com/)是一款计算你与交通相关的碳足迹的应用。该应用程序使用智能手机的 3G 或 GPS 连接来跟踪你的日常活动，并给出答案，非常详细地解释你正在摧毁这个世界。

到目前为止，计算你每天通勤的碳足迹充其量也是乏味的。虽然有网络工具可以让这变得非常容易，但你仍然需要在一天中持续地精确记录旅行的英里数和交通方式。我不知道谁有时间做那件事。

GiveO2 会显示你步行、骑自行车或开车的里程，然后使用一种独特的算法来计算你的碳排放量。您会以一种易于理解的方式看到这些统计数据。例如，该应用程序会告诉你，在你开车去费城的过程中，碳排放量等于 X 棵树的数量。

通过该应用程序，你还可以购买额外的碳信用来抵消你的碳排放量。“我们提议在巴塔哥尼亚种植本土树木，”联合创始人 Joaquin Dufeu 说，“不久我们将在加拿大和美国开展项目。”

与 Foursquare 非常相似，GiveO2 的用户可以因抵消碳足迹而获得积分和徽章。作为额外的激励，这些积分可以在亚马逊、GAP、锐步和匡威等零售商那里兑换折扣。

GiveO2 现已在 iOS 应用商店[和](https://web.archive.org/web/20230224135430/https://itunes.apple.com/us/app/giveo2/id604223260?l=es&ls=1&mt=8)上市，Android 版本将在未来几个月发布。
