# BilltoMobile 与 Discover 合作推出移动支付服务

继移动支付服务公司[bill to mobile](https://web.archive.org/web/20230129214504/http://www.billtomobile.com/)[推出移动网络一键式结账解决方案](https://web.archive.org/web/20230129214504/https://techcrunch.com/2012/01/18/mobile-payments-company-billtomobile-launches-one-click-checkout-for-the-mobile-web/)之后，该公司今天宣布与 Discover 合作。通过该协议，Discover 的电子商务商家现在可以在结账时提供运营商账单作为一种可能的支付选项。

作为背景，BilltoMobile 已经与美国四大运营商建立了[关系(威瑞森，AT & T，Sprint 和 T-Mobile)，允许消费者在结账时使用他们的手机号码作为支付选项。这对于那些不习惯在线输入信用卡信息的人，以及那些不想在手机小键盘上输入信息的人来说很有用。](https://web.archive.org/web/20230129214504/https://techcrunch.com/2011/09/09/billtomobile-lands-deal-with-t-mobile-now-has-direct-relationships-with-the-top-4-u-s-carriers/)

在年初推出较新的一键式系统之前，所涉及的过程稍微繁琐一些。用户必须手动输入他们的号码，有时是邮政编码，然后等待通过短信发送到他们手机上的验证码。该公司解释说，由于与运营商合作伙伴的深度整合，一键式系统现在能够在结账时自动识别用户的手机号码。

与 Discover 的新合作伙伴关系被誉为“营销协议”，旨在帮助推动运营商计费在数字商品、移动商务和普通移动支付领域的应用。

目前，BilltoMobile 通过其运营商关系为美国约 2.9 亿用户提供服务。该公司的战略投资者和大股东、韩国首尔的 Danal Co. Ltd .说，去年它已经处理了 20 亿美元(迄今为止为 40 亿美元)，去年超过 30%的交易来自购买实物商品。该公司相信，这种新的合作伙伴关系将有助于提高这些数字，特别是在数字产品和服务领域。
