# 扎克伯格:移动用户更有可能成为日常活跃用户

在首次公开募股后的首次公开露面中，脸书首席执行官马克·扎克伯格在 TechCrunch Disrupt SF 会议上发表了演讲，并接受了 TechCrunch 创始人迈克尔·阿灵顿的采访。采访谈到了 IPO 后的股票下跌，但扎克伯格首先谈到了移动平台的优势——这是当今社交网络游说的最重要的问题之一。“拥有手机的人比拥有电脑的人多得多，”扎克伯格说。“移动用户更有可能成为每日活跃用户，”他补充道。

他说，移动用户每周使用脸书超过 6 或 7 次的可能性是普通用户的两倍多。他补充道，随着新的 iOS 应用程序的推出，被消费的故事数量翻了一番。移动用户花在脸书上的时间更多，而且每次他们花在桌面上的钱也更多。尽管移动广告必须更深入地融入移动体验，但它们比脸书的右侧广告做得更好。他承认，问题当然是如何到达那里。扎克伯格还谈到了移动开发的挑战，承认脸书在 [HTML5 上长达两年的押注是其最大的错误之一](https://web.archive.org/web/20221209140949/https://beta.techcrunch.com/2012/09/11/mark-zuckerberg-our-biggest-mistake-with-mobile-was-betting-too-much-on-html5)。

至于神话中的脸书手机？扎克伯格再次驳斥了这些谣言，称这没有意义。相反，他将脸书的移动战略描述为有三个重点领域:移动网络、iOS 和 Android。他说，移动网络“对我们来说是巨大的”，但是更加有限。扎克伯格说，iOS 应用是该战略的另一个关键部分，10 个顶级 iOS 应用中有 8 个集成了脸书。即将到来的 iOS 6 集成也将在该平台上为脸书提供更多机会。在 Android 上，由于它的设计方式，你可以更深入地与操作系统集成，这也有利于脸书。

“我基本上靠我的移动设备生活，”扎克伯格说，暗示他认真对待移动设备。“想知道一些有趣的事情吗？”他告诉阿林顿，“你知道 S-1 中创始人的信，嗯，[我在我的手机上写的](https://web.archive.org/web/20221209140949/https://beta.techcrunch.com/2012/09/11/zuckerberg-wrote-all-2179-words-of-facebooks-s-1-founder-letter-on-his-phone)。”
