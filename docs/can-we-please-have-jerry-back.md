# 我们能让杰瑞回来吗？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/09/22/can-we-please-have-jerry-back/>

去年 11 月，我们都知道雅虎联合创始人[杨致远](https://web.archive.org/web/20221210002241/http://www.crunchbase.com/person/jerry-yang)将[在灾难性的 CEO 任期后卸任](https://web.archive.org/web/20221210002241/http://www.beta.techcrunch.com/2008/11/17/yang-to-step-down-from-yahoo/)。他抛弃了微软，却没有意识到后果，他也没有能力为公司描绘出另一条道路。我们并不是唯一一个呼吁解雇他的人，我们希望雅虎能找到合适的领导者来恢复他们曾经的辉煌。他们没有。

在杰里辞职和雅虎卡罗尔·巴茨时代开始之间的几个月里，硅谷有很多关于谁会领导这个曾经伟大的公司的猜测。我采访过的人认为雅虎会走两条路。第一个是试图找到伟大的产品梦想家来领导公司前进。他们的比尔·盖茨或史蒂夫·乔布斯(马克·扎克伯格有一天可能会在名单上)。有了正确的产品愿景，雅虎可以大胆地进入新的领域，重新努力创造一个持久的品牌和公司。第二条路是雇人出售整个或部分公司，并在短期内实现股东价值最大化。

很明显，雅虎选择了第二扇门，并选择了一个能够就我们这个时代下一个伟大的产品愿景进行谈判的人。你不能真的责怪他们——真正的远见者很少。在艰难的重建时期，他们不太可能想在雅虎逆流而上。

于是巴茨来了，交易开始发生。我们基本上保持沉默。任何一个新 CEO 都应该有一个蜜月期，巴茨在她进入雅虎的第一天就对记者咆哮，让他们不要发表意见。雅虎之外的人决定雅虎应该做什么，不应该做什么。这种情况必须停止。”

但是当雅虎以近乎零的价格签下其最重要的资产 T1 时，蜜月期结束了。雅虎从令人羡慕的搜索引擎第二的位置变成了另一个门户网站，尽管是一个大门户。尽管巴茨说，她坚持要一大笔预付现金。微软从未屈服，雅虎屈服了。现在，他们将看着自己的搜索市场份额缩小，就像美国在线在本世纪初将搜索业务拱手让给谷歌后所做的那样。由于所有的人都已经离开或正在离开，雅虎没有办法恢复他们曾经拥有的东西。如果政府拒绝这项搜索交易，他们会怎么样？他们几乎马上就会陷入非常严重的财务困境。我几乎怀疑微软是否真的希望这种情况发生。

巴茨下了一盘精彩的跳棋。只是她的[对手](https://web.archive.org/web/20221210002241/http://www.crunchbase.com/person/steve-ballmer)正在下棋。历史书也不会仁慈。

那么，雅虎的下一步是什么？有人谈论社交，但没人相信。产品正在被出售和关闭左右。如果没有强制出售，雅虎可能会蹒跚前行数年。但没有真正的产品大师留下来做任何引人注目和有风险的事情。雅虎的 iPhone 相当于什么？不管是什么，他们都没有勇气去尝试。

巴茨说不要担心，美国中产阶级[仍然喜欢他们](https://web.archive.org/web/20221210002241/http://www.beta.techcrunch.com/2009/09/22/yahoo-will-spend-more-than-100-million-to-try-to-connect-with-you/) : *“当你离开纽约和硅谷，每个人都喜欢雅虎。”你知道还有谁在他们开始漫长而缓慢的衰落时说过这句话吗？我的空间。但他们忽略了一点，那就是无论他们生活在哪里，新一代互联网用户都只关注脸书和谷歌。如果没有一个大胆而冒险的计划来重塑公司，就没有办法挽回那些用户。*

我真的希望雅虎把杨致远带回来吗？不，不是真的。他热爱公司，但他肯定不是雅虎需要的领导人。他担任首席执行官是一件悲伤的事情。但他确实对产品有热情，而这正是巴茨所缺乏的。坦率地说，他似乎愿意拒绝微软的每一个提议。拒绝最初的收购提议后，也许这是最好的策略。放弃搜索看起来并不具有战略意义。

我想念雅虎。他们曾经是战士，有一口支持他们的吠声。你可以整天扔 F 炸弹，但如果你没有产品肌肉来支持这种咆哮，最终它只会变老。
