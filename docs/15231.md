# 易贝 PayPal 收购 IronPearl，推动用户增长超过 1.23 亿

大约一个月前，我写了一篇关于斯坦·丘德诺夫斯基和詹姆斯·库里耶领导的一个叫做铁珍珠的秘密公司的文章。在“增长黑客”这个词出现之前，他们就是增长黑客。事实上，他们甚至不太喜欢“成长黑客”这个词。

多年来，Chudnovsky 一直为 Goodreads、Path、Wanelo、Poshmark、Lyft 和 Highlight 等公司提供如何获取用户的建议。他的秘密创业公司 [IronPearl](https://web.archive.org/web/20230404163833/http://ironpearl.com/) 正在将这些建议系统化。他和他的合作伙伴一直在开发一套优化工具，这些工具将通过一个网站或应用程序跟踪用户，并测试哪些组合最能让他们在一周或一个月后回来——尽管目标可以是你想要的任何东西。

PayPal 和 GoodReads(最近被亚马逊收购)都是早期的测试者。

显然，IronPearl 的产品和两人的专业知识非常有价值，以至于易贝的 PayPal 部门决定在推出之前直接收购该公司。我们听到的价格是“两位数百万”，尽管两家公司都没有谈论价格。

根据协议，Chudnovsky 将成为 Paypal 的增长副总裁，而 Currier 将成为增长顾问。(Currier 不会全职工作。)

PayPal 总裁大卫·马库斯说:“创建一个增长型团队是我们的基础。“PayPal 的用户几乎已经增长到了 1.25 亿，我们从未像现在这样快速增长。世界上只有极少数世界知名的成长黑客，斯坦就是其中之一。”

Chudnovsky 和 Currier 是一家名为 Tickle 的 Web 1.0 公司的负责人，该公司从事性格测验和测试。后来被求职网站 Monster 以约 1 亿美元收购。然后，他进入游戏领域，将 Wonderhill 卖给了 midcore 开发商 Kabam。然后，在创办铁珍珠之前，他进入了咨询行业。

“我决定出售的主要原因是因为我以前建立过公司，”Chudnovsky 说。“我知道那条路看起来像什么。当你看到像贝宝这样的东西时，就不一样了。这不仅仅是一个纯粹的网络，更像是一个市场。商人和人们相互交易。”

Chudnovsky 将领导一个增长团队，专注于建立超过 1.23 亿注册用户的 PayPal 客户群。
