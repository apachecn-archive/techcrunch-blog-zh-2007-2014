# BlackBerry Beta v 2.0 ticket master 现已面向 Beta Zone 成员推出

如果你是[黑莓](https://web.archive.org/web/20230203083442/https://techcrunch.com/tag/blackberry/)测试区的成员，你应该很高兴听到 [Ticketmaster](https://web.archive.org/web/20230203083442/https://techcrunch.com/tag/ticketmaster/) 刚刚发布了黑莓应用的 Ticketmaster 测试版 2.0。这意味着您可以直接从手机上选择活动，购买门票，然后上路，所有这些都可以通过 QWERTY 键盘舒适地完成。

该应用程序已经进行了彻底的改进，现在拥有一个漂亮、干净的用户界面。但真正的好处在于更新的搜索功能，它为您提供了几种不同的方式来找到合适的活动。首先，你可以按艺术家、团队或场地进行基本搜索。GPS 介入并确保你的搜索被过滤，只显示附近发生的事件，这样你就不会对 200 英里外发生的黑眼豆豆演唱会感到兴奋。还可以根据事件日期进一步筛选搜索。您可以选择“本周末”、“未来 90 天”，或者构建自己的自定义日期过滤器。

另一个很酷的增强是日历集成。有了 ticket master for BlackBerry Beta 2.0，从手机上购票的用户现在可以将日期与他们的 BlackBerry 日历同步，以保持有序。总的来说，这是黑莓应用程序世界的一个很好的未来补充，我们希望它能尽快走向大众。
