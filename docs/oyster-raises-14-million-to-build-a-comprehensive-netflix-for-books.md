# Oyster 筹集了 1400 万美元来建立一个综合性的图书网飞

据[纽约时报](https://web.archive.org/web/20230126215346/http://dealbook.nytimes.com/2014/01/14/oyster-a-start-up-for-e-reading-raises-14-million/)报道，总部位于纽约的初创公司 [Oyster](https://web.archive.org/web/20230126215346/https://techcrunch.com/2012/10/10/oyster-raises-3-million-from-founders-fund-to-finally-create-an-unlimited-subscription-service-for-books/) 刚刚宣布了新一轮融资。新投资者[高地资本合伙人](https://web.archive.org/web/20230126215346/http://www.crunchbase.com/financial-organization/highland-capital-partners)领投，现有投资者[创始人基金](https://web.archive.org/web/20230126215346/http://www.crunchbase.com/financial-organization/founders-fund)也参与其中。

提醒一下，Oyster 是电子书的限量订阅服务。每月支付 9.95 美元，你就可以在你的 iPhone 或 iPad 上访问成千上万的书籍——Android 支持应该很快就会到来。

对于媒体创业公司来说，内容交易非常重要。目前，哈珀柯林斯已经同意了，但其他四家大出版商尚未被说服(阿歇特、麦克米伦、西蒙和舒斯特以及企鹅兰登书屋)。阅读独立出版商 Workman Publishing、Perseus 和无数其他出版商的书籍是一项伟大的服务。

Oyster 的合作出版商之一 Smashwords[向其作者讲述了 Oyster 实际上是如何构建其内容交易的。如果订阅者阅读了一本书的 10%以上，出版商将获得该书零售价的 60%。](https://web.archive.org/web/20230126215346/http://www.the-digital-reader.com/2013/10/25/smashwords-reveals-author-payment-terms-oysters-ebook-subscription-service/#.UtVqP5Uasy4)

这些交易对出版商非常友好，但也表明这项服务依赖于临时订户。如果一个读者一个月阅读超过 3 或 4 本书，Oyster 将无法支付 9.95 美元的订阅费用。

Oyster 已经集成了轻量级的社区功能，以保留现有用户，并与亚马逊的[借阅库](https://web.archive.org/web/20230126215346/http://www.amazon.com/gp/feature.html?docId=1000739811)区分开来，例如浏览你的朋友正在阅读或打算阅读的内容。该公司每周还推出专题报道。

该公司的下一个重要步骤可能是让更多的主流出版商加入进来。这款产品很好用——这款应用设计精良，浏览图书馆是一种享受。许多出版商对此深信不疑。现在，让我们看看签约各大出版社是不是一个困难的壮举。
