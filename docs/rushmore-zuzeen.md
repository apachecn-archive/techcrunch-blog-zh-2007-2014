# Betaworks 联合创建的音乐社区 Rushmore 与直播视频平台 Zuzeen 合并

总部位于伦敦的音乐社区初创公司 Rushmore.fm 由[虚构的 Kin 和 Betaworks](https://web.archive.org/web/20230330013223/https://techcrunch.com/2012/05/15/betaworks-novel-way-of-funding-early-stage-ideas-a-studio-deal-with-brooklyns-fictive-kin/) 创立，最初与[一起发起了一个相当崇高而崇高的使命](https://web.archive.org/web/20230330013223/https://techcrunch.com/2013/05/14/rushmore-fm/)来修复音乐产业。它的核心前提是粉丝和艺术家之间存在脱节，通过它的平台将这两个群体拉近，就可以赚钱——这些钱可以重新投入音乐和初创公司自己的口袋。

如今，由理查德·布兰森(Richard Branson)的维珍集团(Virgin Group)前全球在线主管亚历克斯·亨特(Alex Hunter)领导的这家公司宣布合并，以支持这一主张。它与马德里的初创公司 Zuzeen(T7)联手，这是一项允许音乐家向他们的粉丝直播互动视频表演的服务。与此同时，Rushmore 披露了来自 Kibo Ventures 和 Betaworks 的 120 万美元新一轮融资。

Rushmore 表示，合并将使其能够通过让艺术家拥有他们的 Rushmore 个人资料的所有权，并通过网络或移动应用程序向他们的粉丝直播来增强其产品。

它建立在拉什莫尔和 Zuzeen 去年的合作基础上，这两家公司与古典音乐家郎朗和 telco Telefonica 合作。十多天来，40，000 名粉丝通过他的拉什莫尔频道追随郎朗，“从机场到酒店，到新闻发布会，再到座无虚席的音乐厅。”该广播还包括一个现场独家大师班。与 Telefonica 的合作可能是因为 Zuzeen 是 Wayra 的校友，Wayra 是电信公司的加速器项目。

在乐迷方面，拉什莫尔计划增加新功能，使他们能够找到新音乐，并及时了解他们感兴趣的音乐新闻、演出和发行。目前，用户“追随”他们所喜爱的艺术家，尽管许多相应的内容是基于用户的贡献(页面有一个维基式的“编辑”按钮)。

允许艺术家声明自己的个人资料的举措显然是一种让他们对内容拥有更多控制权的方式，而 Zuzeen 的合并将确保一些内容是独一无二的——不仅是它为粉丝提供的价值，还有 Rushmore 作为一个平台为艺术家提供的价值，这有助于将其与其他艺术家对粉丝的服务区分开来。
