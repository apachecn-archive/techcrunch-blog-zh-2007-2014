# DIY 电子自行车移位器

当你可以把你的自行车变成一个机器人移动的超级机器时，为什么要像其他人一样骑车呢？程序员兼制作人 [Nabil Tewolde](https://web.archive.org/web/20221207225938/http://www.nabiltewolde.com/) 向我们展示了如何创建一个按钮控制的电子拨链器系统，你可以在当地的电子商店买到它的零件。

该系统的关键是 HiTec 的一个[迷你伺服系统以及一个小的](https://web.archive.org/web/20221207225938/http://www.hitecrcd.com/products/analog/micro-mini/hs-225mg.html) [Arduino](https://web.archive.org/web/20221207225938/https://beta.techcrunch.com/tag/Arduino) 板。两个按钮控制上下移动。这有点复杂，但它是一个疯狂的黑客，可以让你忙上几天，如果不是几周的话。

你为什么想这么做？正如特沃尔德所写的:“给任何东西添加电子设备总是很酷！”
