# 使用 Xtranormal 快速制作您自己的 3D 电影剪辑

[![](img/87f446770c4660593357130415749e02.png)](https://web.archive.org/web/20221006113505/http://www.xtranormal.com/)

试试上周启动的 3D 电影制作初创公司 [Xtranormal](https://web.archive.org/web/20221006113505/http://www.xtranormal.com/) ，你很有可能在下午剩下的时间里什么也做不了。正如 Ouriel Ohayon [昨晚在法国 TechCrunch 上报道](https://web.archive.org/web/20221006113505/http://fr.beta.techcrunch.com/2008/10/15/fr-devenez-realisateur-danimations-3d-avec-xtra-normal/)的那样，该网站允许用户使用直观的拖放界面，从各种模型和背景中快速构建自己的 3D 视频剪辑。

要开始创建一个场景，你可以选择一组类似模拟人生中角色的“现实”演员，或者一组类似乐高的卡通模型。然后网站会给你一个“剧本”，你可以在里面填入你希望演员用计算机生成的声音背诵的文本。您还可以将每个角色的动作拖放到脚本中，如面部表情和手势。如果你真的想完全控制场景，你可以手动定位摄像机(如果你不想处理这个，网站会自动放置它)。

[http://www.xtranormal.com/players/jwplayer.swf](https://web.archive.org/web/20221006113505/http://www.xtranormal.com/players/jwplayer.swf)

超常规剪辑与你从皮克斯(Pixar)或梦工厂(Dreamworks)等公司看到的 3D 电影完全不同——角色看起来很傻，背景细节极少，可能的动画数量有限。但是电影对他们有一种特殊的魅力，这使得电影的质量变得无关紧要。因为你可以让他们说任何你想说的话，他们有无限的可能性。朋友们可以很容易地重现他们分享的一些经典记忆，并产生欢闹的效果，当一些类似乐高的角色表演出熟悉的电影场景时，很难不笑出来(见上文的一个很好的例子)。

还有许多其他网站，如 [Oddcast](https://web.archive.org/web/20221006113505/http://www.oddcast.com/) 、 [Gizmoz](https://web.archive.org/web/20221006113505/http://www.gizmoz.com/) 和 [Beema](https://web.archive.org/web/20221006113505/http://www.beema.com/) 让用户创建自己的 3D 角色，但其中大多数仅限于头像，而不是整个场景。
