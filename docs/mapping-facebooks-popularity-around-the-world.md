# 描绘脸书在世界各地的受欢迎程度

[![](img/b863e14d851bee14d776ff562ff3c26d.png)](https://web.archive.org/web/20230203043755/http://www.checkfacebook.com/)

我们都知道脸书正在像野草一样疯长，尤其是在国际上，但是哪些国家真的增长最快呢？TechCrunch 校友 [Nick Gonzalez](https://web.archive.org/web/20230203043755/http://www.crunchbase.com/person/nick-gonzalez) 开发了一款名为 [CheckFacebook](https://web.archive.org/web/20230203043755/http://www.checkfacebook.com/) 的新工具，旨在帮助用户做到这一点。该网站主要面向广告客户，他们可以用它来衡量在哪里开展活动，但它也以易于理解的格式提供了每个国家基本人口统计信息的有趣视图。该网站从脸书向广告商公开披露的数据中提取数据，但通常不容易获得，因为这些数据隐藏在广告注册过程中。

地图上的每个国家都根据其在脸书的受欢迎程度进行了着色，最暗的绿色代表最受欢迎的国家(毫不奇怪，美国是目前最受欢迎的国家，尽管它只占脸书总观众的 30%左右)。您还可以看到一些基本的人口统计信息，包括每个国家的性别和年龄分布。

该网站在这一点上仍然非常基础——我希望看到一些更深入的趋势分析和历史数据——但它显示出了潜力。Gonzalez 说，数据表明，随着脸书首次渗透到一个国家，有一种年轻的趋势，随着该国用户群的增长，18-24 岁和 25-34 岁的人口统计数据稳定在 33%左右。

同样值得注意的是，根据该系统，脸书的总覆盖人数约为 1 . 86 亿——低于本月早些时候公布的 2 亿。这并不十分令人惊讶，因为脸书认为这些数字是“估计”(这有点奇怪，因为他们有确切的数据)。Gonzalez 认为，这些数据可能已经过时了几周，或者脸书可能无法在某些地区销售广告库存，这将减少报告的用户总数。
