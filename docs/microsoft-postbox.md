# 微软是否正在策划一个名为 Postbox 的 Yelp 竞争对手？

![](img/8ef3f1dffa18162b20aed000accfaf18.png)

今年早些时候，彭博商业周刊[报道](https://web.archive.org/web/20230203073040/http://www.businessweek.com/magazine/content/10_07/b4166023271880.htm)称[微软](https://web.archive.org/web/20230203073040/http://www.crunchbase.com/company/microsoft)，而不仅仅是[谷歌](https://web.archive.org/web/20230203073040/http://www.crunchbase.com/company/google)，做出了收购商业目录和评论网站 [Yelp](https://web.archive.org/web/20230203073040/http://www.crunchbase.com/company/yelp) 的重大尝试。据“两位参与谈判的人士”透露，这家位于雷德蒙德的软件巨头甚至出价高达 7 亿美元收购该公司。Yelp 反而选择保持独立，不管是好是坏。

谷歌用 [Places](https://web.archive.org/web/20230203073040/http://www.google.com/local/add/analyticsSplashPage?service=lbc&pli=1) 和[火锅](https://web.archive.org/web/20230203073040/http://www.google.com/hotpot)来攻击它。现在一些信息浮出水面，表明微软*也可能*正在计划 Yelp 的竞争对手。

上周四，也就是 2010 年 12 月 16 日，微软申请了[【邮箱】](https://web.archive.org/web/20230203073040/http://www.trademarkia.com/postbox-85200057.html)的美国联邦商标注册。

起初，我认为这可能与初创公司[邮箱](https://web.archive.org/web/20230203073040/http://www.crunchbase.com/company/postbox)有关，该公司为 Windows 和 Mac OS X 提供[桌面电子邮件软件](https://web.archive.org/web/20230203073040/http://www.postbox-inc.com/)。

也许微软已经悄悄地收购了他们？

但随后，这个商标的描述和类别打动了我。

邮箱商标被归入“广告、商业和零售服务”类别。

更有趣的是，提供给 USPTO 的关于邮箱的描述是:

> “以餐馆、酒吧、夜总会、旅游景点、娱乐和体育活动、季节性和假日活动为特色的在线商业目录；并提供关于餐馆、酒吧、夜总会、旅游景点、娱乐和体育活动、季节性和假日活动的在线信息”。

如果你把两个和两个放在一起，意思是邮箱的定义(公众可以将发出的邮件放入其中的盒子)和上述商标的描述，微软似乎正在策划一个在线本地业务和活动目录，邀请用户提交有关这些业务的信息，即评论、排名等。

这显然纯粹是我们的猜测，但仔细想想，这个商标也可能涉及到 Foursquare 的竞争对手。

无论哪种方式，微软显然不会对本地商业广告的巨大潜力视而不见，商标申请表明他们正计划建立一个名为“邮箱”的东西，这将是其在该领域努力的一部分。

当然，微软可以将任何服务与它的大众搜索引擎[必应](https://web.archive.org/web/20230203073040/http://www.bing.com/)捆绑在一起。

我们应该注意到[已经有了](https://web.archive.org/web/20230203073040/http://www.trademarkia.com/postbox-77556134.html)一个“邮箱”的美国商标，由旧金山一家名为 [Mailbox Technologies，Inc.](https://web.archive.org/web/20230203073040/http://www.yelp.com/biz/mailbox-technologies-inc-san-francisco) 的初创公司注册。评论者指出，为该公司列出的地址与邮箱地址相同，而邮箱地址[实际上是](https://web.archive.org/web/20230203073040/http://www.facebook.com/photo.php?fbid=476368388667&set=a.468048103667.249617.50124613667)该商标在美国的真正所有者[。因此，同一家公司。](https://web.archive.org/web/20230203073040/http://twitter.com/#!/Postbox/status/15540517936824321)

我已经联系了微软了解更多信息，但到目前为止我还没有收到回复(美国现在才刚刚醒来，所以一旦我们得到他们的回复，我会尽快更新)。
