# Powerset 将在未来几周发布技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/04/05/powerset-will-launch-in-coming-weeks/>

该公司昨日告诉我，总部位于 T2 旧金山的 Powerset(T4)将在未来几周公开发布期待已久的测试版服务。他们正在研发一种新的搜索引擎，这种引擎将理解自然语言搜索，并与当今主导搜索的关键词匹配引擎竞争。

昨天在他们的办公室向我展示的搜索引擎的早期版本，已经提供给他们的 [Powerlabs](https://web.archive.org/web/20221006011611/http://www.labs.powerset.com/) 网站的一些用户。但在大部分时间里，它一直保持着安静。

该服务的早期版本将作为用户界面和引擎本身的展示，但是它后面没有完整的 web 索引。目前，Powerset 将只查询维基百科和 Freebase。但是当我测试服务时，我得到了一些非常类似于“啊哈”的东西这种感觉在我第一次使用谷歌的时候就已经出现了。简而言之，这是搜索领域的一次进化，也可能是革命性的进步。

我会缓和这种说法，因为该公司目前只提供两个网站的索引。尤其值得一提的是，Powerset 不必为垃圾邮件控制和其他相关问题(这也是谷歌推出时如此出色的原因)烦恼，这意味着它还不能被视为搜索领域的任何挑战者。但是任何使用它的人都可以看到这个引擎的潜在价值，当它被放在一个完整的网络索引前面的时候。

目前，该公司对引擎的特定特性保密，但我可以说，自 2007 年年中发布[截屏](https://web.archive.org/web/20221006011611/http://www.beta.techcrunch.com/2007/06/11/first-public-view-of-powerset-results/)以来，它已经有了长足的发展。

在发布前的准备工作中，Powerset 团队的一些成员发誓在产品发布前不会刮脸。他们正在一个名为 [Powerstache](https://web.archive.org/web/20221006011611/http://powerstache.com/) 的网站上记录他们的面部毛发探险，杰西卡·古恩在《洛杉矶时报》的[栏目中对此进行了报道。](https://web.archive.org/web/20221006011611/http://www.latimes.com/business/la-fi-powerstache5apr05,1,1399448.story)

谣言也在整个公司流传。一些消息来源称，Powerset 正在寻求额外的资金。该公司似乎也搁置了雇佣新首席执行官的计划——创始人[巴尼·佩尔](https://web.archive.org/web/20221006011611/http://www.crunchbase.com/person/barney-pell)仍然牢牢掌控着公司。

Powerset 是我们密切关注的三个新搜索引擎之一。另外两个， [Cuill](https://web.archive.org/web/20221006011611/http://www.crunchbase.com/company/cuill) (发音为“酷”)和 [Blekko](https://web.archive.org/web/20221006011611/http://www.crunchbase.com/company/blekko) ，仍然处于深度隐身模式。
