# WhereIsTheCool.com:一个更懒的男人 Pinterest TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/11/11/pinterest-for-men/>

凭借其粉色和白色的配色方案以及对自我表达的强调， [Pinterest](https://web.archive.org/web/20230204213229/https://techcrunch.com/2011/11/06/rise-pinterest-shift-search-discovery/) 已经找到了迅速增长的女性受众，投资者也在排队。但是男生可能在 Pinterest 的主页上找不到适合他们的内容。见鬼，甚至它的汽车&摩托车频道也充斥着火辣的粉红色悍马豪华轿车。进入 WhereIsTheCool.com 的，这是一个展示男人渴望的生活方式的黑白照片网站——快艇、冲浪板和漂亮的晚礼服。它是为休闲浏览而设计的，就像一本男性时尚杂志，而不是像女朋友一样需要你投入时间的东西。

最初，创始人杰克·阿彻(甚至他的名字也很时髦)试图将《GQ》翻译成每月发行一期的 iPad 杂志。事实证明，2010 年市场还没有为全数字杂志做好准备，所以他将其重新设计为一个不断更新的 photostream 网站。这满足了现代人的内容瘾，现在该网站每月有超过 100 万的页面浏览量和 27，000 多名 Tumblr 订户。

《酷在哪里》的主页显示了大约 20 套顶层公寓、粗犷的演员以及随行的女性。点击一下就能看到另一个值得渴望的页面。无限滚动选项会更快，但如此多生动的内容会让人不知所措。游客可以点击查看照片，通过社交媒体分享照片，并查看照片首次发布的位置。就是这样，没有华而不实的技术，只有好品味和好设计。

阿彻过得很艰难。他的工作是在互联网上搜寻最酷的东西，并从用户提交的内容中筛选出来。“Contribute”页面的特色是一个丑陋的、老式的网络表单，它应该只让你输入照片的 URL，因为网站上的大多数条目都不包含任何文本。虽然他有一个团队提供支持，但这基本上是一个人的自助行动。

为了赚钱，阿彻偶尔会加入像波特先生这样的男士时装零售商赞助的照片，他们的产品可以很容易地登陆网站。酷在哪里还与路虎(Land Rover)和香蕉共和国(Banana Republic)等品牌签署了横幅广告的大单。

即使游客买不起它所展示的战斗机或度假屋，仅仅是激发对这些物品的渴望就能成为一种令人满意的男性体验。在男人渴望分散注意力的文化中，[酷在哪里](https://web.archive.org/web/20230204213229/http://whereisthecool.com/)以简洁取胜。
