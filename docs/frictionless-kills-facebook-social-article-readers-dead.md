# 无摩擦杀死脸书“社会”文章读者死亡

Nik Cubrilovic 和他的伙伴 Brian Kennish 创造了一些我可能每天都会用到的东西。这本书名叫《无摩擦》,它让脸书的新闻阅读摆脱了中间人的角色。它还没有完全准备好进入黄金时段，所以我建议在尝试时要谨慎，但它很有效。

你知道那些华盛顿邮报和雅虎新闻报道你看到一个脸书？你知道它是如何要求你安装一个“应用程序”来阅读链接的吗？好吧，在 Chrome 中无摩擦劫持这个过程，直接把你送到有问题的新闻故事。这有点不可靠，因为它仍然迫使你在弄清楚发生了什么之前看到应用程序页面，但一旦系统将你弹出到实际的文章，这是一个救命稻草。

我从来不知道为什么像《华盛顿邮报》这样的人首先需要这些脸书应用程序，我担心给已经过度刺激的脸书体验添加一堆垃圾。正是这种小的变通方法让网络变得更简单、更美好——就像十年前一样。
