# Twitter 开始向拥有账户的公司收费？

根据英国贸易杂志[Marketing](https://web.archive.org/web/20230313042449/http://www.marketingmagazine.co.uk/news/879748/Twitter-begin-charging-brands-commercial-use/)(brand Republic 的一部分)对联合创始人比兹·斯通的采访，将 [Twitter](https://web.archive.org/web/20230313042449/http://twitter.com/) 用于商业目的的公司可能很快开始为此活动收费。

据说斯通是这样说的:

> “我们注意到越来越多的公司在使用 Twitter，越来越多的个人在关注他们。我们可以找到让这种体验更有价值的方法，并对商业账户收费。”

这没什么大不了的，因为这经常被称为 Twitter 可能开始创收的最明显的举措之一，尽管许多人对这家初创公司抱有更高的期望，尽管它已经成立了近 3 年，但却因缺乏明显的商业模式而臭名昭著。斯通还表示，他们不会开始向个人用户收费，此举可以“创造创收功能，以利用品牌将 Twitter 作为混合营销和客户服务工具的方式。”

斯通没有给出任何关于定价的细节，也没有给出 Twitter 向用户收费的具体方式和具体收费标准。提醒一下:迄今为止，这家初创公司已经筹集了 2000 万美元的风险投资，最近还拒绝了脸书的收购要约。

公司将 Twitter 用于商业目的的最新例子之一是戴尔，据报道，戴尔通过微共享工具在假期期间实现了 100 万美元的销售额，最近开始专门向其粉丝提供[折扣。](https://web.archive.org/web/20230313042449/https://techcrunch.com/2009/02/03/dell-starts-offering-exclusive-discounts-through-twitter/)

毫无疑问，我们将在未来看到更多这类行为，但我想知道 Twitter 管理层究竟认为什么是“商业用途”:它是指推广产品或服务的任何方式，还是仅指与企业账户相关的销售活动？公司会愿意为使用这项服务付费吗？

带着这个问题，营销部门联系了戴尔社区和对话副总裁鲍勃·皮尔森(Bob Pearson)，并得到了明确的答复:“如果事情变得复杂且成本高昂，我们的直觉会是搬到别处。”

**更新:**正如彼得·卡夫卡在评论中指出的，两天前在[纽约杂志](https://web.archive.org/web/20230313042449/http://nymag.com/news/media/54069/)上有一篇很好的文章，揭示了 Twitter 的更多发展方向。

**更新 2:** 推特上的比兹·斯通写了一篇关于谣言的[帖子](https://web.archive.org/web/20230313042449/http://blog.twitter.com/2009/02/nothing-to-report-just-yet.html):

> 然而，重要的是要注意，无论我们想出什么，Twitter 都将免费供每个人使用——个人、公司、名人等等。我们考虑的是在我们已经看到吸引力的地方增加价值，而不是在现有服务上强加费用。我们仍然处于非常早期的想法阶段，尽管最近投机活动激增，但我们还没有任何东西可以分享。当我们做的时候，我们一定会让你知道。
