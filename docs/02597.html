Windows Live Messenger现在允许您在网站上添加聊天内容

为了扩大Windows Live Messenger聊天服务的覆盖范围，微软今天发布了一些工具来为网站添加聊天功能。任何网站都可以添加聊天功能，并利用3.2亿活跃的Windows Live Messenger用户。这是向任何网站添加社交元素的快速而肮脏的方式，同时它也是Live Messenger的又一个分发点(如果用户想要在网站上用该功能聊天，如果他们还没有帐户，就必须注册。).

越来越多的人接受了提供网站而不仅仅是聊天的脸书联通T4，这使得其他网络公司也开始提供类似的服务。聊天是大多数网站不会为自己建立的那种功能，但它可以增加访问者在网站上的参与度(甚至是与其他用户的参与度，而不是与网站本身的参与度)。

目标，球在你的球场上。
