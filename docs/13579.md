# 4moms 为其电子婴儿装备筹集了 2000 万美元

4moms 是一家总部位于匹兹堡的小公司，它通过将机器人技术、电子技术和创新工程融入婴儿车、婴儿座椅和游戏围栏等产品中来重塑婴儿用品行业，已经从贝恩资本投资公司筹集了 2000 万美元。该公司的姐妹基金贝恩资本(Bain Capital)也投资了玩具反斗城(Toys R' Us)和金宝贝(Gymboree)，因此这里似乎有一些知识共享和营销的机会。

至于 4moms 产品，如果你没看过的话——嗯，它们相当疯狂。TechCrunch 的小工具团队多年来一直在实践这些东西，并给予称赞“这是自最初的 TiVo 以来我见过的最酷的小工具……它只是一辆该死的婴儿车。”说真的，这些事情几乎让你想泵出婴儿(或更多的婴儿)只是为了尝试他们。差不多吧。

是什么让 4moms 产品如此不同？首先，它们不是典型的婴儿用品——它们基本上是小玩意。例如，这款[折纸婴儿车只需点击一个按钮](https://web.archive.org/web/20230320020009/https://techcrunch.com/2011/11/28/the-4moms-oragami-look-at-this-robotic-stroller-look-at-it/)即可打开和关闭。

如果您没有孩子，您可能不会意识到其中一些技术有多令人印象深刻。真实故事:我和我丈夫第一次尝试时，不得不在谷歌上搜索“如何建立游戏围栏”。我不好意思承认，我们不得不看了一段 YouTube 视频才弄明白。还有一次，我们忘了提前为保姆(咳咳，祖父母)安排，后来发现他们只是让孩子熬夜到凌晨 1 点，因为她无处安眠。让我告诉你，她睡眠不足的后果不是我希望发生在任何人身上的事。*曾经*。所以，耶:有人正在努力构建所有这些东西的更好版本，并制造出任何人都可以使用的产品。
