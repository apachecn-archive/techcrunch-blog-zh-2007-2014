# Famigo 希望帮助家庭发现适合孩子的移动应用

对于许多家庭来说，在苹果应用商店和安卓市场的上百万个应用程序中找到最好的儿童友好型应用程序可能会令人望而生畏。进入 [Famigo](https://web.archive.org/web/20230203133652/http://www.famigo.com/) ，这是一个应用程序目录和监管平台，帮助家庭找到适合孩子的游戏、电子书和其他应用程序。家庭现在可以使用 Family App Review 来发现、选择和分享苹果 iOS 和安卓移动平台上最喜欢的应用。

基本上，Famigo 不仅提供了一个应用程序目录，还提供了各种以家庭为中心的类别的苹果或安卓最佳应用程序，如按年龄组划分的最佳游戏等。家长还可以注册接收每周一封的电子邮件，为他们选择的平台(iOS 或 Android)推荐适合他们家庭需求的最佳应用。用户还可以通过排除广告、亵渎、暴力和应用内购买等特征来优化搜索。

Famigo 还声称，每个被推荐的应用程序都经过了内部审查团队的测试。应用程序的评级是由一个内部评审团队确定的，他们根据“易用性”、“教育价值”、“娱乐性”和“家庭实用性”等主题对应用程序进行评分
