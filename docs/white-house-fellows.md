# 白宫正在寻找更多的创新伙伴

白宫对其内部住宅黑客项目非常满意，以至于正在为另一轮就业机会腾出空间。总统创新研究员计划的成员在首席技术官办公室下工作，主要负责向公众开放政府数据存储，并使工具更具参与性。

【YouTube http://www.youtube.com/watch?v = KK0HOIqOcHw

自从两年前在我们自己的 TechCrunch Disrupt 大会上首次推出该项目以来，研究员们已经设计了一些项目，让数百万美国人能够下载他们的健康记录，并建立有助于减少能源消耗的工具。

一些项目从最初启动到现在仍在进行中，急需帮助解决 healthcare.gov 的崩溃，比如 RFP-EZ，一个帮助小企业申请政府合同的项目。我只是希望，随着白宫将这个住宅黑客项目制度化，它会更加认真地对待他们的项目，并通过主要法律来实施他们的想法。
