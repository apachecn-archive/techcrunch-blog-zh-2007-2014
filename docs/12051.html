Evernote推出了两款新的iPhone应用:“食物”和“你好”

今天在巴黎举行的 Le Web 大会上， Evernote 在推出了两款适用于iPhone和iPod touch设备的新应用。第一个叫做食物，旨在帮助人们轻松保存和‘重温’所有他们喜欢的食物和美食体验。

第二个应用程序被称为 Hello ，主要是希望用户在见过他们之后更容易记住他们，并记录关于这些人的想法和经历。

印象笔记美食 ( 下载链接)

这个美食应用程序本质上是给用户提供记录他们吃过的菜肴和饮料的方法，建立一个定制食谱的目录，查看餐馆并记录他们在附近或国外的美食体验，甚至管理他们的饮食。当然，它还附带了方便的共享选项。

该应用程序允许人们通过图片、位置、文本和图片说明捕捉一顿饭的所有方面。所有关于这顿饭的信息都会自动与Evernote同步，便于以后追踪。更多信息见此处。

Evernote Hello ( 下载链接)

Hello应用程序稍微复杂一点。基本上，移动应用程序旨在让人们更容易记住关于相遇、分享经历和与他人见面时的印象的一切。更多信息见此处。

从球场上看:

Evernote Hello旨在使标准问候仪式现代化。有了这个应用程序，当两个人见面时，他们交换手机，并迅速创建一个基本的个人资料。一旦添加到Evernote Hello，个人就会显示在一个可滑动的人脸马赛克中。

点击后，一张脸就会变成一个档案，显示两人分享的每次相遇的历史，以及参加相同会议的其他人、位置信息和用户Evernote帐户中的任何相关内容，如笔记和照片。

听起来真的很酷，除了“交换手机”这件事…

与美食应用一样，人物和遭遇会自动同步到Evernote。

这两款应用现在都可以在App Store上免费下载。
