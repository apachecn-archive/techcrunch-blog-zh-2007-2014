# GPhone 将在印度推出，现在就开始排队吧！

![2016.jpg](img/fc999c0c5a6cf58b0253a06f7c6e89f7.png)

Rediff India overseas 说，谷歌将在某个时候发布某个东西，也许是两周后，这似乎是谣言、影射和编造的混合体？—在印度，正在与印度最受欢迎的两家运营商 Bharti Airtel 和 Vodafone Essar 进行谈判。

在这个听起来像尼日利亚网络钓鱼的骗局中，谷歌可能会也可能不会在印度提供一个移动的东西，它可能不会也可能是一部手机、一台小型电脑，或者一只在消化系统应该在的地方植入了 TRS-80 的电脑海龟。

听着:谷歌可能在制造手机，但我对此深表怀疑。他们没有跟踪记录，除非这个东西花费 5 美元并且运行 Ubuntu Linux，否则它将是一个失败。我不在乎你在上面放了多少搜索内容——谷歌不从事硬件业务。

我知道在这一点上我会吃我的帽子，我最近一直在连败——我把这归咎于我办公室的温度——但是读一下这个:

> 谷歌发言人表示，“我们不对市场传言或猜测发表评论。然而，谷歌致力于为用户提供获取世界信息的途径，而移动设备在这些努力中变得日益重要。我们正在与世界各地的合作伙伴合作，将谷歌搜索和应用带给世界各地的移动用户。然而，我们目前没有什么可以宣布的。”

所以他们懂软件。也许一款搭载 ARM 兼容操作系统的运营商原型可以取代 WinMo6，但它不是手机。虽然那只乌龟听起来很酷。
