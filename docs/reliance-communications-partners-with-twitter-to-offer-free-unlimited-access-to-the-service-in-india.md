# Reliance Communications 与 Twitter 合作，在印度提供免费、无限制的服务

对于那些在美国和其他地方足够幸运能够为他们的智能手机购买大量数据包的人来说，考虑根据可能产生的成本决定“发微博还是不发微博”是一个外国概念，一语双关。对于印度的手机用户来说，这是一个非常真实的情况，Reliance Communications 已经与 Twitter 合作，为其预付费 GSM 用户提供免费、无限制的社交网络访问。

这是另一个例子，说明 Twitter 在我们的日常生活中变得多么重要，以及这个交流平台对于世界各地的人们来说是多么不可或缺。这项服务将与该国最受欢迎的板球比赛直播捆绑在一起。

Twitter 应用程序的定制版本已经创建，提醒客户由于 Reliance Mobile，他们可以免费访问。如果有人点击外部网站的链接，他们会被提醒这样做可能会产生额外的费用。

Reliance 是印度第一家与 Twitter 合作的运营商，其首席无线收入官 Nilanjan Mukherjee 分享了以下内容:

> 我们很高兴成为印度第一家与 Twitter 在 Twitter 接入方面合作的运营商，并在我们卓越的网络上首次提供无限制的 Twitter 接入。我们与印度 Twitter 的合作进一步加强了我们在社交媒体平台上的产品，这符合我们为客户提供价格合理的创新产品的持续努力。

由于预付费手机在像印度这样的国家很受欢迎，签署这样的协议使得服务更有吸引力。不过，回到板球对印度文化的重要性。慕克吉认为，这一声明可能会导致板球迷“重大转变”，转向信实公司。
