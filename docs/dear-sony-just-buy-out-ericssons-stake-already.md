# 亲爱的索尼:刚刚买断了爱立信的股份

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/06/dear-sony-just-buy-out-ericssons-stake-already/>

目前，索尼爱立信是世界第六大手机制造商，但索尼可能会考虑单干。根据《华尔街日报》的最新报道，索尼即将完成收购爱立信在合资公司中的股份。

根据通常未透露姓名的消息来源，目前的谈判充其量是脆弱的。这似乎并没有挫伤索尼首席执行官霍华德·斯金格的情绪:据报道，他希望这笔交易尽快敲定。

索尼的大问题似乎源于控制，特别是在索尼爱立信的手机产品线中缺乏控制。《华尔街日报》认为，通过将手机设计和生产带入内部，索尼可以利用他们的“技术优势，更快地开发新的创新手机。”

我不知道我有多赞同这种观点，但我要说:这值得一试。

索尼爱立信的实验可能已经走到了尽头。当制造一部受欢迎的手机所需要的就是将多媒体功能塞进手机时，这两个人一起设法抢回了相当大的一部分市场。时代变了，更不用说人们的期望了，现在索尼爱立信发现自己是智能手机行业的常年失败者。

让我问一个诚实的问题:你上一次看着索尼爱立信手机，感觉到内心有东西在动是什么时候？你的答案几乎肯定会与我的不同，但我敢打赌已经有一段时间了。我的是最早的 Xperia X1——一部按了我太多按钮的手机，以至于我忽略了它运行 Windows Mobile 的事实。这款手机是三年前推出的——在我向其提问的人中，没有一个人说出比它更新的手机。

早在 2001 年，合资企业就有了各种意义，甚至在那之后的几年里也是如此，但现在索尼爱立信似乎已经变得无精打采了。他们慢慢地生产出一部又一部智能手机，而观众对它们的需求一年比一年少。

我正努力不去贬低爱立信，因为他们并没有做错什么。尽管如此，他们的主要业务是开发和铺设电信网络。这并不是说爱立信搞砸了什么，而是两家公司可能没有相同的优先事项或愿景。他们 2009 年与意法半导体达成的协议并没有真正帮助事情，因为像 HTC 这样的公司[最近一直在使用他们的产品](https://web.archive.org/web/20230203054741/http://www.eetimes.com/electronics-news/4228137/ST-Ericsson-NovaThor-HTC-smartphone)。

简而言之，爱立信不像索尼那样需要手机业务。

索尼爱立信首席执行官伯特·诺德伯格似乎明白了这一点。在上周的一次采访中，诺德伯格提到该公司与索尼的关系比与爱立信更密切。在他的指导下，索尼爱立信的目标是到明年年中成为一家 100%专注于智能手机的公司。如果索尼被收购，诺德伯格在一家母公司的全力支持下运营索尼移动，这家母公司希望控制整个消费电子市场，我们可能会看到一只全新的野兽。
