# 谷歌喷气飞机开始使用美国宇航局跑道，当地社区*愤怒*

[![googleairplanenasa.jpg](img/f1f1cc651f0e2bc6630c8c236ae90011.png)](https://web.archive.org/web/20221005091811/http://old.crunchgear.com/wp-content/uploads/googleairplanenasa.jpg "googleairplanenasa.jpg") 
艺术家的再现。实际尺寸可能有所不同。

对于一个不作恶的公司来说(忽略它与中国的交易)，谷歌不会因为这个举动交任何朋友。或者会，我知道什么？距离该公司位于加州山景城的总部 4 英里的地方是莫菲特机场，这是一个由美国宇航局运营的机场，通常不对私人飞机开放。另一方面，谷歌修改的 767 是完全可以接受的。是的，谷歌的创始人(这架飞机主要是用来运送这两个人的)可以在机场使用他们的飞机，而其他同样富有(但不是像那样富有)的高管却不能。

但是有一个问题。总是有的。

美国宇航局同意让谷歌使用禁区跑道，以换取科学研究。似乎美国宇航局将为飞机配备各种各样的科学小玩意和部件，以便在飞机飞越城镇时收集数据。就像他们说的，双赢。

自然，据说其他首席执行官和“重要”人物嫉妒，机场周围的当地人不喜欢机场扩建的前景，包括击退联邦快递和 UPS 以及他们使用机场的计划。

就像爷爷辛普森曾经说过的，“哦，婊子，婊子，婊子。”
