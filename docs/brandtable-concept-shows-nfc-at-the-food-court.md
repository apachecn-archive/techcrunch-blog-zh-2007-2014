# BrandTable 概念在美食广场展示 NFC

澳大利亚设计公司 [S-Digital](https://web.archive.org/web/20230204231947/http://www.s-digital.com.au/) 为 NFC(近场通信)在美食广场的使用创造了一个有趣的概念应用。是的，美食广场——就像你当地商场里的那些。该项目被称为“ [BrandTable](https://web.archive.org/web/20230204231947/http://www.s-digital.com.au/news/2011/10/19/nfc-brand-table.html) ”，包括固定在桌面上的圆形贴纸，内含 NFC 标签，当用 NFC 手机点击时，就会启动特定餐厅的菜单和点菜页面。

使用移动网站，顾客可以下订单，甚至付款，而不必去柜台或排队等候。当订单准备好了，电话就会响，这样你就可以去拿了。

BrandTable 结合 Java 和 Android SDK 使用 NFC 来调出菜单订购系统，并使用谷歌的三星 Nexus S 进行演示(见下文)。

S-Digital 公司与悉尼大学、Mojo 和失忆剃刀鱼公司共同建造并设计了这个概念。

虽然 NFC 的广泛采用仍需几年时间，但像这样的概念有助于我们想象当该技术比今天更无处不在和更容易获得时，世界可能会是什么样子。而且，承认吧——这看起来不像点餐的一种有趣方式吗？
