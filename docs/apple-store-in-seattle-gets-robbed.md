# 西雅图的苹果店被抢

![613843950_l.jpg](img/b37ed6dd74b5aa3cb3153e7d1d18628d.png)
这看起来像是会偷苹果的人吗？

周一早上，西雅图大学村苹果商店神秘地被抢走了等待进货的维修过的笔记本电脑和翻新过的 iPhones。窃贼似乎是从天花板进来的，甚至偷走了一名技术人员，偷走了锁在隔间里的笔记本电脑。巧合的是，从周一下午开始，我们自己的 Matt Hickey 就一直试图向我们出售“打折”的 iMacs 和 iPhones。这只是巧合还是马特是罪犯之一？我不确定，但我买了一台新的 24 寸 iMac，价钱很便宜。:)
