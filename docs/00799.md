# 手掌折叠 Foleo TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/09/04/palm-folds-the-foleo/>

今天来自硅谷的爆炸性新闻。Palm 的大老板 Ed Colligan 刚刚发表了一篇博客文章，称 Palm 将不会发售 Foleo。你可能还记得，Foleo 是一个“智能手机伴侣”，不完全是笔记本电脑，但比 PDA 更好。科里根是该设备的大力支持者，因此这无疑是 Palm 要做出的一个艰难决定。

在博客中，克里根说:

> 在过去几个月的过程中，很明显，Palm 的正确道路是围绕这一新平台设计提供单一、一致的用户体验，并专注于我们的平台开发工作。为此，经过深思熟虑，我决定取消 Foleo 移动伴侣产品目前的配置…

在这一点上，我们真诚地支持他。虽然 Foleo 是一个有趣的想法，但我们认为执行的时机很差，并且与 Treo 系列智能手机可能的发展方向不同步。通过更快地发布 Treo 更新，Palm 将表明它并不是许多人认为的停滞不前的公司。

也许在新平台发布和硬件更新后，Palm 可以重新访问 Foleo。这看起来只是一家公司把事情做好的一个例子，我们对此表示赞赏。
