# 研究:对脸书的感情在朋友间传播

根据加州大学圣地亚哥分校的一项研究，对脸书的感情会像病毒一样在用户之间传播。换句话说，你朋友的情绪会积极或消极地影响你的情绪。

众所周知，在面对面的情况下，人类可以相互传播自己的情绪。坏脾气的同事会让你变得暴躁，而快乐的另一半会让你振作起来。研究人员写道，这种情况也发生在“陌生人或近乎陌生的人之间”但是没有人弄清楚它是否通过社交媒体传播。直到现在。

来自脸书的博士生 Lorenzo Coviello、Adam D. I. Kramer 和 Cameron Marlow 使用匿名数据查看了 1180 天内的英语状态更新。通过自动文本分析，他们评估了每个帖子的情绪和基调，以获得“情感内容”

他们发现了什么？首先，他们发现，当特定地理位置下雨时，该位置的情绪往往会变得消极，积极的帖子会减少。这可以被视为一种“话题传染”。然而，研究人员想看看在该地区以外没有受到雨水影响的朋友会发生什么。这些影响虽然很小，但绝对是有趣的。研究人员写道:

那么，被雨淋的人情绪表达的变化是否会导致他们的朋友保持干爽呢？是的。根据这项研究，每增加一个负面帖子，会在朋友中多产生 1.29 个负面帖子，而每增加一个正面帖子，会在朋友中多产生 1.75 个正面帖子。

“如果一个人的情绪变化蔓延并导致许多人的变化，那么我们可能会大大低估改善心理和身体健康的努力的有效性，”加州大学圣地亚哥分校医学院医学遗传学和政治学教授兼主要作者詹姆斯·福勒说。“我们应该尽我们所能来衡量社交网络的影响，并学习如何放大它们，以便我们可以创造一种幸福的流行病。”

简而言之，他们发现，比如说，你的情绪和你朋友在脸书的情绪之间存在某种关联。当你感觉不好的时候，它实际上会以数字方式传播给你的朋友，这应该会让我们下次在 Twitter 上发布“生活糟透了”的时候停下来。研究人员认为，这可以通过社交媒体和广泛的信息传播来帮助改善公众福祉。更有理由选择笑脸而不是悲伤的脸。
