# 在智能手机大战中，这是 iOS 与 Android 的较量，也是 Windows Phone 与其他手机的较量 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/08/07/in-the-smartphone-wars-its-ios-vs-android-and-windows-phone-vs-the-rest/>

今天是[智能手机平台市场份额日](https://web.archive.org/web/20221206211621/https://beta.techcrunch.com/2013/08/07/android-nears-80-market-share-in-global-smartphone-shipments-as-ios-and-blackberry-share-slides-per-idc/)，这意味着你在[当地的书呆子们对最新的数字感到兴奋](https://web.archive.org/web/20221206211621/http://www.techmeme.com/130807/p17#a130807p17):Android 的持续大幅增长，iOS 的年同比销量增长放缓，奇怪的是，今天 Windows Phone 的数字是否重要。

你看，根据 IDC 的数据，Windows Phone 发布了最高的年同比销量增长[，从 2012 年第二季度到 2013 年第二季度增长了 77.6%。这一增长率的一个简单因素是，Windows Phone 作为一个平台与 Android 相比非常小，Android 同期的单位销量增长了 73.5%。](https://web.archive.org/web/20221206211621/http://www.idc.com/getdoc.jsp?containerId=prUS24257413)

Windows Phone 粉丝很快抓住他们选择的平台获得的好消息，这是完全合理的。当然，Android 的粉丝们正在狂欢，因为 iOS 的市场份额占全球市场份额的百分比下降到了 13.2%。Android 本身在本季度飙升至 79.3%。

请记住，Windows Phone 比 Android 更容易实现大比例增长，因为 Android 必须卖出比 Windows Phone 多得多的手机才能锁定每一个百分点的销量增长。

然而，它的工作非常出色。根据 IDC 的数据，Android 2013 年第二季度的出货量为 1.874 亿部，高于 2012 年第二季度智能手机市场的出货量*1.562 亿部。这简直是疯了。*

另一个有助于你理清思路的统计数据是:尽管 Windows Phone 今年的销量增长了 77.6%，但同期的市场份额仅从 3.1%增长到 3.7%。这是因为更大的智能手机市场在过去一年中有所增长，第二季度同比增长 51.3%。

那好吧。我们究竟能从这些该死的数据中得到什么？这里有两个独立的叙述，我们不应该混淆或合并，因为如果我们这样做，我们将一无所获。看待智能手机市场最简单的方式是，这是两场独立的战争:iOS 对 Android，Windows Phone 对其他所有人。

这个古老的笑话很适合:两个男人被一只老虎追赶。一个人从背包里拿出一双跑鞋，开始系鞋带。你在干什么，他的朋友喊道，他们来了！你跑不过老虎！不，朋友说，他的鞋子已经牢牢地穿上了。我只要跑得比你快。

这就是目前的 Windows Phone，因为它面临着黑莓和其他越来越边缘化的智能手机平台。iOS 和 Android 的出货规模只是将它们放在一个单独的类别中；Windows Phone 必须让开发者相信它是可靠的第三代移动平台，并且它能够卖出足够多的手机来满足他们的平台需求。

我要说的是，最近的移动市场份额数据肯定了这两点:Windows Phone 正在增长，出货量位居第三，所有其他平台的出货量都低于其数量。

因此，看待智能手机市场的一个简单方法是，Android 不断给苹果产品增加摩擦，随着市场本身的扩张速度超过出货量，苹果产品的市场份额正在下降。Windows Phone 虽然仍是遥遥领先的第三名，但已经成功地巩固了自己在移动领域的地位。

一旦一个季度的 Windows Phone 手机出货量超过 1000 万部，这一事实将变得更加可信。然而，除非市场出现戏剧性逆转，否则这一数字应该会在 2013 年内达到。

谈到苹果，我不是专家，所以我将把分析苹果目前在智能手机大战中的地位——尽管很难抵制添加“困难”的诱惑——留给那些撰写此类文章的人。然而，鉴于 IDC 今天披露的情况，我们基本上可以将市场视为一场三方战争，但 Windows Phone 甚至没有开始挑战现有公司。
