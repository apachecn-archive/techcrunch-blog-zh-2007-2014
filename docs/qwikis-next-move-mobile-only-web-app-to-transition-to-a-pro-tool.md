# Qwiki 的下一步棋:仅移动端；Web 应用程序过渡到专业工具

TechCrunch Disrupt 的赢家 [Qwiki](https://web.archive.org/web/20221207080445/http://www.qwiki.com/) 似乎正在将其重心从基于网络的创作工具转移到仅面向移动设备。

作为背景，Qwikis 本质上是多媒体演示，配有叙述。五月，Qwiki [宣布为出版商和博客作者提供一个新的创作平台，被 ABC 新闻](https://web.archive.org/web/20221207080445/https://beta.techcrunch.com/2012/05/23/qwiki-publisher-platform/)和其他媒体使用。今年 6 月，初创公司[首次与微软](https://web.archive.org/web/20221207080445/https://beta.techcrunch.com/2012/06/12/qwiki-bing/)合作，将 Qwikis 嵌入到必应搜索结果中，并可以直接在搜索页面中播放。Qwiki 还提供了一个 iPad 应用程序，迄今已有 100 万次下载。

据这家初创公司的网站称，在 Qwiki 看到人们对 iPad 应用的反应，并开始尝试创作工具和 Bing 集成后，该公司意识到用户更喜欢移动而不是网络。因此，Qwiki 现在已经专门为 iPhone 建立了一种新的体验。web 应用程序将转变为专业工具，详细信息将很快公布。

在 Qwiki 的网站上，“我们希望你对 Qwiki for iPhone 感到兴奋。这款应用结合了我们在媒体自动化和 iOS 开发方面的专业知识，我们相信这款产品代表了讲故事的未来。”

我们联系了联合创始人兼首席执行官 Doug Imbruce，他告诉我们，“基于我们在 iPad 上现有的成功以及用户需求，将我们的大部分努力集中在创造尽可能最好的移动体验上是完全有意义的。我们的 iPhone 应用程序已经开发了一年多，随着我们越来越关注它，它将有望为所有 iPhone 用户带来绝对精致的体验。最重要的是，它实现了 Qwiki 对自动媒体生成的承诺，我相信这将使我们的消费产品从新奇走向极端实用。”他说这款应用应该会在一个月后发布。

Qwiki 已经从包括脸书联合创始人爱德华多·萨维林、Greylock Partners 和 Lightbank 在内的投资者那里筹集了 1050 万美元。

正如我们过去所写的，Qwiki 的愿景是呈现一种新的信息体验，而不仅仅是包装现有的内容；很明显，这个计划的一部分集中在移动平台上。看起来我们很快会看到更多关于 Qwiki 的细节，所以请继续关注。
