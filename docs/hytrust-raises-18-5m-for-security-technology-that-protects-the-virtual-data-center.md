# HyTrust 为保护虚拟数据中心的安全技术筹集了 1850 万美元

[HyTrust](https://web.archive.org/web/20221206141815/http://www.hytrust.com/) 已经为其安全技术筹集了 1850 万美元，该技术专为虚拟化数据中心的新现实而设计，虚拟化数据中心需要新的方法来防止安全漏洞。

C 轮超额认购融资来自英特尔投资和 Fortinet，以及最近的投资者 VMware 和 [In-Q-Tel](https://web.archive.org/web/20221206141815/http://www.hytrust.com/company/news/press-releases/hytrust-announces-new-funding-strategic-partnership-q-tel) 。之前的投资者 Granite Ventures、思科系统、Trident Capital 和 Epic Ventures 也参与了此轮投资。

HyTrust 技术旨在保护虚拟化数据中心，理论上，虚拟化数据中心将所有计算、存储和网络置于一个软件层中。在当今的现代数据中心，IT 管理员使用管理平台，使组织面临相当大的风险。管理员可以相对轻松地擦除整个数据中心或复制虚拟机。HyTrust 提供了一种在管理员和虚拟基础架构之间管理数据的方法。它提供了一个基于角色的系统，可以帮助监控一个人正在做什么，而不是他们应该做什么。

如今，实时可见性是大多数运行自己的数据中心的公司的不同之处。攻击更加残忍，破坏性更大。关键代码、锁在笼子里的服务器甚至武装警卫都是愚蠢的，因为最大的威胁来自于无形的力量，这些力量经常从完全不同的大陆发起攻击。
