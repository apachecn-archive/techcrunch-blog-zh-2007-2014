# Xamarin 筹集了 1600 万美元的 B 轮融资，由 Lead Edge Capital 牵头，通过了 20，000 个付费开发者席位

iOS、Android、OS X 和 Windows 原生应用的跨平台开发平台 Xamarin 今天宣布，它已完成由 Lead Edge Capital 领投的机会主义的 1600 万美元 B 轮融资。该公司的 A 轮投资者 Charles River Ventures、Ignition Partners 和 Floodgate 也参与了此轮投资。包括 1，200 万美元的首轮融资在内，Xamarin 目前总共筹集了 2，800 万美元。

该公司专注于使企业的移动应用程序开发更容易，今天还宣布其开发者社区现已发展到超过 350，000 名成员，包括 20，000 个付费开发者席位。这家公司现在显然很兴旺。今年早些时候，Xamarin 举办了第一次[开发者大会](https://web.archive.org/web/20230322160429/http://xamarin.com/evolve/2013)，会议门票很快售罄，并推出了许多创新产品，包括其新的[移动 UI 测试平台](https://web.archive.org/web/20230322160429/https://techcrunch.com/2013/04/16/xamarin-launches-test-cloud-automated-mobile-ui-testing-platform-acquires-mobile-test-company-lesspainful/)。

正如 Xamarin 的首席执行官 Nat Friedman 告诉我的那样，该公司计划利用这笔资金增加对其开发者平台的投资，“特别是在预构建的组件和应用程序方面，以帮助开发者更快地构建优秀的应用程序”，以及其测试云。Xamarin 还计划在今年年底前将其销售团队扩大一倍。

“在不久的将来，每一个业务流程和客户互动都将发生在移动设备上，”弗里德曼今天在一份声明中说。“Xamarin 的独特方法使企业能够快速为多种设备平台提供完全原生的应用程序，这带来了令人难以置信的收入增长和市场发展势头。这一成功，加上这次机会主义的 B 轮融资，为公司未来的指数级增长奠定了基础。”

他还告诉我，投资者接触了该公司，但结束这一轮交易背后的驱动力是“将 Xamarin 定位为主导移动开发市场。”他认为，该公司的增长速度明显快于其竞争对手，而且“再来一轮，为这一成功火上浇油。”
