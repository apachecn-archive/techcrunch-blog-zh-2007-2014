# 少数派报告——在社交媒体上预测骚乱是不可能的

在英国范围内的骚乱之后，越来越多的证据表明，像黑莓信息系统这样的移动应用帮助暴徒组织和[扩大他们的活动](https://web.archive.org/web/20230204093530/http://eu.beta.techcrunch.com/2011/08/11/absolute-explosion-%e2%80%94-how-blackberry-bbm-fed-the-riots-says-contact/)，英国政府正在考虑对社交媒体进行审查。具体来说，内政大臣特里萨·梅计划与推特、脸书和 RIM 坐下来讨论这些问题。那会是一次有趣的谈话。

她直言不讳:“脸书和 Twitter 等社交网站以及黑莓 Messenger 等信息服务被用来协调犯罪活动，并比警方领先一步……我将召集 ACPO、警方和社交媒体行业的代表开会，研究如何提高警方的技术和相关法律能力。”
