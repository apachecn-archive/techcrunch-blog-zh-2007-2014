# 有史以来第一个 i/o 风险投资演示日:应用程序、市场和脸书天啊！

在旧金山瓦伦西亚街的 [i/o Ventures](https://web.archive.org/web/20230404205615/http://www.ventures.io/) 孵化器，又名*【一个真正的极客咖啡馆】*这里，六家 I/o Ventures[公司刚刚结束向媒体和投资者进行演示。虽然这些公司包括一些我们以前写过的公司和一些还没有推出的公司(我们正在看着你](https://web.archive.org/web/20230404205615/http://ioventures.com/) [Skyara](https://web.archive.org/web/20230404205615/http://skyara.com/) )，但因为我等不及了，我决定发布我对下面介绍的公司的粗略记录，并提供我们以前报道的链接:

[Appbistro](https://web.archive.org/web/20230404205615/http://appbistro.com/) — [我们的报道](https://web.archive.org/web/20230404205615/https://techcrunch.com/2010/10/12/appbistro-wants-to-give-every-service-with-an-api-its-own-app-store/) —一个脸书商业相关应用的市场，Appbistro 试图规避一些围绕脸书粉丝页面的挑战。Appbistro 最终希望为任何有 API 的服务建立一个应用商店，让品牌可以选择花费数千美元来营销他们的应用，通过应用商店让用户为企业评级和评论应用，脸书还没有提供这种服务。市场是巨大的:目前有 2000 万脸书粉丝页面，其中一半是本地企业。App Bistro 打算通过与这些公司 70/30 的收入分成以及上市费和赞助投放来盈利。

[Socialvision](https://web.archive.org/web/20230404205615/http://youtubesocial.com/) — [我们的报道](https://web.archive.org/web/20230404205615/https://techcrunch.com/2010/09/22/videos-with-friends/) — SocialVision 认为视频观看的未来在于他们所谓的“观看聚会”和他们的消费者应用程序，YouTube Social 允许你通过脸书邀请你的朋友，以便同时观看视频。目前与 Direct TV 合作的 Social Vision 打算通过为内容公司提供平台来举办自己的观看聚会来赚钱。

[异常](https://web.archive.org/web/20230404205615/http://thecadmus.com/) —异常是一种试图*“使网络个人化”*的尝试，它通过提供一种算法，为用户提供个性化的策划进化的网络，这是一种永无止境的信息消防。异常算法会找出你感兴趣的内容，并按照相关性的顺序提供给你。他们最近的 Twitter 应用 thecadmus.com 被誉为“Twitter 的新闻源”,显然增加了三倍的 Twitter 用户流参与度。

[Damntheradio](https://web.archive.org/web/20230404205615/http://damntheradio.com/) — [我们的报道](https://web.archive.org/web/20230404205615/https://techcrunch.com/2010/08/12/damntheradio-launches-ups-the-ante-on-facebook-branding/) — *“如果你要推广任何东西，你需要有社交媒体的存在，”基于奖励的营销平台 Damntheradio 的创始人 Johnny Hwin 说，该平台自 7 月以来一直有正现金流。利用获得更多粉丝的普遍需求，Damntheradio 帮助企业奖励粉丝，以换取他们的社交行为，如“喜欢”粉丝页面。目前就在脸书，Damntheradio 社交奖励平台最终希望帮助企业连接所有消费者接触点，社交网络，如 Foursquare 和 Twitter，移动设备，谷歌电视，最终谷歌汽车。:-)*

[sky ara](https://web.archive.org/web/20230404205615/http://skyara.com/)——sky ara 是一个 Etsy/Airbnb 体验网站，目前尚未推出，主要面向那些问*“有什么可做的？”有经验可提供的企业可以与寻找事情做的人联系。作为一个体验推荐引擎，Skyara 有潜力在 Opentable 之外的 SF 中提供酷的食物体验，例如与 i/o Ventures 的合作伙伴 [Paul Bragiel](https://web.archive.org/web/20230404205615/http://www.crunchbase.com/person/paul-bragiel) 一起去品尝啤酒。尽管 Skyara 仍处于隐形模式，但它已经拥有了一个丰富的市场和一系列商业产品。*

[AppRats](https://web.archive.org/web/20230404205615/http://apprats.com/) — AppRats 是脸书的应用程序，迎合不同品牌的名人，而不是电视和广播，假设 YouTube 名人需要不同的互联网技能，并为希望通过脸书应用程序与粉丝互动的名人提供平台。AppRats 目前与 YouTube 前 100 名名人中的 50 人合作，并希望增加更多人。
