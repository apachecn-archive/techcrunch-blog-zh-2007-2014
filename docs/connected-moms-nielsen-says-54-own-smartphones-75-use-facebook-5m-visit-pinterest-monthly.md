# 联网妈妈:尼尔森称 54%的人拥有智能手机；75%使用脸书；每月访问 Pinterest 万次

尼尔森一直在对不同的人口统计数据及其在数字世界中的影响进行一些研究，他们的最新研究——很可能是对本周末母亲节的致敬——是对美国妈妈们的[审视](https://web.archive.org/web/20230401180215/http://blog.nielsen.com/nielsenwire/?p=31809),这个群体你可能会认为是技术上的勒德分子，但实际上他们表现得很好，非常感谢。

事实证明，妈妈们是高于平均水平的智能手机用户，是脸书和 Pinterest 的忠实粉丝，“妈妈博客”的概念实际上是现实。调查结果还显示，他们是网上参与度最高的消费者。

尼尔森说，大约 54%的美国妈妈正在使用智能手机:考虑到美国人口上个月[刚刚暗示 50%的人拥有智能手机(准确地说是 50.4%)，这意味着母亲拥有高于平均水平的智能手机。](https://web.archive.org/web/20230401180215/https://techcrunch.com/2012/05/07/nielsen-smartphones-used-by-50-4-of-u-s-consumers-android-48-5-of-them/)

谈到社交网络的使用，他们同样活跃。大约 75%的母亲使用脸书，Pinterest 每月超过三分之一的观众，即大约 500 万用户，是由母亲组成的。尼尔森指出，这表明母亲访问 Pinterest 的可能性比普通消费者高 61%。至少这种对社交媒体的积极使用部分归因于大量的移动使用:尼尔森还指出，超过一半的母亲通过移动设备使用社交媒体，而普通人群中只有 37%。

事实证明，母亲不仅仅是内容的消费者，也是创造者(妈妈们，我们太有创造力了)。尼尔森说，在美国所有的博客中，大约三分之一是母亲。这有助于促成另一个有趣的事实:美国 52%的博客作者是有年幼孩子仍住在家里的父母。

他们似乎也在网上参与了更多的活动。在过去的 30 天里，妈妈们成为某个品牌粉丝或在线关注该品牌的可能性比普通消费者高出 38%。他们更热衷于网上购物:35%的人更喜欢在网上购买衣服；50%的人会购买玩具，29%的人会购买音乐，23%的人会购买电子书。

如果有什么不同的话，那似乎表明电子商务公司有机会扩大他们在网上的业务，以瞄准母亲消费者。在英国，网上杂货店购物是一项巨大的业务(这是我这些天在杂货店“购物”的唯一方式)，但也许因为这在美国似乎不那么普遍，尼尔森的调查结果中没有包括这种用法的统计数据。
