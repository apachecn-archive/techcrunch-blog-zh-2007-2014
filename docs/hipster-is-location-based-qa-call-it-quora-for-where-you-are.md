# Hipster 是一个基于位置的问答网站——你可以把它叫做 Quora

![](img/c551787519bf58d3e0476fbfe9bda6db.png)
突然人们在[谈论](https://web.archive.org/web/20230204140124/https://techcrunch.com/2011/01/17/hipster-2/)关于[潮人](https://web.archive.org/web/20230204140124/http://sanfrancisco.usehipster.com/)。

好吧，让我们澄清一下。旧金山的一群科技爱好者正在讨论潮人到底是什么，因为潮人首席执行官道格·勒德洛至今拒绝透露潮人到底是什么。它正在产生一个病毒式营销的案例研究。

幸运的是，所有这些讨论都产生了答案，TechCrunch 的读者肯定已经在我们的[评论](https://web.archive.org/web/20230204140124/https://techcrunch.com/2011/01/17/hipster-2/)中找到了答案。

For Hipster 似乎是一个围绕您当前位置建立的问答网站。让我们称之为“位置 Quora”它甚至看起来像 Quora。从谷歌[缓存](https://web.archive.org/web/20230204140124/http://webcache.googleusercontent.com/search?q=cache:suTSLVuHsg4J:boulder.usehipster.com/questions/where-is-the-best-happy-hour+/search%3Fhl%3Den%26q%3D%2Bsite:usehipster.com%2B%2522use%2Bhipster%2522&cd=5&hl=en&ct=clnk&gl=us)中查看下面的屏幕截图。

但更有趣的是，这项业务已经以 Loqly 的形式存在，这是谷歌搜索、本地专家和实时答案的混搭。请看下面的视频。
