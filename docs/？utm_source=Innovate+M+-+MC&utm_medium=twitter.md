# iTunes 的竞争对手 Rdio 推出了礼品卡，可在网上和 Target 购买

Skype 创始人尼可拉斯·曾斯特罗姆和杰纳斯·弗里斯的流媒体音乐服务 Rdio 今天推出了两种新的音乐礼物选择——实体礼品卡和在线积分。消费者将能够以 10 美元、25 美元、50 美元和 100 美元的价值或用户选择的金额购买虚拟礼品卡。与此同时，价值 10 美元和 25 美元的实体卡将在美国 600 多家目标零售店出售

Rdio 礼品卡是灵活的，因为它没有指定如何使用这笔钱。接收者可以选择将其用于每月订阅、离线音乐下载或与[最近宣布的 Rdio 无限家庭计划](https://web.archive.org/web/20230204192751/https://techcrunch.com/2011/07/27/rdio-to-offer-family-unlimited-plan-broke-music-fans-rejoice/)一起使用。

**Rdio 与 iTunes——“租借”与“拥有”**

礼品卡是 Rdio 首次在美国零售商店销售。尽管它们将与最大的竞争对手 iTunes 一起销售，但这两种服务实际上非常不同。iTunes 会对您下载的音乐收费，这些音乐可以保存并同步到您的 Apple 设备上。相反，Rdio 为其近 1100 万首歌曲目录中的所有音乐提供流媒体访问，每月收费。换句话说，在 iTunes 中，你拥有你购买的音乐，但在 Rdio 中，大多数情况下，你是租借的。

就像苹果即将推出的 iTunes Match 服务一样，Rdio 也采用了一种匹配工具来搜索你的本地图书馆(在 iTunes 或 Windows Media Player 中),以寻找其目录中已经存在的艺术家、专辑和歌曲的名称。这包含在 Rdio 的月费中，但苹果将收取 25 美元/年的费用。然而，苹果也将在比赛时将曲目升级为高质量的 DRM 格式。

此外，如果需要，Rdio 用户可以购买精选曲目进行离线收听——这一功能与传统的 iTunes 服务更为相似。这种选择通常是由于音乐标签和/或版权所有者对流媒体的限制。

新的 Rdio 礼品卡现在可以在 Target 或 Rdio 的网站[www.rdio.com/gift](https://web.archive.org/web/20230204192751/http://www.rdio.com/gift)获得。
