# 云存储平台 Box.net 以超过 6 亿美元的估值从 Salesforce、SAP 融资 8100 万美元 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/11/cloud-storage-platform-box-net-raises-81m-from-salesforce-sap-at-600m-plus-valuation/>

云存储平台[Box.net](https://web.archive.org/web/20230203064641/http://box.net/)已经从战略投资者 Salesforce.com 和 SAP Ventures 筹集了 8100 万美元的 D 轮资金，Bessemer Venture Partners，NEA，以及包括 Andreessen Horowitz 和 Draper Fisher Jurvetson Growth 在内的优先投资者参与了此轮投资。新的投资使 Box 的总资本增加到 1.62 亿美元(我们几周前报道的[5000 万美元轮](https://web.archive.org/web/20230203064641/https://techcrunch.com/2011/09/28/cloud-storage-platform-box-net-raises-50-million-from-salesforce-and-others/)是这轮 D 轮的一部分)。我们从接近该公司的消息来源获悉，该盒子的估值超过 6 亿美元。

Box 拥有 700 万用户，存储超过 3 亿份文档，是一个面向企业的云存储平台，具有协作、社交和移动功能。Box 已经[发展](https://web.archive.org/web/20230203064641/https://techcrunch.com/2011/01/20/box-net-upgrades-cloud-storage-platform-with-new-ui-collaborative-features-and-more/)不仅仅是一个 fils 存储平台，而且已经成为一个成熟的协作应用程序，企业可以在其中实际交流文档更新、[远程同步文件](https://web.archive.org/web/20230203064641/http://www.techcrunchit.com/2010/06/10/box-net-moves-beyond-storage-in-the-cloud-adds-file-syncing-to-the-mix/)，甚至添加来自 Salesforce、Google Apps、NetSuite、Yammer 等的功能。

该公司于 2005 年由 Aaron Levie 和 Dylan Smith 在他们的宿舍里创建，目标是让人们在任何地方都能轻松访问和分享他们的所有内容。如今，该公司为 77%的财富 500 强企业提供存储解决方案。目前，有 10 万家企业在使用 Box 的服务，每月有 25 万新用户加入。今年，Box 与宝洁公司达成了迄今为止最大的企业交易(也是有史以来最大的云 SaaS 交易之一)，拥有 18，000 个席位

Draper Fisher Jurvetson 的董事总经理乔希·斯坦恩(Josh Stein)早在 2006 年就是 Box 的早期信徒。他说，当时有两件事非常突出，表明这个产生于列维和史密斯宿舍的想法是有潜力的。首先，他解释说，史密斯和李维拥有他所见过的最好的职业道德。

他回忆说，当公司向李维和史密斯询问有关产品和业务的问题时，这两位当时初出茅庐的企业家会在一天的所有时间里用几页深思熟虑的电子邮件来回应。第二，早期产品的质量，在当时是针对消费者和企业的，是直观的，设计良好的，有价值的功能。

那么，为什么 Box 能够从宿舍里的一个想法发展成近十亿美元的生意呢？斯坦说，Box 的云存储平台是一个“了不起的产品”，可以满足各种行业的需求。他说，在过去的几年里，Box 在企业产品中创造了一个新的类别，公司内部网本可以成为这个类别。

NEA 合伙人 Kittu Kolluri 同意 Stein 的观点，认为 Box 已经成为内部网的延伸，但他补充说，Box 也已经成为下一代外部网，因为 Box 允许公司与公司外部的客户共享文件和通信。

Kolluri 还强调了 IT 消费化的增长趋势。他说，NEA 研究了企业存储和协作领域的其他几家公司，但 Box 拥有最具吸引力的解决方案，能够为企业提供安全性和易用性。

由于其引人注目的产品、不断增长的收入和强大的用户基础，该公司甚至成为了一个昂贵的收购目标，收到了 5 亿美元的收购要约。Kolluri 表示，Box 拒绝这些收购是一个明智的决定(NEA 在这些收购被拒绝后进行了投资)。Kolluri 全心全意地相信 Box 能够也将会成为一家上市公司。

Stein 也相信 Box 在公开市场的潜在实力。“Box 绝对应该是一家上市公司，”他表示，并确认上市是该公司的未来计划。斯坦说，作为一家 SaaS 公司，Box 的经常性收入模式使其成为一家理想的上市公司。斯坦补充说，当然，筹集这轮增长资金让 Box 可以灵活决定何时进入公开市场。

今年早些时候，Levie 从 Andreessen Horowitz 和其他人那里获得了 4800 万美元，但由于强劲的风险市场、公司的强劲表现和云计算的增长，一轮扩张是有意义的。

“企业中发生了如此多的变化，我们试图为人们如何使用数据、工作和协作建立一个平台，”Levie 解释道。其中一部分是与微软这样的老牌企业竞争。“我们正在重新定义企业在 Box 上共享和管理内容的方式，同时也在构建一个强大、开放的合作伙伴和开发人员生态系统，以帮助我们的客户从他们的信息中获得前所未有的价值和灵活性。”

其中一部分是围绕平台创建一个生态系统。 in 11 月，Box 计划推出 Box 创新网络，该网络将为基于 Box 的开发者提供资金、咨询和其他资源。如今，Box 集成了 120 个应用程序，包括领先的云解决方案，包括 Salesforce、SAP StreamWork、Google Apps 和 NetSuite。

Box 计划将这笔新资金用于产品开发和国际扩张，并进一步建设其在美国的基础设施，在 2012 年开设第三个数据中心。 

斯坦说，当然，这种扩张也会带来挑战。快速增长会给组织带来压力。Box 的团队从 2011 年的 125 名员工迅速扩大到近 300 名员工，其中包括来自思科、EMC、微软、NetSuite 和甲骨文等公司的关键高级员工。当然，这些高管拥有与高增长公司打交道的经验，能够帮助公司在持续增长的同时进行创新。

就战略合作伙伴而言，Salesforce 和 SAP 是有趣的选择。Levie 说，考虑到公司在云领域的两项投资，Salesforce 作为合作伙伴尤其有意义。Box 将[首次推出与 Salesforce 的企业聊天社交网络的集成](https://web.archive.org/web/20230203064641/https://techcrunch.com/2011/09/28/box-net/)，并将在未来继续推出更深层次的集成。Levie 说，Box 的大多数客户也使用 Salesforce。

SAP Ventures 董事总经理 Jai Das 表示:“Box 通过为业务信息带来新的创新，帮助组织更快地做出更好的决策。“作为投资者和合作伙伴，我们对 Box 如何重塑内容管理和协作感到兴奋，并期待与他们合作，提高客户的工作效率。”

“最终，我们不希望企业受到更多限制,”李维说。但是他知道要做这个盒子需要快速创新。“史蒂夫·乔布斯提醒了我们时间的挑战，”他解释道。“我们试图充分利用我们的时间，快速完成大量工作。利用这种云计算增长的能力取决于我们上市的速度。”

在接下来的几个月里，我们肯定会期待更多的产品和特性问世。很明显，我们最早也可能在明年看到公开募股。对于云存储公司来说，2012 年应该是有趣的一年。
