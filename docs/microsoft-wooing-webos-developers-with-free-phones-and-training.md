# 微软用免费手机和培训吸引 WebOS 开发者

在惠普决定砍掉所有 WebOS 产品之后，许多开发者的问题是“下一步是什么？”虽然我确信一些 WebOS 的顽固分子将继续在这个平台上工作(并希望为所有廉价的触摸屏创造一些伟大的新应用)，但微软已经向这些被剥夺权利的开发者张开了双臂。

上周五，微软 Windows Phone 7 开发高级总监布兰登·沃森(Brandon Watson)在推特上向已发布的 WebOS 开发者发出邀请:微软将为他们提供成功所需的一切，从免费手机到开发工具和培训。

沃森的一个大胆举动，似乎已经得到了回报:[他在不到 24 小时内收到了来自感兴趣的开发者的 500 多封电子邮件](https://web.archive.org/web/20230205035629/https://twitter.com/#!/BrandonWatson/status/105036025478381569)，可以理解的是，沃森对这些回复感到“震惊”。这不是沃森第一次冒险让自己受到开发人员的喜爱:过去他在 Twitter 上分享他的个人手机号码来回答关于开发过程的问题。

除了热烈的欢迎之外，前 WebOS 开发者可能会在过渡中挣扎。许多 WebOS 应用程序是用 HTML 和 JavaScript 编写的，而要充分利用 Windows Phone 7，需要熟悉 C#和。NET 框架。微软承诺向新的 WinPho 开发者提供资源，这可能正是他们适应新环境所需要的。

有了这样一个慷慨的提议，微软似乎意识到打造强大的开发者关系对他们的移动平台的成功至关重要。Windows Marketplace 的应用数量正在迅速接近 30，000 个，布兰登对开发者关系的欢迎态度可能会帮助雷德蒙进一步增加他们的数量。微软的移动操作系统竞争对手有数十万个应用程序，虽然质量肯定比数量重要，但大型应用程序商店数量是衡量平台健康状况的快速(如果目光短浅的话)方法。

在前 WebOS 开发者中获得牵引力只是微软移动开发工作的冰山一角。他们的下一个重点是什么？沃森说[今年与学生合作对微软来说将是“巨大的”](https://web.archive.org/web/20230205035629/https://twitter.com/#!/BrandonWatson/status/105020267599900672)。
