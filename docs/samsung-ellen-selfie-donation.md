# 三星将向艾伦选择的慈善机构捐赠 300 万美元，称这是“有机地”包含在她的奥斯卡自拍中

那么……艾伦·德詹尼丝在奥斯卡上已经很出名的名人自拍是怎么回事？她为什么用看起来像是三星 Galaxy Note 的东西来拍照？

我的意思是，没有反对 Galaxy Note 的意思，但是[有一些关于主持仪式的德杰尼勒斯是否从三星获得了使用手机的报酬的争论。三星发言人刚刚给我发了一份声明，暗示事实并非如此。](https://web.archive.org/web/20230207020219/https://twitter.com/felixsalmon/status/440527481159036928)

是的，三星是奥斯卡的赞助商，显然“与美国广播公司(ABC)(转播仪式的网络)有所融合”，但该公司表示，是德杰尼勒斯选择“将设备有机地融入让所有人都在谈论的自拍时刻。”

三星还表示，它对结果非常满意，将分别向德杰尼勒斯选择的两家慈善机构捐赠 150 万美元(共计 300 万美元)——圣裘德儿童研究医院和美国人道协会。

以下是完整的声明:

> 虽然我们是奥斯卡的赞助商，并与美国广播公司进行了整合，但我们很高兴看到艾伦将设备有机地融入到人人都在谈论的自拍时刻。让每个人都大吃一惊的是，她捕捉到了一些没人预料到的东西。为了纪念这一史诗般的时刻，当然，还有近 300 万条推文令人难以置信的回应，我们想向艾伦选择的慈善机构捐款:圣裘德和人道协会。三星将向每个慈善机构捐赠 150 万美元。
