美国国家安全局赢得了一场不可逆转的战斗，但也可能输掉这场战争TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2013/07/25/the-nsa-won-yesterdays-battle-but-it-could-lose-the-war/

美国国家安全局(National Security Agency)失去了维持其有争议的互联网和电话搜索网间谍活动所需的政治支持。昨天，7月24日，众议院几乎批准了对的最厚颜无耻的修正案，彻底切断对任何广泛的国家安全局间谍项目的资金支持(失败205-217)。有了更多的时间来建立基层势头，并制定一项减少NSA间谍权力的不那么暴力的法律，下一项法案可能会获得足够的支持，以赢得胜利。

只有几天的准备时间，众议员贾斯汀·阿马什(Justin Amash)成功推动了国防拨款法案的修正案，该修正案将否认国家安全局将资金用于广泛监视美国人的项目的能力。令人惊讶的是，大多数民主党人反对他们自己的领袖奥巴马总统，支持修正案，111-83。

[推特https://twitter.com/jamesrbuk/status/360205168609992705]

自从美国国家安全局大规模收集电话记录和互联网浏览行为的消息被披露后，国内间谍项目的支持者担心这些法律不会被更新。具体来说，国家安全局从9/11时代的《爱国者法案》第215条中获得法律授权。

“第215条将于2015年底到期，”爱国者法案作者众议员吉姆·森伯伦纳在本月的国会听证会上告诉他的同事。除非你意识到你有一个问题，否则这不会被更新。众议院没有投票延长第215条”。换句话说，到2015年，国会不太可能更新允许NSA有争议项目的法律。

在过去的一个月里，有几项法律被提议限制NSA监视美国人的能力。例如，众议员史蒂夫·科恩的FISA责任法案，将要求国会和最高法院大法官任命新法官到批准国家安全局间谍活动请求的法院( FISA )，而不是给保守派首席大法官约翰·罗伯茨自己任命他们的权力。FISA法院批准了几乎每一个NSA的间谍请求，理论上，这将任命对第四修正案更友好的法官(根据Wonkblog，目前有10名共和党法官和1名民主党法官)。

对NSA的批评者来说，不幸的是，没有一项提议得到全票通过，因此众议院议员永远不必宣布他们是支持还是反对现状。

现在，我们有确凿的证据表明，近一半的民主党人和共和党人支持大幅削减国家安全局的监控能力。更多的人可能同意应该有一些改变。

至少，NSA搜索网的法律基础不太可能超过《爱国者法案》2015年的更新日期。预计到这种损失，情报机构可能不得不寻找某种妥协，通过国会，而不是冒着失去所有权力的风险。
