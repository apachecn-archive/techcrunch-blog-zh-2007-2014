# DevHub 可以让你一步到位地发布和盈利网站

DevHub 今天向公众发布了一个解决方案，让用户能够创建和发布多个网站，并提供一种在同一过程中赚钱的方法。

基于网络的商业出版平台并不是突破性的东西，但 DevHub 允许网络出版商利用 API 进入一个相当大的网站和附属网络的聚合网络，拥有超过 35 个互联网资产，如 Careerbuilder、Superpages、Shopping.com、Priceline、Homegain 等。).这家初创公司表示，这种集成的货币化方法、丰富的所见即所得编辑功能和广泛的报告功能使他们有别于其他在线网站建设工具(还有很多这样的工具)。

在私人测试期间，西雅图公司已经用中央平台开发了超过 55，000 个网站，今天它进入了公开测试，允许任何人注册并乘坐它。该公司声称建立一个网站需要 5-10 分钟，所以你不需要因为时间不够而停止测试。

![](img/46a40c4fbe73e773d09c3bc2875277fa.png)

他们注册新用户的方式也很有趣:虽然 DevHub 用户可以在 DevHub 平台上管理无限数量的网站，但他们必须通过第一组 10 个网站的质量门槛，才能获得下 10 个网站的席位。目前还不清楚这项质量测试具体需要什么。

该产品主要针对那些希望在没有大量技术知识的情况下建立在线媒体网络的互联网企业家。

![](img/ec10ee46765a574511667a41f5a6a098.png)

DevHub 实际上是 EVO Media Group 公司的产品，由 BuddyTV 和 icanhascheezeburger.com/failblog.com 网站背后的同一个天使集团支持。在过去的 1.5 年里，他们一直在用私人投资者不到 100 万美元的资金开发这个平台。
