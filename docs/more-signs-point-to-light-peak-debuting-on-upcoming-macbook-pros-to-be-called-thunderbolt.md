# 更多迹象表明，Light Peak 将在即将推出的 MacBook Pros 上首次亮相，被称为 Thunderbolt

[![](img/cdaa10b0f257b8a899f80dbba5c2ffa2.png "thunderbolt")](https://web.archive.org/web/20230205041642/https://techcrunch.com/wp-content/uploads/2011/02/thunderbolt.jpg) 苹果谣言满天飞[左](https://web.archive.org/web/20230205041642/https://techcrunch.com/2011/02/22/apple-rumor-mill-on-fire/)右[右](https://web.archive.org/web/20230205041642/https://techcrunch.com/2011/02/22/os-x-lion-summer-2011/)，但最新的可能是其中最疯狂的。该传言称，即将推出的 MacBook Pro 型号将采用英特尔的 Light Peak 高速数据连接端口，苹果标签为 Thunderbolt 高速 I/O。尽管苹果很少(我不能说出一个)推出未经验证的行业技术，但据说该端口与 Mini Displayport 共享一个连接器。甚至有一张照片声称是有问题的连接器，在所述端口旁边有一个小闪电。

[谣言始于昨天](https://web.archive.org/web/20230205041642/http://www.crunchgear.com/2011/02/22/intel-to-debut-light-peak-alongside-apples-new-macbook-pros/)英特尔的一份声明谈到了下周四的一项公告，这恰好是传闻中的苹果事件的同一天，关于“一种即将出现在市场上的新技术”。然后德国网站 [fsklog](https://web.archive.org/web/20230205041642/http://www.fscklog.com/2011/02/foto-specs-des-neuen-13-mbp-mit-core-i5-und-thunderbolt-anschluss.html) 发布了一份据称是即将发布的 13 英寸 MacBook Pro 的规格表。除了新的双核英特尔酷睿 i5 集成显卡 CPU 之外，还有 Thunderbolt。然后，MacRumors 得到了看起来完全相同的英文规格表，其中也提到了这一点。上面写着“Thunderbolt port 支持高速 I/O 和迷你 DisplayPort 设备。”

这个谣言也不是新的。早在 2009 年就有传言称苹果将实施轻击巅峰[。这显然没有成功，但苹果喜欢就实施哪些行业标准做出深思熟虑的决定。](https://web.archive.org/web/20230205041642/http://www.crunchgear.com/2009/09/30/yeah-apple-wasnt-integral-in-light-peak-transfer-technology/)[英特尔预测](https://web.archive.org/web/20230205041642/http://www.crunchgear.com/2010/04/15/intel-predicts-light-peak-to-replace-usb-3-0/)甚至在更快的 USB 规格上市之前，Light Peak 就会超过 USB 3.0。也许苹果公司也有同样的感觉，这可以解释为什么 USB 3.0 在任何 Mac 上都不可用。

尽管如此，认为这种新的革命性互连将与 Mini DisplayPort 共享一个端口还是有点疯狂。这将完全否定两者的好处，迫使用户选择一个或另一个模糊的技术。当然苹果不介意强迫用户使用适配器和加密狗。也许苹果官方的解决方案是使用大约 40 美元的分离器。

这可能会在下周的苹果发布会上实现。哦，等等，那也只是谣言。
