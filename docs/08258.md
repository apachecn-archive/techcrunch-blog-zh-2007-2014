# 脸书推出独立的 iPhone/Android Messenger 应用程序(这是白鲸)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/08/09/facebook-launches-standalone-mobile-messenger-app-and-it%e2%80%99s-beluga/amp/>

早在二月份，脸书[收购了 Beluga](https://web.archive.org/web/20230404092347/https://techcrunch.com/2011/03/01/facebook-beluga/) ，这是一家热门的群发信息初创公司，以拥有一个杀手级的 3 人团队而闻名。当时，TC 的另一位作者 MG Siegler 大声质疑脸书是否会发布一个独立的消息应用程序(Beluga 就是这样)，或者它是否有计划将技术和团队融入其最近改进的 Messages 产品中，该产品于去年秋天由[发布](https://web.archive.org/web/20230404092347/https://techcrunch.com/2010/11/15/facebook-messaging/)。

今天，我们有了答案:脸书正在为 iOS 和 Android 推出一款名为 *Messenger* 的独立应用，它将白鲸的最佳功能与脸书消息(以及脸书的 7.5 亿用户)深度融合。该应用程序将于今天在苹果应用商店推出，并应立即在安卓市场上提供。**更新**:这里是 [iPhone 应用](https://web.archive.org/web/20230404092347/http://itunes.apple.com/us/app/facebook-messenger/id454638411?mt=8)和 [Android 应用](https://web.archive.org/web/20230404092347/https://market.android.com/details?id=com.facebook.orca)的链接。(看起来该应用程序仍在向一些服务器推出，因此您可能会收到一个错误)。

如果你以前用过 Beluga，这个应用对你来说会非常熟悉。如果你没有看过，那也应该很熟悉——因为它很像网络版的脸书消息。

点击 Messenger，您将立即看到您最近在脸书上进行的对话列表(请记住，因为脸书的所有消息现在都统一为同一个平台的一部分，您将在此处看到最近的聊天和直接消息)。

基本的信息功能如你所料:点击你现有的一个对话，你就可以发送一条短消息。如果你的一个朋友(或朋友的朋友)在应用程序未打开时给你发了一条信息，你会立即收到推送通知。

您可以通过键入朋友的姓名轻松建立一个群，联系人来自您的脸书朋友列表和手机通讯录。给脸书朋友的信息显然是通过脸书信息发送的，给不在脸书的联系人的信息是通过短信发送的。您还可以重命名照片并将其附加到群消息主题中。

白鲸特有的特征也在这里。如果你的朋友在他们的信息中分享他们的位置，你可以在地图上标出你的小组成员的位置。如果你需要一些安静的时间，你可以一次静音一个小时，或者直到第二天早上 8 点(如果你的朋友让你很烦，你也可以完全静音)。

app 没有任何惊喜，但也不需要。我们的目标是让更多的人使用脸书消息，只需轻点一下你的收件箱(而不是藏在‘主’脸书应用程序里)，就能做到这一点。

从长远来看，脸书在信息方面显然有着雄心勃勃的计划。Messenger 不会很快取代你手机的短信应用程序，因为所有发送出去的短信都来自脸书的一个短码，而不是你的电话号码。但是如果你所有的朋友都在使用 Messenger(这是一个巨大的*如果】*)，那么你就不需要给他们发短信了。

今天的发布会也很引人注目，因为这是脸书首次为其一款产品发布独立的移动应用程序(历史上所有的移动应用程序都被称为“脸书”，包含了脸书能包含的所有功能)。这可能会为更多的独立应用铺平道路，比如正在开发的 secret [Photos 应用](https://web.archive.org/web/20230404092347/https://techcrunch.com/2011/06/15/facebook-photo-sharing-app/)。

最后，虽然信使团队(即前白鲸团队)表示白鲸目前将继续奔跑，但尚不清楚脸书将来会拿它做什么。
