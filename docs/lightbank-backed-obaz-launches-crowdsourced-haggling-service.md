# Lightbank 支持的 oBaz 推出众包砍价服务

我一直听说商店和零售商愿意在价格上与客户进行交易，包括许多连锁店，但我从来没有勇气与销售人员实际讨价还价。老实说，我喜欢打折，但我真的不擅长讨价还价。现在，随着 [oBaz、](https://web.archive.org/web/20230205001341/http://www.obaz.com/)的推出，我可以简单地让这项新服务帮我完成所有的砍价工作。总部位于芝加哥的 oBaz 宣布，由 Groupon 联合创始人埃里克·莱夫科夫斯基和布拉德·基威尔创办的风险投资公司 Lightbank 已经为这家初创公司提供了投资。投资金额没有披露。

oBaz 是 online bazaar 的缩写，它允许任何人创建自己的志同道合的买家群体，希望在同一产品或服务上获得好价格。你只需在 oBaz 上发布你想要的商品，等待人们加入这个群体(至少要有 25 个人在一个群体中为一件商品讨价还价)，然后让 oBaz 发挥它的魔力。oBaz 砍价师将接触商家或制造商，利用团队和他们自己的谈判专业知识来获得尽可能好的交易。

团体越大，奥巴兹讨价还价团队的筹码就越多。用户可以通过与脸书、推特和电子邮件分享来建立群组。每个小组都有 7 天的时间来建立一个临界质量。一旦交易谈妥，小组就不对公众开放，小组成员会通过电子邮件获得报价。然而，如果团体成员对交易不满意，他们没有义务购买。交易直接在商家网站或店内使用促销代码或打印的优惠券兑换。

当然，你可能想知道这是否真的有效。但在该公司的试运行中，会员能够在产品上获得相当不错的交易。最近 oBaz 上的团体交易包括:通常零售价为 900 美元的尼康相机套装优惠 300 美元；售价 130 美元的雷朋 Wayfarer 太阳镜仅售 80 美元，BOB 婴儿车售价 120 美元；在线 GMAT 课程五折优惠；购买 Longchamp 手提袋可免费获得价值 45 美元的零钱包；一根零售价为 105 美元的长曲棍球棒在 oBaz 卖 30 美元。

创始人布莱恩·菲乔和格雷格·卡普兰是 Lightbank 的前雇员。菲乔告诉我们，oBaz 的美妙之处在于它不受地理位置的限制，因此人们不需要位于某个特定的区域就可以达成交易。

就我个人而言，我会用 oBaz。正如我上面提到的，我没有很好的砍价技巧，但我确实喜欢打折(谁没有呢)。外包这项行动似乎是最好的选择。如果 oBaz 能让足够多的人开始与该网站互动，它就能真正起飞。就我个人而言，我看中了一副[的 Bose 降噪耳机。](https://web.archive.org/web/20230205001341/http://www.obaz.com/groups/276)还有其他人吗？
