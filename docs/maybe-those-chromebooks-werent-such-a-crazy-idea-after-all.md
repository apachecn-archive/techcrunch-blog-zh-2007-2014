# 或许这些 Chromebooks 毕竟不是一个疯狂的想法 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/02/03/maybe-those-chromebooks-werent-such-a-crazy-idea-after-all/>

当谷歌在 2010 年初推出基于 Linux 的 Chrome 操作系统和当年晚些时候推出的 Chromebook 试点项目时，大多数专家并不同意我们的 MG 西格勒的假设，即谷歌已经向微软投下了一枚核弹几年后，看起来微软肯定不会对 Chrome 操作系统感到害怕。尽管起步缓慢，但 Chrome OS 的势头似乎正在慢慢回升。

谷歌传统上对 Chromebook 的销售保持沉默，主要关注大型企业和教育安装。这显然也是 Chrome OS 生态系统目前的目标市场，但当宏碁表示其 199 美元的 C7 Chromebook 现在占其美国出货量的 5%至 10%时，很明显其中一些设备肯定也流向了普通用户。

宏碁显然在用其 199 美元的 Wi-Fi 专用设备玩价值游戏。三星也提供了 249 美元的 Chromebook 和功能更全的 449 美元版本。这两家公司在 Chromebook 计划的早期就与谷歌合作。然而，现在联想也开始参与这场游戏，其售价 429 美元的 ThinkPad X131e Chromebook 将于本月晚些时候上市。如果没有看到 Chromebooks 的发展势头，联想也不太可能进入这个市场。

[![Features of Chromebooks](img/a67cc52b15f1aa725558fbdfbda7cc6e.png)](https://web.archive.org/web/20221108213622/https://beta.techcrunch.com/?attachment_id=750132) 在低端，价格显然使这些 Chromebooks 成为有吸引力的笔记本电脑替代品，特别是考虑到即使是这些实惠的设备实际上也不觉得比通常的 Windows 笔记本电脑便宜，后者往往起价在 400 美元左右。自从谷歌去年转向更传统的窗口管理系统以来，使用 Chrome 操作系统的整体体验肯定也让许多用户感到更加舒适，这些用户以前只看 Chrome 就不感兴趣。

尽管如此，现在有超过 2000 所学校使用 Chromebooks，这些设备的真正市场似乎肯定是在教育领域。在过去一年左右的时间里，我从一些教育工作者那里听说，他们更喜欢让学生使用 Chromebooks，而不是 iPad(他们的学生可能会想得正好相反……)。它们不仅明显更便宜，并配有服务合同和管理员管理控制台，而且在许多教师和学校管理员看来，全键盘和更大的屏幕显然使它们成为有吸引力的设备。与传统的笔记本电脑相比，这种设备定期自动更新，而且非常安全，不会感染病毒，这也是一个很有吸引力的卖点。当然，对于谷歌来说，这也意味着这些孩子中的许多人将在谷歌的 Gmail 和 Google Drive 生态系统中成长，一旦这些学生继续上大学或找到工作，这肯定不会有什么影响。

[![bagofapps](img/c64c12a77dda538fa53883d6b2921626.png)](https://web.archive.org/web/20221108213622/https://beta.techcrunch.com/?attachment_id=750133) 谷歌推出 Chrome OS 的时候，可能有点超前。在发布会上，谷歌[表示](https://web.archive.org/web/20221108213622/http://mashable.com/2010/12/07/chrome-os-launch/)希望“重新思考网络的个人计算体验”，但 Wi-Fi 和 3G/4G 连接当时并不普遍，今天仍然如此，因此谷歌开始为 Chrome OS 添加更多离线功能和文件管理功能，并为其 Chromebooks 添加更大的硬盘。网络应用在 2010 年也没有那么强大，但现在，由于 HTML5 的强大，你真的不再需要传统桌面了。

谷歌显然会长期参与其中。它不会很快挑战 Windows 和 OS X 在笔记本电脑上的市场份额，但它肯定在过去两年里已经发展成为一个有趣的平台。也许谷歌将其浏览器与 Linux 和精简笔记本电脑相结合的想法并不那么疯狂。
