# 恐怖之处:Kindle Touch 3G 禁用了 3G 网络浏览

那些狡猾的亚马逊人！Kindle Touch 3G 将内置一个实验性的浏览器，但它只能在 Wi-Fi 上运行，不能在 3G 上运行。自从第一款 [Kindle](https://web.archive.org/web/20230203124201/https://techcrunch.com/tag/Kindle) 问世以来，就支持通过 3G 网络浏览，公平地说，这种体验简直糟糕透了。然而，人们喜欢免费的东西，显然这可能是一个交易破坏者。

在[的一个客户论坛](https://web.archive.org/web/20230203124201/https://techcrunch.com/tag/customer)上，亚马逊发言人写道:

> 我们对造成的混乱表示歉意。我们的新 Kindle Touch 3G 使您能够通过 3G 或 Wi-Fi 连接到 Kindle 商店，下载书籍和期刊，以及访问维基百科。Kindle Touch 3G 上的实验性网络浏览(维基百科之外)只能通过 Wi-Fi 进行。
> 
> 我们的 Kindle Keyboard 3G 将继续通过 3G 或 Wi-Fi 提供实验性的网络浏览。

你还会注意到，他们称老款 Kindle 为“Kindle 键盘”。希望他们不会为孩子做 Kindle 键盘。
