# 摩托罗拉·科尔:仅仅是一个心率监测器？

虽然我们[相当确定【Moto 上周将在平板电脑线上全力进军韩国，但看起来](https://web.archive.org/web/20230203053256/https://techcrunch.com/2011/08/05/is-the-next-motorola-tablet-called-the-kore-all-signs-point-to-yes/) [KORE 只是一个心率监测器和耳机组合的商标](https://web.archive.org/web/20230203053256/http://www.slashgear.com/motorola-kore-is-fitness-gadget-not-tablet-tips-trademark-11170999/)，可与[摩托罗拉](https://web.archive.org/web/20230203053256/https://techcrunch.com/tag/Motorola)手机配合使用。流浪者镇。

> 该商标仅限于以下商品和服务:“用于测量运动、个人能量消耗和身体活动水平的电子非医疗便携式身体活动监视器和健身设备及其配件，即耳机；用于开发和监控个人活动和锻炼计划、训练计划、训练目标并对所取得的结果进行反馈的软件”。

KORE 还可以是一个[音乐播放器/健身设备](https://web.archive.org/web/20230203053256/http://www.droid-life.com/2011/07/28/motorola-to-release-an-ipod-nano-wait-thats-not-right/)，这无疑会使它比一副耳机更有趣。很明显，这只是猜测和克里姆林宫学，所以我们将拭目以待。
