# 适用于 iPhone 的原生 AIM 和 IRC 应用

![buddylist.png](img/6a1e1b8192f3eb171f1467fe5734ba0e.png)

孩子们，不要再用 Safari IM 解决方案了。iPhone 有自己的原生消息应用程序。

首先，有一个为 iPhone 设计的超级测试版 AIM 客户端 [Apolloim](https://web.archive.org/web/20220928032821/http://apolloim.googlecode.com/files/ApolloIM.zip) 。看起来很管用，但是完全没有经过测试。接下来，你有 [ircm](https://web.archive.org/web/20220928032821/http://code.google.com/p/ircm/) ，一个 IRC 客户端，所以你可以告诉 n00bs 到 GTFO。两者都有完整的源代码，看看他们离测试版还有多长时间会很有趣。
