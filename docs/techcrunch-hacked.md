# TechCrunch 被黑了

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/01/26/techcrunch-hacked/>

正如一些人[注意到的](https://web.archive.org/web/20230222074402/http://www.techmeme.com/#a100126p8)，在太平洋时间周一晚上 10:30 左右，TechCrunch 网络的主要站点——beta.techcrunch.com——被黑客攻击并被重定向。该网站在晚上 11:30 短暂恢复，但很快又关闭了。截止到凌晨 2:00，网站已经恢复，看起来很稳定。

在这一点上，我们仍然在收集关于网站如何被破坏的信息，并将更新更多的信息。
