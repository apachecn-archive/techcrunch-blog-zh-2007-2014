# 招聘:富媒体移动广告标准。解决方法:ORMMA？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/11/18/wanted-a-rich-media-mobile-ad-standard-solution-ormma/>

T2 技术人员喜欢凭感觉飞行，不断重复和打破规则。但在某些时候，如果你想建立一个大企业，尤其是如果你想建立一个大的广告业务，你需要行业的稳定性和标准。

想象一下，如果你必须为拥有东芝和索尼的人编写电视广告，或者如果没有标准的广告单元和在线显示广告技术。你认为纽约媒体的大买家会发起大型活动吗？当然不是。如果没有标准化，移动应用程序和移动网络上的富媒体广告也不会成为数十亿美元的大生意。简而言之，这个行业需要一个真正的不可知论者 double click——一个不偏好广告网络而提供广告的公司。

但是谁来推动这种标准化呢？嗯，一群富媒体移动广告公司和出版商正在尝试一个新的联盟，名为 [ORMMA](https://web.archive.org/web/20221006171650/http://ormma.org/) 或开放富媒体移动广告倡议。它今天发布，创始成员包括[天气频道](https://web.archive.org/web/20221006171650/http://www.weather.com/)、 [Crisp 无线](https://web.archive.org/web/20221006171650/http://www.crispwireless.com/)和 [TringApps](https://web.archive.org/web/20221006171650/http://www.tringapps.com/) ，并得到 PointRoll 和 Jumptap 的支持。移动广告平台 Crisp Wireless 的首席执行官鲍里斯·弗里德曼(Boris Fridman)承认，移动营销协会(Mobile Marketing Association)目前正在对其进行审查，如果要取得成功，还需要该机构和互动广告局的认可。

MMA 将在未来三个月内发布一份关于该标准的白皮书。“我们必须消除摩擦，否则我们不会发展这个行业，”弗里德曼说。Crisp 显然是一家盈利性公司，所以这里有一个自私的议程，但它努力让足够多的其他人加入进来，并且通常在科技行业推动这个问题。这似乎对整个行业都有好处，不管 ORMMA 本身是否有吸引力。

弗里德曼说，我们的目标是成为一个厂商无关的 HTLM5 标准，可以在任何平台、任何手机上工作，不需要十几个 SDK 和不断的更新和调整。所有的出版商和开发者都可以加入并参与进来。重量级人物越多，它成为标准的可能性就越大。

但是显然需要一个标准。气象频道移动和数字应用高级副总裁 Cameron Clayton 说:“SDK solutions 不能勒索出版商。”“如果苹果或任何操作系统推出新版本，我们不能等待富媒体供应商或分析提供商来适应新版本的操作系统。我们需要尽快为客户提供最佳解决方案。ORMMA 让这成为可能。现在还为时尚早，但是 ORMMA 显示出了很大的前景，作为一个开源解决方案，它还允许业界将它做得更好，真正成为一个“标准”。我们还没有到那一步，但我们已经很接近了。”

在科技行业从来没有 100%的双赢，毫无疑问，一些通过消除现有混乱而蓬勃发展的大型移动广告网络将会对减少对高价中间商的需求的标准感到不安。其他人声称奥玛人在谈论一个根本不存在的问题。但底线是，如果广告商能够以最少的工作获得跨手机和平台的规模，每个人都将赚更多的钱，整个行业将倾向于一个公平的选择——否则会有太多的风险。与超过 200 亿美元的在线富媒体相比，移动广告现在大约是一个 5 亿美元的产业。智能手机应用有很多固有的优势——它们是无聊杀手，人们在无聊时更有可能回复广告。同样，当涉及到参与度和位置时，他们可以用一种甚至网络都无法做到的方式来衡量各种各样的事情。

现在，无论有没有标准，大多数应用程序都没有足够的眼球来证明大规模广告活动的合理性。但是如果没有更好的标准，世界上所有的眼球都不会让移动广告成为一个十亿美元的产业。
