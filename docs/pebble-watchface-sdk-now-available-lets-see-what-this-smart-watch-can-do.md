# Pebble Watchface SDK 现已上市，让我们看看这款智能手表能做些什么

Pebble 在承诺于 4 月第二周发布 SDK 后，正式发布了 SDK。这是合格的，如果只是，并伴随固件更新版本 1.1 为 PebbleOS。硬件的新软件更新支持使用 SDK 构建的定制手表表面，以及禁用背光和振动的新选项，以及对 iOS 错误的修复。

SDK 本身目前只是用于创建手表表面，而不是用于构建具有其他功能的应用程序，尽管我们可以看到一些有这些限制的创造性软件。Pebble 表示其运动应用 SDK 即将推出，这将有助于开发者模仿 RunKeeper 推出的各种功能[。](https://web.archive.org/web/20230131214002/http://forums.getpebble.com/discussion/3646/runkeeper-for-pebble)

这是第三方开发者第一次公开访问 Pebble 平台的开发者工具，因此它应该会给我们一个未来的提示。iOS 的固件更新修复包括一个使“允许 Pebble 通信…”对话框出现频率大大降低的修复，这是一个好消息，因为对于使用 iPhones 设备的人来说，这是一个非常令人讨厌的错误。
