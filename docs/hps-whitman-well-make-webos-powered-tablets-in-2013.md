# 惠普的惠特曼:我们将在 2013 年生产基于 WebOS 的平板电脑

惠普首席执行官梅格·惠特曼今天早上终于宣布了操作系统 WebOS 的命运，此前该公司曾宣布将停止所有运行 Palm WebOS 平台的智能手机和平板电脑的开发。实际软件的命运是[尚未确定。](https://web.archive.org/web/20221208031913/https://beta.techcrunch.com/2011/08/18/okay-hp-lets-make-some-lemonade/)惠普[今天](https://web.archive.org/web/20221208031913/http://www.hp.com/hpinfo/newsroom/press/2011/111209xa.html?mtxs=rss-corp-news)表示将会[让 webOS 代码开源。](https://web.archive.org/web/20221208031913/https://beta.techcrunch.com/2011/12/09/hp-to-keep-webos-alive-by-making-it-open-source/)

我们采访了惠特曼和惠普董事会成员马克·安德森，讨论了这一消息及其对公司的意义。她向 TechCrunch 解释说:“我们对 webOS 感到非常兴奋，并意识到最好的办法是开源这项技术，为它做出贡献并投资它。”“我们将利用社区的力量，使它成为其他操作系统的更好替代品。”

惠普今天表示，开发者、合作伙伴、惠普工程师和其他硬件制造商可以向市场提供持续的增强功能和新版本的 WebOS。惠普还将在不久的将来向社区贡献 WebOS 的应用框架 ENYO，以及用户空间的其余组件的计划。

你可以在这里看到惠特曼给惠普员工的全部备忘录[。](https://web.archive.org/web/20221208031913/https://beta.techcrunch.com/2011/12/09/meg-whitman-hp-webos-memo/)

那么平板电脑和 WebOS 呢？Andreessen 表示，由于开源模式，我们将拥有基于 webOS 的平板电脑的未来，并表示惠普将成为开发 webOS 平板电脑的公司之一。惠特曼表示，这可能不会在 2012 年发生，但很可能会在 2013 年发生。她在几周前提到了这一点，但惠普将在 WebOS 操作系统上开发未来的硬件产品，这确实很有趣。

**更新:**惠普想澄清惠特曼说惠普可以在 2013 年生产基于 WebOS 的平板电脑。

虽然我们知道惠普已经[停止在手机上开发](https://web.archive.org/web/20221208031913/https://beta.techcrunch.com/2011/08/18/its-official-hp-kills-off-webos-phones-and-the-touchpad/)WebOS，但有趣的是，该公司仍在继续押注平板电脑上的操作系统。

2012 年，惠特曼正计划推出 Windows 8 平板电脑。“我们将继续投资于现有的平板电脑生态系统”，近期“将在 Windows 上下大赌注。”

正如我的同事马特·伯恩斯(Matt Burns)最近写道的那样，在经历了一段艰难时期后，惠普在惠特曼的领导下看起来更好了。在[执掌](https://web.archive.org/web/20221208031913/https://beta.techcrunch.com/2011/09/22/its-official-at-hp-apotheker-is-out-meg-whitmen-named-president-and-ceo/)之后，惠特曼[决定](https://web.archive.org/web/20221208031913/https://beta.techcrunch.com/2011/10/27/hp-reconsiders-spinning-off-pc-division-personal-systems-group-will-remain-part-of-the-company/)将个人系统部门留在惠普。该公司第四季度的业绩超过了华尔街的预期。

关于惠普是否可以成为消费者和企业公司，惠特曼认为该公司可以继续做到这两者，并表示该公司目前的优势是它能够在这两个产品领域取得成功。“如果你在街上问任何一个人惠普以什么闻名，那一定是电脑，”她说。
