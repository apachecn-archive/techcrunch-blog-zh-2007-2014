# 硅谷继续用它可怕的说唱折磨人们

[![Screen Shot 2013-01-23 at 12.55.00 PM](img/7f0bcd87d6f9c2c402202b555a4f5a41.png)](https://web.archive.org/web/20221204082546/https://beta.techcrunch.com/wp-content/uploads/2013/01/screen-shot-2013-01-23-at-12-55-00-pm.png) 需要有一个词来形容什么时候有人可以有效地模仿自己，因为这种倾向在社交媒体时代被夸大了。

每个人都知道有人整天在脸书上发布鼓舞人心的名言，或者名字在 Path 上不断下降，或者在 Twitter 上使用愚蠢和武断的标签。#Twitter 上的#标签#

没有人可以幸免，事实上，硅谷作为一个纯净的回音室，似乎一直在鼓励这种自我模仿，不管有没有意识到(这并不总是[清晰](https://web.archive.org/web/20221204082546/http://www.youtube.com/watch?v=fbdd_Fasz0k))。也许这些比喻中最被滥用的一个是“程序员、博客作者和投资者说唱”；有些人[成功了](https://web.archive.org/web/20221204082546/https://beta.techcrunch.com/2012/12/19/nothing-says-happy-holidays-like-first-round-capital-gangnam-style-and-carly-rae-jepson/)，有些人[没有](https://web.archive.org/web/20221204082546/http://www.youtube.com/watch?v=QxVlBlhVuY8)，但不管怎样，我们做了多少都是不可思议的。

该系列中最新的是 [TechCrunch Disrupt 决赛选手](https://web.archive.org/web/20221204082546/https://beta.techcrunch.com/2012/09/10/undrip-takes-all-the-nonsense-out-of-your-social-feeds/) [Undrip](https://web.archive.org/web/20221204082546/http://undrip.com/) 为慈善说唱(几乎总是为慈善，除了为[筹资](https://web.archive.org/web/20221204082546/https://beta.techcrunch.com/2011/11/16/startup-rap-angel-funding-undrip/)或[crunches](https://web.archive.org/web/20221204082546/https://beta.techcrunch.com/events/crunchies-2012/event-info-2/)[投票](https://web.archive.org/web/20221204082546/https://beta.techcrunch.com/2013/01/10/obligatory-grindr-joke/))。
