# 谷歌推出计算引擎，挑战亚马逊网络服务

谷歌在过去十年里建立了大量的基础设施来支持自己的服务和应用。现在，任何人都可以利用所有这些投资，因为谷歌正在推出一系列基础设施即服务产品，可以与行业巨头亚马逊网络服务公司(Amazon Web Services)竞争。

谷歌已经建立了一个最大的数据中心和连接这些数据中心的网络。2008 年，谷歌创建了 App Engine，工程副总裁 Urs Holzle 是谷歌首批 10 名员工之一，他说现在支持超过 100 万个活跃应用程序和每天 75 亿次点击。这是世界上最大的公共 NoSQL 数据存储。

Holzle 展示了名为 Google Compute Engine 的新产品，该产品将提供 Google 规模的 Linux 虚拟机。目标是为应用程序开发人员提供启动虚拟机的能力，以及高性能存储和通过谷歌网络与用户的连接。
