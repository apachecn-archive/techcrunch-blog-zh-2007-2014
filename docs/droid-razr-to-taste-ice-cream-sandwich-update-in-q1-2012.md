# Droid Razr 将于 2012 年上半年推出冰激凌三明治升级版

摩托罗拉的 Droid Razr 昨天引起了很大的轰动，部分归功于其令人印象深刻的硬件，但由于 T2 冰淇淋三明治，它运行的 Android 版本现在看起来有点古怪。不过不用担心:如果你在 11 月份收到 Droid Razr，你应该会在 2012 年初得到 Android 4.0 [更新。](https://web.archive.org/web/20230203155346/http://www.techradar.com/news/phone-and-communications/mobile-phones/motorola-razr-android-4-0-update-confirmed-for-2012-1035086)

冰淇淋三明治发布窗口的消息来自摩托罗拉移动公司投资组合和产品管理 SVP Alain Mutricy。他在柏林的新闻发布会上发言时似乎非常积极，提到 Droid Razr 用户将在 2012 年 Q1 的某个时候收到更新，摩托罗拉正在努力让 Droid Razr 的标志性功能在冰淇淋三明治中发挥作用。

这些功能中包括 Motocast，这是一种将媒体和内容从 PC 直接传输到 Droid Razr 的绝妙方式。谁会忘记智能操作(Smart Actions)这款根据用户定义的情况管理设备设置的上下文敏感应用呢？没有它们，Droid Razr 就不是 Droid Razr 了，所以需要一点额外的时间来让它工作。

对于那些想要(或希望)看到摩托罗拉的硬件与谷歌最新最棒的操作系统相结合的人来说，这是一个令人失望的消息，但至少 Droid 的忠实用户有所期待。

**更新:**一位摩托罗拉代表已经联系我们，澄清 ICS 将在 2012 年上半年的某个时候进入美国 Droid Razrs，而不是最初报道的第一季度。
