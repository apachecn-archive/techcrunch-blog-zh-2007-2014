# 你是怎么错过这个疯狂的 iPhone 控制的咖啡水龙头的？

一个朋友刚刚给我发了这个，声明“这是疯子。”的确是——一个咖啡龙头！！！)由丹麦制造商 [Scanomat](https://web.archive.org/web/20220930221124/http://www.scanomat.com/) 开发，可以让你通过 iPhone 应用程序冲泡咖啡。

我不管它是不是假的~~而且它已经[被覆盖](https://web.archive.org/web/20220930221124/http://www.google.com/search?aq=f&sourceid=chrome&ie=UTF-8&q=iPhone-Controlled#sclient=psy-ab&hl=en&source=hp&q=topbrewer&pbx=1&oq=topbrewer&aq=f&aqi=g-s4&aql=&gs_sm=e&gs_upl=53501l63008l0l63409l28l15l10l2l4l0l315l2585l2.8.4.1l27l0&bav=on.2,or.r_gc.r_pw.,cf.osb&fp=681ef6e9afe787e2&biw=1019&bih=708)到处~~，我要一个。

顶级酿造器有两个隐藏的研磨机，除了 15 秒钟的咖啡和 25 秒钟的浓缩咖啡，你还可以制作果汁或热可可。

它通过 iPhone 和 iPad 控制，并照看你的孩子。好吧，最后一个我是开玩笑的。哦，它能自我清洁，真的，而且不知何故同时节省了能源。这东西比男朋友强。
