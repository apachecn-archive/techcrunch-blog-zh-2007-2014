# 脸书确实很难留住高管|技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/03/31/facebook-sure-does-have-trouble-keeping-execs/>

脸书将在不到两年的时间里任命第三任首席财务官，该公司今天通过《华尔街日报》的“T2”泄密事件宣布了这一消息。[空手从迪拜回来的](https://web.archive.org/web/20230404134421/http://www.crunchbase.com/person/gideon-yu)[T4，就是](https://web.archive.org/web/20230404134421/https://techcrunch.com/2008/10/31/facebooks-growing-problem/)[出来的](https://web.archive.org/web/20230404134421/https://techcrunch.com/2009/03/31/confirmed-facebook-loses-cfo-gideon-yu/)。寻找他的继任者的工作现在开始了。

脸书表示，这是为了获得一名具有上市公司经验的首席财务官(*“我们已经聘请了斯宾塞·斯图亚特(Spencer Stuart)来领导我们寻找新的首席财务官的工作，并将寻找具有上市公司经验的人。”*)。**这完全是胡说八道(而且是经过深思熟虑的胡说八道)，**因为俞在 YouTube 上呆了一小段时间，在红杉资本(Sequoia Capital)工作更短时间后，担任了雅虎的财务主管兼金融。这是一家非常公开的公司。

那他为什么要离开？我们现在不得不进行推测，但或许他不相信该公司能以董事会希望的估值获得新一轮融资。也可能是其他上千件事中的任何一件。但这肯定不是因为他没有上市公司的经验。

不到两年前首席执行官马克·扎克伯格(Mark Zuckerberg谈到俞*“我认为我们能够在这里招募他是一次政变……他太棒了。”*

我想知道他为什么没有加上*“我只是希望为了他，脸书不会太成功，因为如果我们在某个时候进行首次公开募股，他就出局了。”*
