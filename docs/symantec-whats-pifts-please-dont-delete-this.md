**更新:**鉴定 PIFTS。一条评论将我们引向[这一页](https://web.archive.org/web/20221007231958/http://www.thetechherald.com/article.php/200911/3179/Symantec-explains-PIFTS-and-debunks-conspiracy-theories-Update)，它解释说“产品信息框架故障排除程序”是“我们为旧产品发布的诊断补丁”,并且没有经过赛门铁克的正确签名就发布了，提示了防火墙的响应。至于黑幕论坛删帖？他们说大约在同一时间，一个 4chan 垃圾邮件发送者进入了系统。很方便，但是他们说他们的政策是不删除。各位，谜团解开了！回家吧！

我不知道发生了什么，但是我的朋友迈克刚刚把这个发给了我，我希望把这个消息传播出去。昨天的某个时候，他收到了一个诺顿程序警报，说:“PIFTS 正在试图访问互联网。”作为一个书呆子，他在 IP 上做了 nslookup，它回到了 SwapDrive，这是赛门铁克拥有的公司。然后，他在诺顿/赛门铁克论坛上发帖，但他的帖子被删除，帐户被终止。
