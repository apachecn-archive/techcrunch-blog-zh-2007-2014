# 诺基亚:在智能手机领域，我们已经屈服于 Windows Phone 8——“我们将创新、智胜、超越所有人”

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/11/14/nokia-weve-stooped-in-smartphones-to-conquer-with-windows-phone-8-we-will-innovate-outmanoeuvre-outsell-everyone/>

诺基亚正在欧洲为其新的 Windows Phone 8 设备 Lumia 920 和 820 打广告，上面印有“人人都爱东山再起”的字样。你不能指责曾经强大的移动之王对其目前在智能手机排名中的卑微地位没有幽默感——与其说是无可争议的冠军，不如说是平庸。房间里的 800 磅大猩猩当然是谷歌的 Android 操作系统——根据 IDC 的估计，它在第三季度占据了全球市场的四分之三(75%)——相比之下，Windows Phone 的份额仅为 2%(加上微软的旧 Windows Mobile 操作系统)。这个数字包括了所有使用 Windows Phone 的原始设备制造商，而不仅仅是诺基亚——从诺基亚微小的市场份额中多刮了一点点。诺基亚的传统操作系统 Symbian 仅占 2.3%的市场份额，仍然高于 Windows Phone。

尽管如此，第三季度恰好在 Windows Phone 8 发布之前下跌，因此对于该操作系统或诺基亚来说，这永远不会是一个出色的季度——只剩下一系列孤立的 Windows Phone 7.5 设备可供销售。因此，尽管诺基亚的日子不可否认地不好过，但该公司的西欧副总裁康纳·皮尔斯(Conor Pierce)却提出了强硬、充满斗志的言论。随着闪亮的新 Windows Phone 8 硬件的销售，诺基亚正在旋转，它已经屈尊-最终-征服。

### Windows Phone OEMs 厂商越多越好

皮尔斯说诺基亚确信它已经度过了 WP 最糟糕的时期。在经历了一年左右的低迷期后——主要靠自己在平台上挥汗如雨——皮尔斯说，其他手机制造商突然再次参与进来的事实表明，更公平的风将很快扬起 Windows Phone 的风帆。他告诉 TechCrunch，其新 WP8 Lumias 的销售“经过许多许多月的规划，有了一个真正坚实的开端”。“我们的设备和 Windows 生态系统从未像现在这样令人兴奋。很高兴看到我们的竞争对手来到谈判桌前，因为这将真正有助于传播信息和生态系统意识。还有很多工作要做，但看起来不错。”

“我们认为 Windows Phone 的机会才刚刚开始。现在竞争已经进入市场，这当然会带来压力——健康的压力，因为它会迫使所有参与者为了消费者和市场的利益而创新，”他补充道。

Pierce 甚至暗示，如果微软推出自己的 Windows Phone 手机——传言中的“Surface Phone”——诺基亚甚至不会皱起眉头。当被问及如果微软决定开放自己的手机硬件，诺基亚是否有准备好的战略时，pierce 坚持他的观点，即“这个生态系统中的参与者越多，人们越健康，因为他们有更多的选择”。尽管在后续评论中，他言外之意呼吁微软不要涉足手机制造业务。

“我们需要做的是确保我们继续在我们的产品组合中推动[Windows Phone]生态系统的创新，继续推动微软内部的信心，即我们只会为 WP8 做最好的工作——这是我们的愿景，”他说。“与我们的其他竞争对手不同，我们在这背后没有任何干扰。我们将继续前进，你在[Lumia] 920 和 820 上看到的只是我们带来的差异化的开始。”

### Windows 8 让 WP8“大不相同”

目前吹向诺基亚方向的最大能量冲击波是 Windows 8 的发布，与 Windows 7 + Windows Phone 7 不同，Windows 8 与 Windows Phone 8 有着相同的外观(磁贴)和感觉(触摸)。“在未来 12 个月内，我们预计全球将有 3 亿或 4 亿人通过 Windows 升级或新手机销售或平板电脑等体验 Windows 平铺界面，这与我们过去的体验截然不同，”皮尔斯指出。

他还认为，Windows Phone 7 有效地为运营商预热，让他们为销售 Windows Phone 8 而努力。“商店和呼叫中心的员工现在对我们正在做的事情有了更强、更深的熟悉和接受，因此我们已经打破了这种局面，现在我们能够营造这种势头，”他声称，并补充道:“(销售 Windows Phone 8 设备)符合(运营商)的利益，因为拥有强大的第三种选择符合人们的利益。”。这不是一个健康的市场，也不是一个由两个过于强大的玩家驱动的行业。它扼杀创新，扼杀选择。"

“当你试图在任何行业建立一个全新的东西时，这不会在一天之内发生，”他补充道。“我们现在知道，要让人们意识到需要考虑，让考虑变成偏好，需要付出多大的努力。这需要付出巨大的努力，显然，作为制造商，诺基亚在过去 12 个月里承担了所有的重任。”

第三个生态系统的论点之前已经提出过，当时 Windows Phone 7 进入了一个由 Android 和 iOS 主导的市场，并且它们仍然占据主导地位。但是皮尔斯认为 Window 8 的推出改变了游戏——给了运营商一个更直接的故事来推销。他还指出，诺基亚在信息传递和如何选择将资源集中在 Windows Phone 8 上时是故意选择的——例如，为其旗舰产品 Lumia 920 选择一些[运营商独家服务，这似乎是一家有销量问题的公司目前最不需要的东西。](https://web.archive.org/web/20221007155310/https://beta.techcrunch.com/2012/10/04/goodbye-lumia-920-the-att-exclusive-superphone-will-probably-flop/)

### “一切都是为了简化信息”

“我们知道我们的立场。我们知道还有多少工作要做，才能取得我们所有人都渴望的成功，所以在全球推出战略时，我们必须非常有选择性，就像任何参与者一样， 任何行业中的任何挑战者——所以我们一直非常关注和选择我们进入的市场，在这些市场中，我们还与我们认为最忠诚的合作伙伴合作，我们认为这些合作伙伴不仅为发布活动，也为长尾提供了最大的机会，”当被问及为什么要重新推出运营商专属服务时，他说。

“关键在于执行，关键在于简化信息，让信息真正与更广泛的受众相关并产生共鸣。在这方面，像任何公司一样，我们需要非常关注我们的投资方向，”他补充道。

可以说，诺基亚最近的金融表现是一个更迫切的关注理由。上个月，[公布了第三季度的另一项重大损失](https://web.archive.org/web/20221007155310/https://beta.techcrunch.com/2012/10/18/another-big-loss-for-nokia-in-q3-754m-operating-loss-only-2-9m-lumias-sold-as-it-waits-on-wp8-launch/)(7.54 亿美元)，并报告现金储备缩水，截至该季度末，净现金降至 47 亿美元。对于今天更精简的、更吝啬的和(主要)以微软为动力的诺基亚来说，用巨额收入和利润资助跨多个平台的内部软件和硬件的辉煌时代已经成为过去。

### 与微软的关系“非常健康”

Pierce 将诺基亚与微软的关系描述为“非常健康”,并声称当 Redmond 在 6 月份推出 Windows Phone 8 时，诺基亚没有受到不利影响——让它出售一系列明显“过时”的 Windows Phone 7 手机。相反，他委婉地谈到任何操作系统都有“自然进化”，并补充道，“事实上，我认为(WP8 的新闻)给了(WP)一针强心剂”。Pierce 还声称，自从 Windows 8 和 WP8 发布以来，诺基亚实际上卖出了更多的 Windows Phone 7 手机。“我们目前的 Lumia 系列——特别是 800 和 610——从未比现在卖得更好，”他声称。

(在这一点上，值得注意的是，Pierce 没有提到诺基亚不得不在各种市场上对 Lumia 800 等 WP7 设备打折——所以他看起来是在掩盖 Windows Phone 引发的另一个财务痛苦。)

Pierce 继续说道:“我们正在几个方面与微软密切合作:1)合作伙伴参与，2)零售执行或渠道执行——确保信息非常简单，因为在这个行业中，对于诺基亚和所有制造商，我们可能会列出一长串我们想要谈论的事情。有了微软和他们带来的平台生态系统…有很多好东西可以谈论。关键是我们如何将其浓缩成一个非常可靠、相关的信息，并针对正确的受众—消费者或企业。

“除此之外，我们还在应用生态系统方面与他们密切合作。我们知道，在保护市场上的一些杀手级应用程序方面，我们还有一些工作要做，但由于开发人员的参与欲望，我们有信心继续推动这一进程。”

### 在地面上滑行

在[微软发布 Surface](https://web.archive.org/web/20221007155310/https://beta.techcrunch.com/2012/10/16/microsoft-surface-rt-priced-32gb-for-499-without-touch-cover-599-with-64gb-for-699/) 之后，有传言称诺基亚不得不搁置 Windows 平板电脑计划，Pierce 对此不予置评——如果属实，这将是微软推动的对其业务的又一次破坏。“今天，我们没有也不打算代表诺基亚宣布任何平板电脑战略。我不知道这种说法(诺基亚的平板电脑计划因 Surface 推迟)。我们知道我们要去哪里，我们有一个非常强大的投资组合，发生的任何事情都不会分散我们对我们想做的事情的注意力，”他说。

当被问及 Window Phone 7 的未来角色——以及诺基亚是否可以扩大其对这一版本 WP 的使用，甚至可能将其作为其智能设备组合中非常低端(诺基亚[Asha](https://web.archive.org/web/20221007155310/https://beta.techcrunch.com/2012/09/25/nokia-unboxes-cheapest-touchscreen-phones-yet-series-40-based-asha-308-asha-309/)设备目前所处的位置)的 40 系列操作系统的替代产品时，Pierce 没有立即放弃这一想法，但表示简单的 Windows Phone 不是诺基亚或微软现在想要发出的信息。“所有原始设备制造商继续推动体验降低价格点符合(WP)生态系统和微软的利益，但最重要的是，我们不会危及体验的质量:这一点至关重要，因为目前建立这种与人和企业同步的势头非常重要，这需要巨大的努力和专注。”

### 引领 Windows Phone 8 的未来

所以现在，正如你所料，诺基亚正将其剩余的精力集中在 Windows Phone 8 上，并确保其努力在新来者中脱颖而出。

在讨论诺基亚在未来 12 个月可能如何发展其 WP 差异化努力时，Pierce 给出了一些关于导航的沉重暗示——注意:这次采访发生在诺基亚宣布 [Here Maps](https://web.archive.org/web/20221007155310/https://beta.techcrunch.com/2012/11/13/nokia-introduces-here-a-rebranding-of-its-maps-service-that-goes-head-to-head-with-google/) 及其 [Earthmine 收购](https://web.archive.org/web/20221007155310/https://beta.techcrunch.com/2012/11/13/nokia-steps-up-in-its-mapping-quest-buys-earthmine-for-3d-street-level-imaging/)之前——Pierce 告诉 TechCrunch:“在设计之外……以及我们正在展示的一些东西， 我们还通过我们的 PureView 和卡尔·蔡司镜头以及我们正在构建的一些诺基亚相机镜头实现了令人难以置信的成像创新，所以如果你采用这种技术，并将其与我们的导航(我们的诺基亚地图平台)相结合，那么我们就有了非常坚实的东西，就对消费者的意义而言，我们只是冰山一角。

“例如，在导航方面，五分之四的汽车目前都在使用诺基亚地图，我们已经有一些非常大的参与者，如梅赛德斯、大众、福特等购买了这一产品，并使用诺基亚地图作为他们的导航平台，然后我们还——记住，在整个 Windows 生态系统中，诺基亚地图将是导航、位置商务平台，这真的很好，它对所有参与者开放，无论是原始设备制造商，无论是您想购买的任何设备或软件，但重要的是*我们拥有该资产*。

“我们将做的，我们将继续推动的是带来这些差异化点，无论是在设计上，还是在想象上，还是在导航上——所以我认为这是一个‘观察这个空间’的问题。我们将在生态系统中创新，超越策略，超越所有人，而不仅仅是我们的竞争对手，因为这才是真正推动这个生态系统的因素。”
