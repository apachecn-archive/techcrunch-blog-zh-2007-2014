# 图像广告网络 GumGum 从 NEA 筹集了 700 万美元，第一轮和 GRP

图像广告网络 GumGum 已经从 NEA 筹集了 700 万美元的新资金，GRP 和首轮资本参与了该轮投资。这使得 GumGum 的资金总额达到 1100 万美元。

GumGum 的图片广告平台拥有 1.1 亿独立访客，每月提供数十亿张图片广告。GumGum 于 2007 年由 Ophir Tanz 共同创立，与时代华纳、赫斯特公司、论坛报、TMZ、Gannett 和 Glam Network 等在线出版商合作，通过在图像上叠加文本、横幅或 Flash 广告单元，将照片转化为互动体验。

对于广告客户来说，GumGum 专有的图像识别和定位技术旨在通过高度针对性的活动来获得参与度。GumGum 表示，其广告活动的点击率是传统广告网络的 20 倍。该公司自 2011 年初以来一直盈利。

新的资金将用于发展广告销售基础设施和团队，开发专有的图像识别、广告服务和广告匹配技术，并扩展到其他地区。

GumGum 面临来自 [Luminate](https://web.archive.org/web/20230203065205/https://techcrunch.com/2011/07/27/adsense-for-images-pixazza-rebrands-as-luminate-launches-platform-for-in-image-apps/) 和 [Image Space Media 的竞争。](https://web.archive.org/web/20230203065205/http://www.crunchbase.com/company/picad-media)
