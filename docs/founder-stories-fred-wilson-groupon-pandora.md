# 弗雷德·威尔逊解释了他为什么不投资 Groupon 或 Pandora

很少有风投公司现在比弗雷德·威尔逊更有底气。他的公司 Union Square Ventures 是 Twitter、Zynga、Foursquare、Tumblr、Etsy、Clickable 和 [more](https://web.archive.org/web/20230202235240/http://www.crunchbase.com/financial-organization/union-square-ventures) 的投资者。在这一集的*创始人故事*中，他与主持人克里斯·迪克森谈论联合广场的投资主题已经从追求所有的网络应用程序转变为“建立一个庞大的参与用户网络”的公司(看上面的视频)。

它必须是一个大的网络和参与的用户。根据这个要求，他说他不会投资 Pandora(虽然这是几周前录制的，但昨天刚刚完成首次公开募股),因为 Pandora 的听众只是袖手旁观。他说:“尽管潘多拉是一家伟大的公司，但用户在潘多拉上什么也不做。“同样，他也不会投资 Groupon。不是因为他认为这是一个糟糕的行业，只是这不是他关注的领域。”“Groupon 是一个广告网络，”他说，“我们不会投资它。“在电子商务领域，他认为市场(如 Etsy)确实符合他的定义，但像 Diapers.com 或捷步达康这样的公司不符合。威尔逊还提到了一些逃脱的公司，他希望自己投资了这些公司: [AirBnB](https://web.archive.org/web/20230202235240/http://www.airbnb.com/) 和 [Bump](https://web.archive.org/web/20230202235240/http://bu.mp/) ，他把这些公司输给了红杉。

迪克森和威尔逊还谈到了创始人和风投之间的关系，以及当一家公司需要扩大规模时注入资本的重要性，就像 Twitter 和 Tumblr 一样。迪克森回忆起一项研究，该研究表明，风险投资离一家公司越远，它做得越好。威尔逊同意这一点，并以 Twitter 和 Zynga 为例，这两家公司都位于加州，而 Union Square 位于纽约(作为两家公司的早期投资者，这给了两家公司大约一年的最少干预时间)。

在下面的视频片段中，Wilson 和 Dixon 深入探讨了风险投资业务的机制，并谈论了 Union Square 的新[、1.65 亿美元的机会基金](https://web.archive.org/web/20230202235240/https://techcrunch.com/2011/01/15/union-square-network/)、风险投资公司向投资者收取的“癌症管理费”，以及为什么让初创公司创始人在 IPO 或出售前把钱拿出来可能是一件好事。“这允许他们冒更多的风险，”威尔逊说，“因为不是所有的事情都取决于公司。”
