# 谷歌推出所有更新之母:一个更大的搜索框

谷歌通过算法和结果显示的方式不断更新它的搜索体验。但当涉及到搜索框本身时，它基本上不去管它。当然，它为建议的结果增加了[下拉框](https://web.archive.org/web/20221209131530/http://www.beta.techcrunch.com/2009/05/20/google-suggest-adds-hyperlinks-personalization-and-yes-ads/)，但是这个框本身仍然是一个薄薄的输入框。但现在看来，谷歌可能正在考虑改变。

今天，在使用 Safari 浏览器时，我们注意到搜索框变大了，按钮变成了方形。我们不是唯一注意到变化的人。这似乎是一个微不足道的更新，但请记住，这是谷歌，数百万人每天使用它来做搜索，一个用户界面的变化，无论多么小，都不是微不足道的。

注意现在的输入框与谷歌标志相比有多大(见下图)。

还要记住，谷歌会对它所做的每一个微小的改变进行数据检查，以确定这种改变是否值得。看起来这就是它现在正在做的，因为许多其他人还没有看到变化。

除了 Safari 之外，还有人在其他浏览器中注意到这个变化吗？

更新:额外的报道称他们在火狐的一些版本中也看到了它。

**更新 2** :不亚于谷歌负责搜索产品和用户体验的副总裁[玛丽莎·梅耶尔](https://web.archive.org/web/20221209131530/http://www.crunchbase.com/person/marissa-mayer)，刚刚[就此事发了](https://web.archive.org/web/20221209131530/http://googleblog.blogspot.com/2009/09/now-s-u-p-e-r-sized.html)。

迈尔说:

> 也就是搜索。对我们来说，搜索一直是我们的重点。从今天开始，你会注意到在我们的主页和搜索结果页面上，我们的搜索框越来越大。虽然这是一个非常简单的想法，甚至更简单的改变，但我们对此感到兴奋——因为它象征着我们对搜索的关注，因为它使我们干净、简约的主页更容易使用，也更有趣。
