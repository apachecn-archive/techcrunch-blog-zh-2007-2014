# Glam Media 的应用平台上线 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/11/18/glam-medias-application-platform-goes-live/>

[![](img/3a39a763e7ff4f27fdfb48bdd0e2b199.png)](https://web.archive.org/web/20221006040920/http://www.glamapps.com/)

[Glam Media](https://web.archive.org/web/20221006040920/http://www.glammedia.com/) 向公众推出了名为 [Glam Apps Atako](https://web.archive.org/web/20221006040920/http://www.glamapps.com/) 的应用平台。该平台最初于 7 月[发布](https://web.archive.org/web/20221006040920/http://www.glammedia.com/about_glam/news/2008/07/22/glam-media-announces-glam-apps-platform%E2%80%94code-named-%E2%80%9Catako%E2%80%9D-enabling-a-new-class-of-widgets-and-applications-designed-for-monetization-for-vertical-content-networks/)，直到现在仍处于私人测试阶段。

每个应用程序都允许博客作者和网站所有者在他们的网站中快速实现新的功能，并提供联合内容、添加富媒体和通过评论和投票增强帖子的选项。发布时可用的应用程序包括 Sphere、BuzzFeed、Meebo、PollDaddy、PicApp、JS-Kit 和 Kwanzoo，以及一个内部开发的名为 GlamTV 的视频交付小工具。Glam 还在构建平台时考虑到了货币化，提供了一个集成的货币化系统，内置了对 rev-shares 的支持。Glam 平台由 Google Gadgets 构建，也支持 OpenSocial 和 OpenID。Glam 首席执行官 Samir Arora 表示，该平台本身也将是开源的。

这些新的应用程序似乎和 WordPress 插件一样是博客增强器，但没有脸书、MySpace 或 LinkedIn 等应用程序平台上的社交功能。尽管如此，该平台仍可能是 Glam 的稳定收入来源，因为该公司将从每个应用程序获得的收入中抽取一部分。

该平台与最近发布的 SproutMixer 有相似之处，SproutMixer 也让品牌和其他人为出版商开发 Flash 应用/广告。

Glam Media 是一家多元化的公司，提供主要面向女性的网站网络(尽管它最近推出了面向男性的网络)，以及向其网络上的其他博客和附属机构提供广告的广告网络。

尽管在过去的一年里我们已经对 Glam 采取了很多措施，但我不得不说。他们咄咄逼人，不断挑战货币化的极限。如果你是一个迷人的出版商，让我们知道这是如何为你工作的。
