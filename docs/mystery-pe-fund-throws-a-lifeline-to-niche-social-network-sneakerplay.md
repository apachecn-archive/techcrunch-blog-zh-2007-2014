# 神秘私募股权基金为利基社交网络 Sneakerplay 抛出了一条救命稻草

一家新的私募股权基金已经接手了为运动鞋迷提供的社交网络 [Sneakerplay](https://web.archive.org/web/20221207082845/http://www.sneakerplay.com/) ，去年年底当它的创始人几乎放弃这项服务的时候，我们[放弃了这项服务(但保持网站在线，希望有买家来接手)。](https://web.archive.org/web/20221207082845/http://www.beta.techcrunch.com/2008/12/02/sneakerplay-social-network-for-sneaker-heads-shuts-down/)

该基金名为 [Keroch](https://web.archive.org/web/20221207082845/http://www.keroch.com/) ，已经收购了 Sneakerplay 的资产和知识产权，并计划将这个利基社区网站发展成为一个更具相关性的产业。交易的条款没有披露，但 Sneakerplay 一直是一个完全自举的企业，所以我们可以有把握地假设创始人对出售感到满意，特别是因为他们基本上已经离开这个项目一段时间了。

令我震惊的是，每个人似乎都在尽力隐瞒私人股本基金相关人员的姓名。这个网站不会教你任何关于这方面的东西，除了这个团队由四个合伙人组成的花絮，他们“在技术/互联网行业拥有超过 30 年的综合经验，包括像雅虎这样的大公司！敬小型初创企业。”。到目前为止，我们唯一想到的名字是[布莱恩·罗森伯格](https://web.archive.org/web/20221207082845/http://www.linkedin.com/in/brianrothenberg)，雅虎前产品经理。房地产。

无论如何，收购 Sneakerplay 是 Keroch 的第一步，他们计划通过赞助和路线图上的现场广告，将 sneakerheads 的小社区变成一个相关的利基社交网络，拥有更多功能和清晰的盈利路径。

顺便说一句，当我放弃这家公司时，有些人很不高兴(现在我当然很高兴地承认他们是对的)，但看起来这篇文章至少让 Sneakerplay 团队获得了与潜在收购者进行竞购战的必要关注。联合创始人[罗布莱·贾马](https://web.archive.org/web/20221207082845/http://www.crunchbase.com/person/robleh-jama)已经[承诺在他的个人博客](https://web.archive.org/web/20221207082845/http://robjama.com/2009/03/24/its-official/)上发布一篇文章，介绍与投资银行家打交道、媒体报道、竞购战以及[爆出的条款清单](https://web.archive.org/web/20221207082845/http://www.beta.techcrunch.com/2009/03/22/the-nasty-exploding-term-sheet/)等所有有趣的细节，这应该是一篇有趣的文章。
