# Spree 从 True Ventures、Aol 获得 150 万美元用于开源电子商务平台

基于 Ruby on Rails 的开源电子商务解决方案 Spree Commerce 今天宣布，它已经获得了由 True Ventures 领投的 150 万美元种子资金。参与此轮投资的还有美国在线风投公司，以及肖恩格拉斯(Sean Glass)等天使投资人。Spree 还带来了一些著名的顾问，包括 Dries Buytaert(Drupal 的创始人)、Luke Kanies(Puppet 的创始人)、Tom Preston-Werner(Github 的联合创始人)和 James linden Baum(Heroku 的联合创始人)。

一些读者可能对 Github 的 Spree 很熟悉，这家初创公司收到了数百篇投稿，并被翻译成 30 多种语言，使其成为社交编码平台上最受欢迎的开源项目之一。随着 Spree 越来越受欢迎，创始人肖恩·施菲尔德还创建了 Rails Dog，这是一家服务公司，专注于基于 Spree 的平台构建高度定制的网站，使没有工程经验的人能够实施该解决方案。

施菲尔德说，展望未来，Rails Dog 将继续提供咨询服务，但从今天开始，Spree 已正式注册为 Spree Commerce Inc .，并将主要专注于完善其电子商务平台，使其更加灵活和可定制。施菲尔德还表示，尽管该团队计划为其社区提供额外的支持和服务，但该平台将继续开源，许可和使用条款将保持不变，旨在用于个人和商业应用。

[![](img/b9e9d636d1779c3ad23bd8bf97e49dba.png "Screen shot 2011-10-10 at 11.09.36 AM")](https://web.archive.org/web/20221206042836/https://beta.techcrunch.com/wp-content/uploads/2011/10/screen-shot-2011-10-10-at-11-09-36-am.png)

对于那些可能对 Spree 到底做什么还有点困惑的人，该平台提供了托管和支持，以及一个简单的定制过程。具体来说，Spree 允许企业完全访问其代码，以便企业可以创建定制的结账流程，通过单页或多页结账，根据企业需求定制步骤数量。Spree 还使用语义 HTML，它允许用户从极简风格开始，可以根据用户的要求进行定制和添加。

该平台还提供灵活的产品管理系统、库存跟踪，以及添加扩展以基于现有产品、订单处理等创建产品组合的能力。就后者而言，对于电子商务公司来说，找到正确的网关和商家账户可能是一个巨大的痛苦，因此 Spree 可以与 50 多种不同的支付网关和服务集成，包括 Authorize.net 和贝宝。
