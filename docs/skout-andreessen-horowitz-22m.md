# Skout 的故事:从死亡之门到 Andreessen Horowitz 领导的 2200 万美元 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/04/03/skout-andreessen-horowitz-22m/>

18 个月前，定位战似乎已经结束。Foursquare 刚刚完成了一轮 2000 万美元的 B 轮融资，而 Gowalla、Loopt 和 Brightkite 等竞争对手却没有同样的影响力。与此同时，Christian Wiklund 的公司 Skout 在烧掉了自 2007 年成立以来筹集的 460 万美元的大部分资金后，只剩下三个人。

“我们快要破产了。我们没有牵引力，”维克伦德说。“但我们真的很固执，我们绝不会让这种情况发生。”

首次创业的维克伦德和他的联合创始人尼克拉斯·林德斯特罗姆埋头苦干。大约在那个时候，他们改变了应用程序，专注于一个新的人群:想要结识陌生人的年轻人和想要调情的单身人士。

它触动了人们的神经，Skout 的新应用在市场上非常幸运地出现了。爬上 app store 的排行榜还没有变得非常昂贵，所以 Skout 很早就在社交网络类别中占据了前 25 名的位置，直到现在仍然存在。

之后，它看到了“数千万”的下载量，该公司分享了一些值得注意的参与度统计数据。该应用程序每月支持约 3 亿条消息。普通用户每天登录八到九次，平均花 45 分钟聊天、交换礼物和上传照片。周末用的时候，几个小时内就有十几条消息。不过，Wiklund 没有透露它每月或每天有多少活跃用户。

惊喜，惊喜。它每月注册约 100 万用户。这与 Skout 的竞争对手 Foursquare 的增长速度相同。

在收入方面，该公司在过去一年的四到五个月里实现了盈利。它的收入足以支持包括承包商在内的 50 名员工，而在那段黑暗的日子里，Skout 只有 3 个人。现在，该公司有意重回亏损状态，以便专注于增长。

因此，在这一转变的背后，Skout 在 Andreessen Horowitz 领导的一轮融资中筹集了 2200 万美元。“这是世界上极少数我们愿意合作的风险投资公司之一，”Wiklund 说。

Andreessen Horowitz 将 Skout 作为移动约会领域的一个游戏，这是一个刚刚开始被破解的领域，尽管 OKCupid、Flirtomatic 和 Badoo 等网站也有其他移动约会应用程序。Wiklund 和公司都强调这是在“扩大”资金。

“从 Loopt 到 Whrrl，每个人都试图在本地和移动领域淘金，”安德森·霍洛维茨公司的普通合伙人斯科特·韦斯说，他是 IronPort Systems 的联合创始人。“但 Skout 公司的人正是通过找到适合调情和约会的产品市场做到了这一点。”

他说，Skout 和 Foursquare 现在已经足够不同，安德森投资这两家公司不会有利益冲突。

韦斯补充说，“尼克拉斯和克里斯蒂安是非常能干、聪明的企业家，他们对如何破解这一密码有很多好主意。”

Skout 的有趣之处在于，它的盈利方式不同于社交网络类别中的其他应用。虽然 Instagram、Pinterest 或 Tumblr 等其他移动应用可能会依赖广告，但 Skout 拥有来自应用内购买的健康收入流。用户可以支付点数来发送虚拟礼物，查看谁在查看他们的个人资料，并发送“眨眼炸弹”

“这种类型的礼物与所有虚拟角色社区一直在尝试的没有什么不同，”韦斯说。"这是关于继续扮演一个角色并引起别人的注意."

这使得 Skout 的收入组合更像 FarmVille，而不像 Match.com。如果用户想避开广告，也可以下载一款付费应用。也就是说，Wiklund 设想的业务也将依赖于本地广告。

Skout 为确定模型所做的另一件事是非常强调社区管理。该公司每月禁止 40，000 多台设备，以确保新用户感到舒适。“它需要一个安全、干净、健康的网络，”Wiklund 说。"我相信你还记得聊天轮盘是如何让一些人伤痕累累的."

有了这笔资金，斯库特计划找到一个能容纳 120 人的新办公室。即使是现在，也还是要保持低调。

“我们采取了谦逊的态度。我们甚至还没有真正庆祝过，”维克伦德说。“它还在下沉。但一旦你拿了钱，你最好兑现。否则为什么要稀释？”
