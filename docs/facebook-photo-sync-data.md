# 脸书通过大力推广照片同步技术获得大量数据

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/12/01/facebook-photo-sync-data/>

脸书已经每天拍摄 3 亿张照片，而且这个数字还将大幅增加。现在，它正引导用户进入它的[后台上传功能照片同步](https://web.archive.org/web/20221208180451/https://beta.techcrunch.com/2012/11/30/facebook-cloud-storage/),在它的移动应用的新闻提要顶部有一个大横幅。只需轻点两下，你最近的 20 张照片以及你未来拍摄的每一张照片就会自动上传到私人相册，你可以分享这些照片，脸书也可以从中挖掘元数据。

[昨天开始的照片同步](https://web.archive.org/web/20221208180451/http://newsroom.fb.com/News/542/Photo-Sync)的推出可能是除了好友标记和手动手机上传之外，脸书照片发生的最大的事情。五年前，脸书的成长伴随着一股自恋情结。如果有人在一张照片上给你加了标签，你可能会在收到通知的第一时间查看这张照片，以及相册中的所有其他照片。后来，它开始提供手机照片分享，但它最大的问题仍然是费力的上传过程。

![PhotoSync Mobile](img/6f9e7a4d9184063c8c643a468b1155b2.png) [照片同步](https://web.archive.org/web/20221208180451/https://www.facebook.com/help/photosync)打磨光滑的大理石表面，这将获得大量用户。脸书的 iPhone 和 Android 应用程序每个月都有超过 1.15 亿的用户，他们最终会看到这个横幅，当实际上是一个严肃的决定时，它会漫不经心地要求你“开始”。

一旦启用，你拍摄的每张照片都会被上传，无需你做任何工作。这意味着这六个人都想拍出完美的肖像，你拍的 30 张美丽日落的照片，你想买的可爱裤子的照片，以及这之间的一切。你只需选择要分享的内容，过一会儿，你在脸书上的朋友就可以看到这些内容了。

当你分享一张照片时，你可以分享四张。你可以分享更多的风景和其他抽象的照片，而不是眯着眼睛在手机上输入描述，因为当你从桌面上分享它们时，你可以添加描述。这将生成更多朋友的照片，吸引他们回到网站，并在新闻提要中为他们提供内容。

如果脸书将其与面部识别功能相结合，您将看到照片同步的全部力量。脸书不会手动标记朋友，而是会分析你同步相册中朋友的照片，自动应用标记，你只需批准即可。

与此同时，脸书可能会吸收一些违反服务条款和其他非常隐私的照片。即使人们知道其他人看不到他们，当他们在脸书浏览器中看到他们喝醉、邋遢、裸体或其他有损形象的照片时，一些人可能会恐慌。目前，脸书没有很好地解释该功能是如何工作的，所以它可能会导致一些人感到惊讶，惊慌失措，并关闭它。

但对脸书来说，这可能是值得的，因为照片同步将提供大量的照片元数据。[正如我昨天写的](https://web.archive.org/web/20221208180451/https://beta.techcrunch.com/2012/11/30/facebook-cloud-storage/)，这个功能本质上是脸书进入云存储。虽然你可能不会分享你上传的所有照片，但脸书仍然可以调出地理定位功能，帮助它找出你现在的位置，并向你展示当地的商业广告..最终，品牌认知度，比如在照片中识别耐克鞋的能力，也会有助于广告定位。

昨天没有大型发布会，因为脸书不需要。事实上，它可能不想要一个，考虑到它甚至没有像往常一样通知像我这样的博客。在其 iOS 和 Android 应用程序的新闻提要顶部的横幅可以轻松地让数千万用户启用照片同步。如果它在社交网络和现在的云存储领域的竞争对手没有意识到发生了什么，他们将无法做出反应。但是脸书刚刚找到了一种方法，可以比第一次更快地获得下 2190 亿张照片。
