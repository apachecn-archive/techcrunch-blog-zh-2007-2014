# 古腾堡计划创始人迈克尔·斯特恩·哈特，1947-2011

1971 年，迈克尔·哈特在施乐公司的主机上得到一些操作员的时间，他感到有义务创造一些有价值的东西。他灵机一动，一个字母一个字母地输入了《独立宣言》的文本，这是 7 月 4 日给他的一份副本。然后，他试图将它发送给每个连接到主机的人，侥幸躲过了让他们头上的网络(尽管如此)瘫痪。科技史上最具前瞻性的项目之一古腾堡项目(Project Gutenberg)的开始并不顺利。

正如我们所知，网络的基本原理体现在古登堡计划中。数字化和免费提供世界信息的想法在那里找到了第一个真正的牵引力。当然，YouTube、iTunes 和 Google Books 仅仅是第一种动力的放大版，将书写的页面转化为 1 和 0。

在一个版权冲突、专利纠纷和商标欺凌的时代，该项目提醒我们，子孙后代对我们徒劳的争吵漠不关心。

事实上，哈特和他的志愿者团队所做的决定不仅预测了未来的市场和技术，还超越了那些将被用于盈利和商业目的的讨厌时期。寻找将他人作品货币化的方法是亚马逊、Adobe 和苹果的工作。古腾堡总是让尽可能多的文学作品变得容易理解。这就是为什么这些书都是普通的 ASCII 码(最近又扩展到其他一些可访问的格式)，从 Amigas 到 iPads 都支持。这是互联网最纯粹的形式:以最简单、最开放的方式不受约束地传播信息。

古腾堡计划的方法是第一个，现在仍然是最好的。虽然公司和行业可能会像雷莫拉一样抓住进步的方舟，在排他性和适销性时期榨取尽可能多的利润，但我们前进的方向是不变的。哈特是一个真正的梦想家，他的眼光比后来的许多人都要高。

本周早些时候，哈特在伊利诺伊州的家中去世。人们将深切怀念他的勤奋、慈善和谦逊。[这里已经贴出了讣告](https://web.archive.org/web/20230205033517/http://www.gutenberg.org/w/index.php?title=Michael_S._Hart)。[哈特自己的项目历史可以在这里找到](https://web.archive.org/web/20230205033517/http://www.gutenberg.org/wiki/Gutenberg:The_History_and_Philosophy_of_Project_Gutenberg_by_Michael_Hart)。
