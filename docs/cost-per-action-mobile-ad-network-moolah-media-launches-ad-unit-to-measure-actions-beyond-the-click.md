# 移动广告网络 Moolah Media 推出广告单元来衡量点击以外的行为

最近推出的移动广告网络 Moolah Media 采用了按行为计费和按线索计费的模式，随着其新的 SmartMoolah 层的推出，它正在为广告商和出版商推出一种新的性能优化的广告单元。

去年[推出的网络](https://web.archive.org/web/20230205031043/https://techcrunch.com/2010/12/14/moolah-media-launches-cost-per-action-mobile-advertising-network/)允许广告商将呼入电话直接接到呼叫中心，收集注册和注册线索，同时还可以跟踪转化率。Moolah 承诺为出版商提供更高的报酬和 100%的填充率，广告可以放在应用程序中，移动网络上或短信中。

今年早些时候，[初创公司为展示或应用内广告增加了](https://web.archive.org/web/20230205031043/https://techcrunch.com/2011/07/12/cost-per-action-mobile-ad-network-moolah-media-debuts-3m-developer-fund-and-new-formats/)多种新的广告形式。目前，Moolah 在美国拥有 2000 万用户，每月在其网络上提供 10 亿次广告展示。

今天发布的最新广告单元旨在衡量点击之外的消费者行为，随后帮助为每位消费者提供更相关的广告。这家初创公司将关联技术比作消费者的潘多拉体验。不同于传统的每次印象成本、每次点击成本或每次下载成本广告，SmartMoolah 衡量的是点击后的行为。SmartMoolah 广告是基于每次行动的成本来衡量的移动显示单元。操作包括下载、表单提交、点击呼叫和交互式 SMS。

当消费者购买、点击内容或与他人分享体验时，点击后发生的动作向发布者显示。基于消费者的行为，Moolah 可以在未来向该消费者提供更多相关的广告。

SmartMoolah 广告可以通过该公司的 SDK 部署在 Android 和 iOS 移动应用程序中，并使用基于标准的 API 部署在移动网站中。这家初创公司表示，由 SmartMoolah 广告优化的广告活动比没有针对性的广告活动表现好 8 倍，产生 0.96 美元的 eCPMs，而不是 0.12 美元。SmartMoolah 广告还承诺将点击率提高一倍。
