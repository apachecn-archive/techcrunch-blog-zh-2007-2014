# 热衷于…推销自己:社交媒体如何不能帮你找到工作

如果你想学习如何推销自己，请阅读丹·肖贝尔的《T2》词条。或者，你可以读一读斯考贝尔的新书(下周出版)，书名恰如其分地叫做 [*提升自己:职业成功的新规则*](https://web.archive.org/web/20221006134904/http://www.amazon.com/Promote-Yourself-Rules-Career-Success/dp/1250044553/ref=sr_1_1?s=books&ie=UTF8&qid=1377586539&sr=1-1/&tag=techcrunch098-20) 。29 岁的 Schawbel 是咨询和营销公司 [Millennial Branding](https://web.archive.org/web/20221006134904/http://millennialbranding.com/) 的管理合伙人，他成功地成为了*千禧一代的工作习惯和时尚权威。现在，随着*推销自己*，肖贝尔将他的巨大才能用于指导千禧一代如何在当今竞争日益激烈的市场中取得成功。*

在与美国运通合作研究*推销自己*的过程中，Schawbel 得出了一些有趣的结论，这些结论是关于我们数字经济中职业成功的新规则——特别是在脸书和推特这样的社交媒体网络的真正价值方面。“社交媒体在工作场所不受尊重，”他告诉我。事实上，他发现 40%的学生认为过多的社交媒体使用伤害了他们所谓的“软技能”。Schawbel 坚持认为，极客不会接管工作场所。相反，在当今经济中取得成功的关键是复制丹·肖贝尔(Dan Schawbel)本人，并掌握一个利基市场，这样你就能成为某个特定领域的可信权威。

所以，年轻人，如果你想要一份体面的工作，就离开 Twitter 和脸书，学习肖贝尔所说的“人际交往技巧”(即:如何与他人交谈)。他提醒他的千禧一代同事，坏消息是，在今天的工作场所，不是每个人都能获得奖杯。但丹·斯考贝尔认为，好消息是，那些对自己的技能和知识“负责”的人仍然可以成为当今网络经济的赢家。
