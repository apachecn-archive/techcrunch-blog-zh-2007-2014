# 点击脸书广告获得社会

![](img/58c6232c10a8a5e624849425c5bef1d0.png)

很快将有可能并排比较搜索和社交广告活动的表现。[可点击的](https://web.archive.org/web/20221208222829/http://www.clickable.com/),[广告管理平台](https://web.archive.org/web/20221208222829/https://beta.techcrunch.com/2009/02/09/clickable-launches-version-20-of-powerful-search-ad-management-software-suite/)，让搜索营销人员衡量和跟踪他们在不同搜索引擎和广告网络的在线营销活动的表现，将在 4 月 12 日增加脸书广告作为一个选项。

这意味着，在谷歌上购买点击付费广告的广告商可以测试这些搜索广告相对于脸书上针对特定社会人口的点击付费广告的表现。搜索广告与社交广告，都在一个仪表板中。按此[报名页面上的](https://web.archive.org/web/20221208222829/http://www.clickable.com/facebook/)可点击:

> Clickable Pro 现在允许你批量创建和上传点击付费脸书广告。和可点击的转换跟踪，你可以跟踪脸书和搜索营销收入和转换一个简单的标签放在您的网站上。您甚至可以生成定制的白标报告，将搜索和社交表现并列显示。

Clickable 是脸书的一家合作伙伴，它已经将其服务与脸书广告 API 整合在一起，这将允许 Clickable 的客户直接从 Clickable 创建、上传和管理脸书广告。这种整合将适用于点击付费文本或图像广告，但还不适用于常规显示广告。

脸书广告现在为社交网络带来了实际收入。但是它们与搜索广告相比如何呢？
