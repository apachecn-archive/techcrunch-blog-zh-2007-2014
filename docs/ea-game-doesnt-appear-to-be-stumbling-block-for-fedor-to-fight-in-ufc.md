# EA 游戏似乎不会成为费多尔征战 UFC 的绊脚石

![fedorr](img/0a3a7502be46c70432e312a23a6a9348.png "fedorr")

下面是我前几天在上提到的一些事情的更新:费多尔加入 EA 的 MMA 视频游戏是否会阻止他参加 UFC？显然不会，如果你相信 UFC 主席白大拿[昨天在](https://web.archive.org/web/20221007200459/http://www.f4wonline.com/content/view/10128/)的新闻发布会上所说的话。

昨天的新闻发布会讨论了很多问题，包括蒂托·奥蒂兹的回归和与英国 ESPN 的交易。(ESPN 将在英国推出一个新频道，主要是展示它刚刚赢得转播权的所有英超比赛。)但我们在 CrunchGear 担心的是 EA MMA 游戏。

白大拿过去说过，除了兰迪·科图，任何报名参加 EA 比赛的人都不会参加 UFC 比赛。费多尔似乎是另一个例外。怀特说他给了费多尔“一大笔钱”，这是他在其他综合格斗领域永远找不到的交易。不管出于什么原因，费多尔的人不同意这笔交易。(有传言说，费多尔的人希望共同推动任何可能的 UFC 战斗，大概是对布洛克·莱斯纳，这是 UFC 不想同意的。)

因此，从字里行间来看，费多尔加入 EA 游戏*应该不会对任何可能的 UFC 战斗产生影响*。其他事情可能会阻碍交易，但看起来不像是游戏。
