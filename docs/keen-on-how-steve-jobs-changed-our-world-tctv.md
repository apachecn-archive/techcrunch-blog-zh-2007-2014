# 热衷于…史蒂夫·乔布斯如何改变我们的世界(TCTV)

早在 2000 年，在 iPod、iPhone 或 iPad 出现之前的史前时代，著名的科技记者艾伦·多伊奇曼写了一本非常受欢迎的传记，名为《史蒂夫·乔布斯的第二次到来:T2、T4 和 T5》。现在，十多年后，德国人又回到了乔布斯身边，并且刚刚写了一篇他称之为电子切片的文章，题为《史蒂夫·乔布斯如何改变了我们的世界》。

当我在 Skype 上采访他时，德国人告诉我，乔布斯两次改变了我们的世界。首先，他在 80 年代早期的黑暗日子里振兴了美国商业，Deutschman 解释说，从索尼等日本公司手中夺回了创新的衣钵，甚至复兴了美国制造业的理想。Deutschman 告诉我，最重要的是，史蒂夫·乔布斯在 2000 年后首先抓住并实现了他所谓的“数字生活方式”和设计的理念，从而改变了美国工业。

德国人的分析有点苦乐参半。是的，他说的没错，乔布斯是一个复兴主义者——至少在两个场合拯救了美国工业。然而，今天，当美国工业比 80 年代初更加低迷的时候，乔布斯走了，永远地失去了，就在我们最需要他的时候。

这是对德国人两部分采访的第一部分。明天再来看看“为什么史蒂夫·乔布斯如此重要”。
