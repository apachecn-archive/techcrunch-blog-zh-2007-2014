# 丰田 iQ 获得 55+ MPG，一月来欧洲

![toyota-iq-small-car-europe-01](img/df43af4c1be3ba246000c19d4ee5b939.png "toyota-iq-small-car-europe-01")

瓦斯，笨蛋。这是丰田 iQ，一辆真正的汽车，现在可以在欧洲预订，将于明年 1 月在那里上路。是的，它很小，但它可以容纳四个人——丰田推荐三个成人和一个孩子——最棒的是，它每加仑能跑相当惊人的 55 英里以上。

将有三种型号可供选择；iQ 使用 1 升汽油发动机每加仑行驶 55 英里，iQ2 使用 1.4 升柴油发动机每加仑行驶 57 英里，第三个版本仍在开发中，可能是 iQ3，它将使用一些疯狂的发动机，当你停下来时，它会完全关闭，当你松开刹车时，它会重新启动。把它想象成你汽车的睡眠模式。那一个将在明年的某个时候准备好。

![toyota](img/939f75a8a6673881a753f30efcac03ad.png "toyota")

没有消息说我们什么时候能在美国看到这些，但是希望很快。[《消费者指南》似乎认为](https://web.archive.org/web/20221007160526/http://consumerguideauto.howstuffworks.com/2010-toyota-iq.htm)我们会在“不早于 2009 年秋季”在这里看到这款车，价格可能在 13，000 美元到 17，500 美元之间。
