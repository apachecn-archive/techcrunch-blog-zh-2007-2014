# 纽约州长取消“iPod 税”:刺激资金将弥补预算缺口

![noipodtax](img/d6eb2bead9a098bb392289167c047a62.png "noipodtax")

纽约州抱怨奥巴马总统综合支出法案的 Itunes 用户应该闭嘴，因为州长帕特森已经[放弃了](https://web.archive.org/web/20221001205032/http://wcbstv.com/topstories/david.paterson.taxes.2.956197.html)对 [iPod 征税](https://web.archive.org/web/20221001205032/http://www.crunchgear.com/2008/12/17/ny-governor-proposes-ipod-tax-among-others-to-make-up-budget-shortfall/)的计划，这让*很多人*感到不安。一些刺激资金将用于弥补“iPod 税”本应弥补的预算缺口。

拟议的国家预算中总共取消了 13 亿美元的税收。是的，以一种迂回的方式，这仍然是纳税人的钱来弥补差额，但现在它不是可怕的“iPod 税”，所以。
