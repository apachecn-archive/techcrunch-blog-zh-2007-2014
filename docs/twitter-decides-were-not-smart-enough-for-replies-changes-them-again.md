# Twitter 认为我们不够聪明，无法处理@回复，再次修改它们 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/05/12/twitter-decides-were-not-smart-enough-for-replies-changes-them-again/>

![](img/25495ab77bb43a1ab473de4ac03b10dc.png)

Twitter 正式变得愚蠢了。

在不到两个月的时间里，Twitter 第二次[改变了](https://web.archive.org/web/20230214132504/http://blog.twitter.com/2009/05/small-settings-update.html)它的@reply 系统，这一次是通过删除一个已经存在了好几个月的选项来安抚困惑的新来者。

@reply 系统背后的基本前提是，它允许你与另一个 Twitter 用户建立半公开的对话。为了防止你不得不收听你可能不关心的对话，默认设置长期以来一直是只在你关注对话中的两个人的情况下显示这些@回复。这是大多数人坚持的选择。

但是有一个选项可以接收你关注的任何用户的所有回复消息。这导致噪音增加，但也让你接触到新的 Twitter 用户和对话，否则你可能会错过。我已经开了一年多了。但是很明显，这个选项让太多人感到困惑，所以 Twitter 正在扼杀它。

来自 Twitter 博客:

> 我们更新了设置中的通知部分，以更好地反映人们如何使用 Twitter 进行回复。根据使用模式和反馈，我们了解到大多数人希望看到他们关注的人对他们关注的另一个人的回复，这是一种保持联系的好方法。然而，通过回复那些你在时间线中没有关注的人来接收片面的片段是不可取的。今天的更新删除了这个不受欢迎和令人困惑的选项。
> 
> 迷茫？这是可以理解的，也是我们进行更新的原因。

哎呀，谢谢推特。我没有意识到我手动激活的选项是不受欢迎的。还有什么我不喜欢的事你想让我知道的吗？

如果说旧系统有什么不尽如人意的地方，那就是 Twitter 没有很好地解释它，而不是功能本身不需要。考虑到这个选项不是默认选项，而且被隐藏在设置菜单下，这又有什么关系呢？如果太多的人感到困惑，为什么不干脆把它隐藏起来(也许在“高级”标签下)？

几个月以来，Twitter 在主流中的吸引力越来越大，特别是自从它在 Oprah 上首次亮相以来，一些 Twitter 的早期采用者对服务的变化表示了担忧。随着越来越多的名人和媒体出现(更不用说垃圾邮件发送者)，他们担心这项服务会失去其紧密的感觉。在今晚之前，我从来没有关注过这种思路——毕竟，在 Twitter 上，我可以关注我关心的人，忽略我不关心的人。但很明显，Twitter *关注的是吸引更主流的受众，如果这需要让一个非常简单的服务变得更加简单，那么天哪，这就是他们要做的。*

**更新 1** :许多 Twitter 用户对这一变化表示强烈反对，在频道 [#fixreplies](https://web.archive.org/web/20230214132504/http://search.twitter.com/search?q=%23fixreplies) 下表达他们的不满，这是 Twitter 上目前最热门的术语。

**更新 2** :推特首席执行官伊万·威廉姆斯刚刚[在推特上](https://web.archive.org/web/20230214132504/http://twitter.com/ev/status/1781925468)谈到了这一变化:“阅读人们对回复问题的想法。我们正在考虑替代方案。感谢您的反馈。”
