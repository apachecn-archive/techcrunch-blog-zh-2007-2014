Facebook Messenger For Windows桌面聊天客户端在泄露后启动

原文：https://web.archive.org/web/https://techcrunch.com/2012/03/05/facebook-chat-desktop-client/amp/

如果你不用打开浏览器窗口，你可能会花更多的时间使用脸书，所以今天脸书正式发布了它的Messenger for Windows免费下载桌面客户端。但是由于早期测试者在12月泄露了下载链接，而脸书通过公开做出了回应，你可能已经拥有它了。客户端允许您持续聊天、接收通知，并从桌面阅读您的新闻提要。

脸书说“对于那些苹果电脑用户，我们现在正忙着为苹果电脑开发Messenger。敬请期待。”Facebook Messenger for Windows可以在这里下载 for Windows 7，它将在未来几周内向全球Windows用户推广。脸书已经耗尽了许多国家的大量新用户。该产品表明，它正转向专注于从现有用户那里榨取最后一滴参与度。

事实证明，异地脸书聊天非常受欢迎。根据 AppData 的数据，微软的Messenger Live网站是脸书最受欢迎的应用，允许用户脸书聊天、阅读朋友的更新和分享，拥有2240万月活跃用户和惊人的1910万日活跃用户，粘性百分比高达85.5%。Messenger for Windows可以在没有Hotmail、SkyDrive或其他微软功能来分散用户注意力的地方使用这些功能。

脸书喜欢在全球推广之前，在一小部分用户群中测试其产品，以检查使用模式和采用率。它甚至有一个专门设计的叫做看门人的基础设施来管理谁可以看什么。在测试基于浏览器的产品时，脸书拥有完全的控制权，可以随意向不同的用户显示或隐藏功能，实验对象通常不能选择加入或退出测试。

在测试其首批可下载的桌面产品之一时，脸书突然发现实验是由豚鼠来进行的。当以色列TechIT 的一位博主在11月份产品发布后被允许加入Messenger for Windows测试小组时，他得到了一个未受保护的下载链接。他很快发表了它。我们听到了这条新闻，建议脸书面对现实，公开发布链接给那些急于下载的人。这正是它所做的，使客户在帮助中心可用，但没有正式宣布。

今天，这份声明发布了。脸书告诉我，自泄漏以来的更新基本上只是微调和漏洞修复，没有添加任何功能，测试人员也可以进行异地聊天。就像脸书独立的Messenger移动应用程序一样，Windows客户端快速响应，让用户避免了需要不断查看脸书的新活动的恼人感觉。

正如我在12月指出的，客户将在两个核心方面帮助脸书:

  • 持续访问聊天会增加主要用户的参与度，也会吸引他们的朋友花更多时间在脸书上
  • 持续访问启动Facebook.com的通知、消息和好友请求，可能会比用户随意访问网站查看这些提醒带来更多的回访

有了这个和即将推出的Mac客户端，脸书可能已经找到了一种方法来打破自己创下的每月平均7小时以上的网站时间纪录。
