# 科里·布克向公众开放拦截搜身数据。这就是为什么它可能会有帮助。TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/08/27/corybooker-opens-stop-and-frisk-data-to-the-public-heres-why-it-might-help/>

![4754231502_940e0fe7f1_z](img/60ef91182c50667cb7de4f6bef26cc3e.png)

纽瓦克市长和参议员候选人科里·布克刚刚开始测试一种创新的解决方案，以解决困扰执法部门使用拦截搜身的种族问题:通过向公众公开每个拦截的细节来追究警察的责任。有争议的“拦截搜身”允许警察搜身检查任何看起来有点可疑的公民；执法部门声称这是在过度拥挤的城市打击犯罪的重要工具，而公民自由组织声称这不公平地针对少数族裔(在纽约州，[少数族裔占](https://web.archive.org/web/20221020042232/http://newyork.cbslocal.com/2013/02/05/nypd-report-minorities-account-for-90-percent-of-stop-and-frisks/)所有拦截搜身事件的 90%)。

但是，当纽约市市长迈克尔彭博正在与公民自由团体斗争以保持这种做法时，布克与他当地的美国公民自由联盟合作，以找到一种彻底透明的解决方案。每个月，纽瓦克都会发布拦截搜身事件报告，详细记录种族、性别、年龄、强制使用和逮捕情况。“这是警察的胜利，因为他们能够获得社区成员的信任，”纽瓦克美国公民自由联盟负责人乌迪·奥弗说。“这对公众来说是一个胜利，因为公众能够破译是否有任何滥用行为发生。”

对于那些没有意识到的人来说，无论如何，停下来搜身都是一种相当宽泛的策略。根据宪法权利中心的数据，NYPD 每 143 次对非裔美国人的拦截中，只有 1 次会导致 T4 没收枪支、毒品或其他武器。

**数据、无意的种族歧视和安全**

尽管大多数人不会认为自己是种族主义者，但我们中的许多人可能怀有潜在的偏见。对于警察来说，每天看到同样的少数民族暴力可能会加剧我们所有人的偏见。

加州大学洛杉矶分校教授菲利普·戈夫(Philip Goff)在给我的一封电子邮件中解释道:“公众眼中的‘种族歧视’不仅是由隐性偏见造成的，也是由一些社区和发誓保护他们的警察之间的沟通不畅和不信任历史造成的。”。“换句话说，如果不了解人性，就不可能理解警务中的偏见，而我们误解他人的倾向可能是危险的”

发布停车搜身的数据是警察承认无意的潜在种族偏见的友好方式。没有什么可隐瞒的，当局可以不丢面子地承认错误，与社区成员合作调查真相。例如，有可能官员没有种族偏见，但有地理偏见，无意中将资源集中在特定的社区。

“在他们的实践中更加透明将导致与社区更好的关系，这将导致一个更安全的城市，”Ofer 说。

透明度不仅仅是保护公民自由；这也是关于社区帮助他们的邻居变得更安全。挖掘数据可能会产生有价值的见解，执法分析人员没有时间用他们有限的人员去思考这些见解。

例如，也许警察发现太少的白人女性携带毒品，而阻止了太多的中年非裔美国人涉嫌持枪进入 T2。在将拦截搜身的成功率与实际被判有罪的人口统计数据进行比较后，研究人员将帮助警方找到不寻常的犯罪源头。因此，在寻找非裔美国公民权利的过程中，独立统计学家可能会向当局提供未知的毒品分销渠道，并帮助官员更好地识别社区中谁实际上携带武器。

请原谅我对数学的兴奋，但我们可以在数据中找到各种有趣的统计模式，这让每个人都站在了找出真相和拯救生命的同一侧。

**需要更多信息**

当然，所有这些美妙的好处都是理论上的。目前，警方将每月的数据集转储到笨重的 PDF 文件中，将这些数据转换成电子表格进行统计分析是一件非常麻烦的事情。纽瓦克警方需要一些开发者朋友，他们可以把他们的数据放在统计友好的网站上。

更有问题的是，数据输入完全由警方完成。这就像让学生给自己的作业打分一样。即使有最好的意图，所有的报道仍然受制于心理偏见。

这个问题就是为什么 Ofer 在 TechCrunch 上首次公开支持使用警用翻领摄像机，类似于目前警车上的仪表盘摄像机。只有当公众能够从官员的角度回顾停车时，我们才能真正了解所有数据以及停车背后的意图。

**布克再次给人留下深刻印象**
