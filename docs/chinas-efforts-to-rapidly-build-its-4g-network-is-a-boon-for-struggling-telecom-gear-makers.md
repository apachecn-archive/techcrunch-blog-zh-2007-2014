# 中国快速建设 4G 网络的努力对苦苦挣扎的电信设备制造商来说是一个福音

据报道，中国政府[最迟将在今年年底或 2014 年初开始发放 4G 牌照](https://web.archive.org/web/20221006011830/http://translate.google.com/translate?sl=zh-CN&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&eotf=1&u=http%3A%2F%2Ftech.sina.com.cn%2Ft%2F3g%2F2013-05-08%2F08478317076.shtml&act=url)(通过谷歌翻译链接)，此前有消息称，中国移动[最早将于本月进行 4G 网络建设招标](https://web.archive.org/web/20221006011830/http://www.china.org.cn/business/2013-05/08/content_28760867.htm)。中国尽快建设 TD-LTE 网络的努力对电信设备制造商来说是一个潜在的有利可图的机会，这些设备制造商受到了全球经济低迷的影响。

除了华为和中兴等国内运营商之外，希望将其增长与中国移动 4G 网络扩张挂钩的外国电信设备和芯片制造商包括爱立信、诺基亚西门子、阿尔卡特朗讯、高通和三星电子。

中国移动[上个月表示，今年将在网络上花费 1902 亿元人民币(301 亿美元)](https://web.archive.org/web/20221006011830/http://www.reuters.com/article/2013/03/14/chinamobile-iphone-idUSL3N0C61J920130314?feedType=RSS&feedName=technologySector&rpc=43)，其中大约四分之一用于 TD-LTE 技术。该公司是全球用户数量最大的电信运营商，预计将建设 20 万个 TD-LTE 基站，以及 11 万个 3G 基站。根据 IHS iSuppli 的估计，中国移动的 4G 用户群预计将在 2017 年达到 2.288 亿，占中国 4.399 亿 4G 用户总数的 52%。))

全球最大的 4G 供应商爱立信尤其渴望在中国的 4G 市场上分得更大的一杯羹。爱立信亚太区高级副总裁 Mats H. Olsson 在 2 月份的世界移动通信大会上表示:“我们对爱立信去年在中国移动第一轮 4G 招标中取得的成绩不满意。”。“过去，爱立信非常关注美国、日本和韩国等国家，主要关注 FDD-LTE 网络的部署。现在我们已经把目光投向了中国和 TD-LTE 技术。”

然而，国内电信运营商中兴和华为预计将与中国三大运营商(中国移动、中国联通、中国电信)签订最多的 4G 网络合同，因为它们享有中国政府的支持。获得合同对中兴来说[尤其重要，因为它的表现一直落后于华为。](https://web.archive.org/web/20221006011830/https://beta.techcrunch.com/2013/03/24/zte-is-betting-on-chinas-nascent-4g-network-to-bolster-its-flagging-profitability/)

目前还不清楚中国移动和中国电信是将一起还是分开运营 TD-LTE 网络，因为[的竞争对手更喜欢](https://web.archive.org/web/20221006011830/http://www.marketwatch.com/story/competition-tries-to-curb-china-mobiles-4g-push-2013-05-08)。中国第三大电信运营商中国联通希望尽可能长时间地坚持 3G TD-SCDMA，因为它尚未收回 2007 年开始建设的 1000 亿元人民币(163 亿美元)的低速网络投资。TD-LTE 的前身 [TD-SCDMA 最初是为了成为全球 3G 标准](https://web.archive.org/web/20221006011830/http://www.isc.org.cn/english/Events&News/Industry_News/listinfo-25390.html)而开发的，但其使用最终仅限于中国。
