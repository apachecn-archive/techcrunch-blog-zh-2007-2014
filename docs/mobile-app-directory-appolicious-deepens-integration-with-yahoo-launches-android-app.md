# 移动应用目录 Appolicious 深化与雅虎的整合；推出 Android 应用程序

社交移动应用目录 [Appolicious](https://web.archive.org/web/20221203132354/http://www.appolicious.com/) 正在深化与雅虎的整合，推出了“yap.appolicious.com”网站，帮助人们找到在雅虎平台上运行的应用，并重新推出和设计了[AndroidApps.com](https://web.archive.org/web/20221203132354/http://www.androidapps.com/)，帮助人们找到安卓应用。今年早些时候，Appolicious 宣布与雅虎合作，共同打造 Appolicious 应用目录。该公司的创始人 Alan Warms(他将之前的公司卖给了雅虎)表示，由于这种合作关系，Appolicious 的流量正在激增。

Appolicious 通过今年早些时候对 AppVee 的收购而拥有了 AndroidApps.com，现在将在其所有垂直领域(包括新的垂直领域(雅虎应用)提供雅虎品牌的移动应用体验。

与 Appolicious 的主网站类似，Yap.Appolicious.com 和 AndroidApps.com 关于移动应用的原创文章的链接将成为雅虎媒体资产的重要内容，包括新闻、体育和金融。这两个网站现在都将包含“与雅虎合作”的品牌标志。

Appolicious 还宣布了一个新的跨三个属性的联合搜索功能，允许用户访问用户生成的应用程序列表，关于搜索相关应用程序的文章，以及来自 Appolicious 推荐引擎的应用程序。此外，Appolicious 今天发布了一款 Android 应用，与它的 [iPhone 表亲一样，](//web.archive.org/web/20221203132354/https://www.beta.techcrunch.com/2009/12/21/appolicious-now-delivers-personalized-app-recommendations-to-the-iphone/)允许用户将他们的应用目录导入网站，评级、评论、个人资料、精选应用列表、应用页面等等。

除了与雅虎的整合，Appolicious 在产品改进方面也度过了重要的一年。这家初创公司推出了[精选列表](https://web.archive.org/web/20221203132354/https://beta.techcrunch.com/2010/01/14/appolicious-takes-a-page-out-of-twitters-playbook-lets-users-create-personalized-lists-of-apps/)和[推出了](https://web.archive.org/web/20221203132354/https://beta.techcrunch.com/2010/03/05/mobile-app-directory-appolicious-rolls-out-facebook-app-url-shortener-and-more/)appo . me 应用链接网址缩写，以及 Twicker 广告模式，允许用户在网站上查看推文和赞助商的报价。去年年底，Appolicious 筹集了[150 万美元](https://web.archive.org/web/20221203132354/http://www.beta.techcrunch.com/2009/12/16/appolicious-lands-2-million-for-iphone-app-directory/)的资金，并推出了一款 [iPhone 应用。](https://web.archive.org/web/20221203132354/http://www.beta.techcrunch.com/2009/12/21/appolicious-now-delivers-personalized-app-recommendations-to-the-iphone/)
