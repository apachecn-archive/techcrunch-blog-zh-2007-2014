# 点评:LG 42LD450 液晶电视

电视价格过高。人们花费数千美元购买最新的、功能更多的梦幻电视。对我来说，它总是归结为图片质量和为此付出的代价。这就是 LD450 击败所有人的地方。精彩画面，无特色低价位精彩功能。我并不特别喜欢 LG 这个品牌，因为几年前我遇到过一个功能不正常的 DVD 播放器。因为 LD450，我对 LG 的看法发生了转变。

除了一个很好的遥控器、一本说明书，当然还有带支架的电视机，你在盒子里没有什么东西。电视的设计相当单调:一个黑色的塑料平板围绕着一个黑色的大液晶显示器。然而，右下角有一个指示灯，显示电视是开着还是关着。这个可以关了，那就好。按键和连接器都在后面和侧面；从前面什么也看不见。总共有两个 HDMI 输入，虽然这对一些人来说可能不够，但对我来说足够了。使用 2X10W 隐藏式扬声器，声音相当不错。只有一个用于音频连接的光学输出，这对声音爱好者来说可能是个问题。

尽管这是一台 50Hz 的电视机，但画质还是非常好。常规电视频道看起来不错，高清当然更棒。这种屏幕看起来比售价 8-900 美元的类似尺寸的电视机更亮。设置图片非常简单。我喜欢它，因为它是开箱即用的，但如果你想为自己设置它，有一个非常简单的向导。有一种生态模式可以拯救地球免于某种厄运，我立即关闭了它，因为它降低了屏幕的亮度。输入选择是手动完成的；不幸的是，set 看不到哪个输入上有信号。电视会一直等待最近使用的输入信号。您可以从 24p Real Cinema 等多种图像模式中进行选择，还可以设置并保存自己的图像设置。

![](img/89a88122e8d9b7d60182d627f7f14bcc.png "lg450back")另一个(唯一的)重要功能是 USB 输入。您可以插入 USB 硬盘或拇指驱动器，通过电视菜单访问图片和音乐。现在，如果你够聪明，有一个简单的通用遥控器，你就可以播放视频了。支持各种格式，包括 mkv。这意味着你只需要一个外置硬盘就可以在这台电视机上观看高清电影。

LG42LD450 是一款性价比极高的液晶电视。它可能缺乏一些功能，但如果你不想花很多钱，想买一台新电视，这可能是一个选择。由于增加了通过 USB 观看高清电影的功能，这款电视成了我可以向任何人推荐的绝佳选择。你可以花 566 美元从[亚马逊](https://web.archive.org/web/20230328193103/http://www.amazon.com/LG-42LD450-42-Inch-1080p-HDTV/dp/B0039RRC7U/ref=sr_1_1?ie=UTF8&s=electronics&qid=1283352857&sr=8-1)买到它。
