# 大宇发布首款蓝光播放器| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/30/daewoo-unveils-its-first-blu-ray-player/>

![](img/5b37c3a1341969a06e49430b3bfc3182.png)

在 IFA 期间，生产各种廉价和进口产品的大宇公司宣布，它将凭借其第一款播放器 DBP-1000 涉足蓝光市场。光滑的蓝色外观放在一边，播放器是苗条的，支持完整的 BD-R 规格。它可以访问实时内容，通过 HDMI 以 1080p 格式输出，将 DVD 和照片放大到全尺寸，还可以播放 DivX 文件。

DBP-1000 还可以播放任何刻录的 CD 或 DVD，以及刻录的 BD-R 和 BD-RE 光盘，以防您的朋友有大量数据(或刻录的电影)。您所期望的所有输入都可用，包括数字声音，闪存驱动器的前面还包括一个 USB 端口。听起来是个很棒的球员。尽管我很想染指它，但大宇没有提及任何关于价格或可用性的事情。
