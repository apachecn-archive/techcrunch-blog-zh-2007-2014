# Smule 再次超越自己:Leaf 长号是即时 iPhone 经典 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/04/15/smule-outdoes-itself-again-leaf-trombone-is-an-instant-iphone-classic/amp/>

去年 iPhone 应用商店推出时，从钢琴到鼓等一系列“虚拟”音乐应用涌入市场。其中大多数在 iPhone 的小屏幕上表现不佳，除了一个非常突出的产品: [Smule 的](https://web.archive.org/web/20230118102210/http://www.smule.com/) [Ocarina](https://web.archive.org/web/20230118102210/http://techcrunch.com/2008/11/07/smules-ocarina-a-textbook-example-of-how-to-build-a-great-iphone-app/) ，它非常适合 iPhone，初学者也很容易上手。这款应用获得了巨大的成功，并成为 App Store 有史以来最受欢迎的应用之一。今天，Smule 发布了 [Leaf 长号:世界舞台](https://web.archive.org/web/20230118102210/http://itunes.apple.com/WebObjects/MZStore.woa/wa/viewSoftware?id=309080428&mt=8)，这是一种新的虚拟乐器，几乎在所有方面都比陶笛好。

新手使用陶笛的最大问题之一是没有内置的音乐指导——用户必须访问 Smule 的网站，在那里他们可以下载他们最喜欢的歌曲的“标签”。如果没有标签，播放熟悉的音乐就成了一项令人沮丧的练习，我怀疑许多用户从未登录过网站，只是试图自己做到最好。

Leaf 长号通过内置的教程模式解决了这个问题，它可以向玩家显示他们应该将手指放在哪里(它还有一个很好的背景音乐，让你听起来更好)。使用集成浏览器，用户可以对其他用户创作的课程进行分类，从《D 中的佳能》(Canon in D)等经典歌曲到《超级马里奥兄弟》(Super Mario Bros .)主题曲等更现代的歌曲。单单这个特性就能让这个应用比 Ocarina 更容易上手(也更容易上瘾)。

这款应用的“世界舞台”也不仅仅是为了展示。虽然陶笛有某种程度的内置社区(你可以在其他陶笛演奏者演奏歌曲时实时收听)，但实际上没有任何方法可以与他们互动。Leaf 长号邀请用户在演奏时评判他们的同伴，允许他们实时显示表情符号和发送评论。Leaf Trombone 通过使用 Smule 的创新音频广播技术实现了这种实时交互，该技术可以再现玩家呼吸控制的细微差别，而不必传输海量音乐文件。其他功能，如内置的成就系统(想想摇滚乐队)，允许用户在完成特定任务时获得奖励，为应用程序增加了另一层深度。

我非常喜欢 Leaf 长号——它利用了 iPhone 的网络效果，演奏起来很有趣，任何人都可以轻松上手。但是我有一个主要的担心:你正在演奏一把长号。不是侮辱所有的长号手，但这种乐器对耳朵来说并不特别舒服。与陶制卵形笛不同，陶制卵形笛被神秘的气氛所包围，具有宁静、舒缓的声音，长号则有点…刺耳。也就是说，应用程序不仅仅弥补了这一点，如果这成为一个问题，Smule 几乎可以肯定地用一组新的声音更新应用程序。
