# TwittEarth 让 Twitter 成为一种全球体验

Twitter 可视化并不新鲜，但它们总是提供了一个浪费时间的好方法。最新的 [TwittEarth](https://web.archive.org/web/20221208225556/http://twittearth.com/) 是最吸引眼球的——一个迷人但几乎无用的娱乐，它让你呆在太空中，并让你在全球范围内每十秒钟就能看到一条新的地理位置推文。推文伴随着静止在地图上的小而傻的图标，最终提供了一个有趣的使用分布的表示。

在这一点上，应用程序本身非常简单。似乎没有什么设置可言，也没有办法偏离默认视图。用户可以登录该应用程序并发送自己的消息，但更有趣的是，用户可以分区并观看全球各地弹出的推文。

TwittEarth 是 Windows 的屏幕保护程序(Mac 版本即将推出)，由 T4 的 Digitas France SA 开发。
