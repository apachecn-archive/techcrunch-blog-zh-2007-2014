# 谷歌解释如何不成为一个玻璃洞

以下是佩戴谷歌眼镜的[注意事项和](https://web.archive.org/web/20221203171140/https://sites.google.com/site/glasscomms/glass-explorers)注意事项。就在谷歌上。

显然——我知道这可能令人震惊——你不应该站在房间的角落里用谷歌眼镜记录人们。根据这个列表，你会变成一个玻璃洞。

在这一点上，谷歌的挑战不是建立眼镜平台，而是培训普通公众欢迎眼镜佩戴者进入社会。谷歌眼镜的未来很大程度上取决于公众对这项技术的接受程度。如果像蓝牙耳机一样，它被视为书呆子，或者更糟的是，如果谷歌眼镜被归入美国国家安全局隐私扫描，这项技术将是一个失败者。很多事情都取决于谷歌眼镜探索者。

谷歌推出了谷歌眼镜，但该公司并没有向公众宣传这项技术。对于大多数人来说，他们与设备的唯一交互是与一个随机佩戴谷歌眼镜的人进行的。这些所谓的探索者，无论好坏，都是玻璃的拥护者。“不允许使用玻璃”运动清楚地表明，这些倡导者没有把玻璃放在最好的位置。

在[的“该做什么，不该做什么”列表中的最后一点](https://web.archive.org/web/20221203171140/https://sites.google.com/site/glasscomms/glass-explorers)指出:

> **不要让人毛骨悚然或粗鲁无礼(又名“玻璃洞”)。尊重他人，如果他们对玻璃有疑问，不要生气。礼貌地解释一下 Glass 的功能，并且记住，一个快速的演示会大有帮助。在不允许手机拍照的地方，同样的规则也适用于玻璃。如果你被要求关掉手机，也关掉谷歌眼镜。打破规则或无礼不会让企业对谷歌眼镜感到兴奋，反而会毁了其他探索者的谷歌眼镜。**
