# Prompt.ly 筹集了 150 万美元的种子资金，成为时间和服务 TechCrunch 的 OpenTable

> 原文：<https://web.archive.org/web/http://techcrunch.com/2013/11/25/prompt-ly-raises-1-5m-seed-to-become-the-opentable-for-time-and-services/>

我们已经看到了像 TaskRabbit 和 FancyHands 这样的初创公司形式的“按需”经济的崛起。我们较少看到的是一个成熟的“平台游戏”，它不是关于连接买家和卖家(TaskRabbit)，而是关于根据时间管理业务(如 OpenTable)。在英国和美国运营一项服务的 BookingBug 已经这样做了三年，而且相当成功。但直到最近，它在平台领域或多或少都是一个人。

BookingBug 从网络中发展出来，拥有也接受支付的[移动应用](https://web.archive.org/web/20230210105447/https://techcrunch.com/2013/03/08/europes-square-izettle-now-powers-mobile-payments-in-third-party-apps-sees-first-rollout-of-api-with-bookingbug/)，但这里的需求是为*任何*类型的服务提供商提供一个更加以移动为中心的 SaaS 平台，使任何企业都能加入这场全球移动革命，在这场革命中，与音乐老师、清洁工、商业教练或私人教练相处只需轻轻一敲。现在，在这个更加面向移动的时代，一家新的创业公司即将崭露头角。

Prompt.ly 有一个明星制的团队，包括著名的投资者和顾问。领导该公司的是理查德·泰特斯，他已经是一位经验丰富的科技企业家。你可以[在这里](https://web.archive.org/web/20230210105447/http://prompt.ly/#betasignup)注册该应用的测试版，但目前只有 Android 应用可用。iPhone 应用程序正在等待批准。

Prompt.ly 现在已经获得了几个关键天使的投资，包括来自脸书的 Ethan Beard 和包括减法资本(由 PayPal 的 Jason Portnoy 和 Paul Willard 创立的新基金)在内的几个基金。Titus 表示，他们将在圣诞节前完成 150 万美元的种子资金，“首轮讨论正在升温。”

泰特斯是洛杉矶 Razorfish 的联合创始人，也是联合诺斯克利夫数字公司的首席执行官，该公司创建了《每日邮报在线》，并使 BBC 的数字部门成形。他的联合创始人 Eli-Shaoul Khedouri 是首席技术官，曾是 Everest(另一家移动创业公司)的创始人，也是纽约信息安全公司(infosec)的前首席架构师。

Titus 告诉我，Prompt.ly 以“时间被打破”为前提，需要一个深度的、专门构建的平台来进行日程管理和时间管理。

在这个领域攻击企业是很难的，但是按需任务有一个巨大的无形经济(正如 Task Rabbit 和其他人所展示的)。

他称这个市场为“隐形经济”。它由遛狗者、私人教练、巡回网页设计师、杂工组成，你能想到的都有。而且都是基于时间的。

但是他们已经做了研究。利用来自小企业管理局和国税局的数据，他们估计仅在美国就有 2000 万到 6000 万人陷入这种“小时间段”的工作模式，使其在美国的价值估计为 1.27 万亿美元，占五分之一的购买量。

大多数购买是基于时间的——按小时或按服务。

在对流动工人进行了数百次采访后，他们发现了 1200 种按小时或时段出售时间的人。从律师到保姆。“这是经济的一个巨大部分，但它没有后台，”Titus 说。“不到 150 名受访者拥有电脑，但都有智能手机。”

他或她没有后台；是移动优先还是仅移动；接受现金和/或信用卡。他说，这相当于美国国内生产总值的 30%，5600 万个工作岗位和美国就业人数的 44%。

这不像 Square(这是零售空间)，它是关于销售多余的时间，所以它是关于时间的付款而不是商品的付款跟踪。现在这是一个非常破碎的经历。“我们更接近开放的桌子或广场，而不是 Yelp，”Titus 说。

这一部门经常会丢失账单和收款记录，从而造成收入损失。现在它是通过电话和几十条短信来管理的。一旦他们失去工作，他们就无能为力了。此外，他们很难填补未预订的时间。

Prompt.ly 将采用免费增值模式——先免费，然后每月付费 9.99 美元，按功能分层提供服务。从理论上来说，无论服务是什么，第一次重新预订就可以收回成本。
