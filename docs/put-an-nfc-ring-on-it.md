# 这个 NFC 戒指将无线传输技术放在你的手指上，这样你就可以用拳头碰手机了

科技界可能希望苹果在不久的将来推出一些可穿戴技术，但当 NFC 戒指已经成为一件事情时，谁会关心 iWatch 呢？NFC 粉丝——你知道，[这些渴望随身携带无线传输技术的人](https://web.archive.org/web/20230306065259/http://www.youtube.com/watch?v=AwTzPnWsz20)——应该把目光投向这个总部位于英国[的 Kickstarter 项目](https://web.archive.org/web/20230306065259/http://mclear.co.uk/)。

NFC 戒指旨在与支持 NFC 的小工具(如智能手机和平板电脑、NFC 门锁等)配合使用，因此佩戴者可以通过握拳或张开手掌的手势来执行一些任务，如解锁手机、打开前门或向另一名智能手机用户发送联系信息、个人网站或比特币地址。

如果你愿意戴上一点珠光宝气，这个相对纤细的戒指可以避免一些与碰撞和摩擦 NFC 手机进行数据传输相关的尴尬(又名“[手机性爱](https://web.archive.org/web/20230306065259/https://twitter.com/karaswisher/status/344934064278630400)”)。作为被动 NFC，它也不需要任何电池电源来工作。所有的编程都是通过应用程序完成的，包括第三方应用程序，如果你想用它来解锁你的手机。NFC 戒指的创建者还制作了他们自己的开源软件，用于向戒指写入动作。

这款戒指实际上有两个 NFC 镶嵌物，一个是较大的“公共”区域，设计用于佩戴在顶部，通过拳头撞击手势发送联系数据等信息，另一个是较小的“私人”区域，设计用于佩戴，因此可以隐藏在手掌中，只能通过张开手掌的手势激活。该区域旨在存储更多敏感数据，如前门和手机解锁信息，创建者约翰·麦克利尔说。

双 NFC 设计旨在帮助防止戒指主人的最敏感数据被附近的 NFC 读卡器窃取。但值得指出的是，珠宝很少会固定不动，因此 NFC 戒指持有者可能会花很多时间来检查他们的珠宝是否正确对齐。或者喝醉了，没有把联系信息发给他们在酒吧里试图搭讪的人，而是试图打开一扇不存在的前门。这是一个全新层次的极客失礼。

NFC 戒指基本上是 NFC 智能手机有时附带的 NFC 标签的更方便(即可佩戴)版本。但是由于尴尬是 NFC 更广泛使用的一个障碍，任何有助于减少摩擦的东西对非接触式兄弟会来说都是一个受欢迎的发展。

NFC 戒指正在快速接近其 3 万英镑的 Kickstarter 筹资目标——在活动还剩 27 天的时候，近 1100 名支持者承诺了近 2.95 万英镑。创作者表示，如果一切按计划进行，戒指将于 10 月下旬上市。每枚戒指的成本约为 25/38 美元，但创作者还发布了一个 CAD 设计，因此基本的戒指形式可以被黑客攻击、定制和 3D 打印。然后，他们提供更便宜的承诺，只运输 NFC 镶嵌物，用于固定在这些 3D 打印的戒指内。

值得注意的是，“正常尺寸的戒指”(约 7 毫米宽)并不适用于所有的 NFC 手机——特别是三星 Galaxy S4 和黑莓 Z10 不会玩得很好。对于这些设备，创造者警告说“你将需要一个阿尔法大小的戒指”，他们补充说“真的只适合手指大的家伙”。
