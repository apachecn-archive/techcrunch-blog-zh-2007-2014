# 大宇 DPC-8099OD-I 是一款真正的便携式 DVD/iPod 播放器

![](img/819b205c5b77eff16be0b453c4ed88dd.png)

尽管这个东西的型号名称很可笑，但大宇最新的便携式 DVD 播放器看起来像是将所有正确的功能都打包到了一个小外形中。首先，在你插入 DVD 的地方旁边有一个 iPod 的插槽，这意味着如果你用完了《发展受阻》第二季的光盘，你可以在旅途中从你的 iPod 上观看视频。还支持 VCD、CD、CD-R、MP3、JPG 和 DivX 视频。

你将在一个宽大的 8 英寸液晶显示器上观看你所有的喜剧表演，这个尺寸正好适合去海滩短途旅行或坐飞机。有一个 USB 端口和存储卡读卡器，以防您在跑步时想要观看媒体。太糟糕了，还没有给出价格或发布日期。保持在 150 美元以下，我会买一个支持 iPod 的便携式 DVD 播放器。
