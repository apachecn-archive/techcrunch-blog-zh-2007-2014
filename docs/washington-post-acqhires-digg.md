# Digg 团队要去 WaPo，但是资产不去

《华盛顿邮报》是否收购了 Digg，[如下期网络](https://web.archive.org/web/20221209064713/http://thenextweb.com/insider/2012/05/01/rumor-digg-to-be-acquired-by-the-washington-post/)所报道？算是吧。

我们听说邮报已经雇佣了 Digg 团队，但是没有收购网站或者技术。换句话说，这是一次人才收购，事实上 Digg 的财产、专利和资产仍在出售。

TheNextWeb 的报道惹了一些“[什么鬼？](https://web.archive.org/web/20221209064713/https://twitter.com/mathewi/status/197092738515206144)[推特上的反应](https://web.archive.org/web/20221209064713/https://twitter.com/edbott/status/197093754597945344)，但是一个人才收购更有意义一点。《邮报》并没有试图重振一个苦苦挣扎的社交新闻网站。相反，它正在招募一个团队来帮助开发其 Trove 社交新闻平台，以及 Trove 驱动的产品，如社交阅读器和个人帖子。

与此同时，Digg 似乎正在衰落，尤其是在 2010 年有争议的重新设计之后。联合创始人[凯文·罗斯于 2010 年初接任首席执行官](https://web.archive.org/web/20221209064713/https://beta.techcrunch.com/2010/04/05/kevin-rose-one-of-us-has-to-leave/)，但[一年后卸任，担任更具顾问性质的角色](https://web.archive.org/web/20221209064713/https://beta.techcrunch.com/2011/03/18/kevin-rose-resigns-from-digg-closing-round-on-new-startup/)。(他的新创公司 [Milk 被谷歌](https://web.archive.org/web/20221209064713/https://beta.techcrunch.com/2012/03/15/winning-a-bidding-war-with-facebook-google-picks-up-the-entire-milk-team/)收购。)据推测，[由于与脸书时间轴](https://web.archive.org/web/20221209064713/https://beta.techcrunch.com/2012/02/24/be-yourself/)的整合，流量已经开始再次增长，但这可能还不足以让该公司或其投资者相信这里有一个真正的未来。

我已经联系了《华盛顿邮报》和 Digg 寻求评论，如果有回音，我会及时更新。(**更新:**邮报发言人表示公司无可奉告。)
