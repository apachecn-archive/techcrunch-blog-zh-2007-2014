# Outlook.com 用户现在可以和他们的谷歌朋友聊天了

这里有一些你可能没有预见到的事情:[Outlook.com](https://web.archive.org/web/20230206225416/http://outlook.com/)刚刚[启用了](https://web.archive.org/web/20230206225416/http://blogs.office.com/b/microsoft-outlook/archive/2013/05/14/outlook-com-now-lets-you-chat-with-google-friends.aspx)与 [Google Talk](https://web.archive.org/web/20230206225416/http://www.google.com/talk/) 的聊天互操作性。这项新功能将在未来几天内在全球推出，它允许 Outlook.com 用户在谷歌上与他们的朋友聊天，就像他们已经可以与他们的脸书朋友聊天一样。鉴于微软和谷歌之间有些紧张的关系，这一举动有点令人惊讶，但看起来微软并不认为这一推出会有任何问题。

新的聊天功能将在许多与 Outlook.com 相关的产品中提供，包括您的收件箱、日历、通讯录和 [SkyDrive](https://web.archive.org/web/20230206225416/https://skydrive.live.com/) ，例如，您可以在处理文档时与您的朋友在谷歌上聊天。

正如微软 Outlook.com 高级产品经理 Dharmesh Mehta 昨天告诉我的，微软从用户那里听说聊天互操作性是“阻碍人们从 Gmail 切换到 outlook . com 的原因之一。”他补充说，许多已经切换的用户说，这是“他们在切换后错过的”一个功能

[![outlook-with-chat](img/a91cd6617f689d867beaa9bbacd696e2.png)](https://web.archive.org/web/20230206225416/https://techcrunch.com/2013/05/14/outlook-com-users-can-now-chat-with-their-google-friends/outlook-with-chat/)

要在 Outlook.com 启用谷歌聊天，用户只需使用谷歌的标准 OAuth 系统连接他们的账户，让微软访问他们的账户。之后，他们可以将鼠标悬停在 Gmail 用户的联系线上，或者直接从标准聊天窗格开始新的聊天。

不过，有一件事目前不太管用，那就是启动包括 Gmail 和脸书用户在内的群聊。Mehta 保留了微软在未来实现这一功能的可能性，但目前，该团队还没有构建出允许微软在网络之间传递信息的部件。

外界普遍预计，谷歌将于本周晚些时候在 I/O 上发布自己的文本、音频和视频聊天功能的更新。然而，这些不太可能对微软今天宣布的新功能有任何影响。
