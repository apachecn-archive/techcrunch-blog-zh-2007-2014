# Voxy 筹集 280 万美元帮助人们通过智能手机学习语言

TechCrunch Disrupt 决赛入围者 [Voxy](https://web.archive.org/web/20230203115747/http://www.voxy.com/) 今天宣布新一轮 A 轮投资，来自投资者 [Seavest Inc](https://web.archive.org/web/20230203115747/http://seavestinc.com/) 、 [ff Venture](https://web.archive.org/web/20230203115747/http://ffventure.com/) 和 [Contour Ventures](https://web.archive.org/web/20230203115747/http://www.contourventures.com/) 的 280 万美元。相对较新的投资者 Seavest 最近将另一家与教育相关的投资组合公司 Schoolnet 以 2.3 亿美元的价格出售给了教育出版商 Pearson。

除了融资之外，Voxy 还推出了它的最新产品，一款教葡萄牙语使用者英语的 iPhone 应用程序(到目前为止，它所有的产品都是教西班牙语使用者英语)。

在与 Live Mocha 相同的领域，Voxy 通过从媒体和对话等日常生活材料中挑选内容来区分自己，并且首先是移动的。“我们相信，在许多发展中市场，人们将会跳过基于浏览器的应用程序，转而使用移动设备”创始人 [Paul Gollash](https://web.archive.org/web/20230203115747/http://www.crunchbase.com/person/paul-gollash) 告诉我。Voxy 提供免费和付费服务——这款智能手机产品为希望增加功能的用户提供每月 2.99 美元的起价。

Gollash 计划利用新的资金雇佣更多的工程师，扩大产品规模，改进现有技术。他的未来计划包括提供英语到西班牙语版本的服务，并扩展到其他语言和市场。他表示:“我们将继续利用移动技术来满足那些缺乏时间、金钱和渠道来使用传统的基于浏览器的离线方法的用户的需求。”。

Gollash 还告诉我，在巴西市场推出后的一天，Voxy 已经是教育类别的前十名应用程序。巴西的人口是一亿九千三百万。
