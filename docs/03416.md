# 脸书再次转向 Twitter 寻求灵感，将@ Tagging 引入状态更新| TechCrunch

脸书今天推出了一项有趣的功能:可以在发布者的状态更新和其他消息中标记朋友。和 Twitter 一样，你用“@”符号后跟朋友的名字(或脸书页面或群组的名称)来标记一些东西，但显示的消息没有符号，只有一个指向此人个人资料的链接。

“标记”朋友的功能是脸书几年前通过照片标记功能普及的。正如脸书在宣布新的“@”功能的博客文章中指出的，标记是社交网络上最受欢迎的功能之一。我敢说，这可能是让脸书的社交技术如此独特的早期特征之一。

但是 Twitter，或者至少是它的用户，开创了“@”功能来标记朋友，这在脸书的状态更新中已经消失很久了。也就是说，无论是从用户界面的角度来看，还是从脸书的标签工作方式来看，都有一些关键的区别。在 Twitter 上，你必须记住朋友的用户名(或者使用具有自动完成功能的第三方客户端)，而在脸书上，当你开始在“@”符号后输入朋友的名字时，你会看到一个下拉菜单，让你从朋友、群组、活动和应用程序列表中进行选择。

虽然 Twitter 上的@replies 经常用于对话，但在脸书上的使用情况有所不同。当您在状态更新中标记某人时，此人将会收到通知和墙贴，从而将此人链接到您的状态更新。与照片中的标记类似，用户可以选择移除标记。据脸书内部[报道](https://web.archive.org/web/20230404070649/http://www.insidefacebook.com/2009/09/10/facebook-launching-status-tagging-for-friends-pages-events-and-groups-today/)该功能将在未来几周内缓慢推出。

脸书一直在稳步发展其社交网络(发展速度比 Twitter 更快),[最近进行了一项收购，这是脸书](https://web.archive.org/web/20230404070649/https://techcrunch.com/2009/08/10/facebook-takes-friendfeed-to-take-on-twitter/)[在与 Twitter 的稳步竞争中的积极策略](https://web.archive.org/web/20230404070649/https://techcrunch.com/2009/06/17/phase-4-of-facebooks-systematic-attack-on-twitter-the-everyone-button/)的代表。虽然与收购 FriendFeed 相比，状态更新的“@”功能可能很小，但它对于脸书和 Twitter 之间正在进行的竞争来说意义重大。
