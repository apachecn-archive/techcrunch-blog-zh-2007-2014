# 喷火战机乐队将于明晚在脸书直播演唱会

喷火战机乐队将于明晚在脸书直播一场演唱会，直播将从该乐队位于洛杉矶的 606 工作室总部进行。你可以在这里回复活动[，在这里](https://web.archive.org/web/20221207191653/http://www.facebook.com/event.php?eid=317195110480)直播明天太平洋时间下午 7 点开始的活动[。与脸书最近的其他视频流媒体活动一样，用户将被邀请更新他们对演唱会的想法，这将吸引更多的粉丝，因为他们可以在新闻订阅中看到更新。现场视频将由](https://web.archive.org/web/20221207191653/http://www.facebook.com/foofighters?v=app_315273185327) [Livestream](https://web.archive.org/web/20221207191653/http://www.livestream.com/) 提供支持。

对于网络上最受欢迎的内容和社交网站来说，现场音乐会流似乎是一种增长趋势。上周，MySpace [播放了他们在旧金山的秘密演唱会。本周早些时候，YouTube](https://web.archive.org/web/20221207191653/http://blogs.myspace.com/index.cfm?fuseaction=blog.view&friendId=438372074&blogId=515285549) [在加州帕萨迪纳的玫瑰碗直播了一场 U2 的演唱会。该活动取得了巨大的成功，吸引了](https://web.archive.org/web/20221207191653/http://www.beta.techcrunch.com/2009/10/25/tonight-you-too-can-watch-u2-on-youtube-live/)[报道](https://web.archive.org/web/20221207191653/http://www.beta.techcrunch.com/2009/10/29/with-not-without-u2-youtube-saw-10-million-streams-sunday-night/)1000 万的流量。YouTube 还在八月份直播了旧金山的户外音乐节。Hulu 也加入了这一行动，在五月推出了戴夫·马修斯乐队的流媒体服务。

对乐队的吸引力是显而易见的——他们有机会回馈粉丝，推广他们的专辑，并提高他们在社交网络上的存在。像脸书这样的网站受益于吸引更多的艺术家到他们的平台，以及来自广告和赞助协议的显而易见的收入机会。

尚不清楚脸书是否真的赞助了这个视频流，还是喷火战机乐队自己选择了这个平台。我们已经联系了脸书进行澄清。
