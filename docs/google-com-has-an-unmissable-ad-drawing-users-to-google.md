# Google.com 有一个不容错过的广告吸引用户使用 Google+

驶向 Google.com，你会发现自己正凝视着或许是迄今为止最大胆的涂鸦。不，谷歌没有超越自己，推出了另一款[音乐致敬](https://web.archive.org/web/20230204231851/https://techcrunch.com/2010/10/08/imagine-google-features-its-first-video-doodle-in-honor-of-john-lennons-70th/)或[互动游戏](https://web.archive.org/web/20230204231851/https://techcrunch.com/2010/05/24/googles-pac-man-logo-will-live-forever/)——它正在播放一则广告，引导用户试用 Google+。但这不仅仅是普通的文本广告或横幅:谷歌在其顶部导航栏的左侧画了一个指向你的 Google+名称的蓝色大箭头。点击它，你将进入 Google+主页并注册。

你不会错过的。

这则广告恰逢今天 Google+的[公开发布](https://web.archive.org/web/20230204231851/https://techcrunch.com/2011/09/20/google-plus-beta/)，现在任何人都可以注册这项服务(到目前为止，你需要一个邀请)。这肯定会推动新注册人数的激增。

诚然，这并不是谷歌第一次在主页上投放广告来推广自己的产品:例如，它以前曾为其 [Nexus 手机](https://web.archive.org/web/20230204231851/https://techcrunch.com/2010/01/06/nexus-one-google-homepage/)投放广告，最近[又为某些地区的用户投放了文字广告来推广谷歌的产品。但就我记忆所及，所有这些广告都很微妙——一些文本，也许是搜索栏下方居中的一个小图形。](https://web.archive.org/web/20230204231851/http://www.pcmag.com/article2/0,2817,2392318,00.asp)

但是这个蓝色的箭头很难被忽略(注意这个箭头实际上移动了一点，因为它是在你第一次登陆页面时“画”出来的)。如果没有别的，这是又一次提醒人们谷歌对帮助 Google+取得成功是多么认真，这表明它愿意利用其最大的资产来实现这一使命。
