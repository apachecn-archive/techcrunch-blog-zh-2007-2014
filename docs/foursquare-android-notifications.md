# 通过“超越签到”通知，Foursquare 实现了 Android 优先 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/07/05/foursquare-android-notifications/>

# 随着“超越签到”通知的出现，Foursquare 将 Android 放在了首位

在其历史上，Foursquare 一直是一家以 iPhone 为先的公司。该应用程序最初于 2009 年在 iPhone 上推出，新功能通常会首先在 iPhone 上推出。但是今天，Foursquare 有了一个新功能，首先是 Android(和 web)。

这个新功能很不错:[通知](https://web.archive.org/web/20230205045523/http://blog.foursquare.com/2011/07/05/notifications/)。与你习惯在 Foursquare 移动应用上看到的签到推送通知不同，这些新通知专注于 Foursquare 上“签到之外”的其他活动。这意味着像签到和照片的评论，朋友注册时的提醒，关于小费的提醒，当你被罢免市长时的提醒，蜂拥而至的提醒等等。

但是，同样有趣的是新的 Android 优先的方法。正如 Foursquare 所言:

> Android-iverse 中的人们，你们是第一个玩这个的；下载今天的更新开始使用。其余的人，去 foursquare.com 看看都是些什么(只需点击“通知”)。

在帖子的底部，Foursquare 重申这项功能目前仅适用于 Android 和 web，但承诺 iPhone 和黑莓用户将“尽快”获得新功能。

多年来，Foursquare 几乎不是唯一一个专注于 iPhone 优先的主要应用程序。事实上，许多大公司仍然如此。这让一些人感到困惑，因为 Android 拥有更大的整体市场份额。这对其他公司来说可能是一个预兆吗？或者这次是不是更容易为 Android 完成和推出这个特别的功能？

**更新**:我问联合创始人丹尼斯·克劳利关于新的 Android 优先的方法，他说，“Android 需要一些早期接入的爱！”“作为一家公司，我们努力工作，让不同的产品在不同的客户身上有不同的节奏——所以不总是 iPhone，Android，然后是黑莓，”他继续说。

“此外，它给了我们一个更小的子集来测试，”克劳利还说，并指出该团队正处于他们“启动”的点上。他还暗示我们应该期待更多的定期推送。
