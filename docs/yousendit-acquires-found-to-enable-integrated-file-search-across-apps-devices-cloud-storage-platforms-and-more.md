# YouSendIt 收购 Found 以实现跨应用程序、设备、云存储平台等的集成文件搜索

云协作公司 [YouSendIt](https://web.archive.org/web/20221209150436/http://www.yousendit.com/) 已经收购了 [Found Software](https://web.archive.org/web/20221209150436/https://www.foundapp.com/) ，这是一家允许用户跨设备和云服务搜索、查找和发现他们的文件、文档和数据的初创公司。收购价格没有透露，但我们听说是现金加股票的交易。

Found 的 Mac 应用程序为用户提供了跨桌面、应用程序、云存储提供商等的单一集成文件搜索功能。你可以从 Dropbox、SkyDrive、Evernote、Google Drive、Gmail 附件、你的电脑等查看你的文件。

当您键入时，find 会显示您家用机器上的相关文档以及您在云中的各种帐户。当您将鼠标悬停在每个搜索结果上时，find 会显示该文件的完整图像预览。它还允许您将找到的搜索结果中的文件拖放到 Gmail 邮件中，以便快速发送和共享文件。由于 Found 运行在 Mac 上，它本质上是苹果 Spotlight 搜索功能的更好版本。

作为背景，YouSendIt 是文件共享的[先驱之一，提供的服务主要吸引防火墙后的客户，他们由于微软 Exchange 的限制而无法通过电子邮件发送大文件。该公司随后扩展到跨多个平台的文件共享，包括手机和桌面。去年，YouSendIt](https://web.archive.org/web/20221209150436/https://beta.techcrunch.com/2011/12/13/yousendit-ups-its-game-with-new-desktop-mobile-apps/) [聘请](https://web.archive.org/web/20221209150436/https://beta.techcrunch.com/2012/05/15/brad-garlinghouse-becomes-ceo-of-booming-file-sharing-site-yousendit/) Brad Garlinghouse 担任首席执行官，因为该公司扩大了其在企业存储和协作市场的足迹。

仅去年一年，YouSendIt 就实现了 5700 万美元的收入，比 2011 年增长了 50%，注册用户超过了 4000 万。

加林豪斯解释说，通过这次收购，YouSendIt 的用户将能够通过许多不同的应用和服务访问文件。他解释说，认为企业用户只会使用一家存储提供商或内容提供商是天真的想法，因此 YouSendIt 正在走向“云-nostic”的未来。

一旦 Found 被整合到 YouSendIt 中，用户就不必在这些多种云和设备中寻找信息和应用程序。我们被告知，已经从 NEA 和其他人那里筹集了 100 万美元资金的 Found 将继续作为一个独立的应用程序向用户开放。该应用的技术也将用于 YouSendIt 上的搜索。

在未来，Found 的技术还将利用文件中的元数据为用户提供见解。例如，在参加会议之前，您将能够快速方便地知道谁访问了某个文件、该文件包含什么内容以及更多信息。
