# 尽管许多投资者都在打转，但 Turntable.fm 还没有挑选出一位 DJ TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/07/07/investors-turntable-fm/>

尽管许多投资者都在寻找投资于 [Turntable.fm](https://web.archive.org/web/20230203045247/http://turntable.fm/) 的机会，这家热门音乐初创公司还没有挑选 DJ，尽管有相反的报道。 [Business Insider 声称【Turntable 已经以 3750 万美元的估值筹集了 750 万美元，而且“条款清单确实是昨天签署的。”但几个小时前，当他登上飞机时，联合创始人兼董事长塞思·戈尔茨坦告诉我，“我们还没有完成任何新的融资。”](https://web.archive.org/web/20230203045247/http://www.businessinsider.com/turntablefm-rumored-to-be-raising-5-10-million-at-a-40-million-valuation-2011-7?op=1)

风投们肯定对转盘有很大兴趣，他们希望为转盘的下一轮融资。风险投资者议论纷纷，认为这笔交易竞争激烈，尤其是联合广场的弗雷德·威尔逊(Fred Wilson)、Accel 和凯鹏华盈(Kleiner Perkins)之间。威尔逊显然是最受欢迎的(转盘总部位于纽约市)，但他被 Accel 和 Kleiner 击败。

传闻是威尔逊向转盘提供了 2500 万美元的预投资估值，而 Accel 和 Kleiner 提供了双倍的估值。这很容易被推高到 3，000 万美元的前期投资，在这种情况下，3，750 万美元的数字将作为投资后的估值。(只要记住，风投一直都在发签过字的投资意向书。不代表公司要接受条款)。

不仅顶级风投对转盘感到兴奋，甚至一些潜在的收购者也在四处打探，包括美国在线、脸书和索尼。据我所知，没有任何正式报价，因为联合创始人戈尔茨坦和比利·蔡森刚刚开始，不想出售。此外，他们显然在筹集资金方面没有任何问题。

为什么每个人都对一家推出了完全不同的产品 [Stickybits](https://web.archive.org/web/20230203045247/http://www.stickybits.com/) 的初创公司如此着迷，而[从未走向任何地方](https://web.archive.org/web/20230203045247/https://techcrunch.com/2010/09/20/stickybits-object-check-ins/)？转盘是一个社交音乐网站，你可以进入不同的试听室，玩家可以成为 DJ，一边旋转音乐，一边与彼此的头像聊天。克里斯·萨卡(Chris Sacca)喜欢在那里闲逛(他是 First Round 和 Polaris 的前投资者)。它已经获得了一些令人印象深刻的早期牵引力，尽管你仍然需要认识一些人才能进入。

这是最好的社交音乐发现。你可以在各种不同主题的音乐室里听其他玩家挑选的几个小时的全长歌曲。DJ 越好，其他人奖励给他的分数就越多。如果你喜欢一首歌，你可以把它添加到你的播放列表中，或者通过亚马逊或 iTunes 购买。

但这项业务并非一帆风顺。Turntable 像 Pandora 或其他“网络电台”服务一样支付每流费用。唱片公司[可能会决定打击转盘上的](https://web.archive.org/web/20230203045247/http://allthingsd.com/20110621/turntable-fm-really-is-awesome-is-it-legal/),并试图收取更高的费用。但转盘的早期增长和参与数字太高，不容忽视。人们在这些房间里呆上几个小时。也许这一次，在试图达成协议之前，唱片公司不会扼杀它。尽管如此，转盘将不得不寻找其他赚钱的方式——也许是通过数字商品或更好的化身，赞助商的房间，或者一些人可能愿意付费成为一名特色 DJ。
