# Xobni 来到黑莓(泄露的图片)

![](img/8e2095f0817e63436163303854d6a1f8.png)

电子邮件初创公司 [Xobni](https://web.archive.org/web/20221209001853/http://www.xobni.com/) 从黑莓合作伙伴基金获得[投资才一个月，这使其总投资](https://web.archive.org/web/20221209001853/http://www.xobni.com/blog/2009/03/25/blackberry-partners-fund-invests-in-xobni-and-we%E2%80%99ve-launched-xobni-17-no-more-beta/) [B 轮](https://web.archive.org/web/20221209001853/http://www.beta.techcrunch.com/2009/01/05/xobni-closes-7-million-b-round-led-by-cisco/)达到 1000 万美元，并且已经为即将推出的黑莓应用程序提供了工作原型。Xobni 高管昨晚在旧金山的一个移动会议上展示了这款应用，上面的截图进入了我的收件箱(你知道，它的拼写是“xobni”)。

据我的消息来源称，该应用正在运行，可能会在今年夏天的某个时候发布。上面的照片显示了黑莓 Bold 上的应用程序，似乎在展示其联系人搜索功能。你输入几个字母，它会返回你收件箱中每一个匹配项的联系信息(甚至是你还没有添加到你的地址簿中的人)。我想知道它还能做什么。

Xobni 是 Outlook 的一个插件，整合了来自各种社交网络的数据，目前还没有移动客户端。但该公司已经聘请了一个小型工程师团队来开发移动应用程序，黑莓是第一个获得移动应用程序的设备。

不仅与黑莓合作伙伴基金有联系，而且我在过去被告知 Xobni 用户和黑莓用户之间有大约 50%的重叠。(必须有一个 Outlook-Blackberry 心理轴心存在)。因此，首先追逐黑莓是有道理的。但这家初创公司不能忽视 iPhone。可以吗？
