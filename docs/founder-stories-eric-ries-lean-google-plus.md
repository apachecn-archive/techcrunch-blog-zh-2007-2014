# Eric Ries:Google+的发布有多精益？

在克里斯·迪克森采访*精益创业*作者[埃里克·里斯](https://web.archive.org/web/20230209125331/http://www.crunchbase.com/person/eric-ries)的最后一集[中，迪克森问他谷歌推出 Google+时是否“精益”？迪克森说，从某些方面来看似乎是这样，因为谷歌正在慢慢“推出一系列不同的东西，进行实验，而不是让我们说 Buzz 和 Wave。”](https://web.archive.org/web/20230209125331/http://www.crunchbase.com/person/chris-dixon)

里斯部分同意，但也回答说:“我仍然认为他们一开始做了很多不必要的宣传，他们一直在报告他们的虚荣指标:‘现在我们在 Google+上有 1000 万用户。’这让我很紧张。有多少参与用户？里斯还认为，当品牌变大时，就很难进行增量推广，所以更大的公司应该“在其他一些市场上，以不同的品牌名称私下进行实验，以便在媒体开始蜂拥而上之前，搞清楚事情基本上是不是行得通。"

两人总结时，迪克森问里斯他对创业公司的前景有什么看法。里斯认为流入创业领域的现金和创始人的河流最终会干涸，并引用他的帖子[冬天即将来临](https://web.archive.org/web/20230209125331/http://www.startuplessonslearned.com/2011/08/winter-is-coming.html)来说明这一点。然而，里斯认为这是一件好事，因为强者生存。

从繁荣/萧条周期中分离出来，里斯告诉迪克森，总的来说，“一个根本性的变化正在发生，我喜欢称之为创业运动。我相信，有史以来第一次，创业现在成了一项可行的职业。”他将其比作“100 年前管理职业的诞生”，并表示，“几十年后，企业家将是你名片上的一个头衔，即使你在一家大公司工作。”
