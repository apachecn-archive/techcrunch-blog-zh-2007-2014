第九城市推出移动游戏平台& SDK，让开发者进入中国市场TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/10/01/the9-launches-mobile-gaming-platform-sdk-to-give-developers-access-the-chinese-market/

规模庞大的纳斯达克上市中国游戏发行商和开发商第九城市去年在美国游戏市场进行了大量投资。(似乎反映出亚洲对美国公司的兴趣正在上升，尤其是游戏业。)作为其国际战略的一部分，第九城市一直在全速前进，为国际游戏公司和开发商创造更好的方式，以打入中国移动和社交游戏市场，这一直是非中国开发商难以有效进入(并盈利)的传统领域。

例如，今年5月，第九城市与英特尔和时代华纳联手对众星进行了2300万美元的投资。今年早些时候，它还创建了一只1亿美元的基金(名为Fund9)，专注于投资中国和海外的游戏公司。

与此同时，第九城市还宣布与Aurora Feint签署了一份为期5年的许可协议，允许其在中国使用该公司的OpenFeint移动社交游戏网络软件(可在iOS和Android上运行)。

此后不久，该公司开始使用1亿美元的基金，与OpenFeint合作，将国际Android开发商的精选游戏引入中国市场。

今天，第九城市正在添加剩余的部分，再次利用其OpenFeint许可协议和可观的游戏基金——但这一次是iOS。该公司今天宣布，它将为其移动和社交游戏平台发布一个iOS软件开发包(SDK)，名为The9 Game Zone(当然，它由OpenFeint提供支持)。这很有趣，因为iOS上的Game Zone使国际开发者能够“在在线和移动游戏中实现社交功能”，这是为“中国8亿多游戏市场”量身定制的，第九城市移动业务副总裁Chris Shen说。

当进入中国移动游戏市场时，美国和国际开发商遇到了许多障碍，包括本地化要求和碎片化。沈说，与美国和西方世界相比，中国的应用市场很复杂，有100多家应用商店在运营，每个大型中国原始设备制造商以及大量第三方都在试图进入这个市场。因此，访问这么多应用商店以保证大规模分发、外国应用在中国市场的本地化、特定市场的货币化等等是一个问题。

那么，第九城市的平台和SDK的优势在于，该公司已经与30多家中国最大的应用商店合作(据该团队称，这些商店占中国应用的90%)，包括中国三大电信运营商推出的应用商店。

因此，通过在iOS上使用九城游戏专区，手机游戏开发商可以将他们的独立手机游戏改造成针对中国市场优化的互动社交游戏。很快，开发者也将能够在这30个应用商店上交叉推广他们的游戏，用虚拟商品和货币赚钱，利用展示广告等。

九城真的试图为希望接触中国观众的国际开发者提供一站式解决方案。由于中国大多数应用商店对游戏包装有不同的要求，九城帮助开发者本地化和包装他们的游戏，将它们发布到这些不同的渠道，使他们能够进入比其他方式更大的市场。

第九城市还为游戏开发者开发了一项免费使用的咨询服务，将为他们提供各种社交和互动集成，并帮助他们将中国的应用内微交易模式转化到他们的游戏中。沈说，中国的游戏玩家大多避免为游戏付费，这使得游戏开发商有必要通过游戏内广告解决方案、虚拟货币等来赚钱。他们的咨询服务将帮助开发人员更有效地开展这项工作。

然后，第九城市(和分销渠道，即应用商店)采用收入分成模式，从游戏收入中抽取大约50%的分成。不完全有利，但能接触到中国庞大的移动/社交受众，很可能值回票价。

由于九城与OpenFeint的合作，在iOS上使用Game Zone的开发者将能够访问该公司的游戏功能，如排行榜、成就、挑战、论坛和聊天，并将它们集成到他们的中文版本中。九城还允许整合中国的社交网站，如新浪微博(中国的Twitter)、腾讯和人人网。

目前有超过120家游戏开发商和发行商与第九城市合作，在中国发行了超过500款游戏。游戏区在iOS和Android上都可用，这两个系统都提供OpenFeint的功能。

考虑到该平台使世界各地的开发者能够穿过碎片化的中国移动游戏市场，进入全球第二大应用市场，并且为寻求获得中国市场免费打包和本地化服务的美国开发者提供一站式服务，这是一个非常好的解决方案。
