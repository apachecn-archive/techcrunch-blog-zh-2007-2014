# BBM 音乐获得百思买加拿大预发行喊出来

自去年八月宣布成立以来，BBM 音乐一直保持低调，但最近泄露的一张图片表明，它可能很快就会重见天日。 [Crackberry](https://web.archive.org/web/20230203155333/http://crackberry.com/bbm-music-already-being-advertised-best-buy) 得到了一份似乎是百思买加拿大版的通告，该通告宣传黑莓 Bold 9900 和 RIM 奇怪的社交音乐服务。

向所有的音乐分享爱好者道歉——看起来百思买的撰稿人并不知道任何新的信息。我们仍然需要具体的细节，如可用性和发布日期，但如果广告是任何迹象，它可能会比预期更早准备好黄金时间。

虽然 BBM 音乐可能会从炒作中受益，但考虑到它仍处于测试阶段，这对 RIM 和百思买来说似乎是一个奇怪的举动。它出现在广告中可能只是某种生产失误；也许其中一个文案需要填补空间，抓住他们能找到的第一个模糊有趣的黑莓花絮。

这当然是最简单的解释，但我不确定我是否买账。制作传单和购买指南需要相当长的时间，所以也有可能 BBM 音乐打算在一段时间前推出，广告的布局是为了突出它。

如果你是黑莓用户，并且所有这些关于 BBM 音乐的讨论已经让你恼火，不要难过:在[黑莓测试区](https://web.archive.org/web/20230203155333/https://www.blackberry.com/beta/defaults/login/?cont=portals&act=index&item=)，它已经为你准备好了。
