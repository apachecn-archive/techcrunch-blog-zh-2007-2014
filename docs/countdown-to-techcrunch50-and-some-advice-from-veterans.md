# TechCrunch50 倒计时，以及一些来自资深人士的建议

距离第三届年度(也是第三届售罄)TechCrunch50 发布会不到一天了。五十家创业公司正准备在数千人面前推出他们的新产品。他们紧张吗？没错。但这都是作为一名创业企业家进入奇妙的混乱世界的仪式的一部分。

在过去的几天里，有一堆给创业提供建议的好文章被贴了出来。我们在这里链接到他们，因为当 CNET 和其他人回顾 TechCrunch50 2009 和[把赢家](https://web.archive.org/web/20221209130544/http://news.cnet.com/8301-27076_3-10350281-248.html)和输家分类时，你会想在那个赢家名单上。所以，认真对待这些头发花白的老兵的建议。然后忽略其中的大部分，打造自己的道路，就像所有优秀的企业家一样:

如何经营一个贸易展览摊位的 22 个建议:Jason Calacanis 就如何经营你的 TechCrunch50(或任何会议)摊位给出了可靠、有用的建议。

[致所有 TechCrunch50 2009 初创公司的一封公开信:TC“Bump”，它真正的含义是什么&如何驾驭它](https://web.archive.org/web/20221209130544/http://www.watchingwebsites.com/archives/an-open-letter-to-all-techcrunch50-2009-startups-the-tc-bump-what-it-really-means-how-to-navigate-it) : Sean Power 和 Alistair Croll 谈论他们在 TechCrunch50 2008 上推出 [Akoha](https://web.archive.org/web/20221209130544/http://www.akoha.com/) 的经历，以及如何最大限度地利用你将获得的流量和关注。好建议:*“被人议论没什么。被人记住就是一切。”*

[不要喝你自己的 Kool-Aid(幸存 TC50)](https://web.archive.org/web/20221209130544/http://www.bothsidesofthetable.com/2009/09/13/dont-drink-your-own-kool-aid-surviving-tc50) :马克·苏斯特说不要太在意你会在会议期间受到的大肆宣传和关注。纪念它，但不要相信太多的宣传。用所有的注意力来建立一个可持续发展的企业。
