# 新的 Team Stream 应用程序 的看台报告在 iPad 上发布

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/20/bleacher-report-arrives-on-ipad-with-new-team-stream-app/?renderMode=ie11>

体育迷的热门网站，现在号称每月有 2200 万个独特的，刚刚为 iPad 发布了它的[团队流应用](https://web.archive.org/web/20230319083133/http://itunes.apple.com/us/app/team-stream-hd/id484725748?mt=8&ign-mpt=uo%3D6)。与 iPhone 和 Android 智能手机的 Team Stream 类似，新的 Team Stream HD 应用程序具有新闻和分析功能，并支持突发新闻的推送通知。

这是该公司首次推出针对平板电脑优化的体验，但不是唯一的:该公司还推出了针对 iPad 优化的 HTML5 浏览体验。

现在，当 iPad 用户访问 www.BleacherReport.com时，他们将被引导到一个平板电脑友好版本的网站，该网站具有增强的导航、全屏照片查看和故事之间的特殊过渡效果。该网站还将支持分享和评论其编辑作品。

与此同时，Team Stream HD 应用程序在主屏幕上提供了一个个性化的仪表盘，其中包含来自团队的头条新闻、头条新闻和推文以及您认为最喜欢的主题。在手机上，你可以选择你想关注的球队、运动员和体育记者，包括来自 NFL、大学橄榄球、MLB、NBA、NHL、大学篮球提示、足球、网球、高尔夫、MMA、拳击、WWE 和 NASCAR 的运动员。新的应用程序布局也很好，在故事和易于使用的 UI(用户界面)之间有一点额外的空间。

BleacherReport 刚刚在去年夏天又筹集了 2200 万美元，现在包括一个由 1000 多名专栏作家和 6000 名撰稿人组成的编辑团队。超过 150 万用户订阅其个性化电子邮件简讯。其内容现已整合到国家和地区出版商，包括《今日美国》、《洛杉矶时报》、《Philly.com》、《旧金山纪事报》、《休斯顿纪事报》和《西雅图邮报》。

要获得新的 BleacherReport iPad 应用程序，[点击这里进入 iTunes](https://web.archive.org/web/20230319083133/http://itunes.apple.com/us/app/team-stream-hd/id484725748?mt=8&ign-mpt=uo%3D6)。
