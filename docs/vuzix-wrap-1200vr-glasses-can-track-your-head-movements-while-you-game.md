# Vuzix Wrap 1200VR 眼镜可以在你玩游戏时跟踪你的头部运动

正在寻找一副漂亮的新眼镜？让你感觉像在 75 英寸显示器上观看 3D 内容的眼镜怎么样？这是你提出的一个非常具体的要求，但是 Vuzix 有你要的东西。该公司今天宣布了一款名为 Wrap 1200VR 的新型虚拟现实眼镜。

这些几乎与我们在 8 月份看到的的 [Wrap 1200 3D 眼镜相同，但具有针对游戏玩家的特殊头部跟踪功能。每只眼睛前面的 WVGA 显示器为 800×480，但可以处理 720p 的视频输入，并且可以为每只眼睛单独对焦。用户还可以定位显示器以考虑它们独特的瞳孔间距。](https://web.archive.org/web/20230204113935/https://techcrunch.com/2011/08/11/vuzix-wrap-1200-3d-glasses-hang-a-75-inch-screen-in-front-of-your-nose/)

至于头部追踪技术，它配备了一个指南针和九个传感器。这款新眼镜支持 100 多种 3D 视频和/或头部跟踪功能，包括*COD:Modern warfare*、*辐射 3* 和 *F.E.A.R* 等。

如果你戴隐形眼镜甚至眼镜，不用担心。Vizux Wrap 1200VR 可以“舒适地”安装在大多数处方眼镜上。就像你去看电影，戴上那副超酷的 RealD 眼镜一样，你的隐形眼镜不会受到影响。

这款眼镜可以连接几乎任何 Windows 笔记本电脑或台式机，支持 32 位和 64 位版本的 Windows 7、Vista 和 XP。可选接口还允许连接到 PlayStation 3 和 Xbox 360 等视频设备。

如果你手头有 600 美元，并且不怕看起来像个小玩意极客，那就试试现在在 Vuzix.com[出售的 Vuzix Wrap 1200 虚拟现实眼镜。](https://web.archive.org/web/20230204113935/http://www.vuzix.com/consumer/products_wrap_1200vr.html)
