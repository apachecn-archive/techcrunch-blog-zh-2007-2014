# 首款 Chrome OS 上网本将于今年秋季上市

[![](img/252380c353d9bbaeeceed03e4fd632dd.png "google-chrome-os-netbook-specifications")](https://web.archive.org/web/20221208072711/https://beta.techcrunch.com/wp-content/uploads/2010/06/google-chrome-os-netbook-specifications.jpg) 
谷歌计划今年秋天推出 Chrome OS。[我们昨天](https://web.archive.org/web/20221208072711/http://www.crunchgear.com/2010/06/02/google-chrome-os-set-to-launch-this-fall/)发现的。我们还推测，谷歌 Chrome OS 平板电脑或上网本很可能会有硬件合作伙伴。看起来很可能就是要发生的事情。

一位谷歌副总裁显然在 Computex 上泄露了秘密，尽管他没有提到谁将制造这款预计在今年秋天上市的上网本。有可能是宏碁，但我们现在真的不知道。越来越多。
