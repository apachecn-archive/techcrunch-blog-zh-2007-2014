# 沃尔玛收购面向初创公司 OneRiot TechCrunch 的移动和社交广告

沃尔玛[收购了](https://web.archive.org/web/20230216132249/http://blog.oneriot.com/post/10182214386/oneriot-acquired-by-walmart) [OneRiot](https://web.archive.org/web/20230216132249/http://www.oneriot.com/) ，这是一家以移动和社交广告为目标的初创公司。交易的财务条款没有披露，OneRiot 将加入沃尔玛实验室。

正如我们过去写的那样，OneRiot 最近改变了它的商业模式。关闭其实时搜索门户，专注于通过其实时广告网络赚钱。这家初创公司随后转向为应用内的移动广告提供社交定位服务。

从历史的角度来看，OneRiot first [在 2009 年凭借](https://web.archive.org/web/20230216132249/https://techcrunch.com/2009/09/30/oneriot-believes-it-has-a-way-to-monetize-the-realtime-web-with-riotwise/) [RiotWise、](https://web.archive.org/web/20230216132249/http://www.crunchbase.com/product/riotwise)一种将内容放在他们[实时反馈中重点位置的广告形式，冒险进入了](https://web.archive.org/web/20230216132249/https://techcrunch.com/2009/05/12/twitter-is-talking-real-time-link-search-but-oneriot-is-launching-it-today/)广告世界。 OneRiot 还[推出了](https://web.archive.org/web/20230216132249/https://techcrunch.com/2009/12/22/oneriot-monetizes-whats-hot-on-the-web-with-realtime-trending-ads/) [RiotWise 趋势广告、](https://web.archive.org/web/20230216132249/http://oneriotdevelopernetwork.com/riotwise-api/realtime-display-ad)一系列与社交网络上出现的趋势话题相对应的广告，并推出了[自刷新](https://web.archive.org/web/20230216132249/https://techcrunch.com/2010/03/19/oneriot-rolls-out-realtime-ad-unit-that-refreshes-to-match-trending-topics/)实时趋势广告和一款自助版本的 RiotWise。

现在，OneRiot [允许广告商](https://web.archive.org/web/20230216132249/https://techcrunch.com/2011/04/24/oneriots-social-interest-score-defines-mobile-audience-segments-for-advertisers/)在移动设备上接触目标受众，从忙碌的母亲到技术影响者到体育爱好者到时尚人士。细分和定位基于诸如观众兴趣概况、人口统计、社会影响和实时对话等因素。OneRiot 的受众资料是通过挖掘和分析来自服务(即 Twitter)的公共大数据社交流创建的。

正是这种数据挖掘吸引了沃尔玛对 OneRiot 及其技术的兴趣。从沃尔玛的声明 : *OneRiot 已经开发了一些非常漂亮的技术，可以分析来自 Twitter 和脸书等热门网络的社交媒体信号，以提供与消费者兴趣相关的广告。随着我们的团队就大数据、快速数据和机器学习技术的细节展开辩论，我们很清楚，我们找不到比 OneRiot 的同事更好的同事了。*

沃尔玛在社交、移动和目标市场领域做出了一些有趣的尝试。今年早些时候，该公司收购了社交媒体初创公司 Kosmix。Kosmix 加入了沃尔玛实验室，并一直在[致力于将社交技术](https://web.archive.org/web/20230216132249/https://techcrunch.com/2011/08/08/how-your-social-data-will-power-walmarts-e-commerce-revolution/)整合到沃尔玛的在线和移动产品中，包括其电子商务网站、脸书页面等。
