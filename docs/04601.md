# Soluto 找出是什么让你的电脑陷入困境(并告诉你如何修复它)

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/05/24/soluto-figures-out-whats-bogging-down-your-pc-and-tells-you-how-to-fix-it/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+Techcrunch+%28TechCrunch%29>

人们说，一辆车从停车场开出来的那一刻起，就失去了一半的价值。对于一台全新的电脑来说，情况似乎也是如此——在闪电般的首次启动一两次后，电脑就会逐渐陷入停滞，直到新的四核处理器看起来并不比上一台快多少。进入 TC Disrupt 决赛的 [Soluto](https://web.archive.org/web/20221207064702/http://www.soluto.com/) ，这是一家希望帮助你的电脑恢复往日辉煌的初创公司。更好的是，它将让那些臃肿、占用 CPU 的应用程序负起责任。

Soluto 开发了一种软件，可以监控你的电脑上可能会惹恼用户的事情——打印问题、不稳定的应用程序、资源占用，以及那些随机导致你的鼠标每次几秒钟无用的令人沮丧的应用程序。它会记录中断时哪些应用程序正在运行，并分析低级事件来跟踪用户甚至没有意识到的事情，例如哪些应用程序正在争用内存。

更好的是:如果世界上某个地方的某个用户在某个应用程序上一直有问题，然后进行了修改并修复了它，Soluto 就可以告诉其他有同样问题的用户关于修复的信息。

该产品看起来非常引人注目，具有令人印象深刻的用户界面。在 Soluto 分析了你的机器后，它可以提出许多修复建议，包括“不用动脑筋”的，比如从你的启动中省略某些应用程序。它可以显示您的计算机随时间的速度，绘制出您何时安装了某个应用程序，这样您就可以看到是什么导致了问题。

该公司的 PCGenome 项目实际上可以帮助 T5 解决这些问题。Soluto 从其应用程序中获取用户提交的数据，并使用它来计算出哪种软件在每台计算机上运行得最好。它可以识别哪个硬件的哪个软件出现了问题。它还可以根据其他用户的情况，帮助你挑选出在你使用的特定型号电脑上运行良好的软件。

为了赚钱，Soluto 有几个选择。它将提供一种免费增值模式，有两种计划:手动，适合已经精通计算机的人。这个选项告诉你你的电脑出了什么问题以及如何修复，但这取决于你自己。第二个选项是自动的:设置它，Soluto 将在后台工作以修复任务，无需您的干预。在这一点上，这种自动设置会做一些事情，如修复驱动程序设置，将来它可能会下载比您当前使用的应用程序在您的机器上运行得更好的应用程序。

**第二轮业务演示**
Soluto 是被选中参加第二轮 TechCrunch Disrupt 创业战场的公司之一，在那里他们谈论了他们的商业模式。以下是我对那次演讲的笔记:

我们最初会和极客们一起营销。有几亿人在痛苦中。消费者在安全软件上的支出为 36 亿美元。蛇油解决方案公司赚了 3 亿美元。

我们还在寻求中小企业市场。价值很容易量化——我们可以有效地观察劳动力的增加。我们还延长了电脑的有效寿命，因为人们不会厌倦电脑而决定需要一台新的。我们还削减了维护/支持费用。
