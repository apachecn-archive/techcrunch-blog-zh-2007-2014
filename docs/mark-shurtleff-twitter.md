# 犹他州司法部长马克·舒尔特勒夫使用推特宣布处决

![](img/ab4407b1a509d26f9a13091accf27828.png)

这个时代的一个标志，尽管许多人可能会觉得它令人反感，或者更糟:犹他州司法部长马克·舒尔特勒夫使用移动推特客户端发出了一条推文宣布即将由行刑队处决被定罪的杀人犯罗尼·李·加德纳。

正如英国广播公司注意到的那样，这是一种非常现代的方式来宣布一个非常传统的死亡。

总的来说，就在几个小时前，AG 从他的 iPhone 上发出了 3 条关于该事件的推文，最近的一条是大家都太熟悉的(在 Twitter 上)自我宣传的一条。

> 1)庄严的一天。除非得到最高法院的批准，否则在我的最终同意下，犹他将使用最极端的权力&处决一名杀手。悼念他的受害者。公正
> 
> 2)我刚刚批准惩教主任对加德纳执行死刑。愿上帝赐予他宽恕，他拒绝了他的受害者。
> 
> 3)当我得知加德纳的死讯时，我们将会直播我的新闻发布会。在 www.attorneygeneral.Utah.gov/live.html 看
