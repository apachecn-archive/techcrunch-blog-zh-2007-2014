# 跟随网飞的脚步，百视达创建了一个独立的爆米花公司

在网飞宣布将把 DVD 和网络流媒体业务分开之后，旧的 DVD 业务被重新命名为 Qwikster，顾客和 T2 投资者的强烈反对开始了。伴随这种反弹而来的是恶搞视频。上面那张由[有线电视](https://web.archive.org/web/20230205033223/http://www.youtube.com/user/TheLandline)的人拍摄的照片显示了百视达的回应。它也将其业务拆分为电影租赁和爆米花业务。它在收银台出售的那些爆米花现在将成为一项名为 Plopydop 的新业务。

Plopydop 将开始试验各种新的爆米花，包括“咖啡口味的，能读出别人思想的爆米花”。。。免费看牙医附赠的爆米花。。。还有一桶爆米花，外加免费电影租赁，现在每个人都讨厌网飞了。”

如果模拟视频看起来很熟悉，那是因为它是基于网飞首席执行官雷德·哈斯汀斯和新 Qwikster 首席执行官安迪·伦迪奇录制的道歉视频，以解释网飞的变化。我在下面转贴了原文。
