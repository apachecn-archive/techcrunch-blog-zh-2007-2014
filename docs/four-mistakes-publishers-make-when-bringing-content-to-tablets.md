# 出版商在将内容引入平板电脑时犯的四个错误 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/11/four-mistakes-publishers-make-when-bringing-content-to-tablets/>

**编者按:** *[米奇·耶戈](https://web.archive.org/web/20221007171500/http://www.twitter.com/mitchlazar/)是新闻阅读器初创公司 [Taptu](https://web.archive.org/web/20221007171500/http://www.taptu.com/) 的 CEO。他是 CNN.com 的联合创始人，也是 CNN 移动和卡通网络移动的创始人。*

许多革命都被电视转播了，但是出版革命已经数字化了，而且现在被移动化了。

毫无疑问，2011 年下半年对报纸和杂志出版商来说是一段艰难的时期。发行审计局的报告显示[消费者杂志的单本销量在一年内下降了近 10%](https://web.archive.org/web/20221007171500/http://mediadecoder.blogs.nytimes.com/2012/02/07/magazine-newsstand-sales-suffered-sharp-falloff-in-second-half-of-2011/),而报摊销量最高的五种杂志也都报告了大幅下降。最重要的是，销售额的下降影响了收入，这使得出版业快速实现贸易现代化变得比以往任何时候都更加重要。

随着读者转向平板电脑和手机，毫无疑问，这些新的阅读设备将决定媒体行业的成败。成功的出版商将能够在这些设备上再现他们的数字内容。那么，为什么如此多的出版商在他们的移动战略中磕磕绊绊？从对众多移动平台的过度承诺，到平淡无奇的应用体验，我们看到了许多不应该重复的错误:

**1。试图重新发明轮子，但失败了。**

许多大大小小的出版商拥有一流的技术团队和重要的资源，但往往陷入这样一个陷阱，即认为只有他们自己的团队才能创造出最好的平台和体验。不是真的。

合作伙伴关系是大型和小型媒体公司在新媒体世界中成功建立受众的主要方式。小型创业公司正在创造惊人的技术，可以帮助出版商发展他们的发行计划。通过利用这些才华横溢、专注的团队，出版界可以利用明天的趋势，而不是昨天的机会，以令人信服和引人入胜的方式快速分发内容。

不要重新发明轮子，因为当你这样做的时候，一个新的轮子已经在运转了。

**2。被排除在外。**

如果你仔细想想，听收音机里的音乐或者去夜总会会让你接触到你可能还没有发现的新音乐。多亏了 DJ 和探索服务，我们都能找到我们喜欢并想要分享的新音乐。这种管理和分享经验现在已经进入了数字出版的世界。现代社交新闻聚合器本质上是内容 DJ，通过有趣和简单的体验向消费者提供精彩的内容，无论是通过轻击、点击还是翻转设备屏幕。做对了这一点的出版商正在经历数字读者的激增，这完全是因为新的发现工具和网络，如脸书和 Twitter，将新读者转向了伟大的推荐内容。

新闻需要传播。过去，出版商把报纸放在每个旅馆房间的门下，许多家庭的前门或街角。今天，成功取决于出版商加入和参与社交媒体和新闻革命的程度。像新闻阅读器这样的发现服务可以提供帮助。

可悲的是，一些出版商回避这些发现工具。他们希望他们的内容只存在于他们控制的空间中，或者拥有只包含他们原创内容的频道。但是消费者要求更多。通过新闻阅读器，他们可以利用搜索技术来浏览和发现他们从来不知道存在的新内容和来源，这些搜索技术为发现、分享和推荐创造了意外之喜。

新闻阅读器用户正在建立跨类型的精选主题流，并从各种出版物的编辑那里接收大量内容。以超级碗为例。在过去的日子里，你不得不在 ESPN、CNNSI 或雅虎体育的每一个社论版本中寻找答案。现在，你可以播放自己的新闻组合，看看体育编辑和社会大众对超级碗各方面的看法，使用户体验引人入胜，节省时间，远远超越传统的新闻搜索。当用户喜欢他们看到的东西时，他们会与朋友、家人和粉丝分享故事——证明自己是成功分发的关键因素。最终，新闻阅读器和其他发现服务驱使更多的人回到收银机响起的媒体目的地。

**3。忽视品牌潜力。**

大品牌出版商手头有来自许多不同品牌或标签的惊人的内容宝库。他们每天都在创造大量的内容。事实上，一些最大的媒体公司有几个惊人的报纸或杂志，但许多人没有冒险将他们各种出版物的内容混合和融合成一种新的令人兴奋的品牌体验。

在这个快速变化的数字环境中，试水推出新的聚合服务的时机已经成熟。成本不是很大，但收益会很大。它改变了传统的分销方式，专注于与现有品牌的深度垂直，让出版商可以尝试新的分销策略，而不会侵蚀他们现有的受众和收入。

以 MSN 的 [Glo 为例。通过与 Hachette Filipacci Media 和 BermanBraun 合作，他们为女性打造了一个顶级的生活方式目的地，其中包含了来自各个领域的聚合媒体的精彩组合。他们利用印刷世界中的现有内容，为数字观众创造了一种新的途径来消费他们的精彩内容，同时利用机会以相对较低的成本建立新的业务。](https://web.archive.org/web/20221007171500/http://msnblog.msn.com/blogpost.aspx?post=1732776)

**4。在错误的地方寻找。**

出版商内容的分发和发现过去主要发生在传统的搜索引擎中，如 Google、Yahoo！和 Bing 在搜索引擎优化(SEO)技术方面的传统投资，引导用户寻找一个特定的查询，以发现来自另一个相关渠道的内容。以某种方式标记的内容在算法搜索结果中显示得更高，促使用户点击它，当 SEO 做得正确时，出版商将获得更多受众的好处。这是一种免费广告出版商和媒体所有者在他们的发行计划中使用的方式。然而，像 Taptu，Flipboard，Pulse 和 Zite 这样的新闻阅读器正在展示 SEO 的现代形式，用户可以发现和分享与每个用户都有完美背景和相关性的故事。

在阅读内容流时，人们会接触到相关的故事，或者从各种出版商那里获得其他类似的故事，从而使用户能够分享、发推特或通过链接回到大型媒体和出版商。因此，举例来说，如果用户搜索“2012 年 NFL 模拟选秀”，他们会立即找到各种新的来源，这些来源已经成为该主题的专家，如[沃尔特足球](https://web.archive.org/web/20221007171500/http://walterfootball.com/draft2012.php)。哪个沃尔特？是的，沃尔特足球队。欢迎来到移动搜索的新世界。

在与世界各地 100 多家数字出版商的交谈中，我们听到的一致意见是，“我们知道移动是至关重要的，但走向移动说起来容易做起来难。”希望出版业可以从我每天看到的事情中吸取教训，采取简单、经济的措施在移动领域取得胜利，而不会让历史重演。
