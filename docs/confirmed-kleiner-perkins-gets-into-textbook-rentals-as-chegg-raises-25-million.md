# 确认:凯鹏华盈进入教科书租赁市场，Chegg 筹集了 2500 万美元

[![](img/d7efabc13b852064f284d477b1a3b7e9.png)](https://web.archive.org/web/20221006214131/http://www.chegg.com/)

[Chegg](https://web.archive.org/web/20221006214131/http://www.chegg.com/) 部分确认了我们上个月[的初始报告](https://web.archive.org/web/20221006214131/http://www.beta.techcrunch.com/2008/11/09/textbook-rentals-big-business-kleiner-perkins-goes-after-chegg/)，宣布已经结束了由著名风险投资公司凯鹏华盈(Kleiner Perkins cau field)&Byers 牵头的价值 2500 万美元的 C 系列融资。基金会资本以及现有投资者加布里埃尔风险合作伙伴和 Primera Capital 也参与了这一轮融资，他们很可能帮助这一轮融资超过了我们在 11 月听到的 1500 万美元。

Chegg 是一家教科书租赁初创公司，允许大学生以他们在校园书店通常支付的价格的一小部分租赁书籍。在网站上找到他们需要的书后，Chegg 会在八个工作日内将课本寄给学生。一旦他们在学期末完成了这些书，学生们只需把它们扔进一个预先付费的盒子里，然后把它们送回 Chegg。这项服务在全国 4000 所大学提供，据报道，每季度为一些学生节省了高达 650 美元的教科书费用。

这是一项伟大的服务，一旦更多的学生意识到他们能省多少钱，这项服务很可能会像野火一样蔓延开来。多年来，教科书在大学校园里的价格一直是出了名的高，甚至在大多数学生会发现的教科书回购计划也是明目张胆的敲诈，通常只需支付一本书原价的 15%或 20%。由于利率让这些项目相形见绌，Chegg 似乎是一个确定的赢家。也就是说，直到教科书最终数字化，学生们可以把它们扔在他们的 Kindles 上。
