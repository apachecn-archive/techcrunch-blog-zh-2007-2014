# 谷歌即将因更快的搜索索引而受到影响

![](img/a6566fd60ade9be00aeb05d6ccfafd0b.png)

谷歌对实时搜索越来越认真，在搜索结果中加入了 Twitter、脸书和 MySpace 流，并加入了令人上瘾的最近“更新”选项。但在幕后，谷歌也准备推出一种全新的网络索引方式。[代号为“咖啡因，](https://web.archive.org/web/20230315052422/https://techcrunch.com/2009/08/10/caffeine-its-google-on-red-bull-or-something/)”谷歌从去年夏天开始在一个数据中心进行测试，现在准备在整个谷歌推广。一些有搜索引擎优化思想的情报人员说，他们开始注意到更快的搜索响应时间，但谷歌坚持认为咖啡因要到新年后才会推出(部分原因是为了让每个人都能享受假期)。

当咖啡因真的产生时，你能期待什么？大多数人甚至不会注意到。谷歌搜索结果的外观和感觉不会改变。咖啡因是对谷歌搜索索引算法的升级。但这将大大加快谷歌展示结果的速度，尤其是在不同媒体类型之间，如照片和视频。总的来说，搜索将更加实时，因为谷歌认为它的索引速度是一个竞争优势。

SEO 公司 [Summit Media](https://web.archive.org/web/20230315052422/http://www.summit.co.uk/) 在 9 月份进行的一项独立研究表明，对于通用搜索词，咖啡因对新闻和社交媒体结果的权重更大，而更具体的关键词更有可能出现关于该主题的网站。Summit Media 研究将咖啡因与谷歌目前的约 10，000 个关键词的索引进行了比较。峰会的调查结果包括:

**   排名不太可能波动。从一个客户的所有关键词来看，排名应该会有很小的百分比变化。*   排名下降的网站很可能依赖于多年没有更新的旧的存档内容。保持你的网站新鲜，让咖啡因来抓取是很重要的。*   没有任何变化使一个部门比另一个部门有任何新的优势。*   与咖啡因中的长尾词相比，通用词显示出新闻、信息和社交媒体更优先——这符合通用词在搜索者意图方面不太明确的情况。*   只有 5%的 URL 包含搜索关键字的精确匹配，总分数的 6%由精确匹配的 URL 构成。*

当咖啡因真的上线时，不仅谷歌会感觉更快，而且许多结果会感觉更有新闻性和实时性，特别是通用搜索。我想知道在长尾搜索中出现更多的网站是否会试图通过模仿新闻网站或在首页粘贴 Twitter 和脸书的更新来抓住咖啡因的热潮。我只希望咖啡因能分辨出真消息和假消息。
