# Linux Mobile 走向标准化

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/27/linux-mobile-moves-towards-standardization/>

免费的操作系统 Linux 在手机行业获得了越来越多的支持。像 LiMo 基金会和 Linux 电话标准(LiPS)论坛这样的实体最近在 Linux 世界大会上会面，支持 Linux 作为移动电话操作系统。来自世界各地的支持者使 Linux 成为移动电话行业的主要参与者。

虽然像 LiMo 基金会和 LiPS 论坛这样的支持者有时是由相互竞争的公司组成的，但这些团体的目标并不一定相互冲突。但是时间限制可能会造成一些困难。正如 IMS 研究主管 John Devlin 指出的那样，“LiMo 基金会打算开发一个真正的移动 Linux 平台，该平台可能符合 LiPS 论坛建立的标准。然而，由于 LiMo 基金会表示希望产品最早在 2008 年第二季度(2008 年第二季度)上市，他们正在制定 LiPS 论坛将发布的最终标准。这为 LiMo 基金会平台成为事实上的竞争标准创造了潜力。”

移动 Linux 的未来可能会看到由整合和公司间协议带来的行业标准的创建。在最近发表的一份报告“蜂窝 Linux 的影响”中，IMS Research 分析师 Alison Bogle 准确地预测了我们所看到的大部分情况。Bogle 说，“在未来几年里，将会有一段时间的收购和整合，移动 Linux 社区最终将在一个或两个领先的基于统一 Linux 的平台上实现标准化，这将成为事实上的标准，Linux 将在手机领域得到广泛采用，这取决于支持它的公司的实力。”

如果移动 Linux 能够继续它的标准化之旅，它将成为手机操作系统领域的主要玩家。我希望看到一个主要的移动操作系统成为世界标准。这将使来自不同服务提供商的各种服务的连接更容易在世界各地的所有移动设备上翻译。但这个愿望来自于一个想在 Xbox 360 上玩他的旧 Commodore 64 游戏的人。
