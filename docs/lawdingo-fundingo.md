# Lawdingo 又筹集了 69 万美元，帮助您立即与律师交谈 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/30/lawdingo-fundingo/>

Y Combinator 孵化的法律初创公司 [Lawdingo](https://web.archive.org/web/20221006090702/http://www.lawdingo.com/) 宣布，它又筹集了 69 万美元的资金。

该公司的目标是让用户更方便、更实惠地联系律师(相比之下，LegalZoom 等服务提供的是法律表格，而不是与律师的实际咨询)。用户可以搜索网站，浏览基于律师位置和专业知识的个人资料，然后安排预约或立即与他们交谈。

创始人兼首席执行官 Nikhil Nirmel 表示，自从我上次写了关于公司的[以来，更受欢迎的功能之一是“现在就能接到电话”的能力——law dingo 可以通过电话立即将用户与相关律师联系起来。律师向 Lawdingo 提供他们的可用性，因此当请求进来时，系统会联系那些有空的并且具有相关专业知识的人，直到找到一个可以自由交谈的人。](https://web.archive.org/web/20221006090702/https://beta.techcrunch.com/2013/03/15/lawdingo-y-combinator/)

Nirmel 补充说，这是律师寻找新客户的有效途径。他们可以参加这些谈话，因为律师“不会像在严格的预约时间表上那样运作……除非他们在法庭上，否则他们仍然可以接客户电话。”

商业模式也有所调整，Lawdingo 放弃了一项由律师竞价推广其上市的计划。此后，该公司又恢复了最初的固定订阅费。

Lawdingo 现在在每个州都有律师，最集中的是加利福尼亚、纽约和马萨诸塞州(Nirmel 说在这些州都有 200 多名律师)。为了解决管理不同类别的供应和需求的挑战，他说他的团队建立了一个功能，当网站需要在给定区域吸引更多客户时，它会自动在有针对性的广告上花费更多资金。

说到广告，Lawdingo 还制作了一个半开玩笑的营销视频，我把它放在了下面。尼尔梅尔说，他的目标是“点亮法律行业”，他说，法律行业“把自己看得太重了——他们和其他人一样都是服务提供商。”

nimmel[在今年早些时候参加了硅谷的 Y Combinator 项目](https://web.archive.org/web/20221006090702/https://beta.techcrunch.com/2013/03/15/lawdingo-y-combinator/)，但后来搬到了纽约。Lawdingo 的 15 人团队仍然分布在多个地点，Nirmal 使用 GitHub 和 Skype 等工具管理每个人。当被问及这是否真的是经营公司的有效方式时，他回答说，“老实说，我不知道它能扩大到多大规模，但就目前而言，我认为它运作得很好。”

新的资金来自天使投资者(以及天使投资者运营的基金)，包括 Nathaniel Stevens、Kartik Hosanagar、Gene Alston、Altair Capital、Atsany Captial 和 Andrew Moroz。这使得 Lawdingo 的资金总额达到 85 万美元。
