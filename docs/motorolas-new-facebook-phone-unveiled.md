# 摩托罗拉的新脸书手机透露

摩托罗拉对把社交媒体塞进他们的设备并不陌生(我正看着你， [MotoBlur](https://web.archive.org/web/20230205023329/https://techcrunch.com/tag/motoblur/) )，但一款新设备通过[蓝牙 SIG 网站](https://web.archive.org/web/20230205023329/https://www.bluetooth.org/tpg/EPL_Detail.cfm?ProductID=18358)展示了摩托罗拉对脸书一体化有多认真。

在我们进一步讨论之前，我不得不要求你降低对摩托罗拉·脸书手机的期望。再多一点。然后…我们开始吧。

EX225 是摩托罗拉第一款带有脸书按钮的手机，加入了 HTC ChaCha/Status、HTC Salsa 和[沃达丰 Blue 555](https://web.archive.org/web/20230205023329/https://techcrunch.com/2011/07/27/vodafone-blue-555-the-facebook-phone-that-came-too-late/) 的杰出行列。不幸的是，EX225 在执行方面似乎倾向于沃达丰，因为它似乎缺乏任何智能手机的 DNA，只有 2.4 英寸的显示屏。很难说摩托罗拉的图像有多大，但 UnwiredView 认为它运行的是基于 BREW 的操作系统，考虑到摩托罗拉最近推出的功能手机，这是有道理的。

这是一款 GSM 设备，事实上，双 SIM 卡变体(当然是 EX226)的存在可能意味着这款设备将在全球范围内发挥作用。目前还没有关于国内可用性的消息，但我有预感 EX225 最终将被定位为一款快速消息设备，在 AT&T 上有一个炫耀的名字。

蓝色巨人仍然希望能够为不一定需要智能手机数据速率的客户提供令牌设备，EX225 作为过渡手机的吸引力刚刚好。用户将能够尝到数据的滋味，一旦他们迷上了，一部成熟的智能手机很可能就在他们的未来。
