# AppLoop 已经衰落了吗？

去年 7 月，当 [AppLoop](https://web.archive.org/web/20221002234116/http://www.apploop.com/) [在](https://web.archive.org/web/20221002234116/http://www.beta.techcrunch.com/2008/07/01/location-based-mobile-ad-network-apploop-launches/)推出其在移动应用上追踪和广告的自助服务平台时，我们印象深刻。利用 iPhone 3G 的原生 GPS 功能，AppLoop 的地理感知移动广告网络能够判断消费者何时靠近特定的商业地址，并相应地为该商业提供广告。

去年 10 月，这家初创公司推出了[应用生成器](https://web.archive.org/web/20221002234116/http://www.beta.techcrunch.com/2008/10/20/apploop-transforms-blogs-into-native-iphone-applications/)，这是一个漂亮的工具，可以将任何带有 RSS 订阅源的在线出版物转换成一个单独的 iPhone 应用程序。

现在，我们听到传言说这家初创公司陷入了困境，这些传言似乎有些道理。首先，该公司的网站在过去两天已经关闭。更糟糕的是，在 Twitter 上快速浏览一下就会发现，该服务的宕机也导致使用 AppLoop 库的 iPhone 应用程序崩溃。

我们不希望看到这家年轻的公司逐渐消失，所以我们希望这只是一个技术故障，需要很长时间才能修复。但这两家初创公司的创始人在他们的博客和 Twitter 账户上对停机时间出奇地沉默，所以这可能是徒劳的希望。

我们已经联系了 AppLoop，一旦我们对正在发生的事情有了一些了解，我们将相应地更新这篇文章。

**更新:**在联合创始人埃里克·克尔(Eric Kerr)发出一封电子邮件后，我们将该公司置于[死亡池](https://web.archive.org/web/20221002234116/http://www.beta.techcrunch.com/tag/deadpool/)中，因为他们不得不“出于各种财务和法律原因”关闭这项服务。克尔声称用户在几周前就收到了一封电子邮件，有些人可能没有收到。我不太确定，但我绝对认为不在你的网站或博客上发布公司倒闭的消息是不好的。
