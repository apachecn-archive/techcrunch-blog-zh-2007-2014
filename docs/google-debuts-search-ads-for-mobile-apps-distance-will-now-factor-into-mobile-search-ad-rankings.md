# 谷歌推出移动应用搜索广告；距离现在将成为移动搜索广告排名的因素

随着假日购物季的临近，越来越多的消费者将使用智能手机来搜索产品、商店等。谷歌今天首次推出了一系列新的移动广告形式，包括一种专门为搜索体验设计的形式。许多应用程序在应用程序中具有搜索功能(即 Yelp)，因此通过这些新格式，应用程序开发人员可以在应用程序的搜索中放置上下文广告。

另一个广告单元允许企业使用移动搜索广告，在已经安装在手机上的移动应用程序内，将某人直接引导到目的地。例如，如果有人在移动设备上搜索 iPad，他们可能会看到一个广告，直接将他们带到他们安装在手机上的购物应用程序。

谷歌上 40%的移动搜索与位置相关，本地移动广告是一个巨大的市场。事实上，点击呼叫现在每周带来数百万次呼叫。除了应用内搜索广告，谷歌还发布了许多与本地移动搜索广告相关的公告。这家搜索巨头宣布，接近度现在将成为移动搜索广告排名的一个因素。

因此，用户和广告商的商业地点之间的距离将影响广告在 AdWords 中的表现。当然，只有当消费者选择分享他们的设备位置用于移动搜索时，该功能才会有效。随着假日购物季的到来，这可能是一件大事，会有更多的本地产品和商店搜索。

谷歌还推出了一种面向零售商的新广告形式——通知。当有人点击搜索或显示广告(在桌面、移动或平板设备上)时，他们可能会看到通知，其中包含相关产品和特价商品的照片。用户可以通过电子邮件将通知发送到他们的手机上，然后走进当地的商店，亮出手机，兑换优惠。
