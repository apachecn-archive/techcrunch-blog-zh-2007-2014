# 新闻阅读器 Pulse 首次推出跨平台同步

[Pulse，](https://web.archive.org/web/20230203093216/http://www.pulse.me/)一款面向移动设备的创新新闻阅读应用，首次推出了一种新的跨平台同步功能，允许用户跨设备访问他们的帐户和保存的来源。

Pulse 是由 [Alphonso 实验室开发的，](https://web.archive.org/web/20230203093216/http://www.crunchbase.com/company/alphonso-labs)去年在[推出，通过一个 iPad 应用程序](https://web.archive.org/web/20230203093216/https://techcrunch.com/2010/05/31/pulse-ipad/)作为一种更无缝(视觉上更吸引人)的方式来阅读你的 RSS 订阅。但是 Pulse [抛弃了 RSS](https://web.archive.org/web/20230203093216/https://techcrunch.com/2011/02/16/pulse-rss-apis/) ,转而使用 API 来访问内容。Pulse 的主屏幕在一个动态马赛克界面上呈现来自你的订阅源的故事，并通过一个触摸界面，允许你上下滑动以查看各种来源的标题，左右滑动以浏览特定来源的故事。Pulse 用户还可以从 60 个来源获取新闻。

最近，Pulse 首次推出了创建个人账户的功能，你可以在那里保存故事供以后阅读。注册后，你可以使用[书签功能](https://web.archive.org/web/20230203093216/https://techcrunch.com/2011/08/29/with-save-to-pulse-bookmarklet-and-chrome-extension-pulse-enters-instapapers-turf/)在你的 Android 手机或网络上为一个故事添加书签，然后在你的另一个设备上阅读。在这个版本中，Pulse 还允许您跨设备同步您的源。

基本上，这一更新使得不同设备之间的阅读体验更加统一。添加的每个源都将同步到你有 Pulse 的所有设备，包括该服务的 iPhone，iPad，Android 和 Android 平板电脑应用程序。
