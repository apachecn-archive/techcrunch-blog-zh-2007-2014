# 谷歌的玛丽莎·梅耶尔将离开搜索团队，接管位置和本地

如果你必须列出谷歌最受公众关注的人物，玛丽莎·梅耶尔很有可能名列榜首。梅耶尔于 1999 年作为一名早期员工加入谷歌，多年来一直担任搜索产品和用户体验副总裁，领导着谷歌至关重要的搜索团队，她在其中发挥了重要作用(她还经常领导谷歌的新闻活动)。现在，梅耶尔即将承担新的角色:彭博[向](https://web.archive.org/web/20221218150710/http://www.bloomberg.com/news/2010-10-12/google-s-marissa-mayer-takes-new-role-overseeing-location-local-services.html)报道，梅耶尔将辞去她在搜索团队的职位，领导谷歌的位置和本地业务。

这是一件大事，它有希望意味着这些谷歌产品将会有很大的变化。谷歌的位置服务 Latitude 甚至很少在与 Foursquare 的相同对话中被提及，尽管谷歌正在像热卖蛋糕一样销售 Android 手机。同样，谷歌的本地搜索结果似乎过于接受该公司的简朴美学。

根据彭博的报道，梅耶尔在搜索团队中的位置将被网络搜索工程副总裁 Udi Manber 取代。报道还称，梅耶尔正在加入谷歌精英运营委员会的行列。

梅耶尔没有浪费任何时间为她的新角色做准备:正如你所见，她最近在 Foursquare 上非常活跃。
