索尼NEX-7将在今年晚些时候带来1080p的录制

你可能想暂缓购买索尼 NEX-3 或 NEX-5 相机。他们可能是现在的热门，但似乎他们的大哥就在拐角处。NEX 7听起来非常好。

它不会采用使3和5型号如此吸引人的超薄外壳，但它也不应该是一个庞然大物，因为它也将采用无镜设计，带有APS-C传感器。然而，这一个将能够1080p@60、1080p@24和720p@120/60录制。没错，一个慢动作模式。

NEX-7还将配备全手动控制和体内图像稳定功能，这在一定程度上解释了更大的车身。总之，今年晚些时候，NEX-7将会是一个合适的微型4/3rds的竞争者。我们等不及了。[ EOSHD 通过 SlashGear
