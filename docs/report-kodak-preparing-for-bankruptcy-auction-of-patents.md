# 据报道，柯达准备破产拍卖专利

差[柯达](https://web.archive.org/web/20230209125256/https://techcrunch.com/tag/kodak/)。在这一点上，他们只是逢场作戏。过去的几年对他们来说很艰难，他们做了一些没有成功的重大决定。我必须承认，虽然[我主动给他们的建议](https://web.archive.org/web/20230209125256/https://techcrunch.com/2011/10/17/kodak-time-to-go-invisible/)是合理的，但它可能必须在十年前实施，以避免目前的情况。事实上，[《华尔街日报》获悉，他们正计划申请破产保护](online.wsj.com/article/SB10001424052970203471004577140841495542810.html?mod=googlenews_wsj),在法院监督下拍卖他们的许多数码成像专利。

这很可悲，但事实是，尽管柯达仍然是一家很有价值的公司，但它已经不再是一家可行的企业了。他们改变业务的努力来得太晚了——现在他们正在走向破产。

在某些方面，这是一个很好的决定(并不是说我是一个大的破产专家)，但它也有风险。柯达一直试图出售 1000 多项专利，以获得维持运营所需的现金。目前还不清楚他们在卖掉公司最有价值的部分后会做什么，但他们可能已经注意到“好死不如赖活着”，并选择了偿付能力和生路。

不管他们的意图是什么，专利还没有卖出去。[他们卖掉了自己的传感器业务相关资产](https://web.archive.org/web/20230209125256/https://techcrunch.com/2011/11/07/short-on-cash-kodak-sells-its-sensor-business/)，但是专利还在架子上。为什么？柯达实际上发明了数码相机，他们的一些知识产权肯定对索尼或三星等公司有用。

我猜，问题在于:既然知道商店要倒闭了，为什么还要付全价？如果索尼和三星都想要一套专利，柯达会在对方出价时通知他们，这样他们的目标就不会从眼皮底下溜走。与此同时，柯达无休止地走向破产，在这一点上，整个专利组合将被公开招标和无与伦比的价格。

不幸的是，柯达不太可能从此反弹。他们唯一能活下去的方式就是卖掉自己的核心资产，借一大笔贷款，之后除了卖掉品牌，找一份给别人产品贴牌的工作之外，别无选择。哦，最可耻的命运！但这是他们自己做的。
