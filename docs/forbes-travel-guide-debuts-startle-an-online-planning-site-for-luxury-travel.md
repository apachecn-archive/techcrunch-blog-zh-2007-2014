# 福布斯旅游指南首次亮相，一个豪华旅游在线规划网站

福布斯旅游指南首次推出 Startle.com，这是一个新的豪华旅游在线旅游规划网站。在 72 个美国市场和 5 个国际市场(香港、澳门、北京、新加坡和多米尼加共和国)推出，skeager 独特地结合了来自福布斯旅游指南检查员、时尚人士以及酒店、餐厅和水疗中心专家的内容。

这些网站包括《福布斯旅游指南》对酒店、餐馆和水疗中心的评论，这些评论是基于由匿名访问每个地方的专业检查员进行的 500 项标准服务评估。用户可以访问酒店、活动甚至交通方面的内幕交易。

skeager 提供了一个问答平台，消费者可以跟踪问题，并获得专家和知名
品酒师、侍酒师、门房和豪华酒店总经理对关键旅行规划问题的精选答案。

skeaker 面临着包括 Oyster 和 Jetsetter 在内的众多面向消费者的旅游规划网站的竞争。
