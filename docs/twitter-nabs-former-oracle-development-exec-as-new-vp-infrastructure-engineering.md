# Twitter 聘请前甲骨文开发主管担任基础设施工程副总裁

Twitter 今天刚刚宣布了一项关键的工程招聘。[亚当·梅辛格](https://web.archive.org/web/20230203090210/http://www.linkedin.com/in/adammessinger)将加入公司，担任基础设施工程副总裁。

Messinger 曾任 Oracle 开发副总裁，负责管理 Oracle Coherence、Oracle JRockit、Oracle WebLogic Operations Control 和其他 web 层产品。在加入甲骨文之前，他曾在 Smartforest Ventures 和 O'Reilly AlphaTech Ventures 担任风险投资家。

在 Oracle，他负责开发中间件云服务器、Java SE、FX、ME 和 Card、Coherence、虚拟装配构建器、中间件生命周期和应用生命周期。在他担任风险投资之前，他创建了 Gauntlet Systems Corp .，该公司被 Borland 收购。

这对 Twitter 来说是一个大的发展雇佣，因为该公司说 Messinger“在许多人使用网络之前就在做网络工程。”

10 月，Twitter 的工程副总裁[麦克·阿伯特离开公司](https://web.archive.org/web/20230203090210/http://allthingsd.com/20111013/exclusive-vp-engineering-mike-abbott-departs/)，以 EIR 的身份加入 Benchmark。
