# Atlassian 为 Git 和 Mercurial SourceTree 购买 Mac 客户端

去年，产品开发软件公司 [Atlassian](https://web.archive.org/web/20230217091319/http://www.atlassian.com/) 筹集了高达[6000 万美元](https://web.archive.org/web/20230217091319/https://techcrunch.com/2010/07/14/atlassian-accel-60-million/)的新资金，其中一部分将用于并购& A。该公司[去年收购了](https://web.archive.org/web/20230217091319/https://techcrunch.com/2010/09/29/atlassian-buys-mercurial-project-hosting-site-bitbucket/)[Bitbucket.org](https://web.archive.org/web/20230217091319/http://bitbucket.org/)，这是一项代码协作托管服务。今天，Atlassian 正在收购 [SourceTree](https://web.archive.org/web/20230217091319/http://www.sourcetreeapp.com/) ，这是 Git 和 Mercurial 分布式版本控制系统(DVCS)以及 Subversion 源代码控制的流行客户端。交易的财务条款没有披露。

Atlassian 的软件开发和协作工具，帮助团队构思、规划、构建和推出产品。产品包括 JIRA(问题跟踪器)、Confluence(内容协作)和 BitBucket。该公司的产品被全球 26，000 名客户使用，包括脸书、Zynga、思科和 Adobe。

SourceTree 帮助管理来自 Git 和 Mercurial DVCS 系统的存储库。正如 Atlassian 解释的那样，客户端剥离了分布式版本控制系统的复杂性，使其实际可用。

那么 Atlassian 为什么要收购 SourceTree 呢？Atlassian 表示，该客户将扩展他们当前的 Git 和 Mercurial 产品。随着最近宣布在 Bitbucket 中支持 Git，这次收购为用户提供了一个桌面客户端，可以使用任何 Git 或 Mercurial 存储库。因此，开发人员可以在 Bitbucket 或任何其他托管工具中托管他们的源代码，然后在 SourceTree 中管理该源代码。

SourceTree 从年开始在 Mac App Store 上出售，也可以从 SourceTree 网站上直接下载，每份许可证的价格为 59.99 美元。收购完成后，Atlassian 将在有限的时间内免费提供 SourceTree。该公司在收购 BitBucket 时也采取了类似的举措。

Atlassian 总裁 Jay Simons 表示，Atlassian 将在不久的将来进行更多收购，敬请关注。
