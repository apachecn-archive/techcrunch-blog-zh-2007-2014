# CrunchPad 诉讼的状态 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2010/09/15/status-of-crunchpad-litigation/>

去年年底，在发布前夕，我宣布 CrunchPad 项目的[结束，悲伤地结束了一个非常令人兴奋的](https://web.archive.org/web/20230205032819/https://techcrunch.com/2009/11/30/crunchpad-end/)[一年半的努力](https://web.archive.org/web/20230205032819/https://techcrunch.com/2008/07/21/we-want-a-dead-simple-web-tablet-help-us-build-it/)来创造一个非常便宜的触摸屏平板电脑。

我们的合作伙伴 Fusion Garage 莫名其妙地决定终止合作关系，原因“无非是贪婪、嫉妒和沟通不畅。”几周后，他们以新的名字推出了我们一起研发的设备，我们[提起诉讼](https://web.archive.org/web/20230205032819/https://techcrunch.com/2009/12/11/crunchpad-federal-lawsuit-filed-some-additional-thoughts/)，指控其欺诈和欺骗、盗用商业创意、违反信托责任、不公平竞争以及违反《兰哈姆法》。

丑陋的东西。

只不过当时我们真的不知道有多丑。经过近一年的发现，很明显 Fusion Garage 与他们的公关公司 [McGrath Power](https://web.archive.org/web/20230205032819/http://www.mcgrathpower.com/) 合谋(没有其他词可以形容)，几个月来一直在努力戏剧性地破坏合作关系，无视我们对项目的所有权，并在没有我们的情况下进入市场。麦格拉思的乔纳森·布鲁姆是主要头目，他和 Fusion Garage 实际上从头到尾策划了整个爆炸事件。他们唯一没想到的是一场真正的诉讼。

![](img/c9438922d09bca0397ee03459ebdae61.png)

这次爆发包括一项与媒体接触的计划，该计划以一个令人困惑的故事与 TechCrunch 竞争，该故事暗示我们应该负主要责任。没有任何证据表明这一点，但许多媒体都报道了这个容易写的故事。

听起来很疯狂？确实是。真的很疯狂。但这一切都发生了，内部邮件也证明了这一点。

在这一点上，我们已经通过了许多常规的预审材料，我们已经修改了我们的投诉，以指出我们发现的最糟糕的新证据。我在下面嵌入所有相关文档。Fusion Garage 的律师花了很多时间试图将这些文件封存起来，但它们最近被公开了。

下面的第一份文件读起来像一部小说，坦率地说，它可能会成为一部伟大的电影。我想，如果我不是被操纵的中心人物，我会认为这非常有趣。我们的律师实际上在第 19-20 页做了一个表格，显示他们如何向我们承诺一件事，而与他们的公关公司合谋做完全相反的事情。

例如，据称来自 Fusion Garage 投资者的[邮件实际上是 Fusion Garage 和公关公司写的。有很多关于语气和语言的来回，使这份欺诈性文件似乎是真实的。](https://web.archive.org/web/20230205032819/https://techcrunch.com/2009/12/04/crunchpad-litigation/)

![](img/42e265cf884dfd9c814b6359820fed97.png)

另一个例子是 Fusion Garage 的员工斯图尔特·谭(Stuart Tan)，他在脸书的个人资料中仍然将 Crunchpad 列为他正在进行的项目，他给女朋友发电子邮件告诉她，他“真的很骗这些人”他还提到，他将在我们的办公室进行这个项目，我们已经在投诉中添加了图片，显示 Fusion Garage 团队与我们的团队在我们的办公室和 CrunchPad 一起工作。

![](img/a6ebcd31a87c819798c5f2a551c76577.png)

乐趣并没有就此结束。还有其他文件显示公关公司编写 Fusion Garage 的语言，这样它就不会“暗示我们采用了 crunchpad 的概念，并进一步发展了它，做了 joojoo，现在正在宣布它……**我们需要清楚地表明这是一个不同名称的 crunchpad。**

“现在剧本读起来好像我们按照他想要的方式建造了它，现在从他那里拿走了产品并简单地改变了名称。这不是我想要我们去的地方。”

![](img/da53e74811427c12866ee3c2e4beb7af.png)
![](img/3186cd5ddeeda6bf300f51ce4060b44b.png)

最后，负责我们案件的法官做出了初步裁定，即“TechCrunch 做出了可信的证明，它可能能够证明存在一家合资企业，Fusion Garage 在该合资企业下负有某些信托责任。此类关税可能会阻止 Fusion Garage 在没有采取适当措施解除关系并对 TechCrunch 进行补偿的情况下将 joojoo 推向市场。在此，双方在共同开发平板电脑方面的行为比他们之间可能存在的任何不确定性更有说服力，因为他们之间的关系将如何构建。"
