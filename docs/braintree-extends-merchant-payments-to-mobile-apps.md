# Braintree 将商户支付扩展到移动应用

在线支付提供商 Braintree 为移动应用开发者推出了一套新工具，允许商家在移动应用内接受支付，而不是通过网络浏览器。作为背景，Braintree 为在线商家和公司的在线支付提供动力和自动化。该公司提供商业账户、支付网关、重复计费、信用卡存储、移动和国际支付支持以及 PCI 合规性解决方案。

当用户在移动设备上输入敏感的信用卡数据时，Braintree 的新产品通过加密这些数据来帮助开发人员避免 PCI 合规性问题。加密的数据从商家的服务器传递到 Braintree 进行处理，只有 Braintree 可以使用私钥解密信息，防止商家暴露敏感的信用卡数据。这些库支持运行 Android、iOS 和 Windows Phone 7 操作系统的手机和平板设备。

布伦特里坚持认为，在应用程序上输入信用卡信息的客户是最后一个看到它的人。该公司表示，其他移动应用支付解决方案通常使用伪装成应用的网络浏览器。

Braintree 的客户名单包括 LivingSocial、37signals、OpenTable、Fab.com、GitHub、Airbnb、Heroku、Engine Yard、Animoto、Shopify 和 HotelTonight。Braintree 每年处理超过 40 亿美元的信用卡交易量，并且每月新增 100 多家商户。该公司去年还从 Accel 获得了[3400 万美元](https://web.archive.org/web/20221207071559/https://beta.techcrunch.com/2011/06/29/accel-puts-34-million-in-online-payments-platform-braintree/)的资金。
