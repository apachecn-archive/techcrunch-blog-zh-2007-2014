# 个性化旅游搜索引擎 Room 77 现在允许您预订酒店房间 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/11/09/personalized-travel-search-engine-room-77-now-allows-you-to-book-hotel-rooms/>

[Room 77，](https://web.archive.org/web/20230205021050/http://www.room77.com/)一个专注于酒店客房的综合搜索引擎和评论网站，正在推出从其网站实际预订酒店的功能。从今天开始，旅行者现在可以使用 77 号房间比较各种房间类型的价格，并预订全球超过 120，000 家酒店。

今年 2 月，[在公测中正式推出的【Room 77 已经收集并索引了 2500 家酒店的 50 多万间客房的数据，还众包了游客的评论和评级。该网站为游客提供了每间酒店房间的具体细节，包括房间类别、面积、床类型、电梯距离以及是否是连通房间。对于每个房间，Room 77 还会生成一个虚拟房间视图，使用支持谷歌地球的技术模拟从该房间的窗口看到的实际视图。](https://web.archive.org/web/20230205021050/https://techcrunch.com/2011/02/24/room-77s-hotel-database-wants-to-make-sure-you-find-a-room-with-a-view/)

对于通过 77 号房预订的三星级至五星级酒店，旅行者可以获得免费的客房服务，该服务利用 77 号房专有的 RoomMatch 技术为每位旅行者确定最佳房间，然后直接与酒店合作，尝试代表旅行者获得一个房间。

有了新的预订功能，77 号房将直接在 Orbitz、Expedia、Travelocity、Priceline、Hotels.com、Cheaptickets 等网站的搜索结果中显示价格。您可以在网格视图中比较多个房间类别的价格，这使您可以看到下一个类别的房间包括什么，以及会多花多少钱。

除了价格、星级、位置和便利设施等常用过滤器之外，旅行者还可以根据独特的酒店和房间标准来优化酒店搜索，如景观类型、免费 Wi-Fi、免费停车、免费早餐和免费机场班车。对数千家酒店而言，77 号房分享了其团队“在大厅听到的”内部提示，提供最佳房间的意见、安静度和大小，以及:

正如我们过去所写的，77 号房是一项有用的服务，只要它能收集到关于酒店的正确情报。这就是为什么这家初创公司一直在与酒店集团合作，以提供准确的信息。将这一点与预订功能结合起来很有意义，可以让它在与用户评论巨头猫途鹰的竞争中占据优势。
