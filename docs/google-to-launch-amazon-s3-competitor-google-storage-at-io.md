# 谷歌将在 I/O 上推出亚马逊 S3 竞争对手的“谷歌存储”

本周，亚马逊的云存储服务将迎来另一个主要竞争对手:谷歌。我们听说本周在 T2 举行的 I/O 大会上，谷歌将宣布一项新服务，这是亚马逊 S3 云存储的直接竞争对手。谷歌的服务将被称为面向开发者的谷歌存储，简称 GS。我们相信它最初会有一个私人测试版。我们还听说，这项服务的定位将使现有的 S3 客户能够非常容易地转换到谷歌存储。

特性将包括 REST API、使用 Google 账户提供认证下载的能力以及数据冗余。开发人员将能够使用命令行工具来管理他们的数据，也将有一个 web 界面。

我们之前曾[报道过](https://web.archive.org/web/20230117164219/https://techcrunch.com/2010/04/12/google-plans-to-expand-cloud-computing-services-think-aws-on-steroids/)谷歌正在寻求扩展其云服务产品，但它将主要专注于“增值”服务，这些服务利用了谷歌内部一直在使用的技术，如其翻译工具和视频处理。我们听说这种增值服务将*而不是*成为此次发布的一部分，但它们很可能会在未来推出。这是这里的关键——竞争对手将很难赶上谷歌花了多年时间开发的一系列技术和基础设施。
