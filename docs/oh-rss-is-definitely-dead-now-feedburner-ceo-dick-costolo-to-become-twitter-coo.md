# 哦，RSS 现在肯定死了:Feedburner 首席执行官迪克·科斯特罗将成为 Twitter 的首席运营官

前谷歌高管，RSS 服务 Feedburner 的联合创始人兼首席执行官[迪克·科斯特罗](https://web.archive.org/web/20230302062259/http://www.crunchbase.com/person/dick-costolo)是 Twitter 的新任首席运营官，我们从多个渠道获悉。科斯特洛于 2007 年以 1 亿美元的价格将 Feedburner 卖给了谷歌，[于 7 月离开谷歌](https://web.archive.org/web/20230302062259/https://techcrunch.com/2009/07/07/feedburner-founderceo-dick-costolo-to-leave-google/)。我们听说他想开一家新公司，但显然 Twitter 突然介入并抓住了他。

Steve Gillmor 当然会喜欢这个，因为他宣称 RSS 已死，Twitter 是新的消息协议总线，或者类似的东西。*“安息吧，RSS，”* [他写了](https://web.archive.org/web/20230302062259/http://www.techcrunchit.com/2009/05/05/rest-in-peace-rss/)，说*“是时候彻底脱离 RSS，转投 Twitter 了……我所有的 RSS 订阅都在谷歌阅读器里。我不再去那里了。由于我所有的订阅都在谷歌阅读器里，而且我不去那里，所以我不再使用 RSS 了。”*

Twitter 现任运营总监 Santosh Jayaram(也来自谷歌)可能会继续留在公司，向 Costolo 汇报。

科斯特洛也是 Twitter 的早期投资者之一，他拥有构建可扩展基础设施的实际经验，而这正是 Twitter 迫切需要的。该公司在最近的记忆中没有推出任何新功能，并继续有[定期停机](https://web.archive.org/web/20230302062259/https://techcrunch.com/2009/08/06/oooh-dramatic-twitter-gets-ddosed/)。事实上，Twitter 无法构建功能并保持服务的实时性是一个严重的竞争劣势。科斯特洛大概可以解决所有这些问题。

我们听说，Twitter 正在积极招聘更多的高管。[7 月](https://web.archive.org/web/20230302062259/https://techcrunch.com/2009/07/12/twitter-expanding-executive-team-hires-general-counsel-from-google-looking-for-cfo/)他们聘请[谷歌产品和知识产权副总法律顾问 Alexander Macgillivray](https://web.archive.org/web/20230302062259/http://www.crunchbase.com/person/alexander-macgillivray) 为他们的新总法律顾问。
