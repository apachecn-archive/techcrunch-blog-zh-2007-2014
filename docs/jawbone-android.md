# Jawbone 发布 Android UP 应用，在欧洲苹果商店销售腕带

Jawbone [今天宣布其用于该公司运动追踪腕带](https://web.archive.org/web/20221005190532/http://www.sys-con.com/node/2585634) [UP](https://web.archive.org/web/20221005190532/https://jawbone.com/up) 的应用程序现在可以在 Google Play 上免费下载。129 美元的 UP 此前只兼容 iOS。这种腕带现在也可以在欧洲苹果商店买到，并将于下个月在亚洲和澳大利亚上市。

Jawbone 产品管理和战略副总裁 Travis Bogard 在一份声明中表示:“我们很高兴通过引入对 Android 的支持、iOS 的 11 种新语言以及在全球超过 25 个国家的产品可用性来扩大 UP 社区。”

当与它的应用程序结合时，UP 腕带允许用户跟踪他们的睡眠、运动、食物和情绪。下个月，亚洲和澳大利亚的苹果商店将开始销售这款产品，欧洲、亚洲、澳大利亚和中东的其他零售点也是如此。

虽然 Android 用户现在可以使用 Jawbone UP，但该公司表示，目前没有计划将 UP 应用程序发布到黑莓 10 或 Windows Phone。
