# 推特上充斥着声称收到姜饼的 Nexus One 用户

[![](img/39cda0e9a55d59c0e1e8c8400743b69f.png "Ginerbread part one")](https://web.archive.org/web/20221006073724/http://old.mobilecrunch.com/wp-content/uploads/2010/11/gingerbread3.jpg) 现在，这是一个未经证实的篮子，因为[我见过的唯一证明](https://web.archive.org/web/20221006073724/http://www.intomobile.com/2010/11/23/gingerbread-ota/)Nexus one 升级是两部分升级的第一部分，但[一些](https://web.archive.org/web/20221006073724/http://twitter.com/#!/macintosh/status/6982450550939648) [Twitter](https://web.archive.org/web/20221006073724/http://search.twitter.com/search?had_popular=true&q=gingerbread+android&result_type=recent) 用户正在紧张不安，声称 Android 2.3，又名[姜饼](https://web.archive.org/web/20221006073724/http://search.beta.techcrunch.com/query.php?s=gingerbread)，已经在他们的设备上发布。

现在，正如我之前所说，我看到的唯一更新是两阶段更新的第一阶段。重要的是要认识到第二阶段是姜饼；首先是准备好你的设备。

很难说收到第一次更新和姜饼被推送到你的手机之间会有多长时间的间隔。据我们所知，第二次更新可能还要一个月才会到来。

如果我听说有人接受了关键的第二阶段，我会让你知道。

我们正在耐心等待谷歌关于姜饼发布日期的任何官方消息，你可以肯定的是，一旦我们听到任何消息，我们都会发布。

**更新:**是的，那个更新看起来只是一个 bug 修复，[不是姜饼](https://web.archive.org/web/20221006073724/http://twitter.com/eclipsed4utoo/statuses/7029230818820097)。
此外，感谢 Varunkumar Nagarajan 在评论中确认这只是 Android 2.2.1 FRG83D，而不是为姜饼准备设备的更新。
