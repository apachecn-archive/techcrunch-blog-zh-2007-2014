# Changers 推出售价 149 美元的移动社交太阳能充电系统

今天，Changers 将发布其新的太阳能充电系统，该系统旨在通过一个简单的一键式设备为你的 USB 设备充电，包括 iPhones、iPods、Android 手机和 Kindles。[转换器启动套件](https://web.archive.org/web/20230203105438/http://www.changers.com/en/shop/us/starterkit)，包括该公司的“Kalhuohfummi”(TM)太阳能电池和转换器“Maroshi”(TM)太阳能模块，每小时可产生高达 4 瓦的电能。

这个系统也有社会因素。它将产生的电能上传到位于 Changers.com[的在线能源市场](https://web.archive.org/web/20230203105438/http://www.changers.com/)，在那里你可以比较并与朋友分享你节省的能源。

科罗拉多州生产的太阳能电池板可以安装在任何窗户或阳光充足的表面。该套件的太阳能模块可以在四小时内为系统的 16 瓦电池充电。该公司表示，当给你的移动设备通电时，该系统能够像墙上插座一样快速充电。

此外，你的太阳能充电数据被在线上传到 Changers.com，在那里你可以看到你节省的能源。你也可以与其他社区成员比较你的分数，或者在脸书和推特上与朋友分享。这些节省下来的能源被转化为“兑换信用”,可以在公司的零售合作伙伴处使用。(在发布时，目前只有一家:Holstee[出售用回收材料制成的商品。100 瓦能让你在网站上获得 10 美元)。](https://web.archive.org/web/20230203105438/http://shop.holstee.com/)

[Changers](https://web.archive.org/web/20230203105438/http://www.changers.com/) 现在接受其[首发套件](https://web.archive.org/web/20230203105438/http://www.changers.com/en/shop/us/starterkit)的预购，合理定价为 149 美元。最初只有有限的数量。该公司成立于柏林，但正在美国推出，以吸引早期用户群。
