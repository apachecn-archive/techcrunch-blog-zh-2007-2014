# 图表:在四年时间里，苹果售出的 iPhones 比所有 MAC 电脑都多

如果有人对 iPhones、iPads 和其他 iOS 设备是苹果的未来有任何疑问，只要看看上面来自 Asymco 的图表就知道了。它显示了过去 28 年里累计售出的所有 iOS 产品与 OS X 售出的所有 MAC 产品的对比。iPhone 问世才四年，但在这段时间里，iPhone 的销量超过了苹果历史上所有 MAC 电脑的销量。

MAC 总销量为 1.22 亿台。仅在 2011 年，如果你把包括 iPods 和 iPads 在内的所有 iOS 设备加起来，就卖出了 1.56 亿台——比所有 MAC 电脑卖出的都多。在*一*年。总的来说，苹果在过去几年里已经售出了 3.16 亿台 iOS 设备，就数量而言，iOS 设备完全压倒了 OS X 设备。难怪苹果会让 OS X 的外观和感觉[更像 iOS。](https://web.archive.org/web/20230102135938/https://techcrunch.com/2012/02/16/mountain-lion-os-x-ios/)

我甚至不知道如何描述 iPhone 销售曲线的陡峭程度。即使是“暴涨”也不公平。
