谷歌下周正式推出Chrome扩展

几周前，谷歌推出了它的Chrome扩展网站，因为有迹象显示将在他们的浏览器中全面推广的扩展支持。不幸的是，这个网站只对被允许上传他们的作品到谷歌的扩展开发者开放。在页面上，谷歌承诺寻找这些扩展的最终用户将“很快”有办法做到这一点我们了解到，这将在下周发生。

两名知情人士称，谷歌计划在下周的某个时候公布其扩展库，可能是在本周中期。这是有意义的，因为下周五将举行一个专门讨论浏览器插件的会议，谷歌Chrome是该活动的黄金赞助商。显然，谷歌可能希望有一些他们可以在活动中展示的东西，而不仅仅是一个开发者仪表板。

显然，扩展库将会很像Chrome的主题库。这将是一个页面，列出了一堆的扩展，并有一个按钮，一键下载你想要的。想必还会有一个链接来了解更多关于每个扩展的实际功能。

一些开发者已经为Chrome准备好了他们的扩展。我们最近对 Aviary的和 Shareaholic的进行了简介。事实上，一段时间以来，通过像 Chrome Extensions 这样的网站，已经有数百个非官方的Chrome扩展可供使用。今天早上我们对在那里发现的11个最好的进行了简介。

最初，扩展支持将只针对基于Windows的Chrome版本。尽管Mac版Chrome测试版即将推出，但该版本不会内置扩展支持。然而，最新版本的Mac版Chromium(Chrome所基于的开源浏览器)确实支持扩展，甚至有一个有效的扩展管理器。看起来Chrome的Linux版本只要有测试版就支持扩展。

扩展对Chrome非常重要，因为它试图在未来几年内达到谷歌宣称的10%市场份额目标。扩展一直是Firefox成功的关键之一，因为它继续从曾经完全占主导地位的ie浏览器那里窃取市场份额。
