Windows 7通过测试，准备投入生产| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2009/07/22/windows-7-passes-the-test-is-ready-for-manufacturing/amp/

微软最新版本的操作系统 Windows 7 终于进入了发布制造(RTM)阶段，因此该操作系统将很快预装在新电脑上。尽管尚未正式发布，但Windows 7预计将会大受欢迎。例如，在亚马逊英国网站上仅仅8个小时后，Windows 7的预购量就超过了17周内Vista的预购总量。

根据微软的说法，提供七个不同版本操作系统的Windows 7已经经历了重要的测试、质量保证和达到RTM阶段所需的验证。独立软件和硬件供应商最早将于8月6日开始下载Windows 7 RTM。微软将在8月中下旬向其他合作伙伴推出Windows 7。从8月7日开始，企业客户和开发人员将能够下载英文版的Windows 7 RTM，随后不久将发布针对Windows 7的其他语言功能。

对于普通人/消费者来说，Windows 7将于10月22日开始在零售店和新电脑上销售，这我们已经知道了。在收到beta测试人员的热烈响应后，微软也为Windows 7提供了一个“家庭包”,允许在最多3台电脑上安装。该公司还在正式发布了 Expression 3的新版本，这是微软为开发者提供的基于Silverlight构建应用的工具集。

当然，Windows 7的正式RTM发布正值微软对该操作系统的垄断受到最近发布的谷歌Chrome OS的挑战。谷歌计划在今年晚些时候发布Chrome操作系统的开源代码，这可能会在10月份Windows 7发布时推出。第一批Chrome OS电脑要到明年才会上市。虽然谷歌表示Chrome操作系统目前的目标是上网本，但谷歌操作系统确实有潜力扩展到其他类型的电脑，这给了微软一些思考的空间。
