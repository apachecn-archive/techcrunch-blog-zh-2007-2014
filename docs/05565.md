# 中国的企业家是如何帮助它获胜的

[![](img/dadf2117f2ec6abc1368fb3001815946.png "Win In China")](https://web.archive.org/web/20221005095111/https://beta.techcrunch.com/wp-content/uploads/2010/11/win-in-china.jpg) 鲍勃·康普顿和我终于有了意见一致的地方。

这位华盛顿特区的风险投资家制作了一部引人注目的纪录片，名为[200 万*分钟*](https://web.archive.org/web/20221005095111/http://www.2mminutes.com/) ，记录了六名学生——美国、印度和中国各两名——在高中最后一年的经历。它展示了印度和中国学生努力学习数学和科学，以及美国人聚会和玩电子游戏。鲍勃总结说，印度人和中国人会吃我们孩子的午餐，因为他们受到了更好的教育。[纪录片中出现了我，我同意印度和中国的孩子确实比美国孩子更加努力；他们从小就相信教育就是一切，将决定成功与饥饿；他们童年的大部分时间都花在背高级学科的书上。然而，我认为，对美国竞争力而言，事情远没有纪录片中表现的那么糟糕。我的团队对全球工程教育的研究表明，超过 95%的印度人和中国人没有接受良好的教育，即使他们接受了良好的教育，也比美国人花费更长的时间来培养关键的现实技能。是的，美国青少年兼职、社交、聚会。但是他们培养的独立性和社交技能在他们参加工作时给了他们很大的优势。他们学会实验，挑战规范，并承担风险。他们从一开始就创新。](https://web.archive.org/web/20221005095111/http://www.businessweek.com/technology/content/may2008/tc20080527_245830.htm)

正如我在这篇文章中解释的那样，尽管教育系统薄弱，印度已经成功地建立了创新和研发能力:它的私营企业对工程毕业生进行再教育。中国工业没有同等的劳动力发展实践，因此该国在研发外包和创新方面落后。中国通过升级基础设施实现了奇迹；让数亿人摆脱贫困；建设世界一流的大学和最先进的研究设施。但是它的国有企业——在工业中占主导地位——仍然是官僚的、腐败的，并且由共产党员独裁管理。跨国公司大肆宣传他们在中国的研发，但事实上，他们在中国做的大部分工作是他们技术的本地化，而不是创新新产品的设计。此外，尽管有令人印象深刻的数字，但中国学者发表的论文和研究人员申请的专利并不值得印刷它们的纸张——它们大多是抄袭或不相关的。

但中国的情况正在迅速变化。它的人民变得非常有企业家精神。而且，正如莎拉·莱西[写的](https://web.archive.org/web/20221005095111/https://beta.techcrunch.com/2009/06/01/why-china-isn%E2%80%99t-%E2%80%9Cthe-next-silicon-valley%E2%80%9D/) [写的](https://web.archive.org/web/20221005095111/https://beta.techcrunch.com/2010/09/15/how-did-i-get-more-bullish-on-chinese-innovation-than-kai-fu-lee-2/)，一些人开始超越复制西方技术；他们开始创新。康普顿在一部名为 [*赢在中国:中国创业大爆发*](https://web.archive.org/web/20221005095111/http://www.youtube.com/user/WinInChinaMovie) 的新电影中准确记录了创业趋势。这显示了中国向资本主义经济的快速转变——由企业家带头。

在本周的北京之行中，我目睹了同样的趋势。我去那里教了一个由加州大学伯克利分校创业和技术中心主办的课程(我是该中心的讲师)。我教了大约 50 名学生，并会见了几十名当地的科技企业家。这些学生与我在加州大学伯克利分校和杜克大学教的学生没有什么不同:同样聪明，思想开放，富有创新精神，有改变世界的动力。这些企业家也出乎意料地像我在硅谷遇到的企业家一样——雄心勃勃，无所畏惧，决心建立世界级的公司。在过去的 5 年里，我注意到与我之前去中国时看到的一个很大的不同是，他们更能接受失败，并乐于承认他们一直在抄袭美国的技术:他们在 TechCrunch 上读到的任何东西。但他们中的大多数人都谈到了利用他们的经验来打造针对中国市场的新一代产品。他们对为西方制造产品毫无兴趣。他们也比前几代人更加自信和成熟。

所以鲍勃和我完全同意中国创业革命的重要性。对于我们的孩子必须担心的问题，我们不再有任何分歧:当涉及到全球竞争力时，中国人(和印度人)将吃掉我们孩子的午餐。更多的创新是好的，无论它发生在哪里，但是美国将面临来自东方新兴力量的激烈竞争。我们最好鼓励我们的孩子提高他们的技能，了解全球市场，并为他们复制中国和印度的技术做好准备。
