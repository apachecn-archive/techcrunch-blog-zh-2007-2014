# 联想 ThinkPad 蜂巢平板电脑上市销售，即将发货

联想的 ThinkPad 平板电脑是为商务人士设计的，但它可能是目前最好的蜂巢平板电脑。上个月晚些时候，它与面向消费者的 IdeaPad K1 和 Windows 版本的 P1 一起发布。好吧，联想终于准备好拿你的信用卡来换取航运公司的一个位置。平板电脑[的 16GB 纯 WiFi 版起价为](https://web.archive.org/web/20230204223324/http://shop.lenovo.com/SEUILibrary/controller/e/web/LenovoPortal/en_US/catalog.workflow:category.details?current-catalog-id=12F0696583E04D86B9B79B0FEC01C087&current-category-id=00CE3A2177ABA63564CDF6AE8CD837B3)499 美元，64GB 的价格攀升至 669 美元——3G 移动宽带要到 10 月份才能上市。

ThinkPad 平板电脑穿着得体；看起来像是 ThinkPad 平板电脑。线条笔直，商务风格，甚至在可选的 30 美元有源数字化手写笔上还有一个红色的小点。平板电脑的其余部分是一个标准事件，围绕 Tegra 2 平台构建。有一个全尺寸 3 合 1 SD 卡读卡器，全尺寸 USB 主机，微型 USB 端口，迷你 HDMI，双摄像头，和一个由大猩猩玻璃保护的 IPS 屏幕。

ThinkPad 平板电脑唯一缺少的是有趣的应用程序。联想在这款平板电脑上安装了 suits 喜欢的应用程序和~~与它的弟弟 IdeaPad K1 不同，ThinkPad 平板电脑没有网飞认证，因此它没有安装流媒体应用程序~~。不过，对于 499 美元，你很难找到更好的蜂窝平板电脑——当然，如果你真的想要一个蜂窝平板电脑的话。

**更新**:我的不好。ThinkPad 平板电脑确实配有网飞。奖金！
