Google+现在是苹果应用商店中最热门的免费应用

在上线不到24小时后， Google+ iPhone应用击败了“Draw ' n ' Go:awesome！”争夺第一名。

这是相当令人印象深刻的，考虑到这个应用程序，开始时有问题，并且在Erick见过的“iPhone应用程序的最快更新”之后继续有问题，目前在实地测试中最多只与估计的1800万人相关，并且谷歌有自己的移动操作系统Android需要关注。

让这一里程碑变得重要的是，Google+的竞争对手脸书目前拥有有史以来下载量最高的免费iOS应用——所以Google+是一个好公司。当然，谷歌确实有很多营销杠杆来推动它需要的东西，也就是它的高流量主页。
