# TaskRabbit 有了新的首席执行官，埃里克·格罗斯

消费者任务市场 [TaskRabbit](https://web.archive.org/web/20221224150633/http://www.taskrabbit.com/) 今天早上更换首席执行官，从联合创始人兼 IBM 程序员 [Leah Busque](https://web.archive.org/web/20221224150633/http://www.crunchbase.com/person/leah-busque) 到前 Hotwire 首席执行官 [Eric Grosse](https://web.archive.org/web/20221224150633/http://www.linkedin.com/profile/view?id=6106828&authType=NAME_SEARCH&authToken=SfDA&locale=en_US&srchid=9450f494-410a-4f19-9b31-b48c63bacab7-0&srchindex=1&srchtotal=5&goback=%2Efps_PBCK_*1_Eric_Grosse_*1_*1_*1_*1_*2_*1_Y_*1_*1_*1_false_1_R_true_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2_*2&pvs=ps&trk=pp_profile_name_link) 。

当我本周早些时候拜访他们两人时，格罗斯似乎对公司(和他)的新方向充满信心，“就像旅行在早期——90 年代末和 21 世纪初——帮助电子商务加速发展一样，我真的认为[这就是 TaskRabbit 在这里要做的事情。协作消费这一更广泛的概念将成为下一个十年如何展开的一个非常重要的部分。"

如果有人不清楚 Taskrabbit 是做什么的，这里有一篇非常有深度的文章#humblebrag。该公司目前正从[第一轮资本](https://web.archive.org/web/20221224150633/http://www.firstround.com/)、[沙斯塔风险投资公司、](https://web.archive.org/web/20221224150633/http://www.shastaventures.com/)等机构获得[685 万美元](https://web.archive.org/web/20221224150633/https://techcrunch.com/2011/05/04/taskrabbit-gets-5m-to-help-people-get-stuff-done/)的资金。祝你好运埃里克。
