# 微获取:TwitterCounter 包含 Twitaholic

![](img/83edcd2678a631124e7c26c0fe969a1f.png)

在有[推特](https://web.archive.org/web/20230222201818/http://www.crunchbase.com/company/twitter)统计网站[推特计数器](https://web.archive.org/web/20230222201818/http://twittercounter.com/)之前，有[推特狂人](https://web.archive.org/web/20230222201818/http://www.crunchbase.com/product/twitterholic)(现在叫[推特狂人](https://web.archive.org/web/20230222201818/http://twitaholic.com/))。现在，Twitter Counter 已经收购了 Twitaholic，以支持其 Twitter 统计服务，合并后的实体将被命名为 TwittaHolicCounterTweet(这里只是开玩笑)。

你还在嘲笑我吗？

说真的，Twitaholic 是第一个在 Twitter 上发布前 100 名用户排名的网站，而 TwitterCounter 是第一个提供详细的[个人统计数据](https://web.archive.org/web/20230222201818/https://techcrunch.com/2009/09/10/twittercounter-wants-to-count-dollars-too/)和可嵌入的小工具和横幅来吹嘘你的粉丝数量的网站，所以这是有意义的。

自 2009 年初以来，TwitterCounter 一直在与 Twitaholic 背后的人交谈，twit aholic 对其创始人来说一直是一个次要项目，但现在已经直接收购了该网站(显然不是大笔收购，尽管细节仍未披露)。

TwitterCounter 表示，它可以通过提供 [premium statistics](https://web.archive.org/web/20230222201818/http://twittercounter.com/pages/premium) (目前被微软、易贝、可口可乐和脸书等公司使用)和特色用户点来赚钱，这给人们提供了一个获得更多粉丝的简单方法。

TwitterCounter 可能看起来有点噱头，但它实际上是一个相当不错的小企业——该公司有 3 名全职员工，2010 年的收入约为 50 万美元，并且仍在增长。

Twitaholic 的创始人正在转向其他项目。虽然 TwitterCounter 说可能会对布局或功能进行一些小的改动，但该网站将保持活跃。

为了充分披露:原来 Twitaholic 是持有 Emurse 的同一家公司的一部分，Emurse 在 2009 年 4 月被我们的母公司 AOL 收购(然而 AOL 对 Twitaholic 没有兴趣)。Twitaholic 的创始人现在将专注于开放 AOL 全新的奥兰多办公室并在那里工作(实际上下周开放)。
