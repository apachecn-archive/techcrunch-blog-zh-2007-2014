# Pixel Qi 完成 B 轮融资，获得 3M 投资

创始人兼首席执行官 Mary Lou Jepson 一直对 Pixel Qi 的低功耗、超反射液晶显示器寄予厚望。虽然这种显示器出现在像[概念墨水亚当](https://web.archive.org/web/20230204225722/https://techcrunch.com/tag/notion-ink-adam/)这样的小众平板电脑中，但它们还没有达到杰普森一直想要的那种大众市场渗透率。

Jepson 的愿望可能很快就会实现，如果今天的声明有任何迹象的话:工业巨头 3M 公司[正在向 Pixel Qi 投资一笔未披露的金额](https://web.archive.org/web/20230204225722/http://www.marketwatch.com/story/3m-invests-in-pixel-qi-corp-2011-09-12)，作为该公司 B 轮投资的一部分。

Pixel Qi 的显示器由于其高度适应性，近年来引起了相当大的反响。当背光打开时，屏幕看起来像任何其他 LCD，但当背光关闭时，高反射显示器呈现 eInk 屏幕的外观。最终的结果是，屏幕在阳光下和在黑暗中一样好用，电池续航时间也很长。

这笔交易是通过 3M 公司的新风险投资部门进行的，其细节仍被掩盖着，但它应该是 Jepson 在圣布鲁诺运营的强心针。

“在我们与 3M 的合作中，我们有能力加速这项技术的大规模应用，”杰普森说。3M 的投资将使 Pixel Qi 加快生产，这是有充分理由的:Pixel Qi 将继续致力于将其显示器应用于更多面向消费者的设备，同时该技术也正在被用于数字标牌和“触摸应用”如果一切顺利，第一批面向大众市场的 Pixel Qi 设备将很快上市。
