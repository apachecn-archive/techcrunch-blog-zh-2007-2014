# 更多孵化器爱柏林:德国创业集团提供(高达)€25 万美元投资

又是一天，[另一个](https://web.archive.org/web/20221006195313/https://beta.techcrunch.com/2012/07/19/from-startup-to-incubator-games-publisher-hitfox-launches-hitfox-game-ventures/)新的孵化器从德国出来(事实上，感觉整个欧洲最近都有点孵化器疯了):[德国创业公司集团](https://web.archive.org/web/20221006195313/http://www.germanstartupsgroup.de/)是由 Christoph Gerlinger 和 Alexander klpin 成立的一个新基金，他们两人看起来很适合利用柏林当地的“数字”创业公司，投资金额在€5 万到€25 万之间，尽管不排除来自更远地方的申请。

在被 Gamesforge 收购之前，Christoph Gerlinger 是游戏发行商 [Frogster](https://web.archive.org/web/20221006195313/http://www.crunchbase.com/company/frogster-interactive-pictures) 的创始人兼首席执行官，而 lpin 是 BerlinWebWeek 的联合创始人，并作为位置标记代理 BerlinPartner 的部门总监，以营销的身份与当地初创公司合作。

就像所有其他孵化器一样，早期投资者希望将他们对交易流的兴趣描绘成小狗追逐自己尾巴的热情，德国初创公司集团(Startups Group)正在宣扬其企业家友好性(“我们爱初创公司”)。除了为少数股权注入现金，该基金还将在整个创业阶段为公司提供初始办公空间、实用建议和支持。

或许令人耳目一新(或不令人耳目一新)，它也强调了享受乐趣的需要。“对于几年的工作关系来说，特别重要的是我们一起工作很开心。因此，双方的化学反应必须匹配”，Gerlinger 说。

在投资标准方面，德国创业公司集团(Startups Group)正在寻找“产品或商业模式的颠覆性创新、高可扩展性和创业人才”，对于申请人，它还承诺“直截了当和标准化的投资合同”，无论这意味着什么。(这才是创业者真正需要做[功课](https://web.archive.org/web/20221006195313/https://beta.techcrunch.com/2011/07/06/in-a-historic-move-15-european-investors-agree-on-standard-term-sheet-for-startups/)的地方。)

尽管德国创业集团表示已经筹集了“数百万欧元”，但它也在邀请其他投资者加入。

它加入了一系列活跃在柏林的孵化器，包括 Startupbootcamp、makeastartup.com、youisnow.de、Founderslink、Wayra Berlin、Team Europe/Point Nine Capital、FoundFair、Betahaus 和 V-Check。
