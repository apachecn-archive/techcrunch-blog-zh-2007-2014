# 脸书雇佣记者安德鲁·诺伊斯增加在华府的影响力

不可避免的事实是，那些对美国人的生活和/或钱包产生重大影响的公司最好在华盛顿特区有一些坚实的基础——问问像微软和谷歌这样的公司就知道了。

另一个网络巨头脸书也不例外，因此毫不奇怪，他们从国家首都招募了另一个人来帮助管理公共政策沟通，就在公司公共政策总监蒂姆·斯帕拉帕尼的领导下。

认识一下[安德鲁·诺伊斯](https://web.archive.org/web/20230212121428/http://www.andrewnoyes.net/bio.html) (29 岁)，一位经验丰富的记者，他最近[为《国家杂志》的《国会日报》撰写了关于“网络世界的政治和政策”的博客](https://web.archive.org/web/20230212121428/http://techdailydose.nationaljournal.com/)，并为《国家杂志》撰写了一些文章。(阅读他的简历，了解他在过去十年里还做了什么)

上周五内部宣布了新公共政策经理的聘用，并由诺伊斯[在其推特账户](https://web.archive.org/web/20230212121428/http://twitter.com/anoyes/status/5100111680)上公开确认(是的，他也在脸书上[)。](https://web.archive.org/web/20230212121428/http://www.facebook.com/andrew.noyes1)

以下是脸书公司企业沟通和公共政策总监巴里·施尼特向军队传达这一消息的方式:

> 我想让您知道华盛顿特区脸书公司的一名新员工，29 岁的安德鲁·诺伊斯将加入公司，担任公共政策传播经理。Andrew 将于 11 月中旬正式开始工作，他将带来丰富的相关经验，帮助脸书管理和进一步发展在美国首都的关系。
> 
> 随着脸书不断创新和发展，用户群体迅速增长，超过 3 亿人，我们必须扩展能力**向政策制定者、媒体和公众**解释我们如何让人们在更可信的环境中分享，帮助世界变得更加互联，并推动经济增长。安德鲁和脸书 DC 办公室的其他人议程上的具体政策问题包括**加强网络安全和在线安全，通过用户控制数据扩大数字隐私保护，以及保护言论自由。**
