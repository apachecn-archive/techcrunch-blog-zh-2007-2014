# 由 Accel TechCrunch 领投的电话营销公司 Invoca 融资 2000 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/13/invoca-series-c/>

# 电话营销公司 Invoca 融资 2000 万美元，由 Accel 牵头

电话营销公司[的前身是 ring revenue](https://web.archive.org/web/20221210033307/http://www.invoca.com/invoca-launches-to-unlock-the-power-of-the-phone/)Invoca 宣布，它已经筹集了 2000 万美元的 C 轮融资。

该公司目前已经筹集了超过 3000 万美元，本轮由 Accel Partners 领投。advanced Ventures 和 Rincon Venture Partners 也参与其中，Accel 的 Kobie Fuller 也加入了该公司的董事会。

富勒在资金发布会上表示:“尽管他们的最高价值线索是通过电话获得的，但首席营销官通常会受到电话渠道的阻碍，因为他们认为电话渠道缺乏其他数字渠道所具备的同样的衡量和优化。”。“Invoca 决心通过向企业营销人员提供最有效的解决方案来改变这种心态。”

Invoca 的产品包括呼叫跟踪、呼叫分析，以及基于位置和他们是否是新客户等因素的定制处理。它还集成了其他销售和营销产品，包括 Salesforce.com、Adobe、Kenshoo、Marketo、iCrossing、Cake、Tealium 和 HasOffers。

据说该公司去年的年增长率为 200%。据说它被“超过 3000 名营销人员”使用，包括自由互助保险、OpenTable、Answer Financial 和 DirecTV。

Invoca 的一位发言人告诉我，该公司计划在明年开设一家湾区办事处(总部位于加州圣巴巴拉)，并将员工人数增加一倍，达到 150 多人。这位发言人说，它还将在销售、营销和产品开发方面投入更多资金。
