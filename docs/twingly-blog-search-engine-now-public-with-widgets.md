# Twingly 博客搜索引擎现已公开(带插件)

[![](img/de36d022ccacffb0f8555e58b750ad5c.png "twingly-logo")](https://web.archive.org/web/20221006041903/http://www.crunchbase.com/company/twingly)

瑞典创业公司[结对](https://web.archive.org/web/20221006041903/http://www.twingly.com/)今早向公众开放了其博客搜索引擎。它现在覆盖了 3000 万个博客，搜索结果可以根据权威(链接数量)或最近的帖子进行排序。还有一个无垃圾邮件的搜索选项，只能从数十万个已知真实的白名单博客中返回结果。(阅读我们之前的[报道](https://web.archive.org/web/20221006041903/http://www.beta.techcrunch.com/2008/04/02/europe-focused-blog-search-engine-twingly-goes-into-private-beta/)，了解更多细节)。

在这一点上，任何博客搜索引擎都很难与谷歌博客搜索或 Technorati 抗衡，但 Twingly 已经放弃了。它对欧洲的关注和翻译特色可以帮助它开拓出一个利基市场。

首席执行官马丁·卡尔斯托姆在博客[上强调了他的搜索引擎的一些特性:](https://web.archive.org/web/20221006041903/http://blog.twingly.com/2008/06/12/twingly-launching-out-of-beta-with-new-widgets-for-bloggers/)

> *–无垃圾邮件搜索
> –社交搜索。用户通过对他们喜欢的帖子投票来增强搜索结果。博主通过链接到他们喜欢的帖子来增强搜索结果
> –通过 RSS 订阅搜索结果并通过电子邮件发送提醒
> –语言功能:翻译搜索结果并基于语言进行过滤
> –Twingly widget 平台。Twingly.com 的部分地区可以被纳入博客
> ——现在很热。博客圈热门话题概述
> –通过公开投票的技术计划进行用户导向的开发。*

任何搜索词都可以通过 RSS 订阅。成绩可以投票上去。最棒的是，用户可以投票决定 Twingly 下一步应该做什么。

今天，该公司还发布了一些小部件，博客可以把它们放在自己的网站上，显示最近的帖子、最近的链接和最多链接的帖子。这里是 TechCrunch 的[示例，你可以在这里](https://web.archive.org/web/20221006041903/http://www.twingly.com/blogs/profile?url=beta.techcrunch.com)创建自己的小工具[。](https://web.archive.org/web/20221006041903/http://www.twingly.com/help#widget)
