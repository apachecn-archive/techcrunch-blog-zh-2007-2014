# 出版公司使用苹果的 iBooks 作者工具来制作 iPad 版的莎士比亚作品

美国图书出版商 Sourcebooks 已经使用[苹果的 iBooks 作者工具](https://web.archive.org/web/20221006163428/https://beta.techcrunch.com/2012/01/19/apple-unveils-new-ibooks-author-tool-not-just-for-textbooks/)在苹果的 iBooks 2 商店推出了三部莎士比亚戏剧的 iPad 版。iBooks 包括专注于每部戏剧表演历史的额外内容，包括来自不同演员的照片、视频和音频片段，以及便于学习的主要内容，如词汇表资源、笔记和突出显示功能。

这一系列的 iBooks 有一个相当令人畏缩的标题:T2 的《震撼体验》。这三部戏剧是:《哈姆雷特》、《奥赛罗》和《罗密欧与朱丽叶》。每部戏剧的不同表演和阅读的片段和片断被纳入 iPad 版本，目的是吸引读者和学生。演员德里克·雅各比爵士介绍了音频剪辑，他是 iBooks 中的一系列莎士比亚演员之一。

“老师告诉我们，学生需要三周时间才能熟悉莎士比亚的语言。*莎士比亚体验*旨在改变你阅读莎士比亚的方式，”出版商在 iBooks 的营销材料中写道。

这里有更多来自[资料书的简介](https://web.archive.org/web/20221006163428/http://www.sourcebooks.com/spotlight/shakesperience.html)

> 每部*莎士比亚戏剧都由著名的莎士比亚学者开发，是学生的理想之选，提供了增强读者体验的特殊功能，使之成为你所能想象的最友好、最迷人的莎士比亚作品。*

Sourcebooks 首席执行官兼出版商多米尼克·拉克在一份声明中指出:“我们已经接受了传统上困难和普通的经历，并提供了更积极和生动的学习机会。”“专家们会喜欢它，但我们最兴奋的是它对教育者和学生的影响。”

苹果的 iBooks Author 工具于 1 月发布，当时苹果宣布在其 iBooks 生态系统中大力推动教科书，为 iPad 推出 iBooks 2 的[。](https://web.archive.org/web/20221006163428/https://beta.techcrunch.com/2012/01/19/apple-announces-ibook-2-a-new-textbook-experience-for-the-ipad/)

当时我们说 [iBooks Author tool](https://web.archive.org/web/20221006163428/http://www.apple.com/ibooks-author/) 为潜在出版商提供了构建任何类型 iBook 的模板，而不仅仅是教科书。Sourcebook 的《Shakesperience》是一种混合体——你不能称之为纯粹的教科书——这表明苹果公司有很大的空间继续扩大其商店中添加的交互式 iPad iBooks 的数量。

我们已经联系了苹果，询问其 iBooks 2 应用商店中有多少图书，并将根据任何回复进行更新。(**更新**:苹果发言人说:“我们披露的唯一数字是 iBookstore 上的‘几十万’本书。”)

苹果的 iBooks Author 软件旨在通过拖放界面添加小工具和其他图形、交互和多媒体元素，从而轻松增加文本。该工具可从 Mac App Store 免费下载。
