# 谷歌收购神经网络初创公司 DNNresearch，以提升其语音和图像搜索技术

好吧，谷歌的 M&A 战略如果没有多样化的侧重点，那就什么都不是。11 月，[收购包裹递送初创公司 Bufferbox](https://web.archive.org/web/20230223090216/https://techcrunch.com/2012/11/30/why-did-google-buy-bufferbox-because-the-entire-mail-and-package-delivery-system-is-broken/) 。上个月，谷歌[进行了今年的首次收购](https://web.archive.org/web/20230223090216/https://techcrunch.com/2013/02/06/google-acquires-channel-intelligence-to-boost-product-recommendations-and-e-commer-with-users/)，收购了电子商务初创公司 Channel Intelligence。今天，谷歌挖到了多伦多大学的计算机科学系，收购了由 Geoffrey Hinton 教授和他的两名研究生 Alex Krizhevsky 和 Ilya Sutskever 创办的年轻创业公司 [DNNresearch](https://web.archive.org/web/20230223090216/http://www.dnnresearch.com/) 。

这家初创公司去年成立，网站上明显没有任何识别信息——只有一个空白的黑屏。虽然交易的财务条款没有披露，但谷歌渴望获得这家初创公司对神经网络的研究——以及背后的人才——以帮助它在识别内容、图像、语音、文本等方面的能力超越传统的搜索算法。在今天的公告中，多伦多大学表示，该团队的研究“对语音识别、计算机视觉和语言理解等领域有着深远的影响。”

此外，Hinton 教授是伦敦大学学院 Gatsby 计算神经科学部门的创始主任，拥有机器学习方面的加拿大研究职位，并且是加拿大高级研究所资助的“神经计算和自适应感知”项目的主任。同样是英国皇家学会的成员，辛顿教授因其在神经网络方面的工作和对“具有丰富感觉输入的神经网络的无监督学习程序”的研究而闻名。

多伦多大学在声明中表示，Krizhevsky 和 Sutskever 都将转到谷歌，而 Hinton 将“在大学研究和谷歌工作之间分配时间”，既在谷歌的多伦多办公室，也在山景城的谷歌总部。

对谷歌来说，这意味着获得访问权，特别是该团队对改善物体识别的研究，因为该公司希望提高其图像搜索和面部识别能力的质量。继 2011 年收购 PittPatt 和 2006 年收购 Neven Vision 两家类似的初创公司之后，该公司最近收购了拥有多项面部识别专利的 Viewdle 。

此外，谷歌一直在寻求改善其语音识别、自然语言处理和机器学习，将其与知识图相结合，以帮助开发一个勇敢的新搜索引擎。谷歌已经在网络上拥有深度图片搜索能力，但是，随着智能手机的普及，它将寻求改善移动端的体验。

[在 DNNresearch 的三位创始人最近发表的一篇论文](https://web.archive.org/web/20230223090216/http://www.cs.toronto.edu/~hinton/absps/imagenet.pdf)中，该团队发现，“尽管 CNNs 卷积神经网络]具有吸引人的品质，尽管它们的本地架构相对高效，但它们在大规模应用于高分辨率图像时仍然过于昂贵……[然而，其研究结果]表明，大型深度卷积神经网络能够使用纯监督学习在高度挑战性的数据集上实现破纪录的结果。”

明白了吗？

在收购 DNNresearch 之前，谷歌向 Hinton 及其研究团队提供了 60 万美元的礼物，以支持他们在神经网络方面的工作。继其行善理论之后，该公司承诺通过其“[重点研究奖励计划](https://web.archive.org/web/20230223090216/http://research.google.com/university/relations/focused_research_awards.html)”来“支持计算机科学和工程领域的雄心勃勃的研究”，该计划提供无限制的两到三年的资助，并让接受者获得谷歌的“工具、技术和专业知识”

因此，看起来谷歌通过其奖励计划发现了 DNNresearch，并看到该团队的工作可能对语音识别、语言处理和图像识别领域产生的影响——所有这些都是其核心产品的核心——决定一笔赠款是不够的。

多伦多大学校长大卫·奈洛尔在一份声明中说:“杰弗里·辛顿的研究是植根于基础研究的破坏性创新的一个伟大例子。”。杰出研究人员的发现，由他们的专业知识、好奇心和直觉自由引导，最终导致无人能想象的实际应用，更不用说被征用了。
