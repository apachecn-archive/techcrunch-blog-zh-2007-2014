# 玛丽莎·梅耶尔在 Le Web TechCrunch 谈论谷歌搜索、Chrome、在线新闻等

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/12/09/marissa-mayer-le-web-2009/>

今天下午，TechCrunch 的迈克·阿灵顿在法国巴黎举行的 [Le Web](https://web.archive.org/web/20230215200627/http://leweb.net/) 会议上采访了谷歌负责搜索产品和用户体验的副总裁[玛丽萨·梅尔](https://web.archive.org/web/20230215200627/http://www.crunchbase.com/person/marissa-mayer)。

去年，[阿林顿也在 Le Web 的舞台上采访了梅耶尔](https://web.archive.org/web/20230215200627/https://techcrunch.com/2008/12/10/marissa-mayer-at-le-web-the-almost-complete-interview/)，谷歌当天将 [Chrome 退出了 beta](https://web.archive.org/web/20230215200627/https://techcrunch.com/2008/12/11/chrome-shines-a-little-brighter-drops-the-beta-tag-with-new-release/) ，[谈到了 Searchwiki](https://web.archive.org/web/20230215200627/https://techcrunch.com/2008/12/10/google-search-wiki-to-soon-include-an-off-button-thank-you-marissa/) 。今年，没有发布任何公告，但梅耶尔分享了她对本周早些时候所有谷歌新闻的看法，包括 Chrome 浏览器的新版本和她对新闻未来的看法。

**恭喜[获得](https://web.archive.org/web/20230215200627/https://techcrunch.com/2009/11/06/marissa-mayer-gets-a-little-love-from-glamour-magazine/)魅力杂志年度女性大奖，蛮酷的。**

是的，我感到非常荣幸。

**先说新闻。谷歌本周发布了很多公告，你能概括一下吗？**

是的，当然。因此，我们举办了“搜索的未来”活动，并展示了许多令人兴奋的新想法:谷歌眼镜、语音搜索扩展、按位置搜索，以及关于我们实时搜索努力以及与脸书和 MySpace 合作的重大新闻。

一周内新闻不断。

是的，所有这些都围绕着四个关键部分:

–形态(如何搜索)
–媒体(更丰富、互动性更强、实时内容等。)
–语言(翻译)
–个性化搜索体验

去年，你在 Le Web 上发布了谷歌搜索维基。我从来没有喜欢过它，很高兴看到你集成了一个关闭按钮。但是你什么时候才能永远摆脱它呢？

我们认识到，我们需要改善体验，使其更加完整。

你是说杀了它？

没有。(笑)

我觉得 Goggles 很有意思，你知道，它能让谷歌通过视觉告诉我某人是什么东西或者是谁。我们离把这个概念变成主流还有多远？

Google Goggles 在一定程度上是索引，但它主要是结合了位置和 OCR 元素的图像识别。随着这些技术的发展，我们也会发展。在这一点上，语音到文本比图像和视频识别技术走得更远，所以我们现在在起跑线上。

**对了，谢谢你在后台给我看你的谷歌手机。**

(笑)我没有，那是我的 iPhone。你也知道我不能对猜测发表评论。

**让我们来谈谈移动搜索。这对谷歌有多重要，我的意思是，手机搜索占多大比例？**

我不能分享具体的数字，但它肯定在增长。这一比例正在上升，甚至在一年内翻了一番。

**转向谷歌 Chrome 和 ChromeOS，谷歌 Chrome 终于在本周早些时候发布了 Mac 版测试版。下一步是什么？**

我们昨天发布了一些关于 Chrome 浏览器和扩展的有趣公告，我们认为这将导致许多有趣的事情发生。对于浏览器，谷歌的重点是用户体验、加快浏览速度等等。至于 ChromeOS，我顺便说一下，它是一个反操作系统，我们离进入设备还有一段时间，但我们正在接近它。

你密切关注 Chrome 浏览器的下载量吗？有什么可以分享的吗？

我们不能透露具体数字，但我们说的是数千万用户。

**我是 Mac 用户，所以没怎么玩过。你一直在使用它吗？**

我使用所有的浏览器，速度对我来说非常重要。我喜欢 Safari，我用它是因为它在 MAC 上非常占优势，但 Chrome 太快了。

**让我们转到谷歌新闻。你如何看待关于新闻未来的整个骚动？**

显然，我们内部对此进行了一些头脑风暴，关于如何让用户更多地参与在线新闻，尝试付费内容模式和更有利可图的广告模式等等。你知道，新闻业的核心是不变的，但是它的传播方式发生了变化和发展。我们是这样想的:如果你要重新发明新闻和这个行业的商业模式，你会怎么做？我们不得不做出很多改变的结论。

我觉得有趣的是，如果你去维基百科，你可以在那里看到新闻，而不是在网上报纸网站上。但是只有一篇维基百科文章是关于一个给定的主题的，而你可以从不同的媒体上找到关于一个主题的多个角度。他们都在竞争位置。

我们也想了很多关于个性化新闻，我们可以使用什么指标。脸书和 Twitter 普及了流媒体，但为什么新闻没有像现在这样，更加个性化，跨平台和设备同步，甚至可以感知位置？将所有这些烘焙在一起，将会是一个非常引人注目的产品；我称之为超个性化新闻流。

**有意思。你认为像《纽约时报》这样的公司能够适应吗？**

我也这么认为我们已经和他们合作了一段时间，他们实际上是进步的思想家。他们真的想学习如何重塑自我。

**你会和默多克/新闻集团合作吗？**

嗯，我们和 MySpace 有合作关系…

**但你和鲁珀特·默多克的关系不同。**

是的。(微笑)

你认为新闻集团最终会从搜索引擎中撤出他们的内容吗？

我希望不会，因为我们想要全面。我们必须尊重版权所有者。如果我们不这样做，我们就没有最好的搜索引擎，我们也不会以质量为荣。

**您会考虑为内容付费吗，无论是独家还是非独家？**

很难说。我们已经在帮助出版商解决他们的一些盈利问题，只要看看我们由于谷歌 AdSense 向世界各地的出版商支付的 50 亿美元就知道了。还可以考虑免费为媒体公司提供某些见解的分析。我们提供可以帮助解决他们问题的技术解决方案。

**先说音乐搜索。在欧洲这里有吗？**

不，只有在美国，因为许可协议。

**这种观众有什么办法试用吗？**

这是基于 IP 的限制，所以答案是否定的。

我喜欢音乐搜索，不仅是因为搜索结果中的流媒体功能，还有整合的歌词搜索和购买选项。我经常听广播，不时地查找歌词，现在这项服务让我将两种体验结合起来

**音乐搜索的进一步计划？**

我们对此很高兴，但这绝对是一个开始。但是想想这个:歌词是谷歌上搜索次数第二多的词，音乐排在第九位。显然，用户需求很大。

**完全不同的事情:你在社交网络上玩一些 scammy 社交游戏吗，比如 Farmville？**

我知道他们，但我还没有玩过 Farmville。

到现在为止，你已经了解了这些企业的阴暗面。但是所有这些公司都在谷歌上做广告。你如何对抗他们的行为？

这肯定是一个难题，但我们在本周早些时候对诈骗广告客户提起了诉讼，所以我不能发表更多评论，因为我们正在诉讼中。

我发现自己越来越多地消费基于转发、朋友在脸书上分享的内容等。这是一个趋势，移动到内容的发现而不是搜索。作为搜索引擎提供商，你将如何利用这一现象？

好吧，看看我们今年宣布的两个版本:一个是社交搜索，允许人们搜索来自网络的公开结果和来自朋友的内容，这些内容在各种社交媒体上共享。第二个产品是实时搜索，我发现自己经常使用它。我大部分时间都相信网站上的内容，即使它们是公开的，不是来自朋友。我们显然需要这个领域的行业标准化，我们需要更多的 API，等等，但这是一个重要的概念。

实时搜索的一个大问题似乎是过滤、谣言的病毒式传播(无论真假)、聚众闹事等等。你认为你能解决这个问题吗？

很难说。我们不能简单地将 PageRank 算法应用于实时共享的内容，但我们可以观察生态系统，并检测我们可以用来揭示权威的信号。这很难，但是有数据点可以用来过滤。

**如果你能帮忙解决那个问题，那就太好了。**

同意，也因为当你找到真正好的源推文时，它们几乎是不可思议的。从突发新闻和谣言开始已经很有帮助了。

**让我们来谈谈 Google Wave。它引起了我们的注意，但有些困惑。是我们的错，还是产品需要更多的调整？**

像往常一样，我们很早就发布了，并且在发布过程中不断迭代。我们知道，在这一点上还缺少一些重要的特征。故事的另一部分是临界质量，因为当更多你认识或一起工作的人开始使用它时，价值就会增加。

**Google Wave 内部使用吗？**

每个队都不一样。

**最后一个问题:有谷歌手机，不是吗？否则你会断然否认，而不是给我扑克脸。**

(笑)
