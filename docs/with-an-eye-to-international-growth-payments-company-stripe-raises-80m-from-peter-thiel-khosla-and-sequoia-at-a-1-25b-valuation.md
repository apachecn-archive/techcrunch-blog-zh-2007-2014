# 着眼于国际增长，Stripe 以 17.5 亿美元的估值从 Founders Fund、Khosla 和 Sequoia 融资 8000 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/22/with-an-eye-to-international-growth-payments-company-stripe-raises-80m-from-peter-thiel-khosla-and-sequoia-at-a-1-25b-valuation/>

支付行业有了一家新的价值 10 亿美元的公司。在线支付公司 [Stripe](https://web.archive.org/web/20230404041554/https://stripe.com/) 宣布获得超过 8000 万美元的 C 轮融资，估值为 17.5 亿美元。彼得·泰尔的创始人基金领投，新投资者科斯拉(基思·拉布瓦)和现有投资者红杉资本加入，艾伦公司也提供了资金。这使得 Stripe 的总资金超过 1.3 亿美元。

要理解 Stripe 现在和过去为何具有颠覆性，最好回到几年前该公司背后的使命。2011 年，这家由帕特里克和约翰·科利森兄弟创立的公司明确阐述了它的使命:“我们认为，让网上交易成为可能是源于代码的问题，而不是财务问题，我们希望帮助更多的网站开展业务。”

虽然 PayPal 和其他公司当时存在，允许网站建立电子商务和支付，但这些产品笨重，对开发者不友好。此外，他们还没有适应新技术和用户互动。2010 年，科利森兄弟着手改变这种状况。他们从爱尔兰来到美国，很早就加入了 Y Combinator，当时他们的公司 [Auctomatic](https://web.archive.org/web/20230404041554/http://www.crunchbase.com/company/auctomatic) 在 2008 年被出售(他们当时只有十几岁)。

在竞争方面，Stripe 与 PayPal 针锋相对，PayPal 最近收购了 Braintree，也推动了它的处理能力。但是，来自五分之三的 PayPal 联合创始人和现在 PayPal 早期员工 Rabois 的投资信号表明，这个领域有多个参与者的机会，Stripe 有巨大的增长潜力，特别是在全球范围内。Patrick 向我们解释说，他看到了 PayPal 的一些重叠之处，但从根本上来说，这两家公司处理该领域问题的方式不同。PayPal 已经进入了线下世界，目前，Stripe 还没有朝这个方向前进。

科利森解释说，Stripe 的一个主要区别在于，它在公司内部创建了一个真正的平台。科利森兄弟认为 Stripe 与 AWS 在构建托管平台方面拥有相同的机会。约翰·科利森说，他们想创造一个围绕网络和移动支付的基础设施，就像 AWS 改变了人们建立网站的方式一样。这里的机会是进一步将支付 API 扩展到移动、结账、欺诈等领域。新资金的大量注入只会有助于这一愿景成为现实。

**产品**

Stripe 最初是 John 和 Patrick 在大学的一个兼职项目，但他们很快意识到让在线支付变得更容易有内在价值。他们的雄心不仅仅是促进在线支付，他们认为，如果支付更容易，更多的业务将在网上产生。有了来自贝宝创始人彼得·泰尔和埃隆·马斯克、红杉资本(迈克尔·莫里茨亲自参与)、安德森·霍洛维茨和 SV Angel 的种子资金，科利森兄弟着手为开发者打造一个“不错”的支付平台

2011 年，Stripe 推出了对开发者友好的在线支付系统，允许网站所有者避免设立商家账户和与银行交易，同时仍然确保交易安全。凭借对设计和简单性的关注，该平台在开发人员中一炮而红。在后端，Stripe 支持多种编程语言，可以在几分钟内完成部署；在前端，消费者的体验也很简单。该公司的 API 使商家能够轻松地大规模存储卡、启用订阅和直接向银行账户付款。

Stripe 还允许开发者构建自己的支付表单，品牌化体验，并允许网站所有者留在网站上体验结账体验，这最后一点非常重要，因为许多商家不希望用户必须访问外部网站才能结账，并可能在此过程中放弃交易。控制支付的用户界面和体验对任何商家来说都是一种胜利，因为它能够消除任何重定向。

**全球化的未来**

与支付领域的其他公司不同，这家初创公司没有围绕其处理量的数字浮动。但我们被告知，它每年有数十亿客户，包括 Lyft、Rackspace 和[以及许多其他](https://web.archive.org/web/20230404041554/https://stripe.com/gallery)。2012 年，该公司又筹集了 2000 万美元，这帮助它在旧金山开设了新的办公室，更积极地招聘员工，并开始国际扩张。C

在过去的两年里，该公司在国际上取得了爆炸性的增长。目前 Stripe 支持 12 个国家的支付，但有机会最终支持 200 个国家。仅在 2013 年，Stripe 就增加了对十个国家的支持，今天，它宣布对澳大利亚、比利时、芬兰、法国、德国、卢森堡、荷兰和西班牙进行公测。事实上，Stripe 在国际上的发展速度比在美国更快。新的资金将用于扩大国际市场的业务，并雇用国家负责人来管理每个市场。此外，Stripe 还将寻求在美国培养工程人员。目前，Stripe 约有 85 名员工，其中 22%是前创始人，近 40%来自美国以外

约翰·科利森告诉我们:“为网络创造这一层并使支付基础设施现代化是一项艰巨的任务。”。“这意味着我们必须扩展到更多的国家，并有可能超越信用卡支付。我们希望让支付在全球范围内为所有商家提供良好的服务。”科里森解释说，其中一部分将与当地银行合作，为每个市场整合支付选项。从运营的角度来看，Stripe 将在每个市场招聘国家经理，并从那里开始扩张。

就收入而言，我们被告知 2013 年是重要的一年，该公司表示，它看到了快速增长，但没有透露数字。帕特里克·科利森告诉我们，大约六个月前，Stripe 开始受到大公司的更多关注，尤其是那些财富 500 强的公司。据报道，该公司正在与 Twitter 进行谈判，以推动该网络的电子商务计划。科利森暗示将在 2014 年与更大的公司建立更多的合作关系。

至于今天宣布的融资，当科利森想要筹集新的资金时，他们会想到谁将是他们的梦之队，他们将提供运营和支付方面的专业知识。泰尔、拉博伊斯和迈克尔·莫里茨是一个令人印象深刻的团队，任何公司都可以扩大规模，并有可能创造出数十亿美元的公司。

> Stripe 的目标是增加互联网的 GDP。

拉布瓦作为一个新的投资者是一个特别有趣的合适人选，考虑到他在 Square、Slide 的成长经历，以及在当时羽翼未丰的初创公司 PayPal 工作的经历等等。拉布瓦在科里森夫妇创办 Stripe 时见过他们，虽然他没有投资，但他帮助安排了与富国银行的合作。他与兄弟们保持联系，当他离开广场加入科斯拉时，他开始更认真地与帕特里克和约翰交谈。事实上，他回忆说，在他与 Patrick 的第一次会面中，两人花了 3 个多小时谈论支付行业和 Stripe 的机会。

“Stripe 的机会是巨大的，尤其是因为移动设备的爆炸式增长，”Rabois 说。“我们只是处于移动革命的第一局和第二局，很难量化这一点，但看起来 Stripe 将变得不可思议。”

当然，机遇和成长伴随着挑战。拉布瓦说，Stripe 将选择执行哪些产品和创意，并将员工人数扩大到数百人。

Patrick 认识到，2014 年将是执行年，他和 John 的目标是实现 Stripe 在线支付基础设施 AWS。" Stripe 的目标是增加互联网的 GDP . "
