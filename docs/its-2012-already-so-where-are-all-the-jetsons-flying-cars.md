# 现在已经是 2012 年了，那么所有的杰特森飞行汽车都在哪里呢

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/01/01/its-2012-already-so-where-are-all-the-jetsons-flying-cars/>

当我们进入 2012 年时，我们现在不应该都在开着飞行汽车四处旅行吗？这是杰特森卡通电视节目中的预测。这部未来派系列电影由汉娜·巴贝拉于 1962 年首次制作，背景设定在 2062 年，距离今天正好 50 年。2012 年是中点，所以在我们驾驶飞行汽车之前，我们还有一些时间。但是，杰特森家族的许多其他技术进步已经伴随着我们。

杰森一家代表了对技术和未来的乐观看法。正如杰弗里·塔克写的那样，“这既不是乌托邦，也不是反乌托邦”，“技术是最好的(但当然它仍然会失灵，就像今天一样。)"

杰森一家是在一个今天几乎无法想象的技术时代创建的。电视仍然是部分黑白的。《杰森一家》是美国广播公司(ABC)在 T2 播出的第一个彩色节目。没有个人电脑，没有手机，当然，也没有互联网。但是，让我们看看到目前为止，杰特森对未来的一些设想实现得有多好。

**飞行汽车**:杰特森飞行汽车[的设计灵感来自于 1954 年的福特概念车，FX-Atmos，以其全玻璃气泡座舱盖、仪表盘雷达屏幕和喷气式飞机般的尾翼而闻名。我们还没有完全实现，但是由 Terrafugia 公司制造的名为 Transition 的汽车/飞机已经在两年前进行了首次飞行。](https://web.archive.org/web/20221213110541/http://www.imdb.com/title/tt0055683/trivia)

![](img/eae69fe86ec55f0cbfb2fedd03c86255.png "Image (1) terrafugia_formation_headline.jpg for post 79369")

已经有[几十辆其他的飞行汽车建造了](https://web.archive.org/web/20221213110541/http://en.wikipedia.org/wiki/Flying_car_(aircraft))，但大多数人都没见过头顶上有一辆。虽然乔治·杰特森实际上驾驶/驾驶他的汽车，但我们确实有[谷歌的自动驾驶汽车](https://web.archive.org/web/20221213110541/https://beta.techcrunch.com/2010/10/09/google-automated-cars/)。

**RUDI，乔治的“参考通用数字索引器”工作计算机**:我们有[雅虎](https://web.archive.org/web/20221213110541/http://www.crunchbase.com/company/yahoo)，“又一个等级森严的多管闲事的甲骨文”和数字索引[谷歌](https://web.archive.org/web/20221213110541/http://www.crunchbase.com/company/google)。我们也可以问问题，并从 Siri 得到答案。

视频聊天:乔治可以从这个屏幕上看到他的老板和家人，并与之交谈。我们有很多这样的应用，包括 Skype 视频通话、Google+视频聊天、思科网真和 T21 等等。

电视观众:在家里，乔治可以从这个屏幕上看新闻。在 20 世纪 60 年代，从屏幕上获取信息是一个奇怪的概念。现在，这是主要的方法。

乔治·杰特森的每周 3 天，每天 3 小时工作制:蒂姆·费里斯已经做到了这一点，并且在他的畅销书《每周 4 小时工作制》中做得更多。

乔治的工作是按按钮:在最初的 20 世纪 60 年代剧集中，乔治的工作包括按按钮、旋钮、刻度盘和开关。更多的剧集是在 20 世纪 80 年代制作的，乔治的办公桌[升级为](https://web.archive.org/web/20221213110541/http://en.wikipedia.org/wiki/The_Jetsons)平板按钮和灯光明亮的控制台。虽然不完全是一个控制台，但我们中有多少人整天都在键盘上按键？

**Spacely Space 链轮**:福布斯杂志[在他们的“25 家最大的虚构公司”名单上，开玩笑地将 Spacely 链轮估值为 13 亿美元。如今，苹果的市值约为 3760 亿美元。](https://web.archive.org/web/20221213110541/http://www.forbes.com/2007/12/10/largest-fictional-companies-oped-books-fict1507-cx_mn_de_1211company_slide_26.html?thisSpeed=15000)

**机器人吸尘器** : [iRobot 的](https://web.archive.org/web/20221213110541/http://www.crunchbase.com/company/irobot) Roomba 吸尘器。

家用机器人 Rosey:TechCrunch[在 2008 年写了一篇关于 ReadyBot](https://web.archive.org/web/20221213110541/https://beta.techcrunch.com/2008/07/14/readybot-r2d2-rosie-from-the-jetsons-roomba/) 的文章，这是一个可以捡起玩具和倒垃圾的机器人。与 [Anybots](https://web.archive.org/web/20221213110541/http://www.crunchbase.com/company/anybots) 机器人也有些许相似之处。

朱迪的数字日记 Didi:朱迪的日记是私人的，所以这个类比并不完美，但现在朱迪可能会在[脸书](https://web.archive.org/web/20221213110541/http://www.crunchbase.com/company/facebook)上分享她的生活信息。还有很多私人日记日记应用。

**移动人行道**:移动人行道在机场很常见，但不像在杰森一家的世界里那么普遍。但是，这个提议移动让车辆从下面通过的中国公交车在那个世界看起来就像在家一样。


史派利先生告诉乔治·杰特森“你被解雇了”:没必要解释我们都听过这句话。

杰森一家完全忽略的未来技术的一大部分是电子邮件和短信。也许到 2062 年，这些交流方式将不复存在。
