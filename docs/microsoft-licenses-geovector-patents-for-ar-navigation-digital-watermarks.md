# 微软授权地理矢量专利用于增强现实导航，数字水印

GeoVector 刚刚宣布与微软合作，将他们的专利 AR 技术引入 Windows 设备。该公司专门从事基于 AR 的导航，并拥有一种特殊的“建筑识别”软件，可以根据形状和大小识别网站。例如，正如你在他们的专利信息中看到的，你可以将手机对准首都，识别各种建筑，并用网址给它们贴上标签。

该公司目前有一个基于 AR 的 iPhone 应用程序，名为[世界冲浪](https://web.archive.org/web/20230203041529/http://itunes.apple.com/us/app/world-surfer/id329792332?mt=8)，可以让你拿起一个有指南针功能的手机，找到任何景点、商店和餐馆。

根据一份新闻稿，“微软公司(纳斯达克股票代码:MSFT)已与全球领先的增强现实和基于移动设备的定点本地搜索解决方案公司 GeoVector Corporation 达成许可协议。这项非排他性的多年协议赋予微软使用 GeoVector 专利所涵盖概念的权利。
