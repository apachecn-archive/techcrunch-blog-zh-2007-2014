# 二屏 App Yap。通过脸书开放图形集成，电视变得更加社会化

[狂言。TV](https://web.archive.org/web/20221004212018/http://www.yap.tv/) 于 2010 年末推出，旨在吸引越来越多的用户，他们打开 iPads 和手机收看电视。从那以后，它在一个相当拥挤的第二屏幕应用市场中保持了自己的地位。但是最新版本的 Yap。电视应用程序可以通过与脸书的深度整合来推动这一发展，这将允许用户看到他们的朋友正在观看的电视节目，并与其他有共同兴趣的用户联系。

一旦用户登录 Yap。电视通过脸书连接，应用程序导入他们所有的脸书电视“喜欢”和他们的朋友。然后，他们可以直接在应用程序中与朋友比较他们对各种电视节目的兴趣，并根据自己喜欢的节目时间创建一个收藏夹列表。他们可以看到不同内容得到了多少评论和分享，也可以分享、评论或喜欢它们。

该应用程序的主要新功能之一是在应用程序的主级别提供的“feed ”,它为用户提供与他们在脸书喜欢的节目相关的个性化内容流。它也被添加到 yap 中。电视节目页面显示应用程序内的相关内容。

另一个大的新变化是能够看到哪些朋友目前在线，无论是在 yap 上。电视应用程序，或在脸书，由于新的脸书检测功能。他们还可以直接在应用程序中与这些朋友聊天。

当然，Yap。电视仍然有很多竞争，来自像 Miso，GetGlue 这样的应用程序，以及其他一些应用程序。此外，还有前 BBC iPlayer 首席技术官安东尼·罗斯(Anthony Rose)的 Zeebox，它甚至还没有在美国推出。现在宣布赢家还为时过早，但这只是意味着社交电视和第二屏幕应用仍有很大的调整空间。
