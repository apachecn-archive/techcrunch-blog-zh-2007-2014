更新 : TBS现已确认收购Bleacher Report。该部门将成为特纳体育部门的一部分。该集团目前代表NBA、NCAA和PGA管理数字财产，并监督NASCAR.COM的广告销售以及与雅虎的战略销售关系体育。它有8600万独立访客。该交易的价格尚未披露，但据信在2亿美元左右，据华尔街日报报道，在1.75亿美元左右。

“Bleacher Report是一项战略收购，进一步增强了特纳的体育产品组合，也反映了我们全年为粉丝提供更多内容的持续承诺，”特纳广播公司销售、分销和体育总裁大卫·利维在一份声明中说。“作为品牌建设者和内容提供商，我们被Bleacher Report的快速增长吸引到了领先的市场地位和有价值的消费目的地。该网站将继续创新，为用户和体育迷提供品牌新闻和信息。”

下面是关于这笔交易的更多背景的原始故事。

美国联邦贸易委员会已经批准了 Bleacher Report 和时代华纳之间的交易——根据来自彭博和AllThingsD的报道，这是时代华纳部门特纳广播系统正在完成对体育网站网络的收购的信号。

据报道，价格在2亿美元左右:彭博表示，这笔交易不到2亿美元。 SAI 纸币的价格为1.75亿美元。 AllThingsD 指出，Bleacher报告称投资者正在寻找2亿美元以上的退出机会。Bleacher Report已经从包括Crosslink Capital、Oak Investment Partners和Hillsven Capital在内的投资者那里筹集了4050万美元。这笔交易至少价值6820万美元，这是引发联邦贸易委员会调查的门槛。

联邦贸易委员会不仅调查并购，还调查投资。从周五开始，Bleacher Report和时代华纳出现在同一个目录上，通用大西洋伙伴公司和Box也是如此。7月底，云公司宣布了由风投公司牵头的一轮1.25亿美元的融资。

时代华纳对《露天看台报告》感兴趣是因为这家广播公司已经在今年早些时候将对《体育画报》网站和高尔夫门户网站的控制权移交给了时代公司——尽管这两家网站本身仍以特纳广播公司的名义运营。这可能意味着，如果这是一项收购，并通过了，这些网站将继续保持独立，至少部分。

彭博写道，成立于2006年的Bleacher Report每月有大约1000万用户，它拥有令人震惊的6000名撰稿人，其中大多数人免费写作。它是由四个朋友发起的，大卫·菲诺基奥(现在是内容&产品副总裁)戴夫·奈梅兹(业务发展副总裁)布莱恩·戈德堡(收入副总裁)和赞德·弗罗因德，他自称是《漂白报告》会员的原始“面孔”，是社区总经理。他已经离开了公司，现在是其他创业公司的顾问，同时仍然是Bleacher Report的股东。布莱恩·格雷现任首席执行官。

发布时间:2012年8月6日

特纳广播系统公司收购看台报告
领先的体育数字财产加入特纳体育投资组合

特纳广播系统公司收购了Bleacher Report (B/R)，这是互联网上增长最快的体育数字资产之一，特纳广播公司销售、分销和体育总裁David Levy今天宣布。总部位于旧金山的B/R(http://www.bleacherreport.com)及其广受欢迎的移动产品现在将作为特纳体育部门的一部分运营。

B/R提供原创和有趣的体育编辑内容，是领先的体育媒体目的地，每月有超过1000万的独立访问者。该公司开发了下一代出版模式，由不断扩大的热情和知识丰富的贡献者群体提供支持，这些贡献者在团队和主题层面上提供读者渴望的高质量内容。

“Bleacher Report是一项战略性收购，它进一步增强了特纳的体育产品组合，并反映了我们在全年为粉丝提供更多内容的持续承诺，”Levy说。“作为品牌建设者和内容提供商，我们被Bleacher Report的快速增长吸引到了领先的市场地位和有价值的消费目的地。该网站将继续创新，为用户和体育迷提供品牌新闻和信息。凭借我们广泛的数字版权和资源，特纳将进一步确保Bleacher Report的持续增长和成功。”

Levy补充说:“收购Bleacher Report后，我们的数字和移动资产组合又增加了一个维度，这将使我们能够为我们的广告合作伙伴提供跨所有屏幕和演示的集成机会，从而提高我们全年从体育节目中赚钱的能力。”

“Bleacher Report的创建是为了将粉丝放在第一位，我们很高兴能够将我们的创新内容发布模式和多平台消费者数字媒体体验融入特纳体育的世界级电视和数字内容产品组合，”Brian Grey说，他最近担任了新扩大的角色，即Bleacher Report首席执行官兼特纳体育高级副总裁该组合将共同定义内容创作者、消费者和营销人员如何围绕体育团队走到一起，以及粉丝将在每个屏幕上消费的主题内容体验。"

特纳体育目前代表NBA、NCAA和PGA管理数字财产，监督NASCAR.COM的广告销售，并与雅虎有战略销售关系体育。贝加莱将补充特纳数字产品组合中卓越的品牌系列，涵盖娱乐、体育、儿童、新闻和喜剧目的地，涵盖宽带、移动和平板电脑。特纳数字产品组合目前每月有8600万独立访客。

除了其数字产品组合，特纳体育还在TBS、TNT和truTV上全年转播多项赛事，包括NBA、MLB、NASCAR、PGA锦标赛和NCAA级男子篮球锦标赛。它播出的播音员和分析师名单包括马夫·阿尔伯特、厄尼·约翰逊、查尔斯·巴克利、肯尼·史密斯、雷吉·米勒、史蒂夫·科尔、沙奎尔·奥尼尔、卡尔·里普肯、丹尼斯·埃克斯雷、约翰·斯莫茨、罗恩·达林、大卫·威尔斯和凯尔·佩蒂等。

Grey将继续在旧金山办公室监督B/R的日常运营，并直接向特纳体育执行副总裁兼首席运营官Lenny Daniels报告。Grey和特纳体育的高级副总裁兼运营总经理Matt Hong将共同合作，对《看台报告》的运营进行全面管理，并将其整合到特纳体育的投资组合中。

Dave Finocchio，《看台报告》创始人兼首席内容和产品官，特纳体育副总裁；特纳体育的《露天看台报告》首席技术官兼副总裁Sam Parnell和《露天看台报告》首席收入官兼特纳数字广告销售高级副总裁Rich Calacci都将继续担任扩大后的职务。

在这笔交易中，特纳广播公司由威尔基·法尔和加拉格尔·LLP担任顾问，漂白报告由加拿大皇家银行资本市场和莱瑟姆&沃特金斯·LLP担任顾问。

关于Bleacher Report
Bleacher Report(B/R)是互联网上原创和娱乐体育内容的领先出版商，也是美国发展最快的体育媒体网站之一。自2008年推出以来，B/R的网站已发展到每月拥有超过1000万独立访问者。该公司杰出的编辑团队领导着2000多名核心撰稿人，并指导Bleacher Report独特的数据驱动方法来创建和编程内容。其结果是一流的体育评论，为B/R的观众提供他们想要实时阅读的关于球队和他们最感兴趣的话题的故事。B/R的编辑内容还通过辛迪加发布给许多国家和地区出版合作伙伴，包括《今日美国》、《洛杉矶时报》、《巴尔的摩太阳报》、《Philly.com》、《旧金山纪事报》、《休斯顿纪事报》和《西雅图邮报》。此外，超过200万名球迷订阅了B/R的体育和团队电子邮件简讯，超过100万名用户安装了B/R的领先团队流 移动和平板电脑应用程序，这些应用程序提供了全面的内容组合，在团队和主题层面吸引了广泛的体育球迷。

关于特纳体育
特纳体育是特纳广播系统公司的一个部门，是电视和在线体育节目的行业领导者，在TBS、TNT和truTV上播放冠军级体育赛事，并管理互联网上一些最受欢迎的体育网站。特纳体育的电视节目阵容包括NBA、美国职业棒球大联盟、NCAA级男子篮球锦标赛、NASCAR和职业高尔夫。该公司的数字产品组合包括NASCAR.COM、NCAA.com和三月疯狂直播、PGATOUR.COM和PGA.com，以及配套的移动网站和联网设备应用程序。特纳体育和NBA还共同管理NBA Digital，包括NBA TV、NBA.com、NBA LEAGUE PASS、NBA Mobile、NBA Game Time应用程序、NBADLEAGUE.com和WNBA.com。

时代华纳旗下的特纳广播系统公司制作和播放品牌新闻；娱乐；动画和年轻人；以及电视和其他平台上的体育媒体环境。
