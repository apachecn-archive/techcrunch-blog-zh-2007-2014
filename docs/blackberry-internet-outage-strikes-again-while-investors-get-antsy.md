# 黑莓网络中断再次罢工，投资者坐立不安

哦天啊。似乎昨天的 BIS 中断对 RIM 来说还不够糟糕，看起来他们想出的任何解决方案都不太管用。又一次停电的报道开始流传，欧洲、非洲和中东的用户再次无法浏览网页、发送电子邮件或发送 BBM 消息。

RIM 已经通过他们的官方 [BlackBerryHelp](https://web.archive.org/web/20230205030006/https://twitter.com/#!/BlackBerryHelp) Twitter 账户承认了这个问题，称“一些地区存在信息延迟和浏览障碍。”他们的推文听起来并不太糟糕，但是来自 [T-Mobile 英国](https://web.archive.org/web/20230205030006/https://twitter.com/#%21/TMobileUKhelp/status/123750850437971969)、[沃达丰埃及](https://web.archive.org/web/20230205030006/https://twitter.com/#%21/VodafoneEgypt/status/123750511374630912)、巴林运营商 [Batelco](https://web.archive.org/web/20230205030006/https://twitter.com/#!/Batelco/status/123731599236927489) 和肯尼亚运营商 [Safaricom](https://web.archive.org/web/20230205030006/https://twitter.com/#!/bobcollymore/status/123734744872660993) 的代表都证实他们的客户受到了影响。

虽然 RIM 在努力解决问题，但他们也可能面临一些离家更近的问题。激进投资者 Jaguar Financial 呼吁 RIM 实施一些重大变革已经有一个月了，但是路透社报道称该公司已经成功获得了其他黑莓股东的支持。目前，捷豹声称有足够多的投资者支持，占 RIM 总股票的 8%。

捷豹首席执行官 Vic Alboini 提到，在这些股东的支持下，捷豹将能够要求召开股东会议，以获得更多的动力来实现他们对 RIM 的愿景。这个愿景到底是什么？捷豹希望 RIM 经历一个“价值最大化”的过程，这可能涉及出售 RIM 的专利，甚至整个公司。此外，捷豹呼吁对 RIM 管理层进行改组，首先是罢免联席首席执行官迈克拉扎里迪斯(Mike Lazaridis)和吉姆巴尔西利(Jim Balsillie)。

在这一点上，对 RIM 来说，把他们的事情处理好已经非常重要了，因为事情开始变得严峻起来。RIM 扭转局面当然不是不可能，但这些天来，他们的道路似乎越来越艰难。
