# Fandango 开始推出移动电影票，让电影观众无纸化

![](img/b44d5944479b6e9d9c9ac92e729e5e2a.png)

排队买电影票仍然是去看电影最糟糕的部分(除非你要去看*赏金猎人*)。有这么多手机电影应用程序，很容易找到附近影院正在上映的电影，甚至可以直接从手机上购买门票，但你仍然需要从自动售货机或票务代理那里获得纸质门票。但是你的票可以很容易地通过 2D 条形码发送到你的手机上。

今天， [Fandango](https://web.archive.org/web/20221207075013/http://www.fandango.com/) 在八个城市推出了移动售票计划，让电影观众最终实现无纸化。您的车票会通过短信或彩信发送到您的手机上，短信或彩信与 2D 条形码相连，检票员可以扫描该条形码。电影院需要为他们的观众配备特殊的扫描仪，这就是为什么它只在少数市场提供。(MovieTickets.com 正在[测试](https://web.archive.org/web/20221207075013/http://www.filmjournal.com/filmjournal/content_display/news-and-features/news/technology-and-new-products/e3ia62d19a7acb838dbd19805c7a34239cb)一个类似的程序)。

以下是参与 Fandango 首次推出的影院:

*   纽约:城市电影院 1、2 和 3，安吉丽卡电影中心，东 86 街电影院，村庄东电影院，比克曼剧院，巴黎剧院。
*   新泽西:曼维尔 12 层。
*   休斯顿:安吉丽卡电影中心。
*   达拉斯/普莱诺:安吉利卡达拉斯；安吉利卡·普莱诺。
*   圣地亚哥:拉梅萨格罗斯蒙特中心，克莱尔蒙特镇广场体育场。
*   贝克斯菲尔德:山谷广场 16 号。
*   索诺马县:罗纳特公园 16 号。
*   夏威夷:沃德体育场、卡哈拉剧院、卡波雷 16、米利拉尼体育场。
