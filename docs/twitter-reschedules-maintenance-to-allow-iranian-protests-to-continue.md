# Twitter 重新安排维护时间，允许伊朗人继续抗议

Twitter 计划今晚停机 90 分钟进行维护。考虑到现在伊朗发生的事情，这是个问题。所以推特决定[重新安排](https://web.archive.org/web/20230403221048/http://blog.twitter.com/2009/06/down-time-rescheduled.html)的维护时间，这样抗议就可以继续了。

这是 Twitter 的一个好举措。它显然不希望不得不移动它称之为“关键网络升级”的维护窗口，但在趋势话题列表上停留了一整天的 [#nomaintenance](https://web.archive.org/web/20230403221048/http://twitter.com/timeline/search?q=%23nomaintenance) 标签非常清楚地表明，Twitter 的用户根本不希望服务在这个重要时刻宕机。因此，Twitter 与其网络合作伙伴 NTT 美洲公司合作，将维护时间重新安排在太平洋时间下午 2-3 点，也就是伊朗的凌晨 1:30，而不是白天。

Twitter 用这篇文章的其余部分来赞扬 NTT 美国的灵活性，但实际上这都是关于伊朗的。那里的人们使用 Twitter 作为传播正在发生的事情的首选工具，即使其他交流渠道被封锁。
