# Zendesk 加速了 Twitter 的整合；允许用户将推文转化为服务台票证

随着越来越多的消费者转向 Twitter 表达他们对品牌和客户服务的不满，公司需要密切关注社交网络，以监控和回应这些对话和投诉。服务台初创公司 [Zendesk](https://web.archive.org/web/20221208035206/http://www.zendesk.com/) 今天正在加强其 Twitter 整合，允许客户在其客户服务平台内利用 Twitter 上的社交互动。

Zendesk 本质上提供了一个基于网络的帮助台在线票务系统，用于客户支持。新的集成使用户能够将推文转化为 Zendesk 门票。你也可以在 Twitter 上公开回应来自 Zendesk 平台的投诉。因此，你可以记录任何发生在 Zendesk 平台上的 Twitter 对话，也可以将对话从 Twitter 转移到电子邮件中。

该平台集成了社交媒体监控工具，如 HootSuite、TweetDeck、Twitter.com 以及 Twitter iPhone 和 Android 应用程序。考虑到 Twitter 现在已经成为消费者谈论品牌和公司的集中场所，这个新功能是有意义的。当然，其他服务台平台，如 Salesforce 的[服务云也具有 Twitter](https://web.archive.org/web/20221208035206/https://beta.techcrunch.com/2009/09/08/everything-you-need-to-know-about-salesforces-service-cloud-2/) 集成功能。

刚刚突破 5000 客户大关的 Zendesk 最近提高了价格，这引起了客户的强烈不满。Zendesk 回复到之前的定价模式来应对反弹。
