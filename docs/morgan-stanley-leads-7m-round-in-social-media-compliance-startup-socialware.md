# 摩根士丹利领投社交媒体合规初创公司 Socialware 万美元

帮助公司管理社交媒体和销售合规性的社交软件(Socialware)已经在由摩根士丹利扩张资本(Morgan Stanley Expansion Capital)牵头的 C 轮融资中筹集了 700 万美元(T3)。今年早些时候，初创公司[从](https://web.archive.org/web/20230205024035/https://techcrunch.com/2011/02/09/socialware-lands-3-million-to-put-social-networks-to-work/) [FLOODGATE](https://web.archive.org/web/20230205024035/http://www.crunchbase.com/financial-organization/floodgate) 、 [G-51 Capital](https://web.archive.org/web/20230205024035/http://www.crunchbase.com/financial-organization/g-51-capital) 和 [Silverton Partners 筹集了 300 万美元](https://web.archive.org/web/20230205024035/http://www.crunchbase.com/financial-organization/silverton-partners)。迄今为止，Socialware 已经筹集了 1200 万美元。

Socialware 提供技术和社交网络合规性工具，使高度监管的金融服务行业能够通过社交网络在稳固的框架内与客户和合作伙伴互动。该公司还在脸书、LinkedIn 和 Twitter 等社交销售渠道中提供挖掘潜在客户的工具。

该公司表示，目前它为近 100 家客户提供服务，其中包括美国五家顶级保险公司以及其他主要金融品牌。目前的客户包括美国投资组合、剑桥投资研究、嘉德人寿和纽约人寿。
