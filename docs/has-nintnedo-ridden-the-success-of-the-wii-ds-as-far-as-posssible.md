# 任天堂有没有尽可能地骑过 Wii，DS 的成功？

![upnintendo](img/e5e88bbbca79374d3da67f2b60e12e9a.png "upnintendo")

任天堂终于耗尽了 Wii 和 DS 的所有动力了吗？德意志银行似乎是这样认为的，今天早些时候，该行将其对日本超级快乐公司的评级下调至“卖出”。逻辑很简单:对任天堂来说，这很可能是最好的结果，所以投资者最好现在就套现，并收集他们在过去几年中积累的任何利润。

至少财务人员会这么告诉你，而他们关心的只是利润和增长，以及你最有可能在美国消费者新闻与商业频道听到的诸如此类的词汇。

事实上，该公司的股票今天一直呈下跌趋势。

就像我过去说过的，当涉及到所有这些关于我们玩的游戏的金融故事时，看，只要你对你的系统和游戏满意，就没有必要为附加率、按地区划分的销售额等争论不休。尽情享受吧。
