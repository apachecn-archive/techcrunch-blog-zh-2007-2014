# 不要在网上使用那个越狱的 PS3

你可能想推迟破解你的 PS3 的时间，尤其是如果你花很多时间在线的话。原来索尼可以检测出用于越狱的程序所使用的软件 ID，然后远程封禁你的 PS3。有什么解决办法？要么不要在 Playstation 网络上使用你被黑的 PS3，要么一开始就不要在上面使用黑的。[TG daily](https://web.archive.org/web/20221208112141/http://www.tgdaily.com/games-and-entertainment-features/51210-can-sony-detect-jailbroken-ps3s)报道称，一些制造该黑客事件的人警告人们不要在运行该软件时上网。当然，你可能不需要上网，因为你可以运行 PS3 软件的备份，但是考虑一下这个合理的警告。
