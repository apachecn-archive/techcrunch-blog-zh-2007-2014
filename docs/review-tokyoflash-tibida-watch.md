# 回顾:Tokyoflash Tibida 手表

[![](img/761df2f390d72d214b6076889193ba55.png "scaleddhqpzj6f_23q93j7z4z_b")](https://web.archive.org/web/20221007094202/https://beta.techcrunch.com/wp-content/uploads/2008/11/scaleddhqpzj6f_23q93j7z4z_b.jpg) 
一直对[东京闪](https://web.archive.org/web/20221007094202/http://tokyoflash.com/)情有独钟，但一直都是隔空相望。她是如此迷人，但又如此令人费解。我既敬畏又迷惑。我们终于和新的 Tibida 在一起了。最棒的是她在我手臂上的样子，光滑而黝黑。当人们看到她时，都是“哇”、“酷”和“她花了你多少钱？”她很聪明，比我想象的要聪明得多。盯着她看，你会情不自禁地被她蓝白色的光芒所迷惑。
 现在有了蒂必达，我开始理解她了。我们大多用分钟模式交流。她在上面用圆点显示小时，但在底部用她的 30 个 LED 显示屏用数字拼出分钟。

有几次，我们进入了小时模式，这个模式用数字显示小时，每过 5 分钟在顶部显示一个点(如上所示，大约 2:35)。二进制模式还是像外语一样。我正在学习。总有一天我会准备好的。也许不是今天。也许不是明天，但总有一天，很快。
