# 热衷于…大卫·艾伦:如何在美国把事情做好(TCTV)

我整个星期都想写这篇文章。但是你知道事情是怎样的——总有更好的事情要做，总有更有趣的事情，总有一个或另一个借口来推迟。如果你像我一样，努力把事情做好，那么大卫·艾伦可能就是你要找的人。他是自助界的[先生](https://web.archive.org/web/20230203095842/http://www.davidco.com/)，一位拥有 130 万[推特粉丝的效率大师](https://web.archive.org/web/20230203095842/http://twitter.com/#!/GTDGUY)，以他 2002 年的经典著作[成事](https://web.archive.org/web/20230203095842/http://www.amazon.com/Getting-Things-Done-Stress-Free-Productivity/dp/0142000280)而闻名，该书至今仍是畅销书。

现在，美国需要大卫·艾伦。所以当他走进 TechCrunchTV 演播室时，我问他美国，尤其是华盛顿 DC，如何才能学会把事情做好。他所说的话——关于如何让我们的政治和个人生活更有效率——可能会让你吃惊。如果你今天设法完成了一件事，那应该是看大卫·艾伦。
