# Kinect SDK 将于 2012 年初发布(我们回顾了过去的黑客攻击)

【YouTube = http://www . YouTube . com/watch？v=T_QLguHvACs]

紧随其[设想的未来视频](https://web.archive.org/web/20230203094117/https://techcrunch.com/2011/10/27/microsofts-vision-of-the-future-includes-touch-sensitive-everything-and-beautiful-people-only/)之后，微软在 YouTube 上发布了一个鼓舞人心的 Kinect 图像。该视频概述了 Kinect 在其一年的历史中有些意想不到的用途，包括教育和医疗应用，这可能是为了在 Kinect SDK 的商业发布之前提高人们的认识。

自 Kinect 首次推出仅几天后，开发人员就一直在入侵该平台，并为其创造新的用途和应用，但这标志着公司首次能够从他们的 Kinect 创造中获利。据 [CNET](https://web.archive.org/web/20230203094117/http://news.cnet.com/8301-13506_3-20127797-17/early-2012-launch-for-kinect-for-windows-commercial-sdk/?part=rss&subj=news&tag=2547-1_3-0-20) 称，Kinect SDK 将于 2012 年初向任何准备好并愿意的人开放。

为了庆祝这个重要的时刻，我们认为有必要回顾一下开发人员在 Kinect 上做了些什么。希望你会像我们一样喜欢这种对过去的看法。
