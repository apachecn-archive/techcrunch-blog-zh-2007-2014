# 谷歌 Chrome 现在内置了 Flash

![](img/d7cbc5726154e1040d75214443d3a386.png)

去年三月， [Adobe](https://web.archive.org/web/20221231221354/http://www.crunchbase.com/company/adobe-systems) 和 [Google](https://web.archive.org/web/20221231221354/http://www.crunchbase.com/company/google) 联合宣布 Flash Player 将很快[内置于后者的](https://web.archive.org/web/20221231221354/https://techcrunch.com/2010/03/30/flash-player-to-come-bundled-with-google-chrome-new-browser-plugin-api-coming/) [Chrome](https://web.archive.org/web/20221231221354/http://www.crunchbase.com/product/chrome) 浏览器中，用户无需单独下载、安装和更新。

周四晚上，谷歌向 Linux、Mac 和 Windows 上的稳定频道发布了 Chrome 5.0.375.86 ，修复了一些安全问题。更重要的是，集成的 Flash 播放器现在已经默认启用。

正如 CNET 的 Stephen Shankland 指出的那样，内置 Flash 以前只在基于 WebKit 的 speedy 浏览器的开发者和测试版本中可用，而稳定渠道的发布意味着集成插件现在在其主流版本中可用。

谷歌在禁用集成 Flash Player 插件一段时间后，在[测试频道](https://web.archive.org/web/20221231221354/http://googlechromereleases.blogspot.com/2010/06/beta-channel-update_23.html)默认重新启用了该插件，仅两天后就发布了更新。

谷歌不仅对 Adobe 的 Flash 技术投下了又一张信任票([本周早些时候](https://web.archive.org/web/20221231221354/http://www.adobe.com/aboutadobe/pressroom/pressreleases/201006/06222010FlashPlayerAvailability.html)[宣布的](https://web.archive.org/web/20221231221354/https://techcrunch.com/2010/06/22/flash-player-mobile/)移动版 Flash Player 10.1 将首先在 Android 2.2 手机上推出)，而且这一整合还意味着 Flash Player 的任何更新都将直接通过谷歌 Chrome 的更新系统交付，最终将使用过时软件和组件时可能出现的安全风险降至最低。
