# 迪士尼-美国广播公司与亚马逊和网飞签署流媒体协议

给[亚马逊](https://web.archive.org/web/20230203081759/http://phx.corporate-ir.net/phoenix.zhtml?c=176060&p=irol-newsArticle&ID=1623496&highlight=)…给[网飞](https://web.archive.org/web/20230203081759/http://www.prnewswire.com/news-releases/netflix-and-disney-abc-television-group-announce-extension-of-existing-licensing-agreement-to-stream-tv-shows-132899323.html)加一个。

这两家公司刚刚宣布了与华特·迪士尼公司的单独流媒体协议，这将为订户提供更多来自米奇公司的内容。根据这些协议，两家流媒体服务将看到更多迪士尼/美国广播公司的内容，扩大各自当前和过去的电视节目库。

与网飞的协议允许该公司继续获得数百集。它还增加了《出生时被调换》、《踢屁股的人》前一季的剧集以及《别名》的全部内容。如果这笔交易未能签署，像《迷失》、《兄弟姐妹》、《实习医生格蕾》这样的电影将会从网飞的电影阵容中消失，这可能会签署公司的死亡通知。光是负面新闻就可能做到这一点。

亚马逊和迪士尼签署的协议为其主要即时视频阵容增加了 800 个产品。内容来源于美国广播公司工作室、迪士尼频道、美国广播公司家庭和漫威，包括《迷失》、《实习医生格蕾》、《菲尼亚斯与费伯》和漫威的《x 战警进化》。这笔交易是在与福克斯和哥伦比亚广播公司的一笔重大交易几周后达成的。这些额外的内容将亚马逊的即时流媒体视频库放在了 13，000 部电视节目和电影中，亚马逊今天在首页自豪地宣布了这一事实。

这两笔交易的财务条款都没有公布。

这些流媒体交易正变得越来越重要，因为两家公司都试图加强其图书馆。处于自由落体状态的网飞(T4)需要这些协议来留住用户，而亚马逊正试图踢出一个立足点。亚马逊的图书馆仍然比不上网飞的图书馆，但最近发展迅速，似乎正在快速签约内容提供商。两家公司目前正处于激烈的赛跑中，但这不是短跑。
