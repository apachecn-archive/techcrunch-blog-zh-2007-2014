# 谷歌对搜索做出重大界面改变:搜索维基

![](img/e3d4d3c77dbc6abcd1c554b79856bc17.png)
![](img/d3e5c16a697ba91b7ffa25cd01a7cd62.png)

我们注意到越来越多的人在进行大规模的测试(只对总用户的一定比例进行产品变更测试)时发邮件，谷歌已经[进行了](https://web.archive.org/web/20221208221046/http://www.beta.techcrunch.com/2008/07/14/google-bucket-testing-new-digg-like-search-interface/)好几个月——在搜索结果(也改变了排名)和用户评论中加入了类似 Digg 的投票功能。

今晚，谷歌显然说了“搞什么鬼”，并为每个人打开了[。](https://web.archive.org/web/20221208221046/http://googleblog.blogspot.com/2008/11/searchwiki-make-search-your-own.html)

这些变化被称为搜索维基，与谷歌简化的算法规则搜索方法有很大的不同。它利用 Digg 的功能允许用户对网站结果进行投票，还利用 [Wikia Search](https://web.archive.org/web/20221208221046/http://www.beta.techcrunch.com/2008/06/03/jimmy-wales-wikia-search-finally-doesnt-suck/) 的功能允许用户添加评论、移动搜索结果、添加搜索结果等。结果是定制的结果，在您以后每次进行搜索时都会出现(假设您已经登录)。
