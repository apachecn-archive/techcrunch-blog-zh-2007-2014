# 雕塑 123D iPad 应用程序让雕塑再次变得酷起来

我不知道如果米开朗基罗看到这个应用程序，他是会在坟墓里翻身还是会高兴得跳起来。话说回来，米开朗基罗可能会因为看到一个灯泡而晕倒，所以谁知道他会对平板电脑应用程序做出什么反应。在任何情况下，Autodesk 的 Sculpt 123D [iPad](https://web.archive.org/web/20230204201846/https://techcrunch.com/tag/ipad) 应用程序看起来都是一种非常酷的方式，可以用一种我们大多数人都没有体验过的艺术形式来弄脏自己的手:雕刻。

你从一个基本的形状开始，像一辆车或一个人，然后开始使用不同的工具来“雕刻”成你想要的形状。该应用程序还具有印章，你可以调整并添加到雕塑上。如果有你在 Sculpt 123D 库中找不到的东西，Autodesk 也有。这款应用可以让你用 iPad 拍照，然后在雕塑上刻下图像。所以从理论上来说，你可以为一个朋友或者你自己(如果你有自恋的情绪)创造一个非常逼真的雕塑。

假设即使你已经拍过照片，浏览过形状库十亿次，你仍然找不到这个非常重要的雕塑项目所需要的东西。嗯，Autodesk 有五个额外的形状包，可以在应用程序中购买。项目完成后，你可以通过脸书、Flickr、电子邮件或 YouTube 分享这个杰作的截图。“YouTube？”你问。“我为什么要上传一张静止图像到 YouTube？”因为 Sculpt 123D 可以让你制作最终产品的转盘视频。

这款应用目前仅适用于 iPad，并且在苹果[应用商店](https://web.archive.org/web/20230204201846/http://itunes.apple.com/app/123d-sculpt/id446119510?mt=8)限时免费。每当“有限时间”结束时，该应用程序将花费 0.99 美元。跳过休息时间，查看 Sculpt 123D 演示视频。
