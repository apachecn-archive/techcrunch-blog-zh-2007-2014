# Thermaltake BlacX 5G 是一款 USB 3.0 硬盘扩展坞

[![](img/0359869810ffb13ff16a2cf2ef7d2b71.png "201062110581157814")](https://web.archive.org/web/20221208063734/https://beta.techcrunch.com/wp-content/uploads/2010/06/201062110581157814.jpg) 
Thermaltake 最新的硬盘扩展坞可能看起来像以前的型号，但在后面是蓝色的 [USB 3.0](https://web.archive.org/web/20221208063734/http://www.crunchgear.com/tag/usb-30/) 端口。这意味着用户可以将硬盘放入坞站，以高达 5Gpbs 的速度传输文件。太神奇了。

BlacX 5G 在 OS X 和 Windows 上都支持高达 2TB 的硬盘，售价仅为 49.99 美元。如果您的 rig 缺少 USB 3.0，请不要担心，dock 也可以与 USB 2.0 和 1.1 兼容。
