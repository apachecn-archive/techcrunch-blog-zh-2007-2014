# Andreessen Horowitz 领投智能手机安全公司 Lookout TechCrunch 4000 万美元

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/21/andreessen-horowitz-leads-40m-round-in-smartphone-security-company-lookout/>

[Lookout，](https://web.archive.org/web/20230205004030/https://www.mylookout.com/)一家在[为许多智能手机提供安全服务的公司](https://web.archive.org/web/20230205004030/https://techcrunch.com/2010/05/18/lookout-lands-11-million-from-accel-khosla-and-others-for-smartphone-security-platform/)，已经筹集了 4000 万美元的新资金，由 Andreessen Horowitz 牵头。目前的投资者 Khosla Ventures、Accel Partners 和 Index Ventures 也参与了这轮投资。安德森霍洛维茨基金的杰夫乔丹将加入该公司的董事会。这使得 Lookout 的资金总额达到 7500 万美元。

事实是，随着越来越多的消费者采用智能手机(根据 Gartner 的数据，到 2015 年，这一数字将达到数十亿)，这些设备面临着移动应用恶意软件、网络钓鱼诈骗、驾车下载和移动浏览器上的恶意网站的风险。与消费者保护电脑的方式类似，也需要一个保护智能手机的应用程序。Lookout 基本上凭借简单易用的产品占据了这个市场。

作为背景，Lookout 为 Android、Windows Mobile 和黑莓手机开发的基于网络的云连接应用程序可以帮助用户防止丢失手机，并识别和阻止消费者手机上的威胁。用户只需将该软件下载到一个设备上，它就会像下载到电脑上的安全软件一样，充当跟踪应用程序、数据备份和病毒保护程序。

Lookout 还提供了一个[移动浏览应用](https://web.archive.org/web/20230205004030/https://techcrunch.com/2011/06/15/lookout-expands-mobile-security-app-to-browsing-lands-marketing-deal-with-sprint/)，它将自动检查用户访问的每个网站，从手机上的应用程序、电子邮件或浏览器，以防止钓鱼网站窃取个人数据和恶意软件安装在设备上。它承诺了许多浏览器在桌面上提供的相同类型的安全性(但针对移动设备进行了增强)。

在过去的一年里，Lookout 的用户已经增长到了 1200 万，现在每月还在增加 100 万用户。Lookout 的应用程序实际上由 Lookout 的移动威胁网络提供支持，该网络不断分析全球威胁数据，以识别并通过无线应用程序更新快速阻止新的威胁。原本需要几天时间的威胁检测现在只需几分钟；在用户意识到威胁存在之前有效地保护他们。

移动威胁网络由超过 700，000 个移动应用程序组成的数据集提供支持，随着世界各地的应用程序商店中添加越来越多的应用程序，该数据集每天都在增长。平均每天有超过 1000 个应用程序添加到移动威胁网络中。

![](img/1ab2ffde2e0d32945725f0950a934419.png)该公司已经能够通过 API 提供这种威胁网络来进入企业业务。基本上，Lookout 的移动安全 API 可以将其保护扩展到任何应用商店或下载网站。

威瑞森是第一家使用移动安全 API 为客户提供 V CAST 应用程序的运营商，这些应用程序可以实时防御从威瑞森移动店面下载的应用程序受到的威胁。该 API 在 App Store 的后端工作，以确定任何应用程序是否包含恶意软件或其他安全威胁。Sprint 和 T-Mobile 也在使用 Lookout 的 API。考虑到美国四大航空公司中有三家正在使用 lookout，第四家(美国电话电报公司)可能很快也会加入进来。

Jordan 谈到这笔投资时说:*移动设备正迅速成为占主导地位的计算平台，随着用户越来越多地使用这些设备，安全性对于保护他们的安全至关重要……Lookout 从一开始就明白，移动安全需要一种截然不同的方法。通过将安全检测和分析迁移到云中，他们提供了出色的用户体验，同时构建了最快、最强大的安全基础架构。它的巨大增长证明了它的成功。*

科斯拉风险投资公司的维诺德·科斯拉[上周在 TechCrunch Disrupt](https://web.archive.org/web/20230205004030/https://techcrunch.com/2011/09/13/meat-two-point-o/) 的舞台上提到了 Lookout，这是他投资的更令人兴奋的公司之一。他说…“如果你没有，下载它…两年前我们投资了一两百万美元…现在他们有一千一百万用户，他们正在迅速扩大，我们很高兴继续投资。”

Lookout 的首席执行官兼联合创始人约翰·赫林(John Hering)在一次采访中告诉我们(你可以在下面观看)，新的资金将用于拓展国际市场，用于产品开发、招聘和发展美国以外的类似运营商合作伙伴关系。“现在的目标是我们如何获得 1 亿用户，”赫林说。他相信这很快就会发生。

“我们比包括赛门铁克、迈克菲在内的竞争对手大几个数量级，我们的收入增长非常可观，”他解释道。“我们从消费者和企业那里获得了实实在在的收入。”赫林拒绝透露该公司的收入。他补充说，Lookout 将在未来宣布一些新的运营商合作伙伴关系。
