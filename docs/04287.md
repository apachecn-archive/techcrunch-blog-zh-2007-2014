脸书刚刚收购了由 Y Combinator 支持的初创公司 Divvyshot，[于去年 3 月推出了 T3。Divvyshot 正在筹集一轮天使投资，这时脸书提出了一个更好的提议。收购金额尚未披露，但可能很小。购买 Divvyshot 是脸书的人才收购。创始人萨姆·奥迪奥和 Divvyshot 团队的其他两名成员将加入脸书，为世界上最大的照片共享服务脸书照片公司工作。](https://web.archive.org/web/20230328202432/https://techcrunch.com/2009/03/04/y-combinators-divvyshot-launches-dead-simple-group-photo-sharing/)

Divvyshot 是一个[群体分享图片网站](https://web.archive.org/web/20230328202432/https://techcrunch.com/2009/03/04/y-combinators-divvyshot-launches-dead-simple-group-photo-sharing/)，它采取了一种与其他图片网站略有不同的方式。大多数照片网站认为照片是和相机联系在一起的。Odio 设计了 Divvyshot，以便照片与地点、人物和事件相关联。因此，50 个人可以共享同一张照片，而不是同一事件的 50 张不同照片。脸书照片公司的工作原理与此类似，只是方式更为特殊。人们上传照片并给照片中的人贴标签。当你在其他人的照片中被标记时，它会与你共享。如果 Divvyshot 的收购是一个迹象，那么未来将有更多的方式通过脸书自动分享照片。

Divvyshot 本身将被关闭。该网站有 4 万名活跃但忠诚的用户。相反，奥迪奥和他的团队将把他们在 Divvyshot 学到的东西应用到更大的脸书照片画布上。现有的 Divvyshot 用户将能够继续使用该网站，但不能创建新的帐户，其 iPhone 应用程序将从 iTunes 中删除。

目前，Divvyshot 相册可以由多个用户编辑，并可以设置为私人(只允许指定的朋友上传照片)或公共(允许任何人上传他们的照片到相册)。所有照片都以相当高的分辨率呈现在网站上，也可以以原始质量下载(您也可以一次下载整个相册。zip 文件)。Divvyshot 与包括 Flickr、脸书和 Twitter 在内的照片服务深度整合，因此你可以从这些网站上传所有照片。该网站最近推出了一款 iPhone 应用,让用户可以实时上传和监控事件活动，并且包含了 web 应用的许多核心功能。
