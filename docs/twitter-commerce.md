# Twitter 也考虑将 PayPal 作为其广场市场的竞争对手 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/16/twitter-commerce/>

Twitter 现在是一家上市公司，对如何赚钱有更多的公众责任，它已经把大部分筹码放在了从广告中获取收入上。但它似乎正在探索的另一个领域是商业——利用它的平台不仅有助于传播品牌信息，还能帮助人们从他们那里购买商品。

今天早些时候，Recode 发表了一篇关于 Twitter 如何在商业平台上运作的[故事](https://web.archive.org/web/20221207214012/http://recode.net/2014/01/16/twitter-gets-serious-about-commerce-nears-deal-with-payments-startup-stripe/?utm_source=rc_twitter)。这也是我们已经听说了一段时间的事情，以及一些更多的细节:

Twitter 正在考虑在其平台上建立一个市场，类似于 Square 在[广场市场](https://web.archive.org/web/20221207214012/https://squareup.com/market)提供的市场，提供不同类别的商品列表。“他们正在研究一个类似于 Square 的概念，”有消息称。(没错，就是 Twitter 董事长杰克·多西担任首席执行官的那个广场。但平心而论，这听起来不像是会竞争；只是这是它的一个模型。)

尚不完全清楚，但听起来 marketplace 可能会以独立的形式存在，但也会存在于 Twitter 的主要河流上。“他们希望与更大的零售商合作，并在平台上拥有结账卡，而不仅仅是商业标签，”另一位消息人士说。

Hashtag commerce 是对 Twitter 上迄今为止存在的一些努力的参考，这些努力使用 Hashtag 来触发购买(由像 [Chirpify](https://web.archive.org/web/20221207214012/https://beta.techcrunch.com/2013/09/19/chirpify-drops-its-hashtag-commerce-consumer-marketplace-to-focus-on-big-brand-marketing-and-actiontags/) 这样的公司开创)或折扣(如 2013 年 2 月与[美国运通](https://web.archive.org/web/20221207214012/https://beta.techcrunch.com/2013/02/12/amex-pay-by-tweet/)的合作)。

从我们的理解来看，Twitter 可以说是为这项服务提供了“店面”，但不是商业后台。

“他们说他们不想处理整个商业方面的事情，”这是一个消息来源对我们说的。Recode 报道称，Twitter 一直在与 Stripe 对话。我们也听说过支付处理领域的另一个主要参与者:PayPal。目前还不清楚 PayPal 是否还在。Stripe、PayPal 和 Twitter 都没有回复置评请求。

除了前 Ticketmaster 首席执行官内森·哈伯德(Nathan Hubbard)在 8 月份被任命为 Twitter 商务主管之外，还有其他一些人专注于构建商务解决方案。

一个关键人物是 Tarun Jain，他既是前谷歌员工，也是前微软员工。我们听说他一直在与商业/支付领域的初创公司和企业家交谈，如果你去他在 AngelList 上的个人资料，你可以看到他也关注了其中一些人。不清楚这是为了潜在的收购或合作，还是仅仅为了他的个人利益。

关于 Twitter 与商业公司合作的话题，如果你想一想 Twitter 是如何通过开发人员友好的 API 扩大其受众范围的，这些 API 帮助在许多第三方应用程序中播种该服务，这将是有意义的。

至于为什么 Twitter 可能会推进商业努力，这也许是不可避免的。今天的公司是以广告为主要收入来源的。增加商务将是 Twitter 向市场展示的一种方式，它不是一匹只会一招的小马，而是一个真正的交易平台，不仅是在思想和新闻方面，也是在金钱方面。

但它是否会成功是一个不同的问题。另一个在商业领域做出努力的社交网络是脸书，但今天它的支付业务只占不到 11%的收入——T4，2013 年第三季度 20.16 亿美元的销售额中的 2.18 亿美元。
