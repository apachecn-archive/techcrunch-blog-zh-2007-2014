# NHK 展示超高清(7，680 x 4，320 像素)原型系统

[![](img/ff8d6c59e428df91f8075cf20fbf0e4c.png "nhk_ultra_hd")](https://web.archive.org/web/20221207025633/http://old.crunchgear.com/wp-content/uploads/2008/06/nhk_ultra_hd.jpg)

日本国家电视台 [NHK](https://web.archive.org/web/20221207025633/http://www.nhk.or.jp/pr/koho-e.htm) 在新加坡举行的科技博览会[上展示了一套令人惊叹的投影系统。](https://web.archive.org/web/20221207025633/http://www.communicasia.com/)

NHK 使用了 2 台 Lcos 投影仪(8000 流明)和一个 6.6 x 3.7 米的屏幕来展示他们的 3300 万像素、8K 视频投影技术(7,680 x 4,320 像素)。超高清格式产生的图像是全高清格式的四倍宽，四倍高。NHK 还在演示屏幕上连接了 22.2 声道的影院音响系统。

该电视台使用了用自主开发的 8K 演播室摄像机录制的镜头。这是 NHK 首次在亚洲(不包括日本)向公众展示他们的技术。
