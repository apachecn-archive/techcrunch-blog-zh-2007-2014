# 微软将向搜索引擎用户提供现金返还，以对抗谷歌 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2008/05/20/microsoft-to-offer-cash-back-to-search-engine-users/>

**更新:**比尔盖茨主题演讲直播流和直播搜索返现新闻稿[在此](https://web.archive.org/web/20221224072248/http://techcrunch.com/2008/05/21/live-streaming-bill-gates-keynote-at-advance08/)。

**更新 2:** [我们完整的直播搜索返现分析在此](https://web.archive.org/web/20221224072248/http://techcrunch.com/2008/05/22/the-empire-strikes-back-our-analysis-of-microsoft-live-search-cashback/)。

微软将于明天在华盛顿州雷蒙德市举行的 Advance08 大会上宣布一个新的搜索广告模式——该网站的某些部分已经在微软 now 上直播(见下面的截屏)。新服务的核心将是一套新的 18 个新的垂直搜索产品，将为用户从广告商那里购买的任何产品提供现金返还。

许多知名的电子商务网站正在参与该计划的早期阶段，该计划被称为“Live Search Cashback ”,至少部分基于微软于 2007 年收购的水母公司开发的技术。水母网站上的一条消息称，该网站已关闭“目前离线，以执行必要的服务升级和增强。”

当然，目标是从谷歌那里吸引高价值的搜索。总搜索量中只有一小部分是非常有价值的，通常是因为广告商正处于向搜索者出售某些东西的风口浪尖(例如，对书籍或抵押贷款的搜索)。

微软希望以只有在有人购买时才付费的承诺来吸引广告商，这与谷歌的点击付费模式不同，谷歌的点击付费模式风险更大，因为搜索者可能不会完成交易。通过向广告商收取一定比例的费用，微软希望说服搜索者通过 Live.com 搜索引擎完成交易的最后一英里，为微软创造更多广告收入，同时伤害主要竞争对手谷歌。

新产品将于明天早上由微软平台服务部门总裁凯文·约翰逊宣布。我们将在博客上直播这次活动。

[Farecast](https://web.archive.org/web/20221224072248/http://www.microsoft.com/) ，他们[上个月](https://web.archive.org/web/20221224072248/http://techcrunch.com/2008/04/17/microsoft-acquires-farecast-for-115m/)收购的一家公司，很可能会为 18 个垂直行业之一的旅行提供动力。

**雅虎角**

这符合微软最近与雅虎的重新合作以及接管雅虎搜索业务的新提议。微软将会立刻在广告客户中引起轰动。通过消除风险(从 CPC 转移到 CPA ),他们将实现部分目标。但微软也需要向广告客户提供足够多的库存，让他们觉得物有所值——雅虎搜索流量做到了这一点。

但即使没有雅虎，微软也可能会受到广告商的热烈欢迎，他们目前认为谷歌在搜索广告领域处于虚拟垄断地位。让尽可能多的强者争夺他们的业务符合他们的最大利益，因此任何与谷歌的竞争都值得他们花费时间。
