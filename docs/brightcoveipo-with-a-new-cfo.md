# Brightcove 与新首席财务官一起准备 IPO

![](img/f05d9f810a899bd75b5310194aa6b784.png)

在线视频平台 [Brightcove](https://web.archive.org/web/20221007120619/http://www.brightcove.com/en/) 向[最终 IPO](https://web.archive.org/web/20221007120619/https://beta.techcrunch.com/2009/12/24/top-ten-ipo-candidates-2010/) 迈出了又一步，今天雇佣了新的首席财务官。克里斯托弗·梅纳德是新的首席财务官。此前，他是 Phase Forward 的首席财务官，Phase Forward 是一家临床试验企业软件公司[，去年 4 月被甲骨文](https://web.archive.org/web/20221007120619/http://www.oracle.com/us/corporate/press/068204)以 6.85 亿美元收购。在甲骨文收购之前，Phase Forward 是一家上市公司。

在宣布 Menard 的新角色时，Brightcove 强调，他“在六项收购的执行和整合、首次公开募股和二次股票发行的成功执行以及全球金融系统的实施中发挥了不可或缺的作用。”梅纳德还在波士顿学院商学院教授一门关于 M&A 的课程。正是你在申请首次公开募股前想要的那种人。要么这样，要么离开 M&A。

在过去的一年里，Brightcove 一直在加强其高管团队，增加了新的总裁和首席运营官(David Mendels)，新的董事会成员(前 Macromedia CFO[Betsey Nelson](https://web.archive.org/web/20221007120619/http://www.crunchbase.com/person/betsey-nelson))，新的北美销售高级副总裁(Paul Goetz)，以及负责不断增长的亚太业务的新副总裁(Dennis Rose)。
