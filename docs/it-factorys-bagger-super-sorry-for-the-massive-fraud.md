# IT 工厂的 Bagger 超级抱歉大规模欺诈

围绕丹麦 IT 工厂兴衰的一个引人入胜的故事(可能是一部好莱坞电影)开始浮出水面。11 月下旬，著名的首席执行官 Stein Bagger 在迪拜失踪——后来发现，随着他的公司分崩离析，他抛弃了妻子和孩子，逃到了纽约。

然后，他借了一个朋友的车和信用卡，踏上了前往加州的旅程，在那里他迅速向当地警方(T4)自首。

丹麦媒体对他进行了一次[采访](https://web.archive.org/web/20221006201217/http://ekstrabladet.dk/nyheder/samfund/article1097057.ece)(这里的翻译版本是)。他们问的第一个问题是:你为什么要延续一场长达四年的欺诈，涉及大约 1.7 亿美元的欺诈收入(IT 工厂 90%的收入可能都是谎言)？

> 我自己和家人的生命都受到过威胁，我受到过 25 到 30 次的火器威胁。他不肯说是谁威胁了他，但告诉他们会从他那里拿钱。

每个人被抓后都会后悔。
