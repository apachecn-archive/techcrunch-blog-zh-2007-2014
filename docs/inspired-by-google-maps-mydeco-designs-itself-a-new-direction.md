# 受谷歌地图的启发，mydeco 为自己设计了一个新的方向

对 lastminute.com 背后的管理团队创办的家居设计和家具初创公司 mydeco 来说，这是一段有趣的旅程，它是欧洲有史以来最成功的在线企业之一。早在 2008 年，该网站就开创性地使用了 3D 成像技术，以支持交互式规划和可视化工具。它甚至有一个社交网络。但是尽管做得足够好，游戏已经改变了。消费者现在受到选择的冲击——3D 设计工具的功能总是比普通人需要或想要的略多。mydeco 不是设计无限数量的房间，并购买其中的物品，而是希望通过成为一家专门从事策展的零售商来吸引用户。正如首席执行官妮可·范德比尔特告诉我的，“我们有点像 ASOS。我们从一个聚合商开始，现在越来越像一个零售商。零售商拥有顾客，因此利润更高。但我们最终希望成为谷歌室内地图——无论你想在哪里建造漂亮的房间，我们都会在那里。”
