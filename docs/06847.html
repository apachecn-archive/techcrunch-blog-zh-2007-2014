移动通讯初创公司Kik融资800万美元；启动群聊和照片共享

移动通讯初创公司 Kik Interactive 已经从 RRE风险投资、星火资本和联合广场风险投资获得了800万美元的首轮融资。Kik还宣布，联合广场风险投资公司(Union Square Ventures)的弗雷德·威尔森(Fred Wilson)和RRE风险投资公司的亚当·路丁(Adam Ludwin)将加入公司董事会。

Kik Messenger于2010年10月推出，是一款速度极快的移动通讯应用。它类似于黑莓信使(BBM)，除了它可以跨智能手机平台工作。一旦下载，该应用程序将扫描您的地址簿，并告诉您哪些朋友下载了Kik。你可以开始和这些朋友聊天，就像BBM一样，信息会实时更新。您还可以查看联系人是否以及何时阅读了您的邮件。你可以通过文本、推文或电子邮件向朋友发送包含安装链接的邀请，Kik也会向你推荐用户。

在Kik Messenger对公众开放的头15天里，它经历了病毒式增长，获得了超过100万用户。但去年11月，当RIM 从黑莓世界(Kik推出iOS、Android和黑莓应用)撤下Kik的应用时，这家初创公司引发了一点争议。RIM称，Kik“违反了合同义务”人们认为Kik对地址簿的使用可能是一个问题(这也是iOS 术语的一个问题，但这似乎没问题)。火上浇油的是，RIM 起诉Kik 专利侵权。

Kik今天还为iOS和Android设备发布了一个新版本的Kik Messenger 】,可以进行多达10人的实时群聊。Kik Messenger的新图片分享功能允许用户即时分享他们看到的内容。

正如新董事会成员兼投资者Fred Wilson所说，“Kik是一个快速、简单、优雅的跨设备移动通讯解决方案，我们认为这是目前最令人兴奋的投资机会之一…我们相信Kik团队已经表明，他们
知道如何以正确的方式提供正确的功能集…”

Kik现在有341万用户，因此它似乎在稳步增长(尽管可能没有推出时那么快)。值得注意的是，Kik面临着许多类似移动聊天应用的竞争，包括 Beluga (刚刚被脸书收购) HighNote 和 PingChat 。
