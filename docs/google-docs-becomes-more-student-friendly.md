# 谷歌文档变得对学生更加友好

![](img/24bf9fa26322fbab79b735e7d84ca668.png)

谷歌一直在积极地向学校推销谷歌应用程序，最近[推出了](https://web.archive.org/web/20221006212805/http://www.beta.techcrunch.com/2009/09/08/google-goes-after-impressionable-minds-with-education-apps/)一个旨在招募大学和学院的集中网站。现在，谷歌正在[调整](https://web.archive.org/web/20221006212805/http://googleblog.blogspot.com/2009/09/back-to-school-with-google-docs.html)谷歌文档，这是谷歌企业应用套件的一部分，增加了一些学生友好的功能。

谷歌文档增加了一个方程编辑器，这样学生就可以在一个文档中完成数学问题，让学生不仅可以写包含数字和方程的论文，还可以使用谷歌文档在定量课上做笔记。谷歌还增加了插入上标和下标的功能，这对于写出化合物或代数表达式很有用。

谷歌还试图让文档对那些人文专业的学生有吸引力，让用户从各种创建大纲的项目列表样式中进行选择，并让学生能够打印脚注作为学期论文的尾注。几周前，谷歌[在谷歌文档中推出了](https://web.archive.org/web/20221006212805/http://www.beta.techcrunch.com/2009/08/27/translate-google-docs-and-share-them-around-the-world/)翻译功能。

正如我们过去所写的，谷歌招募教育机构是明智的，因为许多人在那里接受培训，开始依赖并形成对生产力应用的品牌忠诚。借鉴苹果的战略，谷歌知道品牌忠诚度肯定是在这些学校中形成的，并正在稳步开发其产品，以变得对学生更具吸引力。竞争对手微软也针对学生用户推出了基于网络的 Office 产品。初创公司 Zoho 提供免费的基于网络的生产力套件。
