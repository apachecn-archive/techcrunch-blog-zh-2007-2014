# 东芝的梦想:每锅一只鸡，每张 SD 卡都有 WiFi

如果你有一张 [Eye-Fi](https://web.archive.org/web/20221208071818/http://www.crunchgear.com/tag/eye-fi/) 卡，你可能会意识到那里的便利因素。电线是一种痛苦，我们拥有的其他一切都是无线的，为什么我们的存储介质不是呢？当然，Eye-Fi 在很久以前就解决了这个问题，但尽管他们取得了成功，但他们仍然是一个精品品牌。东芝希望继承他们开创的事业，并使之成为标准。

我对此没有异议，但我想说的是，设备本身也越来越多地使用无线技术。这里有无线同步，那里有自动上传到 YouTube 和 Flickr，以及其他各种功能。有独立 WiFi 设置的卡真的有空间吗？当然可以。出于这样或那样的原因，某些型号总是没有某些功能。为了让价格低于临界点或在关键测量中的尺寸，无线等可选的东西往往被排除在外，目前仍将如此。

东芝正在与 Trek 2000 合作的规范还远未完成(实际上，它才刚刚开始)，但标清到标清通信的想法令人兴奋。最受欢迎的存储介质之间的通用接口将有助于一些有趣的设备和应用。我甚至可以想象一个只有一个目的的小设备:方便两个 SD 卡之间的数据共享。我有点期待。
