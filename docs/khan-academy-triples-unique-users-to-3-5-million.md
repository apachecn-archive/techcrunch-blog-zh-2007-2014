# 可汗学院的独立用户增加了两倍，达到 350 万

今天在三藩市的 Web 2.0 峰会上，[可汗学院](https://web.archive.org/web/20230325160348/http://www.khanacademy.org/)的创始人萨尔曼·可汗上台分享了他的在线视频教育平台的一些快速增长数据。对于那些不熟悉的人来说，正如约翰·贝特尔今天下午指出的，汗学院是比尔·盖茨最喜欢的教育机构之一。

它碰巧也是我的一本书，但是我想你可能会和比尔·盖茨有更多的共鸣。但是可汗学院是萨尔曼·可汗的机构，他带来了教育年轻人的想法，这些年轻人是自发的，他们以自己的速度在线学习。“教育”类型的 YouTube 视频已经存在多年了，但可汗学院的资料库却非常可笑。这家教育初创公司现在的图书馆中有超过 2600 个视频，有从算术到物理的各种课程或课程，包括 211 个练习，让学生观看视频并按照自己的节奏学习。

虽然 Khan 是一个非营利组织，但该学院已经收到了盖茨基金会的捐款，并赢得了谷歌的“10^100".项目”有了谷歌和盖茨的 200 多万美元，Khan 今天在 Web 2.0 上告诉观众，该学院每月有 3900 万次页面浏览量和 350 万独立用户。350 万独立用户同比增长 309%。

看起来所有的宣传和牵引可汗学院最近真的开始得到回报，这是很高兴看到的。
