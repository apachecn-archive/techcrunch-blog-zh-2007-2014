# 雅虎又失去两个:法耶兹和扎沃德尼

![](img/ed8ceb8465136a09e7b1ac8beca99e5a.png)

昨天[我们向](https://web.archive.org/web/20230124074416/http://techcrunch.com/2008/06/11/jeff-weiners-departure-from-yahoo-imminent-speculation-on-successor-begins/)报道了[雅虎网络部门的执行副总裁杰夫·韦纳](https://web.archive.org/web/20230124074416/http://www.crunchbase.com/person/jeff-weiner)已经递交了辞呈。现在，我们又听到了这家陷入困境的公司的两起更为突出的离职事件。

NYT·比特公司[宣布](https://web.archive.org/web/20230124074416/http://bits.blogs.nytimes.com/2008/06/12/more-top-yahoos-heading-for-the-exits/)首席数据官兼研究和战略数据解决方案副总裁[乌萨马·法耶兹](https://web.archive.org/web/20230124074416/http://www.crunchbase.com/person/usama-fayyad)离职，预计将于今天晚些时候正式宣布。根据这份报告，法耶兹一直“负责挖掘该公司收集的万亿字节数据，以改善向雅虎用户定向广告和内容等事情。”

资深开发者 Jeremy Zawodny 也在他的博客上宣布他将在未来几周内交出他的紫色徽章。Zawodny 自 1999 年以来一直在雅虎工作，帮助领导一些重要项目，如雅虎开发者网络。

对于雅虎来说，这不是一个好日子。自从与谷歌的联合声明和与微软的谈判破裂的消息在早些时候传出后，它的股票直线下跌。

对雅虎的人力资源来说，今年也不是一个好年头。仅在二月份，雅虎高级技术部门的负责人[布拉德利·霍洛维茨](https://web.archive.org/web/20230124074416/http://techcrunch.com/2008/02/12/yahoo-exec-bails-bradley-horowitz-leaves-for-google/)；[雅虎 Brickhouse 负责人 Salim Ismail](https://web.archive.org/web/20230124074416/http://techcrunch.com/2008/02/12/yahoo-brickhouse-left-headless/)；而[社交搜索副总裁杰夫·邦福特](https://web.archive.org/web/20230124074416/http://techcrunch.com/2008/02/15/another-vice-president-parachutes-from-yahoo-lands-as-ceo-of-xobni/)在更大范围的裁员中离职。然后在 4 月初，雅虎音乐副总裁伊恩·罗杰斯也辞职加入了一家秘密创业公司。

**更新:**国际业务和产品管理高级总监马修·韦拉尔多也将[离开](https://web.archive.org/web/20230124074416/http://www.forbes.com/businesswire/feeds/businesswire/2008/06/12/businesswire20080612006223r1.html)到 LiveJournal 去填补副总裁和总经理的职位。
