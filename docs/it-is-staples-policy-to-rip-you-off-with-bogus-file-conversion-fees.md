# 用虚假的“文件转换”费来敲诈你是史泰博的“政策”

我刚从斯台普斯回来，在那里我遇到了我所见过的最糟糕的虚假费用。诡计:如果你让他们打印任何不是 PDF 格式的文件，他们会收取 2 美元的“转换费”，即使没有发生文件转换。如果他们对只需点击一下鼠标而完全不用花钱的东西收费，那就够糟糕的了，但这太荒谬了

跳楼后的全部细节和故事……
 几天前，我去了曼哈顿上城 184 街和百老汇街的史泰博，为了打印一个简单的 JPEG 文件。复印/打印区的员工给了我一个似乎有点高的价格，但没有任何解释。我询问了价格的细节，她说他们增加了 2 美元的“转换费”,因为文件不是 PDF。我告诉她，你不需要把它转换成 PDF，因为 JPEGS 完全可以自己打印出来。更糟糕的是，她实际上没有做任何转换。她只是打印了某个图片程序的图片。她说她无能为力，这是公司的政策。我要求和经理说话。他承认了我的投诉并免除了费用。

快进到今天。我去同一个订书钉店做日常打印和传真。这次，我需要打印的文件是 Word(。doc)文件。不幸的是，我没有给予足够的重视，直到我离开商店并查看收据时才意识到他们再次收取了“转换费”(见上图)。)

这其中的问题是显而易见的:

1)打印任何这些文件都不需要转换

2)实际上没有进行任何转换来打印这些文件。

3)他们不会告诉你这笔费用，而是告诉你费用是多少，并要求你支付。如果你不看你的收据，你不知道你为此被收费。

4)即使我的前两点不正确，转换文件也不需要任何成本。

简而言之，永远看你的收据，永远不要相信史泰博。
