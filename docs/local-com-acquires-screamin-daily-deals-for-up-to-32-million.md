# Local.com 以高达 3250 万美元的价格收购 Screamin ' Daily 交易

![](img/e202e4d36266eb039fd8fc6b31403b7a.png)

当地在线媒体公司[Local.com](https://web.archive.org/web/20230203133912/http://www.crunchbase.com/company/local-com)今天早上宣布，它已经收购了 [Screamin' Media Group](https://web.archive.org/web/20230203133912/http://www.crunchbase.com/company/screamin-daily-deals) ，该公司是 ~~Groupon 克隆~~ daily deals 网站 [Screamin' Daily Deals](https://web.archive.org/web/20230203133912/http://www.screamindailydeals.com/) 的幕后公司。

Local.com 以现金、股票和债务的形式支付了 1250 万美元，如果在交易完成后的两年时间内满足确定无疑的高端财务表现标准，Screamin' Media Group 的股东有机会获得额外的 2000 万美元。

Screamin' Daily Deals 成立于 2010 年初，拥有约 60 名员工，为全美 14 个市场的“数十万”用户提供服务，这些用户与当地商家达成交易。该公司最近还推出了旅游优惠。

据说这家初创公司在 2010 年创造了约 240 万美元的未经审计的收入，在 2011 年上半年创造了 440 万美元的收入。

Screamin' Media Group 将迁至 Local.com 位于加州尔湾的公司总部，并被后者的[团购业务](https://web.archive.org/web/20230203133912/https://techcrunch.com/2011/05/02/local-com-buys-local-search-company-krillion-for-3-5m-debuts-daily-deal-service/)部门 [Spreebird](https://web.archive.org/web/20230203133912/http://www.spreebird.com/) 所吸收。

这家初创公司的三位联合创始人，即前总裁、首席执行官和首席运营官，将分别成为 Spreebird 的销售和营销副总裁、产品副总裁和运营副总裁。

根据最近提交给美国证券交易委员会的文件，Local.com 刚刚与谷歌签订了一份为期两年的协议，以实施“某些广告和搜索服务”，这导致股价上涨。
