# 所以，是的，将会有一部关于谷歌实习生生活的文斯·沃恩和欧文·威尔逊的电影

【YouTube = http://www . YouTube . com/watch？从地理上来说，好莱坞距离硅谷数百英里(T2)，但从隐喻的角度来看，这两者似乎越来越近了。

最新的例子是 *[《实习医生》](https://web.archive.org/web/20221006172255/http://www.imdb.com/title/tt2234155/)* ，由文斯·沃恩和欧文·威尔逊主演的新电影《老友记》将于今年夏天首映。在这部电影中，沃恩和威尔逊扮演被解雇的中年推销员，他们设法在谷歌找到了实习机会。戴着眼镜的天才 20 多岁的年轻人是他们的老板，他们并不完全适合，当然欢乐随之而来。

毫不奇怪，当今科技行业有些古怪的世界对娱乐圈类型来说是有吸引力的故事素材: *[《社交网络》](https://web.archive.org/web/20221006172255/http://www.imdb.com/title/tt1285016/?ref_=sr_1)* 展示了这些故事[如何能在大银幕上引人注目。多年来，在脸书和谷歌这样的公司工作的感觉对主流社会有着特殊的吸引力(正如我在我的](https://web.archive.org/web/20221006172255/https://beta.techcrunch.com/2010/09/20/terror-nerd/)[披露表](https://web.archive.org/web/20221006172255/https://beta.techcrunch.com/author/colleen-taylor/)中所说的，我的配偶在谷歌工作——每个假期，他都会回答关于广泛报道的工作福利的问题，比如甚至来自我们最不插电的家庭成员的免费食物。)

但是令人惊讶的是谷歌似乎在很大程度上参与了《实习医生 T2》的制作。据说演员阵容[在他们准备拍摄这部电影时可以进入](https://web.archive.org/web/20221006172255/http://www.thesun.co.uk/sol/homepage/showbiz/film/4793956/vince-vaughn-owen-wilson-filming-the-internship-google-hq.html#ixzz2KpgGVzcs)谷歌的山景城总部，谷歌自己也发表了一份[热情支持的声明](https://web.archive.org/web/20221006172255/http://www.thesun.co.uk/sol/homepage/showbiz/film/4793956/vince-vaughn-owen-wilson-filming-the-internship-google-hq.html#ixzz2KpgGVzcs)说:“我们很兴奋文斯·沃恩和欧文·威尔逊选择谷歌校园作为他们自《婚礼傲客》以来第一部合作电影的背景。”昨天，谷歌的[主页](https://web.archive.org/web/20221006172255/http://www.google.com/)(一个著名的[稀疏空间](https://web.archive.org/web/20221006172255/http://www.huffingtonpost.com/2012/03/27/google-design-sergey-brin_n_1384074.html)，通常只偶尔添加自己的新产品或大型慈善事业的链接)甚至包括一个[显著链接](https://web.archive.org/web/20221006172255/https://plus.google.com/101560853443212199687/posts/RvKdeJt8RLy)到一个[与沃恩和威尔逊的 Google+hang out](https://web.archive.org/web/20221006172255/https://twitter.com/loyalelectron/status/301587481336356864/photo/1)，这显然是电影宣传的一部分。

这与脸书对《社交网络》的回应大相径庭，后者大多保持沉默，偶尔遭到断然拒绝。当然，这是两种非常不同类型的电影，但看到一家公司在这里如此合作是很有趣的。

总之。预告片今天刚上线，嵌在上面。
