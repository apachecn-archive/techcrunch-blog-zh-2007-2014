# 谷歌收购波士顿动力公司，大狗的创造者

谷歌宣布他们已经收购了波士顿动力公司，该公司是四踏板和双踏板机器人的创造者，如 T2 大狗和 T4 宠物人。这是谷歌的第八次机器人收购。

该公司没有透露出售的细节。

该声明出现在《纽约时报》上，波士顿动力公司首席执行官马克·雷伯特表示，尽管谷歌不会正式成为军事承包商，但他们将履行 DARPA 的军事合同。

该公司成立于 1992 年，过去十年一直致力于研发独立的气体驱动机器人。这些机器人能够自行恢复正常，而且非常有弹性。像大狗这样的机器人可以扔煤渣砖，处理岩石地形，并以 [16 英里/小时](https://web.archive.org/web/20230327124722/http://www.youtube.com/watch?v=wE3fmFTtP9g)的速度奔跑。

此次收购的幕后黑手安迪·鲁宾(Andy Rubin)在将一个鲜为人知的移动操作系统变成巨头后，于今年 3 月辞去了谷歌 Android 业务负责人的职务。拉里·佩奇在他的 Google+页面上写道:“他的最后一个大赌注，Android，开始是一个疯狂的想法，最终把一台超级计算机放进了数亿个口袋。”“这还为时尚早，但我迫不及待地想看到进展。”
