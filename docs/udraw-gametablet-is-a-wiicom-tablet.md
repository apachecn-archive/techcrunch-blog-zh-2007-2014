# THQ 的 uDraw GameTablet 是…一款 Wiicom 平板电脑

感谢上帝，另一家外设制造商做出了我可以用“Wii”玩的东西其他选择已经过时了。所以，这个 Wiicom 平板电脑(参考，如果你不知道的话)是一个附件，可以让你在 Wii 游戏里面画画。太好了！不确定是否有很多游戏真的需要这个，因为你已经控制了一个自由光标,“绘图”游戏在 DS 上运行得更好，但是嘿。

它将配备一个素描程序，让你保存或重放你的涂鸦，还有一个基于绘图的平台以及不可避免的图片。我哪天都会拿纸笔看图，因为太神奇了，但是我知道那是*期待*的喜悦，所以我不会嫉妒那些选择 Wii 版本的人。
