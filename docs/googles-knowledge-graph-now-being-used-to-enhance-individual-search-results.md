# 谷歌的知识图谱现在被用来增强个人搜索结果

谷歌今天宣布，谷歌的[知识图谱](https://web.archive.org/web/20221208204458/http://www.google.com/insidesearch/features/search/knowledge.html)正在融入实际搜索结果中。在谷歌官方搜索博客上的一篇[博文](https://web.archive.org/web/20221208204458/http://insidesearch.blogspot.com/2014/01/more-information-about-websites-to-help.html)中，谷歌解释了你现在如何能够通过点击搜索结果中给定链接旁边出现的名称来了解更多你正在搜索的主题。

这些附加的信息部件目前只出现在桌面搜索中。

在所提供的示例中，搜索“内战”可能会标记有另一个小链接，单击该链接会显示一个总结该页面主题的下拉框。如图所示(见上图)，一个名为“内战信托”的额外链接弹出一个框，用一句话解释所指的非营利组织，包括其成立日期。

这些新链接很小而且是灰色的，这样不会影响搜索体验。它们出现在较大的蓝色链接的下方，实际站点 URL(绿色链接)的旁边。

谷歌表示，额外的信息只会提供给那些“被广泛认为是值得关注的在线网站”，这意味着你不太可能在谷歌搜索结果的第 10 页看到某个随机出现的人的博客得到同样的待遇。

额外的信息由谷歌的知识图提供支持，这是一项语义搜索工作，该公司最早于 2012 年[推出了](https://web.archive.org/web/20221208204458/https://beta.techcrunch.com/2012/05/16/google-just-got-a-whole-lot-smarter-launches-its-knowledge-graph/)，通过在其搜索索引中注入对人、地点和事物的理解，使谷歌搜索更加智能。

从那时起，知识图谱几乎触及了谷歌搜索的每个方面，包括[谷歌趋势](https://web.archive.org/web/20221208204458/https://beta.techcrunch.com/2013/12/03/google-trends-gets-smarter-by-enlisting-data-from-knowledge-graph/)、其[移动卡片式界面](https://web.archive.org/web/20221208204458/https://beta.techcrunch.com/2013/09/26/google-improves-knowledge-graph-with-comparisons-and-filters-brings-cards-to-mobile-search-adds-cross-platform-notifications/)、涉及统计数据的[搜索](https://web.archive.org/web/20221208204458/https://beta.techcrunch.com/2013/05/15/google-knowledge-graph-gets-smarter-adds-statistics-adds-4-new-languages/)等等，此外还在谷歌搜索结果的右侧(有时在顶部)提供一目了然的信息。

像之前的一些知识图增强一样，新的信息弹出窗口可能会影响搜索结果中返回页面的点击量。例如，如果搜索者只是寻找简要的解释或事实核查，弹出窗口就足够了。但谷歌声称这些额外信息是为了帮助你从蓝色链接列表中选择正确的结果，而不是避开它们。
