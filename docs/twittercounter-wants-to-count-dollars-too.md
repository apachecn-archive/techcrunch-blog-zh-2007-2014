# TwitterCounter 也想数美元

TwitterCounter 是一项相当基础但受欢迎的服务，它让用户了解他们在 Twitter 上的关注者和推文数量方面做得如何，它正在将收入开关切换到“开”。

相当数量的 Twitter 用户定期访问 TwitterCounter 网站，根据他们的账户名称获得统计数据——Compete 估计每月的 uniques 数量大约为[。650，000](https://web.archive.org/web/20230222201818/http://siteanalytics.compete.com/twittercounter.com/)——这项服务背后的团队认为，公司和组织很可能愿意每月向他们支付费用，以获得更多功能和更详细的统计数据的优质服务。

在该公司所谓的[高级仪表板](https://web.archive.org/web/20230222201818/http://twittercounter.com/pages/dashboard_info)中，付费用户可以比较和跟踪多个账户，并获得一年多的统计数据，而不是非付费用户最多 3 个月的数据。此外，Twitter counter‘pro’用户能够以 CSV 格式导出统计数据，并以更大的格式欣赏他们的图表。

TwitterCounter 的 Patrick De Laive 告诉我们，支持多账户跟踪和比较是一个经常被请求的功能，该团队也是名为 [TwitterRemote](https://web.archive.org/web/20230222201818/http://twittercounter.com/pages/remote) 的“MyBlogLog for Twitter”服务和阿姆斯特丹下一次网络会议等活动的幕后支持者，他们很高兴最终能够提供这一功能。这并不是说这家自筹资金的初创公司没有试图通过网络服务赚钱:他们还为用户提供了一种获得更多关注者的方式，即以每次观看的价格展示他们(例如，10 万次观看，289 美元)。

Premium Dashboard 的价格基于您想要跟踪的用户数量，5 个用户每月 25 美元，100 个用户每月 198 美元。你可以说我疯了，但尽管价格可能听起来很高，我实际上可以理解为什么公司——尤其是它们的公关和营销部门——愿意为这种类型的服务付钱。

与此同时，让 Twitter 自己解决如何以及是否会赚钱的问题。
