# (视频演示)比尔·格罗斯插话说

本周早些时候， [UberMedia](https://web.archive.org/web/20230205042429/http://www.ubermedia.com/) (和 Idealab)首席执行官比尔·格罗斯在 Web 2.0 峰会上推出了一款名为 [Chime.in](https://web.archive.org/web/20230205042429/http://chime.in/) 的新产品。这是一个关注你关心的人和事的兴趣流。你甚至可以[跟踪一个人的一部分](https://web.archive.org/web/20230205042429/https://techcrunch.com/2011/10/18/gross-chime-in-follow-part-person/)(只跟踪他们分享的关于科技的东西，不跟踪他们分享的关于猫的东西)。格罗斯给了我一个 [iPhone 应用](https://web.archive.org/web/20230205042429/http://itunes.apple.com/us/app/chime.in-for-iphone/id470952970?mt=8)的演示，你可以在上面看到。
