# 松下创造铁人三项机器人

为了纪念在夏威夷举行的铁人三项赛，松下创造了一个小小的 EVolt 吉祥物，它将在大约一周内跑完全程 142 英里。这个小小的机器人将使用 EVolt 电池，由于他比人类小 10 倍，所以他需要更长的时间来完成课程。

这个小机器人将游泳 2.36 英里，骑自行车 111.8 英里，然后跑完整个马拉松。我怀疑这个小家伙可能会有一些后备机器人来帮助他，如果他死在大海中间的话。
