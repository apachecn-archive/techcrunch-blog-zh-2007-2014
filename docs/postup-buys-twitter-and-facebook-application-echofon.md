# PostUp 收购 Twitter 和脸书应用开发商 EchoFon

根据今天发布的[新闻稿](https://web.archive.org/web/20230202231756/http://www.businesswire.com/news/home/20110105005767/en/PostUp-Acquires-Echofon-Leading-Twitter-App-iPhone)，Twitter 搜索和广告初创公司 [PostUp](https://web.archive.org/web/20230202231756/http://www.postup.com/) ( [前身为 TweetUp](https://web.archive.org/web/20230202231756/https://techcrunch.com/2010/08/30/tweetup-now-includes-updates-from-linkedin-and-facebook-changes-name-to-postup/) )已经收购了 Twitter 应用开发商 [Echofon、](https://web.archive.org/web/20230202231756/http://www.echofon.com/)。交易条款没有披露。

Echofon 过去被称为 TwitterFon，有超过 300 万人使用。初创公司[为 iPhone 提供了一个](https://web.archive.org/web/20230202231756/https://techcrunch.com/2009/09/29/echofon-formerly-twitterfon-unveils-a-simple-and-fast-native-mac-twitter-client/) Twitter 应用程序，一个本地 Twitter 客户端，一个为 iPhone 提供的脸书应用程序，以及一个 Twitter iPad 应用程序。

Echofon 最引人注目的功能之一是，它可以确定你已经在移动应用程序之间看过哪些推文，所以你不会重复阅读相同的内容。

由 [idealab 孵化的 PostUp，](https://web.archive.org/web/20230202231756/http://www.idealab.com/)去年为其[实时搜索](https://web.archive.org/web/20230202231756/https://techcrunch.com/2010/04/11/twitter-idealab-tweetup-adsense-bill-gross//)竞价市场开放注册，并在此过程中获得了许多 Twitter 客户。这家初创公司最近[收购了](https://web.archive.org/web/20230202231756/https://techcrunch.com/2010/07/06/tweetup-twidroid-popurls/) Twitter Android 客户端 Twidroid 和新闻聚合服务 [popurls。](https://web.archive.org/web/20230202231756/http://popurls.com/) PostUp 的搜索算法和市场已经被整合到许多网站上，包括 TechCrunch、Topix.com 和 Businessinsider.com。

PostUp 计划扩展 EchoFon 的特性和功能，并表示该开发商的产品是该公司 Android 手机 Twitter 应用程序的“自然补充”。
