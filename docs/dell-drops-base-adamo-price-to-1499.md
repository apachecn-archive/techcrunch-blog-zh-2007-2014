# 戴尔将 Adamo 的基本价格降至 1499 美元

![dell](img/ac7c4097abf4aa2bd09f3303403650aa.png)

看起来戴尔把超薄 Adamo 的起价降到了 1499 美元——当这款机器在三月正式发布的时候，起价是 1999 美元。

基本配置将为您提供以下功能:

*   1.2GHz 英特尔酷睿 2 双核 CPU
*   64 位 Windows Vista 家庭高级版
*   2GB 内存
*   128GB 固态硬盘

那是给“佩服”版的。升级到“Desire”配置，现在已经降低到 2299 美元的起价，你将获得 1.4GHz 的 CPU，两倍的 RAM 和移动宽带芯片。

如果我没有对 MacBook Air 进行武断的比较，那就是我的失职了，所以我要说:

Adamo 1499 美元的基础价格现在和 MacBook Air(T4)一样。与 Adamo 的 1.2GHz CPU、固态硬盘和英特尔显卡相比，MacBook Air 拥有 1.86GHz CPU、120GB SATA 硬盘和 NVIDIA GeForce 9400M 显卡。Adamo 的重量也增加了约 1 磅:4 磅对 3 磅。更高端的 MacBook Air 起价为 1799 美元，而 Adamo 起价为 2299 美元。MacBook Air 配备了 2.13GHz CPU、2GB 内存、NVIDIA 显卡和 128GB 固态硬盘，而 Adamo 配备了 4GB 内存和移动宽带，尽管 CPU 速度较慢。
