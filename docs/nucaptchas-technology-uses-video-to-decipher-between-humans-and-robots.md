# NuCaptcha 的技术使用视频来破译人类和机器人之间的信息

我们都遇到过自己的[有趣的验证码](https://web.archive.org/web/20230205031546/https://techcrunch.com/2010/03/31/facebook-cruel-captcha/)毫无意义。现有的验证码技术，如谷歌的 reCAPTCHA(T4)也不能幸免于安全攻击。 [NuCaptcha](https://web.archive.org/web/20230205031546/http://www.nucaptcha.com/comingsoon) 今天[发布](https://web.archive.org/web/20230205031546/http://www.prnewswire.com/news-releases/video-based-captchas-now-available-for-sites-and-blogs-97471319.html)，希望颠覆现有的验证码空间。该平台使用视频来确定人是否真的是人，而不是机器。

如你所知，验证码是你在网站上发现的安全问题，它要求你破译和输入单词或数字，并检测用户是人还是垃圾邮件。你看到的大多数验证码是转录的，基于文本的验证码。但 NuCaptcha 说，随着电脑黑客和垃圾邮件机器人变得越来越普遍，公司已经使验证码更难阅读，对用户不友好。

NuCaptcha 的视频动画验证码技术声称既容易阅读又安全，动画使验证码更容易被人类破解(你可以在下面测试一下)。此外，NuCaptcha 平台使用行为智能向合法用户提供非常简单的验证码，并向攻击者提供越来越难的验证码。

该产品对网站和博客免费，每月提供多达 25，000 个验证码。NuCaptcha Basic 提供各种主题的验证码，目前包括环境、运动和摘要。网站可以选择显示一个或多个最适合其外观和内容的主题。

NuCaptcha 的竞争对手是该领域现有的验证码技术，如谷歌的 reCAPTCHA 或微软的 [KittenAuth。虽然这家羽翼未丰的初创公司正在与科技巨头和更老牌的公司竞争，但 NuCaptcha 的技术或许能够为这一组合添加一个令人信服的选项。](https://web.archive.org/web/20230205031546/http://thepcspy.com/kittenauth/)
