# 视频:OTOY 在行动。你得看看这个。

大约一年前，当我们第一次看到 OTOY 时，这家小公司正试图[提供一种服务器端 3D 渲染技术](https://web.archive.org/web/20230203224534/https://techcrunch.com/2008/07/09/otoy-developing-server-side-3d-rendering-technology/)，这种技术可以让现代视频游戏在几乎任何客户端上运行。当然，这是一个崇高的目标。然后 [OnLive](https://web.archive.org/web/20230203224534/http://www.onlive.com/) 在三月份的 GDC 上[亮相](https://web.archive.org/web/20230203224534/http://www.crunchgear.com/2009/03/25/video-demo-of-onlive/)，它在游戏世界的大部分地方掀起了类似的基于云的游戏概念的涟漪(有好有坏)。但是 OTOY 认为它有一种更轻量级和更优雅的方式来做事情，也更具可扩展性。它有几个重要的合作伙伴来证明这一点:EA 和 AMD。

让我们快速浏览一下一些细节:OTOY 是 100%基于浏览器的，works 是所有现代网络浏览器。它只需要一个宽带连接，这将为你提供 720p(高清)的图形，没有插件和下载。也有一种方法可以获得 1080p 的图像，尽管这显然要更复杂一些。但关键是，这一切都是在 OTOY 的服务器上完成的，并从云端传输到你想要的任何客户端上，基本上是即时的。

再说一次，在 Xbox 360 和 PS3 这样的巨型耗电系统是玩图形密集型游戏的唯一方式的世界里，所有这些似乎很难相信。所以对你来说，最好的方法就是看一段视频。请注意，虽然在下面的视频中，玩家使用的是 Xbox 360 控制器，但他是在连接到电视的电脑上玩的。还要注意，这个演示是从 400 英里以外的服务器上播放的。
