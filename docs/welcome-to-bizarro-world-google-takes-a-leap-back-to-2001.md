# 欢迎来到比扎罗世界——谷歌一跃回到 2001 年

[![](img/2aaae836faf59d6d32c6b93b216129e2.png)](https://web.archive.org/web/20221006213624/http://www.google.com/search2001.html)

作为其[10 岁生日庆祝](https://web.archive.org/web/20221006213624/http://www.beta.techcrunch.com/2008/09/02/happy-tenth-birthday-google-when-are-we-celebrating/)的一部分，谷歌向其搜索引擎的[过时版本](https://web.archive.org/web/20221006213624/http://www.google.com/search2001.html)敞开大门，该版本可追溯到 2001 年。该门户网站拥有旧网站的所有装饰，包括愚蠢的标志(带有感叹号)，一个缺失的“我感觉很幸运”按钮，以及不那么谦虚的声称已经索引了 1326920000 个网页。

看到事物原来的样子很酷(尽管它们并没有改变太多)，但是真正的乐趣在于运行一些搜索。谷歌的[博客文章](https://web.archive.org/web/20221006213624/http://googleblog.blogspot.com/2008/09/2001-search-odyssey.html)提供了几个例子:iPod 提到了一个文档处理系统，而 YouTube 的点击率为零。

最令人羞愧的是:搜索 TechCrunch 会得到一个链接，指向一个现已关闭的学习网站。

因为自 2001 年以来，网络已经发生了很大的变化，谷歌也加入了互联网存档回溯机的链接，其中包含了那个时期网页的缓存副本。谷歌表示，该网站将保持开放一个月——之后，它将成为历史。
