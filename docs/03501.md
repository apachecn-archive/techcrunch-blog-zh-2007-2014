# Nonoba 的“游戏网站宁”将为印度游戏平台 Zapak 提供支持

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/10/05/nonobas-ning-for-gaming-sites-to-power-indian-gaming-platform-zapak/amp/>

 <amp-img src="https://web.archive.org/web/20230304152316im_/https://techcrunch.com/wp-content/uploads/2009/10/zapak.jpg" class="amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 

今年早些时候，我们[写了](https://web.archive.org/web/20230304152316/https://techcrunch.com/2009/03/17/nonoba-enters-the-flash-games-arena-with-gamerise-a-ning-for-gaming-sites/)关于 [Nonoba 的](https://web.archive.org/web/20230304152316/http://nonoba.com/) [GameRise，](https://web.archive.org/web/20230304152316/http://nonoba.com/developers/gamerise)一个让任何人都可以创建游戏网站的“宁 for Flash games”。印度最大的休闲游戏平台， [Zapak，](https://web.archive.org/web/20230304152316/http://www.zapak.com/)现在正使用 GameRise 作为其多人游戏门户的白标解决方案。Zapak 的单人游戏将建立在一个额外的游戏网站上。Reliance 旗下的 Zapak 拥有超过 600 万注册用户，其平台上有超过 5000 款游戏。

GameRise 本质上是一个 CMS，开发和维护定制的基于 Flash 的游戏网站，提供游戏目录和社交服务，如聊天室和论坛。Nonoba 的部分盈利计划是将 GameRise 的 API 作为白标服务出售。自 2009 年 3 月推出以来，GameRise 已经为其白标产品注册了 100 多个客户。

创建一个“类似 Ning”的游戏门户网站的想法很有创意，但在 Zapak 上使用 GameRise 白牌服务的网站例子非常简单，可以使用更时尚的界面。 [Mochi Media](https://web.archive.org/web/20230304152316/http://www.mochimedia.com/) 在游戏领域做着类似的事情，但更侧重于 Flash 游戏的货币化、分发和分析。
