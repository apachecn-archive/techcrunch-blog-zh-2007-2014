从你的浏览器搜索你的云:Greplin增加了一个Chrome扩展

对于格雷普林来说，这是活跃的2011年。二月份，我们报道了社交搜索服务的公开发布，仅仅几天后，我们报道了它的关闭一轮由红杉支持的400万美元的风险投资。

自那以后，Greplin一直在迅速扩大其可以搜索和索引的社交应用，上个月在其已经健康的网络和服务中添加了Yammer、Highrise和Google Contacts。除此之外，你可以授权它搜索Dropbox、Twitter、脸书、LinkedIn、谷歌日历、谷歌文档、Evernote、Basecamp和Gmail。只需一个简单的搜索查询，砰！，Greplin几乎可以在您的整个个人云中搜索您似乎找不到的地址、联系人或邀请。

更重要的是，今天Greplin宣布在其搜索存在中添加一个Chrome扩展，这样你就可以直接从浏览器中搜索你的社交数据，而不必去Greplin的网站登录。看看右边图片中的扩展和Greplin的小部件。

Greplin采用了新的 Chrome Omnibox API ，所以你可以得到自动完成的结果，不管你在哪个网站。对于那些不熟悉Omnibox的人来说，它是Chrome搜索栏的另一种说法，它允许你在一个统一的地方输入URL和常规的旧关键字搜索，而不必去Google.com。然后，Omnibox的API允许扩展开发人员通过添加他们自己的关键字命令来使工具更加强大，这样用户就可以键入以关键字为前缀的查询，使扩展能够建议潜在的完成并对用户的输入做出反应。

该公司还宣布加入 Aaron Sarazan (之前在 Spark Plug Games )，他将致力于创建一个本地移动应用程序。联合创始人罗比·沃克(Robby Walker)告诉我，Greplin目前正在寻求增强其员工队伍，因此所有希望参与一家试图为我们提供“搜索的另一面】的年轻热门创业公司的专业工程师都应该听从这一召唤。

沃克表示，Greplin将继续增加其可搜索社交应用的列表，Basecamp和Salesforce是目前私人测试版中提供的服务之一。(接下来是Box.net和谷歌语音。)和一个火狐插件。TC的读者可以通过给support@greplin.com发电子邮件来预览以上所有内容。
