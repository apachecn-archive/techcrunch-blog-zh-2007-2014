# Cloudflare 首席执行官:“我们的营销策略是与世界上所有的国际犯罪分子签约”[TCTV]

![](img/ed279150120e2ed602699c4845d3b4cd.png "cloudflare") Disrupt [亚军](https://web.archive.org/web/20230127204353/https://techcrunch.com/2011/02/24/disrupt-cloudflare-omfg/)， [Cloudflare](https://web.archive.org/web/20230127204353/http://www.cloudflare.com/) 最近受到了很多关注，这要感谢该公司在帮助 [LulzSec 的](https://web.archive.org/web/20230127204353/https://techcrunch.com/2011/06/25/after-50-days-of-attacks-hacker-group-lulzsec-says-its-done/)网站保持在线方面发挥的作用。事实上，黑客们甚至在他们的 Twitter feed 上给了 Cloudflare 一个大喊——提供用朗姆酒换取高级账户——导致客户注册量激增。

当然，联合创始人兼首席执行官马修·普林斯(Matthew Prince)很快指出，该公司采取了一种中立的方式来托管 LulzSec，保护黑客只会让 Cloudflare 的系统对所有其他客户更具弹性。然而，对于一家承诺保护网站免受 DDOS 攻击和其他邪恶活动的公司来说，这是一个相当讽刺的转折。

我渴望了解该公司在帮助黑客和与当局共享用户数据方面的立场，于是邀请普林斯到 TCTV 演播室进行一次简短的采访(我现在在洛杉矶用 Skype，所以请原谅我糟糕的连接)。在下面的视频中，我们讨论了 Cloudflare 如何发展到一个月处理超过 50 亿次页面访问，帮助黑客的道德观，中立的重要性，一两个阴谋论，以及(更具讽刺意味的是)普林斯和他的团队是如何在接到国土安全部的电话后受到启发创办该公司的。

哦，普林斯还解释了 Cloudflare 的早期成功在很大程度上是由于在 Disrupt 上的发布——以及获得第二名(“没有什么比获得第二名更能激励工程师”)。最后，他向最终的赢家慷慨提议:“我们可以帮助他们更快地读出维基百科的文章。”

不过，我先问普林斯，联邦调查局是敲门还是直接破门而入…
