# Zimperium 为移动安全筹集了 800 万美元，扭转了攻击者的局面

假设你在机场，通过 wifi 在 iPhone 上查看邮件。你已经犯了第一个错误。您相信 airport 网络足够安全，可以保护您免受黑客攻击。

Zimperium 今天宣布，它已经从 [Sierra Ventures](https://web.archive.org/web/20221209133738/http://www.crunchbase.com/financial-organization/sierra-ventures) 筹集了 800 万美元，用于开发其移动技术，旨在保护您免受机场或任何其他地方的攻击。

该公司还为 IT 管理员提供了一个工具来监控所有注册到网络的设备。该公司现在有一个超级秘密项目正在进行中，以扭转对黑客的局面。为此，他们打算雇佣更多的黑客，这显然是有道理的。考虑到世界上最著名的黑客之一[凯文·米特尼克](https://web.archive.org/web/20221209133738/http://en.wikipedia.org/wiki/Kevin_Mitnick)是该公司的顾问委员会成员，这应该不会太难。

Zimperium 有一个名为“ [zAnti](https://web.archive.org/web/20221209133738/http://zantiapp.com/anti.html) 的移动应用程序，旨在让日常用户能够测试网络，看看他们是否有漏洞，如“[中间人](https://web.archive.org/web/20221209133738/http://en.wikipedia.org/wiki/Man-in-the-middle_attack)”攻击的危险。这些攻击劫持用户的会话，然后伪装成受信任的参与者，充当沉默的窃听者。

例如，弗兰克和维罗妮卡相互了解，相互信任。但是他们的网络被黑了，这个家伙，莱斯特，把自己放在中间，假装是维罗妮卡。弗兰克很高兴看到维罗妮卡的链接，这就是全部。莱斯特已经成功地接管了对话，并获得了维罗妮卡的 taco truck 应用程序网络的访问权限——或者莱斯特现在试图利用的她拥有的任何信息。莱斯特通过向弗兰克发送一个他知道弗兰克会被引诱打开的链接，说服他允许弗兰克访问他的网络。

Zimperium 背后的人不是开玩笑的。首席执行官 Itzhak“Zuk”AVR aham 在以色列国防军(IDF)工作了三年。所以你可以想象他在那里做了什么。

难怪 Avraham 会吹嘘这个新的绝密项目将能够阻止诸如 Stuxnet 这样的攻击，stu xnet 是一种旨在攻击核操作的计算机蠕虫。

移动—下一个前沿—是安全专家的金矿。天使名单上列出了 108 家专注于移动安全的公司。但是 Zimperium 已经证明了它的能力。现在，它必须证明这也可能是一个糟糕的行业。
