# 谷歌是否应该偏执于失去 Android 这个名字？TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/04/30/should-google-be-paranoid-about-losing-the-android-name/>

“Android”这个名字是谷歌移动计划的核心。它甚至成功地让人们不再称这些设备为“谷歌手机”或“GPhones”，这在发布之前似乎是不可能的。但是谷歌可能会面临失去这个名字的严重麻烦——或者至少不得不支付一大笔钱来保留它。

据福布斯报道，经营一家小中西部数据公司的埃里克·斯派奇特在 2002 年申请并获得了安卓名称的商标。谷歌？在 2007 年，就在安卓大规模公关宣传之前，它曾试图注册这个名字，但几个月后就被驳回了。尽管如此，谷歌仍努力争取对该名称的合法权利，但其上诉一次又一次被驳回，其商标申请显然在去年 11 月被暂停。

这意味着什么呢？嗯，Specht 因侵犯 Android 名称向谷歌(以及开放手机联盟的所有其他成员)索赔 9400 万美元。他将于下周出庭，谷歌将有 60 天时间回应他。鉴于谷歌遭到专利商标局的拒绝，这似乎是一个高价解决方案，谷歌可以继续使用这个名称，同时向 Specht 支付远低于 9400 万美元的费用。但是，事情还没有那么简单。

你看，Specht 实际上拥有“Android Data”这个名称的商标，而不仅仅是“Android”。但在文件中，它显然将“数据”描述为仅仅是一个描述性的词，这显然是 Specht 的律师在夸大其词。过去，谷歌曾试图打出“Specht 不在乎名字”这张牌，并一度解散公司，但这并没有奏效。尽管如此，如果它能证明人们清楚地知道 Android 移动软件和 Specht 的小公司之间的区别，谷歌可能有机会让这个案子被驳回。

其他需要考虑的事情:也许谷歌真的很幸运，现在市场上的安卓手机没有它喜欢的那么多。如果真的达成和解，他们可以省下一大笔钱。

但不管怎样，就在谷歌试图通过几十种新设备来提升 Android 平台，以及即将推出的新的 1.5 Android 软件更新时，谷歌不会对现在不得不处理这个问题感到高兴。实际上，这一切都有点似曾相识。如果你还记得 2007 年，就在苹果公司发布 iPhone 之后，思科公司对其提起诉讼，思科公司拥有该名称的权利。双方很快[达成和解](https://web.archive.org/web/20221208223051/http://www.msnbc.msn.com/id/17268277/)——但那是两家大公司，不是一家大公司和一个小人物。

记住谷歌，[不要作恶](https://web.archive.org/web/20221208223051/http://en.wikipedia.org/wiki/Don%27t_be_evil)。
