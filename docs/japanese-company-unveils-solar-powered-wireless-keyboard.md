# 日本公司推出太阳能无线键盘

[![](img/fd3648159881362351342719287e2e9a.png "buffalo_keyboard")](https://web.archive.org/web/20230304004642/http://old.crunchgear.com/wp-content/uploads/2008/07/buffalo_keyboard.jpg)

日本似乎一天比一天更绿了。现在布法罗宣布他们将从下周开始销售一款由太阳能供电的[无线键盘](https://web.archive.org/web/20230304004642/http://buffalo-kokuyo.jp/products/new/129/)【JP】。BSKBW01SB 的定价为 120 美元。

2.4GHz 无线设备的特点是在数字键盘块上方有一个小型太阳能电池板，也可以使用 2 节 AAA 电池供电。
