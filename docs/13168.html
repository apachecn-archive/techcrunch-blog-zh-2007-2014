微软推出面向政府的Office 365

几周前，谷歌赢得了对微软的重要胜利，赢得了T2 3500万美元的美国政府合同，为内政部提供基于云的办公解决方案。当然，微软的传统解决方案仍然是政府办公室的主要产品，但随着越来越多的机构希望将他们的生产力和协作服务转移到云，微软正在冒着失去这个利润丰厚的市场的风险。然而今天，该公司推出了一项新服务，应该会给更多的政府客户提供转向云的选择，这些客户往往有非常严格的数据安全和隐私规定。用微软自己的话说，微软新推出的Office 365 for Government是“一种新的多租户服务，将美国政府数据存储在一个隔离的社区云中。”

谷歌上周宣传了其谷歌企业应用套件的ISO 27001认证，Office 365 for Government也有资格获得该认证。就像它的前身商业生产力在线套件联邦，微软的新服务也支持过多的其他认证，包括SAS70 Type II，美国健康保险便携性，责任法案(HIPAA)和美国联邦信息安全管理法案(FISMA) 。微软还计划很快支持刑事司法信息安全政策。这项服务也将很快提供对IPv6的支持。

微软的企业解决方案和这种政府云的主要区别在于，政府数据存储在自己独立的基础设施上。除此之外——以及额外的认证——微软的政府解决方案包括与企业版几乎相同的服务，包括Exchange Online、Lync Online、SharePoint Online和Office Professional Plus。鉴于微软的企业解决方案现在也通过了FISMA认证，这项新服务主要是为那些有着超出这一认证要求的机构提供的。
