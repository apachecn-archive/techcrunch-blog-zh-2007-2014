# CloudFlare 透露来自 Union Square Ventures 的 5000 万美元融资 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2013/12/17/cloudflare-reveals-50m-round-from-union-square-ventures/>

据首席执行官兼联合创始人马修·普林斯(Matthew Prince)称， [CloudFlare](https://web.archive.org/web/20230128111351/http://www.crunchbase.com/company/cloudflare) 透露，它在 2012 年 12 月筹集了 5000 万美元的 C 轮融资，这很可能是在筹集新一轮融资之前提高估值的一个举措。

如果你对 CloudFlare 不熟悉，这是一项为网站所有者提供的服务，它提供了对在线威胁的保护，加快了页面加载时间，并优化了跨设备的内容。该公司于 2009 年成立，此后共筹集了 7210 万美元，包括由 Pelion Venture Partners 和 Venrock 牵头的 205 万美元 A 轮融资，以及由现有投资者和 New Enterprise Associates 共同投资的 2000 万美元 B 轮融资。

那么这一轮 5000 万美元是从哪里来的呢？

普林斯透露，大约一年前，联合广场风险投资公司联合现有投资者向该公司注资 5000 万美元。

在此期间，CloudFlare 收到了许多条款清单，其中一些的估值高达 12 亿美元。最终，CloudFlare 选择了 Union Square Ventures 的报价，因为 [Brad Burnham](https://web.archive.org/web/20230128111351/http://www.crunchbase.com/person/brad-burnham) ，该报价较低(但未披露)。大约在那时，我们还报道了 CloudFlare 正在以近 10 亿美元的估值对[进行融资。](https://web.archive.org/web/20230128111351/https://techcrunch.com/2012/04/17/cloudflare-billion-valuation/)

故事是这样的，Burnham 和 CloudFlare 的共同创始人 Michelle Zatlyn 是 FCC 网络中立委员会的成员。在他们相互了解之后，伯纳姆证明了他在 CloudFlare 潜在的网络中立性相关障碍中可能是一笔宝贵的资产，普林斯和扎特林得出结论，他们希望 USV 和伯纳姆成为投资者。

虽然伯纳姆不会加入董事会，但他将从更远的地方观察并帮助提供建议。

自去年获得资金以来，CloudFlare 的用户群已经翻了一番，达到 150 万，占互联网上 web 请求的 5%。事实上，现在每 20 个站点中就有一个在 CloudFlare 上。

但并不都是阳光和玫瑰。CloudFlare 是互联网历史上最大的 DDoS 攻击之一[的中心](https://web.archive.org/web/20230128111351/http://blog.cloudflare.com/the-ddos-that-almost-broke-the-internet)，这对该公司来说是一个动荡的时期。

尽管如此，普林斯透露，自去年以来，收入增长了 460%，成本低于预期，订阅收入比预期高 10%或 15%。

普林斯表示:“我们面临着明年 1 月实现盈利的真正风险。”。

当然，他不会透露实际数字，但他确实说过，该公司目前的估值远远高于一年前的 12 亿美元。

这就引出了我的下一个问题:为什么 CloudFlare 现在要共享所有这些信息？

Prince 的官方回答和展示你的作品有关。

普林斯说:“硅谷的风气是，当你筹集资金时，首先要使劲捶胸，但这不是我们的风气。”。“通常，你筹集资金是基于你将要做什么的承诺，而不是你已经做了什么。我们想用不同的方式来做。”

这也不是第一次 [CloudFlare 发布最新消息](https://web.archive.org/web/20230128111351/https://techcrunch.com/2011/07/12/oh-by-the-way-cloudflare-raised-20-million-last-november/)。

当然，更现实的答案与 CloudFlare 的下一步发展有关。一家公司会在一年后披露融资情况，原因有很多。也许他们试图吸引新的人才或增加可信度，同时获得更多的用户。但考虑到故事发生的时间，这说不通。

另一方面，该公司可能计划在未来几个月进行另一轮大规模融资，并寻求提高估值。事实上，CloudFlare 未来考虑 IPO 也不是不可能。

就我个人而言，我的钱是在另一个大的融资轮，但只有时间会告诉我们。
