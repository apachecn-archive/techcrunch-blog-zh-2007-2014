# 红杉资本合伙人马克·克瓦梅将执掌俄亥俄州的发展部

![](img/c65bdad3143b13d585893f2e89e7e651.png) [红杉资本](https://web.archive.org/web/20230202231809/http://www.crunchbase.com/financial-organization/sequoia-capital)合伙人[马克·克瓦梅](https://web.archive.org/web/20230202231809/http://www.crunchbase.com/person/mark-kvamme)将全职离开公司，出任俄亥俄州发展部主任，在那里他将专注于推动俄亥俄州的经济增长。他被即将上任的州长约翰·卡西奇任命。

Kvamme 于 1999 年加入红杉，并领导了 LinkedIn、MarkLogic、Funny or Die、StrongMail 和 CastIron Systems(被 IBM 收购)的早期投资。在加入红杉之前，他是广告公司 CKS 集团的首席执行官。

Kvamme 说，他对推动经济增长有很好的想法，并指出今天政府中没有多少人具有良好的商业头脑。“数十亿美元的决策导致了不可持续的解决方案，”他昨日在电话中表示。“我想让俄亥俄州回到正轨，并推动人们和企业前往该州。”

他说，他有一个具体的变革蓝图，将在未来几周内与大家分享。红杉资本任命他为他们的第一位“特殊有限合伙人”，他将继续在公司担任有限的职位。

Kvamme 也是早期的冠军 [UJAM](https://web.archive.org/web/20230202231809/http://www.ujam.com/) ，[在 TechCrunch Disrupt:New York startup 战地竞赛中获得第二名](https://web.archive.org/web/20230202231809/https://techcrunch.com/2010/05/26/techcrunch-disrupt-winner-soluto/)。
