# 基准合伙人彼得·芬顿谈投资者运气、科技 IPO 等

在过去的一周里，我们有幸在 TechCrunch Europe 的后台与基准合伙人兼 Twitter 董事会成员[彼得·芬顿](https://web.archive.org/web/20230128194603/http://www.crunchbase.com/person/peter-fenton)坐在一起，谈论他的神奇魅力。

在硅谷的所有投资者中，芬顿是精英风投之一，拥有最多正在上市或将于 2014 年上市的公司。这个群体包括 Twitter、Zendesk、Zuora、New Relic 和其他一些网站。我们问芬顿，他挑选有实力上市的公司的秘诀是什么。

芬顿承认，运气在他的一些成功和赌注中发挥了重要作用。他分享说，他爱上了他投资的公司，这种激情来自于看到一家以使命为基础的公司的活力以及员工如何与初创公司一起成长。他补充称，目前这种环境下的动态是关注首次公开募股(IPO)还是退出(他表示，M&A 在上一代科技公司中更受欢迎)。
