# Salesforce.com 将提供 DaaS 服务，新的定价模式，竞争

 [](https://web.archive.org/web/20230129101611/http://www.force.com/) CRM 和 SaaS 提供商[Salesforce.com](https://web.archive.org/web/20230129101611/http://www.salesforce.com/)宣布，Force.com 云计算架构(我们的评论[在此](https://web.archive.org/web/20230129101611/http://techcrunch.com/2007/09/13/salesforce-enters-custom-application-market-with-forcecom/))现在将提供开发即服务(DaaS)、新的定价结构和开发者竞赛。

DaaS 服务由一组新的开发工具和 API 组成，允许企业开发人员利用云计算。这些工具提供对 Force.com 平台的数据库、逻辑和用户界面功能的全面访问，将开发和 IT 协作工具与 Force.com 平台即服务相统一。新服务包括元数据 API、集成开发环境(IDE)、沙箱和代码共享，所有开发人员都可以使用它们来构建企业软件即服务应用程序。

新的定价模式包括针对 Force.com 平台和开发即服务的按登录付费的实用程序定价模式。这种模式为那些不经常使用云应用的公司提供了一种更便宜的选择，理论上使服务更实惠。Force.com 云(每次登录)的标价为每次登录 5.00 美元，每个用户每月最多登录 5 次，并将在 2008 年底以每次登录 0.99 美元的价格提供。对于更频繁的用户(每月超过 5 次登录)，必须注册 Force.com 的每个用户每月 50 美元的无限定价计划。

Salesforce.com 和 Emergence Capital Partners 还宣布了一个新的比赛，Force.com 100 万美元挑战赛，这是一个为建立在 Force.com 平台上的企业家和公司举办的创业比赛。获胜者将获得 Emergence Capital 的 100 万美元投资，以及 Salesforce.com 的 AppExchange 孵化器设施为期一年的空间。获胜者将在 11 月的 Dreamforce 2008 上宣布。
