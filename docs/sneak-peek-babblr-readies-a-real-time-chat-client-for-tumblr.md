# 先睹为快:Babblr 为 Tumblr TechCrunch 准备了一个实时聊天客户端

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/05/sneak-peek-babblr-readies-a-real-time-chat-client-for-tumblr/>

如果说 [Tumblr](https://web.archive.org/web/20221207055342/http://www.tumblr.com/) 比博客平台更像是一个社交网络[，那么](https://web.archive.org/web/20221207055342/https://beta.techcrunch.com/2013/01/24/with-todays-update-tumblr-starts-to-look-more-like-a-fully-featured-twitter-than-blogging-platform/) [Babblr](https://web.archive.org/web/20221207055342/http://babblr.me/) 可能正是它的用户所渴望的:它是一个平台的集成聊天客户端，让你可以与 Tumblr 的朋友和粉丝交流，并将你的社区组织成小组。

这个想法目前处于测试阶段，来自目前在文凤仙雨社会公益电子商务公司工作的几个人。他们自己都是 Tumblr 的活跃用户，他们创建 Babblr 是因为他们对 Tumblr 和社交媒体的热情。

据 Babblr 联合创始人布兰登·索沃斯(Brandon Sowers)称，他和 Babblr 的其他创始人布雷特·威廉姆斯和特雷弗·克拉克(Trevor Clarke)在 Tumblr 上共有 70 万名粉丝。“我们一直在寻找新的、创新的方式来与我们的粉丝沟通，”Sowers 说。

如今，Tumblr 的社交功能更加轻量级——其用户可以关注他人的博客、喜欢的帖子，并在添加自己的 2 美分的情况下转发博客内容，但用户之间的直接交流是不可能的。

“人们希望与他们的追随者实时交流，”索沃斯解释说，“这是一个很好的工具。”

Babblr 不是过于复杂的软件。事实上，它与其他现场聊天客户端没有太大区别。不同的是，它建立在 Tumblr API 的基础上，通过几周内推出的 Chrome 扩展，它将直接集成到您的 Tumblr 仪表板中。

[![babblr](img/02c501ccb12c8e2eb838d5f34e4c85ea.png)](https://web.archive.org/web/20221207055342/https://beta.techcrunch.com/2013/04/05/sneak-peek-babblr-readies-a-real-time-chat-client-for-tumblr/babblr/)

通过 Tumblr 认证后，Babblr 界面会向您显示关注者和您正在关注的人的列表，并允许您在 Babblr 上发起类似即时消息的聊天。您还可以将您的关注者组织成组，这对于拥有较大社区的人或只想进行更直接对话的人来说非常有用。

这款短信应用的设计也是对家庭友好的，因为它会将脏话转化为愚蠢或无意义的短语，如“胡说八道”或“胡说八道”。(分别是 F 字和 S 字。)

Babblr 团队已经为这个客户端工作了大约六个月，从两个月前开始，他们允许人们通过搜索和其他发布的链接自行发现它，以便进行试用。截至目前，Babblr 已有 25000 名用户。事实上，威廉姆斯在他自己的 Tumblr 博客上发布的一篇取笑客户发展的帖子最终有超过 20 万条“注释”

“那天我们收到了 25，000 封电子邮件，他们想知道当时的聊天内容，”索尔斯说。“当你看到这样的东西时，这是相当疯狂的，”他补充道。

现在有 25，000 名用户建立了 Babblr 账户，其中大多数是女性，年龄在 16 至 24 岁之间，这支持了 Tumblr 在年轻人群中受欢迎的报道。大约 80%的注册用户也来自美国。

今天 Babblr 的“发布”与其说是公开亮相，不如说是先睹为快。该客户端处于测试阶段，在 Chrome 扩展完成之前不会出现在 Tumblr 仪表盘上。但是 Sowers 说这部分已经基本完成了。

当 Babblr 在几周内推出时，它将是免费下载的，但计划是在消息传开后不久开始收取 99 美分的下载费。

索沃斯说，三位联合创始人仍在文凤仙雨工作，用他们自己的“激情、汗水和泪水”来启动这个项目。
