# FatSecret 希望通过新的 API 成为营养数据的中心

![](img/f8d9c026c9740aa841465e21253c326a.png)[【fat secret】](https://web.archive.org/web/20221209115226/http://www.fatsecret.com/)，一个专注于营养和减肥的社交网络，我们[早在 2007 年就报道过](https://web.archive.org/web/20221209115226/http://www.beta.techcrunch.com/2007/04/29/fatsecret-for-fat-people-who-want-to-be-less-so/)，今晚将推出一个新的 API，允许第三方网站和服务接入其营养数据、锻炼信息和其他健康统计数据的数据库。你可以在这里访问新的 FatSecret 平台[。](https://web.archive.org/web/20221209115226/http://platform.fatsecret.com/)

首席执行官 Rodney Moses 说 FatSecret 允许开发者免费访问 API，希望将 FatSecret 变成可靠和准确的营养信息资源。他指出，尽管网上有很多包含各种食物营养信息的饮食网站，但大部分数据是杂乱无章的——没有人们首先求助的公认的综合来源。FatSecret 希望成为这类信息的权威枢纽。该网站已经从美国农业部等许多公开资源中收集了数据，也有许多用户在其社交网络上提交的条目。Moses 说，所有的数据都经过了筛选，以确保准确性。

新 API 的另一个组件是一个品牌实用程序，它邀请食品和饮料品牌所有者将他们的营养事实提交到系统中，以便可以使用 FatSecret API 检索它们。

摩西说，网站本身仍在稳步增长，每月有 50 万访客，如果包括通过移动应用程序等其他方式访问网站的用户，这个数字将会翻一番。
