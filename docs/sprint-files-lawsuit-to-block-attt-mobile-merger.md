# Sprint 提起诉讼阻止美国电话电报公司/T-Mobile 合并

Sprint 显然不喜欢美国电话电报公司/T-mobile 即将进行的合并:在最近的记忆中，他们已经承诺向国会投诉，委托一项研究向 T2 说明这将会有多糟糕，T4 为司法部叫好，因为他们提起了反垄断诉讼以终止诉讼。

现在，Sprint 不再只是在一旁冷嘲热讽，而是对形势采取更加积极的姿态。他们的第一步是:对美国电话电报公司、T-Mobile 和德国电信提起诉讼。

在司法部的法律操纵下，Sprint 最近提起诉讼，声称美国电话电报公司/T-Mobile 的潜在合并违反了《克雷顿反垄断法》第 7 条。Sprint 声称，如果允许的话，这一合并将(除其他外)提高价格，扼杀创新，进一步巩固威瑞森和美国电话电报公司的权力，并损害 Sprint 和其他较小的运营商。

Sprint 的诉讼副总裁苏珊·z·哈勒(Susan Z. Haller)表示，他们“正在继续代表消费者和竞争进行宣传”——这是一个值得称赞的目标，但却充满了公关手段的味道。虽然他们对合并对市场的潜在影响提出了一些严肃的观点，但 Sprint 确实在为生存而战。

他们在这里打得很好:诉讼是一个精心计算的，适时的举动，你可以打赌 Sprint 将继续骑他们的消费者友好的高马，只要他们能。Sprint 将继续强调他们的英雄角度，希望无论这一切如何结束，潜在客户将记住他们在这一切中的角色。
