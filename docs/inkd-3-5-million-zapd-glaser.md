# Pressplane 为移动网络出版应用 Zapd 筹集 350 万美元，Rob Glaser 加入董事会

![](img/4586c83eaa586161001ba77b2b7d512c.png)

西雅图初创公司 Pressplane 刚刚获得了由 Madrona Ventures 领投的 350 万美元新一轮融资。[罗伯·格拉泽](https://web.archive.org/web/20230203040801/http://www.crunchbase.com/person/rob-glaser)，RealNetworks 的创始人，现在是 Accel 的合伙人，他也进行了个人投资，并加入了董事会。第二大道合伙人兼董事长[米卡·萨尔米](https://web.archive.org/web/20230203040801/http://www.crunchbase.com/person/mika-salmi)(前 MTV 网络和 Atom 娱乐公司)也参加了。

这笔钱将用于进一步开发 Zapd，这是一个移动发布平台，让人们可以直接从他们的 iPhones 上创建好看的定制网站。(你可以在*的[这一集](https://web.archive.org/web/20230203040801/https://techcrunch.com/2011/04/28/fly-die-tweetdeck-iphone/)中看到它的作用。自从三月份发布以来，Zapd 已经被下载了 320，000 次，几乎没有什么营销活动。*

“我们认为是时候重新思考网站了。它们是如何制造的。他们是如何消费的，”首席执行官凯莉·史密斯说。他表示，新的资金将用于增加更多功能，如实时主题、互动小工具和社交网络挂钩。Pressplane 的另一项产品是 Inkd，这是一项定制印刷、标识和设计服务。
