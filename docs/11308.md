# Zynga 的最新游戏《城堡》(CastleVille)结合了丰富的故事情节、幻想和中世纪世界的社交游戏 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/03/zyngas-newest-title-castleville-combines-rich-storylines-fantasy-and-social-gaming-in-a-medieval-world/amp/>

正如 Zynga [在几周前的一次活动中宣布的](https://web.archive.org/web/20230209103040/https://techcrunch.com/2011/10/11/zynga-announces-newest-game-in-the-ville-franchise-castleville/)，包括 FarmVille、FrontierVille 和 CityVille 在内的“Ville”系列的最新名称是 [CastleVille](https://web.archive.org/web/20230209103040/http://www.facebook.com/castleville) 。来自社交游戏巨头的最新游戏允许玩家从城堡、工艺美术、盔甲和贸易中建立迷你帝国，并在中世纪世界中保卫你的王国。这款游戏由 Zynga 达拉斯团队开发，该团队是社交游戏巨头收购 Bonfire Studios 的一部分。

Zynga 达拉斯总监比尔·杰克逊(Bill Jackson)解释说，工作室采用了每个“Ville”名称的关键元素，并将最佳元素带入了 CastleVille。他解释说，CastleVille 结合了 FarmVille 的自我表达元素，用 CityVille 构建梦想中的幻想城市的能力，以及 FrontierVille 丰富的叙事和讲故事功能。这个游戏也在尝试社交和自我表达的新方式。

当第一次开始游戏时，你需要选择你的角色和建立你的外观。杰克逊说，这是迄今为止最具扩展性的角色构建功能，有几十种外观可供选择。一旦沉浸在 CastleVille 的中世纪世界中，农民、公主、海盗(和玩家)将通过遇见新的和不同的角色、完成任务和探索神秘区域来选择自己在游戏中的道路。

游戏的基础是建造你梦想中的城堡，将未知的土地变成你自己的王国。一路上，你会遇到许多角色，他们都参与到游戏的故事情节中。你首先会看到杜克，他会帮你建造城堡，他来自邻近的王国。有伊维特(美丽的女歌手)，吉赛尔(可爱的少女)，乔治(友好的乡村海盗)，索尼娅(性感的海盗)，拉斐尔(花花公子樵夫)和汤姆(粗暴的屠龙者)。

正如杰克逊所说，这款游戏拥有最密集的故事情节，这也是为什么会有这么多角色的原因。Zynga 还希望你感觉自己完全沉浸在这个世界中，融入幻想和冒险的元素。“你必须自己努力，成为游戏成功冒险的一部分，”他解释道。你选择自己的道路去发现神秘的地点，寻找稀有物品，并揭示游戏中隐藏的角色。

另一个使王国更加真实的特点是，这是迄今为止最丰富的图形，最具表现力的游戏。你将会看到像鱼在池塘里跳跃，树摇动，和其他与游戏动作相关的音效，

 <amp-img class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" src="https://web.archive.org/web/20230209103040im_/https://techcrunch.com/wp-content/uploads/2011/11/cat.png" alt="" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 在社交方面，你可以访问朋友的王国，赠送他们艺术品、盔甲和药剂，帮助完成任务，挑战黑暗势力。还有一个声誉系统，根据你在游戏中的社交程度给你更多的分数。基本上，你的社交越多，你获得的声誉点就越多，你在游戏中的进步就越快。显然，这给了你更多与朋友互动和购买虚拟商品的动力。杰克逊说，这种简化的社交体验使与朋友一起玩游戏变得更容易、更有趣。

CastleVille 将在未来几周内免费播放 17 种语言(英语、法语、意大利语、德语、西班牙语、葡萄牙语、印度尼西亚语、土耳其语、繁体中文、韩语、马来语、挪威语、瑞典语、丹麦语、荷兰语、日语和泰语)。

过去几个月，Zynga 一直在大量生产游戏。公司推出了[黑手党战争 2](https://web.archive.org/web/20230209103040/https://techcrunch.com/2011/10/03/zynga-goes-3d-with-crime-world-facebook-game-mafia-wars-2-launches-in-a-record-16-languages/) 、[冒险世界](https://web.archive.org/web/20230209103040/https://techcrunch.com/2011/09/01/zynga-debuts-largest-most-feature-rich-multi-dimensional-facebook-game-to-date-adventure-world/)、[拓荒者小道、](https://web.archive.org/web/20230209103040/https://techcrunch.com/2011/08/12/after-frontierville-zynga-channels-the-wild-west-again-for-facebook-game-the-pioneer-trail/)、[帝国&盟友](https://web.archive.org/web/20230209103040/https://techcrunch.com/2011/05/31/zynga-channels-settlers-of-catan-in-new-social-combat-game-empires-allies-debuts-in-12-languages/)、[与朋友厮混。](https://web.archive.org/web/20230209103040/https://techcrunch.com/2011/06/06/zynga-turns-hangman-into-a-social-ios-game-with-the-debut-of-hanging-with-friends/)

我们知道开发新的热门游戏是 Zynga 商业模式的一个重要组成部分，而且由于 2011 年上半年没有任何新游戏推出，今年 Q2 的利润有所下降。我们还没有看到第三季度的数据，所以很难判断这些新产品是否真的对 Zynga 的收入产生了有意义的影响。

根据 Inside Network 的 [AppData 排行榜，](https://web.archive.org/web/20230209103040/http://www.appdata.com/)Empires&alliances 似乎是所有新游戏中做得最好的，拥有 1980 万月活跃用户。其次是黑手党战争 2，用户数为 1370 万。当然，CityVille 和 FarmVille 是 Zynga 的顶级游戏，分别拥有 5420 万和 2820 万月活跃用户。也许下一个名列榜首的城市将是卡斯特利维尔。
