# 独立移动开发者从 Chartboost，Execution Labs 获得新的交叉推广服务

总部位于蒙特利尔的游戏孵化器 Execution Labs 正与移动游戏广告和盈利初创公司 Chartboost 合作，创建一个新的配对程序[，该程序将帮助独立游戏开发者推广彼此的作品。](https://web.archive.org/web/20221005183057/http://executionlabs.com/en/lab-partners/)

它将允许较小的开发商利用交叉推广，这种策略已经帮助较大的游戏开发商在从一个游戏标题穿梭到下一个游戏标题时留住他们的数百万玩家。Zynga 和 Rovio 等大型游戏开发商通常会用旧游戏来推广他们的新游戏。这使得他们为一个新游戏获得数百万用户的成本更低。

与此同时，对于全新的游戏开发商来说，打入市场变得比以往任何时候都更加困难。昨天， [Distimo 的一份报告显示，iPhone 应用商店前 250 名发行商](https://web.archive.org/web/20221005183057/https://beta.techcrunch.com/2013/02/20/the-app-stores-are-getting-full-only-2-of-iphone-top-publishers-in-u-s-are-newcomers-3-on-google-play/)中只有 2%是“新人”Android 也好不到哪里去，在 Android 应用商店 Google Play 中，只有 3%的人有资格成为“新人”。

这是你在软件平台上看到的典型转变，因为先行者利用较低的营销成本来获得影响力，并排挤后来者。

Execution Labs 称这种配对服务为“实验室伙伴”项目。它将帮助独立手机游戏开发商找到类似的开发商，并与他们建立广告交易。他们将能够通过平台和类型搜索其他游戏来寻找潜在的合作伙伴。因此，如果玩家进入一家开发商的游戏，他们会看到另一家工作室类似游戏的广告。

这项服务是免费的，由 Chartboost 的 Direct Deals 平台提供支持。Chartboost 是由一些前 Tapulous 员工创建的，最初是通过促进中型和大型移动开发商之间的直接广告交易起家的。

随着时间的推移，它发展成为一个以游戏为中心的移动广告网络，现在涉及约 12，000 种产品。该公司的收入增长非常快，今年早些时候，这家初创公司吸引了红杉资本(Sequoia Capital)领投的 1900 万美元融资。

现在，Chartboost 将自己设想为游戏的商业引擎。虽然该公司尚未分享其未来一年的产品路线图，但不难想象它以后会开发出保留和货币化的工具。

至于 Execution Labs，它是一家总部位于蒙特利尔的孵化器，由 BDC Venture Capital、Real Ventures 和 White Star Capital 支持。
