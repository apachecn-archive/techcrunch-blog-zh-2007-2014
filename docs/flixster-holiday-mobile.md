# App-y 假期:iOS 狼吞虎咽，Android 吃饱了，RIM 吃剩了，Windows Phone 挨饿了

已经有大量数据表明苹果度过了一个疯狂的假期。几周后他们宣布收益时，我们就能确定了。不过，就目前而言，我们必须根据第三方开发者看到的应用下载量来判断。一家名为 [Flixster](https://web.archive.org/web/20221007034413/http://flixster.com/) 的公司给了我们一些关键细节，告诉我们他们的手机安装从圣诞节到新年是如何分解的。

Flixster 能够在短短一周内达到一个重要的里程碑:移动设备上的安装量刚刚超过 100 万。在本周的 1，027，000 次安装中，51%是在 iPhone 上，30.1%是在 Android 设备上，10.5%是在 iPad 上。7.9%使用 RIM，0.5%使用 Windows Phone。

换句话说，如果你把这两个 iOS 设备加在一起，苹果吞噬了 Flixster 61.5%的下载量。与此同时，安卓系统以 30.1%的份额令人愉快地增长。RIM 拿到了剩菜。而 Windows Phone 饿了。后者的 0.5%相当于假期的 5000 次安装。换句话说，Windows Phone 在整个期间的总安装量与苹果每 90 分钟的安装量相同。甚至 RIM 每天都有 10，000 次安装*。*

 *显然，考虑到 Windows Phone 是一个新平台，而 iOS 等其他平台已经存在了几年，对所有平台进行比较并不完全公平。尽管如此，在竞争对手看到 100 倍的安装量时，假日期间的 5000 安装量显然不太好。

Flixster 还指出，他们的 iPad 应用程序昨天的用户总数达到了 100 万。联合创始人乔·格林斯坦说，这是第九个达到这一里程碑的 Flixster 平台。其他的是 Flixster.com、RottenTomatoes.com，以及他们为脸书、MySpace、iGoogle、iPhone、Android 和黑莓开发的应用。

"*今年圣诞节，我们在四个不同的移动平台上平均每天安装超过 10，000 次。格林斯坦说:“看看我们预计在 2011 年表现良好的设备，明年这个数字很可能是 7 甚至 8。**
