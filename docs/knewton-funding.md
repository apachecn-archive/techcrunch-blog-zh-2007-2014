# 教育技术公司 Knewton 多赚了 1250 万美元

在线教育科技初创公司 [Knewton](https://web.archive.org/web/20230310025753/http://www.knewton.com/) 完成了由纽约市风险投资公司 [FirstMark Capital](https://web.archive.org/web/20230310025753/http://www.crunchbase.com/financial-organization/firstmark-capital) 牵头的[1250 万美元的一轮融资](https://web.archive.org/web/20230310025753/http://www.prnewswire.com/news-releases/knewton-announces-completion-of-125-million-financing-led-by-firstmark-capital-91460439.html)，一年多前获得了[600 万美元](https://web.archive.org/web/20230310025753/https://techcrunch.com/2009/04/07/knewton-bags-6-million-series-b-round-for-adaptive-learning-platform/)B 轮融资。

回归的投资者包括 [Accel Partners](https://web.archive.org/web/20230310025753/http://www.crunchbase.com/financial-organization/accel-partners) 、[Bessemer Venture Partners](https://web.archive.org/web/20230310025753/http://www.crunchbase.com/financial-organization/bessemer-venture-partners)、[首轮资本](https://web.archive.org/web/20230310025753/http://www.crunchbase.com/financial-organization/first-round-capital)和天使投资人[雷德·霍夫曼](https://web.archive.org/web/20230310025753/http://www.crunchbase.com/person/reid-hoffman)，他们通过这第三轮将注入公司的资本总额提高到[近 2000 万美元](https://web.archive.org/web/20230310025753/http://www.crunchbase.com/company/knewton)。

Knewton 表示，它将利用新的融资来加速扩张，以满足对其自适应学习引擎的需求，该引擎使用专有算法来识别和解决每个学生的独特需求。该公司的平台几乎可以跟踪每个学生的每一个动作，并根据学生在哪些概念上最薄弱，相应地对每天的学习概念进行优先排序。

随后，Knewton 系统以最适合个别学生的任何学习形式传递这些概念，如一口大小、媒体格式、一天中的时间以及伴随练习题的数量和难度。

当 Erick Schonfeld [在 2008 年末回顾这项服务](https://web.archive.org/web/20230310025753/https://techcrunch.com/2008/12/23/knewton-takes-adaptive-learning-to-the-next-level/)时，他写道:

> 适应性学习测试在计算机上进行。根据每个学生的回答，问题会变得越来越难或越来越容易。因此，他们适应每个学生的知识和能力。Knewton 采用了适应性学习的概念，并首先将其应用于在线备考服务。
> 
> …
> 
> 该服务结合了在白板环境中与讲师的实时视频聊天，以及自学的示例问题和教程。Knewton 找到了它能找到的最好的老师，每小时付给他们 500 到 800 美元。除了虚拟教室，Knewton 还跟踪每个学生在掌握每次考试中可能涉及的一千多个概念方面的进展。一个“概念队列”让学生们及时了解他们已经掌握了哪些概念，哪些概念是他们的弱项。他们可以点击每个概念标签深入了解。

Knewton 已经向市场推出了一套自适应学习备考产品(T2 刚刚推出了一款用于 SAT 考试的产品)，并表示它现在正致力于开放平台供任何人使用。

Knewton 面临着来自像 [Brightstorm](https://web.archive.org/web/20230310025753/http://www.brightstorm.com/) 和 [Grockit](https://web.archive.org/web/20230310025753/http://grockit.com/) 这样公司的竞争。
