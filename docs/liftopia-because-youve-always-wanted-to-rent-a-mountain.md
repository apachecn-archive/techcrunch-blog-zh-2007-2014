# Liftopia:因为你一直想租一座山 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/11/04/liftopia-because-youve-always-wanted-to-rent-a-mountain/>

终于，我有了一种方法，可以用我的披萨楔子转弯来切开斜坡，而不必忍受陌生人居高临下的假笑。

这是因为初创公司 Liftopia 为你提供滑雪门票和其他与雪有关的活动的大幅折扣，它刚刚增加了一个新功能:全山租赁。这意味着，如果你愿意花足够的钱，你可以独自去滑雪，或者，如果你想走更务实的路线，你可以带 249 个朋友或同事一起去。

根据度假村的不同，你可以租下一整座山，起价 3000 美元。如果你不需要容纳这么多人，一些度假村会让你出租其中的一部分(一个场馆甚至会让你租一个专用的管道山)。Liftopia 将该功能定位为公司外部或俱乐部下雪天的一个好主意，这可能比我孤独的披萨-韦奇幻想更有意义。

到目前为止，可用的租赁项目包括普拉特基尔山度假村(纽约)、亚伯兰山(缅因州)、惠勒贝克山(新罕布什尔州)和卡姆登雪球(缅因州)，更多项目将很快推出。

Liftopia，对于那些没有使用过它的人来说，让你提前在线订购滑雪缆车票，通常有很大的折扣(在一些度假村，你可以节省高达 80%的免门票费用)。这在很大程度上是一个双赢的局面:消费者省钱，度假村可以调整价格以适应需求，因此他们的休息日更少。

该公司早在 2006 年就成立了，但由于最初的行业怀疑，直到 2008/09 年事情才开始[起飞](https://web.archive.org/web/20221127052221/https://beta.techcrunch.com/2009/03/10/liftopias-ski-lift-ticketing-system-begins-to-snowball/)。上个月，[完成了一轮 130 万美元的融资，现在名单上全是知名的滑雪场。事实上，该公司今天还宣布，它已将四个著名的阿斯彭/斯诺马斯山脉加入其目录。](https://web.archive.org/web/20221127052221/https://beta.techcrunch.com/2011/10/19/ski-lift-ticket-retailer-liftopia-raises-1-3-million/)
