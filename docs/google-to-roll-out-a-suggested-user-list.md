# Google+推出了一个类似 Twitter 的推荐用户列表

谷歌的[布拉德利·霍洛维茨](https://web.archive.org/web/20230204141444/http://www.crunchbase.com/person/bradley-horowitz)刚刚[在推特](https://web.archive.org/web/20230204141444/http://twitter.com/#!/elatable/status/109695085951729664)上发布了一条消息，表明谷歌即将向 Google+添加一个类似 Twitter 的建议用户列表。来自推文:*我们即将在 Google+上试行类似“建议用户”的机制。如果你在 Twitter 上有超过 10 万名粉丝，那就给我发信息——让我们聊聊吧！*正如消息中所说，Horowitz 正在寻找在 Twitter 上拥有超过 100，000 名粉丝的用户。

除了 Horowitz 写的内容，没有太多关于 Google+将如何决定谁会上榜的细节。Twitter 的[建议用户名单](https://web.archive.org/web/20230204141444/https://techcrunch.com/2010/01/22/twitter-sul-stats-2/)一直备受争议[因为名单上的人每天肯定会获得数千粉丝，而那些被遗漏的人认为这不公平。去年，Twitter](https://web.archive.org/web/20230204141444/https://techcrunch.com/2009/10/20/web-2-0-summit-a-conversation-with-twitters-ev-williams/)[修改了](https://web.archive.org/web/20230204141444/https://techcrunch.com/2010/01/21/twitter-sul-suggestions/)列表，现在[也允许用户](https://web.archive.org/web/20230204141444/https://techcrunch.com/2011/06/14/twitters-follow-recommendations-lets-users-make-personalized-suggested-user-lists/)创建个性化的推荐用户列表。

谷歌已经变得更加品牌友好[,对名人、公众人物和被加入(未定义)“大量”圈子的人的个人资料进行验证](https://web.archive.org/web/20230204141444/https://techcrunch.com/2011/08/20/google-rolls-out-verified-profiles-still-struggles-with-real-name-policy/)。目前还不清楚建议的用户列表是否只包括已验证的个人资料。

谷歌如何决定谁上榜将非常有趣。很明显，Twitter 的追随者们在发挥着某种作用，这很有趣。你不得不怀疑脸书或网络上的其他追随者是否也会被使用。Horowitz 补充道,谷歌在推荐方面有一些“围绕特定语境相关性的惊人想法”。
