# 谷歌窃取苹果的想法和其他意外的商业间谍故事 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/07/11/google-stealing-apples-ideas-and-other-tales-of-accidental-corporate-espionage/>

![](img/7b474bcc9a0534a893537dce4c88f290.png)今天早上，我起床后在 [Techmeme](https://web.archive.org/web/20221207145802/http://www.techmeme.com/) 上看到一个有趣的标题，作者是福布斯作家布莱恩·考尔菲尔德:[为什么谷歌正在窃取苹果的创意](https://web.archive.org/web/20221207145802/http://www.forbes.com/2009/07/10/google-apple-schmidt-technology-enterprise-tech-google.html)。哇，一个涉及全球最大的两家科技公司和丑闻的故事？这会很棒的。

然后我读了这个故事，它被证明是谷歌的一个奇怪的热门作品，没有明显的原因，试图暗示谷歌以某种方式绕过了苹果的“著名秘密”，并使用其想法来促进其新的 [Chrome OS](https://web.archive.org/web/20221207145802/http://www.beta.techcrunch.com/2009/07/07/google-drops-a-nuclear-bomb-on-microsoft-and-its-made-of-chrome/) 的开发。不知何故，谷歌首席执行官埃里克·施密特是幕后黑手。

考尔菲尔德声称，苹果让施密特留在苹果董事会，可能“错过了一个位置”。首先，为什么每个人似乎都认为苹果并不清楚埃里克·施密特是谷歌的首席执行官？套用约翰·格鲁伯的话，真的有人认为史蒂夫·乔布斯太害羞而不敢面对施密特吗？当然，考尔菲尔德马上接着说，施密特实际上并不是间谍，而是苹果和谷歌已经成为“偶然的竞争对手”，只是他们还没有意识到这一点。没错。

但施密特的观点是可以原谅的。有可能，不管多么不可思议，埃里克·施密特不知何故意外地滥用了他在苹果公司的职位，获得了内幕信息，然后他利用这些信息授权给谷歌，这一切都在乔布斯的鼻子底下，而没有意识到他正在与苹果公司竞争。毕竟，他是一个非常聪明的人。

因此，谷歌显然抄袭了苹果，从愚蠢到完全荒谬。

考尔菲尔德指出，Chrome OS 运行在 x86 和 ARM 处理器上，基于 Unix。但是看:苹果的 OS X *也*运行在 x86 和 ARM 处理器上，基于 Unix 变种，NeXT Step(为 OS X 奠定了基础)被用于构建网络。坦白地说，我不知道最后一点是什么(也许 Chrome 是为浏览网页而优化的？).接下来，OS X 可以在多种设备上运行，包括 iPhone、服务器和 Mac 电脑。显然，Chrome 计划支持上网本并最终支持台式机，这完全源于苹果允许一个操作系统在多种产品上运行的想法。

考尔菲尔德没有指出这两种操作系统之间的主要明显差异。首先，Chrome OS 可能会完全由浏览器驱动——许多细节仍然是猜测性的，但 Chrome PCs 很可能会直接启动到浏览器中，启动时间在个位数秒内。应用程序都将依赖于“云”，而不是在本地存储数据。相比之下，苹果的 OS X 是完全不同的 T2。它不是基于云的。这是一个成熟的操作系统，不会引导到浏览器中。似乎考尔菲尔德专注于几乎所有其他明显的相似之处，而忽略了堆积如山的差异。

考尔菲尔德接着谈到了 iPhone 和 Android 之间的相似之处。这是一个众所周知的领域，苹果和谷歌在这里绝对是竞争对手。但试图说谷歌剽窃了苹果智能手机操作系统的想法是很奇怪的——苹果和谷歌对他们的软件采取了非常不同的方法(谷歌是免费的，并在多家供应商的设备上使用，而苹果不是)。苹果并不是第一个涉足移动领域的公司，尽管他们确实革新了触摸屏。所以，好吧，我们可以看看考尔菲尔德会怎么做。

然后:

> 苹果公司在 2007 年 6 月为视窗操作系统 Safari 推出了网络浏览器。谷歌在 2008 年 9 月推出了 Chrome 网络浏览器。苹果为 Mac 电脑提供了一整套办公软件。谷歌提供了自己的基于网络的替代产品，谷歌文档。苹果公司有电子邮件服务。谷歌也是。

我们一个一个来。《金融时报》刚刚引用埃里克·施密特的话说“[谷歌联合创始人拉里·佩奇和谢尔盖·布林]自创立公司以来就想做这个项目”。换句话说，谷歌考虑这么做已经超过了 T2 十年(T3)。但是谷歌直到 Safari [登陆 Windows](https://web.archive.org/web/20221207145802/http://www.beta.techcrunch.com/2007/06/11/apple-safari-3-for-windows-coming/)一年多之后才发布 Chrome。我猜是考尔菲德角。当然，按照这种逻辑，苹果从一开始就在“偷”微软的整个 Windows 浏览器。

接下来，苹果办公效率套件 [iWork](https://web.archive.org/web/20221207145802/http://www.apple.com/iwork/) vs 谷歌文档。我不认为任何真正尝试过使用 iWork 和 Google Docs 的人会把它们放在同一个范围内。Google Docs 是一项基于云的服务，允许轻松协作。但这并不是你要做任何繁重的文档搬运工作的地方——我无法想象在里面写超过几页的东西，更不用说创建一个媒体密集型的演示文稿了。是的，你可以在这两个软件中编辑文档，但是称之为“偷来的”想法是非常可笑的。

最后，苹果的电子邮件服务 vs Gmail。这个太可笑了。当 Gmail 在 2004 年发布时，它彻底改变了网络邮件，为用户提供了完整的千兆字节存储空间(更不用说一个伟大的界面)。邮件服务在当时被限制在其中的很小一部分——事实上，是苹果的溢价。一年花费 100 美元的 Mac 电脑只能提供少得可怜的 15 兆字节的存储空间。

即使所有这些观点都站得住脚，考尔菲尔德也可以使用几乎完全相同的论点来指控谷歌从微软“偷窃”。微软也有一个浏览器，以及一个名为 Office 的小型办公套件和一个名为 Hotmail 的电子邮件服务。你可能听说过他们。谷歌不是偷窃，而是竞争。它正在发布新产品，试图击败竞争对手。同样，苹果也想打败微软，所以这两家公司有时会发布类似的产品。如果有什么不同的话，这是一个[收敛进化](https://web.archive.org/web/20221207145802/http://en.wikipedia.org/wiki/Convergent_evolution)的案例，而不是秘密间谍活动。
