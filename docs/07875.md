# 索尼随身听 A 系列泄密，显示触摸屏 PMP 远未死亡

迎接即将到来的索尼随身听 A 系列。这个小家伙出现在英国一家主要零售商的网站上，与当代人有着明显的不同。实际上，我敢说，很性感。
