# 终于！Sonos 将于 3 月发布安卓控制器应用

啊，[索诺斯](https://web.archive.org/web/20221210061603/http://www.sonos.com/)。一旦你尝试过，你就不会回头。问题是，过去你需要花费数千美元来设置你的家庭，以便将音乐从任何 wifi 连接的硬盘传输到 Sonos 扬声器。这是我 2006 年对 T4 的评论。

然后事情开始发生了。最引人注目的是他们发布了 [Sonos ZonePlayer S5](https://web.archive.org/web/20221210061603/http://www.crunchgear.com/2009/10/13/sonos-500-mo-money-mo-powerful-wireless-speakers/) ，这是一款可以通过 iPhone 或 iPod Touch 控制的低成本 Sonos 设备。去年，他们终于有了一款 [iPad 控制器应用](https://web.archive.org/web/20221210061603/https://beta.techcrunch.com/2010/06/22/sonos-ipad-controller/)。

但对 Android 来说没有。总是承诺在不久的将来，但我们不得不等待。然后等待。

现在等待结束了。Sonos 称安卓应用将于三月发布。[详情在此](www.sonos.com/android.)。

有一些 iOS 用户没有的漂亮特性。例如，更深层次的硬件集成允许你通过 Android 手机上的物理音量按钮来控制 Sonos 设备上的音量。

*“我们认为我们为 iPhone 制造了最好的无线音乐系统。现在，我们也为安卓系统制造最好的无线音乐系统，”Sonos 营销总监 Thomas Meyer 说。*

美好时光。
