# 诺基亚 Windows Phone 8 新闻发布会现场直播

我们正在多雨的纽约市现场直播诺基亚和微软的联合新闻发布会。今天的庆祝活动包括推出一两款新的 Windows Phone 8 手机，代号为“Arrow”和“Phi”。

根据上个月[泄露的信息](https://web.archive.org/web/20221124021537/https://beta.techcrunch.com/2012/08/23/nokia-said-to-reveal-arrow-and-phi-windows-phone-8-devices-on-september-5/)，这两款设备预计将被命名为 Lumia 820 和 Lumia 920，并配备 PureView。他们能与 HTC 和三星这样的公司竞争吗？我们会在当地时间上午 10 点新闻发布会开始时知道。
