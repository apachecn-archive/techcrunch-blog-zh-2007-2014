# 瑞士媒体公司收购当地 Groupon clone Deindeal.ch 多数股权

瑞士领先的每日交易网站 dein deal . ch 60%的股份被瑞士媒体公司荣格收购。总部位于苏黎世的 Deindeal.ch 据说在该地区规模超过了 Groupon，并继续保持良好的增长势头。条款没有透露。

Deindeal.ch 自 2010 年 3 月上线以来，拥有 100 多名员工。它按照 Groupon 的模式，每天提供生活产品至少 50%的折扣(这种模式已被彻底抛弃，但也许在利基市场行得通？).

管理层将继续留在公司。
