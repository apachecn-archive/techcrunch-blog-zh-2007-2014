# Twitter 首次推出用户评价，呃，故事:当 140 个字符不够时

去年，Twitter [推出了一项名为](https://web.archive.org/web/20230204230621/https://techcrunch.com/2010/08/17/twitter-jumps-on-user-stories-bandwagon-debuts-tales/) [Twitter 故事](https://web.archive.org/web/20230204230621/http://tales.twitter.com/)的营销活动，展示了用户与微博平台互动的方式。今天，该公司正在[推出](https://web.archive.org/web/20230204230621/http://blog.twitter.com/2011/11/introducing-twitter-stories.html)一个类似的活动，名为 [Twitter Stories，](https://web.archive.org/web/20230204230621/http://stories.twitter.com/)它似乎展示了有趣的推文和网站用户通过平台交流的方式。

例如，Maureen Evans [使用 Twitter](https://web.archive.org/web/20230204230621/http://stories.twitter.com/en/1/maureen_evans.html) 与用户分享食谱。亚伦·杜兰德利用推特拯救了他母亲的书店。知名影评人罗杰·伊伯特[在失声后使用推特](https://web.archive.org/web/20230204230621/http://stories.twitter.com/en/1/roger_ebert.html)作为配音。

有些故事甚至还配有视频(嵌在下面)。用户可以通过标签#TwitterStory 或者通过提到@twitterstories 来发送他们的 Twitter 故事，Twitter 将选择展示某些故事。

用户故事总是一个很好的营销工具。脸书去年围绕他们的 5 亿用户里程碑发起了一场类似的营销活动，名为脸书故事。谷歌还推出了谷歌故事(Google Stories)，收集用户分享的关于谷歌如何影响他们生活的故事。
