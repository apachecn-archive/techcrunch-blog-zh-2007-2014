# Jawbone 聘请了首位数据副总裁，负责可穿戴设备、量化自我和个性化健康的交叉领域

Jawbone 进行了一次关键的招聘，显示了该公司在量化自我和可穿戴设备发展方面的一些方向。健身设备 UP 的开发和制造商邀请了前 LinkedIn 高级数据科学家 Monica Rogati 担任该公司的第一位数据副总裁。

就人才而言，罗加蒂是 Jawbone 的巨大胜利。根据[的一份福布斯简介，在 LinkedIn，她开发了一个将职位与候选人匹配的系统，并为你可能认识的人创建了第一个机器学习模型。](https://web.archive.org/web/20221205113325/http://www.forbes.com/sites/danwoods/2011/11/27/linkedins-monica-rogati-on-what-is-a-data-scientist/)在 Jawbone，她领导着数据团队，专注于可穿戴计算、量化自我和个性化医疗保健的交叉领域。

Jawbone 首席执行官兼创始人侯赛因·拉赫曼解释说，罗加蒂是该领域最优秀的数据科学家之一，所以当试图招募她来领导 Jawbone 的数据工作时，这是显而易见的。拉赫曼在一次采访中说:“她不仅有深厚的技术背景，而且她可以获取数据，并将这些数据转化为个性化的、有意义的产品体验。”

今年早些时候，Jawbone [收购了 MassiveHealth](https://web.archive.org/web/20221205113325/http://gigaom.com/2013/02/04/jawbone-buys-visera-massivehealth-to-marry-data-design-with-wearable-computing/) 以加强其数据挖掘工作。该公司还收购了 Body Media 公司，这是一款可穿戴的健康追踪设备。我们假设 Rogati 将致力于如何围绕 UP 和其他设备的用户上传的数据添加创新的软件增强功能和产品。

拉赫曼进一步解释说，她的聘用代表了 Jawbone 数据挖掘工作的加速。在以数据为中心的方法中，Jawbone 正在处理各种因素，如睡眠、天气、营养等如何影响和相互关联。

“我们希望我们的用户能够让他们了解自己的数据，然后做出更明智的选择，”拉赫曼说。他补充说，其中一部分是能够为用户提供更个性化的健康和数据视图。

拉赫曼说:“这仅仅是一个开始，当涉及到获取数据并实际看到它对消费者意味着什么时，还会有更多的事情发生。”

看看 Rogati 开发了哪些产品来为 UP 甚至 Body Media band 处理的数据创建一个更加个性化和更具洞察力的视图应该很有趣。我们一直在猜测 Jawbone 不仅仅是开发健身设备，而是开发成熟的健康监测设备。能够处理这些数据并从中创造有价值的见解的软件将是这项工作成功的关键。
