# Tableau 更新了它的数据可视化软件，不幸的是，这是他们今年最大的产品新闻

上周，我同意了一个关于该公司最新发布的 [Tableau 8.0](https://web.archive.org/web/20221007055254/http://www.tableausoftware.com/new-features/8.0) 的简报，他们说这是今年最大的产品新闻。嗯，我不感兴趣。

新的 Tableau 软件是健壮的，这应该是这家世界级的数据可视化公司所期望的。该版本提供了对谷歌分析等新数据集的访问，并且该公司首次建立了自己的 API。

[维梅奥·http://www.vimeo.com/61654440 w = 400 & h = 300]

但该公司尚未发布其作为软件即服务(SaaS)提供的软件的商业版本。他们只说继续关注。他们说，大多数数据仍在本地，因此主要集中在软件上。

事情是这样的。公司服务器上的数据开始从它的铁腕控制中松动。正在分手上线。像 [Chartio](https://web.archive.org/web/20221007055254/http://chartio.com/) 这样的服务正在为客户提供信息，并且[提供不需要任何复杂软件安装的可视化工具](https://web.archive.org/web/20221007055254/https://beta.techcrunch.com/2012/11/13/chartio-improves-the-way-you-visualize-your-data/)。只需把它拉进来，用 Chartio library 的服务进行可视化。很简单。

还有很多其他服务即将上线。这里有一张你现在可以使用的 20 个的列表。

Tableau Software 确实有叫做 [Tableau Public](https://web.archive.org/web/20221007055254/http://www.tableausoftware.com/public) 的免费软件。它需要 Windows 来安装。呻吟。

[![tableaupublic](img/f6ebd1ea468aab021d417e7058430293.png)](https://web.archive.org/web/20221007055254/https://beta.techcrunch.com/2013/03/21/tableau-updates-its-data-visualization-software-and-unfortunately-its-their-biggest-product-news-of-the-year/tableaupublic/)

Tableau Software 是一家世界级的公司，但这次发布表明他们落后了。APIs Tableau 是[提供](https://web.archive.org/web/20221007055254/http://www.tableausoftware.com/public/blog/2013/02/java-api-1831)说明该公司正在采取一个新的方向。但是 API 已经成为标准很多年了。

Tableau 提供了通过可视化带来简单性的软件。在这些年中，出现了一波创新，使得几年前不可能实现的可视化变得更加容易。通过不提供 SaaS 功能，Tableau 留下了很多东西，并迫使该公司与许多其他遗留软件提供商建立更深的关系。
