# 制作自己的 iWatch TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/09/10/make-your-own-iwatch/>

我想做一个 [iWatch](https://web.archive.org/web/20221208075528/http://www.crunchgear.com/2010/09/09/the-ipod-nano-watch-fashion-win-or-fashion-fail/) ，但我并不觉得有必要再买一个 [Nano](https://web.archive.org/web/20221208075528/http://crunchgear.com/tag/Nano) 。也就是说，如果你想在 iWatchery 一试身手，那就跳过[去 GnomonWatches](https://web.archive.org/web/20221208075528/http://www.gnomonwatches.com/NATOStrap.htm) 或者甚至是[易贝](https://web.archive.org/web/20221208075528/http://shop.ebay.com/?_from=R40&_trksid=p3907.m570.l1313&_nkw=nylon+bond+strap&_sacat=See-All-Categories)，寻找你喜欢的风格的“邦德”表带。顺便提一下，这些带子被称为“邦德带”，因为吉米在一些电影中戴着它们来压住他的劳力士。现代邦德穿欧米茄，我觉得对于一个有着无尽神秘感的男人来说是更好的选择。
