# NeuStar 将以 6.5 亿美元收购按需客户洞察提供商 TARGUSinfo

专门为网络、电信、娱乐、广告和营销公司提供数据见解和情报的 NeuStar 今天宣布，它以 6.5 亿美元的现金收购了 TARGUSinfo(又名 Targus Information Corp .)，以加强其实时分析和客户情报产品。

NeuStar 今天还发布了其[第三季度财务业绩](https://web.archive.org/web/20230203073417/http://www.marketwatch.com/story/neustar-reports-results-for-third-quarter-2011-2011-10-11)，该公司在其中表示，预计此次收购将使其明年的每股收益增加 20 美分以上，并将年收入提高至约 7.5 亿美元。市场也将此次收购视为 data insights 公司未来发展的良好迹象，因为今天的股票在延长交易中上涨了 3%。

[在今天](https://web.archive.org/web/20230203073417/http://www.neustar.biz/blog/neustar-insights/why-neustar-is-acquiring-targusinfo/)的一篇博客文章中，该公司向其股东解释了此次收购，称其成立的宗旨是确保客户在移动或更换电话运营商时，“仍可通过同一号码联系到他们”。该公司还表示，自 90 年代中期推出以来，它已经监管了数万亿笔交易，并建立了“世界上最大的电话号码数据库”。

在过去的二十年中，NeuStar 开始将其管理大型数据集的专业知识应用于 web，为 Web、电信和其他行业提供基于电话和 Web 数据的洞察力和智能。通过收购 Targus，该公司现在可以着眼于成为基于其现在复合的实时信息和分析为其客户提供决策支持的领先提供商。利用两家公司的安全信息池，使 NeuStar 在这个日益分散和增长的市场中进一步扩大规模。

在应用中，由于 Targus 是最大的来电显示服务提供商之一，NeuStar 现在将能够确保客户在更换运营商时不仅保留相同的号码，而且客户将能够验证到底是谁在呼叫他们。所以每个人都赢了。

这两家公司有大量类似的产品，目标市场相当，因此此次收购对双方来说似乎是双赢的，让双方都能够更快地达到他们否则可能无法达到的规模。
