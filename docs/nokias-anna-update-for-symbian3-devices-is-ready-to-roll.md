# 诺基亚面向 Symbian^3 设备的 Anna 更新已准备就绪 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/18/nokias-anna-update-for-symbian3-devices-is-ready-to-roll/>

尽管该公司在 Windows Phone 7 上采用了新的操作系统，但诺基亚一直承诺将继续支持其首款操作系统 Symbian。也就是说，诺基亚 N8、C7、C6-01 和 E7 的用户将从今天开始下载 Symbian Anna 软件更新。此次更新包括对浏览器和地图的改进，以及新的图标。

此次更新还将带来一个纵向 QWERTY 键盘，以及文本输入时的分屏查看，这样你就可以在输入时查看网页、表格和线索。就地图改进而言，Ovi 地图现在将拥有更好的搜索功能以及公共交通路线。此次更新还包括社交网络签到，以及通过电子邮件和短信分享你的位置。

诺基亚还在 Anna 更新中加入了新的业务特性，为电子邮件带来了会议请求支持、硬件加速数据加密，并为需要 IPSEC 和 SSL VPN 支持的用户提供了安全的内部网访问。不幸的是，更新中最好的部分只针对 C7 的用户，但这是一个非常好的改进。一旦 C7 用户下载了 Symbian Anna 更新，手机中的 NFC 硬件将被激活。

诺基亚建议通过 PC 上的 Ovi 套件下载，因为它相当大，可能会通过 OTA 失败，除非你的 Wi-Fi 连接超级可靠。哦，在我离开之前还有一个小问题:这个更新目前在美国还不可用，而且由于诺基亚想要停止它在北美的 Symbian 业务，它可能永远也不会了。目前，比利时、中国、捷克共和国、丹麦、埃及、芬兰、德国、印度、印度尼西亚、科威特、挪威、波兰、俄罗斯、瑞典、泰国、阿联酋和英国将率先获得更新，其他国家将在未来三周内跟进。
