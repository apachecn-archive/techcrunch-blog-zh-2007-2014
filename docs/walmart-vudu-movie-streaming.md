# 小心，网飞。沃尔玛现在在 Walmart.com 点播视频

网飞又多了一些竞争对手。沃尔玛今天推出了视频流媒体服务，不要只是把它当成“我也是”的服务。这是一个严肃的提议。

沃尔玛在 18 个月前收购了 Vudu，并慢慢发展了这项业务。像网飞一样，Vudu 可以在许多机顶盒、游戏系统和媒体流上使用。用户在这些设备上使用时，可以访问 20，000 个标题，包括他们播放 DVD 当天的标题。这项刚刚推出的服务将 Vudu 流媒体直接放在 Walmart.com 上进行即时观看。Walmart.com 服务似乎无法访问整个图书馆，但仍有很大一部分可用，这可能足以让网飞放慢脚步。

这不是订阅服务。它是按需提供的，租金从 1 美元到 5.99 美元不等，购买价格从 4.99 美元起。这是 Vudu 一贯的工作方式。只不过这次它可以直接在美国最大零售商的网站上买到。

就在两周前，网飞将一项广受欢迎的计划提高了 60%。尽管武都的图书馆规模较小，但它一直被认为是网飞的主要选择，现在看来，沃尔玛似乎准备给予它适当的支持和布局，使其成为真正的竞争对手。
