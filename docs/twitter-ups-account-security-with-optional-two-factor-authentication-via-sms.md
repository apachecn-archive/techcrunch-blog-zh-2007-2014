# Twitter 通过短信的双重认证提升了网络安全，但共享账户仍有危险

几个月前，几十个账户可能受到威胁后，Twitter 今日(T2)通过短信推出了双重认证，以保护人们免受黑客攻击和网络钓鱼欺诈。不幸的是，它可能无法帮助像大品牌和新闻机构这样的共享帐户，其中多人需要能够登录和退出，但只有一个电话号码可以获得登录验证码。

继二月的 [Twitter 安全事件之后，数十万账户不得不重置他们的证书，科技界要求 Twitter 提供双重认证。《连线》杂志的 Mat Honan](https://web.archive.org/web/20221208041302/https://beta.techcrunch.com/2013/02/01/twitter-hacked-250k-affected-just-go-change-your-password-now-though/) [上个月报道称，Twitter 正在内部测试这项功能。但是从那以后，包括美联社在内的几个著名的账户已经被安全功能本可以阻止的网络钓鱼欺骗攻击了。现在有了双因素身份认证，我们有望看到更少的个人帐户受到损害。](https://web.archive.org/web/20221208041302/http://www.wired.com/threatlevel/2013/04/twitter-authentication/)

![Screen Shot 2013-05-22 at 12.36.49 PM](img/5c494fa2e2557c90d02b2fa0bacee835.png)

然而，那些对黑客来说账户最有价值的品牌和新闻机构可能不会从这一功能中受益。他们只能设置一个电话号码作为双因素身份验证代码的接收方，但可能有几个员工需要访问该帐户。如果他们启用了这一功能，无论谁携带了在 Twitter 上注册的手机，都必须将代码转发给所有其他员工，以便让需要它的人得到它。这种麻烦可能会阻止共享账户开启登录验证，因此黑客攻击可能会继续。

Twitter 将自己的安全博客贴上“登录验证入门”的标签，希望这一事实意味着可能会有更多的进步来保护共享账户。Twitter 的产品安全团队成员 Jim O'Leary 写道:“发布这一功能所需的大量服务器端工程工作已经为我们在未来提供更多的帐户安全增强功能扫清了道路。敬请期待。”

### Twitter 双因素是如何工作的

该功能现在正在推出。如果你没有在你的[账户设置](https://web.archive.org/web/20221208041302/https://twitter.com/settings/account)中看到它，你应该很快就会看到了。要启用双因素身份验证，请选中帐户安全旁边解释“登录时需要验证码”的框如果你还没有用 Twitter 保存你的电话号码，你需要输入你的电话号码。一旦您在手机上收到确认短信，您就可以完成安全功能的激活。

从那时起，当你输入用户名和密码登录 Twitter.com 时，你会收到一条短信，其中包含一个验证码，你需要输入该验证码来证明你是该帐户的所有者。这个想法是，如果有人窃取了你的名字和密码，他们可能也没有你的手机，他们需要这两者来作为你登录。然而，Twitter 的“登录验证”并不适用于其移动应用，所以当你在小屏幕上登录时，你需要使用[临时应用密码](https://web.archive.org/web/20221208041302/https://twitter.com/settings/applications)来确保安全。
