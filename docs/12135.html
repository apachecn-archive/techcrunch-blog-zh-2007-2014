Google+扩大了“直播”的范围——名人可能会注意到TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2011/12/14/google-expands-on-air-hangouts-and-celebrities-may-take-notice/

当Google+在今年夏天首次推出时，其最大的特色之一是一项名为Hangouts的免费视频聊天服务。乍一看，它像是以前做过的事情:视频聊天，最多十个人之间。但它是免费的，而且与大多数需要朋友互相邀请的视频聊天服务不同，Hangouts更像是那些深夜的大学自习室讨论，人们可以随心所欲地进出。还有一些漂亮的面部切换技术，可以将摄像头对准正在说话的人。

今天，Google的宣布了对Hangouts工作方式的一些重大改进——以及它们如何更深入地融入你的Google+流。

你可能注意到的第一个变化是在你和你的朋友发布的各种状态和照片更新下方增加了一个“Hangout”按钮。点击它，网站将启动一个专门针对该状态更新的链接，这样你就可以通过视频实时聊天，谈论刚刚发布的链接、故事或照片。其他用户可以通过点击帖子评论部分的链接来加入这个社区。

另一项重大举措是扩展Google+的“直播”功能，至少在短期内，这对这项服务的发展更为重要“直播”本质上是一个标准的聚会(最多可以有10个人)，让主持人向无限数量的观众公开播放。能够参加聚会的人数仍然有限制——但是数百万人有可能收听直播。在过去的几个月里，谷歌已经利用这一功能广播了几个备受瞩目的事件，包括与达赖喇嘛的对话，但用户还无法使用这一功能。今天，谷歌说，它开始向该网站的“数百名”知名用户推出这一功能，最终将向所有人开放。

现在，当然有可能通过其他服务来协调公开广播的视频聊天，但这样做往往比它应该做的更痛苦(TCTV工作人员实际上在上周末没有太多运气的情况下正在寻找一种简单的方法来做到这一点)。这就是为什么“On Air”会有很大的吸引力——它是自助式的，所以你可以随时准备好，谷歌会自动将视频流存档在主持人的YouTube账户中(默认情况下是私人的，但可以改为公共的)。

由于这种易用性，“直播”可能会引起名人或政治候选人的注意，比如说，他们可能想进行即兴现场采访。每一项活动都有可能吸引成千上万的新用户首次使用Google+(当然，他们是否会回来是另一回事)。

一些其他的调整:当你浏览Google+时，Google将会更加突出显示Hangouts(它们将会出现在侧边栏的一个小部件中)。谷歌还让Hangouts对那些没有带摄像头的电脑(或高速互联网)的人更有用:它可以让你用电话打电话到一个Hangouts。最后，Hangouts用户在自己身上覆盖驯鹿的鹿角(这实际上是最近推出的Hangouts API 功能的展示——谷歌为八字胡做了类似的事情)。
