# PiCloud 向公众推出无服务器计算平台

初创公司 [PiCloud](https://web.archive.org/web/20230322201112/http://www.picloud.com/) 向公众推出其平台，允许批处理、高性能计算和科学计算应用。PiCloud 是在[光速投资](https://web.archive.org/web/20230322201112/http://www.lightspeedvp.com/summerfellowships/)孵化的，声称“不需要服务器”，使开发者、科学家和工程师能够轻松地在云上建立应用程序。

PiCloud 平台旨在通过三个行动来简化云计算。Cloud.call 在云上运行您的自定义函数，cloud.result 检索您的函数的返回值，cloud.map 将您的函数映射到多个参数。这家初创公司的库支持 Python 代码，提取字节码、源代码和将用户计算转移到云所需的数据。PiCloud 还提供了一个仪表板来管理提交的作业和分析使用情况。

用户也不需要围绕特定的平台 API 开发他们的软件；用户可以将他们现有的数据存储与 PiCloud 结合使用，无论他们的数据是在 Rackspace、Amazon Web Services 上，还是在他们的私人数据中心。
