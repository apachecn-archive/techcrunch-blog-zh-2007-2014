# FeeFighters 推出支付网关武士

总部位于芝加哥的初创公司 [FeeFighters](https://web.archive.org/web/20230203092730/http://feefighters.com/) ，一家[为信用卡处理者提供的比较购物网站](https://web.archive.org/web/20230203092730/https://techcrunch.com/2009/09/25/transfs-launches-comparison-shopping-site-for-credit-card-processing-fees/)正在推出一款新产品来简化在线支付。FeeFighters 的[武士](https://web.archive.org/web/20230203092730/http://samurai.feefighters.com/)作为在线商家接受支付的网关首次亮相。

FeeFighters 是一个针对信用卡处理者的[比较购物网站](https://web.archive.org/web/20230203092730/https://techcrunch.com/2011/01/12/feefighters-raises-1-6-million-to-be-the-billshrink-for-business-services/)。这家初创公司旨在帮助商家节省信用卡费用，并进行反向拍卖，以征求信用卡处理公司的竞标。

那么，FeeFighters 为什么要推出支付网关呢？首席执行官 Sean Harper 表示，初创公司帮助找到最佳信用卡处理器的许多客户往往被他们的网关提供商锁定，这意味着他们无法转换。Samurai 允许任何企业将任何交易发送到任何处理器，没有锁定。

Samurai 的功能包括将交易发送到多个商家账户的能力，并将自动将国际和小额交易发送到最便宜的处理器。Samurai 还安全地存储客户的信用卡号码，并允许商家在几分钟内实现 PCI 合规性。

Samurai 还包括通用兼容性，因此商家可以向任何支付处理器发送和接受交易。Samurai 提供所有功能，价格为每月 10 美元，每次交易 0.10 美元。
