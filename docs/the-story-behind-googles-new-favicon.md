# 谷歌新图标背后的故事

![google-favicon-its.png](img/c95ce11afe526e189d5fecf376dd3c01.png)

最近人们已经注意到，当你访问一个谷歌网站时，你浏览器地址栏中的小图标谷歌图标已经从熟悉的大写 G 变成了小写。谷歌在八年多的时间里没有改变这种设计，所以这种改变引起了更多设计变化的猜测。事实证明，新的 favicon 是谷歌为不同场合设计的新图标家族的一部分，从其标志到浏览器 Favicon 再到移动应用程序。玛丽莎·梅耶尔在[谷歌博客上解释道:](https://web.archive.org/web/20230201152116/http://googleblog.blogspot.com/2008/06/one-fish-two-fish-red-fish-blue-fish.html)

> *原因是我们希望开发一组图标，能够更好地适应一些新平台，如 iPhone 和其他移动设备。因此，新的 favicon 就是其中之一，但我们也开发了一组基于徽标的图标，它们作为一个统一的集合挂在一起。*

以上是谷歌从数百个候选人中挑选出来的一些不同的候选人。下面是目前的五个。我只是很高兴谷歌没有用彩虹主题。那就太独角兽了。随着设计过程的不断发展，favicon 仍然会改变，谷歌正在寻求反馈。也许它应该尝试一种众包方法/竞赛来设计新图标，并允许人们实际提交自己的作品。
