# Seedcamp Week 2011 -与决赛选手、获胜者和一位好心人见面 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/09/12/seedcamp-week-2011-meet-the-finalists-the-winners-and-one-good-samaritan/>

![](img/9c7a98bc67a5809e63f937b61c764452.png)

正如我上周在写的那样，当 [Seedcamp](https://web.archive.org/web/20230205031628/http://www.seedcamp.com/) 发布了一系列公告，包括[新资金](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/seedcamp)和雄心勃勃的国际扩张计划时，我上周在伦敦花了一些时间去了解一些有趣的欧洲初创公司。

以下是 2011 年 Seedcamp Week】所有 20 名决赛选手的概述，从 3 名(实际上是 4 名)开始。

但首先，值得注意的是，Seedcamp 已经为 2011 年 Seedcamp 周期间展示其产品的 20 家公司中的 16 家提供了资金和其他支持——我可以补充说，一次又一次地——与其余四家初创公司进行了面试，并决定也向它们注入一些种子资本。所以，在某种程度上，他们都有点“赢”了。

此外，[加速器集团](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/financial-organization/the-accelerator-group) (TAG)已经向五家 Seedcamp 初创公司发出了投资要约，尽管还不清楚具体是哪些，以及他们提出投资多少。

就个人而言，我真的发现大多数 Seedcamp 初创公司的质量令人惊叹。

以下是比赛的前三名获胜者:

![](img/d4558a1c6cf265734e367c2dd9b0226f.png) ** Vox.io ** ( [网站](https://web.archive.org/web/20230205031628/http://vox.io/)–[CrunchBase 概要](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/vox-io))

Vox.io 是一项简单的服务，可以让你直接通过浏览器进行语音和视频通话。不需要下载、安装或防火墙配置。

![](img/dcc1cdaa4b19986ff18ce02f06c979c3.png)**([网站](https://web.archive.org/web/20230205031628/http://www.transferwise.com/)–[CrunchBase profile](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/transferwise))**

 **TransferWise 是一种在线货币兑换服务，本质上允许人们以固定的费用使用中间市场汇率兑换货币。

![](img/5d1d6aaa13b7867f6879d907c3acaa14.png) ** GrabCAD ** ( [网站](https://web.archive.org/web/20230205031628/http://grabcad.com/)–[CrunchBase 概要](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/grabcad))

GrabCAD 是一个服务市场，它将世界各地的制造和产品开发公司与 CAD(计算机辅助设计)工程师联系起来。

GrabCAD 获得了黄金投资奖(€25，000 英镑种子资金)。然而，该公司决定，由于它最近从 Matrix Partners 和 Atlas Venture 等投资者那里筹集了足够的资金，他们应该要求 Seedcamp 将这笔钱投资于另一家初创公司 Farmeron。善业。

这让我们迎来了第四位冠军:

![](img/d58f393d9adb5f220066a6dff77bd8cf.png) ** Farmeron ** ( [网站](https://web.archive.org/web/20230205031628/http://beta.farmeron.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/farmeron))

Farmeron 帮助世界各地的农民在线管理他们的农业数据，根据综合统计数据进行农场绩效分析，并与其他农民合作。

以下是 2011 seed camp 周剩余的 16 名决赛选手，按字母顺序排列:

**AppExtras** ( [网站](https://web.archive.org/web/20230205031628/http://www.appextras.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/appextras))

AppExtras 是一个面向移动应用发行商的交叉推广网络。这家初创公司的服务背后的想法是，移动应用开发商和出版商可以获得更多的下载量、新用户和额外的网络安装收入。

**Bilbus** ( [网站](https://web.archive.org/web/20230205031628/http://bilbus.com/)——还没有 CrunchBase 档案)

小企业通常不了解他们的现金流，而是完全依赖它。Bilbus 是一种“面向中小企业的 Wonga ”,允许它们根据现金流借入 1，000 万至 5，000 万美元。

Bilbus 有效地帮助匹配借款人的需求和贷款偏好，并在一个地方提供所有关键信息，从而使商业贷款人更容易向企业贷款。

**开花** ( [网站](https://web.archive.org/web/20230205031628/http://www.blossom.io/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/blossom))

“精益创业团队”的项目管理工具。如果你愿意，可以说是敏捷开发软件。

如果“精益是新的敏捷”，那么精益项目管理和问题跟踪的工具就缺失了，所以 Blossom 想要纠正这一点。竞争对手每个用户每月花费 25 美元，所以他们将免费提供。

**Campalyst** ( [网站](https://web.archive.org/web/20230205031628/http://www.campalyst.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/campalyst))

Campalyst 是一款社交分析软件，可以衡量脸书页面的收入和转化率，并帮助品牌了解他们的粉丝是否转化为客户。

这听起来难以置信，但这家初创公司声称，其软件衡量了从脸书的状态更新到广告商网站购买的完整转换漏斗。

该团队表示，他们已经交付了工作产品，并获得了客户的支持。代理机构和内部社交媒体团队可以按月订阅该软件。

**编译** ( [网站](https://web.archive.org/web/20230205031628/http://compilr.com/)–[CrunchBase 概要](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/compilr))

Compilr 是一个开发工具，它允许程序员通过 web 浏览器编写代码。想想谷歌程序员文档。

它拥有 6 万名注册开发者。普通用户是来自美国的 23 岁男性——听起来没错。它声称是唯一真正编译代码的在线 IDE。Cloud9 是他们最大的竞争对手，但它有来自 Accel 和 Atlassian 的 4550 万美元。

**Crashpadder** ( [网站](https://web.archive.org/web/20230205031628/http://www.crashpadder.com/)——还没有 CrunchBase 简介)

Crashpadder.com 表示，它是英国最大的点对点住宿网络，类似于 Airbnb。公平地说，至少在英国，你确实比后者更容易听到这个词。

但如果 Airbnb 允许某人自己出租公寓(有时结果很糟糕)，Crashpadder 的主人会留在他们的地方，当然，也会遇到 Crashpadding 的人。

他们于 2008 年推出 P2P 住宿服务，并于 2010 年 1 月“硬推出”。他们声称在 93 个国家的 1800 个地方有 16000 台主机，到目前为止有 25000 个用户。

Crashpadder.com 从每笔订单中赚取 10%的佣金，外加 3 英镑的管理费。

**人群** ( [网站](https://web.archive.org/web/20230205031628/http://www.usecrowd.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/crowd))

Crowd 是一种聚合和搜索服务(移动应用程序+基于网络的小部件),帮助人们可视化世界各地随时拍摄的带有地理标记和时间戳的公共照片流。

人群基本上是在什么时候给人们一种正在发生的事情的感觉，就在现在和任何地方。

**高效云** ( [网站](https://web.archive.org/web/20230205031628/http://efficientcloud.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/the-efficient-cloud))

这家初创公司使托管提供商能够为基于 PHP 和 Ruby on Rails 的 Web 应用程序提供经济高效的云托管。高效云提供平台即服务(PaaS)解决方案，该解决方案由云计算后端、管理仪表板、供应系统和易于使用的终端用户控制面板组成。

**分形** ( [网站](https://web.archive.org/web/20230205031628/http://getfractal.com/)–[CrunchBase 轮廓](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/fractal))

分形验证所有主要电子邮件客户端的 HTML 电子邮件，并可以自动修复已知的电子邮件怪癖。

**霍尔维** ( [网站](https://web.archive.org/web/20230205031628/http://holvi.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/holvi))

Holvi 的目标是用一种为团体活动提供智能支票账户的服务来取代银行账户。

帐户和预算由来自不同团体的多人共享，提供了完全的透明度。

**moapp**([网站](https://web.archive.org/web/20230205031628/http://www.mopapp.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/mopapp))

Mopapp 跟踪和分析多个应用商店的移动应用销售，并通过生成易于理解的报告来帮助开发者和发行商获得洞察力。

**驱逐我** ( [网站](https://web.archive.org/web/20230205031628/http://oust.me/)——还没有 CrunchBase 档案)

Oust.me 是一款“地理定位游戏”,它根据 Foursquare、Gowalla 和脸书 Places 平台提供的数据将位置转化为领土。

**Psykosoft** ( [网站](https://web.archive.org/web/20230205031628/http://www.psykosoft.net/)–[CrunchBase 档案](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/psykosoft))

这家法国初创公司开发了基于网络的软件 Psykopaint，可以将照片变成令人惊叹的艺术品。

**Rentlord** ( [网站](https://web.archive.org/web/20230205031628/http://www.rentlord.com/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/rentlord))

Rentlord 通过为租户、合租者和房东定制的社交网络，为人们提供启动、管理、续订和终止租赁或合租所需的一切。

**Sportlyzer** ( [网站](https://web.archive.org/web/20230205031628/http://www.sportlyzer.com/)–[CrunchBase 个人资料](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/sportlyzer))

Sportlyzer 是一个有趣的社交在线锻炼应用程序，数字教练可以在其中建议锻炼并提高锻炼动机。

**Zin.gl** ( [网站](https://web.archive.org/web/20230205031628/http://www.zin.gl/)–[CrunchBase 简介](https://web.archive.org/web/20230205031628/http://www.crunchbase.com/company/zingle))

Zin.gl 的目标是[根据他们的脸书、Foursquare 和其他可用的在线数据，将单身者与和他们有很多共同点的其他单身者](https://web.archive.org/web/20230205031628/https://techcrunch.com/2011/05/05/zin-gl-mines-social-network-data-to-help-singles-find-their-perfect-match/)配对。**
