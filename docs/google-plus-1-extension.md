# 谷歌刚刚在全国电视上泄露了他们的+1 Chrome 扩展

如果你还没有看过谷歌新的[“亲爱的索菲”视频](https://web.archive.org/web/20230203023234/http://www.youtube.com/watch?v=R4vkVHijdQk&feature=player_embedded)，我强烈推荐你去看看。太棒了。

仔细观察。

在视频的不同点上，正在使用的 Chrome 浏览器展示了一个未发布的产品:Google +1 扩展。为了更好地观看，请跳到 0:51 分。它就在那里，前面中间，就在被点击的 Picasa 按钮旁边。

最有趣的部分是，谷歌实际上正在国家电视台播放这个视频，作为他们新的 Chrome 营销推广的一部分。所以谷歌刚刚向数百万人泄露了他们自己的新功能。至少那些密切关注的人。

那么扩展是做什么的呢？很难说，因为他们从来没有真正点击它在视频中。但是你可以假设这是一个简单的方法来+1 你正在访问的任何网站。大约一个月前，谷歌正式推出了+1 ,作为一种与朋友分享谷歌搜索结果的社交方式，并帮助谷歌用社交果汁对这些结果进行排名。

我还想知道这个扩展是否没有更多的社交功能——比如分享 Twitter 和其他网络上的链接以及+1。你可能还记得，当他们开发+1 时，[我们不断得到关于它的消息](https://web.archive.org/web/20230203023234/https://techcrunch.com/2010/12/02/google-plus-one-brin/)。其中之一是 [+1 工具栏](https://web.archive.org/web/20230203023234/https://techcrunch.com/2010/12/07/google-plus-one-pic/)，这显然是一种与其他社交网络分享的方式。

谷歌还表示，出版商网站的+1 按钮即将出现。但是自从最初的声明之后，我们就再也没有听到任何消息了。

无论如何，一个+1 的 Chrome 扩展显然正在路上。谷歌让我们知道它要么是通过一个聪明的潜意识营销行动，要么是在整个美国公众面前犯了一个大错误。

不管怎样，很棒的广告。+1，真的。
