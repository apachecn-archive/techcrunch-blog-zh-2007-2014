Twitter首席执行官和前即兴喜剧演员迪克·科斯特罗在本周 Web 2.0峰会的开幕晚宴上发表了近一个小时的讲话，充满了他独特的幽默和洞察力。

当主持人约翰·巴特利引用最近一篇芝加哥论坛报的文章开始讲话时，“科斯特洛的工作是把从斯努基、安东尼·韦纳和金·卡戴珊的指尖散发出来的140个角色变成一台强大到足以承受泡沫破裂的赚钱机器，”科斯特洛自己也附和道，“……对于一个如此有限的杰出人物来说，这个任务似乎无法完成。”

亮点: Battelle对Costolo关于Google最近在Google+上发布的标签和实时搜索的质询；“现在还为时过早，”科斯特洛开始说道，对此巴特尔回应道，“这是一个糟糕的答案……‘现在还为时过早’……‘这不是一个零和游戏’..我可以替你回答吗？”

“一个人不能再抛出主题句了，”科斯特洛打趣道，增加了已经非常自我意识的采访的整体自我意识(在某些时候，两人抛出了采访可能产生的博客文章标题——巴特尔误称我为“亚历克西斯”)。

至少有半个小时左右的时间是值得观看的，如果只是为了体验一位杰出的科技公司首席执行官对荒谬的敏锐感觉。
