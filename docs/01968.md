首批系统将用于冷却英特尔的酷睿 2 至尊芯片。Thermaltake 正在开发对双 CPU 甚至 GPU 冷却的支持。一切仍在测试中，所以还没有价格或可用性的消息，但它可能会介于水冷和 VapoChill 设置之间。
