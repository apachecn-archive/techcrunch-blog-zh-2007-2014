# HomeAway 凭借两名关键员工和每年 2 亿美元的收入向 IPO 迈进

知情人士透露，大型度假房屋租赁服务公司 HomeAway 正准备上市，可能在 2011 年。考虑到今天宣布的公司[聘用的两名关键高管——新的首席产品官](https://web.archive.org/web/20230125174904/http://www.homeaway.com/info/media-center/press-releases/2010_pressreleases/bellm-hale)[汤姆·黑尔](https://web.archive.org/web/20230125174904/http://www.crunchbase.com/person/tom-hale)(之前林登实验室的首席产品官)和首席运营官[布伦特·贝尔姆](https://web.archive.org/web/20230125174904/http://www.crunchbase.com/person/brett-bellm)(之前是贝宝，他的姓需要一个元音字母)。

一年半前，在一轮 2.5 亿美元的融资后，该公司的估值为 14 亿美元。我们听说，当时的收入约为 1.5 亿美元。现在营收更像是 2 亿美元/年，纯利润 7000 万美元左右。简而言之，这件事就是抛出现金，而且是大量的现金。
