# TechCrunch 采访了(非常开心的)Skype 首席执行官乔希·西尔弗曼

Skype 首席执行官 Josh Silverman 在我们今天早上在门洛帕克录制的视频采访中笑个不停。难怪——尽管有[严重的法律和衍生剧](https://web.archive.org/web/20221007193350/http://www.beta.techcrunch.com/2009/10/08/the-fight-for-skype/)，Silverman 还是设法[完成了](https://web.archive.org/web/20221007193350/http://www.beta.techcrunch.com/2009/11/19/ebay-skype-sale/)他从易贝剥离的价值数十亿美元的 Skype。他的法律纠纷已经烟消云散。Skype 像杂草一样生长。他设法保住了经营公司的工作。生活是美好的。

他现在要做的就是与这个星球上两个更有影响力的人物——新投资者[马克·安德森](https://web.archive.org/web/20221007193350/http://www.crunchbase.com/person/marc-andreessen)和联合创始人/新投资者[尼可拉斯·曾斯特罗姆](https://web.archive.org/web/20221007193350/http://www.crunchbase.com/person/niklas-zennstrom)一起管理董事会会议。

嗯，那就让这列火车[保持在轨道上](https://web.archive.org/web/20221007193350/http://www.beta.techcrunch.com/2009/10/21/skype-hits-521-million-users-and-185-million-in-quarterly-revenue/)。Silverman 在采访中说:“Skype 的用户已经超过 5 亿，并且每天增加 30 万。”。1/3 的使用量是视频，尽管视频通话只能是 1-1。语音呼叫是多方通话。收入保持在 1.85 亿美元/季度，利润率为 24.2%。任何时候都有多达 2000 万人在使用 Skype。

我们在采访中稍微谈了一下 Skype 的业务。但是 2010 年的大部分焦点。*“你会看到 Skype 在更多的地方变得越来越普遍，无论是移动设备还是嵌入式设备，”*他说，并补充道*“期待在更多的平台上看到我们。”*

我们还谈到 Skype 是一个开发者平台。Extras 现在已经消失很久了，但 Silverman 重申 Skype 将很快在 T2 推出更强大的开发工具，将 Skype 变成一项服务。这意味着 Skype 可以在 Skype 客户端之外运行。这并不一定意味着浏览器中的 Skype，他们仍然说 Flash 不够强大，无法在浏览器中运行 Skype。但也许我们会看到 Skype 代码被直接嵌入浏览器。

我们还简单谈了谈 T4 的 Chrome 操作系统 T5。谷歌的新操作系统不允许用户安装软件，这意味着 Skype 被淘汰了。Flash 是谷歌声称将包含的唯一第三方插件。像 Silverlight 和 Skype 这样的东西被搁置了，除非谷歌决定把它们包括进来。Silverman 以*结束了采访:“如果谷歌的人有兴趣将 Skype 嵌入 Chrome，我们当然有兴趣进行对话。”*

我希望他们能尽快进行这样的对话。
