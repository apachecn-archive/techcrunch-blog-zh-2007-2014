# 你的机械师启动一个 iPhone 应用程序，让你在旅途中与机械师联系

[TechCrunch Disrupt 赢家](https://web.archive.org/web/20221210045726/https://beta.techcrunch.com/2012/09/12/techcrunch-disrupt-sf-2012-winner/) [你的机械师](https://web.archive.org/web/20221210045726/http://www.yourmechanic.com/)刚刚获得了一个 iPhone 应用程序，它将帮助用户更好地诊断所需的维修。该应用程序还将在用户的汽车正在维修时向他们发送通知，甚至可以让他们看到正在进行的工作的照片。

YourMechanic 于去年推出，为车主提供了一种简单的方式来预约机械师。这个想法是创建一个技工市场，让用户可以为他们的汽车提供服务

但是有一个问题:大多数人在车里的时候就意识到他们的车有问题。然后，他们不得不走到一台电脑前，将问题传到网上，并试图在家里或办公室内进行诊断。

新的移动应用解决了这个问题，让用户在发现问题时可以报告，并找到附近的机械师。这尤其有助于诊断非特定问题，如液体泄漏或汽车发出的奇怪噪音，用户可以通过手机而不是电脑更好地报告这些问题。

即使对于那些只想完成日常维护的人来说，这款应用也能简化一些事情。用户可以直接从应用程序中获得可用机制的列表，包括个人资料和评级，以及日程安排时间。他们还会收到推送通知，让他们知道某项工作何时完成，或者随时获得更新。

推送通知也有助于提醒用户何时该进行定期维护，因为它知道机械师最后一次看汽车是什么时候。

目前，YourMechanic 应用程序只能在 iPhone 上使用，尽管联合创始人阿布哈斯·阿特·阿格拉瓦尔(Abhas Art Agrawal)表示，该公司正在开发安卓版本。YourMechanic 已经从投资者那里筹集了 180 万美元，投资者包括 Y Combinator、SV Angel、尤里·米尔纳、安德森·霍洛维茨、Lerer Ventures、Launch Capital、杰夫·克拉维尔、CrunchFund(由 TechCrunch 创始人迈克尔·阿灵顿所有)、佩奇·克雷格、A 级投资公司 Jawed Karim、贾斯汀·沃尔德伦、约书亚·沙赫特和凯文·弗里德曼。
