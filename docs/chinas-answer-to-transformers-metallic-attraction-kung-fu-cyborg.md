# 中国的变形金刚:金属吸引力功夫机器人

![kung_fu_cyborg](img/190ee91b4fecfa014d97e8c512c82662.png "kung_fu_cyborg")

这部电影的一些早期片段已经在网上流传了几个月，但今天官方发布了预告片。与此同时，被出色命名的[《金属吸引力:功夫机器人》](https://web.archive.org/web/20221006202224/http://www.hkcinemagic.com/en/movie.asp?id=10719#gal)已经演变成某种中国版的变形金刚。

不足为奇的是，有一些不同之处:在《功夫机器人》中，人们变成了战斗机器人和车辆(据我所知)。这部中国“翻拍”电影的预算为 1400 万美元，对于一部香港电影来说很昂贵，但比《变形金刚》(预算:1.5 亿美元)和《变形金刚 2》(2 亿美元)便宜得多。它似乎是喜剧、科幻和浪漫电影的混合体。

观看下面的预告片，它非常酷(用英语替换)。作为一个超级香港影迷，我一定会看这部电影。《机器人功夫》将于 8 月 13 日在中文地区首映。因此，预计今年秋天会有英文版的 DVD。
