# Justin.tv 重新设计，让广播更容易，聊天更明显

![live-broadcast1](img/04d512226899327e2264f7a5cf7df9a3.png "live-broadcast1")现在有很多视频直播竞争，但 [Justin.tv](https://web.archive.org/web/20230326021917/http://justin.tv/) 仍然是最大的(尽管 [Ustream](https://web.archive.org/web/20230326021917/http://www.ustream.tv/) 说这有待讨论)。它希望通过今天发布的重新设计以及一些新功能来保持这一领先地位。

新网站有一个整体更干净和简化的外观。简化是另一个大变化的关键:增加大头版广播。当你第一次加载网站时，你会看到前面和中央有一个大的视频播放器，上面写着“一键直播”如果你点击它下面的红色大按钮，你会打开你的相机选项屏幕，在这里你可以选择一个相机进行记录。从那里你可以登录或创建一个帐户开始广播。

以前，您必须先登录或设置一个帐户，然后才能在不同的页面上开始录制。这种新方法使得如何立即进行录制变得更加明显。

另一个变化是在网站的每个页面上增加了一个 Meebo 即时聊天栏。以前，聊天只存在于频道聊天室，但现在 Meebo 酒吧将允许您连续聊天，无论您在网站的哪个位置。Justin.tv 表示，它是第一个使用 Meebo bar 的直播视频网站，已经有 10%的聊天信息是通过即时消息发送的，而不是在频道的聊天区。我们被告知，这相当于每天 600 万条信息中的 60 万条。预计这个数字会上升，因为它已经遍布整个网站。

正如我所说，Justin.tv 仍然是最大的流媒体直播网站，但它的竞争对手最近都在采取行动，试图刺激使用。一个大动作是 [Mogulus 最近更名为直播](https://web.archive.org/web/20230326021917/https://techcrunch.com/2009/05/18/mogulus-gets-a-killer-domain-for-rebranding-livestreamcom/)。所有这些网站[的流量似乎都在下滑](https://web.archive.org/web/20230326021917/http://trends.google.com/websites?q=justin.tv%2C+ustream.tv%2C+stickam.com%2C+livestream.com%2C+mogulus.com&geo=all&date=all&sort=0)。但是当一年前 [Ustream 推出重新设计](https://web.archive.org/web/20230326021917/https://techcrunch.com/2008/07/01/ustreamtv-just-got-a-redesign-but-justintv-is-still-beating-its-pants-off/)时，它帮助启动了增长。

上周，Justin.tv 刚刚在我们的实时流媒体服务器上发布了它的实时视频。这种新的简化的 Justin.tv 主体验，似乎是简约理念的延伸，我很喜欢。Justin.tv 上个月还推出了一个 DVR 功能。
