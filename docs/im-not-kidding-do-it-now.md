# 亚马逊:你需要马上改变你愚蠢的顾客评论政策

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/03/22/im-not-kidding-do-it-now/>

很难为一位大获成功的作家感到遗憾，但就迈克尔·刘易斯而言，我将破例一次。就这一次。

Lewis 的新书《大空头:世界末日机器内部》在不到一周前出版，已经在亚马逊上排名第一。这是一个令人印象深刻的壮举，特别是对于一部严肃的非小说作品来说，没有巫师、吸血鬼或古老的代码，这是-*天啊！*–藏在显眼的地方，以提高销量。但这一壮举更令人印象深刻的是，毫不夸张地说，这本书的许多评论都非常负面，以至于纽伦堡审判看起来乐观而慷慨。

在撰写本文时，有 64 篇[一星评论](https://web.archive.org/web/20221220102241/http://www.amazon.com/Big-Short-Inside-Doomsday-Machine/product-reviews/0393072231/ref=cm_cr_pr_hist_1?ie=UTF8&showViewpoints=0&filterBy=addOneStar)——比 2-5 星评论的总和还多——向潜在的购买者传达了一个明确的信息:这本书可能很受欢迎，但也完全是一派胡言。不要浪费你的钱。

这个信息只有一个问题:只有不到一半的一星书评是在评论这本书。

相反，大多数评论家的愤怒与出版商 WW Norton 决定不在发行精装版的同时发行该书的 Kindle 版本有关。一位名叫本·卡普兰的评论家写道:

> “我想把我的名字加入那些对这本书没有 Kindle 版本感到非常失望的人的名单中。不，我没有读过这本书，但是我想读——在我的 Kindle 上！如果所有这些一星评论导致销量下降，我认为这将是一个很好的结果，也是作者/出版商的一个极好的教训。”

去吧，本！

不，说真的，走吧。

我曾在之前写过关于出版商推迟 Kindle 发布以保护精装书销量的错误想法。这是一个愚蠢的举动，但由于亚马逊对最受欢迎的 Kindle 图书收取 9.99 美元的政策，许多出版商感到被迫这样做。对于亚马逊和出版商来说，这是亏本定价，但低数字价格点阻碍了精装书的销售，这至少在短期内导致了整体收入的下降。

作为一个花了数百美元购买 Kindle 的人，我完全理解那些对不得不等到一本书的平装本发行才能拿到电子版感到恼火的人的沮丧。但是，认为 one star reviews 将为“作者”提供一个很好的教训，让他们更早发布电子书的想法是最低级的废话。

作为一名[作者](https://web.archive.org/web/20221220102241/http://www.bringingnothingtotheparty.com/)，当我说我们对我们的书何时发行、以何种形式发行、以何种价格发行完全没有发言权时，我有着痛苦的经历。然而，我们是亚马逊负面评论中损失最大的人。一本书的总体星级是亚马逊页面上最突出的信息之一，许多读者——相当合理地——将低平均评级等同于一本写得很差的书。这损害了书的销量，也损害了我们作为作家的声誉。几乎没有人——除非他们点击并阅读负面评论的全文——看到一星评级，并认为这是对出版商拒绝电子版决定的评论。

经济评论员、《T4 救助国家》的作者 Barry Ritholtz 就这个话题写了一篇精彩的博文，将一星评论描述为“只不过是集体欺凌”。给我你的午餐钱(kindle 版)，否则我会在休息时间揍你一顿(给你一星评论)”并呼吁亚马逊改变他们的评论政策。

Ritholz 是对的——但这个问题远远超出了当前 Kindle 的争论。亚马逊的书评政策一直是我和大多数其他作者的心头大患。太多时候，尤其是对于有争议的作者，最负面的评论来自那些甚至没有读过这本书的人。他们的愤怒，以及由此产生的一星评论，只是对作者和他们所做的一切普遍反感的声明。

我知道有几个作者不得不要求亚马逊撤下诽谤性的一星评论，这些评论集中在他们的性别、种族、政治观点和他们性格的几乎任何其他方面。虽然这些可能是关于作者的一般性讨论中的相关因素，但它们肯定不会出现在对其作品的评论中。他们当然也不应该为帮助读者决定一本书是否物有所值的标题星级评分做贡献。

最令人沮丧的是这个问题有多容易修复。亚马逊需要——立即——改变它的评论政策，只有真正买过书的人才被允许评论它。这将一举消除人们对无法在 Kindle 上购买特定书籍或不愿意购买精装书的抱怨，也将驱逐所有一般的憎恨者(他们永远不会为他们憎恨的对象花钱)以及一生中从未买过书的孩子和巨魔。

事实上，很难想象将评论限制在某本书的实际购买者身上会有什么坏处。当然，这是对书评家在不付钱的情况下表达对一本书或一个作者的看法的权利的限制，但这是应该的。亚马逊不是一个普通的书评网站，它是一个商店——因此，它的读者评论只为回答一个问题而存在:这本书——不管它以何种形式出版——值我的钱吗？

这个问题只能由真正的购物者来回答，而不是疯狂的 Kindle 粉丝或各种心怀鬼胎的人。
