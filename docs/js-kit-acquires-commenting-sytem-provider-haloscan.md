# JS-Kit 收购评论系统提供商 HaloScan

为博客提供 Javascript 评论、评级和民意调查工具的供应商 [![haloscan-js-kit](img/8255b76eee2380768911b6660bda5699.png)](https://web.archive.org/web/20230218162503/http://www.crunchbase.com/company/js-kit) [JS-Kit](https://web.archive.org/web/20230218162503/http://js-kit.com/) 宣布收购最大的托管评论服务供应商之一 [HaloScan](https://web.archive.org/web/20230218162503/http://www.haloscan.com/) 。此次发布还与几项主要功能的发布相协调。财务条款没有披露。

HaloScan 之前在 1 月份与 JS-Kit 合作，为他们评论系统的用户提供“一键”部署 JS-Kit 的评级小工具(提供文章评级，而不是评论评级)。此次收购将导致 JS-Kit 客户群的指数级增长，为超过 520，000 个参与站点提供新的访问，使其总覆盖范围达到约 550，000 个站点。JS-Kit 还声称，通过这次新的收购，它将每天注册 300 多个新网站。HaloScan 的评论系统将与 JS-Kit 的评级、投票、评论、导航和顾问小部件集成。JS-Kit 的评论还带有完整的 Akismet 垃圾邮件保护和脏话过滤器。

Blogger 最初的评论系统只允许来自其他 Blogger 用户的评论，因此 HaloScan 作为其固有系统的替代方案很早就受到欢迎。Blogger 还推荐它作为提供引用链接的系统。由于不稳定的服务器问题(可能与过渡有关)，它的使用量最近有所下降，所以希望 JS-Kit 可以在大约 30 天内完成过渡时解决这些问题。

JS-Kit 将利用其新获得的用户推出重要的新功能。其中之一是实现基于开放标准的可移植用户配置文件。用户可以通过 OpenID 登录系统访问任何 JS-Kit 参与站点上的所有评论(如下图所示)。可通过托管站点上的弹出窗口访问可移植配置文件。这确实有助于更容易的发现，这可能有助于新出版商的采用。

[![](img/bbc9c70ca64a785ffed9600dbcca35e0.png "jskit-haloscan")](https://web.archive.org/web/20230218162503/https://techcrunch.com/wp-content/uploads/2008/07/jskithaloscan.png) 另一个功能将更容易与 WordPress 和 Blogger 同步。这可以让现有的博客更容易地采用，从任何一个网站自动导入旧的评论。除了更容易导入之外，JS-Kit 还会在部件中有新评论时更新参与博客的 WordPress 或 Blogger 评论。

这也与 JS-Kit 正在实现的另一个新特性 SEO 支持密切相关。JS-Kit 现在建立了一个用于索引评论内容的静态页面，您可以将它作为子域托管在您的服务器上，这样搜索引擎就可以看到您站点上的内容，而不是 JS-Kit 的内容。

JS-Kit 没有像竞争评论提供商 [Disqus](https://web.archive.org/web/20230218162503/http://www.crunchbase.com/company/disqus) 和 [Sezwho](https://web.archive.org/web/20230218162503/http://www.crunchbase.com/company/sezwho) 那样的后端网络。Disqus 和 Sezwho 基于参与博客的评论提供社交网络，从而更好地发现内容。该竞争对手的用户数量明显少于 JS-Kit(据报道，有 550，000 个参与网站，800 万用户，剩下 8，000 万条评论)，因此 JS-Kit 为 Haloscan 已经庞大的用户群添加社交网络功能可能会使他们处于市场顶端。但是 Disqus 已经获得了很大的发展势头(据报道有 20，000 个参与博客)，可能会给 JS-Kit 带来一些真正的竞争。

一个主要的区别是 Disqus 使用专有的登录系统，而 JS-Kit 使用 OpenID，这是社交网络的发展趋势。为了参与 Disqus 社区的博客，用户被引导到 Disqus 网站，而不是像 JS-Kit 一样留在博客上，这可能解释了为什么下面的竞争图显示 Disqus 在 Haloscan 上增长如此之快。该图没有计算 widget 用户。
