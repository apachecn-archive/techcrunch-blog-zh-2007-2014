# Qwilt 推出网络视频传输平台

网络视频基础设施技术提供商 Qwilt，首次推出其视频传输解决方案，旨在让运营商控制其网络中的互联网视频流量。

作为背景，Qwilt 帮助互联网服务提供商更高效、更经济地向客户提供高质量的视频。这意味着更多的消费者可以随时看到来自网飞、YouTube 和 Hulu 的尽可能高质量的视频。

Qwilt 的视频技术公司解决了一个相当大的问题，这个问题与最近通过互联网(通过网飞、YouTube、Hulu 等)传送的视频爆炸有关。)，在多个屏幕上。视频现在占所有在线活动的三分之二以上，并且有望超过这个数字。仅来自网飞服务的流媒体视频就占了今天晚上高峰时段互联网流量的 30%。所有这些新内容给互联网提供商带来了巨大的压力。

这家初创公司的技术可以被运营商用来识别、监控、存储和交付互联网视频，并允许运营商创建一个通用的视频交付层，该层透明地工作，不会中断或改变内容提供商或网络基础设施。

Qwilt 的技术使用获得专利的视频内容分类和分析技术来检测运营商互联网流量流中的视频内容。然后，它添加了优化的存储
和交付能力，以创建一个更加统一的设备，提供运营商级的性能、可扩展性和可靠性。这家初创公司表示，它的产品是一种即插即用的设备，可以很容易地放入现有的网络，而不需要任何基础设施的改变。

据报道，这款新产品在过去两年中一直处于隐身状态，其性能至少是竞争产品的五倍。Qwilt 宣布日本的三菱公司正在使用它的产品。

Qwilt 最近从 Accel Partners、Redpoint Ventures 和其他公司筹集了 2400 万美元。
