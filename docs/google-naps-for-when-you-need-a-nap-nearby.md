# 谷歌小睡，当你需要在附近小睡一会儿

亲爱的谷歌，请不要起诉那些制作了这个可爱的谷歌地图的家伙。

继续今天关于懒惰的主题，来自荷兰的两个人 Kakhiel 和 Venour 开发了一个智能的小工具来帮助你找到最近的午睡时间。

遇见，[谷歌午睡](https://web.archive.org/web/20221006234459/http://www.googlenaps.info/googlenaps.php)。

当然，这只是一个恶搞应用，根据开发者的说法，他们在这个 web 应用中直接向谢尔盖·布林、拉里·佩奇和所有谷歌员工发布了一个可爱的免责声明:

> 亲爱的谢尔盖&拉里(和其他谷歌员工)
> 你好，请不要生气这只是一个玩笑，一个恶搞。我们无意损害你的品牌或任何东西，我们只是想给谷歌粉丝的脸上带来笑容。所以请不要把这个告上法庭，我们银行里只有几百欧元。我们也不想进监狱，因为目前我们忙于其他事情。但是无论何时你在荷兰，如果你愿意，你可以在我们的沙发上小睡一会儿，只要发电子邮件给我们:kakhiel@gmail.com。如果你喜欢，我们还可以煮咖啡和烤鸡蛋(价格便宜)。

该网站的外观和感觉几乎与谷歌地图相同，但让用户标记他们最喜欢的户外午睡地点，如桥梁、长椅和开阔的田野。

你可以在你所在的区域浏览其他人发布的午睡地点，或者简单地记下你最喜欢的午睡地点与他人分享。当然，也是为了占领你的领地。

显然，像这样的产品在现实世界中的可行性几乎为零，但这是一个开始你的周五的好方法。
