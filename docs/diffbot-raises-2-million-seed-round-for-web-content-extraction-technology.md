# Diffbot 为网络内容提取技术 TechCrunch 筹集了 200 万美元的天使投资

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/05/31/diffbot-raises-2-million-seed-round-for-web-content-extraction-technology/>

超级极客/令人敬畏的视觉学习机器人技术 Diffbot ，旨在以人类的方式“看”网络，今天宣布了新的资本注入。该公司已经从一些技术资深人士那里获得了 200 万美元的资金，其中包括 EarthLink 创始人[Sky Dayton](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/person/sky-dayton)； [Andy Bechtolsheim](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/person/andy-bechtolsheim) ，太阳微系统公司联合创始人； [Joi Ito](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/person/joi-ito) ，麻省理工学院媒体实验室主任；[布拉德·加林豪斯](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/person/brad-garlinghouse)，YouSendIt ( [前 TechCrunch 母公司 AOL](https://web.archive.org/web/20221001020612/https://beta.techcrunch.com/2012/05/15/brad-garlinghouse-becomes-ceo-of-booming-file-sharing-site-yousendit/) )的首席执行官，[梅纳德·韦伯](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/person/maynard-webb)，LiveOps 的董事长，前易贝·首席运营官；Twitter 公司战略副总裁 Elad Gil；脸书前技术运营副总裁 Jonathan HeiligerRedbeacon 联合创始人[亚伦李](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/person/aaron-lee)；和生命体征[的创始人蒙哥马利·克斯滕](https://web.archive.org/web/20221001020612/http://www.linkedin.com/pub/montgomery-kersten/16/26a/5b0)。

Matrix Partners 也参与了这一轮。在新的投资者中，Sky Dayton 将是第一个加入 Diffbot 董事会的人，并将在公司中发挥积极作用，包括参与各种 Diffbot 项目的计划。

去年八月，[该公司公开发布了它的第一个 API](https://web.archive.org/web/20221001020612/https://beta.techcrunch.com/2011/08/25/diffbot-sees-the-web-like-people-do-now-free-for-developers/)，它允许开发者构建能够自动从网页中提取含义的应用。例如，Front Page API 能够分析站点主页，并理解文章文本、标题、署名、广告等之间的区别。然后，文章 API 可以提取干净的文章文本、图像和视频。Diffbot 的另一个例子是“follow API”，它可以跟踪对网站所做的更改。

今天，Diffbot 已经将网络分为大约 20 种不同的页面类型，包括主页和文章页面，这是它现在可以识别的前两种类型。展望未来，Diffbot 计划训练其机器人识别所有其他类型的页面，包括产品页面、社交网络简介、食谱页面、评论页面等。

它的 API 已经被 AOL(再次声明:披露，TC 母公司)用于其新闻杂志 [AOL Editions](https://web.archive.org/web/20221001020612/http://editions.com/) ，以及像 [Nuance](https://web.archive.org/web/20221001020612/http://nuance.com/) 、 [SocMetrics](https://web.archive.org/web/20221001020612/http://socmetrics.com/) 等公司。Diffbot 表示，它现在每月代表客户处理 1 亿个 API 调用。该公司指出，成千上万的开发者正在使用这些 API，但是付费用户只有“几十个”更正:我们现在被告知他们有“更多！”

Diffbot 创始人兼首席执行官 Michael Tung(又名“Diffbot Mike”)表示，新资金将用于招聘新员工和扩大资源。“不仅如此，我们还获得了退伍军人的巨大信任票，他们建立了大型公司，了解规模建设、维持正常运行时间和提供绝对最高服务标准的优点。”

Tung 是一名专利律师，也是斯坦福大学的博士生，由于斯坦福大学孵化器 [StartX](https://web.archive.org/web/20221001020612/http://www.crunchbase.com/company/startx) 的种子基金，他放弃了博士项目，转而从事 Diffbot。Diffbot 是 StartX 的第一笔投资。有了今天的资金，Diffbot 总共筹集到 200 万美元和零钱。
