# OpenSocial 再次被黑| TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/11/05/opensocial-hacked-again/>

就在 Plaxo 上的 RockYou OpenSocial 应用程序公开发布 45 分钟后，同一个人[又黑了它。](https://web.archive.org/web/20230316005832/https://techcrunch.com/2007/11/02/first-opensocial-application-hacked-within-45-minutes/)

这一次，他声称已经轻松访问了宁上的 iLike 应用程序。具体来说，他说他可以在用户的播放列表中添加和删除歌曲。更具破坏性的是，他还可以在客户端代码中访问用户的好友列表。给他阿宁用户名，他可以给你他们朋友的详细信息:与用户的关系，最后更新日期，照片，个人资料创建日期和部分电子邮件地址。

为了证明自己的观点，他调出了 Ning 联合创始人的好友列表，并与我分享了其中的一部分。我不会在这里发布它，但它表明他获得了应用程序的访问权限。

黑 iLike on Ning 总时间:20 分钟。

与 RockYou/Plaxo 黑客攻击一样，并没有造成真正的损害，但它表明，在匆忙将应用程序推出市场的过程中，对安全性的关注可能被忽视了。

HarmonyGuy 现在有了一个博客，他在上面写了他对 OpenSocial 应用程序的黑客攻击。[看到这里](https://web.archive.org/web/20230316005832/http://theharmonyguy.com/)。他[注意到](https://web.archive.org/web/20230316005832/http://theharmonyguy.com/2007/11/06/rockyous-emote-on-plaxo/)rock you 的应用仍然没有打补丁。
