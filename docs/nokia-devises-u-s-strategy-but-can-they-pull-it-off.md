# 诺基亚设计了美国的战略，但是他们能成功吗？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/10/28/nokia-devises-u-s-strategy-but-can-they-pull-it-off/>

诺基亚的新款 Windows Phones 至少还有几个月才会登陆美国，但是诺基亚的高层已经在努力设计他们的策略来打开美国市场。据 [Digits](https://web.archive.org/web/20230203134956/http://blogs.wsj.com/digits/2011/10/28/nokias-u-s-plan-target-first-time-smartphone-users/?mod=WSJBlog) 报道，韦伯和诺基亚希望吸引首次使用智能手机的用户，他们对市场上的其他选择感到不知所措，不想为新设备支付高昂的费用。

这是一个合乎逻辑的策略，但很容易搞砸。如果没有适当的关注，这些新的诺基亚设备可能会像旧的一样命运。为了了解诺基亚现在的处境有多艰难，让我们快速看看他们在美国的地位。

诺基亚一直是一家产品线广泛的公司，但他们在美国尝试过这种方法，多年来都没有奏效。说真的，去你最近的手机店或大型零售商那里找找诺基亚手机。

你可能会发现少数几个，但没有什么比 QWERTY 包装的文本机或迟钝的触摸屏更强大的了。这就是诺基亚此时此地占据的空间:不会让潜在客户眨眼两次的低端手机。

简而言之，这些 Windows 手机将是诺基亚多年来首次高调推出的产品，诺基亚需要抓住这个机会。这里有一些想法可以帮助他们赢得胜利。

**巩固和征服:** 
诺基亚和微软的合作肯定会利用人们的名字识别技能，但这种情况不仅仅是利用品牌名称。据诺基亚美国老板 Chris Weber 称，明年将会有一批 Windows 手机登陆多家运营商。我认为诺基亚可以削减他们的产品组合，专注于推出一些杀手级设备。

如果有什么不同的话，微软的操作系统在这方面有所帮助——不像 Android 那样一个月出现在 50 部手机上，Windows Phone 只出现在少数几部手机上。如果诺基亚集中精力推出几款物美价廉的手机，他们会比散兵游勇更有机会成名。更不用说，通过专注于更少的手机，每个设备都获得了更大的广告预算份额。意识是这里的关键，诺基亚已经很久没有在美国享受过了。

**关注销售人员:** 
管理产品线只是等式的一半——没有运营商的支持，诺基亚将一事无成。在这一点上，这些关系可能也很危险，考虑到唯一根据合同销售诺基亚设备的运营商是美国电话电报公司和 T-Mobile，即使这样也是低端产品。如果诺基亚把最初的几次发射打得落花流水，诺基亚就能赢回他们多年来失去的大部分航空母舰支持。

诺基亚如何为强劲的上市做准备？与零售商和他们的员工搞好关系——如果他们不在乎，那么相当多的潜在顾客也不会在乎。销售人员对诺基亚的 Windows Phones 手机及其产品了解得越多，他们就越有可能在销售过程中出现。例如，惠普给销售人员的房间提供免费的 Veer 4Gs，让人们用起来更舒服。

当然，该设备总体上没有持续很长时间，但我敢说 Veer 因为这个噱头而获得了更多的客户，因为销售人员对 Veer 的功能和预期的利基有了更好的理解。我并不是说诺基亚必须放弃他们的硬件，但培训活动和推广活动只能帮助零售执行。

大力推进零售业务也至关重要，因为诺基亚在智能手机领域最大的竞争可能来自 iPhone。两家公司都将自己定位为提供优雅的智能手机解决方案，因此销售人员越了解 iPhone 的替代品(尤其是价格极具竞争力的产品)，就越有可能获得客户的认可。

如果我在这里听起来过于乐观，请原谅，但诺基亚的新手机(特别是 Lumia 800)是第一款让我*渴望 Windows Phone 的手机。我当然不是一个普通的用户，但我不认为这种吸引力止于我。它很可能是在正确的时间，在正确的包中的正确的软件。但是，除非诺基亚让接下来的几个月变得有意义，否则这可能无关紧要。*
