# 说唱天才计划解释摇滚、诗歌和圣经的意义 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/11/04/rap-genius-rock-poetry/>

如果像“真正的 G 像千层面一样在沉默中移动”这样的说唱歌词难倒了你，看看[说唱天才](https://web.archive.org/web/20230203081719/http://rapgenius.com/)。众包嘻哈歌词解释网站会通知你，李尔·韦恩正在描述熟练的歹徒如何不发出声音就能暗杀某人，就像“千层面”中的字母“g”。

但是 Rap Genius 背后的三人团队并不满足于他们目前的垂直市场。今天他们告诉我，他们正准备推出一个名为 Stereo IQ 的摇滚乐歌词网站，并希望有一天能扩展到乡村音乐、诗歌、法律文件和宗教文本的网站。虽然目前专注于用户增长，网站网络可以通过赞助解释文本中提到的或与之相关的品牌来赚钱。

Rap Genius 的联合创始人 Mahbod Moghadam 告诉我,“Google 想要索引所有的文本。但更有趣的是，*试图解释*所有文本，任何适合逐行分析的东西。”虽然雄心勃勃，但这个自举式的初创网站有了一个良好的开端，每月接待数百万独立访客。

这个行业有很多跑道，因为 Mogahadam 说所有谷歌搜索的 2%是歌词。现在有几十个歌词网站，但大多数都被不准确和充斥着侵入性广告的糟糕用户体验所困扰。这包括领跑者 MetroLyrics，几年前它在[每月有 1350 万单曲](https://web.archive.org/web/20230203081719/http://www.nytimes.com/2010/05/10/business/media/10lyrics.html?src=busln)，每年有 1000 万美元的收入。

莫加哈达姆和他的联合创始人受到了 VH1 弹出视频的启发，最初自己编写了网站的所有说明。最终，它向使用维基百科风格的同行评议系统的贡献者开放。莫加达姆说，“我有这种妄想症，认为众包解释会是狗屎，但如果你有一个强大的审查等级，众包内容甚至会比使用专业人士更好。”

很快，人们开始给各种文本添加解释，而不仅仅是说唱歌词。该网站现在以《权利法案》、《圣经》节选[、《摇滚歌曲》和艾米莉·狄金森的诗歌为特色。目前，所有这些都托管在 Rap Genius 上，但 Mogahadam 希望推出独立的网站，让紧密团结的社区可以围绕特定内容聚集在一起。](https://web.archive.org/web/20230203081719/http://rapgenius.com/Matthew-gospel-of-matthew-chapter-25-lyrics)

当被问及 Quora 时，有时会收到关于歌词含义的问题，Mogahadam 说“对发起它的人非常尊重，但我认为它太大了——试图在一个网站上做所有的知识。如果你看看科技以外领域的答案，它们就没那么好了。”他说，他愿意放弃单一网站的 SEO 优势，转而创建特定垂直领域的爱好者会不断回来寻求最新解释的目的地。

对于货币化，Rap Genius 目前提供的品牌商品是推广的两倍，并且不愿意出售广告。我告诉该公司，我认为最好的办法是让那些在歌词或解释中被点名的品牌购买赞助或网站接管。如果你刚刚发现你最喜欢的说唱歌手在他的歌曲中谈论 Ciroc 伏特加或奥迪汽车，你可能很容易受到这些品牌营销的影响。

Rap Genius 正在迅速成为一些歌曲的首选结果，并看到更多“[歌曲标题] rap genius”查询。虽然单个说唱歌词可能难以解读，但独立的摇滚、乡村、法律和宗教文本网站可以提供对整首歌曲或段落含义的宏观分析。原始作者也有可能添加官方解释，从而吸引更多人访问这些网站。莫加哈达姆说，“我想看看笔画是如何看待它们自己的歌词的。”
