# 自动减肥机器人现在在 Nog 季节上市了

我已经跟随 Autom 很多年了，首先在中国与这个快乐的小机器人互动，然后看着她成长为一个年轻、智能的饮食机器人。她是做什么的？她旨在通过每天与你交谈并了解你的习惯和过失来帮助你减肥。

Autom 坐在你的柜台上，问你在吃什么。你告诉她，通过屏幕上的提示，她提醒你称体重，并给你精神上的支持。虽然你不会认为一个小小的塑料机器人可以帮助人们减肥，但你会惊讶于一个没有偏见的塑料婴儿可以为你的自尊做些什么。在测试过程中，用户开始给他们的机器人穿上有趣的衣服，并把它们当成朋友。

你可能会说 iPhone 应用程序和其他工具可以更便宜地完成 Autom 的工作。然而，设计师以一种让用户以独特和高度个性化的方式与她互动的方式构建了 Autom，确保你永远不会感到无聊或忘记记录你的食物摄入量。她就像一个不能动的私人教练。

该网站称:

你可能以前尝试过一两个减肥计划。也许你把自己限制在某些种类的食物上。或者每周被计数到某个神奇的数字。或者试图遵循某本书或杂志文章的处方。
有了 Autom，饮食本身并不重要。选择任何饮食，她会帮助你真正坚持下去。她为你提供成功所需的动力和支持。当 Amna 在页面顶部分享视频时，Autom 立即成为她家庭的一部分。

Autom 售价 199 美元，包括营养帮助在内的减肥服务每月收费 20 美元。他们向 [TechCrunch 读者](https://web.archive.org/web/20230127215219/http://www.myautom.com/?coupon_code=FREE_SHIPPING_CRUNCH)提供优惠券代码为“FREE_SHIPPING_CRUNCH”的免费送货，当他们开始发送时，我们将测试(并赠送)一台自动售货机。
