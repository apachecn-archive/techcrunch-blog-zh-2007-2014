# 视频:阿尔法狗是一种更大、更快、更安静的大狗

我相信你们都记得[大狗](https://web.archive.org/web/20230203133446/https://techcrunch.com/2008/03/17/bigdog-robot-imagine-this-thing-coming-at-you-through-the-woods/)，这是一个非常可怕的仿生[机器人](https://web.archive.org/web/20230203133446/https://techcrunch.com/tag/robots/)平台，它用怪异的腿在崎岖的地形上跳舞，发出的声音就像充满了愤怒的蜜蜂。你不会认为他们会放弃吧？不，这显然是一个令人振奋的成功，以至于他们决定建造一个在各方面都超越原版的。

他们称之为阿尔法狗，基本上是大狗的实用版本，虽然引人注目和可怕，但声音太大，不能携带足够的东西。另一方面，AlphaDog 将在一天内携带 400 磅的装备行驶 20 英里，无需加油。

[YouTube w = 640 http://www.youtube.com/watch?v=SSbZrQp-HOk]

一定要把视频看完。现在传统的“推四足机器人平台”很棒，然后你会看到一个*新的*令人毛骨悚然的行为，因为它以一种不真实的方式从躺着的位置调整自己。

现在它由外部供电，在 rails 上运行，但在现场考虑这一点并不困难。有了全球定位系统和一些基本的深度感应相机，它可以自信地导航大多数地形或跟随它的“主人”沿着小路前进。尽管我怀疑这个特性是否会在 2012 年成为 AlphaDogs 的第一个产品。
