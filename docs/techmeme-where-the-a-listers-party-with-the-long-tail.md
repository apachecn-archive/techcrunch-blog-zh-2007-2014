# Techmeme:一线明星的长尾派对。

[![techmeme-chart.png](img/0433554a317b1a596c796738acd1a82c.png)](https://web.archive.org/web/20221208223745/http://www.crunchbase.com/company/techmeme)
[Techmeme](https://web.archive.org/web/20221208223745/http://www.techmeme.com/)上最受欢迎的 [bitchmeme 的](https://web.archive.org/web/20221208223745/http://www.parislemon.com/2007/10/case-for-bitchmeme.html)之一是，它是最受欢迎的博客和新闻标题网站，跟踪网络上最受关注的科技故事，它被拥有全职写作人员的一流博客和新闻网站所主导。这种观点认为，由于博客世界的专业化，孤独的博客作者被挤出了对话。在这场争论中，TechCrunch 有时会被拿出来作为证据 A，这就是为什么我很高兴看到来自 [StatBot](https://web.archive.org/web/20221208223745/http://thestatbot.com/2008/05/05/one-third-of-techmeme%e2%80%99s-headlines-come-from-the-long-tail-the-statbot/) 的上图。它显示了 Techmeme 上标题在 [Techmeme 排行榜](https://web.archive.org/web/20221208223745/http://www.techmeme.com/lb)上的排名分布。

![techmeme-lb-small.png](img/84d159f4a259756d1b69a9b152a42ca5.png)虽然大约 30%的标题被 Techmeme 排行榜上的前十大来源占据(见左边的表格)，但另外整整三分之一来自甚至不在前 100 名的博客和网站。这意味着如果你有有趣的事情要说，不管你是谁，其他博客都会找到你并链接到你。现在这将包括 Statbot 上的帖子，它是由一个自称“来自印度的 17 岁想要成为极客”的 Yuvi 写的。欢迎加入对话，尤维。上 Techmeme 的一个万全之策是。。。写一下 Techmeme。但是也有很多其他方法可以达到目的。

就个人而言，我认为图表中显示的分布是 Techmeme 如此引人注目的原因。它总是包括一个相当稳定的可靠来源列表，但是在这些列表中混杂着大量的不确定因素，这些因素反过来会成为他们自己的主导声音。这就是我喜欢我的新闻的方式:三分之一来自相对知名的来源，三分之一来自完全出乎意料的来源，其余的来自两者之间。

参见 TechMeme 排行榜上的[顶级个人博主](https://web.archive.org/web/20221208223745/http://www.beta.techcrunch.com/2008/04/20/who-are-the-top-tech-bloggers/)。
