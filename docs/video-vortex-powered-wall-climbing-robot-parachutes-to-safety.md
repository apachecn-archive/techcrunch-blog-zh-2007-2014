# 视频:涡流动力爬壁机器人安全跳伞

种类繁多的机器人令人惊叹。就在几年前，它们能做的最好的事情是翻滚或僵硬地用关节不良的四肢行走。现在我们有了像鸟一样奔跑的机器人、[滑行的机器人](https://web.archive.org/web/20230203181035/https://techcrunch.com/2010/09/02/carnegie-mellons-snakebot-does-its-tree-climbing-trick-again/)、[。不过，这种交通方式对我来说是完全陌生的。Paraswift](https://web.archive.org/web/20230203181035/https://techcrunch.com/2011/07/22/festos-smartbird-robot-flies-through-the-air-at-ted/)[通过产生吸力爬上墙壁，然后通过跳下并展开一个小降落伞安全降落。为什么不呢，对吗？](https://web.archive.org/web/20230203181035/http://www.newscientist.com/article/dn20881-basejumping-robot-throws-itself-off-buildings.html)

看看这个视频。行动发生在中途。

[YouTube w = 640 http://www.youtube.com/watch?v=ovxv7FPk9WY]

Paraswift 是迪士尼研究所和瑞士联邦研究所的合作项目。这次演示是在巴黎皮埃尔和玛丽·居里大学举行的国际攀爬和行走机器人会议期间进行的。

它通过在中心管中制造一个“像迷你龙卷风一样”的漩涡来产生吸力；漩涡有一个低压中心，这个压力梯度，通过一个被称为*巫术*的过程，导致机器人贴在墙上。只有轮子需要与攀爬表面接触，这使得它对于像石头和混凝土这样的攀爬表面更加有用，这些表面很难在周围形成真空密封。

一旦它爬够了，小翅膀就会展开，展开滑翔机/降落伞，跳到相对安全的地方。我说相对是因为它看起来在末端有一点凸起。他们将在版本 2 中解决这个问题。

不幸的是，我能想到的这些东西的第一个用途是让它们爬上公寓大楼，刺杀里面的人类。所以这一个被归档在 [Robocalypse](https://web.archive.org/web/20230203181035/https://techcrunch.com/tag/robocalypse/) 下。
