# Keynote 以 9000 万美元收购云移动应用测试和监控平台 DeviceAnywhere

[Keynote Systems](https://web.archive.org/web/20230203080134/http://www.crunchbase.com/company/keynote-systems) ，一家提供网络和移动云监控服务的公司，[收购了开发](https://web.archive.org/web/20230203080134/http://www.businesswire.com/news/home/20111010005388/en/Keynote-Acquire-DeviceAnywhere-Leader-Mobile-Testing-Quality) [DeviceAnywhere](https://web.archive.org/web/20230203080134/http://www.deviceanywhere.com/news-and-events.html) 的 Mobile Complete ，这是一个基于云的平台，用于测试和监控移动网站和应用程序，收购价格为 6000 万美元现金，外加 3000 万美元的潜在收益。

DeviceAnywhere 从摩托罗拉风险投资公司和 Innovacom 获得了资金，为开发人员提供了一个基于云的平台，以测试和监控移动应用程序和网站的功能、可用性、性能和可用性。

SaaS 允许您在 7 个国家的全球网络中测试和监控 2，000 台设备上的应用。开发人员还可以自动测试移动应用程序，并使用 24-7 应用程序性能监控来检测潜在问题。

DeviceAnywhere 表示，包括戴尔、Salesforce、谷歌、微软和其他公司在内的 1200 多家公司正在使用该软件。

Keynote Systems 开发和销售服务、硬件和软件，以测量、测试和监控 web 和移动站点。Keynote 计划将 SaaS 加入其移动监控业务，并预计 DeviceAnywhere 的加入将使 Keynote“成为一项价值 1 亿美元的业务”
