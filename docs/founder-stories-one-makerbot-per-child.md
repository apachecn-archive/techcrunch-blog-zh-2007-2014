# Bre Pettis 的抱负:“每个孩子一个 MakerBot”(TCTV)

在这一集的[创始人故事](https://web.archive.org/web/20230204115705/http://www.techcrunch.tv/show/founder-stories)中，主持人[克里斯·迪克森](https://web.archive.org/web/20230204115705/http://www.crunchbase.com/person/chris-dixon)一边看着 3D 打印机，一边和 [MakerBot](https://web.archive.org/web/20230204115705/http://www.makerbot.com/) 、 [Bre Pettis](https://web.archive.org/web/20230204115705/http://www.crunchbase.com/person/bre-pettis) 背后的制造者交谈。

如果[打印 3D 物体](https://web.archive.org/web/20230204115705/https://techcrunch.com/2010/12/31/3d-printing-prediction/)听起来令人印象深刻，那么想想这个。佩蒂斯认为“现在还为时过早”，并将其与早期的个人电脑(如 Altair)相提并论。

大约有一个迷你冰箱大小，Makerbot 售价 1299 美元，允许用户通过 CAD 软件创建自己的对象(或者用户可以从 [Thingiverse](https://web.archive.org/web/20230204115705/http://www.thingiverse.com/) 上的“超过 10000 个东西”中选择),并在几分钟内打印出来。产品范围从浴帘环和眼镜边缘到迷你建筑和可移动的塑料心脏。我们实际上在我们的工作室测试了 Makerbot，并带走了一枚火箭(Dixon 在系列的最后拿出了它)。

当迪克森问佩蒂斯，当公司模型的价格在 2 万美元到 100 万美元之间时，MakerBot 是如何以低于 1500 美元的价格制造和销售 3d 打印机的，佩蒂斯回答说，“你知道，我们只是从一个完全不同的地方来的。他们服务的市场要求它必须工作，必须完美，必须达到 99.9999%的精确度，我们的目标是，让它工作，我们能做的绝对最低限度是什么，然后让我们试试，如果它工作，就发货。”

根据他们下面的对话，Pettis 讨论了他的背景(前木偶师)、竞争和基于家庭的 3D 打印的未来。当迪克森问佩蒂斯“5 到 10 年后”的景观会是什么样时，佩蒂斯回答说，他希望“每个孩子面前都有一个，每个孩子都有一个 Makerbot。”

请务必观看这两个剪辑以获得更多见解，包括最近推出的新功能 Makebot。
