# 惊喜！闪迪仍在生产 MP3 播放器

令人惊讶的是，有人仍在为大众市场制造 MP3 播放器。我很高兴向大家展示售价 70 美元的 Sandisk Sansa Clip Zip，这是一款小巧的便携式媒体播放器，有 4 GB 和 8 GB 两种型号(4GB 型号售价 50 美元)。

剪辑压缩有 7 种颜色，包括红色，蓝色和橙色，几乎兼容任何音频格式。它还有一个 microSD 插槽，因此您可以将播放器扩展到 32 GB。它还有一个非常小的液晶显示屏和一个简单的界面。

很久以前，SanDisk 以其功能齐全的 MP3 播放器而闻名，但那些日子已经一去不复返了。然而，如果你正在为孩子或祖父母寻找一个简单的播放器，这可能是正确的选择:它是白痴证明，坚固耐用，而且不像 Zune 或 iPod 那样昂贵。
