# Urban Airship 收购苹果 Passbook 的 PassTools 制造商泰洛，因为并非所有品牌都需要全功能应用 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/12/04/urban-airship-acquires-tello-maker-of-passtools-for-apples-passbook-because-not-all-brands-need-fully-featured-apps/>

在 [Urban Airship 收购位置初创公司 SimpleGeo](https://web.archive.org/web/20221206061729/https://beta.techcrunch.com/2011/10/31/urban-airships-strategic-partnership-with-simplegeo-turns-into-an-acquisition/) 一年多后，该公司通过全股票交易收购风险投资支持的[泰洛](https://web.archive.org/web/20221206061729/http://www.tello.com/)，再次扩大了其功能集。这家初创公司[于 2010 年在 TechCrunch Disrupt](https://web.archive.org/web/20221206061729/https://beta.techcrunch.com/2010/09/28/tello-gives-businesses-consumer-feedback-on-the-fly/)上首次亮相，最初专注于为商业客户构建实时反馈工具，但最近重新将精力放在了苹果 Passbook 的可视化 pass builder[PassTools](https://web.archive.org/web/20221206061729/https://passtools.com/)上。正是后一种产品吸引了 Urban Airship 的注意。

据泰洛创始人兼首席执行官 Joe Beninato 称，他公司的业务反馈工具至今只有不到 100 名客户，这也是他们选择转移到另一个领域的原因。“另一个产品做得不错，但我们看到了这个机会，并说‘我们认为这个可以大得多’”，他解释道。那个假设被证明是正确的。Beninato 说，泰洛在 2012 年 9 月推出了 PassTools，现在，仅仅几个月后，就有数千名客户使用其平台创建了数万张通行证。

虽然现在有[几十个](https://web.archive.org/web/20221206061729/https://beta.techcrunch.com/2012/09/19/passrocket-lets-any-business-create-loyalty-cards-for-apples-new-passbook-app-for-free/)[初创公司](https://web.archive.org/web/20221206061729/http://www.passsource.com/) [工作中](https://web.archive.org/web/20221206061729/http://passk.it/) [在](https://web.archive.org/web/20221206061729/https://beta.techcrunch.com/2012/09/27/social-marketing-platform-woobox-lets-businesses-send-apple-passbook-coupons-via-sms/) [这个](https://web.archive.org/web/20221206061729/http://passhop.passauthority.com/) [空间](https://web.archive.org/web/20221206061729/https://www.facebook.com/PassWallet)中，PassTools 是最早推出一款所见即所得编辑器用于建筑通行证的。要使用这项服务，企业客户只需上传一个徽标，选择一个模板并添加一些文本，他们基本上可以在五到十分钟内创建一个通行证。这个过程非常适合没有自己内部开发团队的中小型企业。

然而，为了满足大型企业客户的需求，该公司在几周前开始悄悄地提供对 API 的访问。有了 API，企业可以使用与上面相同的过程构建模板，然后以编程方式将数据动态地插入到各个字段中。现在只有少数客户在使用这种 API，公司目前还不能透露。这些客户包括零售领域的客户、一家大型电信公司和各种使用优惠券的广告代理商。这些努力的更多细节将在 2013 年 Q1 上公布。

这两家公司，Urban Airship 和泰洛，与 True Ventures 有着共同的投资者，这就是泰洛创始人兼首席执行官乔·贝尼纳托最初被介绍到 Urban Airship 团队的原因。这也是 Urban Airship 选择泰洛而不是其他竞争者的原因之一，因为投资者已经对他们进行了一定程度的尽职调查。但正如城市飞艇 CMO·布兰特·希格尔克解释的那样，与泰洛的交易也有其他原因。“我们真的对乔和团队印象深刻，我们喜欢他们采取的方法，他们是正确的人，有着正确的成熟度和经验，我们对此感到非常满意。从文化上来说，他们适应得很好，只是有很多无形的东西，”他补充道。

此外，虽然许多 DIY Pass 建设者都是新的早期创业公司，但泰洛也受益于风险投资，在 2 月份的种子期后获得了总计 370 万美元的外部资金。该公司的投资者不仅包括 True Ventures，还包括 500 Startups、Bullpen Capital、Founder Collective、Chris Sacca、Aydin Senkut、Russ Siegelman、Marc Goines、罗恩·康韦、Naval Ravikant 和 Shervin Pishevar。(顺便说一句，泰洛现在不再需要筹集资金是件好事，因为 Beninato 最近与泰洛的早期支持者之一 500 Startups 断绝了关系。今年秋天，他辞去了 500 Startups mentor 的职位，因为他觉得戴夫·麦克卢尔(Dave McClure)对 WalletKit 的投资被背叛了。尽管事实上在种子阶段来自风投的竞争性投资相当普遍。)

**整合将给城市飞艇带来一批全新的客户**

至于 PassTools 与城市飞艇的核心产品整合，那还是一条出路。虽然 Hieggelke 说他会“喜欢”明年年初，但在这一点上没有办法贴上时间戳，因为交易刚刚结束，两个团队甚至还没有开始讨论整合计划。不过，他所知道的是，城市飞艇的客户对这种服务的需求正在增长。“我们正处于真正倾听客户(关于 Passbook)意见的阶段，我们开始听到对 Passbook 的热情高涨，”他表示。“我不会说这是一场熊熊的森林大火，但就人们而言，这是一个缓慢而有条不紊的进步——随着他们开始内化 Passbook 的功能，他们开始变得非常兴奋。”

他还解释说，Urban Airship 在推送通知和定位方面的专业知识将有利于寻求 Passbook 解决方案的客户。或许更重要的是，Passbook 支持为公司打开了一系列全新的潜在客户的大门。

Hieggelke 说:“这让许多没有开发应用程序的公司有能力通过开发通行证和试水来进入移动领域。”“App Store 中有超过一百万个应用程序，”他继续说道，*(注意:他指的是* *[本报告](https://web.archive.org/web/20221206061729/http://www.wired.com/gadgetlab/2012/11/report-1-million-apps/)* *，其中引用了提交和批准的应用程序，明确地说，不是实时应用程序)，*“所以我们不需要仅仅为了拥有一个应用程序而发布大量应用程序。可能有一些品牌和客户真的没有一个全功能应用的用例。”这里想到的一个明显的例子是消费品公司。“我可能不需要薯片应用程序，”Hieggelke 笑着说，对于这些公司来说，通过 Passbook 而不是独立的应用程序推出激励和特价更有意义。

贝尼纳托同意道:“一波浪潮即将到来。”。“广告公司需要一点时间来搞清楚事情，而背后的品牌也需要一点时间来理解为什么这一点会如此强大。”

目前形式的 PassTools 网站将继续运行，直到城市飞艇整合完成。尚未对泰洛之前的产品做出决定。与此同时，泰洛的五人团队将在帕洛阿尔托建立新的城市飞艇办公室，这是对 UA 在波特兰和旧金山的总部的补充。Urban Airship 目前有 65，000 个账户，每月在其平台上看到几十亿条通知。
