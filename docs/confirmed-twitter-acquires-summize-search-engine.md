# 确认:Twitter 收购 Summize 搜索引擎

**更新:** [视频采访全文在此](https://web.archive.org/web/20230213030333/https://techcrunch.com/2008/07/15/interview-with-evan-william-summize-acquisition-api-issues-and-their-revenue-model/)。

[![](img/a85571ef992a460f3c9d96affe379cf4.png)](https://web.archive.org/web/20230213030333/http://www.crunchbase.com/company/summize) 事实证明，上周的那些[传言](https://web.archive.org/web/20230213030333/https://techcrunch.com/2008/07/07/twitter-may-buy-summize/)是准确的。微博网站 [Twitter](https://web.archive.org/web/20230213030333/http://www.crunchbase.com/company/twitter) 已经收购了 [Summize](https://web.archive.org/web/20230213030333/http://www.summize.com/) 搜索引擎，Twitter 联合创始人伊万·威廉姆斯向我证实。尽管交易价格是以“大部分股票”支付的，但交易规模并未披露。

Twitter 还雇佣了 Summize 名员工中的 5 名。创始人兼首席执行官 Jay Verdy 将开始新的项目。

加入 Twitter 的五名 Summize 员工都是工程师，增加了目前在 Twitter 工作的十二名工程师。Summize 首席技术官 Greg Pass 将成为 Twitter 的首席技术人员，担任工程和运营总监。

Twitter 联合创始人伊万·威廉姆斯早些时候告诉我，这笔交易已经讨论了一段时间。6 月份，当 Twitter 与 Summize 合作，帮助他们在苹果 iPhone 3G 史蒂夫乔布斯主题演讲期间保持 Twitter 平台稳定时，这两家公司已经准备好了一份条款清单。这笔交易在上周完成了。

[John Borthwick](https://web.archive.org/web/20230213030333/http://www.crunchbase.com/person/john-borthwick-2) ，sum mize investor[beta works](https://web.archive.org/web/20230213030333/http://www.crunchbase.com/financial-organization/betaworks)的合伙人，也是伊万·威廉姆斯在 1999 年共同创立的 Pyra Labs (Blogger.com)的投资者。换句话说，这两家公司已经是近亲了。
