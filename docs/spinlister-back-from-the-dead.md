# Airbnb For Bikes 的初创公司 Spinlister 从死亡池归来，现在有了新的管理层 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/19/spinlister-back-from-the-dead/>

如果你像我一样是自行车和 Airbnb 类型的共享经济初创公司的忠实粉丝，你可能会失望地获悉，P2P 自行车租赁初创公司 [Liquid (née Spinlister)](https://web.archive.org/web/20221007130300/https://beta.techcrunch.com/2012/10/30/strike-spinlister-strike-liquid/) 几个月前被关闭了。这项服务于去年春天在纽约市推出，几个月前刚刚向公众开放，之后就被叫停了。好吧，[现在又回来了](https://web.archive.org/web/20221007130300/https://www.spinlister.com/)，由它的一个种子投资者复活，而创始团队继续追求其他利益。

Spinlister 在全美多个市场缓慢推出其点对点自行车租赁服务后，于去年 9 月在旧金山的 [TechCrunch Disrupt 战场](https://web.archive.org/web/20221007130300/https://beta.techcrunch.com/2012/09/11/spinlister-makes-it-into-disrupt-battlefield-with-its-airbnb-for-bikes/)推出，并在[在全国推出后不久](https://web.archive.org/web/20221007130300/https://beta.techcrunch.com/2012/09/26/spinlister-national-launch/)。

但后来事情变得很奇怪。几个月后，这家小小的创业公司[更名为 Liquid](https://web.archive.org/web/20221007130300/https://beta.techcrunch.com/2012/10/30/strike-spinlister-strike-liquid/) ，这表明其自行车租赁市场可能会扩展到其他领域。然后，Spinlister 和 Liquid 都陷入了黑暗，因为团队关闭了租赁功能，并考虑他们下一步要做什么。

几周前，我会见了 Spinlister/Liquid 的联合创始人威尔·丹尼斯(Will Dennis)，他谈到了团队变得更加热衷的一些想法。他们喜欢视频发现和视频信息的想法，他给我看了一些他们正在做的东西的早期原型。但很明显，他们对继续专注于自行车租赁并不感兴趣。

好消息是，巴西企业家、这家初创公司的种子投资者马塞洛·洛雷罗对保持 P2P 市场的活力很感兴趣。考虑到这一点，他接管了这个网站，并在本月早些时候悄悄地让旧的 Spinlister 网站——以及 Spinlister 品牌——起死回生。

洛雷罗是白手起家的连续创业者的典型代表，之前曾在服装、餐饮和饮料行业创建公司。他创建了巴西服装创业公司[Mandi.net](https://web.archive.org/web/20221007130300/http://www.mandi.net/site/)，并且是几年前[卖给坎帕里](https://web.archive.org/web/20221007130300/http://beveragemanager.net/Article-Single-News.176.0.html?&tx_ttnews%5Bpointer%5D=53&tx_ttnews%5Btt_news%5D=4125&tx_ttnews%5BbackPid%5D=182&cHash=4647c3adbcbdf6e222a02979cbeb5225)的一家酒类公司 Sagatiba Brasil 的一部分。在此之前，他将一家互联网门户公司卖给了葡萄牙电信，还在巴西经营了一家连锁餐厅。

当他投资 Spinlister 时，他喜欢这家初创公司的计划，即更好地利用未充分利用的资产，让人们获得他们原本必须拥有的东西。虽然“Airbnb-for-x”模式仍不是主流，但洛雷罗认为，公司仍有巨大的机会利用 P2P 商业模式。

“P2P 业务还需要一段时间才能起步，”洛雷罗在电话中告诉我。“但我还是觉得这个想法真的很好。”考虑到这一点，他重新启用了旧的 Spinlister 网站，允许美国各地的用户再次通过该平台列出和租赁自行车。

虽然网站关闭后没有太大变化，但洛雷罗对 Spinlister 有大计划。虽然网站还没有投入使用，但已经有 500 多辆自行车被列出来了，但他希望更新网站，使其更有吸引力，并扩大社区。这意味着更多的社交功能，以及推出一款手机应用程序，让租房变得更加无缝。

洛雷罗希望在阵亡将士纪念日之前发布一些重大更新，届时他计划围绕这项服务进行更大的营销推广。这也可能意味着开放美国以外的租赁服务，基本上使 Spinlister 在世界各地都可以使用。

在我们最初对话的后续电子邮件中，洛雷罗对 Spinlister 表现出了极大的热情，尤其是在酒类行业待了这么久之后。坦率地说，这令人耳目一新——尤其是对于一个最初没有创办他现在经营的初创公司的企业家来说。

“正如我昨天告诉你的，我不指望现在运行 Spinlister，”他写道。“但我当然很享受这一过程，并对企业的潜力充满信心。我不知道我是否会成功。但我不怕失败。”
