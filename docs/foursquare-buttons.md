# Foursquare 开始与《华尔街日报》一起测试“添加到 Foursquare”按钮

知道一项服务做得好不好的一个可靠方法是当你看到他们的按钮开始出现在网络上的时候。我们已经用[脸书](https://web.archive.org/web/20221209073539/https://beta.techcrunch.com/2010/03/25/facebook-to-release-a-like-button-for-the-whole-darn-internet/)见过它，我们已经用 Twitter ( [Tweetmeme 按钮](https://web.archive.org/web/20221209073539/https://beta.techcrunch.com/2009/08/21/what-exactly-do-16-billion-retweet-buttons-get-you-about-6-million-actual-retweets/)见过它，现在我们将开始用 [Foursquare](https://web.archive.org/web/20221209073539/http://foursquare.com/) 见到它。

需要说明的是,“加入 Foursquare”按钮并不像网络上的其他分享按钮那样无处不在。相反，这个按钮允许内容提供商给读者一键访问，将场地添加到他们的 Foursquare 待办事项列表中。第一个对此进行测试的合作伙伴是《华尔街日报》。从今天开始，报纸的网站将在文章(特别是餐馆评论和其他文化报道)中的地点旁边展示这些按钮。

随着点击按钮，地点被添加到待办事项列表中，还会有一个由《华尔街日报》编辑编写的提示出现在用户的 Foursquare 页面上。我们被告知，该提示将包含一个链接，链接到关于会场的文章，因此它也将把流量带回《华尔街日报》。

这个按钮目前只会出现在《华尔街日报》上，但是“*我们绝对有兴趣在未来找到一种方法来扩展这个功能，以便其他品牌可以利用它，*”four square 的发言人告诉我们。Foursquare 拒绝透露这对他们来说是否是一项创收交易。

在某些方面，这个添加到 Foursquare 按钮看起来更像是 Twitter 的新的随处按钮。正如我们所提到的，这不是直接分享，而是将网站的功能扩展到网络的其他部分。我知道我经常在网上读到关于新餐馆的消息，但从来不记得把它们放在我的 Foursquare 待办事项列表中——所以这很有意义。
