# 红杉在 MongoDB 大数据数据库提供商 10gen 中领投 2000 万美元

为企业提供基于 MongoDB 的大数据数据库的公司 10gen 已经筹集了 2000 万美元的融资，由红杉资本牵头，10gen 的其他现有投资者 Flybridge Capital 和 Union Square Ventures 也参与了融资。这使得该公司的资金总额超过 3000 万美元。

与 Cloudera 将 Apache Hadoop 商业化的方式类似，10gen 为托管 NoSQL 平台 MongoDB 的开源数据库提供商业化版本、培训和支持。MongoDB 是一个开源的、面向文档的数据库，设计时考虑了可伸缩性和开发人员的敏捷性。

与使用关系数据库将数据存储在表和行中不同，在 MongoDB 中，您使用动态模式存储类似 JSON 的文档。MongoDB 的目标是在键值存储(快速且可伸缩)和关系数据库(功能丰富)之间架起一座桥梁。

10gen 允许公司扩展存储和处理海量数据的数据库。正如梅里曼所说，10Gen 让公司变得“灵活，快速完成工作”。“作为背景，10gen 是由 DoubleClick 创始人兼首席技术官德怀特·梅里曼和 DoubleClick 前工程师兼 ShopWiki 创始人兼首席技术官艾略特·霍洛维茨创立的

正如梅里曼告诉我的，许多处理大量数据的公司都使用像甲骨文这样的遗留系统。但是，当您需要扩展时，问题就来了，购买更多的服务器可能会非常昂贵，而且是一项具有挑战性的任务。

随着大数据和云继续渗透到企业中，10gen 和 MongoDB 经历了重大发展。该公司现在为超过 400 家客户的主要项目提供支持，包括迪士尼、Foursquare、爱立信、维亚康姆、西班牙电信和 SAP。事实上，10gen 的客户群比前一年增加了 150%以上。开发人员对 MongoDB 的支持持续增长，每月下载量超过 100，000 次。

新的资金将用于招聘以及国际扩张。10gen 正在爱尔兰和英国开设新的办事处，并在欧洲和亚洲扩大其业务、客户服务和研发。
