# 雅虎老板以某种形式从微软交易中幸存；细节仍然模糊

经过几个月的沉默，雅虎的老板团队开始向沮丧的第三方开发者敞开心扉，谈论强大的搜索平台 T4 的未来。几个小时前，雅虎的 Ashim Chhabra 在 BOSS group 论坛上发表了一篇[帖子](https://web.archive.org/web/20221209121217/http://tech.groups.yahoo.com/group/ysearchboss/message/2422)，解释了为什么雅虎花了这么长时间才向开发者传递信息，并让他们对 BOSS 的命运有所了解。好消息是什么？BOSS 将以某种形式继续存在，但目前还不清楚事情将如何变化，哪些服务将由微软技术提供支持，而且可能会涉及费用。这种不确定性可能会让一些开发人员紧张不安，但至少他们知道这个项目没有被完全废弃。

Chhabra 的帖子显然是由一些沮丧的 BOSS 开发者的行为引发的，他们厌倦了被蒙在鼓里，于是[找到司法部](https://web.archive.org/web/20221209121217/http://www.beta.techcrunch.com/2010/01/07/yahoo-boss-doj/)谈论 BOSS 将如何受到雅虎/微软搜索交易的影响。DOJ 听到了他们的抱怨，安排了下周与他们的电话会议。查布拉的帖子可能有助于暂时安抚他们。我们将他的完整帖子收录如下:

> 乡亲们，
> 
> 感谢您的反馈。我们理解你的沮丧。由于我们与微软协议的复杂性，这一过程对我们所有人来说都是漫长的，我们感谢您的耐心。
> 
> 根据该协议，雅虎！被允许继续提供 BOSS 网络服务，其搜索结果将整合雅虎！由微软提供算法结果的服务和内容。一如既往，我们的目的是提供一个 BOSS 产品，只要这样做有商业和经济意义。我们仍在研究 BOSS 提供的服务将由什么组成，一些由微软支持的服务，雅虎！目前提供，并有可能增加雅虎！未来的内容。
> 
> 在雅虎宣布之前！-微软搜索协议，我们已经分享了我们为 BOSS 探索收费结构的意图。随着 BOSS 未来的发展，我们将继续探索合适的收费结构或其他收入模式。
> 
> 如你所知，在与微软的搜索交易实际实施之前，我们必须获得监管机构的许可。只有这样才能最终确定 BOSS 的未来形态。当然，我们会尽可能提供额外的清晰性和确定性。
> 
> 感谢您的关注！
> 
> 雅虎！BOSS 团队
