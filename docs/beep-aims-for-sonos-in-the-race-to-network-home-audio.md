# Beep 的目标是在家庭音频网络竞赛中推出 Sonos

对于大多数制造商来说，破解整个家庭音频之谜是一种不切实际的追求。唯一一家真正做到这一点的公司，Sonos，仍然稳坐音频发烧友的位置，而三星、苹果、索尼等公司还没有在任何方面做出什么贡献。现在，希望[能完成](https://web.archive.org/web/20221001020352/http://www.beepdevices.com/)。

Beep 是一个非常简单的平台，可以在网络上的任何地方控制音乐。最初的系统是一个 99 美元(预购期后为 149 美元)的盒子，前面有一个大表盘，旨在成为一个满足所有流媒体需求的哑终端。它可以从你自己的收藏或者像 Pandora 这样的服务中抓取音乐，你也可以从你的手机中播放音乐。拨号盘控制音量，可以暂停程序，每一声哔哔声都有一条线路输出。该套件没有内置放大器，但计划将 Beep 技术放入其他制造商的扬声器中。

这个由前谷歌员工丹尼尔·康拉德和肖恩·刘易斯领导的团队从 Squeezebox 和 Pandora 那里挖来人才，制造出他们一直想要的盒子。虽然它现在有点动力不足，但随着今年春天发布的临近，这对夫妇计划增加更多功能。

作为 Sonos 的长期爱好者，我真的很高兴看到一个更便宜、功能齐全的整体家居解决方案。哔哔就是那个吗？也许吧。它有很好的血统，看起来很棒。他们给我看的黄铜和银模型制作精良，易于使用。这些家伙致力于他们的事业，他们有一些很好的经验。谢天谢地，我认为这足以帮助他们解决一个古老的谜题，即如何让音乐在你房子的每个房间里播放。
