# 谷歌钱包与新泽西州运输合作伙伴关系加速发展

如果谷歌和 NJ Transit 对此有什么要说的话，在你的口袋里搜寻公共汽车票价的日子已经结束了。新泽西州州长克里斯·克里斯蒂今天早些时候宣布，该国第三大交通系统[与谷歌](https://web.archive.org/web/20230203094809/http://www.njtransit.com/tm/tm_servlet.srv?hdnPageAction=PressReleaseTo&PRESS_RELEASE_ID=2718)合作，将他们的无线钱包支付系统引入东北部的公交线路和铁路线。

客户将能够在少数几个地点使用他们的 Nexus S 4Gs 来快速跟踪他们的日常通勤。到目前为止，受支持的地点包括纽约宾州车站和纽瓦克自由国际机场火车站的售票亭和窗口。谷歌钱包还可以在 6 路、43 路、80 路、81 路、87 路和 120 路公交车上使用，126 路的一些公交车也支持。

就在昨天，谷歌宣布了他们基于 NFC 的支付技术的少数新零售合作伙伴，但这是谷歌进入公共交通领域的第一大步。对谷歌来说，这是一个聪明的，尽管有些明显的举动。为了让谷歌钱包成为该公司希望的交易处理革命，它需要开始超越零售场所。

将非接触式支付集成到交通系统中当然不是一个新概念，但它有望在公共交通成为一种生活方式的领域推动 NFC 的采用。据美国商务部副总裁斯蒂芬妮·蒂勒纽斯(Stephanie Tilenius)称，“交通一直是全球每一次成功的 NFC 努力的共同元素，也是谷歌钱包成功的关键组成部分。”

谷歌和三星期待已久的 [Galaxy Nexus](https://web.archive.org/web/20230203094809/https://techcrunch.com/2011/10/19/in-depth-hands-on-video-galaxy-nexus-and-ice-cream-sandwich-android-4-0/) 等必备新手机将搭载 NFC 芯片，这绝对没有坏处。随着越来越多的制造商加入 NFC 潮流，谷歌钱包有望成为技术娴熟者的首选支付方式。

从更个人的角度来说，这对于一所通勤学校的前学生来说是一个非常受欢迎的消息。早上第一件事就是处理轻微皱折的美钞和挑剔的机器，这通常会在赶去上课时增加不必要的头痛。感谢新泽西的纳税人(包括我自己)，这笔交易没有花他们一分钱。毕竟，如果谷歌能够证明他们的支付系统能够一天顺利处理数千笔交易，那么执行的代价将是非常值得的。
