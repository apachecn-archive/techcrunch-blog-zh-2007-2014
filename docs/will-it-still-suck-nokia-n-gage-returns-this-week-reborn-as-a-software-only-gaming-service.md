# 还会烂吗？诺基亚 N-Gage 本周回归:重生为纯软件游戏服务

[![nokiareturns.jpg](img/2994471db836a17c2c179c3eedbec5db.png)](https://web.archive.org/web/20221001062114/https://beta.techcrunch.com/wp-content/uploads/2007/08/nokiareturns.jpg "nokiareturns.jpg")

男孩哦男孩做的 [N-Gage](https://web.archive.org/web/20221001062114/http://crunchgear.com/2006/09/12/nokia-n-gage/) 吸。(侧话，有人吗？)但诺基亚本周将把它带回来，这让……嗯，有人感到兴奋吗？是啊，有人对 N-Gage 回来感到兴奋吗？

不过，这次不同了。

N-Gage 不会是一个有蹩脚游戏的蹩脚手机。(我很确定雪球比 N-Gage 游戏有更先进的图形。)不，N-Gage 正在作为一种游戏服务获得新生，它得到了包括艺电在内的几家大玩家的支持，希望在不断增长的移动视频游戏市场中获利。我敢肯定，我不是唯一一个在上课或开会时玩几个小时 *Lumines* 的人。

当 N-Gage 本周推出时，它将只与几部 60 系列手机兼容。当然，随着几周变成几个月，诺基亚将发布越来越多支持这项服务的手机。

你可能会问，N-Gage 还有哪些不同之处？显然，诺基亚去了一家顶级设计公司并与之合作，以确保服务尽可能地人性化。你会发现一些功能和设置的快捷方式，这些功能和设置通常隐藏在菜单结构的某个地方。玩在线游戏时，诺基亚有一个匹配的服务排队，这样你就可以和你认识的人或技能相当的人进行游戏。听起来诺基亚一直在研究是什么让 Xbox Live 如此受欢迎。

正如你所料，游戏会偏向休闲类——益智游戏、体育运动等。我想，对我们来说没有*龙之任务:手持版*。

从我经常出错的角度来看，诺基亚似乎从 N-Gage 的第一版中吸取了教训。它现在不仅抛弃了整个专用游戏手机的东西——如果我想要一个专用的便携式游戏设备，我会拿出我的 DS——而且还清理了用户的游戏体验。如果我们有机会玩这个服务，我们会在这里竖起大拇指或者竖起大拇指。

再弹一遍，诺基亚。第三次了。【纽约时报】
