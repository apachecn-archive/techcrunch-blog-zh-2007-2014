# Listia 与百思买合作，让你以旧换新

由于与百思买的新合作关系，在 Y Combinator 和安德里森·霍洛维茨([和其他人](https://web.archive.org/web/20230328182938/https://techcrunch.com/2011/04/19/andreessen-horowitz-leads-1-75m-round-in-freebie-marketplace-listia/))支持的市场初创公司 [Listia](https://web.archive.org/web/20230328182938/http://www.listia.com/) 上赠送二手商品变得越来越有诱惑力。

当你在 Listia 上送出“旧东西”时，你会获得积分，这些积分可以兑换其他用户正在送出的物品，以及通过[公司的奖励商店](https://web.archive.org/web/20230328182938/http://www.listia.com/rewards)提供的新商品。这些奖励已经涵盖了相当广泛的范围(当商店去年夏天推出时，商品的价值从 20 美元到 16000 美元不等)，但增加 iPads、电视、视频游戏机和百思买的其他产品也不会有什么坏处。

据联合创始人 Gee Chuang 说，这种组合每天都会发生变化。例如，今天，商店里列出的第一批商品包括[一套 Beats by Dre 耳机](https://web.archive.org/web/20230328182938/http://www.listia.com/auction/7913743-beats-by-dr-dre-beats-solo-high-on-ear)和[一台 iPod Nano](https://web.archive.org/web/20230328182938/http://www.listia.com/auction/7895306-apple-ipod-nano-16gb-mp3-player-7th)——这两款产品都由百思买提供。这家商店利用了百思买的 API，但庄通过电子邮件告诉我，这是“一种密切的合作关系”，因此用户可以从百思买发货，甚至在店内提货。

“目前，在这一级别与他们合作的其他公司是信用卡奖励公司，如花旗和大通……但我们的合作伙伴关系是独特的，因为人们可以交易实际商品，并获得最佳购买商品作为回报(而不仅仅是使用信用卡积分等)。)”他说。

该公司表示，超过 200 万用户在该网站上交易了 700 万件商品，移动使用率同比增长了 500%。
