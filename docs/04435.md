# iPhone 泄露变得丑陋:警察突袭 Gizmodo 编辑的房子，没收电脑

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/04/26/the-iphone-leak-gets-ugly-police-raid-gizmodo-editors-house-confiscate-computers/amp/>

 <amp-img src="https://web.archive.org/web/20230311084007im_/https://techcrunch.com/wp-content/uploads/2010/04/jasoniphone.jpg" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 哇。上周，Gizmodo [发布了](https://web.archive.org/web/20230311084007/https://techcrunch.com/2010/04/19/iphone-hd-4g/)一个巨大的[独家新闻](https://web.archive.org/web/20230311084007/http://gizmodo.com/5520164/this-is-apples-next-iphone)当他们得到了最有可能是下一代 iPhone 的东西。当时有很多关于 Gizmodo 行为合法性的讨论(因为他们承认为这个设备支付了 5000 美元)。现在 Gizmodo 刚刚[发布了一个帖子](https://web.archive.org/web/20230311084007/http://gizmodo.com/5524843/)说编辑陈以桐昨晚被加州快速执法联合计算机小组没收了他的四台电脑和两台服务器，他们带着搜查令进入了这所房子。

Gawker 的首席运营官·加布·达比希尔引用加州刑法第 1524(g)条对这些行为做出了回应，该条规定“不得对《证据法》第 1070 条中描述的任何物品签发授权令”，该条款保护为保护新闻机构而获得的信息。达比希尔还指出，加州上诉法院之前已经发现这些保护措施适用于网络记者( *O'Grady 诉高等法院*)。

在 Gizmodo 的帖子中，陈讲述了昨晚的事件。突击搜查开始时，陈不在家，他回家时警察已经在他家呆了几个小时。陈的门被撞开了，因为他不在家。他没有被逮捕，但警方缴获了外部硬盘，四台电脑，两台服务器，电话，等等。

详细说明警方打算没收什么的文件提到了苹果的“4G iPhone 原型”，也提到了“被盗”(Gizmodo 辩称，该设备是在一家酒吧找到的，而不是被盗)。还要注意，所有这些都是在周五晚上发生的，Gizmodo 直到今天才说什么。
