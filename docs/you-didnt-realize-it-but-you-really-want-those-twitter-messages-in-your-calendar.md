# 你没有意识到，但你真的想在你的日历上看到这些 Twitter 消息

 [](https://web.archive.org/web/20230203001955/http://www.crunchbase.com/assets/images/original/0001/7708/17708v1.jpg) 比利时 [Twistory](https://web.archive.org/web/20230203001955/http://twistory.net/) 今天在 [MobileWebCamp](https://web.archive.org/web/20230203001955/http://mobilewebcamp.wikispaces.com/) 推出私人测试版。

这是一个非常简单的工具，由 [Tijs Vrolix](https://web.archive.org/web/20230203001955/http://www.crunchbase.com/person/tijs-vrolix) 打造，用来展示他的编码和设计技能:在任何流行的桌面或在线日历应用程序(iCal、Google Calendar 等)中订阅来自任何 Twitter 用户的消息。).这些消息会在适当的日期和时间自动添加到日历中。

有用吗？我不知道。密切监视/跟踪他人(或你自己)当然很有用。如果你想把我每天的智慧之言添加到你的日历中，我的页面是[这里是](https://web.archive.org/web/20230203001955/http://twistory.net/user/techcrunch)(其中还包括一张每天 twitter 总使用量的图表。
