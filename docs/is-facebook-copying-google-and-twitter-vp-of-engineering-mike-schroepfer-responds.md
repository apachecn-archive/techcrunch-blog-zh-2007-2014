# 脸书在抄袭 Google+和 Twitter 吗？迈克·斯科洛普夫工程副总裁回应 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/09/14/is-facebook-copying-google-and-twitter-vp-of-engineering-mike-schroepfer-responds/>

脸书工程副总裁迈克·斯科洛普夫在今天早上的 TechCrunch Disrupt 大会上与杰森·金凯交谈。对话的主题集中在脸书的许多最新功能上，包括最近推出的好友列表和“[订阅](https://web.archive.org/web/20230203124033/https://techcrunch.com/2011/09/14/facebook-launches-twitter-like-subscriptions-lets-you-share-with-unlimited-users/)”选项，该选项允许用户以非常类似于 Twitter 和 Google+的模式关注他人的公共更新。

当然，这引发了一场关于脸书从其他网站获得了多少“灵感”的讨论。Google+现在正在影响脸书的功能集吗？

杰森今天早些时候在博客上写道，如果你错过了新闻，这个[的“订阅”功能解决了拥有很多朋友的脸书用户最大的痛点之一。这意味着想要关注公众人物更新的超级用户现在有了一种方法，而不必向相关人员提交好友请求。它还为用户提供了一种突破脸书此前限制的 5000 个好友上限的方式。](https://web.archive.org/web/20230203124033/https://techcrunch.com/2011/09/14/facebook-launches-twitter-like-subscriptions-lets-you-share-with-unlimited-users/)

但是脸书·佩奇呢？，杰森想知道。这个功能对比如何？

“对于为一个品牌管理页面的多人来说，Pages 是一个很棒的产品，”Schroepfer 解释说，但 subscribe 是为了与朋友和其他人更广泛地分享。(嗯…听起来像推特，不是吗？)

此外，[脸书还推出了一个新功能](https://web.archive.org/web/20230203124033/https://techcrunch.com/2011/09/14/facebook-launches-twitter-like-subscriptions-lets-you-share-with-unlimited-users/)，可以让你自定义好友更新的种类。例如，如果你只想看到状态更新，而不是游戏的更新，你现在可以这样做。

还讨论了最近增加的自动好友列表。不像 Google+那样把朋友拖放到圈子里，这些列表不需要人工操作。如果人们想使用它们，列表就在那里，否则，它们就会从现在所在的左侧导航中消失。并不过分复杂。

但是单向跟随，从你的流中得到游戏的方法，朋友列表？社交网络领域还有谁在做这件事？Twitter 和 Google+。

杰森问，那么为什么现在推出这些功能呢？(这意味着脸书正直接受到竞争对手产品的影响)。

“一段时间以来，我们一直致力于朋友列表和订阅隐喻的大量迭代，”Schroepfer 回避道。“没有人愿意拖拖拉拉地度过下午，”他说。但他补充道，直到现在，脸书都无法让好友列表真正发挥作用。

在回答一个更直接的问题，即 Schroepfer 最喜欢 Google+的什么功能时，Schroepfer 无话可说。然而，他后来解释说，“这方面的挑战是，无论何时我们推出某种产品，你都可以说这是对 X 或 y 的回应，”但他认为，竞争是巨大的，人们可以选择。

然而，并非所有脸书模仿其他社交网络或新趋势的尝试都获得了回报。最近失败的例子有 Deals(一种类似 Groupon 的服务)、Questions(一种类似于 Quora 或 Yahoo Questions 等的问答产品)和 Places(一种模仿 Foursquare 的签到功能)。所有这些要么在很大程度上降低了重要性，要么完全消失了。

“为了创新，你必须进行实验，”Schroepfer 说。但是要把事情做好，你必须失败，他说。例如，脸书的办公室挂着一个牌子，上面写着*这项工作完成了 1%*。

最重要的是，脸书希望能够快速交付代码……它希望比一家拥有 100 万用户的初创公司更快地向其 7.5 亿用户交付变更。当然，快速移动可能意味着更多的错误。但是处理这个问题是今年的一大焦点，Schroepfer 说。他看到的每一个指标都表明情况正在好转。

但采访中最好的问题可能是杰森大胆尝试获取一些脸书新产品的信息，即脸书音乐和即将推出的 iPad 应用程序。

我们会看到脸书音乐集成到 iPad 应用程序中吗？，杰森问。

“死亡之光”是愉快的，但显然非礼节性的反应。
