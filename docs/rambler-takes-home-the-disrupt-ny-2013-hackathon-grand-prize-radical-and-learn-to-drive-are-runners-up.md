# 漫步者将 Disrupt NY 2013 黑客马拉松大奖带回家，学车和激进是亚军 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/28/rambler-takes-home-the-disrupt-ny-2013-hackathon-grand-prize-radical-and-learn-to-drive-are-runners-up/>

对于 Disrupt NY Hackathon 上的数百名黑客来说，过去的 24 小时刚刚过去，但太阳终于升起来了，是时候对他们以咖啡因为燃料的项目做出评判了。事实证明，这里有很多这样的人——有 164 个注册项目，这是我们迄今为止最大的黑客马拉松，每个演示者只有 60 秒的时间让我们的评委惊叹(更不用说其他观众了)。正如你可能猜到的那样，在一天之内不乏令人惊叹的项目，但我们的评委只能选择一个团队将我们的 5000 美元大奖带回家。

无论如何，我已经受够了——来见见我们最新的黑客马拉松冠军吧！

### 赢家:漫步者

由威廉·霍基、扎克·佩雷特和迈克尔·凯利创建的漫步者是一款网络应用，用户可以在地图上查看他们的信用卡和借记卡交易。在开发过程中，团队利用 Foursquare API 获取位置信息，利用 Plaid API 获取用户消费数据。

### 亚军#1:学习驾驶

由 Jared Zoneraich、Jemma Issroff、Kenny Song 和 Nicholas Joseph 创建的 Learn To Drive 是一款面向通用汽车平台的应用程序，它通过大声说出驾驶指令并显示驾驶统计数据，如行驶英里数、行驶小时数和夜间行驶小时数，来充当虚拟驾驶教练。

### 亚军#2:激进

由萨姆·萨科内、卡尔·塞德纳乌伊和杰夫·埃斯卡兰特创建的[激进](https://web.archive.org/web/20230126030600/http://jenius-radical.jitsu.com/auth)，允许用户用一行代码创建吸引人的日历并嵌入网页。

这三个团队也将在周三下午的主要 Disrupt 舞台上展示他们的项目，但这并不是说其他人都将空手而归。黑客马拉松赞助商 Appery.io、美国电话电报公司、CrunchBase、通用汽车、微软 Bizspark、微软 Skydrive、NewAer、Pearson、三星、Twilio、Visa、Wrigley 和 Yammer 也慷慨地为其 API 和服务的最具创新性和趣味性的使用颁发了自己的奖项。

又是谁决定了这些睡眠不足的黑客的命运呢？我们的评委小组包括 Mahaya 首席执行官塔里克·科鲁拉、Path101 联合创始人查理·安东内尔、Muse 创始人/首席执行官凯瑟琳·明舍、bit.ly 首席科学家希拉里·梅森、FuturePerfect Ventures 创始合伙人贾拉克·乔班普特拉和 BoxGroup 董事总经理大卫·蒂施。
