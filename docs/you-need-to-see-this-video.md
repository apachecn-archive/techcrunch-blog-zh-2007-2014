# 你需要看这个视频(1981 年关于互联网新闻诞生的电视报道)

> “想象一下，如果你愿意，坐下来喝早咖啡，打开家用电脑看当天的报纸。嗯，这并不像看起来那么牵强。”

这是一段 1981 年 KRON 报告的视频开头，该报告预测了互联网新闻报道的崛起。

你得看看这个，这是纯金的。

【YouTube http://www.youtube.com/watch?v=5WCTn4FljUQ&hl=en&fs=1]

我最喜欢的名言:

> 大卫·科尔(SF 考官):“这是一个实验。作为编辑和记者，我们试图弄清楚它对我们意味着什么，以及它对家庭用户意味着什么。我们不是为了赚钱，我们可能不会损失很多，但也不会赚很多。”
> 
> 克朗记者:“这只是计算机办报的第一步。工程师们现在预测，我们通过家用电脑获取所有报纸和杂志的那一天将会到来，但这还需要几年时间。”

这篇文章是由罗宾·沃特斯(拥有家用电脑)写的。:-)
