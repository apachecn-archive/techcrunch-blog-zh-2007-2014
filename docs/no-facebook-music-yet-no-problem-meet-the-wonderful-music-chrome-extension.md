# 还没有脸书音乐吗？没问题！遇见精彩+音乐 Chrome 扩展

[脸书](https://web.archive.org/web/20230204221648/http://www.crunchbase.com/company/facebook)正在与[的一些合作伙伴](https://web.archive.org/web/20230204221648/http://gigaom.com/2011/06/19/revealed-facebook%E2%80%99s-music-plans-involve-spotify-others/)一起准备在[推出一项音乐服务](https://web.archive.org/web/20230204221648/http://news.cnet.com/8301-31001_3-20076098-261/spotify-facebook-in-talks-on-music-service/)——可能被称为[“Vibes”](https://web.archive.org/web/20230204221648/http://siliconfilter.com/vibes-facebooks-upcoming-music-feature/)——在这个令人震惊的流行社交网站上。在你等待的时候，我建议你查看一下为[谷歌 Chrome](https://web.archive.org/web/20230204221648/http://www.crunchbase.com/product/chrome) 开发的惊人的[+音乐扩展](https://web.archive.org/web/20230204221648/http://swarm.fm/+music/)(这里有一个[到该扩展的直接链接](https://web.archive.org/web/20230204221648/https://chrome.google.com/webstore/detail/ddjklapimfghfjjinidpblloipjnnpgb))。

挺酷的。安装后，你会在 Chrome 工具栏中发现一个新图标，可以让你快速搜索艺术家，播放音乐，并在脸书、Rdio、维基百科、Last.fm 等网站上找到艺术家简介的链接。音乐从博客 MP3 或 Rdio(需要订阅)获取，你可以点击图标播放和暂停音乐混合，跳转到下一首曲目或排队歌曲。

在你访问的任何网页上，你现在都可以高亮显示某个艺术家的名字，点击右键，然后立即播放或排队播放该艺术家的歌曲。下面的视频向你展示了这是如何工作的。

这一切都很好，但有趣的是。+Music 与脸书深度集成，允许您与朋友分享完整的歌曲，即使他们没有安装扩展。

它还在脸书的基础上增加了一层实用性，所以当你访问这个社交网络时，你会看到许多增强功能，让你只需点击鼠标就能享受音乐。

首先，在脸书的顶部，你会看到一个迷你播放器，可以让你播放、暂停和跳过曲目，因此不再需要通过单击工具栏中的图标来打开+Music 对话框。

在“分享”部分，你还会看到一个“音乐”选项，可以让你搜索你通常会发布状态更新的艺术家、专辑或曲目。你可以用它来分享 YouTube 视频或即时播放的歌曲，甚至是整张专辑(网络博客上或 Rdio 上的 MP3)。

下面的截图让你对它的样子有一个概念。

为了实现上述目标，该插件使用脸书的 [Open Graph](https://web.archive.org/web/20230204221648/http://developers.facebook.com/docs/opengraph/) 协议在你分享曲目时嵌入流媒体音乐——不需要认证——而 [swarm.fm](https://web.archive.org/web/20230204221648/http://swarm.fm/) 网站则充当通过+Music 分享的任何内容的家园(开发者彼得·沃茨告诉我，还会有更多内容)。

它不止于此。+Music 扩展还可以美化艺术家或乐队页面，在他们的名字旁边添加一个按钮，用于即时音乐流或排队。此外，该扩展会在脸书页面的左栏显示相似的艺术家或乐队。再次，见下面截图。

+Music 扩展背后的大部分技术都是由[Echo Nest](https://web.archive.org/web/20230204221648/http://the.echonest.com/)支持的。

如果你对其他漂亮的音乐应用感兴趣，一定要试试 [Ex.fm](https://web.archive.org/web/20230204221648/http://ex.fm/) 和 [Shuffler.fm](https://web.archive.org/web/20230204221648/http://shuffler.fm/) 。
