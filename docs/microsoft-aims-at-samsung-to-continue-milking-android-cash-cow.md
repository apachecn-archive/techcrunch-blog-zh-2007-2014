# 微软瞄准三星继续榨取安卓摇钱树

![](img/9962b51e1e877042558983ef8a456c02.png)还记得那个[有点令人困惑的等式](https://web.archive.org/web/20230204125201/http://www.mobilecrunch.com/2011/05/27/microsoft-has-made-5x-more-money-from-android-than-they-have-from-windows-phone-7/)留给[微软](https://web.archive.org/web/20230204125201/http://crunchgear.com/tag/microsoft)比 [Windows Phone](https://web.archive.org/web/20230204125201/http://www.mobilecrunch.com/tag/wp7/) 多 5 倍的 [Android](https://web.archive.org/web/20230204125201/http://mobilecrunch.com/tag/android) 利润吗？我们已经详细描述了这种情况，但对于那些错过的人，微软在 Android 技术的某些方面拥有专利。正因为如此，HTC 销售的每一部基于安卓系统的智能手机都能为微软赚取 5 美元的授权费，这是两家公司在庭外达成的协议。我们以为微软已经覆盖了它的 Android 平台，并与重量级公司如 LG 和三星达成了更秘密的协议，但路透社报道称，攫取现金才刚刚开始，这一次三星正处于十字路口。

更重要的是，微软已经提高了赌注，要求每销售一台三星制造的 Android 设备收取 15 美元。这比微软与 HTC 达成的协议价格高出了不少，但我想只有胸怀大志，以后再谈判才是明智之举。据未透露姓名的行业官员称，三星可能会与微软及其 Windows Phone 7 平台结成更强大的联盟，以便将许可费降至每部手机约 10 美元。

根据计算，微软从 HTC 制造的 Android 手机中赚了大约 1.5 亿美元，相比之下，从 Windows Phone 授权中获得了大约 3000 万美元。微软完全没有理由不想对三星做同样的事情，三星即将在下个季度成为世界上最大的手机制造商。现在唯一的问题是，谁是下一个？小心背后，LG。
