# 来源:谷歌正在收购 EtherPad | TechCrunch 背后的公司 AppJet

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/12/04/google-acquires-appjet-etherpad/amp/>

[<amp-img src="https://web.archive.org/web/20230223163450im_/https://techcrunch.com/wp-content/uploads/2009/12/etherpad.png" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230223163450/http://www.etherpad.com/)我们听说谷歌正在收购 [AppJet](https://web.archive.org/web/20230223163450/http://www.appjet.com/) ，这是一家开发流行协作工具 [EtherPad](https://web.archive.org/web/20230223163450/http://www.etherpad.com/) 的小型创业公司。许多前谷歌员工都是 AppJet 的高管，包括首席执行官亚伦·伊巴(Aaron Iba)、首席技术官 J.D. Zamfirescu 和首席运营官·丹尼尔·克莱门斯。收购价格显然只有八位数。我们正在调查更多的细节。**更新** : EtherPad 刚刚[确认了](https://web.archive.org/web/20230223163450/https://techcrunch.com/2009/12/04/google-acquires-etherpad/)这笔交易。

[EtherPad](https://web.archive.org/web/20230223163450/https://techcrunch.com/2008/11/19/etherpad-shows-google-docs-how-its-done/) 是一个强大的实时协作工具，类似于(但优于)谷歌文档文本编辑器。用户可以真正实时地协作，几乎没有延迟。AppJet 实际上是作为[构建](https://web.archive.org/web/20230223163450/https://techcrunch.com/2007/12/12/appjet-makes-simple-web-apps-a-breeze/)简单网络应用的工具推出的。该团队构建了供内部使用的 EtherPad，但后来决定向公众发布。此后，产品重心转移到 EtherPad，开始腾飞。

EtherPad 可能是谷歌文档的一个威胁，但谷歌对该公司的兴趣也很可能与谷歌 Wave 有关。这两种产品有一些重叠的功能，但 AppJet 已经建立了可以部署在公司防火墙后的技术。如果谷歌决定将 Wave 作为企业产品推出，这似乎是合乎逻辑的，这项技术和团队的经验可能会被证明是有价值的。

AppJet 是 Y Combinator 项目的一部分，已经筹集了大约 70 万美元的种子资金。
