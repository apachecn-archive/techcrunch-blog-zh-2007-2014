# 谷歌眼镜更新增加了视频播放器、Path 和 Evernote 的语音操作以及新卡片

谷歌在最新软件更新中为谷歌眼镜添加了一些新功能，用于其实验性的人脸计算设备。有一个新的视频播放器，可以简单地点击开始/暂停，滑动快进和倒带命令，以及新的语音命令，用于发布更新或记笔记，默认情况下使用 Path 或 Evernote glassware，如果你安装了它们。

谷歌选择这些合作伙伴似乎是基于机会，并承诺将有更多受支持的玻璃器皿应用程序可以及时使用这些和其他新的默认语音操作。因此，据推测，一旦脸书提供了一款不仅仅是照片分享的谷歌眼镜应用，你就可以选择它作为发布更新的默认选项，尽管目前还不清楚除了 Path 和 Evernote 之外，事情会如何发展。

谷歌还在设置中添加了一个新的音量控制卡，可以在进行视频通话时检查你的时间线和使用谷歌眼镜的其他功能。这将使您的摄像机静音并关闭您的 POV 视频源，直到您将调用返回到前台。

此外，新的功能还包括免提操作导航时的上下文语音命令、照片和视频的语音字幕以及为两者添加标签的能力、更擅长一次性识别字符串的语音命令、更好的短信支持(显示从手机和谷歌眼镜发送的消息)以及一些新的 Google Now 卡。新卡包括从你的 Gmail 中提取的预订、附近有海报的电影放映时间、公共警报以及为生日提醒卡发送生日消息的能力。

谷歌又一次在谷歌眼镜更新中放弃了一系列新功能，看起来它真的在朝着让这款产品更普遍可用的方向前进。到目前为止，所有的更新都集中在一些功能上，这些功能要么使使用谷歌眼镜的体验更加自然，要么使谷歌眼镜在日常活动中的功能更加丰富。这是否会产生一种正常人真的想戴在头上的产品还有待观察。
