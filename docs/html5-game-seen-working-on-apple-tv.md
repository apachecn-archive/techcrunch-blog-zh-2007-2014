# 苹果电视上的 HTML5 游戏

一名黑客成功地让一款基于 HTML5 的游戏在 Apple TV (2G)上运行，此前他首先越狱了该设备并安装了[沙发冲浪者](https://web.archive.org/web/20230203120407/http://support.firecore.com/entries/259306-couch-surfer-pro)应用程序。(沙发冲浪者为越狱的苹果电视提供网络浏览器。)

这个游戏是一个简单的 21 点纸牌游戏，坦白地说，它看起来并不怎么样。但这一增加值得注意，因为它证明了 Apple TV 是基于 HTML5 的应用程序的功能平台。

当然，苹果是否希望开放 Apple TV 来支持广泛的 HTML5 游戏和应用程序是另一回事。虽然 HTML5 将为简单的基于 Flash 的游戏提供一个替代选择，但它也可能及时成为苹果原生应用商店应用程序的一个令人信服的替代选择。(AppleTV 目前不运行应用程序，但有传言称它最终会运行。)
