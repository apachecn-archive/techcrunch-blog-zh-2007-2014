# 教 Siri 与 RememberTheMilk 互动

这并不是一个新的技巧——你或许可以使用同样的技术向任何 CalDAV 兼容系统推送提醒，但待办事项列表制作人 [RememberTheMilk](https://web.archive.org/web/20230205035605/http://blog.rememberthemilk.com/2011/10/we-taught-siri-to-add-tasks-to-remember-the-milk/) 的人已经让提醒在网上自动弹出。

诀窍是将 RememberTheMilk 添加为 CalDAV 帐户。在设置中，选择邮件、通讯录、日历。添加一个新日历，并在服务器字段中输入 rememberthemilk.com。然后输入您的登录名和密码。然后将 RememberTheMilk 添加到您的默认帐户中。

现在，当你创建一个提醒时，iPhone 会推送提醒以记住米尔克*和*并将其存储在本地。这是绕过苹果目前对 Siri API 的控制的一种方式，尽管它的实用性目前相当有限。
