# 科技遇上地缘科技:俄罗斯搜索巨头 Yandex 以 100 万美元收购地震科技 TechCrunch 的股份

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/06/22/tech-meets-geo-tech-russias-search-giant-yandex-takes-1m-25-stake-in-seismotech/>

Yandex 可能会被视为反谷歌者，因为它追求的是在俄罗斯和邻近国家发展搜索门户业务的战略，而不是在全球范围内大举扩张，但它正在使用另一种途径来扩展业务:通过投资。今天，该公司证实以 100 万美元收购了俄罗斯地球物理勘探公司[Seimotech](https://web.archive.org/web/20221130063043/http://www.seismotech.ru/)25%的战略股份。

这是 Yandex 的首次此类投资，此前 Yandex 的融资活动仅限于电子商务和新技术等领域的科技公司。(它是面部识别公司 Face.com 的早期支持者之一，该公司本周正式被脸书收购。)本质上，这是一个战略性的大数据游戏:作为交易的一部分，Yandex 表示，它将为 Seimotech 提供计算资源和专有技术，用于分析测绘地震数据，以用于石油和天然气勘探等目的。

Yandex 的一位发言人告诉我:“这绝对是一项实验，因为我们正在一个完全不同的领域应用我们关键产品(搜索)背后的技术。”。“搜索仍然是我们的核心业务。”他补充说，如果成功，将会有更多此类投资:“我们可能已经准备好发现其他机会，将我们的知识和经验应用于分布式计算，这是一种我们在核心产品中使用的技术。”

Yandex 提供的技术围绕分布式计算技术领域，允许 Seismotech 在潜在的数千台服务器上处理数据，并强调了大数据游戏如何扩展到其他垂直领域的趋势。

“近年来，IT 行业一直是计算能力的主要消费者。难怪最新、最有效的数据处理技术正是在这个领域发展起来的。与此同时，还有其他计算密集型领域——地球物理学就是一个典型的例子。Yandex 首席执行官 Arkady Volozh 今天早些时候在圣彼得堡国际经济论坛上宣布了这一消息。

这也是一个迹象，表明那些传统上专注于技术投资的人正在进一步向其他领域扩张。巧合的是，在一个

[panel discussion](https://web.archive.org/web/20221130063043/http://www.youtube.com/watch?v=uxv93N_iKYA&feature=plcp)

本周早些时候，在伦敦的 LeWeb 上，三位顶级科技投资者凯文·罗斯、查德·赫利和尼可拉斯·曾斯特罗姆都表示，尽管非科技领域(太空探索是当天提出的一个领域)很有趣，但他们目前没有专注于投资这些领域。

Yandex 表示，通过这笔交易，Seismotech 成为最大的地震数据处理公司之一。它已经拥有像挪威国家石油公司和俄罗斯卢克石油公司这样的大型石油公司作为客户，并且雄心勃勃地要在国际上走得更远。

“Yandex 向我们提供的计算资源和技术将大大提高我们的处理能力，并使我们有机会将我们的服务提升到另一个水平。过去需要我们几个月才能完成的计算现在只需几个小时。这使我们在开发新的地球物理勘探服务并将其提供给我们的客户方面处于非常有利的地位，”地震技术公司的首席执行官 Dmitry Mosyakov 在一份声明中说。

这可能是 Yandex 的第一次投资实验，但不是该公司与地球科学的第一次联系:沃洛赫和他的联合创始人伊利亚·塞加洛维奇(现为首席技术官)都是地质学家的儿子，沃洛赫本人拥有古布金石油和天然气研究所的数学学位。
