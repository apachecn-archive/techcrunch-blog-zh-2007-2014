# 不再是尴尬的少年？游戏化成长

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/12/no-longer-an-awkward-teenager-gamification-grows-up/>

**编者按:** *特约撰稿人 [Joseph Puopolo](https://web.archive.org/web/20230305235841/https://twitter.com/#%21/jpuopolo) 是一位企业家和创业爱好者，他在各种主题上写博客，包括绿色倡议、技术和营销。*

在过去的一年里，你可能已经注意到，曾经小众的趋势不仅悄悄进入主流，而且开始真正引起轰动。游戏化已经成为业内最热门的词汇之一，可能正在接管你身边的网站或用户体验。

对于外行来说，游戏化，简单地说，就是使用游戏设计技术和机制来解决问题和吸引观众。在过去的一年里，甚至大公司和企业也开始加入这场游戏，Gartner 称所有首席信息官都应该关注游戏化，M2 研究公司预测到 2016 年游戏化市场的直接支出将达到 28 亿美元。

好的，我们知道这一趋势正在上升，但是让我们来看看一些将这一趋势推向下一个阶段的参与者。特别是三家公司目前在这个领域引起了一些轰动: [Badgeville](https://web.archive.org/web/20230305235841/http://www.badgeville.com/) 、 [Bigdoor](https://web.archive.org/web/20230305235841/http://www.bigdoor.com/) 和 [Bunchball](https://web.archive.org/web/20230305235841/http://www.bunchball.com/) 。

2009 年秋天，Badgeville 在 Disrupt 的舞台上引起了轰动。该公司在 Disrupt 上获得了[观众选择奖，此后一直顺风顺水，并准备迎接一个伟大的 2012 年。(点击此处查看 Rip 关于 Badgeville 前景的原始帖子。](https://web.archive.org/web/20230305235841/https://techcrunch.com/2011/02/12/techcrunch-disrupt-badgeville-lessons/))

Badgeville 的联合创始人兼首席执行官克里斯·达根(Kris Duggan)在谈到最引人注目和最早采用游戏化的签到王 Foursquare 时毫不留情。这位首席执行官表示，Foursquare 在游戏化方面处于早期尝试，但其激励模式仍存在根本性缺陷。

Duggan 指出了 Foursquare 中的“市长制”:*“你实际上有数百人，只有一个相互排斥的认可点，那就是市长。其他几百人会怎么样？他们不仅没有参与，而且你没有考虑到不同类型的用户。”* Duggan 认为你不仅需要吸引重度用户，还需要吸引中度和轻度用户。你可以吸引每种类型的用户并相应地激励他们，而不是一刀切的方法。

从营销人员的角度来看，这是 Foursquare 为
品牌进行游戏化的一个巨大漏洞。品牌拥有参与战略的能力是其发展模式的关键。一个普通的徽章策略只能让他们到此为止，如果没有对体验和奖励的真正控制，这将不会是一个丰富的体验。

大门公司(T4)的首席执行官 Keith Smith 见证了他们游戏化平台的广泛应用。特别是，首席执行官认识到向营销人员销售的巨大机会:*“今天的营销人员在获取用户上花的钱比努力留住这些用户并为他们提供重返网站的理由还要多”*。通过专注于留住客户，营销人员可以从他们的营销资金中获得更多价值。

Big Door 已经能够获得包括戴尔在内的大客户。戴尔将游戏化超越了他们的网站，并将其用于现场活动，以创建多种方式吸引与会者参与旅程，这是活动期间会议曲目的一部分。每个旅程都是根据特定的会议轨道定制的。QR 码遍布整个活动，与会者被鼓励扫描代码，并因每扫描一次 QR 码和完成一次旅程而获得奖励。

Big Door 甚至制作了一个非常简单易用的小部件，我在我的博客上实现了它。实施之后，我开始发现我的网站上的互动增加了。对于那些希望在自己的网站上尝试游戏化的人来说，这是一个很好的赌注。

Bunchball 的数字战略副总裁 Molly Kittle 指出，直到最近，游戏玩家面临的最大挑战之一是教育潜在客户游戏化和社交游戏之间的区别。莫莉认为未来将从教育转向游戏化策略的采用。

“我承认有些游戏化确实很糟糕，在网站上贴一些徽章既不专业也不强大，我们不希望我们的客户这样做，”她说。对大多数品牌来说，这种权宜之计是不可持续的。莫莉认为，越来越多的企业采用游戏化来解决真正的商业挑战，这将推动整个行业的观念发生巨大变化。嘿，甚至还有 gamify.com，这是市场上最好的域名收购之一。

旅游公司也开始涉足游戏化，Trippy、Gogobot 等公司为他们的用户体验提供游戏元素。但是，再一次，创业公司和公司要想在这种方法中取得成功，游戏化必须超越徽章。

Badgeville、Bigdoor 和 Bunchball 在
游戏化领域看到了一些共同的趋势:

1.  这个空间已经从他们教育客户游戏化的地方迅速发展到客户更早地理解他们的产品
2.  无摩擦安装是必要的。他们都认识到，如果很难实现，那么客户就不会想要实现它。
3.  徽章之外的世界:徽章将变得越来越不重要，真正的虚拟货币的崛起将变得更加突出，以徽章的形式为用户提供更多的虚荣成就。

那么，游戏化的未来会是怎样的呢？

很明显，它正在揭示一个更大的市场机会——达根所说的“行为生命周期管理”的概念游戏化的一些元素将进一步渗透到其他应用中。为了增加用户交互，希望看到某种形式的游戏动态出现在你身边的商业应用程序中。在过去的一年里，游戏化已经摆脱了粉刺和笨拙的声音，走向成熟。
