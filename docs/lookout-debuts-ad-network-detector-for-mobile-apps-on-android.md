# Lookout 推出 Android 手机应用广告网络检测器

有没有想过你的移动应用运行的是什么广告网络？今天，[为许多智能手机平台提供安全服务](https://web.archive.org/web/20221007040113/https://beta.techcrunch.com/2010/05/18/lookout-lands-11-million-from-accel-khosla-and-others-for-smartphone-security-platform/)的公司 [Lookout，](https://web.archive.org/web/20221007040113/https://www.mylookout.com/)发布了一款新的免费 Android 应用，旨在为广告网络提供透明度，帮助你了解手机上发生的事情——从了解你的设备上有哪些网络到了解它们的能力。

正如 Lookout 所说，有时应用程序访问信息的原因并不明显。作为进一步确保用户隐私的一种方式，Lookout 的广告网络检测器应用程序将向您显示手机应用程序中的广告网络可以访问哪些信息。例如，一些广告网络会从您的手机中收集信息，用于更有针对性的广告营销活动。这个应用程序只是让你深入了解哪些信息正在被使用。

Lookout 解释说，他们特别警惕那些将应用外广告推至默认 Android 通知栏的广告网络，在移动桌面上放置通用设计的图标，并改变浏览器设置，如书签或主页。

为了帮助用户了解他们的手机上是否有这些广告网络，几周前 Lookout 发布了一款名为推送广告检测器的应用。当推送广告检测器在手机应用程序中识别出这六个广告网络之一时，它会列出集成了该广告网络的应用程序。它还解释了广告网络可以访问哪些私人信息。

广告检测器如此受欢迎，以至于 Lookout 决定更进一步，将所有广告网络跟踪包括在内。它扫描你的安卓设备上的应用程序，寻找 35 个主流广告网络中的任何一个，包括那些能够显示应用程序外广告的网络。可以检测到的广告网络和交易所包括谷歌的 AdMob、Burstly、GreyStripe、JumpTap、InMobi、Medialets、Millenial、MobClix、MoPub 等。
