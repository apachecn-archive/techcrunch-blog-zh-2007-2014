# 这家初创公司声称其 80%的脸书广告点击来自机器人

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/07/30/startup-claims-80-of-its-facebook-ad-clicks-are-coming-from-bots/>

[![](img/0f3812f17f10c21b4edbdd392b71bbe0.png "Robot Kid")](https://web.archive.org/web/20230404090757/http://wanderingstan.com/otherpics/okinawa/Stan%20Okinawa.jpg)

**更新完毕。许多人喜欢在主要的网络平台上抱怨他们的经历，如[【脸书】](https://web.archive.org/web/20230404090757/http://www.facebook.com/)，但是他们中的大多数人还是坚持作为用户，觉得利大于弊。但是 [Limited Run](https://web.archive.org/web/20230404090757/http://limitedrun.com/) ，一家为音乐人和唱片公司提供软件平台以销售黑胶唱片等实体产品的初创公司，表示在脸书做小企业广告的经历已经让它忍无可忍，因此它将完全退出社交网络平台。**

核心问题是，Limited Run 表示，它发现脸书上收到的 80%的点击似乎来自机器人，而不是真人。该公司在其脸书页面上的一条消息[中解释了这一情况(Limited Run 表示将在未来几天删除这条消息)，这条消息也被黑客新闻](https://web.archive.org/web/20230404090757/https://www.facebook.com/limitedpressing/posts/209534972507958)上越来越受欢迎的帖子转载:

> “几个月前，当我们准备推出新的限量版时，我们开始尝试脸书广告。不幸的是，在测试他们的广告系统时，我们注意到一些非常奇怪的事情。脸书向我们收取点击费，但我们只能确认其中的 20%确实出现在我们的网站上。起初，我们以为这是我们的分析服务。我们试着和其他一些知名公司签约，但我们仍然无法核实超过 15-20%的点击量。所以我们做了任何优秀开发者都会做的事情。我们开发了自己的分析软件。以下是我们的发现:在脸书向我们收费的大约 80%的点击中，JavaScript 没有打开。如果点击广告的人没有 JavaScript，分析服务很难验证点击。
> 
> 这里重要的是，根据我们多年的经验，只有大约 1-2%的人禁用了 JavaScript，而不是 80%的点击来自脸书。所以我们做了任何优秀开发者都会做的事情。我们构建了一个页面记录器。任何时候页面被加载，我们都会跟踪它。你知道我们发现了什么吗？我们付费的 80%的点击来自机器人。没错。机器人加载页面，抬高了我们的广告成本。"

#### 追踪的更多细节

我联系了 Limited Run 的联合创始人汤姆·芒戈，他提供了更多的背景信息。在建立自己的分析程序之前，芒果和他的联合创始人使用了六七个外部分析服务，包括[点击](https://web.archive.org/web/20230404090757/http://getclicky.com/)和[谷歌分析](https://web.archive.org/web/20230404090757/http://analytics.google.com/)。这些都在客户端显示了与 Limited Run 自己的分析程序“大致相同的统计数据”——脸书点击的 80%的访问者没有注册 Limited Run 网站的图片，这可能表明它不是真正被人访问过。接下来，Limited Run 在其应用程序的服务器端内置了跟踪功能，任何来自脸书广告活动的 URL 都会被记录到 Limited Run 的数据库中。这些请求也没有加载任何 Limited Run 的客户端资产。

此外，芒果说，这些访问者在 Limited Run 的服务器上显示为“非标准”用户代理。“本质上，他们不是你标准的 Chrome / Firefox / Safari / IE / iOS 用户代理，”Mango 说。"它们是你会看到的随机爬虫和机器人."我们要求提供更多相关信息——所谓僵尸程序的 IP 地址，一个用户代理字符串的例子——并将确保用 Mango 发送的任何内容更新这些信息。

当 Limited Run 联系到脸书时，他们说他们遇到的是冷漠。芒果说:“遗憾的是，当我们试图对抗脸书时，他只会自动回复‘所有分析服务的表现都不一样’。“需要说明的是，我们花了大约一个月的时间来测试和确认这一切。这不是一个周末项目。”

有限运行清楚地表明，他们没有指控脸书是机器人的幕后黑手。芒果向我承认，这很可能是一种拒绝服务类型的攻击，比如说，来自一个竞争对手——他想增加 Limited Run 的广告账单，知道该公司经营着一家小型自助商店。最后，Limited Run 表示，它只是对脸书不承认这种情况或似乎不做任何事情来阻止它感到失望。

#### 时机问题

在我看来，这里的时间可能有点问题:这一切都发生在“今年早些时候”，芒果告诉我。那么，为什么要等待公布这一情况呢？他说:“自从我们的网站几周前刚刚推出测试版以来，我们一直都很忙，没有时间发布相关内容。”芒果网表示，今天发布这条消息的另一个原因是，Limited Run 与脸书有另一个不利的互动——该公司[表示，它被要求](https://web.archive.org/web/20230404090757/https://www.facebook.com/limitedpressing/posts/209534972507958)花费 2000 美元做广告，以改变其脸书商业页面上的名称。芒果说，那是“最后一根稻草”。

这很公平，但这也有助于 Limited Run 在推出后的头几周获得大量关注。多年来在脸书广告上投入了数百万美元的主要营销人员难道没有注意到这样一件事吗？有可能，但会令人惊讶。

不管怎样，Limited Run 已经停止了在脸书的广告，现在正计划彻底退出社交网络。至于社交网络，Limited Run 计划依靠 [Twitter](https://web.archive.org/web/20230404090757/https://twitter.com/limitedrun) 。“希望其他人在听说了我们的经历后，能更多地意识到他们在未来能从脸书广告中得到什么或不能得到什么，”芒果告诉我。

#### 一个真正以人为本的社交网络

我们已经联系了脸书对这一情况进行评论，并将确保更新我们收到的任何反馈。**更新:**一位脸书发言人发邮件发来以下声明:“我们目前正在调查他们的说法。对于他们的页面名称更改问题，似乎有一些误解。我们不收取页面更改名称的费用。我们的团队正在着手解决这个问题。”

在该公司上周的收益电话会议上，脸书首席财务官大卫·埃伯斯曼说，脸书正在积极努力确保其社交网络只由真人组成，而不是机器人。在 2012 年第二季度，他说:

> “我们改进和完善了识别重复或虚假账户的方法。这些改进导致我们对重复或虚假账户的估计相对于我们之前的全球估计有所增加，这主要是由土耳其和印度尼西亚等新兴市场推动的…由于真实身份对脸书体验如此重要，我们将继续尝试和改进我们的用户管理技术，目标是确保脸书上的每个账户都代表一个真实的唯一个人。”
