# CyberHomes 从测试版中脱颖而出，声称拥有更好的房地产数据 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/11/11/cyberhomes-emerges-from-beta-with-claims-of-better-real-estate-data/>

[![cyberhomes-logo.png](img/feb273f7ed3229b9625c364d8f7e65bb.png)](https://web.archive.org/web/20221209115919/http://www.cyberhomes.com/) 即使房地产市场开始走下坡路，房地产网站仍在继续坚守。周二，在测试一年后， [CyberHomes](https://web.archive.org/web/20221209115919/http://www.cyberhomes.com/) 将正式推出，并加入一个拥挤的领域，包括 Zillow、Trulia、Redfin 和 Realtor.com。CyberHomes 的首席执行官马蒂·弗雷姆认为:“即使房地产市场下滑，网上的观众人数也达到了历史最高水平，而且还在增长。我认为，这里的基本情况是，你只能通过人们的担忧来接近他们，而现在有很多担忧。”作为 Realtor.com 的前首席技术官，弗雷姆非常清楚过去几年人们对房地产的态度是如何转变的。

像许多其他网站一样，CyberHomes 允许房主和潜在买家评估特定房产的价值以及总体市场状况。对于任何一处房产，你都可以看到一份类似房屋的清单，以及它们的售价。一个新的模块化仪表板(基于微软的 [Silverlight](https://web.archive.org/web/20221209115919/http://www.beta.techcrunch.com/2007/04/30/silverlight-the-web-just-got-richer/) )将根据每个访问者的喜好个性化主页。还有一些有用的热图，显示给定城市或地区的住房趋势，一目了然。例如，这是硅谷房价的热图(区域越红，房价越贵):

但究竟是什么让网络家园与其他十几个房地产网站有所不同呢？(毕竟 Trulia 有热图)。Frame 说他的数据更好。

是的，我们以前听说过。(Zillow Zestimate 有人吗？).但网络家园实际上可能有优势，因为它由富达国家金融公司所有，富达国家金融公司是美国最大的产权保险公司之一。因此，Fidleity 国家金融公司对国内的每一笔家庭交易都有直接的看法。“数据的关键优势在于我们是它的主要聚合者。”曼桢说。除了经纪人使用的多重列表服务(MLS)，任何想要获得已出售房产、房屋所有权或抵押贷款发放信息的公司都需要从两个来源之一获得许可:要么是富达国家金融公司，要么是第一美国公司(Zillow 通过一家名为 Dataquick 的中介公司获得数据)。由于 CyberHomes 可以直接进入富达国家金融数据库，Frame 认为，他的网站上的数据更新更加及时，涵盖了 1 亿份美国房地产记录。从明年第一季度开始，它也将开始包括实际的 MLS 数据。

有趣的是，CyberHomes 至少能够找到我住的大楼，而 Zillow 却不能。然而，我很确定 CyberHome 对我的公寓价值的估计是错误的(高达 30%)。这可能是因为 CyberHomes 除了它的位置和面积之外，没有太多关于我的公寓的信息。当然，我可以用不同的滑块进行调整，以更好地反映当地的市场状况，以及我的公寓状况。这使价格进入正确的范围。但是如果我房子的价格是错的，我怎么能相信其他的估价呢？所以我想我仍然在等待一个完美的房地产网站出现。在那之前，网络家园热图是相当不错的房地产色情。
