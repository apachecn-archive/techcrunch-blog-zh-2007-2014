# “哭泣”:史密斯飞船的史蒂芬·泰勒未能起诉匿名博客作者

名人在网络上被模仿。他们很有名——每个人都是匿名的——这是常有的事。大多数名人只是忽略它；但有些人会生气。上周，Kanye West 对 Twitter 用户冒充他的行为非常生气。本周，史密斯飞船乐队的主唱史蒂芬·泰勒大动肝火——事实上，他要将匿名的 T4 博客们告上法庭。

据 NBC 洛杉矶报道，泰勒试图起诉一群博主，他称这些博主冒充他，分享私人事实，甚至在网络上使用他的肖像发表虚假声明。唯一的问题？整个“匿名”的事情。鉴于没有人真正知道这些博客作者是谁，他们自然不会费心出庭。见鬼，我很确定他们甚至不知道他们应该在法庭上。所以法官驳回了这个案子。

显然，泰勒对这些匿名博客作者发表一些关于他去年去世的母亲的评论感到非常不安。我没能找到这些，但我找到了一个围绕“假史蒂芬·泰勒”的强大网络社区在摇滚乐队论坛中有一个受欢迎的[，一个关于脸书的](https://web.archive.org/web/20230114051809/http://www.rockband.com/forums/showthread.php?t=69773)[群组](https://web.archive.org/web/20230114051809/http://www.facebook.com/group.php?gid=23393042538)(也是基于他的摇滚乐队头像)，甚至还有[一个维基百科页面](https://web.archive.org/web/20230114051809/http://en.wikipedia.org/wiki/User:Fake_Steven_Tyler)和[一个古怪的 YouTube 视频](https://web.archive.org/web/20230114051809/http://www.youtube.com/watch?v=m8M_yzaKcco)(嵌入在下面)。

在他的要求下，Twitter 关闭了伪造的 Kanye West 账户，但很难看出泰勒或法院在这种情况下能做些什么。好的第一步是弄清楚你到底在起诉谁。
