# 网飞现在是北美最大的互联网流量来源

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/05/17/netflix-largest-internet-traffic/amp/>

 <amp-img src="https://web.archive.org/web/20230404045648im_/https://techcrunch.com/wp-content/uploads/2011/05/sandvine-bandwidth-chart.png" class="amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 

根据 Sandvine 的一份新[报告，网飞视频流现在是美国互联网流量峰值的最大来源。流媒体视频服务现在占峰值下行流量的 29.7%，高于去年秋天的 21%。](https://web.archive.org/web/20230404045648/http://www.sandvine.com/news/global_broadband_trends.asp)

这使得网飞在北美高峰时段作为下游流量来源的排名超过了 HTTP 网站(18%)、BitTorrent(11%)和 YouTube(10%)。(BitTorrent 仍占所有上游流量的一半)。总体而言，2011 年 3 月，“实时娱乐”(主要是视频流，但也包括流媒体音乐)占下游流量的 49%，而 P2P 文件共享占 19%，网页浏览占 17%。

视频文件如此之大，以至于就消耗的带宽而言，它不需要太多的使用就可以接管。但是这些数字明确指出了一个未来，视频将在互联网上占据越来越多的流量。就在去年 11 月，仅网络视频一项就占了互联网流量的 37%。

<amp-img src="https://web.archive.org/web/20230404045648im_/https://techcrunch.com/wp-content/uploads/2011/05/sandvine-top-traffic-apps.jpg" class="amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>

但是正如网飞首席执行官雷德·哈斯汀斯指出的那样，家庭带宽随着需求不断增加——他预计家庭千兆带宽在十年内将会普及。正如他[本月早些时候告诉我](https://web.archive.org/web/20230404045648/https://techcrunch.com/2011/05/06/hastings-netflix-dvd-down/)“流媒体是我们业务的核心，”但他也在下面的视频中指出，大多数家庭视频都是在网络边缘缓存的，而不是通过主干网传输。
