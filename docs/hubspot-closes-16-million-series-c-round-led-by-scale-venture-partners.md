# HubSpot 以 1600 万美元结束由 Scale Venture Partners tech crunch 领导的 C 系列首轮融资

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/10/19/hubspot-closes-16-million-series-c-round-led-by-scale-venture-partners/>

# 由 Scale Venture Partners 牵头，HubSpot 完成价值 1600 万美元的 C 系列首轮融资

![](img/9c40ab2b35f883aa9e66446b7bb15f69.png)数字营销初创公司[HubSpot](https://web.archive.org/web/20221208222821/http://www.hubspot.com/)[为](https://web.archive.org/web/20221208222821/http://www.hubspot.com/blog/bid/5176/HubSpot-Closes-16-Million-Series-C-Financing)筹集了第三轮资金，总额达到 1600 万美元，使投资于马萨诸塞州坎布里奇公司的资金总额达到 3350 万美元。本轮由[规模风险投资伙伴](https://web.archive.org/web/20221208222821/http://www.crunchbase.com/financial-organization/scale-venture-partners)牵头，之前的投资者[矩阵合作伙伴](https://web.archive.org/web/20221208222821/http://www.crunchbase.com/financial-organization/matrix-partners)和[通用催化剂合作伙伴](https://web.archive.org/web/20221208222821/http://www.crunchbase.com/financial-organization/general-catalyst-partners)加入。

根据宣布投资的博客文章，随着新一轮融资，该公司的估值再次上升。

这家初创公司表示，这些资金将主要用于增加在 R&D 的投资，以解决目前许多公司使用的“破损”的海外营销工具。HubSpot 声称其收入比去年增长了 350%，并补充说其客户数量的增长也是一致的:

![](img/536b0ae5106756e3ce19290bc69adbcd.png)

HubSpot 提供了一个互联网营销应用程序，帮助企业制定搜索引擎优化策略，引导销售线索的生成和转换。该公司落后于[网站评级工具](https://web.archive.org/web/20221208222821/http://www.beta.techcrunch.com/2008/02/04/website-grader-gives-out-free-seo-tips/)，后者本质上是对网站的搜索引擎友好性进行审计，并自夸有专门针对[推特](https://web.archive.org/web/20221208222821/http://www.beta.techcrunch.com/2008/12/22/the-state-of-the-twittersphere-hubspot-edition/)和[脸书](https://web.archive.org/web/20221208222821/http://www.beta.techcrunch.com/2009/01/19/facebook-grader-for-the-sad-sad-people-out-there/)的类似工具。
