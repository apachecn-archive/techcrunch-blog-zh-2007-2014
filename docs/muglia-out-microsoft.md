# 头在云端，鲍尔默将在微软 TechCrunch 工作了 23 年的 STB 负责人穆利亚推出

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/01/10/muglia-out-microsoft/>

微软的核心优势之一是他们的服务器和工具业务(STB)。事实上，对这个巨人来说，这是一个每年 150 亿美元的生意。首席执行官[史蒂夫·鲍尔默](https://web.archive.org/web/20230203162203/http://www.crunchbase.com/person/steve-ballmer)今天在[给员工的电子邮件](https://web.archive.org/web/20230203162203/http://www.microsoft.com/presspass/press/2011/jan11/01-10steveb-mail.mspx)中引用了这个数字，宣布该部门负责人[鲍勃·穆利亚](https://web.archive.org/web/20230203162203/http://www.crunchbase.com/person/bob-muglia)离职。

那么，为什么穆格利亚要离开他一手打造的如此成功的事业呢？因为鲍尔默认为是时候改变了。

鲍尔默写道:“考虑变革的最佳时机是当你处于优势地位时，这就是我们今天在机顶盒方面的情况——领导服务器业务，成功使用我们的开发工具，并准备领导迅速崛起的云未来。"*Bob Muglia 和我一直在谈论整体业务，以及加快我们增长所需的资源。在这种背景下，我认为现在是时候让 STB* 的新领导层就位了，”他继续说道。

翻译:现在是时候认真对待云了，否则就太晚了。

不出所料，穆利亚不会接受降级，而是会离开微软。鲍尔默说，这将在今年夏天发生。

事实上，这有点令人惊讶，因为 Muglia 之前在公司内部被降职过——但在 2009 年就已经开始工作了。[正如我们两年前提到的](https://web.archive.org/web/20230203162203/http://www.techcrunchit.com/2009/01/05/microsoft-promotes-bob-muglia-as-one-of-four-presidents/):

> 在 Hailstorm 未能在博客主领导的隐私挑战中幸存下来之后，服务器和工具高级副总裁 Bob Muglia 从降级中复出，晋升为微软集团总裁。穆利亚现在掌管着微软的一个子公司，该公司 600 亿美元的收入中有 22%来自这个部门。

正如我们当时所写的那样，那次晋升将穆利亚置于“四人帮”之中。这包括娱乐主管罗比·巴赫、办公室业务主管史蒂芬·埃洛普和在线服务集团负责人陆弃权力组合。两年后，除了一人(陆，他刚加入微软)外，其他人都离开了微软。(同样[离去的](https://web.archive.org/web/20230203162203/https://techcrunch.com/2010/10/18/microsofts-ray-ozzie-to-step-down-as-chief-software-architect/)，当然是雷·奥茨。)

你可能还记得，几个月前，当他发表评论暗示他负责的一项技术 Silverlight 被[挤出](https://web.archive.org/web/20230203162203/https://techcrunch.com/2010/10/30/rip-silverlight-on-the-web/)而支持 HTML5 之类的技术时，Muglia 又一次在媒体上惹上了麻烦。这种反弹来得又快又激烈，微软不得不[做出回应](https://web.archive.org/web/20230203162203/https://techcrunch.com/2010/11/01/silverlight-silverlight/)，并收回成命。

" *[Muglia]将继续积极运营 STB，我将从内部和外部寻找新的领导者。鲍尔默写道:“鲍勃将成为新领导的一员，并将为我完成额外的项目。*

下面，找到鲍尔默给部队的邮件全文:

> **发自**:史蒂夫·鲍尔默
> **发自**:2011 年 1 月 10 日星期一
> **发至**:微软–全体员工
> **主题**:机顶盒–建立成功，向前迈进
> 
> 软件业很少有 150 亿美元的生意，微软是唯一一家建立了这三项业务的公司。虽然 Windows 和 Office 是家喻户晓的词汇，但我们的服务器和工具业务已经悄然稳步发展，成为服务器计算领域无可争议的领导者。我们推动了行业的发展，并为整整一代商业应用奠定了基础。我们已经克服了重大的竞争挑战。在过去的 20 年里，STB 中每个人的杰出领导使其成为今天价值 150 亿美元的企业。
> 
> 我们现在准备在成功的基础上继续前进，进入云计算时代。微软和我们的机顶盒团队再一次定义了商业计算的未来。10 月份，我们完成了一个非常成功的 PDC，其中我们详细描述了云的未来，概述了平台即服务，并展示了 Windows Azure 的快速发展。
> 
> 考虑改变的最佳时机是当你处于优势地位时，这就是我们今天在机顶盒方面的情况——领导服务器业务，成功使用我们的开发工具，并准备领导迅速崛起的云未来。Bob Muglia 和我一直在谈论整体业务，以及加速我们增长所需的资源。在这种情况下，我决定现在是时候为 STB 安排新的领导了。这只是认识到所有企业都会经历周期，需要新的和不同的人才来管理这些周期。在整个过程中，Bob 一直是一个出色的合作伙伴，他和他的领导团队制定了正确的战略。
> 
> 随着领导层的变动，Bob 决定在今年夏天离开微软。在我从内部和外部寻找新领导的过程中，他将继续积极运营 STB。Bob 将成为新领导的一员，并将为我完成其他项目。
> 
> 从一开始，Bob 就是我们服务器业务的创始人和领导者。他领导过我们的开发人员、办公和移动设备部门，以及 Windows NT 和我们的在线服务业务的关键部分。这些年来，我曾以多种身份与他共事，我一直很欣赏他对客户的关注、技术深度、人员领导技能以及他的积极能量。我要感谢 Bob 的辛勤工作，许多成就，以及他 23 年来专注于将微软放在首位。
> 
> 我们进入了这个新的十年，机顶盒为当今的商业解决方案提供了平台，并且在推动云计算的未来方面具有独特的优势。我相信 STB 将继续为我们的客户提供卓越的产品和服务，并为我们的业务带来卓越的业绩。
> 
> 谢谢，史蒂夫
