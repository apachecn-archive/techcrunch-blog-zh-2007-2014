在过去的几年里，关于专利价值的争论越来越激烈。特别是软件专利已经受到初创企业生态系统中思想领袖的审查，然而像脸书这样的后期公司继续支付巨额资金收购专利组合。

初创企业对专利有多重视？我们决定直接看数据，而不是推测。我们从 CrunchBase 和 USPTO 中提取信息，建立了一个超过12，000家受资助技术公司的专利活动数据库。然后我们把它插入到RJMetrics中，这是一个让在线业务的数据更容易理解和操作的工具。以下是我们的发现:

  • 三分之一的受资助公司已经提交了专利申请。
  • 19%的受资助公司在获得任何资助之前至少提交了一份专利申请。
  • 自2005年以来，与前一年成立的公司相比，普通初创公司申请专利的可能性越来越小。
  • 半导体行业的公司最有可能申请专利(65.2%)，电子商务行业的公司最不可能申请专利(10.5%)。
  • 在投资了至少100家公司的公司中，投资组合中专利申请者最多的公司是DAG Ventures (59%)、Menlo Ventures (57%)和Kleiner Perkins (56%)。
  • 直言不讳的反软件专利投资者布拉德·菲尔德 (Foundry Group，26.2%)和弗雷德·威尔逊 (Union Square Ventures，18%)身体力行——他们的投资组合中的专利申请人比一般人少。

我们是如何做到的

我们使用了 CrunchBase API 来识别12，404家科技公司的行业和支持者。然后，我们将这些公司的名称与美国专利商标局(“USPTO”)发布的历史专利申请数据库中的记录进行匹配。一旦我们有了数据，我们就能把它上传到 RJMetrics ，只需点击几下就能得到这里显示的见解。

为了提供尽可能最新的数据，我们将研究仅限于已发表的专利申请(未发布的专利，这些专利通常在申请几年后才发表)。USPTO通常在申请18个月后公布非临时专利申请。因此，一些最近的公司结果可能会偏低，因为最近的专利申请可能还没有时间被美国专利商标局公布。

按行业划分的专利率

我们的研究结果显示，33%的受资助公司(12，404家中的4，050家)至少有一项公开的专利申请，但专利率因行业而异。

半导体和生物技术行业的公司专利申请率最高(分别为65%和62%)。这些“硬技术”行业的公司在早期阶段需要大量投资，它们的专利更常用于保护知识产权。相反，电子商务、网络和视频游戏等“软技术”行业的专利率低于20%。

投资者影响

我们的发现阐明了顶级投资者不同的专利哲学。首先，值得注意的是 18.9%的受资助公司甚至在获得第一轮资助之前就提交了专利申请。相比之下，在所有受资助的公司中，有33%在其存在期间的任何时候提交过专利申请。

在拥有至少20家投资组合公司的投资者中，拥有专利申请的公司的平均值和中值均为43%。换句话说，一个典型的风险投资所投资的公司中有43%会在某个时候申请专利。然而，根据基金的不同，这一数字可能会有很大差异。

大公司的融资部门明显倾向于高度重视专利。例如，88%由三星风险投资投资的公司，86%由约翰逊&约翰逊发展公司投资的公司，以及81%由摩托罗拉风险投资投资的公司至少有一项公开的专利申请。

同样，医疗保健投资者似乎非常重视专利，专利申请率最高的是 De Novo Ventures (96%)和 Delphi Ventures (91%)。

一些主要的硅谷投资者，如科斯拉风险投资公司(66%)、达格风险投资公司(59%)、门洛风险投资公司(57%)和T21凯鹏华盈公司(56%)的专利申请率大大高于平均水平。

然而，其他多产的投资者，如 Accel Partners (34%)和红杉资本 (39%)则处于中间位置，他们的专利申请率与普通投资者没有显著差异。

种子期投资者的投资组合中专利申请人最少， 500创业公司 (6.5%)，SV天使 (17.7%)，德国风险投资高科技集团 (15.7%)排在最后。

一些投资者直言不讳地反对软件专利。例如，Union Square Ventures的Fred Wilson 和Foundry Group的Brad Feld 认为软件专利应该被废除。事实上，这两只基金的投资理念似乎反映了这些观点。Union Square Ventures (18%)和Foundry Group (26.2%)的专利申请率都远低于43%的平均水平。

专利队列分析

我们想看看近年来初创公司申请专利的可能性是增加了还是减少了。为了对不同年份的公司进行比较，我们求助于我们选择的武器:RJMetrics中的群组分析。

我们按照公司成立的年份(如CrunchBase中所示)将公司分组，并观察了一段时间内每个公司提交的平均专利申请数量。结果很说明问题。

每条线代表当年成立的公司“队列”(即浅蓝色线代表2005年成立的所有公司)。任何给定的数据点都是该群组中的普通公司在其生命周期中的该点提交的专利申请数量。

这张图表的显著之处在于，它完美地按年份排列了队列线。较低的线意味着一般公司不太可能申请专利。因为每一年的线都比前一年低，一般公司在他们存在的每一年申请专利的可能性都在下降。

同样值得注意的是，随着时间的推移，这些线的斜率会发生什么变化。在2005年和2006年的队列中，线的斜率在以后几年增加。如果你假设公司随着年龄的增长和规模的扩大更有可能开始申请专利，这是有道理的。然而，2007年的线在第2年和第4年之间有一个近乎恒定的斜率。然后，在2008年，我们看到这条线的斜率实际上在下降。换句话说，随着时间的推移，2008年成立的公司申请专利的可能性越来越小。这一趋势在我们所掌握的2009年和2010年的数据中得以延续。

结论

在过去几年中，专利的平均受欢迎程度在受资助的科技初创公司中稳步下降。

单个公司选择申请专利的原因可能很复杂，但关键特征可能会产生重大影响。例如，一家公司的行业和投资者似乎是他们申请专利可能性的有意义的指标。

尽管申请活动整体下降，但那些选择追求专利的公司比以往任何时候都更加积极。这表明市场中的二分法越来越多，一些思想领袖积极反对某些类型的专利，而专利组合则被买卖以获取高额利润。
