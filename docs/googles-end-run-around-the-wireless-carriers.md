# 谷歌的终点是无线运营商

在最近发布的一项专利中，谷歌描述了一个开放无线世界的愿景，在这个世界中，移动设备(尤其是智能手机)不再与特定的蜂窝服务提供商绑定。

今天，当你在美国购买一部手机时，你通常必须签署一份合同，防止你在预定的时间内与一个以上的供应商使用该手机。当你购买笔记本电脑时，你不会遇到这样的要求，它可以用来在任何时间通过任何服务提供商连接到互联网。

谷歌的“灵活通信系统和方法”专利主张，手机用户也应该有通过各种网络和方法连接的自由，他们在任何特定时间和地点选择的通信服务应该由竞争市场力量决定。

![](img/6895405ac25f5c55c0eaf91935f06c4a.png)

这个想法是，例如，当你在家时，你可以通过 WiFi 在智能手机上打电话和浏览互联网，当你在市中心时，你可以在威瑞森，当你在乡下时，你可以在美国电话电报公司。你会根据价格和服务质量做出决定，而你当前位置的覆盖质量起着主要作用。

在某种程度上，iPhone 已经让我们领略到了这将会是什么样子。当靠近 WiFi 热点时，您可以随时决定是使用 3G/EDGE 还是 WiFi 上网。大多数人选择 WiFi 是因为它更快(而且很可能是免费的，除非你在机场或咖啡馆)。但你可能会选择 3G/EDGE，因为它更安全(不用担心开放 WiFi 网络上出现的流量嗅探)。随着 iPhone 上的 VoIP 应用程序的出现，你也可以为你的电话通话做出这样的决定。

现在想象一下，这种选择在移动时可用，并且您有五个服务提供商可供选择，而不是只有两个。不难想象，竞争会导致更低的成本和更好的服务。更不用说，在移动或旅行到一个信号覆盖差的地方后，你不会被困在一个破旧的运营商那里。

正如 Unwired View 所强调的，专利也明确概述了这一点，这样一个系统将需要“一个透明的拍卖市场，无线供应商实时竞价，为用户提供通信服务。”谷歌可能非常适合建立这样一个市场，因为它有 AdWords 和 AdSense 的经验。运营商自己会竭尽全力抵制这种情况，因为他们已经成为愚蠢的管道提供商，无法再将用户锁定在合同中。

这项专利是谷歌更广泛议程的一部分，目的是让尽可能多的人用尽可能多的设备上网。因此有了 [gPhone](https://web.archive.org/web/20221002001720/http://www.beta.techcrunch.com/2008/09/23/some-thoughts-on-the-t-mobile-g1/) ，它对 FCC 的[压力](https://web.archive.org/web/20221002001720/http://www.beta.techcrunch.com/2007/07/22/the-fcc-needs-to-listen-to-google/)，以及拉里·佩奇的[对开放空白空间的支持](https://web.archive.org/web/20221002001720/http://voices.washingtonpost.com/posttech/2008/09/google_guy_white_spaces_test_r.html)。尤其是空白空间的开放可能会为移动设备带来更多的连接点，这些连接点是对无线运营商提供的连接点的一种有吸引力的替代。安卓手机可能是首批利用灵活连接系统的手机之一。
