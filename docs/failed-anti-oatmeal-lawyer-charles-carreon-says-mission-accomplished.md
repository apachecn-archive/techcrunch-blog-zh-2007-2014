# 失败的反燕麦片律师说“任务完成”

在似乎是最后一次寻求名声和古怪行为的爆发中， [Funnyjunk 的律师 Charles Carreon](https://web.archive.org/web/20221231195526/https://techcrunch.com/tag/bearsex/) 告诉 ars，他赢了，因为现在他“出名”和“臭名昭著”。

“任务完成，”他说，显然没有讽刺或自我意识。

燕麦片在 EFF 的律师写道:

“我认为他(卡瑞翁)认识到诉讼不会有任何进展，我们提交了一份摘要，非常详细地概述了他为什么站在法律的错误一边。”

我不变的希望这不会鼓励其他处于事业低谷的人为了得到一点谷歌的果汁而采取可见的网络存在。显然，这在一定程度上是有效的，因为[的网站](https://web.archive.org/web/20221231195526/https://www.google.com/webhp?sourceid=chrome-instant&ie=UTF-8&ion=1#hl=en&output=search&sclient=psy-ab&q=charles%20carreon&oq=&gs_l=&pbx=1&fp=d898465e17f33b65&ion=1&bav=on.2,or.r_gc.r_pw.r_cp.r_qf.,cf.osb&biw=1450&bih=677)在关于凯斯的新闻报道列表中名列前茅。任何一个在 2020 年想要雇佣律师的理智的 web 3.0 企业家，可能会因为看到他们的代表欺负一个漫画家而有点不愿意。
