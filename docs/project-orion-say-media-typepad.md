# 猎户座计划:Say Media 计划将 TypePad 改造成 CMS，并成为网络的 Conde Nast

今天，在 T2 收购科技博客读写网后，我终于见到了 Say Media 首席执行官马特·桑切斯。他没有证实[500 万美元的价格](https://web.archive.org/web/20221005161729/https://beta.techcrunch.com/2011/12/14/price-readwriteweb-5-million/)(他也没有否认)，但他透露了一些其他信息，这些信息揭示了 Say Media 的整体战略。在内部，他们称之为猎户座。

桑切斯说，这是一个专业的现代内容管理系统(CMS)，专门用于处理“从内容创作到广告如何出现”的一切。通常，内容和广告由不同的系统控制。重点将是创造一个精简的，“美丽的体验”，桑切斯描述为“在盒子里有一个艺术部门。”它将有一个“更平滑的界面”，支持多个贡献者，社交分享和后端分析，以帮助编辑更好地规划他们的网站。它也将从一开始就适用于手机和网络。读写网将在第一季度迁移到猎户座，进行重新设计。

Orion 是 TypePad 的定制版本(据说 Media [是去年与 Six Apart](https://web.archive.org/web/20221005161729/https://beta.techcrunch.com/2010/09/21/video-egg-will-acquire-six-apart-and-rename-itself-say-media/) 一起购买的——TypePad 本身将继续作为订阅博客平台得到支持)。Orion 将成为一些 Say Media 自有和附属资产的内容引擎。现在有 16 个这样的网站横跨科技领域(读写网；齿轮巡逻；GdgtTechDirtSplatFAndroid & Me)、Style(XO Jane；时尚达人；老实说…wtf；红毯时尚大奖)、美食(认真吃；Food52 厨房；业余美食家)，以及生活(RemodelistaDogster)。所有人都可以选择迁移，但不会被要求迁移。

这个想法是通过编辑和广告创造优质体验。所有的网站都将有一个干净的外观，网页将有一个高影响力的广告。这种策略类似于美国在线的魔鬼广告搭配优质社论。

“我们可以继续将垂直杂志转移到印刷机上，”桑切斯说，暗示将有更多的收购。“我们的论点是，专业出版物已经发展到将会出现大规模整合的阶段。”

如果你仔细想想。博客行业始于 FM Publishing，它在许多利基资产上提供广告规模。但 FM 并不拥有这些资产，当他们足够大时，他们开始出售自己的广告(这正是 TechCrunch 的情况，它最初是 FM 的附属网站)。桑切斯想提供的不仅仅是广告。他想提供发布平台，帮助举办活动，等等。“这是康泰纳仕，”他说，但对网络而言。
