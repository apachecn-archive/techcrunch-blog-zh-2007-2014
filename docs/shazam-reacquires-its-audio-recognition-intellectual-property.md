# Shazam 重新获得其音频识别知识产权

移动音乐发现平台 Shazam 正在重新获得其在 2005 年出售给身体质量指数广播音乐公司的知识产权。当时，这家初创公司已经出售了涉及 Shazam 核心音频识别技术的知识产权，以帮助资助其开发。Shazam 首席执行官安德鲁·费希尔表示，鉴于 Shazam 最近的增长和向音乐识别以外的其他领域的扩张，现在是该公司重新获得这些权利的恰当时机。

Shazam 作为一款智能应用首次亮相，它可以“听”你收音机上播放的音乐，以便告诉你正在播放什么歌曲。然而，在[最近](https://web.archive.org/web/20230303093800/http://eu.beta.techcrunch.com/2011/01/20/shazam-continues-to-invade-the-living-room-partners-with-cable-channel-syfy/) [几个月](https://web.archive.org/web/20230303093800/http://eu.beta.techcrunch.com/2010/12/06/shazam-hits-100m-user-mark-targets-the-living-room-for-growth/)里，该公司一直致力于将同样的技术用于电视广告。11 月初，[公司还与](https://web.archive.org/web/20230303093800/https://techcrunch.com/2011/11/08/shazam-delivery-agent-partner-to-let-you-buy-your-favorite-tv-characters-stuff/)和[交付代理](https://web.archive.org/web/20230303093800/https://techcrunch.com/tag/delivery-agent/)合作，允许你“标记”(这是 Shazam 行话，意思是“识别”)电视节目本身。第一家参与这一项目的工作室是 NBC 环球，它现在允许你标记“秘密事件”集，以发现电视角色穿的衣服和配饰，并为你提供购买相关 NBC 商品的机会。MTV 和各种广告商也加入了这个被恰当地称为“Shazam for TV”的倡议。

该公司在一份声明中解释说，通过重新获得其核心技术，Shazam 可以将其更充分地集成到其服务中。这不仅意味着音乐 ID 服务，也意味着电视 ID 服务。

首席执行官安德鲁·费希尔说:

> “去年，Shazam 取得了惊人的增长，每周新增 150 多万人，全球 Shazam 用户总数现已超过 1.65 亿。这是回购技术和专利的最佳时机，这样我们就可以继续增加和保护我们不断增长的产品组合，进而使 Shazam 扩大我们的产品范围，进一步开发用于电视和其他创新产品的 Shazam。”

Shazam 回购其知识产权的价格没有披露。根据协议条款，Shazam 将获得知识产权的所有权，而身体质量指数将仍然能够在 B2B 领域使用该技术，通过其身体质量指数识别服务来检测和识别音乐表演。身体质量指数也将成为 Shazam 的股东，两家公司将在未来合作开展其他联合营销项目和服务。

出售知识产权可能有助于 Shazam 在早期起步，但该公司后来转向外部投资，以筹集更多资本。今年夏天，Shazam 筹集了 3200 万美元的巨额资金，由 Kleiner Perkins cau field&Byers 和 Institutional Venture Partners 牵头，DN Capital 参与其中。
