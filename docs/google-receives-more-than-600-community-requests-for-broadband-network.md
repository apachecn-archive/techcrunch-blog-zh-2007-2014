# 谷歌收到 600 多个社区对宽带网络的请求

![](img/f8d6680afe47b4cfb626b21e0f0bb7c7.png)昨天，我们[给](https://web.archive.org/web/20230303201416/https://techcrunch.com/2010/03/25/what-lengths-will-cities-go-to-be-googles-broadband-guinea-pig/)写了一篇文章，讲述了许多城市和城镇为吸引谷歌的注意而付出的巨大努力，希望这个搜索巨头会选择他们的社区作为自己的[实验性光纤网络。](https://web.archive.org/web/20230303201416/https://techcrunch.com/2010/02/10/google-fiber-optic-network-home/)所选城市的宽带网络将完全免费(只有使用服务的消费者需要付费)，1Gb/秒的光纤速度将比大多数美国人今天的网速快大约 100 倍。谷歌的计划是让这个实验覆盖 50，000 到 500，000 人。市长们想尽一切办法来吸引谷歌，包括跳进冰冷的湖泊和鲨鱼池，甚至将一座城市重新命名为“谷歌”。申请成为谷歌试验品的截止日期今天结束，根据一篇[博客文章，](https://web.archive.org/web/20230303201416/http://googleblog.blogspot.com/2010/03/next-steps-for-our-experimental-fiber.html)谷歌已经收到了来自美国各地社区和城镇的 600 多份申请

这是一个令人印象深刻的数字，申请过程还没有结束，离截止日期还有四个多小时。谷歌要求感兴趣的市政当局填写一份信息请求(RFI ),以帮助确定该实验的最佳社区。谷歌估计，每个城市的表格/请求需要大约 4 个小时才能完成。个人和公民也可以提交支持城市的信件；谷歌表示，在过去的一个月里，已经有超过 19 万封信件被提交。

谷歌表示，下一步是审查所有的城市申请和个人支持函，进行实地考察，咨询第三方组织，并与感兴趣的城市的当地官员沟通。最终的城市/城镇预计将在年底宣布。

正如我们昨天所写的，谷歌计划旨在补充美国政府的十年宽带计划，该计划的目标之一是补贴农村地区的宽带连接，并为美国的每个社区带来 1 千兆连接。但政府的战略有许多缺陷，即这些计划不够雄心勃勃。例如，在新的计划下，大约 85%的家庭在供应商方面没有选择，由于缺乏竞争，可能会将用户锁定在更高的价格上。

因此，最终，谷歌代表了一盏小小的希望之灯，因为它可以为其他社区或宽带提供商提供一个具体的榜样。虽然谷歌的最初计划涉及一个非常非常小规模的光纤网络，但如果一切顺利，谷歌最终可能会在全国范围内扩展该项目。

谷歌将在今晚公布最终的申请人数。我们会用最终的统计结果来更新这个帖子。
