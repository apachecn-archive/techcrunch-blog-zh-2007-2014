# 4chan 推出自助广告以保持精简，并让用户付费互相攻击

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/03/will-pay-for-lulz/amp/>

4chan，网络上最狂野的角落之一，正在成长却没有失去它的优势。图像委员会刚刚为斗志昂扬的企业和愿意为 lulz 付费的巨魔开放了一个自助广告工具。创始人 Moot 告诉我“我们已经看到用户用它来互相攻击，为特定的线程或他们喜欢的板做广告，等等。我认为人们有很大的潜力以创造性的方式使用自助广告。”

被称为克里斯托弗·普尔的 IRL，穆特告诉我“我经营 4chan 更多的是作为一种爱好，而不是生意。”该网站允许任何人在它的 60 多个主题图片板上张贴图片和评论。它们的范围从安全的工作话题如[神奇宝贝](https://web.archive.org/web/20230220133402/http://boards.4chan.org/vp/ "Pokémon")和[帕拉诺玛](https://web.archive.org/web/20230220133402/http://boards.4chan.org/x/) l 到明显的成人话题如“性感美女”和“非常/另类”(我没有链接到它，你已经被警告)。4chan 最出名的是/b/

穆特试图保持网站真实的反文化根源，但希望有一种方法使它更可持续，特别是当他把时间投入到他的风险投资支持的初创公司 DrawQuest 的时候，该公司试图激励初出茅庐的艺术家。“这些年来，我们很难做到收支平衡——这让大多数人感到震惊，因为 4chan 每月为 2500 万独立访客提供大约 5.75 亿次浏览量，”Moot 说。

4chan 一直在管理展示广告销售，但这很难在保持精简的同时适应小企业的长尾，因为它需要销售人员。为了在打击垃圾邮件的同时帮助赚钱，4chan 推出了 20 美元的“4chan pass ”,让用户绕过发布到网站上必须完成的验证码。

现在，自助广告可以让网站保持健康，而不需要人员膨胀或外部资金。除了广告创意审批之外，这几乎完全是 DIY，因此自助广告可以带来现金来支付服务器，而团队不必做太多工作。

任何人[都可以购买针对特定主板的](https://web.archive.org/web/20230220133402/https://www.4chan.org/advertise?selfserve)页眉、页面中间和页脚显示横幅。尽管广告可能会出现难以形容的放荡图片，但它们不可能是 NSFW。穆特表示，4chan 广告的一大优势是“起价为每分钟 0.2 美元，最低购买价格为 20 美元，非常便宜。”

所有这些真正吸引人的是，这些广告不仅仅是针对广告商的。最初，自助广告常见问题解答解释说“自助广告是为任何希望推广产品、网站、Kickstarter 项目等的人设计的，或者只是为了好玩/在 4chan 社区前获得消息(“MODS=FAGS”，推广董事会或线程/竞赛等)。”等等，什么？“MODS=FAGS”？我告诉过你 4chan 有点粗糙。这是 4chan 俚语，意思是“这个论坛的版主很烦人”，这是该网站叛逆用户的标志，他们讨厌保持网站可用所需的哪怕一丁点审查和限制。

这句话已经从常见问题中删除了，但是，是的，4chan 鼓励用户购买广告来称呼对方为同性恋憎恶者。我没有纵容这一点。但这个建议可能是为了向该网站的铁杆社区成员表明，尽管它正在走向成熟并更加注重广告，但它仍然牢牢植根于有争议的互联网文化。考虑到 4chan 的种族主义者版块上的血腥、触须和丰富的同性恋色情内容，这些都是攻击性的。

不可否认，4chan 是一个有自己道德准则的社区。通过使其广告产品民主化，它可以避免强势的大品牌或风险投资，并无限期地保留其身份。

为了让人们体验用 4chan 广告能买到什么，这里有一些该网站自己运营的荒谬广告。]
