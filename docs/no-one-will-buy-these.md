# 如今的平板电脑也是如此:HTC Jetstream 和索尼 S 和 P 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/31/no-one-will-buy-these/>

今天不会被明天记住。今天没有重要的事情发生。史蒂夫·鲍尔默没有辞职，戴尔没有退出个人电脑业务，我也没有停止拖延，而是去健身房锻炼。不，就像任何一个无聊的夏日一样，几款产品发布了，初创公司宣布融资，另一家无线运营商[提高了费用](https://web.archive.org/web/20230204175816/https://techcrunch.com/2011/08/31/is-sprints-early-termination-fee-jumping-to-350/)。今天又是新的一天。

但是，当我们还在这里的时候，八月的最后一天，几个主要品牌的平板电脑发布了，值得更多的讨论。索尼终于向[透露了 P 和 S 平板电脑的完整规格和发布细节](https://web.archive.org/web/20230204175816/https://techcrunch.com/2011/08/31/sony-s-p-tablets/)。同样，[的 HTC 也对普契尼的平板电脑](https://web.archive.org/web/20230204175816/https://techcrunch.com/2011/08/31/htc-jetstream-atts-10-1-inch-lte-tablet-launches-september-4/)做了同样的事情，它现在拥有一个相当可预测的名字 Jetstream。你看，这些平板电脑就像所有其他被炒作了几个月的蜂窝标签一样，最终被砰的一声击中，而不是砰的一声。

这年头做个安卓同情者很难吧？我曾经坚定地认为自己属于那个阵营——几个月前，我买了一台 iPad 2，perspective 给了我一记耳光。Android 标签依赖于他们的制造商来弥补 Honeycomb 的空白应用市场。如果没有帮助，它们只是另一个互联网门户设备，与笔记本电脑相比，它的存在毫无意义。这种对生命的呼唤使得笔记本电脑，而不是 iPad，成为蜂巢式平板电脑最大的竞争对手。

今天发布的索尼平板电脑试图用索尼的 Qriocity 媒体套件来证明其新颖性。这种有点模糊的产品几乎可以在索尼制造的所有联网设备上使用，包括 PS3、Bravia 电视、索尼爱立信 Android 手机、PSP 以及所有的机顶盒和联网蓝光播放器。考虑到运行这项服务的设备数量，它的安装基础甚至可能超过 iTunes。但是，如果没有这个媒体平台，索尼平板电脑，嗯，只是更多的安卓平板电脑。

索尼确实内置了强大的 Playstation 支持，这可能是一个杀手级功能——有一天。Tablet S 是第一款能够通过模拟器玩 PSP 和 Playstation 游戏的平板电脑。但是平板电脑只有两个游戏(Crash Bandicoot 和 Pinball Heroes ),根据索尼今天早些时候的新闻发布会，增加游戏似乎不是一个巨大的优先事项。

索尼平板电脑发布后不久，就出现了一些早期评论和动手操作。来自 Engadget [的 Tim Stevens 称](https://web.archive.org/web/20230204175816/http://www.engadget.com/2011/08/31/sony-tablet-s-preview/)索尼平板 S“并没有明显好于”Galaxy Tab 10.1，理由是其奇怪的外形、易刮擦的玻璃和轻薄的手感。所以这是一个通行证？

评论真的需要针对我们受保护的世界之外的消费者。与其他产品相比，它值得你花钱吗？没有警告，没有“如果-那么”语句。迄今为止，还没有一款售价在 500 美元左右的桌子能与 iPad 抗衡。

HTC Jetstream 比索尼平板 s 更难证明这一点。价格为 699 美元*，两年合约*，这款相当笨重的平板电脑只有一个 LTE 无线电和手写笔支持，他们的[传单](https://web.archive.org/web/20230204175816/https://techcrunch.com/2011/06/03/review-htc-flyer/)只能勉强工作。除此之外，平板电脑只是另一个毫无特色的 Android 3.1 标签。这款会比 499 美元的 TouchPad 故障更快。

又是一天，又一个 Android 平板电脑加入了死亡池。但是希望制造商不要把每一次失败都看作是一次浪费的机会，而是一次学习的经历。你可以打赌苹果注意到了这一点，因为 iPad 并不是完美平板电脑的定义。随着每一个失败的平板电脑，通往完美平板设备的道路变得更短了一点，我们最终会走到一起。希望这片应许之地不是一个长满苹果树的封闭花园，而是一片开阔地，到处是嬉戏的机器人。但在那之前，让我们先把所有的蜂巢药片收集起来，然后扔进死亡池。没人会注意到。
