# 晚安，猛男:按出价付费拍卖网站已经死了

![](img/c0028ae56814fea2182ecfd43c1efcbd.png)

当我在 2008 年第一次写下“猛扑 T2”的时候，我觉得它令人憎恶。简而言之，这是一种伪装成拍卖网站的赌博形式。你为出价付费——你买的出价越多，你就越有可能为某件商品支付更低的价格。真正的钱来自抬高价格的傻瓜。之前所有 1 美元的出价都在这个过程中被废弃了。

他们称之为娱乐购物。然而现在，我称之为死亡。

该公司于 23 日在德国申请破产，尽管网站似乎因“技术困难”而关闭，但我认为困难更多的是财务方面。[技术人员](https://web.archive.org/web/20230203041512/http://technologizer.com/2011/03/25/swoopo-quietly-files-for-bankruptcy/)发现该公司正在寻找清算人来剥离其资产，目前与该公司有余额的所有投标人都是 SOLwoopo。

Swoopo 的一些竞争对手仍然存在(我觉得我必须避免链接到他们，除非在指责，所以我会避免在这里)，但希望他们会遭受同样的命运。俗话说，傻瓜和他们的钱很快就会分开。当分手的人把牌堆得对自己绝对有利时，这就变得不道德了。
