# Box.net 在戴尔的新迷你笔记本电脑上获得亮点；我们有五个要送出

![](img/e91e29247721c6f812cab26867bcc24b.png)

在线文件存储和共享服务[Box.net](https://web.archive.org/web/20221207202654/http://www.box.net/)正在帮助戴尔的第一台上网本 [Inspiron Mini 9](https://web.archive.org/web/20221207202654/http://www.crunchgear.com/2008/09/04/meet-the-dell-inspiron-mini-9-netbook-not-too-shabby/) 安装“云”。

只有 4GB 的内置硬盘空间，戴尔需要一些方法来提高容量。因此，它在桌面上放置了一个默认的盒子图标，导致 2GB 的免费互联网存储空间(是盒子免费提供的正常 1GB 的两倍)。

目前，这个图标只是 Box 网站的一个快捷方式，用户可以在这里注册这项服务。然而，这家位于帕洛阿尔托的初创公司暗示，更深层次的桌面整合正在进行中。

不想为这些小家伙掏 350 美元吗？Box 已经向我们的读者免费赠送了其中的五本。每个都将配备 25GB 的存储空间。
