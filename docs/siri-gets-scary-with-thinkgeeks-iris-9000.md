# Siri 用 ThinkGeek 的 IRIS 9000 变得吓人

一次又一次的事实证明，Siri 有一种特殊的幽默感，但是如果在快乐的外表下隐藏着更邪恶的东西呢？那么，将你的 iPhone 4S 接入 ThinkGeek 的新 [IRIS 9000 语音控制模块](https://web.archive.org/web/20230205044527/http://www.thinkgeek.com/gadgets/cellphone/eb7d/)，你可能很快就会知道答案。

理论上，IRIS 9000 非常简单——一旦 iPhone 就位，只需随身携带附带的遥控器，就可以开始工作了。每当你需要 Siri 安排约会(或者[唱首歌](https://web.archive.org/web/20230205044527/http://www.youtube.com/watch?v=kSH6JgAQb14)，按下遥控器上的按钮，让你的请求飞起来。

当然，它本质上只是一个美化的扬声器基座，但实际上看起来完全是另一回事。将 Siri 坚定不移的乐于助人的愿望(即使涉及到[谋杀案](https://web.archive.org/web/20230205044527/http://1876.voxcdn.com/wp-content/uploads/2011/10/siri_weird_verge81.jpg))与 HAL 标志性的红眼结合在一起，注定会让最顽固的科幻迷感到不寒而栗。

我唯一的遗憾？扬声器不会将 Siri 的声音扭曲成更接近原声的声音。话说回来，为了我的精神健康，我想这可能是最好的。遗憾的是，IRIS 9000 还没有完全准备好出货，但这仅仅意味着 ThinkGeek 有时间解决所有(致命的)问题。
