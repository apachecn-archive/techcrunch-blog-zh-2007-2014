# 谷歌 API 分析 Apigee 推出高端产品 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/09/22/google-analytics-for-apis-apigee-launches-premium-products-2/amp/>

 <amp-img src="https://web.archive.org/web/20230221012654im_/https://techcrunch.com/wp-content/uploads/2010/09/free-api-tools-and-enterprise-api-management-products-and-services-apigee.png" alt="" title="Free API tools and enterprise API management products and services | Apigee" class="alignright size-full wp-image-6047 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 我们已经[写过关于](https://web.archive.org/web/20230221012654/https://techcrunch.com/2009/08/25/apigees-ambition-is-to-be-the-google-analytics-for-apis/) [Apigee 的](https://web.archive.org/web/20230221012654/http://www.apigee.com/)，它为 Web 开发者和发布者提供了一个仪表板，用于管理他们提供给其他人的 API 和他们自己使用的 API。它本质上是一个[谷歌 API 分析](https://web.archive.org/web/20230221012654/https://techcrunch.com/2010/04/14/apigee-twitter-api/)。对于依赖来自其他网站和公司的数据的开发人员，它监控他们使用了多少数据(消息请求)、错误率、响应时间和下游用户。目前，每月 2，500 GB 的数据和每秒钟 25，000 条消息在 Apigee 的技术上处理。这项服务是免费的，现在增加了付费功能。

高级功能包括无限制的 API 流量计划(Apigee 每小时每 API 免费高达 50，000 个请求)；关于 API 和应用使用情况的高级和定制分析；开发者密钥供应和管理，以支持复杂的分析、策略、管理和访问层；缓存以获得更好更快的性能；以及对 OAuth 1.0 和 2.0 的支持。

免费工具包括学习、测试和调试 API 的能力，获得 API 性能和使用的分析，以及应用基本速率限制来保护他们的服务。

根据功能和流量，定价为每月 50 美元至每月 1，000 美元。目前，该平台拥有 7000 名免费用户和一些知名企业客户，包括网飞、康卡斯特和 MTV Networks。
