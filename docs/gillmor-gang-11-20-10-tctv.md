# 吉尔摩帮 11.20.10 (TCTV)

在旧金山的 Web 2.0 峰会之后，吉尔摩一伙于本周召开了会议。然而，大部分对话都是关于 TechCrunch 编辑迈克尔·阿灵顿的水危机。我们确实有机会混合一些关于[脸书的](https://web.archive.org/web/20221207102121/http://www.crunchbase.com/company/facebook)新消息平台的态度，很明显只有 [Robert Scoble](https://web.archive.org/web/20221207102121/http://www.crunchbase.com/person/robert-scoble) 已经获得了一个活跃账户。

关于脸书在数据可移植性领域的可信度，该团体分成了两派，阿林顿和 searchengineland 的编辑[丹尼·沙利文](https://web.archive.org/web/20221207102121/http://www.crunchbase.com/person/danny-sullivan)认为脸书是一群骗子，而[Salesforce.com](salesforce.com)cloud blogger[约翰·塔斯切克](https://web.archive.org/web/20221207102121/http://www.crunchbase.com/person/john-taschek)说硅谷的每个人在真实性方面都受到了挑战。吉尔摩，当然，只是为了争论而争论。阿林顿最终转向了对虚拟现实的讨论，留给我们的想法是，无论是谁发明了虚拟现实，都将得到所有的钱。
