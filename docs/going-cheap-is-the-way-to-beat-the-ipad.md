# 华硕变压器现在在沃尔玛的售价为 349 美元

还记得华硕 Eee Pad Transformer 是继激光唱片之后最火的东西吗？还记得当初它以花哨的 IPS 屏幕，低廉的价格，甜美的名字被誉为 iPad 杀手吗？汽车人，团结起来！沃尔玛目前的售价为 350 美元，处于蜂巢标签价格范围的低端。玩笑归玩笑，对于一台安卓平板来说，这确实是一笔大买卖。

问题中的机型当然是[入门级 16GB](https://web.archive.org/web/20230204134702/http://www.walmart.com/ip/Asus-TF101-A1/16419502) WiFi 机型。它拥有大量的输入/输出，一个伟大的 10.1 英寸 IPS 屏幕和双摄像头。建筑质量其实相当不错。这感觉不像是 350 美元，尽管沃尔玛的一些人显然不同意。

这款平板电脑似乎并没有什么特别的销售。运费仅为 0.97 美元，349 美元，这是主流蜂巢平板电脑的租金。Wally's World 销售的宏碁 Icona Tab A500 售价 398 美元，8GB 东芝 Thrive 售价 398 美元。多花钱少办事，嗯。寻找一个 Android 平板电脑和 ThinkPad 平板电脑太贵了？请选择这个。
