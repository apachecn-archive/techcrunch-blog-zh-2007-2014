# 霍尼韦尔推出信号校正 HDMI 电缆

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/29/honeywell-introduces-signal-correcting-hdmi-cables/>

我从没想过我会写关于电缆的博客，但现在我做到了，这太棒了。霍尼韦尔推出了我称之为智能电缆的产品，这是一种内置纠错功能的标准 HDMI 电缆。太酷了。

有线电视的“大脑”寻找损坏的 HDCP 和 EDID 数据，并即时修复，这意味着您的图像更清晰，更逼真，音频更清晰。你会注意到宽高比会更准确，颜色也会更准确，更不用说环绕声的削波会更少。

还没有定价，但预计将支付溢价。如果你正在升级或者没有高端家庭影院系统，它不适合你。但是如果你是一个铁杆音频或视频发烧友，你可能已经对着你的键盘流口水了。今年圣诞节，在你的电子商店寻找它们。
