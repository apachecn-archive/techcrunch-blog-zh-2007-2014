# 《一家人》本周一登陆 Xbox Live

[![famguyxbl.jpg](img/a2597249649d4199384a82555a0b6da7.png)](https://web.archive.org/web/20220928201625/https://beta.techcrunch.com/wp-content/uploads/2007/08/famguyxbl.jpg "famguyxbl.jpg")

谁不爱*家的家伙*？(好吧，我，但现在这无关紧要。)从周一开始，微软 Xbox Live 的用户将可以下载《精神分裂症》漫画前两季的剧集。值得注意的是，随着令人印象深刻的 DVD 销售， *Family Guy* 代表了微软帽子上的羽毛，因为该节目尚未在 iTunes 上提供。每集大约要花费 2 美元，由微软伪造的博洛尼亚在线货币兑换而来。
