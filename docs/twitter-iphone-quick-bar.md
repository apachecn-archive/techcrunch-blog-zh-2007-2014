# 有了“快速栏”，Twitter 已经知道如何将广告塞进信息流

今天，Twitter 刚刚为其 iOS 版应用程序推出了一个非常重要的新更新。几个新功能看起来不错，比如新的 Tweet box 设计，以及通过电话联系人更容易找到朋友。但最有趣的新功能是“快速栏”，因为它最终给了 Twitter 一种从应用中赚钱的方式。这已经惹恼了一些用户。

需要说明的是，Quick Bar 目前只在 iPhone 版本的 app 上找到。它位于推文流的顶部，并在当前热门话题中循环(你可以自己滑动来循环)。但关键是这里还包括了推广的趋势话题。换句话说，它将广告推送到推文流中。

现在，我同意这些比你在网上看到的大多数广告要好——尤其是在其他应用程序中。但它们仍然是广告。iPhone Twitter 客户端的用户(即使当时它还被称为 Tweetie)之前并不习惯看到广告。因此，已经出现一些反弹也就不足为奇了。仅在我的流中，对新特性的第一反应是想知道是否有办法关闭它。(据我所知，你可以禁用局部趋势，但不能禁用整体功能。)

在很大程度上，人们看起来最不高兴是因为这个功能被推到你面前，而不一定是因为广告是它的一部分。(事实上，如果你不在流的顶端，这个功能就会覆盖在推文上，这很烦人。)但很明显，广告是这个专题的原因。

无论如何，Twitter 在努力赚钱的过程中尝试这种方法是一个很明显的想法。我迫不及待地想知道这些推广的趋势话题的点击率会是多少。意外点击率会是多少…
