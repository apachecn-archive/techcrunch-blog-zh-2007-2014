# 视频:日本防卫省开发出令人敬畏的球形无人机

今年六月，世界第一次看到了“世界上第一个球形飞行器”，但是它的制造者，[日本防卫省](https://web.archive.org/web/20230204120532/http://www.mod.go.jp/e/index.html)，几天前决定是时候在东京公开展示它了。正如你在下面的视频中看到的，他们的球无人机非常棒。

无人机可以在半空中静止不动，以高达 60 公里/小时的速度垂直和水平地穿过狭窄的空间，并且(这非常酷)在撞到地面或墙壁时保持移动。由于它体内的三个陀螺仪传感器，即使它被障碍物击中，它也可以保持飞行。

同样令人印象深刻的是，这架无人机是由总成本仅为 1400 美元的零部件制成的。
