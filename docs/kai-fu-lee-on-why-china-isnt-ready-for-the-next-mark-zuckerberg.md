# 李开复谈为什么中国还没准备好迎接下一个马克·扎克伯格

[InnovationWorks](https://web.archive.org/web/20230203064657/http://www.chuangxin.com/) 创始人和前谷歌员工[李开复](https://web.archive.org/web/20230203064657/http://www.crunchbase.com/person/kai-fu-lee)今天和[莎拉·莱西](https://web.archive.org/web/20230203064657/http://www.crunchbase.com/person/sarah-lacy)一起来到 TC Disrupt Beijing，谈论中国的创业生态系统。

InnovationWorks 是一家专注于中国早期互联网公司的中国孵化器——对初创公司的兴趣正在爆炸式增长，InnovationWorks 看到了 7000 份简历，以回应其第一批公开申请，“没有什么比得上中国企业家对成功的渴望，”他说。

李开复认为创新工场的未来就像一个中国的理想实验室，是安德森·霍洛维茨/Y 组合公司的结合体。孵化器越来越向 VC 靠拢，新基金总额近 2 亿美元。Lee 想给 Andreesen 的内部完成一切的战略加上中国特色，“我们利用了高估价和低建筑成本的优势。缺点是我们没有十年的等待时间。”

李开复表示，创新工场目前在中国初创企业创始人身上寻找三个特征，除了勤奋、tec、a)做足功课 b)以往经验——真正的技能和增值空间 c)成为领导者的特征

在“以前的经验”这一点上，莱西提到了李开复的观点，即中国还没有准备好迎接具有颠覆性的“头顶灯泡”式想法的企业家——祖克伯格、安德森，甚至杨致远或陈士骏。

“中国社会并不支持失败，”李开复解释说，正因为如此，“在目前的状态下，很难找到一个 20 岁的辍学者有新颖的想法。”

李开复并没有排除中国的马克·扎克伯格(也就是一个改变世界的年轻人)的想法，但暗示腾讯创始人马化腾是一个例外，相反，创新工场指望像人人网的杰克·肖、文森特·唐和马恺文这样的人，“都是有经验的人。”

“我们认为我们的命中率非常高，”他说，“我们需要花时间与那些会提高命中率的人在一起。”
