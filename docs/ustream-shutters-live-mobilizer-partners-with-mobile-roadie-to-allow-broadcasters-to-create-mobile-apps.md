# Ustream Shutters Live Mobilizer 与 Mobile Roadie 合作，允许广播公司创建移动应用程序

去年，流行的直播视频平台 Ustream [推出了](https://web.archive.org/web/20230203044433/https://techcrunch.com/2010/06/17/ustreams-live-mobilizer-gives-bands-and-brands-live-streaming-iphone-apps/)一个名为 Ustream Live Mobilizer 的移动应用开发平台，为品牌、名人和乐队提供一个可定制的 iPhone 应用，该应用具有 Ustream 的直播功能。但今天，Ustream 关闭了 Live Mobilizer，并宣布与 [Mobile Roadie、](https://web.archive.org/web/20230203044433/http://www.mobileroadie.com/)合作，允许任何人开发和创建流畅的富媒体 [iPhone](https://web.archive.org/web/20230203044433/http://techcrunch.com/2009/04/20/mobile-roadie-builds-bands-custom-iphone-apps-on-the-cheap/) 和 [Android](https://web.archive.org/web/20230203044433/https://techcrunch.com/2010/02/22/mobile-roadie-now-creates-apps-for-android-ecosystem/) 应用。

Mobile Roadie 将与 Ustream 广播公司合作创建移动应用程序，展示他们的 Ustream 直播频道以及许多其他免费功能。例如，Mobile Roadie 的 Ustream 应用程序将在广播公司上线时自动发送推送消息，允许访问者“签到”流媒体并分享到脸书和 Twitter，收集广播公司直播流媒体的实时分析和位置数据，并允许用户在观看直播流媒体时相互聊天。

广播公司也将能够利用该应用赚钱，并可以向 Ustream 内容的用户收取一次性或每月费用。Mobile Roadie 表示，使用 Live Mobilizer 的现有 Ustream 客户端将可以选择迁移到新平台。

将移动应用程序开发外包给 Mobile Roadie 对 Ustream 来说很有意义，他们可以专注于他们最擅长的事情——直播。Mobile Roadie 已经成为许多知名品牌和名人使用其平台的事实上的移动应用程序开发平台，包括泰勒斯威夫特(参见本文中 Ustream 集成的例子)、麦当娜、Live Nation、李维斯、暮光之城和维拉王。2009 年，Mobile Roadie 还为 [LeWeb、](https://web.archive.org/web/20230203044433/http://www.leweb.net/)开发了官方 iPhone 应用程序，与[兰登书屋](https://web.archive.org/web/20230203044433/https://techcrunch.com/2009/12/21/mobile-roadie-random-house-parnter/)达成协议，为作者开发 iPhone 应用程序，与[索尼音乐](https://web.archive.org/web/20230203044433/https://techcrunch.com/2011/01/20/mobile-roadie-and-sony-partner-to-launch-mobile-apps-for-music-artists/)为音乐品牌的艺术家网络开发移动应用程序。
