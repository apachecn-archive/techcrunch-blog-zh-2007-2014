Vox Media通过推出Vox Creative加大广告力度，首席执行官Jim Bankoff表示公司今年将实现盈利TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2013/04/29/vox-media-steps-up-its-ad-push-with-the-launch-of-vox-creative-an-in-house-marketing-design-agency/

随着Vox创意的推出，Vox媒体加大了广告力度，首席执行官吉姆·班考夫表示，公司今年将实现盈利

Vox Media 是 SBNation 、 Verge 和 Polygon 博客的出版商，随着Vox Creative的推出，Vox Media 开始着手将其内容货币化，Vox Creative是该公司的一个新部门，将作为一项内部服务来创建营销和广告，与其他内容服务一起运行。TCDisrupt的首席执行官吉姆·班克夫(Jim Bankoff)今天在台上宣布了这一消息，他称赞了“优质内容”的优点，并远离了他认为当今互联网内容普遍存在的“逐底竞争”。他还表示，Vox预计今年整个投资组合都将实现盈利。

“你会看到丑陋的广告……但我们认为，优质、高质量的广告有巨大的市场可挖，”他说。Vox的目标是一个2500亿美元的广告产业:Bankoff说其中65%的支出用于品牌建设。在互联网领域，这一数字甚至更高，250亿美元的营销中有80%用于品牌建设和直接回应。“它必须成长，我们希望在那里抓住它，”他说。

重点是制作视频，强调高质量的设计和讲述故事，“使用我们使用的相同工具。”该公司去年推出了一个名为 Chorus 的新内容管理系统，以敏捷的方式开发自己的博客内容。Vox Creative将由Chad Mumm负责，他曾负责Vox工作室，是Vox Media旗下三个网站的视频主管。尚不清楚他之前的角色是否会被新角色取代或合并。

Vox Creative的想法是利用Vox围绕Verge和SBNation等网站建立的声誉。Bankoff说，这是品牌营销者接近Vox开发广告的结果；他将他们的宣传描述为:“我们希望以Verge与观众沟通的方式与观众沟通。它对消费者有吸引力。”由于多次听到这种说法，Vox决定采取行动。

他说，Verge和SBNation已经盈利，Polygon仍处于增长阶段，但将在今年晚些时候盈利。该公司一直在逐步扩大三者涵盖的编辑内容，最初被视为“科技博客”的Verge现在也进入了科学等领域，主编约书亚·托波尔斯基(Joshua Topolsky)称之为“现在的文化”Bankoff今天说，除了广告，公司还计划进入新的垂直市场，但没有具体说明是什么。

他表示，此举是为了让广告像Vox平台上的内容一样引人注目、引人入胜。“这是对行业的伤害，它伤害了我们作为消费者和作为投资的出版业，”他在谈到不良广告时说。

他还很快指出，虽然这是关于利用Vox在内容和设计方面的声誉，但这并不意味着这将是广告。“我们不相信赞助帖。我们有一堵非常坚固的墙。我们看上下文，”他坚持说。

此举是该公司试图撼动新旧媒体的最新举措，包括其竞争对手美国在线，该公司获得了约2350万美元的风投资金。“我们的投资者看着我们，问我们是否可以像其他人扰乱医疗保健和其他行业一样扰乱媒体，”Bankoff说。
