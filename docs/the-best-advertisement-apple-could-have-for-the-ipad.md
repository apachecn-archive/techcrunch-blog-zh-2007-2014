# 还有比这更好的 iPad 广告吗？

【YouTube = http://www . YouTube . com/watch？v=gQFcgUqTg0M&fs=1&hl=630]

毫无疑问，苹果在制作能引起消费者共鸣的广告方面有着悠久的历史。随着[最近的 Facetime 广告，](https://web.archive.org/web/20221208212721/https://beta.techcrunch.com/2010/07/10/apple-facetime-commercial/)苹果公司更进一步，吸引了人类的情感。但老实说，有时候最好的广告并不是由大牌机构或著名导演制作的。TechCrunch 读者 [Paul Sanduleac](https://web.archive.org/web/20221208212721/http://paulsanduleac.com/) 给我们发来了他四岁弟弟使用 iPad 的视频。简直令人印象深刻。

这个小男孩不仅可以在设备上滑动，还可以玩各种应用程序，包括键盘应用程序和一些游戏。你可以看到他正在享受他的生活，同时还与内容互动，学习字母和颜色。

自从苹果首次推出这款设备以来，iPad 一直被誉为儿童与应用程序和媒体互动的革命性方式。但有时只需一个简单的视频就能提醒你这种设备对孩子有多有效。
