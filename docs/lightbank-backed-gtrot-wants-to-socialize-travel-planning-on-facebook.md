# Lightbank 支持的 Gtrot 希望将脸书的旅游规划社会化

![](img/ada248b02742d198ea924ce3a1c56e69.png)Gtrot，一个利用脸书帮助人们做出旅行决定的服务，宣布了来自 [Lightbank](https://web.archive.org/web/20230204225423/http://www.crunchbase.com/financial-organization/lightbank) 的新一轮资助。该公司拒绝透露这轮融资的确切规模，但表示不到 100 万美元。

Gtrot 由布列塔尼·拉夫林和扎卡里·史密斯创建，帮助用户通过他们的社交图从其他朋友那里获得旅行计划、交易、提示和推荐。例如，如果你要去巴黎旅游，需要旅行建议，你可以使用 Gtrot 的脸书应用程序或网站来查看你的哪些朋友去过或住在巴黎，这样你就可以联系他们，获得个性化的建议和推荐。

与 TripIt 类似，您可以通过电子邮件向 Gtrot 发送您的旅行确认，该网站将自动管理您的行程，并从脸书的朋友那里获取您要去的城市的信息。Gtrot 会找有相交计划的朋友，住在目的地的朋友或者之前去过目的地的朋友。您还可以设置提醒，提醒朋友计划前往您正在访问或居住的城市。你也可以上传过去旅行的照片和评论，与脸书的朋友分享。

拉夫林和史密斯说，像猫途鹰这样的在线论坛已经成为推荐的事实来源，但是这些网站不能从你信任的人那里给你推荐。这些网站无法将你和你的朋友联系起来，这些朋友要么去过度假胜地，要么在那里生活过。

自 2010 年进入测试阶段以来，gtrot 已经吸引了近 6，000 名用户，他们共享了全球 40，000 多个城市的信息，并计划将资金用于进一步的产品开发和扩展。
