# GPS 与蜂窝 GPS:笔记本电脑上的头对头

全球定位系统是很酷的玩具。不要误解我的意思，它们对某些人来说是很重要的设备。我希望我的出租车司机使用它们。但在大多数情况下，它们并不是必不可少的:没有它们，我们已经生存了几千年，所以是时候承认它们是一种便利了。
 所以虽然没有它我也能活下去，但我宁愿不要。对我们所有人来说幸运的是，现在有两种口味:安装在仪表板或车把上的独立 GPS 单元，以及手机 GPS，它使用你的蜂窝网络来获取数据，手机本身接收坐标。

现在，[我们已经对各种类型的 GPS 单元做了自己的总结](https://web.archive.org/web/20221007171020/http://crunchgear.com/2007/08/06/help-key-how-to-shop-for-a-gps-unit/)，但是笔记本电脑在周二将 GPS 与蜂窝 GPS 的枪战放在一起，这超出了我们自己指南的范围。这是经过深思熟虑的，很容易理解。如果你想多了解一点 GPS 和手机上有 GPS 的区别，请看一看。我们个人对手机版更满意，但是，我们倾向于知道我们要去哪里。
