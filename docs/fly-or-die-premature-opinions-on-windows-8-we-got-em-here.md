# 飞还是死:对 Windows 8 的不成熟看法？我们找到他们了。

Erick 和我在今年 Build 随我们一起发送的 Windows 8 预发布平板电脑上做了一次快速的飞行或死亡，我们被分为蓝色(死亡屏幕)状态/灰色(死亡屏幕)线。我发现预发布版有所欠缺且漏洞百出(我并不孤单)，但我[知道我看到的未来](https://web.archive.org/web/20230205033826/https://techcrunch.com/2011/09/13/previewing-the-future-hands-on-with-windows-8/)。不管你喜不喜欢，这是下一代 Windows 8，正如我所说，这是一个“PC”操作系统，而不是平板电脑、笔记本电脑、触摸屏或移动操作系统。是 Windows。

另一方面，Erick 并没有被打动，但我相信一旦他意识到交互面板实际上非常酷，并且一旦更多的应用程序支持全屏 UI，愚蠢的 Windows/Metro 转换将不再是一个问题，他就会回心转意。我同意这个版本很奇怪，需要大量的工作，但这是微软的大推进，他们必须让它不可思议。我愿意给他们多一点时间，但我不确定在 Android 和 iOS 将他们缝合之前，他们还能在 slate 市场混多久。
