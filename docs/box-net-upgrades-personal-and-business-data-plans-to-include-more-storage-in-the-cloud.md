# Box.net 升级个人和商业数据计划，在云中包含更多存储

![](img/28722373fbbf09c48b9c445b1637f6f9.png)基于云的存储和共享应用[Box.net](https://web.archive.org/web/20221208050930/http://www.box.net/)今天对其免费和付费计划进行大数据存储升级。该公司正在[增加](https://web.archive.org/web/20221208050930/http://blog.box.net/2010/10/28/box-and-the-infinite-cloud/)其个人、商业和企业计划的数据存储量。

Box 的个人订阅计划现在提供 5GB 的免费网络存储空间。Box 的商业计划正在被略微调整，这样公司就不用为个人用户付费了(以前每个用户给公司 15GB 的空间)。现在，企业将自动获得 500GB 的数据存储，是普通企业用户可访问的存储量的三倍。Box.net 的首席执行官 Aaron Levie 说，企业客户将可以访问无限量的数据存储，使公司能够在云端管理万亿字节的数据(Box 向企业客户收取每个用户每月 35 美元的费用)。

为了适应这些变化，Box 建立了两个企业级数据中心，并获得了 SAS 70 Type II 认证。据该公司称，包括夏威夷航空公司、T-Mobile 和 ABC News 在内的 60，000 多家企业使用 Box 作为数据存储库。Levie 的未来愿景是，企业和个人将永远不必担心数据存储，Box 将成为访问他所谓的“无限云”的首选平台。

今年早些时候，Box 筹集了 1500 万美元的新资金，对于一家与微软 Sharepoint 等公司竞争的初创公司来说，它已经实现了稳步增长。自 2005 年推出以来，Box.net 已经积累了超过 400 万用户，有数十万商家在使用该应用。去年，这家初创公司的收入增长了 500%，由于与奥普拉温弗瑞网络、沃尔沃和诺基亚西门子的交易，今年的销售额也有所增长。

这家初创公司在过去一年里还不断推出新功能和新产品，包括[文件同步功能、](https://web.archive.org/web/20221208050930/http://www.techcrunchit.com/2010/06/10/box-net-moves-beyond-storage-in-the-cloud-adds-file-syncing-to-the-mix/)安卓应用、 [HTML5 功能](https://web.archive.org/web/20221208050930/https://beta.techcrunch.com/2010/05/28/box-net-adopts-html5-adds-drag-and-drop-functionality-between-the-web-and-desktop/)等等。
