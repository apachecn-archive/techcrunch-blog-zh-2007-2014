# SEC 观察:云存储初创公司 Box.net 再融资 1860 万美元

根据一份提交给 SEC 的文件，Box.net 已经筹集了 3500 万美元中的 1860 万美元。我们已经向公司证实了这一点。这使得 Box 的总资金达到 9600 万美元。

Box 拥有 600 万用户，存储 3 亿份文档，是一个面向企业的云存储平台，具有协作、社交和移动功能。Box 已经[发展](https://web.archive.org/web/20230204212934/https://techcrunch.com/2011/01/20/box-net-upgrades-cloud-storage-platform-with-new-ui-collaborative-features-and-more/)不仅仅是一个 fils 存储平台，而且已经成为一个成熟的协作应用程序，企业可以在其中实际交流文档更新、[远程同步文件](https://web.archive.org/web/20230204212934/http://www.techcrunchit.com/2010/06/10/box-net-moves-beyond-storage-in-the-cloud-adds-file-syncing-to-the-mix/)，甚至添加来自 Salesforce、Google Apps、NetSuite、Yammer 等的功能。

今年早些时候，Box 刚刚从 Andreessen Horowitz 和其他人那里筹集了 4800 万美元，但该公司的首席执行官兼联合创始人 Aaron Levie 告诉我们，由于强劲的风险市场、公司的强劲表现以及在云领域的增长，新一轮的“扩张”是有意义的。他表示，这是今年早些时候 D 融资的后续轮次，将包括现有投资者和新投资者。

Levie 说，这一轮还没有完全结束，可能会发生变化。文件中列出的是目前的投资者 Draper Fisher Jurvetson，U.S. Venture Partners 和 Scale Venture Partners。
