# 涡轮税制造商集中数百万美元游说反对简化纳税申报

在技术最先进的国家，提交纳税申报表是免费的、简单的和快速的:政府根据已经收集的银行记录和工资提供他们所欠的估计数，而不是纳税人自己费力地计算数字。广受欢迎的税务准备软件 TurboTax 的制造商 Intuit 已经筹集了数百万资金反对 T2 为减轻纳税日的痛苦所做的一切努力。

Intuit 已经花了 1150 万美元游说联邦政府，比苹果或亚马逊还多。前加州参议员 Tom Campbell 在他的加州简易文件系统提案中感受到了 Intuit 的力量，[写道](https://web.archive.org/web/20230406191504/http://articles.latimes.com/2006/may/01/opinion/oe-campbell1)他“从未见过游说力量将私人利益置于公共利益之上”

多年来，Intuit 公司漫长而昂贵的竞选活动表明，基于国税局的服务是“美国政府通过一个大型政府项目的大规模扩张”Intuit 的努力已经结交了国会中最懂技术的成员，包括共和党众议院多数党领袖埃里克·坎托( [CrunchGov Grade](https://web.archive.org/web/20230406191504/https://techcrunch.com/policy/leaderboard/) : A)和众议员佐伊·洛夫格伦( [CrunchGov Grade](https://web.archive.org/web/20230406191504/https://techcrunch.com/policy/leaderboard/) : A)。

“政府应该避免阻碍私营部门的创新，因为商业创新几乎完全发生在私营部门，”前 Lofgren 在 2005 年写给她的同事。“政府复制私营部门的创新可能看起来有短期优势，但培育私营部门创新的长期好处通常会丧失。”(提交调查报告的 Publica 表示，Lofgren 拒绝对此事发表评论，但她的办公室很快回复了我们这封信。TurboxTax 的这类创新例子包括拥有税务专家和自动减免税功能的 live Q & A。

尽管如此，一些精通技术的政策制定者相信，政府系统不会阻止创新。“这是自愿的，”奥巴马的前首席经济学家奥斯坦·古尔斯比说，他撰写了一份长篇报告，论述免申报如何能为纳税人节省数十亿美元。“如果你不信任政府，你可以不做。”[ [PDF](https://web.archive.org/web/20230406191504/http://www.brookings.edu/~/media/Research/Files/Papers/2006/7/useconomics%20goolsbee/200607goolsbee.PDF)

此外，参议员罗恩·怀登( [CrunchGov Grade](https://web.archive.org/web/20230406191504/https://techcrunch.com/policy/leaderboard/) : A)在 2011 年提出了一项两党税收改革法案，其中包括一项免退税的文件计划。怀登和洛夫格伦很少在技术问题上持反对意见，这使得反对国税局系统变得很奇怪。

有一天，美国人可能会像爱沙尼亚、[丹麦和瑞典](https://web.archive.org/web/20230406191504/http://www.oecd.org/tax/administration/36280368.pdf)一样先进。在那之前，尽量不要拖延纳税——这是一个漫长而复杂的过程。
