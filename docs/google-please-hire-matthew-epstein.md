# 谷歌，请雇用这个家伙

马修·爱泼斯坦想去 T2 的谷歌工作，很糟糕。为了在公司的产品营销团队谋得一份工作，他建立了[这个很酷的网站](https://web.archive.org/web/20230203123311/http://googlepleasehire.me/)，更有趣的是，还制作了下面的视频。

(Via [Hacker News](https://web.archive.org/web/20230203123311/http://news.ycombinator.com/item?id=2840233) ，他显然受到了那些似乎极度热衷于无情批评他人的人的批评)。

Googlepleasehire.me(懂了吧，我？马修·爱泼斯坦？哈！)已经上线几天了，在发布的时候，Epstein [写道](https://web.archive.org/web/20230203123311/http://www.matthew-epstein.com/land-a-job-at-google-journey/)这只是他在谷歌谋得一份工作的开始:

> 我有几个大的营销和公关计划，我不能等待谷歌和其他人看到它们。接下来的 21 天将会证明我是言出必行还是言出必行。
> 
> 这不是向人们证明我擅长营销，而是向自己证明我有热情和激情解决复杂问题的能力。

此外，制作相当有趣的视频。

值得一提的是，谷歌已经与爱泼斯坦取得了联系，但通知他应该在那里通过标准的招聘程序(他对此没有意见)。

不过，我的蜘蛛感觉打赌 Epstein 会很快被录用。
