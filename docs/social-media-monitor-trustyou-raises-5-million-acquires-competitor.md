# 社交媒体监测 TrustYou 融资 500 万美元，收购竞争对手

社交媒体监测和语义技术初创公司 TrustYou 已经筹集了 500 万美元的 A 轮融资。它还收购了总部位于达拉斯的竞争对手 [ReviewAnalyst](https://web.archive.org/web/20230203201451/http://reviewanalyst.com/) 。这轮融资由法国农业信贷私人股本牵头，将用于扩大其德国慕尼黑总部的全球业务。其“情感搜索平台”针对的是美国和欧洲的酒店、餐厅和旅游行业。
