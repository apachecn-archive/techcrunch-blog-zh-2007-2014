WordPress Twitter Blackbird Pie工具带回了老派的转发方式！

在发布关于 WordPress实现Twitter的黑鸟派工具的帖子时，我注意到了一些有趣的事情。这个工具带回了Twitter早已废弃的东西:老派的转发！

现在，当你在WordPress.com的博客文章中插入一个Tweet URL时，它会自动在那篇文章中生成一条漂亮的Tweet，并带有工作链接。但它也在推文下方添加了一个“转发”链接。点击这个会把你带到Twitter.com，在那里一条推文会自动为你填充:“RT @USERNAME TWEET HERE”。

Retweet是Twitter用户在早期创建的一项功能。它基本上只是一种引用推文的方式，但使用语法“RT”让其他人知道这是其他人的推文。Twitter最终意识到，有这么多的人在使用它，他们必须创造一些方法让所有用户(主要是新用户)更容易使用它。所以他们创造了新的转发，它不使用“RT”语法，而是将实际的推文放回关注转发者的用户的时间表中。

许多用户起初不喜欢这种新的转发方式，并对此发起了一场风暴。主要的问题是新的方式不允许你给你转发的推文添加评论。Twitter或多或少地说了句“太糟糕了”，然后继续前进。现在，新的方式几乎是公认的标准，但许多第三方服务仍然允许您使用旧的方式。

现在Twitter用这个工具把它带了回来(更新如下)。看起来他们不得不这样做——这几乎是唯一不需要网站安装某种插件的方法，这种插件允许Twitter.com上的新型转发。

欢迎回来，老派转发！

更新 : Twitter已经表示WordPress负责转发实现，所以他们不会正式恢复老派的转发。尽管如此，如果这项功能获得成功，它可能会以一种主要的方式回到Twitter上。
