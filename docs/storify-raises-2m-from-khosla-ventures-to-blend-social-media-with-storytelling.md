# Storify 从 Khosla Ventures 筹集了 100 万美元 2M 资金，用于将社交媒体与讲故事结合起来

现在，Twitter 和脸书等社交媒体平台比以往任何时候都更是实时故事和新闻报道不可或缺的一部分。[去年秋天在 TechCrunch Disrupt](https://web.archive.org/web/20230203003002/https://techcrunch.com/2010/09/27/storify/) 上推出， [Storify](https://web.archive.org/web/20230203003002/http://storify.com/) 首次推出其内容监管平台，该平台汇集了来自社交网站的推文、脸书状态更新、视频等，以创建一个故事或问题的实时视图。今天，这家初创公司宣布已经从科斯拉风险投资公司获得了 200 万美元的新资金。

Storify 仍处于私人测试阶段，它允许你围绕一系列推文、Flickr 照片、脸书状态更新、YouTube 视频等构建和嵌入一个故事(我们应该注意 Twitter [提供了自己的技术](https://web.archive.org/web/20230203003002/https://techcrunch.com/2010/05/04/twitter-blackbird-pie/)来嵌入推文)。在 Storify 的平台上，你可以简单地搜索并把内容拖到你的 Storify 故事中。一旦你用所有这些精选的内容创建了一个故事，你就可以通过一个类似的 Javascript 将真实的故事嵌入到你的博客或内容管理系统中。

它非常简单易读，你可以从我们在这篇文章中嵌入的关于 Storify 资助的故事样本中看到(这是一篇关于最近埃及抗议活动的 Storify 文章)。虽然处于私下测试阶段，但 Storify 已经与媒体机构签署了许多高调的合作伙伴关系，包括《华盛顿邮报》、《西雅图时报》、NPR 和半岛电视台。自去年秋天以来，合作伙伴已经创建了 10，000 个 Storify 故事，这些故事吸引了 450 万次观看。

该公司的联合创始人伯特·赫尔曼(实际上是一名前记者)说，新的资金将用于建设新的能力和出版选择。Storify 已经连接到博客平台 Posterous 和 Tumblr，因此用户可以直接在 Storify 中将故事发布到 Posterous 和 Tumblr。

赫尔曼表示，对于任何对使用 Storify 感兴趣的记者或媒体机构来说，这家初创公司的平台应该会在未来几个月向公众开放。
