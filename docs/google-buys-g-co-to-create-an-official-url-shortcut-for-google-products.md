# 谷歌收购 G.Co，为谷歌产品创建官方网址快捷方式

谷歌今天加入互联网巨头 Twitter([T.Co](https://web.archive.org/web/20230316161016/http://www.t.co/))、Overstock([O.co](https://web.archive.org/web/20230316161016/http://www.o.co/))和亚马逊([A.Co](https://web.archive.org/web/20230316161016/http://www.a.co/)、[Z.Co](https://web.archive.org/web/20230316161016/http://www.z.co/)和[K.Co](https://web.archive.org/web/20230316161016/http://www.k.co/))的行列，接管[G.Co](https://web.archive.org/web/20230316161016/http://www.g.co/)域名，购买顶级域名，为 GMail、文档和照片等谷歌产品建立官方 URL 快捷方式。而来自。Co registry 不愿对交易的具体定价发表评论。联合创始人朱安·迭戈·卡勒最近告诉路透社记者，单字母域名的总成本超过 150 万美元。

谷歌表示，它将使用该域名，以便使用格式[g . co/【XYZ product/service】](https://web.archive.org/web/20230316161016/http://g.co/%5BXYZproduct/service%5D)为其所有产品和服务创建快捷方式，该域名将于今天下午晚些时候上线。谷歌消费者营销副总裁加里·布里格斯(Gary Briggs)在一次新闻发布会上说:“你可以访问 G.CO 的快捷方式，相信你会一直停留在谷歌产品或服务的页面上。”。

跑出哥伦比亚(在哥伦比亚政府的支持下)。Co domain 的核心目标是大大小小的科技公司。今天公布的还有 [500 家创业公司](https://web.archive.org/web/20230316161016/http://www.500startups.com/)。联合品牌重塑；今年秋天，孵化器将从 500Startups.com 搬到 500.co。虽然不是主流，500 家创业公司的举动可能与谷歌新闻一样具有里程碑意义，因为它给了域名街信誉，并巩固了“公司”的概念。Co 的意思是公司。
