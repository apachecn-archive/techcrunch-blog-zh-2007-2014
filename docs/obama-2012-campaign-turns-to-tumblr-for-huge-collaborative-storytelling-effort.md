# 奥巴马 2012 竞选团队转向 Tumblr 寻求“巨大的合作叙事努力”

奥巴马总统又增加了一项新的网络服务: [Tumblr](https://web.archive.org/web/20230204210050/http://www.tumblr.com/) ，这个热门的博客服务刚刚又为[筹集了 8500 万美元](https://web.archive.org/web/20230204210050/https://techcrunch.com/2011/09/26/tumblr-raises-85-million-round-from-richard-branson-vcs/)的资金。你可以在这里找到新博客。

该网站由 2012 年奥巴马/拜登竞选团队建立，该团队还运营着他的[脸书页面](https://web.archive.org/web/20230204210050/https://www.facebook.com/barackobama)和[推特](https://web.archive.org/web/20230204210050/https://twitter.com/#!/barackobama)账户(白宫最近也在[four square](https://web.archive.org/web/20230204210050/https://techcrunch.com/2011/08/15/obama-checks-in-you-can-now-follow-our-president-on-foursquare/)上推出了一个账户)。

Tumblr 通常以年轻受众(尤其是青少年)而闻名，它的重新博客功能将有助于竞选帖子像野火一样在整个服务中传播。Tumblr 的人们无疑是微笑的——总统的出现可以作为一种荣誉徽章，通常也会导致大量免费的主流媒体报道。

在他博客的第一篇文章中，奥巴马的团队写道，他们希望 Tumblr“成为一个巨大的合作讲故事的平台——一个让全国人民分享我们各自领域正在发生的事情以及我们如何参与这场运动以不断改善它的地方。”

为此，该网站将通过 Tumblr 提交功能接受提交内容。幸运的是，这项活动并不幼稚——他们已经先发制人地要求提交者在发送任何令人作呕的东西之前，先想到他们的母亲。

> 你们中间会有巨魔:这我们都知道。我们只要求你记住我们是人——相当好的人——你的母亲希望你有礼貌。
