# 证实:爱思唯尔以 6900 万至 1 亿美元收购了门德利，以扩大其开放的社会教育数据工作

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/04/08/confirmed-elsevier-has-bought-mendeley-for-69m-100m-to-expand-open-social-education-data-efforts/>

教育出版商爱思唯尔(Elsevier)正在深入开放和社交教育数据的世界:它已经收购了总部位于伦敦/纽约的 T2 门德利(Mendeley)公司，该公司为学者和组织提供了一个通过社交网络分享研究和与其他人合作的平台。该交易的条款尚未公开披露，但我们知道其金额在 6900 万美元至 1 亿美元之间。1 月 5 日，当这笔交易还在谈判中的时候，我们第一次透露了这个消息。

此次收购将加剧爱思唯尔和其他进入 ed-tech 的大型出版商之间的竞争，其中包括拥有门德利的竞争对手 [EndNote](https://web.archive.org/web/20230405112258/http://www.endnote.com/) 的[汤姆森路透](https://web.archive.org/web/20230405112258/http://www.thomsonreuters.com/)。

Mendeley 既是 Elsevier 的技术/平台收购，也是收购。首席执行官 Victor Henning 是 Mendeley 的三位博士联合创始人之一，他在一次采访中告诉我们，Mendeley 的 50 名员工都将加入爱思唯尔，Henning 将成为公司的战略副总裁，这表明爱思唯尔可能正在为该领域的更多活动和可能的收购做准备。(下面是门德利团队早先拍摄照片，配有俗气的全息图。)

爱思唯尔学术和政府研究市场总经理奥利维尔·德鲁贝说，该公司不打算将门德利与现有的爱思唯尔产品合并，这些产品曾经与之竞争，如 [Scopus](https://web.archive.org/web/20230405112258/http://www.scopus.com/home.url) ，“但我们会更好地整合它。”

爱思唯尔指出，在某些方面，它自 2009 年以来一直与门德利合作。该公司在一份关于收购的声明中表示:“爱思唯尔已将用户推荐给门德利，邀请门德利使用其开放 API 在 ScienceDirect 上开发应用程序，并赞助门德利关于开放科学的科学在线伦敦会议。”。

门德利成立于 2008 年，已经从投资者那里筹集了不到 1200 万美元的资金，投资者包括 [Access Industries](https://web.archive.org/web/20230405112258/http://www.crunchbase.com/financial-organization/access-industries) 、 [Passion Capital](https://web.archive.org/web/20230405112258/http://www.crunchbase.com/financial-organization/passion-capital) 和 [Tom Glocer](https://web.archive.org/web/20230405112258/http://www.crunchbase.com/person/tom-glocer) 。门德利将继续运行其主要平台和机构版，旨在帮助大学和其他大型组织实时跟踪研究和正在阅读的内容。Mendeley 目前在其平台上拥有 230 万用户(高于 1 月份的 210 万)，以及遍布北美、欧洲和亚洲的 24 所“高知名度”机构。该计划旨在扩大免费服务的范围，比如将个人存储空间增加一倍，达到 2gb。

它还将保持其 API 免费开放使用:该 API 目前被大约 300 个应用程序使用，比 1 月份的 260 个有所增加。

Henning 说 Mendeley 将继续从不同的地方收集数据——而不仅仅是关注 Elsevier 发布或拥有的内容。“如果人们想获得神经生物学的最新研究，限制这一点是没有意义的，”亨宁说。“爱思唯尔将通过丰富我们的内容来帮助我们，但当涉及到其他出版商时，它也会增加进入他们的中转路线。”

这次收购也意味着门德利可以稍微改变一下策略。过去，它向用户收取额外费用，如更多的存储空间、更大团队的使用包等，这些服务帮助该公司在过去一年将收入增加了两倍。Henning 表示，如果它保持独立，将会有更大的压力来推动收入增长。“但我认为，有了爱思唯尔，我们可以有更长远的眼光，”他说。“我们的用户想要什么，而不是必须过早变现。重点将是扩大我们的用户群，而不是试图马上赚钱。”

作为其中的一部分，门德利现在还希望将资源集中在一些项目上，这些项目是它一段时间以来一直想解决的。该公司已经在 iOS 设备上有了强大的用途；现在，它打算在今年晚些时候推出一款安卓应用。

当我们第一次透露两家公司正在进行收购谈判的消息时，我注意到在这篇报道的评论中和其他地方有很多负面的言论——事实上，你可能会认为增加存储空间等一些有吸引力的交易有助于留住持怀疑态度的用户——许多人担心产品会因此变得过于专有或商业化。亨宁坚持认为这不会在这里发生。

“如果你观察任何收购，无论是 Dropbox 还是我们收购它的邮箱，人们总是担心服务会发生什么，”他说。“对我们来说，这将加速我们的视野。”

**更新**:自昨晚发布以来，Mendeley's 发表了一篇关于这笔交易的 Q&A[在此](https://web.archive.org/web/20230405112258/http://blog.mendeley.com/press-release/qa-team-mendeley-joins-elsevier/)，我已经联系了其他几个人，传递其他价格信息，其中一个人声称接近这笔交易，建议将€的售价降低到 5000 万英镑(6500 万美元)。
