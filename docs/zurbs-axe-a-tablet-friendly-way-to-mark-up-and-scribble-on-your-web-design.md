# Zurb 的 Axe:一种对平板电脑友好的方式来标记你的网页设计

交互设计公司 Zurb 刚刚发布了一款新工具，可以让你很容易地用 iPad 来标记和添加关于网站设计的评论。它叫[斧](https://web.archive.org/web/20230203065522/http://www.axeapp.com/)(域名是【axeapp.com】的)，不，它和男体喷雾没有任何关系。注意，你不能在电脑上使用它，所以你必须拿出一个 iPad 来查看它。

这款应用非常简单:一旦你从平板电脑上访问它，就会被提示输入你想要标记的网站的网址(比如说 beta.techcrunch.com)。然后，该应用程序将抓取该网站的图像，你可以立即开始在上面涂鸦(去掉你不喜欢的东西)和圈出(突出你做的事情)。你也可以添加文本框来解释你的涂鸦到底是什么意思。如果你对涂鸦有点过分热情，你可以摇动 iPad 来擦除它。

Zurb 营销负责人 Dmitry Dragilev 表示，该服务针对 iPad 进行了优化，但也应该适用于 Android 平板电脑(尽管速度较慢)。而且是免费的。不过，有一点我希望看到它的加入:在这一点上，Axe 主要是针对那些设计(或拥有)网站和交流思想的人；我希望看到一个选项，让网站嵌入一个面向消费者的按钮，让用户提交建议。

事实上，这只是 Zurb 多年来发布的一套免费工具中的最新一款——我们之前写过类似于 [Reel](https://web.archive.org/web/20230203065522/https://techcrunch.com/2011/08/30/zurbs-reel-lets-you-get-instant-feedback-on-powerpoint-presentations/) (让你获得演示文稿的即时反馈)和 [Clue](https://web.archive.org/web/20230203065522/https://techcrunch.com/2010/09/24/clue-tracks-what-people-remember-about-your-website/) 的应用，让你对用户在浏览你的网站后记得的内容进行一种记忆测试。

他们的另一个免费工具是 [Bounce](https://web.archive.org/web/20230203065522/http://www.bounceapp.com/) ，它类似于 Axe，但针对你的 PC 浏览器而不是平板电脑进行了优化。

Dragilev 解释说，Zurb 将这些工具视为提高咨询服务意识的一种方式。他们也有一些高级工具，比如 Zurb [Verify](https://web.archive.org/web/20230203065522/http://verifyapp.com/) 。
