# 这就是胜利的感觉，Shaker 版

> *“‘什么？.'“我们做它是为了捣乱，”萨拉平静地说。我们大声尖叫，把整栋楼都吵醒了。"*

尖叫结束后，总部位于以色列的 [Shaker](https://web.archive.org/web/20230204134315/http://www.atshaker.com/) 放弃了在旧金山为 TechCrunch Disrupt SF 2011 彩排的燃烧人计划，制定了一个计划，为他们赢得一些时间(问他们是否可以让所有 13 位创始人都上台)，并让这个计划适得其反(我们说“是”)).

在经历了 Erick 告诉他们他们的演讲是“我听过的最无聊的演讲”之一，并消化了他慷慨激昂的请求后，“带我去 Shaker，让我感觉像在酒吧一样！团队没有列出 25 个新增功能，而是“给我一个好的体验”，在上台之前，他们花了一个不眠之夜练习他们的演示。

“故事的其余部分是[很好地记录了](https://web.archive.org/web/20230204134315/https://techcrunch.com/2011/09/14/and-the-winner-of-techcrunch-disrupt-is-shaker/)，”Yonatan 说，“不起作用的音乐，就在我们上台时死机的电脑，以色列的团队在酒吧里参加 Shaker 舞蹈，向观众释放红心，当然还有 Mike 的-充满机智的-宣布。”

难怪 Shaker 团队没有作为[获胜者](https://web.archive.org/web/20230204134315/https://techcrunch.com/2010/09/30/this-is-what-winning-feels-like/)参加庆功宴，而是在他们的酒店房间里崩溃了(上图)。

Yonatan 说，“照片中的人显然是我们的财务主管安迪……他是我们公司里扣留每一美元的人——直到他真的带着它上床睡觉时，他才放下那张纸板支票。”
