# 威瑞森无线(卡西欧)G 'zOne 型审查

![imgp0638.JPG](img/15cbe0269e68354eb62d7a14a95ca6b0.png)
它会融合吗？
我们都曾一次、两次或每隔五分钟摔过手机，对吗？我知道我有很多次，每一滴都会让我退缩。电池或某些部件似乎每次都会自动脱落，但之后就不一样了。我已经把我的手机送去更换，没有水印贴纸显示我把手机掉在厕所里，但这种好运不会永远持续下去。你有没有不小心把手机扔进明火里的经历？一旦余烬冷却下来，它看起来很酷，但你是 SOL 否则。还有那些随机宿醉的早晨，我发现我的手机在冰箱里。奇怪。我知道。我可能是一个笨手笨脚或倒霉的人，我还没有想出来。
 我们这些似乎被诅咒了‘黄油手指’的人还有希望吗？不完全是，但是如果你是威瑞森的订户，那么一切都很好。考虑到我和这个国家最大的供应商的又爱又恨的关系，这是一个奇怪的说法，不是吗？无论如何，万一你的手机不能处理游泳池里的浸泡，10 层楼的坠落，红袜队输了(洋基队输了，所以没关系)的愤怒，或者明火，那就去看看卡西欧 G'zOne 吧。

没有必要深入了解运营商方面的事情，因为这是你已经习惯的相同的威瑞森~~废话~~用户界面和应用程序，所以我们将深入研究 G'zOne 必须提供什么，这是相当多的。我与 G 的第一次接触粉碎了任何认为这只是一个聪明的营销策略，它不够坚固，无法承受打击的想法。代表们在房间里来回移动它，把它砸到地上，把它沉入水中，然而，它仍然工作，没有任何并发症。他们甚至向我展示了一些滑雪板玩家扔进火中的一个，它看起来像假的便便，但仍然可以使用。有些人可能仍然持怀疑态度，我也是，所以我们对它进行了各种 CG 测试。通过了吗？

当然通过了。第一系列测试包括将它从几英尺高的地方扔到地板上，沿着楼梯发射，然后让它沉入池底。无论我把手机摔了多少次，它都不会停止跳动。我甚至跳上去，用脚使劲踢墙。一点也没影响。通话清晰度也从未降低，电池盖也从未脱落。卡西欧足够聪明，实现了一个闩锁，以确保后盖和电池永远不会脱落。引用波拉特的话，"那是个骗局。"

但许多手机可以承受一两滴，并继续突突前进，没有问题，对不对？它们能经受住马桶里的一滴水或游泳池里的一泡吗？不尽然，这是 G 优于市场上任何其他移动设备的一个方面。我把卡西欧放在我女朋友的泳池底部大约半个小时，不出所料，它仍然工作。然后，我打开外部手电筒，把它浸入一杯水中，又过了半个小时。你猜对了，仍然有效。嗯。我还能做什么？我知道！我把它扔进冰柜，半小时后看它走得多慢。大多数手机在低温下会变得非常迟钝，有时甚至无法使用。显然，一旦它们被加热，它们就会复活，但是 G 的命运却大不相同。菜单滚动有一分钟的延迟时间，考虑到这种情况，这是可以理解的，但很高兴知道你可以把它带到山上，而不用担心它不能正常工作。最多花 45 秒到 1 分钟，一切都畅通无阻了。

G'zOne Type-S 坚固耐用，但它并不吝啬其功能。翻盖不是市场上最薄的(这一荣誉属于鳍)，但我喜欢我的手机稍微长一点，2.04 x 3.84 x 1.13 英寸，重量略超过 4.5 盎司。通常的嫌疑人包括蓝牙，VGA 3x 倍变焦摄像头/带自拍功能的视频(128MB 闪存)，扬声器，电话簿(500 个联系人)，日历，计算器，秒表，定时器和语音备忘录/命令/拨号，所有这些都隐藏在聚氨酯涂层的塑料框架中。如果你是 VZ 导航用户，它甚至有全球定位系统。通话时间约为 180 分钟，支持短信/短信/彩信和手机即时消息。你真的不需要更多了。你现在可以在当地的威瑞森商店购买，但如果你在网上购买，你可以额外获得 50 美元的折扣。只是不要把它掉在厕所里。
