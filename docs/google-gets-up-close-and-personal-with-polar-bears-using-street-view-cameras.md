# 谷歌用街景相机近距离接触北极熊

过去，谷歌曾将街景拍摄到一艘阿姆斯特丹驳船(恰好是初创公司 [MobyPicture](https://web.archive.org/web/20230130230745/http://mobypicture.com/) 的总部)和[神秘博士](https://web.archive.org/web/20230130230745/https://www.google.com/maps/@51.492137,-0.192878,3a,75y,284.12h,82t/data=!3m5!1e1!3m3!1sc9UMhWP_MWm9U0L48xEjYw!2e0!3e5?hl=en-US)的内部。

现在是*带着[北极熊在自然栖息地的惊人街景照片](https://web.archive.org/web/20230130230745/http://www.google.com/maps/about/behind-the-scenes/streetview/treks/churchill-canada/)再次去那里*，正好赶上今天的国际北极熊日。拍摄于马尼托巴省丘吉尔市——显然是“世界北极熊之都”——并与北极熊国际组织合作，谷歌在一辆“苔原小车”上安装了街景 Trekker 相机，并将谷歌地图带到加拿大的苔原上，以收集北极熊在苔原——它们的自然栖息地——的 360 度街景图像。

但也有严肃的一面。将街景引入加拿大苔原，建立了他们所说的与特定地理空间数据相关的“图像基线记录”。这种数据可以在以后保护这种神奇的生物。

如果我们自己这么说的话，这是很酷的东西。
