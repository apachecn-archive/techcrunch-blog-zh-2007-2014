# Teneros，一家在微软交易所每一次崩溃时都能赚钱的初创公司，在系列 D 中筹集了 4000 万美元

[![teneros-logo.png](img/6a8a918f0107dc4ecc9d9e122609aa8e.png)](https://web.archive.org/web/20221207192816/https://beta.techcrunch.com/wp-content/uploads/2008/02/teneros-logo.png "teneros-logo.png") 把这个放在你的讽刺文件夹里。一家名为 [Teneros](https://web.archive.org/web/20221207192816/http://www.teneros.com/) 的硅谷初创公司今天完成了由 Advanced Equities 牵头的 4000 万美元 D 系列融资。这使得该公司自成立以来筹集的资金总额达到 8450 万美元(其他投资者包括高盛、新企业协会和塞文·罗森基金)。

特纳罗斯是做什么的？通过提供基本上备份服务器的远程管理的“连续性设备”，它使崩溃的 Microsoft Exchange 电子邮件服务器无法阻止公司访问电子邮件。(也许它应该考虑将其名称更改为 Tenemos，如“我们收到了您的邮件。”)

大约三分之二的公司使用微软交换中心发送电子邮件，即使一年中有几个小时的停机时间也会让公司损失大量的资金。我觉得讽刺的是，这笔资金，实际上是这家公司，完全建立在微软交换的缺陷上。其投资者愿意以 8450 万美元打赌，微软将无法修复交易所，以消除对 Teneros 产品的需求。无论微软本身在交易所投资多少，总会有崩溃的时候。顺便说一下，这就是拯救雅虎的同一家公司。

(另见 [Azaleos](https://web.archive.org/web/20221207192816/http://www.azaleos.com/) ，该公司去年 11 月[募集了 1000 万美元](https://web.archive.org/web/20221207192816/http://blog.seattlepi.nwsource.com/venture/archives/125102.asp))。
