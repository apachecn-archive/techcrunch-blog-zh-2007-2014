# TC50:情感界面监控并解读网络用户的面部表情 TechCrunch

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/09/15/tc50-affective-interfaces-monitors-and-deciphers-facial-expressions-of-web-users/>

![](img/61612a2370d9e012a2829ba9afa0ee7e.png)

情感检测正迅速成为解读推文、新闻故事和网络上其他内容的一种流行方法。今天在商业应用会议上发布的 [TechCrunch50](https://web.archive.org/web/20230203155730/http://www.techcrunch50.com/2009/) 初创公司 [Affective Interfaces，](https://web.archive.org/web/20230203155730/http://affectiveinterfaces.com/Welcome.html)已经建立了情感传感技术，通过网络摄像头检测网络用户的面部表情。

情感接口公司的 SaaS 使用网络摄像头来监控面部表情，如微笑、皱眉和其他情绪。该软件分析这些表达以及它们如何与网站的不同部分相关联。例如，情感界面会比较用户在 CNN.com 和 Gawker.com 的情绪。或者，该软件可以很好地用于确定 Zappos.com 等电子商务网站上的情绪，以了解什么样的情绪会影响参与度和购买决策。情感界面通过视频广告、媒体等收集客户内容。，然后为客户收集和处理数据，并创建一个解读用户情绪的报告。

这项技术似乎类似于让一个在线焦点小组测试一个产品，除了该软件实际上会分解情感数据。Affective Interfaces 为每次会议提供完整的报告和图表，深入分析产品参与度。这家初创公司表示，它正在发布 API，供人们将其数据导入该软件，该软件可用于在线媒体指标、娱乐、网页开发、游戏网站和电子商务。
