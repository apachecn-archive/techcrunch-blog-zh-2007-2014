# 争取 40 亿移动用户，Twitter 与印度最大运营商达成短信协议

在美国，我们很多人通过 Twitter 网站或使用其 API 的第三方客户端使用 Twitter。但是，Twitter 的一个原始的关键，也是最大限度地影响受众的一个非常重要的关键，是短信。有一段时间，看起来这个功能要死了，[在荒谬的运营商短信费用下窒息了](https://web.archive.org/web/20230404181335/https://techcrunch.com/2008/08/13/twitter-ends-sms-support-in-uk-says-costs-up-to-1000useryear/)，但是 Twitter 已经在[努力让它回到](https://web.archive.org/web/20230404181335/https://techcrunch.com/2009/09/22/twitter-begins-emphasizing-sms-again/)。今天，他们[宣布了一项新的交易](https://web.archive.org/web/20230404181335/http://blog.twitter.com/2009/10/hello-bharti-airtel.html)来进一步扩展服务:与印度最大的运营商 Bharti Airtel 合作。

该协议允许 Bharti Airtel 用户通过短信免费接收推文，并以标准短信费率发送推文。这似乎不是什么大事，但 Twitter 去年不得不在许多国家关闭短信服务的原因是，运营商坚持 Twitter 向他们支付高额费用，以弥补用户通过短信免费接收的所有推文。Twitter 显然已经开始达成服务和运营商现在都能同意的交易。

手机短信在印度非常重要，这不仅是因为该国有 10 亿居民(其中超过 1 亿人使用 Bharti Airtel)，还因为印度许多人无法接入高速互联网(他们中的大多数人也没有提供廉价的免费数据套餐的 iPhones)。这就是[脸书决定首先在印度测试其](https://web.archive.org/web/20230404181335/https://techcrunch.com/2009/08/12/what-facebook-lite-actually-is-hint-its-not-twitter/)[脸书 Lite 项目](https://web.archive.org/web/20230404181335/https://techcrunch.com/2009/08/11/facebook-begins-testing-facebook-lite-a-faster-simpler-version-of-the-service/)的原因。但是短信是一座桥梁，在像印度这样的地方，更多的人可以通过它方便地访问 Twitter。

要开始在 Bharti Airtel 手机上使用 Twitter，只需发送短信 start 到 53000。请记住，发送这些短信会让你付出代价，即使它们让运营商付出了成本[基本上什么都没有](https://web.archive.org/web/20230404181335/http://digital.venturebeat.com/2008/12/29/text-messaging-further-exposed-as-a-colossal-rip-off/)。

Twitter 不会回避崇高的(至少可以这么说)目标:

> 地球上有超过 10 亿人可以上网，但有超过 40 亿人有手机，Twitter 可以在所有手机上工作，因为即使是最简单的手机也有短信功能。

Twitter 上有 40 亿人？也许麦莉·克里斯甚至会回来。
