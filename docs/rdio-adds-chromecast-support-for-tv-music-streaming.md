# Rdio 增加了对电视音乐流的 Chromecast 支持

流媒体音乐服务 Rdio 增加了对谷歌 Chromecast 媒体附件的支持，该公司今天宣布。这意味着你可以将你的 Rdio 音乐从你的 iPhone，iPad，Android 手机，Chrome for Mac 和 Chrome for Windows 无线发送到你的 Chromecast 连接的电视或 HDMI 显示器上。使用 Rdio 现有的远程功能，您还可以在登录 Rdio 帐户的任何地方控制 Chromecast 上的播放。

与其他支持 Chromecast 的应用程序一样，Chromecast 发现的图标现在出现在 iOS、Mac 和 Chrome 的 Rdio 应用程序中。如果你已经设置了 Chromecast，当你点击图标时，它应该会出现，你可以立即开始播放。Rdio 此前宣布将为其服务提供 Chromecast 支持，但竞争对手 Spotify 尚未添加对谷歌 streamer 的支持，尽管它确实提供了自己的 Spotify Connect Wi-Fi 音频流协议。

[iOS 应用](https://web.archive.org/web/20230129094345/https://itunes.apple.com/en/app/rdio/id335060889?mt=8)和 [Android 应用](https://web.archive.org/web/20230129094345/https://play.google.com/store/apps/details?id=com.rdio.android.ui)的更新应该很快就能实现，而且该功能也在网上直播。
