# 微软开放 2013 创新杯注册，奖金翻倍至 30 万美元

过去 10 年来，微软一直组织其[想象杯](https://web.archive.org/web/20230325002940/http://www.imaginecup.com/)学生技术竞赛，今天，[公司为该赛事的 2013 年版本开放注册](https://web.archive.org/web/20230325002940/http://www.imaginecup.com/news)。16 岁及以上的学生现在可以注册参加他们的全国性赛事，这些地方赛事的获胜者将飞往俄罗斯圣彼得堡，全球总决赛将于 7 月 8 日至 11 日在那里举行。对于这一届的想象杯，微软将奖金提高了一倍，达到 30 万美元。

微软还围绕三个新的核心领域重组了竞赛:世界公民、游戏和创新。此前，旗舰活动是软件设计比赛，今年一群乌克兰学生[在开发出能够](https://web.archive.org/web/20230325002940/https://techcrunch.com/2012/07/10/imagine-cup-winners-quadsquad-enabletalk/)[将手语翻译成语音](https://web.archive.org/web/20230325002940/https://techcrunch.com/2012/07/09/enable-talk-imagine-cup/)的手套后赢得了比赛。

微软表示，自 2003 年首届创新杯以来，全球已有超过 165 万名学生参加了创新杯，许多进入决赛的团队(以及许多未能进入决赛的团队)继续创业。例如，2011 年决赛选手[来自约旦](https://web.archive.org/web/20230325002940/http://oasys-solutions.com/)的 OaSys 团队目前正致力于将其允许四肢瘫痪者控制电脑的系统推向市场。为了帮助决赛选手将他们的想法商业化，微软还允许他们申请为期三年、价值 300 万美元的创新杯资助。

以下是微软对新核心竞争的描述:

*   **世界公民:**表彰在微软平台上开发的最有潜力对人类产生积极影响的软件应用。例如，一个项目可能解决教育、社会或医疗保健相关的问题。
*   **游戏:**奖励面向青少年的最具吸引力和娱乐性的游戏，这些游戏基于微软平台(Windows 8、Windows Phone、Kinect for Windows 软件开发工具包和 Xbox Indie 游戏)。
*   **创新:**表彰那些给消费者带来灵感和创新的应用，无论是社交网络、在线购物还是搜索的新玩法，都是用微软的工具和技术构建的。

每场比赛的获胜者将获得 5 万美元。

除了核心竞赛之外，学生还可以参加一些侧重于特定技术和平台的在线挑战，包括 Windows 8、Windows Azure 和 Windows Phone。
