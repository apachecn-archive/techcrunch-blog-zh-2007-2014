# 来源:微软和雅虎会谈

雅虎和微软的多个消息来源告诉我们，两家公司之间的官方谈判已经恢复。但是我们听到了不同于 CNET T2 的消息——谈判是关于一次全面收购，而不是一次甜言蜜语的纯搜索交易。

我们得到的信息很少，但是有一个消息来源说微软正在谈一个比他们在五月谈判破裂时提供的 33 美元更低的价格。鉴于雅虎最近的股价(今天低于 21 美元)，以及除了董事会和高管以外的几乎所有人都在公开呼吁达成交易的事实，我并不感到惊讶。

微软官方的评论是“无可奉告”，这实际上包含了比看起来更多的信息。一个多月以来，微软官方一直表示他们对雅虎不再感兴趣。他们今天没这么说。

**更新(太平洋标准时间上午 11:44):**其他消息来源称，雅虎董事会在签署[谷歌搜索交易](https://web.archive.org/web/20230213021326/https://techcrunch.com/2008/06/12/yahoo-announces-non-exclusive-search-agreement-with-google/)之前，提出以“每股 30 美元的低价，低于微软的原始报价”出售给微软。

**更新(太平洋标准时间上午 11:47):**来自美国消费者新闻与商业频道:

> 正如 13:29 所提到的，美国消费者新闻与商业频道评论员说，一位非常接近微软(MSFT)的消息人士，他“知道”两者之间的谈判，同情地说，整个 YHOO 没有交易，到今天为止什么也没有改变。注意到两人关于 YHOO 搜索业务的讨论一直都是公开的。重申整个公司不会有任何交易。

我们所听到的是，双方目前正在就完全收购进行讨论，不一定有交易，甚至微软也没有提出任何形式的确定报价。微软的另一位消息人士向我们重申，他们是价格合适的买家，但没有透露价格。
