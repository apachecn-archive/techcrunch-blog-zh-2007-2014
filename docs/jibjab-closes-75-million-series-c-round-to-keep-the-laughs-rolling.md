# JibJab 完成了 750 万美元的 C 轮融资，让笑声不断

[![](img/c3a2861280ea08bf253ac987cf250e0d.png)](https://web.archive.org/web/20221201164932/http://www.jibjab.com/)

因恶搞视频和不敬言论而受欢迎的网站 JibJab 已经完成了 750 万美元的 C 轮融资，新投资者 Overbrook Entertainment(威尔·史密斯共同创立的制作公司)和索尼电影娱乐公司以及现有投资者 Polaris Venture Partners 参与了融资。该公司之前已经筹集了大约 940 万美元的资金。该网站每年都会产生巨大的流量，根据 Quantcast 的统计，它是美国 100 个最受欢迎的网站之一(目前排名第 73)。

节日期间，JibJab 记录了 6000 万次访问，用户在该网站上创建了 3500 万张个性化电子贺卡——这些数字可能受到非常受欢迎的 ElfYourself 视频的推动，该视频让你把脸贴在跳舞的精灵上。

该网站因其广泛的政治模仿而闻名(如果你在 2004 年选举期间没有看到[这个土地](https://web.archive.org/web/20221201164932/http://www.jibjab.com/originals/this_land)视频，你可能正躲在一个山洞里)，但近年来也扩展到提供[模仿小品](https://web.archive.org/web/20221201164932/http://www.beta.techcrunch.com/2006/10/20/jibjabs-great-sketch-experiment/)和个性化的[电子问候](https://web.archive.org/web/20221201164932/http://www.beta.techcrunch.com/2007/10/30/happy-halloween-from-the-sick-people-at-jibjab/)。该公司通过订阅服务提供优质电子贺卡和内容以及各种免费卡片来创收，还销售数字下载和个性化礼品，如杯子和鼠标垫。
