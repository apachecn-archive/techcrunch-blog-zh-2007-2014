# 谷歌收购电子书技术 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/01/12/google-acquires-ebook-technologies/>

根据目前出现在公司主页上的通知，谷歌刚刚收购了[电子书技术公司](https://web.archive.org/web/20230203005435/http://ebooktechnologies.com/)。

谷歌证实了这次收购，一位发言人给了我们这样的声明:

> “我们很高兴欢迎 eBook Technologies 的团队加入谷歌。我们希望在平板电脑、电子阅读器和其他便携式设备上提供更丰富的阅读体验。"

那么电子书技术到底是什么？主页上的细节很少——该网站只有一个横幅，上面写着，“电子书技术公司提供一系列智能阅读设备和许可技术，实现自动出版和对内容分发的控制。”网站上的大多数链接，包括产品页面，都会重定向到这个主页。然而，稍微挖掘一下谷歌的缓存就会发现更多的细节。

请注意，目前还不清楚谷歌打算使用这些技术中的哪一种，但它显然可以利用这些技术来支持其谷歌图书产品，该产品最近推出了电子书商店。

一个[设备页面](https://web.archive.org/web/20230203005435/http://webcache.googleusercontent.com/search?q=cache:http://www.ebooktechnologies.com/devices.htm)提到了该公司开发的硬件，但它似乎已经过时了，一种产品在 2007 年首次亮相。看起来更有可能的是，谷歌是在追求该公司的分销技术，这在一个[缓存的](https://web.archive.org/web/20230203005435/http://webcache.googleusercontent.com/search?q=cache:http://ebooktechnologies.com/products.htm)产品页面上有所概述:

> 电子书技术公司(“ETI”)平台由四部分组成:电子阅读设备(电子书设备)及其相关的用户界面固件技术、“在线书架”、基于互联网的内容销售和交付系统(电子书商店)以及电子书内容转换/出版工具。
> 
> …(关于电子书阅读器的模糊语言)
> 
> 设备设计包括用于管理熟悉的阅读功能的直观控件，包括:翻页、搜索文本、标记文本和查找定义。这些“纸质书”的功能通过技术得到了增强:即搜索是即时的，文本标记可以直接“写”在触摸屏上，标记的文本可以擦除，定义可以根据虚拟词典自动索引。ETI 阅读设备系列目前包括两种型号。ETI-1 有一个 8.5 英寸对角线，全 VGA 彩色屏幕设备，紧凑型闪存，56K 调制解调器和一个以太网端口。ETI-2 有一个半 VGA 5.5 英寸灰度屏幕，带有 8MB 内存和 SmartMedia 卡扩展，一个 33.6K 调制解调器和一个 USB 端口。
> 
> ETI 电子书设备技术中使用的核心底层固件可以获得许可。它由以下组件组成:嵌入式操作系统、文件系统、网络基础设施、图形子系统以及最重要的直观 ETI 阅读体验层。大量使用一本书的隐喻避免了整个 UI 中以计算机为中心的交互，加强了阅读的沉浸式质量，并在 ETI 阅读设备中毫不妥协地复制它。
> 
> 电子书平台的一个关键特征，也是狂热读者和专业客户的一个突破，是随时随地购买和/或访问阅读材料的能力。通过电子书设备上的内置调制解调器、USB 或以太网连接，可以即时访问内容。一旦通过互联网连接到 ETI 电子书网络，客户就可以从电子书商店浏览、选择、访问和有选择地购买电子书内容。在消费者部署中，客户还可以从他们自己的个人“在线书架”中检索任何最近或以前购买的内容。从“在线书架”上检索到的内容被下载到 ETI 设备上，然后可以离线阅读。内容以安全格式传送，只能由指定的 ETI 电子书设备解密。任何专有的用户信息，如信用卡和联系信息，都由电子书网络安全地维护。
> 
> ETI 电子书快速管理器为企业客户提供电子书网络的核心元素，使企业能够集中管理对个人或电子书用户群的企业内容的交付、访问和更新。个人电子书设备可以集中管理，所有内容的传输和更新都很容易审计。
> 
> ETI 电子书商店
> 
> eBookstore 是一个复杂的电子商务系统，用于向电子书客户营销、销售和分发数字内容。电子书商店与 20 多家主要出版商建立了关系，被认为是国内畅销书、流行期刊和主要报纸的主要提供商之一。此外，电子书商店允许顾客直接通过电子书设备、连接互联网的个人电脑上的网络浏览器、离线目录或者通过替代的网络电子零售商来浏览、选择和购买内容。
> 
> 关于 ETI 网络服务和 ETI 电子书商店的更多详细信息，请点击此处。
> 
> 内容转换和发布工具
> 
> ETI 电子书格式是基于一种工业标准的基于 XML 的标记格式(开放电子书标准),这是 ETI 创始人帮助开创的。这种格式是大多数电子书阅读系统(例如，微软、Palm 和 MobiPocket)使用的源格式。
> 
> ETI 提供了一个内容创建和预览工具，将现有的内容和文件转换成 ETI 电子书格式。该软件在 Windows 和 Macintosh 桌面系统上运行，能够将文本、通用开放式电子书文本和 Microsoft Word 文件处理成特定于设备的 ETI 电子书标题。自动脚本允许每天，每周或每月下载到 ETI 电子书设备，只要数据从内容供应商可用。
> 
> ETI 发布工具的服务器端组件，提供分页、压缩和加密功能。
