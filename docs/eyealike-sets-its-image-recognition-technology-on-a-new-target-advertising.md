# Eyealike 将其图像识别技术瞄准了一个新目标:广告

[![](img/38ff310fac01d639b75693a38a3e7956.png)](https://web.archive.org/web/20221007041113/http://www.eyealike.com/)

让你使用照片识别帮助你找到理想伴侣的初创公司 [Eyealike](https://web.archive.org/web/20221007041113/http://www.eyealike.com/) ，正在扩张，将其图像处理技术应用到一个新的市场:广告。该公司表示，新系统将允许企业在网站上出现的照片旁边放置高度针对性的广告(长期以来很难赚钱)。

目前，图像识别仅限于识别照片中人的身体特征，分类包括年龄、性别、头发颜色和肤色。在我看到的演示中，结果令人印象深刻:婴儿照片与婴幼儿产品搭配，化妆品广告显示在女性照片附近。

![](img/230ceafa56652c8621ce7146a1da8f17.png)

该公司还处于识别照片中公司标志的早期阶段(可以与匹配的产品配对)，并最终希望包括对更多对象的支持，如车辆识别和旅行应用。

Eyealike 并不意味着是一种消费品。相反，它是在企业级别上获得许可的，有一个可定制的后端，Eyealike 将为每个客户的需求量身定制。如果系统运行良好，Eyealike 在寻找客户方面应该不会有任何问题——像 MySpace 这样拥有数十亿张照片的网站很乐意更有效地利用它们赚钱。

Eyealike 试图解决的问题并不是一个新问题，因为许多其他创业公司都试图实现自己版本的图像识别。不幸的是，暗淡的精确度长期以来阻碍了这种技术的发展。Eyealike 声称其算法在 90%的时间内有效——这是一个令人印象深刻的数字，无疑会带来更高的广告收入。但由于这项技术没有消费者前端，我无法亲自尝试。Eyealike 表示，它正在谈判在一个大型社交网络中实施其系统，因此我们可能很快就会亲自尝试。
