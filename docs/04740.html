LinkedIn突破7000万用户；包括超过一百万个公司简介

职业社交网络LinkedIn 的主页显示，该公司目前拥有7000万会员。该公司在二月份在T4拥有6000万用户，并且增长迅速，尤其是在国际市场。LinkedIn网络的首席执行官杰夫·韦纳在去年秋天的一篇博客文章中表示，LinkedIn的会员有一半是国际会员。该公司还列出了一百万份公司简介，类似于用户简介。

韦纳告诉我们，LinkedIn增长最快的是国际市场。
LinkedIn通过在荷兰、和印度、这两个LinkedIn增长迅速的地区扩张国际办公室，减缓了国际市场的增长。

在过去的六个月里，该网络还在不断增加功能，以扩大LinkedIn在网络上的影响力，增加了更深入的Twitter集成，开放了一个API，提供了一个带有 Microsoft Outlook 的插件，并增强了共享选项。

但是，尽管LinkedIn希望将其平台引入更大的网络，但利用网络上的数据为用户和公司增加额外功能的潜力巨大。韦纳最近告诉我们，数据是LinkedIn作为一个网络的未来非常重要的一部分。虽然我们目前不能透露这些数据特征是什么，但可以肯定地说，我们将看到LinkedIn利用网络上丰富的专业内容推出许多新举措。

Weiner 不会对LinkedIn近期的IPO计划发表评论，但该公司显然看到了增长，实现了盈利，并正在努力为该平台增加功能。看看2010年下半年职业社交网络会带来什么，应该会很有趣。
