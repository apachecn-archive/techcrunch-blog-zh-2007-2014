# MakerBot 的 Bre Pettis 说:“我们从三个人、一台激光切割机和一个梦想开始。”

Bre Pettis 继续与《创始人故事》主持人 [Chris Dixon](https://web.archive.org/web/20230204223417/http://www.crunchbase.com/person/chris-dixon) 交谈，两人讨论了经营一家实际上需要螺母和螺栓装配的企业所面临的挑战。在这种情况下，扩展带来了许多软件初创公司不熟悉的一系列全新挑战。

硬件只是一个不同的游戏。例如，正如佩蒂斯所说，“我们用完了发动机，我们买了世界上所有的发动机，我们去买更多的，他们就像你已经得到了所有的发动机。所以我们必须弄清楚如何为我们制造和定制马达。所以在硬件方面，这是一系列的挑战。”

迪克森承认，有时“我们软件人很难发现它有多么不同。”(披露:迪克森是通过方正集体投资 Makerbot 的投资者)。

当时局变得艰难时，艰难的人会去中国。一定要听完整场交流，因为佩蒂斯还谈到了面对面交流作为完成工作的润滑剂的重要性。

下面，佩蒂斯谈谈 Makerbot 是如何起步的。“我们从三个人、一台激光切割机和一个梦想开始，”他说。

迪克森询问了一些模仿者，他们可能会采用 [MakerBot](https://web.archive.org/web/20230204223417/http://www.crunchbase.com/company/makerbot) 的概念，并在这样做的时候将 MakerBot 变成它以前的影子。佩蒂斯似乎一点也不担心，并部分回应道，“我们已经有了一个品牌，我不知道如何准确量化，但你知道，当你想到 3D 打印时，你就是在制作机器人，这很强大。”

两人继续讨论 Makerbot 的开源平台、网站黑客攻击和公司的现金流。
