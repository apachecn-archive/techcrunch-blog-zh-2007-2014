# Grouply 给雅虎/谷歌集团带来了一点脸书

[![grouply](img/f39b5467cf0e9c9252744236a7a15154.png) ](https://web.archive.org/web/20221209174523/http://grouply.com/) [Grouply](https://web.archive.org/web/20221209174523/http://grouply.com/) 是一家试图改善目前超过 1 亿注册用户使用的在线“群组”系统(雅虎/谷歌群组)的初创公司。他们的第一个目标是创建一个简单的管理工具，以便在两个网络上轻松跟踪您的小组的更新。你给 Grouply 你的账户凭证，他们以更方便的方式组织你的账户(见我们之前的[评论](https://web.archive.org/web/20221209174523/http://www.beta.techcrunch.com/2007/11/08/grouply-gives-a-much-needed-upgrade-to-the-board-scape/))。他们的第二个目标是让这些系统跟上最新的社交网络增强功能。

新推出的功能被统称为“群体社交”，包括你所期望的所有社交网络功能。用户资料页面显示您的兴趣、个人历史和联系信息。这些页面还像大多数社交网络一样支持多媒体，允许用户分享来自 YouTube 和 Slide 等网站的照片、视频和“小工具”。成员们也可以互相成为朋友，拥有完全的隐私控制。你可以决定谁可以访问你的个人资料，他们可以看到什么部分，类似于脸书。

当谈到可用性和参与度时，互联网的其余部分显然超过了这些旧的“团体”服务。像 [Tangler](https://web.archive.org/web/20221209174523/http://tangler.com/) 、 [Wetpaint](https://web.archive.org/web/20221209174523/http://wetpaint.com/) 和 [Klostu](https://web.archive.org/web/20221209174523/http://klostu.com/) 这样的网站正在创造全新的系统来提高在线论坛的速度。正如我们之前所说，Grouply 正在采取一种进化的方法，从现有系统中吸收用户和数据，并增强它们的功能。最近 grouply[募集了](https://web.archive.org/web/20221209174523/http://www.beta.techcrunch.com/2008/01/11/grouply-closes-13-million-round/)超过 130 万美元。
