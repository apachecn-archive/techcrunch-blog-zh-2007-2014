# 联想 CEO 抨击 iPad，无视自己平板电脑的缺陷

联想是全球增长最快的个人电脑制造商。过去 7 个季度，这家中国电脑公司的增长超过了所有其他个人电脑制造商，目前是全球销量第三大的个人电脑制造商。毫无疑问，联想是台式机和笔记本电脑领域的主要玩家，而且很快会是平板电脑，其首席执行官杨元庆如是说。

联想最近发布了三款平板电脑，Honeycomb ThinkTab 和 IdeaPad K1 以及 Windows 操作系统的 IdeaPad P1。前两个刚刚随着 Windows tab 在今年秋天上市。凭借这三款各有独特目标人群的平板电脑，联想希望在平板电脑竞赛中一马当先。当然，这意味着要与 iPad 正面交锋，这款产品被联想的首席执行官视为顶级产品，其 500 美元的价格让“小城市、乡镇、低工资阶层、低收入阶层”的人买不起。也许有人应该告诉元清，他的垃圾 IdeaPad 只少了 50 美元。

让我们来看完整的引文，[由金融时报](https://web.archive.org/web/20230204213045/http://www.ft.com/intl/cms/s/2/8c59de6a-cbb4-11e0-8c4b-00144feabdc0.html#ixzz1VlIQ0yKE)提供:

> “苹果只覆盖顶层。用 500 美元的价格，你不能去小城市，乡镇，低工资阶层，低收入阶层。我不想说我们想要大幅降低价格，而是说我们的战略是提供更多的类别，以覆盖不同的细分市场。”

很难反对这种策略。不同的消费者希望他们的平板电脑做不同的事情，一种产品无法覆盖所有的基础。联想 IdeaPad K1 的目标消费者是拥有弯曲外观和网飞应用程序的消费者。prosumer ThinkPad 包含强大的安全套件、主动数字化手写笔和足够的直线，在任何角落办公室都可以看得见。IdeaPad P1 是为出于某种原因仍然需要/想要在平板电脑上安装 Windows 的消费者和公司准备的。

这种多人口统计方法与其他公司的做法大相径庭。但这并不是一个好的开始。

IdeaPad P1 评论开始出现。我拒绝了一个评论样本；现在所有的蜂巢平板电脑本质上都是一样的。Engadget 给了它一个 6/10 的分数，而在 this my next 上的乔安娜·斯特恩[给了它一个更诚实的 4.5/10 分。面向消费者的 Honeycomb 平板电脑上唯一值得注意的功能是网飞应用程序，但实际上受到了 Honeycomb 本身和廉价版本的反击。IdeaPad P1 可能会让一些用户感到高兴，但 449 美元的售价很难让人接受，因为 iPad 只贵 50 美元。当只有 50 美元的价格差异时，很难看出联想如何看待他们的平板电脑冲击低收入人群。](https://web.archive.org/web/20230204213045/http://thisismynext.com/2011/08/19/lenovo-ideapad-k1-review/)

联想正在冲击平板电脑市场，但尚不清楚它是否会有回报。IdeaPad K1 只是另一个陈旧的蜂巢标签，而联想其余三分之二的平板电脑产品面向拥有复制笔记本功能的平板电脑的企业。如果联想想让自己的平板电脑有别于 iPad——他们应该这么做 Android 平板电脑应该首先证明它们如何提高生产率并适应商业世界——这是一个连 iPad 都没有做到的技巧。
