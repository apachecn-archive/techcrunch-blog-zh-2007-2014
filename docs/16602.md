# 多米诺杂志作为一个数字的、可购买的出版物重新推出

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/10/03/domino-magazine-re-launches-as-a-digital-shoppable-publication/>

对于家居装饰爱好者来说，2009 年是黑暗的一年，标志着康泰纳仕的杂志《多米诺》在发行四年后关闭。今天杂志[起死回生](https://web.archive.org/web/20230316161011/http://domino.com/)。

有一些明显的区别。它现在是一家从康泰纳仕自主运营的初创公司。印刷杂志将每季度在报摊上出售，但该团队不出售订阅。它正在深入融合内容和在线商务。

多米诺现在由 [Project Décor](https://web.archive.org/web/20230316161011/http://projectdecor.com/) 运营，这是一家设计电子商务初创公司，由 Andy Appelbaum、Aaron Wallace **、**和 Cliff Sirlin 于 2011 年共同创立。装饰工程团队开始与康泰纳仕接洽，希望找到一种方法来利用休眠的多米诺品牌，很明显，这是领导杂志复兴的合适团队。

Appelbaum 说:“在一个大组织中创业可能不是最好的策略。“所以(我们说)，让我们利用他们的资产、他们的品牌、他们的专业知识、他们的头衔、他们的广告支持、他们的消费者营销数据库，并通过米歇尔·亚当斯(Michelle Adams)将它们与编辑声音一起栓接到我们的技术基础设施上。”

亚当斯是前多米诺编辑，在多米诺关闭后，他创办了在线庇护所杂志 [Lonny](https://web.archive.org/web/20230316161011/http://www.lonny.com/) 。她现在回到 Domino 2.0 来推动内容，这意味着社论的观点将与抓住杂志最初狂热追随者的心的观点基本相同。

康泰纳仕的母公司 Advance Publications 是装饰工程的少数股东。这家初创公司还获得了 LaunchCapital 和 iNovia Capital 以及一些天使投资者的支持。Etsy 的前首席执行官玛丽亚·托马斯(Maria Thomas)加入了达美乐，成为其首任顾问。虽然康泰纳仕将为新的 Domino 提供支持和档案内容，但该杂志现在是一家独立运营的企业。

重新推出的部分动机是，以前的 print Domino 无法发挥其作为购物平台的潜力。

“多米诺走在了时代的前面，”康泰纳仕总裁 Bob Sauerberg 说。“技术进步了，现在让我们能够以一种更具互动性的方式重新想象品牌。该品牌一直在设计爱好者中引起高度共鸣，现在，凭借商业和社交能力，他们可以轻松实现他们想要的外观。”

新的多米诺骨牌是一个大的内容/商业游戏。虽然收入是由商业方面推动的，但该团队强调，他们首先是一家编辑公司。

该网站为访问者提供了许多查看选项，每个选项都经过了优化，以提供略有不同的购物服务。每篇文章的顶部都有按钮，可以阅读文章、购物和查看照片。

在阅读模式下，文章的文字和图片布局给人一种明显的杂志般的感觉。购物这个故事向用户展示了故事中使用的家居用品——如果是关于可可·香奈儿在巴黎的公寓，Domino 将展示与古董相似的产品。“查看照片”模式切换到类似 Pinterest 或 Selby 的平铺格式。它显示了所有没有出现在文章正文中的照片，所有这些照片当然都是可购买的。

在不同的查看模式之间切换可能听起来很多，但每种模式都有非常不同的用途。尽管多米诺的目的是帮助人们购物——以前的印刷版是女性购物杂志《幸运》(Lucky)的姐妹出版物——但将故事和购物垂直领域分开意味着观众不会经常被购买按钮搞得晕头转向。另外，它看起来很漂亮。

就电子商务而言，达美乐避开了代销商模式，转而采用零售商模式。

“出于多种原因，我们从未想成为附属机构。链接和坐在板条箱和桶上的概念破坏了体验的魔力和让读者留在我们平台上的价值。这也扼杀了我们的利润，”Sirlin 说。

该团队表示，商业可能会占收入的 85-90%，尽管他们仍在等待人们对此的反应。目前在报摊上的印刷杂志有 132 页长，其中 128 页是达美乐的内容，相对于其他严重依赖广告的杂志来说，这是一个非常低的比例。

印刷杂志将每季度出版一期，网站的目标是每天发布新内容。除了新内容，达美乐还将在未来六个月发布该杂志档案中的故事，并使它们可以在网上购买。

编辑部现在占了 20 人团队的一半。像任何初创公司一样，这意味着扮演多重角色:例如，内部摄影师负责拍摄所有照片，同时还负责修图。

截至今天，Domino 在传统和移动网站上都已上线，第一版在报摊上。该团队表示，Domino 将在未来六个月内推出一款应用。
