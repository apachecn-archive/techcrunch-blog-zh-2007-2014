# Pinterest 联合创始人保罗·西拉离职，本·希伯尔曼正式担任首席执行官

Pinterest 即将迎来重大变革！

据我们自己的独立消息来源证实，Pinterest 联合创始人保罗·赛拉将离开这家初创公司。这很复杂:虽然几周前从钉扎和公司事务中休息了六周的希拉被列在该公司的 [SEC 文件](https://web.archive.org/web/20230131132325/http://sec.gov/Archives/edgar/data/1506293/000150629310000001/xslFormDX01/primary_doc.xml)上，担任首席执行官，但联合创始人[本·希伯尔曼](https://web.archive.org/web/20230131132325/https://twitter.com/#!/8en)通过他的行动非正式地担任了这一角色。

西尔伯曼是如此令人信服，以至于媒体已经授予他首席执行官头衔，尽管事实是直到上周五他和希拉还在共同运营这家公司。Pinterest 的第三位联合创始人埃文·夏普(Evan Sharp)和同事贾斯汀·埃德蒙(Justin Edmund)专注于产品和设计。

随着 Sciarra 的离职，Silbermann 将成为 Pinterest 的官方(也是唯一的)首席执行官。我们听说 Sciarra 不清楚他下一步会做什么，可能会考虑在一家顶级风险投资公司担任 EIR 的角色。到目前为止，Pinterest 已经从安德里森·霍洛维茨、贝塞麦、罗恩·康韦和其他人那里筹集了 3570 万美元，所以我们确信他会是最棒的。

**更新** : Sciarra 现在[证实](https://web.archive.org/web/20230131132325/https://techcrunch.com/2012/04/03/pinterest-co-founder-paul-sciarra-is-now-eir-at-andreessen-horowitz/)他将成为 Andreessen Horowitz 的常驻企业家。

我们还听说 Pinterest 团队在过去几周一直在埋头准备一些适度的短期产品发布以及更具雄心的长期产品发布。传闻已久的 iPad 应用程序和公共 API 是否即将到来？
