# Joyent 任命新首席执行官，并为新兴的横向扩展企业推出 Joyent7

Joyent 任命 Henry Wasik 为新任首席执行官，并发布 Joyent7 来帮助公司发展新兴的、向外扩展的全球企业。

瓦西克接替了联合创始人贾森·霍夫曼，自大卫·杨今年早些时候离开公司以来，贾森·霍夫曼一直担任临时首席执行官。Wasik 最近担任 Force10 Networks 的总裁兼首席执行官。霍夫曼也是 Joyent 的 CTO。

自动化基础设施环境 Joyent7 是该公司的下一代[智能数据中心](https://web.archive.org/web/20221007033644/http://www.joyent.com/products/smartdatacenter)。像许多打破这一领域壁垒的公司一样，Joyent 通过服务器和网络虚拟化、运营管理和客户自助服务提供了提取硬件的新方法。

此前，西班牙电信推出了基于 Joyent 技术的[即时服务器](https://web.archive.org/web/20221007033644/http://cloud.telefonica.com/instantservers/)。这项服务将在西班牙和英国上线，接下来是巴西、德国和捷克共和国。与印度和非洲公司合作的计划也在发展中。

Joyent7 是这种合作关系的核心:它出现在 Telefonica 数据中心或 Joyent 运营的设备上。该设备配备了可以说是最现代的互联网就绪型横向扩展技术。它包括 Node.js，用于部署数据密集型应用程序的服务器端 JavaScript 引擎。NoSQL 数据库是预先配置好的。 [Percona](https://web.archive.org/web/20221007033644/http://percona.com/) 提供其基于云的可扩展虚拟机，以实现更好的 MySQL 性能。[Riak](https://web.archive.org/web/20221007033644/http://docs.basho.com/riak/latest/)(Basho 产品)是集成的一部分，允许横向扩展、横向分布客户的存储环境。

Joyent7 从几个方面与 Telefonica 建立了关系。但最重要的是，整个世界正在走向移动化，大型企业将需要采用新的运营模式来开展业务。由于 Telefonica 遍布全球，客户可以使用该基础设施在 Joyent 开展业务的美国和世界其他地方进行部署。

Joyent 是那些看起来与众不同的公司之一。它没有赶任何一种潮流——该公司制定了自己的议程，构建不需要投资封闭的融合系统的开源技术。

看来我们即将看到 Joyent 会变成什么样子。这将取决于 Telefonica 等运营商如何看待在智能操作系统上运行智能数据中心的优势。
