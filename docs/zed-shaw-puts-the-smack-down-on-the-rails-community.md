# Zed Shaw 抨击 Rails 社区 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2008/01/01/zed-shaw-puts-the-smack-down-on-the-rails-community/>

 [](https://web.archive.org/web/20230202212654/http://www.flickr.com/photos/imajes/1817811449/) 哇，你不会每天都看到这样的夸夸其谈。Zed Shaw，他创造了 [Mongrel](https://web.archive.org/web/20230202212654/http://mongrel.rubyforge.org/) ，一个几乎每个人都用来为 rails 应用服务的库和 web 服务器，基本上是 Rails 社区的神。在今天一篇名为“ [Rails 是贫民窟](https://web.archive.org/web/20230202212654/http://www.zedshaw.com/rants/rails_is_a_ghetto.html)”的文章中，他抨击了许多领先的 Rails 开发人员、整个社区和许多初创公司(以及谷歌)。

这个帖子很长，但非常有趣，即使对我们这些核心社区之外的人来说也是如此。基本上，Zed 刚刚开始了一场风暴，这场风暴将会在社区中持续几个月。

最有趣的引用:*“这正是 Rails 成为贫民窟的原因。一群半吊子的前 PHP 白痴，他们从来不愿意坐下来真正学习他们在大学里学不到的计算机科学。”*

还有:*“杂种狗之后我找不到一帮猴子来强奸我，所以什么工作都忘了吧。当然，人们会因为他们的小公司而联系我，但我最终会明白，他们只是想利用我来实现他们的想法。他们的想法极其站不住脚。我发誓，如果有人说他们要建立一个社交网络，我会用我的鞋子打败他们。”*

还有:*“还有社交网络白痴。他们都有一个社交网络，还有一些他妈的愚蠢的东西要卖，但当然没有哪个 MBA 真的会编程，所以他们跑来找我。然而，有一个小问题。你看，我有一个商业学位你们这些混蛋。如果你告诉我，你的社交网络会因为包含婴儿照片而取代 facebook，我会当面嘲笑你。他们是有中央情报局支持的老牌玩家。你不会消灭他们的。*

谷歌完全是一场骚乱。他们给了我两次工作机会。我去了，他们从来没有回应。可能是因为他们提供给我的工作——一个已经做了 21 年编程，15 年专业编程的人——是初级系统管理员。一个初级系统管理员在谷歌到底做什么？这可能就像在皇后区的荣耀洞拖地板一样。我告诉他们审查我的简历，给我一个真正的职位。"

(最后一个):*“有了 Rails，我就有了患腕管综合症的骨瘦如柴的公鸡吸盘跟我说话，好像他们要吃掉我的孩子。他们被 PHP 感染的虚弱的头脑不能理解像对象或闭包这样的高级东西。当你用一个愚蠢的框架把愚蠢的企业和愚蠢的人结合起来，这个框架是建立在一个糟糕的平台上的一个他妈的大谎言的基础上，你会得到一个像我这样的人找不到工作的完美风暴。”*
