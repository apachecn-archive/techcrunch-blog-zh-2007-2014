# OneRiot 力争进入新一轮 700 万美元的比赛

![screen-shot-2009-08-27-at-63625-pm](img/18b1f3a2b8fb6da408fd9aca4716d14d.png "screen-shot-2009-08-27-at-63625-pm") [实时搜索引擎 OneRiot](https://web.archive.org/web/20221006071341/http://oneriot.com/) 刚刚宣布了由 Appian Ventures、Commonwealth Capital Ventures 和 Spark Capital 领投的新一轮700 万美元 C 轮融资。公司博客刚刚发布了更多信息。

虽然实时搜索空间目前特别热门，OneRiot 一直专注于[开放](https://web.archive.org/web/20221006071341/http://www.beta.techcrunch.com/2009/07/09/oneriot-real-time-search-api-now-open-to-all/)并扩展其 API 以允许其他人利用其数据。合作伙伴包括雅虎和微软，当然，他们也将很快在搜索领域走得更近。与此同时，搜索巨头谷歌据说对实时空间非常感兴趣，并正在探索自己的做事方式。当然还有 Twitter，它目前在收购 Summize 的基础上提供搜索服务，但据说还有更大的计划。

Twitter 搜索扩展的大部分讨论都围绕着[查看链接数据](https://web.archive.org/web/20221006071341/http://www.beta.techcrunch.com/2009/05/07/google-twitter-to-start-indexing-links-for-search/)(显然 Twitter 前几天[在取消测试前](https://web.archive.org/web/20221006071341/http://www.beta.techcrunch.com/2009/08/25/twitter-wants-to-track-your-clicks/)也在做这件事)。这也恰好是 OneRiot 的专长，[搜索实时空间](https://web.archive.org/web/20221006071341/http://www.beta.techcrunch.com/2009/05/12/twitter-is-talking-real-time-link-search-but-oneriot-is-launching-it-today/)，抓取链接，而不仅仅是状态更新。但他们也清楚地意识到 Twitter 在实时空间的力量，因为他们最近推出了一款 [RiotFeeds 产品](https://web.archive.org/web/20221006071341/http://www.beta.techcrunch.com/2009/08/19/riotfeeds-bundles-mundane-less-tweets-to-give-you-the-best-relevant-links/)，将 Twitter 上的链接分成不同的类别。

该公司上一次融资 1500 万美元是在 2007 年夏天，当时它还不叫 OneRiot。那时候，它被称为 Me.dium，更像是 StumbleUpon 类型的产品。新一轮融资使该公司的总融资额在三轮中达到 2700 万美元。我们被告知，这笔新资金将用于改善三个关键领域的服务:速度、规模和相关性。

OneRiot 的首席执行官是[金巴尔·马斯克](https://web.archive.org/web/20221006071341/http://www.crunchbase.com/person/kimbal-musk)、[埃隆·马斯克](https://web.archive.org/web/20221006071341/http://www.crunchbase.com/person/elon-musk)的弟弟。
