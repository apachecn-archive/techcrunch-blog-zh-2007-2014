AppRedeem推出应用内交易，将免费应用的用户推向付费版本

今年1月， AppRedeem 从SV Angel和Blue Run Ventures筹集了75万美元，为移动应用开发者提供创新、有效的解决方案，旨在推动应用安装和参与。自那以后，这家初创公司一直在悄悄地进入一个热门的新领域:交易。今天，这家总部位于山景城的初创公司正式宣布AppRedeem Deals，这是一个新的移动应用内推广网络，奖励用户持续参与应用，同时帮助开发者推动他们的应用转化为付费版本。

为了试水，这家初创公司从“低能儿测试2”开始，这是目前iTunes应用商店排名第三的免费应用。为了使用这个应用程序作为AppRedeem Deals如何工作的例子，AppRedeem为Moron Test的付费应用程序提供了一个促销折扣。这家初创公司目前在其免费应用程序中提供50%到100%的折扣。AppRedeem的联合创始人谢菲尔德·诺兰(Sheffield Nolan)表示，与Tapjoy不同，AppRedeem Deals旨在确保利用折扣的人已经安装了免费版本，没有任何动机，因此购买付费版本的动机与苹果及其政策完全一致。

诺兰表示，这使得免费应用程序的用户能够利用免费升级的机会，而对于开发者来说，他们的付费应用程序可以更有效地货币化，他们可以将这些推广资金用于那些已经尝试并喜欢他们的应用程序的人。

自6周前推出测试版以来，AppRedeem的会员人数已从55，000人增长到580，000人以上，换句话说，自添加交易以来，会员人数每天增长超过34，000人。

以下是来自前6周AppRedeem“交易”的关键数字。Nolan告诉我，首先，AppRedeem将使用他们的新交易网络作为获取客户的工具。目前，他们没有从提供给玩家的折扣中提成，所以从折扣中赚到的钱都直接流向了开发商。该团队计划在未来提供折扣，并从交易中抽取一小部分。

这位联合创始人表示，这家初创公司的下一个重点是开始帮助开发者提高他们应用的参与度，并开始与更多合作伙伴达成协议，以实现应用内购买促销优惠的更多功能。诺兰说，在上周，AppRedeem达到了盈利水平，因此他们也将在未来几个月寻求A轮融资。(他们还将宣布在不久的将来增加一家大型交易公司的高管。)
