# 英特尔又进行了一次收购:黑客联盟，一个黑客马拉松的平台，现在是 Mashery 的一部分

[英特尔的](https://web.archive.org/web/20230111052319/http://www.intel.com/)收购狂潮仍在继续，最近的一次收购是为了加强其最近的一次收购。它正在收购[黑客联盟](https://web.archive.org/web/20230111052319/https://www.hackerleague.org/)，这是一个管理黑客马拉松的流行平台，将与 API 管理公司[Mashery](https://web.archive.org/web/20230111052319/http://www.mashery.com/)(2013 年 4 月[收购](https://web.archive.org/web/20230111052319/https://techcrunch.com/2013/04/17/source-mashery-is-selling-to-intel-for-more-than-180m/))合并。黑客联盟交易的条款尚未披露，但我们知道这笔交易的金额远远低于据报道英特尔为 Mashery 支付的 1.8 亿美元。只有平台、知识产权和其他资产将移交给英特尔；三位联合创始人迈克·斯威夫特(Mike Swift)、阿贝·斯坦韦(Abe Stanway)和伊恩·詹宁斯(Ian Jennings)都是 20 岁出头，却不是。

Mashery 的创始人柳文欢·米歇尔斯(Michael Michels)告诉我，此次收购背后的想法是增强其已经在日常基础上开展的开发者拓展工作。“我们收购了黑客联盟的资产，以获得一个让黑客马拉松变得伟大的产品，这样我们就可以做更多的事情来支持开发者，”他说。“它增加了一些我们可以支持开发人员社区的东西，并最终随着时间的推移将部分内容构建到我们的核心产品中。”

加入黑客联盟将意味着 Mashery(和英特尔)将大大增加他们为开发者举办的活动。Mashery 每年参与约 80 场黑客马拉松，而黑客联盟自 2011 年 10 月以来在全球范围内举办了约 460 场活动，数据库中有来自这些活动的约 6000 次黑客攻击。(它支持的黑客马拉松包括启动我们的旗舰会议 TechCrunch Disrupt 的版本。)

Mashery 和黑客联盟在此之前就已经有了友好的关系，在黑客马拉松巡回赛中相遇。“我们非常尊重混搭，”斯威夫特告诉我。“这是黑客联盟最好的去处。Mashery 把它带到我们不能去的地方的想法让我非常兴奋。”

斯威夫特与斯坦维和詹宁斯一起创建了黑客联盟，当时他们三人还是罗格斯大学的学生。斯威夫特说，这从一开始就是一个附带项目，他们从黑客马拉松中创造了更多他们自己想要的东西，而不是真正的创业公司。它一直是自举的，直到今年夏天(他们开始考虑黑客联盟长期会发生什么)或多或少都是作为一个副业项目运行的，三位联合创始人在 SendGrid、Etsy 和 PubNub 等公司兼职。他们正在继续其他项目的售后服务。这三人都曾是 HackNY 计划的一部分，该计划组织大学生黑客马拉松等活动，这可以被称为 HackNY 的第一次退出。(斯威夫特正在继续他对学生黑客马拉松的兴趣；他是[职业黑客大联盟](https://web.archive.org/web/20230111052319/http://mlh.io/)的“委托人”，竞争成为校际竞争性学生黑客马拉松的官方联盟。

虽然 Mashery 已经有了一项针对开发人员的现有业务——这对于英特尔及其在该社区的思想共享来说总体上是有益的——但看看黑客联盟的平台在未来如何以及在哪里得到使用将是有趣的。英特尔可以使用该平台的一个领域是它自己的专注于硬件的黑客马拉松。和其他像 Appcelerator 这样专注于 API 的企业一样，Mashery 本身也在和企业做生意，帮助他们进行自己的内部开发，所以这也有可能被用来帮助 Mashery 在该领域拓展业务。
