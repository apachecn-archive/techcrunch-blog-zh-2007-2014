# 谷歌推出知识单元，货币化维基百科

[![](img/a484ee3cf3c7764da3940d918f50528e.png)](https://web.archive.org/web/20221007192816/http://www.crunchbase.com/product/knol)

今天，谷歌[推出了](https://web.archive.org/web/20221007192816/http://www.techmeme.com/080723/p71#a080723p71) [知识单元](https://web.archive.org/web/20221007192816/http://knol.google.com/k)，它是维基百科的替代品，让作者对他们写的文章负责。每篇文章都是由一个作者团队创作的，他们接受署名，并被允许参与他们页面上 AdSense 广告的 rev-share。其他用户可以提交修改，但他们必须得到文章原作者的批准，才能在网站上发布(这基本上是一个经过审核的维基百科)。

谷歌表示，它将允许多人就同一主题创建 Knols。知识单元允许用户对 Knols 进行评级和评论，并可能将排名作为搜索结果的一部分，这样我们就不必筛选相同主题的无数文章。你可以在这里看到一篇示例文章(一篇告诉你如何写诺尔的知识单元文章)[。](https://web.archive.org/web/20221007192816/http://knol.google.com/k/knol-help/basics-of-writing-knols/utb1CqLeS/i7fa36)

这里的大新闻是，通过分配所有权和允许作者在他们的文章中包含 AdSense 广告，谷歌有效地提供了创造好内容的金钱激励。理论上，最好的文章会得到最多的关注，进而获得最多的收入。

不幸的是，这一计划可能会适得其反。我们将开始看到一系列关于最受欢迎内容的文章——预计在接下来的几天里会看到几十篇关于巴拉克·奥巴马和约翰·麦凯恩的传记。对于这些受欢迎的主题，系统应该运行良好——一些幸运的(希望可信的)文章将上升到顶部，其余的将逐渐消失。但是对于不太受欢迎的话题，任何人都没有动力去写任何东西。

维基百科运作良好，因为它几乎就像一个慈善组织。每个人都尽其所能，希望增进世界知识。知识单元的社区可能更关心赚钱，而不是一般的福利，这可能会损害其信誉和社区的参与程度。
