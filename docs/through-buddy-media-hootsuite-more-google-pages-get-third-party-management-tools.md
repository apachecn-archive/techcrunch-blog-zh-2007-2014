# 通过 Buddy Media、HootSuite & More、Google+ Pages 获得第三方管理工具 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/11/15/through-buddy-media-hootsuite-more-google-pages-get-third-party-management-tools/>

上周，Google+推出了“Pages”,让品牌、产品、公司、企业和阳光下的其他所有人都能够建立自己量身定制的 Google+形象。

Google+页面听起来可能会让人想起脸书页面，的确如此，尽管有一些重要的变化。([在这里，乔希深入探讨了 Google+如何在页面争夺战中胜过对手](https://web.archive.org/web/20230203083209/https://techcrunch.com/2011/11/07/google-facebook-pages/)。正如人们可能会猜测的那样，Pages 为品牌和企业提供了与单个 Google+类似的体验，因为品牌可以将人们放入圈子，与选定的群体分享内容，推出视频聊天，与客户进行面对面的对话，所有这些都通过该网站的移动应用程序来实现。

随着 G+页面向所有用户推出，谷歌希望为希望管理其页面的企业提供一套集成，这将增强他们在该平台上的体验，并给他们增加灵活性。今天，Google+ [宣布](https://web.archive.org/web/20230203083209/http://adwords.blogspot.com/2011/11/third-party-tools-to-help-manage-your.html)将推出一个试点项目，基本上允许企业和品牌使用一些第三方应用程序来管理他们的 Google+页面，比如 Hootsuite 和 Buddy Media 等等。

总共有六家社交媒体公司将提前获得 Google+ API，用谷歌的话说，就是“在他们的管理工具中测试 Google+的功能”。

这六家公司包括 Buddy Media、Context Optional、Hearsay Social、HootSuite、Involver 和 Vitrue，它们将向页面所有者提供其工具的子集，使他们能够管理圈子、发布到 Google+、访问见解和监控使用情况。根据 Google+的说法，这六家公司被选中是因为“他们在帮助品牌和企业管理和评估他们在社交网络上的存在方面拥有丰富的经验”。

虽然现在还为时尚早，但这些社交媒体集成似乎为页面所有者提供了一些广泛功能的开端，特别是在目标定位、高级页面管理和分析方面。

例如，从 Heresay 的公告中可以看出，该公司与 Google+的合作将允许页面所有者跨品牌页面安排帖子和活动，使用该公司的 CRM 工具来管理客户关系，并利用社交分析来查看每个品牌和本地页面的指标。

或许更重要的是，Heresay 将允许其用户将内容定位于特定的 Google+圈子，允许营销人员发布(和安排)针对特定受众的活动。(Hootsuite 也为特定的商业受众群体提供类似的信息定制服务。)例如，营销人员将能够创建一个“忠实客户”圈子，并能够发布特定的信息、交易等。对那些超级用户来说。

因为 Google+不允许多个用户访问和管理一个特定的页面，所以与这些社交媒体公司的新整合似乎将是一个很好的替代。现在，只需要谷歌向个人账户开放这种功能。
