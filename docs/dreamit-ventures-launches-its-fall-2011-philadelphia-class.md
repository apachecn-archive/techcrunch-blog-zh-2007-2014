# DreamIt Ventures 推出 2011 年秋季费城课程

创业加速器 [DreamIt Ventures](https://web.archive.org/web/20230205013507/http://www.crunchbase.com/financial-organization/dreamit-ventures) 今天推出了 2011 年费城秋季班，其中包括与[康卡斯特风险投资](https://web.archive.org/web/20230205013507/http://www.crunchbase.com/financial-organization/comcast-interactive-capital)合作选出的五家少数股权公司。今年 5 月， [DreamIt 和 Comcast 联手](https://web.archive.org/web/20230205013507/https://techcrunch.com/2011/05/19/dreamit-and-comcast-partner-for-new-program-for-minority-entrepreneurs/)，从 DreamIt 的秋季项目开始，向少数族裔领导的初创公司提供种子资金、培训和指导。

目前的班级包括宾夕法尼亚大学沃顿商学院、普林斯顿大学、杜克大学、斯坦福大学、哈佛大学、耶鲁大学、哥伦比亚大学和麻省理工学院的学生和校友。初创公司的创始人有在谷歌、雅虎、英特尔、高盛和摩根大通的工作经验。

其中五家公司是由 DreamIt 和康卡斯特公司的风险投资子公司康卡斯特风险投资公司共同选择的，作为其少数民族企业家加速器计划(T7)(MEAP)的一部分。除了 DreamIt 为少数族裔领导的创业公司提供的资金外，该计划还提供了 35 万美元的额外资金。目前的集团包括业主是非洲裔美国人，亚洲人，西班牙人和印度人。

MEAP 是康卡斯特创建的 2000 万美元基金的第一个项目，作为收购 NBC 环球的一部分。该基金还将用于投资 DreamIt 以外的其他少数族裔领导的创业公司和倡议。
