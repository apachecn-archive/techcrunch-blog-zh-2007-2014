# 未被征服的国家，被官僚们削弱了

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/05/30/unconquered-nation-crippled-ethiopia-internet/>

![](img/05e1acc8112e2aee7ef39a4c3ced1514.png "addis") 这里似乎是撒哈拉以南的月份:先是萨拉·莱西[去了](https://web.archive.org/web/20221007140629/https://beta.techcrunch.com/2011/05/13/yes-there-are-tech-startups-in-nigeria-here-are-my-favorites/) [去了](https://web.archive.org/web/20221007140629/https://beta.techcrunch.com/2011/05/14/you-think-hollywood-is-rough-welcome-to-the-chaos-excitement-and-danger-of-nollywood/) [尼日利亚](https://web.archive.org/web/20221007140629/https://beta.techcrunch.com/2011/05/15/the-chilling-story-of-genius-in-a-land-of-chronic-unemployment/)，现在我在[亚的斯亚贝巴](https://web.archive.org/web/20221007140629/http://en.wikipedia.org/wiki/Addis_Ababa)，埃塞俄比亚的首都和非洲第四大城市。感觉像一个新兴城市。到处都是起重机和建筑工地，到处都是崭新的玻璃钢建筑，到处都是出售电脑和手机的商店。主要的大街上挤满了制造、交易、修理、卸货、销售和一般忙碌的人们。

不要误解我:这仍然是一个贫穷的国家。停电是经常发生的事情，在城市宽阔的大街上巡逻的出租车是生锈的拉达，小街上满是垃圾，两旁是铁皮棚屋，还有乞丐和野狗在巡逻。但是我只是偶尔看到一些有毒的停滞，而我在撒哈拉以南的其他地方经常看到这些停滞。这感觉像是发生事情的地方。如果不是因为一个巨大的障碍:它自己的政府，这个城市和国家可能会在技术的催化下处于真正变革的风口浪尖。

“哦，他们很棒，”一名外籍人士面无表情地谈到埃塞俄比亚电信(ETC)，这家政府垄断企业控制着全国所有的电话、手机和互联网服务，房间里的每个人都大笑起来。他摇摇头。“不，不。他们太可怕了。”不仅仅是审查，尽管这已经够糟糕的了:整个 blogspot.com 域名都被屏蔽了，还有脸书的各种页面和报纸。但更让人不爽的是 ETC 的彻头彻尾的无能。

对埃塞俄比亚的网吧非常简略的了解，就会很匆忙地确认他们所说的一切。连接速度变化很大，如果你能连接的话，速度会非常慢。埃塞俄比亚仍然没有连接上为东非带来宽带的海洋光纤；结果，整个国家只有 1.2 千兆的带宽供其 8500 万人使用，而每天有更多的人上网。你算算。

如果你有一个连接问题，事情会变得更糟；据说，ETC 的客户关怀让康卡斯特看起来像劳斯莱斯。一名当地的安全黑客和渗透测试人员(“只有白帽子，”他咧嘴笑着向我保证)做了个难以置信的鬼脸，想起了:他也在一家大型非政府组织工作，上次他们遇到严重的连接问题时，“我不得不去(埃塞俄比亚电信的)数据中心，自己解决。”

但是人们肯定可以用一道 [VSAT](https://web.archive.org/web/20221007140629/http://en.wikipedia.org/wiki/Very_small_aperture_terminal) 菜肴来解决这个问题吗？娱乐片《T4》等的创作者烂透了”脸书·佩奇摇摇头:“没有 VSAT，这不可能。”除了最强大的组织，政府禁止他们；他估计这个国家只有不到六个这样的机构，为联合国、世界银行、非洲联盟工作。繁文缛节还不止于此:你需要许可证来进口“任何带有 IP 号码的东西”，这需要一个月的时间——而他们通常会说不。

这是一个值得骄傲的国家，这是有原因的:埃塞俄比亚是唯一一个打败了潜在的欧洲殖民者并在整个殖民时代保持独立的非洲国家。但是他们需要开始把非洲其他地区作为榜样。“他们在肯尼亚远远领先于我们，”当地黑客绝望地说，这意味着那里的移动和互联网提供商之间的激烈竞争，以及因此而蓬勃发展的接入和创新。

唉，埃塞俄比亚政府似乎喜欢垄断、保护主义和官僚主义。我相信移动互联网接入是一种变革力量，可以将非洲国家变成经济雄狮，与亚洲四小龙相抗衡——但前提是它速度快、价格低、无处不在。如果埃塞俄比亚的每一点互联网都被恐龙垄断所控制，没有竞争激励去改善，那么这在这里永远不会发生。
