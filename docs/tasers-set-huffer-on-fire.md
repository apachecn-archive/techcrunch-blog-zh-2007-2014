# 泰瑟枪点燃了赫夫

很明显，这个人在我的家乡俄亥俄州哥伦布市被警察用电击枪击中后着火了。虽然完全有可能是泰瑟枪烧伤了他，但也有可能是年轻的丹尼尔·伍德自燃了。

> 本周，在警察收到泰瑟枪的一天后，其中两人拍灭了丹尼尔·伍德身上的火焰，丹尼尔·伍德是一名 31 岁的无家可归者，据报道，他一直在吸入喷雾罐中的化学物质以获得快感。兰开斯特警察局长大卫·贝利说，伍德伤势不重，但为了以防万一，他已被送往医院。

我的意思是，这位老兄在喘气，所以他可能全身都是甲苯，使他对明火和高温毫无反应。这不是大火焰，因为他们能够拍出他，然后逮捕他。

另外，我想这家伙是我的高中同学。
