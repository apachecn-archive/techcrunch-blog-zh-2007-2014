# 在 pivot 之后，SoundCloud 拥有 300 万注册用户

音频平台 SoundCloud 最初面向音乐制作人，但现在面向更广泛的消费者，该平台宣布其注册用户已达 300 万*。*

 *考虑到这家总部位于柏林的初创公司就在去年 5 月还拥有[百万用户](https://web.archive.org/web/20230203160321/http://eu.beta.techcrunch.com/2010/05/18/now-a-million-on-soundcloud-this-startup-is-scaling-globally/)——如果你愿意的话，可以称之为 pre-pivot——方向的改变似乎是有回报的，或者至少没有对公司的增长轨迹产生负面影响。也就是说，这些都是注册用户，只是不是活跃用户，但增长看起来确实稳定。

为了进一步说明这一点，SoundCloud 表示，在过去的 100 天里，它增加了 100 万用户，创始人兼首席执行官 Alexander Ljung 将其描述为“指数级增长”，证明这家初创公司“走在正确的道路上”。对于 SoundCloud 的支持者来说，这很可能是悦耳的音乐——这是双关语。这家初创公司现在更直接地与 [Cinch](https://web.archive.org/web/20230203160321/http://www.cinchcast.com/) 和 [Audioboo](https://web.archive.org/web/20230203160321/http://audioboo.fm/) 等公司竞争，最近证实了来自 Index Ventures 和 Union Square Ventures 的新一轮未披露的融资。在此之前，2009 年 4 月，它从 Doughty Hanson Ventures 为€融资 250 万英镑。

最近，在乐队 R.E.M .发起了一场众包比赛，将他们即将发行的专辑《Collapse Into Now》中的歌曲进行混音后，SoundCloud 获得了一些不错的宣传。[选择](https://web.archive.org/web/20230203160321/https://techcrunch.com/2011/02/08/r-e-m-remix-soundcloud-garageband/)上传和分享的平台是 SoundCloud，这不会对 growth 造成任何伤害。

这似乎与新闻报道不谋而合，新闻报道称，这家初创公司已在旧金山开展业务，以更好地服务于美国用户群，并“与有意使用其 API 的美国科技公司进行更密切的合作。”*
