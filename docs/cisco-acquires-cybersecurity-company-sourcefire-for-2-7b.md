# 思科以 27 亿美元收购网络安全公司 Sourcefire

思科[刚刚宣布](https://web.archive.org/web/20230113155032/http://www.marketwire.com/press-release/cisco-announces-agreement-to-acquire-sourcefire-nasdaq-csco-1813520.htm)收购 [Sourcefire，](https://web.archive.org/web/20230113155032/http://www.sourcefire.com/)一家创造网络安全产品以保护公司免受攻击的公司。收购价格为 27 亿美元，即每股 76 美元的现金加基于留任的激励。

总部位于马里兰州的 Sourcefire 成立于 2001 年，提供大量安全产品，确保威胁检测和网络防护，包括预防系统、防火墙和高级恶意软件防护。大中型组织和政府机构使用 Sourcefire 来保护其动态网络、端点、移动设备和虚拟环境。

正如思科解释的那样，此次收购“为思科增加了一支具有深度安全 DNA 的团队，并将加速思科防御、发现和补救高级威胁的安全战略的交付。”2012 年底，2007 年上市的 Sourcefire 报告收入 2.231 亿美元，同比增长 35%。

交易完成后，Sourcefire 员工将加入思科安全团队。

思科一直在积极构建其安全技术组合。今年早些时候，[公司收购了](https://web.archive.org/web/20230113155032/https://techcrunch.com/2013/01/29/on-an-international-buying-campaign-cisco-acquires-cyber-threat-protection-startup-cognitive-security/)捷克网络安全初创公司 Cognitive Security。去年，思科收购了数据中心安全公司 Virtuata。网络安全无疑是越来越多大公司关注的一个领域，显然思科希望成为抵御任何威胁的首选。
