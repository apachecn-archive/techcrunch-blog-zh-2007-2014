# 新版 gOS 将在 CES 上发布，配有 Google Gears

[![gos.jpg](img/451cec33a19892177b096d4dd329a3d1.png) ](https://web.archive.org/web/20221007222018/http://www.thinkgos.com/) Linux 发行版 [gOS](https://web.archive.org/web/20221007222018/http://www.thinkgos.com/) ，我们第一次写到了它在沃尔玛[11 月 1 日](https://web.archive.org/web/20221007222018/http://www.beta.techcrunch.com/2007/11/01/google-pc-at-wal-mart-for-200/)以 199 美元的价格出售的时候，它的新版本[即将在下周拉斯维加斯的 CES 上发布](https://web.archive.org/web/20221007222018/http://www.businesswire.com/portal/site/home/index.jsp?epi_menuItemID=887566059a3aedb6efaaa9e27a808a0c&ndmViewId=news_view&ndmConfigId=1000017&newsId=20080103005831&newsLang=en)。

在你问(就像一些人之前问的那样)为什么 TechCrunch 会写一个 Linux 发行版之前，gOS 是不同的。这是第一个以 web 应用为标准的主要操作系统，正如我在去年 11 月写的，它并不完美，但它很可能是未来的一个标志；在这个世界里，网络应用是操作系统的主要焦点，而不是本地桌面应用。

最值得注意的是，gOS 2.0 将标配 Google Gears。在 Google Gears 上运行的基于网络的应用还不多，但是 Google Reader(用于 RSS 阅读)[和 Zoho Writer](https://web.archive.org/web/20221007222018/http://www.beta.techcrunch.com/2007/08/21/zoho-goes-offline-in-a-good-way/) (用于 MS Word 替代品)是其中之一。

gOS 2.0 的其他功能包括基于浏览器的网络摄像头应用程序 gBooth，该应用程序专注于与 YouTube 和脸书等网站合作。

访问 [CrunchGear](https://web.archive.org/web/20221007222018/http://www.crunchgear.com/) 查看 CES 的完整报道，希望他们也能在本周参观 gOS 展台。
