# 铱星卫星与俄罗斯卫星相撞

移动卫星电话服务提供商[铱星](https://web.archive.org/web/20221007003120/http://www.iridium.com/)发现它的一颗轨道卫星在周二与一颗不工作的俄罗斯卫星在西伯利亚上空 500 英里处相撞时离线。根据美国国家航空航天局的说法，事故产生的碎片具有潜在的危险，尽管它“非常小，在可接受的范围内”科学家将追踪这些碎片。

总部位于马里兰州贝塞斯达的铱星公司表示，它的一些无线客户端可能会出现中断，但该公司计划在今天之前让系统正常工作，并计划在不到一个月的时间内发射一颗备份卫星。铱星公司表示，这次碰撞是一次“极不寻常、概率极低的事件”

俄罗斯军用卫星发射于 1993 年，显然在两年内出现了故障。据美国宇航局称，美国宇航局计划在本月底发射的发现号航天飞机不太可能受到坠机事件的影响，国际空间站也不会受到伤害，但能够“在必要时进行碎片规避机动”。
