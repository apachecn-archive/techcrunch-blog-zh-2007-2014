# 三星回应更多操作系统收购传闻:不，我们也不会收购 MeeGo

米果悬而未决。

诺基亚不久前拒绝了 MeeGo。与此同时，英特尔承诺继续支持开源社区，但是[报道称](https://web.archive.org/web/20221231035707/https://techcrunch.com/2011/09/02/report-intel-to-kill-meego-development-temporarily/)它已经决定暂时停止平台的开发。于是一个强大的谣言工厂开始运转。

[Mobiledia 周二报道称](https://web.archive.org/web/20221231035707/http://www.mobiledia.com/news/106629.html)三星有兴趣收购 MeeGo，引用了未确认的行业消息来源。诚然，三星确实需要在 OS 部门实现多元化。谷歌罗拉交易应该会对所有主要的安卓合作伙伴产生影响。此外，还有整个[“微软利用安卓赚钱”](https://web.archive.org/web/20221231035707/https://techcrunch.com/2011/07/06/microsoft-aims-at-samsung-to-continue-milking-android-cash-cow/)的事情要担心。

但是三星有更多的担忧。该公司是苹果最喜欢的诉讼目标(也是其最大的竞争对手之一)，Android 是苹果在移动领域占据统治地位的最大障碍。新的操作系统将大大有助于苹果摆脱三星的束缚。但是 MeeGo？虽然三星可能会收购 MeeGo，但这种可能性不大。

三星有自己的软件平台:Bada。如果该公司真的想部分脱离 Android，为什么不在它已经拥有的平台上工作，并使之完善呢？此外，就连 MeeGo 的联合创始人诺基亚也不太喜欢这个平台。正如我之前提到的，英特尔似乎也已经释怀了。为什么三星要抢走一个已经被制造商抛弃的平台？

三星没有在平台上倾销，而是采取了外交途径回应:“Meego 是一个开源项目，不能成为收购的目标，”三星代表 James Chung 告诉 CNET。

就软件收购而言，所有的目光都投向了三星。当 webOS 被惠普部分抛弃时，也有传言称[三星可能有兴趣收购它](https://web.archive.org/web/20221231035707/https://techcrunch.com/2011/08/29/samsung-may-buy-webos-recruits-former-hp-exec/)。三星[否认这些谣言](https://web.archive.org/web/20221231035707/https://techcrunch.com/2011/09/02/samsung-ceo-we-dont-want-hps-garbage-or-something-like-that/)就像他们否认这些一样快。

Chung 提到，“三星一直在大力投资于自己的软件、解决方案和内容，以使其产品与众不同，并为消费者提供更丰富的体验。”换句话说，听起来 Sammy 将暂时坚持使用 Android，同时继续修补 Bada 作为备份项目。
