# 看似无法回答的问题:政府能利用位置数据追捕我们吗？

你记得 [Locationgate](https://web.archive.org/web/20230204230206/https://techcrunch.com/2011/04/20/surprise-your-iphone-is-tracking-your-every-move/) 吧？在两名德国研究人员发现 [iPhone](https://web.archive.org/web/20230204230206/https://techcrunch.com/tag/iphone) 会自动追踪并存储位置数据后，这起巨大的全国性丑闻让[苹果](https://web.archive.org/web/20230204230206/https://techcrunch.com/tag/apple)和[谷歌](https://web.archive.org/web/20230204230206/https://techcrunch.com/tag/google)成为了我们不满的焦点。这一丑闻引发了许多关于我们应该在位置跟踪方面划清界限的调查，其中一项特别获得了国家安全局马修·奥尔森(Matthew Olsen)的智慧，他是国家反恐中心的负责人，也是国家安全局的总法律顾问。

在参议院情报特别委员会的确认听证会上，奥尔森回答了我们一直在问的问题:政府可以使用我们的位置数据来跟踪我们吗？

据《华尔街日报》报道，俄勒冈州参议员罗恩·怀登提出了这个问题，询问该男子是否可以“使用手机网站数据来追踪美国人在国内的位置”奥尔森的回答充其量是模棱两可的，尽管他似乎暗示政府事实上确实拥有这种权力:“在某些情况下，这种权力可能存在，”奥尔森说。“这是一个非常复杂的问题。”

那就清楚了，不是吗？还记得过去的好时光吗？那时联邦政府实际上不得不做一些工作来寻找犯罪嫌疑人。美好时光。无论如何，由于奥尔森的回答缺乏任何清晰感，“情报界”——另一个非常模糊的参考——打算写一份备忘录，实际上回答这个问题。加州参议员黛安娜·范斯坦要求在委员会 9 月再次召开会议时完成备忘录。

参议员罗恩·怀登提出的问题本质上属于第四修正案的范畴。第四修正案承诺保护公民和他或她的财产免受不合理的搜查。搜索是否不合理取决于被搜索者对自己隐私的主观和客观信念(以及其他因素)。

一个罪犯可能会主观地认为他的旧烟头是私有财产，但这不会阻止法律与秩序侦探用它来获取 DNA 证据。但是，如果社会作为一个整体集体认为对隐私的期望是合理的，那么从搜索中收集的证据必须被扔掉。换句话说，如果肮脏的秘密没有保存在私人的地方，你就可以对你的隐私保护说再见了。

但是手机是私人的吗？我们用密码把它们锁起来，我们当然不会像“借”衣服、书和 DVD 那样“借”对方的手机。那么，我们的私人手机数据如何被用来追踪我们呢？

Locationgate 之所以让这么多人感到不安，是因为它打破了我们的手机被锁在小保险箱里的看法，这些保险箱保存并守护着我们所有的秘密。但苹果和谷歌都坚称，收集的数据是加密的，如果发送回公司，将是匿名的，这再次证实了我们最初的想法:我们的手机是隐私的，当涉及到你的手机时，期待隐私是合理的。

希望这份备忘录也是如此。
