# 必须有人说:iTunes Lite TechCrunch 的时代到了

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/09/09/somebody-has-to-say-it-its-time-for-itunes-lite/>

![9donotwant](img/456c8f9a973eb4397131b7db1ca66331.png "9donotwant")
脸书一体化。在推特上发布我的音乐爱好。苹果电视。全屏专辑临时演员。App 管理。一整个网上商店。智能播放列表。CD 刻录。铃声创作。88MB 时最大。所有这些都在 OS X 上的*唯一的*音乐播放器中。在某种程度上，这已经够了，对我来说，几年前就够了。看在上帝的份上，苹果，*我只想演奏我的音乐。*

多年来，苹果一直在增加 iTunes 的功能，虽然一些改进受到了欢迎，但许多改进只是增加了臃肿。是时候了——真的是时候了——苹果应该推出一些轻量级的基本产品了。我知道 iTunes 对苹果来说是一个楔子(和武器)，我不建议将其摧毁，但考虑到该程序没有可信的替代品，我觉得苹果已经不再简单地增加功能自助餐，并开始强迫用户。

在我的电脑上，我使用 Winamp——用了很多年，非常喜欢。我知道我不能复制 OS X，但至少给我一个没有 80%我永远不会用的功能的程序。

我只是更喜欢做一两件事，并且做得很好的应用程序——当然我不是唯一的一个。我在 MPlayerOSX 或 VLC 播放电影，并自己整理，我知道很多人都这样做。即使我真的使用 Twitter，我也不会想在 Twitter 上发布我正在听或购买的东西；LastFM 很好地解决了这个问题，并且已经有了一个客户端，或者可以嵌入到许多服务中。喜欢脸书的东西也一样。还有 App 管理？我没有 iPhone 或 iPod，为什么我希望我的媒体播放器包含支持？你可以隐藏它的一部分，但远非全部，让我不安的是它总是潜伏在那里，就在表面之下。等待中。

我猜，Songbird 是存在的(我应该换一个)，但它仍然无耻地模仿 iTunes，并添加了更多我不想在音乐播放器和浏览器中使用的功能。我已经有浏览器了，伙计们。使用*中的*为您带来丰富有趣的内容。[还有什么？](https://web.archive.org/web/20221007023851/http://en.wikipedia.org/wiki/Comparison_of_media_players#Audio_players_2) [三极管](https://web.archive.org/web/20221007023851/http://www.panic.com/Audion/index.html)，弃用这五年？ [Cog](https://web.archive.org/web/20221007023851/http://cogx.org/) ，弃用一年半？[女妖](https://web.archive.org/web/20221007023851/http://banshee-project.org/2008/11/12/banshee-14-released/)和 [Vox](https://web.archive.org/web/20221007023851/http://www.voxapp.net/) ，六个月？我看到论坛上有人建议在虚拟机中运行 Winamp 以节省 RAM！我们在一个镀金的笼子里，Mac 用户们，不像 iPhone 的有门有巡逻的移动娱乐花园，这不是一个你应该满意的地方。

每当我有勇气提到一款珍贵、完美的苹果产品的膨胀时，许多 Mac 用户都会愤怒地附和。相信有些人可能只是想播放他们的音乐，或者可能以不同于 iTunes 的方式组织和浏览音乐，这真的是一种想象吗？我认为，对于每一个兴奋地让 Genius 挑选他们聚会的音乐的人来说，都有人无法忍受播放列表的工作方式。对于每一个喜欢 iTunes 组织专辑和歌曲的人来说，有一些人对其展示收藏的方式感到沮丧和倒退。例如:

那么 iTunes 不适合的人应该做些什么呢？多年来，苹果和 OS X 开发者的回答一直是“应付就好”

与此同时，在我的 XP(和 7)上的几个选项中，有非常有竞争力的免费和/或开源的替代品，可以替代微软的 iTunes，Windows Media Player——这并不像每个人说的那样糟糕，但上帝啊，这并不好。看看[Winamp](https://web.archive.org/web/20221007023851/http://www.winamp.com/player/features)中可用的特性。现在观察以下屏幕:

为什么我没有一个 iTunes？当我说我永远不会使用 Genius 或脸书集成时，苹果是不是不信任我？难道他们没有能力将这些颓废的边车应用从他们的媒体播放器的基本功能中分离出来吗？无论如何都是失败的。

冒着我的批评过于笼统的风险，某些种类的软件缺乏多样性是对 OS X 无可辩驳的抱怨之一。我的 PC 上有几十个程序在 Mac 上没有对等的(或只有一个很弱的)程序，很少是因为，正如人们常说的那样，功能在 OS 或诸如此类的操作系统中是重复的。如果我不离开我的电脑，我会把它们列出来给你。但 iTunes 这件事是那个更大问题的症状。由于没有其他选择，苹果的选择变得越来越根深蒂固，随着它变得根深蒂固，它可怕地扩展了它的触角，并产生了像现在的 iTunes 这样的东西(在较小程度上，还有 iMovie、iPhoto 和其他东西)。在今天的恶作剧之后，这个项目比以往任何时候都更加庞大。

最后，在我看来，苹果制造一个更小的媒体播放器太容易了，他们一定是做了一个*选择*不这么做。考虑到没有其他选择，这个决定对像我这样的用户来说是非常有害的。不幸的是，这不太可能改变，因为它显然没有损害他们的业务。看来这将继续是他们的方式或高速公路，除了没有高速公路。

想要我在苹果贬低为不可用的平台上拥有多年的东西并不算被宠坏。我认为我的要求很合理，尽管我并不期望世俗的方式会因为我的方便而被搁置。这是我想要的:
