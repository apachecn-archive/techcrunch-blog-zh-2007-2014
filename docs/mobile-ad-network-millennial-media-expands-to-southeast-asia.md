# 移动广告网络 Millennial Media 向东南亚扩张

移动广告网络 Millennial Media 宣布向亚太地区扩张。在 AdMob [被谷歌](https://web.archive.org/web/20230204123850/https://techcrunch.com/2009/11/09/google-acquires-admob/)收购，苹果[收购 Quattro 之后，Millennial 是现存最大的独立广告网络之一。](https://web.archive.org/web/20230204123850/https://techcrunch.com/2010/01/04/apple-acquires-quattro-wireless/)毫无疑问，许多科技公司都将千禧一代视为收购目标，但尽管移动广告领域的[整合](https://web.archive.org/web/20230204123850/https://techcrunch.com/2011/04/22/valueclick-to-acquire-mobile-ad-network-greystripe/)加剧，该公司仍设法保持独立。

千禧传媒已经在美国和西欧占有一席之地。该公司将邀请罗伯特·伍尔夫瑞(Robert Woolfrey)帮助领导公司在东南亚的业务。Woolfrey 将专注于扩大 Millennial 在该地区的市场品牌和代理销售。

那么为什么是东南亚呢？Gartner 预测，到 2015 年，亚太地区的移动广告行业将从 16 亿美元增长到 69 亿美元，千禧一代希望能够利用这一点。该公司还将韩国、日本和中国视为其他扩张地区。

国际扩张可能是该公司上市前计划的一部分。首席执行官兼创始人保罗·帕尔米里有雄心让公司上市，考虑到今年似乎是科技公司 IPO 之年，时机可能是正确的。今年 5 月，彭博报道称 Millennial [正在与银行家](https://web.archive.org/web/20230204123850/http://www.bloomberg.com/news/2011-05-18/millennial-media-is-said-to-talk-with-banks-about-ipo-as-mobile-ads-surge.html)讨论 IPO 事宜，IPO 可能在今年秋季或 2012 年初进行，对该公司的估值高达 7 亿至 10 亿美元(AdMob 以 7 . 5 亿美元的价格被谷歌收购)。公司现在的收入是 5000 万美元。
