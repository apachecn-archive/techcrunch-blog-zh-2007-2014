# Yelp 因运行“勒索计划”而遭到集体诉讼

两家律师事务所，来自迈阿密的贝克·李和圣地亚哥的威斯顿律师事务所分别向洛杉矶联邦法院提起集体诉讼，指控当地商业评论和评级网站运营商 Yelp 的不公平商业行为。

该诉讼的原告是加利福尼亚州长滩的一家兽医医院，据说他要求 Yelp 从网站上删除一篇负面评论，据称遭到了这家旧金山初创公司的拒绝，之后其销售代表多次联系医院，要求每月支付大约 300 美元，以换取隐藏或删除评论。

你说，听起来很熟悉？

你可能会想到去年，当时东湾快报刊登了一篇爆炸性的报道，基本上是指控 Yelp 参与了[“敲诈勒索 2.0 业务”](https://web.archive.org/web/20230317123924/http://www.eastbayexpress.com/eastbay/yelp-and-the-business-of-extortion-20/Content?oid=1176635)，其中涉及了类似的内容。记者 Kathleen Richards 发表这篇文章后不久，Yelp 激烈地否认了一切，并称她的文章不准确。

现在，该公司将不得不在法庭上为自己辩护，而不是在公司博客上。

诉讼实质上是指控这家[资金雄厚的](https://web.archive.org/web/20230317123924/http://www.crunchbase.com/company/yelp)初创公司实施了一项“敲诈计划”，并实施了“不道德的销售行为”来创造收入，即公司员工打电话给企业，要求他们以广告合同为幌子每月付款，以换取删除或修改负面评论。

该案件名为猫狗动物医院诉 Yelp 公司，于 2010 年 2 月 23 日提交，目前正在美国加州中区地方法院待审。我们收到了 Yelp 的电子邮件，正在等待回复。

**更新:**一位 Yelp 代表评论如下:

> Yelp 基于我们可信的内容，为数百万消费者和企业提供有价值的服务。这些指控显然是错误的，因为许多在 Yelp 上做广告的企业都有正面和负面的评论。这些企业意识到这两种反馈都提供了真实性和价值。经营一家好企业很难；提起诉讼很容易。虽然我们还没有看到有问题的诉讼，我们将积极地争论它。

这起集体诉讼发生的几周前，Yelp 从 Elevation Partners 获得了一笔巨额投资，几个月后，我们报道了这家公司[退出了](https://web.archive.org/web/20230317123924/https://techcrunch.com/2009/12/20/yelp-walks-away-from-google-deal-and-half-a-billion-dollars/)[5.5 亿美元收购谷歌](https://web.archive.org/web/20230317123924/https://techcrunch.com/2009/12/17/google-acquire-buy-yelp/)的交易。
