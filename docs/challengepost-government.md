# 挑战邮政成为政府承包商

![](img/389b90aea5232da967885e0dd9074cd7.png)

我们能通过博弈机制得到更好的政府吗？我们马上就会知道了。今天， [ChallengePost](https://web.archive.org/web/20221216141339/http://www.challengepost.com/) 被[命名为](https://web.archive.org/web/20221216141339/http://www.gsa.gov/Portal/gsa/ep/contentView.do?contentType=GSA_BASIC&contentId=29692)美国联邦政府的官方在线“挑战平台”。任何想要发起挑战以获取公民想法的政府机构都可以使用 ChallengePost 作为收集想法的方式，甚至为最佳想法提供现金奖励。

ChallengePost 举办了纽约市大型应用程序大赛和米歇尔·奥巴马的健康儿童应用程序挑战赛。这些挑战中有许多都围绕着以创造性的方式利用政府数据，或者开发出具有某种公民福利的软件。

ChallengePost 将创建一个新的政府机构挑战中心。它将于 7 月推出。把所有的挑战放在一个地方会更容易被发现。然而，并不要求机构使用质疑公告。如果愿意，他们可以使用其他平台。例如，美国宇航局使用 [InnoCentive](https://web.archive.org/web/20221216141339/http://www.innocentive.com/) ，因为许多科学家和工程师都在那里工作。但是 ChallengePost 是经过审核和批准的官方版本。政府的挑战也将出现在 ChallengePost.com。

并不是每天都有创业公司获得联邦政府的客户，但这是一种“零成本”合同。“这个平台是完全免费的，”美国总务管理局新媒体和公民参与中心主任 Bev Godwin 说。换句话说，ChallengePost 不会直接从中获利。

但这是 ChallengePost 的一个很好的推广参考，该公司可以对额外的服务收费，例如围绕如何定义挑战的咨询。首席执行官布兰登·凯斯勒确认说:“将有额外的服务是有价格的”。然而，他不会透露这些服务的细节，也不会透露他计划向纳税人收取多少费用。啊，透明政府在行动！
