# 随着 IE8 开始下滑，IE 浏览器份额最终跌破 50%

![](img/d72a7c4088be6ef8abcdd7494a0aa57b.png "ie")

根据网络分析公司 [StatCounter](https://web.archive.org/web/20221006112953/http://gs.statcounter.com/#browser-ww-monthly-200809-201009) 的数据，就在两年前，微软的 Internet Explorer 网络浏览器控制了 67%的全球市场。从那以后就一直在走下坡路。

根据该公司的最新数据，上个月，即 2010 年 9 月，标志着 IE 在过去十年中首次跌破 50%的份额大关。当然，这些数据只是统计了 StatCounter 收集的数据，但他们声称这是一个全球范围内的庞大数据集——来自数百万个网站的数十亿次访问。趋势很明显:两年前，三分之二的人用 IE 浏览网页，现在只有不到一半。

那么 IE 为什么一直跌呢？首先，在过去五年左右的时间里，它并不是一个好的浏览器。微软似乎终于[试图用 IE9](https://web.archive.org/web/20221006112953/https://beta.techcrunch.com/2010/09/15/a-tour-of-ie9-with-internet-explorers-senior-director-microsoft/) 来改变这一点——但这还没有正式发布(它[刚刚测试过](https://web.archive.org/web/20221006112953/https://beta.techcrunch.com/2010/09/15/ie9-download/))。值得注意的是，在 StatCounter 的细分统计中，IE8 有史以来第一次失去了市场份额，这导致 IE 的总份额低于 50%。

由于 IE 一直在下滑，竞争对手火狐、Chrome 和 Safari 一直在崛起。但是，尽管 Firefox 和 Safari 在过去两年都有不同程度的增长，Chrome 显然是赢家。这款浏览器已经从两年前的 1%上升到上个月的 11.5%。这使得它遥遥领先于老款 Safari 浏览器(4.4%)，仅次于 Firefox 浏览器(31.5%)和 IE 浏览器。

在[讨厌 IE6](https://web.archive.org/web/20221006112953/https://beta.techcrunch.com/2010/03/05/ie6-funeral/) 的同时，[继续堕落](https://web.archive.org/web/20221006112953/https://beta.techcrunch.com/2010/06/01/statcounter-browser-share/)。根据数据，现在已经下降到 7.4%。两年前，它独自占据了近 30%的市场份额。

上一次 IE 低于 50%的市场份额似乎是在 20 世纪 90 年代末。当然，在那个时候，微软为了将当时的领导者网景赶出市场，采取了一些粗略的(在某些情况下甚至可能是非法的)手段。成功了。到 2005 年中期，微软已经占据了超过 90%的市场份额。

如果 TechCrunch 的更懂技术的观众是未来趋势的任何迹象，微软将需要 IE9 开箱即用，否则他们就有麻烦了。就访问 TechCrunch 的流量而言，IE 目前是排名第*四*的浏览器。火狐、Chrome、Safari 都毁了它。是的，值得注意的是， [Chrome 将很快成为王者](https://web.archive.org/web/20221006112953/https://beta.techcrunch.com/2010/09/02/chrome-firefox-techcrunc/)。
