# Moonfruit 与黑仔脸书整合，打造店铺形象

到目前为止，Moonfruit 已经被认为是一个很好很容易的网站建设者。尽管他们是这个游戏的长期玩家(这家公司已经 12 岁了)，但他们在将产品迭代到其他平台方面做得非常好，筹集了[可观的资金](https://web.archive.org/web/20230321154633/http://eu.beta.techcrunch.com/2010/08/24/moonfruit-secures-2-25m-funding-to-accelerate-internationally/)，让一些重要的名字参与进来，一些(如 500 家初创公司)来自硅谷，并看到他们的用户群最近受到了一些严重的牵引。但是今天，他们进入了一个新的领域，在我看来，这最终会使他们成为拥有我称之为明天的“[销售者](https://web.archive.org/web/20230321154633/http://trendwatching.com/trends/sellsumers/)”的“商店身份”的玩家。
