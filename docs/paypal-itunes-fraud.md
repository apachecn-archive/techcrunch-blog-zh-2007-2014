# 欺诈者通过 iTunes TechCrunch 榨干 PayPal 账户

> 原文：<https://web.archive.org/web/https://techcrunch.com/2010/08/23/paypal-itunes-fraud/>

![](img/bead413eeefe12692bc7782ac84117c0.png)

今早有报道称，与贝宝相关的 iTunes 账户存在重大安全漏洞。至少有一群骗子已经找到了通过贝宝向 iTunes 账户收取数千美元的方法。一位目标客户告诉我们，“我的账户被扣了 4700 多美元。我给 PayPal 的安全部门打了电话，被告知大量 iTunes 商店账户遭到了破坏。”他的电子邮件里有近 50 张 PayPall 的收据，每张 99.99 美元。(**更新**:他们是“CastleCraft，龙晶(10000 包)，卖家:自由诗公司”)。他能够在他的银行向贝宝支付资金之前发现它。

但是其他人就没那么幸运了。在推特和脸书上快速搜索一下就能发现这个问题并不是孤立的。乔伊·布鲁斯在推特[上哀叹](https://web.archive.org/web/20230214001039/https://twitter.com/Joey_Bruce/status/21913079830):

> 有人黑了我的 iTunes/paypal 账户，取走了我银行账户里的所有东西。生活就是在我失意时踢我的蛋蛋。

搜索脸书的公共状态更新，发现更多的人有同样的问题:

> 该死…多糟糕的一天！有人入侵了我的 itunes 账户，买了一大堆下载，还清空了我的 paypal 账户。。。贝宝非常合作，但几乎没有办法获得 itunes。我打电话给贝宝，他们向我保证，他们已经联系了 itunes，这将是照顾我的利益。
> 
> 所以苹果/itunes 出现了安全漏洞&有人通过我的 paypal 账户购买了价值超过 500 美元的音乐。这正是我在圣地亚哥想要的东西。厉害！！！
> 
> 每个人都密切关注你的 itunes 账户。我刚刚被黑了价值近 1000.00 美元的软件、视频和音乐。希望贝宝能全部退款。。。。这发生在过去的几个小时内。一笔又一笔的交易。

至少 PayPal 意识到了这个问题，但是问题似乎出在 iTunes 这边。

**更新**:苹果现在处于旋转模式，几份后续报告指责用户成为普通网络钓鱼骗局的主题。虽然可能是这样(到目前为止，这些事件似乎并不普遍)，但仍有一些事情不合情理。根据上述目标客户的说法，在他与 PayPal 的讨论中，显然 PayPal 认为问题出在 iTunes 上。PayPal 和 iTunes 之间有直接的计费关系，不知何故这种关系被利用了。如果仅仅是他的贝宝账户被入侵，那么一个聪明的骗子会从各种来源传播虚假指控，而不仅仅是 iTunes。其他报告的事件似乎遵循同样的模式。

此外，他还收到了 46 张来自苹果公司的收据，注明了相同商品的相同费用(见我在下面添加的截图)。我看了这些，并将标题信息与合法的苹果收据进行了比较，它们似乎是相同的。所以肯定是通过 iTunes。

也就是说，有许多 iTunes 账户与 PayPal 相关联，这些账户可能会受到多种方式的攻击。这可能只是简单的密码被盗。苹果公司肯定没有任何迹象表明有什么异常。如果同样的事情发生在你身上，请在评论中分享你的经历。

以下是 Paypal 发言人的声明:

> 我可以确认 PayPal 系统本身没有受到损害，我们没有看到任何受此问题影响的实际 PayPal 帐户的欺诈或未经授权的登录。通过 PayPal 发送的未授权费用正在报销。

来自苹果公司:

> 我们一直致力于增强 iTunes 用户的帐户安全性。如果您的信用卡或 iTunes 密码被盗，并在 iTunes 上使用，您应该联系您的金融机构，就任何未经授权的购买进行退款，并确保立即更改您的 iTunes 密码。关于保护你账户安全的提示，请访问 www.apple.com/support/itunes。

**更新 2** :苹果发言人回应道:“我可以告诉你 iTunes 没有被黑。我们的服务器没有受到威胁。”
