# 京瓷 KR2 路由器现已发货

当我在原来的工作单位拿到 PCMCIA 卡时，我已经用了京瓷的 KR1 路由器好几个月了。这是一个像样的路由器，基本上是唯一一个可以工作的，所以我当时没有太多的选择。直到现在我才意识到 KR2 的存在。既然我升级到了 MBP，我就不用担心买路由器的问题了，但我知道你们很多经常出差的人可能会用到。

KR2 的零售价为 250 美元，并增加了对 USB 和 ExpressCards 的支持。为了吸引新顾客，他们降低了 [15%](https://web.archive.org/web/20221007030409/http://store.kyocera-wireless.com/registration_home.aspx) 的价格。
