# 已证实:谷歌收购了 ether pad TechCrunch 的制造商 AppJet

> 原文：<https://web.archive.org/web/https://techcrunch.com/2009/12/04/google-acquires-etherpad/amp/>

 <amp-img src="https://web.archive.org/web/20230213151648im_/https://techcrunch.com/wp-content/uploads/2009/12/appjetgoogle.png" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img> 没多久。不到半小时前，我们[爆料](https://web.archive.org/web/20230213151648/https://techcrunch.com/2009/12/04/google-acquires-appjet-etherpad/)谷歌正在收购强大的实时文档编辑器 [EtherPad](https://web.archive.org/web/20230213151648/http://www.etherpad.com/) 背后的初创公司 [AppJet](https://web.archive.org/web/20230213151648/http://www.appjet.com/) 。官方的 EtherPad [博客](https://web.archive.org/web/20230213151648/http://etherpad.com/ep/blog/posts/google-acquires-appjet)刚刚更新确认了这个消息。

正如我们之前报道的，我们听说收购价格在八位数左右。EtherPad 的帖子还说，他们将加入总部位于澳大利亚的谷歌 Wave 团队。首席执行官 Aaron Iba、总裁兼首席科学家 David Greenspan、首席技术官 J.D. Zamfirescu 和首席运营官 Daniel Clemens 都将前往。

如果你是一个 EtherPad 用户，你会想看看该公司的博客[帖子](https://web.archive.org/web/20230213151648/http://etherpad.com/ep/blog/)中关于导出数据的说明。这项服务将持续到 2010 年 3 月，但不能创建新的免费公共平台，该公司也不允许新客户注册。
