# NTT DoCoMo 交易为数百万日本 Android 用户带来 FlyScreen

![](img/111cf4b893eec4978a37e1905c5f4d16.png)

优秀的 fly screen(T4)移动应用程序的制造商[celllogic](https://web.archive.org/web/20230204120514/http://www.crunchbase.com/company/cellogic)的一笔大(营销和分销)交易:他们与日本大型运营商 [NTT DoCoMo](https://web.archive.org/web/20230204120514/http://www.crunchbase.com/company/ntt-docomo) 合作。

根据合作协议的条款，NTT DoCoMo 和一家名为 [Brilliant](https://web.archive.org/web/20230204120514/http://www.brilliantservice.co.jp/company/) 的日本本地定位服务公司将把 FlyScreen 推向日本市场，首先带给数百万 Android 手机用户。

FlyScreen 本质上是一个动态的、交互式的、基于小部件的替代物，取代了 Android 和 Symbian 手机的标准锁屏。

NTT DoCoMo 与一家名为 Guidog 的现有 LBS 服务提供商合作，Guidog 的服务已经与 FlyScreen 应用程序集成在一起，并将以该名称销售该应用程序[。](https://web.archive.org/web/20230204120514/https://market.android.com/details?id=com.FlyScreen.Docomo)

Guidog 是一个一站式 LBS 服务，为移动用户提供天气信息、当地优惠券、火车时刻表、新闻、地图等。由于锁屏取代了 Cellogic 技术支持的小工具，用户无需解锁手机屏幕就可以访问该应用程序，从而减少了加载时间。
