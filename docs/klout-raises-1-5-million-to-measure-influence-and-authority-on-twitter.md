# Klout 筹集了 150 万美元来衡量 Twitter 上的影响力和权威

![](img/307630c7e3ad4d2d86d54084713b51fb.png) [Klout](https://web.archive.org/web/20221207201941/http://klout.com/) ，一家[在 Twitter 上衡量](https://web.archive.org/web/20221207201941/https://beta.techcrunch.com/2009/11/17/klout-influence-twitter-list-authority/)影响力的初创公司，已经从许多投资者那里获得了 150 万美元的首轮融资，其中包括[梅菲尔德基金的艾伦·摩根](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/person/allen-morgan)，ff 资产管理公司，鲍比·亚兹达尼，[Lucid Ventures 的诺瓦·斯皮瓦克](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/person/nova-spivack)，[泽尔科瓦风险投资](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/financial-organization/zelkova-ventures)，[佩奇·克雷格](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/person/paige-craig)，[汤姆·麦金纳尼](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/person/tom-mcinerney)，[迈克尔·亚冯迪特](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/person/michael-yavonditte)

Klout 最近[发布了一个新的 API，](https://web.archive.org/web/20221207201941/https://beta.techcrunch.com/2010/04/13/twitter-influence-authority-klout-releases-new-version-of-api/)通过复杂的排名算法和内容的语义分析来评估 Twitter 用户的行为，以衡量个人和主题在网络上的影响力。自 2008 年公司成立以来，Klout 已经注册了超过 250，000 名用户和超过 300 个 API 合作伙伴，包括 [CoTweet、](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/company/cotweet) [HootSuite、](https://web.archive.org/web/20221207201941/http://www.crunchbase.com/company/hootsuite)和[新推出的 Tweetup、](https://web.archive.org/web/20221207201941/https://beta.techcrunch.com/2010/04/11/twitter-idealab-tweetup-adsense-bill-gross/)，每天处理数百万个 API 调用。

新的资金将用于扩大产品开发规模，并推出一款更加以消费者为中心的新网络应用。有了大量的 API 合作伙伴，显然 Twitter 影响力初创公司正在获得牵引力，尤其是在第三方 Twitter 客户端。Twitter 总有可能会开发出自己可以使用的权威衡量标准，但就目前而言，Klout 似乎在衡量 Twitter 影响力方面处于领先地位。
