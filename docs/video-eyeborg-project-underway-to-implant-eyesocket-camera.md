# 视频:植入眼袋摄像头的“眼睛机器人”项目正在进行中

【注意:视频略有图文。此外，邮政服务警告]

似乎现在每个人都想换掉他们的眼睛。然而，区分眼内摄像机和视觉替代系统是很重要的。[这位加拿大电影制作人](https://web.archive.org/web/20221207214530/http://www.eyeborgblog.com/)和[这位女士](https://web.archive.org/web/20221207214530/http://www.crunchgear.com/2008/11/17/woman-wants-to-replace-her-glass-eye-with-a-webcam/)，正试图制作一个相当于戴在眼睛上的相机，而其他人(在医疗领域)正试图[创造一个与视网膜](https://web.archive.org/web/20221207214530/http://www.crunchgear.com/2008/03/09/yes-that-is-a-bionic-eye/)或大脑视觉系统的接口。猜猜哪个更难？嗯，老实说，它们都很难，尤其是当你没有预算的时候。

到目前为止，该项目取得了喜忧参半的成功(正如你在视频中看到的)，但在这一领域所做的每一点工作都有助于整体。一个项目的绊脚石是下一个项目的垫脚石，所以也许用不了多久，你就会为了好玩而换掉你的眼睛。
