# Next 新网络收购 BarelyPolitical.com

[![nexnewnetworks.png](img/f636cae958f900a59dbd2e8afb143843.png)](https://web.archive.org/web/20221006234548/http://www.nextnewnetworks.com/) 微型电视网络提供商 [Next New Networks](https://web.archive.org/web/20221006234548/http://www.nextnewnetworks.com/) 已经收购了[BarelyPolitical.com](https://web.archive.org/web/20221006234548/http://www.barelypolitical.com/)，这是奥巴马女孩视频背后的网站。

此次收购是 Next New Networks 首次涉足政治，现在几乎没有政治将与 Next New Networks 现有内容一起提供，包括频道运营商。交易条款没有披露。

虽然奥巴马女孩视频系列(不止一个)可能是一种习惯，但它的病毒式成功非常显著，原始视频已经被观看了 1 亿次。

根据协议，BarelyPolitical 将定期推出多个节目，包括其视频博客和记者的频繁报道。BarelyPolitical 与 Next New Networks 合作推出的第一个项目是“我喜欢一个男孩”[，这是一个与美国伊战阿富汗退伍军人协会共同制作的音乐视频，在第一周就获得了 150 多万次在线视频观看，并在各种媒体上播放，包括美国有线电视新闻网(CNN)的《美国早晨》、福克斯新闻频道和美国广播公司的《早安美国》。](https://web.archive.org/web/20221006234548/http://www.youtube.com/watch?v=XFnzbjftMwc)
