# Flattr 向公众开放，现在任何人都可以用真金白银“喜欢”一个网站

由前海盗湾合伙人创办的小额支付创业公司[Flattr](https://web.archive.org/web/20230125155046/http://eu.beta.techcrunch.com/2010/07/06/is-flattr-the-new-facebook-like-but-this-time-with-real-money/)今天向公众开放。你不再需要一个邀请就可以把 Flattr 按钮作为发布者添加到你的网站上，或者用真金白银支持你访问的网站。

Flattr 的微支付商业模式是基于人们给他们喜欢的内容、Digg 或者脸书喜欢的按钮小费的想法——但这次是用真金白银。用户可以通过建立一个 Flattr 账户来“Flattr”内容，这个账户设定了一个月费——最低的€新协议——他们愿意为任何种类的在线内容付费。当用户找到他们喜欢的带有 Flattr 按钮的东西时，他们可以点击该按钮来“奖励”内容提供商。到了月底，用户的月租费在他们“奉承”的内容持有者之间平分。

然而，Flattr 并不是唯一一家致力于这种微支付模式的初创公司。Rewrd 也提供类似的服务，尽管最终两个网站都在争夺出版商和网络消费者的注意力，很可能最终只有一个赢家。在这一点上， [Flattr 是一个良好的开端](https://web.archive.org/web/20230125155046/http://eu.beta.techcrunch.com/2010/08/02/wikileaks-will-fund-itself-via-flattr-pirate-bay-founders-startup/)。

**更新:**随着向公众开放，Flattr 增加了一些新功能，如 Twitter 和脸书集成。更多细节[通过他们的官方博客](https://web.archive.org/web/20230125155046/http://blog.flattr.com/2010/08/open-beta/)。

**更新 2:** 在这个领域运营的另一家公司是 [Kachingle](https://web.archive.org/web/20230125155046/http://www.kachingle.com/) 。
