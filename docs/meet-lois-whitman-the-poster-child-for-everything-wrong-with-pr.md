# 见见露易丝·惠特曼，公关失败的典型代表

会见来自纽约公关公司 [HWH](https://web.archive.org/web/20230212044140/http://www.hwhpr.com/) 的露易丝·惠特曼-赫斯。洛伊斯是你见过的最令人讨厌的公关人员之一，也是这个行业所有问题的典型代表。考虑一下我正在进行的关于行业的[讨论的第二部分。](https://web.archive.org/web/20230212044140/https://techcrunch.com/2008/12/17/death-to-the-embargo/)

露易丝以让人痛苦为乐，她的专长是发垃圾邮件。她第一次在 CrunchGear 引起我们的注意，该网站[报道了她与](https://web.archive.org/web/20230212044140/http://www.crunchgear.com/2008/12/17/pr-and-the-fine-art-of-not-being-crazy/) [PhoneScoop](https://web.archive.org/web/20230212044140/http://www.phonescoop.com/) 的交流。

PhoneScoop 的人注册了 CES，没有意识到他们的联系信息会被任何想要它的公关公司爆料。当他们收到 Lois 发来的一封非常跑题的推销邮件(关于三星电视)时，他们回邮件说*“请把我从你的名单中删除。我的出版物不包括这些类型的产品。我没有注册接收这个类别的信息，也没有任何接近。根据消费电子展的指导方针，我不应该收到这封邮件，这已经危险地接近垃圾邮件了。这反映了你公司的不良形象。谢谢大家。”*

坏主意。他们收到了露易丝发来的两条信息:

> CES 出版了一份媒体名单。你是几千人中的一个。出于各种原因，每个人都可以接触到那份名单。它是公开出版的。作为一家公关公司，我们利用这份名单来为展台设备招揽媒体。我希望你能理解。如果你没有，我给你介绍一下“删除”按钮。或者以后不要报名 CES 的新闻人。此外，不要威胁我的公司。我不需要你告诉我什么是对的，什么是错的。我从事消费电子行业已经 42 年了。

和

> 我见过像你这样令人讨厌的人消失得比雪球在雨中上山还快。我在等一个道歉。也许我们可以在消费电子展上见面拥抱一下或者喝一杯。又及:我刚刚访问了你的网站。我很难称你的博客为出版物，但是，你确实有非常有趣的内容，我们有很多客户，你想知道更多，以帮助你的努力。打电话给我。

耻辱堂的介绍到此为止。但是快速浏览 Lois 的博客[会发现她愿意不停地发垃圾邮件，直到她得到她想要的:](https://web.archive.org/web/20230212044140/http://loiswhitman.blogspot.com/)

![](img/a7363d2c577b16c7d27ccbadf2690d68.png)

九月，不良广告博客(一个新宠)也指责 Lois 发送垃圾信息，但这种行为仍在继续。我不明白的是，为什么她的[客户](https://web.archive.org/web/20230212044140/http://www.hwhpr.com/clients.html)继续资助这项活动。你应该雇用这些人，因为他们与媒体有关系，而不是因为他们知道如何有效地发送垃圾邮件和辱骂。

是的，我要继续揭露这些人和他们的荒谬行为。
