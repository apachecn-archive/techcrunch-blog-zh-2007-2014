# Placecast 为零售商智能手机应用程序带来基于位置的购物提醒

Placecast 的 [ShopAlerts 服务](https://web.archive.org/web/20230204215726/https://techcrunch.com/2010/02/25/placecast-debuts-location-based-mobile-marketing-technology-shopalerts/)通过手机向消费者提供零售商和商家的优惠和折扣，今天宣布了新功能——与移动智能手机应用程序集成。

正如我们过去所写的，ShopAlerts 是一个白标地理围栏平台，面向希望通过手机向客户提供地理触发优惠的企业。消费者可以选择以多种方式接收短信——在商店、网上、通过短信、手机网站或脸书。地理围栏是围绕物理位置设置的虚拟区域；当选择加入的消费者在地理围栏区域内时，她会收到来自企业的消息，包括折扣、销售通知、商店方向或其他内容。

通过这项新功能，部署移动应用程序的零售商可以允许消费者从零售商最近的商店或其他相关位置接收基于位置的信息，而无需打开应用程序。就像 ShopAlerts 程序可以在任何手机上工作一样，这项新功能可以在任何智能手机平台上工作。

在零售商集成了 ShopAlerts 之后，客户(选择接收警报的人)将在地理围栏区域内收到基于位置的推送通知。零售商还可以利用消费者表达的偏好，如忠诚度或过去的购买数据，来增加信息或交易的相关性。还可以考虑一天中的时间和天气来增加相关性，当附近商店有特定产品时，品牌还可以启用基于位置的通知。

Placecast 表示，它将直接与零售商合作，将商店警报集成到他们的应用程序中，帮助他们设计程序和创建信息策略，选择地理围栏的数量和大小，并激活位置。

与零售商智能手机应用的整合对 Placecast 来说意义重大。随着线上到线下商务的增长，Placecast 为零售商提供了另一种瞄准消费者并提供相关交易和信息的方式。目标是将消费者带入商店，通过智能手机应用程序发送的信息可以帮助实现这一目标。
