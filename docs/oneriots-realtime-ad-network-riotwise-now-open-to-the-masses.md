# OneRiot 的实时广告网络 RiotWise 现在向大众开放

![](img/7b12a5327e70d824bbc72fdd197bb86e.png)

[OneRiot 的](https://web.archive.org/web/20221231100638/http://www.oneriot.com/)最近[凭借](https://web.archive.org/web/20221231100638/http://techcrunch.com/2009/09/30/oneriot-believes-it-has-a-way-to-monetize-the-realtime-web-with-riotwise/) [RiotWise、](https://web.archive.org/web/20221231100638/http://www.crunchbase.com/product/riotwise)一种将内容放在他们的[实时反馈中突出位置的广告形式，冒险进入](https://web.archive.org/web/20221231100638/http://techcrunch.com/2009/05/12/twitter-is-talking-real-time-link-search-but-oneriot-is-launching-it-today/)广告界。几周前，实时搜索引擎[推出了](https://web.archive.org/web/20221231100638/http://techcrunch.com/2009/12/22/oneriot-monetizes-whats-hot-on-the-web-with-realtime-trending-ads/)一个 RiotWise 趋势广告的试点项目，这是一系列与社交网络上出现的趋势话题相对应的广告。今天，在与精选的 iPhone 应用程序、Twitter 客户端和搜索引擎(包括 UberTwitter 和 Digsby)合作后，OneRiot 的实时广告网络 RiotWise 现已对所有开发者开放。

RiotWise 的广告由类似于谷歌的内容组成，通过搜索关键词提供。但与谷歌不同的是，广告商不会对关键词进行竞价。相反，内容制作者与 OneRiot 达成协议，将他们的内容放在搜索引擎中他们的实时提要中的一个强调(但明确标记)的位置。最终，OneRiot 的目标是帮助内容制作商提高点击率，向其发送旨在寻找特定内容的高目标读者。

RiotWise 的趋势广告会将趋势主题与显示广告相匹配，这些显示广告与 Twitter 客户端或 iPhone 应用程序等应用程序中的相同主题高度相关。OneRiot 表示，与搜索引擎结果旁边显示的广告类似，实时相关性会提高广告的点击率。该系统由 OneRiot 的实时搜索技术和 PulseRank 关联算法支持。OneRiot 帮助开发者将 Twitter 应用、IM 客户端和 iPhone 应用等社交网络应用货币化，并与[应用开发者分享广告收入。](https://web.archive.org/web/20221231100638/http://oneriotdevelopernetwork.com/)

现在 OneRiot 报告说，RiotWise 广告的表现是实时网络应用(Twitter 客户端、iPhone 应用等)中广告平均行业标准点击率(CTR)的 3 到 4 倍。).开发者还可以灵活地在信息流中显示 RiotWise 广告或与热门话题相关的广告。

当然，通过这些实时广告，OneRiot 面临着出现不相关或垃圾内容的风险。但作为一个实时搜索引擎，OneRiot 在防止垃圾邮件方面投入了大量资金，并不断整理数百万条内容，以确定哪些是相关的，哪些是不相关的。只要 OneRiot 生产高质量的广告内容，这种新的广告形式对出版商和开发商来说都是一种可行且有利可图的盈利工具。
