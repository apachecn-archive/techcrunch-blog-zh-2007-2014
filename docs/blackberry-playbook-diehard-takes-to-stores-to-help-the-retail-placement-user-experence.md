# 黑莓 PlayBook Diehard 带去商店帮助零售，用户体验

黑莓行动手册有一些忠实的粉丝。当然，人数不多，但他们肯定在那里。一个这样的粉丝带着一个关于如何支持 RIM 和 PlayBook 的颇为新奇的想法来到 CrackBerry.com 论坛:零售福音派，忍者风格。

他的计划至少在一定程度上是为了帮助 PlayBook，因为他会偷偷把一张展示一些小技巧、小窍门和基本功能的海报带到商店里。他认为，这将有助于通过向消费者展示设备的基本功能来增加 PlayBook 的销售。因为，你知道，高质量的产品需要基本的说明。

撇开笑话不谈，这家伙说得有道理。零售演示通常是由工资低、欣赏不足的员工组装而成，他们只是想完成工作。公司代表应该来检查这些展示品，但作为一个在零售领域花了大量时间的人，我可以告诉你这种情况很少发生。就在几周前，我去办公室马克斯一些文件。(顺便问一下，纸什么时候变得这么贵了？)他们的行动手册甚至没有电源，所以一张小小的塑料卡根本没有用。

这个人计划的重点是引起人们对行动手册的注意，这是 RIM 似乎不愿意做的事情。iPad，甚至惠普 TouchPad，往往有花哨的带浮华海报的终端显示器，PlayBook 经常和随机的 Android 平板电脑一起被塞进去。这是 RIM 没有为 PlayBook 找到合适的零售空间的错，它的粉丝们认为他们有责任纠正这家价值数十亿美元的公司的疏忽。坏消息不断。【经由 [TheLoop](https://web.archive.org/web/20230203100623/http://www.loopinsight.com/2011/08/04/rim-customers-try-to-save-the-playbook/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+loopinsight%2FKqJb+%28The+Loop%29) ]
