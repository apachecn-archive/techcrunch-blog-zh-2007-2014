微软仍然统治着台式机:2011年42%的机器将运行Windows 7

Windows 7现在是最流行的桌面操作系统，Gartner估计目前42%的电脑将运行该操作系统，而94%的新电脑将运行Win7。

相比之下，OS X获得了4%的份额，而Linux牢牢占据了2%的份额。甚至IT部门也开始在他们的桌面上大规模部署Win7，这一举动将这个相当新的操作系统推到了猫鸟的位子上。然而，随着IT部门转向托管计算和虚拟化，Gartner预计这将是最后一次在商用电脑上安装独立的操作系统映像。

然而，并非一切都是阳光和玫瑰:

Gartner研究总监Annette Jump表示:“2010年和2011年IT预算的稳步增长有助于加快Windows 7在美国和亚太地区企业市场的部署，从2010年第4季度开始，Windows 7在这些市场开始大规模迁移。“但是，西欧的经济不确定性、中东和非洲(MEA)部分国家的政治不稳定以及日本在2011年3月地震和海啸后的经济放缓可能会导致Windows 7在这些地区的部署略显迟缓。”

如果你是订阅者，你可以在这里查看报告，但是一般来说，即使面对OS X和手机的流行，Windows仍然很强大。
