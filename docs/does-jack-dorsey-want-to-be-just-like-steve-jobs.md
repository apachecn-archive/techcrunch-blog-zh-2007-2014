# 为什么杰克·多西想成为史蒂夫·乔布斯那样的人？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/08/16/does-jack-dorsey-want-to-be-just-like-steve-jobs/>

世界上没有一个首席执行官不在某种程度上努力模仿已故的史蒂夫·乔布斯。他是一个有远见的人，把苹果从一个相对较低的点带到了世界历史上最成功的公司之一。但是试图成为他，或者像他一样，将是徒劳的努力。

一位爆笑的消息人士让我们注意到，Twitter 联合创始人、Square 创始人兼首席执行官杰克·多西可能正在尝试后者。事实上，告密者很有风度地将所有证据打包在一个可爱的小 [Tumblr](https://web.archive.org/web/20221006080921/http://stevejobsspirit.tumblr.com/post/29544010983/thoughts-on-jack) 里。

### 证据一:关于被解雇

1987 年，乔布斯被苹果解雇，之后他继续在 NeXT 和皮克斯工作，后来又回到苹果担任首席执行官。但当他被要求离开他帮助建立的公司时，他告诉《花花公子》以下内容:

> 我觉得有人在我肚子上打了一拳。

多尔西经历了类似的情况，他被任命为 Twitter 的董事长，但不再是一名员工。他告诉《名利场》:

> 就像被人在肚子上打了一拳。

### 证据 B:“这是前所未有的”

史蒂夫·乔布斯是许多前所未有的变革的一部分。iPod 改变了音乐产业。iPad 改变了计算机行业。iPhone？iPhone 改变了一切。

但是杰克·多西凭自己的能力是一个先锋。他建立了一个巨大的社交网络，改变了新闻传播的方式。他完全颠覆了信用卡行业。因此，两人对自己的成就有相同的感受是有道理的。但显然，他们也有相同的确切措辞来描述它。

### 证据 C:“为我们*没有*做的事情而自豪”

2008 年，乔布斯解释说，苹果比其他科技公司更成功的部分原因是，他们成功地在如此少的设备上获得了如此巨大的利润。他对 [CNN 的钱](https://web.archive.org/web/20221006080921/http://money.cnn.com/galleries/2008/fortune/0803/gallery.jobsqna.fortune/6.html)说:

> 事实上，我为许多我们没有做过的事情和我们已经做过的事情感到骄傲。最明显的例子是，我们被迫做了多年的 PDA，有一天我意识到 90%使用 PDA 的人只在路上从它那里获取信息。他们不把信息放进去。很快手机将会做到这一点，所以 PDA 市场将会缩小到目前规模的一小部分，这将是不可持续的。所以我们决定不深究。如果我们当时投入进去，我们就不会有资源来做 iPod 了。我们可能不会看到它的到来。
