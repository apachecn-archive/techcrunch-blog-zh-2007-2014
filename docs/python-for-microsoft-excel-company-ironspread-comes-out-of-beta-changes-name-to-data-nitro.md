# python For Microsoft Excel Company iron spread 退出测试版，将名称改为 DataNitro

y Combinator~~Winter~~2012 年夏季毕业生 [Data Nitro](https://web.archive.org/web/20221206120004/https://datanitro.com/) (原名 IronSpread)有一个简单的提议:它能让你在 Microsoft Excel 中使用流行的编程语言 Python。该插件对个人非商业用户是免费的，企业将为其特权付费。到目前为止，它只适用于 Windows 的 Excel 2007 和 2010。该公司没有支持 OSX 的计划。

![datanitro screenshot](img/bb2852f564cd76563cfb34140060614e.png "datanitro screenshot")

尽管高级用户长期以来一直能够使用微软的 Visual Basic for Applications (VBA)来编写 Excel 脚本，并且微软也有望在 Office 中支持 JavaScript，Python 是 Excel jock 工具箱的强大补充。根据 T2 的多项指标来看，Python 对于程序员来说就算不是顶级语言，也是顶级语言。它干净、直接的语法使它成为初学者和[编程入门课程](https://web.archive.org/web/20221206120004/http://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-00sc-introduction-to-computer-science-and-programming-spring-2011/)的热门选择，但它在许多地方的产品中也很受欢迎[，包括 Google](https://web.archive.org/web/20221206120004/http://techreport.com/discussions.x/16713) 和 NASA 。这种语言有很多令人喜欢的地方，花在学习 Python for Excel 脚本上的努力可以很好地移植到其他应用程序中。

好消息是:你仍然可以使用你所有的旧 VBA 脚本，Data Nitro 不会阻止它们工作。另外，您可以使用现有的 Python 库。坏消息是:你发送电子表格的任何人都需要安装插件才能运行 Python 脚本。

在 Excel 中处理 Python 代码的其他工具包括 [Pyvot](https://web.archive.org/web/20221206120004/http://pytools.codeplex.com/wikipage?title=Pyvot) 和 [Pyxll](https://web.archive.org/web/20221206120004/http://pyxll.com/) (链接见[黑客新闻社区](https://web.archive.org/web/20221206120004/http://news.ycombinator.com/item?id=4085052))。
