# OpenDesk.cc 就像开源狂热者的宜家

家具可能是大多数开源支持者最不想要的东西，但现在不必了。OpenDesk 是一个免费的开源家具系列，你可以自己制作，也可以从制造商那里用数控机床订购。家具不仅便宜——如果你有木头和五金件的话是免费的——而且看起来很酷。

OpenDesk 由[乔尼](https://web.archive.org/web/20221006175004/https://twitter.com/jonisteiner) & [大卫·斯坦纳](https://web.archive.org/web/20221006175004/http://www.davidsteiner.co.uk/)和 Development 00 创建，是第一个与 [FabHub](https://web.archive.org/web/20221006175004/https://www.fabhub.io/) 兼容的项目，该网站允许你搜索你所在地区的制造商。你也可以找一个有商店机器人的人来帮你切割碎片，然后把它们放在你的住所或办公室。

设计[可从下载](https://web.archive.org/web/20221006175004/https://www.opendesk.cc/designs/desk/get-it/download)。dwg 和。dxf 格式，机器可读的模型，允许你驱动一个机器人数控系统。如果你想尝试手工制作硬件，你也可以下载 PDF 格式的计划。他们有许多物品，包括几张凉爽的桌子和一个凳子。

虽然宜家在廉价家具领域可能仍然安全，但很明显，DIY 家具正变得越来越简单。再加上一个机器人金属切割机和一些 3D 打印机，你可以在几天内建造一整套办公设备——包括笔筒。
