# 海盗湾的复制品已经出现了

![tpbcopy](img/9f22c9d4678913a101e27119b2554333.png "tpbcopy")

这向你展示了如果海盗湾变得“合法”，有人会多快取代它。你可能知道，有人下载了 TPB 服务器上的每一个种子文件；那个种子文件被上传回 TPB，在那里它现在从数百个来源被播种。新鲜的是，有人拿走了 TPB 的副本，然后[把它上传到*另一个*网站](https://web.archive.org/web/20221006023000/http://www.btarena.net/)，这实质上是复制了 TPB。

所有的 873，671 种子现在都在 BTArena 中存活。现在还没有办法搜索索引(这对服务器来说压力太大了)，但是您仍然可以很好地浏览目录。

请记住，当前副本仅用于备份。如果 TPB 在接下来的 24 小时内爆炸，那么是的，这就是副本可以派上用场的地方。也就是说，它不是 TPB2 的意思。
