# FunChat 将文本、聊天、恶作剧、虚拟商品和社交游戏集于一个免费应用 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2011/06/16/funchat-combines-text-chat-pranks-virtual-goods-and-social-gaming-all-in-one-free-app/>

[![](img/49fb4a89dd997b1a5966bfc39fc22f1a.png "Screen shot 2011-06-16 at 12.46.07 AM")](https://web.archive.org/web/20221007085324/https://beta.techcrunch.com/wp-content/uploads/2011/06/screen-shot-2011-06-16-at-12-46-07-am.png)[【fun mobility】](https://web.archive.org/web/20221007085324/http://www.funmobility.com/),[FunMail](https://web.archive.org/web/20221007085324/http://funmail.com/)(一款图片信息应用，将你的文本与适当(有趣)的图片有机地配对) [FunTones](https://web.archive.org/web/20221007085324/http://itunes.apple.com/us/app/funtones-2-000-funny-ringtones/id293067873?mt=8) (一个有趣铃声的大集合)和 [FunMe](https://web.archive.org/web/20221007085324/http://www.funmobility.com/funme/) (一套供消费者创建、分享和发布他们自己的用户生成内容的工具)的制造商，显然在努力传递一种特定的品牌信息。我觉得跟“好玩”有关。

今天，FunMobility 为它的乐趣动物园引入了一个新的应用程序，名为 [FunChat](https://web.archive.org/web/20221007085324/http://www.funchat.com/) ，它将聊天、多人 HTML5 游戏、虚拟货币、奖励和成就融合到 iPhone、iPod Touch 和 Android 的实时用户体验中。

虽然这些都是很棒的功能，但 FunChat 的致命一击(至少对我来说)是它包含了第一个移动恶作剧引擎。没错，FunChat 用户可以将音频恶作剧推送到智能手机上，例如狼哨、布朗克斯欢呼和愤怒的猪，即使手机处于待机模式或你没有运行 FunChat。伙计，阿什顿·库彻怎么没先想到这个？我以为他是一个技术布道者。

FunChat 通过“Chattitude”提供可定制的聊天，该功能使用户能够使用新动画头像、聊天气泡和一系列虚拟商品来个性化聊天体验，满足你能想到的每种个性、情绪或场合。用户可以在“公共聚会”或私下与朋友一起利用这些定制。

该应用的多人 HTML5 游戏室包括休闲游戏，如 concentration、嘉年华射鸭、match-3、赛车等，并针对 iPhone 和 Android 设备上的 HTML5 进行了优化。在 FunChat 应用程序中，用户可以争夺 FunGold，这是该应用程序的虚拟货币，通过在应用程序上玩游戏和乱搞来赚取，然后可以在应用程序中兑换，以获得头像、聊天气泡和恶作剧等商品和奖品。

这里的关键是，这些游戏室也是聊天室的一部分，所以用户可以夸耀他们在游戏中的成就，诘问对手，挑战他们的朋友进行游戏决斗，或者称赞他们射击了一整群鸭子——所有这些都是实时的。通过游戏化的聊天用户界面、恶作剧和应用内虚拟奖励，FunChat 试图建立一种令人上瘾的社交游戏体验，鼓励大量吸引年轻人的活动和互动。

[![](img/9fa80797d851cc8eada5013ef0f9b143.png "Screen shot 2011-06-16 at 1.50.30 AM")](https://web.archive.org/web/20221007085324/https://beta.techcrunch.com/wp-content/uploads/2011/06/screen-shot-2011-06-16-at-1-50-30-am.png) 显然，随着[智能手机的日益普及](https://web.archive.org/web/20221007085324/http://www.ctia.org/advocacy/research/index.cfm/aid/10323)以及短信和游戏的使用不断升级(美国的手机消费者每月发送 1870 亿条短信)，FunMobility 将短信、聊天和游戏集于一身是明智之举。在移动应用程序中发短信和聊天是一种天生的社交和游戏化体验，许多消息应用程序([正如我最近写的 GOGII】)正在考虑如何最好地融合这些互补的功能。因此，TextChat 相对较早地进入游戏是明智的。](https://web.archive.org/web/20221007085324/https://beta.techcrunch.com/2011/06/03/scaling-scaling-scaled-textplus-turns-two-hits-10-billion-messages-sent-milestone/)

最后，似乎区分这些社交化和游戏化消息平台的是用户对不同 ui 和设计的反应——每个都创造了独特的用户体验——而这一切都是关于什么对你有意义。对我来说，FunChat 看起来很棒。当然，就品牌信息而言，到处散布“乐趣”从来都不是一个坏主意。当然，这也有助于在 iOS 和 Android 上免费使用 FunChat，以及应用内聊天和消息。

哦，我们也欢迎开发者:【FunChat 背后的‘秘方’的一个关键成分在于我们的 HTML5 多人游戏引擎，它可以让高质量的游戏在原生 iOS 和 Android 客户端和浏览器之间同时进行，”fun mobility 首席执行官亚当·拉文(Adam Lavine)说。*“这个专有的游戏引擎是我们可扩展的 FunChat 移动参与平台的一部分，该平台允许开发人员轻松制作开放标准的 HTML5 游戏，这些游戏可以在 iOS 和 Android 平台上运行，为消费者提供了实时共享、发现和一起玩的新方式”。*
