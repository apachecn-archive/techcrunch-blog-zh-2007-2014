# 《创始人故事》本·勒勒谈创业淘金热

克里斯·迪克森(Chris Dixon)结束了与 [Thrillist](https://web.archive.org/web/20230204222531/http://www.crunchbase.com/company/thrillist-com) 联合创始人和[勒勒风险投资](https://web.archive.org/web/20230204222531/http://www.crunchbase.com/financial-organization/lerer-ventures)合伙人[本·勒勒](https://web.archive.org/web/20230204222531/http://www.crunchbase.com/person/ben-lerer)的谈话，勒勒给出了他对当今创业环境的评估。在他看来，这是一个大杂烩。(注:这是在最近的金融危机之前录制的)

“自从我们开始投资以来，我们已经看到更多的资金进入市场。。。因此，作为种子期投资者更难了，因为估值正在上升。”然而，Lerer 继续说“我们的优势是在首轮融资阶段有更多的钱，所以我们更多的公司有机会继续下去。”

迪克森通过转述他从一些投资者那里听到的谈话，对当前的形势进行了评论。他表示:“在当前环境下，困扰他们的不是估值如此之高，而是时间如此紧迫，他们不得不在三天内做出(投资)决定。”狄克逊的意见？“我认为这是双方的错误。”

在下面的视频中，迪克森问勒尔，在投资方面，竞争对手是谁。勒尔回应说，其中很多来自那些对科技投资游戏不熟悉的有财力的人，他们“向它扔钱”并且“不了解生态系统”。他继续说道，“有人正在开一些愚蠢的支票。”
