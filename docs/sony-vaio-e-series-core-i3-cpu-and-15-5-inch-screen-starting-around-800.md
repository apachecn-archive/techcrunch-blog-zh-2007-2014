# 索尼 VAIO E 系列:酷睿 i3 CPU 和 15.5 英寸屏幕，起价约 800 美元

![](img/1764c3034100d155bd5d3379786ca3e5.png "EB_Brown_-_Front_to_Back_copy_lg")

如果你正在寻找一个不需要倾家荡产的普通台式机替代品，你可能会发现索尼的新 VAIO E 系列适合你的需求。起价约为 800 美元，配备英特尔酷睿 i3 CPU、4GB 内存、500GB 硬盘、HDMI 输出、蓝牙和 Windows 7。

这种基本配置将于 3 月初开始发货，但未来的配置将提供酷睿 i5 处理器、蓝光和高达 8GB 的内存。每个型号都有一个快速启动的“web”按钮，可以让你快速进入 WEB 浏览器，而不需要将机器完全启动到 Windows。

可供选择的款式将包括“五种半透明的高光颜色——加勒比绿、彩虹蓝、椰子白、芙蓉粉和熔岩黑——以及三种哑光颜色——枫褐色、银白色和青铜色。在光滑机型的盖子上可以找到微妙的图案，掌托反映了 PC 的颜色，为设备增加了一点亮度。”

你甚至可以用各种颜色的可选外壳装饰全尺寸键盘。Core i3 型号现在可以预购，更多选项(包括 Core i5 芯片)将于 2 月 7 日公布。电池寿命应该在三小时左右，标准电池的系统重量在不到六磅左右。
