# 是时候了:HTC 将智能手机的努力重新聚焦于“英雄”设备

这些天来，HTC 已经变得非常擅长生产一款又一款手机，但它开始感觉每一款新发布的手机都太少太快了。令人欣慰的是，Android 硬件的教父已经醒悟过来，正如《今日移动》所揭示的那样。

据 HTC 英国负责人 Phil Roberson 称，2012 年将会给他们的客户“一些特别的东西”为此，HTC 计划专注于数量较少的“英雄”智能手机，并减少它们在平板电脑市场的份额。

“我们必须重新专注于让我们变得伟大的东西——令人惊叹的硬件和出色的客户体验，”罗伯森说。“2011 年结束时，我们推出的产品比开始时多得多。我们试图做得太多了。”

我自己说得再好不过了。HTC 最近的哲学似乎是在前辈们有机会喘息之前，通过推出新硬件来不断追逐前沿。听说(呃，读到)HTC 的某人也有同样的感觉，这令人耳目一新，因为过多的选择最终会成为消费者的一个大问题。

《T2》The Verge 的 Chris Ziegler 指出，在美国，T-Mobile 的高端 HTC Amaze 4G 在该运营商之前的旗舰 Android 手机上市 120 天后首次亮相。众所周知，技术进步的速度非常快，但在四个月内推出一款新的旗舰设备，可能会很容易对那些为了让自己适应未来而在以前的型号上挥霍的客户造成打击。

处于风口浪尖是很好的，除非受伤的是你的客户。说到这种行为，HTC 并不是唯一一个——三星的 LTE 包装飙升盖过了他们最初的 Galaxy S II 型号，摩托罗拉的 [Droid RAZR Maxx](https://web.archive.org/web/20221207181951/https://beta.techcrunch.com/2012/01/24/299-droid-razr-maxx-to-hit-verizon-shelves-on-january-26/) 用几毫米换取了一些出色的电池寿命，同时让最初的 Droid RAZR 用户感叹他们的运气。尽管如此，当三星的市场饱和计划做得很好的时候，摩托罗拉已经宣布他们打算在 2012 年削减他们的智能手机产品。

与此同时，他们平板电脑计划的披露(或缺乏计划)并不令人震惊。这家台湾公司在 2011 年发布了两款真正的平板电脑，此外还附带了一些运营商专用的变种。唉，平板电脑市场已经变得越来越难啃了，虽然 HTC 不会完全放弃它，但他们会在主要的赚钱机器:智能手机上花费更多的时间和资源。

HTC 最近的财务状况比以前更不稳定，当时他们称移动领域的“竞争加剧”是他们最近季度收益[结束六个季度增长势头](https://web.archive.org/web/20221207181951/https://beta.techcrunch.com/2012/01/06/htcs-growth-spurt-is-done-for-now/)的原因之一。这一新举措是激烈的，但考虑到他们曾经稳固的方法似乎不再那么稳固，这值得一试。随着世界移动通信大会即将召开，我们应该很快就会更好地了解 HTC 的新战略，尽管它是否真的有效完全是另一回事。
