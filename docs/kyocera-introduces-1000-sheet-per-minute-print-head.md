# 京瓷推出每分钟 1000 张的打印头

![kyocerafastestprinter.jpg](img/76375f3c7c2d49493301e344254e01ad.png)

这里有一个有趣的发展:[京瓷](https://web.archive.org/web/20221007040120/http://www.crunchgear.com/tag/kyocera/)正在建造一个新的 KJ4 系列打印头，它可以以每分钟 150 米的速度在 600×600 dpi 下打印，相当于每分钟 1，000 张纸。检查你的盒子中的[当前打印机的](https://web.archive.org/web/20221007040120/http://www.crunchgear.com/tag/printers/)速度，并感到惊讶。

这个头下个月应该会投入印刷。京瓷使用[陶瓷压电](https://web.archive.org/web/20221007040120/http://www.2dayblog.com/blog/2008/03/31/kyocera-annouced-the-world%e2%80%99s-fastest-high-resolution-inkjet-printhead/?lang=en)使这个 4 英寸的头喷出墨水的速度比乌贼节上的乌贼还快。抱歉，最后一个笑话。时间还早。
