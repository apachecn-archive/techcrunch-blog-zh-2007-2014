# Ev 解释了 Twitter 进军移动应用的原因:“否则我们就是在辜负用户”

![](img/842c48cd79abd46c9a81cee898d966c6.png)

上周，当 Twitter 开始填补其产品线中的漏洞(主要是通过移动应用)时，它在开发者中引起了很大的恐慌。具体来说，它推出了自己的[黑莓应用](https://web.archive.org/web/20221208220015/https://beta.techcrunch.com/2010/04/09/twitter-fills-its-first-hole-with-an-official-blackberry-app/)和[收购了 iPhone 上最受欢迎的 Twitter 应用之一 Tweetie](https://web.archive.org/web/20221208220015/https://beta.techcrunch.com/2010/04/09/twitter-acquires-tweetie/) 背后的初创公司。

在 Chirp 今天的[演示中，首席执行官伊万·威廉姆斯试图解释为什么 Twitter 觉得有必要在这一领域与应用开发者竞争。“我们需要以更好的方式应对这些市场，”他告诉与会的开发商。一些人可能会认为这意味着他们的移动应用程序很糟糕，Twitter 将会介入解决问题。但 Ev 的观点更多的是，移动太重要了，Twitter 不能没有自己的官方应用，尤其是对于那些被各种移动应用商店中太多选择所困惑的新手用户。](https://web.archive.org/web/20221208220015/https://beta.techcrunch.com/2010/04/14/live-twitter-ceo-ev-williamss-chirp-keynote/)

为了强调他的观点，他展示了一段痛苦的视频，视频中一名妇女在一个焦点小组中努力寻找手机上的 Twitter 应用程序。她不知道选哪一个，因为太多了，而且没有一个明显的。“我们发现我们为用户提供的服务不足，”他说。“我们意识到，我们必须在这些主要平台上拥有核心体验，就像我们在网络上一样，否则我们就是失败的用户。”

最大的挑战是试图“接纳”新用户，Twitter 决定不能依赖其他公司来为他们做这件事。但是也有控制体验的因素。Ev 注意到这一策略正在发挥作用。新的黑莓应用程序正在“推动大量的使用”自黑莓官方应用推出以来，它负责 7%至 8%的新注册，即三天内 10 万个新注册。它本身也已经贡献了近 2%的推文。

因此，虽然 Twitter 可能会填补更多的漏洞，并与其他开发商直接竞争，但 Ev 对此持积极态度。他在会议上对开发者说:“我们能为你们做的最好的事情就是把用户群扩大到数亿新用户。”总的来说，这是真的，但我现在不想成为一名 iPhone 或黑莓 Twitter 应用程序的开发者。
