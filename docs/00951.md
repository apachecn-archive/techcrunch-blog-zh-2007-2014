# MYSQL 首次公开募股聊天再次升温技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/10/26/mysql-ipo-chatter-picking-up-again/amp/>

[<amp-img src="https://web.archive.org/web/20230215180318im_/https://techcrunch.com/wp-content/uploads/2007/10/mysql.png" class="shot2 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" alt="mysql.png" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230215180318/https://techcrunch.com/wp-content/uploads/2007/10/mysql.png "mysql.png")今年最受期待的科技首次公开募股是开源数据库公司 MySQL。看起来他们已经准备好在年初上市了。现在，我听到对冲基金圈子里有传言称，申请可能迫在眉睫。据我所知，还没有向证交会提交任何文件。包括基准、指数、IVP、英特尔和思爱普在内的投资者迄今已投入超过 3900 万美元。

MySQL 声称拥有数据库市场 25%的份额。尚不清楚它的经济效益有多有吸引力，但如果它的首次公开募股做得好(假设它曾经发生过)，这将最终为其他开源初创公司打开大门，比如 [Openads](https://web.archive.org/web/20230215180318/http://www.openads.org/) ( [开源广告服务器](https://web.archive.org/web/20230215180318/https://techcrunch.com/2007/06/13/openads-opensource-project-raises-5-million/))和[automatic](https://web.archive.org/web/20230215180318/http://automattic.com/)(已经在开源博客平台 WordPress 上建立了业务)等等。
