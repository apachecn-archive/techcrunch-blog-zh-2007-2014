# 脸书起诉有隐私控制到位。是的，说真的。

![](img/abab035b812e8b629573d49930e2de63.png)

上周，[Priceline.com](https://web.archive.org/web/20221005184607/http://www.crunchbase.com/company/priceline)的共同发明人杰伊·s·沃克创立的“发明公司”[沃克数码](https://web.archive.org/web/20221005184607/http://www.walkerdigital.com/)又打了一场专利官司。

好家伙，这球是从左外野直接打来的。

[彭博商业周刊](https://web.archive.org/web/20221005184607/http://www.businessweek.com/news/2010-11-24/facebook-sued-by-inventor-behind-priceline-com.html)上周三首次报道了这一消息，但不知什么原因，该报道搞错了诉讼专利的数量(应该是 [5，884，272](https://web.archive.org/web/20221005184607/http://www.google.com/patents/about?id=TDEYAAAAEBAJ&dq=5,884,272) 而不是 [5，884，271](https://web.archive.org/web/20221005184607/http://www.google.com/patents/about?id=SzEYAAAAEBAJ&dq=5,884,271) ，导致后续报道中出现大量[混乱](https://web.archive.org/web/20221005184607/http://www.wiredvc.com/exclusive-facebook-603sued-over-universal-electronic-transaction-card-technology/)。

今天早上我收到了投诉(见下文)，它实际上是一个很好的阅读材料。

让我们先来看看这个专利，题目是“建立和维护用户控制的匿名通信的方法和系统”。它是这样描述的:

> 一种用于建立匿名通信的系统包括多个参与方终端、多个请求方终端和一个中央控制器。该系统接收并存储有关各方的各方数据。
> 
> 在从请求者终端接收到有关各方的标准和来自各方的授权后，中央控制器向请求者方释放与各方相关联的各方。该系统还在各方和请求者之间建立通信渠道，同时保持他们的匿名性。

基本上，这项专利描述了用户如何管理和控制关于他们自己或他们身份的信息的发布。此外，它还显示了如何通过使用授权请求(例如，使陌生人成为朋友)立即或逐步公开此类个人信息。

对于一项专利来说，这太宽泛了，但是好吧，这种情况经常发生。这起诉讼的有趣之处在于，沃克数字公司声称脸书侵犯了其(已有十年历史的)专利。

根据诉状，脸书是一个社交工具，允许用户与朋友和其他个人的社区互动，但是，原告说，“用户信息的共享仍然通过用户可选择的隐私设置来控制。”

它继续说道:

> “‘社会效用’就像专利一样，是关于分享的。脸书的隐私控制让你有权决定分享什么和分享多少。
> 
> 就像这项专利一样，一旦你找到你认识的人，然后点击“添加为好友”按钮，一个好友请求就会被发送给那个人。"

简而言之，由于脸书通过让用户决定你与谁分享哪些信息，让人们能够在一定程度上控制他们在流行社交网站上的隐私，沃克数字公司认为该公司侵犯了其一项“发明”。

如果我对投诉的理解正确，而且整件事不是愚人节的玩笑，那么这一整套衣服就是可笑的。

作为对彭博那篇文章的回应，脸书的一位发言人表示，他们将积极应诉，称其“完全没有意义”。这一次，我不能不同意他们的观点。
