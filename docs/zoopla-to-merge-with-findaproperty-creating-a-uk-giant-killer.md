# zoo 计划与 FindaProperty 合并，打造英国巨头黑仔

拥有《每日邮报》和其他媒体权益的《每日邮报和普通信托》计划将其由 FindaProperty.com 和 Primelocation.com 组成的数字财产集团的在线财产业务与风险投资支持的房地产初创公司 Zoopla 合并。根据提议的合并，T4 媒体(DMGT 的消费者部门)将在新合并的实体中保留 55%的股份。该提议提出后，DMGT 股市上涨。英国在线房地产市场的主导者是 [Rightmove](https://web.archive.org/web/20230204234150/http://rightmove.co.uk/) ，但这次合并产生了一个强大的竞争对手。Rightmove 的市值为 13 亿英镑
