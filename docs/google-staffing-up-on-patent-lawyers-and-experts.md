# 谷歌招聘专利律师和专家

众所周知，谷歌正在寻找更多的专利加入其投资组合。这个搜索巨头最近[在北电 6000 项移动和无线专利的竞标中输给了一个由苹果、微软、RIM 和索尼等科技公司组成的财团。正如我的同事 MG Siegler 当时所写的那样，谷歌控制着不到 1000 项专利，这与其一些竞争对手相比是很低的。谷歌目前](https://web.archive.org/web/20230203052602/https://techcrunch.com/2011/07/01/apple-microsoft-rim-google-nortel-patents/)[总共拥有 701 项专利](https://web.archive.org/web/20230203052602/http://parislemon.com/post/7279421090/google-searching-for-patents-after-nortel-loss)，而微软去年获得了 3121 项专利。不幸的是，由于谷歌并不拥有大量专利，该公司将继续容易受到专利诉讼。

也许这就是谷歌正在增加专利律师、职员和法律专家的原因。在其[法律职位列表页面](https://web.archive.org/web/20230203052602/http://www.google.com/intl/en/jobs/uslocations/mountain-view/legal/index.html)上，谷歌正在寻找六个与专利相关的职位，包括专利代理人、专利顾问、专利书记员、专利诉讼顾问、专利律师助理和战略专利许可和收购经理。

例如，[战略专利许可和收购经理](https://web.archive.org/web/20230203052602/http://www.google.com/intl/en/jobs/uslocations/mountain-view/legal/strategic-patent-licensing-and-acquisitions-manager-mountain-view/index.html)评估和评价潜在的专利收购和许可机会，并就这些交易进行谈判(也就是为谷歌找到更多专利)。

除了寻找新的专利机会，谷歌还有一个重大的专利诉讼要处理——甲骨文的诉讼。8 月，甲骨文起诉谷歌安卓移动操作系统，称谷歌侵犯了该操作系统及其 Dalvik 虚拟机的七项 Java 专利。甲骨文正在寻求从谷歌获得数十亿美元的奖励，如果甲骨文赢得这个奖项，无疑对谷歌和安卓都是坏消息。
