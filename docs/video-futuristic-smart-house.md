# 视频:未来派智能房屋

[![](img/6e5a501bc4b538515e26288ea4bbfe92.png "sekusui_house")](https://web.archive.org/web/20230202111300/http://www.crunchgear.com/2011/03/10/video-futuristic-smart-house/sekusui_house/)

这不是我们迄今为止报道的第一个“下一代住宅”，但是日本主要房屋制造商[积水房屋](https://web.archive.org/web/20230202111300/http://www.sekisuihouse.co.jp/english/)的所谓“智能住宅”绝对是迄今为止最酷的概念之一。除了积水，电信巨头 [NTT Docomo](https://web.archive.org/web/20230202111300/http://www.mobilecrunch.com/tag/docomo/) 、 [NEC](https://web.archive.org/web/20230202111300/http://www.crunchgear.com/tag/nec/) 和 Namco Bandai Games 都参与了最近在日本展示的原型房屋的建造。

简而言之，这个想法是将不同的技术结合起来，以创造一个集成的概念，使节能，监控电动汽车的位置，存储房屋的“维护历史”，跟踪你周围人的健康状况等等成为可能。一切都可以用智能手机管理——房子里没有开关。

例如，智能房屋允许人们监控年长亲戚的健康状况:如果他或她在早上打开灯，然后开着联网的电动汽车去超市，你就知道他们在哪里，知道他们没事(你甚至可以将房屋收集的生物信号发送给你)。
