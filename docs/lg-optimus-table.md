# LG Optimus 平板电脑今天赢得了互联网

![](img/e48695a0f22467c3ab73d38b8c6b09f7.png "LG_Tablet_2")
你知道吗？LG 营销副总裁应该加薪。《华尔街日报》称 LG Optimus 平板电脑[将比](https://web.archive.org/web/20221206212808/http://www.crunchgear.com/2010/08/20/lg-our-tablet-will-be-better-than-the-ipad-thats-confidence/) [iPad](https://web.archive.org/web/20221206212808/http://www.crunchgear.com/tag/ipad/) 更好赢得了今天的互联网。毫无疑问，这是今天谈论最多的技术故事之一，这意味着营销副总裁只是做了他的工作。

[我们在平板电脑上的帖子](https://web.archive.org/web/20221206212808/http://www.crunchgear.com/2010/08/20/lg-our-tablet-will-be-better-than-the-ipad-thats-confidence/)是今天整个 TechCrunch 网络上评论最多的故事。这是 [Engadget 的](https://web.archive.org/web/20221206212808/http://www.engadget.com/2010/08/20/ce-oh-no-he-didnt-lgs-ma-promises-optimus-tablet-will-be-be/)当天评论第二多的帖子，也是 [Gizmodo](https://web.archive.org/web/20221206212808/http://gizmodo.com/5617543/lgs-tablet-will-be-surprisingly-productive-and-better-than-the-ipad) 上迄今为止访问量最高的帖子之一。它甚至占据了 Techmeme 大部分时间的榜首，有 25 个博客加入了讨论。都是因为一句愚蠢的引言。

古老的格言“没有坏名声”今天仍然适用。我没有看到一篇关于 LG 平板电脑的文章是全力以赴的赞美，或者至少有几次没有提到 iPad。大多数人都至少有点怀疑副总统的声明。

但是没关系。在今天早上这个故事曝光之前，我甚至不知道 LG 正在开发一款平板电脑。对我们的大多数读者来说，这可能是真的。几个月后，毫无疑问，经过几次精心策划的“泄露”和预发布，平板电脑将最终进入零售市场，我们都将在脑海中记住这个东西应该比 iPad 更好。

副总裁的说法可能有点夸张。那是他的工作，对吗？他应该吹嘘他公司的产品，好像它们是世界上最好的东西。毕竟，大多数时候，区分销售人员和营销人员的唯一标准是大学文凭。他们的目标往往是相同的。但是这里有一条细线。当产品离发布还有几个月的时候，大谈特谈是可以的。我们是一群善变的人，随着产品接近发布，更多的信息变得可用，我们的期望可能会变得平稳。

也许 LG 平板电脑比 iPad 更好。我们没见过。我们不知道是不是。然而，如果 LG 继续像一个真正的 iPad 杀手一样谈论它，它最好能兑现。或许它应该被称为 LG 乐观派。
