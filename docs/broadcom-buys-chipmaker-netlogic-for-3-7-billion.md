# 博通以 37 亿美元收购芯片制造技术公司 Netlogic

[Broadcom](https://web.archive.org/web/20230204235617/http://www.crunchbase.com/company/broadcom) 今晨[宣布](https://web.archive.org/web/20230204235617/http://www.netlogicmicro.com/News/pr/2011/11-09-12merger.asp)已同意收购 [NetLogic 微系统](https://web.archive.org/web/20230204235617/http://www.crunchbase.com/company/netlogic-microsystems)，后者提供一系列半导体解决方案。根据协议，NetLogic 的股东将在约 37 亿美元的交易中获得每股 50 美元，扣除现金假设。

Netlogic 的股价上周五收于 31.91 美元。

通过此次收购，Broadcom 通过大量新的[产品线](https://web.archive.org/web/20230204235617/http://www.netlogicmicro.com/Products/)和技术扩展了其基础设施组合，包括基于知识的处理器、多核嵌入式处理器和数字前端处理器。

NetLogic Microsystems 总部位于加利福尼亚州圣克拉拉市，在北美、亚洲和欧洲设有办事处和设计中心。

该交易已获得 Broadcom 和 NetLogic Microsystems 董事会的批准，并符合惯例成交条件。该交易预计将于 2012 年上半年完成。

Netlogic 的股票在开市前上涨了近 20%。(**更新:**现在涨了百分之五十)

博通表示，预计该交易将使 2012 年的非 GAAP 收益增加 10 美分。该公司今天还重申了 2011 年第三季度的业务前景——公司预计收入将在 19 亿至 20 亿美元之间。
