# 任天堂正在计划 3DS 游戏中虚拟物品的销售

似乎像智能手机游戏革命、社交游戏的兴起和视频游戏销售萎缩这样的因素确实对任天堂的商业战略产生了影响。日本各种媒体，即该国最大的商业日报日经新闻，今天报道称[任天堂](https://web.archive.org/web/20230203084937/https://techcrunch.com/tag/nintendo)正计划为 [3DS](https://web.archive.org/web/20230203084937/https://techcrunch.com/tag/3ds) 游戏提供游戏内虚拟物品销售。

这里的背景是，任天堂的“数字”业务(例如，经典游戏的下载)的销售额目前占总销售额的不到 5%。对于其他“传统”视频游戏制造商来说，这个数字要高得多:例如，Konami 现在通过社交游戏(在日本非常受欢迎)比通过套装软件赚得更多。

据报道，任天堂计划最早在下个月通过固件升级在 3DS 上启用微交易系统。支持该系统的游戏预计将于 2012 年发布:这些游戏将提供一些功能(如额外的关卡、额外的角色等)。)只有买家向任天堂支付一定费用才能解锁。

我的猜测是，新的支付系统也将使该公司有可能在未来提供基于订阅的游戏模式，例如 RPG。任天堂一再表示，它不会提供智能手机游戏，其中许多游戏是通过虚拟物品销售赚钱的。
