# Pokki 计划将其应用推荐服务引入 Android，从其 Windows 滩头阵地 TechCrunch 扩展开来

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/01/08/pokki-plans-to-take-its-app-recommendation-service-to-android-expanding-from-its-windows-beachhead/>

今天早上， [Sweet Labs](https://web.archive.org/web/20221209072652/http://www.crunchbase.com/company/sweetlabs) 的产品 [Pokki](https://web.archive.org/web/20221209072652/http://www.pokki.com/) 宣布，它将把应用推荐引擎扩展到 Android 设备，并为该公司所谓的“多模式”设备——那些非传统硬件规格的设备——增强其应用商店。

2013 年，Pokki 的安装量大幅增长，通过与两家大型原始设备制造商(宏碁和联想)合作，下载量超过 5000 万次。这些交易将 Pokki 带到了新地点的新系统上(该公司还扩大了其语言支持基础，以允许其登陆更多市场)，但该公司的产品在很大程度上仍是面向 Windows 的服务。

因此，Pokki 通过替换微软最新 Windows 版本上丢失的 Windows 开始按钮实现了早期增长，这并不奇怪。

现在，Pokki 希望扩大其业务范围，但不是通过增加新的原始设备制造商作为合作伙伴——它拒绝告诉 TechCrunch 它正在与谁谈判，以及这些交易何时会有结果——而是通过扩展到 Android，并调整其核心产品，以更好地适应不是以台式机和笔记本电脑流行的老式方式制造的电脑。

Pokki 既是 Windows 上的应用商店，也是应用推荐服务。其 OEM 合作伙伴销售的电脑预装了 Pokki，为新电脑用户提供应用程序。所得收入由 Pokki 和 OEM 分享。鉴于硬件公司在新设备上的微利，这一点很重要。

## 外形因素

尤其是随着 Windows 8 的发布，PC 变了形。现在可以买到没有键盘的大触摸屏电脑，也可以买到可以转换的可拆卸机器，以及可以左右翻转的平板电脑。这导致了一个 PC 现实，一个最适合笔记本电脑的应用程序在不同形式的机器上可能不那么好。

Pokki 正在努力调整其技术，根据安装该技术的电脑类型推荐不同的应用程序。该公司称，这项技术尚未投入使用，但将于今年上半年登陆。

我测试了结果，跨设备类别查看了 Pokki，并喜欢我所看到的。我在我的电脑上使用的软件当然会根据它们的形式而有所不同，所以我认为用户会喜欢不同的应用程序。此外，Pokki 团队指出，如果它可以改善应用程序推荐，它可以改善最终用户的体验，设备退货和原始设备制造商本身的其他成本可以降低。

实质上，Pokki 正在努力调整其系统，以支持不断变化的个人电脑市场。

## 安卓

Pokki 在你可以称之为 CES 回合的新闻中宣布的后半部分是一个不足为奇的事实，即它正在将其系统升级到 Android。然而，与 Pokki for Windows 不同，该公司展示了传统桌面应用程序和为其平台构建的应用程序的组合，而在 Android 上，Pokki 将作为一个应用程序分类系统，引导用户到 Google Play 执行下载。

Pokki for Android 将于下半年登陆。

原始设备制造商越来越多地在以前 Windows 机器占主导地位的地方制造基于 Android 的电脑。安卓一体机，甚至安卓笔记本电脑都可能成为常态。鉴于英特尔推动 Windows 和 Android 在同一系统中的统一，这一趋势可能会变得更加平淡无奇。

—

以上简单的要点是，Pokki 正在努力改善新计算机上的最终用户体验，同时从这些新出售的系统中产生收入。通过与 OEM 合作伙伴分享这一收入，Pokki 可以将应用广告预算转化为自己和合作伙伴硬件公司的有意义收入，这些公司长期以来一直难以从高收入但利润微薄的新 PC 中获得真正的利润。

Pokki 在 CES 期间所做的是它下一步要去哪里的细节，但它尚未向市场发布产品，也没有证明新技术会引起消费者的共鸣。该公司目前每天的应用下载量约为 100 万次，如果它在未来一年获得新的 OEM 交易，这一数字可能会上升。结合改进的建议，可以看到它更好地转换其安装基础。

目前，Pokki 拥有强大的安装渠道来推动其增长。下一个问题是，它能利用这种势头做什么。
