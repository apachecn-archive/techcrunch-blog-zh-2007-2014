# 数据可移植性在微软获得了另一个转换

[![dataportability-logo.png](img/123008ca4b133281f25435b5e83efef3.png)](https://web.archive.org/web/20230316005822/http://www.dataportability.org/) 如今，你在网上拥有自己的数据，并可以从一个社交网络或网站带到另一个社交网络或网站的概念越来越受欢迎。雅虎、MySpace、 [LinkedIn](https://web.archive.org/web/20230316005822/https://techcrunch.com/2008/01/10/linkedin-joins-the-dataportability-work-group/) 、谷歌、Plaxo、[甚至脸书](https://web.archive.org/web/20230316005822/https://techcrunch.com/2008/01/08/this-day-will-be-remembered-facebook-google-and-plaxo-join-the-dataportability-workgroup/)都加入了[数据可移植性工作组](https://web.archive.org/web/20230316005822/http://www.dataportability.org/)来制定标准。现在，[微软也加入了](https://web.archive.org/web/20230316005822/http://dev.live.com/blogs/devlive/archive/2008/01/23/203.aspx)。有 4.2 亿 Windows Live IDs 与用户资料相关，微软的参与令人鼓舞。数据锁定的日子真的已经过去了吗？

别指望了。
