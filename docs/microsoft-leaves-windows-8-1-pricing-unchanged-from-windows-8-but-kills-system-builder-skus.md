# 微软保持 Windows 8.1 的定价不变，但取消了系统构建商 SKU

微软今天[宣布](https://web.archive.org/web/20221206125833/http://blogs.windows.com/windows/b/bloggingwindows/archive/2013/09/17/pricing-and-packaging-for-windows-8-1.aspx)即将推出的 Windows 8.1 操作系统的定价结构。Windows 8 用户将免费获得新代码，但对于那些目前没有安装微软最新操作系统的用户来说，该软件的基本版价格为 119.99 美元，“专业版”为 199.99 美元。

微软还提供了一个独立的 Windows 8.1“Pro Pack”，我不明白。当我追查到底是什么，我会更新这个帖子。

需要注意的是，上面列出的 Windows 8.1 价格是针对完整的 Windows 8.1 操作系统，而不仅仅是从以前安装的 Windows 升级而来。这是一种改变。以前，微软出售 Windows 的升级版和独立的“系统构建”版本。

不会了。正如该公司在一篇博客文章中指出的，通过提供“完整版软件”，不强迫用户安装 Windows 的先前版本，Windows 8.1 可以以更大的“灵活性”进行部署。实际上，Windows 现在只有两个普通的 SKU:普通版和专业版。这很简单。还有来自微软，这几乎令人难以置信。

微软称“反馈”是其不再销售升级版和独立“新鲜”操作系统的原因之一。想必这是新常态，微软未来会保持简化的销售结构。正如该公司指出的那样，这一变化将使在虚拟环境中运行 Windows 8.1 变得简单得多。

下面是[的往事](https://web.archive.org/web/20221206125833/http://windows.microsoft.com/en-us/windows-8/upgrade-to-windows-8#section_5):

> 如果您想要构建自己的 PC 并安装 Windows 8 或 Windows 8 Pro，或者想要在本地虚拟机或独立分区(包括 Mac)中运行额外的操作系统，您可以购买 Windows 8 或 Windows 8 Pro System Builder 产品(OEM 版本)。如果您所在的国家或地区提供 Windows 8 和 Windows 8 Pro System Builder 产品，您可以在参与商店购买，但您需要询问销售代表了解更多信息。这个版本不包括客户支持。

现在，从 Windows 8.1 开始，你购买的任何 Windows 版本都将是完整的代码集，不管你的电脑上有什么。这是一个美好而长久需要的改变。

通过简化 Windows 8.1 的购买和安装，是否有助于推动需求和销售？也许有一点，但这不是果汁单位体积的移动。对于爱好者和运行虚拟环境的人来说，这是一个受欢迎的变化。
