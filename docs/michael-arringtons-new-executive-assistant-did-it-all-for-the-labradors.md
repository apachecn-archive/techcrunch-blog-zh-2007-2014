# 迈克尔·阿灵顿的新行政助理做的一切都是为了拉布拉多犬

迈克尔·阿灵顿决定通过让 TechCrunch 的读者建议他一天应该完成的任务，来测试他的新行政助理[格雷格·巴尔托](https://web.archive.org/web/20230204221501/http://twitter.com/#!/Bartozone)实际能做的极限。使用 [Zaarly](https://web.archive.org/web/20230204221501/http://www.zaarly.com/) ，葛瑞格被派去完成一天的冒险任务(冒着生命和尊严的危险？)都是由社区提供的，并且都是为了慈善事业。幸运的是，这一切都在下面的视频中捕捉到了。建议观众自行判断。

第一:格雷格在索玛的一个屋顶上差点被上百只蜜蜂蛰到。

第二个问题:在一次会议上，格雷格教了整个公司如何做导盲犬。

Zaarly #3: Greg 给某人的男朋友发了一封唱歌电报，给他和后街男孩唱布兰妮。

第四个问题:格雷格差点因为跳进公司办公室外的喷泉而被捕。

第五个问题:格雷格带着五只狗穿过拉斐特公园。

第六:格雷格在联合广场为迈克唱了一首歌，向他表达了爱意。
