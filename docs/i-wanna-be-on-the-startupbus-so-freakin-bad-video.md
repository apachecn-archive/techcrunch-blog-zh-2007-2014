# 我想成为“创业乐观主义者”...糟糕透顶(视频)的 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2012/02/10/i-wanna-be-on-the-startupbus-so-freakin-bad-video/>

[YouTube http://www.youtube.com/watch?v=2f70gxTUt5U]

只要一点糖果，就能开始你的周末生活。

为了让 StartupBus 注意到它的状态，卢西亚纳的一群创新者 T2 本周与创始人埃利亚斯·比赞尼斯会面，参加了[巴吞鲁日地区商会](https://web.archive.org/web/20221226033556/http://www.brac.org/)2 月的[创新外卖](https://web.archive.org/web/20221226033556/http://siliconbayounews.com/2012/02/06/startupbus-founder-elias-bizannes-visiting-baton-rouge-this-thursday/ "StartupBus Founder Elias Bizannes visiting Baton Rouge this Thursday")活动。在比赞尼斯的演讲(最后他宣布了路易斯安那州的第一个 StartupBus！)，当地科技社区的成员发布了这个恶搞视频:*“我想坐上创业巴士。”*

很有意思。此外，在纽约和硅谷之外创业总是令人兴奋的。

如果这首歌听起来很熟悉，那是因为它很熟悉。[来源于](https://web.archive.org/web/20221226033556/http://www.youtube.com/watch?v=8aRor905cCw)。这段视频也在巴吞鲁日市中心的大屏幕上播出。

[startup buss](https://web.archive.org/web/20221226033556/http://startupbus.com/)，对于你们中不熟悉的人来说，是大约三年前由 Elias Bizannes 发起的黑客马拉松/公路旅行组合，前往西南偏南地区。2012 年，该组织举办的活动数量翻了一番。今年，最多将会有 12 辆公交车，包括那些从旧金山、纽约、洛杉矶、克里夫兰和坦帕出发的公交车，以及其他在这里投票的公交车。
