# 脸书考虑 HTML5，不会谈 iPad 应用

今天早些时候，有很多关于脸书的传言，据说他们的网站支持 HTML5 视频播放。这当然可以让脸书的视频在 iPad 和 iPhone 的浏览器中自然播放。更重要的是，这将是对 Adobe Flash 格式的又一次重大打击。唯一的问题？这些脸书视频根本没有使用 HTML5。

是的，当从 iPad 或 iPhone 上点击时，脸书上的视频确实可以播放，但只有读写网[指出](https://web.archive.org/web/20221204183501/http://www.readwriteweb.com/archives/facebook_testing_html5_video_for_ipad_apparently_not.php)这是通过 h264 播放器(iPhone 和 iPad 都一直支持)，而不是实际上任何类型的 HTML5 实现。我们已经向脸书证实了这一点，他说，“*你是对的，这不是 html 5。所有新视频都是以 h264 格式编码的，所以我们在 iPad 上本地播放视频，因为它支持 h264 编码的视频*。

好吧，但更重要的是，脸书有计划支持 HTML5 吗，就像苹果、谷歌和许多其他公司已经开始做的那样？一位脸书的代表告诉我们:“*我们会考虑未来的*”。

我还借此机会询问了一款脸书本土 iPad 应用的状态。脸书的 iPhone 应用程序无疑是 App Store 中最好的应用程序之一，但由于它不是通用的(这意味着 iPhone 和 iPad 都有版本)，它像所有其他 iPhone 应用程序一样，在 iPad 上运行时放大到两倍大时看起来有点垃圾。“关于你的 iPad 问题，我们没有什么要宣布的，”我被告知的就是这些。

脸书是 2008 年 App Store 首次推出时准备好的首批应用程序之一，但它在 iPad 应用商店中一直不见踪影。这是一个问题，因为很多人都在寻找它，只找到廉价的仿制品。你可能还记得，脸书 iPhone 应用程序的创建者，[乔·休伊特，](https://web.archive.org/web/20221204183501/http://www.crunchbase.com/person/joe-hewitt)，[因为](https://web.archive.org/web/20221204183501/https://beta.techcrunch.com/2009/11/11/joe-hewitt-developer-of-facebooks-massively-popular-iphone-app-quits-the-project/)[不喜欢应用商店的一些政策而停止了开发。脸书带来了一个团队来代替他，但是到目前为止，什么都没有。](https://web.archive.org/web/20221204183501/https://beta.techcrunch.com/2009/08/24/facebook-app-developer-to-apple-tear-down-this-app-store-wall/)
