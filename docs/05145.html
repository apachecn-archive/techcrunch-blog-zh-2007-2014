Diaspora澄清:9月15日开源，10月消费者Alpha

几天前，我们注意到散居国外的还有三周就要发布他们开源的脸书替代方案。但是今天一个小小的更新澄清了重要的一点。9月15日，Diaspora将发布该项目的开源代码。但直到10月份，第一款面向消费者的alpha才会上市。

这不是一个大问题，这只是一个更新，以确保人们不会期望在9月15日使用Diaspora。相反，在那个时候，开发者将有机会修补团队在过去几周内提出的东西——他们在6月份通过Kickstarter从6500人那里筹集了20万美元。

但是10月的alpha可能是这个项目更重要的时间。虽然所有开源和分布式网络的东西听起来都很棒，但是对于几千名开发人员和超级书呆子来说也很棒。如果Diaspora真的想获得任何牵引力，他们将需要一个杀手级前端，让人们有一个实际的理由使用它，而不是脸书。(或者他们需要其他人根据他们的代码来制作——但这显然最早也要几个月以后。)

幸运的是，听起来团队在这方面说的是对的。以下是他们几天前更新的关键部分:

我们花了大量的时间专注于建立清晰的、上下文相关的共享。这意味着用户可以通过一种直观的方式来决定，而不是注意决定，哪些内容发给他们的同事，哪些内容发给他们的酒友。我们知道这是一个很难解决的UI问题，我们对此非常重视。你们给我们的宣传和资金让我们能够通过她的新项目LUXr与Janice Frasier这样的伟大设计师合作，她的新项目不断提醒我们，我们不是用户，这让我们保持诚实和专注。Pivotal Labs还帮助我们进行了优先级排序，我们已经将插件和API等更多技术功能推回，以支持简单和高价值的功能。我们最初的目标保持不变，这些功能仍然在我们的时间表中。

我个人迫不及待地想看看他们想出了什么。

以下是他们今天的最新消息:

附录(8/30):澄清一下，9月15日将是我们的开源开发者发布日。那时，我们将开放我们的 github 库，发布我们的路线图，并将我们的开发风格转变为更加面向社区。我们打算在10月推出面向消费者的alpha。加入我们的邮件列表获得邀请。
