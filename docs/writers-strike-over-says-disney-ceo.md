# 前迪士尼首席执行官称，作家罢工

[photo press:threadovermaybe . jpg，full，center]

作家罢工可能结束了。美国消费者新闻与商业频道和其他人引用前迪士尼首席执行官迈克尔·埃斯纳的话说，“一切都结束了。他们达成了协议，他们就此握手。一般来说，这将在周六进行……我们已经达成协议，他们很快就会回来工作。”

自上周末以来，交易的谣言一直在流传，昨天洛杉矶的纠察队似乎在减少。

我真诚地希望作家们得到他们想要的。我会全力以赴的。

**更新:**《T2 日报》对艾斯纳的采访进行了全面总结。是的，让那个*成为前任*首席执行官。
