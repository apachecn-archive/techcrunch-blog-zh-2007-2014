# 索尼继续致力于谷歌电视，在 CES 2012 上发布两款新的 GTV 产品

索尼在 2010 年发布了首款谷歌电视产品。自那以后，智能电视平台经受住了一年多的批评和一次重大检修。它离行业领导者还差得很远，这个头衔是由三星 SmartHub 领先的，但谷歌仍在坚持。索尼也是如此。

今天，在 2012 年国际消费电子展上，该公司发布了一款新的流媒体播放器和蓝光播放器，这两款播放器都内置了谷歌电视。像第一个版本一样，这些型号具有几个以索尼为中心的应用程序。将索尼应用程序 Media RemoteTM 和 TrackID 用于 Android 和 iOS 移动设备，用户可以实现跨设备连接。TrackID 是索尼 Shazam 的克隆版，内置了从 Android 或 iOS 设备控制 GTV 所需的商品。

[![](img/615a73ab1cff28476cf7cfb5efac42ae.png "NSZ-GS7_Network_Media_Player")](https://web.archive.org/web/20221007102514/https://beta.techcrunch.com/wp-content/uploads/2012/01/nsz-gs7_network_media_player.jpg) 
两款机型都配备了重新设计的背光键盘，支持手势和三轴运动控制的触摸板。蓝光型号甚至为谷歌语音搜索配备了内置麦克风。因为，你知道，人们对着电视大喊大叫还不够。

NSZ-GS7 网络媒体播放器和 NSZ-GP9 蓝光光盘播放器将在今年夏天以未公布的价格上市。
