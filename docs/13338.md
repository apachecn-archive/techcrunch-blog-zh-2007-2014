# 谷歌:美国和欧洲超过 500 个学区现在使用 Chromebooks | TechCrunch

看起来学校是谷歌 Chromebook 计划取得成功的一个市场。今天，该公司[宣布](https://web.archive.org/web/20230404045519/http://googleenterprise.blogspot.com/2012/06/in-schools-all-you-need-is-web.html)美国和欧洲现在有 500 个学区在积极使用 Chromebooks。谷歌还宣布了最近决定使用谷歌以网络为中心的笔记本电脑的几个新校区，包括北卡罗来纳州的罗金厄姆乡村学校、北卡罗来纳州的特兰西瓦尼亚县学校和威斯康星州的丰迪拉克学区。

谷歌能够在教育领域推动 Chromebooks 的原因之一是，其笔记本电脑符合智能平衡评估联盟和大学和职业准备评估伙伴关系(PARCC)制定的[新硬件和操作系统指南](https://web.archive.org/web/20230404045519/http://www.smarterbalanced.org/news/smarter-balanced-and-parcc-issue-guidance-for-new-instructional-technology-purchases/)。

[<amp-img src="https://web.archive.org/web/20230404045519im_/https://techcrunch.com/wp-content/uploads/2012/06/apps-for-schools.png?w=300" alt="" title="apps for schools" class="alignright size-medium wp-image-577672 amp-wp-enforced-sizes i-amphtml-layout-intrinsic i-amphtml-layout-size-defined" srcset="https://web.archive.org/web/20230404045519im_/https://techcrunch.com/wp-content/uploads/2012/06/apps-for-schools.png 400w, https://web.archive.org/web/20230404045519im_/https://techcrunch.com/wp-content/uploads/2012/06/apps-for-schools.png?resize=150,146 150w, https://web.archive.org/web/20230404045519im_/https://techcrunch.com/wp-content/uploads/2012/06/apps-for-schools.png?resize=300,292 300w, https://web.archive.org/web/20230404045519im_/https://techcrunch.com/wp-content/uploads/2012/06/apps-for-schools.png?resize=32,32 32w, https://web.archive.org/web/20230404045519im_/https://techcrunch.com/wp-content/uploads/2012/06/apps-for-schools.png?resize=50,50 50w" layout="intrinsic" i-amphtml-layout="intrinsic"><i-amphtml-sizer class="i-amphtml-sizer"></i-amphtml-sizer></amp-img>](https://web.archive.org/web/20230404045519/https://techcrunch.com/2012/06/25/chromebooks-education-500-school-districts/apps-for-schools/)除了这些部署公告，谷歌还在今天的国际教育技术学会( [ISTE](https://web.archive.org/web/20230404045519/http://www.iste.org/conference/ISTE-2012.aspx) )会议上为学生和管理员推出了新的工具和网络应用程序集合。谷歌表示，这些工具将“让整个学校更容易找到、使用、安装和管理网络应用程序。”年级级应用程序包可从 Chromebook 管理控制台安装，其功能应用程序与谷歌教育应用程序紧密集成。特定于组织的 web 应用程序集合允许管理员从 Chrome Web Store 中挑选应用程序，并将其推荐给学生、教师和员工。

许多公司也在今天的 ISTE 会议上推出了新的 Chrome 教育应用，包括 [ST Math](https://web.archive.org/web/20230404045519/https://chrome.google.com/webstore/detail/ahgfknfdplnpiiiiejiimkojdkjlfmdo) 、 [VoiceThread](https://web.archive.org/web/20230404045519/https://chrome.google.com/webstore/detail/gajclnhcflhoicggnpmgkedchldikjgn) 和 [Achieve3000](https://web.archive.org/web/20230404045519/https://chrome.google.com/webstore/detail/dmihfidhdkkjnmkcngbolncoijgkldfi) 。

在今天的声明中，谷歌说“在学校里，你需要的只是网络。”这是有争议的，但谷歌显然在教育领域做了一些正确的事情，尽管即使有了最新一代基于 ChromeOS 的硬件，Chromebooks 暂时仍可能是利基产品。然而，对于学校来说，Chromebooks 是传统笔记本电脑的有吸引力的替代品。毕竟，这些笔记本电脑的总拥有成本相对较低，因为它们很容易管理，几乎不受病毒影响，并且可以自我更新。
