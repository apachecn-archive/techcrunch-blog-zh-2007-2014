# Automattic 收购 Gravatar 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2007/10/17/automattic-acquires-gravatar/>

# 自动获取 Gravatar

[![gravatar.jpg](img/0faa1c486420388d142dbd259ef5b100.png)](https://web.archive.org/web/20230215180303/http://www.gravatar.com/)[automatic，](https://web.archive.org/web/20230215180303/http://www.crunchbase.com/company/automattic)WordPress.com 和 Akismet 背后的公司，已经收购了博客头像提供商 [Gravatar](https://web.archive.org/web/20230215180303/http://www.gravatar.com/) 。

Gravatar 提供了一个“全球公认的头像”，一个 80×80 像素的头像图像，可以在博客之间跟随用户，当他们在支持 gravatar 的网站上发表评论时，会出现在他们的名字旁边。

Automattic 的马特·莫楞威格在 Gravatar 博客上写道，Gravatar 正面临 Automattic 有能力处理的“经典规模问题”。立即生效，所有 Gravatar premium 功能现在都是免费的，任何在过去 60 天内购买了 premium 包的人都可以退款。Gravatar 支持现在将向 WordPress.com 用户提供，并将集成到所有 WordPress.com 模板中。

交易条款没有披露。
