# 微软想要创造下一个 Flickr

微软想创造下一个 Flickr 吗？他们的招聘广告无疑表明了这一点:

> 来让 Windows Live 成为共享您的数字记忆的最佳场所吧！听说过 Flickr 吗？YouTube？怎么样？麦克。该职位将在新的 Windows Live 部门与 Spaces、SkyDrive、Messenger 和 Hotmail 等团队合作，为微软在照片和视频共享方面构建一个成功的战略。这个角色包括领导团队设计战略，然后构建和实施一个赢得数百万客户喜爱的解决方案。这个功能团队正在构建下一代照片和视频共享服务，将与 flickr、smugmug 和当今其他照片网络解决方案竞争。这是一个“v1”机会。

其余的招聘广告[在这里](https://web.archive.org/web/20221209123219/http://members.microsoft.com/careers/search/details.aspx?JobID=809D02C9-2FC7-4B51-AD38-111FFF07884F)。
