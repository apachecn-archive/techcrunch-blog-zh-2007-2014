# 佳能的 ELPHs 军团增加了两个

![87blackf.jpg](img/230b7f8b23f6078a5123572c173f35c9.png)
870 黑衣

这有点无聊了，佳能。我不想再写你的新相机了。谁去叫救护车。随着 SD950 IS 和 SD870 IS digis 的加入，PowerShot ELPH 产品线变得更加出色。950 是一款 1210 万像素的全钛螺柱，配有 3.7 倍光学变焦和 2.5 英寸 LCD。870 只能聚集 800 万像素，但它有一个 3.8 倍广角变焦镜头和 3 英寸液晶显示器。两款 ELPH 都采用了 OIS、高级人脸检测和 DIGIC III 处理器。

950

![950.jpg](img/73ac02bdf00e1b70d4429475cb512dee.png)
950s 背面

PowerShot SD950 是 ELPH 数码相机，SD870 是 ELPH 数码相机，将于 9 月上市。950 的售价为 449.99 美元，870 有黑色和银色两种颜色，零售价为 399.99 美元。它们都包括可充电电池组、电池充电器、32MB MMC 存储卡和软件以及 USB 接口电缆、AV 接口电缆和腕带。
