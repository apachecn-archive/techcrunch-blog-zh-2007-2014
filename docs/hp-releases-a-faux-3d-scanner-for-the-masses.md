# 惠普发布面向大众的仿 3D 扫描仪

首先，这不是你所想的:这台扫描仪实际上不允许你扫描 3D 物体，但它允许你扫描物理物体。把它想象成一个更大的相机。尽管如此，惠普 TopShot 扫描仪显然是陷入困境的惠普的一个大胆举措。售价 399 美元的 MFP 可以打印、复印和扫描，它有一个特殊的手臂，可以摆动起来扫描 3D 物体。例如，你可以把一个模型放在上面，从不同的角度“扫描”它(真的“给它拍照”)。

这是我见过的第一个具有这种功能的扫描仪，如果你是 Etsy 制造商或易贝迷，这可能是一个不需要昂贵的灯箱和灯就能拍摄物品的好方法。否则，它只是一个普通的标准多功能打印机，可以放在任何人的书房或小办公室里。

尚未上市，但知道惠普 it 可能会在 47 天内停产[ [【可选音效】](https://web.archive.org/web/20230204224223/http://instantrimshot.com/index.php?sound=rimshot&play=true)。
