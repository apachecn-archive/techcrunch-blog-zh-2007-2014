# 独家:Lockerz 收购社交照片分享应用 Plixi(原 TweetPhoto)

![](img/35b3ae0a5e8c17d1d366ebfd72ae3362.png)社交商务网络[洛克兹](https://web.archive.org/web/20230205034236/http://www.lockerz.com/)已经收购了照片分享应用[普利西](https://web.archive.org/web/20230205034236/http://www.crunchbase.com/company/plixi)(即[原名 TweetPhoto](https://web.archive.org/web/20230205034236/http://www.readwriteweb.com/archives/tweetphoto_becomes_plixi_presents_new_vision_for_l.php) )，我们独家获悉。交易的条款没有透露，但我们听说收购价格在 1000 万到 1500 万美元之间。

**更新**:虽然商业内幕[声称](https://web.archive.org/web/20230205034236/http://www.businessinsider.com/plixi-lockerz-2011-1)这笔交易的价格低于 500 万美元，但我们有两个独立的消息来源证实了我们的范围，我们袖手旁观我们的报告。

Plixi 是一个非常受欢迎的社交照片共享应用程序，每月提供超过 10 亿张照片。这家初创公司去年从“Tweetphoto”更名而来，因为照片共享网络从 Twitter 扩展到了脸书和 MySpace 等其他平台，并增加了基于位置的签到功能。虽然 Plixi 提供了一个独立的 web 和 iPhone 应用程序，但超过 300 名开发人员集成了 Plixi 的照片共享技术，包括 Seesmic、TweetDeck 等，这家初创公司每天处理超过 6700 万个 API 请求。

Lockerz 已经从 Kleiner Perkins 新推出的 sFund 中筹集了 3000 万美元，其核心理念是社交网络中的影响者可以成为品牌和内容的倡导者，并影响他们朋友的行为。

Lockerz 在全球拥有近 1800 万会员，主要面向 13 至 30 岁的男性和女性，试图建立一个热爱购物、游戏和网络联系的潮流引领者和时尚先锋社区。用户可以通过在网站上分享内容来获得品牌积分和折扣。由前亚马逊和美鹰 Outfitters 高管 Kathy Savitt 创立；Lockerz 最终希望成为青少年的商务主页。

Savitt 表示，之前使用 Plixi 的用户体验将是一样的，只是服务的名称将会改变。洛克兹将把普利西的名字改为洛克兹照片。虽然开发人员仍然可以像使用 Plixi 一样使用 Lockerz 照片，但社交商务网络将使用 Plixi 的平台为平台添加照片共享体验。Lockerz 的会员可以在 Lockerz.com 上传、与朋友分享、评论和评价他们的照片。

随着时间的推移，Lockerz 和 Plixi 计划开发新的媒体分享功能来吸引客户，例如 Lockerz 成员可以在其他社交网络上分享他们的活动、照片和视频。新合并的公司还将专注于增加和
增强名人内容。

对于 Plixi 来说，被不断增长的社交网络收购是有意义的。很难与脸书、Foursquare 和 Twitter 竞争，它们都在其网络内提供照片分享。虽然还不清楚社交网络领域是否有洛克兹这样的竞争者，但该公司有针对性的人口统计和社交商务重点可能会被证明是一个可靠的赌注。
