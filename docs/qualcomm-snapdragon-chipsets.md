# 高通彻底改变了骁龙品牌，抛弃了令人困惑的芯片组名称

如果你有任何不是苹果或三星制造的智能手机，很可能是由高通的骁龙芯片组驱动的。现在，至于你能说出*拥有哪个*骁龙芯片组的可能性……是的，几乎为零。

问题:除了最重要的“骁龙”品牌之外，高通在区分哪个芯片组上做得并不好。MSM8260？MSM855T？MSM8930？OMGWTFBBQ？我这样做是为了我的*工作*，如果没有小抄，我*还是*无法告诉你哪个是哪个。

今天早上，高通宣布对骁龙品牌进行一次小小的改革。这会让事情不那么混乱。

简而言之，他们正在从几乎不可能记住的型号转向分层系统。他们首先会有四个层级(或者他们称之为“系统”):S1、S2、S3 和 S4。数字越高，规格越好。

例如，S1 是“大众市场”手机——高通手机的低端产品(1Ghz 或以下)。比如 HTC Status，或者 Droid Incredible。S2 是“高性能智能手机和平板电脑”(高达 1.4 Ghz)，如 HTC Thunderbolt，而 S3 是目前最高端的东西(高达 1.5Ghz)，如 EVO 3D。

S4 囊括了 1.6 Ghz 到 2.5 Ghz 之间的一切——正如你可能知道的，它实际上还不包括任何东西。该系列的第一批设备最早要到明年才会上市。

有趣的是，高通指出，S2 设备将永远被认为是 S2 设备——S2 不会成为 S1，S3 不会成为 S2，等等。据我所知，听起来高通计划随着时间的推移不断增加新的系统层级。男人，S10 就要 ***可笑了。***

几乎可以肯定的是，高通仍然会以这样或那样的方式为我们中的超级极客提供又大又丑的型号，以区分类似的芯片组——他们只是不希望有人真的记得它们。
