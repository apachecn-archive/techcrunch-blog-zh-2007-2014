# 微软推出第二屏幕功能

微软刚刚宣布了一项名为 Xbox SmartGlass 的新功能，这是一项“第二屏幕”功能，允许您在移动设备或 PC 的屏幕上查看媒体信息。

SmartGlass 允许您在多种设备上观看电影，例如，在平板电脑上观看电影，然后将视频传输到屏幕上，并在家中或外出时提供电视和大屏幕上播放的电影的详细媒体信息。

例如，在屏幕演示中，当动作出现在屏幕上时，SmartGlass 提供了《权力的游戏》的实时地图，为视频创建了一个“智能伴侣”。

SmartGlass 还允许您使用 Internet Explorer 在 Xbox 360 上浏览网页。平板电脑或手机变成了键盘，你可以轻松地浏览网页，而无需在客厅里安装物理键盘。

在游戏方面，SmartGlass 可以连接屏幕上正在玩的游戏，并允许玩家计划体育游戏的玩法或在平板电脑上查看库存。这项服务适用于 iOS 平板电脑和手机，以及 Win8 平板电脑和 Windows Phone 手机。

该功能与任天堂的 Wii U 控制器非常相似，但不需要专有硬件。微软称之为“为电视而改变的网络”
