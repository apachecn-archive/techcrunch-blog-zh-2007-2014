# 罗纳德·里根图书馆获得苹果 iPod 之旅

![](img/3778eb4ce45c6388e0c913282c238996.png "ipod_touch_4g")

罗纳德·里根总统图书馆最近进行了耗资 1500 万美元的技术改造。图书馆的技术令人印象深刻:180 个视频屏幕，两个提词器，一个绿色屏幕和 250 个 iPod 触摸被添加进来，使事情变得有趣。

[iPod touches](https://web.archive.org/web/20221208043105/http://www.tuaw.com/2011/02/08/ronald-reagan-library-adds-250-ipod-touch-tour-guides/)将用于引导游客参观图书馆，当人们走动时会有音频解说。此外，iPods 将允许客人拍摄视频和照片，以纪念这次经历，并在[脸书](https://web.archive.org/web/20221208043105/http://www.crunchgear.com/tag/facebook)和[推特](https://web.archive.org/web/20221208043105/http://www.crunchgear.com/tag/twitter)上分享你是如何有教养的。多好玩啊！
