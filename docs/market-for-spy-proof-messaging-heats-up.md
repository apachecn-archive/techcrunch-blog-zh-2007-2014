# 防间谍信息市场升温 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2013/07/12/market-for-spy-proof-messaging-heats-up/>

既然第四修正案不再是针对广泛的政府监控的保证，科技公司归还用户隐私的闸门已经打开。海盗湾联合创始人彼得·桑德的短信应用 Heml.is 已经达到了融资目标，而类似 Skype 的 Unsene 平台今天推出了自己的 Indiegogo 活动。

万一海盗湾没有惹恼足够多的政府人员，Sunde [希望建立一个防间谍的消息服务](https://web.archive.org/web/20221006121409/http://mashable.com/2013/07/10/hemlis/)作为短信或苹果 iMessage 的替代，据报道，这两种服务都受到情报机构的监控。[heml . is](https://web.archive.org/web/20221006121409/https://heml.is/)(“hem lis”在瑞典语中是“秘密”的意思)，使用一种被称为[端到端加密](https://web.archive.org/web/20221006121409/http://en.wikipedia.org/wiki/End-to-end_encryption)的安全协议，这意味着只有用户和发送者拥有密钥来解密通过互联网发送的信息。

Sunde 在视频中解释道:“我们决定建立一个信息平台，在这个平台上，没有人可以监视你，甚至我们也不行。”

Sunde 已经决定不在这项服务上出售广告，所以他正在向用户征集 10 万美元的启动资金。

苹果还声称，由于端到端的加密，它的 messages 服务是防 NSA 的，但 Ars Technica 的好人发现了一些豁免，允许政府机构监视短信。苹果既可以备份 iMessages，也可以让应用程序与之互动，从而让黑客获得访问权限。

Sunde 还警告用户，他们的应用程序不是 100%防间谍，但希望不那么容易受到间谍活动的影响。在撰写本文时，Heml.is 已经实现了 54，000 美元的融资目标。

回到美国，另一个名为 [Unsene](https://web.archive.org/web/20221006121409/http://www.indiegogo.com/projects/unsene-secure-and-private-chat-calls-and-photo-and-file-sharing) 的安全通信平台发起了一项 Indiegogo 筹资活动，筹集了 3 万美元。

Unsene 目前提供加密的网络信息、视频通话和文件共享，本质上是 Skype 的替代产品。一旦 Indiegogo 活动结束，该服务于 8 月中旬向公众开放，该计划将扩展到提供个人防火墙和一个可以取代电子邮件的信息系统。

Unsene 的首席执行官和联合创始人克里斯·基茨告诉我们，Skype 和电子邮件服务的问题在于，信息通过一个中央服务器，而不是点对点传输。如果政府要求，谷歌或雅虎会交出你的电子邮件。与 Heml.is 非常相似，Unsene 将加密信息的密钥放在个人手中。

Unsene 不保护现有的 Skype 或 Gmail 帐户，因此两个人必须同时拥有该系统才能进行通信。幸运的是，Unsene 的标准信息包是免费的，所以这不应该成为潜在用户的巨大障碍。问题是，Unsene 是否能够让用户远离 Skype 和 Gmail，人们是否会出于不同的目的使用这两个系统，以及他们是否真的那么关心保护他们的通信。

Kitze 告诉 TechCrunch，很难理解互联网间谍活动的范围，但目标是将隐私放回个人手中。他们想要多强烈地保护他们的信息取决于他们自己，所以 Unsene 正在为那些想要更多安全的人提供更高等级的高级套餐。Kitze 说，土耳其、中国和中东的人都对 Unsene 感兴趣。华尔街、政治组织和律师也表示了兴趣。

最终，目标是让应用开发者能够将他们的产品集成到 Usene 的 API 中，比如支付系统。

不过，Kitze 承认互联网安全是一个容易出错的系统。

“我认为没有人能在营销中宣称他们的锁牢不可破，但这将是一场猫捉老鼠的游戏。我们会装上更大的锁，人们会想办法进来的。”
