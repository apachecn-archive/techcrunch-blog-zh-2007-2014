# GameStop 即将推出安卓平板电脑

我们已经提到过几次 [GameStop 可能会推出自己的 GameStop 品牌的平板电脑](https://web.archive.org/web/20230205015206/https://techcrunch.com/2011/04/06/gamestops-future-plans-include-tablets-and-tvs/)。GameStop 总裁托尼巴特尔说，事实证明，这款神秘的平板电脑已经处于测试阶段。这款平板电脑不是由 GameStop 开发的，但应该是商店里出售的“GameStop 认证的游戏平台”。

当托尼·巴特尔告诉 GamaSutra，GameStop 计划进军平板电脑领域时，他说如果他们找不到符合自己需求的平板电脑，他们就会自己开发。显然，某个地方的平板电脑制造商有它需要的东西，尽管 GameStop 没有透露他们正在与哪个开发商合作。

“我们肯定已经选好了，”巴特尔告诉 [GamesIndustry.biz](https://web.archive.org/web/20230205015206/http://www.gamesindustry.biz/articles/2011-09-12-gamestop-chooses-android-for-own-brand-gaming-tablet) 。“我们现在正处于测试阶段。但我们对推出这款平板电脑的前景感到兴奋。我称之为“GameStop 认证的游戏平台”。我们查看了所有的平板电脑，这些是真正适用于游戏的平板电脑，我们将为您提供一些在其他地方无法获得的优势。"

巴特尔对细节守口如瓶。到目前为止，我们真正知道的是，这款平板电脑将运行安卓系统，预装游戏，GameStop 计划在未来某个时候将主机游戏整合到平板电脑中。最近有报道称，GameStop】计划在其商店销售 iPod、iPhone 和 iPad。

虽然这两种产品之间会有明显的差异——即 iOS 和 Android——但这两款平板电脑在 GameStop 货架上并排放置几个月后的销售统计数据会很有趣。GameStop 计划如何追加销售自己的 slate 还不知道，但肯定有必要让 iPad 对着房间另一端的顾客微笑。
