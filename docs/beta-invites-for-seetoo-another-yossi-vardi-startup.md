# 测试版邀请另一家 Yossi Vardi 创业公司 SeeToo

[![seetoo-logo.png](img/06f79c970bbbfca408541963387103bf.png)](https://web.archive.org/web/20221206194937/http://www.seetoo.com/)

我猜今天是 TechCrunch 的 [Yossi Vardi 创业](https://web.archive.org/web/20221206194937/http://www.beta.techcrunch.com/2007/11/27/yossi-vardi-invests-in-media-timeline-startup-allofme/)日。ICQ 背后多产的以色列投资者现在正在支持一个名为 [SeeToo](https://web.archive.org/web/20221206194937/http://www.seetoo.com/) 的视频聊天服务，该服务创建于 2007 年 1 月。SeeToo 一个月前完成了约 100 万美元的天使轮投资，Vardi 是最大的投资者，也是一名活跃的董事会成员。这家位于纽约市的以色列初创公司上周悄悄推出了一个私人测试版，我们收到了 500 份邀请，邀请首批 TechCrunch 读者注册。

SeeToo 背后的基本概念是一个共享的聊天室，在这里你可以和正在聊天的人同时观看视频。这是为了你的孩子或旅行的个人视频，而不是为了和朋友一起看好莱坞电影，尽管这在技术上也是可能的。今天早些时候，市场营销和业务开发主管 Yoav Ilan 带我进行了现场演示。他给我看了他 7 个月大的孩子和一艘大游艇的录像，那是他在纽约的公寓里拍的。当视频播放的时候，我们在聊天框里打字。我不需要下载任何东西。我只是点击了他发给我的一个网站链接，就成功了。

SeeToo 并不是唯一一家以这种方式将视频和聊天结合起来的公司。 [PalTalk](https://web.archive.org/web/20221206194937/http://www.paltalk.com/) 允许在一个聊天室中与比 SeeToo 目前支持的人数多得多的人同时共享观看视频(目前只有两方，但这可能会改变)。你可以用 Meebo Rooms 做类似的事情，它可以让你和一群朋友一起观看 YouTube 和其他嵌入式视频。

SeeToo 可以轻松地将更多的人加入聊天，并正在对此进行研究。但主要的区别在于 SeeToo 对点对点技术的巧妙运用。你从不上传视频。相反，如果你想分享一个视频，你可以下载一个 600 千字节的小应用程序，它可以把你桌面上的任何视频压缩，然后直接从你的电脑上传到主持聊天的 SeeToo 网页。该应用程序目前只适用于 Windows，但 Mac 版本将于明年推出。

聊天结束后，视频会从网络上消失(它会保留在您的电脑上)。例如，游艇视频是一个 120 MB 的文件。SeeToo 不需要托管或存储任何视频，消费者也不需要等待视频上传后才能分享。服务的简单性是它最大的优点。
