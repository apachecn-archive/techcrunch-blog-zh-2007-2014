CloudBees筹集了1120万美元的C轮融资，由威瑞森风险投资公司牵头，以扩大其以Java为中心的企业平台即服务(PaaS)

CloudBees ，一个以Java为中心的企业平台即服务，我们在过去报道过几次，今天宣布它已经筹集了由威瑞森风险投资公司领导的1120万美元的C轮融资。现有投资者Matrix Partners和Lightspeed Ventures也参与了此轮投资，还有新投资者Blue Cloud Ventures。

自2010年成立以来，这项服务已经筹集了2570万美元。该公司计划利用这笔资金通过扩大其功能集和全球销售足迹来推动收入增长，并“扩大CloudBees品牌的覆盖范围”

CloudBees之前与威瑞森有过合作关系，尽管两家公司在今年早些时候宣布了合作关系。作为交易的一部分，威瑞森在其最近推出的威瑞森云平台上提供CloudBees的服务。该公司告诉我，它选择威瑞森作为主要投资者，因为它“认真对待成为顶级投资者和投资最好的公司。”

就像类似的以企业为中心的PaaS服务一样，CloudBees允许开发人员在公司的平台上构建和测试他们的应用程序。然而，让CloudBees与众不同的是它对Java的专注。

该公司也支持几乎所有基于JVM的语言，包括Spring、JRuby、Grails、Scala、Groovy等。CloudBees上的开发者使用Jenkins来跟踪Git和SVN库中的代码变化，并使用Maven、Gradle和Ant的链接来触发构建。
