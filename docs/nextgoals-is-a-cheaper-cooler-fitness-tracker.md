# NextGoals 是一款更便宜、更酷的健身追踪器

[NextGoals](https://web.archive.org/web/20230204225302/http://nextgoals.com/)

是一个试图激励昏睡的人去锻炼和保持健康的应用程序。它是如何工作的？让你锻炼，然后确认你真的从沙发上滚下来去了健身房。

该系统使用一种叫做 G 波段的设备。这是一个戴在手腕上的计步器，可以在你的电脑和手机上提供实时数据。你可以衡量自己的进步，也可以通过各种社交网络与朋友和家人联系。

仪表板非常干净和智能，它会根据您当前的活动发送状态更新。你也可以使用 NextGoals 的协作工具与朋友一起锻炼身体。

与其他健康追踪器相比，G 波段相当引人注目。首先，它只有 29 美元，比任何其他迷你跟踪设备都便宜。他们立即将产品瞄准中国，以后可能会转移到海外。他们还与该国的健身房合作。
