# 感恩节+黑色星期五手机流量比 2010 年增长 60%

根据 Usablenet 的一项新研究，移动正在成为电子商务的一种媒介，用户在访问实体店之前通过手机和平板电脑进行交易，该公司为包括 JCPenney、Aeropostale 和 REI 在内的 100 家美国顶级零售商提供移动网站，在感恩节和黑色星期五期间跟踪了 1800 万页面浏览量和 100 万移动用户。它看到其客户的移动流量比去年同期增长了 60%，感恩节发送的流量超过了第二天。Usablenet 还发现，iOS 设备占据了 42%的流量，超过了 Android，并击败了 Windows 和诺基亚设备的微小流量。

今天早些时候，Leena 报告称[黑色星期五在线零售支出今年增长了 24.3%](https://web.archive.org/web/20221202023616/https://beta.techcrunch.com/2011/11/26/ibm-black-friday-online-retail-spending-up-24-3-percent/)，移动端增加了其在总流量和销售额中的份额。然而，除了直接推动销售之外，手机还促进了线下销售和产品提货。Usablenet 告诉我，“感恩节活动的重点是寻找和购买交易，如当天的交易，这是由电子邮件营销以及在线购买和店内提货选项的高使用率驱动的。”

在黑色星期五，移动使用集中在寻找商店位置、浏览评论和访问之前保存的愿望清单，而人们则在当地商家的过道中行走。这些见解可以帮助零售商为明年做计划。他们应该在感恩节早期发送电子邮件营销和交易通知，因为这是人们决定去哪里购物的时候。这可能比在黑色星期五开始时分发促销活动效果更好，因为顾客可能已经制定了购物日程。

移动操作系统的流量也反映了销售额，iOS 的降幅最大，但低于去年。安卓系统正在崛起，从 2010 年占移动流量的 28%上升到 34%。黑莓仍占相当可观的 15%的移动流量，而 Windows 和诺基亚各占不到 3%。如果这种趋势持续下去，到明年，大型零售商提供 Android 应用程序可能会和 iOS 应用程序一样重要。
