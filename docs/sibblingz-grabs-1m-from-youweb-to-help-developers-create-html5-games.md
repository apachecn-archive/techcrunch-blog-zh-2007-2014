# Sibblingz 从 YouWeb 获得 100 万美元，帮助开发者开发 HTML5 游戏 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2011/08/04/sibblingz-grabs-1m-from-youweb-to-help-developers-create-html5-games/>

多设备社交游戏平台 [Sibblingz](https://web.archive.org/web/20230204203530/http://www.crunchbase.com/company/sibblingz) 已经从孵化器 YouWeb 融资 100 万美元。YouWeb 之前在 2008 年向 Sibblingz 投资了 50 万美元的种子资金。

Sibblingz [允许开发者](https://web.archive.org/web/20230204203530/https://techcrunch.com/2010/05/12/sibblingz-launches-multi-platform-social-game-engine-to-the-masses/)在脸书以及 iOS 和 Android 设备上同时创建游戏，允许玩家在设备之间切换时继续相同的社交游戏。该平台还为开发者提供了利用虚拟商品从免费游戏中赚钱的能力。

该公司基于云的游戏创作平台 [Spaceport](https://web.archive.org/web/20230204203530/https://techcrunch.com/2011/04/27/sibblingz-launches-new-version-of-multi-device-social-gaming-platform/) ，允许游戏开发者使用 Javascript 构建一个游戏，并让它在所有智能手机操作系统上运行，具有类似原生应用的体验和快速的性能。基于网络的游戏作为混合 html 5-native 应用程序运行，游戏代码用简单的脚本语言编写一次，因此它们可以在任何设备上运行。开发者也可以使用 Spaceport 创建原生 iOS 应用。

对于使用该平台的开发人员来说，更新过程也相对简单。有了 Spaceport，开发者只需要在 HTML 页面中为他们的游戏修改一次 JavaScript 代码。这样做之后，他们将看到游戏在每台设备上的即时更新。

YouWeb 的创始人 Peter Relan 认为，随着移动游戏的兴起和开发者将 HTML5 视为跨平台技术，游戏领域目前正处于一个重要的十字路口。正如我的同事 MG Siegler [在今年夏天早些时候报道的](https://web.archive.org/web/20230204203530/https://techcrunch.com/2011/06/15/facebook-project-spartan/)，脸书有一个秘密计划，叫做 [Project Spartan](https://web.archive.org/web/20230204203530/https://techcrunch.com/2011/06/30/facebook-ipad-spartan/) ，通过 HTML5 将应用程序带到移动网络上。

Relan 说开发者们都在研究移动化的方法。在这中间坐落着 Sibblingz。Sibblingz 的 Spaceport 允许开发人员通过适用于 iOS 和 Android 的 HTML5 画布渲染引擎创建与 Project Spartan 兼容的 HTML5 游戏。

Sibblingz 创始人 Ben Savage 表示，该公司正在与多家脸书游戏开发商合作，这些开发商正在为“斯巴达项目”做准备。他们都想成为手机游戏界的 Zynga。

Savage 和 Relan 认为 Sibblingz 可以通过其原生渲染和 HTML5 游戏创作技术来帮助实现这一目标。

盈利的 Sibblingz 已经从使用其技术的游戏分享费用中获得了“每年数百万美元”的收入。随着越来越多的开发者利用游戏引擎来创建基于 HTML5 的跨平台游戏，这笔资金将帮助 Sibblingz 扩大 Spaceport。

Spaceport 将很快公布其脸书游戏合作伙伴(我们知道该引擎正在由另一家 YouWeb 公司 CrowdStar 使用)，并将在未来几个月向所有脸书和移动开发者公开。
