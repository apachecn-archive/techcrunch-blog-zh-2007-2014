# News360 将 Edelman 和 Network For Good 作为广告商加入其新的赞助内容计划

本周早些时候，个性化新闻应用程序 [News360](https://web.archive.org/web/20221006214938/http://www.news360.com/) [的制造商宣布](https://web.archive.org/web/20221006214938/http://paidcontent.org/2013/06/06/news360-puts-native-advertising-into-its-personal-news-app/)他们已经启动了一项赞助内容计划，公司可以付费向相关消费者推广他们的内容。

甚至在宣布之前，首席执行官[罗曼·卡拉钦斯基已经在谈论他将广告引入应用](https://web.archive.org/web/20221006214938/https://beta.techcrunch.com/2013/03/03/news360-android-ipad-demo/)的计划，News360 已经为 Cincom 系统开展了一次试点活动。现在它已经为两个新的广告客户发起了活动，营销代理 Edelman 和筹款平台 Network for Good。

这并不完全反映了这一推出的成功，因为 News360 在上周之前就与两家广告商签约了。然而，他们一起暗示了广告计划的潜在广度。

首先，卡拉钦斯基指出，当他谈到赞助内容时，他不仅仅指文章——视频也是活动的一部分。卡拉钦斯基说，超越文本确实带来了独特的挑战。毕竟，News360 向广告商兜售的是它对用户兴趣的理解，以及利用现有内容推荐技术投放与这些用户相关的广告的能力。卡拉钦斯基说，要做到这一点，这里需要“更多的手工工作”，特别是转录视频的部分内容。

Karachinsky 还指出，由于爱德曼的“更智能的燃料未来”活动包括一个潜在的政治话题，即汽油和乙醇，它可以利用 News360 的能力，根据可能的政治派别锁定内容。这意味着赞助内容可以针对更有可能对其信息产生共鸣的读者。

虽然现在知道这些新努力的效果还为时过早，但在最初的 Cincom 活动中，News360 表示，它在 iPhone 上实现了令人印象深刻的 12.5%的独立点击率，在 iPad 上实现了 5%的独立点击率。即使这些是独特的点击率(换句话说，如果有人在点击之前多次看到一个推广的标题，他们仍然被算作一次点击)，Karachinsky 说，与标准的点击率相比，这种方法测量结果并没有太大的区别。
