# 摩托罗拉 Droid X2 升级到 Android 2.3 姜饼

Droid X2 是那种你很快就能学会不通过边框来判断的手机。尽管这款手机的硬件与其功能较弱的前代产品几乎相同，但它还是有一些非常显著的改进。但是仍然缺少一样东西:[姜饼](https://web.archive.org/web/20230204233603/https://techcrunch.com/tag/gingerbread)。

幸运的是，[威瑞森](https://web.archive.org/web/20230204233603/https://techcrunch.com/tag/verizon)今天早上发布了 Android 2.3 姜饼更新，所以机器人 X2 可以继续伪装成 2010 年的手机，同时表现得像刚从盒子里拿出来一样。

当然，改进的复制和粘贴、更快的性能和更详细的电池信息都是姜饼更新的一部分，但[摩托罗拉](https://web.archive.org/web/20230204233603/https://techcrunch.com/tag/motorola)也有自己的一些好东西。MotoBlur UI 现在将展示新的颜色和小部件，基于软件的键盘已经通过多点触摸得到了增强。相机应用程序现在允许对照片进行地理标记，日历应用程序也有一些改进。

更新通知应该会自动出现在 Droid X2 上，但如果你感到不安，请在设置应用的系统更新部分手动检查更新。但一定要先接入 WiFi 网络，否则你可能要等一段时间:更新是 101MB。
