10亿次震动后，Urbanspoon追逐OpenTable TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2011/08/05/one-billion-shakes-urbanspoon-opentable/

手机上最受欢迎的餐厅应用之一是Urbanspoon ( iTunes link )。它仍然是iPhone上十大旅游应用之一。该应用程序的标志性手势是摇一摇，以找到附近的餐馆。Urbanspoon刚刚通过了10亿次摇一摇，在所有移动平台(iOS、Android、黑莓)上已经被下载了2000万次。2010年3月，点击 5亿次，所以不到一年半就翻了一倍。

但是奶昔的价值到底是什么？嗯，这些应用程序和网站一样运行广告。这些应用和网络每月有1亿的浏览量，摇一摇就变成了广告印象。但这里真正重要的是餐厅预订。Urbanspoon要去追 OpenTable 。

去年，Urbanspoon 推出了Rezbook ，这是一款iPad应用程序，允许餐馆直接从Urbanspoon应用程序和网站预订。“两年前，在线预订系统需要大量的前期投资，”IAC负责Urbanspoon(和CityGrid Media)的高级副总裁卡拉·诺特曼指出。现在，他们只需要一台iPad和一个简单的应用程序。

目前，在Urbanspoon的80万家餐馆中，只有大约800家餐馆在Rezbook系统中。当他们出现在Urbanspoon时，用户可以预订一张桌子。诺特曼说，Urbanspoon每月在这800家餐厅安排19，000名用餐者，也就是说每家餐厅每月约有24名用餐者。相比之下，OpenTable平均每个月为每家餐厅提供450个座位，但它已经存在很长时间了。

Urbanspoon只挖掘了其列表中的千分之一的餐馆，在测试了这项服务后，现在正开始更加认真地与它们签约。“随着越来越多的消费者认为Urbanspoon是一个发现和跟踪实时预订的地方，我们应该会看到这个数字的增长，”Nortman说。

她将自己的移动应用视为竞争优势，因为它更直接。通过Urbanspoon应用程序进行的预订，22%在一小时内完成，77%在24小时内完成。当然，OpenTable也有一个很棒的移动预订应用程序，现在覆盖了更多的餐厅。但Urbanspoon看到了一个机会，并正在努力争取。
