# LG 通过 3D 游戏转换器支持 Optimus 3D 应用程序库

引人注目的内容是确保 3D 智能手机不仅仅是昙花一现的关键，而现在，没有太多的内容在流传。Optimus 3D 手机的制造商 LG 旨在通过最近发布的新 3D 游戏转换软件来解决内容问题。

LG 的新 3D 游戏转换器将在 2011 年柏林 IFA 上首次展示，正如其名字所暗示的那样。在 Optimus 3D 上启动转换器后，大约 50 款兼容的 2D 游戏将接受 3D 改造，而不支持的 OpenGL 游戏可以手动调整以实现所需的效果。

这些游戏将得到相当全面的检查，或者说 LG 的 3D 福音传道者 Henry Nho 博士会让我们相信:“LG 的 3D 游戏转换器会根据每个物体的位置自动识别深度信息，并为每只眼睛分离 2D 图形图像。使用现有的深度信息，3D 游戏转换器生成两幅不同的图像——一幅用于前景，一幅用于背景。”

LG 承诺，到今年年底，另外 50 款游戏将获得 3D 支持，这是对现有 3D 内容库的重大补充。对于游戏开发者和他们的客户来说，这是一个双赢的局面:开发者可以轻松地提供 3D 功能，而客户可以在没有额外成本的情况下扩展他们的 3D 内容库。该转换器将作为 Optimus 3D 新维护版本的一部分，目前计划于 10 月发布。

值得称赞的是，LG 似乎意识到，尽管 3D 智能手机噱头十足，但它们的生死取决于用户体验的质量。一些用户会更乐意拍摄立体视频，直到他们脸色发青，但这种新鲜感通常只会持续很久。该公告通过提供更多内容为 3D 体验增加了一点价值，但不可避免的问题是“这就够了吗？”

游戏是一回事，但为了超越它们的噱头地位，3D 智能手机需要更多的应用程序，这些应用程序是从一开始就考虑到 3D 设计的，而不是事后才想到的。
