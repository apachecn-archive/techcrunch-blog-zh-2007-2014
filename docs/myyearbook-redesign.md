# 我的年鉴推出众包重新设计

[![](img/d71ed7b34381172659a46b4e03c87bf8.png)](https://web.archive.org/web/20221208221028/http://www.myyearbook.com/) 本周，社交网站 [MyYearbook](https://web.archive.org/web/20221208221028/http://www.myyearbook.com/) 推出了全新的设计，改变了一些关键的用户界面元素，以更好地反映人们使用该网站的方式。重新设计更有趣的一点不是它的外观，而是它实际上是如何组装的。MyYearbook 不是简单地推出一个新设计，而是实际上向用户群征求建议，然后起草一些模型供用户投票。结果呢？用户喜欢新设计的比例超过 3 比 1，没有其他网站在重新设计中看到的那种[反弹](https://web.archive.org/web/20221208221028/http://www.beta.techcrunch.com/2009/03/19/facebook-polls-users-on-redesign-94-hate-it/)。

该网站最明显的变化是标志，它已经从卡通变成了更专业的东西(尽管它仍然有几个笑脸)。导航栏也被简化了。首席执行官杰夫·库克表示，随着时间的推移，MyYearBook 已经将一系列游戏菜单选项集成到了主导航区域，但没有对真正使该网站成为社交网络的核心功能进行太多区分。现在，它们已被移到页眉的不同部分。

这实际上只是两部分变化的第一阶段；很快，我的年鉴将修改个人资料页面，以更加突出 Chatter 流。Chatter 流在视觉上类似于脸书的新闻源，但更强调结识新朋友，而不仅仅是看到朋友的内容。默认的 Chatter 摘要显示您附近的其他用户的最近内容更新(假设您已经告诉我的年鉴您在哪里)，并且有选项卡仅查看您的朋友的更新。

因为 MyYearBook 是一个社交网络，它显然知道每个用户的一些关键人口统计信息。有了这些，网站收集了一些有趣的数据，分析哪些成员支持新设计，哪些不支持。结论呢？虽然在所有参与调查的成员中，新设计比新设计受欢迎的比例超过 3 比 1，但当考虑到年龄时，有一个明显的趋势:用户年龄越大，他们越反感新设计。这可能并不特别令人惊讶，但有趣的是看到下面的图表是多么一致。
