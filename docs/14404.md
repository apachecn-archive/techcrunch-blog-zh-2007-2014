# 阿里巴巴集团旗下的淘宝被美国从“恶名市场”名单中移除

美国政府周四宣布，中国电子商务巨头[阿里巴巴集团](https://web.archive.org/web/20221221050144/http://www.alibaba.com/)旗下的[淘宝商城](https://web.archive.org/web/20221221050144/http://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&ved=0CDEQFjAA&url=http%3A%2F%2Fwww.taobao.com%2F&ei=y7zKUNzwIcn4mAW9-IDYCg&usg=AFQjCNHZa0G51qN9CeUToajXzTeE1oc9lw&bvm=bv.1355325884,d.dGY)已经从其年度全球最“恶名市场”名单中移除，这是一个盗版和假冒商品猖獗的市场。

美国贸易代表办公室的报告称，淘宝“已经与版权所有者合作，大幅减少通过其网站销售的侵权产品，并承诺继续努力简化投诉程序，以进一步减少假冒产品。”此外，由领先的中国互联网公司搜狐运营的中国搜索引擎搜狗被从名单中删除，“因为有报道称，搜狗也做出了显著努力，与版权所有者合作，解决其网站上侵权内容的问题。”

然而，这一举动可能更多的是一种外交策略，而不是假货消失取得任何进展的迹象(例如，淘宝上仍然有大量便宜得令人怀疑的[蔻驰](https://web.archive.org/web/20221221050144/http://search8.taobao.com/search?q=coach&commend=all&ssid=s5-e&pid=mm_14507416_2297358_8935934)、[路易威登](https://web.archive.org/web/20221221050144/http://search8.taobao.com/search?q=Louis+Vuitton+&pid=mm_14507416_2297358_8935934&unid=0&mode=63&from_bt=1&initiative_id=staobaoz_20121214)和[爱马仕](https://web.archive.org/web/20221221050144/http://search8.taobao.com/search?q=hermes&ex_q=&fs=1&filterFineness=&atype=&from_bt=1&unid=0&mode=63&pid=mm_14507416_2297358_8935934&initiative_id=staobaoz_20121214)商品)。今年的美中商贸联合委员会将讨论“知识产权执法”等议题，会议定于下周举行。早在一月份，中国商务部[对淘宝被列入名单](https://web.archive.org/web/20221221050144/http://online.wsj.com/article/SB10001424052970203735304577168473341149692.html)表示“强烈关注和不满”，并呼吁美国“做出公正的评估，避免给中国企业造成不必要的负面影响。”淘宝和搜狗的回归或许会让会议上的讨论不那么乏味。

另一家登上“恶名市场”名单的中国公司是搜索巨头百度(被称为中国的谷歌)，它在 2011 年与环球音乐集团、华纳音乐集团和索尼公司达成协议，向其社交音乐平台上的版权所有者支付费用后，于 2011 年从名单中删除。
