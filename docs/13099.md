# 乐天首席执行官谈及 1 亿美元的 Pinterest 融资:我们希望 Pinterest 用户使用我们的 ID TechCrunch 发布图片并购买

日本电子商务巨头乐天主导了对 Pinterest 的 1 亿美元投资，对该公司的估值为 15 亿美元，这将为基于图像的社交网络的下一阶段增长做出两大贡献:将基于图像的社交网络带入新的国际市场的资金和商业模式。

首先，乐天的本土市场日本，“Pinterest 在那里增长非常快，”乐天的首席执行官三木谷浩史在接受 TechCrunch 采访时指出。他希望乐天通过使用乐天的服务成为网上购物的基础，与它一起成长。

这方面已经有了一些基础。首先，乐天已经通过至少几个官方账号锁定了 Pinterest:乐天商业和乐天旅游。

另一方面，乐天在中国已有电子商务业务。三木谷浩史指出，75%的日本网民——相当于大约 8000 万人——已经拥有一个乐天 ID——这类似于苹果 ID 或亚马逊 ID，因为有与之相关的支付细节。

正是这个 ID 有可能成为 Pinterest 上一项商业服务的关键:“我们希望让我们的用户能够用我们的 ID 来标记他们自己的图片，”他说。“用户可以用它来点击和购买，未来我们可以创造更多的新服务。”他指出,“丰富的、图形化的社交网络”可以用于“使用乐天 ID 的许多有趣的想法”据 TechCrunch 了解，另一个领域是用户使用 Rakuten IDs 登录，锁定图片，然后在 Rakuten 自己的 Rakuten Ichiba 网站上使用这些 pin 从 Pinterest 购买商品。

因为 Pinterest 现在非常流行，它可以被用来重新点燃乐天的一些传统业务。该公司有很多三木谷浩史称之为“沉睡的顾客”。这些人有身份证，但不是乐天服务的固定用户。“这是让他们重新开始使用这些乐天 id 的好方法。”

他希望看到更多发展的另一个领域是移动商务领域。三木谷浩史告诉我，乐天在日本的销售额已经有 25%来自移动设备，而且这一比例还在上升。“在这方面，应该会与 Pinterest 产生巨大的协同效应，”他说。“我们将利用我们在日本的影响力来推广他们的应用。”

**国际增长**。这笔 1 亿美元的投资由乐天牵头，参与投资的有 Andreessen Horowitz、Bessemer Venture Partners 和 FirstMark Capital，以及一些天使投资者，这笔投资也将为 Pinterest 提供财务实力，将其服务扩展到新的国际市场。或许最重要的是，Pinterest 作为一种国际产品有很大的潜力——一种可以跨越国界的产品——因为虽然脸书和 Twitter 等网站是文本主导的，但 Pinterest 专注于图像，因此较少受到语言障碍的限制。

当然，Pinterest 今天和未来的许多增长都是非商业性的，但商业潜力也很大。(前几天我们写了另一个例子， [Curalate](https://web.archive.org/web/20230204140852/https://techcrunch.com/2012/05/15/look-out-pinterest-marketing-platform-curalate-lands-750k-seed-from-nea-first-round-mentortech/) ，它为品牌创建了一个社交营销服务，可以直观地跟踪他们的图片在网站上如何以及在哪里被使用。)在这一趋势下，乐天也在考虑如何在日本以外的国际业务中利用对 Pinterest 的投资。

三木谷浩史指出，尽管脸书是一个“极其强大的社交网络”，但在购物和电子商务方面，Pinterest 的图片导向服务“潜力更大”

他说，脸书和推特是把你和你的朋友和联系人联系起来，而 Pinterest 是通过图片把有相同兴趣的人联系起来。除了产品摆在那里让你看的事实之外，你还可以想象这种社交设置如何发展成一种商业模式(这里想到的就是团购)。

乐天的控股公司远不止日本，还包括美国的 Buy.com、Kobo 电子阅读器和电子书、法国的 Priceminister 等公司，其业务遍布 10 个国家，总共覆盖 17 个国家。三木谷浩史表示，他希望它想在日本打下的基础也能扩展到其他地方。

例如，三木谷指出，Kobo 已经与脸书建立了“良好的合作伙伴关系”，鼓励人们使用 Kobo 发布节选和阅读更多内容，它也希望将这种合作扩展到 Pinterest:“脸书是 Kobo 目前发展如此迅速的原因，”他说。"我想我们也会在 Pinterest 上看到更多 Kobo 的身影."buy.com——它正慢慢地、逐渐地被重新命名为乐天——是另一个你能想象到的与 Pinterest 联系更紧密的网站。

但这并不排除与其他电子商务公司的其他合作关系。“我们对其他电子商务合作伙伴完全开放，”他说。“Pinterest 应该和他们合作。”
